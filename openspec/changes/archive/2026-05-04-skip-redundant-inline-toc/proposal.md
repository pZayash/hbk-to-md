# Why

Встроенное оглавление в content-страницах дублирует ссылки, которые уже присутствуют в теле после конвертации HTML→Markdown (например, «Константы» и подраздел «Поля»). Это увеличивает шум и токены без навигационной пользы. Нужно вставлять блок TOC только когда он добавляет хотя бы одну новую ссылку на существующую страницу.

## What Changes

- Решение **целиком**: либо вставляется полный блок `<!-- toc:start -->` … `## Оглавление` … `<!-- toc:end -->` как сейчас по структуре дерева, либо блок **не вставляется** — без частичного вырезания отдельных строк внутри TOC.
- **Критерий избыточности**: если каждая ссылка, которая попала бы в TOC с непустым путём к **существующему** в `--out` файлу `.md`, уже встречается в исходном тексте страницы (до крошек и до TOC) как литерал `](<тот же относительный путь>)`, TOC не вставляется.
- **Ссылки без пути** (`](` с пустым target) в TOC **не выводятся**; для проверки избыточности они **не учитываются** (считаются «всегда покрытыми»). То же для целей, указывающих на **несуществующий** в выходном каталоге файл.
- **Пустой TOC**: если после отбора не остаётся ни одной ссылки на существующую страницу, блок TOC **не вставляется**; при повторном прогоне существующий пустой/устаревший блок удаляется, если логика требует отсутствия TOC.
- Отдельного режима «только перестроить TOC» нет — только в рамках обычного `convert.py`.

## Capabilities

### New Capabilities

_(нет — поведение остаётся в рамках конвертера и существующего требования про парную content-страницу)_

### Modified Capabilities

- `hbk-to-md-converter`: уточнено требование к встроенному оглавлению на content-страницах с парным `prefix` — условная вставка целиком, фильтрация пустых целей, критерий «все целевые пути уже в теле», проверка существования целевого файла в `--out`.

## Impact

- `convert.py`: `render_inline_toc`, `inject_inline_toc_into_content`, этап инжекции TOC (в связке с `inject_all_breadcrumbs` / запись в `out_dir`).
- Тесты: `tests/test_toc.py` — юнит-тесты на критерий вставки и краевые случаи.
- Vault: меньше блоков `## Оглавление` там, где навигация уже продублирована в теле; исчезновение битых строк `]()`.
