ДиаграммаГанта (GanttChart)

ПолучитьКартинку (GetPicture)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьКартинку(<Ширина>, <Высота>, <ФорматКартинки>)

Параметры:

<Ширина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Ширина картинки в пикселях. Если <Ширина> или <Высота> = 0, то на клиенте в неформе клиентского приложения используются текущие размеры диаграммы Ганта. В остальных случаях используются размеры по умолчанию.  
Значение по умолчанию: 0.

<Высота> (необязательный)

Тип: [Число](lang__def_Number.md).   
Высота картинки в пикселях. Если <Высота> или <Ширина> = 0, то на клиенте в неформе клиентского приложения используются текущие размеры диаграммы Ганта. В остальных случаях используются размеры по умолчанию.  
Значение по умолчанию: 0.

<ФорматКартинки> (необязательный)

Тип: [ФорматКартинки](ФорматКартинки_(PictureFormat).md).   
Задает формат картинки.  
Допустимые значения для растровых картинок: [BMP](ФорматКартинки.BMP_(PictureFormat.BMP).md), [PNG](ФорматКартинки.PNG_(PictureFormat.PNG).md), [SVG](ФорматКартинки.SVG_(PictureFormat.SVG).md).  
Значение по умолчанию: [BMP](ФорматКартинки.BMP_(PictureFormat.BMP).md).

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md).   

Описание:

Получает картинку диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
