Файл (File)

ЭтоКаталог (IsDirectory)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоКаталог()

Возвращаемое значение:

Тип: Булево.   
Истина - объект соответствует каталогу; Ложь - объект соответствует файлу.

Описание:

Определяет, соответствует ли файловый объект каталогу или файлу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применим только к файлам, которые реально существуют.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
