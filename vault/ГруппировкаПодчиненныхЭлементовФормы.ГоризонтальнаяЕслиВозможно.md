ГруппировкаПодчиненныхЭлементовФормы (ChildFormItemsGroup)

ГоризонтальнаяЕслиВозможно (HorizontalIfPossible)

Доступен, начиная с версии 8.3.7.

Описание:

Горизонтальная группировка при наличии достаточного места по ширине формы.  
В противном случае - вертикальная группировка.

Использование в версии:

Доступен, начиная с версии 8.3.7.
