Расширение элементов управления принадлежащих табличному документу (Spreadsheet document controls extension)

ПолучитьДействие (GetAction)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДействие(<ИмяСобытия>)

Параметры:

<ИмяСобытия> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя события, для которого необходимо получить действие.

Возвращаемое значение:

Тип: [Действие](Действие_(Action).md).   

Описание:

Получает действие, которое будет исполняться при возникновении указанного события.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
