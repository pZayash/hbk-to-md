ОбъектМетаданных: Документ (MetadataObject: Document)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](ДокументМенеджер.Имя_документа.ОбработкаПослеЗаписиВерсийИсторииДанных_(DocumentManager.Document_name.AfterWriteDataHistoryVersionsProcessing).md) предоставляется возможность обработать запись версии после обновления истории.  
При записи данных указанное в конфигураторе значение устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_Документ.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных_(MetadataObject_Document.ExecuteAfterWriteDataHistoryVersionProcessing).md). Значение свойства можно переопределить.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
