КолонкиОписанияИсточникаДанных (DataSourceDescriptionColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаОписанияИсточникаДанных](КолонкаОписанияИсточникаДанных_(DataSourceDescriptionColumn).md).   
Искомая колонка.

Возвращаемое значение:

Индекс колонки, или -1, в случае, если колонка отсутствует в коллекции.

Описание:

Получить индекс колонки в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
