ОтчетОбОшибке (ErrorReport)

Записать (Write)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Записать(<Файл>, <Интерактивно>)

Параметры:

<Файл> (необязательный)

Тип: Строка.   
Имя файла на диске, в который будет записан отчет.

<Интерактивно> (необязательный)

Тип: Булево.   
Если параметр имеет значение Истина, то перед сохранением откроется форма создания отчета (где пользователь сможет дополнить отчет).   
Если параметр имеет значение Ложь, то отчет сохранится без свойств, которые не разрешены для автоматического сохранения.   
Если параметр не установлен, то форма откроется, если есть свойства, для записи которых требуется разрешение пользователя.

Описание:

Записывает отчет в файл. Файл имеет формат архива. Архив может содержать файл отчета, приложенные пользователм файлы и дамп, сгенерированный при возникновении ошибки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[НастройкиОбработкиОшибок](НастройкиОбработкиОшибок.md)

Доступен, начиная с версии 8.3.17.
