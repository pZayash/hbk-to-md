ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

Прокси (Proxy)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md).   
Настройки прокси-сервера, которые будут использоваться при установке соединения, если свойство [ИспользоватьПроксиОС](ПараметрыWebSocketКлиентСоединения.ИспользоватьПроксиОС_(WebSocketClientConnectionParameters.UseOSProxy).md) установлено в значение [Ложь](lang__def_BooleanFalse.md). Это свойство необходимо устанавливать в том случае, например, когда прокси-сервер требует авторизации.  
Значение по умолчанию: неинициализированный объект [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md), прокси не используется.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
