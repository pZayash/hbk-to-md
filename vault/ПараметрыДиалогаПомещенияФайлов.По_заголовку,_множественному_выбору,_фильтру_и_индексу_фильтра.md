ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

По заголовку, множественному выбору, фильтру и индексу фильтра

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ПараметрыДиалогаПомещенияФайлов(<Заголовок>, <МножественныйВыбор>, <Фильтр>, <ИндексФильтра>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Заголовок диалога помещения файлов.

<МножественныйВыбор> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет использование в диалоге выбора нескольких файлов.

<Фильтр> (необязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Фильтр файлов, доступных для выбора с помощью диалога.

<ИндексФильтра> (необязательный)

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Индекс элемента в фильтре диалога, который будет выбран по умолчанию.  
Значение по умолчанию: 0.

Использование в версии:

Доступен, начиная с версии 8.3.15.
