Расширение поля формы для поля планировщика (Form field extension for a planner)

ОбработкаФормированияКоманд (CommandGenerateProcessing)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОбработкаФормированияКоманд(<ПараметрыКоманд>, <Команды>, <КомандаПоУмолчанию>)

Параметры:

<ПараметрыКоманд>

Тип: [ПараметрыФормированияКомандПоляПланировщика](ПараметрыФормированияКомандПоляПланировщика.md).   
Параметры формирования команд планировщика.

<Команды>

Тип: [Массив](Массив.md).   
Список команд, которые можно добавить. Значения элементов массива имеют тип [ОписаниеКомандыПоляПланировщика](ОписаниеКомандыПоляПланировщика.md). Значения другого типа игнорируются.   
При вызове список заполнен командами контекстного меню по умолчанию. Команды, добавленные в контекстное меню через свойство поля [КонтекстноеМеню](ПолеФормы.КонтекстноеМеню.md), не будут добавлены в данный массив.  
Возвращаемое значение.

<КомандаПоУмолчанию>

Тип: [ОписаниеКомандыПоляПланировщика](ОписаниеКомандыПоляПланировщика.md), Неопределено.   
Определяет команду по умолчанию по нажатию левой кнопки мыши. Если команда задана, то при нажатии левой кнопки мыши она выполняется сразу.  
Если команда не задана, то при нажатии на левой кнопки мыши отображается контекстное меню.  
Возвращаемое значение.

Описание:

Событие при нажатии правой или левой кнопки мыши в поле планировщика. Возникает до события [ПередСозданием](Расширение_поля_формы_для_поля_планировщика.ПередСозданием.md), [ПередНачаломБыстрогоРедактирования](Расширение_поля_формы_для_поля_планировщика.ПередНачаломБыстрогоРедактирования.md), [НажатиеНаЭлементеИзмерения](Расширение_поля_формы_для_поля_планировщика.НажатиеНаЭлементеИзмерения.md),  
[НажатиеНаЭлементеШкалыВремени](Расширение_поля_формы_для_поля_планировщика.НажатиеНаЭлементеШкалыВремени.md),   
[НажатиеНаПеренесенномЗаголовкеШкалыВремени](Расширение_поля_формы_для_поля_планировщика.НажатиеНаПеренесенномЗаголовкеШкалыВремени.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
