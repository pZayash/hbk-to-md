# Расширение таблицы формы для значений параметров компоновки данных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Интерфейс.md">Интерфейс</a> › <a href="obsidian://open?file=Таблица_формы.md">Таблица формы</a> › [Расширения для системы компоновки данных](Расширения_для_системы_компоновки_данных.md)

Доступен, начиная с версии 8.2.

Свойства:

[ОграниченияИспользования (UseRestrictions)](Расширение_таблицы_формы_для_значений_параметров_компоновки_данных.ОграниченияИспользования.md)  

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы.md), применимые для значений параметров компоновки данных.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока.md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель.md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки.md), используется идентификатор типа [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных.md).  
Используется для работы с данными следующих типов:

- [ОформлениеКомпоновкиДанных](ОформлениеКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных.md);
- [ЗначенияПараметровВыводаТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаТаблицыКомпоновкиДанных.md);
- [ЗначенияПараметровДанныхКомпоновкиДанных](ЗначенияПараметровДанныхКомпоновкиДанных.md).

Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные.md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки.md) возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
