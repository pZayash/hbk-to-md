ФабрикаXDTO (XDTOFactory)

Создать (Create)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Создание значения из значения

Синтаксис:

Создать(<Тип>, <Значение>)

Параметры:

<Тип> (обязательный)

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO.md).   
Тип, значение которого необходимо создать.

<Значение> (обязательный)

Тип: Произвольный.   
Произвольное значение, на основе которого должно быть создано значение XDTO.

Описание варианта метода:

Создает значение XDTO по произвольному значению.

Вариант синтаксиса: Создание значения по лексическому представлению

Синтаксис:

Создать(<Тип>, <Строка>)

Параметры:

<Тип> (обязательный)

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO.md).   
Тип, значение которого необходимо создать.

<Строка> (обязательный)

Тип: Строка.   
Лексическое представление значения.

Описание варианта метода:

Создает значение XDTO по лексическому представлению значения. Лексическое представление значения должно удовлетворять ограничениям, определяемым данным типом значения XDTO.

Вариант синтаксиса: Создание объекта

Синтаксис:

Создать(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [ТипОбъектаXDTO](ТипОбъектаXDTO.md).   
Тип, объект которого необходимо создать.

Описание варианта метода:

Создаёт объект XDTO, указанного типа.  
При этом метод вернет значение типа [ОбъектXDTO](ОбъектXDTO.md).

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO.md).   

Описание:

Создает значение XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
