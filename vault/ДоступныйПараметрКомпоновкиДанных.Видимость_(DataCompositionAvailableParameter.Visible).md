ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

Видимость (Visible)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Видимость параметра при редактировании значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - параметр не отображается.

Использование в версии:

Доступен, начиная с версии 8.1.
