РазмерКартинки (PictureSize)

ПоРазмеруШрифта (ByFontSize)

Доступен, начиная с версии 8.3.19.

Описание:

Данное значение интерпретируется следующим образом:

- высота картинки считается равной высоте шрифта (об используемой высоте шрифта см. далее),
- ширина картинки рассчитывается пропорционально исходным размерам картинки,
- картинка отображается пропорционально исходным размерам.

В качестве высоты шрифта используются следующие значения:

- Для ячеек табличного документа - шрифт ячейки табличного документа.
- Для рисунков табличного документа - шрифт, используемый табличным документом в качестве шрифта по умолчанию.
- Для декораций типа [Картинка](ЭлементГрафическойСхемыДекорация.Картинка_(GraphicalSchemaItemDecoration.Picture).md) - шрифт, заданный при помощи свойства декорации [Шрифт](ЭлементГрафическойСхемыДекорация.Шрифт_(GraphicalSchemaItemDecoration.Font).md).
- Для расширения поля формы для картинки - шрифт, заданный при помощи свойства [Шрифт](Расширение_поля_формы_для_поля_картинки.Шрифт_(Form_field_extension_for_a_picture_field.Font).md).
- Для следующих элементов графической схемы используется шрифт, заданный при помощи свойства Шрифт соответствующего элемента графической схемы:
  - [ЭлементГрафическойСхемыДекорация](ЭлементГрафическойСхемыДекорация_(GraphicalSchemaItemDecoration).md),
  - [ЭлементГрафическойСхемыДействие](ЭлементГрафическойСхемыДействие_(GraphicalSchemaItemActivity).md),
  - [ЭлементГрафическойСхемыРазделение](ЭлементГрафическойСхемыРазделение_(GraphicalSchemaItemSplit).md),
  - [ЭлементГрафическойСхемыУсловие](ЭлементГрафическойСхемыУсловие_(GraphicalSchemaItemCondition).md),
  - [ЭлементГрафическойСхемыЗавершение](ЭлементГрафическойСхемыЗавершение_(GraphicalSchemaItemCompletion).md),
  - [ЭлементГрафическойСхемыСтарт](ЭлементГрафическойСхемыСтарт_(GraphicalSchemaItemStart).md),
  - [ЭлементГрафическойСхемыСлияние](ЭлементГрафическойСхемыСлияние_(GraphicalSchemaItemJoin).md),
  - [ЭлементГрафическойСхемыВложенныйБизнесПроцесс](ЭлементГрафическойСхемыВложенныйБизнесПроцесс_(GraphicalSchemaItemSubBusinessProcess).md),

- В остальных случаях - шрифт стиля "Обычный шрифт текста" (по умолчанию - Arial, 10pt).

Доступен, начиная с версии 8.3.19.
