ПерерасчетыМенеджер.<Имя регистра расчета> (RecalcsManager.<Calculation register name>)

<Имя перерасчета> (<Recalculation name>)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ПерерасчетМенеджер.\<Имя перерасчета\>](ПерерасчетМенеджер.Имя_перерасчета_(RecalculationManager.Recalculation_name).md).   
Набор свойств содержит менеджеры отдельных перерасчетов. Доступ к менеджеру осуществляется по имени, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
