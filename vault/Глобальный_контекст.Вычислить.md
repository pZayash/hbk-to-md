Глобальный контекст (Global context)

Вычислить (Eval)

Доступен, начиная с версии 8.0.

Синтаксис:

Вычислить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.   
Выражение, которое требуется вычислить.

Возвращаемое значение:

Результат вычисления выражения. Тип возвращаемого значения зависит от типа выражения.

Описание:

Вычисляет переданное выражение. С помощью метода можно вычислять математические и логические выражения, а также операции конкатенации. Подробнее про выражения см. раздел "Выражения языка".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

В режимах запуска мобильного приложения метод не поддерживается в операционной системе iOS.

Пример:

|  |
| --- |
| А = Вычислить("500 + 500\*20/100"); А = Вычислить("Истина и Ложь"); А = Вычислить("""Иван"" + "" "" + ""Иванович"""); |

Использование в версии:

Доступен, начиная с версии 8.0.
