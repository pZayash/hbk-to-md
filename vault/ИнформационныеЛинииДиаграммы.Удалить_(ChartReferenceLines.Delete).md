ИнформационныеЛинииДиаграммы (ChartReferenceLines)

Удалить (Delete)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Удалить(<Линия>)

Параметры:

<Линия> (обязательный)

Тип: [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы_(ChartReferenceLine).md).   
Информационная линия, которую требуется удалить.

Описание:

Удаляет указанную информационную линию из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
