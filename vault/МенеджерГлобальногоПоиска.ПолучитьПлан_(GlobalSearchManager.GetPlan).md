# МенеджерГлобальногоПоиска.ПолучитьПлан

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Глобальный поиск](Глобальный_поиск.md) › [МенеджерГлобальногоПоиска](МенеджерГлобальногоПоиска_(GlobalSearchManager).md) › [Методы](МенеджерГлобальногоПоиска__Методы.md)

МенеджерГлобальногоПоиска (GlobalSearchManager)

ПолучитьПлан (GetPlan)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьПлан()

Возвращаемое значение:

Тип: [ПланГлобальногоПоиска](ПланГлобальногоПоиска_(GlobalSearchPlan).md).   

Описание:

Возвращает текущий план выполнения глобального поиска.  
В качестве начального значения используется план, заполненный всеми видами стандартного поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
