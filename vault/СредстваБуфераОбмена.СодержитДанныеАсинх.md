СредстваБуфераОбмена (ClipboardTools)

СодержитДанныеАсинх (ContainsDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СодержитДанныеАсинх(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена.md), Строка.   
Проверяемый формат данных.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - тип [Булево](ТипЗначенияJSON.Булево.md).

Описание:

Асинхронная функция. Возвращает признак того, что буфер обмена содержит данные указанного формата. Данный метод рекомендуется использовать перед вызовом метода [ПолучитьДанныеАсинх](СредстваБуфераОбмена.ПолучитьДанныеАсинх.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте может быть отображен запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается [Обещание](Обещание.md), результат которого – Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.24.
