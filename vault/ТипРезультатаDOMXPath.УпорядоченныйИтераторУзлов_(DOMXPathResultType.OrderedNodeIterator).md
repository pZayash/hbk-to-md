ТипРезультатаDOMXPath (DOMXPathResultType)

УпорядоченныйИтераторУзлов (OrderedNodeIterator)

Доступен, начиная с версии 8.1.

Описание:

Результат - упорядоченный набор узлов (согласно порядку в документе). Изменение документа делает набор недействительным.

Использование в версии:

Доступен, начиная с версии 8.1.
