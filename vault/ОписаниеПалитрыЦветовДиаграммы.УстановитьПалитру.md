ОписаниеПалитрыЦветовДиаграммы (ChartColorPaletteDescription)

УстановитьПалитру (SetPalette)

Доступен, начиная с версии 8.3.12.

Синтаксис:

УстановитьПалитру(<Палитра>)

Параметры:

<Палитра> (обязательный)

Тип: [Массив](Массив.md).   
Массив цветов палитры. Используются только элементы типа [Цвет](Цвет.md). Элементы другого типа игнорируются.

Описание:

Устанавливает свойство [ПалитраЦветов](ОписаниеПалитрыЦветовДиаграммы.ПалитраЦветов.md) в значение [Произвольная](ПалитраЦветовДиаграммы.Произвольная.md). Использует цвета из переданного массива в качестве палитры. Цвета записываются в палитру в том же порядке, что и в переданном массиве.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
