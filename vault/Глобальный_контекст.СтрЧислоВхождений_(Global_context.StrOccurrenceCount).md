# Глобальный контекст.СтрЧислоВхождений

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Функции работы со значениями типа Строка](Функции_работы_со_значениями_типа_Строка.md)

Глобальный контекст (Global context)

СтрЧислоВхождений (StrOccurrenceCount)

Доступен, начиная с версии 8.0.

Синтаксис:

СтрЧислоВхождений(<Строка>, <ПодстрокаПоиска>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Исходная строка.

<ПодстрокаПоиска> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомая подстрока.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Число вхождений.

Описание:

Вычисляет число вхождений подстроки поиска в исходной строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Результат = СтрЧислоВхождений(ИсходныйТекст, "Число"); Сообщить("Слово ""Число"" встречается " + Результат + " раз."); |

Использование в версии:

Доступен, начиная с версии 8.0.
