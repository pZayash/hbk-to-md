СоставКопииБазыДанных (DatabaseCopyContent)

Удалить (Delete)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [Число](lang__def_Number.md), [ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных_(DatabaseCopyContentItem).md).   
Позиция элемента состава копии базы данных или сам элемент.

Описание:

Удаляет элемент состава из база данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
