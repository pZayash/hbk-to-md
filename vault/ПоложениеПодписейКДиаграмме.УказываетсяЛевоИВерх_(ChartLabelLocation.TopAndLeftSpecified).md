ПоложениеПодписейКДиаграмме (ChartLabelLocation)

УказываетсяЛевоИВерх (TopAndLeftSpecified)

Доступен, начиная с версии 8.3.10.

Описание:

Расположение подписи устанавливается с помощью координат.  
Если объект не имеет свойств, отвечающих за координаты подписи, то поведение аналогично [Центр](ПоложениеПодписейКДиаграмме.Центр_(ChartLabelLocation.Center).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
