Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОписаниеХарактеристик](ОписаниеХарактеристик.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания характеристик.  
Возможно обращение к описанию характеристики посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ОписанияХарактеристик.Индекс.md)  
[Количество (Count)](ОписанияХарактеристик.Количество.md)  
[Получить (Get)](ОписанияХарактеристик.Получить.md)  

Описание:

Содержит коллекцию объектов [ОписаниеХарактеристик](ОписаниеХарактеристик.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
