# ПланОбменаСсылка.<Имя плана обмена>.УникальныйИдентификатор

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаСсылка.<Имя плана обмена>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md) › [Методы](ПланОбменаСсылка.Имя_плана_обмена__Методы.md)

ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ПланОбменаМенеджер.Имя_плана_обмена.ПолучитьСсылку_(ExchangePlanManager.Exchange_plan_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
