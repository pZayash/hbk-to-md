ОбъявлениеАтрибутаXS (XSAttributeDeclaration)

Ссылка (Reference)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [РасширенноеИмяXML](РасширенноеИмяXML.md).   
Имя объявления, на которое ссылается данное объявление свойства (атрибута/элемента).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет значение атрибута ref объявления атрибута (<http://www.w3.org/TR/xmlschema-1/#element-attribute>) или элемента (<http://www.w3.org/TR/xmlschema-1/#element-element>).

Использование в версии:

Доступен, начиная с версии 8.1.
