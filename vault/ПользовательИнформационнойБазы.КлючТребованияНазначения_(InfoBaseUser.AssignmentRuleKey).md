ПользовательИнформационнойБазы (InfoBaseUser)

КлючТребованияНазначения (AssignmentRuleKey)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Ключ требования назначения функциональности, заданный для пользователя информационной базы.  
Значение по умолчанию: Пустая строка.  
  
Изменение свойства возможно только при наличии у текущего пользователя права "Администрирование".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.27.
