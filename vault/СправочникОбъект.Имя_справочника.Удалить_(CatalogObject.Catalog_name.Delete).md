# СправочникОбъект.<Имя справочника>.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Методы](СправочникОбъект.Имя_справочника__Методы.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет элемент справочника из базы данных.  
Важно! В отличие от метода [УстановитьПометкуУдаления](СправочникОбъект.Имя_справочника.УстановитьПометкуУдаления_(CatalogObject.Catalog_name.SetDeletionMark).md), данный метод производит непосредственное удаление элемента справочника без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для иерархического справочника также удаляются все подчиненные элементы.

Пример:

|  |
| --- |
| Объект.Удалить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
