ОбъектМетаданных: Макет (MetadataObject: Template)

ТипМакета (TemplateType)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ТипМакета](ТипМакета_(TemplateType).md).   
Определяет тип макета (например, [ТабличныйДокумент](ТипМакета.ТабличныйДокумент_(TemplateType.SpreadsheetDocument).md), [ТекстовыйДокумент](ТипМакета.ТекстовыйДокумент_(TemplateType.TextDocument).md), [СхемаКомпоновкиДанных](ТипМакета.СхемаКомпоновкиДанных_(TemplateType.DataCompositionSchema).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
