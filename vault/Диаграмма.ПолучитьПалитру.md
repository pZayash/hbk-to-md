Диаграмма (Chart)

ПолучитьПалитру (GetPalette)

Не рекомендуется использовать, начиная с версии 8.3.12.

Рекомендуется использовать:

- [ПолучитьПалитру](ОписаниеПалитрыЦветовДиаграммы.ПолучитьПалитру.md)

Доступен, начиная с версии 8.3.10 (в режиме совместимости с версией 8.3.12 и последующими).

Синтаксис:

ПолучитьПалитру()

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Элементами массива являются значения типа [Цвет](Цвет.md).

Описание:

Возвращает массив цветов текущей палитры.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10 (в режиме совместимости с версией 8.3.12 и последующими).

Не рекомендуется использовать, начиная с версии 8.3.12.

Описание изменено в версии 8.3.15.
