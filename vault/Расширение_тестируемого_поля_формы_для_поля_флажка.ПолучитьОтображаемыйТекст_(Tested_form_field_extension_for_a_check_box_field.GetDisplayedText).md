Расширение тестируемого поля формы для поля флажка (Tested form field extension for a check box field)

ПолучитьОтображаемыйТекст (GetDisplayedText)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьОтображаемыйТекст()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает отображаемое представление значения флажка (в форме тумблера) в поле ввода.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
