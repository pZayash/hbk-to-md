Запрос (Query)

ТребуемоеВремяАктуальностиДанных (RequiredDataRelevanceTime)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Дата](lang__def_Date.md), [СтандартнаяДатаНачала](СтандартнаяДатаНачала_(StandardBeginningDate).md).   
Минимальное допустимое время актуальности данных, которые необходимо получить. Свойство имеет смысл только если [ТребуемаяАктуальностьДанных](Запрос.ТребуемаяАктуальностьДанных_(Query.RequiredDataRelevance).md) равна [Авто](ТребуемаяАктуальностьДанных.Авто_(RequiredDataRelevance.Auto).md) или [Любые](ТребуемаяАктуальностьДанных.Любые_(RequiredDataRelevance.Any).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.16.
