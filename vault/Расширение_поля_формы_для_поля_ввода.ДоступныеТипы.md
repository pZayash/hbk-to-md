Расширение поля формы для поля ввода (Form field extension for a text box)

ДоступныеТипы (AvailableTypes)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов.md).   
Позволяет задать набор типов, доступных для выбора при редактировании значения типа [ОписаниеТипов](ОписаниеТипов.md). Так же задается ограничение для квалификаторов примитивных типов Число, Строка, Дата.  
Если свойство не установлено, пользователю будет предоставлен для выбора список всех возможных типов данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
