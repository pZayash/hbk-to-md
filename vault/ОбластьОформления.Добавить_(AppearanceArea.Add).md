ОбластьОформления (AppearanceArea)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Представление>, <ТипПоля>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным добавляемого поля.

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md).   
Представление добавляемого поля.  
Значение по умолчанию: Пустая строка.

<ТипПоля> (необязательный)

Тип: [ТипОбластиОформления](ТипОбластиОформления_(AppearanceAreaType).md).   
Тип добавляемого поля.  
Значение по умолчанию: [Поле](ТипОбластиОформления.Поле_(AppearanceAreaType.Field).md).

Возвращаемое значение:

Тип: [ЭлементОбластиОформления](ЭлементОбластиОформления_(AppearanceAreaItem).md).   

Описание:

Добавляет поле в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
