ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ДополнительнаяФормаСписка (AuxiliaryListForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Описывает форму, которая будет вызвана системой для просмотра списка плана видов расчета, в случае если основная форма не установлена или не может быть использована в конкретном варианте использования 1С:Предприятия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
