ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновнойРежимЗапуска (DefaultRunMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РежимЗапускаКлиентскогоПриложения](РежимЗапускаКлиентскогоПриложения_(ClientRunMode).md).   
Устанавливает основной режим исполнения приложения.  
Определяет режим, в котором исполняется 1С:Предприятие при запуске толстого клиента пользователем, для которого режим запуска установлен Авто.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
