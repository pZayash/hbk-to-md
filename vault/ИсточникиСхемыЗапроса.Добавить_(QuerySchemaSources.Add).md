ИсточникиСхемыЗапроса (QuerySchemaSources)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: Добавить таблицу схемы запроса

Синтаксис:

Добавить(<ДоступнаяТаблица>, <Псевдоним>)

Параметры:

<ДоступнаяТаблица> (обязательный)

Тип: [Строка](lang__def_String.md), [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса_(QuerySchemaAvailableTable).md), [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса_(QuerySchemaAvailableNestedTable).md).   
Имя таблицы, доступная таблица или доступная вложенная таблица, полученные из доступных таблиц схемы запроса.  
Если передано имя таблицы, и по нему не удалось найти таблицу, то будет сгенерировано исключение.

<Псевдоним> (необязательный)

Тип: [Строка](lang__def_String.md).   
Псевдоним добавляемого источника. Если не указан, то псевдоним добавляемого источника сгенерируется автоматически. Если параметр задан, сначала будет выполнена проверка корректности псевдонима. Если указан недопустимый псевдоним, генерируется исключение, содержащее описание ошибки.

Описание варианта метода:

Добавляет таблицу схемы запроса.

Вариант синтаксиса: Добавить вложенный запрос

Синтаксис:

Добавить(<Тип>, <Псевдоним>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Параметр должен содержать значение типа [ВложенныйЗапросСхемыЗапроса](ВложенныйЗапросСхемыЗапроса_(QuerySchemaNestedQuery).md).

<Псевдоним> (необязательный)

Тип: [Строка](lang__def_String.md).   
Параметр содержит псевдоним, который будет присвоен источнику. Перед присвоением будет выполнена проверка корректности псевдонима. Если указан недопустимый псевдоним, будет сгенерировано исключение, текст которого будет содержать описание ошибки.

Описание варианта метода:

Добавляет вложенный запрос или описание временной таблицы.

Вариант синтаксиса: Добавить описание временной таблицы

Синтаксис:

Добавить(<Тип>, <Имя>, <Псевдоним>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Параметр может содержать значение типа [ОписаниеВременнойТаблицыСхемыЗапроса](ОписаниеВременнойТаблицыСхемыЗапроса_(QuerySchemaTempTableDescription).md).

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Параметр содержит имя временной таблицы. Псевдоним будет сгенерирован автоматически. Имя может начинаться с символа "&",

<Псевдоним> (необязательный)

Тип: [Строка](lang__def_String.md).   
Псевдоним добавляемой временной таблицы. Если параметр не указан, псевдоним генерируется автоматически.

Описание варианта метода:

Добавляет вложенный запрос или описание временной таблицы.

Возвращаемое значение:

Тип: [ИсточникСхемыЗапроса](ИсточникСхемыЗапроса_(QuerySchemaSource).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
