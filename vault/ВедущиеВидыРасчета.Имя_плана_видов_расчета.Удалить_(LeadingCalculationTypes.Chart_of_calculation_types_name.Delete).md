# ВедущиеВидыРасчета.<Имя плана видов расчета>.Удалить

ВедущиеВидыРасчета.<Имя плана видов расчета> (LeadingCalculationTypes.<Chart of calculation types name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [ВедущиеВидыРасчетаСтрока.](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(LeadingCalculationTypesRow.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(LeadingCalculationTypesRow.Chart_of_calculation_types_name).md)>.   
Индекс удаляемой строки в табличной части, либо сама удаляемая строка.

Описание:

Удаляет строку с указанным индексом из табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данный метод следует использовать осмотрительно в том случае, если в табличной части есть предопределенные строки (т.е. имеются строки, для которых свойство [Предопределенный](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета.Предопределенный_(LeadingCalculationTypesRow.Chart_of_calculation_types_name.Predefined).md) имеет значение [Истина](lang__def_BooleanTrue.md)). Если удалить предопределенную строку табличной части, то не будет возможности записать такие изменения.  
Использование метода допустимо только в том случае, если табличная часть получена из свойства объекта. Если табличная часть получена из свойства ссылки (или выборки), то использование этого метода будет вызывать ошибку выполнения.

Использование в версии:

Доступен, начиная с версии 8.0.
