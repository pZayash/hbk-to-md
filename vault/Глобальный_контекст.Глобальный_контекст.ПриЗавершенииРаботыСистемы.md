Глобальный контекст (Global context)

ПриЗавершенииРаботыСистемы (OnExit)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗавершенииРаботыСистемы()

Описание:

Возникает перед завершением работы 1С:Предприятия в режиме внешнего соединения. В данной процедуре могут быть выполнены действия, необходимые при выходе из программы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), событие [ПередЗавершениемРаботыСистемы](Глобальный_контекст.Глобальный_контекст.ПередЗавершениемРаботыСистемы.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
