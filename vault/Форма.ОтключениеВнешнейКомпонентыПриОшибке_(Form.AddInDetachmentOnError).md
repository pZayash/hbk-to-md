Форма (Form)

ОтключениеВнешнейКомпонентыПриОшибке (AddInDetachmentOnError)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ОтключениеВнешнейКомпонентыПриОшибке(<Местоположение>, <Имя>)

Параметры:

<Местоположение>

Тип: [Строка](lang__def_String.md).   
Строка, определяющая местоположение внешней компоненты.  
Соответствует первому параметру метода [ПодключитьВнешнююКомпоненту](Глобальный_контекст.ПодключитьВнешнююКомпоненту_(Global_context.AttachAddIn).md).

<Имя>

Тип: [Строка](lang__def_String.md).   
Символическое имя отключаемой внешней компоненты.  
Соответствует второму параметру метода [ПодключитьВнешнююКомпоненту](Глобальный_контекст.ПодключитьВнешнююКомпоненту_(Global_context.AttachAddIn).md).

Описание:

Вызывается при падении процесса внешней компоненты.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
