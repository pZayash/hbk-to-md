ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [РазмерКартинки](РазмерКартинки.md).   
Свойство используется, чтобы подогнать размеры картинки под размеры ячейки. Если установлено значение [Черепица](РазмерКартинки.Черепица.md) или [Растянуть](РазмерКартинки.Растянуть.md), то значение свойства [ПоложениеТекстаОтносительноКартинки](ОбластьЯчеекТабличногоДокумента.ПоложениеТекстаОтносительноКартинки.md) игнорируется и считается равным [Поверх](ПоложениеТекстаОтносительноКартинки.Поверх.md). Значение по умолчанию – [Пропорционально](РазмерКартинки.Пропорционально.md).  
Значение [АвтоРазмерБезУчетаМасштаба](РазмерКартинки.АвтоРазмерБезУчетаМасштаба.md) интерпретируется как [АвтоРазмер](РазмерКартинки.АвтоРазмер.md).  
Значение [РеальныйРазмерБезУчетаМасштаба](РазмерКартинки.РеальныйРазмерБезУчетаМасштаба.md) интерпретируется как [РеальныйРазмер](РазмерКартинки.РеальныйРазмер.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.16.
