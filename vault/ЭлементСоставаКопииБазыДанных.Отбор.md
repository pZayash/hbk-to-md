ЭлементСоставаКопииБазыДанных (DatabaseCopyContentItem)

Отбор (Filter)

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор.md).   
Поддерживается только для элементов состава копии, у которых свойство [Метаданные](ЭлементСоставаКопииБазыДанных.Метаданные.md) содержит периодический регистр свдеений или регистр накопления.  
Отбор содержит один элемент "Период". Для элемента отбора доступны следующие виды сравнения:

- [Больше](ВидСравнения.Больше.md),
- [БольшеИлиРавно](ВидСравнения.БольшеИлиРавно.md),
- [Интервал](ВидСравнения.Интервал.md),
- [ИнтервалВключаяГраницы](ВидСравнения.ИнтервалВключаяГраницы.md),
- [ИнтервалВключаяНачало](ВидСравнения.ИнтервалВключаяНачало.md),
- [ИнтервалВключаяОкончание](ВидСравнения.ИнтервалВключаяОкончание.md),
- [Меньше](ВидСравнения.Меньше.md),
- [МеньшеИлиРавно](ВидСравнения.МеньшеИлиРавно.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).
