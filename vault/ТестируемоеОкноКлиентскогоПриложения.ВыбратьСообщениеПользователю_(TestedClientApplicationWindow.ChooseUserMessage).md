ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

ВыбратьСообщениеПользователю (ChooseUserMessage)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ВыбратьСообщениеПользователю(<ТекстСообщения>)

Параметры:

<ТекстСообщения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст сообщения, который необходимо "выбрать" в тестируемом приложении.  
Допустимо использование знаков подстановки "\* "и "?".  
При выполнении перебирается список сообщений, начиная с начала, и выбирается первое "похожее" сообщение.

Описание:

Имитирует щелчок мышью на сообщении в списке сообщений.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
