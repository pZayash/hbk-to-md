ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Добавить (Add)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md).   
У нового созданного объекта [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md) свойства [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях.md) и [КолонкаКоэффициентовВУзлах](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВУзлах.md) имеют значение Неопределено.

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
