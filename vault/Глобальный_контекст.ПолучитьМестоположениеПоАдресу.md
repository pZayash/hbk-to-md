Глобальный контекст (Global context)

ПолучитьМестоположениеПоАдресу (GetLocationByAddress)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМестоположениеПоАдресу(<Адрес>)

Параметры:

<Адрес> (необязательный)

Тип: [ДанныеАдреса](ДанныеАдреса.md).   
Адрес местоположения. Состав адреса может различаться в разных государствах.  
Для России определены поля:

- Страна (Country);
- Регион (State);
- Город (City);
- Улица (Street);
- Дом (House);
- Индекс (ZipCode).

Возвращаемое значение:

Тип: [ГеографическиеКоординаты](ГеографическиеКоординаты.md), Неопределено.   
Неопределено - данные не получены.

Описание:

Получает географические координаты по адресу.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
