Доступен, начиная с версии 8.3.1.

Описание:

Тип: Число.   
Содержит номер порта главного менеджера кластера, запущенного на данном рабочем сервере. Используется, если [MainPort](Рабочий_сервер.MainPort.md) = Истина.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
