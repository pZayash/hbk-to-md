ЧтениеPDF (PDFReader)

НачатьПроверкуПодписей (BeginVerifySignatures)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПроверкуПодписей(<ОписаниеОповещения>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со   
следующими параметрами:

- <Сертификаты> - тип [Массив](Массив.md). В параметре возвращаются сертификаты, с помощью которого была произведена подпись (если сертификат включен в данные подписи).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Сертификаты> - тип [Массив](Массив.md). В параметре возвращаются сертификаты, с помощью которого была произведена подпись (если сертификат включен в данные подписи).  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).





<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии.md).   
Менеджер криптографии.

Описание:

Проверяет действительность присоединенных к PDF-документу подписей. В случае ошибки генерируется исключение и вызывается процедура обработки ошибок, указанная в параметре <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В веб-клиенте требуется установка расширения для работы с криптографией.

Использование в версии:

Доступен, начиная с версии 8.3.16.
