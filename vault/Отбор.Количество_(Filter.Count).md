Отбор (Filter)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество элементов отбора, входящих в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Для Индекс = 0 По ПостроительОтчета.Отбор.Количество()-1 Цикл     ПолеОтбора = ПостроительОтчета.Отбор[Индекс];     // обработка отбора      // ...   КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
