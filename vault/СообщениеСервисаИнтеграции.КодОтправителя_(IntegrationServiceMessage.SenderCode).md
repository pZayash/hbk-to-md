СообщениеСервисаИнтеграции (IntegrationServiceMessage)

КодОтправителя (SenderCode)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Код участника интеграции, который отправил сообщение.   
Предполагается использовать при организации взаимодействия запрос-ответ.   
В сообщении-запросе в свойстве указывается код участника, которому направляется ответ.  
Свойство предоставляет удобный способ работы с параметром сообщения Participant-Sender-Code. Значение, установленное в свойстве, доступно также через параметр.   
  
Можно устанавливать только при отправке, при попытке изменить в полученном сообщении генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
