ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПринадлежностьОбъекта (ObjectBelonging)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Определяет, относится ли объект к данной конфигурации или был заимствован из расширяемой конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
