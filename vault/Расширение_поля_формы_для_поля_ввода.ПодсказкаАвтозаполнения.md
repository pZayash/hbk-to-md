Расширение поля формы для поля ввода (Form field extension for a text box)

ПодсказкаАвтозаполнения (AutoFillHint)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ПодсказкаАвтозаполненияПоляВвода](ПодсказкаАвтозаполненияПоляВвода.md).   
Подсказка автозаполнения поля ввода.  
Поддерживается в мобильном и веб-клиенте.  
В тонком клиенте свойство игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
