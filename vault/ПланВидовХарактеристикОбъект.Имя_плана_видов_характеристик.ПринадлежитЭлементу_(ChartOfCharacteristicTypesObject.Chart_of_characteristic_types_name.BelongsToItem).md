# ПланВидовХарактеристикОбъект.<Имя плана видов характеристик>.ПринадлежитЭлементу

ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ПринадлежитЭлементу (BelongsToItem)

Доступен, начиная с версии 8.0.

Синтаксис:

ПринадлежитЭлементу(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).   
Элемент - группа, для которой определяется принадлежность элемента.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - элемент подчинен группе; [Ложь](lang__def_BooleanFalse.md) - в противном случае.  
Если в качестве параметра указать сам элемент, то [Ложь](lang__def_BooleanFalse.md).

Описание:

Определяет подчиненность элемента вида характеристики группе с учетом всех уровней иерархии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определяет подчиненность группе непосредственно на момент вызова метода путем последовательного считывания всех вышестоящих элементов.

Пример:

|  |
| --- |
| Если ОбъектСсылка.ПринадлежитЭлементу(ЭлементСсылка) Тогда     Сообщить("Принадлежит"); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
