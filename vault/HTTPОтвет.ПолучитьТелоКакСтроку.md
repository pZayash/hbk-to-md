HTTPОтвет (HTTPResponse)

ПолучитьТелоКакСтроку (GetBodyAsString)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьТелоКакСтроку(<Кодировка>)

Параметры:

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Указывает кодировку, в которой должно интерпретироваться тело ответа.   
Если не задан, то кодировка определяется из HTTP-заголовка Content-Type.  
Если сервер не указал кодировку в заголовке, то используется [UTF8](КодировкаТекста.UTF8.md).  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.  
В веб-клиенте в браузерах Internet Explorer и Microsoft Edge (до версии 18) параметр игнорируется. Используется кодировка utf-8.

Возвращаемое значение:

Тип: Строка, Неопределено.   
Неопределено - если при вызове методов [ОтправитьДляОбработки](HTTPСоединение.ОтправитьДляОбработки.md) или [Получить](HTTPСоединение.Получить.md) был установлен параметр <ИмяВыходногоФайла>.

Описание:

Получает тело объекта как строку.   
  
В веб-клиенте в браузерах Internet Explorer и Microsoft Edge (до версии 18) метод не поддерживается - возвращает значение Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.22.
