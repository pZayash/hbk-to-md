Расширение табличного поля списка видов характеристик (Characteristic types list table box extension)

ПередИзменениемРодителя (BeforeParentChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИзменениемРодителя(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от изменения родителя. Если в теле процедуры-обработчика установить данному параметру значение Истина, изменение выполнено не будет.  
Значение по умолчанию: Ложь.

Описание:

Вызывается перед выполнением изменения родителя текущего элемента с помощью команды списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
