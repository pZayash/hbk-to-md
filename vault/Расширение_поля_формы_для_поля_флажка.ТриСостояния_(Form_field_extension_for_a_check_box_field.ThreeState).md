# Расширение поля формы для поля флажка.ТриСостояния

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля флажка](Расширение_поля_формы_для_поля_флажка_(Form_field_extension_for_a_check_box_field).md) › [Свойства](Расширение_поля_формы_для_поля_флажка__Свойства.md)

Расширение поля формы для поля флажка (Form field extension for a check box field)

ТриСостояния (ThreeState)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что флажок может иметь три возможных состояния. При этом типом редактируемого значения является [Число](lang__def_Number.md).  
[Истина](lang__def_BooleanTrue.md) - состояние разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
