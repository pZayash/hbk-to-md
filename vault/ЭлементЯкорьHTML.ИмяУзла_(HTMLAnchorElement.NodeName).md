ЭлементЯкорьHTML (HTMLAnchorElement)

ИмяУзла (NodeName)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя узла HTML. Имя узла присутствует у каждого типа узла HTML.  
Узел Имя узла

- [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - [Имя](АтрибутDOM.Имя_(DOMAttribute.Name).md);
- [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - содержит текст комментария;
- [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - документ;
- [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) - [ИмяЭлемента](ЭлементHTML.ИмяЭлемента_(HTMLElement.TagName).md);
- [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) - текст.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В модели HTML узла имена узлов не чувствительны к регистру.  
Имя узла, в отличие от локального имени узла, может содержать префикс, который отсутствует у локального имени.

Использование в версии:

Доступен, начиная с версии 8.2.
