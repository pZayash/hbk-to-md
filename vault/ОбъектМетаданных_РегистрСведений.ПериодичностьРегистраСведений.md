ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ПериодичностьРегистраСведений (InformationRegisterPeriodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПериодичностьРегистраСведений](ПериодичностьРегистраСведений.md).   
Указывает, с какой периодичностью регистр сохраняет значения ресурсов (например, [Месяц](ПериодичностьРегистраСведений.Месяц.md), [День](ПериодичностьРегистраСведений.День.md), [Секунда](ПериодичностьРегистраСведений.Секунда.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
