МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьНавигационнуюСсылкуВнешнегоДоступа (GetExternalAccessURL)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьНавигационнуюСсылкуВнешнегоДоступа(<Пользователь>, <Обсуждение>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Внешний пользователь, для которого формируется ссылка внешнего доступа.

<Обсуждение> (необязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Если параметр задан, то возвращается ссылка, при переходе по которой, будет открыто указанное обсуждение.

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает навигационную ссылку-приглашение для внешнего доступа.  
Если переданный пользователь не является внешним (свойство [Внешний](ПользовательСистемыВзаимодействия.Внешний.md) имеет значение Ложь), то возвращается пустая строка.  
Требуется наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ОтправитьНавигационнуюСсылкуВнешнегоДоступа](МенеджерСистемыВзаимодействия.ОтправитьНавигационнуюСсылкуВнешнегоДоступа.md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
