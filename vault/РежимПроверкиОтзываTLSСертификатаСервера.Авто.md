РежимПроверкиОтзываTLSСертификатаСервера (ServerTLSCertificateRevocationCheckMode)

Авто (Auto)

Доступен, начиная с версии 8.3.24.

Описание:

Режим проверки отзыва определяется в файле conf.cfg, в параметре IgnoreServerCertificatesChainRevocationSoftFail.  
По умолчанию используется значение true, что соответствует мягкому режиму.

Использование в версии:

Доступен, начиная с версии 8.3.24.
