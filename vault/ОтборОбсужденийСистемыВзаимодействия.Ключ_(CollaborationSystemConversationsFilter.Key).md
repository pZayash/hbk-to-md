ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

Ключ (Key)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит ключ обсуждения.  
Если не пустая строка - возвращается обсуждение с заданным ключом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию - пустая строка.

Использование в версии:

Доступен, начиная с версии 8.3.11.
