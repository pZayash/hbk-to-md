СредстваТелефонии (TelephonyTools)

ПоддерживаетсяЖурналSMS (SMSLogSupported)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПоддерживаетсяЖурналSMS()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

- [Истина](lang__def_BooleanTrue.md) - для Android при наличии телефонного модуля,
- [Ложь](lang__def_BooleanFalse.md) - для остальных операционных систем.

Описание:

Определяет доступность журнала SMS-сообщений в используемой операционной системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПолучитьЖурналSMS](СредстваТелефонии.ПолучитьЖурналSMS_(TelephonyTools.GetSMSLog).md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.
