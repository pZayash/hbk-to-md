Файл (File)

ПолучитьПредставлениеФайлаБиблиотекиМобильногоУстройства (GetMobileDeviceLibraryFilePresentation)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьПредставлениеФайлаБиблиотекиМобильногоУстройства()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает представление файла в библиотеке мультимедиа мобильного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
