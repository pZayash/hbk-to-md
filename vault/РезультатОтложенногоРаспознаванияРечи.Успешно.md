РезультатОтложенногоРаспознаванияРечи (DelayedSpeechToTextResult)

Успешно (Success)

Доступен, начиная с версии 8.3.23.

Только чтение.

Описание:

Тип: Булево.   
Указывает, что отложенное распознавание завершено успешно.  
Если отложенное распознавание не завершено, то имеет значение Ложь.

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
