ДоступныеПараметры (AvailableParameters)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДоступныеПараметрыКомпоновкиДанных](ДоступныеПараметрыКомпоновкиДанных_(DataCompositionAvailableParameters).md).

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
