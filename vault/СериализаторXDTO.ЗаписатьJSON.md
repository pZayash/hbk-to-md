СериализаторXDTO (XDTOSerializer)

ЗаписатьJSON (WriteJSON)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ЗаписатьJSON(<ЗаписьJSON>, <Значение>, <НазначениеТипаXML>)

Параметры:

<ЗаписьJSON> (обязательный)

Тип: [ЗаписьJSON](ЗаписьJSON.md).   
Объект, через который осуществляется запись JSON.

<Значение> (обязательный)

Тип: Произвольный.   
Записываемое в поток JSON значение.

<НазначениеТипаXML> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML.md).   
Вариант назначения типа элемента данных XDTO.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное.md).

Описание:

Записывает значения в формате JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В JSON могут быть записаны значения следующих типов: Неопределено, Null, Булево, Число, Строка, Дата, Тип, [УникальныйИдентификатор](УникальныйИдентификатор.md), [ДвоичныеДанные](ДвоичныеДанные.md), [ХранилищеЗначения](ХранилищеЗначения.md), [ОписаниеТипов](ОписаниеТипов.md), а также ссылки объектов данных и сами объекты данных, наборы записей регистров, менеджер значения константы. В дальнейшем список может пополняться.  
При попытке записи значения неподходящего типа будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
