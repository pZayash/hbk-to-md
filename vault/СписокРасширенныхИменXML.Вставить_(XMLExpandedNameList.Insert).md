# СписокРасширенныхИменXML.Вставить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md) › [СписокРасширенныхИменXML](СписокРасширенныхИменXML_(XMLExpandedNameList).md) › [Методы](СписокРасширенныхИменXML__Методы.md)

СписокРасширенныхИменXML (XMLExpandedNameList)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Имя>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс - местоположение вставляемого расширенного имени.

<Имя> (обязательный)

Тип: [РасширенноеИмяXML](РасширенноеИмяXML_(XMLExpandedName).md).   
Расширенное имя XML.

Описание:

Вставляет расширенное имя в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
