ОтображениеГруппыКнопок (ButtonGroupRepresentation)

Авто (Auto)

Доступен, начиная с версии 8.3.8.

Описание:

Вид группы кнопок командной панели определяется автоматически:

- если у группы кнопок есть родительская группа, используется значение свойства [Отображение](Расширение_группы_формы_для_группы_кнопок.Отображение_(Form_group_extension_for_a_button_group.Representation).md) родительской группы,
- Если родительской группы нет, используется значение [Обычное](ОтображениеГруппыКнопок.Обычное_(ButtonGroupRepresentation.Usual).md).

Использование в версии:

Доступен, начиная с версии 8.3.8.
