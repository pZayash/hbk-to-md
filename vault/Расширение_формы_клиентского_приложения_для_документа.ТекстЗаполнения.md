Расширение формы клиентского приложения для документа (Client application form extension for documents)

ТекстЗаполнения (FillingText)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: Строка.   
Текст, введенный в поле ввода, при открытии формы в режиме выбора.  
Если реквизит Номер находится в списке [ВводПоСтроке](ОбъектМетаданных_Документ.ВводПоСтроке.md) и не установлен признак [Автонумерация](ОбъектМетаданных_Документ.Автонумерация.md), то значение присваивается реквизиту Номер.  
В противном случае присваивание не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.3.
