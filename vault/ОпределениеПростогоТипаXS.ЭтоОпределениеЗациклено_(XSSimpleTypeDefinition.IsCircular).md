ОпределениеПростогоТипаXS (XSSimpleTypeDefinition)

ЭтоОпределениеЗациклено (IsCircular)

Доступен, начиная с версии 8.1.

Синтаксис:

ЭтоОпределениеЗациклено()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - определение зациклено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет зациклинность определения.  
Под зацикленностью определения понимается:

- зацикленность по базовым типам;
- зацикленность ссылок на определения групп атрибутов;
- зацикленность ссылок на определения групп модели.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
