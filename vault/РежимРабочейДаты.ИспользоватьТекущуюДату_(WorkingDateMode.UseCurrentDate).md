РежимРабочейДаты (WorkingDateMode)

ИспользоватьТекущуюДату (UseCurrentDate)

Доступен, начиная с версии 8.0.

Описание:

Использовать текущую дату в качестве рабочей. В этом режиме рабочая дата не может быть изменена ни интерактивно, ни программно.

Использование в версии:

Доступен, начиная с версии 8.0.
