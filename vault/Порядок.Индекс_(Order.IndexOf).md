Порядок (Order)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ЭлементПорядка>)

Параметры:

<ЭлементПорядка> (обязательный)

Тип: [ЭлементПорядка](ЭлементПорядка_(OrderingItem).md).   
Элемент порядка, индекс которого требуется определить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента порядка в коллекции порядков.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
