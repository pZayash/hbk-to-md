ВариантПоложенияОкна (WindowLocationVariant)

НеПерекрыватьВладельца (DontOverlapOwner)

Доступен, начиная с версии 8.0.

Описание:

При открытии окно не перекроет окно владельца. При условии, что целиком поместится в пределах родительского окна.

Использование в версии:

Доступен, начиная с версии 8.0.
