`ПустаяСтрока`(`<Строка>`: `Строка`) → `Булево`
<!-- signature:end -->

Глобальный контекст (Global context)

ПустаяСтрока (IsBlankString)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСтрока(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Исходная строка.

Возвращаемое значение:

Тип: Булево.   
Истина - строка не содержит значащих символов или не содержит ни одного символа (пустая); Ложь - в противном случае.

Описание:

Проверяет строку на наличие значащих символов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

К незначащим символам относятся символы "пробел", "неразрывный пробел" ([НПП](Символы.НПП.md)), "табуляция" (горизонтальная [Таб](Символы.Таб.md) и вертикальная [ВТаб](Символы.ВТаб.md)), "возврат каретки" ([ВК](Символы.ВК.md)), "перевод строки" ([ПС](Символы.ПС.md)), "перевод формы (страницы)" ([ПФ](Символы.ПФ.md)).

Пример:

|  |
| --- |
| Если ПустаяСтрока(Сотрудник.Фамилия) Тогда     Предупреждение(НСтр("ru = 'Укажите фамилию сотрудника.';"      + " en = 'Define employee surname.'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
