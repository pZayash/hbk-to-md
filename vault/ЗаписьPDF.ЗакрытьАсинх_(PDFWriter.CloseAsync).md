# ЗаписьPDF.ЗакрытьАсинх

ЗаписьPDF (PDFWriter)

ЗакрытьАсинх (CloseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗакрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Закрывает открытый файл. Закрытие файла выполняется автоматически при удалении объекта [ЗаписьPDF](ЗаписьPDF_(PDFWriter).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [Закрыть](ЗаписьPDF.Закрыть_(PDFWriter.Close).md)  
[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [НачатьЗакрытие](ЗаписьPDF.НачатьЗакрытие_(PDFWriter.BeginClose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
