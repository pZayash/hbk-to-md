Глобальный контекст (Global context)

Base64Значение (Base64Value)

Доступен, начиная с версии 8.0.

Синтаксис:

Base64Значение(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Строка, закодированная по алгоритму base64.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   

Описание:

Получает из строки закодированной по алгоритму base64 двоичные данные.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Base64Строка](Глобальный_контекст.Base64Строка.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
