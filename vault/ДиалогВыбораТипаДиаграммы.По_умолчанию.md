ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ДиалогВыбораТипаДиаграммы()

Описание:

Создает новый диалог выбора типа диаграммы.   
По умолчанию свойство [ДоступныеТипыДиаграммы](ДиалогВыбораТипаДиаграммы.ДоступныеТипыДиаграммы_(ChartTypeChooseDialog.AvailableChartTypes).md) содержит коллекцию со всеми типами диаграммы, и свойство [ТипДиаграммы](ДиалогВыбораТипаДиаграммы.ТипДиаграммы_(ChartTypeChooseDialog.ChartType).md) имеет значение [ГистограммаОбъемная](ТипДиаграммы.ГистограммаОбъемная_(ChartType.Column3D).md).

Пример:

|  |
| --- |
| ВыборТипаДиаграммы = Новый ДиалогВыбораТипаДиаграммы; |

Использование в версии:

Доступен, начиная с версии 8.3.23.
