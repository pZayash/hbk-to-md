# ФормаКлиентскогоПриложения.ВводДоступен

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Форма клиентского приложения](Форма_клиентского_приложения.md) › [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) › [Методы](ФормаКлиентскогоПриложения__Методы.md)

ФормаКлиентскогоПриложения (ClientApplicationForm)

ВводДоступен (IsInputAvailable)

Доступен, начиная с версии 8.2.

Синтаксис:

ВводДоступен()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, возможен ли ввод данных в форму. Может быть использован для того, чтобы определить, активизирована ли текущая форма.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
