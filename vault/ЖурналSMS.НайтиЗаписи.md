ЖурналSMS (SMSLog)

НайтиЗаписи (FindRecords)

Доступен, начиная с версии 8.3.11.

Синтаксис:

НайтиЗаписи(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных.md).   
Задает условия поиска.  
В качестве условий отбора могут быть указаны следующие поля:

- НомераТелефонов – тип массив строк, Строка. Допускаются виды сравнения:
  - Для массива: [ВСписке](ВидСравненияКомпоновкиДанных.ВСписке.md), [НеВСписке](ВидСравненияКомпоновкиДанных.НеВСписке.md).
  - Для строки: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md), [Содержит](ВидСравненияКомпоновкиДанных.Содержит.md), [НеСодержит](ВидСравненияКомпоновкиДанных.НеСодержит.md).
- ДатаПолучения – тип Дата. Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше.md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше.md), [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно.md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).
- ДатаОтправки – тип Дата. Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше.md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше.md), [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно.md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).
- Текст – тип Строка. Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md), [Содержит](ВидСравненияКомпоновкиДанных.Содержит.md), [НеСодержит](ВидСравненияКомпоновкиДанных.НеСодержит.md).
- ТипСообщения – тип значение перечисления [ТипSMSСредствТелефонии](ТипSMSСредствТелефонии.md). Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).
- Прочитано – тип Булево. Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив элементов типа [ЗаписьЖурналаSMS](ЗаписьЖурналаSMS.md).

Описание:

Выполняет поиск записей в журнале SMS-сообщений, отвечающих заданному отбору.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.11.
