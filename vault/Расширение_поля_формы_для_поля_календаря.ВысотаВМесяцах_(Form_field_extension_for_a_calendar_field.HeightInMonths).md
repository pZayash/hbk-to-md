# Расширение поля формы для поля календаря.ВысотаВМесяцах

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля календаря](Расширение_поля_формы_для_поля_календаря_(Form_field_extension_for_a_calendar_field).md) › [Свойства](Расширение_поля_формы_для_поля_календаря__Свойства.md)

Расширение поля формы для поля календаря (Form field extension for a calendar field)

ВысотаВМесяцах (HeightInMonths)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Высота календаря в месяцах. Если равна 0, то используется значение свойства [Высота](ОписаниеОтображаемогоОбъектаPDF.Высота_(PDFRepresentationObjectDescription.Height).md). Значение по умолчанию 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Имеет смысл только для режимов совместимости выше [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md). В противном случае свойство доступно, но его изменение не производится.

Использование в версии:

Доступен, начиная с версии 8.3.3.
