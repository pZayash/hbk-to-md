# РезультатXPath.РазмерСнимка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [РезультатXPath](РезультатXPath_(XPathResult).md) › [Свойства](РезультатXPath__Свойства.md)

РезультатXPath (XPathResult)

РазмерСнимка (SnapshotLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит количество узлов в результирующем снимке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если тип результата не равен [НеупорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.НеупорядоченныйСнимокУзлов_(DOMXPathResultType.UnorderedNodeSnapshot).md) или [УпорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.УпорядоченныйСнимокУзлов_(DOMXPathResultType.OrderedNodeSnapshot).md), то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
