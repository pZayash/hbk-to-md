ПолеВвода (TextBox)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак специального режима для ввода пароля, в котором все вводимые символы отображаются символом "\*" (звездочка).  
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
