Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоВыборНезаполненного (AutoChoiceIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Указывает необходимость предоставления выбора незаполненного значения в поле ввода. Если значение свойства [Неопределено](lang__def_Undefined.md) (в Конфигураторе выбрано значение Авто), то при автоматическом определении значения используются текущие значения свойств [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного_(Form_field_extension_for_a_text_box.AutoMarkIncomplete).md) и [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного_(Form_field_extension_for_a_text_box.MarkIncomplete).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
