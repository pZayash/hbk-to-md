# ЭлементТаблицаHTML.УстановитьПользовательскиеДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md) › [ЭлементТаблицаHTML](ЭлементТаблицаHTML_(HTMLTableElement).md) › [Методы](ЭлементТаблицаHTML__Методы.md)

ЭлементТаблицаHTML (HTMLTableElement)

УстановитьПользовательскиеДанные (SetUserData)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьПользовательскиеДанные(<Ключ>, <Данные>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ пользовательских данных.

<Данные> (обязательный)

Тип: Произвольный.   
Пользовательские данные, которые необходимо сохранить в узле.

Возвращаемое значение:

Прежние пользовательские данные для указанного ключа или [Неопределено](lang__def_Undefined.md), если указанному ключу не соответствовали никакие пользовательские данные.

Описание:

Устанавливает данные пользователя.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
