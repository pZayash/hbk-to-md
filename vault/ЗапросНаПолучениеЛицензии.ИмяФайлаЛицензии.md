ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

ИмяФайлаЛицензии (LicenseFileName)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя записываемого файла лицензии без расширения (будет добавлено расширение .lic) или пустая строка.   
По умолчанию пустая строка.   
Если указана пустая строка, имя файла будет построено на основании времени создания запроса на получение лицензии в формате "YYYYMMDDhhmmss".   
Если имя файла задано явно и файл уже существует, то он будет перезаписан, если не используется. В противном случае генерируется исключение.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
