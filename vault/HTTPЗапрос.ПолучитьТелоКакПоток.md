HTTPЗапрос (HTTPRequest)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: [Поток](Поток.md).   

Описание:

Получает поток для записи тела запроса.   
  
В веб-клиенте в браузерах Internet Explorer и Microsoft Edge (до версии 18) метод не поддерживается, и может возвращать пустой поток.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
