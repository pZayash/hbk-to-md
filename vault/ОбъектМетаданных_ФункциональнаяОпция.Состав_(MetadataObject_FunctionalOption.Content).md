ОбъектМетаданных: ФункциональнаяОпция (MetadataObject: FunctionalOption)

Состав (Content)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СоставФункциональнойОпции](СоставФункциональнойОпции_(FunctionalOptionContent).md).   
Содержит состав функциональной опции. Состояние функциональной опции будет влиять на указанные в этом свойстве объекты метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
