# МенеджерКриптографии.НачатьПроверкуСертификата

МенеджерКриптографии (CryptoManager)

НачатьПроверкуСертификата (BeginCheckingCertificate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПроверкуСертификата(<ОписаниеОповещения>, <Сертификат>, <РежимПроверки>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения проверки сертификата со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

В случае ошибки генерируется исключительная ситуация и вызывается процедура обработки ошибок из [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Проверяемый сертификат.

<РежимПроверки> (необязательный)

Тип: [РежимПроверкиСертификатаКриптографии](РежимПроверкиСертификатаКриптографии_(CryptoCertificateCheckMode).md), [Массив](Массив_(Array).md).   
Указывает режим проверки.   
Содержит один объект или массив объектов режимов проверки.

Описание:

Начинает проверку сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
