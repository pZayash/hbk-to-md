ОписаниеКомандыПоляВвода (InputFieldCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [СтандартнаяКомандаПоляВвода](СтандартнаяКомандаПоляВвода.md), [ОписаниеОповещения](ОписаниеОповещения.md), [Массив](Массив.md), Неопределено.   
Содержит описание команды:

- [СтандартнаяКомандаПоляВвода](СтандартнаяКомандаПоляВвода.md) – стандартная команда;
- [ОписаниеОповещения](ОписаниеОповещения.md) – процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
  - <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md);
- [Массив](Массив.md) – задает подменю. Массив может содержать элементы обозначенных выше типов;
- Неопределено – разделитель;
- Любой другой тип - пункт меню не отображается.

  
Если свойство содержит значение типа [СтандартнаяКомандаПоляВвода](СтандартнаяКомандаПоляВвода.md), игнорируются прочие свойства объекта.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
