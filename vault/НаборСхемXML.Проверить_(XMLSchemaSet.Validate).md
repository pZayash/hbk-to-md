# НаборСхемXML.Проверить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Схема XML](Схема_XML.md) › [НаборСхемXML](НаборСхемXML_(XMLSchemaSet).md) › [Методы](НаборСхемXML__Методы.md)

НаборСхемXML (XMLSchemaSet)

Проверить (Validate)

Доступен, начиная с версии 8.1.

Синтаксис:

Проверить(<РазыменовательСхем>)

Параметры:

<РазыменовательСхем> (необязательный)

Тип: [НаборСхемXML](НаборСхемXML_(XMLSchemaSet).md).   
Набор схем XML, используемый для разыменования ссылок на схему из проверяемого набора.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - успешная проверка набора схем; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет набора схем XML на правильность в соответствии со стандартом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
