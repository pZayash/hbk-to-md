# ПерерасчетНаборЗаписей.<Имя перерасчета>.Сдвинуть

ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Запись>, <Смещение>)

Параметры:

<Запись> (обязательный)

Тип: [ПерерасчетЗапись.\<Имя перерасчета\>](ПерерасчетЗапись.Имя_перерасчета_(RecalculationRecord.Recalculation_name).md).   
Запись, которую требуется сдвинуть.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца набора записей (увеличение индекса), отрицательное - к началу.

Описание:

Сдвигает запись в наборе на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
