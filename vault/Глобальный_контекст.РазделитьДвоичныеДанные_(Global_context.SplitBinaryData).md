# Глобальный контекст.РазделитьДвоичныеДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Процедуры и функции работы с двоичными данными](Процедуры_и_функции_работы_с_двоичными_данными.md)

Глобальный контекст (Global context)

РазделитьДвоичныеДанные (SplitBinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

РазделитьДвоичныеДанные(<ДвоичныеДанные>, <РазмерЧасти>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Объект типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

<РазмерЧасти> (обязательный)

Тип: [Число](lang__def_Number.md).   
Размер одной части данных.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание:

Разделяет двоичные данные на части заданного размера. Размер задается в байтах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [РазделитьФайл](Глобальный_контекст.РазделитьФайл_(Global_context.SplitFile).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
