СлойГеографическойСхемы (GeographicalSchemaLayer)

УстановитьЗначение (SetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьЗначение(<Объект>, <Серия>, <Значение>, <Подсказка>, <Расшифровка>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   
Объект, для которого будет устанавливаться значение серии.

<Серия> (обязательный)

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы_(GeographicalSchemaLayerDataSeries).md).   
Серия, для которой устанавливается значение.

<Значение> (необязательный)

Тип: Произвольный.   
Устанавливаемое значение. Если значение не задано, текущее значение будет удалено.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит текст всплывающей подсказки.

<Расшифровка> (необязательный)

Тип: Произвольный.   
Значение, используемое при расшифровке.

Описание:

Устанавливает значение для указанной серии. Дополнительно может быть установлена расшифровка значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
