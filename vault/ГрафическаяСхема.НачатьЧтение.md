ГрафическаяСхема (GraphicalSchema)

НачатьЧтение (BeginReading)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЧтение(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения чтения файла со следующими параметрами:

- <Результат> - результат операции, в настоящее время не используется, всегда имеет значение Истина,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла графической схемы.

Описание:

Вызывает начало чтения графической схемы из файла.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

[ГрафическаяСхема](ГрафическаяСхема.md), метод [Прочитать](ГрафическаяСхема.Прочитать.md)  
[ГрафическаяСхема](ГрафическаяСхема.md), метод [ПрочитатьАсинх](ГрафическаяСхема.ПрочитатьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.7.

Описание изменено в версии 8.3.18.
