# СписокПолнотекстовогоПоиска.ПолноеКоличество

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Полнотекстовый поиск](Полнотекстовый_поиск.md) › [СписокПолнотекстовогоПоиска](СписокПолнотекстовогоПоиска_(FullTextSearchList).md) › [Методы](СписокПолнотекстовогоПоиска__Методы.md)

СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолноеКоличество (TotalCount)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолноеКоличество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Полное количество результатов. Например, всего результатов 10000, а первая порция - с 1 до 20.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Количество объектов, попадающих в результаты поиска, не учитывает права доступа пользователя к объектам. Поэтому количество объектов в результатах поиска может быть меньше количества, возвращаемого данным методом.  
Полное количество объектов может изменяться при переходе между частями списка результатов поиска ([СписокПолнотекстовогоПоиска](СписокПолнотекстовогоПоиска_(FullTextSearchList).md)).

Использование в версии:

Доступен, начиная с версии 8.1.
