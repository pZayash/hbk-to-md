ВыборкаИзРезультатаЗапроса (QueryResultSelection)

Группировка (Group)

Доступен, начиная с версии 8.0.

Синтаксис:

Группировка()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Имя группировки текущей записи. Если группировки нет, возвращается пустая строка.

Описание:

Получает имя группировки текущей записи выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТекГруппировка = Выборка.Группировка(); |

Использование в версии:

Доступен, начиная с версии 8.0.
