ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, в который нужно записать пакет отображаемых документов.

<ТипФайла> (обязательный)

Тип: [ТипФайлаПакетаОтображаемыхДокументов](ТипФайлаПакетаОтображаемыхДокументов_(RepresentableDocumentBatchFileType).md).   
Содержит формат файла, в котором будет сохранен пакет отображаемых документов.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания:

- [Истина](lang__def_BooleanTrue.md) - если запись была инициирована с помощью расширения браузера и пользователь согласился на его получение, то есть запись файла была завершена;
- [Ложь](lang__def_BooleanFalse.md) - если запись была инициирована с помощью расширения браузера, но пользователь не согласился на его получение, то есть запись файла не была завершена;
- [Неопределено](lang__def_Undefined.md) - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

Описание:

Асинхронная функция. Записывает пакет отображаемых документов в файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
