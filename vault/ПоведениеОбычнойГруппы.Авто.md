ПоведениеОбычнойГруппы (UsualGroupBehavior)

Авто (Auto)

Доступен, начиная с версии 8.3.20.

Описание:

Для формы отчета интепретируется следующим образом:

- В тонком и веб-клиентах используется значение [Обычная](ТипФормы.Обычная.md),
- В мобильном клиенте:
  - Если задан заголовок группы, то интерпретируется как [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая.md), при этом не учитывается значение свойства группы [ОтображатьЗаголовок](Расширение_группы_формы_для_обычной_группы.ОтображатьЗаголовок.md);
  - Если не задан заголовок группы, то интерпретируется как [Обычная](ТипФормы.Обычная.md).

Для остальных форм используется значение [Обычная](ТипФормы.Обычная.md).

Использование в версии:

Доступен, начиная с версии 8.3.20.
