Глобальный контекст (Global context)

ВвестиСтроку (InputString)

Доступен, начиная с версии 8.0.

Синтаксис:

ВвестиСтроку(<Строка>, <Подсказка>, <Длина>, <Многострочность>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Доступная в модуле переменная. В эту переменную будет помещена введенная в диалоге строка. Начальное значение переменной будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода строки. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<Длина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Длина вводимой строки. Если параметр не указан, то строка неограниченной длины.  
Значение по умолчанию: 0.

<Многострочность> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим ввода многострочного текста: [Истина](lang__def_BooleanTrue.md) - ввод многострочного текста с разделителями строк; [Ложь](lang__def_BooleanFalse.md) - ввод простой строки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - строка введена; [Ложь](lang__def_BooleanFalse.md) - пользователь отказался от ввода строки.

Описание:

Вызывает диалог для ввода строки.  
При вызове метода в веб-клиенте генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [ПоказатьВводСтроки](Глобальный_контекст.ПоказатьВводСтроки_(Global_context.ShowInputString).md).

Пример:

|  |
| --- |
| Текст = ""; Подсказка = "Введите текст напоминания"; Если ВвестиСтроку(Текст, Подсказка, 0, Истина) Тогда     // запомнить текст напоминания  КонецЕсли; |
