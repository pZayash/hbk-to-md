СвойствоXDTO (XDTOProperty)

ТипВладелец (OwnerType)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Владелец свойства - тип объекта, которому принадлежит данное свойство или объект (с открытой моделью содержания), которому принадлежит данное свойство.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
