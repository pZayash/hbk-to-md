Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfileAddIn (CreateSecurityProfileAddIn)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfileAddIn()

Возвращаемое значение:

Тип: [Внешний компонент](Внешний_компонент_(ISecurityProfileAddIn).md).   

Описание:

Создает новый разрешенный внешний компонент для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
