# ДокументОбъект.<Имя документа>.Движения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Свойства](ДокументОбъект.Имя_документа__Свойства.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Движения (RegisterRecords)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияДвижений](КоллекцияДвижений_(RegisterRecordsCollection).md).   
Предоставляет доступ к коллекции наборов записей движений документа. Свойства коллекции содержат наборы записей движений документа, которые включены для данного документа в конфигурации. Имена свойств совпадают с именами объектов конфигурации, по которым могут совершаться движения, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
