ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
