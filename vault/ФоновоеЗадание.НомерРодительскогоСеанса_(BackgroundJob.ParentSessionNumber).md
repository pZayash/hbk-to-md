ФоновоеЗадание (BackgroundJob)

НомерРодительскогоСеанса (ParentSessionNumber)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Номер родительского сеанса для фонового задания, из которого данное фоновое задание было запущено.  
В остальных случаях имеет значение 0.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
