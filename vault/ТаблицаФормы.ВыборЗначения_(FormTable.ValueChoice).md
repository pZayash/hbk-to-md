ТаблицаФормы (FormTable)

ВыборЗначения (ValueChoice)

Доступен, начиная с версии 8.2.

Синтаксис:

ВыборЗначения(<Значение>, <СтандартнаяОбработка>)

Параметры:

<Значение>

Выбранное значение или массив выбранных значений в случае, если свойство [МножественныйВыбор](ТаблицаФормы.МножественныйВыбор_(FormTable.MultipleChoice).md) = [Истина](lang__def_BooleanTrue.md).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при выборе значения из таблицы в режиме выбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
