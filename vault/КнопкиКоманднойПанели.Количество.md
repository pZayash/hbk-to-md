КнопкиКоманднойПанели (CommandBarButtons)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество кнопок в командной панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| КП = ЭлементыФормы.КоманднаяПанель1; Сообщить("Итого кнопок " + КП.Кнопки.Количество()); |

Использование в версии:

Доступен, начиная с версии 8.0.
