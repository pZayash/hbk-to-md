ЧтениеФайлаАрхива (ArchiveFileReader)

На основании имени файла

Доступен, начиная с версии 8.3.26.

Синтаксис:

Новый ЧтениеФайлаАрхива(<Пароль>, <ТипФайлаАрхива>)

Параметры:

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к файлу архива, если файл зашифрован.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива_(ArchiveFileType).md).   
Тип файла архива, который требуется открыть для чтения.   
По умолчанию, тип определяется по расширению имени файла.

Описание:

Открывает для чтения указанный файл архива. Аналогичен методу [Открыть](ЧтениеФайлаАрхива.Открыть_(ArchiveFileReader.Open).md).

Использование в версии:

Доступен, начиная с версии 8.3.26.
