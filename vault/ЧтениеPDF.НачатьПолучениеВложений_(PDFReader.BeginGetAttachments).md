ЧтениеPDF (PDFReader)

НачатьПолучениеВложений (BeginGetAttachments)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПолучениеВложений(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <Вложения> - объект типа [КоллекцияВложенийPDF](КоллекцияВложенийPDF_(PDFAttachmentCollection).md), содержащий вложения PDF-документов.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Инициирует получение массива, содержащего описания присоединенных к PDF-документу вложений.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
