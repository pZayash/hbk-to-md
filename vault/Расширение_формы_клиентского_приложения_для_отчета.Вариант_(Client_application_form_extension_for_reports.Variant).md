Расширение формы клиентского приложения для отчета (Client application form extension for reports)

Вариант (Variant)

Доступен, начиная с версии 8.2.

Описание:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md), [Строка](lang__def_String.md).   
Настройки варианта отчета. Указываются либо сами настройки, либо в строке адрес хранилища, из которого добываются настройки.

Использование в версии:

Доступен, начиная с версии 8.2.
