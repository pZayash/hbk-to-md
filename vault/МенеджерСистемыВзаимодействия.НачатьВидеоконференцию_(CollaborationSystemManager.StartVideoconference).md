МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьВидеоконференцию (StartVideoconference)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьВидеоконференцию(<ОписаниеОповещения>, <Участники>, <Обсуждение>, <ПоказатьВыборУчастников>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после отправки приглашения участникам.  
Процедура вызывается со следующими параметрами:

- <Начата> - тип [Булево](lang__Булево_(Boolean).md), признак начала видеоконференции:
  - [Истина](lang__def_BooleanTrue.md) - всем участникам видеоконференции отправлены приглашения,
  - [Ложь](lang__def_BooleanFalse.md) - видеоконференция отменена (в диалоге выбора участников нажата кнопка "Отмена");
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

  
Если конференция не была создана по причине ошибки (например, не подключен микрофон), то вызывается процедура обработки ошибки (если указана).

<Участники> (необязательный)

Тип: [КоллекцияИдентификаторовПользователейСистемыВзаимодействия](КоллекцияИдентификаторовПользователейСистемыВзаимодействия_(CollaborationSystemUserIDCollection).md).   
Если <ПоказатьВыборУчастников> равен [Истина](lang__def_BooleanTrue.md), то содержит список пользователей, отображаемых в диалоге участников видеоконференции.  
Если <ПоказатьВыборУчастников> равен [Ложь](lang__def_BooleanFalse.md), то содержит список пользователей, которым будет отправлен запрос на участие в видеоконференции.

<Обсуждение> (необязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Устанавливает связь видеоконференции с обсуждением.  
Если <ПоказатьВыборУчастников> имеет значение [Истина](lang__def_BooleanTrue.md), то список участников обсуждения отображается в диалоге выбора участников видеоконференции.

<ПоказатьВыборУчастников> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отображения диалога выбора участников видеоконференции.  
[Истина](lang__def_BooleanTrue.md) - диалог отображается.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Начинает новую видеоконференцию.  
Если не заданы <Участники>, тогда:

- Если <ПоказатьВыборУчастников> имеет значение [Истина](lang__def_BooleanTrue.md), открывается диалог выбора участников с пустым списком участников (добавлять участников можно подбором),
- Если <ПоказатьВыборУчастников> имеет значение [Ложь](lang__def_BooleanFalse.md), то видеоконференция начинается без участников.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.
