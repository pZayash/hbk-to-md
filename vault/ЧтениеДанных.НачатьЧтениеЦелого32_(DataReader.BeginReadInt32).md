# ЧтениеДанных.НачатьЧтениеЦелого32

ЧтениеДанных (DataReader)

НачатьЧтениеЦелого32 (BeginReadInt32)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтениеЦелого32(<ОписаниеОповещения>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - тип Число. Результат чтения.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый при чтении числа.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

[Неопределено](lang__def_Undefined.md) - если не удалось прочитать число, поскольку был достигнут конец потока.

Описание:

Начать чтение 32-битного целого числа из потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
