Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [КонтрольныеТочкиИтоговСхемыЗапроса](КонтрольныеТочкиИтоговСхемыЗапроса.md).   
Содержит контрольные точки итогов схемы запроса.  
Соответствует списку контрольных точек из выражения ИТОГИ ПО языка запросов.  
Контрольные точки представляют собой значения, для которых будут вычисляться значения итоговых полей.  
Доступно только, если у схемы запроса выключен режим компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
