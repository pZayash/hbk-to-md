# КоллекцияОбластейТабличногоДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличный документ](Табличный_документ-2.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md) или [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются области или рисунки табличного документа.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Свойства:

[\<Имя области\> (\<Area name\>)](КоллекцияОбластейТабличногоДокумента.Имя_области_(SpreadsheetDocumentAreaCollection.Area_name).md)  

Методы:

[Индекс (IndexOf)](КоллекцияОбластейТабличногоДокумента.Индекс_(SpreadsheetDocumentAreaCollection.IndexOf).md)  
[Количество (Count)](КоллекцияОбластейТабличногоДокумента.Количество_(SpreadsheetDocumentAreaCollection.Count).md)  
[Найти (Find)](КоллекцияОбластейТабличногоДокумента.Найти_(SpreadsheetDocumentAreaCollection.Find).md)  
[Получить (Get)](КоллекцияОбластейТабличногоДокумента.Получить_(SpreadsheetDocumentAreaCollection.Get).md)  

Описание:

Представляет собой коллекцию областей табличного документа, включая именованные рисунки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), свойство [Области](ТабличныйДокумент.Области_(SpreadsheetDocument.Areas).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
