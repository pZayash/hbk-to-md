Картинка (Picture)

Преобразовать (Convert)

Не рекомендуется использовать, начиная с версии 8.3.14.

Доступен, начиная с версии 8.0.

Синтаксис:

Преобразовать(<ФорматКартинки>, <ВсеВарианты>, <ПодобратьВариант>)

Параметры:

<ФорматКартинки> (необязательный)

Тип: [ФорматКартинки](ФорматКартинки_(PictureFormat).md).   
Определяет формат картинки.

<ВсеВарианты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки. Если значение [Ложь](lang__def_BooleanFalse.md) - будет преобразована одна картинка. Если значение [Истина](lang__def_BooleanTrue.md) - все варианты картинки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ПодобратьВариант> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md), [Неопределено](lang__def_Undefined.md).   
Данный параметр используется только, если картинка представлена архивом, содержащим варианты этой картинки, и <ВсеВарианты> установлен в значение [Ложь](lang__def_BooleanFalse.md).  
Пары "Ключ-Значение" соответствуют xml-тегам в манифесте вариантов картинки. Если параметры выбора варианта не заданы, будет выбран вариант по умолчанию. Если параметры указывают на несуществующий в архиве вариант, то будет возвращено [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Создает копию картинки и преобразовывает ее в заданный формат.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Картинки растровых форматов и векторного формата SVG преобразовываются в форматы BMP и PNG.  
Картинки других векторных форматов не преобразовываются.  
Для многокадровых картинок (Icon и анимированный GIF) преобразуется первый кадр.  
В мобильной версии платформы доступно преобразование только в форматы PNG и JPEG.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.14.

Описание изменено в версии 8.3.22.
