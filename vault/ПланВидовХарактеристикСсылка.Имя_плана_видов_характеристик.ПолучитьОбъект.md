Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект, предназначенный для модификации плана видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
