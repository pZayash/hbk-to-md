ПолеКартинки (PictureBox)

ТекстНевыбраннойКартинки (NonselectedPictureText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст, отображаемый, если картинка не выбрана.

Доступность:

Толстый клиент.

Примечание:

Текст всегда многострочный, выровненный по центру (по горизонтали и вертикали).

Использование в версии:

Доступен, начиная с версии 8.0.
