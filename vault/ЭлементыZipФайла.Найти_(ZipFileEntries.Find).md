ЭлементыZipФайла (ZipFileEntries)

Найти (Find)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Найти](ЭлементыФайлаАрхива.Найти_(ArchiveFileEntries.Find).md)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяЭлемента>)

Параметры:

<ИмяЭлемента> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента.

Возвращаемое значение:

Тип: [ЭлементZipФайла](ЭлементZipФайла_(ZipFileEntry).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента по его имени.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
