ПланВидовХарактеристикВыборка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesSelection.<Chart of characteristic types name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующий элемент выбран; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующий элемент из выборки. Для обхода плана видов характеристик нужно после получения выборки вызвать данный метод для позиционирования на первый элемент и далее вызывать до тех пор, пока не будет возвращено значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Выборка = ПланыВидовХарактеристик.СвойстваОбъектов.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка выборки  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
