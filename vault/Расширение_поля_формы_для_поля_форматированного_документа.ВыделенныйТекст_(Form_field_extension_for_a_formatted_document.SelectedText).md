# Расширение поля формы для поля форматированного документа.ВыделенныйТекст

Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ВыделенныйТекст (SelectedText)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выделенный пользователем текст поля ввода на момент чтения свойства.   
При записи в данное свойство нового текста этот текст заменяет текст, выделенный в настоящий момент. При этом выделение сбрасывается.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Не используется, если поле расположено в таблице.

Использование в версии:

Доступен, начиная с версии 8.2.
