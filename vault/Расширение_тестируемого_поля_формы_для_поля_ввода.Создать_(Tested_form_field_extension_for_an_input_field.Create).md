Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

Создать (Create)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Создать()

Описание:

Создает новый элемент из поля ввода (аналогично нажатию кнопки "Создать" в списке выбора).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
