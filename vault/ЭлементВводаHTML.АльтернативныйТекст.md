ЭлементВводаHTML (HTMLInputElement)

АльтернативныйТекст (Alt)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Альтернативный текст элемента ввода (атрибут alt), который отображается в случае, если элемент ввода не может быть правильно отображен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
