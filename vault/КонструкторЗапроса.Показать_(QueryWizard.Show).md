КонструкторЗапроса (QueryWizard)

Показать (Show)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершении никакая процедура вызвана не будет. Если параметр указан, то содержит описание процедуры, которая будет вызвана после закрытия диалога конструктора со следующими параметрами:

- <Текст> - Текст, если конструктор закрыли по кнопке "OK", [Неопределено](lang__def_Undefined.md) - в противном случае;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта ОписаниеОповещения.

Описание:

Открывает немодально конструктор запроса.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[КонструкторЗапроса](КонструкторЗапроса_(QueryWizard).md), метод [ОткрытьАсинх](КонструкторЗапроса.ОткрытьАсинх_(QueryWizard.DoAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.18.
