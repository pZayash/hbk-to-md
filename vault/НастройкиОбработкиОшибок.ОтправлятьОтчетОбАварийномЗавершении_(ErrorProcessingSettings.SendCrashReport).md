НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОтчетОбАварийномЗавершении (SendCrashReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ОтправлятьОтчетОбАварийномЗавершенииНаКлиенте](НастройкиОбработкиОшибок.ОтправлятьОтчетОбАварийномЗавершенииНаКлиенте_(ErrorProcessingSettings.SendCrashReportOnClient).md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость отправки информации об аварийном завершении.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

См. также:

[НастройкиОбработкиОшибок](НастройкиОбработкиОшибок_(ErrorProcessingSettings).md), свойство [ОтправлятьОтчетОбАварийномЗавершенииНаСервере](НастройкиОбработкиОшибок.ОтправлятьОтчетОбАварийномЗавершенииНаСервере_(ErrorProcessingSettings.SendCrashReportOnServer).md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.
