МенеджерКриптографии (CryptoManager)

ПолучитьДанныеИзПодписи (GetDataFromSignature)

Доступен, начиная с версии 8.3.24.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.  
Если параметр типа [Строка](lang__def_String.md), то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), то он задает данные подписи.

Описание варианта метода:

Данные возвращаются методом как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Сохранение данных в файл или запись их в поток

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.  
Если параметр типа [Строка](lang__def_String.md), то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), то он задает данные подписи.

<ВыходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Имя файла или поток, в который будут выгружены данные, содержащиеся в подписи.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   
В зависимости от варианта метода, либо [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), либо [Неопределено](lang__def_Undefined.md).

Описание:

Извлекает подписанные данные из электронной подписи.   
Если данные отсутствуют в подписи, т.е. она является отсоединенной, то будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md), метод [ПолучитьДанныеИзПодписиАсинх](МенеджерКриптографии.ПолучитьДанныеИзПодписиАсинх_(CryptoManager.GetDataFromSignatureAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.24.
