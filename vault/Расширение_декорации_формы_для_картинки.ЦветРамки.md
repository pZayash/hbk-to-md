Расширение декорации формы для картинки (Form decoration extension for a picture)

ЦветРамки (BorderColor)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет.md).   
Содержит цвет рамки вокруг поля картинки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется, если рамка имеет стиль [Выпуклая](ТипРамкиЭлементаУправления.Выпуклая.md) или [Вдавленная](ТипРамкиЭлементаУправления.Вдавленная.md) - в этом случае рамка рисуется притененным цветом от цвета фона.

Использование в версии:

Доступен, начиная с версии 8.2.
