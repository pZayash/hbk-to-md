ТипКаноническогоXML (XMLCanonicalizationType)

КаноническийXML1\_1СКомментариями (XMLCanonicalization1\_1WithComments)

Доступен, начиная с версии 8.3.9.

Описание:

Приведение к каноническому виду по стандарту W3C "Canonical XML 1.0". Комментарии учитываются.

Использование в версии:

Доступен, начиная с версии 8.3.9.
