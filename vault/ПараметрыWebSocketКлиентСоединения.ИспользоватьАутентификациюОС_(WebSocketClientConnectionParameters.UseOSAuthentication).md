ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

ИспользоватьАутентификациюОС (UseOSAuthentication)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость использовать аутентификацию операционной системы на WebSocket-сервере (указывается пользователь компьютера/сети, где размещен WebSocket-сервер).  
Если имя пользователя и пароль не указаны, для аутентификации используется имя пользователя, от имени которого работает сервис, обслуживающий WebSocket-соединение.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

В веб-клиенте не поддерживается. При чтении возвращается [Истина](lang__def_BooleanTrue.md), записи значения [Ложь](lang__def_BooleanFalse.md) вызывается исключение "Свойство недоступно в веб-клиенте".

Использование в версии:

Доступен, начиная с версии 8.3.27.
