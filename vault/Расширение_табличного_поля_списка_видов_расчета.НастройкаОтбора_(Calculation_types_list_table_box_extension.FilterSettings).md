Расширение табличного поля списка видов расчета (Calculation types list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора_(FilterSettings).md).   
Предоставляет доступ к настройкам отбора списка видов расчета. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для вида расчетов элементов отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
