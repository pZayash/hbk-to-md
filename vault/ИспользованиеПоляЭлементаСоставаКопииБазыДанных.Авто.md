ИспользованиеПоляЭлементаСоставаКопииБазыДанных (DatabaseCopyContentItemFieldUse)

Авто (Auto)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Описание:

Значение по умолчанию.  
Использование и значение поля вычисляется, начиная с собственного свойства поля в порядке Поле - Элемент состава - Копия. Вычисление останавливается, если значение установлено явно. Если значение свойства установлено явно, то получается оно. Если свойство поля установлено в значение Авто, то проверяется свойство ИспользованиеПоляЭлементаСоставаКопииБазыДанных. В зависимости от типа поля проверяется следующее свойство:

- Для табличной части - [ИспользованиеТабличныхЧастей](ЭлементСоставаКопииБазыДанных.ИспользованиеТабличныхЧастей.md),
- Для простых полей в зависимости от типа:
  - ХранилищеЗначения - [ИспользованиеХранилищЗначения](ЭлементСоставаКопииБазыДанных.ИспользованиеХранилищЗначения.md),
  - Строка неограниченной длины - [ИспользованиеСтрокНеограниченнойДлины](ЭлементСоставаКопииБазыДанных.ИспользованиеСтрокНеограниченнойДлины.md),
  - Прочие типы - [ИспользованиеПрочихПолей](ЭлементСоставаКопииБазыДанных.ИспользованиеПрочихПолей.md).

Если свойство установлено явно, то берется оно. Если установлено значение Авто, то проверяется аналогичное свойство копии: если оно установлено явно, то берется оно. Если поле в копии имеет значение Авто, то применяются следующие правила вычисления использования для полей:

- Копия со стандартной репликацией:
  - Поле - табличная часть, реквизит типа [ХранилищеЗначения](ХранилищеЗначения.md) – [НеИспользовать](ИспользованиеПоляЭлементаСоставаКопииБазыДанных.НеИспользовать.md).
  - Прочие типы – [Использовать](ИспользованиеПоляЭлементаСоставаКопииБазыДанных.Использовать.md).
- Копия с внешней репликацией:
  - Для всех полей – [Использовать](ИспользованиеПоляЭлементаСоставаКопииБазыДанных.Использовать.md).

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
