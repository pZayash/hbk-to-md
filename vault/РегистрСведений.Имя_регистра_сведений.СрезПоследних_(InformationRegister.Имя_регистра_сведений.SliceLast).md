Описание:

Предназначена для получения наиболее поздних записей регистра сведений на указанную дату (включительно). Включает только активные записи. По каждой комбинации измерений будет найдена наиболее поздняя запись, но не более поздняя, чем указанная дата.  
В режиме совместимости [Версия8\_3\_17](РежимСовместимости.Версия8_3_17_(CompatibilityMode.Version8_3_17).md) и ниже включает записи независимо от признака активности.

Пример:

|  |
| --- |
| // Получение всех наиболее поздних записей регистра сведений  ВЫБРАТЬ \* ИЗ РегистрСведений.КурсыВалют.СрезПоследних()  // Получение всех наиболее поздних записей регистра сведений   // на указанную дату и по указанной валюте.  ВЫБРАТЬ \* ИЗ РегистрСведений.КурсыВалют.СрезПоследних(&ПараметрДата,      Валюта = &ПараметрВалюта) |
