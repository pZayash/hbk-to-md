**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Схема компоновки данных](Схема_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[НаборДанныхЗапросСхемыКомпоновкиДанных](НаборДанныхЗапросСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetQuery).md), [НаборДанныхОбъектСхемыКомпоновкиДанных](НаборДанныхОбъектСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetObject).md), [НаборДанныхОбъединениеСхемыКомпоновкиДанных](НаборДанныхОбъединениеСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetUnion).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы данных.  
Возможно обращение к набору данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
