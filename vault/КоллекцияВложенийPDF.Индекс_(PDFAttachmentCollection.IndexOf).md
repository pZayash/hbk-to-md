КоллекцияВложенийPDF (PDFAttachmentCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ВложениеPDF](ВложениеPDF_(PDFAttachment).md).   
Элемент, индекс которого требуется получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Если элемент не найден возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
