КолонкиРезультатаМоделиПрогноза (PredictionModelResultColumns)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Колонка>, <Позиция>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаРезультатаМоделиПрогноза](КолонкаРезультатаМоделиПрогноза_(PredictionModelResultColumn).md).   
Устанавливаемая колонка.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить колонку.

Описание:

Устанавливает колонку в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
