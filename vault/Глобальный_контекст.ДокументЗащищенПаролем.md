Глобальный контекст (Global context)

ДокументЗащищенПаролем (DocumentIsPasswordProtected)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: Из файла

Синтаксис:

ДокументЗащищенПаролем(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла для проверки наличия защиты паролем.

Описание варианта метода:

Проверяет наличие защиты паролем у файла. Применяется для файлов форматов:

- [ДокументPDF](ДокументPDF.md).

Вариант синтаксиса: Из потока

Синтаксис:

ДокументЗащищенПаролем(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [ФайловыйПоток](ФайловыйПоток.md), [ПотокВПамяти](ПотокВПамяти.md), [Поток](Поток.md).   
Поток для проверки наличия защиты паролем.

Описание варианта метода:

Проверяет наличие защиты паролем у файла. Применяется для файлов форматов:

- [ДокументPDF](ДокументPDF.md).

Возвращаемое значение:

Тип: Булево.   

Описание:

Проверяет наличие защиты паролем у файла. Применяется для файлов форматов:

- [ДокументPDF](ДокументPDF.md).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
