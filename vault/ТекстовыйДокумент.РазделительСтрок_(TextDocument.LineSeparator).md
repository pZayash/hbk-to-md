ТекстовыйДокумент (TextDocument)

РазделительСтрок (LineSeparator)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет разделитель строк. Может принимать только следующие значения: [ВК](Символы.ВК_(Chars.CR).md), [ПС](Символы.ПС_(Chars.LF).md) или [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).  
При попытке установки других значений будет инициировано исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию [ПС](Символы.ПС_(Chars.LF).md).

Использование в версии:

Доступен, начиная с версии 8.1.
