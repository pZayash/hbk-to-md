# ВедущиеВидыРасчета.<Имя плана видов расчета>.Вставить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [ВедущиеВидыРасчета.<Имя плана видов расчета>](ВедущиеВидыРасчета.Имя_плана_видов_расчета_(LeadingCalculationTypes.Chart_of_calculation_types_name).md) › [Методы](ВедущиеВидыРасчета.Имя_плана_видов_расчета__Методы.md)

ВедущиеВидыРасчета.<Имя плана видов расчета> (LeadingCalculationTypes.<Chart of calculation types name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс позиции в табличной части, в которую необходимо вставить строку.

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ВедущиеВидыРасчетаСтрока.\<Имя плана видов расчета\>](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(LeadingCalculationTypesRow.Chart_of_calculation_types_name).md).   
Вставленная строка.

Описание:

Вставляет строку с указанным индексом в табличную часть.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода допустимо только в том случае, если табличная часть получена из свойства объекта. Если табличная часть получена из свойства ссылки (или выборки), то использование этого метода будет вызывать ошибку выполнения.

Использование в версии:

Доступен, начиная с версии 8.0.
