# СущностьDOM.НайтиПрефикс

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [СущностьDOM](СущностьDOM_(DOMEntity).md) › [Методы](СущностьDOM__Методы.md)

СущностьDOM (DOMEntity)

НайтиПрефикс (LookupPrefix)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПрефикс(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен, префикс которого необходимо найти.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Производит поиск префикса для URI пространства имен, начиная с данного узла. Алгоритм поиска префикса описан в спецификации DOM Core.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
