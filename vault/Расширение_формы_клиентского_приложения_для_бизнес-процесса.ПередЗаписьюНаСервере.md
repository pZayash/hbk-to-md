Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

ПередЗаписьюНаСервере (BeforeWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗаписьюНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись произведена не будет и будет вызвано исключение.  
Значение по умолчанию: Ложь.

<ТекущийОбъект>

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса.md).   
Записываемый объект.

<ПараметрыЗаписи>

Структура, содержащая параметры записи. Для формы бизнес-процесса существует один предопределенный параметр Старт. При отработке команд "Старт", "Стартовать и закрыть" он устанавливается в Истина.

Описание:

Вызывается перед записью объекта на сервере.   
Можно отменить запись. Выполняется вне транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
