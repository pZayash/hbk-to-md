ПараметрыФормированияКомандПоляПланировщика (PlannerFieldCommandGenerateParameters)

Источник (Source)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ИсточникКомандПоляПланировщика](ИсточникКомандПоляПланировщика.md).   
Содержит информацию о том, на какой объект было произведено нажатие.  
В зависимости от того, какой элемент содержит форматированную строку с гиперссылкой, заполняются разные свойства объекта типа [ПараметрыФормированияКомандПланировщика](ПараметрыФормированияКомандПланировщика.md).  
Если источник равен [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка.md), тогда:

- Заполнено свойство либо [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md), либо [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md), либо [Измерение](ПараметрыФормированияКомандПланировщика.Измерение.md),
- если навигационная ссылка находится в заголовке элемента измерения, то свойство [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md), либо [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md) заполнен, иначе содержит Неопределено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата.md) содержит Неопределено,
- [НавигационнаяСсылка](ПараметрыФормированияКомандПланировщика.НавигационнаяСсылка.md) заполнено,
- [Действие](ПараметрыФормированияКомандПланировщика.Действие.md) содержит Неопределено,
- Команды содержат стандартные команды [ПерейтиПоНавигационнойСсылке](СтандартнаяКомандаПланировщика.ПерейтиПоНавигационнойСсылке.md), [КопироватьНавигационнуюСсылку](СтандартнаяКомандаПланировщика.КопироватьНавигационнуюСсылку.md),
- КомандаПоУмолчанию заполнено командой ПерейтиПоНавигационнойСсылке.

Если источник равен [Действие](ИсточникКомандПланировщика.Действие.md) тогда:

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md) заполнено,
- свойства [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md), [Измерение](ПараметрыФормированияКомандПланировщика.Измерение.md), [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md), содержит Неопределено, [Дата](ПараметрыФормированияКомандПланировщика.Дата.md), [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md), [НавигационнаяСсылка](ПараметрыФормированияКомандПланировщика.НавигационнаяСсылка.md) содержат Неопределено,
- Список команд не содержит стандартных команд,
- <КомандаПоУмолчанию> заполнено командой [ВыполнитьДействие](СтандартнаяКомандаПланировщика.ВыполнитьДействие.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
