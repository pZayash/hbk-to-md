НастройкаВторогоФактораАутентификации (SecondAuthenticationFactorSetting)

Параметры (Parameters)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Настраиваемые параметры, которые будут использованы в шаблоне при аутентификации. Элемент соответствия представляет собой пару из имени параметра и его значения.  
Например:   
Если задано соответствие "sms\_phone\_number, +71234567890", тогда все вхождения подстроки "sms\_phone\_number" внутри свойств объекта [ШаблонНастройкиВторогоФактораАутентификации](ШаблонНастройкиВторогоФактораАутентификации_(SecondAuthenticationFactorSettingTemplate).md) будут заменены на +71234567890.  
Если в качестве шаблона используется объект типа [ШаблонНастройкиВторогоФактораАутентификации](ШаблонНастройкиВторогоФактораАутентификации_(SecondAuthenticationFactorSettingTemplate).md), тогда будет выполнен поиск вхождений параметра в свойствах:

- по структуре [HTTPЗапросНаАутентификацию](ШаблонНастройкиВторогоФактораАутентификации.HTTPЗапросНаАутентификацию_(SecondAuthenticationFactorSettingTemplate.HTTPAuthenticationRequest).md):
  - [АдресРесурса](HTTPЗапрос.АдресРесурса_(HTTPRequest.ResourceAddress).md),
  - [Заголовки](HTTPЗапрос.Заголовки_(HTTPRequest.Headers).md),
  - в теле файла, устанавливаемом любым возможным способом.
- по тексту в [МетодHTTPЗапросаНаАутентификацию](ШаблонНастройкиВторогоФактораАутентификации.МетодHTTPЗапросаНаАутентификацию_(SecondAuthenticationFactorSettingTemplate.HTTPAuthenticationRequestMethod).md);
- по структуре [МетодHTTPЗапросаНаПроверкуРезультатаАутентификации](ШаблонНастройкиВторогоФактораАутентификации.МетодHTTPЗапросаНаПроверкуРезультатаАутентификации_(SecondAuthenticationFactorSettingTemplate.AuthenticationResultVerificationHTTPRequestMethod).md):
  - [АдресРесурса](HTTPЗапрос.АдресРесурса_(HTTPRequest.ResourceAddress).md),
  - [Заголовки](HTTPЗапрос.Заголовки_(HTTPRequest.Headers).md),
  - в теле файла, устанавливаемом любым возможным способом..
- по тексту в [МетодHTTPЗапросаНаПроверкуРезультатаАутентификации](ШаблонНастройкиВторогоФактораАутентификации.МетодHTTPЗапросаНаПроверкуРезультатаАутентификации_(SecondAuthenticationFactorSettingTemplate.AuthenticationResultVerificationHTTPRequestMethod).md).

Зарезервированные переменные:

- secret - на место этой переменной будет подставлен секретный токен второго фактора, сгенерированной платформой. Длина токена фиксирована и равна 5.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
