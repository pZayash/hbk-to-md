ОбластьОформления (AppearanceArea)

УстановитьДоступныеПоля (SetAvailableFields)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьДоступныеПоля(<ДоступныеПоля>)

Параметры:

<ДоступныеПоля> (обязательный)

Тип: [ПоляНастройки](ПоляНастройки.md).   
Доступные поля для настройки.

Описание:

Устанавливает поля, которые возможно использовать в данной коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
