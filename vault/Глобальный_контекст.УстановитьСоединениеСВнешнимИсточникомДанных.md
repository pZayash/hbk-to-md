Глобальный контекст (Global context)

УстановитьСоединениеСВнешнимИсточникомДанных (ConnectExternalDataSource)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьСоединениеСВнешнимИсточникомДанных(<ИмяВнешнегоИсточникаДанных>)

Параметры:

<ИмяВнешнегоИсточникаДанных> (необязательный)

Тип: Строка.   
Имя внешнего источника данных, с которым нужно установить соединение.

Описание:

Устанавливает соединение с внешним источником данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
