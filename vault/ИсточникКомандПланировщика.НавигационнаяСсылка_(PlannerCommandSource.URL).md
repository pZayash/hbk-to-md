Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [НавигационнаяСсылка](ИсточникКомандПоляПланировщика.НавигационнаяСсылка_(PlannerCommandSource.URL).md)

Описание:

Тип: [ИсточникКомандПланировщика](ИсточникКомандПоляПланировщика_(PlannerCommandSource).md).   
Значение, которое устанавливается при нажатии на навигационную ссылку форматированной строки измерения планировщика, элемента измерения планировщика, метки интервала фона планировщика, метки элемента шкалы времени.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
