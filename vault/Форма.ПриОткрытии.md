Форма (Form)

ПриОткрытии (OnOpen)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриОткрытии()

Описание:

Возникает при открытии формы, до показа окна пользователю.

Доступность:

Толстый клиент.

Примечание:

В обработчике события отказаться от открытия формы нельзя.  
При программном переключении режима прокручиваемых страниц следует учитывать следующее:

- В обработчике события разрешена только установка режима прокручиваемых страниц. Снятие режима прокручиваемых страниц в этом методе игнорируется.
- При попытке открытия уже открытой формы, события [ПередОткрытием](Форма.ПередОткрытием.md) и [ПриОткрытии](Форма.ПриОткрытии.md) не возникают.

Использование в версии:

Доступен, начиная с версии 8.0.
