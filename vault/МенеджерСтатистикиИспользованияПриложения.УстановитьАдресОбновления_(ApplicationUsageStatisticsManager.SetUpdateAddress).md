МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

УстановитьАдресОбновления (SetUpdateAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьАдресОбновления(<Адрес>)

Параметры:

<Адрес> (обязательный)

Тип: [Строка](lang__def_String.md).   
Если [Неопределено](lang__def_Undefined.md), используется следующая по приоритету настройка регистрации.  
Если указанный адрес недоступен, используется последняя эффективная настройка регистрация, скаченная по адресу.  
Если указанный адрес недоступен и ранее ни разу не удалось скачать настройку, настройка регистрации будет отключена.

Описание:

Устанавливает новый адрес автоматического обновления настроек регистрации событий/экранов статистики использования приложения.  
Если метод был вызван со значением [Неопределено](lang__def_Undefined.md), адрес обновления не установлен.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
