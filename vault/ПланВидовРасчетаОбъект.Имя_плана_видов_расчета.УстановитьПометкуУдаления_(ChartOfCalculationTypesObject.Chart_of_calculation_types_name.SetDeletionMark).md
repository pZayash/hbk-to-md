# ПланВидовРасчетаОбъект.<Имя плана видов расчета>.УстановитьПометкуУдаления

ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет состояние пометки на удаление. [Истина](lang__def_BooleanTrue.md) - объект помечается на удаление, [Ложь](lang__def_BooleanFalse.md) - пометка на удаление снимается.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает/снимет пометку удаления объекта. Объект при этом не удаляется из базы данных. Окончательное удаление помеченных на удаление видов расчета происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПередЗаписью_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПометкаУдаления_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Использование в версии:

Доступен, начиная с версии 8.0.
