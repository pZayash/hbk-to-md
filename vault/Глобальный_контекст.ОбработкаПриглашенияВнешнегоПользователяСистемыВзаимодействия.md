Глобальный контекст (Global context)

ОбработкаПриглашенияВнешнегоПользователяСистемыВзаимодействия (CollaborationSystemExternalUserInvitationProcessing)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ОбработкаПриглашенияВнешнегоПользователяСистемыВзаимодействия(<Обсуждение>, <СтандартнаяОбработка>)

Параметры:

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Если команда приглашения вызвана из списка обсуждения, то в параметре передается Неопределено.

<СтандартнаяОбработка>

Тип: Булево.   
Если после вызова обработчика параметр <СтандартнаяОбработка> содержит значение Истина, отображается системный диалог приглашения внешнего участника.  
Значение по умолчанию: Истина.

Описание:

Обработчик вызывается при выборе команды "Пригласить внешнего участника".  
Если при выходе из обработчика <СтандартнаяОбработка> имеет значение Истина, то отображается системный диалог приглашения внешнего участника.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.20.
