НастройкаОтбора (FilterSettings)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все элементы управления отбором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не доступен для настройки отбора динамических списков (свойство НастройкаОтбора объектов [СправочникСписок.](СправочникСписок.Имя_справочника.md)<[Имя справочника](СправочникСписок.Имя_справочника.md)>, [ДокументСписок.](ДокументСписок.Имя_документа.md)<[Имя документа](ДокументСписок.Имя_документа.md)> и т.д.).

Использование в версии:

Доступен, начиная с версии 8.0.
