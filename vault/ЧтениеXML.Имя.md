ЧтениеXML (XMLReader)

Имя (Name)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя текущего узла XML.  
Содержит квалифицированное имя для текущего узла. Имя зависит от типа текущего узла:

- [Атрибут](ТипУзлаXML.Атрибут.md) - имя атрибута;
- [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента.md) - имя типа документа;
- [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md) - имя элемента;
- [КонецЭлемента](ТипУзлаXML.КонецЭлемента.md) - имя элемента;
- [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность.md) - имя ссылки на сущность;
- [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки.md) - адрес инструкции обработки;
- [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML.md) - строка xml.

Для узлов, для которых имя не определено, свойство содержит пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
