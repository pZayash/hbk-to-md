# ПланОбменаОбъект.<Имя плана обмена>.<Имя табличной части>

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланОбменаТабличнаяЧасть.<Имя плана обмена>.<Имя табличной части>.   
Набор свойств содержит табличные части узла. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланОбменаТабличнаяЧасть.<Имя плана обмена>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
