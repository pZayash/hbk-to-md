Расширение поля формы для поля планировщика (Form field extension for a planner)

ПриАктивизации (OnActivate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПриАктивизации()

Описание:

Возникает при активизации (выделении или снятии выделения) элементов планировщика.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
