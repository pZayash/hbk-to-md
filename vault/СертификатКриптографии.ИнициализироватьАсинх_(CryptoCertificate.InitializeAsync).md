# СертификатКриптографии.ИнициализироватьАсинх

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Криптография](Криптография-2.md) › [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md) › [Методы](СертификатКриптографии__Методы.md)

СертификатКриптографии (CryptoCertificate)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Исходные данные сертификата в виде пути к файлу или потока.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - созданный объект типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Описание:

Асинхронная функция. Начинает инициализацию объекта [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), метод [НачатьИнициализацию](СертификатКриптографии.НачатьИнициализацию_(CryptoCertificate.BeginInitialization).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
