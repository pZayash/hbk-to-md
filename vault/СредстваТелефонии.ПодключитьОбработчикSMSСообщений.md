СредстваТелефонии (TelephonyTools)

ПодключитьОбработчикSMSСообщений (AttachSMSMessageHandler)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПодключитьОбработчикSMSСообщений(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: Строка, [ОписаниеОповещения](ОписаниеОповещения.md).   
Имя процедуры, подключаемой в качестве обработчика.

Описание:

Выполняет подключение обработчика входящих SMS-сообщений.  
При получении входящего сообщения вызывается обработчик со следующими параметрами:

- Сообщение: [SMSСообщение](SMSСообщение.md),
- Дополнительный параметр (если используется [ОписаниеОповещения](ОписаниеОповещения.md)).

Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяПриемSMS](СредстваТелефонии.ПоддерживаетсяПриемSMS.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Если функциональность не поддерживается, будет выполнен возврат из метода без вызова обработчика.

См. также:

[СредстваТелефонии](СредстваТелефонии.md), метод [ПоддерживаетсяПриемSMS](СредстваТелефонии.ПоддерживаетсяПриемSMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
