Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfileVirtualDirectory (CreateSecurityProfileVirtualDirectory)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfileVirtualDirectory()

Возвращаемое значение:

Тип: [Виртуальный каталог](Виртуальный_каталог.md).   

Описание:

Создает новый виртуальный каталог для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
