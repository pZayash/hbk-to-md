ДиаграммаГанта (GanttChart)

РазвернутаСерия (SeriesExpanded)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутаСерия(<Серия>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md).   
Серия диаграммы Ганта.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - серия развернута; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, развернута ли серия диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
