РежимОткрытияОкнаФормы (FormWindowOpeningMode)

Независимый (Independent)

Доступен, начиная с версии 8.2.

Описание:

Обычный режим открытия формы, при котором возможны действия пользователя в окне-владельце до закрытия формы:

- В интерфейсе "В отдельных окнах" открывается отдельное окно.
- В интерфейсах "В закладках" и "Такси" форма открывается в рабочей области основного окна.

Использование в версии:

Доступен, начиная с версии 8.2.
