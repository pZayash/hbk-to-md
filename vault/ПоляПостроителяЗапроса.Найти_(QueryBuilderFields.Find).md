ПоляПостроителяЗапроса (QueryBuilderFields)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого поля.

Возвращаемое значение:

Тип: [ПолеПостроителяЗапроса](ПолеПостроителяЗапроса_(QueryBuilderField).md), [Неопределено](lang__def_Undefined.md).   
Если поле не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск поля по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
