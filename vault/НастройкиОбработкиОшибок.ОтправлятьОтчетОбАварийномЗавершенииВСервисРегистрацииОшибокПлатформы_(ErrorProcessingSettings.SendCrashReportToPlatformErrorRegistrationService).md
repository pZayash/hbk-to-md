НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОтчетОбАварийномЗавершенииВСервисРегистрацииОшибокПлатформы (SendCrashReportToPlatformErrorRegistrationService)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ОтправлятьОтчетОбАварийномЗавершенииНаКлиентеВСервисРегистрацииОшибокПлатформы](НастройкиОбработкиОшибок.ОтправлятьОтчетОбАварийномЗавершенииНаКлиентеВСервисРегистрацииОшибокПлатформы_(ErrorProcessingSettings.SendCrashReportOnClientToPlatformErrorRegistrationService).md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость отправки информации об аварийном завершении в техподдержку фирмы "1С".

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
