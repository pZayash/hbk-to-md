АдминистрированиеСеанс (AdministrationSession)

НомерСеанса (SessionNumber)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер сеанса. Целое число, уникальное среди всех сеансов данной информационной базы.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
