КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Добавить(<Значение>, <ТипЭлемента>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Элемент, добавляемый в коллекцию.  
В зависимости от параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.  
Если значение параметра [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), то в коллекцию будет добавлено строковое представление переданного значения.  
Если значение параметра [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md), то тип параметра <Значение> должен быть [Картинка](Картинка.md), [ДвоичныеДанные](ДвоичныеДанные.md) или Строка, содержащая URL картинки. В противном случае вызывается исключение.   
Если значение параметра [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), то значение параметра игнорируется и в коллекцию добавляется перевод строки.  
Если значение параметров Неопределено, а также коллекция элементов форматированного документа содержит только параграфы, то в коллекцию добавляется пустой параграф.  
Значение по умолчанию: Неопределено.

<ТипЭлемента> (необязательный)

Тип: Тип.   
Тип элемента, который нужно добавить в коллекцию. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Элементами коллекции могут быть:  
1) [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md)  
2) [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md)  
3) [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md)  
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Значение по умолчанию: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md).

Возвращаемое значение:

Тип: [ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента.md), [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md).   
Возвращает добавленный элемент.

Описание:

Добавляет элемент в конец коллекции элементов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Добавление картинки доступно только на сервере.  
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.21.
