# АнализДанныхПоискАссоциаций

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md) › [Поиск ассоциаций](Поиск_ассоциаций.md)

Доступен, начиная с версии 8.0.

Описание:

Тип анализа данных - поиск ассоциаций.  
Параметры:

- МинимальныйПроцентСлучаев (MinSupport) - минимальный процент случаев, в которых наблюдается группа элементов (Число);
- МинимальнаяДостоверность (MinConfidence) - минимальная достоверность правила (Число);
- МинимальнаяЗначимость (MinImportance) - минимальная значимость правила (Число);
- ПоискПоИерархии (FindInHierarchy) - необходимость поиска по иерархии (Булево);
- ТипОтсеченияПравил (PruneRulesType) - тип отсечения найденных правил (ТипОтсеченияПравилАссоциации);
- ТипИсточникаДанных (DataSourceType) - тип источника данных (ТипИсточникаДанныхПоискаАссоциаций);
- ИспользованиеЧисловыхЗначений (NumericValueUse) - как интерпретировать числовые значения ТипИспользованияЧисловыхЗначенийАнализаДанных);
- ИгнорироватьНезаполненныеЗначения (IgnoreIncompleteValues) - как использовать пустые значения (Булево);
- Порядок (Order) - порядок, в котором выводить найденные правила. (Перечисление ТипУпорядочиванияПравилАссоциацииАнализаДанных).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
