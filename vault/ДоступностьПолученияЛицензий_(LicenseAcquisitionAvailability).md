Описание:

Содержит данные о возможности записи файла программной лицензии в каталог лицензий.  
Объект может быть получен только с помощью метода [ПолучитьДоступностьПолученияЛицензии](МенеджерПолученияЛицензий.ПолучитьДоступностьПолученияЛицензии_(LicenseAcquisitionManager.GetLicenseAcquisitionAvailability).md).

Доступность:

Тонкий клиент, сервер, внешнее соединение.   
Сериализуется.

См. также:

[МенеджерПолученияЛицензий](МенеджерПолученияЛицензий_(LicenseAcquisitionManager).md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
