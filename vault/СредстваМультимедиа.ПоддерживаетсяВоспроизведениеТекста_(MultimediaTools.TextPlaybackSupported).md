СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяВоспроизведениеТекста (TextPlaybackSupported)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПоддерживаетсяВоспроизведениеТекста(<Язык>)

Параметры:

<Язык> (необязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Язык](ОбъектМетаданных_Язык_(MetadataObject_Language).md).   
Идентификатор языка согласно стандарту BCP 47 (например, "en-US" для английского или "ru-RU" для русского). Если не указан, то проверяется возможность произнести текст на текущем языке операционной системы.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md), если на устройстве установлен голосовой пакет для запрашиваемого языка.

Описание:

Определяет, поддерживается ли воспроизведение текста на указанном языке.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ВоспроизвестиТекст](СредстваМультимедиа.ВоспроизвестиТекст_(MultimediaTools.PlayText).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
