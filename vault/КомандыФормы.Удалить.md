КомандыФормы (FormCommands)

Удалить (Delete)

Доступен, начиная с версии 8.2.

Синтаксис:

Удалить(<Команда>)

Параметры:

<Команда> (обязательный)

Тип: [КомандаФормы](КомандаФормы.md).   
Удаляемая команда.

Описание:

Удаляет команду из коллекции.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступно удаление только ранее программно добавленного элемента.

Использование в версии:

Доступен, начиная с версии 8.2.
