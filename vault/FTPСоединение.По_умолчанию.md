FTPСоединение (FTPConnection)

По указанному серверу

Доступен, начиная с версии 8.0.

Синтаксис:

Новый FTPСоединение(<Сервер>, <Порт>, <ИмяПользователя>, <ПарольПользователя>, <Прокси>, <ПассивноеСоединение>, <Таймаут>, <ЗащищенноеСоединение>, <УровеньИспользованияЗащищенногоСоединения>)

Параметры:

<Сервер> (обязательный)

Тип: [Строка](lang__def_String.md).   
Сервер, с которым осуществляется соединение.

<Порт> (необязательный)

Тип: [Число](lang__def_Number.md).   
Порт сервера, с которым осуществляется соединение.  
Значение по умолчанию для FTP соединений равно 21, для защищенных FTPS соединений - 990.  
Значение по умолчанию: порт по умолчанию для используемого протокола.

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя на указанном сервере.

<ПарольПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя на указанном сервере.

<Прокси> (необязательный)

Тип: [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md).   
Прокси, используемый для соединения с сервером.  
Объект [FTPСоединение](FTPСоединение_(FTPConnection).md) должен использоваться только с механизмом "FTP over HTTP".  
Если при создании объекта использован прокси-сервер типа native FTP, то при обращении к нему будет выдана ошибка.

<ПассивноеСоединение> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет тип ftp-соединения.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Таймаут в секундах на установку FTP соединения и выполнение FTP операций. 0 - не устанавливать таймаут.  
Значение по умолчанию: 0.

<ЗащищенноеСоединение> (необязательный)

Тип: [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md), [Неопределено](lang__def_Undefined.md).   
Объект защищенного соединения для осуществления FTPS-соединения, если указан.  
Если параметр не задан, выполняется попытка подключения к серверу FTP или FTPES в незащищенном режиме. При этом, если сервер требует установки защищенного режима, будет сгенерирована ошибка.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<УровеньИспользованияЗащищенногоСоединения> (необязательный)

Тип: [УровеньИспользованияЗащищенногоСоединенияFTP](УровеньИспользованияЗащищенногоСоединенияFTP_(FTPSecureConnectionUsageLevel).md).   
Определяет требования по использованию защищенного SSL-соединения.  
Значение по умолчанию: [Авто](УровеньИспользованияЗащищенногоСоединенияFTP.Авто_(FTPSecureConnectionUsageLevel.Auto).md).

Описание:

Создает объект [FTPСоединение](FTPСоединение_(FTPConnection).md).

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.24.
