# ИнтернетПочта.УдалитьСообщения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Почта](Почта-2.md) › [ИнтернетПочта](ИнтернетПочта.md) › [ИнтернетПочта](ИнтернетПочта_(InternetMail).md) › [Методы](ИнтернетПочта__Методы.md)

ИнтернетПочта (InternetMail)

УдалитьСообщения (DeleteMessages)

Доступен, начиная с версии 8.0.

Синтаксис:

УдалитьСообщения(<МассивУдаляемыхДанных>)

Параметры:

<МассивУдаляемыхДанных> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий либо заголовки сообщений, либо серверные идентификаторы сообщений, которые необходимо удалить с сервера.  
Для варианта работы с [IMAP](ПротоколИнтернетПочты.IMAP_(InternetMailProtocol.IMAP).md) протоколом также допускается передать массив порядковых номеров сообщений (целые числа) в текущем почтовом ящике ([ТекущийПочтовыйЯщик](ИнтернетПочта.ТекущийПочтовыйЯщик_(InternetMail.CurrentMailbox).md)).

Описание:

При работе с POP3 удаляет с сервера все сообщения, указанные либо объектами [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md), либо идентификаторами, находящимися в массиве, принимаемом в качестве параметра.  
При работе с IMAP помечает как удаленные на сервере все сообщения, указанные одним из следующих способов:

- объектами [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md),
- идентификаторами, находящимися в массиве, принимаемом в качестве параметра,
- порядковыми номерами сообщений в текущем почтовом ящике, заданном свойством [ТекущийПочтовыйЯщик](ИнтернетПочта.ТекущийПочтовыйЯщик_(InternetMail.CurrentMailbox).md).

Помеченные сообщения можно окончательно удалить методом [ОчиститьУдаленныеСообщения](ИнтернетПочта.ОчиститьУдаленныеСообщения_(InternetMail.ClearDeletedMassages).md) или снять отметку удаления с помощью метода [ОтменитьУдалениеСообщений](ИнтернетПочта.ОтменитьУдалениеСообщений_(InternetMail.UndeleteMessages).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
