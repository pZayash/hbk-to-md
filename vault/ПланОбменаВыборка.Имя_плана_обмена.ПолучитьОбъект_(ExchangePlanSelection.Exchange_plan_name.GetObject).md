ПланОбменаВыборка.<Имя плана обмена> (ExchangePlanSelection.<Exchange plan name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для модификации и записи узла, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = Выборка.ПолучитьОбъект(); |

Использование в версии:

Доступен, начиная с версии 8.0.
