Описание:

Определяет положение подписей в диаграмме. Для гистограмм с накоплением и нормированных гистограмм всегда используется вариант [Центр](ПоложениеПодписейКДиаграмме.Центр_(ChartLabelLocation.Center).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartLabelLocation.
