Расширение табличного поля дерева значений (Value tree table box extension)

ПередРазворачиванием (BeforeExpand)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередРазворачиванием(<Строка>, <Отказ>)

Параметры:

<Строка>

Строка табличного поля.

<Отказ>

Тип: Булево.   
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение Истина, то стандартной обработки события не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед раскрытием узла табличного поля, отображаемого в виде дерева.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
