WebSocketКлиент (WebSocketClient)

ПодключатьАвтоматически (AutoConnect)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, что подключение к серверу надо выполнять автоматически, при запуске системы "1С:Предприятие" и переустанавливать соединение при разрыве связи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
