РежимИспользованияТабличныхПространствБазыДанных (DatabaseTablespacesUseMode)

Использовать (Use)

Доступен, начиная с версии 8.3.23.

Описание:

Использование табличных пространств разрешено.  
При использовании режима совместимости конфигурации выше [Версия8\_3\_22](РежимСовместимости.Версия8_3_22_(CompatibilityMode.Version8_3_22).md) используется всегда.
