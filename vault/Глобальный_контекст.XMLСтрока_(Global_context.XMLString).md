Глобальный контекст (Global context)

XMLСтрока (XMLString)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLСтрока(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Неопределено](lang__def_Undefined.md), [Null](lang__def_Null.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md); значения перечислений, значения системных перечислений ([ВидСчета](ВидСчета_(AccountType).md), [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии_(AccountingRecordType).md), [ВидДвиженияНакопления](ВидДвиженияНакопления_(AccumulationRecordType).md), [ДопустимыйЗнак](ДопустимыйЗнак_(AllowedSign).md), [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md), [ЧастиДаты](ЧастиДаты_(DateFractions).md)) или все ссылки на объекты базы данных.   
Значение, для которого должно быть получено строковое представление, которое может быть использовано как текст элемента или значение атрибута XML.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает XML представление значения для помещения в текст элемента или значение атрибута XML.  
XML представление зависит от типа значения:

- [Строка](lang__def_String.md) - как есть;
- [Булево](lang__Булево_(Boolean).md): [Истина](lang__def_BooleanTrue.md) - как "true", [Ложь](lang__def_BooleanFalse.md) - как "false";
- [Дата](lang__def_Date.md) - XML представление соответствует представлению типа dateTime схемы XML (см. <http://www.w3.org/TR/xmlschema-2/#dateTime>);
- [Число](lang__def_Number.md) - XML представление соответствует представлению типа decimal схемы XML (см. <http://www.w3.org/TR/xmlschema-2/#decimal>);
- Значение системного перечисления (для перечислений, поддерживающих XML сериализацию) - как имя значения перечисления;
- [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md) - как строковое представление уникального идентификатора;
- Ссылки - как XML представление уникального идентификатора ссылки;
- [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) - как строка в формате base64.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьXML = Новый ЗаписьXML; ЗаписьXML.ЗаписатьНачалоЭлемента("payment"); ЗаписьXML.ЗаписатьНачалоЭлемента("date"); ЗаписьXML.ЗаписатьТекст(XMLСтрока(ДатаПлатежа)); ЗаписьXML.ЗаписатьКонецЭлемента(); ЗаписьXML.ЗаписатьНачалоЭлемента("amount"); ЗаписьXML.ЗаписатьТекст(XMLСтрока(РазмерПлатежа)); ЗаписьXML.ЗаписатьКонецЭлемента(); ЗаписьXML.ЗаписатьКонецЭлемента(); |
