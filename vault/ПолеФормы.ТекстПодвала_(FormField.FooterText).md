# ПолеФормы.ТекстПодвала

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [ПолеФормы](ПолеФормы_(FormField).md) › [Свойства](ПолеФормы__Свойства.md)

ПолеФормы (FormField)

ТекстПодвала (FooterText)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст подвала колонки. Данное свойство имеет смысл только для полей, расположенных в таблице (представляющих собой колонки).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
