РежимОбработкиПодкаталоговZIP (ZIPSubDirProcessingMode)

НеОбрабатывать (DontProcess)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [НеОбрабатывать](РежимОбработкиПодкаталоговФайлаАрхива.НеОбрабатывать_(ArchiveFileSubDirProcessingMode.DontProcess).md)

Доступен, начиная с версии 8.0.

Описание:

При задании маски в качестве имени для архивации подкаталоги не будут обработаны.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
