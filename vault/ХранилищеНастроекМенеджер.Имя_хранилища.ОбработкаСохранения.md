ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ОбработкаСохранения (SaveProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаСохранения(<КлючОбъекта>, <КлючНастроек>, <Настройки>, <ОписаниеНастроек>, <Пользователь>)

Параметры:

<КлючОбъекта>

Тип: Произвольный.   
Ключ объекта, для которого сохраняются настройки.

<КлючНастроек>

Тип: Произвольный.   
Ключ настроек.

<Настройки>

Тип: Произвольный.   
Настройки, которые нужно сохранить.

<ОписаниеНастроек>

Тип: [ОписаниеНастроек](ОписаниеНастроек.md), Неопределено.   
Описание настроек.  
Неопределено - описание настроек сохранять не нужно.

<Пользователь>

Тип: Строка.   
Идентификатор пользователя, для которого сохраняются настройки.  
Если представляет пустую строку, то будут сохраняться настройки для текущего пользователя.

Описание:

Возникает при сохранении настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
