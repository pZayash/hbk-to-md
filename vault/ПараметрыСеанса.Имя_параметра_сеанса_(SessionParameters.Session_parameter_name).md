ПараметрыСеанса (SessionParameters)

<Имя параметра сеанса> (<Session parameter name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения параметров сеанса. Доступ к значению осуществляется по имени. Имена свойств совпадают с именами параметров сеанса, как они заданы в Конфигураторе.  
Позволяет получить или установить значение одноименного параметра сеанса. Типы, допустимые для значения параметра сеанса, определяются при создании параметра сеанса в Конфигураторе. Среди допустимых могут быть следующие типы:

- [Число](lang__def_Number.md);
- [Строка](lang__def_String.md);
- [Дата](lang__def_Date.md);
- [Булево](lang__Булево_(Boolean).md);
- [Неопределено](lang__def_Undefined.md);
- [Null](lang__def_Null.md);
- [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md);
- [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md);
- [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md);
- [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md);
- [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md);
- [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md);
- [ВидСчета](ВидСчета_(AccountType).md);
- [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии_(AccountingRecordType).md);
- [ВидДвиженияНакопления](ВидДвиженияНакопления_(AccumulationRecordType).md);
- [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md);
- [ВидДвиженияНакопления](ВидДвиженияНакопления_(AccumulationRecordType).md);
- [ВидСчета](ВидСчета_(AccountType).md);
- [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии_(AccountingRecordType).md);
- Любая ссылка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При попытке обращения к неинициализированному параметру сеанса (в течении сеанса не происходило присвоение параметра) генерируется исключение.  
Элементами значений типов [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md), [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md) могут быть значения только перечисленных типов.
