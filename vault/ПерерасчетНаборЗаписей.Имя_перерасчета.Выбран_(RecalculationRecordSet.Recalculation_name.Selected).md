# ПерерасчетНаборЗаписей.<Имя перерасчета>.Выбран

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры расчета](Регистры_расчета.md) › [Перерасчеты](Перерасчеты.md) › [ПерерасчетНаборЗаписей.<Имя перерасчета>](ПерерасчетНаборЗаписей.Имя_перерасчета_(RecalculationRecordSet.Recalculation_name).md) › [Методы](ПерерасчетНаборЗаписей.Имя_перерасчета__Методы.md)

ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Выбран (Selected)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбран()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - набор не изменялся; [Ложь](lang__def_BooleanFalse.md) - изменялся, в частности, если набор не считывался и не записывался, а также, если записывался с добавлением записей.

Описание:

Определяет, считан ли набор записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При изменении отбора считается, что набор перестает быть считанным.

Использование в версии:

Доступен, начиная с версии 8.0.
