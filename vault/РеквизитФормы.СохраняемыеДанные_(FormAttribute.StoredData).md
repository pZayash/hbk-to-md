РеквизитФормы (FormAttribute)

СохраняемыеДанные (StoredData)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - указывает, что это сохраняемый при записи реквизит.  
Соответственно, система будет устанавливать модифицированность и выполнять блокировку при интерактивном изменении этих данных.  
Если [Истина](lang__def_BooleanTrue.md) и значение изменено, будет выведен вопрос о необходимости сохранения.   
Используется только для реквизитов верхнего уровня.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
