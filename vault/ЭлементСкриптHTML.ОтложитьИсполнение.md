ЭлементСкриптHTML (HTMLScriptElement)

ОтложитьИсполнение (Defer)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает HTML-документу, что не нужно дожидаться загрузки скрипта для продолжения загрузки и обработки оставшейся части страницы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
