Описание:

Данный объект представляет собой коллекцию объектов, описывающих оформление диаграммы.  
Для оформления ячейки таблицы используются следующие элементы оформления:

- ТипДиаграммы (ChartType) - тип диаграммы (тип [ТипДиаграммы](ТипДиаграммы.md));
- ТипДиаграммы.АвтоРаздвижениеСерий (ChartType.AutoSeriesSeparation) - режим раздвижения (тип [АвтоРаздвижениеСерий](АвтоРаздвижениеСерий.md));
- ТипДиаграммы.БазовоеЗначение (ChartType.BaseValue) – базовое значение (тип Число);
- ТипДиаграммы.ВидПодписей (ChartType.LabelType) - состав подписей (тип [ВидПодписейКДиаграмме](ВидПодписейКДиаграмме.md));
- ТипДиаграммы.ВлияниеРазмераНаПузырек (ChartType.ChartBubbleSizing) - определяет зависимость размера пузырька от значения (тип [ВлияниеРазмераНаПузырекДиаграммы](ВлияниеРазмераНаПузырекДиаграммы.md));
- ТипДиаграммы.ВнутреннийРадиусКольцевойДиаграммы (ChartType.DonutChartInnerRadius) - настройка размера внутреннего радиуса кольцевой диаграммы (тип Число).
- ТипДиаграммы.ГлубинаОбъемнойДиаграммы (ChartType.Chart3DDepth) - глубина объемной диаграммы. Используется в круговой, кольцевой и объемных гистограммах (тип Число);
- ТипДиаграммы.Градиент (ChartType.Gradient) – градиент (тип Булево);
- ТипДиаграммы.ДополнительнаяОсьЗначений (ChartType.AdditionalValuesAxis) - настройка дополнительной оси значений (тип [ОсьДиаграммы](ОсьДиаграммы.md));
- ТипДиаграммы.ДополнительнаяШкалаЗначений (ChartType.AdditionalValuesScale) - настройка дополнительной шкалы значений (тип [ШкалаДиаграммы](ШкалаДиаграммы.md));
- ТипДиаграммы.ЗаполнениеПодсказкиЗначений (ChartType.ValuesToolTipFillType) - настройка режима заполнения всплывающей подсказки диаграммы (тип [ЗаполнениеПодсказкиЗначенийДиаграммы](ЗаполнениеПодсказкиЗначенийДиаграммы.md));
- ТипДиаграммы.ИнформационныеИнтервалыЗначений (ChartType.ValuesReferenceBands) - информационные интервалы значений в диаграмме (тип [ИнформационныеИнтервалыДиаграммы](ИнформационныеИнтервалыДиаграммы.md));
- ТипДиаграммы.ИнформационныеИнтервалыТочек (ChartType.PointsReferenceBands) - информационные интервалы точек в диаграмме (тип [ИнформационныеИнтервалыДиаграммы](ИнформационныеИнтервалыДиаграммы.md));
- ТипДиаграммы.ИнформационныеЛинииЗначений (ChartType.ValuesReferenceLines) - информационные линии значений в диаграмме (тип [ИнформационныеЛинииДиаграммы](ИнформационныеЛинииДиаграммы.md));
- ТипДиаграммы.ИнформационныеЛинииТочек (ChartType.PointsReferenceLines) - информационные линии точек в диаграмме (тип [ИнформационныеЛинииДиаграммы](ИнформационныеЛинииДиаграммы.md));
- ТипДиаграммы.ИспользованиеНечисловыхЗначений (ChartType.NonnumericValuesUse) - настройка использования значений, которые не приводятся к числу (тип [ИспользованиеНечисловыхЗначенийДиаграммы](ИспользованиеНечисловыхЗначенийДиаграммы.md));
- ТипДиаграммы.ИсточникЗначенийОсиТочек (ChartType.ChartPointsAxisValuesSource) - определяет вид значений на оси точек (тип [ИсточникЗначенийОсиТочекДиаграммы](ИсточникЗначенийОсиТочекДиаграммы.md));
- ТипДиаграммы.ИсточникЗначенияРазмераПузырька (ChartType.BubbleSizeValueSource) - источник значения размера пузырьков (тип [ИсточникЗначенияРазмераПузырькаДиаграммы](ИсточникЗначенияРазмераПузырькаДиаграммы.md));
- ТипДиаграммы.КлючСохраненияЦветов (ChartType.SaveColorsKey) - ключ, по которому в системном хранилище настроек лежит [СохранениеОформленияКомпоновкиДанных](СохранениеОформленияКомпоновкиДанных.md). Если по ключу ничего нет, то создает новую запись. Если по ключу лежит объект другого типа, то параметр ТипДиаграммы.КлючСохраненияЦветов игнорируется (тип Строка).
- ТипДиаграммы.КонечныйУгол (ChartType.EndAngle) - угол конца отображения секторов. Используется в круговой и кольцевой диагарммах (тип Число);
- ТипДиаграммы.ЛинииСоединенияЗначенийПоСериям (ChartType.ValuesBySeriesConnectionLines) - стиль и толщина линий соединений (тип [Линия](Линия.md));
- ТипДиаграммы.ЛинииШкал (ChartType.ScaleLines) - стиль линий шкалы (тип [Линия](Линия.md));
- ТипДиаграммы.МаксимальноеЗначение (ChartType.MaxValue) - максимальное значение (тип Число);
- ТипДиаграммы.МаксимальныйРазмерПузырька (ChartType.MaxBubbleSize) - максимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения) если для размера пузырьков указана серия значений (тип Число);
- ТипДиаграммы.МинимальноеЗначение (ChartType.MinValue) - минимальное значение (тип Число);
- ТипДиаграммы.МинимальныйРазмерПузырька (ChartType.MinBubbleSize) - минимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения), если для размера пузырьков указана серия значений (тип Число);
- ТипДиаграммы.НатяжениеСглаживания (ChartType.SplineStrain) - целочисленное значение определяет "натяжение" кривой для режима сглаживания [ГладкаяКривая](РежимСглаживанияДиаграммы.ГладкаяКривая.md);
- ТипДиаграммы.НачальныйУгол (ChartType.BeginAngle) - угол начала отображения секторов по часовой стрелке от вертикальной оси. Используется в круговой и кольцевой диагарммах (тип Число);
- ТипДиаграммы.ОбщаяСерияРазмераПузырька (ChartType.BubbleSizeCommonSeries) - серия, значения которой будут использоваться для размеров пузырьков для всех серий по оси ординат (тип Произвольн);
- ТипДиаграммы.Окантовка (ChartType.Outline) – окантовка (тип Булево);
- ТипДиаграммы.ОписаниеПалитрыЦветов (ChartType.ColorPaletteDescription) - настройка палитры цветов серий в диаграмме (тип [ОписаниеПалитрыЦветовДиаграммы](ОписаниеПалитрыЦветовДиаграммы.md));
- ТипДиаграммы.ОписаниеПалитрыЦветовИнформационныхИнтервалов (ChartType.ReferenceBandsColorPaletteDescription) - настройка палитры цветов для информационных интервалов (тип ОписаниеПалитрыЦветовДиаграммы)
- ТипДиаграммы.ОсьЗначений (ChartType.ValuesAxis) - настройки оси значений в диаграмме (тип [ОсьДиаграммы](ОсьДиаграммы.md));
- ТипДиаграммы.ОсьТочек (ChartType.PointsAxis) - настройки оси точек в диаграмме (тип [ОсьДиаграммы](ОсьДиаграммы.md));
- ТипДиаграммы.ОтображатьТаблицуДанных (ChartType.ShowDataTable) - отображать таблицу данных (тип Булево);
- ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииЗначений (ChartType.ShowValuesPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале значений при показе подсказки (тип [ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы](ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы.md));
- ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииТочек (ChartType.ShowPointsPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале точек при показе подсказки диаграммы (тип [ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы](ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы.md));
- ТипДиаграммы.ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы (ChartType.BubbleChartNegativeValuesShowMode) - указывает вариант работы с отрицательными значениями размера (тип [ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы](ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы.md));
- ТипДиаграммы.ПорядокОтображенияТочекГоризонтальнойГистограммы (ChartType.BarChartPointsOrder) - порядок вывода точек в горизонтальных диаграммах (тип [ПорядокОтображенияТочекГоризонтальнойГистограммы](ПорядокОтображенияТочекГоризонтальнойГистограммы.md));
- ТипДиаграммы.ПорядокСерийВЛегенде (ChartType.SeriesOrderInLegend) - определяет порядок отображения серий в легенде диаграммы (тип [ПорядокСерийВЛегендеДиаграммы](ПорядокСерийВЛегендеДиаграммы.md));
- ТипДиаграммы.ПропускатьБазовоеЗначение (ChartType.HideBaseValue) - пропускать точки с базовым значением (тип Булево);
- ТипДиаграммы.РазмерПузырькаПоУмолчанию (ChartType.DefaultBubbleSize) - указывает базовый диаметр пузырька в процентах от минимума из ширины и высоты области построения (тип Число);
- ТипДиаграммы.РасположениеЛегенды (ChartType.LegendPlacement) - расположение легенды в диаграммы (тип [РасположениеЛегендыДиаграммыКомпоновкиДанных](РасположениеЛегендыДиаграммыКомпоновкиДанных.md));
- ТипДиаграммы.РасположениеРесурсов (ChartType.ResourcesPlacement) - расположение ресурсов в диаграмме (тип [РасположениеРесурсовВДиаграммеКомпоновкиДанных](РасположениеРесурсовВДиаграммеКомпоновкиДанных.md));
- ТипДиаграммы.РежимВыделения (ChartType.SelectionMode) - настройка режима выделения диаграммы (тип [РежимВыделенияДиаграммы](РежимВыделенияДиаграммы.md));
- ТипДиаграммы.РежимОтображенияПодсказкиЗначений (ChartType.ValuesToolTipShowMode) - настройка режима заполнения всплывающей подсказки диаграммы (тип [ЗаполнениеПодсказкиЗначенийДиаграммы](ЗаполнениеПодсказкиЗначенийДиаграммы.md));
- ТипДиаграммы.РежимПолупрозрачности (ChartType.SemitransparencyMode) - определяет режим отображения для перекрывающихся элементов диаграмм (тип [РежимПолупрозрачностиДиаграммы](РежимПолупрозрачностиДиаграммы.md));
- ТипДиаграммы.РежимСглаживания (ChartType.SplineMode) - определяет режим сглаживания линий в диаграммах "непрерывного" вида (тип [РежимСглаживанияДиаграммы](РежимСглаживанияДиаграммы.md));
- ТипДиаграммы.СерияНаОсиТочек (ChartType.PointsAxisSeries) - серия, значения которой откладываются на оси точек (тип Произвольный);
- ТипДиаграммы.СоединениеЗначенийПоСериям (ChartType.ValuesBySeriesConnection) - тип соединения значений по сериям в гистограммах с накоплением (тип [ТипСоединенияЗначенийПоСериямДиаграммы](ТипСоединенияЗначенийПоСериямДиаграммы.md));
- ТипДиаграммы.СоединениеТочек (ChartType.PointsConnection) - соединять точки линиями для диаграмм, в которых это имеет смысл ([График](ТипДиаграммы.График.md), [РадарныйГрафик](ТипДиаграммы.РадарныйГрафик.md), [Точечная](ТипДиаграммы.Точечная.md), [Пузырьковая](ТипДиаграммы.Пузырьковая.md)) (тип [ТипСоединенияТочекДиаграммы](ТипСоединенияТочекДиаграммы.md));
- ТипДиаграммы.СоединениеТочекПриПропущенныхЗначениях (ChartType.PointConnectionAcrossSkippedValues) - настройка режима отрисовки при пропущенных значениях (тип [ТипСоединенияТочекПриПропущенныхЗначенияхДиаграммы](ТипСоединенияТочекПриПропущенныхЗначенияхДиаграммы.md)).
- ТипДиаграммы.СохранениеЦветов (ChartType.SaveColors) - настройка варианта сохранения цветов (тип ([СохранениеОформленияКомпоновкиДанных](СохранениеОформленияКомпоновкиДанных.md));
- ТипДиаграммы.СтильЛинии (ChartType.BorderStyle) - стиль линии вокруг диаграммы (тип [Линия](Линия.md));
- ТипДиаграммы.ФорматВТаблицеДанных (ChartType.DataTableFormat) - формат значений в таблице данных (тип Строка);
- ТипДиаграммы.ФорматЗначенийВПодписях (ChartType.ValueLabelFormat) - формат представления значений в подписях к диаграмме (тип Строка);
- ТипДиаграммы.ЦветСоединенияЗначенийПоСериям (ChartType.ValuesBySeriesConnectionColor) – цвет линий соединений (тип [Цвет](Цвет.md));
- ТипДиаграммы.ЦветФона (ChartType.BackColor) - цвет фона диаграммы (тип [Цвет](Цвет.md));
- ТипДиаграммы.ЦветШкал (ChartType.ScaleColor) - цвет линий шкалы (тип [Цвет](Цвет.md));
- ТипДиаграммы.ШкалаЗначений (ChartType.ValuesScale) - настройка шкалы значений (тип [ШкалаДиаграммы](ШкалаДиаграммы.md));
- ТипДиаграммы.ШкалаСерий (ChartType.SeriesScale) - настройка шкалы серий (тип [ШкалаДиаграммы](ШкалаДиаграммы.md));
- ТипДиаграммы.ШкалаТочек (ChartType.PointsScale) - настройка шкалы точек (тип [ШкалаДиаграммы](ШкалаДиаграммы.md));
- ТипДиаграммы.Шрифт (ChartType.Font) - шрифт диаграммы (тип [Шрифт](Шрифт.md));

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: ChartAppearance.
