ДиаграммаГанта (GanttChart)

СвернутьТочку (CollapsePoint)

Доступен, начиная с версии 8.0.

Синтаксис:

СвернутьТочку(<Точка>, <СПодчиненными>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта.md).   
Точка диаграммы Ганта.

<СПодчиненными> (необязательный)

Тип: Булево.   
Определяет необходимость сворачивания точки вместе с подчиненными точками. Истина - свернуть вместе с подчиненными.  
Значение по умолчанию: Ложь.

Описание:

Сворачивает точки, возможно вместе с подчиненными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
