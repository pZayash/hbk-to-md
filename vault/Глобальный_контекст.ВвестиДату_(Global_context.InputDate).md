Глобальный контекст (Global context)

ВвестиДату (InputDate)

Доступен, начиная с версии 8.0.

Синтаксис:

ВвестиДату(<Дата>, <Подсказка>, <ЧастьДаты>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Имя доступной в модуле переменной. В эту переменную будет помещено введенное значение даты. Начальное значение переменной будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода даты. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<ЧастьДаты> (необязательный)

Тип: [ЧастиДаты](ЧастиДаты_(DateFractions).md).   
Вводимая в диалоге часть (или части) даты.  
Значение по умолчанию: [ДатаВремя](ЧастиДаты.ДатаВремя_(DateFractions.DateTime).md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - дата введена; [Ложь](lang__def_BooleanFalse.md) - пользователь отказался от ввода даты.

Описание:

Вызывает диалог для ввода даты.  
При вызове метода в веб-клиенте генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [ПоказатьВводДаты](Глобальный_контекст.ПоказатьВводДаты_(Global_context.ShowInputDate).md).

Пример:

|  |
| --- |
| ДатаНапоминания = РабочаяДата; Подсказка = "Введите дату и время"; ЧастьДаты = ЧастиДаты.ДатаВремя; Если ВвестиДату(ДатаНапоминания, Подсказка, ЧастьДаты) Тогда     // запомнить дату напоминания  КонецЕсли; |
