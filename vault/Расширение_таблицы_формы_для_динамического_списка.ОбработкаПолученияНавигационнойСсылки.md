Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ОбработкаПолученияНавигационнойСсылки (URLGetProcessing)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ОбработкаПолученияНавигационнойСсылки(<Элемент>, <Ключ>, <Представление>, <НавигационнаяСсылка>, <СтандартнаяОбработка>)

Параметры:

<Элемент>

Тип: [ТаблицаФормы](ТаблицаФормы.md).   
Таблица, для которой выполняется получение списка навигационных ссылок.

<Ключ>

Тип: Произвольный.   
В этом параметре передается значение свойства <Ключ> выбранного элемента списка ссылок, полученного из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](Расширение_таблицы_формы_для_динамического_списка.ОбработкаПолученияСпискаНавигационныхСсылок.md). Если <СписокНавигационыхСсылок>, полученный из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](Расширение_таблицы_формы_для_динамического_списка.ОбработкаПолученияСпискаНавигационныхСсылок.md), пуст, то <Ключ> содержит значение Неопределено.

<Представление>

Тип: Строка.   
Представление ссылки, полученное из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](Расширение_таблицы_формы_для_динамического_списка.ОбработкаПолученияСпискаНавигационныхСсылок.md) или введенное пользователем.

<НавигационнаяСсылка>

Тип: Строка.   
Параметр, в который требуется поместить ссылку.  
Полученная ссылка будет обработана (недопустимые символы будут закодированы) и в дальнейшем может быть использована для перехода по ней, копирования в буфер, помещения в избранное и т.д.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события.  
Параметр может иметь следующие значения:

- Истина - если в параметре <Ключ> используется один из стандартных вариантов;
- Ложь - если в параметре <Ключ> один из вариантов, добавленных в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](Расширение_таблицы_формы_для_динамического_списка.ОбработкаПолученияСпискаНавигационныхСсылок.md).

Описание:

Вызывается при выборе в диалоге получения навигационной ссылки одного из вариантов ссылки, как добавленного в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](Расширение_таблицы_формы_для_динамического_списка.ОбработкаПолученияСпискаНавигационныхСсылок.md), так и стандартного.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.21.
