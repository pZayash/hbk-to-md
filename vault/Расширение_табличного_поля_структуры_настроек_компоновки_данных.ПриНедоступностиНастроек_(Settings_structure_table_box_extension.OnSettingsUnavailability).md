Расширение табличного поля структуры настроек компоновки данных (Settings structure table box extension)

ПриНедоступностиНастроек (OnSettingsUnavailability)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ДействиеПриНедоступностиНастроекКомпоновкиДанных](ДействиеПриНедоступностиНастроекКомпоновкиДанных_(OnUnavailabilityDataCompositionSettingsAction).md).   
Определяет способ управления редакторами настроек, связанными с текущими данными структуры настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
