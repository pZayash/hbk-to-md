ЗаписьДанных (DataWriter)

НачатьЗаписьЦелого32 (BeginWriteInt32)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗаписьЦелого32(<ОписаниеОповещения>, <Число>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Целое число, которое будет записано в целевой поток. Значение числа должно находиться в диапазоне от 0 до 2^32-1.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа перед записью в поток. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Значение по умолчанию: Неопределено.

Описание:

Начать запись целого 32-битного числа в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
