Рабочий сервер (IWorkingServerInfo)

DedicatedManagers (DedicatedManagers)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит выделенные менеджеры под каждый сервис.   
[Истина](lang__def_BooleanTrue.md) – на данном рабочем сервере для каждого вида сервиса требуется запуск отдельного процесса менеджера кластера. Может использоваться в процессе опытной эксплуатации.   
[Ложь](lang__def_BooleanFalse.md) – все сервисы кластера на данном рабочем сервере назначаются на один процесс менеджер кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
