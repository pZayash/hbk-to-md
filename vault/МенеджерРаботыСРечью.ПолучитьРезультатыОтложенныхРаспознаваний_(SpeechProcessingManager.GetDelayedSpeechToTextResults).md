МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьРезультатыОтложенныхРаспознаваний (GetDelayedSpeechToTextResults)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьРезультатыОтложенныхРаспознаваний(<МассивКлючей>)

Параметры:

<МассивКлючей> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ИдентификаторОтложенногоРаспознаванияРечи](ИдентификаторОтложенногоРаспознаванияРечи_(SpeechToTextDelayedID).md), полученные с помощью метода [ВыполнитьОтложенноеРаспознавание](МенеджерРаботыСРечью.ВыполнитьОтложенноеРаспознавание_(SpeechProcessingManager.ExecuteDelayedSpeechToText).md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [РезультатОтложенногоРаспознаванияРечи](РезультатОтложенногоРаспознаванияРечи_(DelayedSpeechToTextResult).md).

Описание:

Получает результаты отложенного распознавания или ошибок, если результат по каким-то ключам никогда не будет получен (неизвестный ключ, истек срок хранения результатов, неподдерживаемый формат и т.д.).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
