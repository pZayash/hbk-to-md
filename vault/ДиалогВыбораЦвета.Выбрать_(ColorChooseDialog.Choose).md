ДиалогВыбораЦвета (ColorChooseDialog)

Выбрать (Choose)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - цвет выбран; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает диалог выбора цвета.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [Показать](ДиалогВыбораЦвета.Показать_(ColorChooseDialog.Show).md).
