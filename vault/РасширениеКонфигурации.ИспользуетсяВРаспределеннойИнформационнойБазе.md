РасширениеКонфигурации (ConfigurationExtension)

ИспользуетсяВРаспределеннойИнформационнойБазе (UsedInDistributedInfoBase)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то данное расширение конфигурации будет передаваться в распределенных информационных базах, организованных планами обмена, у которых свойство [РаспределеннаяИнформационнаяБаза](ОбъектМетаданных_ПланОбмена.РаспределеннаяИнформационнаяБаза.md) установлено в значение Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Изменение свойства на клиенте требует обращения к серверу.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации.md), свойство [ГлавныйУзел](РасширениеКонфигурации.ГлавныйУзел.md)  
[ОбъектМетаданных: ПланОбмена](ОбъектМетаданных_ПланОбмена.md), свойство [ВключатьРасширенияКонфигурации](ОбъектМетаданных_ПланОбмена.ВключатьРасширенияКонфигурации.md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
