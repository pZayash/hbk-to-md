БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ВедущаяЗадача (HeadTask)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   
Содержит ссылку на ведущую задачу, которую бизнес-процесс будет автоматически помечать как выполненную при своем завершении. Это свойство используется при формировании вложенных бизнес-процессов. В этом случае вложенный бизнес-процесс выполняет ведущую задачу, что приводит к продвижению родительского бизнес-процесса дальше по маршруту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
