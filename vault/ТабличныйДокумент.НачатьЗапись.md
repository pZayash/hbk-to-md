ТабличныйДокумент (SpreadsheetDocument)

НачатьЗапись (BeginWriting)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ИмяФайла>, <ТипФайлаТаблицы>, <Вложения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения записи файла со следующими параметрами:

- <Результат> - результат операции:
  - Истина - если запись была инициирована с помощью расширений для браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
  - Ложь - если запись была инициирована с помощью расширений для браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
  - Неопределено - если запись была инициирована скачиванием файла средствами браузера и не известно - был ли файл записан или нет.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в котором сохраняется табличный документ.

<ТипФайлаТаблицы> (необязательный)

Тип: [ТипФайлаТабличногоДокумента](ТипФайлаТабличногоДокумента.md).   
Формат, в котором будет сохранен табличный документ.  
Значение по умолчанию: [MXL](ТипФайлаТабличногоДокумента.MXL.md).

<Вложения> (необязательный)

Тип: [КоллекцияВложенийPDF](КоллекцияВложенийPDF.md).   
Объект содержит вложения PDF-файлов. Используется только при записи в форматах PDF и PDF/A-3.

Описание:

Вызывает начало записи табличного документа в файл.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

При работе на веб-клиенте нет возможности сохранять в формате XLS95.  
Не поддерживается сохранение в [PDF](ТипФайлаТабличногоДокумента.PDF.md) картинок типа Метафайл.  
При работе на веб-клиенте вызов метода выполняет обращение к серверу.  
При выводе в формат PDF шрифты, которые нельзя внедрять, а также шрифты, не являющиеся TrueType или OpenType, будут заменены на шрифт Arial. Если данный шрифт не обнаружен, то будет использован шрифт Helvetica.
