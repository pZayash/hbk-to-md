Рекомендуется использовать:

- [ПанельКнопокСообщенияСистемыВзаимодействия](ПанельКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanel).md)

Доступен, начиная с версии 8.3.18.

Элементы коллекции:

[ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются действия сообщения системы взаимодействия.  
Возможно обращение к элементам коллекции посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Представляет собой коллекцию действий сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
