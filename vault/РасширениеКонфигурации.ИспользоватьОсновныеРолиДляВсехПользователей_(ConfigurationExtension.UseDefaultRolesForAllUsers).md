РасширениеКонфигурации (ConfigurationExtension)

ИспользоватьОсновныеРолиДляВсехПользователей (UseDefaultRolesForAllUsers)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то для сеансов, в которых подключено расширение любой пользователь будет обладать правами из основных ролей данного расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
