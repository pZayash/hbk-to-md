ДокументHTML (HTMLDocument)

КодировкаИсточника (InputEncoding)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Кодировка, в которой записан документ (информация имеется на момент разбора).  
Пустая строка, если кодировка неизвестна (например, документ формируется в памяти).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
