ЭлементСвязьHTML (HTMLLinkElement)

Атрибуты (Attributes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияАтрибутовHTML](КоллекцияАтрибутовHTML.md).   
Содержит коллекцию атрибутов узла. Коллекция атрибутов доступна только для узла Element.  
Узел Атрибуты

- [Атрибут](ТипУзлаDOM.Атрибут.md) - Неопределено;
- [Комментарий](ТипУзлаDOM.Комментарий.md) - Неопределено;
- [Документ](ТипУзлаDOM.Документ.md) - Неопределено;
- [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - Неопределено;
- [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) - Неопределено;
- [Элемент](ТипУзлаDOM.Элемент.md) - [КоллекцияАтрибутовHTML](КоллекцияАтрибутовHTML.md);
- [Текст](ТипУзлаDOM.Текст.md) - Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
