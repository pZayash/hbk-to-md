Кластер серверов (IClusterInfo)

LoadBalancingMode (LoadBalancingMode)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Режим выбора рабочего процесса для соединения.  
0 - приоритет выбора по доступной производительности;  
1 - приоритет выбора по доступной памяти;  
Значение по умолчанию 0.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
