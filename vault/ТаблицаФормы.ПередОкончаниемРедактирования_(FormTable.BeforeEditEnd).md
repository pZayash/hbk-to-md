# ТаблицаФормы.ПередОкончаниемРедактирования

ТаблицаФормы (FormTable)

ПередОкончаниемРедактирования (BeforeEditEnd)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередОкончаниемРедактирования(<НоваяСтрока>, <ОтменаРедактирования>, <Отказ>)

Параметры:

<НоваяСтрока>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если строка была добавлена или скопирована.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если произошла отмена редактирования.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается перед окончанием редактирования строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Примечание:

В обработчике данного события нельзя использовать серверные методы формы с директивой компиляции &НаСервере, а также изменять свойства, которые могут привести к серверному вызову.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.15.
