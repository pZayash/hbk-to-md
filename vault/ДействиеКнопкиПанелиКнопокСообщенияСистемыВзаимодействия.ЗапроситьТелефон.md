ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonAction)

ЗапроситьТелефон (RequestPhone)

Доступен, начиная с версии 8.3.25.

Описание:

Действие кнопки создает сообщение с запросом контактной информации (номер телефона) пользователя.  
Действие не доступно в тонком, толстом, веб-клиенте, но доступно в мобильном клиенте и для использования в интеграциях с мессенджерами.

Использование в версии:

Доступен, начиная с версии 8.3.25.
