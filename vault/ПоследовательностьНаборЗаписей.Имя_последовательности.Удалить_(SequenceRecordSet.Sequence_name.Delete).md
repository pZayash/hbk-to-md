# ПоследовательностьНаборЗаписей.<Имя последовательности>.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Последовательности](Последовательности.md) › [ПоследовательностьНаборЗаписей.<Имя последовательности>](ПоследовательностьНаборЗаписей.Имя_последовательности_(SequenceRecordSet.Sequence_name).md) › [Методы](ПоследовательностьНаборЗаписей.Имя_последовательности__Методы.md)

ПоследовательностьНаборЗаписей.<Имя последовательности> (SequenceRecordSet.<Sequence name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [Число](lang__def_Number.md), [ПоследовательностьЗапись.](ПоследовательностьЗапись.Имя_последовательности_(SequenceRecord.Sequence_name).md)<[Имя последовательности](ПоследовательностьЗапись.Имя_последовательности_(SequenceRecord.Sequence_name).md)>.   
Удаляемая из набора запись. Может быть указан индекс записи в наборе или сама запись.

Описание:

Удаляет указанную запись регистрации документа из набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
