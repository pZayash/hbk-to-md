Соединение с агентом сервера (IServerAgentConnection)

GetClusterAdmins (GetClusterAdmins)

Доступен, начиная с версии 8.1.

Синтаксис:

GetClusterAdmins(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Описание кластера с интерфейсом [Администратор](Администратор_(IRegUserInfo).md), администраторов которых необходимо получить.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив описаний администраторов. Каждый элемент массива является объектом с интерфейсом IRegUserInfo.

Описание:

Получает массив администраторов кластера. Про каждого администратора доступна вся информация, кроме его пароля. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера. Если не зарегистрирован ни один администратор кластера, то перед выполнением этого метода следует выполнить метод [Authenticate](Соединение_с_агентом_сервера.Authenticate_(IServerAgentConnection.Authenticate).md)(Кластер, "", ""), где Кластер - кластер, массив администраторов которого требуется получить.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
