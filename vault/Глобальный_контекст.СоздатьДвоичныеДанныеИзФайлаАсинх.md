Глобальный контекст (Global context)

СоздатьДвоичныеДанныеИзФайлаАсинх (CreateBinaryDataFromFileAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СоздатьДвоичныеДанныеИзФайлаАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, из которого надо создать [ДвоичныеДанные](ДвоичныеДанные.md).

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [ДвоичныеДанные](ДвоичныеДанные.md), содержащий данные из файла.

Описание:

Асинхронная функция. Создает двоичные данные из файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [НачатьСозданиеДвоичныхДанныхИзФайла](Глобальный_контекст.НачатьСозданиеДвоичныхДанныхИзФайла.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
