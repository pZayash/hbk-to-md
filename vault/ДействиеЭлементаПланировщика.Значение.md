Доступен, начиная с версии 8.3.18.

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение действия. Используется для поиска с помощью метода [Найти](КоллекцияДействийЭлементаПланировщика.Найти.md) коллекции действий элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
