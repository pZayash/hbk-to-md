Описание:

Идентифицирует выгрузку данных системы взаимодействия.  
При сравнении идентификаторов средствами встроенного языка сравнение выполняется по значению идентификаторов.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
