ОформлениеЯчейки (CellAppearance)

УстановитьКартинку (SetPicture)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьКартинку(<Картинка>)

Параметры:

<Картинка> (обязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка.

Описание:

Устанавливает свойство [Картинка](ОформлениеЯчейки.Картинка_(CellAppearance.Picture).md) объекта [ОформлениеЯчейки](ОформлениеЯчейки_(CellAppearance).md) в значение, переданное в параметре <Картинка>, и устанавливает свойство [ОтображатьКартинку](ОформлениеЯчейки.ОтображатьКартинку_(CellAppearance.ShowPicture).md) в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
