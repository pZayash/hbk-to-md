КнопкаПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButton)

ТекстСообщения (MessageText)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст сообщения, которое будет отправлено при нажатии на кнопку, если действие кнопки [ОтправитьСообщение](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщение.md) или [ОтправитьСообщениеСДанными](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщениеСДанными.md).  
Если свойство не заполнено, то текст сообщения будет содержать значение из свойства [Текст](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Текст.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
