ТипМаркераДиаграммы (ChartMarkerType)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Маркер устанавливается автоматически:

- для серий - [Чередование](ТипМаркераДиаграммы.Чередование.md),
- для линий тренда - результирующий маркер из серии,
- для области уравнения - результирующий маркер из линии тренда.

Использование в версии:

Доступен, начиная с версии 8.3.10.
