НастройкаУсловногоОформления (ConditionalAppearanceSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента настройки условного оформления.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойУсловногоОформления](ЭлементУправленияНастройкойУсловногоОформления_(ConditionalAppearanceSettingControl).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента по имени элемента настройки условного оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
