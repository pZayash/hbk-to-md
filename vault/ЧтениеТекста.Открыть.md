ЧтениеТекста (TextReader)

Открыть (Open)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть файл

Синтаксис:

Открыть(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <МонопольныйРежим>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя читаемого файла.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Определяет кодировку текста читаемого текстового файла (согласован с текстовым документом). Если параметр не задан, формат текста будет определен автоматически по сигнатуре BOM в начале файла; если сигнатура BOM в файле отсутствует, файл будет открыт в кодировке [ANSI](КодировкаТекста.ANSI.md).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Указывает строку, являющуюся разделителем строк в читаемом файле.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Указывает разделитель строк в файле при конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<МонопольныйРежим> (необязательный)

Тип: Булево.   
Определяет режим открытия файла. Истина - монопольный, Ложь - разделяемый.  
Значение по умолчанию: Истина.

Описание варианта метода:

Открывает текстовый файл для чтения в монопольном режиме.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения текста.

<Кодировка> (необязательный)

Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Указывает строку, являющуюся разделителем строк в читаемом потоке.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Указывает разделитель строк в потоке при конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

Описание варианта метода:

Установить поток для чтения.

Описание:

Открывает текстовый файл или поток для чтения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Открыть("d:\dos.txt", КодировкаТекста.OEM); |

Использование в версии:

Доступен, начиная с версии 8.0.
