КоллекцияАтрибутовHTML (HTMLAttributeMap)

Элемент (Item)

Доступен, начиная с версии 8.2.

Синтаксис:

Элемент(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс атрибута в коллекции.

Возвращаемое значение:

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md).   

Описание:

Получает узел атрибута HTML по индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
