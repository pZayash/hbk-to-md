# Рабочий сервер.CreatePortRange

Рабочий сервер (IWorkingServerInfo)

CreatePortRange (CreatePortRange)

Доступен, начиная с версии 8.1.

Синтаксис:

CreatePortRange()

Возвращаемое значение:

Тип: [Диапазон портов](Диапазон_портов_(IPortRangeInfo).md).   

Описание:

Создает диапазон IP-портов.

Доступность:

Интеграция.

Примечание:

Созданный диапазон можно заполнить и использовать для добавления к списку диапазонов методом [InsertPortRange](Рабочий_сервер.InsertPortRange_(IWorkingServerInfo.InsertPortRange).md).

Использование в версии:

Доступен, начиная с версии 8.1.
