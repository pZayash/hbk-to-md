ТестируемоеПолеФормы (TestedFormField)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбъект(<Тип>, <ТекстЗаголовка>, <Имя>, <Таймаут>)

Параметры:

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Тип объекта, который нужно найти.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ТекстЗаголовка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Время в секундах, в течение которого выполняется поиск объекта. Если значение параметра не задано, время поиска не ограничено.  
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: [ТестируемаяФорма](ТестируемаяФорма_(TestedForm).md), [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы_(TestedFormGroup).md), [ТестируемаяКнопкаФормы](ТестируемаяКнопкаФормы_(TestedFormButton).md), [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы_(TestedFormTable).md), [ТестируемоеПолеФормы](ТестируемоеПолеФормы_(TestedFormField).md), [ТестируемаяДекорацияФормы](ТестируемаяДекорацияФормы_(TestedFormDecoration).md), [Неопределено](lang__def_Undefined.md).   
Первый найденный объект. Если объект не найден, генерируется исключительная ситуация.

Описание:

Выполняет поиск объекта среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
