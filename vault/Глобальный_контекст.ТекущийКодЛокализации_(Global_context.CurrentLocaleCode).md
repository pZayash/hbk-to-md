Глобальный контекст (Global context)

ТекущийКодЛокализации (CurrentLocaleCode)

Доступен, начиная с версии 8.2.

Синтаксис:

ТекущийКодЛокализации()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает код локализации (язык, страна) текущего сеанса, который может быть ключом командной строки /VL или, при запуске из Конфигуратора, установкой параметров запуска.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
