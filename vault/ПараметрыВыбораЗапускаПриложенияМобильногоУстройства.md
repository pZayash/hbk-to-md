Описание:

Содержит данные для уточнения фильтров, которые будут использованы для поиска приложения мобильного устройства при запуске с помощью методов [ЗапуститьБезОжидания](ЗапускПриложенияМобильногоУстройства.ЗапуститьБезОжидания.md) и [ЗапуститьАсинх](ЗапускПриложенияМобильногоУстройства.ЗапуститьАсинх.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
