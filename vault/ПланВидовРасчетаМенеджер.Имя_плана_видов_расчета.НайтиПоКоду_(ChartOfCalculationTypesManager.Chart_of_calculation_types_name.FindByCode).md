# ПланВидовРасчетаМенеджер.<Имя плана видов расчета>.НайтиПоКоду

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [ПланВидовРасчетаМенеджер.<Имя плана видов расчета>](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md) › [Методы](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета__Методы.md)

ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

НайтиПоКоду (FindByCode)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>)

Параметры:

<Код> (необязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md).   
Код для поиска. [Строка](lang__def_String.md) или [Число](lang__def_Number.md) в зависимости от типа кода плана видов расчета, заданного в для данного объекта конфигурации.

Возвращаемое значение:

Тип: [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)>, [Неопределено](lang__def_Undefined.md).   
Если объект по заданному коду не найдет, возвращает пустую ссылку на вид расчета.  
Если код не задан (длина = 0), то будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск вида расчета по коду.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
