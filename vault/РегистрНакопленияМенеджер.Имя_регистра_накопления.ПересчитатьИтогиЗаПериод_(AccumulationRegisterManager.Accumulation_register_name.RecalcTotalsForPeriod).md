РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

ПересчитатьИтогиЗаПериод (RecalcTotalsForPeriod)

Доступен, начиная с версии 8.1.

Синтаксис:

ПересчитатьИтогиЗаПериод(<НачалоПериода>, <КонецПериода>)

Параметры:

<НачалоПериода> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Начало периода пересчета. Если параметр не задан, то пересчет выполняется с самого начала.

<КонецПериода> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Конец периода пересчета. Если параметр не задан, то пересчет выполняется до самого конца.

Описание:

Выполняет пересчет итогов регистра накопления за период.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Регистры.Остатки.ПересчитатьИтогиЗаПериод(,НачалоМесяца(ТекущаяДата())); |

Использование в версии:

Доступен, начиная с версии 8.1.
