МенеджерРаботыСРечью (SpeechProcessingManager)

УдалитьДополнительнуюГрамматикуИнформационнойБазы (DeleteInfoBaseAdditionalGrammar)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УдалитьДополнительнуюГрамматикуИнформационнойБазы(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор удаляемой грамматики.

Описание:

Удаляет дополнительную грамматику распознавания речи из информационной базы (с учетом используемых разделителей сеанса).  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
