# ЧтениеСообщенияОбмена.ЧтениеXML

ЧтениеСообщенияОбмена (ExchangeMessageReader)

ЧтениеXML (XMLReader)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM_(DOMNodeReader).md), [ЧтениеFastInfoset](ЧтениеFastInfoset_(FastInfosetReader).md), [ЧтениеXML](ЧтениеXML_(XMLReader).md), [Неопределено](lang__def_Undefined.md).   
Содержит объект, с помощью которого осуществляется чтение сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md) и после обращения к методу [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение_(ExchangeMessageReader.EndRead).md) или [ПрерватьЧтение](ЧтениеСообщенияОбмена.ПрерватьЧтение_(ExchangeMessageReader.CancelRead).md) значением данного свойства будет [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.0.
