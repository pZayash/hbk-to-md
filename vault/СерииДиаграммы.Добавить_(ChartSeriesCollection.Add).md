# СерииДиаграммы.Добавить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Диаграмма](Диаграмма.md) › [СерииДиаграммы](СерииДиаграммы_(ChartSeriesCollection).md) › [Методы](СерииДиаграммы__Методы.md)

СерииДиаграммы (ChartSeriesCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Название серии.

Возвращаемое значение:

Тип: [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   

Описание:

Добавляет новую серию в конец коллекции серий диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Диаграмма.Очистить(); Для Серия = 1 По 4 Цикл     НоваяСерия = Диаграмма.Серии.Добавить("Серия №" + Серия); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
