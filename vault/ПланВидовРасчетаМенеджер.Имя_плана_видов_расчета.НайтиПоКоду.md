ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

НайтиПоКоду (FindByCode)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>)

Параметры:

<Код> (необязательный)

Тип: Строка, Число.   
Код для поиска. Строка или Число в зависимости от типа кода плана видов расчета, заданного в для данного объекта конфигурации.

Возвращаемое значение:

Тип: [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)>, Неопределено.   
Если объект по заданному коду не найдет, возвращает пустую ссылку на вид расчета.  
Если код не задан (длина = 0), то будет возвращено Неопределено.

Описание:

Осуществляет поиск вида расчета по коду.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
