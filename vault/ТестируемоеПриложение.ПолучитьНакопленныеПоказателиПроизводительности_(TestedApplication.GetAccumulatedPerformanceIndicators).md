ТестируемоеПриложение (TestedApplication)

ПолучитьНакопленныеПоказателиПроизводительности (GetAccumulatedPerformanceIndicators)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьНакопленныеПоказателиПроизводительности(<ОчиститьНакопленные>)

Параметры:

<ОчиститьНакопленные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Очищает информацию о накопленных вызовах в клиенте тестирования перед возвратом.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая следующие поля:

- Вызовы (Calls) – Тип значения – [Число](lang__def_Number.md). Количество вызовов сервера с момента последней очистки истории накопленных.
- Длительность (Duration) – Тип значения – [Число](lang__def_Number.md). Время, затраченное на все вызовы с момента последней очистки истории накопленных.
- Отправлено (Sent) – Тип значения – [Число](lang__def_Number.md). Количество отправленных байт с момента последней очистки истории накопленных.
- Принято (Received) – Тип значения – [Число](lang__def_Number.md). Количество принятых байт с момента последней очистки истории накопленных.

  
Имена полей зависят от варианта встроенного языка, выбранного на стороне менеджера тестирования.

Описание:

Получает информацию о производительности тестируемого клиента.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
