# СжатиеДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.0.

Конструкторы:

[По степени сжатия](СжатиеДанных.По_умолчанию.md)  

Описание:

Реализует алгоритм сжатия данных "Deflation". Может быть использован для сжатия данных объектом [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Хранилище = Новый ХранилищеЗначения(Значение, Новый СжатиеДанных(9)); |

См. также:

[ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md), конструктор [На основании значения](ХранилищеЗначения.По_умолчанию.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
