ВнешнийОтчет (ExternalReport)

ПриКомпоновкеРезультата (OnComposeResult)

Доступен, начиная с версии 8.1.

Синтаксис:

ПриКомпоновкеРезультата()

Описание:

Вызывается при выполнении отчета с помощью метода [СкомпоноватьРезультат](ВнешнийОтчет.СкомпоноватьРезультат_(ExternalReport.ComposeResult).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
