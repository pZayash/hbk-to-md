ПостроительDOM (DOMBuilder)

Прочитать (Read)

Доступен, начиная с версии 8.1.

Синтаксис:

Прочитать(<ИсточникДанныхXML>)

Параметры:

<ИсточникДанныхXML> (обязательный)

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM_(DOMNodeReader).md), [ЧтениеFastInfoset](ЧтениеFastInfoset_(FastInfosetReader).md), [ЧтениеXML](ЧтениеXML_(XMLReader).md), [ЧтениеHTML](ЧтениеHTML_(HTMLReader).md).   
Объект чтения данных XML или чтения данных HTML.

Возвращаемое значение:

Тип: [ДокументDOM](ДокументDOM_(DOMDocument).md), [ДокументHTML](ДокументHTML_(HTMLDocument).md).   
Если выполняется чтение HTML с помощью объекта [ЧтениеHTML](ЧтениеHTML_(HTMLReader).md), данный метод вернет объект типа [ДокументHTML](ДокументHTML_(HTMLDocument).md).

Описание:

Создает документ DOM. Производит чтение документа XML из источника данных XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Параметры DOM в созданный документ не переносятся.  
При чтении документа XML обработка схемы Document Type Definition (DTD) из читаемого документа не выполняется.

Использование в версии:

Доступен, начиная с версии 8.1.
