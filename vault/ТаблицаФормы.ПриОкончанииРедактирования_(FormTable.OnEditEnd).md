ТаблицаФормы (FormTable)

ПриОкончанииРедактирования (OnEditEnd)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриОкончанииРедактирования(<НоваяСтрока>, <ОтменаРедактирования>)

Параметры:

<НоваяСтрока>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак редактирования новой строки. Имеет значение [Истина](lang__def_BooleanTrue.md), если строка была добавлена или скопирована.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если произошла отмена редактирования.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается после выхода из режима редактирования строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Примечание:

В обработчике данного события нельзя использовать серверные методы формы с директивой компиляции &НаСервере, а также изменять свойства, которые могут привести к серверному вызову.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.24.
