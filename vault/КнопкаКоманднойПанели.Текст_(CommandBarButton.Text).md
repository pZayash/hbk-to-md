КнопкаКоманднойПанели (CommandBarButton)

Текст (Text)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст, отображаемый на кнопке, при соответствующем стиле отображения, определяемом свойством [Отображение](КнопкаКоманднойПанели.Отображение_(CommandBarButton.Representation).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
