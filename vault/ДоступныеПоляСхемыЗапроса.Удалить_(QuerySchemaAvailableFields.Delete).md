ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Удалить (Delete)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого элемента.

Описание:

Удаляет элемент из коллекции. Метод используется только для доступных полей описания временной таблицы ([ДоступныеПоля](ОписаниеВременнойТаблицыСхемыЗапроса.ДоступныеПоля_(QuerySchemaTempTableDescription.AvailableFields).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
