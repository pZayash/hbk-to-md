Описание:

Объекты этого типа предназначены для доступа к настройкам, хранящимся в стандартном хранилище.   
Для доступа к настройкам вариантов отчетов объект этого типа должен быть получен из свойства глобального контекста [ХранилищеВариантовОтчетов](Глобальный_контекст.ХранилищеВариантовОтчетов.md).   
Для доступа к пользовательским настройкам отчетов объект этого типа должен быть получен из свойства глобального контекста [ХранилищеПользовательскихНастроекОтчетов](Глобальный_контекст.ХранилищеПользовательскихНастроекОтчетов.md).   
Для доступа к пользовательским настройкам данных форм объект этого типа должен быть получен из свойства глобального контекста [ХранилищеНастроекДанныхФорм](Глобальный_контекст.ХранилищеНастроекДанныхФорм.md).   
Для доступа к общим настройкам объект этого типа должен быть получен из свойства глобального контекста [ХранилищеОбщихНастроек](Глобальный_контекст.ХранилищеОбщихНастроек.md).   
Для доступа к системным настройкам объект этого типа должен быть получен из свойства глобального контекста [ХранилищеСистемныхНастроек](Глобальный_контекст.ХранилищеСистемныхНастроек.md).  
Для доступа к пользовательским настройкам динамических списков объект этого типа должен быть получен из свойства глобального контекста [ХранилищеПользовательскихНастроекДинамическихСписков](Глобальный_контекст.ХранилищеПользовательскихНастроекДинамическихСписков.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).
