ПочтовоеСообщение (MailMessage)

Непрочтено (IsUnread)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Прочтенное или не прочтенное сообщение. Имеет смысл только для полученных сообщений.  
[Истина](lang__def_BooleanTrue.md) - непрочтенное.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
