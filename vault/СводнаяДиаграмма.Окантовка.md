СводнаяДиаграмма (PivotChart)

Окантовка (Outline)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак окантовки черным цветом областей на сводной диаграмме. Применяется в диаграммах, имеющих закрашиваемые области. Вне зависимости от цвета областей (серий) окантовка наносится тонкой черной линией.  
Истина - окантовка используется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл для диаграмм с закрашиваемыми областями.

Использование в версии:

Доступен, начиная с версии 8.0.
