# Расширение поля формы для поля флажка.ВысотаЗаголовкаЭлемента

Расширение поля формы для поля флажка (Form field extension for a check box field)

ВысотаЗаголовкаЭлемента (ItemTitleHeight)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Высота заголовка элемента. 0 - определять автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Используется при виде флажка [Тумблер](ВидФлажка.Тумблер_(CheckBoxType.Tumbler).md).

Использование в версии:

Доступен, начиная с версии 8.3.2.
