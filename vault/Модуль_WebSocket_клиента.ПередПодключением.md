Модуль WebSocket клиента (WebSocket client module)

ПередПодключением (BeforeConnect)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПередПодключением(<Клиент>)

Параметры:

<Клиент>

Тип: [WebSocketКлиент](WebSocketКлиент.md).   
Ссылка на объект клиента.

Описание:

Обработчик вызывается перед попыткой установить соединение. Позволяет изменить настройки соединения.  
Пример:

|  |
| --- |
| Процедура ПередПодключением(Соединение) Соединение.Параметры.Заголовки["Authorization"] = "Bearer " + token; Соединение.Параметры.ДополнительныеПараметры = "LoggingLevel: debug"; КонецПроцедуры |

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
