Описание:

Значения параметров вывода для группировок таблиц.  
Содержит значения следующих параметров:  
1. РасположениеИтогов (TotalsPlacement) - расположение итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных.md));  
2. РасположениеПолейГруппировки (GroupFieldsPlacement) - расположение полей группировки (тип [РасположениеПолейГруппировкиКомпоновкиДанных](РасположениеПолейГруппировкиКомпоновкиДанных.md));  
3. РасположениеГруппировки (GroupPlacement) - расположение группировки (тип [РасположениеГруппировкиКомпоновкиДанных](РасположениеГруппировкиКомпоновкиДанных.md));  
4. РасположениеРеквизитов (AttributePlacement) - расположение реквизитов (тип [РасположениеРеквизитовКомпоновкиДанных](РасположениеРеквизитовКомпоновкиДанных.md));  
5. РасположениеРесурсов (ResourcePlacement) - расположение ресурсов (тип [РасположениеРесурсовКомпоновкиДанных](РасположениеРесурсовКомпоновкиДанных.md));  
6. РасположениеОбщихИтогов (OverallsPlacement) - расположение общих итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных.md));  
7. ТипЗаголовкаПолей (FieldsTitleType) - тип заголовка полей (тип [ТипЗаголовкаПолейКомпоновкиДанных](ТипЗаголовкаПолейКомпоновкиДанных.md));  
8. КоличествоЗаписей (RecordsCount) - количество записей, которое необходимо выводить в группировке (тип Число);  
9. ВыводитьЗаголовок (TitleOutput) - выводить заголовок (тип [ТипВыводаТекстаКомпоновкиДанных](ТипВыводаТекстаКомпоновкиДанных.md));  
10. Заголовок (Title) - заголовок (тип Строка);  
11. ВыводитьОтбор (FilterOutput) - выводить отбор (тип [ТипВыводаТекстаКомпоновкиДанных](ТипВыводаТекстаКомпоновкиДанных.md));  
12. МакетОформления (AppearanceTemplate) - имя макета оформления (тип Строка);  
13. Процент записей (RecordsPercent) - процент от общего количества записей, выводимый в группировке (тип Число);  
14. АвтоПозицияРесурсов (ResourcesAutoPosition) - авто позиция ресурсов (тип [АвтоПозицияРесурсовКомпоновкиДанных](АвтоПозицияРесурсовКомпоновкиДанных.md));  
15. ВариантИспользованияГруппировки (GroupUseVariant) - вариант использования группировки (тип [ВариантИспользованияГруппировкиКомпоновкиДанных](ВариантИспользованияГруппировкиКомпоновкиДанных.md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/settings}. Имя типа XDTO: TableGroupOutputParameterValues.
