ЭлементЯчейкаТаблицыHTML (HTMLTableCellElement)

ОбластьВидимости (Scope)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Этот атрибут определяет набор ячеек, для которых информация заголовка устанавливается текущим заголовком.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
