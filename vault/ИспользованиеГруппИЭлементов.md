Значения

[Группы (Folders)](ИспользованиеГруппИЭлементов.Группы.md)  
[ГруппыИЭлементы (FoldersAndItems)](ИспользованиеГруппИЭлементов.ГруппыИЭлементы.md)

Описание:

Определяет способ использования групп и элементов справочника.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |
