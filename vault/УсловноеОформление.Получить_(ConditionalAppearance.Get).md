УсловноеОформление (ConditionalAppearance)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс области оформления.

Возвращаемое значение:

Тип: [ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md).   

Описание:

Получить область оформления по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
