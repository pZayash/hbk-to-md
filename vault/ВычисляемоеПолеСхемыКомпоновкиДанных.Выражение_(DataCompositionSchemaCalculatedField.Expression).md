ВычисляемоеПолеСхемыКомпоновкиДанных (DataCompositionSchemaCalculatedField)

Выражение (Expression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение, по которому будет вычисляться значение данного поля. В выражении могут быть использованы только поля наборов данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование в выражении вычисляемых полей не допускается.

Использование в версии:

Доступен, начиная с версии 8.1.
