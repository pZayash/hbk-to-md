ОбъектМетаданных: Ресурс (MetadataObject: Resource)

ПризнакУчетаСубконто (ExtDimensionAccountingFlag)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: ПризнакУчетаСубконтоПланаСчетов](ОбъектМетаданных_ПризнакУчетаСубконтоПланаСчетов_(MetadataObject_ExtDimensionAccountingFlag).md).   
Объект метаданных, описывающий признак учета субконто соответствующего плана счетов. Определяет виды субконто, в разрезе которых будет вестись учет в данном ресурсе регистра бухгалтерии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется для объектов метаданных, описывающих ресурсы регистра бухгалтерии.

Использование в версии:

Доступен, начиная с версии 8.1.
