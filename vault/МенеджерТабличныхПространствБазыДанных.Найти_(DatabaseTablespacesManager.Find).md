МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

Найти (Find)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя менеджера табличного пространства.

Возвращаемое значение:

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных_(DatabaseTablespaceManager).md).   

Описание:

Возвращает табличное пространство по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
