АтрибутDOM (DOMAttribute)

ЯвляетсяИдентификатором (IsId)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, является ли данный атрибут идентификатором владеющего им элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - данный атрибут является идентификатором владеющего им элемента, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.1.
