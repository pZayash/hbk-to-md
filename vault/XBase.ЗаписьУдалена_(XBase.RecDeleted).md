XBase (XBase)

ЗаписьУдалена (RecDeleted)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписьУдалена()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - запись удалена; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Получает пометки удаления текущей записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB.ОтображатьУдаленные = Истина; xB.Первая(); Пока Не xB.ВКонце() Цикл     Если xB.ЗаписьУдалена() Тогда         Сообщить("Удаленная запись: " + xB.НомерЗаписи());     КонецЕсли;     xB.Следующая(); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
