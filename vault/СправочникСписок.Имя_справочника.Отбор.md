СправочникСписок.<Имя справочника> (CatalogList.<Catalog name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор.md).   
Позволяет установить отбор по реквизитам справочника. Имена элементов отбора совпадают с именами колонок списка справочника, а также дополняются критериями отбора, затрагивающими справочник.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[СправочникСписок.\<Имя справочника\>](СправочникСписок.Имя_справочника.md), свойство [Колонки](СправочникСписок.Имя_справочника.Колонки.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
