МенеджерКриптографии (CryptoManager)

НачатьПолучениеКонтейнераПодписейКриптографии (BeginGettingCryptoSignaturesContainer)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НачатьПолучениеКонтейнераПодписейКриптографии(<ОписаниеОповещения>, <Подпись>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения получения информации со следующими параметрами:

- <КонтейнерПодписейКриптографии> - помещается [КонтейнерПодписейКриптографии](КонтейнерПодписейКриптографии.md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Подпись> (необязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Подпись для разбора.  
Если параметр типа Строка, тогда он обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные.md) или Поток, тогда он задает данные подписи.

Описание:

Выполняет разбор двоичных данных подписи.   
  
По завершении будет вызвана процедура, указанная в <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
