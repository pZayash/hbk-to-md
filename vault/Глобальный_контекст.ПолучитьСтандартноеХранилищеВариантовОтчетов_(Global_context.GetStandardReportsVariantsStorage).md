Глобальный контекст (Global context)

ПолучитьСтандартноеХранилищеВариантовОтчетов (GetStandardReportsVariantsStorage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСтандартноеХранилищеВариантовОтчетов()

Возвращаемое значение:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md).   

Описание:

Получает объект, с помощью которого осуществляется доступ к содержимому стандартного хранилища вариантов отчетов, независимо от того, заполнено ли свойство конфигурации ([ХранилищеВариантовОтчетов](ОбъектМетаданныхКонфигурация.ХранилищеВариантовОтчетов_(ConfigurationMetadataObject.ReportsVariantsStorage).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
