Глобальный контекст (Global context)

ПередЗавершениемРаботыСистемы (BeforeExit)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗавершениемРаботыСистемы(<Отказ>, <ТекстПредупреждения>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выхода из программы. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то работа с программой не будет завершена.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ТекстПредупреждения>

Тип: [Строка](lang__def_String.md).   
Если параметру <Отказ> установлено значение [Истина](lang__def_BooleanTrue.md) и <ТекстПредупреждения> содержит непустую строку, то окно сообщения с этой строкой будет выведено на экран и пользователю будет предложено на выбор - остаться или покинуть окно без сохранения данных.   
Если параметру <Отказ> установлено значение [Истина](lang__def_BooleanTrue.md) и <ТекстПредупреждения> содержит пустую строку, то пользователю будет отображено окно сообщения с общим текстом "Работа в данном окне не завершена" и предложено - остаться или покинуть окно без сохранения данных.  
Если параметру <Отказ> установлено значение [Ложь](lang__def_BooleanFalse.md), то значение параметра <ТекстПредупреждения> будет проигнорировано, и процесс завершения работы приложения будет продолжен.  
Примечание: В браузере Mozilla Firefox окно сообщения не будет содержать текста предупреждения.

Описание:

Возникает перед завершением работы в режиме [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) до закрытия главного окна. В процедуре-обработчике могут быть выполнены необходимые проверки и, при необходимости, может быть установлен параметр отказа от выхода из программы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), событие [ПриЗавершенииРаботыСистемы](Глобальный_контекст.ПриЗавершенииРаботыСистемы_(Global_context.OnExit).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
