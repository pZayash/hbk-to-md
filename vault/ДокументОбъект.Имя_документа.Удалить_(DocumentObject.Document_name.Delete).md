# ДокументОбъект.<Имя документа>.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет документ из базы данных.  
Важно! В отличие от метода [УстановитьПометкуУдаления](ДокументОбъект.Имя_документа.УстановитьПометкуУдаления_(DocumentObject.Document_name.SetDeletionMark).md), данный метод производит непосредственное удаление документа без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТекстВопроса = "Удалить документ без возможности восстановления"; Ответ = Вопрос(ТекстВопроса, РежимДиалогаВопрос.ДаНет); Если Ответ = КодВозвратаДиалога.Да Тогда     РасходнаяНакладная.Удалить(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
