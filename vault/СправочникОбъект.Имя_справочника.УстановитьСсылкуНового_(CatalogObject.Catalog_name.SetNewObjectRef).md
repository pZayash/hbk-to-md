# СправочникОбъект.<Имя справочника>.УстановитьСсылкуНового

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Методы](СправочникОбъект.Имя_справочника__Методы.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

УстановитьСсылкуНового (SetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСсылкуНового(<Ссылка>)

Параметры:

<Ссылка> (обязательный)

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   
Ссылка, которая будет назначена при записи нового объекта.

Описание:

Устанавливает значение для нового (созданного и еще не записанного) объекта, которое будет назначено при записи в качестве ссылки. Значение не может равняться ссылке какого-либо из имеющихся в базе данных объекта данного типа. Уникальность ссылки проверяется при записи объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СправочникМенеджер.\<Имя справочника\>](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md), метод [ПолучитьСсылку](СправочникМенеджер.Имя_справочника.ПолучитьСсылку_(CatalogManager.Catalog_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
