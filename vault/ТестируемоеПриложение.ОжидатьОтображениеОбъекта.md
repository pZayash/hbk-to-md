ТестируемоеПриложение (TestedApplication)

ОжидатьОтображениеОбъекта (WaitForObjectDisplayed)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОжидатьОтображениеОбъекта(<Тип>, <ТекстЗаголовка>, <Имя>, <Таймаут>)

Параметры:

<Тип> (необязательный)

Тип: Тип.   
Тип объекта, который должен появится.  
Значение по умолчанию: Неопределено.

<ТекстЗаголовка> (необязательный)

Тип: Строка.   
Заголовок объекта, если это применимо. Может содержать знаки подстановки "\* "и "?".

<Имя> (необязательный)

Тип: Строка.   
Имя объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Таймаут> (необязательный)

Тип: Число.   
Время ожидания в секундах.  
Значение по умолчанию: 60.

Возвращаемое значение:

Тип: Булево.   
Истина - объект появился; Ложь - закончился тайм-аут.

Описание:

Ожидает появления объекта с указанными параметрами.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
