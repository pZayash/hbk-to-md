Доступен, начиная с версии 8.3.17.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции [КлючИЗначение](КлючИЗначение.md). Ключом является значение типа [КатегорияОшибки](КатегорияОшибки.md), значением - [ТекстыСообщенияОбОшибке](ТекстыСообщенияОбОшибке.md).  
Возможно обращение к значениям посредством оператора [...]. В качестве аргумента передается значение типа [КатегорияОшибки](КатегорияОшибки.md).

Содержит тексты сообщений об ошибках по категориям.
