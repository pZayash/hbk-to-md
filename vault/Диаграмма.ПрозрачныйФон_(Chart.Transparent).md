Диаграмма (Chart)

ПрозрачныйФон (Transparent)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак прозрачности фона. При выводе комбинируется с аналогичными свойствами областей. При использовании диаграммы в табличном документе комбинируется со свойством фона рисунка табличного документа.  
[Истина](lang__def_BooleanTrue.md) - фон прозрачный.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
