ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ОбработкаПроверкиВыполнения (OnCheckExecutionProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаПроверкиВыполнения(<Результат>)

Параметры:

<Результат>

Тип: Булево.   
Если в параметре установить Истина, то будет вызван обработчик [ОбработкаПроверкиВыполнения](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаПроверкиВыполнения.md).  
Значение по умолчанию: Ложь.

Описание:

Возникает при выполнении метода [ПроверитьВыполнение](ЗадачаОбъект.Имя_задачи.ПроверитьВыполнение.md). Вызывается перед вызовом обработчика события [ОбработкаПроверкиВыполнения](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаПроверкиВыполнения.md) у соответствующей точки маршрута бизнес-процесса. Предназначено для проверки достаточных условий, чтобы считать задачу выполненной. Это отличает его от события [ПередВыполнением](ЗадачаОбъект.Имя_задачи.ПередВыполнением.md), которое предназначено для проверки необходимых условий выполнения задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Не вызывается, если задача уже выполнена.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
