Период (Period)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Дата.   
Содержит дату и время записи регистра накопления. Совместно с [Регистратор](РегистрНакопленияВыборка.Имя_регистра_накопления.Регистратор.md) и [НомерСтроки](РегистрНакопленияВыборка.Имя_регистра_накопления.НомерСтроки.md) определяет последовательность движений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
