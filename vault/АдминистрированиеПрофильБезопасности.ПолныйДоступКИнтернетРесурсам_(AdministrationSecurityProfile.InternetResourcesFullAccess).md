АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКИнтернетРесурсам (InternetResourcesFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет использование сети Интернет.  
[Истина](lang__def_BooleanTrue.md) – разрешено обращение к любым ресурсам Интернета. [Ложь](lang__def_BooleanFalse.md) – разрешено обращение только к ресурсам сети Интернет, перечисленным в списке разрешенных ресурсов интернет профиля.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
