ДинамическийСписок (DynamicList)

ОсновнаяТаблица (MainTable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Указывает динамическому списку таблицу, которая является главной в запросе.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Например, "Справочник.Номенклатура".

Использование в версии:

Доступен, начиная с версии 8.2.
