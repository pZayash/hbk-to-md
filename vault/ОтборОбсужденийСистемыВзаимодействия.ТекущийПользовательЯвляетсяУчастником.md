ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

ТекущийПользовательЯвляетсяУчастником (CurrentUserIsMember)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если установлено значение Истина, будут получены обсуждения для текущего пользователя.  
Если установлено Ложь, будут получены все обсуждения, в которых участвуют все пользователи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
