РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии> (AccountingRegisterManager.<Accounting register name>)

ВыбратьПоРегистратору (SelectByRecorder)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьПоРегистратору(<Регистратор>)

Параметры:

<Регистратор> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа.md).   
Значение регистратора.

Возвращаемое значение:

Тип: [РегистрБухгалтерииВыборка.\<Имя регистра бухгалтерии\>](РегистрБухгалтерииВыборка.Имя_регистра_бухгалтерии.md).   

Описание:

Формирует выборку записей регистра бухгалтерии по регистратору.

Доступность:

Сервер, толстый клиент, внешнее соединение.
