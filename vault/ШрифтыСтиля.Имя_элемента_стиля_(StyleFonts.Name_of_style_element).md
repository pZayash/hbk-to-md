ШрифтыСтиля (StyleFonts)

<Имя элемента стиля> (<Name of style element>)

Доступен, начиная с версии 8.0.

Описание:

Тип: [Шрифт](Шрифт_(Font).md).   
Набор значений содержит шрифты элементов стиля, созданные при конфигурировании прикладной задачи. Доступ к шрифту элемента стиля осуществляется по имени. Имена значений совпадают с именами элементов стиля типа [Шрифт](ВидЭлементаСтиля.Шрифт_(StyleElementType.Font).md), как они заданы в Конфигураторе.

Использование в версии:

Доступен, начиная с версии 8.0.
