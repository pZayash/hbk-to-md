# ПланОбменаОбъект.<Имя плана обмена>.ПриОтправкеДанныхГлавному

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [События](ПланОбменаОбъект.Имя_плана_обмена__События.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПриОтправкеДанныхГлавному (OnSendDataToMaster)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриОтправкеДанныхГлавному(<ЭлементДанных>, <ОтправкаЭлемента>)

Параметры:

<ЭлементДанных>

При вызове обработчика события данный параметр содержит элемент данных, изменение которого зарегистрировано и который должен быть помещен в сообщение обмена данными. Элементами данных могут быть [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)>, объекты базы данных, наборы записей регистров, последовательностей или перерасчетов.

<ОтправкаЭлемента>

Тип: [ОтправкаЭлементаДанных](ОтправкаЭлементаДанных_(DataItemSend).md).   
При вызове события данный параметр содержит значение [Авто](ОтправкаЭлементаДанных.Авто_(DataItemSend.Auto).md), что означает, что элемент данных будет помещен в сообщение. В обработчике события можно присвоить параметру другое значение.   
Значение [Удалить](ОтправкаЭлементаДанных.Удалить_(DataItemSend.Delete).md) означает, что в сообщение будет помещено значение, предназначенное для удаления соответствующего элемента данных в той информационной базе, которой адресовано сообщение. В случае объекта базы данных в сообщение будет помещен объект типа [УдалениеОбъекта](УдалениеОбъекта_(ObjectDeletion).md), содержащий ссылку на объект базы данных. В случае набора записей будет помещен пустой набор записей. Однако в случае элемента данных [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)> действие, производимое значением [Удалить](ОтправкаЭлементаДанных.Удалить_(DataItemSend.Delete).md), не отличается от [Авто](ОтправкаЭлементаДанных.Авто_(DataItemSend.Auto).md).  
Значение [Игнорировать](ОтправкаЭлементаДанных.Игнорировать_(DataItemSend.Ignore).md) означает, что в сообщение не будет помещено ничего, связанного с элементом данных.  
Значение по умолчанию: [Авто](ОтправкаЭлементаДанных.Авто_(DataItemSend.Auto).md).

Описание:

Обмен изменениями данных и конфигурации в распределенной информационной базы производится с помощью сообщений обмена данными. Различаются сообщения, с помощью которых производится передача изменений от главного узла подчиненному и от подчиненного главному. Обработчик данного события вызывается перед помещением образа элемента данных в сообщение обмена данными, отправляемого подчиненным узлом главному. Обработчик вызывается для каждого элемента данных, помещаемого в сообщение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.22.
