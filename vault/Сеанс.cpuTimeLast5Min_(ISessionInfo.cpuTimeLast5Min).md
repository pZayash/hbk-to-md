Сеанс (ISessionInfo)

cpuTimeLast5Min (cpuTimeLast5Min)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Время, которое затрачено процессором на обработку серверных вызовов сеанса за последние 5 минут, в миллисекундах.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.13.
