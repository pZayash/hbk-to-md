ВнешнийИсточникДанныхТаблицаОбъект.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableObject.<External source name>.<External data source table name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRef.External_source_name.External_data_source_table_name).md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](ВнешнийИсточникДанныхТаблицаОбъект.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.УстановитьСсылкуНового_(ExternalDataSourceTableObject.External_source_name.External_data_source_table_name.SetNewObjectRef).md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
