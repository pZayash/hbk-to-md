ТипДвустороннейПечати (DuplexPrintingType)

ИспользоватьНастройкиПринтера (UsePrinterSettings)

Доступен, начиная с версии 8.3.9.

Описание:

При печати платформа специально не настраивает режим двусторонней печати. Используется настройка двусторонней печати принтера.

Использование в версии:

Доступен, начиная с версии 8.3.9.
