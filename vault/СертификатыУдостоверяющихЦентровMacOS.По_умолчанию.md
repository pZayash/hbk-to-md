СертификатыУдостоверяющихЦентровMacOS (MacOSCertificationAuthorityCertificates)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый СертификатыУдостоверяющихЦентровMacOS()

Описание:

Создает объект сертификатов удостоверяющих центров OpenSSL, загружаемых из системного хранилища сертификатов операционной системы macOS.

Использование в версии:

Доступен, начиная с версии 8.3.13.
