ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

АдресИнформацииОКонфигурации (ConfigurationInformationAddress)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Ссылка на информацию о конфигурации. Может задаваться как с префиксом схемы («http://»), так и без него. Слева в верхней части страниц справки указывается наименование конфигурации, определенное в свойстве [КраткаяИнформация](ОбъектМетаданныхКонфигурация.КраткаяИнформация_(ConfigurationMetadataObject.BriefInformation).md) (если свойство не задано, то выбирается из значения свойства [Синоним](ОбъектМетаданныхКонфигурация.Синоним_(ConfigurationMetadataObject.Synonym).md), если свойство [Синоним](ОбъектМетаданныхКонфигурация.Синоним_(ConfigurationMetadataObject.Synonym).md) не задано, то выбирается из свойства объекта [ОбъектМетаданныхКонфигурация](ОбъектМетаданныхКонфигурация_(ConfigurationMetadataObject).md)). Показываемое наименование конфигурации имеет ссылку, заданную данным свойством.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
