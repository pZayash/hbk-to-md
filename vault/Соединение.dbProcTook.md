Соединение (IInfoBaseConnectionInfo)

dbProcTook (dbProcTook)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Если в момент получение списка соединений информационной базы методом [GetInfoBaseConnections](Соединение_с_рабочим_процессом.GetInfoBaseConnections.md) данное соединение выполняло обращение к серверу баз данных, то свойство содержит время в секундах, в течение которого выполняется данное обращение к серверу баз данных. В противном случае – 0.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
