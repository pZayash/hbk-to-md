ПараметрыДоступа (AccessParameters)

ОграничениеУсловием (RestrictionByCondition)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит ограничение доступа условием.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md), если ограничение доступа задано, [Ложь](lang__def_BooleanFalse.md) - в противном случае.  
Если права на какое-либо действие нет, то ограничение условием всегда [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
