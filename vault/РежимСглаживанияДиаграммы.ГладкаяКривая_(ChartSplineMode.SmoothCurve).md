РежимСглаживанияДиаграммы (ChartSplineMode)

ГладкаяКривая (SmoothCurve)

Доступен, начиная с версии 8.3.1.

Описание:

Режим сплайна, проходящего через все точки. Строится гладкая кривая, соединяющая все точки серии.

Использование в версии:

Доступен, начиная с версии 8.3.1.
