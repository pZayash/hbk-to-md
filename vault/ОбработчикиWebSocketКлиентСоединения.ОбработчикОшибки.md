ОбработчикиWebSocketКлиентСоединения (WebSocketClientConnectionHandlers)

ОбработчикОшибки (OnErrorHandler)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя экспортируемой процедуры обработчика события возникновения ошибки со следующими параметрами:

- <Соединение> - объект типа [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md).
- <КодОшибки> - объект типа Строка. Содержит код ошибки.
- <Описание> - объект типа Строка. Содержит описание ошибки.

  
Пустая строка, если это соединение [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
