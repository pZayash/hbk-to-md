Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileCOMClass (UnregSecurityProfileCOMClass)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileCOMClass(<Кластер>, <ИмяПрофиля>, <ComClassName>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должна быть отменена регистрация разрешенного COM-класса.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<ComClassName> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя [COM-класс](COM-класс_(ISecurityProfileCOMClass).md), регистрация которого отменяется.

Описание:

Отменяет регистрацию разрешенного [COM-класс](COM-класс_(ISecurityProfileCOMClass).md).

Доступность:

Интеграция.

Примечание:

Если профиля с именем, указанным в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если [COM-класс](COM-класс_(ISecurityProfileCOMClass).md) с ключом, указанным в параметре <ComClassName>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
