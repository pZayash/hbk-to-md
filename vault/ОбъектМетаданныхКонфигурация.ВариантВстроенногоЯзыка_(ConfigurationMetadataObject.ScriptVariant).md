ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ВариантВстроенногоЯзыка (ScriptVariant)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ВариантВстроенногоЯзыка](ВариантВстроенногоЯзыка_(ScriptVariant).md).   
Определяет основной язык программирования ([Русский](ВариантВстроенногоЯзыка.Русский_(ScriptVariant.Russian).md) или [Английский](ВариантВстроенногоЯзыка.Английский_(ScriptVariant.English).md)). Выбор определяет, на каком языке будет формироваться языковые конструкции в модулях (например, при использовании Синтакс–Помощника) и выдаваться информация о примитивных типах данных. Вне зависимости от значения свойства можно использовать как русский, так и английский вариант языковых конструкций. При смене значения свойства вариант написания введенных языковых конструкции не изменяется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
