ЧтениеFastInfoset (FastInfosetReader)

ЛокальноеИмяАтрибута (AttributeLocalName)

Доступен, начиная с версии 8.1.

Синтаксис:

ЛокальноеИмяАтрибута(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс атрибута.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает локальное имя атрибута по индексу. Так как атрибуты могут быть только у элемента XML, доступ к ним может осуществляться только при прочтении узла, имеющего тип [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
