СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

ФорматДанныхСтроки (StringDataFormat)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.   
Если свойство [Данные](СообщениеСистемыВзаимодействия.Данные.md) содержит данные типа Строка, то содержит MIME-тип.данных. Иначе – Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
