ЭлементПланировщика (PlannerItem)

ДатаЗамещения (ReplacementDate)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Дата, Неопределено.   
Если элемент является замещающим в расписании, содержит дату, по которой данный элемент замещает повторенный элемент из расписания. Иначе - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
