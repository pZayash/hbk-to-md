ЭлементDOM (DOMElement)

ПространствоИменПоУмолчанию (IsDefaultNamespace)

Доступен, начиная с версии 8.1.

Синтаксис:

ПространствоИменПоУмолчанию(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Проверяет, является ли пространство имен пространством имен по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
