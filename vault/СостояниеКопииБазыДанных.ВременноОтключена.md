СостояниеКопииБазыДанных (DatabaseCopyState)

ВременноОтключена (TemporarilyTurnedOff)

Доступен, начиная с версии 8.3.27.

Описание:

Копия не может выполнять запросы из-за нехватки памяти Дата акселератору. По истечении таймаута платформа выполнит попытку снова использовать копию для выполнения запросов.

Использование в версии:

Доступен, начиная с версии 8.3.27.
