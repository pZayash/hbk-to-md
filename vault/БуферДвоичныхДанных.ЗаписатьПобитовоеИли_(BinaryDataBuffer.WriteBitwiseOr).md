БуферДвоичныхДанных (BinaryDataBuffer)

ЗаписатьПобитовоеИли (WriteBitwiseOr)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЗаписатьПобитовоеИли(<Позиция>, <Буфер>, <Количество>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Начальная позиция в буфере.

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Буфер, с которым выполняется объединение.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется объединить.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Объединить заданное количество байтов, начиная с заданной позиции, с байтами из заданного буфера с использованием побитового оператора ИЛИ. Если количество байтов не указано, объединяются все байты до конца буфера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
