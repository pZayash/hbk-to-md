МенеджерКриптографии (CryptoManager)

ПроверитьМеткуВремениАсинх (VerifyTimestampAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПроверитьМеткуВремениАсинх(<Подпись>, <МеткаВремени>)

Параметры:

<Подпись> (необязательный)

Тип: [ПодписьКриптографии](ПодписьКриптографии.md).   
Подпись из контейнера подписей.

<МеткаВремени> (необязательный)

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии.md).   
Проверяемая метка времени.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
В случае успешного результата проверки результатом выполнения обещания является Неопределено. В случае ошибки объект [Обещание](Обещание.md) будет содержать исключение, которое будет выброшено из оператора Ждать, аргументом которого будет данное [Обещание](Обещание.md).

Описание:

Асинхронная функция. Метод выполняет проверку метки времени для переданной подписи криптографии:

- Соответствие хеша, указанного в метке времени, хешу полей подписи.
- Корректность сертификата метки.
- Корректность подписи на метке времени.

При возникновении ошибки проверки, будет сгенерировано исключение с описанием ошибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
