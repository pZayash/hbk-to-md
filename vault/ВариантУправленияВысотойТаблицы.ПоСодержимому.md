ВариантУправленияВысотойТаблицы (TableHeightControlVariant)

ПоСодержимому (UseContentHeight)

Доступен, начиная с версии 8.3.9.

Описание:

Высота элемента задается так, чтобы высота рабочей области элемента совпадала с высотой всех строк, заголовка и подвала. Если количество строк превышает 50, режим управления высотой переходит в режим [ВСтрокахФормы](ВариантУправленияВысотойТаблицы.ВСтрокахФормы.md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
