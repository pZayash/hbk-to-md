Форма сохранения (Save form)

КлючОбъекта (ObjectKey)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Идентификатор объекта, для которого будет сохраняться настройка.

Использование в версии:

Доступен, начиная с версии 8.2.
