СтрокаДереваЗначений (ValueTreeRow)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс строки.

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение.

Описание:

Устанавливает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Пример:

|  |
| --- |
| СтрокаДерева.Уровень(Индекс); |

Использование в версии:

Доступен, начиная с версии 8.0.
