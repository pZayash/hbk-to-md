ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

Использование (Use)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеПараметраКомпоновкиДанных](ИспользованиеПараметраКомпоновкиДанных_(DataCompositionParameterUse).md).   
Если свойство имеет значение [Авто](ИспользованиеПараметраКомпоновкиДанных.Авто_(DataCompositionParameterUse.Auto).md), необходимость использования данного параметра получается из настроек компоновки данных.   
Если свойство имеет значение [Всегда](ИспользованиеПараметраКомпоновкиДанных.Всегда_(DataCompositionParameterUse.Always).md), то параметр всегда считается используемым и всегда используется его значение, указанное в настройках компоновки.  
Свойство заполняется автоматически на основании значения данного свойства параметра схемы компоновки данных.  
Значение по умолчанию [Авто](ИспользованиеПараметраКомпоновкиДанных.Авто_(DataCompositionParameterUse.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
