КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных (DataCompositionAvailableSettingsObjectCollection)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор объекта.

Возвращаемое значение:

Тип: [ДоступныйОбъектНастройкиКомпоновкиДанных](ДоступныйОбъектНастройкиКомпоновкиДанных_(DataCompositionAvailableSettingsObject).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
