МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПоддерживаетсяРаботаСВложениями (AttachmentsSupported)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПоддерживаетсяРаботаСВложениями()

Возвращаемое значение:

Тип: Булево.   

Описание:

Возвращает Истина, если внешнее хранилище файлов подключено к серверу взаимодействия.  
Если информационная база не зарегистрирована в системе взаимодействия, то генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
