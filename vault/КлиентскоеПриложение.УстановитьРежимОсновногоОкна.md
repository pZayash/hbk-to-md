КлиентскоеПриложение (ClientApplication)

УстановитьРежимОсновногоОкна (SetMainWindowMode)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьРежимОсновногоОкна(<РежимОсновногоОкна>)

Параметры:

<РежимОсновногоОкна> (обязательный)

Тип: [РежимОсновногоОкнаКлиентскогоПриложения](РежимОсновногоОкнаКлиентскогоПриложения.md).   
Устанавливаемый режим основного окна клиентского приложения.

Описание:

Устанавливает режим основного окна клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

- Недоступен в режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение.md).
- Метод можно вызывать только внутри обработчика события [ПередНачаломРаботыСистемы](Глобальный_контекст.Глобальный_контекст.ПередНачаломРаботыСистемы.md) управляемого приложения.

[КлиентскоеПриложение](КлиентскоеПриложение.md), метод [ПолучитьРежимОсновногоОкна](КлиентскоеПриложение.ПолучитьРежимОсновногоОкна.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
