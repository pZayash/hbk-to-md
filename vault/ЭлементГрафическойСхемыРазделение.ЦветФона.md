ЭлементГрафическойСхемыРазделение (GraphicalSchemaItemSplit)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет.md).   
Содержит цвет фона элемента графической схемы. Если установлен [АвтоЦвет](ВидЦвета.АвтоЦвет.md), то используется цвет, принятый для данного элемента графической схемы (зеленый для точки старта, желтый для точки действия, красный для условия и т.д.).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ЭлементГрафическойСхемыРазделение.ПрозрачныйФон.md) имеет значение Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
