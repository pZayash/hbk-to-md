Расширение элементов управления, расположенных в табличном поле (Extension for controls located in a table box)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Предоставляет доступ к значению, которое редактируется в элементе управления. Тип значения зависит от типа элемента управления, предоставившего значение.

Доступность:

Толстый клиент.

Примечание:

Только для элементов управления, поддерживающих связь с данными.

Использование в версии:

Доступен, начиная с версии 8.0.
