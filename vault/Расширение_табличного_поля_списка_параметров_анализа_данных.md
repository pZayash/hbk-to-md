Свойства:

[НастройкаПараметров (ParametersSetting)](Расширение_табличного_поля_списка_параметров_анализа_данных.НастройкаПараметров.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка параметров анализа данных. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ПараметрАнализаДанных](ПараметрАнализаДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
