ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

Загрузить (Load)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Загрузить(<ТаблицаЗначений>)

Параметры:

<ТаблицаЗначений> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   
Таблица значений, по которой производится заполнение набора записей.

Описание:

Загружает набор записей значениями из переданной таблицы значений. При этом все прежние записи набора удаляются. Заполняются значения доступных для записи свойств записей таблицы внешнего источника данных, имена которых совпали c именами колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
