СводнаяДиаграмма (PivotChart)

КоличествоЗначенийГоризонтальнойШкалы (HorizontalScaleValueCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет количество значений, одновременно отображаемых на горизонтальной оси (оси точек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется при [ГоризонтальнаяПоддержкаМасштаба](СводнаяДиаграмма.ГоризонтальнаяПоддержкаМасштаба_(PivotChart.HorizontalScaleKeeping).md) = [КоличествоЗначений](ПоддержкаМасштабаСводнойДиаграммы.КоличествоЗначений_(PivotChartScaleKeeping.ValuesCount).md).

Использование в версии:

Доступен, начиная с версии 8.0.
