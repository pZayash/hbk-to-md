# КонтейнерПодписейКриптографии

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Криптография](Криптография-2.md)

Доступен, начиная с версии 8.3.20.

Свойства:

[Подписи (Signatures)](КонтейнерПодписейКриптографии.Подписи_(CryptoSignaturesContainer.Signatures).md)  

Описание:

Контейнер подписей криптографии. Содержит массив элементов [ПодписьКриптографии](ПодписьКриптографии_(CryptoSignature).md).  
Объект может быть получен с помощью метода [ПолучитьКонтейнерПодписейКриптографии](МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии_(CryptoManager.GetCryptoSignaturesContainer).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| КонтейнерПодписей = МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии(РезультатПодписи); Для каждого Подпись Из КонтейнерПодписей.Подписи Цикл     Сообщить(Подпись.ТипПодписи); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.23.
