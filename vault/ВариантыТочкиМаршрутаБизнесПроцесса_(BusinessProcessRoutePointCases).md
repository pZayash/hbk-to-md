# ВариантыТочкиМаршрутаБизнесПроцесса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ВариантТочкиМаршрутаБизнесПроцесса](ВариантТочкиМаршрутаБизнесПроцесса_(BusinessProcessRoutePointCase).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты.  
Возможно обращение к варианту посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента в коллекции.  

Свойства:

[\<Имя варианта\> (\<Variant name\>)](ВариантыТочкиМаршрутаБизнесПроцесса.Имя_варианта_(BusinessProcessRoutePointCases.Variant_name).md)  

Методы:

[Количество (Count)](ВариантыТочкиМаршрутаБизнесПроцесса.Количество_(BusinessProcessRoutePointCases.Count).md)  
[Получить (Get)](ВариантыТочкиМаршрутаБизнесПроцесса.Получить_(BusinessProcessRoutePointCases.Get).md)  

Описание:

Представляет коллекцию вариантов точки маршрута бизнес-процесса.  
Только для точек вида [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта_(BusinessProcessRoutePointType.Switch).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Процедура ВыборСогласованияОбработкаВыбораВарианта(         ТочкаВыбораВарианта,          Результат)     Если Случай = 1 Тогда         Результат = ТочкаВыбораВарианта.Варианты.УпрощенныйВариант;     Иначе          Если Случай = 2 Тогда             Результат = ТочкаВыбораВарианта.Варианты.ПолныйВариант;         Иначе              Если Случай = 3 Тогда                 Результат = ТочкаВыбораВарианта.Варианты.ВариантДиректора;             Иначе                         Результат = ТочкаВыбораВарианта.Варианты.ВариантБезСогласования;             КонецЕсли;         КонецЕсли;     КонецЕсли; КонецПроцедуры |

См. также:

[ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md), свойство [Варианты](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.Варианты_(BusinessProcessRoutePointRef.Business_process_name.Cases).md)
