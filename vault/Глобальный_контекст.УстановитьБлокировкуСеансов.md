Глобальный контекст (Global context)

УстановитьБлокировкуСеансов (SetSessionsLock)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьБлокировкуСеансов(<Блокировка>)

Параметры:

<Блокировка> (обязательный)

Тип: [БлокировкаСеансов](БлокировкаСеансов.md).   
Новое описание блокировки сеансов.

Описание:

Устанавливает режим блокировки сеансов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
