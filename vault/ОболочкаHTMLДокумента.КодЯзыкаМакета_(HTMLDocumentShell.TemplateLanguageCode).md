ОболочкаHTMLДокумента (HTMLDocumentShell)

КодЯзыкаМакета (TemplateLanguageCode)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит код языка, который будет использован для получения HTML-документа требуемой локализации.   
Если свойство содержит значение [Неопределено](lang__def_Undefined.md), язык макета определяется из текущих настроек конфигурации.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
