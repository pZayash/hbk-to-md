МаскаXS (XSWildcard)

ЭтаМаскаПодмножество (IsWildcardSubset)

Доступен, начиная с версии 8.1.

Синтаксис:

ЭтаМаскаПодмножество(<Маска>)

Параметры:

<Маска> (обязательный)

Тип: [МаскаXS](МаскаXS_(XSWildcard).md).   
Маска - предполагаемое подмножество.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, является ли данная маска подмножеством указанной.  
Используется алгоритм, описанный в http://www.w3.org/TR/xmlschema-1/#cos-ns-subset.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
