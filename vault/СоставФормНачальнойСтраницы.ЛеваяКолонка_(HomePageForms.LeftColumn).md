СоставФормНачальнойСтраницы (HomePageForms)

ЛеваяКолонка (LeftColumn)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит список форм начальной страницы, размещенных в левой колонке.  
Можно использовать стандартные формы.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Элемент массива (тип [Строка](lang__def_String.md)) содержит полное имя объекта метаданных формы.

Использование в версии:

Доступен, начиная с версии 8.3.5.
