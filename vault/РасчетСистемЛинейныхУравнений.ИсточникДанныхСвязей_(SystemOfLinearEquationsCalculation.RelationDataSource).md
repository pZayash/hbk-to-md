ИсточникДанныхСвязей (RelationDataSource)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   
Таблица, содержащая следующие колонки: колонка номера уравнений, колонка номера переменных, множество колонок коэффициентов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При использовании результата запроса механизм работает существенно быстрее.

Использование в версии:

Доступен, начиная с версии 8.3.14.
