ДействиеЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemAction)

Текст (Text)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Содержит текст, отображаемый в действии элемента результата глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если не задан ни <Текст>, ни <Картинка>, то в качестве текста гиперссылки выводится представление <Значения>.

Использование в версии:

Доступен, начиная с версии 8.3.18.
