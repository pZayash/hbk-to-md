SMSСообщение (SMSMessage)

На основании записи в журнале СМС

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый SMSСообщение(<ЗаписьЖурналаSMS>, <КопироватьВложения>)

Параметры:

<ЗаписьЖурналаSMS> (обязательный)

Запись в журнале SMS-сообщений.

<КопироватьВложения> (необязательный)

Тип: Булево.   
Указывает, будут ли скопированы вложения и поля [Вложения](ЗаписьЖурналаSMS.Вложения.md).  
Истина - вложения будут скопированы.  
Значение по умолчанию: Ложь.

Описание:

Создает SMS-сообщение на основе записи журнала SMS. Свойства объекта заполняются по следующим принципам:

- [Получатели](SMSСообщение.Получатели.md) - из поля [НомераТелефонов](ЗаписьЖурналаSMS.НомераТелефонов.md) копируются все цифровые номера. Имена сервисов, на которые невозможно отправить сообщения не копируются;
- [Текст](SMSСообщение.Текст.md) - получается из поля [Текст](ЗаписьЖурналаSMS.Текст.md);
- [Вложения](SMSСообщение.Вложения.md) - вложения копируются из поля [Вложения](ЗаписьЖурналаSMS.Вложения.md), если параметр <КопироватьВложения> установлен в значение Истина;
- Поля [Отправитель](SMSСообщение.Отправитель.md) и [ДатаПолучения](SMSСообщение.ДатаПолучения.md) не заполняются.

Использование в версии:

Доступен, начиная с версии 8.3.11.
