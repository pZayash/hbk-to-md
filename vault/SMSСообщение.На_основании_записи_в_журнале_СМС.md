SMSСообщение (SMSMessage)

На основании записи в журнале СМС

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый SMSСообщение(<ЗаписьЖурналаSMS>, <КопироватьВложения>)

Параметры:

<ЗаписьЖурналаSMS> (обязательный)

Запись в журнале SMS-сообщений.

<КопироватьВложения> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, будут ли скопированы вложения и поля [Вложения](ЗаписьЖурналаSMS.Вложения_(SMSLogRecord.Attachments).md).  
[Истина](lang__def_BooleanTrue.md) - вложения будут скопированы.  
Значение по умолчанию: Ложь.

Описание:

Создает SMS-сообщение на основе записи журнала SMS. Свойства объекта заполняются по следующим принципам:

- [Получатели](SMSСообщение.Получатели_(SMSMessage.To).md) - из поля [НомераТелефонов](ЗаписьЖурналаSMS.НомераТелефонов_(SMSLogRecord.PhoneNumbers).md) копируются все цифровые номера. Имена сервисов, на которые невозможно отправить сообщения не копируются;
- [Текст](SMSСообщение.Текст_(SMSMessage.Text).md) - получается из поля [Текст](ЗаписьЖурналаSMS.Текст_(SMSLogRecord.Text).md);
- [Вложения](SMSСообщение.Вложения_(SMSMessage.Attachments).md) - вложения копируются из поля [Вложения](ЗаписьЖурналаSMS.Вложения_(SMSLogRecord.Attachments).md), если параметр <КопироватьВложения> установлен в значение [Истина](lang__def_BooleanTrue.md);
- Поля [Отправитель](SMSСообщение.Отправитель_(SMSMessage.From).md) и [ДатаПолучения](SMSСообщение.ДатаПолучения_(SMSMessage.DateReceived).md) не заполняются.

Использование в версии:

Доступен, начиная с версии 8.3.11.
