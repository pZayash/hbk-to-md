Файл (File)

НачатьИнициализацию (BeginInitialization)

Не рекомендуется использовать, начиная с версии 8.3.6.

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения инициализации со следующими параметрами:

- <Файл> – инициализированный объект [Файл](Файл_(File).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла или каталога, с которым будет связан объект.

Описание:

Начинает инициализацию объекта [Файл](Файл_(File).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Файл](Файл_(File).md), метод [ИнициализироватьАсинх](Файл.ИнициализироватьАсинх_(File.InitializeAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
