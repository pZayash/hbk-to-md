ВидГруппыФормы (FormGroupType)

КонтекстноеМеню (ContextMenu)

Доступен, начиная с версии 8.2.

Описание:

Содержит контекстное меню элемента.   
Группа данного вида может содержать кнопки и группы видов [ГруппаКнопок](ВидГруппыФормы.ГруппаКнопок_(FormGroupType.ButtonGroup).md) и [Подменю](ВидГруппыФормы.Подменю_(FormGroupType.Popup).md).

Использование в версии:

Доступен, начиная с версии 8.2.
