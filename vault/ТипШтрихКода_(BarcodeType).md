Описание:

Предназначен для определения типов штрихкодов, которые система должна распознавать при вызове сканирования.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.26.
