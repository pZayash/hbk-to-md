РежимОтображенияРезультатаОтчета (ReportResultViewMode)

Авто (Auto)

Доступен, начиная с версии 8.3.13.

Описание:

Режим отображения результата отчета определяется автоматически:

- В мобильной версии - [Компактный](РежимОтображенияРезультатаОтчета.Компактный_(ReportResultViewMode.Compact).md),
- В платформе для ПК - [Обычный](РежимОтображенияРезультатаОтчета.Обычный_(ReportResultViewMode.Default).md).

Использование в версии:

Доступен, начиная с версии 8.3.13.
