РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRow)

Удалить (Delete)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемой кнопки в ряду кнопок.

Описание:

Удаляет элемент коллекции по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
