ПараметрСхемыКомпоновкиДанных (DataCompositionSchemaParameter)

ЗапрещатьНезаполненныеЗначения (DenyIncompleteValues)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в [Истина](lang__def_BooleanTrue.md), то при выполнении компоновки данных, если включено использование данного параметра и значение параметра содержит значение по умолчанию для типа значения параметра, система выдаст ошибку, в которой будет указано, что нужно ввести значение данного параметра.   
Если свойство установлено в [Ложь](lang__def_BooleanFalse.md), то отчет будет исполняться и с пустым значением параметра.  
Значение по умолчанию [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
