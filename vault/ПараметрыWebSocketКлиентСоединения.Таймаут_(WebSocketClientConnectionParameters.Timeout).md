ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

Таймаут (Timeout)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Таймаут установки соединения с WebSocket-сервером, в секундах.  
Значение по умолчанию: 30.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
