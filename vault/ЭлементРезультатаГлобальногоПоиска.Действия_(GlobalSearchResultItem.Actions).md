ЭлементРезультатаГлобальногоПоиска (GlobalSearchResultItem)

Действия (Actions)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияДействийЭлементаРезультатаГлобальногоПоиска](КоллекцияДействийЭлементаРезультатаГлобальногоПоиска_(GlobalSearchResultItemActionCollection).md).   
Содержит список действий, которые будут отображаться в виде гиперссылок при отображении результата поиска, где:

- Текст ссылки – значение свойства [Представление](ЭлементСпискаЗначений.Представление_(ValueListItem.Presentation).md) элемента списка значений. Если Представление содержит пустую строку, то гиперссылка для этого элемента не отображается;
- Если задана [Картинка](Картинка_(Picture).md), то у гиперссылки отображается картинка;
- В обработчик события [ПриВыбореДействияРезультатаГлобальногоПоиска](Глобальный_контекст.ПриВыбореДействияРезультатаГлобальногоПоиска_(Global_context.OnGlobalSearchResultActionChoice).md) передается значение, заданное в свойстве [Значение](ЭлементСпискаЗначений.Значение_(ValueListItem.Value).md) элемента списка значений.

Текст ссылки – значение свойства [Представление](ЭлементСпискаЗначений.Представление_(ValueListItem.Presentation).md) элемента списка значений. Если Представление содержит пустую строку, то гиперссылка для этого элемента не отображается;  
Если задана [Картинка](Картинка_(Picture).md), то у гиперссылки отображается картинка;  
В обработчик события [ПриВыбореДействияРезультатаГлобальногоПоиска](Глобальный_контекст.ПриВыбореДействияРезультатаГлобальногоПоиска_(Global_context.OnGlobalSearchResultActionChoice).md) передается значение, заданное в свойстве [Значение](ЭлементСпискаЗначений.Значение_(ValueListItem.Value).md) элемента списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
