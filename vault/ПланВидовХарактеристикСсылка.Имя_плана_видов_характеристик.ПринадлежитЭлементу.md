ПринадлежитЭлементу (BelongsToItem)

Доступен, начиная с версии 8.0.

Синтаксис:

ПринадлежитЭлементу(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md).   
Группа (или элемент для иерархического плана видов характеристик с иерархией элементов), для которой определяется принадлежность элемента.

Возвращаемое значение:

Тип: Булево.   
Истина - элемент подчинен группе; Ложь - в противном случае.  
Если в качестве параметра указать сам элемент, то Ложь.

Описание:

Определяет подчиненность элемента плана видов характеристик группе с учетом всех уровней иерархии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определяет подчиненность группе непосредственно на момент вызова метода путем последовательного считывания всех вышестоящих элементов.

Доступен, начиная с версии 8.0.
