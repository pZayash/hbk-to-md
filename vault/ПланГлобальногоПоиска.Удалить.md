ПланГлобальногоПоиска (GlobalSearchPlan)

Удалить (Delete)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента в коллекции.

Вариант синтаксиса: По элементу

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементПланаГлобальногоПоиска](ЭлементПланаГлобальногоПоиска.md).   
Удаляемый элемент.

Вариант синтаксиса: По виду поиска

Синтаксис:

Удалить(<ВидПоиска>)

Параметры:

<ВидПоиска> (обязательный)

Тип: [СтандартныйВидГлобальногоПоиска](СтандартныйВидГлобальногоПоиска.md).   
Вид поиска.

Вариант синтаксиса: По обработчику

Синтаксис:

Удалить(<ИмяПроцедуры>, <Модуль>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: Строка.   
Имя процедуры общего модуля.

<Модуль> (обязательный)

Тип: Строка.   
Имя общего модуля.

Описание:

Удаляет элемент из коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
