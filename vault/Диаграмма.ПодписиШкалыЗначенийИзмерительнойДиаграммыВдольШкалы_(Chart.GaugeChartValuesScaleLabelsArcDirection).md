Диаграмма (Chart)

ПодписиШкалыЗначенийИзмерительнойДиаграммыВдольШкалы (GaugeChartValuesScaleLabelsArcDirection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет ориентацию подписей к шкале измерительной диаграммы вдоль шкалы (по окружности).  
[Истина](lang__def_BooleanTrue.md) - подписи ориентированы вдоль шкалы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
