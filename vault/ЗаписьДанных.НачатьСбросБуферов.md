ЗаписьДанных (DataWriter)

НачатьСбросБуферов (BeginFlush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьСбросБуферов(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает сбрасывать все внутренние буферы в целевой поток, после чего будет вызван метод [СброситьБуферы](Поток.СброситьБуферы.md) целевого потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [СброситьБуферы](ЗаписьДанных.СброситьБуферы.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [СброситьБуферыАсинх](ЗаписьДанных.СброситьБуферыАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
