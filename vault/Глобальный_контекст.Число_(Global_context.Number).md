# Глобальный контекст.Число

Глобальный контекст (Global context)

Число (Number)

Доступен, начиная с версии 8.0.

Синтаксис:

Число(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Булево](lang__Булево_(Boolean).md).   
Исходное значение.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Полученное значение.

Описание:

Преобразует полученный параметр в значение типа [Число](lang__def_Number.md).  
Значение типа [Строка](lang__def_String.md) преобразуется в число в том случае, если параметр является строковым представлением литерала численного типа. При этом строковое представление анализируется как с учетом региональных установок информационной базы, так и без их учета. В противном случае будет выдано сообщение об ошибке выполнения функции.  
Параметр типа [Булево](lang__Булево_(Boolean).md) преобразуется в число по следующим правилам:

- [Ложь](lang__def_BooleanFalse.md) преобразуется в 0;
- [Истина](lang__def_BooleanTrue.md) преобразуется в 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| КонтрольноеЧисло = Число(Прав(СтраховойНомер,2)); |

Использование в версии:

Доступен, начиная с версии 8.0.
