СлойГеографическойСхемы (GeographicalSchemaLayer)

ВыбратьОбъекты (SelectObjects)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьОбъекты(<УсловиеОтбора>)

Параметры:

<УсловиеОтбора> (обязательный)

Тип: [Структура](Структура.md).   
Структура, в которой передается условие отбора. Ключ - имя серии. Значение - значение отбора.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Формирует выборку объектов в слое по переданному условию, проверяя его на равенство.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Структура = Новый Структура; Структура.Вставить(СерияСлоя.Имя, ЗначениеПоиска) МассивОбъектов = Слой.ВыбратьОбъекты(Структура); |

Использование в версии:

Доступен, начиная с версии 8.0.
