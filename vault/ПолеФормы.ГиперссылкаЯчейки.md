ПолеФормы (FormField)

ГиперссылкаЯчейки (CellHyperlink)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Истина - в колонке таблицы текст в ячейках отображает как гиперссылка.   
Выбор ячейки с установленным свойством равнозначен нажатию клавиши Enter.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
