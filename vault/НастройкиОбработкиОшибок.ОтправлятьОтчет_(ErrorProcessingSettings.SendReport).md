НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОтчет (SendReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ОтправлятьОтчетНаКлиенте](НастройкиОбработкиОшибок.ОтправлятьОтчетНаКлиенте_(ErrorProcessingSettings.SendReportOnClient).md)
- [ОтправлятьОтчетНаСервере](НастройкиОбработкиОшибок.ОтправлятьОтчетНаСервере_(ErrorProcessingSettings.SendReportOnServer).md)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость отправки отчета.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

Использование в версии:

Доступен, начиная с версии 8.3.20.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
