Расширение колонки набора записей регистра расчета (Extension of the calculation register recordset table box column)

ОтображатьСтандартнуюКартинку (DisplayDefaultPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения в колонке стандартной картинки.

Доступность:

Толстый клиент.

Примечание:

Если [Истина](lang__def_BooleanTrue.md), то при выводе свойство [КартинкиСтрок](КолонкаТабличногоПоля.КартинкиСтрок_(TableBoxColumn.RowsPictures).md) колонки табличного поля не используется.

Использование в версии:

Доступен, начиная с версии 8.0.
