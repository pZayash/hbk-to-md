АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

СоздатьРазрешенныйИнтернетРесурс (CreateAllowedInternetResource)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьРазрешенныйИнтернетРесурс()

Возвращаемое значение:

Тип: [РазрешенныйИнтернетРесурс](РазрешенныйИнтернетРесурс.md).   

Описание:

Создает новый разрешенный ресурс сети Интернет для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
