# СписокПолнотекстовогоПоиска.НачальнаяПозиция

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Полнотекстовый поиск](Полнотекстовый_поиск.md) › [СписокПолнотекстовогоПоиска](СписокПолнотекстовогоПоиска_(FullTextSearchList).md) › [Методы](СписокПолнотекстовогоПоиска__Методы.md)

СписокПолнотекстовогоПоиска (FullTextSearchList)

НачальнаяПозиция (StartPosition)

Доступен, начиная с версии 8.1.

Синтаксис:

НачальнаяПозиция()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Начальная позиция поиска. После вызова метода [ПерваяЧасть](СписокПолнотекстовогоПоиска.ПерваяЧасть_(FullTextSearchList.FirstPart).md) равна 0. При вызове методов [СледующаяЧасть](СписокПолнотекстовогоПоиска.СледующаяЧасть_(FullTextSearchList.NextPart).md), [ПредыдущаяЧасть](СписокПолнотекстовогоПоиска.ПредыдущаяЧасть_(FullTextSearchList.PreviousPart).md), начальная позиция будет меняться.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
