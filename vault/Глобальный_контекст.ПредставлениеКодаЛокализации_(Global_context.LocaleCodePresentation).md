Глобальный контекст (Global context)

ПредставлениеКодаЛокализации (LocaleCodePresentation)

Доступен, начиная с версии 8.2.

Синтаксис:

ПредставлениеКодаЛокализации(<КодЛокализации>)

Параметры:

<КодЛокализации> (обязательный)

Тип: [Строка](lang__def_String.md).   
Код локализации, для которого должно быть получено представление.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Локализованное наименование языка с заданным идентификатором.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьДопустимыеКодыЛокализации](Глобальный_контекст.ПолучитьДопустимыеКодыЛокализации_(Global_context.GetAvailableLocaleCodes).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
