МенеджерРаботыСРечью (SpeechProcessingManager)

ОстановитьПотоковоеРаспознавание (StopStreamingSpeechToText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОстановитьПотоковоеРаспознавание(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор потокового распознавания.  
Если не указан, то останавливаются все сессии потокового распознавания.

Описание:

Останавливает сессию потокового распознавания по ключу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
