АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

СоздатьРазрешенныйВиртуальныйКаталог (CreateAllowedVirtualDirectory)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьРазрешенныйВиртуальныйКаталог()

Возвращаемое значение:

Тип: [РазрешенныйВиртуальныйКаталог](РазрешенныйВиртуальныйКаталог.md).   

Описание:

Создает новый виртуальный каталог для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
