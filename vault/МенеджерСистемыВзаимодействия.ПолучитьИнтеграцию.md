МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьИнтеграцию (GetIntegration)

Доступен, начиная с версии 8.3.17.

Вариант синтаксиса: По идентификатору

Синтаксис:

ПолучитьИнтеграцию(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия.md).   
Идентификатор интеграции.

Вариант синтаксиса: По ключу

Синтаксис:

ПолучитьИнтеграцию(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Ключ интеграции.

Возвращаемое значение:

Тип: [ИнтеграцияСистемыВзаимодействия](ИнтеграцияСистемыВзаимодействия.md), Неопределено.   

Описание:

Получает интеграцию по идентификатору или ключу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.23.
