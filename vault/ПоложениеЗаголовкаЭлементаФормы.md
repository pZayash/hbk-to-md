Описание:

Содержит варианты положения подчиненного элемента относительно элемента формы. В зависимости от вида элемента и его размещения, не все варианты могут отрабатываться.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
