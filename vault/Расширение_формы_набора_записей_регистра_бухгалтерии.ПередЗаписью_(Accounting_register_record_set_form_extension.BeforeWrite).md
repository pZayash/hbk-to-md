Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи набора из формы. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.ПередЗаписью_(AccountingRegisterRecordSet.Accounting_register_name.BeforeWrite).md) объекта [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
