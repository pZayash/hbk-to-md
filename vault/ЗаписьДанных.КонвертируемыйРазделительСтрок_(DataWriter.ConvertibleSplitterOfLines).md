ЗаписьДанных (DataWriter)

КонвертируемыйРазделительСтрок (ConvertibleSplitterOfLines)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Конвертируемый разделитель строк. Этот параметр влияет на поведение метода [ЗаписатьСимволы](ЗаписьДанных.ЗаписатьСимволы_(DataWriter.WriteChars).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [ЗаписатьСимволы](ЗаписьДанных.ЗаписатьСимволы_(DataWriter.WriteChars).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
