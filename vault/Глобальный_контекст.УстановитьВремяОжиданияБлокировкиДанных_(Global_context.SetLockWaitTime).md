Глобальный контекст (Global context)

УстановитьВремяОжиданияБлокировкиДанных (SetLockWaitTime)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьВремяОжиданияБлокировкиДанных(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Время ожидания блокировки данных (в секундах).  
Значение по умолчанию: 20.

Описание:

Устанавливает время ожидания блокировки данных (в секундах).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.  
Изменение времени ожидания блокировки данных действует сразу после выполнения для всех СУБД, кроме IBM DB2. На IBM DB2 после изменения времени ожидания блокировки данных требуется перезапуск СУБД.

Использование в версии:

Доступен, начиная с версии 8.1.
