ПеречислимыеСвойстваОбъектовМетаданных (MetadataObjectEnumeratedProperties)

АвтоПрефиксНомераЗадачи (TaskNumberAutoPrefix)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [АвтоПрефиксНомераЗадачи](АвтоПрефиксНомераЗадачи.md).   
Если выбрано значение [НомерБизнесПроцесса](АвтоПрефиксНомераЗадачи.НомерБизнесПроцесса.md), то префикс номера задачи формируется автоматически из номера соответствующего ей бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
