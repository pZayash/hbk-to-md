**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Порядок](Порядок-3.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки порядка.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
