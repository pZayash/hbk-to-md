МенеджерКалендарей (CalendarsManager)

ПолучитьСобытие (GetEvent)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьСобытие(<Событие>)

Параметры:

<Событие> (обязательный)

Тип: [ЛокальныйКлючСобытияКалендаря](ЛокальныйКлючСобытияКалендаря_(LocalCalendarEventKey).md).   
Локальный ключ события календаря.

Возвращаемое значение:

Тип: [ДанныеСобытияКалендаряУчетнойЗаписи](ДанныеСобытияКалендаряУчетнойЗаписи_(AccountCalendarEventData).md).   

Описание:

Получает данные события по локальному ключу.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
