ДанныеАдреса (AddressData)

СтрокаАдресаДополнительная (AddressLine2)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текстовое представление номера дома.  
Свойство не используется в свойствах объекта [ДанныеКонтакта](ДанныеКонтакта_(ContactData).md).  
В методе [ПолучитьМестоположениеПоАдресу](Глобальный_контекст.ПолучитьМестоположениеПоАдресу_(Global_context.GetLocationByAddress).md) имеет приоритет перед свойством [СтрокаАдреса](ДанныеАдреса.СтрокаАдреса_(AddressData.AddressLine).md).

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
