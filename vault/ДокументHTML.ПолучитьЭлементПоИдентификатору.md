ДокументHTML (HTMLDocument)

ПолучитьЭлементПоИдентификатору (GetElementById)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЭлементПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: Строка.   
Идентификатор искомого элемента.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML.md), Неопределено.   
Узел, у которого имеется атрибут со свойством [ЯвляетсяИдентификатором](АтрибутHTML.ЯвляетсяИдентификатором.md), установленным в Истина.

Описание:

Возвращает элемент по идентификатору.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
