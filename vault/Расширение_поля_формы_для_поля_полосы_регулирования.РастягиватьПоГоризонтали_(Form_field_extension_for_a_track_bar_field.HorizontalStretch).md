Расширение поля формы для поля полосы регулирования (Form field extension for a track bar field)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает возможность растягивания поля полосы регулирования по горизонтали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
