ВидГруппыФормы (FormGroupType)

Страница (Page)

Доступен, начиная с версии 8.2.

Описание:

Страница панели. Группа данного вида может содержать тот же набор элементов, что и группа вида [ОбычнаяГруппа](ВидГруппыФормы.ОбычнаяГруппа_(FormGroupType.UsualGroup).md).

Использование в версии:

Доступен, начиная с версии 8.2.
