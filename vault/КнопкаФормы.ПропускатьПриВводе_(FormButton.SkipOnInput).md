# КнопкаФормы.ПропускатьПриВводе

КнопкаФормы (FormButton)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Указывает необходимость пропуска поля при вводе данных.  
Если [Неопределено](lang__def_Undefined.md), то это [Истина](lang__def_BooleanTrue.md) для кнопки по умолчанию и [Ложь](lang__def_BooleanFalse.md) для всех остальных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
