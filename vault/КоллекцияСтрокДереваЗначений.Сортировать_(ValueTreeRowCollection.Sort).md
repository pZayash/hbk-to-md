КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Сортировать (Sort)

Доступен, начиная с версии 8.0.

Синтаксис:

Сортировать(<СтрокаСортировки>, <ВключатьПодчиненные>, <ОбъектСравнения>)

Параметры:

<СтрокаСортировки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Список имен колонок, разделенных запятыми, по которым производится сортировка таблицы. После каждого имени колонки через пробел может быть указано направление упорядочивания. Направление определяется: "Убыв" ("Desc") - упорядочивать по убыванию; "Возр" ("Asc") - упорядочивать по возрастанию. По умолчанию сортировка таблица производится по возрастанию. Порядок указания имен колонок таблицы определяет порядок сортировки. Это означает, что сначала таблица сортируется по колонке, указанной первой. Затем группы строк с одинаковым значением в этой колонке сортируются по колонке, которая указана второй, и так далее.

<ВключатьПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли отсортированы коллекции подчиненных строк (если таковые имеются). [Истина](lang__def_BooleanTrue.md) - коллекции будут отсортированы.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ОбъектСравнения> (необязательный)

Тип: [СравнениеЗначений](СравнениеЗначений_(CompareValues).md).   
Объект для сравнения элементов. Если объект сравнения не задан, сравнение выполняется по следующим правилам:

- элементы, чьи типы не совпадают, сравниваются по коду типа;
- элементы простых типов сравниваются по значению;
- элементы остальных типов сравниваются по строковому представлению.

Описание:

Сортирует строки коллекции в соответствии с указанными правилами сортировки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СтрокиСостава.Сортировать("Норма Убыв", Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
