Описание:

Содержит свойства, методы и стандартные параметры объекта [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md), применимые для формы плана видов характеристик.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
