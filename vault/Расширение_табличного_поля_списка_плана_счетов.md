Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка счетов. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ПланСчетовСписок.](ПланСчетовСписок.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСписок.Имя_плана_счетов.md)>.
