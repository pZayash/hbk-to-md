ФабрикаXDTO (XDTOFactory)

На основе набора схем XML

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ФабрикаXDTO(<НаборСхемXML>, <Пакеты>)

Параметры:

<НаборСхемXML> (необязательный)

Тип: [НаборСхемXML](НаборСхемXML_(XMLSchemaSet).md).   
Набор схем XML, на основе которого будет создана фабрика XDTO.

<Пакеты> (необязательный)

Тип: [Массив](Массив_(Array).md), [КоллекцияПакетовXDTO](КоллекцияПакетовXDTO_(XDTOPackageCollection).md).   
Коллекция пакетов XDTO, которые необходимо добавить в фабрику XDTO. Если в наборе схем присутствует схема с URI пространства имен, соответствующему пакету XDTO из коллекции пакетов, то будет использован имеющийся пакет, а не создан новый.

Описание:

Создает фабрику типов XDTO по набору схем XML.

Использование в версии:

Доступен, начиная с версии 8.1.
