РежимРедактированияЗначенийДиаграммы (ChartValuesEditMode)

Авто (Auto)

Доступен, начиная с версии 8.3.16.

Описание:

Для диаграммы используется значение [НеИспользовать](РежимРедактированияЗначенийДиаграммы.НеИспользовать_(ChartValuesEditMode.DontUse).md).

Доступен, начиная с версии 8.3.16.
