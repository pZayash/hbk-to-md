**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Свойства](Свойства.md)

Глобальный контекст (Global context)

НавигационнаяСсылкаЗапуска (LaunchURL)

Не рекомендуется использовать, начиная с версии 8.3.18.

Рекомендуется использовать:

- [ДанныеПереходаПоНавигационнойСсылкеЗапуска](Глобальный_контекст.ДанныеПереходаПоНавигационнойСсылкеЗапуска_(Global_context.LaunchURLNavigationData).md)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [ДанныеПереходаПоНавигационнойСсылке](ДанныеПереходаПоНавигационнойСсылке_(URLNavigationData).md), [Неопределено](lang__def_Undefined.md).   
Используется для доступа к ссылке, по которой был произведен запуск приложения.   
Свойство доступно во всех клиентских обработчиках событий, начиная с обработчика события [ПередНачаломРаботыСистемы](Глобальный_контекст.ПередНачаломРаботыСистемы_(Global_context.BeforeStart).md).  
В мобильной платформе помимо внешней ссылки запуск может быть осуществлен с помощью ассоциативной ссылки со специальной схемой (maps://, mailto://, tel:, и т.д.).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.18.
