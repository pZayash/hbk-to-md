МенеджерГлобальногоПоиска (GlobalSearchManager)

ПолучитьПодсказкуВвода (GetInputHint)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьПодсказкуВвода()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает подсказку поля ввода, установленную при вызове метода [УстановитьПодсказкуВвода](МенеджерГлобальногоПоиска.УстановитьПодсказкуВвода.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
