ПолеВвода (TextBox)

ВыборНезаполненного (ChoiceIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Свойство указывает, нужно ли сейчас в поле в нужный момент выполнять автовыбор значения или нет. Если свойство имеет значение Истина, то независимо от хранимых в поле ввода данных будет выполняться автовыбор значения, если Ложь - то независимо от хранимых в поле ввода данных автовыбор выполняться не будет.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
