ВнешнийИсточникДанныхКубНаборЗаписей.<Имя внешнего источника>.<Имя куба> (ExternalDataSourceCubeRecordSet.<External source name>.<Cube name>)

Получить (Get)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс записи.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхКубЗапись.\<Имя внешнего источника\>.\<Имя куба\>](ВнешнийИсточникДанныхКубЗапись.Имя_внешнего_источника.Имя_куба_(ExternalDataSourceCubeRecord.External_source_name.Cube_name).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.3.3.
