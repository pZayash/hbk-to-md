# РезультатЧтенияДанных.МаркерНайден

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с двоичными данными](Работа_с_двоичными_данными-2.md) › [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md) › [Свойства](РезультатЧтенияДанных__Свойства.md)

РезультатЧтенияДанных (ReadDataResult)

МаркерНайден (MarkerFound)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак обнаружения маркера:

- [Истина](lang__def_BooleanTrue.md) - если в процессе чтения данных был обнаружен маркер.
- [Ложь](lang__def_BooleanFalse.md) - если маркер не был найден или операция не предполагала поиска маркера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
