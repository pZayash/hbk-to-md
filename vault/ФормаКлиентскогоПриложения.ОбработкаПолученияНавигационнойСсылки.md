ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаПолученияНавигационнойСсылки (URLGetProcessing)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ОбработкаПолученияНавигационнойСсылки(<Ключ>, <Представление>, <НавигационнаяСсылка>, <СтандартнаяОбработка>)

Параметры:

<Ключ>

Тип: Произвольный.   
В этом параметре передается значение свойства [Ключ](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.Ключ.md) выбранного элемента списка ссылок, полученного из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок.md). Если СписокНавигационыхСсылок, полученный из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок.md), пуст, то <Ключ> содержит значение Неопределено.

<Представление>

Тип: Строка.   
Представление ссылки, полученное из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок.md) или введенное пользователем. Представление может быть необходимо для формирования навигационной ссылки одним из методов глобального контекста или одного израсширений формы.

<НавигационнаяСсылка>

Тип: Строка.   
Параметр, в который требуется поместить ссылку.  
Полученная ссылка будет обработана (недопустимые символы будут закодированы) и в дальнейшем может быть использована для копирования в буфер или для помещения в избранное.

<СтандартнаяОбработка>

Тип: Строка.   
В данный параметр передается признак выполнения стандартной (системной) обработки события.  
<СтандартнаяОбработка> имеет значение Истина в одном из случаев:

- если в параметре <Ключ> один из стандартных вариантов,
- в параметре <Ключ> - Неопределено и <АвтоНавигационнаяСсылка> установлено в значение Ложь.

<СтандартнаяОбработка> имеет значение Ложь в одном из случаев:

- если в параметре <Ключ> один из вариантов, добавленных в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок.md);
- в параметре Ключ Неопределено и <АвтоНавигационнаяСсылка> имеет значение Истина.

Описание:

Вызывается при выборе в диалоге получения навигационной ссылки или добавления в избранное одного из вариантов ссылки, как добавленного в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок.md), так и стандартного.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
