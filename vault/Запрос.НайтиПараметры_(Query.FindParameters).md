Запрос (Query)

НайтиПараметры (FindParameters)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПараметры()

Возвращаемое значение:

Тип: [ОписаниеПараметровЗапроса](ОписаниеПараметровЗапроса_(QueryParametersDescription).md).   
Если не найдено ни одного параметра, то коллекция будет пустая.

Описание:

Осуществляет поиск параметров, которые присутствуют в тексте запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
