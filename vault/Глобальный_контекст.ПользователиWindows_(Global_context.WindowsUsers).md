# Глобальный контекст.ПользователиWindows

Глобальный контекст (Global context)

ПользователиWindows (WindowsUsers)

Не рекомендуется использовать, начиная с версии 8.2.

Рекомендуется использовать:

- [ПользователиОС](Глобальный_контекст.ПользователиОС_(Global_context.OSUsers).md)

Доступен, начиная с версии 8.2.

Синтаксис:

ПользователиWindows()

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [Массив](Массив_(Array).md).   
Информация возвращается в виде таблицы значений, в колонках которой содержится следующая информация:

- ИмяДомена (DomainName) - имя домена (тип [Строка](lang__def_String.md));
- ИмяСервера (ServerName) - имя компьютера, являющегося контроллером домена (тип [Строка](lang__def_String.md));
- Пользователи (Users) - массив строк с именами пользователей домена;
- Локальный (Local) - признак того, что этому домену принадлежат локальные пользователи данного компьютера.

Описание:

Предоставляет доступ к информации о доменах и пользователях операционной системы.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

При использовании в тонком клиенте, информация возвращается в виде массива структур со свойствами (аналогичными колонкам возвращаемой таблицы значений).

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.2.
