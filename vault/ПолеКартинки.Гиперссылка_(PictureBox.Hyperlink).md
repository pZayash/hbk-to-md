ПолеКартинки (PictureBox)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что поле картинки является гиперссылкой. Если значение свойства [Истина](lang__def_BooleanTrue.md), то при щелчке мышью в поле картинки возникает событие [Нажатие](ПолеКартинки.Нажатие_(PictureBox.Click).md).

Доступность:

Толстый клиент.

См. также:

[ПолеКартинки](ПолеКартинки_(PictureBox).md), событие [Нажатие](ПолеКартинки.Нажатие_(PictureBox.Click).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
