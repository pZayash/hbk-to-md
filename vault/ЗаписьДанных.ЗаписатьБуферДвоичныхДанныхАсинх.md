ЗаписьДанных (DataWriter)

ЗаписатьБуферДвоичныхДанныхАсинх (WriteBinaryDataBufferAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Запись буфера

Синтаксис:

ЗаписатьБуферДвоичныхДанныхАсинх(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер двоичных данных, все байты которого будут записаны в целевой поток.

Описание варианта метода:

Записывает всё содержимое заданного буфера в целевой поток.

Вариант синтаксиса: Запись части байтов из буфера

Синтаксис:

ЗаписатьБуферДвоичныхДанныхАсинх(<Буфер>, <ПозицияВБуфере>, <Количество>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер двоичных данных, который используется в качестве источника данных для записи в целевой поток.

<ПозицияВБуфере> (обязательный)

Тип: Число.   
Позиция в буфере, начиная с которой выполняется чтение байтов для записи в целевой поток.

<Количество> (обязательный)

Тип: Число.   
Количество байтов, которые требуется записать в целевой поток.

Описание варианта метода:

Записывает заданное количество байтов из заданного буфера, начиная с заданной позиции.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Записывает байты из буфера двоичных данных в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.
