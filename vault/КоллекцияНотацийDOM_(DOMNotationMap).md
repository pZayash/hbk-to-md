# КоллекцияНотацийDOM

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[НотацияDOM](НотацияDOM_(DOMNotation).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются узлы.  
Возможно обращение к узлу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияНотацийDOM.Количество_(DOMNotationMap.Count).md)  
[ПолучитьИменованныйЭлемент (GetNamedItem)](КоллекцияНотацийDOM.ПолучитьИменованныйЭлемент_(DOMNotationMap.GetNamedItem).md)  
[УдалитьИменованныйЭлемент (RemoveNamedItem)](КоллекцияНотацийDOM.УдалитьИменованныйЭлемент_(DOMNotationMap.RemoveNamedItem).md)  
[УстановитьИменованныйЭлемент (SetNamedItem)](КоллекцияНотацийDOM.УстановитьИменованныйЭлемент_(DOMNotationMap.SetNamedItem).md)  
[Элемент (Item)](КоллекцияНотацийDOM.Элемент_(DOMNotationMap.Item).md)  

Описание:

Коллекция узлов нотаций DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), свойство [Нотации](ОпределениеТипаДокументаDOM.Нотации_(DOMDocumentType.Notations).md)
