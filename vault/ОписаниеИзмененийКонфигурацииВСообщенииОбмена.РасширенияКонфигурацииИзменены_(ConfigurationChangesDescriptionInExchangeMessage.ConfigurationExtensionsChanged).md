ОписаниеИзмененийКонфигурацииВСообщенииОбмена (ConfigurationChangesDescriptionInExchangeMessage)

РасширенияКонфигурацииИзменены (ConfigurationExtensionsChanged)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что чтение сообщения приводит к изменению состава расширений конфигурации, т.е. одно или несколько расширений были изменены, добавлены, удалены или исключены из участия в распределенной информационной базе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
