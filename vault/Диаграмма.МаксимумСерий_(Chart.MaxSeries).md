Диаграмма (Chart)

МаксимумСерий (MaxSeries)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [МаксимумСерий](МаксимумСерий_(MaxSeries).md).   
Определяет режим ограничения максимального количества отображаемых серий диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Диаграмма](Диаграмма_(Chart).md), свойство [МаксимумСерийКоличество](Диаграмма.МаксимумСерийКоличество_(Chart.MaxSeriesCount).md)  
[Диаграмма](Диаграмма_(Chart).md), свойство [МаксимумСерийПроцент](Диаграмма.МаксимумСерийПроцент_(Chart.MaxSeriesPercent).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
