ДанныеКонтакта (ContactData)

НомераТелефонов (PhoneNumbers)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив номеров телефонов в виде объектов типа [ЭлементДанныхКонтакта](ЭлементДанныхКонтакта_(ContactDataItem).md).   
Свойство [ТипДанных](ЭлементДанныхКонтакта.ТипДанных_(ContactDataItem.DataType).md) содержит значение перечисления [ТипНомераТелефонаДанныхКонтакта](ТипНомераТелефонаДанныхКонтакта_(ContactDataPhoneNumberType).md) или строку. Свойство [Значение](ЭлементДанныхКонтакта.Значение_(ContactDataItem.Value).md) содержит строку с номером телефона.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
