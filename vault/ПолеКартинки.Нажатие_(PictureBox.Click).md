ПолеКартинки (PictureBox)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на поле картинки, если у него свойство [Гиперссылка](ПолеКартинки.Гиперссылка_(PictureBox.Hyperlink).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.

См. также:

[ПолеКартинки](ПолеКартинки_(PictureBox).md), свойство [Гиперссылка](ПолеКартинки.Гиперссылка_(PictureBox.Hyperlink).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
