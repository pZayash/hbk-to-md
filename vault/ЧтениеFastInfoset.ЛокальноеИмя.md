ЧтениеFastInfoset (FastInfosetReader)

ЛокальноеИмя (LocalName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит локальное имя для текущего узла XML. Имя зависит от типа текущего узла:

- [Атрибут](ТипУзлаXML.Атрибут.md) - локальное имя атрибута;
- [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента.md) - имя типа документа;
- [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md) - локальное имя элемента;
- [КонецЭлемента](ТипУзлаXML.КонецЭлемента.md) - локальное имя элемента;
- [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность.md) - имя ссылки на сущность;
- [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки.md) - адрес инструкции обработки;
- [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML.md) - строка xml.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для узлов, для которых локальное имя не определено, – пустая строка.

Использование в версии:

Доступен, начиная с версии 8.1.
