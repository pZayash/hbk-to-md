Диаграмма (Chart)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначение(<Точка>, <Серия>)

Параметры:

<Точка> (обязательный)

Тип: Число, [ТочкаДиаграммы](ТочкаДиаграммы.md).   
Индекс точки либо значение сама точка диаграммы.

<Серия> (обязательный)

Тип: Число, [СерияДиаграммы](СерияДиаграммы.md).   
Индекс серии либо сама серия диаграммы.

Возвращаемое значение:

Тип: [ЗначениеДиаграммы](ЗначениеДиаграммы.md).   
Значение диаграммы в заданной точке и серии.

Описание:

Получает значение из заданной точки и серии диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ОбъемПродаж = Диаграмма.ПолучитьЗначение(Точка, Серия).Значение; |

См. также:

[Диаграмма](Диаграмма.md), метод [УстановитьЗначение](Диаграмма.УстановитьЗначение.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
