ОбъектМетаданных: WebSocketКлиент (MetadataObject: WebSocketClient)

Пароль (Password)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Пароль пользователя, который используется для стандартной аутентификации или аутентификации ОС на WebSocket-сервере.  
Значение по умолчанию: пустая строка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
