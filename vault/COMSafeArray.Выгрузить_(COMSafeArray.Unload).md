COMSafeArray (COMSafeArray)

Выгрузить (Unload)

Доступен, начиная с версии 8.0.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Создает массив значений и копирует в него содержимое [COMSafeArray](COMSafeArray_(COMSafeArray).md).  
Если [COMSafeArray](COMSafeArray_(COMSafeArray).md) имеет более чем одно измерение, то элементами полученного массива значений будут вложенные массивы значений, и так от старшего индекса до самого младшего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| МассивЗначений = Массив.Выгрузить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
