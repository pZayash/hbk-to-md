ОписаниеИзмененийКонфигурацииВСообщенииОбмена (ConfigurationChangesDescriptionInExchangeMessage)

ИзмененныеМетаданныеРасширенийКонфигурацииИзменяютСтруктуруДанных (ChangedConfigurationExtensionsMetadataModifiesDataStructure)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что добавленные или удаленные расширения влияют на структуру данных, либо изменения расширений касаются объектов влияющих на структуру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
