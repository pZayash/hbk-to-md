МенеджерРаботыСРечью (SpeechProcessingManager)

ОтменитьОтложенноеРаспознавание (CancelDelayedSpeechToText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОтменитьОтложенноеРаспознавание(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [ИдентификаторОтложенногоРаспознаванияРечи](ИдентификаторОтложенногоРаспознаванияРечи_(SpeechToTextDelayedID).md).   
Ключ задания отложенного распознавания.

Описание:

Отменяет отложенное распознавание речи по ключу.  
При отмене для объекта [РезультатОтложенногоРаспознаванияРечи](РезультатОтложенногоРаспознаванияРечи_(DelayedSpeechToTextResult).md) устанавливаются следующие значения свойств:

- [Завершено](РезультатОтложенногоРаспознаванияРечи.Завершено_(DelayedSpeechToTextResult.Completed).md) - [Истина](lang__def_BooleanTrue.md),
- [Успешно](РезультатОтложенногоРаспознаванияРечи.Успешно_(DelayedSpeechToTextResult.Success).md) - [Ложь](lang__def_BooleanFalse.md),
- [Ошибка](РезультатОтложенногоРаспознаванияРечи.Ошибка_(DelayedSpeechToTextResult.Error).md) - "Распознавание отменено".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
