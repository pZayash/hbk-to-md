ОтображениеЗначенийСводнойДиаграммы (PivotChartValuesShowMode)

ВсеЗначения (AllValues)

Доступен, начиная с версии 8.0.

Описание:

Самостоятельно отображать все значения независимо от положения в иерархии.

Использование в версии:

Доступен, начиная с версии 8.0.
