Соединение с агентом сервера (IServerAgentConnection)

GetServiceDataDirectoriesForTransfer (GetServiceDataDirsForTransfer)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetServiceDataDirectoriesForTransfer(<Кластер>, <Сервер>, <ИмяСервиса>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого требуется получить список каталогов для переноса.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Сервер, для которого требуется получить список каталогов для переноса.

<ИмяСервиса> (обязательный)

Тип: Строка.   
Имя сервиса, для которого требуется получить список каталогов.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Элементы массива содержат имя сервиса, имя информационной базы, каталога, откуда надо перенести данные, и каталога, куда надо перенести данные. Если настройка применена, то данные для нее не отображаются в списке.

Описание:

Получает список каталогов для переноса данных сервисов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
