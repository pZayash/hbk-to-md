ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак выделения текста ячеек, первый значимый символ которого "-", специальным цветом стиля [ЦветОтрицательногоЧисла](ЦветаСтиля.ЦветОтрицательногоЧисла_(StyleColors.NegativeTextColor).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - текст выделяется.

Использование в версии:

Доступен, начиная с версии 8.2.
