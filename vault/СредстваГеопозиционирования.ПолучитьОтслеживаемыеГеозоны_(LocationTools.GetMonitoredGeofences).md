СредстваГеопозиционирования (LocationTools)

ПолучитьОтслеживаемыеГеозоны (GetMonitoredGeofences)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьОтслеживаемыеГеозоны()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [Геозона](Геозона_(Geofence).md).

Описание:

Получает список отслеживаемых геозон для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
