ЗащищенноеСоединениеКриптоПро (CryptoProSecureConnection)

СертификатыУдостоверяющихЦентров (CertificationAuthorityCertificates)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [СертификатыУдостоверяющихЦентровФайл](СертификатыУдостоверяющихЦентровФайл_(FileCertificationAuthorityCertificates).md), [СертификатыУдостоверяющихЦентровОС](СертификатыУдостоверяющихЦентровОС_(OSCertificationAuthorityCertificates).md), [Неопределено](lang__def_Undefined.md).   
Объект сертификатов удостоверяющих центров, если используются.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если [Неопределено](lang__def_Undefined.md) - сертификат сервера не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.24.
