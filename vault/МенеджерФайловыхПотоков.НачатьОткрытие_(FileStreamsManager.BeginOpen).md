# МенеджерФайловыхПотоков.НачатьОткрытие

МенеджерФайловыхПотоков (FileStreamsManager)

НачатьОткрытие (BeginOpen)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: С указанием режима открытия

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <Имя файла>, <РежимОткрытия>, <РазмерБуфера>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <ФайловыйПоток> - Файловый поток для файла с заданным именем.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Имя файла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла_(FileOpenMode).md).   
Режим открытия файла.

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера для операций с файлом.

Описание варианта метода:

Начинает открытие файла в заданном режиме с возможностью чтения и записи.   
Файл открывается в режиме разделяемого чтения.

Вариант синтаксиса: С указанием режимов открытия и доступа

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <РежимОткрытия>, <РежимДоступа>, <ИмяФайла>, <РазмерБуфера>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Поток> - Объект типа [ФайловыйПоток](ФайловыйПоток_(FileStream).md) для файла с указанным именем.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла_(FileOpenMode).md).   
Режим открытия файла.

<РежимДоступа> (обязательный)

Тип: [ДоступКФайлу](ДоступКФайлу_(FileAccess).md).   
Режим доступа к файлу.

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера для операций с файлом.

Описание варианта метода:

Открыть файл в выбранном режиме, с заданным уровнем доступа, с общим доступом на чтение.

Описание:

Открывает файл в заданном режиме с возможностью чтения и записи.   
Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

При вызове данного метода запрашивается разрешение пользователя на выполнение операции.

Использование в версии:

Доступен, начиная с версии 8.3.9.
