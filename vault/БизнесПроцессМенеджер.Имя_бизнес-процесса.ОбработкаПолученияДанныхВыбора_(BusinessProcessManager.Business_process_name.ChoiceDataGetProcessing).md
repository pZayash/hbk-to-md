# БизнесПроцессМенеджер.<Имя бизнес-процесса>.ОбработкаПолученияДанныхВыбора

БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ОбработкаПолученияДанныхВыбора (ChoiceDataGetProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаПолученияДанныхВыбора(<ДанныеВыбора>, <Параметры>, <СтандартнаяОбработка>)

Параметры:

<ДанныеВыбора>

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
В обработчике можно сформировать и передать в этом параметре данные для выбора. При этом параметр <СтандартнаяОбработка> должен быть установлен в [Ложь](lang__def_BooleanFalse.md).  
В элемент списка значений можно помещать не только конкретное значение, но и структуру.  
Анализ значения на предмет того, является ли она структурой или просто значением, используется полем ввода формы клиентского приложения в случаях:

- Выбора из списков, возвращаемых событиями поля ввода;
- Выбора из списка в поле ввода, содержащегося в свойстве [СписокВыбора](Расширение_поля_формы_для_поля_ввода.СписокВыбора_(Form_field_extension_for_a_text_box.ChoiceList).md).

В случае, если в значение элемента списка значений используется [Структура](Структура_(Structure).md), она должна иметь следующий состав свойств:

- Значение (Value) – собственно значение (обязательный элемент структуры);
- ПометкаУдаления (DeletionMark) – ([Булево](lang__Булево_(Boolean).md)) содержит признак пометки значения на удаление. Если [Истина](lang__def_BooleanTrue.md), то при выборе такого элемента из списка значений выдаст стандартное предупреждение;
- Предупреждение (Warning) – ([Строка](lang__def_String.md)) содержит текст предупреждения, которое будет показано при выборе элемента из списка значений.

<Параметры>

Тип: [Структура](Структура_(Structure).md).   
Содержит параметры выбора.  
Их можно дополнительно настроить в обработчике события.  
Структура, в которой обязательно должны содержаться следующие ключи:

- СтрокаПоиска (SearchString) - строка, используемая при поиске данных, [Неопределено](lang__def_Undefined.md) - осуществляется быстрый выбор;
- Отбор (Filter) - отбор, используемый при поиске данных;
- ВыборГруппИЭлементов (ChoiceFoldersAndItems) - указывается использование групп и элементов; тип [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов_(FoldersAndItemsUse).md); параметр может отсутствовать;
- СпособПоискаСтроки (StringSearchMode) - указывается способ поиска при вводе по строке; тип [СпособПоискаСтрокиПриВводеПоСтроке](СпособПоискаСтрокиПриВводеПоСтроке_(SearchStringModeOnInputByString).md); параметр может отсутствовать;
- ПолнотекстовыйПоиск (FullTextSearch) - указывается необходимость использования полнотекстового поиска при вводе по строке; тип - [ПолнотекстовыйПоискПриВводеПоСтроке](ПолнотекстовыйПоискПриВводеПоСтроке_(FullTextSearchOnInputByString).md); параметр может отсутствовать.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает на сервере перед стандартным формированием списка при вводе по строке, автоподборе текста и быстром выборе, а также при выполнении метода [ПолучитьДанныеВыбора](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьДанныеВыбора_(BusinessProcessManager.Business_process_name.GetChoiceData).md).

Доступность:

Сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
