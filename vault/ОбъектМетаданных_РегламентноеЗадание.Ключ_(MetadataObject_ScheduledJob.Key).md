ОбъектМетаданных: РегламентноеЗадание (MetadataObject: ScheduledJob)

Ключ (Key)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Ключ регламентного задания. Два регламентных задания с одинаковым значением ключа могут быть выполнены только последовательно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Максимальная длина 128 символов.

Использование в версии:

Доступен, начиная с версии 8.1.
