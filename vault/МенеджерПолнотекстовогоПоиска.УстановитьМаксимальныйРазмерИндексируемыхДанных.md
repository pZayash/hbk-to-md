МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

УстановитьМаксимальныйРазмерИндексируемыхДанных (SetMaxIndexedDataSize)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УстановитьМаксимальныйРазмерИндексируемыхДанных(<МаксимальныйРазмер>)

Параметры:

<МаксимальныйРазмер> (обязательный)

Тип: Число.   
Новое значение максимального размера индексируемых данных в байтах.   
Если равен нулю, индексируются все данные.

Описание:

Устанавливает максимальный размер индексируемых данных в байтах.  
Требует наличия у пользователя права "Администрирование". Отсутствие права приводит к возникновению исключения.  
Метод должен быть вызван из сеанса, в котором все разделители не используются.  
Данные, размер которых превышает <МаксимальныйРазмер>, не индексируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.8.
