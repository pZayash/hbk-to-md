ОбъектМетаданных: КаналСервисаИнтеграции (MetadataObject: IntegrationServiceChannel)

ВТранзакции (Transactioned)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если свойство установлено в значение Истина, то код процедуры, обрабатывающий полученное сообщение, будет выполняться в транзакции.  
Если при фиксации успешно выполненной обработки сообщения произойдет сбой, то успешно выполненный обработчик будет выполнен повторно. При разработке обработчика необходимо учитывать данную особенность.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
