Описание:

Предоставляет компоновщику настроек необходимую информацию: наборы доступных полей для различных использований, наборы объектов доступных для вложенной настройки, доступные параметры данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
