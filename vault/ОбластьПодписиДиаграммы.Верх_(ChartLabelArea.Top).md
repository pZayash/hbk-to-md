ОбластьПодписиДиаграммы (ChartLabelArea)

Верх (Top)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит положение верхней границы.   
Может принимать значения от 0 до 1.   
0 - привязана к верхнему краю, 1 - к нижнему.  
Используется вместе со свойством [УказываетсяЛевоИВерх](ПоложениеПодписейКДиаграмме.УказываетсяЛевоИВерх_(ChartLabelLocation.TopAndLeftSpecified).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ПоложениеПодписейКДиаграмме](ПоложениеПодписейКДиаграмме_(ChartLabelLocation).md), свойство [УказываетсяЛевоИВерх](ПоложениеПодписейКДиаграмме.УказываетсяЛевоИВерх_(ChartLabelLocation.TopAndLeftSpecified).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
