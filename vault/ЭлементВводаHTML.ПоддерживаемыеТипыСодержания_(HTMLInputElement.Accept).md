ЭлементВводаHTML (HTMLInputElement)

ПоддерживаемыеТипыСодержания (Accept)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Список типов MIME, разделенных запятыми, указывающий MIME тип переданного файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется только с типом(type)="file".

Использование в версии:

Доступен, начиная с версии 8.2.
