# КоллекцияВстроенныхТаблиц

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличный документ](Табличный_документ-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[СводнаяТаблица](СводнаяТаблица_(PivotTable).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются [СводнаяТаблица](СводнаяТаблица_(PivotTable).md).  
Возможно обращение к коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) или имя таблицы.

Описание:

Содержит коллекцию сводных таблиц, встроенных в табличный документ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), свойство [ВстроенныеТаблицы](ТабличныйДокумент.ВстроенныеТаблицы_(SpreadsheetDocument.EmbeddedTables).md)

Доступен, начиная с версии 8.0.
