ЭлементКнопкаHTML (HTMLButtonElement)

Тип (Type)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит тип кнопки. Возможные значения: submit (отправить форму), reset (установить элементы управления в начальные значения), push (для выполнения сценариев на стороне клиента).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
