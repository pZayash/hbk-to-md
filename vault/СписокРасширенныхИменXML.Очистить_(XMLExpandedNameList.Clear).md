# СписокРасширенныхИменXML.Очистить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md) › [СписокРасширенныхИменXML](СписокРасширенныхИменXML_(XMLExpandedNameList).md) › [Методы](СписокРасширенныхИменXML__Методы.md)

СписокРасширенныхИменXML (XMLExpandedNameList)

Очистить (Clear)

Доступен, начиная с версии 8.1.

Синтаксис:

Очистить()

Описание:

Удаляет все элементы коллекции расширенных имен XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
