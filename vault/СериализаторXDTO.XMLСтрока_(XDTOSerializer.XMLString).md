СериализаторXDTO (XDTOSerializer)

XMLСтрока (XMLString)

Доступен, начиная с версии 8.1.

Синтаксис:

XMLСтрока(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Неопределено](lang__def_Undefined.md), [Null](lang__def_Null.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md); значения перечислений, значения системных перечислений ([ВидСчета](ВидСчета_(AccountType).md), [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии_(AccountingRecordType).md), [ВидДвиженияНакопления](ВидДвиженияНакопления_(AccumulationRecordType).md), [ДопустимыйЗнак](ДопустимыйЗнак_(AllowedSign).md), [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md), [ЧастиДаты](ЧастиДаты_(DateFractions).md)) или все ссылки на объекты базы данных.   
Значение, для которого должно быть получено строковое представление, которое может быть использовано как текст элемента или значение атрибута XML. Как строка XML могут быть представлены значения следующих типов:  
[Null](lang__def_Null.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и все ссылки на объекты базы данных.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает XML представление значения для помещения в текст элемента или значение атрибута XML.  
XML представления различных типов платформы 1С:Предприятие 8:

- "Строка" - как есть;
- "Булево": "Истина" - "true", "Ложь" -"false";
- "Дата" - XML представление соответствует представлению типа dateTime схемы XML (см. <http://www.w3.org/TR/xmlschema-2/#dateTime>);
- "Число" - XML представление соответствует представлению типа decimal схемы XML (см. <http://www.w3.org/TR/xmlschema-2/#decimal>);
- Значение системного перечисления (для перечислений, поддерживающих XML сериализацию) - имя значения перечисления;
- "УникальныйИдентификатор" - строковое представление уникального идентификатора;
- "Ссылки" - XML представление уникального идентификатора ссылки;
- "ХранилиищеЗначений" и "ДвоичныеДанные" - строка в формате base64.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
