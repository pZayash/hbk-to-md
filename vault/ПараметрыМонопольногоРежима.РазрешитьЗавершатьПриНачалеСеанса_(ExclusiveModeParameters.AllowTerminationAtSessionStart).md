# ПараметрыМонопольногоРежима.РазрешитьЗавершатьПриНачалеСеанса

ПараметрыМонопольногоРежима (ExclusiveModeParameters)

РазрешитьЗавершатьПриНачалеСеанса (AllowTerminationAtSessionStart)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Текущий сеанс может быть завершен при начале нового сеанса пользователя с правом ЗавершениеМонопольногоРежимаПриНачалеСеанса.   
По умолчанию - [Ложь](lang__def_BooleanFalse.md).   
Значение [Истина](lang__def_BooleanTrue.md) может быть установлено, если выполняемое в монопольном режиме действие может быть прервано без опасности разрушения прикладных данных информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
