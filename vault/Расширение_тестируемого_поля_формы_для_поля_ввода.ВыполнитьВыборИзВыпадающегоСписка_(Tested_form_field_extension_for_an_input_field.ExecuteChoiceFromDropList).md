Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ВыполнитьВыборИзВыпадающегоСписка (ExecuteChoiceFromDropList)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ВыполнитьВыборИзВыпадающегоСписка(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md).   
Представление выбираемого значения или номер значения (индекс строки, нумерация с 0).

Описание:

Выбирает значение из открытого выпадающего списка по его представлению в списке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
