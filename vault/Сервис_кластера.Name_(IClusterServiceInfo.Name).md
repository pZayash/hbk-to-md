Сервис кластера (IClusterServiceInfo)

Name (Name)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Строковый идентификатор сервиса.   
Каждый сервис имеет постоянный идентификатор, не зависящий от текущих национальных настроек.   
Определены следующие идентификаторы сервисов:

- ClusterConfigService - сервис конфигурации кластера;
- ClusterStateService - сервис состояния кластера;
- SpeechToTextModelManagementService - сервис управления моделями распознавания речи;
- SpeechToTextService - сервис распознавания речи.

- AuxuliaryService - сервис вспомогательных функций кластера;
- ClientTestingService - сервис тестирования;
- CounterService - cервис мониторинга счетчиков потребления ресурсов;
- DataBaseConfigurationUpdateService - сервис обновления конфигурации базы данных;
- DatabaseTableNumberingService - сервис нумерации таблиц и полей базы данных;
- DataEditLockService - сервис блокировки объектов;
- DbCopiesService - сервис копий базы данных;
- DbCopiesTimeService - сервис времени копий базы данных;
- DebugService - сервис управления предметами отладки (TCP);
- EventLogService - сервис журналов регистрации;
- ExternalDataSourceODBCService - сервис работы с внешними источниками данных через ODBC;
- ExternalDataSourceXMLAService - сервис работы с внешними источниками данных через XMLA;
- ExternalSessionManagerService - сервис внешнего управления сеансами;
- FulltextSearchService - сервис полнотекстового поиска;
- IntegrationDataService - сервис интеграционных данных (обеспечивают функционирование сервисов интеграции);
- LicenseService - сервис лицензирования;
- NumerationService - сервис нумерации;
- OpenID2ProviderContextService - сервис провайдера OpenID2;
- DataAcceleratorService - сервис Дата акселератора;
- SessionDataService - сервис сеансовых данных;
- SessionReuseService - cервис повторного использования сеансов;
- SettingsService - сервис пользовательских настроек;
- TimestampService - сервис времени;
- TransactionLockService - сервис транзакционных блокировок;
- BinaryDataStorageService - сервис хранилища двоичных данных (только для 64-битной платформы).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.17.
