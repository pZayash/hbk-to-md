СредстваТелефонии (TelephonyTools)

ПодключитьОбработчикSMSСообщений (AttachSMSMessageHandler)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПодключитьОбработчикSMSСообщений(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md), [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Имя процедуры, подключаемой в качестве обработчика.

Описание:

Выполняет подключение обработчика входящих SMS-сообщений.  
При получении входящего сообщения вызывается обработчик со следующими параметрами:

- Сообщение: [SMSСообщение](SMSСообщение_(SMSMessage).md),
- Дополнительный параметр (если используется [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md)).

Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяПриемSMS](СредстваТелефонии.ПоддерживаетсяПриемSMS_(TelephonyTools.SMSReceivingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Если функциональность не поддерживается, будет выполнен возврат из метода без вызова обработчика.

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПоддерживаетсяПриемSMS](СредстваТелефонии.ПоддерживаетсяПриемSMS_(TelephonyTools.SMSReceivingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
