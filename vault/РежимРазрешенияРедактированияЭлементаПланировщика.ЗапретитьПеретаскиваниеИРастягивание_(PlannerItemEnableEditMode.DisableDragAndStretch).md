РежимРазрешенияРедактированияЭлементаПланировщика (PlannerItemEnableEditMode)

ЗапретитьПеретаскиваниеИРастягивание (DisableDragAndStretch)

Доступен, начиная с версии 8.3.18.

Описание:

Отключает возможности:

- перетаскивание элемента внутри поля планировщика,
- растягивание элемента планировщика.

Использование в версии:

Доступен, начиная с версии 8.3.18.
