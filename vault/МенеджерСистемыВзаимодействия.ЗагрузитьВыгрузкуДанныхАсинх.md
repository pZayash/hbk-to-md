МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ЗагрузитьВыгрузкуДанныхАсинх (LoadDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ЗагрузитьВыгрузкуДанныхАсинх(<ПараметрыДиалога>, <Описание>)

Параметры:

<ПараметрыДиалога> (необязательный)

Тип: [ПараметрыДиалогаПомещенияФайлов](ПараметрыДиалогаПомещенияФайлов.md).   
Параметры диалога загрузки выгрузки данных.

<Описание> (обязательный)

Тип: Строка.   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания – [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.md).  
Обещание выполняется при окончании загрузки резервной копии.

Описание:

Асинхронная функция. Загружает выгрузку данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ЗагрузитьВыгрузкуДанных](МенеджерСистемыВзаимодействия.ЗагрузитьВыгрузкуДанных.md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
