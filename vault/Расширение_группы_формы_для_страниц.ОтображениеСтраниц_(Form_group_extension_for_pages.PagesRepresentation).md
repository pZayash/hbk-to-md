Расширение группы формы для страниц (Form group extension for pages)

ОтображениеСтраниц (PagesRepresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеСтраницФормы](ОтображениеСтраницФормы_(FormPagesRepresentation).md).   
Содержит режим отображения закладок страниц.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

На мобильной платформе варианты [ЗакладкиСправаГоризонтально](ОтображениеСтраницФормы.ЗакладкиСправаГоризонтально_(FormPagesRepresentation.TabsOnRightHorizontal).md) и [ЗакладкиСлеваГоризонтально](ОтображениеСтраницФормы.ЗакладкиСлеваГоризонтально_(FormPagesRepresentation.TabsOnLeftHorizontal).md), интерпретируются как [ЗакладкиСверху](ОтображениеСтраницФормы.ЗакладкиСверху_(FormPagesRepresentation.TabsOnTop).md).

Использование в версии:

Доступен, начиная с версии 8.2.
