ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Формат (Format)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, задающую формат вывода данных в ячейках таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Формат](Глобальный_контекст.Формат_(Global_context.Format).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
