# ТекстDOM.ВставитьДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ТекстDOM](ТекстDOM_(DOMText).md) › [Методы](ТекстDOM__Методы.md)

ТекстDOM (DOMText)

ВставитьДанные (InsertData)

Доступен, начиная с версии 8.1.

Синтаксис:

ВставитьДанные(<Смещение>, <Строка>)

Параметры:

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение, по которому определяется позиция вставки символьных данных.

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, вставляемая в символьные данные.

Описание:

Вставляет строку в символьные данные.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных;
- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
