ВыраженияИтоговСхемыЗапроса (QuerySchemaTotalExpressions)

НайтиПоПолю (FindByField)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НайтиПоПолю(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки для поиска.

Возвращаемое значение:

Тип: [ВыражениеИтогаСхемыЗапроса](ВыражениеИтогаСхемыЗапроса_(QuerySchemaTotalExpression).md).   

Описание:

Выполняет поиск выражения по свойству [Поле](ВыражениеИтогаСхемыЗапроса.Поле_(QuerySchemaTotalExpression.Field).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
