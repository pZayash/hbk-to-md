# ОбъектМетаданных: Справочник.ОграничиватьКоличествоУровней

ОбъектМетаданных: Справочник (MetadataObject: Catalog)

ОграничиватьКоличествоУровней (LimitLevelCount)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то максимальное количество уровней иерархии ограничено значением свойства [КоличествоУровней](ОбъектМетаданных_Справочник.КоличествоУровней_(MetadataObject_Catalog.LevelCount).md). В противном случае максимальное количество уровней иерархии неограниченно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
