ОбластьЗаголовкаСводнойДиаграммы (PivotChartTitleArea)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона сводной диаграммы.

Сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ОбластьЗаголовкаСводнойДиаграммы.ПрозрачныйФон_(PivotChartTitleArea.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Доступен, начиная с версии 8.0.
