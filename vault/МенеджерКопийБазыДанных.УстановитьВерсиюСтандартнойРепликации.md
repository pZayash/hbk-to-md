МенеджерКопийБазыДанных (DatabaseCopiesManager)

УстановитьВерсиюСтандартнойРепликации (SetStandardReplicationVersion)

Доступен, начиная с версии 8.3.21.

Синтаксис:

УстановитьВерсиюСтандартнойРепликации(<ВерсияСтандартнойРепликации>)

Параметры:

<ВерсияСтандартнойРепликации> (обязательный)

Тип: [ВерсияСтандартнойРепликацииКопийБазыДанных](ВерсияСтандартнойРепликацииКопийБазыДанных.md).   
Устанавливаемая версия реализации стандартной репликации копий базы данных.

Описание:

Устанавливает версию реализации стандартной репликации копий базы данных.  
При вызове данного метода сбрасывается состояние копий с типом репликации [Стандартная](ТипРепликацииКопииБазыДанных.Стандартная.md) (копии не будут доступны в запросах до выполнения обновления копий), при следующем обновлении каждой копии будет выполнена полная перезаливка данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если установлена [Версия2](ВерсияСтандартнойРепликацииКопийБазыДанных.Версия2.md), то для информационных баз, использующих в качестве основной СУБД Microsoft SQL Server (версии 2008 и выше).

Использование в версии:

Доступен, начиная с версии 8.3.21.
