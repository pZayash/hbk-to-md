Вставить(<Индекс>, <Начало>, <Конец>)

<Индекс> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить период отображения.

<Начало> (обязательный)

Тип: Дата.   
Начало периода отображения.

<Конец> (обязательный)

Тип: Дата.   
Конец периода отображения.

Вставляет период отображения в коллекцию по указанному индексу

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
