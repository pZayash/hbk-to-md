Элементы коллекции:

[ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных_(UsedDatabaseCopy).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются .  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя копии.
