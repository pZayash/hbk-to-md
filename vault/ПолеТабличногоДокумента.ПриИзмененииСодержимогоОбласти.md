ПолеТабличногоДокумента (SpreadsheetDocumentField)

ПриИзмененииСодержимогоОбласти (OnChangeAreaContent)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриИзмененииСодержимогоОбласти(<Область>)

Параметры:

<Область>

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md), [РисунокТабличногоДокумента](РисунокТабличногоДокумента.md).   
Область ячеек или рисунок табличного документа, содержимое которых было изменено.

Описание:

Возникает при изменении содержимого области табличного документа.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
