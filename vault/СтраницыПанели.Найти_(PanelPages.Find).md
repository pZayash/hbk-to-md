СтраницыПанели (PanelPages)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяСтраницы>)

Параметры:

<ИмяСтраницы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя страницы.

Возвращаемое значение:

Тип: [СтраницаПанели](СтраницаПанели_(PanelPage).md), [Неопределено](lang__def_Undefined.md).   
Искомая страница в коллекции.  
Если не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск страницы по имени страницы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
