HTTPСоединение (HTTPConnection)

Изменить (Patch)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ИзменитьАсинх](HTTPСоединение.ИзменитьАсинх.md)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Изменить(<HTTPЗапрос>)

Параметры:

<HTTPЗапрос> (обязательный)

Тип: [HTTPЗапрос](HTTPЗапрос.md).   
Запрос, передаваемый на сервер.

Возвращаемое значение:

Тип: [HTTPОтвет](HTTPОтвет.md).   

Описание:

Изменяет данные на сервере при помощи PATCH-запроса.  
Передаются только изменения объекта.

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При использовании мобильной платформы на устройствах под управлением ОС Android версий 2.3 - 4.4 данный метод недоступен, и его вызов приведет к исключению.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
