Значения

[Плавный (Smooth)](РежимСглаживанияИндикатора.Плавный_(ProgressBarSmoothingMode.Smooth).md)

Описание:

Содержит варианты стилей заполнения индикатора. Используется для определения свойства [СтильОтображения](Индикатор.СтильОтображения_(ProgressBar.ShowStyle).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

См. также:
