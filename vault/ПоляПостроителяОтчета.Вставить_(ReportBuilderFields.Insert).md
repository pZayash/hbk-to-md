ПоляПостроителяОтчета (ReportBuilderFields)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным поля.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: [ПолеПостроителяОтчета](ПолеПостроителяОтчета_(ReportBuilderField).md).   

Описание:

Вставляет поле в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПостроительОтчета.ВыбранныеПоля.Вставить("Период","Период",              ПостроительОтчета.ВыбранныеПоля.Количество()+1); |

Использование в версии:

Доступен, начиная с версии 8.0.
