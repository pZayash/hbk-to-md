КонтрольнаяТочкаИтоговСхемыЗапроса (QuerySchemaTotalCalculationField)

ТипКонтрольнойТочки (TotalCalculationFieldType)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ТипКонтрольнойТочкиСхемыЗапроса](ТипКонтрольнойТочкиСхемыЗапроса_(QuerySchemaTotalCalculationFieldType).md).   
Указывает тип контрольной точки итогов схемы запроса.  
Контрольная точка представляет собой значение, для которого будут вычисляться значения итоговых полей

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
