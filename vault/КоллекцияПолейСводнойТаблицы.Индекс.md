Синтаксис:

Индекс(<Поле>)

Параметры:

<Поле> (обязательный)

Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Получает индекс элемента коллекции сводной таблицы.

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
