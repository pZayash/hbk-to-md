МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьДанныеРаботыСРечьюИнформационнойБазы (GetInfoBaseSpeechProcessingData)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДанныеРаботыСРечьюИнформационнойБазы()

Возвращаемое значение:

Тип: [ДанныеИнформационнойБазыРаботыСРечью](ДанныеИнформационнойБазыРаботыСРечью_(SpeechProcessingInfoBaseData).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает данные информационной базы распознавания речи (для текущей области данных) или [Неопределено](lang__def_Undefined.md), если нет данных.  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
