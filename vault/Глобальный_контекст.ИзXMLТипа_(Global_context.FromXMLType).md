Глобальный контекст (Global context)

ИзXMLТипа (FromXMLType)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По типу XML

Синтаксис:

ИзXMLТипа(<ТипXML>)

Параметры:

<ТипXML> (обязательный)

Тип: [ТипДанныхXML](ТипДанныхXML_(XMLDataType).md).   
Тип XML, для которого нужно получить соответствующий тип.

Вариант синтаксиса: По имени типа и URI пространства имен

Синтаксис:

ИзXMLТипа(<ИмяТипа>, <URIПространстваИмен>)

Параметры:

<ИмяТипа> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя типа XML.

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен типа XML.

Возвращаемое значение:

Тип: [Тип](lang__def_Type.md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md), если соответствующий тип данных не обнаружен.

Описание:

Получает тип, соответствующий типу данных XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
