КнопкаФормы (FormButton)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает возможность растягивания кнопки по горизонтали.   
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
