Элементы коллекции:

[ЭлементПакетаОтображаемыхДокументов](ЭлементПакетаОтображаемыхДокументов.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы пакета.  
Возможно обращение к элементу пакета посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
