Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

ВыбиратьСрез (SelectSlice)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеСреза](ИспользованиеСреза.md).   
Содержит вариант получения среза записей для периодического регистра сведений.

Доступность:

Толстый клиент.

Примечание:

Для непериодического регистра сведений всегда имеет значение [НеИспользовать](ИспользованиеСреза.НеИспользовать.md).

Использование в версии:

Доступен, начиная с версии 8.0.
