ЭлементыДендрограммы (DendrogramItemCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ЭлементДендрограммы](ЭлементДендрограммы_(DendrogramItem).md).   

Описание:

Вставляет элемент в заданную позицию коллекции элементов дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
