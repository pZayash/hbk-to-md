РежимСовместимости (CompatibilityMode)

Версия8\_3\_8 (Version8\_3\_8)

Доступен, начиная с версии 8.3.9.

Описание:

Усовершенствования версии 8.3.9 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.8.

Доступен, начиная с версии 8.3.9.
