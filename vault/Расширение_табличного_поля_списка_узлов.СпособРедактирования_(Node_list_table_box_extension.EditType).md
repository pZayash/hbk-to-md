Расширение табличного поля списка узлов (Node list table box extension)

СпособРедактирования (EditType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СпособРедактированияСписка](СпособРедактированияСписка_(ListEditMode).md).   
Предоставляет доступ к управлению способом редактирования элементов списка узлов (в диалоге или в списке).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
