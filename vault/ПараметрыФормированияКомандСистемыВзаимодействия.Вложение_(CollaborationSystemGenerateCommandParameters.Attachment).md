ПараметрыФормированияКомандСистемыВзаимодействия (CollaborationSystemGenerateCommandParameters)

Вложение (Attachment)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [ВложениеСистемыВзаимодействия](ВложениеСистемыВзаимодействия_(CollaborationSystemAttachment).md), [Неопределено](lang__def_Undefined.md).   
Указывает вложение, если источником команд является вложение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
