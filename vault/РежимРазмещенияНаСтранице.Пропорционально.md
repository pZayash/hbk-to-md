РежимРазмещенияНаСтранице (FitPageMode)

Пропорционально (Proportionally)

Доступен, начиная с версии 8.0.

Описание:

Выбор режима пропорционального масштабирования.   
Если высота или ширина графической схемы больше высоты (ширины) страницы, то производится пропорциональное масштабирование (пропорциональное изменение размеров по X и по Y) так, чтобы вся схема уместилась на одну страницу.

Использование в версии:

Доступен, начиная с версии 8.0.
