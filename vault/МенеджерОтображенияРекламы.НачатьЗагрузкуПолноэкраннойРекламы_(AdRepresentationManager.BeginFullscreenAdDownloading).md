МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуПолноэкраннойРекламы (BeginFullscreenAdDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуПолноэкраннойРекламы(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Ссылка на процедуру, которая будет вызвана после завершения загрузки рекламы из сети Интернет. Процедура должна иметь следующие параметры:

- Статус (Status) – тип [СтатусРекламы](СтатусРекламы_(AdStatus).md). Показывает статус рекламного блока по результатам загрузки.
- Идентификатор (ID) – тип [Строка](lang__def_String.md). Рекламный идентификатор, для которого закончилась загрузка.
- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Загружает из сети полноэкранную рекламу. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [ПоказатьПолноэкраннуюРекламу](МенеджерОтображенияРекламы.ПоказатьПолноэкраннуюРекламу_(AdRepresentationManager.ShowFullScreenAd).md) реклама будет показана без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы_(AdRepresentationManager).md), метод [ЗагрузитьПолноэкраннуюРекламуАсинх](МенеджерОтображенияРекламы.ЗагрузитьПолноэкраннуюРекламуАсинх_(AdRepresentationManager.DownloadFullscreenAdAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
