СписокСтрокDOM (DOMStringList)

Элемент (Item)

Доступен, начиная с версии 8.1.

Синтаксис:

Элемент(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс строки в коллекции.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает строку по указанному индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
