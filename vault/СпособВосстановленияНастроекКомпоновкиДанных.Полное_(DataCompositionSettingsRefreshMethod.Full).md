СпособВосстановленияНастроекКомпоновкиДанных (DataCompositionSettingsRefreshMethod)

Полное (Full)

Доступен, начиная с версии 8.2.

Описание:

При восстановлении настроек отключается использование у всех настроек, содержащих недоступные поля.

Использование в версии:

Доступен, начиная с версии 8.2.
