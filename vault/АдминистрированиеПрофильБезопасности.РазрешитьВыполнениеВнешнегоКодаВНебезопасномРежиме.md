АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

РазрешитьВыполнениеВнешнегоКодаВНебезопасномРежиме (AllowExternalCodeExecutionInUnsafeMode)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет разрешение запуска внешнего кода (внешние отчеты, внешние обработки, оператор Выполнить, Вычислить, расширения конфигурации) без безопасного режима, если это указано в конфигурации:

- Истина – для любого кода.
- Ложь – разрешен запуск без включения безопасного режима только внешних модулей, перечисленных в списке разрешенных внешних модулей.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
