ИнтернетТекстыПочтовогоСообщения (InternetMailTexts)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: С указанием типа текста

Синтаксис:

Добавить(<Текст>, <ТипТекста>)

Параметры:

<Текст> (необязательный)

Тип: Строка.   
Добавляемый текст.

<ТипТекста> (необязательный)

Тип: Строка, [ТипТекстаПочтовогоСообщения](ТипТекстаПочтовогоСообщения.md).   
Указывает тип добавляемого текста.   
Если Строка, то свойство [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста.md) устанавливается в [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст.md).  
При генерации отправляемого текста письма, в поле Content-Type письма добавляется (или заменяется) параметр charset, получаемый из кодировки, указанной в свойстве [Кодировка](ИнтернетТекстПочтовогоСообщения.Кодировка.md).  
Значение по умолчанию: [ПростойТекст](ТипТекстаПочтовогоСообщения.ПростойТекст.md).

Вариант синтаксиса: Без указания типа текста

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.   
Добавляемый текст.

Возвращаемое значение:

Тип: [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения.md).   

Описание:

Добавляет текст в коллекцию почтовых текстов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если для всех параметров метода используются значения по умолчанию, то [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста.md) добавленного объекта [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения.md) будет равен [HTML](ТипТекстаПочтовогоСообщения.HTML.md). Если же указан первый параметр <Текст>, то в качестве типа текста будет установлен [ПростойТекст](ТипТекстаПочтовогоСообщения.ПростойТекст.md).

Использование в версии:

Доступен, начиная с версии 8.0.
