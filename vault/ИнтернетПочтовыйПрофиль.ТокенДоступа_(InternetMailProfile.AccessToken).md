ИнтернетПочтовыйПрофиль (InternetMailProfile)

ТокенДоступа (AccessToken)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строковое значение полученного ранее токена доступа OAuth – access\_token.  
При вызове метода [Подключиться](ИнтернетПочта.Подключиться_(InternetMail.Logon).md) объекта [ИнтернетПочта](ИнтернетПочта_(InternetMail).md) аутентификация будет выполнена с использованием этого токена, если в свойстве [АутентификацияПоТокену](ИнтернетПочтовыйПрофиль.АутентификацияПоТокену_(InternetMailProfile.TokenAuthentication).md) указано [Использовать](ИспользованиеАутентификацииИнтернетПочтыПоТокену.Использовать_(UseInternetMailTokenAuthentication.Use).md) или [Авто](ИспользованиеАутентификацииИнтернетПочтыПоТокену.Авто_(UseInternetMailTokenAuthentication.Auto).md), и подключение осуществляется по протоколу IMAP или SMTP.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.19.
