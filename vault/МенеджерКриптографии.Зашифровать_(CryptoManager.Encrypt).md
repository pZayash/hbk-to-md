МенеджерКриптографии (CryptoManager)

Зашифровать (Encrypt)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: Данные в возвращаемом значении

Синтаксис:

Зашифровать(<ИсходныеДанные>, <Получатели>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Исходные данные для шифрования.   
Данные могут размещаться в файле (в этом случае указывается имя файла), а также представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md).

<Получатели> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), [Массив](Массив_(Array).md).   
Параметр может содержать один сертификат или массив объектов сертификатов.   
Только получатели из указанного списка (получатели, обладающие переданными сертификатами) смогут расшифровать данные, используя свои закрытые ключи.

Описание варианта метода:

Зашифрованные данные возвращаются методом как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Сохранение данных в файл

Синтаксис:

Зашифровать(<ИсходныеДанные>, <ВыходныеДанные>, <Получатели>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Исходные данные для шифрования.   
Данные могут размещаться в файле (в этом случае указывается имя файла), а также представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md).

<ВыходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Имя файла или потока, в который будут выгружены зашифрованные данные.

<Получатели> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), [Массив](Массив_(Array).md).   
Параметр может содержать один сертификат или массив сертификатов.  
Только получатели из указанного списка (получатели, обладающие переданными сертификатами) смогут расшифровать данные, используя свои закрытые ключи.

Описание варианта метода:

Зашифрованные данные записываются в файл. Возвращает [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Неопределено](lang__def_Undefined.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Осуществляет шифрование данных для определенного круга получателей. Только получатели из указанного списка смогут расшифровать данные, используя свои закрытые ключи.  
Формат зашифрованных данных - CMS (базируется на PKCS#7).  
Используется алгоритм, заданный в свойстве [АлгоритмШифрования](МенеджерКриптографии.АлгоритмШифрования_(CryptoManager.EncryptAlgorithm).md).  
Размер входных данных для шифрования ограничен 2 Гб (2147483647 байт).  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
