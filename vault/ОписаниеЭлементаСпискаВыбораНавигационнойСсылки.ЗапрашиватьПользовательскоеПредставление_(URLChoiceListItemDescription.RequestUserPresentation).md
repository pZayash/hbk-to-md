ОписаниеЭлементаСпискаВыбораНавигационнойСсылки (URLChoiceListItemDescription)

ЗапрашиватьПользовательскоеПредставление (RequestUserPresentation)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость запросить представление навигационной ссылки у пользователя.   
Если [Истина](lang__def_BooleanTrue.md), то форма получения навигационной ссылки запросит у пользователя представление ссылки, которое будет использовано для сохранения внешних данных ссылки и для отображения ссылки, например, в списке избранного.   
Если [Ложь](lang__def_BooleanFalse.md), будет использовано значение свойства [Представление](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.Представление_(URLChoiceListItemDescription.Presentation).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
