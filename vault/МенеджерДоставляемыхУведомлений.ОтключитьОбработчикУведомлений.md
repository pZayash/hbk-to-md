МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ОтключитьОбработчикУведомлений (DetachNotificationHandler)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОтключитьОбработчикУведомлений(<Процедура>)

Параметры:

<Процедура> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Строка.   
Имя процедуры, которую необходимо исключить из обработки уведомления.

Описание:

Отключает вызов указанной процедуры.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
