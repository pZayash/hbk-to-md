Automation сервер (Automation server)

Visible (Visible)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает/скрывает пользовательский интерфейс 1С:Предприятия.

Доступность:

Интеграция.

Примечание:

[Истина](lang__def_BooleanTrue.md) - пользовательский интерфейс показан; [Ложь](lang__def_BooleanFalse.md) - скрыт.   
Начальное значение после вызова метода [Connect](Automation_сервер.Connect_(Automation_server.Connect).md) – [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.1.
