КоллекцияДействийСообщенияСистемыВзаимодействия (CollaborationSystemMessageActionCollection)

Найти (Find)

Не рекомендуется использовать, начиная с версии 8.3.25.

Доступен, начиная с версии 8.3.18.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению действия сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Выполняет поиск действия сообщения системы взаимодействия по значению свойства <Значение>. Возвращает первое действие с совпадающим значением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
