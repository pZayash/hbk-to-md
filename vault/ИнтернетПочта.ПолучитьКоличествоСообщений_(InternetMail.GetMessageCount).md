# ИнтернетПочта.ПолучитьКоличествоСообщений

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Почта](Почта-2.md) › [ИнтернетПочта](ИнтернетПочта.md) › [ИнтернетПочта](ИнтернетПочта_(InternetMail).md) › [Методы](ИнтернетПочта__Методы.md)

ИнтернетПочта (InternetMail)

ПолучитьКоличествоСообщений (GetMessageCount)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьКоличествоСообщений()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество сообщений [IMAP](ПротоколИнтернетПочты.IMAP_(InternetMailProtocol.IMAP).md) в текущем почтовом ящике ([ТекущийПочтовыйЯщик](ИнтернетПочта.ТекущийПочтовыйЯщик_(InternetMail.CurrentMailbox).md)) или количество сообщений в ящике POP3.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
