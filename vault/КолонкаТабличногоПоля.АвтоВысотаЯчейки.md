КолонкаТабличногоПоля (TableBoxColumn)

АвтоВысотаЯчейки (AutoCellHeight)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает необходимость вычисления высоты ячейки данной колонки. При этом в свойстве [ВысотаЯчейки](КолонкаТабличногоПоля.ВысотаЯчейки.md) указывается максимальная высота ячейки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
