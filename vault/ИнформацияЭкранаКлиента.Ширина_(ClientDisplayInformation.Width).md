ИнформацияЭкранаКлиента (ClientDisplayInformation)

Ширина (Width)

Доступен, начиная с версии 8.3.2.

Использование:

Только чтение.

Описание:

Ширина экрана в точках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выдаётся в оригинальной ориентации, без учета поворота. Для телефонов и некоторых планшетов это портретная ориентация, для большинства планшетов - ланшафтная.

Использование в версии:

Доступен, начиная с версии 8.3.2.
