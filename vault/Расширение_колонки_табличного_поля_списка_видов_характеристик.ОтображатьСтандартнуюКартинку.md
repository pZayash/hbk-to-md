Расширение колонки табличного поля списка видов характеристик (Characteristic type list table box column extension)

ОтображатьСтандартнуюКартинку (DisplayDefaultPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак отображения в колонке стандартной картинки, обозначающей вид характеристики и его состояние.  
При отображении в колонке иерархии для показа иерархии используются стандартные картинки.

Доступность:

Толстый клиент.

Примечание:

Если Истина, то при выводе свойство [КартинкиСтрок](КолонкаТабличногоПоля.КартинкиСтрок.md) колонки табличного поля не используется.

Использование в версии:

Доступен, начиная с версии 8.0.
