# ВидКлючаДинамическогоСписка.НомерСтроки

ВидКлючаДинамическогоСписка (DynamicListKeyType)

НомерСтроки (RowNumber)

Доступен, начиная с версии 8.3.14.

Описание:

Используется, если в динамическом поле отсутствуют ключевые поля.  
В качестве значений ТекущаяСтрока используются числа - порядковые номера записей от начала выборки.   
При использовании ключей такого вида, функциональность динамического списка ограничена.  
Перед использованием такого вида ключей, рекомендуется рассмотреть варианты использования видов ключей [КлючСтроки](ВидКлючаДинамическогоСписка.КлючСтроки_(DynamicListKeyType.RowKey).md) или [ЗначениеПоля](ВидКлючаДинамическогоСписка.ЗначениеПоля_(DynamicListKeyType.FieldValue).md).

Использование в версии:

Доступен, начиная с версии 8.3.14.
