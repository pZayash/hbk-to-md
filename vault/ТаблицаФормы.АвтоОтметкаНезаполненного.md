ТаблицаФормы (FormTable)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Автоматически управляет свойством [ОтметкаНезаполненного](ТаблицаФормы.ОтметкаНезаполненного.md) при добавлении и удалении строк таблицы.  
Если нет ни одной строки - ставит [ОтметкаНезаполненного](ТаблицаФормы.ОтметкаНезаполненного.md) в Истина, иначе Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
