Соединение с агентом сервера (IServerAgentConnection)

GetWorkingProcesses (GetWorkingProcesses)

Доступен, начиная с версии 8.1.

Синтаксис:

GetWorkingProcesses(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, для которого должен быть получен массив рабочих процессов.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив рабочих процессов. Каждый рабочий процесс является объектом с интерфейсом [Рабочий процесс](Рабочий_процесс.md).

Описание:

Позволяет получить массив рабочих процессов, зарегистрированных в заданном кластере. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
