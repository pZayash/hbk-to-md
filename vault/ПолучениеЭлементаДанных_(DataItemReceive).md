Значения

[Авто (Auto)](ПолучениеЭлементаДанных.Авто_(DataItemReceive.Auto).md)  
[Игнорировать (Ignore)](ПолучениеЭлементаДанных.Игнорировать_(DataItemReceive.Ignore).md)  
[Принять (Accept)](ПолучениеЭлементаДанных.Принять_(DataItemReceive.Accept).md)  

Описание:

Содержит варианты значений выходных параметров обработчиков событий [ПриПолученииДанныхОтГлавного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтГлавного_(ExchangePlanObject.Exchange_plan_name.OnReceiveDataFromMaster).md) и [ПриПолученииДанныхОтПодчиненного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтПодчиненного_(ExchangePlanObject.Exchange_plan_name.OnReceiveDataFromSlave).md) объектов [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>. Данное значение определяет поведение системы при получении элементов данных из сообщения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.
