ДиалогВыбораШрифта (FontChooseDialog)

ВыбратьАсинх (ChooseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВыбратьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Шрифт](ДиалогВыбораШрифта.Шрифт_(FontChooseDialog.Font).md) - выбранный шрифт, [Неопределено](lang__def_Undefined.md) - если пользователь цвет не выбрал.

Описание:

Асинхронная функция. Открывает немодальный диалог выбора шрифта.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

[ДиалогВыбораШрифта](ДиалогВыбораШрифта_(FontChooseDialog).md), метод [Выбрать](ДиалогВыбораШрифта.Выбрать_(FontChooseDialog.Choose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
