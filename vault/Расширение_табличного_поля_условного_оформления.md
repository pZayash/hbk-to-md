Доступен, начиная с версии 8.0.

Свойства:

[НастройкаУсловногоОформления (ConditionalAppearanceSetting)](Расширение_табличного_поля_условного_оформления.НастройкаУсловногоОформления.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для условного оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ЭлементУсловногоОформления](ЭлементУсловногоОформления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
