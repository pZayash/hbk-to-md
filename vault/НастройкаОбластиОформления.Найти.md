НастройкаОбластиОформления (AppearanceAreaSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента управления областью оформления.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойОбластиОформления](ЭлементУправленияНастройкойОбластиОформления.md), Неопределено.   
Если не найдено, возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента по имени элемента управления областью оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
