КонстантыНабор (ConstantsSet)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: Булево.   
Истина - ошибок не обнаружено, Ложь - в противном случае.

Описание:

Проверка заполнения выполняется менеджерами значений констант, входящих в набор и допускающих запись с текущим набором прав.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к последовательному возникновению события [ОбработкаПроверкиЗаполнения](КонстантаМенеджерЗначения.Имя_константы.ОбработкаПроверкиЗаполнения.md) для каждой константы, входящей в набор.

Использование в версии:

Доступен, начиная с версии 8.2.
