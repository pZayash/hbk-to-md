СертификатКлиентаФайл (FileClientCertificate)

ИмяФайла (FileName)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, из которого был загружен клиентский сертификат и его приватный ключ.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
