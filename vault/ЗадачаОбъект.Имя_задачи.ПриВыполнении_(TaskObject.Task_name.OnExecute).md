# ЗадачаОбъект.<Имя задачи>.ПриВыполнении

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [События](ЗадачаОбъект.Имя_задачи__События.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПриВыполнении (OnExecute)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриВыполнении(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при выполнении метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу_(TaskObject.Task_name.ExecuteTask).md) или [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно_(TaskObject.Task_name.ExecuteTaskInteractively).md). Выполняется в транзакции записи при выполнении задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
