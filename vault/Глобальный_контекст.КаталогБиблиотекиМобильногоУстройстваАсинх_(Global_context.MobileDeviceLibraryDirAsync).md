Глобальный контекст (Global context)

КаталогБиблиотекиМобильногоУстройстваАсинх (MobileDeviceLibraryDirAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КаталогБиблиотекиМобильногоУстройстваАсинх(<ТипКаталога>)

Параметры:

<ТипКаталога> (обязательный)

Тип: [ТипКаталогаБиблиотекиМобильногоУстройства](ТипКаталогаБиблиотекиМобильногоУстройства_(MobileDeviceLibraryDirType).md).   
Тип каталога библиотеки мобильного устройства.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Строка](lang__def_String.md), имя каталога.

Описание:

Асинхронная функция. Получает путь к библиотекам мобильного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[Глобальный контекст](Глобальный_контекст.md), метод [КаталогБиблиотекиМобильногоУстройства](Глобальный_контекст.КаталогБиблиотекиМобильногоУстройства_(Global_context.MobileDeviceLibraryDir).md)

Доступен, начиная с версии 8.3.18.
