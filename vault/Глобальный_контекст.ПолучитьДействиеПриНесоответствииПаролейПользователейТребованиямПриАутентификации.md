ПолучитьДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации (GetActionOnUserPasswordRequirementsViolationOnAuthentication)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации()

Возвращаемое значение:

Тип: [ДействиеПриНесоответствииПароляТребованиямПриАутентификации](ДействиеПриНесоответствииПароляТребованиямПриАутентификации.md).   

Описание:

Возвращает действие при несоответствии паролей пользователей требованиям при аутентификации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
