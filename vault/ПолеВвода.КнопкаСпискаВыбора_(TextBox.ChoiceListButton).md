ПолеВвода (TextBox)

КнопкаСпискаВыбора (ChoiceListButton)

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.3.2 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак наличия/отсутствия кнопки выбора из списка.  
[Истина](lang__def_BooleanTrue.md) - кнопка есть.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.3.2 и последующими).

Описание изменено в версии 8.3.15.
