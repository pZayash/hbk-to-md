# Расширение таблицы формы для значений параметров компоновки данных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [Расширения для системы компоновки данных](Расширения_для_системы_компоновки_данных.md)

Доступен, начиная с версии 8.2.

Свойства:

[ОграниченияИспользования (UseRestrictions)](Расширение_таблицы_формы_для_значений_параметров_компоновки_данных.ОграниченияИспользования_(Form_table_extension_for_data_composition_parameter_values.UseRestrictions).md)  

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы_(FormTable).md), применимые для значений параметров компоновки данных.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель_(FormTable.CurrentParent).md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки_(FormTable.SelectedRows).md), используется идентификатор типа [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md).  
Используется для работы с данными следующих типов:

- [ОформлениеКомпоновкиДанных](ОформлениеКомпоновкиДанных_(DataCompositionAppearance).md);
- [ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных_(DataCompositionChartGroupOutputParameterValues).md);
- [ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных_(DataCompositionGroupOutputParameterValues).md);
- [ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных_(DataCompositionTableGroupOutputParameterValues).md);
- [ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных_(DataCompositionChartOutputParameterValues).md);
- [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных_(DataCompositionOutputParameterValues).md);
- [ЗначенияПараметровВыводаТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаТаблицыКомпоновкиДанных_(DataCompositionTableOutputParameterValues).md);
- [ЗначенияПараметровДанныхКомпоновкиДанных](ЗначенияПараметровДанныхКомпоновкиДанных_(DataCompositionDataParameterValues).md).

Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные_(FormTable.CurrentData).md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки_(FormTable.RowData).md) возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
