# МенеджерКриптографии.ПроверитьМеткуВремениАсинх

МенеджерКриптографии (CryptoManager)

ПроверитьМеткуВремениАсинх (VerifyTimestampAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПроверитьМеткуВремениАсинх(<Подпись>, <МеткаВремени>)

Параметры:

<Подпись> (необязательный)

Тип: [ПодписьКриптографии](ПодписьКриптографии_(CryptoSignature).md).   
Подпись из контейнера подписей.

<МеткаВремени> (необязательный)

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии_(CryptoTimestamp).md).   
Проверяемая метка времени.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
В случае успешного результата проверки результатом выполнения обещания является [Неопределено](lang__def_Undefined.md). В случае ошибки объект [Обещание](Обещание_(Promise).md) будет содержать исключение, которое будет выброшено из оператора Ждать, аргументом которого будет данное [Обещание](Обещание_(Promise).md).

Описание:

Асинхронная функция. Метод выполняет проверку метки времени для переданной подписи криптографии:

- Соответствие хеша, указанного в метке времени, хешу полей подписи.
- Корректность сертификата метки.
- Корректность подписи на метке времени.

При возникновении ошибки проверки, будет сгенерировано исключение с описанием ошибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
