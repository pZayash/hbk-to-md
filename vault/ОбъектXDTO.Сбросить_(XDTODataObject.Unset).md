ОбъектXDTO (XDTODataObject)

Сбросить (Unset)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По выражению XPath

Синтаксис:

Сбросить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение XPath, соответствующее свойству, у которого необходимо сбросить значение.

Вариант синтаксиса: По свойству

Синтаксис:

Сбросить(<Свойство>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md).   
Свойство, значение которого необходимо сбросить.

Описание:

Сбрасывает значения указанного свойства. Метод [Установлено](ОбъектXDTO.Установлено_(XDTODataObject.IsSet).md) будет возвращать значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для списковых свойств выполнение данного метода приведет к очистке списка.

Использование в версии:

Доступен, начиная с версии 8.1.
