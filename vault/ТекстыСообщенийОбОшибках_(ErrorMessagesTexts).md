**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Обработка ошибок](Обработка_ошибок-2.md)

Доступен, начиная с версии 8.3.17.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение_(KeyAndValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции [КлючИЗначение](КлючИЗначение_(KeyAndValue).md). Ключом является значение типа [КатегорияОшибки](КатегорияОшибки_(ErrorCategory).md), значением - [ТекстыСообщенияОбОшибке](ТекстыСообщенияОбОшибке_(ErrorMessageTexts).md).  
Возможно обращение к значениям посредством оператора [...]. В качестве аргумента передается значение типа [КатегорияОшибки](КатегорияОшибки_(ErrorCategory).md).

Описание:

Содержит тексты сообщений об ошибках по категориям.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.22.
