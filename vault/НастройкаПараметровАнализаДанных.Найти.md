НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента параметров.

Возвращаемое значение:

Тип: [ЭлементУправленияПараметрамиАнализаДанных](ЭлементУправленияПараметрамиАнализаДанных.md), Неопределено.   
Если не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента управления параметров по имени элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
