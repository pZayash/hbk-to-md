БизнесПроцессСсылка.<Имя бизнес-процесса> (BusinessProcessRef.<Business process name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку_(BusinessProcessManager.Business_process_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса_(BusinessProcessManager.Business_process_name).md), метод [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку_(BusinessProcessManager.Business_process_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
