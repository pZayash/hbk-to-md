ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>.   
Набор свойств содержит табличные части видов расчета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>).  
Предопределенные табличные части [ВедущиеВидыРасчетаСтрока.](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(LeadingCalculationTypesRow.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(LeadingCalculationTypesRow.Chart_of_calculation_types_name).md)>, [БазовыеВидыРасчетаСтрока.](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета_(BaseCalculationTypesRow.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета_(BaseCalculationTypesRow.Chart_of_calculation_types_name).md)> и [ВытесняющиеВидыРасчетаСтрока.](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(DisplacingCalculationTypesRow.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(DisplacingCalculationTypesRow.Chart_of_calculation_types_name).md)> доступны через специальные соответствующие свойства объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
