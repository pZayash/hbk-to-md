Расширение тестируемого поля формы для текстового документа (Tested form field extension for a text document)

ВвестиТекст (InputText)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ВвестиТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Вводимый текст.

Описание:

Осуществляет ввод текста в поле ввода. Переданное значение целиком замещает предыдущий текст.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
