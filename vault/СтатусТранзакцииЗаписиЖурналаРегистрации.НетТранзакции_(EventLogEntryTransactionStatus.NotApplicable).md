СтатусТранзакцииЗаписиЖурналаРегистрации (EventLogEntryTransactionStatus)

НетТранзакции (NotApplicable)

Доступен, начиная с версии 8.1.

Запись журнала не зависит от транзакции, свойство статуса транзакции к ней не применяется.

Доступен, начиная с версии 8.1.
