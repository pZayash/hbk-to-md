<!-- signature:end -->

ДиаграммаГанта (GanttChart)

ПоказатьУровеньТочек (ShowPointLevel)

Доступен, начиная с версии 8.0.

Синтаксис:

ПоказатьУровеньТочек(<КоличествоУровней>)

Параметры:

<КоличествоУровней> (необязательный)

Тип: Число.   
Число уровней, начиная от верхнего, которые требуется раскрыть. Если 0, то раскрыть все уровни.  
Значение по умолчанию: 0.

Описание:

Раскрывает заданное число уровней точек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ДиаграммаГанта.ПоказатьУровеньТочек(0); |

Использование в версии:

Доступен, начиная с версии 8.0.
