Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

ПриИзмененииСодержимогоОбласти (OnChangeAreaContentEvent)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриИзмененииСодержимогоОбласти(<Область>, <ДополнительныеПараметры>)

Параметры:

<Область>

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Область ячеек табличного документа, содержимое которых было изменено.

<ДополнительныеПараметры>

Тип: [Структура](Структура.md).   
Если событие было сформировано диаграммой Ганта, то структура будет содержать поля "Интервал" (Interval) и "Отмена" (Cancel) (аналогичные параметрам события диаграммы Ганта [ПриОкончанииРедактированияИнтервала](Расширение_поля_формы_для_поля_диаграммы_Ганта.ПриОкончанииРедактированияИнтервала.md)). Если событие было сформировано диаграммой, то структура будет содержать поля "ЗначениеДиаграммы" (ChartValue) и "СостояниеРедактирования" (EditState) (аналогичные параметрам события диаграммы [ПриРедактированииЗначения](Диаграмма.ПриРедактированииЗначения.md)). В других случаях структура будет пустая.

Описание:

Возникает при изменении содержимого области табличного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
