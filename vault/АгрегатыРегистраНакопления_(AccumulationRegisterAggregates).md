Доступен, начиная с версии 8.2.

Элементы коллекции:

[АгрегатРегистраНакопления](АгрегатРегистраНакопления_(AccumulationRegisterAggregate).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются агрегаты.  
Возможно обращение к агрегату посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Количество (Count)](АгрегатыРегистраНакопления.Количество_(AccumulationRegisterAggregates.Count).md)  
[Получить (Get)](АгрегатыРегистраНакопления.Получить_(AccumulationRegisterAggregates.Get).md)  

Описание:

Представляет собой коллекцию агрегатов регистра накопления.

Доступность:

Сервер, толстый клиент, внешнее соединение.
