МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьРежимИспользованияХранилищаДвоичныхДанных (SetBinaryDataStorageUseMode)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьРежимИспользованияХранилищаДвоичныхДанных(<РежимИспользованияХранилищаДвоичныхДанных>)

Параметры:

<РежимИспользованияХранилищаДвоичныхДанных> (обязательный)

Тип: [РежимИспользованияХранилищаДвоичныхДанных](РежимИспользованияХранилищаДвоичныхДанных.md).   
Режим использования хранилища двоичных данных.  
Для созданной информационной базы использование хранилище двоичных данных выключено (установлено в [Запретить](РежимИспользованияХранилищаДвоичныхДанных.Запретить.md)).

Описание:

Устанавливает режим использования хранилища двоичных данных.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
