ГрафическаяСхема (GraphicalSchema)

Записать (Write)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в котором сохраняется графическая схема.

Описание варианта метода:

Запись графической схемы в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи графической схемы.

Описание варианта метода:

Записывает графическую схему в поток.

Описание:

Записывает графическую схему в файл.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Пример:

|  |
| --- |
| Схема.Записать("C:\My Documents\Схема.grs"); |

См. также:

[ГрафическаяСхема](ГрафическаяСхема_(GraphicalSchema).md), метод [ЗаписатьАсинх](ГрафическаяСхема.ЗаписатьАсинх_(GraphicalSchema.WriteAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
