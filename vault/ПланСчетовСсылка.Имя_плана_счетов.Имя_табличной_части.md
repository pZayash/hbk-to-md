Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовТабличнаяЧасть.<Имя плана счетов>.<Имя табличной части>.   
Набор свойств содержит табличные части счета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланСчетовТабличнаяЧасть.<Имя плана счетов>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
