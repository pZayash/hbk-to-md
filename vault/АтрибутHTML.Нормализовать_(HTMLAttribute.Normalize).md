# АтрибутHTML.Нормализовать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md) › [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md) › [Методы](АтрибутHTML__Методы.md)

АтрибутHTML (HTMLAttribute)

Нормализовать (Normalize)

Доступен, начиная с версии 8.2.

Синтаксис:

Нормализовать()

Описание:

Приводит все узлы [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) на полной глубине поддерева данного узла, включая узлы [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), в "нормальную" форму, где только структуры (например, элементы, комментарии) разделяются текстовыми узлами, т.е. смежные текстовые узлы отсутствуют.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
