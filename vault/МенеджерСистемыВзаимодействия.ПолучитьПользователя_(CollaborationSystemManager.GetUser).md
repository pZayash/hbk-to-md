МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьПользователя (GetUser)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьПользователя(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Идентификатор получаемого пользователя.

Возвращаемое значение:

Тип: [ПользовательСистемыВзаимодействия](ПользовательСистемыВзаимодействия_(CollaborationSystemUser).md).   

Описание:

Возвращает пользователя системы взаимодействия по его идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если в качестве параметра используется значение [ВсеПользователиПриложения](СтандартныеПользователиСистемыВзаимодействия.ВсеПользователиПриложения_(CollaborationSystemStandardUsers.AllApplicationUsers).md), будет сгенерирована ошибка времени выполнения.
