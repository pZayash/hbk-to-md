КнопкиКоманднойПанели (CommandBarButtons)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число, [КнопкаКоманднойПанели](КнопкаКоманднойПанели.md).   
Позиция кнопки в командной панели или кнопка.

Описание:

Удаляет кнопку из командной панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| // КоманднаяПанель1 командная панель содержит подменю Заполнить  // Удаление из подменю кнопки ИзТерминалаСбора (сбор данных)  Кнопки = ЭлементыФормы.КоманднаяПанель1.Кнопки.Заполнить.Кнопки; ИндексКнопки = КоманднаяПанель1.Индекс(Кнопки.ИзТерминалаСбора); Кнопки.Удалить(ИндексКнопки); |

Использование в версии:

Доступен, начиная с версии 8.0.
