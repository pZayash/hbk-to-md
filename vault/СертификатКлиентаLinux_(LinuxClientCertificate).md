# СертификатКлиентаLinux

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Интернет](Интернет-2.md) › [Сертификаты защищенных соединений](Сертификаты_защищенных_соединений.md)

Доступен, начиная с версии 8.3.21.

Конструкторы:

[По умолчанию](СертификатКлиентаLinux.По_умолчанию.md)  

Описание:

Объект клиентского сертификата OpenSSL, загружаемого из хранилища сертификатов ОС Linux.  
Поддерживаются хранилища сертификатов КриптоПро CSP. Поиск библиотек КриптоПро выполняется в стандартных каталогах. Также в файле conf.cfg, в параметре CryptoAPILibraryLocation можно указать пути к файлу библиотеки или список путей, разделенных точками с запятой.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md)  

Использование в версии:

Доступен, начиная с версии 8.3.21.
