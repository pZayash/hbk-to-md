Действие (Action)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Произвольный.   
Указывает действие, если источником команд является гиперссылка на действие.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
