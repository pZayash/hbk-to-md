# Расширение поля формы для поля табличного документа.Высота

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля табличного документа](Расширение_поля_формы_для_поля_табличного_документа_(Form_field_extension_for_a_spreadsheet_document_field).md) › [Свойства](Расширение_поля_формы_для_поля_табличного_документа__Свойства.md)

Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту табличного документа. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
