Описание:

Предоставляет доступ к физической NFC-метке. Используется для операций чтения/записи. Объекты данного типа не нужно создавать самостоятельно. При обнаружении NDEF-метки объект будет создан и передан обработчику автоматически.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
