МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВидеоконференцияАсинх (VideoconferenceAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВидеоконференцияАсинх(<Участники>, <Обсуждение>, <ПоказатьВыборУчастников>)

Параметры:

<Участники> (необязательный)

Тип: [КоллекцияИдентификаторовПользователейСистемыВзаимодействия](КоллекцияИдентификаторовПользователейСистемыВзаимодействия.md).   
Если <ПоказатьВыборУчастников> равен Истина, то содержит список пользователей, отображаемых в диалоге участников видеоконференции.  
Если <ПоказатьВыборУчастников> равен Ложь, то содержит список пользователей, которым будет отправлен запрос на участие в видеоконференции.

<Обсуждение> (необязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Устанавливает связь видеоконференции с обсуждением.  
Если <ПоказатьВыборУчастников> имеет значение Истина, то список участников обсуждения отображается в диалоге выбора участников видеоконференции.

<ПоказатьВыборУчастников> (необязательный)

Тип: Булево.   
Признак отображения диалога выбора участников видеоконференции.  
Истина - диалог отображается.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - значение типа Булево:

- Истина - всем участникам видеоконференции отправлены приглашения,
- Ложь - видеоконференция отменена (в диалоге выбора участников нажата кнопка "Отмена").

Описание:

Асинхронная функция. Начинает новую видеоконференцию.  
Если не заданы <Участники>, тогда:

- Если <ПоказатьВыборУчастников> имеет значение Истина, открывается диалог выбора участников с пустым списком участников (добавлять участников можно подбором),
- Если <ПоказатьВыборУчастников> имеет значение Ложь, то видеоконференция начинается без участников.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [НачатьВидеоконференцию](МенеджерСистемыВзаимодействия.НачатьВидеоконференцию.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
