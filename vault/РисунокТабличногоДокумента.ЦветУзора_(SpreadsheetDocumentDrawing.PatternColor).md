РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ЦветУзора (PatternColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет узора рисунка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не применимо для рисунков типа [Прямая](ТипРисункаТабличногоДокумента.Прямая_(SpreadsheetDocumentDrawingType.Line).md), [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
