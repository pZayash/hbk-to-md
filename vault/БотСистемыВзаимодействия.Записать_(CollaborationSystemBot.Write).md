БотСистемыВзаимодействия (CollaborationSystemBot)

Записать (Write)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Записать()

Описание:

Записывает бота в информационную базу.  
При записи существующего бота (полученного с помощью методом [ПолучитьБота](МенеджерСистемыВзаимодействия.ПолучитьБота_(CollaborationSystemManager.GetBot).md) или [ПолучитьБотов](МенеджерСистемыВзаимодействия.ПолучитьБотов_(CollaborationSystemManager.GetBots).md)), если изменено свойство [Пользователь](БотСистемыВзаимодействия.Пользователь_(CollaborationSystemBot.User).md), то вначале удаляется запись существующего бота.  
Требует наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
