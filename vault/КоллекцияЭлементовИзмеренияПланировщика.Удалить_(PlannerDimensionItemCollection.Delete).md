КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md);.   
Индекс удаляемого элемента в коллекции или сам элемент.

Описание:

Удаляет элемент измерения в заданной позиции коллекции элементов измерения планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
