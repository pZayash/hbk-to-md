КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить элемент измерения планировщика.

<Значение> (обязательный)

Тип: Произвольный.   
Значение элемента измерения планировщика. Значение должно быть уникальным в пределах коллекции.

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, отображаемый в заголовке элемента измерения.  
Рекомендуется заполнить, так как это позволит уменьшить количество обращений к серверу для получения представлений.

Возвращаемое значение:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md).   

Описание:

Вставляет элемент измерения планировщика в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
