Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор.md).   
Содержит объект [Отбор](Отбор.md), по которому осуществляется текущая фильтрация элементов плана видов характеристик в списке. Имена элементов отбора совпадают с именами реквизитов плана видов характеристик и критериев отбора, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
