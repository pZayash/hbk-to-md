АвтоФиксация (AutoFixation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак включенной автофиксации колонок/строк сводной таблицы.   
[Истина](lang__def_BooleanTrue.md) - автофиксация включена (фиксируются заголовки таблицы).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
