Рабочий сервер (IWorkingServerInfo)

ConnectionsPerWorkingProcessLimit (ConnectionsPerWorkingProcessLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит ограничение количества соединений с информационными базами через один рабочий процесс. При достижении этого количества на рабочий процесс не могут назначаться новые соединения с информационными базами.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
