ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

Страна (Country)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Код или наименование страны. Рекомендуется использовать двухбуквенный код.   
При повторном получении лицензии, полученной платформой версии 8.3.10 и ниже (в 2017 году и раньше), требуется указание наименования страны, как оно было указано при первоначальном получении лицензии.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
