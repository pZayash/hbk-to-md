ДанныеКонтакта (ContactData)

АдресаМгновенныхСообщений (InstantMessagingAddresses)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Содержит массив адресов мгновенных сообщений в виде объектов типа [ЭлементДанныхКонтактаМгновенныеСообщения](ЭлементДанныхКонтактаМгновенныеСообщения.md).   
Свойство [ТипДанных](ЭлементДанныхКонтактаМгновенныеСообщения.ТипДанных.md) содержит значение перечисления [ТипАдресаМгновенныхСообщенийДанныхКонтакта](ТипАдресаМгновенныхСообщенийДанныхКонтакта.md) или строку. Свойство [ТипМгновенныхСообщений](ЭлементДанныхКонтактаМгновенныеСообщения.ТипМгновенныхСообщений.md) содержит строку (например, Skype, ICQ и т.д.). Свойство [Идентификатор](ЭлементДанныхКонтактаМгновенныеСообщения.Идентификатор.md) содержит строку – идентификатор пользователя в системе мгновенных сообщений.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
