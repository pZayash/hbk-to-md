ОбъектXDTO (XDTODataObject)

Добавить (Add)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Установка значения нового произвольного свойства

Синтаксис:

Добавить(<Форма>, <URIПространстваИмен>, <ЛокальноеИмя>, <ЭлементДанных>, <НазначениеТипа>)

Параметры:

<Форма> (обязательный)

Тип: [ФормаXML](ФормаXML_(XMLForm).md).   
Форма XML представления произвольного свойства.  
Может принимать значения [Элемент](ФормаXML.Элемент_(XMLForm.Element).md) или [Атрибут](ФормаXML.Атрибут_(XMLForm.Attribute).md).

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен XML представления произвольного свойства.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя произвольного свойства.

<ЭлементДанных> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Элемент данных XDTO (значение или объект), являющийся значением произвольного свойства.

<НазначениеТипа> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Назначение типа данных произвольного свойства при сериализации.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Описание варианта метода:

Имя свойства будет создано автоматически, на основе локального имени XML представления свойства.  
В случае, если произвольное свойство с полученным именем уже существует и если предыдущая установка значения произвольного свойства выполнялась для данного произвольного свойства, либо другого свойства в случае наличия последовательности, свойство становится списковым с неограниченной верхней границей.   
В противном случае будет инициировано исключение.  
Тип создаваемого свойства всегда устанавливается anyType пространства имен схемы XML.

Вариант синтаксиса: Установка значения существующего произвольного свойства

Синтаксис:

Добавить(<Имя>, <ЭлементДанных>, <НазначениеТипа>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя произвольного свойства.

<ЭлементДанных> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Элемент данных XDTO (значение или объект), являющийся значением произвольного свойства.

<НазначениеТипа> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Назначение типа данных произвольного свойства при сериализации.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Описание варианта метода:

Если свойства с указанным именем не существует, то будет инициировано исключение.  
Если предыдущая установка значения произвольного свойства выполнялась для данного произвольного свойства, либо другого свойства в случае наличия последовательности, - свойство становится списковым с неограниченной верхней границей.   
В противном случае будет инициировано исключение.

Описание:

Добавляет значение произвольного свойства.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вызов метода может быть осуществлен только для объектов типов с установленным свойством Открытый, в противном случае будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.25.
