ОткрытьПотокДляЧтения (OpenStreamForRead)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ОткрытьПотокДляЧтения()

Возвращаемое значение:

Тип: [Поток](Поток_(Stream).md).   

Описание:

Возвращает поток для чтения содержимого вложения системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.
