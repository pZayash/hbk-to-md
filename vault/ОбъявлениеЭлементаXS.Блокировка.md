ОбъявлениеЭлементаXS (XSElementDeclaration)

Блокировка (Block)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОбъединениеНедопустимыхПодстановкиXS](ОбъединениеНедопустимыхПодстановкиXS.md).   
Определяет значение, запрещающее подстановку вместо элемента данного типа элементов, имеющих тип, являющийся расширением или ограничением данного типа. Также может запрещаться подстановка элементов из группы подстановки возглавляемой данным элементом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
