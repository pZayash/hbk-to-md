Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает возможность растягивания поля по горизонтали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.21.
