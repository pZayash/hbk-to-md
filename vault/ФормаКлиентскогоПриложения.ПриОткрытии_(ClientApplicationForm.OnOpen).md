ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриОткрытии (OnOpen)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриОткрытии(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от открытия формы. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то открытие формы произведено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает на клиенте при открытии формы, до показа окна пользователю. Следует учитывать, что открытие в данном обработчике другой формы влечет за собой сначала активацию открываемой формы, а затем активацию текущей формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
