# Расширение формы клиентского приложения для документа.Записать

Расширение формы клиентского приложения для документа (Client application form extension for documents)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Состав параметров произвольный. Содержит предопределенные параметры записи РежимЗаписи и РежимПроведения. Предопределенные параметры передаются в метод объекта [Записать](ДокументОбъект.Имя_документа.Записать_(DocumentObject.Document_name.Write).md).  
Параметры записи можно обработать в событиях формы [ПередЗаписью](Расширение_формы_клиентского_приложения_для_документа.ПередЗаписью_(Client_application_form_extension_for_documents.BeforeWrite).md), [ПередЗаписьюНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПередЗаписьюНаСервере_(Client_application_form_extension_for_documents.BeforeWriteAtServer).md), [ПриЗаписиНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПриЗаписиНаСервере_(Client_application_form_extension_for_documents.OnWriteAtServer).md), [ПослеЗаписиНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПослеЗаписиНаСервере_(Client_application_form_extension_for_documents.AfterWriteAtServer).md), [ПослеЗаписи](Расширение_формы_клиентского_приложения_для_документа.ПослеЗаписи_(Client_application_form_extension_for_documents.AfterWrite).md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - успешно записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
