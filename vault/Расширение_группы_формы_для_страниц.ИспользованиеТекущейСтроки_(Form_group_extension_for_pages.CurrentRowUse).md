Расширение группы формы для страниц (Form group extension for pages)

ИспользованиеТекущейСтроки (CurrentRowUse)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеТекущейСтроки](ИспользованиеТекущейСтроки_(CurrentRowUse).md).   
Если установлено значение [Использует](ИспользованиеТекущейСтроки.Использует_(CurrentRowUse.Use).md), то в мобильном клиенте:

- Группа автоматически скрывается из формы;
- В контекстное меню строки таблицы добавляется команда, при выборе которой отображается страница со всеми такими группами, заполненными данными, связанными со строкой, из контекстного меню которой была вызвана команда;
- Заголовок команды составляется из заголовков групп через запятую и заканчивается многоточием.

Доступность:

Мобильный клиент, сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
