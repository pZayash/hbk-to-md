АвтоИзменениеРегистраПриВводеТекста (AutoCapitalizationOnTextInput)

Предложения (Sentences)

Доступен, начиная с версии 8.3.9.

Описание:

Регистр автоматически изменяется на верхний для первой буквы первого слова в каждом предложении.

Использование в версии:

Доступен, начиная с версии 8.3.9.
