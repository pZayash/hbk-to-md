ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

ТекущийПользовательЯвляетсяУчастником (CurrentUserIsMember)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), будут получены обсуждения для текущего пользователя.  
Если установлено [Ложь](lang__def_BooleanFalse.md), будут получены все обсуждения, в которых участвуют все пользователи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
