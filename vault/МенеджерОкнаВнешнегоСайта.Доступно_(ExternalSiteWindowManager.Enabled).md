МенеджерОкнаВнешнегоСайта (ExternalSiteWindowManager)

Доступно (Enabled)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит информацию о том, доступно ли окно внешнего сайта для веб-клиента.  
Для тонкого, толстого, мобильного клиентов значение свойства всегда [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
