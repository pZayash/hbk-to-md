ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ФункциональныеОпции (FunctionalOptions)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все функциональные опции, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ФункциональнаяОпция](ОбъектМетаданных_ФункциональнаяОпция.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
