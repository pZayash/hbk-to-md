ТабличноеПоле (TableBox)

ПриИзмененииФлажка (OnCheckChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриИзмененииФлажка(<Колонка>)

Параметры:

<Колонка>

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md).   
Редактируемая колонка.

Описание:

Возникает при изменении пометки в колонке табличного поля, если он не является элементом управления.

Доступность:

Толстый клиент.

Примечание:

Если выделено несколько строк табличного поля, то событие возникает только для той строки, в которой произошло изменение флажка.

Использование в версии:

Доступен, начиная с версии 8.0.
