Описание:

Содержит настройки получения файлов в виде архива. Используется для работы методов [НачатьПолучениеФайловССервера](Глобальный_контекст.НачатьПолучениеФайловССервера.md) и [ПолучитьФайлыССервераАсинх](Глобальный_контекст.ПолучитьФайлыССервераАсинх.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
