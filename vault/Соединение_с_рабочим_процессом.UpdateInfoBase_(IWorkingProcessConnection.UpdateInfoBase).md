Соединение с рабочим процессом (IWorkingProcessConnection)

UpdateInfoBase (UpdateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateInfoBase(<ИнформационнаяБаза>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база_(IInfoBaseInfo).md).   
Информационная база с новыми значениями параметров. Новые значения могут иметь только следующие свойства объекта:

- [Descr](Информационная_база.Descr_(IInfoBaseInfo.Descr).md),
- [SessionsDenied](Информационная_база.SessionsDenied_(IInfoBaseInfo.SessionsDenied).md),
- [DBMS](Информационная_база.DBMS_(IInfoBaseInfo.DBMS).md),
- [dbServerName](Информационная_база.dbServerName_(IInfoBaseInfo.dbServerName).md),
- [dbName](Информационная_база.dbName_(IInfoBaseInfo.dbName).md),
- [dbUser](Информационная_база.dbUser_(IInfoBaseInfo.dbUser).md),
- [dbPassword](Информационная_база.dbPassword_(IInfoBaseInfo.dbPassword).md),
- [DeniedFrom](Информационная_база.DeniedFrom_(IInfoBaseInfo.DeniedFrom).md),
- [DeniedTo](Информационная_база.DeniedTo_(IInfoBaseInfo.DeniedTo).md),
- [DeniedMessage](Информационная_база.DeniedMessage_(IInfoBaseInfo.DeniedMessage).md),
- [DeniedParameter](Информационная_база.DeniedParameter_(IInfoBaseInfo.DeniedParameter).md),
- [ExternalSessionManagerConnectionString](Информационная_база.ExternalSessionManagerConnectionString_(IInfoBaseInfo.ExternalSessionManagerConnectionString).md),
- [ExternalSessionManagerRequired](Информационная_база.ExternalSessionManagerRequired_(IInfoBaseInfo.ExternalSessionManagerRequired).md),
- [LicenseDistributionAllowed](Информационная_база.LicenseDistributionAllowed_(IInfoBaseInfo.LicenseDistributionAllowed).md),
- [PermissionCode](Информационная_база.PermissionCode_(IInfoBaseInfo.PermissionCode).md),
- [SafeModeSecurityProfileName](Информационная_база.SafeModeSecurityProfileName_(IInfoBaseInfo.SafeModeSecurityProfileName).md),
- [ScheduledJobsDenied](Информационная_база.ScheduledJobsDenied_(IInfoBaseInfo.ScheduledJobsDenied).md),
- [SecurityProfileName](Информационная_база.SecurityProfileName_(IInfoBaseInfo.SecurityProfileName).md)

.

Описание:

Устанавливает новые параметры существующей информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
