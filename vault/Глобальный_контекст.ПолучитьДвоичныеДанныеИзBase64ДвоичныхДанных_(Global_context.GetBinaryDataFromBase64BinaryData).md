Глобальный контекст (Global context)

ПолучитьДвоичныеДанныеИзBase64ДвоичныхДанных (GetBinaryDataFromBase64BinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДвоичныеДанныеИзBase64ДвоичныхДанных(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Двоичные данные, закодированные по методу Base64.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Преобразует двоичные данные из формата Base64 в [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
