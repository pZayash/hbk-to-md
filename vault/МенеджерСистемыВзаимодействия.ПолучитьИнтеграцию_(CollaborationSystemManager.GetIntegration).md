МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьИнтеграцию (GetIntegration)

Доступен, начиная с версии 8.3.17.

Вариант синтаксиса: По идентификатору

Синтаксис:

ПолучитьИнтеграцию(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия_(CollaborationSystemIntegrationID).md).   
Идентификатор интеграции.

Вариант синтаксиса: По ключу

Синтаксис:

ПолучитьИнтеграцию(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ интеграции.

Возвращаемое значение:

Тип: [ИнтеграцияСистемыВзаимодействия](ИнтеграцияСистемыВзаимодействия_(CollaborationSystemIntegration).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Получает интеграцию по идентификатору или ключу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.23.
