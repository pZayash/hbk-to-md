АдминистрированиеКластер (AdministrationCluster)

ПринятьТребованияНазначения (ApplyAssignmentRules)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПринятьТребованияНазначения(<ПереназначитьВсеСервисы>)

Параметры:

<ПереназначитьВсеСервисы> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает способ применения изменений конфигурации кластера:

- [Ложь](lang__def_BooleanFalse.md) - В изменении назначения не могут принимать участия сервисы, использующие файловые хранилища данных:
  - сервис журналов регистрации,
  - сервис полнотекстового поиска,
  - сервис пользовательских настроек.
- [Истина](lang__def_BooleanTrue.md) - Выполняется переназначение всех сервисов.

  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Применяет требования назначения. После выполнения метода внесенные изменения в конфигурацию кластера начинают оказывать влияние на управление работой кластера.  
Если изменения настроек кластера противоречат друг другу, то применение изменений не выполняется и генерируется исключение.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
