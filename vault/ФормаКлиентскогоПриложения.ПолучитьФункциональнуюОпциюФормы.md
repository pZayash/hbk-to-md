ФормаКлиентскогоПриложения (ClientApplicationForm)

ПолучитьФункциональнуюОпциюФормы (GetFormFunctionalOption)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьФункциональнуюОпциюФормы(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя функциональной опции.

Возвращаемое значение:

Тип: Произвольный.   
Значение функциональной опции, применительно к экземпляру формы. Для получения значения опции используются параметры, установленные ранее с помощью метода [УстановитьПараметрыФункциональныхОпцийФормы](ФормаКлиентскогоПриложения.УстановитьПараметрыФункциональныхОпцийФормы.md).

Описание:

Получает функциональную опцию формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

Примечание:

Результат выполнения не кэшируется, вызов сервера осуществляется при каждом вызове метода.   
Если в качестве параметра функциональной опции формы была передана ссылка на группу, а реквизит, в котором хранится значение функциональной опции, не используется для группы, то при получении значения функциональной опции будет возращено Null.

Использование в версии:

Доступен, начиная с версии 8.2.
