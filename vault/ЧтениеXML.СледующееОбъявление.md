ЧтениеXML (XMLReader)

СледующееОбъявление (NextDeclaration)

Доступен, начиная с версии 8.1.

Синтаксис:

СледующееОбъявление()

Возвращаемое значение:

Тип: Булево.   
Истина - позиционирование на объявление прошло удачно, Ложь - в противном случае.

Описание:

Осуществляет переход к следующему определению DTD.  
В случае, если текущий узел является [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента.md) (DTD), текущий узел позиционируется на следующее объявление DTD.  
Если позиционирование на первое объявление еще не выполнялось, то метод аналогичен [ПерейтиКСодержимому](ЧтениеFastInfoset.ПерейтиКСодержимому.md).  
Вызов метода [ЧтениеXML.Прочитать](ЧтениеXML.Прочитать.md) прерывает процесс чтения объявлений и позиционирует текущий узел на следующий после DTD.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
