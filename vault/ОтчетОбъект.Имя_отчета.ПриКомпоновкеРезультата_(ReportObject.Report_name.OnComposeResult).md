# ОтчетОбъект.<Имя отчета>.ПриКомпоновкеРезультата

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Отчеты](Отчеты.md) › [ОтчетОбъект.<Имя отчета>](ОтчетОбъект.Имя_отчета_(ReportObject.Report_name).md) › [События](ОтчетОбъект.Имя_отчета__События.md)

ОтчетОбъект.<Имя отчета> (ReportObject.<Report name>)

ПриКомпоновкеРезультата (OnComposeResult)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриКомпоновкеРезультата(<ДокументРезультат>, <ДанныеРасшифровки>, <СтандартнаяОбработка>)

Параметры:

<ДокументРезультат>

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Документ, в который выводится результат.

<ДанныеРасшифровки>

Переменная, в которую необходимо поместить данные расшифровки.  
Если данный параметр имеет значение Null, то данные расшифровки создавать не нужно.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при выполнении отчета с помощью метода [СкомпоноватьРезультат](ОтчетОбъект.Имя_отчета.СкомпоноватьРезультат_(ReportObject.Report_name.ComposeResult).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
