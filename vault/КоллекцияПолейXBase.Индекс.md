КоллекцияПолейXBase (XBaseFieldsCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ПолеXBase>)

Параметры:

<ПолеXBase> (обязательный)

Тип: [ПолеXBase](ПолеXBase.md).   
Поле объекта XBase, индекс которого требуется определить.

Возвращаемое значение:

Тип: Число.   
Индекс поля. Если такого поля нет, возвращает -1.

Описание:

Получает индекс поля в коллекции полей XBase.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
