ЗаписьPDF (PDFWriter)

ЗаписатьУдалениеВложений (WriteAttachmentsDeletion)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ЗаписатьУдалениеВложений(<Вложения>)

Параметры:

<Вложения> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий имена файлов.

Описание:

Записывает в PDF-документ информацию о удалении вложения.  
Если в метод передан файл версии меньше 1.5, то генерируется исключение.

Доступность:

Тонкий клиент, сервер, толстый клиент.
