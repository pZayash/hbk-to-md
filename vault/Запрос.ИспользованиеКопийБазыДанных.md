Запрос (Query)

ИспользованиеКопийБазыДанных (DatabaseCopiesUse)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеКопийБазыДанных](ИспользованиеКопийБазыДанных.md).   
Свойство позволяет указать, где будет выполнен запрос - на основной базе данных или на одной из копий.   
Значение по умолчанию: [Авто](ИспользованиеКопийБазыДанных.Авто.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
