ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ИспользованиеВИтогах (UseInTotals)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то данное измерение исключается из хранимых итогов регистра.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для регистра накопления остатков (значение свойства [ВидРегистра](ОбъектМетаданных_РегистрНакопления.ВидРегистра_(MetadataObject_AccumulationRegister.RegisterType).md) равно [Остатки](ВидРегистраНакопления.Остатки_(AccumulationRegisterType.Balance).md)) это свойство не используется.  
Используется для измерений регистров накопления.

Использование в версии:

Доступен, начиная с версии 8.1.
