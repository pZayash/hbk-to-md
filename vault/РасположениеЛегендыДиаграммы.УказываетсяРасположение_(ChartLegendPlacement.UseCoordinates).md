РасположениеЛегендыДиаграммы (ChartLegendPlacement)

УказываетсяРасположение (UseCoordinates)

Доступен, начиная с версии 8.3.12.

Описание:

Расположение легенды указывается с помощью свойств [Лево](ОбластьЛегендыДиаграммы.Лево_(ChartLegendArea.Left).md), [Верх](ОбластьЛегендыДиаграммы.Верх_(ChartLegendArea.Top).md), [Низ](ОбластьЛегендыДиаграммы.Низ_(ChartLegendArea.Bottom).md) и [Право](ОбластьЛегендыДиаграммы.Право_(ChartLegendArea.Right).md) области легенды.

Использование в версии:

Доступен, начиная с версии 8.3.12.
