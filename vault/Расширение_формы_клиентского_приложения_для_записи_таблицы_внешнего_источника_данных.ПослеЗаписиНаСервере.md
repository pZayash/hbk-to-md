Расширение формы клиентского приложения для записи таблицы внешнего источника данных

ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [РегистрСведенийМенеджерЗаписи.\<Имя регистра сведений\>](РегистрСведенийМенеджерЗаписи.Имя_регистра_сведений.md).   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура.md).   
Структура, содержащая параметры записи.

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
