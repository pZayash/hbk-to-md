РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

Представление (Presentation)

Доступен, начиная с версии 8.3.20.

Синтаксис:

Представление(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение, представление которого требуется получить.

Возвращаемое значение:

Тип: Строка.   
Представление значения, аналогичное тому, которое строится методом глобального контекста [Строка](Глобальный_контекст.Строка.md). В качестве региональных установок используются данные этого объекта [РегиональныеНастройкиИнформационнойБазы](РегиональныеНастройкиИнформационнойБазы.md).

Описание:

Получает представление значения по правилам, определяемым значениями свойств данного объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
