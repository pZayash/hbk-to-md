ПапкаПолейНабораДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetFieldFolder)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит путь к данным, под которым поле будет фигурировать в настройках и выражениях. Обратим внимание, что поле будет фигурировать именно под данным именем, а не под именем, указанным в свойстве [Поле](ПолеНабораДанныхСхемыКомпоновкиДанных.Поле_(DataCompositionSchemaDataSetField.Field).md).   
В случае, если несколько наборов данных имеют поля с одинаковыми путями к данным, будет использовано поле набора данных, являющимся родительским набором данных. Указание одинаковых путей к данным полей не связанных наборов данных не допускается.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В системе компоновки данных не допускается использовать в качестве пути к данным имена, совпадающие с ключевыми словами:  
ВЫБОР (CASE)  
ЕСТЬ (IS)  
ИНАЧЕ (ELSE)  
КОГДА (WHEN)  
КОНЕЦ (END)  
НЕ (NOT)  
ПОДОБНО (LIKE)  
РАЗЛИЧНЫЕ (DISTINCT)  
СПЕЦСИМВОЛ (ESCAPE)  
ТОГДА (THEN)

Использование в версии:

Доступен, начиная с версии 8.1.
