# ЗадачаОбъект.<Имя задачи>.ОбработкаПроверкиВыполнения

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ОбработкаПроверкиВыполнения (OnCheckExecutionProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаПроверкиВыполнения(<Результат>)

Параметры:

<Результат>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в параметре установить [Истина](lang__def_BooleanTrue.md), то будет вызван обработчик [ОбработкаПроверкиВыполнения](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаПроверкиВыполнения_(BusinessProcessRoutePointRef.Business_process_name.OnCheckExecuteProcessing).md).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при выполнении метода [ПроверитьВыполнение](ЗадачаОбъект.Имя_задачи.ПроверитьВыполнение_(TaskObject.Task_name.CheckExecution).md). Вызывается перед вызовом обработчика события [ОбработкаПроверкиВыполнения](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаПроверкиВыполнения_(BusinessProcessRoutePointRef.Business_process_name.OnCheckExecuteProcessing).md) у соответствующей точки маршрута бизнес-процесса. Предназначено для проверки достаточных условий, чтобы считать задачу выполненной. Это отличает его от события [ПередВыполнением](ЗадачаОбъект.Имя_задачи.ПередВыполнением_(TaskObject.Task_name.BeforeExecute).md), которое предназначено для проверки необходимых условий выполнения задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Не вызывается, если задача уже выполнена.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
