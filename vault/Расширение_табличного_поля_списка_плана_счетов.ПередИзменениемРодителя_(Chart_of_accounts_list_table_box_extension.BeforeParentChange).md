# Расширение табличного поля списка плана счетов.ПередИзменениемРодителя

Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

ПередИзменениемРодителя (BeforeParentChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИзменениемРодителя(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения родителя. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), изменение выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается перед выполнением изменения родителя текущего элемента с помощью команды списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
