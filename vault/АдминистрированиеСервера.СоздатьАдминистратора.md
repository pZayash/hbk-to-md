АдминистрированиеСервера (ServerAdministration)

СоздатьАдминистратора (CreateAdministrator)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьАдминистратора()

Описание:

Создает администратора кластера с пустыми значениями свойств. Этот объект может быть использован для регистрации нового администратора после установки требуемых значений свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
