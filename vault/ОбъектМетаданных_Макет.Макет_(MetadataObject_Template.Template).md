ОбъектМетаданных: Макет (MetadataObject: Template)

Макет (Template)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
Для получения макета можно использовать метод глобального контекста [ПолучитьОбщийМакет](Глобальный_контекст.ПолучитьОбщийМакет_(Global_context.GetCommonTemplate).md)() или методы ПолучитьМакет() соответствующих прикладных объектов.

Использование в версии:

Доступен, начиная с версии 8.1.
