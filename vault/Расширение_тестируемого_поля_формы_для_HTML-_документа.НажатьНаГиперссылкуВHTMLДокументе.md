Расширение тестируемого поля формы для HTML- документа (Tested form field extension for a HTML-document)

НажатьНаГиперссылкуВHTMLДокументе (ClickHTMLDocumentHyperlink)

Доступен, начиная с версии 8.3.25.

Синтаксис:

НажатьНаГиперссылкуВHTMLДокументе(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Строка, Число.   
Представление или порядковый номер гиперссылки в документе (индекс строки, нумерация с 0).

Описание:

Генерирует событие обработки нажатия ссылки в HTML-документе.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.

Описание изменено в версии 8.3.26.
