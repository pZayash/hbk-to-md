**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Решение СЛУ](Решение_СЛУ.md)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

[ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.  
Возможно обращение к к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Описание:

Представляет собой коллекцию объектов типа [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
