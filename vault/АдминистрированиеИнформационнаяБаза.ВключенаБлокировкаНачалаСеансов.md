АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ВключенаБлокировкаНачалаСеансов (SessionsLockEnabled)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет режимом блокировки сеансов.   
Если установлено значение Истина и текущее время находится в диапазоне, определяемом свойствами [ВремяНачалаБлокировки](АдминистрированиеИнформационнаяБаза.ВремяНачалаБлокировки.md) и [ВремяОкончанияБлокировки](АдминистрированиеИнформационнаяБаза.ВремяОкончанияБлокировки.md), то начало новых сеансов с информационной базой запрещено. Если при этом попытаться установить новый сеанс, то будет сгенерировано сообщение об ошибке, содержащее текст из значения свойства [СообщениеБлокировки](АдминистрированиеИнформационнаяБаза.СообщениеБлокировки.md).   
Для начала сеанса вопреки блокировке сеансов необходимо указать код разрешения, определяемый свойством [КодРазрешенияНачалаСеансов](АдминистрированиеИнформационнаяБаза.КодРазрешенияНачалаСеансов.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
