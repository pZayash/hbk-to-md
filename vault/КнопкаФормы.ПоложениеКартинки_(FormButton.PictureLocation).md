ПоложениеКартинки (PictureLocation)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеКартинкиКнопкиФормы](ПоложениеКартинкиКнопкиФормы_(FormButtonPictureLocation).md).   
Задает расположение картинки для кнопки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
