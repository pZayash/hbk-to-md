# МакетОформленияКомпоновкиДанных.Очистить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макет оформления](Макет_оформления.md) › [МакетОформленияКомпоновкиДанных](МакетОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplate).md) › [Методы](МакетОформленияКомпоновкиДанных__Методы.md)

МакетОформленияКомпоновкиДанных (DataCompositionAppearanceTemplate)

Очистить (Clear)

Доступен, начиная с версии 8.1.

Синтаксис:

Очистить()

Описание:

Очищает коллекцию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод может быть вызван в том случае, если коллекция не связана с доступными параметрами. В противном случае генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
