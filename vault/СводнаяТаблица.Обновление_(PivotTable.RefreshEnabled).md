СводнаяТаблица (PivotTable)

Обновление (RefreshEnabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак, отвечающий за необходимость обновления данных табличного документа при изменениях состава измерений на осях и ресурсов.  
[Истина](lang__def_BooleanTrue.md) - требуется обновление.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Перед установкой колонок рекомендуется установить [Ложь](lang__def_BooleanFalse.md).   
После завершения настройки установите [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
