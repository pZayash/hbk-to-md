ПоляПостроителяОтчета (ReportBuilderFields)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеПостроителяОтчета](ПолеПостроителяОтчета_(ReportBuilderField).md).   
Искомое поле.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
