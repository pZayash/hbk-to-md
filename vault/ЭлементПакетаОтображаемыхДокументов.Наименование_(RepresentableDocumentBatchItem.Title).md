ЭлементПакетаОтображаемыхДокументов (RepresentableDocumentBatchItem)

Наименование (Title)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Наименование элемента пакета отображаемых документов. При сохранении пакета отображаемых документов в формате XLS, XLSX, ODS значение данного поля используется в качестве имени листа итогового документа.  
Следующие символы при записи будут заменены на символ подчеркивания "\_":   
":", "/", "\", "?", "\*", "[", "]", "`".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если длина значения превышает 31 символ, при записи она будет сокращена до 31 символа.

Использование в версии:

Доступен, начиная с версии 8.3.12.
