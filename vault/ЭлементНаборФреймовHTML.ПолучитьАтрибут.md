ЭлементНаборФреймовHTML (HTMLFrameSetElement)

ПолучитьАтрибут (GetAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьАтрибут(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.   
Имя атрибута.

Возвращаемое значение:

Тип: Строка, Неопределено.   

Описание:

Возвращает значение атрибута по имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение отсутствует или атрибут не обнаружен, то будет возвращено значение Неопределено.

Использование в версии:

Доступен, начиная с версии 8.2.
