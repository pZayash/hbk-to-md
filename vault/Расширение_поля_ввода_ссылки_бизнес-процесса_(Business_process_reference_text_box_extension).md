# Расширение поля ввода ссылки бизнес-процесса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора бизнес-процесса в элементе управления [ПолеВвода](ПолеВвода_(TextBox).md) по введенному тексту. По умолчанию поиск осуществляется по полю "Номер".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
