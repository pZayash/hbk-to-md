ОписаниеОтображаемогоОбъектаPDF (PDFRepresentationObjectDescription)

Высота (Height)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит высоту объекта (в миллиметрах) на странице PDF-документа.   
Если не указано, то высота будет определена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Используется, если значение свойства [ВысотаВМесяцах](Расширение_поля_формы_для_поля_календаря.ВысотаВМесяцах.md)=0 или для режимов совместимости до [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md).

Использование в версии:

Доступен, начиная с версии 8.3.18.
