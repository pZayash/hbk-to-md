ЗаписьТекста (TextWriter)

Открыть (Open)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть файл

Синтаксис:

Открыть(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <Дописывать>, <КонвертируемыйРазделительСтрок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя записываемого файла.

<Кодировка> (необязательный)

Тип кодировки текста. Тип кодировки может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Разделяет строки в записываемом файле.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<Дописывать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр принимает значение [Истина](lang__def_BooleanTrue.md), то содержимое файла сохраняется, если файл уже существует. Если файла не существует, создается новый с именем, указанным в первом параметре метода.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает разделитель строк в файле при конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание варианта метода:

Открывает текстовый файл для записи в монопольном режиме.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Разделяет строки в записываемом потоке.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает разделитель строк в потоке при конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли добавлена метка порядка байт (BOM) в начало потока.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Устанавливает поток для записи.

Описание:

Открывает текстовый файл или поток для записи.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Открыть("d:\dos.txt", КодировкаТекста.OEM); |

Использование в версии:

Доступен, начиная с версии 8.0.
