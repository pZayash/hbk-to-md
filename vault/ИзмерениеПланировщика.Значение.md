ИзмерениеПланировщика (PlannerDimension)

Значение (Value)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Значение измерения, по которому осуществляется поиск с помощью метода [Найти](КоллекцияИзмеренийПланировщика.Найти.md) коллекции измерений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[КоллекцияИзмеренийПланировщика](КоллекцияИзмеренийПланировщика.md), метод [Найти](КоллекцияИзмеренийПланировщика.Найти.md)

Доступен, начиная с версии 8.3.6.
