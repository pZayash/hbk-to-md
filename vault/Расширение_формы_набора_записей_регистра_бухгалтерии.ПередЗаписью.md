Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением записи набора из формы. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.ПередЗаписью.md) объекта [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
