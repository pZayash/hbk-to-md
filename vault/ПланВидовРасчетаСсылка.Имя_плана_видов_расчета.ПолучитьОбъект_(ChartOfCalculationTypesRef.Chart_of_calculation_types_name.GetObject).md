ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)>, [Неопределено](lang__def_Undefined.md).

Описание:

Получает по ссылке объект базы данных, который можно изменить, записать или удалить.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
