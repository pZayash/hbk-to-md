МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьПриложенияАбонента (GetSubscriberApplications)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьПриложенияАбонента()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [ПриложениеСистемыВзаимодействия](ПриложениеСистемыВзаимодействия_(CollaborationSystemApplication).md).

Описание:

Возвращает список приложений абонента системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
