ЭлементПланировщика (PlannerItem)

Родитель (Parent)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), [Неопределено](lang__def_Undefined.md).   
Если элемент является замещающим в расписании, свойство содержит оригинальный элемент, в котором задано расписание и один из повторенных элементов которого замещает данный элемент. Иначе - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
