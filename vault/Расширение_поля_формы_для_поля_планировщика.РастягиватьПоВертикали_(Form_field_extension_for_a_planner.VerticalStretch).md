Расширение поля формы для поля планировщика (Form field extension for a planner)

РастягиватьПоВертикали (VerticalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает возможность растягивания поля по вертикали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
