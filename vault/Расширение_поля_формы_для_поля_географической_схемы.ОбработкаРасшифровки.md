Расширение поля формы для поля географической схемы (Form field extension for a geographical schema field)

ОбработкаРасшифровки (DetailProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаРасшифровки(<Расшифровка>, <СтандартнаяОбработка>)

Параметры:

<Расшифровка>

Тип: Произвольный.   
Расшифровка выбранного значения.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается при выборе значения.

Доступность:

Веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
