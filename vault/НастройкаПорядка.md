Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияПорядком](ЭлементУправленияПорядком.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки порядка.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция элементов управления порядком.

Доступность:

Сервер, толстый клиент, внешнее соединение.
