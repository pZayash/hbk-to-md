ФормаКлиентскогоПриложения (ClientApplicationForm)

ПроверятьЗаполнениеАвтоматически (AutoFillCheck)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает необходимость автоматической проверки заполнения формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

На клиенте, тонком клиенте и Веб-клиенте свойство доступно только для чтения.

Использование в версии:

Доступен, начиная с версии 8.2.
