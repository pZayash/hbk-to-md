ФормаКлиентскогоПриложения (ClientApplicationForm)

Элементы (Items)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ВсеЭлементыФормы](ВсеЭлементыФормы.md).   
Содержит коллекцию всех элементов формы клиентского приложения для прямого доступа вне зависимости от иерархии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
