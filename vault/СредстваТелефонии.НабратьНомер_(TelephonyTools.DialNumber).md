СредстваТелефонии (TelephonyTools)

НабратьНомер (DialNumber)

Доступен, начиная с версии 8.3.5.

Синтаксис:

НабратьНомер(<Номер>, <ВызватьСразу>)

Параметры:

<Номер> (обязательный)

Тип: [Строка](lang__def_String.md).   
Набираемый номер.

<ВызватьСразу> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли набранный номер сразу вызван. Если нет, то пользователь должен сам нажать кнопку вызова.

Описание:

Запускает системное приложение "Телефон" с переданным номером.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства следует использовать метод [ПоддерживаетсяНаборНомера](СредстваТелефонии.ПоддерживаетсяНаборНомера_(TelephonyTools.NumberDialingSupported).md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
