Элементы коллекции:

[ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса](ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса.md), [ПараметрыТаблицыСхемыЗапроса](ПараметрыТаблицыСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры компоновки данных таблицы.  
Возможно обращение к параметру компоновки данных таблицы посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПараметрыКомпоновкиДанныхТаблицыСхемыЗапроса.Индекс.md)  
[Количество (Count)](ПараметрыКомпоновкиДанныхТаблицыСхемыЗапроса.Количество.md)  
[Получить (Get)](ПараметрыКомпоновкиДанныхТаблицыСхемыЗапроса.Получить.md)  

Описание:

Содержит коллекцию параметров компоновки данных схемы запроса. Количество параметров и тип каждого из них зависит от самой таблицы. Каждый параметр может быть одного из следующих типов:

- ПараметрТаблицыСхемыЗапроса - в таком случае параметр компоновки может принимать значение только параметра, указанного с символом "", например, Парам;
- ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса - в таком случае параметр представляет собой набор параметров, каждый из которых может принимать либо значение обычного параметра (имя которого указано с символом "", например, Парам), либо имя доступного поля данной таблицы схемы запроса. При использовании поля таблицы схемы запроса параметр может содержать признак [ИспользоватьРеквизиты](ПолеВыбораКомпоновкиДанныхСхемыЗапроса.ИспользоватьРеквизиты.md) и псевдоним.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
