Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: Число, [ПоследовательностьЗапись.](ПоследовательностьЗапись.Имя_последовательности.md)<[Имя последовательности](ПоследовательностьЗапись.Имя_последовательности.md)>.   
Удаляемая из набора запись. Может быть указан индекс записи в наборе или сама запись.

Описание:

Удаляет указанную запись регистрации документа из набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
