# МенеджерОтображенияРекламы.ЗагрузитьПолноэкраннуюРекламуАсинх

МенеджерОтображенияРекламы (AdRepresentationManager)

ЗагрузитьПолноэкраннуюРекламуАсинх (DownloadFullscreenAdAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗагрузитьПолноэкраннуюРекламуАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Булево](lang__Булево_(Boolean).md), признак загрузки видеообъявления.

Описание:

Асинхронная функция. Загружает из сети видеообъявление. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [ПоказатьВидеообъявлениеСВознаграждением](МенеджерОтображенияРекламы.ПоказатьВидеообъявлениеСВознаграждением_(AdRepresentationManager.ShowRewardedVideo).md) видеообъявление будет показано без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы_(AdRepresentationManager).md), метод [НачатьЗагрузкуПолноэкраннойРекламы](МенеджерОтображенияРекламы.НачатьЗагрузкуПолноэкраннойРекламы_(AdRepresentationManager.BeginFullscreenAdDownloading).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
