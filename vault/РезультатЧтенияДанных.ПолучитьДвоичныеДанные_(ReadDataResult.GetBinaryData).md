РезультатЧтенияДанных (ReadDataResult)

ПолучитьДвоичныеДанные (GetBinaryData)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьДвоичныеДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Получает результат чтения в виде двоичных данных.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
