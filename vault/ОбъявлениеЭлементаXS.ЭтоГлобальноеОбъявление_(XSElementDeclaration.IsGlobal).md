ОбъявлениеЭлементаXS (XSElementDeclaration)

ЭтоГлобальноеОбъявление (IsGlobal)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, имеет ли объявление глобальную область видимости.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - объявление имеет глобальную область видимости; [Ложь](lang__def_BooleanFalse.md) - объявление содержится внутри определений составного типа, группы атрибутов, группы модели.

Использование в версии:

Доступен, начиная с версии 8.1.
