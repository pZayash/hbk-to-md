СериализаторXDTO (XDTOSerializer)

ВозможностьЧтенияXML (CanReadXML)

Доступен, начиная с версии 8.1.

Синтаксис:

ВозможностьЧтенияXML(<ЧтениеXML>)

Параметры:

<ЧтениеXML> (обязательный)

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM.md), [ЧтениеFastInfoset](ЧтениеFastInfoset.md), [ЧтениеXML](ЧтениеXML.md).   
Объект, через который производится чтение XML.

Возвращаемое значение:

Тип: Булево.   
Истина - тип 1С:Предприятия существует; Ложь - в противном случае.

Описание:

Производит проверку возможности чтения из объекта [ЧтениеXML](ЧтениеXML.md) значения.   
Данный метод получает тип данных XML из объекта [ЧтениеXML](ЧтениеXML.md) аналогично методу [ПолучитьXMLТип](СериализаторXDTO.ПолучитьXMLТип.md), а затем пытается определить, имеется ли соответствующий тип 1С:Предприятия аналогично тому, как это делает метод [ИзXMLТипа](СериализаторXDTO.ИзXMLТипа.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не производит чтения. Он определяет возможность чтения указанного типа в текущем положении объекта чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
