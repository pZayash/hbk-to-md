Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для бизнес-процессов. Для данного расширения свойство [ТекущаяСтрока](ПолеСписка.ТекущаяСтрока.md) содержит значение типа [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [БизнесПроцессСписок.](БизнесПроцессСписок.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСписок.Имя_бизнес-процесса.md)>.

Доступность:

Толстый клиент.

Доступен, начиная с версии 8.0.
