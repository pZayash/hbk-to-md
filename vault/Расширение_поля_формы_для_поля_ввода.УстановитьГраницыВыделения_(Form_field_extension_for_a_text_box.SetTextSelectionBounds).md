Расширение поля формы для поля ввода (Form field extension for a text box)

УстановитьГраницыВыделения (SetTextSelectionBounds)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: По позициям начала и конца выделения в тексте

Синтаксис:

УстановитьГраницыВыделения(<НачальнаяПозиция>, <КонечнаяПозиция>)

Параметры:

<НачальнаяПозиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция начала выделения в тексте.

<КонечнаяПозиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Конечная позиция выделения в тексте.

Вариант синтаксиса: По номерам строк и колонок начала и конца выделения в тексте

Синтаксис:

УстановитьГраницыВыделения(<НачалоСтроки>, <НачалоКолонки>, <КонецСтроки>, <КонецКолонки>)

Параметры:

<НачалоСтроки> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер строки начала выделения.

<НачалоКолонки> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер колонки начала выделения.

<КонецСтроки> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер строки конца выделения.

<КонецКолонки> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер колонки конца выделения.

Описание варианта метода:

Внимание! Данный вариант метода не поддерживается в мобильном приложении и мобильном клиенте.

Описание:

Устанавливает границы выделения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод можно использовать при открытом окне, в том числе в серверных вызовах.

Использование в версии:

Доступен, начиная с версии 8.2.
