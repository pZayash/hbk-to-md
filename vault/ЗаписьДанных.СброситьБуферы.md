ЗаписьДанных (DataWriter)

СброситьБуферы (Flush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

СброситьБуферы()

Описание:

Сбрасывает все внутренние буферы в целевой поток, после чего вызывает метод [СброситьБуферы](Поток.СброситьБуферы.md) целевого потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [НачатьСбросБуферов](ЗаписьДанных.НачатьСбросБуферов.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [СброситьБуферыАсинх](ЗаписьДанных.СброситьБуферыАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
