[ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных_(DataCompositionParameterValue).md)

Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Параметры редактирования компоновки данных. Реализованы в виде значений параметров. Для оформления используются следующие элементы оформления:

- Маска (Mask) - тип значения [Строка](lang__def_String.md);
- СвязиПараметровВыбора (ChoiceParameterLinks) - тип значения [СвязиПараметровВыбораКомпоновкиДанных](СвязиПараметровВыбораКомпоновкиДанных_(DataCompositionChoiceParameterLinks).md);
- СвязьПоТипу (TypeLink) - тип значения [СвязьПоТипуКомпоновкиДанных](СвязьПоТипуКомпоновкиДанных_(DataCompositionTypeLink).md);
- ФормаВыбора (ChoiceForm) - тип значения [Строка](lang__def_String.md);
- ФорматРедактирования (EditFormat) - тип значения [Строка](lang__def_String.md);
- ВыборГруппИЭлементов (ChoiceFoldersAndItems) - возможность выбора групп и элементов - тип значения [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов_(FoldersAndItemsUse).md);
- ПараметрыВыбора (ChoiceParameters) - параметры выбора - тип значения [ПараметрыВыбораКомпоновкиДанных](ПараметрыВыбораКомпоновкиДанных_(DataCompositionChoiceParameters).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: Appearance.
