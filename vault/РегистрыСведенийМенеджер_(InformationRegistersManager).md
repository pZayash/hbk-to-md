# РегистрыСведенийМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[РегистрСведенийМенеджер.\<Имя регистра сведений\>](РегистрСведенийМенеджер.Имя_регистра_сведений_(InformationRegisterManager.Information_register_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров сведений.  
Возможно обращение к менеджеру регистра сведений посредством оператора [...]. В качестве аргумента передается имя регистра сведений, как оно задано в конфигурации.  

Свойства:

[\<Имя регистра сведений\> (\<Information register name\>)](РегистрыСведенийМенеджер.Имя_регистра_сведений_(InformationRegistersManager.Information_register_name).md)  

Описание:

Предназначен для управления регистрами сведений и предоставляет доступ к значениям типа [РегистрСведенийМенеджер.](РегистрСведенийМенеджер.Имя_регистра_сведений_(InformationRegisterManager.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийМенеджер.Имя_регистра_сведений_(InformationRegisterManager.Information_register_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [РегистрыСведений](Глобальный_контекст.РегистрыСведений_(Global_context.InformationRegisters).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| // менеджер регистров сведений предоставляет доступ к менеджеру  // конкретного регистра (Курсы валют).  Форма = РегистрыСведений.КурсыВалют.ПолучитьФормуСписка(); Форма.Открыть(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [РегистрыСведений](Глобальный_контекст.РегистрыСведений_(Global_context.InformationRegisters).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
