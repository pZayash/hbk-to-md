Файл (File)

СуществуетАсинх (ExistsAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СуществуетАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Истина](lang__def_BooleanTrue.md), если файл существует, иначе - [Ложь](lang__def_BooleanFalse.md).

Описание:

Асинхронная функция. Проверяет, существует ли файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы метода на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.
