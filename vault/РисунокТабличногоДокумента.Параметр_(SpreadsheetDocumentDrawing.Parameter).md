РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Параметр (Parameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя параметра, значение которого будет использоваться при выводе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступно только для рисунка типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md).

Использование в версии:

Доступен, начиная с версии 8.0.
