Расширение формы клиентского приложения для документа (Client application form extension for documents)

ЗначениеКопирования (CopyingValue)

Доступен, начиная с версии 8.2.

Описание:

Значение для копирования.

Примечание:

Тип параметра соответствует типу объекта, копия которого будет создана.

Использование в версии:

Доступен, начиная с версии 8.2.
