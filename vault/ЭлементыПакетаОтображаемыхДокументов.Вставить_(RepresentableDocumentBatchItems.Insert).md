ЭлементыПакетаОтображаемыхДокументов (RepresentableDocumentBatchItems)

Вставить (Insert)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Вставить(<Индекс>, <АдресДокумента>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<АдресДокумента> (необязательный)

Тип: [Строка](lang__def_String.md).   
Адрес временного хранилища, в котором расположен документ.

Возвращаемое значение:

Тип: [ЭлементПакетаОтображаемыхДокументов](ЭлементПакетаОтображаемыхДокументов_(RepresentableDocumentBatchItem).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
