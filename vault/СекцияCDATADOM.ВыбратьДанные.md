СекцияCDATADOM (DOMCDATASection)

ВыбратьДанные (SubstringData)

Доступен, начиная с версии 8.1.

Синтаксис:

ВыбратьДанные(<Смещение>, <Длина>)

Параметры:

<Смещение> (обязательный)

Тип: Число.   
Смещение от начала символьных данных.

<Длина> (обязательный)

Тип: Число.   
Длина выделяемой подстроки.

Возвращаемое значение:

Тип: Строка.   

Описание:

Выделяет подстроки из символьных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных.

Использование в версии:

Доступен, начиная с версии 8.1.
