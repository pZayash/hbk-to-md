Планировщик (Planner)

ЕдиницаПериодическогоВарианта (PeriodicVariantUnit)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ТипЕдиницыШкалыВремени](ТипЕдиницыШкалыВремени.md).   
Используется вместе со свойством [КратностьПериодическогоВарианта](Планировщик.КратностьПериодическогоВарианта.md) для указания длительности шкалы времени до её переноса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[Планировщик](Планировщик.md), свойство [КратностьПериодическогоВарианта](Планировщик.КратностьПериодическогоВарианта.md)
