`Послать`(`<Сообщение>`: [`ПочтовоеСообщение`](ПочтовоеСообщение.md), `<ЗапросПодтверждения>`: `Булево`)

Почта (Mail)

Послать (Send)

Доступен, начиная с версии 8.0.

Синтаксис:

Послать(<Сообщение>, <ЗапросПодтверждения>)

Параметры:

<Сообщение> (обязательный)

Тип: [ПочтовоеСообщение](ПочтовоеСообщение.md).   
Почтовое сообщение.

<ЗапросПодтверждения> (необязательный)

Тип: Булево.   
Определяет, требуется ли запрос на подтверждение о получении письма адресатом.  
Значение по умолчанию: Ложь.

Описание:

Посылает сообщение типа [ПочтовоеСообщение](ПочтовоеСообщение.md).

Доступность:

Тонкий клиент, толстый клиент.

Пример:

|  |
| --- |
| Почта = Новый Почта; Сообщение = Новый ПочтовоеСообщение; Почта.Подключиться(); Почта.Послать(Сообщение); Почта.Отключиться(); |

Использование в версии:

Доступен, начиная с версии 8.0.
