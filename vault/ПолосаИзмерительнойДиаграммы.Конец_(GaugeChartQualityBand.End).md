ПолосаИзмерительнойДиаграммы (GaugeChartQualityBand)

Конец (End)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит конец полосы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Измеряется в абсолютной величине. Значение не должно быть меньше значение свойства [Начало](ПолосаИзмерительнойДиаграммы.Начало_(GaugeChartQualityBand.Begin).md).

Использование в версии:

Доступен, начиная с версии 8.0.
