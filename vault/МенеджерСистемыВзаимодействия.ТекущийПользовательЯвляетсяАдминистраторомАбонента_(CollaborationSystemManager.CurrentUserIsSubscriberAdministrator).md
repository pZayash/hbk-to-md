МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ТекущийПользовательЯвляетсяАдминистраторомАбонента (CurrentUserIsSubscriberAdministrator)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ТекущийПользовательЯвляетсяАдминистраторомАбонента()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Возвращает [Истина](lang__def_BooleanTrue.md), если текущий пользователь системы взаимодействия является администратором абонента.  
Если приложение не зарегистрировано на сервере системы взаимодействия, то возвращается значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
