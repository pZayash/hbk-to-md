ЧтениеPDF (PDFReader)

ОткрытьАсинх (OpenAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Открыть файл

Синтаксис:

ОткрытьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для чтения PDF-документа.

Вариант синтаксиса: Установить поток

Синтаксис:

ОткрытьАсинх(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения PDF-документа.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Открывает PDF-документ для чтения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
