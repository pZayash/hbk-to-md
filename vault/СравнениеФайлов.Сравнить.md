СравнениеФайлов (FileCompare)

Сравнить (Compare)

Доступен, начиная с версии 8.0.

Синтаксис:

Сравнить()

Возвращаемое значение:

Тип: Булево.   
Истина - файлы равны, Ложь – в противном случае.

Описание:

Сравнивает файлы. Позволяет узнать, есть ли в них различия.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
