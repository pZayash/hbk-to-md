БазовыеВидыРасчета.<Имя плана видов расчета> (BaseCalculationTypes.<Chart of calculation types name>)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Значение>, <Колонки>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Искомое значение.

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка имен колонок, разделенных запятыми, в которых будет осуществляться поиск.  
Если параметр не указан, поиск осуществляется по всей табличной части.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [БазовыеВидыРасчетаСтрока.](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета_(BaseCalculationTypesRow.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета_(BaseCalculationTypesRow.Chart_of_calculation_types_name).md)>, [Неопределено](lang__def_Undefined.md).   
Строка табличной части, которая содержит искомое значение. Если значение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск значения в указанных колонках табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод эффективно использовать для поиска уникальных значений.

Использование в версии:

Доступен, начиная с версии 8.0.
