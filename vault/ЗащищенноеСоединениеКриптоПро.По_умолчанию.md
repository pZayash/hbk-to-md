ЗащищенноеСоединениеКриптоПро (CryptoProSecureConnection)

По объектам используемых сертификатов

Доступен, начиная с версии 8.3.24.

Синтаксис:

Новый ЗащищенноеСоединениеКриптоПро(<СертификатКлиента>, <СертификатыУдостоверяющихЦентров>, <РежимПроверкиОтзываTLSСертификатаСервера>)

Параметры:

<СертификатКлиента> (необязательный)

Тип: [СертификатКлиентаОС](СертификатКлиентаОС.md), [СертификатКриптографии](СертификатКриптографии.md), [СертификатКлиентаФайл](СертификатКлиентаФайл.md), Неопределено.   
Объект источника клиентского сертификата.  
Если не указан, то клиентский сертификат не предоставляется.  
Значение по умолчанию: Неопределено.

<СертификатыУдостоверяющихЦентров> (необязательный)

Тип: [СертификатыУдостоверяющихЦентровФайл](СертификатыУдостоверяющихЦентровФайл.md), [СертификатыУдостоверяющихЦентровОС](СертификатыУдостоверяющихЦентровОС.md), Неопределено.   
Объект источника сертификатов удостоверяющих центров.  
Если не указан, то проверка сертификата сервера не осуществляется.  
Значение по умолчанию: Неопределено.

<РежимПроверкиОтзываTLSСертификатаСервера> (необязательный)

Тип: [РежимПроверкиОтзываTLSСертификатаСервера](РежимПроверкиОтзываTLSСертификатаСервера.md).   
Режим проверки отзыва TLS-сертификата сервера.  
Значение по умолчанию: Неопределено.

Описание:

Создает объект защищенного соединения КриптоПро,  
использующего указанные источники клиентского сертификата и сертификатов удостоверяющих центров.  
В Linux и macOS, при вызове конструктора, если не установлена КриптоПРО СPS, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.24.
