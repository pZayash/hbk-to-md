ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

КонецСтраницы (PageBottom)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что данная область находится последней на странице.  
[Истина](lang__def_BooleanTrue.md) - область является последней.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо только для областей строк/колонок.

Использование в версии:

Доступен, начиная с версии 8.2.
