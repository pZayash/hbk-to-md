Доступен, начиная с версии 8.1.

Элементы коллекции:

[ТипЗначенияXDTO](ТипЗначенияXDTO.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются типы значений XDTO.  
Возможно обращение к типу значения XDTO посредством оператора [...]. В качестве аргумента передается индексу (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияТиповЗначенийXDTO.Количество.md)  
[Получить (Get)](КоллекцияТиповЗначенийXDTO.Получить.md)  

Описание:

Коллекция типов значений модели XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
