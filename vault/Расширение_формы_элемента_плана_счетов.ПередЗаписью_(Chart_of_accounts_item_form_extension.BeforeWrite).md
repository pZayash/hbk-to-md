# Расширение формы элемента плана счетов.ПередЗаписью

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [Расширение формы элемента плана счетов](Расширение_формы_элемента_плана_счетов_(Chart_of_accounts_item_form_extension).md) › [События](Расширение_формы_элемента_плана_счетов__События.md)

Расширение формы элемента плана счетов (Chart of accounts item form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи элемента. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись элемента выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи объекта из формы. Процедура-обработчик данного события вызывается перед вызовом процедуры обработчика события [ПередЗаписью](ПланСчетовОбъект.Имя_плана_счетов.ПередЗаписью_(ChartOfAccountsObject.Chart_of_accounts_name.BeforeWrite).md) объекта [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
