Описание:

Используется для доступа к свойствам командной панели в целом. Представляет собой древовидную структуру кнопок командного интерфейса.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма = Справочники.Конкуренты.ПолучитьФормуСписка(); КП = Форма.ЭлементыФормы.КоманднаяПанель1; НоваяРамка   = Новый Рамка(ТипРамкиЭлементаУправления.Двойная); КП.Рамка     = НоваяРамка; КП.ЦветФона  = WebЦвета.ГолубойСКраснымОттенком; КП.Подсказка = "Выводится при наведении курсора на рамку панели"; Форма.Открыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
