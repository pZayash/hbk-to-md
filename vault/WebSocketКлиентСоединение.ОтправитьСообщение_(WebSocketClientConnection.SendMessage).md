WebSocketКлиентСоединение (WebSocketClientConnection)

ОтправитьСообщение (SendMessage)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ОтправитьСообщение(<Сообщение>)

Параметры:

<Сообщение> (необязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Сообщение для отправки.

Описание:

Выполняет отправку данных (в виде сообщения) серверу, с которым выполнено соединение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
