ДиалогВыбораФайла (FileDialog)

ПолноеИмяФайла (FullFileName)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит начальное полное имя файла. После выбора файла содержит имя выбранного файла (кроме режима множественного выбора файлов или выбора каталога).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
