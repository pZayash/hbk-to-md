ДиалогВыбораПользователейИсторииДанных (DataHistoryUsersChooseDialog)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ДиалогВыбораПользователейИсторииДанных(<Данные>, <Пользователи>)

Параметры:

<Данные> (обязательный)

Тип: [СправочникСсылка.](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)<[Имя справочника](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)<[Имя документа](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)>, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)<[Имя задачи](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)>.   
Данные, по которым отбираются версии. Используется для формирования списка доступных пользователей-авторов версий.

<Пользователи> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив полных имен пользователей.  
Устанавливается, если необходимо заполнить параметры отбора при открытии диалога. Если диалог был закрыт нажатием кнопки "OK", параметр содержит результат выбора.  
Значение по умолчанию: Пустой массив.

Описание:

Создает диалог отбора пользователей версий указанных данных.

Использование в версии:

Доступен, начиная с версии 8.3.13.
