HTTPСоединение (HTTPConnection)

ОтправитьДляОбработки (Post)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ОтправитьДляОбработкиАсинх](HTTPСоединение.ОтправитьДляОбработкиАсинх.md)

Доступен, начиная с версии 8.0.

Синтаксис:

ОтправитьДляОбработки(<HTTPЗапрос>, <ИмяВыходногоФайла>)

Параметры:

<HTTPЗапрос> (обязательный)

Тип: [HTTPЗапрос](HTTPЗапрос.md).   
HTTP-запрос.

<ИмяВыходногоФайла> (необязательный)

Тип: Строка.   
Имя выходного файла, в который записываются полученные с сервера данные.  
Если не указан или содержит пустую строку, то тело ответа может быть получено из объекта [HTTPОтвет](HTTPОтвет.md).

Возвращаемое значение:

Тип: [HTTPОтвет](HTTPОтвет.md).   

Описание:

Отправляет данные на указанный адрес для обработки при помощи HTTP-запроса POST.

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
