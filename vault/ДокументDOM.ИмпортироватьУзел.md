ДокументDOM (DOMDocument)

ИмпортироватьУзел (ImportNode)

Доступен, начиная с версии 8.1.

Синтаксис:

ИмпортироватьУзел(<Узел>, <Рекурсивно>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Импортируемый узел DOM.

<Рекурсивно> (обязательный)

Тип: Булево.   
Признак "глубокого" импортирования.   
Истина - будут рекурсивно импортированы все дочерние узлы импортируемого узла. Исключения составляют узлы [Атрибут](ТипУзлаDOM.Атрибут.md) и [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md).   
Для [Атрибут](ТипУзлаDOM.Атрибут.md) дочерние узлы импортируются всегда.  
Для [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) дочерние узлы не импортируются.   
Ложь - импортируется только переданный узел.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   

Описание:

Импортирует узел из другого документа DOM в данный.  
Для каждого узла импортируются свойства ТипУзла, ИмяУзла, а также свойства, относящиеся к пространствам имен: ЛокальноеИмя, Префикс, URIПространстваИмен. Пользовательские данные не переносятся.  
Правила импорта для типов узлов:

- [Атрибут](ТипУзлаDOM.Атрибут.md) - [ЭлементВладелец](АтрибутDOM.ЭлементВладелец.md) устанавливается в Неопределено, признак [Указан](АтрибутDOM.Указан.md) устанавливается в Истина, импортируются все потомки узла;
- [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - если признак "глубокого" импортирования установлен, то все потомки фрагмента рекурсивно копируются в новый фрагмент, в противном случае просто создается пустой узел [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md);
- [Документ](ТипУзлаDOM.Документ.md) - не может быть импортирован;
- [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) - не может быть импортирован;
- [Элемент](ТипУзлаDOM.Элемент.md) - атрибуты с установленным свойством [Указан](АтрибутDOM.Указан.md) импортируются, для нового узла-элемента создаются атрибуты, имеющие значения по умолчанию;
- [Сущность](ТипУзлаDOM.Сущность.md) - узлы могут быть импортированы, даже если документ находится в режиме Только для чтения. Свойства [ПубличныйИдентификатор](СущностьDOM.ПубличныйИдентификатор.md), [СистемныйИдентификатор](СущностьDOM.СистемныйИдентификатор.md) и [ИмяНотации](СущностьDOM.ИмяНотации.md) копируются;
- [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) - импортируется только данный узел. Если в данном документе определена сущность с таким же именем, то формируется значение сущности для импортированной ссылки;
- [Нотация](ТипУзлаDOM.Нотация.md) - узлы могут быть импортированы, даже если документ находится в режиме Только для чтения. Свойства [ПубличныйИдентификатор](НотацияDOM.ПубличныйИдентификатор.md), [СистемныйИдентификатор](НотацияDOM.СистемныйИдентификатор.md) копируются;
- [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md) - копируются свойства [Цель](ИнструкцияОбработкиDOM.Цель.md) и [Данные](ИнструкцияОбработкиDOM.Данные.md);
- [Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md), [Комментарий](ТипУзлаDOM.Комментарий.md) - копируются свойства [ТекстDOM.Данные](ТекстDOM.Данные.md), [КомментарийDOM.Данные](КомментарийDOM.Данные.md), [СекцияCDATADOM.Данные](СекцияCDATADOM.Данные.md) и [ТекстDOM.Размер](ТекстDOM.Размер.md), [КомментарийDOM.Размер](КомментарийDOM.Размер.md), [СекцияCDATADOM.Размер](СекцияCDATADOM.Размер.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- импортируемые имена содержат символы, недопустимые стандартом XML данного документа.

Использование в версии:

Доступен, начиная с версии 8.1.
