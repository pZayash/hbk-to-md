# КоллекцияПолейСводнойТаблицы.Вставить

КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Поле>, <Позиция>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, [ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md).   
Вставляемое поле.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую будет вставлено поле или коллекция.

Возвращаемое значение:

Тип: [ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md).   

Описание:

Вставляет поле или другую коллекцию полей в указанную позицию коллекции полей сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
