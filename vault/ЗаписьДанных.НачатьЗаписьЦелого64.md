ЗаписьДанных (DataWriter)

НачатьЗаписьЦелого64 (BeginWriteInt64)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗаписьЦелого64(<ОписаниеОповещения>, <Число>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Число> (обязательный)

Тип: Число.   
Число, которое будет записано в поток.  
Значение числа должно находиться в диапазоне от 0 до 2^64-1.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов.md).   
Порядок байтов, который будет использован для кодировки числа перед записью в поток. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных.md).  
Значение по умолчанию: Неопределено.

Описание:

Начинает запись целого 16-битного числа в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьЦелое64](ЗаписьДанных.ЗаписатьЦелое64.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьЦелое64Асинх](ЗаписьДанных.ЗаписатьЦелое64Асинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
