Индикатор (ProgressBar)

МаксимальноеЗначение (MaxValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальное значение диапазона изменения. Если значение больше или равно максимальному, то оно будет отображаться "полным" индикатором (100%).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
