ПараметрыДиалогаПолученияФайлов (GetFilesDialogParameters)

По заголовку, выбору каталога и каталогу

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ПараметрыДиалогаПолученияФайлов(<Заголовок>, <ВыборКаталога>, <Каталог>)

Параметры:

<Заголовок> (необязательный)

Тип: Строка, Неопределено.   
Заголовок диалога получения файлов.

<ВыборКаталога> (необязательный)

Тип: Булево, Неопределено.   
Определяет, откроется ли диалог для выбора каталога.

<Каталог> (необязательный)

Тип: Строка, Неопределено.   
Содержит значение текущего каталога при открытии диалога.

Использование в версии:

Доступен, начиная с версии 8.3.15.
