Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

ОбработкаНавигационнойСсылки (URLProcessing)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ОбработкаНавигационнойСсылки(<НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: [Строка](lang__def_String.md).   
Значение гиперссылки форматированной строки.  
Параметр передается по ссылке.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при нажатии навигационной ссылки в форматированной строке ячейки или на рисунке табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
