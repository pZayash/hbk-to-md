ИнтернетТекстПочтовогоСообщения (InternetMailText)

Данные (Data)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Содержит текстовые данные сообщения. Данные были только раскодированы, но работы над кодировкой текстов не производилось.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
