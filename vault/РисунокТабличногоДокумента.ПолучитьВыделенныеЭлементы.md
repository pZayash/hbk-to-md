РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ПолучитьВыделенныеЭлементы (GetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьВыделенныеЭлементы()

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Элементами массива являются значения следующих типов: [ТочкаДиаграммы](ТочкаДиаграммы.md), [СерияДиаграммы](СерияДиаграммы.md), [ЗначениеДиаграммы](ЗначениеДиаграммы.md).

Описание:

Возвращает массив выделенных элементов для рисунка типа [Диаграмма](ТипРисункаТабличногоДокумента.Диаграмма.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
