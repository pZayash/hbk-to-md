МенеджерСистемыАналитики (AnalyticsSystemManager)

ПолучитьАдресСервераСистемыАналитики (GetAnalyticsSystemServerAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьАдресСервераСистемыАналитики()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает адрес соединения с системой аналитики, настроенного на данную область данных (если в информационной базе не определены разделители – на всю базу).

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
