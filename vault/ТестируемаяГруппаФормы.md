Описание:

Представляет абстракцию элемента управления [ГруппаФормы](ГруппаФормы.md). Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством [Вид](ТестируемаяГруппаФормы.Вид.md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
