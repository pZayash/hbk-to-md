ИспользованиеАутентификацииИнтернетПочтыПоТокену (UseInternetMailTokenAuthentication)

Использовать (Use)

Доступен, начиная с версии 8.3.13.

Описание:

При подключении принудительно использовать протокол OAuth2.   
Исключение генерируется в следующих случаях:

- Если сервер не поддерживает данную возможность,
- Если не указано значение свойства [ТокенДоступа](ИнтернетПочтовыйПрофиль.ТокенДоступа_(InternetMailProfile.AccessToken).md).

Использование в версии:

Доступен, начиная с версии 8.3.13.
