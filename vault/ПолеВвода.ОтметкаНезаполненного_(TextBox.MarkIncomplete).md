ПолеВвода (TextBox)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство указывает, нужно ли сейчас в поле ввода отображать особую отметку или нет. Если свойство имеет значение [Истина](lang__def_BooleanTrue.md), то независимо от хранимых в поле ввода данных будет отображаться особая отметка, если [Ложь](lang__def_BooleanFalse.md), то независимо от хранимых в поле ввода данных особая отметка отображаться не будет.  
При начале редактирования свойство сбрасывается, даже если [АвтоОтметкаНезаполненного](ПолеВвода.АвтоОтметкаНезаполненного_(TextBox.AutoMarkIncomplete).md) = [Ложь](lang__def_BooleanFalse.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
