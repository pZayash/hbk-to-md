Невидимый (Hidden)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Невидимый](ЭлементФайлаАрхива.Невидимый_(ArchiveFileEntry.Hidden).md)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, установлен ли у файла флаг "Невидимый".  
[Истина](lang__def_BooleanTrue.md) - установлен.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
