ЭлементГрафическойСхемыДействие (GraphicalSchemaItemActivity)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - отображает элемент графической схемы как гиперссылку. При выборе элемента с установленным свойством будет порождаться событие [Выбор](ПолеГрафическойСхемы.Выбор_(GraphicalSchemaField.Selection).md).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
