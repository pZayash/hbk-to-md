Глобальный контекст (Global context)

ЗагрузитьВнешнююКомпоненту (LoadAddIn)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗагрузитьВнешнююКомпоненту(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла внешнего компонента.  
Файл внешнего компонента не обязательно должен находиться в каталоге исполняемых файлов программы.

Описание:

Загружает внешний компонент и подключает ее объекты к 1С:Предприятию. Объекты этого компонента должны быть построены по технологии внешних компонентов COM.

Доступность:

Толстый клиент.

Примечание:

Внешние компоненты совместимы с компонентами 1С:Предприятия 7.7.

Пример:

|  |
| --- |
| Попытка     ЗагрузитьВнешнююКомпоненту("scanopos.dll");     Сообщить("Компонента scanopos загружена"); Исключение     Сообщить("Компонента scanopos не найдена!", СтатусСообщения.Внимание); КонецПопытки |
