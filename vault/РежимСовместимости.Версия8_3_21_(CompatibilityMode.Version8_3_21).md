РежимСовместимости (CompatibilityMode)

Версия8\_3\_21 (Version8\_3\_21)

Доступен, начиная с версии 8.3.22.

Описание:

Усовершенствования версии 8.3.22 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.21.

Использование в версии:

Доступен, начиная с версии 8.3.22.
