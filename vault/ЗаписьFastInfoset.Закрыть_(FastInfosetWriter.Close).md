ЗаписьFastInfoset (FastInfosetWriter)

Закрыть (Close)

Доступен, начиная с версии 8.1.

Синтаксис:

Закрыть()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   
Если запись производилась в двоичные данные, значение двоичных данных будет возвращено в качестве результата.  
[Неопределено](lang__def_Undefined.md) - если инициализация объекта выполнялась методом [ОткрытьФайл](ЗаписьFastInfoset.ОткрытьФайл_(FastInfosetWriter.OpenFile).md).

Описание:

Завершает запись данных Fast Infoset в файл или двоичные данные.   
Если запись производилась в файл, файл будет закрыт.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
