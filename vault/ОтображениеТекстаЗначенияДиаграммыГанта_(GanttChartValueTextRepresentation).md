Описание:

Содержит варианты отображения текста значения диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChartValueTextRepresentation.

[ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md), свойство [ОтображениеТекстаЗначения](ДиаграммаГанта.ОтображениеТекстаЗначения_(GanttChart.ValueTextRepresentation).md)
