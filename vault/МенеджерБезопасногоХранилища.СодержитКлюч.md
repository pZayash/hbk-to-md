МенеджерБезопасногоХранилища (SecureStorageManager)

СодержитКлюч (ContainsKey)

Доступен, начиная с версии 8.3.15.

Синтаксис:

СодержитКлюч(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: Строка.   
Ключ, занятость которого требуется проверить.

Возвращаемое значение:

Тип: Булево.   

Описание:

Проверяет занятость переданного ключа. Вызов метода никогда не требует пользовательской проверки.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
