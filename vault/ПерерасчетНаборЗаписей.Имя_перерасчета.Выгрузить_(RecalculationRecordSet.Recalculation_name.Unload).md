ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.1.

Синтаксис:

Выгрузить(<Строки>, <Колонки>)

Параметры:

<Строки> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив строк для выгрузки. Если не указан, выгружаются все строки набора записей.

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Список колонок для выгрузки в формате: "Колонка1, Колонка2...". Если список не задан, то будут выгружены все колонки.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   

Описание:

Создает таблицу значений и копирует в нее набор записей перерасчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Рег = РегистрыРасчета.ДополнительныеНачисленияСотрудниковОрганизации; Перерасчеты = Рег.Перерасчеты.ПерерасчетДополнительныхНачислений; НаборЗаписей = Перерасчеты.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.ОбъектПерерасчета.Значение = Ссылка; Данные = НаборЗаписей.Выгрузить(); |

Использование в версии:

Доступен, начиная с версии 8.1.
