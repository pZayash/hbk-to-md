УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

ТребоватьДляУправления (RequireForControl)

Доступен, начиная с версии 8.3.9.

Описание:

Для передачи управляющих команд требуется использование защищенного (TLS) соединения. Данные при этом могут передаваться как по открытому, так и по защищенному каналу, в зависимости от настроек сервера.

Использование в версии:

Доступен, начиная с версии 8.3.9.
