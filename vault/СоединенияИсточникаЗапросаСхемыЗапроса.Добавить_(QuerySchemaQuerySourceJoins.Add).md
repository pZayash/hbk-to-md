СоединенияИсточникаЗапросаСхемыЗапроса (QuerySchemaQuerySourceJoins)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Источник>, <Условие>)

Параметры:

<Источник> (обязательный)

Тип: [ИсточникСхемыЗапроса](ИсточникСхемыЗапроса_(QuerySchemaSource).md), [Строка](lang__def_String.md).   
Источник или его псевдоним, с которым требуется создать связь. Если передан псевдоним и по нему найти источник не удалось, то генерируется исключение.

<Условие> (необязательный)

Тип: [Строка](lang__def_String.md).   
Условие связи двух источников. Если параметр не задан, условие связи будет создано автоматически.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В случае успеха возвращает значение [Истина](lang__def_BooleanTrue.md), иначе [Ложь](lang__def_BooleanFalse.md). Если условие связи не допустимо, вызывается исключение, содержащее сообщение об ошибке.

Описание:

Добавляет новое соединение с источником.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
