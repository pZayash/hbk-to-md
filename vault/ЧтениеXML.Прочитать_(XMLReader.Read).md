ЧтениеXML (XMLReader)

Прочитать (Read)

Доступен, начиная с версии 8.1.

Синтаксис:

Прочитать()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - очередной узел прочитан; [Ложь](lang__def_BooleanFalse.md) - текст XML завершился.

Описание:

Считывает очередной узел XML. При этом свойствам [ЧтениеXML.ТипУзла](ЧтениеXML.ТипУзла_(XMLReader.NodeType).md), [ЧтениеXML.Имя](ЧтениеXML.Имя_(XMLReader.Name).md), [ЧтениеXML.ЛокальноеИмя](ЧтениеXML.ЛокальноеИмя_(XMLReader.LocalName).md), [ЧтениеXML.Префикс](ЧтениеXML.Префикс_(XMLReader.Prefix).md), [ЧтениеXML.URIПространстваИмен](ЧтениеXML.URIПространстваИмен_(XMLReader.NamespaceURI).md) и [ЧтениеXML.Значение](ЧтениеXML.Значение_(XMLReader.Value).md) присваиваются значения, соответствующие прочитанным данным.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Чтение.ОткрытьФайл("c:/docs/data.xml"); Пока Чтение.Прочитать() Цикл     // Обработка узла XML  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.1.
