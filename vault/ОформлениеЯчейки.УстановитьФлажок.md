ОформлениеЯчейки (CellAppearance)

УстановитьФлажок (SetCheck)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьФлажок(<Флажок>)

Параметры:

<Флажок> (обязательный)

Тип: Булево, Число.   
Значение флажка.

Описание:

Устанавливает свойство [Флажок](ОформлениеЯчейки.Флажок.md) объекта. Устанавливает свойство [ОтображатьФлажок](ОформлениеЯчейки.ОтображатьФлажок.md) в значение Истина.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
