ЗаписьДанных (DataWriter)

ЗаписатьБайт (WriteByte)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьБайт(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Целое число, которое будет записано в целевой поток. Значение числа должно находиться в диапазоне от 0 до 255.

Описание:

Записывает один байт в целевой поток.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
