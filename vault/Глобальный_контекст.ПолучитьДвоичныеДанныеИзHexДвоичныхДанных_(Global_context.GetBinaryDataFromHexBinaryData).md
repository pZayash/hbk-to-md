Глобальный контекст (Global context)

ПолучитьДвоичныеДанныеИзHexДвоичныхДанных (GetBinaryDataFromHexBinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДвоичныеДанныеИзHexДвоичныхДанных(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Двоичные данные в формате Base 16 (Hex).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Преобразует двоичные данные из формата Base 16 (Hex) в [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
