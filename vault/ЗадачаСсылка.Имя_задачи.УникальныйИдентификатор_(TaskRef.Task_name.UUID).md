ЗадачаСсылка.<Имя задачи> (TaskRef.<Task name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку_(TaskManager.Task_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md), метод [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку_(TaskManager.Task_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
