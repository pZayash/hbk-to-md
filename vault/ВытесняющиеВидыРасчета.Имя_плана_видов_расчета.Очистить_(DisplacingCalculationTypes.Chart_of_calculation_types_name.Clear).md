# ВытесняющиеВидыРасчета.<Имя плана видов расчета>.Очистить

ВытесняющиеВидыРасчета.<Имя плана видов расчета> (DisplacingCalculationTypes.<Chart of calculation types name>)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все строки табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данный метод следует использовать осмотрительно в том случае, если в табличной части есть предопределенные строки (т.е. имеются строки, для которых свойство [Предопределенный](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета.Предопределенный_(DisplacingCalculationTypesRow.Chart_of_calculation_types_name.Predefined).md) имеет значение [Истина](lang__def_BooleanTrue.md)). Если удалить все, в том числе, предопределенные строки табличной части, то не будет возможности записать такой вид расчета.  
Использование метода допустимо только в том случае, если табличная часть получена из свойства объекта. Если табличная часть получена из свойства ссылки (или выборки), то использование этого метода будет вызывать ошибку выполнения.
