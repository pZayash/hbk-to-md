БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.0.

Синтаксис:

Заблокирован()

Возвращаемое значение:

Тип: Булево.   
Истина - бизнес-процесс заблокирован; Ложь - в противном случае.

Описание:

Определяет, заблокирован ли бизнес-процесс данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Следует учитывать, что этот метод используется для проверки блокировки объекта базы данных конкретным объектом встроенного языка. Он не может быть использован, чтобы проверить, заблокирован ли вообще объект базы данных.

Пример:

|  |
| --- |
| Если ВыбБизнесПроцесс.Заблокирован() Тогда     ВыбБизнесПроцесс.Разблокировать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
