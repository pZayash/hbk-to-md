# КоллекцияВстроенныхТаблиц

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Табличный документ](objects__catalog63__catalog64.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[СводнаяТаблица](СводнаяТаблица.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются [СводнаяТаблица](СводнаяТаблица.md).  
Возможно обращение к коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) или имя таблицы.

Описание:

Содержит коллекцию сводных таблиц, встроенных в табличный документ.

Доступность:

Сервер, толстый клиент, внешнее соединение.
