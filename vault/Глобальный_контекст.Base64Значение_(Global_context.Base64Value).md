# Глобальный контекст.Base64Значение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Прочие процедуры и функции](Прочие_процедуры_и_функции.md)

Глобальный контекст (Global context)

Base64Значение (Base64Value)

Доступен, начиная с версии 8.0.

Синтаксис:

Base64Значение(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, закодированная по алгоритму base64.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Получает из строки закодированной по алгоритму base64 двоичные данные.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Base64Строка](Глобальный_контекст.Base64Строка_(Global_context.Base64String).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
