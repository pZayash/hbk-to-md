ИспользованиеАтрибутаXS (XSAttributeUse)

Ограничение (Constraint)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОграничениеЗначенияXS](ОграничениеЗначенияXS_(XSConstraint).md).   
Представляет вид ограничения значения объявления атрибута (http://www.w3.org/TR/xmlschema-1/#a-value\_constraint).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
