ЛинияТрендаДиаграммы (ChartTrendline)

ОтображатьВЛегенде (ShowInLegend)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в [Истина](lang__def_BooleanTrue.md), линия тренда будет отображаться в легенде диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
