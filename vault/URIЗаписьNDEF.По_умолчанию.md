URIЗаписьNDEF (URIRecordNDEF)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый URIЗаписьNDEF(<URI>, <ИдентификаторЗаписи>)

Параметры:

<URI> (обязательный)

Тип: Строка.   
Непустая строка размером не более 10 Мбайт.

<ИдентификаторЗаписи> (необязательный)

Тип: Строка.   
Строка размером не более 255 байт.  
Обычно не указывается для экономии места на метке.

Описание:

Если в параметрах обнаружена ошибка, будет сгенерировано исключение категории [ОшибкаВоВремяВыполненияВстроенногоЯзыка](КатегорияОшибки.ОшибкаВоВремяВыполненияВстроенногоЯзыка.md) с описанием ошибки.

Использование в версии:

Доступен, начиная с версии 8.3.23.
