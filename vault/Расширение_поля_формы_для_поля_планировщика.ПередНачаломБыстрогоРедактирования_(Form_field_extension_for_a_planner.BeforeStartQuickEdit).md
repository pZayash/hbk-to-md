Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередНачаломБыстрогоРедактирования (BeforeStartQuickEdit)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередНачаломБыстрогоРедактирования(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Ложь](lang__def_BooleanFalse.md) приведет к отмене появления окна быстрого редактирования.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает перед отображением окна быстрого редактирования. Элементы, над которыми выполнялось действие, можно получить с помощью свойства [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы_(Form_field_extension_for_a_planner.SelectedItems).md) поля.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
