МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПоддерживаетсяИсторияПриобретений (PurchaseHistorySupported)

Синтаксис:

ПоддерживаетсяИсторияПриобретений(<ИдентификаторПокупки>)

Параметры:

<ИдентификаторПокупки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор покупки, для которого необходимо определить возможность получения истории.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md) в следующих случаях:

- iOS - если идентификатор покупки не передан, то всегда. С идентификатором покупки - если доступно более одной записи о данной покупке;
- Windows Phone если идентификатор покупки не передан, то всегда. С идентификатором покупки - если доступно более одной записи о данной покупке.

Описание:

Определяет, возможно ли получение истории покупок на текущей платформе. Если параметром указан идентификатор покупки, то метод будет определять возможно ли получение истории для конкретной покупки.   
Важно! Метод вернет [Истина](lang__def_BooleanTrue.md) для конкретного идентификатора покупки, если записей о приобретении данной покупки больше одной.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
