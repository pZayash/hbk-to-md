РегистрРасчетаЗапись.<Имя регистра расчета> (CalculationRegisterRecord.<Calculation register name>)

Активность (Active)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак активности записи. Влияние записи на фактический период действия, перерасчеты, работу методов [ПолучитьБазу](РегистрРасчетаЗапись.Имя_регистра_расчета.ПолучитьБазу_(CalculationRegisterRecord.Calculation_register_name.GetBase).md) и [ПолучитьДанныеГрафика](РегистрРасчетаЗапись.Имя_регистра_расчета.ПолучитьДанныеГрафика_(CalculationRegisterRecord.Calculation_register_name.GetScheduleData).md).  
[Истина](lang__def_BooleanTrue.md) - запись активна.

Доступность:

Сервер, толстый клиент, внешнее соединение.
