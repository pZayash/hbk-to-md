# ДокументОбъект.<Имя документа>.Скопировать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md).   

Описание:

Создает новый документ копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Объект = ВыбДокумент.Скопировать(); |

[ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md), событие [ПриКопировании](ДокументОбъект.Имя_документа.ПриКопировании_(DocumentObject.Document_name.OnCopy).md)

Доступен, начиная с версии 8.0.
