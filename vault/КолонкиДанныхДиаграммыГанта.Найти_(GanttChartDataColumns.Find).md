КолонкиДанныхДиаграммыГанта (GanttChartDataColumns)

Найти (Find)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Найти(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки.

Возвращаемое значение:

Тип: [КолонкаДанныхДиаграммыГанта](КолонкаДанныхДиаграммыГанта_(GanttChartDataColumn).md), [Неопределено](lang__def_Undefined.md).   
Если указанная колонка отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск колонки данных диаграммы Ганта по имени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
