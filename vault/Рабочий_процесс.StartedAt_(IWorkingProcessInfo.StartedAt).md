Рабочий процесс (IWorkingProcessInfo)

StartedAt (StartedAt)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит момент запуска рабочего процесса. Если процесс не запущен, то содержит нулевую дату.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
