ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриСозданииНаСервере (OnCreateAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСозданииНаСервере(<Отказ>, <СтандартнаяОбработка>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от создания формы. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то форма создана не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает на сервере при создании формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
