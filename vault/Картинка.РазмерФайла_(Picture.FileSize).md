# Картинка.РазмерФайла

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Картинка](Картинка.md) › [Картинка](Картинка_(Picture).md) › [Методы](Картинка__Методы.md)

Картинка (Picture)

РазмерФайла (FileSize)

Доступен, начиная с версии 8.3.14.

Синтаксис:

РазмерФайла()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Если картинка создана с вариантами, то возвращает информацию по текущему варианту.  
Если картинка является частью коллекции картинок, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает размер изображения на диске (в байтах).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
