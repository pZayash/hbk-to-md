РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRow)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Индекс(<Кнопка>)

Параметры:

<Кнопка> (обязательный)

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   
Кнопка панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс указанной кнопки в ряду.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
