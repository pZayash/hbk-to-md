КолонкаТабличногоПоля (TableBoxColumn)

ОтображатьИерархию (ShowHierarchy)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак вывода линий иерархии и пиктограмм раскрытия узлов дерева.  
[Истина](lang__def_BooleanTrue.md) - отображается.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если табличное поле отображается в виде дерева.

См. также:

[Расширение табличного поля списка справочника](Расширение_табличного_поля_списка_справочника_(Catalog_list_table_box_extension).md), свойство [Дерево](Расширение_табличного_поля_списка_справочника.Дерево_(Catalog_list_table_box_extension.Tree).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
