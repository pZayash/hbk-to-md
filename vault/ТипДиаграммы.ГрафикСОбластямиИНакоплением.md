ТипДиаграммы (ChartType)

ГрафикСОбластямиИНакоплением (StackedArea)

Доступен, начиная с версии 8.0.

Описание:

Линейный график с закрашиванием области и накоплением. Значения в каждой серии суммируются со значениями в предыдущих, поэтому области разных серий не перекрывают друг друга. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График.md).

Использование в версии:

Доступен, начиная с версии 8.0.
