ДиаграммаГанта (GanttChart)

УстановитьТочку (SetPoint)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТочку(<Значение>, <ЗначениеРодителя>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение.

<ЗначениеРодителя> (необязательный)

Тип: Произвольный.   
Значение родительской точки.

Возвращаемое значение:

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта.md).   

Описание:

Устанавливает точку диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если точка с ключом <Значение> существует, она возвращается вне зависимости от параметра <ЗначениеРодителя>. Если точка с ключом <Значение> не существует, но существует точка с ключом <ЗначениеРодителя>, то создаётся новая точка с ключом <Значение>, подчинённая точке с ключом <ЗначениеРодителя>. Иначе (т.е. не существуют точки с ключами <Значение> и <ЗначениеРодителя>), создаётся новая точка с ключом <Значение> на верхнем уровне (т.е. точка с ключом <ЗначениеРодителя> автоматически не создаётся никогда).

Использование в версии:

Доступен, начиная с версии 8.0.
