СтандартныйПериод (StandardPeriod)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СтандартныйПериод()

Описание:

Создает период с вариантом периода [ПроизвольныйПериод](ВариантСтандартногоПериода.ПроизвольныйПериод_(StandardPeriodVariant.Custom).md) и пустыми датами начала и окончания.

Использование в версии:

Доступен, начиная с версии 8.2.
