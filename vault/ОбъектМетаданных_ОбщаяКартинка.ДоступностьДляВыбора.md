ОбъектМетаданных: ОбщаяКартинка (MetadataObject: CommonPicture)

ДоступностьДляВыбора (AvailabilityForChoice)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает доступность картинки для выбора в форме выбора картинки, открываемой для редактирования картинки в ячейке, рисунке или колонтитуле табличного документа в режиме 1С:Предприятие.   
По умолчанию содержит Ложь.   
В режиме совместимости [Версия8\_3\_15](РежимСовместимости.Версия8_3_15.md) форма выбора картинки это свойство игнорирует.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
