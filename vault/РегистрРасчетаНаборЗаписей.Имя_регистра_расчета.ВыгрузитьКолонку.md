РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.   
Имя поля записей набора, из которого значения будут выгружены в массив.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Создает массив и копирует в него значения заданного поля записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения записей выгружаются в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.0.
