Глобальный контекст (Global context)

ПоказатьНаКарте (ShowOnMap)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьНаКарте(<Местоположение>)

Параметры:

<Местоположение> (обязательный)

Тип: [ГеографическиеКоординаты](ГеографическиеКоординаты_(GeographicCoordinates).md), [СписокЗначений](СписокЗначений_(ValueList).md).   
Местоположение, которое нужно показать.  
Для отображения нескольких точек используется [СписокЗначений](СписокЗначений_(ValueList).md), в котором значение элемента списка - это географическая координата, а представление - текст, отображаемый в точке (на булавке).

Описание:

Отображает на карте переданные координаты.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяОтображениеКарты](Глобальный_контекст.ПоддерживаетсяОтображениеКарты_(Global_context.MapRepresentationSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПоддерживаетсяОтображениеКарты](Глобальный_контекст.ПоддерживаетсяОтображениеКарты_(Global_context.MapRepresentationSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.
