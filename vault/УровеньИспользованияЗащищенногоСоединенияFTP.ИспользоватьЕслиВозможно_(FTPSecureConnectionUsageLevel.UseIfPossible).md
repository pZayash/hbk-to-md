УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

ИспользоватьЕслиВозможно (UseIfPossible)

Доступен, начиная с версии 8.3.9.

Описание:

Использовать защищенное SSL-соединение, если возможно. Фактический уровень защиты определяется сервером.

Использование в версии:

Доступен, начиная с версии 8.3.9.
