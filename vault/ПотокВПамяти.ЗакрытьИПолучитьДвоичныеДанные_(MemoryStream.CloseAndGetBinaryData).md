# ПотокВПамяти.ЗакрытьИПолучитьДвоичныеДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с двоичными данными](Работа_с_двоичными_данными-2.md) › [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md) › [Методы](ПотокВПамяти__Методы.md)

ПотокВПамяти (MemoryStream)

ЗакрытьИПолучитьДвоичныеДанные (CloseAndGetBinaryData)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗакрытьИПолучитьДвоичныеДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Возвращает экземпляр объекта [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), содержащего данные, записанные в поток.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если поток создан на основе объекта[БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md) и буфер позволяет изменения ([ТолькоЧтение](БуферДвоичныхДанных.ТолькоЧтение_(BinaryDataBuffer.ReadOnly).md)установлено в [Ложь](lang__def_BooleanFalse.md)), то выполняется копирование данных.  
В любом другом случае копирование данных не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.9.
