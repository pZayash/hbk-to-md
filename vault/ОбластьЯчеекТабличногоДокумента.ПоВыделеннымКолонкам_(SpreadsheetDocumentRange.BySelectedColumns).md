ПоВыделеннымКолонкам (BySelectedColumns)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак горизонтального выравнивания по выделенным колонкам.  
[Истина](lang__def_BooleanTrue.md) - горизонтальное выравнивание.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
