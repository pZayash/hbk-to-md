# Расширение формы клиентского приложения для набора записей.Записать

Расширение формы клиентского приложения для набора записей (Managed form extension for record set)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Состав параметров произвольный.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - успешно записаны; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает набор записей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
