РежимЗамещения (ReplacementMode)

Удаление (Delete)

Доступен, начиная с версии 8.3.25.

Описание:

В результате записи из регистра будут удалены записи, совпадающие по значению ключевых полей с записями в наборе записей.

Использование в версии:

Доступен, начиная с версии 8.3.25.
