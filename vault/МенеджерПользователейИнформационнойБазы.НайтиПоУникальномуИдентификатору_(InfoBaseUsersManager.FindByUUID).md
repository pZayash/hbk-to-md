# МенеджерПользователейИнформационнойБазы.НайтиПоУникальномуИдентификатору

МенеджерПользователейИнформационнойБазы (InfoBaseUsersManager)

НайтиПоУникальномуИдентификатору (FindByUUID)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоУникальномуИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор пользователя.

Возвращаемое значение:

Тип: [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md), [Неопределено](lang__def_Undefined.md).   
Если значение не найдено, то для пользователя с административными правами возвращается значение [Неопределено](lang__def_Undefined.md), для других пользователей вызывается исключение.

Описание:

Осуществляет поиск информации о пользователе с указанным идентификатором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В случае, если пользователь имеет административные права, то допускается поиск любого пользователя. Если пользователь не имеет административных прав, то по уникальному идентификатору допускается поиск только того пользователя, под которым данный пользователь аутентифицировался.

Использование в версии:

Доступен, начиная с версии 8.0.
