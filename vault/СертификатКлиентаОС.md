Описание:

Объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы:

- В ОС Windows работает полностью аналогично объекту [СертификатКлиентаWindows](СертификатКлиентаWindows.md).
- В ОС Linux работает полностью аналогично объекту [СертификатКлиентаLinux](СертификатКлиентаLinux.md).
- В ОС macOS работает полностью аналогично объекту [СертификатКлиентаMacOS](СертификатКлиентаMacOS.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
