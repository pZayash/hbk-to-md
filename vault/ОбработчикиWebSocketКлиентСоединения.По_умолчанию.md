ОбработчикиWebSocketКлиентСоединения (WebSocketClientConnectionHandlers)

По умолчанию

Доступен, начиная с версии 8.3.27.

Синтаксис:

Новый ОбработчикиWebSocketКлиентСоединения(<ОбработчикОткрытияСоединения>, <ОбработчикПолученияСообщения>, <ОбработчикОшибки>, <ОбработчикЗакрытияСоединения>, <Модуль>)

Параметры:

<ОбработчикОткрытияСоединения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры обработчика события открытия соединения.

<ОбработчикПолученияСообщения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры-обработчика события получения сообщения.

<ОбработчикОшибки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры обработчика события возникновения ошибки.

<ОбработчикЗакрытияСоединения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры обработчика события закрытия соединения.

<Модуль> (необязательный)

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md), [ОбщийМодуль](ОбщийМодуль_(CommonModule).md), [Строка](lang__def_String.md).   
Модуль процедур-обработчиков событий.

Описание:

Создает объект [ОбработчикиWebSocketКлиентСоединения](ОбработчикиWebSocketКлиентСоединения_(WebSocketClientConnectionHandlers).md) и связывает его с вызовом указанных процедур модуля.

Использование в версии:

Доступен, начиная с версии 8.3.27.
