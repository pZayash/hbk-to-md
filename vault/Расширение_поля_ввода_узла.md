Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода.md), предназначенного для ввода ссылки на узел.  
Расширение предоставляет возможность автоматического поиска и подбора узла по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
