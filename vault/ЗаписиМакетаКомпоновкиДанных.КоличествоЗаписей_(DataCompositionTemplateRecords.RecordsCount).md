ЗаписиМакетаКомпоновкиДанных (DataCompositionTemplateRecords)

КоличествоЗаписей (RecordsCount)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Указывает количество записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если значение равно -1, выводятся все записи.

Использование в версии:

Доступен, начиная с версии 8.1.
