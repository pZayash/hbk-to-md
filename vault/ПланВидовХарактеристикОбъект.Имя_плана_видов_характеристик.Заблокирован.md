ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.0.

Синтаксис:

Заблокирован()

Возвращаемое значение:

Тип: Булево.   
Истина - объект заблокирован; Ложь - в противном случае.

Описание:

Определяет, заблокирован ли объект.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Следует учитывать, что этот метод используется для проверки блокировки объекта базы данных конкретным объектом встроенного языка. Он не может быть использован для проверки, заблокирован ли вообще объект базы данных, например, другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.0.
