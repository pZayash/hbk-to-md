Глобальный контекст (Global context)

ПолучитьПериодЖурналаРегистрации (GetEventLogPeriod)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьПериодЖурналаРегистрации()

Возвращаемое значение:

Тип: [Диапазон](Диапазон.md).   
[Начало](Диапазон.Начало.md) - дата самого старого события журнала регистрации,   
[Конец](Диапазон.Конец.md) - дата самого нового события журнала регистрации.   
Если и Начало, и Конец имеют значение Неопределено, то журнал не содержит ни одного события.

Описание:

Возвращает начало и конец периода, за который журнал регистрации хранит события.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[Глобальный контекст](Глобальный_контекст.md), метод [СократитьЖурналРегистрации](Глобальный_контекст.СократитьЖурналРегистрации.md)

Доступен, начиная с версии 8.3.20.
