Доступен, начиная с версии 8.2.

Значения

[Авто (Auto)](ПоискВТаблицеПриВводе.Авто.md)  
[Использовать (Use)](ПоискВТаблицеПриВводе.Использовать.md)  
[НеИспользовать (DontUse)](ПоискВТаблицеПриВводе.НеИспользовать.md)  

Описание:

Содержит варианты режимов выполнения поиска при вводе символов в таблице, связанной с данными типа [ДинамическийСписок](ДинамическийСписок.md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы.md), свойство [ПоискПриВводе](ТаблицаФормы.ПоискПриВводе.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
