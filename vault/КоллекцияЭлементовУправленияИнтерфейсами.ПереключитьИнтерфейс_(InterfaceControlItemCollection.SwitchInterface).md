# КоллекцияЭлементовУправленияИнтерфейсами.ПереключитьИнтерфейс

КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

ПереключитьИнтерфейс (SwitchInterface)

Доступен, начиная с версии 8.0.

Синтаксис:

ПереключитьИнтерфейс(<ИменаИнтерфейсов>)

Параметры:

<ИменаИнтерфейсов> (необязательный)

Тип: [Строка](lang__def_String.md).   
Перечисленные через запятую имена интерфейсов, которые следует сделать видимыми.

Описание:

Способ группового управления видимостью командных интерфейсов для режима запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md). При вызове данного метода сначала становятся невидимыми все интерфейсы, кроме имеющих значение [Ложь](lang__def_BooleanFalse.md) у свойства [Переключаемый](ЭлементУправленияИнтерфейсом.Переключаемый_(InterfaceControlItem.Switchable).md), а затем становятся видимыми перечисленные в параметре интерфейсы.  
При работе в режиме [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) вызов данного метода игнорируется.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ГлавныйИнтерфейс.ПереключитьИнтерфейс("Основной,Маркетинг"); |

Использование в версии:

Доступен, начиная с версии 8.0.
