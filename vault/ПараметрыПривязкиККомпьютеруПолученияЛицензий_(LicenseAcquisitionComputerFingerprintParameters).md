Описание:

Данные о привязке программной лицензии на платформу к оборудованию компьютера. Может быть получен только с помощью метода [ПолучитьПараметрыПривязкиККомпьютеру](МенеджерПолученияЛицензий.ПолучитьПараметрыПривязкиККомпьютеру_(LicenseAcquisitionManager.GetComputerFingerprintParameters).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.   
Сериализуется.
