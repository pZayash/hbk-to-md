ГруппировкаКомпоновкиДанных (DataCompositionGroup)

УсловноеОформление (ConditionalAppearance)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных_(DataCompositionConditionalAppearance).md).   
Содержит условное оформление, применяемое в группировке.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
