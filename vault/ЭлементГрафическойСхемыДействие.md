Описание:

Элемент графической схемы "Действие". Отображает основную точку маршрута бизнес-процесса, по которой выдаются и выполняются задачи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
