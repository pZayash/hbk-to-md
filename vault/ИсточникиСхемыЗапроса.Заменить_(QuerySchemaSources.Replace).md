ИсточникиСхемыЗапроса (QuerySchemaSources)

Заменить (Replace)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Заменить(<Индекс>, <НовыйИсточник>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс заменяемого источника.

<НовыйИсточник> (обязательный)

Тип: [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса_(QuerySchemaAvailableTable).md), [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса_(QuerySchemaAvailableNestedTable).md).   
Доступная таблица или доступная вложенная таблица, полученные из доступных таблиц схемы запроса.

Описание:

Заменяет источник новым источником, полученным из доступных таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
