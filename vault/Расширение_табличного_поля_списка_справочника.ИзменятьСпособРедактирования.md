Расширение табличного поля списка справочника (Catalog list table box extension)

ИзменятьСпособРедактирования (ChangeEditingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Устанавливает доступность интерактивного способа редактирования объекта (в строке или в отдельной форме).

Доступность:

Толстый клиент.

Примечание:

Ложь - интерактивная установка способа редактирования объекта не доступна пользователю.

Использование в версии:

Доступен, начиная с версии 8.0.
