Элементы коллекции:

[ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметров.  
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Данный объект представляет собой коллекцию, содержащую следующие элементы оформления:

- Расшифровка (Details) - элемент оформления; тип [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных.md);
- Формат (Format) - формат, применяемый к значению группировки; тип Строка;
- ЦветВДиаграмме (ColorInChart) - цвет, используемый для вывода группировки в диаграмме; тип [Цвет](Цвет.md);
- ВыводитьНаОсиТочекДиаграммы (PutOnPointsAxis) - указывает, что серия, полученная из данной группировки, откладывается на оси точек в пузырьковой и точечной диаграммах; тип Булево;
- ИспользоватьКакОбщуюСериюРазмераПузырькаВДиаграмме (UseAsCommonBubbleSizeSeries) - указывает, что серия, полученная из данной группировки, используется в качестве общей серии размера пузырька в пузырьковой диаграмме; тип Булево;
- ЛинияВДиаграмме (LineInChart) - тип линии серии, полученной из данной группировки; тип [Линия](Линия.md);
- МаркерВДиаграмме (MarkerInChart) - тип маркера серии, полученной из данной группировки; тип [ТипМаркераДиаграммы](ТипМаркераДиаграммы.md);
- ИндикаторВДиаграмме (IndicatorInChart) указывает, является ли индикатором серия, полученная из данной группировки; тип Булево;
- ОтображатьГрафическоеПредставлениеДанныхВДиаграмме (ShowGraphicalDataRepresentationInChart) - указывает, что серия, полученная из данной группировки, отображается на диаграмме; тип [ОтображениеВДиаграмме](ОтображениеВДиаграмме.md);
- ОтображатьГрафическоеПредставлениеДанныхВЛегендеДиаграммы (ShowGraphicalDataRepresentationInChartLegend) - указывает, что серия, полученная из данной группировки, отображается в легенде диаграммы; тип [ОтображениеВЛегендеДиаграммы](ОтображениеВЛегендеДиаграммы.md);
- ИнформационныеЛинииВДиаграмме (ReferenceLinesInChart) - содержит коллекцию информационных линий для точки диаграммы, полученной из данной группировки; тип [Соответствие](Соответствие.md);
- ИнформационныеИнтервалыВДиаграмме (ReferenceBandsInChart) - содержит коллекцию информационных интервалов для точки диаграммы, полученной из данной группировки; тип [Соответствие](Соответствие.md);
- ТипГрафическогоПредставленияВДиаграмме (GraphicalRepresentationTypeInChart) - тип серии, отличный от типа диаграммы; тип [ТипГрафическогоПредставленияСерииДиаграммы](ТипГрафическогоПредставленияСерииДиаграммы.md);
- ТипНакопленияВДиаграмме (StackTypeInChart) - тип накопления серии, отличный от типа диаграммы. Группа накопления серии создаётся автоматически, на основе элемента условного оформления. Разные элементы условного оформления имеют разные группы;тип [ТипНакопленияСерииДиаграммы](ТипНакопленияСерииДиаграммы.md);
- ИспользуемаяОсьЗначенийВДиаграмме (UsedValueAxisInChart) - настраивает использование серией дополнительной оси значений;тип [ИспользуемаяОсьЗначенийДиаграммы](ИспользуемаяОсьЗначенийДиаграммы.md);
- ГруппаНакопленияВДиаграмме (StackGroupInChart) - группа накопления серии; тип Строка;
- ФорматЗначенийВПодписяхДиаграммы (ChartValueLabelFormat) - формат подписей значений в подписях; тип Строка;
- ФорматВТаблицеДанныхДиаграммы (ChartDataTableFormat) - формат значений в таблице данных; тип Строка};

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: ChartGroupAppearance.
