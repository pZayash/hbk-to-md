КонструкторЗапроса (QueryWizard)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - текст, если конструктор был закрыт нажатием кнопки "OK".

Описание:

Асинхронная функция. Открывает немодально конструктор запроса.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[КонструкторЗапроса](КонструкторЗапроса.md), метод [Показать](КонструкторЗапроса.Показать.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
