Расширение поля формы для поля планировщика (Form field extension for a planner)

Выбор (Selection)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Выбор(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка – вызов события ПриНачалеРедактирования – производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке левой кнопки мыши на элементе. Выбранный элемент можно получить с помощью свойства поля [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы_(Form_field_extension_for_a_planner.SelectedItems).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
