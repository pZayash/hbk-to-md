# Расширение группы формы для обычной группы.СквозноеВыравнивание

Расширение группы формы для обычной группы (Form group extension for a usual group)

СквозноеВыравнивание (ThroughAlign)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [СквозноеВыравнивание](СквозноеВыравнивание_(ThroughAlign).md).   
Устанавливает сквозное выравнивание заголовков элементов группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
