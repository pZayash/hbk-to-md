Расширение группы формы для обычной группы (Form group extension for a usual group)

ПутьКДаннымЗаголовка (TitleDataPath)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит путь к реквизиту, который отображается в заголовке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Установка значения возможна, если значение еще не задано.

Использование в версии:

Доступен, начиная с версии 8.2.
