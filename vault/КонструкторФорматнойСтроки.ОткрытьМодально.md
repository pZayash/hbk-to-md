КонструкторФорматнойСтроки (FormatStringWizard)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: Булево.   
Истина - конструктор закрыли по кнопке "OK"; Ложь - в противном случае.

Описание:

Открывает модально конструктор форматной строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [Показать](КонструкторФорматнойСтроки.Показать.md).

См. также:

[КонструкторФорматнойСтроки](КонструкторФорматнойСтроки.md), метод [ОткрытьАсинх](КонструкторФорматнойСтроки.ОткрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
