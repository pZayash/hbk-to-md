ЧтениеPDF (PDFReader)

ОткрытьАсинх (OpenAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Открыть файл

Синтаксис:

ОткрытьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: Строка.   
Имя файла для чтения PDF-документа.

Вариант синтаксиса: Установить поток

Синтаксис:

ОткрытьАсинх(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения PDF-документа.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Открывает PDF-документ для чтения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [Открыть](ЧтениеPDF.Открыть.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [НачатьОткрытие](ЧтениеPDF.НачатьОткрытие.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
