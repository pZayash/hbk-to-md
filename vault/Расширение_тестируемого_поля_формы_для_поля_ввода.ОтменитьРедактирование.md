Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ОтменитьРедактирование (CancelEdit)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОтменитьРедактирование()

Описание:

Вызывает отмену изменений в поле ввода (аналогично нажатию клавиши Esc).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
