ПоискВТаблицеПриВводе (SearchInTableOnInput)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.2.

Описание:

При вводе символов в таблицу, связанную с данными типа [ДинамическийСписок](ДинамическийСписок.md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией.md) поиск в таблице не выполняется.

Использование в версии:

Доступен, начиная с версии 8.2.
