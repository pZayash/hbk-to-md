ПараметрыЧтенияXML (XMLReaderSettings)

СекцииCDATAКакТекст (CDATASectionAsText)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак интерпретации секций CDATA как текста при чтении XML.  
Истина - интерпретируется как текст. Значение по умолчанию Ложь.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
