КоллекцияДвижений (RegisterRecordsCollection)

<Имя регистра> (<Register name>)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Предоставляет доступ к набору записей соответствующего регистра. Содержит объекты вида [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>, или [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)>, или [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)>.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
