НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы (InfoBaseUserAuthenticationLockSettings)

ДлительностьБлокировки (LockDuration)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Время (в секундах), на которое блокируется пользователь после превышения максимального количества попыток аутентификации.   
Значение не может быть отрицательным или равным нулю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
