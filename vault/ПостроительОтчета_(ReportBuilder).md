Описание:

Предназначен для интеллектуального построения отчетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // ТекстЗапроса - элемент управления, в котором формируется   // запрос (например, с помощью конструктора запросов)  Построитель = Новый ПостроительОтчета; Попытка     Построитель.Текст = ЭлементыФормы.ТекстЗапроса.ПолучитьТекст();     Построитель.Выполнить();     Если РезультатВТабличныйДокумент Тогда         // Вывод результатов в табличный документ          Построитель.Вывести();     Иначе         // В сводную таблицу           СводнаяТаблица.ИсточникДанных = Построитель;          СводнаяТаблица.Обновить();     КонецЕсли; Исключение     Предупреждение(ОписаниеОшибки()); КонецПопытки; |
