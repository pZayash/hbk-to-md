ВертикальнаяПрокруткаФормы (VerticalFormScroll)

ИспользоватьБезРастягивания (UseWithoutStretch)

Доступен, начиная с версии 8.3.7.

Описание:

При изменении размера формы элементы формы не сжимаются и не растягиваются по вертикали.

Использование в версии:

Доступен, начиная с версии 8.3.7.
