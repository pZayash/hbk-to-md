ТекстовыйДокумент (TextDocument)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что текстовый документ открыт в режиме просмотра. В режиме просмотра содержимое текстового документа не доступно для редактирования.  
Истина - режим просмотра.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
