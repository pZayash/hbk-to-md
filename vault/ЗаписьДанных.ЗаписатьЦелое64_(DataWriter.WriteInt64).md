# ЗаписьДанных.ЗаписатьЦелое64

ЗаписьДанных (DataWriter)

ЗаписатьЦелое64 (WriteInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьЦелое64(<Число>, <ПорядокБайтов>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Целое число, которое будет записано в целевой поток. Значение числа должно находиться в диапазоне от 0 до 2^64-1.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа при записи. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Значение по умолчанию: Неопределено.

Описание:

Записывает целое 16-битное число в целевой поток.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
