# СписокПолнотекстовогоПоиска.ПолучатьОписание

СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолучатьОписание (GetDescription)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак получения описания для результатов поиска. Если установлено [Истина](lang__def_BooleanTrue.md), то для каждого из результатов поиска будет заполнено значение [Описание](ЭлементСпискаПолнотекстовогоПоиска.Описание_(FullTextSearchListItem.Description).md), помогающее понять контекст найденных слов. Установка [ПолучатьОписание](СписокПолнотекстовогоПоиска.ПолучатьОписание_(FullTextSearchList.GetDescription).md) в [Ложь](lang__def_BooleanFalse.md) ускоряет поиск.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
