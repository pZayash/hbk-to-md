АтрибутHTML (HTMLAttribute)

СледующийСоседний (NextSibling)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md), [Неопределено](lang__def_Undefined.md).   
Следующий соседний узел HTML или [Неопределено](lang__def_Undefined.md), если такового нет.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
