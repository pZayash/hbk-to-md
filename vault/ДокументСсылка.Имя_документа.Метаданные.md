ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Документ](ОбъектМетаданных_Документ.md).   

Описание:

Предоставляет доступ к объекту описания метаданных документа. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.Документы.РасходнаяНакладная.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Для каждого Реквизит Из ДокументСсылка.Метаданные().Реквизиты Цикл     Сообщить(Реквизит.Представление()); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
