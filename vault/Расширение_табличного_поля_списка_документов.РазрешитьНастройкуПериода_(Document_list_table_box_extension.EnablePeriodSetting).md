Расширение табличного поля списка документов (Document list table box extension)

РазрешитьНастройкуПериода (EnablePeriodSetting)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство имеет значение [Истина](lang__def_BooleanTrue.md), то команда "Установить интервал дат" доступна даже в том случае, если запрещен интерактивный отбор по дате.

Доступность:

Толстый клиент.

Примечание:

Значение по умолчанию [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
