# СлойГеографическойСхемы.НайтиОбъект

СлойГеографическойСхемы (GeographicalSchemaLayer)

НайтиОбъект (FindObject)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиОбъект(<Серия>, <Значение>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы_(GeographicalSchemaLayerDataSeries).md).   
Серия, по которой ищется объект.

<Значение> (обязательный)

Тип: Произвольный.   
Значение, по которому ищется объект.

Возвращаемое значение:

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - элемент не найден.

Описание:

Осуществляет поиск объекта в слое.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
