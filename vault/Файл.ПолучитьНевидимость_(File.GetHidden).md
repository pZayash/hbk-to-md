Файл (File)

ПолучитьНевидимость (GetHidden)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьНевидимость()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - у файла установлен атрибут "Скрытый"; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Получает значение атрибута файла "Скрытый" ("Hidden").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применим только к файлам или каталогам, которые реально существуют.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.

Пример:

|  |
| --- |
| Если ВыбФайл.ПолучитьНевидимость() Тогда     Сообщить("Файл "+СокрЛП(ВыбФайл.Имя)+" невидим"); КонецЕсли; |
