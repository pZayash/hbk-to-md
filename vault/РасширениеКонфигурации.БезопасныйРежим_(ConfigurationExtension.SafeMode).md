РасширениеКонфигурации (ConfigurationExtension)

БезопасныйРежим (SafeMode)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Булево](lang__Булево_(Boolean).md).   
Устанавливает использование безопасного режима для расширения:

- [Истина](lang__def_BooleanTrue.md) – Расширение использует профиль безопасности безопасного режима.
- [Ложь](lang__def_BooleanFalse.md) – Расширение использует основной профиль безопасности, при этом хеш расширения должен быть прописан в профиле.
- Строка с именем профиля – расширение будет исполняться с указанным профилем.

Изменение свойства возможно только при наличии у текущего пользователя информационной базы права "Администрирование".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
