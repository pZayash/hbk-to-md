НастройкаПериода (PeriodSettings)

ДатаНачала (StartDate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату начала периода.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если свойство [ВариантНачала](НастройкаПериода.ВариантНачала_(PeriodSettings.BeginningVariant).md) имеет значение [КонкретнаяДата](ВариантГраницыИнтервала.КонкретнаяДата_(IntervalBoundVariant.SpecificDate).md).

Использование в версии:

Доступен, начиная с версии 8.0.
