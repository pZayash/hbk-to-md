ПараметрыЗаписиXML (XMLWriterSettings)

ОтступАтрибутов (IndentAttributes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Показывает, будут ли в результирующем тексте делаться отступы для более удобного отображения атрибутов элемента XML.   
Истина - отступ устанавливается.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
