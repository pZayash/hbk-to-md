ПередПереоткрытиемСДругогоСервера (BeforeReopenFromOtherServer)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПередПереоткрытиемСДругогоСервера(<Параметры>, <Отказ>)

Параметры:

<Параметры>

Тип: [Структура](Структура_(Structure).md).   
Ключевые параметры исходной формы.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.

Описание:

Возникает перед тем, как форма будет переоткрыта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
