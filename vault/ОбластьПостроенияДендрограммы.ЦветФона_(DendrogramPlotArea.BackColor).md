ОбластьПостроенияДендрограммы (DendrogramPlotArea)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона области построения дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ОбластьПостроенияДендрограммы.ПрозрачныйФон_(DendrogramPlotArea.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
