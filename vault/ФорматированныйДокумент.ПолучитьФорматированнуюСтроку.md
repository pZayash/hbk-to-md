ФорматированныйДокумент (FormattedDocument)

ПолучитьФорматированнуюСтроку (GetFormattedString)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьФорматированнуюСтроку()

Возвращаемое значение:

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   

Описание:

Получает форматированную строку.  
В результирующую форматированную строку помещаются только те картинки, которые заданы URL’ом на библиотеку картинок или на внешний ресурс. Остальные картинки игнорируются (целиком).  
Форматирование, не поддерживаемое форматированной строкой:  
Выравнивание;  
Отступы;  
Межстрочный интервал.  
игнорируется.  
Символы перевода строки и начала параграфа преобразуются в перевод строки.  
Элементы маркированных и нумерованных списков преобразуются в элементы разделенные переводом строк с символом ● или номером вначале.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
