ОбъектМетаданных: СервисИнтеграции (MetadataObject: IntegrationService)

ПринадлежностьОбъекта (ObjectBelonging)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ПринадлежностьОбъекта](ПринадлежностьОбъекта_(ObjectBelonging).md).   
Устанавливает, относится ли объект к данной конфигурации или был заимствован из расширяемой конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
