ЭлементDOM (DOMElement)

УстановитьСоответствиеПространстваИмен (SetNamespaceMapping)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Основной

Синтаксис:

УстановитьСоответствиеПространстваИмен(<Префикс>, <URIПространстваИмен>, <Согласовать>)

Параметры:

<Префикс> (обязательный)

Тип: [Строка](lang__def_String.md).   
Префикс пространства имен.

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

<Согласовать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак необходимости согласования префиксов дочерних элементов и атрибутов для указываемого пространства имен. Для атрибутов согласование выполняется только в случае непустого существующего префикса.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Если для данного префикса уже установлено соответствие пространству имен, то никаких действий производиться не будет, и будет возвращено [Неопределено](lang__def_Undefined.md).  
Причины вызова исключений:

- указанный префикс содержит символы, недопустимые стандартом XML, соответствующим версии XML, указанной в свойстве [ВерсияXML](ДокументDOM.ВерсияXML_(DOMDocument.XMLVersion).md);
- данный узел в режиме Только для чтения;
- <Префикс> и <URIПространстваИмен> не соответствуют спецификации XML Namespaces.

Вариант синтаксиса: Автоматическое создание префикса

Синтаксис:

УстановитьСоответствиеПространстваИмен(<URIПространстваИмен>, <Согласовать>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

<Согласовать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак необходимости согласования префиксов дочерних элементов и атрибутов для указываемого пространства имен. Для атрибутов согласование выполняется только в случае непустого существующего префикса.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Префикс соответствия пространства имен будет сгенерирован автоматически. Если для данного пространства имен уже установлено соответствие, то никаких действий производиться не будет и будет возвращен соответствующий узел-атрибут.  
Причины вызова исключений:

- указанный URI пространства имен содержит символы, недопустимые стандартом XML, соответствующим версии XML, указанной в свойстве [ВерсияXML](ДокументDOM.ВерсияXML_(DOMDocument.XMLVersion).md).
- данный узел в режиме Только для чтения.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [Неопределено](lang__def_Undefined.md).   
Возвращает - атрибут, соответствующий созданному соответствию.  
Если для данного префикса уже установлено соответствие пространству имен, то никаких действий производиться не будет, и будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Создает и устанавливает в элемент атрибут соответствия пространства имен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
