Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

ИспользуемоеИмяФайла (UsedFileName)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, которое будет использоваться при сохранении документа при вызове пользователем команды "Сохранить".

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.22.
