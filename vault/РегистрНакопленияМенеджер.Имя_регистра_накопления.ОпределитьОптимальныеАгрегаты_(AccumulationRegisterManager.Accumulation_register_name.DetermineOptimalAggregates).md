РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

ОпределитьОптимальныеАгрегаты (DetermineOptimalAggregates)

Доступен, начиная с версии 8.2.

Синтаксис:

ОпределитьОптимальныеАгрегаты(<МаксимальныйОтносительныйРазмер>)

Параметры:

<МаксимальныйОтносительныйРазмер> (необязательный)

Тип: [Число](lang__def_Number.md).   
Максимальная сумма размеров агрегатов в % от таблицы движений.  
Если не указан или равен нулю, то не накладывается ограничений на размер агрегатов.  
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: [ИнформацияОбАгрегатах](ИнформацияОбАгрегатах_(AggregatesInformation).md).   

Описание:

Выполняет построение оптимальных агрегатов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
