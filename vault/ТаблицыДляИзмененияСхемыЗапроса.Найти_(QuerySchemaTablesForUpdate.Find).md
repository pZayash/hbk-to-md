ТаблицыДляИзмененияСхемыЗапроса (QuerySchemaTablesForUpdate)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<ИмяТаблицы>)

Параметры:

<ИмяТаблицы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы для поиска.

Возвращаемое значение:

Тип: [ТаблицаДляИзмененияСхемыЗапроса](ТаблицаДляИзмененияСхемыЗапроса_(QuerySchemaTableForUpdate).md).   

Описание:

Выполняет поиск по свойству [ИмяТаблицы](ТаблицаДляИзмененияСхемыЗапроса.ИмяТаблицы_(QuerySchemaTableForUpdate.TableName).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
