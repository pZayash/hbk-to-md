КонстантаКлючЗначения.<Имя константы> (ConstantValueKey.<Constant name>)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый КонстантаКлючЗначения.<Имя константы>(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (необязательный)

Тип: [Структура](Структура_(Structure).md), [Неопределено](lang__def_Undefined.md).   
Имя ключа - имя общего реквизита, в качестве значения поля – необходимое значение этого реквизита.   
Если параметр не задан или равен [Неопределено](lang__def_Undefined.md), создается пустой ключ константы.

Описание:

Создает новый объект на основе переданных параметров.  
Для типов метаданных нужно использовать функциональную форму оператора Новый(<Тип>[, <ПараметрыКонструктора>]).

Использование в версии:

Доступен, начиная с версии 8.3.13.
