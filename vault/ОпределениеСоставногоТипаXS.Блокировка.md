ОпределениеСоставногоТипаXS (XSComplexTypeDefinition)

Блокировка (Block)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОбъединениеЗапрещенныхПодстановокXS](ОбъединениеЗапрещенныхПодстановокXS.md).   
Определяет значение, запрещающее подстановку вместо элемента данного типа элементов, имеющих тип, являющийся расширением или ограничением данного типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
