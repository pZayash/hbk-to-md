ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

АутентификацияОС (OSAuthentication)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак необходимости соединения с внешним источником данных при помощи аутентификации операционной системы.  
[Неопределено](lang__def_Undefined.md) - признак необходимости аутентификации операционной системы выбирается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - соединение необходимо выполнить при помощи аутентификации операционной системы. [Ложь](lang__def_BooleanFalse.md) - аутентификация не используется.

Использование в версии:

Доступен, начиная с версии 8.3.1.
