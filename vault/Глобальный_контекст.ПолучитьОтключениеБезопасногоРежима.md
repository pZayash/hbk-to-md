Глобальный контекст (Global context)

ПолучитьОтключениеБезопасногоРежима (GetSafeModeDisabled)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьОтключениеБезопасногоРежима()

Возвращаемое значение:

Тип: Булево.   
Истина - выключение безопасного режима для текущего метода; Ложь - в противном случае.

Описание:

Определяет выключение безопасного режима для текущего метода командой на время выполнения обработчиков событий объектов метаданных или вызовом метода [УстановитьОтключениеБезопасногоРежима](Глобальный_контекст.УстановитьОтключениеБезопасногоРежима.md) с параметром Истина непосредственно в коде метода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
