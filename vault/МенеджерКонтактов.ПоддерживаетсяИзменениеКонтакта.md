МенеджерКонтактов (ContactManager)

ПоддерживаетсяИзменениеКонтакта (ContactEditingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяИзменениеКонтакта(<Контакт>)

Параметры:

<Контакт> (обязательный)

Тип: [ЛокальныйКлючКонтакта](ЛокальныйКлючКонтакта.md).   
Проверяемый контакт.

Возвращаемое значение:

Тип: Булево.   
Для устройств под управлением Windows Phone всегда возвращает Ложь.  
Для устройств под управлением Android и iOS всегда возвращает Истина.

Описание:

Определяет возможность редактирования или удаления контакта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКонтактов](МенеджерКонтактов.md), метод [ИзменитьКонтакт](МенеджерКонтактов.ИзменитьКонтакт.md)  
[МенеджерКонтактов](МенеджерКонтактов.md), метод [УдалитьКонтакт](МенеджерКонтактов.УдалитьКонтакт.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
