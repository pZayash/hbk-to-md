РасписаниеРегламентногоЗадания (JobSchedule)

ДеньВМесяце (DayInMonth)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Номер дня в месяце, когда задание может быть запущено. Положительные значения считают относительно начала месяца, отрицательные - относительно конца (1 - первый день месяца, -1 - последний день месяца).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
