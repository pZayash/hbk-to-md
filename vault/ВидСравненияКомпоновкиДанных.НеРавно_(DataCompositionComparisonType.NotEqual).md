ВидСравненияКомпоновкиДанных (DataCompositionComparisonType)

НеРавно (NotEqual)

Доступен, начиная с версии 8.1.

Описание:

Проверка на неравенство. Для строковых значений регистр символов не учитывается.

Использование в версии:

Доступен, начиная с версии 8.1.
