ТабличныйДокумент (SpreadsheetDocument)

МасштабПечати (PrintScale)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит масштаб изображения при выводе на печать в процентах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл, если свойство [АвтоМасштаб](ТабличныйДокумент.АвтоМасштаб_(SpreadsheetDocument.FitToPage).md) имеет значение [Ложь](lang__def_BooleanFalse.md).  
Если не задано, то значение свойства будет выбираться из настроек принтера, на котором осуществляется печать.

Использование в версии:

Доступен, начиная с версии 8.0.
