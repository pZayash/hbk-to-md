Глобальный контекст (Global context)

ОбработкаВыбораШаблонаСообщенияСистемыВзаимодействия (CollaborationSystemMessageTemplateChoiceProcessing)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОбработкаВыбораШаблонаСообщенияСистемыВзаимодействия(<Обсуждение>, <СтандартнаяОбработка>)

Параметры:

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если после вызова обработчика параметр <СтандартнаяОбработка> содержит значение [Истина](lang__def_BooleanTrue.md), отображается системный диалог отправки шаблонного сообщения.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при выборе команды "Отправить шаблонное сообщение".  
Если после завершения работы процедуры обработки события <СтандартнаяОбработка> установлено в [Истина](lang__def_BooleanTrue.md), отображается системный диалог отправки шаблонного сообщения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.23.
