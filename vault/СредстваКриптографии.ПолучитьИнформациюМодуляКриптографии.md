СредстваКриптографии (CryptoTools)

ПолучитьИнформациюМодуляКриптографии (GetCryptoModuleInformation)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИнформациюМодуляКриптографии(<ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>)

Параметры:

<ИмяМодуляКриптографии> (обязательный)

Тип: Строка.   
Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы MS Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: Строка.   
Путь к файлу-библиотеке, реализующей требуемый набор функций.  
Параметр используется, если менеджер создается для операционных систем Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: Число.   
Задает тип модуля криптографии.

Возвращаемое значение:

Тип: Неопределено: [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии.md).   
Если модуль криптографии не поддерживается (не найден по имени для MS Windows, либо не экспортируются требуемые функции на Linux), то возвращает Неопределено.

Описание:

Получает информацию о модуле криптографии.  
  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
