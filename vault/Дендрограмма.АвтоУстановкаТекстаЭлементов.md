Описание:

Тип: Булево.   
Содержит признак генерации текста для новых элементов по схеме "ЭлементN". Если Ложь, то у нового элемента будет пустой текст.

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
