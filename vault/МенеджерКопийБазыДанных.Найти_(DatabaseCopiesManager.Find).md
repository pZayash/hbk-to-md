МенеджерКопийБазыДанных (DatabaseCopiesManager)

Найти (Find)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Найти(<ИмяКопии>)

Параметры:

<ИмяКопии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя копии.

Возвращаемое значение:

Тип: [МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md).   

Описание:

Осуществляет поиск менеджера копии по его имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
