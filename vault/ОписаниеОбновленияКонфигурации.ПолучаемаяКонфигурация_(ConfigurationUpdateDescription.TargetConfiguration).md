ОписаниеОбновленияКонфигурации (ConfigurationUpdateDescription)

ПолучаемаяКонфигурация (TargetConfiguration)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [ОписаниеКонфигурации](ОписаниеКонфигурации_(ConfigurationDescription).md).   
Данные конфигурации, которая должна быть получена в результате применения к одной из исходных конфигураций данного обновления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
