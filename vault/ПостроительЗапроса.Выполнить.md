ПостроительЗапроса (QueryBuilder)

Выполнить (Execute)

Доступен, начиная с версии 8.0.

Синтаксис:

Выполнить()

Описание:

Выполняет запрос и записывает результат в свойство [Результат](ПостроительЗапроса.Результат.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Выставляет внутренний флаг необходимости перевыполнения запроса при получении результата запроса.

Использование в версии:

Доступен, начиная с версии 8.0.
