# ДоступныеТаблицыСхемыЗапроса

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Работа_с_запросами.md">Работа с запросами</a> › [Схема запроса](objects__catalog213__catalog2171.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ГруппаДоступныхТаблицСхемыЗапроса](ГруппаДоступныхТаблицСхемыЗапроса.md), [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются группа доступных таблиц или доступная таблица.  
Возможно обращение к группе доступных таблиц или к доступной таблице посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ДоступныеТаблицыСхемыЗапроса.Индекс.md)  
[Количество (Count)](ДоступныеТаблицыСхемыЗапроса.Количество.md)  
[Найти (Find)](ДоступныеТаблицыСхемыЗапроса.Найти.md)  
[Получить (Get)](ДоступныеТаблицыСхемыЗапроса.Получить.md)  

Описание:

Коллекция доступных таблиц схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
