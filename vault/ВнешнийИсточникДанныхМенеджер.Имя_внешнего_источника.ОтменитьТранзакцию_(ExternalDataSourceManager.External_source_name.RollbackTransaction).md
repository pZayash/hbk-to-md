ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

ОтменитьТранзакцию (RollbackTransaction)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ОтменитьТранзакцию()

Описание:

Отменяет транзакцию.  
Изменения, внесенные в процессе транзакции, будут отменены.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
