Панель (Panel)

Рамка (Border)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Рамка](Рамка.md).   
Рамка вокруг страницы и закладок. В режиме отображения закладок рамка бывает только простая одинарная, если закладки не отображаются или установлен режим [РежимПрокручиваемыхСтраниц](Панель.РежимПрокручиваемыхСтраниц.md), то рамка может принимать любое значение типа [Рамка](Рамка.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
