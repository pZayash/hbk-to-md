ПолучитьПараметрыСоединенияПользователя (GetUserConnectionParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПараметрыСоединенияПользователя(<ПользовательИБ>)

Параметры:

<ПользовательИБ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, параметры которого получаются.

Возвращаемое значение:

Тип: [ПараметрыСоединенияВнешнегоИсточникаДанных](ПараметрыСоединенияВнешнегоИсточникаДанных_(ExternalDataSourceConnectionParameters).md).   

Описание:

Получает параметры соединения для пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
