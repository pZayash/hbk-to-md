СредстваТелефонии (TelephonyTools)

ПоддерживаетсяЖурналЗвонков (CallLogSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяЖурналЗвонков()

Возвращаемое значение:

Тип: Булево.   

- Истина - для Android при наличии телефонного модуля,
- Ложь - для остальных операционных систем.

Описание:

Определяет, доступен ли журнал звонков на данном устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Доступен, начиная с версии 8.3.5.
