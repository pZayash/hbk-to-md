СоставПланаОбмена (ExchangePlanContent)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Объект метаданных, который необходимо найти в элементе состава плана обмена.  
Тип параметра - ОбъектМетаданных, описывающий объекты конфигурации, которые могут входить в состав плана обмена (например: [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник_(MetadataObject_Catalog).md), [ОбъектМетаданных: Документ](ОбъектМетаданных_Документ_(MetadataObject_Document).md) и т.д.).

Возвращаемое значение:

Тип: [ЭлементСоставаПланаОбмена](ЭлементСоставаПланаОбмена_(ExchangePlanContentItem).md), [Неопределено](lang__def_Undefined.md).   
Если объекта метаданных с таким именем нет, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск в составе плана обмена элемента, содержащего указанный объект метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
