ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: Булево.   
Истина - изменяется еще ни разу не записанный вид расчета; Ложь - вид расчета уже записан.

Описание:

Определяет, записан ли вид расчета в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.0.
