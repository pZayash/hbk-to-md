Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ВыпадающийСписокОткрыт (DropListIsOpen)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ВыпадающийСписокОткрыт()

Возвращаемое значение:

Истина - список открыт.  
Ложь - список закрыт.

Описание:

Проверяет, открыт ли выпадающий список на экране

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
