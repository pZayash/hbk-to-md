# СправочникОбъект.<Имя справочника>.УстановитьПометкуУдаления

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>, <ВключаяПодчиненные>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак установки / снятие пометки на удаление.  
[Истина](lang__def_BooleanTrue.md) - пометка будет установлена; [Ложь](lang__def_BooleanFalse.md) - пометка будет снята.

<ВключаяПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр установлен в [Истина](lang__def_BooleanTrue.md), то будут помечены на удаление, кроме данного элемента, все его подчиненные элементы в данном справочнике и во всех подчиненных справочниках.  
Если параметр установлен в [Ложь](lang__def_BooleanFalse.md), то будет помечен на удаление только данный элемент.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает/снимет пометку на удаление. Объект при этом не удаляется из базы данных. Окончательное удаление помеченных элементов справочника происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](СправочникОбъект.Имя_справочника.ПередЗаписью_(CatalogObject.Catalog_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](СправочникОбъект.Имя_справочника.ПометкаУдаления_(CatalogObject.Catalog_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Пример:

|  |
| --- |
| Объект.УстановитьПометкуУдаления(Истина,Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
