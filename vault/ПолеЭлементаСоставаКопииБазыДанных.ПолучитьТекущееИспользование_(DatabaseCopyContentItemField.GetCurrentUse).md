ПолеЭлементаСоставаКопииБазыДанных (DatabaseCopyContentItemField)

ПолучитьТекущееИспользование (GetCurrentUse)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

ПолучитьТекущееИспользование()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Получает фактическое (вычисляемое) использование поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
