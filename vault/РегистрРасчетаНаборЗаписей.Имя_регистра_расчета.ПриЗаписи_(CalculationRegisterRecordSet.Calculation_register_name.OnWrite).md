РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>, <Замещение>, <ТолькоЗапись>, <ЗаписьФактическогоПериодаДействия>, <ЗаписьПерерасчетов>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Замещение>

Тип: [Булево](lang__Булево_(Boolean).md).   
Режим записи набора.  
[Истина](lang__def_BooleanTrue.md) - запись осуществляется с заменой существующих в базе данных записей набора.  
[Ложь](lang__def_BooleanFalse.md) - запись осуществляется с "дописыванием" текущего набора записей.

<ТолькоЗапись>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что при записи набора выполняется только запись набора записей без пересчета фактического периода действия и без обновления перерасчетов.

<ЗаписьФактическогоПериодаДействия>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что при записи набора выполняется расчет фактического периода действия.   
Анализировать этот параметр имеет смысл только в том случае, если параметр обработчика <ТолькоЗапись> имеет значение [Ложь](lang__def_BooleanFalse.md).

<ЗаписьПерерасчетов>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что при записи набора выполняется запись перерасчетов регистра.   
Анализировать этот параметр имеет смысл только в том случае, если параметр обработчика <ТолькоЗапись> имеет значение [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта. Процедура-обработчик вызывается после записи объекта в базу данных, но до окончания транзакции записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
