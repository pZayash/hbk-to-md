МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтправитьНавигационнуюСсылкуВнешнегоДоступа (SendExternalAccessURL)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ОтправитьНавигационнуюСсылкуВнешнегоДоступа(<Пользователь>, <Обсуждение>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Внешний пользователь, которому будет отправлена ссылка для доступа.

<Обсуждение> (необязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Если параметр задан, то будет отправлена ссылка, при переходе по которой, будет открыто это обсуждение.

Описание:

Отправляет на [АдресЭлектроннойПочтыДляАутентификации](ПользовательСистемыВзаимодействия.АдресЭлектроннойПочтыДляАутентификации.md) переданного пользователя навигационную ссылку-приглашение для внешнего доступа.  
Если пользователь не является внешним, вызов метода игнорируется.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ПолучитьНавигационнуюСсылкуВнешнегоДоступа](МенеджерСистемыВзаимодействия.ПолучитьНавигационнуюСсылкуВнешнегоДоступа.md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
