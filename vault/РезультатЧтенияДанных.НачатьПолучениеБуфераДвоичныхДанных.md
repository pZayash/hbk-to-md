РезультатЧтенияДанных (ReadDataResult)

НачатьПолучениеБуфераДвоичныхДанных (BeginGetBinaryDataBuffer)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьПолучениеБуфераДвоичныхДанных(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - тип [БуферДвоичныхДанных](БуферДвоичныхДанных.md). Буфер, содержащий результат получения данных.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начать получение результата в виде буфера двоичных данных.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
