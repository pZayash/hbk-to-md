# Расширение колонки табличного поля списка документов.ОтображатьСтандартнуюКартинку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [Расширение колонки табличного поля списка документов](Расширение_колонки_табличного_поля_списка_документов_(Document_list_table_box_column_extension).md) › [Свойства](Расширение_колонки_табличного_поля_списка_документов__Свойства.md)

Расширение колонки табличного поля списка документов (Document list table box column extension)

ОтображатьСтандартнуюКартинку (DisplayDefaultPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения в колонке стандартной картинки, обозначающей документ и его состояние.

Доступность:

Толстый клиент.

Примечание:

Если [Истина](lang__def_BooleanTrue.md), то при выводе свойство [КартинкиСтрок](КолонкаТабличногоПоля.КартинкиСтрок_(TableBoxColumn.RowsPictures).md) колонки табличного поля не используется.

Использование в версии:

Доступен, начиная с версии 8.0.
