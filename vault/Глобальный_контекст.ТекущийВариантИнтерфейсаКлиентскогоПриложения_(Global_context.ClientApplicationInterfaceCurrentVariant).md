Глобальный контекст (Global context)

ТекущийВариантИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceCurrentVariant)

Не рекомендуется использовать, начиная с версии 8.3.10.

Рекомендуется использовать:

- [ТекущийВариантИнтерфейса](КлиентскоеПриложение.ТекущийВариантИнтерфейса_(ClientApplication.CurrentInterfaceVariant).md)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущийВариантИнтерфейсаКлиентскогоПриложения()

Возвращаемое значение:

Тип: [ВариантИнтерфейсаКлиентскогоПриложения](ВариантИнтерфейсаКлиентскогоПриложения_(ClientApplicationInterfaceVariant).md).   

Описание:

Получает текущий вариант интерфейса.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Использование в версии:

Доступен, начиная с версии 8.3.3.

Не рекомендуется использовать, начиная с версии 8.3.10.
