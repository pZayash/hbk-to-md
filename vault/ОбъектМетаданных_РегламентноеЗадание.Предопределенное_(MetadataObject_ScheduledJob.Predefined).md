ОбъектМетаданных: РегламентноеЗадание (MetadataObject: ScheduledJob)

Предопределенное (Predefined)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то в информационной базе автоматически будет создано предопределенное регламентное задание.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
