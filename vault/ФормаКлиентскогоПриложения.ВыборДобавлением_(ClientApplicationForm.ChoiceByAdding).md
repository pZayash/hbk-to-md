ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыборДобавлением (ChoiceByAdding)

Доступен, начиная с версии 8.3.23.

Использование:

Ключевой.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет способ обработки для выбранных значений в поле ввода в режиме расширенного редактирования множественных значений.  
  
Параметр принимает значение [Истина](lang__def_BooleanTrue.md), если поле ввода находится в режиме расширенного редактирования множественных значений и форма открывается в обработчике [НачалоВыбора](Расширение_поля_формы_для_поля_ввода.НачалоВыбора_(Form_field_extension_for_a_text_box.StartChoice).md), когда [СтандартнаяОбработка](html.md)=[Истина](lang__def_BooleanTrue.md).  
  
В остальных случаях параметр принимает значение [Ложь](lang__def_BooleanFalse.md).

[Расширение поля формы для поля ввода](Расширение_поля_формы_для_поля_ввода_(Form_field_extension_for_a_text_box).md)

Доступен, начиная с версии 8.3.23.
