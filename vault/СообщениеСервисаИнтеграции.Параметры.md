СообщениеСервисаИнтеграции (IntegrationServiceMessage)

Параметры (Parameters)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие.md).   
При отправке сообщения позволяет указать специфичный для приложения набор полей, который будет использоваться получателями сообщения для его обработки.  
При получении сообщения содержит специфичный для приложения набор полей, который указан отправителем сообщения. Может использоваться для обработки сообщения.  
Можно устанавливать только при отправке, при попытке изменить в полученном сообщении генерируется исключение.  
В качестве ключа и значения в соответствии используется Строка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
