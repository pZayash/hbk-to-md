Только чтение.

Описание:

Тип: [БиблиотекаМакетовОформленияКомпоновкиДанных](БиблиотекаМакетовОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplateLib).md).   
Библиотека макетов оформления компоновки данных. Доступны следующие предопределенные макеты оформления: Основной(Main), Яркий(Gaudy), Море(Sea), Арктика(Arctic), Зеленый(Green), Античный(Antique).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
