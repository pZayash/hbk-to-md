# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ОбработкаИнтерактивнойАктивации

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md) › [События](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса__События.md)

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ОбработкаИнтерактивнойАктивации (InteractiveActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаИнтерактивнойАктивации(<ТочкаМаршрута>, <Задача>, <СтандартнаяОбработка>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, связанная с задачей, на которой происходит интерактивная активация задачи.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Задача, которую активирует пользователь.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), обработка интерактивной активации производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при интерактивной активации пользователем задач, соответствующих данной точке маршрута бизнес-процесса. Вызов этого обработчика происходит при вызове метода [АктивироватьИнтерактивно](ЗадачаОбъект.Имя_задачи.АктивироватьИнтерактивно_(TaskObject.Task_name.ActivateInteractively).md) у задачи или метода [АктивироватьИнтерактивно](БизнесПроцессОбъект.Имя_бизнес-процесса.АктивироватьИнтерактивно_(BusinessProcessObject.Business_process_name.ActivateInteractively).md) у бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md).  
В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) не вызывается.

Использование в версии:

Доступен, начиная с версии 8.0.
