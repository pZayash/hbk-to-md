Доступен, начиная с версии 8.3.6.

Элементы коллекции:

[ПериодОтображенияПланировщика](ПериодОтображенияПланировщика_(PlannerRepresentationPeriod).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются периоды отображения.  
Возможно обращение к периоду отображения посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию текущих периодов отображения планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
