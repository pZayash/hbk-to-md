ПланыОбменаМенеджер (ExchangePlansManager)

ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации (ReadConfigurationAndConfigurationExtensionsChanges)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации(<ЧтениеСообщенияОбмена>)

Параметры:

<ЧтениеСообщенияОбмена> (обязательный)

Тип: [ЧтениеСообщенияОбмена](ЧтениеСообщенияОбмена_(ExchangeMessageReader).md).   
Объект, через который осуществляется чтение сообщения обмена данными. У этого объекта должен быть выполнен метод [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md), но еще не выполнен метод [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение_(ExchangeMessageReader.EndRead).md).  
После вызова метода объект может быть использован в вызове [ПрочитатьИзменения](ПланыОбменаМенеджер.ПрочитатьИзменения_(ExchangePlansManager.ReadChanges).md), если вызов не привёл к изменениям конфигурации, расширений конфигурации или возникновению исключения.

Возвращаемое значение:

Тип: [ОписаниеИзмененийКонфигурацииВСообщенииОбмена](ОписаниеИзмененийКонфигурацииВСообщенииОбмена_(ConfigurationChangesDescriptionInExchangeMessage).md).   
Для этого метода флаг [РасширенияКонфигурацииИзменены](ОписаниеИзмененийКонфигурацииВСообщенииОбмена.РасширенияКонфигурацииИзменены_(ConfigurationChangesDescriptionInExchangeMessage.ConfigurationExtensionsChanged).md) означает, что расширения конфигурации были изменены и требуется перезапуск сеанса для их применения  
Если флаг [ИзмененныеМетаданныеРасширенийКонфигурацииИзменяютСтруктуруДанных](ОписаниеИзмененийКонфигурацииВСообщенииОбмена.ИзмененныеМетаданныеРасширенийКонфигурацииИзменяютСтруктуруДанных_(ConfigurationChangesDescriptionInExchangeMessage.ChangedConfigurationExtensionsMetadataModifiesDataStructure).md) установлен, а [РасширенияКонфигурацииИзменены](ОписаниеИзмененийКонфигурацииВСообщенииОбмена.РасширенияКонфигурацииИзменены_(ConfigurationChangesDescriptionInExchangeMessage.ConfigurationExtensionsChanged).md) – нет, это значит, что расширения содержали изменения структуры и не смогли быть применены либо из-за невозможности установить монопольную блокировку, либо из-за запуска метода в фоновом задании.

Описание:

Выполняет чтение из сообщения обмена данными изменений конфигурации и расширений конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Доступен, начиная с версии 8.3.22.
