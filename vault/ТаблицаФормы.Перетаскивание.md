ТаблицаФормы (FormTable)

Перетаскивание (Drag)

Доступен, начиная с версии 8.2.

Синтаксис:

Перетаскивание(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Строка>, <Поле>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания.md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

<Строка>

Содержит порядковый номер строки или ссылку на текущий объект.  
Если таблица связана с [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md) или [ДанныеФормыДерево](ДанныеФормыДерево.md), то параметр содержит идентификатор строки над которой находится курсор.   
Для таблиц, связанных с [ДинамическийСписок](ДинамическийСписок.md), содержит ссылку на текущий объект строки таблицы.

<Поле>

Поле, с которым связана данная колонка таблицы, над которой находится объект.

Описание:

Возникает при окончании перетаскивания в поле - приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл.md), то перетаскивание файлов игнорируется, и обработчик данного события не возникает.

Использование в версии:

Доступен, начиная с версии 8.2.
