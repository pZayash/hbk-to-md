Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

Обновить (Refresh)

Доступен, начиная с версии 8.0.

Синтаксис:

Обновить()

Описание:

Обновляет набор записей перечитыванием данных из базы данных.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| НаборЗаписей.Обновить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
