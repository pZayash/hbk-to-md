ИнформацияОбАгрегате (AggregateInformation)

Размер (Size)

Доступен, начиная с версии 8.2.

Только чтение.

Описание:

Тип: Число.   
Содержит расчетный размер агрегата, в байтах на момент перестроения использования агрегата.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
