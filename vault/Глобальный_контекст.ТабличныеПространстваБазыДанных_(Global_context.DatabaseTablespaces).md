Глобальный контекст (Global context)

ТабличныеПространстваБазыДанных (DatabaseTablespaces)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [МенеджерТабличныхПространствБазыДанных](МенеджерТабличныхПространствБазыДанных_(DatabaseTablespacesManager).md).   
Предоставляет доступ к табличным пространствам базы данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.23.
