Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Тип: [Строка](lang__def_String.md).   
Вид интеграции, например, "Telegram".  
Доступно для изменения только у новой незаписанной интеграции.

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
