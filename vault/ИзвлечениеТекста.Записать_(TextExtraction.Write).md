ИзвлечениеТекста (TextExtraction)

Записать (Write)

Доступен, начиная с версии 8.1.

Синтаксис:

Записать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет записан извлеченный текст.

Описание:

Извлекает текст из файла и записывает его в формате [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md) в указанный файл.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Извлечение производится небольшими порциями, поэтому выполнение не требовательно к оперативной памяти.

Использование в версии:

Доступен, начиная с версии 8.1.
