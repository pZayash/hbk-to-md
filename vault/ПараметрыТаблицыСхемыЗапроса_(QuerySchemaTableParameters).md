**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ПараметрТаблицыСхемыЗапроса](ПараметрТаблицыСхемыЗапроса_(QuerySchemaTableParameter).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры таблицы.  
Возможно обращение к параметру таблицы посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПараметрыТаблицыСхемыЗапроса.Индекс_(QuerySchemaTableParameters.IndexOf).md)  
[Количество (Count)](ПараметрыТаблицыСхемыЗапроса.Количество_(QuerySchemaTableParameters.Count).md)  
[Найти (Find)](ПараметрыТаблицыСхемыЗапроса.Найти_(QuerySchemaTableParameters.Find).md)  
[Получить (Get)](ПараметрыТаблицыСхемыЗапроса.Получить_(QuerySchemaTableParameters.Get).md)  

Описание:

Содержит коллекцию параметров таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
