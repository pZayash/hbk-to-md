КонтекстПространствИменXML (XMLNamespaceContext)

Префиксы (Prefixes)

Доступен, начиная с версии 8.1.

Синтаксис:

Префиксы(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен, для которого необходимо получить список префиксов.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает все префиксы, которым соответствует указанное URI пространства имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
