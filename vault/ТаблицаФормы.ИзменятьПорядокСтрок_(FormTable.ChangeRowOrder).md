# ТаблицаФормы.ИзменятьПорядокСтрок

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [ТаблицаФормы](ТаблицаФормы_(FormTable).md) › [Свойства](ТаблицаФормы__Свойства.md)

ТаблицаФормы (FormTable)

ИзменятьПорядокСтрок (ChangeRowOrder)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет изменением порядка строк.   
[Истина](lang__def_BooleanTrue.md) - менять разрешено.  
Данное свойство имеет смысл только для тех данных, редактирование которых не запрещено прикладным смыслом и ограничениями по правам. А также для тех данных, тип которых подразумевает упорядоченность списка (табличные части, [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
