ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Очистить (Clear)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Очистить()

Описание:

Очищает коллекцию. Метод доступен только для доступных полей описания временной таблицы ([ДоступныеПоля](ОписаниеВременнойТаблицыСхемыЗапроса.ДоступныеПоля_(QuerySchemaTempTableDescription.AvailableFields).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
