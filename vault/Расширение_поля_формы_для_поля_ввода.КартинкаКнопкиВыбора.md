Расширение поля формы для поля ввода (Form field extension for a text box)

КартинкаКнопкиВыбора (ChoiceButtonPicture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, отображаемую в кнопке выбора.  
Оптимальный размер картинки определяется размером элемента управления и размером его шрифта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.2.
