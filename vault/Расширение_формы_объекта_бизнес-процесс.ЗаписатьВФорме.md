Расширение формы объекта бизнес-процесс (Business process object form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: Булево.   
Истина - бизнес-процесс успешно записан; Ложь - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
