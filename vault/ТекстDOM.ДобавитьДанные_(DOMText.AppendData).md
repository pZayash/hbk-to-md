# ТекстDOM.ДобавитьДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ТекстDOM](ТекстDOM_(DOMText).md) › [Методы](ТекстDOM__Методы.md)

ТекстDOM (DOMText)

ДобавитьДанные (AppendData)

Доступен, начиная с версии 8.1.

Синтаксис:

ДобавитьДанные(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, добавляемая к символьным данным.

Описание:

Добавляет строку к символьным данным.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
