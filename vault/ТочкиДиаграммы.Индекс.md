ТочкиДиаграммы (ChartPoints)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Точка>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммы](ТочкаДиаграммы.md).   
Точка диаграммы.

Возвращаемое значение:

Тип: Число.   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс указанной точки диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ИндексТочки = Диаграмма.Точки.Индекс(ВыбраннаяТочка); |

Использование в версии:

Доступен, начиная с версии 8.0.
