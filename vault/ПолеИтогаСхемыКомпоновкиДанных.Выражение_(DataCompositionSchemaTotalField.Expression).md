ПолеИтогаСхемыКомпоновкиДанных (DataCompositionSchemaTotalField)

Выражение (Expression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение, по которому будет осуществляться расчет итога по полю. Например: "Сумма(КоличествоКонечныйОстаток)".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
