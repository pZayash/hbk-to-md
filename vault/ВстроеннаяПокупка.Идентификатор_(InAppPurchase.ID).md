ВстроеннаяПокупка (InAppPurchase)

Идентификатор (ID)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Уникальный строковый идентификатор, определяющий конкретную покупку в сервисе-магазине.

Доступность:

Мобильный клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
