МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьМодельРаспознаванияВоВременноеХранилище (GetSpeechToTextModelToTempStorage)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьМодельРаспознаванияВоВременноеХранилище(<ИдентификаторМодели>)

Параметры:

<ИдентификаторМодели> (обязательный)

Тип: [ИдентификаторМоделиРаспознаванияРечи](ИдентификаторМоделиРаспознаванияРечи_(SpeechToTextModelID).md).   
Уникальный идентификатор модели распознавания речи.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает файл модели, из которого модель будет загружена, и сохраняет его во временное хранилище.   
Далее модель можно сохранить локально либо установить для использования.  
Если модель доступна локально, то модель не будет выгружаться из сервиса в Интернет, а будет использоваться локальная модель. Это позволяет выгрузить и сохранить на диск, например, уже установленные в информационной базе модели.  
Если модель локально недоступна, то она может быть получена из сервиса (при наличии сети Интернет).   
При ошибке скачивания модели из сети будет сгенерировано исключение. Для скачивания публичных моделей аутентификация не требуется. Но при предоставлении токена могут стать доступными приватные модели.  
Внимание! В мобильной платформе работа с локальными моделями распознавания не поддерживается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.27.
