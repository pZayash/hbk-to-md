ЧтениеPDF (PDFReader)

НачатьПроверкуПодписи (BeginVerifySignature)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПроверкуПодписи(<ОписаниеОповещения>, <Имя>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <Сертификат> - тип [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md). В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Сертификат> - тип [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md). В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).





<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя подписи для проверки.

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

Описание:

Проверяет действительность присоединенной к PDF-документу подписи с указанным именем. В случае ошибки генерируется исключение и вызывается процедура обработки ошибок, указанная в <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
