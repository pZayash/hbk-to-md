Форма (Form)

ИмяСохраненияПоложенияОкна (WindowOptionsName)

Не рекомендуется использовать, начиная с версии 8.2.

Рекомендуется использовать:

- [КлючСохраненияПоложенияОкна](Форма.КлючСохраненияПоложенияОкна.md)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя для сохранения размеров и положения окна данной формы. Параметры окна, настроенные и сохраненные с данным именем, впоследствии восстанавливаются при следующем показе формы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.2.

Описание изменено в версии 8.2.
