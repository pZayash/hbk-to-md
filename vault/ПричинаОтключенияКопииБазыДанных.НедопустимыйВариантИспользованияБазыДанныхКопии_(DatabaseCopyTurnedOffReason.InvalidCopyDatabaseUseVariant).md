ПричинаОтключенияКопииБазыДанных (DatabaseCopyTurnedOffReason)

НедопустимыйВариантИспользованияБазыДанныхКопии (InvalidCopyDatabaseUseVariant)

Доступен, начиная с версии 8.3.27.

Описание:

Копия не может быть использована для выполнения запроса из-за того, что база данных используется другой копией или информационной базой.

Использование в версии:

Доступен, начиная с версии 8.3.27.
