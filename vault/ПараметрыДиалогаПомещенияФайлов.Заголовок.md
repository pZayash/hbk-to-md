ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

Заголовок (Title)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.   
Заголовок диалога.   
Если имеет значение Неопределено, то используется стандартный заголовок.  
Значение по умолчанию: Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
