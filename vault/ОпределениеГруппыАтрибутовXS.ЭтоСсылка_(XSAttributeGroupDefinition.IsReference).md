ОпределениеГруппыАтрибутовXS (XSAttributeGroupDefinition)

ЭтоСсылка (IsReference)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, является ли данный компонент ссылкой на глобальную группу атрибутов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - компонент является ссылкой.

Использование в версии:

Доступен, начиная с версии 8.1.
