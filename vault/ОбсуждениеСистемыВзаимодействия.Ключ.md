ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

Ключ (Key)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Устанавливает уникальный ключ обсуждения. Уникальность поддерживается в пределах информационной базы в разрезе всех независимых разделителей.  
Доступно для записи только у нового обсуждения (до первого вызова метода [Записать](ОбсуждениеСистемыВзаимодействия.Записать.md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
