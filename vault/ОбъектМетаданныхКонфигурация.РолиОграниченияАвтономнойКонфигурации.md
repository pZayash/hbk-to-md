ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

РолиОграниченияАвтономнойКонфигурации (StandaloneConfigurationRestrictionRoles)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Свойство предназначено для хранения списка ролей, ограничивающих права при работе автономного режима.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
