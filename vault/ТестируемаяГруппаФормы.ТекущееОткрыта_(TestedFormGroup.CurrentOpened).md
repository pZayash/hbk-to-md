ТестируемаяГруппаФормы (TestedFormGroup)

ТекущееОткрыта (CurrentOpened)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ТекущееОткрыта()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), в случаях:

- Группа не свертываемая.
- Группа свертываемая и в данный момент развернута.
- Группа всплывающая и в данный момент отображается.

[Ложь](lang__def_BooleanFalse.md) в случаях:

- Группа свертываемая и в данный момент скрыта.
- Группа всплывающая и в данный момент не отображается.

Описание:

Для свертываемой или всплывающей группы проверяет ее состояние - развернута или нет.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
