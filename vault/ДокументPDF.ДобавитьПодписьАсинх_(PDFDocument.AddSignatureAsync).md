# ДокументPDF.ДобавитьПодписьАсинх

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с PDF](Работа_с_PDF-2.md) › [ДокументPDF](ДокументPDF_(PDFDocument).md) › [Методы](ДокументPDF__Методы.md)

ДокументPDF (PDFDocument)

ДобавитьПодписьАсинх (AddSignatureAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ДобавитьПодписьАсинх(<МенеджерКриптографии>, <ОписаниеПодписи>, <ПредставлениеПодписи>, <УчитыватьОриентациюСтраницы>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

<ОписаниеПодписи> (обязательный)

Тип: [ОписаниеПодписиPDF](ОписаниеПодписиPDF_(PDFSignatureDescription).md).   
Объект, содержащий описание электронной подписи PDF-документа.

<ПредставлениеПодписи> (необязательный)

Тип: [Картинка](Картинка_(Picture).md), [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Объект, представляющий собой оформление (штамп) электронной подписи PDF-документа. Если параметр не указан, оформление не создается (подпись не отображается).

<УчитыватьОриентациюСтраницы> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость учитывать ориентацию страницы при добавлении отображаемого объекта. Если содержит [Неопределено](lang__def_Undefined.md) (или не указан), то в режиме совместимости конфигурации выше [Версия8\_3\_25](РежимСовместимости.Версия8_3_25_(CompatibilityMode.Version8_3_25).md) ориентация страницы учитывается, иначе ориентация страницы не учитывается.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   

Описание:

Асинхронная функция. Создает присоединенную к PDF-документу подпись. Располагает оформление подписи (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Для работы метода в веб-клиенте требуется наличие расширения работы с файлами.

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.24.
