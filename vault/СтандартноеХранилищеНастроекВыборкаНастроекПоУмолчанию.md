Описание:

Объект этого типа возвращается методом [ВыбратьНастройкиПоУмолчанию](СтандартноеХранилищеНастроекМенеджер.ВыбратьНастройкиПоУмолчанию.md) объекта типа [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md) и представляет собой специализированный способ перебора настроек по умолчанию, содержащихся в хранилище. Обход выборки выполняется системой динамически. Это означает, что использование выборки не считывает все настройки по умолчанию из хранилища сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки хранилища, содержащие большие количества настроек по умолчанию и не загружает в память всю выборку.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
