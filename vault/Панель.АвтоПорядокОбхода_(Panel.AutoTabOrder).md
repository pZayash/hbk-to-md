Панель (Panel)

АвтоПорядокОбхода (AutoTabOrder)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Истина](lang__def_BooleanTrue.md), то порядок обхода элементов управления определяется автоматически при их добавлении или перемещении.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
