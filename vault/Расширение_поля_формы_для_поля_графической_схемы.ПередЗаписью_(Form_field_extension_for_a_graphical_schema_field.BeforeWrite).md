# Расширение поля формы для поля графической схемы.ПередЗаписью

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля графической схемы](Расширение_поля_формы_для_поля_графической_схемы_(Form_field_extension_for_a_graphical_schema_field).md) › [События](Расширение_поля_формы_для_поля_графической_схемы__События.md)

Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если выполняется команда "Сохранить как".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) позволяет отказаться от записи.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Событие, возникающее перед интерактивной записью.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
