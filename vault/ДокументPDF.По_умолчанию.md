ДокументPDF (PDFDocument)

По умолчанию

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый ДокументPDF()

Описание:

Формирует неинициализированный объект ДокументPDF. Для инициализации объекта используются методы [Прочитать](ДокументPDF.Прочитать.md) и [ПрочитатьАсинх](ДокументPDF.ПрочитатьАсинх.md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
