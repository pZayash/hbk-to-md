Свойства:

[НастройкаКолонок (ColumnsSetting)](Расширение_табличного_поля_списка_колонок_анализа_данных.НастройкаКолонок.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md), предназначенные для интерактивной работы со списком колонок анализа данных. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [КолонкаАнализаДанных](КолонкаАнализаДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
