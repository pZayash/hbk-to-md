XBase (XBase)

Следующая (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующая()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - получена следующая запись; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Позиционирует объект на следующей записи. В зависимости от того, установлен ли текущий индекс, позиционирование происходит либо в порядке возрастания физического номера записи, либо в порядке возрастания значения выражения текущего индекса (если индекс создавался с установленным флагом "Убывание", то в порядке убывания).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Обнуляет значения всех полей всех существующих записей  xB.Первая(); Пока Истина Цикл     xB.Очистить();     xB.Записать();     Если Не xB.Следующая() Тогда         Прервать;     КонецЕсли; КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
