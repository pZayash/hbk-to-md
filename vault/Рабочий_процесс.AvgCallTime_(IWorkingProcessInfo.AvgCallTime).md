Рабочий процесс (IWorkingProcessInfo)

AvgCallTime (AvgCallTime)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Показывает среднее время обслуживания рабочим процессом одного клиентского обращения. Оно складывается из: значений свойств [AvgServerCallTime](Рабочий_процесс.AvgServerCallTime_(IWorkingProcessInfo.AvgServerCallTime).md), [AvgDBCallTime](Рабочий_процесс.AvgDBCallTime_(IWorkingProcessInfo.AvgDBCallTime).md), [AvgLockCallTime](Рабочий_процесс.AvgLockCallTime_(IWorkingProcessInfo.AvgLockCallTime).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
