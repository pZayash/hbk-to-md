ОбъектМетаданных: Подсистема (MetadataObject: Subsystem)

ИспользоватьОднуКоманду (UseOneCommand)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то подсистема должна содержать только одну команду.  
В клиентском приложении при выборе такого раздела (подсистемы) будет сразу выполнена команда.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.21.
