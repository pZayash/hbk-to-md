МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблонСообщенияАсинх (GetMessageTemplateAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблонСообщенияАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия.md).   
Идентификатор шаблона сообщения.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - значение типа [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия.md) или Неопределено:

- Для интеграции типа WhatsApp возвращает шаблон сообщения по его идентификатору.
- Для остальных интеграций или если шаблон не найден возвращает Неопределено.

Описание:

Асинхронная функция. Получает шаблон сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
