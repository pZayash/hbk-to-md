# НастройкаУсловногоОформления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Оформление](Оформление-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияНастройкойУсловногоОформления](ЭлементУправленияНастройкойУсловногоОформления_(ConditionalAppearanceSettingControl).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки условного оформления.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[Найти (Find)](НастройкаУсловногоОформления.Найти_(ConditionalAppearanceSetting.Find).md)  
[Очистить (Clear)](НастройкаУсловногоОформления.Очистить_(ConditionalAppearanceSetting.Clear).md)  
[Получить (Get)](НастройкаУсловногоОформления.Получить_(ConditionalAppearanceSetting.Get).md)  
[Удалить (Delete)](НастройкаУсловногоОформления.Удалить_(ConditionalAppearanceSetting.Delete).md)  

Описание:

Набор элементов управления условного оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.
