**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Работа_с_запросами.md">Работа с запросами</a> › [Выполнение и работа с запросами во встроенном языке](Выполнение_и_работа_с_запросами_во_встроенном_языке.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаРезультатаЗапроса](КолонкаРезультатаЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс колонки (нумерация с 0).  

Свойства:

[\<Имя колонки\> (\<Column name\>)](КоллекцияКолонокРезультатаЗапроса.Имя_колонки.md)  

Методы:

[Индекс (IndexOf)](КоллекцияКолонокРезультатаЗапроса.Индекс.md)  
[Количество (Count)](КоллекцияКолонокРезультатаЗапроса.Количество.md)  
[Найти (Find)](КоллекцияКолонокРезультатаЗапроса.Найти.md)  
[Получить (Get)](КоллекцияКолонокРезультатаЗапроса.Получить.md)  

Описание:

Представляет собой коллекцию колонок результата запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

[РезультатЗапроса](РезультатЗапроса.md), свойство [Колонки](РезультатЗапроса.Колонки.md)
