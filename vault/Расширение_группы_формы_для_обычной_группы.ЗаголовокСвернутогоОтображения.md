Расширение группы формы для обычной группы (Form group extension for a usual group)

ЗаголовокСвернутогоОтображения (CollapsedRepresentationTitle)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст заголовка, отображаемый, когда группа свернута.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Если содержит пустую строку, то в свернутом виде отображается обычный заголовок.

Использование в версии:

Доступен, начиная с версии 8.3.3.
