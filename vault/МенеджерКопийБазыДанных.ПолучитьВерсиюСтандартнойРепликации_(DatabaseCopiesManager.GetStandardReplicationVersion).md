МенеджерКопийБазыДанных (DatabaseCopiesManager)

ПолучитьВерсиюСтандартнойРепликации (GetStandardReplicationVersion)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПолучитьВерсиюСтандартнойРепликации()

Возвращаемое значение:

Тип: [ВерсияСтандартнойРепликацииКопийБазыДанных](ВерсияСтандартнойРепликацииКопийБазыДанных_(DatabaseCopiesStandardReplicationVersion).md).   

Описание:

Возвращает версию реализации стандартной репликации копий базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
