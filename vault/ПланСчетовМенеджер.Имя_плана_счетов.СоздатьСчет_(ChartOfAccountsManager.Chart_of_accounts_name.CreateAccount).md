ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

СоздатьСчет (CreateAccount)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьСчет()

Возвращаемое значение:

Тип: [ПланСчетовОбъект.\<Имя плана счетов\>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md).   

Описание:

Создает новый счет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Счет = ПланыСчетов.Хозрасчетный.СоздатьСчет(); |

Использование в версии:

Доступен, начиная с версии 8.0.
