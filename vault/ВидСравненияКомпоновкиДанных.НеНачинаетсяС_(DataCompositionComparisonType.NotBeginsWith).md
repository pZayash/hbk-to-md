ВидСравненияКомпоновкиДанных (DataCompositionComparisonType)

НеНачинаетсяС (NotBeginsWith)

Доступен, начиная с версии 8.3.5.

Описание:

Сравниваемое значение не начинается с подстроки, заданной в условии сравнения.

Использование в версии:

Доступен, начиная с версии 8.3.5.
