РежимОтображенияЭлементаНастройкиКомпоновкиДанных (DataCompositionSettingsItemViewMode)

БыстрыйДоступ (QuickAccess)

Доступен, начиная с версии 8.1.

Описание:

Этот элемент настройки предназначен для режима быстрого доступа.  
Он будет доступен в табличном поле, если у расширения табличного поля отбора свойство РежимОтображения будет установлено в значение Все или БыстрыйДоступ.

Использование в версии:

Доступен, начиная с версии 8.1.
