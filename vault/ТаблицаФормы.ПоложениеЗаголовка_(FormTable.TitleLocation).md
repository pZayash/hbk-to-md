# ТаблицаФормы.ПоложениеЗаголовка

ТаблицаФормы (FormTable)

ПоложениеЗаголовка (TitleLocation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеЗаголовкаЭлементаФормы](ПоложениеЗаголовкаЭлементаФормы_(FormItemTitleLocation).md).   
Управляет показом в форме заголовка к таблице. Если заголовок к таблице не нужен, следует установить данное свойство в [Нет](ПоложениеЗаголовкаЭлементаФормы.Нет_(FormItemTitleLocation.None).md), в то время как пустая строка в свойстве [Заголовок](ТаблицаФормы.Заголовок_(FormTable.Title).md) означает автоматическое формирование заголовка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
