Описание:

Предназначен для описания координат местоположения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен (http://v8.1c.ru/8.3/data/ext). Имя типа XDTO: GeographicCoordinates.

Использование в версии:

Доступен, начиная с версии 8.3.3.
