ИнформацияОбОшибке (ErrorInfo)

ЯвляетсяОшибкойКатегории (IsErrorOfCategory)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ЯвляетсяОшибкойКатегории(<Категория>, <УчитыватьПричины>)

Параметры:

<Категория> (обязательный)

Тип: [КатегорияОшибки](КатегорияОшибки_(ErrorCategory).md).   
Проверяемая категория ошибки.

<УчитыватьПричины> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр имеет значение [Истина](lang__def_BooleanTrue.md), то учитывается вся цепочка причин у описания ошибки.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Для категории [ВсеОшибки](КатегорияОшибки.ВсеОшибки_(ErrorCategory.AllErrors).md) всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Проверяет, что ошибка принадлежит к указанной категории.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
