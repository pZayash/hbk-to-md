Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), предназначенные для интерактивной работы с записями регистра сведений. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) содержит значение типа [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [РегистрСведенийСписок.](РегистрСведенийСписок.Имя_регистра_сведений_(InformationRegisterList.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийСписок.Имя_регистра_сведений_(InformationRegisterList.Information_register_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
