Описание:

Элемент графической схемы "Старт". Отображает точку маршрута бизнес-процесса, с которой начинается выполнение бизнес-процесса. Для бизнес-процесса, имеющего несколько точек "Старт", при запуске должна быть указана нужная точка старта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md), метод [Старт](БизнесПроцессОбъект.Имя_бизнес-процесса.Старт_(BusinessProcessObject.Business_process_name.Start).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
