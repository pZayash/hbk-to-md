ДействиеЭлементаПланировщика (PlannerItemAction)

Доступность (Enabled)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если Истина, то при нажатии на элемент будет вызвано событие [НажатиеНаДействиеПланировщика](Расширение_поля_формы_для_поля_планировщика.НажатиеНаДействиеПланировщика.md).   
Значение по умолчанию: Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
