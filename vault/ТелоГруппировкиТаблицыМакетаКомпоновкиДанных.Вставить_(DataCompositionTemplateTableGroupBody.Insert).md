ТелоГруппировкиТаблицыМакетаКомпоновкиДанных (DataCompositionTemplateTableGroupBody)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [МакетГруппировкиТаблицыМакетаКомпоновкиДанных](МакетГруппировкиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroupTemplate).md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroup).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableRecords).md).

Возвращаемое значение:

Тип: [ТелоГруппировкиТаблицыМакетаКомпоновкиДанных](ТелоГруппировкиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroupBody).md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroup).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableRecords).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
