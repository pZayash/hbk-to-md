СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

Данные (Data)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Данные, содержащиеся в сообщении.  
Значение должно поддерживать XDTO-сериализацию.  
При сохранении объекта значение сериализуется в формат JSON.  
Десериализация выполняется при обращении к свойству.  
Размер данных (в сериализованном виде) ограничен 100 Кб.  
При превышении ограничения будет сгенерировано исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Свойство доступно для записи только на сервере.

Использование в версии:

Доступен, начиная с версии 8.3.11.
