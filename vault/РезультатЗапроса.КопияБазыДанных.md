РезультатЗапроса (QueryResult)

КопияБазыДанных (DatabaseCopy)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит имя копии, на которой выполнен запрос или пустую строку, если запрос выполнен на основной базе данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.16.
