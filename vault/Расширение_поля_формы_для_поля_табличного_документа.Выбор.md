Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<Область>, <СтандартнаяОбработка>)

Параметры:

<Область>

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Выбранное значение.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

Описание:

Событие, возникающее при выборе диаграммы (например, при двойном щелчке мыши).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
