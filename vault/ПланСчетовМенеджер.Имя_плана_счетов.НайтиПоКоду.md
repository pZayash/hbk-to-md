ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

НайтиПоКоду (FindByCode)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>, <Родитель>)

Параметры:

<Код> (обязательный)

Тип: Строка.   
Искомый код.

<Родитель> (необязательный)

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов.md).   
Родитель, в пределах которого нужно выполнять поиск. Если не указан, то во всем плане счетов.

Возвращаемое значение:

Тип: [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>, Неопределено.   
Если не существует ни одного элемента с требуемым кодом, то будет возвращена пустая ссылка.  
Если код не задан (длина = 0), то будет возвращено Неопределено.

Описание:

Осуществляет поиск элемента по его коду.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Родитель = ПланыСчетов.Основной.НайтиПоКоду(КодРодителя); |

Использование в версии:

Доступен, начиная с версии 8.0.
