# Сервис кластера.MainOnly

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Сервис кластера](Сервис_кластера_(IClusterServiceInfo).md) › [Свойства](Сервис_кластера__Свойства.md)

Сервис кластера (IClusterServiceInfo)

MainOnly (MainOnly)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит признак поддержки главным и дополнительным менеджером кластера.  
1, если сервис поддерживается только главным менеджером кластера и не может быть перенесена в процесс дополнительного менеджера кластера.   
0, если сервис может поддерживаться как главным, так и дополнительным менеджером кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
