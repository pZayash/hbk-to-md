# Расширение группы формы для обычной группы.Показать

Расширение группы формы для обычной группы (Form group extension for a usual group)

Показать (Show)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Показать()

Описание:

В зависимости от значения свойства [Поведение](Расширение_группы_формы_для_обычной_группы.Поведение_(Form_group_extension_for_a_usual_group.Behavior).md) группы:

- [Обычное](ПоведениеОбычнойГруппы.Обычное_(UsualGroupBehavior.Usual).md) - никакие действия не выполняются,
- [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая_(UsualGroupBehavior.Collapsible).md) - разворачивает группу,
- [Всплывающая](ПоведениеОбычнойГруппы.Всплывающая_(UsualGroupBehavior.PopUp).md) - группа отображается на экране. Если другая группа была видима, она скрывается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
