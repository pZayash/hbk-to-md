Форма (Form)

Активизировать (Activate)

Доступен, начиная с версии 8.0.

Синтаксис:

Активизировать()

Описание:

Активизирует уже открытую форму.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если Форма.Открыта() Тогда     Форма.Активизировать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
