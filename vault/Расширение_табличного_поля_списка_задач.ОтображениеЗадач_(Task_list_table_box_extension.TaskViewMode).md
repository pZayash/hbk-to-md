Расширение табличного поля списка задач (Task list table box extension)

ОтображениеЗадач (TaskViewMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РежимСпискаЗадач](РежимСпискаЗадач_(TaskListMode).md).   
Управляет режимом отображения задач.

Доступность:

Толстый клиент.

Примечание:

При изменении режима сбрасываются установленные ранее отборы.

Использование в версии:

Доступен, начиная с версии 8.0.
