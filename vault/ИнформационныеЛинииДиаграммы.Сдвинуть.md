ИнформационныеЛинииДиаграммы (ChartReferenceLines)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Сдвинуть(<Линия>, <Смещение>)

Параметры:

<Линия> (обязательный)

Тип: [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы.md).   
Информационная линия, которую требуется сдвинуть.

<Смещение> (обязательный)

Тип: Число.   
Количество позиций, на которое необходимо сдвинуть информационную линию.

Описание:

Сдвигает информационную линию на указанное количество позиций.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
