Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

РазрешитьВыборКорня (AllowRootChoice)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность выбора корня в форме с динамическим списком, отображаемом в виде дерева.  
При открытии формы списка со значением параметра [Истина](lang__def_BooleanTrue.md) и параметром [РежимВыбора](Расширение_формы_клиентского_приложения_для_динамического_списка.РежимВыбора_(Client_application_form_extension_for_dynamic_lists.ChoiceMode).md)=[Истина](lang__def_BooleanTrue.md) для всех таблиц, связанных с основным реквизитом типа [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), автоматически будут переопределены значения некоторых свойств:

- свойство [ОтображатьКорень](Расширение_таблицы_формы_для_динамического_списка.ОтображатьКорень_(Form_table_extension_for_dynamic_list.ShowRoot).md) будет установлено в [Истина](lang__def_BooleanTrue.md);
- свойство [РазрешитьВыборКорня](Расширение_таблицы_формы_для_динамического_списка.РазрешитьВыборКорня_(Form_table_extension_for_dynamic_list.AllowRootChoice).md) будет установлено в [Истина](lang__def_BooleanTrue.md);
- свойство [Отображение](ТаблицаФормы.Отображение_(FormTable.Representation).md) будет установлено в [Дерево](ОтображениеТаблицы.Дерево_(TableRepresentation.Tree).md), если оно было не [Дерево](ОтображениеТаблицы.Дерево_(TableRepresentation.Tree).md).

Команда "Переместить в группу" при открытии формы передает параметр со значением [Истина](lang__def_BooleanTrue.md).  
В конструкторах форм списков для всех списков, связанных с основным реквизитом, свойству [НачальноеОтображениеДерева](ТаблицаФормы.НачальноеОтображениеДерева_(FormTable.InitialTreeView).md) устанавливается значение [РаскрыватьВерхнийУровень](НачальноеОтображениеДерева.РаскрыватьВерхнийУровень_(InitialTreeView.ExpandTopLevel).md).

Использование в версии:

Доступен, начиная с версии 8.2.
