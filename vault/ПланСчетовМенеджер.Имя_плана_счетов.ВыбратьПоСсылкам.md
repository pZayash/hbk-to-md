ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на элементы плана счетов.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [ПланСчетовВыборка.\<Имя плана счетов\>](ПланСчетовВыборка.Имя_плана_счетов.md).   

Описание:

Формирует выборку элементов плана счетов по заданному источнику ссылок на элементы плана счетов соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы плана счетов соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    Хозрасчетный.Ссылка КАК Ссылка     |ИЗ     |    ПланСчетов.Хозрасчетный КАК Хозрасчетный     |"; Результат = Запрос.Выполнить(); Выборка = ПланыСчетов.Хозрасчетный.ВыбратьПоСсылкам(Результат, 0); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
