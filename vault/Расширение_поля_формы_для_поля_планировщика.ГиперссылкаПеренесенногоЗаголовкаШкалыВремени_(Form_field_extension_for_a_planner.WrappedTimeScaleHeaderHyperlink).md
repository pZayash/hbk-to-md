Расширение поля формы для поля планировщика (Form field extension for a planner)

ГиперссылкаПеренесенногоЗаголовкаШкалыВремени (WrappedTimeScaleHeaderHyperlink)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет внешний вид и поведение поля при нажатии левой кнопкой мыши на перенесенный заголовок шкалы времени.

- [Истина](lang__def_BooleanTrue.md) - вызывается событие [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд_(Form_field_extension_for_a_planner.CommandGenerateProcessing).md).
- [Ложь](lang__def_BooleanFalse.md) - никакие действия не выполняются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
