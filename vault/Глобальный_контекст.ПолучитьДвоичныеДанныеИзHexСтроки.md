Глобальный контекст (Global context)

ПолучитьДвоичныеДанныеИзHexСтроки (GetBinaryDataFromHexString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДвоичныеДанныеИзHexСтроки(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Строка в формате Base 16 (Hex).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   

Описание:

Преобразует строку в формате Base 16 (Hex) в двоичные данные.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
