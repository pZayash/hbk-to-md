ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

РежимСокращенияТипа (TypeReductionMode)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: [РежимСокращенияТипа](РежимСокращенияТипа_(TypeReductionMode).md).   
Действия над значениями стандартного реквизита справочника "Владелец" при обновлении конфигурации базы данных, если один или несколько владельцев были исключены из состава владельцев. По умолчанию [Запрещать](РежимСокращенияТипа.Запрещать_(TypeReductionMode.Deny).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.25.
