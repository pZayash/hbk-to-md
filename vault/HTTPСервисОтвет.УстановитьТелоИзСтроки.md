HTTPСервисОтвет (HTTPServiceResponse)

УстановитьТелоИзСтроки (SetBodyFromString)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТелоИзСтроки(<Тело>, <Кодировка>, <ИспользоватьBOM>)

Параметры:

<Тело> (обязательный)

Тип: Строка.   
Строка, которая будет записана в тело ответа.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Определяет кодировку, в которой будет отправлено тело HTTP-ответа. Если параметр не задан, то тело отправляется в кодировке [UTF8](КодировкаТекста.UTF8.md).  
Установка этого параметра не вызывает автоматической установки соответствующего HTTP-заголовка.  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

- HZ-GB-2312

.

<ИспользоватьBOM> (необязательный)

Тип: [ИспользованиеByteOrderMark](ИспользованиеByteOrderMark.md).   
Использовать метку порядка байтов (BOM) для кодировок, которые ее поддерживают: UTF-8, UTF-16 и др.  
В режиме совместимости конфигурации значение по умолчанию - [Использовать](ИспользованиеByteOrderMark.Использовать.md).  
Значение по умолчанию: [Авто](ИспользованиеByteOrderMark.Авто.md).

Описание:

Устанавливает тело ответа из строки.

Доступность:

Сервер, толстый клиент.

[HTTPСервисОтвет](HTTPСервисОтвет.md), метод [УстановитьТелоИзДвоичныхДанных](HTTPСервисОтвет.УстановитьТелоИзДвоичныхДанных.md)
