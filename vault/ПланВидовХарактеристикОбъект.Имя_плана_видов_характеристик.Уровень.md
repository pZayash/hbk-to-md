ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Уровень (Level)

Доступен, начиная с версии 8.0.

Синтаксис:

Уровень()

Возвращаемое значение:

Тип: Число.   
Уровень элемента плана видов характеристик.

Описание:

Получает уровень элемента плана видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл только для многоуровневых планов видов характеристик. Следует учитывать, что уровень элемента может меняться, например, при переносе в другую группу (смене родителя). Для элемента, не имеющего родителя, уровень будет равняться 0.

Пример:

|  |
| --- |
| Сообщить("Данный элемент принадлежит уровню " +           СсылкаНаЭлемент.Уровень()); |

Использование в версии:

Доступен, начиная с версии 8.0.
