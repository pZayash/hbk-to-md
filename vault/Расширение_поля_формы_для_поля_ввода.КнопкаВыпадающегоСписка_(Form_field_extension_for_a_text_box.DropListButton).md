Расширение поля формы для поля ввода (Form field extension for a text box)

КнопкаВыпадающегоСписка (DropListButton)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет, будет ли показываться кнопка открытия выпадающего списка. [Истина](lang__def_BooleanTrue.md) - кнопка показывается.  
Если режим совместимости не используется или выше [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md) и значение равно [Неопределено](lang__def_Undefined.md), то кнопка отображается у полей ввода, редактирующих значения следующих типов:

- Ссылочный тип, в том числе перечисление;
- Системное перечисление;
- [Булево](lang__Булево_(Boolean).md);
- [СтандартнаяДатаНачала](СтандартнаяДатаНачала_(StandardBeginningDate).md);
- [СтандартныйПериод](СтандартныйПериод_(StandardPeriod).md);
- Любого типа, если свойство [РежимВыбораИзСписка](Расширение_поля_формы_для_поля_ввода.РежимВыбораИзСписка_(Form_field_extension_for_a_text_box.ListChoiceMode).md) имеет значение [Истина](lang__def_BooleanTrue.md).

При установленном режиме совместимости [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md) и ниже значение [Неопределено](lang__def_Undefined.md) игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
