МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

УстановитьНаклейку (SetBadge)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьНаклейку(<Наклейка>)

Параметры:

<Наклейка> (обязательный)

Тип: [Число](lang__def_Number.md).   
Содержит количество новых объектов, тип которых зависит от конфигурации.

Описание:

Устанавливает значение наклейки на значке.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Работает только под управлением операционной системы iOS.

Использование в версии:

Доступен, начиная с версии 8.3.6.
