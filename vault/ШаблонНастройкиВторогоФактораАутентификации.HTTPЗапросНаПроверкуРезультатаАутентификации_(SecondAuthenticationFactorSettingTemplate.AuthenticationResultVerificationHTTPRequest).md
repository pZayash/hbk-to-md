HTTPЗапросНаПроверкуРезультатаАутентификации (AuthenticationResultVerificationHTTPRequest)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [HTTPЗапрос](HTTPЗапрос_(HTTPRequest).md), [Неопределено](lang__def_Undefined.md).   
Запрос к провайдеру, проверяющий результат аутентификации в случае, если провайдер выполняет аутентификацию самостоятельно. [Неопределено](lang__def_Undefined.md), если опрос результата не требуется.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
