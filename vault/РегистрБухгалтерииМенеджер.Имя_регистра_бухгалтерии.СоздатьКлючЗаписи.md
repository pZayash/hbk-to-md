РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии> (AccountingRegisterManager.<Accounting register name>)

СоздатьКлючЗаписи (CreateRecordKey)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьКлючЗаписи(<ЗначенияКлюча>)

Параметры:

<ЗначенияКлюча> (обязательный)

Тип: [Структура](Структура.md).   
Структура, содержащая значения для заполнения свойств ключа записи.  
Идентификаторы элементов структуры должны соответствовать идентификаторам ключевых полей.

Возвращаемое значение:

Тип: [РегистрБухгалтерииКлючЗаписи.\<Имя регистра бухгалтерии\>](РегистрБухгалтерииКлючЗаписи.Имя_регистра_бухгалтерии.md).   

Описание:

Создает ключ записи по переданной структуре.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Отбор = Новый Структура; Отбор.Вставить("Организация", ВыбОрганизация); Отбор.Вставить("Валюта", ВыбВалюта); Рег = РегистрыБухгалтерии.Хозрасчетный; КлючЗаписи = Рег.СоздатьКлючЗаписи(Отбор); |

Использование в версии:

Доступен, начиная с версии 8.0.
