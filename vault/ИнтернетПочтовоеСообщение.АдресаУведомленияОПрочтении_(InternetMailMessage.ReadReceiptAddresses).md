ИнтернетПочтовоеСообщение (InternetMailMessage)

АдресаУведомленияОПрочтении (ReadReceiptAddresses)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИнтернетПочтовыеАдреса](ИнтернетПочтовыеАдреса_(InternetMailAddresses).md).   
Коллекция адресов, на которые будет высылаться уведомление о прочтении.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
