ЧтениеFastInfoset (FastInfosetReader)

ОткрытьФайл (OpenFile)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьФайл(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла в формате Fast Infoset.

Описание:

Открывает Fast Infoset-файл для чтения данным объектом. Если перед вызовом данного метода уже производилось чтение данных из другого файла, то чтение прекращается и объект инициализируется для чтения из указанного файла.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Чтение = Новый ЧтениеFastInfoset; Чтение.ОткрытьФайл("d:/performance.fi"); |

Использование в версии:

Доступен, начиная с версии 8.1.
