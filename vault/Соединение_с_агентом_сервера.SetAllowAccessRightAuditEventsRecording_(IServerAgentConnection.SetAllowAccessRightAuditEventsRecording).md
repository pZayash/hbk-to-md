Соединение с агентом сервера (IServerAgentConnection)

SetAllowAccessRightAuditEventsRecording (SetAllowAccessRightAuditEventsRecording)

Доступен, начиная с версии 8.3.26.

Синтаксис:

SetAllowAccessRightAuditEventsRecording(<РазрешитьЗаписьСобытийАудита>)

Параметры:

<РазрешитьЗаписьСобытийАудита> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешает запись событий аудита прав доступа информационных баз кластера.  
Если [Истина](lang__def_BooleanTrue.md), то запись событий аудита выполняется.  
К событиям аудита относятся:

Описание:

Устанавливает значение свойства [Встроенное](ТипХранилищаДвоичныхДанных.Встроенное_(BinaryDataStorageType.Embedded).md) (разрешение записи событий аудита).  
К событиям аудита относятся:

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
