РежимИспользованияСинхронныхВызововРасширенийПлатформыИВнешнихКомпонент (SynchronousPlatformExtensionAndAddInCallUseMode)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.3.6.

Описание:

В режиме "Конфигуратор":

- При синтаксическом контроле из глобального контекста синхронные методы будут удалены.
- Расширенная проверка конфигурации выдает ошибки при обнаружении синхронных методов на клиенте.
- В контекстной подсказке синхронные методы не отображаются.

.  
В режиме "1С:Предприятие" при вызове синхронных методов будет генерироваться исключение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
