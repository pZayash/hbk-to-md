КоллекцияОбластейТабличногоДокумента (SpreadsheetDocumentAreaCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество элементов в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| МакетСоставаПоказателей = ЭтотОбъект.ПолучитьМакет("Показатели"); Для Индекс = 0 По МакетСоставаПоказателей.Области.Количество() - 1 Цикл     ТекОбласть    = МакетСоставаПоказателей.Области[Индекс];     ИмяСтраницы   = ТекОбласть.Имя;     // ...  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
