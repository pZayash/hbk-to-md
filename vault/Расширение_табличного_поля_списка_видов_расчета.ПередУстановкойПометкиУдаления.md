Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то пометка удаления изменена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед установкой пометки удаления в списке видов расчета.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПередЗаписью.md) и [ПриЗаписи](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПриЗаписи.md) объекта [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
