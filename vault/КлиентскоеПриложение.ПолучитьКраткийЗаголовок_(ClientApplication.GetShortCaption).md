КлиентскоеПриложение (ClientApplication)

ПолучитьКраткийЗаголовок (GetShortCaption)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьКраткийЗаголовок()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает краткий заголовок приложения, размещенный в заголовке окна приложения перед текстом системного заголовка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Недоступен в режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md).

См. также:

[КлиентскоеПриложение](КлиентскоеПриложение_(ClientApplication).md), метод [УстановитьКраткийЗаголовок](КлиентскоеПриложение.УстановитьКраткийЗаголовок_(ClientApplication.SetShortCaption).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
