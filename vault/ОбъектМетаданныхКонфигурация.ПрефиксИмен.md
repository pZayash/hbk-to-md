ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПрефиксИмен (NamePrefix)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Свойство доступно только для расширений конфигурации. При работе в режиме конфигуратора имена новых собственных объектов будут начинаться со значения, указанного в данном свойстве. В режиме "1С:Предприятие" данное свойство на поведение не влияет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
