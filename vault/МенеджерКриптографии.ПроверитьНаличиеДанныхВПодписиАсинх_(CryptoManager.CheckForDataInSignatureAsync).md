МенеджерКриптографии (CryptoManager)

ПроверитьНаличиеДанныхВПодписиАсинх (CheckForDataInSignatureAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПроверитьНаличиеДанныхВПодписиАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Электронная подпись, в которой определяется наличие данных.  
Данные подписи могут размещаться в файле (в этом случае указывается имя файла) или представлены в явном виде как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md) или [ФайловыйПоток](ФайловыйПоток_(FileStream).md).

Возвращаемое значение:

Тип: Обещание.   
Результатом выполнения обещания является параметр типа [Булево](lang__Булево_(Boolean).md), содержащий признак наличия данных: [Истина](lang__def_BooleanTrue.md) – если заданная подпись содержит данные и [Ложь](lang__def_BooleanFalse.md) – в противном случае.

Описание:

Определяет наличия данных в электронной подписи: присоединенная подпись или отсоединенная.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
