Переключатель (RadioButton)

Заголовок (Caption)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст переключателя.

Доступность:

Толстый клиент.

Примечание:

Если текст не умещается по ширине, он автоматически разбивается на несколько строк с переносом по словам. Если перенос по словам невозможен, видимый текст будет заканчиваться троеточием. Если разбитый на строки текст не умещается по высоте, то он будет прижат к верхней границе независимо от установленного способа выравнивания по вертикали, не уместившиеся строки не будут показаны.

Использование в версии:

Доступен, начиная с версии 8.0.
