СхемаЗапроса (QuerySchema)

РежимКомпоновкиДанных (DataCompositionMode)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет режимом схемы запроса.  
Если [Истина](lang__def_BooleanTrue.md), то для каждого запроса в пакете запросов схемы запроса будут недоступны поля [ОбщиеИтоги](ЗапросВыбораСхемыЗапроса.ОбщиеИтоги_(QuerySchemaSelectQuery.Overall).md), [КонтрольныеТочкиИтогов](ЗапросВыбораСхемыЗапроса.КонтрольныеТочкиИтогов_(QuerySchemaSelectQuery.TotalCalculationFields).md) и [ВыраженияИтогов](ЗапросВыбораСхемыЗапроса.ВыраженияИтогов_(QuerySchemaSelectQuery.TotalExpressions).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
