ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия (CollaborationSystemUsersChoiceFormGetProcessing)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия(<НазначениеВыбора>, <Параметры>, <ВыбраннаяФорма>, <СтандартнаяОбработка>)

Параметры:

<НазначениеВыбора>

Тип: [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md).   
Назначение выбора пользователя системы взаимодействия.

<Параметры>

Тип: [Структура](Структура_(Structure).md).   
Параметры получения формы.

<ВыбраннаяФорма>

Тип: [Строка](lang__def_String.md).   
Имя открываемой формы.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если после вызова обработчика параметр <СтандартнаяОбработка> содержит значение [Истина](lang__def_BooleanTrue.md), то в качестве формы выбора пользователей системы взаимодействия используется форма, заданная в свойстве [ОсновнаяФормаВыбораПользователейСистемыВзаимодействия](ОбъектМетаданныхКонфигурация.ОсновнаяФормаВыбораПользователейСистемыВзаимодействия_(ConfigurationMetadataObject.DefaultCollaborationSystemUsersChoiceForm).md).  
Если свойство [ОсновнаяФормаВыбораПользователейСистемыВзаимодействия](ОбъектМетаданныхКонфигурация.ОсновнаяФормаВыбораПользователейСистемыВзаимодействия_(ConfigurationMetadataObject.DefaultCollaborationSystemUsersChoiceForm).md) содержит пустое значение, то форма выбора не используется (в выпадающем списке подбора пользователей гиперссылка "Показать все" не отображается).

Описание:

Возникает на клиенте при автоподборе пользователя системы взаимодействия (получателя сообщения, участника видеозвонка).  
Обработчик вызывается на клиенте после каждого вызова обработчика [АвтоПодборПользователейСистемыВзаимодействия](ФормаКлиентскогоПриложения.АвтоПодборПользователейСистемыВзаимодействия_(ClientApplicationForm.CollaborationSystemUsersAutoComplete).md).  
Данный обработчик не вызывается при добавлении участников в существующую видеоконференцию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
