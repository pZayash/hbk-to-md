# ПолеФормы.ПредупреждениеПриРедактировании

ПолеФормы (FormField)

ПредупреждениеПриРедактировании (WarningOnEdit)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, выдаваемую в качестве предупреждения при начале редактирования в случае, если [ОтображениеПредупрежденияПриРедактировании](ПолеФормы.ОтображениеПредупрежденияПриРедактировании_(FormField.WarningOnEditRepresentation).md) установлено в значение [Отображать](ОтображениеПредупрежденияПриРедактировании.Отображать_(WarningOnEditRepresentation.Show).md). Текст предупреждения дополняется строкой ". <Перевод строки>Продолжить редактирование?".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
