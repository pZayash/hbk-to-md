Глобальный контекст (Global context)

ПрочитатьЗначениеJSON (ReadJSONValue)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПрочитатьЗначениеJSON(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
JSON-строка, которую требуется десериализовать.   
Если значение содержит строку, не являющуюся валидной строкой JSON, будет сгенерировано исключение.

Возвращаемое значение:

Тип: [Структура](Структура_(Structure).md), [Массив](Массив_(Array).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Преобразует JSON-строку в значение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
