ЧтениеFastInfoset (FastInfosetReader)

Закрыть (Close)

Доступен, начиная с версии 8.1.

Синтаксис:

Закрыть()

Описание:

Прекращает чтение данных Fast Infoset из файла. Если никакого чтения в момент вызова данного метода не выполнялось, то метод ничего не делает.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
