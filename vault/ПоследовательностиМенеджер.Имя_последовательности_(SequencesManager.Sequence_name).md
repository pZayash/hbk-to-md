ПоследовательностиМенеджер (SequencesManager)

<Имя последовательности> (<Sequence name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПоследовательностьМенеджер.\<Имя последовательности\>](ПоследовательностьМенеджер.Имя_последовательности_(SequenceManager.Sequence_name).md).   
Набор свойств содержит менеджеры отдельных последовательностей. Доступ к менеджеру осуществляется по имени, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
