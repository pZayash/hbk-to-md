Сеанс (ISessionInfo)

cpuTimeAll (cpuTimeAll)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит время, которое затрачено процессором на обработку серверных вызовов с момента начала сеанса, в миллисекундах.

Доступность:

Интеграция.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.13.
