Описание:

Дополнительные свойства, методы и стандартные параметры формы объекта [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md), применимые для формы любого объекта, кроме документа, бизнес-процесса, задачи, справочника, плана видов характеристик.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
