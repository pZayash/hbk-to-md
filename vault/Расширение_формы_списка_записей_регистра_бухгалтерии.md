Описание:

Дополнительные свойства объекта [Форма](Форма.md) в том случае, если основной реквизит формы имеет тип [РегистрБухгалтерииСписок.](РегистрБухгалтерииСписок.Имя_регистра_бухгалтерии.md)<[Имя регистра бухгалтерии](РегистрБухгалтерииСписок.Имя_регистра_бухгалтерии.md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Приходные = Документы.ПриходнаяНакладная; Док = Приходные.НайтиПоНомеру("00003",ТекущаяДата()); Остатки = РегистрыНакопления.УчетНоменклатуры; Форма = Остатки.ПолучитьФормуСписка(); Форма.ПараметрОтборПоРегистратору = Док; Форма.Открыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
