АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКВнешнимМодулям (ExternalModuleFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет разрешение на доступ к внешним модулям:

- Истина - разрешен доступ к любым внешним модулям,
- Ложь - разрешен доступ только к модулям, перечисленным в списке разрешенных модулей (список разрешенных модулей можно получить с помощью метода [ПолучитьРазрешенныеВнешниеМодули](АдминистрированиеПрофильБезопасности.ПолучитьРазрешенныеВнешниеМодули.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
