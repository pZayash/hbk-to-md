РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак выделения текста, первый значимый символ которого "-", специальным цветом стиля [ЦветОтрицательногоЧисла](ЦветаСтиля.ЦветОтрицательногоЧисла.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - текст выделяется.  
Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), [Примечание](ТипРисункаТабличногоДокумента.Примечание.md).

Использование в версии:

Доступен, начиная с версии 8.0.
