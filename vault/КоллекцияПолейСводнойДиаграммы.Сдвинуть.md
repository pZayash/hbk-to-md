КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Поле>, <Смещение>)

Параметры:

<Поле> (обязательный)

Тип: [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы.md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы.md).   
Перемещаемое поле.

<Смещение> (обязательный)

Тип: Число.   
Число полей, на которое необходимо сдвинуть данное поле.

Описание:

Сдвигает поле сводной диаграммы на указанное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
