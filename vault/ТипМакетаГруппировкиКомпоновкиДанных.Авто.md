ТипМакетаГруппировкиКомпоновкиДанных (DataCompositionGroupTemplateType)

Авто (Auto)

Доступен, начиная с версии 8.1.

Описание:

Автоматическое определение расположения выбранных полей:  
Если в группировке есть вложенная таблица, диаграмма, вложенный отчет или группировка с типом макета группировки [Вертикальный](ТипМакетаГруппировкиКомпоновкиДанных.Вертикальный.md), то выбранные поля располагаются вертикально, иначе горизонтально.

Использование в версии:

Доступен, начиная с версии 8.1.
