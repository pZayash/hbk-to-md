# Расширение элементов управления, расположенных в форме.ИзменяетДанные

Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

ИзменяетДанные (ModifiesData)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что данный элемент управления будет менять данные.   
[Истина](lang__def_BooleanTrue.md) - изменяет.  
Рекомендуется использовать, например, для кнопок, при нажатии которых выполняется заполнение объекта или его изменения.

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
