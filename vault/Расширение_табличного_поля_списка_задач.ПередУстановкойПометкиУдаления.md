Расширение табличного поля списка задач (Task list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то пометка удаления изменена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед установкой пометки удаления в списке задач.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ЗадачаОбъект.Имя_задачи.ПередЗаписью.md) и [ПриЗаписи](ЗадачаОбъект.Имя_задачи.ПриЗаписи.md)объекта [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
