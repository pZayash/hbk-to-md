КоллекцияВложенийСистемыВзаимодействия (CollaborationSystemAttachmentCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Вставить(<Индекс>, <Поток>, <Наименование>, <ТипСодержимого>, <Отображаемое>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс вставляемого значения.

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, из которого будет создано вложение системы взаимодействия.

<Наименование> (обязательный)

Тип: Строка.   
Наименование создаваемого вложения системы взаимодействия.

<ТипСодержимого> (необязательный)

Тип: Строка.   
Тип содержимого создаваемого вложения.  
Если параметр не задан, то выполняется попытка автоматически определить тип содержимого вложения. Если автоматически определить тип содержимого не удалось, для вложения используется тип "application/octet-stream".

<Отображаемое> (необязательный)

Тип: Булево.   
Устанавливает признак отображения вложения в интерфейсе.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ВложениеСистемыВзаимодействия](ВложениеСистемыВзаимодействия.md).   

Описание:

Вставляет элемент в коллекцию по указанному индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
