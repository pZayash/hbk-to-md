РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Запись>, <Смещение>)

Параметры:

<Запись> (обязательный)

Тип: [Число](lang__def_Number.md), [РегистрРасчетаЗапись.](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)>.   
Индекс сдвигаемой записи, либо сама запись.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца набора записей (увеличение индекса), отрицательное - к началу.

Описание:

Сдвигает запись в наборе на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
