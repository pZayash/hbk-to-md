# ТабличноеПоле.АвтоВводНовойСтроки

ТабличноеПоле (TableBox)

АвтоВводНовойСтроки (AutoInsertNewRow)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического ввода новой строки. В этом режиме новая строка вводится в табличное поле автоматически после окончания редактирования последнего элемента предыдущей строки.  
[Истина](lang__def_BooleanTrue.md) - новая строка вводится автоматически.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
