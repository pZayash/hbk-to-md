Расширение табличного поля списка записей регистра бухгалтерии (Accounting register record list table box extension)

ИзменятьАвтоОбновление (ChangeAutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено как [Ложь](lang__def_BooleanFalse.md), то пользователь не имеет доступа к интерактивному управлению параметрами автообновления, устанавливаемыми на закладке "Прочее" диалога настройки просмотра списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
