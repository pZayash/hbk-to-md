ОбъектМетаданных: Ресурс (MetadataObject: Resource)

ОбъектРасширяемойКонфигурации (ExtendedConfigurationObject)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Свойство используется для заимствованных объектов расширения.   
Если для расширения установлено свойство "Поддерживать соответствие объектам конфгурации", значение данного свойства заполняется при заимствовании объекта и содержит уникальный идентификатор объекта в расширяемой конфигурации (отображается как имя объекта расширяемой конфигурации или <Объект не найден>).  
Если после переименования заимствованного объекта ему стал соответствовать другой объект конфигурации и для расширения установлено свойство "Поддерживать соответствие объектам расширяемой конфигурации при переименовании", то значение свойства будет обновлено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Непосредственное редактирование значения свойства не поддерживается.

Использование в версии:

Доступен, начиная с версии 8.3.15.
