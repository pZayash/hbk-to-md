УникальныйИдентификатор (UUID)

С указанием версии

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый УникальныйИдентификатор(<Версия>, <ПространствоИмен>, <Имя>)

Параметры:

<Версия> (обязательный)

Тип: [ВерсияУникальногоИдентификатора](ВерсияУникальногоИдентификатора.md).   
Задает версию генерируемого уникального идентификатора.

<ПространствоИмен> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Определяет пространство имен при генерации уникального идентификатора.  
Используется только для идентификаторов версии 3 и 5.

<Имя> (необязательный)

Тип: Строка.   
Задает имя при генерации уникального идентификатора.  
Используется только для идентификаторов версии 3 и 5.

Описание:

Создает уникальный идентификатор заданной версии.  
  
В режиме совместимости конфигурации с [Версия8\_3\_24](РежимСовместимости.Версия8_3_24.md) конструктор создает уникальные идентификаторы для версий 3 и 5, которые могут содержать ошибку (не полностью удовлетворять стандарту RFC4122), но алгоритм генерации – такой же, как в версии до 8.3.24 включительно.  
  
При выключенном режиме совместимости используется алгоритм с исправленной ошибкой.  
  
На алгоритм генерации версий 1 и 4 режим совместимости не влияет.

Использование в версии:

Доступен, начиная с версии 8.3.21.
