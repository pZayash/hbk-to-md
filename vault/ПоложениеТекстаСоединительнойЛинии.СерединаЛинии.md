ПоложениеТекстаСоединительнойЛинии (ConnectorTextLocation)

СерединаЛинии (Middle)

Доступен, начиная с версии 8.0.

Описание:

Текст размещается в районе середины линии (середина вычисляется как половина суммы длин всех сегментов линии).

Использование в версии:

Доступен, начиная с версии 8.0.
