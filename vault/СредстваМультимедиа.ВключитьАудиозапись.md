СредстваМультимедиа (MultimediaTools)

ВключитьАудиозапись (StartAudioRecording)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ВключитьАудиозапись(<ОбработчикОстановкиАудиозаписи>, <ПараметрыАудиозаписи>, <РасположениеКнопки>)

Параметры:

<ОбработчикОстановкиАудиозаписи> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана при остановке записи звука со следующими параметрами:

- <РезультатАудиозаписи> - тип [ДанныеМультимедиа](ДанныеМультимедиа.md) или Неопределено.
- Неопределено - запись не выполнена.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ПараметрыАудиозаписи> (необязательный)

Тип: [ПараметрыАудиозаписи](ПараметрыАудиозаписи.md).   
Набор параметров, используемых при записи аудио.

<РасположениеКнопки> (необязательный)

Тип: [РасположениеКнопкиОстановкиЗаписиМультимедиа](РасположениеКнопкиОстановкиЗаписиМультимедиа.md), Неопределено.   
Устанавливает расположение кнопки для остановки воспроизведения. Кнопка мигает, чтобы напоминать пользователю о процессе записи. Кнопку можно перемещать по экрану в удобное для работы место.  
Если параметр не указан или имеет значение Неопределено, то кнопка будет размещена там, куда ее переместил пользователь, или в правом нижнем углу, если пользователь ее не двигал.  
Значение по умолчанию: Неопределено.

Описание:

Запускает запись аудио встроенным микрофоном в фоновом для пользователя режиме. Остановить можно с помощью метода [ОтключитьАудиозапись](СредстваМультимедиа.ОтключитьАудиозапись.md), или если пользователь нажал кнопку остановки записи на экране устройства.   
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяАудиозапись](СредстваМультимедиа.ПоддерживаетсяАудиозапись.md). Если запись звука не поддерживается, то будет вызвано исключение: "Ошибка работы с микрофоном."  
Если запись уже идет, то будет вызвано исключение: "Микрофон занят".  
Если переданные параметры не поддерживаются устройством, то будет вызвано исключение:  
"Устройство не поддерживает переданные параметры записи звука."  
Если в процессе записи приложение было свернуто, то аудиозапись приостанавливается и пользователю показывается локальное уведомление: "Запись звука приостановлена."

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
