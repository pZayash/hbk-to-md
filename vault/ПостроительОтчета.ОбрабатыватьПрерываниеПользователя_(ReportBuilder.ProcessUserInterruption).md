ПостроительОтчета (ReportBuilder)

ОбрабатыватьПрерываниеПользователя (ProcessUserInterruption)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак разрешения прерывания пользователем вывода отчета нажатием комбинации клавиш Ctrl + Break. При прерывании вывода отчета сгенерируется исключение встроенного языка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию [Истина](lang__def_BooleanTrue.md) (прерывание разрешено).

Использование в версии:

Доступен, начиная с версии 8.0.
