КоллекцияИнтерваловФонаПланировщика (PlannerBackgroundIntervalCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Вставить(<Индекс>, <Начало>, <Конец>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить интервал фона.

<Начало> (обязательный)

Тип: Дата.   
Дата начала интервала фона.

<Конец> (обязательный)

Тип: Дата.   
Дата конца интервала фона.

Возвращаемое значение:

Тип: [ИнтервалФонаПланировщика](ИнтервалФонаПланировщика.md).   

Описание:

Вставляет интервал фона в коллекцию по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
