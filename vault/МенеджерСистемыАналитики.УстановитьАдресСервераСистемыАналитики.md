МенеджерСистемыАналитики (AnalyticsSystemManager)

УстановитьАдресСервераСистемыАналитики (SetAnalyticsSystemServerAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьАдресСервераСистемыАналитики(<АдресСервера>)

Параметры:

<АдресСервера> (необязательный)

Тип: Строка.   
Адрес сервера системы аналитики.

Описание:

Установливает адрес соединения с системой аналитики, настроенного на эту область данных (если в информационной базе не определены разделители – на всю базу). Действие доступно только для пользователя с правом "Администрирование".

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
