Доступен, начиная с версии 8.3.26.

Элементы коллекции:

[ДополнительныйИндекс](ДополнительныйИндекс.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются дополнительные индексы.  
Возможно обращение к дополнительному индексу посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Количество (Count)](ДополнительныеИндексы.Количество.md)  
[Получить (Get)](ДополнительныеИндексы.Получить.md)  

Описание:

Представляет собой коллекцию дополнительных индексов объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
