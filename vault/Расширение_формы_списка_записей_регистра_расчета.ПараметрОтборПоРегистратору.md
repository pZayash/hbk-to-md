Расширение формы списка записей регистра расчета (Calculation register record list form extension)

ПараметрОтборПоРегистратору (FilterByRecorderParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа.md).   
Устанавливает начальное значение отбора по регистратору при открытии или повторном открытии формы списка регистра расчета.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
