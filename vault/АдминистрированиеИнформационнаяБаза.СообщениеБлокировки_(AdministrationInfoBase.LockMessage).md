АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

СообщениеБлокировки (LockMessage)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, который будет частью сообщения об ошибке при попытке установки соединения с заблокированной информационной базой.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
