СекцияCDATADOM (DOMCDATASection)

ЗначениеУзла (NodeValue)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит значение узла DOM. Значение узла присутствует не для каждого типа узла DOM.  
Узел Значение узла

- [Атрибут](ТипУзлаDOM.Атрибут.md) - [Значение](АтрибутDOM.Значение.md);
- [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md) - [СекцияCDATADOM.Данные](СекцияCDATADOM.Данные.md), содержит CDATA секцию;
- [Комментарий](ТипУзлаDOM.Комментарий.md) - [СекцияCDATADOM.Данные](СекцияCDATADOM.Данные.md), содержит комментарий;
- [Документ](ТипУзлаDOM.Документ.md) - Неопределено;
- [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - Неопределено;
- [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) - Неопределено;
- [Элемент](ТипУзлаDOM.Элемент.md) - Неопределено;
- [Сущность](ТипУзлаDOM.Сущность.md) - Неопределено;
- [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) - Неопределено;
- [Нотация](ТипУзлаDOM.Нотация.md) - Неопределено;
- [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md) - [Данные](ИнструкцияОбработкиDOM.Данные.md);
- [Текст](ТипУзлаDOM.Текст.md) - [СекцияCDATADOM.Данные](СекцияCDATADOM.Данные.md), содержит текст узла.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
