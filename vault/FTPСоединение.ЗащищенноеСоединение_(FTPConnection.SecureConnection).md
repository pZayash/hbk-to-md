FTPСоединение (FTPConnection)

ЗащищенноеСоединение (SecureConnection)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md), [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
