ПринадлежностьОбъекта (ObjectBelonging)

Заимствованный (Adopted)

Доступен, начиная с версии 8.3.6.

Описание:

Объект заимствован из основной конфигурации. Такие объекты не доступны во время запуска конфигурации в режиме "1С:Предприятие". Они либо расширяют существующие объекты расширяемой конфигурации, либо только используют ссылки на них.

Использование в версии:

Доступен, начиная с версии 8.3.6.
