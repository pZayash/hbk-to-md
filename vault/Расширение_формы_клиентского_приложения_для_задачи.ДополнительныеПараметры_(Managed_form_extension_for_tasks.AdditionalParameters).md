Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

ДополнительныеПараметры (AdditionalParameters)

Доступен, начиная с версии 8.3.4.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Дополнительные параметры формы, открываемой при нажатии кнопки "Создать".  
На мобильной платформе параметр не используется.

Использование в версии:

Доступен, начиная с версии 8.3.4.
