РасширениеКонфигурации (ConfigurationExtension)

ПроверитьВозможностьПрименения (CheckCanApply)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПроверитьВозможностьПрименения(<Данные>, <ПроверятьСледующие>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Проверка применимости будет учитывать, что текущее расширение конфигурации обновлено предоставленными двоичными данными.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ПроверятьСледующие> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если содержит значение [Истина](lang__def_BooleanTrue.md), будут проверены все расширения, последующие за данным в порядке загрузки расширения конфигурации.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Возвращает массив объектов типа [ИнформацияОПроблемеПримененияРасширенияКонфигурации](ИнформацияОПроблемеПримененияРасширенияКонфигурации_(ConfigurationExtensionApplicationIssueInformation).md)

Описание:

Проверяет возможность применения данного расширения конфигурации. При проверке учитываются расширения, загруженные ранее.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
