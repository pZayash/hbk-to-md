РегистрРасчетаКлючЗаписи.<Имя регистра расчета> (CalculationRegisterRecordKey.<Calculation register name>)

НомерСтроки (LineNumber)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета.Регистратор_(CalculationRegisterRecordKey.Calculation_register_name.Recorder).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
