Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md), предназначенные для интерактивной работы с записями регистра сведений. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [РегистрСведенийСписок.](РегистрСведенийСписок.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийСписок.Имя_регистра_сведений.md)>.

Доступность:

Толстый клиент.

Доступен, начиная с версии 8.0.
