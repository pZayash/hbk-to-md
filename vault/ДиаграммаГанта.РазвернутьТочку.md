ДиаграммаГанта (GanttChart)

РазвернутьТочку (ExpandPoint)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутьТочку(<Точка>, <СПодчиненными>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта.md).   
Точка диаграммы Ганта.

<СПодчиненными> (обязательный)

Тип: Булево.   
Определяет необходимость раскрытия точки вместе с подчиненными точками. Истина - раскрывать вместе с подчиненными.

Описание:

Раскрывает точку, возможно вместе с подчинёнными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
