ДанныеКонтакта (ContactData)

АдресаМгновенныхСообщений (InstantMessagingAddresses)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив адресов мгновенных сообщений в виде объектов типа [ЭлементДанныхКонтактаМгновенныеСообщения](ЭлементДанныхКонтактаМгновенныеСообщения_(ContactDataItemInstantMessaging).md).   
Свойство [ТипДанных](ЭлементДанныхКонтактаМгновенныеСообщения.ТипДанных_(ContactDataItemInstantMessaging.DataType).md) содержит значение перечисления [ТипАдресаМгновенныхСообщенийДанныхКонтакта](ТипАдресаМгновенныхСообщенийДанныхКонтакта_(ContactDataInstantMessagingAddressType).md) или строку. Свойство [ТипМгновенныхСообщений](ЭлементДанныхКонтактаМгновенныеСообщения.ТипМгновенныхСообщений_(ContactDataItemInstantMessaging.InstantMessagingType).md) содержит строку (например, Skype, ICQ и т.д.). Свойство [Идентификатор](ЭлементДанныхКонтактаМгновенныеСообщения.Идентификатор_(ContactDataItemInstantMessaging.ID).md) содержит строку – идентификатор пользователя в системе мгновенных сообщений.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
