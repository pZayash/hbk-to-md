ВыраженияСхемыЗапроса (QuerySchemaExpressions)

Удалить (Delete)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого элемента.

Описание:

Удаляет элемент из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
