СхемаСистемыАналитики (AnalyticsSystemSchema)

УстановитьИзСтроки (SetFromString)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьИзСтроки(<СхемаКакСтрока>)

Параметры:

<СхемаКакСтрока> (обязательный)

Тип: [Строка](lang__def_String.md).   
Содержит схему системы аналитики в виде строки.

Описание:

Устанавливает схему системы аналитики по переданной строке.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
