ЭлементУсловногоОформленияКомпоновкиДанных (DataCompositionConditionalAppearanceItem)

ПредставлениеПользовательскойНастройки (UserSettingPresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Используется для отображения объекта в пользовательских настройках.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
