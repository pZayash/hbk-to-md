# ПланСчетовОбъект.<Имя плана счетов>.УстановитьПометкуУдаления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Методы](ПланСчетовОбъект.Имя_плана_счетов__Методы.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>, <ВключаяПодчиненные>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак установки / снятие пометки на удаление.  
[Истина](lang__def_BooleanTrue.md) - пометка будет установлена; [Ложь](lang__def_BooleanFalse.md) - пометка будет снята.

<ВключаяПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр установлен в [Истина](lang__def_BooleanTrue.md), то будут помечены на удаление, кроме данного элемента, все его подчиненные элементы в данном счете и во всех подчиненных счетах.  
Если параметр установлен в [Ложь](lang__def_BooleanFalse.md), то будет помечен на удаление только данный элемент.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает/снимет пометку на удаление. Объект при этом не удаляется из базы данных. Окончательное удаление помеченных счетов происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](ПланСчетовОбъект.Имя_плана_счетов.ПередЗаписью_(ChartOfAccountsObject.Chart_of_accounts_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](ПланСчетовОбъект.Имя_плана_счетов.ПометкаУдаления_(ChartOfAccountsObject.Chart_of_accounts_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Пример:

|  |
| --- |
| Объект.УстановитьПометкуУдаления(Истина,Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
