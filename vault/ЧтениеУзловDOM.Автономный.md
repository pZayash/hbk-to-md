ЧтениеУзловDOM (DOMNodeReader)

Автономный (Standalone)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит значение атрибута standalone элемента объявления XML.   
Если атрибут не указан или объявление не прочитано (отсутствует), то содержит Ложь.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
