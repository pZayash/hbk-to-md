# ОформлениеМакетаОформленияКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макет оформления](Макет_оформления.md)

Доступен, начиная с версии 8.1.

Описание:

Коллекция элементов оформления макета оформления компоновки данных. Для оформления используются следующие элементы оформления:  
1. ЦветФона (BackColor) - тип значения [Цвет](Цвет_(Color).md);  
2. ЦветТекста (TextColor) - тип значения [Цвет](Цвет_(Color).md);  
3. ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет_(Color).md);  
4. СтильГраницы (BorderStyle) - тип значения [Линия](Линия_(Line).md);  
5. СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия_(Line).md);  
6. СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия_(Line).md);  
7. СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия_(Line).md);  
8. СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия_(Line).md);  
9. Шрифт (Font) - тип значения [Шрифт](Шрифт_(Font).md);  
10. Отступ (Indent) - тип значения [Число](lang__def_Number.md);  
11. АвтоОтступ (AutoIndent) - тип значения [Число](lang__def_Number.md);  
12. ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md);  
13. ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md);  
14. Размещение (Placement) - размещение текста; тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных_(DataCompositionTextPlacementType).md);  
15. ОриентацияТекста (TextOrientation) - тип значения [Число](lang__def_Number.md);  
16. Формат (Format) - тип значения [Строка](lang__def_String.md) (форматная строка);  
17. ВыделятьОтрицательные (MarkNegatives) - тип значения [Булево](lang__Булево_(Boolean).md);  
18. МинимальнаяШирина (MinWidth) - тип значения [Число](lang__def_Number.md);  
19. МаксимальнаяШирина (MaxWidth) - тип значения [Число](lang__def_Number.md);  
20. МинимальнаяВысота (MinHeight) - тип значения [Число](lang__def_Number.md);  
21. МаксимальнаяВысота (MaxHeight) - тип значения [Число](lang__def_Number.md);  
22. РежимИзмененияРазмераКолонки (ColumnSizeChangeMode) - возможность изменять размер колонки ячейки [РежимИзмененияРазмера](РежимИзмененияРазмера_(SizeChangeMode).md).  
23. ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную; тип значения [Булево](lang__Булево_(Boolean).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: Appearance.
