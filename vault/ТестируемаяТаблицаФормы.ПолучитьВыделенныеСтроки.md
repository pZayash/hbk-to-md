Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив.md).   

Описание:

Возвращает массив соответствий с описаниями выделенных строк. Ключ соответствия содержит заголовок колонки, значение - текст ячейки.

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
