ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageConnectionParameters)

Каталог (Directory)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Подкаталог внутри контейнера (корзины, bucket). Полный путь до объекта будет иметь вид: "<URL-адрес подключения к хранилищу>/<Каталог>".

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.27.
