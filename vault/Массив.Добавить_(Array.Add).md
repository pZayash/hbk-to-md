Массив (Array)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Добавляемое значение. Если не указан, то будет добавлено значение типа [Неопределено](lang__def_Undefined.md).

Описание:

Добавляет элемент в конец массива.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При добавлении количество элементов массива увеличивается на 1.

Пример:

|  |
| --- |
| Массив.Добавить("Первый"); Массив.Добавить("Второй"); |

Использование в версии:

Доступен, начиная с версии 8.0.
