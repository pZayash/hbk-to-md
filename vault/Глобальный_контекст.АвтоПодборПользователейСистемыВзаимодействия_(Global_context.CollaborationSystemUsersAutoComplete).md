Глобальный контекст (Global context)

АвтоПодборПользователейСистемыВзаимодействия (CollaborationSystemUsersAutoComplete)

Доступен, начиная с версии 8.3.15.

Синтаксис:

АвтоПодборПользователейСистемыВзаимодействия(<Текст>, <ДанныеВыбора>, <НазначениеВыбора>, <Форма>, <Обсуждение>)

Параметры:

<Текст>

Тип: [Строка](lang__def_String.md).   
Строка текста, введенная в поле ввода пользователя системы взамиодействия.

<ДанныеВыбора>

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Содержит список подобранных значений.  
В качестве значения элемента списка могут использоваться значения следующих типов:

- [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md),
- [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md) - идентификатор пользователя информационной базы,
- Массив - должен содержать значения только следующих типов:
  - [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md),
  - [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md) - идентификатор пользователя информационной базы,
  - [Неопределено](lang__def_Undefined.md) - допустимо только, если [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md) имеет значение [ПолучательСообщения](НазначениеВыбораПользователейСистемыВзаимодействия.ПолучательСообщения_(CollaborationSystemUsersChoicePurpose.MessageRecipient).md).

<НазначениеВыбора>

Тип: [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md).   
Назначение выбора пользователей системы взаимодействия.

<Форма>

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [Неопределено](lang__def_Undefined.md).   
Форма, в которой находится контекстное обсуждение, если выполняется автоподбор получателей сообщения.

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md), [Неопределено](lang__def_Undefined.md).   
Обсуждение, в котором выполняется подбор пользователей.

Описание:

Возникает при автоподборе пользователя системы взаимодействия (получателя сообщения, участника видеозвонка).  
При подборе пользователей обработчик вызывается в том случае, если у формы не задан обработчик события [АвтоПодборПользователейСистемыВзаимодействия](ФормаКлиентскогоПриложения.АвтоПодборПользователейСистемыВзаимодействия_(ClientApplicationForm.CollaborationSystemUsersAutoComplete).md), или в параметре <СтандартнаяОбработка> возвращено значение [Истина](lang__def_BooleanTrue.md).  
При вызове в параметр <ДанныеВыбора> передается сформированный системой список выбора.  
Если для параметра <ДанныеВыбора> в качестве значения элемента списка содержится массив, то он задает группы пользователей для выбора.  
Значение [Неопределено](lang__def_Undefined.md) допустимо только, если параметр <НазначениеВыбора> имеет значение [ПолучательСообщения](НазначениеВыбораПользователейСистемыВзаимодействия.ПолучательСообщения_(CollaborationSystemUsersChoicePurpose.MessageRecipient).md) и интерпретируется как "Никого не оповещать". В остальных случаях значение [Неопределено](lang__def_Undefined.md) игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
