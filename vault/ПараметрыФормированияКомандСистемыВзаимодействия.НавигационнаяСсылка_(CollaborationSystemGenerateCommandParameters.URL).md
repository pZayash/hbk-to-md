ПараметрыФормированияКомандСистемыВзаимодействия (CollaborationSystemGenerateCommandParameters)

НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Указывает навигационную ссылку, если источником команд является навигационная ссылка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
