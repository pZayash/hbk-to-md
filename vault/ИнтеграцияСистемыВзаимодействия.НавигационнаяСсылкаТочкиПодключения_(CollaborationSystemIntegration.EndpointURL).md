ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

НавигационнаяСсылкаТочкиПодключения (EndpointURL)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит адрес точки подключения для интеграции типа Webhook.  
Для остальных видов интеграции содержит пустую строку.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
