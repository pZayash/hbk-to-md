`УстановитьОграничениеПовторенияПаролейПользователейСредиПоследних`(`<ОграничениеПовторенияПаролейСредиПоследних>`: `Число`)

Глобальный контекст (Global context)

УстановитьОграничениеПовторенияПаролейПользователейСредиПоследних (SetUserPasswordReuseLimit)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьОграничениеПовторенияПаролейПользователейСредиПоследних(<ОграничениеПовторенияПаролейСредиПоследних>)

Параметры:

<ОграничениеПовторенияПаролейСредиПоследних> (необязательный)

Тип: Число.   
Численное значение ограничения повторения паролей среди последних.  
Значение по умолчанию: 0.

Описание:

Устанавливает численное значение ограничения повторения паролей пользователей среди последних.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
