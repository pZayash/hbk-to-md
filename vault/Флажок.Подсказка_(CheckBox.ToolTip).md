Флажок (CheckBox)

Подсказка (ToolTip)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст всплывающей подсказки, которая появляется при наведении курсора.

Доступность:

Толстый клиент.

Примечание:

1. Если текст значения полностью не помещается в элементе управления и подсказка не задана, то показывается текст значения.  
2. Если текст значения помещается и подсказка задана, то показывать текст подсказки.  
3. Если текст значения не помещается и подсказка задана, то показывается текст значения, а также подсказка с новой строки.

Использование в версии:

Доступен, начиная с версии 8.0.
