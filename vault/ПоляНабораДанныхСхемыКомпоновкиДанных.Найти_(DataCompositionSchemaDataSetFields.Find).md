ПоляНабораДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetFields)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<ПутьКДанным>)

Параметры:

<ПутьКДанным> (необязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным поля.

Возвращаемое значение:

Тип: [ПолеНабораДанныхСхемыКомпоновкиДанных](ПолеНабораДанныхСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetField).md), [ПапкаПолейНабораДанныхСхемыКомпоновкиДанных](ПапкаПолейНабораДанныхСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetFieldFolder).md), [Неопределено](lang__def_Undefined.md).   
Если элемент не найден, будет возвращено значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента коллекции по пути к данным.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
