Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

ТекстЗаполнения (FillingText)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: Строка.   
Текст, введенный в поле ввода, при открытии формы в режиме выбора.  
Если основное представление [ВВидеНаименования](ОсновноеПредставлениеЗадачи.ВВидеНаименования.md) и реквизит Наименование находится в списке [ВводПоСтроке](ОбъектМетаданных_Задача.ВводПоСтроке.md), то значение присваивается реквизиту Наименование.  
Если основное представление ввиденомера и реквизит Номер находится в списке [ВводПоСтроке](ОбъектМетаданных_Задача.ВводПоСтроке.md) и не установлен признак [Автонумерация](ОбъектМетаданных_Задача.Автонумерация.md), то значение присваивается реквизиту Номер.  
В противном случае присваивание не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.3.
