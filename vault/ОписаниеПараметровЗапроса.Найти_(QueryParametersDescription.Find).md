ОписаниеПараметровЗапроса (QueryParametersDescription)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого параметра.

Возвращаемое значение:

Тип: [ОписаниеПараметраЗапроса](ОписаниеПараметраЗапроса_(QueryParameterDescription).md), [Неопределено](lang__def_Undefined.md).   
Если искомое значение не найдено, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск параметра запроса по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
