ОписаниеЗначенияПараметраГолосаСинтезаРечи (TextToSpeechVoiceParameterValueDescription)

Значение (Value)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Ключ значения для использования в методах генерации речи [НачатьСинтез](МенеджерРаботыСРечью.НачатьСинтез_(SpeechProcessingManager.StartTextToSpeech).md) и [Синтезировать](МенеджерРаботыСРечью.Синтезировать_(SpeechProcessingManager.Synthesize).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
