ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - при выводе измерения с отрицательным значением значение будет отображено особым образом.  
Имеет смысл только для измерений типа [Число](lang__def_Number.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
