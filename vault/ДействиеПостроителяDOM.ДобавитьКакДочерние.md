ДействиеПостроителяDOM (DOMBuilderAction)

ДобавитьКакДочерние (AppendAsChildren)

Доступен, начиная с версии 8.1.

Описание:

Добавить как дочерние узлы.  
Результат чтения добавляется в узел контекста документа в качестве дочерних узлов. В качестве узла контекста могут выступать узлы: [Элемент](ТипУзлаDOM.Элемент.md) или [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md).

Использование в версии:

Доступен, начиная с версии 8.1.
