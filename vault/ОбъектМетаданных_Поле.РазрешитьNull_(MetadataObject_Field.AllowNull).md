РазрешитьNull (AllowNull)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что поле может принимать значение [Null](lang__def_Null.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
