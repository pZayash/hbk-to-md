РазрешенныйИнтернетРесурс (AllowedInternetResource)

Протокол (Protocol)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит разрешенный протокол ресурса Интернет. Регистр букв не различается:

- IMAP – сервер электронной почты с протоколом IMAP;
- POP3 – сервер электронной почты с протоколом POP3;
- SMTP – сервер электронной почты с протоколом SMTP;
- HTTP – веб-сервер;
- HTTPS – защищенное соединение с веб-сервером;
- FTP – FTP-сервер;
- FTPS – защищенное соединение с ftp-сервером.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
