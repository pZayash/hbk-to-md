Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

ЗначениеКопирования (CopyingValue)

Доступен, начиная с версии 8.2.

Описание:

Значение для копирования.

Примечание:

Тип параметра соответствует типу объекта, копия которого будет создана.

Использование в версии:

Доступен, начиная с версии 8.2.
