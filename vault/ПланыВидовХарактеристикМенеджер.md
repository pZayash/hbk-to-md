Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПланВидовХарактеристикМенеджер.\<Имя плана видов характеристик\>](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов характеристик.  
Возможно обращение к менеджеру плана видов характеристик посредством оператора [...]. В качестве аргумента передается имя плана видов характеристик, как оно задано в конфигурации.  

Свойства:

[\<Имя плана видов характеристик\> (\<Chart of characteristic types name\>)](ПланыВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПланыВидовХарактеристикМенеджер.ТипВсеСсылки.md)  

Описание:

Предназначен для управления планами видов характеристик и предоставляет доступ к значениям типа [ПланВидовХарактеристикМенеджер.](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)>. Доступ к объекту осуществляется через свойство глобального контекста [ПланыВидовХарактеристик](Глобальный_контекст.ПланыВидовХарактеристик.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.
