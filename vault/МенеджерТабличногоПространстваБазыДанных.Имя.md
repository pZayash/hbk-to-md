МенеджерТабличногоПространстваБазыДанных (DatabaseTablespaceManager)

Имя (Name)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя табличного пространства. Используется во встроенном языке для идентификации табличного пространства. Имя пространства должно быть уникальным, может содержать только буквы латинского алфавита и цифры. Допускается длина не более 16 символов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
