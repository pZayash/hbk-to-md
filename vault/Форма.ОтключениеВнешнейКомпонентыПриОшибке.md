Форма (Form)

ОтключениеВнешнейКомпонентыПриОшибке (AddInDetachmentOnError)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ОтключениеВнешнейКомпонентыПриОшибке(<Местоположение>, <Имя>)

Параметры:

<Местоположение>

Тип: Строка.   
Строка, определяющая местоположение внешней компоненты.  
Соответствует первому параметру метода [ПодключитьВнешнююКомпоненту](Глобальный_контекст.ПодключитьВнешнююКомпоненту.md).

<Имя>

Тип: Строка.   
Символическое имя отключаемой внешней компоненты.  
Соответствует второму параметру метода [ПодключитьВнешнююКомпоненту](Глобальный_контекст.ПодключитьВнешнююКомпоненту.md).

Описание:

Вызывается при падении процесса внешней компоненты.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
