ДоставляемоеУведомление (DeliverableNotification)

Данные (Data)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит дополнительные данные уведомления. С помощью данного свойства отправитель может передать получателю короткую команду. Например, запрос сеанса связи. В уведомлении не отображается.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - "" (пустая строка).

Использование в версии:

Доступен, начиная с версии 8.3.6.
