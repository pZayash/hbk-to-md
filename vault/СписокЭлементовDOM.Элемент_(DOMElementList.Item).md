СписокЭлементовDOM (DOMElementList)

Элемент (Item)

Доступен, начиная с версии 8.1.

Синтаксис:

Элемент(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента DOM в списке.

Возвращаемое значение:

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md).   

Описание:

Получает элемент DOM по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
