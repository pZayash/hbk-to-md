СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: [Поток](Поток.md).   
Возвращает поток со следующими свойствами в зависимости от состояния сообщения:

- При создании сообщения - [ДоступнаЗапись](Поток.ДоступнаЗапись.md) имеет значение Истина, [ДоступноЧтение](Поток.ДоступноЧтение.md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции.md) имеют значение Ложь.
- При отправке и для отправленного сообщения - [ДоступноЧтение](Поток.ДоступноЧтение.md) имеет значение Истина, [ДоступнаЗапись](Поток.ДоступнаЗапись.md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции.md) имеют значение Ложь.
- Для полученного сообщения - [ДоступноЧтение](Поток.ДоступноЧтение.md) имеет значение Истина, [ДоступнаЗапись](Поток.ДоступнаЗапись.md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции.md) имеют значение Ложь.

Описание:

Получить тело сообщения как поток, доступный для чтения и записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
