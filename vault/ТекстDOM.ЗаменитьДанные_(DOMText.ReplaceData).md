# ТекстDOM.ЗаменитьДанные

ТекстDOM (DOMText)

ЗаменитьДанные (ReplaceData)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаменитьДанные(<Смещение>, <Длина>, <Строка>)

Параметры:

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение от начала символьных данных.

<Длина> (обязательный)

Тип: [Число](lang__def_Number.md).   
Длина замещаемых символьных данных.

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, замещающая символьные данные.

Описание:

Заменяет диапазон символьных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных;
- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
