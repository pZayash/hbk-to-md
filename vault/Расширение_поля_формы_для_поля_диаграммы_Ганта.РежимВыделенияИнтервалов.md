Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

РежимВыделенияИнтервалов (IntervalsSelectionMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [РежимВыделенияИнтерваловДиаграммыГанта](РежимВыделенияИнтерваловДиаграммыГанта.md).   
Определяет режим выделения интервалов в диаграмме. Значение по умолчанию - [Авто](РежимВыделенияИнтерваловДиаграммыГанта.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
