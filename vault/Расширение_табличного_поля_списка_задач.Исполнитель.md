Расширение табличного поля списка задач (Task list table box extension)

Исполнитель (Performer)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Ссылка на объект информационной базы.   
Параметр содержит ссылку на исполнителя.

Доступность:

Толстый клиент.

Примечание:

Используется при установленном свойстве [ОтображениеЗадач](Расширение_табличного_поля_списка_задач.ОтображениеЗадач.md) = [ПоИсполнителю](РежимСпискаЗадач.ПоИсполнителю.md).

Использование в версии:

Доступен, начиная с версии 8.0.
