ФормаКлиентскогоПриложения (ClientApplicationForm)

ГоризонтальноеПоложениеПодчиненных (ChildItemsHorizontalAlign)

Доступен, начиная с версии 8.3.7.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента_(ItemHorizontalLocation).md).   
Устанавливает горизонтальное выравнивание подчиненных элементов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.7.
