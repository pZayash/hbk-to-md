Расширение формы клиентского приложения для документа (Client application form extension for documents)

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: [ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md).   
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
