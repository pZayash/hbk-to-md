ТаблицаФормы (FormTable)

ГоризонтальноеПоложениеВГруппе (HorizontalAlignInGroup)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента_(ItemHorizontalLocation).md).   
Устанавливает горизонтальное выравнивание элемента внутри группы. Значение [Авто](ГоризонтальноеПоложениеЭлемента.Авто_(ItemHorizontalLocation.Auto).md) означает, что будет использоваться значение свойства [ГоризонтальноеПоложениеВГруппе](ГруппаФормы.ГоризонтальноеПоложениеВГруппе_(FormGroup.HorizontalAlignInGroup).md) родительской группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
