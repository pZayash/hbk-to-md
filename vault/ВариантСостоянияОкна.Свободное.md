ВариантСостоянияОкна (WindowStateVariant)

Свободное (Float)

Доступен, начиная с версии 8.0.

Описание:

Свободное расположение: окно может свободно перемещаться по всему экрану (не только внутри рабочей области программы).

Использование в версии:

Доступен, начиная с версии 8.0.
