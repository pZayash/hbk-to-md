Диаграмма (Chart)

ПробелыВоронки (FunnelSpace)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет относительное суммарное пространство зазоров между сегментами для диаграммы вида [ВоронкаНормированная](ТипДиаграммы.ВоронкаНормированная_(ChartType.NormalizedFunnel).md) в %% от общей высоты. Целое число от 0 до 100.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
