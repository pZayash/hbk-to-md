НастройкаПериода (PeriodSettings)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый НастройкаПериода()

Описание:

Создает новую настройку периода. По умолчанию у созданного объекта свойства определены следующим образом:

- [РедактироватьКакИнтервал](НастройкаПериода.РедактироватьКакИнтервал_(PeriodSettings.EditAsInterval).md) - [Истина](lang__def_BooleanTrue.md);
- [РедактироватьКакПериод](НастройкаПериода.РедактироватьКакПериод_(PeriodSettings.EditAsPeriod).md) - [Истина](lang__def_BooleanTrue.md);
- [ВариантНачала](НастройкаПериода.ВариантНачала_(PeriodSettings.BeginningVariant).md)- [БезОграничения](ВариантГраницыИнтервала.БезОграничения_(IntervalBoundVariant.WithoutRestriction).md);
- [ВариантОкончания](НастройкаПериода.ВариантОкончания_(PeriodSettings.EndingVariant).md) - [БезОграничения](ВариантГраницыИнтервала.БезОграничения_(IntervalBoundVariant.WithoutRestriction).md);
- [ВариантНастройки](НастройкаПериода.ВариантНастройки_(PeriodSettings.SettingsMode).md) - [Интервал](ВариантНастройкиПериода.Интервал_(PeriodSettingsVariant.Interval).md).

Также может быть создан с использованием функциональной формы оператора Новый.

Пример:

|  |
| --- |
| Период = Новый (Тип("НастройкаПериода")); Период.ВариантНастройки = ВариантНастройкиПериода.Интервал; Период.УстановитьПериод(НачалоДня(ДатаНач), КонецДня(ДатаКон))); |

Использование в версии:

Доступен, начиная с версии 8.0.
