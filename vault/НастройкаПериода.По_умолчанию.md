НастройкаПериода (PeriodSettings)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый НастройкаПериода()

Описание:

Создает новую настройку периода. По умолчанию у созданного объекта свойства определены следующим образом:

- [РедактироватьКакИнтервал](НастройкаПериода.РедактироватьКакИнтервал.md) - Истина;
- [РедактироватьКакПериод](НастройкаПериода.РедактироватьКакПериод.md) - Истина;
- [ВариантНачала](НастройкаПериода.ВариантНачала.md)- [БезОграничения](ВариантГраницыИнтервала.БезОграничения.md);
- [ВариантОкончания](НастройкаПериода.ВариантОкончания.md) - [БезОграничения](ВариантГраницыИнтервала.БезОграничения.md);
- [ВариантНастройки](НастройкаПериода.ВариантНастройки.md) - [Интервал](ВариантНастройкиПериода.Интервал.md).

Также может быть создан с использованием функциональной формы оператора Новый.

Пример:

|  |
| --- |
| Период = Новый (Тип("НастройкаПериода")); Период.ВариантНастройки = ВариантНастройкиПериода.Интервал; Период.УстановитьПериод(НачалоДня(ДатаНач), КонецДня(ДатаКон))); |

Использование в версии:

Доступен, начиная с версии 8.0.
