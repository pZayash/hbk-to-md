# Цвет.ПолучитьАбсолютный

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Оформление](Оформление.md) › [Цвет](Цвет_(Color).md) › [Методы](Цвет__Методы.md)

Цвет (Color)

ПолучитьАбсолютный (GetAbsolute)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьАбсолютный()

Возвращаемое значение:

Тип: [Цвет](Цвет_(Color).md).   

Описание:

Возвращает цвет, который определен абсолютным значением (сочетанием красного, синего и зеленого цветов).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
