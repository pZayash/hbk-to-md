РезультатРегистрацииИнформационнойБазыСистемыВзаимодействия (CollaborationSystemInfoBaseRegistrationResult)

ТекстСообщения (MessageText)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Если свойство [РегистрацияВыполнена](РезультатРегистрацииИнформационнойБазыСистемыВзаимодействия.РегистрацияВыполнена_(CollaborationSystemInfoBaseRegistrationResult.RegistrationCompleted).md) имеет значение [Ложь](lang__def_BooleanFalse.md), то в данном свойстве содержится текст инструкции для пользователя с дальнейшими действиями.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
