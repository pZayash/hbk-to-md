ПодписьКриптографии (CryptoSignature)

НеподтвержденноеВремяПодписи (UnverifiedSignatureTime)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата в формате UTC.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Значение данного свойства ничем не подтверждается, использование этой даты небезопасно.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.26.
