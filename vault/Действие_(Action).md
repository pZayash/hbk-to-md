# Действие

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Общие интерфейсные объекты](Общие_интерфейсные_объекты.md)

Доступен, начиная с версии 8.0.

Конструкторы:

[На основании процедуры](Действие.По_умолчанию.md)  

Описание:

Предназначен для получения действия, которое можно назначить в качестве обработчика события элемента управления или команды пользовательского интерфейса.   
Для обработки события может быть вызвана процедура или функция модуля формы или общего модуля, имя которой задано в конструкторе.  
При использовании в форме в качестве имени процедуры может использоваться только процедура модуля данной формы.

Доступность:

Толстый клиент.   
Сериализуется.

Пример:

|  |
| --- |
| // В модуле описана процедура Сформировать(), выполнение которой  // назначим кнопке "Сформировать" в качестве действия при нажатии  ОбработкаНажатия = Новый Действие("Сформировать"); КнопкаСформировать = ЭлементыФормы.КнопкаСформировать; КнопкаСформировать.УстановитьДействие("Нажатие", ОбработкаНажатия); |
