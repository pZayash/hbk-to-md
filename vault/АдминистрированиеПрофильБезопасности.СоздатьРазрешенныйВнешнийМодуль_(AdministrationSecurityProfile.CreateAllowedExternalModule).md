АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

СоздатьРазрешенныйВнешнийМодуль (CreateAllowedExternalModule)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьРазрешенныйВнешнийМодуль()

Возвращаемое значение:

Тип: [РазрешенныйВнешнийМодуль](РазрешенныйВнешнийМодуль_(AllowedExternalModule).md).   

Описание:

Создает новый внешний модуль для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
