ХарактеристикиКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionCharacteristics)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции, который требуется удалить.  
Индексация начинается с 0.

Описание:

Удаляет элемент из коллекции по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
