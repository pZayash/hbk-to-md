ОписаниеКомандыВходящегоЗапросаПоделиться (IncomingShareRequestCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: [СтандартнаяКомандаВходящегоЗапросаПоделиться](СтандартнаяКомандаВходящегоЗапросаПоделиться_(IncomingShareRequestStandardCommand).md), [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), [Неопределено](lang__def_Undefined.md).   
Содержит описание команды:

- [СтандартнаяКомандаВходящегоЗапросаПоделиться](СтандартнаяКомандаВходящегоЗапросаПоделиться_(IncomingShareRequestStandardCommand).md) - стандартная команда;
- [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md) - процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
  - <Данные> - тип [ДанныеЗапросаПоделиться](ДанныеЗапросаПоделиться_(ShareRequestData).md),
  - <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md);;
- [Неопределено](lang__def_Undefined.md) - разделитель;
- Любой другой тип - пункт меню не отображается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
