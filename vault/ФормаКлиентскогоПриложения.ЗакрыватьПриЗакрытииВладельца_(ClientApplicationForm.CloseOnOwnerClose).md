ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗакрыватьПриЗакрытииВладельца (CloseOnOwnerClose)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость закрытия формы при закрытии владельца.  
При создании формы значение данного параметра устанавливается в свойство [ЗакрыватьПриЗакрытииВладельца](ФормаКлиентскогоПриложения.ЗакрыватьПриЗакрытииВладельца_(ClientApplicationForm.CloseOnOwnerClose).md).  
[Истина](lang__def_BooleanTrue.md) - форма будет закрыта при закрытии владельца.

Использование в версии:

Доступен, начиная с версии 8.2.
