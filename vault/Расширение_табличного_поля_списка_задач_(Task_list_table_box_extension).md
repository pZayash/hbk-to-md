Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для списка задач. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) содержит значение типа [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)<[Имя задачи](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ЗадачаСписок.](ЗадачаСписок.Имя_задачи_(TaskList.Task_name).md)<[Имя задачи](ЗадачаСписок.Имя_задачи_(TaskList.Task_name).md)>.
