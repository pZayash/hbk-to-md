# Расширение поля формы для поля графической схемы.Высота

Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит ширину графической схемы. Задается в символах.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
