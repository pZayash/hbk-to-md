ПолеВыбора (ComboBox)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак возможности/невозможности интерактивного изменения значения, введенного в поле.

Доступность:

Толстый клиент.

Примечание:

В отличие от свойства [Доступность](ПолеВыбора.Доступность_(ComboBox.Enabled).md), если свойство ТолькоПросмотр равно [Истина](lang__def_BooleanTrue.md), становится невозможным редактирование поля выбора, однако на остальную работу с полем это не влияет. Например, можно выделить строковое представление содержимого поля выбора и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
