СерииСлояГеографическойСхемы (GeographicalSchemaLayerSeries)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы_(GeographicalSchemaLayerDataSeries).md).

Удаляет указанный объект из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
