УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.3.9.

Описание:

Не использовать защищенное соединение. Вся информация передается по открытым каналам.

Использование в версии:

Доступен, начиная с версии 8.3.9.
