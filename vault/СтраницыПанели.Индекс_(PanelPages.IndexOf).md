СтраницыПанели (PanelPages)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Страница>)

Параметры:

<Страница> (обязательный)

Тип: [СтраницаПанели](СтраницаПанели_(PanelPage).md).   
Страница панели.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной страницы в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс страницы в коллекции страниц панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ИндексТекСтраницы = Панель.Страницы.Индекс(Панель.ТекущаяСтраница); |

Использование в версии:

Доступен, начиная с версии 8.0.
