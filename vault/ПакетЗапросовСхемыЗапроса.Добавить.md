ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса.md).  
Значение по умолчанию: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md).

Возвращаемое значение:

Тип: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод возвращает Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
