ПредставлениеНавигационнойСсылки (URLPresentation)

Текст (Text)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Визуальное представление ссылки.   
Содержит пустую строку, если ссылка некорректна, в одном из случаев:

- ссылка была неправильно сформирована,
- объект метаданных отсутствует,
- объект отсутствует,
- у пользователя нет прав доступа к объекту.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
