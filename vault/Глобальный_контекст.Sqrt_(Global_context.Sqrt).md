Глобальный контекст (Global context)

Sqrt (Sqrt)

Доступен, начиная с версии 8.0.

Синтаксис:

Sqrt(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Аргумент функции. Неотрицательное число.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат вычисления квадратного корня.

Описание:

Вычисляет квадратный корень параметра <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

При отрицательном значении параметра происходит ошибка "Неправильное значение аргумента встроенной функции (Sqrt)".

Пример:

|  |
| --- |
| // Вычисление радиуса шара по площади поверхности  Pi = 3.1415926535897932; S = 10; R = Sqrt(S/(4\*Pi)); Сообщить("Радиус шара с площадью поверхности = "     + S + "; равен " + Формат(R,"ЧДЦ=3")); |

Использование в версии:

Доступен, начиная с версии 8.0.
