ТелоМакетаКомпоновкиДанных (DataCompositionTemplateBody)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [МакетОбластиМакетаКомпоновкиДанных](МакетОбластиМакетаКомпоновкиДанных_(DataCompositionTemplateAreaTemplate).md), [ГруппировкаМакетаКомпоновкиДанных](ГруппировкаМакетаКомпоновкиДанных_(DataCompositionTemplateGroup).md), [ИерархическаяГруппировкаМакетаКомпоновкиДанных](ИерархическаяГруппировкаМакетаКомпоновкиДанных_(DataCompositionTemplateHierarchicalGroup).md), [ЗаписиМакетаКомпоновкиДанных](ЗаписиМакетаКомпоновкиДанных_(DataCompositionTemplateRecords).md), [ТаблицаМакетаКомпоновкиДанных](ТаблицаМакетаКомпоновкиДанных_(DataCompositionTemplateTable).md), [ДиаграммаМакетаКомпоновкиДанных](ДиаграммаМакетаКомпоновкиДанных_(DataCompositionTemplateChart).md), [ВложенныйОбъектМакетаКомпоновкиДанных](ВложенныйОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateNestedObject).md).

Возвращаемое значение:

Тип: [МакетОбластиМакетаКомпоновкиДанных](МакетОбластиМакетаКомпоновкиДанных_(DataCompositionTemplateAreaTemplate).md) , [ГруппировкаМакетаКомпоновкиДанных](ГруппировкаМакетаКомпоновкиДанных_(DataCompositionTemplateGroup).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md), [ЗаписиМакетаКомпоновкиДанных](ЗаписиМакетаКомпоновкиДанных_(DataCompositionTemplateRecords).md), [ТаблицаМакетаКомпоновкиДанных](ТаблицаМакетаКомпоновкиДанных_(DataCompositionTemplateTable).md), [ДиаграммаМакетаКомпоновкиДанных](ДиаграммаМакетаКомпоновкиДанных_(DataCompositionTemplateChart).md), [ВложенныйОбъектМакетаКомпоновкиДанных](ВложенныйОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateNestedObject).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
