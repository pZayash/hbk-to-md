НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы (InfoBaseUserAuthenticationLockSettings)

МаксимальноеКоличествоНеуспешныхПопыток (MaxUnsuccessfulAttemptsCount)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальное значение количества неуспешных попыток аутентификации, превышение которого приведет к блокировке аутентификации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
