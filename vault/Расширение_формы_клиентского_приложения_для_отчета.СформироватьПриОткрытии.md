Расширение формы клиентского приложения для отчета (Client application form extension for reports)

СформироватьПриОткрытии (GenerateOnOpen)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.   
Указывает необходимость формирования отчета при открытии.  
Истина - отчет будет выполнен сразу на сервере, после обработчика события [ПриСозданииНаСервере](ФормаКлиентскогоПриложения.ПриСозданииНаСервере.md).  
Если в обработчике события установить СтандартнаяОбработка = Ложь, то параметр не отрабатывается, т.е. отчет не формируется.

Использование в версии:

Доступен, начиная с версии 8.2.
