# БизнесПроцессОбъект.<Имя бизнес-процесса>.ПередУдалением

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПередУдалением (BeforeDelete)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУдалением(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак удаления бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то удаление бизнес-процесса выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед непосредственным удалением объекта из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
