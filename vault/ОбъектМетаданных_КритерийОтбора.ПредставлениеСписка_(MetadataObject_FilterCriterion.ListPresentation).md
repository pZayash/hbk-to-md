ОбъектМетаданных: КритерийОтбора (MetadataObject: FilterCriterion)

ПредставлениеСписка (ListPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием списка элементов, соответствующих условию отбора, и используется при отображении стандартных команд.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
