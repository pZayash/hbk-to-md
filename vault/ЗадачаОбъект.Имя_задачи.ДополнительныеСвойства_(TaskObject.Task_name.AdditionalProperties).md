# ЗадачаОбъект.<Имя задачи>.ДополнительныеСвойства

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Свойства](ЗадачаОбъект.Имя_задачи__Свойства.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ДополнительныеСвойства (AdditionalProperties)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Может использоваться в тех случаях, когда необходимо хранить некоторые значения, связанные с объектом, на время выполнения некоторых операций, без изменения объекта. Например, при обработке событий в подписке на события.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
