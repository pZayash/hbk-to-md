СписокРасширенныхИменXML (XMLExpandedNameList)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс расширенного имени в списке.

Возвращаемое значение:

Тип: [РасширенноеИмяXML](РасширенноеИмяXML_(XMLExpandedName).md).   

Описание:

Возвращает расширенное имя XML по индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
