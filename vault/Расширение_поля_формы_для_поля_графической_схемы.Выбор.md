Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор()

Описание:

Возникает при двойном щелчке мыши (нажатии клавиши Enter) на элементе графической схемы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
