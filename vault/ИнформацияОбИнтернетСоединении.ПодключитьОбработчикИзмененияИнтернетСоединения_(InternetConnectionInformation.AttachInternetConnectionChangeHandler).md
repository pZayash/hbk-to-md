ИнформацияОбИнтернетСоединении (InternetConnectionInformation)

ПодключитьОбработчикИзмененияИнтернетСоединения (AttachInternetConnectionChangeHandler)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПодключитьОбработчикИзмененияИнтернетСоединения(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Имя процедуры, подключаемой в качестве обработчика изменения интернет-соединения.  
Процедура должна иметь следующие параметры.

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Подключает указанную процедуру в качестве обработчика, вызываемого при смене текущего типа или скорости соединения сети. Вызов обработчика выполняется в следующих случаях:

- Изменение типа Интернет-соединения:
  - С использования мобильных данных выполнение перехода на использование Wi-Fi-соединения,
  - Отключение соединения всех доступных типов (например, при переходе в режим полета);
- Изменение скорости Интернет-соединения при смене используемого стандарта передачи данных (GPRS, EDGE, LTE, и т.д.);
- При подключении или отключении роуминга в мобильной сети.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
