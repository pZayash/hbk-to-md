Глобальный контекст (Global context)

ПолучитьРежимВнешнихРесурсов (GetExternalResourcesMode)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьРежимВнешнихРесурсов()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

- "D" - режим по умолчанию.
- "А" - альтернативный режим.

Описание:

Получает режим работы с внешними ресурсами.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
