Доступен, начиная с версии 8.0.

Только чтение.

Описание:

Тип: [ПланСчетовМенеджер.\<Имя плана счетов\>](ПланСчетовМенеджер.Имя_плана_счетов_(ChartOfAccountsManager.Chart_of_accounts_name).md).

Использование в версии:

Доступен, начиная с версии 8.0.
