# Расширение поля формы для поля планировщика.Перетаскивание

Расширение поля формы для поля планировщика (Form field extension for a planner)

Перетаскивание (Drag)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Перетаскивание(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Дата>, <ЗначенияИзмерений>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания_(DragParameters).md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Дата>

Тип: [Дата](lang__def_Date.md).   
Дата в планировщике, на которую произведено перетаскивание.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Коллекция значений измерений. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения. В качестве значения — [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения .

Описание:

Возникает при окончании перетаскивания в поле - приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
