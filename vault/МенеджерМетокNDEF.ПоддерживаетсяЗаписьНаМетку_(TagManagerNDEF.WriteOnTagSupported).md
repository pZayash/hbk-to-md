МенеджерМетокNDEF (TagManagerNDEF)

ПоддерживаетсяЗаписьНаМетку (WriteOnTagSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяЗаписьНаМетку()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - устройство поддерживает запись на метку и NFC включено,  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, что устройство поддерживает операции записи на метку.   
Для устройств на Android дополнительно проверяется, что пользователь включил NFC в настройках телефона.  
Если запись не поддерживается, то методы [ЗаписатьСообщениеАсинх](МеткаNDEF.ЗаписатьСообщениеАсинх_(TagNDEF.WriteMessageAsync).md) и [ЗаписатьСообщениеАсинх](МеткаNDEF.ЗаписатьСообщениеАсинх_(TagNDEF.WriteMessageAsync).md) будут недоступны.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МеткаNDEF](МеткаNDEF_(TagNDEF).md), метод [ПоддерживаетсяЗаписьНаМеткуАсинх](МеткаNDEF.ПоддерживаетсяЗаписьНаМеткуАсинх_(TagNDEF.WriteOnTagSupportedAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
