# ШкалаДиаграммы.Отображение

ШкалаДиаграммы (ChartScale)

Отображение (ShowInChart)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеВДиаграмме](ОтображениеВДиаграмме_(ShowInChart).md).   
Управляет отображением шкалы диаграммы.  
Если установлено значение [Отображать](ОтображениеВДиаграмме.Отображать_(ShowInChart.Show).md) и нет ни одной серии, которая выводится на этой шкале, то шкала дублирует основную.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
