ДокументDOM (DOMDocument)

ПолучитьЭлементПоИдентификатору (GetElementById)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьЭлементПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор искомого элемента.

Возвращаемое значение:

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md), [Неопределено](lang__def_Undefined.md).   
Узел, у которого имеется атрибут со свойством [ЯвляетсяИдентификатором](АтрибутDOM.ЯвляетсяИдентификатором_(DOMAttribute.IsId).md), установленным в [Истина](lang__def_BooleanTrue.md).

Описание:

Выбирает все элементы по идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
