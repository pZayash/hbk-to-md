КоллекцияИндексовXBase (XBaseIndexesCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ИндексXBase>)

Параметры:

<ИндексXBase> (обязательный)

Тип: [ИндексXBase](ИндексXBase_(XBaseIndex).md).   
Искомый индекс XBase.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс индекса в коллекции индексов XBase.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
