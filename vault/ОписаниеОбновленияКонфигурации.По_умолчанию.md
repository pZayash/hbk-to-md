ОписаниеОбновленияКонфигурации (ConfigurationUpdateDescription)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ОписаниеОбновленияКонфигурации(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные файла, из которого будет сформирован объект [ОписаниеОбновленияКонфигурации](ОписаниеОбновленияКонфигурации.md).

Описание:

Создает объект на основе двоичных данных файла в формате \*.cfu.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации.md), метод [ПолучитьДанные](РасширениеКонфигурации.ПолучитьДанные.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
