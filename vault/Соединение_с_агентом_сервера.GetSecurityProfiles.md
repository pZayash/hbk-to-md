Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfiles (GetSecurityProfiles)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfiles(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого должен быть получен список профилей безопасности.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив объектов [Профиль безопасности](Профиль_безопасности.md).

Описание:

Получает список профилей безопасности кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
