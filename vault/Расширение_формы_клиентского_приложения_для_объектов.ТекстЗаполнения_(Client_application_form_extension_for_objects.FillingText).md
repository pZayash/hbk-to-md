# Расширение формы клиентского приложения для объектов.ТекстЗаполнения

Расширение формы клиентского приложения для объектов (Client application form extension for objects)

ТекстЗаполнения (FillingText)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: [Строка](lang__def_String.md).   
Текст, введенный в поле ввода, при открытии формы в режиме выбора.  
Если основное представление ВВидеНаименования и реквизит Наименование находится в списке ВводПоСтроке, то значение присваивается реквизиту Наименование.  
Если основное представление ВВидеКода и реквизит Код находится в списке ВводПоСтроке и не установлен признак Автонумерация, то значение присваивается реквизиту Код.  
В противном случае присваивание не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.3.
