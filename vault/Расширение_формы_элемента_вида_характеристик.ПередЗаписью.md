Расширение формы элемента вида характеристик (Characteristic type item form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи элемента. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись элемента выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением записи элемента вида характеристики из формы. Процедура-обработчик данного события вызывается перед вызовом процедуры обработчика события [ПередЗаписью](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередЗаписью.md) объекта [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
