ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи набора записей в базу данных. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи набора записей. Процедура-обработчик вызывается после начала транзакции записи, но до начала записи набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
