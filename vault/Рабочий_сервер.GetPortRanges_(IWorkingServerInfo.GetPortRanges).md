Рабочий сервер (IWorkingServerInfo)

GetPortRanges (GetPortRanges)

Доступен, начиная с версии 8.1.

Синтаксис:

GetPortRanges()

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Метод возвращает [COMSafeArray](COMSafeArray_(COMSafeArray).md) из объектов [Диапазон портов](Диапазон_портов_(IPortRangeInfo).md).

Описание:

Получает выделенные диапазоны IP-портов для динамического занятия.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
