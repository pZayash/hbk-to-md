ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - изменяется еще ни разу не записанный элемент; [Ложь](lang__def_BooleanFalse.md) - элемент уже записан.

Описание:

Определяет, записан ли элемент в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Если Объект.ЭтоНовый() Тогда     Объект.Наименование = "Новый счет"; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
