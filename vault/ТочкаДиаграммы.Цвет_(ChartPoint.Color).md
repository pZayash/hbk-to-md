ТочкаДиаграммы (ChartPoint)

Цвет (Color)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет точки диаграммы.  
Используется вместо цвета серии, если у точки установлено свойство [ПриоритетЦвета](ТочкаДиаграммы.ПриоритетЦвета_(ChartPoint.ColorPriority).md) и у серии такое свойство не установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не для всех типов диаграмм.

См. также:

[ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), свойство [ПриоритетЦвета](ТочкаДиаграммы.ПриоритетЦвета_(ChartPoint.ColorPriority).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
