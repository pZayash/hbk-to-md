КлиентскоеПриложение (ClientApplication)

ПолучитьЗаголовок (GetCaption)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьЗаголовок()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает заголовок приложения, который используется при формировании заголовков окон.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) получает заголовок главного окна.

См. также:

[КлиентскоеПриложение](КлиентскоеПриложение_(ClientApplication).md), метод [УстановитьЗаголовок](КлиентскоеПриложение.УстановитьЗаголовок_(ClientApplication.SetCaption).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
