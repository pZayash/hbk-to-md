БлокировкаСеансов (SessionsLock)

КодРазрешения (KeyCode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит код разрешения в виде строки, которая будет добавлена к параметру командной строки "/uc", чтобы начать новый сеанс с информационной базой вопреки блокировке сеансов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
