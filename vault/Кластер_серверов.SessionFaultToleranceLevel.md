Кластер серверов (IClusterInfo)

SessionFaultToleranceLevel (SessionFaultToleranceLevel)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет уровень отказоустойчивости кластера: количество рабочих серверов кластера, выход из строя которых не приводит к аварийному завершению сеансов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
