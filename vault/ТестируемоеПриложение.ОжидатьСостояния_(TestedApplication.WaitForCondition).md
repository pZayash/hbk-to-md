ТестируемоеПриложение (TestedApplication)

ОжидатьСостояния (WaitForCondition)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОжидатьСостояния(<ОписаниеОповещения>, <Таймаут>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание функции, которая будет возвращать:

- [Истина](lang__def_BooleanTrue.md) - если состояние достигнуто,
- [Ложь](lang__def_BooleanFalse.md) - в противном случае.

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Время в секундах, в течение которого выполняется ожидание состояния.  
Значение по умолчанию: 60.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если функция, проверяющая достижение состояния, вернула [Истина](lang__def_BooleanTrue.md).  
[Ложь](lang__def_BooleanFalse.md) - закончился тайм-аут или функция проверки вернула [Ложь](lang__def_BooleanFalse.md).

Описание:

Ожидает наступления некоторого состояния за счет вызова функции, проверяющей условия его достижения с периодичностью 1 раз в секунду (без цикла ожидания по времени).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
