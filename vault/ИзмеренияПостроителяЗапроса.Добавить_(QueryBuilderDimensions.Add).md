ИзмеренияПостроителяЗапроса (QueryBuilderDimensions)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Имя>, <Тип>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным для измерения.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя измерения.

<Тип> (необязательный)

Тип: [ТипИзмеренияПостроителяЗапроса](ТипИзмеренияПостроителяЗапроса_(QueryBuilderDimensionType).md).   
Тип измерения.

Возвращаемое значение:

Тип: [ИзмерениеПостроителяЗапроса](ИзмерениеПостроителяЗапроса_(QueryBuilderDimension).md).   

Описание:

Добавляет измерение в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
