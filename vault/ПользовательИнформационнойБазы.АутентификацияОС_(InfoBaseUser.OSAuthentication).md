ПользовательИнформационнойБазы (InfoBaseUser)

АутентификацияОС (OSAuthentication)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, использовать ли аутентификацию Windows для данного пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - аутентификация Windows разрешена; [Ложь](lang__def_BooleanFalse.md) - запрещена.  
Доступно только пользователю с административными правами.

Использование в версии:

Доступен, начиная с версии 8.0.
