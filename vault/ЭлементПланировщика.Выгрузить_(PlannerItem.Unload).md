ЭлементПланировщика (PlannerItem)

Выгрузить (Unload)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: [Структура](Структура_(Structure).md).   

Описание:

Создает структуру и копирует в нее соответствующие свойства элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), метод [Загрузить](ЭлементПланировщика.Загрузить_(PlannerItem.Load).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
