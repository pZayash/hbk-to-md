ОбъектМетаданных: Документ (MetadataObject: Document)

УдалениеДвижений (RegisterRecordsDeletion)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [УдалениеДвижений](УдалениеДвижений.md).   
Определяет варианты удаления движений документа по регистрам при проведении и отмене проведения (например, [УдалятьАвтоматически](УдалениеДвижений.УдалятьАвтоматически.md), [НеУдалятьАвтоматически](УдалениеДвижений.НеУдалятьАвтоматически.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
