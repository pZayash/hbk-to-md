МенеджерПолученияЛицензий (LicenseAcquisitionManager)

СохранитьЗапросНаПолучениеЛицензии (SaveLicenseAcquisitionRequest)

Доступен, начиная с версии 8.3.20.

Синтаксис:

СохранитьЗапросНаПолучениеЛицензии(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md).   
Данные запроса на получение лицензии.

Описание:

Сохраняет данные для получения лицензии в каталог данных приложений всех пользователей или текущего пользователя операционной системы.  
Сохраненные данные могут быть прочитаны методом [ЗагрузитьЗапросНаПолучениеЛицензии](МенеджерПолученияЛицензий.ЗагрузитьЗапросНаПолучениеЛицензии.md) и использованы в диалоге получения лицензии, встроенным в платформу.  
Значение свойства [Пинкод](ЗапросНаПолучениеЛицензии.Пинкод.md) не сохраняется.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
