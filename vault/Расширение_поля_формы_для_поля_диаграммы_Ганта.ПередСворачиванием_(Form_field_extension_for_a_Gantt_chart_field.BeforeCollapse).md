# Расширение поля формы для поля диаграммы Ганта.ПередСворачиванием

Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ПередСворачиванием (BeforeCollapse)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПередСворачиванием(<ИдентификаторЗначения>, <Отказ>)

Параметры:

<ИдентификаторЗначения>

Тип: [ИдентификаторЗначенияДиаграммыГанта](ИдентификаторЗначенияДиаграммыГанта_(GanttChartValueID).md).   
Содержит идентификатор значения, точка/серия которого будет свернута.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то стандартной обработки события не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается перед сворачиванием точки/серии диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
