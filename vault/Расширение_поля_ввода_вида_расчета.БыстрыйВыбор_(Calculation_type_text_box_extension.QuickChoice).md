# Расширение поля ввода вида расчета.БыстрыйВыбор

Расширение поля ввода вида расчета (Calculation type text box extension)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Предоставляет доступ к управлению режимом выбора значения. Если свойство установлено в [Истина](lang__def_BooleanTrue.md), то выбор будет выполняться из выпадающего списка, если [Ложь](lang__def_BooleanFalse.md) - из окна выбора вида расчета.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
