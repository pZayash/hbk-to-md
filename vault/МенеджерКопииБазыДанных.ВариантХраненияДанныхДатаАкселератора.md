МенеджерКопииБазыДанных (DatabaseCopyManager)

ВариантХраненияДанныхДатаАкселератора (DataAcceleratorDataLocationVariant)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ВариантХраненияДанныхДатаАкселератора](ВариантХраненияДанныхДатаАкселератора.md).   
Вариант хранения копии базы данных, размещаемой на Дата акселераторе:

- для копии на Дата акселераторе - используется вариант хранения базы данных копии,
- для других копий - всегда имеет значение [ВОперативнойПамятиИНаДиске](ВариантХраненияДанныхДатаАкселератора.ВОперативнойПамятиИНаДиске.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
