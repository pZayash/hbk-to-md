ЧтениеДанных (DataReader)

НачатьЧтениеВБуферДвоичныхДанных (BeginReadIntoBinaryDataBuffer)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: Чтение в новый буфер

Синтаксис:

НачатьЧтениеВБуферДвоичныхДанных(<ОписаниеОповещения>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Буфер> - тип [БуферДвоичныхДанных](БуферДвоичныхДанных.md). Буфер, содержащий прочитанные байты.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые требуется прочитать. Если не задано, будут прочитаны все данные до конца потока.  
Значение по умолчанию: Неопределено.

Описание варианта метода:

Прочитать заданное количество байтов в новый буфер.

Вариант синтаксиса: Чтение в существующий буфер

Синтаксис:

НачатьЧтениеВБуферДвоичныхДанных(<ОписаниеОповещения>, <Буфер>, <ПозицияВБуфере>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <РеальноеКоличество> - тип Число. Количество байтов, которые были фактически прочитаны и записаны в буфер.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер двоичных данных, в который требуется записать прочитанные байты.

<ПозицияВБуфере> (обязательный)

Тип: Число.   
Позиция в буфере, начиная с которой будут записаны прочитанные данные.

<Количество> (обязательный)

Тип: Число.   
Количество байтов, которое требуется прочитать.

Описание варианта метода:

Прочитать заданное количество байтов в существующий буфер, начиная с заданной позиции в буфере.

Описание:

Начинает чтение байтов из потока в [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[ЧтениеДанных](ЧтениеДанных.md), метод [ПрочитатьВБуферДвоичныхДанных](ЧтениеДанных.ПрочитатьВБуферДвоичныхДанных.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
