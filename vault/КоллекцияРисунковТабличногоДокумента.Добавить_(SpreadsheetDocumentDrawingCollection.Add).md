КоллекцияРисунковТабличногоДокумента (SpreadsheetDocumentDrawingCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ТипРисунка>)

Параметры:

<ТипРисунка> (обязательный)

Тип: [ТипРисункаТабличногоДокумента](ТипРисункаТабличногоДокумента_(SpreadsheetDocumentDrawingType).md).   
Тип добавляемого рисунка.

Возвращаемое значение:

Тип: [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md).   

Описание:

Добавляет рисунок в конец коллекции рисунков табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Нельзя добавлять в коллекцию рисунок типа [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.0.
