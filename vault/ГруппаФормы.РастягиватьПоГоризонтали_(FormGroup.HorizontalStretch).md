# ГруппаФормы.РастягиватьПоГоризонтали

ГруппаФормы (FormGroup)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Устанавливает возможность растягивания группы по горизонтали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.  
[Неопределено](lang__def_Undefined.md) - автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
