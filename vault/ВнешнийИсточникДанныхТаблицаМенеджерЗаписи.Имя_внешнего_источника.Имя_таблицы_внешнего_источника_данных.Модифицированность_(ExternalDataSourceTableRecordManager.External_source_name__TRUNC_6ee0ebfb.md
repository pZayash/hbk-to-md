ВнешнийИсточникДанныхТаблицаМенеджерЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordManager.<External source name>.<External data source table name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект изменен; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, изменен ли объект по отношению к считанным значениям.

Доступность:

Толстый клиент.

Примечание:

Метод не позволяет определить, была ли изменена соответствующая запись другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.3.5.
