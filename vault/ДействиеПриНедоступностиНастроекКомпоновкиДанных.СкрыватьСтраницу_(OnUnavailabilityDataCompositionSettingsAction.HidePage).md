ДействиеПриНедоступностиНастроекКомпоновкиДанных (OnUnavailabilityDataCompositionSettingsAction)

СкрыватьСтраницу (HidePage)

Доступен, начиная с версии 8.1.

Описание:

Станица скрывается, если она содержит только элементы управления, связанные с текущими данными структуры настроек, которые не имеют смысла для текущей страницы.  
Наличие или отсутствие других элементов управления на других страницах формы не анализируется.

Использование в версии:

Доступен, начиная с версии 8.1.
