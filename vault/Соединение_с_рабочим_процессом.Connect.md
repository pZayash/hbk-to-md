Соединение с рабочим процессом (IWorkingProcessConnection)

Connect (Connect)

Доступен, начиная с версии 8.1.

Синтаксис:

Connect(<ИнформационнаяБаза>, <ИмяПользователя>, <Пароль>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база.md).   
Информационная база, для которой должно быть установлено значение свойства [Name](Информационная_база.Name.md).

<ИмяПользователя> (обязательный)

Тип: Строка.   
Имя пользователя информационной базы 1С:Предприятия, который устанавливает соединение.

<Пароль> (обязательный)

Тип: Строка.   
Пароль пользователя информационной базы. Если пароля нет, то нужно указать пустую строку.

Возвращаемое значение:

Тип: [Внешнее соединение](Внешнее_соединение.md).   

Описание:

Устанавливает соединение с информационной базой, зарегистрированной в кластере серверов 1С:Предприятия, аналогичное результату метода [Connect](Менеджер_COM-соединений.Connect.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
