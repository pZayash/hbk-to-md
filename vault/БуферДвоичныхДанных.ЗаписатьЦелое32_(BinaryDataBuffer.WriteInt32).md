БуферДвоичныхДанных (BinaryDataBuffer)

ЗаписатьЦелое32 (WriteInt32)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьЦелое32(<Позиция>, <Значение>, <ПорядокБайтов>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, на которой требуется записать число.

<Значение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число, которое требуется записать.  
Если значение не помещается в 32 бита, будет вызван исключение.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа при записи в буфер. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Записать целое 32-битное положительное число в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
