РазрешенныйCOMКласс (AllowedCOMClass)

ИмяКомпьютера (ComputerName)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя компьютера, на котором может быть создан COM-объект. Используется при создании нового COM-объекта. Специальные значения:

- localhost – текущий компьютер;
- пустая строка – произвольный компьютер.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
