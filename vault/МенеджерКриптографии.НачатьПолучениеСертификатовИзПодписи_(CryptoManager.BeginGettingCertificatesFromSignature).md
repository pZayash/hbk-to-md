МенеджерКриптографии (CryptoManager)

НачатьПолучениеСертификатовИзПодписи (BeginGettingCertificatesFromSignature)

Не рекомендуется использовать, начиная с версии 8.3.24.

Рекомендуется использовать:

- [ПолучитьСертификаты](МенеджерКриптографии.ПолучитьСертификаты_(CryptoManager.GetCertificates).md)
- [ПолучитьСертификатыАсинх](МенеджерКриптографии.ПолучитьСертификатыАсинх_(CryptoManager.GetCertificatesAsync).md)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеСертификатовИзПодписи(<ОписаниеОповещения>, <ИсходныеДанные>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения получения сертификатов из подписи со следующими параметрами:

- <Сертификаты> - массив объектов [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Исходные данные, из которых будет извлечен сертификат.  
Данные могут размещаться в файле (в этом случае указывается имя файла) или представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание:

Начинает извлечение массива сертификатов из данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md), метод [ПолучитьСертификатыИзПодписиАсинх](МенеджерКриптографии.ПолучитьСертификатыИзПодписиАсинх_(CryptoManager.GetCertificatesFromSignatureAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.24.
