# ПланСчетовОбъект.<Имя плана счетов>.Уровень

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Методы](ПланСчетовОбъект.Имя_плана_счетов__Методы.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Уровень (Level)

Доступен, начиная с версии 8.0.

Синтаксис:

Уровень()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает уровень счета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл только для многоуровневых планов счетов. Следует учитывать, что уровень элемента может меняться, например, при переносе в другую группу (смене родителя).  
Уровень счета начинается с 0. Для элемента, не имеющего родителя, уровень будет равен 0.

Пример:

|  |
| --- |
| Сообщить("Данный элемент принадлежит уровню " +           Объект.Уровень()); |

Использование в версии:

Доступен, начиная с версии 8.0.
