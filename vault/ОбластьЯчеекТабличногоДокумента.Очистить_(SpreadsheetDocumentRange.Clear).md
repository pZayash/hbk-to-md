# ОбластьЯчеекТабличногоДокумента.Очистить

ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Очистить (Clear)

Доступен, начиная с версии 8.2.

Синтаксис:

Очистить(<ОчищатьТекст>, <ОчищатьФорматирование>, <ОчищатьРамки>)

Параметры:

<ОчищатьТекст> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость очищения текста ячеек табличного документа. [Истина](lang__def_BooleanTrue.md) - очищать текст, [Ложь](lang__def_BooleanFalse.md) - не очищать текст.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ОчищатьФорматирование> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость очищения формата ячеек табличного документа. [Истина](lang__def_BooleanTrue.md) - очищать формат, [Ложь](lang__def_BooleanFalse.md) - не очищать формат.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ОчищатьРамки> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость удаления рамок ячеек области табличного документа. [Истина](lang__def_BooleanTrue.md) - удалять рамки; [Ложь](lang__def_BooleanFalse.md) - не удалять рамки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Очищает содержимое области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Область.Очистить(); // очищает текст области Область |

Использование в версии:

Доступен, начиная с версии 8.2.
