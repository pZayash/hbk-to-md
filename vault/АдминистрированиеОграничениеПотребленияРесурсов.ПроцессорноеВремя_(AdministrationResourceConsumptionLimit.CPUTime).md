АдминистрированиеОграничениеПотребленияРесурсов (AdministrationResourceConsumptionLimit)

ПроцессорноеВремя (CPUTime)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит предельное значение времени, которое было затрачено процессором на обработку серверных вызовов, в миллисекундах.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
