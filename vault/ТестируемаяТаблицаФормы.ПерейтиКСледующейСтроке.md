ТестируемаяТаблицаФормы (TestedFormTable)

ПерейтиКСледующейСтроке (GotoNextRow)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПерейтиКСледующейСтроке(<ПереключитьВыделение>)

Параметры:

<ПереключитьВыделение> (необязательный)

Тип: Булево.   
Если установлено Ложь, поведение метода эмулирует обычный переход курсора к строке при помощи клавиатуры.  
Если установлено Истина, поведение метода эмулирует обычный переход курсора к строке при помощи клавиатуры с нажатой клавишей Shift.  
Значение по умолчанию: Ложь.

Описание:

Перемещает курсор к следующей строке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
