МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблоныСообщения (GetMessageTemplates)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблоныСообщения(<Интеграция>)

Параметры:

<Интеграция> (обязательный)

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия.md).   
Идентификатор интеграции, для которой возвращается список шаблонов сообщений.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Для интеграции типа WhatsApp возвращает список шаблонов сообщений.  
Для остальных интеграций возвращает пустой массив.

Описание:

Получает массив, содержащий шаблоны сообщения системы взаимодействия.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ПолучитьШаблоныСообщенияАсинх](МенеджерСистемыВзаимодействия.ПолучитьШаблоныСообщенияАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
