Элементы коллекции:

[ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.

Описание:

Предназначен для моделирования коллекций в данных формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
