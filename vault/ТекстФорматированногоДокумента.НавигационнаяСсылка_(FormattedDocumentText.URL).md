ТекстФорматированногоДокумента (FormattedDocumentText)

НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит навигационную ссылку текста форматированного документа.  
Пустая строка, если в тексте нет никаких частей гиперссылок. Непустая строка, если весь текст - гиперссылка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
