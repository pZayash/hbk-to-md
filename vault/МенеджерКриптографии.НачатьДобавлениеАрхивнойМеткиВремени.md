МенеджерКриптографии (CryptoManager)

НачатьДобавлениеАрхивнойМеткиВремени (BeginAddingArchiveTimestamp)

Доступен, начиная с версии 8.3.20.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

НачатьДобавлениеАрхивнойМеткиВремени(<ОписаниеОповещения>, <Подпись>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения проверки подписи со следующими параметрами:

- <ВыходныеДанные> - [ДвоичныеДанные](ДвоичныеДанные.md) c результатом.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md) или [ФайловыйПоток](ФайловыйПоток.md).

Вариант синтаксиса: Данные подписи сохраняются в файл

Синтаксис:

НачатьДобавлениеАрхивнойМеткиВремени(<ОписаниеОповещения>, <Подпись>, <ВыходныеДанные>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения проверки подписи со следующими параметрами:

- <ВыходныеДанные> - [ДвоичныеДанные](ДвоичныеДанные.md) с результатом.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md) или [ФайловыйПоток](ФайловыйПоток.md).

<ВыходныеДанные> (обязательный)

Тип: Строка, [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Имя файла или поток, в который будут выгружены данные подписи.

Описание:

Инициирует добавление в подпись архивной метки времени 3 версии.   
При добавлении метки выполняется обращение к серверам меток времени, указанным в поле [АдресаСерверовМетокВремени](МенеджерКриптографии.АдресаСерверовМетокВремени.md).   
  
По завершении, будет вызвана процедура, указанная в <ОписанииОповещения>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Доступен, начиная с версии 8.3.20.
