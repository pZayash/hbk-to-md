<Имя измерения> (<Dimension name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения измерений перерасчета. Доступ к значению осуществляется по имени, как оно задано в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Тип свойства определяется измерением или реквизитом родительского регистра расчета. В качестве типа допускается использовать любые типы, кроме [УникальныйИдентификатор](УникальныйИдентификатор.md), [ДвоичныеДанные](ДвоичныеДанные.md) и строки неограниченной длины.

Использование в версии:

Доступен, начиная с версии 8.0.
