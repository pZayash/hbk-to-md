ОбъектМетаданных: Отчет (MetadataObject: Report)

ДополнительнаяФорма (AuxiliaryForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Описывает форму, которая будет вызвана системой для выполнения стандартных действий с критерием отбора, в случае если основная форма не установлена или не может быть использована в конкретном варианте использования 1С:Предприятия.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
