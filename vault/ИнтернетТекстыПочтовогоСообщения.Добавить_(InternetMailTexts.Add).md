# ИнтернетТекстыПочтовогоСообщения.Добавить

ИнтернетТекстыПочтовогоСообщения (InternetMailTexts)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: С указанием типа текста

Синтаксис:

Добавить(<Текст>, <ТипТекста>)

Параметры:

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Добавляемый текст.

<ТипТекста> (необязательный)

Тип: [Строка](lang__def_String.md), [ТипТекстаПочтовогоСообщения](ТипТекстаПочтовогоСообщения_(InternetMailTextType).md).   
Указывает тип добавляемого текста.   
Если [Строка](lang__def_String.md), то свойство [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста_(InternetMailText.TextType).md) устанавливается в [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст_(InternetMailTextType.CustomText).md).  
При генерации отправляемого текста письма, в поле Content-Type письма добавляется (или заменяется) параметр charset, получаемый из кодировки, указанной в свойстве [Кодировка](ИнтернетТекстПочтовогоСообщения.Кодировка_(InternetMailText.Encoding).md).  
Значение по умолчанию: [ПростойТекст](ТипТекстаПочтовогоСообщения.ПростойТекст_(InternetMailTextType.PlainText).md).

Вариант синтаксиса: Без указания типа текста

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Добавляемый текст.

Возвращаемое значение:

Тип: [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md).   

Описание:

Добавляет текст в коллекцию почтовых текстов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если для всех параметров метода используются значения по умолчанию, то [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста_(InternetMailText.TextType).md) добавленного объекта [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md) будет равен [HTML](ТипТекстаПочтовогоСообщения.HTML_(InternetMailTextType.HTML).md). Если же указан первый параметр <Текст>, то в качестве типа текста будет установлен [ПростойТекст](ТипТекстаПочтовогоСообщения.ПростойТекст_(InternetMailTextType.PlainText).md).

Использование в версии:

Доступен, начиная с версии 8.0.
