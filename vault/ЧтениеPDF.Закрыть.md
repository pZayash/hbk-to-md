ЧтениеPDF (PDFReader)

Закрыть (Close)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый файл.   
Закрытие файла выполняется автоматически при удалении объекта [ЧтениеPDF](ЧтениеPDF.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [ЗакрытьАсинх](ЧтениеPDF.ЗакрытьАсинх.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [НачатьЗакрытие](ЧтениеPDF.НачатьЗакрытие.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
