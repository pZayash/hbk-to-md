Описание:

Содержит варианты форматов файлов для сохранения табличного документа. Используется для определения параметра <ТипФайлаТаблицы> метода [Записать](ТабличныйДокумент.Записать.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), метод [Записать](ТабличныйДокумент.Записать.md)
