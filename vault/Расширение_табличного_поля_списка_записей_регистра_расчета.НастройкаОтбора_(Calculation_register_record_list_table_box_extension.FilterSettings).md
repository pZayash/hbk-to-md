Расширение табличного поля списка записей регистра расчета (Calculation register record list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора_(FilterSettings).md).   
Предоставляет доступ к настройкам отбора списка записей регистра расчета. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для регистра расчета элементов отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
