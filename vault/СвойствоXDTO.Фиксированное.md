СвойствоXDTO (XDTOProperty)

Фиксированное (Fixed)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак фиксированного значения. Если Истина - объект, соответствующий модели XDTO, должен содержать в данном свойстве указанное значение по умолчанию.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
