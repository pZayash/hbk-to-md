ФрагментДокументаDOM (DOMDocumentFragment)

УдалитьДочерний (RemoveChild)

Доступен, начиная с версии 8.1.

Синтаксис:

УдалитьДочерний(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Удаляемый узел DOM.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Удаляемый узел.

Описание:

Удаляет узел из списка дочерних узлов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Исключение вызывается в случаях:

- если данный узел в режиме Только для чтения;
- если удаляемый узел не найден;
- если удаляемый узел является дочерним узлом узла [Документ](ТипУзлаDOM.Документ.md), исключение генерируется в случае, если реализация не поддерживает удаление узлов [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) и [Элемент](ТипУзлаDOM.Элемент.md).

Использование в версии:

Доступен, начиная с версии 8.1.
