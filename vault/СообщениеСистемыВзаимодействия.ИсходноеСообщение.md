СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

ИсходноеСообщение (SourceMessage)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: [ИдентификаторСообщенияСистемыВзаимодействия](ИдентификаторСообщенияСистемыВзаимодействия.md), Неопределено.   
Содержит идентификатор сообщения, ответом на которое является данное сообщение.  
Заполняется у сообщений, посылаемых нажатием на кнопку (панели кнопок) сообщения и у ответных сообщений, получаемых из интеграций.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
