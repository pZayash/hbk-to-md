МенеджерВстроенныхПокупок (InAppPurchasesManager)

Приобретена (Purchased)

Не рекомендуется использовать, начиная с версии 8.3.18.

Рекомендуется использовать:

- [ПроверитьПриобретение](МенеджерВстроенныхПокупок.ПроверитьПриобретение_(InAppPurchasesManager.CheckPurchased).md)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Приобретена(<Покупка>)

Параметры:

<Покупка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Приобретенная покупка.  
Значение по умолчанию: [ВстроеннаяПокупка](ВстроеннаяПокупка_(InAppPurchase).md); [Строка](lang__def_String.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Возвращает информацию, приобретена ли указанная покупка. Поведение метода зависит от типа покупки (постоянная, подписка или расходуемая) и от используемой ОС:

- Для постоянных покупок и подписок возвращает информацию о том, приобретена ли указанная покупка (активна ли подписка). Работает одинаково на iOS и Android.
- Для расходуемых покупок:
  - На Android - возвращает значение [Истина](lang__def_BooleanTrue.md), если покупка была приобретена и не была израсходована.
  - На iOS - возвращает значение [Ложь](lang__def_BooleanFalse.md). Данные для выполнения данного метода получаются из истории покупок, хранящейся локально, поэтому для актуализации данных следует использовать метод [ОбновитьИнформациюОПриобретении](МенеджерВстроенныхПокупок.ОбновитьИнформациюОПриобретении_(InAppPurchasesManager.UpdatePurchaseInformation).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.

Не рекомендуется использовать, начиная с версии 8.3.18.

Описание изменено в версии 8.3.18.
