Глобальный контекст (Global context)

ПолучитьЧасовойПоясИнформационнойБазы (GetInfoBaseTimeZone)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЧасовойПоясИнформационнойБазы()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Список допустимых часовых поясов может быть получен методом [ПолучитьДопустимыеЧасовыеПояса](Глобальный_контекст.ПолучитьДопустимыеЧасовыеПояса_(Global_context.GetAvailableTimeZones).md).   
Кроме того, возможно задание часовых поясов смещением от универсального времени (часовые пояса GMT) в формате:  
GMT{+/-}h[h][:mm]  
где hh=0:23, mm=0:59.  
[Неопределено](lang__def_Undefined.md), если он не был установлен.

Описание:

Получает строку идентификатора часового пояса информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьЧасовойПоясИнформационнойБазы](Глобальный_контекст.УстановитьЧасовойПоясИнформационнойБазы_(Global_context.SetInfoBaseTimeZone).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
