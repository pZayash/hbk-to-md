НастройкиНачальнойСтраницы (HomePageSettings)

УстановитьСоставФорм (SetForms)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьСоставФорм(<СоставФорм>)

Параметры:

<СоставФорм> (обязательный)

Тип: [СоставФормНачальнойСтраницы](СоставФормНачальнойСтраницы_(HomePageForms).md).   
В элементах массива должны указываться полные имена объектов метаданных форм. Разрешается передавать имена стандартных форм объектов.   
Если переданная строка не является именем формы, то этот элемент массива игнорируется.

Описание:

Формирует в пользовательских настройках отличия переданного состава форм от заданных в конфигураторе.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В составе форм должны быть переданы полные имена объектов метаданных форм. Разрешается передавать имена стандартных форм объектов. Если переданная строка не является именем формы, то этот элемент массива игнорируется.

Использование в версии:

Доступен, начиная с версии 8.3.5.
