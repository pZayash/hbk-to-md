АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ПрофильБезопасностиБезопасногоРежима (SafeModeSecurityProfile)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит профиль безопасности исполнения кода, внешнего по отношению к конфигурации. Если ссылка не указана, то в безопасном режиме исполнения кода информационная база работает с полным запретом на внешнюю активность. Если указано имя профиля безопасности, то поведение информационной базы в безопасном режиме исполнения кода регламентируется этим профилем.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
