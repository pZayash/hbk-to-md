# ПорядокКомпоновкиДанных.ПолучитьОбъектПоИдентификатору

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Настройки компоновки данных](Настройки_компоновки_данных.md) › [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных_(DataCompositionOrder).md) › [Методы](ПорядокКомпоновкиДанных__Методы.md)

ПорядокКомпоновкиДанных (DataCompositionOrder)

ПолучитьОбъектПоИдентификатору (GetObjectByID)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбъектПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md).   
Идентификатор объекта.

Возвращаемое значение:

Тип: [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных_(DataCompositionAutoOrderItem).md), [ЭлементПорядкаКомпоновкиДанных](ЭлементПорядкаКомпоновкиДанных_(DataCompositionOrderItem).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объект, для которого был передан идентификатор, не найден или если он не является частью данного объекта.

Описание:

Получает элементы порядка компоновки данных, соответствующие переданному идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
