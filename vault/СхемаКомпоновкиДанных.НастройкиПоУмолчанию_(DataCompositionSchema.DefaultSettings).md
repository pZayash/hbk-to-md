СхемаКомпоновкиДанных (DataCompositionSchema)

НастройкиПоУмолчанию (DefaultSettings)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Содержат настройки первого варианта настроек, которые будут использоваться как настройки по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
