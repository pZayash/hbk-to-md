ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

РежимЗаписи (WriteMode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [РежимЗаписиРегистра](РежимЗаписиРегистра_(RegisterWriteMode).md).   
Определяет режим добавления записей в регистр сведений (например, [Независимый](РежимЗаписиРегистра.Независимый_(RegisterWriteMode.Independent).md), [ПодчинениеРегистратору](РежимЗаписиРегистра.ПодчинениеРегистратору_(RegisterWriteMode.RecorderSubordinate).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
