Информационная база (IInfoBaseInfo)

DeniedFrom (DeniedFrom)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Начало начала интервала времени, в течение которого действует режим блокировки сеансов.  
Если указана пустая дата ('00010101'), то без ограничения.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
