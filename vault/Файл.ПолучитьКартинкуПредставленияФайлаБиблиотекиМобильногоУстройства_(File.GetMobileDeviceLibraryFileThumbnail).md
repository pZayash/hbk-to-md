Файл (File)

ПолучитьКартинкуПредставленияФайлаБиблиотекиМобильногоУстройства (GetMobileDeviceLibraryFileThumbnail)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьКартинкуПредставленияФайлаБиблиотекиМобильногоУстройства()

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md).   

Описание:

Возвращает картинку представления файла из мультимедиа-библиотек картинок и видео.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Метод работает только с файлами, расположенными в мультимедиа-библиотеках картинок и видео. Путь к файлу картинки или видео должен начинаться со схемы "content:". На мобильном устройстве под управлением Windows возвращает [Неопределено](lang__def_Undefined.md).
