СредстваТелефонии (TelephonyTools)

ПоддерживаетсяЖурналSMS (SMSLogSupported)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПоддерживаетсяЖурналSMS()

Возвращаемое значение:

Тип: Булево.   

- Истина - для Android при наличии телефонного модуля,
- Ложь - для остальных операционных систем.

Описание:

Определяет доступность журнала SMS-сообщений в используемой операционной системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[СредстваТелефонии](СредстваТелефонии.md), метод [ПолучитьЖурналSMS](СредстваТелефонии.ПолучитьЖурналSMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.
