ЧтениеPDF (PDFReader)

ПроверитьПодписьАсинх (VerifySignatureAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьПодписьАсинх(<Имя>, <Сертификат>, <МенеджерКриптографии>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя подписи для проверки.

<Сертификат> (необязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - сертификат типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), с помощью которого была произведена подпись (если сертификат включен в данные подписи).  
  
Если проверка подписи завершилась неудачно, генерируется исключение.

Описание:

Асинхронная функция. Проверяет действительность присоединенной к PDF-документу подписи с указанным именем.

Доступность:

Тонкий клиент, толстый клиент.
