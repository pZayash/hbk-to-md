СертификатКлиентаОС (OSClientCertificate)

По умолчанию

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый СертификатКлиентаОС()

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы. Сертификат подбирается автоматически.

Использование в версии:

Доступен, начиная с версии 8.3.21.
