ДокументHTML (HTMLDocument)

ПереименоватьУзел (RenameNode)

Доступен, начиная с версии 8.2.

Синтаксис:

ПереименоватьУзел(<Узел>, <КвалифицированноеИмя>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md), [ЭлементHTML](ЭлементHTML_(HTMLElement).md).   
Переименовываемый узел HTMLDOM.

<КвалифицированноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Новое квалифицируемое имя переименовываемого узла.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [ДокументDOM](ДокументDOM_(DOMDocument).md), [ЭлементDOM](ЭлементDOM_(DOMElement).md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [НотацияDOM](НотацияDOM_(DOMNotation).md), [СущностьDOM](СущностьDOM_(DOMEntity).md), [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md), [ТекстDOM](ТекстDOM_(DOMText).md), [КомментарийDOM](КомментарийDOM_(DOMComment).md), [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM_(DOMEntityReference).md), [ПространствоИменXPath](ПространствоИменXPath_(XPathNamespace).md).   

Описание:

Переименовывает узлы [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) и [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md).  
Выполняется попытка переименования существующего узла, в случае невозможности, создает новый узел с новым именем, префиксом и заменяет им исходный узел.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- попытка переименования узлов, отличных от [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) или [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), или в случае, если реализация не поддерживает переименование узлов;
- квалифицированное имя содержит символы, недопустимые стандартом XML данного документа;
- попытка переименования узла, созданного другим документом HTML;
- квалифицированное имя содержит префикс, значение которого недопустимо для данного значения URI пространства имен.

Использование в версии:

Доступен, начиная с версии 8.2.
