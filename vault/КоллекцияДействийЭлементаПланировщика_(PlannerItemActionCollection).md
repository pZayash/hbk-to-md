Доступен, начиная с версии 8.3.18.

Элементы коллекции:

[ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика_(PlannerItemAction).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются действия элемента планировщика.  
Возможно обращение к действию элемента планировщика посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию действий элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
