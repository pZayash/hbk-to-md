ПользовательИнформационнойБазы (InfoBaseUser)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор пользователя информационной базы.  
Для нового пользователя равен нулевому уникальному идентификатору.  
Устанавливается неявно после выполнения метода [Записать](ПользовательИнформационнойБазы.Записать.md) для нового пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
