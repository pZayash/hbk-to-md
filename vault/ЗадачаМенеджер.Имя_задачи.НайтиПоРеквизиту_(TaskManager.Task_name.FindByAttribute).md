ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

НайтиПоРеквизиту (FindByAttribute)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоРеквизиту(<ИмяРеквизита>, <ЗначениеРеквизита>)

Параметры:

<ИмяРеквизита> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя реквизита, как он задан в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и строк произвольной длины.

<ЗначениеРеквизита> (обязательный)

Тип: Произвольный.   
Значение реквизита, по которому должен выполняться поиск.

Возвращаемое значение:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Ссылка на найденную задачу. Если не существует ни одной задачи с требуемым значением реквизита, то будет возвращена пустая ссылка.

Описание:

Осуществляет поиск задачи по реквизиту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если существует несколько задач с указанным значением реквизита, то будет найден только один из них.  
Для реквизитов типа [Строка](lang__def_String.md) поиск осуществляется по точному соответствию.

Использование в версии:

Доступен, начиная с версии 8.0.
