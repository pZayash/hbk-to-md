# ДокументОбъект.<Имя документа>.Разблокировать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Разблокировать (Unlock)

Доступен, начиная с версии 8.0.

Синтаксис:

Разблокировать()

Описание:

Выполняет разблокировку документа, если он заблокирован этим объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| РасходнаяНакладная.Разблокировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
