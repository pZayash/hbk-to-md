РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

УстановитьВыделенныеЭлементы (SetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьВыделенныеЭлементы(<МассивЭлементов>)

Параметры:

<МассивЭлементов> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив выделяемых элементов.

Описание:

Устанавливает новое выделение. Элементы, тип которых отличается от [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md) , [ЗначениеДиаграммы](ЗначениеДиаграммы_(ChartValue).md) игнорируются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
