# СтрокиДинамическогоСписка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [Расширение для динамического списка](Расширение_для_динамического_списка.md)

Доступен, начиная с версии 8.3.10.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение_(KeyAndValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа [КлючИЗначение](КлючИЗначение_(KeyAndValue).md), где Ключ - ключ строки, Значение - [СтрокаДинамическогоСписка](СтрокаДинамическогоСписка_(DynamicListRow).md).  
Возможно обращение к значению элемента коллекции посредством оператора [...]. В качестве аргумента передается ключ строки.  

Методы:

[Количество (Count)](СтрокиДинамическогоСписка.Количество_(DynamicListRows.Count).md)  
[Получить (Get)](СтрокиДинамическогоСписка.Получить_(DynamicListRows.Get).md)  
[ПолучитьКлючи (GetKeys)](СтрокиДинамическогоСписка.ПолучитьКлючи_(DynamicListRows.GetKeys).md)  

Описание:

Представляет собой коллекцию строк динамического списка.  
К содержимому объекта можно обращаться только из обработчика [ПриПолученииДанныхНаСервере](Расширение_таблицы_формы_для_динамического_списка.ПриПолученииДанныхНаСервере_(Form_table_extension_for_dynamic_list.OnGetDataAtServer).md). По окончании работы обработчика попытка обращения к содержимому приведет к вызову исключения.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
