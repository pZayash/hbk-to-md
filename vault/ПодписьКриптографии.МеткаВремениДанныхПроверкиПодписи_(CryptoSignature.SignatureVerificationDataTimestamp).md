ПодписьКриптографии (CryptoSignature)

МеткаВремениДанныхПроверкиПодписи (SignatureVerificationDataTimestamp)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии_(CryptoTimestamp).md), [Неопределено](lang__def_Undefined.md).   
Содержит метку времени данных проверки подписи - необходимый для CAdES-X атрибут.  
Для [CAdESXType1](ТипПодписиКриптографии.CAdESXType1_(CryptoSignatureType.CAdESXType1).md) и [CAdESXType2](ТипПодписиКриптографии.CAdESXType2_(CryptoSignatureType.CAdESXType2).md) в данном поле хранятся различные метки времени.  
  
Если метка отсутствует значение поля - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
