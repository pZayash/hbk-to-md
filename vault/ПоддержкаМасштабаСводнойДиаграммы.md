Описание:

Содержит варианты способов поддержки масштаба (количества одновременно отображаемых значений) при изменении размеров области построения.  
Вариант [ВсеЗначения](ПоддержкаМасштабаСводнойДиаграммы.ВсеЗначения.md) предусматривает представление в области построения всех значений. Прокрутка не применяется.  
Вариант [КоличествоЗначений](ПоддержкаМасштабаСводнойДиаграммы.КоличествоЗначений.md) предусматривает представление области построения определенного количества значений, заданное соответствующим свойством. Если реальных значений больше, применяется прокрутка.  
Вариант [МинимальнаяШирина](ПоддержкаМасштабаДендрограммы.МинимальнаяШирина.md) предусматривает задание минимальной ширины значения (в символах). Если размер области построения не позволяет отобразить все значения с шириной каждого, не меньше заданной, применяется прокрутка.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Доступен, начиная с версии 8.0.
