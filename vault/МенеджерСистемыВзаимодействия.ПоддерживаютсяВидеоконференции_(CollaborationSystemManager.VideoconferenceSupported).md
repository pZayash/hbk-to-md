МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПоддерживаютсяВидеоконференции (VideoconferenceSupported)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПоддерживаютсяВидеоконференции()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если есть возможность совершать видеозвонки.

Описание:

Определяет возможность совершать видеозвонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
