ТабличноеПоле (TableBox)

РазрешитьПеретаскивание (EnableDrag)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешает начало операции перетаскивания в табличном поле - приемнике данных.

Доступность:

Толстый клиент.

Примечание:

Истина - перетаскивание разрешено; Ложь - запрещено.

Использование в версии:

Доступен, начиная с версии 8.0.
