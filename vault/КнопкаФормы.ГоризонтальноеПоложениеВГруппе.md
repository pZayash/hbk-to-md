КнопкаФормы (FormButton)

ГоризонтальноеПоложениеВГруппе (HorizontalAlignInGroup)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента.md).   
Устанавливает горизонтальное выравнивание элемента внутри группы. Значение [Авто](ГоризонтальноеПоложениеЭлемента.Авто.md) означает, что будет использоваться значение свойства [ГоризонтальноеПоложениеВГруппе](ГруппаФормы.ГоризонтальноеПоложениеВГруппе.md) родительской группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство используется только для кнопок, расположенных непосредственно на форме, вне командной панели.

Использование в версии:

Доступен, начиная с версии 8.3.6.
