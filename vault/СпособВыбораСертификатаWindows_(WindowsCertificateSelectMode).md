# СпособВыбораСертификатаWindows

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интернет](Интернет.md)

Доступен, начиная с версии 8.3.1.

Значения

[Авто (Auto)](СпособВыбораСертификатаWindows.Авто_(WindowsCertificateSelectMode.Auto).md)  
[Выбирать (Choose)](СпособВыбораСертификатаWindows.Выбирать_(WindowsCertificateSelectMode.Choose).md)  

Описание:

Содержит варианты способов выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы MS Windows.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

См. также:

[СертификатКлиентаWindows](СертификатКлиентаWindows_(WindowsClientCertificate).md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.24.
