ОбъектМетаданных: Константа (MetadataObject: Constant)

ВыборГруппИЭлементов (ChoiceFoldersAndItems)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов_(FoldersAndItemsUse).md).   
Устанавливает вид данных, которые могут быть использованы в качестве значения для константы и устанавливается в значение: Элементы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
