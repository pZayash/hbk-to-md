Лицензия (ILicenseInfo)

MaxUsersAll (MaxUsersAll)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит максимальное количество пользователей, допустимое для данного комплекта, если используется программная лицензия платформы. Иначе совпадает с значением свойства [MaxUsersCur](Лицензия.MaxUsersCur.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
