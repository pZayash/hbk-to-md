ОбъектМетаданных: Интерфейс (MetadataObject: Interface)

Переключаемый (Switchable)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то можно управлять отображением данного интерфейса. В противном случае данный интерфейс будет показываться всегда.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
