Флажок (CheckBox)

ТриСостояния (ThreeState)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что флажок может иметь три возможных состояния. При этом типом редактируемого значения является [Число](lang__def_Number.md).  
[Истина](lang__def_BooleanTrue.md) - состояние разрешено.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
