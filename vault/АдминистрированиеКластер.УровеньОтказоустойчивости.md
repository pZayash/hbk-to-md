Доступен, начиная с версии 8.3.14.

Чтение и запись.

Описание:

Тип: Число.   
Определяет уровень отказоустойчивости кластера - количество рабочих серверов кластера, выход из строя которых не приводит к аварийному завершению сеансов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
