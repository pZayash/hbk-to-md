РезультатXPath (XPathResult)

ПолучитьСледующий (IterateNext)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьСледующий()

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md), Неопределено.   
Неопределено, если следующего узла нет.

Описание:

Получает следующий узел результата.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если тип узла не равен [НеупорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.НеупорядоченныйСнимокУзлов.md) или [УпорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.УпорядоченныйСнимокУзлов.md), то будет вызвано исключение.  
Если итератор имеет недействительное состояние, то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
