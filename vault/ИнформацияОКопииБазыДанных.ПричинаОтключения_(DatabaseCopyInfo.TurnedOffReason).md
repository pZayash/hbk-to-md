ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ПричинаОтключения (TurnedOffReason)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ПричинаОтключенияКопииБазыДанных](ПричинаОтключенияКопииБазыДанных_(DatabaseCopyTurnedOffReason).md), [Неопределено](lang__def_Undefined.md).   
Если копия базы данных может выполнять запросы, то данное свойство имеет значение [Неопределено](lang__def_Undefined.md). В противном случае - причина отключения копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
