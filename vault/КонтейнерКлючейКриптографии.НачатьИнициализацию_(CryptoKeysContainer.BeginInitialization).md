# КонтейнерКлючейКриптографии.НачатьИнициализацию

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Криптография](Криптография-2.md) › [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md) › [Методы](КонтейнерКлючейКриптографии__Методы.md)

КонтейнерКлючейКриптографии (CryptoKeysContainer)

НачатьИнициализацию (BeginInitialization)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяФайла>, <Пароль>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения загрузки контейнера ключей криптографии из файла со следующими параметрами:

- КонтейнерКлючейКриптографии - созданный [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md).
- ДополнительныеПараметры – значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из данных которого будет инициализирован контейнер ключей криптографии.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль контейнера ключей криптографии.

Описание:

Начинает инициализацию контейнера.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md), метод [ИнициализироватьАсинх](КонтейнерКлючейКриптографии.ИнициализироватьАсинх_(CryptoKeysContainer.InitializeAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
