РазмерСтраницы (PageSize)

Доступен, начиная с версии 8.2.

Примечание:

Если используется известный платформе вариант, то в качестве размеров страницы содержатся размеры известного платформе формата. Если используется вариант Custom, то в качестве размеров страницы содержатся размеры, указанные разработчиком. Если используется вариант, отличный от известного и не являющийся Custom, то в качестве размеров страницы содержатся размеры для формата А4.

Использование в версии:

Доступен, начиная с версии 8.2.
