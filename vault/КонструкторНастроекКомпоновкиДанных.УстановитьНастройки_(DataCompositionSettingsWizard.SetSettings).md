КонструкторНастроекКомпоновкиДанных (DataCompositionSettingsWizard)

УстановитьНастройки (SetSettings)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьНастройки(<Настройки>, <ОписаниеОшибки>)

Параметры:

<Настройки> (обязательный)

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Редактируемый объект.

<ОписаниеОшибки> (обязательный)

Тип: [Строка](lang__def_String.md).   
После выполнения метода параметр содержит строку с описанием уменьшения функциональности схемы. Если ошибок нет, параметр содержит пустую строку.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Ложь](lang__def_BooleanFalse.md) - есть возможность редактирования настроек без ограничения их функциональности. [Истина](lang__def_BooleanTrue.md), если придется усекать функциональность настроек.

Описание:

Устанавливает настройки в конструктор и проверяет возможность редактирования этих установок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
