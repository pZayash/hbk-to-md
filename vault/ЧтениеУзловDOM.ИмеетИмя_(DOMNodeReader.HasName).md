ЧтениеУзловDOM (DOMNodeReader)

ИмеетИмя (HasName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак наличия у текущего узла имени.  
[Истина](lang__def_BooleanTrue.md) - для узла определено имя, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
