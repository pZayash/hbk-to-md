ВнешниеОтчетыМенеджер (ExternalReportsManager)

Подключить (Connect)

Доступен, начиная с версии 8.2.

Синтаксис:

Подключить(<Путь>, <Имя>, <БезопасныйРежим>, <ЗащитаОтОпасныхДействий>, <КодЯзыка>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ссылка на реквизит объекта конфигурации, который содержит данные внешней обработки. Ссылка может быть получена методом [ПолучитьНавигационнуюСсылку](Глобальный_контекст.ПолучитьНавигационнуюСсылку_(Global_context.GetURL).md).

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя внешнего отчета, с которым он будет зарегистрирован в системе.  
Наличие одинаковых имен в процессе работы сеанса не допускается. Если имя не указано, будет использовано имя, содержащееся в самом отчете.

<БезопасныйРежим> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Строка](lang__def_String.md).   
Признак подключения внешнего отчета в безопасном режиме. Если параметр не указан, подключение будет осуществлено в безопасном режиме.  
Если указана строка, то внешняя обработка будет исполняться в безопасном режиме под управлением профиля безопасности с этим именем. Профиль безопасности должен иметь [SafeModeProfile](Профиль_безопасности.SafeModeProfile_(ISecurityProfile.SafeModeProfile).md)=[Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ЗащитаОтОпасныхДействий> (необязательный)

Тип: [ОписаниеЗащитыОтОпасныхДействий](ОписаниеЗащитыОтОпасныхДействий_(UnsafeOperationProtectionDescription).md).   
Структура, описывающая параметры защиты от опасных действий для подключаемого модуля.

<КодЯзыка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Код языка внешнего отчета, который должен быть использован при подключении.  
Если параметр не указан, то используется текущий язык аутентифицированного пользователя (т.е. код языка, возвращаемый методом [ТекущийЯзык](Глобальный_контекст.ТекущийЯзык_(Global_context.CurrentLanguage).md) глобального контекста).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Выполняет подключение внешнего отчета по указанному пути. В качестве пути может использоваться только путь к данным в конфигурации и базе данных.  
Возвращает имя подключенного внешнего отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
