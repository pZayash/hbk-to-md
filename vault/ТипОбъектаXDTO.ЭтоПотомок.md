Доступен, начиная с версии 8.1.

Синтаксис:

ЭтоПотомок(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [ТипОбъектаXDTO](ТипОбъектаXDTO.md).   
Проверяемый тип.

Возвращаемое значение:

Тип: Булево.   
Истина - переданный тип является потомком данного, Ложь - в противном случае.

Описание:

Определяет, является ли переданный тип значения потомком данного типа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
