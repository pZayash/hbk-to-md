ИнтернетПочтовоеСообщение (InternetMailMessage)

ПолучитьИсходныеДанные (GetSourceData)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИсходныеДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Возвращает исходные данные почтового сообщения.  
Если сообщение принято с сервера (с помощью метода [Выбрать](ИнтернетПочта.Выбрать_(InternetMail.Get).md) объекта [ИнтернетПочта](ИнтернетПочта_(InternetMail).md)) и не было изменено, то возвращаемые методом данные соответствуют тем, которые были получены с сервера.  
Если объект [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md) создан программно или получен с сервера и был изменен, то данные генерируются на основе содержимого этого объекта, в том виде, в каком они готовились к передаче почтовому серверу при отправке.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Полученные данные являются не аналогом сериализованных данных, а представлением сообщения в том виде, как оно передается почтовому серверу при отправке. В частности, полученные данные не будут содержать информацию о слепых копиях, поскольку эти данные на сервер не передаются.  
Не гарантируется эквивалентность исходного письма и копии, созданной с помощью метода [УстановитьИсходныеДанные](ИнтернетПочтовоеСообщение.УстановитьИсходныеДанные_(InternetMailMessage.SetSourceData).md) на основании этих данных.  
В случае ошибки разбора почтового сообщения, можно использовать метод для попытки восстановить содержимое письма вручную.

Использование в версии:

Доступен, начиная с версии 8.3.5.
