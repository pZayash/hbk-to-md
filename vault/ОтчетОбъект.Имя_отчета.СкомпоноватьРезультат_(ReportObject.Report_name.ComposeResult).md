# ОтчетОбъект.<Имя отчета>.СкомпоноватьРезультат

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Отчеты](Отчеты.md) › [ОтчетОбъект.<Имя отчета>](ОтчетОбъект.Имя_отчета_(ReportObject.Report_name).md) › [Методы](ОтчетОбъект.Имя_отчета__Методы.md)

ОтчетОбъект.<Имя отчета> (ReportObject.<Report name>)

СкомпоноватьРезультат (ComposeResult)

Доступен, начиная с версии 8.1.

Синтаксис:

СкомпоноватьРезультат(<Результат>, <ИнформацияРасшифровки>)

Параметры:

<Результат> (обязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Табличный документ, в который будет выводиться результат.

<ИнформацияРасшифровки> (необязательный)

Переменная, в которую будет записана информация расшифровки.  
Если не указано - расшифровка заполняться не будет.

Описание:

Выполняет компоновку.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Пользователь может при помощи комбинации клавиш Ctrl + Break прервать выполнение отчета. При этом будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
