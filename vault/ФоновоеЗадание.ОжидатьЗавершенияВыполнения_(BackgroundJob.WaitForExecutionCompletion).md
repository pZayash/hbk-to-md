ФоновоеЗадание (BackgroundJob)

ОжидатьЗавершенияВыполнения (WaitForExecutionCompletion)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОжидатьЗавершенияВыполнения(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Таймаут в секундах ожидания завершения задания. Если таймаут не определен, ожидание будет длиться до момента завершения задания.

Возвращаемое значение:

Тип: [ФоновоеЗадание](ФоновоеЗадание_(BackgroundJob).md).   
Возвращает обновленный объект типа [ФоновоеЗадание](ФоновоеЗадание_(BackgroundJob).md) для текущего задания при изменения состояния фонового заданий или по истечении таймаута.

Описание:

Ожидает успешного завершения задания. Если задание завершено, завершено аварийно, отменено, либо наступил таймаут, то ожидание прерывается и возвращается обновленный объект [ФоновоеЗадание](ФоновоеЗадание_(BackgroundJob).md).   
Ожидать завершения задания может только администратор или пользователь, запустивший задание.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Задание = Задание.ОжидатьЗавершенияВыполнения(10); Пока Задание.Состояние = СостояниеФоновогоЗадания.Активно Цикл     Задание = Задание.ОжидатьЗавершенияВыполнения(10); КонецЦикла; // фоновое задание завершено |

Использование в версии:

Доступен, начиная с версии 8.3.13.
