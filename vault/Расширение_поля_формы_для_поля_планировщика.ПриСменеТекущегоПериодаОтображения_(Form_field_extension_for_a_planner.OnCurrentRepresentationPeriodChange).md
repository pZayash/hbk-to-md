Расширение поля формы для поля планировщика (Form field extension for a planner)

ПриСменеТекущегоПериодаОтображения (OnCurrentRepresentationPeriodChange)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПриСменеТекущегоПериодаОтображения(<ТекущиеПериодыОтображения>, <СтандартнаяОбработка>)

Параметры:

<ТекущиеПериодыОтображения>

Тип: [Массив](Массив_(Array).md).   
Массив текущих периодов отображения. Элементы массива имеют тип [ПериодОтображенияПланировщика](ПериодОтображенияПланировщика_(PlannerRepresentationPeriod).md):

- В мобильной платформе массив содержит либо два периода (когда следующий период отображается при прокрутке), либо один (когда единовременно при прокрутке отображается только один период).
- В платформе для ПК массив содержит только один период.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), то смены периода отображения производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при смене периода отображения (при прокрутке с помощью колеса мыши или команд).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
