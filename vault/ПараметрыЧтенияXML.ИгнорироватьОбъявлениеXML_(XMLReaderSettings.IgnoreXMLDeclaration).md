ПараметрыЧтенияXML (XMLReaderSettings)

ИгнорироватьОбъявлениеXML (IgnoreXMLDeclaration)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак игнорирования объявления XML при чтении.  
[Истина](lang__def_BooleanTrue.md) - игнорировать. Значение по умолчанию [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
