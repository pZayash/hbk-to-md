Глобальный контекст (Global context)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Строка>, <ПодстрокаПоиска>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Исходная строка.

<ПодстрокаПоиска> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомая подстрока.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Позиция первого знака найденной подстроки. Нумерация символов в строке начинается с 1. Если строка не содержит указанной подстроки, то возвращается 0.

Описание:

Функция является устаревшей! Рекомендуется использовать функцию [СтрНайти](Глобальный_контекст.СтрНайти_(Global_context.StrFind).md).  
Находит первое вхождение искомой строки как подстроки в исходной строке. Сравнение выполняется с учетом регистра.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

Если параметр <ПодстрокаПоиска> - пустая строка, то возвращает 1.  
Если оба параметра - пустые строки, то возвращает 1.

Пример:

|  |
| --- |
| Если Найти(Сотрудник.Имя, "Борис") > 0 Тогда     Сообщить(Сотрудник.Имя + " таб. №" + Сотрудник.Код); КонецЕсли; |
