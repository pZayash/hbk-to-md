# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПриВыполнении

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриВыполнении (OnExecute)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриВыполнении(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, в которой осуществляется проверка выполнения.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Выполняемая задача.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то задача не будет выполнена.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает в процессе выполнения метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу_(TaskObject.Task_name.ExecuteTask).md) у соответствующей задачи и после обработки задачей события [ПриВыполнении](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПриВыполнении_(BusinessProcessRoutePointRef.Business_process_name.OnExecute).md). Если параметру <Отказ> установлено значение [Истина](lang__def_BooleanTrue.md), то выполнение метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу_(TaskObject.Task_name.ExecuteTask).md) также прерывается. Задача не будет выполнена и выполнение бизнес-процесса не пойдет дальше по маршруту.   
Этот обработчик позволяет, например, проверить выполнение необходимых условий выполнения задачи (проведение документа, установка скидки по счету, отметка о согласовании и пр.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md).  
Вызывается только при выполнении последней задачи, соответствующей данной точке маршрута данного бизнес-процесса.

Использование в версии:

Доступен, начиная с версии 8.0.
