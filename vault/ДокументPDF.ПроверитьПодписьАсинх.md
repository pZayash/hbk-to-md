ДокументPDF (PDFDocument)

ПроверитьПодписьАсинх (VerifySignatureAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПроверитьПодписьАсинх(<Имя>, <МенеджерКриптографии>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя подписи для проверки.

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии.md).   
Менеджер криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - сертификат типа [СертификатКриптографии](СертификатКриптографии.md), с помощью которого была произведена подпись (если сертификат включен в данные подписи).

Описание:

Асинхронная функция. Проверяет действительность присоединенной к PDF-документу подписи с указанным именем.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если проверка подписи завершилась неудачно, генерируется исключение категории [ОшибкаПроверкиПодписи](КатегорияОшибки.ОшибкаПроверкиПодписи.md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
