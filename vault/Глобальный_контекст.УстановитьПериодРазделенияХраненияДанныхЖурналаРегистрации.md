Глобальный контекст (Global context)

УстановитьПериодРазделенияХраненияДанныхЖурналаРегистрации (SetEventLogDataStorageSplitPeriod)

Доступен, начиная с версии 8.3.20.

Синтаксис:

УстановитьПериодРазделенияХраненияДанныхЖурналаРегистрации(<ПериодРазделенияХраненияДанных>)

Параметры:

<ПериодРазделенияХраненияДанных> (обязательный)

Тип: [ПериодРазделенияХраненияДанныхЖурналаРегистрации](ПериодРазделенияХраненияДанныхЖурналаРегистрации.md).   
Период разделения хранения данных журнала регистрации.

Описание:

Устанавливает новый период разделения хранения данных журнала регистрации.   
Требуется наличие у текущего пользователя права "Администрирование".  
На время исполнения устанавливается режим монопольного доступа к информационной базе. Может использоваться только для журнала регистрации в последовательном формате. При любой ошибке генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьПериодРазделенияХраненияДанныхЖурналаРегистрации](Глобальный_контекст.ПолучитьПериодРазделенияХраненияДанныхЖурналаРегистрации.md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
