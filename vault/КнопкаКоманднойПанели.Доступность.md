КнопкаКоманднойПанели (CommandBarButton)

Доступность (Enabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак доступности кнопки командной панели.   
Истина - доступна.  
Может использоваться для кнопок с типом [Действие](ТипКнопкиКоманднойПанели.Действие.md).  
Если кнопка панели запрещена системой, значение свойства игнорируется.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
