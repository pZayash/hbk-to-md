ЭлементФайлаАрхива (ArchiveFileEntry)

ИсходноеПолноеИмя (OriginalFullName)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, включая путь.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Без режима совместимости содержит части пути в значении свойства разделены в зависимости от используемой операционной системы:

- В Linux элементы пути разделяются символом "/".
- В Windows элементы пути разделяются символом"\".
- В режиме совместимости [Версия8\_3\_8](РежимСовместимости.Версия8_3_8_(CompatibilityMode.Version8_3_8).md) и ниже части пути разделены символом "/".

В Linux элементы пути разделяются символом "/".  
В Windows элементы пути разделяются символом"\".


- В режиме совместимости [Версия8\_3\_8](РежимСовместимости.Версия8_3_8_(CompatibilityMode.Version8_3_8).md) и ниже части пути разделены символом "/".

В режиме совместимости [Версия8\_3\_8](РежимСовместимости.Версия8_3_8_(CompatibilityMode.Version8_3_8).md) и ниже части пути разделены символом "/".

Использование в версии:

Доступен, начиная с версии 8.3.26.
