# WSКоллекцияОпераций

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Web-сервисы](Web-сервисы-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[WSОперация](WSОперация_(WSOperation).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) коллекции.  

Методы:

[Количество (Count)](WSКоллекцияОпераций.Количество_(WSOperationCollection.Count).md)  
[Получить (Get)](WSКоллекцияОпераций.Получить_(WSOperationCollection.Get).md)  

Описание:

Коллекция операций веб-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[WSИнтерфейс](WSИнтерфейс_(WSInterface).md), свойство [Операции](WSИнтерфейс.Операции_(WSInterface.Operations).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
