Почта (Mail)

Отключиться (Logoff)

Доступен, начиная с версии 8.0.

Синтаксис:

Отключиться()

Описание:

Производит отключение от почтового сервера. Посылка и получение сообщений с сервера не доступны.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

|  |
| --- |
| Почта = Новый Почта; Почта.Подключиться(); Почта.Послать(Сообщение); Почта.Отключиться(); |

Использование в версии:

Доступен, начиная с версии 8.0.
