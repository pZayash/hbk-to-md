ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриПовторномОткрытии (OnReopen)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриПовторномОткрытии()

Описание:

Возникает на клиенте при повторном открытии формы. Повторное открытие возникает при вызове метода [Открыть](ФормаКлиентскогоПриложения.Открыть_(ClientApplicationForm.Open).md) для уже открытой формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
