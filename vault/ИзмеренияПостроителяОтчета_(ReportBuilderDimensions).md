Доступен, начиная с версии 8.1.

Элементы коллекции:

[ИзмерениеПостроителяОтчета](ИзмерениеПостроителяОтчета_(ReportBuilderDimension).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются измерения.  
Возможно обращение к измерению посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
