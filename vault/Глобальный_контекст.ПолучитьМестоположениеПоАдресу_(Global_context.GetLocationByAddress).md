Глобальный контекст (Global context)

ПолучитьМестоположениеПоАдресу (GetLocationByAddress)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМестоположениеПоАдресу(<Адрес>)

Параметры:

<Адрес> (необязательный)

Тип: [ДанныеАдреса](ДанныеАдреса_(AddressData).md).   
Адрес местоположения. Состав адреса может различаться в разных государствах.  
Для России определены поля:

- Страна (Country);
- Регион (State);
- Город (City);
- Улица (Street);
- Дом (House);
- Индекс (ZipCode).

Возвращаемое значение:

Тип: [ГеографическиеКоординаты](ГеографическиеКоординаты_(GeographicCoordinates).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - данные не получены.

Описание:

Получает географические координаты по адресу.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
