НастройкиКомпоновкиДанных (DataCompositionSettings)

Порядок (Order)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных.md).   
Содержит порядок записей, выводимых в результат. Используется как порядок “по умолчанию” для всех элементов компоновки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
