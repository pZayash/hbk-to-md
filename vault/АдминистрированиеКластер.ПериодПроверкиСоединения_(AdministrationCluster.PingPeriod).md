АдминистрированиеКластер (AdministrationCluster)

ПериодПроверкиСоединения (PingPeriod)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет период отправки проверочных запросов системы отслеживания соединений, в миллисекундах. Допустимые значения от 0 до 65535.  
Если данное свойство и свойство [ТаймаутПроверкиСоединения](АдминистрированиеКластер.ТаймаутПроверкиСоединения_(AdministrationCluster.PingTimeout).md) имеют значения 0, то используются параметры системы отслеживания соединений, установленные с помощью командной строки.  
Значение свойства должно быть установлено вместе со значением свойства [ТаймаутПроверкиСоединения](АдминистрированиеКластер.ТаймаутПроверкиСоединения_(AdministrationCluster.PingTimeout).md).  
Значения по умолчанию:

- ПериодПроверкиСоединения - 1000 мс,
- ТаймаутПроверкиСоединения - 5000 мс.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
