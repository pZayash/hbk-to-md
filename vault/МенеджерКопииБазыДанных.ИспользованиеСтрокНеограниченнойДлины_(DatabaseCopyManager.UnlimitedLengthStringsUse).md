МенеджерКопииБазыДанных (DatabaseCopyManager)

ИспользованиеСтрокНеограниченнойДлины (UnlimitedLengthStringsUse)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеПоляЭлементаСоставаКопииБазыДанных](ИспользованиеПоляЭлементаСоставаКопииБазыДанных_(DatabaseCopyContentItemFieldUse).md).   
Определяет режим использования по умолчанию для полей типа [Строка](lang__def_String.md) неограниченной длины.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
