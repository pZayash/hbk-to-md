WSПрокси (WSProxy)

Таймаут (Timeout)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет время ожидания соединения и операций, осуществляемых объектом WSПрокси, в секундах.  
0 - таймаут не установлен.  
На мобильной платформе таймаут по умолчанию составляет 5 минут.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
