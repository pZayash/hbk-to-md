СоединениеССерверомСистемыАналитики (AnalyticsSystemServerConnection)

ПолучитьСхему (GetSchema)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьСхему(<Путь>, <Отбор>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь для получения схемы.

<Отбор> (необязательный)

Тип: [Структура](Структура_(Structure).md), [Строка](lang__def_String.md).   
Если параметр имеет тип [Строка](lang__def_String.md), то представляет идентификатор передаваемого объекта.   
Если параметр имеет тип [Структура](Структура_(Structure).md), то представляет набор полей и их значений по которым будет осуществляться отбор объектов из системы аналитики.

Возвращаемое значение:

Тип: [СхемаСистемыАналитики](СхемаСистемыАналитики_(AnalyticsSystemSchema).md).   

Описание:

Получает схему по переданному идентификатору или отбору в рамках пути системы аналитики.  
Содержание параметров определяется сервером системы аналитики и описано в его документации.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
