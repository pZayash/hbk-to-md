# СписокЗначений.НайтиПоИдентификатору

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Список значений](Список_значений.md) › [СписокЗначений](СписокЗначений_(ValueList).md) › [Методы](СписокЗначений__Методы.md)

СписокЗначений (ValueList)

НайтиПоИдентификатору (FindByID)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Число](lang__def_Number.md).   
Идентификатор элемента списка значений.

Возвращаемое значение:

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - элемент не найден.

Описание:

Получает элемент списка значений по идентификатору.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
