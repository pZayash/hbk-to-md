ФормаКлиентскогоПриложения (ClientApplicationForm)

НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Задает навигационную ссылку, возвращаемую формой.  
Если [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка.md)=Ложь, метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md) объекта [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md) возвращает значение свойства НавигационнаяСсылка основной формы.  
Если [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка.md)=Истина, метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md) возвращает навигационную ссылку основной формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.
