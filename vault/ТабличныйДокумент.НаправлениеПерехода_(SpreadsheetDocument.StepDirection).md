ТабличныйДокумент (SpreadsheetDocument)

НаправлениеПерехода (StepDirection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипНаправленияПереходаТабличногоДокумента](ТипНаправленияПереходаТабличногоДокумента_(SpreadsheetDocumentStepDirectionType).md).   
Определяет направление перехода при завершении редактирования ячейки таблицы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
