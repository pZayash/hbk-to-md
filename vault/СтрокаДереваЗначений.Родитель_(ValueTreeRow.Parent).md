СтрокаДереваЗначений (ValueTreeRow)

Родитель (Parent)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md).   
Содержит строку-родителя для данной строки дерева значений. Если строка находится в корне дерева (на верхнем уровне иерархии), то свойство имеет значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
