HTTPСервисОтвет (HTTPServiceResponse)

ПолучитьИмяФайлаТела (GetBodyFileName)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИмяФайлаТела()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает имя файла, из которого будет прочитано тело HTTP-ответа.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
