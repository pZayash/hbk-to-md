РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, необходимо ли выделять рисунок, в котором не заполнено значение, либо оно выставлено в значение по умолчанию.   
Если нет заполненного текста, будет установлено [ОтметкаНезаполненного](РисунокТабличногоДокумента.ОтметкаНезаполненного_(SpreadsheetDocumentDrawing.MarkIncomplete).md) в [Истина](lang__def_BooleanTrue.md), иначе [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо только для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md), [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.0.
