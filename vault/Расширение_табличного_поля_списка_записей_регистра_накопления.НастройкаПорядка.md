Расширение табличного поля списка записей регистра накопления (Accumulation register record list table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаПорядка](НастройкаПорядка.md).   
Предоставляет доступ к настройкам порядка списка записей регистра накопления. Представляет собой коллекцию значений типа [ЭлементУправленияПорядком](ЭлементУправленияПорядком.md). Свойства коллекции содержат объекты для управления элементом порядка. Имена свойств совпадают с именами доступных для регистра накопления элементов упорядочивания.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
