СертификатКриптографии (CryptoCertificate)

ДатаОкончания (ValidTo)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату окончания действия сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При чтении полей сертификата платформа использует универсальное время.

Использование в версии:

Доступен, начиная с версии 8.2.
