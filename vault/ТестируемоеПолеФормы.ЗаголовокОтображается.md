ТестируемоеПолеФормы (TestedFormField)

ЗаголовокОтображается (TitleIsShown)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ЗаголовокОтображается()

Возвращаемое значение:

Тип: Булево.   
Истина - заголовок отображается.  
Ложь - заголовок не отображается.

Описание:

Возвращает признак видимости заголовка для указанного поля.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
