ВыраженияПорядкаСхемыЗапроса (QuerySchemaOrderExpressions)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [Число](lang__def_Number.md), [ВыражениеПорядкаСхемыЗапроса](ВыражениеПорядкаСхемыЗапроса_(QuerySchemaOrderExpression).md).   
Индекс элемента или сам элемент, который требуется сдвинуть.  
В режиме совместимости с [Версия8\_3\_10](РежимСовместимости.Версия8_3_10_(CompatibilityMode.Version8_3_10).md) и ниже тип элемента может быть только [Число](lang__def_Number.md).

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое нужно сдвинуть указанный элемент. Положительное смещение обозначает сдвиг в конец, отрицательное - в начало коллекции.

Описание:

Сдвигает элемент в коллекции на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме совместимости [Версия8\_3\_10](РежимСовместимости.Версия8_3_10_(CompatibilityMode.Version8_3_10).md) и ниже перемещает элемент коллекции в указанную позицию.

Использование в версии:

Доступен, начиная с версии 8.3.5.
