Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

МаксимальнаяШирина (MaxWidth)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит максимальную ширину элемента.  
Используется, если свойство [АвтоМаксимальнаяШирина](Расширение_поля_формы_для_поля_PDF-документа.АвтоМаксимальнаяШирина.md) установлено в значение Ложь.  
Ложь.  
Задается в символах.  
Значение 0 означает, что максимальная ширина не ограничена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.21.
