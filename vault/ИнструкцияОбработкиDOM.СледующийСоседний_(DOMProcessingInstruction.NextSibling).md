ИнструкцияОбработкиDOM (DOMProcessingInstruction)

СледующийСоседний (NextSibling)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [Неопределено](lang__def_Undefined.md).   
Следующий соседний узел DOM или [Неопределено](lang__def_Undefined.md), если такового нет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
