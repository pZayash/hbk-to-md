СтандартнаяКомандаПоляПланировщика (PlannerStandardCommand)

КопироватьНавигационнуюСсылку (CopyURL)

Доступен, начиная с версии 8.3.23.

Описание:

Копировать ссылку в буфер обмена.  
Используется только для источника вида [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка.md).  
В контекстном меню команда будет иметь представление "Копировать ссылку".

Использование в версии:

Доступен, начиная с версии 8.3.23.
