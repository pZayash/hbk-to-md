КонтекстПространствИменXML (XMLNamespaceContext)

СоответствияПространствИмен (NamespaceMappings)

Доступен, начиная с версии 8.1.

Синтаксис:

СоответствияПространствИмен()

Возвращаемое значение:

Тип: [Соответствие](Соответствие.md).   

Описание:

Получает префиксы и соответствующие им URI пространств имен, определенные в текущем элементе XML, в рамках которого находится текущая позиция объекта [ЧтениеXML](ЧтениеXML.md).  
Ключом соответствия является значение префикса, значением - URI пространства имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
