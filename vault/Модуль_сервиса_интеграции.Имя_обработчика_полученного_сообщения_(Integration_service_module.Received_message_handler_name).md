Модуль сервиса интеграции (Integration service module)

<Имя обработчика полученного сообщения> (<Received message handler name>)

Доступен, начиная с версии 8.3.17.

Синтаксис:

<Имя обработчика полученного сообщения>(<Сообщение>, <Отказ>)

Параметры:

<Сообщение>

Тип: [СообщениеСервисаИнтеграции](СообщениеСервисаИнтеграции_(IntegrationServiceMessage).md).   
Содержит сообщение сервиса интеграции.

<Отказ>

Признак отмены выполненных в информационной базе изменений.  
Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то изменения, выполненные в информационной базе, отменяются, обработчик будет вызван повторно.

Описание:

Обрабатывает получаемые каналом сервиса интеграции сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
