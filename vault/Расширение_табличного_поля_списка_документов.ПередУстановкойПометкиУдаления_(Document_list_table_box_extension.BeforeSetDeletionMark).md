Расширение табличного поля списка документов (Document list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак изменения пометки удаления документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке документов.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ДокументОбъект.Имя_документа.ПередЗаписью_(DocumentObject.Document_name.BeforeWrite).md) и [ПриЗаписи](ДокументОбъект.Имя_документа.ПриЗаписи_(DocumentObject.Document_name.OnWrite).md) объекта [ДокументОбъект.](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)<[Имя документа](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
