РезультатXPath (XPathResult)

СтроковоеЗначение (StringValue)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полученное строковое значение результата.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение не может быть преобразовано к строковому типу, то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
