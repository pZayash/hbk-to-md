КоллекцияЭлементовПланировщика (PlannerItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению элемента планировщика.

Возвращаемое значение:

Тип: [ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), [Неопределено](lang__def_Undefined.md);.   
Если указанный элемент отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента планировщика по указаному значению в свойстве [Значение](ЭлементПланировщика.Значение_(PlannerItem.Value).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), свойство [Значение](ЭлементПланировщика.Значение_(PlannerItem.Value).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
