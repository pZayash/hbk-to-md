РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

ВыбратьПоРегистратору (SelectByRecorder)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьПоРегистратору(<Регистратор>)

Параметры:

<Регистратор> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Ссылка на документ, являющийся регистратором для записей регистра.

Возвращаемое значение:

Тип: [РегистрНакопленияВыборка.\<Имя регистра накопления\>](РегистрНакопленияВыборка.Имя_регистра_накопления_(AccumulationRegisterSelection.Accumulation_register_name).md).   

Описание:

Формирует выборку по регистратору записей регистра накопления.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Движения = РегистрыНакопления.Продажи.ВыбратьПоРегистратору(Ссылка); |

Использование в версии:

Доступен, начиная с версии 8.0.
