ЧередованиеРасположенияСтраниц (PagePlacementAlternation)

Авто (Auto)

Доступен, начиная с версии 8.3.1.

Описание:

Если свойство ДвусторонняяПечать (для [ТабличныйДокумент](ТабличныйДокумент.md) или [ГрафическаяСхема](ГрафическаяСхема.md)) имеет значение [Нет](ТипДвустороннейПечати.Нет.md), то используется обычный вывод полей и колонтитулов, если ДвусторонняяПечать имеет значение [ПереворотВлево](ТипДвустороннейПечати.ПереворотВлево.md), то используется вывод со значением [ЗеркальноСлева](ЧередованиеРасположенияСтраниц.ЗеркальноСлева.md), если ДвусторонняяПечать имеет значение [ПереворотВверх](ТипДвустороннейПечати.ПереворотВверх.md), то используется вывод со значением [ЗеркальноСверху](ЧередованиеРасположенияСтраниц.ЗеркальноСверху.md).

Использование в версии:

Доступен, начиная с версии 8.3.1.
