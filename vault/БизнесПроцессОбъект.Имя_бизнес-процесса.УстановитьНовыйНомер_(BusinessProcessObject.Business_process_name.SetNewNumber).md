# БизнесПроцессОбъект.<Имя бизнес-процесса>.УстановитьНовыйНомер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [БизнесПроцессОбъект.<Имя бизнес-процесса>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md) › [Методы](БизнесПроцессОбъект.Имя_бизнес-процесса__Методы.md)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

УстановитьНовыйНомер (SetNewNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьНовыйНомер(<ПрефиксНомера>)

Параметры:

<ПрефиксНомера> (необязательный)

Тип: [Строка](lang__def_String.md).   
Префикс устанавливаемого номера.  
Значение по умолчанию: Пустая строка.

Описание:

Устанавливает новый номер бизнес-процесса для заданного префикса номера.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для бизнес-процессов с нумерацией в пределах периода, номер устанавливается в соответствии с установленной датой бизнес-процесса.

Пример:

|  |
| --- |
| ВыбБизнесПроцесс.УстановитьНовыйНомер(ПрефиксИБ); |
