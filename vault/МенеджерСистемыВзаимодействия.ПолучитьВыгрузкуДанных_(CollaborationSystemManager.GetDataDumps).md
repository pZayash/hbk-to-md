ПолучитьВыгрузкуДанных (GetDataDumps)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьВыгрузкуДанных()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ВыгрузкаДанныхСистемыВзаимодействия](ВыгрузкаДанныхСистемыВзаимодействия_(CollaborationSystemDataDump).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
