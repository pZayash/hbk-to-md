Доступен, начиная с версии 8.1.

Элементы коллекции:

[НотацияDOM](НотацияDOM.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются узлы.  
Возможно обращение к узлу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция узлов нотаций DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
