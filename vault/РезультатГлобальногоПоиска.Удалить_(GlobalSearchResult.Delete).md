РезультатГлобальногоПоиска (GlobalSearchResult)

Удалить (Delete)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции.

Вариант синтаксиса: По элементу

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md).   
Удаляемый элемент.

Вариант синтаксиса: По значению

Синтаксис:

Удалить(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение удаляемого элемента.

Описание:

Удаляет элемент из коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
