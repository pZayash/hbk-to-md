XBase (XBase)

ТекущийИндекс (CurrentIndex)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИндексXBase](ИндексXBase_(XBaseIndex).md).   
Содержит текущий используемый индекс. Установка текущего индекса работы с таблицей базы данных оказывает влияние на работу методов поиска и позиционирования: [Найти](XBase.Найти_(XBase.Find).md), [НайтиПоКлючу](XBase.НайтиПоКлючу_(XBase.FindByKey).md), [Первая](XBase.Первая_(XBase.First).md), [Последняя](XBase.Последняя_(XBase.Last).md), [Следующая](XBase.Следующая_(XBase.Next).md), [Предыдущая](XBase.Предыдущая_(XBase.Prev).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
