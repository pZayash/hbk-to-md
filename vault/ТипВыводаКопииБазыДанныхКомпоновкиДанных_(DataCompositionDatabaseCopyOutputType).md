Описание:

Содержит варианты вывода копии базы данных, из которой получены данных.  
Пустая строка, если данные получены из основной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
