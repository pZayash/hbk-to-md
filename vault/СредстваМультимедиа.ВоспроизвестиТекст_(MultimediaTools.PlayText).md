СредстваМультимедиа (MultimediaTools)

ВоспроизвестиТекст (PlayText)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ВоспроизвестиТекст(<Текст>, <Язык>, <Скорость>, <Тембр>, <ОбработчикОстановкиВоспроизведения>, <РазрешитьФоновое>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст, который требуется воспроизвести.

<Язык> (необязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Язык](ОбъектМетаданных_Язык_(MetadataObject_Language).md).   
Идентификатор языка согласно стандарту BCP 47 (например, "en-US" для английского или "ru-RU" для русского). Если не указан, то проверяется возможность произнести текст на текущем языке операционной системы. Если идентификатор указан с ошибкой, либо выбранный язык не поддерживается в системе, будет сгенерировано исключение.

<Скорость> (необязательный)

Тип: [Число](lang__def_Number.md).   
Скорость воспроизведения текста в интервале от 0.5 до 2.0.  
Значение по умолчанию: 1.0.

<Тембр> (необязательный)

Тип: [Число](lang__def_Number.md).   
Смещение тембра голоса в интервале от 0.5 до 2.0.  
Значение по умолчанию: 1.0.

<ОбработчикОстановкиВоспроизведения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана при остановке или прерывании воспроизведения текста со следующими параметрами:

- <Дополнительно> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<РазрешитьФоновое> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешает воспроизведение текста в фоне.  
Значение по умолчанию: Ложь.

Описание:

Воспроизвести заданный текст с помощью синтезатора речи на устройстве.  
Если стороннее приложение воспроизводит звук или выполняется метод [ВоспроизвестиАудио](СредстваМультимедиа.ВоспроизвестиАудио_(MultimediaTools.PlayAudio).md), воспроизведение звука будет остановлено, после чего будет воспроизведен текст.  
Если во время произношения текста на мобильное приложение приходит системное уведомление, или с помощью метода [ВоспроизвестиЗвуковоеОповещение](СредстваМультимедиа.ВоспроизвестиЗвуковоеОповещение_(MultimediaTools.PlaySoundAlert).md) инициируется звуковое оповещение, то мобильная платформа должна приостановить воспроизведение текста и продолжить его после завершения звука уведомления, с позиции, на которой воспроизведение было остановлено.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства, перед вызовом метода следует использовать метод [ПоддерживаетсяАудиозапись](СредстваМультимедиа.ПоддерживаетсяАудиозапись_(MultimediaTools.AudioRecordingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Если в момент вызова метода, уже воспроизводится текст, воспроизведение будет прервано и начнется воспроизведение нового текста. Обработчик завершения для предыдущего воспроизведения текста вызван не будет.

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ПоддерживаетсяАудиозапись](СредстваМультимедиа.ПоддерживаетсяАудиозапись_(MultimediaTools.AudioRecordingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
