Файл (File)

ЭтоФайл (IsFile)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоФайл()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект соответствует файлу; [Ложь](lang__def_BooleanFalse.md) - объект соответствует каталогу.

Описание:

Определяет, соответствует ли файловый объект файлу или каталогу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Противоположен методу [ЭтоКаталог](Файл.ЭтоКаталог_(File.IsDirectory).md). Применим только к файлам, которые реально существуют.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.

| Если ВыбФайл.ЭтоФайл() Тогда     Предупреждение(НСтр("ru = 'Укажите каталог!'; en = 'Define a directory!'")); КонецЕсли; |
