# ПустаяЗаписьNDEF

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [NFC](NFC.md)

Доступен, начиная с версии 8.3.23.

Описание:

Данный объект следует использовать только для обработки пустых записей, обнаруженных в полученном сообщении. Возможность самостоятельного добавления пустой записи в сообщение не предоставляется.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
