# АдминистрированиеКластер.СоздатьРабочийСервер

АдминистрированиеКластер (AdministrationCluster)

СоздатьРабочийСервер (CreateWorkServer)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьРабочийСервер()

Возвращаемое значение:

Тип: [АдминистрированиеРабочийСервер](АдминистрированиеРабочийСервер_(AdministrationWorkServer).md).   

Описание:

Создает новый рабочий сервер. Новый сервер может быть использован, например, для регистрации сервера. Для этого требуется явно задать необходимые значения свойств сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
