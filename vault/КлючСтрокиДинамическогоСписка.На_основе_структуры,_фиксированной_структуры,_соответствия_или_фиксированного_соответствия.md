КлючСтрокиДинамическогоСписка (DynamicListRowKey)

На основе структуры, фиксированной структуры, соответствия или фиксированного соответствия

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый КлючСтрокиДинамическогоСписка(<ИсточникКлюча>)

Параметры:

<ИсточникКлюча> (обязательный)

Тип: [Структура](Структура_(Structure).md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), [Соответствие](Соответствие_(Map).md), [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Коллекция, используемая в качестве источника путей и значений ключевых полей.

Описание:

Создаваемый ключ строки заполняется значениями из переданной коллекции. Ключи элементов коллекции используются в качестве путей ключевых полей, а значения - в качестве значений полей.

Использование в версии:

Доступен, начиная с версии 8.3.14.
