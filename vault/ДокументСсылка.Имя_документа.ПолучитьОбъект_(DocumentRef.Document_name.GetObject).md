# ДокументСсылка.<Имя документа>.ПолучитьОбъект

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументСсылка.<Имя документа>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md) › [Методы](ДокументСсылка.Имя_документа__Методы.md)

ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ДокументОбъект.](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)<[Имя документа](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект для чтения, изменения, добавления и удаления документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = ВыбДокумент.ПолучитьОбъект(); |

Использование в версии:

Доступен, начиная с версии 8.0.
