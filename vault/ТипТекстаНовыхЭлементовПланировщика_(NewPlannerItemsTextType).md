# ТипТекстаНовыхЭлементовПланировщика

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Планировщик](Планировщик.md)

Доступен, начиная с версии 8.3.18.

Значения

[Строка (String)](ТипТекстаНовыхЭлементовПланировщика.Строка_(NewPlannerItemsTextType.String).md)  
[ФорматированнаяСтрока (FormattedString)](ТипТекстаНовыхЭлементовПланировщика.ФорматированнаяСтрока_(NewPlannerItemsTextType.FormattedString).md)  

Описание:

Содержит тип текста новых элементов планировщика. Влияет на тип параметра <Текст> события [ПередСозданием](Расширение_поля_формы_для_поля_планировщика.ПередСозданием_(Form_field_extension_for_a_planner.BeforeCreate).md), а также тип элементов планировщика, создаваемых через методы [Добавить](КоллекцияЭлементовПланировщика.Добавить_(PlannerItemCollection.Add).md) и [Вставить](КоллекцияЭлементовПланировщика.Вставить_(PlannerItemCollection.Insert).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML.

Использование в версии:

Доступен, начиная с версии 8.3.18.
