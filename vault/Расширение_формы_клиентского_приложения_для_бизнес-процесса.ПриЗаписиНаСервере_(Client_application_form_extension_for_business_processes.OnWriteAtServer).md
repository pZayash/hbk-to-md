Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

ПриЗаписиНаСервере (OnWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЗаписиНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись произведена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ТекущийОбъект>

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Структура, содержащая параметры записи. Для формы бизнес-процесса существует один предопределенный параметр Старт. При отработке команд "Старт", "Стартовать и закрыть" он устанавливается в [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает на сервере при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события ПриЗаписи объекта) в базу данных, но до окончания транзакции записи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
