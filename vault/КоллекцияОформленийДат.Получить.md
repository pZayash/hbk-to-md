КоллекцияОформленийДат (DateAppearanceCollection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: Дата.   
Дата, для которой необходимо получить оформление.

Возвращаемое значение:

Тип: [КлючИЗначение](КлючИЗначение.md).   
Ключом является дата, а значением - [ОформлениеДаты](ОформлениеДаты.md).

Описание:

Получает оформление даты для указанной даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.0.
