ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПланыВидовРасчета (ChartsOfCalculationTypes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все планы видов расчета, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ПланВидовРасчета](ОбъектМетаданных_ПланВидовРасчета_(MetadataObject_ChartOfCalculationTypes).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
