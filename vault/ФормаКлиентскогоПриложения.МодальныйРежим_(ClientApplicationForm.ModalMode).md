ФормаКлиентскогоПриложения (ClientApplicationForm)

МодальныйРежим (ModalMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак монопольного открытия формы.  
[Истина](lang__def_BooleanTrue.md) - форма открыта в модальном режиме.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
