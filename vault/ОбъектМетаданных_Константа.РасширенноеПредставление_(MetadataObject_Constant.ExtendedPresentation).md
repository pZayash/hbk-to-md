ОбъектМетаданных: Константа (MetadataObject: Constant)

РасширенноеПредставление (ExtendedPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием константы и используется при формировании команды открытия формы константы в командном интерфейсе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
