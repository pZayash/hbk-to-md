Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

ИнформацияПрограммыПросмотра (UserAgentInformation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит идентификационную строку (User-Agent) браузера, который отображает содержимое HTML-документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
