ПользовательскиеНастройкиКомпоновкиДанных (DataCompositionUserSettings)

ПолучитьОсновныеНастройкиПоИдентификаторуПользовательскойНастройки (GetMainSettingsByUserSettingID)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьОсновныеНастройкиПоИдентификаторуПользовательскойНастройки(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: Строка.   
Идентификатор пользовательской настройки.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Возвращаемый массив может содержать элементы следующих типов:

- [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных.md),
- [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных.md),
- [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных.md),
- [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных.md),
- [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных.md),
- [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных.md),
- [ЭлементУсловногоОформленияКомпоновкиДанных](ЭлементУсловногоОформленияКомпоновкиДанных.md),
- [ЗначениеПараметраНастроекКомпоновкиДанных](ЗначениеПараметраНастроекКомпоновкиДанных.md),
- [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных.md),
- [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных.md),
- [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных.md),
- [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных.md),
- [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных.md),
- [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных.md),
- [КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных](КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных.md),

Если объекты не найдены, метод возвращает пустой массив.

Описание:

Получает массив объектов основных настроек по идентификатору пользовательской настройки. Эти объекты можно использовать для доступа к свойствам, значения которых не установлены у объектов пользовательских настроек, поскольку они недоступны для редактирования пользователю. Например, свойство [ЛевоеЗначение](ЭлементОтбораКомпоновкиДанных.ЛевоеЗначение.md) объекта [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если идентификатор пользовательской настройки объекта имеет тип [СтруктураНастроекКомпоновкиДанных](СтруктураНастроекКомпоновкиДанных.md) из состава пользовательских настроек, то возвращаемый массив будет содержать элемент полной коллекции настроек типа [КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных](КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных.md).

Использование в версии:

Доступен, начиная с версии 8.3.13.
