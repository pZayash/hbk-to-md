# ВременныеТаблицыЗапроса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Выполнение и работа с запросами во встроенном языке](Выполнение_и_работа_с_запросами_во_встроенном_языке.md)

Доступен, начиная с версии 8.3.8.

Элементы коллекции:

[ВременнаяТаблицаЗапроса](ВременнаяТаблицаЗапроса_(QueryTempTable).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются временные таблицы.  
Возможно обращение к временной таблице посредством оператора [...]. В качестве аргумента передается индекс, полное имя.  

Методы:

[Индекс (IndexOf)](ВременныеТаблицыЗапроса.Индекс_(QueryTempTables.IndexOf).md)  
[Количество (Count)](ВременныеТаблицыЗапроса.Количество_(QueryTempTables.Count).md)  
[Найти (Find)](ВременныеТаблицыЗапроса.Найти_(QueryTempTables.Find).md)  
[Получить (Get)](ВременныеТаблицыЗапроса.Получить_(QueryTempTables.Get).md)  

Описание:

Содержит коллекцию временных таблиц запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
