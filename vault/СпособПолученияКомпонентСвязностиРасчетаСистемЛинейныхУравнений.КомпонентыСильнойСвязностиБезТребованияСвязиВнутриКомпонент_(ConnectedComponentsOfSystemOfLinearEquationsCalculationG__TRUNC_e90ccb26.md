СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений (ConnectedComponentsOfSystemOfLinearEquationsCalculationGettingMethod)

КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент (StronglyConnectedComponentsWithNoInnerConnectionRequired)

Доступен, начиная с версии 8.3.15.

Описание:

Расчет компонент связности, наиболее оптимальных для анализа и отладки механизма решения систем линейных уравнений.   
Возвращаются такие компоненты (подграфы), у которых для любой вершины из одной компоненты не существует пути в любую другую вершину любой другой компоненты.

Использование в версии:

Доступен, начиная с версии 8.3.15.
