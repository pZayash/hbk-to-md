ОбъектМетаданных: Документ (MetadataObject: Document)

Автонумерация (Autonumbering)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то новому объекту базы данных система будет автоматически присваивать очередной порядковый номер/код.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
