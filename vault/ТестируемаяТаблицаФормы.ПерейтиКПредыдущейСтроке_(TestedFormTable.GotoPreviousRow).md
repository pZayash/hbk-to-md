ТестируемаяТаблицаФормы (TestedFormTable)

ПерейтиКПредыдущейСтроке (GotoPreviousRow)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПерейтиКПредыдущейСтроке(<ПереключитьВыделение>)

Параметры:

<ПереключитьВыделение> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Ложь](lang__def_BooleanFalse.md), поведение метода эмулирует обычный переход курсора к строке при помощи клавиатуры.  
Если установлено [Истина](lang__def_BooleanTrue.md), поведение метода эмулирует обычный переход курсора к строке при помощи клавиатуры с нажатой клавишей Shift.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Перемещает курсор к предыдущей строке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
