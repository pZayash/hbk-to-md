Регион (State)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Регион - название административной единицы второго уровня, если используется в адресе: область, республика, край, штат, земля, провинция, префектура, графство и т. п.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
