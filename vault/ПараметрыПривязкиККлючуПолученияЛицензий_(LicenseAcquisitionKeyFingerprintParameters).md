Описание:

Данные о привязке программной лицензии на платформу к аппаратной лицензии (ключу защиты).   
Может быть получен только с помощью метода [ПолучитьДоступныеКлючи](МенеджерПолученияЛицензий.ПолучитьДоступныеКлючи_(LicenseAcquisitionManager.GetAvailableKeys).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.   
Сериализуется.
