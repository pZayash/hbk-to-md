КвитанцияВстроеннойПокупки (InAppPurchaseReceipt)

Сервис (Service)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [СервисВстроенныхПокупок](СервисВстроенныхПокупок_(InAppPurchaseService).md).   
Сервис встроенных покупок, от которого получена квитанция.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
