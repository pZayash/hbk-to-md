# ПакетЗапросовСхемыЗапроса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются запрос выбора или запрос уничтожения таблицы.  
Возможно обращение к запросу выбора или запросу уничтожения таблицы посредством оператора [...]. В качестве аргумента передается индекс.
