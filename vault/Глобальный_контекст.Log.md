Глобальный контекст (Global context)

Log (Log)

Доступен, начиная с версии 8.0.

Синтаксис:

Log(<Число>)

Параметры:

<Число> (обязательный)

Тип: Число.   
Исходное число, больше 0.

Возвращаемое значение:

Тип: Число.   
Значение результата вычисления логарифма.

Описание:

Вычисляет натуральный логарифм параметра <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

При отрицательном или нулевом значении параметра происходит ошибка "Неправильное значение аргумента встроенной функции (Log)".

Пример:

|  |
| --- |
| Частота = -5; Попытка     Шкала = Log(Частота);     Сообщить("Log(Частота)="+Шкала); Исключение     Сообщить("При значении параметра "+Частота         +" выдается сообщение об ошибке:"         +Символы.ПС+""""+ОписаниеОшибки()+""""); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
