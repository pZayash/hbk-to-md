ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Боты (Bots)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все боты системы взаимодействия, определенные в конфигурации.  
Элементами коллекции являются объекты типа [БотСистемыВзаимодействия](БотСистемыВзаимодействия_(CollaborationSystemBot).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
