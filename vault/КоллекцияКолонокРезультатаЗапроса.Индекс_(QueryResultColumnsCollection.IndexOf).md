# КоллекцияКолонокРезультатаЗапроса.Индекс

КоллекцияКолонокРезультатаЗапроса (QueryResultColumnsCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаРезультатаЗапроса](КолонкаРезультатаЗапроса_(QueryResultColumn).md).   
Колонка результата запроса, для которой нужно определить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной колонки в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс колонки результата запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| РезультатЗапроса.Колонки.Индекс(Колонка); |

Использование в версии:

Доступен, начиная с версии 8.3.8.
