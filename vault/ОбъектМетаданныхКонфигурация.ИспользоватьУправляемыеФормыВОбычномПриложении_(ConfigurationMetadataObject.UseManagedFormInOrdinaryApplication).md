ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ИспользоватьУправляемыеФормыВОбычномПриложении (UseManagedFormInOrdinaryApplication)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то при проверке модулей в режимах "Толстый клиент (обычное приложение)" и "Толстый клиент (обычное приложение) вариант клиент-сервер" модули формы клиентского приложения и модули команд не проверяются. При этом при получении формы по умолчанию в толстом клиенте в режиме обычное приложение, если в качестве формы по умолчанию используется форма клиентского приложения, будет сгенерирована обычная форма.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
