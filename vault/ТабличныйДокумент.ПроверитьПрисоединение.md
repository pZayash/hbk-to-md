ТабличныйДокумент (SpreadsheetDocument)

ПроверитьПрисоединение (CheckAttachment)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьПрисоединение(<Таблицы>)

Параметры:

<Таблицы> (обязательный)

Тип: [Массив](Массив.md).   
Массив из проверяемых таблиц.

Возвращаемое значение:

Тип: Булево.   
Истина - умещаются; Ложь - в противном случае.  
Истина, если [АвтоМасштаб](ТабличныйДокумент.АвтоМасштаб.md)=Истина.

Описание:

Проверяет, умещаются ли переданные таблицы по ширине страницы при печати.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При возникновении проблем с получением информации о текущем принтере (например, в системе не установлено ни одного принтера), будет вызвано исключение.

Пример:

|  |
| --- |
| // в примере Таблицы - это массив таблиц  // Колонка - это присоединяемая область (имеет тип ТабличныйДокумент)  Таблицы.Добавить(Колонка); Если Не ТабДок.ПроверитьПрисоединение(Таблицы) Тогда     Сообщить("Документ не умещается по ширине страницы!"); КонецЕсли; |
