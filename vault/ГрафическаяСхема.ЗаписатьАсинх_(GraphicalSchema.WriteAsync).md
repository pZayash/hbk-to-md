# ГрафическаяСхема.ЗаписатьАсинх

ГрафическаяСхема (GraphicalSchema)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в котором сохраняется графическая схема.

Описание варианта метода:

Запись графической схемы в файл.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи графической схемы.

Описание варианта метода:

Записывает графическую схему в поток.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания:

- [Истина](lang__def_BooleanTrue.md) - если запись была инициирована с помощью расширения браузера и пользователь подтвердил получение файла, то есть запись файла была завершена.
- [Ложь](lang__def_BooleanFalse.md) - если запись была инициирована с помощью расширения браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
- [Неопределено](lang__def_Undefined.md) - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

Описание:

Асинхронная функция. Записывает графическую схему в файл.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ГрафическаяСхема](ГрафическаяСхема_(GraphicalSchema).md), метод [Записать](ГрафическаяСхема.Записать_(GraphicalSchema.Write).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
