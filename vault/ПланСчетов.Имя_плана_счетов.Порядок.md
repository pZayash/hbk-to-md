Порядок (Order)

Тип: Строка.   
Хранит строку с порядком, по которому можно выполнить упорядочивание выборки.  
Свойство предназначено для сортировки счетов в случаях, когда порядок следования по полю [Код](ПланСчетов.Имя_плана_счетов.Код.md) или [Наименование](ПланСчетов.Имя_плана_счетов.Наименование.md) не приемлем. Если длина поля [Порядок](ПланСчетов.Имя_плана_счетов.Порядок.md) > 0, то при формировании выборок будет автоматически устанавливаться сортировка по полю [Порядок](ПланСчетов.Имя_плана_счетов.Порядок.md). При этом основное представление по полю [Код](ПланСчетов.Имя_плана_счетов.Код.md) или [Наименование](ПланСчетов.Имя_плана_счетов.Наименование.md) влиять на упорядочивание счетов не будет.  
Если при конфигурировании у плана счетов установлено свойство "Автопорядок по коду", то при формировании результатов запроса будет отмечено, что сортировка выполняется по полю [Код](ПланСчетов.Имя_плана_счетов.Код.md), однако на самом деле будет использоваться сортировка по полю [Порядок](ПланСчетов.Имя_плана_счетов.Порядок.md).
