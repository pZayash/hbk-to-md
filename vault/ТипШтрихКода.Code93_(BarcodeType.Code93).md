ТипШтрихКода (BarcodeType)

Code93 (Code93)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Code 93. Отличается от Code 39 большей плотностью и защищенностью данных.

Использование в версии:

Доступен, начиная с версии 8.3.25.
