Следующий (Next)

Доступен, начиная с версии 8.1.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [ЭлементРезультатаКомпоновкиДанных](ЭлементРезультатаКомпоновкиДанных_(DataCompositionResultItem).md), [Неопределено](lang__def_Undefined.md).

Описание:

Получает следующий элемент результата компоновки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
