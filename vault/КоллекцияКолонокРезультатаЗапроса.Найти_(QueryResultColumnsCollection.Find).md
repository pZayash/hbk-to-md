КоллекцияКолонокРезультатаЗапроса (QueryResultColumnsCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<НаименованиеКолонки>)

Параметры:

<НаименованиеКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки результата запроса.

Возвращаемое значение:

Тип: [КолонкаРезультатаЗапроса](КолонкаРезультатаЗапроса_(QueryResultColumn).md), [Неопределено](lang__def_Undefined.md).   
Если указанная колонка отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск колонки результата запроса по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если РезультатЗапроса.Колонки.Найти("Цена") <> Неопределено Тогда     ЦеныВЗапросе = Истина; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
