ВидСравнения (ComparisonType)

ИнтервалВключаяНачало (IntervalIncludingLowerBound)

Доступен, начиная с версии 8.0.

Описание:

Сравниваемое значение лежит в заданном интервале, включая начальную и не включая конечную границу интервала.

Использование в версии:

Доступен, начиная с версии 8.0.
