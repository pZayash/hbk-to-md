СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

СохранитьНастройкуПоУмолчанию (SaveDefaultSetting)

Доступен, начиная с версии 8.3.26.

Синтаксис:

СохранитьНастройкуПоУмолчанию(<КлючОбъекта>, <Настройки>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ объекта настройки.

<Настройки> (обязательный)

Тип: Произвольный.   
Содержит настройку, которую нужно сохранить в хранилище в качестве настройки по умолчанию для заданного ключа объекта.

Описание:

Сохраняет настройку по умолчанию в хранилище.  
Вызов метода:

- требует наличия права "СохранениеДанныхПользователя",
- допускается только для хранилища системных настроек (ХранилищеСистемныхНастроек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
