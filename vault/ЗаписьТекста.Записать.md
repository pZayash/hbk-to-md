ЗаписьТекста (TextWriter)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Записываемая строка.

Описание:

Записывает строку текста в файл. Разделитель за записываемой строкой не ставится.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Записать("Добро пожаловать!"); // записывает в файл строку "Добро пожаловать!" |

Использование в версии:

Доступен, начиная с версии 8.0.
