# МенеджерХранилищаДвоичныхДанных.ОчиститьНеиспользуемоеМестоПоУниверсальнойДате

МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ОчиститьНеиспользуемоеМестоПоУниверсальнойДате (ClearUnusedSpaceByUniversalDate)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ОчиститьНеиспользуемоеМестоПоУниверсальнойДате(<УниверсальнаяДатаДоКоторойВыполнятьОчистку>)

Параметры:

<УниверсальнаяДатаДоКоторойВыполнятьОчистку> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата и время (в формате UTC) очистки неиспользуемоего места.

Описание:

Физически удаляет помеченные на удаление данные хранилища двоичных данных до указанной даты.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены.

Доступность:

Сервер.

Примечание:

Дата и время очистки неиспользуемого места в формате UTC.  
  
Дата и время указывается исключительно (то есть данные, у которых дата и время точно совпадает с датой очистки, не очищаются, остаются неиспользуемыми)

Использование в версии:

Доступен, начиная с версии 8.3.24.
