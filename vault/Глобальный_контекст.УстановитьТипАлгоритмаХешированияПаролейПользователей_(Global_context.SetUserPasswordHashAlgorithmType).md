УстановитьТипАлгоритмаХешированияПаролейПользователей (SetUserPasswordHashAlgorithmType)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьТипАлгоритмаХешированияПаролейПользователей(<ТипАлгоритмаХешированияПаролейПользователей>)

Параметры:

<ТипАлгоритмаХешированияПаролейПользователей> (обязательный)

Тип: [ТипАлгоритмаХешированияПаролейПользователей](ТипАлгоритмаХешированияПаролейПользователей_(UserPasswordHashAlgorithmType).md).   
Новый тип алгоритма хеширования паролей пользователей информационной базы.

Описание:

Устанавливает используемый алгоритм хеширования паролей пользователей информационной базы.  
Требует наличие права "Администрирование" у текущего пользователя.  
По умолчанию используется алгоритм SHA-1. Только алгоритм SHA-1 поддерживается всеми версиями платформы. Другие алгоритмы поддерживаются платформой версии 8.3.26 и выше. После изменения алгоритма хранимое значение пароля у каждого пользователя будет заменено на новое в соответствии с новым алгоритмом при очередной аутентификации пользователя или при очередном изменении значения пароля пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md), свойство [ТипАлгоритмаХешированияПароля](ПользовательИнформационнойБазы.ТипАлгоритмаХешированияПароля_(InfoBaseUser.PasswordHashAlgorithmType).md)  

Использование в версии:

Доступен, начиная с версии 8.3.26.
