РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - записи изменены; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, были ли изменены записи в наборе после последнего считывания или записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, были ли изменены соответствующие записи другими пользователями.

Пример:

|  |
| --- |
| Если НаборЗаписей.Модифицированность() Тогда     // Запись набора записей.      Попытка         НаборЗаписей.Записать();     Исключение         Предупреждение(НСтр("ru = 'Не удалось записать: ';"            + " en = 'Can't write: '") + ОписаниеОшибки());         Отказ = Истина;     КонецПопытки; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
