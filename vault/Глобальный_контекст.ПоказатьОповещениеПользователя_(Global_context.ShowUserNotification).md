# Глобальный контекст.ПоказатьОповещениеПользователя

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Процедуры и функции интерактивной работы](Процедуры_и_функции_интерактивной_работы.md)

Глобальный контекст (Global context)

ПоказатьОповещениеПользователя (ShowUserNotification)

Доступен, начиная с версии 8.2.

Синтаксис:

ПоказатьОповещениеПользователя(<Текст>, <ДействиеПриНажатии>, <Пояснение>, <Картинка>, <СтатусОповещенияПользователя>, <КлючУникальности>)

Параметры:

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст оповещения.

<ДействиеПриНажатии> (необязательный)

Тип: [Строка](lang__def_String.md), [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Если тип [Строка](lang__def_String.md), то она содержит навигационную ссылку, по которой будет выполнен переход при нажатии на окно оповещения. После перехода окно оповещения будет закрыто.  
Если тип [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), то содержит описание процедуры, которая будет вызвана при нажатии на окно оповещения c параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

После вызова процедуры окно оповещения будет закрыто.  
При использовании этого параметра рекомендуется установить текст, содержащий указание на необходимость реакции пользователя на оповещение.

<Пояснение> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пояснение оповещения.

<Картинка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка, которая будет показана в оповещении.

<СтатусОповещенияПользователя> (необязательный)

Тип: [СтатусОповещенияПользователя](СтатусОповещенияПользователя_(UserNotificationStatus).md).   
Параметр определяет важность оповещения пользователя.  
Статус устанавливается в зависимости от критичности информации для пользователя.  
В режимах интерфейса, отличных от [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md), данный параметр игнорируется.  
Значение по умолчанию: [Информация](СтатусОповещенияПользователя.Информация_(UserNotificationStatus.Information).md).

<КлючУникальности> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ уникальности оповещения. Если ключ задан и существует открытое оповещение с таким же ключом, то оно будет заменено на новое. Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то оповещение считается уникальным.  
В режимах интерфейса, отличных от [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md), этот параметр игнорируется.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Показывает окно оповещения, в котором может располагаться текст с пояснением, картинка.  
Для оповещений со статусом [Важное](СтатусОповещенияПользователя.Важное_(UserNotificationStatus.Important).md) окно автоматически скрывается через 10 секунд. Одновременно может быть показано несколько оповещений. Нажатие на окно оповещения либо вызывает процедуру, определенную в описании оповещения, либо выполняет переход по навигационной ссылке.  
Для оповещений со статусом [Информация](СтатусОповещенияПользователя.Информация_(UserNotificationStatus.Information).md) окно постепенно затухает и исчезает с экрана. Одновременно может быть показано только одно оповещение. В окне может располагаться ссылка, нажатие на которую либо вызывает процедуру, определенную в описании оповещения, либо выполняет переход по навигационной ссылке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
