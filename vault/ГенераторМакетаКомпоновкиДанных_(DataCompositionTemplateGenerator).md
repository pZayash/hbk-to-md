# ГенераторМакетаКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макет компоновки данных](Макет_компоновки_данных.md)

Доступен, начиная с версии 8.3.5.

Описание:

Используется для генерации макета для вывода в табличный документ. Данный объект не может быть создан. Он может быть только передан в качестве параметра метода [Выполнить](КомпоновщикМакетаКомпоновкиДанных.Выполнить_(DataCompositionTemplateComposer.Execute).md).

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
