КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Получить (Get)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс измерения.

Возвращаемое значение:

Тип: [ИзмерениеПланировщика](ИзмерениеПланировщика_(PlannerDimension).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.3.6.
