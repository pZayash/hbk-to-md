ЧтениеУзловDOM (DOMNodeReader)

ИмеетИмя (HasName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Признак наличия у текущего узла имени.  
Истина - для узла определено имя, Ложь - в противном случае.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
