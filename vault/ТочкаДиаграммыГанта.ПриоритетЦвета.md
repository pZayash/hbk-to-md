ТочкаДиаграммыГанта (GanttChartPoint)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Вместе со свойством [ПриоритетЦвета](СерияДиаграммыГанта.ПриоритетЦвета.md) соответствующей серии определяет цвет значения. При равных значениях свойства используется цвет серии.  
Истина - будет использоваться значение [Цвет](ТочкаДиаграммыГанта.Цвет.md) точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
