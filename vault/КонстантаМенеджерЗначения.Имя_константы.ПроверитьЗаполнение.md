КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: Булево.   
Истина - ошибок не обнаружено, Ложь - в противном случае.

Описание:

Проверяет заполнение константы.   
У константы свойство "Проверка заполнения" может быть установлено в значение "Показывать ошибку". В таком случае, если константа не заполнена, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](КонстантаМенеджерЗначения.Имя_константы.ОбработкаПроверкиЗаполнения.md).
