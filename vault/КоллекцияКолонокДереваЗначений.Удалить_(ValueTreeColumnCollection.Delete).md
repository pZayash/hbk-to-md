КоллекцияКолонокДереваЗначений (ValueTreeColumnCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [КолонкаДереваЗначений](КолонкаДереваЗначений_(ValueTreeColumn).md).   
Индекс удаляемой колонки в коллекции или колонка.

Описание:

Удаляет колонку в заданной позиции коллекции колонок дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| КолонкиСостава.Удалить(1); |

Использование в версии:

Доступен, начиная с версии 8.0.
