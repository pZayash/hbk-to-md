ЗаписьДанных (DataWriter)

ЗаписатьЦелое16Асинх (WriteInt16Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьЦелое16Асинх(<Число>, <ПорядокБайтов>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число, которое будет записано в целевой поток.  
Значение числа должно находиться в диапазоне от 0 до 65535.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа при записи. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Записывает целое 16-битное число в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.
