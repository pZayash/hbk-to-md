# Расширение табличного поля списка справочника.ПередСворачиванием

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [Расширение табличного поля списка справочника](Расширение_табличного_поля_списка_справочника_(Catalog_list_table_box_extension).md) › [События](Расширение_табличного_поля_списка_справочника__События.md)

Расширение табличного поля списка справочника (Catalog list table box extension)

ПередСворачиванием (BeforeCollapse)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСворачиванием(<Строка>, <Отказ>)

Параметры:

<Строка>

Строка табличного поля.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то стандартной обработки события не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед закрытием узла табличного поля, отображаемого в виде дерева.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
