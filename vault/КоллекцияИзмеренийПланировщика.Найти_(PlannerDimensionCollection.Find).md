КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению измерения планировщика.

Возвращаемое значение:

Тип: [ИзмерениеПланировщика](ИзмерениеПланировщика_(PlannerDimension).md), [Неопределено](lang__def_Undefined.md).   
Если указанная колонка отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск измерения по значению свойства [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИзмерениеПланировщика](ИзмерениеПланировщика_(PlannerDimension).md), свойство [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
