ТаблицаФормы (FormTable)

АвтоВводНезаполненного (AutoAddIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак автоматического ввода новой строки.   
[Истина](lang__def_BooleanTrue.md) - при активации таблицы (мышкой или кнопкой TAB), если в поле нет ни одной строки, происходит ввод новой строки.  
Если значение свойства [Неопределено](lang__def_Undefined.md) (в Конфигураторе выбрано значение Авто), то при автоматическом определении значения, помимо значения свойства [АвтоОтметкаНезаполненного](ТаблицаФормы.АвтоОтметкаНезаполненного_(FormTable.AutoMarkIncomplete).md), используется также значение свойства [ОтметкаНезаполненного](ТаблицаФормы.ОтметкаНезаполненного_(FormTable.MarkIncomplete).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
