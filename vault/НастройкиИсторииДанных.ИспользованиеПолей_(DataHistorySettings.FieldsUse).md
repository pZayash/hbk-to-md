НастройкиИсторииДанных (DataHistorySettings)

ИспользованиеПолей (FieldsUse)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит настройки использования для полей объекта метаданных.  
В качестве ключа соответствия используются [Строка](lang__def_String.md), имя поля (Реквизит, Измерение, Ресурс, СтандартныйРеквизит, Реквизит ТабличнойЧасти). Для реквизита табличной части указывается имя табличной части.  
Значение имеет тип [Булево](lang__Булево_(Boolean).md):

- [Истина](lang__def_BooleanTrue.md) - поле используется в истории данных,
- [Ложь](lang__def_BooleanFalse.md) - поле не используется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
