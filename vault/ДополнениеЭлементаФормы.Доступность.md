Доступен, начиная с версии 8.3.5.

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак доступности элемента.

Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
