СводнаяДиаграмма (PivotChart)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона сводной диаграммы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если свойство [ПрозрачныйФон](СводнаяДиаграмма.ПрозрачныйФон_(PivotChart.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
