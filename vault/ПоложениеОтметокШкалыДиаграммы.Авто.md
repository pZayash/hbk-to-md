ПоложениеОтметокШкалыДиаграммы (ChartScaleMarkLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Положение отметок определяется автоматически:

- если подписи диаграммы отображаются снаружи -- [Снаружи](ПоложениеОтметокШкалыДиаграммы.Снаружи.md),
- если подписи диаграммы отображаются внутри -- [Внутри](ПоложениеОтметокШкалыДиаграммы.Внутри.md).

Использование в версии:

Доступен, начиная с версии 8.3.11.
