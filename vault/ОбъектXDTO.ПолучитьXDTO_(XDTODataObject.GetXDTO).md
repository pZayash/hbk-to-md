ОбъектXDTO (XDTODataObject)

ПолучитьXDTO (GetXDTO)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьXDTO(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя свойства, значение которого необходимо получить.

Вариант синтаксиса: По свойству

Синтаксис:

ПолучитьXDTO(<Свойство>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md).   
Свойство XDTO, значение которого необходимо получить.

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Получает значение XDTO для указанного свойства.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
