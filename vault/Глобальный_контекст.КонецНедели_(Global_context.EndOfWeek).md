# Глобальный контекст.КонецНедели

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Функции работы со значениями типа Дата](Функции_работы_со_значениями_типа_Дата.md)

Глобальный контекст (Global context)

КонецНедели (EndOfWeek)

Доступен, начиная с версии 8.0.

Синтаксис:

КонецНедели(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Исходная дата.

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   
Дата и время конца недели.

Описание:

Определяет дату и время конца недели для указанной даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Сообщить(КонецНедели(ТекущаяДата())); |

Использование в версии:

Доступен, начиная с версии 8.0.
