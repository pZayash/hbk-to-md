# HTTPОтвет.ПолучитьТелоКакПоток

HTTPОтвет (HTTPResponse)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   

Описание:

Получает поток для чтения тела ответа.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.21.
