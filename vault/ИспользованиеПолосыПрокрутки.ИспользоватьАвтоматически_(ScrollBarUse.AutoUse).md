ИспользованиеПолосыПрокрутки (ScrollBarUse)

ИспользоватьАвтоматически (AutoUse)

Доступен, начиная с версии 8.0.

Описание:

Полоса прокрутки появляется, если содержимое элемента формы не может быть размещено полностью в отведенной для него области.

Использование в версии:

Доступен, начиная с версии 8.0.
