ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗакрыватьПриВыборе (CloseOnChoice)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость закрытия формы при выборе значения.  
При создании формы значение данного параметра устанавливается в свойство [ЗакрыватьПриВыборе](ФормаКлиентскогоПриложения.ЗакрыватьПриВыборе_(ClientApplicationForm.CloseOnChoice).md).  
[Истина](lang__def_BooleanTrue.md) - форма будет закрыта при выборе.

Использование в версии:

Доступен, начиная с версии 8.2.
