Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоМаксимальнаяВысота (AutoMaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет использование автоматического ограничения максимальной высоты элемента.  
Истина - автоматическое ограничение используется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
