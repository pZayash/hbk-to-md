ПолитикаПаролейПользователей (UserPasswordPolicy)

Удалить (Delete)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Удалить()

Описание:

Удаляет сведения о политике паролей пользователей.  
  
Доступен только пользователю с административными правами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
