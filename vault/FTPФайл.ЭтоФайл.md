FTPФайл (FTPFile)

ЭтоФайл (IsFile)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоФайл()

Возвращаемое значение:

Тип: Булево.   
Истина - объект соответствует файлу; Ложь - объект соответствует каталогу.

Описание:

Определяет, соответствует ли файловый объект файлу или каталогу.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Противоположен методу [ЭтоКаталог](FTPФайл.ЭтоКаталог.md). Применим только к файлам или каталогам, которые реально существуют.

Использование в версии:

Доступен, начиная с версии 8.0.
