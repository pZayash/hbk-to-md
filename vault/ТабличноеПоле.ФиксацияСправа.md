ТабличноеПоле (TableBox)

ФиксацияСправа (FixedRight)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит количество зафиксированных справа колонок. Зафиксированные колонки остаются на месте при горизонтальной прокрутке табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
