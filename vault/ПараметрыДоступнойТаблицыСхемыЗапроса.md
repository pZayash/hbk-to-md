Элементы коллекции:

[ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры доступной таблицы.  
Возможно обращение к параметру доступной таблицы посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПараметрыДоступнойТаблицыСхемыЗапроса.Индекс.md)  
[Количество (Count)](ПараметрыДоступнойТаблицыСхемыЗапроса.Количество.md)  
[Найти (Find)](ПараметрыДоступнойТаблицыСхемыЗапроса.Найти.md)  
[Получить (Get)](ПараметрыДоступнойТаблицыСхемыЗапроса.Получить.md)  

Описание:

Коллекция параметров доступной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
