Automation сервер (Automation server)

NewObject (NewObject)

Доступен, начиная с версии 8.1.

Синтаксис:

NewObject(<Имя>, <ПараметрыКонструктора>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя типа создаваемого объекта аналогичное тому, которое может быть задано в операторе встроенного языка Новый.

<ПараметрыКонструктора> (необязательный)

Тип: Произвольный.   
Список параметров для конструктора объекта создаваемого типа, перечисленных через запятую в том порядке, в котором они определены в конструкторе.

Возвращаемое значение:

Тип: [COMОбъект](COMОбъект.md).   

Описание:

Создает объект, для которого предусмотрен конструктор, и возвращает ссылку на него.

Доступность:

Интеграция.

Пример:

|  |
| --- |
| connector = Новый COMObject("V83.Application"); connector.Connect("File=c:\InfoBases\Trade;Usr=Director;") ТДокумент = connector.NewObject("ТекстовыйДокумент"); ТДокумент.ВставитьСтроку(1, "Новая строка"); Сообщить(ТДокумент. КоличествоСтрок()); |

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.27.
