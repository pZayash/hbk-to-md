WSПрокси (WSProxy)

Прокси (Proxy)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md).   
Прокси, через который осуществляются HTTP-соединения с веб-сервисом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
