ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

ПолучитьОбновлениеПредопределенныхДанных (GetPredefinedDataUpdate)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбновлениеПредопределенныхДанных()

Возвращаемое значение:

Тип: [ОбновлениеПредопределенныхДанных](Системные_перечисления.ОбновлениеПредопределенныхДанных.md).   

Описание:

Получает текущее значение режима обновления предопределенных элементов. Вычисляется из значений, установленных в метаданных и в данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
