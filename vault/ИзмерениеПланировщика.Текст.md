ИзмерениеПланировщика (PlannerDimension)

Текст (Text)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Текст, отображаемый в заголовке измерения в поле планировщика. Если текст не задан, то для отображения используется текст, полученный из свойства [Значение](ИзмерениеПланировщика.Значение.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
