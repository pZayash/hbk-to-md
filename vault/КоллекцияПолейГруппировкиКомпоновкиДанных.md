# КоллекцияПолейГруппировкиКомпоновкиДанных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › <a href="obsidian://open?file=objects__catalog63__catalog1064.md">Система компоновки данных</a> › [Настройки компоновки данных](Настройки_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеГруппировкиКомпоновкиДанных](ПолеГруппировкиКомпоновкиДанных.md), [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля группировки.  
Возможно обращение к полю группировки посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[Удалить (Delete)](КоллекцияПолейГруппировкиКомпоновкиДанных.Удалить.md)  

Описание:

Коллекция полей группировки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
