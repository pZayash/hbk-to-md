Расширение группы формы для обычной группы (Form group extension for a usual group)

ЦветФонаЗаголовкаСкрытогоОтображения (HiddenRepresentationTitleBackColor)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет.md).   
Если группа [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая.md), используется в качестве цвета фона заголовка в свернутом состоянии.  
Если группа [Всплывающая](ПоведениеОбычнойГруппы.Всплывающая.md), используется в качестве цвета фона заголовка в любом состоянии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

По умолчанию - [АвтоЦвет](ВидЦвета.АвтоЦвет.md), который трактуется как прозрачный цвет.

Использование в версии:

Доступен, начиная с версии 8.3.12.
