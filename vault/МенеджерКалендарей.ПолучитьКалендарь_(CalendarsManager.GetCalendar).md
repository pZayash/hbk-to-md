МенеджерКалендарей (CalendarsManager)

ПолучитьКалендарь (GetCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьКалендарь(<Календарь>)

Параметры:

<Календарь> (обязательный)

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря_(LocalCalendarKey).md).   
Локальный ключ календаря.

Возвращаемое значение:

Тип: [ДанныеКалендаряУчетнойЗаписи](ДанныеКалендаряУчетнойЗаписи_(AccountCalendarData).md).   

Описание:

Получает данные календаря по локальному ключу.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
