Системная форма различий версий истории данных (System form for the Data history version differences)

Данные (Data)

Доступен, начиная с версии 8.3.13.

Описание:

Тип: Произвольный.   
Объект данных, для которого отображаются различия версий.  
Допускаются следующие типы: [СправочникСсылка.](СправочникСсылка.Имя_справочника.md)<[Имя справочника](СправочникСсылка.Имя_справочника.md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>.
