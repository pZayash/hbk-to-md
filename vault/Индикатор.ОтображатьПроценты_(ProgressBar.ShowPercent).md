# Индикатор.ОтображатьПроценты

Индикатор (ProgressBar)

ОтображатьПроценты (ShowPercent)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что на индикаторе либо рядом с ним следует выводить процентное представление индикатора (0% - минимальный, 100% - максимальный размер полосы индикатора).  
[Истина](lang__def_BooleanTrue.md) - отображаются.

Доступность:

Толстый клиент.

Примечание:

В режиме заполнения [Плавный](РежимСглаживанияИндикатора.Плавный_(ProgressBarSmoothingMode.Smooth).md) проценты показываются в центре, в режиме [Прерывистый](РежимСглаживанияИндикатора.Прерывистый_(ProgressBarSmoothingMode.Broken).md) и [ПрерывистыйНаклонный](РежимСглаживанияИндикатора.ПрерывистыйНаклонный_(ProgressBarSmoothingMode.BrokenTilt).md) - слева от индикатора.

Использование в версии:

Доступен, начиная с версии 8.0.
