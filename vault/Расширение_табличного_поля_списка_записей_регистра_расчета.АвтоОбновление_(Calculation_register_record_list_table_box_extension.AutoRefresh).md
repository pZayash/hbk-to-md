# Расширение табличного поля списка записей регистра расчета.АвтоОбновление

Расширение табличного поля списка записей регистра расчета (Calculation register record list table box extension)

АвтоОбновление (AutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического обновления табличного поля. Если признак установлен как [Истина](lang__def_BooleanTrue.md), то автоматически будет вызываться автоматическое обновление списка. Время между обновлениями списка задается свойством [ПериодАвтоОбновления](Расширение_табличного_поля_списка_записей_регистра_расчета.ПериодАвтоОбновления_(Calculation_register_record_list_table_box_extension.AutoRefreshPeriod).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
