# ПараметрыСоединенияВнешнегоИсточникаДанных.ИмяПользователя

ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

ИмяПользователя (UserName)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит имя пользователя, которое нужно использовать при стандартной аутентификации.  
Если содержит [Неопределено](lang__def_Undefined.md), то имя пользователя получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
