ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

Напечатать (Print)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Напечатать(<РежимОтображенияДиалогаПечати>)

Параметры:

<РежимОтображенияДиалогаПечати> (необязательный)

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати.md).   
Указывает необходимость отображения диалога печати.

Описание:

Отправляет документы на печать.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Под управлением ОС Windows:

- При печати пакета создается одна очередь на принтер.
- При печати в файл отображается один диалог, запрашивающий имя файла, и пакет сохраняется в файл.

Под управлением ОС Linux:

- Если параметры печати (ориентация страниц, размер бумаги) документов, входящих в пакет, одинаковые, то поведение совпадает с Windows.
- Если параметры различаются, то каждый документ печатается в отдельном задании.

В режиме совместимости конфигурации [Версия8\_3\_7](РежимСовместимости.Версия8_3_7.md) и ниже каждый документ пакета печатается в отдельном задании.

Использование в версии:

Доступен, начиная с версии 8.3.3.
