СравнениеФайлов (FileCompare)

УчитыватьРегистр (CaseSensitive)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в [Истина](lang__def_BooleanTrue.md), то при сравнении текстов регистры различаются.

Доступность:

Толстый клиент.

Примечание:

Для нетекстовых режимов сравнения свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.0.
