# ПараметрыСеанса

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Параметры сеанса](Параметры_сеанса.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

Произвольный  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры сеанса.  

Свойства:

[\<Имя параметра сеанса\> (\<Session parameter name\>)](ПараметрыСеанса.Имя_параметра_сеанса.md)  

Методы:

[Очистить (Clear)](ПараметрыСеанса.Очистить.md)  

Описание:

Является значением свойства глобального контекста [ПараметрыСеанса](Глобальный_контекст.ПараметрыСеанса.md) и служит для доступа к значениям параметров сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| // Установка значения параметра сеанса.  ПараметрыСеанса.ТекущийОтдел = "Производство"; // Получение значения параметра сеанса.  МойОтдел = ПараметрыСеанса.ТекущийОтдел; // Обращение к параметру сеанса из ограничения доступа к данным.  ГДЕ Отдел = &ТекущийОтдел |
