ОтчетОбОшибке (ErrorReport)

Отправить (Send)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Отправить(<Интерактивно>)

Параметры:

<Интерактивно> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр имеет значение [Истина](lang__def_BooleanTrue.md), то перед отправкой откроется форма создания отчета (где пользователь сможет дополнить отчет).   
Если параметр имеет значение [Ложь](lang__def_BooleanFalse.md), то отчет отправится без свойств, которые не разрешены для автоматической отправки.   
Если параметр не установлен, то форма откроется, если есть свойства, для записи которых требуется разрешение пользователя.

Описание:

Отправляет отчет в сервис сбора ошибок. Если отправка отчетов запрещена или адрес сервера не установлен, то метод не выполняет никаких действий.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
