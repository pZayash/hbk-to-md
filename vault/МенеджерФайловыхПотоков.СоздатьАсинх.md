МенеджерФайловыхПотоков (FileStreamsManager)

СоздатьАсинх (CreateAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СоздатьАсинх(<ИмяФайла>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя создаваемого файла.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер буфера.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [ФайловыйПоток](ФайловыйПоток.md) для файла с указанным именем.

Описание:

Асинхронная функция. Создает или перезаписывает файл и открывает поток с возможностью чтения и записи в файл. Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, толстый клиент.
