Расширение поля формы для поля ввода (Form field extension for a text box)

ОтображатьФлажкиВВыпадающемСпискеПриВводеМножественныхЗначений (ShowCheckBoxesInDropListWhenInputMultipleValues)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Содержит признак отображения флажков в выпадающем списке, если поле ввода редактирует данные типа [СписокЗначений](СписокЗначений.md) или [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md), то есть поле ввода находится в режиме ввода множественных значений. В остальных случаях значения данного свойства игнорируется.

- Истина – флажки отображаются.
- Ложь – флажки не отображаются.
- Неопределено – отображение флажков определяется автоматически:
  - Если [РазрешитьДублированиеМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.РазрешитьДублированиеМножественныхЗначений.md) имеет значение Ложь - флажки отображаются,
  - Если [РазрешитьДублированиеМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.РазрешитьДублированиеМножественныхЗначений.md) имеет значение Истина - флажки не отображаются.

Значение по умолчанию: Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
