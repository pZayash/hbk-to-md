ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

Фильтр (Filter)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Строка, определяющая все расширения файлов, доступные для выбора через диалог. Может состоять из нескольких элементов фильтра, разделенных символом «|» (вертикальная черта).  
Элемент фильтра – подстрока, входящая в фильтр, которая определяет набор расширений файлов. Состоит из описания и набора расширений файлов, разделенных символом «|» (вертикальная черта). В наборе расширений можно указать несколько расширений, используя в качестве разделителя символ «;».  
Например, фильтр: «Текстовый документ|\*.txt;\*.rtf |Табличный документ|\*.mxl» состоит из двух элементов фильтра:  
1) «Текстовый документ|\*.txt;\*.rtf» - имеет описание «Текстовый документ», включает в себя расширения файлов .txt и .rtf.  
2) «Табличный документ|\*.mxl» - имеет описание «Табличный документ», включает в себя расширение файлов .mxl.  
  
Пустая строка означает отсутствие фильтра.  
Значение по умолчанию: пустая строка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
