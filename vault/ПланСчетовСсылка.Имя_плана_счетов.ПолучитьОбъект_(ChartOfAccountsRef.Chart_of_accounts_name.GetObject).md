ПланСчетовСсылка.<Имя плана счетов> (ChartOfAccountsRef.<Chart of accounts name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект, предназначенный для модификации счета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
