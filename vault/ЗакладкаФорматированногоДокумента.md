# ЗакладкаФорматированногоДокумента

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Форматированный документ](objects__catalog63__catalog1934.md)

Доступен, начиная с версии 8.2.

Описание:

Содержит закладку форматированного документа. Определяет некоторую позицию в [ФорматированныйДокумент](ФорматированныйДокумент.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/formatted-document}. Имя типа XDTO: FormattedDocumentBookmark.

Использование в версии:

Доступен, начиная с версии 8.2.
