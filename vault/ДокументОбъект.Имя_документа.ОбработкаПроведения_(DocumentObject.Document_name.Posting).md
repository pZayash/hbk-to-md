ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

ОбработкаПроведения (Posting)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаПроведения(<Отказ>, <РежимПроведения>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак проведения документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то проведение документа выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<РежимПроведения>

Тип: [РежимПроведенияДокумента](РежимПроведенияДокумента_(DocumentPostingMode).md).   
В данный параметр передается текущий режим проведения.

Описание:

Возникает при проведении документа. Основное назначение процедуры-обработчика данного события - генерация движений по документу. Выполняется в транзакции записи.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если для данного вида документа в конфигурации установлено автоматическое удаление движений, то перед возникновением события все движения по документу будут удалены.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Доступен, начиная с версии 8.0.
