ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: Булево.   
Истина - изменяется еще ни разу не записанный узел; Ложь - узел уже записан.

Описание:

Определяет, записан ли узел в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если Объект.ЭтоНовый() Тогда     Объект.Наименование = "Новый узел"; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
