СправочникСсылка.<Имя справочника> (CatalogRef.<Catalog name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](СправочникМенеджер.Имя_справочника.ПолучитьСсылку.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СправочникМенеджер.\<Имя справочника\>](СправочникМенеджер.Имя_справочника.md), метод [ПолучитьСсылку](СправочникМенеджер.Имя_справочника.ПолучитьСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
