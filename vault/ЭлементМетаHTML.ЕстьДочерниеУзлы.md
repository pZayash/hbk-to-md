ЭлементМетаHTML (HTMLMetaElement)

ЕстьДочерниеУзлы (HasChildNodes)

Доступен, начиная с версии 8.2.

Синтаксис:

ЕстьДочерниеУзлы()

Возвращаемое значение:

Тип: Булево.   
Истина - узел имеет дочерние узлы, Ложь - в противном случае.

Описание:

Проверяет, имеются ли у узла дочерние узлы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
