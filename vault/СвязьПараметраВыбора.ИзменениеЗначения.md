СвязьПараметраВыбора (ChoiceParameterLink)

ИзменениеЗначения (ValueChange)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РежимИзмененияСвязанногоЗначения](РежимИзмененияСвязанногоЗначения.md).   
Содержит способ изменения связанного значения при изменении ведущего значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
