ИнтернетПочтовыйПрофиль (InternetMailProfile)

Пароль (Password)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Пароль доступа к почтовому ящику. Поддерживаются только символы, входящие в кодовую таблицу US-ASCII.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИнтернетПочтовыйПрофиль](ИнтернетПочтовыйПрофиль_(InternetMailProfile).md), свойство [Пользователь](ИнтернетПочтовыйПрофиль.Пользователь_(InternetMailProfile.User).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
