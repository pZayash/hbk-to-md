# ПриЗаписиНаСервере

Расширение формы клиентского приложения для объекта таблицы внешнего источника данных

ПриЗаписиНаСервере (OnWriteAtServer)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПриЗаписиНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись произведена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ТекущийОбъект>

Тип: [СправочникОбъект.\<Имя справочника\>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи.

Описание:

Возникает на сервере при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ВнешнийИсточникДанныхТаблицаОбъект.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.ПриЗаписи_(ExternalDataSourceTableObject.External_source_name.External_data_source_table_name.OnWrite).md) объекта) в базу данных, но до окончания транзакции записи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
