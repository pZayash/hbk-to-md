Глобальный контекст (Global context)

УстановитьЗапретЗасыпанияКомпьютера (SetComputerSleepModeProhibition)

Доступен, начиная с версии 8.3.16.

Синтаксис:

УстановитьЗапретЗасыпанияКомпьютера(<Включить>)

Параметры:

<Включить> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли установлен запрет засыпания компьютера:

- [Истина](lang__def_BooleanTrue.md) - запрет установлен;
- [Ложь](lang__def_BooleanFalse.md) - запрет снят.

Описание:

Устанавливает режим, при котором мобильное устройство не переходит в спящий режим.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
