# СправочникОбъект.<Имя справочника>.Ссылка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Свойства](СправочникОбъект.Имя_справочника__Свойства.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   
Содержит ссылку на элемент справочника. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Например, данное свойство можно использовать, если ссылку на созданный и записанный элемент справочника нужно присвоить реквизиту другого справочника.

Использование в версии:

Доступен, начиная с версии 8.0.
