ВнешнийИсточникДанныхТаблицаОбъект.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableObject.<External source name>.<External data source table name>)

Заполнить (Fill)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение объекта.

Описание:

Вызывает заполнение элемента данными из какого-либо другого объекта. При этом в модуле объекта вызывается процедура [ОбработкаЗаполнения](ВнешнийИсточникДанныхТаблицаОбъект.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.ОбработкаЗаполнения_(ExternalDataSourceTableObject.External_source_name.External_data_source_table_name.Filling).md). В ней может размещаться алгоритм, заполняющий данные элемента из переданного значения.  
Для заполнения может использоваться значение любого типа. Для этого типа в процедуре [ОбработкаЗаполнения](ВнешнийИсточникДанныхТаблицаОбъект.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.ОбработкаЗаполнения_(ExternalDataSourceTableObject.External_source_name.External_data_source_table_name.Filling).md) должен быть описан алгоритм заполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
