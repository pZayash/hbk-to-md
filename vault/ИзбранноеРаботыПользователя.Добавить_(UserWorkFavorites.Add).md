ИзбранноеРаботыПользователя (UserWorkFavorites)

Добавить (Add)

Доступен, начиная с версии 8.2.

Синтаксис:

Добавить(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ЭлементИзбранногоРаботыПользователя](ЭлементИзбранногоРаботыПользователя_(UserWorkFavoritesItem).md).   
Добавляемый элемент избранного.

Возвращаемое значение:

Тип: [ЭлементИзбранногоРаботыПользователя](ЭлементИзбранногоРаботыПользователя_(UserWorkFavoritesItem).md).   

Описание:

Добавляет элемент избранного в коллекцию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
