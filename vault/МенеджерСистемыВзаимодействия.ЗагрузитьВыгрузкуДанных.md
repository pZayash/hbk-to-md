МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ЗагрузитьВыгрузкуДанных (LoadDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ЗагрузитьВыгрузкуДанных(<ПутьКФайлу>, <Описание>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: Строка.   
Путь к файлу с выгрузкой данных.

<Описание> (обязательный)

Тип: Строка.   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.md).   

Описание:

Загружает выгрузку данных системы взаимодействия. Возвращает управление при окончании загрузки резервной копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ЗагрузитьВыгрузкуДанныхАсинх](МенеджерСистемыВзаимодействия.ЗагрузитьВыгрузкуДанныхАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
