СохранениеОформленияКомпоновкиДанных (SaveDataCompositionAppearance)

ДляТекущегоРезультата (ForCurrentResult)

Доступен, начиная с версии 8.3.16.

Описание:

Используются значения [ОформлениеЗначений](ОформлениеЗначений_(ValueAppearance).md), создаваемые только на момент вывода отчета, без сохранения между формированием отчета.

Использование в версии:

Доступен, начиная с версии 8.3.16.
