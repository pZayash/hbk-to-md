ПараметрыПотоковогоРаспознаванияРечи (SpeechToTextStreamingParameters)

ОбработчикОстановкиАудиозаписи (AudioRecordingStopHandler)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана для сохранения результата записи аудио со следующими параметрами:

- <РезультатАудиозаписи> - тип [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Если параметры не заданы, запись не выполняется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
