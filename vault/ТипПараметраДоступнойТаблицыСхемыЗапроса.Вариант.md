ТипПараметраДоступнойТаблицыСхемыЗапроса (QuerySchemaAvailableTableParameterType)

Вариант (Variant)

Доступен, начиная с версии 8.3.5.

Описание:

В параметре указывается один из возможных вариантов.  
Возможные варианты получаются из свойства [Варианты](ПараметрДоступнойТаблицыСхемыЗапроса.Варианты.md) объекта типа [ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса.md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
