АтрибутDOM (DOMAttribute)

Значение (Value)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текстовое значение атрибута.  
Значение формируется из всех дочерних узлов в порядке их следования. Ссылки на символы и основные сущности заменяются на их символьные представления. При установке значения все дочерние узлы удаляются, и создается один дочерний узел [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) с содержимым данного свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
