# Свойства объектов метаданных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › [Системные перечисления](Системные_перечисления.md)

Системные перечисления предназначены для определения некоторого набора предопределенных значений. Доступ к системным перечислениям этой группы осуществляется как к свойствам объекта [ПеречислимыеСвойстваОбъектовМетаданных](ПеречислимыеСвойстваОбъектовМетаданных.md). Конкретные значения указываются через точку от имени системного перечисления. Системные перечисления этой группы используются, как правило, для определения значений свойств объектов метаданных в логических выражениях.

Пример:

|  |
| --- |
| Сообщить(НСтр("ru = 'Периодичность номера документа - год: ';"      + " en = 'Document number periodicity - year: '")      + (Метаданные.Документы.Счет.ПериодичностьНомера=      Метаданные.СвойстваОбъектов.ПериодичностьНомераДокумента.Год)); |
