# Свойства объектов метаданных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md)

Системные перечисления предназначены для определения некоторого набора предопределенных значений. Доступ к системным перечислениям этой группы осуществляется как к свойствам объекта [ПеречислимыеСвойстваОбъектовМетаданных](ПеречислимыеСвойстваОбъектовМетаданных_(MetadataObjectEnumeratedProperties).md). Конкретные значения указываются через точку от имени системного перечисления. Системные перечисления этой группы используются, как правило, для определения значений свойств объектов метаданных в логических выражениях.

Пример:

|  |
| --- |
| Сообщить(НСтр("ru = 'Периодичность номера документа - год: ';"      + " en = 'Document number periodicity - year: '")      + (Метаданные.Документы.Счет.ПериодичностьНомера=      Метаданные.СвойстваОбъектов.ПериодичностьНомераДокумента.Год)); |
