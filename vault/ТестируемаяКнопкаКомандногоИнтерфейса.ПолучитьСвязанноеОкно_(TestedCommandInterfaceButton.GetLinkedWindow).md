ТестируемаяКнопкаКомандногоИнтерфейса (TestedCommandInterfaceButton)

ПолучитьСвязанноеОкно (GetLinkedWindow)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьСвязанноеОкно()

Описание:

Возвращает объект [ТестируемоеОкноКлиентскогоПриложения](ТестируемоеОкноКлиентскогоПриложения_(TestedClientApplicationWindow).md), связанный с кнопкой, если она расположена в панели открытых окон.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
