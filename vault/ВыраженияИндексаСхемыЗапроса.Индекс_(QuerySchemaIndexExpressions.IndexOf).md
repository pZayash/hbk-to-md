ВыраженияИндексаСхемыЗапроса (QuerySchemaIndexExpressions)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ВыражениеИндексаСхемыЗапроса](ВыражениеИндексаСхемыЗапроса_(QuerySchemaIndexExpression).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
