ГеографическаяСхема (GeographicalSchema)

ПроверитьПоРасположению (CheckByLocation)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьПоРасположению(<Объект>, <ГдеПроверять>, <КакПроверять>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md).   
Объект поиска.

<ГдеПроверять> (обязательный)

Тип: [Массив](Массив.md).   
Массив может содержать объекты следующих типов: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md), [ГеографическиеКоординаты](ГеографическиеКоординаты.md),   
[ПрямоугольникГеографическойСхемы](ПрямоугольникГеографическойСхемы.md).

<КакПроверять> (обязательный)

Тип: [ТипПоискаОбъектовГеографическойСхемы](ТипПоискаОбъектовГеографическойСхемы.md).   
Определяет способ поиска объекта географической схемы.

Возвращаемое значение:

Тип: Булево.   
Истина в противном случае.

Описание:

Проверяет, попадают ли указанные объекты в местоположение других.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
