# КоллекцияВстроенныхТаблиц.Индекс

КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ВстроеннаяТаблица>)

Параметры:

<ВстроеннаяТаблица> (обязательный)

Тип: [СводнаяТаблица](СводнаяТаблица_(PivotTable).md).   
Искомая встроенная таблица.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента коллекции встроенных таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
