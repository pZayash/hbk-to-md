Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileUnSafeExternalModule (UnregSecurityProfileUnSafeExternalModule)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileUnSafeExternalModule(<Кластер>, <ИмяПрофиля>, <ИмяМодуля>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должна быть отменена регистрация внешнего модуля.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<ИмяМодуля> (обязательный)

Тип: Строка.   
Имя внешнего модуля, регистрация которого отменяется.

Описание:

Отменяет регистрацию внешнего модуля.

Доступность:

Интеграция.

Примечание:

Если профиля с именем, указанным в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если внешнего модуля с именем, указанным в параметре <ИмяМодуля>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
