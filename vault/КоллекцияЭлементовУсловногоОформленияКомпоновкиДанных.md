Элементы коллекции:

[ЭлементУсловногоОформленияКомпоновкиДанных](ЭлементУсловногоОформленияКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы условного оформления.  
Возможно обращение к элементам условного оформления посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
