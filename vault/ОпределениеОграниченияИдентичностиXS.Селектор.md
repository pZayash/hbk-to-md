ОпределениеОграниченияИдентичностиXS (XSIdentityConstraintDefinition)

Селектор (Selector)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОпределенияXPathXS](ОпределенияXPathXS.md).   
Содержит компонент ограничивающего выражения XPath.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Представляет свойство selector (<http://www.w3.org/TR/xmlschema-1/#selector>) информационного пространства определения. По данному выражению XPath производится выбор узлов документа XML, для которых необходимо применить ограничения идентичности.

Использование в версии:

Доступен, начиная с версии 8.1.
