КоллекцияОформленийДат (DateAppearanceCollection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата, для которой необходимо получить оформление.

Возвращаемое значение:

Тип: [КлючИЗначение](КлючИЗначение_(KeyAndValue).md).   
Ключом является дата, а значением - [ОформлениеДаты](ОформлениеДаты_(DateAppearance).md).

Описание:

Получает оформление даты для указанной даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.0.
