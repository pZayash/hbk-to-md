# Расширение поля формы для поля дендрограммы.Выбор

Расширение поля формы для поля дендрограммы (Form field extension for a dendrogram field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<ЗначениеДендрограммы>, <СтандартнаяОбработка>)

Параметры:

<ЗначениеДендрограммы>

Тип: [ЭлементДендрограммы](ЭлементДендрограммы_(DendrogramItem).md), [СвязьДендрограммы](СвязьДендрограммы_(DendrogramLink).md).   
Выбранное значение.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Событие, возникающее при выборе дендрограммы (например, при двойном щелчке мыши).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
