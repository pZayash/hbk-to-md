# РежимОбработкиПодкаталоговZIP

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [ZIP](ZIP.md)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [РежимОбработкиПодкаталоговФайлаАрхива](РежимОбработкиПодкаталоговФайлаАрхива_(ArchiveFileSubDirProcessingMode).md)

Доступен, начиная с версии 8.0.

Значения

[НеОбрабатывать (DontProcess)](РежимОбработкиПодкаталоговZIP.НеОбрабатывать_(ZIPSubDirProcessingMode.DontProcess).md)  
[ОбрабатыватьРекурсивно (ProcessRecursively)](РежимОбработкиПодкаталоговZIP.ОбрабатыватьРекурсивно_(ZIPSubDirProcessingMode.ProcessRecursively).md)  

Описание:

Содержит варианты обработки масок файлов в именах для упаковки в ZIP архив.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЗаписьZipФайла](ЗаписьZipФайла_(ZipFileWriter).md), метод [Добавить](ЗаписьZipФайла.Добавить_(ZipFileWriter.Add).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.

Описание изменено в версии 8.3.26.
