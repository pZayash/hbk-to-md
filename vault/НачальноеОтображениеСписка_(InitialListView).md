# НачальноеОтображениеСписка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Значения

Описание:

Определяет набор вариантов перехода на строку связанного с табличным полем списка при открытии табличного поля.  
Используется для определения свойства [НачальноеОтображениеСписка](ТабличноеПоле.НачальноеОтображениеСписка_(TableBox.InitialListView).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется.
