ФайловыйПоток (FileStream)

Размер (Size)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Размер()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает размер данных в байтах.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Примечание:

Если [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) установлено в [Ложь](lang__def_BooleanFalse.md), то при вызове этого метода будет сгенерировано исключение.
