МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

ДобавитьВСписокИзПаролей (AddToListFromPasswords)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ДобавитьВСписокИзПаролей(<СписокПроверкиРаскрытияПароля>)

Параметры:

<СписокПроверкиРаскрытияПароля> (обязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md).   
Список проверки раскрытия пароля.  
Список может быть размещен в массиве строк, в файле, содержащем пароли, разделенные переносом строки (в этом случае указывается имя файла), передан в виде потока или представлен как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание:

Добавляет в список проверки раскрытия пароля значения, удаляя дубликаты. Количество значений после добавления не может превышать 100000.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| СписокПроверкиРаскрытияПароля = Новый Массив(); СписокПроверкиРаскрытияПароля.Добавить("Qwerty"); СписокПроверкиРаскрытияПароля.Добавить("12345"); СписокПроверкиРаскрытияПароля.Добавить("Password"); ДополнительныеНастройкиАутентификации. СписокПроверкиРаскрытияПароля.ДобавитьВСписокИзПаролей(СписокПроверкиРаскрытияПароля); |
