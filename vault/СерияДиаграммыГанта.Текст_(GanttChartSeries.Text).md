СерияДиаграммыГанта (GanttChartSeries)

Текст (Text)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет представления серии в диаграмме Ганта. Если не задано, используется строковое представление значения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
