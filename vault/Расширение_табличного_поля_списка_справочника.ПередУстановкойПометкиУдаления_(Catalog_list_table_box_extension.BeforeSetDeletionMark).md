Расширение табличного поля списка справочника (Catalog list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке справочника.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](СправочникОбъект.Имя_справочника.ПередЗаписью_(CatalogObject.Catalog_name.BeforeWrite).md) и [ПриЗаписи](СправочникОбъект.Имя_справочника.ПриЗаписи_(CatalogObject.Catalog_name.OnWrite).md) объекта [СправочникОбъект.](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)<[Имя справочника](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
