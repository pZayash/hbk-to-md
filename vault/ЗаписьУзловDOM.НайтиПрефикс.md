`НайтиПрефикс`(`<URIПространстваИмен>`: `Строка`) → `Строка, Неопределено`
<!-- signature:end -->

ЗаписьУзловDOM (DOMNodeWriter)

НайтиПрефикс (LookupPrefix)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПрефикс(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен, для которого необходимо найти префикс.

Возвращаемое значение:

Тип: Строка, Неопределено.   
Если префикс не обнаружен (соответствие URI пространства имен не было записано), возвращается Неопределено.

Описание:

Осуществляет поиск префикса для указанного URI пространства имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Префикс должен быть поставлен в соответствие указанному пространству имен при помощи методов [ЗаписьУзловDOM.ЗаписатьСоответствиеПространстваИмен](ЗаписьУзловDOM.ЗаписатьСоответствиеПространстваИмен.md) или [ЗаписьУзловDOM.ЗаписатьАтрибут](ЗаписьУзловDOM.ЗаписатьАтрибут.md).

Использование в версии:

Доступен, начиная с версии 8.1.
