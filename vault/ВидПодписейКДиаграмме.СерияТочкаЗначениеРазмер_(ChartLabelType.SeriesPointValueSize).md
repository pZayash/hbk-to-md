ВидПодписейКДиаграмме (ChartLabelType)

СерияТочкаЗначениеРазмер (SeriesPointValueSize)

Доступен, начиная с версии 8.3.10.

Описание:

В качестве подписи выводится представление серии, представление точки, значение серии в точке. Дополнительно в пузырьковой диаграмме выводится значение размера.

Использование в версии:

Доступен, начиная с версии 8.3.10.
