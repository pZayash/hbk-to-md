БизнесПроцессВыборка.<Имя бизнес-процесса> (BusinessProcessSelection.<Business process name>)

ПометкаУдаления (DeletionMark)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак пометки на удаление бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - бизнес-процесс помечен на удаление; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.0.
