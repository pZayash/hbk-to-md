ОпределениеТипаДокументаDOM (DOMDocumentType)

КлонироватьУзел (CloneNode)

Доступен, начиная с версии 8.1.

Синтаксис:

КлонироватьУзел(<ГлубокоеКопирование>)

Параметры:

<ГлубокоеКопирование> (обязательный)

Тип: Булево.   
Признак "глубокого" копирования.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Копия данного узла.

Описание:

Копирует (клонирует) данный узел.  
Для копии не определены родительский узел и пользовательские данные.  
Копирование узла [Элемент](ТипУзлаDOM.Элемент.md) копирует все атрибуты и их значения, включая сгенерированные XML процессором значения по умолчанию.   
Копирование дочерних узлов производится только в случае установленного признака "глубокого" копирования. "Глубокое" копирование включает текст, содержащийся в узле [Элемент](ТипУзлаDOM.Элемент.md), если текст содержится как дочерний узел [Текст](ТипУзлаDOM.Текст.md).  
Копирование узлов [Атрибут](ТипУзлаDOM.Атрибут.md) отличается от копирования их как части узла [Элемент](ТипУзлаDOM.Элемент.md) тем, что признак явного указания свойства [Указан](АтрибутDOM.Указан.md) будет установлен в значение Истина. Копирование узлов [Атрибут](ТипУзлаDOM.Атрибут.md) всегда выполняется с копированием дочерних узлов (представляющих значение атрибута), вне зависимости от режима копирования.  
Копирование узла [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) автоматически конструирует поддерево, если соответствующий узел [Сущность](ТипУзлаDOM.Сущность.md) доступен, вне зависимости от режима копирования.  
Копирование других типов узлов просто возвращает копию данного узла.   
В случае копирования поддерева в режиме Только для чтения создает редактируемую копию, но дочерние узлы копии узла [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) останутся в режиме Только для чтения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
