ЗащищенноеСоединениеOpenSSL (OpenSSLSecureConnection)

По объектам используемых сертификатов

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый ЗащищенноеСоединениеOpenSSL(<СертификатКлиента>, <СертификатыУдостоверяющихЦентров>, <РежимПроверкиОтзываTLSСертификатаСервера>)

Параметры:

<СертификатКлиента> (необязательный)

Тип: [СертификатКлиентаОС](СертификатКлиентаОС_(OSClientCertificate).md), [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), [СертификатКлиентаФайл](СертификатКлиентаФайл_(FileClientCertificate).md), [Неопределено](lang__def_Undefined.md).   
Клиентский сертификат OpenSSL, если используется.  
Если не указан, то клиентский сертификат не предоставляется.  
В веб-клиенте параметр игнорируется.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<СертификатыУдостоверяющихЦентров> (необязательный)

Тип: [СертификатыУдостоверяющихЦентровФайл](СертификатыУдостоверяющихЦентровФайл_(FileCertificationAuthorityCertificates).md), [СертификатыУдостоверяющихЦентровОС](СертификатыУдостоверяющихЦентровОС_(OSCertificationAuthorityCertificates).md), [Неопределено](lang__def_Undefined.md).   
Сертификаты удостоверяющих центров OpenSSL, если используются.  
Если не указан, то проверка сертификата сервера не осуществляется.  
В веб-клиенте параметр игнорируется.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<РежимПроверкиОтзываTLSСертификатаСервера> (необязательный)

Тип: [РежимПроверкиОтзываTLSСертификатаСервера](РежимПроверкиОтзываTLSСертификатаСервера_(ServerTLSCertificateRevocationCheckMode).md).   
Режим проверки отзыва TLS-сертификата сервера.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Создает защищенное соединение OpenSSL, использующего указанные источники клиентского сертификата и сертификатов удостоверяющих центров.  
Если не указывается источник сертификатов удостоверяющих центров, то сертификат сервера не проверяется.  
Если не указывается источник клиентского сертификата, то соединение возможно только с серверами, не требующими предоставления клиентского сертификата.  
В веб-клиенте параметры <СертификатКлиента>, <СертификатыУдостоверяющихЦентров> игнорируются

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.24.
