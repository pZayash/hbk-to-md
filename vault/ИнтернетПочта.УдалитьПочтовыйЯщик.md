ИнтернетПочта (InternetMail)

УдалитьПочтовыйЯщик (DeleteMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя почтового ящика. Может содержать в себе иерархию. В качестве разделителя используется символ, который доступен через свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md). В большинстве случаев это символ '/'.

Описание:

Удаляет заданный почтовый ящик IMAP со всем его содержимым.  
Символ-разделитель иерархии почтовых ящиков, используемый текущим почтовым сервером, доступен с помощью свойства [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Работает только при подключении, установленном по протоколу IMAP.

Пример:

|  |
| --- |
| Почта.УдалитьПочтовыйЯщик("inbox/maibox1/maibox2"); |

См. также:

[ИнтернетПочта](ИнтернетПочта.md), свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
