Описание:

Объект этого типа возвращается методом [Выбрать](БизнесПроцессМенеджер.Имя_бизнес-процесса.Выбрать.md) у объекта типа [БизнесПроцессМенеджер.](БизнесПроцессМенеджер.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессМенеджер.Имя_бизнес-процесса.md)> и представляет собой специализированный способ перебора бизнес-процессов.  
Обход бизнес-процессов выполняется системой динамически. Это означает, что использование выборки не считывает все бизнес-процессы сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большие списки бизнес-процессов и не загружает в память всех элементов выборки.  
Важно! Данному объекту доступны все свойства объекта [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>.

Доступность:

Сервер, толстый клиент, внешнее соединение.
