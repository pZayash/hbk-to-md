ТаблицыДляИзмененияСхемыЗапроса (QuerySchemaTablesForUpdate)

Получить (Get)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции, который требуется получить.

Возвращаемое значение:

Тип: [ТаблицаДляИзмененияСхемыЗапроса](ТаблицаДляИзмененияСхемыЗапроса_(QuerySchemaTableForUpdate).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
