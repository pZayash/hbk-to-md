ВыраженияСхемыЗапроса (QuerySchemaExpressions)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: Для группировки

Синтаксис:

Добавить(<ПолеГруппировки>)

Параметры:

<ПолеГруппировки> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md), [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса.md), Строка.   
Выражение или доступное поле из источника схемы запроса.

Описание варианта метода:

Добавляет выражение в группировку.

Вариант синтаксиса: Для отбора

Синтаксис:

Добавить()

Описание варианта метода:

Добавляет выражение в отбор.

Вариант синтаксиса: Для отбора из выражения схемы запроса или строки

Синтаксис:

Добавить(<Условие>)

Параметры:

<Условие> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md), Строка.   
Указывает выражение или условие в виде строки.

Описание варианта метода:

Добавляет выражение в отбор.

Вариант синтаксиса: Для отбора из доступного поля

Синтаксис:

Добавить(<Условие>)

Параметры:

<Условие> (обязательный)

Тип: [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса.md).   
Доступное поле, по которому будет создано условие отбора. Имя параметра будет сгенерировано автоматически.  
Например:  

|  |
| --- |
| ПриходнаяНакладная.Проведен = &Проведен |

Описание варианта метода:

Добавляет выражение в отбор.

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
