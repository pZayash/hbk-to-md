ДиалогВыбораШрифта (FontChooseDialog)

Шрифт (Font)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт_(Font).md).   
Содержит шрифт, устанавливаемый при открытии диалога.  
После выбора содержит выбранный шрифт.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.0.
