# Расширение таблицы формы для табличных частей

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Интерфейс.md">Интерфейс</a> › [Таблица формы](Таблица_формы.md)

Доступен, начиная с версии 8.2.

Свойства:

[ОтборСтрок (RowFilter)](Расширение_таблицы_формы_для_табличных_частей.ОтборСтрок.md)  

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы.md), применимые для табличных частей.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока.md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель.md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки.md), используется идентификатор строки.  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные.md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки.md) возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Таблица1 = ЭлементыФормы.ТаблицаОстатков; Отбор = Новый Структура; Отбор.Вставить("НашДолг", 10); Таблица1.ОтборСтрок = Новый ФиксированнаяСтруктура(Отбор); |

Использование в версии:

Доступен, начиная с версии 8.2.
