ФорматДняШкалыВремени (TimeScaleDayFormat)

ДеньНеделиДеньМесяца (WeekDayMonthDay)

Доступен, начиная с версии 8.0.

Описание:

Предпочтительный формат дня для элемента шкалы времени может быть представлен как ДеньНедели + ДеньМесяца.

Использование в версии:

Доступен, начиная с версии 8.0.
