Расширение формы клиентского приложения для отчета (Client application form extension for reports)

АвтоОтображениеСостояния (AutoShowState)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимАвтоОтображенияСостояния](РежимАвтоОтображенияСостояния_(AutoShowStateMode).md).   
Содержит режим отображения состояния отчета.  
Значение [Авто](РежимАвтоОтображенияСостояния.Авто_(AutoShowStateMode.Auto).md) эквивалентно значению [Отображать](РежимАвтоОтображенияСостояния.Отображать_(AutoShowStateMode.Show).md). По умолчанию данное свойство имеет значение [Авто](РежимАвтоОтображенияСостояния.Авто_(AutoShowStateMode.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
