# МенеджерПрогрессивногоВебПриложения.Установлено

МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

Установлено (IsInstalled)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Установлено()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В веб-клиенте возвращает значение [Истина](lang__def_BooleanTrue.md), если прогрессивное веб-приложение поддерживается и установлено.  
Для тонкого, толстого, мобильного клиентов возвращаемое значение всегда [Ложь](lang__def_BooleanFalse.md).

Описание:

Определяет, установлено ли прогрессивное веб-приложение в веб-клиенте.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
