Файл (File)

ЭтоКаталогАсинх (IsDirectoryAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЭтоКаталогАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Истина](lang__def_BooleanTrue.md), если связанный с текущим объектом файловый объект каталогом.

Описание:

Асинхронная функция. Определяет, соответствует ли файловый объект каталогу или файлу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Применима только к файлам, которые реально существуют.  
Для работы функции на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
