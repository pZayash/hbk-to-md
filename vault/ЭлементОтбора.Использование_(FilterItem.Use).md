ЭлементОтбора (FilterItem)

Использование (Use)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования элемента отбора в фильтре. Если свойство имеет значение [Истина](lang__def_BooleanTrue.md), то будет выполняться фильтрация по данному элементу отбора, иначе элемент отбора не будет участвовать в фильтрации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
