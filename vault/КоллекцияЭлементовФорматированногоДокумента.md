Доступен, начиная с версии 8.3.1.

Элементы коллекции:

[ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента.md), [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы форматированного документа.  
Возможно обращение к элементу форматированного документа посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит коллекцию элементов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
