# КоллекцияЗамещающихЭлементовПланировщика

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md)

Доступен, начиная с версии 8.3.6.

Элементы коллекции:

[ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются замещающие элементы.  
Возможно обращение к замещающим элементам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию замещающих элементов планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
