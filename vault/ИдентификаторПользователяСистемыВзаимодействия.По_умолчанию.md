ИдентификаторПользователяСистемыВзаимодействия (CollaborationSystemUserID)

По умолчанию

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ИдентификаторПользователяСистемыВзаимодействия()

Описание:

Создает "пустой" идентификатор, который может быть использован при подборе получателей сообщения системы взаимодействия в значении "Никого не оповещать".

Использование в версии:

Доступен, начиная с версии 8.3.15.
