СводнаяДиаграмма (PivotChart)

МинимальнаяШиринаЗначенияГоризонтальнойШкалы (HorizontalScaleValueMinWidth)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет минимальную ширину значения (в символах), отображаемого на горизонтальной оси (оси точек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется при [ГоризонтальнаяПоддержкаМасштаба](СводнаяДиаграмма.ГоризонтальнаяПоддержкаМасштаба_(PivotChart.HorizontalScaleKeeping).md) = МинимальнаяШирина.

Использование в версии:

Доступен, начиная с версии 8.0.
