Форма (Form)

Открыть (Open)

Доступен, начиная с версии 8.0.

Синтаксис:

Открыть()

Описание:

Открывает форму и делает ее окно активным.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма = Справочники.Номенклатура.ПолучитьФормуСписка(); Форма.Открыть(); |
