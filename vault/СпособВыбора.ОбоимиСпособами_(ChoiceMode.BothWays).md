СпособВыбора (ChoiceMode)

ОбоимиСпособами (BothWays)

Доступен, начиная с версии 8.0.

Описание:

Способ выбора определяется свойством БыстрыйВыбор расширения поля ввода для установленного в поле ввода типа.

Использование в версии:

Доступен, начиная с версии 8.0.
