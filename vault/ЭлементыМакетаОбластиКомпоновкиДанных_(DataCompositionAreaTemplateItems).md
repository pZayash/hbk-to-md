Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных_(DataCompositionAreaTemplateField).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы макета области компоновки данных.  
Возможно обращение к элементу области макета компоновки данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Данный объект представляет собой коллекцию объектов типа [ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных_(DataCompositionAreaTemplateField).md). Данные объекты могут содержаться в коллекции в произвольном порядке. Данные, содержащиеся в данных объектах, используются при выводе в документы различных форматов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
