ОбъектМетаданных: ПланВидовХарактеристик (MetadataObject: ChartOfCharacteristicTypes)

ДополнительнаяФормаГруппы (AuxiliaryFolderForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Описывает форму, которая будет вызвана системой для выполнения просмотра и редактирования группы плана видов характеристик, в случае если основная форма не установлена или не может быть использована в конкретном варианте использования 1С:Предприятия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
