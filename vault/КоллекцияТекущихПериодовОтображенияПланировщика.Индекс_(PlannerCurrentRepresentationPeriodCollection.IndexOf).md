КоллекцияТекущихПериодовОтображенияПланировщика (PlannerCurrentRepresentationPeriodCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ПериодОтображенияПланировщика](ПериодОтображенияПланировщика_(PlannerRepresentationPeriod).md).   
Элемент, для которого необходимо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс указанного элемента из коллекции текущих периодов отображения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
