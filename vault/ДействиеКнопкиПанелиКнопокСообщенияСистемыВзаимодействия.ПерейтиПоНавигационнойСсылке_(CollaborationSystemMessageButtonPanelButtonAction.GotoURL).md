ПерейтиПоНавигационнойСсылке (GotoURL)

Доступен, начиная с версии 8.3.25.

Описание:

Действие кнопки выполняет переход по навигационной ссылке, указанной в свойстве [НавигационнаяСсылка](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.НавигационнаяСсылка_(CollaborationSystemMessageButtonPanelButton.URL).md) кнопки.

Доступен, начиная с версии 8.3.25.
