ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

ИспользованиеРазделенияДанных (DataSeparationUse)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: ПараметрСеанса](ОбъектМетаданных_ПараметрСеанса_(MetadataObject_SessionParameter).md).   
Содержит параметр сеанса, значение которого задает использование разделителя в данном сеансе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
