ПользовательИнформационнойБазы (InfoBaseUser)

АутентификацияOpenIDConnect (OpenIDConnectAuthentication)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, разрешена ли пользователю аутентификация по протоколу OpenID Connect.   
Истина - аутентификация разрешена.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
