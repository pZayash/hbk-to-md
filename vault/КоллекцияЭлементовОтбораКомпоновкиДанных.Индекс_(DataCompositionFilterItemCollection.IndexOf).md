КоллекцияЭлементовОтбораКомпоновкиДанных (DataCompositionFilterItemCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных_(DataCompositionFilterItem).md), [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных_(DataCompositionFilterItemGroup).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
