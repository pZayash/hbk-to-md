# Расширение формы записи регистра сведений.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md) › [Расширение формы записи регистра сведений](Расширение_формы_записи_регистра_сведений_(Information_register_record_form_extension).md) › [События](Расширение_формы_записи_регистра_сведений__События.md)

Расширение формы записи регистра сведений (Information register record form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.ПриЗаписи_(InformationRegisterRecordSet.Information_register_name.OnWrite).md) объекта [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
