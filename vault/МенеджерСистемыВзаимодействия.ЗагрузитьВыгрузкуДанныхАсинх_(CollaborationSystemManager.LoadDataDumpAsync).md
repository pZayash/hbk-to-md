МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ЗагрузитьВыгрузкуДанныхАсинх (LoadDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ЗагрузитьВыгрузкуДанныхАсинх(<ПараметрыДиалога>, <Описание>)

Параметры:

<ПараметрыДиалога> (необязательный)

Тип: [ПараметрыДиалогаПомещенияФайлов](ПараметрыДиалогаПомещенияФайлов_(PutFilesDialogParameters).md).   
Параметры диалога загрузки выгрузки данных.

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).  
Обещание выполняется при окончании загрузки резервной копии.

Описание:

Асинхронная функция. Загружает выгрузку данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
