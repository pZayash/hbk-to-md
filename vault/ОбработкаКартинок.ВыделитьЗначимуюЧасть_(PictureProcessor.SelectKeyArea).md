ОбработкаКартинок (PictureProcessor)

ВыделитьЗначимуюЧасть (SelectKeyArea)

Не рекомендуется использовать, начиная с версии 8.3.14.

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВыделитьЗначимуюЧасть(<Картинка>, <Чувствительность>)

Параметры:

<Картинка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка, в которой нужно выделить значимую часть.

<Чувствительность> (необязательный)

Тип: [Число](lang__def_Number.md).   
Определяет степень включения деталей исходного изображения в результирующее при выделении значимой части. Число от 1 до 10.  
Чем меньше значение чувствительности, тем больше "шумов изображения" отсекается.  
Значение по умолчанию: 5.

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md).   
Картинка, в которой выделена подпись.

Описание:

Выделяет значимую часть в картинке. Например, подпись или печать для дальнейшего размещения их в печатаемых документах.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Не рекомендуется использовать, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
