Расширение группы формы для страницы (Form group extension for a page)

ВертикальнаяПрокруткаПриСжатии (VerticalScrollOnReduceSize)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Задает использование вертикальной прокрутки, если страница растягивается по вертикали и высота ее содержимого больше высоты самой страницы.   
При этом свойство [ВертикальнаяПрокрутка](ФормаКлиентскогоПриложения.ВертикальнаяПрокрутка_(ClientApplicationForm.VerticalScroll).md) формы должно иметь значение [ИспользоватьПриНеобходимости](ВертикальнаяПрокруткаФормы.ИспользоватьПриНеобходимости_(VerticalFormScroll.UseIfNecessary).md).  
Если [Неопределено](lang__def_Undefined.md), использование вертикальной прокрутки определяется автоматически и значение свойства интерпретируется как [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
