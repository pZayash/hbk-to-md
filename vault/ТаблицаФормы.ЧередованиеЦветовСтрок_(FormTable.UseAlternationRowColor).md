ТаблицаФормы (FormTable)

ЧередованиеЦветовСтрок (UseAlternationRowColor)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Управляет режимом чередования цветов строк.  
[Истина](lang__def_BooleanTrue.md) - цвета чередуются.  
[Ложь](lang__def_BooleanFalse.md) - цвета не чередуются.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).  
Для таблиц, сохраненных в предыдущих версиях платформы, при открытии форм, отображающих таблицы, значение [Истина](lang__def_BooleanTrue.md) интерпретируется как [Неопределено](lang__def_Undefined.md)(Авто).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.27.
