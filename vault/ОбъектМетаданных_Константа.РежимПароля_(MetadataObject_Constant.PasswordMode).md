# ОбъектМетаданных: Константа.РежимПароля

ОбъектМетаданных: Константа (MetadataObject: Constant)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает отображение данных константы в режиме пароля (символы заменяются звездочками при отображении).   
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
