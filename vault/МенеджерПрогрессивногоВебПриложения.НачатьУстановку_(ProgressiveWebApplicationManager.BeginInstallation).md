# МенеджерПрогрессивногоВебПриложения.НачатьУстановку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Прогрессивное веб-приложение](Прогрессивное_веб-приложение-2.md) › [МенеджерПрогрессивногоВебПриложения](МенеджерПрогрессивногоВебПриложения_(ProgressiveWebApplicationManager).md) › [Методы](МенеджерПрогрессивногоВебПриложения__Методы.md)

МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

НачатьУстановку (BeginInstallation)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьУстановку()

Описание:

Инициирует установку прогрессивного веб-приложения, показывая соответствующий диалог.  
Если прогрессивное веб-приложение не доступно к установке (свойство [Доступно](МенеджерОкнаВнешнегоСайта.Доступно_(ExternalSiteWindowManager.Enabled).md) в значении [Ложь](lang__def_BooleanFalse.md)), тогда при вызове данного метода будет сгенерировано исключение "Прогрессивное веб-приложение не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
