# ПланВидовРасчетаОбъект.<Имя плана видов расчета>.Записать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [ПланВидовРасчетаОбъект.<Имя плана видов расчета>](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md) › [Методы](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета__Методы.md)

ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает объект вид расчета. При записи выполняется проверка правильности таблицы вытесняющих видов расчета и ведущих видов расчета. При заполнении предопределенных табличных частей [ВедущиеВидыРасчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.ВедущиеВидыРасчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name.LeadingCalculationTypes).md) и [ВытесняющиеВидыРасчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ВытесняющиеВидыРасчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.DisplacingCalculationTypes).md) проверяется отсутствие циклических ссылок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
