Конструкторы:

[По умолчанию](СертификатыУдостоверяющихЦентровОС.По_умолчанию.md)  

Описание:

Объект сертификатов удостоверяющих центров OpenSSL, загружаемых из системного хранилища сертификатов операционной системы.

- В ОС Windows работает полностью аналогично объекту [СертификатыУдостоверяющихЦентровWindows](СертификатыУдостоверяющихЦентровWindows.md).
- В ОС Linux работает полностью аналогично объекту [СертификатыУдостоверяющихЦентровLinux](СертификатыУдостоверяющихЦентровLinux.md):
  - Поддерживаются только сертификаты, записанные в PEM-формате.
  - Если в каталоге находится много файлов, инициализация объекта [СертификатыУдостоверяющихЦентровLinux](СертификатыУдостоверяющихЦентровLinux.md) может занять длительное время.
  - При необходимости инициализации различных соединений рекомендуется создавать объект [СертификатыУдостоверяющихЦентровОС](СертификатыУдостоверяющихЦентровОС.md) один раз и использовать его повторно, в противном случае в ОС Linux возможно снижение производительности.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.8.
