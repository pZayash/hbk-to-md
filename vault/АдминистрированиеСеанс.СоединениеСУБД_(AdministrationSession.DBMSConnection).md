# АдминистрированиеСеанс.СоединениеСУБД

АдминистрированиеСеанс (AdministrationSession)

СоединениеСУБД (DBMSConnection)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит номер соединения с СУБД в терминах СУБД в том случае, если в момент получения списка выполняется запрос к СУБД, открыта транзакция или определены временные таблицы (то есть захвачено соединение с СУБД).   
Пустая строка означает, что соединение с СУБД не захвачено.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
