# Диаграмма.УстановитьЗначение

Диаграмма (Chart)

УстановитьЗначение (SetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьЗначение(<Точка>, <Серия>, <Значение>, <Расшифровка>, <Подсказка>)

Параметры:

<Точка> (обязательный)

Тип: [Число](lang__def_Number.md), [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md).   
Индекс точки либо сама точка диаграммы.

<Серия> (обязательный)

Тип: [Число](lang__def_Number.md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Индекс серии либо сама серия диаграммы.

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение. При построении диаграммы значения с типами, отличными от [Число](lang__def_Number.md), будут интерпретироваться как числа.

<Расшифровка> (необязательный)

Тип: Произвольный.   
Значение, используемое при расшифровке диаграммы.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Содержит текст всплывающей подсказки, которая появляется при наведении курсора к значению диаграммы.  
Если параметр задан, то стандартная подсказка, формируемая системой как <Серия> + <Точка> + <Значение>, будет заменена на указанный текст.

Описание:

Устанавливает значение в заданной точке и серии диаграммы. Дополнительно может быть установлена расшифровка значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Диаграмма.УстановитьЗначение(Точка, Серия, ОбъемПродаж); |

Доступен, начиная с версии 8.0.
