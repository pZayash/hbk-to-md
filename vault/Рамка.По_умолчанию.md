Рамка (Border)

Заданного типа и толщины

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Рамка(<ТипРамки>, <Толщина>)

Параметры:

<ТипРамки> (обязательный)

Тип: [ТипРамкиЭлементаУправления](ТипРамкиЭлементаУправления_(ControlBorderType).md).   
Тип рамки.

<Толщина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Толщина рамки (в логических единицах). Если в параметре <ТипРамки> указано значение [БезРамки](ТипРамкиЭлементаУправления.БезРамки_(ControlBorderType.WithoutBorder).md), то максимальная толщина рамки - 1.  
Значение по умолчанию: 1.

Описание:

Создает рамку требуемого типа и толщины.

Пример:

|  |
| --- |
| Рамка = Новый Рамка(ТипРамкиЭлементаУправления.Двойная, 2); |

Использование в версии:

Доступен, начиная с версии 8.0.
