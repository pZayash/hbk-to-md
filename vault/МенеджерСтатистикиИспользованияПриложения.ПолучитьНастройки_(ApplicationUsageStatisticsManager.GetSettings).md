МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьНастройки()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает текст настроек событий/экранов статистики использования приложения, установленный с помощью метода [УстановитьНастройки](МенеджерСтатистикиИспользованияПриложения.УстановитьНастройки_(ApplicationUsageStatisticsManager.SetSettings).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).
