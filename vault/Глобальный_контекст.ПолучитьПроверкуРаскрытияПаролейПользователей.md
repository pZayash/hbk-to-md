Глобальный контекст (Global context)

ПолучитьПроверкуРаскрытияПаролейПользователей (GetUserPasswordCompromiseCheck)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьПроверкуРаскрытияПаролейПользователей()

Возвращаемое значение:

Тип: Булево.   

Описание:

Проверяет, установлены ли ограничения на ввод пользователями скомпрометированных паролей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
