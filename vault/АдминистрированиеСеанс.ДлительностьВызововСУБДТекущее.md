АдминистрированиеСеанс (AdministrationSession)

ДлительностьВызововСУБДТекущее (DBMSCallDurationCurrent)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит интервал времени, прошедший с момента начала выполнения запроса, в случае, если сеанс выполняет запрос к СУБД, в миллисекундах.   
Если сеанс не выполняет запроса к СУБД – 0.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
