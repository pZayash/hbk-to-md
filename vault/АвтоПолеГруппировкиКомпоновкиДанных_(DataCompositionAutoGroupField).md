# АвтоПолеГруппировкиКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Настройки компоновки данных](Настройки_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Свойства:

[Использование (Use)](АвтоПолеГруппировкиКомпоновкиДанных.Использование_(DataCompositionAutoGroupField.Use).md)  

Описание:

Автоматическое поле группировки. Перед использованием оно будет преобразовано в набор полей группировки.  
Формирование набора происходит следующим образом: выбираются используемые выбранные поля, которые:

- доступны для использования в полях группировки;
- не являются ресурсами;
- не являются реквизитами других выбранных полей;
- не являются реквизитами существующих полей группировки.

Если поле уже включено в данные поля группировки, повторно оно не добавляется.  
В случае, если один элемент структуры настроек содержит [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных_(DataCompositionAutoGroupField).md), [АвтоВыбранноеПолеКомпоновкиДанных](АвтоВыбранноеПолеКомпоновкиДанных_(DataCompositionAutoSelectedField).md) и [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных_(DataCompositionAutoOrderItem).md), они преобразовываются в след. порядке:  
1. [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных_(DataCompositionAutoGroupField).md);  
2. [АвтоВыбранноеПолеКомпоновкиДанных](АвтоВыбранноеПолеКомпоновкиДанных_(DataCompositionAutoSelectedField).md);  
3. [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных_(DataCompositionAutoOrderItem).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/settings}. Имя типа XDTO: GroupItemAuto.

Использование в версии:

Доступен, начиная с версии 8.1.
