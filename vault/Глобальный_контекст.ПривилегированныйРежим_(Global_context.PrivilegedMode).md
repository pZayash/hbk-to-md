Глобальный контекст (Global context)

ПривилегированныйРежим (PrivilegedMode)

Доступен, начиная с версии 8.2.

Синтаксис:

ПривилегированныйРежим()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - включен привилегированный режим исполнения кода; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет включение привилегированного режима исполнения кода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В клиентском коде клиент-серверной информационной базы всегда возвращает [Ложь](lang__def_BooleanFalse.md).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьПривилегированныйРежим](Глобальный_контекст.УстановитьПривилегированныйРежим_(Global_context.SetPrivilegedMode).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
