ДинамическийСписок (DynamicList)

УстановитьОбязательноеИспользование (SetRequiredUse)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьОбязательноеИспользование(<Поле>, <Использование>)

Параметры:

<Поле> (обязательный)

Тип: [Строка](lang__def_String.md).   
Поле, значение которого требуется обязательно получать на клиенте.

<Использование> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает обязательность выбора поля.  
[Истина](lang__def_BooleanTrue.md) - поле обязательно для выборки.  
[Ложь](lang__def_BooleanFalse.md) - поле может не выбираться.

Описание:

Устанавливает признак обязательной выборки указанной колонки и передачи на сторону клиента.   
Если для колонки обязательность выборки не установлена и колонки не видимы (например скрыта колонка в таблице), данные по этой колонке выбираться не будут.  
Если требуется обращаться к данным из языка, для такой колонки необходимо установить признак обязательного использования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
