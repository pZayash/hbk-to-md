ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтключитьОбработчикОжидания (DetachIdleHandler)

Доступен, начиная с версии 8.2.

Синтаксис:

ОтключитьОбработчикОжидания(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры, которую необходимо исключить из обработки ожидания.

Описание:

Отключает ранее подключенную процедуру - обработчик ожидания, подключенный ранее методом формы [ПодключитьОбработчикОжидания](ФормаКлиентскогоПриложения.ПодключитьОбработчикОжидания_(ClientApplicationForm.AttachIdleHandler).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
