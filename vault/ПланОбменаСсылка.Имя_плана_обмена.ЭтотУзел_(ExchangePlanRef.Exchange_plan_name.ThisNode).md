ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

ЭтотУзел (ThisNode)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то данный объект является текущим узлом информационной базы. В противном случае, объект является другим узлом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
