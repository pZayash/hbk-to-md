Расширение табличного поля структуры настроек компоновки данных (Settings structure table box extension)

ПодробнаяНастройка (DetailedSetting)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Режим подробной настройки влияет на состав полей текущих данных табличного поля и на состав свойств объекта [ТекущиеДанныеСтруктурыНастроекКомпоновкиДанных](ТекущиеДанныеСтруктурыНастроекКомпоновкиДанных_(DataCompositionSettingsStructureCurrentData).md).  
Полями текущих данных могут быть: [ПараметрыСхемыКомпоновкиДанных](ПараметрыСхемыКомпоновкиДанных_(DataCompositionSchemaParameters).md), [ПоляГруппировкиКомпоновкиДанных](ПоляГруппировкиКомпоновкиДанных_(DataCompositionGroupFields).md), [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных_(DataCompositionSelectedFields).md), [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md), [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных_(DataCompositionOrder).md), [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных_(DataCompositionConditionalAppearance).md), [ЗначенияПараметровВыводаТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаТаблицыКомпоновкиДанных_(DataCompositionTableOutputParameterValues).md), [ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных_(DataCompositionChartOutputParameterValues).md), [ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных_(DataCompositionTableGroupOutputParameterValues).md), [ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных_(DataCompositionGroupOutputParameterValues).md), [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных_(DataCompositionOutputParameterValues).md), [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных_(DataCompositionOutputParameterValues).md), [ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных_(DataCompositionChartGroupOutputParameterValues).md), [ПользовательскиеПоляКомпоновкиДанных](ПользовательскиеПоляКомпоновкиДанных_(DataCompositionUserFields).md).  
В случае подробных настроек текущие данные всегда относятся к текущей строке структуры настроек, а их состав определяется наличием у объекта соответствующих свойств.  
В случае кратких настроек напрямую от текущей строки зависят только поля группировки. Остальные поля текущих данных постоянно позволяют редактировать основные настройки, т.е. соответствующие свойства объекта "Настройки компоновки данных", которому принадлежит текущая строка структуры настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
