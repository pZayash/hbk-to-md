ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Примечание (Comment)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md).   
Содержит примечание к области.   
Обратите внимание: для установки текста примечания следует использовать свойство [Текст](РисунокТабличногоДокумента.Текст_(SpreadsheetDocumentDrawing.Text).md) (например: Область.Примечание.Текст = "Щелкните левой кнопкой мыши для увеличения").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.2.
