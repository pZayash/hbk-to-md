# КоллекцияВыделенныхДат.Удалить

КоллекцияВыделенныхДат (SelectedDateCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По дате

Синтаксис:

Удалить(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Удаляемая дата.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Позиция удаляемой даты.

Описание:

Удаляет конкретную дату из коллекции выделенных дат.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
