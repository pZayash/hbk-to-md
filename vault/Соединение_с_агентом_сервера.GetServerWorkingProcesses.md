Соединение с агентом сервера (IServerAgentConnection)

GetServerWorkingProcesses (GetServerWorkingProcesses)

Доступен, начиная с версии 8.1.

Синтаксис:

GetServerWorkingProcesses(<Кластер>, <Сервер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, рабочие процессы которого необходимо получить.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Рабочий сервер, для которого необходимо получить запускаемые на нем рабочие процессы.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Каждое описание соединения представлено объектом с интерфейсом [Рабочий процесс](Рабочий_процесс.md).

Описание:

Получает массив рабочих процессов, зарегистрированных на заданном рабочем сервере заданного кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
