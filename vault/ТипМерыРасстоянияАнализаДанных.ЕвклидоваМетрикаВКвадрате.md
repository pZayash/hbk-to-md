ТипМерыРасстоянияАнализаДанных (DataAnalysisDistanceMetricType)

ЕвклидоваМетрикаВКвадрате (SquaredEuclidean)

Доступен, начиная с версии 8.0.

Описание:

Расстояние между двумя объектами вычисляется по формуле:  
D = sum( Wi \* (Xi - Yi)^2), где Xi, Yi - значения атрибутов объектов, Wi - весовой коэффициент. i - номер атрибута, от 1 до n, где n - число атрибутов.

Использование в версии:

Доступен, начиная с версии 8.0.
