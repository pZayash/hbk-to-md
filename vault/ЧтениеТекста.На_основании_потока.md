ЧтениеТекста (TextReader)

На основании потока

Доступен, начиная с версии 8.3.10.

Синтаксис:

Новый ЧтениеТекста(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения текста.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Определяет кодировку текста читаемого потока (согласован с текстовым документом). Если параметр не задан, формат текста будет определен автоматически по сигнатуре BOM в начале файла; если сигнатура BOM в файле отсутствует, файл будет открыт в кодировке [ANSI](КодировкаТекста.ANSI_(TextEncoding.ANSI).md).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет строку, разделяющую строки в текстовом файле.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание:

Открывает поток для чтения. Аналогичен методу [Открыть](ЧтениеТекста.Открыть_(TextReader.Open).md).  
Метод [Закрыть](ЧтениеТекста.Закрыть_(TextReader.Close).md) при использовании данного конструктора не выполняет никаких действий.

Пример:

|  |
| --- |
| ДлинныйТекст = Новый ЧтениеТекста("c:\temp\file.txt", КодировкаТекста.UTF16); |

Доступен, начиная с версии 8.3.10.
