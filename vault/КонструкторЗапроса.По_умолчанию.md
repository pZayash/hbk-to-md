КонструкторЗапроса (QueryWizard)

На основании текста запроса

Доступен, начиная с версии 8.0.

Синтаксис:

Новый КонструкторЗапроса(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.   
Содержит текст редактируемого запроса.  
Значение по умолчанию: Пустая строка.

Описание:

Создает объект [КонструкторЗапроса](КонструкторЗапроса.md).

Пример:

|  |
| --- |
| КонструкторЗапроса = Новый КонструкторЗапроса; |

Использование в версии:

Доступен, начиная с версии 8.0.
