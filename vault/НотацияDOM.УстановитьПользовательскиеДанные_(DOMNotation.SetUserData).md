НотацияDOM (DOMNotation)

УстановитьПользовательскиеДанные (SetUserData)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьПользовательскиеДанные(<Ключ>, <Данные>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ пользовательских данных.

<Данные> (обязательный)

Тип: Произвольный.   
Пользовательские данные, которые необходимо сохранить в узле.

Возвращаемое значение:

Прежние пользовательские данные для указанного ключа или [Неопределено](lang__def_Undefined.md), если указанному ключу не соответствовали никакие пользовательские данные.

Описание:

Устанавливает данные пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
