ЧтениеДанных (DataReader)

НачатьКопированиеВ (BeginCopyTo)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьКопированиеВ(<ОписаниеОповещения>, <Поток>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Описание процедуры, которая будет вызвана после завершения операции.  
Процедура должна содержать следующие параметры:

- <Количество> - тип Число. Фактическое количество скопированных байтов;
- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md)

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md), [ЗаписьДанных](ЗаписьДанных.md).   
Поток или объект [ЗаписьДанных](ЗаписьДанных.md), в который требуется скопировать данные.

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые требуется скопировать. Если не задано, то будут скопированы все данные до конца вложенного потока.

Описание:

Начинает копирование заданного количества байтов из объекта [ЧтениеДанных](ЧтениеДанных.md) в указанный поток или объект [ЗаписьДанных](ЗаписьДанных.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЧтениеДанных](ЧтениеДанных.md), метод [КопироватьВ](ЧтениеДанных.КопироватьВ.md)  
[ЧтениеДанных](ЧтениеДанных.md), метод [КопироватьВАсинх](ЧтениеДанных.КопироватьВАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
