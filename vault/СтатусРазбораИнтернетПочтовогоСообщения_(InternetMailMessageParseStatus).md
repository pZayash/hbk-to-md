# СтатусРазбораИнтернетПочтовогоСообщения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Почта](Почта.md)

Доступен, начиная с версии 8.3.6.

Значения

[ОбнаруженыОшибки (ErrorsDetected)](СтатусРазбораИнтернетПочтовогоСообщения.ОбнаруженыОшибки_(InternetMailMessageParseStatus.ErrorsDetected).md)  
[ОшибокНеОбнаружено (ErrorsNotDetected)](СтатусРазбораИнтернетПочтовогоСообщения.ОшибокНеОбнаружено_(InternetMailMessageParseStatus.ErrorsNotDetected).md)  

Описание:

Содержит виды статусов разбора почтового сообщения:

- [ОбнаруженыОшибки](СтатусРазбораИнтернетПочтовогоСообщения.ОбнаруженыОшибки_(InternetMailMessageParseStatus.ErrorsDetected).md) - ошибки в структуре письма при разборе были обнаружены ошибки. Объект [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md) может содержать ошибки.
- [ОшибокНеОбнаружено](СтатусРазбораИнтернетПочтовогоСообщения.ОшибокНеОбнаружено_(InternetMailMessageParseStatus.ErrorsNotDetected).md) - при разборе письма ошибки обнаружены не были.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
