ОбъектМетаданных: ПараметрФункциональныхОпций (MetadataObject: FunctionalOptionsParameter)

Использование (Use)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Измерение](ОбъектМетаданных_Измерение_(MetadataObject_Dimension).md), [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник_(MetadataObject_Catalog).md).   
Указывает на объект, значение которого будет являться параметром для получения значения функциональной опции.  
Параметры используется в случае, если для функциональной опции в свойстве [Хранение](ОбъектМетаданных_ФункциональнаяОпция.Хранение_(MetadataObject_FunctionalOption.Location).md) указан реквизит справочника или ресурс регистра сведений, т.е. в случае, когда в одном объекте (справочнике или регистре сведений) может храниться набор значений опций.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство должно быть заполнено.

Использование в версии:

Доступен, начиная с версии 8.2.
