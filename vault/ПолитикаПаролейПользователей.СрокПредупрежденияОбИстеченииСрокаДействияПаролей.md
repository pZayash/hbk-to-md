ПолитикаПаролейПользователей (UserPasswordPolicy)

СрокПредупрежденияОбИстеченииСрокаДействияПаролей (PasswordExpirationNotificationPeriod)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Срок предупреждения об истечении срока действия паролей в секундах.  
  
Значение должно быть в интервале от 0 до 4294967295.  
Значение по умолчанию: 432000 (5 суток).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
