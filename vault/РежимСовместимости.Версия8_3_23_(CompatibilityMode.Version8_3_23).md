РежимСовместимости (CompatibilityMode)

Версия8\_3\_23 (Version8\_3\_23)

Доступен, начиная с версии 8.3.24.

Описание:

Усовершенствования версии 8.3.24 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.23.

Использование в версии:

Доступен, начиная с версии 8.3.24.
