Доступен, начиная с версии 8.3.26.

Элементы коллекции:

[ЭлементФайлаАрхива](ЭлементФайлаАрхива_(ArchiveFileEntry).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элемент коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Методы:

[Количество (Count)](ЭлементыФайлаАрхива.Количество_(ArchiveFileEntries.Count).md)  
[Найти (Find)](ЭлементыФайлаАрхива.Найти_(ArchiveFileEntries.Find).md)  
[Получить (Get)](ЭлементыФайлаАрхива.Получить_(ArchiveFileEntries.Get).md)  

Описание:

Содержит коллекцию элементов файла архива.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
