Глобальный контекст (Global context)

ПолучитьBase64СтрокуИзБуфераДвоичныхДанных (GetBase64StringFromBinaryDataBuffer)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьBase64СтрокуИзБуфераДвоичныхДанных(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Буфер двоичных данных.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Преобразует буфер двоичных данных в строку формата Base64.  
Полученный текст разбивается на строки длиной 64 символа. В качестве разделителя строк используется сочетание символов CR+LF.   
Если целевой декодер Base64 не поддерживает переводы строк, требуется дополнительно обработать строки, чтобы удалить из нее разделители.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
