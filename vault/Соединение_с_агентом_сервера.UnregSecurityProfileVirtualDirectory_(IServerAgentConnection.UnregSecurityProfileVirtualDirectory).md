Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileVirtualDirectory (UnregSecurityProfileVirtualDirectory)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileVirtualDirectory(<Кластер>, <ИмяПрофиля>, <ЛогическийURL>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должна быть отменена регистрация виртуального каталога.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<ЛогическийURL> (обязательный)

Тип: [Строка](lang__def_String.md).   
Логический URL виртуального каталога, регистрация которого отменяется.

Описание:

Отменяет регистрацию виртуального каталога.

Доступность:

Интеграция.

Примечание:

Если профиля, указанного в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если логического URL виртуального каталога, указанного в параметре <ЛогическийURL>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
