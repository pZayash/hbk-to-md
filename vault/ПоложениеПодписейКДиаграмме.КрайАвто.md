КрайАвто (EdgeAuto)

Доступен, начиная с версии 8.2.

Описание:

Подписи отображаются аналогично [КрайВнутри](ПоложениеПодписейКДиаграмме.КрайВнутри.md) если места достаточно, в противном случае поведение аналогично [Край](ПоложениеПодписейКДиаграмме.Край.md).

Использование в версии:

Доступен, начиная с версии 8.2.
