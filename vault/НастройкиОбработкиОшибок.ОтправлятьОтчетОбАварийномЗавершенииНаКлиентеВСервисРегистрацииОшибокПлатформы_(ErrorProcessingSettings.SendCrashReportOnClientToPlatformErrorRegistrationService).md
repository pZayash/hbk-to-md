НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОтчетОбАварийномЗавершенииНаКлиентеВСервисРегистрацииОшибокПлатформы (SendCrashReportOnClientToPlatformErrorRegistrationService)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость отправки информации об аварийном завершении клиента в сервис фирмы "1С".

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).
