МенеджерОкнаВнешнегоСайта (ExternalSiteWindowManager)

ОтключитьОбработчикСообщений (DetachMessageHandler)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОтключитьОбработчикСообщений(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, заданной при вызове метода [ПодключитьОбработчикСообщений](МенеджерОкнаВнешнегоСайта.ПодключитьОбработчикСообщений_(ExternalSiteWindowManager.AttachMessageHandler).md).

Описание:

Отключает ранее подключенную процедуру-обработчик.   
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Если окно внешнего сайта не доступно (свойство [Доступно](МенеджерОкнаВнешнегоСайта.Доступно_(ExternalSiteWindowManager.Enabled).md) имеет значение [Ложь](lang__def_BooleanFalse.md)), тогда вызов данного метода приведет к исключению "Окно внешнего сайта не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[СообщениеВнешнегоСайта](СообщениеВнешнегоСайта_(MessageFromExternalSite).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.
