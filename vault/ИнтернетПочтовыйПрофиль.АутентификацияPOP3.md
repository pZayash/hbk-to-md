ИнтернетПочтовыйПрофиль (InternetMailProfile)

АутентификацияPOP3 (POP3Authentication)

Не рекомендуется использовать, начиная с версии 8.3.1.

Рекомендуется использовать:

- [ТолькоЗащищеннаяАутентификацияPOP3](ИнтернетПочтовыйПрофиль.ТолькоЗащищеннаяАутентификацияPOP3.md)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [СпособPOP3Аутентификации](СпособPOP3Аутентификации.md).   
Содержит способ аутентификации при подключении на POP3 сервере. Значение по умолчанию - [Обычная](objects__catalog2__catalog2086__POP3AuthenticationMode__properties__General5792.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.1.

Описание изменено в версии 8.3.21.
