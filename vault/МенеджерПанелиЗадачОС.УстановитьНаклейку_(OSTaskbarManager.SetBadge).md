# МенеджерПанелиЗадачОС.УстановитьНаклейку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [МенеджерПанелиЗадачОС](МенеджерПанелиЗадачОС_(OSTaskbarManager).md) › [Методы](МенеджерПанелиЗадачОС__Методы.md)

МенеджерПанелиЗадачОС (OSTaskbarManager)

УстановитьНаклейку (SetBadge)

Доступен, начиная с версии 8.3.8.

Вариант синтаксиса: ИзКартинки

Синтаксис:

УстановитьНаклейку(<Картинка>, <СкрытьПриВыборе>)

Параметры:

<Картинка> (обязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Иконка, которая будет отображаться в правом нижнем углу кнопки программы на панели задач.  
Формат картинки – любой, но рекомендуется использовать файл в формате ".ico", размер картинки – 16x16 пикселей (рекомендуется также иметь картинки для размеров 20х20 и 24х24 пикселей).

<СкрытьПриВыборе> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), то иконка будет убрана при нажатии на кнопку программы на панели задач или при программной активации окна приложения (например, при получении окном фокуса, при открытии формы).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Устанавливает значок из картинки.

Вариант синтаксиса: ИзЧисла

Синтаксис:

УстановитьНаклейку(<Число>, <СкрытьПриВыборе>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число от 0 до 99, которое будет отображаться в правом нижнем углу кнопки программы на панели задач.

<СкрытьПриВыборе> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установленно значение [Истина](lang__def_BooleanTrue.md), иконка будет убрана при нажатии на кнопку программы на панели задач или при программной активации окна приложения (например, при получении окном фокуса, при открытии формы).  
В ОС Unity параметр игнорируется.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Устанавливает числовой значок.

Описание:

Устанавливает наклейку уведомления поверх кнопки программы на панели задач.  
Работает под Windows 7 и старше, а также Ubuntu Unity. Под другими ОС не производит никаких действий.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

Вариант метода с установкой картинки в качестве наклейки уведомления работает только в Windows 7 и старше.  
Наклейки на кнопках приложений в панели задач Windows отображаются только, если в свойствах панели задач флажок "Использовать маленькие значки" снят, а в свойствах меню "Пуск" флажок "Крупные значки" - установлен.  
Число, которое устанавливается в качестве наклейки уведомления в Windows 7 и старше, должно принадлежать отрезку [0, 99], под Ubuntu Unity - [0, 9999], иначе оно будет заменено на соответствующее граничное значение.  
В ОС Ubuntu Unity параметр &СкрытьПриВыборе игнорируется.

Пример:

|  |
| --- |
| ПанельЗадачОС.УстановитьНаклейку(БиблиотекаКартинок.Иконка1, Истина); ПанельЗадачОС.УстановитьНаклейку(77); |

Использование в версии:

Доступен, начиная с версии 8.3.8.
