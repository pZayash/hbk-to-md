ИспользованиеАтрибутаXS (XSAttributeUse)

ЛексическоеЗначение (LexicalValue)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Представляет лексическое представление значения атрибутов fixed или default объявления атрибута (http://www.w3.org/TR/xmlschema-1/#a-value\_constraint).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
