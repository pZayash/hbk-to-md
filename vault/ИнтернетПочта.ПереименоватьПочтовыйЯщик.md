ИнтернетПочта (InternetMail)

ПереименоватьПочтовыйЯщик (RenameMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

ПереименоватьПочтовыйЯщик(<Имя>, <НовоеИмя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя почтового ящика, которое требуется переименовать.

<НовоеИмя> (обязательный)

Тип: Строка.   
Новое имя, которое будет назначено почтовому ящику.

Описание:

Выполняет переименование почтового ящика.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Работает только при подключении, установленном по протоколу IMAP.

Использование в версии:

Доступен, начиная с версии 8.2.
