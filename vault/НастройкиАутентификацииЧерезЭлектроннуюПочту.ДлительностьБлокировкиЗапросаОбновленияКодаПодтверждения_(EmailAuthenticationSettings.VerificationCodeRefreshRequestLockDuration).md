НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

ДлительностьБлокировкиЗапросаОбновленияКодаПодтверждения (VerificationCodeRefreshRequestLockDuration)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет длительность блокировки запроса отправки нового кода аутентификации через электронную почту (в секундах).  
Целое число в диапазоне от 30 до 999999999. Значение по умолчанию: 30.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
