СерияДиаграммы (ChartSeries)

ТипГрафическогоПредставления (GraphicalRepresentationType)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [ТипГрафическогоПредставленияСерииДиаграммы](ТипГрафическогоПредставленияСерииДиаграммы_(ChartSeriesGraphicalRepresentationType).md).   
Задает тип графического представления серии, отличный от типа диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Используется только для диаграмм следующих видов:  
[ГистограммаСНакоплением](ТипДиаграммы.ГистограммаСНакоплением_(ChartType.StackedColumn).md),  
[ГистограммаСНакоплениемГоризонтальная](ТипДиаграммы.ГистограммаСНакоплениемГоризонтальная_(ChartType.StackedBar).md),  
[График](ТипДиаграммы.График_(ChartType.Line).md),  
[ГрафикПоШагам](ТипДиаграммы.ГрафикПоШагам_(ChartType.Step).md),  
[ГрафикСНакоплением](ТипДиаграммы.ГрафикСНакоплением_(ChartType.StackedLine).md),  
[ГрафикСОбластями](ТипДиаграммы.ГрафикСОбластями_(ChartType.Area).md),  
[ГрафикСОбластямиИНакоплением](ТипДиаграммы.ГрафикСОбластямиИНакоплением_(ChartType.StackedArea).md),  
[ГрафикСОбластямиНормированный](ТипДиаграммы.ГрафикСОбластямиНормированный_(ChartType.NormalizedArea).md),  
[ГистограммаОбъемная](ТипГрафическогоПредставленияСерииДиаграммы.ГистограммаОбъемная_(ChartSeriesGraphicalRepresentationType.Column3D).md),  
[ГистограммаСНакоплениемОбъемная](ТипДиаграммы.ГистограммаСНакоплениемОбъемная_(ChartType.StackedColumn3D).md),  
[ГистограммаНормированнаяОбъемная](ТипДиаграммы.ГистограммаНормированнаяОбъемная_(ChartType.NormalizedColumn3D).md),

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).
