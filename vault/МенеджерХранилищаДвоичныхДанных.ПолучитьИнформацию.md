МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ПолучитьИнформацию (GetInformation)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьИнформацию()

Возвращаемое значение:

Тип: [ИнформацияХранилищаДвоичныхДанных](ИнформацияХранилищаДвоичныхДанных.md).   

Описание:

Получает информацию о текущем состоянии хранилища двоичных данных (в том числе время последней очистки хранилища).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
