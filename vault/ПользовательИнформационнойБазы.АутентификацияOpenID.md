ПользовательИнформационнойБазы (InfoBaseUser)

АутентификацияOpenID (OpenIDAuthentication)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, разрешена ли пользователю аутентификация по протоколу OpenID.   
Истина - аутентификация разрешена.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.2.
