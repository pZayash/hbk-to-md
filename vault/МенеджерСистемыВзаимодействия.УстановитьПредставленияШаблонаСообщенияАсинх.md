МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьПредставленияШаблонаСообщенияАсинх (SetMessageTemplatePresentationsAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПредставленияШаблонаСообщенияАсинх(<Идентификатор>, <ПредставлениеШаблона>, <ПредставленияПодстановок>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия.md).   
Идентификатор шаблона сообщения, для которого устанавливается описание.

<ПредставлениеШаблона> (обязательный)

Тип: Строка.   
Представление шаблона.

<ПредставленияПодстановок> (необязательный)

Тип: [Массив](Массив.md).   
Массив строк - список представлений подстановок.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания – Неопределено.

Описание:

Асинхронная функция. Для интеграции типа WhatsApp устанавливает представление шаблона и представления его подстановок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [УстановитьПредставленияШаблонаСообщения](МенеджерСистемыВзаимодействия.УстановитьПредставленияШаблонаСообщения.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
