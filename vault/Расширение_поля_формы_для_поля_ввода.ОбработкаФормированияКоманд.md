Расширение поля формы для поля ввода (Form field extension for a text box)

ОбработкаФормированияКоманд (CommandGenerateProcessing)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОбработкаФормированияКоманд(<ПараметрыКоманд>, <Команды>)

Параметры:

<ПараметрыКоманд>

Тип: [ПараметрыФормированияКомандПоляВвода](ПараметрыФормированияКомандПоляВвода.md).   
Параметры формирования команд поля ввода.

<Команды>

Тип: [Массив](Массив.md).   
Список команд, которые можно добавить. Значения элементов массива имеют тип [ОписаниеКомандыПоляВвода](ОписаниеКомандыПоляВвода.md). Значения другого типа игнорируются.   
При вызове список заполнен командами контекстного меню по умолчанию. Команды, добавленные в контекстное меню через свойство поля формы [КонтекстноеМеню](ПолеФормы.КонтекстноеМеню.md), не будут добавлены в данный массив.

Описание:

Событие при нажатии правой кнопки мыши в поле ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
