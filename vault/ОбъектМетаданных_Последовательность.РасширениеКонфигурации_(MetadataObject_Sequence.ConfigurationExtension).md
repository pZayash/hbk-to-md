ОбъектМетаданных: Последовательность (MetadataObject: Sequence)

РасширениеКонфигурации (ConfigurationExtension)

Доступен, начиная с версии 8.3.8.

Синтаксис:

РасширениеКонфигурации()

Возвращаемое значение:

Тип: [РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объект относится к основной конфигурации.

Описание:

Получает объект описания расширения конфигурации, с помощью которого объект метаданных был добавлен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
