Запрос (Query)

МенеджерВременныхТаблиц (TempTablesManager)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [МенеджерВременныхТаблиц](МенеджерВременныхТаблиц_(TempTablesManager).md), [Неопределено](lang__def_Undefined.md).   
Содержит менеджер временных таблиц, в котором будут создаваться временные таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Неопределено](lang__def_Undefined.md) - для запроса не установлен менеджер временных таблиц.

Использование в версии:

Доступен, начиная с версии 8.1.
