# МенеджерКриптографии.НачатьПолучениеКонтейнераПодписейКриптографии

МенеджерКриптографии (CryptoManager)

НачатьПолучениеКонтейнераПодписейКриптографии (BeginGettingCryptoSignaturesContainer)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НачатьПолучениеКонтейнераПодписейКриптографии(<ОписаниеОповещения>, <Подпись>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения получения информации со следующими параметрами:

- <КонтейнерПодписейКриптографии> - помещается [КонтейнерПодписейКриптографии](КонтейнерПодписейКриптографии_(CryptoSignaturesContainer).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Подпись> (необязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Подпись для разбора.  
Если параметр типа [Строка](lang__def_String.md), тогда он обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или Поток, тогда он задает данные подписи.

Описание:

Выполняет разбор двоичных данных подписи.   
  
По завершении будет вызвана процедура, указанная в <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
