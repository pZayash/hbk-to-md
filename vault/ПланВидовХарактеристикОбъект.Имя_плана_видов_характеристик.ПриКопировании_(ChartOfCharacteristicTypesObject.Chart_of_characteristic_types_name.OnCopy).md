ПриКопировании (OnCopy)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриКопировании(<ОбъектКопирования>)

Параметры:

<ОбъектКопирования>

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md).   
Исходный элемент, из которого выполняется копирование.

Описание:

Возникает при создании элемента плана видов характеристик копированием.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
