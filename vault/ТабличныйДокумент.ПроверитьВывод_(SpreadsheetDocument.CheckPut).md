ТабличныйДокумент (SpreadsheetDocument)

ПроверитьВывод (CheckPut)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьВывод(<Таблицы>)

Параметры:

<Таблицы> (обязательный)

Тип: [Массив](Массив_(Array).md), [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Массив из проверяемых таблиц или табличный документ.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - умещаются; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, умещаются ли переданные табличные документы на страницу при печати.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При возникновении проблем с получением информации о текущем принтере (например, в системе не установлено ни одного принтера), будет вызвано исключение.  
Следует учитывать, если для табличного документа установлены поля, размер которых меньше размера полей, установленных для принтера, на котором документ будет напечатан, то при печати содержимое некоторых строк может не уместиться на странице, даже если метод возвращает значение [Истина](lang__def_BooleanTrue.md).

Пример:

|  |
| --- |
| Т.Добавить(ШапкаДокумента); Т.Добавить(СтрокаДокумента); Т.Добавить(ПодвалДокумента); Если Не ТабДок.ПроверитьВывод(Т) Тогда     Сообщить("Документ не умещается на страницу!"); КонецЕсли; |

Доступен, начиная с версии 8.0.
