# ПараметрыЗаписиJSON.СимволыОтступа

ПараметрыЗаписиJSON (JSONWriterSettings)

СимволыОтступа (PaddingSymbols)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет символы отступа, используемые при записи документа JSON.  
Свойство не используется, если свойство [ПереносСтрокJSON](ПереносСтрокJSON_(JSONLineBreak).md) установлено в значение [Нет](ПереносСтрокJSON.Нет_(JSONLineBreak.None).md).  
После создания объекта данное свойство имеет значение - пустая строка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
