# Кластер серверов.MaxMemoryTimeLimit

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Кластер серверов](Кластер_серверов_(IClusterInfo).md) › [Свойства](Кластер_серверов__Свойства.md)

Кластер серверов (IClusterInfo)

MaxMemoryTimeLimit (MaxMemoryTimeLimit)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет максимальный период превышения критического объема памяти в секундах.   
Рабочий процесс будет перезапущен, если объем виртуального адресного пространства рабочего процесса постоянно превышает критический объем памяти в течение указанных в свойстве секунд.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
