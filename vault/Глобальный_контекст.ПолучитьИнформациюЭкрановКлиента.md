Глобальный контекст (Global context)

ПолучитьИнформациюЭкрановКлиента (GetClientDisplaysInformation)

Доступен, начиная с версии 8.3.2.

Синтаксис:

ПолучитьИнформациюЭкрановКлиента()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив.md).   
Неопределено, если сеанс не связан с клиентом (фоновое задание, внешнее соединение).

Описание:

Получает информацию об экранах подключенных устройств.   
Информация предоставляется в виде фиксированного массива объектов [ИнформацияЭкранаКлиента](ИнформацияЭкранаКлиента.md). Первый элемент массива описывает основной экран.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме запуска "веб-клиент":

- в массиве всегда возвращается один элемент; для браузеров Mozilla Firefox и Google Chrome - это монитор, на котором открыто главное окно, для MS Internet Explorer и Apple Safari - это основной монитор;
- [DPI](ИнформацияЭкранаКлиента.DPI.md) заполняется только в MS Internet Explorer, для остальных браузеров - 0.

В режиме запуска "сервер":  
Информация об экранах передается с клиента при старте сеанса или при вызове метода [ОбновитьИнтерфейс](Глобальный_контекст.ОбновитьИнтерфейс.md).

Использование в версии:

Доступен, начиная с версии 8.3.2.
