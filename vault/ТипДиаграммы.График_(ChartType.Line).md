ТипДиаграммы (ChartType)

График (Line)

Доступен, начиная с версии 8.0.

Описание:

Линейный график. Соседние значения внутри одной серии соединяются прямыми линиями, значения в точках могут быть отмечены маркерами. Строится на основании данных, представленных одной или несколькими сериями значений для нескольких точек.

Использование в версии:

Доступен, начиная с версии 8.0.
