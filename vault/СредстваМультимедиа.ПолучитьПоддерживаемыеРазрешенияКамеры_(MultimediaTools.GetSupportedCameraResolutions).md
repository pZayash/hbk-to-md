ПолучитьПоддерживаемыеРазрешенияКамеры (GetSupportedCameraResolutions)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьПоддерживаемыеРазрешенияКамеры(<ТипКамеры>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства_(DeviceCameraType).md).   
Показывает, для какой камеры будут получены разрешения.  
Значение по умолчанию: [Авто](ТипКамерыУстройства.Авто_(DeviceCameraType.Auto).md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md), [Неопределено](lang__def_Undefined.md).   
Массив [РазрешениеКамерыУстройства](РазрешениеКамерыУстройства_(DeviceCameraResolution).md). Для iOS будет возвращен тип [Неопределено](lang__def_Undefined.md).

Описание:

Получает массив доступных разрешений для выбранной камеры.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок_(MultimediaTools.PhotoSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок_(MultimediaTools.PhotoSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
