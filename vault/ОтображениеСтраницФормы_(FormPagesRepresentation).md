Описание:

Содержит варианты отображения закладок страниц в панели в форме клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Расширение группы формы для страниц](Расширение_группы_формы_для_страниц_(Form_group_extension_for_pages).md), свойство [ОтображениеСтраниц](Расширение_группы_формы_для_страниц.ОтображениеСтраниц_(Form_group_extension_for_pages.PagesRepresentation).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
