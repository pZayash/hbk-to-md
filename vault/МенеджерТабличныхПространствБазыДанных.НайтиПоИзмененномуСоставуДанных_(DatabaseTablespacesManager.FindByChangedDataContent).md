# МенеджерТабличныхПространствБазыДанных.НайтиПоИзмененномуСоставуДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличные пространства базы данных](Табличные_пространства_базы_данных.md) › [МенеджерТабличныхПространствБазыДанных](МенеджерТабличныхПространствБазыДанных_(DatabaseTablespacesManager).md) › [Методы](МенеджерТабличныхПространствБазыДанных__Методы.md)

МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

НайтиПоИзмененномуСоставуДанных (FindByChangedDataContent)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиПоИзмененномуСоставуДанных(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [ЭлементСоставаТабличногоПространстваБазыДанных](ЭлементСоставаТабличногоПространстваБазыДанных_(DatabaseTablespaceContentItem).md), ОбъектМетаданных.   
Объект метаданных, для которого выполняется поиск табличного пространства.

Возвращаемое значение:

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных_(DatabaseTablespaceManager).md).   

Описание:

Возвращает табличное пространство, которое будет хранить данные таблиц объекта метаданных после обновления размещения таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
