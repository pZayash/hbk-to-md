ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

МоментВремени (PointInTime)

Доступен, начиная с версии 8.1.

Синтаксис:

МоментВремени()

Возвращаемое значение:

Тип: [МоментВремени](МоментВремени_(PointInTime).md).   

Описание:

Получает момент времени документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // (в модуле документа ПроведениеПоПартиям)  Запрос = Новый Запрос; Запрос.УстановитьПараметр("МоментВремени", МоментВремени()); Запрос.Текст =  "ВЫБРАТЬ |    ПроведениеПоПартиям.Ссылка КАК РегламентныйДокумент |ИЗ |    Документ.ПроведениеПоПартиям КАК ПроведениеПоПартиям | |ГДЕ |    ПроведениеПоПартиям.МоментВремени >= МоментВремени"; Выборка = Запрос.Выполнить().Выбрать(); |

Доступен, начиная с версии 8.1.
