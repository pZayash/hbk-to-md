ЭлементГрафическойСхемыДействие (GraphicalSchemaItemActivity)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, которой соответствует данный элемент графической схемы.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Имеет смысл только для элемента графической схемы, полученной методами [ПолучитьКартуМаршрута](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьКартуМаршрута_(BusinessProcessManager.Business_process_name.GetFlowchart).md) и [ПолучитьКартуМаршрута](БизнесПроцессОбъект.Имя_бизнес-процесса.ПолучитьКартуМаршрута_(BusinessProcessObject.Business_process_name.GetFlowchart).md).

Использование в версии:

Доступен, начиная с версии 8.0.
