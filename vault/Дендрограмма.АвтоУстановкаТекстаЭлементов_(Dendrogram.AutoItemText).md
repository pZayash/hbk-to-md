Дендрограмма (Dendrogram)

АвтоУстановкаТекстаЭлементов (AutoItemText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак генерации текста для новых элементов по схеме "ЭлементN". Если [Ложь](lang__def_BooleanFalse.md), то у нового элемента будет пустой текст.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
