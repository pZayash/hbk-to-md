ПараметрыОбменаДанными (DataExchangeParameters)

Получатели (Recipients)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НаборУзлов](НаборУзлов.md).   
Набор узлов, для которых будут регистрироваться изменения при записи или удаления объекта данных, к которому относится объект [ПараметрыОбменаДанными](ПараметрыОбменаДанными.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
