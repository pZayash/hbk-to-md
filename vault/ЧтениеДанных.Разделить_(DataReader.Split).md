# ЧтениеДанных.Разделить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с двоичными данными](Работа_с_двоичными_данными-2.md) › [ЧтениеДанных](ЧтениеДанных_(DataReader).md) › [Методы](ЧтениеДанных__Методы.md)

ЧтениеДанных (DataReader)

Разделить (Split)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: Разделение по двоичному разделителю

Синтаксис:

Разделить()

Описание варианта метода:

Разделить данные по двоичному разделителю.

Вариант синтаксиса: По двоичному разделителю

Синтаксис:

Разделить(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Двоичный разделитель данных.

Описание варианта метода:

Разделяет остаток данных по двоичному разделителю.

Вариант синтаксиса: Разделение по строковому разделителю

Синтаксис:

Разделить(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста. Если не задана, то используется кодировка, заданная для текущего экземпляра объекта [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Описание варианта метода:

Разделить данные по строковому разделителю.

Вариант синтаксиса: По строковому разделителю

Синтаксис:

Разделить(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: [Строка](lang__def_String.md).   
Разделитель данных в виде строки.

Описание варианта метода:

Разделяет остаток данных по строковому разделителю.

Вариант синтаксиса: Разделение по набору смешанных разделетелей

Синтаксис:

Разделить(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста. Если не установлена, используется кодировка, заданная для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Описание варианта метода:

Разделить данные по набору смешанных разделителей.

Вариант синтаксиса: По набору смешанных разделителей

Синтаксис:

Разделить(<Разделители>)

Параметры:

<Разделители> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив разделителей может содержать элементы типа [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md) или [Строка](lang__def_String.md).

Описание варианта метода:

Разделяет остаток данных по набору смешанных разделителей.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив значений типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Описание:

Разделяет остаток данных по заданным разделителям.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
