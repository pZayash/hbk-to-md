Планировщик (Planner)

АвтоМинимальнаяШиринаКолонки (AutoColumnMinWidth)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет использование автоматического ограничения минимальной ширины колонки.  
Если значение [Ложь](lang__def_BooleanFalse.md), автоматическое ограничение не используется, и минимальная ширина определяется свойством [МинимальнаяШиринаКолонки](Планировщик.МинимальнаяШиринаКолонки_(Planner.MinColumnWidth).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
