ИнтернетПочта (InternetMail)

СимволРазделитель (DelimeterChar)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит разделитель, используемый для разделения папок в иерархии папок.  
Разделитель настраивается на сервере IMAP.   
На клиенте свойство указывает, какой разделитель нужно использовать при адресации иерархии папок.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
