# Форма.РазрешитьЗакрытие

Форма (Form)

РазрешитьЗакрытие (AllowClose)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет интерактивной возможностью закрытия формы.

Доступность:

Толстый клиент.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - форму нельзя закрыть интерактивно. В этом случае форму можно закрыть только методом [Закрыть](Форма.Закрыть_(Form.Close).md).

Использование в версии:

Доступен, начиная с версии 8.1.
