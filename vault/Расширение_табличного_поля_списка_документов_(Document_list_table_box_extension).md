Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для списка документов. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) содержит значение типа [ДокументСсылка.](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)<[Имя документа](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ДокументСписок.](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)<[Имя документа](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
