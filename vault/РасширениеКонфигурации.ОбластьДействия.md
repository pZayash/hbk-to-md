РасширениеКонфигурации (ConfigurationExtension)

ОбластьДействия (Scope)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации.md), Неопределено.   
Область действия расширения конфигурации.  
Для нового объекта [РасширениеКонфигурации](РасширениеКонфигурации.md) значение свойства может быть прочитано как Неопределено. При записи такого расширения это значение будет автоматически заменено на одно из значений типа [ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации.md) по следующим правилам:

- Если включено разделение расширений конфигурации, то будет выбрано значение [РазделениеДанных](ПользовательИнформационнойБазы.РазделениеДанных.md).
- В остальных случаях будет выбрано значение [ИнформационнаяБаза](ОбластьДействияРасширенияКонфигурации.ИнформационнаяБаза.md).

Эти значения также можно установить явно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации.md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
