ИнтернетПочтовыйПрофиль (InternetMailProfile)

ПортPOP3 (POP3Port)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Порт протокола POP3.   
0 - означает использовать 995 порт для SSL соединений и 110 - для незащищенных.  
Значение по умолчанию: 0.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ИнтернетПочтовыйПрофиль](ИнтернетПочтовыйПрофиль.md), свойство [АдресСервераPOP3](ИнтернетПочтовыйПрофиль.АдресСервераPOP3.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
