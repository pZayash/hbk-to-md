РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение рисунка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступно только для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), если в рисунке хранится значение (свойство [СодержитЗначение](РисунокТабличногоДокумента.СодержитЗначение.md) равно Истина).

Использование в версии:

Доступен, начиная с версии 8.0.
