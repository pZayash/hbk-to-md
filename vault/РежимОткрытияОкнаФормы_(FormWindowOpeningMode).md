Значения

[БлокироватьВесьИнтерфейс (LockWholeInterface)](РежимОткрытияОкнаФормы.БлокироватьВесьИнтерфейс_(FormWindowOpeningMode.LockWholeInterface).md)

Описание:

Содержит варианты открытия формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), свойство [РежимОткрытияОкнаФормы](ФормаКлиентскогоПриложения.РежимОткрытияОкнаФормы_(ClientApplicationForm.FormWindowOpeningMode).md)
