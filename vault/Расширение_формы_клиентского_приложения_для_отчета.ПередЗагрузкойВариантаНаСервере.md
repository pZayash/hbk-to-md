Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПередЗагрузкойВариантаНаСервере (BeforeLoadVariantAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗагрузкойВариантаНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md).   
Загружаемые настройки.

Описание:

Вызывается перед загрузкой настроек в компоновщик настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
