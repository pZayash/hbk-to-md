АдминистрированиеХранилищеДвоичныхДанных (AdministrationBinaryDataStorage)

СоздатьПолнуюРезервнуюКопию (CreateFullBackup)

Доступен, начиная с версии 8.3.27.

Синтаксис:

СоздатьПолнуюРезервнуюКопию(<Путь>)

Параметры:

<Путь> (обязательный)

Тип: Строка.   
Путь к файлу резервной копии хранилища на сервере.

Описание:

Создает полную резервную копию хранилища двоичных данных. Только для встроенного хранилища.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
