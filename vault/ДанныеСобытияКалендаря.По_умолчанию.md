ДанныеСобытияКалендаря (CalendarEventData)

По умолчанию

Доступен, начиная с версии 8.3.4.

Синтаксис:

Новый ДанныеСобытияКалендаря(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата начала события.

<Конец> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата окончания события.

Описание:

Создает данные события календаря.

Использование в версии:

Доступен, начиная с версии 8.3.4.
