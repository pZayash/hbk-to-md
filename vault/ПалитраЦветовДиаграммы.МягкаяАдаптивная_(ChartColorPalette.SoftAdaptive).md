МягкаяАдаптивная (SoftAdaptive)

Доступен, начиная с версии 8.3.10.

Описание:

Адаптивная палитра на основе мягкой палитры.  
Для линий используются более яркие цвета.

Доступен, начиная с версии 8.3.10.
