РежимСовместимости (CompatibilityMode)

Версия8\_3\_4 (Version8\_3\_4)

Доступен, начиная с версии 8.3.5.

Описание:

Усовершенствования версии 8.3.5 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.4.

Использование в версии:

Доступен, начиная с версии 8.3.5.
