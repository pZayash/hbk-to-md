# АтрибутDOM.ЕстьДочерниеУзлы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [АтрибутDOM](АтрибутDOM_(DOMAttribute).md) › [Методы](АтрибутDOM__Методы.md)

АтрибутDOM (DOMAttribute)

ЕстьДочерниеУзлы (HasChildNodes)

Доступен, начиная с версии 8.1.

Синтаксис:

ЕстьДочерниеУзлы()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - узел имеет дочерние узлы, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, имеются ли у узла дочерние узлы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
