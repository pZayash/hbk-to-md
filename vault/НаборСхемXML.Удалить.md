НаборСхемXML (XMLSchemaSet)

Удалить (Delete)

Доступен, начиная с версии 8.1.

Синтаксис:

Удалить(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен удаляемой схемы.

Описание:

Удаляет схему с указанным URI пространства имен из набора схем.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
