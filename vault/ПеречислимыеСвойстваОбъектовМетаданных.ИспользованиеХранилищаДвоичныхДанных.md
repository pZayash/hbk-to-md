ПеречислимыеСвойстваОбъектовМетаданных (MetadataObjectEnumeratedProperties)

ИспользованиеХранилищаДвоичныхДанных (BinaryDataStorageUseMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [РежимИспользованияХранилищаДвоичныхДанных](РежимИспользованияХранилищаДвоичныхДанных.md).   
Определяет вариант использования хранилища двоичных данных.  
Данное свойство недоступно при работе в режиме совместимости 8.3.22 и выше (по умолчанию, хранилище может использоваться).  
Данное свойство недоступно при работе в режиме совместимости ниже [Версия8\_3\_10](РежимСовместимости.Версия8_3_10.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
