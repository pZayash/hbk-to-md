ИнформацияОЗаписиВерсииИсторииДанных (WriteDataHistoryVersionInformation)

РазличияВерсий (VersionDifferences)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md), Неопределено.   
Различия между версиями или Неопределено, если версия создана по удалению данных.  
Значение свойства соответствует значению, возвращаемому методом [ПолучитьРазличияВерсий](МенеджерИсторииДанных.ПолучитьРазличияВерсий.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
