ОболочкаHTMLДокумента (HTMLDocumentShell)

ПолучитьДокументHTML (GetHTMLDocument)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьДокументHTML()

Возвращаемое значение:

Тип: [ДокументHTML](ДокументHTML_(HTMLDocument).md).   

Описание:

Получает исходный HTML-документ в виде дерева DOM и объекта [ДокументHTML](ДокументHTML_(HTMLDocument).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
