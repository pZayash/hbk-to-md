Описание:

Представляет абстракцию элемента управления [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md). Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством [Вид](ТестируемаяДекорацияФормы.Вид_(TestedFormDecoration.Type).md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.25.
