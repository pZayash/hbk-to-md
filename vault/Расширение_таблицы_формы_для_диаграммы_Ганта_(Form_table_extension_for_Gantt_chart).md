# Расширение таблицы формы для диаграммы Ганта

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md)

Доступен, начиная с версии 8.3.21.

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы_(FormTable).md), применимые для таблицы значений диаграммы Ганта.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель_(FormTable.CurrentParent).md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки_(FormTable.SelectedRows).md), используется идентификатор типа [ИдентификаторЗначенияДиаграммыГанта](ИдентификаторЗначенияДиаграммыГанта_(GanttChartValueID).md).  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные_(FormTable.CurrentData).md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки_(FormTable.RowData).md) возвращают объект типа [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md). Именами свойств структуры будут являться имена колонок диаграммы Ганта. Изменение этих данных не изменит диаграмму Ганта.
