Расширение поля ввода вида расчета (Calculation type text box extension)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Предоставляет доступ к управлению режимом выбора значения. Если свойство установлено в Истина, то выбор будет выполняться из выпадающего списка, если Ложь - из окна выбора вида расчета.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
