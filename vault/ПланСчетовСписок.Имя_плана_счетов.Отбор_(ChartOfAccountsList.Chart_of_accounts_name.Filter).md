ПланСчетовСписок.<Имя плана счетов> (ChartOfAccountsList.<Chart of accounts name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Содержит объект [Отбор](Отбор_(Filter).md), по которому осуществляется текущая фильтрация элементов плана счетов в списке. Имена элементов отбора совпадают с именами реквизитов плана счетов и критериев отбора, затрагивающими план счетов, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
