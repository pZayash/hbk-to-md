# Кластерный анализ

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Одним из методов математической обработки данных является кластерный анализ. Кластерный анализ представляет собой средство исследования топологической структуры совокупности объектов. Он позволяет разбить множество объектов на классы близких между собой объектов статистической совокупности. Обнаруженные этим методом классы родственных элементов, называются кластерами.
