Модуль WebSocket клиента (WebSocket client module)

ПриОшибке (OnError)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПриОшибке(<Соединение>, <КодОшибки>, <Описание>)

Параметры:

<Соединение>

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md).   
Ссылка на объект соединения.

<КодОшибки>

Тип: [Строка](lang__def_String.md).   
Код ошибки.

<Описание>

Тип: [Строка](lang__def_String.md).   
Описание ошибки.

Описание:

Обработчик ошибки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
