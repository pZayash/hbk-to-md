Соединение с рабочим процессом (IWorkingProcessConnection)

CreateBinaryDataStorageFullBackup (CreateBinaryDataStorageFullBackup)

Доступен, начиная с версии 8.3.27.

Синтаксис:

CreateBinaryDataStorageFullBackup(<BinaryDataStorage>, <Path>)

Параметры:

<BinaryDataStorage> (обязательный)

Тип: [Хранилище двоичных данных](Хранилище_двоичных_данных.md).   
Хранилище двоичных данных.

<Path> (обязательный)

Тип: Строка.   
Путь к файлу полной резервной копии на сервере.

Описание:

Создает полную резервную копию хранилища двоичных данных.Только для встроенного хранилища.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
