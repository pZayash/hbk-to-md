ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент (DataCompositionResultSpreadsheetDocumentOutputProcessor)

ВывестиЭлемент (OutputItem)

Доступен, начиная с версии 8.1.

Синтаксис:

ВывестиЭлемент(<ЭлементРезультата>)

Параметры:

<ЭлементРезультата> (обязательный)

Тип: [ЭлементРезультатаКомпоновкиДанных](ЭлементРезультатаКомпоновкиДанных_(DataCompositionResultItem).md).   
Элемент результата, выводимый в результирующий документ.

Описание:

Вывести элемент результата компоновки отчета в документ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
