ЗначенияИзмерений (DimensionValues)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ЗначенияИзмерений](ПараметрыФормированияКомандПоляПланировщика.ЗначенияИзмерений_(PlannerFieldCommandGenerateParameters.DimensionValues).md)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md), [Неопределено](lang__def_Undefined.md).   
Содержит значения измерений, однозначно определяющие заголовок измерения,если он был нажат.   
Иначе - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
