# МенеджерСистемыВзаимодействия.ВерсияСервера

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система взаимодействия](Система_взаимодействия-2.md) › [МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md) › [Методы](МенеджерСистемыВзаимодействия__Методы.md)

МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВерсияСервера (ServerVersion)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВерсияСервера()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает версию сервера системы взаимодействия, на котором зарегистрирована информационная база.  
Если приложение не зарегистрировано на сервере системы взаимодействия, то возвращает пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
