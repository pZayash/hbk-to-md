Расширение табличного поля набора записей регистра расчета (Calculation register recordset table box extension)

Обновить (Refresh)

Доступен, начиная с версии 8.0.

Синтаксис:

Обновить()

Описание:

Обновляет набор записей перечитыванием данных из базы данных.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
