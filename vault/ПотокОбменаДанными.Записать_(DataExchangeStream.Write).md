ПотокОбменаДанными (DataExchangeStream)

Записать (Write)

Доступен, начиная с версии 8.3.22.

Синтаксис:

Записать(<Значение>)

Параметры:

<Значение> (необязательный)

Записываемое значение, например: Справочник.Объект, Справочник.Ссылка, и т.д.  
  
В обработчиках [ПередНачаломОтправкиДанныхГлавному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхГлавному_(ExchangePlanObject.Exchange_plan_name.BeforeBeginSendDataToMaster).md), [ПередНачаломОтправкиДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхПодчиненному_(ExchangePlanObject.Exchange_plan_name.BeforeBeginSendDataToSlave).md) можно передать объект данных типа [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), для передачи сырых данных и обработать в обработчиках [ПриПолученииДанныхОтГлавного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтГлавного_(ExchangePlanObject.Exchange_plan_name.OnReceiveDataFromMaster).md), [ПриПолученииДанныхОтПодчиненного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтПодчиненного_(ExchangePlanObject.Exchange_plan_name.OnReceiveDataFromSlave).md).

Описание:

Записывает объект, который может быть передан в обработчики [ПередСозданиемНачальногоОбраза](ПланОбменаОбъект.Имя_плана_обмена.ПередСозданиемНачальногоОбраза_(ExchangePlanObject.Exchange_plan_name.BeforeCreateInitialImage).md), [ПередНачаломОтправкиДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхПодчиненному_(ExchangePlanObject.Exchange_plan_name.BeforeBeginSendDataToSlave).md) для записи объектов в обмене данными.

Доступность:

Мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
