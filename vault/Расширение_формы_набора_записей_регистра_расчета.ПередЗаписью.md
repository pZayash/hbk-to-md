Расширение формы набора записей регистра расчета (Calculation register record set form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением записи набора. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.ПередЗаписью.md) объекта [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
