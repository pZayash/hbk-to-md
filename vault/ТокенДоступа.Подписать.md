ТокенДоступа (AccessToken)

Подписать (Sign)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Подписать(<АлгоритмПодписи>, <ИнформацияОКлюче>)

Параметры:

<АлгоритмПодписи> (обязательный)

Тип: [АлгоритмПодписиТокенаДоступа](АлгоритмПодписиТокенаДоступа.md).   
Алгоритм подписи токена доступа.

<ИнформацияОКлюче> (необязательный)

Тип: Строка.   
Строка в формате Base64.  
Информация о ключе, который используется для формирования подписи в формате PEM.  
Данный параметр является необязательным только, если не указан алгоритм подписи, т.е. при вызове этого метода <АлгоритмПодписи> имеет значение [Нет](АлгоритмПодписиТокенаДоступа.Нет.md).

Описание:

Добавляет токену доступа подпись по указанному в параметрах алгоритму.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.26.
