Файл (File)

НачатьУстановкуТолькоЧтения (BeginSettingReadOnly)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьУстановкуТолькоЧтения(<ОписаниеОповещения>, <ТолькоЧтение>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ТолькоЧтение> (обязательный)

Тип: Булево.   
Истина - установить атрибут "Только чтение", Ложь - снять атрибут "Только чтение".

Описание:

Начинает установку атрибута файла "Только чтение".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Файл](Файл.md), метод [УстановитьТолькоЧтение](Файл.УстановитьТолькоЧтение.md)  
[Файл](Файл.md), метод [УстановитьТолькоЧтениеАсинх](Файл.УстановитьТолькоЧтениеАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
