# НаправлениеСообщенияКаналаСервисаИнтеграции

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Свойства объектов метаданных](Свойства_объектов_метаданных.md)

Доступен, начиная с версии 8.3.17.

Значения

[Отправка (Send)](НаправлениеСообщенияКаналаСервисаИнтеграции.Отправка_(IntegrationServiceChannelMessageDirection.Send).md)  
[Получение (Receive)](НаправлениеСообщенияКаналаСервисаИнтеграции.Получение_(IntegrationServiceChannelMessageDirection.Receive).md)  

Описание:

Содержит варианты направления передачи сообщений в канале сервиса интеграции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.1/data/enterprise. Имя типа XDTO: IntegrationServiceChannelMessageDirection.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
