ПодписьКриптографии (CryptoSignature)

Данные (Data)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   
Содержит подписанные данные, хранящиеся в самой подписи. Если подпись отсоединенная и не хранит данных, то содержит [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.24.
