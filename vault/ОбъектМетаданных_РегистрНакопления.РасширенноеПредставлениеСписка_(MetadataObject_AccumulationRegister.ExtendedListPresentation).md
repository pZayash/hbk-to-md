ОбъектМетаданных: РегистрНакопления (MetadataObject: AccumulationRegister)

РасширенноеПредставлениеСписка (ExtendedListPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием списка записей регистра накопления и используется при формировании заголовка формы списка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
