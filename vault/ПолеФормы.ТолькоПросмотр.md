ПолеФормы (FormField)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак возможности/невозможности редактирования отображаемых данных.  
Данное свойство имеет смысл только для тех данных, редактирование которых не запрещено прикладным смыслом и ограничениями по правам. Также некоторые виды полей не обладают возможностью редактировать отображаемые данные.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
