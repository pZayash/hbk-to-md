ПолеКалендаря (CalendarBox)

Обновить (Refresh)

Доступен, начиная с версии 8.0.

Синтаксис:

Обновить()

Описание:

Инициирует событие [ПриВыводеПериода](ПолеКалендаря.ПриВыводеПериода_(CalendarBox.OnPeriodOutput).md) для всех видимых в календаре месяцев.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ЭлементыФормы.ПроизводственныйКалендарь.Обновить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
