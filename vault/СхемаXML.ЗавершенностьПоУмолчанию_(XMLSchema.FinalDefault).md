СхемаXML (XMLSchema)

ЗавершенностьПоУмолчанию (FinalDefault)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОбъединениеЗавершенностиСхемыXS](ОбъединениеЗавершенностиСхемыXS_(XSSchemaFinalUnion).md).   
Содержит виды ограничений наследования по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет значение finalDefault (<http://www.w3.org/TR/xmlschema-1/#element-schema>) информационного пространства определения.

Использование в версии:

Доступен, начиная с версии 8.1.
