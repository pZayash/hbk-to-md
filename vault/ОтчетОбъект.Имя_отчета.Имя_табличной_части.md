ОтчетОбъект.<Имя отчета> (ReportObject.<Report name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ОтчетТабличнаяЧасть.<Имя отчета>.<Имя табличной части>.   
Набор свойств содержит табличные части отчета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ОтчетТабличнаяЧасть.<Имя отчета>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
