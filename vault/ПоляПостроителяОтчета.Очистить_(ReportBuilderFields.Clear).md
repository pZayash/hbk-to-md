ПоляПостроителяОтчета (ReportBuilderFields)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Очищает коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПостроительОтчета.ВыбранныеПоля.Очистить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
