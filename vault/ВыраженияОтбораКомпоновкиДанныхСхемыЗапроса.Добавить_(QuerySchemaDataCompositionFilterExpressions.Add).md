# ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса.Добавить

ВыраженияОтбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionFilterExpressions)

Добавить (Add)

Доступен, начиная с версии 8.3.14.

Вариант синтаксиса: Из строки

Синтаксис:

Добавить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строковое представление параметра (начинающееся с символа "", например, Парам) или имя доступного поля, добавляемого в выражения отбора.

Описание варианта метода:

Добавляет выражение отбора компоновки данных из строки.

Вариант синтаксиса: Из выражения схемы запроса

Синтаксис:

Добавить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md).   
Выражение схемы запроса, содержащее строковое представление параметра (начинающееся с символа "", например, Парам) или имя доступного поля, добавляемого в выражения отбора.

Описание варианта метода:

Добавляет выражение отбора компоновки данных из выражения схемы запроса.

Вариант синтаксиса: Из доступного поля схемы запроса

Синтаксис:

Добавить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса_(QuerySchemaAvailableField).md).   
Доступное поле схемы запроса, добавляемое в выражения отбора компоновки данных.

Описание варианта метода:

Добавляет выражение отбора компоновки данных из доступного поля схемы запроса.

Возвращаемое значение:

Тип: [ВыражениеОтбораКомпоновкиДанныхСхемыЗапроса](ВыражениеОтбораКомпоновкиДанныхСхемыЗапроса_(QuerySchemaDataCompositionFilterExpression).md).   

Описание:

Добавляет новый элемент коллекции. Если тип нового элемента не соответствует данной коллекции, добавление не выполняется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
