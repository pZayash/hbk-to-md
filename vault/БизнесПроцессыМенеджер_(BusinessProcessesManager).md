# БизнесПроцессыМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса_(BusinessProcessManager.Business_process_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры бизнес-процессов.  
Возможно обращение к менеджеру бизнес-процесса посредством оператора [...]. В качестве аргумента передается имя бизнес-процесса, как оно задано в конфигурации.  

Свойства:

[\<Имя бизнес-процесса\> (\<Business process name\>)](БизнесПроцессыМенеджер.Имя_бизнес-процесса_(BusinessProcessesManager.Business_process_name).md)  

Методы:

[ТипВсеСсылки (AllRefsType)](БизнесПроцессыМенеджер.ТипВсеСсылки_(BusinessProcessesManager.AllRefsType).md)  
[ТипВсеСсылкиТочекМаршрутаБизнесПроцессов (RoutePointsAllRefsType)](БизнесПроцессыМенеджер.ТипВсеСсылкиТочекМаршрутаБизнесПроцессов_(BusinessProcessesManager.RoutePointsAllRefsType).md)  

Описание:

Предназначен для управления бизнес-процессами и предоставляет доступ к значениям типа [БизнесПроцессыМенеджер](БизнесПроцессыМенеджер_(BusinessProcessesManager).md). Доступ к объекту осуществляется через свойство глобального контекста [БизнесПроцессы](Глобальный_контекст.БизнесПроцессы_(Global_context.BusinessProcesses).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

Пример:

|  |
| --- |
| // менеджер бизнес-процессов предоставляет доступ к менеджеру  // бизнес-процесса ВыпискаСчета.  БизнесПроцесс = БизнесПроцессы.ВыпискаСчета.СоздатьБизнесПроцесс(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [БизнесПроцессы](Глобальный_контекст.БизнесПроцессы_(Global_context.BusinessProcesses).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
