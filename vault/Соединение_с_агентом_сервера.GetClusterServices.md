Соединение с агентом сервера (IServerAgentConnection)

GetClusterServices (GetClusterServices)

Доступен, начиная с версии 8.2.

Синтаксис:

GetClusterServices(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив объектов [Сервис кластера](Сервис_кластера.md).

Описание:

Получает список сервисов кластера. Список сервисов фиксирован для кластера серверов 1С:Предприятия определенной версии. Каждый сервис, кроме сервиса сеансовых данных, работает в главном или каком-нибудь из дополнительных менеджеров кластера. Сервис сеансовых данных может работать на нескольких менеджерах одновременно. Сервис конфигурации кластера, Сервис блокировок кластера и Сервис управления предметами отладки могут работать только в главном менеджере кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
