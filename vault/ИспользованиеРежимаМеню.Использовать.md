ИспользованиеРежимаМеню (UseMenuMode)

Использовать (Use)

Доступен, начиная с версии 8.0.

Описание:

Если свойство установлено, то на кнопке справа рисуется треугольник. При нажатии на кнопку не отрабатывается событие нажатия, а появляется выпадающее меню, откуда пользователь выбирает команду.

Использование в версии:

Доступен, начиная с версии 8.0.
