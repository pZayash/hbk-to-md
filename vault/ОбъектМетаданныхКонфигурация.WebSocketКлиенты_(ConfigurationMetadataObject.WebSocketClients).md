# ОбъектМетаданныхКонфигурация.WebSocketКлиенты

ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

WebSocketКлиенты (WebSocketClients)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все WebSocket-клиенты, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент_(MetadataObject_WebSocketClient).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
