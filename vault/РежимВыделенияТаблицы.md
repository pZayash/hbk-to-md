Описание:

Содержит варианты выделения строк таблицы формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы.md), свойство [РежимВыделения](ТаблицаФормы.РежимВыделения.md)
