Описание:

Содержит варианты способов выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы MS Windows.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

См. также:

[СертификатКлиентаWindows](СертификатКлиентаWindows.md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.24.
