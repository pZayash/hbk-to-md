# ЧтениеДанных.НачатьЧтениеЦелого64

ЧтениеДанных (DataReader)

НачатьЧтениеЦелого64 (BeginReadInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтениеЦелого64(<ОписаниеОповещения>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - тип [Число](lang__def_Number.md). Результат чтения.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Описание:

Начинает чтение 64-битного целого числа из потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[ЧтениеДанных](ЧтениеДанных_(DataReader).md), метод [ПрочитатьЦелое64Асинх](ЧтениеДанных.ПрочитатьЦелое64Асинх_(DataReader.ReadInt64Async).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
