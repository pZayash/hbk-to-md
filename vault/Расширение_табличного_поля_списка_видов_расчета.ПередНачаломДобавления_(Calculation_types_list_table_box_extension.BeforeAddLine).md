# Расширение табличного поля списка видов расчета.ПередНачаломДобавления

Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления вида расчета. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), добавление выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

Описание:

Вызывается при вызове пользователем добавления вида расчета при редактировании в строке.

Доступность:

Толстый клиент.

Примечание:

Расширяется набор параметров по сравнению с одноименным событием объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

См. также:

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [ПередНачаломДобавления](ТабличноеПоле.ПередНачаломДобавления_(TableBox.BeforeAddRow).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
