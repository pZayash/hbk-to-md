КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число, [ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента.md), [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md).   
Индекс сдвигаемого элемента, либо сам сдвигаемый элемент.

<Смещение> (обязательный)

Тип: Число.   
Указывается, на сколько позиций осуществляется сдвиг.

Описание:

Сдвигает элемент форматированного документа на заданное количество позиций.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
