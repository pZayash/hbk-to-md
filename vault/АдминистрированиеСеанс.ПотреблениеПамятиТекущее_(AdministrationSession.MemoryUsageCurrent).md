АдминистрированиеСеанс (AdministrationSession)

ПотреблениеПамятиТекущее (MemoryUsageCurrent)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит объем памяти, занятый с начала выполнения текущего вызова (в байтах).   
Если в данный момент вызов не выполняется, содержит 0.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
