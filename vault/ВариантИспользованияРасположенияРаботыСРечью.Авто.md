ВариантИспользованияРасположенияРаботыСРечью (SpeechProcessingLocationUseVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ВариантИспользованияРасположенияРаботыСРечью](ВариантИспользованияРасположенияРаботыСРечью.md).   
Поиск модели выполняется локально, после чего используется внешнее распознавание, если подключено.  
В мобильном приложении (клиент, сервер) используется только внешнее распознавание.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.25.
