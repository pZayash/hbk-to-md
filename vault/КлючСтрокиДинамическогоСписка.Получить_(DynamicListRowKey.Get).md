КлючСтрокиДинамическогоСписка (DynamicListRowKey)

Получить (Get)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Получить(<ПутьКлючевогоПоля>)

Параметры:

<ПутьКлючевогоПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к ключевому полю, значение которого требуется получить.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Получает значение по ключу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
