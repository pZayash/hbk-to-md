СтраницыПанели (PanelPages)

Получить (Get)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс страницы.

Вариант синтаксиса: По имени

Синтаксис:

Получить(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя страницы.

Возвращаемое значение:

Тип: [СтраницаПанели](СтраницаПанели_(PanelPage).md).   

Описание:

Получает значение. Работает аналогично оператору [].

Доступность:

Толстый клиент.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
