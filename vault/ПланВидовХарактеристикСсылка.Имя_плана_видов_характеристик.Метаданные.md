ПланВидовХарактеристикСсылка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesRef.<Chart of characteristic types name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланВидовХарактеристик](ОбъектМетаданных_ПланВидовХарактеристик.md).   

Описание:

Предоставляет доступ к объекту описания метаданных плана видов характеристик. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.ПланыВидовХарактеристик.СвойстваОбъектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
