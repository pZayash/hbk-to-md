ПолеСводнойДиаграммы (PivotChartField)

КоличествоОткрытыхУровней (OpenLevelCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит количество открытых уровней измерения. Если = 0, то измерение не расположено ни на одной оси.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
