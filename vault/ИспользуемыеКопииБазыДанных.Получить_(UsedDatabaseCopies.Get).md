ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Получить (Get)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс используемой копии.

Возвращаемое значение:

Тип: [ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных_(UsedDatabaseCopy).md).   

Описание:

Получает используемую копию по индексу в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
