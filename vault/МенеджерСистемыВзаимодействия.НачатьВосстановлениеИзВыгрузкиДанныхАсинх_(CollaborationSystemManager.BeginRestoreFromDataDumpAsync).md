МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьВосстановлениеИзВыгрузкиДанныхАсинх (BeginRestoreFromDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

НачатьВосстановлениеИзВыгрузкиДанныхАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – [Неопределено](lang__def_Undefined.md).  
Обещание выполняется при начале восстановления данных из выгрузки.

Описание:

Асинхронная функция. Начинает восстановление данных системы взаимодействия из выгрузки данных с указанным идентификатором.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [НачатьВосстановлениеИзВыгрузкиДанных](МенеджерСистемыВзаимодействия.НачатьВосстановлениеИзВыгрузкиДанных_(CollaborationSystemManager.BeginRestoreFromDataDump).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
