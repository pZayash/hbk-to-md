ФормаКлиентскогоПриложения (ClientApplicationForm)

ОписаниеОповещенияОЗакрытии (CallbackDescriptionOnClose)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Неопределено.   
Содержит описание процедуры, которая будет вызвана после закрытия формы со следующими параметрами:  
<РезультатЗакрытия> - значение, переданное при вызове метода [Закрыть](ФормаКлиентскогоПриложения.Закрыть.md) открываемой формы,  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.22.
