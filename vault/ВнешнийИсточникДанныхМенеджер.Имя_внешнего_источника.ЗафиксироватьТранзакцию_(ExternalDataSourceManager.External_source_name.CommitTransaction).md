ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

ЗафиксироватьТранзакцию (CommitTransaction)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ЗафиксироватьТранзакцию()

Описание:

Завершает успешную транзакцию. Все изменения, внесенные в базу данных внешнего источника данных в процессе транзакции, будут записаны.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
