Свойства:

[\<Имя перерасчета\> (\<Recalculation name\>)](РегистрРасчетаПерерасчетыМенеджер.Имя_регистра_расчета.Имя_перерасчета_(CalculationRegisterRecalcsManager.Calculation_register_name.Recalculation_name).md)  

Описание:

Менеджер всех менеджеров перерасчетов регистра расчетов.  
У менеджера регистра расчета имеется свойство [Перерасчеты](РегистрРасчетаМенеджер.Имя_регистра_расчета.Перерасчеты_(CalculationRegisterManager.Calculation_register_name.Recalculations).md), предоставляющее доступ к объекту типа [РегистрРасчетаПерерасчетыМенеджер.](РегистрРасчетаПерерасчетыМенеджер.Имя_регистра_расчета_(CalculationRegisterRecalcsManager.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаПерерасчетыМенеджер.Имя_регистра_расчета_(CalculationRegisterRecalcsManager.Calculation_register_name).md)>.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

См. также:

[РегистрРасчетаМенеджер.\<Имя регистра расчета\>](РегистрРасчетаМенеджер.Имя_регистра_расчета_(CalculationRegisterManager.Calculation_register_name).md), свойство [Перерасчеты](РегистрРасчетаМенеджер.Имя_регистра_расчета.Перерасчеты_(CalculationRegisterManager.Calculation_register_name.Recalculations).md)

Доступен, начиная с версии 8.0.
