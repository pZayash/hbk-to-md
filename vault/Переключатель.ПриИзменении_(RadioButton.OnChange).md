ПриИзменении (OnChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриИзменении()

Описание:

Возникает при изменении положения переключателя.

Доступность:

Толстый клиент.

Примечание:

Всегда связано с первым в группе элементом.

Использование в версии:

Доступен, начиная с версии 8.0.
