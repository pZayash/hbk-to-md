Поток (Stream)

УстановитьРазмерАсинх (SetSizeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьРазмерАсинх(<Размер>)

Параметры:

<Размер> (обязательный)

Тип: Число.   
Устанавливаемый размер потока.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Устанавливает размер потока.   
Если текущий размер превышает заданный, поток будет сокращен до заданного размера, а информация, превышающая заданный размер, будет потеряна.  
Если текущий размер потока меньше заданного, то содержимое потока между старым и новым размером не определено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Установить длину потока можно только в том случае, если в потоке разрешены запись и изменение позиции. В противном случае будет сгенерировано исключение.
