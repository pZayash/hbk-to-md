ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

Разблокировать (Unlock)

Доступен, начиная с версии 8.0.

Синтаксис:

Разблокировать()

Описание:

Выполняет разблокировку узла, если он заблокирован данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
