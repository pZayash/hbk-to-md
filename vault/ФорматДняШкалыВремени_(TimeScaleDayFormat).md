Описание:

Содержит варианты предпочтительного формата дня для элемента шкалы времени с единицей времени [День](ТипЕдиницыШкалыВремени.День_(TimeScaleUnitType.Day).md) (т.е. трактовать день как день недели, месяца и в каком порядке).  
Реальный формат зависит также от масштаба.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: TimeScaleDayFormat.
