Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<ИндексЭлементаКоллекции>)

Параметры:

<ИндексЭлементаКоллекции> (обязательный)

Тип: Число.   
Индекс элемента коллекции.

Возвращаемое значение:

Тип: [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения.md).

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
