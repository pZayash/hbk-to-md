ТестируемаяДекорацияФормы (TestedFormDecoration)

ТекущаяВидимость (CurrentVisible)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущаяВидимость()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект видимый; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Возвращает признак того, что пользователь может увидеть элемент без настройки формы, но путем переключения закладок и полос прокрутки.  
Если объект является колонкой таблицы и таблица не находится в режиме редактирования, при вызове метода генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
