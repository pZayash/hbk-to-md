XBase (XBase)

Ключ (Key)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КлючXBase](КлючXBase_(XBaseKey).md).   
Предоставляет доступ к объекту [КлючXBase](КлючXBase_(XBaseKey).md). Состав свойств объекта повторяет набор свойств полей объекта [XBase](XBase_(XBase).md). Значения свойств используются для вычисления выражения индекса при использовании метода [НайтиПоКлючу](XBase.НайтиПоКлючу_(XBase.FindByKey).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[XBase](XBase_(XBase).md), метод [НайтиПоКлючу](XBase.НайтиПоКлючу_(XBase.FindByKey).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
