Глобальный контекст (Global context)

КаталогБиблиотекиМобильногоУстройства (MobileDeviceLibraryDir)

Доступен, начиная с версии 8.3.9.

Синтаксис:

КаталогБиблиотекиМобильногоУстройства(<ТипКаталога>)

Параметры:

<ТипКаталога> (обязательный)

Тип: [ТипКаталогаБиблиотекиМобильногоУстройства](ТипКаталогаБиблиотекиМобильногоУстройства_(MobileDeviceLibraryDirType).md).   
Тип каталога.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает путь к библиотекам мобильного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[Глобальный контекст](Глобальный_контекст.md), метод [КаталогБиблиотекиМобильногоУстройстваАсинх](Глобальный_контекст.КаталогБиблиотекиМобильногоУстройстваАсинх_(Global_context.MobileDeviceLibraryDirAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
