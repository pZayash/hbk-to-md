СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

УстановитьКомментарийВерсииИсторииДанных (SetDataHistoryVersionComment)

Не рекомендуется использовать, начиная с версии 8.3.15.

Рекомендуется использовать:

- [КомментарийВерсии](ПараметрыЗаписиИсторииДанных.КомментарийВерсии_(WriteDataHistoryParameters.VersionComment).md)

Доступен, начиная с версии 8.3.11.

Синтаксис:

УстановитьКомментарийВерсииИсторииДанных(<Комментарий>)

Параметры:

<Комментарий> (обязательный)

Тип: [Строка](lang__def_String.md).   
Комментарий версии объекта.  
Строка, превышающая максимально допустимую длину в 1024 символа, будет обрезана.

Описание:

Устанавливает комментарий версии объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.

Не рекомендуется использовать, начиная с версии 8.3.15.

Описание изменено в версии 8.3.15.
