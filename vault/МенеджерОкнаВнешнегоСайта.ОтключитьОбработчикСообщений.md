МенеджерОкнаВнешнегоСайта (ExternalSiteWindowManager)

ОтключитьОбработчикСообщений (DetachMessageHandler)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОтключитьОбработчикСообщений(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, заданной при вызове метода [ПодключитьОбработчикСообщений](МенеджерОкнаВнешнегоСайта.ПодключитьОбработчикСообщений.md).

Описание:

Отключает ранее подключенную процедуру-обработчик.   
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Если окно внешнего сайта не доступно (свойство [Доступно](МенеджерОкнаВнешнегоСайта.Доступно.md) имеет значение Ложь), тогда вызов данного метода приведет к исключению "Окно внешнего сайта не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[СообщениеВнешнегоСайта](СообщениеВнешнегоСайта.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.
