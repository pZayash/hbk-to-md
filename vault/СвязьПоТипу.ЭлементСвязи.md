СвязьПоТипу (TypeLink)

ЭлементСвязи (LinkItem)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит номер вида субконто для случая, если реквизит, с которым выбрана связь по типу, имеет значение типа Ссылка на план счетов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
