СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

Действия (Actions)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [ПанельКнопок](СообщениеСистемыВзаимодействия.ПанельКнопок.md)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияДействийСообщенияСистемыВзаимодействия](КоллекцияДействийСообщенияСистемыВзаимодействия.md).   
Содержит список действий, которые будут выведены в виде гиперссылок при отображении сообщения:

- Текст ссылки – значение свойства [Текст](ДействиеСообщенияСистемыВзаимодействия.Текст.md) объекта типа [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия.md). Если [Текст](ДействиеСообщенияСистемыВзаимодействия.Текст.md) содержит пустую строку, то гиперссылка для этого элемента не отображается.
- В обработчик, подключенный при помощи вызова [ПодключитьОбработчикДействияСообщения](МенеджерСистемыВзаимодействия.ПодключитьОбработчикДействияСообщения.md) передается значение, заданное в свойстве [Значение](ДействиеСообщенияСистемыВзаимодействия.Значение.md) объекта типа [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
