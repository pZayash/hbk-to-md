Глобальный контекст (Global context)

УстановитьВремяЗасыпанияПассивногоСеанса (SetPassiveSessionHibernateTime)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьВремяЗасыпанияПассивногоСеанса(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: Число.   
Интервал времени в секундах, по истечении которого неактивный сеанс переводится в спящий режим.  
Значение по умолчанию: 1200.

Описание:

Устанавливает интервал времени в секундах, по истечении которого неактивный сеанс переводится в спящий режим.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
