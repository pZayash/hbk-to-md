ОбъектМетаданных: РегистрБухгалтерии (MetadataObject: AccountingRegister)

ДлинаУточненияПериода (PeriodAdjustmentLength)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: Число.   
Уточнение периода предназначено для дополнительного упорядочивания движений в регистре на оси времени: при сравнении двух движений с равными периодами хронологически более ранним будет считаться движение с меньшим значением уточнения периода.  
Тип - целое число от 0 до 3.  
Если свойство имеет значение 0, регистр бухгалтерии не поддерживает уточнение периода.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
