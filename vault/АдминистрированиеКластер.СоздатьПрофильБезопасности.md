АдминистрированиеКластер (AdministrationCluster)

СоздатьПрофильБезопасности (CreateSecurityProfile)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьПрофильБезопасности()

Возвращаемое значение:

Тип: [АдминистрированиеПрофильБезопасности](АдминистрированиеПрофильБезопасности.md).   

Описание:

Создает новый профиль безопасности кластера для его последующей настройки и регистрации. Все свойства созданного профиля имеют значения по умолчанию для соответствующих типов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
