ДокументHTML (HTMLDocument)

ПолучитьЭлементыПоИмени (GetElementByTagName)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЭлементыПоИмени(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя требующегося элемента (коллекции элементов).

Возвращаемое значение:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML.md).   

Описание:

Возвращает коллекцию элементов HTML документа с заданным именем.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для элемента HTML имена элементов не чувствительны к регистру.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.18.
