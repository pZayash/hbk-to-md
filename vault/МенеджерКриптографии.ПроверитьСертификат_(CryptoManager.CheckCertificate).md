# МенеджерКриптографии.ПроверитьСертификат

МенеджерКриптографии (CryptoManager)

ПроверитьСертификат (CheckCertificate)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьСертификат(<Сертификат>, <РежимПроверки>, <УниверсальнаяДатаПроверки>)

Параметры:

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Проверяемый сертификат.

<РежимПроверки> (необязательный)

Тип: [РежимПроверкиСертификатаКриптографии](РежимПроверкиСертификатаКриптографии_(CryptoCertificateCheckMode).md), [Массив](Массив_(Array).md).   
Указывает режим проверки.   
Содержит один объект или массив объектов режимов проверки.

<УниверсальнаяДатаПроверки> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Время проверки и построения цепочки сертификатов, указывается в универсальном времени (UTC).

Описание:

Проверяет сертификат на корректность.  
  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В случае ошибки генерируется исключительная ситуация.
