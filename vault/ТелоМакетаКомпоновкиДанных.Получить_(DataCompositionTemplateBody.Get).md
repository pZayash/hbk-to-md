ТелоМакетаКомпоновкиДанных (DataCompositionTemplateBody)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [МакетОбластиМакетаКомпоновкиДанных](МакетОбластиМакетаКомпоновкиДанных_(DataCompositionTemplateAreaTemplate).md) , [ГруппировкаМакетаКомпоновкиДанных](ГруппировкаМакетаКомпоновкиДанных_(DataCompositionTemplateGroup).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md), [ЗаписиМакетаКомпоновкиДанных](ЗаписиМакетаКомпоновкиДанных_(DataCompositionTemplateRecords).md), [ТаблицаМакетаКомпоновкиДанных](ТаблицаМакетаКомпоновкиДанных_(DataCompositionTemplateTable).md), [ДиаграммаМакетаКомпоновкиДанных](ДиаграммаМакетаКомпоновкиДанных_(DataCompositionTemplateChart).md), [ВложенныйОбъектМакетаКомпоновкиДанных](ВложенныйОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateNestedObject).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
