Описание:

Объект для создания и редактирования запросов из встроенного языка.  
Для корректной работы с объектами, полученными из схемы запроса, родительские объекты таких объектов должны существовать.   
Например, нельзя получить [ПоляСхемыЗапроса](ПоляСхемыЗапроса_(QuerySchemaFields).md) из [ОператорВыбратьСхемыЗапроса](ОператорВыбратьСхемыЗапроса_(QuerySchemaSelectOperator).md), затем удалить этот оператор из [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), и потом пробовать добавить новые выражения в [ПоляСхемыЗапроса](ПоляСхемыЗапроса_(QuerySchemaFields).md).  
Также не допускается использование объектов, полученных из одной схемы запроса, в другой схеме запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.3.5.
