# СертификатКриптографии.ВыгрузитьАсинх

СертификатКриптографии (CryptoCertificate)

ВыгрузитьАсинх (UnloadAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

ВыгрузитьАсинх()

Описание варианта метода:

Передает выгруженные данные в параметре типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Данные в файл

Синтаксис:

ВыгрузитьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выгружен сертификат.

Описание варианта метода:

Выгружает сертификат в файл.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - данные типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или имя файла, в который выгружен сертификат.

Описание:

Асинхронная функция. Выгружает сертификат.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), метод [Выгрузить](СертификатКриптографии.Выгрузить_(CryptoCertificate.Unload).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
