СтандартныйПериод (StandardPeriod)

ДатаОкончания (EndDate)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
В случае, если вариант периода - [ПроизвольныйПериод](ВариантСтандартногоПериода.ПроизвольныйПериод.md), содержит дату, которая будет использоваться в качестве конца периода.  
В случае, если вариант периода не [ПроизвольныйПериод](ВариантСтандартногоПериода.ПроизвольныйПериод.md), содержит дату, которая будет использоваться в качестве конца периода если компоновку выполнить сейчас.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
