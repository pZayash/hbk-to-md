МенеджерРаботыСРечью (SpeechProcessingManager)

УдалитьДополнительнуюГрамматикуСеанса (DeleteSessionAdditionalGrammar)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УдалитьДополнительнуюГрамматикуСеанса(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор удаляемой грамматики.

Описание:

Удаляет дополнительную грамматику распознавания речи для сеанса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
