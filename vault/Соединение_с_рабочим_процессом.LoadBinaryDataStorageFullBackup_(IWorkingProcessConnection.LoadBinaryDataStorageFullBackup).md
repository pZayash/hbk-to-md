Соединение с рабочим процессом (IWorkingProcessConnection)

LoadBinaryDataStorageFullBackup (LoadBinaryDataStorageFullBackup)

Доступен, начиная с версии 8.3.27.

Синтаксис:

LoadBinaryDataStorageFullBackup(<BinaryDataStorage>, <Path>)

Параметры:

<BinaryDataStorage> (обязательный)

Тип: [Хранилище двоичных данных](Хранилище_двоичных_данных_(IBinaryDataStorageInfo).md).   
Хранилище двоичных данных.

<Path> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу полной резервной копии на сервере.

Описание:

Восстанавливает хранилище двоичных данных из полной резервной копии. Только для встроенного хранилища.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
