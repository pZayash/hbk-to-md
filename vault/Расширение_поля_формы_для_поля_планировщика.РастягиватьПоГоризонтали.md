Расширение поля формы для поля планировщика (Form field extension for a planner)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает возможность растягивания поля по горизонтали.   
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
