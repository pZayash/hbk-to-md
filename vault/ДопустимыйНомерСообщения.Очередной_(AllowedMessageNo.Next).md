ДопустимыйНомерСообщения (AllowedMessageNo)

Очередной (Next)

Доступен, начиная с версии 8.0.

Описание:

Данное значение показывает, что номер принимаемого сообщения должен быть на 1 больше, чем максимальный номер принятого сообщения. Максимальный номер принятого сообщения можно узнать, обратившись к реквизиту [НомерПринятого](ПланОбменаОбъект.Имя_плана_обмена.НомерПринятого_(ExchangePlanObject.Exchange_plan_name.ReceivedNo).md) узла плана обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
