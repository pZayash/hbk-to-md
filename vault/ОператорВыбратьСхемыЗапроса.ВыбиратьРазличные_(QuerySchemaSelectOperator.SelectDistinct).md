ОператорВыбратьСхемыЗапроса (QuerySchemaSelectOperator)

ВыбиратьРазличные (SelectDistinct)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость выбора различных значений записей.  
Соответствует выражению РАЗЛИЧНЫЕ языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
