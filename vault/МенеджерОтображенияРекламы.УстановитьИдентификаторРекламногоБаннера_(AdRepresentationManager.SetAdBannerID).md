МенеджерОтображенияРекламы (AdRepresentationManager)

УстановитьИдентификаторРекламногоБаннера (SetAdBannerID)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УстановитьИдентификаторРекламногоБаннера(<ИдентификаторРекламногоБлока>)

Параметры:

<ИдентификаторРекламногоБлока> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор рекламного блока для рекламы Google.   
Параметр является обязательным, если реклама используется в приложении на Android. В остальных случаях параметр игнорируется.

Описание:

Устанавливает идентификатор рекламного блока, который будет использован для рекламного баннера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
