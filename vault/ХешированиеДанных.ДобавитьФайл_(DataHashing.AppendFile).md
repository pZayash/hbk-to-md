ХешированиеДанных (DataHashing)

ДобавитьФайл (AppendFile)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ДобавитьФайл(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла с двоичными данными.

Описание:

Обновляет хеш-сумму, с учетом добавления двоичных данных из указанного файла.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
