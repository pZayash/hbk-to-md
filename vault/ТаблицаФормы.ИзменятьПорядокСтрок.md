ТаблицаФормы (FormTable)

ИзменятьПорядокСтрок (ChangeRowOrder)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет изменением порядка строк.   
Истина - менять разрешено.  
Данное свойство имеет смысл только для тех данных, редактирование которых не запрещено прикладным смыслом и ограничениями по правам. А также для тех данных, тип которых подразумевает упорядоченность списка (табличные части, [ТаблицаЗначений](ТаблицаЗначений.md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
