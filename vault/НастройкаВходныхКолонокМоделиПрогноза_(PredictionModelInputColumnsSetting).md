# НастройкаВходныхКолонокМоделиПрогноза

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[НастройкаВходнойКолонкиМоделиПрогноза](НастройкаВходнойКолонкиМоделиПрогноза_(PredictionModelInputColumnSetting).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются настройки входных колонок.  
Возможно обращение к настройке посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Свойства:

[\<Имя колонки прогноза\> (\<Forecast column name\>)](НастройкаВходныхКолонокМоделиПрогноза.Имя_колонки_прогноза_(PredictionModelInputColumnsSetting.Forecast_column_name).md)

Описание:

Содержит соответствия колонок источника данных колонкам модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.
