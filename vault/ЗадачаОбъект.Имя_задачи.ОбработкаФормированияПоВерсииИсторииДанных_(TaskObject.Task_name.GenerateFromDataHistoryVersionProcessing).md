ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ОбработкаФормированияПоВерсииИсторииДанных (GenerateFromDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОбработкаФормированияПоВерсииИсторииДанных(<ДанныеВерсии>, <НомерВерсии>, <СтандартнаяОбработка>, <ИсключитьДанные>)

Параметры:

<ДанныеВерсии>

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md).   
Фиксированная структура с данными версии, по которой выполняется формирование.

<НомерВерсии>

Тип: [Число](lang__def_Number.md).   
Номер версии, по которой выполняется формирование.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ИсключитьДанные>

Тип: [Массив](Массив_(Array).md).   
Массив с именами полей и табличных частей, которые будут исключены из стандартной обработки.

Описание:

Возникает при формировании объекта по версии истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерИсторииДанных](МенеджерИсторииДанных_(DataHistoryManager).md), метод [СформироватьПоВерсии](МенеджерИсторииДанных.СформироватьПоВерсии_(DataHistoryManager.GenerateFromVersion).md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
