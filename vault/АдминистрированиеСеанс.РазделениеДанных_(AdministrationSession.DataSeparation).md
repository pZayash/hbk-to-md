РазделениеДанных (DataSeparation)

Доступен, начиная с версии 8.3.15.

Только чтение.

<значение разделителя1>,...,<значение разделителяN>.

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
