Описание:

Объект не может быть создан самостоятельно. Объект данного типа представляет собой значение свойства ОбменДанными объектов, представляющих данные, по которым ведется учет изменений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект.ОбменДанными.Получатели.Заполнить(); |

См. также:

[УдалениеОбъекта](УдалениеОбъекта_(ObjectDeletion).md), свойство [ОбменДанными](УдалениеОбъекта.ОбменДанными_(ObjectDeletion.DataExchange).md)

Доступен, начиная с версии 8.0.
