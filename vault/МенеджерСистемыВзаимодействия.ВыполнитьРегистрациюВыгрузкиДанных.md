МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВыполнитьРегистрациюВыгрузкиДанных (RegisterDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ВыполнитьРегистрациюВыгрузкиДанных(<ПутьКФайлу>, <Описание>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: Строка.   
Путь к файлу с выгрузкой данных на сервере системы взаимодействия.

<Описание> (обязательный)

Тип: Строка.   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.md).   

Описание:

Регистрирует выгрузку данных, если администратор самостоятельно записал ее на сервер взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ВыполнитьРегистрациюВыгрузкиДанныхАсинх](МенеджерСистемыВзаимодействия.ВыполнитьРегистрациюВыгрузкиДанныхАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
