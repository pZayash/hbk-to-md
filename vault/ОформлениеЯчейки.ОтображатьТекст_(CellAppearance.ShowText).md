ОформлениеЯчейки (CellAppearance)

ОтображатьТекст (ShowText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения текста.  
[Истина](lang__def_BooleanTrue.md) - отображается.

Доступность:

Толстый клиент.

Примечание:

Текст ячейки можно не отображать, если с ней связано значение тип [Булево](lang__Булево_(Boolean).md) или [Число](lang__def_Number.md) и для отображения этого значения используется флажок.

Использование в версии:

Доступен, начиная с версии 8.0.
