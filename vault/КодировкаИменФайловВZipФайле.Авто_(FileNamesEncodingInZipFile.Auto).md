КодировкаИменФайловВZipФайле (FileNamesEncodingInZipFile)

Авто (Auto)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Авто](КодировкаИменФайловВФайлеАрхива.Авто_(FileNamesEncodingInArchiveFile.Auto).md)

Доступен, начиная с версии 8.3.9.

Описание:

Значение кодировки вычисляется из настроек конфигурационного файла. Если настройки в конфигурационном файле отсутствуют, используется значение UTF8.

Использование в версии:

Доступен, начиная с версии 8.3.9.

Не рекомендуется использовать, начиная с версии 8.3.26.
