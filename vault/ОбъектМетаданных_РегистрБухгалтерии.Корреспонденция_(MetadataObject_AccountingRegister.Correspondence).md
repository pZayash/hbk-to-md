ОбъектМетаданных: РегистрБухгалтерии (MetadataObject: AccountingRegister)

Корреспонденция (Correspondence)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), регистр бухгалтерии поддерживает корреспонденцию счетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
