Расширение формы клиентского приложения для записи регистра сведений (Client application form extension for information register records)

ЗначенияЗаполнения (FillingValues)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Параметры заполнения новой записи.  
Ключом структуры является "Период" или название измерения, свойства или ресурса.

Использование в версии:

Доступен, начиная с версии 8.2.
