Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) при отображении в нем таблицы значений. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) имеют значения типа [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений_(ValueTableRow).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
