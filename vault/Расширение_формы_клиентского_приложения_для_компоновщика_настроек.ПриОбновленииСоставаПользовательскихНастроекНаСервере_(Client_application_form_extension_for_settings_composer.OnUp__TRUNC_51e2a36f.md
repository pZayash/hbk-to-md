Расширение формы клиентского приложения для компоновщика настроек (Client application form extension for settings composer)

ПриОбновленииСоставаПользовательскихНастроекНаСервере (OnUpdateUserSettingSetAtServer)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПриОбновленииСоставаПользовательскихНастроекНаСервере(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в обработчике установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), то стандартная обработка данного события не выполняется.  
В стандартной обработке события расширение формы компоновщика настроек при помощи метода [СоздатьЭлементыФормыПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_компоновщика_настроек.СоздатьЭлементыФормыПользовательскихНастроек_(Client_application_form_extension_for_settings_composer.CreateUserSettingsFormItems).md) создает в указанной группе элементы формы для редактирования пользовательских настроек.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при обновлении состава пользовательских настроек:

- При получении формы;
- После изменения настроек, если изменился состав настроек.

.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
