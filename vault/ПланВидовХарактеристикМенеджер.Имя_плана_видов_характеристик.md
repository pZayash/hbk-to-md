Описание:

Предназначен для управления планом видов характеристик, как объектом конфигурации. Доступ к объекту осуществляется через свойства объекта [ПланыВидовХарактеристикМенеджер](ПланыВидовХарактеристикМенеджер.md).   
Предназначен для создания новых элементов плана видов характеристик, а также поиска элементов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.
