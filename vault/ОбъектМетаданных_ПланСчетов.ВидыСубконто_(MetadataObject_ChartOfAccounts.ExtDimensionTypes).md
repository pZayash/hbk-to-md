ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ВидыСубконто (ExtDimensionTypes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: ПланВидовХарактеристик](ОбъектМетаданных_ПланВидовХарактеристик_(MetadataObject_ChartOfCharacteristicTypes).md).   
Объект метаданных, описывающий план видов характеристик. В этом плане видов характеристик содержатся виды субконто, используемые в данном плане счетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
