# Расширение колонки табличного поля дерева значений.ДанныеКартинки

Расширение колонки табличного поля дерева значений (Value tree table box column extension)

ДанныеКартинки (PictureData)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя колонки дерева значений, из которой следует брать картинку для отображения в колонке табличного поля. Если в колонке таблицы значений хранится картинка, будет отображена она. Если в колонке хранится число, оно будет интерпретировано как индекс картинки в наборе картинок, указанном в свойстве [КартинкиСтрок](КолонкаТабличногоПоля.КартинкиСтрок_(TableBoxColumn.RowsPictures).md) колонки табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
