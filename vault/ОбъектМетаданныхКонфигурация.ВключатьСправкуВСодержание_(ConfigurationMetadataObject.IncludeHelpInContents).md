# ОбъектМетаданныхКонфигурация.ВключатьСправкуВСодержание

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [ОбъектМетаданныхКонфигурация](ОбъектМетаданныхКонфигурация_(ConfigurationMetadataObject).md) › [Свойства](ОбъектМетаданныхКонфигурация__Свойства.md)

ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ВключатьСправкуВСодержание (IncludeHelpInContents)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то cправочная информация, относящаяся к объекту метаданных конфигурация (свойство [Справка](ОбъектМетаданныхКонфигурация.Справка_(ConfigurationMetadataObject.Help).md)), включается в общее описание конфигурации.  
В противном случае справочная информация не включается в общее описание конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
