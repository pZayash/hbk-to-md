КонструкторНастроекКомпоновкиДанных (DataCompositionSettingsWizard)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - в конструкторе была нажата клавиша OK. [Ложь](lang__def_BooleanFalse.md) - была нажата клавиша Отмена.

Описание:

Открывает конструктор настроек компоновки данных в модальном режиме.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.1.
