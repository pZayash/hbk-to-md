КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных (DataCompositionChartStructureItemCollection)

ИдентификаторПользовательскойНастройки (UserSettingID)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Предназначен для идентификации пользовательской настройки.   
Если идентификатор не пустой, объект включается в состав пользовательских настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
