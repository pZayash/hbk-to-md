ПланыОбменаМенеджер (ExchangePlansManager)

СоздатьНачальныйОбраз (CreateInitialImage)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьНачальныйОбраз(<Узел>, <СтрокаСоединения>)

Параметры:

<Узел> (обязательный)

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [ПланОбменаМенеджер.](ПланОбменаМенеджер.Имя_плана_обмена_(ExchangePlanManager.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаМенеджер.Имя_плана_обмена_(ExchangePlanManager.Exchange_plan_name).md)>.   
Ссылка на узел плана обмена или объект, представляющий узел плана обмена. Данный узел должен относиться к плану обмена распределенной информационной базы и не должен быть главным узлом для данного узла распределенной информационной базы.

<СтрокаСоединения> (обязательный)

Тип: [Строка](lang__def_String.md).   
[Строка соединения](lang__Строка_соединения.md), указывающая информационную базу, в которой будет создан начальный образ подчиненного узла распределенной ИБ. Информационная база для создания начального образа должна быть пустой или не должна существовать вовсе.

Описание:

Создает начальный образ подчиненного узла распределенной информационной базы. Начальный образ выгружается в информационную базу, которая создается в процессе создания начального образа или должна быть пустой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При создании начального образа удаляются все записи регистрации элементов данных и конфигурации для данного узла.

Пример:

|  |
| --- |
| ПланыОбмена.СоздатьНачальныйОбраз(Узел,     "File=c:/1cv8/MDDistr/New;Locale=ru\_RU"); |

Использование в версии:

Доступен, начиная с версии 8.0.
