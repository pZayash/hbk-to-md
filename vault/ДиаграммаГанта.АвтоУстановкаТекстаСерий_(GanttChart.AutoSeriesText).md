ДиаграммаГанта (GanttChart)

АвтоУстановкаТекстаСерий (AutoSeriesText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак генерации текста для новых серий по схеме "СерияN". Если [Ложь](lang__def_BooleanFalse.md), то у новой серии будет пустой текст.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
