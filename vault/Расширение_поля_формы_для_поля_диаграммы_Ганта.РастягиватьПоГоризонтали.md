Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает возможность растягивания поля диаграммы Ганта по горизонтали.   
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
