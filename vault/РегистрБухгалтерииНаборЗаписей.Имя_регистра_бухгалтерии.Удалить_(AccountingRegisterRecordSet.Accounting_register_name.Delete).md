Удалить (Delete)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция записи в наборе.

Вариант синтаксиса: По объекту записи

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [РегистрБухгалтерииЗапись.\<Имя регистра бухгалтерии\>](РегистрБухгалтерииЗапись.Имя_регистра_бухгалтерии_(AccountingRegisterRecord.Accounting_register_name).md).   
Удаляемая запись.

Описание:

Удаляет запись из набора записей регистра бухгалтерии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
