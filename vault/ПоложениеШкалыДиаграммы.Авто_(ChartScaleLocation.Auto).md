ПоложениеШкалыДиаграммы (ChartScaleLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Положение выбирается автоматически:

- для шкалы значений - [Край](ПоложениеШкалыДиаграммы.Край_(ChartScaleLocation.Edge).md),
- для шкалы точек в точечной и пузырьковой диаграммах - [Край](ПоложениеШкалыДиаграммы.Край_(ChartScaleLocation.Edge).md),
- во всех остальных случаях - [БазовоеЗначение](ПоложениеШкалыДиаграммы.БазовоеЗначение_(ChartScaleLocation.BaseValue).md).

Использование в версии:

Доступен, начиная с версии 8.3.11.
