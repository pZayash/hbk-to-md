Доступен, начиная с версии 8.0.

Описание:

Тип анализа данных - дерево решений.  
Параметры:

- МинимальноеКоличествоСлучаев (MinCaseCount) - минимальное количество случаев в узле (Число);
- МаксимальнаяГлубина (MaxDepth) - максимальная глубина дерева (Число);
- ТипУпрощения (SimplificationType) - тип упрощения дерева решений (ТипУпрощенияДереваРешений).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
