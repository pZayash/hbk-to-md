# ФайловыйПоток.ПолучитьПотокТолькоДляЧтения

ФайловыйПоток (FileStream)

ПолучитьПотокТолькоДляЧтения (GetReadonlyStream)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьПотокТолькоДляЧтения()

Возвращаемое значение:

Тип: [Поток](Поток_(Stream).md).   

Описание:

Возвращает поток, который разделяет данные и текущую позицию с данным потоком, но не разрешает запись.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
