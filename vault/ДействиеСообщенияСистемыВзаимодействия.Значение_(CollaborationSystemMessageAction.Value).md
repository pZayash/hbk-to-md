ДействиеСообщенияСистемыВзаимодействия (CollaborationSystemMessageAction)

Значение (Value)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Данные](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Данные_(CollaborationSystemMessageButtonPanelButton.Data).md)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение действия. Используется для поиска с помощью метода [Найти](КоллекцияДействийСообщенияСистемыВзаимодействия.Найти_(CollaborationSystemMessageActionCollection.Find).md) коллекции действий сообщения системы взаимодействия, а также для передачи в событие [ПриВыбореДействияСообщенияСистемыВзаимодействия](Глобальный_контекст.ПриВыбореДействияСообщенияСистемыВзаимодействия_(Global_context.OnCollaborationSystemMessageActionChoice).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
