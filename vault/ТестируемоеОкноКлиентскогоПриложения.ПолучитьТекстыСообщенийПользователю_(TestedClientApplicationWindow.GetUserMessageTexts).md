ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

ПолучитьТекстыСообщенийПользователю (GetUserMessageTexts)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьТекстыСообщенийПользователю()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   

Описание:

Получает фиксированный массив с текстами сообщений пользователю, отображаемых в данный момент в окне. Если окно сообщений закрыто, генерируется ошибка "Метод не может быть вызван для данного объекта".

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
