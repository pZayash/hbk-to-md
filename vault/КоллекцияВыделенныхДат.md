Доступен, начиная с версии 8.0.

Элементы коллекции:

Дата  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Добавить (Add)](КоллекцияВыделенныхДат.Добавить.md)  
[Количество (Count)](КоллекцияВыделенныхДат.Количество.md)  
[Очистить (Clear)](КоллекцияВыделенныхДат.Очистить.md)  
[Получить (Get)](КоллекцияВыделенныхДат.Получить.md)  
[Содержит (Contains)](КоллекцияВыделенныхДат.Содержит.md)  
[Удалить (Delete)](КоллекцияВыделенныхДат.Удалить.md)  

Описание:

Представляет собой коллекцию выделенных в календаре дат.

Доступность:

Толстый клиент.

[ПолеКалендаря](ПолеКалендаря.md), свойство [ВыделенныеДаты](ПолеКалендаря.ВыделенныеДаты.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
