ОбъектМетаданных: Документ (MetadataObject: Document)

ИспользоватьСтандартныеКоманды (UseStandardCommands)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено в значение [Ложь](lang__def_BooleanFalse.md), принадлежащие данному объекту метаданных команды не будут включаться в соответствующие фрагменты командного интерфейса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
