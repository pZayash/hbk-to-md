Рабочий сервер (IWorkingServerInfo)

GetPortRanges (GetPortRanges)

Доступен, начиная с версии 8.1.

Синтаксис:

GetPortRanges()

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Метод возвращает [COMSafeArray](COMSafeArray.md) из объектов [Диапазон портов](Диапазон_портов.md).

Описание:

Получает выделенные диапазоны IP-портов для динамического занятия.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
