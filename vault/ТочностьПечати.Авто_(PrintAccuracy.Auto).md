ТочностьПечати (PrintAccuracy)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

При печати возможны изменения полей и масштаба.  
При печати из веб-клиента способом "PDF" печатаемый табличный документ открывается в отдельном окне.

Использование в версии:

Доступен, начиная с версии 8.2.
