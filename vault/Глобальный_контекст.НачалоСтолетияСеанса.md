Глобальный контекст (Global context)

НачалоСтолетияСеанса (SessionBeginningOfCentury)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачалоСтолетияСеанса()

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает год начала столетия, действующий в текущем сеансе.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
