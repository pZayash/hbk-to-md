# ОтображениеПодсказкиЗначенийДиаграммы.ОтображатьДляБлижайшего

ОтображениеПодсказкиЗначенийДиаграммы (ChartValuesToolTipShowMode)

ОтображатьДляБлижайшего (ShowForNearestValue)

Доступен, начиная с версии 8.3.14.

Описание:

Всплывающая подсказка отображается для ближайшего значения. Подсказка располагается около значения. При перемещении курсора, подсказка перемещается с анимацией от предыдущего значения к следующему.

Использование в версии:

Доступен, начиная с версии 8.3.14.
