Описание:

Элемент графической схемы "Слияние". Отображает точку маршрута бизнес-процесса, в которую сходятся параллельные пути исполнения, начавшиеся в точке "Разделение". Пока исполнение задач по всем параллельным путям не придет в точку слияния, переход к следующей за слиянием точке не будет выполнен. Одной точке слияния всегда соответствует одна точка разделения (но не наоборот, т.к. могут быть точки разделения без точек слияния).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
