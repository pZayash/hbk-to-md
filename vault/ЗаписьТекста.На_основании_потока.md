ЗаписьТекста (TextWriter)

На основании потока

Доступен, начиная с версии 8.3.10.

Синтаксис:

Новый ЗаписьТекста(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка записываемого текста (согласована с текстовым документом).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

- ISO-2022-KR

- ISO-2022-CN-EXT
- HZ-GB-2312

  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8.md).

<РазделительСтрок> (необязательный)

Тип: Строка.   
Определяет строку, разделяющую строки в потоке.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Определяет разделение строк в потоке для конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<ЗаписатьBOM> (необязательный)

Тип: Булево.   
Если в начало потока требуется записать метку порядка байтов (BOM) для используемой кодировки текста, то данный параметр должен иметь значение Истина.  
Значение по умолчанию: Ложь.

Описание:

Устанавливает поток для записи. Аналогичен методу [Открыть](ЗаписьТекста.Открыть.md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
