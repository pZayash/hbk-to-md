ЗаписьТекста (TextWriter)

На основании потока

Доступен, начиная с версии 8.3.10.

Синтаксис:

Новый ЗаписьТекста(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка записываемого текста (согласована с текстовым документом).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

Значение по умолчанию: [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет строку, разделяющую строки в потоке.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет разделение строк в потоке для конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в начало потока требуется записать метку порядка байтов (BOM) для используемой кодировки текста, то данный параметр должен иметь значение [Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Устанавливает поток для записи. Аналогичен методу [Открыть](ЗаписьТекста.Открыть_(TextWriter.Open).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
