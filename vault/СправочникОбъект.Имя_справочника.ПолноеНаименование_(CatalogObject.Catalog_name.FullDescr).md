# СправочникОбъект.<Имя справочника>.ПолноеНаименование

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Методы](СправочникОбъект.Имя_справочника__Методы.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПолноеНаименование (FullDescr)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолноеНаименование()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Строка, включающая наименование элемента и наименования всех вышестоящих элементов.  
Наименования выводятся слева направо, начиная с самого верхнего уровня, разделяются символом "/".

Описание:

Получает наименование элемента справочника с учетом наименований всех вышестоящих элементов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Сообщить("Полное  наименование данного элемента: " +           Объект.ПолноеНаименование()); |

Использование в версии:

Доступен, начиная с версии 8.0.
