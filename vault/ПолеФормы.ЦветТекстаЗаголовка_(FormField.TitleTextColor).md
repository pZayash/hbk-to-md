ПолеФормы (FormField)

ЦветТекстаЗаголовка (TitleTextColor)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет текста заголовка. При размещении в таблице определяет цвет текста шапки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
