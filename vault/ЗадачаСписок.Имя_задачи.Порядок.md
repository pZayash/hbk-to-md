Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Универсальные_объекты.Порядок.md).   
Содержит объект [Порядок](Универсальные_объекты.Порядок.md), по которому осуществляется текущее упорядочивание задач в списке. В качестве имен упорядочивания объекта могут выступать имена предопределенных реквизитов Код и Наименование.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
