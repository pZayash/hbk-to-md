ИнтернетТекстПочтовогоСообщения (InternetMailText)

ПроизвольныйТипТекста (CustomTextType)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Текстовые данные сообщения.  
Если при отправке тип текста у [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения.md) установлен в [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст.md), то содержимое поля Content-Type для отправляемого письма будет получено из данного свойства.  
Если в получаемом сообщении не удалось определить тип текста, то свойство [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста.md) принимает значение [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст.md). Но в любом случае, данное свойство при чтении, будет содержать актуальный Content-Type полученного сообщения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
