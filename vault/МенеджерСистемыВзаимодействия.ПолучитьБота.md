МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьБота (GetBot)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьБота(<Пользователь>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Идентификатор пользователя системы взаимодействия.

Возвращаемое значение:

Тип: [БотСистемыВзаимодействия](БотСистемыВзаимодействия.md), Неопределено.   
Неопределено - если переданный пользователь не является ботом.

Описание:

Возвращает описание бота, назначенного переданному пользователю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
