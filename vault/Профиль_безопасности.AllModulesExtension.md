Профиль безопасности (ISecurityProfile)

AllModulesExtension (AllModulesExtension)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет, разрешено ли изменение всех модулей в расширении конфигурации:  
Истина - разрешено расширять любой модуль,  
Ложь - разрешено расширять модули из списка разрешенных (указаны в свойстве [ModulesAvailableForExtension](Профиль_безопасности.ModulesAvailableForExtension.md)).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
