XBase (XBase)

ТекущийИндекс (CurrentIndex)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИндексXBase](ИндексXBase.md).   
Содержит текущий используемый индекс. Установка текущего индекса работы с таблицей базы данных оказывает влияние на работу методов поиска и позиционирования: [Найти](XBase.Найти.md), [НайтиПоКлючу](XBase.НайтиПоКлючу.md), [Первая](XBase.Первая.md), [Последняя](XBase.Последняя.md), [Следующая](XBase.Следующая.md), [Предыдущая](XBase.Предыдущая.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
