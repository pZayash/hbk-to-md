Цвет (Color)

Конструктор по умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Цвет()

Описание:

Создает значение цвета, которое будет использоваться как значение по умолчанию (цвет "Авто").

Пример:

|  |
| --- |
| Процедура СправочникСписокПриПолученииДанных(Элемент, ОформленияСтрок)     Если Элемент.ТекущаяСтрока=Неопределено Тогда         Возврат;     КонецЕсли;          Для Каждого Офр Из ОформленияСтрок Цикл         Если Офр.ДанныеСтроки.Важно Тогда             Офр.ЦветФона = Новый Цвет(197, 255, 185);         Иначе             Офр.ЦветФона = Новый Цвет();         КонецЕсли;     КонецЦикла; КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
