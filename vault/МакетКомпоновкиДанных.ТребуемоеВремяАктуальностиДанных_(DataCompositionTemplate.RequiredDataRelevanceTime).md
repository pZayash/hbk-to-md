МакетКомпоновкиДанных (DataCompositionTemplate)

ТребуемоеВремяАктуальностиДанных (RequiredDataRelevanceTime)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Дата](lang__def_Date.md).   
Минимальное допустимое время актуальности данных, которые необходимо получить.  
Свойство имеет смысл только если [ТребуемаяАктуальностьДанных](МакетКомпоновкиДанных.ТребуемаяАктуальностьДанных_(DataCompositionTemplate.RequiredDataRelevance).md) равна [Авто](ТребуемаяАктуальностьДанных.Авто_(RequiredDataRelevance.Auto).md) или [Любые](ТребуемаяАктуальностьДанных.Любые_(RequiredDataRelevance.Any).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.16.
