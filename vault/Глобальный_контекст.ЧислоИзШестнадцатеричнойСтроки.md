`ЧислоИзШестнадцатеричнойСтроки`(`<ШестнадцатеричнаяСтрока>`: `Строка`) → `Число`
<!-- signature:end -->

Глобальный контекст (Global context)

ЧислоИзШестнадцатеричнойСтроки (NumberFromHexString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЧислоИзШестнадцатеричнойСтроки(<ШестнадцатеричнаяСтрока>)

Параметры:

<ШестнадцатеричнаяСтрока> (обязательный)

Тип: Строка.   
Шестнадцатеричная строка.

Возвращаемое значение:

Тип: Число.   

Описание:

Преобразует шестнадцатеричную строку в значение типа Число.  
Строка должна начинаться с префикса "0x", после которого следуют одна или несколько шестнадцатеричных цифр. Другие символы в строке не допускаются.  
Поддерживаются только натуральные числа.  
Отрицательные и числа с дробной частью не поддерживаются.  
Примеры допустимых строк: "0x003A", "0xabcde".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Значение1 = ЧислоИзШестнадцатеричнойСтроки("0xabcde"); Значение2 = ЧислоИзШестнадцатеричнойСтроки("0х003A"); |

Использование в версии:

Доступен, начиная с версии 8.3.10.
