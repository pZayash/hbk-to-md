МенеджерКриптографии (CryptoManager)

НачатьПолучениеИнформацииМодуляКриптографии (BeginGettingCryptoModuleInformation)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеИнформацииМодуляКриптографии(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения получения информации со следующими параметрами:

- <ИнформацияМодуля> - [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии.md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает получение информации о модуле криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
