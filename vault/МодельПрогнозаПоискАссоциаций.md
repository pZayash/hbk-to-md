Описание:

Модель для прогноза ассоциаций.  
Модель содержит следующие колонки:

- Следствие (Consequent) - элемент-следствие;
- Достоверность (Confidence) - достоверность возникновения элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.
