Описание:

Предназначен для получения и записи сведений о политике паролей пользователей информационной базы.   
Изменение политик доступно только пользователям с административными правами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.26.
