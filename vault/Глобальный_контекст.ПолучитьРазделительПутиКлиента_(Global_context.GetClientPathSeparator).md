Глобальный контекст (Global context)

ПолучитьРазделительПутиКлиента (GetClientPathSeparator)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьРазделительПутиКлиента()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает значение разделителя пути, используемого в операционной системе, в которой работает клиентское приложение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
