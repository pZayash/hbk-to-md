Скопировать (Copy)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаОбъект.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаОбъект.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   

Описание:

Создает новый объект таблицы внешнего источника данных копированием существующего.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных внешнего источника данных.

Использование в версии:

Доступен, начиная с версии 8.3.5.
