# СписокЗначений.Вставить

СписокЗначений (ValueList)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Представление>, <Пометка>, <Картинка>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс позиции, на которую надо вставить элемент.

<Значение> (необязательный)

Тип: Произвольный.   
Значение, которое необходимо вставить.

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Строковое выражение, содержащее представление добавляемого значения.  
Значение по умолчанию: Пустая строка.

<Пометка> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, создавать новое значение с установленной пометкой или нет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Картинка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Визуальное представление значения.  
Значение по умолчанию: Пустая картинка.

Возвращаемое значение:

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).   
Новый элемент списка значений.

Описание:

Вставляет новый элемент в список значений на позицию с указанным индексом.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СписокТиповЦен.Добавить("Все наценки",, Истина); ВыборкаТиповЦен = Справочники.ТипыЦен.Выбрать(); Пока ВыборкаТиповЦен.Следующий() Цикл     СписокТиповЦен.Вставить(0, ВыборкаТиповЦен.Ссылка,                              ВыборкаТиповЦен.Наценка, Ложь); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
