# МенеджерВстроенныхПокупок.ПолучитьСписок

МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьСписок (GetList)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПолучитьСписок(<ИдентификаторыПокупок>, <ДоступныеДляПокупки>)

Параметры:

<ИдентификаторыПокупок> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив идентификаторов покупок, информация о которых получается.

<ДоступныеДляПокупки> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), в списке покупок отображаются только покупки, коорые можно приобрести в данный момент. Приобретенные постоянные покупки и активные подписки из списка исключаются.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ВстроеннаяПокупка](ВстроеннаяПокупка_(InAppPurchase).md).

Описание:

Возвращает список покупок, доступных в приложении. Для работы данного метода наличие соединения с сетью Интернет не требуется. Данные получаются из истории покупок, хранящейся локально на устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
