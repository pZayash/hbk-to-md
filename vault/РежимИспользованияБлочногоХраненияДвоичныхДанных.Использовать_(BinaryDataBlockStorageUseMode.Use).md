РежимИспользованияБлочногоХраненияДвоичныхДанных (BinaryDataBlockStorageUseMode)

Использовать (Use)

Доступен, начиная с версии 8.3.26.

Описание:

Использовать режим блочного хранения данных. Разрешает использовать режимы размещения копий данных в хранилище двоичных данных.  
При установке данного режима совместно с [РежимХранилищаДвоичныхДанных](ОбъектМетаданныхКонфигурация.РежимХранилищаДвоичныхДанных_(ConfigurationMetadataObject.BinaryDataStorageMode).md) в значении [Использовать](РежимХранилищаДвоичныхДанных.Использовать_(BinaryDataStorageMode.Use).md), будет создана служебная таблица BinaryData для хранения двоичных данных.   
В режиме Использовать или при отключении режима совместимости двоичные данные будут записаны в эту таблицу, если их размер превышает 2 Кб, и они относятся к таблицам объектов метаданных и не относятся к системным таблицам.   
При включении данного режима двоичные данные не перемещаются из таблиц объектов метаданных в таблицу BinaryData.  
Содержимое каждого поля записи двоичных данных в таблице не превышает 960 Мб. Если записываемые двоичные данные превышают 960 Мб, то они разбиваются на несколько записей-блоков, не более 960 Мб каждый. Таблица не входит в состав разделителей, и при записи одинаковых двоичных данных из разных областей будут использованы различные уникальные идентификаторы (GUID).

Использование в версии:

Доступен, начиная с версии 8.3.26.
