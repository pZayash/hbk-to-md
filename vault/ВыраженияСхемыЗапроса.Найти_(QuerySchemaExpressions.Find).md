ВыраженияСхемыЗапроса (QuerySchemaExpressions)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение для поиска.

Возвращаемое значение:

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md).   

Описание:

Выполняет поиск по выражению. При этом форматирование игнорируется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
