Соединение с агентом сервера (IServerAgentConnection)

ApplyServiceSettings (ApplyServiceSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ApplyServiceSettings(<Кластер>, <Сервер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого требуется применить настройки сервисов.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   
Сервер, для которого требуется применить настройки сервисов.

Описание:

Применяет настройки сервисов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
