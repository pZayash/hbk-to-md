ТаблицаФормы (FormTable)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Указывает необходимость пропуска поля при вводе данных.  
Если [Неопределено](lang__def_Undefined.md), то [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
