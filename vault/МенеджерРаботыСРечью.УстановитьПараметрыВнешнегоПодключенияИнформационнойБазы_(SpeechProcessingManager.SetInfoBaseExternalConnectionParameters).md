МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьПараметрыВнешнегоПодключенияИнформационнойБазы (SetInfoBaseExternalConnectionParameters)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПараметрыВнешнегоПодключенияИнформационнойБазы(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: [ПараметрыВнешнегоПодключенияРаботыСРечью](ПараметрыВнешнегоПодключенияРаботыСРечью_(SpeechProcessingExternalConnectionParameters).md).   
Параметры внешнего распознавания речи, которые будут использованы в информационной базе.  
[Неопределено](lang__def_Undefined.md), если требуется отключить информационную базу или область от сервиса внешнего распознавания речи.  
Поля [АдресСервера](ПараметрыВнешнегоПодключенияРаботыСРечью.АдресСервера_(SpeechProcessingExternalConnectionParameters.ServerAddress).md) и [Токен](ПараметрыВнешнегоПодключенияРаботыСРечью.Токен_(SpeechProcessingExternalConnectionParameters.Token).md) должны быть заполнены обязательно.

Описание:

Сохраняет параметры доступа к облачному сервису работы с речью в информационной базе (с учетом используемых разделителей сеанса).  
Требуется наличие права "Администрирование данных".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
