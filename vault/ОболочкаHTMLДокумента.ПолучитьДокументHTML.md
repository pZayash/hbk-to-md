Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьДокументHTML()

Возвращаемое значение:

Тип: [ДокументHTML](ДокументHTML.md).   

Описание:

Получает исходный HTML-документ в виде дерева DOM и объекта [ДокументHTML](ДокументHTML.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
