ШкалаДиаграммы (ChartScale)

ЛинияСетки (GridLine)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: [Линия](Линия_(Line).md), [Неопределено](lang__def_Undefined.md).   
Тип линии для отображения линий сетки.  
Если [Неопределено](lang__def_Undefined.md), тип линии выбирается из свойства [ЛинииШкал](ОбластьПостроенияДиаграммы.ЛинииШкал_(ChartPlotArea.ScaleLines).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
