Глобальный контекст (Global context)

УстановитьИспользуемыйСервер (SetUsedServer)

Доступен, начиная с версии 8.3.16.

Синтаксис:

УстановитьИспользуемыйСервер(<ИспользуемыйСервер>)

Параметры:

<ИспользуемыйСервер> (обязательный)

Тип: [ИспользуемыйСервер](ИспользуемыйСервер_(UsedServer).md), [Неопределено](lang__def_Undefined.md).   
Используемый сервер.

Описание:

Указывает, какой из серверов должен использоваться мобильным клиентом без явного указания сервера. Установка действует до выхода из области видимости локальных переменных.

Доступность:

Мобильный клиент.

Примечание:

Если параметр <ИспользуемыйСервер> имеет значение [Неопределено](lang__def_Undefined.md), возвращает предыдущее состояние.

См. также:

[Глобальный контекст](Глобальный_контекст.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
