КолонкаТабличногоПоля (TableBoxColumn)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак выделения текста отрицательного числа специальным цветом [ЦветОтрицательногоЧисла](ЦветаСтиля.ЦветОтрицательногоЧисла_(StyleColors.NegativeTextColor).md).

Доступность:

Толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - текст выделяется.  
Применимо только для колонок с данными типа [Число](lang__def_Number.md).

Использование в версии:

Доступен, начиная с версии 8.0.
