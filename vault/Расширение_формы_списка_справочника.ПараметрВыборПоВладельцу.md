Расширение формы списка справочника (Catalog list form extension)

ПараметрВыборПоВладельцу (ChoiceByOwnerParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника.md).   
Содержит ссылку на элемент-владелец подчиненного справочника, по которому может быть осуществлен выбор элемента.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
