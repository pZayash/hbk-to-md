Надпись (Label)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что надпись является гиперссылкой. Если значение свойства [Истина](lang__def_BooleanTrue.md), то текст надписи выводится с подчеркиванием, и при нажатии на надпись возникает событие [Нажатие](Надпись.Нажатие_(Label.Click).md).

Доступность:

Толстый клиент.

См. также:

[Надпись](Надпись_(Label).md), событие [Нажатие](Надпись.Нажатие_(Label.Click).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
