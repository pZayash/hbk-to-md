ПоследовательностьXDTO (XDTOSequence)

ПолучитьЗначениеXDTO (GetXDTOValue)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьЗначениеXDTO(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента последовательности.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Получает элемент данных XDTO по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если по указанному индексу записан текст, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
