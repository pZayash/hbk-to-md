ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ОбработкаУстановкиОписания (SetDescriptionProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаУстановкиОписания(<КлючОбъекта>, <КлючНастроек>, <ОписаниеНастроек>, <Пользователь>)

Параметры:

<КлючОбъекта>

Тип: Произвольный.   
Ключ объекта, для которого устанавливается описание.

<КлючНастроек>

Тип: Произвольный.   
Ключ настроек.

<ОписаниеНастроек>

Тип: [ОписаниеНастроек](ОписаниеНастроек.md).   
Описание настроек.

<Пользователь>

Тип: Строка.   
Идентификатор пользователя, для которого устанавливается описание.  
Если не указан или представляет пустую строку, то следует загружать настройки для текущего пользователя.

Описание:

Возникает при установке описания настроек.  
В данном обработчике необходимо реализовать установку описания настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.2.
