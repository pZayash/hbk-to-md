ТабличноеПоле (TableBox)

Шрифт (Font)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт_(Font).md).   
Содержит шрифт, используемый при отображении текста в ячейках табличного поля.

Доступность:

Толстый клиент.

Примечание:

Указанный шрифт используется, если для ячейки указан вид шрифта [АвтоШрифт](ВидШрифта.АвтоШрифт_(FontType.AutoFont).md).

Использование в версии:

Доступен, начиная с версии 8.0.
