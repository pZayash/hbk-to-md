Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПриСохраненииПользовательскихНастроекНаСервере (OnSaveUserSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСохраненииПользовательскихНастроекНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных_(DataCompositionUserSettings).md).   
Сохраняемые настройки.

Описание:

Вызывается при сохранении пользовательских настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
