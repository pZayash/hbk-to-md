Расширение формы записи регистра сведений (Information register record form extension)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РегистрСведенийМенеджерЗаписи.\<Имя регистра сведений\>](РегистрСведенийМенеджерЗаписи.Имя_регистра_сведений.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
