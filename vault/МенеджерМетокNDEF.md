Описание:

Предоставляет доступ к функциям сканирования меток NDEF.  
Для работы с объектом требуется функциональность NFC.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
