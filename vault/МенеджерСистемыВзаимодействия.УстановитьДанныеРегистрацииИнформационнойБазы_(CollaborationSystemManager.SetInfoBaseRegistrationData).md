МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьДанныеРегистрацииИнформационнойБазы (SetInfoBaseRegistrationData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьДанныеРегистрацииИнформационнойБазы(<ДанныеРегистрации>)

Параметры:

<ДанныеРегистрации> (обязательный)

Тип: [ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия](ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия_(CollaborationSystemInfoBaseRegistrationData).md), [Неопределено](lang__def_Undefined.md).   
Данные регистрации информационной базы на сервере системы взаимодействия.

Возвращаемое значение:

Если у пользователя нет права "АдминистрированиеДанных", генерируется исключение.

Описание:

Устанавливает данные регистрации информационной базы на сервере системы взаимодействия (для текущей области данных).  
Если передано [Неопределено](lang__def_Undefined.md), то регистрация информационной базы отменяется. При повторной регистрации после выполнения этого метода данные системы взаимодействия восстановлены не будут.  
Доступен только пользователям с правом "АдминистрированиеДанных".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
