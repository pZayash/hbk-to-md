ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

ИспользоватьПроксиОС (UseOSProxy)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, что при установке соединения в WebSocket-сервером будут использоваться (значение [Ложь](lang__def_BooleanFalse.md)) настройки прокси-сервера, указанные в свойстве [Прокси](ПараметрыWebSocketКлиентСоединения.Прокси_(WebSocketClientConnectionParameters.Proxy).md). Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то при установке соединения будут использоваться настройки прокси-сервера, указанные в настройках операционной системы того компьютера, с которого выполняется соединение.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
