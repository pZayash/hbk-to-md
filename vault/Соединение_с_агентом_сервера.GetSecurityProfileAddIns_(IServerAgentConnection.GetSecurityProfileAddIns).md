Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfileAddIns (GetSecurityProfileAddIns)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfileAddIns(<Кластер>, <Профиль>)

Параметры:

<Кластер> (необязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер серверов.

<Профиль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности, которому принадлежит список.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [Внешний компонент](Внешний_компонент_(ISecurityProfileAddIn).md).

Описание:

Получает разрешенные внешние компоненты.

Доступность:

Интеграция.

Примечание:

Если профиль с этим именем не найден, то исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
