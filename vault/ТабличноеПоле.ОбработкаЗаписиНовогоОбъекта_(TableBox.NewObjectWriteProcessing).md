ТабличноеПоле (TableBox)

ОбработкаЗаписиНовогоОбъекта (NewObjectWriteProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаЗаписиНовогоОбъекта(<Объект>, <СтандартнаяОбработка>)

Параметры:

<Объект>

Тип: Ссылка на объект информационной базы.   
Ссылка на добавленный объект.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при добавлении объекта в одной из подчиненных форм или при вызове в одной из подчиненных форм метода формы [ОповеститьОЗаписиНовогоОбъекта](Форма.ОповеститьОЗаписиНовогоОбъекта_(Form.NotifyWriteNewObject).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
