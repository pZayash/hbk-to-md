ВнешнийОтчет (ExternalReport)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: Строка, [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет.md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент.md), [ТекстовыйДокумент](ТекстовыйДокумент.md); другой объект, который может быть макетом.   

Описание:

Получает макет внешнего отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
