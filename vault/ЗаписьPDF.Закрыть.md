ЗаписьPDF (PDFWriter)

Закрыть (Close)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый файл.   
Закрытие файла выполняется автоматически при удалении объекта [ЗаписьPDF](ЗаписьPDF.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF.md), метод [ЗакрытьАсинх](ЗаписьPDF.ЗакрытьАсинх.md)  
[ЗаписьPDF](ЗаписьPDF.md), метод [НачатьЗакрытие](ЗаписьPDF.НачатьЗакрытие.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
