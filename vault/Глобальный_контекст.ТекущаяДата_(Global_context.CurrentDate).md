Глобальный контекст (Global context)

ТекущаяДата (CurrentDate)

Доступен, начиная с версии 8.0.

Синтаксис:

ТекущаяДата()

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   
Текущая (системная) дата.

Описание:

Определяет текущую (системную) дату на компьютере.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Если ДатаНачала = '00010101000000' Тогда     ДатаНачала = НачалоМесяца(ТекущаяДата()); КонецЕсли;  Если ДатаОкончания = '00010101000000' Тогда     ДатаОкончания = КонецМесяца(ТекущаяДата()); КонецЕсли; |

[Глобальный контекст](Глобальный_контекст.md), метод [ТекущаяУниверсальнаяДата](Глобальный_контекст.ТекущаяУниверсальнаяДата_(Global_context.CurrentUniversalDate).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
