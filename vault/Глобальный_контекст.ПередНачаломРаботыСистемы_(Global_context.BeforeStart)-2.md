Глобальный контекст (Global context)

ПередНачаломРаботыСистемы (BeforeStart)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередНачаломРаботыСистемы(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от запуска программы. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то программа не будет запущена.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при старте 1С:Предприятия в режиме приложения до открытия главного окна. В процедуре-обработчике данного события могут быть выполнены необходимые проверки и, при необходимости, установлен параметр отказа от запуска программы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Для режима запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) в данной процедуре не допускается открытие форм и других окон, не поддерживается выдача сообщений, установка текста в панели состояния, а также другие действия, требующие наличия главного окна.

См. также:

Использование в версии:

Доступен, начиная с версии 8.2.
