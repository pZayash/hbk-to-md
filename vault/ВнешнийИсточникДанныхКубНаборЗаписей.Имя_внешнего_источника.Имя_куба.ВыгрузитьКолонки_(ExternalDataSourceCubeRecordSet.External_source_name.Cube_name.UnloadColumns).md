ВнешнийИсточникДанныхКубНаборЗаписей.<Имя внешнего источника>.<Имя куба> (ExternalDataSourceCubeRecordSet.<External source name>.<Cube name>)

ВыгрузитьКолонки (UnloadColumns)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ВыгрузитьКолонки(<Колонки>)

Параметры:

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Список колонок для выгрузки в формате: "Колонка1, Колонка2...". Если список не задан, то будут выгружены все колонки.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   

Описание:

Создает таблицу значений с заданным списком колонок набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Выгружается только структура набора записей, данные не выгружаются.

Использование в версии:

Доступен, начиная с версии 8.3.3.
