Доступен, начиная с версии 8.0.

Описание:

Считывает записи из базы данных по установленному отбору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Рег = РегистрыБухгалтерии.Хозрасчетный; НаборЗаписей = Рег.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.Регистратор.Значение = Документ.Регистратор; НаборЗаписей.Прочитать(); |

Доступен, начиная с версии 8.0.
