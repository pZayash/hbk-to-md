ИнтернетПочтовыйПрофиль (InternetMailProfile)

ТокенДоступа (AccessToken)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит строковое значение полученного ранее токена доступа OAuth – access\_token.  
При вызове метода [Подключиться](ИнтернетПочта.Подключиться.md) объекта [ИнтернетПочта](ИнтернетПочта.md) аутентификация будет выполнена с использованием этого токена, если в свойстве [АутентификацияПоТокену](ИнтернетПочтовыйПрофиль.АутентификацияПоТокену.md) указано [Использовать](ИспользованиеАутентификацииИнтернетПочтыПоТокену.Использовать.md) или [Авто](ИспользованиеАутентификацииИнтернетПочтыПоТокену.Авто.md), и подключение осуществляется по протоколу IMAP или SMTP.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.19.
