Элементы коллекции:

[ОписаниеМакетаСхемыКомпоновкиДанных](ОписаниеМакетаСхемыКомпоновкиДанных_(DataCompositionSchemaTemplateDescription).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания макета.  
Возможно обращение к описанию посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Доступность:
