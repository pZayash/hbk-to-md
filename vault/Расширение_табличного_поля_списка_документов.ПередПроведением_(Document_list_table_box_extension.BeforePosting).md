# Расширение табличного поля списка документов.ПередПроведением

Расширение табличного поля списка документов (Document list table box extension)

ПередПроведением (BeforePosting)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередПроведением(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак проведения документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то проведение документа выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивным проведением документа в списке.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
