РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Загрузить (Load)

Доступен, начиная с версии 8.0.

Синтаксис:

Загрузить(<ТаблицаЗначений>)

Параметры:

<ТаблицаЗначений> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   
Источник для заполнения набора записей.

Описание:

Загружает набор записей значениями из переданной таблицы значений. При этом все прежние записи набора удаляются. Заполняются значения доступных для записи свойств записей регистра расчета, имена которых совпали c именами колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Рег = РегистрыРасчета.ОсновныеНачисленияСотрудниковОрганизации; НаборЗаписей = Рег.СоздатьНаборЗаписей(); Таб = Новый ТаблицаЗначений; // формирование таблицы  // ...  НаборЗаписей.Загрузить(Таб); |

Использование в версии:

Доступен, начиная с версии 8.0.
