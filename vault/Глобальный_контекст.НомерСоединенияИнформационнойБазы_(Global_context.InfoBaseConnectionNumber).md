Глобальный контекст (Global context)

НомерСоединенияИнформационнойБазы (InfoBaseConnectionNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

НомерСоединенияИнформационнойБазы()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает номер текущего соединения с информационной базой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[СоединениеИнформационнойБазы](СоединениеИнформационнойБазы_(InfoBaseConnection).md), свойство [НомерСоединения](СоединениеИнформационнойБазы.НомерСоединения_(InfoBaseConnection.ConnectionNumber).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
