МаскаXS (XSWildcard)

ОбновитьЭлементDOM (UpdateDOMElement)

Доступен, начиная с версии 8.1.

Синтаксис:

ОбновитьЭлементDOM()

Описание:

Производит синхронизацию свойств компонента и структуры элемента DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
