# Расширение табличного поля списка узлов.ПериодАвтоОбновления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [Расширение табличного поля списка узлов](Расширение_табличного_поля_списка_узлов_(Node_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_узлов__Свойства.md)

Расширение табличного поля списка узлов (Node list table box extension)

ПериодАвтоОбновления (AutoRefreshPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Задает период (в секундах), с которым будет производиться автоматическое обновление списка. Не может быть меньше 5 секунд. Если свойство [АвтоОбновление](Расширение_табличного_поля_списка_узлов.АвтоОбновление_(Node_list_table_box_extension.AutoRefresh).md) установлено как [Ложь](lang__def_BooleanFalse.md), то обновление не производится.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
