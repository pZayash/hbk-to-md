Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ПолучитьОтображениеСостояния (GetStatePresentation)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьОтображениеСостояния()

Возвращаемое значение:

Тип: [ОтображениеСостояния](ОтображениеСостояния.md), Неопределено.   

Описание:

Возвращает объект, описывающий состояние текущего отображения поля табличного документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
