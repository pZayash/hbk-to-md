# РезультатXPath.БулевоЗначение

РезультатXPath (XPathResult)

БулевоЗначение (BooleanValue)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит полученное логическое значение результата.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение не может быть преобразовано к типу [Булево](lang__Булево_(Boolean).md), то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
