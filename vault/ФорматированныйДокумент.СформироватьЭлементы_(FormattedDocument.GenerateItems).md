ФорматированныйДокумент (FormattedDocument)

СформироватьЭлементы (GenerateItems)

Доступен, начиная с версии 8.3.1.

Синтаксис:

СформироватьЭлементы(<Начало>, <Конец>)

Параметры:

<Начало> (необязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Закладка, указывающая на начало диапазона, элементы которого надо вернуть в массиве.  
Если закладка не задана, то элементы будут выбраны с самого начала документа.

<Конец> (необязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Закладка, указывающая на конец диапазона, элементы которого надо вернуть в массиве.  
Если закладка не задана, то элементы будут выбраны до самого конца документа.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов.

Описание:

Получает точный набор элементов, входящих в диапазон между заданными закладками. При этом документ, при необходимости, меняется - выполняется разделение элементов так, чтобы указанная область содержала элементы полностью.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
