КодировкаИменФайловВZipФайле (FileNamesEncodingInZipFile)

КодировкаОСДополнительноUTF8 (OSEncodingWithUTF8)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [КодировкаОСДополнительноUTF8](КодировкаИменФайловВФайлеАрхива.КодировкаОСДополнительноUTF8_(FileNamesEncodingInArchiveFile.OSEncodingWithUTF8).md)

Доступен, начиная с версии 8.3.9.

Описание:

Имена файлов внутри ZIP-архива будут записаны в двух кодировках: текущая кодировка операционной системы и UTF8.

Использование в версии:

Доступен, начиная с версии 8.3.9.

Не рекомендуется использовать, начиная с версии 8.3.26.
