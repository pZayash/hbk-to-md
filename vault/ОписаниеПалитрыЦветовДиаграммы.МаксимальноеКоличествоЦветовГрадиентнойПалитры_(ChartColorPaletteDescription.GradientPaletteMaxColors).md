ОписаниеПалитрыЦветовДиаграммы (ChartColorPaletteDescription)

МаксимальноеКоличествоЦветовГрадиентнойПалитры (GradientPaletteMaxColors)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Максимальное количество цветов в градиентной палитре (полученные цвета повторяются).  
Если значение равно 0, используется количество серий.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
