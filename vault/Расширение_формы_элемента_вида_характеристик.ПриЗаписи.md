Расширение формы элемента вида характеристик (Characteristic type item form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПриЗаписи.md) объекта [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
