ФормаКлиентскогоПриложения (ClientApplicationForm)

СохраняемыеВНастройкахДанныеМодифицированы (SavedInSettingsDataModified)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак модифицированности сохраняемых в настройках формы данных.  
Если [Истина](lang__def_BooleanTrue.md) и у формы при конфигурировании установлена необходимость автоматически сохранять данные формы, то при закрытии формы данные формы сохраняются с вызовом обработчика.   
При изменении сохраняемых данных формы платформа автоматически помещает значение [Истина](lang__def_BooleanTrue.md) в свойство.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
