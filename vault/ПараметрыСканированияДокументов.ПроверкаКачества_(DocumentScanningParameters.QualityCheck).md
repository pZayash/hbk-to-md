ПараметрыСканированияДокументов (DocumentScanningParameters)

ПроверкаКачества (QualityCheck)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ПроверкаКачестваСканированияДокументов](ПроверкаКачестваСканированияДокументов_(DocumentScanningCheckingQuality).md).   
Управляет требованием определенного уровня качества отсканированного документа и/или предупреждением пользователя о необходимости отсканировать документ повторно.  
Если требуемое качество документа и его границы не достигнуто, то в окне редактирования страницы документа кнопки "Удалить" и "Готово" меняются на "Переснять" и "Оставить".  
Значение по умолчанию: [ПредупреждатьНижеСреднего](ПроверкаКачестваСканированияДокументов.ПредупреждатьНижеСреднего_(DocumentScanningCheckingQuality.WarnBelowMedium).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
