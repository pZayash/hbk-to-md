Описание:

Содержит варианты обновления текста редактирования в поле ввода формы клиентского приложения при установке значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

См. также:

[Расширение поля формы для поля ввода](Расширение_поля_формы_для_поля_ввода_(Form_field_extension_for_a_text_box).md), свойство [ОбновлениеТекстаРедактирования](Расширение_поля_формы_для_поля_ввода.ОбновлениеТекстаРедактирования_(Form_field_extension_for_a_text_box.EditTextUpdate).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
