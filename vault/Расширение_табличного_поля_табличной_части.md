Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md), предназначеные для интерактивной работы с табличной частью. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) имеют значения типа [Строка табличной части](Строка_табличной_части.md)

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
