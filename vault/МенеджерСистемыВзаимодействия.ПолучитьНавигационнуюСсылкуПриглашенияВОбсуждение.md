МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьНавигационнуюСсылкуПриглашенияВОбсуждение (GetInviteUrlToConversation)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьНавигационнуюСсылкуПриглашенияВОбсуждение(<Обсуждение>, <ТолькоВидеоконференция>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Обсуждение.

<ТолькоВидеоконференция> (необязательный)

Тип: Булево.   
Если Ложь, то в обсуждении есть возможность читать и писать сообщения.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: Строка.   
Если подключена работа с внешними пользователями, то возвращает внешнюю ссылку, иначе – внутреннюю.  
  
Если не подключена работа с видеозвонками и <ТолькоВидеоконференция> имеет значени Истина, то возвращает пустую ссылку.

Описание:

Возвращает навигационную ссылку-приглашение, по которой можно получить доступ к видеозвонку и обсуждению, если <ТолькоВидеоконференция> имеет значение Ложь.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
