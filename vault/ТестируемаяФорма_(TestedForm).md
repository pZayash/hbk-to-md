Описание:

Представляет абстракцию формы (управляемой и обычной). Позволяет вызвать закрытие окна и получить список подчиненных элементов управления. Может содержать объекты типа [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы_(TestedFormGroup).md), [ТестируемаяКнопкаФормы](ТестируемаяКнопкаФормы_(TestedFormButton).md), [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы_(TestedFormTable).md), [ТестируемоеПолеФормы](ТестируемоеПолеФормы_(TestedFormField).md), [ТестируемаяДекорацияФормы](ТестируемаяДекорацияФормы_(TestedFormDecoration).md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
