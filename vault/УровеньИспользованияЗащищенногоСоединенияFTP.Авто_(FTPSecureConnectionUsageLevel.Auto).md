УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Уровень использования защищенного соединения определяется автоматически:

- Если параметр [ЗащищенноеСоединение](html.md) не задан, то производится попытка подключиться по защищенному каналу, если это возможно. Если сервер не поддерживает защищенное соединение, передача данных происходит по открытому каналу.
- Если параметр [ЗащищенноеСоединение](html.md) задан, то защищенное соединение требуется и как для управляющих команд, так и для данных.

Использование в версии:

Доступен, начиная с версии 8.3.9.
