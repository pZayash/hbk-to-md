СтандартнаяКомандаПоляПланировщика (PlannerStandardCommand)

СоздатьЭлемент (CreateItem)

Доступен, начиная с версии 8.3.23.

Описание:

Создает элемент планировщика в месте нажатия правой кнопки мыши. Используется только для источника типа [ПустаяОбластьЭлементов](ИсточникКомандПоляПланировщика.ПустаяОбластьЭлементов_(PlannerCommandSource.EmptyItemsArea).md) .  
В контекстном меню команда будет иметь представление "Создать".

Использование в версии:

Доступен, начиная с версии 8.3.23.
