# КолонкиСписка

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Универсальные объекты](Универсальные_объекты.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаСписка](КолонкаСписка.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются объекты описания колонки списка.  
Возможно обращение к описанию колонки списка посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) колонки.  

Свойства:

[\<Имя колонки\> (\<Column name\>)](КолонкиСписка.Имя_колонки.md)  

Методы:

[Добавить (Add)](КолонкиСписка.Добавить.md)  
[Количество (Count)](КолонкиСписка.Количество.md)  
[Найти (Find)](КолонкиСписка.Найти.md)  
[Очистить (Clear)](КолонкиСписка.Очистить.md)  
[Получить (Get)](КолонкиСписка.Получить.md)  
[Удалить (Delete)](КолонкиСписка.Удалить.md)  
[Установить (Set)](КолонкиСписка.Установить.md)  

Описание:

Предназначен для управления набором колонок списка прикладных объектов, используемого для отображения в табличном поле.

Доступность:

Сервер, толстый клиент, внешнее соединение.
