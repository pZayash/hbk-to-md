Элементы коллекции:

Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются действия элемента результата глобального поиска.  
Возможно обращение к элементам коллекции посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Представляет собой коллекцию действий элемента результата глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).
