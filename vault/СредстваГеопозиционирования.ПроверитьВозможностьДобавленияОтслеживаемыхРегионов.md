СредстваГеопозиционирования (LocationTools)

ПроверитьВозможностьДобавленияОтслеживаемыхРегионов (CheckAvailabilityOfEnablingGeofencesMonitoring)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПроверитьВозможностьДобавленияОтслеживаемыхРегионов(<Количество>)

Параметры:

<Количество> (необязательный)

Тип: Число.   
Количество геозон для включения (должно быть строго больше нуля).

Возвращаемое значение:

Тип: Булево.   
Истина - отслеживание возможно, и Ложь - если попытка отслеживания вызовет исключение.

Описание:

Проверяет возможность включить одну (по умолчанию) или несколько геозон для отслеживания в приложении.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Если параметры заданы неверно (количество геозон нулевое или отрицательно), будет сгененировано исключение.

См. также:

[СредстваГеопозиционирования](СредстваГеопозиционирования.md), метод [ВключитьОтслеживаниеГеозон](СредстваГеопозиционирования.ВключитьОтслеживаниеГеозон.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
