Глобальный контекст (Global context)

СтрЗаканчиваетсяНа (StrEndsWith)

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Синтаксис:

СтрЗаканчиваетсяНа(<Строка>, <СтрокаПоиска>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, окончание которой проверяется на совпадение с подстрокой поиска.

<СтрокаПоиска> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, содержащая предполагаемое окончание строки.  
В случае если переданное значение является пустой строкой генерируется исключительная ситуация.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - строка заканчивается указанной подстрокой; [Ложь](lang__def_BooleanFalse.md) - в противном случае

Описание:

Определяет, заканчивается ли строка указанной подстрокой. Определение выполняется с учетом регистра, аналогично функции [Найти](Глобальный_контекст.Найти_(Global_context.Find).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [СтрНачинаетсяС](Глобальный_контекст.СтрНачинаетсяС_(Global_context.StrStartsWith).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Описание изменено в версии 8.3.15.
