ОбъектМетаданных: Поле (MetadataObject: Field)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает на отображение данных реквизита в режиме пароля (символы заменяются звездочками при отображении).   
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл для реквизитов типа [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
