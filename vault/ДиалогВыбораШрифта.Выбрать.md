ДиалогВыбораШрифта (FontChooseDialog)

Выбрать (Choose)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать()

Возвращаемое значение:

Тип: Булево.   
Истина - шрифт выбран; Ложь - в противном случае.

Описание:

Открывает диалог выбора шрифта.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [Показать](ДиалогВыбораШрифта.Показать.md).

См. также:

[ДиалогВыбораШрифта](ДиалогВыбораШрифта.md), метод [ВыбратьАсинх](ДиалогВыбораШрифта.ВыбратьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
