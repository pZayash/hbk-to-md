Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы.md).  
Расширение предоставляет возможность доступа к свойствам динамического списка.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока.md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель.md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки.md), используются значения ссылок на объекты, которые содержит главная таблица динамического списка. Например, СправочникСсылка.Товары. Также эти свойства могут содержать значения типа [СтрокаГруппировкиДинамическогоСписка](СтрокаГруппировкиДинамическогоСписка.md).  
Если главная таблица не задана или является виртуальной таблицей регистра, в качестве значений будут использоваться числа, являющиеся идентификаторами строки в выборке динамического списка.  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные.md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки.md) возвращают объект типа [ДанныеФормыСтруктура](ДанныеФормыСтруктура.md). Их изменение не отразится на данных формы.   
Имена свойств этого объекта соответствуют путям компоновки данных для полей, отображаемых в таблице.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
