КнопкаФормы (FormButton)

Картинка (Picture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Содержит картинку, отображаемую на кнопке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.2.
