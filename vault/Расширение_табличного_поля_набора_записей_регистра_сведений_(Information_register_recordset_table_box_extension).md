Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для набора записей регистра сведений. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержат значение типа [РегистрСведенийЗапись.](РегистрСведенийЗапись.Имя_регистра_сведений_(InformationRegisterRecord.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийЗапись.Имя_регистра_сведений_(InformationRegisterRecord.Information_register_name).md)>.
