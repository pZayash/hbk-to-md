ДействиеПриНесоответствииПароляТребованиямПриАутентификации (ActionOnThePasswordRequirementsViolationOnAuthentication)

ПредложитьСменуПароля (SuggestPasswordChange)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Отображается диалог смены пароля с возможностью пропустить действие, и выполняется запись события в журнал регистрации.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
