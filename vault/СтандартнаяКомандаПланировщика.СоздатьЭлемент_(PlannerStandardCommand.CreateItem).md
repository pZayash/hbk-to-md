# СтандартнаяКомандаПланировщика.СоздатьЭлемент

СтандартнаяКомандаПланировщика (PlannerStandardCommand)

СоздатьЭлемент (CreateItem)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [СоздатьЭлемент](СтандартнаяКомандаПоляПланировщика.СоздатьЭлемент_(PlannerStandardCommand.CreateItem).md)

Доступен, начиная с версии 8.3.16.

Описание:

Создает элемент планировщика в месте нажатия правой кнопки мыши. Используется только для источника типа [ПустаяОбластьЭлементов](ИсточникКомандПланировщика.ПустаяОбластьЭлементов_(PlannerCommandSource.EmptyItemsArea).md) .  
В контекстном меню команда будет иметь представление "Создать".

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
