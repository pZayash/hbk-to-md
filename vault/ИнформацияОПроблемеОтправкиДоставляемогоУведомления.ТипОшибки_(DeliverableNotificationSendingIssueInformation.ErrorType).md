Только чтение.

Описание:

Тип: [ТипОшибкиОтправкиДоставляемогоУведомления](ТипОшибкиОтправкиДоставляемогоУведомления_(DeliverableNotificationSendErrorType).md).

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
