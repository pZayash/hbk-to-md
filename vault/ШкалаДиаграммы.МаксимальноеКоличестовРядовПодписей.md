ШкалаДиаграммы (ChartScale)

МаксимальноеКоличестовРядовПодписей (MaxLabelRows)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Устанавливает максимальное количество рядов подписей.   
0 - количество рассчитывается автоматически: если все подписи умещаются в один ряд, они будут отображаться в один ряд.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
