Диаграмма (Chart)

ПоложениеПодписей (LabelLocation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеПодписейКДиаграмме](ПоложениеПодписейКДиаграмме.md).   
Определяет положение подписей к значениям гистограмм.  
Значение по умолчанию: [Авто](ПоложениеПодписейКДиаграмме.Авто.md). В режиме совместимости [Версия8\_3\_23](РежимСовместимости.Версия8_3_23.md) и ниже - [Край](ПоложениеПодписейКДиаграмме.Край.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
