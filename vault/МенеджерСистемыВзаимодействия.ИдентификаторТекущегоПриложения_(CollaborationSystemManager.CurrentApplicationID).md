МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ИдентификаторТекущегоПриложения (CurrentApplicationID)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ИдентификаторТекущегоПриложения()

Возвращаемое значение:

Тип: [ИдентификаторПриложенияСистемыВзаимодействия](ИдентификаторПриложенияСистемыВзаимодействия_(CollaborationSystemApplicationID).md).   

Описание:

Возвращает идентификатор текущего приложения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
