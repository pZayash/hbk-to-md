ПроцессорВыводаРезультатаКомпоновкиДанныхВКоллекциюЗначений (DataCompositionResultValueCollectionOutputProcessor)

ВывестиЭлемент (OutputItem)

Доступен, начиная с версии 8.2.

Синтаксис:

ВывестиЭлемент(<ЭлементРезультата>)

Параметры:

<ЭлементРезультата> (обязательный)

Тип: [ЭлементРезультатаКомпоновкиДанных](ЭлементРезультатаКомпоновкиДанных_(DataCompositionResultItem).md).   
Элемент результата, выводимый в результирующий документ.

Описание:

Выводит элемент результата компоновки отчета в документ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
