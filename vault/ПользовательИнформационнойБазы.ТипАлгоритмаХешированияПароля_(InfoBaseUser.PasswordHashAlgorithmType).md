ПользовательИнформационнойБазы (InfoBaseUser)

ТипАлгоритмаХешированияПароля (PasswordHashAlgorithmType)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [ТипАлгоритмаХешированияПаролейПользователей](ТипАлгоритмаХешированияПаролейПользователей_(UserPasswordHashAlgorithmType).md).   
Содержит тип хеш-функции, используемой для хранения пароля этого пользователя. Поле предназначено для выявления пользователей, у которых используется старая хеш-функция. При переносе пользователей между базами это поле не требуется копировать, т.к. оно является производным от значения поля [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
