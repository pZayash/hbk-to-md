ДиаграммаГанта (GanttChart)

СвернутьСерию (CollapseSeries)

Доступен, начиная с версии 8.0.

Синтаксис:

СвернутьСерию(<Серия>, <СПодчиненными>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта.md).   
Сворачиваемая серия.

<СПодчиненными> (необязательный)

Тип: Булево.   
Определяет необходимость сворачивания серии вместе с подчиненными сериями. Истина - свернуть вместе с подчиненными.  
Значение по умолчанию: Ложь.

Описание:

Сворачивает серии, возможно вместе с подчиненными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
