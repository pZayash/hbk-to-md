# МенеджерВстроенныхПокупок.ПоддерживаетсяРасходованиеПокупок

МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПоддерживаетсяРасходованиеПокупок (PurchaseConsumingSupported)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоддерживаетсяРасходованиеПокупок()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

- [Истина](lang__def_BooleanTrue.md) - для Windows Phone и для Android при наличии установленных служб Google Play Services телефонного модуля,
- [Ложь](lang__def_BooleanFalse.md) - для iOS.

Описание:

Определяет, доступно ли "расходование" покупок на текущей платформе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерВстроенныхПокупок](МенеджерВстроенныхПокупок_(InAppPurchasesManager).md), метод [ИзрасходоватьПокупку](МенеджерВстроенныхПокупок.ИзрасходоватьПокупку_(InAppPurchasesManager.ConsumePurchase).md)  

Использование в версии:

Доступен, начиная с версии 8.3.8.
