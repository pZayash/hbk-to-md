ЭлементСоставаПланаОбмена (ExchangePlanContentItem)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Содержит объект метаданных, описывающий объекты конфигурации, которые могут входит в состав плана обмена (например: [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник_(MetadataObject_Catalog).md), [ОбъектМетаданных: Документ](ОбъектМетаданных_Документ_(MetadataObject_Document).md) и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
