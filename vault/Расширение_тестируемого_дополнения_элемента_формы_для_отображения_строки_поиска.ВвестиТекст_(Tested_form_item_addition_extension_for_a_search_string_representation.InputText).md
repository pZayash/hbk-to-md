Расширение тестируемого дополнения элемента формы для отображения строки поиска (Tested form item addition extension for a search string representation)

ВвестиТекст (InputText)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ВвестиТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Вводимый текст.

Описание:

Осуществляет ввод текста в строку поиска. Переданное значение целиком замещает весь текст, ранее содержащийся в строке поиска.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
