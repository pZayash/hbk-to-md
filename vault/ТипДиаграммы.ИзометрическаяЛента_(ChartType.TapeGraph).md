ТипДиаграммы (ChartType)

ИзометрическаяЛента (TapeGraph)

Доступен, начиная с версии 8.0.

Описание:

Изометрическая диаграмма, значения внутри одной серии объединяются лентой. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График_(ChartType.Line).md).

Использование в версии:

Доступен, начиная с версии 8.0.
