МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблонСообщения (GetMessageTemplate)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблонСообщения(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplateID).md).   
Идентификатор шаблона сообщения.

Возвращаемое значение:

Тип: [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplate).md), [Неопределено](lang__def_Undefined.md).   
Для интеграции типа WhatsApp возвращает шаблон сообщения по его идентификатору.  
Для остальных интеграций или если шаблон не найден возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Получает шаблон сообщения системы взаимодействия.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.
