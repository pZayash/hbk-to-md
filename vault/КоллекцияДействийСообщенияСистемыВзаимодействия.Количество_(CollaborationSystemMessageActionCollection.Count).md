Количество (Count)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Количество](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Количество_(CollaborationSystemMessageButtonPanelButtonRow.Count).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество действий в коллекции действий сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
