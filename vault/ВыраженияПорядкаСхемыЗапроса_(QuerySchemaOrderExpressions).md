Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ВыражениеПорядкаСхемыЗапроса](ВыражениеПорядкаСхемыЗапроса_(QuerySchemaOrderExpression).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения порядка.  
Возможно обращение к выражению порядка посредством оператора [...]. В качестве аргумента передается индекс.
