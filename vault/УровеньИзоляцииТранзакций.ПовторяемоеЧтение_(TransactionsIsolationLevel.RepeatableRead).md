УровеньИзоляцииТранзакций (TransactionsIsolationLevel)

ПовторяемоеЧтение (RepeatableRead)

Доступен, начиная с версии 8.3.5.

Описание:

Уровень изоляции транзакций внешнего источника данных, при котором операция чтения возвращает данные, зафиксированные в других транзакциях на момент начала транзакции, операция записи на измененных в других транзакциях данных завершается с ошибкой (в зависимости от внешнего источника данных, может быть обеспечено поведение уровня [Упорядочиваемость](УровеньИзоляцииТранзакций.Упорядочиваемость_(TransactionsIsolationLevel.Serializable).md)).

Использование в версии:

Доступен, начиная с версии 8.3.5.
