Гистограмма (Column)

Доступен, начиная с версии 8.0.

Описание:

Вертикальная гистограмма. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График_(ChartType.Line).md).

Использование в версии:

Доступен, начиная с версии 8.0.
