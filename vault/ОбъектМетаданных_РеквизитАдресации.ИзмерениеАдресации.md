ИзмерениеАдресации (AddressingDimension)

Только чтение.

Описание:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Описывает реквизиты адресации задачи.

Использование в версии:

Доступен, начиная с версии 8.1.
