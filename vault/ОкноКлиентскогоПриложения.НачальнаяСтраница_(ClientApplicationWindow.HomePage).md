ОкноКлиентскогоПриложения (ClientApplicationWindow)

НачальнаяСтраница (HomePage)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Для окна, представляющего собой начальную страницу, свойство имеет значение [Истина](lang__def_BooleanTrue.md).  
Для остальных окон значение свойства - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
