ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Сгруппировать (Group)

Доступен, начиная с версии 8.2.

Синтаксис:

Сгруппировать(<ИмяГруппы>, <РасположениеЗаголовка>)

Параметры:

<ИмяГруппы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Название группы.

<РасположениеЗаголовка> (необязательный)

Тип: [РасположениеЗаголовкаГруппировкиТабличногоДокумента](РасположениеЗаголовкаГруппировкиТабличногоДокумента_(SpreadsheetDocumentGroupHeaderPlacement).md).   
Расположение заголовка группировки.  
Значение по умолчанию: [Авто](РасположениеЗаголовкаГруппировкиТабличногоДокумента.Авто_(SpreadsheetDocumentGroupHeaderPlacement.Auto).md).

Описание:

Группирует область, состоящую из строк или колонок.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Область.Сгруппировать("Товар"); |

См. также:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md), метод [Разгруппировать](ОбластьЯчеекТабличногоДокумента.Разгруппировать_(SpreadsheetDocumentRange.Ungroup).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
