ТестируемоеПриложение (TestedApplication)

По умолчанию

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый ТестируемоеПриложение(<ИмяКомпьютера>, <Порт>, <ИдентификаторКлиента>)

Параметры:

<ИмяКомпьютера> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя или IP-адрес компьютера на котором запущено тестируемое приложение. Например: TEST1 или 165.15.27.11.  
Значение по умолчанию: 127.0.0.1.

<Порт> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер порта TCP/IP, который нужно использовать для передачи данных. По умолчанию используется порт 1538.  
При соединении с тонким клиентом - порт по умолчанию или порт, который указан через ключ командной строки клиента /TESTCLIENT -TPort<Номер порта TCP-IP>.  
При соединении с веб-клиентом - порт на веб-сервере с номером по умолчанию или который указан в конфигурационном файле testcfg.xml.  
  
При тестировании через веб-клиент информационной базы, которая расположена на сервере "1С:Предприятия", необходимо подключаться по адресу и порту менеджера кластера сервера (порт по умолчанию 1541).  
Значение по умолчанию: 1538.

<ИдентификаторКлиента> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка, идентифицирующая веб-клиент на стороне сервера.  
Не используется в случае работы с тонким клиентом.

Описание:

Создает новый объект на основе переданных параметров.

Использование в версии:

Доступен, начиная с версии 8.3.1.
