Расширение дополнения элемента формы для отображения управления поиском (Form item addition extension for search control)

МаксимальнаяШирина (MaxWidth)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную ширину элемента.  
Используется, если свойство [АвтоМаксимальнаяШирина](Расширение_дополнения_элемента_формы_для_отображения_управления_поиском.АвтоМаксимальнаяШирина_(Form_item_addition_extension_for_search_control.AutoMaxWidth).md) установлено в значение [Ложь](lang__def_BooleanFalse.md).  
Задается в символах.  
Значение 0 означает, что максимальная ширина не ограничена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
