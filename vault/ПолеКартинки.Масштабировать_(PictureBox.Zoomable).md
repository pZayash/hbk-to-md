ПолеКартинки (PictureBox)

Масштабировать (Zoomable)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено ([Истина](lang__def_BooleanTrue.md)), то для картинки, не помещающейся в области размещения, появляются линейки прокрутки.   
Изменить размер картинки можно с помощью команд контекстного меню.

Доступность:

Толстый клиент.

Примечание:

Для картинок, размещенных в режиме [РеальныйРазмер](РазмерКартинки.РеальныйРазмер_(PictureSize.RealSize).md).

Использование в версии:

Доступен, начиная с версии 8.0.
