Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfileApplication (RegSecurityProfileApplication)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfileApplication(<Кластер>, <ИмяПрофиля>, <Приложение>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должно быть зарегистрировано приложение.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<Приложение> (обязательный)

Тип: [Приложение](Приложение_(ISecurityProfileApplication).md).   
Регистрируемое приложение.

Описание:

Выполняет регистрацию нового разрешенного приложения или обновление существующего.

Доступность:

Интеграция.

Примечание:

Если разрешенное приложение существует, то выполняется обновление, в противном случае - добавление.  
Если профиль не найден, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
