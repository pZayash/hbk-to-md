Описание:

Содержит данные, необходимые для получения программной лицензии на платформу.  
Может быть сохранен в настройках.  
Может использоваться в методе [ЗаполнитьЗначенияСвойств](Глобальный_контекст.ЗаполнитьЗначенияСвойств_(Global_context.FillPropertyValues).md) в качестве источника и в качестве приемника.

Доступность:

Тонкий клиент, сервер, внешнее соединение.   
Сериализуется.

См. также:

[МенеджерПолученияЛицензий](МенеджерПолученияЛицензий_(LicenseAcquisitionManager).md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.27.
