МенеджерОтображенияРекламы (AdRepresentationManager)

УстановитьИдентификаторПолноэкраннойРекламы (SetFullScreenAdID)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УстановитьИдентификаторПолноэкраннойРекламы(<ИдентификаторРекламногоБлока>)

Параметры:

<ИдентификаторРекламногоБлока> (необязательный)

Тип: Строка.   
Идентификатор рекламного блока для рекламы Google.   
Параметр является обязательным, если реклама используется в приложении на Android. В остальных случаях параметр игнорируется.

Описание:

Устанавливает идентификатор рекламного блока, который будет использован для полноэкранной рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
