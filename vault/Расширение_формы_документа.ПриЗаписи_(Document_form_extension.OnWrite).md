# Расширение формы документа.ПриЗаписи

Расширение формы документа (Document form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись документа выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ДокументОбъект.Имя_документа.ПриЗаписи_(DocumentObject.Document_name.OnWrite).md) объекта [ДокументОбъект.](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)<[Имя документа](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

См. также:

[ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md), событие [ПриЗаписи](ДокументОбъект.Имя_документа.ПриЗаписи_(DocumentObject.Document_name.OnWrite).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
