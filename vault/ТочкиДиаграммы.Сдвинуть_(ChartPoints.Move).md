# ТочкиДиаграммы.Сдвинуть

ТочкиДиаграммы (ChartPoints)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Сдвинуть(<Точка>, <Смещение>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md).   
Сдвигаемая точка.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое необходимо сдвинуть точку.

Описание:

Сдвигает точку на указанное количество позиций.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
