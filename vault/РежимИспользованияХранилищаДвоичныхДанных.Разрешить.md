РежимИспользованияХранилищаДвоичныхДанных (BinaryDataStorageUseMode)

Разрешить (Enable)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Разрешает использование хранилища двоичных данных.

Доступность:

Сервер.

Примечание:

Если хранилище двоичных данных не существует, оно будет создано.

Использование в версии:

Доступен, начиная с версии 8.3.23.
