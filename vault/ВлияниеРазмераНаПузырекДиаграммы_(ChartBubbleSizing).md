Описание:

Содержит варианты зависимости размера пузырька от значения размера точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartBubbleSizing.

Использование в версии:

Доступен, начиная с версии 8.3.10.
