# Расширение поля формы для поля диаграммы Ганта.ТекущийИнтервал

Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ТекущийИнтервал (CurrentInterval)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ИдентификаторИнтервалаДиаграммыГанта](ИдентификаторИнтервалаДиаграммыГанта_(GanttChartIntervalID).md).   
Содержит идентификатор выделенного интервала диаграммы. Если устанавливаемый идентификатор интервала не является допустимым для данной диаграммы, то такая установка игнорируется. Значение по умолчанию - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
