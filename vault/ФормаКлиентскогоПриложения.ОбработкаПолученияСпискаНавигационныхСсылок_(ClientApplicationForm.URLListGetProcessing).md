ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаПолученияСпискаНавигационныхСсылок (URLListGetProcessing)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ОбработкаПолученияСпискаНавигационныхСсылок(<СписокНавигационыхСсылок>, <КлючПоУмолчанию>)

Параметры:

<СписокНавигационыхСсылок>

Тип: [СписокВыбораНавигационнойСсылки](СписокВыбораНавигационнойСсылки_(URLChoiseList).md).   
Предварительно заполняется описаниями вариантов стандартных навигационных ссылок, поддерживаемых платформой. В обработчике можно добавить новые варианты, переставить или удалить стандартные варианты.  
Если у формы свойство [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка_(ClientApplicationForm.AutoURL).md) имеет значение [Ложь](lang__def_BooleanFalse.md), то обработчику передается пустой <СписокНавигационыхСсылок>.

<КлючПоУмолчанию>

Тип: Произвольный.   
В данный параметр нужно поместить значение ключа из списка ссылок, соответствующего ссылке, которую нужно использовать по умолчанию.

Описание:

Вызывается перед открытием диалогов получения навигационной ссылки и добавления в избранное. Параметр <СписокНавигационыхСсылок> предварительно заполняется описаниями вариантов стандартных навигационных ссылок, поддерживаемых платформой. В обработчике можно добавить новые варианты, переставить или удалить стандартные варианты, указать ключ любого из элементов списка, в т.ч. стандартного, в качестве ключа по умолчанию.  
Если у формы установлено свойство [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка_(ClientApplicationForm.AutoURL).md) имеет значение [Ложь](lang__def_BooleanFalse.md), то обработчику передается пустой <СписокНавигационыхСсылок>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
