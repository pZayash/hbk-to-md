ТипШтрихКода (BarcodeType)

UPCE (UPCE)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Universal Product Code. Код из 8 цифр, эквивалентный UPCA, но использующий систему сжатия лишних нулей.

Использование в версии:

Доступен, начиная с версии 8.3.25.
