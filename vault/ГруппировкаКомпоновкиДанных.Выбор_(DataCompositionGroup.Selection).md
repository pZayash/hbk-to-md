ГруппировкаКомпоновкиДанных (DataCompositionGroup)

Выбор (Selection)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных_(DataCompositionSelectedFields).md).   
Содержит набор полей, выводимых в группировке.  
Данное свойство недоступно для чтения и записи у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
