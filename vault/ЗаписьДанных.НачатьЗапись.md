ЗаписьДанных (DataWriter)

НачатьЗапись (BeginWrite)

Доступен, начиная с версии 8.3.8.

Вариант синтаксиса: Запись двоичных данных

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ДвоичныеДанные>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Объект, данные которого будут записаны в поток.

Описание варианта метода:

Записывает в поток данные из объекта типа [ДвоичныеДанные](ДвоичныеДанные.md).

Вариант синтаксиса: Запись результата чтения данных

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <РезультатЧтенияДанных>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<РезультатЧтенияДанных> (обязательный)

Тип: [РезультатЧтенияДанных](РезультатЧтенияДанных.md).   
Объект типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md), данные которого будут записаны в поток.

Описание варианта метода:

Записывает в целевой поток данные из объекта типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Описание:

Вызывает начало записи данных в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [Записать](ЗаписьДанных.Записать.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьАсинх](ЗаписьДанных.ЗаписатьАсинх.md)

Доступен, начиная с версии 8.3.8.
