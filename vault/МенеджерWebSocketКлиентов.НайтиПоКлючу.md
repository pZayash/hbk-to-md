МенеджерWebSocketКлиентов (WebSocketClientsManager)

НайтиПоКлючу (FindByKey)

Доступен, начиная с версии 8.3.27.

Синтаксис:

НайтиПоКлючу(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Идентификатор соединения.

Возвращаемое значение:

Тип: [WebSocketКлиент](WebSocketКлиент.md), Неопределено.   

Описание:

Возвращает [WebSocketКлиент](WebSocketКлиент.md) с ключом или Неопределено, если такого не предопределенного нет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
