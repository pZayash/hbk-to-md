УниверсальнаяДата (UniversalDate)

Доступен, начиная с версии 8.3.6.

Описание:

Дата в поясе UTC.   
Пример для формата ISO: "2009-02-15T06:00:00Z".

Использование в версии:

Доступен, начиная с версии 8.3.6.
