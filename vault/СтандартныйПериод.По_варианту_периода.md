СтандартныйПериод (StandardPeriod)

По варианту периода

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СтандартныйПериод(<Вариант>)

Параметры:

<Вариант> (обязательный)

Тип: [ВариантСтандартногоПериода](ВариантСтандартногоПериода.md).   
Вариант периода, с которым нужно создать стандартный период.

Описание:

Создает стандартный период с указанным вариантом периода.

Использование в версии:

Доступен, начиная с версии 8.2.
