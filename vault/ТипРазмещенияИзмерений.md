Доступен, начиная с версии 8.0.

Значения

[Вместе (Together)](ТипРазмещенияИзмерений.Вместе.md)  
[Отдельно (Separately)](ТипРазмещенияИзмерений.Отдельно.md)  
[ОтдельноИТолькоВИтогах (SeparatelyAndInTotalsOnly)](ТипРазмещенияИзмерений.ОтдельноИТолькоВИтогах.md)  

Описание:

Содержит варианты расположения измерений в сводной таблице и построителе отчета.  
Для объекта [СводнаяТаблица](СводнаяТаблица.md) значения Отдельно и ОтдельноИТолькоВИтогах интерпретируются как одно и то же значение - ОтдельноИТолькоВИтогах.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.
