# Рабочий сервер.TemporaryAllowedProcessesTotalMemoryTimeLimit

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md) › [Свойства](Рабочий_сервер__Свойства.md)

Рабочий сервер (IWorkingServerInfo)

TemporaryAllowedProcessesTotalMemoryTimeLimit (TemporaryAllowedProcessesTotalMemoryTimeLimit)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Определяет длительность периода превышения временно допустимого объема памяти процессами кластера, после которого рабочие процессы перезапускаются.  
Параметр будет действовать, если его значение отлично от 0, и значение параметра [ВременноДопустимыйОбъемПамятиПроцессов](АдминистрированиеРабочийСервер.ВременноДопустимыйОбъемПамятиПроцессов_(AdministrationWorkServer.TemporaryAllowedProcessesTotalMemory).md) отлично от -1.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
