Расширение табличного поля списка документов (Document list table box extension)

ВосстанавливатьТекущуюСтроку (RestoreCurrentRow)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак восстановления текущей позиции в табличном поле при открытии формы, которая его содержит.

Доступность:

Толстый клиент.

Примечание:

Истина - текущая позиция восстанавливается.

Использование в версии:

Доступен, начиная с версии 8.0.
