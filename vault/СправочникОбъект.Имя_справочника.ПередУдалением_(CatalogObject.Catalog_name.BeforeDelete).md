# СправочникОбъект.<Имя справочника>.ПередУдалением

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [События](СправочникОбъект.Имя_справочника__События.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПередУдалением (BeforeDelete)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУдалением(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), удаление выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед непосредственным удалением объекта из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
