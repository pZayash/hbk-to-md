КолонкиМоделиПрогноза (PredictionModelColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>, <Тип>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомой колонки.

<Тип> (обязательный)

Тип: [ТипКолонкиМоделиПрогноза](ТипКолонкиМоделиПрогноза_(PredictionModelColumnType).md).   
Тип искомой колонки.

Возвращаемое значение:

Тип: [КолонкаМоделиПрогноза](КолонкаМоделиПрогноза_(PredictionModelColumn).md), [Неопределено](lang__def_Undefined.md).   
Если колонка не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск колонки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
