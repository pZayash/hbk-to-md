# КоллекцияДоступныхПолейКомпоновкиДанных

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных.md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля.  
Возможно обращение к доступному полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияДоступныхПолейКомпоновкиДанных.Индекс.md)  
[Количество (Count)](КоллекцияДоступныхПолейКомпоновкиДанных.Количество.md)  
[Найти (Find)](КоллекцияДоступныхПолейКомпоновкиДанных.Найти.md)  
[Получить (Get)](КоллекцияДоступныхПолейКомпоновкиДанных.Получить.md)  

Описание:

Коллекция доступных полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
