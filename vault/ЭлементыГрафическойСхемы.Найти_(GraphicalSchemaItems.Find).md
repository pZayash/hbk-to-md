ЭлементыГрафическойСхемы (GraphicalSchemaItems)

Найти (Find)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По точке бизнес-процесса

Синтаксис:

Найти(<ТочкаМаршрутаБизнесПроцесса>)

Параметры:

<ТочкаМаршрутаБизнесПроцесса> (необязательный)

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, для которой ищется элемент графической схемы.

Вариант синтаксиса: По имени

Синтаксис:

Найти(<ИмяЭлементаГрафическойСхемы>)

Параметры:

<ИмяЭлементаГрафическойСхемы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента графической схемы, которую требуется найти.

Возвращаемое значение:

Тип: [ЭлементГрафическойСхемыСоединительнаяЛиния](ЭлементГрафическойСхемыСоединительнаяЛиния_(GraphicalSchemaItemConnectionLine).md), [ЭлементГрафическойСхемыДекоративнаяЛиния](ЭлементГрафическойСхемыДекоративнаяЛиния_(GraphicalSchemaItemDecorativeLine).md), [ЭлементГрафическойСхемыДекорация](ЭлементГрафическойСхемыДекорация_(GraphicalSchemaItemDecoration).md), [ЭлементГрафическойСхемыДействие](ЭлементГрафическойСхемыДействие_(GraphicalSchemaItemActivity).md), [ЭлементГрафическойСхемыРазделение](ЭлементГрафическойСхемыРазделение_(GraphicalSchemaItemSplit).md), [ЭлементГрафическойСхемыУсловие](ЭлементГрафическойСхемыУсловие_(GraphicalSchemaItemCondition).md), [ЭлементГрафическойСхемыЗавершение](ЭлементГрафическойСхемыЗавершение_(GraphicalSchemaItemCompletion).md), [ЭлементГрафическойСхемыСтарт](ЭлементГрафическойСхемыСтарт_(GraphicalSchemaItemStart).md), [ЭлементГрафическойСхемыСлияние](ЭлементГрафическойСхемыСлияние_(GraphicalSchemaItemJoin).md), [ЭлементГрафическойСхемыВложенныйБизнесПроцесс](ЭлементГрафическойСхемыВложенныйБизнесПроцесс_(GraphicalSchemaItemSubBusinessProcess).md), [ЭлементГрафическойСхемыОбработка](ЭлементГрафическойСхемыОбработка_(GraphicalSchemaItemProcessing).md), [ЭлементГрафическойСхемыВыборВарианта](ЭлементГрафическойСхемыВыборВарианта_(GraphicalSchemaItemSwitch).md), [Неопределено](lang__def_Undefined.md).   
Элемент графической схемы. Если элемент не найден, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента графической схемы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
