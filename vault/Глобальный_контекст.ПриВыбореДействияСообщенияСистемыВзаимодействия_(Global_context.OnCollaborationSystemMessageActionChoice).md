Глобальный контекст (Global context)

ПриВыбореДействияСообщенияСистемыВзаимодействия (OnCollaborationSystemMessageActionChoice)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия](Глобальный_контекст.ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия_(Global_context.OnCollaborationSystemMessageButtonPanelButtonClick).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПриВыбореДействияСообщенияСистемыВзаимодействия(<Сообщение>, <Действие>)

Параметры:

<Сообщение>

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).   
Сообщение, в котором нажата гиперссылка действия. В обработчик передается копия объекта.

<Действие>

Тип: Произвольный.   
Действие, заданное для нажатой гиперссылки.

Описание:

Вызывается при нажатии на гиперссылку в сообщении системы взаимодействия, заданную в свойстве [Действия](СообщениеСистемыВзаимодействия.Действия_(CollaborationSystemMessage.Actions).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
