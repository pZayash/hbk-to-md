# КоллекцияАтрибутовHTML

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[АтрибутHTML](АтрибутHTML_(HTMLAttribute).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются атрибуты HTML.  
Возможно обращение к атрибуту HTML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияАтрибутовHTML.Количество_(HTMLAttributeMap.Count).md)  
[ПолучитьИменованныйЭлемент (GetNamedItem)](КоллекцияАтрибутовHTML.ПолучитьИменованныйЭлемент_(HTMLAttributeMap.GetNamedItem).md)  
[УдалитьИменованныйЭлемент (RemoveNamedItem)](КоллекцияАтрибутовHTML.УдалитьИменованныйЭлемент_(HTMLAttributeMap.RemoveNamedItem).md)  
[УстановитьИменованныйЭлемент (SetNamedItem)](КоллекцияАтрибутовHTML.УстановитьИменованныйЭлемент_(HTMLAttributeMap.SetNamedItem).md)  
[Элемент (Item)](КоллекцияАтрибутовHTML.Элемент_(HTMLAttributeMap.Item).md)  

Описание:

Коллекция узлов атрибутов узла HTML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
