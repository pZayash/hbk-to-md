Конец периода (необязательный)

Тип параметра: [Дата](lang__def_Date.md), [МоментВремени](МоментВремени_(PointInTime).md), [Граница](Граница_(Boundary).md).   
Указывается конец периода расчета оборотов. Значение этого параметра по умолчанию включается в период расчета оборотов. Для исключения значения из периода его следует задавать объектом [Граница](Граница_(Boundary).md) с исключением граничного значения.  
Если параметр не задан, обороты рассчитываются по самую последнюю запись.
