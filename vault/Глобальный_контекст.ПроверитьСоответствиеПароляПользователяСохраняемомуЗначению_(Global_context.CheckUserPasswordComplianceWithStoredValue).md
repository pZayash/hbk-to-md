Глобальный контекст (Global context)

ПроверитьСоответствиеПароляПользователяСохраняемомуЗначению (CheckUserPasswordComplianceWithStoredValue)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПроверитьСоответствиеПароляПользователяСохраняемомуЗначению(<Пароль>, <ЗначениеДляПроверки>)

Параметры:

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя информационной базы для проверки.

<ЗначениеДляПроверки> (обязательный)

Тип: [Строка](lang__def_String.md), [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).   
Хранимое значение пароля для проверки соответствия.

- Если [Строка](lang__def_String.md), то она может быть получена из значения свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md) или с помощью метода [ВычислитьСохраняемоеЗначениеПароляПользователя](Глобальный_контекст.ВычислитьСохраняемоеЗначениеПароляПользователя_(Global_context.EvaluateStoredUserPasswordValue).md).
- Если указан [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md), то используется значение его свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если пароль соответствует сохраняемому значению пароля, переданному в параметре <ЗначениеДляПроверки>, или содержащемуся в объекте [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md), переданному в параметре <ЗначениеДляПроверки>, с учетом установленного в информационной базе режима проверки сложности паролей пользователей, получаемого с помощью метода [ПолучитьПроверкуСложностиПаролейПользователей](Глобальный_контекст.ПолучитьПроверкуСложностиПаролейПользователей_(Global_context.GetUserPasswordStrengthCheck).md).

Описание:

Проверяет, соответствует ли заданный пароль сохраняемому значению пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
