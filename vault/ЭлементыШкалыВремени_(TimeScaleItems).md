**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Шкала времени](Шкала_времени.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЭлементШкалыВремени](ЭлементШкалыВремени_(TimeScaleItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы шкалы времени.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).
