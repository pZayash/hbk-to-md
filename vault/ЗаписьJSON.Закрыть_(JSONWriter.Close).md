# ЗаписьJSON.Закрыть

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [JSON](JSON-2.md) › [ЗаписьJSON](ЗаписьJSON_(JSONWriter).md) › [Методы](ЗаписьJSON__Методы.md)

ЗаписьJSON (JSONWriter)

Закрыть (Close)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Закрыть()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Завершает запись текста JSON. Если производилась запись в файл, то файл закрывается. Если производилась запись в строку, то результирующая строка будет получена в качестве возвращаемого значения метода. Если производилась запись в файл, то метод вернет пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; ЗаписьJSON.УстановитьСтроку(); // Запись документа JSON  ... СтрJSON = ЗаписьJSON.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
