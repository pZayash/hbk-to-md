Глобальный контекст (Global context)

ПолучитьОкна (GetWindows)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОкна()

Возвращаемое значение:

Тип: [ОкнаКлиентскогоПриложения](ОкнаКлиентскогоПриложения.md).   

Описание:

Получает окна приложения. Порядок элементов в коллекции не определен.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение.md) возвращает Неопределено.

См. также:

[ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
