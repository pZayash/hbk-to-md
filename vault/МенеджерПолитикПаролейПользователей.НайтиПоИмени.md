МенеджерПолитикПаролейПользователей (UserPasswordPolicyManager)

НайтиПоИмени (FindByName)

Доступен, начиная с версии 8.3.17.

Синтаксис:

НайтиПоИмени(<ИмяПолитики>)

Параметры:

<ИмяПолитики> (необязательный)

Тип: Строка.   
Наименование политики паролей пользователей.

Возвращаемое значение:

Тип: [ПолитикаПаролейПользователей](ПолитикаПаролейПользователей.md).   

Описание:

Возвращает политику паролей пользователей по имени. Если политики с таким именем нет, возвращается Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
