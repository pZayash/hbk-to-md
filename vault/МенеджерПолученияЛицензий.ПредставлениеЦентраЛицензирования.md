МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПредставлениеЦентраЛицензирования (LicensingCenterPresentation)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПредставлениеЦентраЛицензирования(<СтранаЦентраЛицензирования>)

Параметры:

<СтранаЦентраЛицензирования> (необязательный)

Тип: Строка.   
Код локализации, на основании которого определяется центр лицензирования, используемый для получения лицензии:

- ru, ru\_RU – Российская Федерация;
- uk, uk\_UA, ru\_UA – Украина;
- kk, kk\_KZ – Казахстан;
- пустая строка – центр лицензирования определяется кодом локализации системы;
- другой текст – все остальные страны.

  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: Строка.   

Описание:

Позволяет получить читаемое представление на языке интерфейса центра лицензирования, используемого для получения лицензии.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
