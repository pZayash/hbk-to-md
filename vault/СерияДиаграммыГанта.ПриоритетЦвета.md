СерияДиаграммыГанта (GanttChartSeries)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Вместе со свойством [ПриоритетЦвета](ТочкаДиаграммыГанта.ПриоритетЦвета.md) соответствующей точки определяет цвет значения. При равных значениях свойства используется цвет серии.  
Истина - будет использоваться значение [Цвет](СерияДиаграммыГанта.Цвет.md) серии.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
