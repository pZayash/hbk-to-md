КолонкаТабличногоПоля (TableBoxColumn)

ОтображатьВШапке (ShowInHeader)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак отображения шапки колонки в шапке табличного поля.  
Истина - отображается.

Доступность:

Толстый клиент.

Примечание:

Шапку колонки в шапке табличного поля можно не выводить, если колонки расположены одна под другой.

Использование в версии:

Доступен, начиная с версии 8.0.
