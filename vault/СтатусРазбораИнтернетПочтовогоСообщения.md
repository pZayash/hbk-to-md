Описание:

Содержит виды статусов разбора почтового сообщения:

- [ОбнаруженыОшибки](СтатусРазбораИнтернетПочтовогоСообщения.ОбнаруженыОшибки.md) - ошибки в структуре письма при разборе были обнаружены ошибки. Объект [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение.md) может содержать ошибки.
- [ОшибокНеОбнаружено](СтатусРазбораИнтернетПочтовогоСообщения.ОшибокНеОбнаружено.md) - при разборе письма ошибки обнаружены не были.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
