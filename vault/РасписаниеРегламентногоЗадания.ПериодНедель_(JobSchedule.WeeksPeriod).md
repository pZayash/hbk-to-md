РасписаниеРегламентногоЗадания (JobSchedule)

ПериодНедель (WeeksPeriod)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Период времени в неделях, через который нужно повторять запуск регламентного задания.

- 1, задание может быть запущено каждую неделю;
- 2 - через неделю;
- и т.д.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
