МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ПолучитьКоличествоЗаданийИндексирования (GetIndexingJobsCount)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьКоличествоЗаданийИндексирования()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает максимальное количество фоновых заданий, которое может быть запущено одновременно для обновления индекса. Значение параметра может быть установлено при помощи метода [УстановитьКоличествоЗаданийИндексирования](МенеджерПолнотекстовогоПоиска.УстановитьКоличествоЗаданийИндексирования.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме совместимости конфигурации [Версия8\_3\_11](РежимСовместимости.Версия8_3_11.md) и ниже, если количество заданий равно 0, фоновые задания не запускаются и индексация выполняется в одном потоке.

Использование в версии:

Доступен, начиная с версии 8.3.12.
