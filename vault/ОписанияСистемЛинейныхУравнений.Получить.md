ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Получить (Get)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].   
Индексация начинается с 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
