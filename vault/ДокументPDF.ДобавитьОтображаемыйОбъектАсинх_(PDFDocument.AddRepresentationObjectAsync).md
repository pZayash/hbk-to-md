# ДокументPDF.ДобавитьОтображаемыйОбъектАсинх

ДокументPDF (PDFDocument)

ДобавитьОтображаемыйОбъектАсинх (AddRepresentationObjectAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ДобавитьОтображаемыйОбъектАсинх(<ОписаниеОтображаемогоОбъекта>, <УчитыватьОриентациюСтраницы>)

Параметры:

<ОписаниеОтображаемогоОбъекта> (обязательный)

Тип: [ОписаниеОтображаемогоОбъектаPDF](ОписаниеОтображаемогоОбъектаPDF_(PDFRepresentationObjectDescription).md).   
Объект, содержащий описание отображения объекта в PDF-документе.

<УчитыватьОриентациюСтраницы> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость учитывать ориентацию страницы при добавлении отображаемого объекта. Если содержит [Неопределено](lang__def_Undefined.md) (или не указан), то в режиме совместимости конфигурации выше [Версия8\_3\_25](РежимСовместимости.Версия8_3_25_(CompatibilityMode.Version8_3_25).md) ориентация страницы учитывается, иначе ориентация страницы не учитывается.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Располагает отображение объекта (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Для работы метода в веб-клиенте требуется наличие расширения работы с файлами.

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.24.
