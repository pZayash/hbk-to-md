# ВытесняющиеВидыРасчета.<Имя плана видов расчета>.Выгрузить

ВытесняющиеВидыРасчета.<Имя плана видов расчета> (DisplacingCalculationTypes.<Chart of calculation types name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.0.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   

Описание:

Создает таблицу значений и копирует в нее табличную часть. Имена колонок таблицы значений совпадают с именами колонок табличного поля. Колонка, содержащая номер строки, выгружается в колонку с именем [НомерСтроки](Строка_табличной_части.НомерСтроки_(Line_of_a_tabular_section.LineNumber).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
