Расширение тестируемого поля формы для поля полосы регулирования (Tested form field extension for a track bar field)

ПерейтиКЗначению (GotoValue)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПерейтиКЗначению(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Число.   
Значение в процентах, на которое необходимо установить "движок" полосы регулирования.

Описание:

Устанавливает значение полосы регулирование на указанное в процентах.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
