# ОбъектыСлояГеографическойСхемы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Добавить (Add)](ОбъектыСлояГеографическойСхемы.Добавить_(GeographicalSchemaLayerObjects.Add).md)  
[Количество (Count)](ОбъектыСлояГеографическойСхемы.Количество_(GeographicalSchemaLayerObjects.Count).md)  
[Очистить (Clear)](ОбъектыСлояГеографическойСхемы.Очистить_(GeographicalSchemaLayerObjects.Clear).md)  
[Получить (Get)](ОбъектыСлояГеографическойСхемы.Получить_(GeographicalSchemaLayerObjects.Get).md)  
[Удалить (Delete)](ОбъектыСлояГеографическойСхемы.Удалить_(GeographicalSchemaLayerObjects.Delete).md)  

Описание:

Предоставляет доступ к коллекции объектов, входящих в слой географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md), свойство [Объекты](СлойГеографическойСхемы.Объекты_(GeographicalSchemaLayer.Objects).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
