# Глобальный контекст.СтрСоединить

Глобальный контекст (Global context)

СтрСоединить (StrConcat)

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Синтаксис:

СтрСоединить(<Строки>, <Разделитель>)

Параметры:

<Строки> (обязательный)

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md), [Массив](Массив_(Array).md).   
Массив, содержащий объединяемые строки.

<Разделитель> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка, которая будет вставлена между объединяемыми строками.  
Если параметр не задан, строки будут объединены слитно друг с другом.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Строка, содержащая соединенные исходные строки с разделителем между ними.

Описание:

Соединяет массив переданных строк в одну строку с указанным разделителем.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [СтрРазделить](Глобальный_контекст.СтрРазделить_(Global_context.StrSplit).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).
