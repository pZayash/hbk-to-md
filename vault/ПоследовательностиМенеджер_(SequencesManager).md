# ПоследовательностиМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Последовательности](Последовательности.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПоследовательностьМенеджер.\<Имя последовательности\>](ПоследовательностьМенеджер.Имя_последовательности_(SequenceManager.Sequence_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры последовательностей.  
Возможно обращение к менеджеру последовательности посредством оператора [...]. В качестве аргумента передается имя последовательности, как оно задано в конфигурации.  

Свойства:

[\<Имя последовательности\> (\<Sequence name\>)](ПоследовательностиМенеджер.Имя_последовательности_(SequencesManager.Sequence_name).md)  

Методы:

[Восстановить (Restore)](ПоследовательностиМенеджер.Восстановить_(SequencesManager.Restore).md)  

Описание:

Предназначен для управления последовательностями и предоставляет доступ к значениям типа [ПоследовательностьМенеджер.](ПоследовательностьМенеджер.Имя_последовательности_(SequenceManager.Sequence_name).md)<[Имя последовательности](ПоследовательностьМенеджер.Имя_последовательности_(SequenceManager.Sequence_name).md)>. Объект также позволяет восстанавливать сразу несколько последовательностей. Доступ к объекту осуществляется через свойство глобального контекста [Последовательности](Глобальный_контекст.Последовательности_(Global_context.Sequences).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Последовательности](Глобальный_контекст.Последовательности_(Global_context.Sequences).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
