ГруппировкаДиаграммыКомпоновкиДанных (DataCompositionChartGroup)

Отбор (Filter)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md).   
Содержит отбор, применяемый к записям, попадающим в группировку.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
