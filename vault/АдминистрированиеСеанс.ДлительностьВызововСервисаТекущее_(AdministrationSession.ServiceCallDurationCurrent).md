АдминистрированиеСеанс (AdministrationSession)

ДлительностьВызововСервисаТекущее (ServiceCallDurationCurrent)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Время, в течение которого сеанс выполняет текущий вызов сервиса кластера, в миллисекундах.   
0, если в данный момент вызов сервиса кластера не выполняется.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
