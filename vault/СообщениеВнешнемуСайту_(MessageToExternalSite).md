Описание:

Предназначен для хранения параметров сообщения внешнему сайту в случае, когда в него интегрирован веб-клиент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[МенеджерОкнаВнешнегоСайта](МенеджерОкнаВнешнегоСайта_(ExternalSiteWindowManager).md), метод [ОтправитьСообщение](МенеджерОкнаВнешнегоСайта.ОтправитьСообщение_(ExternalSiteWindowManager.PostMessage).md)

Доступен, начиная с версии 8.3.16.
