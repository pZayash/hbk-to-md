# ОписаниеОтображаемогоОбъектаPDF.Высота

ОписаниеОтображаемогоОбъектаPDF (PDFRepresentationObjectDescription)

Высота (Height)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту объекта (в миллиметрах) на странице PDF-документа.   
Если не указано, то высота будет определена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Используется, если значение свойства [ВысотаВМесяцах](Расширение_поля_формы_для_поля_календаря.ВысотаВМесяцах_(Form_field_extension_for_a_calendar_field.HeightInMonths).md)=0 или для режимов совместимости до [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md).

Использование в версии:

Доступен, начиная с версии 8.3.18.
