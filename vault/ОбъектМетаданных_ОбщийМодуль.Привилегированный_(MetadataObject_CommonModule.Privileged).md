ОбъектМетаданных: ОбщийМодуль (MetadataObject: CommonModule)

Привилегированный (Privileged)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то выполнение процедур общего модуля будет выполняться на сервере без контроля прав доступа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
