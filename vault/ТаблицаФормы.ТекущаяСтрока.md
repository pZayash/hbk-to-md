ТаблицаФормы (FormTable)

ТекущаяСтрока (CurrentRow)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Идентификатор текущей строки таблицы.  
В режиме совместимости выше [Версия8\_3\_22](РежимСовместимости.Версия8_3_22.md), для поля диаграммы Ганта при изменении значения данного свойства меняется значение свойства [ТекущееЗначение](Расширение_поля_формы_для_поля_диаграммы_Ганта.ТекущееЗначение.md) расширения поля формы для поля диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если устанавливаемое значение не является допустимым (такое значение отсутствует) для данной таблицы, то такая установка игнорируется.  
Тип определяется расширением таблицы.
