МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВыполнитьРегистрациюИнформационнойБазыАсинх (RegisterInfoBaseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВыполнитьРегистрациюИнформационнойБазыАсинх(<ПараметрыРегистрации>)

Параметры:

<ПараметрыРегистрации> (обязательный)

Тип: [ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия](ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия_(CollaborationSystemInfoBaseRegistrationParameters).md).   
Параметры регистрации базы в системе взаимодействия.  
Параметры [АдресСервера](ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия.АдресСервера_(CollaborationSystemInfoBaseRegistrationParameters.ServerAddress).md) и [АдресЭлектроннойПочты](ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия.АдресЭлектроннойПочты_(CollaborationSystemInfoBaseRegistrationParameters.Email).md) являются обязательными.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [РезультатРегистрацииИнформационнойБазыСистемыВзаимодействия](РезультатРегистрацииИнформационнойБазыСистемыВзаимодействия_(CollaborationSystemInfoBaseRegistrationResult).md).

Описание:

Асинхронная функция. Выполняет регистрацию информационной базы в системе взаимодействия.  
Требуется наличие у текущего пользователя права "РегистрацияИнформационнойБазыСистемыВзаимодействия".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Примечание:

Если в параметрах регистрации не указан [АдресЭлектроннойПочты](ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия.АдресЭлектроннойПочты_(CollaborationSystemInfoBaseRegistrationParameters.Email).md) и в информационной базе нет объекта [ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия](ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия_(CollaborationSystemInfoBaseRegistrationData).md), при вызове данного метода будет сгенерировано исключение.
