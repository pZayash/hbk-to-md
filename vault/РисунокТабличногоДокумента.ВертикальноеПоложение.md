Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложение](ВертикальноеПоложение.md).   
Определяет способ вертикального выравнивания текста в рисунке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), [Примечание](ТипРисункаТабличногоДокумента.Примечание.md).
