# ТочкаДиаграммы.ПриоритетЦвета

ТочкаДиаграммы (ChartPoint)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Вместе со свойством [ПриоритетЦвета](СерияДиаграммы.ПриоритетЦвета_(ChartSeries.ColorPriority).md) соответствующей серии определяет цвет значения. При равных значениях свойства используется [Цвет](СерияДиаграммы.Цвет_(ChartSeries.Color).md) серии.  
[Истина](lang__def_BooleanTrue.md) - будет использоваться значение [Цвет](ТочкаДиаграммы.Цвет_(ChartPoint.Color).md) точки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для следующих типов диаграмм:

- [Гистограмма](ТипДиаграммы.Гистограмма_(ChartType.Column).md);
- [ГистограммаОбъемная](ТипДиаграммы.ГистограммаОбъемная_(ChartType.Column3D).md);
- [ГистограммаГоризонтальная](ТипДиаграммы.ГистограммаГоризонтальная_(ChartType.Bar).md);
- [ГистограммаГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаГоризонтальнаяОбъемная_(ChartType.Bar3D).md);
- [Изометрическая](ТипДиаграммы.Изометрическая_(ChartType.BarGraph).md);
- [ИзометрическаяПирамида](ТипДиаграммы.ИзометрическаяПирамида_(ChartType.PyramidGraph).md).

Использование в версии:

Доступен, начиная с версии 8.0.
