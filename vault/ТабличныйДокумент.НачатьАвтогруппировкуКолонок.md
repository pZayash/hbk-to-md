ТабличныйДокумент (SpreadsheetDocument)

НачатьАвтогруппировкуКолонок (StartColumnAutoGrouping)

Доступен, начиная с версии 8.0.

Синтаксис:

НачатьАвтогруппировкуКолонок()

Описание:

Начинает автоматическую группировку колонок. При автоматической группировке учитываются значения уровней, заданные при выводе колонок методом [Присоединить](ТабличныйДокумент.Присоединить.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), метод [ЗакончитьАвтогруппировкуКолонок](ТабличныйДокумент.ЗакончитьАвтогруппировкуКолонок.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
