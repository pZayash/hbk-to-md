РазмерКартинки (PictureSize)

РеальныйРазмер (RealSize)

Доступен, начиная с версии 8.0.

Описание:

Сохранять реальные размеры картинки. Если область, в которой отображается картинка, меньше ее размеров, из середины картинки вырезается прямоугольник нужного размера.

Использование в версии:

Доступен, начиная с версии 8.0.
