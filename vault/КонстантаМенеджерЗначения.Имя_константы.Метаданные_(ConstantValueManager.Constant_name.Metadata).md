# КонстантаМенеджерЗначения.<Имя константы>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md) › [КонстантаМенеджерЗначения.<Имя константы>](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md) › [Методы](КонстантаМенеджерЗначения.Имя_константы__Методы.md)

КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Константа](ОбъектМетаданных_Константа_(MetadataObject_Constant).md).   

Описание:

Предоставляет доступ к объекту описания метаданных константы. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.Константы.Валюта.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
