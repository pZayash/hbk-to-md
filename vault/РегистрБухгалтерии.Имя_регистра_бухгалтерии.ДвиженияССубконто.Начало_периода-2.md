# Начало периода

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Таблицы.md">Таблицы</a> › <a href="obsidian://open?file=РегистрБухгалтерии-2.md">РегистрБухгалтерии</a> › [РегистрБухгалтерии.<Имя регистра бухгалтерии>.ДвиженияССубконто](tables__catalog36__table42.md)

Начало периода (необязательный)

Тип параметра: Дата, [МоментВремени](Прикладные_объекты.МоментВремени.md), [МоментВремениСУточнениемПериода](МоментВремениСУточнениемПериода.md), [Граница](Граница.md).   
Начало периода, за который будут выбираться проводки.   
Если начало периода не совпадает с первой границей периодичности, то в результат запроса могут попасть записи с датой, совпадающей с началом периода, указанного в параметрах виртуальной таблицы.  
Например: Если задана периодичность "Год", а начало периода установлено "03.01.2018", то если в 2018 году были движения по регистру, все данные в результате запроса будут указаны с периодом "03.01.2018".  
Значение по умолчанию: Неопределено.
