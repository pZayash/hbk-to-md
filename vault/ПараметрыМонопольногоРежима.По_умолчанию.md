ПараметрыМонопольногоРежима (ExclusiveModeParameters)

По умолчанию

Доступен, начиная с версии 8.3.19.

Синтаксис:

Новый ПараметрыМонопольногоРежима(<Сообщение>, <РазрешитьЗавершатьПриНачалеСеанса>)

Параметры:

<Сообщение> (необязательный)

Тип: Неопределено, Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Текст сообщения, которое будет отображено пользователю при попытке входа в информационную базу или область данных, на которую установлен режим монопольного доступа.  
Значение по умолчанию: Неопределено.

<РазрешитьЗавершатьПриНачалеСеанса> (необязательный)

Тип: Булево.   
Текущий сеанс может быть завершен при начале нового сеанса.   
По умолчанию - Ложь.   
Значение Истина может быть установлено, если выполняемое в монопольном режиме действие может быть прервано без опасности разрушения прикладных данных информационной базы.  
Значение по умолчанию: Ложь.

Описание:

Создает объект на основании строки - текста сообщения и признкак разрешения завершать сеанс при начале сеанса.

Пример:

|  |
| --- |
| // Сообщение является строкой  ПараметрыМонопольного = Новый ПараметрыМонопольногоРежима(     "Вход в ИБ запрещен. Идёт пересчёт итогов.",     Истина);  // Сообщение является форматированной строкой.  ТФС = Тип("ФорматированнаяСтрока"); ФС = СериализаторXDTO.XMLЗначение(ТФС, Сообщение); ПараметрыМонопольного = Новый ПараметрыМонопольногоРежима(     ФС, Истина); |

Использование в версии:

Доступен, начиная с версии 8.3.19.
