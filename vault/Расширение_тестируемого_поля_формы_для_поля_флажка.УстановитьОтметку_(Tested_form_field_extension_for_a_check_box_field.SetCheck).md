Расширение тестируемого поля формы для поля флажка (Tested form field extension for a check box field)

УстановитьОтметку (SetCheck)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УстановитьОтметку()

Описание:

Устанавливает циклически новое значение флажка.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
