FTPСоединение (FTPConnection)

Записать (Put)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Записать из файла

Синтаксис:

Записать(<ИмяВходногоФайла>, <АдресРесурса>)

Параметры:

<ИмяВходногоФайла> (обязательный)

Тип: Строка.   
Имя файла, из которого будут прочитаны данные для записи на FTP-сервере.

<АдресРесурса> (обязательный)

Тип: Строка.   
Относительный адрес ресурса на FTP-сервере, в который будут записаны данные.

Описание варианта метода:

Записывает данные из файла в ресурс по заданному адресу.

Вариант синтаксиса: Записать из потока

Синтаксис:

Записать(<АдресРесурса>, <ВходнойПоток>)

Параметры:

<АдресРесурса> (обязательный)

Тип: Строка.   
Относительный адрес ресурса на FTP-сервере, в который будут записаны данные.

<ВходнойПоток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, из которого будут прочитаны данные для записи в ресурс.

Описание варианта метода:

Записывает данные из потока в ресурс по указанному адресу.

Описание:

Записывает данные в заданный ресурс на FTP-сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
