ФасетМинимальногоВключающегоЗначенияXS (XSMinInclusiveFacet)

Включающий (Inclusive)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - значение фасета включается в диапазон ограничения, в противном случае - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение свойства противоположно значению свойства [ФасетМинимальногоВключающегоЗначенияXS.Исключающий](ФасетМинимальногоВключающегоЗначенияXS.Исключающий_(XSMinInclusiveFacet.Exclusive).md).

Использование в версии:

Доступен, начиная с версии 8.1.
