КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля, которое требуется найти.

Возвращаемое значение:

Тип: [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы_(PivotChartFieldCollection).md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы_(PivotChartField).md), [Неопределено](lang__def_Undefined.md).   
Если поле отсутствует, возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск поля по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
