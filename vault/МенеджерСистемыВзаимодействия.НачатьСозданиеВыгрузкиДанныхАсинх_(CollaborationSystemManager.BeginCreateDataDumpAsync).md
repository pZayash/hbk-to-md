МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьСозданиеВыгрузкиДанныхАсинх (BeginCreateDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

НачатьСозданиеВыгрузкиДанныхАсинх(<Описание>)

Параметры:

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).  
Обещание выполняется при начале процесса создания резервной копии.

Описание:

Асинхронная функция. Начинает создание выгрузки данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [НачатьСозданиеВыгрузкиДанных](МенеджерСистемыВзаимодействия.НачатьСозданиеВыгрузкиДанных_(CollaborationSystemManager.BeginCreateDataDump).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
