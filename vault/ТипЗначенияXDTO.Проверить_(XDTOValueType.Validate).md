ТипЗначенияXDTO (XDTOValueType)

Проверить (Validate)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Проверка лексического представления значения

Синтаксис:

Проверить(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Лексическое представление значения.

Вариант синтаксиса: Проверка значения

Синтаксис:

Проверить(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Проверяемое значение.

Описание:

Осуществляет проверку строки или значения на соответствие типу значения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
