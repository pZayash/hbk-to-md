СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника.md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор.md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](СправочникСсылка.Имя_справочника.УникальныйИдентификатор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

См. также:

[СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника.md), метод [УникальныйИдентификатор](СправочникСсылка.Имя_справочника.УникальныйИдентификатор.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
