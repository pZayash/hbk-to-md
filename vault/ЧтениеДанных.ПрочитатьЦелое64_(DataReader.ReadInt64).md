# ЧтениеДанных.ПрочитатьЦелое64

ЧтениеДанных (DataReader)

ПрочитатьЦелое64 (ReadInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьЦелое64(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Устанавливает порядок байтов, используя который число будет прочитано. Если порядок байтов не задан, то используется порядок байтов, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
[Неопределено](lang__def_Undefined.md) - если не удалось прочитать число, поскольку был достигнут конец потока.

Описание:

Считывает 64-битное целое число из потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
