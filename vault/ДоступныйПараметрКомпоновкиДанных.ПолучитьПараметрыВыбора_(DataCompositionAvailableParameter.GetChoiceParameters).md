ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

ПолучитьПараметрыВыбора (GetChoiceParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПараметрыВыбора()

Возвращаемое значение:

Тип: [ПараметрыВыбораКомпоновкиДанных](ПараметрыВыбораКомпоновкиДанных_(DataCompositionChoiceParameters).md).   

Описание:

Получает параметры выбора, которые используются для выбора значения поля или параметра.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
