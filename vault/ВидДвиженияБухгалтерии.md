Значения

[Дебет (Debit)](ВидДвиженияБухгалтерии.Дебет.md)  
[Кредит (Credit)](ВидДвиженияБухгалтерии.Кредит.md)  

Описание:

Предназначен для определения вида движения записи регистра бухгалтерии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise}. Имя типа XDTO: AccountingRecordType.
