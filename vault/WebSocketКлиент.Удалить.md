WebSocketКлиент (WebSocketClient)

Удалить (Delete)

Доступен, начиная с версии 8.3.27.

Синтаксис:

Удалить()

Описание:

Удаляет данные о WebSocket-клиенте из информационной базы. Все параметры сбрасываются к значениям, указанным в метаданных.  
При попытке удалить предопределенного клиента будет сгенерировано исключение.  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
