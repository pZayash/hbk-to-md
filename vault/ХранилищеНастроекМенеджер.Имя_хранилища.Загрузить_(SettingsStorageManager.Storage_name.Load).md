# ХранилищеНастроекМенеджер.<Имя хранилища>.Загрузить

ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

Загрузить (Load)

Доступен, начиная с версии 8.2.

Синтаксис:

Загрузить(<КлючОбъекта>, <КлючНастроек>, <ОписаниеНастроек>, <ИмяПользователя>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: Произвольный.   
Ключ объекта настройки.

<КлючНастроек> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ загружаемых настроек.

<ОписаниеНастроек> (необязательный)

Тип: [ОписаниеНастроек](ОписаниеНастроек_(SettingsDescription).md).   
Описание настроек.  
Если не указан, то описание настройки не загружается.

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, настройки которого загружаются.  
Если не указан, то загружаются настройки текущего пользователя.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Загружает настройку из хранилища.  
Результатом работы являются настройки, загруженные из хранилища.  
Данный метод вызывает обработчик хранилища настроек [ОбработкаЗагрузки](ХранилищеНастроекМенеджер.Имя_хранилища.ОбработкаЗагрузки_(SettingsStorageManager.Storage_name.LoadProcessing).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
