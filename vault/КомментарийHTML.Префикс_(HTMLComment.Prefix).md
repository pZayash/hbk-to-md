КомментарийHTML (HTMLComment)

Префикс (Prefix)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Префикс узла HTML.   
Свойство имеет значение [Неопределено](lang__def_Undefined.md) для узлов, отличных от [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md) и [ЭлементHTML](ЭлементHTML_(HTMLElement).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
