МенеджерДополнительныхНастроекАутентификации (AdditionalAuthenticationSettingsManager)

ПолучитьНастройкиАвтоматическогоСохраненияАутентификации (GetAuthenticationAutoSaveSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьНастройкиАвтоматическогоСохраненияАутентификации()

Возвращаемое значение:

Тип: [НастройкиАвтоматическогоСохраненияАутентификации](НастройкиАвтоматическогоСохраненияАутентификации_(AuthenticationAutoSaveSettings).md).   

Описание:

Получает настройки автоматического сохранения параметров аутентификации.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| НастройкиАвтоматическогоСохраненияПараметровАутентификации = ДополнительныеНастройкиАутентификации.ПолучитьНастройкиАвтоматическогоСохраненияПараметровАутентификации); НастройкиАвтоматическогоСохраненияПараметровАутентификации.РазрешитьСохранение = Истина;  НастройкиАвтоматическогоСохраненияПараметровАутентификации.СохранятьПоУмолчанию = Истина; НастройкиАвтоматическогоСохраненияПараметровАутентификации = 56000; ДополнительныеНастройкиАутентификации.УстановитьНастройкиАвтоматическогоСохраненияПараметровАутентификации(НастройкиАвтоматическогоСохраненияПараметровАутентификации); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
