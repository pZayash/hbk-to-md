ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

ВыбратьИерархически (SelectHierarchically)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьИерархически(<Родитель>, <Отбор>, <Порядок>)

Параметры:

<Родитель> (необязательный)

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов.md).   
Отбор по родителю. Если параметр не задан, то отбор по родителю не производится.

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Ключ структуры описывает имя поля, а значение структуры - значение отбора по этому полю. Могут задаваться только "Код", "Наименование" и реквизиты счета, для которых в конфигураторе признак индексирования установлен в значение "Индексировать" или в значение "Индексировать с доп. упорядоч.".   
Важно! Структура может содержать только один элемент.  
Если параметр не указан, то отбор не используется.

<Порядок> (необязательный)

Тип: Строка.   
Строка с именем реквизита счета, определяющая упорядочивание элементов в выборке. Может быть указан "Код", "Наименование" или имя одного из реквизитов примитивного типа (Число, Строка, Дата, Булево), для которого установлен признак "Индексирование" в значение "Индексировать" или "Индексировать с доп. упорядоч." в метаданных. После имени реквизита через пробел может быть указано направление сортировки. Направление определяется: "Убыв" ("Desc") - упорядочивать по убыванию; "Возр" ("Asc") - упорядочивать по возрастанию. По умолчанию сортировка производится по возрастанию. Если параметр не указан, то порядок определяется основным представлением счета.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [ПланСчетовВыборка.\<Имя плана счетов\>](ПланСчетовВыборка.Имя_плана_счетов.md).   

Описание:

Формирует иерархическую выборку элементов справочника по заданным условиям. При иерархической выборке для каждого элемента сначала выбираются элементы, для которых он является родителем, а затем уже выбираются элементы следующего уровня.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
