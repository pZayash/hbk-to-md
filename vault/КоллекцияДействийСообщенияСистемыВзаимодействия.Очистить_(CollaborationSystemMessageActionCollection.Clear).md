КоллекцияДействийСообщенияСистемыВзаимодействия (CollaborationSystemMessageActionCollection)

Очистить (Clear)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Очистить](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Очистить_(CollaborationSystemMessageButtonPanelButtonRow.Clear).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Очистить()

Описание:

Удаляет все элементы коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
