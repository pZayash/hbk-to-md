Планировщик (Planner)

КратностьПериодическогоВарианта (PeriodicVariantRepetition)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Используется вместе со свойством [ЕдиницаПериодическогоВарианта](Планировщик.ЕдиницаПериодическогоВарианта_(Planner.PeriodicVariantUnit).md) для указания длительности шкалы времени до её переноса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Планировщик](Планировщик_(Planner).md), свойство [ЕдиницаПериодическогоВарианта](Планировщик.ЕдиницаПериодическогоВарианта_(Planner.PeriodicVariantUnit).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
