**Вариант синтаксиса: По выражению**

**Синтаксис:**ВызватьИсключение <Выражение>

**Параметры:**

<Выражение>

Результат вычисления выражения преобразуется к строке и данная строка используется в качестве описания исключения.

**Описание варианта:**При использовании данной формы оператора вызывается новое исключение.

**Вариант синтаксиса: По выражению и дополнительным параметрам**

**Синтаксис:**ВызватьИсключение ([Выражение], [Категория], [Код], [ДополнительнаяИнформация], [Причина])

**Параметры:**

<Выражение>

Результат вычисления выражения преобразуется к строке и данная строка используется в качестве описания исключения.

Использование оператора только с параметром [Выражение] работает аналогично вызову **ВызватьИсключение <Выражение>**.

<Категория> (необязательный)  
Тип: [КатегорияОшибки](КатегорияОшибки.md).

Устанавливает категорию ошибки. Категория будет отображена пользователю.    
Значение по умолчанию: Неопределено.  
Не допускается использовать следующие категории ошибок:

- ВсеОшибки,
- ПрочаяОшибка,
- ОшибкаКомпиляцииВстроенногоЯзыка,
- ОшибкаВоВремяВыполненияВстроенногоЯзыка,
- ИсключениеВызванноеИзВстроенногоЯзыка.

<Код>(необязательный)  
Тип: Строка.

Устанавливает код ошибки, которое может использоваться в обработчиках ошибок для классификации исключений. Подробнее см. [Код](ИнформацияОбОшибке.Код.md).  
Значение по умолчанию: Пустая строка.

<ДополнительнаяИнформация>(необязательный)  
Тип: Строка.

Дополнительная информация о формируемой ошибке. Подробнее см. ДополнительнаяИнформация.  
Значение по умолчанию: Пустая строка.

<Причина>(необязательный)  
Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md)

**Описание варианта:**

При использовании данной формы оператора вызывается новое исключение и применяются указанные параметры исключения.

**Пример:**

```
 ВызватьИсключение("Ошибка на клиенте с категорией", КатегорияОшибки.ОшибкаКонфигурации, "error.client.config", "error.client.config additional info");
```

**См. также:**  
Описание оператора [Попытка](lang__Попытка.md).

**Использование в версии:**

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
