Структура (Structure)

На основании фиксированной структуры

Доступен, начиная с версии 8.3.2.

Синтаксис:

Новый Структура(<ФиксированнаяСтруктура>)

Параметры:

<ФиксированнаяСтруктура> (необязательный)

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md).   
Фиксированная структура.

Описание:

Создаваемая структура заполняется свойствами из исходной фиксированной структуры.

Использование в версии:

Доступен, начиная с версии 8.3.2.
