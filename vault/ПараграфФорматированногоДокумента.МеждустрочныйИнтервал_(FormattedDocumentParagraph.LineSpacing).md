ПараграфФорматированногоДокумента (FormattedDocumentParagraph)

МеждустрочныйИнтервал (LineSpacing)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит междустрочный интервал между параграфами форматированного документа в относительных единицах (множитель от размера шрифта текущего текста).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
