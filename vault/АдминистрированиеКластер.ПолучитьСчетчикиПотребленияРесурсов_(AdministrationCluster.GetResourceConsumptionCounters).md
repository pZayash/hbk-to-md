АдминистрированиеКластер (AdministrationCluster)

ПолучитьСчетчикиПотребленияРесурсов (GetResourceConsumptionCounters)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСчетчикиПотребленияРесурсов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеСчетчикПотребленияРесурсов](АдминистрированиеСчетчикПотребленияРесурсов_(AdministrationResourceConsumptionCounter).md).

Описание:

Получает массив счетчиков потребления ресурсов, зарегистрированных в кластере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
