# Расширение формы списка записей регистра сведений.ПараметрТекущаяСтрока

Расширение формы списка записей регистра сведений (Information register record list form extension)

ПараметрТекущаяСтрока (CurrentLineParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РегистрСведенийКлючЗаписи.\<Имя регистра сведений\>](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md).   
Осуществляет позиционирование на строку содержащегося в форме списка, отображающую устанавливаемое значение. Позиционирование происходит при открытии или повторном открытии формы. Если форма используется для выбора или подбора, данная установка имеет более высокий приоритет, чем свойство формы [НачальноеЗначениеВыбора](Форма.НачальноеЗначениеВыбора_(Form.ChoiceInitialValue).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
