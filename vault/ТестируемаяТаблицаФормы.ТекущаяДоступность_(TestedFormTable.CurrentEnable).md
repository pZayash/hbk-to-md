# ТестируемаяТаблицаФормы.ТекущаяДоступность

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Автоматизированное тестирование](Автоматизированное_тестирование-2.md) › [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы_(TestedFormTable).md) › [Методы](ТестируемаяТаблицаФормы__Методы.md)

ТестируемаяТаблицаФормы (TestedFormTable)

ТекущаяДоступность (CurrentEnable)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущаяДоступность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект доступен; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Возвращает признак того, что пользователь может активизировать элемент без настройки формы, но путем переключения закладок и полос прокрутки.  
Если объект является колонкой таблицы и таблица не находится в режиме редактирования, при вызове метода генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
