ОбъектМетаданных: Измерение (MetadataObject: Dimension)

РежимСокращенияТипа (TypeReductionMode)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: [РежимСокращенияТипа](РежимСокращенияТипа.md).   
Определяет действия над данными независимого регистра сведений в процессе обновления конфигурации базы данных, если у измерения изменился тип на более узкий, и преобразование значения измерения к новому типу приводит к потере данных, результатом которой может быть преобразование разных значений к одному и тому же значению. По умолчанию [ПреобразовыватьЗначения](РежимСокращенияТипа.ПреобразовыватьЗначения.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.25.
