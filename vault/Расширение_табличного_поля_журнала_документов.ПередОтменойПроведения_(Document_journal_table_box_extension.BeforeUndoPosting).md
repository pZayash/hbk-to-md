# Расширение табличного поля журнала документов.ПередОтменойПроведения

Расширение табличного поля журнала документов (Document journal table box extension)

ПередОтменойПроведения (BeforeUndoPosting)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередОтменойПроведения(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отмены проведения документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то отмена проведения документа выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивной отменой проведения документа в списке.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
