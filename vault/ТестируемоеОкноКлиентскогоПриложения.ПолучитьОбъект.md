ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ТестируемоеОкноКлиентскогоПриложения](ТестируемоеОкноКлиентскогоПриложения.md), [ТестируемаяФорма](ТестируемаяФорма.md), [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы.md), [ТестируемаяКнопкаФормы](ТестируемаяКнопкаФормы.md), [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы.md), [ТестируемоеПолеФормы](ТестируемоеПолеФормы.md), [ТестируемаяДекорацияФормы](ТестируемаяДекорацияФормы.md), Неопределено.   
Первый найденный объект. Если объект не найден, генерируется исключительная ситуация.

Описание:

Выполняет поиск объекта среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
