ИдентификаторВыгрузкиДанныхСистемыВзаимодействия (CollaborationSystemDataDumpID)

Из строки

Доступен, начиная с версии 8.3.22.

Синтаксис:

Новый ИдентификаторВыгрузкиДанныхСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строковое представление идентификатора.

Описание:

Создает идентификатор выгрузки данных из указанной строки.  
Валидность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.22.
