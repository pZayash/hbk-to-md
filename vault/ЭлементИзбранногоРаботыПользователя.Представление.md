ЭлементИзбранногоРаботыПользователя (UserWorkFavoritesItem)

Представление (Presentation)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит представление элемента избранного.  
Не пустое значение используется для представления навигационной ссылки при показе избранного (в панели избранного и в меню избранного).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
