НачальноеОтображениеСписка (InitialListView)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [НачальноеОтображениеСписка](НачальноеОтображениеСписка_(InitialListView).md).   
Содержит режим начального отображения списка в таблице.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
