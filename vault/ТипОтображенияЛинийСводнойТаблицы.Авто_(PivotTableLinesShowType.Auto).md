ТипОтображенияЛинийСводнойТаблицы (PivotTableLinesShowType)

Авто (Auto)

Доступен, начиная с версии 8.0.

Описание:

Необходимость вывода линий определяется автоматически. Линии будут выводиться для группировок, в которые вложены другие группировки.

Использование в версии:

Доступен, начиная с версии 8.0.
