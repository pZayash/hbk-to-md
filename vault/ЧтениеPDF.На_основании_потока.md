На основании потока

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЧтениеPDF(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток.md).   
Поток, из которого будет производиться чтение данных.

Описание:

Открывает поток для чтения.

Использование в версии:

Доступен, начиная с версии 8.3.16.
