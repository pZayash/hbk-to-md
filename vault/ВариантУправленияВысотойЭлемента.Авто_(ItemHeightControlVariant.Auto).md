ВариантУправленияВысотойЭлемента (ItemHeightControlVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Вариант управления высотой определяется автоматически:

- Для многострочного поля ввода - [ПоСодержимому](ВариантУправленияВысотойЭлемента.ПоСодержимому_(ItemHeightControlVariant.UseContentHeight).md),
- Для однострочного поля ввода - [ВСтрокахФормы](ВариантУправленияВысотойЭлемента.ВСтрокахФормы_(ItemHeightControlVariant.UseHeightInFormRows).md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
