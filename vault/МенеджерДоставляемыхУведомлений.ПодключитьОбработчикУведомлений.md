МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ПодключитьОбработчикУведомлений (AttachNotificationHandler)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПодключитьОбработчикУведомлений(<Процедура>)

Параметры:

<Процедура> (обязательный)

Имя процедуры, подключаемой в качестве обработчика уведомления.  
Значение по умолчанию: [ОписаниеОповещения](ОписаниеОповещения.md); Строка.

Описание:

Подключает процедуру-обработчик получения уведомления.  
Процедура должна иметь следующие параметры:

- Уведомление - Объект, содержащий данные уведомления, у которого заполнены только поля "Текст" и "Данные", остальные содержат значения по умолчанию.
- Локальное - тип Булево. Истина - уведомление было создано локально операционной системой. Ложь - было получено Push-уведомление.
- Показано - тип Булево. Указывает, что уведомление было показано пользователю средствами операционной системы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
