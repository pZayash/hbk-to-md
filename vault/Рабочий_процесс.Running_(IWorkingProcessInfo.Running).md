Рабочий процесс (IWorkingProcessInfo)

Running (Running)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Состояние рабочего процесса.

Доступность:

Интеграция.

Примечание:

0 – процесс неактивен (либо не загружен в память, либо не может выполнять клиентские запросы); 1 – процесс активен (работает).

Использование в версии:

Доступен, начиная с версии 8.1.
