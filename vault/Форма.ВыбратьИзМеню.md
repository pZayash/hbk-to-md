Форма (Form)

ВыбратьИзМеню (ChooseFromMenu)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьИзМеню(<СписокЗначений>, <ЭлементФормы>)

Параметры:

<СписокЗначений> (обязательный)

Тип: [СписокЗначений](СписокЗначений.md).   
Список значений, содержащий значения, из которых будет осуществляться выбор.

<ЭлементФормы> (необязательный)

Элемент управления формы, около которого будет открыт выпадающий список. Для некоторых элементов (табличный документ, табличное поле) выпадающий список будет показан около активной области. Если не указан, то будет выбран активный элемент формы.  
Примечание: В качестве этого параметра нельзя указывать кнопки, расположенные в командной панели или подменю.

Возвращаемое значение:

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md), Неопределено.   
Выбранное значение из меню или Неопределено, если пользователь отказался от выбора.

Описание:

Вызывает меню для выбора значения. Набор значений для выбора задается списком значений. Меню будет расположено около переданного элемента формы или около его активной области.

Доступность:

Толстый клиент.

Примечание:

Если в списке значений указаны пометки или картинки, то они отображаются в меню.

Пример:

|  |
| --- |
| Список = Новый СписокЗначений; Список.Добавить("Доставка",,Истина); Список.Добавить("Разгрузка",,Ложь); Список.Добавить("Картинка",,,БиблиотекаКартинок.Картинка1); Вид = ВыбратьИзМеню(Список, ЭлементыФормы["Кнопка1"]); |

Использование в версии:

Доступен, начиная с версии 8.0.
