ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

КомментарийВерсии (VersionComment)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Комментарий версии истории данных, создаваемой при записи данных.  
Значение по умолчанию - пустая строка.  
Строка, превышающая максимально допустимую длину в 1024 символа, будет обрезана.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
