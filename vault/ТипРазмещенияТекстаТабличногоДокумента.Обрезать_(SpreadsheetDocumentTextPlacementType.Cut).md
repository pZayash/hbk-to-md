ТипРазмещенияТекстаТабличногоДокумента (SpreadsheetDocumentTextPlacementType)

Обрезать (Cut)

Доступен, начиная с версии 8.0.

Описание:

Видимая часть текста будет ограничена левой и правой границами ячейки.

Использование в версии:

Доступен, начиная с версии 8.0.
