Описание:

Объект сертификатов удостоверяющих центров OpenSSL, загружаемых из файла.  
Формат файла определяется по его расширению:  
\*.PFX; \*.P12 - файл сертификатов формата P12;  
\*.PEM - файл сертификатов формата PEM.  
по умолчанию - файл сертификатов формата PEM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL.md)
