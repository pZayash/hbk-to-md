МенеджерКриптографии (CryptoManager)

ПроверитьНаличиеДанныхВПодписи (CheckForDataInSignature)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПроверитьНаличиеДанныхВПодписи(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Электронная подпись, в которой определяется наличие данных.  
Данные подписи могут размещаться в файле (в этом случае указывается имя файла) или представлены в явном виде как [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md) или [ФайловыйПоток](ФайловыйПоток.md).

Возвращаемое значение:

Тип: Булево.   
Признак наличия данных: Истина – если заданная подпись содержит данные и Ложь – в противном случае.

Описание:

Определяет наличие данных в электронной подписи: присоединенная подпись или отсоединенная.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПроверитьНаличиеДанныхВПодписиАсинх](МенеджерКриптографии.ПроверитьНаличиеДанныхВПодписиАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.24.
