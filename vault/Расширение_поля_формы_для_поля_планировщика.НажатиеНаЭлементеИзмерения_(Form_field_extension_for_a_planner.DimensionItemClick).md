# Расширение поля формы для поля планировщика.НажатиеНаЭлементеИзмерения

Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаЭлементеИзмерения (DimensionItemClick)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НажатиеНаЭлементеИзмерения(<ЭлементИзмерения>, <ЗначенияИзмерений>, <СтандартнаяОбработка>)

Параметры:

<ЭлементИзмерения>

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md).   
Элемент измерения, на который было произведено нажатие.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Фиксированное соответствие, однозначно определяющее заголовок измерения, на котором было произведено нажатие.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения стандартной обработки.  
 Если [Истина](lang__def_BooleanTrue.md), то показывается значение элемента измерения.

Описание:

Возникает при нажатии левой кнопкой мыши на элементе измерения в случае, если значение свойства поля планировщика [ГиперссылкаЭлементаИзмерения](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаИзмерения_(Form_field_extension_for_a_planner.DimensionItemHyperlink).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
