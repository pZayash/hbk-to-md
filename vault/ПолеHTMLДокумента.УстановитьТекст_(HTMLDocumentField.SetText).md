# ПолеHTMLДокумента.УстановитьТекст

ПолеHTMLДокумента (HTMLDocumentField)

УстановитьТекст (SetText)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Устанавливаемый текст HTML-документа.

Описание:

Устанавливает текст поля HTML-документа.  
Этот метод устанавливает текст, не формируя адреса для HTML документа.  
Если для элемента управления установлен режим просмотра, то при использовании метода в данном элементе не будут работать внутренние ссылки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
