АдминистрированиеСчетчикПотребленияРесурсов (AdministrationResourceConsumptionCounter)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить()

Описание:

Отменяет регистрацию счетчика потребления ресурсов в кластере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
