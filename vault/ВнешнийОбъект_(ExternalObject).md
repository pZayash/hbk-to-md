# ВнешнийОбъект

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.2.

Описание:

Свойства, методы и события объектов данного типа являются динамическими, т.е. зависят от самого объекта.  
Для значений типа [ВнешнийОбъект](ВнешнийОбъект_(ExternalObject).md), свойства, являющиеся динамическими объектами, имеют тип [ВнешнийОбъект](ВнешнийОбъект_(ExternalObject).md).

Доступность:

Веб-клиент.

См. также:

[Расширение поля формы для поля HTML-документа](Расширение_поля_формы_для_поля_HTML-документа_(Form_extension_for_a_HTML_document_field).md), свойство [Документ](Расширение_поля_формы_для_поля_HTML-документа.Документ_(Form_extension_for_a_HTML_document_field.Document).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
