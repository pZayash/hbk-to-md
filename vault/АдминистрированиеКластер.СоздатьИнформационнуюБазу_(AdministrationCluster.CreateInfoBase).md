АдминистрированиеКластер (AdministrationCluster)

СоздатьИнформационнуюБазу (CreateInfoBase)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьИнформационнуюБазу()

Возвращаемое значение:

Тип: [АдминистрированиеИнформационнаяБаза](АдминистрированиеИнформационнаяБаза_(AdministrationInfoBase).md).   

Описание:

Создает информационную базу с пустыми значениями свойств. Требуется аутентификация администратора кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
