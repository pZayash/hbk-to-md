# Расширение формы элемента справочника.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [Расширение формы элемента справочника](Расширение_формы_элемента_справочника_(Catalog_item_form_extension).md) › [События](Расширение_формы_элемента_справочника__События.md)

Расширение формы элемента справочника (Catalog item form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](СправочникОбъект.Имя_справочника.ПриЗаписи_(CatalogObject.Catalog_name.OnWrite).md) объекта [СправочникОбъект.](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)<[Имя справочника](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
