Глобальный контекст (Global context)

ПроверитьРаскрытиеПароля (CheckPasswordCompromise)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПроверитьРаскрытиеПароля(<Пароль>)

Параметры:

<Пароль> (обязательный)

Тип: Строка.   
Пароль, раскрытие которого требуется проверить.

Возвращаемое значение:

Тип: Булево.   
Истина - если было выявлено раскрытие пароля,  
Ложь - в противном случае.

Описание:

Проверяет раскрытие пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
