СредстваМультимедиа (MultimediaTools)

СделатьАудиозапись (MakeAudioRecording)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СделатьАудиозапись(<ПараметрыАудиозаписи>, <НачатьЗаписьСразу>, <ОтображатьОкноПрослушивания>)

Параметры:

<ПараметрыАудиозаписи> (необязательный)

Набор параметров, используемых при записи аудио.

<НачатьЗаписьСразу> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Запустить запись звука автоматически.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ОтображатьОкноПрослушивания> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Отключает интерфейс предварительного прослушивания записи звука. Метод завершается при завершении записи.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - запись не выполнена.

Описание:

Выполняет аудиозапись встроенным микрофоном.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяАудиозапись](СредстваМультимедиа.ПоддерживаетсяАудиозапись_(MultimediaTools.AudioRecordingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ПоддерживаетсяАудиозапись](СредстваМультимедиа.ПоддерживаетсяАудиозапись_(MultimediaTools.AudioRecordingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.18.
