ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаЗаписиНового (NewWriteProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаЗаписиНового(<НовыйОбъект>, <Источник>, <СтандартнаяОбработка>)

Параметры:

<НовыйОбъект>

Тип: Произвольный.   
Добавленный в подчиненной форме объект.

<Источник>

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [Форма](Форма_(Form).md).   
Форма - источник события.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Стандартная обработка заключается в позиционировании на записанный элемент основной таблицы формы списка (если форма является формой списка типа записываемого объекта).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает на клиенте при записи объекта в одной из подчиненных форм или при вызове в одной из подчиненных форм метода [ОповеститьОЗаписиНового](ФормаКлиентскогоПриложения.ОповеститьОЗаписиНового_(ClientApplicationForm.NotifyWritingNew).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
