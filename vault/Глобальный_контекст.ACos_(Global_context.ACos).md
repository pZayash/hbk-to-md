Глобальный контекст (Global context)

ACos (ACos)

Доступен, начиная с версии 8.0.

Синтаксис:

ACos(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Аргумент функции. Определен в диапазоне -1 ... 1.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат вычисления арккосинуса в радианах.

Описание:

Вычисляет арккосинус от аргумента <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

Если аргумент выходит за пределы диапазона, то происходит ошибка "Неправильное значение аргумента встроенной функции (ACos)".

Пример:

|  |
| --- |
| Pi = 3.1415926535897932; Аргумент = 0.5; Попытка     УголВГрадусах = 360\*ACos(Аргумент)/Pi/2;     Сообщить("Значение угла по значению Cos(): "         + Окр(УголВГрадусах)); Исключение     Сообщить("При значении аргумента " + Аргумент         + " выдается сообщение об ошибке:"         + Символы.ПС + """" + ОписаниеОшибки() + """"); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
