ПараметрыДоступнойТаблицыСхемыЗапроса (QuerySchemaAvailableTableParameters)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя параметра для поиска.

Возвращаемое значение:

Тип: [ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса_(QuerySchemaAvailableTableParameter).md).   

Описание:

Выполняет поиск по свойству [Имя](ПараметрДоступнойТаблицыСхемыЗапроса.Имя_(QuerySchemaAvailableTableParameter.Name).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
