# Расширение формы списка записей регистра сведений.ПараметрОтборПоРегистратору

Расширение формы списка записей регистра сведений (Information register record list form extension)

ПараметрОтборПоРегистратору (FilterByRecorderParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Предоставляет возможность установить начальное значение отбора по регистратору при открытии или повторном открытии формы списка регистра сведений.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только для регистров сведений, подчиненных регистраторам.

Использование в версии:

Доступен, начиная с версии 8.0.
