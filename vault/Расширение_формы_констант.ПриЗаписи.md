Расширение формы констант (Constant form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](КонстантаМенеджерЗначения.Имя_константы.ПриЗаписи.md) объекта [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы.md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы.md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
