СтраницыПанели (PanelPages)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяСтраницы>)

Параметры:

<ИмяСтраницы> (обязательный)

Тип: Строка.   
Имя страницы.

Возвращаемое значение:

Тип: [СтраницаПанели](СтраницаПанели.md), Неопределено.   
Искомая страница в коллекции.  
Если не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск страницы по имени страницы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
