СерииДиаграммы (ChartSeriesCollection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс серии.

Возвращаемое значение:

Тип: [СерияДиаграммы](СерияДиаграммы.md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
