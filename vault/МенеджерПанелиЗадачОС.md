Описание:

Используется для управления кнопкой программы в панели задач.  
Функционал работает под ОС Windows 7 и выше, Ubuntu Unity. Под неподдерживаемыми ОС не производит никаких действий.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
