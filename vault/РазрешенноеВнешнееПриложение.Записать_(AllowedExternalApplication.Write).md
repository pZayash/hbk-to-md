РазрешенноеВнешнееПриложение (AllowedExternalApplication)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового разрешенного приложения или обновление существующего.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
