РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [Число](lang__def_Number.md), [РегистрРасчетаЗапись.](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)>.   
Индекс удаляемой записи либо сама запись.

Описание:

Удаляет запись из набора по указанному индексу записи или объекту записи регистра.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
