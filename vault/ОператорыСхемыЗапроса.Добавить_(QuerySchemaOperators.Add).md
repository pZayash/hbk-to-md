ОператорыСхемыЗапроса (QuerySchemaOperators)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ОператорВыбратьСхемыЗапроса](ОператорВыбратьСхемыЗапроса_(QuerySchemaSelectOperator).md).   
Если параметр указан по умолчанию, будет выполнено добавление нового объекта, если в параметре передан оператор выбора схемы запроса (должен быть из того же запроса), он будет скопирован.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [ОператорВыбратьСхемыЗапроса](ОператорВыбратьСхемыЗапроса_(QuerySchemaSelectOperator).md).   

Описание:

Добавляет новый элемент или копирует элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
