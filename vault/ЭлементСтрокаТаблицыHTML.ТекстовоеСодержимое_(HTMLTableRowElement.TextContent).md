# ЭлементСтрокаТаблицыHTML.ТекстовоеСодержимое

ЭлементСтрокаТаблицыHTML (HTMLTableRowElement)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Текстовое содержимое узла HTML.  
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.  
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство [ЭтоПробельныеСимволыЭлемента](ТекстHTML.ЭтоПробельныеСимволыЭлемента_(HTMLText.IsElementContentWhitespace).md)).  
Принцип формирования текстового значения узла:  
[Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md) - объединение текстового содержимого каждого из дочерних узлов, исключая [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md). Пустая строка, если дочерних узлов нет;  
[Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md), [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - значение свойства ЗначениеУзла;  
[Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md) - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
