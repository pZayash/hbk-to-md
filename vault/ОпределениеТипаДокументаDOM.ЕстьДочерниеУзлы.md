ОпределениеТипаДокументаDOM (DOMDocumentType)

ЕстьДочерниеУзлы (HasChildNodes)

Доступен, начиная с версии 8.1.

Синтаксис:

ЕстьДочерниеУзлы()

Возвращаемое значение:

Тип: Булево.   
Истина - узел имеет дочерние узлы, Ложь - в противном случае.

Описание:

Проверяет, имеются ли у узла дочерние узлы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
