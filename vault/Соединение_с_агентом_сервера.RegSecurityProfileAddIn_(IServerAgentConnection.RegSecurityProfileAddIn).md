Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfileAddIn (RegSecurityProfileAddIn)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfileAddIn(<Кластер>, <ИмяПрофиля>, <ВнешнийКомпонент>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должен быть зарегистрирован внешний компонент.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<ВнешнийКомпонент> (обязательный)

Тип: [Внешний компонент](Внешний_компонент_(ISecurityProfileAddIn).md).   
Регистрируемый внешний компонент.

Описание:

Выполняет регистрацию нового разрешенного внешнего компонента или обновляет существующий.

Доступность:

Интеграция.

Примечание:

Если внешний компонент с заданным именем существует, то выполняется обновление, в противном случае - добавление.  
Если профиль не найден, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
