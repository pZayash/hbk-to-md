# ЧтениеДанных.ПрочитатьБайт

ЧтениеДанных (DataReader)

ПрочитатьБайт (ReadByte)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьБайт()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
[Неопределено](lang__def_Undefined.md), если не удалось прочитать байт поскольку был достигнут конец потока.

Описание:

Выполняет чтение одного байта из потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
