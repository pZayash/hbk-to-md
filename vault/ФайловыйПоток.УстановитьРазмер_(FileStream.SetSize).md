# ФайловыйПоток.УстановитьРазмер

ФайловыйПоток (FileStream)

УстановитьРазмер (SetSize)

Доступен, начиная с версии 8.3.9.

Синтаксис:

УстановитьРазмер(<Размер>)

Параметры:

<Размер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Устанавливаемый размер потока.

Описание:

Устанавливает размер потока.  
Если текущий размер превышает заданный, поток будет сокращен до заданного размера, а информация, превышающая заданный размер, будет потеряна.  
Если текущий размер потока меньше заданного, то содержимое потока между старым и новым размером не определено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Примечание:

Установить длину потока можно только в том случае, если в потоке разрешены запись и изменение позиции. В противном случае будет сгенерировано исключение.
