Описание:

Определяет вид периода регистра расчета - базовый период, период действия или период регистрации.  
Данное перечисление используется как параметр метода регистра расчета, предназначенного для получения данных графика, связанного с регистром расчета.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.
