# НачальноеОтображениеСписка

Доступен, начиная с версии 8.0.

Значения

[Авто (Auto)](НачальноеОтображениеСписка.Авто.md)  
[Конец (End)](НачальноеОтображениеСписка.Конец.md)  
[Начало (Beginning)](НачальноеОтображениеСписка.Начало.md)  

Описание:

Определяет набор вариантов перехода на строку связанного с табличным полем списка при открытии табличного поля.  
Используется для определения свойства [НачальноеОтображениеСписка](ТабличноеПоле.НачальноеОтображениеСписка.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется.
