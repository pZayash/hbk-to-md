Форма (Form)

ОткрытьСправкуФормы (OpenFormHelp)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьСправкуФормы()

Возвращаемое значение:

Тип: Булево.   
Истина - окно со справочной информацией по форме открыто; Ложь - в противном случае.

Описание:

Открывает окно справочной информации по форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ОткрытьСправку(); |

Использование в версии:

Доступен, начиная с версии 8.0.
