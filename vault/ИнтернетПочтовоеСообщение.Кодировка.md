ИнтернетПочтовоеСообщение (InternetMailMessage)

Кодировка (Encoding)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит кодировку всего сообщения, включая тему, отображаемые имена, тексты и наименования вложений, если для этих частей сообщения кодировка не указана. По умолчанию используется [UTF8](КодировкаТекста.UTF8.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
