[МакетГруппировкиТаблицыМакетаКомпоновкиДанных](МакетГруппировкиТаблицыМакетаКомпоновкиДанных.md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных.md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных.md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы тела группировки таблицы.  
Возможно обращение к элементу тела группировки таблицы посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных.md).
