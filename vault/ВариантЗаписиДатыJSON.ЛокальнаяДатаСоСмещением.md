ВариантЗаписиДатыJSON (JSONDateWritingVariant)

ЛокальнаяДатаСоСмещением (LocalDateWithOffset)

Доступен, начиная с версии 8.3.6.

Описание:

Локальная дата с указание смещения относительно всемирного координированного времени (UTC).  
Пример для формата ISO: 2009-02-15T10:00:00+04:00.

Использование в версии:

Доступен, начиная с версии 8.3.6.
