МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

ТекущийСпособБиометрическойПроверки (CurrentBiometricVerificationMethod)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ТекущийСпособБиометрическойПроверки()

Возвращаемое значение:

Тип: [СпособБиометрическойПроверки](СпособБиометрическойПроверки.md), Неопределено.   
Если получить информацию о наличии биометрической проверки невозможно (в Windows Runtime), возвращает Неопределено.

Описание:

Возвращает способ биометрической проверки, который поддерживает устройство.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
