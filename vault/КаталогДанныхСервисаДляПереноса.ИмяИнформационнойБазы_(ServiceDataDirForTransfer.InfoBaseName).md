КаталогДанныхСервисаДляПереноса (ServiceDataDirForTransfer)

ИмяИнформационнойБазы (InfoBaseName)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя информационной базы, для которой задан каталог данных, или пустая строка, если данные сервиса не разделены по информационным базам.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
