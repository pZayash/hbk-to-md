ПараметрыМакетаТекстовогоДокумента (TextDocumentTemplateParameters)

<Имя параметра> (<Parameter name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит параметры, используемые в макете текстового документа. Доступ к параметру осуществляется по имени. Имена свойств совпадают с именами параметров.  
Параметры служат для вывода в текстовый документ данных, содержащихся в базе данных, либо получаемых в результате расчетов. Для этого параметрам текстового документа нужно присвоить необходимые по смыслу значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
