РасписаниеРегламентногоЗадания (JobSchedule)

ТребуетсяВыполнение (ExecutionRequired)

Доступен, начиная с версии 8.1.

Синтаксис:

ТребуетсяВыполнение(<ДатаДляПроверки>, <ДатаНачалаПоследнегоЗапуска>, <ДатаЗавершенияПоследнегоЗапуска>)

Параметры:

<ДатаДляПроверки> (необязательный)

Тип: Дата.   
Дата, для которой нужно проверить расписание. Если дата не задана, используется текущая дата.

<ДатаНачалаПоследнегоЗапуска> (необязательный)

Тип: Дата.   
Дата начала последнего запуска регламентного задания. Если дата задана, то используется для проверки таких условий как [ПериодПовтораДней](РасписаниеРегламентногоЗадания.ПериодПовтораДней.md), [ПериодНедель](РасписаниеРегламентногоЗадания.ПериодНедель.md), [ПериодПовтораВТечениеДня](РасписаниеРегламентногоЗадания.ПериодПовтораВТечениеДня.md). Если дата не задана, то считается, что задание не выполнялось ни разу и проверка этих условий не производится.

<ДатаЗавершенияПоследнегоЗапуска> (необязательный)

Тип: Дата.   
Дата завершения последнего запуска регламентного задания. Если дата задана, то используется для проверки условия [ПаузаПовтора](РасписаниеРегламентногоЗадания.ПаузаПовтора.md). Если дата не задана, то считается, что задание ни разу не завершалось, и проверка этого условия не производится.

Возвращаемое значение:

Тип: Булево.   

Описание:

Проверяет, выполняется ли расписание для заданной даты.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
