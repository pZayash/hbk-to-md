МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ПолучитьДоступностьХранилищаДвоичныхДанных (GetBinaryDataStorageAvailability)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДоступностьХранилищаДвоичныхДанных()

Возвращаемое значение:

Тип: Булево.   
Истина - если хранилище доступно,  
Ложь - если хранилище недоступно (не поддерживается).

Описание:

Проверяет, поддерживается ли хранилище двоичных данных в текущем варианте платформы.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
