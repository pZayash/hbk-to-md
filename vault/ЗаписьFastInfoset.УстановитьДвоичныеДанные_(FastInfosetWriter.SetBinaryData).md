ЗаписьFastInfoset (FastInfosetWriter)

УстановитьДвоичныеДанные (SetBinaryData)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьДвоичныеДанные()

Описание:

Производит инициализацию объекта записи для записи Fast Infoset-файла в двоичные данные.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
