СвязиДендрограммы (DendrogramLinkCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество связей в дендрограмме.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для Н = 0 По Диаграмма.Точки.Количество() - 1 Цикл     // обработка точки  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
