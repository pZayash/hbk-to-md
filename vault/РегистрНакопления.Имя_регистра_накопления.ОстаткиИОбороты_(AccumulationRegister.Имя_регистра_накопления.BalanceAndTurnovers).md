Описание:

Предназначена для получения остатков и оборотов за период по регистру накопления. Данные представляют собой итоги ресурсов.   
При расчете итогов учитываются только активные записи.  
Выборка выдает периоды, за которые остатки равны нулю, но были обороты, даже, если обороты в данном запросе не запрашивались.  
Таблица существует только для регистров остатков.  
Агрегация производится по измерениям указанным в запросе.
