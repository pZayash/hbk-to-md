КолонкаОписанияИсточникаДанных (DataSourceDescriptionColumn)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Путь к данным колонки. Описывает взаимосвязь между колонками. Поля, отделенные точкой от других полей, считаются зависимыми от полей. В случае, если зависимое поле называется Представление, такое поле будет считаться представлением родительского поля.  
Например, имеем две колонки, в первой путь к данным равен Контрагент, во второй Контрагент.Представление. В таком случае вторая колонка будет считаться представлением первой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
