БиблиотекаМакетовОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateLib)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента макетов оформления компоновки данных.

Возвращаемое значение:

Тип: [ЭлементБиблиотекиМакетовОформленияКомпоновкиДанных](ЭлементБиблиотекиМакетовОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplateLibItem).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
