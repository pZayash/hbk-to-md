КоллекцияПолейXBase (XBaseFieldsCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля XBase.

Возвращаемое значение:

Тип: [ПолеXBase](ПолеXBase_(XBaseField).md), [Неопределено](lang__def_Undefined.md).   
Если указанного поля нет, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск поля по наименованию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
