Запрос (Query)

ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [ИспользуемыеКопииБазыДанных](ИспользуемыеКопииБазыДанных_(UsedDatabaseCopies).md).   
Свойство содержит список копий базы данных, на которых можно выполнять запрос.   
Если список пустой, то может быть выбрана любая из копий базы данных, определенных в информационной базе. Будет ли запрос выполняться на одной из копий, зависит от свойств запроса [ИспользованиеКопийБазыДанных](Запрос.ИспользованиеКопийБазыДанных_(Query.DatabaseCopiesUse).md), [ТребуемаяАктуальностьДанных](МакетКомпоновкиДанных.ТребуемаяАктуальностьДанных_(DataCompositionTemplate.RequiredDataRelevance).md), и [ТребуемоеВремяАктуальностиДанных](МакетКомпоновкиДанных.ТребуемоеВремяАктуальностиДанных_(DataCompositionTemplate.RequiredDataRelevanceTime).md).  
Выбирается первая подходящая из списка копия (в порядке следования элементов в коллекции [ИспользуемыеКопииБазыДанных](ИспользуемыеКопииБазыДанных_(UsedDatabaseCopies).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
