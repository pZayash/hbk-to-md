КатегорияОшибки (ErrorCategory)

ПринудительноеЗавершениеРаботы (ForcedShutdown)

Доступен, начиная с версии 8.3.25.

Описание:

Ошибка связана с прерыванием серверного вызова с помощью средств администрирования или с завершением сеанса средствами платформы.

Использование в версии:

Доступен, начиная с версии 8.3.25.
