ОператорВыбратьСхемыЗапроса (QuerySchemaSelectOperator)

КоличествоПолучаемыхЗаписей (RetrievedRecordsCount)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: Число. Неопределено.   
Количество получаемых записей.  
Соответствует ключевому слову ПЕРВЫЕ языка запросов.  
Если Неопределено, ограничение на количество выбираемых записей отсутствует.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
