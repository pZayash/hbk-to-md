Доступен, начиная с версии 8.0.

Элементы коллекции:

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры бизнес-процессов.  
Возможно обращение к менеджеру бизнес-процесса посредством оператора [...]. В качестве аргумента передается имя бизнес-процесса, как оно задано в конфигурации.  

Свойства:

[\<Имя бизнес-процесса\> (\<Business process name\>)](БизнесПроцессыМенеджер.Имя_бизнес-процесса.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](БизнесПроцессыМенеджер.ТипВсеСсылки.md)  
[ТипВсеСсылкиТочекМаршрутаБизнесПроцессов (RoutePointsAllRefsType)](БизнесПроцессыМенеджер.ТипВсеСсылкиТочекМаршрутаБизнесПроцессов.md)  

Описание:

Предназначен для управления бизнес-процессами и предоставляет доступ к значениям типа [БизнесПроцессыМенеджер](БизнесПроцессыМенеджер.md). Доступ к объекту осуществляется через свойство глобального контекста [БизнесПроцессы](Глобальный_контекст.БизнесПроцессы.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

Пример:

|  |
| --- |
| // менеджер бизнес-процессов предоставляет доступ к менеджеру  // бизнес-процесса ВыпискаСчета.  БизнесПроцесс = БизнесПроцессы.ВыпискаСчета.СоздатьБизнесПроцесс(); |
