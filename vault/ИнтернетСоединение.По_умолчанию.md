ИнтернетСоединение (InternetConnection)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ИнтернетСоединение(<НаименованиеСоединения>)

Параметры:

<НаименованиеСоединения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя соединения для подключения по коммутируемым линиям. Пустая строка используется в случае, если соединение с Интернетом постоянно или требуется использование коммутируемого соединения по умолчанию.  
Значение по умолчанию: Пустая строка.

Описание:

Создает объект [ИнтернетСоединение](ИнтернетСоединение_(InternetConnection).md).

Использование в версии:

Доступен, начиная с версии 8.0.
