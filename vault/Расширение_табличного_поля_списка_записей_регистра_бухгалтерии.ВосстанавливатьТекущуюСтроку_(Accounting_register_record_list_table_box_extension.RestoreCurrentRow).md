Расширение табличного поля списка записей регистра бухгалтерии (Accounting register record list table box extension)

ВосстанавливатьТекущуюСтроку (RestoreCurrentRow)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак восстановления текущей позиции в табличном поле при открытии формы, в которой табличное поле размещено.

Доступность:

Толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - текущая позиция восстанавливается.

Использование в версии:

Доступен, начиная с версии 8.0.
