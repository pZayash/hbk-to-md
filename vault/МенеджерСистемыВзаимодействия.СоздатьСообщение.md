МенеджерСистемыВзаимодействия (CollaborationSystemManager)

СоздатьСообщение (CreateMessage)

Доступен, начиная с версии 8.3.10.

Синтаксис:

СоздатьСообщение(<Обсуждение>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения.

Возвращаемое значение:

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия.md).   

Описание:

Создает новое сообщение в обсуждении. Для отправки сообщения необходимо вызвать метод [Записать](СообщениеСистемыВзаимодействия.Записать.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
