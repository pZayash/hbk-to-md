ЖурналДокументовСписок.<Имя журнала документов> (DocumentJournalList.<Document journal name>)

Колонки (Columns)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КолонкиСписка](КолонкиСписка.md).   
Содержит колонки списка журнала документов. В списке журнала могут быть использованы колонки с графами журнала, а также с датой, номером и видом документа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
