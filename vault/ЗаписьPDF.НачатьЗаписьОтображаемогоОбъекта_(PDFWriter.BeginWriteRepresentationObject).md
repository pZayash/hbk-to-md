ЗаписьPDF (PDFWriter)

НачатьЗаписьОтображаемогоОбъекта (BeginWriteRepresentationObject)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗаписьОтображаемогоОбъекта(<ОписаниеОповещения>, <ОписаниеОтображаемогоОбъекта>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещенияОЗакрытии](ФормаКлиентскогоПриложения.ОписаниеОповещенияОЗакрытии_(ClientApplicationForm.CallbackDescriptionOnClose).md).

<ОписаниеОтображаемогоОбъекта> (необязательный)

Тип: [ОписаниеОтображаемогоОбъектаPDF](ОписаниеОтображаемогоОбъектаPDF_(PDFRepresentationObjectDescription).md).   
Объект, содержащий описание отображения объекта в PDF-документе.

Описание:

Инициирует отображение объекта (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [ЗаписатьОтображаемыйОбъект](ЗаписьPDF.ЗаписатьОтображаемыйОбъект_(PDFWriter.WriteRepresentationObject).md)  
[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [ЗаписатьОтображаемыйОбъектАсинх](ЗаписьPDF.ЗаписатьОтображаемыйОбъектАсинх_(PDFWriter.WriteRepresentationObjectAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.
