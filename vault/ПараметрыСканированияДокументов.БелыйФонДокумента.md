ПараметрыСканированияДокументов (DocumentScanningParameters)

БелыйФонДокумента (WhiteBackgroundDocument)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет, учитывается ли цвет документа при поиске его границ:

- Истина - при поиске границ документа учитывается белый цвет фона документа. Изображение будет очищено от теней, сломов, что сделает его более ровным.
- Ложь - граница документа определяется без учета цвета документа.

Значение по умолчанию: Истина.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
