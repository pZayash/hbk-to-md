ОбъектМетаданных: Бот (MetadataObject: Bot)

Предопределенный (Predefined)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, является ли бот предопределенным.   
Предопределенные боты определяются в метаданных. Предопределенных ботов можно изменять, но нельзя удалять. Создание и удаление предопределенных ботов выполняется автоматически.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
