ВариантГраницыИнтервала (IntervalBoundVariant)

Месяц (Month)

Доступен, начиная с версии 8.0.

Описание:

Начало интервала задается датой начала месяца рабочей даты, конец интервала - датой конца месяца рабочей даты.

Использование в версии:

Доступен, начиная с версии 8.0.
