Форма загрузки (Load form)

СтандартныеНастройки (StandardSettings)

Доступен, начиная с версии 8.2.

Описание:

Тип: [СписокЗначений](СписокЗначений.md).   
Список значений, содержащий стандартные настройки, предусмотренные для объекта сохранения. Значением элемента списка является идентификатор стандартной настройки. Представлением элемента списка является представление стандартной настройки, отображаемое пользователю.

Использование в версии:

Доступен, начиная с версии 8.2.
