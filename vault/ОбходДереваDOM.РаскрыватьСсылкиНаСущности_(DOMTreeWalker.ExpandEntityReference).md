# ОбходДереваDOM.РаскрыватьСсылкиНаСущности

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ОбходДереваDOM](ОбходДереваDOM_(DOMTreeWalker).md) › [Свойства](ОбходДереваDOM__Свойства.md)

ОбходДереваDOM (DOMTreeWalker)

РаскрыватьСсылкиНаСущности (ExpandEntityReference)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак обхода ссылок на сущности при обходе узлов DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.

Использование в версии:

Доступен, начиная с версии 8.1.
