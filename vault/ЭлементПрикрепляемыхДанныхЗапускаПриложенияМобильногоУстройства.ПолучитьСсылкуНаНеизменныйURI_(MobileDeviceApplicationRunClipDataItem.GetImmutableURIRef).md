ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipDataItem)

ПолучитьСсылкуНаНеизменныйURI (GetImmutableURIRef)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьСсылкуНаНеизменныйURI()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Ссылка на ресурсы, если она есть в прикрепляемых данных, иначе [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает ссылку на ресурсы из элемента прикрепляемых данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
