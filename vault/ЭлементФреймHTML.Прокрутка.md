ЭлементФреймHTML (HTMLFrameElement)

Прокрутка (Scrolling)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Указывает наличие полос прокрутки во фрейме.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

YES - полосы прокрутки должны присутствовать, даже если обе не нужны;  
NO - полосы прокрутки не должны присутствовать, даже если хотя бы одна из них нужна;  
AUTO - HTML документ определяет необходимость полос прокрутки.

Использование в версии:

Доступен, начиная с версии 8.2.
