ИспользованиеАутентификацииИнтернетПочтыПоТокену (UseInternetMailTokenAuthentication)

Авто (Auto)

Доступен, начиная с версии 8.3.13.

Описание:

Автоматический режим. Если сервер поддерживает аутентификацию по протоколу OAuth2, то будет использована она, в противном случае – аутентификация по логину и паролю. Если значение свойства [ТокенДоступа](ИнтернетПочтовыйПрофиль.ТокенДоступа.md) не указано, то аутентификация по протоколу OAuth2 не используется.

Использование в версии:

Доступен, начиная с версии 8.3.13.
