МенеджерСредствПередачиДанныхНаУстройстве (DeviceDataSharingToolsManager)

ПоделитьсяДанными (ShareData)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПоделитьсяДанными(<Данные>, <ТипСодержимогоДляВыбораПриложения>, <Заголовок>)

Параметры:

<Данные> (обязательный)

Тип: Строка, [ДанныеМультимедиа](ДанныеМультимедиа.md), [Картинка](Картинка.md), [Массив](Массив.md).   
Данные для отправки. У объекта [ДанныеМультимедиа](ДанныеМультимедиа.md) должно быть указано свойство [РасширениеФайла](ДанныеМультимедиа.РасширениеФайла.md). Объект [Картинка](Картинка.md) должен быть известного формата. Иначе генерируется исключение.

<ТипСодержимогоДляВыбораПриложения> (необязательный)

Тип: Строка.   
Определяет тип передаваемых данных в стандарте MIME на устройствах под управлением ОС Android.   
Значение по умолчанию: "\*/\*".

<Заголовок> (необязательный)

Тип: Строка.   
Устанавливает заголовок в окне выбора на устройствах под управлением ОС Android.

Возвращаемое значение:

Тип: Булево.   
Истина, если найдены приложения, которые могут принимать данные из нашего приложения. Иначе возвращает Ложь. На iOS всегда Истина.

Описание:

Выводит системный диалог выбора приложений, которые могут принять данные из приложения 1С:Предприятия, что позволяет пользователям делиться информацией быстро и просто, используя пользовательские приложения и сервисы.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ВыбратьПоСсылкам](ПланСчетовМенеджер.Имя_плана_счетов.ВыбратьПоСсылкам.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.

Описание изменено в версии 8.3.26.
