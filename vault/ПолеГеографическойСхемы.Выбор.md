ПолеГеографическойСхемы (GeographicalSchemaField)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<ОбъектСхемы>, <СтандартнаяОбработка>)

Параметры:

<ОбъектСхемы>

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md).   
Объект географической схемы, который был интерактивно выбран.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Возникает при интерактивном выборе объекта географической схемы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
