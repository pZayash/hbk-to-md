ЭлементПланаГлобальногоПоиска (GlobalSearchPlanItem)

По обработчику

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ЭлементПланаГлобальногоПоиска(<ИмяПроцедуры>, <Модуль>, <НаСервере>, <Фоновый>, <Порядок>, <ДополнительныеПараметры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: Строка.   
Имя экспортируемой процедуры общего модуля.  
Процедура будет вызвана с параметрами:

- СтрокаПоиска - тип Строка. Строка. Строка, по которой выполняется поиск.
- РезультатыПоиска - тип [РезультатГлобальногоПоиска](РезультатГлобальногоПоиска.md). Выходной параметр. Содержит результаты поиска.
- ДополнительныеПараметры - тип Произвольный. Параметры, заданные в свойстве [ДополнительныеПараметры](ЭлементПланаГлобальногоПоиска.ДополнительныеПараметры.md) у [ЭлементПланаГлобальногоПоиска](ЭлементПланаГлобальногоПоиска.md).

<Модуль> (обязательный)

Тип: Строка.   
Имя общего модуля.

<НаСервере> (обязательный)

Тип: Булево.   
Признак выполнения поиска на сервере.

<Фоновый> (необязательный)

Тип: Булево.   
Признак выполнения поиска в фоновом задании.  
Значение по умолчанию: Ложь.

<Порядок> (необязательный)

Тип: Число.   
Порядок выдачи результатов поиска.  
Значение по умолчанию: 0.

<ДополнительныеПараметры> (необязательный)

Тип: Произвольный.   
Дополнительные параметры, передаваемые в вызываемую процедуру.

Описание:

Создает элемент плана глобального поиска по обработчику.

Использование в версии:

Доступен, начиная с версии 8.3.15.
