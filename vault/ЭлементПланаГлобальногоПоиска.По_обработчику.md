ЭлементПланаГлобальногоПоиска (GlobalSearchPlanItem)

По обработчику

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ЭлементПланаГлобальногоПоиска(<ИмяПроцедуры>, <Модуль>, <НаСервере>, <Фоновый>, <Порядок>, <ДополнительныеПараметры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры общего модуля.  
Процедура будет вызвана с параметрами:

- СтрокаПоиска - тип [Строка](lang__def_String.md). Строка. Строка, по которой выполняется поиск.
- РезультатыПоиска - тип [РезультатГлобальногоПоиска](РезультатГлобальногоПоиска_(GlobalSearchResult).md). Выходной параметр. Содержит результаты поиска.
- ДополнительныеПараметры - тип Произвольный. Параметры, заданные в свойстве [ДополнительныеПараметры](ЭлементПланаГлобальногоПоиска.ДополнительныеПараметры_(GlobalSearchPlanItem.AdditionalParameters).md) у [ЭлементПланаГлобальногоПоиска](ЭлементПланаГлобальногоПоиска_(GlobalSearchPlanItem).md).

<Модуль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя общего модуля.

<НаСервере> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения поиска на сервере.

<Фоновый> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения поиска в фоновом задании.  
Значение по умолчанию: Ложь.

<Порядок> (необязательный)

Тип: [Число](lang__def_Number.md).   
Порядок выдачи результатов поиска.  
Значение по умолчанию: 0.

<ДополнительныеПараметры> (необязательный)

Тип: Произвольный.   
Дополнительные параметры, передаваемые в вызываемую процедуру.

Описание:

Создает элемент плана глобального поиска по обработчику.

Использование в версии:

Доступен, начиная с версии 8.3.15.
