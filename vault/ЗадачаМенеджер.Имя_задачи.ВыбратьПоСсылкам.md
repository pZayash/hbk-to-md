ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на задачи.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [ЗадачаВыборка.\<Имя задачи\>](ЗадачаВыборка.Имя_задачи.md).   

Описание:

Формирует выборку задач по заданному источнику ссылок на задачи соответствующего типа. Если источник содержит значения, отличные от ссылок на задачи соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    БюджетнаяЗадача.Ссылка КАК Ссылка     |ИЗ     |    Задача.БюджетнаяЗадача КАК БюджетнаяЗадача     |"; РезультатЗапроса = Запрос.Выполнить(); Выборка = Задачи.БюджетнаяЗадача.ВыбратьПоСсылкам(РезультатЗапроса); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
