# КонстантыНабор.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md) › [КонстантыНабор](КонстантыНабор_(ConstantsSet).md) › [Методы](КонстантыНабор__Методы.md)

КонстантыНабор (ConstantsSet)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверка заполнения выполняется менеджерами значений констант, входящих в набор и допускающих запись с текущим набором прав.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к последовательному возникновению события [ОбработкаПроверкиЗаполнения](КонстантаМенеджерЗначения.Имя_константы.ОбработкаПроверкиЗаполнения_(ConstantValueManager.Constant_name.FillCheckProcessing).md) для каждой константы, входящей в набор.

Использование в версии:

Доступен, начиная с версии 8.2.
