ПоследовательностьXDTO (XDTOSequence)

ПолучитьТекст (GetText)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьТекст(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: Число.   
Индекс в последовательности.

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает текст, содержащийся в последовательности по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если по указанному индексу содержится свойство, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
