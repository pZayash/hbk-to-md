МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьПараметрыВнешнегоПодключенияИнформационнойБазы (GetInfoBaseExternalConnectionParameters)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьПараметрыВнешнегоПодключенияИнформационнойБазы()

Возвращаемое значение:

Тип: [ПараметрыВнешнегоПодключенияРаботыСРечью](ПараметрыВнешнегоПодключенияРаботыСРечью.md).   
Неопределено, если параметры не установлены.

Описание:

Получает параметры доступа к облачному сервису работы с речью в информационной базе (для текущей области, если для сеанса определены разделители).  
Требует наличия права "Администрирование данных".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
