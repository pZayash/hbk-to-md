ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, необходимо ли выделять ячейку, в которой не заполнено значение, либо оно выставлено в значение по умолчанию.  
Если нет ни одной заполненной ячейки, будет установлено [ОтметкаНезаполненного](ТаблицаФормы.ОтметкаНезаполненного_(FormTable.MarkIncomplete).md) в [Истина](lang__def_BooleanTrue.md), иначе [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
