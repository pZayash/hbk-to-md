ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

НайтиПоКоду (FindByCode)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>)

Параметры:

<Код> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомый код.

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   
Если не существует ни одного узла с требуемым кодом, то возвращается пустая ссылка.

Описание:

Осуществляет поиск узла плана обмена по коду.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
