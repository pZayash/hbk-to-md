ПользовательСистемыВзаимодействия (CollaborationSystemUser)

Внешний (External)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), пользователь является внешним.  
Установить свойство можно только для нового объекта.   
При записи нового внешнего объекта должно быть записано свойство [АдресЭлектроннойПочтыДляАутентификации](ПользовательСистемыВзаимодействия.АдресЭлектроннойПочтыДляАутентификации_(CollaborationSystemUser.AuthenticationEmail).md). Если уже есть внешний пользователь с таким же значением свойства [АдресЭлектроннойПочтыДляАутентификации](ПользовательСистемыВзаимодействия.АдресЭлектроннойПочтыДляАутентификации_(CollaborationSystemUser.AuthenticationEmail).md), будет сгенерировано исключение.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
