ФайловыйПоток (FileStream)

НачатьКопированиеВ (BeginCopyTo)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьКопированиеВ(<ОписаниеОповещения>, <ЦелевойПоток>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Количество> - тип Число. Количество фактически скопированных байтов.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ЦелевойПоток> (обязательный)

Тип: [Поток](Поток.md).   
Поток, в который будет производиться копирование.

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые требуется скопировать в целевой поток. Если не задано, то будут скопированы все данные до конца текущего потока.

Описание:

Начинает копирование данных из текущего потока в другой поток. Дополнительно можно установить количество байтов, которые будут скопированы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Поток](Поток.md), метод [КопироватьВ](Поток.КопироватьВ.md)  
[Поток](Поток.md), метод [КопироватьВАсинх](Поток.КопироватьВАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
