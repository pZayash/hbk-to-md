# ТекстовыйДокумент.Вывести

ТекстовыйДокумент (TextDocument)

Вывести (Put)

Доступен, начиная с версии 8.0.

Синтаксис:

Вывести(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md).   
Выводимый текстовый документ.

Описание:

Добавляет текстовый документ в результирующий текстовый документ, добавляя его со следующей строки вслед за самой нижней выведенной строкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Этот метод имеет смысл использовать только при работе с текстовым документом, содержащим макет.  
При этом строки текстового документа, не входящие ни в одну из областей макета, выводиться не будут.

Использование в версии:

Доступен, начиная с версии 8.0.
