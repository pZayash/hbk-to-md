Диаграмма (Chart)

ИспользованиеНечисловыхЗначений (NonnumericValueUse)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеНечисловыхЗначенийДиаграммы](ИспользованиеНечисловыхЗначенийДиаграммы_(NonnumericChartValueUse).md).   
Определяет использование значений, которые не приводятся к числу.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.13.
