МенеджерМетокNDEF (TagManagerNDEF)

ПоддерживаетсяЗаписьНаМетку (WriteOnTagSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяЗаписьНаМетку()

Возвращаемое значение:

Тип: Булево.   
Истина - устройство поддерживает запись на метку и NFC включено,  
Ложь - в противном случае.

Описание:

Проверяет, что устройство поддерживает операции записи на метку.   
Для устройств на Android дополнительно проверяется, что пользователь включил NFC в настройках телефона.  
Если запись не поддерживается, то методы [ЗаписатьСообщениеАсинх](МеткаNDEF.ЗаписатьСообщениеАсинх.md) и [ЗаписатьСообщениеАсинх](МеткаNDEF.ЗаписатьСообщениеАсинх.md) будут недоступны.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МеткаNDEF](МеткаNDEF.md), метод [ПоддерживаетсяЗаписьНаМеткуАсинх](МеткаNDEF.ПоддерживаетсяЗаписьНаМеткуАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
