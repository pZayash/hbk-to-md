# Расширение декорации формы для картинки.Масштабировать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Декорация формы](Декорация_формы.md) › [Расширение декорации формы для картинки](Расширение_декорации_формы_для_картинки_(Form_decoration_extension_for_a_picture).md) › [Свойства](Расширение_декорации_формы_для_картинки__Свойства.md)

Расширение декорации формы для картинки (Form decoration extension for a picture)

Масштабировать (Zoomable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено ([Истина](lang__def_BooleanTrue.md)), то для картинки, не помещающейся в области размещения, появляются линейки прокрутки.   
Изменить размер картинки можно с помощью команд контекстного меню.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
