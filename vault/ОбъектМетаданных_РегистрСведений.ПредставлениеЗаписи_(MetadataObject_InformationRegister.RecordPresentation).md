ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ПредставлениеЗаписи (RecordPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием записи регистра сведений и используется при отображении стандартных команд.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
