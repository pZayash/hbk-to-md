Соединение с агентом сервера (IServerAgentConnection)

CreateAssignmentRule (CreateAssignmentRule)

Доступен, начиная с версии 8.3.1.

Синтаксис:

CreateAssignmentRule()

Возвращаемое значение:

Тип: [Требование назначения сервисов](Требование_назначения_сервисов.md).   

Описание:

Создает новое требование назначения для его последующего добавления в список требований. После создания необходимо установить значения его свойств.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
