МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблонСообщения (GetMessageTemplate)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблонСообщения(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия.md).   
Идентификатор шаблона сообщения.

Возвращаемое значение:

Тип: [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия.md), Неопределено.   
Для интеграции типа WhatsApp возвращает шаблон сообщения по его идентификатору.  
Для остальных интеграций или если шаблон не найден возвращает Неопределено.

Описание:

Получает шаблон сообщения системы взаимодействия.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ПолучитьШаблонСообщенияАсинх](МенеджерСистемыВзаимодействия.ПолучитьШаблонСообщенияАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
