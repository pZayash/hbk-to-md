АдресЦентраЛицензирования (LicensingCenterAddress)

Доступен, начиная с версии 8.3.20.

Синтаксис:

АдресЦентраЛицензирования()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Позволяет получить URL-адрес сайта центра лицензирования для самостоятельного получения лицензии на носителе.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
