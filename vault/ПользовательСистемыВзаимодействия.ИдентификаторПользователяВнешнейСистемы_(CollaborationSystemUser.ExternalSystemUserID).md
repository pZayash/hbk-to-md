ПользовательСистемыВзаимодействия (CollaborationSystemUser)

ИдентификаторПользователяВнешнейСистемы (ExternalSystemUserID)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит идентификатор пользователя во внешней системе.  
[Неопределено](lang__def_Undefined.md) - если пользователь не является пользователем внешней системы.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
