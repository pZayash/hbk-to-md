ЭлементDOM (DOMElement)

ПолучитьЭлементыПоИмени (GetElementByTagName)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьЭлементыПоИмени()

Описание варианта метода:

Может использоваться предопределенный шаблон '\*', означающий необходимость выбора всех узлов [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) дерева.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ПолучитьЭлементыПоИмени(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.  
Может использоваться предопределенный шаблон '\*', означающий необходимость выбора узлов [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) дерева документа с произвольным URI пространства имен.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя.  
Может использоваться предопределенный шаблон '\*', означающий необходимость выбора узлов [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) дерева документа с произвольным локальным именем.

Возвращаемое значение:

Тип: [СписокЭлементовDOM](СписокЭлементовDOM_(DOMElementList).md).   
Список узлов DOM (элементов), являющихся дочерними для данного узла, в порядке их следования в документе.

Описание:

Выбирает все элементы с указанным именем.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
