ДокументPDF (PDFDocument)

ПрочитатьАсинх (ReadAsync)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: Из файла

Синтаксис:

ПрочитатьАсинх(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для чтения PDF-документа.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к PDF-документу.

Описание варианта метода:

Асинхронная функция. Считывает документ PDF из файла.  
Позволяет считывать PDF-документ из файла формата "документ PDF".  
Если документ защищен паролем и пароль не указан или является не верным, то генерируется исключение.  
  
Доступность:  
Тонкий клиент, толстый клиент, веб-клиент.

Вариант синтаксиса: Из потока

Синтаксис:

ПрочитатьАсинх(<Поток>, <Пароль>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения PDF-документа.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к PDF-документу.

Описание варианта метода:

Асинхронная функция. Считывает документ PDF из потока.  
Позволяет считывать PDF-документ из файла формата "документ PDF".  
Если документ защищен паролем и пароль не указан или является не верным, то генерируется исключение.  
  
Доступность:  
Тонкий клиент, толстый клиент, веб-клиент.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   

Описание:

Асинхронная функция. Считывает PDF-документ из файла.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Для работы метода в веб-клиенте требуется наличие расширения работы с файлами.

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.26.
