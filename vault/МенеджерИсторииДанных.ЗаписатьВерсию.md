МенеджерИсторииДанных (DataHistoryManager)

ЗаписатьВерсию (WriteVersion)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ЗаписатьВерсию(<Данные>, <Дата>, <Пользователь>, <ИмяПользователя>, <ПолноеИмяПользователя>, <ВидИзмененияДанных>, <Комментарий>, <Транзакция>, <Узел>)

Параметры:

<Данные> (обязательный)

Тип: [СправочникОбъект.](СправочникОбъект.Имя_справочника.md)<[Имя справочника](СправочникОбъект.Имя_справочника.md)>, [ДокументОбъект.](ДокументОбъект.Имя_документа.md)<[Имя документа](ДокументОбъект.Имя_документа.md)>, [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>, [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>, [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>, [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>, [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>, [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)>, [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы.md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы.md)>.   
Данные для формирования версии.

<Дата> (обязательный)

Тип: Дата.   
Дата создания версии.

<Пользователь> (обязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Идентификатор пользователя, создавшего версию.

<ИмяПользователя> (обязательный)

Тип: Строка.   
Имя пользователя, создавшего версию.  
Строка, превышающая максимально допустимую длину в 256 символов, будет обрезана.

<ПолноеИмяПользователя> (обязательный)

Тип: Строка.   
Полное имя пользователя, создавшего версию.  
Строка, превышающая максимально допустимую длину в 256 символов, будет обрезана.

<ВидИзмененияДанных> (обязательный)

Тип: [ВидИзмененияДанных](ВидИзмененияДанных.md).   
Вид изменения данных, которое привело к созданию версии.

<Комментарий> (необязательный)

Тип: Строка.   
Комментарий к версии.  
Строка, превышающая максимально допустимую длину в 1024 символа, будет обрезана.  
Значение по умолчанию: Пустая строка.

<Транзакция> (необязательный)

Тип: Строка.   
Идентификатор транзакции версии.  
Идентификатор должен иметь вид: "02.01.0001 0:21:41 (865831)". Следует использовать только значения, полученные средствами платформы из журнала регистрации, подсистемы версионирования и др., или значение Неопределено. В противном случае значение может оказаться недопустимым.  
Если указана пустая строка, идентификатор транзакции не устанавливается.  
Значение по умолчанию: Пустая строка.

<Узел> (необязательный)

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>, Неопределено.   
Узел плана обмена, из которого получено изменение данных.  
Значение по умолчанию: Неопределено.

Описание:

Добавляет версию данных для переданного объекта или набора записей регистра сведений.  
Метод предназначен для переноса истории данных из других механизмов версионирования.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Пример:

|  |
| --- |
| Данные = Справочники.Валюты.НайтиПоНаименованию("Рубль")             .ПолучитьОбъект(); ДатаСоздания = Дата(2016, 1, 1, 1, 1, 1); Пользователь = ПользователиИнформационнойБазы             .НайтиПоИмени("Создателев"); ИсторияДанных.ЗаписатьВерсию(         Данные,         ДатаСоздания,         Пользователь.УникальныйИдентификатор,         Пользователь.Имя,         Пользователь.ПолноеИмя,         ВидИзмененияДанных.Добавление,         "Загрузка данных"); |

Использование в версии:

Доступен, начиная с версии 8.3.11.
