Глобальный контекст (Global context)

ПолучитьВнешнююНавигационнуюСсылку (GetExternalURL)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьВнешнююНавигационнуюСсылку(<БазоваяНавигационнаяСсылка>, <ОтносительнаяНавигационнаяСсылка>, <ДополнительныеПараметрыНавигационнойСсылки>)

Параметры:

<БазоваяНавигационнаяСсылка> (необязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Часть ссылки, по которой должен быть выполнен переход в приложение. Если [Неопределено](lang__def_Undefined.md), базовая часть будет выбрана автоматически:

- В "1С:Предприятии" для персонального компьютера поведение остается как при вызове метода [ПолучитьНавигационнуюСсылкуИнформационнойБазы](Глобальный_контекст.ПолучитьНавигационнуюСсылкуИнформационнойБазы_(Global_context.GetInfoBaseURL).md).
- В мобильной платформе будет выбран из списка перехватываемых ссылок, объявленных в конфигурации. Приоритет отдается в пользу глубинных ссылок. Если список пуст, возвращаемое значение метода – пустая строка.

<ОтносительнаяНавигационнаяСсылка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Относительная часть ссылки, указывающая на объект внутри информационной базы.

<ДополнительныеПараметрыНавигационнойСсылки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Дополнительные параметры ссылки, которые будут указаны после знака вопроса, где:

- Ключ – тип [Строка](lang__def_String.md), имя параметра.
- Значение – тип [Строка](lang__def_String.md) или [Массив](Массив_(Array).md) строк, значение параметра.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает внешнюю навигационную ссылку на объект информационной базы, по которой можно перейти из другого (внешнего) приложения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
