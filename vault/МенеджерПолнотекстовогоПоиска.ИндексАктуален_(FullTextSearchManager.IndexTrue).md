# МенеджерПолнотекстовогоПоиска.ИндексАктуален

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Полнотекстовый поиск](Полнотекстовый_поиск.md) › [МенеджерПолнотекстовогоПоиска](МенеджерПолнотекстовогоПоиска_(FullTextSearchManager).md) › [Методы](МенеджерПолнотекстовогоПоиска__Методы.md)

МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ИндексАктуален (IndexTrue)

Доступен, начиная с версии 8.1.

Синтаксис:

ИндексАктуален()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если индекс полнотекстового поиска полностью соответствует текущему состоянию информационной базы.

Описание:

Осуществляет проверку актуальности индекса полнотекстового поиска.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если режим полнотекстового поиска запрещен, то использование метода вызывает исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
