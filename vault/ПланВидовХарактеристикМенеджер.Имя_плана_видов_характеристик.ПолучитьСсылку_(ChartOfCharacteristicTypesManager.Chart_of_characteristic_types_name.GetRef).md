# ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик>.ПолучитьСсылку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик>](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md) › [Методы](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик__Методы.md)

ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик> (ChartOfCharacteristicTypesManager.<Chart of characteristic types name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.УникальныйИдентификатор_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name.UUID).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

Пример:

|  |
| --- |
| Значение = ХМЛ.ПолучитьАтрибут("Ид"); Попытка     Ид = Новый УникальныйИдентификатор(Значение);     Свойство = ПланыВидовХарактеристик.ДопСвойства.ПолучитьСсылку(Ид); Исключение     Возврат; КонецПопытки; |
