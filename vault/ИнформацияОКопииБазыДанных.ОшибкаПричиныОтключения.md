ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ОшибкаПричиныОтключения (TurnedOffReasonError)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md), Неопределено.   
При отсутствии ошибок свойство содержит значение Неопределено. При возникновении ошибки объект, описывающий ошибку, которая привела к недоступности копии для выполнения запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
