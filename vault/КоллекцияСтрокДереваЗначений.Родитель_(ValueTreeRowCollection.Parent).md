КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Родитель (Parent)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md), [Неопределено](lang__def_Undefined.md).   
Строка, которой принадлежит данная коллекция строк (родительская строка).  
Если данная коллекция строк является корневой коллекцией дерева, то имеет значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
