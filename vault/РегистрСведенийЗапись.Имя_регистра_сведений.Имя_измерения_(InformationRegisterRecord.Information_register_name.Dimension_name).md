РегистрСведенийЗапись.<Имя регистра сведений> (InformationRegisterRecord.<Information register name>)

<Имя измерения> (<Dimension name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения измерений регистра сведений. Имена свойств совпадают с именами измерений, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В качестве типа допускается использовать любые типы, кроме [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) и строки неограниченной длины.

Использование в версии:

Доступен, начиная с версии 8.0.
