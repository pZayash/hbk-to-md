Кластер серверов (IClusterInfo)

KillByMemoryWithDump (KillByMemoryWithDump)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак формирования дампа при завершении процессов, останавливаемых принудительно при превышении значения параметра рабочего сервера "Критический объем памяти процессов".

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
