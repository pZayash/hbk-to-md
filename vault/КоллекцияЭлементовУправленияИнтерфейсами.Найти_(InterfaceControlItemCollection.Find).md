КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяИнтерфейса>)

Параметры:

<ИмяИнтерфейса> (обязательный)

Тип: [Строка](lang__def_String.md).   
Наименование интерфейса, как он назван в конфигураторе.

Возвращаемое значение:

Тип: [ЭлементУправленияИнтерфейсом](ЭлементУправленияИнтерфейсом_(InterfaceControlItem).md), [Неопределено](lang__def_Undefined.md).   
Если интерфейс отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента управления интерфейсом по его имени.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
