КнопкиКоманднойПанели (CommandBarButtons)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяКнопки>)

Параметры:

<ИмяКнопки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Наименование кнопки.

Возвращаемое значение:

Тип: [КнопкаКоманднойПанели](КнопкаКоманднойПанели_(CommandBarButton).md), [Неопределено](lang__def_Undefined.md).   
Если кнопка с указанным именем отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск кнопки командной панели по ее имени.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
