Планировщик (Planner)

ТекущиеПериодыОтображения (CurrentRepresentationPeriods)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияТекущихПериодовОтображенияПланировщика](КоллекцияТекущихПериодовОтображенияПланировщика_(PlannerCurrentRepresentationPeriodCollection).md).   
Коллекция, содержащая элементы типа [ПериодОтображенияПланировщика](ПериодОтображенияПланировщика_(PlannerRepresentationPeriod).md). Диапазон дат, отображаемый планировщиком, определяется, исходя из элементов этой коллекции. Некоторый диапазон будет отображён, если он входит в хотя бы один из периодов отображения, находящихся в коллекции.  
По умолчанию в коллекции содержится один элемент с датами, вычисленными на основании текущей даты, а также свойств [ЕдиницаПериодическогоВарианта](Планировщик.ЕдиницаПериодическогоВарианта_(Planner.PeriodicVariantUnit).md) и [КратностьПериодическогоВарианта](Планировщик.КратностьПериодическогоВарианта_(Planner.PeriodicVariantRepetition).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[Планировщик](Планировщик_(Planner).md), свойство [ЕдиницаПериодическогоВарианта](Планировщик.ЕдиницаПериодическогоВарианта_(Planner.PeriodicVariantUnit).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
