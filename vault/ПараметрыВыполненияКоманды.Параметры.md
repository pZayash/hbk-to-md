ПараметрыВыполненияКоманды (CommandExecuteParameters)

Параметры (Parameters)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [Структура](Структура.md).   
В данный параметр помещаются параметры обработки команды.   
Если команда открывалась по навигационной ссылке, то в значение данного свойства будет помещена структура, полученная из хранилища внешних данных навигационных ссылок по ключу, полученному из параметра extdata, находящегося в ссылке (если он есть). В качестве имени объекта будет использовано имя команды.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.19.
