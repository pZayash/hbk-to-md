РежимАвтоВремя (AutoTimeMode)

Первым (First)

Доступен, начиная с версии 8.0.

Описание:

Установить время объекта (документа, бизнес-процесса, задачи) так, что бы он оказался первым на свою дату. Если при этом дата объекта совпадает с текущей по дню, то время объекта будет установлено не позже, чем текущее.

Использование в версии:

Доступен, начиная с версии 8.0.
