ДиаграммаГанта (GanttChart)

ПриОкончанииРедактированияИнтервала (OnIntervalEditEnd)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриОкончанииРедактированияИнтервала(<Интервал>, <ОтменаРедактирования>)

Параметры:

<Интервал>

Тип: [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md).   
Интервал, который был изменен.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.  
Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при окончании интерактивного редактирования интервала.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.23.
