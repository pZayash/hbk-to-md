МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ПолучитьМинимальныйРазмерЗаписываемыхДанных (GetMinWriteDataSize)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьМинимальныйРазмерЗаписываемыхДанных()

Возвращаемое значение:

Тип: Число.   
Значение по умолчанию: 2048.  
Значение должно находиться в диапазоне от 2048 до 2^64-1.

Описание:

Возвращает минимальный размер записываемых данных в хранилище двоичных данных (размер порции).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
