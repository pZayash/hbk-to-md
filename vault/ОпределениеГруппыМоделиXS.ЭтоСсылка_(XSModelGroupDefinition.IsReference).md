ОпределениеГруппыМоделиXS (XSModelGroupDefinition)

ЭтоСсылка (IsReference)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, является ли данное определение модели группы ссылкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - определение является ссылкой.

Использование в версии:

Доступен, начиная с версии 8.1.
