# Расширение поля формы для поля HTML-документа.Высота

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля HTML-документа](Расширение_поля_формы_для_поля_HTML-документа_(Form_extension_for_a_HTML_document_field).md) › [Свойства](Расширение_поля_формы_для_поля_HTML-документа__Свойства.md)

Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту поля. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
