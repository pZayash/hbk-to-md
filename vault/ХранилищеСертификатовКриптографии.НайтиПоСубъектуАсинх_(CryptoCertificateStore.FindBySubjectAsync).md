ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НайтиПоСубъектуАсинх (FindBySubjectAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НайтиПоСубъектуАсинх(<СтрокаПоиска>)

Параметры:

<СтрокаПоиска> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка поиска субъекта.  
Данные субъекта должны содержать переданную в этом параметре строку.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив объектов [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Описание:

Асинхронная функция. Выполняет поиск сертификатов криптографии по переданному имени субьекта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
