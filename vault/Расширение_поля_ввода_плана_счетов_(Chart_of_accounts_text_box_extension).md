# Расширение поля ввода плана счетов

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md)

Доступен, начиная с версии 8.0.

Свойства:

[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_плана_счетов.БыстрыйВыбор_(Chart_of_accounts_text_box_extension.QuickChoice).md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на план счетов.  
Расширение предоставляет возможность автоматического поиска и подбора счета по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
