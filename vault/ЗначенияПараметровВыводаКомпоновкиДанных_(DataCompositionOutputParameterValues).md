Описание:

Значения параметров вывода для настроек.  
Содержит значения следующих параметров:

- АвтоПозицияРесурсов (ResourcesAutoPosition) - автопозиция ресурсов (тип [АвтоПозицияРесурсовКомпоновкиДанных](АвтоПозицияРесурсовКомпоновкиДанных_(DataCompositionResourcesAutoPosition).md));

- ВертикальноеРасположениеОбщихИтогов (VerticalOverallPlacement) - вертикальное расположение общих итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных_(DataCompositionTotalPlacement).md));
- ВыводитьЗаголовок (TitleOutput) - выводить заголовок (тип [ТипВыводаТекстаКомпоновкиДанных](ТипВыводаТекстаКомпоновкиДанных_(DataCompositionTextOutputType).md));
- ВыводитьОтбор (FilterOutput) - выводить отбор (тип [ТипВыводаТекстаКомпоновкиДанных](ТипВыводаТекстаКомпоновкиДанных_(DataCompositionTextOutputType).md));

- Заголовок (Title) - заголовок (тип [Строка](lang__def_String.md));
- КоличествоЗаписей (RecordsCount) - количество записей, которое необходимо выводить в группировке (тип [Число](lang__def_Number.md));

- РасположениеРеквизитов (AttributePlacement) - расположение реквизитов (тип [РасположениеРеквизитовКомпоновкиДанных](РасположениеРеквизитовКомпоновкиДанных_(DataCompositionAttributesPlacement).md));
