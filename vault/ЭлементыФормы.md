Описание:

Используется для доступа к элементам управления, расположенным на форме. Типы элементов управления описаны в [Элементы управления](Элементы_управления.md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Сообщить(ЭлементыФормы.ТаблПоле.Колонки.Количество()); |
