СущностьDOM (DOMEntity)

ЗначениеУзла (NodeValue)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит значение узла DOM. Значение узла присутствует не для каждого типа узла DOM.  
Узел Значение узла

- [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - [Значение](АтрибутDOM.Значение_(DOMAttribute.Value).md);

- [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - [Неопределено](lang__def_Undefined.md);

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
