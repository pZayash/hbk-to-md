СредстваТелефонии (TelephonyTools)

ПоддерживаетсяОтправкаSMS (SMSSendingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяОтправкаSMS(<Интерактивно>)

Параметры:

<Интерактивно> (обязательный)

Тип: Булево.   
Указывает, используется ли интерактивный режим для отправки SMS-сообщений.

Возвращаемое значение:

Тип: Булево.   
При интерактивной отправке возвращает Истина в следующих случаях:

- Android - при наличии телефонного модуля,
- iOS - если возможна отправка SMS-сообщений,
- Windows Mobile - всегда.

При не интерактивной отправке возвращает Истина в следующих случаях:

- Android - при наличии телефонного модуля,
- в остальных операционных системах - никогда.

Описание:

Определяет доступность отправки сообщений SMS с данного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[СредстваТелефонии](СредстваТелефонии.md), метод [ПослатьSMS](СредстваТелефонии.ПослатьSMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
