ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

<Имя предопределенного объекта> (<The name of a predefined object.>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланВидовРасчетаСсылка.\<Имя плана видов расчета\>](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md).   
Содержит ссылки на предопределенный объект плана видов расчета. Доступ к значению осуществляется по имени, как они заданы в Конфигураторе. Имена имеют только предопределенные виды расчета. Количество таких свойств задается списком предопределенных объектов плана видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
