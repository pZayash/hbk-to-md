# Дерево решений

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Анализ данных](objects__catalog63__catalog818.md)

Дерево решений — схематическое представление процесса принятия решений по определенной задаче, изображаемое графически в виде древовидной структуры.  
  
Дерево решений представляет собой схему, описывающую процесс принятия решения и последствия выбора каждой из возможных альтернатив. Каждый вариант характеризуется вероятностью рисков и затратами или доходами от выбора каждой логической последовательности событий и будущих решений.
