# ПользовательИнформационнойБазы.ДатаУстановкиПароля

ПользовательИнформационнойБазы (InfoBaseUser)

ДатаУстановкиПароля (PasswordSettingDate)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата установки пароля в формате UTC.  
Доступно для чтения только пользователю с административными правами.  
Значение данного свойства у локального объекта [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md) не изменяется после установки значения для свойств [Пароль](ПользовательИнформационнойБазы.Пароль_(InfoBaseUser.Password).md), [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md), [ЗаписываемаяДатаУстановкиПароля](ПользовательИнформационнойБазы.ЗаписываемаяДатаУстановкиПароля_(InfoBaseUser.WritablePasswordSettingDate).md) и после записи пользователя.  
Чтобы получить правильную дату установки пароля, после установки пользователю нового пароля следует записать пользователя с измененным паролем, а затем прочитать его.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
