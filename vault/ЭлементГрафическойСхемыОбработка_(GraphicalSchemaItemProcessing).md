Описание:

Элемент графической схемы "Обработка". Отображает точку маршрута бизнес-процесса, выполняемую в автоматическом режиме и не имеющую адресата.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
