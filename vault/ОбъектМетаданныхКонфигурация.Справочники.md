ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Справочники (Catalogs)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все справочники, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
