Глобальный контекст (Global context)

УстановитьПроверкуСложностиПаролейПользователей (SetUserPasswordStrengthCheck)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьПроверкуСложностиПаролейПользователей(<СложныйПароль>)

Параметры:

<СложныйПароль> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - установить; [Ложь](lang__def_BooleanFalse.md) - снять.

Описание:

Устанавливает или отключает следующие ограничения на сложность паролей пользователей информационной базы:

- Длина не менее 7 символов либо не меньше значения, установленного в параметре "Минимальная длина паролей пользователей", если оно превышает 7;
- Содержит не менее 3-х типов символов из перечисленных ниже:

- заглавные буквы;
- строчные буквы;
- цифры;
- спец. символы.

- Не совпадает с именем пользователя;
- Не является последовательностью символов.

Действует только:

- при изменении пароля существующего пользователя или при создании нового пользователя, если включена [АутентификацияСтандартная](ПользовательИнформационнойБазы.АутентификацияСтандартная_(InfoBaseUser.StandardAuthentication).md) или новый пароль не пустой;
- при включении [АутентификацияСтандартная](ПользовательИнформационнойБазы.АутентификацияСтандартная_(InfoBaseUser.StandardAuthentication).md) у существующего пользователя, если пароль пустой.

Метод доступен только из сеанса, в котором все разделители не используются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.22.
