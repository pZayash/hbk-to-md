Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileInternetResource (UnregSecurityProfileInternetResource)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileInternetResource(<Кластер>, <ИмяПрофиля>, <ИмяРесурса>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должна быть отменена регистрация разрешенного ресурса Интернет.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<ИмяРесурса> (обязательный)

Тип: Строка.   
Имя ресурса сети Интернет, регистрация которого отменяется.

Описание:

Отменяет регистрацию разрешенного ресурса сети Интернет.

Доступность:

Интеграция.

Примечание:

Если профиля, указанного в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если разрешенного ресурса сети Интернет с именем, указанным в параметре <ИмяРесурса>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
