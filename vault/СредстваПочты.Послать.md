СредстваПочты (MailTools)

Послать (Send)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Послать(<Сообщение>, <ОбрабатыватьТексты>)

Параметры:

<Сообщение> (обязательный)

Тип: [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение.md).   
Почтовое сообщение.  
Внимание! Тип текстов почтовых сообщений [РазмеченныйТекст](ТипТекстаПочтовогоСообщения.РазмеченныйТекст.md) не поддерживается.

<ОбрабатыватьТексты> (необязательный)

Тип: [ОбработкаТекстаИнтернетПочтовогоСообщения](ОбработкаТекстаИнтернетПочтовогоСообщения.md).   
Управляет обработкой текстов перед отправкой:

- [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать.md) - Локальные картинки из текста, помещаются в сообщение в виде вложений. Содержимое текста письма изменяется таким образом, чтобы теги <img> указывали на картинки из вложений.
- [НеОбрабатывать](ДопустимыеДействияПеретаскивания.НеОбрабатывать.md) - Текст посылается без внутренней обработки.
- Загрузка картинок при отправке HTML-документов отключена.

  
Значение по умолчанию: [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать.md).

Описание:

Отправляет почтовое сообщение.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяОтправка](СредстваПочты.ПоддерживаетсяОтправка.md).  
Внимание! Тип текстов почтовых сообщений [РазмеченныйТекст](ТипТекстаПочтовогоСообщения.РазмеченныйТекст.md) не поддерживается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваПочты](СредстваПочты.md), метод [ПоддерживаетсяОтправка](СредстваПочты.ПоддерживаетсяОтправка.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
