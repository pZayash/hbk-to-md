ТабличноеПоле (TableBox)

ИзменятьНастройкуКолонок (ChangeSettingOfColumns)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет возможность настройки всех колонок табличного поля.

Доступность:

Толстый клиент.

Примечание:

Если свойство имеет значение Ложь, то при открытии формы у табличного поля будет восстановлена стандартная настройка колонок.

Использование в версии:

Доступен, начиная с версии 8.0.
