ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Вставить (Insert)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Вставить(<Индекс>, <Имя>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Позиция вставляемой копии.

<Имя> (обязательный)

Тип: Строка.   
Имя вставляемой копии.

Возвращаемое значение:

Тип: [ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных.md).   

Описание:

Вставляет используемую копию в определенную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
