РезультатXPath (XPathResult)

РазмерСнимка (SnapshotLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит количество узлов в результирующем снимке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если тип результата не равен [НеупорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.НеупорядоченныйСнимокУзлов.md) или [УпорядоченныйСнимокУзлов](ТипРезультатаDOMXPath.УпорядоченныйСнимокУзлов.md), то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
