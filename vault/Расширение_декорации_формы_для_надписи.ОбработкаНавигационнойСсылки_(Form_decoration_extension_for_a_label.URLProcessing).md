Расширение декорации формы для надписи (Form decoration extension for a label)

ОбработкаНавигационнойСсылки (URLProcessing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОбработкаНавигационнойСсылки(<НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: [Строка](lang__def_String.md).   
Значение гиперссылки форматированной строки.  
Передается по ссылке.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при нажатии на гиперссылку форматированной строки, расположенной в элементе, у которого свойство Гиперссылка имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.1.
