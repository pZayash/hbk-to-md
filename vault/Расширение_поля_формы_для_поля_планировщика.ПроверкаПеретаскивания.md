Расширение поля формы для поля планировщика (Form field extension for a planner)

ПроверкаПеретаскивания (DragCheck)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПроверкаПеретаскивания(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Дата>, <ЗначенияИзмерений>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания.md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

<Дата>

Тип: Дата.   
Дата, над которой находится курсор мыши.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие.md).   
Коллекция значений измерений. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение.md) измерения, в качестве значения выступает [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md) элемента измерения.

Описание:

Возникает при движении курсора в поле-приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
