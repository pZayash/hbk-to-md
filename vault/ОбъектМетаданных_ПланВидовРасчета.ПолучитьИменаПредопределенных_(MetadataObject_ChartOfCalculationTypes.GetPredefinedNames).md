ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ПолучитьИменаПредопределенных (GetPredefinedNames)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИменаПредопределенных()

Описание:

Возвращает имена предопределенных, заданные в конфигурации в виде фиксированного массива из имен предопределенных типа [Строка](lang__def_String.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
