КоллекцияКолонокДереваЗначений (ValueTreeColumnCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<НаименованиеКолонки>)

Параметры:

<НаименованиеКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Наименование колонки дерева значений.

Возвращаемое значение:

Тип: [КолонкаДереваЗначений](КолонкаДереваЗначений_(ValueTreeColumn).md), [Неопределено](lang__def_Undefined.md).   
Если указанная колонка отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск колонки дерева значений по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // ДеревоСтраниц (дерево значений) содержит коллекцию страниц отчета  Для каждого ТекКол Из ПользовательскиеНастройки.Колонки Цикл     Если ДеревоСтраниц.Колонки.Найти(ТекКол.Имя) = Неопределено Тогда         // удаляем отсутствующую колонку          Индекс = ПользовательскиеНастройки.Колонки.Индекс(ТекКол);         ПользовательскиеНастройки.Колонки.Удалить(Индекс);     КонецЕсли; КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
