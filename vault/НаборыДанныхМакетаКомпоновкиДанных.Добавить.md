НаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSets)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [НаборДанныхЗапросМакетаКомпоновкиДанных](НаборДанныхЗапросМакетаКомпоновкиДанных.md), [НаборДанныхОбъектМакетаКомпоновкиДанных](НаборДанныхОбъектМакетаКомпоновкиДанных.md), [НаборДанныхОбъединениеМакетаКомпоновкиДанных](НаборДанныхОбъединениеМакетаКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [НаборДанныхЗапросМакетаКомпоновкиДанных](НаборДанныхЗапросМакетаКомпоновкиДанных.md), [НаборДанныхОбъектМакетаКомпоновкиДанных](НаборДанныхОбъектМакетаКомпоновкиДанных.md), [НаборДанныхОбъединениеМакетаКомпоновкиДанных](НаборДанныхОбъединениеМакетаКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод вернет Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
