XBase (XBase)

НайтиПоКлючу (FindByKey)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКлючу(<Режим>)

Параметры:

<Режим> (обязательный)

Тип: Строка.   
Режим поиска записей:

- "=" - ищет запись на точное соответствие ключу;
- ">=" - ищет запись на точное соответствие с ключом или большую;
- ">" - ищет запись с большим ключом;
- "<=" - ищет запись на точное соответствие с ключом или меньшую;
- "<" - ищет запись с меньшим ключом.

Возвращаемое значение:

Тип: Булево.   
Истина - запись найдена; Ложь в противном случае.

Описание:

Осуществляет поиск записи, соответствующей значениям свойств [Ключ](XBase.Ключ.md) по текущему индексу и режиму поиска, переданному в качестве параметра. Указатель устанавливается на найденную запись. Перед вызовом метода следует установить значения всех свойств объекта типа [Ключ](XBase.Ключ.md), которые участвуют в вычислении выражения текущего индекса.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Поиск по составному ключу  xB.ТекущийИндекс = xB.Индексы.IDXCNAME; // индекс CODE + TRIM(NAME)  xB.Ключ.CODE = "00001"; xB.Ключ.NAME = "Системный блок"; ЗаписьНайдена = xB.НайтиПоКлючу("="); |
