# Расширение поля формы для поля диаграммы.Высота

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля диаграммы](Расширение_поля_формы_для_поля_диаграммы_(Form_field_extension_for_a_chart_field).md) › [Свойства](Расширение_поля_формы_для_поля_диаграммы__Свойства.md)

Расширение поля формы для поля диаграммы (Form field extension for a chart field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту поля диаграммы. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
