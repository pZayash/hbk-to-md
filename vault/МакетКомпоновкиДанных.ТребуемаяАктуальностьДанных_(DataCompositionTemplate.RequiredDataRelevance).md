МакетКомпоновкиДанных (DataCompositionTemplate)

ТребуемаяАктуальностьДанных (RequiredDataRelevance)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [ТребуемаяАктуальностьДанных](ТребуемаяАктуальностьДанных_(RequiredDataRelevance).md).   
Актуальность данных, которые необходимо получить.  
Если свойство [ИспользованиеКопийБазыДанных](Запрос.ИспользованиеКопийБазыДанных_(Query.DatabaseCopiesUse).md) имеет значение [ИспользоватьТолькоКопии](ИспользованиеКопийБазыДанных.ИспользоватьТолькоКопии_(DatabaseCopiesUse.UseCopiesOnly).md), то свойство может влиять только на то, какая именно копия будет выбрана.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
