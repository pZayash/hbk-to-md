Соединение с агентом сервера (IServerAgentConnection)

UpdateInfoBase (UpdateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateInfoBase(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должны быть изменены параметры информационной базы.

<ИнформационнаяБаза> (обязательный)

Тип: [Описание информационной базы](Описание_информационной_базы.md).   
Новое описание информационной базы. Изменено может быть только значение свойства [Descr](Описание_информационной_базы.Descr.md).

Описание:

Позволяет изменить параметры информационной базы, зарегистрированной в кластере. Изменять можно только словесное описание информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
