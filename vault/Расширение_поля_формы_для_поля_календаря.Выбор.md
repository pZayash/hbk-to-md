Расширение поля формы для поля календаря (Form field extension for a calendar field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<ВыбраннаяДата>)

Параметры:

<ВыбраннаяДата>

Тип: Дата.   
Выбранная дата.

Описание:

Возникает при выборе даты.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
