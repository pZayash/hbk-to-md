Свойства:

[ПараметрТекущаяСтрока (CurrentLineParameter)](Расширение_формы_списка_плана_счетов.ПараметрТекущаяСтрока.md)  

Описание:

Дополнительные свойства объекта [Форма](Форма.md) в том случае, если основной реквизит формы имеет тип [ПланСчетовСписок.](ПланСчетовСписок.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСписок.Имя_плана_счетов.md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |

Использование в версии:

Доступен, начиная с версии 8.0.
