Модуль команды (Command module)

ОбработкаКоманды (CommandProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаКоманды(<ПараметрКоманды>, <ПараметрыВыполненияКоманды>)

Параметры:

<ПараметрКоманды>

Тип: Произвольный.   
В параметр передается значение от источника, в котором реализована команда. Может содержать как единичное значение, так и массив значений.

<ПараметрыВыполненияКоманды>

Тип: [ПараметрыВыполненияКоманды](ПараметрыВыполненияКоманды_(CommandExecuteParameters).md).   
В обработчике команды можно изменить значение свойств параметра [Окно](ПараметрыВыполненияКоманды.Окно_(CommandExecuteParameters.Window).md) и [Уникальность](ПараметрыВыполненияКоманды.Уникальность_(CommandExecuteParameters.Uniqueness).md).

Описание:

В теле процедуры-обработчике выполняются действия по обработке выполнения команды.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
