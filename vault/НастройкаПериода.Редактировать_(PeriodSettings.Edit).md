НастройкаПериода (PeriodSettings)

Редактировать (Edit)

Доступен, начиная с версии 8.0.

Синтаксис:

Редактировать()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - нажата кнопка "ОК" (выбор сделан); [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает диалог ввода периода в модальном режиме. В зависимости от значения свойств [РедактироватьКакИнтервал](НастройкаПериода.РедактироватьКакИнтервал_(PeriodSettings.EditAsInterval).md) и [РедактироватьКакПериод](НастройкаПериода.РедактироватьКакПериод_(PeriodSettings.EditAsPeriod).md) диалог визуальной настройки имеет обе закладки или одну из них. Если использовано обе закладки активность той или иной из них задается свойством [ВариантНастройки](НастройкаПериода.ВариантНастройки_(PeriodSettings.SettingsMode).md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод; НастройкаПериода.Редактировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
