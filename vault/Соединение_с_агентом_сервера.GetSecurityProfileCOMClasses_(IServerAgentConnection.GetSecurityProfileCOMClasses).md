Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfileCOMClasses (GetSecurityProfileCOMClasses)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfileCOMClasses(<Кластер>, <Профиль>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должен быть получен список COM-классов.

<Профиль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности, которому принадлежит список.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [COM-класс](COM-класс_(ISecurityProfileCOMClass).md).

Описание:

Получает разрешенные COM-классы.

Доступность:

Интеграция.

Примечание:

Если профиль с этим именем не найден, то исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
