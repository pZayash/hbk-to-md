МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьДоступностьИспользованияЛицензийДляРазработчика (GetDeveloperLicenseUsageAvailability)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДоступностьИспользованияЛицензийДляРазработчика()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если в диалоге получения лицензии должно быть показано предложение активировать лицензию для разработчиков. Для этого должны быть выполнены следующие условия:

- Используются региональные настройки для Российской Федерации,
- Доступны ресурсы фирмы 1С (Центр лицензирования и developer.1c.ru).

[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, доступно ли в диалоге получения лицензии получение лицензии для разработчиков.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
