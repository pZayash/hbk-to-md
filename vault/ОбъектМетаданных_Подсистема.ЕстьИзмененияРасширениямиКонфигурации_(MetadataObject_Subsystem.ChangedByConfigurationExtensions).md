ОбъектМетаданных: Подсистема (MetadataObject: Subsystem)

ЕстьИзмененияРасширениямиКонфигурации (ChangedByConfigurationExtensions)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЕстьИзмененияРасширениямиКонфигурации()

Описание:

Возвращает [Истина](lang__def_BooleanTrue.md), если сам объект или один из его дочерних объектов изменен (в том числе, и добавлением нового дочернего объекта) расширениями в текущем сеансе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
