АдминистрированиеКластер (AdministrationCluster)

ПолучитьОграниченияПотребленияРесурсов (GetResourceConsumptionLimits)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьОграниченияПотребленияРесурсов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеОграничениеПотребленияРесурсов](АдминистрированиеОграничениеПотребленияРесурсов_(AdministrationResourceConsumptionLimit).md).

Описание:

Получает массив ограничений потребления ресурсов с пустыми значениями свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
