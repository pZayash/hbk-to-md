ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

РазрешитьИтогиСрезПервых (EnableTotalsSliceFirst)

Доступен, начиная с версии 8.3.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - при обновлении данных набора записей регистра сведений будут обновлены записи таблицы итогов для среза первых.  
Обновление записей таблицы итогов для среза первых может быть отключено с помощью метода [УстановитьИспользованиеИтогов](РегистрСведенийМенеджер.Имя_регистра_сведений.УстановитьИспользованиеИтогов_(InformationRegisterManager.Information_register_name.SetTotalsUsing).md).  
Запросы по срезу будут выполняться по дополнительной таблице "Срез первых" за исключением случаев, когда параметры среза содержат условия, заданные по полям, отличным от измерений и разделителей, или когда в правилах ограничения доступа к данным используются поля, отличные от измерений и разделителей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.2.
