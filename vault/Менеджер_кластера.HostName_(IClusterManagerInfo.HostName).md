Менеджер кластера (IClusterManagerInfo)

HostName (HostName)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя или адрес рабочего сервера, на котором должен быть запущен процесс rmngr менеджера кластера. Должен совпадать с именем или адресом одного из рабочих серверов, определенных в кластере. Изменение значения допустимо только для нового дополнительного менеджера кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
