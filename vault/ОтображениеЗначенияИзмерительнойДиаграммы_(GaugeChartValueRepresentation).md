Описание:

Определяет способ отображения значений в измерительной диаграмме.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GaugeChartValueRepresentation.

См. также:

[Диаграмма](Диаграмма_(Chart).md), свойство [ОтображениеЗначенияИзмерительнойДиаграммы](Диаграмма.ОтображениеЗначенияИзмерительнойДиаграммы_(Chart.GaugeChartValueRepresentation).md)

Доступен, начиная с версии 8.0.
