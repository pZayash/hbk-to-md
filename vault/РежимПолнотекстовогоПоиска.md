Доступен, начиная с версии 8.2.

Значения

[Запретить (Disable)](РежимПолнотекстовогоПоиска.Запретить.md)  
[Разрешить (Enable)](РежимПолнотекстовогоПоиска.Разрешить.md)  

Описание:

Разрешает (запрещает) все операции полнотекстового поиска: обновление индексов, очистка индексов, поиск.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
