# ЯчейкиТаблицыОбластиКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макеты областей](Макеты_областей.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЯчейкаТаблицыОбластиКомпоновкиДанных](ЯчейкаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableCell).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ячейки таблицы области компоновки данных.  
Возможно обращение к ячейке таблицы области компоновки данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция ячеек строки таблицы.   
Ячейкой является прямоугольная область таблицы, содержащая внутри себя выводимые данные и их оформление.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
