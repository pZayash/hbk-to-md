ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [РазмерКартинки](РазмерКартинки_(PictureSize).md).   
Свойство используется, чтобы подогнать размеры картинки под размеры ячейки. Если установлено значение [Черепица](РазмерКартинки.Черепица_(PictureSize.Tile).md) или [Растянуть](РазмерКартинки.Растянуть_(PictureSize.Stretch).md), то значение свойства [ПоложениеТекстаОтносительноКартинки](ОбластьЯчеекТабличногоДокумента.ПоложениеТекстаОтносительноКартинки_(SpreadsheetDocumentRange.TextPositionRelativeToPicture).md) игнорируется и считается равным [Поверх](ПоложениеТекстаОтносительноКартинки.Поверх_(TextPositionRelativeToPicture.OnTop).md). Значение по умолчанию – [Пропорционально](РазмерКартинки.Пропорционально_(PictureSize.Proportionally).md).  
Значение [АвтоРазмерБезУчетаМасштаба](РазмерКартинки.АвтоРазмерБезУчетаМасштаба_(PictureSize.AutoSizeIgnoreScale).md) интерпретируется как [АвтоРазмер](РазмерКартинки.АвтоРазмер_(PictureSize.AutoSize).md).  
Значение [РеальныйРазмерБезУчетаМасштаба](РазмерКартинки.РеальныйРазмерБезУчетаМасштаба_(PictureSize.RealSizeIgnoreScale).md) интерпретируется как [РеальныйРазмер](РазмерКартинки.РеальныйРазмер_(PictureSize.RealSize).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.16.
