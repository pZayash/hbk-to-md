Глобальный контекст (Global context)

ЧислоИзДвоичнойСтроки (NumberFromBinaryString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЧислоИзДвоичнойСтроки(<ДвоичнаяСтрока>)

Параметры:

<ДвоичнаяСтрока> (обязательный)

Тип: [Строка](lang__def_String.md).   
Двоичная строка.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Преобразует двоичную строку в значение типа [Число](lang__def_Number.md).  
Двоичная строка должна начинаться с префикса "0b", после которого следуют двоичные цифры (одна или больше).   
Поддерживаются только натуральные числа.  
Примеры корректных строк: "0b0011001", "0b101001111".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
