ПолеВыбора (ComboBox)

КнопкаОткрытия (OpenButton)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак наличия/отсутствия кнопки открытия значения поля.  
Истина - кнопка есть.

Доступность:

Толстый клиент.

Примечание:

Стандартной обработки нажатия на кнопку открытия значения не предусмотрено. Необходимые действия можно провести в обработчике события Открытие.

Использование в версии:

Доступен, начиная с версии 8.0.
