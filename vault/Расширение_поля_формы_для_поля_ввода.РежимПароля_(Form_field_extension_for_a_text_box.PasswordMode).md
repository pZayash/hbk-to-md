# Расширение поля формы для поля ввода.РежимПароля

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля ввода](Расширение_поля_формы_для_поля_ввода_(Form_field_extension_for_a_text_box).md) › [Свойства](Расширение_поля_формы_для_поля_ввода__Свойства.md)

Расширение поля формы для поля ввода (Form field extension for a text box)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак специального режима для ввода пароля, в котором все вводимые символы отображаются символом "\*" (звездочка).  
Если тип значения [Неопределено](lang__def_Undefined.md), то поведение осуществляется в соответствии с настройками конфигурации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
