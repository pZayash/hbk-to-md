ПолучитьХранилищаДвоичныхДанных (GetBinaryDataStorages)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьХранилищаДвоичныхДанных()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
