Описание:

Тип записи сообщений NDEF, использующийся для хранения и передачи данных в нестандартном формате.   
Записи внешнего типа следует использовать в случае, когда поставленная задача не решается при помощи записей стандартных типов ([ТекстоваяЗаписьNDEF](ТекстоваяЗаписьNDEF.md), [URIЗаписьNDEF](URIЗаписьNDEF.md), [МедиаЗаписьNDEF](МедиаЗаписьNDEF.md)).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
