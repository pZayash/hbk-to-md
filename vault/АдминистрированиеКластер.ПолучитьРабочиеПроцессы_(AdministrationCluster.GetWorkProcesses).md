АдминистрированиеКластер (AdministrationCluster)

ПолучитьРабочиеПроцессы (GetWorkProcesses)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьРабочиеПроцессы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеРабочийПроцесс](АдминистрированиеРабочийПроцесс_(AdministrationWorkProcess).md).

Описание:

Получает массив рабочих процессов, зарегистрированных на заданном рабочем сервере заданного кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
