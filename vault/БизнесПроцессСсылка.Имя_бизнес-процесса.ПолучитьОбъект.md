БизнесПроцессСсылка.<Имя бизнес-процесса> (BusinessProcessRef.<Business process name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект для чтения, изменения, добавления и удаления бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Объект = ВыбБизнесПроцесс.ПолучитьОбъект(); |

Использование в версии:

Доступен, начиная с версии 8.0.
