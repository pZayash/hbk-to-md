АдминистрированиеРабочийСервер (AdministrationWorkServer)

КоличествоСоединенийНаПроцесс (SingleProcessConnectionsCount)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит ограничение количества соединений с информационными базами через один рабочий процесс. При достижении этого количества на рабочий процесс не могут назначаться новые соединения с информационными базами.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
