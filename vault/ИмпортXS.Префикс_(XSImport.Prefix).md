ИмпортXS (XSImport)

Префикс (Prefix)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Префикс для URI пространства имен импортируемой схемы. Не несет информационной нагрузки, используется только при генерации XML документа для объектной модели схемы XML.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
