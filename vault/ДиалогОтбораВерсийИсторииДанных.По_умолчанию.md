По умолчанию

Синтаксис:

Параметры:

<Данные> (обязательный)

Тип: [СправочникСсылка.](СправочникСсылка.Имя_справочника.md)<[Имя справочника](СправочникСсылка.Имя_справочника.md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>.   
Данные, по которым отбираются версии. Используется для формирования доступных для отбора полей и пользователей-авторов версий.

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Используется для установки параметров отбора при открытии диалога.  
Структура идентична структуре, содержащейся в параметре <Отбор> метода [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии.md).  
Если диалог был закрыт пользователем нажатием кнопки "OK", содержит параметры отбора, установленные пользователем.  
Значение по умолчанию: пустая структура.  
Значение по умолчанию: [Структура](Структура.md).

Описание:

Создает диалог отбора версий указанных данных.

Использование в версии:

Доступен, начиная с версии 8.3.13.
