# HTTPОтвет.ПолучитьТелоКакДвоичныеДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Интернет](Интернет-2.md) › [Работа с HTTP](Работа_с_HTTP.md) › [HTTPОтвет](HTTPОтвет_(HTTPResponse).md) › [Методы](HTTPОтвет__Методы.md)

HTTPОтвет (HTTPResponse)

ПолучитьТелоКакДвоичныеДанные (GetBodyAsBinaryData)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьТелоКакДвоичныеДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - если при вызове методов [ОтправитьДляОбработки](HTTPСоединение.ОтправитьДляОбработки_(HTTPConnection.Post).md) или [Получить](HTTPСоединение.Получить_(HTTPConnection.Get).md) был установлен параметр <ИмяВыходногоФайла>.

Описание:

Получает тело HTTP-ответа сервера в виде объекта двоичных данных.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
