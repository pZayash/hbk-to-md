ЭлементыФормы (Controls)

ДобавитьActiveX (AddActiveX)

Доступен, начиная с версии 8.0.

Синтаксис:

ДобавитьActiveX(<Идентификатор>, <Имя>, <Видимость>, <Панель>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, идентифицирующая ActiveX. (ProgID).

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор создаваемого элемента управления. По имени объекта возможно получить доступ к создаваемому данным методом элементу управления.

<Видимость> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, создавать новый элемент управления формы видимым или нет. [Истина](lang__def_BooleanTrue.md) - элемент создается видимым.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Панель> (необязательный)

Тип: [Панель](Панель_(Panel).md).   
Определяет, какой именно панели формы принадлежит создаваемый элемент. Если параметр не указан, создаваемый элемент будет принадлежать непосредственно форме.

Описание:

Добавляет в форму объект ActiveX.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
