Описание:

Может быть получен с помощью метода [GetWorkingProcesses](Соединение_с_агентом_сервера.GetWorkingProcesses_(IServerAgentConnection.GetWorkingProcesses).md).  
Представляет собой объект с интерфейсом ISrvrProcessInfo. Содержит параметры рабочего процесса.
