ФорматированныйДокумент (FormattedDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который будет записан форматированный документ.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента.md).   
Указывает тип файла.

Описание варианта метода:

Запись форматированного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи документа.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента.md).   
Указывает формат записи документа.

Описание варианта метода:

Запись форматированного документа в поток.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания:

- Истина - если запись была инициирована с помощью расширения браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
- Ложь - если запись была инициирована с помощью расширения браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
- Неопределено - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

.

Описание:

Асинхронная функция. Сохраняет форматированный документ в файл.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
