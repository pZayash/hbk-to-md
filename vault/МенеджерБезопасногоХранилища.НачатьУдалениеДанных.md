МенеджерБезопасногоХранилища (SecureStorageManager)

НачатьУдалениеДанных (BeginDeleteData)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьУдалениеДанных(<Ключ>, <Обработчик>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Ключ, под которым были сохранены данные.

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения удаления данных со следующими параметрами:

- <Ключ> - тип Строка. Переданный ключ.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

При возникновении ошибки вызывается обработчик ошибки.

Возвращаемое значение:

Если переданный ключ не существует, будет сгенерировано сообщение об ошибке.

Описание:

Удаляет данные из защищенного хранилища.  
Вызов метода не блокирует исполнение кода языка, все результаты возвращаются в переданный обработчик оповещения.   
Выполняемая операция не является интерактивной и не может быть отменена пользователем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерБезопасногоХранилища](МенеджерБезопасногоХранилища.md), метод [УдалитьДанныеАсинх](МенеджерБезопасногоХранилища.УдалитьДанныеАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
