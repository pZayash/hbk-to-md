ИнтернетПочта (InternetMail)

СоздатьПочтовыйЯщик (CreateMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

СоздатьПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя почтового ящика. Может содержать иерархию. В качестве разделителя используется символ, который доступен через свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md). В большинстве случаев это символ '/'.

Описание:

Создает IMAP почтовый ящик по заданному имени. Ящик создаётся от корня. Имя может содержать иерархический путь к почтовому ящику.   
Символ-разделитель иерархии почтовых ящиков, используемый текущим почтовым сервером, доступен с помощью свойства [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Работает только при подключении, установленном по протоколу IMAP.

Пример:

|  |
| --- |
| Почта.СоздатьПочтовыйЯщик("inbox/maibox1/maibox2"); |

См. также:

[ИнтернетПочта](ИнтернетПочта.md), свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
