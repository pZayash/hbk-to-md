# Расширение табличного поля списка плана счетов.ПередНачаломДобавления

Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>, <Родитель>, <ЭтоГруппа>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то строка списка плана счетов добавлена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

<Родитель>

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   
Ссылка на элемент, который будет использован при добавлении в качестве родителя.

<ЭтоГруппа>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, что добавляется - группа или счет. [Истина](lang__def_BooleanTrue.md) - добавляется группа.

Описание:

Возникает перед добавлением строки списка плана счетов. Данное событие для табличного поля плана счетов возникает "вместо" одноименного события объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

Доступность:

Толстый клиент.

Примечание:

Расширяется набор параметров по сравнению с одноименным событием объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

См. также:

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [ПередНачаломДобавления](ТабличноеПоле.ПередНачаломДобавления_(TableBox.BeforeAddRow).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
