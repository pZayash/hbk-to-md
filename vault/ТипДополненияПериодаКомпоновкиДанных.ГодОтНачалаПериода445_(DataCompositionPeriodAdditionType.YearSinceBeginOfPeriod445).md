# ТипДополненияПериодаКомпоновкиДанных.ГодОтНачалаПериода445

ТипДополненияПериодаКомпоновкиДанных (DataCompositionPeriodAdditionType)

ГодОтНачалаПериода445 (YearSinceBeginOfPeriod445)

Доступен, начиная с версии 8.3.12.

Описание:

Тип дополнения периода "Год", которое будет отсчитываться от начальной даты периода дополнения.  
При этом считается, что размер первого и второго месяца равен четырем неделям, третьего - пяти неделям, четвертого и пятого - четырем неделям, для шестого - пяти, и т.д.

Использование в версии:

Доступен, начиная с версии 8.3.12.
