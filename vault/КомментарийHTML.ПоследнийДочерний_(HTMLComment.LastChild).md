КомментарийHTML (HTMLComment)

ПоследнийДочерний (LastChild)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КомментарийHTML](КомментарийHTML_(HTMLComment).md), [Неопределено](lang__def_Undefined.md).   
Последний дочерний узел HTML или [Неопределено](lang__def_Undefined.md), если дочерних узлов нет.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
