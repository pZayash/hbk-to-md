Доступен, начиная с версии 8.3.10.

Описание:

Определяет необходимость использования хранилища сертификатов в данных приложения. Используется на мобильной платформе.  
В "1С:Предприятии" для персонального компьютера реализовано для совместимости и содержит пустое хранилище сертификатов.

Использование в версии:

Доступен, начиная с версии 8.3.10.
