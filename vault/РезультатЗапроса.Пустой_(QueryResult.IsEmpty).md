РезультатЗапроса (QueryResult)

Пустой (IsEmpty)

Доступен, начиная с версии 8.0.

Синтаксис:

Пустой()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - нет ни одной записи; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, есть ли в результате записи (без учета общих итогов).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если Запрос.Выполнить().Пустой() Тогда     Возврат Ложь; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
