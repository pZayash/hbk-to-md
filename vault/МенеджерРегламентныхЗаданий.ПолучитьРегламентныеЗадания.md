МенеджерРегламентныхЗаданий (ScheduledJobsManager)

ПолучитьРегламентныеЗадания (GetScheduledJobs)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьРегламентныеЗадания(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Структура, задающая отбор. Значениями структуры могут быть: [УникальныйИдентификатор](РегламентноеЗадание.УникальныйИдентификатор.md), [Ключ](РегламентноеЗадание.Ключ.md), [Метаданные](РегламентноеЗадание.Метаданные.md), [Предопределенное](РегламентноеЗадание.Предопределенное.md), [Использование](РегламентноеЗадание.Использование.md), [Наименование](РегламентноеЗадание.Наименование.md). Если отбор не задан, получаются все регламентные задания.  
Если выполняется отбор по метаданным, то в качестве значения [Метаданные](РегламентноеЗадание.Метаданные.md) можно указать либо объект метаданных регламентного задания, либо его имя.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Получает массив регламентных заданий по заданному отбору. Получение регламентных заданий возможно только для администратора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
