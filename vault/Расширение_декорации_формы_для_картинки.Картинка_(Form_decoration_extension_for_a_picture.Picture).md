# Расширение декорации формы для картинки.Картинка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Декорация формы](Декорация_формы.md) › [Расширение декорации формы для картинки](Расширение_декорации_формы_для_картинки_(Form_decoration_extension_for_a_picture).md) › [Свойства](Расширение_декорации_формы_для_картинки__Свойства.md)

Расширение декорации формы для картинки (Form decoration extension for a picture)

Картинка (Picture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Отображаемая картинка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.2.
