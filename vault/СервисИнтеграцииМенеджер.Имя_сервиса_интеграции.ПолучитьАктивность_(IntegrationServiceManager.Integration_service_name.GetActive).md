# СервисИнтеграцииМенеджер.<Имя сервиса интеграции>.ПолучитьАктивность

СервисИнтеграцииМенеджер.<Имя сервиса интеграции> (IntegrationServiceManager.<Integration service name>)

ПолучитьАктивность (GetActive)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьАктивность()

Возвращаемое значение:

Тип: Булево.   

Описание:

При получении активности сервиса интеграции выполняются следующие действия:

- Если в сервисе интеграции определены разделители:
  - Генерируется исключение, если выполняется одно из перечисленных условий:
    - В сеансе используется разделитель, который не разделяет сервис интеграции.
    - В сеансе не используется разделитель, который разделяет сервис интеграции.
  - Если для области данных установлена активность сервиса, то возвращается активность сервиса интеграции для области данных, в противном случае - [Ложь](lang__def_BooleanFalse.md) .
- Если в сервисе интеграции не определены разделители:
  - Генерируется исключение, если в сеансе используется любой разделитель.
  - Если в информационной базе установлена активность сервиса, то возвращается активность сервиса интеграции для информационной базы, иначе - [Ложь](lang__def_BooleanFalse.md).

При выполнении метода проверяется наличие следующих прав:

- Если в сервисе используются разделители - "Администрирование";
- Если в сервисе не используются разделители - "Администрирование данных".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
