ОформлениеГруппировкиДиаграммыОбластиКомпоновкиДанных (DataCompositionAreaTemplateChartGroupAppearance)

НайтиЗначениеПараметра (FindParameterValue)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиЗначениеПараметра(<Параметр>)

Параметры:

<Параметр> (необязательный)

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md).   
Параметр, значение которого нужно найти.

Возвращаемое значение:

Тип: [ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных_(DataCompositionParameterValue).md), [Неопределено](lang__def_Undefined.md).   
Если значение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск значения параметра.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
