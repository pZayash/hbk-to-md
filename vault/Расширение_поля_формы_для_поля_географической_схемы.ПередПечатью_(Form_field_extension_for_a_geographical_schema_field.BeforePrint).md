Расширение поля формы для поля географической схемы (Form field extension for a geographical schema field)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати_(PrintDialogUseMode).md).   
Способ вызова печати:

- [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать_(PrintDialogUseMode.DontUse).md) - если выбрана команда "Печать с текущими настройками",
- [Использовать](РежимИспользованияДиалогаПечати.Использовать_(PrintDialogUseMode.Use).md) - если выбрана команда "Печать".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то печать выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
