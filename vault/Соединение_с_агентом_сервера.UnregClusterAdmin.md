Соединение с агентом сервера (IServerAgentConnection)

UnregClusterAdmin (UnregClusterAdmin)

Доступен, начиная с версии 8.1.

Синтаксис:

UnregClusterAdmin(<Кластер>, <Имя>)

Параметры:

<Кластер> (обязательный)

Тип: [Администратор](Администратор.md).   
Кластер серверов, в котором необходимо отменить регистрацию администратора.

<Имя> (обязательный)

Тип: Строка.   
Имя администратора, который должен быть удален. Если администратора с таким именем в кластере не зарегистрировано, то никакие действия не выполняются.

Описание:

Отменяет регистрацию администратора кластера. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
