ГруппаФормы (FormGroup)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак возможности/невозможности редактирования отображаемых данных. Установка данного свойства в [Истина](lang__def_BooleanTrue.md) приведет все дочерние элементы в состояние запрещения редактирования вне зависимости от значения их свойств ТолькоПросмотр.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
