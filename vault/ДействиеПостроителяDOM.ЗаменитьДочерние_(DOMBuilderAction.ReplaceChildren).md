ДействиеПостроителяDOM (DOMBuilderAction)

ЗаменитьДочерние (ReplaceChildren)

Доступен, начиная с версии 8.1.

Описание:

Заменить дочерние узлы. Результат чтения заменяет дочерние узлы контекста документа. В качестве узла контекста могут выступать: [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) или [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md).

Использование в версии:

Доступен, начиная с версии 8.1.
