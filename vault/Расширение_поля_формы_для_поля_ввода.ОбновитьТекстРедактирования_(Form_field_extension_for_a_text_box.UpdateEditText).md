# Расширение поля формы для поля ввода.ОбновитьТекстРедактирования

Расширение поля формы для поля ввода (Form field extension for a text box)

ОбновитьТекстРедактирования (UpdateEditText)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбновитьТекстРедактирования()

Описание:

Обновляет текст в поле ввода в соответствии со значением, которое содержится в данных, редактируемых в поле ввода. При этом не предпринимается попыток формирования значения по имеющемуся в поле ввода тексту.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не зависит от режима совместимости.

Использование в версии:

Доступен, начиная с версии 8.2.
