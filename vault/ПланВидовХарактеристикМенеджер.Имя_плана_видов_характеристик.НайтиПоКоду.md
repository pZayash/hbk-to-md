Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>, <Родитель>)

Параметры:

<Код> (обязательный)

Тип: Строка.   
Искомый код.

<Родитель> (необязательный)

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md).   
Родитель, в пределах которого нужно выполнять поиск Если не указано, то во всем плане видов характеристик.

Возвращаемое значение:

Тип: [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, Неопределено.   
Если не существует ни одного элемента с требуемым кодом, то будет возвращена пустая ссылка.  
Если код не задан (длина = 0), то будет возвращено Неопределено.

Описание:

Осуществляет поиск элемента по его коду.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
