ТабличныйДокумент (SpreadsheetDocument)

Скрыть (Hide)

Доступен, начиная с версии 8.1.

Синтаксис:

Скрыть()

Описание:

Скрывает окно табличного документа.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Пример:

|  |
| --- |
| ТабДок.Скрыть(); |

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [Показать](ТабличныйДокумент.Показать_(SpreadsheetDocument.Show).md)

Доступен, начиная с версии 8.1.
