ЭлементАплетHTML (HTMLAppletElement)

Код (Code)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя файла класса, содержащего скомпилированный подкласс аплета или путь, по которому можно получить класс вместе с файлом класса.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
