ФиксированныйМассив (FixedArray)

ВГраница (UBound)

Доступен, начиная с версии 8.0.

Синтаксис:

ВГраница()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает наибольший индекс элемента массива. Соответствует Количество()-1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Рег = РегистрРасчета.ОсновныеНачисленияСотрудниковОрганизации; Для К = 0 По Рег.ФактическийПериодДействия.ВГраница() Цикл     // обработка элемента фиксированного массива   КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
