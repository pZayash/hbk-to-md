Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ВытесняющиеВидыРасчетаСтрока.\<Имя плана видов расчета\>](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(DisplacingCalculationTypesRow.Chart_of_calculation_types_name).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
