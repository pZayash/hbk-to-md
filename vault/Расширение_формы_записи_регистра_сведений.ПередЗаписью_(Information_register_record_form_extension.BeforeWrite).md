# Расширение формы записи регистра сведений.ПередЗаписью

Расширение формы записи регистра сведений (Information register record form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи объекта. Процедура-обработчик вызывается перед вызовом процедуры обработчика события [ПередЗаписью](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.ПередЗаписью_(InformationRegisterRecordSet.Information_register_name.BeforeWrite).md) объекта [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
