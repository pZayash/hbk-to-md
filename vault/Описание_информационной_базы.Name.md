Описание информационной базы (IInfoBaseShort)

Name (Name)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит ссылочное имя информационной базы. Совпадает со значением свойства [Name](Информационная_база.Name.md). Должно быть уникальным среди всех информационных баз, зарегистрированных в данном реестре. При сравнении регистр букв не имеет значения.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
