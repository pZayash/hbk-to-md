АдминистрированиеРабочийСервер (AdministrationWorkServer)

ДобавитьДиапазонПортов (AddPortRange)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ДобавитьДиапазонПортов(<ДиапазонПортов>)

Параметры:

<ДиапазонПортов> (необязательный)

Тип: [АдминистрированиеДиапазонПортов](АдминистрированиеДиапазонПортов_(AdministrationPortRange).md).   
Диапазон IP-портов.

Описание:

Добавляет диапазон портов рабочего сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
