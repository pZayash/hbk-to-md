ИнтернетПочтовоеСообщение (InternetMailMessage)

УведомитьОПрочтении (RequestReadReceipt)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Уведомить о прочтении почтового сообщения. Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).  
[Истина](lang__def_BooleanTrue.md) - уведомление необходимо.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
