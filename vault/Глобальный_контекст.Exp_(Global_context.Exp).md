Глобальный контекст (Global context)

Exp (Exp)

Доступен, начиная с версии 8.0.

Синтаксис:

Exp(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Аргумент функции.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат возведения в степень.

Описание:

Вычисляет результат возведения основания натурального логарифма (числа e) в степень <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

Если значение параметра настолько велико, что вызывает при вычислении арифметическое переполнение (параметр превышает 709.78271289338397, при этом значение последнего разряда не оказывает влияние на результат), то происходит ошибка "Неправильное значение аргумента встроенной функции (Exp)".

Пример:

|  |
| --- |
| Попытка     r = Exp(709.78271289338397);   //709.782712893      Сообщить("Это предельное значение функции: " + Символы.ПС + R);     r = Exp(709.78271289338398);   //709.782712893  Исключение     Сообщить("При значении аргумента 709.78271289338398"         + " выдается сообщение об ошибке:"         + Символы.ПС + """" + ОписаниеОшибки() + """"); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
