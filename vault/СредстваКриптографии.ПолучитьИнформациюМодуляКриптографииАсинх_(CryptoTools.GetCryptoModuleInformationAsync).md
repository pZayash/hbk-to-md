СредстваКриптографии (CryptoTools)

ПолучитьИнформациюМодуляКриптографииАсинх (GetCryptoModuleInformationAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьИнформациюМодуляКриптографииАсинх(<ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>)

Параметры:

<ИмяМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы MS Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу-библиотеке, реализующей требуемый набор функций.  
Параметр используется, если менеджер создается для операционных систем Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: [Число](lang__def_Number.md).   
Задает тип модуля криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии_(CryptoModuleInformation).md).  
Если модуль криптографии не поддерживается (не найден по имени на Windows, либо не экспортируются требуемые функции на Linux), то результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Получает информацию о модуле криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
