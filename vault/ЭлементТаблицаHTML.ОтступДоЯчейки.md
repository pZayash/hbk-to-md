ЭлементТаблицаHTML (HTMLTableElement)

ОтступДоЯчейки (CellPadding)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Устанавливает величину пространства (по горизонтали и по вертикали) между границей ячейки и содержанием.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию 1.

Использование в версии:

Доступен, начиная с версии 8.2.
