Форма (Form)

Открыта (IsOpen)

Доступен, начиная с версии 8.0.

Синтаксис:

Открыта()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - открыта; [Ложь](lang__def_BooleanFalse.md) - не открыта.

Описание:

Определяет, открыта ли данная форма.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если Не Форма.Открыта() Тогда     Форма.Открыть(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
