МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьОтображениеОповещенийОбсуждения (GetConversationNotificationRepresentation)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьОтображениеОповещенийОбсуждения(<Обсуждение>, <Пользователь>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения, для которого устанавливается режим оповещений.

<Пользователь> (необязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Если задано, то возвращается режим наблюдения для соответствующего пользователя.  
Если передано значение, отличное от идентификатора текущего пользователя, и не установлен привилегированный режим, то генерируется исключение.

Возвращаемое значение:

Тип: [ОтображениеОповещенийСистемыВзаимодействия](ОтображениеОповещенийСистемыВзаимодействия_(CollaborationSystemNotificationRepresentation).md).   

Описание:

Возвращает режим отображения оповещений системы взаимодействия.

Доступность:

Мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
