# ОбластьПостроенияДиаграммы.ЦветФона

ОбластьПостроенияДиаграммы (ChartPlotArea)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона области построения диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ОбластьПостроенияДиаграммы.ПрозрачныйФон_(ChartPlotArea.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
