ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ДопустимыеТипыВходящихЗапросовПоделиться (AllowedIncomingShareRequestTypes)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Список типов данных, перехватываемых мобильным приложением через механизм "Поделиться".  
Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.24.
