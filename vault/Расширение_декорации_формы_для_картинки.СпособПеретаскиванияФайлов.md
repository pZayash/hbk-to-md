Расширение декорации формы для картинки (Form decoration extension for a picture)

СпособПеретаскиванияФайлов (FileDragMode)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [СпособПеретаскиванияФайлов](СпособПеретаскиванияФайлов.md).   
Указывает способ перетаскивания файлов. Если установлен способ [КакФайл](СпособПеретаскиванияФайлов.КакФайл.md), то при перетаскивании файла (файлов) на элемент управления в качестве значения параметра перетаскивания будет передан объект типа [Файл](Файл.md) (массив объектов типа [Файл](Файл.md)), а если установлен способ [КакСсылкаНаФайл](СпособПеретаскиванияФайлов.КакСсылкаНаФайл.md) – объект типа [СсылкаНаФайл](СсылкаНаФайл.md) (массив объектов типа [СсылкаНаФайл](СсылкаНаФайл.md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В веб-клиенте при перетаскивании файла (файлов) как [СсылкаНаФайл](СсылкаНаФайл.md) обработчик события [ПроверкаПеретаскивания](Расширение_поля_формы_для_поля_картинки.ПроверкаПеретаскивания.md) у элементов управления вызывается один раз в момент отпускания кнопки мыши перед обработчиком события [Перетаскивание](Расширение_поля_формы_для_поля_картинки.Перетаскивание.md) в отличие от тонкого клиента, в котором обработчик события [ПроверкаПеретаскивания](Расширение_декорации_формы_для_картинки.ПроверкаПеретаскивания.md) вызывается в процессе перетаскивания файла (файлов).  
В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл.md), то перетаскивание файлов игнорируется, и обработчики событий [ПроверкаПеретаскивания](Расширение_декорации_формы_для_картинки.ПроверкаПеретаскивания.md) и [Перетаскивание](Расширение_поля_формы_для_поля_картинки.Перетаскивание.md) не вызываются.

Использование в версии:

Доступен, начиная с версии 8.3.12.
