ДокументHTML (HTMLDocument)

СоздатьАтрибут (CreateAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

СоздатьАтрибут(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.   
Имя создаваемого атрибута.

Возвращаемое значение:

Тип: [АтрибутHTML](АтрибутHTML.md).   

Описание:

Создает новый узел [АтрибутHTML](АтрибутHTML.md).  
Причины вызова исключений:

- Имя атрибута содержит символы, недопустимые стандартом XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
