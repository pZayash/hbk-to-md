МенеджерКриптографии (CryptoManager)

НачатьПолучениеХранилищаСертификатов (BeginGettingCertificateStore)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеХранилищаСертификатов(<ОписаниеОповещения>, <ТипХранилища>, <РасположениеХранилища>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения установки внешнего компонента со следующими параметрами:

- <Хранилище> - объект типа [ХранилищеСертификатовКриптографии](ХранилищеСертификатовКриптографии_(CryptoCertificateStore).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ТипХранилища> (необязательный)

Тип: [ТипХранилищаСертификатовКриптографии](ТипХранилищаСертификатовКриптографии_(CryptoCertificateStoreType).md).   
Определяет тип хранилища сертификатов, который должен вернуть метод.   
Если параметр не задан, возвращает хранилище, содержащее сертификаты всех доступных типов.

<РасположениеХранилища> (необязательный)

Тип: [РасположениеХранилищаСертификатовКриптографии](РасположениеХранилищаСертификатовКриптографии_(CryptoCertificateStorePlacement).md).   
Определяет, к какому хранилищу предоставить доступ: к хранилищу сертификатов пользователя операционной системы или к хранилищу сертификатов локального компьютера.  
Значение по умолчанию: [ДанныеПользователяОС](РасположениеХранилищаСертификатовКриптографии.ДанныеПользователяОС_(CryptoCertificateStorePlacement.OSUserData).md).

Описание:

Начинает получение хранилища сертификатов, содержащее сертификаты заданного типа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
