ПланСчетовВыборка.<Имя плана счетов> (ChartOfAccountsSelection.<Chart of accounts name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ПланСчетовТабличнаяЧасть.<Имя плана счетов>.<Имя табличной части>.   
Набор свойств содержит табличные части плана счетов. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланСчетовТабличнаяЧасть.<Имя плана счетов>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
