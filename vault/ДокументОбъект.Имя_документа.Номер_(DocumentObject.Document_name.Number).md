# ДокументОбъект.<Имя документа>.Номер

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Номер (Number)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Содержит номер документа. Тип значения зависит от установки в Конфигураторе представления номера документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
