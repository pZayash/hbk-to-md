ИнтернетТекстыПочтовогоСообщения (InternetMailTexts)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<ИндексЭлементаКоллекции>)

Параметры:

<ИндексЭлементаКоллекции> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента коллекции.

Возвращаемое значение:

Тип: [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
