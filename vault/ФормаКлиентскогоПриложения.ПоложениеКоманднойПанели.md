ФормаКлиентскогоПриложения (ClientApplicationForm)

ПоложениеКоманднойПанели (CommandBarLocation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеКоманднойПанелиФормы](ПоложениеКоманднойПанелиФормы.md).   
Определяет положение автоматической командной панели относительно формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
