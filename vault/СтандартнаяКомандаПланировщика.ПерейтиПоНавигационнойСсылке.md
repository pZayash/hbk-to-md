СтандартнаяКомандаПланировщика (PlannerStandardCommand)

ПерейтиПоНавигационнойСсылке (GotoURL)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ПерейтиПоНавигационнойСсылке](СтандартнаяКомандаПоляПланировщика.ПерейтиПоНавигационнойСсылке.md)

Доступен, начиная с версии 8.3.18.

Описание:

Переход к событию [НажатиеНаНавигационнойСсылке](Расширение_поля_формы_для_поля_планировщика.НажатиеНаНавигационнойСсылке.md).  
Используется только для источника вида [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка.md).  
В контекстном меню команда будет иметь представление "Перейти".

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
