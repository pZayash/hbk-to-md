Доступен, начиная с версии 8.0.

Чтение и запись.

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет, которым отображаются надписи в области легенды (подписи к элементам легенды и надписи в масштабной линейке).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
