НаборСхемXML (XMLSchemaSet)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<СхемаXML>)

Параметры:

<СхемаXML> (обязательный)

Тип: [СхемаXML](СхемаXML_(XMLSchema).md).   
Схема XML, добавляемая к набору схем.

Описание:

Добавляет схему в набор.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
