Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управление размером порций выбора данных. Истина - динамический список должен выбирать данные только при необходимости их отображения на экране.

Мобильный клиент, сервер, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
