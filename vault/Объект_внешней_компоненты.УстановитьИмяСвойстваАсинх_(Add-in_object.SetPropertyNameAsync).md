# Объект внешней компоненты.Установить<ИмяСвойства>Асинх

Объект внешней компоненты (Add-in object)

Установить<ИмяСвойства>Асинх (Set<PropertyName>Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Установить<ИмяСвойства>Асинх(<НовоеЗначение>)

Параметры:

<НовоеЗначение> (необязательный)

Тип: Произвольный.   
Устанавливаемое значение свойства.

Возвращаемое значение:

Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронно устанавливает значение свойства внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

[Объект внешней компоненты](Объект_внешней_компоненты_(Add-in_object).md), метод [НачатьУстановку\<ИмяСвойства\>](Объект_внешней_компоненты.НачатьУстановкуИмяСвойства_(Add-in_object.BeginSettingPropertyName).md)
