СерииСлояГеографическойСхемы (GeographicalSchemaLayerSeries)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяСерии>)

Параметры:

<ИмяСерии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомой серии.

Возвращаемое значение:

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы_(GeographicalSchemaLayerDataSeries).md), [Неопределено](lang__def_Undefined.md).   
Если не найдена, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск серии в коллекции по переданному имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
