# СпособВыбораСертификатаMacOS

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интернет](Интернет.md)

Доступен, начиная с версии 8.3.13.

Значения

[Авто (Auto)](СпособВыбораСертификатаMacOS.Авто_(MacOSCertificateSelectMode.Auto).md)  
[Выбирать (Choose)](СпособВыбораСертификатаMacOS.Выбирать_(MacOSCertificateSelectMode.Choose).md)  

Описание:

Содержит варианты способов выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы macOS.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

См. также:

[СертификатКлиентаWindows](СертификатКлиентаWindows_(WindowsClientCertificate).md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.24.
