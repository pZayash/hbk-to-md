# Расширение поля ввода справочника

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md)

Свойства:

[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_справочника.БыстрыйВыбор_(Catalog_text_box_extension.QuickChoice).md)  
[ВыборГруппИЭлементов (ChoiceFoldersAndItems)](Расширение_поля_ввода_справочника.ВыборГруппИЭлементов_(Catalog_text_box_extension.ChoiceFoldersAndItems).md)  
[ВыборПоВладельцу (ChoiceByOwner)](Расширение_поля_ввода_справочника.ВыборПоВладельцу_(Catalog_text_box_extension.ChoiceByOwner).md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на элемент справочника.  
Расширение предоставляет возможность автоматического поиска и подбора элемента справочника по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Клиент = ЭлементыФормы.ПолеВводаКонтрагента; Клиент.БыстрыйВыбор = Истина; Клиент.ВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Группы; |

Использование в версии:

Доступен, начиная с версии 8.0.
