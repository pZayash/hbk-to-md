ПоложениеПодписейКДиаграмме (ChartLabelLocation)

КрайАвто (EdgeAuto)

Доступен, начиная с версии 8.2.

Описание:

Подписи отображаются аналогично [КрайВнутри](ПоложениеПодписейКДиаграмме.КрайВнутри_(ChartLabelLocation.EdgeInside).md) если места достаточно, в противном случае поведение аналогично [Край](ПоложениеПодписейКДиаграмме.Край_(ChartLabelLocation.Edge).md).

Использование в версии:

Доступен, начиная с версии 8.2.
