ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыбратьИзСпискаАсинх (ChooseFromListAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВыбратьИзСпискаАсинх(<СписокЗначений>, <ЭлементФормы>, <НачальноеЗначение>)

Параметры:

<СписокЗначений> (обязательный)

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Список значений, содержащий значения, из которых будет осуществляться выбор.

<ЭлементФормы> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md), [КнопкаФормы](КнопкаФормы_(FormButton).md).   
Элемент формы, около которого будет открыт выпадающий список. Для некоторых элементов выпадающий список будет показан около активной области.   
Если не указан, то будет выбран активный элемент формы.  
Примечание: В качестве этого параметра нельзя указывать кнопки, расположенные в командной панели или подменю.

<НачальноеЗначение> (необязательный)

Тип: [Число](lang__def_Number.md), [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).   
Задает начальный элемент, который будет активным при открытии списка.   
Если параметр не указан, в списке не будет выбрано начальное значение.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - выбранный элемент списка или [Неопределено](lang__def_Undefined.md), если пользователь отказался от выбора.

Описание:

Асинхронная функция. Выбор значения из выпадающего списка. Набор значений для выбора задаются списком значений, выпадающий список будет расположен около переданого элемента формы или около его активной области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
