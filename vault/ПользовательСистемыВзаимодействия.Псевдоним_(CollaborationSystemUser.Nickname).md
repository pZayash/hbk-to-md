ПользовательСистемыВзаимодействия (CollaborationSystemUser)

Псевдоним (Nickname)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит псевдоним пользователя, с которым пользователь отображается во внешних системах (например, в интеграции типа "WebChat").  
В интерфейсе клиентских приложений "1С:Предприятия" данное имя не отображается.  
В отборах пользователей системы взаимодействия ([ОтборПользователейСистемыВзаимодействия](ОтборПользователейСистемыВзаимодействия_(CollaborationSystemUsersFilter).md)) данное свойство не используется.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
