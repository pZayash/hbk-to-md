НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияКолонкамиАнализаДанных](ЭлементУправленияКолонкамиАнализаДанных.md).   
Искомый элемент управления колонками анализа данных.

Возвращаемое значение:

Тип: Число.   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
