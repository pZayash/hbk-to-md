# СправочникОбъект.<Имя справочника>.Скопировать

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [СправочникОбъект.\<Имя справочника\>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md).   

Описание:

Создает новый элемент справочника копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Объект = Справочник.Скопировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
