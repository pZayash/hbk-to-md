АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Включен (TurnedOn)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливается кластером при необходимости запустить или остановить рабочий процесс:

- [Истина](lang__def_BooleanTrue.md) - процесс должен быть запущен и будет запущен при возможности.
- [Ложь](lang__def_BooleanFalse.md) - процесс должен быть остановлен и будет остановлен после отсоединения всех пользователей или после истечения времени, заданного свойством [ВремяПринудительногоЗавершения](АдминистрированиеКластер.ВремяПринудительногоЗавершения_(AdministrationCluster.ForceTerminationTime).md) объекта [АдминистрированиеКластер](АдминистрированиеКластер_(AdministrationCluster).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
