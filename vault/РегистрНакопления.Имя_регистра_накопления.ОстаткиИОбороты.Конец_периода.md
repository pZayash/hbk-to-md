Конец периода (необязательный)

Тип параметра: Дата, [МоментВремени](Прикладные_объекты.МоментВремени.md), [Граница](Граница.md).   
Указывается конец периода расчета итогов. Значение конца периода по умолчанию включается в период расчета итогов. Для исключения значения из периода его следует задавать объектом [Граница](Граница.md) с исключением граничного значения.  
Если параметр не задан, итоги рассчитываются по самую последнюю запись.
