ПриСохраненииПользовательскихНастроекНаСервере (OnSaveUserSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСохраненииПользовательскихНастроекНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных.md).   
Сохраняемые настройки.

Вызывается при сохранении пользовательских настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
