Глобальный контекст (Global context)

РазделитьДвоичныеДанные (SplitBinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

РазделитьДвоичныеДанные(<ДвоичныеДанные>, <РазмерЧасти>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Объект типа [ДвоичныеДанные](ДвоичныеДанные.md).

<РазмерЧасти> (обязательный)

Тип: Число.   
Размер одной части данных.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив объектов типа [ДвоичныеДанные](ДвоичныеДанные.md).

Описание:

Разделяет двоичные данные на части заданного размера. Размер задается в байтах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [РазделитьФайл](Глобальный_контекст.РазделитьФайл.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
