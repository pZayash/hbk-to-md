МенеджерИсторииДанных (DataHistoryManager)

УдалитьИзОбработкиПослеЗаписиВерсий (DeleteFromAfterWriteVersionsProcessing)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: Удаление из обработки записей версий указанных объектов метаданных

Синтаксис:

УдалитьИзОбработкиПослеЗаписиВерсий(<ВключитьМетаданные>, <ИсключитьМетаданные>, <Дата>)

Параметры:

<ВключитьМетаданные> (необязательный)

Тип: Неопределено, [Массив](Массив.md).   
Объекты метаданных, по данным которых выполняется удаление из обработки записей версий.  
Если значение параметра Неопределено или пустой массив, то выполняется удаление из обработки всех записей версий, за исключением указанных в параметре <ИсключитьМетаданные>.  
Значение по умолчанию: Неопределено.

<ИсключитьМетаданные> (необязательный)

Тип: Неопределено, [Массив](Массив.md).   
Объекты метаданных, по данным которых не выполняется удаление из обработки о записи версий.  
Значение по умолчанию: Неопределено.

<Дата> (необязательный)

Тип: Дата.   
Дата. Метод удаляет из обработки записи все версии указанных объектов метаданных, созданные ранее этой даты.

Описание варианта метода:

Если метод вызывается из сеанса, в котором разделитель не используется:

- Удаляются версии данных из всех областей данных.
- Требуется право "Администрирование".

Если метод вызывается из сеанса, в котором разделитель используется:

- Удаляются версии данных текущей области.
- Требуется право "Администрирование данных".
- При определении области данных учитываются все независимые разделители конфигурации.

Вариант синтаксиса: Удаление из обработки записей версий указанных данных

Синтаксис:

УдалитьИзОбработкиПослеЗаписиВерсий(<Данные>, <НомерВерсии>)

Параметры:

<Данные> (обязательный)

Тип: [СправочникСсылка.](СправочникСсылка.Имя_справочника.md)<[Имя справочника](СправочникСсылка.Имя_справочника.md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>, [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>, [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)>, [КонстантаКлючЗначения.](КонстантаКлючЗначения.Имя_константы.md)<[Имя константы](КонстантаКлючЗначения.Имя_константы.md)>.   
Данные, по которым создана версия. Значение соответствует значению колонки "Данные" таблицы значений, возвращаемой методом [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии.md).

<НомерВерсии> (необязательный)

Тип: Число.   
Номер созданной версии. Значение аналогично значению в колонке "Данные" таблицы значений, возвращаемой методов [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии.md).  
Если номер не установлен, то из обработки удаляются все версии по указанным данным.

Описание:

Удаляет версии из обработки после записи версий.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
