ПараметрыАудиозаписи (AudioRecordingParameters)

СкоростьКодирования (EncodingBitrate)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Устанавливает скорость кодирования звука в битах в секунду для записи.  
По умолчанию: 128000.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
