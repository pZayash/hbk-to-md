СхемаСистемыАналитики (AnalyticsSystemSchema)

Из двоичных данных

Доступен, начиная с версии 8.3.17.

Синтаксис:

Новый СхемаСистемыАналитики(<СхемаКакДвоичныеДанные>, <Данные>)

Параметры:

<СхемаКакДвоичныеДанные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные, на основе которых будут созданы данные системы аналитики.

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные, на основе которых будут созданы данные системы аналитики.

Описание:

Создает данные системы аналитики на основе переданных двоичных данных.

Использование в версии:

Доступен, начиная с версии 8.3.17.
