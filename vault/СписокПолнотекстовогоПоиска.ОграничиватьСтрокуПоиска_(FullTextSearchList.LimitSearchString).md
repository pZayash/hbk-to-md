# СписокПолнотекстовогоПоиска.ОграничиватьСтрокуПоиска

СписокПолнотекстовогоПоиска (FullTextSearchList)

ОграничиватьСтрокуПоиска (LimitSearchString)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак использования ограничения запроса полнотекстового поиска. Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), запрос будет проанализирован на предмет превышения ограничений.   
Максимальная длина строки запроса - 1000 символов, максимальное количество слов в запросе - 32. При превышении любого из этих лимитов будет сгенерировано исключение.  
Значение по умолчанию зависит от режима совместимости. В режиме совместимости конфигурации [Версия8\_3\_17](РежимСовместимости.Версия8_3_17_(CompatibilityMode.Version8_3_17).md) и ниже значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).   
В режиме совместимости с более новыми версиями значение по умолчанию - [Истина](lang__def_BooleanTrue.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
