ЭлементОбъектHTML (HTMLObjectElement)

Префикс (Prefix)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Префикс узла HTML.   
Свойство имеет значение Неопределено для узлов, отличных от [АтрибутHTML](АтрибутHTML.md) и [ЭлементHTML](ЭлементHTML.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
