ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

Заголовок (Title)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Заголовок диалога.   
Если имеет значение [Неопределено](lang__def_Undefined.md), то используется стандартный заголовок.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
