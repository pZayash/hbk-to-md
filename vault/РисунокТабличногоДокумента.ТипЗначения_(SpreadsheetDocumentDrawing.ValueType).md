РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ТипЗначения (ValueType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Содержит объект, описывающий допустимые типы значений для рисунка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступно только для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md), если в рисунке хранится значение (свойство [СодержитЗначение](РисунокТабличногоДокумента.СодержитЗначение_(SpreadsheetDocumentDrawing.ContainsValue).md) равно [Истина](lang__def_BooleanTrue.md)).

Использование в версии:

Доступен, начиная с версии 8.0.
