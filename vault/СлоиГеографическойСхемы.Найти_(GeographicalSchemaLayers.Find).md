СлоиГеографическойСхемы (GeographicalSchemaLayers)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст, по которому будет искаться слой в коллекции.

Возвращаемое значение:

Тип: [СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск объекта по указанному тексту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
