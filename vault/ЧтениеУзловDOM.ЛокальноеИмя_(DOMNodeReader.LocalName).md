ЧтениеУзловDOM (DOMNodeReader)

ЛокальноеИмя (LocalName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит локальное имя для текущего узла XML. Имя зависит от типа текущего узла:

- [Атрибут](ТипУзлаXML.Атрибут_(XMLNodeType.Attribute).md) - локальное имя атрибута;
- [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md) - имя типа документа;
- [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md) - локальное имя элемента;
- [КонецЭлемента](ТипУзлаXML.КонецЭлемента_(XMLNodeType.EndElement).md) - локальное имя элемента;
- [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md) - имя ссылки на сущность;
- [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки_(XMLNodeType.ProcessingInstruction).md) - адрес инструкции обработки;
- [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML_(XMLNodeType.XMLDeclaration).md) - строка xml.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для узлов, для которых локальное имя не определено, – пустая строка.

Использование в версии:

Доступен, начиная с версии 8.1.
