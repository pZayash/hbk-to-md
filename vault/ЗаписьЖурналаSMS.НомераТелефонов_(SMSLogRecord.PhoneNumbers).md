ЗаписьЖурналаSMS (SMSLogRecord)

НомераТелефонов (PhoneNumbers)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Один или несколько номеров (список рассылки) телефона, с которого или на который было отправлено SMS-сообщение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.11.
