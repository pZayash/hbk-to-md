КолонкиМоделиПрогноза (PredictionModelColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаМоделиПрогноза](КолонкаМоделиПрогноза_(PredictionModelColumn).md).   
Искомая колонка.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
