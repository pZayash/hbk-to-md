СредстваТелефонии (TelephonyTools)

ОтключитьОбработчикЗвонков (DetachCallsHandler)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ОтключитьОбработчикЗвонков()

Описание:

Отключает ранее подключенный обработчик звонков.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
