Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются запрос выбора или запрос уничтожения таблицы.  
Возможно обращение к запросу выбора или запросу уничтожения таблицы посредством оператора [...]. В качестве аргумента передается индекс.
