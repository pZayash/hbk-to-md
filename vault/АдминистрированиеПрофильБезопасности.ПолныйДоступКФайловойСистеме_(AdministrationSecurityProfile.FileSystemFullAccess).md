АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКФайловойСистеме (FileSystemFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет полный доступ к файловой системе сервера.  
[Истина](lang__def_BooleanTrue.md) – доступна файловая система. [Ложь](lang__def_BooleanFalse.md) – доступ к файловой системе запрещен, можно использовать только виртуальные каталоги, перечисленные в списке виртуальных каталогов профиля безопасности.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
