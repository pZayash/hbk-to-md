БитНаПиксел24 (BitPerPixel24)

Доступен, начиная с версии 8.3.14.

Описание:

Для хранения одного пикселя используется 24 бита (картинка многоцветная).

Использование в версии:

Доступен, начиная с версии 8.3.14.
