Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfile (CreateSecurityProfile)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfile()

Возвращаемое значение:

Тип: [Профиль безопасности](Профиль_безопасности.md).   

Описание:

Создает новый профиль безопасности кластера для его последующей настройки и регистрации. Все свойства созданного профиля имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
