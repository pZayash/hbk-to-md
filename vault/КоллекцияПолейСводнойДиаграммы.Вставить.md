КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Поле>, <Позиция>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы.md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы.md).   
Позиция, в которую будет вставлено поле или коллекция.

<Позиция> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы.md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы.md).   

Описание:

Вставляет поле или другую коллекцию в указанную позицию коллекции полей сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
