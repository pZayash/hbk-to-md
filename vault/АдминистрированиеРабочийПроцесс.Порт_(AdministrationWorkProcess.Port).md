АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Порт (Port)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер основного IP-порта рабочего процесса. Этот порт выделяется динамически при старте рабочего процесса из диапазонов портов, определенных для соответствующего рабочего сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
