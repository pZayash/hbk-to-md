Глобальный контекст (Global context)

Предупреждение (DoMessageBox)

Доступен, начиная с версии 8.0.

Синтаксис:

Предупреждение(<ТекстПредупреждения>, <Таймаут>, <Заголовок>)

Параметры:

<ТекстПредупреждения> (обязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст предупреждения.

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Интервал времени в секундах, в течение которого система будет ожидать ответа пользователя. По истечении интервала окно предупреждения будет закрыто. Если параметр не указан, то время ожидания не ограничено.  
Если параметр имеет отрицательное значение, будет сгенерировано исключение.  
Значение по умолчанию: 0.

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит заголовок окна предупреждения.  
Если не установлен, то у диалога заголовок не отображается.

Описание:

Выводит на экран окно предупреждения.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [ПоказатьПредупреждение](Глобальный_контекст.ПоказатьПредупреждение_(Global_context.ShowMessageBox).md).

Пример:

|  |
| --- |
| Предупреждение(НСтр("ru = 'Выберите документ!'; en = 'Select a document!'"), 10); |

Использование в версии:

Доступен, начиная с версии 8.0.
