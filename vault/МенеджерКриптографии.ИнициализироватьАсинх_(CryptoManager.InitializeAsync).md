# МенеджерКриптографии.ИнициализироватьАсинх

МенеджерКриптографии (CryptoManager)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Без инициализации модуля криптографии

Синтаксис:

ИнициализироватьАсинх(<ИспользованиеИнтерактивногоРежима>)

Параметры:

<ИспользованиеИнтерактивногоРежима> (обязательный)

Тип: [ИспользованиеИнтерактивногоРежимаКриптографии](ИспользованиеИнтерактивногоРежимаКриптографии_(CryptoInteractiveModeUse).md).   
Определяет использование интерактивного режима работы модуля криптографии.  
Значение по умолчанию: [НеИспользовать](ИспользованиеИнтерактивногоРежимаКриптографии.НеИспользовать_(CryptoInteractiveModeUse.DontUse).md).

Вариант синтаксиса: С инициализацией модуля криптографии

Синтаксис:

ИнициализироватьАсинх(<ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>, <ИспользованиеИнтерактивногоРежима>)

Параметры:

<ИмяМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы MS Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу-библиотеке, реализующей требуемый набор функций.  
Параметр используется при создании менеджера в Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: [Число](lang__def_Number.md).   
Тип модуля криптографии.

<ИспользованиеИнтерактивногоРежима> (необязательный)

Тип: [ИспользованиеИнтерактивногоРежимаКриптографии](ИспользованиеИнтерактивногоРежимаКриптографии_(CryptoInteractiveModeUse).md).   
Определяет использование интерактивного режима работы модуля криптографии.  
Значение по умолчанию: [НеИспользовать](ИспользованиеИнтерактивногоРежимаКриптографии.НеИспользовать_(CryptoInteractiveModeUse.DontUse).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).

Описание:

Асинхронная функция. Начинает инициализацию объекта [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md) с переданными параметрами.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе параметры <ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии> игнорируются.

См. также:

[МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md), метод [НачатьИнициализацию](МенеджерКриптографии.НачатьИнициализацию_(CryptoManager.BeginInitialization).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.

Описание изменено в версии 8.3.24.
