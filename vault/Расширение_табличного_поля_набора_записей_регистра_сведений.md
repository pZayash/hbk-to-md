Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для набора записей регистра сведений. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержат значение типа [РегистрСведенийЗапись.](РегистрСведенийЗапись.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийЗапись.Имя_регистра_сведений.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
