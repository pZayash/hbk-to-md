ЭлементыZipФайла (ZipFileEntries)

Найти (Find)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Найти](ЭлементыФайлаАрхива.Найти.md)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяЭлемента>)

Параметры:

<ИмяЭлемента> (обязательный)

Тип: Строка.   
Имя элемента.

Возвращаемое значение:

Тип: [ЭлементZipФайла](ЭлементZipФайла.md), Неопределено.   
Если не найден, возвращается Неопределено.

Описание:

Осуществляет поиск элемента по его имени.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
