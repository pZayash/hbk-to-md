Значения

[Строка (String)](ТипТекстаНовыхЭлементовПланировщика.Строка.md)  
[ФорматированнаяСтрока (FormattedString)](ТипТекстаНовыхЭлементовПланировщика.ФорматированнаяСтрока.md)  

Описание:

Содержит тип текста новых элементов планировщика. Влияет на тип параметра <Текст> события [ПередСозданием](Расширение_поля_формы_для_поля_планировщика.ПередСозданием.md), а также тип элементов планировщика, создаваемых через методы [Добавить](КоллекцияЭлементовПланировщика.Добавить.md) и [Вставить](КоллекцияЭлементовПланировщика.Вставить.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML.

Использование в версии:

Доступен, начиная с версии 8.3.18.
