ВыборкаИзРезультатаЗапроса (QueryResultSelection)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующая запись выбрана; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующую запись из результата запроса. Для обхода результата запроса нужно после получения выборки вызвать данный метод для позиционирования на первый элемент и далее вызывать до тех пор, пока не будет возвращено значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если при получении записей выбираются все записи со значением поля, равным значению поля, заданному при предыдущем вызове метода [СледующийПоЗначениюПоля](ВыборкаИзРезультатаЗапроса.СледующийПоЗначениюПоля_(QueryResultSelection.NextByFieldValue).md), при попытке получить записи со значением поля, отличающимся от полученного методом [СледующийПоЗначениюПоля](ВыборкаИзРезультатаЗапроса.СледующийПоЗначениюПоля_(QueryResultSelection.NextByFieldValue).md), будет возвращено значение [Ложь](lang__def_BooleanFalse.md).

Пример:

|  |
| --- |
| Пока Выборка.Следующий() Цикл     ... КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
