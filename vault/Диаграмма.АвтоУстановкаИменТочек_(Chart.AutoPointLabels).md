Диаграмма (Chart)

АвтоУстановкаИменТочек (AutoPointLabels)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.1 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматической генерации текста для новых точек диаграммы, состоящих из слова "Точка" и номера точки без пробела.  
[Истина](lang__def_BooleanTrue.md) - имена точек формируются автоматически.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0 (в режиме совместимости с версией 8.1 и последующими).

Не рекомендуется использовать, начиная с версии 8.2.
