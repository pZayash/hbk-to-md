КатегорияСодержимогоHTML (HTMLContentCategory)

ImportАтрибуты (ImportAttributes)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все элементы, у которых есть атрибуты rel="import" и href с любым значением, т.е. будут удалены выражения вида: <link rel="import" href="any/href/link"/>.

Использование в версии:

Доступен, начиная с версии 8.3.13.
