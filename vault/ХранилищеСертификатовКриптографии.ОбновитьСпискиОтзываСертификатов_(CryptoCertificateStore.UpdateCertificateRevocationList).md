ХранилищеСертификатовКриптографии (CryptoCertificateStore)

ОбновитьСпискиОтзываСертификатов (UpdateCertificateRevocationList)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ОбновитьСпискиОтзываСертификатов(<Сертификаты>, <ОбновлятьПриНаличииРасширенияСАдресомOCSP>)

Параметры:

<Сертификаты> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md) – список сертификатов, по которым необходимо обновить списки отзыва. По умолчанию выполняется обновление всех сертификатов хранилища.

<ОбновлятьПриНаличииРасширенияСАдресомOCSP> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, требуется ли обновлять списки отзыва по сертификату, если в нем есть адрес OCSP-сервера.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, удалось ли обновить хотя бы один список отзыва или не требуется обновлять списки отзыва.

Описание:

Обновляет списки отзыва по списку сертификатов и их цепочек. Рекомендуется добавить в регламентные задания при необходимости использования улучшенной подписи. Позволяет поддерживать списки отзыва в актуальном состоянии в асинхронном режиме (не задерживая функции создания подписи).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Хранилище = МенеджерКриптографии.ПолучитьХранилищеСертификатов(Тип); Сертификаты = Хранилище.ПолучитьВсе(); Хранилище.ОбновитьСпискиОтзываСертификатов(Сертификаты, Ложь); |
