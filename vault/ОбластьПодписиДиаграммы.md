Описание:

Используется для настройки подписи диаграммы. Позволяет устанавливать для подписи цвет, фон, шрифт и другие параметры.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartLabelArea.

Использование в версии:

Доступен, начиная с версии 8.3.10.
