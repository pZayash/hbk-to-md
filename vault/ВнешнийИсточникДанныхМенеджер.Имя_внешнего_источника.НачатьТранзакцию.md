ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

НачатьТранзакцию (BeginTransaction)

Доступен, начиная с версии 8.3.5.

Синтаксис:

НачатьТранзакцию(<РежимБлокировок>, <УровеньИзоляции>)

Параметры:

<РежимБлокировок> (необязательный)

Тип: [РежимУправленияБлокировкойДанных](РежимУправленияБлокировкойДанных.md).   
Установка параметра имеет смысл, если для свойства внешнего источника данных "Режим управления блокировкой данных" выбрано значение "Автоматический и Управляемый".  
Если значение параметра [Автоматический](РежимУправленияБлокировкойДанных.Автоматический.md), то данная транзакция будет выполняться в режиме автоматических блокировок.  
Если значение параметра [Управляемый](РежимУправленияБлокировкойДанных.Управляемый.md), то в данной транзакции будут выполняться управляемые блокировки.  
Если к моменту вызова данного метода была начата транзакция, выполняющаяся в автоматическом режиме управления блокировками, то установка значения параметра [Управляемый](РежимУправленияБлокировкойДанных.Управляемый.md) не приведет к изменению режима управления блокировками.  
Если к моменту вызова данного метода была начата транзакция, выполняющаяся в управляемом режиме блокировок, то установка значения параметра [Автоматический](РежимУправленияБлокировкойДанных.Автоматический.md) приведет к возникновению исключительной ситуации, которая может быть обработана конструкцией Попытка... Исключение... КонецПопытки.  
Если для свойства конфигурации "Режим управления блокировкой данных" выбрано значение "Управляемый", то значение параметра по умолчанию [Управляемый](РежимУправленияБлокировкойДанных.Управляемый.md).  
Значение по умолчанию: [Автоматический](РежимУправленияБлокировкойДанных.Автоматический.md).

<УровеньИзоляции> (необязательный)

Тип: [УровеньИзоляцииТранзакций](УровеньИзоляцииТранзакций.md).   
Параметр указывает необходимый уровень изоляции транзакции.   
Если параметр установлен в [Авто](УровеньИзоляцииТранзакций.Авто.md), то фактический уровень изоляции будет выбран на основании параметра РежимБлокировок, иначе будет выбран указанный уровень. Если данный уровень не поддерживается базой данных внешнего источника данных, то уровень будет автоматически повышен.  
Значение по умолчанию: [Авто](УровеньИзоляцииТранзакций.Авто.md).

Описание:

Открывает транзакцию. Транзакция предназначена для записи в базу данных внешнего источника данных согласованных изменений. Все изменения, внесенные после начала транзакции, будут затем либо полностью записаны, либо полностью отменены.

Доступность:

Сервер, толстый клиент, внешнее соединение.
