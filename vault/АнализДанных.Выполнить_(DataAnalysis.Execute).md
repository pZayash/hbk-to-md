АнализДанных (DataAnalysis)

Выполнить (Execute)

Доступен, начиная с версии 8.0.

Синтаксис:

Выполнить()

Возвращаемое значение:

Тип: [РезультатАнализаДанныхДеревоРешений](РезультатАнализаДанныхДеревоРешений_(DataAnalysisDecisionTreeResult).md), [РезультатАнализаДанныхКластеризация](РезультатАнализаДанныхКластеризация_(DataAnalysisClusterizationResult).md), [РезультатАнализаДанныхПоискАссоциаций](РезультатАнализаДанныхПоискАссоциаций_(DataAnalysisAssociationRulesResult).md), [РезультатАнализаДанныхПоискПоследовательностей](РезультатАнализаДанныхПоискПоследовательностей_(DataAnalysisSequentialPatternsResult).md), [РезультатАнализаДанныхОбщаяСтатистика](РезультатАнализаДанныхОбщаяСтатистика_(DataAnalysisSummaryStatisticsResult).md).   
Результат анализа зависит от типа анализа.

Описание:

Выполняет анализ и получает результат.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
