# ВыборкаИзРезультатаЗапроса.Сбросить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Выполнение и работа с запросами во встроенном языке](Выполнение_и_работа_с_запросами_во_встроенном_языке.md) › [ВыборкаИзРезультатаЗапроса](ВыборкаИзРезультатаЗапроса_(QueryResultSelection).md) › [Методы](ВыборкаИзРезультатаЗапроса__Методы.md)

ВыборкаИзРезультатаЗапроса (QueryResultSelection)

Сбросить (Reset)

Доступен, начиная с версии 8.0.

Синтаксис:

Сбросить()

Описание:

Позиционирует выборку в начало.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

После вызова метода [Следующий](ВыборкаИзРезультатаЗапроса.Следующий_(QueryResultSelection.Next).md) выборка спозиционируется на первую запись в ней.

Использование в версии:

Доступен, начиная с версии 8.0.
