МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьКвитанцииВстроенныхПокупок (GetInAppPurchaseReceipts)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьКвитанцииВстроенныхПокупок(<ИдентификаторыПокупок>)

Параметры:

<ИдентификаторыПокупок> (обязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md).   
Массив идентификаторов покупок, для которых будут возвращены квитанции. Возможна передача одного идентификатора в строке - в таком случае будет возвращен массив квитанций с одним элементом.  
При работе под управлением iOS параметр игнорируется.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [КвитанцияВстроеннойПокупки](КвитанцияВстроеннойПокупки_(InAppPurchaseReceipt).md).  
Важно! При работе под управлением iOS параметр <ИдентификаторыПокупок> будет игнорирован, а массив квитанций будет всегда содержать массив из одного элемента.

Описание:

Возвращает массив актуальных квитанций покупок для указанных идентификаторов. Если покупка с таким идентификатором не приобреталась, то поле "Данные" в квитанции будет пустым.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаютсяПокупки](МенеджерВстроенныхПокупок.ПоддерживаютсяПокупки_(InAppPurchasesManager.PurchasesSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.13.
