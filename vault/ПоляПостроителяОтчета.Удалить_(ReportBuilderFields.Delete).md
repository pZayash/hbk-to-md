ПоляПостроителяОтчета (ReportBuilderFields)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеПостроителяОтчета](ПолеПостроителяОтчета_(ReportBuilderField).md).   
Удаляемое поле.

Описание:

Удаляет поле из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПолеВыбора=ПостроительОтчета.ВыбранныеПоля.Найти(ИмяПоля); ПостроительОтчета.ВыбранныеПоля.Удалить(ПолеВыбора); |

Использование в версии:

Доступен, начиная с версии 8.0.
