ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик> (ChartOfCharacteristicTypesManager.<Chart of characteristic types name>)

<Имя предопределенного вида характеристик> (<Predefined characteristic type name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).   
Содержит ссылки на предопределенный объект плана видов характеристик. Доступ к значению осуществляется по имени, как они заданы в Конфигураторе. Имена имеют только предопределенные виды характеристик. Количество таких свойств задается списком предопределенных объектов плана видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
