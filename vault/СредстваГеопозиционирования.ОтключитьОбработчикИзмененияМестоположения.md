СредстваГеопозиционирования (LocationTools)

ОтключитьОбработчикИзмененияМестоположения (DetachLocationChangeHandler)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ОтключитьОбработчикИзмененияМестоположения(<Процедура>, <ИмяПровайдера>)

Параметры:

<Процедура> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Строка.   
Процедура обработки изменения местоположения, которую необходимо исключить.

<ИмяПровайдера> (необязательный)

Тип: Строка.   
Содержит имя провайдера.

Описание:

Отключает вызов указанной процедуры обработки изменения местоположения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
