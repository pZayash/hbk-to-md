ОграниченияИспользованияДоступныхПолейКомпоновкиДанных (DataCompositionAvailableFieldUseRestrictions)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [ОграничениеИспользованияДоступногоПоляКомпоновкиДанных](ОграничениеИспользованияДоступногоПоляКомпоновкиДанных_(DataCompositionAvailableFieldUseRestriction).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
