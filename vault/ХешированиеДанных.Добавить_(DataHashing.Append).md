ХешированиеДанных (DataHashing)

Добавить (Append)

Доступен, начиная с версии 8.3.1.

Вариант синтаксиса: Добавить двоичные данные

Синтаксис:

Добавить(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Добавляемые двоичные данные.

Описание варианта метода:

Обновляет хеш-сумму, с учетом добавления двоичных данных.

Вариант синтаксиса: Добавить текстовые данные

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Добавляемые текстовые данные (в [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md) кодировке).

Описание варианта метода:

Обновляет хеш-сумму, с учетом добавления текстовых данных.

Вариант синтаксиса: Добавить данные из потока

Синтаксис:

Добавить(<Поток>, <Количество>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md).   
Поток, из которого получаются данные для обновления хеш-суммы.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые читаются из потока и участвуют в обновлении хеш-суммы. Если количество не задано, то читаются все данные до конца потока.

Описание варианта метода:

Обновляет хеш-сумму с учетом добавления данных из потока.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
