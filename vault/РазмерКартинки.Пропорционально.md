РазмерКартинки (PictureSize)

Пропорционально (Proportionally)

Доступен, начиная с версии 8.0.

Описание:

Картинка растягивается или сжимается таким образом, чтобы занять всю высоту или ширину отведенной для нее области, сохраняя при этом соотношение высоты и ширины оригинальной картинки.

Использование в версии:

Доступен, начиная с версии 8.0.
