ВсеЭлементыФормы (FormAllItems)

Переместить (Move)

Доступен, начиная с версии 8.2.

Синтаксис:

Переместить(<Элемент>, <Родитель>, <МестоРасположения>)

Параметры:

<Элемент> (обязательный)

Тип: [ДекорацияФормы](ДекорацияФормы.md), [ГруппаФормы](ГруппаФормы.md), [КнопкаФормы](КнопкаФормы.md), [ТаблицаФормы](ТаблицаФормы.md), [ПолеФормы](ПолеФормы.md).   
Перемещаемый элемент.

<Родитель> (обязательный)

Тип: [ГруппаФормы](ГруппаФормы.md), [ТаблицаФормы](ТаблицаФормы.md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md).   
Новый родитель элемента. Может совпадать со старым.

<МестоРасположения> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы.md), [ДекорацияФормы](ДекорацияФормы.md), [КнопкаФормы](КнопкаФормы.md), [ТаблицаФормы](ТаблицаФормы.md), [ПолеФормы](ПолеФормы.md).   
Элемент, перед которым нужно разместить перемещаемый элемент. Если не задан, то перемещается в конец коллекции.

Описание:

Перемещает элемент коллекции.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
