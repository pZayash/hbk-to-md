ДополнительныйИндекс (AdditionalIndex)

Таблица (Table)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит название таблицы (основной, табличной части либо виртуальной), по которой строится дополнительный индекс объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
