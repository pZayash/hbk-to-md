# HTTPСоединение.Удалить

HTTPСоединение (HTTPConnection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<HTTPЗапрос>)

Параметры:

<HTTPЗапрос> (обязательный)

Тип: [HTTPЗапрос](HTTPЗапрос_(HTTPRequest).md).   
HTTP-запрос.

Возвращаемое значение:

Тип: [HTTPОтвет](HTTPОтвет_(HTTPResponse).md).   

Описание:

Удаляет ресурс на сервере при помощи HTTP-запроса DELETE.

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
