**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Типы свойств объектов метаданных](Типы_свойств_объектов_метаданных.md) › [Дополнительные индексы](Дополнительные_индексы.md)

Доступен, начиная с версии 8.3.26.

Элементы коллекции:

[ДополнительныйИндекс](ДополнительныйИндекс_(AdditionalIndex).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются дополнительные индексы.  
Возможно обращение к дополнительному индексу посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Количество (Count)](ДополнительныеИндексы.Количество_(AdditionalIndexes.Count).md)  
[Получить (Get)](ДополнительныеИндексы.Получить_(AdditionalIndexes.Get).md)  

Описание:

Представляет собой коллекцию дополнительных индексов объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
