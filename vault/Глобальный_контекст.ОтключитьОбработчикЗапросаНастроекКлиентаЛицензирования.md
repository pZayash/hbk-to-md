Глобальный контекст (Global context)

ОтключитьОбработчикЗапросаНастроекКлиентаЛицензирования (DetachLicensingClientParametersRequestHandler)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ОтключитьОбработчикЗапросаНастроекКлиентаЛицензирования()

Описание:

Отключает обработчик запроса настройки клиента лицензирования, подключенный ранее при помощи [ПодключитьОбработчикЗапросаНастроекКлиентаЛицензирования](Глобальный_контекст.ПодключитьОбработчикЗапросаНастроекКлиентаЛицензирования.md) .

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Пример:

|  |
| --- |
| ОтключитьОбработчикЗапросаНастройкиКлиентаЛицензирования(); |

Использование в версии:

Доступен, начиная с версии 8.3.7.
