КатегорияСодержимогоHTML (HTMLContentCategory)

AreaТеги (AreaTags)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все атрибуты href, содержащиеся в элементах <area>. Таким образом, <area href="dest"> примет вид <area>.

Использование в версии:

Доступен, начиная с версии 8.3.13.
