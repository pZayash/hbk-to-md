ОбъектМетаданных: ФункциональнаяОпция (MetadataObject: FunctionalOption)

Хранение (Location)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Константа](ОбъектМетаданных_Константа_(MetadataObject_Constant).md), [ОбъектМетаданных: Ресурс](ОбъектМетаданных_Ресурс_(MetadataObject_Resource).md), [ОбъектМетаданных: Реквизит](ОбъектМетаданных_Реквизит_(MetadataObject_Attribute).md).   
Описывает объект, который является источником значения для функциональной опции. Таким источником может быть константа, реквизит справочника или ресурс регистра сведений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство должно быть заполнено.

Использование в версии:

Доступен, начиная с версии 8.2.
