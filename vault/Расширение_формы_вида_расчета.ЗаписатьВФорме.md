Расширение формы вида расчета (Calculation type form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: Булево.   
Истина - объект записан; Ложь - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Объект записан."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
