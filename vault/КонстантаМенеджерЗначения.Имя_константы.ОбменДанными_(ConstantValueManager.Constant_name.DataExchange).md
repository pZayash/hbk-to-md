# КонстантаМенеджерЗначения.<Имя константы>.ОбменДанными

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md) › [КонстантаМенеджерЗначения.<Имя константы>](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md) › [Свойства](КонстантаМенеджерЗначения.Имя_константы__Свойства.md)

КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md).   
Используется для управления обменом данных.  
С помощью данного свойства настраивается состав узлов-получателей, для которых будут регистрироваться изменения данных, узел-отправитель, из которого получена записываемая информация, а также устанавливается режим [Загрузка](ПараметрыОбменаДанными.Загрузка_(DataExchangeParameters.Load).md), указывающий, что выполняется перенос информации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
