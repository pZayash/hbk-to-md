ЭлементИзмеренияПланировщика (PlannerDimensionItem)

Элементы (Items)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовИзмеренияПланировщика](КоллекцияЭлементовИзмеренияПланировщика_(PlannerDimensionItemCollection).md).   
Содержит коллекцию вложенных элементов измерения. Если коллекция пустая, то элемент измерения отображается обычным образом (не как группа).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
