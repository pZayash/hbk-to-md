Диаграмма (Chart)

АвтоТранспонирование (AutoTransposition)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то в зависимости от типа диаграммы и текущих данных диаграмма может менять местами серии и точки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Недоступно в режиме источника данных.

Использование в версии:

Доступен, начиная с версии 8.0.
