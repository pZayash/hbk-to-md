# ДокументDOM.URIПространстваИмен

ДокументDOM (DOMDocument)

URIПространстваИмен (NamespaceURI)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
URI пространства имен узла.  
Свойство имеет значение [Неопределено](lang__def_Undefined.md) для узлов, отличных от [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) и [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), или для узлов, созданных при помощи метода DOM Level 1 (например, [СоздатьЭлемент](ДокументDOM.СоздатьЭлемент_(DOMDocument.CreateElement).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
