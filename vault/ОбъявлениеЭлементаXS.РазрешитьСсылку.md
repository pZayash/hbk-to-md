ОбъявлениеЭлементаXS (XSElementDeclaration)

РазрешитьСсылку (ResolveReference)

Доступен, начиная с версии 8.1.

Синтаксис:

РазрешитьСсылку()

Возвращаемое значение:

Тип: [ОбъявлениеЭлементаXS](ОбъявлениеЭлементаXS.md), Неопределено.   
Компонент (объявление атрибута или элемента), на который ссылается данное объявление, или Неопределено, если такого компонента нет.

Описание:

Разрешить ссылку на объявление атрибута или элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
