КолонкаОписанияИсточникаДанных (DataSourceDescriptionColumn)

Итог (Total)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение для расчета итога по данному полю.  
Например, Сумма(КоличествоОборот).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
