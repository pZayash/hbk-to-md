НастройкиТаблицыДинамическогоСписка (DynamicListTableSettings)

ТекущаяСтрока (CurrentRow)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Текущая строка динамического списка.  
Тип свойства соответствует типу значения свойства [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md) таблицы формы, связанной с динамическим списком, или [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
