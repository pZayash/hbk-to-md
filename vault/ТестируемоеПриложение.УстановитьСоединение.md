ТестируемоеПриложение (TestedApplication)

УстановитьСоединение (Connect)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УстановитьСоединение()

Описание:

Устанавливает соединение с тестируемым приложением.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

Если соединение не установлено, генерируется исключительная ситуация.

Использование в версии:

Доступен, начиная с версии 8.3.1.
