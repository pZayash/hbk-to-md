НастройкиВосстановленияПароля (PasswordRecoverySettings)

ПортSMTP (SMTPPort)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет порт протокола SMTP.   
0 - означает использовать 465 порт для SSL соединений и 25 - для незащищенных.  
Значение по умолчанию: 0.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
