# ЧтениеДанных.ПрочитатьЦелое32Асинх

ЧтениеДанных (DataReader)

ПрочитатьЦелое32Асинх (ReadInt32Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьЦелое32Асинх(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - результат чтения типа [Число](lang__def_Number.md).  
Если не удалось прочитать число из-за того, что был достигнут конец потока, результатом обещания будет значение [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Считывает 32-битное целое число из потока. Результат выполнения обещания - результат чтения типа Число.

Доступность:

Тонкий клиент, толстый клиент.
