Описание:

Содержит возможные типы платформы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[СистемнаяИнформация](СистемнаяИнформация_(SystemInfo).md), свойство [ТипПлатформы](СистемнаяИнформация.ТипПлатформы_(SystemInfo.PlatformType).md)  

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.24.
