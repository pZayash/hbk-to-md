HTTPСервисОтвет (HTTPServiceResponse)

УстановитьТелоИзДвоичныхДанных (SetBodyFromBinaryData)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТелоИзДвоичныхДанных(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Двоичные данные, которые будут записаны в тело HTTP-ответа.

Описание:

Устанавливает тело HTTP-ответа из двоичных данных.

Доступность:

Сервер, толстый клиент.

См. также:

[HTTPСервисОтвет](HTTPСервисОтвет_(HTTPServiceResponse).md), метод [УстановитьТелоИзСтроки](HTTPСервисОтвет.УстановитьТелоИзСтроки_(HTTPServiceResponse.SetBodyFromString).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
