ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

РазделениеАутентификации (AuthenticationSeparation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РазделениеАутентификацииОбщегоРеквизита](РазделениеАутентификацииОбщегоРеквизита_(CommonAttributeAuthenticationSeparation).md).   
Указывает, требуется ли установка значения этого разделителя для однозначного выбора пользователя информационной базы по имени пользователя и по имени пользователя операционной системы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
