ОбъектМетаданных: Документ (MetadataObject: Document)

МодульМенеджера (ManagerModule)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
При запуске Конфигуратора в пакетном режиме из командной строки существует возможность выгрузить тексты модулей конфигурации. Для этого следует использовать параметр запуска /DumpConfigFiles с ключом –Module.

Использование в версии:

Доступен, начиная с версии 8.2.
