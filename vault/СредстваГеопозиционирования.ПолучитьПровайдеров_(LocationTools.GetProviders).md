СредстваГеопозиционирования (LocationTools)

ПолучитьПровайдеров (GetProviders)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьПровайдеров(<ТолькоДоступных>)

Параметры:

<ТолькоДоступных> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр имеет значение [Истина](lang__def_BooleanTrue.md) - метод вернет список провайдеров, доступных в данный момент.  
Если параметр имеет значение [Ложь](lang__def_BooleanFalse.md) - метод вернет список всех провайдеров, присутствующих на устройстве.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив из значений типа [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования_(LocationProviderInformation).md), содержащий найденные провайдеры геопозиционирования.

Описание:

Получает информацию о провайдерах геопозиционирования, которые доступны в системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
