ОбъектМетаданных: Ресурс (MetadataObject: Resource)

ЗначениеЗаполнения (FillingValue)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Неопределено.   
Значение типа, совпадающего с типом реквизита. Данное значение будет присваиваться реквизиту процедурой стандартной обработки заполнения, если реквизит не заполнен из данных заполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
