Описание:

Текстовые данные письма.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Сообщение = Новый ИнтернетПочтовоеСообщение; Текст = Сообщение.Тексты.Добавить("Привет, мир!"); Текст.ТипТекста = ТипТекстаПочтовогоСообщения.ПростойТекст; |
