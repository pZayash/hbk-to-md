СписокВыбораНавигационнойСсылки (URLChoiseList)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Индекс(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ элемента коллекции.

Возвращаемое значение:

Тип: Число.   
Если элемент не найден, то возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
