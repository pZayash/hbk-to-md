ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

УстановитьЭлементУправления (SetControl)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьЭлементУправления(<ТипЭлементаУправления>)

Параметры:

<ТипЭлементаУправления> (обязательный)

Тип: Тип.   
Тип устанавливаемого элемента управления.

Описание:

Устанавливает элемент управления, которым будут изменяться ячейки в области табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
