БотСистемыВзаимодействия (CollaborationSystemBot)

Предопределенный (Predefined)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если бот создан из объекта метаданных, у которого свойство [Предопределенный](ОбъектМетаданных_Бот.Предопределенный_(MetadataObject_Bot.Predefined).md) установлено в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
