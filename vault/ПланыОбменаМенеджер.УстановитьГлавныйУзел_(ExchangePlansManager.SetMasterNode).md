# ПланыОбменаМенеджер.УстановитьГлавныйУзел

ПланыОбменаМенеджер (ExchangePlansManager)

УстановитьГлавныйУзел (SetMasterNode)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьГлавныйУзел(<ГлавныйУзел>)

Параметры:

<ГлавныйУзел> (необязательный)

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>, [Неопределено](lang__def_Undefined.md).   
Если в качестве значения параметра передается значение типа [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)> или [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>, то план обмена, к которому относится ссылка или объект должен быть планом обмена распределенной информационной базы.   
Если в качестве значения передано значение [Неопределено](lang__def_Undefined.md), то назначение главного узла отменяется.

Описание:

Устанавливает главный узел распределенной информационной базы для текущей информационной базы или отменяет назначение главного узла.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПланыОбмена.УстановитьГлавныйУзел(ГлУзел); |

Использование в версии:

Доступен, начиная с версии 8.0.
