КлючXBase (XBaseKey)

<Имя поля> (<Field name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [Булево](lang__Булево_(Boolean).md).   
Набор свойств содержит значения полей в текущей записи таблицы базы данных формата DBF, состав и тип которых зависит от структуры файла базы данных, с которым связан конкретный объект. Доступ к значению осуществляется по имени поля таблицы базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
