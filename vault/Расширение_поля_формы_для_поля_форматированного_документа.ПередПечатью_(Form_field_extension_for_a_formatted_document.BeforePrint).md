Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати_(PrintDialogUseMode).md).   
Указывает способ вызова печати:

- [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать_(PrintDialogUseMode.DontUse).md) - Если вызвана команда "Печать с текущими настройками",
- [Использовать](РежимИспользованияДиалогаПечати.Использовать_(PrintDialogUseMode.Use).md) - если вызвана команда "Печать".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), печать выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает на клиенте перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
