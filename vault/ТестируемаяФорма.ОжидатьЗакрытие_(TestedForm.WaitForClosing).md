# ТестируемаяФорма.ОжидатьЗакрытие

ТестируемаяФорма (TestedForm)

ОжидатьЗакрытие (WaitForClosing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОжидатьЗакрытие(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Таймаут в секундах ожидания закрытия объекта. Если таймаут не определен, ожидание будет длиться до момента закрытия.  
Значение по умолчанию: 60.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) – текущий объект закрыт или отсутствует.  
[Ложь](lang__def_BooleanFalse.md) - объект не был закрыт в течение указанного времени.

Описание:

Ожидает закрытия объекта.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
