Поток (Stream)

КопироватьВАсинх (CopyToAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КопироватьВАсинх(<ЦелевойПоток>, <Количество>)

Параметры:

<ЦелевойПоток> (обязательный)

Тип: [Поток](Поток_(Stream).md).   
Поток, в который будет производиться копирование.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется скопировать в целевой поток. Если не задано, то будут скопированы все данные до конца текущего потока.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - количество фактически скопированных байтов типа [Число](lang__def_Number.md).

Описание:

Асинхронная функция. Копирует данные из текущего потока в другой поток.  
Дополнительно можно установить количество байтов, которые будут скопированы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
