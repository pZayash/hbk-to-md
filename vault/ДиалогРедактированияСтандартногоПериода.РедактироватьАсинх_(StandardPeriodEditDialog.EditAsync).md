РедактироватьАсинх (EditAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РедактироватьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Период](ДиалогРедактированияСтандартногоПериода.Период_(StandardPeriodEditDialog.Period).md) - диалог был закрыт нажатием кнопки "OK"; [Неопределено](lang__def_Undefined.md) - в противном случае.

Описание:

Асинхронная функция. Начинает интерактивное редактирование.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
