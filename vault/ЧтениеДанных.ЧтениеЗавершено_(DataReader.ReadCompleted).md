ЧтениеДанных (DataReader)

ЧтениеЗавершено (ReadCompleted)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что во входном потоке больше нет данных для чтения. Изначально устанавливается в значение [Ложь](lang__def_BooleanFalse.md). Если при очередном чтении было прочитано меньше данных, чем было запрошено, принимает значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Значение данного свойство можно ипользовать в качестве выхода из цикла при чтении в цикле.

Использование в версии:

Доступен, начиная с версии 8.3.9.
