ОформлениеЗначений (ValueAppearance)

Получить (Get)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Получить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ оформления.

Возвращаемое значение:

Тип: Произвольный, [Неопределено](lang__def_Undefined.md).   
Если запись с таким ключом не найдена, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Выполняет поиск записи по ключу.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
