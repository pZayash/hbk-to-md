МенеджерКалендарей (CalendarsManager)

ПоддерживаетсяДобавлениеКалендарей (CalendarsAddingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяДобавлениеКалендарей()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
На устройствах под управлением Windows Phone возвращает [Ложь](lang__def_BooleanFalse.md).  
Возвращает [Истина](lang__def_BooleanTrue.md) на устройствах под управлением:

- Android – начиная с версии 4.1,
- iOS – всегда.

Описание:

Определяет возможность добавления календарей.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей_(CalendarsManager).md), метод [ДобавитьКалендарь](МенеджерКалендарей.ДобавитьКалендарь_(CalendarsManager.AddCalendar).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
