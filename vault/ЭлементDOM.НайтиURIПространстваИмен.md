`НайтиURIПространстваИмен`(`<Префикс>`: `Строка`) → `Строка, Неопределено`
<!-- signature:end -->

ЭлементDOM (DOMElement)

НайтиURIПространстваИмен (LookupNamespaceURI)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиURIПространстваИмен(<Префикс>)

Параметры:

<Префикс> (обязательный)

Тип: Строка.   
Префикс, для которого необходимо найти соответствующее ему URI пространства имен.

Возвращаемое значение:

Тип: Строка, Неопределено.   
URI пространства имен, соответствующее указанному префиксу, или Неопределено.

Описание:

Производит поиск URI пространства имен для указанного префикса, начиная с данного узла. Алгоритм поиска префикса описан в спецификации DOM Core.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
