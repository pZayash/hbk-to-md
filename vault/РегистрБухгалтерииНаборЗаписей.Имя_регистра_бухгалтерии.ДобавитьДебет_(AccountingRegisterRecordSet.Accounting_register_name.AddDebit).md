РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

ДобавитьДебет (AddDebit)

Доступен, начиная с версии 8.0.

Синтаксис:

ДобавитьДебет()

Описание:

Добавляет запись с установленным видом движения "Дебет".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл только для регистров без корреспонденции.

Использование в версии:

Доступен, начиная с версии 8.0.
