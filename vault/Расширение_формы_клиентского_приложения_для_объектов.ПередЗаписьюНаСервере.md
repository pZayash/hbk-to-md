Расширение формы клиентского приложения для объектов (Client application form extension for objects)

ПередЗаписьюНаСервере (BeforeWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗаписьюНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от выполнения действия. Если в теле процедуры-обработчика установить данному параметру значение Истина, запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: Ложь.

<ТекущийОбъект>

Тип: [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>, [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>.   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура.md).   
Структура, содержащая параметры записи.

Описание:

Вызывается перед записью объекта на сервере. Можно отменить запись. Выполняется вне транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
