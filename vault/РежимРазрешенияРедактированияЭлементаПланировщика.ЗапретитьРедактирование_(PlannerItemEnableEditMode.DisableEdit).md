РежимРазрешенияРедактированияЭлементаПланировщика (PlannerItemEnableEditMode)

ЗапретитьРедактирование (DisableEdit)

Доступен, начиная с версии 8.3.18.

Описание:

Делает недоступными следующие возможности элемента планировщика:

- стандартная команда планировщика [РедактироватьЭлемент](СтандартнаяКомандаПланировщика.РедактироватьЭлемент_(PlannerStandardCommand.EditItem).md),
- окна редактирования элемента планировщика,
- растягивание элемента планировщика,
- перетаскивание элемента внутри поля планировщика.

Использование в версии:

Доступен, начиная с версии 8.3.18.
