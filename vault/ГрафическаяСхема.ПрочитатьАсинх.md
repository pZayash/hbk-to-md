ГрафическаяСхема (GraphicalSchema)

ПрочитатьАсинх (ReadAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Из файла

Синтаксис:

ПрочитатьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла графической схемы.

Описание варианта метода:

Чтение графической схемы из файла.

Вариант синтаксиса: Из потока

Синтаксис:

ПрочитатьАсинх(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения графической схемы.

Описание варианта метода:

Чтение графической схемы из потока.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Считывает графическую схему из файла.

Доступность:

Тонкий клиент, толстый клиент.
