КолонкиРезультатаМоделиПрогноза (PredictionModelResultColumns)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Колонка>, <Смещение>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаРезультатаМоделиПрогноза](КолонкаРезультатаМоделиПрогноза_(PredictionModelResultColumn).md).   
Сдвигаемая колонка.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает колонку на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
