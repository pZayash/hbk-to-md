РежимВыбораНезаполненного (IncompleteChoiceMode)

ПриНажатииEnter (OnEnterPressed)

Доступен, начиная с версии 8.0.

Описание:

Выбор незаполненного значения производится при нажатии клавиши Enter.

Использование в версии:

Доступен, начиная с версии 8.0.
