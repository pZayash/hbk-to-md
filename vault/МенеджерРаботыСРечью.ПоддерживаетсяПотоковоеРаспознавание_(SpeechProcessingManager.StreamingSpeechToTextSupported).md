МенеджерРаботыСРечью (SpeechProcessingManager)

ПоддерживаетсяПотоковоеРаспознавание (StreamingSpeechToTextSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяПотоковоеРаспознавание()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В тонком и толстом клиентах возвращает:

- [Ложь](lang__def_BooleanFalse.md) - при работе в операционных системах MS Windows Server 2003/Windows XP,
- [Истина](lang__def_BooleanTrue.md) - в остальных случаях.

В веб-клиенте:

- [Истина](lang__def_BooleanTrue.md), если веб-клиент опубликован по HTTPS или на localhost и запущен в браузере Chrome (а также всех браузерах на базе движка Chromium), и версия Chrome не ниже 66,
- [Ложь](lang__def_BooleanFalse.md) - в остальных случаях.

Описание:

Определяет возможность выполнять потоковое распознавание речи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
