ОпределениеСоставногоТипаXS (XSComplexTypeDefinition)

ИспользованиеАтрибутов (AttributeUses)

Доступен, начиная с версии 8.1.

Синтаксис:

ИспользованиеАтрибутов()

Возвращаемое значение:

Тип: [ФиксированныйСписокКомпонентXS](ФиксированныйСписокКомпонентXS_(XSComponentFixedList).md).   

Описание:

Формирует фактический список атрибутов данного типа (включает атрибуты базовых типов).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
