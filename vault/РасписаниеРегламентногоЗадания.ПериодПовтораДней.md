РасписаниеРегламентногоЗадания (JobSchedule)

ПериодПовтораДней (DaysRepeatPeriod)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Период времени в днях, через который нужно повторять запуск регламентного задания.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если значение свойства равно 0, задание будет запущено в течение только одного дня (задание может быть запущено в течение одного дня несколько раз, в зависимости от [ПериодПовтораВТечениеДня](РасписаниеРегламентногоЗадания.ПериодПовтораВТечениеДня.md)). Если [ПериодПовтораДней](РасписаниеРегламентногоЗадания.ПериодПовтораДней.md) = 1, задание будет запущено каждый день, [ПериодПовтораДней](РасписаниеРегламентногоЗадания.ПериодПовтораДней.md) = 2, задание будет запущено через день и т.д.

Использование в версии:

Доступен, начиная с версии 8.1.
