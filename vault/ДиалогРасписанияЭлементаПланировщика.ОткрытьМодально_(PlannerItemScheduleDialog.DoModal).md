ДиалогРасписанияЭлементаПланировщика (PlannerItemScheduleDialog)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Открывает диалог расписания в модальном режиме.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).
