ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

ЗапуститьБезОжидания (RunWithoutWaiting)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗапуститьБезОжидания()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В зависимости от успешности запуска приложения возвращается значение [Истина](lang__def_BooleanTrue.md) или [Ложь](lang__def_BooleanFalse.md).

Описание:

Запускает приложение без ожидания результата. Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЗапуск](ЗапускПриложенияМобильногоУстройства.ПоддерживаетсяЗапуск_(MobileDeviceApplicationRun.RunningSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
