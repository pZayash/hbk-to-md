ЧтениеСообщенияОбмена (ExchangeMessageReader)

Отправитель (Sender)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [Неопределено](lang__def_Undefined.md).   
Ссылка на узел, являющийся отправителем данного сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md) и после обращения к методу [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение_(ExchangeMessageReader.EndRead).md) или [ПрерватьЧтение](ЧтениеСообщенияОбмена.ПрерватьЧтение_(ExchangeMessageReader.CancelRead).md) значением данного свойства будет [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.0.
