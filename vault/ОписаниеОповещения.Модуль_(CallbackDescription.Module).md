ОписаниеОповещения (CallbackDescription)

Модуль (Module)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md), [ОбщийМодуль](ОбщийМодуль_(CommonModule).md).   
Программный модуль, процедура которого будет вызвана.  
При использовании типа [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) будет вызван метод модуля указанной формы клиентского приложения.  
При использовании типа [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md) будет вызван метод модуля команды командного интерфейса.  
При использовании типа [ОбщийМодуль](ОбщийМодуль_(CommonModule).md) будет вызван метод общего неглобального модуля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
