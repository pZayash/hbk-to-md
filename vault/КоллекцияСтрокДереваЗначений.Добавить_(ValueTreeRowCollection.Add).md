КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md).   

Описание:

Добавляет строку в конец коллекции строк данного уровня дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| НовыйЭлемент = СтрокиСостава.Добавить(); НовыйЭлемент.Затрата = Затрата; НовыйЭлемент.Норма = Количество; |

Использование в версии:

Доступен, начиная с версии 8.0.
