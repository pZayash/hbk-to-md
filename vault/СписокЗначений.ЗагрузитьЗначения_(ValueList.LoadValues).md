# СписокЗначений.ЗагрузитьЗначения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Список значений](Список_значений.md) › [СписокЗначений](СписокЗначений_(ValueList).md) › [Методы](СписокЗначений__Методы.md)

СписокЗначений (ValueList)

ЗагрузитьЗначения (LoadValues)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗагрузитьЗначения(<МассивЗначений>)

Параметры:

<МассивЗначений> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий значения для заполнения списка значений.

Описание:

Загружает список значений значениями из переданного массива. При этом все прежние элементы списка удаляются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СписокТиповЦен.ЗагрузитьЗначения(МассивЭлементов); |

Использование в версии:

Доступен, начиная с версии 8.0.
