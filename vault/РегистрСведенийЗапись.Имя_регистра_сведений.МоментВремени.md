РегистрСведенийЗапись.<Имя регистра сведений> (InformationRegisterRecord.<Information register name>)

МоментВремени (PointInTime)

Доступен, начиная с версии 8.1.

Синтаксис:

МоментВремени()

Возвращаемое значение:

Тип: [МоментВремени](Прикладные_объекты.МоментВремени.md).   

Описание:

Получает момент времени, соответствующий записи регистра. Применяется для регистров сведений, для которых в Конфигураторе установлен режим записи "Подчинение регистратору".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
