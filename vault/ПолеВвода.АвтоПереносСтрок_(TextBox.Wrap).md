ПолеВвода (TextBox)

АвтоПереносСтрок (Wrap)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического переноса длинных строк текста.  
[Истина](lang__def_BooleanTrue.md) - выполняется автоперенос строк.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, только если свойство [МногострочныйРежим](ПолеВвода.МногострочныйРежим_(TextBox.MultiLine).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
