Описание:

Предназначен для представления структурированной информации об ошибке (исключении).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Сериализуется.

Пример:

|  |
| --- |
| Попытка     ... Исключение     Инфо = ИнформацияОбОшибке();     Сообщить(НСтр("ru='Описание=';en='Description='") + Инфо.Описание + "'");     Сообщить(НСтр("ru='ИмяМодуля=';en='ModuleName='") + Инфо.ИмяМодуля + "'");     Сообщить(НСтр("ru='НомерСтроки=';en='LineNumber='") + Инфо.НомерСтроки + "'");     Сообщить(НСтр("ru='ИсходнаяСтрока=';en='SourceLine='") + Инфо.ИсходнаяСтрока + "'"); КонецПопытки; |
