ЗначенияПараметровДанныхКомпоновкиДанных (DataCompositionDataParameterValues)

ДоступныеПоляПараметров (ParameterAvailableFields)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДоступныеПоляКомпоновкиДанных](ДоступныеПоляКомпоновкиДанных_(DataCompositionAvailableFields).md).   
Содержит доступные поля для редактирования параметров данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
