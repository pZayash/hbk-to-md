ПоляГруппировкиКомпоновкиДанных (DataCompositionGroupFields)

ПолучитьИдентификаторПоОбъекту (GetIDByObject)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИдентификаторПоОбъекту(<Объект>)

Параметры:

<Объект> (необязательный)

Тип: [ПолеГруппировкиКомпоновкиДанных](ПолеГруппировкиКомпоновкиДанных_(DataCompositionGroupField).md), [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных_(DataCompositionAutoGroupField).md).   
Объект, для которого требуется получить идентификатор.

Возвращаемое значение:

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - переданный объект не является частью данного объекта.

Описание:

Получает идентификатор для переданного объекта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
