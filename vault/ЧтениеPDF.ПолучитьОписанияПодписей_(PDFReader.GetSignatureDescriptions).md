ЧтениеPDF (PDFReader)

ПолучитьОписанияПодписей (GetSignatureDescriptions)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьОписанияПодписей(<МенеджерКриптографии>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Содержит объекты типа [ОписаниеПодписиPDF](ОписаниеПодписиPDF_(PDFSignatureDescription).md).

Описание:

Получает массив, содержащий описания электронных подписей, присоединенных к PDF-документу.

Доступность:

Тонкий клиент, сервер, толстый клиент.
