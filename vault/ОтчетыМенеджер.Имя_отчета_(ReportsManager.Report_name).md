ОтчетыМенеджер (ReportsManager)

<Имя отчета> (<Report name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ОтчетМенеджер.\<Имя отчета\>](ОтчетМенеджер.Имя_отчета_(ReportManager.Report_name).md).   
Набор свойств содержит менеджеры отдельных отчетов. Доступ к менеджеру осуществляется по имени. Имена свойств совпадают с именами отчетов, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
