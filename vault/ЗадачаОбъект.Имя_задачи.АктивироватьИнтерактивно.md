ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

АктивироватьИнтерактивно (ActivateInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

АктивироватьИнтерактивно()

Описание:

Выполняет интерактивную активацию задачи. Это приводит к возникновению события [ОбработкаИнтерактивнойАктивации](ЗадачаОбъект.Имя_задачи.ОбработкаИнтерактивнойАктивации.md) задачи, а потом, если разрешена стандартная обработка, события [ОбработкаИнтерактивнойАктивации](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаИнтерактивнойАктивации.md) у точки бизнес-процесса.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
