Почта (Mail)

Подключиться (Logon)

Доступен, начиная с версии 8.0.

Синтаксис:

Подключиться(<Профиль>, <Пароль>)

Параметры:

<Профиль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Профиль пользователя для подключения. Если указана пустая строка, то используется профиль пользователя, установленный по умолчанию.  
Значение по умолчанию: Пустая строка.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя, если он требуется при подключении к серверу.  
Значение по умолчанию: Пустая строка.

Описание:

Осуществляет подключение к почтовому серверу для получения/посылки почты.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

|  |
| --- |
| Почта.Подключиться(); |

Использование в версии:

Доступен, начиная с версии 8.0.
