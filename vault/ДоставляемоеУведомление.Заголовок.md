ДоставляемоеУведомление (DeliverableNotification)

Заголовок (Title)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.   
Заголовок уведомления, который будет отображаться в системной области. Не используется в уведомлениях для iOS версии ниже 8.2.  
Для значения Неопределено в заголовке уведомления отображается название приложения мобильной платформы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - Неопределено.

Использование в версии:

Доступен, начиная с версии 8.3.6.
