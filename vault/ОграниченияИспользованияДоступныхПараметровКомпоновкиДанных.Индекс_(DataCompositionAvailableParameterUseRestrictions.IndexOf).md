ОграниченияИспользованияДоступныхПараметровКомпоновкиДанных (DataCompositionAvailableParameterUseRestrictions)

Индекс (IndexOf)

Доступен, начиная с версии 8.2.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ОграничениеИспользованияДоступногоПоляКомпоновкиДанных](ОграничениеИспользованияДоступногоПоляКомпоновкиДанных_(DataCompositionAvailableFieldUseRestriction).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
