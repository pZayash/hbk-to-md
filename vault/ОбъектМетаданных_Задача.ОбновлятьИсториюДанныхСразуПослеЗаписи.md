ОбъектМетаданных: Задача (MetadataObject: Task)

ОбновлятьИсториюДанныхСразуПослеЗаписи (UpdateDataHistoryImmediatelyAfterWrite)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если установлено значение Истина, то после завершения транзакции  
выполняется обновление истории данных по записанным данным,   
в фоновом задании без ожидания выполнения завершения фонового задания.  
Значение по умолчанию - Ложь.  
При записи данных значение, указанное в конфигураторе, записывается в свойство [ОбновлятьИсториюСразуПослеЗаписи](ПараметрыЗаписиИсторииДанных.ОбновлятьИсториюСразуПослеЗаписи.md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
