Расширение табличного поля отбора (Filter table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора.md).   
Предоставляет доступ к настройкам отбора списка отборов. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для списка отборов элементов отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
