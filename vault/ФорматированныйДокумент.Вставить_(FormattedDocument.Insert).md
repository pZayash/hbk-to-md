ФорматированныйДокумент (FormattedDocument)

Вставить (Insert)

Доступен, начиная с версии 8.2.

Синтаксис:

Вставить(<Закладка>, <Значение>, <ТипЭлемента>)

Параметры:

<Закладка> (обязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Определяет позицию вставки, в которую необходимо вставить элемент форматированного документа.

<Значение> (необязательный)

Тип: Произвольный.   
Содержит вставляемое в форматированный документ значение.  
В зависимости от значения параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.  
Если значение параметра [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), то в указанную позицию документа будет вставлено строковое представление переданного значения.  
Если значение [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md), то тип параметра должен быть [Картинка](Картинка_(Picture).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Строка](lang__def_String.md), содержащая URL картинки. В противном случае вызывается исключение. Картинка будет вставлена в указанное место.  
Если значение параметра [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md), то независимо от значения параметра в указанную позицию вставляется перевод строки.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ТипЭлемента> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Указывает тип элемента, который нужно вставить. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Элементами коллекции могут быть:  
1) [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md)  
2) [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md)  
3) [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md)  
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Значение по умолчанию: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md).

Возвращаемое значение:

Тип: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md).   
Возращает элемент, вставленный в форматированный документ.

Описание:

Вставляет строку, картинку или перевод строки в указанную позицию форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Вставка картинки доступна только на сервере.  
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
