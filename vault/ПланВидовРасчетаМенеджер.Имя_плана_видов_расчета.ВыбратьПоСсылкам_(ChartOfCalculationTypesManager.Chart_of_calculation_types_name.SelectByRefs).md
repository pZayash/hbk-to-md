# ПланВидовРасчетаМенеджер.<Имя плана видов расчета>.ВыбратьПоСсылкам

ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [Массив](Массив_(Array).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Источник ссылок на элементы плана видов расчета.

<Колонка> (необязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Колонка источника ссылок. [Число](lang__def_Number.md) представляет индекс колонки источника. [Строка](lang__def_String.md) представляет имя колонки. Если значением параметра является [Неопределено](lang__def_Undefined.md), то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив_(Array).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [ПланВидовРасчетаВыборка.\<Имя плана видов расчета\>](ПланВидовРасчетаВыборка.Имя_плана_видов_расчета_(ChartOfCalculationTypesSelection.Chart_of_calculation_types_name).md).   

Описание:

Формирует выборку элементов плана видов расчета по заданному источнику ссылок на элементы плана видов расчета соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы плана видов расчета соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    Мобильные.Ссылка КАК Ссылка     |ИЗ     |    ПланОбмена.Мобильные КАК Мобильные     |"; РезультатЗапроса = Запрос.Выполнить(); Выборка = ПланыОбмена.Мобильные.ВыбратьПоСсылкам(РезультатЗапроса); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
