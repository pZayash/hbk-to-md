СправочникСсылка.<Имя справочника> (CatalogRef.<Catalog name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [СправочникОбъект.\<Имя справочника\>](СправочникОбъект.Имя_справочника.md).   

Описание:

Создает новый элемент справочника копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Если Не ТоварСсылка.Пустая() Тогда     Копия = ТоварСсылка.Скопировать();     Копия.Записать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
