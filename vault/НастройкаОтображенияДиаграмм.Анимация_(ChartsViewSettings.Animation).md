НастройкаОтображенияДиаграмм (ChartsViewSettings)

Анимация (Animation)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [АнимацияДиаграммы](АнимацияДиаграммы_(ChartAnimation).md).   
Указывает необходимость использования анимации для диаграмм. Используется для диаграмм, у которых свойство [Анимация](Диаграмма.Анимация_(Chart.Animation).md) имеет значение [Авто](АнимацияДиаграммы.Авто_(ChartAnimation.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
