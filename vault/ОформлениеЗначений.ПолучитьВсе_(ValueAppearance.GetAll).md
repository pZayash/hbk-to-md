ОформлениеЗначений (ValueAppearance)

ПолучитьВсе (GetAll)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьВсе(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Тип ключей в возвращаемом соответствии.

Возвращаемое значение:

Тип: [Соответствие](Соответствие_(Map).md).   

Описание:

Возвращает соответствие, в котором все ключи имеют тип, указанный в параметре <Тип>.   
Если тип не указан, то возвращает соответствие, содержащее все записи.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
