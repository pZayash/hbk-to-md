СпособВыбораСертификатаWindows (WindowsCertificateSelectMode)

Выбирать (Choose)

Доступен, начиная с версии 8.3.1.

Описание:

При наличии более одного подходящего клиентского сертификата предоставлять выбор через системный диалог выбора сертификатов.

Использование в версии:

Доступен, начиная с версии 8.3.1.
