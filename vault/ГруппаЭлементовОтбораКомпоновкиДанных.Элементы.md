ГруппаЭлементовОтбораКомпоновкиДанных (DataCompositionFilterItemGroup)

Элементы (Items)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовОтбораКомпоновкиДанных](КоллекцияЭлементовОтбораКомпоновкиДанных.md).   
Содержит вложенные элементы отбора.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
