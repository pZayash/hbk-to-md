Действие (Action)

На основании процедуры

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Действие(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры модуля.

Описание:

Создает объект из процедуры модуля.

Пример:

|  |
| --- |
| Процедура ОбработкаНажатия()     Сообщить("Нажата кнопка"); КонецПроцедуры;  Действие = Новый Действие("ОбработкаНажатия"); ЭлементыФормы.НоваяКнопка.УстановитьДействие("Нажатие", Действие); |

Использование в версии:

Доступен, начиная с версии 8.0.
