# ОбработкаОбъект.<Имя обработки>.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Обработки](Обработки.md) › [ОбработкаОбъект.<Имя обработки>](ОбработкаОбъект.Имя_обработки_(DataProcessorObject.Data_processor_name).md) › [Методы](ОбработкаОбъект.Имя_обработки__Методы.md)

ОбработкаОбъект.<Имя обработки> (DataProcessorObject.<Data processor name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](ОбработкаОбъект.Имя_обработки.ОбработкаПроверкиЗаполнения_(DataProcessorObject.Data_processor_name.FillCheckProcessing).md).

Доступен, начиная с версии 8.2.
