МенеджерДополнительныхНастроекАутентификации (AdditionalAuthenticationSettingsManager)

УстановитьНастройкиПроверкиРаскрытияПароля (SetPasswordCompromiseCheckSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьНастройкиПроверкиРаскрытияПароля(<НастройкиПроверкиРаскрытияПароля>)

Параметры:

<НастройкиПроверкиРаскрытияПароля> (обязательный)

Тип: [НастройкиПроверкиРаскрытияПароля](НастройкиПроверкиРаскрытияПароля_(PasswordCompromiseCheckSettings).md).   
Устанавливаемые настройки.

Описание:

Изменяет настройки проверки раскрытия пароля пользователей информационной базы.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
