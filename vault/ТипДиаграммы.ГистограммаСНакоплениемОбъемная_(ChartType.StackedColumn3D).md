ТипДиаграммы (ChartType)

ГистограммаСНакоплениемОбъемная (StackedColumn3D)

Доступен, начиная с версии 8.0.

Описание:

Вертикальная объемная гистограмма с накоплением: значения в каждой серии суммируются со значениями в предыдущих. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График_(ChartType.Line).md).

Использование в версии:

Доступен, начиная с версии 8.0.
