ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

УстановитьНовыйНомер (SetNewNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьНовыйНомер(<ПрефиксНомера>)

Параметры:

<ПрефиксНомера> (необязательный)

Тип: Строка.   
Префикс устанавливаемого номера.  
Значение по умолчанию: Пустая строка.

Описание:

Устанавливает новый номер задачи для заданного префикса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:
