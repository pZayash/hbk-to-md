ЧтениеPDF (PDFReader)

На основании имени файла

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЧтениеPDF(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из которого будет выполнено чтение данных.

Описание:

Создает объект [ЧтениеPDF](ЧтениеPDF_(PDFReader).md) из заданного файла.   
При этом будет файл, указанный в параметре ИмяФайла, будет автоматически открыт на чтение.   
Если файл с таким именем не существует, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
