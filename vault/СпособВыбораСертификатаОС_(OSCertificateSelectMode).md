# СпособВыбораСертификатаОС

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интернет](Интернет.md)

Доступен, начиная с версии 8.3.21.

Значения

[Авто (Auto)](СпособВыбораСертификатаОС.Авто_(OSCertificateSelectMode.Auto).md)  
[Выбирать (Choose)](СпособВыбораСертификатаОС.Выбирать_(OSCertificateSelectMode.Choose).md)  

Описание:

Содержит варианты способов выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы.  
При работе в ОС Linux значение игнорируется.  
В ОС Windows и macOS действует аналогично объектам [СпособВыбораСертификатаWindows](СпособВыбораСертификатаWindows_(WindowsCertificateSelectMode).md) и [СпособВыбораСертификатаMacOS](СпособВыбораСертификатаMacOS_(MacOSCertificateSelectMode).md).

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

См. также:

[СертификатКлиентаОС](СертификатКлиентаОС_(OSClientCertificate).md)  

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.24.
