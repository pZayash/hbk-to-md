ОбъектМетаданных: ШаблонURLHTTPСервиса (MetadataObject: HTTPServiceURLTemplate)

Методы (Methods)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
HTTP-методы, которые обрабатываются по путям, удовлетворяющим шаблону.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: МетодHTTPСервиса](ОбъектМетаданных_МетодHTTPСервиса_(MetadataObject_HttpServiceMethod).md)

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
