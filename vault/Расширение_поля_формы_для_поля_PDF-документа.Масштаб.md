Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

Масштаб (Scale)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет дополнительный (к масштабу формы) масштаб документа PDF.  
Значение по умолчанию: 100.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
