Глобальный контекст (Global context)

ПоказатьВводДаты (ShowInputDate)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьВводДаты(<ОписаниеОповещенияОЗавершении>, <Дата>, <Подсказка>, <ЧастьДаты>)

Параметры:

<ОписаниеОповещенияОЗавершении> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия окна ввода даты со следующими параметрами:

- <Дата> - выбранное значение даты или [Неопределено](lang__def_Undefined.md), если пользователь отказался от ввода,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Дата> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Начальное значение даты, отображаемое в окне ввода.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода даты. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<ЧастьДаты> (необязательный)

Тип: [ЧастиДаты](ЧастиДаты_(DateFractions).md).   
Вводимая в диалоге часть (или части) даты.  
Значение по умолчанию: [ДатаВремя](ЧастиДаты.ДатаВремя_(DateFractions.DateTime).md).

Возвращаемое значение:

[Истина](lang__def_BooleanTrue.md) - дата введена; [Ложь](lang__def_BooleanFalse.md) - пользователь отказался от ввода даты.

Описание:

Открывает окно для ввода даты, но при этом не ожидает завершения ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| ДатаНапоминания = РабочаяДата; Подсказка = "Введите дату и время"; ЧастьДаты = ЧастиДаты.ДатаВремя; Оповещение = Новый ОписаниеОповещения("ПослеВводаДаты", , Параметры); ПоказатьВводДаты(Оповещение, ДатаНапоминания, Подсказка, ЧастьДаты); //...   &НаКлиенте Процедура ПослеВводаДаты(Дата, Параметры) Экспорт     Если НЕ Дата = Неопределено Тогда         // запомнить дату напоминания      КонецЕсли; КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.22.
