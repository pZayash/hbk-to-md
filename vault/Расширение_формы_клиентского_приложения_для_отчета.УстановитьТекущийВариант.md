Расширение формы клиентского приложения для отчета (Client application form extension for reports)

УстановитьТекущийВариант (SetCurrentVariant)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьТекущийВариант(<КлючВарианта>)

Параметры:

<КлючВарианта> (обязательный)

Тип: Произвольный.   
Ключ варианта, который нужно загрузить.

Описание:

Загружает вариант и пользовательские настройки, которые последними использовались для этого варианта. Заполняет свойства [КлючТекущегоВарианта](Расширение_формы_клиентского_приложения_для_отчета.КлючТекущегоВарианта.md), [ПредставлениеТекущегоВарианта](Расширение_формы_клиентского_приложения_для_отчета.ПредставлениеТекущегоВарианта.md), [КлючТекущихПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.КлючТекущихПользовательскихНастроек.md), [ПредставлениеТекущихПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.ПредставлениеТекущихПользовательскихНастроек.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
