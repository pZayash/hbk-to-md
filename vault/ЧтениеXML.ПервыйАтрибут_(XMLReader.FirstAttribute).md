# ЧтениеXML.ПервыйАтрибут

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md) › [ЧтениеXML](ЧтениеXML_(XMLReader).md) › [Методы](ЧтениеXML__Методы.md)

ЧтениеXML (XMLReader)

ПервыйАтрибут (FirstAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

ПервыйАтрибут()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - атрибут прочитан, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Позиционирует текущее состояние объекта на первый атрибут текущего элемента.   
При этом свойствам [ЧтениеXML.ТипУзла](ЧтениеXML.ТипУзла_(XMLReader.NodeType).md), [ЧтениеXML.Имя](ЧтениеXML.Имя_(XMLReader.Name).md), [ЧтениеXML.ЛокальноеИмя](ЧтениеXML.ЛокальноеИмя_(XMLReader.LocalName).md), [ЧтениеXML.Префикс](ЧтениеXML.Префикс_(XMLReader.Prefix).md), [ЧтениеXML.URIПространстваИмен](ЧтениеXML.URIПространстваИмен_(XMLReader.NamespaceURI).md), [ЧтениеXML.Значение](ЧтениеXML.Значение_(XMLReader.Value).md) и т.д. присваиваются значения, соответствующие прочитанным данным атрибута.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл, если текущим узлом является [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md) или [Атрибут](ТипУзлаXML.Атрибут_(XMLNodeType.Attribute).md).

Использование в версии:

Доступен, начиная с версии 8.1.
