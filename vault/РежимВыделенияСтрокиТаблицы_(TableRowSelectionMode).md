# РежимВыделенияСтрокиТаблицы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[Строка (Row)](РежимВыделенияСтрокиТаблицы.Строка_(TableRowSelectionMode.Row).md)  
[Ячейка (Cell)](РежимВыделенияСтрокиТаблицы.Ячейка_(TableRowSelectionMode.Cell).md)  

Описание:

Содержит варианты режима выделения строки таблицы формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы_(FormTable).md), свойство [РежимВыделенияСтроки](ТаблицаФормы.РежимВыделенияСтроки_(FormTable.RowSelectionMode).md)  

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.27.
