ГеографическаяСхема (GeographicalSchema)

Вывести (Put)

Доступен, начиная с версии 8.0.

Синтаксис:

Вывести(<Схема>)

Параметры:

<Схема> (обязательный)

Тип: [ГеографическаяСхема](ГеографическаяСхема_(GeographicalSchema).md).   
Выводимая схема.

Описание:

Выводит в схему данные из макета географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
