# Расширение группы формы для группы кнопок.Отображение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Группа формы](Группа_формы.md) › [Расширение группы формы для группы кнопок](Расширение_группы_формы_для_группы_кнопок_(Form_group_extension_for_a_button_group).md) › [Свойства](Расширение_группы_формы_для_группы_кнопок__Свойства.md)

Расширение группы формы для группы кнопок (Form group extension for a button group)

Отображение (Representation)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеГруппыКнопок](ОтображениеГруппыКнопок_(ButtonGroupRepresentation).md).   
Задает отображение группы кнопок командной панели.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
