Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступныйПараметрКомпоновкиДанных](ДоступныйПараметрКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные параметры.  
Возможно обращение к доступному параметру посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[Найти (Find)](КоллекцияДоступныхПараметровКомпоновкиДанных.Найти.md)  
[Получить (Get)](КоллекцияДоступныхПараметровКомпоновкиДанных.Получить.md)  

Описание:

Коллекция доступных параметров.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
