# ПланСчетовОбъект.<Имя плана счетов>.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Методы](ПланСчетовОбъект.Имя_плана_счетов__Методы.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет элемент плана счетов из базы данных.  
Важно! В отличие от метода [УстановитьПометкуУдаления](ПланСчетовОбъект.Имя_плана_счетов.УстановитьПометкуУдаления_(ChartOfAccountsObject.Chart_of_accounts_name.SetDeletionMark).md), данный метод производит непосредственное удаление элемента плана счетов без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Также удаляются все подчиненные элементы.

Пример:

|  |
| --- |
| Объект.Удалить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
