# ЗаписьДанных.СброситьБуферыАсинх

ЗаписьДанных (DataWriter)

СброситьБуферыАсинх (FlushAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СброситьБуферыАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Сбрасывает все внутренние буферы в целевой поток, после чего вызывает метод [СброситьБуферы](Поток.СброситьБуферы_(Stream.Flush).md) у целевого потока.

Доступность:

Тонкий клиент, толстый клиент.

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [НачатьСбросБуферов](ЗаписьДанных.НачатьСбросБуферов_(DataWriter.BeginFlush).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
