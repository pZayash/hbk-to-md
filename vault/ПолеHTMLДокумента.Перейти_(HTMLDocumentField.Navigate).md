ПолеHTMLДокумента (HTMLDocumentField)

Перейти (Navigate)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По URL

Синтаксис:

Перейти(<URL>)

Параметры:

<URL> (обязательный)

Тип: [Строка](lang__def_String.md).   
URL ресурса, который будет загружен в элемент управления.

Вариант синтаксиса: По объекту метаданных

Синтаксис:

Перейти(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет_(MetadataObject_Template).md).   
Макет HTML-документа, на который должен перейти элемент управления.

Описание:

Переводит поле HTML-документа на страницу с заданным адресом.

Доступность:

Толстый клиент.

Примечание:

Не доступен в режиме работы [Редактирование](РежимПоляHTMLДокумента.Редактирование_(HTMLDocumentFieldMode.Design).md).

Пример:

|  |
| --- |
| // в форме расположен элемент управления ПолеHTMLДокумента1   // типа "ПолеHTMLДокумента"   Процедура ПриОткрытии()     // ...      // для варианта метода "по URL":      ЭлементыФормы.ПолеHTMLДокумента1.Перейти("www.example.com");     // ...  КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
