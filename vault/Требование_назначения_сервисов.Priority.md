Требование назначения сервисов (IAssignmentRule)

Priority (Priority)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Приоритет требования. Для объекта требования при размещении вычисляется эффективное требование для каждого из серверов. Объект будет размещен там, где приоритет выше (при одинаковых типах требования).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
