Расширение поля формы для поля ввода (Form field extension for a text box)

ПутьКДаннымКартинкиМножественногоЗначения (MultipleValuePictureDataPath)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит путь к реквизиту, содержащему картинку множественного значения.   
Для получения картинки множественных значений в свойстве поля ввода "ПутьКДаннымКартинкиМножественногоЗначения" может быть указана колонка c типом [Картинка](Картинка_(Picture).md), [Строка](lang__def_String.md), [Число](lang__def_Number.md) или [Булево](lang__Булево_(Boolean).md):

- [Строка](lang__def_String.md) - значение в колонке представляет собой навигационную ссылку с адресом картинки.
- [Число](lang__def_Number.md) - значение в колонке представляет собой индекс в наборе картинок, задаваемых свойством [КартинкаМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.КартинкаМножественныхЗначений_(Form_field_extension_for_a_text_box.MultipleValuesPicture).md).
- [Булево](lang__Булево_(Boolean).md) - если [Истина](lang__def_BooleanTrue.md) используется индекс 0, если [Ложь](lang__def_BooleanFalse.md) используется индекс 1.

Доступность:

Сервер.
