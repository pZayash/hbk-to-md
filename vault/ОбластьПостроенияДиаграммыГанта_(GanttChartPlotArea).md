Описание:

Используется для доступа к области построения диаграммы Ганта для настройки параметров оформления.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md), свойство [ОбластьПостроения](ДиаграммаГанта.ОбластьПостроения_(GanttChart.PlotArea).md)

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.23.
