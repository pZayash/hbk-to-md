ДанныеКонтакта (ContactData)

Отношения (Relationships)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Содержит массив отношений контакта в виде объектов типа [ЭлементДанныхКонтакта](ЭлементДанныхКонтакта.md).   
Свойство [ТипДанных](ЭлементДанныхКонтакта.ТипДанных.md) содержит значение перечисления [ТипОтношенийДанныхКонтакта](ТипОтношенийДанныхКонтакта.md) или строку. Свойство [Значение](ЭлементДанныхКонтакта.Значение.md) содержит строку с именем человека.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
