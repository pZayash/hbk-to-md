ВложенныйНаборДанныхСхемыКомпоновкиДанных (DataCompositionSchemaNestedDataSet)

Заголовок (Title)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит заголовок вложенного набора данных. Этот заголовка будет использоваться для отображения в настройках отчета и результате отчета.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
