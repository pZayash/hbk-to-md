ДанныеПереходаПоНавигационнойСсылке (URLNavigationData)

ПараметрыНавигационнойСсылки (URLParameters)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит параметры ссылки, которые следуют после знака вопроса:

- Ключ – тип [Строка](lang__def_String.md), имя параметра.
- Значение – [Строка](lang__def_String.md) или [Массив](Массив_(Array).md) строк, значение параметра.

Доступность:

Тонкий клиент, веб-клиент, сервер, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
