Диаграмма (Chart)

НачальныйУголИзмерительнойДиаграммы (BeginGaugeAngle)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит начальный угол шкалы измерительной диаграммы. Задается в градусах от положительной оси Х в диапазоне 0 - 360.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Должен быть больше значения свойства [КонечныйУголИзмерительнойДиаграммы](Диаграмма.КонечныйУголИзмерительнойДиаграммы_(Chart.EndGaugeAngle).md).

Использование в версии:

Доступен, начиная с версии 8.0.
