ВременныеТаблицыЗапроса (QueryTempTables)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Индекс(<ВременнаяТаблица>)

Параметры:

<ВременнаяТаблица> (обязательный)

Тип: [ВременнаяТаблицаЗапроса](ВременнаяТаблицаЗапроса.md).   
Временная таблица, для которой нужно определить индекс.

Возвращаемое значение:

Тип: Число.   
Индекс указанной колонки в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс колонки результата запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| РезультатЗапроса.Колонки.Индекс(Колонка); |

Использование в версии:

Доступен, начиная с версии 8.3.8.
