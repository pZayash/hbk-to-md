ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Измерение>, <Смещение>)

Параметры:

<Измерение> (обязательный)

Тип: [ИзмерениеПостроителяОтчета](ИзмерениеПостроителяОтчета_(ReportBuilderDimension).md).   
Измерение, которое надо переместить.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число элементов, на которое необходимо сместить измерение.

Описание:

Сдвигает измерение на указанное смещение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
