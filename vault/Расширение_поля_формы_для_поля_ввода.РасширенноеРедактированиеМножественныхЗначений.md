Расширение поля формы для поля ввода (Form field extension for a text box)

РасширенноеРедактированиеМножественныхЗначений (MultipleValuesExtendedEdit)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Значение по умолчанию: Истина.  
  
Если свойство установлено Истина, то ввод множественных значений осуществляется непосредственно внутри поля ввода посредством ввода с клавиатуры, через форму выбора или выпадающий список. Кнопка открытия открывает диалог редактирования списка значений, если источником данных для множественных значений является [СписокЗначений](СписокЗначений.md).  
  
В противном случае, если поле ввода редактирует [СписокЗначений](СписокЗначений.md), то ввод множественных значений осуществляется только через диалог редактирования списка значений, который открывается при нажатии на кнопку выбора. Если же поле ввода связано с реквизитом типа [ТаблицаЗначений](ТаблицаЗначений.md) или [Табличная часть](Табличная_часть.md), множественные значения отображаются в поле ввода без возможности редактирования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
