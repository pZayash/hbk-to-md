ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

РежимСовместимости (CompatibilityMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РежимСовместимости](РежимСовместимости_(CompatibilityMode).md).   
Показывает, установлен ли режим совместимости конфигурации с платформой указанной версии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
