КолонкиТабличногоПоля (TableBoxColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md).   
Колонка табличного поля.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной колонки в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс колонки в коллекции колонок.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
