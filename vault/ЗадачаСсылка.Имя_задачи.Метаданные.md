ЗадачаСсылка.<Имя задачи> (TaskRef.<Task name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Задача](ОбъектМетаданных_Задача.md).   

Описание:

Предоставляет доступ к объекту описания метаданных задачи. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.Задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для каждого Реквизит Из Задача.Метаданные().Реквизиты Цикл     Сообщить(Реквизит.Представление()); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
