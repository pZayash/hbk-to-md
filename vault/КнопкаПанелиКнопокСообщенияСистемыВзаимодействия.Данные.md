КнопкаПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButton)

Данные (Data)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит данные кнопки, передаваемые в сообщении, которое будет отправлено при нажатии на кнопку, если действие кнопки - [ОтправитьСообщениеСДанными](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщениеСДанными.md).  
Значение должно поддерживать XDTO-сериализацию.  
При сохранении объекта значение сериализуется в формат JSON.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
