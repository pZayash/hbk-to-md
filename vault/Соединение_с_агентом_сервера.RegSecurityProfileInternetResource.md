Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfileInternetResource (RegSecurityProfileInternetResource)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfileInternetResource(<Кластер>, <ИмяПрофиля>, <РесурсИнтернет>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должен быть зарегистрирован ресурс Интернет.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<РесурсИнтернет> (обязательный)

Тип: [Ресурс интернет](Ресурс_интернет.md).   
Регистрируемый ресурс сети Интернет.

Описание:

Выполняет регистрацию нового разрешенного ресурса Интернета или обновление существующего.

Доступность:

Интеграция.

Примечание:

Если разрешенный ресурс существует, то выполняется обновление, в противном случае - добавление.  
Если профиль не найден, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
