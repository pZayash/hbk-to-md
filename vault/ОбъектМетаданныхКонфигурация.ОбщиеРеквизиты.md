ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОбщиеРеквизиты (CommonAttributes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все общие реквизиты, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ОбщийРеквизит](ОбъектМетаданных_ОбщийРеквизит.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
