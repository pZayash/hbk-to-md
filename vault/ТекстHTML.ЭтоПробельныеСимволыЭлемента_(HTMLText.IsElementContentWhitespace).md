ЭтоПробельныеСимволыЭлемента (IsElementContentWhitespace)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Содержит признак пробельных символов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - текст является пробельными символами узла [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md). [Ложь](lang__def_BooleanFalse.md) - в противном случае (игнорируемые пробельные символы).

Использование в версии:

Доступен, начиная с версии 8.2.
