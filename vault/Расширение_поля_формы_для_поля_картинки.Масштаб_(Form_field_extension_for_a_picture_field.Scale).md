# Расширение поля формы для поля картинки.Масштаб

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля картинки](Расширение_поля_формы_для_поля_картинки_(Form_field_extension_for_a_picture_field).md) › [Свойства](Расширение_поля_формы_для_поля_картинки__Свойства.md)

Расширение поля формы для поля картинки (Form field extension for a picture field)

Масштаб (Scale)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет дополнительный (к масштабу формы) масштаб картинки.  
Значение по умолчанию: 100.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме совместимости [Версия8\_3\_6](РежимСовместимости.Версия8_3_6_(CompatibilityMode.Version8_3_6).md) и ниже изменение свойства игнорируется.

Использование в версии:

Доступен, начиная с версии 8.3.15.
