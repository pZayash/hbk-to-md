Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ПолучитьРазмерОбластиДанныхДокументаПоГоризонтали (GetDocumentDataAreaHorizontalSize)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьРазмерОбластиДанныхДокументаПоГоризонтали()

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает максимальный номер колонки, содержащей данные любого типа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
