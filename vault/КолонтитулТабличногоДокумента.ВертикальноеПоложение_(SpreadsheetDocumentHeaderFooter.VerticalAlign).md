КолонтитулТабличногоДокумента (SpreadsheetDocumentHeaderFooter)

ВертикальноеПоложение (VerticalAlign)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md).   
Определяет способ выравнивания текста колонтитула по вертикали.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
