# Глобальный контекст.ПоместитьФайлы

Глобальный контекст (Global context)

ПоместитьФайлы (PutFiles)

Не рекомендуется использовать, начиная с версии 8.3.13.

Рекомендуется использовать:

- [НачатьПомещениеФайловНаСервер](Глобальный_контекст.НачатьПомещениеФайловНаСервер_(Global_context.BeginPutFilesToServer).md)

Доступен, начиная с версии 8.2.

Синтаксис:

ПоместитьФайлы(<ПомещаемыеФайлы>, <ПомещенныеФайлы>, <НаборПомещаемыхФайлов>, <Интерактивно>, <УникальныйИдентификаторФормы>)

Параметры:

<ПомещаемыеФайлы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Массив объектов типа [ОписаниеПередаваемогоФайла](ОписаниеПередаваемогоФайла_(TransferableFileDescription).md).  
Каждый объект описывает получаемый файл:

- [Хранение](ОписаниеПередаваемогоФайла.Хранение_(TransferableFileDescription.Location).md) - расположение данных во временном хранилище;
- [Имя](ОписаниеПередаваемогоФайла.Имя_(TransferableFileDescription.Name).md) - это локальный путь, по которому должен быть получен файл.

Массив может быть заполнен полностью, в этом случае известен набор помещаемых файлов и адреса, по которым они должны быть сохранены.  
Массив может быть заполнен частично, у элементов массива заполнены только имена. В этом случае, набор помещаемых файлов определен, а места хранения неизвестны. Файлы будут размещены в новых временных хранилищах.  
Массив может быть не заполнен. В этом случае набор помещаемых файлов определяется по значениям остальных параметров метода (поиском по маске либо интерактивным выбором).

<ПомещенныеФайлы> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ОписаниеПереданногоФайла](ОписаниеПереданногоФайла_(TransferredFileDescription).md).  
Каждый объект описывает помещенный файл:  
[Хранение](ОписаниеПереданногоФайла.Хранение_(TransferredFileDescription.Location).md) - расположение данных во временном хранилище;  
[Имя](ОписаниеПереданногоФайла.Имя_(TransferredFileDescription.Name).md) - это локальный путь, по которому получен файл.

<НаборПомещаемыхФайлов> (необязательный)

Тип: [ДиалогВыбораФайла](ДиалогВыбораФайла_(FileDialog).md), [Строка](lang__def_String.md).   
Определяет набор помещаемых файлов.  
Может быть задан строкой поиска для метода [НайтиФайлы](Глобальный_контекст.НайтиФайлы_(Global_context.FindFiles).md) либо заранее подготовленным диалогом выбора файлов в режиме открытия файлов.  
Если параметр определен, то параметр <ПомещаемыеФайлы> должен быть не определен либо быть пустым массивом.

<Интерактивно> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает использование интерактивного режима, при котором пользователю показывается диалог, в котором он может выбрать набор помещаемых файлов.

<УникальныйИдентификаторФормы> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор формы, который можно получить с помощью свойства [УникальныйИдентификатор](ФормаКлиентскогоПриложения.УникальныйИдентификатор_(ClientApplicationForm.UUID).md) управляемой формы.   
Файл помещается во временное хранилище и автоматически удаляется после удаления объекта формы.  
Если параметр не указан, помещенное значение будет удалено, после очередного запроса сервера из общего модуля, при контекстном и неконтекстном серверном вызове из формы, при серверном вызове из модуля команды или при получении формы.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Помещает набор файлов из локальной файловой системы во временное хранилище.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
