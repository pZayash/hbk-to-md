# ЭлементТаблицаHTML.ВставитьСтроку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md) › [ЭлементТаблицаHTML](ЭлементТаблицаHTML_(HTMLTableElement).md) › [Методы](ЭлементТаблицаHTML__Методы.md)

ЭлементТаблицаHTML (HTMLTableElement)

ВставитьСтроку (InsertRow)

Доступен, начиная с версии 8.2.

Синтаксис:

ВставитьСтроку(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс строки, после которой будет добавлена новая строка.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML_(HTMLElement).md).   

Описание:

Предназначен для добавления строки в таблицу HTML-документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
