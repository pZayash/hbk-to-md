МенеджерОформленияОтчетов (ReportsAppearanceManager)

УстановитьОбщийМакетОформления (SetCommonAppearanceTemplate)

Доступен, начиная с версии 8.3.22.

Синтаксис:

УстановитьОбщийМакетОформления(<ИмяМакета>)

Параметры:

<ИмяМакета> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя макета оформления компоновки данных. Чтобы использовать системный макет оформления, необходимо передать пустую строку.

Описание:

Устанавливает общий для всех пользователей макет оформления компоновки данных, используемый по умолчанию. Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.22.
