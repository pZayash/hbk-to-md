Расширение поля формы для поля ввода (Form field extension for a text box)

УстановитьВыделенныеМножественныеЗначения (SetSelectedMultipleValues)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьВыделенныеМножественныеЗначения(<Идентификаторы>)

Параметры:

<Идентификаторы> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [Число](lang__def_Number.md).  
Массив идентификаторов объектов [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md) или [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md) для выделения множественных значений.  
Эти идентификаторы не привязаны к позиции элемента в списке или таблице значений.

Описание:

Устанавливает выделение множественных значений.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
