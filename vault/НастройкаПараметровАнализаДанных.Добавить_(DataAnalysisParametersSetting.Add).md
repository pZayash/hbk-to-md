НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента параметров.

<Доступность> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Добавляет элемент управления параметрами в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
