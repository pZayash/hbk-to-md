ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

Интеграция (Integration)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия.md), Неопределено.   
Содержит идентификатор интеграции, если это обсуждение с пользователем внешней системы.  
  
Установка свойства возможна только для нового незаписанного обсуждения.  
Если свойство установлено, то при записи создается обсуждение с внешней системой интеграции.  
При получении первого сообщения в это обсуждение в список участников будет добавлен пользователь интеграции. Если список участников пуст, то также будут добавлены "пользователи интеграции по умолчанию".  
Создание обсуждения с внешней системой интеграции имеет смысл только для интеграций типа WhatsApp и Webhook.

Доступность:

Тонкий клиент, веб-клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.27.
