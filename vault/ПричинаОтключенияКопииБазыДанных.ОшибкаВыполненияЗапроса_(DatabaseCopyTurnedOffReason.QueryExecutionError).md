ПричинаОтключенияКопииБазыДанных (DatabaseCopyTurnedOffReason)

ОшибкаВыполненияЗапроса (QueryExecutionError)

Доступен, начиная с версии 8.3.27.

Описание:

Во время выполнения запроса произошла ошибка СУБД, в которой расположена копия.

Использование в версии:

Доступен, начиная с версии 8.3.27.
