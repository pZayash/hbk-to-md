ОписаниеКомандыПланировщика (PlannerCommandDescription)

Команда (Command)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [Команда](ОписаниеКомандыПланировщика.Команда_(PlannerCommandDescription.Command).md)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [СтандартнаяКомандаПланировщика](СтандартнаяКомандаПоляПланировщика_(PlannerStandardCommand).md), [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), [Массив](Массив_(Array).md), [Неопределено](lang__def_Undefined.md).   
Содержит описание команды:

- [СтандартнаяКомандаПланировщика](СтандартнаяКомандаПоляПланировщика_(PlannerStandardCommand).md) - стандартная команда;
- [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md) – процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
  - <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md);
- [Массив](Массив_(Array).md) – задает подменю. Массив может содержать элементы обозначенных выше типов;
- [Неопределено](lang__def_Undefined.md) – разделитель;
- Любой другой тип - пункт меню не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
