# Расширение табличного поля списка параметров анализа данных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Свойства:

[НастройкаПараметров (ParametersSetting)](Расширение_табличного_поля_списка_параметров_анализа_данных.НастройкаПараметров_(Data_analysis_parameter_list_table_box_extension.ParametersSetting).md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для списка параметров анализа данных. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [ПараметрАнализаДанных](ПараметрАнализаДанных_(DataAnalysisParameter).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
