РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRow)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   
Индекс объекта или сам объект, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое необходимо сдвинуть ряд кнопок.

Описание:

Сдвигает указанную кнопку на заданное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
