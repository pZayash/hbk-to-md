ОбластьПостроенияДиаграммыГанта (GanttChartPlotArea)

ОтображатьТекстТочек (ShowPointsText)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеВДиаграммеГанта](ОтображениеВДиаграммеГанта_(ShowInGanttChart).md).   
Настройка отображения текста точек в области построения.   
Значение [Авто](ОтображениеВДиаграммеГанта.Авто_(ShowInGanttChart.Auto).md) интерпретируется следующим образом: если у диаграммы как элемента формы включено отображение таблицы в качестве предопределенного элемента (значение свойства [ПоложениеТаблицы](Расширение_поля_формы_для_поля_диаграммы_Ганта.ПоложениеТаблицы_(Form_field_extension_for_a_Gantt_chart_field.TableLocation).md) поля диаграммы не равно [Нет](ПоложениеТаблицыДиаграммыГанта.Нет_(GanttChartTableLocation.None).md)), то [НеОтображать](ОтображениеВДиаграммеГанта.НеОтображать_(ShowInGanttChart.DontShow).md), иначе - [Отображать](ОтображениеВДиаграммеГанта.Отображать_(ShowInGanttChart.Show).md).  
Значение по умолчанию - [Авто](ОтображениеВДиаграммеГанта.Авто_(ShowInGanttChart.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
