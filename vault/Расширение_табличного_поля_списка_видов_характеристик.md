Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для вида характеристик. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ПланВидовХарактеристикСписок.](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик.md)>.

Доступность:

Толстый клиент.
