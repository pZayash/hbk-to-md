МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьНовыйКлючАдминистрирования (GetNewAdministrationKey)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПолучитьНовыйКлючАдминистрирования()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает новый ключ администрирования абонента. Предыдущий ключ администрирования перестает действовать.  
Если текущий пользователь не является администратором абонента, то генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
