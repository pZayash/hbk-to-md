ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Расписание](ДиалогРасписанияРегламентногоЗадания.Расписание_(ScheduledJobDialog.Schedule).md), если диалог был закрыт нажатием кнопки "OK", [Неопределено](lang__def_Undefined.md) - в противном случае.

Описание:

Асинхронная функция. Открывает немодальный диалог расписания.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
