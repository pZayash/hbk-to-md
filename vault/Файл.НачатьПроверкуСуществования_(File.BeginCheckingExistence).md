Файл (File)

НачатьПроверкуСуществования (BeginCheckingExistence)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПроверкуСуществования(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <Существует> - помещается признак существования, тип [Булево](lang__Булево_(Boolean).md). [Истина](lang__def_BooleanTrue.md), если файл существует.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Начинает проверку существования файла.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Файл](Файл_(File).md), метод [Существует](Файл.Существует_(File.Exists).md)  
[Файл](Файл_(File).md), метод [СуществуетАсинх](Файл.СуществуетАсинх_(File.ExistsAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
