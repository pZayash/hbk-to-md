# ДиаграммаГанта.РазвернутьТочку

ДиаграммаГанта (GanttChart)

РазвернутьТочку (ExpandPoint)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутьТочку(<Точка>, <СПодчиненными>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

<СПодчиненными> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость раскрытия точки вместе с подчиненными точками. [Истина](lang__def_BooleanTrue.md) - раскрывать вместе с подчиненными.

Описание:

Раскрывает точку, возможно вместе с подчинёнными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
