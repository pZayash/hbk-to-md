# Расширение поля формы для поля диаграммы Ганта.Выбор

Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<Значения>, <СтандартнаяОбработка>, <Дата>)

Параметры:

<Значения>

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий выбранные объекты.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Дата>

Тип: [Дата](lang__def_Date.md).   
Дата, соответствующая точке выбора. Применима для значения и интервала диаграммы Ганта. При выборе других объектов принимает значение [Неопределено](lang__def_Undefined.md).

Описание:

Событие, возникающее при выборе диаграммы Ганта (например, при двойном щелчке мыши).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
