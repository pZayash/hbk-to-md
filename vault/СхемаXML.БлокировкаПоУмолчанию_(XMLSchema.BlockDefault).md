СхемаXML (XMLSchema)

БлокировкаПоУмолчанию (BlockDefault)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОбъединениеНедопустимыхПодстановкиXS](ОбъединениеНедопустимыхПодстановкиXS_(XSDisallowedSubstitutionsUnion).md).   
Содержит значение блокировки подстановки по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет значение blockDefault (<http://www.w3.org/TR/xmlschema-1/#element-schema>) информационного пространства определения.

Использование в версии:

Доступен, начиная с версии 8.1.
