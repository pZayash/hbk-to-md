# ЗаписьУзловDOM.ЗаписатьСсылкуНаСущность

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ЗаписьУзловDOM](ЗаписьУзловDOM_(DOMNodeWriter).md) › [Методы](ЗаписьУзловDOM__Методы.md)

ЗаписьУзловDOM (DOMNodeWriter)

ЗаписатьСсылкуНаСущность (WriteEntityReference)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьСсылкуНаСущность(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя ссылки на сущность.

Описание:

Осуществляет запись ссылки на сущность.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
