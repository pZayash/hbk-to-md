# КонтейнерКлючейКриптографии.ВыгрузитьАсинх

КонтейнерКлючейКриптографии (CryptoKeysContainer)

ВыгрузитьАсинх (UnloadAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

ВыгрузитьАсинх()

Описание варианта метода:

Передает выгруженные данные в параметре типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Данные в файл

Синтаксис:

ВыгрузитьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выгружен контейнер ключей криптографии.

Описание варианта метода:

Выгружает контейнер ключей криптографии в файл.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или имя файла, в который выгружен сертификат.

Описание:

Асинхронная функция. Возвращает данные, использованные при создании или инициализации объекта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md), метод [Выгрузить](КонтейнерКлючейКриптографии.Выгрузить_(CryptoKeysContainer.Unload).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
