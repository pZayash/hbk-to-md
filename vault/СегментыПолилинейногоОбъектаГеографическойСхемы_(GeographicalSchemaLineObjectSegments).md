# СегментыПолилинейногоОбъектаГеографическойСхемы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СегментПолилинейногоОбъектаГеографическойСхемы](СегментПолилинейногоОбъектаГеографическойСхемы_(GeographicalSchemaPolylineObjectSegment).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Добавить (Add)](СегментыПолилинейногоОбъектаГеографическойСхемы.Добавить_(GeographicalSchemaLineObjectSegments.Add).md)  
[Количество (Count)](СегментыПолилинейногоОбъектаГеографическойСхемы.Количество_(GeographicalSchemaLineObjectSegments.Count).md)  
[Очистить (Clear)](СегментыПолилинейногоОбъектаГеографическойСхемы.Очистить_(GeographicalSchemaLineObjectSegments.Clear).md)  
[Получить (Get)](СегментыПолилинейногоОбъектаГеографическойСхемы.Получить_(GeographicalSchemaLineObjectSegments.Get).md)  
[Удалить (Delete)](СегментыПолилинейногоОбъектаГеографическойСхемы.Удалить_(GeographicalSchemaLineObjectSegments.Delete).md)  

Описание:

Предоставляет доступ к коллекции сегментов полилинейного объекта, из которых состоит линия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), свойство [Сегменты](ПолилинейныйОбъектГеографическойСхемы.Сегменты_(GeographicalSchemaObjectPolyline.Segments).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
