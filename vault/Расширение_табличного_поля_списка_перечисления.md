Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка перечисления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ПеречислениеСсылка.](ПеречислениеСсылка.Имя_перечисления.md)<[Имя перечисления](ПеречислениеСсылка.Имя_перечисления.md)>.
