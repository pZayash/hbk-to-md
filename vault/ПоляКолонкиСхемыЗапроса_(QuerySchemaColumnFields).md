**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md), [ВложеннаяТаблицаСхемыЗапроса](ВложеннаяТаблицаСхемыЗапроса_(QuerySchemaNestedTable).md), [Неопределено](lang__def_Undefined.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля колонки.  
Возможно обращение к полю колонки посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПоляКолонкиСхемыЗапроса.Индекс_(QuerySchemaColumnFields.IndexOf).md)

[Установить (Set)](ПоляКолонкиСхемыЗапроса.Установить_(QuerySchemaColumnFields.Set).md)  

Описание:

Коллеция полей, которые входят в колонку.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
