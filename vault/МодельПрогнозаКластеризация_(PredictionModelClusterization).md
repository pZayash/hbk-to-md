Описание:

Модель прогноза для кластерного анализа.  
Модель содержит следующие колонки:

- Кластер (Cluster) - номер ближайшего кластера для анализируемого объекта;
- <Имя прогнозируемой колонки> - решение анализа данных - наиболее вероятное решение. Содержит вложенные колонки:
  - Прогноз (Prediction) - объект анализа данных. Прогноз для колонки. Содержит вложенные колонки:
    - Значение (Value) - наиболее вероятное значение;
    - ЗначениеС (ValueFrom) - наиболее вероятное начальное значение интервала;
    - ЗначениеПо (ValueTo) - наиболее вероятное начальное значение интервала.
  - Вероятность (Probability) - вероятность правильного решения.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.
