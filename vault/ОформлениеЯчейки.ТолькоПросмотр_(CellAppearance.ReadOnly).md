ОформлениеЯчейки (CellAppearance)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность редактирования ячейки.

Доступность:

Толстый клиент.

Примечание:

Если свойство ТолькоПросмотр равно [Истина](lang__def_BooleanTrue.md), становится невозможным редактирование ячейки табличного поля, однако на остальную работу с ячейкой это не влияет. Например, можно выделить строковое представление содержимого ячейки и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
