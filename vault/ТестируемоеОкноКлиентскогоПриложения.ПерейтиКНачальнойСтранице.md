ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

ПерейтиКНачальнойСтранице (GotoStartPage)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПерейтиКНачальнойСтранице()

Описание:

Для объекта [ТестируемоеОкноКлиентскогоПриложения](ТестируемоеОкноКлиентскогоПриложения.md), у которого свойство [Основное](ТестируемоеОкноКлиентскогоПриложения.Основное.md) установлено в значение Истина, выполняет переход на начальную страницу.  
Для остальных окон генерирует исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
