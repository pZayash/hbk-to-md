# НастройкиСправки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Настройки](Настройки.md)

Доступен, начиная с версии 8.2.

Описание:

Содержит настройки системы справочной информации:

- список строк поиска в режиме "Индекс";
- список строк поиска в режиме "Поиск".

Доступность:

Сервер, толстый клиент.   
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.2.
