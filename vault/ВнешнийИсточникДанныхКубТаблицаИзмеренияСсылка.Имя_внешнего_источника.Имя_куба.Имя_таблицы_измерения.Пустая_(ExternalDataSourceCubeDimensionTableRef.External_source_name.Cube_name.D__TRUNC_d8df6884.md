ВнешнийИсточникДанныхКубТаблицаИзмеренияСсылка.<Имя внешнего источника>.<Имя куба>.<Имя таблицы измерения> (ExternalDataSourceCubeDimensionTableRef.<External source name>.<Cube name>.<Dimension table name>)

Пустая (IsEmpty)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Пустая()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ссылка не указывает ни на какой объект (пустая ссылка); [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, является ли ссылка пустой или нет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
