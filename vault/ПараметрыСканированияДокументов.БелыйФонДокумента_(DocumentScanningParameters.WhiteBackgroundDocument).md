ПараметрыСканированияДокументов (DocumentScanningParameters)

БелыйФонДокумента (WhiteBackgroundDocument)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, учитывается ли цвет документа при поиске его границ:

- [Истина](lang__def_BooleanTrue.md) - при поиске границ документа учитывается белый цвет фона документа. Изображение будет очищено от теней, сломов, что сделает его более ровным.
- [Ложь](lang__def_BooleanFalse.md) - граница документа определяется без учета цвета документа.

Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
