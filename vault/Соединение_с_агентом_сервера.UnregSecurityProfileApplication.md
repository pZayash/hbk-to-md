Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileApplication (UnregSecurityProfileApplication)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileApplication(<Кластер>, <ИмяПрофиля>, <ИмяПриложения>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должна быть отменена регистрация разрешенного приложения.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<ИмяПриложения> (обязательный)

Тип: Строка.   
Имя приложения, регистрация которого отменяется.

Описание:

Отменяет регистрацию разрешенного приложения.

Доступность:

Интеграция.

Примечание:

Если профиля, указанного в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если разрешенного приложения с именем, указанным в параметре <ИмяПриложения>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
