Глобальный контекст (Global context)

ТекущаяУниверсальнаяДатаВМиллисекундах (CurrentUniversalDateInMilliseconds)

Доступен, начиная с версии 8.2.

Синтаксис:

ТекущаяУниверсальнаяДатаВМиллисекундах()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает текущую универсальную дату в миллисекундах (в UTC, начиная с 01.01.0001 00:00:00).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
