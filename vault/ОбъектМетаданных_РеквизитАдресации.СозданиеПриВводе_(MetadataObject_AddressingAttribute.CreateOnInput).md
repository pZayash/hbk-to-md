ОбъектМетаданных: РеквизитАдресации (MetadataObject: AddressingAttribute)

СозданиеПриВводе (CreateOnInput)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [СозданиеПриВводе](СозданиеПриВводе_(CreateOnInput).md).   
Режим создания при вводе. Если значение = [Авто](СозданиеПриВводе.Авто_(CreateOnInput.Auto).md), то участие в механизме создания при вводе определяется значением свойства СозданиеПриВводе объекта метаданных этого типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Расширение поля формы для поля ввода](Расширение_поля_формы_для_поля_ввода_(Form_field_extension_for_a_text_box).md), свойство [КнопкаСоздания](Расширение_поля_формы_для_поля_ввода.КнопкаСоздания_(Form_field_extension_for_a_text_box.CreateButton).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.
