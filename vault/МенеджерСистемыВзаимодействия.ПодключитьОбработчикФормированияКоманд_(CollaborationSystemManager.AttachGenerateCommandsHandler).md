МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПодключитьОбработчикФормированияКоманд (AttachGenerateCommandsHandler)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПодключитьОбработчикФормированияКоманд(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после подключения обработчика со следующими параметрами:

- <ПараметрыКоманд> - тип [ПараметрыФормированияКомандСистемыВзаимодействия](ПараметрыФормированияКомандСистемыВзаимодействия_(CollaborationSystemGenerateCommandParameters).md). Описывает параметры вызова обработчика.
- <Команды> - тип [Массив](Массив_(Array).md). Значения элементов массива имеют тип [ОписаниеКомандыСистемыВзаимодействия](ОписаниеКомандыСистемыВзаимодействия_(CollaborationSystemCommandDescription).md). Значения другого типа игнорируются. Задает список команд для меню. При вызове список заполнен командами меню по умолчанию.
- <КомандаПоУмолчанию> - тип [ОписаниеКомандыСистемыВзаимодействия](ОписаниеКомандыСистемыВзаимодействия_(CollaborationSystemCommandDescription).md), [Неопределено](lang__def_Undefined.md). Определяет команду по умолчанию по значению из списка команд. При вызове содержит команду по умолчанию. Эта команда выполняется сразу при нажатии левой кнопки мыши на ссылку, вложение или действие. Если [Неопределено](lang__def_Undefined.md), то при нажатии левой кнопки мыши показывается контекстное меню.
- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Подключает обработчик, который будет вызван при нажатии левой или правой кнопки мыши на гиперссылке в тексте сообщения, на вложение, гиперссылке действия или нажатие правой кнопки мыши на сообщение.  
В обработчик передается список команд по умолчанию.  
В обработчике можно сформировать свой состав команд.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.23.
