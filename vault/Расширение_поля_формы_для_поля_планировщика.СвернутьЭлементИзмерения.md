Расширение поля формы для поля планировщика (Form field extension for a planner)

СвернутьЭлементИзмерения (CollapseDimensionItem)

Доступен, начиная с версии 8.3.17.

Синтаксис:

СвернутьЭлементИзмерения(<ЗначенияИзмерений>, <СПодчиненными>)

Параметры:

<ЗначенияИзмерений> (обязательный)

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие.md).   
Фиксированное соответствие, однозначно определяющее элемент измерения, который нужно свернуть.   
В качестве ключа используется [Значение](ИзмерениеПланировщика.Значение.md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md) элемента измерения.

<СПодчиненными> (необязательный)

Тип: Булево.   
Определяет необходимость разворачивания элемента измерения вместе с подчиненными элементами измерения. Истина – развернуть вместе с подчиненными.  
Значение по умолчанию: Ложь.

Описание:

Сворачивает элемент измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
