ТочкаДиаграммыГанта (GanttChartPoint)

Цвет (Color)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Если используются цвета точки (см. свойство [ПриоритетЦвета](ТочкаДиаграммыГанта.ПриоритетЦвета_(GanttChartPoint.ColorPriority).md)), используется для вывода интервалов.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md), свойство [ПриоритетЦвета](ТочкаДиаграммыГанта.ПриоритетЦвета_(GanttChartPoint.ColorPriority).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
