КоллекцияСтраницPDF (PDFPagesCollection)

Получить (Get)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции, который требуется получить.

Возвращаемое значение:

Тип: [СтраницаPDF](СтраницаPDF_(PDFPage).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
