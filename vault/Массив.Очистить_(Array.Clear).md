# Массив.Очистить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Массив](Массив_(Array).md) › [Методы](Массив__Методы.md)

Массив (Array)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все значения из массива.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| МассивСтрок.Очистить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
