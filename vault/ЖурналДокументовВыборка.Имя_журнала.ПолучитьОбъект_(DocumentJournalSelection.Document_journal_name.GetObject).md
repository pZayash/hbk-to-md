ЖурналДокументовВыборка.<Имя журнала> (DocumentJournalSelection.<Document journal name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ДокументОбъект.](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)<[Имя документа](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для чтения, изменения, добавления и удаления документа, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
