ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа.md).   

Описание:

Получает пустое значение ссылки на документ данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Пример:

|  |
| --- |
| Запрос.УстановитьПараметр("ПустойЗаказ", Документы.ВнутреннийЗаказ.ПустаяСсылка()); |

Использование в версии:

Доступен, начиная с версии 8.0.
