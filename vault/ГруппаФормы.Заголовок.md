ГруппаФормы (FormGroup)

Заголовок (Title)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Заголовок к таблице. Пустая строка означает автоматическое формирование. Для того чтобы не располагать в форме заголовок, следует установить свойство [ПоложениеЗаголовка](ТаблицаФормы.ПоложениеЗаголовка.md) в [Нет](ПоложениеЗаголовкаЭлементаФормы.Нет.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
