# МенеджерWebSocketКлиентов.НайтиПредопределенный

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [WebSocket](WebSocket.md) › [МенеджерWebSocketКлиентов](МенеджерWebSocketКлиентов_(WebSocketClientsManager).md) › [Методы](МенеджерWebSocketКлиентов__Методы.md)

МенеджерWebSocketКлиентов (WebSocketClientsManager)

НайтиПредопределенный (FindPredefined)

Доступен, начиная с версии 8.3.27.

Синтаксис:

НайтиПредопределенный(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент_(MetadataObject_WebSocketClient).md).   
Объект метаданных WebSocket-клиента.

Возвращаемое значение:

Тип: [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md), связанный с предопределенным WebSocket-клиентом или [Неопределено](lang__def_Undefined.md), если такого предопределенного нет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
