Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md), [ВложеннаяТаблицаСхемыЗапроса](ВложеннаяТаблицаСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражение схемы запроса или вложенная таблица схемы запроса.  
Возможно обращение к выражению схемы запроса или вложенной таблице схемы запроса посредством оператора [...]. В качестве аргумента передается индекс.
