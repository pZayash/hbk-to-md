МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьОповещенияТекущегоПользователя (GetCurrentUserNotifications)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьОповещенияТекущегоПользователя()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ОповещениеСистемыВзаимодействия](ОповещениеСистемыВзаимодействия_(CollaborationSystemNotification).md).

Описание:

Возвращает список оповещений.  
Метод возвращает оповещения для всех обсуждений, независимо от значения свойства [Отображаемое](ОбсуждениеСистемыВзаимодействия.Отображаемое_(CollaborationSystemConversation.Displayed).md) обсуждения.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
