ТипШтрихКода (BarcodeType)

UPCA (UPCA)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Universal Product Code. Является частным случаем, подмножеством кода EAN-13.

Использование в версии:

Доступен, начиная с версии 8.3.25.
