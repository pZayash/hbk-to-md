Расширение формы клиентского приложения для справочника (Client application form extension for catalogs)

ЭтоГруппа (IsFolder)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.   
Определяет вид нового объекта - группа или элемент.

Примечание:

Используется для форм иерархических объектов (справочников, планов видов характеристик).

Использование в версии:

Доступен, начиная с версии 8.2.
