Картинка (Picture)

ПолучитьДвоичныеДанные (GetBinaryData)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДвоичныеДанные(<ВсеВарианты>, <ПодобратьВариант>)

Параметры:

<ВсеВарианты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки. Если значение [Ложь](lang__def_BooleanFalse.md) - будет выгружен один вариант. Если значение [Истина](lang__def_BooleanTrue.md) - все варианты в виде архива.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ПодобратьВариант> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md), [Неопределено](lang__def_Undefined.md).   
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки, и <ВсеВарианты> установлен в значение [Ложь](lang__def_BooleanFalse.md).  
Пары "Ключ-Значение" соответствуют xml-тегам в манифесте вариантов картинки. Если параметры выбора варианта не заданы, будет выбран вариант по умолчанию. Если параметры указывают на несуществующий в архиве вариант, то будет возвращено [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Преобразует картинку в тип [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Картинка](Картинка_(Picture).md), конструктор [Формирование из двоичных данных](Картинка.Формирование_из_двоичных_данных.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
