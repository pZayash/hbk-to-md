АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

СтрокаПараметровВнешнегоУправленияСеансами (ExternalSessionManagementConnectionString)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Свойство предназначено для внешнего управления сеансами.  
В качестве значения используется строка параметров веб сервиса внешнего управления сеансами в формате:  
 <Имя параметра>=<Значение> через символ ';' (точка с запятой).  
Если <Значение> содержит пробелы или символы ';', то строку значения параметра следует заключать в кавычки ("). При этом каждый из символов кавычки ("), присутствующий в строке <Значение>, должен быть экранирован ("").  
Параметры:

- - wsdl – URL-адрес для получения wsdl веб-сервиса;
- - ns – пространство имен веб-сервиса;
- - srvc – имя веб-сервиса;
- - port – имя порта веб-сервиса.

Например:  
wsdl=http://server/034983/ws/esc.1cws?wsdl;ns=http://www.sessioncontrol.org;srvc=ExternalSessionControl;port=ExternalSessionControlSoap.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
