ТестируемаяТаблицаФормы (TestedFormTable)

Развернут (Expanded)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Развернут(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие.md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.  
Если не указан, проверка выполняется для текущей строки.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Булево.   
Истина - узел для текущей строки развернут.

Описание:

Проверяет развернутость узла для текущей строки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
