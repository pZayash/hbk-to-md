МенеджерПроверкиВстроенныхПокупок (InAppPurchasesValidationManager)

ПроверитьКвитанциюВстроеннойПокупки (ValidateInAppPurchaseReceipt)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПроверитьКвитанциюВстроеннойПокупки(<Квитанция>, <КлючПроверки>, <ПоляКвитанции>)

Параметры:

<Квитанция> (обязательный)

Тип: [КвитанцияВстроеннойПокупки](КвитанцияВстроеннойПокупки_(InAppPurchaseReceipt).md).   
Проверяемая квитанция.

<КлючПроверки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ для проверки квитанции.  
Публичный ключ приложения Google Play для проверки квитанции на Android. Либо "secret key" для проверки квитанций на iOS.

<ПоляКвитанции> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Поля квитанции.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращаемое значение показывает подлинность квитанции в отношении платформы.  
Дополнительно метод возвращает распакованные значения полей квитанции, что облегчает ее дальнейшую проверку.

Описание:

Выполняет проверку квитанции:

- на Android - с помощью переданного ключа проверки проверяет электронную подпись квитанции,
- на iOS - отправляет запрос на сервис проверки квитанций. Запрос отправляется по адресу https://buy.itunes.apple.com/verifyReceipt (на этапе тестирования - https://sandbox.itunes.apple.com/verifyReceipt).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
