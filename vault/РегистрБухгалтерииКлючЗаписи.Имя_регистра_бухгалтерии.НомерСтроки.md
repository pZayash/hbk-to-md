Доступен, начиная с версии 8.0.

Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрБухгалтерииКлючЗаписи.Имя_регистра_бухгалтерии.Регистратор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
