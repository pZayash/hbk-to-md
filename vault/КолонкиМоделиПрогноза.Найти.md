КолонкиМоделиПрогноза (PredictionModelColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>, <Тип>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомой колонки.

<Тип> (обязательный)

Тип: [ТипКолонкиМоделиПрогноза](ТипКолонкиМоделиПрогноза.md).   
Тип искомой колонки.

Возвращаемое значение:

Тип: [КолонкаМоделиПрогноза](КолонкаМоделиПрогноза.md), Неопределено.   
Если колонка не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск колонки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
