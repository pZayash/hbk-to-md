ПоискВТаблицеПриВводе (SearchInTableOnInput)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

При вводе символов в таблицу, связанную с данными типа [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), выполняется поиск в таблице.  
При вводе символов в таблицу, связанную с [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md), поиск выполняется только в режиме совместимости конфигурации [Версия8\_2\_13](РежимСовместимости.Версия8_2_13_(CompatibilityMode.Version8_2_13).md).

Использование в версии:

Доступен, начиная с версии 8.2.
