БиблиотекаСтилей (StyleLib)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<СсылкаНаСтиль>)

Параметры:

<СсылкаНаСтиль> (обязательный)

Тип: [Стиль](Стиль.md).   
Ссылка на стиль коллекции, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: Число.   
Индекс указанного стиля в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс указанного стиля в коллекции стилей.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| ИндексСтиля = БиблиотекаСтилей.Индекс(ТекущийСтиль); |

Использование в версии:

Доступен, начиная с версии 8.0.
