Глобальный контекст (Global context)

АктивноеОкно (ActiveWindow)

Доступен, начиная с версии 8.2.

Синтаксис:

АктивноеОкно()

Возвращаемое значение:

Тип: [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md), Неопределено.   
В режиме совместимости [Версия8\_3\_19](РежимСовместимости.Версия8_3_19.md) и ниже, если активна начальная страница, метод возвращает:

- в тонком клиенте - Неопределено,
- в веб-клиенте - объект типа [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md), у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница.md) имеет значение Ложь.

В режиме совместимости выше [Версия8\_3\_19](РежимСовместимости.Версия8_3_19.md) в тонком и веб-клиенте, в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси.md) возвращает объект типа [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md), у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница.md) имеет значение Истина. И свойство [Содержимое](ОкноКлиентскогоПриложения.Содержимое.md) содержит формы начальной страницы.  
Если активна начальная страница то и данный метод, и свойство [Окно](ФормаКлиентскогоПриложения.Окно.md) формы начальной страницы возвращают один и тот же объект.

Описание:

Получает текущее активное окно.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска клиентского приложения "Обычное приложение" возвращает Неопределено.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
