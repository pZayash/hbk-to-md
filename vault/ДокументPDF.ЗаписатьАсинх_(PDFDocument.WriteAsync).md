ДокументPDF (PDFDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <Пароль>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для записи PDF-документа.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: [ТипФайлаДокументаPDF](ТипФайлаДокументаPDF_(PDFDocumentFileType).md), [Неопределено](lang__def_Undefined.md).   
Тип файла документа PDF. Если содержит [Неопределено](lang__def_Undefined.md), то в качестве значения параметра используется свойство [ТипФайла](ДокументPDF.ТипФайла_(PDFDocument.FileType).md) объекта [ДокументPDF](ДокументPDF_(PDFDocument).md).

Описание варианта метода:

Асинхронная функция. Записывает документ PDF.  
Если документ имеет электронную цифровую подпись и выполняется запись с паролем, то генерируется исключение.  
  
Доступность:  
Тонкий клиент, толстый клиент, веб-клиент.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <Пароль>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи PDF-документа.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: [ТипФайлаДокументаPDF](ТипФайлаДокументаPDF_(PDFDocumentFileType).md), [Неопределено](lang__def_Undefined.md).   
Тип файла документа PDF. Если содержит [Неопределено](lang__def_Undefined.md), то в качестве значения параметра используется свойство [ТипФайла](ДокументPDF.ТипФайла_(PDFDocument.FileType).md) объекта [ДокументPDF](ДокументPDF_(PDFDocument).md).

Описание варианта метода:

Асинхронная функция. Записывает изменения в документ PDF.  
Если документ имеет электронную цифровую подпись и выполняется запись с паролем, то генерируется исключение.  
  
Доступность:  
Тонкий клиент, толстый клиент, веб-клиент.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   

Описание:

Асинхронная функция. Записывает изменения в PDF-документ.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Для работы метода в веб-клиенте требуется наличие расширения работы с файлами.

См. также:

[ДокументPDF](ДокументPDF_(PDFDocument).md), метод [Записать](ДокументPDF.Записать_(PDFDocument.Write).md)  

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.24.
