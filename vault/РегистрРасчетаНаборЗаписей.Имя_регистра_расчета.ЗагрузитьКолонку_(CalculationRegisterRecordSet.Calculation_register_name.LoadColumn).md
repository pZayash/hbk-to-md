РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

ЗагрузитьКолонку (LoadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗагрузитьКолонку(<Массив>, <ИмяПоля>)

Параметры:

<Массив> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий значения для заполнения колонки набора.

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор поля записи, в которую будут загружены значения из массива.

Описание:

Загружает значения из массива в заданное поле набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения из массива загружаются в поле в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.0.
