ПолеВыбора (ComboBox)

КнопкаОчистки (ClearButton)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак наличия/отсутствия кнопки очистки поля.  
Истина - кнопка есть.

Доступность:

Толстый клиент.

Примечание:

Стандартной обработки нажатия на кнопку очистки не предусмотрено. Необходимые действия можно провести в обработчике события Очистка.

Использование в версии:

Доступен, начиная с версии 8.0.
