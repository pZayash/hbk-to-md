СравнениеФайлов (FileCompare)

ИгнорироватьПустоеПространство (IgnoreWhiteSpace)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство установлено в Истина, то при сравнения текстов игнорируется пустое пространство.

Доступность:

Толстый клиент.

Примечание:

Значение по умолчанию Истина.  
Для нетекстовых режимов сравнения свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.0.
