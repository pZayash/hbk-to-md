ОтображениеСтраницФормы (FormPagesRepresentation)

Авто (Auto)

Доступен, начиная с версии 8.3.20.

Описание:

Для формы настроек отчета, формы редактирования варианта отчета:

- В тонком и веб-клиентах используется [ЗакладкиСверху](ОтображениеСтраницФормы.ЗакладкиСверху.md),
- На мобильных клиентах используется значение [ЗакладкиСлеваГоризонтально](ОтображениеСтраницФормы.ЗакладкиСлеваГоризонтально.md).

Для остальных форм используется значение [ЗакладкиСверху](ОтображениеСтраницФормы.ЗакладкиСверху.md).

Использование в версии:

Доступен, начиная с версии 8.3.20.
