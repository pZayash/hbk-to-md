ТестируемаяФорма (TestedForm)

ТекущаяДоступность (CurrentEnable)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ТекущаяДоступность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект доступен; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Возвращает признак того, что пользователь может взаимодействовать с элементами формы.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
