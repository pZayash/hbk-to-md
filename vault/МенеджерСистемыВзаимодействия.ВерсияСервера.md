МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВерсияСервера (ServerVersion)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВерсияСервера()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает версию сервера системы взаимодействия, на котором зарегистрирована информационная база.  
Если приложение не зарегистрировано на сервере системы взаимодействия, то возвращает пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
