ПоследовательностьXDTO (XDTOSequence)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Вставка текста

Синтаксис:

Вставить(<Индекс>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс вставки текста в последовательность.

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст, вставляемый в последовательность.

Вариант синтаксиса: Вставка значения

Синтаксис:

Вставить(<Индекс>, <Свойство>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс вставки элемента в последовательность.

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md).   
Свойство объекта, для которого вставляется элемент.

<Элемент> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [Неопределено](lang__def_Undefined.md).   
Элемент данных XDTO.

Описание:

Вставляет элемент в последовательность по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
