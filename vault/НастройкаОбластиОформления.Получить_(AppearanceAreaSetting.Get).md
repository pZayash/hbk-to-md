НастройкаОбластиОформления (AppearanceAreaSetting)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента управления областью оформления.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойНастройкиОформления](ЭлементУправленияНастройкойНастройкиОформления_(AppearanceSettingsSettingControl).md).   

Описание:

Получает элемент управления областью оформления по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
