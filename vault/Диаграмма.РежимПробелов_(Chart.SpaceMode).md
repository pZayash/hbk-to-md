Диаграмма (Chart)

РежимПробелов (SpaceMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РежимПробеловДиаграммы](РежимПробеловДиаграммы_(ChartSpaceMode).md).   
Определяет режим вывода пробелов между столбцами (областями).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для диаграмм, имеющих столбцы или области.

Использование в версии:

Доступен, начиная с версии 8.0.
