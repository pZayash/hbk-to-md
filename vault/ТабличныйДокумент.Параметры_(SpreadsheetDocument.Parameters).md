ТабличныйДокумент (SpreadsheetDocument)

Параметры (Parameters)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыМакетаТабличногоДокумента](ПараметрыМакетаТабличногоДокумента_(SpreadsheetDocumentTemplateParameters).md).   
Содержит коллекцию параметров макета табличного документа.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
