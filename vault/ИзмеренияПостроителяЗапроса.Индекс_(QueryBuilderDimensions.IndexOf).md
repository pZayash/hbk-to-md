ИзмеренияПостроителяЗапроса (QueryBuilderDimensions)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Измерение>)

Параметры:

<Измерение> (обязательный)

Тип: [ИзмерениеПостроителяЗапроса](ИзмерениеПостроителяЗапроса_(QueryBuilderDimension).md).   
Искомое измерение.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
