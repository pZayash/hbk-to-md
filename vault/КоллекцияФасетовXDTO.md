Элементы коллекции:

[ФасетXDTO](ФасетXDTO.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются фасеты XDTO.  
Возможно обращение к фасету XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
