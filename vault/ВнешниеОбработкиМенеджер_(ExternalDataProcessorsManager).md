Описание:

Предназначен для управления внешними обработками и позволяет получать значения типа [ВнешняяОбработка](ВнешняяОбработка_(ExternalDataProcessor).md) при помощи метода [Создать](ВнешниеОбработкиМенеджер.Создать_(ExternalDataProcessorsManager.Create).md). Доступ к объекту осуществляется через свойство глобального контекста [ВнешниеОбработки](Глобальный_контекст.ВнешниеОбработки_(Global_context.ExternalDataProcessors).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ВнешниеОбработки](Глобальный_контекст.ВнешниеОбработки_(Global_context.ExternalDataProcessors).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.22.
