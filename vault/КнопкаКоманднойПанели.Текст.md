КнопкаКоманднойПанели (CommandBarButton)

Текст (Text)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст, отображаемый на кнопке, при соответствующем стиле отображения, определяемом свойством [Отображение](КнопкаКоманднойПанели.Отображение.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
