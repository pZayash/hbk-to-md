# ЗадачаОбъект.<Имя задачи>.Ссылка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Свойства](ЗадачаОбъект.Имя_задачи__Свойства.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Содержит ссылку на задачу. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Например, данное свойство можно использовать, если ссылку на созданную и записанную задачу нужно присвоить реквизиту другой задачи.

Использование в версии:

Доступен, начиная с версии 8.0.
