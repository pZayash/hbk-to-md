# СтрокиДинамическогоСписка

Доступен, начиная с версии 8.3.10.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа [КлючИЗначение](КлючИЗначение.md), где Ключ - ключ строки, Значение - [СтрокаДинамическогоСписка](СтрокаДинамическогоСписка.md).  
Возможно обращение к значению элемента коллекции посредством оператора [...]. В качестве аргумента передается ключ строки.  

Методы:

[Количество (Count)](СтрокиДинамическогоСписка.Количество.md)  
[Получить (Get)](СтрокиДинамическогоСписка.Получить.md)  
[ПолучитьКлючи (GetKeys)](СтрокиДинамическогоСписка.ПолучитьКлючи.md)  

Описание:

Представляет собой коллекцию строк динамического списка.  
К содержимому объекта можно обращаться только из обработчика [ПриПолученииДанныхНаСервере](Расширение_таблицы_формы_для_динамического_списка.ПриПолученииДанныхНаСервере.md). По окончании работы обработчика попытка обращения к содержимому приведет к вызову исключения.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
