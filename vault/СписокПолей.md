Доступен, начиная с версии 8.0.

Элементы коллекции:

[Поле](Поле.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Индекс (IndexOf)](СписокПолей.Индекс.md)  
[Количество (Count)](СписокПолей.Количество.md)  
[Найти (Find)](СписокПолей.Найти.md)  
[Получить (Get)](СписокПолей.Получить.md)  

Описание:

Представляет собой коллекцию полей ввода по строке.  
Используется для свойств ВводПоСтроке объектов метаданных и свойства ПоляКлюча объектов метаданных - таблиц внешних источников данных.  
 Свойство ВводПоСтроке такого типа есть у всех объектов описания метаданного, которые определяют наличие в системе типа ...Ссылка, таких как [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>, [СправочникСсылка.](СправочникСсылка.Имя_справочника.md)<[Имя справочника](СправочникСсылка.Имя_справочника.md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>, [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>.  
В качестве полей ввода по строке могут выступать такие предопределенные реквизиты, как "Код", "Наименование", "Дата", "Номер" и реквизиты объектов, имеющие примитивный тип данных, для которых свойство объекта метаданных "Индексировать" принимает значение "Индексирование" или "Индексировать с доп. упорядочиванием".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
