Описание:

Объект этого типа возвращается методами [Выбрать](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.Выбрать.md) и [ВыбратьИерархически](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.ВыбратьИерархически.md) у объекта типа [ПланВидовХарактеристикМенеджер.](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.md)> и представляет собой специализированный способ перебора элементов плана видов характеристик. Обход элементов выполняется системой динамически. Это означает, что использование выборки не считывает все элементы сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большие списки справочников и не загружает в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение.
