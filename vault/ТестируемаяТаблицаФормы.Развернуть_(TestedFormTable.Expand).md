ТестируемаяТаблицаФормы (TestedFormTable)

Развернуть (Expand)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Развернуть(<ОписаниеСтроки>, <СПодчиненными>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.

<СПодчиненными> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
При передаче значения [Истина](lang__def_BooleanTrue.md) метод разворачивает все подчинённые узлы переданной строки.  
При передаче значения [Ложь](lang__def_BooleanFalse.md) разворачивается только узел переданной строки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Выполняет попытку развернуть узел для текущей строки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
