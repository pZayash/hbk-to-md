Расширение формы клиентского приложения для объекта таблицы внешнего источника данных

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: [СправочникОбъект.\<Имя справочника\>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md).   
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
