РезультатРаспознаванияРечи (SpeechToTextResult)

РаспознаваниеФразыЗавершено (PhraseRecognitionCompleted)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что гипотеза является окончательной относительно фразы.  
Для файлового распознавания всегда имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
