Форма сохранения (Save form)

КлючТекущихНастроек (CurrentSettingsKey)

Доступен, начиная с версии 8.2.

Описание:

Идентификатор текущей настройки. Данный объект может быть использован для позиционирования в списке настроек на текущую настройку.

Использование в версии:

Доступен, начиная с версии 8.2.
