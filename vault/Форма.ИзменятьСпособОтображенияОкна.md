Форма (Form)

ИзменятьСпособОтображенияОкна (ChangeWindowAppearanceMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИзменениеСпособаОтображенияОкна](ИзменениеСпособаОтображенияОкна.md).   
Разрешает или запрещает изменение способа отображения окна формы.

Доступность:

Толстый клиент.

Примечание:

Управление свойством должно производиться до открытия формы.

Использование в версии:

Доступен, начиная с версии 8.0.
