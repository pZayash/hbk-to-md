ФабрикаXDTO (XDTOFactory)

ПрочитатьXML (ReadXML)

Доступен, начиная с версии 8.1.

Синтаксис:

ПрочитатьXML(<ЧтениеXML>, <Тип>, <НазначениеТипа>)

Параметры:

<ЧтениеXML> (обязательный)

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM.md), [ЧтениеFastInfoset](ЧтениеFastInfoset.md), [ЧтениеXML](ЧтениеXML.md).   
Объект чтения XML.

<Тип> (необязательный)

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO.md), [ТипОбъектаXDTO](ТипОбъектаXDTO.md).   
Тип элемента данных XDTO. Если тип не указан, то будет сделана попытка определить тип данных по текущему состоянию объекта [ЧтениеXML](ЧтениеXML.md).

<НазначениеТипа> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML.md).   
Назначение типа данных произвольных свойства при чтении.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное.md).

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO.md).   

Описание:

Читает элемент данных XDTO указанного типа из объекта чтения XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.25.
