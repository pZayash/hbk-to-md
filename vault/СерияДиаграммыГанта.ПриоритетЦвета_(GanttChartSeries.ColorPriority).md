СерияДиаграммыГанта (GanttChartSeries)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Вместе со свойством [ПриоритетЦвета](ТочкаДиаграммыГанта.ПриоритетЦвета_(GanttChartPoint.ColorPriority).md) соответствующей точки определяет цвет значения. При равных значениях свойства используется цвет серии.  
[Истина](lang__def_BooleanTrue.md) - будет использоваться значение [Цвет](СерияДиаграммыГанта.Цвет_(GanttChartSeries.Color).md) серии.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
