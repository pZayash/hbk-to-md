ПолеВвода (TextBox)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, которая располагается непосредственно в поле ввода. Картинка всегда прижимается влево.

Доступность:

Толстый клиент.

Примечание:

Максимальная ширина картинки, которую можно отобразить в поле ввода 20 пикселей. Оптимальная высота картинки зависит от высоты элемента управления и толщины рамки.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
