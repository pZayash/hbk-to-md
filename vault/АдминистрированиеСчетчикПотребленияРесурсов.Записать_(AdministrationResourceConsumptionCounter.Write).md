АдминистрированиеСчетчикПотребленияРесурсов (AdministrationResourceConsumptionCounter)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового или обновление существующего счетчика потребления ресурсов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
