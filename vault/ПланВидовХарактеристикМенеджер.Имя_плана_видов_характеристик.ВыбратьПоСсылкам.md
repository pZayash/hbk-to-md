ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик> (ChartOfCharacteristicTypesManager.<Chart of characteristic types name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на элементы плана видов характеристик.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [ПланВидовХарактеристикВыборка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикВыборка.Имя_плана_видов_характеристик.md).   

Описание:

Формирует выборку элементов плана видов характеристик по заданному источнику ссылок на элементы плана видов характеристик соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы плана видов характеристик соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    ВидыХарактеристик.Ссылка КАК Ссылка     |ИЗ     |    ПланВидовХарактеристик.ВидыХарактеристик КАК ВидыХарактеристик     |"; РезультатЗапроса = Запрос.Выполнить(); Выборка = ПланыВидовХарактеристик.ВидыХарактеристик     .ВыбратьПоСсылкам(РезультатЗапроса); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
