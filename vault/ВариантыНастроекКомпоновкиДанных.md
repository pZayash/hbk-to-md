Доступен, начиная с версии 8.2.

Элементы коллекции:

[ВариантНастроекКомпоновкиДанных](ВариантНастроекКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты настроек.  
Возможно обращение к варианту настроек посредством оператора [...]. В качестве аргумента передается индекс.

[Удалить (Delete)](ВариантыНастроекКомпоновкиДанных.Удалить.md)  

Описание:

Коллекция вариантов настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
