# Расширение поля формы для поля табличного документа.МаксимальнаяВысота

Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

МаксимальнаяВысота (MaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную высоту элемента.  
Используется, если свойство [АвтоМаксимальнаяВысота](Расширение_поля_формы_для_поля_табличного_документа.АвтоМаксимальнаяВысота_(Form_field_extension_for_a_spreadsheet_document_field.AutoMaxHeight).md) установлено в значение [Ложь](lang__def_BooleanFalse.md).  
Задается в символах. Значение 0 означает, что максимальная высота не ограничена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
