# Расширение табличного поля списка узлов.ИзменятьСпособРедактирования

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [Расширение табличного поля списка узлов](Расширение_табличного_поля_списка_узлов_(Node_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_узлов__Свойства.md)

Расширение табличного поля списка узлов (Node list table box extension)

ИзменятьСпособРедактирования (ChangeEditingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает доступность интерактивного способа редактирования объекта (в строке или в отдельной форме).

Доступность:

Толстый клиент.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - интерактивная установка способа редактирования объекта не доступна пользователю.

Использование в версии:

Доступен, начиная с версии 8.0.
