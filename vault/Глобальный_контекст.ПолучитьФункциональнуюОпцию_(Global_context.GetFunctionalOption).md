Глобальный контекст (Global context)

ПолучитьФункциональнуюОпцию (GetFunctionalOption)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьФункциональнуюОпцию(<Имя>, <Параметры>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя функциональной опции в соответствии с установленными языковыми настройками.

<Параметры> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Параметры функциональных опций, необходимые для получения значения функциональной опции.  
Имеет смысл только для функциональных опций, источниками значения которых является реквизит справочника или ресурс регистра сведений. Для опций констант этот параметр игнорируется.  
Ключ элемента структуры задает имя параметра, которое должно соответствовать имени одному из объектов метаданных типа [ОбъектМетаданных: ПараметрФункциональныхОпций](ОбъектМетаданных_ПараметрФункциональныхОпций_(MetadataObject_FunctionalOptionsParameter).md). Значение элемента структуры должно содержать значение параметра.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Получает значения функциональной опции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметры переданы не полностью, то для функциональной опции с параметрами типа [Булево](lang__Булево_(Boolean).md) учитываются отобранные для указанных параметров значения. Если хотя бы одно из них равно [Истина](lang__def_BooleanTrue.md), то и значение опции равно [Истина](lang__def_BooleanTrue.md). Для функциональной опции остальных типов генерируется исключительная ситуация.  
Если параметры переданы полностью, но подходящего значения не найдено, то значение функциональной опции будет [Ложь](lang__def_BooleanFalse.md).  
Если в качестве параметра функциональной опции была передана ссылка на группу, а реквизит, в котором хранится значение функциональной опции, не используется для группы, то при получении значения функциональной опции будет возращено Null.

Использование в версии:

Доступен, начиная с версии 8.2.
