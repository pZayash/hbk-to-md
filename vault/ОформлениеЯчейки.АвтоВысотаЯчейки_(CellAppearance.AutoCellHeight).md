ОформлениеЯчейки (CellAppearance)

АвтоВысотаЯчейки (AutoCellHeight)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость вычисления высоты ячейки. При этом свойство [ВысотаЯчейки](ОформлениеЯчейки.ВысотаЯчейки_(CellAppearance.CellHeight).md) указывает максимальную высоту ячейки.  
Свойство становится доступным только для чтения, если поле является колонкой предопределенной таблицы поля диаграммы Ганта (в случае записи возникает исключение).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
