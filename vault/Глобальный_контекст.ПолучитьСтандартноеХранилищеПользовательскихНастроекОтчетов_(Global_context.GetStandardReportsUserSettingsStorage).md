Глобальный контекст (Global context)

ПолучитьСтандартноеХранилищеПользовательскихНастроекОтчетов (GetStandardReportsUserSettingsStorage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСтандартноеХранилищеПользовательскихНастроекОтчетов()

Возвращаемое значение:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md).   

Описание:

Получает объект, с помощью которого осуществляется доступ к содержимому стандартного хранилища пользовательских настроек отчетов, независимо от того, заполнено ли свойство конфигурации [ХранилищеПользовательскихНастроекОтчетов](ОбъектМетаданныхКонфигурация.ХранилищеПользовательскихНастроекОтчетов_(ConfigurationMetadataObject.ReportsUserSettingsStorage).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
