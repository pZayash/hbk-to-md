МенеджерГлобальногоПоиска (GlobalSearchManager)

ПолучитьПодсказкуВвода (GetInputHint)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьПодсказкуВвода()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает подсказку поля ввода, установленную при вызове метода [УстановитьПодсказкуВвода](МенеджерГлобальногоПоиска.УстановитьПодсказкуВвода_(GlobalSearchManager.SetInputHint).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
