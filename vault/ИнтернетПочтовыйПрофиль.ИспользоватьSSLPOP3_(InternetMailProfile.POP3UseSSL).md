ИнтернетПочтовыйПрофиль (InternetMailProfile)

ИспользоватьSSLPOP3 (POP3UseSSL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Использовать SSL-соединение для протокола POP3.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если предполагается использование защищенного STARTTLS-соединения, то должно быть установлено в значение [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.1.
