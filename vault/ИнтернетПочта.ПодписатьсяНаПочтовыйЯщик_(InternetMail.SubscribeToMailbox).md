ИнтернетПочта (InternetMail)

ПодписатьсяНаПочтовыйЯщик (SubscribeToMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

ПодписатьсяНаПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя IMAP почтового ящика.

Описание:

Подписать почтовый ящик IMAP для дальнейшей работы с ним.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется для [POP3](ПротоколИнтернетПочты.POP3_(InternetMailProtocol.POP3).md) соединения.

Использование в версии:

Доступен, начиная с версии 8.2.
