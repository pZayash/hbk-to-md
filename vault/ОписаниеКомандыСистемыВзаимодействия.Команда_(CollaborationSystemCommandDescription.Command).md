ОписаниеКомандыСистемыВзаимодействия (CollaborationSystemCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Описывает выполняемую команду. Отображение команды зависит от используемого типа:

- [СтандартнаяКомандаСистемыВзаимодействия](СтандартнаяКомандаСистемыВзаимодействия_(CollaborationSystemStandardCommand).md) - отображается одна из стандартных команд системы взаимодействия;
- [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md) - свойство задает процедуру, которая будет вызвана при выполнении этой команды со следующими параметрами:
  - <ДополнительныеПараметры> - тип Произвольный. Параметры, заданные при создании описания оповещения.
- [Массив](Массив_(Array).md) - отображается подменю с командами;
- [Неопределено](lang__def_Undefined.md) - в меню отображается разделитель;
- Любой другой тип - команда не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
