Глобальный контекст (Global context)

ЗаписатьЗначениеJSON (WriteJSONValue)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗаписатьЗначениеJSON(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Структура](Структура_(Structure).md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), [Массив](Массив_(Array).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Значение, которое требуется сериализовать в JSON.   
Если тип параметра [Структура](Структура_(Structure).md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), [Массив](Массив_(Array).md), или [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md), то значения, содержащиеся в них, должны быть одного из следующих типов:

- [Структура](Структура_(Structure).md),
- [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md),
- [Массив](Массив_(Array).md),
- [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md),
- [Число](lang__def_Number.md),
- [Строка](lang__def_String.md),
- [Булево](lang__Булево_(Boolean).md),
- [Неопределено](lang__def_Undefined.md).

При этом на значения типов [Структура](Структура_(Structure).md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), [Массив](Массив_(Array).md), или [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md) распространяются такие же ограничения, что и для параметра <Значение>.  
Значение типа [Неопределено](lang__def_Undefined.md) преобразуется в NULL.  
  
Если <Значение> не может быть преобразовано в/из JSON из-за неподдерживаемых типов: будет сгенерировано исключение: "Значение содержит данные недопустимых типов."  
  
Если преобразование в строку невозможно из-за циклической зависимости объектов, генерируется исключение: "Значение содержит циклические зависимости".

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
JSON-строка

Описание:

Преобразует значение в JSON-строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
