ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет задачу из базы данных.  
Важно! В отличие от метода [УстановитьПометкуУдаления](ЗадачаОбъект.Имя_задачи.УстановитьПометкуУдаления.md), данный метод производит непосредственное удаление задачи без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Объект.Удалить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
