Расширение поля формы для поля флажка (Form field extension for a check box field)

ОдинаковаяШиринаЭлементов (EqualItemsWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет одинаковую ширину элементов.   
[Неопределено](lang__def_Undefined.md) трактуется как [Истина](lang__def_BooleanTrue.md).   
Если итоговое значение отлично от [Истина](lang__def_BooleanTrue.md), свойство [ШиринаЭлемента](Расширение_поля_формы_для_поля_флажка.ШиринаЭлемента_(Form_field_extension_for_a_check_box_field.ItemWidth).md) не используется.   
Данное свойство используется только если [ВидФлажка](ВидФлажка_(CheckBoxType).md) = [Тумблер](ВидФлажка.Тумблер_(CheckBoxType.Tumbler).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
