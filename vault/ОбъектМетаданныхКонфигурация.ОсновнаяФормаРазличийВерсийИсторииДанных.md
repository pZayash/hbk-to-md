ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновнаяФормаРазличийВерсийИсторииДанных (DefaultDataHistoryVersionDifferencesForm)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Описывает общую форму, которая будет вызвана системой для отображения различий версий истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
