ОтображениеЗаголовкаШкалыДиаграммы (ShowChartScaleTitle)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Заголовок отображается, если есть не пустой текст, который можно вывести (установленный пользователем или сгенерированный автоматически).

Использование в версии:

Доступен, начиная с версии 8.3.10.
