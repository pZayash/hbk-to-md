# ПланСчетовМенеджер.<Имя плана счетов>.ВыбратьПоСсылкам

ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [Массив](Массив_(Array).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Источник ссылок на элементы плана счетов.

<Колонка> (необязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Колонка источника ссылок. [Число](lang__def_Number.md) представляет индекс колонки источника. [Строка](lang__def_String.md) представляет имя колонки. Если значением параметра является [Неопределено](lang__def_Undefined.md), то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив_(Array).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [ПланСчетовВыборка.\<Имя плана счетов\>](ПланСчетовВыборка.Имя_плана_счетов_(ChartOfAccountsSelection.Chart_of_accounts_name).md).   

Описание:

Формирует выборку элементов плана счетов по заданному источнику ссылок на элементы плана счетов соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы плана счетов соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    Хозрасчетный.Ссылка КАК Ссылка     |ИЗ     |    ПланСчетов.Хозрасчетный КАК Хозрасчетный     |"; Результат = Запрос.Выполнить(); Выборка = ПланыСчетов.Хозрасчетный.ВыбратьПоСсылкам(Результат, 0); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
