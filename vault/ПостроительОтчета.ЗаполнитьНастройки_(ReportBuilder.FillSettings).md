ПостроительОтчета (ReportBuilder)

ЗаполнитьНастройки (FillSettings)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаполнитьНастройки()

Описание:

Выполняет автоматическое заполнение настроек построителя из текста запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПостроительОтчетов = Новый ПостроительОтчета; ПостроительОтчетов.Текст = ЭлементыФормы.ТекстЗапроса.ПолучитьТекст(); ПостроительОтчетов.ЗаполнитьНастройки(); |

Использование в версии:

Доступен, начиная с версии 8.0.
