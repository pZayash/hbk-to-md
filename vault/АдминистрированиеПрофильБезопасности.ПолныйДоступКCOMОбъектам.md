АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКCOMОбъектам (COMObjectFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешено использование произвольных COM-объектов.  
Истина – разрешено использование любых COM-объектов. Ложь – допустимо использование только COM-объектов, перечисленных в списке разрешенных COM-объектов профиля безопасности.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
