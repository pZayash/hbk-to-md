ЭлементТаблицаHTML (HTMLTableElement)

ПолучитьСтроки (GetRows)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьСтроки()

Возвращаемое значение:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML_(HTMLCollection).md).   

Описание:

Возвращает коллекцию элементов строк таблицы HTML-документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
