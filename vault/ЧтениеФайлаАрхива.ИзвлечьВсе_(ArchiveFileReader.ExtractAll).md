ЧтениеФайлаАрхива (ArchiveFileReader)

ИзвлечьВсе (ExtractAll)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ИзвлечьВсе(<Путь>, <ВосстанавливатьКаталоги>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь, по которому необходимо распаковать элемент файла архива.

<ВосстанавливатьКаталоги> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает, требуется ли восстанавливать структуру каталогов.  
Значение по умолчанию: [Восстанавливать](РежимВосстановленияПутейФайлаАрхива.Восстанавливать_(ArchiveFileRestoreFilePathsMode.Restore).md).

Описание:

Распаковывает все элементы из файла архива по указанному пути.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
