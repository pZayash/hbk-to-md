Описание:

Данный объект представляет собой коллекцию, содержащую следующие элементы оформления:

- Расшифровка (Details) - элемент оформления; тип [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных_(DataCompositionDetailsID).md);
- Формат (Format) - формат, применяемый к значению группировки; тип [Строка](lang__def_String.md);
- ЦветВДиаграмме (ColorInChart) - цвет, используемый для вывода группировки в диаграмме; тип [Цвет](Цвет_(Color).md);
- ВыводитьНаОсиТочекДиаграммы (PutOnPointsAxis) - указывает, что серия, полученная из данной группировки, откладывается на оси точек в пузырьковой и точечной диаграммах; тип [Булево](lang__Булево_(Boolean).md);
- ИспользоватьКакОбщуюСериюРазмераПузырькаВДиаграмме (UseAsCommonBubbleSizeSeries) - указывает, что серия, полученная из данной группировки, используется в качестве общей серии размера пузырька в пузырьковой диаграмме; тип [Булево](lang__Булево_(Boolean).md);
- ЛинияВДиаграмме (LineInChart) - тип линии серии, полученной из данной группировки; тип [Линия](Линия_(Line).md);
- МаркерВДиаграмме (MarkerInChart) - тип маркера серии, полученной из данной группировки; тип [ТипМаркераДиаграммы](ТипМаркераДиаграммы_(ChartMarkerType).md);
- ИндикаторВДиаграмме (IndicatorInChart) указывает, является ли индикатором серия, полученная из данной группировки; тип [Булево](lang__Булево_(Boolean).md);
- ОтображатьГрафическоеПредставлениеДанныхВДиаграмме (ShowGraphicalDataRepresentationInChart) - указывает, что серия, полученная из данной группировки, отображается на диаграмме; тип [ОтображениеВДиаграмме](ОтображениеВДиаграмме_(ShowInChart).md);
- ОтображатьГрафическоеПредставлениеДанныхВЛегендеДиаграммы (ShowGraphicalDataRepresentationInChartLegend) - указывает, что серия, полученная из данной группировки, отображается в легенде диаграммы; тип [ОтображениеВЛегендеДиаграммы](ОтображениеВЛегендеДиаграммы_(ShowInChartLegend).md);
- ИнформационныеЛинииВДиаграмме (ReferenceLinesInChart) - содержит коллекцию информационных линий для точки диаграммы, полученной из данной группировки; тип [Соответствие](Соответствие_(Map).md);
- ИнформационныеИнтервалыВДиаграмме (ReferenceBandsInChart) - содержит коллекцию информационных интервалов для точки диаграммы, полученной из данной группировки; тип [Соответствие](Соответствие_(Map).md);
- ТипГрафическогоПредставленияВДиаграмме (GraphicalRepresentationTypeInChart) - тип серии, отличный от типа диаграммы; тип [ТипГрафическогоПредставленияСерииДиаграммы](ТипГрафическогоПредставленияСерииДиаграммы_(ChartSeriesGraphicalRepresentationType).md);
- ТипНакопленияВДиаграмме (StackTypeInChart) - тип накопления серии, отличный от типа диаграммы. Группа накопления серии создаётся автоматически, на основе элемента условного оформления. Разные элементы условного оформления имеют разные группы;тип [ТипНакопленияСерииДиаграммы](ТипНакопленияСерииДиаграммы_(ChartSeriesStackType).md);
- ИспользуемаяОсьЗначенийВДиаграмме (UsedValueAxisInChart) - настраивает использование серией дополнительной оси значений;тип [ИспользуемаяОсьЗначенийДиаграммы](ИспользуемаяОсьЗначенийДиаграммы_(UsedChartValuesAxis).md);
- ГруппаНакопленияВДиаграмме (StackGroupInChart) - группа накопления серии; тип [Строка](lang__def_String.md);
- ФорматЗначенийВПодписяхДиаграммы (ChartValueLabelFormat) - формат подписей значений в подписях; тип [Строка](lang__def_String.md);
- ФорматВТаблицеДанныхДиаграммы (ChartDataTableFormat) - формат значений в таблице данных; тип [Строка](lang__def_String.md)};

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: ChartGroupAppearance.
