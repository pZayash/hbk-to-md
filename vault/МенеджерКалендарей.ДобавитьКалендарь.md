МенеджерКалендарей (CalendarsManager)

ДобавитьКалендарь (AddCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ДобавитьКалендарь(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ДанныеКалендаряУчетнойЗаписи](ДанныеКалендаряУчетнойЗаписи.md).   
Данные календаря учетной записи.

Возвращаемое значение:

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря.md).   

Описание:

Добавляет новый календарь в мобильном устройстве и возвращает его локальный ключ.   
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяДобавлениеКалендарей](МенеджерКалендарей.ПоддерживаетсяДобавлениеКалендарей.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей.md), метод [ПоддерживаетсяДобавлениеКалендарей](МенеджерКалендарей.ПоддерживаетсяДобавлениеКалендарей.md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
