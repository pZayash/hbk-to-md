Планировщик (Planner)

ПоведениеЭлементовПриНедостаткеМеста (ItemsBehaviorWhenSpaceInsufficient)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [ПоведениеЭлементовПланировщикаПриНедостаткеМеста](ПоведениеЭлементовПланировщикаПриНедостаткеМеста.md).   
Определяет поведение элементов планировщика при недостатке места в поле планировщика.  
Свойство используется при положении шкалы времени [Верх](ПоложениеШкалыВремени.Верх.md) или [Низ](ПоложениеШкалыВремени.Низ.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение по умолчанию - [СворачиватьЭлементы](ПоведениеЭлементовПланировщикаПриНедостаткеМеста.СворачиватьЭлементы.md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
