ТипОтображенияСерииСлояГеографическойСхемы (GeographicalSchemaLayerSeriesShowMode)

КруговаяСРазмером (SizedPie)

Доступен, начиная с версии 8.0.

Описание:

Режим отображения серии круговой диаграммой, при которой размеры секторов в диаграмме пропорциональны значениям серий, а величина диаметра диаграммы зависит суммы значений.

Использование в версии:

Доступен, начиная с версии 8.0.
