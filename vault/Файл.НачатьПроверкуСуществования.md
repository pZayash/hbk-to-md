Файл (File)

НачатьПроверкуСуществования (BeginCheckingExistence)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПроверкуСуществования(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <Существует> - помещается признак существования, тип Булево. Истина, если файл существует.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает проверку существования файла.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Файл](Файл.md), метод [Существует](Файл.Существует.md)  
[Файл](Файл.md), метод [СуществуетАсинх](Файл.СуществуетАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
