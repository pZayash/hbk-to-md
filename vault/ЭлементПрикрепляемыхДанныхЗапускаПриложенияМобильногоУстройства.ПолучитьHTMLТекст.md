ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipDataItem)

ПолучитьHTMLТекст (GetHTMLText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьHTMLТекст()

Возвращаемое значение:

Тип: Строка, Неопределено.   
HTML-текст, если он есть в прикрепляемых данных, иначе Неопределено.

Описание:

Возвращает HTML-текст из прикрепляемых данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
