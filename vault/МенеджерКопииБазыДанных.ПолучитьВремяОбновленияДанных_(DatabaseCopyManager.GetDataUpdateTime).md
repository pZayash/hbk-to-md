МенеджерКопииБазыДанных (DatabaseCopyManager)

ПолучитьВремяОбновленияДанных (GetDataUpdateTime)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

ПолучитьВремяОбновленияДанных()

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   

Описание:

Получает время последнего обновления копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
