ОбъектМетаданных: HTTPСервис (MetadataObject: HTTPService)

ШаблоныURL (URLTemplates)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных - шаблоны URL-адресов, обрабатываемых сервисом. Элементами коллекции являются объекты типа [ОбъектМетаданных: ШаблонURLHTTPСервиса](ОбъектМетаданных_ШаблонURLHTTPСервиса_(MetadataObject_HTTPServiceURLTemplate).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
