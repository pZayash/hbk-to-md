Глобальный контекст (Global context)

Base64Строка (Base64String)

Доступен, начиная с версии 8.0.

Синтаксис:

Base64Строка(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные, которые необходимо закодировать по алгоритму base64.

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает строку, закодированную по алгоритму base64.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Base64Значение](Глобальный_контекст.Base64Значение.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
