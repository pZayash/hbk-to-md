НеобходимостьЗавершенияСоединения (ConnectionStopRequest)

НеобходимоЗавершить (StopConnection)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - рабочий процесс, с которым установлено текущее соединение выключен. В этом случае соединение должно быть завершено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В файловом варианте всегда равно [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.1.
