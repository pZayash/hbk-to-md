# ТочкиМаршрута

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются точки маршрута бизнес-процесса.  
Возможно обращение к точке маршрута бизнес-процесса посредством оператора [...]. В качестве аргумента передается индекс точки маршрута бизнес-процесса в коллекции (нумерация с 0).  

Свойства:

[\<Имя точки маршрута бизнес-процесса\> (\<Business process route point name\>)](ТочкиМаршрута.Имя_точки_маршрута_бизнес-процесса.md)  

Методы:

[Индекс (IndexOf)](ТочкиМаршрута.Индекс.md)  
[Количество (Count)](ТочкиМаршрута.Количество.md)  
[Получить (Get)](ТочкиМаршрута.Получить.md)  

Описание:

Представляет коллекцию точек маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Количество = БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута.Количество();  Для каждого Точка Из БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута Цикл     Сообщить(НСтр("ru='Точка маршрута: ';en='Route point:'") + Точка.Имя);     Index = БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута.Индекс(Точка);     Сообщить(НСтр("ru='Индекс точки: ';en='Point index: '") + Index); КонецЦикла; |
