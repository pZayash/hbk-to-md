МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьДанныеКвитанцийВстроенныхПокупок (GetInAppPurchaseReceiptsData)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьДанныеКвитанцийВстроенныхПокупок(<Квитанции>, <ИдентификаторыПокупок>)

Параметры:

<Квитанции> (обязательный)

Тип: [Массив](Массив.md), Строка.   
Квитанция или массив объектов типа [КвитанцияВстроеннойПокупки](КвитанцияВстроеннойПокупки.md), поля для которой или которых требуется получить.

<ИдентификаторыПокупок> (необязательный)

Тип: [Массив](Массив.md), Строка.   
При работе под управлением iOS параметр является обязательным.  
Используется для определения покупок, для которых требуется получить [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки.md). Если задан неверный идентификатор, то [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки.md) для него не будут созданы.  
При работе под управлением других ОС параметр игнорируется.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив объектов типа [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки.md). Если переданная квитанция не содержит данных или данные невозможно прочесть, будет сгенерировано исключение.  
Важно! При работе под управлением iOS необходимо указать дополнительный параметр <ИдентификаторыПокупок>. При работе под управлением других ОС данный параметр игнорируется.

Описание:

Возвращает данные квитанций встроенных покупок.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.13.
