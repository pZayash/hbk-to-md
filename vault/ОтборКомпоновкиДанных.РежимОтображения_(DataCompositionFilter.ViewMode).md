ОтборКомпоновкиДанных (DataCompositionFilter)

РежимОтображения (ViewMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтображенияЭлементаНастройкиКомпоновкиДанных](РежимОтображенияЭлементаНастройкиКомпоновкиДанных_(DataCompositionSettingsItemViewMode).md).   
Определяет режим отображения.   
Используется при отображении списка в зависимости от режима отображения табличного поля (см. соответствующие свойства у расширений табличного поля отбора и пользовательских настроек).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md), свойство [ИдентификаторПользовательскойНастройки](ОтборКомпоновкиДанных.ИдентификаторПользовательскойНастройки_(DataCompositionFilter.UserSettingID).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
