# ЗадачиМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры задач.  
Возможно обращение к менеджеру задачи посредством оператора [...]. В качестве аргумента передается имя задачи, как оно задано в конфигурации.  

Свойства:

[\<Имя задачи\> (\<Task name\>)](ЗадачиМенеджер.Имя_задачи_(TasksManager.Task_name).md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ЗадачиМенеджер.ТипВсеСсылки_(TasksManager.AllRefsType).md)  

Описание:

Предназначен для управления задачами и предоставляет доступ к значениям типа [ЗадачаМенеджер.](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md)<[Имя задачи](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [Задачи](Глобальный_контекст.Задачи_(Global_context.Tasks).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Задачи](Глобальный_контекст.Задачи_(Global_context.Tasks).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
