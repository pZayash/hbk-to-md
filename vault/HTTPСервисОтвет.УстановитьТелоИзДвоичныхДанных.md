HTTPСервисОтвет (HTTPServiceResponse)

УстановитьТелоИзДвоичныхДанных (SetBodyFromBinaryData)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьТелоИзДвоичныхДанных(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные, которые будут записаны в тело HTTP-ответа.

Описание:

Устанавливает тело HTTP-ответа из двоичных данных.

Доступность:

Сервер, толстый клиент.

См. также:

[HTTPСервисОтвет](HTTPСервисОтвет.md), метод [УстановитьТелоИзСтроки](HTTPСервисОтвет.УстановитьТелоИзСтроки.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
