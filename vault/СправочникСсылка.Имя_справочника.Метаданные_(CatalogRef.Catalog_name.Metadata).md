# СправочникСсылка.<Имя справочника>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникСсылка.<Имя справочника>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md) › [Методы](СправочникСсылка.Имя_справочника__Методы.md)

СправочникСсылка.<Имя справочника> (CatalogRef.<Catalog name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник_(MetadataObject_Catalog).md).   

Описание:

Предоставляет доступ к объекту описания метаданных справочника. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md).   
Например: Метаданные.Справочники.Номенклатура.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если Не ТоварСсылка.Пустая() Тогда     МетаданныеСправочника = ТоварСсылка.Метаданные();     Сообщить("Представление справочника - " +          МетаданныеСправочника.Представление()); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
