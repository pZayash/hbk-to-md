ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

ТипНомера (NumberType)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ТипНомераБизнесПроцесса](ТипНомераБизнесПроцесса_(BusinessProcessNumberType).md).   
Определяет тип номера бизнес-процесса (например, [Число](ТипНомераБизнесПроцесса.Число_(BusinessProcessNumberType.Number).md), [Строка](ТипНомераБизнесПроцесса.Строка_(BusinessProcessNumberType.String).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
