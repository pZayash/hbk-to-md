ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

УстановитьПараметрыСоединенияСеанса (SetSessionConnectionParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьПараметрыСоединенияСеанса(<ПараметрыСоединения>)

Параметры:

<ПараметрыСоединения> (обязательный)

Тип: [ПараметрыСоединенияВнешнегоИсточникаДанных](ПараметрыСоединенияВнешнегоИсточникаДанных.md).   
Параметры соединения, которые нужно установить.

Описание:

Устанавливает параметры соединения с внешним источником данных для текущего сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если у пользователя нет права на изменение свойства и значение свойства изменено, то выдается исключение. Например, у пользователя нет права ИзменениеСтандартнойАутентификацииСеанса и в параметре было установлено свойство [ИмяПользователя](ПараметрыСоединенияВнешнегоИсточникаДанных.ИмяПользователя.md), то будет выдано исключение.  
Для свойств [ИмяПользователя](ПараметрыСоединенияВнешнегоИсточникаДанных.ИмяПользователя.md), [Пароль](ПараметрыСоединенияВнешнегоИсточникаДанных.Пароль.md), [ПарольУстановлен](ПараметрыСоединенияВнешнегоИсточникаДанных.ПарольУстановлен.md) проверяется права ИзменениеСтандартнойАутентификацииСеанса.  
Для остальных свойств проверяются административные права.

Использование в версии:

Доступен, начиная с версии 8.2.
