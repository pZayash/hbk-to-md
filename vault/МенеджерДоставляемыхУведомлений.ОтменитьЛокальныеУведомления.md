МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ОтменитьЛокальныеУведомления (CancelLocalNotifications)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОтменитьЛокальныеУведомления()

Описание:

Отменяет локальные уведомления, для которых установлена задержка и которые еще не выведены.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
