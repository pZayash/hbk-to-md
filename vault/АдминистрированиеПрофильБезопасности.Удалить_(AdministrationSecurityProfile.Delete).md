# АдминистрированиеПрофильБезопасности.Удалить

АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить()

Описание:

Отменяет регистрацию профиля безопасности кластера. Все объекты (виртуальные каталоги, COM-класс, внешние компоненты, приложения, адреса сети Интернет, внешние модули), связанные с этим профилем, будут удалены.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
