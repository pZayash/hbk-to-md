# РежимУправленияБлокировкойДанныхПоУмолчанию.Автоматический

РежимУправленияБлокировкойДанныхПоУмолчанию (DefaultDataLockControlMode)

Автоматический (Automatic)

Доступен, начиная с версии 8.1.

Описание:

Если в свойстве установлен режим управления блокировкой данных конфигурации, то все транзакции, вне зависимости от установок данного свойства для объектов и параметра вызова метода [НачатьТранзакцию](Глобальный_контекст.НачатьТранзакцию_(Global_context.BeginTransaction).md), будут выполняться в автоматическом режиме.  
Если установлен у любого другого объекта метаданных, а у конфигурации установлено [АвтоматическийИУправляемый](РежимУправленияБлокировкойДанныхПоУмолчанию.АвтоматическийИУправляемый_(DefaultDataLockControlMode.AutomaticAndManaged).md), транзакции, создаваемые автоматически системой, будут выполняться в автоматическом режиме.

Использование в версии:

Доступен, начиная с версии 8.1.
