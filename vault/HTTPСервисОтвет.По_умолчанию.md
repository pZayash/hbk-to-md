HTTPСервисОтвет (HTTPServiceResponse)

По коду состояния, причине и заголовкам

Доступен, начиная с версии 8.3.5.

Синтаксис:

Новый HTTPСервисОтвет(<КодСостояния>, <Причина>, <Заголовки>)

Параметры:

<КодСостояния> (обязательный)

Тип: [Число](lang__def_Number.md).   
Код состояния (ответа), возвращаемый HTTP-сервисом в соответствии с документом: http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html.  
  
Примечание:  
HTTP-сервер Apache, в отличие от IIS, не поддерживает произвольные коды ответа. При использовании Apache рекомендуется использовать коды ответа, указанные в документе: http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html.

<Причина> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит текстовую расшифровку кода HTTP-статуса.  
Для стандартных статусов в ответе сервиса формируется автоматически.

<Заголовки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Коллекция заголовков, которые должны быть переданы на клиента.

Описание:

Ответ HTTP-сервиса формируется по коду состояния, причине и заголовкам.

Использование в версии:

Доступен, начиная с версии 8.3.5.
