ЭлементыЛегендыГеографическойСхемы (GeographicalSchemaLegendItems)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [ЭлементЛегендыГеографическойСхемы](ЭлементЛегендыГеографическойСхемы_(GeographicalSchemaLegendItem).md).   
Объект, индекс которого в коллекции требуется получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс переданного объекта в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
