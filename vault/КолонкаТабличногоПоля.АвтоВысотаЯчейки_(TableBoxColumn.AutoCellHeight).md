КолонкаТабличногоПоля (TableBoxColumn)

АвтоВысотаЯчейки (AutoCellHeight)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость вычисления высоты ячейки данной колонки. При этом в свойстве [ВысотаЯчейки](КолонкаТабличногоПоля.ВысотаЯчейки_(TableBoxColumn.CellHeight).md) указывается максимальная высота ячейки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
