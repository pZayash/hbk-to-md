ЭлементЛинияHTML (HTMLHRElement)

УдалитьАтрибут (RemoveAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя атрибута.

Описание:

Удаляет атрибут с указанным именем.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если данный узел в режиме Только для чтения, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
