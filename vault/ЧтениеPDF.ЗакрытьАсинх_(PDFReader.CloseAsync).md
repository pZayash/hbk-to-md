ЧтениеPDF (PDFReader)

ЗакрытьАсинх (CloseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗакрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Закрывает открытый файл.   
Закрытие файла выполняется автоматически при удалении объекта [ЧтениеPDF](ЧтениеPDF_(PDFReader).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [Закрыть](ЧтениеPDF.Закрыть_(PDFReader.Close).md)  
[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [НачатьЗакрытие](ЧтениеPDF.НачатьЗакрытие_(PDFReader.BeginClose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
