# Расширение формы вида расчета.ПриИзмененииДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [Расширение формы вида расчета](Расширение_формы_вида_расчета_(Calculation_type_form_extension).md) › [События](Расширение_формы_вида_расчета__События.md)

Расширение формы вида расчета (Calculation type form extension)

ПриИзмененииДанных (OnDataChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриИзмененииДанных()

Описание:

Возникает при инициализации формы, перечитывании данных, смене редактируемого объекта.  
Предназначено для отображения данных объекта в элементах управления формы, которые не связаны непосредственно с данными, например, для отображения картинки или текста, который формируется по данным объекта и т.д.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
