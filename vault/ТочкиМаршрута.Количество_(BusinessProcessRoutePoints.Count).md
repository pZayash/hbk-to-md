ТочкиМаршрута (BusinessProcessRoutePoints)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество точек маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Количество = БизнесПроцессы.ВыпискаСчета.ТочкиМаршрута.Количество(); |

Использование в версии:

Доступен, начиная с версии 8.0.
