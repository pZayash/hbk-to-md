# XBase

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Для работы с базами данных формата DBF (dBase III) в системе может использоваться специальный объект - [XBase](XBase_(XBase).md). Механизм работы с базами данных формата DBF предназначен для обеспечения возможности манипулирования ими непосредственно из встроенного языка 1С:Предприятия. Каждый объект [XBase](XBase_(XBase).md) может быть связан с одним файлом базы данных.  
Замечание. Не поддерживается работа с Xbase в 64-разрядной операционной системе Linux.
