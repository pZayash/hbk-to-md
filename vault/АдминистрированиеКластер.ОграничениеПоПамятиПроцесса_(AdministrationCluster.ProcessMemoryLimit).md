АдминистрированиеКластер (AdministrationCluster)

ОграничениеПоПамятиПроцесса (ProcessMemoryLimit)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет объем виртуального адресного пространства (в килобайтах), занятого рабочим процессом, превышение которого приведет к перезапуску рабочего процесса.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
