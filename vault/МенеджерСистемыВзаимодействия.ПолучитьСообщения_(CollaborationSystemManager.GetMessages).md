МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьСообщения (GetMessages)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьСообщения(<Отбор>)

Параметры:

<Отбор> (обязательный)

Тип: [ОтборСообщенийСистемыВзаимодействия](ОтборСообщенийСистемыВзаимодействия_(CollaborationSystemMessagesFilter).md).   
Содержит значения отбора и сортировки.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Возвращает массив элементов типа [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).

Описание:

Возвращают список сообщений обсуждения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
