МенеджерПолученияЛицензий (LicenseAcquisitionManager)

СформироватьКонтрольнуюСуммуСтрокиКодаПолученияЛицензии (GenerateLicenseAcquisitionCodeStringChecksum)

Доступен, начиная с версии 8.3.20.

Синтаксис:

СформироватьКонтрольнуюСуммуСтрокиКодаПолученияЛицензии(<СтрокаЦифр>)

Параметры:

<СтрокаЦифр> (обязательный)

Тип: Строка.   
Последовательность цифр, для которой вычисляется контрольная сумма.

Возвращаемое значение:

Тип: Строка.   
Контрольная сумма:

- 3 десятичные цифры - если исходная строка состояла из цифр,
- 000 - если исходная строка пустая,
- пустая строка - если в исходной строке содержались не только цифры.

Описание:

Вычисляет контрольную сумму строки, состоящей из десятичных цифр.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
