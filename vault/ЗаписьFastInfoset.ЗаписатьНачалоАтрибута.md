ЗаписьFastInfoset (FastInfosetWriter)

ЗаписатьНачалоАтрибута (WriteStartAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Запись с полным именем атрибута

Синтаксис:

ЗаписатьНачалоАтрибута(<ПолноеИмя>)

Параметры:

<ПолноеИмя> (обязательный)

Тип: Строка.   
Полное имя записываемого атрибута. Может включать префикс. Например: "nsp:weight".

Вариант синтаксиса: Запись с локальным именем и URI пространства имен

Синтаксис:

ЗаписатьНачалоАтрибута(<ЛокальноеИмя>, <URIПространстваИмен>)

Параметры:

<ЛокальноеИмя> (обязательный)

Тип: Строка.   
Локальное имя атрибута (без префикса).

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен для имени атрибута.

Описание варианта метода:

Соответствие пространства имен должно быть записано при записи вышестоящего атрибута или при записи данного атрибута.

Описание:

Записывает начало атрибута. Может вызываться после вызова метода [ЗаписьFastInfoset.ЗаписатьНачалоЭлемента](ЗаписьFastInfoset.ЗаписатьНачалоЭлемента.md), но до вызова методов [ЗаписьFastInfoset.ЗаписатьКонецЭлемента](ЗаписьFastInfoset.ЗаписатьКонецЭлемента.md), [ЗаписьFastInfoset.ЗаписатьТекст](ЗаписьFastInfoset.ЗаписатьТекст.md), [ЗаписьFastInfoset.ЗаписатьКомментарий](ЗаписьFastInfoset.ЗаписатьКомментарий.md), [ЗаписьFastInfoset.ЗаписатьСекциюCDATA](ЗаписьFastInfoset.ЗаписатьСекциюCDATA.md) или [ЗаписьFastInfoset.ЗаписатьНачалоАтрибута](ЗаписьFastInfoset.ЗаписатьНачалоАтрибута.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
