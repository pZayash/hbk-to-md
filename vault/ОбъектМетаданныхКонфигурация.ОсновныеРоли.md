ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновныеРоли (DefaultRoles)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийСвойстваОбъектаМетаданных](КоллекцияЗначенийСвойстваОбъектаМетаданных.md).   
Список ролей, которые будут использованы, если список пользователей прикладного решения пуст.   
В этом случае при начале работы системы авторизация доступа не выполняется и права доступа определяются указанным в свойстве набором ролей.   
Если список пользователей пуст, считается, что пользователь обладает административными правами вне зависимости от значения права "Администрирование" у всех ролей, указанных как основные.  
Если основные роли конфигурации не указаны и список пользователей пуст, пользователь работает без ограничения прав доступа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
