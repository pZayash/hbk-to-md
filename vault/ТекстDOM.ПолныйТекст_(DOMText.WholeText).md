# ТекстDOM.ПолныйТекст

ТекстDOM (DOMText)

ПолныйТекст (WholeText)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст логически смежных текстовых узлов.  
Текст формируется из текстовых узлов, смежных с данным, в порядке следования в документе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
