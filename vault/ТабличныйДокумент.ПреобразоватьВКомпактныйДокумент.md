ТабличныйДокумент (SpreadsheetDocument)

ПреобразоватьВКомпактныйДокумент (ConvertToCompactMode)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПреобразоватьВКомпактныйДокумент(<Ширина>, <ИспользованиеШирины>)

Параметры:

<Ширина> (необязательный)

Тип: Число.   
Задает максимальную ширину содержимого табличного документа (в "средних символах"), до которой документ будет сжат.  
Если значение не задано, то максимальная ширина рассчитывается автоматически на основе информации об экране устройства конечного пользователя. Если такой информации нет, то используется стандартное значение, равное 20.

<ИспользованиеШирины> (необязательный)

Тип: [ИспользованиеШириныСжатияТабличногоДокумента](ИспользованиеШириныСжатияТабличногоДокумента.md).   
Задает использование ширины, переданной в качестве первого аргумента.

Описание:

Преобразовывает текущий табличный документ в компактный. Ширина содержимого нового документа не превышает ширину, переданную в качестве аргумента. Если переданная ширина слишком мала, то документ не изменяется.  
Для сжатия табличного документа, расположенного на форме, по размеру поля табличного документа рекомендуется использовать в качестве ширины сжатия результат метода расширения поля формы для табличного документа [ПолучитьРекомендуемуюШиринуСжатия](Расширение_поля_формы_для_поля_табличного_документа.ПолучитьРекомендуемуюШиринуСжатия.md).

Доступность:

Сервер, толстый клиент.
