РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ПолучитьГлубинуГрафа (GetGraphDepth)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьГлубинуГрафа(<НомерКорневойВершины>)

Параметры:

<НомерКорневойВершины> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер неизвестной СЛАУ, которая должна существовать в рамках данной СЛАУ.  
Неотрицательное целое число.   
С вершины графа с указанным номером должен начинаться поиск в глубину графа.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Число, равное максимальной глубине графа, полученной в результате поиска в глубину, начатого с вершины с номером <НомерКорневойВершины>.

Описание:

Рассчитывает глубину графа СЛАУ.  
Для корректной работы требуется установка значений свойств [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей_(SystemOfLinearEquationsCalculation.RelationDataSource).md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях_(SystemOfLinearEquationsCalculation.EquationsInRelationsColumn).md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях_(SystemOfLinearEquationsCalculation.VariablesInRelationsColumn).md), а также чтобы в коллекции [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений_(SystemDescriptions).md) был добавлен один объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md), в котором должна быть указана только [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях_(SystemOfLinearEquationsDescription.FactorsInRelationsColumn).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
