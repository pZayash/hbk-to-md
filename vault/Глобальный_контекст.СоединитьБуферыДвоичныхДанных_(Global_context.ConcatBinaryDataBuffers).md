Глобальный контекст (Global context)

СоединитьБуферыДвоичныхДанных (ConcatBinaryDataBuffers)

Доступен, начиная с версии 8.3.9.

Синтаксис:

СоединитьБуферыДвоичныхДанных(<Части>, <Разделитель>, <ПорядокБайтов>)

Параметры:

<Части> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md), которые требуется объединить.

<Разделитель> (необязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Буфер, вставляемый между частями. Если не указан, буферы будут объединены без использования разделителя.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов.

Возвращаемое значение:

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   

Описание:

Соединяет набор буферов в один буфер. С помощью необязательного параметра <Разделитель> можно задать буфер, который будет использован в качестве разделителя между частями.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
