МенеджерБезопасногоХранилища (SecureStorageManager)

ПоддерживаетсяЗащитаДоступа (AccessProtectionSupported)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПоддерживаетсяЗащитаДоступа(<СпособЗащитыДоступа>, <СпособПроверкиПользователя>)

Параметры:

<СпособЗащитыДоступа> (обязательный)

Тип: [СпособЗащитыДоступаБезопасногоХранилища](СпособЗащитыДоступаБезопасногоХранилища.md).   
Используемый способ защиты доступа.

<СпособПроверкиПользователя> (необязательный)

Тип: [СпособДополнительнойПроверкиПользователя](СпособДополнительнойПроверкиПользователя.md).   
Используется только если <СпособБиометрическойПроверки> равен [ТребуетсяДополнительнаяПроверкаПользователя](СпособЗащитыДоступаБезопасногоХранилища.ТребуетсяДополнительнаяПроверкаПользователя.md), иначе игнорируется.  
Значение по умолчанию: [БиометрическаяИлиВводПароля](СпособДополнительнойПроверкиПользователя.БиометрическаяИлиВводПароля.md).

Возвращаемое значение:

Тип: Булево.   

Описание:

Проверяет наличие возможности хранить данные в безопасном хранилище, защищенном выбранным способом защиты доступа.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
