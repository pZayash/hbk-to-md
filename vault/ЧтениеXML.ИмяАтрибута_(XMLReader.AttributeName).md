# ЧтениеXML.ИмяАтрибута

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md) › [ЧтениеXML](ЧтениеXML_(XMLReader).md) › [Методы](ЧтениеXML__Методы.md)

ЧтениеXML (XMLReader)

ИмяАтрибута (AttributeName)

Доступен, начиная с версии 8.1.

Синтаксис:

ИмяАтрибута(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс в списке атрибутов.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает имя атрибута по индексу. Так как атрибуты могут быть только у элемента XML, доступ к ним может осуществляться только при прочтении узла, имеющего тип [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
