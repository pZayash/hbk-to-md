# ОбъектМетаданных: ОбщаяКоманда.ВключатьСправкуВСодержание

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: ОбщаяКоманда](ОбъектМетаданных_ОбщаяКоманда_(MetadataObject_CommonCommand).md) › [Свойства](ОбъектМетаданных_ОбщаяКоманда__Свойства.md)

ОбъектМетаданных: ОбщаяКоманда (MetadataObject: CommonCommand)

ВключатьСправкуВСодержание (IncludeHelpInContents)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то справка по данному объекту метаданных (свойство [Справка](ОбъектМетаданных_БизнесПроцесс.Справка_(MetadataObject_BusinessProcess.Help).md)) будет включена в общую справку по конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
