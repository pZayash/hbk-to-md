ИспользованиеТекущейСтрокиТаблицы (TableCurrentRowUse)

ОтображениеВыделенияИВыбор (SelectionPresentationAndChoice)

Доступен, начиная с версии 8.3.12.

Описание:

- Текущая строка используется, если у таблицы есть хотя бы одна строка.
- Текущая строка отображается визуально.
- При нажатии на строку вызывается событие [ПриАктивизацииПоля](ТаблицаФормы.ПриАктивизацииПоля_(FormTable.OnActivateField).md). Событие [Выбор](ТаблицаФормы.Выбор_(FormTable.Selection).md) не вызывается.
- В правой части строки отображается кнопка, при нажатии которой вызываются события [ПриАктивизацииПоля](ТаблицаФормы.ПриАктивизацииПоля_(FormTable.OnActivateField).md) и [Выбор](ТаблицаФормы.Выбор_(FormTable.Selection).md).

Использование в версии:

Доступен, начиная с версии 8.3.12.
