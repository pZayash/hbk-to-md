# ТипСтандартизацииАнализаДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Анализ данных](Анализ_данных.md)

Доступен, начиная с версии 8.0.

Значения

[НеСтандартизировать (DontStandardize)](ТипСтандартизацииАнализаДанных.НеСтандартизировать_(DataAnalysisStandardizationType.DontStandardize).md)  
[Стандартизировать (Standardize)](ТипСтандартизацииАнализаДанных.Стандартизировать_(DataAnalysisStandardizationType.Standardize).md)  

Описание:

Содержит варианты стандартизации данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: DataAnalysisStandardizationType.

Использование в версии:

Доступен, начиная с версии 8.0.
