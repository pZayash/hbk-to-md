ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

ПараметрыВнешнейСистемы (ExternalSystemParameters)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [Соответствие](Соответствие_(Map).md), [Неопределено](lang__def_Undefined.md).   
Параметры интеграции с внешним мессенджером. Значение свойства зависит от типа внешней системы.  
Если интеграция получена без права "Администрирование данных" (или не в привилегированном режиме), то содержит [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
