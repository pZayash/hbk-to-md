Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ОбработкаРасшифровки (DetailProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаРасшифровки(<Расшифровка>, <СтандартнаяОбработка>, <Дата>)

Параметры:

<Расшифровка>

Тип: Произвольный, [Массив](Массив_(Array).md).   
Значение расшифровки точки, серии, значения диаграммы Ганта или метки элемента шкалы времени. Если параметр - [Массив](Массив_(Array).md), то первый элемент - расшифровка значения диаграммы Ганта, второй и последующие - расшифровки интервалов диаграммы Ганта (в случае перекрывающихся интервалов).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Дата>

Тип: [Дата](lang__def_Date.md).   
Выбранная дата.

Описание:

Вызывается при выборе значения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
