СпособКодированияНеASCIIСимволовИнтернетПочтовогоСообщения (InternetMailMessageNonASCIISymbolsEncodingMode)

БезКодирования (None)

Доступен, начиная с версии 8.1.

Описание:

Все поля заголовков, содержащие не ASCII символы кодироваться не будут. Все символы будут конвертироваться в кодировку, заданною свойством [Кодировка](ИнтернетПочтовоеСообщение.Кодировка_(InternetMailMessage.Encoding).md).

Использование в версии:

Доступен, начиная с версии 8.1.
