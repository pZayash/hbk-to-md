Расширение поля формы для поля календаря (Form field extension for a calendar field)

КонецПериодаОтображения (EndOfRepresentationPeriod)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Неопределено, Дата.   
Содержит верхнюю границу периода видимых в календаре дат. За указанную границу перемещение в календаре невозможно.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Неопределено - нет ограничения.

Использование в версии:

Доступен, начиная с версии 8.2.
