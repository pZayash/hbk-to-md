ПараметрыСканированияДокументов (DocumentScanningParameters)

РазрешитьАвтоЗахват (AllowAutoCapture)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет отображением элемента управления для автоматического получения фотографии и поиска документа на фотографии в интерфейс сканирования документа.   
Когда элемент управления включен, во время поиска документа на предварительном изображении с камеры устройства выполняется проверка его качества, и если оно признано удовлетворительным, производится автоматическое получение фотографии. На полученной фотографии выполняется поиск документа и показ полученного документа пользователю.   
Элемент управления по умолчанию выключен.   
Текущее состояние элемента управления запоминается между вызовами метода [ПоказатьСканированиеДокументов](СредстваМультимедиа.ПоказатьСканированиеДокументов_(MultimediaTools.ShowDocumentScanning).md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
