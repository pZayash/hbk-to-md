ЭлементГрафическойСхемыСоединительнаяЛиния (GraphicalSchemaItemConnectionLine)

НачалоВариант (BeginCase)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ВариантЭлементаГрафическойСхемыВыборВарианта](ВариантЭлементаГрафическойСхемыВыборВарианта_(GraphicalSchemaItemSwitchCase).md).   
Содержит вариант, из которого исходит линия.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Имеет смысл, только если линия подходит к элементу типа [ЭлементГрафическойСхемыВыборВарианта](ЭлементГрафическойСхемыВыборВарианта_(GraphicalSchemaItemSwitch).md), иначе [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.1.
