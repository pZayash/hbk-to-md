ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

ЭтотУзел (ThisNode)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтотУзел()

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   

Описание:

Получает ссылку на предопределенный узел, соответствующий данной базе данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ВыборкаУзлов = ПланыОбмена.УдаленныеОфисы.Выбрать(); Пока ВыборкаУзлов.Следующий() Цикл     // Производим обмен данными со всеми узлами, кроме текущего (ЭтотУзел)      Если ВыборкаУзлов.Ссылка<>ПланыОбмена.УдаленныеОфисы.ЭтотУзел() Тогда         УзелОбъект = ВыборкаУзлов.ПолучитьОбъект();         // Получение сообщения          УзелОбъект.ПрочитатьСообщениеСИзменениями();         // Формирование сообщения          УзелОбъект.ЗаписатьСообщениеСИзменениями();     КонецЕсли;     КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
