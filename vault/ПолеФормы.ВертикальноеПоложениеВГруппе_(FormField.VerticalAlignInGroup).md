ПолеФормы (FormField)

ВертикальноеПоложениеВГруппе (VerticalAlignInGroup)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложениеЭлемента](ВертикальноеПоложениеЭлемента_(ItemVerticalAlign).md).   
Устанавливает вертикальное расположение элемента внутри группы.  
Если имеет значение [Авто](ВертикальноеПоложениеЭлемента.Авто_(ItemVerticalAlign.Auto).md), то будет испльзовано значение свойства [ВертикальноеПоложение](Расширение_группы_формы_для_обычной_группы.ВертикальноеПоложение_(Form_group_extension_for_a_usual_group.VerticalAlign).md) родительской группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
