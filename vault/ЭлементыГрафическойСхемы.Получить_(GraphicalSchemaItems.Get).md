ЭлементыГрафическойСхемы (GraphicalSchemaItems)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ЭлементГрафическойСхемыСоединительнаяЛиния](ЭлементГрафическойСхемыСоединительнаяЛиния_(GraphicalSchemaItemConnectionLine).md), [ЭлементГрафическойСхемыДекорация](ЭлементГрафическойСхемыДекорация_(GraphicalSchemaItemDecoration).md), [ЭлементГрафическойСхемыДействие](ЭлементГрафическойСхемыДействие_(GraphicalSchemaItemActivity).md), [ЭлементГрафическойСхемыРазделение](ЭлементГрафическойСхемыРазделение_(GraphicalSchemaItemSplit).md), [ЭлементГрафическойСхемыУсловие](ЭлементГрафическойСхемыУсловие_(GraphicalSchemaItemCondition).md), [ЭлементГрафическойСхемыЗавершение](ЭлементГрафическойСхемыЗавершение_(GraphicalSchemaItemCompletion).md), [ЭлементГрафическойСхемыСтарт](ЭлементГрафическойСхемыСтарт_(GraphicalSchemaItemStart).md), [ЭлементГрафическойСхемыСлияние](ЭлементГрафическойСхемыСлияние_(GraphicalSchemaItemJoin).md), [ЭлементГрафическойСхемыВложенныйБизнесПроцесс](ЭлементГрафическойСхемыВложенныйБизнесПроцесс_(GraphicalSchemaItemSubBusinessProcess).md), [ЭлементГрафическойСхемыОбработка](ЭлементГрафическойСхемыОбработка_(GraphicalSchemaItemProcessing).md), [ЭлементГрафическойСхемыВыборВарианта](ЭлементГрафическойСхемыВыборВарианта_(GraphicalSchemaItemSwitch).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
