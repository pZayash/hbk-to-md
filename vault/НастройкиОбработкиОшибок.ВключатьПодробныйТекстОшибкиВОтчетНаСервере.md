НастройкиОбработкиОшибок (ErrorProcessingSettings)

ВключатьПодробныйТекстОшибкиВОтчетНаСервере (IncludeDetailErrorDescriptionInReportOnServer)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке.md).   
Указывает необходимость добавлять отчет об ошибке на сервере подробный текст ошибки.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
