РежимИспользованияМодальности (ModalityUseMode)

Доступен, начиная с версии 8.3.7.

Использование:

Только чтение.

Описание:

Тип: [РежимИспользованияМодальности](РежимИспользованияМодальности.md).   
Содержит варианты режимов использования модальности.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.7.
