Свойства:

[Данные (Data)](НеизвестнаяЗаписьNDEF.Данные_(UnknownRecordNDEF.Data).md)  
[ИдентификаторЗаписи (RecordID)](НеизвестнаяЗаписьNDEF.ИдентификаторЗаписи_(UnknownRecordNDEF.RecordID).md)  

Описание:

Данный объект должен использоваться только для обработки записей неизвестного типа, обнаруженных в полученном сообщении.   
Этот тип записи может быть использован в качестве индикатора того, что фактические данные записи имеют неизвестный формат.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
