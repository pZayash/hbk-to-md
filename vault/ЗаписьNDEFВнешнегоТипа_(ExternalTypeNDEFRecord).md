Описание:

Тип записи сообщений NDEF, использующийся для хранения и передачи данных в нестандартном формате.   
Записи внешнего типа следует использовать в случае, когда поставленная задача не решается при помощи записей стандартных типов ([ТекстоваяЗаписьNDEF](ТекстоваяЗаписьNDEF_(TextRecordNDEF).md), [URIЗаписьNDEF](URIЗаписьNDEF_(URIRecordNDEF).md), [МедиаЗаписьNDEF](МедиаЗаписьNDEF_(MediaRecordNDEF).md)).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
