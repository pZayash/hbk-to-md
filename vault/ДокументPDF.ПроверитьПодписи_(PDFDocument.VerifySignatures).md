# ДокументPDF.ПроверитьПодписи

ДокументPDF (PDFDocument)

ПроверитьПодписи (VerifySignatures)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПроверитьПодписи(<МенеджерКриптографии>, <Сертификаты>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

<Сертификаты> (необязательный)

Тип: [Массив](Массив_(Array).md).   
В параметре возвращаются сертификаты, с помощью которых была произведена подпись (если сертификат включен в данные подписи).

Описание:

Проверяет действительность присоединенных к PDF-документу подписей.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Если проверка подписи завершилась неудачно, генерируется исключение категории [ОшибкаПроверкиПодписи](КатегорияОшибки.ОшибкаПроверкиПодписи_(ErrorCategory.SignatureVerificationError).md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
