ДиалогВыбораФайла (FileDialog)

ПредварительныйПросмотр (Preview)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак показа окна предварительного просмотра файлов.  
Истина - выполняется предварительный просмотр.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Не используется в режиме выбора каталога.  
В веб-клиенте и мобильном приложении данное свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.0.
