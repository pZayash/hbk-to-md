АдминистрированиеМенеджерКластера (AdministrationClusterManager)

ИмяКомпьютера (HostName)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя или IP-адрес рабочего сервера, на котором запущен процесс rmngr менеджера кластера. Значение свойства должно совпадать с именем или адресом одного из рабочих серверов, определенных в кластере. Изменение значения допустимо только для нового дополнительного менеджера кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
