# ЭлементDOM.УстановитьАтрибут

ЭлементDOM (DOMElement)

УстановитьАтрибут (SetAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

УстановитьАтрибут(<Имя>, <Значение>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Значение атрибута.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

УстановитьАтрибут(<URIПространстваИмен>, <ЛокальноеИмя>, <Значение>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен атрибута.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута.

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Значение атрибута.

Описание:

Устанавливает значение атрибута.  
Если атрибута с указанным именем нет в элементе, то он будет создан с указанным значением. В противном случае значение атрибута будет изменено. Значение атрибута при установке не разбирается.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если имя атрибута содержит символы, не допустимые стандартом XML данного документа, или данный узел в режиме Только для чтения, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
