ИспользованиеКопийБазыДанных (DatabaseCopiesUse)

ИспользоватьТолькоКопии (UseCopiesOnly)

Доступен, начиная с версии 8.3.19.

Описание:

Использовать только копии базы данных. При невозможности выполнения на копии генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
