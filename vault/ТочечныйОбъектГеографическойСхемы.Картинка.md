ТочечныйОбъектГеографическойСхемы (GeographicalSchemaObjectPoint)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, используемую для отображения, если объект отображается как картинка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
