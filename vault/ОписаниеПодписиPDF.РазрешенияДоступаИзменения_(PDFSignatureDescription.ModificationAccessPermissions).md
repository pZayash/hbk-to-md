ОписаниеПодписиPDF (PDFSignatureDescription)

РазрешенияДоступаИзменения (ModificationAccessPermissions)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [РазрешенияДоступаИзмененияPDF](РазрешенияДоступаИзмененияPDF_(PDFModificationAccessPermissions).md).   
Содержит разрешенные действия после подписания. Имеет смысл только для подписи типа [Сертифицирующая](ТипПодписиPDF.Сертифицирующая_(PDFSignatureType.Certifying).md).   
По умолчанию: [Нет](РазрешенияДоступаИзмененияPDF.Нет_(PDFModificationAccessPermissions.None).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
