ПостроительЗапроса (QueryBuilder)

ПолучитьЗапрос (GetQuery)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗапрос()

Возвращаемое значение:

Тип: [Запрос](Запрос_(Query).md).   

Описание:

Создать и вернуть запрос. Данный метод недоступен в случае, если установлено свойство [ИсточникДанных](ПостроительЗапроса.ИсточникДанных_(QueryBuilder.DataSource).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
