МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Добавить (Add)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Добавить(<Имя>, <ПараметрыПодключения>, <ПараметрыДоступа>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя внешнего хранилища двоичных данных.

<ПараметрыПодключения> (обязательный)

Тип: [ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных](ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных.md).   
Параметры подключения внешнего хранилища двоичных данных.

<ПараметрыДоступа> (обязательный)

Тип: [ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных](ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных.md).   
Параметры доступа внешнего хранилища двоичных данных.

Возвращаемое значение:

Тип: [МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.md).   

Описание:

Добавляет новое внешнее хранилище двоичных данных.  
Требуется режим монопольного доступа к информационной базе.

Доступность:

Сервер.

Примечание:

Если хранилище с данным именем уже существует, в имени содержатся пробелы, либо оно пустое, то будет сгенерировано исключение, а операция отклонена.  
Если в параметре ПараметрыПодключения поле URL или ТипURL имеют значение Неопределено, то будет сгенерировано исключение, а операция отклонена.  
  
Параметры метода будут записаны в поля <Имя> и <ПараметрыПодключения> соответственно. Изменить их можно будет только пересозданием менеджера.

Пример:

|  |
| --- |
| СтрПодключения = Новый      ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных(); СтрПодключения.URL = "https://bucket.example.com"; СтрПодключения.ТипURL =     ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;  Доступ = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных(); Доступ.ИдентификаторДоступа = "EXAMPLE"; Доступ.СекретныйКлюч = "SecretKey"; Доступ.Регион = "US";  Мнджр = ВнешниеХранилищаДвоичныхДанных             .Добавить("manager", СтрПодключения, Доступ); |
