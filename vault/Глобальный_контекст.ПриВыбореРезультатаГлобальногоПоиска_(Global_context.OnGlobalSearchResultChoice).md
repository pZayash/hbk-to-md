Глобальный контекст (Global context)

ПриВыбореРезультатаГлобальногоПоиска (OnGlobalSearchResultChoice)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПриВыбореРезультатаГлобальногоПоиска(<ЭлементРезультата>, <СтандартнаяОбработка>)

Параметры:

<ЭлементРезультата>

Тип: [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md).   
Выбранный результат поиска.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события.   
Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.

Описание:

Вызывается при выборе результата глобального поиска.   
Если после завершения работы процедуры обработки события <СтандартнаяОбработка> установлено в [Истина](lang__def_BooleanTrue.md), то в зависимости от типа значения, используемого в объекте [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md), выполняется одно из следующих действий:

- [Строка](lang__def_String.md) - переход по навигационной ссылке. Если строка является ссылкой на страницу справочной информации, выполняется открытие справки.
- [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md), [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md) - открытие обсуждения.
- [ИдентификаторСообщенияСистемыВзаимодействия](ИдентификаторСообщенияСистемыВзаимодействия_(CollaborationSystemMessageID).md) - открытие обсуждения и позиционирование на сообщение.
- Другой тип - вызов метода [ПоказатьЗначение](Глобальный_контекст.ПоказатьЗначение_(Global_context.ShowValue).md) с указанным значением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
