КоллекцияЭлементовПланировщика (PlannerItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению элемента планировщика.

Возвращаемое значение:

Тип: [ЭлементПланировщика](ЭлементПланировщика.md), Неопределено;.   
Если указанный элемент отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента планировщика по указаному значению в свойстве [Значение](ЭлементПланировщика.Значение.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементПланировщика](ЭлементПланировщика.md), свойство [Значение](ЭлементПланировщика.Значение.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
