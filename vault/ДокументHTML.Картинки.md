ДокументHTML (HTMLDocument)

Картинки (Images)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML.md).   
Содержит коллекцию элементов всех изображений HTML документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Коллекция содержит элементы типа [ЭлементКартинкаHTML](ЭлементКартинкаHTML.md).

Использование в версии:
