ЭлементДанныхКонтакта (ContactDataItem)

ТипДанных (DataType)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ТипНомераТелефонаДанныхКонтакта](ТипНомераТелефонаДанныхКонтакта.md), [ТипАдресаЭлектроннойПочтыДанныхКонтакта](ТипАдресаЭлектроннойПочтыДанныхКонтакта.md), [ТипАдресаДанныхКонтакта](ТипАдресаДанныхКонтакта.md), [ТипВебАдресаДанныхКонтакта](ТипВебАдресаДанныхКонтакта.md), [ТипОтношенийДанныхКонтакта](ТипОтношенийДанныхКонтакта.md).   
Показывает, как интерпретировать значение.  
Например, тип данных [Рабочий](ТипНомераТелефонаДанныхКонтакта.Рабочий.md) показывает, что в значении хранится номер рабочего телефона.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
