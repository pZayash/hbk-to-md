ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ПолноеНаименование (FullDescr)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолноеНаименование()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Строка, включающая наименование элемента и наименования всех вышестоящих элементов. Наименования выводятся слева направо, начиная с самого верхнего уровня, разделяются символом "/".

Описание:

Получает наименование элемента плана видов характеристик с учетом наименований всех вышестоящих элементов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
