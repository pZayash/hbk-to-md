Массив (Array)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс удаляемого элемента.

Описание:

Удаляет значение из массива по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если указанный в параметре индекс больше, чем [ВГраница](Массив.ВГраница.md), то никаких действий не производится и не выдается сообщение "Индекс находится за границами массива".

Пример:

|  |
| --- |
| Массив.Удалить(4); |

Использование в версии:

Доступен, начиная с версии 8.0.
