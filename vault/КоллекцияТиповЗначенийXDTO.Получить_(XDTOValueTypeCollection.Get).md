КоллекцияТиповЗначенийXDTO (XDTOValueTypeCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента списка. Должен находиться в пределах количества элементов списка.

Возвращаемое значение:

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO_(XDTOValueType).md).   

Описание:

Получает элемент коллекции по индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
