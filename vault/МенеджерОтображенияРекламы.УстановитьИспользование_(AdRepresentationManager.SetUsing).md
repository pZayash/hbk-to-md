МенеджерОтображенияРекламы (AdRepresentationManager)

УстановитьИспользование (SetUsing)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УстановитьИспользование(<Использовать>, <ИдентификаторРекламногоБлока>)

Параметры:

<Использовать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Ложь](lang__def_BooleanFalse.md), компоненты отображения рекламы в интерфейсе будут отключены, вызов отображения полноэкранного баннера будет перенаправлен к обработчику завершения.  
Если [Истина](lang__def_BooleanTrue.md), то в параметре <ИдентификаторРекламногоБлока> требуется передать идентификатор.

<ИдентификаторРекламногоБлока> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор рекламного блока.

Описание:

Устанавливает признак отображения рекламы в приложении.  
Если параметр <Использовать> установлен в значение [Ложь](lang__def_BooleanFalse.md), то компоненты, отображающие рекламу в интерфейсе, будут отключены, а вызов метода для отображения полноэкранного баннера сразу же будет перенаправлен к его обработчику завершения.   
Если значение параметра <Использовать> установлено [Истина](lang__def_BooleanTrue.md), то в качестве второго параметра обязательно следует передать идентификатор рекламного блока.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
