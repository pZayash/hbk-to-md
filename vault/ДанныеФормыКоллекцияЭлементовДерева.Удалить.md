ДанныеФормыКоллекцияЭлементовДерева (FormDataTreeItemCollection)

Удалить (Delete)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента в коллекции.

Вариант синтаксиса: По элементу

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ДанныеФормыЭлементДерева](ДанныеФормыЭлементДерева.md).   
Удаляемый элемент.

Описание:

Удаляет элемент из коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
