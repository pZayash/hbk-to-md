ПланыОбменаМенеджер (ExchangePlansManager)

ПрочитатьОписаниеИзменений (ReadChangesDescription)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПрочитатьОписаниеИзменений(<ЧтениеСообщенияОбмена>)

Параметры:

<ЧтениеСообщенияОбмена> (обязательный)

Тип: [ЧтениеСообщенияОбмена](ЧтениеСообщенияОбмена.md).   
Объект, через который осуществляется чтение сообщения обмена данными. У этого объекта должен быть выполнен метод [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение.md), но еще не выполнен метод [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение.md).  
После вызова метода объект не может быть использован в вызове [ПрочитатьИзменения](ПланыОбменаМенеджер.ПрочитатьИзменения.md).

Возвращаемое значение:

Тип: [ОписаниеИзмененийКонфигурацииВСообщенииОбмена](ОписаниеИзмененийКонфигурацииВСообщенииОбмена.md).   

Описание:

Определяет, какие изменения конфигурации и расширений конфигурации произойдут при чтении сообщения обмена данными.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ПланыОбменаМенеджер](ПланыОбменаМенеджер.md), метод [ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации](ПланыОбменаМенеджер.ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации.md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
