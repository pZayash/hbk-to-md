МенеджерРаботыСРечью (SpeechProcessingManager)

УдалитьМодельРаспознаванияДляИнформационнойБазы (DeleteInfoBaseSpeechToTextModel)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УдалитьМодельРаспознаванияДляИнформационнойБазы(<ИдентификаторМодели>)

Параметры:

<ИдентификаторМодели> (обязательный)

Тип: [ИдентификаторМоделиРаспознаванияРечи](ИдентификаторМоделиРаспознаванияРечи.md).   
Уникальный идентификатор модели распознавания речи.

Описание:

Выгружает из памяти модель по идентификатору, если такая модель была загружена в память. Удаляет запись о модели из информационной базы (с учетом установленных разделителей сеанса).  
  
Требует наличие у текущего пользователя права "Администрирование данных".  
Если модель используется в данный момент, то распознавания, использующие модель, завершаются.  
Внимание! В мобильной платформе работа с локальными моделями распознавания не поддерживается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.25.
