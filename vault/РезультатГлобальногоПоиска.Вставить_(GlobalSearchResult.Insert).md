**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Глобальный поиск](Глобальный_поиск.md) › [РезультатГлобальногоПоиска](РезультатГлобальногоПоиска_(GlobalSearchResult).md) › [Методы](РезультатГлобальногоПоиска__Методы.md)

РезультатГлобальногоПоиска (GlobalSearchResult)

Вставить (Insert)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: По значению

Синтаксис:

Вставить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Значение> (обязательный)

Тип: [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md).   
Вставляемое значение.

Вариант синтаксиса: По параметрам

Синтаксис:

Вставить(<Индекс>, <Значение>, <Представление>, <Картинка>, <ВидПоиска>, <Описание>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Значение> (необязательный)

Тип: Произвольный.   
Значение результата поиска.

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Представление результата поиска.

<Картинка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка результата поиска.

<ВидПоиска> (необязательный)

Тип: [СтандартныйВидГлобальногоПоиска](СтандартныйВидГлобальногоПоиска_(StandardGlobalSearchType).md), [Строка](lang__def_String.md).   
Вид поиска.

<Описание> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Описание результата поиска.

Возвращаемое значение:

Тип: [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md).   

Описание:

Вставляет элемент в заданную позицию коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
