СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяВидеозапись (VideoRecordingSupported)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоддерживаетсяВидеозапись(<ТипКамеры>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства_(DeviceCameraType).md).   
Определяет, имеет ли устройство конкретную камеру (на передней или задней панели), в зависимости от переданного параметра.  
Значение по умолчанию: Авто.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в качестве типа камеры устройства используется значение [Авто](ТипКамерыУстройства.Авто_(DeviceCameraType.Auto).md), метод возвращает [Истина](lang__def_BooleanTrue.md), в следующих случаях:

- на Android - при наличии камеры,
- на iOS - если система поддерживает видеозапись,
- на Windows Phone - при наличии камеры.

Если в качестве типа камеры устройства используется значение [Передняя](ТипКамерыУстройства.Передняя_(DeviceCameraType.Front).md) или [Задняя](ТипКамерыУстройства.Задняя_(DeviceCameraType.Rear).md), метод возвращает [Истина](lang__def_BooleanTrue.md), в следующих случаях:

- на Android - при наличии требуемой камеры,
- на iOS - если система поддерживает видеозапись и есть требуемая камера,
- на Windows Phone - при наличии требуемой камеры.

Описание:

Определяет, поддерживается ли видеосъемка системой.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Не поддерживается в мобильном приложении для Windows Desktop.
