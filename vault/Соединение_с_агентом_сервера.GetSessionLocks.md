Соединение с агентом сервера (IServerAgentConnection)

GetSessionLocks (GetSessionLocks)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessionLocks(<Кластер>, <Сеанс>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, из которого должны быть получены блокировки. Должен быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера.md).

<Сеанс> (обязательный)

Тип: [Сеанс](Сеанс.md).   
Сеанс, от имени которого установлены блокировки объектов. Должен быть получен методом [GetSessions](Соединение_с_агентом_сервера.GetSessions.md) или [GetInfoBaseSessions](Соединение_с_агентом_сервера.GetInfoBaseSessions.md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера.md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив элементов [Блокировка](Блокировка.md).

Описание:

Получает объектные блокировки кластера, установленные от имени заданного сеанса.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
