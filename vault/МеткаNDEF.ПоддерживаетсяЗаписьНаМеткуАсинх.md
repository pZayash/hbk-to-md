МеткаNDEF (TagNDEF)

ПоддерживаетсяЗаписьНаМеткуАсинх (WriteOnTagSupportedAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяЗаписьНаМеткуАсинх()

Возвращаемое значение:

Тип: Булево.   
Истина - физическая метка перезаписываемя и запись на метку не заблокирована,  
Ложь - в противном случае.

Описание:

Проверяет, что физическая NFC-метка доступна для записи. Если произошла ошибка, генерируется исключение категории [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МенеджерМетокNDEF](МенеджерМетокNDEF.md), метод [ПоддерживаетсяЗаписьНаМетку](МенеджерМетокNDEF.ПоддерживаетсяЗаписьНаМетку.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
