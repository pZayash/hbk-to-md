Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для журнала документов. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ЖурналДокументовСписок.](ЖурналДокументовСписок.Имя_журнала_документов.md)<[Имя журнала документов](ЖурналДокументовСписок.Имя_журнала_документов.md)>.

Доступность:

Толстый клиент.
