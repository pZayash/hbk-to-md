ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

ДляВсехПользователей (ForAllUsers)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет размещением файла программной лицензии:

- [Истина](lang__def_BooleanTrue.md) - файл программной лицензии будет записан для всех пользователей. Обычно используются каталоги:
  - Windows - "C:\ProgramData\1C\licenses"
  - Linux - "/var/1C/licenses".
- [Ложь](lang__def_BooleanFalse.md) - файл будет записан в каталог поиска лицензии текущего пользователя. Обычно используются каталоги:
  - Windows - "C:\Users\"Имя пользователя"\AppData\Local\1C\1cv8\conf",
  - Linux - "~\1C\1cv8\conf".

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
