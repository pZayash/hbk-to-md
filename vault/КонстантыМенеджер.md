# КонстантыМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Константы](Константы-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КонстантаМенеджер.\<Имя константы\>](КонстантаМенеджер.Имя_константы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры констант.  
Возможно обращение к менеджеру константы посредством оператора [...]. В качестве аргумента передается имя константы, как оно задано в конфигурации.  

Свойства:

[\<Имя константы\> (\<Constant name\>)](КонстантыМенеджер.Имя_константы.md)  

Методы:

[ПолучитьФорму (GetForm)](КонстантыМенеджер.ПолучитьФорму.md)  
[СоздатьНабор (CreateSet)](КонстантыМенеджер.СоздатьНабор.md)  

Описание:

Применяется в виде свойства [Константы](Глобальный_контекст.Константы.md) глобального контекста для предоставления доступа к константам.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| Константы.ПолучитьФорму().ОткрытьМодально(); |
