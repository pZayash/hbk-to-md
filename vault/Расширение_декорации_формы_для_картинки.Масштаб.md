Расширение декорации формы для картинки (Form decoration extension for a picture)

Масштаб (Scale)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет дополнительный (к масштабу формы) масштаб картинки.  
Значение по умолчанию: 100.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
