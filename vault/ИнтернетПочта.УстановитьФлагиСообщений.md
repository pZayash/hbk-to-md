ИнтернетПочта (InternetMail)

УстановитьФлагиСообщений (SetMessagesFlags)

Доступен, начиная с версии 8.3.12.

Синтаксис:

УстановитьФлагиСообщений(<Флаги>)

Параметры:

<Флаги> (обязательный)

Тип: [Соответствие](Соответствие.md).   
Данный параметр позволяет передать флаги почтовых сообщений.   
Ключом соответствия является серверный идентификатор сообщения, а значением - объект [ФлагиИнтернетПочтовогоСообщения](ФлагиИнтернетПочтовогоСообщения.md) .  
На почтовом сервере флаги сообщения устанавливаются в зависимости от значения свойства объекта [ФлагиИнтернетПочтовогоСообщения](ФлагиИнтернетПочтовогоСообщения.md) , соответствующего IMAP-флагу:

- Если свойство установлено в значение Истина, то на сервере флаг устанавливается.
- Если свойство установлено в значение Ложь, то на сервере флаг снимается.
- Если свойство установлено в значение Неопределено, то на сервере флаг не изменяется.

Если почтовый сервер не поддерживает какой-либо флаг, такой флаг игнорируется.

Описание:

Устанавливает флаги почтовых сообщений на сервере. Метод доступен только при подключении по протоколу IMAP.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.12.
