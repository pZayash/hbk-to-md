ЭлементРазметкаHTML (HTMLHtmlElement)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Текстовое содержимое узла HTML.  
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.  
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство [ЭтоПробельныеСимволыЭлемента](ТекстHTML.ЭтоПробельныеСимволыЭлемента.md)).  
Принцип формирования текстового значения узла:  
[Элемент](ТипУзлаDOM.Элемент.md), [Атрибут](ТипУзлаDOM.Атрибут.md), [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - объединение текстового содержимого каждого из дочерних узлов, исключая [Комментарий](ТипУзлаDOM.Комментарий.md). Пустая строка, если дочерних узлов нет;  
[Текст](ТипУзлаDOM.Текст.md), [Комментарий](ТипУзлаDOM.Комментарий.md) - значение свойства ЗначениеУзла;  
[Документ](ТипУзлаDOM.Документ.md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) - Неопределено.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
