# ТабличныйДокумент.НайтиЗначение

ТабличныйДокумент (SpreadsheetDocument)

НайтиЗначение (FindValue)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиЗначение(<Значение>, <Начало>, <Где>, <ИскатьПоСтрокам>, <ИскатьВперед>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Искомое значение.

<Начало> (необязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md).   
Область, после которой начинать поиск. Если параметр не задан, то поиск с начала табличного документа.

<Где> (необязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md).   
Область, в которой осуществляется поиск. Если параметр не указан, поиск будет выполнен по всем ячейкам табличного документа.

<ИскатьПоСтрокам> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет последовательность обхода ячеек при поиске.  
[Истина](lang__def_BooleanTrue.md) - искать по текущей строке, затем переходить на следующую и т.д. в общем направлении поиска, [Ложь](lang__def_BooleanFalse.md) - искать по текущей колонке, затем переходить на следующую и т.д. в общем направлении поиска.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ИскатьВперед> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет направление поиска: [Истина](lang__def_BooleanTrue.md) - искать вперед; [Ложь](lang__def_BooleanFalse.md) - искать назад.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md), [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md), [Неопределено](lang__def_Undefined.md).   
Если значение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск значения в ячейках табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Область = ТабДок.НайтиЗначение(ВыбЗначение); Если ТипЗнч(Область)=Тип("Неопределено") Тогда     Сообщить("Значение не найдено!"); Иначе     Сообщить("Значение найдено."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.1.
