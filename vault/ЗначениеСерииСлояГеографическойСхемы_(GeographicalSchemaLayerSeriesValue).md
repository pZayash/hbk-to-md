Описание:

Предоставляет доступ к свойствам значения серии слоя географической схемы. Значение серии слоя характеризует некий топологический объект.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Получим значение объекта для серии названия  ЗначениеСерии = ОбъектКарты.Слой.ПолучитьЗначение(ОбъектКарты,                          СерияНазваний); // Если есть такое значение  Если Не ЗначениеСерии = Неопределено Тогда     // Получим реальное название      НазваниеМестоположения = ЗначениеСерии.Значение; КонецЕсли; |

См. также:

[СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md), метод [ПолучитьЗначение](СлойГеографическойСхемы.ПолучитьЗначение_(GeographicalSchemaLayer.GetValue).md)

Доступен, начиная с версии 8.1.
