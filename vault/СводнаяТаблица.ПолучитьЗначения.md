СводнаяТаблица (PivotTable)

ПолучитьЗначения (GetValues)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначения(<Область>)

Параметры:

<Область> (обязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Область, для которой определяются значения.

Возвращаемое значение:

Тип: [Структура](Структура.md), Неопределено.   
В случае, если переданная область не принадлежит сводной таблице, возвращается значение Неопределено, в противном случае - [Структура](Структура.md). Ключом в структуре является имя поля, значением - значение данного поля. В зависимости от положения области в сводной таблице, структура может содержать различные поля.

- В случае, если переданная область находится на пересечении строк и колонок, структура будет содержать имена измерений строк, измерений колонок, на пересечении которых расположена область, и значения всех ресурсов.
- В случае, если область расположена в области заголовков строк или области итогов строк, структура будет содержать имена измерений строк, и значения ресурсов.
- В случае, если область расположена в области заголовков колонок или области итогов колонок, структура будет содержать имена измерений колонок и значения ресурсов.
- В случае, если область будет расположена в области общих итогов, будут возвращены только значения ресурсов.

Описание:

Получает значения всех полей для области табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Независимо от того, какая область будет передана в параметре, "обработана" будет только левая верхняя ячейка этой области.

Использование в версии:

Доступен, начиная с версии 8.0.
