Элементы коллекции:

[ЗначениеПараметраМакетаКомпоновкиДанных](ЗначениеПараметраМакетаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметра.  
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
