XBase (XBase)

АвтоСохранение (AutoSave)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического сохранения изменений в базе данных.  
[Истина](lang__def_BooleanTrue.md) - выполняется автоматическое сохранение изменений.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
