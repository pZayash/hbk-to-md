# Соединение с агентом сервера.DeleteServiceSetting

Соединение с агентом сервера (IServerAgentConnection)

DeleteServiceSetting (DeleteServiceSetting)

Доступен, начиная с версии 8.3.26.

Синтаксис:

DeleteServiceSetting(<Кластер>, <Сервер>, <Настройка>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого требуется удалить настройку.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   
Рабочий сервер, на котором требуется удалить настройку.

<Настройка> (обязательный)

Тип: [Настройка сервиса](Настройка_сервиса_(IServiceSetting).md).   
Настройка сервиса кластера.

Описание:

Удаляет настройку сервиса кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
