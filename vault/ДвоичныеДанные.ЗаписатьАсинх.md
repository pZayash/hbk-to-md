ДвоичныеДанные (BinaryData)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который требуется записать двоичные данные.

Описание варианта метода:

Записать двоичные данные в файл с заданным именем.

Вариант синтаксиса: Запись в файл

Синтаксис:

ЗаписатьАсинх()

Описание варианта метода:

Записывает двоичные данные в файл с указанным именем.

Вариант синтаксиса: Запись в поток

Синтаксис:

ЗаписатьАсинх(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md).   
Поток, в который требуется записать двоичные данные.

Описание варианта метода:

Записывает двоичные данные в заданный поток.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Записывает двоичные данные в файл или поток.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер).

Доступен, начиная с версии 8.3.18.
