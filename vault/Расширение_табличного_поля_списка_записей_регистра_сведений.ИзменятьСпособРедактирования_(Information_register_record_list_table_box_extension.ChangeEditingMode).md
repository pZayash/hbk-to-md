Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

ИзменятьСпособРедактирования (ChangeEditingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает доступность интерактивного способа редактирования объекта (в строке или в отдельной форме).

Доступность:

Толстый клиент.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - интерактивная установка способа редактирования объекта не доступна пользователю.

Использование в версии:

Доступен, начиная с версии 8.0.
