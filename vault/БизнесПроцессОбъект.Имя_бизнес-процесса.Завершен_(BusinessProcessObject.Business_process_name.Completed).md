# БизнесПроцессОбъект.<Имя бизнес-процесса>.Завершен

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [БизнесПроцессОбъект.<Имя бизнес-процесса>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md) › [Свойства](БизнесПроцессОбъект.Имя_бизнес-процесса__Свойства.md)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Завершен (Completed)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак завершения бизнес-процесса. Этот признак устанавливается системой автоматически при переходе бизнес-процесса на точку завершения при условии отсутствия невыполненных задач.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - бизнес-процесс завершен; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.0.
