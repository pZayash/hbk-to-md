ИспользованиеСобытияЖурналаРегистрации (EventLogEventUse)

На основании значения свойства Использование

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ИспользованиеСобытияЖурналаРегистрации(<Использование>, <ОписаниеИспользования>)

Параметры:

<Использование> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет значение свойства [Использование](ИспользованиеСобытияЖурналаРегистрации.Использование_(EventLogEventUse.Use).md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ОписаниеИспользования> (необязательный)

Тип: Произвольный.   
Определяет значение свойства [ОписаниеИспользования](ИспользованиеСобытияЖурналаРегистрации.ОписаниеИспользования_(EventLogEventUse.UseDescription).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Создает объект [ИспользованиеСобытияЖурналаРегистрации](ИспользованиеСобытияЖурналаРегистрации_(EventLogEventUse).md).

Использование в версии:

Доступен, начиная с версии 8.2.
