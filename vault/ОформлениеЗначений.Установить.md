ОформлениеЗначений (ValueAppearance)

Установить (Set)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Установить(<Ключ>, <Значение>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ оформления.

<Значение> (необязательный)

Тип: Произвольный.   
Значение оформления.

Возвращаемое значение:

Тип: Произвольный.   
Значение оформления.

Описание:

Добавляет новую запись по ключу.  
Если такой ключ уже есть, то заменяет значение.   
Если значение не передано и такого ключа нет, то автоматически создает индекс, выбирая свободный в разрезе типа переданного ключа.  
Если значение не передано, и такой ключ есть, то не выполняет никаких действий.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
