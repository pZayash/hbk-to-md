# МенеджерДополнительнойПроверкиПользователя.НачатьПроверку

МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

НачатьПроверку (BeginVerification)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьПроверку(<Способ>, <СообщениеПользователю>, <ОбработчикРезультатовПроверки>)

Параметры:

<Способ> (обязательный)

Тип: [СпособДополнительнойПроверкиПользователя](СпособДополнительнойПроверкиПользователя_(AdditionalUserVerificationMethod).md).   
Способ проверки пользователя.

<СообщениеПользователю> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст сообщения, отображаемый пользователю при проверке доступа.

<ОбработчикРезультатовПроверки> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после выполнения проверки пользователя со следующими параметрами:

- <ОтмененоПользователем> - тип [Булево](lang__Булево_(Boolean).md). Признак того, что проверка пользователя была отменена пользователем.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

При возникновении ошибки вызвается обработчик ошибки.

Описание:

Начинает проверку дополнительную проверку пользователя указанным способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерДополнительнойПроверкиПользователя](МенеджерДополнительнойПроверкиПользователя_(AdditionalUserVerificationManager).md), метод [ПроверитьАсинх](МенеджерДополнительнойПроверкиПользователя.ПроверитьАсинх_(AdditionalUserVerificationManager.VerifyAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
