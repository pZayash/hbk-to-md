ЗаписьУзловDOM (DOMNodeWriter)

ЗаписатьСекциюCDATA (WriteCDATASection)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьСекциюCDATA(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: Строка.   
Символьные данные - содержимое секции CDATA.

Описание:

Осуществляет запись секции CDATA.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
