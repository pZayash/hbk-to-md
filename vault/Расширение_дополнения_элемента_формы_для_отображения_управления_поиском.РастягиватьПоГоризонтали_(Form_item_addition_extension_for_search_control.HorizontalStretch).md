Расширение дополнения элемента формы для отображения управления поиском (Form item addition extension for search control)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак растягивания элемента по горизонтали.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
