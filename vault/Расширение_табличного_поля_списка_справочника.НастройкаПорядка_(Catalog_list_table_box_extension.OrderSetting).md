# Расширение табличного поля списка справочника.НастройкаПорядка

Расширение табличного поля списка справочника (Catalog list table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаПорядка](НастройкаПорядка_(OrderSetting).md).   
Предоставляет доступ к настройкам порядка списка элементов справочника. Представляет собой коллекцию значений типа [ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md). Свойства коллекции содержат объекты для управления элементом порядка. Имена свойств совпадают с именами доступных для справочника элементов упорядочивания.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
