Лицензия (ILicenseInfo)

LicenseType (LicenseType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит тип лицензии:

- 0 - программная лицензия платформы;
- 1 - аппаратная лицензия (ключ защиты программы).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
