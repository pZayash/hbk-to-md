РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Заполнение (FillType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипЗаполненияОбластиТабличногоДокумента](ТипЗаполненияОбластиТабличногоДокумента_(SpreadsheetDocumentAreaFillType).md).   
Определяет тип заполнения содержимого рисунка при выводе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунка типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md).

Использование в версии:

Доступен, начиная с версии 8.0.
