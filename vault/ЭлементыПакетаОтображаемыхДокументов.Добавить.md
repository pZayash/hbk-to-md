ЭлементыПакетаОтображаемыхДокументов (RepresentableDocumentBatchItems)

Добавить (Add)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Добавить(<АдресДокумента>)

Параметры:

<АдресДокумента> (необязательный)

Тип: Строка.   
Содержит адрес временного хранилища, в котором расположен документ.

Возвращаемое значение:

Тип: [ЭлементПакетаОтображаемыхДокументов](ЭлементПакетаОтображаемыхДокументов.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
