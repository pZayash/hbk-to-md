ДанныеАдреса (AddressData)

По умолчанию

Доступен, начиная с версии 8.3.3.

Синтаксис:

Новый ДанныеАдреса(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [Структура](Структура.md).   
Содержит данные адреса в виде структуры.  
Ключи структуры должны совпадать с именами свойств объекта:

- Страна (Country) тип Строка;
- Регион (State) - тип Строка;
- Город (City) - тип Строка;
- Улица (Street) - тип Строка;
- Дом (House) - тип Строка;
- Индекс (ZipCode) - тип Строка;
- Представление (Presentation) - представление адреса в виде строки; тип Строка.

Также можно передать нестандартные поля с помощью соответствия в паре с ключом ИсходныеДанные (RawData) тип [Соответствие](Соответствие.md).  
Если ключи соответствия с нестандартными полями совпадают со стандартными полями, то будут использоваться значения стандартных полей. Например, если в поле Страна передано значение Россия, а в поле ИсходныеДанные передано соответствие с записью Country=Китай, то результирующим значением будет Россия.

Описание:

Формирует адрес из данных в виде соответствия.

Использование в версии:

Доступен, начиная с версии 8.3.3.
