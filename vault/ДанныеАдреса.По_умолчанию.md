ДанныеАдреса (AddressData)

По умолчанию

Доступен, начиная с версии 8.3.3.

Синтаксис:

Новый ДанныеАдреса(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит данные адреса в виде структуры.  
Ключи структуры должны совпадать с именами свойств объекта:

- Страна (Country) тип [Строка](lang__def_String.md);
- Регион (State) - тип [Строка](lang__def_String.md);
- Город (City) - тип [Строка](lang__def_String.md);
- Улица (Street) - тип [Строка](lang__def_String.md);
- Дом (House) - тип [Строка](lang__def_String.md);
- Индекс (ZipCode) - тип [Строка](lang__def_String.md);
- Представление (Presentation) - представление адреса в виде строки; тип [Строка](lang__def_String.md).

Также можно передать нестандартные поля с помощью соответствия в паре с ключом ИсходныеДанные (RawData) тип [Соответствие](Соответствие_(Map).md).  
Если ключи соответствия с нестандартными полями совпадают со стандартными полями, то будут использоваться значения стандартных полей. Например, если в поле Страна передано значение Россия, а в поле ИсходныеДанные передано соответствие с записью Country=Китай, то результирующим значением будет Россия.

Описание:

Формирует адрес из данных в виде соответствия.

Использование в версии:

Доступен, начиная с версии 8.3.3.
