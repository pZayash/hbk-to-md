ТестируемоеПолеФормы (TestedFormField)

ПолучитьПредставлениеДанных (GetDataPresentation)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьПредставлениеДанных()

Возвращаемое значение:

Тип: Строка.   
Строка с текстовым представлением отображаемых объектом данных.

Описание:

Получает текстовое представление данных, связанных с объектом, если применимо.  
Если метод вызывается для поля, являющегося колонкой таблицы, которая не находится в режиме редактирования, будет сгенерирована ошибка.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
