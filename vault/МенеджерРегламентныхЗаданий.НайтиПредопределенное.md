МенеджерРегламентныхЗаданий (ScheduledJobsManager)

НайтиПредопределенное (FindPredefined)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПредопределенное(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: Строка, [ОбъектМетаданных: РегламентноеЗадание](ОбъектМетаданных_РегламентноеЗадание.md).   
Имя или объект метаданных.

Возвращаемое значение:

Тип: [РегламентноеЗадание](РегламентноеЗадание.md).   

Описание:

Осуществляет поиск предопределенного регламентного задания по его метаданным. Получение возможно только для администратора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
