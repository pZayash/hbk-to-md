Планировщик (Planner)

ОтображениеВремениЭлементов (ItemsTimeRepresentation)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеВремениЭлементовПланировщика](ОтображениеВремениЭлементовПланировщика_(PlannerItemsTimeRepresentation).md).   
Устанавливает режим отображения времени на элементах планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
