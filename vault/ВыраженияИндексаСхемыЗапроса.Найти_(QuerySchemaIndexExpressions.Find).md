ВыраженияИндексаСхемыЗапроса (QuerySchemaIndexExpressions)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение для поиска.

Возвращаемое значение:

Тип: [ВыражениеИндексаСхемыЗапроса](ВыражениеИндексаСхемыЗапроса_(QuerySchemaIndexExpression).md).   

Описание:

Выполняет поиск по свойству [Выражение](ВыражениеИндексаСхемыЗапроса.Выражение_(QuerySchemaIndexExpression.Expression).md). Во время поиска форматирование игнорируется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
