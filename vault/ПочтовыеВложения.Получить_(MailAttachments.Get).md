ПочтовыеВложения (MailAttachments)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<ИндексЭлементаКоллекции>)

Параметры:

<ИндексЭлементаКоллекции> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента коллекции.

Возвращаемое значение:

Тип: [ПочтовоеВложение](ПочтовоеВложение_(MailAttachment).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
