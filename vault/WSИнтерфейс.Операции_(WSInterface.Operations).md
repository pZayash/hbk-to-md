WSИнтерфейс (WSInterface)

Операции (Operations)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [WSКоллекцияОпераций](WSКоллекцияОпераций_(WSOperationCollection).md).   
Содержит операции интерфейса веб-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
