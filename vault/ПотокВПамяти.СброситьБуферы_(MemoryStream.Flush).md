ПотокВПамяти (MemoryStream)

СброситьБуферы (Flush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

СброситьБуферы()

Описание:

Сбрасывает все промежуточные буферы и производит запись всех незаписанных данных в целевое устройство.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

[Поток](Поток_(Stream).md), метод [НачатьСбросБуферов](Поток.НачатьСбросБуферов_(Stream.BeginFlush).md)  
[Поток](Поток_(Stream).md), метод [СброситьБуферыАсинх](Поток.СброситьБуферыАсинх_(Stream.FlushAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
