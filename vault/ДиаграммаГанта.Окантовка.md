ДиаграммаГанта (GanttChart)

Окантовка (Outline)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак окантовки черным цветом интервалов на диаграмме Ганта. Вне зависимости от цвета интервалов окантовка наносится тонкой черной линией.  
Истина - окантовка используется.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
