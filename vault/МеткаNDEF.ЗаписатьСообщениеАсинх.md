МеткаNDEF (TagNDEF)

ЗаписатьСообщениеАсинх (WriteMessageAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗаписатьСообщениеАсинх(<Сообщение>)

Параметры:

<Сообщение> (обязательный)

Тип: [СообщениеNDEF](СообщениеNDEF.md).   
Полезная нагрузка (фактические данные метки).

Описание:

Записывает сообщение NDEF на физическую метку.  
Перед записью рекомендуется проверить, доступна ли запись с помощью метода [ПоддерживаетсяЗаписьНаМеткуАсинх](МеткаNDEF.ПоддерживаетсяЗаписьНаМеткуАсинх.md).  
Если произошла ошибка записи, то будет сгенерировано исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерМетокNDEF](МенеджерМетокNDEF.md), метод [ПоддерживаетсяЗаписьНаМетку](МенеджерМетокNDEF.ПоддерживаетсяЗаписьНаМетку.md)  
[МеткаNDEF](МеткаNDEF.md), метод [ПоддерживаетсяЗаписьНаМеткуАсинх](МеткаNDEF.ПоддерживаетсяЗаписьНаМеткуАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
