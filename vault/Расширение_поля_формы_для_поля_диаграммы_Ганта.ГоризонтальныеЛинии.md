Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ГоризонтальныеЛинии (HorizontalLines)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Управляет отображением вертикальных линий в области построения диаграммы.   
Если содержит Ложь, то все вертикальные линии в области построения диаграммы не отображаются.   
Если содержит Истина, то все вертикальные линии в области построения диаграммы отображаются.   
Если содержит Неопределено, то все вертикальные линии в области текстов точек и области таблицы данных не отображаются, а в области значений и интервалов – отображаются.   
Значение по умолчанию - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
