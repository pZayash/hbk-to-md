ЗаписьДанных (DataWriter)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Инициирует вызов метода [СброситьБуферы](ЗаписьДанных.СброситьБуферы_(DataWriter.Flush).md). Если целевой поток был создан при создании объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md), также вызывает начало закрытие целевого потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [Закрыть](ЗаписьДанных.Закрыть_(DataWriter.Close).md)  
[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [ЗакрытьАсинх](ЗаписьДанных.ЗакрытьАсинх_(DataWriter.CloseAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
