# ТипНаправленияПереходаТабличногоДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Табличный документ](Табличный_документ.md)

Доступен, начиная с версии 8.0.

Значения

[БезПерехода (WithoutMove)](ТипНаправленияПереходаТабличногоДокумента.БезПерехода_(SpreadsheetDocumentStepDirectionType.WithoutMove).md)  
[ПоКолонкам (ByColumns)](ТипНаправленияПереходаТабличногоДокумента.ПоКолонкам_(SpreadsheetDocumentStepDirectionType.ByColumns).md)

Описание:

Содержит типы перехода по ячейкам после завершения редактирования ячейки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/spreadsheet}. Имя типа XDTO: SpreadsheetDocumentStepDirectionType.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), свойство [НаправлениеПерехода](ТабличныйДокумент.НаправлениеПерехода_(SpreadsheetDocument.StepDirection).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
