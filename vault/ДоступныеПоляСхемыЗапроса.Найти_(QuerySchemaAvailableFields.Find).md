ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля для поиска.

Возвращаемое значение:

Тип: [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса_(QuerySchemaAvailableField).md), [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса_(QuerySchemaAvailableNestedTable).md).   

Описание:

Выполняет поиск по свойству Имя объектов [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса_(QuerySchemaAvailableField).md) и [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса_(QuerySchemaAvailableNestedTable).md). Поиск выполняется рекурсивно, с обходом доступных полей доступной вложенной таблицы и полей доступного поля. В качестве искомой строки допустимо указать имя поля как с именем таблицы, так и без него.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
