Расширение поля формы для поля ввода (Form field extension for a text box)

ДобавлениеМножественныхЗначений (MultipleValuesAdd)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ДобавлениеМножественныхЗначений(<Значения>, <СтандартнаяОбработка>)

Параметры:

<Значения>

Тип: [Массив](Массив_(Array).md).   
Массив значений, по которым будут сформированы или объекты [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md) и добавлены в конец списка значений, или объекты [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md) и добавлены в конец таблицы значений, если параметр <СтандартнаяОбработка> содержит значение [Истина](lang__def_BooleanTrue.md).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при интерактивном добавлении множественных значений после событий [ОкончаниеВводаТекста](ПолеВвода.ОкончаниеВводаТекста_(TextBox.TextEditEnd).md) и [ОбработкаВыбора](Расширение_поля_формы_для_поля_ввода.ОбработкаВыбора_(Form_field_extension_for_a_text_box.ChoiceProcessing).md), но до вызова обработчика [ПриИзменении](ПолеВвода.ПриИзменении_(TextBox.OnChange).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
