# СправочникОбъект.<Имя справочника>.ПриУстановкеНовогоКода

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПриУстановкеНовогоКода (OnSetNewCode)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриУстановкеНовогоКода(<СтандартнаяОбработка>, <Префикс>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная генерация нового кода производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Префикс>

Тип: [Строка](lang__def_String.md).   
Префикс, который будет использоваться для генерации кода. Может быть изменен в процедуре-обработчике события.

Описание:

Возникает в момент, когда выполняется установка нового кода элемента справочника.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Параметр <Префикс> может быть указан, если код имеет тип [Строка](lang__def_String.md).

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
