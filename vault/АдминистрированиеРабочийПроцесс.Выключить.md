АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Выключить (TurnOff)

Доступен, начиная с версии 8.3.27.

Синтаксис:

Выключить()

Описание:

Останавливает рабочий процесс. Процесс будет остановлен после отключения всех пользователей или после истечения времени, заданного свойством [ВремяПринудительногоЗавершения](АдминистрированиеКластер.ВремяПринудительногоЗавершения.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
