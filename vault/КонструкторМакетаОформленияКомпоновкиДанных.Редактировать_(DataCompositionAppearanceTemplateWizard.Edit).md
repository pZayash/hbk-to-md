КонструкторМакетаОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateWizard)

Редактировать (Edit)

Доступен, начиная с версии 8.1.

Синтаксис:

Редактировать(<Форма>)

Параметры:

<Форма> (необязательный)

Тип: [Форма](Форма_(Form).md).   
Форма-владелец.

Описание:

Открывает окно конструктора макета оформления компоновки данных.  
При завершении редактирования и нажатии на кнопку OK конструктор вызывает событие "Обработка выбора" у формы, которая была назначена в качестве формы владельца вызванного конструктора.

Доступность:

Толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
