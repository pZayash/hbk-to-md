ПолеНабораДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetField)

ОграничениеИспользованияРеквизитов (AttributeUseRestriction)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОграничениеИспользованияПоляСхемыКомпоновкиДанных](ОграничениеИспользованияПоляСхемыКомпоновкиДанных.md).   
Указывает ограничение использования данного поля в пользовательских настройках. Ограничению подлежат те же виды использования, что и для ограничения использования поля.   
Поведение данного свойства аналогично свойству [ОграничениеИспользования](ПолеНабораДанныхСхемыКомпоновкиДанных.ОграничениеИспользования.md).  
Применяется для полей объектного типа. При этом выбрать поле возможно, но нельзя выбрать его реквизиты.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
