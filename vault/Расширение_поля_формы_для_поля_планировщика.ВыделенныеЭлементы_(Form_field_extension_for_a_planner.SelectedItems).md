Расширение поля формы для поля планировщика (Form field extension for a planner)

ВыделенныеЭлементы (SelectedItems)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Содержит элементы планировщика, выделенные на поле.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
