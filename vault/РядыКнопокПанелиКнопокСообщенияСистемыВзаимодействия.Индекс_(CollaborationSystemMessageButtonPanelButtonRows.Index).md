РядыКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRows)

Индекс (Index)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Индекс(<РядКнопок>)

Параметры:

<РядКнопок> (обязательный)

Тип: [РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonRow).md).   
Ряд кнопок панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс указанного ряда кнопок в коллекции рядов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
