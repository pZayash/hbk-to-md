# ДокументОбъект.<Имя документа>.УстановитьСсылкуНового

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

УстановитьСсылкуНового (SetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСсылкуНового(<Ссылка>)

Параметры:

<Ссылка> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Ссылка, которая будет назначена при записи нового объекта.

Описание:

Устанавливает значение для нового (созданного и еще не записанного) объекта, которое будет назначено при записи в качестве ссылки. Значение не может равняться ссылке какого-либо из имеющихся в базе данных объекта данного типа. Уникальность ссылки проверяется при записи объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ДокументМенеджер.\<Имя документа\>](ДокументМенеджер.Имя_документа_(DocumentManager.Document_name).md), метод [ПолучитьСсылку](ДокументМенеджер.Имя_документа.ПолучитьСсылку_(DocumentManager.Document_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
