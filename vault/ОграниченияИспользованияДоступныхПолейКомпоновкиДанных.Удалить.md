ОграниченияИспользованияДоступныхПолейКомпоновкиДанных (DataCompositionAvailableFieldUseRestrictions)

Удалить (Delete)

Доступен, начиная с версии 8.1.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ОграничениеИспользованияПоляСхемыКомпоновкиДанных](ОграничениеИспользованияПоляСхемыКомпоновкиДанных.md), [ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных.md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных.md).   
Удаляемый элемент.  
В случае, если параметром метода является доступное поля или доступное поле отбора, из ограничений будут удалены все элементы, ограничивающие поле, указанное в свойстве данного доступного поля, и его синонимы.

Описание:

Удаляет элемент из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
