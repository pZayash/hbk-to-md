ОписаниеПодписиPDF (PDFSignatureDescription)

НомерСтраницы (PageNumber)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер страницы документа, на которой будет расположена электронная подпись PDF-документа.   
Начинается с 1.  
Если при подписании не указан, то подпись будет выведена на последнюю страницу. Если указан номер несуществующей страницы, то генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
