Глобальный контекст (Global context)

СредстваУстройства (DeviceTools)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [МенеджерСредствУстройства](МенеджерСредствУстройства.md).   
Предоставляет доступ к средствам устройства.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.13.
