ТипШтрихКода (BarcodeType)

Все (All)

Доступен, начиная с версии 8.3.5.

Описание:

При вызове сканирования система должна распознавать штрихкоды всех типов.  
Поддерживаются следующие типы штрихкодов:

- двумерные - QRCode, DataMatrix, PDF417, Aztec, MaxiCode.
- линейные - UPCA, UPCE, EAN13, EAN8, Codabar, Code93, Code128, RSS14, RSSExpanded, Code39, ITF.

Использование в версии:

Доступен, начиная с версии 8.3.5.
