Глобальный контекст (Global context)

XMLЗначение (XMLValue)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLЗначение(<Тип>, <СтрокаXML>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип, значение которого надо получить при преобразовании из строкового представления XML.

<СтрокаXML> (обязательный)

Тип: Строка.   
Строка, содержащая строковое представление значения, полученное из XML.

Возвращаемое значение:

Тип: Null, Булево, Число, Строка, Дата, [УникальныйИдентификатор](УникальныйИдентификатор.md), [ДвоичныеДанные](ДвоичныеДанные.md), [ХранилищеЗначения](ХранилищеЗначения.md); значения перечислений, значения системных перечислений ([ВидСчета](ВидСчета.md), [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии.md), [ВидДвиженияНакопления](ВидДвиженияНакопления.md), [ДопустимыйЗнак](ДопустимыйЗнак.md), [ДопустимаяДлина](ДопустимаяДлина.md), [ЧастиДаты](ЧастиДаты.md), Неопределено или все ссылки на объекты базы данных.   

Описание:

Выполняет преобразование из строки, полученной из текста элемента или значения атрибута XML, в значение в соответствии с указанным типом. Выполняет действие обратное действию метода [XMLСтрока](Глобальный_контекст.XMLСтрока.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
