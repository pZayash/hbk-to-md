# Расширение табличного поля списка задач.ПериодАвтоОбновления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [Расширение табличного поля списка задач](Расширение_табличного_поля_списка_задач_(Task_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_задач__Свойства.md)

Расширение табличного поля списка задач (Task list table box extension)

ПериодАвтоОбновления (AutoRefreshPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Задает период (в секундах), с которым будет производиться автоматическое обновление списка. Не может быть меньше 5 секунд. Если свойство [АвтоОбновление](Расширение_табличного_поля_списка_задач.АвтоОбновление_(Task_list_table_box_extension.AutoRefresh).md) установлено как [Ложь](lang__def_BooleanFalse.md), то обновление не производится.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
