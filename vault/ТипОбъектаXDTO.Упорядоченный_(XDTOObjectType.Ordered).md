# ТипОбъектаXDTO.Упорядоченный

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md) › [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md) › [Свойства](ТипОбъектаXDTO__Свойства.md)

ТипОбъектаXDTO (XDTOObjectType)

Упорядоченный (Ordered)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак упорядоченности свойств объекта данного типа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - порядок следования свойств в объекте должен соответствовать порядку следования свойств в типе объекта. В противном случае порядок следования свойств может быть произвольным.

Использование в версии:

Доступен, начиная с версии 8.1.
