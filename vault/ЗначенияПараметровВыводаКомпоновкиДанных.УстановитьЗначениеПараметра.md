ЗначенияПараметровВыводаКомпоновкиДанных (DataCompositionOutputParameterValues)

УстановитьЗначениеПараметра (SetParameterValue)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьЗначениеПараметра(<Параметр>, <Значение>)

Параметры:

<Параметр> (обязательный)

Тип: Строка, [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных.md).   
Имя параметра или параметр компоновки данных, значение которого нужно установить.

<Значение> (обязательный)

Значение, которое нужно установить.

Описание:

Устанавливает значение параметра и включает свойство [Использование](ЗначениеПараметраКомпоновкиДанных.Использование.md). Если параметр с указанным именем не найден, будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
