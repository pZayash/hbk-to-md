БизнесПроцессСписок.<Имя бизнес-процесса> (BusinessProcessList.<Business process name>)

Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Универсальные_объекты.Порядок.md).   
Содержит объект [Порядок](Универсальные_объекты.Порядок.md), по которому осуществляется текущее упорядочивание бизнес-процессов в списке. В качестве имен упорядочивания объекта могут выступать имена предопределенных реквизитов Дата и Номер и имена реквизитов бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
