# КолонкиОписанияИсточникаДанных

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаОписанияИсточникаДанных](КолонкаОписанияИсточникаДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонке описания посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Свойства:

[\<Имя колонки\> (\<Column name\>)](КолонкиОписанияИсточникаДанных.Имя_колонки.md)  

Методы:

[Индекс (IndexOf)](КолонкиОписанияИсточникаДанных.Индекс.md)  
[Количество (Count)](КолонкиОписанияИсточникаДанных.Количество.md)  
[Получить (Get)](КолонкиОписанияИсточникаДанных.Получить.md)  

Описание:

Содержит колонки описания источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ОписаниеИсточникаДанных](ОписаниеИсточникаДанных.md), свойство [Колонки](ОписаниеИсточникаДанных.Колонки.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
