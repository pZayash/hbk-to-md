ЧтениеHTML (HTMLReader)

Закрыть (Close)

Доступен, начиная с версии 8.2.

Синтаксис:

Закрыть()

Описание:

Прекращает чтение HTML из файла или строки. Если никакого чтения в момент вызова данного метода не выполнялось, то метод ничего не делает.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЧтениеXML.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.2.
