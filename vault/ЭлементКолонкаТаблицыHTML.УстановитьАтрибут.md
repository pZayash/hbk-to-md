ЭлементКолонкаТаблицыHTML (HTMLTableColElement)

УстановитьАтрибут (SetAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьАтрибут(<Имя>, <Значение>)

Параметры:

<Имя> (обязательный)

Имя атрибута.

<Значение> (обязательный)

Тип: Строка.   
Значение атрибута.

Описание:

Устанавливает значение атрибута с указанным именем.  
Если атрибута с таким именем нет, то он будет создан.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если имя атрибута содержит символы, не допустимые стандартом XML данного документа, или данный узел в режиме Только для чтения, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
