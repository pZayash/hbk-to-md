РегистрНакопленияВыборка.<Имя регистра накопления> (AccumulationRegisterSelection.<Accumulation register name>)

Активность (Active)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак, определяющий влияние записи на итоги регистра накопления.  
Если значение [Ложь](lang__def_BooleanFalse.md), то запись не учитывается в итогах регистра.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
