ВнешнийИсточникДанныхТаблицаМенеджер.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableManager.<External source name>.<External data source table name>)

НайтиПоПолю (FindByField)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоПолю(<ИмяПоля>, <ЗначениеПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля, как оно задано в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и строк произвольной длины.

<ЗначениеПоля> (обязательный)

Тип: Произвольный.   
Значение поля, по которому должен выполняться поиск.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRef.External_source_name.External_data_source_table_name).md).   
Если не существует ни одного элемента с требуемым значением поля, то будет возвращена пустая ссылка.

Описание:

Осуществляет поиск элемента по значению поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных [ОбъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.ОбъектныеДанные_(ExternalDataSourceTableDataType.ObjectData).md).  
Если существует несколько элементов с указанным значением поля, то будет найден только один из них.  
Для реквизитов типа [Строка](lang__def_String.md) поиск осуществляется по точному соответствию.

Использование в версии:

Доступен, начиная с версии 8.2.
