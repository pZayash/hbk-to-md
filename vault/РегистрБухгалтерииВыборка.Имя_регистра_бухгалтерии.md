Описание:

Выборка формируется методом [Выбрать](РегистрБухгалтерииМенеджер.Имя_регистра_бухгалтерии.Выбрать.md) и представляет собой специализированный способ перебора элементов регистра бухгалтерии.  
Обход элементов выполняется системой динамически, при котором не производится выборка всех элементы сразу. Это позволяет обходить с помощью выборки большие регистры бухгалтерии.

Доступность:

Сервер, толстый клиент, внешнее соединение.
