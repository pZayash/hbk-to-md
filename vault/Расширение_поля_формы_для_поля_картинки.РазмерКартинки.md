Расширение поля формы для поля картинки (Form field extension for a picture field)

РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РазмерКартинки](РазмерКартинки.md).   
Определяет режим приспособления оригинальных размеров картинки к размеру поля картинки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
