ЗаписьУзловDOM (DOMNodeWriter)

ЗаписатьАтрибут (WriteAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Запись атрибута с полным именем

Синтаксис:

ЗаписатьАтрибут(<ПолноеИмя>, <Значение>)

Параметры:

<ПолноеИмя> (обязательный)

Тип: Строка.   
Полное имя атрибута. Может включать префикс.

<Значение> (обязательный)

Тип: Строка.   
Значение атрибута.

Вариант синтаксиса: Запись атрибута с локальным именем и URI пространства имен

Синтаксис:

ЗаписатьАтрибут(<ЛокальноеИмя>, <URIПространстваИмен>, <Значение>)

Параметры:

<ЛокальноеИмя> (обязательный)

Тип: Строка.   
Локальное имя атрибута.

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен для локального имени атрибута.

<Значение> (обязательный)

Тип: Строка.   
Значение атрибута.

Описание:

Записывает атрибут элемента.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Атрибутов может быть произвольное количество. Атрибут может идентифицироваться полным именем или локальным именем и URI пространства имен. Атрибуты должны записываться после записи начала элемента и перед записью текста элемента.

Пример:

|  |
| --- |
| Запись.ЗаписатьНачалоЭлемента("apple"); Запись.ЗаписатьСоответствиеПространстваИмен("nsp", "http://metrics.org"); Запись.ЗаписатьАтрибут("weight", "http://metrics.org", "12.5"); Запись.ЗаписатьКонецЭлемента(); |

Использование в версии:

Доступен, начиная с версии 8.1.
