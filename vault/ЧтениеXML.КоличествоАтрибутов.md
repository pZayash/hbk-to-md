ЧтениеXML (XMLReader)

КоличествоАтрибутов (AttributeCount)

Доступен, начиная с версии 8.1.

Синтаксис:

КоличествоАтрибутов()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество атрибутов у последнего прочитанного узла XML. Так как атрибуты могут быть только у элемента XML, доступ к ним может осуществляться только при прочтении узла, имеющего тип [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Чтение.ОткрытьФайл("c:/docs/data.xml"); Пока Чтение.Прочитать() Цикл     Если Чтение.КоличествоАтрибутов() > 0 Тогда         Пока Чтение.ПрочитатьАтрибут() Цикл             // Обработка атрибута          КонецЦикла     КонецЕсли КонецЦикла |

Использование в версии:

Доступен, начиная с версии 8.1.
