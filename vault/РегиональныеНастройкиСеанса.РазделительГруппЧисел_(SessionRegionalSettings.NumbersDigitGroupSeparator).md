РегиональныеНастройкиСеанса (SessionRegionalSettings)

РазделительГруппЧисел (NumbersDigitGroupSeparator)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Строка из одного символа. Символ, который должен использоваться для разделения групп цифр при получении представления данных типа [Число](lang__def_Number.md).

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
