ХарактеристикиКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionCharacteristics)

Получить (Get)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции.  
Индексация начинается с 0.

Возвращаемое значение:

Тип: [ХарактеристикаКомпоновкиДанныхСхемыЗапроса](ХарактеристикаКомпоновкиДанныхСхемыЗапроса_(QuerySchemaDataCompositionCharacteristic).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
