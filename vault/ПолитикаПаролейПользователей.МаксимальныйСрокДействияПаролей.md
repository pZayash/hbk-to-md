ПолитикаПаролейПользователей (UserPasswordPolicy)

МаксимальныйСрокДействияПаролей (PasswordMaxEffectivePeriod)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Численное значение максимального срока действия паролей в секундах. Максимальный срок действия паролей не может быть меньше минимального.  
  
Значение должно быть в интервале от 0 до 4294967295.  
Значение по умолчанию: 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
