Глобальный контекст (Global context)

ПриГлобальномПоиске (OnGlobalSearch)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПриГлобальномПоиске(<СтрокаПоиска>, <ПланПоиска>)

Параметры:

<СтрокаПоиска>

Тип: Строка.   
Строка поиска.

<ПланПоиска>

Тип: [ПланГлобальногоПоиска](ПланГлобальногоПоиска.md).   
Текущий план выполнения глобального поиска.

Описание:

Обработчик вызывается при вводе в строку глобального поиска (после стандартной задержки).  
При вызове события в параметре <ПланПоиска> передается копия плана, установленного в менеджере глобального поиска.  
Содержимое параметра <ПланПоиска> можно модифицировать в коде обработчика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
