МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ДобавитьВАдминистраторыАбонента (AddToSubscriberAdministrators)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ДобавитьВАдминистраторыАбонента(<Пользователь>, <КодДоступа>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Пользователь, назначаемый администратором абонента.

<КодДоступа> (необязательный)

Тип: Строка, Неопределено.   
Ключ администрирования абонента или временный код администрирования.  
Значение по умолчанию: Неопределено.

Описание:

Добавляет пользователя в список администраторов абонента.  
Если код доступа не задан, то текущий пользователь должен быть администратором абонента. В противном случае генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
