Описание:

Определяет набор вариантов ответа пользователя на стандартный вопрос. Используется в качестве возвращаемого значения функции [Вопрос](Глобальный_контекст.Вопрос.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Вопрос](Глобальный_контекст.Вопрос.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
