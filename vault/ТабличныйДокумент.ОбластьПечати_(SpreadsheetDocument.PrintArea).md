ТабличныйДокумент (SpreadsheetDocument)

ОбластьПечати (PrintArea)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md).   
Содержит область табличного документа, которая будет выводиться на печать.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если не задано, то будет выведена область, ограниченная значениями свойств [ВысотаТаблицы](ТабличныйДокумент.ВысотаТаблицы_(SpreadsheetDocument.TableHeight).md) и [ШиринаТаблицы](ТабличныйДокумент.ШиринаТаблицы_(SpreadsheetDocument.TableWidth).md).   
Если [ВысотаТаблицы](ТабличныйДокумент.ВысотаТаблицы_(SpreadsheetDocument.TableHeight).md)=0 и [ШиринаТаблицы](ТабличныйДокумент.ШиринаТаблицы_(SpreadsheetDocument.TableWidth).md)=0, то будет выдано сообщение "Нет информации для вывода на печать".  
Если содержит прямоугольную область, то будет выведена она.  
Если содержит область колонок, то будет выведена эта область, ограниченная снизу значением [ВысотаТаблицы](ТабличныйДокумент.ВысотаТаблицы_(SpreadsheetDocument.TableHeight).md).  
Если содержит область строк, то будет выведена эта область, ограниченная справа значением [ШиринаТаблицы](ТабличныйДокумент.ШиринаТаблицы_(SpreadsheetDocument.TableWidth).md).

Использование в версии:

Доступен, начиная с версии 8.0.
