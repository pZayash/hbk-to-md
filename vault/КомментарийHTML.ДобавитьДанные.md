КомментарийHTML (HTMLComment)

ДобавитьДанные (AppendData)

Доступен, начиная с версии 8.2.

Синтаксис:

ДобавитьДанные(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Строка, добавляемая к символьным данным.

Описание:

Добавляет строку к символьным данным.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.2.
