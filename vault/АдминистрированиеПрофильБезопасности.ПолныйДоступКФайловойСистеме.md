АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКФайловойСистеме (FileSystemFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет полный доступ к файловой системе сервера.  
Истина – доступна файловая система. Ложь – доступ к файловой системе запрещен, можно использовать только виртуальные каталоги, перечисленные в списке виртуальных каталогов профиля безопасности.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
