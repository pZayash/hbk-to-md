Расширение группы формы для командной панели (Form group extension for a command bar)

ГоризонтальноеПоложение (HorizontalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента.md).   
Содержит вид выравнивания элементов в командной панели.  
Запрещено использовать значение [Авто](ГоризонтальноеПоложениеЭлемента.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
