# ЗадачаОбъект.<Имя задачи>.Модифицированность

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект изменен; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, был ли изменен объект после считывания из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, был ли изменен объект другими пользователями.

Пример:

|  |
| --- |
| Если Объект.Модифицированность() Тогда     Объект.Записать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
