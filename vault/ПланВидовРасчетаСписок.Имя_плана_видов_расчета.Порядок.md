ПланВидовРасчетаСписок.<Имя плана видов расчета> (ChartOfCalculationTypesList.<Chart of calculation types name>)

Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Универсальные_объекты.Порядок.md).   
Содержит объект [Порядок](Универсальные_объекты.Порядок.md), по которому осуществляется текущее упорядочивание видов расчета в списке. В качестве имен упорядочивания объекта могут выступать имена предопределенных и обычных реквизитов видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
