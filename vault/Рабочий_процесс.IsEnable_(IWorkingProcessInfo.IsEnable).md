Рабочий процесс (IWorkingProcessInfo)

IsEnable (IsEnable)

Не рекомендуется использовать, начиная с версии 8.3.27.

Рекомендуется использовать:

- [TurnedOn](Рабочий_процесс.TurnedOn_(IWorkingProcessInfo.TurnedOn).md)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливается кластером при необходимости запустить или остановить рабочий процесс.  
[Истина](lang__def_BooleanTrue.md) - процесс должен быть запущен и будет запущен при возможности.   
[Ложь](lang__def_BooleanFalse.md) - процесс должен быть остановлен и будет остановлен после отсоединения всех пользователей или после истечения времени, заданного свойством [ExpirationTimeout](Кластер_серверов.ExpirationTimeout_(IClusterInfo.ExpirationTimeout).md) объекта [Кластер серверов](Кластер_серверов_(IClusterInfo).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.3.27.
