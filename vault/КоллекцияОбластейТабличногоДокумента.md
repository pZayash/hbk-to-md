# КоллекцияОбластейТабличногоДокумента

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Табличный документ](objects__catalog63__catalog64.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md) или [РисунокТабличногоДокумента](РисунокТабличногоДокумента.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются области или рисунки табличного документа.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Свойства:

[\<Имя области\> (\<Area name\>)](КоллекцияОбластейТабличногоДокумента.Имя_области.md)  

Методы:

[Индекс (IndexOf)](КоллекцияОбластейТабличногоДокумента.Индекс.md)  
[Количество (Count)](КоллекцияОбластейТабличногоДокумента.Количество.md)  
[Найти (Find)](КоллекцияОбластейТабличногоДокумента.Найти.md)  
[Получить (Get)](КоллекцияОбластейТабличногоДокумента.Получить.md)  

Описание:

Представляет собой коллекцию областей табличного документа, включая именованные рисунки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), свойство [Области](ТабличныйДокумент.Области.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
