Форма (Form)

ОбновлениеОтображения (RefreshDisplay)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбновлениеОтображения()

Описание:

Возникает при обновлении формы или при изменении данных формы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
