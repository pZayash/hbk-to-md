Глобальный контекст (Global context)

ИмпортМоделиXDTO (ImportXDTOModel)

Доступен, начиная с версии 8.1.

Синтаксис:

ИмпортМоделиXDTO(<НаборСхемXML>)

Параметры:

<НаборСхемXML> (обязательный)

Тип: [НаборСхемXML](НаборСхемXML.md).   
Импортируемый набор схем XML.

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO.md).   

Описание:

Осуществляет импорт переданного набора схем XML в модель типов XDTO. Возвращаемое значение – [ОбъектXDTO](ОбъектXDTO.md), имеющий тип {http://v8.1c.ru/8.1/xdto}Model. При импорте никакой проверки правильности схемы не производится. При импорте производится преобразование структуры типов схемы в структуру типов модели. Полученная модель может быть не идентична структуре типов схемы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.20.
