ОбъектМетаданных: Таблица (MetadataObject: Table)

СпособРедактирования (EditType)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [СпособРедактирования](СпособРедактирования_(EditType).md).   
Определяет способ редактирования объектов записей таблицы (например, [ВСписке](СпособРедактирования.ВСписке_(EditType.InList).md), [ВДиалоге](СпособРедактирования.ВДиалоге_(EditType.InDialog).md), [ОбоимиСпособами](СпособРедактирования.ОбоимиСпособами_(EditType.BothWays).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
