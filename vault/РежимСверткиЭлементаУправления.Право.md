РежимСверткиЭлементаУправления (ControlCollapseMode)

Право (Right)

Доступен, начиная с версии 8.0.

Описание:

Элемент управления свертывается к своей правой стороне (левая сторона прижимается к правой).

Использование в версии:

Доступен, начиная с версии 8.0.
