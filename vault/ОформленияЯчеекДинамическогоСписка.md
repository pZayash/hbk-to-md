Доступен, начиная с версии 8.3.10.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа [КлючИЗначение](КлючИЗначение.md), где Ключ - строка, содержащая путь компоновки данных соответствующей колонки,¶Значение - [ОформлениеЯчейкиДинамическогоСписка](ОформлениеЯчейкиДинамическогоСписка.md) - [ОформлениеЯчейкиДинамическогоСписка](ОформлениеЯчейкиДинамическогоСписка.md).  
Возможно обращение к оформлению ячейки посредством оператора [...]. В качестве аргумента передается строка (путь компоновки данных соответствующей колонки).  

Методы:

[Количество (Count)](ОформленияЯчеекДинамическогоСписка.Количество.md)  
[Получить (Get)](ОформленияЯчеекДинамическогоСписка.Получить.md)  

Описание:

Содержит коллекцию оформлений ячеек динамического списка. Коллекция содержит элементы только для видимых ячеек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
