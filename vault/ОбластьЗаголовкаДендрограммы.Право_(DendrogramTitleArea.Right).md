ОбластьЗаголовкаДендрограммы (DendrogramTitleArea)

Право (Right)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Относительное положение правой границы.   
Может содержать значения от 0 до 1.   
0 - левый край, 0,5 - центр, 1 - правый край.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
