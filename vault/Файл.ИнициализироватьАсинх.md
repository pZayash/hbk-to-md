Файл (File)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Полное имя файла или каталога, с которым будет связан объект.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - инициализированный объект типа [Файл](Файл.md).

Описание:

Асинхронная функция. Начинает инициализацию объекта [Файл](Файл.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Файл](Файл.md), метод [НачатьИнициализацию](Файл.НачатьИнициализацию.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
