МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьПользователей (GetUsers)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьПользователей(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [ОтборПользователейСистемыВзаимодействия](ОтборПользователейСистемыВзаимодействия.md).   
Для строковых полей отбор выполняется по полному соответствию.  
Для полей "Имя", "ПолноеИмя", "НомерТелефона", "АдресЭлектроннойПочты", "АдресЭлектроннойПочтыДляАутентификации" сравнение значений выполняется без учета регистра.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Возвращает список пользователей системы взаимодействия.  
Элементами массива являются объекты типа [ПользовательСистемыВзаимодействия](ПользовательСистемыВзаимодействия.md).  
Требуется использование привилегированного режима или наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.20.
