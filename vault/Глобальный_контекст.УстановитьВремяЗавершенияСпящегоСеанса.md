Глобальный контекст (Global context)

УстановитьВремяЗавершенияСпящегоСеанса (SetHibernateSessionTerminateTime)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьВремяЗавершенияСпящегоСеанса(<Таймаут>)

Параметры:

<Таймаут> (необязательный)

Тип: Число.   
Интервал времени в секундах, по истечении которого спящий сеанс завершается.  
Значение по умолчанию: 86400.

Описание:

Установить интервал времени в секундах, по истечении которого спящий сеанс будет завершен.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
