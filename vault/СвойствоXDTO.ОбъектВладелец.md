СвойствоXDTO (XDTOProperty)

ОбъектВладелец (OwnerObject)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектXDTO](ОбъектXDTO.md).   
Содержит объект-владелец данного свойства.  
В случае, если у свойства определен объект-владелец, свойство является пользовательским. Такие свойства могут присутствовать только у объектов открытых типов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
