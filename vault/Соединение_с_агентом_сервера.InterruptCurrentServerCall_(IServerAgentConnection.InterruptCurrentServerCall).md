Соединение с агентом сервера (IServerAgentConnection)

InterruptCurrentServerCall (InterruptCurrentServerCall)

Доступен, начиная с версии 8.3.13.

Синтаксис:

InterruptCurrentServerCall(<Кластер>, <Сеанс>, <СообщениеОбОшибке>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, из которого будет удален сеанс. Должен содержать объект, полученный методом [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md).

<Сеанс> (обязательный)

Тип: [Сеанс](Сеанс_(ISessionInfo).md).   
Сеанс, который будет удален. Должен содержать объект, полученный методом [GetSessions](Соединение_с_агентом_сервера.GetSessions_(IServerAgentConnection.GetSessions).md) или [GetInfoBaseSessions](Соединение_с_агентом_сервера.GetInfoBaseSessions_(IServerAgentConnection.GetInfoBaseSessions).md).

<СообщениеОбОшибке> (необязательный)

Тип: [Строка](lang__def_String.md).   
Сообщение для пользователя, которое содержит причину прерывания текущего серверного вызова.  
Значение по умолчанию: "Выполнение текущей операции прервано администратором".

Описание:

Прерывает выполнение текущего серверного вызова.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.13.
