ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновнаяФормаВариантаОтчета (DefaultReportVariantForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Указывает основную форму варианта отчета, которая будет использована, если у отчета не установлена основная форма варианта отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
