Требование назначения сервисов (IAssignmentRule)

InfoBaseName (InfoBaseName)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя информационной базы, для которой действует требование.   
Пустая строка, если требование должно действовать для всех информационных баз.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
