ЯчейкаМакетаКоллекцииЗначенийОбластиКомпоновкиДанных (DataCompositionAreaTemplateValueCollectionCell)

Значение (Value)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Содержит значение, которое будет выводиться в коллекцию значений.  
Может содержать параметр компоновки данных. В таком случае в результат будет выведено значение параметра.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
