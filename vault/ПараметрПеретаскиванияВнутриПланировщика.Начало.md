ПараметрПеретаскиванияВнутриПланировщика (PlannerInsideDragParameter)

Начало (Begin)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   

- В случае создания элемента планировщика путем выделения пустой области является датой начала области выделения.
- В случае выделения элементов планировщика является датой начала области выделения.
- В случае редактирования элемента планировщика путем перетаскивания является датой начала редактируемого элемента.
- В случае копирования существующего элемента планировщика является датой начала создаваемого элемента.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
