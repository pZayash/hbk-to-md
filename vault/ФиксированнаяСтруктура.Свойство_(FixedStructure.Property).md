# ФиксированнаяСтруктура.Свойство

ФиксированнаяСтруктура (FixedStructure)

Свойство (Property)

Доступен, начиная с версии 8.2.

Синтаксис:

Свойство(<Ключ>, <НайденноеЗначение>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ для поиска свойства.

<НайденноеЗначение> (необязательный)

Тип: Произвольный.   
В данный параметр возвращается найденное значение. Если ключа нет в структуре, то возвращается [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - свойство найдено; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Получает значение свойства структуры.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
