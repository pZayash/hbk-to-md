Доступен, начиная с версии 8.3.22.

Синтаксис:

УстановитьОписаниеВыгрузкиДанныхАсинх(<Идентификатор>, <Описание>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   
Идентификатор выгрузки данных.

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).

Тип: [Обещание](Обещание_(Promise).md).

Асинхронная функция. Устанавливает новое описание выгрузки данных.

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.
