# РегистрСведенийНаборЗаписей.<Имя регистра сведений>.УстановитьАктивность

РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

УстановитьАктивность (SetActive)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьАктивность(<Активность>)

Параметры:

<Активность> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак активности записей. [Истина](lang__def_BooleanTrue.md) - записи активны. [Ложь](lang__def_BooleanFalse.md) - записи не активны.

Описание:

Устанавливает значение свойства [Активность](РегистрСведенийЗапись.Имя_регистра_сведений.Активность_(InformationRegisterRecord.Information_register_name.Active).md) для всех записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| НаборКурсов.УстановитьАктивность(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
