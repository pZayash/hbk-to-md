Форма (Form)

ОбработкаВыбора (ChoiceProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаВыбора(<ЗначениеВыбора>, <Источник>)

Параметры:

<ЗначениеВыбора>

Тип: Произвольный.   
Результат выбора в подчиненной форме. В случае множественного выбора возвращается массив выбранных значений.

<Источник>

Тип: Произвольный.   
Форма - источник события.

Описание:

Возникает при выборе объекта в одной из подчиненных форм или при вызове в одной из подчиненных форм метода [ОповеститьОВыборе](Форма.ОповеститьОВыборе_(Form.NotifyChoice).md).

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), метод [ОповеститьОВыборе](Форма.ОповеститьОВыборе_(Form.NotifyChoice).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
