КоллекцияПользовательскихПолейКомпоновкиДанных (DataCompositionUserFieldCollection)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных.md) и [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных.md), [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных.md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
