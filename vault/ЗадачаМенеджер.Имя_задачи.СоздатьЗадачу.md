ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

СоздатьЗадачу (CreateTask)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьЗадачу()

Возвращаемое значение:

Тип: [ЗадачаОбъект.\<Имя задачи\>](ЗадачаОбъект.Имя_задачи.md).   

Описание:

Создает новую задачу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Создание и запись новой задачи  НовЗадача = Задачи.Задача.СоздатьЗадачу(); НовЗадача.Дата = ТекущаяДата(); // Установка прочих реквизитов задачи  НовЗадача.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
