СеансИнформационнойБазы (InfoBaseSession)

ИмяКомпьютера (ComputerName)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя компьютера, с которого начат сеанс.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для сеанса веб-клиента - пустая строка.

Использование в версии:

Доступен, начиная с версии 8.2.
