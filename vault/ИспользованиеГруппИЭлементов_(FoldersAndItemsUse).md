# ИспользованиеГруппИЭлементов

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Свойства прикладных объектов](Свойства_прикладных_объектов.md)

Доступен, начиная с версии 8.0.

Значения

[Группы (Folders)](ИспользованиеГруппИЭлементов.Группы_(FoldersAndItemsUse.Folders).md)  
[ГруппыИЭлементы (FoldersAndItems)](ИспользованиеГруппИЭлементов.ГруппыИЭлементы_(FoldersAndItemsUse.FoldersAndItems).md)  
[Элементы (Items)](ИспользованиеГруппИЭлементов.Элементы_(FoldersAndItemsUse.Items).md)  

Описание:

Определяет способ использования групп и элементов справочника.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |
