**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Оформление](Оформление-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияНастройкойОбластиОформления](ЭлементУправленияНастройкойОбластиОформления_(AppearanceAreasSettingControl).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы управления областями оформления.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Доступность:
