ФрагментXS (XSParticle)

ЯвляетсяПодмножеством (IsSubset)

Доступен, начиная с версии 8.1.

Синтаксис:

ЯвляетсяПодмножеством()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, является ли указанный фрагмент подмножеством данного.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определение производится в соответствии с алгоритмом http://www.w3.org/TR/xmlschema-1/#cos-particle-restrict.

Использование в версии:

Доступен, начиная с версии 8.1.
