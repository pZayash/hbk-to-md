ПолеГеографическойСхемы (GeographicalSchemaField)

ОтображатьКоординаты (ShowCoordinates)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак показа дополнительной информационной панели, в которой отображаются координаты точки на схеме под курсором мыши.

Доступность:

Толстый клиент.

Примечание:

Если [Истина](lang__def_BooleanTrue.md), то координаты точки отображаются на схеме. Значение по умолчанию [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.0.
