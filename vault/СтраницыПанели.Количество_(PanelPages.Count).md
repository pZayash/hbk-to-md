СтраницыПанели (PanelPages)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество страниц панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Для Номер = 0 По ЭлементыФормы.Панель.Страницы.Количество()-1 Цикл     ТекущаяСтраница = ЭлементыФормы.Панель.Страницы.Получить(Номер);     // обработка страницы  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
