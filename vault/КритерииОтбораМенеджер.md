# КритерииОтбораМенеджер

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КритерийОтбораМенеджер.\<Имя критерия\>](КритерийОтбораМенеджер.Имя_критерия.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры критериев отбора.  
Возможно обращение к менеджеру критерия отбора посредством оператора [...]. В качестве аргумента передается имя критерия отбора, как оно задано в конфигурации.  

Свойства:

[\<Имя критерия отбора\> (\<Filter criteria name\>)](КритерииОтбораМенеджер.Имя_критерия_отбора.md)  

Описание:

Предназначен для управления критериями отбора и предоставляет доступ к значениям типа [КритерийОтбораМенеджер.](КритерийОтбораМенеджер.Имя_критерия.md)<[Имя критерия](КритерийОтбораМенеджер.Имя_критерия.md)>. Доступ к объекту осуществляется через свойство глобального контекста [КритерииОтбора](Глобальный_контекст.КритерииОтбора.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| Форма = КритерииОтбора.Контрагенты.ПолучитьФорму("ОсновнаяФорма"); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [КритерииОтбора](Глобальный_контекст.КритерииОтбора.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
