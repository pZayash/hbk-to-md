Расширение дополнения элемента формы для отображения строки поиска (Form item addition extension for search string)

Шрифт (Font)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт.md).   
Содержит шрифт.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
