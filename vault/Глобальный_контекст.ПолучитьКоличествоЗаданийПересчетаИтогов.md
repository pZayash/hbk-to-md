Глобальный контекст (Global context)

ПолучитьКоличествоЗаданийПересчетаИтогов (GetTotalRecalcJobCount)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьКоличествоЗаданийПересчетаИтогов()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает заданное в информационной базе количество заданий пересчета итогов.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
