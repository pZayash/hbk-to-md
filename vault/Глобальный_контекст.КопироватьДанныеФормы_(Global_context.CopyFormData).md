Глобальный контекст (Global context)

КопироватьДанныеФормы (CopyFormData)

Доступен, начиная с версии 8.2.

Синтаксис:

КопироватьДанныеФормы(<Источник>, <Приемник>)

Параметры:

<Источник> (обязательный)

Тип: [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md), [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md).   
Объект - источник.

<Приемник> (обязательный)

Тип: [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md), [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md).   
Объект приемник.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - копирование произведено. [Ложь](lang__def_BooleanFalse.md) - структура объектов абсолютно несовместима.

Описание:

Копирует данные формы, обладающие совместимой структурой.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Во внешнем соединении метод игнорируется.

Использование в версии:

Доступен, начиная с версии 8.2.
