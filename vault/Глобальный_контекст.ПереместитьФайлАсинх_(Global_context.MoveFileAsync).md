Глобальный контекст (Global context)

ПереместитьФайлАсинх (MoveFileAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПереместитьФайлАсинх(<ИмяФайлаИсточника>, <ИмяФайлаПриемника>)

Параметры:

<ИмяФайлаИсточника> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла-источника (полное исходное имя файла).

<ИмяФайлаПриемника> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла-приемника (полное новое имя файла).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - строка, содержащая путь к файлу. Допускается использование схем http, https и ftp для адресации файлов. При использовании указанных схем в адресах необходимо указывать косые черты '/', а не обратные '\'.

Описание:

Асинхронная функция. Выполняет перемещение (переименование) указанного файла.   
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.  
На веб-клиенте метод работает только с локальными файлами.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
