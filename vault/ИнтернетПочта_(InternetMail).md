Описание:

Предназначен для доступа к почтовым серверам для отправки и получения сообщений (писем). В отличие от объекта [Почта](Почта_(Mail).md), который работает по технологии MAPI и требует от пользователя установленного почтового клиента.  
Использует наиболее распространенные Интернет протоколы SMTP, POP3 и IMAP.   
Не требует установленного почтового клиента и, если почтовый клиент все же установлен, работает с ним(-и) параллельно.  
Для протоколов IMAP, POP3 и SMTP поддерживается работа по SSL, но без возможности проверки сертификата клиента или сервера.  
Важно! Если почтовый сервер не поддерживает работу с идентификаторами, то в качестве идентификаторов используются [ИдентификаторСообщения](ИнтернетПочтовоеСообщение.ИдентификаторСообщения_(InternetMailMessage.MessageID).md) писем.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
