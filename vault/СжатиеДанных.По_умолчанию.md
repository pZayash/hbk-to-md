СжатиеДанных (Deflation)

По степени сжатия

Доступен, начиная с версии 8.0.

Синтаксис:

Новый СжатиеДанных(<СтепеньСжатия>)

Параметры:

<СтепеньСжатия> (необязательный)

Тип: Число.   
Степень сжатия алгоритма Deflation. Целое число в диапазоне -1...9. -1 - степень сжатия по умолчанию. 0 - никакого сжатия, 9 - максимальная степень сжатия.  
Значение по умолчанию: -1.

Описание:

Создает объект [СжатиеДанных](СжатиеДанных.md). В качестве необязательного параметра конструктора может быть задано целое число, определяющее степень сжатия данных.

Пример:

|  |
| --- |
| Сжатие = Новый СжатиеДанных(9); |

Использование в версии:

Доступен, начиная с версии 8.0.
