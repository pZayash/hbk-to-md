МенеджерФайловыхПотоков (FileStreamsManager)

Создать (Create)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Создать(<ИмяФайла>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя создаваемого файла.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер буфера.

Возвращаемое значение:

Тип: [ФайловыйПоток](ФайловыйПоток.md).   

Описание:

Создает или перезаписывает файл и открывает поток с возможностью чтения и записи в файл. Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

[МенеджерФайловыхПотоков](МенеджерФайловыхПотоков.md), метод [НачатьСоздание](МенеджерФайловыхПотоков.НачатьСоздание.md)
