ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

Пользователь (User)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя пользователя, который используется для стандартной аутентификации или аутентификации ОС на WebSocket-сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию - Пустая строка. Аутентификация не используется.

Использование в версии:

Доступен, начиная с версии 8.3.27.
