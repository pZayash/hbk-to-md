ВариантГраницыИнтервала (IntervalBoundVariant)

Смещение (BeforeAfter)

Доступен, начиная с версии 8.0.

Описание:

Начало интервала задается датой за несколько дней до рабочей даты, конец интервала - через несколько дней после рабочей даты.

Использование в версии:

Доступен, начиная с версии 8.0.
