ПостроительОтчета (ReportBuilder)

Выполнить (Execute)

Доступен, начиная с версии 8.0.

Синтаксис:

Выполнить()

Описание:

Выполняет запрос, построенный для отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Выставляет внутренний флаг необходимости перевыполнения запроса при выводе отчета (метод [Вывести](ПостроительОтчета.Вывести.md)) и получении результата запроса.

Пример:

|  |
| --- |
| ПостроительОтчета = Новый ПостроительОтчета; ПостроительОтчета.Текст = ТекстЗапроса; ПостроительОтчета.Выполнить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
