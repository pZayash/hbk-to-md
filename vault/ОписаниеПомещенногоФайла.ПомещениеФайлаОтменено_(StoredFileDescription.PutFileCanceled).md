ОписаниеПомещенногоФайла (StoredFileDescription)

ПомещениеФайлаОтменено (PutFileCanceled)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отказа от помещения во временное хранилище файла.  
[Истина](lang__def_BooleanTrue.md) - если помещение файла было отменено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
