# ПользовательИнформационнойБазы.ПоказыватьВСпискеВыбора

ПользовательИнформационнойБазы (InfoBaseUser)

ПоказыватьВСпискеВыбора (ShowInList)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, показывать ли данного пользователя в списке для выбора при старте системы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл, только если установлено свойство [АутентификацияСтандартная](ПользовательИнформационнойБазы.АутентификацияСтандартная_(InfoBaseUser.StandardAuthentication).md).  
[Истина](lang__def_BooleanTrue.md) - показ данного пользователя разрешен; [Ложь](lang__def_BooleanFalse.md) - запрещен.  
Доступно только пользователю с административными правами.

Использование в версии:

Доступен, начиная с версии 8.0.
