# ПоискВТаблицеПриВводе

Значения

[Авто (Auto)](ПоискВТаблицеПриВводе.Авто_(SearchInTableOnInput.Auto).md)  
[Использовать (Use)](ПоискВТаблицеПриВводе.Использовать_(SearchInTableOnInput.Use).md)

Описание:

Содержит варианты режимов выполнения поиска при вводе символов в таблице, связанной с данными типа [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы_(FormTable).md), свойство [ПоискПриВводе](ТаблицаФормы.ПоискПриВводе_(FormTable.SearchOnInput).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
