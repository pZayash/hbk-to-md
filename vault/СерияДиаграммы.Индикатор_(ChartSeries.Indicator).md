СерияДиаграммы (ChartSeries)

Индикатор (Indicator)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [ТипГрафическогоПредставления](СерияДиаграммы.ТипГрафическогоПредставления_(ChartSeries.GraphicalRepresentationType).md)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - в гистограммах серия показывается как график и никогда не попадает в сводную серию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.16.

Описание изменено в версии 8.3.16.
