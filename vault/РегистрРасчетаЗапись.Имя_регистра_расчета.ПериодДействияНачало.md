РегистрРасчетаЗапись.<Имя регистра расчета> (CalculationRegisterRecord.<Calculation register name>)

ПериодДействияНачало (BegOfActionPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Дата начала периода действия записи.  
Период действия записи (задаваемый датой начала и датой окончания) может не совпадать с фактическим периодом действия, который в общем случае - набор нескольких непересекающихся временных интервалов (см. [ФактическийПериодДействия](objects__catalog125__catalog486__object491__properties__TrueEffectivePeriod2106.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определен для регистров расчета с признаком "Период действия".
