РасширениеКонфигурации (ConfigurationExtension)

ОбластьДействия (Scope)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации_(ConfigurationExtensionScope).md), [Неопределено](lang__def_Undefined.md).   
Область действия расширения конфигурации.  
Для нового объекта [РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md) значение свойства может быть прочитано как [Неопределено](lang__def_Undefined.md). При записи такого расширения это значение будет автоматически заменено на одно из значений типа [ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации_(ConfigurationExtensionScope).md) по следующим правилам:

- Если включено разделение расширений конфигурации, то будет выбрано значение [РазделениеДанных](ПользовательИнформационнойБазы.РазделениеДанных_(InfoBaseUser.DataSeparation).md).
- В остальных случаях будет выбрано значение [ИнформационнаяБаза](ОбластьДействияРасширенияКонфигурации.ИнформационнаяБаза_(ConfigurationExtensionScope.InfoBase).md).

Эти значения также можно установить явно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ОбластьДействияРасширенияКонфигурации](ОбластьДействияРасширенияКонфигурации_(ConfigurationExtensionScope).md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
