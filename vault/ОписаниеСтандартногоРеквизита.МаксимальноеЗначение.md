ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

МаксимальноеЗначение (MaxValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Неопределено.   
Устанавливает максимальное возможное для реквизита значение.  
Имеет смысл для реквизитов типа Число.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
