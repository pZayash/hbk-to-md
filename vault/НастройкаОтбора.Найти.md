НастройкаОтбора (FilterSettings)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента отбора.

Возвращаемое значение:

Тип: [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md), Неопределено.   
Если не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента по имени элемента отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
