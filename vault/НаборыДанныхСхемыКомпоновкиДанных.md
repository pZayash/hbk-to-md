Элементы коллекции:

[НаборДанныхЗапросСхемыКомпоновкиДанных](НаборДанныхЗапросСхемыКомпоновкиДанных.md), [НаборДанныхОбъектСхемыКомпоновкиДанных](НаборДанныхОбъектСхемыКомпоновкиДанных.md), [НаборДанныхОбъединениеСхемыКомпоновкиДанных](НаборДанныхОбъединениеСхемыКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы данных.  
Возможно обращение к набору данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
