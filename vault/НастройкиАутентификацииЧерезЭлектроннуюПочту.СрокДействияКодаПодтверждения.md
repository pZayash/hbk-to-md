НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

СрокДействияКодаПодтверждения (VerificationCodeEffectivePeriod)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет срок действия кода аутентификации через электронную почту (в секундах).  
Значение по умолчанию: 600 (10 минут).  
Диапазон значений: от 1 до 999999999.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
