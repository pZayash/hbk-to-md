# КолонкиАнализаДанных.Найти

КолонкиАнализаДанных (DataAnalysisColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомой колонки.

Возвращаемое значение:

Тип: [КолонкаАнализаДанных](КолонкаАнализаДанных_(DataAnalysisColumn).md), [Неопределено](lang__def_Undefined.md).   
Если колонка не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск колонки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
