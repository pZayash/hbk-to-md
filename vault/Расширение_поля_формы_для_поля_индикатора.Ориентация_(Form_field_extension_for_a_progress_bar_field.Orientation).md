Расширение поля формы для поля индикатора (Form field extension for a progress bar field)

Ориентация (Orientation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОриентацияЭлементаФормы](ОриентацияЭлементаФормы_(FormItemOrientation).md).   
Содержит вариант ориентации индикатора.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
