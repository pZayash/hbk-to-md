ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

Индексы (Indexes)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: [ИндексыСхемыЗапроса](ИндексыСхемыЗапроса_(QuerySchemaIndexes).md).   
Содержит коллекцию описаний индексов, по которым будет индексироваться временная таблица.  
Соответствует выражению ИНДЕКСИРОВАТЬ ПО или ИНДЕКСИРОВАТЬ ПО НАБОРАМ() языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
