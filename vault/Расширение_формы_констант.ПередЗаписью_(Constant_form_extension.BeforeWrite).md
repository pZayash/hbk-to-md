# Расширение формы констант.ПередЗаписью

Расширение формы констант (Constant form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи объекта из формы. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](КонстантаМенеджерЗначения.Имя_константы.ПередЗаписью_(ConstantValueManager.Constant_name.BeforeWrite).md)объекта [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы_(ConstantValueManager.Constant_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
