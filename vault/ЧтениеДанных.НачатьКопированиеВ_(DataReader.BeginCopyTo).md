ЧтениеДанных (DataReader)

НачатьКопированиеВ (BeginCopyTo)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НачатьКопированиеВ(<ОписаниеОповещения>, <Поток>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Описание процедуры, которая будет вызвана после завершения операции.  
Процедура должна содержать следующие параметры:

- <Количество> - тип [Число](lang__def_Number.md). Фактическое количество скопированных байтов;
- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md)

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).   
Поток или объект [ЗаписьДанных](ЗаписьДанных_(DataWriter).md), в который требуется скопировать данные.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется скопировать. Если не задано, то будут скопированы все данные до конца вложенного потока.

Описание:

Начинает копирование заданного количества байтов из объекта [ЧтениеДанных](ЧтениеДанных_(DataReader).md) в указанный поток или объект [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
