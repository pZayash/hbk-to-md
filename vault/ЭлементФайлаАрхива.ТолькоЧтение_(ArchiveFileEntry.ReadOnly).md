ЭлементФайлаАрхива (ArchiveFileEntry)

ТолькоЧтение (ReadOnly)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что у файла установлен признак "Только чтение".  
[Истина](lang__def_BooleanTrue.md) - установлен признак "Только чтение".

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
