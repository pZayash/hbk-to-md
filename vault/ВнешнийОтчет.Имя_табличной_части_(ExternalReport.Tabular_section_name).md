ВнешнийОтчет (ExternalReport)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: ВнешнийОтчетТабличнаяЧасть.<Имя отчета>.<Имя табличной части>.   
Набор свойств содержит табличные части внешнего отчета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ВнешнийОтчетТабличнаяЧасть.<Имя отчета>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
