ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов.md).   

Описание:

Предоставляет доступ к объекту описания метаданных плана счетов. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.ПланыСчетов.Основной.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ВыборкаПоСчету = ПланыСчетов[Счет.Метаданные().Имя].Выбрать(Счет); |

Использование в версии:

Доступен, начиная с версии 8.0.
