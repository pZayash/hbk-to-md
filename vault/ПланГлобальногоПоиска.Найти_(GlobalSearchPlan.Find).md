# ПланГлобальногоПоиска.Найти

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Глобальный поиск](Глобальный_поиск.md) › [ПланГлобальногоПоиска](ПланГлобальногоПоиска_(GlobalSearchPlan).md) › [Методы](ПланГлобальногоПоиска__Методы.md)

ПланГлобальногоПоиска (GlobalSearchPlan)

Найти (Find)

Доступен, начиная с версии 8.3.15.

Вариант синтаксиса: По виду поиска

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [СтандартныйВидГлобальногоПоиска](СтандартныйВидГлобальногоПоиска_(StandardGlobalSearchType).md).   
Вид стандартного поиска.

Вариант синтаксиса: По обработчику

Синтаксис:

Найти(<ИмяПроцедуры>, <Модуль>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортной процедуры общего модуля.

<Модуль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя общего модуля.

Возвращаемое значение:

Тип: [ЭлементПланаГлобальногоПоиска](ЭлементПланаГлобальногоПоиска_(GlobalSearchPlanItem).md), [Неопределено](lang__def_Undefined.md).   
Указанный элемент плана глобального поиска.  
[Неопределено](lang__def_Undefined.md) - если такой элемент не найден.

Описание:

Получает элемент коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
