Описание:

Определяет уровень проверки качества документа после его сканирования в методах [ПоказатьСканированиеДокументов](СредстваМультимедиа.ПоказатьСканированиеДокументов_(MultimediaTools.ShowDocumentScanning).md) и [СканироватьДокументыАсинх](СредстваМультимедиа.СканироватьДокументыАсинх_(MultimediaTools.ScanDocumentsAsync).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
