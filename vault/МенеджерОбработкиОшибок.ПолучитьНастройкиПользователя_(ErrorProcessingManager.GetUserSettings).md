МенеджерОбработкиОшибок (ErrorProcessingManager)

ПолучитьНастройкиПользователя (GetUserSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьНастройкиПользователя(<ИмяПользователя>)

Параметры:

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, для которого получаются настройки.  
Если параметр не передан, то используется текущий пользователь.

Возвращаемое значение:

Тип: [НастройкиОбработкиОшибок](НастройкиОбработкиОшибок_(ErrorProcessingSettings).md).   

Описание:

Возвращает настройки обработки ошибки для переданного пользователя. Для доступа к параметрам другого пользователя требуется наличие права "Администрирование данных".

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.17.
