РасширениеКонфигурации (ConfigurationExtension)

ИспользоватьОсновныеРолиДляВсехПользователей (UseDefaultRolesForAllUsers)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то для сеансов, в которых подключено расширение любой пользователь будет обладать правами из основных ролей данного расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
