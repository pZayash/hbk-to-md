ИнтернетПочта (InternetMail)

ОтменитьУдалениеСообщений (UndeleteMessages)

Доступен, начиная с версии 8.2.

Синтаксис:

ОтменитьУдалениеСообщений(<МассивУдаляемыхДанных>)

Параметры:

<МассивУдаляемыхДанных> (обязательный)

Тип: [Массив](Массив.md).   
Cодержит либо заголовки сообщений, либо серверные идентификаторы сообщений, либо массив порядковых номеров почтовых сообщений, для которых необходимо отменить действие удаления с сервера.

Описание:

Снимает пометку удаления для [IMAP](ПротоколИнтернетПочты.IMAP.md) письма. Позволяет отменить действия метода [УдалитьСообщения](ИнтернетПочта.УдалитьСообщения.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Пометки не снимаются при работе по [POP3](ПротоколИнтернетПочты.POP3.md) протоколу.

См. также:

[ИнтернетПочта](ИнтернетПочта.md), метод [ОтменитьУдалениеСообщений](ИнтернетПочта.ОтменитьУдалениеСообщений.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
