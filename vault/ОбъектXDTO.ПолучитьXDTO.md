ОбъектXDTO (XDTODataObject)

ПолучитьXDTO (GetXDTO)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьXDTO(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя свойства, значение которого необходимо получить.

Вариант синтаксиса: По свойству

Синтаксис:

ПолучитьXDTO(<Свойство>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO.md).   
Свойство XDTO, значение которого необходимо получить.

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO.md), [ЗначениеXDTO](ЗначениеXDTO.md), Неопределено.   

Описание:

Получает значение XDTO для указанного свойства.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
