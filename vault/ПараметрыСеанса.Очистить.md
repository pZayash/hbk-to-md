ПараметрыСеанса (SessionParameters)

Очистить (Clear)

Доступен, начиная с версии 8.2.

Синтаксис:

Очистить(<ОчищаемыеПараметры>)

Параметры:

<ОчищаемыеПараметры> (необязательный)

Тип: [Массив](Массив.md), Неопределено.   
Если параметр опущен или имеет значение Неопределено, то все параметры сеанса будут сброшены в "Не установлено".  
Если параметр имеет значение типа [Массив](Массив.md), то элементами массива должны быть строки, каждая из которых должна содержать имя параметра сеанса. Все перечисленные параметры сеанса получат состояние "Не установлено".  
Значение по умолчанию: Неопределено.

Описание:

Переводит параметры сеанса в состояние "Не установлено".  
Если параметр <ОчищаемыеПараметры> опущен или его значение равно Неопределено, то все параметры сеанса, определенные в конфигурации, переводятся в состояние "Не установлено". Иначе состояние "Не установлено" получают только те параметры сеанса, имена которых перечислены в значении параметра <ОчищаемыеПараметры>. Для выполнения метода текущему пользователю информационной базы требуются права на установку каждого параметра сеанса из <ОчищаемыеПараметры>. Если параметр <ОчищаемыеПараметры> не указан, требуются права на установку всех параметров сеанса, определенных в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ПараметрыСеанса.Очистить();  ОчищаемыеПараметры = Новый Массив; ОчищаемыеПараметры.Добавить("ПараметрСеанса1"); ПараметрыСеанса.Очистить(ОчищаемыеПараметры); |

Использование в версии:

Доступен, начиная с версии 8.2.
