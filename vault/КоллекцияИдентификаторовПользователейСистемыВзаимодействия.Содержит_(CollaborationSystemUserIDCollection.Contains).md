Содержит (Contains)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Содержит(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).

Тип: [Булево](lang__Булево_(Boolean).md).

Определяет, находится ли элемент в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
