# COMSafeArray.IsResizable

COMSafeArray (COMSafeArray)

IsResizable (IsResizable)

Доступен, начиная с версии 8.0.

Синтаксис:

IsResizable()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - у массива можно менять нижнюю границу и количество элементов по старшему измерению; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, можно ли изменять размерности массива.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| МожноМенять = Массив.IsResizable(); Если МожноМенять Тогда     Массив.Resize(0, 10); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
