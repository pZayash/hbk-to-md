РежимЗамещения (ReplacementMode)

Замещение (Replace)

Доступен, начиная с версии 8.3.25.

Описание:

Перед записью набора записей существующие записи регистра, соответствующие текущему отбору, будут удалены.

Использование в версии:

Доступен, начиная с версии 8.3.25.
