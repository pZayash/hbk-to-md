МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьТипыВнешнихСистем (GetExternalSystemTypes)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТипыВнешнихСистем()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Массив элементов типа [Строка](lang__def_String.md).

Описание:

Возвращает поддерживаемые сервером взаимодействия типы внешних систем.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
