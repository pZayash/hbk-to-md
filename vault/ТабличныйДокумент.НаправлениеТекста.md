ТабличныйДокумент (SpreadsheetDocument)

НаправлениеТекста (TextDirection)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Содержит направление текста табличного документа.  
Значение по умолчанию: [СлеваНаправо](НаправлениеТекста.СлеваНаправо.md) или [СправаНалево](НаправлениеТекста.СправаНалево.md) в зависимости от направления текста текущего клиентского приложения на момент создания табличного документа. Если невозможно определить направление текста текущего клиентского приложения, то значение по умолчанию – [СлеваНаправо](НаправлениеТекста.СлеваНаправо.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Значение свойства влияет на отображение документа при экспорте (в тонком, толстом и веб-клиентах).  
Значение свойства влияет на отображение документа в веб-клиенте, а также при выводе документа на печать в веб-клиенте.

Использование в версии:

Доступен, начиная с версии 8.3.17.
