ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

АутентификацияОС (OSAuthentication)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Содержит признак необходимости соединения с внешним источником данных при помощи аутентификации операционной системы.  
Неопределено - признак необходимости аутентификации операционной системы выбирается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - соединение необходимо выполнить при помощи аутентификации операционной системы. Ложь - аутентификация не используется.

Использование в версии:

Доступен, начиная с версии 8.3.1.
