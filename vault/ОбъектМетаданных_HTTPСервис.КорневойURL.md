ОбъектМетаданных: HTTPСервис (MetadataObject: HTTPService)

КорневойURL (RootURL)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Корневой URL-адрес сервиса - первый сегмент в URL-адресе сервиса после "hs".  
Строка может содержать: буквы, цифры, "-", "~", "$", "(", ")", "\_",".".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
