ФабрикаXDTO (XDTOFactory)

ЭкспортМоделиXDTO (ExportXDTOModel)

Доступен, начиная с версии 8.1.

Синтаксис:

ЭкспортМоделиXDTO(<URIПространствИмен>)

Параметры:

<URIПространствИмен> (обязательный)

Тип: [Строка](lang__def_String.md), [Массив](Массив_(Array).md).   
Массив строк или строка, соответствующая URI пространства имен экспортируемых пакетов.

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Объект XDTO, имеющий тип {http://v8.1c.ru/8.1/xdto}Model.

Описание:

Выполняет экспорт пакетов XDTO, имеющих указанные URI пространства имен в модели XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
