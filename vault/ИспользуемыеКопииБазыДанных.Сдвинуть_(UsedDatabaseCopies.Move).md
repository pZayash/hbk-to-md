ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [Число](lang__def_Number.md), [ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных_(UsedDatabaseCopy).md).   
Используемая копия базы данных или позиция копии в коллекции.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение элемента в коллекции.

Описание:

Сдвигает позицию используемой копии базы данных в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
