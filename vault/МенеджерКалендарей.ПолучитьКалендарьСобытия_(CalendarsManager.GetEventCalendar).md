МенеджерКалендарей (CalendarsManager)

ПолучитьКалендарьСобытия (GetEventCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьКалендарьСобытия(<Событие>)

Параметры:

<Событие> (обязательный)

Тип: [ЛокальныйКлючСобытияКалендаря](ЛокальныйКлючСобытияКалендаря_(LocalCalendarEventKey).md).   
Локальный ключ события календаря.

Возвращаемое значение:

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря_(LocalCalendarKey).md).   

Описание:

Получает календарь, которому принадлежит событие.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.4.
