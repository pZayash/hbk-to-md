Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfileCOMClass (RegSecurityProfileCOMClass)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfileCOMClass(<Кластер>, <ИмяПрофиля>, <COMКласс>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должен быть зарегистрирован разрешенный [COM-класс](COM-класс.md).

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<COMКласс> (обязательный)

Тип: [COM-класс](COM-класс.md).   
Регистрируемый COM-класс.

Описание:

Выполняет регистрацию нового разрешенного [COM-класс](COM-класс.md) или обновление существующего. [COM-класс](COM-класс.md) идентифицируется своим идентификатором, по которому он может быть создан операционной системой, и/или текстовым представлением моникера.

Доступность:

Интеграция.

Примечание:

Если разрешенный [COM-класс](COM-класс.md) существует, то выполняется обновление, в противном случае - добавление.  
Если профиль не найден, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
