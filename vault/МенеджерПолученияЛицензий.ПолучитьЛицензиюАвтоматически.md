МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьЛицензиюАвтоматически (AcquireLicenseAutomatically)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьЛицензиюАвтоматически(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md).   
Данные для запроса в центр лицензирования на получение лицензии.

Описание:

Выполняет обращение через Интернет к центру лицензирования и создает файл лицензии на основании параметров текущего компьютера, данных о лицензии и ответа центра лицензирования.   
Файл лицензии помещается на текущий компьютер или на компьютер, определяемый адресом агента сервера 1С:Предприятия, в каталог поиска программной лицензии для всех пользователей или для текущего пользователя.   
При любой ошибке генерируется исключение, и файл лицензии не создается.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
