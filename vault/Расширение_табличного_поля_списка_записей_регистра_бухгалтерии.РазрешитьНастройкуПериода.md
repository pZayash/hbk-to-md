Расширение табличного поля списка записей регистра бухгалтерии (Accounting register record list table box extension)

РазрешитьНастройкуПериода (EnablePeriodSetting)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если Истина, то команда "Установить интервал дат" доступна даже в том случае, если запрещен интерактивный отбор по дате.

Доступность:

Толстый клиент.

Примечание:

Значение по умолчанию Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
