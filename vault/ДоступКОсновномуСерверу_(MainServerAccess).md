Описание:

Объект предназначен для доступа с автономного сервера к основному серверу. Свойствами этого объекта являются серверные общие модули с установленным признаком "Вызов сервера".

Доступность:

Мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
