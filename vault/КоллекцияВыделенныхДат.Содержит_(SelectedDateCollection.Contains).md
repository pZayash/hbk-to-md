КоллекцияВыделенныхДат (SelectedDateCollection)

Содержит (Contains)

Доступен, начиная с версии 8.0.

Синтаксис:

Содержит(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата, наличие которой в коллекции необходимо проверить.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - дата содержится в коллекции; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, содержится ли указанная дата в коллекции.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
