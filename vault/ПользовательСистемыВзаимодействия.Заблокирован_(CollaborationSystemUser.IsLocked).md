ПользовательСистемыВзаимодействия (CollaborationSystemUser)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), пользователю недоступны функции системы взаимодействия:

- установка соединения с сервером системы взаимодействия,
- получение push-уведомлений,
- видеозвонки.

При попытке отправить сообщение такому пользователю, добавить его в список участников обсуждения или в видеоконференцию, будет вызвано исключение.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).  
Изменение свойства возможно только в привилегированном режиме или при наличии у текущего пользователя права "Администрирование данных".

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
