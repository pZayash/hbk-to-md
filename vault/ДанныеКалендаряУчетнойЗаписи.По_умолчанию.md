ДанныеКалендаряУчетнойЗаписи (AccountCalendarData)

По умолчанию

Доступен, начиная с версии 8.3.4.

Синтаксис:

Новый ДанныеКалендаряУчетнойЗаписи(<Данные>, <УчетнаяЗапись>, <ГлобальныйКлючКалендаря>)

Параметры:

<Данные> (обязательный)

Тип: [ДанныеКалендаря](ДанныеКалендаря_(CalendarData).md).   
Данные календаря.

<УчетнаяЗапись> (обязательный)

Тип: [УчетнаяЗаписьКалендарей](УчетнаяЗаписьКалендарей_(CalendarAccount).md).   
Учетная запись, к которой привязан календарь.

<ГлобальныйКлючКалендаря> (обязательный)

Тип: [Строка](lang__def_String.md).   
Глобальный ключ календаря во внешнем хранилище календарей.

Описание:

Создает данные календаря учетной записи.

Использование в версии:

Доступен, начиная с версии 8.3.4.
