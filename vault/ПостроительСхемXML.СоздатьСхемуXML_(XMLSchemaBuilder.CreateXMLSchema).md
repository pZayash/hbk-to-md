ПостроительСхемXML (XMLSchemaBuilder)

СоздатьСхемуXML (CreateXMLSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьСхемуXML(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md).   
Элемент DOM, на основе которого необходимо создать схему XML.

Возвращаемое значение:

Тип: [СхемаXML](СхемаXML_(XMLSchema).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md), если структура элемента не соответствует схеме.

Описание:

Создает схему на основе элемента DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
