Глобальный контекст (Global context)

ПолучитьПроверкуСложностиПаролейПользователей (GetUserPasswordStrengthCheck)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьПроверкуСложностиПаролейПользователей()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ограничения установлены; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, установлены ли ограничения на сложность паролей пользователей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
