ДокументHTML (HTMLDocument)

СоздатьКомментарий (CreateComment)

Доступен, начиная с версии 8.2.

Синтаксис:

СоздатьКомментарий(<Комментарий>)

Параметры:

<Комментарий> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст комментария HTMLDOM.

Возвращаемое значение:

Тип: [КомментарийDOM](КомментарийDOM_(DOMComment).md).   

Описание:

Создает узел HTML - комментарий.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
