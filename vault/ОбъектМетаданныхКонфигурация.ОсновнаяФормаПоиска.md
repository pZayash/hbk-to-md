ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновнаяФормаПоиска (DefaultSearchForm)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Описывает общую форму, которая будет вызвана системой для выполнения полнотекстового поиска в данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если конфигурация была создана в версии 8.3.14 и ниже, то для перехода на версию 8.3.15 и выше значение данного свойства следует очистить.

Использование в версии:

Доступен, начиная с версии 8.3.3.
