ПользовательскиеНастройкиКомпоновкиДанных (DataCompositionUserSettings)

ДополнительныеСвойства (AdditionalProperties)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Дополнительные свойства настроек компоновки данных.  
Может использоваться в тех случаях, когда необходимо хранить некоторые значения, связанные с объектом.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
