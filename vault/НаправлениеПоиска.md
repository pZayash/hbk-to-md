# НаправлениеПоиска

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Системные_перечисления.md">Системные перечисления</a> › [Разные](Разные.md)

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).

Значения

[СКонца (FromEnd)](НаправлениеПоиска.СКонца.md)  
[СНачала (FromBegin)](НаправлениеПоиска.СНачала.md)  

Описание:

Определяет направление поиска подстроки в строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Найти](Глобальный_контекст.Найти.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).
