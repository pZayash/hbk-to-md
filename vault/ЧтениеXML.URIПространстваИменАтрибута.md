ЧтениеXML (XMLReader)

URIПространстваИменАтрибута (AttributeNamespaceURI)

Доступен, начиная с версии 8.1.

Синтаксис:

URIПространстваИменАтрибута(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс атрибута.

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает URI пространства имен атрибута по индексу. Так как атрибуты могут быть только у элемента XML, доступ к ним может осуществляться только при прочтении узла, имеющего тип [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
