ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает задачу в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В процессе записи проверяется уникальность кода, если данный режим установлен в конфигурации.

Пример:

|  |
| --- |
| Попытка     Объект.Записать(); Исключение     Предупреждение(НСтр("ru = 'Не удалось записать объект';"      + " en = 'Can't write object -'") + " '"       + Объект + '"!      |" + ОписаниеОшибки(), 60); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
