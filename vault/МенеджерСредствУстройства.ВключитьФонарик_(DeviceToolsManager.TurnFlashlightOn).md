МенеджерСредствУстройства (DeviceToolsManager)

ВключитьФонарик (TurnFlashlightOn)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ВключитьФонарик(<Включить>)

Параметры:

<Включить> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли включен фонарик:

- [Истина](lang__def_BooleanTrue.md) - включить фонарик;
- [Ложь](lang__def_BooleanFalse.md) - выключить фонарик.

Описание:

Управляет состоянием фонарика.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом метода следует использовать метод [ПоддерживаетсяФонарик](МенеджерСредствУстройства.ПоддерживаетсяФонарик_(DeviceToolsManager.FlashlightSupported).md).

Доступность:

Толстый клиент.

См. также:

[МенеджерСредствУстройства](МенеджерСредствУстройства_(DeviceToolsManager).md), метод [ПоддерживаетсяФонарик](МенеджерСредствУстройства.ПоддерживаетсяФонарик_(DeviceToolsManager.FlashlightSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
