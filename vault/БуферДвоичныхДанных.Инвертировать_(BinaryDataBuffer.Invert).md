БуферДвоичныхДанных (BinaryDataBuffer)

Инвертировать (Invert)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Инвертировать(<Позиция>, <Количество>)

Параметры:

<Позиция> (необязательный)

Тип: [Число](lang__def_Number.md).   
Начальная позиция в буфере.  
Значение по умолчанию: 0.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется инвертировать.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Инвертировать заданное количество байтов в буфере, начиная с указанной позиции. Если позиция не указана, байты инвертируются с начала буфера. Если количество не указано, то выполняется инверсия байтов от начальной позиции до конца буфера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
