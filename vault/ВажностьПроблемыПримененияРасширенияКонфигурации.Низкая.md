ВажностьПроблемыПримененияРасширенияКонфигурации (ConfigurationExtensionApplicationIssueSeverity)

Низкая (Low)

Доступен, начиная с версии 8.3.12.

Описание:

Проблема применения расширения конфигурации с низкой важностью. Наличие такой проблемы может возникнуть, например, после установки свойства [Активно](РасширениеКонфигурации.Активно.md) в значение Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.12.
