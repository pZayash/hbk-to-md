# КоллекцияТекущихПериодовОтображенияПланировщика.Вставить

КоллекцияТекущихПериодовОтображенияПланировщика (PlannerCurrentRepresentationPeriodCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Вставить(<Индекс>, <Начало>, <Конец>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить период отображения.

<Начало> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Начало периода отображения.

<Конец> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Конец периода отображения.

Возвращаемое значение:

Тип: [ПериодОтображенияПланировщика](ПериодОтображенияПланировщика_(PlannerRepresentationPeriod).md).   
Вставленный элемент

Описание:

Вставляет период отображения в коллекцию по указанному индексу

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
