Расширение поля формы для поля картинки (Form field extension for a picture field)

Нажатие (Click)

Доступен, начиная с версии 8.2.

Синтаксис:

Нажатие(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при нажатии на поле картинки при установленном свойстве [Гиперссылка](ПолеКартинки.Гиперссылка_(PictureBox.Hyperlink).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
