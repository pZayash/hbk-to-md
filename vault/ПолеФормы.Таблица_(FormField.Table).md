ПолеФормы (FormField)

Таблица (Table)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [ТаблицаФормы](ТаблицаФормы_(FormTable).md).   
Для объекта с значением свойства [Вид](ПолеФормы.Вид_(FormField.Type).md), равным [ПолеДиаграммыГанта](ВидПоляФормы.ПолеДиаграммыГанта_(FormFieldType.GanttChartField).md), возвращается объект типа [ТаблицаФормы](ТаблицаФормы_(FormTable).md), содержащий предопределенную таблицу поля, для остальных - выбрасывается исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
