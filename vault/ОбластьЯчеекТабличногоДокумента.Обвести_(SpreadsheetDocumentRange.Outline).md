ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Обвести (Outline)

Доступен, начиная с версии 8.2.

Синтаксис:

Обвести(<ГраницаСлева>, <ГраницаСверху>, <ГраницаСправа>, <ГраницаСнизу>)

Параметры:

<ГраницаСлева> (необязательный)

Тип: [Линия](Линия_(Line).md).   
Задает тип границы слева. Если параметр не указан, граница слева не рисуется.

<ГраницаСверху> (необязательный)

Тип: [Линия](Линия_(Line).md).   
Задает тип верхней границы. Если параметр не указан, верхняя граница не рисуется.

<ГраницаСправа> (необязательный)

Тип: [Линия](Линия_(Line).md).   
Задает тип границы справа. Если параметр не указан, граница справа не рисуется.

<ГраницаСнизу> (необязательный)

Тип: [Линия](Линия_(Line).md).   
Задает тип нижней границы. Если параметр не указан, нижняя граница не рисуется.

Описание:

Обводит область рамкой.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Линия = Новый Линия(ТипЛинииЯчейкиТабличногоДокумента.Сплошная,2); Область = ДокументМакет.Область("Шапка"); Область.Обвести(Линия, Линия, Линия, Линия); |

Использование в версии:

Доступен, начиная с версии 8.2.
