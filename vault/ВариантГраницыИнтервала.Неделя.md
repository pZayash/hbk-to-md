ВариантГраницыИнтервала (IntervalBoundVariant)

Неделя (Week)

Доступен, начиная с версии 8.0.

Описание:

Начало интервала задается датой начала недели рабочей даты, конец интервала - датой конца недели рабочей даты.

Использование в версии:

Доступен, начиная с версии 8.0.
