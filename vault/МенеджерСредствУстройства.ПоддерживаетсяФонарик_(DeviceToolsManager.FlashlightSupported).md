МенеджерСредствУстройства (DeviceToolsManager)

ПоддерживаетсяФонарик (FlashlightSupported)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПоддерживаетсяФонарик()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md) в следующих случаях:

- на Android – присутствует фотовспышка,
- на iOS и Windows Mobile – присутствует фотовспышка и поддерживается режим фонарика.

Описание:

Проверяет наличие на устройстве вспышки (она используется в качестве фонарика).

Доступность:

Толстый клиент.

См. также:

Использование в версии:

Доступен, начиная с версии 8.3.13.
