МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

Удалить (Delete)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Удалить(<Пространство>)

Параметры:

<Пространство> (обязательный)

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных.md).   
Менеджер табличного пространства.

Описание:

Удаляет пустое табличное пространство. Может быть вызван только для пустого табличного пространства, иначе будет сгенерировано исключение. Удаление пространства выполняется в момент вызова данного метода.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
