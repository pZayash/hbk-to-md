КонструкторЗапроса (QueryWizard)

РежимКомпоновкиДанных (DataCompositionMode)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает режим компоновки данных в конструкторе запроса. Закладки "Характеристики" и "Компоновка данных" становятся доступны.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - конструктор запроса в режиме компоновки данных.

Использование в версии:

Доступен, начиная с версии 8.1.
