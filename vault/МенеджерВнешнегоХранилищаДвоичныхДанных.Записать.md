МенеджерВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageManager)

Записать (Write)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Записать()

Возвращаемое значение:

Нет

Описание:

Сохраняет изменения менеджера внешнего хранилища двоичных данных в базу данных.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.  
Если режим хранилища по умолчанию на другом внешнем либо встроенном хранилище двоичных данных установлен в Истина, то при попытке установить [ХранилищеПоУмолчанию](МенеджерВнешнегоХранилищаДвоичныхДанных.ХранилищеПоУмолчанию.md) в Истина будет сгенерировано исключение.  
Один и тот же объект метаданных в [СоставХранимыхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.СоставХранимыхДанных.md) не может быть настроен для использования с разными хранилищами двоичных данных в [СоставХранимыхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.СоставХранимыхДанных.md). Если [СоставХранимыхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.СоставХранимыхДанных.md) содержит объект метаданных, который уже был настроен для использования в другом внешнем или встроенном хранилище двоичных данных, будет сгенерировано исключение.

Доступность:

Сервер.

Примечание:

[МинимальныйРазмерЗаписываемыхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.МинимальныйРазмерЗаписываемыхДанных.md) должен быть не менее 2048 байт. При попытке установить меньшее значение, устанавливается значение 2048 байт.
