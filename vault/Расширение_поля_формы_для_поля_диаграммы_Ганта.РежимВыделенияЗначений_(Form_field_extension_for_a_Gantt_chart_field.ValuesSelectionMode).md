# Расширение поля формы для поля диаграммы Ганта.РежимВыделенияЗначений

Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

РежимВыделенияЗначений (ValuesSelectionMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [РежимВыделенияЗначенийДиаграммыГанта](РежимВыделенияЗначенийДиаграммыГанта_(GanttChartValuesSelectionMode).md).   
Определяет режим выделения значений в диаграмме. Значение по умолчанию - [Авто](РежимВыделенияЗначенийДиаграммыГанта.Авто_(GanttChartValuesSelectionMode.Auto).md).  
Если установлено значение [Нет](РежимВыделенияЗначенийДиаграммыГанта.Нет_(GanttChartValuesSelectionMode.None).md), то свойства [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md) таблицы принимает значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный_(GanttChartValuesSelectionMode.Single).md).  
Если установлено значение [Авто](РежимВыделенияЗначенийДиаграммыГанта.Авто_(GanttChartValuesSelectionMode.Auto).md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md) в режиме совместимости с [Версия8\_3\_22](РежимСовместимости.Версия8_3_22_(CompatibilityMode.Version8_3_22).md) и ниже устанавливается значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный_(GanttChartValuesSelectionMode.Single).md). В остальных случаях устанавливается значение {ссылка:Атрибуты; 13383; Множественный}.  
Если данное свойство принимает значение [Множественный](РежимВыделенияЗначенийДиаграммыГанта.Множественный_(GanttChartValuesSelectionMode.Multiple).md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md) таблицы также принимает значение [Множественный](РежимВыделенияТаблицы.Множественный_(TableSelectionMode.MultiRow).md), и наоборот.  
Если данное свойство принимает значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный_(GanttChartValuesSelectionMode.Single).md), то и свойство [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md) таблицы также принимает значение [Одиночный](РежимВыделенияТаблицы.Одиночный_(TableSelectionMode.SingleRow).md), и наоборот.   
Если установлено значение [Нет](РежимВыделенияЗначенийДиаграммыГанта.Нет_(GanttChartValuesSelectionMode.None).md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md) таблицы принимает значение [Одиночный](РежимВыделенияТаблицы.Одиночный_(TableSelectionMode.SingleRow).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
