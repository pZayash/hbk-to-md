ПоложениеПодписейКДиаграмме (ChartLabelLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Место для расположения подписи подбирается автоматически:

- для уравнения линии тренда - [СвободноеМесто](ПоложениеПодписейКДиаграмме.СвободноеМесто_(ChartLabelLocation.EmptySpace).md),
- для столбца диаграммы - [КрайАвто](ПоложениеПодписейКДиаграмме.КрайАвто_(ChartLabelLocation.EdgeAuto).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
