# ЗащищенноеСоединениеNSS

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › <a href="obsidian://open?file=objects__catalog63__catalog578.md">Интернет</a> › [Сертификаты защищенных соединений](Сертификаты_защищенных_соединений.md)

Не рекомендуется использовать, начиная с версии 8.3.8.

Рекомендуется использовать:

- [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL.md)

Доступен, начиная с версии 8.3.1.

Описание:

Объект защищенного соединения NSS.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| nss1 = Новый ЗащищенноеСоединениеNSS(                 "c:/nss/nssdb","password", Истина, "user1");  nss2 = Новый ЗащищенноеСоединениеNSS(     "C:/Users/usrnm/AppData/Roaming/Mozilla/Firefox/Profiles/usrnm"); |

Использование в версии:

Доступен, начиная с версии 8.3.1.

Не рекомендуется использовать, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
