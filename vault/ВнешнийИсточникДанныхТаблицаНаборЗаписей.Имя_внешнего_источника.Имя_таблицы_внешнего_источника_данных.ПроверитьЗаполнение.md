ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: Булево.   
Истина - ошибок не обнаружено, Ложь - в противном случае.

Описание:

Метод вызывает проверку заполнения полей записей. Для полей, у которых свойство [ПроверкаЗаполнения](ОбъектМетаданных_Поле.ПроверкаЗаполнения.md) установлено в значение [ВыдаватьОшибку](ПроверкаЗаполнения.ВыдаватьОшибку.md) и они при этом не заполнены, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
