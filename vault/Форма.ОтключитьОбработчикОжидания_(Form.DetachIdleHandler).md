Форма (Form)

ОтключитьОбработчикОжидания (DetachIdleHandler)

Доступен, начиная с версии 8.0.

Синтаксис:

ОтключитьОбработчикОжидания(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры, которую необходимо исключить из обработки ожидания.

Описание:

Отключает ранее подключенную процедуру - обработчик ожидания, подключенный ранее методом формы [ПодключитьОбработчикОжидания](Форма.ПодключитьОбработчикОжидания_(Form.AttachIdleHandler).md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ОтключитьОбработчикОжидания("ПриОжидании"); |

См. также:

[Форма](Форма_(Form).md), метод [ПодключитьОбработчикОжидания](Форма.ПодключитьОбработчикОжидания_(Form.AttachIdleHandler).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
