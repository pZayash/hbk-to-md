ЭлементЯкорьHTML (HTMLAnchorElement)

Отношение (Rel)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Свойство описывает отношение из текущего документа к якорю, заданному атрибутом <href>.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
