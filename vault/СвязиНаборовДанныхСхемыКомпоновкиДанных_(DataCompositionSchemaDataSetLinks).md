# СвязиНаборовДанныхСхемыКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Схема компоновки данных](Схема_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СвязьНаборовДанныхСхемыКомпоновкиДанных](СвязьНаборовДанныхСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetLink).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются связи.  
Возможно обращение к связь посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит описания связей между наборами данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
