СредстваМультимедиа (MultimediaTools)

ПоказатьСканированиеДокументов (ShowDocumentScanning)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоказатьСканированиеДокументов(<Заголовок>, <ОбработчикСканирования>, <ОбработчикЗавершенияСканирования>, <ПараметрыСканирования>)

Параметры:

<Заголовок> (необязательный)

Тип: Строка.   
Текст заголовка, который будет показан в окне сканирования документов.

<ОбработчикСканирования> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана при успешном сканировании со следующими параметрами:

- <Результат> - тип [СтраницаСканированияДокументов](СтраницаСканированияДокументов.md). Возвращает данные сканирования страницы документа.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md)

<ОбработчикЗавершенияСканирования> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана при выходе из интерфейса сканирования после успешного сканирования документа, серии документов или после закрытия пользователем:

- <ИдентификаторыСтраниц> - массив значений типа Число или Неопределено. Возвращает идентификаторы страниц документа, в порядке заданным пользователем, или Неопределено, если пользователь закрыл интерфейс сканирования документов.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md)

<ПараметрыСканирования> (необязательный)

Тип: [ПараметрыСканированияДокументов](ПараметрыСканированияДокументов.md).   
Набор входных параметров для сканирования документов.

Описание:

Выводит на экран интерфейс сканирования документов.   
Пользователь фотографирует страницы документа с помощью камеры, встроенной в мобильное устройство. После получения фотоснимка выполняется автоматическое сканирование на наличие страницы документа. Найденная страница документа вырезается в отдельную картинку, которая обрабатывается выбранным пользователем фильтром из списка, указанного в параметрах сканирования.   
Метод позволяет выполнить сканирование как отдельных документов, так и серии страниц одного документа.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяСканированиеДокументов](СредстваМультимедиа.ПоддерживаетсяСканированиеДокументов.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
