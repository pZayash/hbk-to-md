ФиксированнаяКоллекция (FixedCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<НаименованиеЭлемента>)

Параметры:

<НаименованиеЭлемента> (обязательный)

Тип: [Строка](lang__def_String.md).   
Наименование элемента коллекции.

Возвращаемое значение:

Тип: Произвольный.   
Если элемента с таким именем не существует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск значения элемента коллекции по его имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
