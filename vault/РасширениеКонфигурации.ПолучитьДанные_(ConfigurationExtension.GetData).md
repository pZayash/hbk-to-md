РасширениеКонфигурации (ConfigurationExtension)

ПолучитьДанные (GetData)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Для нового расширения возвращает [Неопределено](lang__def_Undefined.md) либо [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), если они были переданы при создании объекта.

Описание:

Возвращает текущие используемые данные расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерРасширенийКонфигурации](МенеджерРасширенийКонфигурации_(ConfigurationExtensionsManager).md), метод [Создать](МенеджерРасширенийКонфигурации.Создать_(ConfigurationExtensionsManager.Create).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
