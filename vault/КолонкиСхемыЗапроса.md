Элементы коллекции:

[КолонкаСхемыЗапроса](КолонкаСхемыЗапроса.md), [КолонкаВложеннаяТаблицаСхемыЗапроса](КолонкаВложеннаяТаблицаСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонке посредством оператора [...]. В качестве аргумента передается индекс.
