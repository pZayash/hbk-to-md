Расширение формы записи регистра сведений (Information register record form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.ПриЗаписи.md) объекта [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
