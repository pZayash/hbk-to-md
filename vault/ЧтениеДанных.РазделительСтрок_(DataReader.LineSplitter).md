# ЧтениеДанных.РазделительСтрок

ЧтениеДанных (DataReader)

РазделительСтрок (LineSplitter)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Разделитель строк по-умолчанию. Это свойство влияет на поведение метода [ПрочитатьСтроку](ЧтениеДанных.ПрочитатьСтроку_(DataReader.ReadLine).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЧтениеДанных](ЧтениеДанных_(DataReader).md), метод [ПрочитатьСтроку](ЧтениеДанных.ПрочитатьСтроку_(DataReader.ReadLine).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
