Описание:

Объект предназначен для последовательного чтения XML данных, записанных в бинарном формате Fast Infoset (Recommendation X.891).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
