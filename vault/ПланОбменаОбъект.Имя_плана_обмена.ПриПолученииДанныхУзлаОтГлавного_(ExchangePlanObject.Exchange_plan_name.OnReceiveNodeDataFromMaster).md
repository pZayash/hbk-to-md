# ПланОбменаОбъект.<Имя плана обмена>.ПриПолученииДанныхУзлаОтГлавного

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПриПолученииДанныхУзлаОтГлавного (OnReceiveNodeDataFromMaster)

Доступен, начиная с версии 8.1.

Синтаксис:

ПриПолученииДанныхУзлаОтГлавного(<ЭлементДанных>, <Игнорировать>)

Параметры:

<ЭлементДанных>

Тип: [ПланОбменаОбъект.\<Имя плана обмена\>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md).   
Узел плана обмена, прочитанный из сообщения обмена.

<Игнорировать>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак, управляющий записью узла плана обмена в базу данных. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), узел плана обмена в базу данных записан не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает для обоих узлов плана обмена (источника и приемника сообщения обмена) после их загрузки из сообщения обмена в рамках распределенной информационной базой, но перед записью в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.1.
