Описание:

Предназначен для хранения топологических данных и связанных с ними дополнительной информации.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/map}. Имя типа XDTO: GeographicalSchema.
