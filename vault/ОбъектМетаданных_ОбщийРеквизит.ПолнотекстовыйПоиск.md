ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

ПолнотекстовыйПоиск (FullTextSearch)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеПолнотекстовогоПоиска](ИспользованиеПолнотекстовогоПоиска.md).   
Определяет, использовать или нет данные этого объекта метаданных в полнотекстовом поиске.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для всех реквизитов, кроме реквизитов, описывающих отчеты, обработки и табличные части отчетов и обработок.

Использование в версии:

Доступен, начиная с версии 8.2.
