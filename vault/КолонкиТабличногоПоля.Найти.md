КолонкиТабличногоПоля (TableBoxColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Наименование>)

Параметры:

<Наименование> (обязательный)

Тип: Строка.   
Наименование колонки.

Возвращаемое значение:

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля.md), Неопределено.   
Если колонки с таким именем нет, то возвращается значение Неопределено.

Описание:

Осуществляет поиск колонки с заданным именем.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Функция СравнитьТаблицыЗначений(ТЗ1, ТЗ2)     Если ТипЗнч(ТЗ1) <> Тип("ТаблицаЗначений") ИЛИ          ТипЗнч(ТЗ2) <> Тип("ТаблицаЗначений") Тогда         Возврат Ложь;     КонецЕсли;      Если ТЗ1.Количество() <> ТЗ2.Количество() Тогда         Возврат Ложь;     КонецЕсли;      Если ТЗ1.Колонки.Количество() <> ТЗ2.Колонки.Количество() Тогда         Возврат Ложь;     КонецЕсли;       Для каждого Колонка Из ТЗ1.Колонки Цикл         Если ТЗ2.Колонки.Найти(Колонка.Имя) = Неопределено Тогда             Возврат Ложь;         КонецЕсли;         Для каждого СтрокаТаблицы Из ТЗ1 Цикл             // сравнение строк таблиц              // ...          КонецЦикла;      КонецЦикла;      Возврат Истина; КонецФункции // СравнитьТаблицыЗначений() |

Использование в версии:

Доступен, начиная с версии 8.0.
