ЭлементГрафическойСхемыОбработка (GraphicalSchemaItemProcessing)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона элемента графической схемы. Если установлен [АвтоЦвет](ВидЦвета.АвтоЦвет_(ColorType.AutoColor).md), то используется цвет, принятый для данного элемента графической схемы (зеленый для точки старта, желтый для точки действия, красный для условия и т.д.).

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ЭлементГрафическойСхемыОбработка.ПрозрачныйФон_(GraphicalSchemaItemProcessing.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
