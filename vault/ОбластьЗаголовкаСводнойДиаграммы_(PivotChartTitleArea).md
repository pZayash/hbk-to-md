Описание:

Используется для доступа к области заголовка сводной диаграммы для настройки параметров оформления. В области заголовка может содержаться название сводной диаграммы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Диаграмма](Диаграмма_(Chart).md), свойство [ОбластьЗаголовка](Диаграмма.ОбластьЗаголовка_(Chart.TitleArea).md)

Доступен, начиная с версии 8.1.
