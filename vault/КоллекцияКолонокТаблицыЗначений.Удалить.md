КоллекцияКолонокТаблицыЗначений (ValueTableColumnCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число, [КолонкаТаблицыЗначений](КолонкаТаблицыЗначений.md), Строка.   
Удаляемая колонка (индекс, колонка или имя колонки).

Описание:

Удаляет заданную колонку из коллекции колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
