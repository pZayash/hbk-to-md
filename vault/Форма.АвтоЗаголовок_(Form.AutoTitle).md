Форма (Form)

АвтоЗаголовок (AutoTitle)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического формирования заголовка формы. В этом режиме к заголовку формы будет добавляться информация, определяемая основным реквизитом формы.  
Если [Истина](lang__def_BooleanTrue.md), то заголовок формы формируется системой.   
[Ложь](lang__def_BooleanFalse.md), то используется текст, указанный в [Заголовок](Форма.Заголовок_(Form.Caption).md).

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), свойство [Заголовок](Форма.Заголовок_(Form.Caption).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
