КолонкаСписка (ListColumn)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, необходимо ли удалять колонку при удалении ее из табличного поля.  
Истина - необходимо удалять.

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
