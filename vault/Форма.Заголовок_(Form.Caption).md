Форма (Form)

Заголовок (Caption)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст заголовка формы. По умолчанию содержит значение, заданное при редактировании формы в Конфигураторе.

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), свойство [АвтоЗаголовок](Форма.АвтоЗаголовок_(Form.AutoTitle).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
