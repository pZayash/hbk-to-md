ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

Индекс (Index)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Индексы](ЗапросВыбораСхемыЗапроса.Индексы_(QuerySchemaSelectQuery.Indexes).md)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [ВыраженияИндексаСхемыЗапроса](ВыраженияИндексаСхемыЗапроса_(QuerySchemaIndexExpressions).md).   
Содержит выражения индекса первого элемента коллекции [ИндексыСхемыЗапроса](ИндексыСхемыЗапроса_(QuerySchemaIndexes).md).  
Если коллекция [ИндексыСхемыЗапроса](ИндексыСхемыЗапроса_(QuerySchemaIndexes).md) содержит более одного элемента, то при обращении к данному свойству будет возвращено [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
