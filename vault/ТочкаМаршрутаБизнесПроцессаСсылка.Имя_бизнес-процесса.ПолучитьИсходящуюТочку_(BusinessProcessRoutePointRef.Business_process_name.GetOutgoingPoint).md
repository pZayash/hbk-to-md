# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПолучитьИсходящуюТочку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md) › [Методы](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса__Методы.md)

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПолучитьИсходящуюТочку (GetOutgoingPoint)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Для точки вида "ВыборВарианта"

Синтаксис:

ПолучитьИсходящуюТочку(<Вариант>)

Параметры:

<Вариант> (обязательный)

Тип: [ВариантТочкиМаршрутаБизнесПроцесса](ВариантТочкиМаршрутаБизнесПроцесса_(BusinessProcessRoutePointCase).md).   
Вариант.

Описание варианта метода:

Для точки вида [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта_(BusinessProcessRoutePointType.Switch).md) возможны несколько точек перехода. Число точек определяется числом вариантов.

Вариант синтаксиса: Для точки вида "Условие"

Синтаксис:

ПолучитьИсходящуюТочку(<Условие>)

Параметры:

<Условие> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает точку маршрута бизнес-процесса. Если значение параметра [Истина](lang__def_BooleanTrue.md), то требуется получить исходящую точку выхода по "Да", [Ложь](lang__def_BooleanFalse.md) - по "Нет".

Описание варианта метода:

Для точки вида [Условие](ВидТочкиМаршрутаБизнесПроцесса.Условие_(BusinessProcessRoutePointType.Condition).md) возможны две исходящие точки: при значении условия "Да" и "Нет". Параметр уточняет, какую точку требуется получить.

Вариант синтаксиса: Для всех точек, кроме точек вида "Разделение", "Условие", "ВыборВарианта".

Синтаксис:

ПолучитьИсходящуюТочку()

Описание варианта метода:

Для всех точек, кроме точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение_(BusinessProcessRoutePointType.Split).md), [Условие](ВидТочкиМаршрутаБизнесПроцесса.Условие_(BusinessProcessRoutePointType.Condition).md) и [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта_(BusinessProcessRoutePointType.Switch).md) существует только одна следующая точка.

Возвращаемое значение:

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   

Описание:

Получает точку маршрута бизнес-процесса, следующую за указанной точкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для всех точек маршрута бизнес-процесса, кроме точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение_(BusinessProcessRoutePointType.Split).md).

Использование в версии:

Доступен, начиная с версии 8.0.
