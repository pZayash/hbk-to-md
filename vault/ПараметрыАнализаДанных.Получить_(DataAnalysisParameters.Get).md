ПараметрыАнализаДанных (DataAnalysisParameters)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс параметра.

Возвращаемое значение:

Тип: [ПараметрАнализаДанных](ПараметрАнализаДанных_(DataAnalysisParameter).md).   

Описание:

Получает параметр по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
