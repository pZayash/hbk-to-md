Эффект (Effect)

Доступен, начиная с версии 8.2.

Только чтение.

Описание:

Тип: Число.   
Содержит значение эффекта от использования агрегатов при выполнении запросов по сравнению с таблицей оборотов (на момент построения списка).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
