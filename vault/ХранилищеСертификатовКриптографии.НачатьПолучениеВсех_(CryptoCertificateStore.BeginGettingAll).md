ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьПолучениеВсех (BeginGettingAll)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеВсех(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения получения сертификатов со следующими параметрами:

- <Сертификаты> - массив объектов [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Начинает получение массива всех сертификатов криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
