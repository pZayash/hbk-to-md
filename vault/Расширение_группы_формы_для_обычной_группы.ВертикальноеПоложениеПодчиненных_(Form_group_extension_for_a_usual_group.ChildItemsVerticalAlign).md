Расширение группы формы для обычной группы (Form group extension for a usual group)

ВертикальноеПоложениеПодчиненных (ChildItemsVerticalAlign)

Доступен, начиная с версии 8.3.7.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложениеЭлемента](ВертикальноеПоложениеЭлемента_(ItemVerticalAlign).md).   
Устанавливает вертикальное выравнивание подчиненных элементов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.7.
