ТестируемаяГруппаКомандногоИнтерфейса (TestedCommandInterfaceGroup)

Свернуть (Collapse)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Свернуть()

Описание:

Сворачивает разворачиваемую группу, если это действие доступно. В противном случае генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
