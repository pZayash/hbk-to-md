# ПланыСчетовМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Планы счетов](Планы_счетов.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПланСчетовМенеджер.\<Имя плана счетов\>](ПланСчетовМенеджер.Имя_плана_счетов.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов счетов.  
Возможно обращение к менеджеру плана счетов посредством оператора [...]. В качестве аргумента передается имя плана счетов, как оно задано в конфигурации.  

Свойства:

[\<Имя плана счетов\> (\<Chart of accounts name\>)](ПланыСчетовМенеджер.Имя_плана_счетов.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПланыСчетовМенеджер.ТипВсеСсылки.md)  

Описание:

Предназначен для управления планами счетов и предоставляет доступ к значениям типа [ПланСчетовМенеджер.](ПланСчетовМенеджер.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовМенеджер.Имя_плана_счетов.md)>. Доступ к объекту осуществляется через свойство глобального контекста [ПланыСчетов](Глобальный_контекст.ПланыСчетов.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ПланыСчетов](Глобальный_контекст.ПланыСчетов.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
