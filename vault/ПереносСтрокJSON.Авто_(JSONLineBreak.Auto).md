ПереносСтрокJSON (JSONLineBreak)

Авто (Auto)

Доступен, начиная с версии 8.3.6.

Описание:

Строки переносятся. Для переноса строк используются символы переноса строк в зависимости от текущей операционной системы (Windows или Unix).

Использование в версии:

Доступен, начиная с версии 8.3.6.
