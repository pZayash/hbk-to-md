Глобальный контекст (Global context)

ПолучитьОбновлениеКонфигурацииБазыДанных (GetDataBaseConfigurationUpdate)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьОбновлениеКонфигурацииБазыДанных()

Возвращаемое значение:

Тип: [ОбновлениеКонфигурацииБазыДанных](ОбновлениеКонфигурацииБазыДанных_(DataBaseConfigurationUpdate).md).   

Описание:

Получает информацию о состоянии фонового обновления конфигурации базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен пользователю с административными правами, а также пользователю, имеющему права обновления конфигурации базы данных.

Использование в версии:

Доступен, начиная с версии 8.3.1.
