ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак выделения текста отрицательного числа специальным цветом [ЦветОтрицательногоЧисла](ЦветаСтиля.ЦветОтрицательногоЧисла.md).

Доступность:

Толстый клиент.

Примечание:

Истина - текст выделяется.  
Применимо только для колонок с данными типа Число.

Использование в версии:

Доступен, начиная с версии 8.0.
