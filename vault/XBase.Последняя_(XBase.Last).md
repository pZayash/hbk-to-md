XBase (XBase)

Последняя (Last)

Доступен, начиная с версии 8.0.

Синтаксис:

Последняя()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - последняя запись получена; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Позиционирует объект на последней записи. Если не установлен текущий индекс, объект позиционируется на последнюю запись в таблице базы данных, если установлен - на запись, имеющую самое старшее значение выражения текущего индекса (если индекс создавался с установленным флагом "Убывание", то наоборот).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB.Последняя(); |

Использование в версии:

Доступен, начиная с версии 8.0.
