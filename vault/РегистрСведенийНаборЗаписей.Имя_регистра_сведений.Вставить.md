РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс позиции, на которую надо вставить новую запись.

Возвращаемое значение:

Тип: [РегистрСведенийЗапись.\<Имя регистра сведений\>](РегистрСведенийЗапись.Имя_регистра_сведений.md).   
Созданная запись регистра.

Описание:

Вставляет новую запись в набор на указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступен только для набора записей регистра сведений, в котором записи подчинены регистратору.

Пример:

|  |
| --- |
| НоваяЦена = НаборЦенКонкурентов.Вставить(0); |

Использование в версии:

Доступен, начиная с версии 8.0.
