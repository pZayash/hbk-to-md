СертификатКриптографии (CryptoCertificate)

НачатьВыгрузку (BeginUnloading)

Доступен, начиная с версии 8.3.6.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- <ВыгруженныеДанные> - данные типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание варианта метода:

Передает выгруженные данные в параметре типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Данные в файл

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- <ИмяФайла> - имя файла, в который выгружен сертификат.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выгружен сертификат.

Описание варианта метода:

Выгружает сертификат в файл.

Описание:

Начинает выгрузку сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), метод [Выгрузить](СертификатКриптографии.Выгрузить_(CryptoCertificate.Unload).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
