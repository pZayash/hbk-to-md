# РегистрыРасчетаМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры расчета](Регистры_расчета.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[РегистрРасчетаМенеджер.\<Имя регистра расчета\>](РегистрРасчетаМенеджер.Имя_регистра_расчета_(CalculationRegisterManager.Calculation_register_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров расчета.  
Возможно обращение к менеджеру регистра расчета посредством оператора [...]. В качестве аргумента передается имя регистра расчета, как оно задано в конфигурации.  

Свойства:

[\<Имя регистра расчета\> (\<Calculation register name\>)](РегистрыРасчетаМенеджер.Имя_регистра_расчета_(CalculationRegistersManager.Calculation_register_name).md)  

Описание:

Предоставляет доступ к менеджерам всех регистров расчета (объектам типа [РегистрРасчетаМенеджер.](РегистрРасчетаМенеджер.Имя_регистра_расчета_(CalculationRegisterManager.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаМенеджер.Имя_регистра_расчета_(CalculationRegisterManager.Calculation_register_name).md)>. Доступ к объектам данного типа осуществляется через объект глобального контекста [РегистрыРасчета](Глобальный_контекст.РегистрыРасчета_(Global_context.CalculationRegisters).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [РегистрыРасчета](Глобальный_контекст.РегистрыРасчета_(Global_context.CalculationRegisters).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
