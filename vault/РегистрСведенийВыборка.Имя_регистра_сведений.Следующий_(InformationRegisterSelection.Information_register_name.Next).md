РегистрСведенийВыборка.<Имя регистра сведений> (InformationRegisterSelection.<Information register name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующая запись выбрана; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующую запись из выборки. Для обхода записей регистра нужно после получения выборки вызвать данный метод для позиционирования на первую запись и далее вызывать до тех пор, пока не будет возвращено значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // найдем максимальную цену у конкурентов на товар ВыбТовар  МаркетингЦен = РегистрыСведений.МаркетингЦен; ОтборПоТовару = Новый Структура("Номенклатура"); ОтборПоТовару.Номенклатура = ВыбТовар; ЦеныТовара = МаркетингЦен.Выбрать(,,ОтборПоТовару); МаксЦена =0; Конкурент = НСтр("ru='Не найден'; en='Not found'"); Пока ЦеныТовара.Следующий() Цикл     Если ЦеныТовара.Цена > МаксЦена Тогда         МаксЦена = ЦеныТовара.Цена;         Конкурент = ЦеныТовара.Конкурент.Наименование;      КонецЕсли; КонецЦикла; Предупреждение(НСтр("ru = 'Максимальная цена';"      + " en = 'Maximum price'") +" """      + СокрЛП(ВыбТовар) + """ = "+ МаксЦена      + НСтр("ru=' руб. у '; en=' USD. It is used by '")      + СокрЛП(Конкурент) + """."); |

Использование в версии:

Доступен, начиная с версии 8.0.
