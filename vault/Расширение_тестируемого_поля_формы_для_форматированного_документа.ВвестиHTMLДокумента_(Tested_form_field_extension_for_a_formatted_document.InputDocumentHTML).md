Расширение тестируемого поля формы для форматированного документа (Tested form field extension for a formatted document)

ВвестиHTMLДокумента (InputDocumentHTML)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ВвестиHTMLДокумента(<HTML>, <Вложения>)

Параметры:

<HTML> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текстовая строка, содержащая HTML.

<Вложения> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Структура, содержащая вложения, которые будут установлены в форматированный документ.  
Ключ элемента структуры - название или имя файла рисунка, значение - [Картинка](Картинка.md) .

Описание:

Имитирует ввод кода HTML форматированного документа на основе переданных параметров.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
