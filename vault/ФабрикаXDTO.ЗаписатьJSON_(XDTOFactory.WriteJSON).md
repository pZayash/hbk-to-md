ФабрикаXDTO (XDTOFactory)

ЗаписатьJSON (WriteJSON)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ЗаписатьJSON(<ЗаписьJSON>, <Значение>, <НазначениеТипаXML>)

Параметры:

<ЗаписьJSON> (обязательный)

Тип: [ЗаписьJSON](ЗаписьJSON_(JSONWriter).md).   
Объект, через который осуществляется запись JSON.

<Значение> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [Неопределено](lang__def_Undefined.md).   
Записываемое в поток JSON значение.

<НазначениеТипаXML> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Вариант назначения типа элемента данных XDTO.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Описание:

Записывает указанный элемент данных XDTO в объект записи JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.7.
