СредстваГеопозиционирования (LocationTools)

ПодключитьОбработчикПересеченияГраницОтслеживаемыхГеозон (AttachMonitoredGeofencesBordersCrossingDetectionHandler)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПодключитьОбработчикПересеченияГраницОтслеживаемыхГеозон(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), [Строка](lang__def_String.md).   
Описание процедуры, которая будет вызвана после завершения операции.  
Процедура должна содержать следующие параметры:

- <Геозона> - тип [Геозона](Геозона_(Geofence).md);
- <ПоложениеОтносительноГеозоны> - Значение [Внутри](ПоложениеОтносительноГеозоны.Внутри_(LocationRelativeToGeofence.Inside).md) означает, что пользователь вошел в геозону, [Снаружи](ПоложениеОтносительноГеозоны.Снаружи_(LocationRelativeToGeofence.Outside).md) - вышел.
- <Дополнительный параметр> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md)

Описание:

Выполняет подключение обработчика входа/выхода из геозон, добавленных с помощью метода [ВключитьОтслеживаниеГеозон](СредстваГеопозиционирования.ВключитьОтслеживаниеГеозон_(LocationTools.EnableGeofencesMonitoring).md) для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
