РежимИзмененияСвязанногоЗначения (LinkedValueChangeMode)

НеИзменять (DontChange)

Доступен, начиная с версии 8.2.

Описание:

При изменении ведущего значения не изменять связанное значение.

Использование в версии:

Доступен, начиная с версии 8.2.
