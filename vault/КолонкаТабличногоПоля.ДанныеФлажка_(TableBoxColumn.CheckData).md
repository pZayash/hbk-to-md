КолонкаТабличногоПоля (TableBoxColumn)

ДанныеФлажка (CheckData)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя колонки, значения которой будут отображаться флажком ячейки табличного поля. Колонка должна иметь тип [Булево](lang__Булево_(Boolean).md) или [Число](lang__def_Number.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
