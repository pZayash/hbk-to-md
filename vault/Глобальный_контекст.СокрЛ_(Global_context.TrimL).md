Глобальный контекст (Global context)

СокрЛ (TrimL)

Доступен, начиная с версии 8.0.

Синтаксис:

СокрЛ(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Исходная строка.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Строка, полученная в результате отсечения пробелов .

Описание:

Отсекает незначащие символы, стоящие слева от первого значащего символа в строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

К незначащим символам относятся символы "пробел", "неразрывный пробел" ([НПП](Символы.НПП_(Chars.NBSp).md)), "табуляция" (горизонтальная [Таб](Символы.Таб_(Chars.Tab).md) и вертикальная [ВТаб](Символы.ВТаб_(Chars.VTab).md)), "возврат каретки" ([ВК](Символы.ВК_(Chars.CR).md)), "перевод строки" ([ПС](Символы.ПС_(Chars.LF).md)), "перевод формы (страницы)" ([ПФ](Символы.ПФ_(Chars.FF).md)).

Пример:

|  |
| --- |
| НаименованиеДляПечати = СокрЛ(Товар.Наименование); |

Использование в версии:

Доступен, начиная с версии 8.0.
