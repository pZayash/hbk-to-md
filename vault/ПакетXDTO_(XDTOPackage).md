# ПакетXDTO

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ТипЗначенияXDTO](ТипЗначенияXDTO_(XDTOValueType).md), [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются типы XDTO.  
Возможно обращение к типу пакета посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция типов XDTO, принадлежащих одному пространству имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
