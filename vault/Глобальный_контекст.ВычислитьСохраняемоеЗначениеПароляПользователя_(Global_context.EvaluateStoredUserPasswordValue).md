Глобальный контекст (Global context)

ВычислитьСохраняемоеЗначениеПароляПользователя (EvaluateStoredUserPasswordValue)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВычислитьСохраняемоеЗначениеПароляПользователя(<Пароль>, <ТипАлгоритмаХешированияПароля>)

Параметры:

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль, для которого вычисляется сохраняемое значение.

<ТипАлгоритмаХешированияПароля> (необязательный)

Тип: [ТипАлгоритмаХешированияПаролейПользователей](ТипАлгоритмаХешированияПаролейПользователей_(UserPasswordHashAlgorithmType).md), [Неопределено](lang__def_Undefined.md).   
Тип алгоритма хеширования для получения сохраняемого значения пароля.   
Если [Неопределено](lang__def_Undefined.md), то используется текущий тип алгоритма хеширования паролей информационной базы, доступный с помощью метода [ПолучитьТипАлгоритмаХешированияПаролейПользователей](Глобальный_контекст.ПолучитьТипАлгоритмаХешированияПаролейПользователей_(Global_context.GetUserPasswordHashAlgorithmType).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Вычисляет сохраняемое значение пароля, предназначенное для использования в качестве значения свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md), алгоритмом заданного типа. Для всех типов алгоритмов, кроме [SHA1](ТипАлгоритмаХешированияПаролейПользователей.SHA1_(UserPasswordHashAlgorithmType.SHA1).md), при каждом вызове метода с одним и тем же паролем будут выдаваться различные сохраняемые значения пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
