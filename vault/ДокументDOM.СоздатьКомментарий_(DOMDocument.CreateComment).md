ДокументDOM (DOMDocument)

СоздатьКомментарий (CreateComment)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьКомментарий(<Комментарий>)

Параметры:

<Комментарий> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст комментария DOM.

Возвращаемое значение:

Тип: [КомментарийDOM](КомментарийDOM_(DOMComment).md).   

Описание:

Создает узел DOM - комментарий.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
