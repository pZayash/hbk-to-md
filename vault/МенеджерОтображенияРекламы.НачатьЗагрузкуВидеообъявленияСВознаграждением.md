МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуВидеообъявленияСВознаграждением (BeginRewardedVideoDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуВидеообъявленияСВознаграждением(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Указывает обработчик, вызываемый при завершении загрузки регламы из сети Интернет.  
Процедура должна иметь следующие параметры:

- Статус (Status) – тип [СтатусРекламы](СтатусРекламы.md). Показывает статус рекламного блока по результатам загрузки.
- Идентификатор (ID) – тип Строка. Рекламный идентификатор для которого закончилась загрузка.
- <ДополнительныеПараметры> – Значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Загружает из сети видеообъявление. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [ПоказатьВидеообъявлениеСВознаграждением](МенеджерОтображенияРекламы.ПоказатьВидеообъявлениеСВознаграждением.md) видеообъявление будет показано без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы.md), метод [ЗагрузитьВидеообъявлениеСВознаграждениемАсинх](МенеджерОтображенияРекламы.ЗагрузитьВидеообъявлениеСВознаграждениемАсинх.md)
