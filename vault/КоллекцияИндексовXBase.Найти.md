КоллекцияИндексовXBase (XBaseIndexesCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяИндекса>)

Параметры:

<ИмяИндекса> (обязательный)

Тип: Строка.   
Наименование индекса XBase, который требуется найти.

Возвращаемое значение:

Тип: [ИндексXBase](ИндексXBase.md), Неопределено.   
Если указанного индекса нет, то возвращается значение Неопределено.

Описание:

Осуществляет поиск индекса XBase по его имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
