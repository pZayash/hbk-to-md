ПолеВвода (TextBox)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак возможности/невозможности интерактивного изменения значения, введенного в поле.

Доступность:

Толстый клиент.

Примечание:

В отличие от свойства [Доступность](ПолеВвода.Доступность.md), если свойство ТолькоПросмотр равно Истина, становится невозможным редактирование поля ввода, однако на остальную работу с полем это не влияет. Например, можно выделить строковое представление содержимого поля выбора и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
