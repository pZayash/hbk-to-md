НастройкиОбработкиОшибокПриЗапуске (ErrorOnStartupProcessingSettings)

АдресСервисаОбработкиОшибокПриЗапуске (ErrorOnStartupProcessingServiceAddress)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Адрес, по которому будет осуществлен GET-запрос дополнительной информации.  
Запрос содержит следующие заголовки, которые можно использовать для анализа:

- errorDescr – полный текст ошибки,
- appName – имя приложения,
- appVersion – версия клиентского приложения,
- clientID – идентификатор клиента,
- additionalInfo – дополнительная информация из настроек обработки ошибок.

  
В ответ ожидается JSON-документ со следующей структурой:  

|  |
| --- |
| {  userMessage:string  userMessageEndTime:string } |

  
  
Где:

- userMessage – строка, которая может содержать теги <b> и <a href=…>
- userMessageEndTime – строка даты в формате ISO 8601.

  
Если адрес не пуст и на запрос был получен валидный ответ, то полученная строка отобразится на форме ошибке (ниже строки, указанной в <ТекстПомощи>), если текущее время на клиенте меньше userMessageEndTime. Если userMessageEndTime не задан, то строка отобразится без учета текущего времени.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
