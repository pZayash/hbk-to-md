Глобальный контекст (Global context)

ДанныеФормыВЗначение (FormDataToValue)

Доступен, начиная с версии 8.2.

Синтаксис:

ДанныеФормыВЗначение(<Объект>, <Тип>)

Параметры:

<Объект> (обязательный)

Тип: [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md), [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md), [ДанныеФормыДерево](ДанныеФормыДерево_(FormDataTree).md).   
Данные формы.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип значения, получаемого из данных формы.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Преобразует данные формы в объект прикладного типа.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Нельзя преобразовать в объект типа [Табличная часть](Табличная_часть_(Tabular_section).md).

Использование в версии:

Доступен, начиная с версии 8.2.
