ЗаписьДанных (DataWriter)

Закрыть (Close)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Закрыть()

Описание:

Вызывает метод [СброситьБуферы](ЗаписьДанных.СброситьБуферы.md). Если целевой поток был создан при создании объекта [ЗаписьДанных](ЗаписьДанных.md), целевой поток также закрывается.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [НачатьЗакрытие](ЗаписьДанных.НачатьЗакрытие.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗакрытьАсинх](ЗаписьДанных.ЗакрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
