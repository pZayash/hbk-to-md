# ФормаКлиентскогоПриложения.Закрыть

ФормаКлиентскогоПриложения (ClientApplicationForm)

Закрыть (Close)

Доступен, начиная с версии 8.2.

Синтаксис:

Закрыть(<ПараметрЗакрытия>)

Параметры:

<ПараметрЗакрытия> (необязательный)

Тип: Произвольный.   
Параметр закрытия формы, открытой в модальном режиме. Это значение будет возвращено как результат метода, открывающего форму модально.  
В режиме отказа от модальности параметр передается в процедуру, описанную параметром [ОписаниеОповещенияОЗакрытии](html.md) метода [ОткрытьФорму](Глобальный_контекст.ОткрытьФорму_(Global_context.OpenForm).md) или свойством формы [ОписаниеОповещенияОЗакрытии](ФормаКлиентскогоПриложения.ОписаниеОповещенияОЗакрытии_(ClientApplicationForm.CallbackDescriptionOnClose).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Закрывает форму. Если закрываемая форма была открыта в модальном режиме, то метод может передать команду закрытия методу, открывшему форму.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).  
Вызов метода выполняет обращение к серверу.

Примечание:

Использование метода не допускается в следующих случаях:

- В интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md):
  - для форм, расположенных на начальной странице,
- В интерфейсе [Версия8\_2](ВариантИнтерфейсаКлиентскогоПриложения.Версия8_2_(ClientApplicationInterfaceVariant.Version8_2).md):
  - В режиме открытия форм "В закладках":
    - для форм, расположенных на рабочем столе.
  - В режиме открытия форм "В отдельных окнах":
    - для форм, расположенных на рабочем столе и в рабочей области основного окна.

Использование в версии:

Доступен, начиная с версии 8.2.
