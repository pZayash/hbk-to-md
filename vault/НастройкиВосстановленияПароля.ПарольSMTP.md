НастройкиВосстановленияПароля (PasswordRecoverySettings)

ПарольSMTP (SMTPPassword)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит пароль пользователя для аутентификации на SMTP-сервере.   
Поддерживаются только символы, входящие в кодовую таблицу US-ASCII.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
