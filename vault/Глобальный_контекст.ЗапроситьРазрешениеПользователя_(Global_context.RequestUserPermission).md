Глобальный контекст (Global context)

ЗапроситьРазрешениеПользователя (RequestUserPermission)

Не рекомендуется использовать, начиная с версии 8.3.15.

Рекомендуется использовать:

- [НачатьЗапросРазрешенияПользователя](Глобальный_контекст.НачатьЗапросРазрешенияПользователя_(Global_context.BeginRequestingUserPermission).md)

Доступен, начиная с версии 8.2.

Синтаксис:

ЗапроситьРазрешениеПользователя(<Вызовы>)

Параметры:

<Вызовы> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Содержит массив с описаниями вызовов, для которых получается разрешение.  
Каждое описание вызова это массив. Первым элементом этого массива является строка с именем метода, для которого получается разрешение. Это методы:

- [ЗапуститьПриложение](Глобальный_контекст.ЗапуститьПриложение_(Global_context.RunApp).md);
- [УдалитьФайлы](Глобальный_контекст.УдалитьФайлы_(Global_context.DeleteFiles).md);
- [НайтиФайлы](Глобальный_контекст.НайтиФайлы_(Global_context.FindFiles).md);
- [КопироватьФайл](Глобальный_контекст.КопироватьФайл_(Global_context.FileCopy).md);
- [ПереместитьФайл](Глобальный_контекст.ПереместитьФайл_(Global_context.MoveFile).md);
- [СоздатьКаталог](Глобальный_контекст.СоздатьКаталог_(Global_context.CreateDirectory).md).

Следующие элементы массива - это параметры вызова в правильном порядке.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В режиме запуска тонкий и толстый клиент, а также для клиента мобильного приложения всегда возвращает [Истина](lang__def_BooleanTrue.md).  
Если метод получает пустой запрос (за вычетом проигнорированных записей), то возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Получает у пользователя разрешение на несколько операций с файлами, которые потом будут выполнены без дополнительных вопросов.  
Полученное разрешение хранится до выполнения соответствующего вызова, либо до окончания работы текущего вызова встроенного языка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если сочетание параметров предполагает вызов выбора файлов или поиск файлов по маске, то описание вызова будет проигнорировано при запросе разрешения.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
