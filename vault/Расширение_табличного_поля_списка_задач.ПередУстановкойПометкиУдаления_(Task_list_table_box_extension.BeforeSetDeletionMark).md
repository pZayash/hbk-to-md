Расширение табличного поля списка задач (Task list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке задач.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ЗадачаОбъект.Имя_задачи.ПередЗаписью_(TaskObject.Task_name.BeforeWrite).md) и [ПриЗаписи](ЗадачаОбъект.Имя_задачи.ПриЗаписи_(TaskObject.Task_name.OnWrite).md)объекта [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)<[Имя задачи](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
