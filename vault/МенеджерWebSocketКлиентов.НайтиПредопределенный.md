МенеджерWebSocketКлиентов (WebSocketClientsManager)

НайтиПредопределенный (FindPredefined)

Доступен, начиная с версии 8.3.27.

Синтаксис:

НайтиПредопределенный(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент.md).   
Объект метаданных WebSocket-клиента.

Возвращаемое значение:

Тип: [WebSocketКлиент](WebSocketКлиент.md), Неопределено.   

Описание:

Возвращает [WebSocketКлиент](WebSocketКлиент.md), связанный с предопределенным WebSocket-клиентом или Неопределено, если такого предопределенного нет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
