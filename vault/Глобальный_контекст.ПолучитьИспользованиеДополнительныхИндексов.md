Глобальный контекст (Global context)

ПолучитьИспользованиеДополнительныхИндексов (GetAdditionalIndexesUsage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьИспользованиеДополнительныхИндексов()

Возвращаемое значение:

Тип: Булево.   

Описание:

Возвращает значение признака использования дополнительных индексов.  
  
Результат выполнения этого метода не отражает факт реального наличия или отсутствия дополнительных индексов в базе данных, а только возвращает состояние признака необходимости их создания или удаления при следующих действиях:

- реструктуризация объектов метаданных,
- тестирование и исправление информационной базы в режиме "Проверка и включение функциональности уровня КОРП",
- тестирование и исправление информационной базы в режиме "Реструктуризация таблиц информационной базы".

Получить информацию о фактическом наличии или отсутствии дополнительных индексов в базе данных можно с помощью метода [ПолучитьСтруктуруХраненияБазыДанных](Глобальный_контекст.ПолучитьСтруктуруХраненияБазыДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
