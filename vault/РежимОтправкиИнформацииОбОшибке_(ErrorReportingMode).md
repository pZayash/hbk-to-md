Описание:

Содержит варианты режима отправки различной информации об ошибке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/managed-application/core. Имя типа XDTO: ErrorInfoReportingMode.

Использование в версии:

Доступен, начиная с версии 8.3.17.
