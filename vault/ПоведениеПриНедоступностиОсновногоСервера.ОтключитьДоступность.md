ПоведениеПриНедоступностиОсновногоСервера (OnMainServerUnavalableBehavior)

ОтключитьДоступность (MakeDisable)

Доступен, начиная с версии 8.3.16.

Описание:

При недоступности основного сервера, если форма была открыта с автономного сервера, команда или элемент формы становится недоступным.

Использование в версии:

Доступен, начиная с версии 8.3.16.
