СерияДанныхСлояГеографическойСхемы (GeographicalSchemaLayerDataSeries)

РежимОтображенияЗначений (ValuesShowMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтображенияЗначенийСерии](РежимОтображенияЗначенийСерии_(SeriesValuesDrawingMode).md).   
Определяет, как будут отображаться значения, если серия отображается каким-либо видом графика, а не только гистограммой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
