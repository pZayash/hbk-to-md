# Расширение табличного поля списка видов расчета.ПериодАвтоОбновления

Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ПериодАвтоОбновления (AutoRefreshPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Устанавливает период (в секундах), с которым будет производиться автоматическое обновление списка. Минимальное значение периода - 5 секунд. Если свойство [АвтоОбновление](Расширение_табличного_поля_списка_видов_расчета.АвтоОбновление_(Calculation_types_list_table_box_extension.AutoRefresh).md) установлено как [Ложь](lang__def_BooleanFalse.md), то обновление не производится.

Доступность:

Толстый клиент.
