КнопкаКоманднойПанели (CommandBarButton)

Пометка (Check)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет пометкой кнопки командной панели.   
Истина - кнопка помечена (нажата).  
Может использоваться для кнопок с типом [Действие](ТипКнопкиКоманднойПанели.Действие.md).  
Если кнопка панели помечена системой, значение свойства игнорируется.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
