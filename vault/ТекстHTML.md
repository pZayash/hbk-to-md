Описание:

Объект, представляющий фрагмент текстовой информации внутри HTML документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ДокументHTML](ДокументHTML.md), метод [СоздатьТекстовыйУзел](ДокументHTML.СоздатьТекстовыйУзел.md)
