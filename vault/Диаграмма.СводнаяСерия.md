Диаграмма (Chart)

СводнаяСерия (SummarySeries)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [СерияДиаграммы](СерияДиаграммы.md).   
Содержит сводную серию диаграммы. Это серия формируется автоматически из серий, объединяемых при использовании ограничения на максимальное количество серий диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
