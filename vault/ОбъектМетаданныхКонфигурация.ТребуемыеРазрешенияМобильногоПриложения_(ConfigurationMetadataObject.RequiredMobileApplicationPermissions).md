ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ТребуемыеРазрешенияМобильногоПриложения (RequiredMobileApplicationPermissions)

Не рекомендуется использовать, начиная с версии 8.3.18.

Рекомендуется использовать:

- [ИспользуемаяФункциональностьМобильногоПриложения](ОбъектМетаданныхКонфигурация.ИспользуемаяФункциональностьМобильногоПриложения_(ConfigurationMetadataObject.UsedMobileApplicationFunctionalities).md)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Список требуемых разрешений мобильного приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Не рекомендуется использовать, начиная с версии 8.3.18.

Описание изменено в версии 8.3.18.
