РегистрНакопленияНаборЗаписей.<Имя регистра накопления> (AccumulationRegisterRecordSet.<Accumulation register name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов записей. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.ОбработкаПроверкиЗаполнения_(AccumulationRegisterRecordSet.Accumulation_register_name.FillCheckProcessing).md).

Доступен, начиная с версии 8.2.
