КоллекцияМетокИнтервалаФонаПланировщика (PlannerBackgroundIntervalLabelCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.17.

Вариант синтаксиса: По индексу

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс сдвигаемой метки в коллекции меток интервала фона планировщика.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов в коллекции, на которое нужно сдвинуть метку. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Вариант синтаксиса: По метке

Синтаксис:

Сдвинуть(<Метка>, <Смещение>)

Параметры:

<Метка> (обязательный)

Тип: [МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика.md).   
Сдвигаемая метка в коллекции меток интервала фона планировщика.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов в коллекции, на которое нужно сдвинуть метку. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание:

Сдвигает элемент на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
