ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

КонтрольУникальности (CheckUnique)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то при создании нового объекта базы данных будет выполняться автоматический контроль уникальности его кода/номера.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
