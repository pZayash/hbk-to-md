# ЗадачаОбъект.<Имя задачи>.УстановитьВремя

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Методы](ЗадачаОбъект.Имя_задачи__Методы.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

УстановитьВремя (SetTime)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьВремя(<РежимОпределенияВремени>)

Параметры:

<РежимОпределенияВремени> (необязательный)

Тип: [РежимАвтоВремя](РежимАвтоВремя_(AutoTimeMode).md).   
Устанавливает режим автоматического определения времени при записи новой задачи.  
Значение по умолчанию: [ТекущееИлиПоследним](РежимАвтоВремя.ТекущееИлиПоследним_(AutoTimeMode.CurrentOrLast).md).

Описание:

Устанавливает режим определения времени при записи задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
