Доступен, начиная с версии 8.1.

Элементы коллекции:

[ВложеннаяСхемаКомпоновкиДанных](ВложеннаяСхемаКомпоновкиДанных_(NestedDataCompositionSchema).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются вложенные схемы.  
Возможно обращение к вложенной схеме посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Описание:

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
