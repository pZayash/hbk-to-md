СписокПолнотекстовогоПоиска (FullTextSearchList)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс поиска. Начинается с 0.

Возвращаемое значение:

Тип: [ЭлементСпискаПолнотекстовогоПоиска](ЭлементСпискаПолнотекстовогоПоиска_(FullTextSearchListItem).md).   

Описание:

Получает элемент списка поиска по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
