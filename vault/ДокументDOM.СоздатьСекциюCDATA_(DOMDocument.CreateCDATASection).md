# ДокументDOM.СоздатьСекциюCDATA

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ДокументDOM](ДокументDOM_(DOMDocument).md) › [Методы](ДокументDOM__Методы.md)

ДокументDOM (DOMDocument)

СоздатьСекциюCDATA (CreateCDATASection)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьСекциюCDATA(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [Строка](lang__def_String.md).   
Данные секции CDATA.

Возвращаемое значение:

Тип: [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md).   

Описание:

Создает узел DOM - секция CDATA.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
