СтатусРазбораИнтернетПочтовогоСообщения (InternetMailMessageParseStatus)

ОшибокНеОбнаружено (ErrorsNotDetected)

Доступен, начиная с версии 8.3.6.

Описание:

При разборе почтового соощения ошибок обнаружено не было.

Использование в версии:

Доступен, начиная с версии 8.3.6.
