ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<РежимЗаписи>, <РежимПроведения>)

Параметры:

<РежимЗаписи> (необязательный)

Тип: [РежимЗаписиДокумента](РежимЗаписиДокумента.md).   
Позволяет выбрать один из возможных режимов записи.  
Значение по умолчанию: [Запись](РежимЗаписиДокумента.Запись.md).

<РежимПроведения> (необязательный)

Тип: [РежимПроведенияДокумента](РежимПроведенияДокумента.md).   
Позволяет выбрать один из возможных режимов проведения.  
Значение по умолчанию: [Неоперативный](РежимПроведенияДокумента.Неоперативный.md).

Описание:

Записывает документ в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для документов с поддержкой уникальности номеров в процессе записи, если нужно, проверяется уникальность номера.

Пример:

|  |
| --- |
| Попытка     НовДокумент.Записать(РежимЗаписиДокумента.Проведение); Исключение     Предупреждение(НСтр("ru = 'Невозможно провести документ.';"      + " en = 'Document needs reposting.'")); КонецПопытки; |
