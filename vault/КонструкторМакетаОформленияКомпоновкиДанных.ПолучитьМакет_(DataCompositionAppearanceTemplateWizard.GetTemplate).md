Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьМакет()

Возвращаемое значение:

Тип: [МакетОформленияКомпоновкиДанных](МакетОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplate).md).

Толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
