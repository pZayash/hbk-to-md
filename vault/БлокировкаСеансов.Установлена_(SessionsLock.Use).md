БлокировкаСеансов (SessionsLock)

Установлена (Use)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет установку блокировки.  
[Истина](lang__def_BooleanTrue.md) - блокировка включена.   
[Ложь](lang__def_BooleanFalse.md) - выключена.   
Блокировка не позволяет начинать новые сеансы с информационной базой, если она включена и актуальна.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
