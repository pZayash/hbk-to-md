ПотокОбменаДанными (DataExchangeStream)

Записать (Write)

Доступен, начиная с версии 8.3.22.

Синтаксис:

Записать(<Значение>)

Параметры:

<Значение> (необязательный)

Записываемое значение, например: Справочник.Объект, Справочник.Ссылка, и т.д.  
  
В обработчиках [ПередНачаломОтправкиДанныхГлавному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхГлавному.md), [ПередНачаломОтправкиДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхПодчиненному.md) можно передать объект данных типа [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md), для передачи сырых данных и обработать в обработчиках [ПриПолученииДанныхОтГлавного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтГлавного.md), [ПриПолученииДанныхОтПодчиненного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтПодчиненного.md).

Описание:

Записывает объект, который может быть передан в обработчики [ПередСозданиемНачальногоОбраза](ПланОбменаОбъект.Имя_плана_обмена.ПередСозданиемНачальногоОбраза.md), [ПередНачаломОтправкиДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПередНачаломОтправкиДанныхПодчиненному.md) для записи объектов в обмене данными.

Доступность:

Мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
