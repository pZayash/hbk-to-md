**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Пакет отображаемых документов](Пакет_отображаемых_документов-2.md)

Доступен, начиная с версии 8.3.3.

Элементы коллекции:

[ЭлементПакетаОтображаемыхДокументов](ЭлементПакетаОтображаемыхДокументов_(RepresentableDocumentBatchItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы пакета.  
Возможно обращение к элементу пакета посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция элементов пакета документов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:
