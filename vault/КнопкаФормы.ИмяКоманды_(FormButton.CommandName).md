КнопкаФормы (FormButton)

ИмяКоманды (CommandName)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя команды формы. Если кнопка связана не с командой формы, содержит пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Изменение свойства доступно только на сервере.

Использование в версии:

Доступен, начиная с версии 8.2.
