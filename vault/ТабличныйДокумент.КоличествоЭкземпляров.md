ТабличныйДокумент (SpreadsheetDocument)

КоличествоЭкземпляров (Copies)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит количество экземпляров для вывода табличного документа на печать.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если не задано, то значение свойства будет выбираться из настроек принтера, на котором осуществляется печать.

Использование в версии:

Доступен, начиная с версии 8.0.
