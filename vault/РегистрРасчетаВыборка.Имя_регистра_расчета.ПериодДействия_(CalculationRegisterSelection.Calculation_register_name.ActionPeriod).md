РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

ПериодДействия (ActionPeriod)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Период действия - дата, которая принимает дискретные значения в зависимости от периодичности регистра расчета.  
Дата периода действия приводится к началу месяца от даты [ПериодДействияНачало](objects__catalog125__catalog486__object490__properties__BegOfEffectivePeriod2118.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определен для регистров расчета с признаком "Период действия".

Использование в версии:

Доступен, начиная с версии 8.0.
