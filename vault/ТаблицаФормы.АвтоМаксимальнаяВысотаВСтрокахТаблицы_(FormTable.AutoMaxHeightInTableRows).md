ТаблицаФормы (FormTable)

АвтоМаксимальнаяВысотаВСтрокахТаблицы (AutoMaxHeightInTableRows)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
На максимальную высоту таблицы накладывается автоматическое ограничение.  
Свойство используется, в одном из случаев:

- если свойство [ВариантУправленияВысотой](ТаблицаФормы.ВариантУправленияВысотой_(FormTable.HeightControlVariant).md) имеет значение [ВСтрокахТаблицы](ВариантУправленияВысотойТаблицы.ВСтрокахТаблицы_(TableHeightControlVariant.UseHeightInTableRows).md) или [ПоСодержимому](ВариантУправленияВысотойТаблицы.ПоСодержимому_(TableHeightControlVariant.UseContentHeight).md)
- если значение [Авто](ВариантУправленияВысотойТаблицы.Авто_(TableHeightControlVariant.Auto).md) было разыменовано в эти значения.

В противном случае, значение свойства игнорируется.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.10.
