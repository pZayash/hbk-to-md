СоздатьОбходДерева (CreateTreeWalker)

Доступен, начиная с версии 8.2.

Синтаксис:

Узел HTML - стартовый узел итератора.

Возвращаемое значение:

Тип: [ОбходДереваDOM](ОбходДереваDOM.md).   

Описание:

Создает новый иерархический объект обхода дерева документа HTML, начиная с указанного корневого узла.  
Метод соответствует интерфейсу Treewalker стандарта W3C (Подробнее см. <http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113/traversal.html#Traversal-TreeWalker>).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
