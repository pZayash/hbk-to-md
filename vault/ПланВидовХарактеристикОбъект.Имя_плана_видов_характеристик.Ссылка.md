ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md).   
Содержит ссылку на элемент вида характеристик. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство используется, если ссылку на созданный и записанный элемент вида характеристик нужно присвоить реквизиту другого вида характеристик.

Использование в версии:

Доступен, начиная с версии 8.0.
