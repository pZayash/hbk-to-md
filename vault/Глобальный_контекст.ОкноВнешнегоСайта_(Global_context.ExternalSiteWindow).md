Глобальный контекст (Global context)

ОкноВнешнегоСайта (ExternalSiteWindow)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [МенеджерОкнаВнешнегоСайта](МенеджерОкнаВнешнегоСайта_(ExternalSiteWindowManager).md).   
Используется для доступа к функциям окна внешнего сайта, когда веб-клиент интегрирован в него.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
