КлиентскоеПриложение (ClientApplication)

УстановитьОтображениеЗаголовкаОС (SetOSCaptionRepresentation)

Доступен, начиная с версии 8.3.15.

Синтаксис:

УстановитьОтображениеЗаголовкаОС(<ОтображениеЗаголовка>)

Параметры:

<ОтображениеЗаголовка> (обязательный)

Тип: Булево.   
Задает отображение заголовка операционной системы для главного окна.

Описание:

Для тонкого и толстого клиентов под управлением ОС Windows в интерфейсе "Такси" устанавливает отображение заголовка операционной системы для главного окна.  
Для остальных клиентов и вариантов интерфейса вызов метода игнорируется.  
Вызов метода возможен только в обработчике события [ПередНачаломРаботыСистемы](Глобальный_контекст.Глобальный_контекст.ПередНачаломРаботыСистемы.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
