Глобальный контекст (Global context)

ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (OnCollaborationSystemMessageButtonPanelButtonClick)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия(<Сообщение>, <Кнопка>)

Параметры:

<Сообщение>

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).   
Сообщение, в котором была нажата кнопка.

<Кнопка>

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   
Кнопка, которая была нажата.

Описание:

Обработчик, вызываемый при нажатии кнопки панели кнопок сообщения системы взаимодействия, если действие кнопки [ОбработатьНаКлиенте](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОбработатьНаКлиенте_(CollaborationSystemMessageButtonPanelButtonAction.ProcessOnClient).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.
