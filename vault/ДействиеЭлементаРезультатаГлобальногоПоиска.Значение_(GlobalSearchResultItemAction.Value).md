ДействиеЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemAction)

Значение (Value)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение действия. Используется для поиска с помощью метода [Найти](КоллекцияДействийЭлементаРезультатаГлобальногоПоиска.Найти_(GlobalSearchResultItemActionCollection.Find).md) коллекции действий элемента результата глобального поиска, а также для передачи в событие [ПриВыбореДействияРезультатаГлобальногоПоиска](Глобальный_контекст.ПриВыбореДействияРезультатаГлобальногоПоиска_(Global_context.OnGlobalSearchResultActionChoice).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
