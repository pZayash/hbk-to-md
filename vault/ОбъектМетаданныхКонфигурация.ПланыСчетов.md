ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПланыСчетов (ChartsOfAccounts)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все планы счетов, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
