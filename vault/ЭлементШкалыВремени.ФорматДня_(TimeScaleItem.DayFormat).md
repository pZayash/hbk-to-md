ЭлементШкалыВремени (TimeScaleItem)

ФорматДня (DayFormat)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ФорматДняШкалыВремени](ФорматДняШкалыВремени_(TimeScaleDayFormat).md).   
Предпочтительный формат дня. Используется, если [Единица](ЭлементШкалыВремени.Единица_(TimeScaleItem.Unit).md) = "День".

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
