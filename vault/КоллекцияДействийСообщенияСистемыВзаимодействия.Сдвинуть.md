КоллекцияДействийСообщенияСистемыВзаимодействия (CollaborationSystemMessageActionCollection)

Сдвинуть (Move)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Сдвинуть](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Сдвинуть.md)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По индексу

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс сдвигаемого действия в коллекции действий сообщения системы взаимодействия.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов в коллекции, на которое требуется сдвинуть действие. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание варианта метода:

Сдвигает действие с заданным индексом на указанное смещение.

Вариант синтаксиса: По действию

Синтаксис:

Сдвинуть(<Действие>, <Смещение>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия.md).   
Сдвигаемое действие сообщения системы взаимодействия.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов в коллекции, на которое требуется сдвинуть действие. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание варианта метода:

Сдвигает действие на указанное смещение.

Описание:

Сдвигает действие на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
