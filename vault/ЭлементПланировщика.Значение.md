ЭлементПланировщика (PlannerItem)

Значение (Value)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Произвольное значение, которое можно хранить в элементе планировщика и по которому можно искать элемент в коллекции элементов планировщика с помощью метода [Найти](КоллекцияЭлементовПланировщика.Найти.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
