СпособОпределенияОграничивающегоЗначенияДиаграммы (ChartBoundaryDetectionMethod)

ИспользоватьЗначениеСОграничением (UseValueWithLimitations)

Доступен, начиная с версии 8.3.10.

Описание:

Ограничивающее значение используется, если оно не сокращает диапазон данных. В противном случае в качестве ограничивающего используется максимальное или минимальное значение из данных.

Использование в версии:

Доступен, начиная с версии 8.3.10.
