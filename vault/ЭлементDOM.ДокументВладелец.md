ЭлементDOM (DOMElement)

ДокументВладелец (OwnerDocument)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДокументDOM](ДокументDOM.md), Неопределено.   
Содержит документ DOM, которому принадлежит данный узел DOM.   
Необходим для возможности создания новых узлов.   
Если данный узел - Document или DocumentType, которыми не владеет пока ни один документ, то содержит Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
