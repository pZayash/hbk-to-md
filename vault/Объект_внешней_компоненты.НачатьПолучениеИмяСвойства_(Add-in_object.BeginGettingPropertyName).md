# Объект внешней компоненты.НачатьПолучение<ИмяСвойства>

Объект внешней компоненты (Add-in object)

НачатьПолучение<ИмяСвойства> (BeginGetting<PropertyName>)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучение<ИмяСвойства>(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения вызова метода со следующими параметрами:

- <Значение> - значение свойства
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Начинает получение свойства внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

[Объект внешней компоненты](Объект_внешней_компоненты_(Add-in_object).md), метод [Получить\<ИмяСвойства\>Асинх](Объект_внешней_компоненты.ПолучитьИмяСвойстваАсинх_(Add-in_object.GetPropertyNameAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
