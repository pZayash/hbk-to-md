ЧтениеФайлаАрхива (ArchiveFileReader)

Открыть (Open)

Доступен, начиная с версии 8.3.26.

Вариант синтаксиса: Открыть для чтения из файла

Синтаксис:

Открыть(<ИмяФайла>, <Пароль>, <ТипФайлаАрхива>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла архива, который требуется открыть для чтения.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к файлу архива, если файл зашифрован.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива_(ArchiveFileType).md).   
Тип файла архива, который требуется открыть для чтения.   
По умолчанию тип определяется по расширению имени файла.

Описание варианта метода:

Метод открывает файл архива для чтения.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Пароль>, <ТипФайлаАрхива>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток, из которого будут прочитаны данные архива.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к архиву, если архив зашифрован.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива_(ArchiveFileType).md).   
Тип файла архива, который требуется открыть для чтения.   
По умолчанию, тип определяется по расширению имени файла.

Описание варианта метода:

Устанавливает поток для чтения архива.

Описание:

Открывает файл архива для чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
