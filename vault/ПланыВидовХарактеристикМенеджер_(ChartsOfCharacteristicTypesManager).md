# ПланыВидовХарактеристикМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПланВидовХарактеристикМенеджер.\<Имя плана видов характеристик\>](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов характеристик.  
Возможно обращение к менеджеру плана видов характеристик посредством оператора [...]. В качестве аргумента передается имя плана видов характеристик, как оно задано в конфигурации.  

Свойства:

[\<Имя плана видов характеристик\> (\<Chart of characteristic types name\>)](ПланыВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartsOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПланыВидовХарактеристикМенеджер.ТипВсеСсылки_(ChartsOfCharacteristicTypesManager.AllRefsType).md)  

Описание:

Предназначен для управления планами видов характеристик и предоставляет доступ к значениям типа [ПланВидовХарактеристикМенеджер.](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [ПланыВидовХарактеристик](Глобальный_контекст.ПланыВидовХарактеристик_(Global_context.ChartsOfCharacteristicTypes).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ПланыВидовХарактеристик](Глобальный_контекст.ПланыВидовХарактеристик_(Global_context.ChartsOfCharacteristicTypes).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
