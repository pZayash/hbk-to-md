# ВсеЭлементыФормы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Форма клиентского приложения](Форма_клиентского_приложения.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ГруппаФормы](ГруппаФормы_(FormGroup).md), [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md), [КнопкаФормы](КнопкаФормы_(FormButton).md), [ПолеФормы](ПолеФормы_(FormField).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ДополнениеЭлементаФормы](ДополнениеЭлементаФормы_(FormItemAddition).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается имя элемента.  

Свойства:

[\<Имя элемента управления\> (\<Control name\>)](ВсеЭлементыФормы.Имя_элемента_управления_(FormAllItems.Control_name).md)

Описание:

Коллекция всех элементов формы клиентского приложения не зависит от их места в иерархии. В коллекцию не входит только корневой элемент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
