Расширение группы формы для страницы (Form group extension for a page)

ШиринаПодчиненныхЭлементов (SlaveItemsWidth)

Не рекомендуется использовать, начиная с версии 8.3.7.

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ШиринаПодчиненныхЭлементовФормы](ШиринаПодчиненныхЭлементовФормы_(ChildFormItemsWidth).md).   
Задает относительную ширину подчиненных элементов формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.3.7.
