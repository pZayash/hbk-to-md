КолонкиРезультатаМоделиПрогноза (PredictionModelResultColumns)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Колонка>, <Позиция>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаМоделиПрогноза](КолонкаМоделиПрогноза_(PredictionModelColumn).md).   
Вставляемая колонка.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить колонку.

Возвращаемое значение:

Тип: [КолонкаРезультатаМоделиПрогноза](КолонкаРезультатаМоделиПрогноза_(PredictionModelResultColumn).md).   

Описание:

Вставляет колонку в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
