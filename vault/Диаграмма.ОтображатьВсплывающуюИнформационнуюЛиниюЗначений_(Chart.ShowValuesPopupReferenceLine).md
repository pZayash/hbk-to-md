Диаграмма (Chart)

ОтображатьВсплывающуюИнформационнуюЛиниюЗначений (ShowValuesPopupReferenceLine)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы](ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы_(ShowChartPopupReferenceLine).md).   
Определяет режим отображения всплывающей информационной линии на шкале значений при показе подсказки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
