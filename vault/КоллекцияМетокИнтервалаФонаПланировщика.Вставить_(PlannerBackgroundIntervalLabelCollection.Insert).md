КоллекцияМетокИнтервалаФонаПланировщика (PlannerBackgroundIntervalLabelCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить метку интервала фона планировщика.

<Значение> (необязательный)

Тип: Произвольный.   
Значение метки интервала фона планировщика.

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, отображаемый в метке интервала фона планировщика.

Возвращаемое значение:

Тип: [МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика_(PlannerBackgroundIntervalLabel).md).   

Описание:

Вставляет метку в коллекцию меток интервала фона планировщика по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
