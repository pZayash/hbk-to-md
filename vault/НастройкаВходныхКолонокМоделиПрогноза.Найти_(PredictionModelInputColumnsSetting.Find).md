НастройкаВходныхКолонокМоделиПрогноза (PredictionModelInputColumnsSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомой настройки.

Возвращаемое значение:

Тип: [НастройкаВходнойКолонкиМоделиПрогноза](НастройкаВходнойКолонкиМоделиПрогноза_(PredictionModelInputColumnSetting).md), [Неопределено](lang__def_Undefined.md).   
Если настройка не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск настройки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
