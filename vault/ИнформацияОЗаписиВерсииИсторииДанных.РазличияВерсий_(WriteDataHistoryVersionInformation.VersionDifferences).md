ИнформацияОЗаписиВерсииИсторииДанных (WriteDataHistoryVersionInformation)

РазличияВерсий (VersionDifferences)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md), [Неопределено](lang__def_Undefined.md).   
Различия между версиями или [Неопределено](lang__def_Undefined.md), если версия создана по удалению данных.  
Значение свойства соответствует значению, возвращаемому методом [ПолучитьРазличияВерсий](МенеджерИсторииДанных.ПолучитьРазличияВерсий_(DataHistoryManager.GetVersionDiff).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
