ПользовательскоеПолеВыражениеКомпоновкиДанных (DataCompositionUserFieldExpression)

УстановитьВыражения (SetExpressions)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьВыражения(<ВыражениеДетальныхЗаписей>, <ВыражениеИтоговыхЗаписей>)

Параметры:

<ВыражениеДетальныхЗаписей> (необязательный)

Тип: Строка.   
Устанавливаемое выражение для вывода поля в детальных записях.

<ВыражениеИтоговыхЗаписей> (необязательный)

Тип: Строка.   
Устанавливаемое выражение для вывода поля в итоговых записях.

Описание:

Устанавливает выражения, используемые для полей детальных и итоговых записей.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.10.
