Расширение группы формы для обычной группы (Form group extension for a usual group)

ОтображатьОтступСлева (ShowLeftMargin)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержимое свертываемой группы отображается в развернутом виде с отступом слева.  
Данное свойство не используется, если [Отображение](Расширение_группы_формы_для_обычной_группы.Отображение.md) = [СильноеВыделение](ОтображениеОбычнойГруппы.СильноеВыделение.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
