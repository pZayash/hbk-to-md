НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента настройки колонок.

Возвращаемое значение:

Тип: [ЭлементУправленияКолонкамиАнализаДанных](ЭлементУправленияКолонкамиАнализаДанных.md), Неопределено.   
Если не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента настройки колонок по имени элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
