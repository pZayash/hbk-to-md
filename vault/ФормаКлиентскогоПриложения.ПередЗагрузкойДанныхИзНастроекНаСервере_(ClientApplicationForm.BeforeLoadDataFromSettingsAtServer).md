ФормаКлиентскогоПриложения (ClientApplicationForm)

ПередЗагрузкойДанныхИзНастроекНаСервере (BeforeLoadDataFromSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗагрузкойДанныхИзНастроекНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [Соответствие](Соответствие_(Map).md).   
Настройки формы, в которых находятся значения реквизитов.

Описание:

Вызывается на сервере перед загрузкой данных из настроек:

- Перед открытием формы;
- При выполнении команды "Восстановить параметры...".

В данный обработчик передается объект, из которого будут загружены настройки. Объект представляет из себя соответствие, в котором находятся значения реквизитов. В обработчике можно загрузить собственные свойства из соответствия. Например, загрузить из соответствия отбор.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
