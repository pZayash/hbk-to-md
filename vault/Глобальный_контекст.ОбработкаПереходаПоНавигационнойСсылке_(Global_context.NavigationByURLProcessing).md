Глобальный контекст (Global context)

ОбработкаПереходаПоНавигационнойСсылке (NavigationByURLProcessing)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОбработкаПереходаПоНавигационнойСсылке(<ДанныеПереходаПоНавигационнойСсылке>, <СтандартнаяОбработка>)

Параметры:

<ДанныеПереходаПоНавигационнойСсылке>

Данные перехода по навигационной ссылке.

<СтандартнаяОбработка>

В данный параметр передается признак выполнения стандартной (системной) обработки ссылки. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка перехода по ссылке производиться не будет.

Описание:

Вызывается при переходе по навигационной ссылке внутри приложения или при переходе в приложение по внешней ссылке или ссылке в мобильное приложение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
