ЭлементыМакетаОбластиКомпоновкиДанных (DataCompositionAreaTemplateItems)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных_(DataCompositionAreaTemplateField).md).

Возвращаемое значение:

Тип: [ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных_(DataCompositionAreaTemplateField).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
