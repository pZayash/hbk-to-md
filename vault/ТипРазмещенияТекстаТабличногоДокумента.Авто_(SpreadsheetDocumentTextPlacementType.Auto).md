ТипРазмещенияТекстаТабличногоДокумента (SpreadsheetDocumentTextPlacementType)

Авто (Auto)

Доступен, начиная с версии 8.0.

Описание:

Не помещающийся по ширине текст будет показан в соседних по горизонтали ячейках, если в них нет текста.

Использование в версии:

Доступен, начиная с версии 8.0.
