# ЗадачиМенеджер

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры задач.  
Возможно обращение к менеджеру задачи посредством оператора [...]. В качестве аргумента передается имя задачи, как оно задано в конфигурации.  

Свойства:

[\<Имя задачи\> (\<Task name\>)](ЗадачиМенеджер.Имя_задачи.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ЗадачиМенеджер.ТипВсеСсылки.md)  

Описание:

Предназначен для управления задачами и предоставляет доступ к значениям типа [ЗадачаМенеджер.](ЗадачаМенеджер.Имя_задачи.md)<[Имя задачи](ЗадачаМенеджер.Имя_задачи.md)>. Доступ к объекту осуществляется через свойство глобального контекста [Задачи](Глобальный_контекст.Задачи.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Задачи](Глобальный_контекст.Задачи.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
