ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

Параметр (Parameter)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных.md).   
Параметр, доступный для использования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для поиска нужного параметра в доступных параметрах (перебором) данное свойство использовать нельзя, т.к. доступный параметр может быть многоязыковым, но свойство всегда возвращает один и тот же синоним. Для поиска нужно использовать метод [НайтиПараметр](ДоступныеПараметрыКомпоновкиДанных.НайтиПараметр.md).

Использование в версии:

Доступен, начиная с версии 8.1.
