РежимСовместимости (CompatibilityMode)

Версия8\_1 (Version8\_1)

Доступен, начиная с версии 8.2.

Описание:

Возможности версии 8.2 частично не используются. Гарантируется работа конфигураций, созданных на платформе версии 8.1.

Использование в версии:

Доступен, начиная с версии 8.2.
