ТипКаноническогоXML (XMLCanonicalizationType)

ИсключающийКаноническийXML (XMLExclusiveCanonicalization)

Доступен, начиная с версии 8.1.

Описание:

Приведение к каноническому виду по w3c стандарту Exclusive XML Canonicalization. Комментарии игнорируются.

Использование в версии:

Доступен, начиная с версии 8.1.
