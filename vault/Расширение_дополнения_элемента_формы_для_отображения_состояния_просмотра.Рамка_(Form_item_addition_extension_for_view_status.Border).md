Расширение дополнения элемента формы для отображения состояния просмотра (Form item addition extension for view status)

Рамка (Border)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Рамка](Рамка_(Border).md).   
Содержит параметры рамки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
