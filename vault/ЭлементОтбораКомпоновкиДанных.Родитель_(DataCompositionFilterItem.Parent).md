ЭлементОтбораКомпоновкиДанных (DataCompositionFilterItem)

Родитель (Parent)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md), [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных_(DataCompositionFilterItemGroup).md).   
Родительская группа элементов отбора, [Неопределено](lang__def_Undefined.md) для элементов отбора верхнего уровня.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
