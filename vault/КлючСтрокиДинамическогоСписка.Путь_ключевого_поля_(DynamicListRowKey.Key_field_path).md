КлючСтрокиДинамическогоСписка (DynamicListRowKey)

<Путь ключевого поля> (<Key field path>)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Произвольный.   
Содержит значения ключевого поля. Если путь ключевого поля не является корректным идентификатором, то обращение к нему возможно с использованием оператора [] или метода [Получить](КлючСтрокиДинамическогоСписка.Получить_(DynamicListRowKey.Get).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
