Описание:

Поле списка предназначено для выбора одного или нескольких значений из списка и является визуальным представлением объекта [СписокЗначений](СписокЗначений.md). Строки поля списка могут содержать пометки, аналогичные флажкам, при помощи которых можно отмечать выбранные значения. Все действия по изменению списка (добавлению элементов в список, удаление из списка, сортировка списка) проводятся при помощи связанного с полем списка значений.
