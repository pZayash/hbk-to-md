РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

УстановитьАктивность (SetActive)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьАктивность(<Активность>)

Параметры:

<Активность> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак активности записи. [Истина](lang__def_BooleanTrue.md) - записи учитываются в итогах регистра. [Ложь](lang__def_BooleanFalse.md) - записи не учитываются в итогах регистра.

Описание:

Устанавливает значение свойства [Активность](РегистрБухгалтерииЗапись.Имя_регистра_бухгалтерии.Активность_(AccountingRegisterRecord.Accounting_register_name.Active).md) у всех записей, входящих в набор.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| НаборЗаписей.УстановитьАктивность(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
