ЗащищенноеСоединениеКриптоПро (CryptoProSecureConnection)

СертификатКлиента (ClientCertificate)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [СертификатКлиентаОС](СертификатКлиентаОС_(OSClientCertificate).md), [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), [СертификатКлиентаФайл](СертификатКлиентаФайл_(FileClientCertificate).md), [Неопределено](lang__def_Undefined.md).   
Объект клиентского сертификата, если используется.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если [Неопределено](lang__def_Undefined.md) - сертификат клиента не предоставляется.

Использование в версии:

Доступен, начиная с версии 8.3.24.
