Элементы коллекции:

[КонтурПолигональногоОбъектаГеографическойСхемы](КонтурПолигональногоОбъектаГеографическойСхемы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

[Удалить (Delete)](КонтурыПолигональногоОбъектаГеографическойСхемы.Удалить.md)  

Описание:

Предоставляет доступ к коллекции контуров полигонального объекта (замкнутых линий), из которых состоит полигон.

Доступность:

Сервер, толстый клиент, внешнее соединение.
