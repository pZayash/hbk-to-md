Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения измерений регистра сведений. Имена свойств совпадают с именами измерений, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В качестве типа допускается использовать любые типы, кроме [ДвоичныеДанные](ДвоичныеДанные.md) и строки неограниченной длины.

Доступен, начиная с версии 8.0.
