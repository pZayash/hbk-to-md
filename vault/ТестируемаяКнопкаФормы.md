Описание:

Представляет абстракцию элемента управления [КнопкаФормы](КнопкаФормы.md). Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством [Вид](ТестируемаяКнопкаФормы.Вид.md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
