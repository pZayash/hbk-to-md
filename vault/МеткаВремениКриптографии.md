Доступен, начиная с версии 8.3.20.

Свойства:

[Дата (Date)](МеткаВремениКриптографии.Дата.md)  
[Подписи (Signatures)](МеткаВремениКриптографии.Подписи.md)  

Описание:

Содержит данные метки времени, полученной из подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Пример:

|  |
| --- |
| КонтейнерПодписей = МенеджерКриптографии.ПолучитьКонтейнерПодписейКриптографии(РезультатПодписи); Для каждого Подпись из КонтейнерПодписей.Подписи Цикл     Если (Подпись.МеткаВремениПодписи <> Неопределено) Тогда             Сообщить("Дата: " + Подпись.МеткаВремениПодписи.Дата);             Сообщить("СертификатПодписи.ДатаОкончания: " + Подпись.МеткаВремениПодписи.Подписи[0].СертификатПодписи.ДатаОкончания);         КонецЕсли; КонецЦикла; |

Использование в версии:
