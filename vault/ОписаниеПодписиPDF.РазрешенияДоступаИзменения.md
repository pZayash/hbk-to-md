ОписаниеПодписиPDF (PDFSignatureDescription)

РазрешенияДоступаИзменения (ModificationAccessPermissions)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [РазрешенияДоступаИзмененияPDF](РазрешенияДоступаИзмененияPDF.md).   
Содержит разрешенные действия после подписания. Имеет смысл только для подписи типа [Сертифицирующая](ТипПодписиPDF.Сертифицирующая.md).   
По умолчанию: [Нет](РазрешенияДоступаИзмененияPDF.Нет.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
