СоставФормНачальнойСтраницы (HomePageForms)

ПраваяКолонка (RightColumn)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив.md).   
Содержит список форм начальной страницы, размещенных в правой колонке.  
Можно использовать стандартные формы.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Элемент массива (тип Строка) содержит полное имя объекта метаданных формы.

Использование в версии:

Доступен, начиная с версии 8.3.5.
