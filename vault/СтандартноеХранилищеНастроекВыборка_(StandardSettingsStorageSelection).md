Описание:

Объект этого типа возвращается методом [Выбрать](СтандартноеХранилищеНастроекМенеджер.Выбрать_(StandardSettingsStorageManager.Select).md) объекта [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md) и представляет собой специализированный способ перебора содержимого хранилища. Обход содержимого хранилища выполняется системой динамически. Это означает, что использование выборки не считывает все настройки из хранилища сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить хранилища, содержащие большие количества настроек, и не загружает в память всю выборку.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.1.
