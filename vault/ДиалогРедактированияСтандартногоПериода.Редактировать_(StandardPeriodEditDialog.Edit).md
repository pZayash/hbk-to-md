ДиалогРедактированияСтандартногоПериода (StandardPeriodEditDialog)

Редактировать (Edit)

Доступен, начиная с версии 8.2.

Синтаксис:

Редактировать()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - была нажата кнопка "ОК", [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает диалог редактирования.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [Показать](ДиалогРедактированияСтандартногоПериода.Показать_(StandardPeriodEditDialog.Show).md).
