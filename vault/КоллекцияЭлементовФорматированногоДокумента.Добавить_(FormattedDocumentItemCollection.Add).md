КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Добавить(<Значение>, <ТипЭлемента>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Элемент, добавляемый в коллекцию.  
В зависимости от параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.  
Если значение параметра [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), то в коллекцию будет добавлено строковое представление переданного значения.  
Если значение параметра [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md), то тип параметра <Значение> должен быть [Картинка](Картинка_(Picture).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Строка](lang__def_String.md), содержащая URL картинки. В противном случае вызывается исключение.   
Если значение параметра [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md), то значение параметра игнорируется и в коллекцию добавляется перевод строки.  
Если значение параметров [Неопределено](lang__def_Undefined.md), а также коллекция элементов форматированного документа содержит только параграфы, то в коллекцию добавляется пустой параграф.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ТипЭлемента> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Тип элемента, который нужно добавить в коллекцию. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Элементами коллекции могут быть:  
1) [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md)  
2) [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md)  
3) [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md)  
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Значение по умолчанию: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md).

Возвращаемое значение:

Тип: [ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента_(FormattedDocumentParagraph).md), [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md).   
Возвращает добавленный элемент.

Описание:

Добавляет элемент в конец коллекции элементов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Добавление картинки доступно только на сервере.  
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.21.
