Описание:

Предопределенная табличная часть вида расчета - список ведущих видов расчета. Такая табличная часть определена для всех планов видов расчета.  
Имеет колонки - ВидРасчета(CalculationKind) типа [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)> и Предопределенный(Predefined) типа [Булево](lang__Булево_(Boolean).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
