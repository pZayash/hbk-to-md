ПочтовыеАдреса (MailAddresses)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Строкой

Синтаксис:

Добавить(<ПочтовыйАдрес>)

Параметры:

<ПочтовыйАдрес> (обязательный)

Тип: [Строка](lang__def_String.md).   
Почтовый адрес.

Описание варианта метода:

Добавляет почтовый адрес в коллекцию почтовых адресов как полный почтовый адрес.

Вариант синтаксиса: Пользователь и сервер

Синтаксис:

Добавить(<Пользователь>, <Сервер>)

Параметры:

<Пользователь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя.

<Сервер> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя сервера.

Описание варианта метода:

Добавляет почтовый адрес в коллекцию почтовых адресов как пару (пользователь, сервер).

Возвращаемое значение:

Тип: [ПочтовыйАдрес](ПочтовыйАдрес_(MailAddress).md).   

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
