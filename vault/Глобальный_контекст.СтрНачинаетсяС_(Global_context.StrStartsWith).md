Глобальный контекст (Global context)

СтрНачинаетсяС (StrStartsWith)

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Синтаксис:

СтрНачинаетсяС(<Строка>, <СтрокаПоиска>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, начало которой проверяется на совпадение с подстрокой поиска.

<СтрокаПоиска> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, содержащая предполагаемое начало строки.  
В случае если переданное значение является пустой строкой генерируется исключительная ситуация.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - строка начинается с указанной подстроки, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, что строка начинается с указанной подстроки. Определение выполняется с учетом регистра, аналогично функции [Найти](Глобальный_контекст.Найти_(Global_context.Find).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [СтрЗаканчиваетсяНа](Глобальный_контекст.СтрЗаканчиваетсяНа_(Global_context.StrEndsWith).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6 (в режиме совместимости с версией 8.3.6 и последующими).

Описание изменено в версии 8.3.15.
