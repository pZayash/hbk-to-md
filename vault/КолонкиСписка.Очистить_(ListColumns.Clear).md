КолонкиСписка (ListColumns)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все колонки из списка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Удалим из списка справочника, с которым связано табличное поле,  // все колонки:  Список.Колонки.Очистить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
