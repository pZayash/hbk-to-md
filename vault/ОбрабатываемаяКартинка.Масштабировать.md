ОбрабатываемаяКартинка (ProcessingPicture)

Масштабировать (Scale)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Масштабировать(<Коэффициент>)

Параметры:

<Коэффициент> (обязательный)

Тип: Число.   
Коэффициент масштабирования.

Описание:

Изменяет размеры картинки, масштабируя ее, используя переданный коэффициент в процентах. Коэффициент должен быть больше нуля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
