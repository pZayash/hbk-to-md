StartedAt (StartedAt)

Описание:

Тип: Дата.   
Содержит момент запуска рабочего процесса. Если процесс не запущен, то содержит нулевую дату.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
