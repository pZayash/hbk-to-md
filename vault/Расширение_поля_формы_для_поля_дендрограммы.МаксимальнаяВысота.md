Расширение поля формы для поля дендрограммы (Form field extension for a dendrogram field)

МаксимальнаяВысота (MaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит максимальную высоту элемента.  
Используется, если свойство [АвтоМаксимальнаяВысота](Расширение_поля_формы_для_поля_дендрограммы.АвтоМаксимальнаяВысота.md) установлено в значение Ложь.  
Задается в символах.  
Значение 0 означает, что максимальная высота не ограничена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
