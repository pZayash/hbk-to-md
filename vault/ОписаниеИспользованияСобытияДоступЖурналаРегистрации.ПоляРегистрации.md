ОписаниеИспользованияСобытияДоступЖурналаРегистрации (EventLogAccessEventUseDescription)

ПоляРегистрации (LoggedFields)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Поля, которые в случае участия их в результате выборки запроса будут выведены в таблицу значений (поле Данные.Данные события журнала регистрации).   
Поля регистрации задаются массивом, элементом которого является либо строка - имя поля, либо массив строк - имен полей. Первый уровень массива определяет набор ключевых полей, которые будут зарегистрированы.   
Если элементом массива является массив строк, то в этом массиве описываются альтернативные ключевые поля.   
В случае возможности вывода нескольких альтернативных ключевых полей выводится поле с меньшим индексом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
