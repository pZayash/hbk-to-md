# ДокументМенеджер.<Имя документа>.ВыбратьПоСсылкам

ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [Массив](Массив_(Array).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Источник ссылок на документы.

<Колонка> (необязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Колонка источника ссылок. [Число](lang__def_Number.md) представляет индекс колонки источника. [Строка](lang__def_String.md) представляет имя колонки. Если значением параметра является [Неопределено](lang__def_Undefined.md), то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив_(Array).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [ДокументВыборка.\<Имя документа\>](ДокументВыборка.Имя_документа_(DocumentSelection.Document_name).md).   

Описание:

Формирует выборку документов по заданному источнику ссылок на документы соответствующего типа. Если источник содержит значения, отличнные от ссылок на документы соответствующего типа, или ссылки на несуществующие документы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    РасходТовара.Ссылка КАК Ссылка     |ИЗ     |    Документ.РасходТовара КАК РасходТовара     |"; Таблица = Запрос.Выполнить()     .Выгрузить(ОбходРезультатаЗапроса.Прямой); Выборка = Документы.РасходТовара.ВыбратьПоСсылкам(Таблица, 0); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
