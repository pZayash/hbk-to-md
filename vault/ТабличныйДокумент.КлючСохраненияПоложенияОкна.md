ТабличныйДокумент (SpreadsheetDocument)

КлючСохраненияПоложенияОкна (WindowOptionsKey)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит ключ для сохранения размеров и положения окна данной формы. Параметры окна, настроенные и сохраненные с данным ключом, впоследствии восстанавливаются при следующем показе таблицы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
