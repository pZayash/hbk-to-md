Доступен, начиная с версии 8.0.

Чтение и запись.

Описание:

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Содержит текст рисунка.

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), [Примечание](ТипРисункаТабличногоДокумента.Примечание.md). Для удаления примечания у области табличного документа необходимо установить данному свойству пустую строку.

Доступен, начиная с версии 8.0.
