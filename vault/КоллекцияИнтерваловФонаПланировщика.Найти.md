КоллекцияИнтерваловФонаПланировщика (PlannerBackgroundIntervalCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Цвет](Цвет.md).   
Цвет, который должен быть установлен у искомого элемента.

Возвращаемое значение:

Тип: [ИнтервалФонаПланировщика](ИнтервалФонаПланировщика.md), Неопределено.   
Если указанный элемент отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента измерения планировщика по указанному значению свойства [Цвет](ИнтервалФонаПланировщика.Цвет.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
