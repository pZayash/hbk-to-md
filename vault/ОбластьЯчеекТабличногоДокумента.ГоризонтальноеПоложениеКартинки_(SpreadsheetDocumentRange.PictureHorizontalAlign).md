ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ГоризонтальноеПоложениеКартинки (PictureHorizontalAlign)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md).   
Определяет способ горизонтального выравнивания картинки внутри ячейки табличного документа. По умолчанию – [Лево](ГоризонтальноеПоложение.Лево_(HorizontalAlign.Left).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
