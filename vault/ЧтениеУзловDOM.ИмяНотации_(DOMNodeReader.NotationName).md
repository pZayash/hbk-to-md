ЧтениеУзловDOM (DOMNodeReader)

ИмяНотации (NotationName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя нотации для узлов типа [Сущность](ТипУзлаXML.Сущность_(XMLNodeType.Entity).md) и [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md), определяющих внешнюю неразбираемую сущность (external unparsed entity).  
Для типов узлов, для которых идентификатор не определен, содержит пустую строку.  
Если имя нотации не пустая строка и текущий узел [Сущность](ТипУзлаXML.Сущность_(XMLNodeType.Entity).md) или [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md), то это означает, что сущность (ссылка на сущность) неразбираемая (external unparsed entity) и также должны быть непустыми [ЧтениеУзловDOM.ПубличныйИдентификатор](ЧтениеУзловDOM.ПубличныйИдентификатор_(DOMNodeReader.PublicId).md) или [ЧтениеУзловDOM.СистемныйИдентификатор](ЧтениеУзловDOM.СистемныйИдентификатор_(DOMNodeReader.SystemId).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
