ИтераторУзловDOM (DOMNodeIterator)

РаскрыватьСсылкиНаСущности (ExpandEntityReference)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак обхода ссылок на сущности при обходе узлов DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.

Использование в версии:

Доступен, начиная с версии 8.1.
