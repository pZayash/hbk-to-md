КолонкиСхемыЗапроса (QuerySchemaColumns)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Вариант синтаксиса: По псевдониму

Синтаксис:

Найти(<Псевдоним>)

Параметры:

<Псевдоним> (обязательный)

Тип: Строка.   
Псевдоним для поиска.

Описание варианта метода:

Поиск колонки выполняется по свойству [Псевдоним](КолонкаСхемыЗапроса.Псевдоним.md).

Вариант синтаксиса: По выражению

Синтаксис:

Найти(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md), [ВложеннаяТаблицаСхемыЗапроса](ВложеннаяТаблицаСхемыЗапроса.md).   
Выражение для поиска.

Описание варианта метода:

Поиск по выражению, содержащемся в колонке.

Возвращаемое значение:

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса.md), [КолонкаВложеннаяТаблицаСхемыЗапроса](КолонкаВложеннаяТаблицаСхемыЗапроса.md).   

Описание:

Выполняет поиск колонки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
