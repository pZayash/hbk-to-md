# БизнесПроцессОбъект.<Имя бизнес-процесса>.ПриУстановкеНовогоНомера

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПриУстановкеНовогоНомера (OnSetNewNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриУстановкеНовогоНомера(<СтандартнаяОбработка>, <Префикс>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная генерация нового номера производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Префикс>

Тип: [Строка](lang__def_String.md).   
Префикс, который будет использоваться для генерации номера. Может быть изменен в процедуре-обработчике события.

Описание:

Возникает в момент, когда выполняется установка нового номера бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Параметр <Префикс> может быть указан, если [Номер](БизнесПроцессОбъект.Имя_бизнес-процесса.Номер_(BusinessProcessObject.Business_process_name.Number).md) имеет тип [Строка](lang__def_String.md).

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Доступен, начиная с версии 8.0.
