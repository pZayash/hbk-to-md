Описание:

Динамический список предоставляет разработчику возможность описать в форме запрос практически произвольного вида, указать условное оформление данных, порядок сортировки, отборы и группировки данных.  
Ограничения:

- при указанной основной таблице, в запросах нельзя использовать объединения (иначе основная таблица должна присутствовать в каждой части объединения);
- при указанной основной таблице или не пустом списке ключевых полей, в запросах нельзя использовать упорядочивание.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
