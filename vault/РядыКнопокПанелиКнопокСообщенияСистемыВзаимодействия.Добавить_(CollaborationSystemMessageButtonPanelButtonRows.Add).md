Добавить (Add)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonRow).md).   

Описание:

Добавляет ряд кнопок в коллекцию рядов панели кнопок сообщения системы взаимодействия в конец коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
