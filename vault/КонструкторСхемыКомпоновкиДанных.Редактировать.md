КонструкторСхемыКомпоновкиДанных (DataCompositionSchemaWizard)

Редактировать (Edit)

Доступен, начиная с версии 8.1.

Синтаксис:

Редактировать(<Форма>)

Параметры:

<Форма> (обязательный)

Тип: [Форма](Форма.md).   
Форма-владелец.

Описание:

Открывает окно конструктора схемы компоновки данных.  
При завершении редактирования конструктор вызывает событие [ОбработкаВыбора](Форма.ОбработкаВыбора.md) у формы, которая была назначена в качестве формы владельца вызванного конструктора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.1.
