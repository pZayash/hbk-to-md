Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

МаксимальнаяШирина (MaxWidth)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную ширину элемента.  
Используется, если свойство [АвтоМаксимальнаяШирина](Расширение_поля_формы_для_поля_PDF-документа.АвтоМаксимальнаяШирина_(Form_extension_for_a_PDF_document_field.AutoMaxWidth).md) установлено в значение [Ложь](lang__def_BooleanFalse.md).  
[Ложь](lang__def_BooleanFalse.md).  
Задается в символах.  
Значение 0 означает, что максимальная ширина не ограничена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.21.
