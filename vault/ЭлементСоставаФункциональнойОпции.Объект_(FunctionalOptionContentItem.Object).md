ЭлементСоставаФункциональнойОпции (FunctionalOptionContentItem)

Объект (Object)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Позволяет получить доступ к объекту метаданных, который включен в [Состав](ОбъектМетаданных_ФункциональнаяОпция.Состав_(MetadataObject_FunctionalOption.Content).md) объекта [ОбъектМетаданных: ФункциональнаяОпция](ОбъектМетаданных_ФункциональнаяОпция_(MetadataObject_FunctionalOption).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
