ПолосаРегулирования (TrackBar)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона полосы регулирования.

Доступность:

Толстый клиент.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ПолосаРегулирования.ПрозрачныйФон_(TrackBar.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
