ВнешнийИсточникДанныхКубТаблицаИзмеренияМенеджер.<Имя внешнего источника>.<Имя куба>.<Имя таблицы измерения> (ExternalDataSourceCubeDimensionTableManager.<External source name>.<Cube name>.<Dimension table name>)

ПолучитьДанныеВыбора (GetChoiceData)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьДанныеВыбора(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: [Структура](Структура.md).   
Содержит параметры для формирования списка значений.  
Структура, в которой обязательно должны содержаться следующие ключи:

- Отбор (Filter) - отбор, используемый для поиска данных, тип [Структура](Структура.md); значения элементов отбора могут иметь тип [Массив](Массив.md), [СписокЗначений](СписокЗначений.md), [ФиксированныйМассив](ФиксированныйМассив.md) или иной тип, не являющийся коллекцией; для типов [Массив](Массив.md), [СписокЗначений](СписокЗначений.md), [ФиксированныйМассив](ФиксированныйМассив.md) сравнение выполняется с видом сравнения [ВСписке](ВидСравнения.ВСписке.md), иначе вид сравнения [Равно](ВидСравнения.Равно.md);
- СтрокаПоиска (SearchString) - содержит строку, используемую при поиске данных; для быстрого выбора содержит значение Неопределено;
- СпособПоискаСтроки (StringSearchMode) - указывается способ поиска при вводе по строке; тип [СпособПоискаСтрокиПриВводеПоСтроке](СпособПоискаСтрокиПриВводеПоСтроке.md); параметр может отсутствовать.

Возвращаемое значение:

Тип: [СписокЗначений](СписокЗначений.md).   

Описание:

Формирует список значений данных выбора при вводе по строке, автоподборе текста и быстром выборе.  
Если требуется выполнить поиск более 32 слов или 1000 символов, то свойство [ОграничиватьСтрокуПоиска](СписокПолнотекстовогоПоиска.ОграничиватьСтрокуПоиска.md) требуется установить в значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если в модуле менеджера в методе указано два параметра, то будет вызван метод [ПолучитьДанныеВыбора](Глобальный_контекст.ПолучитьДанныеВыбора.md) глобального контекста.  
Выполнение метода приводит к вызову события [ОбработкаПолученияДанныхВыбора](ВнешнийИсточникДанныхКубТаблицаИзмеренияМенеджер.Имя_внешнего_источника.Имя_куба.Имя_таблицы_измерения.ОбработкаПолученияДанныхВыбора.md).  
  
В режиме совместимости конфигурации [Версия8\_3\_17](РежимСовместимости.Версия8_3_17.md) и ниже рекомендуется устанавливать свойство [ОграничиватьСтрокуПоиска](СписокПолнотекстовогоПоиска.ОграничиватьСтрокуПоиска.md) в значение Истина, чтобы при выполнении сложных запросов генерировалась ошибка.

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.18.
