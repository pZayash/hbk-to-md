СертификатКриптографии (CryptoCertificate)

НачатьВыгрузку (BeginUnloading)

Доступен, начиная с версии 8.3.6.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- <ВыгруженныеДанные> - данные типа [ДвоичныеДанные](ДвоичныеДанные.md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание варианта метода:

Передает выгруженные данные в параметре типа [ДвоичныеДанные](ДвоичныеДанные.md).

Вариант синтаксиса: Данные в файл

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- <ИмяФайла> - имя файла, в который выгружен сертификат.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который будет выгружен сертификат.

Описание варианта метода:

Выгружает сертификат в файл.

Описание:

Начинает выгрузку сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии.md), метод [Выгрузить](СертификатКриптографии.Выгрузить.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
