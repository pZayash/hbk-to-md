Глобальный контекст (Global context)

НачатьПолучениеФайлов (BeginGettingFiles)

Не рекомендуется использовать, начиная с версии 8.3.15.

Рекомендуется использовать:

- [НачатьПолучениеФайловССервера](Глобальный_контекст.НачатьПолучениеФайловССервера.md)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеФайлов(<ОписаниеОповещения>, <ПолучаемыеФайлы>, <РасположениеФайлов>, <Интерактивно>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <ПолученныеФайлы> – массив объектов типа [ОписаниеПереданногоФайла](ОписаниеПереданногоФайла.md) или Неопределено, если файлы не получены.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ПолучаемыеФайлы> (обязательный)

Тип: [Массив](Массив.md).   
Массив объектов типа [ОписаниеПередаваемогоФайла](ОписаниеПередаваемогоФайла.md).  
Каждый объект описывает получаемый файл:

- [Хранение](ОписаниеПередаваемогоФайла.Хранение.md) - расположение данных во временном хранилище или в информационной базе;
- [Имя](ОписаниеПередаваемогоФайла.Имя.md) - это локальный путь, по которому должен быть расположен получаемый файл. Это может быть либо полный путь, либо часть пути, которая будет объединена со значением параметра <РасположениеФайлов>. Это также может быть имя файла или имя каталога (каталогов) + имя файла.

<РасположениеФайлов> (необязательный)

Тип: [ДиалогВыбораФайла](ДиалогВыбораФайла.md), Строка.   
Определяет общее расположение помещаемых файлов в локальной файловой системе.  
Может быть задан строкой либо заранее подготовленным диалогом выбора файлов в режиме выбора каталога или сохранения файлов.

<Интерактивно> (необязательный)

Тип: Булево.   
Истина - задан интерактивный режим, в котором пользователю показывается диалог, в котором он может выбрать локальное расположение файлов.  
Ложь - на сервер передается список файлов, описанный в параметре <ПолучаемыеФайлы>.  
В интерактивном режиме строка, переданная в параметре <РасположениеФайлов>, рассматривается как начальное значение для выбора пути.  
Значение по умолчанию: Истина.

Описание:

Начинает получение набора файлов и сохраняет их в локальную файловую систему пользователя.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьФайлы](Глобальный_контекст.ПолучитьФайлы.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.15.
