Глобальный контекст (Global context)

ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия (CollaborationSystemUsersChoiceFormGetProcessing)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия(<НазначениеВыбора>, <Форма>, <Обсуждение>, <Параметры>, <ВыбраннаяФорма>, <СтандартнаяОбработка>)

Параметры:

<НазначениеВыбора>

Тип: [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md).   
Назначение выбора пользователей системы взаимодействия.

<Форма>

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [Неопределено](lang__def_Undefined.md).   
Форма, в которой находится контекстное обсуждение, если выполняется автоподбор пользователей в контекстном обсуждении.

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md), [Неопределено](lang__def_Undefined.md).   
Обсуждение, в котором выполняется подбор пользователей.

<Параметры>

Тип: [Структура](Структура_(Structure).md).   
Параметры получения формы.

<ВыбраннаяФорма>

Тип: [Строка](lang__def_String.md).   
Имя открываемой формы.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события.  
Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет (выбираются установленные разработчиком форма и параметры).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при автоподборе пользователя системы взаимодействия.  
При подборе пользователей обработчик вызывается в том случае, если у формы не задан обработчик события [ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия](ФормаКлиентскогоПриложения.ОбработкаПолученияФормыВыбораПользователейСистемыВзаимодействия_(ClientApplicationForm.CollaborationSystemUsersChoiceFormGetProcessing).md) или если после его выполнения <СтандартнаяОбработка> имеет значение [Истина](lang__def_BooleanTrue.md).  
При вызове обработчика (и для формы, и для клиентского приложения) в параметре <Параметры> передается структура, в которой заданы следующие значения:

- РежимВыбора – [Истина](lang__def_BooleanTrue.md),
- МножественныйВыбор – [Истина](lang__def_BooleanTrue.md),
- ЗакрыватьПриЗакрытииВладельца – [Истина](lang__def_BooleanTrue.md).

  
Если после вызова обработчика в параметре <ВыбраннаяФорма> содержится не пустая строка, то в выпадающем списке отображается гиперссылка "Показать все".  
При нажатии гиперссылки открывается заданная форма с параметрами, заполненными из структуры, возвращенной в параметре <Параметры>. Владельцем формы выбора устанавливается форма, в которой отображается обсуждение.  
  
При закрытии формы выбора значение, переданное в параметре метода [Закрыть](ФормаКлиентскогоПриложения.Закрыть_(ClientApplicationForm.Close).md), добавляется к списку пользователей.  
При подборе получателей сообщения (<НазначениеВыбора> имеет значение [ПолучательСообщения](НазначениеВыбораПользователейСистемыВзаимодействия.ПолучательСообщения_(CollaborationSystemUsersChoicePurpose.MessageRecipient).md)) значение [Неопределено](lang__def_Undefined.md) интерпретируется как "Никого не оповещать". В остальных случаях значение [Неопределено](lang__def_Undefined.md) игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
