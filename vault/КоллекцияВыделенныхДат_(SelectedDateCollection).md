# КоллекцияВыделенныхДат

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Элементы управления](Элементы_управления.md) › [Поле календаря](Поле_календаря.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[Дата](lang__def_Date.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию выделенных в календаре дат.

Доступность:

Толстый клиент.

[ПолеКалендаря](ПолеКалендаря_(CalendarBox).md), свойство [ВыделенныеДаты](ПолеКалендаря.ВыделенныеДаты_(CalendarBox.SelectedDates).md)
