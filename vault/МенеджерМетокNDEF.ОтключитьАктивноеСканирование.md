МенеджерМетокNDEF (TagManagerNDEF)

ОтключитьАктивноеСканирование (StopActiveScanning)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОтключитьАктивноеСканирование()

Описание:

При вызове метода завершается текущая сессия сканирования, скрывается интерфейс сканирования.   
При вызове данного метода в отсутствие сессии сканирования, то никаких действий выполнено не будет.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
