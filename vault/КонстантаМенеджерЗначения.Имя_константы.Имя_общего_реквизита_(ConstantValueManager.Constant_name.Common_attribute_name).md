КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

<Имя общего реквизита> (<Common attribute name>)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор таких свойств определяется набором общих реквизитов, в которых участвует данная константа и для которых [РазделениеДанных](ОбъектМетаданных_ОбщийРеквизит.РазделениеДанных_(MetadataObject_CommonAttribute.DataSeparation).md) = [Разделять](РазделениеДанныхОбщегоРеквизита.Разделять_(CommonAttributeDataSeparation.Separate).md) и [ИспользованиеРазделяемыхДанных](ОбъектМетаданных_ОбщийРеквизит.ИспользованиеРазделяемыхДанных_(MetadataObject_CommonAttribute.SeparatedDataUse).md) = [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно_(CommonAttributeSeparatedDataUse.IndependentlyAndSimultaneously).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
