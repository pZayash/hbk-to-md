# МенеджерОтображенияРекламы.ЗагрузитьРекламныйБаннерАсинх

МенеджерОтображенияРекламы (AdRepresentationManager)

ЗагрузитьРекламныйБаннерАсинх (DownloadAdBannerAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗагрузитьРекламныйБаннерАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Булево](lang__Булево_(Boolean).md), признак загрузки баннера.

Описание:

Асинхронная функция. Загружает из сети рекламный баннер. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [УстановитьОтображениеРекламногоБаннера](МенеджерОтображенияРекламы.УстановитьОтображениеРекламногоБаннера_(AdRepresentationManager.SetAdBannerRepresentation).md) рекламный баннер будет отображен без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы_(AdRepresentationManager).md), метод [НачатьЗагрузкуРекламногоБаннера](МенеджерОтображенияРекламы.НачатьЗагрузкуРекламногоБаннера_(AdRepresentationManager.BeginAdBannerDownloading).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
