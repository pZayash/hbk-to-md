АдминистрированиеСервера (ServerAdministration)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый АдминистрированиеСервера(<АдресАгента>, <Порт>)

Параметры:

<АдресАгента> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя (символический или IP-адрес) компьютера, на котором расположен сервис ras.

<Порт> (необязательный)

Тип: [Число](lang__def_Number.md).   
Порт, на котором расположен сервис ras.

Описание:

Создает объект администрирования сервера.

Использование в версии:

Доступен, начиная с версии 8.3.14.
