COM-класс (ISecurityProfileCOMClass)

Name (Name)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя COM-класса. Используется в качестве ключа при поиске. Должно быть уникальным для всех COM-классов данного профиля безопасности.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
