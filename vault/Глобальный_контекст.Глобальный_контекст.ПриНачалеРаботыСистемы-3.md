Глобальный контекст (Global context)

ПриНачалеРаботыСистемы (OnStart)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриНачалеРаботыСистемы()

Описание:

Возникает при запуске 1С:Предприятия в режиме приложения после открытия главного окна. В процедуре-обработчике могут быть выполнены различные действия, необходимые при запуске программы, например, открыты некоторые формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
