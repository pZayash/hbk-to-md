# РежимЧтенияЗаписиХранилищаДвоичныхДанных.ТолькоЧтение

РежимЧтенияЗаписиХранилищаДвоичныхДанных (BinaryDataStorageReadWriteMode)

ТолькоЧтение (ReadOnly)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Описывает режим хранилища двоичных данных только для чтения. В этом режиме двоичные данные, записываемые в информационную   
базу, будут записаны в базу данных, а не в хранилище (независимо от того, включено ли хранилище).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
