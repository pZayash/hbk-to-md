ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыравниваниеЭлементовИЗаголовков (ItemsAndTitlesAlign)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ВариантВыравниванияЭлементовИЗаголовков](ВариантВыравниванияЭлементовИЗаголовков_(ItemsAndTitlesAlignVariant).md).   
Устанавливает способ выравнивания элементов и их заголовков.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
