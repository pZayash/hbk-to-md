# ИндексыКоллекции.Удалить

ИндексыКоллекции (CollectionIndexes)

Удалить (Delete)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По объекту

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [ИндексКоллекции](ИндексКоллекции_(CollectionIndex).md).   
Удаляемый индекс (объект) коллекции.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Порядковый индекс объекта в коллекции.

Описание:

Удаляет индекс из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
