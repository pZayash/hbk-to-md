Значения

[Гистограмма (Column)](ТипСводнойДиаграммы.Гистограмма.md)  
[ГистограммаОбъемная (Column3D)](ТипСводнойДиаграммы.ГистограммаОбъемная.md)  

Описание:

Содержит варианты типов сводных диаграмм. Используется для определения свойства [ТипСводнойДиаграммы](СводнаяДиаграмма.ТипСводнойДиаграммы.md) объекта [СводнаяДиаграмма](СводнаяДиаграмма.md).

Доступность:

Тонкий клиент, сервер, толстый клиент.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: PivotChartType.

[СводнаяДиаграмма](СводнаяДиаграмма.md), свойство [ТипСводнойДиаграммы](СводнаяДиаграмма.ТипСводнойДиаграммы.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
