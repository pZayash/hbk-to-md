ЭлементКартинкаHTML (HTMLImageElement)

ЯвляетсяКартой (IsMap)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак того, что изображение использует возможности серверной навигационной карты.  
[Истина](lang__def_BooleanTrue.md) - использует.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
