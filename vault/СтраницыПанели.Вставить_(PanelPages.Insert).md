СтраницыПанели (PanelPages)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<НомерЭлемента>, <Имя>, <Заголовок>, <Значение>, <КартинкаЗаголовка>)

Параметры:

<НомерЭлемента> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую должен быть помещена страница.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя добавляемой страницы.

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка добавляемой страницы. Если страницы панели показываются в виде закладок, то определяет текст на закладке.

<Значение> (необязательный)

Тип: Произвольный.   
Значение, связанное со страницей.

<КартинкаЗаголовка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка, отображаемая на заголовке закладки.

Возвращаемое значение:

Тип: [СтраницаПанели](СтраницаПанели_(PanelPage).md).   

Описание:

Вставляет страницу в нужную позицию коллекции страниц панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| // ТаблицаРегистровНакопления - таблица значений, для которой   // создается страница панели  // СтрокаТаблицы - текущая строка таблицы  СтраницаПанели = ЭлементыФормы.ОсновнаяПанель.Страницы.Вставить(         ТаблицаРегистровНакопления.Индекс(СтрокаТаблицы),          СтрокаТаблицы.Имя,          СтрокаТаблицы.Представление); |

Использование в версии:

Доступен, начиная с версии 8.0.
