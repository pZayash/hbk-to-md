# ЗадачаМенеджер.<Имя задачи>.НайтиПоНомеру

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаМенеджер.<Имя задачи>](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md) › [Методы](ЗадачаМенеджер.Имя_задачи__Методы.md)

ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

НайтиПоНомеру (FindByNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоНомеру(<НомерЗадачи>)

Параметры:

<НомерЗадачи> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Номер искомой задачи. Тип передаваемого значения зависит от установки в конфигураторе представления номера задачи.

Возвращаемое значение:

Тип: [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)<[Имя задачи](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)>, [Неопределено](lang__def_Undefined.md).   
Ссылка на найденную задачу. Если задача не найдена, возвращается пустая ссылка.  
Если номер не задан (длина = 0), то будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск задачи по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
