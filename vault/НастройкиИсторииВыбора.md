Доступен, начиная с версии 8.3.1.

Описание:

Содержит настройки, в которых хранится история выбора значений в поле ввода формы клиентского приложения, а также история выбора участников видеозвонков.

Доступность:

Сервер, толстый клиент.   
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.16.
