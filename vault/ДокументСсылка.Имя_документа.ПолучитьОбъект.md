ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ДокументОбъект.](ДокументОбъект.Имя_документа.md)<[Имя документа](ДокументОбъект.Имя_документа.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект для чтения, изменения, добавления и удаления документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = ВыбДокумент.ПолучитьОбъект(); |

Использование в версии:

Доступен, начиная с версии 8.0.
