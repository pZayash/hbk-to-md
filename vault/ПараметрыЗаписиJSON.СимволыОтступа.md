ПараметрыЗаписиJSON (JSONWriterSettings)

СимволыОтступа (PaddingSymbols)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Определяет символы отступа, используемые при записи документа JSON.  
Свойство не используется, если свойство [ПереносСтрокJSON](ПереносСтрокJSON.md) установлено в значение [Нет](ПереносСтрокJSON.Нет.md).  
После создания объекта данное свойство имеет значение - пустая строка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
