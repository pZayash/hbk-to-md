ПараметрыФормированияКомандПоляПланировщика (PlannerFieldCommandGenerateParameters)

Дата (Date)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Дата, Неопределено.   
Содержит дату, если нажатие было произведено на элемент шкалы времени или на перенесенный заголовок шкалы времени.   
Иначе - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
