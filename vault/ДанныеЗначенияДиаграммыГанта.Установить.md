ДанныеЗначенияДиаграммыГанта (GanttChartValueData)

Установить (Set)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Установить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента.

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение.

Описание:

Установить значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, интеграция.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.3.21.
