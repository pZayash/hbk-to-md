ОбъектМетаданных: Ресурс (MetadataObject: Resource)

Индексирование (Indexing)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Индексирование](Индексирование_(Indexing).md).   
Вид дополнительных индексов, создаваемых для работы с данными объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для ресурсов регистра сведений.

Использование в версии:

Доступен, начиная с версии 8.1.
