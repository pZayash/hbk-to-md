БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: Булево.   
Истина - изменяется еще ни разу не записанный бизнес-процесс; Ложь - бизнес-процесс уже записан.

Описание:

Определяет, записан ли бизнес-процесс в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
