Глобальный контекст (Global context)

ПриИзмененииПараметровЭкрана (OnChangeDisplaySettings)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПриИзмененииПараметровЭкрана()

Описание:

Возникает при изменении информации, собранной в объекте [ИнформацияЭкранаКлиента](ИнформацияЭкранаКлиента_(ClientDisplayInformation).md): разрешение, ориентация и т.п.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.5.
