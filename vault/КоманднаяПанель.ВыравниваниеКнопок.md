КоманднаяПанель (CommandBar)

ВыравниваниеКнопок (ButtonsAlignment)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВыравниваниеКнопокКоманднойПанели](ВыравниваниеКнопокКоманднойПанели.md).   
Определяет тип выравнивания кнопок командной панели (по левому краю, по центру, по правому краю).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
