МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьИдентификаторПользователяИнформационнойБазы (GetInfoBaseUserID)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьИдентификаторПользователяИнформационнойБазы(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Идентификатор пользователя системы взаимодействия.

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает идентификатор того пользователя информационной базы, который соответствует пользователю системы взаимодействия. Если соответствующего пользователя в информационной базе нет, возвращается [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
