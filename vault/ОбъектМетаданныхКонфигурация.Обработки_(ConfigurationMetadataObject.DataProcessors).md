ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Обработки (DataProcessors)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все обработки, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Обработка](ОбъектМетаданных_Обработка_(MetadataObject_DataProcessor).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
