РежимВыделенияДиаграммы (ChartSelectionMode)

ВыделениеТочек (PointsSelection)

Доступен, начиная с версии 8.3.14.

Описание:

Позволяет выделять точки диаграммы с помощью мыши.  
Данный режим доступен только для диаграмм, поддерживающих ось точек и не поддерживающих ось серий (графики, гистограммы, радарные, биржевые).  
В диаграммах остальных типов интепретируется как [Нет](РежимВыделенияДиаграммы.Нет_(ChartSelectionMode.None).md).

Использование в версии:

Доступен, начиная с версии 8.3.14.
