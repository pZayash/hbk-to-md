СтандартныйПериод (StandardPeriod)

По датам периода

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СтандартныйПериод(<НачалоПериода>, <ОкончаниеПериода>)

Параметры:

<НачалоПериода> (обязательный)

Тип: Дата.   
Начальная дата периода.

<ОкончаниеПериода> (обязательный)

Тип: Дата.   
Дата окончания периода.

Описание:

Создает период с произвольным периодом по датам начала и конца периодов.

Использование в версии:

Доступен, начиная с версии 8.2.
