ПараметрыФормированияКомандПоляПланировщика (PlannerFieldCommandGenerateParameters)

Источник (Source)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ИсточникКомандПоляПланировщика](ИсточникКомандПоляПланировщика_(PlannerCommandSource).md).   
Содержит информацию о том, на какой объект было произведено нажатие.  
В зависимости от того, какой элемент содержит форматированную строку с гиперссылкой, заполняются разные свойства объекта типа [ПараметрыФормированияКомандПланировщика](ПараметрыФормированияКомандПланировщика_(PlannerCommandGenerateParameters).md).  
Если источник равен [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка_(PlannerCommandSource.URL).md), тогда:

- Заполнено свойство либо [Элементы](ПараметрыФормированияКомандПланировщика.Элементы_(PlannerCommandGenerateParameters.Items).md), либо [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения_(PlannerCommandGenerateParameters.DimensionItem).md), либо [Измерение](ПараметрыФормированияКомандПланировщика.Измерение_(PlannerCommandGenerateParameters.Dimension).md),
- если навигационная ссылка находится в заголовке элемента измерения, то свойство [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений_(PlannerCommandGenerateParameters.DimensionValues).md), либо [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandGenerateParameters.TimeScaleItem).md) заполнен, иначе содержит [Неопределено](lang__def_Undefined.md),
- [Дата](ПараметрыФормированияКомандПланировщика.Дата_(PlannerCommandGenerateParameters.Date).md) содержит [Неопределено](lang__def_Undefined.md),
- [НавигационнаяСсылка](ПараметрыФормированияКомандПланировщика.НавигационнаяСсылка_(PlannerCommandGenerateParameters.URL).md) заполнено,
- [Действие](ПараметрыФормированияКомандПланировщика.Действие_(PlannerCommandGenerateParameters.Action).md) содержит [Неопределено](lang__def_Undefined.md),
- Команды содержат стандартные команды [ПерейтиПоНавигационнойСсылке](СтандартнаяКомандаПланировщика.ПерейтиПоНавигационнойСсылке_(PlannerStandardCommand.GotoURL).md), [КопироватьНавигационнуюСсылку](СтандартнаяКомандаПланировщика.КопироватьНавигационнуюСсылку_(PlannerStandardCommand.CopyURL).md),
- КомандаПоУмолчанию заполнено командой ПерейтиПоНавигационнойСсылке.

Если источник равен [Действие](ИсточникКомандПланировщика.Действие_(PlannerCommandSource.Action).md) тогда:

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы_(PlannerCommandGenerateParameters.Items).md) заполнено,
- свойства [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения_(PlannerCommandGenerateParameters.DimensionItem).md), [Измерение](ПараметрыФормированияКомандПланировщика.Измерение_(PlannerCommandGenerateParameters.Dimension).md), [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений_(PlannerCommandGenerateParameters.DimensionValues).md), содержит [Неопределено](lang__def_Undefined.md), [Дата](ПараметрыФормированияКомандПланировщика.Дата_(PlannerCommandGenerateParameters.Date).md), [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandGenerateParameters.TimeScaleItem).md), [НавигационнаяСсылка](ПараметрыФормированияКомандПланировщика.НавигационнаяСсылка_(PlannerCommandGenerateParameters.URL).md) содержат [Неопределено](lang__def_Undefined.md),
- Список команд не содержит стандартных команд,
- <КомандаПоУмолчанию> заполнено командой [ВыполнитьДействие](СтандартнаяКомандаПланировщика.ВыполнитьДействие_(PlannerStandardCommand.ExecuteAction).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
