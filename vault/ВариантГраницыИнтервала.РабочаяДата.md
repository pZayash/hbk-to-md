ВариантГраницыИнтервала (IntervalBoundVariant)

РабочаяДата (WorkingDate)

Доступен, начиная с версии 8.0.

Описание:

Начало интервала задается датой началом дня рабочей даты, конец интервала - концом дня рабочей даты.

Использование в версии:

Доступен, начиная с версии 8.0.
