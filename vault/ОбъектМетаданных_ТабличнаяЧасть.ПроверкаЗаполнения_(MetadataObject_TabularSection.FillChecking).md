ОбъектМетаданных: ТабличнаяЧасть (MetadataObject: TabularSection)

ПроверкаЗаполнения (FillChecking)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ПроверкаЗаполнения](ПроверкаЗаполнения_(FillChecking).md).   
Устанавливает включение табличной части в автоматическую проверку заполнения, выполняемую системой.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
