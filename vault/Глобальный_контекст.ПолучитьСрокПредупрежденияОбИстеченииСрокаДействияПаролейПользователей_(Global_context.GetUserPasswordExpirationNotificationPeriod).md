Глобальный контекст (Global context)

ПолучитьСрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей (GetUserPasswordExpirationNotificationPeriod)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьСрокПредупрежденияОбИстеченииСрокаДействияПаролейПользователей()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает срок предупреждения об истечении срока действия паролей пользователей в секундах.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
