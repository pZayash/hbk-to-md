Глобальный контекст (Global context)

ПроверкаВстроенныхПокупок (InAppPurchasesValidation)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [МенеджерПроверкиВстроенныхПокупок](МенеджерПроверкиВстроенныхПокупок_(InAppPurchasesValidationManager).md).   
Предоставляет доступ к средствам проверки квитанций встроенных покупок.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
