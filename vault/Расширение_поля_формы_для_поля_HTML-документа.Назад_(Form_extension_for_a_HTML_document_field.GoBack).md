Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

Назад (GoBack)

Доступен, начиная с версии 8.2.

Синтаксис:

Назад()

Описание:

Перейти назад.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В веб-клиенте во всех веб-браузерах, в случае, когда в одном физическом окне веб-браузера расположены несколько полей HTML-документа (возможно в разных формах), метод, вызванный у любого из этих элементов, действует на все поля в хронологическом порядке. То есть переходы по ссылкам и изменение содержимого образуют общую историю в пределах одного физического окна браузера.  
В веб-браузерах Google Chrome и Safari метод не осуществляет навигацию по содержимому, заданному в виде строки (только по содержимому, заданному URL).

Использование в версии:

Доступен, начиная с версии 8.2.
