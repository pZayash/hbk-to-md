ЭлементDOM (DOMElement)

ПолучитьСоответствияПространствИмен (NamespaceMappings)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьСоответствияПространствИмен()

Возвращаемое значение:

Тип: [Соответствие](Соответствие.md).   

Описание:

Получает соответствие URI пространств имен.  
Ключом соответствия является префикс, значением - URI пространства имен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
