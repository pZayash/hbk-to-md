# Расширение табличного поля журнала документов.ПроверкаОтображенияНовойСтроки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Журналы документов](Журналы_документов.md) › [Расширение табличного поля журнала документов](Расширение_табличного_поля_журнала_документов_(Document_journal_table_box_extension).md) › [Свойства](Расширение_табличного_поля_журнала_документов__Свойства.md)

Расширение табличного поля журнала документов (Document journal table box extension)

ПроверкаОтображенияНовойСтроки (NewRowShowCheck)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВариантПроверкиОтображенияНовойСтроки](ВариантПроверкиОтображенияНовойСтроки_(NewRowShowCheckVariant).md).   
Если данное свойство имеет значение [СообщатьОНесоответствииОтбору](ВариантПроверкиОтображенияНовойСтроки.СообщатьОНесоответствииОтбору_(NewRowShowCheckVariant.FilterMismatchMessage).md), то после окончания ввода новой строки системой будет осуществлена проверка строки и, если строка будет не соответствовать установленному отбору, будет выдано сообщение.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
