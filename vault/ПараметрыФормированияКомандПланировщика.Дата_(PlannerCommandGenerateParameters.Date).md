ПараметрыФормированияКомандПланировщика (PlannerCommandGenerateParameters)

Дата (Date)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [Дата](ПараметрыФормированияКомандПоляПланировщика.Дата_(PlannerFieldCommandGenerateParameters.Date).md)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md), [Неопределено](lang__def_Undefined.md).   
Содержит дату, если нажатие было произведено на элемент шкалы времени или на перенесенный заголовок шкалы времени.   
Иначе - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
