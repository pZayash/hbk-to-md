Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ПередЗаписьюНаСервере (BeforeWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗаписьюНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ТекущийОбъект>

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи.

Описание:

Вызывается перед записью объекта на сервере. Можно отменить запись. Выполняется вне транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
