СправочникВыборка.<Имя справочника> (CatalogSelection.<Catalog name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [СправочникОбъект.](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)<[Имя справочника](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для чтения, изменения, добавления и удаления элемента, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = ВыборкаСправочника.ПолучитьОбъект(); |

Использование в версии:

Доступен, начиная с версии 8.0.
