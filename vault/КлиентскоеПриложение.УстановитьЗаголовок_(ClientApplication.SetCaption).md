КлиентскоеПриложение (ClientApplication)

УстановитьЗаголовок (SetCaption)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьЗаголовок(<ПредставлениеКлиентскогоПриложения>)

Параметры:

<ПредставлениеКлиентскогоПриложения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Представление клиентского приложения.

Описание:

Устанавливает заголовок приложения, которое используется при формировании заголовков окон.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) устанавливает заголовок главного окна.

См. также:

[КлиентскоеПриложение](КлиентскоеПриложение_(ClientApplication).md), метод [ПолучитьЗаголовок](КлиентскоеПриложение.ПолучитьЗаголовок_(ClientApplication.GetCaption).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
