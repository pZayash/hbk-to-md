ДоступныеТаблицыСхемыЗапроса (QuerySchemaAvailableTables)

Найти (Find)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы, которую требуется найти.

Возвращаемое значение:

Тип: [ГруппаДоступныхТаблицСхемыЗапроса](ГруппаДоступныхТаблицСхемыЗапроса_(QuerySchemaAvailableTablesGroup).md), [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса_(QuerySchemaAvailableTable).md).   

Описание:

Находит доступную таблицу, группу доступных таблиц или вложенную таблицу по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
