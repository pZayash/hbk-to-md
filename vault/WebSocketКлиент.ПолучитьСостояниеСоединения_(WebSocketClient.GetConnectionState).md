# WebSocketКлиент.ПолучитьСостояниеСоединения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [WebSocket](WebSocket.md) › [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md) › [Методы](WebSocketКлиент__Методы.md)

WebSocketКлиент (WebSocketClient)

ПолучитьСостояниеСоединения (GetConnectionState)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьСостояниеСоединения()

Возвращаемое значение:

Тип: [СостояниеWebSocketСоединения](СостояниеWebSocketСоединения_(WebSocketConnectionState).md).   

Описание:

Возвращает состояние WebSocket-соединения.  
Если у клиента нет объекта соединения, возвращает [Закрыто](СостояниеWebSocketСоединения.Закрыто_(WebSocketConnectionState.Closed).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
