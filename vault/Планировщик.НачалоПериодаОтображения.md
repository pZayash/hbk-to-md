Планировщик (Planner)

НачалоПериодаОтображения (BeginOfRepresentationPeriod)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Содержит начальную границу периода в планировщике. Перемещаение за указанную границу в планировщике невозможно. Если установлено значение Неопределено, начальная граница периода планировщика не установлена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
