ФакторЛинииТрендаДиаграммы (ChartTrendlineFactor)

НомерТочки (PointNumber)

Доступен, начиная с версии 8.3.10.

Описание:

В качестве фактора линии тренда (значения по оси ординат) используется номер точки.  
Нумерация точек начинается с 1.

Использование в версии:

Доступен, начиная с версии 8.3.10.
