ВидСравнения (ComparisonType)

Интервал (Interval)

Доступен, начиная с версии 8.0.

Описание:

Сравниваемое значение лежит в заданном интервале, не включая границы интервала.

Использование в версии:

Доступен, начиная с версии 8.0.
