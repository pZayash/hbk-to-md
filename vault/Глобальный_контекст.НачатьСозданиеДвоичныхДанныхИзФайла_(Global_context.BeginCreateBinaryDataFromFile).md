Глобальный контекст (Global context)

НачатьСозданиеДвоичныхДанныхИзФайла (BeginCreateBinaryDataFromFile)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьСозданиеДвоичныхДанныхИзФайла(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <ДвоичныеДанные> - Экземпляр объекта [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), содержащий данные из файла.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из которого надо создать [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание:

Начинает создание двоичных данных из файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

При вызове этого метода в веб-клиенте запрашивается разрешение пользователя на выполнение операции.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [СоздатьДвоичныеДанныеИзФайлаАсинх](Глобальный_контекст.СоздатьДвоичныеДанныеИзФайлаАсинх_(Global_context.CreateBinaryDataFromFileAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
