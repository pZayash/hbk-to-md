ТабличныйДокумент (SpreadsheetDocument)

ПолучитьКомпактныйДокумент (GetCompactDocument)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьКомпактныйДокумент(<Ширина>, <ИспользованиеШирины>)

Параметры:

<Ширина> (необязательный)

Тип: Число.   
Максимальная ширина содержимого табличного документа (в "средних символах").  
Если значение не задано, то максимальная ширина рассчитывается автоматически на основе информации об экране устройства пользователя. Если такой информации нет, то используется стандартное значение равное 20.

<ИспользованиеШирины> (необязательный)

Тип: [ИспользованиеШириныСжатияТабличногоДокумента](ИспользованиеШириныСжатияТабличногоДокумента.md).   
Задает использование ширины, переданной в качестве первого аргумента.

Описание:

Создает на основе текущего и возвращает новый табличный документ, сжатый по горизонтали (используется для адаптации документа к ширине экрана мобильного устройства).  
После вызова данного метода можно вызвать метод [УстановитьРастягиваниеПоГоризонтали](ТабличныйДокумент.УстановитьРастягиваниеПоГоризонтали.md) для настройки ширины колонок.

Доступность:

Сервер, толстый клиент.

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), метод [УстановитьРастягиваниеПоГоризонтали](ТабличныйДокумент.УстановитьРастягиваниеПоГоризонтали.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
