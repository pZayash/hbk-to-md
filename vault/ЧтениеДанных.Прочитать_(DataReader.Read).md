# ЧтениеДанных.Прочитать

ЧтениеДанных (DataReader)

Прочитать (Read)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Прочитать(<Количество>)

Параметры:

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется прочитать. Если не задано, то выполняется чтение всех данных до конца потока.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).   
Экземпляр объекта [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Описание:

Выполняет чтение двоичных данных заданного размера из потока. Если размер не указан, будут прочитаны все данные до конца потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
