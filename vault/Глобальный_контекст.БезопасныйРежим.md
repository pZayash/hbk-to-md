Глобальный контекст (Global context)

БезопасныйРежим (SafeMode)

Доступен, начиная с версии 8.2.

Синтаксис:

БезопасныйРежим()

Возвращаемое значение:

Тип: Булево, Строка.   
Строка, если установлен безопасный режим с указанием имени профиля безопасности.  
Истина, если установлен безопасный режим.  
Ложь, если безопасный режим не установлен.

Описание:

Получает текущее значение безопасного режима.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьБезопасныйРежим](Глобальный_контекст.УстановитьБезопасныйРежим.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
