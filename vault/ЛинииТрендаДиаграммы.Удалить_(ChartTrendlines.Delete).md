ЛинииТрендаДиаграммы (ChartTrendlines)

Удалить (Delete)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемой линии тренда.

Описание:

Удаляет линию тренда из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
