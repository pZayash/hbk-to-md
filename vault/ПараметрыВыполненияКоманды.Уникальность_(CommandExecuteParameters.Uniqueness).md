ПараметрыВыполненияКоманды (CommandExecuteParameters)

Уникальность (Uniqueness)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Ключ уникальности, с которым должна быть открыта форма.  
[Ложь](lang__def_BooleanFalse.md) - искать форму с ключом уникальности по умолчанию, [Истина](lang__def_BooleanTrue.md) - не искать форму вообще, любое другое значение устанавливает ключ уникальности, по которому будет произведен поиск.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
