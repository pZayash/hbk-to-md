Глобальный контекст (Global context)

ПредставлениеПрава (RightPresentation)

Доступен, начиная с версии 8.2.

Синтаксис:

ПредставлениеПрава(<ИмяПрава>)

Параметры:

<ИмяПрава> (необязательный)

Тип: Строка.   
Содержит имя права, для которого нужно получить представление.

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает представление права по имени права.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если право с таким именем не существует, генерируется исключительная ситуация.

Использование в версии:

Доступен, начиная с версии 8.2.
