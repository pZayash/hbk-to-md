Глобальный контекст (Global context)

ВыполнитьОбработкуОповещения (RunCallback)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ВыполнитьОбработкуОповещения(<ВыполняемоеОповещение>, <Результат>)

Параметры:

<ВыполняемоеОповещение> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, вызов которой будет произведен со следующими параметрами:

- <Результат> - значение результата, переданное вторым параметром при вызове метода,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта оповещения.

<Результат> (необязательный)

Значение, которое будет передано в процедуру, описанную параметром <ВыполняемоеОповещение>. Содержит результат, который мог быть получен при помощи алгоритма с использованием блокирующих методов.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Неопределено - если в параметре <ВыполняемоеОповещение> передан объект оповещения инициализированный именем процедуры.  
Произвольное значение (результат выполнения функции), если в описание оповещения передана функция.

Описание:

Запускает выполнение процедуры (или функции), описанной в параметре <ВыполняемоеОповещение>.   
Вызывающая процедура может передать процедуру обработки результата в виде объекта [ОписаниеОповещения](ОписаниеОповещения.md), которая будет вызвана после получения результата блокирующего метода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Используется в реализации процедур, которые внутри себя могут открывать блокирующие окна (например, вызывать метод [ПоказатьВопрос](Глобальный_контекст.ПоказатьВопрос.md)) и при этом должны каким-то образом вернуть свой результат в вызывающие процедуры.

См. также:

[ОписаниеОповещения](ОписаниеОповещения.md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.22.
