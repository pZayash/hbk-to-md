ПараметрыАнализаДанных (DataAnalysisParameters)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомого параметра.

Возвращаемое значение:

Тип: [ПараметрАнализаДанных](ПараметрАнализаДанных.md), Неопределено.   
Если параметр не найден, то возвращается значение Неопределено.

Описание:

Осуществляет поиск параметра по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
