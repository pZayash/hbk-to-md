ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

РежимЗаписи (WriteMode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [РежимЗаписиРегистра](РежимЗаписиРегистра.md).   
Определяет режим добавления записей в регистр сведений (например, [Независимый](РежимЗаписиРегистра.Независимый.md), [ПодчинениеРегистратору](РежимЗаписиРегистра.ПодчинениеРегистратору.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
