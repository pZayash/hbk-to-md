ПланыОбменаМенеджер (ExchangePlansManager)

ПрочитатьОписаниеИзменений (ReadChangesDescription)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПрочитатьОписаниеИзменений(<ЧтениеСообщенияОбмена>)

Параметры:

<ЧтениеСообщенияОбмена> (обязательный)

Тип: [ЧтениеСообщенияОбмена](ЧтениеСообщенияОбмена_(ExchangeMessageReader).md).   
Объект, через который осуществляется чтение сообщения обмена данными. У этого объекта должен быть выполнен метод [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md), но еще не выполнен метод [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение_(ExchangeMessageReader.EndRead).md).  
После вызова метода объект не может быть использован в вызове [ПрочитатьИзменения](ПланыОбменаМенеджер.ПрочитатьИзменения_(ExchangePlansManager.ReadChanges).md).

Возвращаемое значение:

Тип: [ОписаниеИзмененийКонфигурацииВСообщенииОбмена](ОписаниеИзмененийКонфигурацииВСообщенииОбмена_(ConfigurationChangesDescriptionInExchangeMessage).md).   

Описание:

Определяет, какие изменения конфигурации и расширений конфигурации произойдут при чтении сообщения обмена данными.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ПланыОбменаМенеджер](ПланыОбменаМенеджер_(ExchangePlansManager).md), метод [ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации](ПланыОбменаМенеджер.ПрочитатьИзмененияКонфигурацииИРасширенийКонфигурации_(ExchangePlansManager.ReadConfigurationAndConfigurationExtensionsChanges).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
