ПочтовыеВложения (MailAttachments)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Файл

Синтаксис:

Добавить(<АдресФайлаВложения>)

Параметры:

<АдресФайлаВложения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Адрес файла-вложения.

Описание варианта метода:

Добавляет [ПочтовоеВложение](ПочтовоеВложение_(MailAttachment).md) в коллекцию почтовых вложений как файл.

Вариант синтаксиса: Двоичные данные

Синтаксис:

Добавить(<ДвоичныеДанные>, <НаименованиеВложения>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные вложения.

<НаименованиеВложения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Наименование вложения.  
Значение по умолчанию: Пустая строка.

Описание варианта метода:

Добавляет [ПочтовоеВложение](ПочтовоеВложение_(MailAttachment).md) в коллекцию почтовых вложений как данные с наименованием.

Возвращаемое значение:

Тип: [ПочтовоеВложение](ПочтовоеВложение_(MailAttachment).md).   

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
