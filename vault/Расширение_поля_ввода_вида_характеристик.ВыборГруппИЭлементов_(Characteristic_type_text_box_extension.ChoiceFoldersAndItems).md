Расширение поля ввода вида характеристик (Characteristic type text box extension)

ВыборГруппИЭлементов (ChoiceFoldersAndItems)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов_(FoldersAndItemsUse).md).   
Предоставляет доступ к управлению режимом выбора групп и элементов плана видов характеристик: любой элемент, только группы или только элементы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
