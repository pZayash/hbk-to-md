# Расширение группы формы для обычной группы.Скрыта

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Группа формы](Группа_формы.md) › [Расширение группы формы для обычной группы](Расширение_группы_формы_для_обычной_группы_(Form_group_extension_for_a_usual_group).md) › [Методы](Расширение_группы_формы_для_обычной_группы__Методы.md)

Расширение группы формы для обычной группы (Form group extension for a usual group)

Скрыта (Hidden)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Скрыта()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В зависимости от значения свойства [Поведение](Расширение_группы_формы_для_обычной_группы.Поведение_(Form_group_extension_for_a_usual_group.Behavior).md) группы, метод возвращает:

- [Обычное](ПоведениеОбычнойГруппы.Обычное_(UsualGroupBehavior.Usual).md) - [Ложь](lang__def_BooleanFalse.md),
- [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая_(UsualGroupBehavior.Collapsible).md) - [Истина](lang__def_BooleanTrue.md), если группа свернута, и [Ложь](lang__def_BooleanFalse.md) в обратном случае,
- [Всплывающая](ПоведениеОбычнойГруппы.Всплывающая_(UsualGroupBehavior.PopUp).md) - [Истина](lang__def_BooleanTrue.md), если группа в данный момент отображается, и [Ложь](lang__def_BooleanFalse.md) в обратном случае.

Описание:

Проверяет состояние группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
