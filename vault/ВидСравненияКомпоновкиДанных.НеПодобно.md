ВидСравненияКомпоновкиДанных (DataCompositionComparisonType)

НеПодобно (NotLike)

Доступен, начиная с версии 8.3.5.

Описание:

Сравниваемое значение не удовлетворяет маске, заданной в условии сравнения.

- % (процент): последовательность, содержащая любое количество произвольных символов.
- \_ (подчеркивание): один произвольный символ.
- / - следующий символ нужно интерпретировать как обычный символ.

Использование в версии:

Доступен, начиная с версии 8.3.5.
