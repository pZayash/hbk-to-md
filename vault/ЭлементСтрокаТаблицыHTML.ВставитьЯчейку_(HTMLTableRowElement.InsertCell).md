# ЭлементСтрокаТаблицыHTML.ВставитьЯчейку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md) › [ЭлементСтрокаТаблицыHTML](ЭлементСтрокаТаблицыHTML_(HTMLTableRowElement).md) › [Методы](ЭлементСтрокаТаблицыHTML__Методы.md)

ЭлементСтрокаТаблицыHTML (HTMLTableRowElement)

ВставитьЯчейку (InsertCell)

Доступен, начиная с версии 8.2.

Синтаксис:

ВставитьЯчейку(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс ячейки, после которой будет добавлена новая ячейка.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML_(HTMLElement).md).   

Описание:

Предназначен для добавления ячейки в строку таблицы HTML-документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
