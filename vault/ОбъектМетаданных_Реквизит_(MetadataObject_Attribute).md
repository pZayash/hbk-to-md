Описание:

Используется для обращения к метаданным реквизита объекта конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ОбъектМетаданных: Задача](ОбъектМетаданных_Задача_(MetadataObject_Task).md), свойство [ОсновнойРеквизитАдресации](ОбъектМетаданных_Задача.ОсновнойРеквизитАдресации_(MetadataObject_Task.MainAddressingAttribute).md)

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.26.
