ФиксированныеНастройки (FixedSettings)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md).   
Содержит настройки, которые дополнительно применяются к основным настройкам, при использовании метода [ПолучитьНастройки](КомпоновщикНастроекКомпоновкиДанных.ПолучитьНастройки.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
