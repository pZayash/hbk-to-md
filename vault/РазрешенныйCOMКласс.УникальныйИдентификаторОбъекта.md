РазрешенныйCOMКласс (AllowedCOMClass)

УникальныйИдентификаторОбъекта (ObjectUUID)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит представление идентификатора COM-класса в формате системного реестра ОС Windows без фигурных скобок, по которому он может быть создан операционной системой. Например, 7F58F27D-5AD8-43a1-AA1E-C982F41BED5C.   
Регистр букв не различается. Используется при выполнении оператора Новый COMОбъект и метода [ПолучитьCOMОбъект](Глобальный_контекст.ПолучитьCOMОбъект.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
