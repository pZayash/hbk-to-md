СерииКодов (CodeSeries)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [СерииКодовПланаВидовХарактеристик](СерииКодовПланаВидовХарактеристик_(CharacteristicKindCodesSeries).md), [СерииКодовПланаСчетов](СерииКодовПланаСчетов_(CharOfAccountCodeSeries).md), [СерииКодовСправочника](СерииКодовСправочника_(CatalogCodesSeries).md).   
Определяет варианты диапазона проверки кода на уникальность и автоматического присвоения кодов ( например, [ВоВсемСправочнике](СерииКодовСправочника.ВоВсемСправочнике_(CatalogCodesSeries.WholeCatalog).md), [ВПределахПодчинения](СерииКодовСправочника.ВПределахПодчинения_(CatalogCodesSeries.WithinSubordination).md), [ВПределахПодчиненияВладельцу](СерииКодовСправочника.ВПределахПодчиненияВладельцу_(CatalogCodesSeries.WithinOwnerSubordination).md)).

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
