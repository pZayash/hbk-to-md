КатегорияСодержимогоHTML (HTMLContentCategory)

W3IncludeАтрибуты (W3IncludeAttributes)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все атрибуты "w3-include-html".   
Например, <div w3-include-html="content.html"> Some text</div>будет преобразован в <div> Some text </div>.

Использование в версии:

Доступен, начиная с версии 8.3.13.
