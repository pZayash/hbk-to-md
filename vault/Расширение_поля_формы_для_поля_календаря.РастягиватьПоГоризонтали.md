Расширение поля формы для поля календаря (Form field extension for a calendar field)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает возможность растягивания календаря по горизонтали.   
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
