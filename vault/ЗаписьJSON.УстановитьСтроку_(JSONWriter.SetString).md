# ЗаписьJSON.УстановитьСтроку

ЗаписьJSON (JSONWriter)

УстановитьСтроку (SetString)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьСтроку(<Параметры>)

Параметры:

<Параметры> (необязательный)

Тип: [ПараметрыЗаписиJSON](ПараметрыЗаписиJSON_(JSONWriterSettings).md).   
Параметры, используемые при записи объекта JSON.  
По умолчанию, содержит [ПараметрыЗаписиJSON](ПараметрыЗаписиJSON_(JSONWriterSettings).md), сгенерированные автоматически.

Описание:

Инициализирует объект для вывода результирующего JSON текста в строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; ЗаписьJSON.УстановитьСтроку(); // Запись документа JSON |

Использование в версии:

Доступен, начиная с версии 8.3.6.
