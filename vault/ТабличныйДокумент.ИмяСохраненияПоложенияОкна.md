ТабличныйДокумент (SpreadsheetDocument)

ИмяСохраненияПоложенияОкна (WindowOptionsName)

Не рекомендуется использовать, начиная с версии 8.2.

Рекомендуется использовать:

- [КлючСохраненияПоложенияОкна](ТабличныйДокумент.КлючСохраненияПоложенияОкна.md)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя значения, в котором сохраняются параметры окна, в котором отображается таблица. Параметры окна, настроенные и сохраненные с данным именем, впоследствии восстанавливаются при следующем показе таблицы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
