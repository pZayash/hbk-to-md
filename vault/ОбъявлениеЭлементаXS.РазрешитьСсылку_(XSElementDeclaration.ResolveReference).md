# ОбъявлениеЭлементаXS.РазрешитьСсылку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Схема XML](Схема_XML.md) › [ОбъявлениеЭлементаXS](ОбъявлениеЭлементаXS_(XSElementDeclaration).md) › [Методы](ОбъявлениеЭлементаXS__Методы.md)

ОбъявлениеЭлементаXS (XSElementDeclaration)

РазрешитьСсылку (ResolveReference)

Доступен, начиная с версии 8.1.

Синтаксис:

РазрешитьСсылку()

Возвращаемое значение:

Тип: [ОбъявлениеЭлементаXS](ОбъявлениеЭлементаXS_(XSElementDeclaration).md), [Неопределено](lang__def_Undefined.md).   
Компонент (объявление атрибута или элемента), на который ссылается данное объявление, или [Неопределено](lang__def_Undefined.md), если такого компонента нет.

Описание:

Разрешить ссылку на объявление атрибута или элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
