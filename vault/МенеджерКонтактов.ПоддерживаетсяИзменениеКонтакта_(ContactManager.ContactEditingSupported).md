МенеджерКонтактов (ContactManager)

ПоддерживаетсяИзменениеКонтакта (ContactEditingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяИзменениеКонтакта(<Контакт>)

Параметры:

<Контакт> (обязательный)

Тип: [ЛокальныйКлючКонтакта](ЛокальныйКлючКонтакта_(LocalContactKey).md).   
Проверяемый контакт.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Для устройств под управлением Windows Phone всегда возвращает [Ложь](lang__def_BooleanFalse.md).  
Для устройств под управлением Android и iOS всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Определяет возможность редактирования или удаления контакта.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКонтактов](МенеджерКонтактов_(ContactManager).md), метод [ИзменитьКонтакт](МенеджерКонтактов.ИзменитьКонтакт_(ContactManager.UpdateContact).md)  
[МенеджерКонтактов](МенеджерКонтактов_(ContactManager).md), метод [УдалитьКонтакт](МенеджерКонтактов.УдалитьКонтакт_(ContactManager.DeleteContact).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
