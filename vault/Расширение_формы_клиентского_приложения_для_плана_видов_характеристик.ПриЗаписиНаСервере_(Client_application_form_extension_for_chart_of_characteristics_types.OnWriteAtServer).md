Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ПриЗаписиНаСервере (OnWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЗаписиНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись произведена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ТекущийОбъект>

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи.

Описание:

Возникает на сервере при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события ПриЗаписи объекта) в базу данных, но до окончания транзакции записи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
