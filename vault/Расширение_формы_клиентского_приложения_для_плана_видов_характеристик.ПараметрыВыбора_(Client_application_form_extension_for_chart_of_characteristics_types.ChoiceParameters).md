Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ПараметрыВыбора (ChoiceParameters)

Доступен, начиная с версии 8.3.6.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Содержит значения, по которым проверяется возможность выбора для форм, находящихся в режиме выбора.

Использование в версии:

Доступен, начиная с версии 8.3.6.
