ВариантСтандартногоПериода (StandardPeriodVariant)

Месяц (Month)

Доступен, начиная с версии 8.2.

Описание:

При выборе данного варианта начальная дата периода становится равной текущей дате плюс один день и минус один месяц, начало дня. Конечной датой периода становится конец текущего дня.

Использование в версии:

Доступен, начиная с версии 8.2.
