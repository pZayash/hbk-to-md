ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonAction)

ОтправитьСообщениеСДанными (SendMessageWithData)

Доступен, начиная с версии 8.3.25.

Описание:

Действие кнопки создает сообщение с текстом кнопки.  
В свойство [Данные](СообщениеСистемыВзаимодействия.Данные.md) создаваемого сообщения копируется значение свойства [Данные](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Данные.md) нажатой кнопки.   
Действие не доступно в интеграциях с мессенджерами.

Использование в версии:

Доступен, начиная с версии 8.3.25.
