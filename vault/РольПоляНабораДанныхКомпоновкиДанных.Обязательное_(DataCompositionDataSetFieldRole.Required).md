РольПоляНабораДанныхКомпоновкиДанных (DataCompositionDataSetFieldRole)

Обязательное (Required)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости обязательного добавления данного поля в набор данных макета компоновки данных в случае, если из набора данных поля в настройках задействовано хотя бы одно поле.  
[Истина](lang__def_BooleanTrue.md) - обязателен.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
