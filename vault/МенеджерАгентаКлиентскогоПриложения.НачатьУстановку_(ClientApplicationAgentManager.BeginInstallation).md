МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

НачатьУстановку (BeginInstallation)

Доступен, начиная с версии 8.3.13.

Синтаксис:

НачатьУстановку()

Описание:

Начинает установку агента клиентского приложения.  
Результат выполнения действия (успешное выполнение или ошибка выполнения) можно получить, подключив обработчик с помощью метода [ПодключитьОбработчикИзмененияСостояния](МенеджерАгентаКлиентскогоПриложения.ПодключитьОбработчикИзмененияСостояния_(ClientApplicationAgentManager.AttachStateChangeHandler).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
