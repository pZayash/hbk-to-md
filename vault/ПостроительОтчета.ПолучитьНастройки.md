ПостроительОтчета (ReportBuilder)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьНастройки(<Отбор>, <Порядок>, <Измерения>, <ВыбранныеПоля>, <УсловноеОформление>)

Параметры:

<Отбор> (необязательный)

Тип: Булево.   
Признак необходимости получения отбора.  
Значение по умолчанию: Истина.

<Порядок> (необязательный)

Тип: Булево.   
Признак необходимости получения порядка.  
Значение по умолчанию: Истина.

<Измерения> (необязательный)

Тип: Булево.   
Признак необходимости получения настроек измерений.  
Значение по умолчанию: Истина.

<ВыбранныеПоля> (необязательный)

Тип: Булево.   
Признак необходимости получения выбранных полей.  
Значение по умолчанию: Истина.

<УсловноеОформление> (необязательный)

Тип: Булево.   
Признак необходимости получения условного оформления.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [НастройкиПостроителяОтчета](НастройкиПостроителяОтчета.md).   

Описание:

Получает текущие настройки построителя отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| СтруктураНастройки = Новый Структура; Настройки = ПостроительОтчета.ПолучитьНастройки(Истина,Ложь); Параметры = ПостроительОтчета.Параметры; СтруктураНастройки.Вставить("НастройкиПостроителя",Настройки); СтруктураНастройки.Вставить("ПараметрыПостроителя",Параметры); ХранилищеДанных = Новый ХранилищеЗначения(СтруктураНастройки); |

Использование в версии:

Доступен, начиная с версии 8.0.
