# КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Настройки компоновки данных](Настройки_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступныйОбъектНастройкиКомпоновкиДанных](ДоступныйОбъектНастройкиКомпоновкиДанных_(DataCompositionAvailableSettingsObject).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные объекты настройки.  
Возможно обращение к доступному объекту настройки посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Индекс_(DataCompositionAvailableSettingsObjectCollection.IndexOf).md)  
[Количество (Count)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Количество_(DataCompositionAvailableSettingsObjectCollection.Count).md)  
[Найти (Find)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Найти_(DataCompositionAvailableSettingsObjectCollection.Find).md)  
[Получить (Get)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Получить_(DataCompositionAvailableSettingsObjectCollection.Get).md)  

Описание:

Коллекция доступных объектов настройки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
