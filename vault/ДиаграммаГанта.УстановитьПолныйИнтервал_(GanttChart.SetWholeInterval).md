ДиаграммаГанта (GanttChart)

УстановитьПолныйИнтервал (SetWholeInterval)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПолныйИнтервал(<ДатаНачала>, <ДатаОкончания>)

Параметры:

<ДатаНачала> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата начала интервала. Не может быть меньше даты "01.01.1000".

<ДатаОкончания> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата окончания интервала. Не может быть больше даты "01.01.3000".

Описание:

Устанавливает полный интервал.  
Следует использовать при [АвтоОпределениеПолногоИнтервала](ДиаграммаГанта.АвтоОпределениеПолногоИнтервала_(GanttChart.AutoDetectWholeInterval).md) = [Ложь](lang__def_BooleanFalse.md).   
ВНИМАНИЕ! Установку полного интервала можно производить только с помощью этого метода. Свойства [НачалоПолногоИнтервала](ДиаграммаГанта.НачалоПолногоИнтервала_(GanttChart.BeginOfWholeInterval).md) и [КонецПолногоИнтервала](ДиаграммаГанта.КонецПолногоИнтервала_(GanttChart.EndOfWholeInterval).md) только для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
