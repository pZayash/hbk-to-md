КонструкторСхемыКомпоновкиДанных (DataCompositionSchemaWizard)

УстановитьСхему (SetSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьСхему(<СхемаКомпоновкиДанных>)

Параметры:

<СхемаКомпоновкиДанных> (обязательный)

Тип: [СхемаКомпоновкиДанных](СхемаКомпоновкиДанных.md).   
Схема компоновки данных, предназначенная для редактирования.

Описание:

Устанавливает схему компоновки данных для редактирования.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.1.
