ФормаКлиентскогоПриложения (ClientApplicationForm)

ВладелецФормы (FormOwner)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ТабличноеПоле](ТабличноеПоле_(TableBox).md), [ПолеВвода](ПолеВвода_(TextBox).md), [Форма](Форма_(Form).md), [ПолеФормы](ПолеФормы_(FormField).md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md).   
Содержит форму клиентского приложения или элемент управления, который является владельцем данной формы. При помощи этого свойства устанавливается связь между формой и владельцем формы. Например, форма элемента подчиненного справочника может иметь в качестве владельца форму элемента справочника владельца, из которой она была открыта. При открытии по текущей строке табличного поля в качестве владельца устанавливается табличное поле.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если у формы есть владелец, то в свойстве [ЗакрыватьПриЗакрытииВладельца](ФормаКлиентскогоПриложения.ЗакрыватьПриЗакрытииВладельца_(ClientApplicationForm.CloseOnOwnerClose).md) устанавливается необходимость закрытия данной формы при закрытии формы-владельца.

Использование в версии:

Доступен, начиная с версии 8.2.
