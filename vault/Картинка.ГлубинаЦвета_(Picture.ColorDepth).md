Картинка (Picture)

ГлубинаЦвета (ColorDepth)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ГлубинаЦвета()

Возвращаемое значение:

Тип: [ГлубинаЦвета](ГлубинаЦвета_(ColorDepth).md), [Неопределено](lang__def_Undefined.md).   
Если картинка создана с вариантами, то возвращает информацию по текущему варианту.  
Для картинок векторных форматов возвращает [Неопределено](lang__def_Undefined.md).  
На мобильной платформе и мобильных клиентах всегда возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает глубину цвета картинки - количество бит для хранения цвета каждой точки изображения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.19.
