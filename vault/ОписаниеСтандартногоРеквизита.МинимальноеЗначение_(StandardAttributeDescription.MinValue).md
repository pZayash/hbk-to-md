ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

МинимальноеЗначение (MinValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Устанавливает минимальное возможное для реквизита значение.  
Имеет смысл для реквизитов типа [Число](lang__def_Number.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
