# МенеджерТабличныхПространствБазыДанных.НайтиПоИспользуемомуСоставуИндексов

МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

НайтиПоИспользуемомуСоставуИндексов (FindByUsedIndexesContent)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиПоИспользуемомуСоставуИндексов(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [ЭлементСоставаТабличногоПространстваБазыДанных](ЭлементСоставаТабличногоПространстваБазыДанных_(DatabaseTablespaceContentItem).md), ОбъектМетаданных.   
Объект метаданных, для которого выполняется поиск табличного пространства.

Возвращаемое значение:

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных_(DatabaseTablespaceManager).md).   

Описание:

Возвращет табличное пространство, в котором физически хранятся индексы таблиц объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.22.
