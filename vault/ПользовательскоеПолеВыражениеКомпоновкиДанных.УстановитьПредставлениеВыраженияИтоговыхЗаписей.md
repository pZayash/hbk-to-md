ПользовательскоеПолеВыражениеКомпоновкиДанных (DataCompositionUserFieldExpression)

УстановитьПредставлениеВыраженияИтоговыхЗаписей (SetTotalRecordExpressionPresentation)

Не рекомендуется использовать, начиная с версии 8.3.10.

Рекомендуется использовать:

- [УстановитьПредставлениеВыражений](ПользовательскоеПолеВыражениеКомпоновкиДанных.УстановитьПредставлениеВыражений.md)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьПредставлениеВыраженияИтоговыхЗаписей(<ПользовательскоеПредставлениеВыражения>)

Параметры:

<ПользовательскоеПредставлениеВыражения> (необязательный)

Тип: Строка.   
Пользовательское представление выражения, используемое при выводе поля в итоговых записях.

Описание:

Устанавливает пользовательское представление выражения, используемое при выводе поля в итоговых записях.  
В случае, если настройки связаны с источником доступных настроек, выражение тоже будет обновлено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.10.

Описание изменено в версии 8.3.15.
