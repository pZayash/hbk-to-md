АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКВнешнимПриложениям (ExternalApplicationsFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет разрешение запуска приложений.  
[Истина](lang__def_BooleanTrue.md) – разрешен запуск любых приложений. [Ложь](lang__def_BooleanFalse.md) – разрешен запуск только приложений, перечисленных в списке разрешенных приложений профиля.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
