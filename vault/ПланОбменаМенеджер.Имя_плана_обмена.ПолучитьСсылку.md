ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена.md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор.md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ПланОбменаСсылка.Имя_плана_обмена.УникальныйИдентификатор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

См. также:

[ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена.md), метод [УникальныйИдентификатор](ПланОбменаСсылка.Имя_плана_обмена.УникальныйИдентификатор.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
