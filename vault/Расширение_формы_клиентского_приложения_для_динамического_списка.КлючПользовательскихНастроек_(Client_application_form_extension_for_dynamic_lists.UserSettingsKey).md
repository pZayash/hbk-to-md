Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

КлючПользовательскихНастроек (UserSettingsKey)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: Произвольный.   
Содержит ключ пользовательских настроек, которые нужно использовать для основного динамического списка формы. Если не установлен параметр <ПользовательскиеНастройки>, то будут загружены пользовательские настройки по указанному ключу.  
Если настройки, заданные ключом, не найдены, используются настройки динамического списка по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.3.3.
