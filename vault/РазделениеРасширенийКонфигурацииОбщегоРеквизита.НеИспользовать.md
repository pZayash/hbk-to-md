РазделениеРасширенийКонфигурацииОбщегоРеквизита (CommonAttributeConfigurationExtensionsSeparation)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.3.10.

Описание:

Данный разделитель не будет влиять на расширения конфигурации, т.е. вне зависимости от значения данного разделителя будет доступен один и тот же набор расширений.

Использование в версии:

Доступен, начиная с версии 8.3.10.
