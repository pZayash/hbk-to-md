ГеографическаяСхема (GeographicalSchema)

Проекция (Projection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипПроекцииГеографическойСхемы](ТипПроекцииГеографическойСхемы_(GeographicalSchemaProjection).md).   
Проекция, в которой отображается географическая схема. По умолчанию установлена цилиндрическая проекция Миллера ([ЦилиндрическаяПроекцияМиллера](ТипПроекцииГеографическойСхемы.ЦилиндрическаяПроекцияМиллера_(GeographicalSchemaProjection.CylindricalMillerProjection).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
