Расширение поля формы для поля картинки (Form field extension for a picture field)

ТекстНевыбраннойКартинки (NonselectedPictureText)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст, отображаемый, если картинка не выбрана.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Не используется если поле размещено в таблице (колонка).

Использование в версии:

Доступен, начиная с версии 8.2.
