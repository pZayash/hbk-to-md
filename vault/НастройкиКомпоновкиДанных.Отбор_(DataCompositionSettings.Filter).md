НастройкиКомпоновкиДанных (DataCompositionSettings)

Отбор (Filter)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md).   
Основной отбор. Прописывается как условия в запросы наборов данных и как отборы для наборов данных, целиком.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
