РезультатXPath (XPathResult)

ЧисловоеЗначение (NumberValue)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит числовое значение результата.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение не может быть преобразовано к числовому типу, то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
