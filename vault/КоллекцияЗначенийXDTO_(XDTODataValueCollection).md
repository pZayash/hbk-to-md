# КоллекцияЗначенийXDTO

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения XDTO.  
Возможно обращение к значению XDTO посредством оператора [...]. В качестве аргумента передается индексу.  

Методы:

[Количество (Count)](КоллекцияЗначенийXDTO.Количество_(XDTODataValueCollection.Count).md)  
[Получить (Get)](КоллекцияЗначенийXDTO.Получить_(XDTODataValueCollection.Get).md)  

Описание:

Коллекция значений простого типа XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), свойство [Список](ЗначениеXDTO.Список_(XDTODataValue.List).md)
