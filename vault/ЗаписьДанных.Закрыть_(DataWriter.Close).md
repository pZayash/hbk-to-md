# ЗаписьДанных.Закрыть

ЗаписьДанных (DataWriter)

Закрыть (Close)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Закрыть()

Описание:

Вызывает метод [СброситьБуферы](ЗаписьДанных.СброситьБуферы_(DataWriter.Flush).md). Если целевой поток был создан при создании объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md), целевой поток также закрывается.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [НачатьЗакрытие](ЗаписьДанных.НачатьЗакрытие_(DataWriter.BeginClose).md)  
[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [ЗакрытьАсинх](ЗаписьДанных.ЗакрытьАсинх_(DataWriter.CloseAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
