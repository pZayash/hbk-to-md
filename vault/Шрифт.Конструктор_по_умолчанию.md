Шрифт (Font)

Конструктор по умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Шрифт()

Описание:

Создает значение шрифта, которое будет использоваться как значение "по умолчанию".

Пример:

|  |
| --- |
| Шрифт = Новый Шрифт; |

Использование в версии:

Доступен, начиная с версии 8.0.
