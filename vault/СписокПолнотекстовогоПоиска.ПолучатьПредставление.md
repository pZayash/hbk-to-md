СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолучатьПредставление (GetPresentation)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Признак получения представления для результатов поиска. Если установлено Истина, то для каждого из результатов поиска будет заполнено значение [Представление](ЭлементСпискаПолнотекстовогоПоиска.Представление.md).  
Если свойство установлено в значение Ложь, поиск выполняется быстрее.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
