# Расширение табличного поля списка перечисления.НастройкаОтбора

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Перечисления](Перечисления.md) › [Расширение табличного поля списка перечисления](Расширение_табличного_поля_списка_перечисления_(Enum_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_перечисления__Свойства.md)

Расширение табличного поля списка перечисления (Enum list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора_(FilterSettings).md).   
Предоставляет доступ к настройкам отбора списка перечисления. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для перечисления элементов отбора (в данном случае - единственное свойство "Ссылка").

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
