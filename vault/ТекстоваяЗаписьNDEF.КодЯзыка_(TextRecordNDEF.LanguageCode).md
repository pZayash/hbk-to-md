ТекстоваяЗаписьNDEF (TextRecordNDEF)

КодЯзыка (LanguageCode)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Код языка по стандарту ISO/реестру IANA.  
Размер не более 63 байт.  
Если [Неопределено](lang__def_Undefined.md), используется "en-US".

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
