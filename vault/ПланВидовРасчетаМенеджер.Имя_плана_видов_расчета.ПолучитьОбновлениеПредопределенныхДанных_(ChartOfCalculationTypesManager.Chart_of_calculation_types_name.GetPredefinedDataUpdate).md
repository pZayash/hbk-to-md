ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

ПолучитьОбновлениеПредопределенныхДанных (GetPredefinedDataUpdate)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбновлениеПредопределенныхДанных()

Возвращаемое значение:

Тип: [ОбновлениеПредопределенныхДанных](ОбновлениеПредопределенныхДанных_(PredefinedDataUpdate).md).   

Описание:

Получает текущее значение режима обновления предопределенных элементов. Вычисляется из значений, установленных в метаданных и в данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
