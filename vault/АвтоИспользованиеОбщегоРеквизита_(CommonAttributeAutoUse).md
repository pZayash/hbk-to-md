Значения

Описание:

Содержит варианты включения в состав общего реквизита объекта метаданных из соответствующего элемента состава общего реквизита, если у элемента включение в состав установлено автоматическим.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
