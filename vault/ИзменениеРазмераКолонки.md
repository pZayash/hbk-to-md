Доступен, начиная с версии 8.0.

Значения

[Изменять (Change)](ИзменениеРазмераКолонки.Изменять.md)  
[НеИзменять (DontChange)](ИзменениеРазмераКолонки.НеИзменять.md)  

Описание:

Определяет набор режимов изменения размера колонки. Используется для определения свойства [ИзменениеРазмера](КолонкаТабличногоПоля.ИзменениеРазмера.md) колонки табличного поля.

Доступность:

Толстый клиент.

См. также:

[КолонкаТабличногоПоля](КолонкаТабличногоПоля.md), свойство [ИзменениеРазмера](КолонкаТабличногоПоля.ИзменениеРазмера.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
