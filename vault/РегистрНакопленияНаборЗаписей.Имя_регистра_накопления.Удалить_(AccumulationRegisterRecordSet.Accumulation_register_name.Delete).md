РегистрНакопленияНаборЗаписей.<Имя регистра накопления> (AccumulationRegisterRecordSet.<Accumulation register name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [Число](lang__def_Number.md), [РегистрНакопленияЗапись.](РегистрНакопленияЗапись.Имя_регистра_накопления_(AccumulationRegisterRecord.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияЗапись.Имя_регистра_накопления_(AccumulationRegisterRecord.Accumulation_register_name).md)>.   
Индекс удаляемой записи либо сама запись.

Описание:

Удаляет запись из набора по указанному индексу записи или объекту записи регистра накопления.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
