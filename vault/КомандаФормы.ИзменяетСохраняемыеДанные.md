КомандаФормы (FormCommand)

ИзменяетСохраняемыеДанные (ModifiesStoredData)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак необходимости сохранения данных при вызове команды (вызов команды приводит к установке флага модифицированности).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Изменение доступно только на сервере.

Использование в версии:

Доступен, начиная с версии 8.2.
