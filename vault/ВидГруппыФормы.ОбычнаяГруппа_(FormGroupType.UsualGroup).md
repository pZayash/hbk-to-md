ВидГруппыФормы (FormGroupType)

ОбычнаяГруппа (UsualGroup)

Доступен, начиная с версии 8.2.

Описание:

Обычная группа. Группа данного вида может содержать поля, таблицы, кнопки и группы видов [ОбычнаяГруппа](ВидГруппыФормы.ОбычнаяГруппа_(FormGroupType.UsualGroup).md), [КоманднаяПанель](ВидГруппыФормы.КоманднаяПанель_(FormGroupType.CommandBar).md), [Страницы](ВидГруппыФормы.Страницы_(FormGroupType.Pages).md).

Использование в версии:

Доступен, начиная с версии 8.2.
