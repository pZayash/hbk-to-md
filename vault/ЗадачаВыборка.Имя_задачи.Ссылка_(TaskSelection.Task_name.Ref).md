ЗадачаВыборка.<Имя задачи> (TaskSelection.<Task name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Содержит ссылку на задачу. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство используется, если ссылку на созданную и записанную задачу нужно присвоить реквизиту другой задачи.

Использование в версии:

Доступен, начиная с версии 8.0.
