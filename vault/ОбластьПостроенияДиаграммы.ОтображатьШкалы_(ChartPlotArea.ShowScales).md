ОбластьПостроенияДиаграммы (ChartPlotArea)

ОтображатьШкалы (ShowScales)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения шкал (координатных линий и делений).  
[Истина](lang__def_BooleanTrue.md) - отображаются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для всех диаграмм, кроме круговых.

Использование в версии:

Доступен, начиная с версии 8.0.
