РегистрРасчетаСписок.<Имя регистра расчета> (CalculationRegisterList.<Calculation register name>)

Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Порядок_(Order).md).   
Содержит объект [Порядок](Порядок_(Order).md), по которому осуществляется текущее упорядочивание записей регистра в списке. В качестве свойств объекта могут выступать предопределенные поля, а также измерения, ресурсы и реквизиты регистра. Имена свойств совпадают с именами полей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
