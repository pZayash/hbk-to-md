Соединение с агентом сервера (IServerAgentConnection)

SetClusterSecurityLevel (SetClusterSecurityLevel)

Доступен, начиная с версии 8.1.

Синтаксис:

SetClusterSecurityLevel(<Кластер>, <Уровень>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, уровень безопасности которого должен быть изменен.

<Уровень> (обязательный)

Тип: [Число](lang__def_Number.md).   
Уровень безопасности всех соединений процесса менеджера кластера (rmngr). 0 - незащищенное соединение, 1 - безопасное соединение только на время выполнения аутентификации пользователя, 2 - безопасное соединение в течении всего сеанса.

Описание:

Изменить уровень безопасности кластера. Он определяет защищенность соединений процесса менеджера кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
