Описание:

Предназначен для выполнения анализа данных (кластерный анализ, поиск ассоциативных правил, поиск последовательностей и другие виды анализа). Результат работы зависит от типа производимого анализа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ПостроительОтчетаАнализаДанных](ПостроительОтчетаАнализаДанных_(DataAnalysisReportBuilder).md), метод [ПолучитьАнализ](ПостроительОтчетаАнализаДанных.ПолучитьАнализ_(DataAnalysisReportBuilder.GetAnalysis).md)

Доступен, начиная с версии 8.0.
