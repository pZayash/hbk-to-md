# ДинамическийСписок.АвтоЗаполнениеДоступныхПолей

ДинамическийСписок (DynamicList)

АвтоЗаполнениеДоступныхПолей (AutoFillAvailableFields)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md) – доступные поля заполняются по правилам автоматического заполнения доступных полей набора данных схемы компоновки данных.  
Если [Ложь](lang__def_BooleanFalse.md) – в качестве доступных полей компоновки данных используются поля, явно добавленные с помощью расширения языка запросов для системы компоновки данных.   
Если свойство [ПроизвольныйЗапрос](ДинамическийСписок.ПроизвольныйЗапрос_(DynamicList.CustomQuery).md) имеет значение [Ложь](lang__def_BooleanFalse.md), значение данного свойства не учитывается.  
В режиме совместимости конфигурации [Версия8\_3\_7](РежимСовместимости.Версия8_3_7_(CompatibilityMode.Version8_3_7).md) и ниже значение данного свойства не влияет на поведение системы - доступные поля всегда заполняются автоматически.

Доступность:

Сервер.

Примечание:

Значение по умолчанию - [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
