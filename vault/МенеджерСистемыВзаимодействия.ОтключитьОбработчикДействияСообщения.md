МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтключитьОбработчикДействияСообщения (DetachMessageActionHandler)

Не рекомендуется использовать, начиная с версии 8.3.18.

Рекомендуется использовать:

- [ПриВыбореДействияСообщенияСистемыВзаимодействия](Глобальный_контекст.ПриВыбореДействияСообщенияСистемыВзаимодействия.md)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ОтключитьОбработчикДействияСообщения(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, заданной при вызове метода [ПодключитьОбработчикДействияСообщения](МенеджерСистемыВзаимодействия.ПодключитьОбработчикДействияСообщения.md).

Описание:

Отключает ранее подключенный обработчик.  
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ПодключитьОбработчикДействияСообщения](МенеджерСистемыВзаимодействия.ПодключитьОбработчикДействияСообщения.md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.

Не рекомендуется использовать, начиная с версии 8.3.18.

Описание изменено в версии 8.3.18.
