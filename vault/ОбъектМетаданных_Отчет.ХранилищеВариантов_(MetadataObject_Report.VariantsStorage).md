ОбъектМетаданных: Отчет (MetadataObject: Report)

ХранилищеВариантов (VariantsStorage)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: ХранилищеНастроек](ОбъектМетаданных_ХранилищеНастроек_(MetadataObject_SettingsStorage).md).   
Содержит хранилище, которое используется для хранения вариантов отчетов, создаваемых в режиме Предприятие.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
