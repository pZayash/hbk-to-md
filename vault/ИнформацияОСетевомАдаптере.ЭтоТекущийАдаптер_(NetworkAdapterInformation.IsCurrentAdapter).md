ИнформацияОСетевомАдаптере (NetworkAdapterInformation)

ЭтоТекущийАдаптер (IsCurrentAdapter)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет, является ли данный сетевой адаптер адаптером, через который установлено соединение в данный момент:

- [Истина](lang__def_BooleanTrue.md) - соединение клиента установлено через данный адаптер,
- [Ложь](lang__def_BooleanFalse.md) - в противном случае.
- Для веб-клиента всегда содержит [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
