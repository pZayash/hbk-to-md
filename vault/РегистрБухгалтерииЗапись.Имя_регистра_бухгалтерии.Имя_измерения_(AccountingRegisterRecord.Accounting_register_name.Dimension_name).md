Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения измерений регистра бухгалтерии. Доступ к значению осуществляется по имени, как оно задано в конфигурации.  
Набор свойств используется только для балансовых измерений.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В качестве типа допускается использовать любые типы, кроме [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) и строки неограниченной длины.

Использование в версии:

Доступен, начиная с версии 8.0.
