ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

НеОбновлятьАвтоматически (DontAutoUpdate)

Доступен, начиная с версии 8.3.3.

Описание:

При реструктуризации информационной базы не будет выполняться автоматическое создание новых предопределенных элементов и обновление их значений.

Использование в версии:

Доступен, начиная с версии 8.3.3.
