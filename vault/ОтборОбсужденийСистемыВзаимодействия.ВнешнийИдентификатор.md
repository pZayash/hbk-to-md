ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

ВнешнийИдентификатор (ExternalID)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Если содержит непустую строку, то возвращается обсуждение с заданным внешним идентификатором.  
Если обсуждение не содержит участников, то для поиска такого обсуждения требуется привилегированный режим.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
