ЧтениеДанных (DataReader)

ПрочитатьСтрокуАсинх (ReadLineAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьСтрокуАсинх(<КодировкаТекста>, <РазделительСтрок>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста. Если не установлена, используется кодировка, заданная для текущего экземпляра [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Указывает строку, являющуюся разделителем строк в читаемых данных. Если параметр не указан, используется разделитель строк, указанный для текущего экземпляра объекта [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - прочитанная Строка.

Описание:

Асинхронная функция. Выполняет чтение строки текста до разделителя строки. Если разделитель не найден, чтение выполняется до конца потока.  
Если разделитель строки не задан явно, используется разделитель строки, указанный для данного экземпляра объекта [ЧтениеДанных](ЧтениеДанных.md).

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

По умолчанию, в качестве разделителя строки выступают следующие строки:

- [ВК](Символы.ВК.md),
- [ПС](Символы.ПС.md),
- [ВК](Символы.ВК.md),
- [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).
