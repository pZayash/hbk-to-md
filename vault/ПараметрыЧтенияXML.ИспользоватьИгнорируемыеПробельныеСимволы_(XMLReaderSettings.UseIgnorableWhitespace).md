# ПараметрыЧтенияXML.ИспользоватьИгнорируемыеПробельныеСимволы

ПараметрыЧтенияXML (XMLReaderSettings)

ИспользоватьИгнорируемыеПробельныеСимволы (UseIgnorableWhitespace)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования узлов [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы_(XMLNodeType.Whitespace).md) при чтении XML.  
[Истина](lang__def_BooleanTrue.md) - тип узла [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы_(XMLNodeType.Whitespace).md) используется. Значение по умолчанию [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
