КнопкаКоманднойПанели (CommandBarButton)

Кнопки (Buttons)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КнопкиКоманднойПанели](КнопкиКоманднойПанели.md).   
Содержит коллекцию кнопок, связанную с данной. Имеет смысл только для кнопок типа [Подменю](ТипКнопкиКоманднойПанели.Подменю.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
