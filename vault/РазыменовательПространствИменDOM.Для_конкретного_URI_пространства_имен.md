РазыменовательПространствИменDOM (DOMNamespaceResolver)

Для конкретного URI пространства имен

Доступен, начиная с версии 8.1.

Синтаксис:

Новый РазыменовательПространствИменDOM(<Префикс>, <URIПространстваИмен>)

Параметры:

<Префикс> (обязательный)

Тип: Строка.   
Префикс пространства имен.

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен, соответствующее префиксу.

Описание:

Объект, разыменовывающий префикс пространства имен, создается на основе конкретного соответствия префикса - URI пространства имен.

Использование в версии:

Доступен, начиная с версии 8.1.
