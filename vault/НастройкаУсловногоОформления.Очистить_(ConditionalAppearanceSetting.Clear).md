НастройкаУсловногоОформления (ConditionalAppearanceSetting)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все элементы управления настройками условного оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
