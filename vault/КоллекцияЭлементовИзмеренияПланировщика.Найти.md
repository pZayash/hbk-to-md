КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>, <ВключатьПодчиненные>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению элемента измерения.

<ВключатьПодчиненные> (необязательный)

Тип: Булево.   
Определяет, будут ли участвовать в поиске элемента измерения подчиненные коллекции (если таковые имеются).   
Истина – элементы измерения подчиненных коллекций участвуют в поиске.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика.md), Неопределено.   
Если указанный элемент отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента измерения планировщика по указанному значению свойства [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
