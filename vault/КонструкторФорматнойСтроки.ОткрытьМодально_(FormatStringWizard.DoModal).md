# КонструкторФорматнойСтроки.ОткрытьМодально

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [КонструкторФорматнойСтроки](КонструкторФорматнойСтроки_(FormatStringWizard).md) › [Методы](КонструкторФорматнойСтроки__Методы.md)

КонструкторФорматнойСтроки (FormatStringWizard)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - конструктор закрыли по кнопке "OK"; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает модально конструктор форматной строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [Показать](КонструкторФорматнойСтроки.Показать_(FormatStringWizard.Show).md).

См. также:

[КонструкторФорматнойСтроки](КонструкторФорматнойСтроки_(FormatStringWizard).md), метод [ОткрытьАсинх](КонструкторФорматнойСтроки.ОткрытьАсинх_(FormatStringWizard.DoAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
