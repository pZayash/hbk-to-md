# Расширение элементов управления, расположенных на панели.УстановитьПривязку

Расширение элементов управления, расположенных на панели (Panel controls extension)

УстановитьПривязку (SetLink)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПривязку(<Граница>, <ПервыйЭлемент>, <ГраницаПервогоЭлемента>, <ВторойЭлемент>, <ГраницаВторогоЭлемента>)

Параметры:

<Граница> (обязательный)

Тип: [ГраницаЭлементаУправления](ГраницаЭлементаУправления_(ControlEdge).md).   
Граница элемента управления, для которой устанавливается привязка.

<ПервыйЭлемент> (необязательный)

Элемент управления, к которому будет жестко привязана указанная граница.

<ГраницаПервогоЭлемента> (необязательный)

Тип: [ГраницаЭлементаУправления](ГраницаЭлементаУправления_(ControlEdge).md).   
Граница первого элемента управления, к которой будет жестко привязана указанная граница.

<ВторойЭлемент> (необязательный)

Элемент управления, для пропорциональной привязки к нему указанной границы.

<ГраницаВторогоЭлемента> (необязательный)

Тип: [ГраницаЭлементаУправления](ГраницаЭлементаУправления_(ControlEdge).md).   
Граница второго элемента, к которой будет пропорционально привязана указанная граница.

Описание:

Устанавливает привязку границы элемента.

Доступность:

Толстый клиент.

Примечание:

Если не указывать необязательные параметры, то производится сброс привязок для границы элемента.

Пример:

|  |
| --- |
| ЭлементыФормы.ПолеВвода1.УстановитьПривязку(     ГраницаЭлементаУправления.Лево,     ЭлементыФормы.Элемент1,     ГраницаЭлементаУправления.Право,     Панель,     ГраницаЭлементаУправления.Лево); |

Использование в версии:

Доступен, начиная с версии 8.0.
