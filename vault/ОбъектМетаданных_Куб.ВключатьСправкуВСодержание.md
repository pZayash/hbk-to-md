ОбъектМетаданных: Куб (MetadataObject: Cube)

ВключатьСправкуВСодержание (IncludeHelpInContents)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак включения справки в содержание.  
Истина - справка по данному объекту метаданных (свойство [Справка](ОбъектМетаданных_Куб.Справка.md)) будет включена в общую справку по конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
