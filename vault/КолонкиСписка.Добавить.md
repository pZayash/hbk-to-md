КолонкиСписка (ListColumns)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Колонка>, <Автоудаление>)

Параметры:

<Колонка> (обязательный)

Тип: Строка.   
Имя колонки списка. Зависит от прикладного объекта.

<Автоудаление> (необязательный)

Тип: Булево.   
Разрешает удалять колонку, если данная колонка была удалена табличным полем.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [КолонкаСписка](КолонкаСписка.md).   

Описание:

Добавляет колонку в список.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Добавим в список справочника, с которым связано табличное поле,  // колонку с кодом элемента:  Список.Колонки.Добавить("Код"); // Для показа кода в табличном поле, вставим в него   // колонку с индексом 1, отображающую колонку "Код" из   // списка справочника:  ЭлементыФормы.Список.Колонки.Вставить(1,"Код").Значение = "Код"; |

Использование в версии:

Доступен, начиная с версии 8.0.
