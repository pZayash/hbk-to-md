ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтображатьЗаголовок (ShowTitle)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет отображением заголовка формы (включая звездочку модификации).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

- Свойство применяется только в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md).
- В режиме основного окна приложения [Обычный](РежимОсновногоОкнаКлиентскогоПриложения.Обычный_(MainClientApplicationWindowMode.Normal).md) свойство применяется только для форм, расположенных на начальной странице.
- В режиме основного окна приложения [РабочееМесто](РежимОсновногоОкнаКлиентскогоПриложения.РабочееМесто_(MainClientApplicationWindowMode.Workplace).md), [ПолноэкранноеРабочееМесто](РежимОсновногоОкнаКлиентскогоПриложения.ПолноэкранноеРабочееМесто_(MainClientApplicationWindowMode.FullscreenWorkplace).md) и [Киоск](РежимОсновногоОкнаКлиентскогоПриложения.Киоск_(MainClientApplicationWindowMode.Kiosk).md) свойство применяется для всех управляемых форм.
- В интерфейсе Такси, если на начальной странице располагается одна форма и свойство <ОтображатьЗаголовок> для этой формы установлено в значение [Истина](lang__def_BooleanTrue.md), то заголовок выводится вместо надписи "Начальная страница".

См. также:

[РежимОсновногоОкнаКлиентскогоПриложения](РежимОсновногоОкнаКлиентскогоПриложения_(MainClientApplicationWindowMode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
