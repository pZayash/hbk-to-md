# Расширение формы записи регистра сведений.ЗаписатьВФорме

Расширение формы записи регистра сведений (Information register record form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Запись занесена в ИБ."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
