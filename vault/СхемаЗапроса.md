Описание:

Объект для создания и редактирования запросов из встроенного языка.  
Для корректной работы с объектами, полученными из схемы запроса, родительские объекты таких объектов должны существовать.   
Например, нельзя получить [ПоляСхемыЗапроса](ПоляСхемыЗапроса.md) из [ОператорВыбратьСхемыЗапроса](ОператорВыбратьСхемыЗапроса.md), затем удалить этот оператор из [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md), и потом пробовать добавить новые выражения в [ПоляСхемыЗапроса](ПоляСхемыЗапроса.md).  
Также не допускается использование объектов, полученных из одной схемы запроса, в другой схеме запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.

Использование в версии:

Доступен, начиная с версии 8.3.5.
