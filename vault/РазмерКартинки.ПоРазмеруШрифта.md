РазмерКартинки (PictureSize)

ПоРазмеруШрифта (ByFontSize)

Доступен, начиная с версии 8.3.19.

Описание:

Данное значение интерпретируется следующим образом:

- высота картинки считается равной высоте шрифта (об используемой высоте шрифта см. далее),
- ширина картинки рассчитывается пропорционально исходным размерам картинки,
- картинка отображается пропорционально исходным размерам.

В качестве высоты шрифта используются следующие значения:

- Для ячеек табличного документа - шрифт ячейки табличного документа.
- Для рисунков табличного документа - шрифт, используемый табличным документом в качестве шрифта по умолчанию.
- Для декораций типа [Картинка](ЭлементГрафическойСхемыДекорация.Картинка.md) - шрифт, заданный при помощи свойства декорации [Шрифт](ЭлементГрафическойСхемыДекорация.Шрифт.md).
- Для расширения поля формы для картинки - шрифт, заданный при помощи свойства [Шрифт](Расширение_поля_формы_для_поля_картинки.Шрифт.md).
- Для следующих элементов графической схемы используется шрифт, заданный при помощи свойства Шрифт соответствующего элемента графической схемы:
  - [ЭлементГрафическойСхемыДекорация](ЭлементГрафическойСхемыДекорация.md),
  - [ЭлементГрафическойСхемыДействие](ЭлементГрафическойСхемыДействие.md),
  - [ЭлементГрафическойСхемыРазделение](ЭлементГрафическойСхемыРазделение.md),
  - [ЭлементГрафическойСхемыУсловие](ЭлементГрафическойСхемыУсловие.md),
  - [ЭлементГрафическойСхемыЗавершение](ЭлементГрафическойСхемыЗавершение.md),
  - [ЭлементГрафическойСхемыСтарт](ЭлементГрафическойСхемыСтарт.md),
  - [ЭлементГрафическойСхемыСлияние](ЭлементГрафическойСхемыСлияние.md),
  - [ЭлементГрафическойСхемыВложенныйБизнесПроцесс](ЭлементГрафическойСхемыВложенныйБизнесПроцесс.md),
  - [ЭлементГрафическойСхемыОбработка](ЭлементГрафическойСхемыОбработка.md),
  - [ЭлементГрафическойСхемыВыборВарианта](ЭлементГрафическойСхемыВыборВарианта.md).
- В остальных случаях - шрифт стиля "Обычный шрифт текста" (по умолчанию - Arial, 10pt).

Использование в версии:

Доступен, начиная с версии 8.3.19.
