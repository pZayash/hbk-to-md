ПоискВТаблицеПриВводе (SearchInTableOnInput)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

При вводе символов в таблицу, связанную с данными типа [ДинамическийСписок](ДинамическийСписок.md), выполняется поиск в таблице.  
При вводе символов в таблицу, связанную с [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией.md), поиск выполняется только в режиме совместимости конфигурации [Версия8\_2\_13](РежимСовместимости.Версия8_2_13.md).

Использование в версии:

Доступен, начиная с версии 8.2.
