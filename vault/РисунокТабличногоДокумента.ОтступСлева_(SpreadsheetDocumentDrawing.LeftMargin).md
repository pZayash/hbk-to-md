РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ОтступСлева (LeftMargin)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Отступ от левого края рисунка (в так называемых средних символах шрифта).  
Применим только для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md) и [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
