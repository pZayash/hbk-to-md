ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Объект, на основании которого производится заполнение.

Описание:

Заполняет данные объекта вид расчета данными из другого объекта (вида расчета или иного объекта базы данных). Метод инициирует событие [ОбработкаЗаполнения](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ОбработкаЗаполнения.md) и вызов соответствующей процедуры-обработчика в модуле объекта. В процедуре может размещаться алгоритм, заполняющий данные объекта на основании переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
