Глобальный контекст (Global context)

ЧасовойПояс (TimeZone)

Доступен, начиная с версии 8.2.

Синтаксис:

ЧасовойПояс()

Возвращаемое значение:

Тип: Строка.   
Список допустимых часовых поясов может быть получен методом [ПолучитьДопустимыеЧасовыеПояса](Глобальный_контекст.ПолучитьДопустимыеЧасовыеПояса.md).   
Кроме того, возможно задание часовых поясов смещением от универсального времени (часовые пояса GMT) в формате:  
GMT{+/-}h[h][:mm]  
где hh=0:23, mm=0:59.

Описание:

Получает идентификатор часового пояса компьютера.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
