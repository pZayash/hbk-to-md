ИнформацияОбАгрегатах (AggregatesInformation)

ОграничениеРазмера (SizeLimit)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную сумму размеров агрегатов в % от таблицы движений, использовавшуюся при последнем перестроении использования агрегатов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
