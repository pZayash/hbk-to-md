WSПрокси (WSProxy)

ЗащищенноеСоединение (SecureConnection)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md), [Неопределено](lang__def_Undefined.md).   
Содержит объект защищенного соединения для осуществления HTTPS соединения с веб-сервисом, если указан.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
