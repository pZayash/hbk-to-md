ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

ОбработкаПолученияПолейПредставления (PresentationFieldsGetProcessing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОбработкаПолученияПолейПредставления(<Поля>, <СтандартнаяОбработка>)

Параметры:

<Поля>

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий имена полей, которые нужны для формирования представления объекта или ссылки.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события.   
Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
При стандартной обработке в массив полей добавляются поля, используемые платформой для формирования стандартного представления объекта или ссылки.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при получении состава полей, из которых формируется представление объекта или ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Результат обработки события кэшируется для сеанса.  
Не рекомендуется в качестве полей представления возвращать поля неограниченной длины, т.к. это может приводить к ошибкам исполнения запроса, если представление будет получено для составных ссылок.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.3.1.
