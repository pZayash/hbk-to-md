ПараграфФорматированногоДокумента (FormattedDocumentParagraph)

Элементы (Items)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовФорматированногоДокумента](КоллекцияЭлементовФорматированногоДокумента_(FormattedDocumentItemCollection).md).   
Содержит коллекцию элементов параграфа форматированного документа.  
В коллекцию помещаются элементы только типов [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md) и [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
