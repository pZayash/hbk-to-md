Доступен, начиная с версии 8.3.19.

Элементы коллекции:

[ОписаниеЭлементаСпискаВыбораНавигационнойСсылки](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания элемента списка выбора навигационной ссылки.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0)..

Описание:

Предназначен для формирования списка выбора навигационных ссылок.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
