Форма (Form)

ОбработкаАктивизацииОбъекта (ObjectActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаАктивизацииОбъекта(<АктивныйОбъект>, <Источник>)

Параметры:

<АктивныйОбъект>

Тип: Ссылка на объект, либо [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md).   
Объект, который был активизирован.

<Источник>

Тип: [Форма](Форма_(Form).md).   
Форма - источник сообщения.

Описание:

Возникает в подчиненной форме, если в форме-владельце изменился активный объект или в форме-владельце был вызван метод [ОповеститьОбАктивизацииОбъекта](Форма.ОповеститьОбАктивизацииОбъекта_(Form.NotifyActivateObject).md).

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), метод [ОповеститьОбАктивизацииОбъекта](Форма.ОповеститьОбАктивизацииОбъекта_(Form.NotifyActivateObject).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
