Расширение поля формы для поля надписи (Form field extension for a label field)

Нажатие (Click)

Доступен, начиная с версии 8.2.

Синтаксис:

Нажатие(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при нажатии мышью или при помощи клавиатуры при установленном свойстве [Гиперссылка](Расширение_поля_формы_для_поля_надписи.Гиперссылка_(Form_field_extension_for_a_label_field.Hyperlink).md).  
Если параметр <СтандартнаяОбработка> не был установлен в [Ложь](lang__def_BooleanFalse.md), происходит открытие отображаемого значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
