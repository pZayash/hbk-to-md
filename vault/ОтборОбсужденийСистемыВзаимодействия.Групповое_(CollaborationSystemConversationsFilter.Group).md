ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

Групповое (Group)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Устанавливает отбор групповых обсуждений:

- [Истина](lang__def_BooleanTrue.md) - возвращаются обсуждения, у которых свойство [Групповое](ОбсуждениеСистемыВзаимодействия.Групповое_(CollaborationSystemConversation.Group).md) имеет значение [Истина](lang__def_BooleanTrue.md),
- [Ложь](lang__def_BooleanFalse.md) - возвращаются обсуждения, у которых свойство [Групповое](ОбсуждениеСистемыВзаимодействия.Групповое_(CollaborationSystemConversation.Group).md) имеет значение [Ложь](lang__def_BooleanFalse.md),
- [Неопределено](lang__def_Undefined.md) - возвращаются все обсуждения в соответствии с другими значениями отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.3.13.
