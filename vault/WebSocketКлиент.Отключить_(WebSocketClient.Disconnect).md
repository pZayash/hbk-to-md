# WebSocketКлиент.Отключить

WebSocketКлиент (WebSocketClient)

Отключить (Disconnect)

Доступен, начиная с версии 8.3.27.

Синтаксис:

Отключить()

Описание:

Выполняет закрытие соединения с сервером. После выполнения метода использовать данное соединение невозможно.  
Параметр [ПодключатьАвтоматически](WebSocketКлиент.ПодключатьАвтоматически_(WebSocketClient.AutoConnect).md) устанавливается в значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
