Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередРазворачиваниемЭлементаИзмерения (BeforeExpandDimensionItem)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПередРазворачиваниемЭлементаИзмерения(<ЭлементИзмерения>, <ЗначенияИзмерений>, <Отказ>)

Параметры:

<ЭлементИзмерения>

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md).   
Элемент измерения, на кнопку разворачивания которого было произведено нажатие.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Фиксированное соответствие, однозначно определяющее элемент измерения, на кнопку разворачивания которого было произведено нажатие. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если значение [Истина](lang__def_BooleanTrue.md), то разворачивание элемента не выполняется.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед разворачиванием элемента измерения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
