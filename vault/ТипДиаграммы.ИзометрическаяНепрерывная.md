ТипДиаграммы (ChartType)

ИзометрическаяНепрерывная (CeilGraph)

Доступен, начиная с версии 8.0.

Описание:

Изометрическая непрерывная диаграмма. Значения внутри одной серии объединяются лентой, области от лент до плоскости базового значения заполняются. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График.md).

Использование в версии:

Доступен, начиная с версии 8.0.
