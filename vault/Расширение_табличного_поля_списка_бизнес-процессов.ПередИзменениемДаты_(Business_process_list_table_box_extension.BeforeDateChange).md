# Расширение табличного поля списка бизнес-процессов.ПередИзменениемДаты

Расширение табличного поля списка бизнес-процессов (Business process list table box extension)

ПередИзменениемДаты (BeforeDateChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИзменениемДаты(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак изменения даты и времени бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то изменение даты бизнес-процесса выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед изменением даты бизнес-процесса при использовании интерактивной команды смены времени.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
