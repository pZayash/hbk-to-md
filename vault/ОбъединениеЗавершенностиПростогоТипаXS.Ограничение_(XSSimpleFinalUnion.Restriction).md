ОбъединениеЗавершенностиПростогоТипаXS (XSSimpleFinalUnion)

Ограничение (Restriction)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если значение перечисления [Ограничение](ЗавершенностьПростогоТипаXS.Ограничение_(XSSimpleFinal.Restriction).md) присутствует в объединении, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
