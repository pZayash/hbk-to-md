ПреобразованиеXSL (XSLTransform)

ЗагрузитьТаблицуСтилейXSL (LoadXSLStylesheet)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЗагрузитьТаблицуСтилейXSL(<ЧтениеXML>)

Параметры:

<ЧтениеXML> (обязательный)

Тип: [ЧтениеXML](ЧтениеXML.md).   
Объект чтения XML, из которого будет загружена таблица стилей XSL.

Описание:

Загружает таблицу стилей XSL.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
