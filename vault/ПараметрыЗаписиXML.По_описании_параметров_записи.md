ПараметрыЗаписиXML (XMLWriterSettings)

По описании параметров записи

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ПараметрыЗаписиXML(<Кодировка>, <Версия>, <Отступ>, <ОтступАтрибутов>, <СимволыОтступа>)

Параметры:

<Кодировка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Кодировка, которая будет использоваться при записи документа XML (актуальна для записи в файл).  
По умолчанию будет использована кодировка UTF-8.

<Версия> (необязательный)

Тип: [Строка](lang__def_String.md).   
Версия стандарта XML, соответствие которому будет проверяться при записи данных XML. Может принимать значения: "1.0" или "1.1".  
Значение по умолчанию: 1.0.

<Отступ> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак наличия отступов при записи структуры элементов документа XML.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ОтступАтрибутов> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак наличия отступов при записи атрибутов элемента XML.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<СимволыОтступа> (необязательный)

Тип: [Строка](lang__def_String.md).   
Последовательность символов, которая будет использована при записи отступов при записи данных XML. В последовательность могут входит только пробельные символы (Пробел, [ВК](Символы.ВК_(Chars.CR).md), [ПС](Символы.ПС_(Chars.LF).md), [Таб](Символы.Таб_(Chars.Tab).md)).  
Значение по умолчанию: [Таб](Символы.Таб_(Chars.Tab).md).

Описание:

Создает объект параметров записи XML.

Использование в версии:

Доступен, начиная с версии 8.1.
