Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати.md).   
Указывает способ вызова печати:

- при выборе команды "Печать с текущими настройками" - [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать.md),
- при выборе команды "Печать" - [Использовать](РежимИспользованияДиалогаПечати.Использовать.md).

<Отказ>

Тип: Булево.   
Установка данного параметра в Истина позволяет отказаться от записи.  
Значение по умолчанию: Ложь.

Описание:

Событие вызывается перед интерактивной печатью (при вызове команд "Печать" и "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
