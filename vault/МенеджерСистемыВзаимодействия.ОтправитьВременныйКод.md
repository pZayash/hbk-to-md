МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтправитьВременныйКод (SendTemporaryCode)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ОтправитьВременныйКод()

Описание:

Отправляет на адрес электронной почты абонента временный код, с помощью которого можно изменять список администраторов абонента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
