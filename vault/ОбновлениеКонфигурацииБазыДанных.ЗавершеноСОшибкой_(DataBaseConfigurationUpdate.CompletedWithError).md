ОбновлениеКонфигурацииБазыДанных (DataBaseConfigurationUpdate)

ЗавершеноСОшибкой (CompletedWithError)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, была ли предыдущая попытка фонового обновления конфигурации базы данных завершена с ошибкой.  
[Истина](lang__def_BooleanTrue.md) - завершена с ошибкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
