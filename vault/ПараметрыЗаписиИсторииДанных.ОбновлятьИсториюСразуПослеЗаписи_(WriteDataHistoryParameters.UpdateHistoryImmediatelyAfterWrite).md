ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ОбновлятьИсториюСразуПослеЗаписи (UpdateHistoryImmediatelyAfterWrite)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), то после завершения транзакции  
выполняется обновление истории данных по записанным данным,   
в фоновом задании без ожидания завершения фонового задания.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).  
При записи данных значение, указанное в конфигураторе, записывается в свойство [ОбновлятьИсториюСразуПослеЗаписи](ПараметрыЗаписиИсторииДанных.ОбновлятьИсториюСразуПослеЗаписи_(WriteDataHistoryParameters.UpdateHistoryImmediatelyAfterWrite).md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
