**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Анализ данных](objects__catalog63__catalog818.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаРезультатаМоделиПрогноза](КолонкаРезультатаМоделиПрогноза.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонкам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит колонки, выводимые в результат модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.
