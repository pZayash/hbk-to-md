Расширение табличного поля списка перечисления (Enum list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора.md).   
Предоставляет доступ к настройкам отбора списка перечисления. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для перечисления элементов отбора (в данном случае - единственное свойство "Ссылка").

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
