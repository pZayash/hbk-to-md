СтрокаГруппировкиДинамическогоСписка (DynamicListGroupRow)

РодительскаяГруппировка (ParentGrouping)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтрокаГруппировкиДинамическогоСписка](СтрокаГруппировкиДинамическогоСписка_(DynamicListGroupRow).md), [Неопределено](lang__def_Undefined.md).   
Содержит значение группировки верхнего уровня.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Неопределено](lang__def_Undefined.md) - если группировка осуществлялась по полю, в котором есть значения типа Null.

Использование в версии:

Доступен, начиная с версии 8.2.
