Глобальный контекст (Global context)

ВозможностьЧтенияXML (CanReadXML)

Доступен, начиная с версии 8.0.

Синтаксис:

ВозможностьЧтенияXML(<ЧтениеXML>)

Параметры:

<ЧтениеXML> (обязательный)

Тип: [ЧтениеXML](ЧтениеXML.md).   
Объект, через который производится чтение XML.

Возвращаемое значение:

Тип: Булево.   
Истина - тип 1С:Предприятия существует; Ложь - в противном случае.

Описание:

Производит проверку возможности чтения из XML значения указанного типа.   
Данный метод получает тип данных XML из объекта [ЧтениеXML](ЧтениеXML.md), а затем пытается определить, имеется ли соответствующий тип 1С:Предприятия, аналогично тому как это делает метод [ПолучитьXMLТип](Глобальный_контекст.ПолучитьXMLТип.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
