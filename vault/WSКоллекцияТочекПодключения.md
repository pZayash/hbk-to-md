# WSКоллекцияТочекПодключения

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Web-сервисы](objects__catalog63__catalog1031.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[WSТочкаПодключения](WSТочкаПодключения.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) коллекции.  

Методы:

[Количество (Count)](WSКоллекцияТочекПодключения.Количество.md)  
[Получить (Get)](WSКоллекцияТочекПодключения.Получить.md)  

Описание:

Коллекция точек подключения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[WSСервис](WSСервис.md), свойство [ТочкиПодключения](WSСервис.ТочкиПодключения.md)  

Использование в версии:

Доступен, начиная с версии 8.1.
