КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого поля.

Возвращаемое значение:

Тип: [ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md), [Неопределено](lang__def_Undefined.md).   
Если поле отсутствует, возвращает значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск поля сводной таблицы по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
