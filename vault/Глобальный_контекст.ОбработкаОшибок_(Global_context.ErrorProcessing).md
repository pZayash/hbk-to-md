Глобальный контекст (Global context)

ОбработкаОшибок (ErrorProcessing)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [МенеджерОбработкиОшибок](МенеджерОбработкиОшибок_(ErrorProcessingManager).md).   
Используется для настройки обработки ошибок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[МенеджерОбработкиОшибок](МенеджерОбработкиОшибок_(ErrorProcessingManager).md)  

Использование в версии:
