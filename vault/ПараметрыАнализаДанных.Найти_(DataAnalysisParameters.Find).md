# ПараметрыАнализаДанных.Найти

ПараметрыАнализаДанных (DataAnalysisParameters)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого параметра.

Возвращаемое значение:

Тип: [ПараметрАнализаДанных](ПараметрАнализаДанных_(DataAnalysisParameter).md), [Неопределено](lang__def_Undefined.md).   
Если параметр не найден, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск параметра по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
