ОбновлениеКонфигурацииБазыДанных (DataBaseConfigurationUpdate)

ПроцентОбработки (ProcessingPercent)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит число (от 0 до 100), отражающее процент отработанных записей на фазе обработки.   
100% означает, что фаза обработки завершена.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
