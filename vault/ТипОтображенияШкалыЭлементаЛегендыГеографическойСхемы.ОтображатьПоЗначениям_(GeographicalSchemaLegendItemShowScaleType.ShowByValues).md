ОтображатьПоЗначениям (ShowByValues)

Доступен, начиная с версии 8.0.

Описание:

Показывать шкалу для элемента легенды.  
Шкала рисуется по максимальному, минимальному и среднему значению серии.

Использование в версии:

Доступен, начиная с версии 8.0.
