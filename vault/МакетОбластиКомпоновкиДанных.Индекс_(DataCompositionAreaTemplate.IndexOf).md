Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableRow).md).

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
