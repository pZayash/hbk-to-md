ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриВставкеИзБуфераОбмена (OnPasteFromClipboard)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПриВставкеИзБуфераОбмена(<Значение>, <СтандартнаяОбработка>)

Параметры:

<Значение>

Тип: [Картинка](Картинка_(Picture).md), [Массив](Массив_(Array).md).   
Значение, полученное из буфера обмена.  
Картинка или массив из элементов типа [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается при вставке картинки или файла (файлов) из буфера обмена на активной форме.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
