ДействиеЭлементаПланировщика (PlannerItemAction)

Видимость (Visible)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то действие отображается в элементе планировщика, иначе скрыто.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
