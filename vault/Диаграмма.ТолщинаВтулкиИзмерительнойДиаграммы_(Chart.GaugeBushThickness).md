Диаграмма (Chart)

ТолщинаВтулкиИзмерительнойДиаграммы (GaugeBushThickness)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит относительную толщину втулки измерительной диаграммы. Задается числом 0 – 10 (0% - 10% от радиуса). В случае отображения значений в виде стрелки, определяет также толщину стрелки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
