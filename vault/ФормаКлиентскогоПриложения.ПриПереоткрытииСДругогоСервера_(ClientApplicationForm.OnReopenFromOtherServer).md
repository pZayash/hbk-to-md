ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриПереоткрытииСДругогоСервера (OnReopenFromOtherServer)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПриПереоткрытииСДругогоСервера(<Источник>, <ПараметрыЗаполнения>, <СтандартнаяОбработка>)

Параметры:

<Источник>

Исходная форма.

<ПараметрыЗаполнения>

Тип: [ПараметрыЗаполненияПриПереоткрытииФормы](ПараметрыЗаполненияПриПереоткрытииФормы_(OnReopenFormFillParameters).md).   
Реквизиты, которые копируются в новую форму.  
Реквизиты, которые не могут быть скопированы, добавляются в исключаемые.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения стандартной обработки.  
Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет (выбираются установленные разработчиком измененные параметры).

Описание:

Возникает при переоткрытии формы клиентского приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
