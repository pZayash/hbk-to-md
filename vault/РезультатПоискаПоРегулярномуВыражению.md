Описание:

Объект, получаемый в результате работы метода [СтрНайтиПоРегулярномуВыражению](Глобальный_контекст.СтрНайтиПоРегулярномуВыражению.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.3/regexp}. Имя типа XDTO: SearchResultByRegularExpression.

Доступен, начиная с версии 8.3.23.
