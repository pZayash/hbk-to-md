АдминистрированиеСервера (ServerAdministration)

СоздатьКластер (CreateCluster)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьКластер()

Возвращаемое значение:

Тип: [АдминистрированиеКластер](АдминистрированиеКластер.md).   

Описание:

Создает новый кластер с пустыми значениями свойств.  
Новый кластер может быть использован, например, для регистрации кластера. Для этого требуется явно задать необходимые значения его свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
