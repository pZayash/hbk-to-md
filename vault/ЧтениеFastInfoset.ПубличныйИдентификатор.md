ЧтениеFastInfoset (FastInfosetReader)

ПубличныйИдентификатор (PublicId)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит публичный идентификатор PiblicId для типов узлов, у которых идентификатор имеет смысл: [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента.md), [Сущность](ТипУзлаXML.Сущность.md), [Нотация](ТипУзлаXML.Нотация.md).  
Для типов узлов, для которых идентификатор не определен, содержит пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
