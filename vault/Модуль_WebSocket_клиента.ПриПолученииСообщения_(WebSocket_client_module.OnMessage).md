Модуль WebSocket клиента (WebSocket client module)

ПриПолученииСообщения (OnMessage)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПриПолученииСообщения(<Соединение>, <Сообщение>)

Параметры:

<Соединение>

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md).   
Ссылка на объект соединения.

<Сообщение>

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные сообщения.

Описание:

Обработчик получения сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
