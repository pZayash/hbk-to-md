ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает новый или измененный объект плана видов характеристик. Инициирует события [ПередЗаписью](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередЗаписью_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.BeforeWrite).md) и [ПриЗаписи](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПриЗаписи_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.OnWrite).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
