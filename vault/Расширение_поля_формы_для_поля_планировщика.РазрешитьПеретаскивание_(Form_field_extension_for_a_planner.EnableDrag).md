Расширение поля формы для поля планировщика (Form field extension for a planner)

РазрешитьПеретаскивание (EnableDrag)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешает начало операции перетаскивания в поле планировщика - приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
