ЗадачаСсылка.<Имя задачи> (TaskRef.<Task name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект для модификации и записи задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос; Запрос.Текст =  "ВЫБРАТЬ |    Ссылка  |ИЗ |    Задача.Задача | |ГДЕ |    Выполнена <> Истина"; Выборка = Запрос.Выполнить().Выбрать(); Пока Выборка.Следующий() Цикл     ТекущаяЗадача = Выборка.Ссылка.ПолучитьОбъект();     // Выполнение требуемых действий с задачей  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
