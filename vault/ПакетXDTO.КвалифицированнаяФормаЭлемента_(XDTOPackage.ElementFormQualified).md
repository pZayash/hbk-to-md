ПакетXDTO (XDTOPackage)

КвалифицированнаяФормаЭлемента (ElementFormQualified)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость квалификации свойств объектов в форме элемента при сериализации в XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
