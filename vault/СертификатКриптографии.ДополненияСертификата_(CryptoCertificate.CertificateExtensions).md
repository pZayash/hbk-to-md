СертификатКриптографии (CryptoCertificate)

ДополненияСертификата (CertificateExtensions)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит список дополнений сертификата, каждый элемент которого представляет собой тип [КлючИЗначение](КлючИЗначение_(KeyAndValue).md), ключом которого является строка, содержащая идентификатор объекта (OID) дополнения сертификата, а значение зависит от конкретного дополнения и представляет собой (ниже представлено в формате "Ключ (Аббревиатура) - Значение"):

- 2.5.29.15 (Key Usage) - тип [Число](lang__def_Number.md), битовая комбинация назначений ключа,
- 2.5.29.16 (Private Key Usage Period) - Массив дат – периода использования закрытого ключа,
- 2.5.29.17 (Subject Alternative Name) - Массив строк альтернативных наименований владельца,
- 2.5.29.19 (Basic Constraints) - тип [Булево](lang__Булево_(Boolean).md), признак того, что сертификат является сертификатом удостоверяющего центра,
- 2.5.29.31 (CRL Distribution Points) - Массив строк адресов (точек) распространения списков отозванных сертификатов,
- 2.5.29.32 (Certificate Policies) - Массив строк OID-ов политик сертификата,
- 2.5.29.37 (Extended Key Usage) - Массив строк OID-ов расширенного назначения ключа,
- 1.3.6.1.5.5.7.1.1 (Authority Information Access) - Массив строк адресов для загрузки сертификатов издателя или адресов OCSP-служб,
- 1.2.643.2.2.49.2 (Ограниченная лицензия КриптоПро CSP) - Строка, содержащая текст лицензии (может отсутствовать), являющаяся признаком использования КриптоПро CSP для работы с итоговым сертификатом в том случае, когда клиентская лицензия на использование КриптоПро CSP просрочена,
- 1.2.643.100.111 (Средство электронной подписи владельца) - Строка наименования, используемого владельцем квалифицированного сертификата, средства электронной подписи,
- 1.2.643.100.112 (Средства электронной подписи и УЦ издателя) - Массив строк наименований средств электронной подписи и средств аккредитованного удостоверяющего центра,
- 1.2.643.100.114 (IdentificationKind) - тип [Число](lang__def_Number.md), способ идентификации владельца сертификата удостоверяющим центром.
