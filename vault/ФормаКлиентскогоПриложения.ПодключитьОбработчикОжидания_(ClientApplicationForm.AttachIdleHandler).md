ФормаКлиентскогоПриложения (ClientApplicationForm)

ПодключитьОбработчикОжидания (AttachIdleHandler)

Доступен, начиная с версии 8.2.

Синтаксис:

ПодключитьОбработчикОжидания(<ИмяПроцедуры>, <Интервал>, <Однократно>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры, подключаемой в качестве обработчика ожидания.  
Процедура не должна содержать параметров.

<Интервал> (обязательный)

Тип: [Число](lang__def_Number.md).   
Интервал времени в секундах с точностью до 1/10 секунды, через который будет осуществляться вызов процедуры (положительное число).  
Если указано значение меньше 1, то значение третьего параметра должно быть равно [Истина](lang__def_BooleanTrue.md).  
Внимание! Не следует использовать данный обработчик в качестве таймера. Фактический интервал срабатывания может отличаться от указанного.

<Однократно> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак однократного выполнения обработчика ожидания.  
[Истина](lang__def_BooleanTrue.md) - указанный обработчик ожидания будет выполнен один раз.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Подключает указанную процедуру в качестве обработчика ожидания. Процедура будет вызываться в период ожидания системы каждый раз по истечению указанного интервала времени.  
Вызов будет осуществляться только в "состоянии покоя", то есть в тот момент, когда программа не выполняет никаких действий.  
Во время временной блокировки, вызванной методом [ЗаблокироватьРаботуПользователя](Глобальный_контекст.ЗаблокироватьРаботуПользователя_(Global_context.LockApplication).md) или соответствующей командой "Временная блокировка", обработчики ожидания не выполняются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Вызов обработчика ожидания продолжается пока форма не будет закрыта или пока не будет вызван метод формы [ОтключитьОбработчикОжидания](ФормаКлиентскогоПриложения.ОтключитьОбработчикОжидания_(ClientApplicationForm.DetachIdleHandler).md).

Использование в версии:

Доступен, начиная с версии 8.2.
