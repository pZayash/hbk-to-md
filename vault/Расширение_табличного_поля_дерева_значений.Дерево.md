Расширение табличного поля дерева значений (Value tree table box extension)

Дерево (Tree)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет тип отображения дерева значений. Если указано Истина - в виде дерева, если Ложь - в виде иерархического списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
