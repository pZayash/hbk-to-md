# МенеджерВнешнихХранилищДвоичныхДанных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Хранилище двоичных данных](Хранилище_двоичных_данных.md)

Доступен, начиная с версии 8.3.24.

Элементы коллекции:

[МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются МенеджерВнешнегоХранилищаДвоичныхДанных.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя внешнего хранилища.  

Методы:

[Добавить (Add)](МенеджерВнешнихХранилищДвоичныхДанных.Добавить.md)  
[Количество (Count)](МенеджерВнешнихХранилищДвоичныхДанных.Количество.md)  
[Найти (Find)](МенеджерВнешнихХранилищДвоичныхДанных.Найти.md)  
[Удалить (Delete)](МенеджерВнешнихХранилищДвоичныхДанных.Удалить.md)  

Описание:

Предоставляет доступ к внешним хранилищам двоичных данных. Может быть получен с помощью свойства глобального контекста [ВнешниеХранилищаДвоичныхДанных](Глобальный_контекст.ВнешниеХранилищаДвоичныхДанных.md).  
Требуется наличие у текущего пользователя права "Администрирование".

Доступность:

Сервер.

Пример:

|  |
| --- |
| // обход всех внешних хранилищ (без ХранилищеДвоичныхДанных)  Для каждого Мнджр Из ВнешниеХранилищаДвоичныхДанных Цикл     Дата = ТекущаяУниверсальнаяДата();     Мнджр.ОчиститьНеиспользуемоеМестоПоУниверсальнойДате(Дата); КонецЦикла;  // доступ к менеджеру по имени  Мнджр = ВнешниеХранилищаДвоичныхДанных["manager"]; |
