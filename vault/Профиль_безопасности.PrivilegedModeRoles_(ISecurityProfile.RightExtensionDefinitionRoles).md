Профиль безопасности (ISecurityProfile)

PrivilegedModeRoles (RightExtensionDefinitionRoles)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Список ролей через ";", использующихся во время активности привилегированного режима вместо ролей пользователя, если включен безопасный режим и для активного профиля безопасности не установлен флаг [FullPrivilegedMode](Профиль_безопасности.FullPrivilegedMode_(ISecurityProfile.PrivilegedModeInSafeModeAllowed).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
