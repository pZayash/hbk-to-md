БлокировкаСеансов (SessionsLock)

Установлена (Use)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет установку блокировки.  
Истина - блокировка включена.   
Ложь - выключена.   
Блокировка не позволяет начинать новые сеансы с информационной базой, если она включена и актуальна.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
