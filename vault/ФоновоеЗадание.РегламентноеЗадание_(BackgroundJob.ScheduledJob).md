ФоновоеЗадание (BackgroundJob)

РегламентноеЗадание (ScheduledJob)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [РегламентноеЗадание](РегламентноеЗадание_(ScheduledJob).md).   
Регламентное задание, связанное с фоновым заданием.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если фоновое задание было запущено явно, то содержит [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.1.
