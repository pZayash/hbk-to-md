ПостроительОтчетаАнализаДанных (DataAnalysisReportBuilder)

Выполнить (Execute)

Доступен, начиная с версии 8.0.

Синтаксис:

Выполнить()

Описание:

Выполняет анализ и записывает результат анализа в свойство [Результат](ПостроительОтчетаАнализаДанных.Результат_(DataAnalysisReportBuilder.Result).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
