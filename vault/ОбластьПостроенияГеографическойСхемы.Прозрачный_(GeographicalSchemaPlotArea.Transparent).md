Не рекомендуется использовать, начиная с версии 8.2.

- [ПрозрачныйФон](ОбластьПостроенияГеографическойСхемы.ПрозрачныйФон_(GeographicalSchemaPlotArea.Transparent).md)

[Истина](lang__def_BooleanTrue.md) - фон прозрачный.

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.2.

Описание изменено в версии 8.2.
