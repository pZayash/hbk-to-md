РежимАвтоВремя (AutoTimeMode)

Последним (Last)

Доступен, начиная с версии 8.0.

Описание:

Установить время объекта (документа, бизнес-процесса, задачи) так, чтобы он оказался последним на свою дату. Если при этом дата объекта совпадает с текущей по дню, то время объекта будет установлено не раньше, чем текущее.

Использование в версии:

Доступен, начиная с версии 8.0.
