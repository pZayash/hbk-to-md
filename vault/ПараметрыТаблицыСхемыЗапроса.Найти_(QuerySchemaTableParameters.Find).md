ПараметрыТаблицыСхемыЗапроса (QuerySchemaTableParameters)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение для поиска.

Возвращаемое значение:

Тип: [ПараметрТаблицыСхемыЗапроса](ПараметрТаблицыСхемыЗапроса_(QuerySchemaTableParameter).md).   

Описание:

Выполняет поиск по свойству [Выражение](ПараметрТаблицыСхемыЗапроса.Выражение_(QuerySchemaTableParameter.Expression).md) с учетом форматирования.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
