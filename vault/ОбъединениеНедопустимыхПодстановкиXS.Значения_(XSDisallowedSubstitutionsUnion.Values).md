ОбъединениеНедопустимыхПодстановкиXS (XSDisallowedSubstitutionsUnion)

Значения (Values)

Доступен, начиная с версии 8.1.

Синтаксис:

Значения()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает массив значений перечисления, объединенных в данном экземпляре объекта объединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
