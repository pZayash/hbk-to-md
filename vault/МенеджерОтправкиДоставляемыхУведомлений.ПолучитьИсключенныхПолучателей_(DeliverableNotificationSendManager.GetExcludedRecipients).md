МенеджерОтправкиДоставляемыхУведомлений (DeliverableNotificationSendManager)

ПолучитьИсключенныхПолучателей (GetExcludedRecipients)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьИсключенныхПолучателей(<ДанныеАутентификации>, <ИспользоватьПромежуточныйСервис>)

Параметры:

<ДанныеАутентификации> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Соответствие](Соответствие_(Map).md).   
Файл сертификата, необходимый для подключения к службе доставки "Apple Push Notification Service", может быть помещен в соответствии с ключом [APNS](ТипПодписчикаДоставляемыхУведомлений.APNS_(DeliverableNotificationSubscriberType.APNS).md).  
Если используется промежуточный сервис - ключ доступа к этому сервису.

<ИспользоватьПромежуточныйСервис> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость использования промежуточного сервиса отправки.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив строк с идентификаторами устройств.

Описание:

Выполняет запрос в сервис APNs Feedback, и получает оттуда список идентификаторов устройств подписчиков, до которых невозможно доставить уведомления.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
