# РежимОтображенияСтрокиПоискаДинамическогоСписка.ОтображатьПриИспользованииПолнотекстовогоПоиска

РежимОтображенияСтрокиПоискаДинамическогоСписка (DynamicListSearchStringViewMode)

ОтображатьПриИспользованииПолнотекстовогоПоиска (ShowOnUsingFullTextSearch)

Доступен, начиная с версии 8.3.26.

Описание:

Строка поиска отображается, если для основной таблицы списка полнотекстовый поиск используется.  
Если полнотекстовый поиск не используется для основной таблицы списка, строка поиска не отображается.  
Если в индексе полнотекстового поиска слишком большое количество не проиндексированных ссылок для основной таблицы списка, строка поиска не отображается.

Использование в версии:

Доступен, начиная с версии 8.3.26.
