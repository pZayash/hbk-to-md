Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаНавигационнойСсылке (URLClick)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НажатиеНаНавигационнойСсылке(<Источник>, <НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<Источник>

Тип: [ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), [ИзмерениеПланировщика](ИзмерениеПланировщика_(PlannerDimension).md), [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md), [МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика_(PlannerBackgroundIntervalLabel).md), [МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени_(TimeScaleItemLabel).md).   
Объект, содержащий форматированную строку, на гиперссылку которой было произведено нажатие.

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: [Строка](lang__def_String.md).   
Значение гиперссылки форматированной строки.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.

Описание:

Возникает при нажатии левой кнопкой мыши на навигационной ссылке форматированной строки элемента измерения планировщика (если свойство [ГиперссылкаЭлементаИзмерения](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаИзмерения_(Form_field_extension_for_a_planner.DimensionItemHyperlink).md) = [Ложь](lang__def_BooleanFalse.md)), метки интервала фона планировщика, метки элемента шкалы времени (если свойство [ГиперссылкаЭлементаШкалыВремени](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаШкалыВремени_(Form_field_extension_for_a_planner.TimeScaleItemHyperlink).md) = [Ложь](lang__def_BooleanFalse.md)), элемента планировщика и измерения планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
