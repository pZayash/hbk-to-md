ОбработчикиWebSocketКлиентСоединения (WebSocketClientConnectionHandlers)

ОбработчикОткрытияСоединения (OnOpenHandler)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры-обработчика события открытия соединения со следующими параметрами:

- <Соединение> - объект типа [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md).

Пустая строка, если это соединение [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент_(MetadataObject_WebSocketClient).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
