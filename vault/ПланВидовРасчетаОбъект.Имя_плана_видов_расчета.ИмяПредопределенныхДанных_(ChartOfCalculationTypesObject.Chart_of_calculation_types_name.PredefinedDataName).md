ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

ИмяПредопределенныхДанных (PredefinedDataName)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя предопределенного элемента, как оно задано в конфигурации.  
Если указано пустое имя, элемент не будет предопределенным.  
Если указано недопустимое имя, будет вызвано исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Поле обладает особенностями сортировки.

Использование в версии:

Доступен, начиная с версии 8.3.3.
