ФорматированныйДокумент (FormattedDocument)

ПолучитьЗакладкуПоПозиции (GetPositionBookmark)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЗакладкуПоПозиции(<Позиция>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция в форматированном документе. Позиция первого символа 0.

Возвращаемое значение:

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   

Описание:

Получает закладку по переданной позиции символа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если переданная позиция слишком велика и выходит за пределы документа - вызывает исключительную ситуацию.

Использование в версии:

Доступен, начиная с версии 8.2.
