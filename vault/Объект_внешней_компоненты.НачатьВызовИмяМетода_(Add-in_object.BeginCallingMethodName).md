# Объект внешней компоненты.НачатьВызов<ИмяМетода>

Объект внешней компоненты (Add-in object)

НачатьВызов<ИмяМетода> (BeginCalling<MethodName>)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьВызов<ИмяМетода>(<ОписаниеОповещения>, <Параметры>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения вызова метода со следующими параметрами:

- <РезультатВызова> - результат вызова метода.
- <ПараметрыВызова> – массив входных параметров. В него помещаются все параметры, переданные в метод, кроме первого ([ОписаниеОповещения](html.md)).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Параметры> (необязательный)

Тип: Произвольный.   
Список входных параметров внешней компоненты в том же порядке, в каком они указаны в компоненте.

Описание:

Начинает вызов метода объекта внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Объект внешней компоненты](Объект_внешней_компоненты_(Add-in_object).md), метод [\<ИмяМетода\>Асинх](Объект_внешней_компоненты.ИмяМетодаАсинх_(Add-in_object.MethodNameAsync).md)  

Использование в версии:
