РежимОбработкиПодкаталоговФайлаАрхива (ArchiveFileSubDirProcessingMode)

НеОбрабатывать (DontProcess)

Доступен, начиная с версии 8.3.26.

При задании маски в качестве имени для архивации подкаталоги не будут обработаны.

Использование в версии:

Доступен, начиная с версии 8.3.26.
