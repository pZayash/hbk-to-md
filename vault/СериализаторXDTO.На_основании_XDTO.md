СериализаторXDTO (XDTOSerializer)

На основании XDTO

Доступен, начиная с версии 8.1.

Синтаксис:

Новый СериализаторXDTO(<Фабрика>)

Параметры:

<Фабрика> (обязательный)

Тип: [ФабрикаXDTO](ФабрикаXDTO_(XDTOFactory).md).   
Фабрика типов XDTO.

Описание:

Создает сериализатор значений платформы 1С:Предприятие 8 в XML на основе XDTO.

Использование в версии:

Доступен, начиная с версии 8.1.
