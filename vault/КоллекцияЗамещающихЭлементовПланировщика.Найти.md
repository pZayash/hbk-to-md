КоллекцияЗамещающихЭлементовПланировщика (PlannerReplacementItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<ДатаЗамещения>)

Параметры:

<ДатаЗамещения> (обязательный)

Тип: Дата.   
Дата, элемент расписания для которой будет замещен новым элементом.  
Новый элемент добавляется в коллекцию замещающих элементов.

Возвращаемое значение:

Если указанный элемент отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск замещающего элемента планировщика по указанному значению свойства [Значение](ЭлементПланировщика.Значение.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
