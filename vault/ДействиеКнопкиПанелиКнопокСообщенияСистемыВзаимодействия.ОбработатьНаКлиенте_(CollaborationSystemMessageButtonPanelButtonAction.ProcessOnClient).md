ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonAction)

ОбработатьНаКлиенте (ProcessOnClient)

Доступен, начиная с версии 8.3.25.

Описание:

Действие кнопки вызывает обработчик [ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия](Глобальный_контекст.ПриНажатииКнопкиПанелиКнопокСообщенияСистемыВзаимодействия_(Global_context.OnCollaborationSystemMessageButtonPanelButtonClick).md) модуля клиентского приложения.   
Действие не доступно в интеграциях с мессенджерами.

Использование в версии:

Доступен, начиная с версии 8.3.25.
