ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ПолучитьПорядокКода (GetCodeOrder)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьПорядокКода()

Возвращаемое значение:

Тип: Строка.   

Описание:

Преобразует код счета в соответствии с маской счета. Если код счета имеет меньшую разрядность, то добавляются пробелы. Числовые фрагменты кода прижимаются вправо.  
Например, для счета "41.1" при маске "@@@.@@@.@@" метод возвращает строку " 41. 1".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для счетов с масками, использующими символы "@" и ".".

Пример:

|  |
| --- |
| Процедура ПередЗаписью(Отказ)     Порядок = ПолучитьПорядокКода(); КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
