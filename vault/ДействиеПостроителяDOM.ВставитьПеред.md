ДействиеПостроителяDOM (DOMBuilderAction)

ВставитьПеред (InsertBefore)

Доступен, начиная с версии 8.1.

Описание:

Вставить до. Результат чтения добавляется непосредственно до узла контекста документа. В качестве узла контекста могут выступать узлы, имеющие в качестве родительского: [Элемент](ТипУзлаDOM.Элемент.md) или [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md).

Использование в версии:

Доступен, начиная с версии 8.1.
