Расширение табличного поля дерева значений (Value tree table box extension)

ИзменятьПорядокСтрок (ChangeRowOrder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет возможность перемещения строк и изменение упорядочивания дерева значений.  
Истина - менять разрешено.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
