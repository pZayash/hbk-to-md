# Табличное поле

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Элементы управления](Элементы_управления.md)

Табличное поле - элемент управления, позволяющий работать с различными списками, наборами записей, табличными полями, а также динамическими наборами данных [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md) и [ДеревоЗначений](ДеревоЗначений_(ValueTree).md).  
Тип значения строки табличного поля зависит от типа связанного с табличным полем значения.
