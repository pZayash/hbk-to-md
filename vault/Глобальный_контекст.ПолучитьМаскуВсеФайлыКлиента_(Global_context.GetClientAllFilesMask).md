Глобальный контекст (Global context)

ПолучитьМаскуВсеФайлыКлиента (GetClientAllFilesMask)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМаскуВсеФайлыКлиента()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Маска возвращается в виде строки. Например: "\*.\*".

Описание:

Получает файловую маску, принятую для выбора всех файлов в операционной системе, в которой выполняется клиентское приложение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
