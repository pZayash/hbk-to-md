ГруппировкаТаблицыКомпоновкиДанных (DataCompositionTableGroup)

ПоляГруппировки (GroupFields)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПоляГруппировкиКомпоновкиДанных](ПоляГруппировкиКомпоновкиДанных_(DataCompositionGroupFields).md).   
Содержит набор полей, по которым осуществляется группировка.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
