АдминистрированиеКластер (AdministrationCluster)

ЗавершатьПроблемныеПроцессы (TerminateProblemProcess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак принудительного завершения проблемных процессов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Примечание:

[Истина](lang__def_BooleanTrue.md) - проблемные процессы завершаются принудительно;  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.3.14.
