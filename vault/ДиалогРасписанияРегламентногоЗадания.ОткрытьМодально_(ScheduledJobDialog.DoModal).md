ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Открывает модально диалог расписания.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [Показать](ДиалогРасписанияРегламентногоЗадания.Показать_(ScheduledJobDialog.Show).md).

См. также:

[ДиалогРасписанияРегламентногоЗадания](ДиалогРасписанияРегламентногоЗадания_(ScheduledJobDialog).md), метод [ОткрытьАсинх](ДиалогРасписанияРегламентногоЗадания.ОткрытьАсинх_(ScheduledJobDialog.DoAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.18.
