Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОписаниеХарактеристик](ОписаниеХарактеристик_(CharacteristicsDescription).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания характеристик.  
Возможно обращение к описанию характеристики посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ОписанияХарактеристик.Индекс_(CharacteristicsDescriptions.IndexOf).md)  
[Количество (Count)](ОписанияХарактеристик.Количество_(CharacteristicsDescriptions.Count).md)  
[Получить (Get)](ОписанияХарактеристик.Получить_(CharacteristicsDescriptions.Get).md)  

Описание:

Содержит коллекцию объектов [ОписаниеХарактеристик](ОписаниеХарактеристик_(CharacteristicsDescription).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
