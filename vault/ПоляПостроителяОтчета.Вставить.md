ПоляПостроителяОтчета (ReportBuilderFields)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: Строка.   
Путь к данным поля.

<Имя> (необязательный)

Тип: Строка.   
Имя поля.

<Позиция> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: [ПолеПостроителяОтчета](ПолеПостроителяОтчета.md).   

Описание:

Вставляет поле в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПостроительОтчета.ВыбранныеПоля.Вставить("Период","Период",              ПостроительОтчета.ВыбранныеПоля.Количество()+1); |

Использование в версии:

Доступен, начиная с версии 8.0.
