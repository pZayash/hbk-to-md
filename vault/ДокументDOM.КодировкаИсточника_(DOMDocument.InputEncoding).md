ДокументDOM (DOMDocument)

КодировкаИсточника (InputEncoding)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Кодировка, в которой записан документ (информация имеется на момент разбора).  
Пустая строка, если кодировка неизвестна (например, документ формируется в памяти).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
