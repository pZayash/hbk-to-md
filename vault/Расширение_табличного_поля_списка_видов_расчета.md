Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md), в котором отображается список видов расчета. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений, которая задается колонками объекта [ПланВидовРасчетаСписок.](ПланВидовРасчетаСписок.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСписок.Имя_плана_видов_расчета.md)>, связанного с табличным полем.

Доступность:
