ДействиеЭлементаПланировщика (PlannerItemAction)

Подсказка (ToolTip)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст всплывающей подсказки, которая появляется при наведении курсора на действие элемента планировщика.  
Значение по умолчанию: Пустая строка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
