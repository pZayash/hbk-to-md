СписокЗначений (ValueList)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: Число, [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md).   
Индекс удаляемого элемента либо сам элемент.

Описание:

Удаляет элемент из списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // удаление по индексу в списке  СписокТиповЦен.Удалить(0);  // удаление по элементу  НайденныйЭлемент = СписокТиповЦен.НайтиПоЗначению(ВыбЗначение); Если НайденныйЭлемент <> Неопределено Тогда     СписокТиповЦен.Удалить(НайденныйЭлемент); Иначе     Сообщить("Значение не найдено!"); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
