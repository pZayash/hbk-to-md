# Расширение поля формы для поля ввода.РежимВыбораИзСписка

Расширение поля формы для поля ввода (Form field extension for a text box)

РежимВыбораИзСписка (ListChoiceMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство регулирует работу поля ввода. Если значение свойства - [Истина](lang__def_BooleanTrue.md), то поле ввода работает как поле выбора и может хранить одно из значений произвольного типа, имеющихся в списке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - при выборе показываются представления значений списка, в противном случае - сами значения.

Использование в версии:

Доступен, начиная с версии 8.2.
