Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

СтандартныйПериод (StandardPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [НастройкаПериода](НастройкаПериода.md).   
Содержит объект [НастройкаПериода](НастройкаПериода.md), в соответствии с которым осуществляется настройка периода списка записей при открытии формы, содержащей данное табличное поле.

Доступность:

Толстый клиент.

Примечание:

Для непериодического регистра сведений смысла не имеет.

См. также:

[НастройкаПериода](НастройкаПериода.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
