Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати.md).   
Указывает способ вызова печати:

- [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать.md) - Если вызвана команда "Печать с текущими настройками",
- [Использовать](РежимИспользованияДиалогаПечати.Использовать.md) - если вызвана команда "Печать".

<Отказ>

Тип: Булево.   
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение Истина, печать выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает на клиенте перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
