# Расширение поля формы для поля форматированного документа.ПередЗаписью

Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если выполняется команда "Сохранить как".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) позволяет отказаться от записи.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивной записью форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
