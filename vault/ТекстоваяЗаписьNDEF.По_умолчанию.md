ТекстоваяЗаписьNDEF (TextRecordNDEF)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ТекстоваяЗаписьNDEF(<Текст>, <КодЯзыка>, <ИдентификаторЗаписи>)

Параметры:

<Текст> (обязательный)

Тип: Строка.   
Текст размером не более 10 Мб.

<КодЯзыка> (необязательный)

Тип: Строка.   
Код языка по стандарту ISO/реестру IANA.  
Размер не более 63 байт.  
Если Неопределено, используется "en-US".

<ИдентификаторЗаписи> (необязательный)

Тип: Строка.   
Идентификатор записи обычно не указывается для экономии места на физической NFC-метке. Размер не более 255 байт.

Описание:

Если в параметрах обнаружена ошибка, будет сгенерировано исключение категории [ОшибкаВоВремяВыполненияВстроенногоЯзыка](КатегорияОшибки.ОшибкаВоВремяВыполненияВстроенногоЯзыка.md) с описанием ошибки.

Использование в версии:

Доступен, начиная с версии 8.3.23.
