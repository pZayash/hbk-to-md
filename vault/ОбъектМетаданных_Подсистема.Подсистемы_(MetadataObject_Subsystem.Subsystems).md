ОбъектМетаданных: Подсистема (MetadataObject: Subsystem)

Подсистемы (Subsystems)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция подсистем, вложенных в подсистему.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Подсистема](ОбъектМетаданных_Подсистема_(MetadataObject_Subsystem).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
