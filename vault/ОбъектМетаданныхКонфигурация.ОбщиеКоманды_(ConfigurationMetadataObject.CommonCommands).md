ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОбщиеКоманды (CommonCommands)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все общие команды, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ОбщаяКоманда](ОбъектМетаданных_ОбщаяКоманда_(MetadataObject_CommonCommand).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
