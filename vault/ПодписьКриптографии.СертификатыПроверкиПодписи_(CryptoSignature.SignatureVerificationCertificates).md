ПодписьКриптографии (CryptoSignature)

СертификатыПроверкиПодписи (SignatureVerificationCertificates)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).  
Сертификаты, которые могут быть использованы для проверки подписи (сертификаты цепочек сертификата подписи (включая сертификат подписи), меток времени TSP, OCSP).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
