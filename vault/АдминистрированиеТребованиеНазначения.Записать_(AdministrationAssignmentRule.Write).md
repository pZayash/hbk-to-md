АдминистрированиеТребованиеНазначения (AdministrationAssignmentRule)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Добавляет или обновляет требование в списке требований рабочего сервера кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
