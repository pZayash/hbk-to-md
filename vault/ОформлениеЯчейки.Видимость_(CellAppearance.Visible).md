ОформлениеЯчейки (CellAppearance)

Видимость (Visible)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет признак видимости ячейки. Если все ячейки в строке составной колонки невидимы, то данная строка удаляется.

Доступность:

Толстый клиент.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - ячейка не отображается.

Использование в версии:

Доступен, начиная с версии 8.0.
