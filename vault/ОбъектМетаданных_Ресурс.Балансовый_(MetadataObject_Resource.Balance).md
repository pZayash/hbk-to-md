ОбъектМетаданных: Ресурс (MetadataObject: Resource)

Балансовый (Balance)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то для ресурса регистра бухгалтерии это означает, что по данному ресурсу будет вестись двойная запись (итог дебетуемых записей будет равен итогу кредитуемых записей).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
