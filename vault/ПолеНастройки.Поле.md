ПолеНастройки (CustomField)

Поле (Field)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Показывает, является ли поле допустимым к использованию в качестве поля построителя отчета.  
Истина - поле допустимо к использованию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
