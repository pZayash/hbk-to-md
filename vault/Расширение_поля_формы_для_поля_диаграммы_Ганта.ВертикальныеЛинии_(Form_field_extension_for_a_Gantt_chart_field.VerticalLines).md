# Расширение поля формы для поля диаграммы Ганта.ВертикальныеЛинии

Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ВертикальныеЛинии (VerticalLines)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Управляет отображением горизонтальных линий в области построения диаграммы.   
Если содержит [Ложь](lang__def_BooleanFalse.md), то все горизонтальные линии в области построения диаграммы не отображаются. Если содержит [Истина](lang__def_BooleanTrue.md), то все горизонтальные линии в области построения диаграммы отображаются.   
Если содержит [Неопределено](lang__def_Undefined.md), то все горизонтальные линии в области построения диаграммы отображаются.   
Значение по умолчанию - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
