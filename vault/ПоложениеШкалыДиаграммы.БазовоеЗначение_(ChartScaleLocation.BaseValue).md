ПоложениеШкалыДиаграммы (ChartScaleLocation)

БазовоеЗначение (BaseValue)

Доступен, начиная с версии 8.3.11.

Описание:

Шкала отображается на базовом значении перпендикулярной оси (для шкалы точек - на базовом значении оси значений, для шкалы значений – на базовом значении оси точек).

Использование в версии:

Доступен, начиная с версии 8.3.11.
