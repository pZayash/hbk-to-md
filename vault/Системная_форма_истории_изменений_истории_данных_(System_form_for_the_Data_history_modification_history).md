# Системная форма истории изменений истории данных

Доступен, начиная с версии 8.3.13.

Параметры формы:

[Данные (Data)](Системная_форма_истории_изменений_истории_данных.Данные_(System_form_for_the_Data_history_modification_history.Data).md)  

Описание:

Форма история изменений истории данных. Предоставляет список версий данных, открывается при вызове команды "История изменений".

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.21.
