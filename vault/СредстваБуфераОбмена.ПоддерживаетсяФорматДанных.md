СредстваБуфераОбмена (ClipboardTools)

ПоддерживаетсяФорматДанных (DataFormatSupported)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПоддерживаетсяФорматДанных(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена.md), Строка.   
Значение параметра типа Строка должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Возвращаемое значение:

Тип: Булево.   
В тонком и толстом клиентах всегда возвращает Истина.

Описание:

Проверяет, поддерживается ли переданный формат данных буфером обмена.  
Данный метод рекомендуется использовать перед вызовом метода [ПоместитьДанныеАсинх](СредстваБуфераОбмена.ПоместитьДанныеАсинх.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте имеются следующие особенности:

- в браузере Chrome (а также всех браузерах на базе движка Chromium) поддерживаются все форматы данных буфера обмена, начиная с версии 104 Chromium. До версии 104 Chromium не поддерживаются форматы данных, заданные строкой,
- в браузерах Firefox, Safari, Internet Explorer поддерживается только формат данных буфера обмена [Текст](СтандартныйФорматДанныхБуфераОбмена.Текст.md).

Использование в версии:

Доступен, начиная с версии 8.3.24.
