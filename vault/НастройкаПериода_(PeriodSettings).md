Описание:

Предназначен для ввода периодов. При помощи объекта можно настраивать интервалы просмотра в различных хронологических списках.  
Также используется для установки интервала просмотра средствами встроенного языка.

Доступность:

Толстый клиент.   
Сериализуется.

Пример:

|  |
| --- |
| НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод; НастройкаПериода.ВариантНачала = ВариантГраницыПериода.Месяц; НастройкаПериода.ВариантОкончания = ВариантГраницыПериода.БезОграничения; НастройкаПериода.РедактироватьКакИнтервал = Истина; НастройкаПериода.РедактироватьКакПериод = Ложь; |
