БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Номер (Number)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Содержит номер бизнес-процесса. Тип значения зависит от установки в Конфигураторе представления номера бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
