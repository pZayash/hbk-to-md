Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет минимальную ширину элемента (в символах), отображаемого на оси.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Используется при ПоддержкаМасштаба = МинимальнаяШирина.

Использование в версии:

Доступен, начиная с версии 8.0.
