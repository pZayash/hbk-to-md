# Расширение табличного поля списка колонок анализа данных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Свойства:

[НастройкаКолонок (ColumnsSetting)](Расширение_табличного_поля_списка_колонок_анализа_данных.НастройкаКолонок_(Data_analysis_column_list_table_box_extension.ColumnsSetting).md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), предназначенные для интерактивной работы со списком колонок анализа данных. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [КолонкаАнализаДанных](КолонкаАнализаДанных_(DataAnalysisColumn).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
