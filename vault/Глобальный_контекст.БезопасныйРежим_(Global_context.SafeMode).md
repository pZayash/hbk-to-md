Глобальный контекст (Global context)

БезопасныйРежим (SafeMode)

Доступен, начиная с версии 8.2.

Синтаксис:

БезопасныйРежим()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md), [Строка](lang__def_String.md).   
[Строка](lang__def_String.md), если установлен безопасный режим с указанием имени профиля безопасности.  
[Истина](lang__def_BooleanTrue.md), если установлен безопасный режим.  
[Ложь](lang__def_BooleanFalse.md), если безопасный режим не установлен.

Описание:

Получает текущее значение безопасного режима.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьБезопасныйРежим](Глобальный_контекст.УстановитьБезопасныйРежим_(Global_context.SetSafeMode).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
