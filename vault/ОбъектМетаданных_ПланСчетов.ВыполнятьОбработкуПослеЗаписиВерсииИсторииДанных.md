ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если имеет значение Истина, то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](ПланСчетовМенеджер.Имя_плана_счетов.ОбработкаПослеЗаписиВерсийИсторииДанных.md) можно обработать запись версии после обновления истории.  
При записи данных значение, указанное в конфигураторе, устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_ПланСчетов.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных.md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.15.
