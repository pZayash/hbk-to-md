Свойства:

[ДолготаВосточнойГраницы (EastBorderLongitude)](ПоказываемаяОбластьГеографическойСхемы.ДолготаВосточнойГраницы_(GeographicalSchemaShowedArea.EastBorderLongitude).md)

[ШиротаСевернойГраницы (NorthBorderLatitude)](ПоказываемаяОбластьГеографическойСхемы.ШиротаСевернойГраницы_(GeographicalSchemaShowedArea.NorthBorderLatitude).md)  
[ШиротаЮжнойГраницы (SouthBorderLatitude)](ПоказываемаяОбластьГеографическойСхемы.ШиротаЮжнойГраницы_(GeographicalSchemaShowedArea.SouthBorderLatitude).md)  

Описание:

Предоставляет доступ к границам показываемой области, в режиме масштабирования [ЗаданнаяОбласть](РежимОтображенияГеографическойСхемы.ЗаданнаяОбласть_(GeographicalSchemaShowMode.SpecifiedArea).md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| // Получим ограничивающий прямоугольник объекта  ОграничивающийПрямоугольник = Объект.ОграничивающийПрямоугольник;  ПоказываемаяОбласть =      ЭлементыФормы.ПолеГеографическойСхемы.ПоказываемаяОбласть;  // Установим у схемы новые границы ограничивающего прямоугольника  ПоказываемаяОбласть.ДолготаВосточнойГраницы =      ОграничивающийПрямоугольник.ДолготаВосточнойГраницы; ПоказываемаяОбласть.ДолготаЗападнойГраницы =      ОграничивающийПрямоугольник.ДолготаЗападнойГраницы; ПоказываемаяОбласть.ШиротаСевернойГраницы =      ОграничивающийПрямоугольник.ШиротаСевернойГраницы; ПоказываемаяОбласть.ШиротаЮжнойГраницы =      ОграничивающийПрямоугольник.ШиротаЮжнойГраницы;  // Установим новый режим масштабирования  Схема.ПоддержкаМасштаба =      РежимОтображенияГеографическойСхемы.ЗаданнаяОбласть |
