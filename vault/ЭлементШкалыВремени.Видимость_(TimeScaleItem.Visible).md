ЭлементШкалыВремени (TimeScaleItem)

Видимость (Visible)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Позволяет без добавления или удаления элементов управлять составом шкалы времени.  
Элемент шкалы может стать невидимым и автоматически, в зависимости от масштаба. На значение данного свойства автоматическое выключение влияния не оказывает.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - элемент шкалы не отображается.

[ЭлементШкалыВремени](ЭлементШкалыВремени_(TimeScaleItem).md), свойство [Формат](ЭлементШкалыВремени.Формат_(TimeScaleItem.Format).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
