ДокументDOM (DOMDocument)

СоздатьФрагментДокумента (CreateDocumentFragment)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьФрагментДокумента()

Возвращаемое значение:

Тип: [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md).   

Описание:

Создает узел DOM - фрагмент документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
