НастройкаНастройкиОформления (AppearanceSettingSettings)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента настройки оформления.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойНастройкиОформления](ЭлементУправленияНастройкойНастройкиОформления.md), Неопределено.   
Если не найдено, возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента по имени элемента настройки оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
