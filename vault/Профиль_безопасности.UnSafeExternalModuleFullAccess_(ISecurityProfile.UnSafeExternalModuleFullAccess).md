Профиль безопасности (ISecurityProfile)

UnSafeExternalModuleFullAccess (UnSafeExternalModuleFullAccess)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет разрешение запуска внешнего кода (внешние отчеты, внешние обработки, оператор Выполнить, [Вычислить](Глобальный_контекст.Вычислить_(Global_context.Eval).md), расширения конфигурации) без безопасного режима, если это указано в конфигурации.  
[Истина](lang__def_BooleanTrue.md) – для любого кода. [Ложь](lang__def_BooleanFalse.md) – разрешен запуск без включения безопасного режима только внешних модулей, перечисленных в списке разрешенных внешних модулей.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
