ТестируемаяТаблицаФормы (TestedFormTable)

ТекущееТолькоПросмотр (CurrentReadOnly)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущееТолькоПросмотр()

Возвращаемое значение:

Тип: Булево.   
Истина - объект не редактируется; Ложь в противном случае.

Описание:

Возвращает признак того, что пользователь может изменить значение элемента без настройки формы, но путем переключения закладок и полос прокрутки.  
Если объект является колонкой таблицы и таблица не находится в режиме редактирования, при вызове метода генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
