ДиалогВыбораФайла (FileDialog)

ВыбратьАсинх (ChooseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВыбратьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив выбранных имен файлов или [Неопределено](lang__def_Undefined.md), если выбор не осуществлен.

Описание:

Асинхронная функция. Вызывает диалог выбора файлов.  
Для работы метода на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ДиалогВыбораФайла](ДиалогВыбораФайла_(FileDialog).md), метод [Выбрать](ДиалогВыбораФайла.Выбрать_(FileDialog.Choose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
