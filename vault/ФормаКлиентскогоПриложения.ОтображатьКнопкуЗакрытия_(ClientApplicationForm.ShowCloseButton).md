ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтображатьКнопкуЗакрытия (ShowCloseButton)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md) - кнопка закрытия отображается (если способ открытия формы предполагает наличие такой кнопки), нажатие клавиши Esс закрывает форму.   
Если свойство установлено в значение [Ложь](lang__def_BooleanFalse.md) - кнопка закрытия скрывается, нажатие на клавишу Esc не приводит к закрытию формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Свойство применяется только в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md) и только в режимах основного окна [РабочееМесто](РежимОсновногоОкнаКлиентскогоПриложения.РабочееМесто_(MainClientApplicationWindowMode.Workplace).md), [ПолноэкранноеРабочееМесто](РежимОсновногоОкнаКлиентскогоПриложения.ПолноэкранноеРабочееМесто_(MainClientApplicationWindowMode.FullscreenWorkplace).md), и [Киоск](РежимОсновногоОкнаКлиентскогоПриложения.Киоск_(MainClientApplicationWindowMode.Kiosk).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
