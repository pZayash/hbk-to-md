ОкноКлиентскогоПриложения (ClientApplicationWindow)

ПолучитьНавигационнуюСсылку (GetURL)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьНавигационнуюСсылку()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает навигационную ссылку в формате 1С:Предприятия на текущую форму.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
