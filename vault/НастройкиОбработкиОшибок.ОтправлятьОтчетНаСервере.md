НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОтчетНаСервере (SendReportOnServer)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке.md).   
Указывает необходимость отправки отчета на сервере.  
Отчеты отправляются для механизмов, не предусматривающих обработку ошибок на клиенте. Например, для фоновых заданий или HTTP-сервисов.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
