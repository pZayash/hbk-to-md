<!-- signature:end -->

Диапазон (Range)

Конец (End)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Дата, Число, Неопределено.   
Верхняя граница диапазона. Значение верхней границы диапазона должно быть не меньше значения нижней границы. Неопределено - диапазон не ограничен сверху.  
Тип значения должен совпадать с типом значения нижней границы.  
Если свойство [Начало](Диапазон.Начало.md) имеет тип значения, отличный от Неопределено, то данное свойство должно иметь такой же тип значения или Неопределено.  
Если свойство [Начало](Диапазон.Начало.md) имеет тип значения Неопределено, то данное свойство не может иметь такой же тип значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
