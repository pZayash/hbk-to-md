ТекстовыйДокумент (TextDocument)

КодЯзыкаМакета (TemplateLanguageCode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит код языка, который используется для получения областей методом [ПолучитьОбласть](ТекстовыйДокумент.ПолучитьОбласть_(TextDocument.GetArea).md) в текстовом макете. В случае, когда свойство содержит значение [Неопределено](lang__def_Undefined.md), код языка макета определяется из настроек текущего пользователя. Если для текущего пользователя не определен код языка, или пользователи не заданы, то код языка выбирается из свойства "Основной язык" конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
