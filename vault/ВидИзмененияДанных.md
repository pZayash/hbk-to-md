Описание:

Содержит варианты видов изменения данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.1/data/enterprise. Имя типа XDTO: DataChangeType.
