ДополнительныйИндекс (AdditionalIndex)

ИндексируемыеПоля (IndexedFields)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Содержит список ключевых полей, по которым строится дополнительный индекс объекта метаданных.  
Элементы фиксированного массива имеют тип [ПолеИндекса](ПолеИндекса_(IndexField).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ПолеИндекса](ПолеИндекса_(IndexField).md)  

Использование в версии:

Доступен, начиная с версии 8.3.26.
