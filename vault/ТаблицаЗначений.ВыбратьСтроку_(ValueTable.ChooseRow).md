# ТаблицаЗначений.ВыбратьСтроку

ТаблицаЗначений (ValueTable)

ВыбратьСтроку (ChooseRow)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьСтроку(<Заголовок>, <НачальнаяСтрока>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок диалогового окна выбора строки. Может использоваться для подсказки пользователю.

<НачальнаяСтрока> (необязательный)

Тип: [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений_(ValueTableRow).md).   
Указывает строку таблицы значений, на которой будет установлен курсор при открытии окна выбора.

Возвращаемое значение:

Тип: [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений_(ValueTableRow).md), [Неопределено](lang__def_Undefined.md).   
Если пользователь отказался от выбора, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Вызывает модальное диалоговое окно для интерактивного выбора строки из таблицы значений.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Адрес = ТаблицаАдресов.ВыбратьСтроку("Выберите получателя"); |

Использование в версии:

Доступен, начиная с версии 8.0.
