# ПланОбменаОбъект.<Имя плана обмена>.ЭтоНовый

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - изменяется еще ни разу не записанный узел; [Ложь](lang__def_BooleanFalse.md) - узел уже записан.

Описание:

Определяет, записан ли узел в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если Объект.ЭтоНовый() Тогда     Объект.Наименование = "Новый узел"; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
