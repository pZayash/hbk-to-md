Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаМоделиПрогноза](КолонкаМоделиПрогноза.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонкам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КолонкиМоделиПрогноза.Индекс.md)

[Найти (Find)](КолонкиМоделиПрогноза.Найти.md)  
[Получить (Get)](КолонкиМоделиПрогноза.Получить.md)  

Описание:

Содержит колонки модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.
