Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ДополнительныеПараметрыСоздания (AdditionalCreateParameters)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md).   
Содержит параметры, которые будут переданы в форму создания нового элемента.  
Структура заполняется из параметров формы списка [ДополнительныеПараметры](Расширение_формы_клиентского_приложения_для_объектов.ДополнительныеПараметры_(Client_application_form_extension_for_objects.AdditionalParameters).md) после вызова обработчика [ПриСозданииНаСервере](ФормаКлиентскогоПриложения.ПриСозданииНаСервере_(ClientApplicationForm.OnCreateAtServer).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
