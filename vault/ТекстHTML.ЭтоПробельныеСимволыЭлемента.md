Использование:

Только чтение.

Описание:

Содержит признак пробельных символов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Истина - текст является пробельными символами узла [Элемент](ТипУзлаDOM.Элемент.md). Ложь - в противном случае (игнорируемые пробельные символы).
