# ЗадачаОбъект.<Имя задачи>.АктивироватьИнтерактивно

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Методы](ЗадачаОбъект.Имя_задачи__Методы.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

АктивироватьИнтерактивно (ActivateInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

АктивироватьИнтерактивно()

Описание:

Выполняет интерактивную активацию задачи. Это приводит к возникновению события [ОбработкаИнтерактивнойАктивации](ЗадачаОбъект.Имя_задачи.ОбработкаИнтерактивнойАктивации_(TaskObject.Task_name.InteractiveActivationProcessing).md) задачи, а потом, если разрешена стандартная обработка, события [ОбработкаИнтерактивнойАктивации](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаИнтерактивнойАктивации_(BusinessProcessRoutePointRef.Business_process_name.InteractiveActivationProcessing).md) у точки бизнес-процесса.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
