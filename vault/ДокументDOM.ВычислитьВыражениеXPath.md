ДокументDOM (DOMDocument)

ВычислитьВыражениеXPath (EvaluateXPathExpression)

Доступен, начиная с версии 8.1.

Синтаксис:

ВычислитьВыражениеXPath(<Выражение>, <УзелКонтекста>, <Разыменователь>, <ТипРезультата>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.   
Строка - выражение XPath.

<УзелКонтекста> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Узел DOM - контекст для вычисления выражения XPath. Узел должен принадлежать данному документу DOM, (в противном случае будет сгенерировано исключение) и может быть следующих видов: [Документ](ТипУзлаDOM.Документ.md), [Элемент](ТипУзлаDOM.Элемент.md), [Атрибут](ТипУзлаDOM.Атрибут.md), [Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md), [Комментарий](ТипУзлаDOM.Комментарий.md), [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md), [ПространствоИменXPath](ТипУзлаDOM.ПространствоИменXPath.md).

<Разыменователь> (обязательный)

Тип: [РазыменовательПространствИменDOM](РазыменовательПространствИменDOM.md).   
Объект разрешения пространств имен в выражении XPath.

<ТипРезультата> (необязательный)

Тип: [ТипРезультатаDOMXPath](ТипРезультатаDOMXPath.md).   
Тип результата вычисления выражения XPath. Если не указан, то результат вычисления выражения может иметь любой тип.  
Значение по умолчанию: [Любой](ТипРезультатаDOMXPath.Любой.md).

Возвращаемое значение:

Тип: [РезультатXPath](РезультатXPath.md).   

Описание:

Производит вычисление выражения ХPath.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если выражение XPath содержит ошибки или в процессе вычисления выражения не удалось разыменовать префикс пространства имен, то будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
