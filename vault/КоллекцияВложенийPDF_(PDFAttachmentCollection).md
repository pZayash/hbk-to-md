Элементы коллекции:

[ВложениеPDF](ВложениеPDF_(PDFAttachment).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементам коллекции посредством оператора [...]. В качестве аргумента передается индекс.
