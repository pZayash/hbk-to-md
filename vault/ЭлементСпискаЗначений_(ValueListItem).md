Описание:

Используется для доступа к свойствам и методам элемента списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ВыборкаТиповЦен = Справочники.ТипыЦен.Выбрать(); ТипыЦен = Новый СписокЗначений; Пока ВыборкаТиповЦен.Следующий() Цикл     ТекНаценка = ВыборкаТиповЦен.Наценка;     Если ТекНаценка > 10 Тогда         ТипыЦен.Добавить(ВыборкаТиповЦен.Ссылка, ТекНаценка);     КонецЕсли; КонецЦикла;  Выбор = ТипыЦен.ВыбратьЭлемент("Укажите требуемую наценку."); Если Выбор <> Неопределено Тогда     ТипЦен = Выбор.Значение; КонецЕсли; |
