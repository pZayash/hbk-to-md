МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблоныСообщенияАсинх (GetMessageTemplatesAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблоныСообщенияАсинх(<Интеграция>)

Параметры:

<Интеграция> (обязательный)

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия.md).   
Идентификатор интеграции, для которой возвращается список шаблонов сообщений.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - массив объектов типа [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия.md).

Описание:

Асинхронная функция. Получает массив, содержащий шаблоны сообщения системы взаимодействия.  
Для интеграции типа WhatsApp возвращает список шаблонов сообщений.  
Для остальных интеграций возвращает пустой массив.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ПолучитьШаблоныСообщения](МенеджерСистемыВзаимодействия.ПолучитьШаблоныСообщения.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
