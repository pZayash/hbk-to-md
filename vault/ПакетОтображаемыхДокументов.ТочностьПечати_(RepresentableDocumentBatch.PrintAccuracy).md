ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ТочностьПечати (PrintAccuracy)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ТочностьПечати](ТочностьПечати_(PrintAccuracy).md), [Неопределено](lang__def_Undefined.md).   
Указывает необходимость получения точной печати при работе в веб-клиенте.   
Используется только из самого верхнего пакета документов (не вложенного пакета).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Неопределено](lang__def_Undefined.md) - для вложенных пакетов.

Использование в версии:

Доступен, начиная с версии 8.3.3.
