# КоллекцияПолейСводнойДиаграммы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Сводная диаграмма](Сводная_диаграмма.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы_(PivotChartFieldCollection).md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы_(PivotChartField).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной диаграммы или коллекция полей сводной диаграммы.  
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс (нумерация с 0) поля.
