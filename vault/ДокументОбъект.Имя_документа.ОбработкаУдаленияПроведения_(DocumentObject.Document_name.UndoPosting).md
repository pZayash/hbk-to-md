# ДокументОбъект.<Имя документа>.ОбработкаУдаленияПроведения

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

ОбработкаУдаленияПроведения (UndoPosting)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаУдаленияПроведения(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отмены проведения документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то отмена проведения документа произведена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при отмене проведения документа. Выполняется в транзакции записи.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
