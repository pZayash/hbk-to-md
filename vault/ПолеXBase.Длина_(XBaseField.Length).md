ПолеXBase (XBaseField)

Длина (Length)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит общую длину поля.   
Существуют следующие ограничения на длину поля в зависимости от его типа:

- числовое поле ("N", "F") - максимальная длина поля 19 символов;
- строковое поле ("S") - максимальная длина поля 254 символа;
- поле типа дата ("D") - длина поля равна 8 символам;
- поле типа булево ("L") - длина поля равна одному символу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
