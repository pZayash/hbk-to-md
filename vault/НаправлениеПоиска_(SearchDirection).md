# НаправлениеПоиска

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Разные](Разные.md)

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).

Значения

[СКонца (FromEnd)](НаправлениеПоиска.СКонца_(SearchDirection.FromEnd).md)  
[СНачала (FromBegin)](НаправлениеПоиска.СНачала_(SearchDirection.FromBegin).md)  

Описание:

Определяет направление поиска подстроки в строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Найти](Глобальный_контекст.Найти_(Global_context.Find).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5 (в режиме совместимости с версией 8.3.5 и последующими).
