ПроверкаУсловия (ConditionCheck)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверкаУсловия(<ТочкаМаршрута>, <Результат>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка условного перехода, для которой вызывается обработчик для определения одной из двух возможных вариантов точек маршрута бизнес-процесса.

<Результат>

Тип: Булево.   
Признак завершения бизнес-процесса. Если в параметре возвращается Истина, то бизнес-процесс идет по той ветке, которая на карте обозначена особым образом (по умолчанию справа).

Описание:

Возникает в процессе определения следующей точки. Если в параметре <Результат> возвращается Истина, то выполнение бизнес-процесса пойдет по той ветке, которая на карте обозначена особым образом (по умолчанию справа).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Условие](ВидТочкиМаршрутаБизнесПроцесса.Условие.md).  
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
