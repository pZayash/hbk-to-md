# СтандартныйПериод.ДатаНачала

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [СтандартныйПериод](СтандартныйПериод_(StandardPeriod).md) › [Свойства](СтандартныйПериод__Свойства.md)

СтандартныйПериод (StandardPeriod)

ДатаНачала (StartDate)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
В случае, если вариант периода - [ПроизвольныйПериод](ВариантСтандартногоПериода.ПроизвольныйПериод_(StandardPeriodVariant.Custom).md), содержит дату, которая будет использоваться в качестве начала периода.  
В случае, если вариант периода не [ПроизвольныйПериод](ВариантСтандартногоПериода.ПроизвольныйПериод_(StandardPeriodVariant.Custom).md), содержит дату, которая будет использоваться в качестве начала периода, если компоновку выполнить сейчас.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
