Менеджер кластера (IClusterManagerInfo)

MainManager (MainManager)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Указывает, является ли менеджер главным или дополнительным менеджером кластера:  
1 - главный менеджер;   
0 - дополнительный менеджер (не главный менеджер).   
В кластере всегда есть один главный менеджер кластера. Он не может быть удален или изменен.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
