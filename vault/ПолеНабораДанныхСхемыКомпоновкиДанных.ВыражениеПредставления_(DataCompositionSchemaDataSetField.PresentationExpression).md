ПолеНабораДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetField)

ВыражениеПредставления (PresentationExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение, по которому вычисляется представление поля. Может быть использовано для переопределения стандартного представления поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
