ПланСчетовСсылка.<Имя плана счетов> (ChartOfAccountsRef.<Chart of accounts name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ПланСчетовМенеджер.Имя_плана_счетов.ПолучитьСсылку_(ChartOfAccountsManager.Chart_of_accounts_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ПланСчетовМенеджер.\<Имя плана счетов\>](ПланСчетовМенеджер.Имя_плана_счетов_(ChartOfAccountsManager.Chart_of_accounts_name).md), метод [ПолучитьСсылку](ПланСчетовМенеджер.Имя_плана_счетов.ПолучитьСсылку_(ChartOfAccountsManager.Chart_of_accounts_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
