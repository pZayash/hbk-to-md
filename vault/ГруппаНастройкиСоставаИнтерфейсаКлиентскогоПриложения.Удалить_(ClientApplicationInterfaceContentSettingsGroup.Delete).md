ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceContentSettingsGroup)

Удалить (Delete)

Доступен, начиная с версии 8.3.4.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого элемента.

Описание:

Удаляет элемент из коллекции по указанному индексу.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
