ПоляПостроителяОтчета (ReportBuilderFields)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество полей в коллекции полей построителя отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для Индекс = 0 По ПостроительОтчета.ВыбранныеПоля.Количество()-1 Цикл     Поле = ПостроительОтчета.ВыбранныеПоля[Индекс];     // обработка поля  КонецЦикла;  Для Индекс = 0 По ПостроительОтчета.ДоступныеПоля.Количество()-1 Цикл     Поле = ПостроительОтчета.ДоступныеПоля[Индекс];     // обработка поля  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
