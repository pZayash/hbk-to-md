ЧтениеPDF (PDFReader)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Закрывает открытый файл. Закрытие файла выполняется автоматически при удалении объекта [ЧтениеPDF](ЧтениеPDF_(PDFReader).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [Закрыть](ЧтениеPDF.Закрыть_(PDFReader.Close).md)  
[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [ЗакрытьАсинх](ЧтениеPDF.ЗакрытьАсинх_(PDFReader.CloseAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
