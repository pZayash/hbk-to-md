HTTPСоединение (HTTPConnection)

Записать (Put)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ЗаписатьАсинх](HTTPСоединение.ЗаписатьАсинх_(HTTPConnection.PutAsync).md)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<HTTPЗапрос>)

Параметры:

<HTTPЗапрос> (обязательный)

Тип: [HTTPЗапрос](HTTPЗапрос_(HTTPRequest).md).   
HTTP-запрос.

Возвращаемое значение:

Тип: [HTTPОтвет](HTTPОтвет_(HTTPResponse).md).   

Описание:

Используется для отправки данных на HTTP-сервер с помощью PUT-запроса.

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
