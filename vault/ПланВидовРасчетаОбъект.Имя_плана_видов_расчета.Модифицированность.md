ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: Булево.   
Истина - объект изменен; Ложь - в противном случае.

Описание:

Определяет, был ли изменен объект после считывания из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, был ли изменен объект другими пользователями.

Пример:

|  |
| --- |
| Если НаборЗаписей.Модифицированность() Тогда     // Запись набора записей.      Попытка         НаборЗаписей.Записать();     Исключение         Предупреждение(НСтр("ru = 'Не удалось записать: ';"            + " en = 'Can't write: '") + ОписаниеОшибки());         Отказ = Истина;     КонецПопытки; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
