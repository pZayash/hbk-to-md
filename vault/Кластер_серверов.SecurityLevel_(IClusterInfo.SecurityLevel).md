# Кластер серверов.SecurityLevel

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Кластер серверов](Кластер_серверов_(IClusterInfo).md) › [Свойства](Кластер_серверов__Свойства.md)

Кластер серверов (IClusterInfo)

SecurityLevel (SecurityLevel)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет уровень безопасности соединений менеджера кластера.

Доступность:

Интеграция.

Примечание:

0 - незащищенное соединение, 1 - защищенное соединение только в процессе аутентификации администратора, 2 - защищенное соединение в течение всего сеанса.

Использование в версии:

Доступен, начиная с версии 8.1.
