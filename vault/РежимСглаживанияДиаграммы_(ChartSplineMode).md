# РежимСглаживанияДиаграммы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md) › [Диаграмма](Диаграмма-2.md)

Доступен, начиная с версии 8.3.1.

Значения

[ГладкаяКривая (SmoothCurve)](РежимСглаживанияДиаграммы.ГладкаяКривая_(ChartSplineMode.SmoothCurve).md)  
[Нет (None)](РежимСглаживанияДиаграммы.Нет_(ChartSplineMode.None).md)  

Описание:

Содержит режимы использования сплайнов ("гладких" кривых) в диаграммах "непрерывного" вида для визуализации соединений между точками серий.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartSplineMode.

Использование в версии:

Доступен, начиная с версии 8.3.1.
