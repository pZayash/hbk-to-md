КоллекцияЭлементовПорядкаКомпоновкиДанных (DataCompositionOrderItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных.md), [ЭлементПорядкаКомпоновкиДанных](ЭлементПорядкаКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных.md), [ЭлементПорядкаКомпоновкиДанных](ЭлементПорядкаКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
