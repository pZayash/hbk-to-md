Панель (Panel)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет.md).   
Содержит цвет фона панели.

Доступность:

Толстый клиент.

Примечание:

Един для всех страниц и закладок. Неактивные закладки отображаются этим же цветом, но немного притенено.  
Не используется, если свойство [ПрозрачныйФон](Панель.ПрозрачныйФон.md) имеет значение Истина.

Использование в версии:

Доступен, начиная с версии 8.0.
