ТипВыводаКопииБазыДанныхКомпоновкиДанных (DataCompositionDatabaseCopyOutputType)

Авто (Auto)

Доступен, начиная с версии 8.3.14.

Описание:

Значение по умолчанию. Имя копии базы данных будет выведено в заголовок отчета, если данные получены из копии базы данных.   
Если данные получены из основной базы данных - не выводится.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
