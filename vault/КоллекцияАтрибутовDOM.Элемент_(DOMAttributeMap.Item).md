КоллекцияАтрибутовDOM (DOMAttributeMap)

Элемент (Item)

Доступен, начиная с версии 8.1.

Синтаксис:

Элемент(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс атрибута в коллекции.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   

Описание:

Получает узел атрибута DOM по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
