МенеджерРаботыСРечью (SpeechProcessingManager)

Синтезировать (Synthesize)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Синтезировать(<Текст>, <Голос>, <ПараметрыСинтеза>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст, по которому синтезируется речь.

<Голос> (необязательный)

Тип: [Строка](lang__def_String.md).   
Голос синтеза речи.  
Если не указан – голос определяется сервисом.

<ПараметрыСинтеза> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Дополнительные параметры синтеза. Возможный список параметров и их значений зависит от выбранного голоса; будет получен с помощью метода [ПолучитьДоступныеГолосаСинтеза](МенеджерРаботыСРечью.ПолучитьДоступныеГолосаСинтеза_(SpeechProcessingManager.GetTextToSpeechAvailableVoices).md).  
  
Заполняется на основании [ОписаниеПараметраГолосаСинтезаРечи](ОписаниеПараметраГолосаСинтезаРечи_(TextToSpeechVoiceParameterDescription).md).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Синтезирует речь и возвращает результат в виде аудиофайла (OPUS в контейнере OGG).

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.25.
