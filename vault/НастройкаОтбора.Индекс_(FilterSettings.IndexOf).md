НастройкаОтбора (FilterSettings)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md).   
Искомый элемент управления отбором.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента управления отбором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
