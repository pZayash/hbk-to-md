ЭлементыМакетаОбластиКомпоновкиДанных (DataCompositionAreaTemplateItems)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [ПолеОбластиКомпоновкиДанных](ПолеОбластиКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
