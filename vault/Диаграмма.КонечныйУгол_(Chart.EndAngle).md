Диаграмма (Chart)

КонечныйУгол (EndAngle)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Угол конца отображения секторов по часовой стрелке от вертикальной оси.  
Используется в круговой и кольцевой диаграммах.  
Значение [Неопределено](lang__def_Undefined.md) интерпретируется в зависимости от типа диаграммы: для круговой и кольцевой диаграммы - 360.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.24.
