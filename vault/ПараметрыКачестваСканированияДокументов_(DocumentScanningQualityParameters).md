Описание:

Содержит результат выполнения методов [ПоказатьСканированиеДокументов](СредстваМультимедиа.ПоказатьСканированиеДокументов_(MultimediaTools.ShowDocumentScanning).md) и [СканироватьДокументыАсинх](СредстваМультимедиа.СканироватьДокументыАсинх_(MultimediaTools.ScanDocumentsAsync).md), возвращаемый в свойстве объекта [СтраницаСканированияДокументов](СтраницаСканированияДокументов_(DocumentScanningPage).md) для контроля уровня качества полученной страницы документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/data/ext. Имя типа XDTO: DocumentScanningQualityParameters.

Использование в версии:

Доступен, начиная с версии 8.3.23.
