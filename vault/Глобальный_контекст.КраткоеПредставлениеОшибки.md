Глобальный контекст (Global context)

КраткоеПредставлениеОшибки (BriefErrorDescription)

Не рекомендуется использовать, начиная с версии 8.3.17.

Рекомендуется использовать:

- [КраткоеПредставлениеОшибки](МенеджерОбработкиОшибок.КраткоеПредставлениеОшибки.md)

Доступен, начиная с версии 8.1.

Синтаксис:

КраткоеПредставлениеОшибки(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Содержит информацию об ошибке.

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает строку, аналогичную той, что показывается в стандартном сообщении об ошибке платформы "1С:Предприятие".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.17.

Описание изменено в версии 8.3.17.
