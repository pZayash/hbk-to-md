Расширение тестируемого поля формы для поля надписи (Tested form field extension for a label field)

НажатьНаГиперссылкуВФорматированнойСтроке (ClickFormattedStringHyperlink)

Доступен, начиная с версии 8.3.13.

Синтаксис:

НажатьНаГиперссылкуВФорматированнойСтроке(<Гиперссылка>)

Параметры:

<Гиперссылка> (обязательный)

Тип: Строка, Число.   
Представление выбираемой гиперссылки или ее номер по порядку отображения (нумерация индекса начинается с 0).

Описание:

Генерирует событие обработки нажатия навигационной ссылки для поля надписи.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
