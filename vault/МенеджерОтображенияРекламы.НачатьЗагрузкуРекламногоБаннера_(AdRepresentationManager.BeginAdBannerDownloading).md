МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуРекламногоБаннера (BeginAdBannerDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуРекламногоБаннера(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Ссылка на процедуру, которая будет вызвана после завершения загрузки рекламы из сети. Процедура должна иметь следующие параметры:

- Статус (Status) – тип [СтатусРекламы](СтатусРекламы_(AdStatus).md). Показывает статус рекламного блока по результатам загрузки.
- Идентификатор (ID) – тип [Строка](lang__def_String.md). Рекламный идентификатор для которого закончилась загрузка.
- <ДополнительныеПараметры> – Значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Загружает из сети рекламный баннер. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [УстановитьОтображениеРекламногоБаннера](МенеджерОтображенияРекламы.УстановитьОтображениеРекламногоБаннера_(AdRepresentationManager.SetAdBannerRepresentation).md) рекламный баннер будет отображен без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы_(AdRepresentationManager).md), метод [ЗагрузитьРекламныйБаннерАсинх](МенеджерОтображенияРекламы.ЗагрузитьРекламныйБаннерАсинх_(AdRepresentationManager.DownloadAdBannerAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
