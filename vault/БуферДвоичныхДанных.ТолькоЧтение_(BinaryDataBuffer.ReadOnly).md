БуферДвоичныхДанных (BinaryDataBuffer)

ТолькоЧтение (ReadOnly)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Значение [Истина](lang__def_BooleanTrue.md) указывает, что данный буфер предназначен только для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
