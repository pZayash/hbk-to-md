Описание:

Поле выбора предназначено для выбора одного элемента из списка. Список элементов для выбора содержится в свойстве [СписокВыбора](ПолеВыбора.СписокВыбора.md) и заполняется перед использованием поля выбора. Кроме кнопки выбора из списка поле выбора также может иметь дополнительные кнопки управления.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
