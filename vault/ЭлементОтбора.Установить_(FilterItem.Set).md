# ЭлементОтбора.Установить

ЭлементОтбора (FilterItem)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Значение>, <Использование>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение отбора.

<Использование> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак использования отбора по значению.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает значение отбора и флаг использования. При выполнении данного метода вид сравнения устанавливается в значение [Равно](ВидСравнения.Равно_(ComparisonType.Equal).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод имеет смысл применять, если необходимо установить и значение и использование отбора.  
Важно! При установке значения первого параметра в [Неопределено](lang__def_Undefined.md) установленный ранее отбор не изменяется.

Пример:

|  |
| --- |
| СписокКурсыВалют.Отбор.Валюта.Установить(Ссылка); |

Использование в версии:

Доступен, начиная с версии 8.0.
