КолонкиСписка (ListColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки.

Возвращаемое значение:

Тип: [КолонкаСписка](КолонкаСписка_(ListColumn).md), [Неопределено](lang__def_Undefined.md).   
Если колонка не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск указанной колонки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
