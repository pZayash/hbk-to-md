ОписаниеЭлементаСпискаВыбораНавигационнойСсылки (URLChoiceListItemDescription)

Представление (Presentation)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Представление ссылки, которое в форме получения ссылки будет использовано в списке выбора ссылки.   
Если не задано, в качестве представления используется представление значения свойства [Ключ](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.Ключ_(URLChoiceListItemDescription.Key).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
