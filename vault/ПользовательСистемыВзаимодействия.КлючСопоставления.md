ПользовательСистемыВзаимодействия (CollaborationSystemUser)

КлючСопоставления (MatchingKey)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Устанавливает ключ, по которому сервер взаимодействия выполняет сопоставление пользователей объединенных информационных баз.  
Изменение свойства возможно только в привилегированном режиме или при наличии у текущего пользователя права "Администрирование данных".

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
