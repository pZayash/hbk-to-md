ИспользованиеНечисловыхЗначенийДиаграммы (NonnumericChartValueUse)

Пропускать (Skip)

Доступен, начиная с версии 8.3.13.

Описание:

Значения, которые не приводятся к числу, пропускаются при отрисовке.

Использование в версии:

Доступен, начиная с версии 8.3.13.
