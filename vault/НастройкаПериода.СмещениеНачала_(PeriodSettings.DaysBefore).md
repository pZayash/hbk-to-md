НастройкаПериода (PeriodSettings)

СмещениеНачала (DaysBefore)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Количество дней до рабочей даты для начала интервала.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если свойство [ВариантНачала](НастройкаПериода.ВариантНачала_(PeriodSettings.BeginningVariant).md) имеет значение [Смещение](ВариантГраницыИнтервала.Смещение_(IntervalBoundVariant.BeforeAfter).md).

Использование в версии:

Доступен, начиная с версии 8.0.
