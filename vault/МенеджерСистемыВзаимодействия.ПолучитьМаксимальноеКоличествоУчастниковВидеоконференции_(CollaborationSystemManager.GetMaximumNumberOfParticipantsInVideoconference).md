МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьМаксимальноеКоличествоУчастниковВидеоконференции (GetMaximumNumberOfParticipantsInVideoconference)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьМаксимальноеКоличествоУчастниковВидеоконференции()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
0 - если использование видеоконференций недоступно.

Описание:

Возвращает максимально возможное количество участников видеоконференции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
