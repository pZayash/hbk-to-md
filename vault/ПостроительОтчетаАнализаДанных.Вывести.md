ПостроительОтчетаАнализаДанных (DataAnalysisReportBuilder)

Вывести (Put)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Вывод свойства результат

Синтаксис:

Вывести(<Документ>)

Параметры:

<Документ> (обязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент.md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента.md).   
Табличный документ, в который будет осуществлен вывод.

Описание варианта метода:

Выводит в табличный документ содержимое свойства [Результат](ПостроительОтчетаАнализаДанных.Результат.md).

Вариант синтаксиса: Вывод произвольного результата

Синтаксис:

Вывести(<Результат>, <Документ>)

Параметры:

<Результат> (обязательный)

Тип: [РезультатАнализаДанныхДеревоРешений](РезультатАнализаДанныхДеревоРешений.md), [РезультатАнализаДанныхКластеризация](РезультатАнализаДанныхКластеризация.md), [РезультатАнализаДанныхПоискАссоциаций](РезультатАнализаДанныхПоискАссоциаций.md), [РезультатАнализаДанныхПоискПоследовательностей](РезультатАнализаДанныхПоискПоследовательностей.md), [РезультатАнализаДанныхОбщаяСтатистика](РезультатАнализаДанныхОбщаяСтатистика.md).   
Выводимый результат.

<Документ> (обязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент.md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента.md).   
Табличный документ, в который будет осуществлен вывод.

Описание варианта метода:

Осуществляет вывод произвольного результата.

Описание:

Выводит результат анализа в табличный документ.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
