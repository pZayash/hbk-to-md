КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяИнтерфейса>)

Параметры:

<ИмяИнтерфейса> (обязательный)

Тип: Строка.   
Наименование интерфейса, как он назван в конфигураторе.

Возвращаемое значение:

Тип: [ЭлементУправленияИнтерфейсом](ЭлементУправленияИнтерфейсом.md), Неопределено.   
Если интерфейс отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента управления интерфейсом по его имени.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
