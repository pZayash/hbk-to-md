Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеСостоянияПросмотра](ПоложениеСостоянияПросмотра.md).   
Задает расположение автоматически сформированного элемента для отображения состояния просмотра таблицы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
