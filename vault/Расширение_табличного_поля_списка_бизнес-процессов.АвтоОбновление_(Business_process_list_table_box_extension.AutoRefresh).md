# Расширение табличного поля списка бизнес-процессов.АвтоОбновление

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [Расширение табличного поля списка бизнес-процессов](Расширение_табличного_поля_списка_бизнес-процессов_(Business_process_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_бизнес-процессов__Свойства.md)

Расширение табличного поля списка бизнес-процессов (Business process list table box extension)

АвтоОбновление (AutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического обновления табличного поля. Если признак установлен как [Истина](lang__def_BooleanTrue.md), то автоматически будет вызываться обновление списка. Время между обновлениями списка задается свойством [ПериодАвтоОбновления](Расширение_табличного_поля_списка_бизнес-процессов.ПериодАвтоОбновления_(Business_process_list_table_box_extension.AutoRefreshPeriod).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
