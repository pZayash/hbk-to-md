ГрафическаяСхема (GraphicalSchema)

Показать (Show)

Доступен, начиная с версии 8.1.

Синтаксис:

Показать(<Заголовок>, <ИмяФайла>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит заголовок окна.

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который может быть сохранен документ.

Описание:

Открывает окно для показа и редактирования графической схемы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если графическая схема является реквизитом управляемой формы, то вызов метода приводит к возникновению исключения.

Использование в версии:

Доступен, начиная с версии 8.1.
