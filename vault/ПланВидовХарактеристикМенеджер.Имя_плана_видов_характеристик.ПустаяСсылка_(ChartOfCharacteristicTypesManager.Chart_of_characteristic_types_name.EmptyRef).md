Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
