ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаПолученияНавигационнойСсылки (URLGetProcessing)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ОбработкаПолученияНавигационнойСсылки(<Ключ>, <Представление>, <НавигационнаяСсылка>, <СтандартнаяОбработка>)

Параметры:

<Ключ>

Тип: Произвольный.   
В этом параметре передается значение свойства [Ключ](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.Ключ_(URLChoiceListItemDescription.Key).md) выбранного элемента списка ссылок, полученного из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок_(ClientApplicationForm.URLListGetProcessing).md). Если СписокНавигационыхСсылок, полученный из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок_(ClientApplicationForm.URLListGetProcessing).md), пуст, то <Ключ> содержит значение [Неопределено](lang__def_Undefined.md).

<Представление>

Тип: [Строка](lang__def_String.md).   
Представление ссылки, полученное из обработчика [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок_(ClientApplicationForm.URLListGetProcessing).md) или введенное пользователем. Представление может быть необходимо для формирования навигационной ссылки одним из методов глобального контекста или одного израсширений формы.

<НавигационнаяСсылка>

Тип: [Строка](lang__def_String.md).   
Параметр, в который требуется поместить ссылку.  
Полученная ссылка будет обработана (недопустимые символы будут закодированы) и в дальнейшем может быть использована для копирования в буфер или для помещения в избранное.

<СтандартнаяОбработка>

Тип: [Строка](lang__def_String.md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события.  
<СтандартнаяОбработка> имеет значение [Истина](lang__def_BooleanTrue.md) в одном из случаев:

- если в параметре <Ключ> один из стандартных вариантов,
- в параметре <Ключ> - [Неопределено](lang__def_Undefined.md) и <АвтоНавигационнаяСсылка> установлено в значение [Ложь](lang__def_BooleanFalse.md).

<СтандартнаяОбработка> имеет значение [Ложь](lang__def_BooleanFalse.md) в одном из случаев:

- если в параметре <Ключ> один из вариантов, добавленных в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок_(ClientApplicationForm.URLListGetProcessing).md);
- в параметре Ключ [Неопределено](lang__def_Undefined.md) и <АвтоНавигационнаяСсылка> имеет значение [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при выборе в диалоге получения навигационной ссылки или добавления в избранное одного из вариантов ссылки, как добавленного в обработчике [ОбработкаПолученияСпискаНавигационныхСсылок](ФормаКлиентскогоПриложения.ОбработкаПолученияСпискаНавигационныхСсылок_(ClientApplicationForm.URLListGetProcessing).md), так и стандартного.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
