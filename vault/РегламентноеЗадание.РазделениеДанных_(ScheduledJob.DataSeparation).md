РегламентноеЗадание (ScheduledJob)

РазделениеДанных (DataSeparation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Содержит значения разделителей. Имя элемента структуры содержит имя общего реквизита; значение – значение общего реквизита.  
Элементами структуры являются значения всех разделителей регламентного задания с типом [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно_(CommonAttributeSeparatedDataUse.IndependentlyAndSimultaneously).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
