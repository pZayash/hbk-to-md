НастройкиОбработкиОшибок (ErrorProcessingSettings)

ВключатьИмяПользователяВОтчет (IncludeUserNameInReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ВключатьИмяПользователяВОтчетНаКлиенте](НастройкиОбработкиОшибок.ВключатьИмяПользователяВОтчетНаКлиенте_(ErrorProcessingSettings.IncludeUserNameInReportOnClient).md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость добавлять в отчет об ошибке имя пользователя.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

См. также:

[НастройкиОбработкиОшибок](НастройкиОбработкиОшибок_(ErrorProcessingSettings).md), свойство [ВключатьИмяПользователяВОтчетНаСервере](НастройкиОбработкиОшибок.ВключатьИмяПользователяВОтчетНаСервере_(ErrorProcessingSettings.IncludeUserNameInReportOnServer).md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
