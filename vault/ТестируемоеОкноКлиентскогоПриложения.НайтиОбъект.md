ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

НайтиОбъект (FindObject)

Доступен, начиная с версии 8.3.2.

Синтаксис:

НайтиОбъект(<Тип>, <Заголовок>, <Имя>, <Таймаут>)

Параметры:

<Тип> (необязательный)

Тип: Тип.   
Тип объекта, который нужно найти.  
Значение по умолчанию: Неопределено.

<Заголовок> (необязательный)

Тип: Строка.   
Заголовок объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Имя> (необязательный)

Тип: Строка.   
Имя объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Таймаут> (необязательный)

Тип: Число.   
Время ожидания в секундах.  
Значение по умолчанию: 0.

Возвращаемое значение:

Первый найденный объект.

Описание:

Выполняет поиск среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.2.
