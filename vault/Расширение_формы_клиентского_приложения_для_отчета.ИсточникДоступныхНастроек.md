Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ИсточникДоступныхНастроек (AvailableSettingsSource)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: [ИсточникДоступныхНастроекКомпоновкиДанных](ИсточникДоступныхНастроекКомпоновкиДанных.md).   
Предоставляет компоновщику настроек отчета необходимую информацию: наборы доступных полей для различных использований, наборы объектов доступных для вложенной настройки, доступные параметры данных.

Использование в версии:

Доступен, начиная с версии 8.3.3.
