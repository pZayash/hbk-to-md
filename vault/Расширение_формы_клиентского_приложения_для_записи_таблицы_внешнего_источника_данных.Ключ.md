Расширение формы клиентского приложения для записи таблицы внешнего источника данных

Ключ (Key)

Доступен, начиная с версии 8.3.5.

Описание:

Тип: [ВнешнийИсточникДанныхТаблицаКлючЗаписи.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   
Тип значения - набор ключевых полей, однозначно идентифицирующих запись, редактируемых в форме.

Примечание:

В случае отсутствия параметра, а также когда параметр содержит пустое значение, создается новая запись с использованием остальных параметров.

Использование в версии:

Доступен, начиная с версии 8.3.5.
