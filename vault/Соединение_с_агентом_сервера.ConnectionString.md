Соединение с агентом сервера (IServerAgentConnection)

ConnectionString (ConnectionString)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит строку, согласно которой было установлено соединение с агентом сервера. Эта строка являлась значением параметра serverName метода [ConnectAgent](Менеджер_COM-соединений.ConnectAgent.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
