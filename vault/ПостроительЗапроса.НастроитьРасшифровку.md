ПостроительЗапроса (QueryBuilder)

НастроитьРасшифровку (InitDetails)

Доступен, начиная с версии 8.0.

Синтаксис:

НастроитьРасшифровку(<Построитель>, <Расшифровка>)

Параметры:

<Построитель> (обязательный)

Тип: [ПостроительЗапроса](ПостроительЗапроса.md).   
Настраиваемый построитель запроса.

<Расшифровка> (обязательный)

Тип: [Структура](Структура.md).   
Структура с расшифровкой. Ключ - имя поля, значение - значение поля.

Описание:

Настраивает построитель для получения расшифровки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
