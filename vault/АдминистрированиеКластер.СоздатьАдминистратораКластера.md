АдминистрированиеКластер (AdministrationCluster)

СоздатьАдминистратораКластера (CreateClusterAdministrator)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьАдминистратораКластера()

Возвращаемое значение:

Тип: [АдминистрированиеАдминистратор](АдминистрированиеАдминистратор.md).   

Описание:

Создает администратора кластера с пустыми значениями свойств. Этот объект может быть использован для регистрации нового администратора после установки правильных значений его свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
