ЗаписьPDF (PDFWriter)

На основании потока

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЗаписьPDF(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток_(Stream).md).   
Поток, в который производится запись данных.

Описание:

Объект создается для записи в заданный поток.  
После завершения работы с объектом, до закрытия потока, переданного в конструктор, объект необходимо закрыть с помощью метода [Закрыть](ЗаписьPDF.Закрыть_(PDFWriter.Close).md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
