ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

НайтиПоНаименованию (FindByDescription)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоНаименованию(<Наименование>, <ТочноеСоответствие>)

Параметры:

<Наименование> (обязательный)

Тип: Строка.   
Строка, содержащая искомое наименование задачи.

<ТочноеСоответствие> (необязательный)

Тип: Булево.   
Определяет режим поиска по полному соответствию.  
Поиск будет успешным, если строка поиска: в случае значения параметра Ложь - будет соответствовать левой части наименования; в случае значения параметра Истина - будет полностью совпадать с наименованием (за исключением "хвостовых" пробелов в наименовании).  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>, Неопределено.   
Ссылка на найденную задачу. Если не существует ни одной задачи с требуемым наименованием, то будет возвращена пустая ссылка.  
Если наименование не задано (длина = 0) и поиск выполняется по полному соответствию, то будет возвращено Неопределено.

Описание:

Осуществляет поиск задачи по ее наименованию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если существует несколько задач с указанным наименованием, то будет найдена только одна из них.

Использование в версии:

Доступен, начиная с версии 8.0.
