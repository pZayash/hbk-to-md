ЭлементыШкалыВремени (TimeScaleItems)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ЭлементШкалыВремени](ЭлементШкалыВремени_(TimeScaleItem).md).   

Описание:

Вставляет элемент с указанным индексом в шкалу.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
