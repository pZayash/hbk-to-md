ПолеКалендаря (CalendarBox)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешает начало операции перетаскивания в поле календаря - источнике данных.

Доступность:

Толстый клиент.

Примечание:

Истина - перетаскивание разрешено; Ложь - запрещено.

Использование в версии:

Доступен, начиная с версии 8.0.
