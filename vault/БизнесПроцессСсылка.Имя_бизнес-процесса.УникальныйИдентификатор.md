БизнесПроцессСсылка.<Имя бизнес-процесса> (BusinessProcessRef.<Business process name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса.md), метод [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку.md)

Доступен, начиная с версии 8.0.
