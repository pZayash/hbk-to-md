Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

НачатьРедактированиеТекущейОбласти (BeginEditCurrentArea)

Доступен, начиная с версии 8.3.1.

Синтаксис:

НачатьРедактированиеТекущейОбласти()

Описание:

Начать редактирование текущей области табличного документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
