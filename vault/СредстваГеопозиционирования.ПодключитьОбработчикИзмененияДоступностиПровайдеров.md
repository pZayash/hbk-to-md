СредстваГеопозиционирования (LocationTools)

ПодключитьОбработчикИзмененияДоступностиПровайдеров (AttachProvidersAvailabilityChangeHandler)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПодключитьОбработчикИзмененияДоступностиПровайдеров(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Строка.   
Имя процедуры, подключаемой в качестве обработчика изменения доступности провайдеров.  
Процедура должна иметь следующие параметры.

- ИмяПровайдера - имя провайдера, чей статус был изменен. Тип Строка.
- Доступен - новое состояние провайдера. Тип Булево.
- <ДополнительныйПараметр> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md), если для установки обработчика используется описание оповещения.

Описание:

Выполняет подключение обработчика изменения статусов провайдеров геопозиционирования присутствующих в системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
