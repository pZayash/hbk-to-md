# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПриЗавершении

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриЗавершении (OnComplete)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗавершении(<ТочкаМаршрута>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса вида [Завершение](ВидТочкиМаршрутаБизнесПроцесса.Завершение_(BusinessProcessRoutePointType.End).md), на которой происходит завершение бизнес-процесса.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от завершения бизнес-процесса. Если необходимые условия для завершения не выполнены (например, когда не выполнены все необходимые действия, предусмотренные данным бизнес-процессом), то в теле процедуры-обработчика следует установить данному параметру значение [Истина](lang__def_BooleanTrue.md). В этом случае бизнес-процесса не будет завершен.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается при переходе бизнес-процесса на указанную в параметре точку завершения до установки ему признака "Завершен". Обычно вызывается вследствие вызова метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу_(TaskObject.Task_name.ExecuteTask).md) у задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точки вида [Завершение](ВидТочкиМаршрутаБизнесПроцесса.Завершение_(BusinessProcessRoutePointType.End).md).  
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
