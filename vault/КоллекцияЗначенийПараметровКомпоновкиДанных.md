# КоллекцияЗначенийПараметровКомпоновкиДанных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Система компоновки данных](objects__catalog63__catalog1064.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных.md), [ЗначениеПараметраНастроекКомпоновкиДанных](ЗначениеПараметраНастроекКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметров.  
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
