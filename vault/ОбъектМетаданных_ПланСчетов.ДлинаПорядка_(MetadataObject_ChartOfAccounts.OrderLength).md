# ОбъектМетаданных: ПланСчетов.ДлинаПорядка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов_(MetadataObject_ChartOfAccounts).md) › [Свойства](ОбъектМетаданных_ПланСчетов__Свойства.md)

ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ДлинаПорядка (OrderLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Максимальная длина поля [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок_(ChartOfAccountsObject.Chart_of_accounts_name.Order).md) плана счетов. Если значение этого свойства равно 0, то по умолчанию список счетов упорядочивается по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код_(ChartOfAccountsObject.Chart_of_accounts_name.Code).md). В противном случае - по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок_(ChartOfAccountsObject.Chart_of_accounts_name.Order).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
