Расширение табличного поля списка справочника (Catalog list table box extension)

ИерархическийПросмотр (HierarchicalView)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Предоставляет доступ к управлению способу просмотра списка справочника. Если свойство установлено в [Истина](lang__def_BooleanTrue.md), то будет выполняться иерархический просмотр (по родителям), иначе будут выводиться сразу все элементы справочника.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
