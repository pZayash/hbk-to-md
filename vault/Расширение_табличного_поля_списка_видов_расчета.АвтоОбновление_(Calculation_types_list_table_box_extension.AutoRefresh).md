# Расширение табличного поля списка видов расчета.АвтоОбновление

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [Расширение табличного поля списка видов расчета](Расширение_табличного_поля_списка_видов_расчета_(Calculation_types_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_видов_расчета__Свойства.md)

Расширение табличного поля списка видов расчета (Calculation types list table box extension)

АвтоОбновление (AutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического обновления табличного поля. Если признак установлен как [Истина](lang__def_BooleanTrue.md), то автоматически будет вызываться автоматическое обновление списка. Время между обновлениями списка задается свойством [ПериодАвтоОбновления](Расширение_табличного_поля_списка_видов_расчета.ПериодАвтоОбновления_(Calculation_types_list_table_box_extension.AutoRefreshPeriod).md).

Доступность:

Толстый клиент.
