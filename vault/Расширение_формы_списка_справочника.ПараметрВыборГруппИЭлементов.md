Расширение формы списка справочника (Catalog list form extension)

ПараметрВыборГруппИЭлементов (ChoiceFoldersAndItemsParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов.md).   
Предоставляет возможность установить при открытии или повторном открытии формы списка справочника, что должно быть выбрано в форме: любой элемент, только группы или только элементы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
