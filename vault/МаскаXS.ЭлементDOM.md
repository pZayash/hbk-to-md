МаскаXS (XSWildcard)

ЭлементDOM (DOMElement)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ЭлементDOM](ЭлементDOM.md).   
Элемент DOM, который соответствует данному компоненту схемы XML.  
Синхронизация значений свойств компонента схемы XML и структуры элемента DOM обеспечивается вызовом метода [МаскаXS.ОбновитьЭлементDOM](МаскаXS.ОбновитьЭлементDOM.md).  
Перед первым вызовом данного метода значение данного свойства может быть Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
