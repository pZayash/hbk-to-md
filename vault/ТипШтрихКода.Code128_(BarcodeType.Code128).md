ТипШтрихКода (BarcodeType)

Code128 (Code128)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Code 128. Позволяет кодировать не только цифры, но и буквы латинского алфавита, а также специальные символы.

Использование в версии:

Доступен, начиная с версии 8.3.25.
