ФормаКлиентскогоПриложения (ClientApplicationForm)

Окно (Window)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md).   
Содержит окно клиентского приложения, в котором расположена форма.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме совместимости [Версия8\_3\_19](РежимСовместимости.Версия8_3_19.md) и ниже для формы на начальной странице в тонком и веб-клиенте содержит Неопределено.  
В режиме совместимости выше [Версия8\_3\_19](РежимСовместимости.Версия8_3_19.md) для формы на начальной странице в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси.md) содержит объект типа {ссылка:Объекты; 1753; ОкноКлиентскогоПриложения}, у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница.md) имеет значение Истина. И свойство [Содержимое](ОкноКлиентскогоПриложения.Содержимое.md) содержит формы начальной страницы.  
Если активна начальная страница, то и данное свойство, и метод [АктивноеОкно](Глобальный_контекст.АктивноеОкно.md) для формы на начальной странице возвращают один и тот же объект.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
