# Соединение с агентом сервера.GetConnections

Соединение с агентом сервера (IServerAgentConnection)

GetConnections (GetConnections)

Доступен, начиная с версии 8.1.

Синтаксис:

GetConnections(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должен быть получен массив описаний соединений. Может быть элементом полученного методом [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md) массива кластеров, зарегистрированных в текущем центральном сервере. Еще кластер может быть создан методом [CreateClusterInfo](Соединение_с_агентом_сервера.CreateClusterInfo_(IServerAgentConnection.CreateClusterInfo).md) с последующим заполнением его свойств.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив описаний соединений кластера. Каждое описание соединения представлено объектом с интерфейсом [Описание соединения](Описание_соединения_(IConnectionShort).md).

Описание:

Получает массив описаний соединений кластера. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера. Про каждое соединение выдается только такая информация, которая не требует аутентификации пользователей информационных баз. Получение полной информации о соединениях и принудительный разрыв соединений возможны только при помощи интерфейса [Соединение с рабочим процессом](Соединение_с_рабочим_процессом_(IWorkingProcessConnection).md) после аутентификации пользователей информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
