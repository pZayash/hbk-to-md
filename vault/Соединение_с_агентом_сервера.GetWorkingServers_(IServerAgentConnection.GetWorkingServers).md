Соединение с агентом сервера (IServerAgentConnection)

GetWorkingServers (GetWorkingServers)

Доступен, начиная с версии 8.1.

Синтаксис:

GetWorkingServers(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должен быть получен список рабочих серверов.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив рабочих серверов, зарегистрированных в кластере. Каждый сервер является объектом с интерфейсом [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).

Описание:

Получает массив рабочих серверов, зарегистрированных в кластере.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
