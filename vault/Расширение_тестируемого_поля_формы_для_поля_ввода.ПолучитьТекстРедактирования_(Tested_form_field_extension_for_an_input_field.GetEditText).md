Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ПолучитьТекстРедактирования (GetEditText)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьТекстРедактирования()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает текущий текст из поля ввода.  
Если метод вызывается для поля, являющегося колонкой таблицы, которая не находится в режиме редактирования, будет сгенерирована ошибка.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
