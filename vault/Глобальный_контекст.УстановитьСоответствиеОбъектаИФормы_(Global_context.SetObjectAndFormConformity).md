Прикладной объект, преобразованный из универсального объекта в значение, для которого необходимо установить соответствие с именем реквизита формы.

<ИдентификаторФормыИИмяРеквизита> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит идентификатор формы и имя реквизита элемента управления, для которых выполняется установка соответствия.   
Структура, передаваемая в параметр, может содержать два элемента с именами ключей:  
1. "ИдентификаторФормы" ("FormID") – уникальный идентификатор формы (тип [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md));  
2. "ИмяРеквизита" ("AttributeName") – имя реквизита в форме (тип [Строка](lang__def_String.md)). Этот элемент может не указываться.

Описание:

Устанавливает соответствие формы клиентского приложения и объектов вида [СообщениеПользователю](СообщениеПользователю_(UserMessage).md) для последующей правильной привязки сообщений к элементам управления формы. Для того, чтобы сообщение использовало данные соответствия объекта и формы, у него необходимо заполнить свойство [ИдентификаторНазначения](СообщениеПользователю.ИдентификаторНазначения_(UserMessage.TargetID).md). Соответствие может быть получено методом [ПолучитьСоответствиеОбъектаИФормы](Глобальный_контекст.ПолучитьСоответствиеОбъектаИФормы_(Global_context.GetObjectAndFormConformity).md). Получать соответствие можно до тех пор, пока существует объект, для которого оно зарегистрированно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
