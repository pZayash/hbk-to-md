Доступен, начиная с версии 8.0.

Описание:

Тип анализа данных - поиск последовательностей.  
Параметры:

- МинимальныйПроцентСлучаев (MinSupport) - минимальное число последовательностей, в которых должен наблюдаться шаблон последовательности (Число);
- ПоискПоИерархии (FindInHierarchy) - необходимо ли осуществлять поиск по иерархии (Булево);
- МинимальныйИнтервал (MinInterval) - признак того, что установлен минимальный интервал между наблюдаемыми событиями (Булево);
- ЕдиницаМинимальногоИнтервала (MinIntervalUnit) - единица минимального интервала (ТипЕдиницыИнтервалаВремениАнализаДанных);
- КратностьМинимальногоИнтервала (MinIntervalRepetition) - кратность минимального интервала (Число);
- МаксимальныйИнтервал (MaxInterval) - признак того, что установлен максимальный интервал между наблюдаемыми событиями (Булево);
- ЕдиницаМаксимальногоИнтервала (MaxIntervalUnit) - единица максимального интервала (ТипЕдиницыИнтервалаВремениАнализаДанных);
- КратностьМаксимальногоИнтервала (MaxIntervalRepetition) - кратность максимального интервала (Число);
- ИнтервалЭквивалентностиВремени (TimeSliceWindow) - признак того, что установлен интервал эквивалентности времени между наблюдаемыми событиями (Булево);
- ЕдиницаИнтервалаЭквивалентностиВремени (TimeSliceWindowUnit) - единица интервала эквивалентности времени (ТипЕдиницыИнтервалаВремениАнализаДанных);
- КратностьИнтервалаЭквивалентностиВремени (TimeSliceWindowRepetition) - кратность интервала эквивалентности времени (Число).
- МинимальнаяДлина (MinLength) - минимальная длина искомых последовательностей (Число);
- Порядок (Order) - порядок, в соответствии с которым выводить последовательности в результат.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
