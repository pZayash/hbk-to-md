Описание:

Сетевые параметры подключения к WebSocket-серверу.  
Объект данного типа может быть передан в метод [ОткрытьСоединение](МенеджерWebSocketКлиентСоединений.ОткрытьСоединение_(WebSocketClientConnectionsManager.OpenConnection).md), и станет доступен через свойства [Параметры](WebSocketКлиентСоединение.Параметры_(WebSocketClientConnection.Parameters).md) (в том числе, и в обработчиках событий WebSocket).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
