WebSocketКлиентСоединение (WebSocketClientConnection)

ИмяПользователяИнформационнойБазы (InfoBaseUserName)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя пользователя, от имени которого которым будет запущен сеанс, в котором будут вызваны обработчики событий. Если имя пользователя не задано, сеанс будет запущен от имени пользователя по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
