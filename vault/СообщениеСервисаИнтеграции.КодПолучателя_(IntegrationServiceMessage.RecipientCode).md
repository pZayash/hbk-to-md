СообщениеСервисаИнтеграции (IntegrationServiceMessage)

КодПолучателя (RecipientCode)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Код участника интеграции, которому направляется сообщение. Предполагается использовать при организации взаимодействия "запрос-ответ".  
Свойство предоставляет удобный способ работы с параметром сообщения Participant-Receiver-Codes. Значение, установленное в свойстве, доступно также через указанный параметр.   
В сообщении-запросе в качестве кода отправителя указывается код участника интеграции, которому будет направлен ответ.  
В сообщении-ответе свойство <КодПолучателя> устанавливается равным свойству [КодОтправителя](СообщениеСервисаИнтеграции.КодОтправителя_(IntegrationServiceMessage.SenderCode).md) запроса.  
  
Можно устанавливать только при отправке. При попытке изменить в полученном сообщении генерируется исключение.  
Принимает коды справочника участников, разделенные символом запятая (,). Если получатели явно не указаны, сообщение будет направлено всем возможным получателям.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
