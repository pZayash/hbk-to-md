ЭлементВводаHTML (HTMLInputElement)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство управляет доступностью поля элемента ввода: если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), пользователь не может изменить значение в поле.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
