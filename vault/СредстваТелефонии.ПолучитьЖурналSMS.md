СредстваТелефонии (TelephonyTools)

ПолучитьЖурналSMS (GetSMSLog)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПолучитьЖурналSMS()

Возвращаемое значение:

Тип: [ЖурналSMS](ЖурналSMS.md), Неопределено.   
Если журнал сообщений не доступен, возвращает Неопределено.

Описание:

Получает журнал SMS-сообщений.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЖурналSMS](СредстваТелефонии.ПоддерживаетсяЖурналSMS.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии.md), метод [ПоддерживаетсяЖурналSMS](СредстваТелефонии.ПоддерживаетсяЖурналSMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.
