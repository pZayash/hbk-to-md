МакетОбластиКомпоновкиДанных (DataCompositionAreaTemplate)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Расположение элемента в коллекции.

<Тип> (необязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных.md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
