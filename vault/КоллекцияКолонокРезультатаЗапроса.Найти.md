КоллекцияКолонокРезультатаЗапроса (QueryResultColumnsCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<НаименованиеКолонки>)

Параметры:

<НаименованиеКолонки> (обязательный)

Тип: Строка.   
Имя колонки результата запроса.

Возвращаемое значение:

Тип: [КолонкаРезультатаЗапроса](КолонкаРезультатаЗапроса.md), Неопределено.   
Если указанная колонка отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск колонки результата запроса по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если РезультатЗапроса.Колонки.Найти("Цена") <> Неопределено Тогда     ЦеныВЗапросе = Истина; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
