ОбъявлениеАтрибутаXS (XSAttributeDeclaration)

ЭтоСсылка (IsReference)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, является ли данное объявление ссылкой.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - объявление является ссылкой.

Использование в версии:

Доступен, начиная с версии 8.1.
