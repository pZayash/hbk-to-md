ПоляПостроителяЗапроса (QueryBuilderFields)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: [ПолеПостроителяЗапроса](ПолеПостроителяЗапроса_(QueryBuilderField).md).   

Описание:

Вставляет поле в указанную позицию коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
