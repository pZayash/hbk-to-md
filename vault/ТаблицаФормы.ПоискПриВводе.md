ТаблицаФормы (FormTable)

ПоискПриВводе (SearchOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПоискВТаблицеПриВводе](ПоискВТаблицеПриВводе.md).   
Задает режим выполнения поиска при вводе символов в таблице, связанной с данными типа [ДинамическийСписок](ДинамическийСписок.md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
