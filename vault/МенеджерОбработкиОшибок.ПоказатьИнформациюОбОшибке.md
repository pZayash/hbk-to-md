МенеджерОбработкиОшибок (ErrorProcessingManager)

ПоказатьИнформациюОбОшибке (ShowErrorInfo)

Доступен, начиная с версии 8.3.17.

Вариант синтаксиса: С сообщением об ошибке

Синтаксис:

ПоказатьИнформациюОбОшибке(<ИнформацияОбОшибке>, <СообщениеОбОшибке>, <ДополнительнаяИнформация>, <ФормироватьОтчетОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Информация об ошибке, которую требуется показать пользователю.

<СообщениеОбОшибке> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Строка отобразится в качестве сообщения об ошибке.

<ДополнительнаяИнформация> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Если параметр заполнен, то дополнительная информация отобразится на форме ошибки.

<ФормироватьОтчетОбОшибке> (необязательный)

Тип: Булево.   
Если параметр не передан, то необходимость формировать отчет об ошибке будет определена автоматически.

Описание варианта метода:

Открывает диалог с информацией об ошибке. Если метод был вызван из обработчика [ОбработкаОтображенияОшибки](Глобальный_контекст.ОбработкаОтображенияОшибки.md), в котором параметр <ТребуетсяЗавершениеСеанса> имеет значение Истина, то перезапуск произойдет после закрытия этого окна.

Вариант синтаксиса: С заданным уровнем подробности

Синтаксис:

ПоказатьИнформациюОбОшибке(<ИнформацияОбОшибке>, <ВариантОтображенияСообщения>, <ДополнительнаяИнформация>, <ФормироватьОтчетОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Информация об ошибке, которую требуется показать пользователю.

<ВариантОтображенияСообщения> (необязательный)

Тип: [ВариантОтображенияСообщенияОбОшибке](ВариантОтображенияСообщенияОбОшибке.md).   
Текст ошибки сформируется с заданным уровнем подробности.  
Значение по умолчанию: [Авто](ВариантОтображенияСообщенияОбОшибке.Авто.md).

<ДополнительнаяИнформация> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Если параметр заполнен, то дополнительная информация отобразится на форме ошибки.

<ФормироватьОтчетОбОшибке> (необязательный)

Тип: Булево.   
Если параметр не передан, то необходимость формировать отчет об ошибке будет определена автоматически.

Описание варианта метода:

Открывает диалог с информацией об ошибке. Если метод был вызван из обработчика [ОбработкаОтображенияОшибки](Глобальный_контекст.ОбработкаОтображенияОшибки.md), в котором параметр <ТребуетсяЗавершениеСеанса> имеет значение Истина, то перезапуск произойдет после закрытия этого окна.

Описание:

Предназначен для показа информации об ошибке пользователю в стандартном диалоге платформы "1С:Предприятие".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Попытка     ... Исключение     Инфо = ИнформацияОбОшибке();     ОбработкаОшибок.ПоказатьИнформациюОбОшибке(Инфо); КонецПопытки; |
