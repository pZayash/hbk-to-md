ИсточникКомандПоляПланировщика (PlannerCommandSource)

ЭлементШкалыВремени (TimeScaleItem)

Доступен, начиная с версии 8.3.23.

Описание:

Значение, которое устанавливается при нажатии на шкалу времени планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд_(Form_field_extension_for_a_planner.CommandGenerateProcessing).md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы_(PlannerCommandGenerateParameters.Items).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения_(PlannerCommandGenerateParameters.DimensionItem).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений_(PlannerCommandGenerateParameters.DimensionValues).md) содержит [Неопределено](lang__def_Undefined.md),
- [Дата](ПараметрыФормированияКомандПланировщика.Дата_(PlannerCommandGenerateParameters.Date).md) заполнено,
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandGenerateParameters.TimeScaleItem).md) заполнено,
- Команды не содержат стандартных команд,
- [КомандаПоУмолчанию](html.md) заполнено командой [ВыбратьЭлементШкалыВремени](СтандартнаяКомандаПланировщика.ВыбратьЭлементШкалыВремени_(PlannerStandardCommand.SelectTimeScaleItem).md).

Использование в версии:

Доступен, начиная с версии 8.3.23.
