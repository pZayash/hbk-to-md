Расширение тестируемого поля формы для HTML- документа (Tested form field extension for a HTML-document)

ПолучитьHTMLДокумента (GetDocumentHTML)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПолучитьHTMLДокумента(<HTML>, <Вложения>)

Параметры:

<HTML> (обязательный)

Тип: [Строка](lang__def_String.md).   
В параметр метод помещает строку, содержащую текст HTML.

<Вложения> (обязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
В параметр метод помещает структуру вложенных в HTML-документ рисунков.   
Ключ элемента структуры - название или имя файла рисунка, значение - [Картинка](Картинка_(Picture).md).

Описание:

Получает код HTML для HTML-документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
