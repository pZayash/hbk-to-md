Администратор (IRegUserInfo)

SysUserName (SysUserName)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя пользователя операционной системы, которое соответствует данному администратору кластера серверов. Формат: \\<имя домена>\<имя пользователя>. Например, \\domain\username.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
