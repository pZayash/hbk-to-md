# Расширение элементов управления, расположенных на панели.Верх

Расширение элементов управления, расположенных на панели (Panel controls extension)

Верх (Top)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит координату верхней границы элемента управления относительно верхней границы панели (в логических единицах).

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
