ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

ИндексФильтра (FilterIndex)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит индекс элемента в фильтре диалога, который будет выбран по умолчанию (нумерация с 0).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Не поддерживается в мобильном приложении для Windows.

Использование в версии:

Доступен, начиная с версии 8.3.17.
