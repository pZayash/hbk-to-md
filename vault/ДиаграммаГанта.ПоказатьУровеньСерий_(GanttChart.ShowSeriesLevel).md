ДиаграммаГанта (GanttChart)

ПоказатьУровеньСерий (ShowSeriesLevel)

Доступен, начиная с версии 8.0.

Синтаксис:

ПоказатьУровеньСерий(<Уровень>)

Параметры:

<Уровень> (необязательный)

Тип: [Число](lang__def_Number.md).   
Раскрываемый уровень.  
Значение по умолчанию: 0.

Описание:

Раскрывает указанный уровень серий.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если <Уровень>=0 или не задан, раскрывает все уровни.

Использование в версии:

Доступен, начиная с версии 8.0.
