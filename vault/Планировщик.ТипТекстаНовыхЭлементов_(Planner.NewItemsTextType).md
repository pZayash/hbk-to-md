Планировщик (Planner)

ТипТекстаНовыхЭлементов (NewItemsTextType)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [ТипТекстаНовыхЭлементовПланировщика](ТипТекстаНовыхЭлементовПланировщика_(NewPlannerItemsTextType).md).   
Содержит тип текста новых элементов планировщика. Влияет на тип параметра <Текст> события [ПередСозданием](Расширение_поля_формы_для_поля_планировщика.ПередСозданием_(Form_field_extension_for_a_planner.BeforeCreate).md), а также тип элементов планировщика, создаваемых через методы [Добавить](КоллекцияЭлементовПланировщика.Добавить_(PlannerItemCollection.Add).md) и [Вставить](КоллекцияЭлементовПланировщика.Вставить_(PlannerItemCollection.Insert).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение по умолчанию - [Строка](ТипТекстаНовыхЭлементовПланировщика.Строка_(NewPlannerItemsTextType.String).md).

Использование в версии:

Доступен, начиная с версии 8.3.18.
