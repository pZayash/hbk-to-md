ОписаниеПодписиPDF (PDFSignatureDescription)

БлокировкаДокумента (DocumentLock)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости блокировки документа после подписания. Имеет смысл только для подписи типа [Утверждающая](ТипПодписиPDF.Утверждающая_(PDFSignatureType.Approving).md).   
По умолчанию: [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
