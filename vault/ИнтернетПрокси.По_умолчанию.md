ИнтернетПрокси (InternetProxy)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ИнтернетПрокси(<ПроксиПоУмолчанию>)

Параметры:

<ПроксиПоУмолчанию> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли объект инициализирован настройками прокси по умолчанию.  
Если значения параметра равно [Истина](lang__def_BooleanTrue.md), то производится попытка инициализировать объект значениями из файла inetcfg.xml. Если файл не существует, то значения берутся из системных настроек.  
При работе в ОС Windows используются системные настройки, задаваемые через Internet Explorer.  
При работе в ОС Linux настройки берутся из следующих переменных среды:

- http\_proxy
- https\_proxy
- ftp\_proxy
- ftps\_proxy

Если значения какой-либо из эти переменных не заданы, то производится попытка взять настройки прокси для соответствующего протокола из переменной среды all\_proxy.  
Если переменные среды содержат некорректные строки, то их значение игнорируется (исключения не происходит).  
Значение по умолчанию: Истина.

Описание:

Создает объект [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md). Его инициализация зависит от значения параметра [ПроксиПоУмолчанию](html.md).

Пример:

|  |
| --- |
| ПроксиСервер = Новый ИнтернетПрокси; ПроксиСервер.Пользователь = ИмяПользователя; ПроксиСервер.Пароль       = ПарольПользователя; Попытка     HTTP = Новый HTTPСоединение(СерверИсточник,,,, ПроксиСервер); Исключение КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
