НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

АдресСервераSMTP (SMTPServerAddress)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Определяет адрес SMTP-сервера, который будет использован для отправки писем при аутентификации через электронную почту.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
