Надпись (Label)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СочетаниеКлавиш](СочетаниеКлавиш.md).   
Содержит сочетание клавиш, по нажатию которых происходит нажатие на надпись.

Доступность:

Толстый клиент.

Примечание:

Используется, если надпись является гиперссылкой (свойство Ссылка имеет значение Истина).

Использование в версии:

Доступен, начиная с версии 8.0.
