ТелоГруппировкиДиаграммыМакетаКомпоновкиДанных (DataCompositionTemplateChartGroupBody)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [МакетГруппировкиДиаграммыМакетаКомпоновкиДанных](МакетГруппировкиДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartGroupTemplate).md), [ГруппировкаДиаграммыМакетаКомпоновкиДанных](ГруппировкаДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartGroup).md), [ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных](ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartHierarchicalGroup).md).

Возвращаемое значение:

Тип: [МакетГруппировкиДиаграммыМакетаКомпоновкиДанных](МакетГруппировкиДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartGroupTemplate).md), [ГруппировкаДиаграммыМакетаКомпоновкиДанных](ГруппировкаДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartGroup).md), [ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных](ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных_(DataCompositionTemplateChartHierarchicalGroup).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. В случае, если тип нового элемента не подходит для данной коллекции, добавления не происходит, и метод возвращает [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
