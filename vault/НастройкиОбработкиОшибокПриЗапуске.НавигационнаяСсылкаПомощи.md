НастройкиОбработкиОшибокПриЗапуске (ErrorOnStartupProcessingSettings)

НавигационнаяСсылкаПомощи (HelpURL)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Ссылка на внешнюю страницу помощи. Если данной свойство не пустое, то на форме ошибки подключения к конфигурации будет отображаться кнопка "Нужна помощь?", при нажатии на которую будет осуществлен переход по указанному адресу.  
Адрес должен содержать протокол, например, "https://...".  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
