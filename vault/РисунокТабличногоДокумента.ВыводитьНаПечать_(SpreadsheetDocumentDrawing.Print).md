РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ВыводитьНаПечать (Print)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак вывода рисунка на печать.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - выводить рисунок на печать; [Ложь](lang__def_BooleanFalse.md) - не выводить.  
Не применимо для рисунков типа [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
