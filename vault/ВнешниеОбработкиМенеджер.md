Описание:

Предназначен для управления внешними обработками и позволяет получать значения типа [ВнешняяОбработка](ВнешняяОбработка.md) при помощи метода [Создать](ВнешниеОбработкиМенеджер.Создать.md). Доступ к объекту осуществляется через свойство глобального контекста [ВнешниеОбработки](Глобальный_контекст.ВнешниеОбработки.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
