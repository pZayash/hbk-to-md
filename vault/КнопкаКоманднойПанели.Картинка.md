КнопкаКоманднойПанели (CommandBarButton)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, выводимую на кнопке, при соответствующем стиле ее отображения, определяемом свойством [Отображение](КнопкаКоманднойПанели.Отображение.md).

Доступность:

Толстый клиент.

Примечание:

Рекомендуемый размер картинки 16х16 пикселей.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
