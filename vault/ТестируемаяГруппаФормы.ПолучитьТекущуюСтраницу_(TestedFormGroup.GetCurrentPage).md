ТестируемаяГруппаФормы (TestedFormGroup)

ПолучитьТекущуюСтраницу (GetCurrentPage)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьТекущуюСтраницу()

Возвращаемое значение:

Тип: [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы_(TestedFormGroup).md).   

Описание:

Для группы формы вида "Группа страниц" возвращает объект, описывающий текущую активную страницу.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
