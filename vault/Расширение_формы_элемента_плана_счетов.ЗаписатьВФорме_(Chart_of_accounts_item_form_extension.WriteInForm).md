Расширение формы элемента плана счетов (Chart of accounts item form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Процедура ВидыСубконтоПередНачаломДобавления(Элемент, Отказ, Копирование)     Если ЭтоНовый() Тогда         Ответ = Вопрос("Счет еще не записан. Записать?", РежимДиалогаВопрос.ОКОтмена);         Если Ответ = КодВозвратаДиалога.ОК Тогда             Отказ = НЕ ЗаписатьВФорме();         Иначе             Отказ = Истина;         КонецЕсли;     КонецЕсли; КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
