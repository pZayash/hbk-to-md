РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Необходимость помечать рисунок.  
[Истина](lang__def_BooleanTrue.md) - отметка выведена, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md), [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.0.
