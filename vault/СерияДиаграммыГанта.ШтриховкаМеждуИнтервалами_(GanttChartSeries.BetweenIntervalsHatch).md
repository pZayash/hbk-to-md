СерияДиаграммыГанта (GanttChartSeries)

ШтриховкаМеждуИнтервалами (BetweenIntervalsHatch)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Истина](lang__def_BooleanTrue.md), то пространство между интервалами выводятся штриховкой, а не сплошным цветом. Цвет выбирается из свойства [ЦветШтриховкиМеждуИнтервалами](СерияДиаграммыГанта.ЦветШтриховкиМеждуИнтервалами_(GanttChartSeries.BetweenIntervalsHatchColor).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
