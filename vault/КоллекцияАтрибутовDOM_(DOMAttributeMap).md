# КоллекцияАтрибутовDOM

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[АтрибутDOM](АтрибутDOM_(DOMAttribute).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются атрибуты DOM.  
Возможно обращение к атрибуту DOM посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция узлов атрибутов узла DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
