РежимСовместимости (CompatibilityMode)

Версия8\_3\_18 (Version8\_3\_18)

Доступен, начиная с версии 8.3.19.

Описание:

Усовершенствования версии 8.3.19 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.18.

Использование в версии:

Доступен, начиная с версии 8.3.19.
