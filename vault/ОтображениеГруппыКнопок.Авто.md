ОтображениеГруппыКнопок (ButtonGroupRepresentation)

Авто (Auto)

Доступен, начиная с версии 8.3.8.

Описание:

Вид группы кнопок командной панели определяется автоматически:

- если у группы кнопок есть родительская группа, используется значение свойства [Отображение](Расширение_группы_формы_для_группы_кнопок.Отображение.md) родительской группы,
- Если родительской группы нет, используется значение [Обычное](ОтображениеГруппыКнопок.Обычное.md).

Использование в версии:

Доступен, начиная с версии 8.3.8.
