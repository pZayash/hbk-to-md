Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

Имя (Name)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя элемента управления.

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.  
В режиме совместимости конфигурации [Версия8\_3\_5](РежимСовместимости.Версия8_3_5.md) и ниже свойство доступно для чтения и записи.

Использование в версии:

Доступен, начиная с версии 8.0.
