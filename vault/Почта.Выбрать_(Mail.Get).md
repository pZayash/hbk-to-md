Почта (Mail)

Выбрать (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать(<НепрочитанныеСообщения>, <Конверты>)

Параметры:

<НепрочитанныеСообщения> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - выбирать только непрочитанные сообщения.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Конверты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - выбирать только конверты сообщений (вложения не переносятся в возвращаемые сообщения).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ПочтовоеСообщение](ПочтовоеСообщение_(MailMessage).md).

Описание:

Используется для получения сообщений с сервера.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

|  |
| --- |
| Почта = Новый Почта; Почта.Подключиться(); Сообщения = Почта.Выбрать(Ложь, Ложь); Почта.Отключиться(); |

Использование в версии:

Доступен, начиная с версии 8.0.
