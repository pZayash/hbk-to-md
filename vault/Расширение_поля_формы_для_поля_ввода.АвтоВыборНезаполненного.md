Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоВыборНезаполненного (AutoChoiceIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Указывает необходимость предоставления выбора незаполненного значения в поле ввода. Если значение свойства Неопределено (в Конфигураторе выбрано значение Авто), то при автоматическом определении значения используются текущие значения свойств [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного.md) и [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
