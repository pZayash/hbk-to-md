ЭлементЯчейкаТаблицыHTML (HTMLTableCellElement)

ПолучитьАтрибут (GetAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьАтрибут(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает значение атрибута по имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение отсутствует или атрибут не обнаружен, то будет возвращено значение [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.2.
