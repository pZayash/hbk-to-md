СистемнаяИнформация (SystemInfo)

ТипПлатформы (PlatformType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ТипПлатформы](ТипПлатформы.md), Неопределено.   
Содержит тип платформы, в рамках которой осуществляется работа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме веб-клиента:

- [Windows\_x86](ТипПлатформы.Windows_x86.md)– для 32-х разрядных браузеров в среде Windows вне зависимости от разрядности операционной системы,
- [Windows\_x86\_64](ТипПлатформы.Windows_x86_64.md) – для 64-х разрядных браузеров в среде 64-х разрядной Windows,
- [Linux\_x86](ТипПлатформы.Linux_x86.md) – для 32-х разрядных браузеров в среде Linux вне зависимости от разрядности операционной системы,
- [Linux\_x86\_64](ТипПлатформы.Linux_x86_64.md) – для 64-х разрядных браузеров в среде 64-разрядной Linux,
- Неопределено - в остальных случаях.

В мобильной версии "1С:Предприятия" в режиме совместимости [Версия8\_3\_22](РежимСовместимости.Версия8_3_22.md) и ниже содержит Неопределено.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
