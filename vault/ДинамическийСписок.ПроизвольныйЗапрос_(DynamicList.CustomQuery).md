ДинамическийСписок (DynamicList)

ПроизвольныйЗапрос (CustomQuery)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак автоматической генерации текста запроса по указанной главной таблице.  
[Ложь](lang__def_BooleanFalse.md) - выполняется автоматическая генерация текста запроса.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вне сервера "1С:Предприятие" не используется.

Использование в версии:

Доступен, начиная с версии 8.2.
