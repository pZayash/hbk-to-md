ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает текущий для выборки объект базы данных, предназначенный для чтения, изменения, добавления и удаления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
