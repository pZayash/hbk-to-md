ТокенДоступа (AccessToken)

ВремяЖизни (LifeTime)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет время жизни токена в секундах. Устанавливает значение для ключа 'exp' равное сумме времени создания и времени жизни.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
