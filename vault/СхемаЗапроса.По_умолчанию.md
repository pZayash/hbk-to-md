СхемаЗапроса (QuerySchema)

По умолчанию

Доступен, начиная с версии 8.3.5.

Синтаксис:

Новый СхемаЗапроса()

Описание:

Создает объект типа [СхемаЗапроса](СхемаЗапроса_(QuerySchema).md). Схема содержит один пакет [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md) и один оператор в пакете.

Использование в версии:

Доступен, начиная с версии 8.3.5.
