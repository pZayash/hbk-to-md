БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение бизнес-процесса.

Описание:

Заполняет данные бизнес-процесса данными из другого объекта. Инициирует событие [ОбработкаЗаполнения](БизнесПроцессОбъект.Имя_бизнес-процесса.ОбработкаЗаполнения.md) и вызов его процедуры-обработчика в модуле бизнес-процесса. В ней может размещаться алгоритм, заполняющий данные бизнес-процесса из переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для заполнения может использоваться значение любого типа.

Пример:

|  |
| --- |
| ВыбБизнесПроцесс.Заполнить(ВыбКонтрагент); |
