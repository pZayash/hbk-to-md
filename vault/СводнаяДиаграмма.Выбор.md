СводнаяДиаграмма (PivotChart)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<ЗначениеДиаграммы>, <СтандартнаяОбработка>)

Параметры:

<ЗначениеДиаграммы>

Тип: [ТочкаДиаграммы](ТочкаДиаграммы.md), [СерияДиаграммы](СерияДиаграммы.md), [ЗначениеДиаграммы](ЗначениеДиаграммы.md).   
Значение сводной диаграммы.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Возникает при двойном щелчке мыши на сводной диаграмме.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если сводная диаграмма расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.

Использование в версии:

Доступен, начиная с версии 8.0.
