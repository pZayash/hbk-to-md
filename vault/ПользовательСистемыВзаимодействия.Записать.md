ПользовательСистемыВзаимодействия (CollaborationSystemUser)

Записать (Write)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Записать()

Описание:

Записывает нового пользователя или сохраняет изменения для существующего.  
Запись нового возможна только в привилегированном режиме.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
