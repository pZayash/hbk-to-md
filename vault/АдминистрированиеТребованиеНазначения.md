Описание:

Определяет одно из множества правил, в соответствии с которыми кластер серверов распределяет соединения с информационными базами и сервисы кластера по рабочим серверам и отдельным процессам.  
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
