Расширение декорации формы для картинки (Form decoration extension for a picture)

Масштабировать (Zoomable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство установлено (Истина), то для картинки, не помещающейся в области размещения, появляются линейки прокрутки.   
Изменить размер картинки можно с помощью команд контекстного меню.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
