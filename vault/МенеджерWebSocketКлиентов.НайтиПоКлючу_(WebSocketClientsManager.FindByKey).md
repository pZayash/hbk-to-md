# МенеджерWebSocketКлиентов.НайтиПоКлючу

МенеджерWebSocketКлиентов (WebSocketClientsManager)

НайтиПоКлючу (FindByKey)

Доступен, начиная с версии 8.3.27.

Синтаксис:

НайтиПоКлючу(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор соединения.

Возвращаемое значение:

Тип: [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md) с ключом или [Неопределено](lang__def_Undefined.md), если такого не предопределенного нет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
