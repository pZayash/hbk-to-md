КоллекцияВложенийСистемыВзаимодействия (CollaborationSystemAttachmentCollection)

Переместить (MoveTo)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Переместить(<Элемент>, <Индекс>)

Параметры:

<Элемент> (обязательный)

Тип: [ВложениеСистемыВзаимодействия](ВложениеСистемыВзаимодействия_(CollaborationSystemAttachment).md).   
Элемент, который требуется переместить.

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс, в который нужно переместить указанный элемент.

Описание:

Перемещает элемент коллекции в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
