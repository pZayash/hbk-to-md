HTTPОтвет (HTTPResponse)

Заголовки (Headers)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Соответствие](Соответствие.md).   
HTTP-заголовки ответа сервера в виде соответствия: "Название заголовка" - "Значение".  
При наличии в ответе нескольких одинаковых заголовков, они будут записаны в качестве значения свойства через запятую.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
