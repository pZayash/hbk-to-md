COMSafeArray (COMSafeArray)

GetType (GetType)

Доступен, начиная с версии 8.0.

Синтаксис:

GetType()

Возвращаемое значение:

Тип: Строка.   
Описание возвращаемых значений типов элемента массива см. в описании параметра <ТипЭлемента> конструктора [По типу элемента 1](COMSafeArray.По_типу_элемента_1.md).

Описание:

Получает тип элемента массива.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ТипЭлемента = Массив.GetType(); |

Использование в версии:

Доступен, начиная с версии 8.0.
