РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

ИспользоватьНастройкиТекущегоСеанса (UseCurrentSessionSettings)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования региональных настроек текущего сеанса при получении представления данных типа [Число](lang__def_Number.md), [Дата](lang__def_Date.md), [Булево](lang__Булево_(Boolean).md):

- [Истина](lang__def_BooleanTrue.md) - используются региональные настройки сеанса, выполняющего получение представления,
- [Ложь](lang__def_BooleanFalse.md) - используются значения других свойств этого объекта, полученного методом [ПолучитьРегиональныеНастройкиИнформационнойБазы](Глобальный_контекст.ПолучитьРегиональныеНастройкиИнформационнойБазы_(Global_context.GetInfoBaseRegionalSettings).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
