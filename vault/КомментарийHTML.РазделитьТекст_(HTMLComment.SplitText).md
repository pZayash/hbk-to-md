# КомментарийHTML.РазделитьТекст

КомментарийHTML (HTMLComment)

РазделитьТекст (SplitText)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.2.

Синтаксис:

РазделитьТекст(<Смещение>)

Параметры:

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение относительно начала символьных данных.

Возвращаемое значение:

Тип: [ТекстDOM](ТекстDOM_(DOMText).md).   

Описание:

Разделяет текстовый узел. В данном узле остаются символьные данные до смещения, новый узел будет содержать символьные данные после смещения. Новый узел добавляется к родителю данного узла в качестве дочернего, следующего за данным.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных;
- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.2.
