Расширение поля формы для поля периода (Form field extension for a period field)

Ширина (Width)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Указывает ширину элемента управления в символах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
