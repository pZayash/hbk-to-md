Доступен, начиная с версии 8.0.

Свойства:

[НастройкаУсловногоОформления (ConditionalAppearanceSetting)](Расширение_табличного_поля_условного_оформления.НастройкаУсловногоОформления_(Conditional_appearance_table_box_extension.ConditionalAppearanceSetting).md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для условного оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
