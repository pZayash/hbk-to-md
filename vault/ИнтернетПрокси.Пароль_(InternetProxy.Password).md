# ИнтернетПрокси.Пароль

ИнтернетПрокси (InternetProxy)

Пароль (Password)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Пароль(<Протокол>)

Параметры:

<Протокол> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя протокола, для которого запрашивается установленный пароль. При указании пустой строки возвращается пароль, установленный по умолчанию.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает пароль, установленный для аутентификации на прокси-сервере для данного протокола.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
