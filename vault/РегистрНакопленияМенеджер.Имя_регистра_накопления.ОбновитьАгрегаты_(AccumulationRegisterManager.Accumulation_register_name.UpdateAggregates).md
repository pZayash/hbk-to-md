РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

ОбновитьАгрегаты (UpdateAggregates)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбновитьАгрегаты(<Порционное>)

Параметры:

<Порционное> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - заполнение происходит порциями; иначе – полностью заполняются все агрегаты, помеченные к использованию.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Отражает в агрегатах движения, происшедшие в период от даты прошлого переноса движений, или пересчета агрегатов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
