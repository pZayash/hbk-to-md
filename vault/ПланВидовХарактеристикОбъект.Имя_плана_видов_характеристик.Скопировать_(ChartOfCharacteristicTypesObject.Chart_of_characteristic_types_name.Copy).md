ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md).   

Описание:

Создает новый элемент плана видов характеристик копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Если Не ОбъектСсылка.Пустая() Тогда     Копия = ОбъектСсылка.Скопировать();     Копия.Записать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
