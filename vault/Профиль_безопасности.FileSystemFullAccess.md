Профиль безопасности (ISecurityProfile)

FileSystemFullAccess (FileSystemFullAccess)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет полный доступ к файловой системе сервера.  
Истина – доступна файловая система. Ложь – доступ к файловой системе запрещен, можно использовать только виртуальные каталоги, перечисленные в списке виртуальных каталогов профиля безопасности.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
