ЗаписьHTML (HTMLWriter)

УстановитьСтроку (SetString)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьСтроку(<ТипКодировки>)

Параметры:

<ТипКодировки> (необязательный)

Тип: [Строка](lang__def_String.md).   
качестве типа кодировки может быть указано имя одной из возможных кодировок. Это никак не влияет на формируемую строку. Но если будет записано объявление HTML-документа, то объявление будет содержать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то атрибут 'encoding' не будет присутствовать в объявлении HTML-документа.

Описание:

Инициализирует объект для вывода результирующего HTML в строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьHTML = Новый ЗаписьHTML; ЗаписьHTML.УстановитьСтроку(); // Запись документа HTML  ... СтрHTML = ЗаписьHTML.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.2.
