Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке видов расчета.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПередЗаписью_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.BeforeWrite).md) и [ПриЗаписи](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПриЗаписи_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.OnWrite).md) объекта [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
