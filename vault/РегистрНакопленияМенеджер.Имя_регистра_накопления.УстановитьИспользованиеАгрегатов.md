РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

УстановитьИспользованиеАгрегатов (SetAggregatesUsing)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьИспользованиеАгрегатов(<Использование>)

Параметры:

<Использование> (обязательный)

Тип: Булево.   
Режим использования агрегатов.  
Истина – использование агрегатов разрешается.   
Ложь – все агрегаты перестают использоваться и очищаются.

Описание:

Устанавливает использование агрегатов.  
Заполнение агрегатов можно выполнить при помощи метода [ОбновитьАгрегаты](РегистрНакопленияМенеджер.Имя_регистра_накопления.ОбновитьАгрегаты.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
