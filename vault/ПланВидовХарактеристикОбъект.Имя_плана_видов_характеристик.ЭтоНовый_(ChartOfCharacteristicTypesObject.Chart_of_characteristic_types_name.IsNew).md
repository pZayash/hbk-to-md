ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - изменяется еще ни разу не записанный элемент; [Ложь](lang__def_BooleanFalse.md) - элемент уже записан.

Описание:

Определяет, записан ли вид характеристик в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Если Объект.ЭтоНовый() Тогда     Объект.Наименование = "Новый элемент"; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
