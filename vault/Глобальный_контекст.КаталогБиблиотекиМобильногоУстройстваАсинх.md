КаталогБиблиотекиМобильногоУстройстваАсинх (MobileDeviceLibraryDirAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КаталогБиблиотекиМобильногоУстройстваАсинх(<ТипКаталога>)

Параметры:

<ТипКаталога> (обязательный)

Тип: [ТипКаталогаБиблиотекиМобильногоУстройства](ТипКаталогаБиблиотекиМобильногоУстройства.md).   
Тип каталога библиотеки мобильного устройства.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - тип Строка, имя каталога.

Описание:

Асинхронная функция. Получает путь к библиотекам мобильного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
