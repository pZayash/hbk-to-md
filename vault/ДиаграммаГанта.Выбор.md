ДиаграммаГанта (GanttChart)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<Значения>, <СтандартнаяОбработка>, <Дата>)

Параметры:

<Значения>

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта.md), [СерияДиаграммыГанта](СерияДиаграммыГанта.md), [Массив](Массив.md), [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта.md), [МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени.md).   
Если тип параметра [Массив](Массив.md), то первый элемент - это [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта.md), второй и последующие [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта.md) (в случае перекрывающихся интервалов).

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

<Дата>

Тип: Дата.   
Дата, соответствующая точке выбора. Применима для значения и интервала диаграммы Ганта. При выборе других объектов принимает значение Неопределено.

Описание:

Возникает при двойном щелчке мыши на диаграмме Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если диаграмма Ганта расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.

Использование в версии:

Доступен, начиная с версии 8.0.
