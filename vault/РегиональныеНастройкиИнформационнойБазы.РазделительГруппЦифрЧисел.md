РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

РазделительГруппЦифрЧисел (NumbersDigitGroupSeparator)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Строка из одного символа. Символ, который должен использоваться для разделения групп цифр при получении представления данных типа Число.   
Если пустая строка, то используется соответствующая региональная настройка текущего сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
