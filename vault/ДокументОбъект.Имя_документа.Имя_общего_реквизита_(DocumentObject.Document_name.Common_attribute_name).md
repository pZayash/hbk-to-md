# ДокументОбъект.<Имя документа>.<Имя общего реквизита>

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Свойства](ДокументОбъект.Имя_документа__Свойства.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

<Имя общего реквизита> (<Common attribute name>)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения общих реквизитов, входящих в объект метаданных. Но только тех общих реквизитов, которые или не являются разделителями ([РазделениеДанных](ОбъектМетаданных_ОбщийРеквизит.РазделениеДанных_(MetadataObject_CommonAttribute.DataSeparation).md) = [НеИспользовать](РазделениеДанныхОбщегоРеквизита.НеИспользовать_(CommonAttributeDataSeparation.DontUse).md)), или являются разделителями ([РазделениеДанных](ОбъектМетаданных_ОбщийРеквизит.РазделениеДанных_(MetadataObject_CommonAttribute.DataSeparation).md) = [Разделять](РазделениеДанныхОбщегоРеквизита.Разделять_(CommonAttributeDataSeparation.Separate).md)) и имеют настройку разделения [ИспользованиеРазделяемыхДанных](ОбъектМетаданных_ОбщийРеквизит.ИспользованиеРазделяемыхДанных_(MetadataObject_CommonAttribute.SeparatedDataUse).md) = [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно_(CommonAttributeSeparatedDataUse.IndependentlyAndSimultaneously).md).   
Доступ к значению осуществляется по имени, как оно задано в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
