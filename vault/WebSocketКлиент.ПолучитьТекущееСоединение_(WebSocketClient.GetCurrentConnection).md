WebSocketКлиент (WebSocketClient)

ПолучитьТекущееСоединение (GetCurrentConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьТекущееСоединение()

Возвращаемое значение:

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md), [Неопределено](lang__def_Undefined.md).   
Возвращает объект [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md) или  
[Неопределено](lang__def_Undefined.md) - если соединение отсутствует.

Описание:

Получает объект текущего соединения с WebSocket-клиентом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
