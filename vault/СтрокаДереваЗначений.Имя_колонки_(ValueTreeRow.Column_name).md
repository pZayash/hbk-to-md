СтрокаДереваЗначений (ValueTreeRow)

<Имя колонки> (<Column name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения в колонках данной строки дерева значений. Доступ к значению осуществляется по имени колонки. Имена свойств совпадают с именами колонок дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
