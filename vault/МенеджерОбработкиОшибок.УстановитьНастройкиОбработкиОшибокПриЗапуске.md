МенеджерОбработкиОшибок (ErrorProcessingManager)

УстановитьНастройкиОбработкиОшибокПриЗапуске (SetErrorOnStartupProcessingSettings)

Доступен, начиная с версии 8.3.21.

Синтаксис:

УстановитьНастройкиОбработкиОшибокПриЗапуске(<Настройки>)

Параметры:

<Настройки> (обязательный)

Тип: [НастройкиОбработкиОшибокПриЗапуске](НастройкиОбработкиОшибокПриЗапуске.md).   
Новые настройки.

Описание:

Устанавливает новые параметры помощи при возникновении ошибки во время запуска.  
Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
