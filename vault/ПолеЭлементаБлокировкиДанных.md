Описание:

Содержит информацию о значении или колонке источника данных, содержащей значения блокировки для определенного поля пространства имен блокировок.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
