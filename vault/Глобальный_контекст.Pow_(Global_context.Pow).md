Глобальный контекст (Global context)

Pow (Pow)

Доступен, начиная с версии 8.0.

Синтаксис:

Pow(<Основание>, <Показатель>)

Параметры:

<Основание> (обязательный)

Тип: [Число](lang__def_Number.md).   
Основание операции возведения в степень.

<Показатель> (обязательный)

Тип: [Число](lang__def_Number.md).   
Показатель степени.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат возведения в степень.

Описание:

Возводит число <X> в степень <Y>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

В случае арифметического переполнения или невозможности вычислить результат происходит ошибка "Неправильное значение аргумента встроенной функции (Pow)".

Пример:

|  |
| --- |
| // Вычисление объема шара по значению радиуса  Pi = 3.1415926535897932; Радиус = 3; V = 4\*Pi\*Pow(Радиус,3)/3; Сообщить("Объем шара с радиусом "      + Радиус + " равен " + Формат(V,"ЧДЦ=2")); |

Использование в версии:

Доступен, начиная с версии 8.0.
