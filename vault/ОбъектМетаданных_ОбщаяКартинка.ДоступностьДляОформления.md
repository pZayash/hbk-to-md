ОбъектМетаданных: ОбщаяКартинка (MetadataObject: CommonPicture)

ДоступностьДляОформления (AvailabilityForAppearance)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает доступность картинки для выбора в форме выбора картинки, открываемой из формы редактирования элемента условного оформления в режиме 1С:Предприятие. Доступно для редактирования только тогда, когда свойство [ДоступностьДляВыбора](ОбъектМетаданных_ОбщаяКартинка.ДоступностьДляВыбора.md) содержит Истина.   
По умолчанию содержит Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
