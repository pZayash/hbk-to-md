# РегистрыРасчетаМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Регистры расчета](Регистры_расчета.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[РегистрРасчетаМенеджер.\<Имя регистра расчета\>](РегистрРасчетаМенеджер.Имя_регистра_расчета.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров расчета.  
Возможно обращение к менеджеру регистра расчета посредством оператора [...]. В качестве аргумента передается имя регистра расчета, как оно задано в конфигурации.  

Свойства:

[\<Имя регистра расчета\> (\<Calculation register name\>)](РегистрыРасчетаМенеджер.Имя_регистра_расчета.md)  

Описание:

Предоставляет доступ к менеджерам всех регистров расчета (объектам типа [РегистрРасчетаМенеджер.](РегистрРасчетаМенеджер.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаМенеджер.Имя_регистра_расчета.md)>. Доступ к объектам данного типа осуществляется через объект глобального контекста [РегистрыРасчета](Глобальный_контекст.РегистрыРасчета.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [РегистрыРасчета](Глобальный_контекст.РегистрыРасчета.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
