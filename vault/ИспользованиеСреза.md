Описание:

Определяет набор вариантов формирования среза периодического регистра сведений. Используется для заполнения свойства [ВыбиратьСрез](Расширение_табличного_поля_списка_записей_регистра_сведений.ВыбиратьСрез.md).

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Расширение табличного поля списка записей регистра сведений](Расширение_табличного_поля_списка_записей_регистра_сведений.md), свойство [ВыбиратьСрез](Расширение_табличного_поля_списка_записей_регистра_сведений.ВыбиратьСрез.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
