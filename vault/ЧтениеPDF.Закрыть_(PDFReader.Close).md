ЧтениеPDF (PDFReader)

Закрыть (Close)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый файл.   
Закрытие файла выполняется автоматически при удалении объекта [ЧтениеPDF](ЧтениеPDF_(PDFReader).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [ЗакрытьАсинх](ЧтениеPDF.ЗакрытьАсинх_(PDFReader.CloseAsync).md)  
[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [НачатьЗакрытие](ЧтениеPDF.НачатьЗакрытие_(PDFReader.BeginClose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
