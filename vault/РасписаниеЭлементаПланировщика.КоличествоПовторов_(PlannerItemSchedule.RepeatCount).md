# РасписаниеЭлементаПланировщика.КоличествоПовторов

РасписаниеЭлементаПланировщика (PlannerItemSchedule)

КоличествоПовторов (RepeatCount)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Устанавливает предельное количество повторения элемента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если установлено значение 0, то количество повторов не ограничено.

Использование в версии:

Доступен, начиная с версии 8.3.6.
