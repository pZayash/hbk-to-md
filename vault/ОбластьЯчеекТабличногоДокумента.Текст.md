ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Текст (Text)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Содержит значение текста ячеек области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

1. Свойство доступно для записи, если ячейка не содержит значения (свойство [СодержитЗначение](ОбластьЯчеекТабличногоДокумента.СодержитЗначение.md) имеет значение Ложь).  
2. Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.24.
