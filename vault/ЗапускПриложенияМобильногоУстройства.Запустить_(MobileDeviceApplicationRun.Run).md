ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

Запустить (Run)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Запустить(<Ожидать>)

Параметры:

<Ожидать> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость ожидания результата запуска приложения.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md).   
Зависит от параметра <Ожидать>.  
Если параметр имеет значение [Ложь](lang__def_BooleanFalse.md), то в зависимости от успешности запуска возвращается значение [Истина](lang__def_BooleanTrue.md) или [Ложь](lang__def_BooleanFalse.md).  
Если параметр имеет значение [Истина](lang__def_BooleanTrue.md), то возвращается числовой код результата работы приложения.

Описание:

Запускает приложение.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЗапуск](ЗапускПриложенияМобильногоУстройства.ПоддерживаетсяЗапуск_(MobileDeviceApplicationRun.RunningSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).
