Блокировка (IObjectLock)

Session (Session)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Сеанс](Сеанс_(ISessionInfo).md).   
Содержит объект с информацией о сеансе, установившем данную блокировку, или [Неопределено](lang__def_Undefined.md), если источником блокировки не является сеанс.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
