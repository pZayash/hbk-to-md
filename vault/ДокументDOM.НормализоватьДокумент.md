ДокументDOM (DOMDocument)

НормализоватьДокумент (NormalizeDocument)

Доступен, начиная с версии 8.1.

Синтаксис:

НормализоватьДокумент()

Описание:

Выполняет нормализацию документа DOM так, как если бы было произведено сохранение и восстановление документа, приводящее документ к нормальной форме.  
Результатом работы данного метода является:  
- обновленное дерево подстановки ссылки на сущность;  
- нормализованные значения текстовых узлов (как определено методом [ДокументDOM.Нормализовать](ДокументDOM.Нормализовать.md)).  
Результат работы метода зависит от конфигурации документа, задаваемой через свойство [КонфигурацияDOM](ДокументDOM.КонфигурацияDOM.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
