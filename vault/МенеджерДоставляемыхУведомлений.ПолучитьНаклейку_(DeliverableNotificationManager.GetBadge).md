МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ПолучитьНаклейку (GetBadge)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьНаклейку()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает текущее значение наклейки (badge).  
Работает только под управлением iOS.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
