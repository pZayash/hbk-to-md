# КоллекцияДействийЭлементаПланировщика.Сдвинуть

КоллекцияДействийЭлементаПланировщика (PlannerItemActionCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По индексу

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс сдвигаемого действия в коллекции действий элемента планировщика.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов в коллекции, на которое нужно сдвинуть действие. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание варианта метода:

Сдвигает действие с заданным индексом на указанное смещение.

Вариант синтаксиса: По действию

Синтаксис:

Сдвинуть(<Действие>, <Смещение>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика_(PlannerItemAction).md).   
Сдвигаемое действие элемента планировщика.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов в коллекции, на которое нужно сдвинуть действие. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание варианта метода:

Сдвигает заданное действие на указанное смещение.

Описание:

Сдвигает элемент из указанного индекса на указанное смещение

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
