ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

РасширенноеПредставлениеОбъекта (ExtendedObjectPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием элемента плана обмена и используется при формировании заголовка формы элемента плана обмена.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
