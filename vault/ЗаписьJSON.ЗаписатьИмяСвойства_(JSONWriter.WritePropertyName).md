# ЗаписьJSON.ЗаписатьИмяСвойства

ЗаписьJSON (JSONWriter)

ЗаписатьИмяСвойства (WritePropertyName)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗаписатьИмяСвойства(<ИмяСвойства>)

Параметры:

<ИмяСвойства> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя свойства.

Описание:

Записывает имя свойства JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; // Запись документа JSON ...  ЗаписьJSON.ЗаписатьИмяСвойства("Строка"); ЗаписьJSON.ЗаписатьЗначение("Строковое значение"); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
