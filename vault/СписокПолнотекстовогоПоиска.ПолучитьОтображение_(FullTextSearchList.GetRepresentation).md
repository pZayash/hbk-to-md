СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолучитьОтображение (GetRepresentation)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьОтображение(<ВидОтображения>)

Параметры:

<ВидОтображения> (обязательный)

Тип: [ВидОтображенияПолнотекстовогоПоиска](ВидОтображенияПолнотекстовогоПоиска_(FullTextSearchRepresentationType).md).   
Вид отображения результата полнотекстового поиска.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [ЧтениеXML](ЧтениеXML_(XMLReader).md).   

Описание:

Получает результат поиска: объект [ЧтениеXML](ЧтениеXML_(XMLReader).md) или строку с текстом в формате HTML, в котором средствами HTML найденные слова подсвечены (полужирный шрифт и цвет фона).  
При этом слова, соответствующие разным словам поискового выражения, подсвечиваются разными цветами из набора в 16 предопределенных цветов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
