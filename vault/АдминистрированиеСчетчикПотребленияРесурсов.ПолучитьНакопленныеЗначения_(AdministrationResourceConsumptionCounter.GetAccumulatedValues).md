АдминистрированиеСчетчикПотребленияРесурсов (AdministrationResourceConsumptionCounter)

ПолучитьНакопленныеЗначения (GetAccumulatedValues)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьНакопленныеЗначения(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит параметры отбора значений счетчика потребления ресурсов.   
Структура может содержать следующие ключи:

- Объект (Object) - тип [Строка](lang__def_String.md). Значение по умолчанию [Неопределено](lang__def_Undefined.md);
- ДатаНачала (StartDate) - тип [Дата](lang__def_Date.md). Значение по умолчанию [Неопределено](lang__def_Undefined.md);
- ДатаОкончания (EndDate) - тип [Дата](lang__def_Date.md). Значение по умолчанию [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает массив накопленных значений счетчика потребления ресурсов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.15.
