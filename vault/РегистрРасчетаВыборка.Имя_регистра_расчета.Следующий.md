РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: Булево.   
Истина - следующая запись выбрана; Ложь - достигнут конец выборки.

Описание:

Получает следующую запись из выборки. Для обхода записей регистра нужно после получения выборки вызвать данный метод для позиционирования на первую запись и далее вызывать до тех пор, пока не будет возвращено значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Регистр = РегистрыРасчета.ОсновныеНачисленияСотрудников; Выборка = Регистр.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка выборки      // ...  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
