ПотокВПамяти (MemoryStream)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Вызывает начало закрытие потока. Выполняемое действие зависит от конкретного типа потока.  
Является асинхронным вариантом метода [Закрыть](Поток.Закрыть_(Stream.Close).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Поток](Поток_(Stream).md), метод [Закрыть](Поток.Закрыть_(Stream.Close).md)  
[Поток](Поток_(Stream).md), метод [ЗакрытьАсинх](Поток.ЗакрытьАсинх_(Stream.CloseAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
