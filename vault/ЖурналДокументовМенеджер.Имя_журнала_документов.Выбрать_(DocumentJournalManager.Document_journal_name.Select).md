# ЖурналДокументовМенеджер.<Имя журнала документов>.Выбрать

ЖурналДокументовМенеджер.<Имя журнала документов> (DocumentJournalManager.<Document journal name>)

Выбрать (Select)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать(<ДатаНачала>, <ДатаОкончания>, <Отбор>, <Порядок>)

Параметры:

<ДатаНачала> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Дата и время начала периода выборки документов. Если параметр не указан, то выбираются все документы, начиная с самого первого документа в базе данных.  
Значение по умолчанию: Пустая дата.

<ДатаОкончания> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Дата окончания периода выбираемых документов. Если параметр не указан, то выбираются все документы, начиная с самого первого документа в базе данных.  
Значение по умолчанию: Пустая дата.

<Отбор> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Ключ структуры описывает имя поля, а значение структуры - значение отбора по этому полю. Может задаваться только "Дата" и графы журнала документов, для которых при конфигурировании установлено индексирование.   
Если параметр не указан, то отбор не используется.

<Порядок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет порядок выдачи документов. Может быть указано поле "Дата" или одна из граф журнала документов при условии, что все включенные в данную графу реквизиты документов имеют один и тот же базовый тип (например, [Число](lang__def_Number.md) или [Строка](lang__def_String.md)) и при этом в такой графе отражаются все включенные в журнал документы и при этом при конфигурировании для графы установлено индексирование. После указания имени поля через пробел может быть указано направление упорядочивания. Для упорядочивания в порядке возрастания - "Возр", иначе - "Убыв". По умолчанию выборка упорядочивается по возрастанию.  
Если параметр не задан, выборка упорядочивается по хронологии документов в порядке возрастания.

Возвращаемое значение:

Тип: [ДокументВыборка.\<Имя документа\>](ДокументВыборка.Имя_документа_(DocumentSelection.Document_name).md).   

Описание:

Формирует выборку документов, регистрируемых в журнале, за определенный период.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ВыборкаИзЖурнала = ЖурналыДокументов.Торговля.Выбрать(, РабочаяДата); |

Использование в версии:

Доступен, начиная с версии 8.0.
