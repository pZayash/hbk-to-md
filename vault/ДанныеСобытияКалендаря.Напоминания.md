ДанныеСобытияКалендаря (CalendarEventData)

Напоминания (Reminders)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит массив напоминаний. Элементом массива является число минут до начала события, когда сработает напоминание.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
