ОписаниеКомандыПланировщика (PlannerCommandDescription)

По умолчанию

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ОписаниеКомандыПланировщика(<Команда>, <Представление>)

Параметры:

<Команда> (обязательный)

Тип: Произвольный.   
Устанавливает команду.   
См. описание свойства [Команда](ОписаниеКомандыПоляПланировщика.Команда_(PlannerFieldCommandDescription.Command).md).

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст команды.

Описание:

Создает объект на основании команды и представления.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.24.
