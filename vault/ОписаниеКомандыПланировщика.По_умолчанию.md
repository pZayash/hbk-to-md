ОписаниеКомандыПланировщика (PlannerCommandDescription)

По умолчанию

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ОписаниеКомандыПланировщика(<Команда>, <Представление>)

Параметры:

<Команда> (обязательный)

Тип: Произвольный.   
Устанавливает команду.   
См. описание свойства [Команда](ОписаниеКомандыПоляПланировщика.Команда.md).

<Представление> (необязательный)

Тип: Строка.   
Текст команды.

Описание:

Создает объект на основании команды и представления.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.24.
