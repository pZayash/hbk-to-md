ПланВидовХарактеристикМенеджер.<Имя плана видов характеристик> (ChartOfCharacteristicTypesManager.<Chart of characteristic types name>)

НайтиПоКоду (FindByCode)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоКоду(<Код>, <Родитель>)

Параметры:

<Код> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомый код.

<Родитель> (необязательный)

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).   
Родитель, в пределах которого нужно выполнять поиск Если не указано, то во всем плане видов характеристик.

Возвращаемое значение:

Тип: [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)>, [Неопределено](lang__def_Undefined.md).   
Если не существует ни одного элемента с требуемым кодом, то будет возвращена пустая ссылка.  
Если код не задан (длина = 0), то будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента по его коду.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
