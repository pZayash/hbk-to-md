ИнформацияОбАгрегатах (AggregatesInformation)

Эффект (Effect)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит значение эффекта от использования агрегатов при выполнении запросов по сравнению с таблицей оборотов (на момент построения списка).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
