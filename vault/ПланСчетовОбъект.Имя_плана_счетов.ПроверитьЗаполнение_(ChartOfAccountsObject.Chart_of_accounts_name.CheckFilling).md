# ПланСчетовОбъект.<Имя плана счетов>.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Методы](ПланСчетовОбъект.Имя_плана_счетов__Методы.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](ПланСчетовОбъект.Имя_плана_счетов.ОбработкаПроверкиЗаполнения_(ChartOfAccountsObject.Chart_of_accounts_name.FillCheckProcessing).md).

Доступен, начиная с версии 8.2.
