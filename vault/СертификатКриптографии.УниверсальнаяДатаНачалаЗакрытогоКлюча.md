СертификатКриптографии (CryptoCertificate)

УниверсальнаяДатаНачалаЗакрытогоКлюча (StartUniversalDateOfPrivateKey)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Дата, Неопределено.   
Содержит дату начала действия ключа электронной подписи, представленной в формете всемирного времени, или Неопределено - если соответствующее дополнение отсутствует.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.27.
