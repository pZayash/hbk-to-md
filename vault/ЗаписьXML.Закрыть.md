ЗаписьXML (XMLWriter)

Закрыть (Close)

Доступен, начиная с версии 8.0.

Синтаксис:

Закрыть()

Возвращаемое значение:

Тип: Строка.   

Описание:

Завершает запись текста XML. Если производилась запись в файл, то файл закрывается. Если производилась запись в строку, то результирующая строка будет получена в качестве возвращаемого значения метода. Если производилась запись в файл, то метод вернет пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьXML = Новый ЗаписьXML; ЗаписьXML.УстановитьСтроку(); // Запись документа XML  ... СтрXML = ЗаписьXML.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
