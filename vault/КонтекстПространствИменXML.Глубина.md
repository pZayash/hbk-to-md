КонтекстПространствИменXML (XMLNamespaceContext)

Глубина (Depth)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Глубина контекста - количество вложенных элементов XML, начиная от текущего узла (текущий узел определяется от объекта владельца - [ЧтениеXML](ЧтениеXML.md) или [ЗаписьXML](ЗаписьXML.md)) до корня документа XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
