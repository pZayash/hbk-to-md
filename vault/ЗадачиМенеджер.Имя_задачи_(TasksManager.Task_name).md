ЗадачиМенеджер (TasksManager)

<Имя задачи> (<Task name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md).   
Набор свойств содержит менеджеры отдельных задач. Имена свойств совпадают с именами задач, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
