СертификатКриптографии (CryptoCertificate)

Субъект (Subject)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md).   
Содержит информацию о субъекте сертификата.   
Строковые элементы со следующими именами:

- CN - commonName;
- O - organizationName;
- OU - organizationUnitName;
- C - countryName;
- ST - stateOrProvinceName;
- L - localityName;
- E - emailAddress;
- Street - streetAddress.

Все поля записываются в структуру вида "Ключ - Значение", где данные для ключа и значения получаются из данных сертификата. Формат ключа имеет вид:  
OID<X1>\_<X2>\_...\_<XN>, где X<i> - это компонента идентификатора OID.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
