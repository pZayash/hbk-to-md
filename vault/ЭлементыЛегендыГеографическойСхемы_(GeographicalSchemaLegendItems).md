**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЭлементЛегендыГеографическойСхемы](ЭлементЛегендыГеографическойСхемы_(GeographicalSchemaLegendItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Добавить (Add)](ЭлементыЛегендыГеографическойСхемы.Добавить_(GeographicalSchemaLegendItems.Add).md)  
[Индекс (IndexOf)](ЭлементыЛегендыГеографическойСхемы.Индекс_(GeographicalSchemaLegendItems.IndexOf).md)  
[Количество (Count)](ЭлементыЛегендыГеографическойСхемы.Количество_(GeographicalSchemaLegendItems.Count).md)  
[Очистить (Clear)](ЭлементыЛегендыГеографическойСхемы.Очистить_(GeographicalSchemaLegendItems.Clear).md)

[Сдвинуть (Move)](ЭлементыЛегендыГеографическойСхемы.Сдвинуть_(GeographicalSchemaLegendItems.Move).md)  
[Удалить (Delete)](ЭлементыЛегендыГеографическойСхемы.Удалить_(GeographicalSchemaLegendItems.Delete).md)  

Описание:

Содержит коллекцию элементов легенды географической схемы. Элементы отрисовываются в том порядке, в котором они расположены в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ОбластьЛегендыГеографическойСхемы](ОбластьЛегендыГеографическойСхемы_(GeographicalSchemaLegendArea).md), свойство [Элементы](ОбластьЛегендыГеографическойСхемы.Элементы_(GeographicalSchemaLegendArea.Items).md)
