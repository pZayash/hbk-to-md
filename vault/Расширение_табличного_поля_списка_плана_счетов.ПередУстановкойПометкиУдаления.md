<!-- signature:end -->

Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то пометка удаления изменена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед установкой пометки удаления в списке счетов.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ПланСчетовОбъект.Имя_плана_счетов.ПередЗаписью.md) и [ПриЗаписи](ПланСчетовОбъект.Имя_плана_счетов.ПриЗаписи.md) объекта [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
