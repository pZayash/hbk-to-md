# ОформленияЯчеекДинамическогоСписка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [Расширение для динамического списка](Расширение_для_динамического_списка.md)

Доступен, начиная с версии 8.3.10.

Элементы коллекции:

[КлючИЗначение](КлючИЗначение_(KeyAndValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа [КлючИЗначение](КлючИЗначение_(KeyAndValue).md), где Ключ - строка, содержащая путь компоновки данных соответствующей колонки,¶Значение - [ОформлениеЯчейкиДинамическогоСписка](ОформлениеЯчейкиДинамическогоСписка_(DynamicListCellAppearance).md) - [ОформлениеЯчейкиДинамическогоСписка](ОформлениеЯчейкиДинамическогоСписка_(DynamicListCellAppearance).md).  
Возможно обращение к оформлению ячейки посредством оператора [...]. В качестве аргумента передается строка (путь компоновки данных соответствующей колонки).  

Методы:

[Количество (Count)](ОформленияЯчеекДинамическогоСписка.Количество_(DynamicListCellAppearances.Count).md)  
[Получить (Get)](ОформленияЯчеекДинамическогоСписка.Получить_(DynamicListCellAppearances.Get).md)  

Описание:

Содержит коллекцию оформлений ячеек динамического списка. Коллекция содержит элементы только для видимых ячеек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
