Описание:

Объект содержит краткую информацию о конфигурации или расширении конфигурации.  
Может быть получен на основе двоичных данных файла конфигурации или в качестве поля описания файла обновления конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ДанныеКонфигурации = Новый ОписаниеКонфигурации                         (Новый ДвоичныеДанные("c:\1cv8.cf"));  ДанныеРасширения = Новый ОписаниеКонфигурации                 (РасширенияКонфигурации.Получить()[0].ПолучитьДанные()); |

Использование в версии:

Доступен, начиная с версии 8.3.13.
