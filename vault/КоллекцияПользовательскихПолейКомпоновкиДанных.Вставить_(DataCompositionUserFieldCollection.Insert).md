КоллекцияПользовательскихПолейКомпоновкиДанных (DataCompositionUserFieldCollection)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных_(DataCompositionUserFieldExpression).md) и [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных_(DataCompositionUserFieldCase).md).

Возвращаемое значение:

Тип: [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных_(DataCompositionUserFieldExpression).md), [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных_(DataCompositionUserFieldCase).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
