ОбрабатываемаяКартинка (ProcessingPicture)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый ОбрабатываемаяКартинка(<Картинка>, <ВсеВарианты>, <ПодобратьВариант>)

Параметры:

<Картинка> (обязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Исходная картинка.

<ВсеВарианты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Параметр имеет смысл для картинки в архиве. Будут выгружены либо все варианты ([Истина](lang__def_BooleanTrue.md)), либо один конкретный ([Ложь](lang__def_BooleanFalse.md)).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ПодобратьВариант> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md), [Неопределено](lang__def_Undefined.md).   
Имеет смысл только для картинки в архиве, и когда параметр [ВсеВарианты](html.md) установлен в значение [Ложь](lang__def_BooleanFalse.md).   
Установленные в соответствии пары Ключ-Значение соответствуют XML-тегам в манифесте вариантов картинки.   
Если параметры выбора варианта не заданы, будет выбран вариант по умолчанию.  
Если параметры указывают на несуществующий в архиве вариант, то будет создан пустой файл.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Конструктор на основании исходной картинки.

Использование в версии:

Доступен, начиная с версии 8.3.14.
