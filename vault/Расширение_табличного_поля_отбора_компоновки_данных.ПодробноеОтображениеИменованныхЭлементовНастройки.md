Расширение табличного поля отбора компоновки данных (Filter table box extension)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство установлено в Ложь, именованные части отбора (свойство [Представление](ЭлементОтбораКомпоновкиДанных.Представление.md) не пустое) отображаются в табличном поле одной строкой с одной ячейкой, в которой выводится представление.  
В противном случае представление редактируется в отдельной колонке, отображение именованных и неименованных элементов отбора не отличается.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
