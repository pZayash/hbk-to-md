МенеджерКопийБазыДанных (DatabaseCopiesManager)

ПолучитьИнформациюОбИспользованииБазыДанныхКопии (GetCopyDatabaseUseInformation)

Доступен, начиная с версии 8.3.26.

Вариант синтаксиса: По параметрам базы данных

Синтаксис:

ПолучитьИнформациюОбИспользованииБазыДанныхКопии(<ТипСУБД>, <СерверБазыДанных>, <БазаДанных>, <ПользовательБазыДанных>, <ПарольПользователяБазыДанных>)

Параметры:

<ТипСУБД> (необязательный)

Тип: [ТипСУБДКопииБазыДанных](ТипСУБДКопииБазыДанных_(DatabaseCopyDBMSType).md).   
Тип СУБД копии базы данных.

<СерверБазыДанных> (необязательный)

Тип: [Строка](lang__def_String.md).   
Адрес сервера базы данных копии. Может содержать значение порта после имени (или ip-адреса) хоста и двоеточия.

<БазаДанных> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя базы данных копии.

<ПользовательБазыДанных> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пользователь базы данных копии.

<ПарольПользователяБазыДанных> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя базы данных копии.

Вариант синтаксиса: По копии

Синтаксис:

ПолучитьИнформациюОбИспользованииБазыДанныхКопии(<Копия>)

Параметры:

<Копия> (необязательный)

Тип: [МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md).   
Проверяемая копия базы данных.

Описание варианта метода:

Для базы данных, указанной в свойствах менеджера копии, даже если копия не сохранена.

Вариант синтаксиса: По имени копии

Синтаксис:

ПолучитьИнформациюОбИспользованииБазыДанныхКопии(<ИмяКопии>)

Параметры:

<ИмяКопии> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя проверяемой копии.

Возвращаемое значение:

Тип: [ИнформацияОбИспользованииБазыДанныхКопии](ИнформацияОбИспользованииБазыДанныхКопии_(CopyDatabaseUseInformation).md).   

Описание:

Возвращает информацию об использовании базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
