Рабочий процесс (IWorkingProcessInfo)

TurnedOn (TurnedOn)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Устанавливается кластером при необходимости запустить или остановить рабочий процесс.  
Истина - процесс должен быть запущен и будет запущен при возможности.   
Ложь - процесс должен быть остановлен и будет остановлен после отсоединения всех пользователей или после истечения времени, заданного свойством [ExpirationTimeout](Кластер_серверов.ExpirationTimeout.md) объекта [Кластер серверов](Кластер_серверов.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
