Дендрограмма (Dendrogram)

КоличествоЭлементовШкалы (ScaleItemCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет количество элементов, одновременно отображаемых на оси.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Используется при ПоддержкаМасштаба = КоличествоЭлементов.

Использование в версии:

Доступен, начиная с версии 8.0.
