НастройкиКомпоновкиДанных (DataCompositionSettings)

ПолучитьОбъектПоИдентификатору (GetObjectByID)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбъектПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных.md).   
Идентификатор объекта.

Возвращаемое значение:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md), [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных.md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных.md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных.md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных.md), [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных.md), [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных.md), [КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных](КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных.md), [КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных](КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных.md), Неопределено.   
Неопределено - объект, для которого был передан идентификатор, не найден или если он не является частью данного объекта.

Описание:

Получает объект структуры настроек компоновки данных, соответствующие переданному идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
