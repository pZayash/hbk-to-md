Файл (File)

НачатьУстановкуНевидимости (BeginSettingHidden)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьУстановкуНевидимости(<ОписаниеОповещения>, <Невидимость>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Невидимость> (обязательный)

Тип: Булево.   
Истина - установить атрибут "Скрытый", Ложь - снять атрибут "Скрытый".

Описание:

Начинает установку значения атрибута файла "Скрытый" ("Hidden").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Файл](Файл.md), метод [УстановитьНевидимость](Файл.УстановитьНевидимость.md)  
[Файл](Файл.md), метод [УстановитьНевидимостьАсинх](Файл.УстановитьНевидимостьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
