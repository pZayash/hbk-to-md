Диаграмма (Chart)

ПалитраЦветов (ColorPalette)

Не рекомендуется использовать, начиная с версии 8.3.12.

Рекомендуется использовать:

- [ОписаниеПалитрыЦветов](Диаграмма.ОписаниеПалитрыЦветов_(Chart.ColorPaletteDescription).md)
- [ПалитраЦветов](ОписаниеПалитрыЦветовДиаграммы.ПалитраЦветов_(ChartColorPaletteDescription.ColorPalette).md)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПалитраЦветовДиаграммы](ПалитраЦветовДиаграммы_(ChartColorPalette).md).   
Содержит палитру цветов, используемую при создании новых серий.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.3.12.

Описание изменено в версии 8.3.15.
