СерииДиаграммыГанта (GanttChartSeriesCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс серии.

Возвращаемое значение:

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md).   

Описание:

Вставляет серию с указанным индексом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
