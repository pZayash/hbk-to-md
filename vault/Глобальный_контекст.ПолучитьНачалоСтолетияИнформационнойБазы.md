Глобальный контекст (Global context)

ПолучитьНачалоСтолетияИнформационнойБазы (GetInfoBaseBeginningOfCentury)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьНачалоСтолетияИнформационнойБазы()

Возвращаемое значение:

Тип: Число, Неопределено.   

Описание:

Возвращает год начала столетия, установленный для информационной базы или Неопределено, если начало столетия не установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
