ВариантОтображенияСообщенияОбОшибке (ErrorMessageDisplayVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.17.

Описание:

Если используется в качестве параметра метода [ПоказатьИнформациюОбОшибке](МенеджерОбработкиОшибок.ПоказатьИнформациюОбОшибке.md), то означает, что будет использовано значение из настроек данного пользователя.   
Если же &Авто получено из настроек, то в режиме совместимости с 8.3.16 означает [КраткоеПредставлениеОшибки](ВариантОтображенияСообщенияОбОшибке.КраткоеПредставлениеОшибки.md), иначе - [СообщениеОбОшибкеДляПользователя](ВариантОтображенияСообщенияОбОшибке.СообщениеОбОшибкеДляПользователя.md).

Использование в версии:

Доступен, начиная с версии 8.3.17.
