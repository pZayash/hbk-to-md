ПараметрыСеанса (SessionParameters)

<Имя параметра сеанса> (<Session parameter name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения параметров сеанса. Доступ к значению осуществляется по имени. Имена свойств совпадают с именами параметров сеанса, как они заданы в Конфигураторе.  
Позволяет получить или установить значение одноименного параметра сеанса. Типы, допустимые для значения параметра сеанса, определяются при создании параметра сеанса в Конфигураторе. Среди допустимых могут быть следующие типы:

- Число;
- Строка;
- Дата;
- Булево;
- Неопределено;
- Null;
- [ХранилищеЗначения](ХранилищеЗначения.md);
- [ФиксированныйМассив](ФиксированныйМассив.md);
- [ФиксированноеСоответствие](ФиксированноеСоответствие.md);
- [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md);
- [УникальныйИдентификатор](УникальныйИдентификатор.md);
- [ОписаниеТипов](ОписаниеТипов.md);
- [ВидСчета](ВидСчета.md);
- [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии.md);
- [ВидДвиженияНакопления](ВидДвиженияНакопления.md);
- [ДвоичныеДанные](ДвоичныеДанные.md);
- [ВидДвиженияНакопления](ВидДвиженияНакопления.md);
- [ВидСчета](ВидСчета.md);
- [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии.md);
- Любая ссылка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При попытке обращения к неинициализированному параметру сеанса (в течении сеанса не происходило присвоение параметра) генерируется исключение.  
Элементами значений типов [ФиксированныйМассив](ФиксированныйМассив.md), [ФиксированноеСоответствие](ФиксированноеСоответствие.md), [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md) могут быть значения только перечисленных типов.

См. также:

[Глобальный контекст](Глобальный_контекст.md), событие [УстановкаПараметровСеанса](Глобальный_контекст.УстановкаПараметровСеанса.md)

Доступен, начиная с версии 8.0.
