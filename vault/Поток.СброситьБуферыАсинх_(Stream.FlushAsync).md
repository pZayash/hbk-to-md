Поток (Stream)

СброситьБуферыАсинх (FlushAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СброситьБуферыАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Сбрасывает все промежуточные буферы и производит запись всех незаписанных данных в целевое устройство.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[Поток](Поток_(Stream).md), метод [НачатьСбросБуферов](Поток.НачатьСбросБуферов_(Stream.BeginFlush).md)  
[Поток](Поток_(Stream).md), метод [СброситьБуферы](Поток.СброситьБуферы_(Stream.Flush).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
