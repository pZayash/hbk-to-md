Глобальный контекст (Global context)

ЗапуститьСистему (RunSystem)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗапуститьСистему(<ДополнительныеПараметрыКоманднойСтроки>, <ДождатьсяЗавершения>, <КодВозврата>)

Параметры:

<ДополнительныеПараметрыКоманднойСтроки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит дополнительные параметры, которые будут переданы новому сеансу. Если такие параметры уже есть среди текущих установок, переданные заменят текущие.  
Если режим приложения ([УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md)/ [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md)) не задан явно дополнительным параметром командной строки, при запуске текущего приложения учитывается использование параметров принудительного включения режима управляемого или обычного приложения /RunModeManagedApplication и /RunModeOrdinaryApplication.

<ДождатьсяЗавершения> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость дождаться завершения запуска системы для продолжения работы.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<КодВозврата> (необязательный)

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Параметр (если он указан) содержит код завершения работы сеанса "1С:Предприятия"- значение, передаваемое платформой операционной системе, если работа была завершена корректно, и если такой код был получен. В противном случае возвращает [Неопределено](lang__def_Undefined.md).  
Для получения кода возврата значение параметра <ДождатьсяЗавершения> должно быть установлено в [Истина](lang__def_BooleanTrue.md), а сеанс должен быть корректно завершен.  
Возможны следующие значения:

- 0 - успешное завершение;
- 1 - ошибка;
- 101 - ошибка в данных (такой код может быть возвращен при обновлении конфигурации, при проверке и исправлении конфигурации, а также при создании дистрибутива конфигурации).

Описание:

Запускает новый сеанс 1С:Предприятия. Новый сеанс будет запущен с текущими установками.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

По умолчанию новая система запускается в режиме 1С:Предприятия. Если в качестве одного из дополнительных параметров будет передан ключ DESIGNER, новая система будет запущена в режиме Конфигуратора.

Использование в версии:

Доступен, начиная с версии 8.1.
