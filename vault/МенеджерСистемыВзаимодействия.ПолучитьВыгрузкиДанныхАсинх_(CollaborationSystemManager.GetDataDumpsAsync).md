Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьВыгрузкиДанныхАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив объектов типа [ВыгрузкаДанныхСистемыВзаимодействия](ВыгрузкаДанныхСистемыВзаимодействия_(CollaborationSystemDataDump).md).

Асинхронная функция. Получает список выгрузок данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.
