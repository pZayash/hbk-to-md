# ПланВидовХарактеристикОбъект.<Имя плана видов характеристик>.УстановитьПометкуУдаления

ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>, <ВключаяПодчиненные>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак установки / снятие пометки на удаление.  
[Истина](lang__def_BooleanTrue.md) - пометка будет установлена; [Ложь](lang__def_BooleanFalse.md) - пометка будет снята.

<ВключаяПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр установлен в [Истина](lang__def_BooleanTrue.md), то будут помечены на удаление, кроме данного элемента, все его подчиненные элементы в данном счете и во всех подчиненных счетах.  
Если параметр установлен в [Ложь](lang__def_BooleanFalse.md), то будет помечен на удаление только данный элемент.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает/снимает пометку на удаление на план видов характеристик. Объект при этом не удаляется из базы данных. Окончательное удаление помеченных счетов происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередЗаписью_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](ПланВидовХарактеристикВыборка.Имя_плана_видов_характеристик.ПометкаУдаления_(ChartOfCharacteristicTypesSelection.Chart_of_characteristic_types_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Использование в версии:

Доступен, начиная с версии 8.0.
