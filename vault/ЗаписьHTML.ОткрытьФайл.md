ЗаписьHTML (HTMLWriter)

ОткрытьФайл (OpenFile)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: По имени и типу кодировки

Синтаксис:

ОткрытьФайл(<ИмяФайлаHTML>, <ТипКодировки>)

Параметры:

<ИмяФайлаHTML> (обязательный)

Тип: Строка.   
Имя файла, в который будет записываться текст HTML.

<ТипКодировки> (необязательный)

Тип: Строка.   
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае файл будет записан в соответствующей кодировке. При этом в объявлении HTML-документа (если таковое будет записано) будет присутствовать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то для записи файла будет использована кодировка [UTF8](КодировкаТекста.UTF8.md), а атрибут 'encoding' не будет присутствовать в объявлении HTML-документа.

Вариант синтаксиса: С параметрами записи

Синтаксис:

ОткрытьФайл(<ИмяФайлаHTML>, <ПараметрыЗаписи>)

Параметры:

<ИмяФайлаHTML> (обязательный)

Тип: Строка.   
Имя файла, в который будет записываться текст HTML.

<ПараметрыЗаписи> (обязательный)

Тип: [ПараметрыЗаписиXML](ПараметрыЗаписиXML.md).   
Параметры записи XML.

Описание:

Открывает файл для записи HTML. Позволяет указать тип кодировки, который будет использован для записи файла HTML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьHTML = Новый ЗаписьHTML; ЗаписьHTML.ОткрытьФайл("c:\docs\data.html", "UTF-8"); |

Использование в версии:

Доступен, начиная с версии 8.2.
