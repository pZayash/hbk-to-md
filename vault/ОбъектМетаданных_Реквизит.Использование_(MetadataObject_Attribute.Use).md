ОбъектМетаданных: Реквизит (MetadataObject: Attribute)

Использование (Use)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеРеквизита](ИспользованиеРеквизита_(AttributeUse).md).   
Для реквизитов справочников, планов видов характеристик это свойство указывает способ использования данных подчиненного объекта конфигурации (например, [ДляЭлемента](ИспользованиеРеквизита.ДляЭлемента_(AttributeUse.ForItem).md), [ДляГруппы](ИспользованиеРеквизита.ДляГруппы_(AttributeUse.ForFolder).md), [ДляГруппыИЭлемента](ИспользованиеРеквизита.ДляГруппыИЭлемента_(AttributeUse.ForFolderAndItem).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для реквизитов справочников или планов видов характеристик.

Использование в версии:

Доступен, начиная с версии 8.1.
