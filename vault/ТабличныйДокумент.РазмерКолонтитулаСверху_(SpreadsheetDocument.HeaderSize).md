ТабличныйДокумент (SpreadsheetDocument)

РазмерКолонтитулаСверху (HeaderSize)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит размер верхнего колонтитула (в миллиметрах). Значение по умолчанию 10.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[КолонтитулТабличногоДокумента](КолонтитулТабличногоДокумента_(SpreadsheetDocumentHeaderFooter).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
