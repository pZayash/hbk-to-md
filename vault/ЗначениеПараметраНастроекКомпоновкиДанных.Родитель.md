ЗначениеПараметраНастроекКомпоновкиДанных (DataCompositionSettingsParameterValue)

Родитель (Parent)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Неопределено, [ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных.md).   
Родительский параметр. Неопределено для значения параметров верхнего уровня.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
