# Описание типов значений

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › [Общие объекты](Общие_объекты.md)

Для описания допустимых типов значений свойств различных объектов в системе используется специальный объект [ОписаниеТипов](ОписаниеТипов.md). При помощи данного объекта можно описывать допустимые типы значений, которые можно будет присваивать свойствам. Для дополнительного ограничения возможных значений примитивных типов Число, Строка и Дата предусмотрены квалификаторы. Квалификаторы описывают такие параметры, как длина строки или числа, допустимые части даты и т.п.
