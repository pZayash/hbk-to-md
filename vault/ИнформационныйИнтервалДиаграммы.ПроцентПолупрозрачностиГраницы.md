ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

ПроцентПолупрозрачностиГраницы (BorderSemitransparencyPercent)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.   
Содержит степень прозрачности линии границы, указанную в процентах. Целочисленное значение от 0 до 100. 100% - полностью прозрачный.  
Если указано Неопределено, прозрачность расчитывается автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
