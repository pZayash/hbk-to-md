ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

НайтиОбъекты (FindObjects)

Доступен, начиная с версии 8.3.2.

Синтаксис:

НайтиОбъекты(<Тип>, <Имя>, <Таймаут>)

Параметры:

<Тип> (необязательный)

Тип: Тип.   
Тип объекта, которое нужно найти.  
Значение по умолчанию: Неопределено.

<Имя> (необязательный)

Тип: Строка.   
Имя объекта, если это применимо. Может содержать знаки подстановки "\*" и "?".

<Таймаут> (необязательный)

Тип: Число.   
Время в секундах, в течение которого выполняется поиск объекта. Если значение параметра не задано, время поиска не ограничено.  
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив.md).   
Фиксированный массив найденных объектов.

Описание:

Выполняет поиск среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.2.
