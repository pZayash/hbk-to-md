ФайловыйПоток (FileStream)

С указанием режима открытия и уровня доступа

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый ФайловыйПоток(<ИмяФайла>, <РежимОткрытия>, <УровеньДоступа>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла_(FileOpenMode).md).   
Режим открытия файла.

<УровеньДоступа> (обязательный)

Тип: [ДоступКФайлу](ДоступКФайлу_(FileAccess).md).   
Уровень доступа к файлу (чтение, запись).

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер временного буфера, используемого для работы.

Описание:

Открывает выбранный файл в заданном режиме с указанным уровнем доступа.

Использование в версии:

Доступен, начиная с версии 8.3.9.
