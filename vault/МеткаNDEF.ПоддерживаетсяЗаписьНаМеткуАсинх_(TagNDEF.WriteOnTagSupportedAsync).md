МеткаNDEF (TagNDEF)

ПоддерживаетсяЗаписьНаМеткуАсинх (WriteOnTagSupportedAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяЗаписьНаМеткуАсинх()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - физическая метка перезаписываемя и запись на метку не заблокирована,  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, что физическая NFC-метка доступна для записи. Если произошла ошибка, генерируется исключение категории [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа_(ErrorCategory.MultimediaToolsError).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерМетокNDEF](МенеджерМетокNDEF_(TagManagerNDEF).md), метод [ПоддерживаетсяЗаписьНаМетку](МенеджерМетокNDEF.ПоддерживаетсяЗаписьНаМетку_(TagManagerNDEF.WriteOnTagSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
