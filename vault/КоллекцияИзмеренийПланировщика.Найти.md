КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению измерения планировщика.

Возвращаемое значение:

Тип: [ИзмерениеПланировщика](ИзмерениеПланировщика.md), Неопределено.   
Если указанная колонка отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск измерения по значению свойства [Значение](ИзмерениеПланировщика.Значение.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИзмерениеПланировщика](ИзмерениеПланировщика.md), свойство [Значение](ИзмерениеПланировщика.Значение.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
