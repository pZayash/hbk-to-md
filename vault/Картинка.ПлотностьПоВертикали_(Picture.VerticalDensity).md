Картинка (Picture)

ПлотностьПоВертикали (VerticalDensity)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПлотностьПоВертикали()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Если картинка создана с вариантами, то возвращает информацию по текущему варианту.  
Если картинка является частью коллекции картинок, возвращает [Неопределено](lang__def_Undefined.md).  
На мобильной платформе и мобильных клиентах всегда возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает плотность изображения по вертикали - число точек на дюйм (DPI).  
Важно! Если формат картинки не предоставляет эту информацию, то будет возвращена стандартная плотность, равная 72 точкам на дюйм (разрешение экрана).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
