ДанныеКвитанцииВстроеннойПокупки (InAppPurchaseReceiptData)

АктивнаяПодписка (ActiveSubscription)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак того, является ли подписка активной в данный момент времени. Используется только для автоматически возобновляемых подписок.  
Важно! При работе под управлением Android ОС может содержать Ложь, если пользователи отменил подписку в сервисе Google Play. При этом спрок подписки с момента ее последнего приобретения может еще не закончиться. Следует учесть данный фактор и обеспечить дополнительную проверку по датам.

Доступность:

Мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
