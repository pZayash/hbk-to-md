Описание:

Используется для доступа к свойствам, методам и событиям диаграммы Ганта в целом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: GanttChart.

См. также:

[ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md), метод [УстановитьПолныйИнтервал](ДиаграммаГанта.УстановитьПолныйИнтервал_(GanttChart.SetWholeInterval).md)

Доступен, начиная с версии 8.1.
