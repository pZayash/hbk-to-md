Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для набора записей регистра сведений. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержат значение типа [РегистрНакопленияЗапись.](РегистрНакопленияЗапись.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияЗапись.Имя_регистра_накопления.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
