Расширение формы клиентского приложения для набора записей (Managed form extension for record set)

ПередЗаписьюНаСервере (BeforeWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередЗаписьюНаСервере(<Отказ>, <ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение Истина, запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: Ложь.

<ТекущийОбъект>

Тип: [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>, [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)>, [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)>, [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)>.   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура.md).   
Структура, содержащая параметры записи.

Описание:

Вызывается перед записью объекта на сервере.   
Можно отменить запись. Выполняется вне транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
