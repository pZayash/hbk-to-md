ХранилищеСертификатовКриптографии (CryptoCertificateStore)

ДобавитьАсинх (AddAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ДобавитьАсинх(<Сертификат>)

Параметры:

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md).   
Добавляемый сертификат.  
В мобильном приложении вместо сертификата может использоваться контейнер ключей криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Добавляет сертификат в хранилище.  
В случае ошибки будет сгененировано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе на iOS нельзя несколько раз добавить один и тот же сертификат или контейнер ключей криптографии, даже в хранилища с разным типом.
