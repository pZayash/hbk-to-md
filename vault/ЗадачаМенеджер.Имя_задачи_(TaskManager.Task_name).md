Описание:

Предназначен для управления задачей, как объектом конфигурации. С помощью этого объекта осуществляется поиск элементов, создание новых элементов и групп, работа с формами и макетами задач. Доступ к объекту осуществляется через свойства объекта [ЗадачиМенеджер](ЗадачиМенеджер_(TasksManager).md). Полное имя типа объекта определяется с учетом имени задачи конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

Пример:

|  |
| --- |
| // Создание и запись новой задачи  НовЗадача = Задачи.Задача.СоздатьЗадачу(); НовЗадача.Дата = ТекущаяДата(); // Установка прочих реквизитов задачи  НовЗадача.Записать(); |
