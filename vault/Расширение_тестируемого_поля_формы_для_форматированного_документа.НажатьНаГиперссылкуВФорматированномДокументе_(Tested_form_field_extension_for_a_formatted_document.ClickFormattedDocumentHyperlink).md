Расширение тестируемого поля формы для форматированного документа (Tested form field extension for a formatted document)

НажатьНаГиперссылкуВФорматированномДокументе (ClickFormattedDocumentHyperlink)

Доступен, начиная с версии 8.3.25.

Синтаксис:

НажатьНаГиперссылкуВФорматированномДокументе(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md).   
Представление или порядковый номер гиперссылки в документе.

Описание:

Генерирует событие обработки нажатия ссылки в форматированном документе.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
