# БизнесПроцессОбъект.<Имя бизнес-процесса>.ПолучитьСсылкуНового

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [БизнесПроцессСсылка.\<Имя бизнес-процесса\>](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](БизнесПроцессОбъект.Имя_бизнес-процесса.УстановитьСсылкуНового_(BusinessProcessObject.Business_process_name.SetNewObjectRef).md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса_(BusinessProcessManager.Business_process_name).md), метод [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку_(BusinessProcessManager.Business_process_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
