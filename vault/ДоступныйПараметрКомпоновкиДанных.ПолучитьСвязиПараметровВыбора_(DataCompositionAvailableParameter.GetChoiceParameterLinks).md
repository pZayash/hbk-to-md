ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

ПолучитьСвязиПараметровВыбора (GetChoiceParameterLinks)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьСвязиПараметровВыбора()

Возвращаемое значение:

Тип: [СвязиПараметровВыбораКомпоновкиДанных](СвязиПараметровВыбораКомпоновкиДанных_(DataCompositionChoiceParameterLinks).md).   

Описание:

Связи параметров выбора используются для выбора значения поля или параметра.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
