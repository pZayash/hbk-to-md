МенеджерСредствПередачиДанныхНаУстройстве (DeviceDataSharingToolsManager)

ПоделитьсяДанными (ShareData)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПоделитьсяДанными(<Данные>, <ТипСодержимогоДляВыбораПриложения>, <Заголовок>)

Параметры:

<Данные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md), [Картинка](Картинка_(Picture).md), [Массив](Массив_(Array).md).   
Данные для отправки. У объекта [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md) должно быть указано свойство [РасширениеФайла](ДанныеМультимедиа.РасширениеФайла_(MultimediaData.FileExtention).md). Объект [Картинка](Картинка_(Picture).md) должен быть известного формата. Иначе генерируется исключение.

<ТипСодержимогоДляВыбораПриложения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет тип передаваемых данных в стандарте MIME на устройствах под управлением ОС Android.   
Значение по умолчанию: "\*/\*".

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Устанавливает заголовок в окне выбора на устройствах под управлением ОС Android.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если найдены приложения, которые могут принимать данные из нашего приложения. Иначе возвращает [Ложь](lang__def_BooleanFalse.md). На iOS всегда [Истина](lang__def_BooleanTrue.md).

Описание:

Выводит системный диалог выбора приложений, которые могут принять данные из приложения 1С:Предприятия, что позволяет пользователям делиться информацией быстро и просто, используя пользовательские приложения и сервисы.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ВыбратьПоСсылкам](ПланСчетовМенеджер.Имя_плана_счетов.ВыбратьПоСсылкам_(ChartOfAccountsManager.Chart_of_accounts_name.SelectByRefs).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.

Описание изменено в версии 8.3.26.
