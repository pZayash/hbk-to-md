КомандаФормы (FormCommand)

Картинка (Picture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Содержит картинку команды.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Изменение доступно только на сервере.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.2.
