ВариантИспользованииБазыДанныхКопии (CopyDatabaseUseVariant)

ИспользуетсяЭтойКопией (UsedByThisCopy)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Базу данных использует копия, для которой был вызван метод.  
Получить информацию об использовании базы данных копии можно только с помощью вариантов метода [ПолучитьИнформациюОбИспользованииБазыДанныхКопии](МенеджерКопийБазыДанных.ПолучитьИнформациюОбИспользованииБазыДанныхКопии_(DatabaseCopiesManager.GetCopyDatabaseUseInformation).md) "По имени" и "По копии".

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
