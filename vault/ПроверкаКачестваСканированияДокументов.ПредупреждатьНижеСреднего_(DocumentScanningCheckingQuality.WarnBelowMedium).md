ПроверкаКачестваСканированияДокументов (DocumentScanningCheckingQuality)

ПредупреждатьНижеСреднего (WarnBelowMedium)

Доступен, начиная с версии 8.3.23.

Описание:

Пользователь может принять документ любого качества.  
Пользователю показываются предупреждения о качестве документа, чье обобщенное качество ниже Среднего (меньше 4).

Использование в версии:

Доступен, начиная с версии 8.3.23.
