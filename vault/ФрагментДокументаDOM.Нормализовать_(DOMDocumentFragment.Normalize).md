ФрагментДокументаDOM (DOMDocumentFragment)

Нормализовать (Normalize)

Доступен, начиная с версии 8.1.

Синтаксис:

Нормализовать()

Описание:

Приводит все узлы [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) на полной глубине поддерева данного узла, включая узлы [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), в "нормальную" форму, где только структуры (например, элементы, комментарии, инструкции обработки, секции CDATA и ссылки на сущности) разделяются текстовыми узлами, т.е. смежные текстовые узлы отсутствуют.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
