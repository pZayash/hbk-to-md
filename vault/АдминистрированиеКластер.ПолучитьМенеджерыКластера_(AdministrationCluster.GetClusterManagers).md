АдминистрированиеКластер (AdministrationCluster)

ПолучитьМенеджерыКластера (GetClusterManagers)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьМенеджерыКластера()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеМенеджерКластера](АдминистрированиеМенеджерКластера_(AdministrationClusterManager).md).

Описание:

Получает список менеджеров, зарегистрированных в кластере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
