# Глобальный контекст.ПриВставкеИзБуфераОбмена

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [События](События.md) › [События приложения](События_приложения.md)

Глобальный контекст (Global context)

ПриВставкеИзБуфераОбмена (OnPasteFromClipboard)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПриВставкеИзБуфераОбмена(<Значение>)

Параметры:

<Значение>

Тип: [Картинка](Картинка_(Picture).md), [Массив](Массив_(Array).md).   
Значение, полученное из буфера обмена.  
Картинка или массив элементов типа [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md).

Описание:

Вызывается при вставке картинки или файла (файлов) из буфера обмена в клиентском приложении.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.24.
