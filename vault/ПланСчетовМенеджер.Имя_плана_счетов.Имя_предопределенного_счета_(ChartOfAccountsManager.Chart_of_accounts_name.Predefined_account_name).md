ПланСчетовМенеджер.<Имя плана счетов> (ChartOfAccountsManager.<Chart of accounts name>)

<Имя предопределенного счета> (<Predefined account name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   
Содержит ссылки на предопределенный объект плана счетов. Доступ к значению осуществляется по имени, как они заданы в Конфигураторе. Имена имеют только предопределенные счета. Количество таких свойств задается списком предопределенных объектов плана счетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
