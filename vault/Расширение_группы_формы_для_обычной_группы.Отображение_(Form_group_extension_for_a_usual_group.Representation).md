# Расширение группы формы для обычной группы.Отображение

Расширение группы формы для обычной группы (Form group extension for a usual group)

Отображение (Representation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеОбычнойГруппы](ОтображениеОбычнойГруппы_(UsualGroupRepresentation).md).   
Отображение группы в форме.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Если группа отобрадается как карточка то любое значение интерпретируется всегда как "Обычное выделение".

Использование в версии:

Доступен, начиная с версии 8.2.
