КоллекцияЗамещающихЭлементовПланировщика (PlannerReplacementItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<ДатаЗамещения>)

Параметры:

<ДатаЗамещения> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата, элемент расписания для которой будет замещен новым элементом.  
Новый элемент добавляется в коллекцию замещающих элементов.

Возвращаемое значение:

Если указанный элемент отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск замещающего элемента планировщика по указанному значению свойства [Значение](ЭлементПланировщика.Значение_(PlannerItem.Value).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
