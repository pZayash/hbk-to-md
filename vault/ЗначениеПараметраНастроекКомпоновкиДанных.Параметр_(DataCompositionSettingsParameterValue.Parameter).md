ЗначениеПараметраНастроекКомпоновкиДанных (DataCompositionSettingsParameterValue)

Параметр (Parameter)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных_(DataCompositionParameter).md).   
Параметр, значение которого предоставляется.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может быть изменен только в том случае, если владеющая им коллекция значений параметров не связана с доступными параметрами. В противном случае генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
