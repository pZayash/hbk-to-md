Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Описание:

Записывает новый или измененный объект плана видов характеристик. Инициирует события [ПередЗаписью](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередЗаписью.md) и [ПриЗаписи](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПриЗаписи.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
