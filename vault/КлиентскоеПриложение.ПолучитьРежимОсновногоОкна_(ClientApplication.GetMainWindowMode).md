КлиентскоеПриложение (ClientApplication)

ПолучитьРежимОсновногоОкна (GetMainWindowMode)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьРежимОсновногоОкна()

Возвращаемое значение:

Тип: [РежимОсновногоОкнаКлиентскогоПриложения](РежимОсновногоОкнаКлиентскогоПриложения_(MainClientApplicationWindowMode).md).   
Текущий режим основного окна.

Описание:

Возвращает текущий режим основного окна клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Не доступен в режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md).

[КлиентскоеПриложение](КлиентскоеПриложение_(ClientApplication).md), метод [УстановитьРежимОсновногоОкна](КлиентскоеПриложение.УстановитьРежимОсновногоОкна_(ClientApplication.SetMainWindowMode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
