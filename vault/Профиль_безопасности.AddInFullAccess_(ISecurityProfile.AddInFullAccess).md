Доступен, начиная с версии 8.3.3.

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешено использование произвольных внешних компонентов.   
[Истина](lang__def_BooleanTrue.md) – разрешено использование произвольных внешних компонентов. [Ложь](lang__def_BooleanFalse.md) – допустимо использование только внешних компонентов, перечисленных в списке разрешенных внешних компонентов профиля.

Доступность:

Интеграция.

Доступен, начиная с версии 8.3.3.
