ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

ОбработкаФормированияПоВерсииИсторииДанных (GenerateFromDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОбработкаФормированияПоВерсииИсторииДанных(<ДанныеВерсии>, <НомерВерсии>, <ИсключитьДанные>, <СтандартнаяОбработка>)

Параметры:

<ДанныеВерсии>

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура.md).   
Фиксированная структура с данными версии, по которой выполняется формирование.

<НомерВерсии>

Тип: Число.   
Номер версии, по которой выполняется формирование.

<ИсключитьДанные>

Тип: [Массив](Массив.md).   
Массив с именами полей и табличных частей, которые будут исключены из стандартной обработки.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

Описание:

Возникает при формировании объекта по версии истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерИсторииДанных](МенеджерИсторииДанных.md), метод [СформироватьПоВерсии](МенеджерИсторииДанных.СформироватьПоВерсии.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
