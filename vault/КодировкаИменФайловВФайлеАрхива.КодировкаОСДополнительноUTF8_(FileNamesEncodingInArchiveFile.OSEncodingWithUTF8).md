КодировкаИменФайловВФайлеАрхива (FileNamesEncodingInArchiveFile)

КодировкаОСДополнительноUTF8 (OSEncodingWithUTF8)

Доступен, начиная с версии 8.3.26.

Описание:

Имена файлов внутри файла архива будут записаны в двух кодировках: текущая кодировка операционной системы и UTF8.

Использование в версии:

Доступен, начиная с версии 8.3.26.
