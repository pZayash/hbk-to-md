# РегистрыНакопленияМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры накопления](Регистры_накопления.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[РегистрНакопленияМенеджер.\<Имя регистра накопления\>](РегистрНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegisterManager.Accumulation_register_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров накопления.  
Возможно обращение к менеджеру регистра накопления посредством оператора [...]. В качестве аргумента передается имя регистра накопления, как оно задано в конфигурации.  

Свойства:

[\<Имя регистра накопления\> (\<Accumulation register name\>)](РегистрыНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegistersManager.Accumulation_register_name).md)  

Описание:

Предназначен для управления регистрами накопления и предоставляет доступ к значениям типа [РегистрНакопленияМенеджер.](РегистрНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegisterManager.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegisterManager.Accumulation_register_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [РегистрыНакопления](Глобальный_контекст.РегистрыНакопления_(Global_context.AccumulationRegisters).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [РегистрыНакопления](Глобальный_контекст.РегистрыНакопления_(Global_context.AccumulationRegisters).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
