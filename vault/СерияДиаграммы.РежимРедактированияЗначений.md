Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [РежимРедактированияЗначенийДиаграммы](РежимРедактированияЗначенийДиаграммы.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
