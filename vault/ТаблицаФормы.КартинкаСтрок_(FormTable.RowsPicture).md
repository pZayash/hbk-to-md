ТаблицаФормы (FormTable)

КартинкаСтрок (RowsPicture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Для получения картинки строк в таблице может быть указана колонка c типом [Число](lang__def_Number.md) или [Булево](lang__Булево_(Boolean).md). Если указана колонка с типом [Число](lang__def_Number.md), то значение в колонке представляет собой индекс в наборе картинок, задаваемых данным свойством. Если указана колонка с типом [Булево](lang__Булево_(Boolean).md), значение [Истина](lang__def_BooleanTrue.md) означает индекс 0, а значение [Ложь](lang__def_BooleanFalse.md) означает индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.2.
