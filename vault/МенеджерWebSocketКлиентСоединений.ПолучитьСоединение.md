МенеджерWebSocketКлиентСоединений (WebSocketClientConnectionsManager)

ПолучитьСоединение (GetConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьСоединение(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Идентификатор соединения.

Возвращаемое значение:

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md), Неопределено.   
Возвращает объект [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md) или  
Неопределено - если соединение отсутствует.

Описание:

Получает объект соединения с WebSocket-клиентом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
