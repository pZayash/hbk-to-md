РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Защита (Protection)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак защиты рисунка от изменений.  
[Истина](lang__def_BooleanTrue.md) - защита включена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Не применимо для рисунков типа [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
