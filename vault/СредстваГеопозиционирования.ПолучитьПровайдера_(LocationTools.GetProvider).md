СредстваГеопозиционирования (LocationTools)

ПолучитьПровайдера (GetProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьПровайдера(<Имя>, <ТолькоДоступного>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя провайдера геопозиционирования.

<ТолькоДоступного> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - информация о требуемом провайдере будет возвращена только если провайдер присутствует в системе и является доступным.  
[Ложь](lang__def_BooleanFalse.md) - информация о требуемом провайдере будет возвращена только если провайдер присутствует в системе.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования_(LocationProviderInformation).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - провайдер не найден.

Описание:

Получает информацию о провайдере геопозиционирования с заданным именем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
