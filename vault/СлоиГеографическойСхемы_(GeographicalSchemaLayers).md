# СлоиГеографическойСхемы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Свойства:

Описание:

Представляет собой коллекцию слоев, входящих в географическую схему.

Доступность:

Сервер, толстый клиент, внешнее соединение.
