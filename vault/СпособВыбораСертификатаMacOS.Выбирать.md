СпособВыбораСертификатаMacOS (MacOSCertificateSelectMode)

Выбирать (Choose)

Доступен, начиная с версии 8.3.13.

Описание:

При наличии более одного подходящего клиентского сертификата предоставлять выбор через системный диалог выбора сертификатов.

Использование в версии:

Доступен, начиная с версии 8.3.13.
