Расширение формы клиентского приложения для записи таблицы внешнего источника данных

Прочитать (Read)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Прочитать()

Описание:

Считывает запись по установленным значениям ключевых атрибутов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
