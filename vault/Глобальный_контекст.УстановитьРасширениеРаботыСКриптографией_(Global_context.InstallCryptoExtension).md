Глобальный контекст (Global context)

УстановитьРасширениеРаботыСКриптографией (InstallCryptoExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [НачатьУстановкуРасширенияРаботыСКриптографией](Глобальный_контекст.НачатьУстановкуРасширенияРаботыСКриптографией_(Global_context.BeginInstallCryptoExtension).md)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьРасширениеРаботыСКриптографией()

Описание:

Устанавливает расширение для работы с криптографией в браузер.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска толстого и тонкого клиентов, а также на клиенте мобильного приложения информирует о том, что расширение установлено.  
Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [НачатьУстановкуРасширенияРаботыСКриптографией](Глобальный_контекст.НачатьУстановкуРасширенияРаботыСКриптографией_(Global_context.BeginInstallCryptoExtension).md).
