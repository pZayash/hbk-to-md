Глобальный контекст (Global context)

НачатьУстановкуРасширенияРаботыСФайлами (BeginInstallFileSystemExtension)

Доступен, начиная с версии 8.3.3.

Синтаксис:

НачатьУстановкуРасширенияРаботыСФайлами(<ОписаниеОповещенияОЗавершении>)

Параметры:

<ОписаниеОповещенияОЗавершении> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения установки расширения работы с файлами со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Описание:

Устанавливает расширение работы с файлами.  
В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.  
Для клиента мобильного приложения информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
