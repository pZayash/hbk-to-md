БизнесПроцессСсылка.<Имя бизнес-процесса> (BusinessProcessRef.<Business process name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md).   

Описание:

Создает новый бизнес-процесс копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| Объект = ВыбБизнесПроцесс.Скопировать(); |
