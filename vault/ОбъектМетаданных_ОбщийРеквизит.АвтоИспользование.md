ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

АвтоИспользование (AutoUse)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [АвтоИспользованиеОбщегоРеквизита](АвтоИспользованиеОбщегоРеквизита.md).   
Содержит способ включения в состав объекта метаданных из соответствующего элемента состава общего реквизита ([Состав](ОбъектМетаданных_ОбщийРеквизит.Состав.md)), если у элемента включение в состав установлено автоматическим.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
