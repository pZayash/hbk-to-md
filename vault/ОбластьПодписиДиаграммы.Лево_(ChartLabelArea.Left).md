ОбластьПодписиДиаграммы (ChartLabelArea)

Лево (Left)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит положение левой границы.   
Может содержать значения от 0 до 1.   
0 - привязана к левому краю, 1 - к правому.  
Используется вместе со свойством [УказываетсяЛевоИВерх](ПоложениеПодписейКДиаграмме.УказываетсяЛевоИВерх_(ChartLabelLocation.TopAndLeftSpecified).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

Использование в версии:

Доступен, начиная с версии 8.3.10.
