# Расширение поля ввода задачи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора задачи в элементе управления [ПолеВвода](ПолеВвода_(TextBox).md) по введенному тексту. По умолчанию поиск осуществляется по полям "Номер" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
