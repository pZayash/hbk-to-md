# Расширение поля формы для поля форматированного документа.Высота

Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту поля. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
