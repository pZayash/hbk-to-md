Расширение табличного поля списка задач (Task list table box extension)

ИзменятьСпособРедактирования (ChangeEditingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Устанавливает доступность интерактивного способа редактирования объекта (в строке или в отдельной форме).

Доступность:

Толстый клиент.

Примечание:

Истина - интерактивная установка способа редактирования объекта доступна пользователю; Ложь - не доступна.

Использование в версии:

Доступен, начиная с версии 8.0.
