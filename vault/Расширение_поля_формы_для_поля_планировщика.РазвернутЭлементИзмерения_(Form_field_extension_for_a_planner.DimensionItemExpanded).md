# Расширение поля формы для поля планировщика.РазвернутЭлементИзмерения

Расширение поля формы для поля планировщика (Form field extension for a planner)

РазвернутЭлементИзмерения (DimensionItemExpanded)

Доступен, начиная с версии 8.3.17.

Синтаксис:

РазвернутЭлементИзмерения(<ЗначенияИзмерений>)

Параметры:

<ЗначенияИзмерений> (обязательный)

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Фиксированное соответствие, однозначно определяющее элемент измерения, состояние которого нужно получить. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает значение [Истина](lang__def_BooleanTrue.md), если элемент измерения развернут, и [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Определяет, развернут ли элемент измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
