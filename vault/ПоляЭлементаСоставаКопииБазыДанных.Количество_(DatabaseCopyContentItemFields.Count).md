ПоляЭлементаСоставаКопииБазыДанных (DatabaseCopyContentItemFields)

Количество (Count)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество полей элемента состава.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
