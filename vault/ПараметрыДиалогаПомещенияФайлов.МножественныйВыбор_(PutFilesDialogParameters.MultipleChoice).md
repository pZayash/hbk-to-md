ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

МножественныйВыбор (MultipleChoice)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Значение свойства определяет, разрешен ли в диалоге выбор нескольких файлов:

- [Истина](lang__def_BooleanTrue.md) – разрешается,
- [Ложь](lang__def_BooleanFalse.md) – не разрешается.

  
Если значение [Неопределено](lang__def_Undefined.md), то в методе [НачатьПомещениеФайлаНаСервер](Глобальный_контекст.НачатьПомещениеФайлаНаСервер_(Global_context.BeginPutFileToServer).md) значение интерпретируется как [Ложь](lang__def_BooleanFalse.md), а в методе [НачатьПомещениеФайловНаСервер](Глобальный_контекст.НачатьПомещениеФайловНаСервер_(Global_context.BeginPutFilesToServer).md) значение интерпретируется как [Истина](lang__def_BooleanTrue.md).  
Значению по умолчанию: [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
