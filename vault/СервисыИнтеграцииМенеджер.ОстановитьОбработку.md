СервисыИнтеграцииМенеджер (IntegrationServicesManager)

ОстановитьОбработку (StopProcessing)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ОстановитьОбработку()

Описание:

Останавливает системные фоновые задания, обрабатывающие сервисы интеграции.   
Требует наличие права "Администрирование".  
Не допускается запуск, если в сеансе установлены разделители.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
