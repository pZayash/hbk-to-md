# ПланСчетовСсылка.<Имя плана счетов>.ПринадлежитЭлементу

ПланСчетовСсылка.<Имя плана счетов> (ChartOfAccountsRef.<Chart of accounts name>)

ПринадлежитЭлементу (BelongsToItem)

Доступен, начиная с версии 8.0.

Синтаксис:

ПринадлежитЭлементу(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   
Счет, для которого определяется принадлежность.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - счет подчинен другому счету; [Ложь](lang__def_BooleanFalse.md) - в противном случае.  
Если в качестве параметра указать сам счет, то [Ложь](lang__def_BooleanFalse.md).

Описание:

Определяет подчиненность счета другому счету (элементу) с учетом всех уровней иерархии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определяет подчиненность непосредственно на момент вызова метода путем последовательного считывания всех вышестоящих элементов.

Пример:

|  |
| --- |
| ПС = ПланыСчетов["Основной"]; Счет01   = ПС.ОсновныеСредства; Счет01\_1 = ПС.ОСвОрганизации; Счет02   = ПС.АмортизацияОсновныхСредств; Счет03   = ПС.ДоходныеВложенияВ\_МЦ; // ...  ВыборкаПоСубконто = РезультатЗапроса.Выбрать(ОбходРезультатаЗапроса.ПоГруппировкам, "Субконто1"); Пока ВыборкаПоСубконто.Следующий() Цикл     ВыборкаПоСчетам = ВыборкаПоСубконто.Выбрать(ОбходРезультатаЗапроса.ПоГруппировкам, "Счет");     Пока ВыборкаПоСчетам.Следующий() Цикл         Если ВыборкаПоСчетам.Счет = Счет01\_1 Тогда             // ...          ИначеЕсли ВыборкаПоСчетам.Счет.ПринадлежитЭлементу(Счет03) Тогда             // ...          КонецЕсли;         // ...      КонецЦикла; КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
