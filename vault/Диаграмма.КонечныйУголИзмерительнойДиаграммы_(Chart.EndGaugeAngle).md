Диаграмма (Chart)

КонечныйУголИзмерительнойДиаграммы (EndGaugeAngle)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит конечный угол шкалы измерительной диаграммы. Задается в градусах от положительной оси х в диапазоне 0 - 360.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Должен быть меньше значения свойства [НачальныйУголИзмерительнойДиаграммы](Диаграмма.НачальныйУголИзмерительнойДиаграммы_(Chart.BeginGaugeAngle).md).

Использование в версии:

Доступен, начиная с версии 8.0.
