РежимСверткиЭлементаУправления (ControlCollapseMode)

Низ (Bottom)

Доступен, начиная с версии 8.0.

Описание:

Элемент управления свертывается к своей нижней стороне (верхняя сторона прижимается к нижней).

Использование в версии:

Доступен, начиная с версии 8.0.
