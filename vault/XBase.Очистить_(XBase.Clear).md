# XBase.Очистить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XBase](XBase.md) › [XBase](XBase_(XBase).md) › [Методы](XBase__Методы.md)

XBase (XBase)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Очищает свойства объекта [XBase](XBase_(XBase).md), соответствующие полям таблицы базы данных.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Очищает значения всех полей первой записи  xB.Первая(); xB.Очистить(); xB.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
