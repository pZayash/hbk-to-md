ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Добавить (Add)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Добавить(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя вставляемой копии.

Возвращаемое значение:

Тип: [ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных_(UsedDatabaseCopy).md).   

Описание:

Добавляет используемую копию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
