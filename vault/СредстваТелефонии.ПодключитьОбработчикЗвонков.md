СредстваТелефонии (TelephonyTools)

ПодключитьОбработчикЗвонков (AttachCallsHandler)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПодключитьОбработчикЗвонков(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: Строка, [ОписаниеОповещения](ОписаниеОповещения.md).   
Имя процедуры, подключаемой в качестве обработчика.

Описание:

Выполняет подключение обработчика звонков.  
Обработчик вызывается со следующими параметрами:

- НомерТелефона - тип Строка. Номер телефона, с которого приходит вызов или на который отправляется вызов.
- Дата - тип Дата. Время возникновения события.
- ВариантСобытия - тип [ВариантСобытияЗвонкаСредствТелефонии](ВариантСобытияЗвонкаСредствТелефонии.md). Одно из значений перечисления.
- ТипЗвонка - тип [ТипЗвонкаЖурналаЗвонков](ТипЗвонкаЖурналаЗвонков.md): входящий, исходящий или пропущенный (если вызов был завершен при входящем звонке).
- Дополнительный параметр (если используется [ОписаниеОповещения](ОписаниеОповещения.md)).

Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяОбработкаЗвонков](СредстваТелефонии.ПоддерживаетсяОбработкаЗвонков.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).
