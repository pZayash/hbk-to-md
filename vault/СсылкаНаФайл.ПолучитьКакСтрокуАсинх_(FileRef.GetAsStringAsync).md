СсылкаНаФайл (FileRef)

ПолучитьКакСтрокуАсинх (GetAsStringAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьКакСтрокуАсинх(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста. Если кодировка не установлена, то используется [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат обещания - [Строка](lang__def_String.md).

Описание:

Асинхронная функция. Возвращает содержимое файла в виде строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.24.
