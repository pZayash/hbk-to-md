ДиаграммаГанта (GanttChart)

ЕдиницаПериодическогоВарианта (PeriodicVariantUnit)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипЕдиницыШкалыВремени](ТипЕдиницыШкалыВремени_(TimeScaleUnitType).md).   
Единица шкалы времени периодического интервала. Значение интервала в выбранных данным свойством единицах указывается в свойстве [КратностьПериодическогоВарианта](ДиаграммаГанта.КратностьПериодическогоВарианта_(GanttChart.PeriodicVariantRepetition).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Используется, если для свойства ПоддержкаМасштаба установлено значение Период.

Использование в версии:

Доступен, начиная с версии 8.0.
