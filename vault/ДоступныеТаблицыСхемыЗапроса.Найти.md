ДоступныеТаблицыСхемыЗапроса (QuerySchemaAvailableTables)

Найти (Find)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя таблицы, которую требуется найти.

Возвращаемое значение:

Тип: [ГруппаДоступныхТаблицСхемыЗапроса](ГруппаДоступныхТаблицСхемыЗапроса.md), [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса.md).

Находит доступную таблицу, группу доступных таблиц или вложенную таблицу по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
