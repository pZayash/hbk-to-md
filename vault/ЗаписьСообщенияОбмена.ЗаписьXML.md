ЗаписьСообщенияОбмена (ExchangeMessageWriter)

ЗаписьXML (XMLWriter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЗаписьУзловDOM](ЗаписьУзловDOM.md), [ЗаписьFastInfoset](ЗаписьFastInfoset.md), [ЗаписьXML](ЗаписьXML.md), Неопределено.   
Содержит объект, с помощью которого осуществляется запись сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу [НачатьЗапись](ЗаписьСообщенияОбмена.НачатьЗапись.md) и после обращения к методу [ЗакончитьЗапись](ЗаписьСообщенияОбмена.ЗакончитьЗапись.md) или [ПрерватьЗапись](ЗаписьСообщенияОбмена.ПрерватьЗапись.md) значение данного свойства равно Неопределено.

Использование в версии:

Доступен, начиная с версии 8.0.
