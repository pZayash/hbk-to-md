ТипСвязиВложенияPDF (PDFAttachmentRelationshipType)

Дополнение (Supplement)

Доступен, начиная с версии 8.3.16.

Описание:

Спецификация файла представляет дополнительную репрезентацию исходного источника или данных, которые могут быть использованы (например, формулы, таблицы отчетов).

Использование в версии:

Доступен, начиная с версии 8.3.16.
