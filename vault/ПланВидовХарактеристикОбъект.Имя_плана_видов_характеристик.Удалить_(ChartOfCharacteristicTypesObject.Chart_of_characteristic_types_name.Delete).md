ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет план видов характеристик из базы данных. Вызывает преопределенную процедуру события [ПередУдалением](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередУдалением_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.BeforeDelete).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для иерархического плана видов характеристик также удаляются все подчиненные элементы.

Использование в версии:

Доступен, начиная с версии 8.0.
