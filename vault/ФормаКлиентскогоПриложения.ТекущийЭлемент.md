ФормаКлиентскогоПриложения (ClientApplicationForm)

ТекущийЭлемент (CurrentItem)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГруппаФормы](ГруппаФормы.md), [ТаблицаФормы](ТаблицаФормы.md), [ПолеФормы](ПолеФормы.md), [КнопкаФормы](КнопкаФормы.md), Неопределено.   
Содержит текущий активный элемент формы клиентского приложения.  
  
Если на форме нет ни одного элемента, свойство содержит объект Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
