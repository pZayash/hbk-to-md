ДокументHTML (HTMLDocument)

СоздатьЭлемент (CreateElement)

Доступен, начиная с версии 8.2.

Синтаксис:

СоздатьЭлемент(<ИмяЭлемента>)

Параметры:

<ИмяЭлемента> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя создаваемого элемента.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML_(HTMLElement).md).   
Новый узел HTML- [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), в свойство ИмяУзла которого устанавливается значение параметра <ИмяЭлемента>. Свойства ЛокальноеИмя и Префикс не устанавливаются.

Описание:

Создает новый узел HTML элемента XML. Если известны атрибуты, имеющие значения по умолчанию, то они автоматически добавляются в элемент.  
Причины вызова исключений:

- Имя элемента содержит символы, недопустимые стандартом XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
