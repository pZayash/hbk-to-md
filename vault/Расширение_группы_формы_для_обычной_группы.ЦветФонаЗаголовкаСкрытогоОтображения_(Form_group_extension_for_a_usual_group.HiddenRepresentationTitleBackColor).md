Расширение группы формы для обычной группы (Form group extension for a usual group)

ЦветФонаЗаголовкаСкрытогоОтображения (HiddenRepresentationTitleBackColor)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Если группа [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая_(UsualGroupBehavior.Collapsible).md), используется в качестве цвета фона заголовка в свернутом состоянии.  
Если группа [Всплывающая](ПоведениеОбычнойГруппы.Всплывающая_(UsualGroupBehavior.PopUp).md), используется в качестве цвета фона заголовка в любом состоянии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

По умолчанию - [АвтоЦвет](ВидЦвета.АвтоЦвет_(ColorType.AutoColor).md), который трактуется как прозрачный цвет.

Использование в версии:

Доступен, начиная с версии 8.3.12.
