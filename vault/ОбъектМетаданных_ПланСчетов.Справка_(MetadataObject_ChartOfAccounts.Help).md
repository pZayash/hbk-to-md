ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

Справка (Help)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
При запуске Конфигуратора в пакетном режиме из командной строки существует возможность выгрузить тексты справки. Для этого следует использовать параметр запуска /DumpConfigFiles с ключом –Help.

Использование в версии:

Доступен, начиная с версии 8.1.
