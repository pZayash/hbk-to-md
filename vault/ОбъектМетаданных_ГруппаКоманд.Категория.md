ОбъектМетаданных: ГруппаКоманд (MetadataObject: CommandGroup)

Категория (Category)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КатегорияГруппыКоманд](КатегорияГруппыКоманд.md).   
Категория, к которой принадлежит группа команд. Устанавливает расположение группы команд в командном интерфейсе приложения. Может принимать значения:

- Панель навигации;
- Панель навигации формы;
- Панель действий;
- Командная панель формы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
