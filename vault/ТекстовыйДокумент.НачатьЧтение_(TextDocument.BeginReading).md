ТекстовыйДокумент (TextDocument)

НачатьЧтение (BeginReading)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЧтение(<ОписаниеОповещения>, <ИмяФайла>, <ТипФайла>, <РазделительСтрок>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения чтения файла со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя текстового файла.

<ТипФайла> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Указывается кодировка текста в открываемом файле.   
В случае, если параметр не задан, формат текста будет определен автоматически (если он хранится в файле с сигнатурой BOM в начале). Если автоматически определить формат файла не удается, файл будет открыт в кодировке [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).  
  
Тип файла может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Задается строка, являющаяся разделителем строк, из которой при чтении файла будет сформирован символ [ПС](Символы.ПС_(Chars.LF).md).   
Если строка пустая, то разделители строк при чтении не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК_(Chars.CR).md);
- [ПС](Символы.ПС_(Chars.LF).md);
- [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание:

Вызывает начало чтения текстового документа из файла.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
