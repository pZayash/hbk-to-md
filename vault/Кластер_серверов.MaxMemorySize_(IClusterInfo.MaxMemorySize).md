# Кластер серверов.MaxMemorySize

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Кластер серверов](Кластер_серверов_(IClusterInfo).md) › [Свойства](Кластер_серверов__Свойства.md)

Кластер серверов (IClusterInfo)

MaxMemorySize (MaxMemorySize)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет объем виртуального адресного пространства (в килобайтах), занятого рабочим процессом, превышение которого приведет к перезапуску рабочего процесса.

Доступность:

Интеграция.

Примечание:

Если значение равно 0, то объем занятого процессом виртуального адресного пространства не ограничен.

Использование в версии:

Доступен, начиная с версии 8.2.
