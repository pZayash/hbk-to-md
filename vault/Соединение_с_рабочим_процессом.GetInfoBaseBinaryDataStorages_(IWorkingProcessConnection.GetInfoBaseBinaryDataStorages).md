Соединение с рабочим процессом (IWorkingProcessConnection)

GetInfoBaseBinaryDataStorages (GetInfoBaseBinaryDataStorages)

Доступен, начиная с версии 8.3.27.

Синтаксис:

GetInfoBaseBinaryDataStorages(<InfoBase>)

Параметры:

<InfoBase> (обязательный)

Тип: [Информационная база](Информационная_база_(IInfoBaseInfo).md).   
Информационная база.

Возвращаемое значение:

Тип: COMSafeArray.   
Массив элементов типа [Хранилище двоичных данных](Хранилище_двоичных_данных_(IBinaryDataStorageInfo).md).

Описание:

Возвращает список хранилищ двоичных данных для информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
