РежимОбработкиПодкаталоговФайлаАрхива (ArchiveFileSubDirProcessingMode)

ОбрабатыватьРекурсивно (ProcessRecursively)

Доступен, начиная с версии 8.3.26.

Описание:

При задании маски в качестве пути для архивации подкаталоги также будут рекурсивно обработаны.

Использование в версии:

Доступен, начиная с версии 8.3.26.
