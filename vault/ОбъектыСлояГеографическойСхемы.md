Доступен, начиная с версии 8.1.

Элементы коллекции:

[ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Получить (Get)](ОбъектыСлояГеографическойСхемы.Получить.md)  
[Удалить (Delete)](ОбъектыСлояГеографическойСхемы.Удалить.md)  

Описание:

Предоставляет доступ к коллекции объектов, входящих в слой географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
