Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<ЭлементСписка>)

Параметры:

<ЭлементСписка>

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).   
Выбранный элемент списка.

Возникает при выборе строки из поля списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
