ЗаписьДанных (DataWriter)

НачатьСбросБуферов (BeginFlush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьСбросБуферов(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Начинает сбрасывать все внутренние буферы в целевой поток, после чего будет вызван метод [СброситьБуферы](Поток.СброситьБуферы_(Stream.Flush).md) целевого потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [СброситьБуферы](ЗаписьДанных.СброситьБуферы_(DataWriter.Flush).md)  
[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [СброситьБуферыАсинх](ЗаписьДанных.СброситьБуферыАсинх_(DataWriter.FlushAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
