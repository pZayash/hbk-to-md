ДействиеЭлементаПланировщика (PlannerItemAction)

Положение (Location)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеДействияЭлементаПланировщика](ПоложениеДействияЭлементаПланировщика.md).   
Положение действия внутри элемента планировщика. Действия отображаются друг за другом, если есть пространство.  
Значение по умолчанию: [ПослеТекста](ПоложениеДействияЭлементаПланировщика.ПослеТекста.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
