КомандаФормы (FormCommand)

ИспользуемаяТаблица (AssociatedTable)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ТаблицаФормы](ТаблицаФормы.md).   
Используется вместе со свойством [ИспользованиеТекущейСтроки](КомандаФормы.ИспользованиеТекущейСтроки.md) для указания таблицы, текущая строка которой используется.  
Таблица формы, которая устанавливается значением данного свойства, должна относиться к той же форме, что и владелец свойства. При выборе таблицы из другой формы, будет сгенерировано исключение.  
Данную особенность следует учитывать при использовании метода [ЗаполнитьЗначенияСвойств](Глобальный_контекст.ЗаполнитьЗначенияСвойств.md)для объектов разных форм.

Доступность:

Мобильный клиент, сервер, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
