Описание:

Предназначен для управления режимом отображения серии в легенде диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ShowInChartLegend.

Использование в версии:

Доступен, начиная с версии 8.3.10.
