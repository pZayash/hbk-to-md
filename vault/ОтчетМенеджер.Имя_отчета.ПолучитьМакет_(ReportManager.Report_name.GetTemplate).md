# ОтчетМенеджер.<Имя отчета>.ПолучитьМакет

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Отчеты](Отчеты.md) › [ОтчетМенеджер.<Имя отчета>](ОтчетМенеджер.Имя_отчета_(ReportManager.Report_name).md) › [Методы](ОтчетМенеджер.Имя_отчета__Методы.md)

ОтчетМенеджер.<Имя отчета> (ReportManager.<Report name>)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет_(MetadataObject_Template).md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md); другой объект, который может быть макетом.   

Описание:

Получает макет отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| МакетОтчета = Отчеты.ОстаткиНоменклатуры.ПолучитьМакет("Основной"); |

Использование в версии:

Доступен, начиная с версии 8.0.
