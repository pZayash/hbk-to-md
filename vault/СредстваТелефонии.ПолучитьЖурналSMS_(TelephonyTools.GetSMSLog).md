СредстваТелефонии (TelephonyTools)

ПолучитьЖурналSMS (GetSMSLog)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПолучитьЖурналSMS()

Возвращаемое значение:

Тип: [ЖурналSMS](ЖурналSMS_(SMSLog).md), [Неопределено](lang__def_Undefined.md).   
Если журнал сообщений не доступен, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Получает журнал SMS-сообщений.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЖурналSMS](СредстваТелефонии.ПоддерживаетсяЖурналSMS_(TelephonyTools.SMSLogSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПоддерживаетсяЖурналSMS](СредстваТелефонии.ПоддерживаетсяЖурналSMS_(TelephonyTools.SMSLogSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.
