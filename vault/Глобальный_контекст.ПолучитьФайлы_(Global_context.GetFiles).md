Глобальный контекст (Global context)

ПолучитьФайлы (GetFiles)

Не рекомендуется использовать, начиная с версии 8.3.13.

Рекомендуется использовать:

- [НачатьПолучениеФайловССервера](Глобальный_контекст.НачатьПолучениеФайловССервера_(Global_context.BeginGetFilesFromServer).md)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьФайлы(<ПолучаемыеФайлы>, <ПолученныеФайлы>, <РасположениеФайлов>, <Интерактивно>)

Параметры:

<ПолучаемыеФайлы> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ОписаниеПередаваемогоФайла](ОписаниеПередаваемогоФайла_(TransferableFileDescription).md).  
Каждый объект описывает получаемый файл:

- [Хранение](ОписаниеПередаваемогоФайла.Хранение_(TransferableFileDescription.Location).md) - расположение данных во временном хранилище или в информационной базе;
- [Имя](ОписаниеПередаваемогоФайла.Имя_(TransferableFileDescription.Name).md) - это локальный путь, по которому должен быть расположен получаемый файл. Это может быть либо полный путь, либо часть пути, которая будет объединена со значением параметра <РасположениеФайлов>. Это также может быть имя файла или имя каталога (каталогов) + имя файла.

<ПолученныеФайлы> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ОписаниеПереданногоФайла](ОписаниеПереданногоФайла_(TransferredFileDescription).md).  
Каждый объект описывает получаемый файл:

- [Хранение](ОписаниеПереданногоФайла.Хранение_(TransferredFileDescription.Location).md) - расположение данных во временном хранилище или в информационной базе;
- [Имя](ОписаниеПереданногоФайла.Имя_(TransferredFileDescription.Name).md) - это локальный путь, по которому должен быть расположен получаемый файл.

<РасположениеФайлов> (необязательный)

Тип: [ДиалогВыбораФайла](ДиалогВыбораФайла_(FileDialog).md), [Строка](lang__def_String.md).   
Определяет общее расположение помещаемых файлов в локальной файловой системе.  
Может быть задан строкой либо заранее подготовленным диалогом выбора файлов в режиме выбора каталога или сохранения файлов.

<Интерактивно> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - задан интерактивный режим, в котором пользователю показывается диалог, в котором он может выбрать локальное расположение файлов.  
[Ложь](lang__def_BooleanFalse.md) - на сервер передается список файлов, описанный в параметре <ПолучаемыеФайлы>.  
В интерактивном режиме строка, переданная в параметре <РасположениеФайлов>, рассматривается как начальное значение для выбора пути.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - получен хотя бы один файл; [Ложь](lang__def_BooleanFalse.md) - получение всех файлов отменено пользователем.

Описание:

Получает набор файлов и сохраняет их в локальную файловую систему пользователя.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.  
В веб-клиенте при работе в Firefox при выполнении метода до окончания загрузки выводится модальное окно с отображением процента получения файла. Выполнение метода заканчивается после окончания получения файла.
