Расширение поля формы для поля ввода (Form field extension for a text box)

ДобавлениеМножественныхЗначений (MultipleValuesAdd)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ДобавлениеМножественныхЗначений(<Значения>, <СтандартнаяОбработка>)

Параметры:

<Значения>

Тип: [Массив](Массив.md).   
Массив значений, по которым будут сформированы или объекты [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md) и добавлены в конец списка значений, или объекты [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md) и добавлены в конец таблицы значений, если параметр <СтандартнаяОбработка> содержит значение Истина.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается при интерактивном добавлении множественных значений после событий [ОкончаниеВводаТекста](ПолеВвода.ОкончаниеВводаТекста.md) и [ОбработкаВыбора](Расширение_поля_формы_для_поля_ввода.ОбработкаВыбора.md), но до вызова обработчика [ПриИзменении](ПолеВвода.ПриИзменении.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
