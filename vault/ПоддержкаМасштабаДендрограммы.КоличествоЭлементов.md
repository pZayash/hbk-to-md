ПоддержкаМасштабаДендрограммы (DendrogramScaleKeeping)

КоличествоЭлементов (ItemCount)

Доступен, начиная с версии 8.0.

Описание:

Содержит заданное количество элементов, отображаемых в видимой области. Если элементов больше, используется прокрутка.

Использование в версии:

Доступен, начиная с версии 8.0.
