ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

ДоступноРедактированиеСообщения (MessageEditingAvailable)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность изменения отправленных сообщений в обсуждениях данной интеграции:

- [Истина](lang__def_BooleanTrue.md) - изменение возможно,
- [Ложь](lang__def_BooleanFalse.md) - в противном случае (для интеграции типа WhatsApp).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
