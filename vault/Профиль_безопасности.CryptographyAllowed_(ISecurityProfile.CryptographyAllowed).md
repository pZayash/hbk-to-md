Профиль безопасности (ISecurityProfile)

CryptographyAllowed (CryptographyAllowed)

Доступен, начиная с версии 8.3.7.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет разрешение на использование криптографического функционала (подпись, проверка подписи, шифрование, расшифрование, работа с хранилищем сертификатов, проверка сертификатов, извлечение сертификатов из подписи) при работе на сервере.  
На клиенте функции криптографии не блокируются.  
[Истина](lang__def_BooleanTrue.md) – исполнение разрешено. [Ложь](lang__def_BooleanFalse.md) – исполнение запрещено.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.7.
