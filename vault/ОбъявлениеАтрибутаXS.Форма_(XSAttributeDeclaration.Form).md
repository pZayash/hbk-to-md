ОбъявлениеАтрибутаXS (XSAttributeDeclaration)

Форма (Form)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ФормаПредставленияXS](ФормаПредставленияXS_(XSForm).md).   
Содержит форму представления свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет атрибут form объявления атрибута (<http://www.w3.org/TR/xmlschema-1/#element-attribute>) или элемента (<http://www.w3.org/TR/xmlschema-1/#element-element>).

Использование в версии:

Доступен, начиная с версии 8.1.
