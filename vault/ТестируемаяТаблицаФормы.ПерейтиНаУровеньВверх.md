ПерейтиНаУровеньВверх (GoOneLevelUp)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПерейтиНаУровеньВверх(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие.md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.  
Значение по умолчанию: Неопределено.

Описание:

Выполняет переход на уровень вверх в иерархическом списке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
