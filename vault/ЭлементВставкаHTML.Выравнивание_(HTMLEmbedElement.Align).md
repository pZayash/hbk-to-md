ЭлементВставкаHTML (HTMLEmbedElement)

Выравнивание (Align)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Выравнивание (атрибут align) вставляемого объекта относительно окружающего контекста в документе.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
