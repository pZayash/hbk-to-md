ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunAdditionalData)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Ключ>, <Значение>, <ИмяТипа>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Указывает ключ добавляемого элемента данных.

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md), [Булево](lang__Булево_(Boolean).md), [Картинка](Картинка_(Picture).md), [УчетнаяЗаписьКонтактов](УчетнаяЗаписьКонтактов_(ContactAccount).md), [ДанныеАдреса](ДанныеАдреса_(AddressData).md), [ДанныеМестоположения](ДанныеМестоположения_(LocationData).md), [Массив](Массив_(Array).md), [ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства](ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства_(MobileDeviceApplicationRunAdditionalData).md).   
Добавляемое значение дополнительных данных.  
Массив может содержать элементы одного из следующих типов: [Строка](lang__def_String.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md).

<ИмяТипа> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает имя типа добавляемого данного.

Возвращаемое значение:

Тип: [ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства](ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства_(MobileDeviceApplicationRunAdditionalDataItem).md).   

Описание:

Добавляет элемент дополнительных данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
