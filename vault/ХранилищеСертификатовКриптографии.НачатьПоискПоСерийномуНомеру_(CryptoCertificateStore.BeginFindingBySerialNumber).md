ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьПоискПоСерийномуНомеру (BeginFindingBySerialNumber)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПоискПоСерийномуНомеру(<ОписаниеОповещения>, <СерийныйНомер>, <Издатель>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения поиска со следующими параметрами:

- <Сертификаты> - массив объектов [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<СерийныйНомер> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Серийный номер искомого сертификата.

<Издатель> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка поиска издателя сертификата.   
Данные издателя должны содержать переданную в этом параметре строку.

Описание:

Начинает поиск сертификатов криптографии по серийному номеру и издателю.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
