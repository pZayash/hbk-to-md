ИнтернетПочтовоеСообщение (InternetMailMessage)

ОбработатьТексты (ProcessTexts)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработатьТексты()

Описание:

Производит предварительную обработку текстов письма, с возможностью последующего изменения содержимого текстов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
