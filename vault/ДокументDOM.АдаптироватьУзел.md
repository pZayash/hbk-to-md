ДокументDOM (DOMDocument)

АдаптироватьУзел (AdoptNode)

Доступен, начиная с версии 8.1.

Синтаксис:

АдаптироватьУзел()

Возвращаемое значение:

Тип: [ДокументDOM](ДокументDOM.md), Неопределено.   
Адаптированный узел или Неопределено, если адаптация не удалась.

Описание:

Выполняет адаптацию (заимствование) узла другого документа. Производит попытку заимствования узла из другого документа в данный.  
Если адаптация поддерживается для узла, то производится замена документа-владельца у данного узла, всех его дочерних узлов и атрибутов рекурсивно.  
Если узел имеет родительский узел, то вначале он удаляется из родительского узла.  
Правила заимствования для типов узлов:  
[Атрибут](ТипУзлаDOM.Атрибут.md) - документ-владелец устанавливается в Неопределено, свойство [Указан](АтрибутDOM.Указан.md) устанавливается в Истина, заимствуются все потомки узла;  
[ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - все потомки фрагмента рекурсивно заимствуются;  
[Документ](ТипУзлаDOM.Документ.md) - не может быть адаптирован;  
[ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) - не может быть адаптирован;  
[Элемент](ТипУзлаDOM.Элемент.md) - атрибуты с установленным свойством [Указан](АтрибутDOM.Указан.md) заимствуются, для нового узла-элемента создаются атрибуты, имеющие значения по умолчанию. Все потомки рекурсивно заимствуются;  
[Сущность](ТипУзлаDOM.Сущность.md) - не может быть адаптирован;  
[СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md) - заимствуется только данный узел. Если в данном документе определена сущность с таким же именем, то формируется значение сущности для заимствованной ссылки;  
[Нотация](ТипУзлаDOM.Нотация.md) - не может быть адаптирован;  
[ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md), [Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md), [Комментарий](ТипУзлаDOM.Комментарий.md) - заимствуются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- попытка адаптации узлов [Документ](ТипУзлаDOM.Документ.md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md);
- текущий документ адаптируемого узла в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
