Глобальный контекст (Global context)

ОтключитьОбработчикОповещения (DetachNotificationHandler)

Доступен, начиная с версии 8.0.

Синтаксис:

ОтключитьОбработчикОповещения(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры модуля приложения (модуля обычного приложения) или общего модуля, используемой в качестве обработчика.

Описание:

Отключает подключенный ранее при помощи метода [ПодключитьОбработчикОповещения](Глобальный_контекст.ПодключитьОбработчикОповещения_(Global_context.AttachNotificationHandler).md) вызов процедуры при вызове метода [Оповестить](Глобальный_контекст.Оповестить_(Global_context.Notify).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Процедура ПриНачалеРаботыСистемы()     //...          ПодключитьОбработчикОповещения("ОбработкаОповещенияГлобальная");     //...      ОтключитьОбработчикОповещения("ОбработкаОповещенияГлобальная");     //...  КонецПроцедуры  // ПриНачалеРаботыСистемы() |
