ОбъектМетаданных: Нумератор (MetadataObject: DocumentNumerator)

ДопустимаяДлинаНомера (NumberAllowedLength)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md).   
Отображает тип допустимой длины номера.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Свойство имеет смысл, если тип номера - [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
