ПоддержкаМасштабаСводнойДиаграммы (PivotChartScaleKeeping)

КоличествоЗначений (ValuesCount)

Доступен, начиная с версии 8.0.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит заданное количество значений, отображаемых в видимой области. Если значений больше, используется прокрутка.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
