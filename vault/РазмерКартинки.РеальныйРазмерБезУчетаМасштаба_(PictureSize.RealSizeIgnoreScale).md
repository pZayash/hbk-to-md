РазмерКартинки (PictureSize)

РеальныйРазмерБезУчетаМасштаба (RealSizeIgnoreScale)

Доступен, начиная с версии 8.3.10.

Описание:

Сохраняет реальные размеры картинки без учета масштаба формы. Если область, в которой отображается картинка, меньше ее размеров, из середины картинки вырезается прямоугольник требуемого размера.

Использование в версии:

Доступен, начиная с версии 8.3.10.
