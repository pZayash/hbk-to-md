Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [СоставТабличногоПространстваБазыДанных](СоставТабличногоПространстваБазыДанных.md).   
Состав объектов, индексы таблиц которых сейчас физически находятся в этом табличном пространстве.   
Возвращаемая коллекция недоступна для изменения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
