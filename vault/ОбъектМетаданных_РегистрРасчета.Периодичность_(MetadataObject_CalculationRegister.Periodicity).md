ОбъектМетаданных: РегистрРасчета (MetadataObject: CalculationRegister)

Периодичность (Periodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПериодичностьРегистраРасчета](ПериодичностьРегистраРасчета_(CalculationRegisterPeriodicity).md).   
Определяет вид периода регистрации записей (например, [Год](ПериодичностьРегистраРасчета.Год_(CalculationRegisterPeriodicity.Year).md), [Квартал](ПериодичностьРегистраРасчета.Квартал_(CalculationRegisterPeriodicity.Quarter).md), [Месяц](ПериодичностьРегистраРасчета.Месяц_(CalculationRegisterPeriodicity.Month).md), [День](ПериодичностьРегистраРасчета.День_(CalculationRegisterPeriodicity.Day).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
