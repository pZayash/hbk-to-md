СекцияCDATADOM (DOMCDATASection)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Текстовое содержимое узла DOM.  
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.  
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство [ЭтоПробельныеСимволыЭлемента](ТекстDOM.ЭтоПробельныеСимволыЭлемента.md)).  
Принцип формирования текстового значения узла:  
[Элемент](ТипУзлаDOM.Элемент.md), [Атрибут](ТипУзлаDOM.Атрибут.md), [Сущность](ТипУзлаDOM.Сущность.md), [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md), [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md) - объединение текстового содержимого каждого из дочерних узлов, исключая [Комментарий](ТипУзлаDOM.Комментарий.md); [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md). Пустая строка, если дочерних узлов нет;  
[Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md), [Комментарий](ТипУзлаDOM.Комментарий.md); [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md) - значение свойства [СекцияCDATADOM.ЗначениеУзла](СекцияCDATADOM.ЗначениеУзла.md);  
[Документ](ТипУзлаDOM.Документ.md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md), [Нотация](ТипУзлаDOM.Нотация.md) - Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
