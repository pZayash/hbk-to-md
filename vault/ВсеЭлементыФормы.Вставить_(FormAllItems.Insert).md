# ВсеЭлементыФормы.Вставить

ВсеЭлементыФормы (FormAllItems)

Вставить (Insert)

Доступен, начиная с версии 8.2.

Синтаксис:

Вставить(<Имя>, <ТипЭлемента>, <Родитель>, <Элемент>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Уникальное имя добавляемого элемента.

<ТипЭлемента> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента.

<Родитель> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md).   
Родитель для добавляемого элемента.  
Если не указан, то вставляется на верхний уровень.

<Элемент> (необязательный)

Тип: [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md), [ГруппаФормы](ГруппаФормы_(FormGroup).md), [КнопкаФормы](КнопкаФормы_(FormButton).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md).   
Элемент, перед которым должен быть вставлен новый элемент. Если не указан, то элемент будет вставлен в конец.

Возвращаемое значение:

Тип: [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md), [ГруппаФормы](ГруппаФормы_(FormGroup).md), [КнопкаФормы](КнопкаФормы_(FormButton).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md).   

Описание:

Вставляет элемент в коллекцию элементов.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
