Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [МакетТелаТаблицыМакетаКомпоновкиДанных](МакетТелаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableBodyTemplate).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
