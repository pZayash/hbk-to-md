# ТаблицаФормы.ЗакончитьРедактированиеСтроки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [ТаблицаФормы](ТаблицаФормы_(FormTable).md) › [Методы](ТаблицаФормы__Методы.md)

ТаблицаФормы (FormTable)

ЗакончитьРедактированиеСтроки (EndEditRow)

Доступен, начиная с версии 8.2.

Синтаксис:

ЗакончитьРедактированиеСтроки(<ОтменаРедактирования>)

Параметры:

<ОтменаРедактирования> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если данный параметр имеет значение [Истина](lang__def_BooleanTrue.md), то происходит отмена редактирования строки.

Описание:

Позволяет закончить или отменить редактирование текущей строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Метод не выполняет действия, если таблица не отображается на форме или соответствующая ему команда таблицы недоступна.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.17.
