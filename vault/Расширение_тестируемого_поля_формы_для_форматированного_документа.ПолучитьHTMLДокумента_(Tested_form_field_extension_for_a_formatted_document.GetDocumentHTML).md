Расширение тестируемого поля формы для форматированного документа (Tested form field extension for a formatted document)

ПолучитьHTMLДокумента (GetDocumentHTML)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПолучитьHTMLДокумента(<HTML>, <Вложения>)

Параметры:

<HTML> (обязательный)

Тип: [Строка](lang__def_String.md).   
В параметр метод помещает строку, содержащую текст HTML.

<Вложения> (обязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
В параметр метод помещает структуру вложенных в форматированный документ рисунков.   
Ключ элемента структуры - название или имя файла рисунка, значение - [Картинка](Картинка_(Picture).md).

Описание:

Получает код HTML форматированного документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
