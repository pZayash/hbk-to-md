Описание:

Флажок предназначен для выбора одного значения. Используются флажки, имеющие два либо три состояния. От типа связанного с флажком значения зависит количество состояний флажка: флажок со значением типа [Булево](lang__Булево_(Boolean).md) может иметь только два состояния (установлен, снят), со значением типа [Число](lang__def_Number.md) может иметь три состояния (установлен - 1, снят - 0, "серый флажок" - 2 ).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
