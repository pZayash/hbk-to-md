ПроверкаПравописанияПриВводеТекста (SpellCheckingOnTextInput)

Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Способ проверки правописания определяется автоматически по типу элемента:

- Для многострочного поля ввода - [Использовать](ПроверкаПравописанияПриВводеТекста.Использовать_(SpellCheckingOnTextInput.Use).md),
- Для однострочного поля ввода - [НеИспользовать](ПроверкаПравописанияПриВводеТекста.НеИспользовать_(SpellCheckingOnTextInput.DontUse).md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
