СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

Текст (Text)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Содержит текст сообщения. Если форматированная строка содержит картинки, они удаляются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.16.
