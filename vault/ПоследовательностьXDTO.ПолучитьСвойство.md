ПоследовательностьXDTO (XDTOSequence)

ПолучитьСвойство (GetProperty)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьСвойство(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: Число.   
Индекс в последовательности.

Возвращаемое значение:

Тип: [СвойствоXDTO](СвойствоXDTO.md), Неопределено.   
Если по указанному индексу в последовательности записан текст, будет возвращено Неопределено.

Описание:

Получает свойство объекта по индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
