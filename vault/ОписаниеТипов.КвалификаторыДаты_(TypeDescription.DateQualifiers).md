ОписаниеТипов (TypeDescription)

КвалификаторыДаты (DateQualifiers)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КвалификаторыДаты](КвалификаторыДаты_(DateQualifiers).md).   
Содержит квалификаторы даты - объект, используемый для описания допустимых значений типа [Дата](lang__def_Date.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл, если в описании типов используется тип [Дата](lang__def_Date.md).
