Доступен, начиная с версии 8.0.

Только чтение.

Описание:

Тип: ВнешняяОбработкаТабличнаяЧасть.<Имя обработки>.<Имя табличной части>.   
Набор свойств содержит табличные части внешней обработки. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ВнешняяОбработкаТабличнаяЧасть.<Имя обработки>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
