ТестируемаяТаблицаФормы (TestedFormTable)

УстановитьПорядок (SetOrder)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьПорядок(<ЗаголовокКолонки>)

Параметры:

<ЗаголовокКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок колонки, по которой требуется установить сортировку, например: "Код".

Описание:

Устанавливает порядок по указанной колонке. Если в момент установки порядка по этой колонке уже выполняется сортировка, порядок сортировки устанавливается на противоположный.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
