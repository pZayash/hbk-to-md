ТестируемаяТаблицаФормы (TestedFormTable)

ВозможноРазвернуть (CanBeExpanded)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВозможноРазвернуть(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие.md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки.  
Если не указан, выполняется проверка возможности развернуть текущую строку.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Булево.   
Истина - разворачивание узла возможно; Ложь - в противном случае.

Описание:

Проверяет возможность разворачивания узла.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
