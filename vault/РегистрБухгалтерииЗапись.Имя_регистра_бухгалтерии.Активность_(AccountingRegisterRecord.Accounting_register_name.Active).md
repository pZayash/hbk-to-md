Активность (Active)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак активности записи. Неактивные записи не отражаются в итогах и не учитываются при подсчете остатков и оборотов.  
[Истина](lang__def_BooleanTrue.md) - запись активна.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
