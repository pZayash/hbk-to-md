ДоставляемоеУведомление (DeliverableNotification)

Получатели (Recipients)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив получателей уведомления в виде объектов типа [ИдентификаторПодписчикаДоставляемыхУведомлений](ИдентификаторПодписчикаДоставляемыхУведомлений_(DeliverableNotificationSubscriberID).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
