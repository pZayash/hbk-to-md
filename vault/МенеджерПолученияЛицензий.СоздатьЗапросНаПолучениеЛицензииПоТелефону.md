МенеджерПолученияЛицензий (LicenseAcquisitionManager)

СоздатьЗапросНаПолучениеЛицензииПоТелефону (CreateLicenseAcquisitionRequestByPhone)

Доступен, начиная с версии 8.3.20.

Синтаксис:

СоздатьЗапросНаПолучениеЛицензииПоТелефону(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md).   
Данные для запроса на получение лицензии.   
Свойства [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента.md), [Порт](ЗапросНаПолучениеЛицензии.Порт.md), [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей.md), [ИмяФайлаЛицензии](ЗапросНаПолучениеЛицензии.ИмяФайлаЛицензии.md) не используются.

Возвращаемое значение:

Тип: Строка.   
Строка, содержащая последовательность цифр, которую требуется передать в центр лицензирования по телефону для получения лицензии.

Описание:

По данным для получения лицензии и параметрам текущего компьютера создает последовательность цифр, которую требуется передать в центр лицензирования для получения лицензии по телефону.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
