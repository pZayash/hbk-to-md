Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора.md).   
Предоставляет доступ к настройкам отбора списка записей регистра сведений. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных элементов отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
