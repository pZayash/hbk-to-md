РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РазмерКартинки](РазмерКартинки.md).   
Содержит способ приспособления оригинальных размеров картинки к размеру рисунка.  
Значение [АвтоРазмерБезУчетаМасштаба](РазмерКартинки.АвтоРазмерБезУчетаМасштаба.md) интерпретируется как [АвтоРазмер](РазмерКартинки.АвтоРазмер.md).  
Значение [РеальныйРазмерБезУчетаМасштаба](РазмерКартинки.РеальныйРазмерБезУчетаМасштаба.md) интерпретируется как [РеальныйРазмер](РазмерКартинки.РеальныйРазмер.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунков типа [Картинка](ТипРисункаТабличногоДокумента.Картинка.md).

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
