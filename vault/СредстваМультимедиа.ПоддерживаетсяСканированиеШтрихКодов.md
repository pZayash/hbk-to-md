СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяСканированиеШтрихКодов (BarcodeScanningSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяСканированиеШтрихКодов()

Возвращаемое значение:

Тип: Булево.   
Возвращает Истина в следующих случаях:

- на Android - при наличии камеры на устройстве,
- на iOS - если система поддерживает видеозапись,
- на Windows Phone и Windows 10 - при наличии задней камеры и доступа к предпросмотру.

Описание:

Определяет, поддерживается ли сканирование штрихкодов.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Не поддерживается в мобильном приложении для следующих ОС:

- Windows Phone 8.1.
- Windows Desktop.

[СредстваМультимедиа](СредстваМультимедиа.md), метод [ПоказатьСканированиеШтрихКодов](СредстваМультимедиа.ПоказатьСканированиеШтрихКодов.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
