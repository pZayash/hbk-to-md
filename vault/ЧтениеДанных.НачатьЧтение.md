ЧтениеДанных (DataReader)

НачатьЧтение (BeginRead)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтение(<ОписаниеОповещения>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - Объект типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md), содержащий прочитанные данные.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые требуется прочитать. Если не установлено, будут прочитаны все данные до конца потока.  
Значение по умолчанию: Неопределено.

Описание:

Начинает чтение двоичных данных заданного размера из потока. Если размер не задан, будут прочитаны все данные до конца потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
