Описание:

Описывает структуру, передаваемую в обработчик команды.  
Содержит следующие свойства:  
Источник – окно ([ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md)) или форма ([Форма](Форма.md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md)), из которого(-ой) вызвана команда.  
Окно – тип [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md). Окно, в котором требуется открыть форму.  
Уникальность - уникальность формы. Ложь - искать с ключом уникальности формы по умолчанию, Истина - не искать форму вообще, любое другое значение устанавливает ключ уникальности, по которому будет произведен поиск.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
