ТабличноеПоле (TableBox)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак возможности/невозможности интерактивного изменения содержимого табличного поля. Если значение свойства [Истина](lang__def_BooleanTrue.md), то возможен только просмотр табличного поля.

Доступность:

Толстый клиент.

Примечание:

В отличие от свойства [Доступность](ТабличноеПоле.Доступность_(TableBox.Enabled).md), если свойство ТолькоПросмотр равно [Истина](lang__def_BooleanTrue.md), становится невозможным редактирование табличного поля, однако на остальную работу с полем это не влияет. Например, можно выделить строковое представление содержимого табличного поля и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
