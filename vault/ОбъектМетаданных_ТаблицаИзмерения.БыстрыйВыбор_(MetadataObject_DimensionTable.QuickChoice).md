# ОбъектМетаданных: ТаблицаИзмерения.БыстрыйВыбор

ОбъектМетаданных: ТаблицаИзмерения (MetadataObject: DimensionTable)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство содержит способ выбора значения в таблице измерения, если свойство "Быстрый выбор" таблицы установлено в значение Авто.  
Если свойство установлено в значение Авто, то при выборе значения учитывается свойство "Быстрый выбор" объекта конфигурации, к которому относится реквизит.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
