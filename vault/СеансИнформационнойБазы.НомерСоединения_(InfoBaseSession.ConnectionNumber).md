СеансИнформационнойБазы (InfoBaseSession)

НомерСоединения (ConnectionNumber)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер соединения, которому назначен сеанс.   
0, если сеанс не назначен соединению.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
