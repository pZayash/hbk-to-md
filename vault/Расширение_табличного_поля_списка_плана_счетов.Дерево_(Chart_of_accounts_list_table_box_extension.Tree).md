Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

Дерево (Tree)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит способ отображения поля списка.  
Если свойство установить в [Истина](lang__def_BooleanTrue.md), то поле будет отображаться в виде дерева.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
