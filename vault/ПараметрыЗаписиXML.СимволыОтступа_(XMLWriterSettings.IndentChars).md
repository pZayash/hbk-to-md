ПараметрыЗаписиXML (XMLWriterSettings)

СимволыОтступа (IndentChars)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет набор символов, используемый при форматировании (создании отступов) в документе XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
