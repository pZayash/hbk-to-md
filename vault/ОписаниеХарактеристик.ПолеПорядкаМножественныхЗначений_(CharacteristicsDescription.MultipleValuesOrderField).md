ОписаниеХарактеристик (CharacteristicsDescription)

ПолеПорядкаМножественныхЗначений (MultipleValuesOrderField)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [Поле](Поле_(Field).md).   
Содержит имя поля таблицы значений характеристик, в котором хранится поле, по которому упорядочиваются множественные характеристики.   
Если не указано, то значения будут упорядочиваться по самому значению характеристики.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
