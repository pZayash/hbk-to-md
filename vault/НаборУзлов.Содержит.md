НаборУзлов (NodeSet)

Содержит (Contains)

Доступен, начиная с версии 8.0.

Синтаксис:

Содержит(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена.md).   
Ссылка на узел плана обмена, который надо проверить на вхождение в набор узлов.

Возвращаемое значение:

Тип: Булево.   
Истина - узел входит в набор узлов; Ложь - в противном случае.

Описание:

Определяет, входит ли указанный узел плана обмена в набор узлов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Узел = ПланыОбмена.ПО1.НайтиПоКоду("Маг11"); Если Об.ОбменДанными.Получатели.Содержит(Узел) Тогда     // Некоторые действия  КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
