ИнтервалДиаграммыГанта (GanttChartInterval)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Связь>)

Параметры:

<Связь> (обязательный)

Тип: [СвязьДиаграммыГанта](СвязьДиаграммыГанта_(GanttChartLink).md).   
Удаляемая связь.

Описание:

Удаляет связь, в которой участвует интервал.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
