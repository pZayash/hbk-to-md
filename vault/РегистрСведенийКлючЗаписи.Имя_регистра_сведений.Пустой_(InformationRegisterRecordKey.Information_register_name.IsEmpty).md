РегистрСведенийКлючЗаписи.<Имя регистра сведений> (InformationRegisterRecordKey.<Information register name>)

Пустой (IsEmpty)

Доступен, начиная с версии 8.2.

Синтаксис:

Пустой()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Возвращает [Истина](lang__def_BooleanTrue.md), если все поля ключа имеют значения по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
