Доступен, начиная с версии 8.1.

Только чтение.

Описание:

Если данный признак установлен в значение [Истина](lang__def_BooleanTrue.md), данное свойство может содержать значения [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.1.
