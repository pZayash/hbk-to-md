ГрафическаяСхема (GraphicalSchema)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Из файла

Синтаксис:

Прочитать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла графической схемы.

Описание варианта метода:

Чтение графической схемы из файла.

Вариант синтаксиса: Из потока

Синтаксис:

Прочитать(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения графической схемы.

Описание варианта метода:

Чтение графической схемы из потока.

Описание:

Считывает графическую схему из файла.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Пример:

|  |
| --- |
| Схема = Новый ГрафическаяСхема; Схема.Прочитать("C:\My Documents\Схема.grs"); |
