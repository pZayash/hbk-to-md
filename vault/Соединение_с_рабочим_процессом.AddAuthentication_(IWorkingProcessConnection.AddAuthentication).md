Соединение с рабочим процессом (IWorkingProcessConnection)

AddAuthentication (AddAuthentication)

Доступен, начиная с версии 8.1.

Синтаксис:

AddAuthentication(<Имя>, <Пароль>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя администратора информационной базы. Административный доступ разрешен только к тем информационным базам, в которых зарегистрирован пользователь с таким именем и он имеет право "Администратор".

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль администратора информационной базы.

Описание:

Выполняет аутентификацию.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
