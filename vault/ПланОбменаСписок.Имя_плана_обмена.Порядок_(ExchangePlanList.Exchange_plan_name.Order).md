ПланОбменаСписок.<Имя плана обмена> (ExchangePlanList.<Exchange plan name>)

Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Порядок_(Order).md).   
Содержит объект [Порядок](Порядок_(Order).md), по которому осуществляется текущее упорядочивание узлов в списке. В качестве имен упорядочивания объекта могут выступать имена предопределенных реквизитов Код и Наименование и имена реквизитов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
