ТаблицаКомпоновкиДанных (DataCompositionTable)

Выбор (Selection)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных_(DataCompositionSelectedFields).md).   
Содержит набор выбранных полей таблицы.  
Данное свойство недоступно для чтения и записи у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
