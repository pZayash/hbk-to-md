# ЗадачаСсылка.<Имя задачи>.Пустая

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаСсылка.<Имя задачи>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md) › [Методы](ЗадачаСсылка.Имя_задачи__Методы.md)

ЗадачаСсылка.<Имя задачи> (TaskRef.<Task name>)

Пустая (IsEmpty)

Доступен, начиная с версии 8.0.

Синтаксис:

Пустая()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ссылка не указывает ни на какой объект (пустая ссылка); [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, является ли ссылка пустой или нет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
