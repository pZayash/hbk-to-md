ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

МногострочныйРежим (MultiLine)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает на многострочность поля. Если свойство установлено (Истина), то текст может быть отредактирован с использованием переноса строк.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
