# ПланОбменаОбъект.<Имя плана обмена>.Заблокирован

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [Методы](ПланОбменаОбъект.Имя_плана_обмена__Методы.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.0.

Синтаксис:

Заблокирован()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект заблокирован; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, заблокирован ли узел данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Следует учитывать, что этот метод используется для проверки блокировки объекта базы данных конкретным объектом встроенного языка. Он не может быть использован для проверки, заблокирован ли вообще объект базы данных, например, другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.0.
