Глобальный контекст (Global context)

ИнициализироватьПредопределенныеДанные (InitializePredefinedData)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ИнициализироватьПредопределенныеДанные()

Описание:

Инициализирует предопределенные данные в текущей области информационной базы.   
Предопределенные элементы создаются у объектов метаданных, при выполнении всех следующих условий:

- В сеансе установлены и используются все разделители, в состав которых включен объект метаданных.
- Автоматическое обновление предопределенных данных для этого объекта метаданных включено.
- Предопределенные данные для этого объекта в этой области данных еще не создавались.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

Примечание:

Если для какого-либо объекта конфигурации не будут установлены все разделители, то:

- В режиме совместимости конфигурации [НеИспользовать](РежимСовместимости.НеИспользовать_(CompatibilityMode.DontUse).md), предопределенные данные для этого объекта не создаются. Исключение при этом не вызывается.
- В режиме совместимости конфигурации [Версия8\_3\_4](РежимСовместимости.Версия8_3_4_(CompatibilityMode.Version8_3_4).md) и ниже будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
