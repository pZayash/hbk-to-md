ВариантИспользованииБазыДанныхКопии (CopyDatabaseUseVariant)

ИспользуетсяДругойКопией (UsedByAnotherCopy)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Базу данных использует другая копия базы данных.   
Размещение копии в базе данных на текущий момент запрещено. Чтобы разместить копию, требуется выполнить команду "Использовать базу данных этой копией".

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
