ТестируемаяФорма (TestedForm)

ВыполнитьВыборИзСписка (ExecuteChoiceFromList)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ВыполнитьВыборИзСписка(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Строка, Число.   
Представление выбираемого значения или номер значения (индекс строки, нумерация с 0).

Описание:

Выбирает значение из открытого списка выбора по его представлению в списке выбора.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
