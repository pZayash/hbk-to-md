Картинка (Picture)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Картинка](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Картинка_(CollaborationSystemMessageButtonPanelButton.Picture).md)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Содержит картинку, отображаемую в действии сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если не задан ни <Текст>, ни <Картинка>, то в качестве текста гиперссылки выводится представление <Значения>.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
