ЦветаСтиля (StyleColors)

<Имя элемента стиля> (<Name of style element>)

Доступен, начиная с версии 8.0.

Описание:

Тип: [Цвет](Цвет.md).   
Набор значений содержит цвета элементов стиля, созданные при конфигурировании прикладной задачи. Доступ к цвету элемента стиля осуществляется по имени. Имена значений совпадают с именами элементов стиля типа [Цвет](ВидЭлементаСтиля.Цвет.md), как они заданы в Конфигураторе.

Использование в версии:

Доступен, начиная с версии 8.0.
