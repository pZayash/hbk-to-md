ВнешнийИсточникДанныхТаблицаКлючЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordKey.<External source name>.<External data source table name>)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ВнешнийИсточникДанныхТаблицаКлючЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (необязательный)

Тип: [Структура](Структура.md), Неопределено.   
Содержит пары Ключ/Значение. Имя ключа - имя поля ключа, в качестве значения поля – необходимое значение этого поля. Значения должны быть перечислены в порядке полей ключа, как они определены в конфигураторе.  
Если параметр не задан или равен Неопределено, или отсутствует, создается пустой ключ таблицы внешнего источника данных.

Описание:

Создает новый объект на основе переданных параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
