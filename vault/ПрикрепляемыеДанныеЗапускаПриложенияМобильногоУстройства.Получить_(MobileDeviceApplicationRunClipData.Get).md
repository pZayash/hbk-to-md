Получить (Get)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента прикрепляемых данных в контейнере. Значение должно быть целым числом в диапазоне от 0 до [Количество](ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства.Количество_(MobileDeviceApplicationRunClipData.Count).md) – 1.

Возвращаемое значение:

Тип: [ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства](ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства_(MobileDeviceApplicationRunClipDataItem).md).   

Описание:

Возвращает элемент из контейнера прикрепляемых данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
