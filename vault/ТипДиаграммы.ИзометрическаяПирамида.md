ТипДиаграммы (ChartType)

ИзометрическаяПирамида (PyramidGraph)

Доступен, начиная с версии 8.0.

Описание:

Изометрическая диаграмма. Значения отмечаются пирамидами разной высоты. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График.md).

Использование в версии:

Доступен, начиная с версии 8.0.
