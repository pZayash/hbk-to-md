ЗначенияПараметровДанныхКомпоновкиДанных (DataCompositionDataParameterValues)

ДоступныеПараметры (AvailableParameters)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДоступныеПараметрыКомпоновкиДанных](ДоступныеПараметрыКомпоновкиДанных_(DataCompositionAvailableParameters).md), [Неопределено](lang__def_Undefined.md).   
Содержит набор доступных параметров. Определяет состав параметров данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Неопределено](lang__def_Undefined.md) - настройки, которым принадлежат параметры данных, не связаны с источником доступных настроек.

Использование в версии:

Доступен, начиная с версии 8.1.
