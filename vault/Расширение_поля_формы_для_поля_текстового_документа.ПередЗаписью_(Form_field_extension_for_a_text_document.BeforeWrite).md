Расширение поля формы для поля текстового документа (Form field extension for a text document)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит значение [Истина](lang__def_BooleanTrue.md), если выполняется команда "Сохранить как...".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивной записью объекта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
