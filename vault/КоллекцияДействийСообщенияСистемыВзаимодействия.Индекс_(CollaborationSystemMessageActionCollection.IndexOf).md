Индекс (IndexOf)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Индекс](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Индекс_(CollaborationSystemMessageButtonPanelButtonRow.IndexOf).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Индекс(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md).   
Действие, для которого необходимо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс указанного действия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
