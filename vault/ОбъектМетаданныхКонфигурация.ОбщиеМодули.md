ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОбщиеМодули (CommonModules)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все общие модули, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ОбщийМодуль](ОбъектМетаданных_ОбщийМодуль.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
