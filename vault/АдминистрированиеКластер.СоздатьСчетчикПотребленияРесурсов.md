АдминистрированиеКластер (AdministrationCluster)

СоздатьСчетчикПотребленияРесурсов (CreateResourceConsumptionCounter)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьСчетчикПотребленияРесурсов()

Возвращаемое значение:

Тип: [АдминистрированиеСчетчикПотребленияРесурсов](АдминистрированиеСчетчикПотребленияРесурсов.md).   

Описание:

Создает счетчик потребления ресурсов с пустыми значениями свойств. Этот объект может быть использован для регистрации нового счетчика потребления ресурсов после установки правильных значений его свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
