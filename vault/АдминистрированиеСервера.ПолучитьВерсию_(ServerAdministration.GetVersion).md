АдминистрированиеСервера (ServerAdministration)

ПолучитьВерсию (GetVersion)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьВерсию()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает строковое представление версии сервера "1С:Предприятие".

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
