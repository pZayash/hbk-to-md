РеквизитФормы (FormAttribute)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый РеквизитФормы(<Имя>, <Тип>, <Путь>, <Заголовок>, <СохраняемыеДанные>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя реквизита.

<Тип> (обязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Тип реквизита.

<Путь> (необязательный)

Тип: [Строка](lang__def_String.md).   
Путь к создаваемому реквизиту. Не включает имя реквизита.  
Значение по умолчанию: Пустая строка.

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит отображаемый текст реквизита.  
Значение по умолчанию: Пустая строка.

<СохраняемыеДанные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Реквизит содержит сохраняемые данные.  
[Истина](lang__def_BooleanTrue.md) - сохраняемый реквизит формы.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Создает описание реквизита формы клиентского приложения на основе переданных параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
