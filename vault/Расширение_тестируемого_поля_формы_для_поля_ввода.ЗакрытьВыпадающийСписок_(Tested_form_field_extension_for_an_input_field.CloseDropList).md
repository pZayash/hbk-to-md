Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ЗакрытьВыпадающийСписок (CloseDropList)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗакрытьВыпадающийСписок()

Описание:

Закрывает выпадающий список, если такая возможность доступна. В противном случае генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
