Описание:

Записывает конец объекта JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; // Запись документа JSON  // Запись строковое значение  ЗаписьJSON.ЗаписатьНачалоОбъекта(); ЗаписьJSON.ЗаписатьИмяСвойства("Строка"); ЗаписьJSON.ЗаписатьЗначение("Строковое значение"); ЗаписьJSON.ЗаписатьКонецОбъекта(); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
