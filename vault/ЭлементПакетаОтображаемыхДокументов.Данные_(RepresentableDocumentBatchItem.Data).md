ЭлементПакетаОтображаемыхДокументов (RepresentableDocumentBatchItem)

Данные (Data)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ПакетОтображаемыхДокументов](ПакетОтображаемыхДокументов_(RepresentableDocumentBatch).md).   
Содержит данные пакета отображаемых документов.  
[Строка](lang__def_String.md) - адрес временного хранилища, в котором расположен документ. [ПакетОтображаемыхДокументов](ПакетОтображаемыхДокументов_(RepresentableDocumentBatch).md) - вложенный пакет документов. В качестве документов может выступать табличный документ.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
