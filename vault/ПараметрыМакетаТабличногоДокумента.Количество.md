ПараметрыМакетаТабличногоДокумента (SpreadsheetDocumentTemplateParameters)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество параметров в коллекции.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Процедура ОчиститьПараметрыОбластиТабличногоДокумента(Область)     Для Параметр = 0 По Область.Параметры.Количество() -1 Цикл       Область.Параметры[Параметр] = Неопределено;     КонецЦикла;  КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
