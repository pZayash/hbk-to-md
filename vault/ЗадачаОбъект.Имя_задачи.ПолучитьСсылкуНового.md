ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](ЗадачаОбъект.Имя_задачи.УстановитьСсылкуНового.md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи.md), метод [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
