СредстваГеопозиционирования (LocationTools)

ОбновитьМестоположение (UpdateLocation)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ОбновитьМестоположение(<ИмяПровайдера>, <Таймаут>)

Параметры:

<ИмяПровайдера> (обязательный)

Тип: [Строка](lang__def_String.md).   
Содержит имя провайдера.

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Определяет предельное время ожидания ответа от провайдера (в секундах).  
0 - не устанавливать таймаут (время ожидания не ограничено).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Ложь](lang__def_BooleanFalse.md) - за выделенное время не был получен ответ от провайдера.

Описание:

Выполняет попытку получения от провайдера актуального местоположения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
