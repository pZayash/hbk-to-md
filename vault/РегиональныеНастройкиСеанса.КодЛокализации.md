РегиональныеНастройкиСеанса (SessionRegionalSettings)

КодЛокализации (LocaleCode)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Один из возможных кодов локализации. Используется при получении представлений значений типа Число, Дата, Булево, при выполнении методов [ЧислоПрописью](Глобальный_контекст.ЧислоПрописью.md) и [ПредставлениеПериода](Глобальный_контекст.ПредставлениеПериода.md). Допустимые коды локализации могут быть получены методом глобального контекста [ПолучитьДопустимыеКодыЛокализации](Глобальный_контекст.ПолучитьДопустимыеКодыЛокализации.md). Представления кодов локализации можно получить методом глобального контекста [ПредставлениеКодаЛокализации](Глобальный_контекст.ПредставлениеКодаЛокализации.md).

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
