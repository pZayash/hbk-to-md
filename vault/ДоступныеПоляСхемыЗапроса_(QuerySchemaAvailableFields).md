# ДоступныеПоляСхемыЗапроса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса_(QuerySchemaAvailableField).md), [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса_(QuerySchemaAvailableNestedTable).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля и доступные вложенные таблицы.  
Возможно обращение к доступным полям и доступным вложенным таблицам посредством оператора [...]. В качестве аргумента передается индекс.
