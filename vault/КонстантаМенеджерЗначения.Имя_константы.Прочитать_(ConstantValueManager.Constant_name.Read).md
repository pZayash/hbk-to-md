КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать()

Описание:

Считывает значение константы из базы данных в свойство [Значение](КонстантаМенеджерЗначения.Имя_константы.Значение_(ConstantValueManager.Constant_name.Value).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
