МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ЗагрузитьВыгрузкуДанных (LoadDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ЗагрузитьВыгрузкуДанных(<ПутьКФайлу>, <Описание>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу с выгрузкой данных.

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   

Описание:

Загружает выгрузку данных системы взаимодействия. Возвращает управление при окончании загрузки резервной копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [ЗагрузитьВыгрузкуДанныхАсинх](МенеджерСистемыВзаимодействия.ЗагрузитьВыгрузкуДанныхАсинх_(CollaborationSystemManager.LoadDataDumpAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
