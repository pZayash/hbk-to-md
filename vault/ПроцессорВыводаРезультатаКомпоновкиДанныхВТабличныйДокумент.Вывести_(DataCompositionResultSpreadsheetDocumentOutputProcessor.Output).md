ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент (DataCompositionResultSpreadsheetDocumentOutputProcessor)

Вывести (Output)

Доступен, начиная с версии 8.1.

Синтаксис:

Вывести(<ПроцессорКомпоновки>, <ВозможностьПрерыванияПользователем>)

Параметры:

<ПроцессорКомпоновки> (обязательный)

Тип: [ПроцессорКомпоновкиДанных](ПроцессорКомпоновкиДанных_(DataCompositionProcessor).md).   
Процессор компоновки данных, из которого будут получаться элементы результата.

<ВозможностьПрерыванияПользователем> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, может ли пользователь прервать вывод при помощи комбинации клавиш Ctrl + Break.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   

Описание:

Выводит весь результат в объект.  
При этом автоматически исполняется метод [НачатьВывод](ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент.НачатьВывод_(DataCompositionResultSpreadsheetDocumentOutputProcessor.BeginOutput).md), перебираются все элементы процессора компоновки и выводятся в результат. После чего исполняется метод [ЗакончитьВывод](ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент.ЗакончитьВывод_(DataCompositionResultSpreadsheetDocumentOutputProcessor.EndOutput).md), результат которого возвращается данным методом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
