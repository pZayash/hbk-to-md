Расширение группы формы для группы колонок (Form group extension for a group of columns)

Группировка (Group)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГруппировкаКолонок](ГруппировкаКолонок.md).   
Вариант группировки колонок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
