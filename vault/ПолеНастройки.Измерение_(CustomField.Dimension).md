ПолеНастройки (CustomField)

Измерение (Dimension)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, является ли поле допустимым к использованию в качестве измерения построителя отчета.  
[Истина](lang__def_BooleanTrue.md) - допустимо к использованию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
