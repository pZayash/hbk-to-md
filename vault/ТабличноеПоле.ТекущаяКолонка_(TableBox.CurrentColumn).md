ТабличноеПоле (TableBox)

ТекущаяКолонка (CurrentColumn)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md).   
Содержит текущую колонку табличного поля. Если данному свойству присвоить [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md), то произойдет активизация указанной колонки, если устанавливается значение [Неопределено](lang__def_Undefined.md), то активная колонка сбрасывается.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
