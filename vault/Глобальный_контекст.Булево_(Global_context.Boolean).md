Глобальный контекст (Global context)

Булево (Boolean)

Доступен, начиная с версии 8.0.

Синтаксис:

Булево(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Число](lang__def_Number.md), [Булево](lang__Булево_(Boolean).md), [Строка](lang__def_String.md).   
Исходное значение.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Полученное значение.

Описание:

Преобразует полученный параметр в значение типа [Булево](lang__Булево_(Boolean).md).  
Преобразование числа к типу Булево производится по следующим правилам:

- 0 преобразуется в [Ложь](lang__def_BooleanFalse.md);
- остальные значения преобразуются в [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Если ТипЗнч(Реквизит) = Тип("Булево")  Тогда     Значение = Булево(Значение); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
