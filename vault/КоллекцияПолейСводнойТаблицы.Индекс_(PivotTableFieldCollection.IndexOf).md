КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md).   
Искомое поле.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента коллекции сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
