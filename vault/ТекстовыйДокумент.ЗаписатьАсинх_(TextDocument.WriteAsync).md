ТекстовыйДокумент (TextDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайла>, <РазделительСтрок>)

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Задается строка, являющаяся разделителем строк, к которой при записи файла будет преобразован символ [ПС](Символы.ПС_(Chars.LF).md). Если строка пустая, то разделители строк при записи не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК_(Chars.CR).md);
- [ПС](Символы.ПС_(Chars.LF).md);
- [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание варианта метода:

Запись текстового документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <ТипФайла>, <РазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи документа.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Задается строка, являющаяся разделителем строк, из которой при чтении файла будет сформирован символ [ПС](Символы.ПС_(Chars.LF).md).   
Если строка пустая, то разделители строк при чтении не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК_(Chars.CR).md);
- [ПС](Символы.ПС_(Chars.LF).md);
- [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли добавлена метка порядка байт (BOM) при сохранении документа.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание варианта метода:

Запись текстового документа в поток.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания:

- [Истина](lang__def_BooleanTrue.md) - если запись была инициирована с помощью расширения браузера и пользователь согласился на его получение, то есть запись файла была завершена.
- [Ложь](lang__def_BooleanFalse.md) - если запись была инициирована с помощью расширения браузера, но пользователь не согласился на его получение, то есть запись файла не была завершена.
- [Неопределено](lang__def_Undefined.md) - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

Описание:

Асинхронная функция. Записывает текст в файл с именем <ИмяФайла>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
