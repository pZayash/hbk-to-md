КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Поле>, <Смещение>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, [ПолеСводнойТаблицы](ПолеСводнойТаблицы.md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы.md).   
Сдвигаемое поле.

<Смещение> (обязательный)

Тип: Число.   
Число полей, на которое необходимо сдвинуть данное поле.

Описание:

Сдвигает поле сводной таблицы на указанное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
