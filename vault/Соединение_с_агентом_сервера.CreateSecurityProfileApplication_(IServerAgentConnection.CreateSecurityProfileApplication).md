Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfileApplication (CreateSecurityProfileApplication)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfileApplication()

Возвращаемое значение:

Тип: [Приложение](Приложение_(ISecurityProfileApplication).md).   

Описание:

Создает новое разрешенное приложение для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
