МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтменитьРегистрациюИнформационнойБазыАсинх (UnregisterInfoBaseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОтменитьРегистрациюИнформационнойБазыАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Отменяет регистрацию информационной базы в сервере взаимодействия.  
Требуется наличие у текущего пользователя права "РегистрацияИнформационнойБазыСистемыВзаимодействия".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
