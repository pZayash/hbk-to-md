Описание:

Топологический объект, отображаемый на географической схеме полигоном (занимает некоторую площадь). Может содержать "дырки".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
