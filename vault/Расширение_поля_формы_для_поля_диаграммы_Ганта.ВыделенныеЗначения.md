Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ВыделенныеЗначения (SelectedValues)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив.md).   
Содержит массив идентификаторов выделенных значений диаграммы.   
Значение по умолчанию - пустая коллекция.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
