РежимАвтоОтображенияКнопкиОткрытия (AutoShowOpenButtonMode)

ТолькоДляЗаполненного (FilledOnly)

Доступен, начиная с версии 8.3.9.

Описание:

Кнопка открытия отображается только если значение в поле ввода отличается от значения по умолчанию для данного типа.

Использование в версии:

Доступен, начиная с версии 8.3.9.
