ИнформацияХранилищаДвоичныхДанных (BinaryDataStorageInformation)

ОбъемУдаленныхДанных (DeletedDataSize)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Размер данных, которые удалены логически, но не удалены физически (в хранилище используется отложенное удаление).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
