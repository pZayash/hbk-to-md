<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Набор свойств содержит табличные части бизнес-процесса. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (БизнесПроцессТабличнаяЧасть.<Имя бизнес-процесса>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
