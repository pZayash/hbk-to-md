ЗаписьФайлаАрхива (ArchiveFileWriter)

На основании потока

Доступен, начиная с версии 8.3.26.

Синтаксис:

Новый ЗаписьФайлаАрхива(<Поток>, <Пароль>, <ТипФайлаАрхива>, <Комментарий>, <МетодСжатия>, <УровеньСжатия>, <МетодШифрования>, <Кодировка>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, в который будет записан архив.

<Пароль> (необязательный)

Тип: Строка.   
Пароль, который будет назначен архиву.   
Если пароль не назначен или является пустой строкой, то шифрование не происходит.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива.md).   
Тип файла архива, который требуется записать.  
Примечание. Допускается только использование значения [ZIP](ТипФайлаАрхива.ZIP.md).  
Значение по умолчанию: [ZIP](ТипФайлаАрхива.ZIP.md).

<Комментарий> (необязательный)

Тип: Строка.   
Комментарий, описывающий файл архива.

<МетодСжатия> (необязательный)

Тип: [МетодСжатияФайлаАрхива](МетодСжатияФайлаАрхива.md).   
Метод сжатия, которым будет сжиматься архив.  
Значение по умолчанию: [Сжатие](МетодСжатияФайлаАрхива.Сжатие.md).

<УровеньСжатия> (необязательный)

Тип: [УровеньСжатияФайлаАрхива](УровеньСжатияФайлаАрхива.md).   
Уровень сжатия данных.  
Значение по умолчанию: [Оптимальный](УровеньСжатияФайлаАрхива.Оптимальный.md).

<МетодШифрования> (необязательный)

Тип: [МетодШифрованияФайлаАрхива](МетодШифрованияФайлаАрхива.md).   
Метод шифрования, которым будет зашифрован архив.  
Значение по умолчанию: [Zip20](МетодШифрованияФайлаАрхива.Zip20.md).

<Кодировка> (необязательный)

Тип: [КодировкаИменФайловВФайлеАрхива](КодировкаИменФайловВФайлеАрхива.md).   
Определяет способ кодировки имен файлов внутри файла архива.  
Значение по умолчанию: [Авто](КодировкаИменФайловВФайлеАрхива.Авто.md).

Описание:

Создает объект записи файла архива и устанавливает поток для записи архива.

Использование в версии:

Доступен, начиная с версии 8.3.26.
