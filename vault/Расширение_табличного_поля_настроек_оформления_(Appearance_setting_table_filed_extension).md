Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для поля настроек оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [ЭлементНастройкиОформления](ЭлементНастройкиОформления_(AppearanceSettingItem).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
