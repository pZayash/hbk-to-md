# СправочникОбъект.<Имя справочника>.ПринадлежитЭлементу

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПринадлежитЭлементу (BelongsToItem)

Доступен, начиная с версии 8.0.

Синтаксис:

ПринадлежитЭлементу(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   
Группа (элемент для иерархического справочника с иерархией элементов), для которой определяется принадлежность элемента.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - элемент подчинен группе; [Ложь](lang__def_BooleanFalse.md) - в противном случае.  
Если в качестве параметра указать сам элемент, то [Ложь](lang__def_BooleanFalse.md).

Описание:

Определяет подчиненность элемента справочника группе с учетом всех уровней иерархии.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Определяет подчиненность группе непосредственно на момент вызова метода путем последовательного считывания всех вышестоящих элементов.

Использование в версии:

Доступен, начиная с версии 8.0.
