ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

АутентификацияСтандартная (StandardAuthentication)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Указывает необходимость использования стандартной аутентификации.  
Если содержит Неопределено, то необходимость стандартной аутентификации получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
