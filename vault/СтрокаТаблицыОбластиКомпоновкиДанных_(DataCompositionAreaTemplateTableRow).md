# СтрокаТаблицыОбластиКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макеты областей](Макеты_областей.md)

Доступен, начиная с версии 8.1.

Свойства:

[ИдентификаторТаблицы (TableID)](СтрокаТаблицыОбластиКомпоновкиДанных.ИдентификаторТаблицы_(DataCompositionAreaTemplateTableRow.TableID).md)  
[Ячейки (Cells)](СтрокаТаблицыОбластиКомпоновкиДанных.Ячейки_(DataCompositionAreaTemplateTableRow.Cells).md)  

Описание:

Строка таблицы. Представляет собой коллекцию ячеек, расположенных горизонтально слева направо. Таким образом, несколько идущих подряд строк таблицы области компоновки данных образуют прямоугольную таблицу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: TableRow.
