БлокировкаСеансов (SessionsLock)

Сообщение (Message)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Содержит текст, который будет частью сообщения об ошибке при попытке начала нового сеанса с заблокированной информационной базой.  
Структура сообщения:  
<системный текст>   
<текст из данного свойства>

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.19.
