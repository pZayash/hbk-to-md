# МенеджерВнешнихХранилищДвоичныхДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Хранилище двоичных данных](Хранилище_двоичных_данных.md)

Доступен, начиная с версии 8.3.24.

Элементы коллекции:

[МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных_(BinaryDataExternalStorageManager).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются МенеджерВнешнегоХранилищаДвоичныхДанных.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя внешнего хранилища.  

Методы:

[Добавить (Add)](МенеджерВнешнихХранилищДвоичныхДанных.Добавить_(BinaryDataExternalStoragesManager.Add).md)  
[Количество (Count)](МенеджерВнешнихХранилищДвоичныхДанных.Количество_(BinaryDataExternalStoragesManager.Count).md)  
[Найти (Find)](МенеджерВнешнихХранилищДвоичныхДанных.Найти_(BinaryDataExternalStoragesManager.Find).md)  
[Удалить (Delete)](МенеджерВнешнихХранилищДвоичныхДанных.Удалить_(BinaryDataExternalStoragesManager.Delete).md)  

Описание:

Предоставляет доступ к внешним хранилищам двоичных данных. Может быть получен с помощью свойства глобального контекста [ВнешниеХранилищаДвоичныхДанных](Глобальный_контекст.ВнешниеХранилищаДвоичныхДанных_(Global_context.BinaryDataExternalStorages).md).  
Требуется наличие у текущего пользователя права "Администрирование".

Доступность:

Сервер.

Пример:

|  |
| --- |
| // обход всех внешних хранилищ (без ХранилищеДвоичныхДанных)  Для каждого Мнджр Из ВнешниеХранилищаДвоичныхДанных Цикл     Дата = ТекущаяУниверсальнаяДата();     Мнджр.ОчиститьНеиспользуемоеМестоПоУниверсальнойДате(Дата); КонецЦикла;  // доступ к менеджеру по имени  Мнджр = ВнешниеХранилищаДвоичныхДанных["manager"]; |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ВнешниеХранилищаДвоичныхДанных](Глобальный_контекст.ВнешниеХранилищаДвоичныхДанных_(Global_context.BinaryDataExternalStorages).md)
