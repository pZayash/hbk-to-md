ВыражениеИтогаСхемыЗапроса (QuerySchemaTotalExpression)

Выражение (Expression)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md).   
Содержит выражение итога. Для установки свойства используется [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md): его нужно создать при помощи конструктора и присвоить свойству.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
