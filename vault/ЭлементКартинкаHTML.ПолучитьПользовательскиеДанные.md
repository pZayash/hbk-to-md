ЭлементКартинкаHTML (HTMLImageElement)

ПолучитьПользовательскиеДанные (GetUserData)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПользовательскиеДанные(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Ключ пользовательских данных.

Возвращаемое значение:

Тип: Произвольный.   
Неопределено, если указанному ключу не соответствовали никакие пользовательские данные.

Описание:

Получает пользовательские данные.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
