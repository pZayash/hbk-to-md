Глобальный контекст (Global context)

Log10 (Log10)

Доступен, начиная с версии 8.0.

Синтаксис:

Log10(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Исходное число, больше 0.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Значение результата вычисления логарифма.

Описание:

Вычисляет десятичный логарифм параметра <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

При отрицательном или нулевом значении параметра происходит ошибка "Неправильное значение аргумента встроенной функции (Log10)".

Пример:

|  |
| --- |
| Частота = 100; Попытка     Шкала = Log10(Частота);     Сообщить("Log10(Частота)="+Шкала); Исключение     Сообщить("При значении параметра "+Частота         +" выдается сообщение об ошибке:"         +Символы.ПС+""""+ОписаниеОшибки()+""""); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
