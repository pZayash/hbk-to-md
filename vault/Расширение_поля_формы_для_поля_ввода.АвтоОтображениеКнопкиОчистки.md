Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоОтображениеКнопкиОчистки (AutoShowClearButton)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Определяет способ автоматического отображения кнопки очистки в поле ввода формы клиентского приложения.

Доступность:

Мобильный клиент, сервер, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
