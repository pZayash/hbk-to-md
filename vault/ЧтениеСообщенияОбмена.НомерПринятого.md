ЧтениеСообщенияОбмена (ExchangeMessageReader)

НомерПринятого (ReceivedNo)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит максимальный номер сообщения, принятого узлом-отправителем данного сообщения от данного узла.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение.md) и после обращения к методу [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение.md) или [ПрерватьЧтение](ЧтениеСообщенияОбмена.ПрерватьЧтение.md) значением данного свойства будет Неопределено.

Использование в версии:

Доступен, начиная с версии 8.0.
