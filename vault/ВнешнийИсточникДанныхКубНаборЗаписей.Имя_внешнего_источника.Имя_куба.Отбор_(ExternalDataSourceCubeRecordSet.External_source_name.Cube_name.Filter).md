Содержит объект [Отбор](Отбор_(Filter).md), по которому осуществляется текущая фильтрация записей при считывании или записи набора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
