ОбластьОформления (AppearanceArea)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Представление>, <ТипПоля>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: Строка.   
Путь к данным добавляемого поля.

<Представление> (необязательный)

Тип: Строка.   
Представление добавляемого поля.  
Значение по умолчанию: Пустая строка.

<ТипПоля> (необязательный)

Тип: [ТипОбластиОформления](ТипОбластиОформления.md).   
Тип добавляемого поля.  
Значение по умолчанию: [Поле](ТипОбластиОформления.Поле.md).

Возвращаемое значение:

Тип: [ЭлементОбластиОформления](ЭлементОбластиОформления.md).   

Описание:

Добавляет поле в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
