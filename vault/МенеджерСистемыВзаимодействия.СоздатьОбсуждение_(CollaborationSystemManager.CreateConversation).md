МенеджерСистемыВзаимодействия (CollaborationSystemManager)

СоздатьОбсуждение (CreateConversation)

Доступен, начиная с версии 8.3.10.

Синтаксис:

СоздатьОбсуждение()

Возвращаемое значение:

Тип: [ОбсуждениеСистемыВзаимодействия](ОбсуждениеСистемыВзаимодействия_(CollaborationSystemConversation).md).   

Описание:

Создает объект [ОбсуждениеСистемыВзаимодействия](ОбсуждениеСистемыВзаимодействия_(CollaborationSystemConversation).md). Реальное обсуждение будет создано после вызова метода [Записать](ОбсуждениеСистемыВзаимодействия.Записать_(CollaborationSystemConversation.Write).md) объекта обсуждения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
