# ОписанияСтандартныхТабличныхЧастей

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Типы свойств объектов метаданных](Типы_свойств_объектов_метаданных.md) › [Стандартные реквизиты и стандартные табличные части](Стандартные_реквизиты_и_стандартные_табличные_части.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОписаниеСтандартнойТабличнойЧасти](ОписаниеСтандартнойТабличнойЧасти_(StandardTabularSectionDescription).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (начиная с 0).  

Описание:

Тип и значение описывают свойства стандартных табличных частей, имеющихся у объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
