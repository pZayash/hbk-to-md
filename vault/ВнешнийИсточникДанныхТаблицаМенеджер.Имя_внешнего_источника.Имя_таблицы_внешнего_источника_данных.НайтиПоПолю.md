ВнешнийИсточникДанныхТаблицаМенеджер.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableManager.<External source name>.<External data source table name>)

НайтиПоПолю (FindByField)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоПолю(<ИмяПоля>, <ЗначениеПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.   
Имя поля, как оно задано в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме [ХранилищеЗначения](ХранилищеЗначения.md) и строк произвольной длины.

<ЗначениеПоля> (обязательный)

Тип: Произвольный.   
Значение поля, по которому должен выполняться поиск.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   
Если не существует ни одного элемента с требуемым значением поля, то будет возвращена пустая ссылка.

Описание:

Осуществляет поиск элемента по значению поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных [ОбъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.ОбъектныеДанные.md).  
Если существует несколько элементов с указанным значением поля, то будет найден только один из них.  
Для реквизитов типа Строка поиск осуществляется по точному соответствию.

Использование в версии:

Доступен, начиная с версии 8.2.
