ЗаписьТекста (TextWriter)

Открыть (Open)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть файл

Синтаксис:

Открыть(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <Дописывать>, <КонвертируемыйРазделительСтрок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя записываемого файла.

Тип кодировки текста. Тип кодировки может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Разделяет строки в записываемом файле.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<Дописывать> (необязательный)

Тип: Булево.   
Если параметр принимает значение Истина, то содержимое файла сохраняется, если файл уже существует. Если файла не существует, создается новый с именем, указанным в первом параметре метода.  
Значение по умолчанию: Ложь.

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Указывает разделитель строк в файле при конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

Описание варианта метода:

Открывает текстовый файл для записи в монопольном режиме.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Разделяет строки в записываемом потоке.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Указывает разделитель строк в потоке при конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<ЗаписатьBOM> (необязательный)

Тип: Булево.   
Определяет, будет ли добавлена метка порядка байт (BOM) в начало потока.  
Значение по умолчанию: Ложь.

Описание варианта метода:

Устанавливает поток для записи.

Описание:

Открывает текстовый файл или поток для записи.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Открыть("d:\dos.txt", КодировкаТекста.OEM); |

Использование в версии:

Доступен, начиная с версии 8.0.
