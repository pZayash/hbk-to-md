Форма (Form)

ОбработкаОповещения (NotificationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаОповещения(<ИмяСобытия>, <Параметр>, <Источник>)

Параметры:

<ИмяСобытия>

Тип: Строка.   
Имя события. Может быть использовано для идентификации сообщений.

<Параметр>

Тип: Произвольный.   
Параметр, переданный в сообщении.

<Источник>

Источник события, переданный в сообщении.

Описание:

Возникает при оповещении всех созданных форм методом глобального контекста [Оповестить](Глобальный_контекст.Оповестить.md).

Доступность:

Толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Оповестить](Глобальный_контекст.Оповестить.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
