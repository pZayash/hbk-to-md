ИнтернетПочтовыйПрофиль (InternetMailProfile)

POP3ПередSMTP (POP3BeforeSMTP)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Способ аутентификации, когда подключение к SMTP серверу происходит после успешного подключения к POP3 серверу.  
Значение по умолчанию - Ложь.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - Подключение к серверу POP3 будет происходить перед подключением к SMTP серверу.

Использование в версии:

Доступен, начиная с версии 8.1.
