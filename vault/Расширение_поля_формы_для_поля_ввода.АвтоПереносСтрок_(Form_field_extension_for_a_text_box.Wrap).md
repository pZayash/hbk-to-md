# Расширение поля формы для поля ввода.АвтоПереносСтрок

Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоПереносСтрок (Wrap)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет автоматическим переносом строк.  
[Истина](lang__def_BooleanTrue.md) - выполняется автоперенос строк.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
