КатегорияСодержимогоHTML (HTMLContentCategory)

ОбработчикиСобытий (EventsHandlers)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все атрибуты, имя которых начинается на on.  
Например, <div onclick="test">Пример текста</div> будет преобразован в <div>Пример текста</div>.

Использование в версии:

Доступен, начиная с версии 8.3.13.
