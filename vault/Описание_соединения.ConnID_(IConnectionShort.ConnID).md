Описание соединения (IConnectionShort)

ConnID (ConnID)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит идентификатор соединения. Позволяет различить разные соединения, установленные одним и тем же приложением с одного и того же клиентского компьютера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
