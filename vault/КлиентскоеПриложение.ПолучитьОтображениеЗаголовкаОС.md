КлиентскоеПриложение (ClientApplication)

ПолучитьОтображениеЗаголовкаОС (GetOSCaptionRepresentation)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьОтображениеЗаголовкаОС()

Возвращаемое значение:

Тип: Булево, Неопределено.   

Описание:

Для тонкого и толстого клиентов под Windows (в режимах интерфейса [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси.md) и [Версия8\_2](ВариантИнтерфейсаКлиентскогоПриложения.Версия8_2.md)) возвращает, отображается ли заголовок операционной системы в главном окне.  
Для остальных клиентов возвращает Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
