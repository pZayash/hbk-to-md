# ДанныеИнформационнойБазыРаботыСРечью

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Работа с речью](Работа_с_речью.md)

Доступен, начиная с версии 8.3.23.

Описание:

Представляет собой данные регистрации информационной базы в сервисе распознавания речи.  
Используется для переноса данных регистрации в другую информационную базу через выгрузку и загрузку данных.   
В этот объект включаются идентификаторы моделей распознавания речи, сохраненные дополнительные грамматики, данные доступа к облаку распознавания.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер).   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/stt/core. Имя типа XDTO: RegistrationData.

Использование в версии:

Доступен, начиная с версии 8.3.23.
