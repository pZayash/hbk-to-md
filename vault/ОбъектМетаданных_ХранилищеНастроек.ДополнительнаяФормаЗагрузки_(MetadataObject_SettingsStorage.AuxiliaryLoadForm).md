ОбъектМетаданных: ХранилищеНастроек (MetadataObject: SettingsStorage)

ДополнительнаяФормаЗагрузки (AuxiliaryLoadForm)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма_(MetadataObject_Form).md) , [Неопределено](lang__def_Undefined.md).   
Указывает форму, которая будет использована для загрузки настроек в случае, если основная форма не установлена или не может быть использована в конкретном варианте использования клиентского приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение [Неопределено](lang__def_Undefined.md), и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
