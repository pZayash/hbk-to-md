# ЭлементВводаHTML.ФлажокУстановлен

ЭлементВводаHTML (HTMLInputElement)

ФлажокУстановлен (Checked)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство указывает, что переключатель или флажок должен быть установлен при первой загрузке HTML-документа.  
[Истина](lang__def_BooleanTrue.md) - устанавливается при первой загрузке.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
