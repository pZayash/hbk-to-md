[ДоступныеЗначения (AvailableValues)](Расширение_поля_ввода_системного_перечисления.ДоступныеЗначения.md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода.md), предназначенного для ввода ссылки на системное перечисление. Предназначен для задания набора значений перечисления, которые попадают в список выбора. Порядок следования значений в списке выбора в этом случае совпадает с порядком в списке доступных значений. В случае, если список пустой, доступными являются все значения.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
