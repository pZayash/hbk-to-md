ТипПараметраДоступнойТаблицыСхемыЗапроса (QuerySchemaAvailableTableParameterType)

Вариант (Variant)

Доступен, начиная с версии 8.3.5.

Описание:

В параметре указывается один из возможных вариантов.  
Возможные варианты получаются из свойства [Варианты](ПараметрДоступнойТаблицыСхемыЗапроса.Варианты_(QuerySchemaAvailableTableParameter.Variants).md) объекта типа [ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса_(QuerySchemaAvailableTableParameter).md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
