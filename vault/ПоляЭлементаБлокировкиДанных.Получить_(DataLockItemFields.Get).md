ПоляЭлементаБлокировкиДанных (DataLockItemFields)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс поля элемента блокировки данных.

Возвращаемое значение:

Тип: [ПолеЭлементаБлокировкиДанных](ПолеЭлементаБлокировкиДанных_(DataLockItemField).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.1.
