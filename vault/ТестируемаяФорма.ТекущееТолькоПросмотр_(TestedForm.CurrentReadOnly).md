# ТестируемаяФорма.ТекущееТолькоПросмотр

ТестируемаяФорма (TestedForm)

ТекущееТолькоПросмотр (CurrentReadOnly)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ТекущееТолькоПросмотр()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект не редактируется;   
[Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Возвращает признак того, что пользователь может изменить значение элемента без настройки формы, но путем переключения закладок и полос прокрутки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
