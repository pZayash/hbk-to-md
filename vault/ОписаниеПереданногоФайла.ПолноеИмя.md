ОписаниеПереданногоФайла (TransferredFileDescription)

ПолноеИмя (FullName)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит полное имя файла на клиенте, включая путь к нему. В веб-клиенте, если расширение работы с файлами не установлено, это свойство содержит пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
