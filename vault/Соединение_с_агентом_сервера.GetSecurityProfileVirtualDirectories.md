Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfileVirtualDirectories (GetSecurityProfileVirtualDirectories)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfileVirtualDirectories(<Кластер>, <Профиль>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого должен быть получен список виртуальных каталогов.

<Профиль> (обязательный)

Тип: Строка.   
Имя профиля безопасности, которому принадлежит список.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив объектов [Виртуальный каталог](Виртуальный_каталог.md).

Описание:

Получает разрешенные виртуальные каталоги.

Доступность:

Интеграция.

Примечание:

Если профиль с этим именем не найден, то исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
