Внешний модуль (ISecurityProfileExternalModule)

ExternalModuleHash (ExternalModuleHash)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит контрольную сумму внешнего модуля, вычисленная алгоритмом SHA-1, преобразованная к строке base64. Для построения контрольной суммы можно использовать:

- объект [ХешированиеДанных](ХешированиеДанных_(DataHashing).md) с указанием хеш-функции [SHA1](ХешФункция.SHA1_(HashFunction.SHA1).md);
- метод глобального контекста [Base64Строка](Глобальный_контекст.Base64Строка_(Global_context.Base64String).md).

Для расширений конфигурации может быть получена из свойства [ХешСумма](РасширениеКонфигурации.ХешСумма_(ConfigurationExtension.HashSum).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
