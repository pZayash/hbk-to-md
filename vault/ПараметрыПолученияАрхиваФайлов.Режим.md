ПараметрыПолученияАрхиваФайлов (GetFilesArchiveParameters)

Режим (Mode)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [РежимПолученияАрхиваФайлов](РежимПолученияАрхиваФайлов.md).   
Определяет, помещать ли все файлы в архив перед получением.  
Значение по умолчанию: [ПолучатьАрхивВсегда](РежимПолученияАрхиваФайлов.ПолучатьАрхивВсегда.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
