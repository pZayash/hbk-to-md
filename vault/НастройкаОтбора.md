Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияОтбором](ЭлементУправленияОтбором.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки отбора.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Добавить (Add)](НастройкаОтбора.Добавить.md)  
[Индекс (IndexOf)](НастройкаОтбора.Индекс.md)  
[Количество (Count)](НастройкаОтбора.Количество.md)  
[Найти (Find)](НастройкаОтбора.Найти.md)  
[Очистить (Clear)](НастройкаОтбора.Очистить.md)  
[Получить (Get)](НастройкаОтбора.Получить.md)  
[Удалить (Delete)](НастройкаОтбора.Удалить.md)  

Описание:

Набор элементов управления отбором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[Расширение табличного поля отбора](Расширение_табличного_поля_отбора.md), свойство [НастройкаОтбора](Расширение_табличного_поля_отбора.НастройкаОтбора.md)
