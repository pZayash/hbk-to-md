ДанныеФормыСтруктура (FormDataStructure)

<Имя свойства> (<Property name>)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Набор свойств универсальной структуры зависит от того, на основе какого объекта построена данная структура. Может включать в себя свойства, значения которых будут являться другие универсальные структуры, универсальные коллекции и наборы записей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
