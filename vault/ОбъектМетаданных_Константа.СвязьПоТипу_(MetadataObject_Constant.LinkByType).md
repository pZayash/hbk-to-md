ОбъектМетаданных: Константа (MetadataObject: Constant)

СвязьПоТипу (LinkByType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СвязьПоТипу](СвязьПоТипу_(TypeLink).md).   
Описывает связь по типу.  
Содержит путь к объекту метаданных, устанавливает связь с реквизитом, ограничивая тип вводимых значений для поля ввода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
