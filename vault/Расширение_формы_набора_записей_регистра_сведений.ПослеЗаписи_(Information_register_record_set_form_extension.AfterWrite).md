# Расширение формы набора записей регистра сведений.ПослеЗаписи

Расширение формы набора записей регистра сведений (Information register record set form extension)

ПослеЗаписи (AfterWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПослеЗаписи()

Описание:

Возникает после записи объекта, и после окончания транзакции записи, то есть к моменту вызова этой процедуры запись объекта полностью завершена.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
