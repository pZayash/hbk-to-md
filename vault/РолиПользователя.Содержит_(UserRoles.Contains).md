РолиПользователя (UserRoles)

Содержит (Contains)

Доступен, начиная с версии 8.0.

Синтаксис:

Содержит(<Роль>)

Параметры:

<Роль> (обязательный)

Тип: [ОбъектМетаданных: Роль](ОбъектМетаданных_Роль_(MetadataObject_Role).md).   
Искомая роль.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - роль содержится в коллекции; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, содержит ли коллекция используемых ролей пользователя указанную роль.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
