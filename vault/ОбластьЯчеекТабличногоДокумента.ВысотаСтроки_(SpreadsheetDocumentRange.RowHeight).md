# ОбластьЯчеекТабличногоДокумента.ВысотаСтроки

ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ВысотаСтроки (RowHeight)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту строк в области (в пунктах). Если свойство [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки_(SpreadsheetDocumentRange.AutoRowHeight).md) установлено [Истина](lang__def_BooleanTrue.md), - содержит максимальную высоту строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При присвоении свойству значения 0, значение свойства [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки_(SpreadsheetDocumentRange.AutoRowHeight).md) выставляется в [Истина](lang__def_BooleanTrue.md). При присвоении других значений свойство [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки_(SpreadsheetDocumentRange.AutoRowHeight).md) выставляется в [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.2.
