ПеречислимыеСвойстваОбъектовМетаданных (MetadataObjectEnumeratedProperties)

ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Доступен, начиная с версии 8.3.7.

Использование:

Только чтение.

Описание:

Тип: [ОбновлениеПредопределенныхДанных](ОбновлениеПредопределенныхДанных_(PredefinedDataUpdate).md).   
Содержит варианты указания режима обновления предопределенных данных. Используется в метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.7.
