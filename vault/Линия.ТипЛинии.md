Линия (Line)

ТипЛинии (LineType)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ТипЛинииРисункаТабличногоДокумента](ТипЛинииРисункаТабличногоДокумента.md), [ТипЛинииЯчейкиТабличногоДокумента](ТипЛинииЯчейкиТабличногоДокумента.md), [ТипЛинииДиаграммы](ТипЛинииДиаграммы.md).   
Содержит тип линии. Тип значения зависит от типа объекта, для которого используются линии. Для линий области табличного документа используется тип [ТипЛинииЯчейкиТабличногоДокумента](ТипЛинииЯчейкиТабличногоДокумента.md), для рисунка табличного документа [ТипЛинииРисункаТабличногоДокумента](ТипЛинииРисункаТабличногоДокумента.md), для линий диаграммы используется [ТипЛинииДиаграммы](ТипЛинииДиаграммы.md), для элементов графической схемы [ТипСоединительнойЛинии](ТипСоединительнойЛинии.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
