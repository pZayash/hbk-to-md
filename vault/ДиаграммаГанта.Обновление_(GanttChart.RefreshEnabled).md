# ДиаграммаГанта.Обновление

ДиаграммаГанта (GanttChart)

Обновление (RefreshEnabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак обновления (пересчета и отрисовки) диаграммы Ганта.   
Запрет обновления диаграммы Ганта рекомендуется на время наполнения данными в целях ускорения работы системы. Если обновление разрешено, пересчет и отрисовка производятся автоматически.  
[Истина](lang__def_BooleanTrue.md) - требуется обновление.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
