Элементы коллекции:

[ЯчейкаМакетаКоллекцииЗначенийОбластиКомпоновкиДанных](ЯчейкаМакетаКоллекцииЗначенийОбластиКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ячейки.  
Возможно обращение к ячейке посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Коллекция ячеек для вывода в коллекцию значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
