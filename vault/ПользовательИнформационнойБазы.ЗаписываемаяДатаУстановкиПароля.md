ПользовательИнформационнойБазы (InfoBaseUser)

ЗаписываемаяДатаУстановкиПароля (WritablePasswordSettingDate)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Доступно для записи только пользователю с административными правами.  
Если данное свойство было изменено пользователем с административными правами, пользователю в качестве даты установки пароля устанавливается заданная дата.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
