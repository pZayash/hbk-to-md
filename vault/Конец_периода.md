Конец периода (необязательный)

Тип параметра: [Дата](lang__def_Date.md), [МоментВремени](МоментВремени_(PointInTime).md), [Граница](Граница_(Boundary).md).   
Указывается конец периода расчета итогов. Значение конца периода по умолчанию включается в период расчета итогов. Для исключения значения из периода его следует задавать объектом [Граница](Граница_(Boundary).md) с исключением граничного значения.  
Если параметр не задан, итоги рассчитываются по самую последнюю запись.
