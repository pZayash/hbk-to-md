# ТочкаДиаграммы.Значение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Диаграмма](Диаграмма.md) › [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md) › [Свойства](ТочкаДиаграммы__Свойства.md)

ТочкаДиаграммы (ChartPoint)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение, идентифицирующее точку в методе диаграммы [УстановитьТочку](Диаграмма.УстановитьТочку_(Chart.SetPoint).md). Строковое представление значения используется для представления точки, если свойство [Текст](ТочкаДиаграммы.Текст_(ChartPoint.Text).md) не установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство [Текст](ТочкаДиаграммы.Текст_(ChartPoint.Text).md) может быть сформировано программой автоматически, если свойство [АвтоУстановкаТекстаТочек](Диаграмма.АвтоУстановкаТекстаТочек_(Chart.AutoPointText).md) = [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
