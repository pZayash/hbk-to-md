# ДокументPDF.ПроверитьПодписиАсинх

ДокументPDF (PDFDocument)

ПроверитьПодписиАсинх (VerifySignaturesAsync)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПроверитьПодписиАсинх(<МенеджерКриптографии>)

Параметры:

<МенеджерКриптографии> (необязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект Массив, содержащий сертификаты, с помощью которого была произведена подпись (если сертификат включен в данные подписи).

Описание:

Асинхронная функция. Проверяет действительность присоединенных к PDF-документу подписей.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если проверка подписи завершилась неудачно, генерируется исключение категории [ОшибкаПроверкиПодписи](КатегорияОшибки.ОшибкаПроверкиПодписи_(ErrorCategory.SignatureVerificationError).md).

Использование в версии:

Доступен, начиная с версии 8.3.21.
