ДиалогОтбораВерсийИсторииДанных (DataHistoryVersionsFilterDialog)

Показать (Show)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога выбора со следующими параметрами:

- <Отбор> - Если диалог был закрыт нажатием кнопки "OK", содержит структуру идентичную структуре в параметре <Отбор> метода [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии.md), иначе Неопределено.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).
- Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Если свойство [Данные](ДиалогОтбораВерсийИсторииДанных.Данные.md) не установлено, генерируется исключение.

Описание:

Начинает отображение диалога отбора версий.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ДиалогОтбораВерсийИсторииДанных](ДиалогОтбораВерсийИсторииДанных.md), метод [ОткрытьАсинх](ДиалогОтбораВерсийИсторииДанных.ОткрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.18.
