АдминистрированиеРабочийСервер (AdministrationWorkServer)

ПолучитьКаталогиДанныхСервисаДляПереноса (GetServiceDataDirsForTransfer)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьКаталогиДанныхСервисаДляПереноса(<ИмяСервиса>)

Параметры:

<ИмяСервиса> (необязательный)

Тип: Строка.   
Имя сервиса кластера как в [АдминистрированиеСервис](АдминистрированиеСервис.md).

Возвращаемое значение:

Тип: Массив.   

Описание:

Возвращает массив каталогов для переноса данных сервисов для измененных настроек. Если настройка применена, то данные для нее не отображаются в списке.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

См. также:

[КаталогДанныхСервисаДляПереноса](КаталогДанныхСервисаДляПереноса.md)  

Использование в версии:

Доступен, начиная с версии 8.3.26.
