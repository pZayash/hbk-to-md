АдминистрированиеСервера (ServerAdministration)

ВыполнитьАутентификацию (Authenticate)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ВыполнитьАутентификацию(<Пароль>, <ИмяПользователя>)

Параметры:

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя для аутентификации пользователя паролем. Пользователь будет аутентифицирован, если совпадает его имя и пароль.  
Значение по умолчанию: Пустая строка.

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя агента администрирования сервера "1С:Предприятие".  
Значение по умолчанию: Пустая строка.

Описание:

Выполняет аутентификацию администратора центрального сервера в агенте администрирования сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
