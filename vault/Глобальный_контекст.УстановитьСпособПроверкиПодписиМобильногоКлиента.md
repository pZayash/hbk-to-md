Глобальный контекст (Global context)

УстановитьСпособПроверкиПодписиМобильногоКлиента (SetMobileClientSignatureVerificationMethod)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьСпособПроверкиПодписиМобильногоКлиента(<СпособПроверки>)

Параметры:

<СпособПроверки> (необязательный)

Тип: [СпособПроверкиПодписиМобильногоКлиента](СпособПроверкиПодписиМобильногоКлиента.md).   
Устанавливаемый способ проверки.

Описание:

Устанавливает новый способ проверки подписи мобильного клиента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Описание изменено в версии 8.3.20.
