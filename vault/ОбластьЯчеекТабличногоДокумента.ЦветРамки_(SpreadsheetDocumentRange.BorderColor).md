ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ЦветРамки (BorderColor)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет рамки области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не используется, если рамка имеет стиль [Выпуклая](ТипРамкиЭлементаУправления.Выпуклая_(ControlBorderType.Embossed).md) или [Вдавленная](ТипРамкиЭлементаУправления.Вдавленная_(ControlBorderType.Indented).md) - в этом случае рамка рисуется притененным цветом от цвета фона.

Использование в версии:

Доступен, начиная с версии 8.2.
