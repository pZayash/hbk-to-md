ПотокВПамяти (MemoryStream)

РазмерАсинх (SizeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазмерАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Число](lang__def_Number.md), размер данных в байтах.

Описание:

Асинхронная функция. Получает размер данных в байтах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) = [Ложь](lang__def_BooleanFalse.md), то при вызове этого метода будет сгенерировано исключение.
