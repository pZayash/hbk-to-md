Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), в котором отображается список видов расчета. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) содержит значение типа [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержит коллекцию значений, которая задается колонками объекта [ПланВидовРасчетаСписок.](ПланВидовРасчетаСписок.Имя_плана_видов_расчета_(ChartOfCalculationTypesList.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСписок.Имя_плана_видов_расчета_(ChartOfCalculationTypesList.Chart_of_calculation_types_name).md)>, связанного с табличным полем.
