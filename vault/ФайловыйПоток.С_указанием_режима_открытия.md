ФайловыйПоток (FileStream)

С указанием режима открытия

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый ФайловыйПоток(<ИмяФайла>, <РежимОткрытия>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла.md).   
Режим открытия файла.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер временного буфера, используемого для работы.

Описание:

Открывает заданный файл в указанном режиме с возможностью чтения и записи, с общим доступом на чтение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
