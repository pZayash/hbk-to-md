Расширение колонки табличного поля таблицы значений (Value table table box column extension)

ОтображатьИтогиВПодвале (ShowTotalsInFooter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения в подвале табличного поля итогов по колонке таблицы значений.  
[Истина](lang__def_BooleanTrue.md) - отображается.

Доступность:

Толстый клиент.

Примечание:

Свойство доступно для колонок, имеющих тип [Число](lang__def_Number.md).

Использование в версии:

Доступен, начиная с версии 8.0.
