# Расширение формы набора записей регистра бухгалтерии.ПриЗаписи

Расширение формы набора записей регистра бухгалтерии (Accounting register record set form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.ПриЗаписи_(AccountingRegisterRecordSet.Accounting_register_name.OnWrite).md) объекта [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
