Индикатор (ProgressBar)

МинимальноеЗначение (MinValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит минимальное значение диапазона изменения. Если значение меньше или равно минимальному, то оно будет отображаться "пустым" индикатором (0%).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
