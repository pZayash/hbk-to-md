Соединение с агентом сервера (IServerAgentConnection)

AuthenticateAgent (AuthenticateAgent)

Доступен, начиная с версии 8.1.

Синтаксис:

AuthenticateAgent(<Имя>, <Пароль>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя администратора центрального сервера.

<Пароль> (обязательный)

Тип: Строка.   
Пароль администратора центрального сервера.

Описание:

Выполняет аутентификацию администратора центрального сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
