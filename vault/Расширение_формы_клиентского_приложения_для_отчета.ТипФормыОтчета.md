Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ТипФормыОтчета (ReportFormType)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ТипФормыОтчета](ТипФормыОтчета.md).   
Содержит тип формы отчета.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
