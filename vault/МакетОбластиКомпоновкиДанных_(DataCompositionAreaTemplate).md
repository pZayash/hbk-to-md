**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макеты областей](Макеты_областей.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableRow).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы макета области.  
Возможно обращение к элемент макета области посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Данный объект представляет собой коллекцию объектов типа [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableRow).md) и содержит описание выводимой области компоновки данных (выводимые поля, цвета, шрифты и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: AreaTemplate.
