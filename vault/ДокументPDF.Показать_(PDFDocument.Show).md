ДокументPDF (PDFDocument)

Показать (Show)

Доступен, начиная с версии 8.3.22.

Синтаксис:

Показать(<Заголовок>, <ИмяФайла>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок окна PDF-документа.

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для сохранения PDF-документа. Если значение не пустое, то при закрытии окна, если документ был изменен, будет предложено сохранить PDF-документ в указанный файл.

Описание:

Открывает окно для показа и редактирования PDF-документа.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если PDF-документ является реквизитом управляемой формы, то вызов метода приводит к возникновению исключения.

Пример:

|  |
| --- |
| ДокPDF.Показать("Заявки от покупателей", "Заявки.pdf"); |

Использование в версии:

Доступен, начиная с версии 8.3.22.
