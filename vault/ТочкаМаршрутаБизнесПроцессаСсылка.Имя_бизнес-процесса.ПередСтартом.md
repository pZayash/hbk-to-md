ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередСтартом (BeforeStart)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСтартом(<ТочкаМаршрута>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса вида [Старт](ВидТочкиМаршрутаБизнесПроцесса.Старт.md), на которой происходит старт бизнес-процесса.

<Отказ>

Тип: Булево.   
Признак отказа от старта бизнес-процесса. Если необходимые условия для начала старта не выполнены (например, не выполнены какие-то внешние условия или не установлены значения реквизитов, необходимые для корректного старта бизнес-процесса), то в теле процедуры-обработчика следует установить данному параметру значение Истина. В этом случае старт бизнес-процесса не будет произведен.  
Значение по умолчанию: Ложь.

Описание:

Вызывается из метода [Старт](БизнесПроцессОбъект.Имя_бизнес-процесса.Старт.md) непосредственно перед выполнением старта бизнес-процесса и до начала перехода на следующую за стартом точку маршрута.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точки вида [Старт](ВидТочкиМаршрутаБизнесПроцесса.Старт.md).  
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
