ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

СпособВыбора (ChoiceMode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [СпособВыбора](СпособВыбора_(ChoiceMode).md).   
Определяет способ выбора объектов базы данных (например, [ИзФормы](СпособВыбора.ИзФормы_(ChoiceMode.FromForm).md), [БыстрыйВыбор](СпособВыбора.БыстрыйВыбор_(ChoiceMode.QuickChoice).md), [ОбоимиСпособами](СпособВыбора.ОбоимиСпособами_(ChoiceMode.BothWays).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
