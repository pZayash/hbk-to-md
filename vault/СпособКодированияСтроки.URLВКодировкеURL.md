СпособКодированияСтроки (StringEncodingMethod)

URLВКодировкеURL (URLInURLEncoding)

Доступен, начиная с версии 8.3.1.

Описание:

Кодировать (раскодировать) строку URL в URL кодировке. Спецсимволы URL ( !#$%'()\*+,/:;=?@[] ) не кодируются.

Использование в версии:

Доступен, начиная с версии 8.3.1.
