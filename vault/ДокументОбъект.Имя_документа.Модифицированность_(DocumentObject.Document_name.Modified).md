# ДокументОбъект.<Имя документа>.Модифицированность

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект изменен; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, был ли изменен объект после считывания из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не позволяет определить, был ли изменен объект другими пользователями.

Пример:

|  |
| --- |
| Если РасходнаяНакладная.Модифицированность() Тогда     Предупреждение(НСтр("ru = 'Документ был изменен.';"      + " en = 'Document was changed.'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
