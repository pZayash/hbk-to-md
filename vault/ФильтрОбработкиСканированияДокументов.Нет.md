ФильтрОбработкиСканированияДокументов (DocumentScanningProcessingFilter)

Нет (None)

Доступен, начиная с версии 8.3.23.

Описание:

Не использовать фильтры при получении документа.   
Данную настройку рекомендуется добавлять на случай непредвиденных ситуаций.  
Если предполагается автоматическая обработка отсканированного документа, рекомендуется оставлять фильтр "Нет" в качестве единственного.

Использование в версии:

Доступен, начиная с версии 8.3.23.
