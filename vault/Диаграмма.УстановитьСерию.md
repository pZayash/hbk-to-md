Диаграмма (Chart)

УстановитьСерию (SetSeries)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСерию(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Серия, которую требуется найти или создать.

Возвращаемое значение:

Тип: [СерияДиаграммы](СерияДиаграммы.md).   
Найденная серия. Если указанной серии нет, создается новая серия, для нее заполняется свойство [Значение](СерияДиаграммы.Значение.md), и эта серия возвращается.

Описание:

Предназначен для создания и доступа к сериям по ключевому значению. Позволяет выполнять заполнение диаграммы в произвольном порядке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Важно! Во время заполнения диаграммы свойство [АвтоТранспонирование](Диаграмма.АвтоТранспонирование.md) не должно быть установлено. Его можно устанавливать после заполнения.

Использование в версии:

Доступен, начиная с версии 8.0.
