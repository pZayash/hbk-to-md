# РегистрНакопленияНаборЗаписей.<Имя регистра накопления>.Записать

РегистрНакопленияНаборЗаписей.<Имя регистра накопления> (AccumulationRegisterRecordSet.<Accumulation register name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [РежимЗамещения](РежимЗамещения_(ReplacementMode).md).   
Определяет режим замещения существующей записи в соответствии с текущими установками отбора. [Истина](lang__def_BooleanTrue.md) - перед записью существующие записи будут удалены. [Ложь](lang__def_BooleanFalse.md) или [Добавление](РежимЗамещения.Добавление_(ReplacementMode.Append).md) - записи будут дописаны к уже существующим в информационной базе записям.  
[Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) - записи регистра, совпадающие с записями набора записей по значениям ключевых полей (Регистратор, НомерСтроки, разделители), будут удалены из регистра.   
[Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) - существующие записи регистра, совпадающие по значениям ключевых полей (Регистратор, НомерСтроки, разделители) с записями набора записей будут перезаписаны. Если для какой-либо записи набора записей нет соответствующей записи регистра, будет сгенерировано исключение.   
[Слияние](РежимЗамещения.Слияние_(ReplacementMode.Merge).md) является недопустимым значением параметра для регистра накопления.  
Режимы [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) и [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) могут быть использованы только в том случае, если свойство набора записей [РасширенныеРежимыЗамещения](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.РасширенныеРежимыЗамещения_(AccumulationRegisterRecordSet.Accumulation_register_name.ExtendedReplaceMode).md) имеет значение [Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает в базу данных набор записей регистра накопления. В зависимости от переданного параметра может быть выполнено добавление записей или их замещение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу набор записей очищается (удаляются записи из набора).

Пример:

|  |
| --- |
| // Очистим записи документа, если он был ранее проведен  Если Проведен Тогда     Рег = РегистрыНакопления.ЗаказыПоставщикам;     НаборЗаказыПоставщикам  = Рег.СоздатьНаборЗаписей();     НаборЗаказыПоставщикам.Отбор.Регистратор.Установить(Ссылка);     НаборЗаказыПоставщикам.Записать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
