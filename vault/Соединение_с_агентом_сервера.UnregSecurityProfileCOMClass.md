Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileCOMClass (UnregSecurityProfileCOMClass)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileCOMClass(<Кластер>, <ИмяПрофиля>, <ComClassName>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должна быть отменена регистрация разрешенного COM-класса.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности.

<ComClassName> (обязательный)

Тип: Строка.   
Имя [COM-класс](COM-класс.md), регистрация которого отменяется.

Описание:

Отменяет регистрацию разрешенного [COM-класс](COM-класс.md).

Доступность:

Интеграция.

Примечание:

Если профиля с именем, указанным в параметре <ИмяПрофиля>, нет, то метод ничего не делает.  
Если [COM-класс](COM-класс.md) с ключом, указанным в параметре <ComClassName>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
