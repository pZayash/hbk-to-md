ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ОсновнаяФормаДляВыбора (DefaultChoiceForm)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма_(MetadataObject_Form).md) , [Неопределено](lang__def_Undefined.md).   
Описывает форму, которая будет вызвана системой для выполнения стандартных действий для выбора элемента вида расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Неопределено](lang__def_Undefined.md) - если основная форма не была задана на этапе конфигурирования. В этом случае система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.1.
