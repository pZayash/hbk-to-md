ОбъектМетаданных: Константа (MetadataObject: Constant)

МногострочныйРежим (MultiLine)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает на многострочность поля. Если свойство установлено ([Истина](lang__def_BooleanTrue.md)), то текст может быть отредактирован с использованием переноса строк.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
