МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьДоступностьПолученияЛицензии (GetLicenseAcquisitionAvailability)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьДоступностьПолученияЛицензии(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии_(LicenseAcquisitionRequest).md).   
Используются только значения свойств: [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md), [Порт](ЗапросНаПолучениеЛицензии.Порт_(LicenseAcquisitionRequest.Port).md), [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей_(LicenseAcquisitionRequest.ForAllUsers).md).   
Если [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md) - пустая строка, то проверку выполняет текущее приложение на текущем компьютере, иначе проверку на своем компьютере выполняет приложение, адрес которого указан параметрами [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md) и [Порт](ЗапросНаПолучениеЛицензии.Порт_(LicenseAcquisitionRequest.Port).md).   
Если [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей_(LicenseAcquisitionRequest.ForAllUsers).md) - [Истина](lang__def_BooleanTrue.md), то проверяется возможность записи файла лицензии в подкаталог лицензий каталога данных приложений всех пользователей, иначе - в подкаталог лицензий каталога данных приложений текущего пользователя.

Возвращаемое значение:

Тип: [ДоступностьПолученияЛицензий](ДоступностьПолученияЛицензий_(LicenseAcquisitionAvailability).md).   

Описание:

Проверяет возможность получения лицензии для всех пользователей или для текущего пользователя заданного компьютера.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
