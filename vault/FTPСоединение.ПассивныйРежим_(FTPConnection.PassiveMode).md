FTPСоединение (FTPConnection)

ПассивныйРежим (PassiveMode)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим работы ftp-соединения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - активный режим, [Истина](lang__def_BooleanTrue.md) - пассивный.

Использование в версии:

Доступен, начиная с версии 8.0.
