# ЗадачаОбъект.<Имя задачи>.ПроверитьВыполнение

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПроверитьВыполнение (CheckExecution)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьВыполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если все достаточные условия выполнения задачи соблюдены, [Ложь](lang__def_BooleanFalse.md) – в противном случае.  
Если обработчики не определены, возвращает [Ложь](lang__def_BooleanFalse.md).

Описание:

Показывает результат вызова обработчиков события [ОбработкаПроверкиВыполнения](ЗадачаОбъект.Имя_задачи.ОбработкаПроверкиВыполнения_(TaskObject.Task_name.OnCheckExecutionProcessing).md) у задачи и обработчика события [ОбработкаПроверкиВыполнения](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаПроверкиВыполнения_(BusinessProcessRoutePointRef.Business_process_name.OnCheckExecuteProcessing).md) у точки. Если обработчик задачи вернул [Ложь](lang__def_BooleanFalse.md), то обработчик у точки не вызывается. Используется, например, для автоматизированного выполнения задач роботами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
