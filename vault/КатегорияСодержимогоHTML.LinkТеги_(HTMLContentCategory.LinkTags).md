КатегорияСодержимогоHTML (HTMLContentCategory)

LinkТеги (LinkTags)

Доступен, начиная с версии 8.3.13.

Описание:

Из документа будут удалены все элементы <link>, у которых атрибут type не равен css, т.е.:  
<link type="text/html"> будет удален, а <link type="css"> удален не будет.

Использование в версии:

Доступен, начиная с версии 8.3.13.
