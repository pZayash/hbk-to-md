РегламентноеЗадание (ScheduledJob)

ИнтервалПовтораПриАварийномЗавершении (RestartIntervalOnFailure)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Интервал в секундах, через который нужно перезапускать задание в случае его аварийного завершения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Возможность чтения и записи доступны только для администратора.

Использование в версии:

Доступен, начиная с версии 8.1.
