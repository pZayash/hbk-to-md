ПолеВвода (TextBox)

ВыборНезаполненного (ChoiceIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство указывает, нужно ли сейчас в поле в нужный момент выполнять автовыбор значения или нет. Если свойство имеет значение [Истина](lang__def_BooleanTrue.md), то независимо от хранимых в поле ввода данных будет выполняться автовыбор значения, если [Ложь](lang__def_BooleanFalse.md) - то независимо от хранимых в поле ввода данных автовыбор выполняться не будет.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
