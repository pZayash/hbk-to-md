Описание:

Содержит варианты заполнения области табличного документа при выводе в результирующий документ.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/data/spreadsheet. Имя типа XDTO: SpreadsheetDocumentAreaFillType.
