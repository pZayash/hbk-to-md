РежимОткрытияФайла (FileOpenMode)

Открыть (Open)

Доступен, начиная с версии 8.3.9.

Описание:

Открытие существующего файла. Если файл не существует, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
