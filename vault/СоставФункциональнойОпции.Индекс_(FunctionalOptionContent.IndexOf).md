СоставФункциональнойОпции (FunctionalOptionContent)

Индекс (IndexOf)

Доступен, начиная с версии 8.2.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементСоставаФункциональнойОпции](ЭлементСоставаФункциональнойОпции_(FunctionalOptionContentItem).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
