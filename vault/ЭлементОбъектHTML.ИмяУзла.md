ЭлементОбъектHTML (HTMLObjectElement)

ИмяУзла (NodeName)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя узла HTML. Имя узла присутствует у каждого типа узла HTML.  
Узел Имя узла

- [Атрибут](ТипУзлаDOM.Атрибут.md) - [Имя](АтрибутDOM.Имя.md);
- [Комментарий](ТипУзлаDOM.Комментарий.md) - содержит текст комментария;
- [Документ](ТипУзлаDOM.Документ.md) - документ;
- [Элемент](ТипУзлаDOM.Элемент.md) - [ИмяЭлемента](ЭлементHTML.ИмяЭлемента.md);
- [Текст](ТипУзлаDOM.Текст.md) - текст.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В модели HTML узла имена узлов не чувствительны к регистру.  
Имя узла, в отличие от локального имени узла, может содержать префикс, который отсутствует у локального имени.

Использование в версии:

Доступен, начиная с версии 8.2.
