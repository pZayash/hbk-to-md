СпособОтображенияОкна (WindowAppearanceMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВариантСпособаОтображенияОкна](ВариантСпособаОтображенияОкна_(WindowAppearanceModeVariant).md).   
Указывает способ отображения окна формы.

Доступность:

Толстый клиент.

Примечание:

Управление свойством должно производиться до открытия формы.

Использование в версии:

Доступен, начиная с версии 8.0.
