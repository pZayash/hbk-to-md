ТабличныйДокумент (SpreadsheetDocument)

ОтменитьПреобразованиеДокумента (RevertToDefaultMode)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОтменитьПреобразованиеДокумента()

Описание:

Отменяет преобразование документа в компактное представление, возвращает документ в исходное состояние.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
