ИнтернетПочтовоеСообщение (InternetMailMessage)

УстановитьПолеЗаголовка (SetField)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьПолеЗаголовка(<ИмяПоля>, <ЗначениеПоля>, <СпособКодирования>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля помещаемого в главный заголовок сообщения. Имя может состоять только из печатных символов набора ASCII (т.е. из символов диапазона от 33 до 126 включительно), за исключением символа "двоеточие" (":"). Данный диапазон содержит все буквы английского алфавита (как строчные, так и прописные) и несколько специальных символов.

<ЗначениеПоля> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [ИнтернетПочтовыйАдрес](ИнтернетПочтовыйАдрес_(InternetMailAddress).md), [ИнтернетПочтовыеАдреса](ИнтернетПочтовыеАдреса_(InternetMailAddresses).md).   
Значение добавляемого поля.

<СпособКодирования> (необязательный)

Тип: [СпособКодированияНеASCIIСимволовИнтернетПочтовогоСообщения](СпособКодированияНеASCIIСимволовИнтернетПочтовогоСообщения_(InternetMailMessageNonASCIISymbolsEncodingMode).md).   
Способ кодирования поля в заголовке. Если параметр не задан, будет использоваться значение из свойства [СпособКодированияНеASCIIСимволов](ИнтернетПочтовоеСообщение.СпособКодированияНеASCIIСимволов_(InternetMailMessage.NonASCIISymbolsEncodingMode).md).

Описание:

Добавляет пользовательское поле в главный заголовок почтового сообщения. Текстовые данные будут конвертироваться в кодировку, заданную свойством [Кодировка](ИнтернетПочтовоеСообщение.Кодировка_(InternetMailMessage.Encoding).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если указанное поле существует, оно перезаписывается новым значением.

Использование в версии:

Доступен, начиная с версии 8.1.
