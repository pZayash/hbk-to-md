Расширение поля формы для поля планировщика (Form field extension for a planner)

ГиперссылкаЭлементаШкалыВремени (TimeScaleItemHyperlink)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет внешний вид и поведение при нажатии левой кнопкой мыши на элемент шкалы времени:

- Истина - вызывается событие [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд.md).
- Ложь - никакие действия не выполняются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
