# Расширение формы клиентского приложения для задачи.Записать

Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Состав параметров произвольный. Содержит предопределенный параметр записи ВыполнитьЗадачу. Если параметр ВыполнитьЗадачу=[Истина](lang__def_BooleanTrue.md), то задача будет выполнена.  
Состав других параметров произвольный.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - успешно записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
