Менеджер COM-соединений (COM connector)

ConnectAgent (ConnectAgent)

Доступен, начиная с версии 8.1.

Синтаксис:

ConnectAgent(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор протокола (по умолчанию tcp), имя компьютера или IP-адрес агента сервера (обязательно), номер IP-порта агента сервера (по умолчанию 1540). Например, значением параметра могут быть следующие строки: «server1», «tcp://server1», «tcp://server1:1540», «server1:1540».

Возвращаемое значение:

Тип: [Соединение с агентом сервера](Соединение_с_агентом_сервера_(IServerAgentConnection).md).   

Описание:

Выполняет подсоединение к агенту сервера 1С:Предприятия 8, работающему на указанном серверном компьютере.

Доступность:

Интеграция.

Примечание:

Если подсоединение невозможно, то вызывается исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
