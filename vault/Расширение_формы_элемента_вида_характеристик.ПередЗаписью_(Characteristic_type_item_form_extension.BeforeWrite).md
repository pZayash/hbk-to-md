# Расширение формы элемента вида характеристик.ПередЗаписью

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [Расширение формы элемента вида характеристик](Расширение_формы_элемента_вида_характеристик_(Characteristic_type_item_form_extension).md) › [События](Расширение_формы_элемента_вида_характеристик__События.md)

Расширение формы элемента вида характеристик (Characteristic type item form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи элемента. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись элемента выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи элемента вида характеристики из формы. Процедура-обработчик данного события вызывается перед вызовом процедуры обработчика события [ПередЗаписью](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПередЗаписью_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.BeforeWrite).md) объекта [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
