Файл (File)

УстановитьТолькоЧтениеАсинх (SetReadOnlyAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьТолькоЧтениеАсинх(<ТолькоЧтение>)

Параметры:

<ТолькоЧтение> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - установить атрибут "Только чтение", [Ложь](lang__def_BooleanFalse.md) - снять атрибут "Только чтение".

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Устанавливает значение атрибута файла "ReadOnly". Применима только к файлам или каталогам, которые реально существуют.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.

[Файл](Файл_(File).md), метод [НачатьУстановкуТолькоЧтения](Файл.НачатьУстановкуТолькоЧтения_(File.BeginSettingReadOnly).md)  
[Файл](Файл_(File).md), метод [УстановитьТолькоЧтение](Файл.УстановитьТолькоЧтение_(File.SetReadOnly).md)

Доступен, начиная с версии 8.3.18.
