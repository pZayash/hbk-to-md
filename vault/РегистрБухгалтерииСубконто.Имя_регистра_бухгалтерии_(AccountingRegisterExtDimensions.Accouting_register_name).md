Элементы коллекции:

[КлючИЗначение](КлючИЗначение_(KeyAndValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются Ключ - вид субконто, Значение - значение субконто.  
Возможно обращение к значению субконто посредством оператора [...]. В качестве аргумента передается вид субконто ([ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)>).

Описание:

Коллекция значений субконто записи регистра бухгалтерии. Установка и получение значения конкретного субконто осуществляется через оператор [], в качестве параметра которому передается вид субконто.  
Возвращается значение типа [КлючИЗначение](КлючИЗначение_(KeyAndValue).md). В качестве ключа вид субконто, в качестве значения значение субконто.

Доступность:

Сервер, толстый клиент, внешнее соединение.
