ОбъектМетаданных: ПланВидовХарактеристик (MetadataObject: ChartOfCharacteristicTypes)

ОсновнаяФормаДляВыбораГруппы (DefaultFolderChoiceForm)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма_(MetadataObject_Form).md) , [Неопределено](lang__def_Undefined.md).   
Описывает форму, которая будет вызвана системой для выполнения стандартных действий при выборе группы плана видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение [Неопределено](lang__def_Undefined.md), и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.1.
