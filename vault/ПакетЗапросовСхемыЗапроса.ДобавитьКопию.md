ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

ДобавитьКопию (AddCopy)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ДобавитьКопию(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента, который нужно скопировать.

Возвращаемое значение:

Тип: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса.md).   
[ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса.md) или [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса.md), в зависимости от того, что находилось по переданному индексу.

Описание:

Добавляет копию запроса. В параметре передается индекс запроса, который требуется скопировать.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
