# XBase.СоздатьИндексныйФайл

XBase (XBase)

СоздатьИндексныйФайл (CreateIndex)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьИндексныйФайл(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя создаваемого индексного файла.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - индексный файл создан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Создает индексный файл. Создаваемый индексный файл будет содержать все индексы, которые были созданы на текущий момент.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Созданный файл будет иметь расширение "CDX", независимо от используемой операционной системы.

Пример:

|  |
| --- |
| xB.СоздатьИндексныйФайл(ИмяФайлаИндекса); |

Использование в версии:

Доступен, начиная с версии 8.0.
