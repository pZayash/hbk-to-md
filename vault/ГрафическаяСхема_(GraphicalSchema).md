Описание:

Используется для доступа к свойствам графической схемы. При помощи этого объекта возможно управление свойствами графической схемы, получение элементов схемы, чтение и запись их свойств.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/graphscheme}. Имя типа XDTO: FlowchartContextType.
