ЗаписьДанных (DataWriter)

НачатьЗапись (BeginWrite)

Доступен, начиная с версии 8.3.8.

Вариант синтаксиса: Запись двоичных данных

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ДвоичныеДанные>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Объект, данные которого будут записаны в поток.

Описание варианта метода:

Записывает в поток данные из объекта типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Запись результата чтения данных

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <РезультатЧтенияДанных>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<РезультатЧтенияДанных> (обязательный)

Тип: [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).   
Объект типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md), данные которого будут записаны в поток.

Описание варианта метода:

Записывает в целевой поток данные из объекта типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Описание:

Вызывает начало записи данных в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Доступен, начиная с версии 8.3.8.
