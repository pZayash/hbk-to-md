Доступен, начиная с версии 8.3.27.

Синтаксис:

УстановитьНастройкиАутентификацииЧерезЭлектроннуюПочту(<НастройкиАутентификацииЧерезЭлектроннуюПочту>)

Параметры:

<НастройкиАутентификацииЧерезЭлектроннуюПочту> (обязательный)

Тип: [НастройкиАутентификацииЧерезЭлектроннуюПочту](НастройкиАутентификацииЧерезЭлектроннуюПочту_(EmailAuthenticationSettings).md).

Описание:

Изменяет настройки аутентификации через электронную почту.

Доступность:

Сервер, толстый клиент.
