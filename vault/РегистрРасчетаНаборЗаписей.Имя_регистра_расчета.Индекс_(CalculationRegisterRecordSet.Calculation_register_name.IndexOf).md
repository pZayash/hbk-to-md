РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [РегистрРасчетаЗапись.\<Имя регистра расчета\>](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md).   
Запись набора, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной записи в наборе. Если не найдено, то возвращается -1.

Описание:

Получает индекс записи в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
