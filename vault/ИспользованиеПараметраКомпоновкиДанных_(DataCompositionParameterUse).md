# ИспользованиеПараметраКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Система компоновки данных](Система_компоновки_данных.md)

Доступен, начиная с версии 8.2.

Значения

[Авто (Auto)](ИспользованиеПараметраКомпоновкиДанных.Авто_(DataCompositionParameterUse.Auto).md)  
[Всегда (Always)](ИспользованиеПараметраКомпоновкиДанных.Всегда_(DataCompositionParameterUse.Always).md)  

Описание:

Содержит возможные варианты использования параметра компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: DataCompositionParameterUse.

Использование в версии:

Доступен, начиная с версии 8.2.
