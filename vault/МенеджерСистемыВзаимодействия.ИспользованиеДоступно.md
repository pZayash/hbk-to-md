МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ИспользованиеДоступно (CanUse)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ИспользованиеДоступно()

Возвращаемое значение:

Тип: Булево.   
Возвращает Истина, если информационная база зарегистрирована, регистрация информационной базы на сервере взаимодействия не отменена, и в базе есть пользователи.

Описание:

Проверяет состояние регистрации информационной базе на сервере взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Проверяет настройки регистрации информационной базы, но не физическую доступность сервера системы взаимодействия.

Использование в версии:

Доступен, начиная с версии 8.3.13.

Описание изменено в версии 8.3.16.
