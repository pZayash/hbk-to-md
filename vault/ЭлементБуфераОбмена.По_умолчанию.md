ЭлементБуфераОбмена (ClipboardItem)

По умолчанию

Доступен, начиная с версии 8.3.24.

Синтаксис:

Новый ЭлементБуфераОбмена(<ФорматДанных>, <Данные>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена_(ClipboardDataStandardFormat).md), [Строка](lang__def_String.md).   
Вариант стандартного формата данных буфера обмена или строка, устанавливающая произвольный формат данных.  
  
В веб-клиенте значение параметра типа [Строка](lang__def_String.md) должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

<Данные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Картинка](Картинка_(Picture).md).   
Данные типа [Строка](lang__def_String.md) могут быть использованы только, если указан формат данных типа [Текст](СтандартныйФорматДанныхБуфераОбмена.Текст_(ClipboardDataStandardFormat.Text).md) или [HTML](СтандартныйФорматДанныхБуфераОбмена.HTML_(ClipboardDataStandardFormat.HTML).md).  
  
Данные типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) могут быть использованы только, если указан формат данных типа [Строка](lang__def_String.md).  
  
Данные типа [Картинка](Картинка_(Picture).md) могут быть использованы только, если указан формат данных типа [Картинка](СтандартныйФорматДанныхБуфераОбмена.Картинка_(ClipboardDataStandardFormat.Picture).md).

Использование в версии:

Доступен, начиная с версии 8.3.24.
