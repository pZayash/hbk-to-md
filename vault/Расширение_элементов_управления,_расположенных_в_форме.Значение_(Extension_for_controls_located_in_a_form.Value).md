Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Предоставляет доступ к значению, которое редактируется в элементе управления. Тип значения зависит от типа элемента управления, предоставившего значение.

Доступность:

Толстый клиент.

Примечание:

Только для элементов управления, поддерживающих связь с данными.  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
