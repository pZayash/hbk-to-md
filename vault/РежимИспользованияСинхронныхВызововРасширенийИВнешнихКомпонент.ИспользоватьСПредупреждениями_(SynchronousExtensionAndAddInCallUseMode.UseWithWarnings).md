РежимИспользованияСинхронныхВызововРасширенийИВнешнихКомпонент (SynchronousExtensionAndAddInCallUseMode)

ИспользоватьСПредупреждениями (UseWithWarnings)

Доступен, начиная с версии 8.3.6.

Описание:

Тип: [РежимИспользованияСинхронныхВызововРасширенийИВнешнихКомпонент](РежимИспользованияСинхронныхВызововРасширенийИВнешнихКомпонент_(SynchronousExtensionAndAddInCallUseMode).md).   
В режиме "Конфигуратор":

- В процессе синтаксического контроля синхронные методы не вызывают ошибки.
- При расширенной проверке конфигурации при обнаружении синхронных методов на клиенте отображается предупреждение.

.  
В режиме "1С:Предприятие" при вызове синхронных методов будет отображаться предупреждение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
