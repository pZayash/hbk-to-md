ДинамическийСписок (DynamicList)

ВидКлюча (KeyType)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [ВидКлючаДинамическогоСписка](ВидКлючаДинамическогоСписка_(DynamicListKeyType).md).   
Вид ключа, используемого динамическим списком. Если основная таблица указана, допускается использовать только значение [Авто](ВидКлючаДинамическогоСписка.Авто_(DynamicListKeyType.Auto).md).

Доступность:

Сервер.
