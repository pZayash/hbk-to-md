Описание:

С помощью данного объекта производится настройка положения, видимости, порядка обхода, порядка отображения и привязок границ элемента управления, размещенного на панели.  
  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, некоторые их свойства становятся недоступны.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| // Пример процедуры для отработки нажатия кнопки "Свертка"  // При нажатии кнопки панель "ПанельОтбор" сворачивается или разворачивается  // В данном примере схлопывание осуществляется по вертикали  Процедура КоманднаяПанельОтбор(Кнопка)     Если НЕ ЭлементыФормы.ПанельОтбор.Свертка = РежимСверткиЭлементаУправления.Верх Тогда         ЭлементыФормы.ПанельОтбор.Свертка = РежимСверткиЭлементаУправления.Верх;     Иначе         ЭлементыФормы.ПанельОтбор.Свертка = РежимСверткиЭлементаУправления.Нет;     КонецЕсли; КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
