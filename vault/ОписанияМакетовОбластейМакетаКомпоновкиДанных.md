Элементы коллекции:

[ОписаниеМакетаОбластиМакетаКомпоновкиДанных](ОписаниеМакетаОбластиМакетаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания макетов.  
Возможно обращение к описанию макета посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
