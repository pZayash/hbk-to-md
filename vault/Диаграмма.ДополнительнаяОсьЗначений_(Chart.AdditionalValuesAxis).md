Диаграмма (Chart)

ДополнительнаяОсьЗначений (AdditionalValuesAxis)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).

Использование:

Только чтение.

Описание:

Тип: [ОсьДиаграммы](ОсьДиаграммы_(ChartAxis).md).   
Содержит настройки дополнительной оси значений в диаграмме.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).
