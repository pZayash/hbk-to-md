# КоллекцияЭлементовПорядкаКомпоновкиДанных

Доступен, начиная с версии 8.1.

Элементы коллекции:

[АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных.md), [ЭлементПорядкаКомпоновкиДанных](ЭлементПорядкаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы порядка.  
Возможно обращение к элементу порядка посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция элементов порядка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
