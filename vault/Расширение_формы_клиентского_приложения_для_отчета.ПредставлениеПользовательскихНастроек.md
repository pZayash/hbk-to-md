Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПредставлениеПользовательскихНастроек (UserSettingsPresentation)

Доступен, начиная с версии 8.2.

Описание:

Тип: Строка.   
Представление пользовательских настроек.   
Если не указан и при этом указан параметр формы [КлючПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.КлючПользовательскихНастроек.md), то будет получено представление пользовательских настроек для переданного ключа пользовательских настроек.

Использование в версии:

Доступен, начиная с версии 8.2.
