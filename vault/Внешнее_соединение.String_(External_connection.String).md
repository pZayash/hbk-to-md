Внешнее соединение (External connection)

String (String)

Доступен, начиная с версии 8.1.

Синтаксис:

String(<Параметр>)

Параметры:

<Параметр> (обязательный)

Тип: Произвольный.   
Выражение произвольного типа.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает строковое представление значения произвольного типа.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
