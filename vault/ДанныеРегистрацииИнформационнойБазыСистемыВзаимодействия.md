# ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Система взаимодействия](objects__catalog63__catalog2454.md)

Доступен, начиная с версии 8.3.10.

Описание:

Представляет данные регистрации информационной базы на сервере системы взаимодействия:

- уникальный идентификатор информационной базы на сервере,
- криптографические ключи для обеспечения безопасной передачи сообщений между клиентской и серверной частями системы взаимодействия,
- адрес сервера взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.3/ecs/core. Имя типа XDTO: RegistrationData.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
