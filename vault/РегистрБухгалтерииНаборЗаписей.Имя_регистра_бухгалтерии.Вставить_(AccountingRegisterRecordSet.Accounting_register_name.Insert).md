РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс позиции, на которую надо вставить новую запись.

Возвращаемое значение:

Тип: [РегистрНакопленияЗапись.\<Имя регистра накопления\>](РегистрНакопленияЗапись.Имя_регистра_накопления_(AccumulationRegisterRecord.Accumulation_register_name).md).   
Созданная запись регистра.

Описание:

Вставляет новую запись в набор на указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
