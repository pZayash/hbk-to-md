Описание:

Позволяет выполнять административные функции над рабочим процессом кластера. Представляет собой объект с интерфейсом IV8ServerConnection.

Доступность:

Интеграция.

См. также:

[Менеджер COM-соединений](Менеджер_COM-соединений_(COM_connector).md), метод [ConnectWorkingProcess](Менеджер_COM-соединений.ConnectWorkingProcess_(COM_connector.ConnectWorkingProcess).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.27.
