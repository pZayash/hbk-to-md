ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Метод вызывает проверку заполнения полей записей. Для полей, у которых свойство [ПроверкаЗаполнения](ОбъектМетаданных_Поле.ПроверкаЗаполнения_(MetadataObject_Field.FillChecking).md) установлено в значение [ВыдаватьОшибку](ПроверкаЗаполнения.ВыдаватьОшибку_(FillChecking.ShowError).md) и они при этом не заполнены, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
