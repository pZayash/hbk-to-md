ПользовательИнформационнойБазы (InfoBaseUser)

ПользовательОС (OSUser)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, идентифицирующую пользователя операционной системы при установленном свойстве [АутентификацияОС](ПользовательИнформационнойБазы.АутентификацияОС_(InfoBaseUser.OSAuthentication).md).  
Формат строки: \\ИмяДомена\ИмяПользователя.   
При установке наличие пользователя в операционной системе не проверяется.  
Доступно только пользователю с административными правами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
