Глобальный контекст (Global context)

Cos (Cos)

Доступен, начиная с версии 8.0.

Синтаксис:

Cos(<Угол>)

Параметры:

<Угол> (обязательный)

Тип: Число.   
Аргумент функции.

Возвращаемое значение:

Тип: Число.   
Результат вычисления косинуса.

Описание:

Вычисляет косинус от аргумента <Угол>, заданного в радианах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Pi = 3.1415926535897932; УголРадианы = 0; УголГрадусы = 0; Для Н=0 По 12 Цикл     Сообщить("Cos("+УголГрадусы+")="+Формат(Cos(УголРадианы),"ЧДЦ=6"));     УголРадианы = УголРадианы + Pi/6;     УголГрадусы = УголГрадусы + 30; КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
