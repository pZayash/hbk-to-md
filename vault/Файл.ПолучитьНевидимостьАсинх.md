Файл (File)

ПолучитьНевидимостьАсинх (GetHiddenAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьНевидимостьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Истина, если файл скрытый, в противном случае - Ложь. Применима только к файлам или каталогам, которые реально существуют.

Описание:

Асинхронная функция. Получает значение атрибута файла "Скрытый" ("Hidden").

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.
