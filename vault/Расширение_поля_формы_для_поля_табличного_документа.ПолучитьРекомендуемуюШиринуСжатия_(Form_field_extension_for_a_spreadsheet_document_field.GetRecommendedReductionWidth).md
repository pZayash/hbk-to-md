Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

ПолучитьРекомендуемуюШиринуСжатия (GetRecommendedReductionWidth)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьРекомендуемуюШиринуСжатия()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает ширину поля табличного документа в средних символах табличного документа.  
Результат данного метода рекомендуется использовать в качестве параметра метода табличного документа [ПреобразоватьВКомпактныйДокумент](ТабличныйДокумент.ПреобразоватьВКомпактныйДокумент_(SpreadsheetDocument.ConvertToCompactMode).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [ПреобразоватьВКомпактныйДокумент](ТабличныйДокумент.ПреобразоватьВКомпактныйДокумент_(SpreadsheetDocument.ConvertToCompactMode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.14.
