# ПостроительОтчетаАнализаДанных.Вывести

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md) › [ПостроительОтчетаАнализаДанных](ПостроительОтчетаАнализаДанных_(DataAnalysisReportBuilder).md) › [Методы](ПостроительОтчетаАнализаДанных__Методы.md)

ПостроительОтчетаАнализаДанных (DataAnalysisReportBuilder)

Вывести (Put)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Вывод свойства результат

Синтаксис:

Вывести(<Документ>)

Параметры:

<Документ> (обязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Табличный документ, в который будет осуществлен вывод.

Описание варианта метода:

Выводит в табличный документ содержимое свойства [Результат](ПостроительОтчетаАнализаДанных.Результат_(DataAnalysisReportBuilder.Result).md).

Вариант синтаксиса: Вывод произвольного результата

Синтаксис:

Вывести(<Результат>, <Документ>)

Параметры:

<Результат> (обязательный)

Тип: [РезультатАнализаДанныхДеревоРешений](РезультатАнализаДанныхДеревоРешений_(DataAnalysisDecisionTreeResult).md), [РезультатАнализаДанныхКластеризация](РезультатАнализаДанныхКластеризация_(DataAnalysisClusterizationResult).md), [РезультатАнализаДанныхПоискАссоциаций](РезультатАнализаДанныхПоискАссоциаций_(DataAnalysisAssociationRulesResult).md), [РезультатАнализаДанныхПоискПоследовательностей](РезультатАнализаДанныхПоискПоследовательностей_(DataAnalysisSequentialPatternsResult).md), [РезультатАнализаДанныхОбщаяСтатистика](РезультатАнализаДанныхОбщаяСтатистика_(DataAnalysisSummaryStatisticsResult).md).   
Выводимый результат.

<Документ> (обязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Табличный документ, в который будет осуществлен вывод.

Описание варианта метода:

Осуществляет вывод произвольного результата.

Описание:

Выводит результат анализа в табличный документ.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
