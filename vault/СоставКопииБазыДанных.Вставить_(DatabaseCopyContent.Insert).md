Вставляемый элемент.  
В режиме совместимости с [Версия8\_3\_15](РежимСовместимости.Версия8_3_15_(CompatibilityMode.Version8_3_15).md) и ниже параметр является необязательным.  
В этом случае объект метаданных можно установить позже через свойство [Метаданные](ЭлементСоставаКопииБазыДанных.Метаданные_(DatabaseCopyContentItem.Metadata).md), доступное на запись.

Возвращаемое значение:

Тип: [ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных_(DatabaseCopyContentItem).md).   

Описание:

Добавляет новый элемент коллекции по указанному индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
