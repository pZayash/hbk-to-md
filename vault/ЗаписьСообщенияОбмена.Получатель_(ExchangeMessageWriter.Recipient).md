ЗаписьСообщенияОбмена (ExchangeMessageWriter)

Получатель (Recipient)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [Неопределено](lang__def_Undefined.md).   
Ссылка на узел, которому предназначено сообщение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

До обращения к методу [НачатьЗапись](ЗаписьСообщенияОбмена.НачатьЗапись_(ExchangeMessageWriter.BeginWrite).md) и после обращения к методу [ЗакончитьЗапись](ЗаписьСообщенияОбмена.ЗакончитьЗапись_(ExchangeMessageWriter.EndWrite).md) или [ПрерватьЗапись](ЗаписьСообщенияОбмена.ПрерватьЗапись_(ExchangeMessageWriter.CancelWrite).md) значение данного свойства равно [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.0.
