СоставФункциональнойОпции (FunctionalOptionContent)

Получить (Get)

Доступен, начиная с версии 8.2.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [ЭлементСоставаФункциональнойОпции](ЭлементСоставаФункциональнойОпции_(FunctionalOptionContentItem).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
