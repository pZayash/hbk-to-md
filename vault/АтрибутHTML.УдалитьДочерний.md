Удаляемый узел.

Описание:

Удаляет узел из списка дочерних узлов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Исключение вызывается в случаях:

- если данный узел в режиме Только для чтения;
- если удаляемый узел не найден;
- если удаляемый узел является дочерним узлом узла [Документ](ТипУзлаDOM.Документ.md);
- если выполняется попытка удалить узел [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md).

Доступен, начиная с версии 8.2.
