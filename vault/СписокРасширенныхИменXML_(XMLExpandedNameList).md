# СписокРасширенныхИменXML

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[РасширенноеИмяXML](РасширенноеИмяXML_(XMLExpandedName).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются расширенные имена XML.  
Возможно обращение к расширенному имени XML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
