ЭлементыШкалыВремени (TimeScaleItems)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементШкалыВремени](ЭлементШкалыВремени.md).   
Сдвигаемый элемент.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов, на которое необходимо переместить элемент. Положительное значение означает, что элемент будет передвинут ближе к концу шкалы (вниз), отрицательное - ближе к началу (вверх).

Описание:

Сдвигает указанный элемент шкалы на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
