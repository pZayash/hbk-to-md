ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ДокументМенеджер.Имя_документа.ПолучитьСсылку.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ДокументМенеджер.\<Имя документа\>](ДокументМенеджер.Имя_документа.md), метод [ПолучитьСсылку](ДокументМенеджер.Имя_документа.ПолучитьСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
