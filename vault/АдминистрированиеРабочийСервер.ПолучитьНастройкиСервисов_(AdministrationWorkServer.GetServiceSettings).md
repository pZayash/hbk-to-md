АдминистрированиеРабочийСервер (AdministrationWorkServer)

ПолучитьНастройкиСервисов (GetServiceSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьНастройкиСервисов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [НастройкаСервиса](НастройкаСервиса_(ServiceSetting).md).

Описание:

Возвращает массив настроек сервисов кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

См. также:

[НастройкаСервиса](НастройкаСервиса_(ServiceSetting).md)  

Использование в версии:

Доступен, начиная с версии 8.3.26.
