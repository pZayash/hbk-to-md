ФиксированныйМассив (FixedArray)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Получает значение по индексу. Тип возвращаемого значения определяется типом хранящихся в коллекции значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
