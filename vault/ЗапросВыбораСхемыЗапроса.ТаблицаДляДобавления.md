ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

ТаблицаДляДобавления (TableToAdd)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя временной таблицы, в которую будут добавлены строки, полученные в результате выполнения запроса.  
Соответствует ключевому слову ДОБАВИТЬ языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
