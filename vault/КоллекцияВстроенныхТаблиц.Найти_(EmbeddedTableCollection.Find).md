# КоллекцияВстроенныхТаблиц.Найти

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличный документ](Табличный_документ-2.md) › [КоллекцияВстроенныхТаблиц](КоллекцияВстроенныхТаблиц_(EmbeddedTableCollection).md) › [Методы](КоллекцияВстроенныхТаблиц__Методы.md)

КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяТаблицы>)

Параметры:

<ИмяТаблицы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя встроенной таблицы, как оно задано в табличном документе.

Возвращаемое значение:

Тип: [СводнаяТаблица](СводнаяТаблица_(PivotTable).md), [Неопределено](lang__def_Undefined.md).   
Если не найдена, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск сводной таблицы по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
