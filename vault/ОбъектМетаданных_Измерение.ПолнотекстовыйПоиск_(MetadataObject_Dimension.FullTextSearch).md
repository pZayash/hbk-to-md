ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ПолнотекстовыйПоиск (FullTextSearch)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеПолнотекстовогоПоиска](ИспользованиеПолнотекстовогоПоиска_(UseFullTextSearch).md).   
Определяет, использовать или нет данные этого объекта метаданных в полнотекстовом поиске.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для измерений регистра бухгалтерии, регистра накопления, регистра расчета, регистра сведений.

Использование в версии:

Доступен, начиная с версии 8.1.
