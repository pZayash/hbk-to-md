Описание:

Описывает структуру, передаваемую в обработчик команды.  
Содержит следующие свойства:  
Источник – окно ([ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md)) или форма ([Форма](Форма_(Form).md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md)), из которого(-ой) вызвана команда.  
Окно – тип [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md). Окно, в котором требуется открыть форму.  
Уникальность - уникальность формы. [Ложь](lang__def_BooleanFalse.md) - искать с ключом уникальности формы по умолчанию, [Истина](lang__def_BooleanTrue.md) - не искать форму вообще, любое другое значение устанавливает ключ уникальности, по которому будет произведен поиск.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
