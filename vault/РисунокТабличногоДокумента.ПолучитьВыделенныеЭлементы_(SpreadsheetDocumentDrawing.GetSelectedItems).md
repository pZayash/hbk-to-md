РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ПолучитьВыделенныеЭлементы (GetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьВыделенныеЭлементы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Элементами массива являются значения следующих типов: [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md), [ЗначениеДиаграммы](ЗначениеДиаграммы_(ChartValue).md).

Описание:

Возвращает массив выделенных элементов для рисунка типа [Диаграмма](ТипРисункаТабличногоДокумента.Диаграмма_(SpreadsheetDocumentDrawingType.Chart).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
