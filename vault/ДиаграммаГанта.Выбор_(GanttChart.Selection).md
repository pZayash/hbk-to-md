ДиаграммаГанта (GanttChart)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<Значения>, <СтандартнаяОбработка>, <Дата>)

Параметры:

<Значения>

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md), [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md), [Массив](Массив_(Array).md), [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта_(GanttChartValue).md), [МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени_(TimeScaleItemLabel).md).   
Если тип параметра [Массив](Массив_(Array).md), то первый элемент - это [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта_(GanttChartValue).md), второй и последующие [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md) (в случае перекрывающихся интервалов).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Дата>

Тип: [Дата](lang__def_Date.md).   
Дата, соответствующая точке выбора. Применима для значения и интервала диаграммы Ганта. При выборе других объектов принимает значение [Неопределено](lang__def_Undefined.md).

Описание:

Возникает при двойном щелчке мыши на диаграмме Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если диаграмма Ганта расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.

Использование в версии:

Доступен, начиная с версии 8.0.
