# ТаблицаФормы.ДанныеСтроки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md) › [ТаблицаФормы](ТаблицаФормы_(FormTable).md) › [Методы](ТаблицаФормы__Методы.md)

ТаблицаФормы (FormTable)

ДанныеСтроки (RowData)

Доступен, начиная с версии 8.2.

Синтаксис:

ДанныеСтроки(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Произвольный.   
Строка таблицы, данные которой необходимо получить. Тип параметра зависит от типа редактируемого значения.

Возвращаемое значение:

Тип: [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md), [ДанныеФормыЭлементДерева](ДанныеФормыЭлементДерева_(FormDataTreeItem).md), [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md), [Неопределено](lang__def_Undefined.md).   
Для таблиц, связанных с разными типами данных, тип возращаемого значения данного метода может различаться:

- Для динамического списка тип свойства - [ДанныеФормыСтруктура](ДанныеФормыСтруктура_(FormDataStructure).md);
- Для дерева значений - [ДанныеФормыЭлементДерева](ДанныеФормыЭлементДерева_(FormDataTreeItem).md);
- Для остальных (таблица значений, табличные части и др.) - [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md).

Если таблица не отображается на форме, если в таблице нет строк или передана строка, отсутствующая в таблице, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает значение, содержащее данные указанной строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.17.
