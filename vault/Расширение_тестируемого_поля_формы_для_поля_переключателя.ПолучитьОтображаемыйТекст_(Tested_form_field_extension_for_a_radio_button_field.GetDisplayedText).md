Расширение тестируемого поля формы для поля переключателя (Tested form field extension for a radio button field)

ПолучитьОтображаемыйТекст (GetDisplayedText)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьОтображаемыйТекст()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает отображаемое представление значения переключателя в поле ввода.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
