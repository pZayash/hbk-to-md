Доступен, начиная с версии 8.1.

Элементы коллекции:

[ОформляемоеПолеКомпоновкиДанных](ОформляемоеПолеКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются оформляемые поля.  
Возможно обращение к оформляемому полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
