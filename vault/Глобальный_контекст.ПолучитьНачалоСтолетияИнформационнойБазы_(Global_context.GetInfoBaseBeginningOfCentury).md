Глобальный контекст (Global context)

ПолучитьНачалоСтолетияИнформационнойБазы (GetInfoBaseBeginningOfCentury)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьНачалоСтолетияИнформационнойБазы()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает год начала столетия, установленный для информационной базы или [Неопределено](lang__def_Undefined.md), если начало столетия не установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
