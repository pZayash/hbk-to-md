ОписаниеХарактеристик (CharacteristicsDescription)

ЗначениеОтбораВидов (TypesFilterValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Содержит значение отбора. Значение может быть примитивного типа или ссылкой на перечисление или предопределенный элемент.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
