ДоступныеПоляСхемыЗапроса (QuerySchemaAvailableFields)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Имя>, <Тип>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя добавляемого поля.

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Содержит описание типа добавляемого элемента.  
Значение по умолчанию: Неопределено.

Описание:

Добавляет новый элемент коллекции. Метод доступен только для доступных полей описания временной таблицы ([ДоступныеПоля](ОписаниеВременнойТаблицыСхемыЗапроса.ДоступныеПоля_(QuerySchemaTempTableDescription.AvailableFields).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
