ИнтернетПочтовыйПрофиль (InternetMailProfile)

ТолькоЗащищеннаяАутентификацияPOP3 (POP3SecureAuthenticationOnly)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает на использование для протокола POP3 только защищенных способов аутентификации из поддерживаемых сервером (на данный момент CRAM-MD5).  
Ложь - разрешить использовать незащищенные способы аутентификации, если сервер поддерживает только их.  
Значение по умолчанию - Ложь.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Ложь - соответствует устаревшему значению свойства АутентификацияPOP3.Обычная.  
Истина - соответствует устаревшим значениям свойств АутентификацияPOP3 APOP, CramMD5.

Использование в версии:

Доступен, начиная с версии 8.3.1.
