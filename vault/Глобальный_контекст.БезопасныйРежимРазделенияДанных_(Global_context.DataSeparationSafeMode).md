Глобальный контекст (Global context)

БезопасныйРежимРазделенияДанных (DataSeparationSafeMode)

Доступен, начиная с версии 8.3.1.

Синтаксис:

БезопасныйРежимРазделенияДанных(<ОбщийРеквизит>)

Параметры:

<ОбщийРеквизит> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя общего реквизита со значением "Использовать" свойства "Разделение данных", для которого должен быть получен безопасный режим разделения данных.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - безопасный режим разделения данных по данному разделителю включен.

Описание:

Показывает, установлен ли безопасный режим разделения данных по данному разделителю. Учитывает установку режима безопасного сеанса, определяемого установкой атрибута safe=”true” элемента zone файла default.vrd.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьБезопасныйРежимРазделенияДанных](Глобальный_контекст.УстановитьБезопасныйРежимРазделенияДанных_(Global_context.SetDataSeparationSafeMode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.
