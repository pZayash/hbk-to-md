СлойГеографическойСхемы (GeographicalSchemaLayer)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначение(<Объект>, <Серия>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md).   
Объект, для которого ищется значение.

<Серия> (обязательный)

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы.md).   
Серия, для которой ищется значение.

Возвращаемое значение:

Тип: [ЗначениеСерииСлояГеографическойСхемы](ЗначениеСерииСлояГеографическойСхемы.md).   

Описание:

Получает значение, соответствующее заданному объекту и серии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
