РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ПолучитьГлубинуГрафа (GetGraphDepth)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьГлубинуГрафа(<НомерКорневойВершины>)

Параметры:

<НомерКорневойВершины> (обязательный)

Тип: Число.   
Номер неизвестной СЛАУ, которая должна существовать в рамках данной СЛАУ.  
Неотрицательное целое число.   
С вершины графа с указанным номером должен начинаться поиск в глубину графа.

Возвращаемое значение:

Тип: Число.   
Число, равное максимальной глубине графа, полученной в результате поиска в глубину, начатого с вершины с номером <НомерКорневойВершины>.

Описание:

Рассчитывает глубину графа СЛАУ.  
Для корректной работы требуется установка значений свойств [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей.md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях.md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях.md), а также чтобы в коллекции [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений.md) был добавлен один объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md), в котором должна быть указана только [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
