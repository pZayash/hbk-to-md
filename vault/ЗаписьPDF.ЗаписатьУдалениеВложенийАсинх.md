ЗаписьPDF (PDFWriter)

ЗаписатьУдалениеВложенийАсинх (WriteAttachmentsDeletionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьУдалениеВложенийАсинх(<Вложения>)

Параметры:

<Вложения> (необязательный)

Тип: [Массив](Массив.md).   
Массив, содержащий имена файлов.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Записывает в PDF-документ информацию о удалении вложения.  
Если в метод передан файл версии меньше 1.5, то генерируется исключение.

Доступность:

Тонкий клиент, толстый клиент.
