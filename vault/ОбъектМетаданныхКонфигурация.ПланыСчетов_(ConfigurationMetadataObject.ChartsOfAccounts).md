# ОбъектМетаданныхКонфигурация.ПланыСчетов

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [ОбъектМетаданныхКонфигурация](ОбъектМетаданныхКонфигурация_(ConfigurationMetadataObject).md) › [Свойства](ОбъектМетаданныхКонфигурация__Свойства.md)

ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПланыСчетов (ChartsOfAccounts)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все планы счетов, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов_(MetadataObject_ChartOfAccounts).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
