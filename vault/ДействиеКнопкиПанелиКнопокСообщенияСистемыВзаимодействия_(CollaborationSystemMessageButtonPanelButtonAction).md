Описание:

Содержит варианты действия кнопки панели кнопок сообщения системы взаимодействия.  
Если действие не поддерживается в системе, тогда кнопка, выполняющая данное действие, не отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ИнтеграцияСистемыВзаимодействия](ИнтеграцияСистемыВзаимодействия_(CollaborationSystemIntegration).md), метод [ПоддерживаетсяДействиеКнопкиПанелиКнопокСообщения](ИнтеграцияСистемыВзаимодействия.ПоддерживаетсяДействиеКнопкиПанелиКнопокСообщения_(CollaborationSystemIntegration.MessageButtonPanelButtonActionSupported).md)
