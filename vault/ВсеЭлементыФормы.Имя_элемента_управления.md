ВсеЭлементыФормы (FormAllItems)

<Имя элемента управления> (<Control name>)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Набор свойств содержит элементы формы клиентского приложения, размещенные на форме. Доступ к элементу формы осуществляется по имени.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
