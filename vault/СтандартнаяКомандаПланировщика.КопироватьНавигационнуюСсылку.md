СтандартнаяКомандаПланировщика (PlannerStandardCommand)

КопироватьНавигационнуюСсылку (CopyURL)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [КопироватьНавигационнуюСсылку](СтандартнаяКомандаПоляПланировщика.КопироватьНавигационнуюСсылку.md)

Доступен, начиная с версии 8.3.18.

Описание:

Копировать ссылку в буфер обмена.  
Используется только для источника вида [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка.md).  
В контекстном меню команда будет иметь представление "Копировать ссылку".

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
