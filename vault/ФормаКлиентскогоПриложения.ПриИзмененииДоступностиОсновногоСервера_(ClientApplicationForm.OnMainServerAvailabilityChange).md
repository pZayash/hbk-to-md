ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриИзмененииДоступностиОсновногоСервера (OnMainServerAvailabilityChange)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПриИзмененииДоступностиОсновногоСервера(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если форма была принудильно открыта с автономного сервера при появлении связи с основным сервером, будет предложено переоткрыть форму с основного сервера.

Описание:

Возникает для формы при недоступности основного сервера.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
