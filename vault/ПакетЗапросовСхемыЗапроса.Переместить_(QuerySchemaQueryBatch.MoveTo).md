ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Переместить (MoveTo)

Доступен, начиная с версии 8.3.11.

Синтаксис:

Переместить(<Элемент>, <НовыйИндекс>)

Параметры:

<Элемент> (обязательный)

Тип: [Число](lang__def_Number.md), [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).   
Индекс элемента или сам элемент, который требуется переместить.

<НовыйИндекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс, в который нужно переместить указанный элемент.

Описание:

Перемещает элемент в коллекции в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
