Расширение поля формы для поля ввода (Form field extension for a text box)

Формат (Format)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит форматную строку. Используется при отображении текста в колонке таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ПолеВвода](ПолеВвода_(TextBox).md), свойство [Формат](ПолеВвода.Формат_(TextBox.Format).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
