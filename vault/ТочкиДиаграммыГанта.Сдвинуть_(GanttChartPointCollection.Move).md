ТочкиДиаграммыГанта (GanttChartPointCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Точка>, <Смещение>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество точек, на которое необходимо переместить указанную. Положительное значение означает, что точка будет передвинута ближе к концу диаграммы (вниз), отрицательное - ближе к началу (вверх).

Описание:

Сдвигает указанную точку на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
