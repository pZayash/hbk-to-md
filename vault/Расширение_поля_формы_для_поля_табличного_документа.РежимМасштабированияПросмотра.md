Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

РежимМасштабированияПросмотра (ViewScalingMode)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [РежимМасштабированияПросмотра](РежимМасштабированияПросмотра.md).   
Режим масштабирования, используемый при просмотре табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

По умолчанию - [Авто](РежимМасштабированияПросмотра.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.4.
