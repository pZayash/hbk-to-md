Диапазон (Range)

Конец (End)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md), [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Верхняя граница диапазона. Значение верхней границы диапазона должно быть не меньше значения нижней границы. [Неопределено](lang__def_Undefined.md) - диапазон не ограничен сверху.  
Тип значения должен совпадать с типом значения нижней границы.  
Если свойство [Начало](Диапазон.Начало_(Range.Begin).md) имеет тип значения, отличный от [Неопределено](lang__def_Undefined.md), то данное свойство должно иметь такой же тип значения или [Неопределено](lang__def_Undefined.md).  
Если свойство [Начало](Диапазон.Начало_(Range.Begin).md) имеет тип значения [Неопределено](lang__def_Undefined.md), то данное свойство не может иметь такой же тип значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
