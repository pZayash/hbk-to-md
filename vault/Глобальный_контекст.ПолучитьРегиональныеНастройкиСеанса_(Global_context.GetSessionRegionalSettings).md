ПолучитьРегиональныеНастройкиСеанса (GetSessionRegionalSettings)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьРегиональныеНастройкиСеанса()

Возвращаемое значение:

Тип: [РегиональныеНастройкиСеанса](РегиональныеНастройкиСеанса_(SessionRegionalSettings).md).

Сервер, толстый клиент, внешнее соединение.
