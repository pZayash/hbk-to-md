# КомментарийHTML.ВыбратьДанные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [HTML](HTML.md) › [КомментарийHTML](КомментарийHTML_(HTMLComment).md) › [Методы](КомментарийHTML__Методы.md)

КомментарийHTML (HTMLComment)

ВыбратьДанные (SubstringData)

Доступен, начиная с версии 8.2.

Синтаксис:

ВыбратьДанные(<Смещение>, <Длина>)

Параметры:

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение от начала символьных данных.

<Длина> (обязательный)

Тип: [Число](lang__def_Number.md).   
Длина выделяемой подстроки.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Выделяет подстроки из символьных данных.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных.

Использование в версии:

Доступен, начиная с версии 8.2.
