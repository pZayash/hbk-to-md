Элементы коллекции:

[МакетГруппировкиТаблицыМакетаКомпоновкиДанных](МакетГруппировкиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroupTemplate).md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroup).md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableRecords).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы тела группировки таблицы.  
Возможно обращение к элементу тела группировки таблицы посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[МакетГруппировкиТаблицыМакетаКомпоновкиДанных](МакетГруппировкиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroupTemplate).md);
