ГеографическаяСхема (GeographicalSchema)

ПроверитьПоРасположению (CheckByLocation)

Доступен, начиная с версии 8.0.

Синтаксис:

ПроверитьПоРасположению(<Объект>, <ГдеПроверять>, <КакПроверять>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   
Объект поиска.

<ГдеПроверять> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив может содержать объекты следующих типов: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md), [ГеографическиеКоординаты](ГеографическиеКоординаты_(GeographicCoordinates).md),   
[ПрямоугольникГеографическойСхемы](ПрямоугольникГеографическойСхемы_(GeographicalSchemaRectangle).md).

<КакПроверять> (обязательный)

Тип: [ТипПоискаОбъектовГеографическойСхемы](ТипПоискаОбъектовГеографическойСхемы_(GeographicalSchemaObjectFindType).md).   
Определяет способ поиска объекта географической схемы.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) в противном случае.

Описание:

Проверяет, попадают ли указанные объекты в местоположение других.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
