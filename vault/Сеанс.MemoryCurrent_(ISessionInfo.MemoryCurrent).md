Сеанс (ISessionInfo)

MemoryCurrent (MemoryCurrent)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит объем памяти в байтах, занятый с начала выполнения текущего вызова. Если в данный момент вызов не выполняется, содержит 0.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
