ДанныеАдреса (AddressData)

СтрокаАдресаДополнительная (AddressLine2)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит текстовое представление номера дома.  
Свойство не используется в свойствах объекта [ДанныеКонтакта](ДанныеКонтакта.md).  
В методе [ПолучитьМестоположениеПоАдресу](Глобальный_контекст.ПолучитьМестоположениеПоАдресу.md) имеет приоритет перед свойством [СтрокаАдреса](ДанныеАдреса.СтрокаАдреса.md).

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
