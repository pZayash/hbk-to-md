КолонкаТабличногоПоля (TableBoxColumn)

ОтображатьИерархию (ShowHierarchy)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак вывода линий иерархии и пиктограмм раскрытия узлов дерева.  
Истина - отображается.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если табличное поле отображается в виде дерева.

См. также:

[Расширение табличного поля списка справочника](Расширение_табличного_поля_списка_справочника.md), свойство [Дерево](Расширение_табличного_поля_списка_справочника.Дерево.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
