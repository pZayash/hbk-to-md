ИнтернетПочтовоеСообщение (InternetMailMessage)

Частичное (Partial)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, является ли сообщение частичным (в случае, если большое сообщение было разбито на части).  
[Истина](lang__def_BooleanTrue.md) - сообщение частично.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Объединить частичные сообщения можно методом [ОбъединитьЧастичные](ИнтернетПочта.ОбъединитьЧастичные_(InternetMail.CombinePartial).md). Тело частичного сообщения сохраняется во вложении.

См. также:

[ИнтернетПочта](ИнтернетПочта_(InternetMail).md), метод [ОбъединитьЧастичные](ИнтернетПочта.ОбъединитьЧастичные_(InternetMail.CombinePartial).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
