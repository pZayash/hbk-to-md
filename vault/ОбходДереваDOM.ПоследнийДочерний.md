ОбходДереваDOM (DOMTreeWalker)

ПоследнийДочерний (LastChild)

Доступен, начиная с версии 8.1.

Синтаксис:

ПоследнийДочерний()

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md), Неопределено.   
Если видимые потомки отсутствуют, то итератор не перемещается и возвращается Неопределено.

Описание:

Выполняет переход к последнему потомку. Итератор перемещается к последнему потомку текущего узла, удовлетворяющему условиям отбора узлов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
