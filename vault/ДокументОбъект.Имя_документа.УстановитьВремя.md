ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

УстановитьВремя (SetTime)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьВремя(<РежимОпределенияВремени>, <ИспользоватьЖурналы>)

Параметры:

<РежимОпределенияВремени> (необязательный)

Тип: [РежимАвтоВремя](РежимАвтоВремя.md).   
Устанавливает режим автоматического определения времени при записи нового документа.  
Значение по умолчанию: [ТекущееИлиПоследним](РежимАвтоВремя.ТекущееИлиПоследним.md).

<ИспользоватьЖурналы> (необязательный)

Тип: Булево.   
Определяет, устанавливать ли время с использованием журналов, в которых отражается данный документ.  
Значение по умолчанию: Ложь.

Описание:

Устанавливает время документа.  
На время выполнения данного метода устанавливается привилегированный режим.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Документ = Документы.БюджетнаяОперация.СоздатьДокумент(); Документ.Дата = ДатаДокумента; Документ.УстановитьНовыйНомер(); Документ.УстановитьВремя(РежимАвтоВремя.ТекущееИлиПоследним); |

Использование в версии:

Доступен, начиная с версии 8.0.
