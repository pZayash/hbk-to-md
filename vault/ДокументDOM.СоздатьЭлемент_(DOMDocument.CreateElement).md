ДокументDOM (DOMDocument)

СоздатьЭлемент (CreateElement)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

СоздатьЭлемент(<ИмяЭлемента>)

Параметры:

<ИмяЭлемента> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя создаваемого элемента.

Описание варианта метода:

Причины вызова исключений:

- имя элемента содержит символы, недопустимые стандартом XML.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

СоздатьЭлемент(<URIПространстваИмен>, <КвалифицированноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен создаваемого элемента.

<КвалифицированноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Квалифицированное имя элемента XML.

Описание варианта метода:

Таблица заполнения значений свойств элемента:  
[ДокументDOM.ИмяУзла](ДокументDOM.ИмяУзла_(DOMDocument.NodeName).md) - по значению параметра <КвалифицированноеИмя>;  
[ДокументDOM.URIПространстваИмен](ДокументDOM.URIПространстваИмен_(DOMDocument.NamespaceURI).md) - по значению параметра <URIПространстваИмен>;  
[ДокументDOM.Префикс](ДокументDOM.Префикс_(DOMDocument.Prefix).md) - префикс, извлеченный из параметра <КвалифицированноеИмя>, или пустая строка, если префикс отсутствует;  
[ДокументDOM.ЛокальноеИмя](ДокументDOM.ЛокальноеИмя_(DOMDocument.LocalName).md) - локальное имя, извлеченное из параметра <КвалифицированноеИмя>;  
[ИмяЭлемента](ЭлементDOM.ИмяЭлемента_(DOMElement.TagName).md) - по значению параметра <КвалифицированноеИмя>.  
Причины вызова исключений:

- квалифицированное имя содержит символы, недопустимые стандартом XML данного документа;
- квалифицированное имя содержит префикс, значение которого недопустимо для данного значения URI пространства имен.

Возвращаемое значение:

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md).   
Новый узел DOM - [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), в свойство ИмяУзла которого устанавливается значение параметра <ИмяЭлемента>. Свойства ЛокальноеИмя, Префикс и URIПространстваИмен не устанавливаются.

Описание:

Создает новый узел DOM элемента XML.   
Если известны атрибуты, имеющие значения по умолчанию, то они автоматически добавляются в элемент.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
