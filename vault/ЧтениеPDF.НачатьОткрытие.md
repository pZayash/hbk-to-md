ЧтениеPDF (PDFReader)

НачатьОткрытие (BeginOpen)

Доступен, начиная с версии 8.3.16.

Вариант синтаксиса: Открыть файл

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИмяФайла> (необязательный)

Тип: Строка.   
Имя файла для чтения PDF-документа.

Вариант синтаксиса: Открыть поток

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <Поток>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения PDF-документа.

Описание:

Начинает открытие PDF-документа для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [Открыть](ЧтениеPDF.Открыть.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [ОткрытьАсинх](ЧтениеPDF.ОткрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
