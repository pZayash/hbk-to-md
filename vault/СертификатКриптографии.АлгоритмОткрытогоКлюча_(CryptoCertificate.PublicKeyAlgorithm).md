СертификатКриптографии (CryptoCertificate)

АлгоритмОткрытогоКлюча (PublicKeyAlgorithm)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит идентификатор объекта (OID) алгоритма ключа проверки электронной подписи (открытого ключа).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.27.
