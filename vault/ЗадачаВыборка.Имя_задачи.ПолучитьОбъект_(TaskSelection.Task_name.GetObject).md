# ЗадачаВыборка.<Имя задачи>.ПолучитьОбъект

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаВыборка.<Имя задачи>](ЗадачаВыборка.Имя_задачи_(TaskSelection.Task_name).md) › [Методы](ЗадачаВыборка.Имя_задачи__Методы.md)

ЗадачаВыборка.<Имя задачи> (TaskSelection.<Task name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)<[Имя задачи](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для чтения, изменения, добавления и удаления элемента, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
