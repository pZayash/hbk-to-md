Форма (Form)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: Булево.   
Истина - ошибок не обнаружено, Ложь - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Толстый клиент.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](Форма.ОбработкаПроверкиЗаполнения.md).

[Форма](Форма.md), событие [ОбработкаПроверкиЗаполнения](Форма.ОбработкаПроверкиЗаполнения.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
