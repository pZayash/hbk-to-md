ОбъектМетаданных: РегламентноеЗадание (MetadataObject: ScheduledJob)

Использование (Use)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Для регламентных заданий установка этого свойства в значение [Истина](lang__def_BooleanTrue.md) указывает, что регламентное задание будет выполняться автоматически согласно расписанию. В противном случае регламентное задание автоматически выполнено не будет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
