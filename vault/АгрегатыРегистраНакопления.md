Элементы коллекции:

[АгрегатРегистраНакопления](АгрегатРегистраНакопления.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются агрегаты.  
Возможно обращение к агрегату посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Количество (Count)](АгрегатыРегистраНакопления.Количество.md)  
[Получить (Get)](АгрегатыРегистраНакопления.Получить.md)  

Описание:

Представляет собой коллекцию агрегатов регистра накопления.

Доступность:

Сервер, толстый клиент, внешнее соединение.
