СоставХранимыхДанныхХранилищаДвоичныхДанных (BinaryDataStorageDataContent)

Добавить (Add)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Добавить(<Метаданные>, <Хранение>)

Параметры:

<Метаданные> (обязательный)

Тип: [ОбъектМетаданных: Реквизит](ОбъектМетаданных_Реквизит_(MetadataObject_Attribute).md), [ОбъектМетаданных: Ресурс](ОбъектМетаданных_Ресурс_(MetadataObject_Resource).md),[ОбъектМетаданных: Константа](ОбъектМетаданных_Константа_(MetadataObject_Constant).md).   
Содержит элемент типа [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md).

<Хранение> (обязательный)

Тип: [ИспользованиеХраненияВХранилищеДвоичныхДанных](ИспользованиеХраненияВХранилищеДвоичныхДанных_(BinaryDataStorageLocationUse).md).   
Режим записи в хранилище.

Возвращаемое значение:

Тип: [ЭлементСоставаХранимыхДанныхХранилищаДвоичныхДанных](ЭлементСоставаХранимыхДанныхХранилищаДвоичныхДанных_(BinaryDataStorageDataContentItem).md).   

Описание:

Добавляет новый элемент в коллекцию состава.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
