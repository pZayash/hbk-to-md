Глобальный контекст (Global context)

РабочаяДата (WorkingDate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит рабочую дату, используемую в текущем сеансе работы с конфигурацией. Доступно для записи в случае, если свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты_(Global_context.WorkingDateUse).md) имеет значение [Назначать](РежимРабочейДаты.Назначать_(WorkingDateMode.Assign).md).

Доступность:

Толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты_(Global_context.WorkingDateUse).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
