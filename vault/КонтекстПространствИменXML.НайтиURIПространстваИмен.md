`НайтиURIПространстваИмен`(`<Префикс>`: `Строка`) → `Строка, Неопределено`
<!-- signature:end -->

КонтекстПространствИменXML (XMLNamespaceContext)

НайтиURIПространстваИмен (LookupNamespaceURI)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиURIПространстваИмен(<Префикс>)

Параметры:

<Префикс> (обязательный)

Тип: Строка.   
Префикс, для которого необходимо получить URI пространства имен.

Возвращаемое значение:

Тип: Строка, Неопределено.   
Если префиксу не соответствует пространство имен, возвращается Неопределено.

Описание:

Получает URI пространства имен по префиксу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
