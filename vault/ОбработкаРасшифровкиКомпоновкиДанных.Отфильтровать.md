ОбработкаРасшифровкиКомпоновкиДанных (DataCompositionDetailsProcess)

Отфильтровать (Filter)

Доступен, начиная с версии 8.1.

Синтаксис:

Отфильтровать(<ИдентификаторРасшифровки>, <Поле>)

Параметры:

<ИдентификаторРасшифровки> (обязательный)

Тип: [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных.md).   
Идентификатор обрабатываемой расшифровки.

<Поле> (обязательный)

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных.md).   
Поле, по значению которого должен быть отфильтрован результат.

Возвращаемое значение:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md), [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных.md).   
Если в настройках используются пользовательские элементы (см. свойства ИдентификаторПользовательскойНастройки различных объектов) и изменения настроек затрагивают только пользовательские элементы, будут возвращены [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных.md), иначе [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md).

Описание:

Перестраивает настройки, позволяя отфильтровать результат по заданному полю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
