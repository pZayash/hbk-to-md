ЧтениеУзловDOM (DOMNodeReader)

Закрыть (Close)

Доступен, начиная с версии 8.1.

Синтаксис:

Закрыть()

Описание:

Прекращает чтение XML из узла DOM. Если никакого чтения в момент вызова данного метода не выполнялось, то метод ничего не делает.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
