# ОформлениеПоляОбластиКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макеты областей](Макеты_областей.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных_(DataCompositionParameterValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметров.  
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция, содержащая элементы оформления:

- Формат (Format) - тип [Строка](lang__def_String.md),
- ЦветФона (BackColor) - тип [Цвет](Цвет_(Color).md),
- ЦветТекста (TextColor) - тип [Цвет](Цвет_(Color).md),
- Шрифт (Font) - тип [Шрифт](Шрифт_(Font).md),
- ВыделятьОтрицательные (MarkNegatives) - тип [Булево](lang__Булево_(Boolean).md),
- Картинка (Picture) - тип [Картинка](Картинка_(Picture).md),
- ПоложениеТекстаОтносительноКартинки (PictureSizeTextPositionRelativeToPicture) - тип [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки_(TextPositionRelativeToPicture).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: FieldAppearance.
