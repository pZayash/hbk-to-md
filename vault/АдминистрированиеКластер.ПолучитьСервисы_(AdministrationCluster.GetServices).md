АдминистрированиеКластер (AdministrationCluster)

ПолучитьСервисы (GetServices)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСервисы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеСервис](АдминистрированиеСервис_(AdministrationService).md).

Описание:

Получает список сервисов кластера. Список сервисов фиксирован для кластера серверов "1С:Предприятия" определенной версии. Каждый сервис, кроме сервиса сеансовых данных, работает в главном или одном из дополнительных менеджеров кластера. Сервис сеансовых данных может работать на нескольких менеджерах одновременно. Сервис конфигурации кластера, Сервис блокировок кластера и Сервис управления предметами отладки могут работать только в главном менеджере кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
