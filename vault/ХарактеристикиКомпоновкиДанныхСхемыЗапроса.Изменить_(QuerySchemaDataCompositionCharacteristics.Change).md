ХарактеристикиКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionCharacteristics)

Изменить (Change)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Изменить(<Индекс>, <Тип>, <ВидыХарактеристикТаблица>, <ВидыЗарактеристикЗапрос>, <ПолеКлюча>, <ПолеИмени>, <ПолеТипаЗначения>, <ЗначенияХарактеристикТаблица>, <ЗначенияХарактеристикЗапрос>, <ПолеОбъекта>, <ПолеВида>, <ПолеЗначения>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс характеристики в коллекции, которую требуется изменить.  
Индексация начинается с нуля.

<Тип> (обязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Типы, для которых описываются характеристики.

<ВидыХарактеристикТаблица> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы, из которой будут получены виды характеристик (Например, ПланВидовХарактеристик.ВидыХарактеристик).  
Если в качестве этого параметра указана не пустая строка, то параметр <ВидыХарактеристикЗапрос> будет игнорирован.   
Если в качестве этого параметра указана пустая строка, то имя таблицы будет игнорировано и в качестве источника видов характеристик будет использован запрос.  
  
Допустимо в качестве этого параметра указать пустую строку, но в таком случае параметр <ВидыХарактеристикЗапрос> обязательно должен быть заполнен корректным текстом запроса.

<ВидыЗарактеристикЗапрос> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст запроса, из которого будут получены виды характеристик.  
Если в качестве параметра <ВидыХарактеристикТаблица> указана не пустая строка, то этот параметр будет игнорирован.   
Если в качестве параметра <ВидыХарактеристикТаблица> была указана пустая строка, то в качестве источника видов характеристик будет использован запрос.  
  
Допустимо в качестве этого параметра отправить пустую строку в том случае, если параметр <ВидыХарактеристикТаблица> заполнен не пустой строкой.

<ПолеКлюча> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля в таблице или запросе видов характеристик, которое содержит ключ вида характеристики.

<ПолеИмени> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля в таблице или запросе видов характеристик, которое содержит имя характеристики.

<ПолеТипаЗначения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля из таблицы или запроса значений характеристик, которое содержит тип значения характеристики.  
Допускается указание пустой строки.

<ЗначенияХарактеристикТаблица> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы, из которой будут получены значения характеристик (Например, РегистрСведений.Характеристики).  
Если в качестве данного параметра указана не пустая строка, то параметр <ЗначенияХарактеристикЗапрос> будет игнорирован.   
Если в качестве этого параметра указана пустая строка, то имя таблицы будет проигнорировано, и в качестве источника значений характеристик будет использован запрос.  
Если указана пустая строка, параметр <ЗначенияХарактеристикЗапрос> должен содержать корректный текст запроса.

<ЗначенияХарактеристикЗапрос> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст запроса, из которого будут получены значения характеристик.  
Если в качестве параметра <ЗначенияХарактеристикТаблица> указана не пустая строка, то этот параметр будет игнорирован. Если в качестве параметра <ЗначенияХарактеристикТаблица> была указана пустая строка, то в качестве источника значений характеристик установится запрос.  
  
В данном параметре допускается указание пустой строки, если параметр <ЗначенияХарактеристикТаблица> содержит не пустую строку.

<ПолеОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля в таблице или запросе значений характеристик, содержащего объект, для которого указана характеристика.

<ПолеВида> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля из таблицы или запроса значений характеристик, которое содержит вид характеристики.

<ПолеЗначения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля из таблицы или запроса значений характеристик, которое содержит значение характеристики.  
Допускается указание пустой строки.

Возвращаемое значение:

Тип: [ХарактеристикаКомпоновкиДанныхСхемыЗапроса](ХарактеристикаКомпоновкиДанныхСхемыЗапроса_(QuerySchemaDataCompositionCharacteristic).md).   

Описание:

Изменяет описание характеристики в коллекции. Если один из параметров задан некорректно, генерируется исключение, содержащее описание ошибки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
