Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

УстановитьВыделенныеОбласти (SetSelectedAreas)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьВыделенныеОбласти(<МассивОбластей>)

Параметры:

<МассивОбластей> (необязательный)

Тип: [Массив](Массив.md).   
Содержит области, которые нужно установить как выделенные.  
Первый элемент становится текущей областью.

Описание:

Устанавливает области табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
