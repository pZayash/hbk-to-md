ОпределениеПростогоТипаXS (XSSimpleTypeDefinition)

ЭлементDOM (DOMElement)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md).   
Элемент DOM, который соответствует данному компоненту схемы XML.  
Синхронизация значений свойств компонента схемы XML и структуры элемента DOM обеспечивается вызовом метода [ОпределениеПростогоТипаXS.ОбновитьЭлементDOM](ОпределениеПростогоТипаXS.ОбновитьЭлементDOM_(XSSimpleTypeDefinition.UpdateDOMElement).md).  
Перед первым вызовом данного метода значение данного свойства может быть [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
