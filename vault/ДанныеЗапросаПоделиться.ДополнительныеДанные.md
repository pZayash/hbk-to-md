Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства](ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства.md).   
Дополнительные данные, которые будут заполнены в случае, если переход в приложение осуществлен с помощью механизма Intent в ОС Android.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
