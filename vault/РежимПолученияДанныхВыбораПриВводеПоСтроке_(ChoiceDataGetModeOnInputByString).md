Описание:

Содержит варианты запуска поиска при вводе по строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/managed-application/logform}. Имя типа XDTO: ChoiceDataGettingModeOnInputByString.

Использование в версии:

Доступен, начиная с версии 8.3.3.
