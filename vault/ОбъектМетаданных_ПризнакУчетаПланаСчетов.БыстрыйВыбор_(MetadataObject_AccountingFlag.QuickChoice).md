ОбъектМетаданных: ПризнакУчетаПланаСчетов (MetadataObject: AccountingFlag)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеБыстрогоВыбора](ИспользованиеБыстрогоВыбора_(UseQuickChoice).md).   
Свойство устанавливает способ выбора значения в поле ввода формы клиентского приложения, если свойство Быстрый Выбор поля формы клиентского приложения установлено в значение Авто.  
  
Если свойство БыстрыйВыбор признака учета плана счетов установлено в значение Авто, то при выборе значения учитывается свойство Быстрый выбор объекта конфигурации Признак учета плана счетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
