ЭлементыЛегендыГеографическойСхемы (GeographicalSchemaLegendItems)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции, который требуется получить.

Возвращаемое значение:

Тип: [ЭлементЛегендыГеографическойСхемы](ЭлементЛегендыГеографическойСхемы_(GeographicalSchemaLegendItem).md).   

Описание:

Получает элемент по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
