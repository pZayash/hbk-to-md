ОписаниеПодписиPDF (PDFSignatureDescription)

БлокировкаДокумента (DocumentLock)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак необходимости блокировки документа после подписания. Имеет смысл только для подписи типа [Утверждающая](ТипПодписиPDF.Утверждающая.md).   
По умолчанию: Ложь.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
