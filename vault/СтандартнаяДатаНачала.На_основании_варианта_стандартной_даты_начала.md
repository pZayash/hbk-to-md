СтандартнаяДатаНачала (StandardBeginningDate)

На основании варианта стандартной даты начала

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СтандартнаяДатаНачала(<Вариант>)

Параметры:

<Вариант> (обязательный)

Тип: [ВариантСтандартнойДатыНачала](ВариантСтандартнойДатыНачала.md).   
Вариант даты.

Описание:

Создает стандартную дату начала по указанному варианту стандартной даты начала.

Использование в версии:

Доступен, начиная с версии 8.2.
