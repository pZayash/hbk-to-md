ИнформационныеЛинииДиаграммы (ChartReferenceLines)

Получить (Get)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс информационной линии, которую требуется получить.

Возвращаемое значение:

Тип: [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы.md).   

Описание:

Возвращает элемент с указанным индексом.  
Если индекс находится за пределами коллекции, генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
