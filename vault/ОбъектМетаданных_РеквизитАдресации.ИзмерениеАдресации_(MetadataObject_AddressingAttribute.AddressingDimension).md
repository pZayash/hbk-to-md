ОбъектМетаданных: РеквизитАдресации (MetadataObject: AddressingAttribute)

ИзмерениеАдресации (AddressingDimension)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Измерение](ОбъектМетаданных_Измерение_(MetadataObject_Dimension).md).   
Объект метаданных, описывающий измерение регистра сведений. Этот регистр сведений хранит информацию о соответствии исполнителей всем остальным реквизитам адресации задач.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Описывает реквизиты адресации задачи.

Использование в версии:

Доступен, начиная с версии 8.1.
