ПолеВвода (TextBox)

РежимВыбораИзСписка (ListChoiceMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Свойство регулирует работу поля ввода. Если значение свойства - Истина, то поле ввода работает как поле выбора и может хранить одно из значений произвольного типа, имеющихся в списке.

Доступность:

Толстый клиент.

Примечание:

Истина - при выборе показываются представления значений списка, в противном случае - сами значения.  
Внимание! Для правильной работы режима выбора из списка в случае его программной установки, перед установкой этого режима в поле ввода нужно сбрасывать тип редактируемых данных:  
  
 ПолеВвода.ТипЗначения = Новый ОписаниеТипов();  
 ПолеВвода.РежимВыбораИзСписка = Истина;

Использование в версии:

Доступен, начиная с версии 8.0.
