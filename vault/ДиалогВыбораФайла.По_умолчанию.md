ДиалогВыбораФайла (FileDialog)

С заданным режимом работы

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ДиалогВыбораФайла(<РежимРаботы>)

Параметры:

<РежимРаботы> (обязательный)

Тип: [РежимДиалогаВыбораФайла](РежимДиалогаВыбораФайла_(FileDialogMode).md).   
Режим работы конструируемого диалога.

Описание:

Создает диалог выбора файла с заданным режимом работы.

Пример:

|  |
| --- |
| Режим = РежимДиалогаВыбораФайла.Открытие; ДиалогОткрытияФайла = Новый ДиалогВыбораФайла(Режим); |

Использование в версии:

Доступен, начиная с версии 8.0.
