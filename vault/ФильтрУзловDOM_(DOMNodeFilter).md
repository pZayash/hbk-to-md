Описание:

Используется для отбора узлов DOM в итераторе и объекте обхода дерева DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
