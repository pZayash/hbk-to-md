ОбъектМетаданных: ПризнакУчетаПланаСчетов (MetadataObject: AccountingFlag)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает отображение данных признака учета плана счетов в режиме пароля (символы заменяются звездочками при отображении).   
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл для признаков типа [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
