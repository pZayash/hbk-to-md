ФормаКлиентскогоПриложения (ClientApplicationForm)

НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Задает навигационную ссылку, возвращаемую формой.  
Если [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка_(ClientApplicationForm.AutoURL).md)=[Ложь](lang__def_BooleanFalse.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку_(ClientApplicationWindow.GetURL).md) объекта [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md) возвращает значение свойства НавигационнаяСсылка основной формы.  
Если [АвтоНавигационнаяСсылка](ФормаКлиентскогоПриложения.АвтоНавигационнаяСсылка_(ClientApplicationForm.AutoURL).md)=[Истина](lang__def_BooleanTrue.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку_(ClientApplicationWindow.GetURL).md) возвращает навигационную ссылку основной формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
