ТипШтрихКода (BarcodeType)

Code39 (Code39)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Code 39. Позволяет кодировать большие латинские буквы (от A до Z), цифры (от 0 до 9) и некоторые специальные символы (например, знак доллара '$'). Может не содержать контрольного знака.

Использование в версии:

Доступен, начиная с версии 8.3.25.
