Process (Process)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Рабочий процесс](Рабочий_процесс_(IWorkingProcessInfo).md), [Неопределено](lang__def_Undefined.md).   
Содержит рабочий процесс, с которым установлено соединение, если сеанс назначен соединению. Иначе - [Неопределено](lang__def_Undefined.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
