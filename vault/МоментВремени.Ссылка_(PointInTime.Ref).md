МоментВремени (PointInTime)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Ссылка на объект информационной базы, [Неопределено](lang__def_Undefined.md).   
Содержит ссылку на объект в базе данных. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
