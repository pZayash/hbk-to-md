Описание:

Содержит варианты ориентации изометрических диаграмм. Используется для определения свойства [Ориентация](Диаграмма.Ориентация_(Chart.Orientation).md) диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartOrientation.

См. также:

[Диаграмма](Диаграмма_(Chart).md), свойство [Ориентация](Диаграмма.Ориентация_(Chart.Orientation).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
