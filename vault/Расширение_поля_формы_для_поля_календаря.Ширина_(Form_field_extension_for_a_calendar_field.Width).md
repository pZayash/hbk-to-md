# Расширение поля формы для поля календаря.Ширина

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля календаря](Расширение_поля_формы_для_поля_календаря_(Form_field_extension_for_a_calendar_field).md) › [Свойства](Расширение_поля_формы_для_поля_календаря__Свойства.md)

Расширение поля формы для поля календаря (Form field extension for a calendar field)

Ширина (Width)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит ширину календаря. Задается в символах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется, если значение свойства [ШиринаВМесяцах](Расширение_поля_формы_для_поля_календаря.ШиринаВМесяцах_(Form_field_extension_for_a_calendar_field.WidthInMonths).md)=0 или для режимов совместимости до [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md).

Использование в версии:

Доступен, начиная с версии 8.2.
