КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных (DataCompositionSettingStructureItemCollection)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных.md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных.md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных.md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных.md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных.md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных.md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных.md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
