Файл (File)

УстановитьНевидимость (SetHidden)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьНевидимость(<Невидимость>)

Параметры:

<Невидимость> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - установить атрибут "Скрытый", [Ложь](lang__def_BooleanFalse.md) - снять атрибут "Скрытый".

Описание:

Устанавливает значение атрибута файла "Скрытый" ("Hidden").

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применим только к файлам или каталогам, которые реально существуют.  
В операционной системе Linux метод не работает.

Пример:

|  |
| --- |
| ВыбФайл.УстановитьНевидимость(Истина); |
