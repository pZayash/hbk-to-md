НотацияDOM (DOMNotation)

КлонироватьУзел (CloneNode)

Доступен, начиная с версии 8.1.

Синтаксис:

КлонироватьУзел(<ГлубокоеКопирование>)

Параметры:

<ГлубокоеКопирование> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак "глубокого" копирования.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [ДокументDOM](ДокументDOM_(DOMDocument).md), [ЭлементDOM](ЭлементDOM_(DOMElement).md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [НотацияDOM](НотацияDOM_(DOMNotation).md), [СущностьDOM](СущностьDOM_(DOMEntity).md), [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md), [ТекстDOM](ТекстDOM_(DOMText).md), [КомментарийDOM](КомментарийDOM_(DOMComment).md), [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM_(DOMEntityReference).md), [ПространствоИменXPath](ПространствоИменXPath_(XPathNamespace).md).   
Копия данного узла.

Описание:

Копирует (клонирует) данный узел.  
Для копии не определены родительский узел и пользовательские данные.  
Копирование узла [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) копирует все атрибуты и их значения, включая сгенерированные XML процессором значения по умолчанию.   
Копирование дочерних узлов производится только в случае установленного признака "глубокого" копирования. "Глубокое" копирование включает текст, содержащийся в узле [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), если текст содержится как дочерний узел [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md).  
Копирование узлов [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) отличается от копирования их как части узла [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) тем, что признак явного указания свойства [Указан](АтрибутDOM.Указан_(DOMAttribute.Specified).md) будет установлен в значение [Истина](lang__def_BooleanTrue.md). Копирование узлов [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) всегда выполняется с копированием дочерних узлов (представляющих значение атрибута), вне зависимости от режима копирования.  
Копирование узла [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность_(DOMNodeType.EntityReference).md) автоматически конструирует поддерево, если соответствующий узел [Сущность](ТипУзлаDOM.Сущность_(DOMNodeType.Entity).md) доступен, вне зависимости от режима копирования.  
Копирование других типов узлов просто возвращает копию данного узла.   
В случае копирования поддерева в режиме Только для чтения создает редактируемую копию, но дочерние узлы копии узла [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность_(DOMNodeType.EntityReference).md) останутся в режиме Только для чтения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
