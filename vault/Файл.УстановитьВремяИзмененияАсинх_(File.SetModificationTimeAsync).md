Файл (File)

УстановитьВремяИзмененияАсинх (SetModificationTimeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьВремяИзмененияАсинх(<ВремяИзменения>)

Параметры:

<ВремяИзменения> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Устанавливаемое местное время изменения файла.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Устанавливает время последнего изменение файла. Применима только к файлам и каталогам, которые реально существуют.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.
