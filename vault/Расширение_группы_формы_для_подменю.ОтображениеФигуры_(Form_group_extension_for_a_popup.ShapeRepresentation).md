Расширение группы формы для подменю (Form group extension for a popup)

ОтображениеФигуры (ShapeRepresentation)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеФигурыКнопки](ОтображениеФигурыКнопки_(ButtonShapeRepresentation).md).   
Задает отображение фигуры кнопки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
