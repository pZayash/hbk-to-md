ОбъектМетаданных: Подсистема (MetadataObject: Subsystem)

ВключатьВКомандныйИнтерфейс (IncludeInCommandInterface)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, будет ли подсистема включена в командный интерфейс.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - включается.

Использование в версии:

Доступен, начиная с версии 8.2.
