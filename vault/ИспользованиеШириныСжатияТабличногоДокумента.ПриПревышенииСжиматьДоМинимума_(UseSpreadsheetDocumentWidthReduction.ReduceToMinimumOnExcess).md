ПриПревышенииСжиматьДоМинимума (ReduceToMinimumOnExcess)

Доступен, начиная с версии 8.3.13.

Описание:

Для областей табличного документа, в которых минимальная ширина больше переданной, минимальная ширина будет рассчитана автоматически

Использование в версии:

Доступен, начиная с версии 8.3.13.
