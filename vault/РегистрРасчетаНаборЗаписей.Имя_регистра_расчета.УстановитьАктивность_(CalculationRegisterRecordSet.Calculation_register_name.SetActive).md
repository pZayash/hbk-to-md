# РегистрРасчетаНаборЗаписей.<Имя регистра расчета>.УстановитьАктивность

РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

УстановитьАктивность (SetActive)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьАктивность(<Активность>)

Параметры:

<Активность> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак активности записей. [Истина](lang__def_BooleanTrue.md) - записи учитываются в итогах регистра. [Ложь](lang__def_BooleanFalse.md) - записи не учитываются в итогах регистра.

Описание:

Устанавливает значение свойства [Активность](РегистрРасчетаЗапись.Имя_регистра_расчета.Активность_(CalculationRegisterRecord.Calculation_register_name.Active).md) у всех записей, входящих в набор.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| НаборЗаписей.УстановитьАктивность(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
