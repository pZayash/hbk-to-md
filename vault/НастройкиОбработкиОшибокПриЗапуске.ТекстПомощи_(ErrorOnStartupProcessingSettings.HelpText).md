НастройкиОбработкиОшибокПриЗапуске (ErrorOnStartupProcessingSettings)

ТекстПомощи (HelpText)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, который будет добавлен на страницу ошибки при проблемах при входе в приложении.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
