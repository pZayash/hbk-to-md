Описание:

Определяет тип действия при превышении предельных значений, указанных в ограничении потребления ресурсов.  
Требуется наличие запущенного сервера администрирования (ras).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
