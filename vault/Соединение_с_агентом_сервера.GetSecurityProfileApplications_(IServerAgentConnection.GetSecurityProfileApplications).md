Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfileApplications (GetSecurityProfileApplications)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfileApplications(<Кластер>, <Профиль>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должен быть получен список разрешенных приложений.

<Профиль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности, которому принадлежит список.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [Приложение](Приложение_(ISecurityProfileApplication).md).

Описание:

Получает разрешенные приложения.

Доступность:

Интеграция.

Примечание:

Если профиль с этим именем не найден, то генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
