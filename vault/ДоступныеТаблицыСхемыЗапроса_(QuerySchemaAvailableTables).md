# ДоступныеТаблицыСхемыЗапроса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ГруппаДоступныхТаблицСхемыЗапроса](ГруппаДоступныхТаблицСхемыЗапроса_(QuerySchemaAvailableTablesGroup).md), [ДоступнаяТаблицаСхемыЗапроса](ДоступнаяТаблицаСхемыЗапроса_(QuerySchemaAvailableTable).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются группа доступных таблиц или доступная таблица.  
Возможно обращение к группе доступных таблиц или к доступной таблице посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ДоступныеТаблицыСхемыЗапроса.Индекс_(QuerySchemaAvailableTables.IndexOf).md)  
[Количество (Count)](ДоступныеТаблицыСхемыЗапроса.Количество_(QuerySchemaAvailableTables.Count).md)  
[Найти (Find)](ДоступныеТаблицыСхемыЗапроса.Найти_(QuerySchemaAvailableTables.Find).md)  
[Получить (Get)](ДоступныеТаблицыСхемыЗапроса.Получить_(QuerySchemaAvailableTables.Get).md)  

Описание:

Коллекция доступных таблиц схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
