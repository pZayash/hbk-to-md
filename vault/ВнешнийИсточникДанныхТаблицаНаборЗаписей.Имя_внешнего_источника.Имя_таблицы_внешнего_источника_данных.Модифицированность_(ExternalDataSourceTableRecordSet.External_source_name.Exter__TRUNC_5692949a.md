ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - записи изменены; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, были ли изменены записи в наборе после последнего считывания или записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, были ли изменены соответствующие записи другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.3.5.
