Описание:

Тип: ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>.   
Набор свойств содержит табличные части видов расчета. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ПланВидовРасчетаТабличнаяЧасть.<Имя плана видов расчета>.<Имя табличной части>).  
Предопределенные табличные части [ВедущиеВидыРасчетаСтрока.](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ВедущиеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)>, [БазовыеВидыРасчетаСтрока.](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)> и [ВытесняющиеВидыРасчетаСтрока.](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета.md)> доступны через специальные соответствующие свойства объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
