# ПеречислениеМенеджер.<Имя перечисления>.ПолучитьДанныеВыбора

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Перечисления](Перечисления.md) › [ПеречислениеМенеджер.<Имя перечисления>](ПеречислениеМенеджер.Имя_перечисления_(EnumManager.Enumeration_name).md) › [Методы](ПеречислениеМенеджер.Имя_перечисления__Методы.md)

ПеречислениеМенеджер.<Имя перечисления> (EnumManager.<Enumeration name>)

ПолучитьДанныеВыбора (GetChoiceData)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьДанныеВыбора(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит параметры для формирования списка значений:  
Структура, в которой обязательно должны содержаться следующие ключи:

- Отбор (Filter) - отбор, используемый для поиска данных, тип [Структура](Структура_(Structure).md); значения элементов отбора могут иметь тип [Массив](Массив_(Array).md), [СписокЗначений](СписокЗначений_(ValueList).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md) или иной тип, не являющийся коллекцией; для типов [Массив](Массив_(Array).md), [СписокЗначений](СписокЗначений_(ValueList).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md) сравнение выполняется с видом сравнения [ВСписке](ВидСравнения.ВСписке_(ComparisonType.InList).md), иначе вид сравнения [Равно](ВидСравнения.Равно_(ComparisonType.Equal).md);
- СтрокаПоиска (SearchString) - содержит строку, используемую при поиске данных; для быстрого выбора содержит значение [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   

Описание:

Формирует список значений данных выбора при вводе по строке, автоподборе текста и быстром выборе.  
Если требуется выполнить поиск более 32 слов или 1000 символов, то свойство [ОграничиватьСтрокуПоиска](СписокПолнотекстовогоПоиска.ОграничиватьСтрокуПоиска_(FullTextSearchList.LimitSearchString).md) требуется установить в значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

Примечание:

Если в модуле менеджера в методе указано два параметра, то будет вызван метод [ПолучитьДанныеВыбора](Глобальный_контекст.ПолучитьДанныеВыбора_(Global_context.GetChoiceData).md) глобального контекста.  
Выполнение метода приводит к вызову события [ОбработкаПолученияДанныхВыбора](ПеречислениеМенеджер.Имя_перечисления.ОбработкаПолученияДанныхВыбора_(EnumManager.Enumeration_name.ChoiceDataGetProcessing).md).  
  
В режиме совместимости конфигурации [Версия8\_3\_17](РежимСовместимости.Версия8_3_17_(CompatibilityMode.Version8_3_17).md) и ниже рекомендуется устанавливать свойство [ОграничиватьСтрокуПоиска](СписокПолнотекстовогоПоиска.ОграничиватьСтрокуПоиска_(FullTextSearchList.LimitSearchString).md) в значение [Истина](lang__def_BooleanTrue.md), чтобы при выполнении сложных запросов генерировалась ошибка.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.18.
