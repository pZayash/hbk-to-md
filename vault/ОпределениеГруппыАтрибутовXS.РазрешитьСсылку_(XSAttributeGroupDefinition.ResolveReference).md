# ОпределениеГруппыАтрибутовXS.РазрешитьСсылку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Схема XML](Схема_XML.md) › [ОпределениеГруппыАтрибутовXS](ОпределениеГруппыАтрибутовXS_(XSAttributeGroupDefinition).md) › [Методы](ОпределениеГруппыАтрибутовXS__Методы.md)

ОпределениеГруппыАтрибутовXS (XSAttributeGroupDefinition)

РазрешитьСсылку (ResolveReference)

Доступен, начиная с версии 8.1.

Синтаксис:

РазрешитьСсылку()

Возвращаемое значение:

Тип: [ОпределениеГруппыАтрибутовXS](ОпределениеГруппыАтрибутовXS_(XSAttributeGroupDefinition).md), [Неопределено](lang__def_Undefined.md).   
Если данная группа атрибутов является ссылкой на глобальную группу атрибутов, то возвращаемым значением будет глобальная группа атрибутов (определенная в схеме-владельце данного компонента или в одной из схем, указанных в директивах импорта или включения) или [Неопределено](lang__def_Undefined.md), если для имени, указанного в свойстве [Ссылка](ОпределениеГруппыАтрибутовXS.Ссылка_(XSAttributeGroupDefinition.Reference).md) группы атрибутов не обнаружено.

Описание:

Производит разыменование ссылки на группу атрибутов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
