КнопкаФормы (FormButton)

ТолькоВоВсехДействиях (OnlyInAllActions)

Не рекомендуется использовать, начиная с версии 8.3.15.

Рекомендуется использовать:

- [ПоложениеВКоманднойПанели](КнопкаФормы.ПоложениеВКоманднойПанели.md)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Указывает, следует ли скрывать кнопку в подменю "Все действия".  
Истина - кнопка скрывается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.3.15.

Описание изменено в версии 8.3.15.
