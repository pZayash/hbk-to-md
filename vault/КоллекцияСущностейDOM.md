Доступен, начиная с версии 8.1.

Элементы коллекции:

[СущностьDOM](СущностьDOM.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются узле сущностей DOM.  
Возможно обращение к узлу сущности DOM посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция узлов сущностей DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
