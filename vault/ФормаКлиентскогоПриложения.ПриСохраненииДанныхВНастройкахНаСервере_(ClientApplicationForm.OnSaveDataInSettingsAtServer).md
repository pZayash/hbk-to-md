ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриСохраненииДанныхВНастройкахНаСервере (OnSaveDataInSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСохраненииДанныхВНастройкахНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [Соответствие](Соответствие_(Map).md).   
Настройки формы.

Описание:

В обработчик передается объект-соответствие, в который будут сохраняться настройки. В объекте уже заполнены значения реквизитов, которые требовалось сохранять.   
В обработчике можно добавить в объект настроек какую-либо информацию. Например, поместить в реквизит настроек отбор.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Вызывается на сервере.

Использование в версии:

Доступен, начиная с версии 8.2.
