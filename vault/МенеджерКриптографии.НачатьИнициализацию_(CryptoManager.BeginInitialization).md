# МенеджерКриптографии.НачатьИнициализацию

МенеджерКриптографии (CryptoManager)

НачатьИнициализацию (BeginInitialization)

Доступен, начиная с версии 8.3.6.

Вариант синтаксиса: Без инициализации модуля криптографии

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИспользованиеИнтерактивногоРежима>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения установки внешнего компонента со следующими параметрами:

- МенеджерКриптографии - объект [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

.

<ИспользованиеИнтерактивногоРежима> (обязательный)

Тип: [ИспользованиеИнтерактивногоРежимаКриптографии](ИспользованиеИнтерактивногоРежимаКриптографии_(CryptoInteractiveModeUse).md).   
Определяет использование интерактивного режима работы модуля криптографии.  
Значение по умолчанию: [НеИспользовать](ИспользованиеИнтерактивногоРежимаКриптографии.НеИспользовать_(CryptoInteractiveModeUse.DontUse).md).

Вариант синтаксиса: С инициализацией модуля криптографии

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>, <ИспользованиеИнтерактивногоРежима>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения установки внешнего компонента со следующими параметрами:

- МенеджерКриптографии - объект [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

.

<ИмяМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы MS Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу-библиотеке, реализующей требуемый набор функций.   
Параметр используется при создании менеджера в Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: [Число](lang__def_Number.md).   
Тип модуля криптографии.

<ИспользованиеИнтерактивногоРежима> (необязательный)

Тип: [ИспользованиеИнтерактивногоРежимаКриптографии](ИспользованиеИнтерактивногоРежимаКриптографии_(CryptoInteractiveModeUse).md).   
Определяет использование интерактивного режима работы модуля криптографии.  
Значение по умолчанию: [НеИспользовать](ИспользованиеИнтерактивногоРежимаКриптографии.НеИспользовать_(CryptoInteractiveModeUse.DontUse).md).

Описание:

Начинает инициализацию объекта [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md) с переданными параметрами.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе параметры <ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии> игнорируются.

См. также:

[МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md), метод [ИнициализироватьАсинх](МенеджерКриптографии.ИнициализироватьАсинх_(CryptoManager.InitializeAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.24.
