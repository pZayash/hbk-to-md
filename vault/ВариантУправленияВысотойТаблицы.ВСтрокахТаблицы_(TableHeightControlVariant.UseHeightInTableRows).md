ВСтрокахТаблицы (UseHeightInTableRows)

Доступен, начиная с версии 8.3.9.

Описание:

Высота элемента устанавливается в соответствии со значением свойства [ВысотаВСтрокахТаблицы](ТаблицаФормы.ВысотаВСтрокахТаблицы_(FormTable.HeightInTableRows).md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
