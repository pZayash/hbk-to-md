ТаблицаФормы (FormTable)

ДанныеСтроки (RowData)

Доступен, начиная с версии 8.2.

Синтаксис:

ДанныеСтроки(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Произвольный.   
Строка таблицы, данные которой необходимо получить. Тип параметра зависит от типа редактируемого значения.

Возвращаемое значение:

Тип: [ДанныеФормыСтруктура](ДанныеФормыСтруктура.md), [ДанныеФормыЭлементДерева](ДанныеФормыЭлементДерева.md), [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md), Неопределено.   
Для таблиц, связанных с разными типами данных, тип возращаемого значения данного метода может различаться:

- Для динамического списка тип свойства - [ДанныеФормыСтруктура](ДанныеФормыСтруктура.md);
- Для дерева значений - [ДанныеФормыЭлементДерева](ДанныеФормыЭлементДерева.md);
- Для остальных (таблица значений, табличные части и др.) - [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md).

Если таблица не отображается на форме, если в таблице нет строк или передана строка, отсутствующая в таблице, то возвращается Неопределено.

Описание:

Возвращает значение, содержащее данные указанной строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.17.
