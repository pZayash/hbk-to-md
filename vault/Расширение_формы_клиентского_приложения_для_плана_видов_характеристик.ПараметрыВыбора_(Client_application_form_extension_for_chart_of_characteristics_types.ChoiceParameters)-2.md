Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ПараметрыВыбора (ChoiceParameters)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md).   
Значения, по которым проверяется возможность выбора для форм, находящихся в режиме выбора.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
