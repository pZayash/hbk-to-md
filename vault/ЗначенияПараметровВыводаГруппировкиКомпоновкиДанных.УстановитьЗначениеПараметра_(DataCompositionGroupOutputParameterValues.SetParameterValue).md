ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных (DataCompositionGroupOutputParameterValues)

УстановитьЗначениеПараметра (SetParameterValue)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьЗначениеПараметра(<Параметр>, <Значение>)

Параметры:

<Параметр> (обязательный)

Тип: [Строка](lang__def_String.md), [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных_(DataCompositionParameter).md).   
Имя параметра или параметр компоновки данных, значение которого нужно установить.

<Значение> (обязательный)

Значение, которое нужно установить.

Описание:

Устанавливает свойство [Значение](ЗначениеПараметраКомпоновкиДанных.Значение_(DataCompositionParameterValue.Value).md) и включает свойство [Использование](ЗначениеПараметраКомпоновкиДанных.Использование_(DataCompositionParameterValue.Use).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если параметр с указанным именем не найден, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
