КоллекцияДействийЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemActionCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого действия в коллекции.

Описание варианта метода:

Удаляет действие по заданному индексу из коллекции действий элемента результата глобального поиска.

Вариант синтаксиса: По действию

Синтаксис:

Удалить(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеЭлементаРезультатаГлобальногоПоиска](ДействиеЭлементаРезультатаГлобальногоПоиска_(GlobalSearchResultItemAction).md).   
Удаляемое действие элемента результата глобального поиска.

Описание варианта метода:

Удаляет заданное действие из коллекции действий элемента результата глобального поиска.

Описание:

Удаляет действие из коллекции действий элемента результата глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
