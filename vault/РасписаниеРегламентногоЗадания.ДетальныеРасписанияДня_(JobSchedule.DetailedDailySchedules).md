# РасписаниеРегламентногоЗадания.ДетальныеРасписанияДня

РасписаниеРегламентногоЗадания (JobSchedule)

ДетальныеРасписанияДня (DetailedDailySchedules)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив детальных расписаний на день. Расписание может содержать только значения следующих свойств: [ВремяНачала](РасписаниеРегламентногоЗадания.ВремяНачала_(JobSchedule.BeginTime).md), [ВремяКонца](РасписаниеРегламентногоЗадания.ВремяКонца_(JobSchedule.EndTime).md), [ВремяЗавершения](РасписаниеРегламентногоЗадания.ВремяЗавершения_(JobSchedule.CompletionTime).md), [ИнтервалЗавершения](РасписаниеРегламентногоЗадания.ИнтервалЗавершения_(JobSchedule.CompletionInterval).md), [ПериодПовтораВТечениеДня](РасписаниеРегламентногоЗадания.ПериодПовтораВТечениеДня_(JobSchedule.RepeatPeriodInDay).md), [ПаузаПовтора](РасписаниеРегламентногоЗадания.ПаузаПовтора_(JobSchedule.RepeatPause).md). Если задано свойство [ДетальныеРасписанияДня](РасписаниеРегламентногоЗадания.ДетальныеРасписанияДня_(JobSchedule.DetailedDailySchedules).md), то основное расписание не должно содержать вышеперечисленные свойства.  
Свойство возвращает копию массива, поэтому просто его модификация не имеет смысла. Измененный массив должен быть обратно присвоен свойству для применения:  

|  |
| --- |
| ДетальноеРасписание = Расписание.ДетальныеРасписанияДня; ДетальноеРасписание.Добавить(Новый РасписаниеРегламентногоЗадания()); Расписание.ДетальныеРасписанияДня = ДетальноеРасписание; |

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.17.
