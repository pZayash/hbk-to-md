ТабличныйДокумент (SpreadsheetDocument)

НачатьГруппуКолонок (StartColumnGroup)

Доступен, начиная с версии 8.0.

Синтаксис:

НачатьГруппуКолонок(<ИмяГруппы>, <ОткрытаЛиГруппа>)

Параметры:

<ИмяГруппы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя группы колонок.

<ОткрытаЛиГруппа> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость открытия группы. [Истина](lang__def_BooleanTrue.md) - выведенная группа будет открыта; [Ложь](lang__def_BooleanFalse.md) - группа будет выведена в свернутом виде.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Начинает новую группу колонок. Можно создавать вложенные группы колонок.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТабДок.НачатьГруппуКолонок("По всем складам", Истина); |

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [ЗакончитьГруппуКолонок](ТабличныйДокумент.ЗакончитьГруппуКолонок_(SpreadsheetDocument.EndColumnGroup).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
