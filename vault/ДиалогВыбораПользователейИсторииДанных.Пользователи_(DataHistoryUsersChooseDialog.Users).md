ДиалогВыбораПользователейИсторииДанных (DataHistoryUsersChooseDialog)

Пользователи (Users)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив полных имен пользователей.  
Устанавливается, если необходимо заполнить параметры отбора при открытии диалога. Если диалог был закрыт нажатием кнопки "OK", содержит результат выбора.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
