ЭлементыШкалыВремени (TimeScaleItems)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементШкалыВремени](ЭлементШкалыВремени_(TimeScaleItem).md).   
Искомый элемент.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента шкалы в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента шкалы в коллекции элементов шкалы времени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
