ОпределениеТипаДокументаDOM (DOMDocumentType)

ПредыдущийСоседний (PreviousSibling)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [Неопределено](lang__def_Undefined.md).   
Предыдущий узел DOM или [Неопределено](lang__def_Undefined.md), если такого нет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
