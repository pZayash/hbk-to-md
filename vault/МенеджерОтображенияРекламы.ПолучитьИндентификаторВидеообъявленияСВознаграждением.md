МенеджерОтображенияРекламы (AdRepresentationManager)

ПолучитьИндентификаторВидеообъявленияСВознаграждением (GetRewardedVideoID)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьИндентификаторВидеообъявленияСВознаграждением()

Возвращаемое значение:

Тип: Строка.   
Если идентификатор не задан, то возвращается пустая строка.

Описание:

Возвращает идентификатор видеообъявления, который используется при загрузке рекламы из сети Интернет.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.14.
