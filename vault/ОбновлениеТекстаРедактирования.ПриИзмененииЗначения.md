ОбновлениеТекстаРедактирования (EditTextUpdate)

ПриИзмененииЗначения (OnValueChange)

Доступен, начиная с версии 8.2.

Описание:

Текст в поле ввода формы клиентского приложения обновляется в том случае, если устанавливаемое в поле ввода значение отличается от имеющегося в данных, редактируемых в поле ввода.

Использование в версии:

Доступен, начиная с версии 8.2.
