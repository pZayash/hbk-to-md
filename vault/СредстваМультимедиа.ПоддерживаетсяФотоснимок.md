СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяФотоснимок (PhotoSupported)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоддерживаетсяФотоснимок(<ТипКамеры>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства.md).   
Определяет, имеет ли устройство конкретную камеру (на передней или задней панели), в зависимости от переданного параметра.  
Значение по умолчанию: Авто.

Возвращаемое значение:

Тип: Булево.   
Если тип камеры устанавливается автоматически, возвращает Истина в случаях:

- Android - при наличии камеры,
- iOS - если система поддерживает фотосъемку,
- Windows Phone - при наличии камеры.

Если тип камеры установлен как [Передняя](ТипКамерыУстройства.Передняя.md) или [Задняя](ТипКамерыУстройства.Задняя.md), возвращает Истина в случаях:

- Android - при наличии требуемой камеры,
- iOS - при наличии требуемой камеры и если система поддерживает фотосъемку,
- Windows Phone - при наличии камеры.

Описание:

Определяет, поддерживается ли фотографирование системой.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Не поддерживается в мобильном приложении для Windows Desktop.
