XBase (XBase)

Открыта (IsOpen)

Доступен, начиная с версии 8.0.

Синтаксис:

Открыта()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - файл открыт; [Ложь](lang__def_BooleanFalse.md) - файл не открыт.

Описание:

Определяет, открыт ли файл таблицы базы данных.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Если Не xB.Открыта() Тогда     Предупреждение(НСтр("ru = 'Не удалось открыть указанный файл!';"      + " en = 'Can't open this file!'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
