ОформлениеЯчейки (CellAppearance)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - отображает значение ячейки табличного поля как гиперссылку. При выборе ячейки с установленным свойством будет порождаться событие [Выбор](ТабличноеПоле.Выбор_(TableBox.Selection).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
