ОкноКлиентскогоПриложения (ClientApplicationWindow)

Содержимое (Content)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Свойство содержит фиксированный массив форм, открытых в этом окне клиентского приложения.  
Элементы массива - объекты типа [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) или [Форма](Форма_(Form).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент, внешнее соединение.

Примечание:

- Для основного окна свойство содержит пустой массив.
- Для начальной страницы свойство содержит список форм на начальной странице.
- Для вспомогательных окон свойство содержит массив со списком форм, открытых в этом окне. Основная форма является первым элементом массива.

Использование в версии:

Доступен, начиная с версии 8.3.5.
