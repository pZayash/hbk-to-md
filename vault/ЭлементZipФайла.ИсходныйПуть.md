Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [ИсходныйПуть](ЭлементФайлаАрхива.ИсходныйПуть.md)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит путь файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Без режима совместимости содержит части пути в значении свойства разделены в зависимости от используемой операционной системы:

- В Linux элементы пути разделяются символом "/".
- В Windows элементы пути разделяются символом"\".
- В режиме совместимости [Версия8\_3\_8](РежимСовместимости.Версия8_3_8.md) и ниже части пути разделены символом "/".

В режиме совместимости [Версия8\_3\_8](РежимСовместимости.Версия8_3_8.md) и ниже части пути разделены символом "/".

Использование в версии:

Доступен, начиная с версии 8.3.11.
