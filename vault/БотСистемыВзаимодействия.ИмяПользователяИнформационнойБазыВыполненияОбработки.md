БотСистемыВзаимодействия (CollaborationSystemBot)

ИмяПользователяИнформационнойБазыВыполненияОбработки (ExecuteProcessingInfoBaseUserName)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя пользователя, от которого будет выполняться сеанс, в котором будет вызван обработчик [ОбработкаСообщенияСистемыВзаимодействия](Модуль_бота.ОбработкаСообщенияСистемыВзаимодействия.md).  
Если имя пользователя не задано, сеанс будет выполняться от имени пользователя по умолчанию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
