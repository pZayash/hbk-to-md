# ОболочкаActiveDocument

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.0.

Методы:

[Записать (Write)](ОболочкаActiveDocument.Записать_(ActiveDocumentShell.Write).md)  
[Получить (Get)](ОболочкаActiveDocument.Получить_(ActiveDocumentShell.Get).md)  

Описание:

Используется для доступа к Active document. Объект может быть записан в файл.  
Может быть получен методами ПолучитьМакет() и [ПолучитьОбщийМакет](Глобальный_контекст.ПолучитьОбщийМакет_(Global_context.GetCommonTemplate).md), если тип макета [ActiveDocument](ТипМакета.ActiveDocument_(TemplateType.ActiveDocument).md).

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
