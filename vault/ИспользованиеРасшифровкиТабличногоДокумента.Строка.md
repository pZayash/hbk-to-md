ИспользованиеРасшифровкиТабличногоДокумента (SpreadsheetDocumentDetailUse)

Строка (Row)

Доступен, начиная с версии 8.0.

Описание:

Обработка расшифровки вызывается для всех ячеек строки, следующей за ячейкой, содержащей значение расшифровки.

Использование в версии:

Доступен, начиная с версии 8.0.
