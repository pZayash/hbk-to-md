ФормаКлиентскогоПриложения (ClientApplicationForm)

Группировка (Group)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГруппировкаПодчиненныхЭлементовФормы](ГруппировкаПодчиненныхЭлементовФормы_(ChildFormItemsGroup).md).   
Содержит способ группировки дочерних элементов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
