ОтборПользователейСистемыВзаимодействия (CollaborationSystemUsersFilter)

АдминистраторАбонента (SubscriberAdministrator)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Если содержит значение типа [Булево](lang__Булево_(Boolean).md), то метод [ПолучитьПользователей](МенеджерСистемыВзаимодействия.ПолучитьПользователей_(CollaborationSystemManager.GetUsers).md) возвращает соответственно только администраторов абонента или не администраторов абонента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
