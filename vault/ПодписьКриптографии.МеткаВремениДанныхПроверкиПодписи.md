ПодписьКриптографии (CryptoSignature)

МеткаВремениДанныхПроверкиПодписи (SignatureVerificationDataTimestamp)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии.md), Неопределено.   
Содержит метку времени данных проверки подписи - необходимый для CAdES-X атрибут.  
Для [CAdESXType1](ТипПодписиКриптографии.CAdESXType1.md) и [CAdESXType2](ТипПодписиКриптографии.CAdESXType2.md) в данном поле хранятся различные метки времени.  
  
Если метка отсутствует значение поля - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
