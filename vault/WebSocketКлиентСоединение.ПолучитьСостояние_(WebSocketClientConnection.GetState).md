WebSocketКлиентСоединение (WebSocketClientConnection)

ПолучитьСостояние (GetState)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьСостояние()

Возвращаемое значение:

Тип: [СостояниеWebSocketСоединения](СостояниеWebSocketСоединения_(WebSocketConnectionState).md).   

Описание:

Возвращает состояние WebSocket-соединения в данный момент времени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
