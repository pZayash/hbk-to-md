КоллекцияЭлементовУправленияИнтерфейсами (InterfaceControlItemCollection)

ПереключитьИнтерфейс (SwitchInterface)

Доступен, начиная с версии 8.0.

Синтаксис:

ПереключитьИнтерфейс(<ИменаИнтерфейсов>)

Параметры:

<ИменаИнтерфейсов> (необязательный)

Тип: Строка.   
Перечисленные через запятую имена интерфейсов, которые следует сделать видимыми.

Описание:

Способ группового управления видимостью командных интерфейсов для режима запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение.md). При вызове данного метода сначала становятся невидимыми все интерфейсы, кроме имеющих значение Ложь у свойства [Переключаемый](ЭлементУправленияИнтерфейсом.Переключаемый.md), а затем становятся видимыми перечисленные в параметре интерфейсы.  
При работе в режиме [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение.md) вызов данного метода игнорируется.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ГлавныйИнтерфейс.ПереключитьИнтерфейс("Основной,Маркетинг"); |

Использование в версии:

Доступен, начиная с версии 8.0.
