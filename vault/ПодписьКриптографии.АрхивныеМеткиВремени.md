ПодписьКриптографии (CryptoSignature)

АрхивныеМеткиВремени (ArchiveTimestamps)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив.md).   
Массив элементов типа [МеткаВремениКриптографии](МеткаВремениКриптографии.md).  
  
Содержит архивные метки времени подписи - необходимый для CAdES-A атрибут.  
Для [CAdESAv2](ТипПодписиКриптографии.CAdESAv2.md) и [CAdESAv3](ТипПодписиКриптографии.CAdESAv3.md) в данном поле содержатся различные типы меток времени.  
  
Если метка отсутствует значение поля - Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
