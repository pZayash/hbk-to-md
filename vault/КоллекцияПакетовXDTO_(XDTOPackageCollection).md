# КоллекцияПакетовXDTO

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПакетXDTO](ПакетXDTO_(XDTOPackage).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются пакеты XDTO.  
Возможно обращение к пакету XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияПакетовXDTO.Количество_(XDTOPackageCollection.Count).md)  
[Получить (Get)](КоллекцияПакетовXDTO.Получить_(XDTOPackageCollection.Get).md)  

Описание:

Коллекция пакетов XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
