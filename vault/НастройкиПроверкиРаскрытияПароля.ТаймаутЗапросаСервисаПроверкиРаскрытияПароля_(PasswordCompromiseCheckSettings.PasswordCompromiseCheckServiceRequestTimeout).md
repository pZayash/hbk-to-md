НастройкиПроверкиРаскрытияПароля (PasswordCompromiseCheckSettings)

ТаймаутЗапросаСервисаПроверкиРаскрытияПароля (PasswordCompromiseCheckServiceRequestTimeout)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Интервал времени в секундах, после истечения которого проверка будет признана неудачной.  
Допустимые значения: от 0 до 4294967295.   
Значение по умолчанию: 1.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
