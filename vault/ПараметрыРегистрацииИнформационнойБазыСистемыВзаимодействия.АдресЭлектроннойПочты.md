ПараметрыРегистрацииИнформационнойБазыСистемыВзаимодействия (CollaborationSystemInfoBaseRegistrationParameters)

АдресЭлектроннойПочты (Email)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Адрес электронной почты абонента.  
Если не указан, то при вызове методов [ВыполнитьРегистрациюИнформационнойБазы](МенеджерСистемыВзаимодействия.ВыполнитьРегистрациюИнформационнойБазы.md), [ВыполнитьРегистрациюИнформационнойБазыАсинх](МенеджерСистемыВзаимодействия.ВыполнитьРегистрациюИнформационнойБазыАсинх.md) и [НачатьРегистрациюИнформационнойБазы](МенеджерСистемыВзаимодействия.НачатьРегистрациюИнформационнойБазы.md) будет сгенерировано исключение, если в информационной базе нет объекта [ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия](ДанныеРегистрацииИнформационнойБазыСистемыВзаимодействия.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.20.
