ЗаписьFastInfoset (FastInfosetWriter)

ЗаписатьИнструкциюОбработки (WriteProcessingInstruction)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьИнструкциюОбработки(<Имя>, <Текст>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя инструкции обработки.

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст инструкции обработки.

Описание:

Записывает инструкцию обработки XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запись.ЗаписатьИнструкциюОбработки("xml-stylesheet",     "type=""text/xsl"" href=""trasform.xsl"""); |

Использование в версии:

Доступен, начиная с версии 8.1.
