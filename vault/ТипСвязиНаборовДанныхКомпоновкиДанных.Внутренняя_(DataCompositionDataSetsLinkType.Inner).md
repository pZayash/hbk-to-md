ТипСвязиНаборовДанныхКомпоновкиДанных (DataCompositionDataSetsLinkType)

Внутренняя (Inner)

Доступен, начиная с версии 8.1.

Описание:

Записи исходного набора данных, для которых не были найдены записи в наборе данных – приемнике, в результат компоновки не попадут.

Использование в версии:

Доступен, начиная с версии 8.1.
