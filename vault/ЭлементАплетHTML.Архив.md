ЭлементАплетHTML (HTMLAppletElement)

Архив (Archive)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Список адресов URI архивов, содержащих ресурсы и классы, которые будут загружаться перед загрузкой самого аплета.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
