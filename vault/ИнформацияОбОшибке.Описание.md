ИнформацияОбОшибке (ErrorInfo)

Описание (Description)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержательное описание ошибки (исключения). Данное свойство может содержать:

- Краткое сообщение об ошибке,
- Код ошибки (если есть),
- Подробную техническую информацию об ошибке (если есть).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
