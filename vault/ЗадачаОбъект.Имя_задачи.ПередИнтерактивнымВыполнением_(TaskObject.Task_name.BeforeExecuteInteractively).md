# ЗадачаОбъект.<Имя задачи>.ПередИнтерактивнымВыполнением

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [События](ЗадачаОбъект.Имя_задачи__События.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПередИнтерактивнымВыполнением (BeforeExecuteInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИнтерактивнымВыполнением(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при выполнении метода [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно_(TaskObject.Task_name.ExecuteTaskInteractively).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) не вызывается.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
