Картинка (Picture)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, отображаемую в действии элемента результата глобального поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если не задан ни <Текст>, ни <Картинка>, то в качестве текста гиперссылки выводится представление <Значения>.

Использование в версии:

Доступен, начиная с версии 8.3.18.
