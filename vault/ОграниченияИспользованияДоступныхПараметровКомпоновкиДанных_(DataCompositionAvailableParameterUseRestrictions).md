Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОграничениеИспользованияДоступногоПараметраКомпоновкиДанных](ОграничениеИспользованияДоступногоПараметраКомпоновкиДанных_(DataCompositionAvailableParameterUseRestriction).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ограничения использования параметров.  
Возможно обращение к ограничениям использования параметров посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция ограничений использования доступных параметров системы компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
