Форма (Form)

ВводДоступен (IsInputAvailable)

Доступен, начиная с версии 8.0.

Синтаксис:

ВводДоступен()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ввод данных в форму возможен (форма активна); [Ложь](lang__def_BooleanFalse.md) - форма не активна.

Описание:

Определяет, возможен ли ввод данных в форму. Может быть использован для того, чтобы определить, активизирована ли текущая форма.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ВводДоступен() Тогда     Сообщить("Введите текст"); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
