Расширение формы клиентского приложения для справочника (Client application form extension for catalogs)

Ключ (Key)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   
Ключ объекта.   
В случае отсутствия или неверного значения создается новый объект с использованием остальных параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
