SessionID (SessionID)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер сеанса. Целое число, уникальное среди всех сеансов данной информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
