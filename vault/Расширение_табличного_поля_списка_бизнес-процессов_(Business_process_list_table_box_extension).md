Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для бизнес-процессов. Для данного расширения свойство [ТекущаяСтрока](ПолеСписка.ТекущаяСтрока_(ListBox.CurrentRow).md) содержит значение типа [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [БизнесПроцессСписок.](БизнесПроцессСписок.Имя_бизнес-процесса_(BusinessProcessList.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессСписок.Имя_бизнес-процесса_(BusinessProcessList.Business_process_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
