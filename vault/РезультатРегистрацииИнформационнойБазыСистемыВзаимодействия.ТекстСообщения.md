Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Если свойство [РегистрацияВыполнена](РезультатРегистрацииИнформационнойБазыСистемыВзаимодействия.РегистрацияВыполнена.md) имеет значение Ложь, то в данном свойстве содержится текст инструкции для пользователя с дальнейшими действиями.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
