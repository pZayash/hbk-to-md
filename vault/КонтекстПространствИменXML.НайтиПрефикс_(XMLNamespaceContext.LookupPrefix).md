# КонтекстПространствИменXML.НайтиПрефикс

КонтекстПространствИменXML (XMLNamespaceContext)

НайтиПрефикс (LookupPrefix)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПрефикс(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен, для которого необходимо получить префикс.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если URI пространству имен не соответствует префикс, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Получает префикс по URI пространства имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
