МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ПолучитьРежимРазделенияСоставныхСлов (GetCompositeWordsSeparationMode)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПолучитьРежимРазделенияСоставныхСлов()

Возвращаемое значение:

Тип: [РежимРазделенияСоставныхСлов](РежимРазделенияСоставныхСлов_(CompositeWordsSeparationMode).md).   

Описание:

Получает текущее значение режима использования разделения составных слов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
