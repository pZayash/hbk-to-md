ФасетМинимальногоИсключающегоЗначенияXS (XSMinExclusiveFacet)

Исключающий (Exclusive)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - значение фасета не включается в диапазон ограничения, в противном случае - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение свойства противоположно значению свойства [ФасетМинимальногоИсключающегоЗначенияXS.Включающий](ФасетМинимальногоИсключающегоЗначенияXS.Включающий_(XSMinExclusiveFacet.Inclusive).md).

Доступен, начиная с версии 8.1.
