ЧтениеFastInfoset (FastInfosetReader)

Имя (Name)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя текущего узла XML.  
Содержит квалифицированное имя для текущего узла. Имя зависит от типа текущего узла:

- [Атрибут](ТипУзлаXML.Атрибут_(XMLNodeType.Attribute).md) - имя атрибута;
- [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md) - имя типа документа;
- [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md) - имя элемента;
- [КонецЭлемента](ТипУзлаXML.КонецЭлемента_(XMLNodeType.EndElement).md) - имя элемента;
- [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md) - имя ссылки на сущность;
- [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки_(XMLNodeType.ProcessingInstruction).md) - адрес инструкции обработки;
- [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML_(XMLNodeType.XMLDeclaration).md) - строка xml.

Для узлов, для которых имя не определено, свойство содержит пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
