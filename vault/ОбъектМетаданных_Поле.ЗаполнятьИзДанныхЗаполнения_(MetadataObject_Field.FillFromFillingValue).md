ОбъектМетаданных: Поле (MetadataObject: Field)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство определяет возможность заполнения поля из данных заполнения. Используется для заполнения полей из значений текущего отбора списков. Если установлено [Истина](lang__def_BooleanTrue.md), то процедура стандартной обработки заполнения будет заполнять поле из данных заполнения. Если в данных заполнения нет данных для заполнения этого поля или свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то поле будет заполнен из свойства [ЗначениеЗаполнения](ОбъектМетаданных_Поле.ЗначениеЗаполнения_(MetadataObject_Field.FillingValue).md), если оно установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
