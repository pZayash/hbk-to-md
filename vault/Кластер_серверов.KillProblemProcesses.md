Кластер серверов (IClusterInfo)

KillProblemProcesses (KillProblemProcesses)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Режим принудительного завершения проблемных процессов.   
Если система мониторинга кластера признает менеджер кластера или рабочий процесс проблемным, то в технологическом журнале выводится событие ATTN. Если данное свойство имеет значение Истина, то проблемный процесс принудительно завершается средствами операционной системы с записью дампа.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
