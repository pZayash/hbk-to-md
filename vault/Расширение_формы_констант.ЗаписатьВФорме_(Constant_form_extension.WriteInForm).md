Расширение формы констант (Constant form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - набор записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает набор констант, открытый в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Запись занесена в ИБ."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
