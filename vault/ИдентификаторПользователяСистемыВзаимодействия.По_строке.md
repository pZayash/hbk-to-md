ИдентификаторПользователяСистемыВзаимодействия (CollaborationSystemUserID)

По строке

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый ИдентификаторПользователяСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Строковое представление идентификатора.

Описание:

Создает идентификатор пользователя на основе указанной строки.  
Данный конструктор можно использовать, если нужно произвести действие с пользователем, идентификатор которого хранится в информационной базе в виде строки.  
Корректность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.11.
