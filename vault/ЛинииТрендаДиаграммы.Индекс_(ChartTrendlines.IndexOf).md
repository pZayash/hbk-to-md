ЛинииТрендаДиаграммы (ChartTrendlines)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Индекс(<Линия тренда>)

Параметры:

<Линия тренда> (необязательный)

Тип: [ЛинияТрендаДиаграммы](ЛинияТрендаДиаграммы_(ChartTrendline).md).   
Линия тренда, индекс которой определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
