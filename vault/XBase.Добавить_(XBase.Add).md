# XBase.Добавить

XBase (XBase)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить()

Описание:

Добавляет новую пустую запись. При этом поля и свойства объекта [Ключ](XBase.Ключ_(XBase.Key).md) очищаются. Для записи изменений в таблицу базы данных необходимо инициировать запись.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB.Добавить(); xB.CODE = "00004"; xB.NAME = "Клавиатура"; xB.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
