<!-- signature:end -->

Расширение табличного поля списка видов расчета (Calculation types list table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от добавления вида расчета. Если в теле процедуры-обработчика установить данному параметру значение Истина, добавление выполнено не будет.  
Значение по умолчанию: Ложь.

<Копирование>

Тип: Булево.   
Определяет режим копирования. Если установлено Истина, то происходит копирование строки.

Описание:

Вызывается при вызове пользователем добавления вида расчета при редактировании в строке.

Доступность:

Толстый клиент.

Примечание:

Расширяется набор параметров по сравнению с одноименным событием объекта [ТабличноеПоле](ТабличноеПоле.md).

См. также:

[ТабличноеПоле](ТабличноеПоле.md), событие [ПередНачаломДобавления](ТабличноеПоле.ПередНачаломДобавления.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
