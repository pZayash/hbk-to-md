МенеджерКонтактов (ContactManager)

ПоддерживаетсяДобавлениеКонтактов (ContactsAddingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяДобавлениеКонтактов()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md) в зависимости от используемой операционной системы:

- Android – всегда,
- iOS – всегда,
- Windows Phone – никогда.

Описание:

Определяет возможность добавления, редактирования, или удаления контактов на данном устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКонтактов](МенеджерКонтактов_(ContactManager).md), метод [ДобавитьКонтакт](МенеджерКонтактов.ДобавитьКонтакт_(ContactManager.AddContact).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
