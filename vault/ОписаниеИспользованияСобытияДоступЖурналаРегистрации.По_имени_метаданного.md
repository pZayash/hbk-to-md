ОписаниеИспользованияСобытияДоступЖурналаРегистрации (EventLogAccessEventUseDescription)

По имени метаданного

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ОписаниеИспользованияСобытияДоступЖурналаРегистрации(<Объект>)

Параметры:

<Объект> (необязательный)

Тип: Строка.   
Полное имя объекта метаданных.

Описание:

Объект создается с указанным полным именем метаданного в качестве объекта и пустыми массивами в свойствах [ПоляДоступа](ОписаниеИспользованияСобытияДоступЖурналаРегистрации.ПоляДоступа.md) и [ПоляРегистрации](ОписаниеИспользованияСобытияДоступЖурналаРегистрации.ПоляРегистрации.md).

Использование в версии:

Доступен, начиная с версии 8.2.
