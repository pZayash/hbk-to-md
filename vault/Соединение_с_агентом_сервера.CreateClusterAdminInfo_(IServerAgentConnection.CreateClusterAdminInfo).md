Соединение с агентом сервера (IServerAgentConnection)

CreateClusterAdminInfo (CreateClusterAdminInfo)

Доступен, начиная с версии 8.1.

Синтаксис:

CreateClusterAdminInfo()

Описание:

Создает администратора кластера с пустыми значениями свойств. Этот объект может быть использован для регистрации нового администратора после установки правильных значений его свойств.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
