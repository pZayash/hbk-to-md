ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

Закрыть (Close)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Закрыть()

Описание:

Закрывает окно приложения, на которое ссылается объект в тестируемом приложении.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
