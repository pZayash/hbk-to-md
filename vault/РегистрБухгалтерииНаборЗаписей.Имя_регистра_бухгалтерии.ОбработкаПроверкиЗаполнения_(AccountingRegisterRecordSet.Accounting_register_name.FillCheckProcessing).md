РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

ОбработкаПроверкиЗаполнения (FillCheckProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаПроверкиЗаполнения(<Отказ>, <ПроверяемыеРеквизиты>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то будет выполнен отказ от продолжения работы после выполнения проверки заполнения.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ПроверяемыеРеквизиты>

Тип: [Массив](Массив_(Array).md).   
Массив путей к реквизитам, для которых будет выполнена проверка заполнения. Массив может быть модифицирован удалением или добавлением путей к необходимым реквизитам.

Описание:

Вызывается перед системной проверкой заполнения реквизитов при записи в форме, а также при выполнении метода [ПроверитьЗаполнение](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.ПроверитьЗаполнение_(AccountingRegisterRecordSet.Accounting_register_name.CheckFilling).md).   
Позволяет разработчику конфигурации выполнить проверку самостоятельно. При этом в обработчике можно полностью отказаться от системной обработки (очистив список проверяемых реквизитов), отказаться от проверки системой части реквизитов (выполнив проверку отдельных реквизитов особенным образом и исключив эти реквизиты из списка), а также добавить для проверки другие реквизиты, проверка которых не была указана.  
Пример:

|  |
| --- |
| Процедура ОбработкаПроверкиЗаполнения(Отказ, ПроверяемыеРеквизиты)  // Сообщим платформе, что мы сами обрабатываем проверку реквизита Организация  ПроверяемыеРеквизиты.Удалить(ПроверяемыеРеквизиты.Найти("Организация"));  Для Каждого Проводка Из ЭтотОбъект Цикл     Индекс = ЭтотОбъект.Индекс(Проводка)     Если Не ЗначениеЗаполнено(Проводка.Организация) Тогда         Сообщение = Новый СообщениеПользователю();         Сообщение.Текст = "В строке " + Индекс + " не указана Организация";         Сообщение.Поле = "[" + Индекс + "].Организация";         Сообщение.УстановитьДанные(ЭтотОбъект);         Сообщение.Сообщить();         Отказ = Истина;     КонецЕсли; КонецЦикла;  КонецПроцедуры |

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.
