Расширение поля формы для поля ввода (Form field extension for a text box)

РазрешитьДублированиеМножественныхЗначений (AllowMultipleValuesDuplicates)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет возможность ввода дубликатов множественных значений:

- Ложь – при вводе контролируется уникальность введенных множественных значений.
  - При добавлении в результате интерактивных действий дублирующиеся значения игнорируется. Проверка на дублирующие значения производится после вызова обработчика [ДобавлениеМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.ДобавлениеМножественныхЗначений.md), перед добавлением значения. Если проверка не проходит, то показывается оповещение пользователю с текстом ""Значение" уже есть в списке".
- Истина – разрешает ввод одинаковых множественных значений.

Значение по умолчанию: Ложь.  
Запрет дублирования множественных значений также распространяется на работу стандартного диалога редактирования списка значений, который вызывается при нажатии на кнопку "Открыть" или сочетанием клавиш Ctrl+Shift+F4, если поле ввода содержит данные типа [СписокЗначений](СписокЗначений.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
