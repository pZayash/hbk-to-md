Поток (Stream)

РазмерАсинх (SizeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазмерАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - тип Число, размер данных в байтах.

Описание:

Асинхронная функция. Получает размер данных в байтах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции.md) = Ложь, то при вызове этого метода будет сгенерировано исключение.
