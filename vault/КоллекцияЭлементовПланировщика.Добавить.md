КоллекцияЭлементовПланировщика (PlannerItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Добавить(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: Дата.   
Дата начала элемента планировщика.

<Конец> (обязательный)

Тип: Дата.   
Дата конца элемента планировщика.

Возвращаемое значение:

Тип: [ЭлементПланировщика](ЭлементПланировщика.md).   
Добавленный элемент планировщика.

Описание:

Добавляет элемент планировщика в конец коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
