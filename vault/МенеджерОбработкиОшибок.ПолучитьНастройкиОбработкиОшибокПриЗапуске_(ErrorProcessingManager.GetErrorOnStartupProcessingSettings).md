МенеджерОбработкиОшибок (ErrorProcessingManager)

ПолучитьНастройкиОбработкиОшибокПриЗапуске (GetErrorOnStartupProcessingSettings)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ПолучитьНастройкиОбработкиОшибокПриЗапуске()

Возвращаемое значение:

Тип: [НастройкиОбработкиОшибокПриЗапуске](НастройкиОбработкиОшибокПриЗапуске_(ErrorOnStartupProcessingSettings).md).   

Описание:

Возвращает текущие настройки помощи при возникновении ошибки во время запуска.  
Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
