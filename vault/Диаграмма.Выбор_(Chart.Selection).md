Диаграмма (Chart)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<ЗначениеДиаграммы>, <СтандартнаяОбработка>)

Параметры:

<ЗначениеДиаграммы>

Тип: [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md), [ЗначениеДиаграммы](ЗначениеДиаграммы_(ChartValue).md).   
Значение диаграммы.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке мыши на диаграмме.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если диаграмма расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.

Использование в версии:

Доступен, начиная с версии 8.0.
