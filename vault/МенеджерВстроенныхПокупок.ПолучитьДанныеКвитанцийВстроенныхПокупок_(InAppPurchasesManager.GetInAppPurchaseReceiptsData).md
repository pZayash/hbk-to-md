МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьДанныеКвитанцийВстроенныхПокупок (GetInAppPurchaseReceiptsData)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьДанныеКвитанцийВстроенныхПокупок(<Квитанции>, <ИдентификаторыПокупок>)

Параметры:

<Квитанции> (обязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md).   
Квитанция или массив объектов типа [КвитанцияВстроеннойПокупки](КвитанцияВстроеннойПокупки_(InAppPurchaseReceipt).md), поля для которой или которых требуется получить.

<ИдентификаторыПокупок> (необязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md).   
При работе под управлением iOS параметр является обязательным.  
Используется для определения покупок, для которых требуется получить [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки_(InAppPurchaseReceiptData).md). Если задан неверный идентификатор, то [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки_(InAppPurchaseReceiptData).md) для него не будут созданы.  
При работе под управлением других ОС параметр игнорируется.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ДанныеКвитанцииВстроеннойПокупки](ДанныеКвитанцииВстроеннойПокупки_(InAppPurchaseReceiptData).md). Если переданная квитанция не содержит данных или данные невозможно прочесть, будет сгенерировано исключение.  
Важно! При работе под управлением iOS необходимо указать дополнительный параметр <ИдентификаторыПокупок>. При работе под управлением других ОС данный параметр игнорируется.

Описание:

Возвращает данные квитанций встроенных покупок.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.13.
