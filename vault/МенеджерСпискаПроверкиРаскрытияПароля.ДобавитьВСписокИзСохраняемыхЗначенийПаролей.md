МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

ДобавитьВСписокИзСохраняемыхЗначенийПаролей (AddToListFromPasswordsStoredValues)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ДобавитьВСписокИзСохраняемыхЗначенийПаролей(<СписокПроверкиРаскрытияПароля>)

Параметры:

<СписокПроверкиРаскрытияПароля> (обязательный)

Тип: [Массив](Массив.md), Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ФайловыйПоток](ФайловыйПоток.md), [ПотокВПамяти](ПотокВПамяти.md).   
Список проверки раскрытия пароля.  
Список может быть размещен:

- в массиве строк,
- в файле, содержащем хэш-значения паролей, вычисленные алгоритмом SHA1 и закодированные в base64, разделенные переносом строки (в этом случае указывается имя файла),
- передан в виде потока или представлен как [ДвоичныеДанные](ДвоичныеДанные.md).

Описание:

Добавляет в список проверки раскрытия пароля значения, удаляя дубликаты. Количество значений после добавления не может превышать 100000.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| СписокПроверкиРаскрытияПароля = Новый Массив(); СписокПроверкиРаскрытияПароля.Добавить(     "NEB8vB0WmKRCDOZWVeN9eA6bcOw="); СписокПроверкиРаскрытияПароля.Добавить(     "NvU34LBQ5TqfqRaWQM3WPvc8IMM="); СписокПроверкиРаскрытияПароля.Добавить(     "QwMDnmjxEwDwc/YMd6rPAMWqSYE="); ДополнительныеНастройкиАутентификации.СписокПроверкиРаскрытияПароля.     ДобавитьВСписокИзСохраняемыхЗначенийПаролей(     СписокПроверкиРаскрытияПароля); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
