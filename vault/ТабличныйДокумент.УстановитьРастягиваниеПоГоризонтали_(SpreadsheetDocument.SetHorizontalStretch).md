ТабличныйДокумент (SpreadsheetDocument)

УстановитьРастягиваниеПоГоризонтали (SetHorizontalStretch)

Доступен, начиная с версии 8.3.13.

Синтаксис:

УстановитьРастягиваниеПоГоризонтали(<Растягивать>)

Параметры:

<Растягивать> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Значение, которое будет использовано в свойстве [РастягиватьПоГоризонтали](ОбластьЯчеекТабличногоДокумента.РастягиватьПоГоризонтали_(SpreadsheetDocumentRange.HorizontalStretch).md) для всех колонок.   
Если передано значение [Истина](lang__def_BooleanTrue.md), то дополнительно для всех колонок в свойство [ВесовойКоэффициентШирины](ОбластьЯчеекТабличногоДокумента.ВесовойКоэффициентШирины_(SpreadsheetDocumentRange.WidthWeightFactor).md) устанавливается текущая ширина ячейки.

Описание:

Для всех колонок табличного документа изменяет значение свойства [РастягиватьПоГоризонтали](ОбластьЯчеекТабличногоДокумента.РастягиватьПоГоризонтали_(SpreadsheetDocumentRange.HorizontalStretch).md).

Доступность:

Сервер, толстый клиент.

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [ПолучитьКомпактныйДокумент](ТабличныйДокумент.ПолучитьКомпактныйДокумент_(SpreadsheetDocument.GetCompactDocument).md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
