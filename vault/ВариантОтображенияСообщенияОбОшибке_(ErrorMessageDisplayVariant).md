Описание:

Содержит варианты отображения сообщения об ошибке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.   
Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/managed-application/core. Имя типа XDTO: ErrorMessageDisplayVariant.

Доступен, начиная с версии 8.3.17.
