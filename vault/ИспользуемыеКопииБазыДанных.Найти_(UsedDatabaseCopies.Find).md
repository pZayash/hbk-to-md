ИспользуемыеКопииБазыДанных (UsedDatabaseCopies)

Найти (Find)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя копии базы данных.

Возвращаемое значение:

Тип: [ИспользуемаяКопияБазыДанных](ИспользуемаяКопияБазыДанных_(UsedDatabaseCopy).md).   

Описание:

Выполняет поиск используемой копии базы данных по имени копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
