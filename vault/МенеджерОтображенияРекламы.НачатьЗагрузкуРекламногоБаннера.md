МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуРекламногоБаннера (BeginAdBannerDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуРекламногоБаннера(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Ссылка на процедуру, которая будет вызвана после завершения загрузки рекламы из сети. Процедура должна иметь следующие параметры:

- Статус (Status) – тип [СтатусРекламы](СтатусРекламы.md). Показывает статус рекламного блока по результатам загрузки.
- Идентификатор (ID) – тип Строка. Рекламный идентификатор для которого закончилась загрузка.
- <ДополнительныеПараметры> – Значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Загружает из сети рекламный баннер. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [УстановитьОтображениеРекламногоБаннера](МенеджерОтображенияРекламы.УстановитьОтображениеРекламногоБаннера.md) рекламный баннер будет отображен без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы.md), метод [ЗагрузитьРекламныйБаннерАсинх](МенеджерОтображенияРекламы.ЗагрузитьРекламныйБаннерАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.18.
