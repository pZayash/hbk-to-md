ВстроеннаяПокупка (InAppPurchase)

Валюта (Currency)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Строка, содержащая международный код валюты в формате ISO 4217. Определяется платежным регионом, к которому привязана учетная запись пользователя устройства: если пользовать зарегистрирован в России, то в качестве валюты будет передано значение "RUB", даже если на устройстве используется язык интерфейса, отличный от русского.

Доступность:

Мобильный клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
