КритерииОтбораМенеджер (FilterCriteriaManager)

<Имя критерия отбора> (<Filter criteria name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КритерийОтбораМенеджер.\<Имя критерия\>](КритерийОтбораМенеджер.Имя_критерия_(FilterCriterionManager.Criterion_name).md).   
Набор свойств содержит менеджеры отдельных критериев отбора. Доступ к менеджеру осуществляется по имени. Имена свойств совпадают с именами критериев отбора, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
