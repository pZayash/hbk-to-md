РазделениеАутентификацииОбщегоРеквизита (CommonAttributeAuthenticationSeparation)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.2.

Описание:

Возможно однозначное определение пользователя информационной базы по его имени без указания значения этого разделителя. Имена пользователей информационной базы уникальны, независимо от значения этого разделителя.

Использование в версии:

Доступен, начиная с версии 8.2.
