ОписаниеОповещения (CallbackDescription)

По умолчанию

Доступен, начиная с версии 8.3.3.

Синтаксис:

Новый ОписаниеОповещения(<ИмяПроцедуры>, <Модуль>, <ДополнительныеПараметры>, <ИмяПроцедурыОбработкиОшибки>, <МодульОбработкиОшибки>)

Параметры:

<ИмяПроцедуры> (необязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Имя экспортируемой процедуры модуля, которая будет вызвана.

<Модуль> (необязательный)

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md), [ОбщийМодуль](ОбщийМодуль_(CommonModule).md), [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Программный модуль, процедура которого будет вызвана.  
При использовании типа [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) будет вызван метод модуля указанной формы клиентского приложения.  
При использовании типа [ОбщийМодуль](ОбщийМодуль_(CommonModule).md) будет вызван метод общего неглобального модуля.  
При использовании типа [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md) будет вызван метод модуля команды командного интерфейса.  
Объект данного типа может быть получен только из модуля команды.  
Для получения ссылки на него необходимо использовать свойство "ЭтотОбъект".  
Оно доступно в контексте модуля команды.

<ДополнительныеПараметры> (необязательный)

Тип: Произвольный.   
Значение любого типа, которое при вызове будет передано в указанную процедуру последним параметром.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ИмяПроцедурыОбработкиОшибки> (необязательный)

Имя экспортируемой процедуры модуля, которая будет вызвана в случае возникновения ошибки.

<МодульОбработкиОшибки> (необязательный)

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md), [ОбщийМодуль](ОбщийМодуль_(CommonModule).md).   
Программный модуль, процедура которого будет вызвана в случае возникновения ошибки.  
При использовании типа [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) будет вызван метод модуля указанной формы клиентского приложения.  
При использовании типа [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md) будет вызван метод модуля команды командного интерфейса.  
При использовании типа [ОбщийМодуль](ОбщийМодуль_(CommonModule).md) будет вызван метод общего неглобального модуля.

Описание:

Создает объект и связывает его с вызовом указанной процедуры модуля.

Использование в версии:

Доступен, начиная с версии 8.3.3.
