Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаНавигационнойСсылке (URLClick)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НажатиеНаНавигационнойСсылке(<Источник>, <НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<Источник>

Тип: [ЭлементПланировщика](ЭлементПланировщика.md), [ИзмерениеПланировщика](ИзмерениеПланировщика.md), [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика.md), [МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика.md), [МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени.md).   
Объект, содержащий форматированную строку, на гиперссылку которой было произведено нажатие.

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: Строка.   
Значение гиперссылки форматированной строки.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.

Описание:

Возникает при нажатии левой кнопкой мыши на навигационной ссылке форматированной строки элемента измерения планировщика (если свойство [ГиперссылкаЭлементаИзмерения](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаИзмерения.md) = Ложь), метки интервала фона планировщика, метки элемента шкалы времени (если свойство [ГиперссылкаЭлементаШкалыВремени](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаШкалыВремени.md) = Ложь), элемента планировщика и измерения планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
