АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

НедоступныеДляРасширенияМодули (ModulesNotAvailableForExtension)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Используется, когда расширение всех модулей разрешено.  
Содержит список полных имен объектов конфигурации или модулей, расширение которых не разрешено, разделенных ";".  
Указание полного имени объекта конфигурации запрещает расширение всех модулей объекта.  
Указание полного имени модуля запрещает расширение конкретного модуля.  
Примеры имен:   
ОбщийМодуль.ОбщийМодуль1,  
Справочник.Товары.Форма.ФормаЭлемента,  
Справочник.Контрагенты.МодульОбъекта,  
Конфигурация.ДемоКонфигурация.МодульСеанса.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
