Расширение тестируемого поля формы для поля надписи (Tested form field extension for a label field)

ПолучитьПредставленияГиперссылокФорматированнойСтроки (GetFormattedStringHyperlinksPresentations)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПолучитьПредставленияГиперссылокФорматированнойСтроки()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив.md).   
Фиксированный массив элементов типа Строка.  
Если форматированная строка не содержит гиперссылок, возвращает пустой массив.  
Если элемент управления не содержит форматированной строки, генерируется исключение.

Описание:

Метод возвращает массив значений с представлениями ссылок в форматированном документе.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
