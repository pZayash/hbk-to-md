КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Добавить(<Значение>, <Текст>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение элемента измерения. Значение должно быть уникальным в пределах коллекции.

<Текст> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Текст, отображаемый в заголовке элемента измерения.  
Рекомендуется заполнить, так как это позволит уменьшить количество обращений к серверу для получения представлений.

Возвращаемое значение:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика.md).   
Добавленный элемент.

Описание:

Добавляет элемент измерения в конец коллекции элементов измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
