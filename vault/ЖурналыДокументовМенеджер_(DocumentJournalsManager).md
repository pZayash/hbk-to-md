# ЖурналыДокументовМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Журналы документов](Журналы_документов.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЖурналДокументовМенеджер.\<Имя журнала документов\>](ЖурналДокументовМенеджер.Имя_журнала_документов_(DocumentJournalManager.Document_journal_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры журналов документов.  
Возможно обращение к менеджеру журнала документов посредством оператора [...]. В качестве аргумента передается имя журнала документов, как оно задано в конфигурации.  

Свойства:

[\<Имя журнала документов\> (\<Document journal name\>)](ЖурналыДокументовМенеджер.Имя_журнала_документов_(DocumentJournalsManager.Document_journal_name).md)  

Описание:

Предназначен для управления журналами документов и предоставляет доступ к значениям типа [ЖурналДокументовМенеджер.](ЖурналДокументовМенеджер.Имя_журнала_документов_(DocumentJournalManager.Document_journal_name).md)<[Имя журнала документов](ЖурналДокументовМенеджер.Имя_журнала_документов_(DocumentJournalManager.Document_journal_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [ЖурналыДокументов](Глобальный_контекст.ЖурналыДокументов_(Global_context.DocumentJournals).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| // менеджер журналов документов предоставляет доступ к   // менеджеру журнала Реализация.  Выборка = ЖурналыДокументов.Реализация.Выбрать(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ЖурналыДокументов](Глобальный_контекст.ЖурналыДокументов_(Global_context.DocumentJournals).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
