КлиентскоеПриложение (ClientApplication)

УстановитьРежимОсновногоОкна (SetMainWindowMode)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьРежимОсновногоОкна(<РежимОсновногоОкна>)

Параметры:

<РежимОсновногоОкна> (обязательный)

Тип: [РежимОсновногоОкнаКлиентскогоПриложения](РежимОсновногоОкнаКлиентскогоПриложения_(MainClientApplicationWindowMode).md).   
Устанавливаемый режим основного окна клиентского приложения.

Описание:

Устанавливает режим основного окна клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

- Недоступен в режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md).
- Метод можно вызывать только внутри обработчика события [ПередНачаломРаботыСистемы](Глобальный_контекст.ПередНачаломРаботыСистемы_(Global_context.BeforeStart).md) управляемого приложения.

См. также:

[КлиентскоеПриложение](КлиентскоеПриложение_(ClientApplication).md), метод [ПолучитьРежимОсновногоОкна](КлиентскоеПриложение.ПолучитьРежимОсновногоОкна_(ClientApplication.GetMainWindowMode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
