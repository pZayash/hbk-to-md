ПолеКартинки (PictureBox)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что поле картинки является гиперссылкой. Если значение свойства Истина, то при щелчке мышью в поле картинки возникает событие [Нажатие](ПолеКартинки.Нажатие.md).

Доступность:

Толстый клиент.

См. также:

[ПолеКартинки](ПолеКартинки.md), событие [Нажатие](ПолеКартинки.Нажатие.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
