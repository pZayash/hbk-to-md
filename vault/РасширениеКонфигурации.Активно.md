РасширениеКонфигурации (ConfigurationExtension)

Активно (Active)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если данное свойство установлено в значение Ложь, то при следующем запуске расширение не будет подключено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.12.
