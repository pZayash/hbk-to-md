ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

ПоддерживаетсяЗапуск (RunningSupported)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоддерживаетсяЗапуск()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

- [Истина](lang__def_BooleanTrue.md) - для Android,
- [Ложь](lang__def_BooleanFalse.md) - для остальных операционных систем.

Описание:

Определяет, поддерживается ли запуск приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
