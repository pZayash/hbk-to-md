РазмерКартинки (PictureSize)

АвтоРазмерБезУчетаМасштаба (AutoSizeIgnoreScale)

Доступен, начиная с версии 8.3.10.

Описание:

Если предоставленного для картинки места достаточно, используется режим [РеальныйРазмер](РазмерКартинки.РеальныйРазмер_(PictureSize.RealSize).md), в противном случае используется режим [Пропорционально](РазмерКартинки.Пропорционально_(PictureSize.Proportionally).md).  
При определении размера картинки не учитывается масштаб формы.

Использование в версии:

Доступен, начиная с версии 8.3.10.
