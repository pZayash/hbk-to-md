Глобальный контекст (Global context)

ПоддерживаетсяОтображениеКарты (MapRepresentationSupported)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПоддерживаетсяОтображениеКарты()

Возвращаемое значение:

Тип: Булево.   

- Истина - в ОС Android, iOS, Windows Phone, Windows 10,
- Ложь - Windows 8 (для планшетов).

Описание:

Проверяет возможность работы с картой.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПоказатьНаКарте](Глобальный_контекст.ПоказатьНаКарте.md)  

Использование в версии:

Доступен, начиная с версии 8.3.11.
