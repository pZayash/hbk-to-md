Расширение таблицы формы для доступных полей компоновки данных (Form table extension for data composition available fields)

ОграниченияИспользования (UseRestrictions)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОграниченияИспользованияДоступныхПолейКомпоновкиДанных](ОграниченияИспользованияДоступныхПолейКомпоновкиДанных_(DataCompositionAvailableFieldUseRestrictions).md).   
Содержит коллекцию ограничений для использования элементов в данной таблице.  
С помощью ограничений разработчик может исключить часть элементов этой коллекции, показ которых для пользователя нецелесообразен.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
