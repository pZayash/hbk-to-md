ИзменениеСпособаОтображенияОкна (WindowAppearanceModeChange)

Авто (Auto)

Доступен, начиная с версии 8.0.

Описание:

Изменение способа отображения зависит от состояния окна.  
Если у формы установлено состояние окна [Обычное](ВариантСостоянияОкна.Обычное_(WindowStateVariant.Normal).md), то изменять способ отображения можно, если установлено состояние окна [Свободное](ВариантСостоянияОкна.Свободное_(WindowStateVariant.Float).md), то нельзя.

Использование в версии:

Доступен, начиная с версии 8.0.
