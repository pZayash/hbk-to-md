Описание:

Содержит данные, необходимые для получения программной лицензии на платформу.  
Может быть сохранен в настройках.  
Может использоваться в методе [ЗаполнитьЗначенияСвойств](Глобальный_контекст.ЗаполнитьЗначенияСвойств.md) в качестве источника и в качестве приемника.

Доступность:

Тонкий клиент, сервер, внешнее соединение.   
Сериализуется.
