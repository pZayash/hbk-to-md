ТестируемоеПриложение (TestedApplication)

ПолучитьМаксимальноеВремяВыполненияДействия (GetMaxActionExecutionTime)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьМаксимальноеВремяВыполненияДействия()

Возвращаемое значение:

Если полученное число равно 0, менеджер тестирования ожидает завершения действия до тех пор, пока оно само не закончит работу.

Описание:

Получает количество секунд, в течение которых действие должно успеть завершить свою работу на стороне клиента тестирования.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
