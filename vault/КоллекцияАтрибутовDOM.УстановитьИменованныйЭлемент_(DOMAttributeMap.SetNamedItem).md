КоллекцияАтрибутовDOM (DOMAttributeMap)

УстановитьИменованныйЭлемент (SetNamedItem)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьИменованныйЭлемент(<Атрибут>)

Параметры:

<Атрибут> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   
Атрибут DOM.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   

Описание:

Устанавливает в коллекцию атрибут DOM, используя в качестве ключа локальное имя и URI пространства имен узла атрибута.  
Если в коллекции присутствует атрибут с таким же URI пространства имен и локальным именем, то он будет замещен и возвращен в качестве возвращаемого значения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
