ЭлементОтбора (FilterItem)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение, по которому будет осуществляться фильтрация.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если [ВидСравнения](ЭлементОтбора.ВидСравнения.md) принимает значение [ВСписке](ВидСравнения.ВСписке.md), [ВСпискеПоИерархии](ВидСравнения.ВСпискеПоИерархии.md), [НеВСписке](ВидСравнения.НеВСписке.md) или [НеВСпискеПоИерархии](ВидСравнения.НеВСпискеПоИерархии.md), то тип значения [СписокЗначений](СписокЗначений.md).

Использование в версии:

Доступен, начиная с версии 8.0.
