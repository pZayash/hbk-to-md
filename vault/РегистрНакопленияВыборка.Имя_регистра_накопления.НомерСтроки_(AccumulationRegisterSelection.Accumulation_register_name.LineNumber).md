РегистрНакопленияВыборка.<Имя регистра накопления> (AccumulationRegisterSelection.<Accumulation register name>)

НомерСтроки (LineNumber)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрНакопленияВыборка.Имя_регистра_накопления.Регистратор_(AccumulationRegisterSelection.Accumulation_register_name.Recorder).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
