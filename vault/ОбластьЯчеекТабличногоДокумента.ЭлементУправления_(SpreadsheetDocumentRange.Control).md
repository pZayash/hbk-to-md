ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ЭлементУправления (Control)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Содержит элемент управления ячеек области.

Доступность:

Толстый клиент.

Примечание:

1. Свойство доступно, только если ячейка содержит значение (свойство СодержитЗначение равно [Истина](lang__def_BooleanTrue.md)).  
2. Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.2.
