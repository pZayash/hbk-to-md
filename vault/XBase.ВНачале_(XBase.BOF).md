XBase (XBase)

ВНачале (BOF)

Доступен, начиная с версии 8.0.

Синтаксис:

ВНачале()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - указатель в файле находится перед первой записью; [Ложь](lang__def_BooleanFalse.md) - начало файла не достигнуто.

Описание:

Получает признак достижения указателем текущей записи начала файла.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Среднее значение поля COST  xB.Последняя(); Стоимость = 0; КоличествоЗаписей = 0; Пока Не xB.ВНачале() Цикл     Стоимость = Стоимость + xB.COST;     КоличествоЗаписей = КоличествоЗаписей + 1;     xB.Предыдущая(); КонецЦикла; Если КоличествоЗаписей > 0 Тогда     Стоимость = Стоимость / КоличествоЗаписей; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
