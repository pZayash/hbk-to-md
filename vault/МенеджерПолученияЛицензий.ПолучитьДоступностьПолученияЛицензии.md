МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьДоступностьПолученияЛицензии (GetLicenseAcquisitionAvailability)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьДоступностьПолученияЛицензии(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md).   
Используются только значения свойств: [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента.md), [Порт](ЗапросНаПолучениеЛицензии.Порт.md), [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей.md).   
Если [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента.md) - пустая строка, то проверку выполняет текущее приложение на текущем компьютере, иначе проверку на своем компьютере выполняет приложение, адрес которого указан параметрами [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента.md) и [Порт](ЗапросНаПолучениеЛицензии.Порт.md).   
Если [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей.md) - Истина, то проверяется возможность записи файла лицензии в подкаталог лицензий каталога данных приложений всех пользователей, иначе - в подкаталог лицензий каталога данных приложений текущего пользователя.

Возвращаемое значение:

Тип: [ДоступностьПолученияЛицензий](ДоступностьПолученияЛицензий.md).   

Описание:

Проверяет возможность получения лицензии для всех пользователей или для текущего пользователя заданного компьютера.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
