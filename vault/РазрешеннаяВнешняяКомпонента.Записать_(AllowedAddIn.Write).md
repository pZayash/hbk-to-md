РазрешеннаяВнешняяКомпонента (AllowedAddIn)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию новой разрешенной внешней компоненты или обновляет существующую.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
