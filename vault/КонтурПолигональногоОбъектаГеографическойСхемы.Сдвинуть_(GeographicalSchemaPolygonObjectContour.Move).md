КонтурПолигональногоОбъектаГеографическойСхемы (GeographicalSchemaPolygonObjectContour)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс объекта в коллекции, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение, на которое необходимо сдвинуть объект.

Описание:

Сдвигает объект из указанного индекса на указанное смещение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
