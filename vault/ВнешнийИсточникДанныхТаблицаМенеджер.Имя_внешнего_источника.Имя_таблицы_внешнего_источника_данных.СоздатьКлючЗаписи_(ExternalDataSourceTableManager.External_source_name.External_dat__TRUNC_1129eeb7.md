ВнешнийИсточникДанныхТаблицаМенеджер.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableManager.<External source name>.<External data source table name>)

СоздатьКлючЗаписи (CreateRecordKey)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СоздатьКлючЗаписи(<ЗначенияКлюча>)

Параметры:

<ЗначенияКлюча> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая значения для заполнения свойств ключа записи.  
Имена элементов структуры должны соответствовать именам ключевых полей.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаКлючЗаписи.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecordKey.External_source_name.External_data_source_table_name).md).   

Описание:

Создает ключ записи по переданной структуре.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных [НеобъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.НеобъектныеДанные_(ExternalDataSourceTableDataType.NonobjectData).md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
