ОбъектМетаданных: РегистрРасчета (MetadataObject: CalculationRegister)

Периодичность (Periodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПериодичностьРегистраРасчета](ПериодичностьРегистраРасчета.md).   
Определяет вид периода регистрации записей (например, [Год](ПериодичностьРегистраРасчета.Год.md), [Квартал](ПериодичностьРегистраРасчета.Квартал.md), [Месяц](ПериодичностьРегистраРасчета.Месяц.md), [День](ПериодичностьРегистраРасчета.День.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
