ОписаниеПодписиPDF (PDFSignatureDescription)

ТипПодписи (SignatureType)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Содержит тип подписи – утверждающая или сертифицирующая.   
По умолчанию: [Утверждающая](ТипПодписиPDF.Утверждающая_(PDFSignatureType.Approving).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
