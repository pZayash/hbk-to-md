СтраницаСканированияДокументов (DocumentScanningPage)

ПараметрыКачества (QualityParameters)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыКачестваСканированияДокументов](ПараметрыКачестваСканированияДокументов_(DocumentScanningQualityParameters).md).   
Содержит результат выполнения методов [ПоказатьСканированиеДокументов](СредстваМультимедиа.ПоказатьСканированиеДокументов_(MultimediaTools.ShowDocumentScanning).md) и [СканироватьДокументыАсинх](СредстваМультимедиа.СканироватьДокументыАсинх_(MultimediaTools.ScanDocumentsAsync).md), предназначенный для контроля уровня качества полученного документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
