Расширение табличного поля списка справочника (Catalog list table box extension)

СпособРедактирования (EditType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СпособРедактированияСписка](СпособРедактированияСписка.md).   
Предоставляет доступ к управлению способом редактирования элементов справочника (в диалоге или в списке).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
