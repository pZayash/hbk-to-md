Надпись (Label)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак выделения текста надписи, первый значимый символ которого "-", специальным цветом стиля [ЦветОтрицательногоЧисла](ЦветаСтиля.ЦветОтрицательногоЧисла_(StyleColors.NegativeTextColor).md).

Доступность:

Толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - текст выделяется.

Использование в версии:

Доступен, начиная с версии 8.0.
