КнопкаФормы (FormButton)

УникальностьКоманды (CommandUniqueness)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак поддержки уникальности кнопок по командам внутри автоматически заполняемой группы формы:

- [Истина](lang__def_BooleanTrue.md) - уникальность кнопок поддерживается. Для каждой команды будет отображаться только одна кнопка. Кнопки, добавленные пользователем, имеют приоритет;
- [Ложь](lang__def_BooleanFalse.md) - уникальность кнопок не поддерживается. В группе может быть несколько кнопок, связанных с одной и той же командой.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.15.
