Свойства:

[ИзменятьПорядокСтрок (ChangeRowOrder)](Расширение_табличного_поля_списка_значений.ИзменятьПорядокСтрок.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md), предназначенные для интерактивной работы со списком значений. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) имеет значение типа [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
