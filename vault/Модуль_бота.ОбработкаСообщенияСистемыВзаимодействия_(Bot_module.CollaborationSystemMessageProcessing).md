Модуль бота (Bot module)

ОбработкаСообщенияСистемыВзаимодействия (CollaborationSystemMessageProcessing)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОбработкаСообщенияСистемыВзаимодействия(<Сообщение>, <ДополнительныеПараметры>)

Параметры:

<Сообщение>

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).   
Полученное сообщение.  
Вызов метода [Записать](СообщениеСистемыВзаимодействия.Записать_(CollaborationSystemMessage.Write).md) у этого сообщения вызывает исключение.

<ДополнительныеПараметры>

Тип: Произвольный.   
Параметры, заданные при создании бота.

Описание:

Обработчик, вызываемый при получении сообщения системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
