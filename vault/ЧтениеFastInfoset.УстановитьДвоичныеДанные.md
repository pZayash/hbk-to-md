ЧтениеFastInfoset (FastInfosetReader)

УстановитьДвоичныеДанные (SetBinaryData)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьДвоичныеДанные(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные, из которых необходимо производить чтение.

Описание:

Открывает двоичные данные для чтения данным объектом.   
Если перед вызовом данного метода уже производилось чтение данных из другого файла или двоичных данных, то чтение прекращается и объект инициализируется для чтения из указанных двоичных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
