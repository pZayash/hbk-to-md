ПеречислениеМенеджер.<Имя перечисления> (EnumManager.<Enumeration name>)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: Строка, [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет.md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент.md), [ТекстовыйДокумент](ТекстовыйДокумент.md); другой объект, который может быть макетом.   

Описание:

Получает макет перечисления.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
