ОбновлениеКонфигурацииБазыДанных (DataBaseConfigurationUpdate)

ИнформацияОВыполнении (ExecutionInformation)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит информацию об объектах, обновляемых в процессе фонового обновления конфигурации базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Элементом массива является объект типа [ЭлементИнформацииОВыполненииОбновленияКонфигурацииБазыДанных](ЭлементИнформацииОВыполненииОбновленияКонфигурацииБазыДанных_(DataBaseConfigurationUpdateExecutionInformationItem).md).

Использование в версии:

Доступен, начиная с версии 8.3.1.
