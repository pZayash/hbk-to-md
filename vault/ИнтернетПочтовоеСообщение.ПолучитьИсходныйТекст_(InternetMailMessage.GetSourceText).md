ИнтернетПочтовоеСообщение (InternetMailMessage)

ПолучитьИсходныйТекст (GetSourceText)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИсходныйТекст(<Кодировка>)

Параметры:

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка, которая будет использована для перекодирования исходных данных в текст. Если параметр не указан, то используется кодировка UTF-8.  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

.  
Значение по умолчанию: "UTF-8".

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает исходный текст почтового сообщения.  
Если сообщение принято с сервера (с помощью метода [Выбрать](ИнтернетПочта.Выбрать_(InternetMail.Get).md) объекта [ИнтернетПочта](ИнтернетПочта_(InternetMail).md)) и не было изменено, то возвращаемый методом текст соответствуют тексту, полученному с сервера  
Если объект [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md) создан программно или получен с сервера и был изменен, то текст генерируется на основе содержимого этого объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
