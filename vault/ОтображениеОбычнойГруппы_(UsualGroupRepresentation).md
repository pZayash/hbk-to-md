Описание:

Содержит варианты отображения группы формы клиентского приложения вида [ОбычнаяГруппа](ВидГруппыФормы.ОбычнаяГруппа_(FormGroupType.UsualGroup).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Расширение группы формы для обычной группы](Расширение_группы_формы_для_обычной_группы_(Form_group_extension_for_a_usual_group).md), свойство [Отображение](Расширение_группы_формы_для_обычной_группы.Отображение_(Form_group_extension_for_a_usual_group.Representation).md)  

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.27.
