ОбъектМетаданных: ПризнакУчетаПланаСчетов (MetadataObject: AccountingFlag)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Истина - при выводе признака учета плана счетов с отрицательным значением значение будет отображено особым образом.  
Имеет смысл только для признаков типа Число.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
