ЗаписьУзловDOM (DOMNodeWriter)

НайтиПрефикс (LookupPrefix)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПрефикс(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен, для которого необходимо найти префикс.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если префикс не обнаружен (соответствие URI пространства имен не было записано), возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск префикса для указанного URI пространства имен.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Префикс должен быть поставлен в соответствие указанному пространству имен при помощи методов [ЗаписьУзловDOM.ЗаписатьСоответствиеПространстваИмен](ЗаписьУзловDOM.ЗаписатьСоответствиеПространстваИмен_(DOMNodeWriter.WriteNamespaceMapping).md) или [ЗаписьУзловDOM.ЗаписатьАтрибут](ЗаписьУзловDOM.ЗаписатьАтрибут_(DOMNodeWriter.WriteAttribute).md).

Использование в версии:

Доступен, начиная с версии 8.1.
