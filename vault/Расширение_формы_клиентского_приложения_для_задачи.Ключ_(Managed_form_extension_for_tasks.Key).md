Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

Ключ (Key)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Ключ объекта.   
В случае отсутствия или неверного значения создается новый объект с использованием остальных параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
