ВложениеСистемыВзаимодействия (CollaborationSystemAttachment)

СодержимоеУдалено (ContentDeleted)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что содержимое вложения было удалено из файлового хранилища по причине превышения лимитов хранений по срокам или объемам.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
