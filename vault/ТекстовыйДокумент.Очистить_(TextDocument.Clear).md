ТекстовыйДокумент (TextDocument)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Очищает содержимое текстового документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ТекстДок.Очистить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
