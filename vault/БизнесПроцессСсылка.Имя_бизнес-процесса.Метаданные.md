БизнесПроцессСсылка.<Имя бизнес-процесса> (BusinessProcessRef.<Business process name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: БизнесПроцесс](ОбъектМетаданных_БизнесПроцесс.md).   

Описание:

Предоставляет доступ к объекту описания метаданных бизнес-процесса. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.БизнесПроцессы.ВыпискаСчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для каждого Реквизит Из БизнесПроцесс.Метаданные().Реквизиты Цикл     Сообщить(Реквизит.Представление()); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
