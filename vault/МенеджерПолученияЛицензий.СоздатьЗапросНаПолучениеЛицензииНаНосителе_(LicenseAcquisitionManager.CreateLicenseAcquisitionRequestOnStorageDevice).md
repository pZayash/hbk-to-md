# МенеджерПолученияЛицензий.СоздатьЗапросНаПолучениеЛицензииНаНосителе

МенеджерПолученияЛицензий (LicenseAcquisitionManager)

СоздатьЗапросНаПолучениеЛицензииНаНосителе (CreateLicenseAcquisitionRequestOnStorageDevice)

Доступен, начиная с версии 8.3.20.

Синтаксис:

СоздатьЗапросНаПолучениеЛицензииНаНосителе(<ЗапросНаПолучениеЛицензии>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии_(LicenseAcquisitionRequest).md).   
Данные для запроса на получение лицензии.   
Свойства [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md), [Порт](ЗапросНаПолучениеЛицензии.Порт_(LicenseAcquisitionRequest.Port).md), [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей_(LicenseAcquisitionRequest.ForAllUsers).md), [ИмяФайлаЛицензии](ЗапросНаПолучениеЛицензии.ИмяФайлаЛицензии_(LicenseAcquisitionRequest.LicenseFileName).md) не используются.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Последовательность символов, которую требуется передать в центр лицензирования на носителе или по электронной почте для получения лицензии. Строка может содержать только символы с кодами, не превышающими 127, и может быть записана в файл при помощи объекта [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md) в кодировке ANSI.

Описание:

По данным для получения лицензии и параметрам компьютера создает последовательность цифр, которую требуется передать в центр лицензирования для получения лицензии на носителе.  
Использует параметры компьютера, который задан адресом агента сервера 1С:Предприятия, или текущего компьютера.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
