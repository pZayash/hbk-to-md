СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ДатаОтправки (SendDate)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата и время, когда сообщение было отправлено. Устанавливается при отправке сообщения.  
Хранится и передается в формате UTC, при получении преобразуется в время в часовом поясе сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
