# РежимИзмененияСвязанногоЗначения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[НеИзменять (DontChange)](РежимИзмененияСвязанногоЗначения.НеИзменять_(LinkedValueChangeMode.DontChange).md)  
[Очищать (Clear)](РежимИзмененияСвязанногоЗначения.Очищать_(LinkedValueChangeMode.Clear).md)  

Описание:

Содержит варианты изменения связанного значения при изменении ведущего значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise}. Имя типа XDTO: LinkedValueChangeMode.
