ОписаниеПалитрыЦветовДиаграммы (ChartColorPaletteDescription)

ПолучитьПалитру (GetPalette)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьПалитру()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Элементами массива являются значения типа [Цвет](Цвет_(Color).md).

Описание:

Возвращает массив цветов текущей палитры.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
