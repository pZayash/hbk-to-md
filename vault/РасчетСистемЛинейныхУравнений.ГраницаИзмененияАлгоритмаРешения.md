РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ГраницаИзмененияАлгоритмаРешения (AlgorithmSelectionBorder)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Число, начиная с которого почти полный подграф считается существенным и обрабатывается алгоритмом для полного графа (а не для разреженных).   
Значение по умолчанию – 10000.   
Корректное значение – любое целое число. Не рекомендуется выставлять слишком маленьким или слишком большим – решение замедлится. Оптимальное значение от 5000 до 100000 (зависит от конкретных данных).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
