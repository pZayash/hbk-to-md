ЭлементПлавающийФреймHTML (HTMLIFrameElement)

ДокументВладелец (OwnerDocument)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ДокументHTML](ДокументHTML_(HTMLDocument).md), [Неопределено](lang__def_Undefined.md).   
Содержит документ HTML, которому принадлежит данный узел HTML.   
Необходим для возможности создания новых узлов.   
Если данный узел - Document или DocumentType, которыми не владеет пока ни один документ, то содержит [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
