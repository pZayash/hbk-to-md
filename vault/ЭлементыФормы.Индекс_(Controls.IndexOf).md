ЭлементыФормы (Controls)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ЭлементФормы>)

Параметры:

<ЭлементФормы> (обязательный)

Элемент управления формы, индекс которого следует получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс переданного элемента формы в коллекции элементов формы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
