Модуль бота (Bot module)

ПриДобавленииВОбсуждениеСистемыВзаимодействия (OnAddToCollaborationSystemConversation)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПриДобавленииВОбсуждениеСистемыВзаимодействия(<ИдентификаторОбсуждения>, <ДополнительныеПараметры>)

Параметры:

<ИдентификаторОбсуждения>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения, в которое добавляется бот.

<ДополнительныеПараметры>

Тип: Произвольный.   
Параметры, заданные при создании бота.

Описание:

Обработчик, вызываемый при добавлении бота в имеющееся обсуждение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
