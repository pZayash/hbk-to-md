ИзвлечениеТекста (TextExtraction)

По имени файла

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ИзвлечениеТекста(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: Строка.   
Полное имя файла, из которого будет извлекаться текст.

Описание:

Используется для создания объекта ИзвлечениеТекста.

Использование в версии:

Доступен, начиная с версии 8.1.
