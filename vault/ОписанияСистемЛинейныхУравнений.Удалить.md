ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Вариант синтаксиса: По элементу

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md).   
Удаляемый элемент.

Описание варианта метода:

Удаляет переданный элемент из коллекции.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Индекс удаляемого элемента коллекции.

Описание:

Удаляет элемент из коллекции по индексу.   
Индексация начинается с 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
