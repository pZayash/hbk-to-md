МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПроверитьПриобретение (CheckPurchased)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьПриобретение(<Покупка>)

Параметры:

<Покупка> (обязательный)

Тип: [ВстроеннаяПокупка](ВстроеннаяПокупка.md), Строка.   
Покупка, приобретение которой необходимо проверить.  
В качестве параметра можно указать идентификатор покупки в виде строки или объект [ВстроеннаяПокупка](ВстроеннаяПокупка.md), у которого будет запрошен идентификатор.

Возвращаемое значение:

Тип: Булево.   

Описание:

Возвращает информацию, приобретена ли указанная покупка. Поведение метода зависит от типа покупки (постоянная, подписка или расходуемая) и от используемой ОС:

- Для постоянных покупок и подписок возвращает информацию о том, приобретена ли указанная покупка (активна ли подписка). Работает одинаково на iOS и Android.
- Для расходуемых покупок:
  - На Android - возвращает значение Истина, если покупка была приобретена и не была израсходована.
  - На iOS - возвращает значение Ложь. Данные для выполнения данного метода получаются из истории покупок, хранящейся локально, поэтому для актуализации данных следует использовать метод [ОбновитьИнформациюОПриобретении](МенеджерВстроенныхПокупок.ОбновитьИнформациюОПриобретении.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).
