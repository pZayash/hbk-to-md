ПолитикаПаролейПользователей (UserPasswordPolicy)

ОграничениеПовторенияПаролейСредиПоследних (PasswordReuseLimit)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Численное значение ограничения повторения паролей пользователей среди последних.  
  
Значение должно быть в интервале от 0 до 99.  
Значение по умолчанию: 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
