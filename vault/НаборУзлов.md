Описание:

Представляет собой значение реквизита [НаборУзлов](НаборУзлов.md) объекта типа [ПараметрыОбменаДанными](ПараметрыОбменаДанными.md), который, в свою очередь является значением свойства ОбменДанными, объектов, представляющих данные, по которым ведется учет изменений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
