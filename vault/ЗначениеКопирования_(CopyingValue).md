Расширение формы клиентского приложения для записи таблицы внешнего источника данных

ЗначениеКопирования (CopyingValue)

Доступен, начиная с версии 8.3.5.

Описание:

Тип: [ВнешнийИсточникДанныхТаблицаКлючЗаписи.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecordKey.External_source_name.External_data_source_table_name).md).   
Запись таблицы внешнего источника данных, копия которой должна быть создана.

Использование в версии:

Доступен, начиная с версии 8.3.5.
