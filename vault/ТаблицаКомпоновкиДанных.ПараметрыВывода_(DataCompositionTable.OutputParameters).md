ТаблицаКомпоновкиДанных (DataCompositionTable)

ПараметрыВывода (OutputParameters)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ЗначенияПараметровВыводаТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаТаблицыКомпоновкиДанных_(DataCompositionTableOutputParameterValues).md).   
Содержит параметры вывода, применяемые ко всей таблице.   
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
