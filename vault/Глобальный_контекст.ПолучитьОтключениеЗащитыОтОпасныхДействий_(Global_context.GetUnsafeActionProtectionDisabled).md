Глобальный контекст (Global context)

ПолучитьОтключениеЗащитыОтОпасныхДействий (GetUnsafeActionProtectionDisabled)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьОтключениеЗащитыОтОпасныхДействий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - защита от опасных действий отключена в обработчике,  
[Ложь](lang__def_BooleanFalse.md) - защита от опасных действий включена.

Описание:

Возвращает состояние защиты от опасных действий для текущего метода. Состояние может быть изменено вызовом [УстановитьОтключениеЗащитыОтОпасныхДействий](Глобальный_контекст.УстановитьОтключениеЗащитыОтОпасныхДействий_(Global_context.SetUnsafeActionProtectionDisabled).md) непосредственно в коде метода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер, интеграция.  
Вызов метода выполняет обращение к серверу.

Пример:

|  |
| --- |
| // включение защиты от опасных действий в потенциально опасных разработчиках  Процедура ПриЗаписи(Отказ)     Если ПолучитьОтключениеЗащитыОтОпасныхДействий() Тогда         УстановитьОтключениеЗащитыОтОпасныхДействий(Ложь);     КонецЕсли;     // здесь состояние защиты от опасных действий будет как до вызова обработчика  КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.3.27.
