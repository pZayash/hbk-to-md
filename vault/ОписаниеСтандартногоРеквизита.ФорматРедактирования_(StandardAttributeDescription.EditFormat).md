ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

ФорматРедактирования (EditFormat)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит форматную строку. Используется при редактировании текста.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
