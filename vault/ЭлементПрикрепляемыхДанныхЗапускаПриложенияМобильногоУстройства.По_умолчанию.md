ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipDataItem)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства(<Текст>, <HTMLТекст>, <ДанныеЗапуска>, <СсылкаНаНеизменныйURI>)

Параметры:

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Простой текст.

<HTMLТекст> (необязательный)

Тип: [Строка](lang__def_String.md).   
HTML-текст. Не может превышать 800 Кбайт. Если нужно передать больше, то следует записать текст в файл и передать через параметр СсылкаНаНеизменныйURI (ImmutableURIRef).

<ДанныеЗапуска> (необязательный)

Тип: [ЗапускПриложенияМобильногоУстройства](ЗапускПриложенияМобильногоУстройства_(MobileDeviceApplicationRun).md).   
Данные запуска.

<СсылкаНаНеизменныйURI> (необязательный)

Тип: [Строка](lang__def_String.md), [Файл](Файл_(File).md).   
Ссылка на ресурсы.

Описание:

Создает элемент прикрепляемых данных.

Использование в версии:

Доступен, начиная с версии 8.3.23.
