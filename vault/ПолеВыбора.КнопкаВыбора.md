ПолеВыбора (ComboBox)

КнопкаВыбора (ChoiceButton)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак наличия/отсутствия кнопки выбора значения в поле.  
Истина - кнопка есть.

Доступность:

Толстый клиент.

Примечание:

Стандартной обработки нажатия на кнопку выбора не предусмотрено. Необходимые действия можно провести в обработчике события [НачалоВыбора](ПолеВыбора.НачалоВыбора.md).

Использование в версии:

Доступен, начиная с версии 8.0.
