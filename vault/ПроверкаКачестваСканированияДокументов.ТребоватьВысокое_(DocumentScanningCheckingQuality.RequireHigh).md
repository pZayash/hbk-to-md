ПроверкаКачестваСканированияДокументов (DocumentScanningCheckingQuality)

ТребоватьВысокое (RequireHigh)

Доступен, начиная с версии 8.3.23.

Описание:

Пользователь не может принять документ, чье обобщенное качество ниже Высокого (меньше 7).   
Пользователю показываются предупреждения о качестве документа.

Использование в версии:

Доступен, начиная с версии 8.3.23.
