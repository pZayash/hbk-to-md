ЗаписьXML (XMLWriter)

ЗаписатьТипДокумента (WriteDocumentType)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Имя и определение

Синтаксис:

ЗаписатьТипДокумента(<Имя>, <ВнутреннееПодмножество>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя документа.

<ВнутреннееПодмножество> (обязательный)

Тип: [Строка](lang__def_String.md).   
Внутреннее подмножество - собственно определение типа документа.

Описание варианта метода:

Осуществляет запись имени и внутреннего подмножества определения типа документа.

Вариант синтаксиса: Имя, системный идентификатор и внутреннее подмножество

Синтаксис:

ЗаписатьТипДокумента(<Имя>, <СистемныйИдентификатор>, <ВнутреннееПодмножество>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя документа.

<СистемныйИдентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Системный идентификатор документа.

<ВнутреннееПодмножество> (обязательный)

Тип: [Строка](lang__def_String.md).   
Внутреннее подмножество - собственно определение типа документа.

Описание варианта метода:

Осуществляет запись имени документа, системного идентификатора и внутреннего подмножества определения типа документа.

Вариант синтаксиса: Имя, публичный идентификатор, системный идентификатор и внутреннее подмножество

Синтаксис:

ЗаписатьТипДокумента(<Имя>, <ПубличныйИдентификатор>, <СистемныйИдентификатор>, <ВнутреннееПодмножество>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя документа.

<ПубличныйИдентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Публичный идентификатор документа.

<СистемныйИдентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Системный идентификатор документа.

<ВнутреннееПодмножество> (обязательный)

Тип: [Строка](lang__def_String.md).   
Внутреннее подмножество - собственно определение типа документа.

Описание варианта метода:

Осуществляет запись имени документа, публичного идентификатора, системного идентификатора и внутреннего подмножества определения типа документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
