# ПланОбменаСсылка.<Имя плана обмена>.Пустая

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаСсылка.<Имя плана обмена>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md) › [Методы](ПланОбменаСсылка.Имя_плана_обмена__Методы.md)

ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

Пустая (IsEmpty)

Доступен, начиная с версии 8.0.

Синтаксис:

Пустая()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ссылка не указывает ни на какой объект (пустая ссылка); [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, является ли ссылка пустой или нет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
