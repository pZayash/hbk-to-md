Глобальный контекст (Global context)

ИмяПользователя (UserName)

Доступен, начиная с версии 8.0.

Синтаксис:

ИмяПользователя()

Возвращаемое значение:

Тип: Строка.   
Имя пользователя.

Описание:

Получает имя пользователя, указанного при запуске программы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТекущийПользователь = ИмяПользователя(); |

[Глобальный контекст](Глобальный_контекст.md), метод [ПолноеИмяПользователя](Глобальный_контекст.ПолноеИмяПользователя.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
