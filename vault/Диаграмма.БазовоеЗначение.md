Диаграмма (Chart)

БазовоеЗначение (BaseValue)

Не рекомендуется использовать, начиная с версии 8.3.10.

Рекомендуется использовать:

- [ОсьЗначений](Диаграмма.ОсьЗначений.md)
- [БазовоеЗначение](ОсьДиаграммы.БазовоеЗначение.md)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит начальное значение координат (ординаты). Используется в случае, когда изменения параметра, отображаемого в диаграмме, несоизмеримо малы по сравнению с его минимальным значением.   
Рекомендуется устанавливать в качестве базового значения минимальное или близкое к нему значение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Следует отводить области изменения параметра большую область построения диаграммы.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.10.

Описание изменено в версии 8.3.15.
