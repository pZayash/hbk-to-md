МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьДляЧтенияАсинх (OpenForReadAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьДляЧтенияАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя открываемого файла.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [ФайловыйПоток](ФайловыйПоток.md) для файла с указанным именем.

Описание:

Асинхронная функция. Открывает существующий файл для чтения с общим доступом на чтение.

Доступность:

Тонкий клиент, толстый клиент.
