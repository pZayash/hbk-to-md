ПолеКартинки (PictureBox)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на поле картинки, если у него свойство [Гиперссылка](ПолеКартинки.Гиперссылка.md) имеет значение Истина.

Доступность:

Толстый клиент.

См. также:

[ПолеКартинки](ПолеКартинки.md), свойство [Гиперссылка](ПолеКартинки.Гиперссылка.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
