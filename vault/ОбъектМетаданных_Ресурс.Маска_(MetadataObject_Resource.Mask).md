ОбъектМетаданных: Ресурс (MetadataObject: Resource)

Маска (Mask)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит маску, с использованием которой значение ресурса редактируется в поле ввода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
