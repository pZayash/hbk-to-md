ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

УстановитьПараметрыВыбораПриложения (SetApplicationChoiceParameters)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПараметрыВыбораПриложения(<Параметры>)

Параметры:

<Параметры> (обязательный)

Тип: [ПараметрыВыбораЗапускаПриложенияМобильногоУстройства](ПараметрыВыбораЗапускаПриложенияМобильногоУстройства.md).   
Параметры для уточнения фильтров, которые будут использованы для поиска приложения мобильного устройства при запуске с помощью методов [ЗапуститьБезОжидания](ЗапускПриложенияМобильногоУстройства.ЗапуститьБезОжидания.md) и [ЗапуститьАсинх](ЗапускПриложенияМобильногоУстройства.ЗапуститьАсинх.md).

Описание:

Устанавливает дополнительные параметры выбора для данного запуска приложения мобильного устройства.   
Изменяет настройки отбора, которые будут использованы для поиска приложения. Если параметры выбора установлены, они будут использованы при попытке найти приложения, которые могут обрабатывать запрос, вместо содержимого основного объекта запуска.   
Это позволяет разработчику создать объект запуска, содержащий общие правила, но с более точным указанием приложения.  
Является аналогом метода Intent.setSelector() в Android.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
