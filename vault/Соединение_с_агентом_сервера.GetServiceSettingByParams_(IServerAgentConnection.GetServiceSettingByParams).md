Соединение с агентом сервера (IServerAgentConnection)

GetServiceSettingByParams (GetServiceSettingByParams)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetServiceSettingByParams(<Кластер>, <Сервер>, <ИмяСервиса>, <ИмяИнформационнойБазы>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого требуется получить настройку.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   
Сервер, для которого требуется получить настройку.

<ИмяСервиса> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя сервиса кластера.

<ИмяИнформационнойБазы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя информационной базы или пустая строка, если сервис кластера не разделен по информационным базам.

Возвращаемое значение:

Тип: [Настройка сервиса](Настройка_сервиса_(IServiceSetting).md).   

Описание:

Возвращает настройку сервиса кластера по указанным параметрам.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
