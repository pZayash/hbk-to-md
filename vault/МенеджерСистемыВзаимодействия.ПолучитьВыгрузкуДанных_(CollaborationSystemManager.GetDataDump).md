МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьВыгрузкуДанных (GetDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьВыгрузкуДанных(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: [ВыгрузкаДанныхСистемыВзаимодействия](ВыгрузкаДанныхСистемыВзаимодействия_(CollaborationSystemDataDump).md).   

Описание:

Получает выгрузку данных по идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение.
