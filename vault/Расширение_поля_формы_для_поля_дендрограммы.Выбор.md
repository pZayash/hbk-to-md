Расширение поля формы для поля дендрограммы (Form field extension for a dendrogram field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<ЗначениеДендрограммы>, <СтандартнаяОбработка>)

Параметры:

<ЗначениеДендрограммы>

Тип: [ЭлементДендрограммы](ЭлементДендрограммы.md), [СвязьДендрограммы](СвязьДендрограммы.md).   
Выбранное значение.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

Описание:

Событие, возникающее при выборе дендрограммы (например, при двойном щелчке мыши).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
