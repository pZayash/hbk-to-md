КоллекцияПользовательскихПолейКомпоновкиДанных (DataCompositionUserFieldCollection)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных_(DataCompositionUserFieldExpression).md), [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных_(DataCompositionUserFieldCase).md).

Возвращаемое значение:

Тип: [ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных_(DataCompositionUserFieldExpression).md), [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных_(DataCompositionUserFieldCase).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
