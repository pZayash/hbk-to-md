ИсточникДанныхСводнойТаблицыКомпоновкиДанных (DataCompositionPivotTableDataSource)

УстановитьНастройки (SetSettings)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьНастройки(<Настройки>)

Параметры:

<Настройки> (необязательный)

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Объект настроек компоновки данных.

Описание:

Устанавливает настройки системы компоновки данных в источник сводной таблицы.  
В настройках ожидается таблица, группировки которой будут соответствовать измерениям сводной таблицы, выбранные поля группировок будут соответствовать ресурсам.  
Выбранные поля настроек будут соответствовать измерениям, отображаемым в сводной таблице.  
Также настройки могут содержать отбор, упорядочивание, которые будут влиять на информацию, отображаемую в сводной таблице. Содержимое сводной таблицы будет обновлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод может быть вызван после установки источника в сводную таблицу.   
До установки источника в сводную таблицу должен быть вызван метод [УстановитьСхему](ИсточникДанныхСводнойТаблицыКомпоновкиДанных.УстановитьСхему_(DataCompositionPivotTableDataSource.SetSchema).md).

Использование в версии:

Доступен, начиная с версии 8.1.
