ТипШтрихКода (BarcodeType)

ITF (ITF)

Доступен, начиная с версии 8.3.25.

Описание:

При вызове сканирования система должна распознавать штрихкоды линейного типа Interleaved 2 of 5. Позволяет кодировать цифры от 0 до 9. Не содержит контрольного знака.

Использование в версии:

Доступен, начиная с версии 8.3.25.
