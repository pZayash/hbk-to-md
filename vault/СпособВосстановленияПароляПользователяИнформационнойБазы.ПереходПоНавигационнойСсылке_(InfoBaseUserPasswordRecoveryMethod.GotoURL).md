СпособВосстановленияПароляПользователяИнформационнойБазы (InfoBaseUserPasswordRecoveryMethod)

ПереходПоНавигационнойСсылке (GotoURL)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

При восстановлении пароля будет выполнен переход по указанной в настройках ссылке.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
