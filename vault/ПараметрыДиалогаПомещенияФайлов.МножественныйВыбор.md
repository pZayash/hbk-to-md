ПараметрыДиалогаПомещенияФайлов (PutFilesDialogParameters)

МножественныйВыбор (MultipleChoice)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Значение свойства определяет, разрешен ли в диалоге выбор нескольких файлов:

- Истина – разрешается,
- Ложь – не разрешается.

  
Если значение Неопределено, то в методе [НачатьПомещениеФайлаНаСервер](Глобальный_контекст.НачатьПомещениеФайлаНаСервер.md) значение интерпретируется как Ложь, а в методе [НачатьПомещениеФайловНаСервер](Глобальный_контекст.НачатьПомещениеФайловНаСервер.md) значение интерпретируется как Истина.  
Значению по умолчанию: Неопределено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
