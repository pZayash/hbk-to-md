СерияДиаграммы (ChartSeries)

ГруппаНакопления (StackGroup)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.16 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Серии с одинаковой группой накопления накапливаются вместе, с разными – раздельно. Порядок накопления соответствует порядку серий.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Используется только для диаграмм следующих видов:
