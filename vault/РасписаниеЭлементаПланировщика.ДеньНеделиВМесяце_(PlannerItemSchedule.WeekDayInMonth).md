РасписаниеЭлементаПланировщика (PlannerItemSchedule)

ДеньНеделиВМесяце (WeekDayInMonth)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Номер дня недели в месяце (например, второе воскресенье месяца). Положительные значения отсчитываются относительно начала месяца, отрицательные - относительно конца (1 - первый день недели в месяце, -1 - последний день недели в месяце).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
