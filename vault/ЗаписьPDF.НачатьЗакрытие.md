ЗаписьPDF (PDFWriter)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Закрывает открытый файл. Закрытие файла выполняется автоматически при удалении объекта [ЗаписьPDF](ЗаписьPDF.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF.md), метод [Закрыть](ЗаписьPDF.Закрыть.md)  
[ЗаписьPDF](ЗаписьPDF.md), метод [ЗакрытьАсинх](ЗаписьPDF.ЗакрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
