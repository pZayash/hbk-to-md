Глобальный контекст (Global context)

ОбработкаОтображенияОшибки (ErrorDisplayProcessing)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ОбработкаОтображенияОшибки(<ИнформацияОбОшибке>, <ТребуетсяЗавершениеСеанса>, <СтандартнаяОбработка>)

Параметры:

<ИнформацияОбОшибке>

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Содержит описание ошибки, при возникновении которой был вызван данный обработчик.

<ТребуетсяЗавершениеСеанса>

Тип: Булево.   
В данный параметр передается признак того, что работа в приложении не может быть продолжена и после окончания работы обработчика будет выполнено завершение работы или перезапуск приложения.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак отображения пользователю стандартного сообщения об ошибке.

Описание:

Вызывается при возникновении неперехваченного исключения, перед отображением пользователю окна с ошибкой.   
В обработчике данного события можно выполнить дополнительные действия по логированию ошибки, показать свою форму ошибки, отправить информацию об ошибке на внешний сервер. Следует помнить, что для некоторых типов ошибок (например, ошибка сети, удаление сеанса) невозможно обращение к серверу (например, для получения формы). В таких случаях можно воспользоваться методами [ПоказатьВопрос](Глобальный_контекст.ПоказатьВопрос.md) и [ПоказатьОповещениеПользователя](Глобальный_контекст.ПоказатьОповещениеПользователя.md).   
Если параметр <ТребуетсяЗавершениеСеанса> имеет значение Истина, то открыть форму можно только использую только метод [ПоказатьИнформациюОбОшибке](МенеджерОбработкиОшибок.ПоказатьИнформациюОбОшибке.md) и объект [ОтчетОбОшибке](ОтчетОбОшибке.md). Любые другие формы не будут влиять на завершение работы после завершения выполнения обработчика.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ОтчетОбОшибке](ОтчетОбОшибке.md)

Использование в версии:

Доступен, начиная с версии 8.3.17.
