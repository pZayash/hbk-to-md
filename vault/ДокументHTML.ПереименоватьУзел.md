ДокументHTML (HTMLDocument)

ПереименоватьУзел (RenameNode)

Доступен, начиная с версии 8.2.

Синтаксис:

ПереименоватьУзел(<Узел>, <КвалифицированноеИмя>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутHTML](АтрибутHTML.md), [ЭлементHTML](ЭлементHTML.md).   
Переименовываемый узел HTMLDOM.

<КвалифицированноеИмя> (обязательный)

Тип: Строка.   
Новое квалифицируемое имя переименовываемого узла.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   

Описание:

Переименовывает узлы [Элемент](ТипУзлаDOM.Элемент.md) и [Атрибут](ТипУзлаDOM.Атрибут.md).  
Выполняется попытка переименования существующего узла, в случае невозможности, создает новый узел с новым именем, префиксом и заменяет им исходный узел.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- попытка переименования узлов, отличных от [Элемент](ТипУзлаDOM.Элемент.md) или [Атрибут](ТипУзлаDOM.Атрибут.md), или в случае, если реализация не поддерживает переименование узлов;
- квалифицированное имя содержит символы, недопустимые стандартом XML данного документа;
- попытка переименования узла, созданного другим документом HTML;
- квалифицированное имя содержит префикс, значение которого недопустимо для данного значения URI пространства имен.

Использование в версии:

Доступен, начиная с версии 8.2.
