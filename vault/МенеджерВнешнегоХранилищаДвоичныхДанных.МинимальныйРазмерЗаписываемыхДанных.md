МенеджерВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageManager)

МинимальныйРазмерЗаписываемыхДанных (MinWriteDataSize)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Минимальный размер записываемых данных (в байтах).

Доступность:

Сервер.

Примечание:

По умолчанию - 2048.  
Не допускается установка значения менее 2048.
