ПользовательскиеНастройкиКомпоновкиДанных (DataCompositionUserSettings)

ПолучитьИдентификаторПоОбъекту (GetIDByObject)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИдентификаторПоОбъекту(<Объект>)

Параметры:

<Объект> (необязательный)

Объект настроек, который можно использовать в пользовательских настройках, для которого нужно получить идентификатор.

Возвращаемое значение:

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md).   

Описание:

Получает идентификатор переданного объекта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
