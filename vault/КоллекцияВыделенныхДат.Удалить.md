КоллекцияВыделенныхДат (SelectedDateCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По дате

Синтаксис:

Удалить(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: Дата.   
Удаляемая дата.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: Число.   
Позиция удаляемой даты.

Описание:

Удаляет конкретную дату из коллекции выделенных дат.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
