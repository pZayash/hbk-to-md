СертификатКриптографии (CryptoCertificate)

ИспользоватьДляПодписи (UseToSign)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, можно ли использовать данный сертификат для подписи.  
[Истина](lang__def_BooleanTrue.md) - сертификат используется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
