ПреобразованиеXSL (XSLTransform)

ЗагрузитьТаблицуСтилейXSLИзФайла (LoadXSLStylesheetFromFile)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ЗагрузитьТаблицуСтилейXSLИзФайла(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из которого должно быть загружено описание преобразования XSL.

Описание:

Загружает описание преобразования XSL из файла.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
