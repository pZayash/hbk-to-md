КоллекцияИзмеренийПланировщика (PlannerDimensionCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [ИзмерениеПланировщика](ИзмерениеПланировщика_(PlannerDimension).md).   
Индекс сдвигаемого измерения в коллекции или само измерение.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое необходимо сдвинуть измерение. Положительное значение соответствует смещению к концу коллекции, отрицательное - к началу коллекции.

Описание:

Сдвигает измерение с заданным индексом на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
