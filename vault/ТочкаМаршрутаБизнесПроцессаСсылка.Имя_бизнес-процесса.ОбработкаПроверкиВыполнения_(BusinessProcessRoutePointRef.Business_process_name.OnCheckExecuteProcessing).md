# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ОбработкаПроверкиВыполнения

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ОбработкаПроверкиВыполнения (OnCheckExecuteProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаПроверкиВыполнения(<ТочкаМаршрута>, <Задача>, <Результат>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, в которой выполняется проверка завершения.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Задача, при выполнении которой приходит событие.

<Результат>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак завершения задачи. Если в параметре возвращается [Ложь](lang__def_BooleanFalse.md), то задача не будет отмечена как завершенная.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при вызове метода [ПроверитьВыполнение](ЗадачаОбъект.Имя_задачи.ПроверитьВыполнение_(TaskObject.Task_name.CheckExecution).md) у задачи. Не вызывается, если задача уже выполнена. Вызывается после вызова обработчика события [ОбработкаПроверкиВыполнения](ЗадачаОбъект.Имя_задачи.ОбработкаПроверкиВыполнения_(TaskObject.Task_name.OnCheckExecutionProcessing).md) у самой задачи и если в теле этого обработчика для параметра <Результат> установлено значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md).  
Допускаются интерактивные вызовы (открытие форм, предупреждений).

Использование в версии:

Доступен, начиная с версии 8.0.
