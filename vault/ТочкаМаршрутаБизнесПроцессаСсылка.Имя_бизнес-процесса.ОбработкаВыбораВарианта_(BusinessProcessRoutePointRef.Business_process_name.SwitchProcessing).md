# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ОбработкаВыбораВарианта

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ОбработкаВыбораВарианта (SwitchProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаВыбораВарианта(<ТочкаВыбораВарианта>, <Результат>)

Параметры:

<ТочкаВыбораВарианта>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка выбора варианта, для которой вызывается обработчик для определения дальнейшей точки маршрута бизнес-процесса.

<Результат>

Тип: [ВариантТочкиМаршрутаБизнесПроцесса](ВариантТочкиМаршрутаБизнесПроцесса_(BusinessProcessRoutePointCase).md).   
Выбранный вариант точки маршрута бизнес-процесса. Должен быть обязательно установлен одним из вариантов.

Описание:

Вызывается после осуществления выбора перехода. В параметре <Результат> возвращается вариант перехода. Обработчик этого события обязательно должен быть назначен, т.к. в противном случае бизнес-процесс не сможет самостоятельно выбрать вариант перехода, что приведет к ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта_(BusinessProcessRoutePointType.Switch).md).  
Это событие неинтерактивное и в нем не рекомендуется открывать какие-либо окна для организации диалога с пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
