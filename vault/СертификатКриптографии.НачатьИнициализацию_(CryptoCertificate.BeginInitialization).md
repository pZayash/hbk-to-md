# СертификатКриптографии.НачатьИнициализацию

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Криптография](Криптография-2.md) › [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md) › [Методы](СертификатКриптографии__Методы.md)

СертификатКриптографии (CryptoCertificate)

НачатьИнициализацию (BeginInitialization)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения загрузки сертификата из файла со следующими параметрами:

- <Сертификат> – созданный [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полный путь к файлу сертификата.

Описание:

Начинает инициализацию объекта [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), метод [ИнициализироватьАсинх](СертификатКриптографии.ИнициализироватьАсинх_(CryptoCertificate.InitializeAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
