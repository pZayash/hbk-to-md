Описание:

Содержит варианты форматов файлов для сохранения табличного документа. Используется для определения параметра <ТипФайлаТаблицы> метода [Записать](ТабличныйДокумент.Записать_(SpreadsheetDocument.Write).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [Записать](ТабличныйДокумент.Записать_(SpreadsheetDocument.Write).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
