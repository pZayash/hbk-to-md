Расширение формы клиентского приложения для отчета (Client application form extension for reports)

УстановитьРежимОтображенияРезультата (SetResultViewMode)

Доступен, начиная с версии 8.3.19.

Синтаксис:

УстановитьРежимОтображенияРезультата(<РежимОтображения>)

Параметры:

<РежимОтображения> (обязательный)

Тип: [РежимОтображенияРезультатаОтчета](РежимОтображенияРезультатаОтчета_(ReportResultViewMode).md).   
Режим отображения результата отчета, который требуется установить.

Описание:

Устанавливает для формы режим отображения результата отчета.   
При изменении режима отображения осуществляется преобразование результата отчета в соответствии с новым значением. При вызове данного метода на клиенте может потребоваться серверный вызов (в случае, когда меняется режим отображения результата).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
