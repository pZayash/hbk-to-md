МенеджерДоставляемыхУведомлений (DeliverableNotificationManager)

ДобавитьЛокальноеУведомление (AddLocalNotification)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ДобавитьЛокальноеУведомление(<Уведомление>)

Параметры:

<Уведомление> (обязательный)

Тип: [ДоставляемоеУведомление](ДоставляемоеУведомление.md).   
Добавляемое уведомление.

Описание:

Добавляет локальное уведомление. У уведомления должно быть заполнено поле "Текст". В противном случае генерируется исключение.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

На платформе Windows Phone уведомление будет передано в обработчик доставляемых уведомлений только при нажатии на него в центре уведомлений

Использование в версии:

Доступен, начиная с версии 8.3.6.
