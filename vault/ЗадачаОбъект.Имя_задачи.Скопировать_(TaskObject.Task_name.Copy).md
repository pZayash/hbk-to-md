# ЗадачаОбъект.<Имя задачи>.Скопировать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Методы](ЗадачаОбъект.Имя_задачи__Методы.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ЗадачаОбъект.\<Имя задачи\>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md).   

Описание:

Создает новую задачу копированием существующей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.0.
