ДокументHTML (HTMLDocument)

Якоря (Anchors)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML_(HTMLCollection).md).   
Содержит коллекцию всех якорей HTML документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Коллекция содержит элементы типа [ЭлементЯкорьHTML](ЭлементЯкорьHTML_(HTMLAnchorElement).md).

Использование в версии:

Доступен, начиная с версии 8.2.
