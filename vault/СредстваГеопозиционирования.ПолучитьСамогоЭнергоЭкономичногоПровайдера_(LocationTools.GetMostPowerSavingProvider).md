СредстваГеопозиционирования (LocationTools)

ПолучитьСамогоЭнергоЭкономичногоПровайдера (GetMostPowerSavingProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьСамогоЭнергоЭкономичногоПровайдера(<ТолькоДоступного>)

Параметры:

<ТолькоДоступного> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если передано значение [Истина](lang__def_BooleanTrue.md), информация о самом экономичном провайдере будет возвращена только если провайдер доступен. Если самый энергоэкономичный провайдер недоступен, будет возвращено [Неопределено](lang__def_Undefined.md).  
Если передано значение [Ложь](lang__def_BooleanFalse.md), будет возвращена информация о самом экономичном провайдере на устройстве.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования_(LocationProviderInformation).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - ни один провайдер не найден, или если параметр <ТолькоДоступного> содержал значение [Истина](lang__def_BooleanTrue.md), а самый экономичный провайдер не доступен.

Описание:

Получает информацию о самом энергоэкономичном провайдере геопозиционирования.  
Если параметр <ТолькоДоступного> содержит значение [Истина](lang__def_BooleanTrue.md), информация о самом экономичном провайдере будет возвращена только в случае, если провайдер доступен.  
Если параметр <ТолькоДоступного> содержит значение [Ложь](lang__def_BooleanFalse.md), информация о самом экономичном провайдере будет возвращена независимо от доступности провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
