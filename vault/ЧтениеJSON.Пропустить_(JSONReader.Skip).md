ЧтениеJSON (JSONReader)

Пропустить (Skip)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Пропустить()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - очередной узел прочитан;  
[Ложь](lang__def_BooleanFalse.md) - был достигнут конец текста JSON.

Описание:

Если текущее значение – начало массива или объекта, то пропускает его содержимое и конец.  
Для остальных типов значений работает аналогично [Прочитать](ЧтениеJSON.Прочитать_(JSONReader.Read).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
