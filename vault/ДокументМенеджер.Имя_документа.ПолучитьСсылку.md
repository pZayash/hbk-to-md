ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа.md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор.md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ДокументСсылка.Имя_документа.УникальныйИдентификатор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан или передано значение Неопределено, то будет сформирована новая уникальная ссылка.

См. также:

[ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа.md), метод [УникальныйИдентификатор](ДокументСсылка.Имя_документа.УникальныйИдентификатор.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
