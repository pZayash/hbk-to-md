Глобальный контекст (Global context)

ПолучитьВремяПредупрежденияОЗавершенииСеансаПриБездействии (GetInactivityTimeForTerminateSessionNotification)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьВремяПредупрежденияОЗавершенииСеансаПриБездействии()

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает интервал неактивности сеанса (в секундах), по истечении которого отобразится предупреждение о завершении сеанса по причине бездействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
