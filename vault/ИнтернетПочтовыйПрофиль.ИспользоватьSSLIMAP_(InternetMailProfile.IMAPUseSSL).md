ИнтернетПочтовыйПрофиль (InternetMailProfile)

ИспользоватьSSLIMAP (IMAPUseSSL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость использования SSL соединения для протокола IMAP.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если предполагается использование защищенного STARTTLS-соединения, то должно быть установлено в значение [Ложь](lang__def_BooleanFalse.md).  
При установке свойству значения [Истина](lang__def_BooleanTrue.md), номер порта IMAP устанавливается в значение по умолчанию - "993". При установке значения [Ложь](lang__def_BooleanFalse.md) - номер порта принимает значение "143".

Использование в версии:

Доступен, начиная с версии 8.3.1.

Описание изменено в версии 8.3.20.
