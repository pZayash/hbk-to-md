Глобальный контекст (Global context)

ПолучитьДополнительныйПараметрКлиентаЛицензирования (GetLicensingClientAdditionalParameter)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьДополнительныйПараметрКлиентаЛицензирования()

Возвращаемое значение:

Тип: Строка.   
Значение дополнительного параметра.

Описание:

Получает дополнительный параметр, используемый для обслуживания информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только пользователям с административными правами.

Пример:

|  |
| --- |
| ДополнительныйПараметрКлиентаЛицензирования = ПолучитьДополнительныйПараметрКлиентаЛицензирования(); |

Использование в версии:

Доступен, начиная с версии 8.3.7.
