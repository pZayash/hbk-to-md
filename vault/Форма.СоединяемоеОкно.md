Форма (Form)

СоединяемоеОкно (ConnectableWindow)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если это свойство установлено (Истина), окно будет соединяться с другими окнами.  
Данное свойство устанавливается окну только в том случае, если положение окна не было ранее сохранено в профиле.

Доступность:

Толстый клиент.

Примечание:

Управление свойством должно производиться до открытия формы.

Использование в версии:

Доступен, начиная с версии 8.0.
