НастройкаПериода (PeriodSettings)

ДатаОкончания (EndDate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Содержит дату окончания периода.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если свойство [ВариантОкончания](НастройкаПериода.ВариантОкончания.md) имеет значение [КонкретнаяДата](ВариантГраницыИнтервала.КонкретнаяДата.md).

Использование в версии:

Доступен, начиная с версии 8.0.
