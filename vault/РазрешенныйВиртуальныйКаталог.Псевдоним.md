РазрешенныйВиртуальныйКаталог (AllowedVirtualDirectory)

Псевдоним (Alias)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Логический URL-адрес каталога. Этот URL-адрес можно использовать в конфигурации для обращения к файлам и подкаталогам данного виртуального каталога. Перед сравнением логические URL-адреса приводятся к каноническому виду. Для этого выполняются следующие действия:

- удаление лишних пробельных символов,
- перевод регистронезависимых частей URL-адреса в нижний регистр,
- замена символов "\\" на "/" в пути к ресурсу,
- преобразование пути – удаление пустых секций и интерпретация секций "." и "..".

Если в контексте информационной базы, области данных, сеанса или режима безопасного исполнения кода конфигурация или платформа выполняет любую операцию с файловой системой по URL-адресу, начало которого не совпадает со значением свойства Alias одного из виртуальных каталогов, то генерируется исключение.   
Существуют несколько предопределенных логических URL-адресов:

- /bin – каталог загрузочных модулей текущей версии "1С:Предприятия" (может быть получен методом [КаталогПрограммы](Глобальный_контекст.КаталогПрограммы.md)),
- /temp – каталог временных файлов (может быть получен методом [КаталогВременныхФайлов](Глобальный_контекст.КаталогВременныхФайлов.md)).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
