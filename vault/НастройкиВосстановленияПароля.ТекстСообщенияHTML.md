НастройкиВосстановленияПароля (PasswordRecoverySettings)

ТекстСообщенияHTML (HTMLMessageText)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст сообщения в формате HTML.  
Текст сообщения обязательно должен содержать параметр &VerificationCode.  
Три параметра, содержащиеся в тексте, &VerificationCode, &UserPresentation, и &ApplicationPresentation заменяются на код восстановления пароля, имя пользователя, и представление приложения соответственно. Два последних параметра необязательны.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
