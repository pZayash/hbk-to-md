ЧтениеXML (XMLReader)

ЭтоПробельныеСимволы (IsWhitespace)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, состоит ли текстовое значение текущего узла только из пробельных символов.  
Применимо только для узлов, обладающих текстовым содержимым, для других узлов содержит [Ложь](lang__def_BooleanFalse.md).  
[Истина](lang__def_BooleanTrue.md) - текст состоит только из пробельных символов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
