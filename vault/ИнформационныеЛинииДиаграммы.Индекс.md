ИнформационныеЛинииДиаграммы (ChartReferenceLines)

Индекс (Index)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Индекс(<Линия>)

Параметры:

<Линия> (обязательный)

Тип: [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы.md).   
Линия, индекс которой в коллекции требуется получить.

Возвращаемое значение:

Тип: Число.   
Если объект не принадлежит этой коллекции, генерируется исключение.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
