Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных.md) [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных.md).

Тип: [ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных.md), [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
