ОбъектМетаданных: ОбщаяКоманда (MetadataObject: CommonCommand)

РежимИспользованияПараметра (ParameterUsageMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РежимИспользованияПараметраКоманды](РежимИспользованияПараметраКоманды.md).   
Перечисление из двух значений: [Одиночный](РежимИспользованияПараметраКоманды.Одиночный.md) и [Множественный](РежимИспользованияПараметраКоманды.Множественный.md). Если свойство имеет значение [Множественный](РежимИспользованияПараметраКоманды.Множественный.md), то в обработчик команды передается массив значений, а не одно значение даже, если в конкретном случае значение одно.   
Массив значений в качестве параметра может быть передан в обработчик команды в том случае, если источником для параметра команды является таблица в режиме множественного выделения. При этом первым элементом массива всегда будет текущая строка в независимости от последовательности выделения строк таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
