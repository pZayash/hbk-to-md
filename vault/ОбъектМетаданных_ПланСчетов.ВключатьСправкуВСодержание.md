ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ВключатьСправкуВСодержание (IncludeHelpInContents)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то справка по данному объекту метаданных (свойство [Справка](ОбъектМетаданных_БизнесПроцесс.Справка.md)) будет включена в общую справку по конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
