ОриентацияПодписейДиаграммы (ChartLabelsOrientation)

ПроизвольныйУголНаклона (CustomAngle)

Доступен, начиная с версии 8.3.11.

Описание:

Наклон подписей устанавливается отдельно, с помощью соответствующего свойства объекта, для которого изменяется угол наклона подписей.

Использование в версии:

Доступен, начиная с версии 8.3.11.
