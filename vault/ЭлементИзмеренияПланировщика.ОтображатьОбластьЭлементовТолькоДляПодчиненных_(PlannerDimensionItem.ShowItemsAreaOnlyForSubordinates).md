ЭлементИзмеренияПланировщика (PlannerDimensionItem)

ОтображатьОбластьЭлементовТолькоДляПодчиненных (ShowItemsAreaOnlyForSubordinates)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если элемент измерения содержит подчиненные элементы (то есть свойство [Элементы](ЭлементИзмеренияПланировщика.Элементы_(PlannerDimensionItem.Items).md) содержит не пустую коллекцию), то данное свойство определяет отображение области элементов данного элемента измерения:

- Если значение [Истина](lang__def_BooleanTrue.md), то область элементов данного элемента измерения не отображается, а отображаются области элементов только для подчиненных элементов измерения.
- Если значение [Ложь](lang__def_BooleanFalse.md), то область элементов данного элемента измерения отображается вместе с областями элементов для подчиненных элементов измерения.

Если элемент измерения не содержит подчиненные элементов, то данное свойство игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
