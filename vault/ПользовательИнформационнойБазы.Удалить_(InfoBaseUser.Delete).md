ПользовательИнформационнойБазы (InfoBaseUser)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет сведения о пользователе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только пользователю с административными правами.

Использование в версии:

Доступен, начиная с версии 8.0.
