ПараметрыЧтенияXML (XMLReaderSettings)

ТипПроверкиПравильности (ValidationType)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ТипПроверкиXML](ТипПроверкиXML_(XMLValidationType).md).   
Содержит тип проверки данных XML на правильность при чтении. Значение по умолчанию [НетПроверки](ТипПроверкиXML.НетПроверки_(XMLValidationType.NoValidate).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
