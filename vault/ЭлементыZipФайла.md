Рекомендуется использовать:

- [ЭлементыФайлаАрхива](ЭлементыФайлаАрхива.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементZipФайла](ЭлементZipФайла.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элемент коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Методы:

[Количество (Count)](ЭлементыZipФайла.Количество.md)  
[Найти (Find)](ЭлементыZipФайла.Найти.md)  
[Получить (Get)](ЭлементыZipФайла.Получить.md)  

Описание:

Содержит коллекцию элементов ZIP файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЧтениеZipФайла](ЧтениеZipФайла.md), свойство [Элементы](ЧтениеZipФайла.Элементы.md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.

Описание изменено в версии 8.3.26.
