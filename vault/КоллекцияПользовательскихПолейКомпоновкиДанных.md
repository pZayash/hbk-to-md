Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПользовательскоеПолеВыражениеКомпоновкиДанных](ПользовательскоеПолеВыражениеКомпоновкиДанных.md), [ПользовательскоеПолеВыборКомпоновкиДанных](ПользовательскоеПолеВыборКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются пользовательские поля.  
Возможно обращение к пользовательскому полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Коллекция пользовательских полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
