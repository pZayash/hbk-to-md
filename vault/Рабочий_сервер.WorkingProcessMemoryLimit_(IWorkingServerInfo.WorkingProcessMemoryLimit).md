# Рабочий сервер.WorkingProcessMemoryLimit

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md) › [Свойства](Рабочий_сервер__Свойства.md)

Рабочий сервер (IWorkingServerInfo)

WorkingProcessMemoryLimit (WorkingProcessMemoryLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит ограничение суммарного объема оперативной памяти, занятой рабочими процессами на этом сервере. Если суммарный объем оперативной памяти, занятой рабочими процессами, превысил эту величину, то на этом сервере новые рабочие процессы запускаться не будут.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
