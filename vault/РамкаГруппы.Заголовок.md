РамкаГруппы (GroupBox)

Заголовок (Caption)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит текст подписи рамки группы.

Доступность:

Толстый клиент.

Примечание:

Для рамок типа [Одинарная](ТипРамкиЭлементаУправления.Одинарная.md) и Двойная заголовок отображается на самой рамке. Для рамок типа [Выпуклая](ТипРамкиЭлементаУправления.Выпуклая.md) и [Вдавленная](ТипРамкиЭлементаУправления.Вдавленная.md) заголовок отображается внутри рамки.

Использование в версии:

Доступен, начиная с версии 8.0.
