БуферДвоичныхДанных (BinaryDataBuffer)

Разделить (Split)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: По двоичному буферу

Синтаксис:

Разделить(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Разделитель.

Описание варианта метода:

Разделяет буфер на части, используя двоичный буфер-разделитель.

Вариант синтаксиса: По массиву разделителей

Синтаксис:

Разделить()

Описание варианта метода:

Разделить буфер на части по заданным разделителям. Элементы массива имеют тип [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Вариант синтаксиса: По массиву разделителей

Синтаксис:

Разделить(<Разделители>)

Параметры:

<Разделители> (обязательный)

Тип: [Массив](Массив.md).   
Массив разделителей. Может содержать значения типа [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Описание варианта метода:

Разделить буфер на части по заданным разделителям.  
Элементы массива должны иметь тип [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Тип элемента массива - [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Описание:

Разделить буфер на части по заданному разделителю.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
