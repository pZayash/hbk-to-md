СертификатКриптографии (CryptoCertificate)

Из двоичных данных

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СертификатКриптографии(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Данные, из которых будет построен сертификат.

Описание:

Создает сертификат из двоичных данных.  
В веб-клиенте, если не установлено расширение работы с криптографией или если недоступно использование синхронных вызовов, при вызове конструктора генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
