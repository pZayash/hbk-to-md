# Массив.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Массив](Массив_(Array).md) › [Методы](Массив__Методы.md)

Массив (Array)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого элемента.

Описание:

Удаляет значение из массива по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если указанный в параметре индекс больше, чем [ВГраница](Массив.ВГраница_(Array.UBound).md), то никаких действий не производится и не выдается сообщение "Индекс находится за границами массива".

Пример:

|  |
| --- |
| Массив.Удалить(4); |

Использование в версии:

Доступен, начиная с версии 8.0.
