АдминистрированиеТребованиеНазначения (AdministrationAssignmentRule)

Приоритет (Priority)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.   
Приоритет требования. Для объекта требования при размещении вычисляется эффективное требование для каждого из серверов. Объект будет размещен там, где приоритет выше (при одинаковых типах требования).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
