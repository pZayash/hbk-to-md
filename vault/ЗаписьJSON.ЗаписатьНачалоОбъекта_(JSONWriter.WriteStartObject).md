# ЗаписьJSON.ЗаписатьНачалоОбъекта

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [JSON](JSON-2.md) › [ЗаписьJSON](ЗаписьJSON_(JSONWriter).md) › [Методы](ЗаписьJSON__Методы.md)

ЗаписьJSON (JSONWriter)

ЗаписатьНачалоОбъекта (WriteStartObject)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗаписатьНачалоОбъекта()

Описание:

Записывает начало объекта JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; // Запись документа JSON  // Запись строковое значение  ЗаписьJSON.ЗаписатьНачалоОбъекта(); ЗаписьJSON.ЗаписатьИмяСвойства("Строка"); ЗаписьJSON.ЗаписатьЗначение("Строковое значение"); ЗаписьJSON.ЗаписатьКонецОбъекта(); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
