ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыбратьИзСписка (ChooseFromList)

Доступен, начиная с версии 8.2.

Синтаксис:

ВыбратьИзСписка(<СписокЗначений>, <ЭлементФормы>, <НачальноеЗначение>)

Параметры:

<СписокЗначений> (обязательный)

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Список значений, содержащий значения, из которых будет осуществляться выбор.

<ЭлементФормы> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md), [КнопкаФормы](КнопкаФормы_(FormButton).md).   
Элемент формы, около которого будет открыт выпадающий список. Для некоторых элементов выпадающий список будет показан около активной области.   
Если не указан, то будет выбран активный элемент формы.  
Примечание: В качестве этого параметра нельзя указывать кнопки, расположенные в командной панели или подменю.

<НачальноеЗначение> (необязательный)

Тип: [Число](lang__def_Number.md), [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).   
Задает начальный элемент, который будет активным при открытии списка.   
Если параметр не указан, в списке не будет выбрано начальное значение.

Возвращаемое значение:

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md), [Неопределено](lang__def_Undefined.md).   
Выбранный элемент списка или [Неопределено](lang__def_Undefined.md), если пользователь отказался от выбора.

Описание:

Вызывает выпадающий список для выбора значения. Набор значений для выбора задается списком значений. Выпадающий список будет расположен около переданного элемента формы или около его активной области.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если в списке значений указаны картинки, то они отображаются в выпадающем списке.  
В веб-клиенте выбор осуществляется из списка, открываемого в отдельном окне.  
Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [ПоказатьВыборИзСписка](ФормаКлиентскогоПриложения.ПоказатьВыборИзСписка_(ClientApplicationForm.ShowChooseFromList).md).

Пример:

|  |
| --- |
| Список = Новый СписокЗначений; Список.Добавить("Доставка"); Список.Добавить("Разгрузка"); Список.Добавить("Картинка",,,БиблиотекаКартинок.Картинка1); Вид = ВыбратьИзСписка(Список, Элементы.Поле1); |
