ЭлементРезультатаГлобальногоПоиска (GlobalSearchResultItem)

Представление (Presentation)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Представление результата поиска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Представление используется только для отображения элемента, внутренние гиперссылки в нем игнорируются.  
Для настройки действий со строкой результата следует использовать свойство [Действия](ЭлементРезультатаГлобальногоПоиска.Действия_(GlobalSearchResultItem.Actions).md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
