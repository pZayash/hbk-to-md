ПоляНастройки (CustomFields)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомого поля.

Возвращаемое значение:

Тип: [ПолеНастройки](ПолеНастройки.md), Неопределено.   
Если поле не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск поля по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
