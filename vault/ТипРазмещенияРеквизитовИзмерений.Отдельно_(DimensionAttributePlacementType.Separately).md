ТипРазмещенияРеквизитовИзмерений (DimensionAttributePlacementType)

Отдельно (Separately)

Доступен, начиная с версии 8.0.

Описание:

Реквизиты располагаются в отдельных строках/колонках.

Использование в версии:

Доступен, начиная с версии 8.0.
