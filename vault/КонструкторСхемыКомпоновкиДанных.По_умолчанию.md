КонструкторСхемыКомпоновкиДанных (DataCompositionSchemaWizard)

На основании схемы

Доступен, начиная с версии 8.1.

Синтаксис:

Новый КонструкторСхемыКомпоновкиДанных(<СхемаКомпоновкиДанных>)

Параметры:

<СхемаКомпоновкиДанных> (необязательный)

Тип: [СхемаКомпоновкиДанных](СхемаКомпоновкиДанных_(DataCompositionSchema).md).   
По умолчанию схема компоновки данных не задается.

Описание:

Создается объект [КонструкторСхемыКомпоновкиДанных](КонструкторСхемыКомпоновкиДанных_(DataCompositionSchemaWizard).md).

Использование в версии:

Доступен, начиная с версии 8.1.
