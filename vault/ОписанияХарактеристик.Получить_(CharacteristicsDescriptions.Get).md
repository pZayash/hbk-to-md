ОписанияХарактеристик (CharacteristicsDescriptions)

Получить (Get)

Доступен, начиная с версии 8.2.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс характеристики.

Возвращаемое значение:

Тип: [ОписаниеХарактеристик](ОписаниеХарактеристик_(CharacteristicsDescription).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
