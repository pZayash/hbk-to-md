ФабрикаXDTO (XDTOFactory)

Пакеты (Packages)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияПакетовXDTO](КоллекцияПакетовXDTO_(XDTOPackageCollection).md).   
Содержит коллекцию пакетов XDTO, составляющих фабрику.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
