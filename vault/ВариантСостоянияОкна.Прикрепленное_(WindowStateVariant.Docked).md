ВариантСостоянияОкна (WindowStateVariant)

Прикрепленное (Docked)

Доступен, начиная с версии 8.0.

Описание:

Прикрепляемое: окно может "прикрепиться" к границам рабочей области программы.

Использование в версии:

Доступен, начиная с версии 8.0.
