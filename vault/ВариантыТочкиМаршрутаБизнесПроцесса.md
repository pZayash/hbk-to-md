# ВариантыТочкиМаршрутаБизнесПроцесса

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ВариантТочкиМаршрутаБизнесПроцесса](ВариантТочкиМаршрутаБизнесПроцесса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются варианты.  
Возможно обращение к варианту посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента в коллекции.  

Свойства:

[\<Имя варианта\> (\<Variant name\>)](ВариантыТочкиМаршрутаБизнесПроцесса.Имя_варианта.md)  

Методы:

[Количество (Count)](ВариантыТочкиМаршрутаБизнесПроцесса.Количество.md)  
[Получить (Get)](ВариантыТочкиМаршрутаБизнесПроцесса.Получить.md)  

Описание:

Представляет коллекцию вариантов точки маршрута бизнес-процесса.  
Только для точек вида [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Процедура ВыборСогласованияОбработкаВыбораВарианта(         ТочкаВыбораВарианта,          Результат)     Если Случай = 1 Тогда         Результат = ТочкаВыбораВарианта.Варианты.УпрощенныйВариант;     Иначе          Если Случай = 2 Тогда             Результат = ТочкаВыбораВарианта.Варианты.ПолныйВариант;         Иначе              Если Случай = 3 Тогда                 Результат = ТочкаВыбораВарианта.Варианты.ВариантДиректора;             Иначе                         Результат = ТочкаВыбораВарианта.Варианты.ВариантБезСогласования;             КонецЕсли;         КонецЕсли;     КонецЕсли; КонецПроцедуры |
