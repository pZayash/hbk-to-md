РазрешенныйCOMКласс (AllowedCOMClass)

ИмяФайла (FileName)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя файла, по которому создается объект методом [ПолучитьCOMОбъект](Глобальный_контекст.ПолучитьCOMОбъект.md) с пустым значением второго параметра. В общем случае строка используется как текстовое представление объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
