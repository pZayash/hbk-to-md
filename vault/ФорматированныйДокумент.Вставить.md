ФорматированныйДокумент (FormattedDocument)

Вставить (Insert)

Доступен, начиная с версии 8.2.

Синтаксис:

Вставить(<Закладка>, <Значение>, <ТипЭлемента>)

Параметры:

<Закладка> (обязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента.md).   
Определяет позицию вставки, в которую необходимо вставить элемент форматированного документа.

<Значение> (необязательный)

Тип: Произвольный.   
Содержит вставляемое в форматированный документ значение.  
В зависимости от значения параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.  
Если значение параметра [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), то в указанную позицию документа будет вставлено строковое представление переданного значения.  
Если значение [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md), то тип параметра должен быть [Картинка](Картинка.md), [ДвоичныеДанные](ДвоичныеДанные.md) или Строка, содержащая URL картинки. В противном случае вызывается исключение. Картинка будет вставлена в указанное место.  
Если значение параметра [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), то независимо от значения параметра в указанную позицию вставляется перевод строки.  
Значение по умолчанию: Неопределено.

<ТипЭлемента> (необязательный)

Тип: Тип.   
Указывает тип элемента, который нужно вставить. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Элементами коллекции могут быть:  
1) [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md)  
2) [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md)  
3) [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md)  
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Значение по умолчанию: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md).

Возвращаемое значение:

Тип: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md).   
Возращает элемент, вставленный в форматированный документ.

Описание:

Вставляет строку, картинку или перевод строки в указанную позицию форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Вставка картинки доступна только на сервере.  
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
