# МенеджерПользователейИнформационнойБазы.СоздатьПользователя

МенеджерПользователейИнформационнойБазы (InfoBaseUsersManager)

СоздатьПользователя (CreateUser)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьПользователя()

Возвращаемое значение:

Тип: [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).   

Описание:

Создает описание нового пользователя информационной базы. Для внесения пользователя в список следует после заполнения свойств нового пользователя использовать метод [Записать](ПользовательИнформационнойБазы.Записать_(InfoBaseUser.Write).md) объекта [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
