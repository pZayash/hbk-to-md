# ОбъектМетаданных: БизнесПроцесс.ПериодичностьНомера

ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

ПериодичностьНомера (NumberPeriodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПериодичностьНомераБизнесПроцесса](ПериодичностьНомераБизнесПроцесса_(BusinessProcessNumberPeriodicity).md), [ПериодичностьНомераДокумента](ПериодичностьНомераДокумента_(DocumentNumberPeriodicity).md).   
Устанавливает пределы контроля уникальности номеров и период повторяемости номеров (например, [Год](ПериодичностьНомераДокумента.Год_(DocumentNumberPeriodicity.Year).md), [Квартал](ПериодичностьНомераДокумента.Квартал_(DocumentNumberPeriodicity.Quarter).md), [Месяц](ПериодичностьНомераДокумента.Месяц_(DocumentNumberPeriodicity.Month).md), [День](ПериодичностьНомераДокумента.День_(DocumentNumberPeriodicity.Day).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
