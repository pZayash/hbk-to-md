МенеджерКалендарей (CalendarsManager)

ДобавитьСобытие (AddEvent)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ДобавитьСобытие(<ЛокальныйКлючКалендаря>, <Данные>)

Параметры:

<ЛокальныйКлючКалендаря> (обязательный)

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря.md).   
Локальный ключ календаря.

<Данные> (обязательный)

Тип: [ДанныеСобытияКалендаря](ДанныеСобытияКалендаря.md).   
Данные события календаря.

Возвращаемое значение:

Тип: [ЛокальныйКлючСобытияКалендаря](ЛокальныйКлючСобытияКалендаря.md).   

Описание:

Добавляет событие в календарь и возвращает его локальный ключ.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИзменениеСобытий](МенеджерКалендарей.ПоддерживаетсяИзменениеСобытий.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей.md), метод [ПоддерживаетсяИзменениеСобытий](МенеджерКалендарей.ПоддерживаетсяИзменениеСобытий.md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
