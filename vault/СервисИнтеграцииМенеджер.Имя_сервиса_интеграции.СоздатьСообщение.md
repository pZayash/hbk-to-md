СервисИнтеграцииМенеджер.<Имя сервиса интеграции> (IntegrationServiceManager.<Integration service name>)

СоздатьСообщение (CreateMessage)

Доступен, начиная с версии 8.3.17.

Синтаксис:

СоздатьСообщение(<ДатаУстаревания>)

Параметры:

<ДатаУстаревания> (необязательный)

Тип: Дата.   
Устанавливает дату устаревания создаваемого сообщения сервиса интеграции.  
Значение по умолчанию: "Текущая дата сеанса + 20 дней".  
  
Дата и время, после которого считается, что сообщение устарело и не должно быть обработано.  
  
Хранится и передается в UTC, при получении преобразуется в время в часовом поясе сеанса, при установке преобразуется из времени в часовом поясе сеанса.

Возвращаемое значение:

Тип: [СообщениеСервисаИнтеграции](СообщениеСервисаИнтеграции.md).   

Описание:

Создает сообщение сервиса интеграции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
