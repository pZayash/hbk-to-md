# СписокЗначений.Количество

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Список значений](Список_значений.md) › [СписокЗначений](СписокЗначений_(ValueList).md) › [Методы](СписокЗначений__Методы.md)

СписокЗначений (ValueList)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество элементов списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Сообщить("Итого типов цен: " + СписокТиповЦен.Количество()); |

Использование в версии:

Доступен, начиная с версии 8.0.
