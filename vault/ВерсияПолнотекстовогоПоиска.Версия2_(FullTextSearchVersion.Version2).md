ВерсияПолнотекстовогоПоиска (FullTextSearchVersion)

Версия2 (Version2)

Доступен, начиная с версии 8.3.22.

Описание:

Новая версия полнотекстового поиска (ППД).  
Внимание! Данный механизм не поддерживается в файловой информационной базе, в мобильной платформе и на автономном сервере.  
Внимание! При выборе данной версии после полной индексации должно оставаться не менее 10% свободного пространства. В противном случае, индексация полнотекстового поиска выполнена не будет.

Использование в версии:

Доступен, начиная с версии 8.3.22.
