Форма (Form)

ЗакрыватьПриЗакрытииВладельца (CloseOnOwnerClose)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что форма должна закрываться при закрытии формы-владельца.  
Ложь - остается открытой.

Доступность:

Толстый клиент.

См. также:

[Форма](Форма.md), свойство [ВладелецФормы](Форма.ВладелецФормы.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
