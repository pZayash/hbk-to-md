Соединение с агентом сервера (IServerAgentConnection)

GetInfoBaseLocks (GetInfoBaseLocks)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBaseLocks(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого необходимо получить список блокировок.

<ИнформационнаяБаза> (обязательный)

Тип: [Описание информационной базы](Описание_информационной_базы_(IInfoBaseShort).md).   
Описание информационной базы, для которой необходимо получить список блокировок.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив блокировок объектов информационной базы. Каждая блокировка представлена объектом с интерфейсом [Блокировка](Блокировка_(IObjectLock).md).

Описание:

Получает массив блокировок объектов информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
