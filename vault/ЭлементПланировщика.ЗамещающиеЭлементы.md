ЭлементПланировщика (PlannerItem)

ЗамещающиеЭлементы (ReplacementItems)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗамещающихЭлементовПланировщика](КоллекцияЗамещающихЭлементовПланировщика.md), Неопределено.   
Коллекция элементов, которые используются для того, чтобы заменить повторенные элементы для случая, когда задано свойство [Расписание](ЭлементПланировщика.Расписание.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
