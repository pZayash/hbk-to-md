ОбластьПостроенияДиаграммыГанта (GanttChartPlotArea)

РазмещениеТекста (TextPlacement)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ТипРазмещенияТекстаДиаграммыГанта](ТипРазмещенияТекстаДиаграммыГанта_(GanttChartTextPlacementType).md).   
Указывает вариант отображения текста точки (значения, интервала). Значение по умолчанию - [Авто](ТипРазмещенияТекстаДиаграммыГанта.Авто_(GanttChartTextPlacementType.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
