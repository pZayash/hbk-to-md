ИнтернетПочтовоеВложение (InternetMailAttachment)

СпособКодирования (EncodingMode)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [СпособКодированияИнтернетПочтовогоВложения](СпособКодированияИнтернетПочтовогоВложения.md).   
Содержит способ кодирования вложений сообщения. По умолчанию используется метод [MIME](СпособКодированияИнтернетПочтовогоВложения.MIME.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
