ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

ВыборГруппИЭлементов (ChoiceFoldersAndItems)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов.md).   
Предоставляет доступ к управлению режимом выбора групп и элементов справочника: любой элемент, только группы или только элементы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
