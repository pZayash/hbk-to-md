ПараметрыПривязкиККомпьютеруПолученияЛицензий (LicenseAcquisitionComputerFingerprintParameters)

СписокОборудования (EquipmentList)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив.md).   
Фиксированный массив элементов типа Строка. Все параметры устройств компьютера, участвующие в привязке лицензии, в формате:  
"имя\_устройства: параметры\_устройства".

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
