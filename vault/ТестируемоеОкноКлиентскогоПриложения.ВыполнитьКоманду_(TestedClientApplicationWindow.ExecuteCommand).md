ВыполнитьКоманду (ExecuteCommand)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВыполнитьКоманду(<Команда>)

Параметры:

<Команда> (обязательный)

Тип: [Строка](lang__def_String.md).

Описание:

Выполняет указанную команду.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
