# ЧтениеУзловDOM.ПерейтиКСодержимому

ЧтениеУзловDOM (DOMNodeReader)

ПерейтиКСодержимому (MoveToContent)

Доступен, начиная с версии 8.2.

Синтаксис:

ПерейтиКСодержимому()

Возвращаемое значение:

Тип: [ТипУзлаXML](ТипУзлаXML_(XMLNodeType).md).   
Значение узла, в который был перемещен текущий узел; [Ничего](ТипУзлаXML.Ничего_(XMLNodeType.None).md) - в случае достижения конца источника данных XML.

Описание:

Проверяет является ли текущий узел "содержимым" ([Текст](ТипУзлаXML.Текст_(XMLNodeType.Text).md), [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md), [КонецЭлемента](ТипУзлаXML.КонецЭлемента_(XMLNodeType.EndElement).md), [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md) или [КонецСущности](ТипУзлаXML.КонецСущности_(XMLNodeType.EndEntity).md)).  
Если текущий узел не является "содержимым", то выполняется пропуск узлов до "содержимого" или же до конца источника данных XML.  
Возвращает значение узла, в который был перемещен текущий узел, [Ничего](ТипУзлаXML.Ничего_(XMLNodeType.None).md) - в случае достижения конца источника данных XML.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Будут пропущены следующие типы узлов: [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML_(XMLNodeType.XMLDeclaration).md), [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки_(XMLNodeType.ProcessingInstruction).md), [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md), [Комментарий](ТипУзлаXML.Комментарий_(XMLNodeType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.2.
