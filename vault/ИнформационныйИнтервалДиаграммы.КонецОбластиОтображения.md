ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

КонецОбластиОтображения (DisplayAreaEnd)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Положение конца области отображения информационного интервала относительно диаграммы.  
Может содержать значения от 0 до 1.  
Значение по умолчанию - 1.  
Для интервалов значений свойство отсчитывается вдоль оси точек, а для интервалов точек - вдоль оси значений.  
Например, для интервала значений диаграммы, размещенной горизонтально:

- 0 - конец области интервала располагается у левой границы области диаграммы;
- 0,5 - конец области интервала располагается в середине области диаграммы;
- 1 - конец области интервала располагается у правой границы области диаграммы.

Если значение меньше или равно свойству [НачалоОбластиОтображения](ИнформационныйИнтервалДиаграммы.НачалоОбластиОтображения.md), интервал отображаться не будет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
