FTPФайл (FTPFile)

ЭтоКаталог (IsDirectory)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоКаталог()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект соответствует каталогу; [Ложь](lang__def_BooleanFalse.md) - объект соответствует файлу.

Описание:

Определяет, соответствует ли файловый объект каталогу или файлу.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применим только к файлам или каталогам, которые реально существуют.

Пример:

| --- |
| Если Не ВыбФайл.ЭтоКаталог() Тогда     Предупреждение(НСтр("ru = 'Укажите каталог!'; en = 'Define a catalog!'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
