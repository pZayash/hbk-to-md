# ПланыВидовРасчетаМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПланВидовРасчетаМенеджер.\<Имя плана видов расчета\>](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов расчета.  
Возможно обращение к менеджеру плана видов расчета посредством оператора [...]. В качестве аргумента передается имя плана видов расчета, как оно задано в конфигурации.  

Свойства:

[\<Имя плана видов расчета\> (\<Chart of calculation types name\>)](ПланыВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartsOfCalculationTypesManager.Chart_of_calculation_types_name).md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПланыВидовРасчетаМенеджер.ТипВсеСсылки_(ChartsOfCalculationTypesManager.AllRefsType).md)  

Описание:

Предназначен для управления планами видов расчета и предоставляет доступ к значениям типа [ПланВидовРасчетаМенеджер.](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [ПланыВидовРасчета](Глобальный_контекст.ПланыВидовРасчета_(Global_context.ChartsOfCalculationTypes).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ПланыВидовРасчета](Глобальный_контекст.ПланыВидовРасчета_(Global_context.ChartsOfCalculationTypes).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
