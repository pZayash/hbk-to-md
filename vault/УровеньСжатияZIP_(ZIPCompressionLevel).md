**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [ZIP](ZIP.md)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [УровеньСжатияФайлаАрхива](УровеньСжатияФайлаАрхива_(ArchiveFileCompressionLevel).md)

Доступен, начиная с версии 8.0.

Значения

[Максимальный (Maximum)](УровеньСжатияZIP.Максимальный_(ZIPCompressionLevel.Maximum).md)  
[Минимальный (Minimum)](УровеньСжатияZIP.Минимальный_(ZIPCompressionLevel.Minimum).md)  
[Оптимальный (Optimal)](УровеньСжатияZIP.Оптимальный_(ZIPCompressionLevel.Optimal).md)  

Описание:

Содержит варианты степени сжатия ZIP файлов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЗаписьZipФайла](ЗаписьZipФайла_(ZipFileWriter).md), метод [Открыть](ЗаписьZipФайла.Открыть_(ZipFileWriter.Open).md)

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.

Описание изменено в версии 8.3.26.
