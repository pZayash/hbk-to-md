ТабличноеПоле (TableBox)

ПередНачаломДобавления (BeforeAddRow)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), строка не будет добавлена.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

Описание:

Возникает перед началом интерактивного добавления строки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
