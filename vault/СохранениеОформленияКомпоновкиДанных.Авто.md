СохранениеОформленияКомпоновкиДанных (SaveDataCompositionAppearance)

Авто (Auto)

Доступен, начиная с версии 8.3.16.

Описание:

Если параметр <ТипДиаграммы.КлючСохраненияЦветов> установлен, то используется значение [ПоКлючуДляПользователя](СохранениеОформленияКомпоновкиДанных.ПоКлючуДляПользователя.md).  
Иначе, в режиме совместимости [Версия8\_3\_15](РежимСовместимости.Версия8_3_15.md) используется значение [НеИспользовать](СохранениеОформленияКомпоновкиДанных.НеИспользовать.md), а если режим совместимости не установлен - [ДляПользователя](СохранениеОформленияКомпоновкиДанных.ДляПользователя.md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
