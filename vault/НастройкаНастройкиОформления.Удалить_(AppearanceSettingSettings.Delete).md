НастройкаНастройкиОформления (AppearanceSettingSettings)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияНастройкойНастройкиОформления](ЭлементУправленияНастройкойНастройкиОформления_(AppearanceSettingsSettingControl).md).   
Удаляемый элемент.

Описание:

Удаляет элемент управления настройкой оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
