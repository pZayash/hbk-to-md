РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

ПериодДействияНачало (BegOfActionPeriod)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата начала периода действия записи.  
Период действия записи (задаваемый датой начала и датой окончания) может не совпадать с фактическим периодом действия, который в общем случае - набор нескольких непересекающихся временных интервалов (см. [ФактическийПериодДействия](РегистрРасчетаВыборка.Имя_регистра_расчета.ФактическийПериодДействия_(CalculationRegisterSelection.Calculation_register_name.ActualActionPeriod).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определен для регистров расчета с признаком "Период действия".
