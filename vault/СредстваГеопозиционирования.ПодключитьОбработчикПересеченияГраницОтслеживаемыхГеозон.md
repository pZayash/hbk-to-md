СредстваГеопозиционирования (LocationTools)

ПодключитьОбработчикПересеченияГраницОтслеживаемыхГеозон (AttachMonitoredGeofencesBordersCrossingDetectionHandler)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПодключитьОбработчикПересеченияГраницОтслеживаемыхГеозон(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Строка.   
Описание процедуры, которая будет вызвана после завершения операции.  
Процедура должна содержать следующие параметры:

- <Геозона> - тип [Геозона](Геозона.md);
- <ПоложениеОтносительноГеозоны> - Значение [Внутри](ПоложениеОтносительноГеозоны.Внутри.md) означает, что пользователь вошел в геозону, [Снаружи](ПоложениеОтносительноГеозоны.Снаружи.md) - вышел.
- <Дополнительный параметр> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md)

Описание:

Выполняет подключение обработчика входа/выхода из геозон, добавленных с помощью метода [ВключитьОтслеживаниеГеозон](СредстваГеопозиционирования.ВключитьОтслеживаниеГеозон.md) для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
