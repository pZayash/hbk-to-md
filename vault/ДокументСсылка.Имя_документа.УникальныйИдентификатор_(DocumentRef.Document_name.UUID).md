# ДокументСсылка.<Имя документа>.УникальныйИдентификатор

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументСсылка.<Имя документа>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md) › [Методы](ДокументСсылка.Имя_документа__Методы.md)

ДокументСсылка.<Имя документа> (DocumentRef.<Document name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ДокументМенеджер.Имя_документа.ПолучитьСсылку_(DocumentManager.Document_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ДокументМенеджер.\<Имя документа\>](ДокументМенеджер.Имя_документа_(DocumentManager.Document_name).md), метод [ПолучитьСсылку](ДокументМенеджер.Имя_документа.ПолучитьСсылку_(DocumentManager.Document_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
