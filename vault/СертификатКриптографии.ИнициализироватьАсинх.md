СертификатКриптографии (CryptoCertificate)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Исходные данные сертификата в виде пути к файлу или потока.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - созданный объект типа [СертификатКриптографии](СертификатКриптографии.md).

Описание:

Асинхронная функция. Начинает инициализацию объекта [СертификатКриптографии](СертификатКриптографии.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии.md), метод [НачатьИнициализацию](СертификатКриптографии.НачатьИнициализацию.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
