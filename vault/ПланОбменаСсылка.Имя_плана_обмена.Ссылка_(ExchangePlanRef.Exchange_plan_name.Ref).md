ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   
Содержит ссылку на узел. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Так как [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)> и [ПланОбменаВыборка.](ПланОбменаВыборка.Имя_плана_обмена_(ExchangePlanSelection.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаВыборка.Имя_плана_обмена_(ExchangePlanSelection.Exchange_plan_name).md)> имеют одноименное свойство, то при обращении в программном модуле к данному свойству переданного в процедуру или функцию параметра не потребуется определять тип параметра: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)> или [ПланОбменаВыборка.](ПланОбменаВыборка.Имя_плана_обмена_(ExchangePlanSelection.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаВыборка.Имя_плана_обмена_(ExchangePlanSelection.Exchange_plan_name).md)>.

Доступен, начиная с версии 8.0.
