МенеджерПользователейИнформационнойБазы (InfoBaseUsersManager)

ПолучитьПользователей (GetUsers)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьПользователей()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает массив, элементами которого являются объекты [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данный метод доступен пользователям, имеющим административные права.

Использование в версии:

Доступен, начиная с версии 8.0.
