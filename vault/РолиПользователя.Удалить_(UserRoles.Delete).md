РолиПользователя (UserRoles)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Роль>)

Параметры:

<Роль> (обязательный)

Тип: [ОбъектМетаданных: Роль](ОбъектМетаданных_Роль_(MetadataObject_Role).md).   
Удаляемая роль.

Описание:

Удаляет указанную роль из коллекции используемых ролей пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
