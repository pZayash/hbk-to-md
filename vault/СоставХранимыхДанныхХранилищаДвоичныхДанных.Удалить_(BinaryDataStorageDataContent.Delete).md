СоставХранимыхДанныхХранилищаДвоичныхДанных (BinaryDataStorageDataContent)

Удалить (Delete)

Доступен, начиная с версии 8.3.26.

Вариант синтаксиса: По элементу

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементСоставаХранимыхДанныхХранилищаДвоичныхДанных](ЭлементСоставаХранимыхДанныхХранилищаДвоичныхДанных_(BinaryDataStorageDataContentItem).md).   
Удаляемый элемент состава хранилища двоичных данных.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента коллекции состава, который необходимо удалить.

Описание:

Удаляет элемент коллекции состава хранимых данных хранилища двоичных данных.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.

Описание изменено в версии 8.3.27.
