Описание:

Содержит параметры диалога помещения файлов.  
Используется в качестве параметра <ПараметрыДиалога> методов [НачатьПомещениеФайлаНаСервер](Глобальный_контекст.НачатьПомещениеФайлаНаСервер.md) и [НачатьПомещениеФайловНаСервер](Глобальный_контекст.НачатьПомещениеФайловНаСервер.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
