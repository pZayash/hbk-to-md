ПланВидовРасчетаВыборка.<Имя плана видов расчета> (ChartOfCalculationTypesSelection.<Chart of calculation types name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: Булево.   
Истина - следующий вид расчета выбран; Ложь - достигнут конец выборки.

Описание:

Получает следующий вид расчета из выборки. Для обхода плана видов расчета нужно после получения выборки вызвать данный метод для позиционирования на первый элемент и далее вызывать до тех пор, пока не будет возвращено значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Выборка = ПланыВидовРасчета.УдержанияОрганизаций.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка выборки      // ...  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
