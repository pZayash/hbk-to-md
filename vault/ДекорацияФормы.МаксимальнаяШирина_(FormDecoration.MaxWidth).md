ДекорацияФормы (FormDecoration)

МаксимальнаяШирина (MaxWidth)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную ширину элемента.  
Используется, если свойство [АвтоМаксимальнаяШирина](ДекорацияФормы.АвтоМаксимальнаяШирина_(FormDecoration.AutoMaxWidth).md) установлено в значение [Ложь](lang__def_BooleanFalse.md).  
Задается в символах.  
Значение 0 означает, что максимальная ширина не ограничена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
