Расширение поля формы для поля планировщика (Form field extension for a planner)

СвернутьЭлементИзмерения (CollapseDimensionItem)

Доступен, начиная с версии 8.3.17.

Синтаксис:

СвернутьЭлементИзмерения(<ЗначенияИзмерений>, <СПодчиненными>)

Параметры:

<ЗначенияИзмерений> (обязательный)

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Фиксированное соответствие, однозначно определяющее элемент измерения, который нужно свернуть.   
В качестве ключа используется [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения.

<СПодчиненными> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость разворачивания элемента измерения вместе с подчиненными элементами измерения. [Истина](lang__def_BooleanTrue.md) – развернуть вместе с подчиненными.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Сворачивает элемент измерения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
