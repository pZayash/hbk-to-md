ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ПараметрКартинки (PictureParameter)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя параметра, из которого картинка получается с помощью методов [Вывести](ТабличныйДокумент.Вывести_(SpreadsheetDocument.Put).md) и [Присоединить](ТабличныйДокумент.Присоединить_(SpreadsheetDocument.Join).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
