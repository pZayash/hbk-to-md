Расширение поля формы для поля ввода (Form field extension for a text box)

ОтображатьФлажкиВВыпадающемСпискеПриВводеМножественныхЗначений (ShowCheckBoxesInDropListWhenInputMultipleValues)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак отображения флажков в выпадающем списке, если поле ввода редактирует данные типа [СписокЗначений](СписокЗначений_(ValueList).md) или [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md), то есть поле ввода находится в режиме ввода множественных значений. В остальных случаях значения данного свойства игнорируется.

- [Истина](lang__def_BooleanTrue.md) – флажки отображаются.
- [Ложь](lang__def_BooleanFalse.md) – флажки не отображаются.
- [Неопределено](lang__def_Undefined.md) – отображение флажков определяется автоматически:
  - Если [РазрешитьДублированиеМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.РазрешитьДублированиеМножественныхЗначений_(Form_field_extension_for_a_text_box.AllowMultipleValuesDuplicates).md) имеет значение [Ложь](lang__def_BooleanFalse.md) - флажки отображаются,
  - Если [РазрешитьДублированиеМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.РазрешитьДублированиеМножественныхЗначений_(Form_field_extension_for_a_text_box.AllowMultipleValuesDuplicates).md) имеет значение [Истина](lang__def_BooleanTrue.md) - флажки не отображаются.

Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
