Профиль безопасности (ISecurityProfile)

SafeModeProfile (SafeModeProfile)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, может ли профиль использоваться в качестве профиля безопасного режима: его имя может быть указано в конфигурациях при использовании внешних отчетов, обработок и расширений конфигурации.   
[Истина](lang__def_BooleanTrue.md) – профиль можно использовать. [Ложь](lang__def_BooleanFalse.md) – профиль нельзя использовать.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
