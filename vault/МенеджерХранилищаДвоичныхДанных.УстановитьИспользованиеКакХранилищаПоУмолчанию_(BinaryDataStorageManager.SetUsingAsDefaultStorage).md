МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьИспользованиеКакХранилищаПоУмолчанию (SetUsingAsDefaultStorage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьИспользованиеКакХранилищаПоУмолчанию(<Использовать>)

Параметры:

<Использовать> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Использовать встроенное хранилище как хранилище по умолчанию.

Описание:

Устанавливает или отключает режим хранилища по умолчанию.  
Один и тот же объект метаданных не может быть настроен для использования с разными хранилищами двоичных данных.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены.  
Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
