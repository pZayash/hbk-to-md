ПолеВыбора (ComboBox)

ОбработкаВыбора (ChoiceProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаВыбора(<ВыбранноеЗначение>, <СтандартнаяОбработка>)

Параметры:

<ВыбранноеЗначение>

Тип: Произвольный.   
Выбранное значение.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается после осуществления выбора значения из списка выбора, но до помещения выбранного значения в элемент управления. В процедуре-обработчике события можно отказаться от выбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
