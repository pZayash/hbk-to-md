ТестируемаяКнопкаФормы (TestedFormButton)

ТекущееПометка (CurrentCheck)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ТекущееПометка()

Возвращаемое значение:

Истина - если кнопка нажата или рядом с ней отображается пометка.  
Ложь - в противном случае.

Описание:

Проверяет, нажата ли кнопка.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
