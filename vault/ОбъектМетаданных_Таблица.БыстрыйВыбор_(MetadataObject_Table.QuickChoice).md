ОбъектМетаданных: Таблица (MetadataObject: Table)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство доступно, если способ выбора равен "Обоими способами". Устанавливает способ выбора в полях ввода по умолчанию.  
[Истина](lang__def_BooleanTrue.md) - используется быстрый выбор.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
