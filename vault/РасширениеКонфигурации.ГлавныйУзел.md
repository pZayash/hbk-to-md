РасширениеКонфигурации (ConfigurationExtension)

ГлавныйУзел (MasterNode)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Неопределено, [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>.   
Содержит узел распределенной информационной базы, в котором создано данное расширение конфигурации. Если текущая информационная база не является узлом распределенной информационной базы или расширение создано локально, то содержит Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации.md), свойство [ИспользуетсяВРаспределеннойИнформационнойБазе](РасширениеКонфигурации.ИспользуетсяВРаспределеннойИнформационнойБазе.md)  
[ОбъектМетаданных: ПланОбмена](ОбъектМетаданных_ПланОбмена.md), свойство [ВключатьРасширенияКонфигурации](ОбъектМетаданных_ПланОбмена.ВключатьРасширенияКонфигурации.md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
