# ЭлементDOM.УстановитьУзелАтрибутаИдентификатора

ЭлементDOM (DOMElement)

УстановитьУзелАтрибутаИдентификатора (SetIdAttributeNode)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьУзелАтрибутаИдентификатора(<Атрибут>, <ЭтоИД>)

Параметры:

<Атрибут> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   
Атрибут элемента.

<ЭтоИД> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак идентификатора.

Описание:

Устанавливает атрибуту признак идентификатора.  
Если значение параметра <ЭтоИД> = [Истина](lang__def_BooleanTrue.md), указанный атрибут становится "идентификатором, определенным пользователем".  
Причины вызова исключений:

- данный узел в режиме Только для чтения;
- указанный атрибут не является атрибутом данного элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не меняет установленной в документе схемы.

Использование в версии:

Доступен, начиная с версии 8.1.
