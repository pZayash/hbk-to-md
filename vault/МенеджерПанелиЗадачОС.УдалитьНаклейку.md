МенеджерПанелиЗадачОС (OSTaskbarManager)

УдалитьНаклейку (RemoveBadge)

Доступен, начиная с версии 8.3.8.

Синтаксис:

УдалитьНаклейку()

Описание:

Удаляет значок уведомления с кнопки программы на панели задач.  
Работает под Windows 7 и старше, а также Ubuntu Unity (удаляет установленное число на кнопке приложения в Unity Launcher).   
При работе в других ОС не производит никаких действий.

Доступность:

Тонкий клиент, толстый клиент.

Пример:

|  |
| --- |
| ПанельЗадачОС.УдалитьЗначок(); |

Использование в версии:

Доступен, начиная с версии 8.3.8.
