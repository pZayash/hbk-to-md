ИзмеренияПостроителяЗапроса (QueryBuilderDimensions)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Тип>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным для измерения.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя измерения.

<Тип> (необязательный)

Тип: [ТипИзмеренияПостроителяЗапроса](ТипИзмеренияПостроителяЗапроса_(QueryBuilderDimensionType).md).   
Тип измерения.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить измерение.

Возвращаемое значение:

Тип: [ИзмерениеПостроителяЗапроса](ИзмерениеПостроителяЗапроса_(QueryBuilderDimension).md).   

Описание:

Вставляет измерение в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
