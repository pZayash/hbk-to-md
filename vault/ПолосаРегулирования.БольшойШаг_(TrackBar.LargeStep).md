ПолосаРегулирования (TrackBar)

БольшойШаг (LargeStep)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит число, на которое изменится значение элемента управления при перемещении на один большой шаг. Перемещение большими шагами осуществляется при нажатии клавиш PgUp и PgDn.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
