ПалитраЦветовДиаграммы (ChartColorPalette)

Градиентная (Gradient)

Доступен, начиная с версии 8.3.10.

Описание:

Градиентная палитра. Используются цвета из свойств диаграммы [ЦветНачалаГрадиентнойПалитры](Диаграмма.ЦветНачалаГрадиентнойПалитры_(Chart.GradientPaletteStartColor).md) и [ЦветКонцаГрадиентнойПалитры](Диаграмма.ЦветКонцаГрадиентнойПалитры_(Chart.GradientPaletteEndColor).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
