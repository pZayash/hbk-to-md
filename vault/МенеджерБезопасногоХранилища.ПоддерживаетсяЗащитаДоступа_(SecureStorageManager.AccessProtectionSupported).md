МенеджерБезопасногоХранилища (SecureStorageManager)

ПоддерживаетсяЗащитаДоступа (AccessProtectionSupported)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПоддерживаетсяЗащитаДоступа(<СпособЗащитыДоступа>, <СпособПроверкиПользователя>)

Параметры:

<СпособЗащитыДоступа> (обязательный)

Тип: [СпособЗащитыДоступаБезопасногоХранилища](СпособЗащитыДоступаБезопасногоХранилища_(SecureStorageAccessProtectionMethod).md).   
Используемый способ защиты доступа.

<СпособПроверкиПользователя> (необязательный)

Тип: [СпособДополнительнойПроверкиПользователя](СпособДополнительнойПроверкиПользователя_(AdditionalUserVerificationMethod).md).   
Используется только если <СпособБиометрическойПроверки> равен [ТребуетсяДополнительнаяПроверкаПользователя](СпособЗащитыДоступаБезопасногоХранилища.ТребуетсяДополнительнаяПроверкаПользователя_(SecureStorageAccessProtectionMethod.AdditionalUserVerificationRequired).md), иначе игнорируется.  
Значение по умолчанию: [БиометрическаяИлиВводПароля](СпособДополнительнойПроверкиПользователя.БиометрическаяИлиВводПароля_(AdditionalUserVerificationMethod.BiometricsOrPassword).md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Проверяет наличие возможности хранить данные в безопасном хранилище, защищенном выбранным способом защиты доступа.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
