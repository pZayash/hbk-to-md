Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ЭтоГруппа (IsFolder)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет вид нового объекта - группа или элемент.

Примечание:

Используется для форм иерархических объектов (справочников, планов видов характеристик).

Использование в версии:

Доступен, начиная с версии 8.2.
