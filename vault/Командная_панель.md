# Командная панель

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Элементы управления](Элементы_управления.md)

Элемент управления [КоманднаяПанель](КоманднаяПанель_(CommandBar).md) предназначен для размещения в форме набора команд, с помощью которых производится управление информацией, размещенной в форме. Командная панель представляет собой древовидную структуру кнопок командной панели. Нажатие мышью на любую из кнопок вызывает немедленное выполнение связанной с этой кнопкой команды.
