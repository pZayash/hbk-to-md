# Командная панель

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Интерфейс-2.md">Интерфейс</a> › [Элементы управления](Элементы_управления.md)

Элемент управления [КоманднаяПанель](КоманднаяПанель.md) предназначен для размещения в форме набора команд, с помощью которых производится управление информацией, размещенной в форме. Командная панель представляет собой древовидную структуру кнопок командной панели. Нажатие мышью на любую из кнопок вызывает немедленное выполнение связанной с этой кнопкой команды.
