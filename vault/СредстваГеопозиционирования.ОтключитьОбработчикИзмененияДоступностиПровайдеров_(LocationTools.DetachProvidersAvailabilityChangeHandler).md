СредстваГеопозиционирования (LocationTools)

ОтключитьОбработчикИзмененияДоступностиПровайдеров (DetachProvidersAvailabilityChangeHandler)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОтключитьОбработчикИзмененияДоступностиПровайдеров(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), [Строка](lang__def_String.md).   
Имя процедуры, подключаемой в качестве обработчика изменения доступности провайдеров.  
Процедура должна иметь следующие параметры.

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Отключает обработчик изменения статусов провайдеров геопозиционирования, присутствующих в системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
