# Расширение формы элемента плана счетов.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [Расширение формы элемента плана счетов](Расширение_формы_элемента_плана_счетов_(Chart_of_accounts_item_form_extension).md) › [События](Расширение_формы_элемента_плана_счетов__События.md)

Расширение формы элемента плана счетов (Chart of accounts item form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланСчетовОбъект.Имя_плана_счетов.ПриЗаписи_(ChartOfAccountsObject.Chart_of_accounts_name.OnWrite).md) объекта [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
