Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

Вывод (Output)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеВывода](ИспользованиеВывода.md).   
Ограничение вывода на печать, сохранения, работы с буфером обмена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Может иметь значение, отличное от значения свойства Вывод документа, с которым это поле связано.

Использование в версии:

Доступен, начиная с версии 8.3.21.
