<!-- signature:end -->

Расширение формы объекта бизнес-процесс (Business process object form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись бизнес-процесса выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](БизнесПроцессОбъект.Имя_бизнес-процесса.ПриЗаписи.md) объекта [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

См. также:

[БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса.md), событие [ПриЗаписи](БизнесПроцессОбъект.Имя_бизнес-процесса.ПриЗаписи.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
