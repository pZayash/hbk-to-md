ПолеСписка (ListBox)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность редактирования поля списка. Если значение свойства [Истина](lang__def_BooleanTrue.md), то возможен только просмотр поля списка.

Доступность:

Толстый клиент.

Примечание:

В отличие от свойства [Доступность](ПолеСписка.Доступность_(ListBox.Enabled).md), если свойство ТолькоПросмотр равно [Истина](lang__def_BooleanTrue.md), становится невозможным редактирование поля списка, однако на остальную работу с полем это не влияет. Например, можно выделить строковое представление содержимого поля списка и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
