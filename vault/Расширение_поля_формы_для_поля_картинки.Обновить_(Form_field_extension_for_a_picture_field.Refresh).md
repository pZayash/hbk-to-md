# Расширение поля формы для поля картинки.Обновить

Расширение поля формы для поля картинки (Form field extension for a picture field)

Обновить (Refresh)

Доступен, начиная с версии 8.2.

Синтаксис:

Обновить()

Описание:

Осуществляет повторное получение картинки, если поле связано с данными типа [Строка](lang__def_String.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
