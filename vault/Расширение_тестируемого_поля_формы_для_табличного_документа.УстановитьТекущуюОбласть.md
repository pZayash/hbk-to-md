Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

УстановитьТекущуюОбласть (SetCurrentArea)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УстановитьТекущуюОбласть(<Область>)

Параметры:

<Область> (необязательный)

Тип: Строка.   
Имя области или адрес в принятом в табличном документе формате, например "R1C1:R2C2".

Описание:

Устанавливает текущую область в табличном документе.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
