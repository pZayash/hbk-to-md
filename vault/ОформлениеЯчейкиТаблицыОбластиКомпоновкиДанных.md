Описание:

Содержит объекты, описывающие оформление ячейки таблицы.  
Для оформления ячейки таблицы используются следующие элементы оформления:  
1. ЦветФона (BackColor) - тип значения [Цвет](Цвет.md);  
2. ЦветТекста (TextColor) - тип значения [Цвет](Цвет.md);  
3. ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет.md);  
4. СтильГраницы (BorderStyle) - тип значения [Линия](Линия.md);  
5. СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия.md);  
6. СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия.md);  
7. СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия.md);  
8. СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия.md);  
9. Шрифт (Font) - тип значения [Шрифт](Шрифт.md);  
10. Отступ (Indent) - тип значения Число;  
11. ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md);  
12. ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение.md);  
13. Размещение (Placement) - размещение текста - тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных.md);  
14. ОриентацияТекста (TextOrientation) - тип значения Число;  
15. ВыделятьОтрицательные (MarkNegatives) - тип значения Булево;  
16. МинимальнаяШирина (MinWidth) - тип значения Число;  
17. МаксимальнаяШирина (MaxWidth) - тип значения Число;  
18. МинимальнаяВысота (MinHeight) - тип значения Число;  
19. МаксимальнаяВысота (MaxHeight) - тип значения Число;  
20. ОбъединятьПоГоризонтали (HorizontalMerge) - тип значения Булево;  
21. ОбъединятьПоВертикали (VerticalMerge) - тип значения Булево;  
22. ГоризонтальныйУровень (HorizontalLevel) - тип значения Число;  
23. ВертикальныйУровень (VerticalLevel) - тип значения Число;  
24. Расшифровка (Details) - тип [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных.md);  
25. РежимИзмененияРазмераКолонки (ColumnSizeChangeMode) - возможность изменять размер колонки ячейки [РежимИзмененияРазмера](РежимИзмененияРазмера.md);  
26. ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную - тип значения Булево;  
27. ФиксироватьСлева (FixLeft) - признак того, что ячейку нужно фиксировать слева Булево;  
28. ФиксироватьСверху (FixTop) - признак того, что ячейку нужно фиксировать сверху Булево;  
29. РастягиватьПоГоризонтали (HorizontalStretch) - тип значения Булево. Указывает, что ширина текущей колонки рассчитывается автоматически;  
30. ВесовойКоэффициентШирины (WidthWeightFactor) - тип значения Число. Весовой коэффициент для автоматического расчета ширины колонки;  
31. Картинка (Picture) - тип [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных.md) или [Картинка](Картинка.md). Если содержит параметр, то картинка получается из выражения параметра.  
32. РазмерКартинки (PictureSize) - тип [РазмерКартинки](РазмерКартинки.md). Способ определения размера картинки;  
33. ПоложениеТекстаОтносительноКартинки (PictureSizeTextPositionRelativeToPicture) - тип [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки.md). Положение текста относительно картинки;  
34. ГоризонтальноеПоложениеКартинки (PictureHorizontalAlign)- тип [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md). Размещение картинки по горизонтали;  
35. ВертикальноеПоложениеКартинки (PictureVerticalAlign) - тип [ВертикальноеПоложение](ВертикальноеПоложение.md). Размещение картинки по вертикали;  
36. ОтступСлева (LeftMargin) - тип значения Число;  
37. ОтступСправа (RightMargin) - тип значения Число;  
38. ОтступСверху (TopMargin) - тип значения Число;  
39. ОтступСнизу (BottomMargin) - тип значения Число.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: TableCellAppearance.
