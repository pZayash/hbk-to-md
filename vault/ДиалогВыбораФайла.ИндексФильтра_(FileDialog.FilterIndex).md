Доступен, начиная с версии 8.0.

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).

Тонкий клиент, веб-клиент, толстый клиент.

Не поддерживается в мобильном приложении для Windows.

Использование в версии:

Доступен, начиная с версии 8.0.
