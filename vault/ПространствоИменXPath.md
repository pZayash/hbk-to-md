Описание:

Специализированный узел DOM. Может появляться только в результате вычисления выражения XPath. Представляет собой узел объявления соответствия URI пространства имен. Узел пространства имен является узлом только для чтения. Все попытки изменения, добавления и удаления узла приводят к вызову исключения.  
Все атрибуты узла DOM, не описанные ниже, имеют либо значение Ложь, либо Неопределено:

- [ПространствоИменXPath.ДокументВладелец](ПространствоИменXPath.ДокументВладелец.md) - (одноименные атрибуты разных объектов) соответствует документу-владельцу элемента-владельца данного узла;
- [ПространствоИменXPath.ИмяУзла](ПространствоИменXPath.ИмяУзла.md) - (одноименные атрибуты разных объектов) всегда равен "#namespace";
- [ПространствоИменXPath.Префикс](ПространствоИменXPath.Префикс.md) - (одноименные атрибуты разных объектов) префикс пространства имен, представляемого данным узлом;
- [ПространствоИменXPath.ЛокальноеИмя](ПространствоИменXPath.ЛокальноеИмя.md) - (одноименные атрибуты разных объектов) то же, что и [ПространствоИменXPath.Префикс](ПространствоИменXPath.Префикс.md);
- [ПространствоИменXPath.ТипУзла](ПространствоИменXPath.ТипУзла.md) - (одноименные атрибуты разных объектов) [ПространствоИменXPath](ПространствоИменXPath.md);
- [ПространствоИменXPath.URIПространстваИмен](ПространствоИменXPath.URIПространстваИмен.md) - URI пространства имен, представляемого данным узлом;
- [ПространствоИменXPath.ЗначениеУзла](ПространствоИменXPath.ЗначениеУзла.md) - (одноименные атрибуты разных объектов) то же, что и [ПространствоИменXPath.URIПространстваИмен](ПространствоИменXPath.URIПространстваИмен.md) (одноименные атрибуты разных объектов);
- [АдаптироватьУзел](ДокументDOM.АдаптироватьУзел.md), [ПространствоИменXPath.КлонироватьУзел](ПространствоИменXPath.КлонироватьУзел.md) и [ИмпортироватьУзел](ДокументDOM.ИмпортироватьУзел.md) - не поддерживаются и вызывают исключения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
