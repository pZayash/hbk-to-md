Форма (Form)

ОткрытьСправкуФормы (OpenFormHelp)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьСправкуФормы()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - окно со справочной информацией по форме открыто; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает окно справочной информации по форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ОткрытьСправку(); |

Использование в версии:

Доступен, начиная с версии 8.0.
