ИдентификаторПриложенияСистемыВзаимодействия (CollaborationSystemApplicationID)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ИдентификаторПриложенияСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строковое представление идентификатора.

Описание:

Создает идентификатор приложения из указанной строки.  
Корректность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.13.
