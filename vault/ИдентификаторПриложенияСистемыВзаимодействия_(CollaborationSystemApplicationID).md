Конструкторы:

[По умолчанию](ИдентификаторПриложенияСистемыВзаимодействия.По_умолчанию.md)  

Описание:

Идентифицирует приложение (информационную базу или область данных информационной базы) в системе взаимодействия.  
При сравнении идентификаторов средствами встроенного языка сравнение выполняется по значению идентификаторов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
