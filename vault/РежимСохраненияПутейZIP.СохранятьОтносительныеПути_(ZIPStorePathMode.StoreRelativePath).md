РежимСохраненияПутейZIP (ZIPStorePathMode)

СохранятьОтносительныеПути (StoreRelativePath)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [СохранятьОтносительныеПути](РежимСохраненияПутейФайлаАрхива.СохранятьОтносительныеПути_(ArchiveFileStorePathMode.StoreRelativePath).md)

Описание:

При помещении в архив сохраняются относительные пути от указанного каталога.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
