Описание:

Предоставляет доступ к свойствам серии слоя географической схемы. Позволяет задать параметры её отображения на схеме.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Создадим в слое географической схемы серию для отображения данных  СерияСуммаОборотаПоРегионам = СлойРегионов.Серии.Добавить(); СерияСуммаОборотаПоРегионам.Текст = "Сумма оборота"; СерияСуммаОборотаПоРегионам.ТипОтображения =      ТипОтображенияСерииСлояГеографическойСхемы.Гистограмма; |
