ЧтениеXML (XMLReader)

СледующийАтрибут (NextAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

СледующийАтрибут()

Возвращаемое значение:

Тип: Булево.   
Истина - атрибут прочитан, Ложь - атрибутов больше нет.

Описание:

Позиционирует текущее состояние объекта на следующий атрибут текущего элемента. Если вызов осуществляется при текущем узле [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md), то вызов аналогичен методу [ЧтениеXML.ПервыйАтрибут](ЧтениеXML.ПервыйАтрибут.md).  
При этом свойствам [ЧтениеXML.ТипУзла](ЧтениеXML.ТипУзла.md), [ЧтениеXML.Имя](ЧтениеXML.Имя.md), [ЧтениеXML.ЛокальноеИмя](ЧтениеXML.ЛокальноеИмя.md), [ЧтениеXML.Префикс](ЧтениеXML.Префикс.md), [ЧтениеXML.URIПространстваИмен](ЧтениеXML.URIПространстваИмен.md), [ЧтениеXML.Значение](ЧтениеXML.Значение.md) и т.д. присваиваются значения, соответствующие прочитанным данным атрибута.  
Вызов метода [ЧтениеXML.Прочитать](ЧтениеXML.Прочитать.md) прерывает процесс чтения атрибутов элемента и позиционирует текущий узел на следующий атрибут после [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
