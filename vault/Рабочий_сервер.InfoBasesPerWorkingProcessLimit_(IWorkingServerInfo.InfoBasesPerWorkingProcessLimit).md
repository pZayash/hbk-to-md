InfoBasesPerWorkingProcessLimit (InfoBasesPerWorkingProcessLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит количество информационных баз, соединения которых могут быть направлены на один рабочий процесс этого сервера.

Доступность:

Интеграция.

Доступен, начиная с версии 8.3.1.
