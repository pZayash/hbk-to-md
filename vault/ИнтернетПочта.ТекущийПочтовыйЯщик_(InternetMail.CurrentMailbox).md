ИнтернетПочта (InternetMail)

ТекущийПочтовыйЯщик (CurrentMailbox)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текущий почтовый ящик на сервере, с которым выполняется работа.  
Данное свойство влияет на методы [ПолучитьЗаголовки](ИнтернетПочта.ПолучитьЗаголовки_(InternetMail.GetHeaders).md), [ПолучитьИдентификаторы](ИнтернетПочта.ПолучитьИдентификаторы_(InternetMail.GetUIDL).md), [УдалитьСообщения](ИнтернетПочта.УдалитьСообщения_(InternetMail.DeleteMessages).md), [ПолучитьКоличествоСообщений](ИнтернетПочта.ПолучитьКоличествоСообщений_(InternetMail.GetMessageCount).md), [Послать](ИнтернетПочта.Послать_(InternetMail.Send).md) для случая, когда используется протокол отправки [IMAP](ПротоколИнтернетПочты.IMAP_(InternetMailProtocol.IMAP).md).   
Если свойство не задано, то действие будет производиться с [IMAP](ПротоколИнтернетПочты.IMAP_(InternetMailProtocol.IMAP).md) почтовым ящиком "Входящие" ("Inbox").

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
