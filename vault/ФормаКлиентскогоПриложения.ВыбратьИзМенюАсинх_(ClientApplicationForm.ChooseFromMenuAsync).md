ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыбратьИзМенюАсинх (ChooseFromMenuAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВыбратьИзМенюАсинх(<СписокЗначений>, <ЭлементФормы>)

Параметры:

<СписокЗначений> (обязательный)

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Список значений, содержащий значения, из которых будет осуществляться выбор.

<ЭлементФормы> (необязательный)

Элемент управления формы, около которого будет открыт выпадающий список. Для некоторых элементов выпадающий список будет показан около активной области. Если не указан, то будет выбран активный элемент формы.  
Примечание: В качестве этого параметра нельзя указывать кнопки, расположенные в командной панели или подменю.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - выбранное значение из меню или [Неопределено](lang__def_Undefined.md), если пользователь отказался от выбора.

Описание:

Асинхронная функция. Показывает меню для выбора значения. Набор значений для выбора задаются списком значений, меню будет расположено около переданого элемента формы или около его активной области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
