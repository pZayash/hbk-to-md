СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

УдалитьНастройкуПоУмолчанию (DeleteDefaultSetting)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УдалитьНастройкуПоУмолчанию(<КлючОбъекта>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Ключ объекта настройки.  
Если [Неопределено](lang__def_Undefined.md), то будут удалены все настройки по умолчанию.

Описание:

Удаляет настройку по умолчанию из хранилища.  
Вызов метода:

- требует наличия права "СохранениеДанныхПользователя",
- допускается только для хранилища системных настроек (ХранилищеСистемныхНастроек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
