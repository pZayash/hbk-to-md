Восстанавливать (Restore)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Восстанавливать](РежимВосстановленияПутейФайлаАрхива.Восстанавливать_(ArchiveFileRestoreFilePathsMode.Restore).md)

Описание:

Требуется восстанавливать пути при извлечении элемента из архива.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
