СводнаяДиаграмма (PivotChart)

ЦветТекстаОбластейПолей (FieldAreasTextColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет текста областей полей сводной диаграммы.  
Области полей - это области, на которых расположены выбранные на соответствующей оси измерения (и, возможно, данные, если одновременно используются несколько ресурсов).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
