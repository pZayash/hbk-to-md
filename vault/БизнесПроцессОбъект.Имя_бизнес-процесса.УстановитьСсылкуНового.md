БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

УстановитьСсылкуНового (SetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСсылкуНового(<Ссылка>)

Параметры:

<Ссылка> (обязательный)

Тип: [БизнесПроцессСсылка.\<Имя бизнес-процесса\>](БизнесПроцессСсылка.Имя_бизнес-процесса.md).   
Ссылка, которая будет назначена при записи нового объекта.

Описание:

Устанавливает значение для нового (созданного и еще не записанного) объекта, которое будет назначено при записи в качестве ссылки. Значение не может равняться ссылке какого-либо из имеющихся в базе данных объекта данного типа. Уникальность ссылки проверяется при записи объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[БизнесПроцессМенеджер.\<Имя бизнес-процесса\>](БизнесПроцессМенеджер.Имя_бизнес-процесса.md), метод [ПолучитьСсылку](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
