Значения

[Включая (Including)](ВидГраницы.Включая.md)  
[Исключая (Excluding)](ВидГраницы.Исключая.md)  

Описание:

Определяет набор видов границ по отношению к граничному значению. Используется для определения значения свойства [ВидГраницы](Граница.ВидГраницы.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

[Граница](Граница.md), конструктор [На основании значения и вида](Граница.По_умолчанию.md)
