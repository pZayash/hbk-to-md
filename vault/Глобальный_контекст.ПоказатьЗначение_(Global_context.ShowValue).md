Глобальный контекст (Global context)

ПоказатьЗначение (ShowValue)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьЗначение(<ОписаниеОповещенияОЗавершении>, <Значение>)

Параметры:

<ОписаниеОповещенияОЗавершении> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия окна ввода значения с параметром <ДополнительныеПараметры>, который содержит значение, указанное при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).  
Если параметр не указан, то по завершении никакая процедура вызвана не будет.

<Значение> (обязательный)

Тип: Произвольный.   
Значение любого типа, который может быть показан системой в диалоге.

Описание:

Показывает в диалоге переданное значение, при этом не ожидается его закрытие. Вид диалога зависит от типа переданного значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Товар = Справочники.Номенклатура.НайтиПоКоду(КодТовара); ПоказатьЗначение(,Товар); |
