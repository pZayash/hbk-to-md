**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Таблица значений](Таблица_значений.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[Строка](lang__def_String.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются имена колонок таблицы значений.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента.  

Описание:

Индекс коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
