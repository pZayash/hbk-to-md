СтандартнаяКомандаВходящегоЗапросаПоделиться (IncomingShareRequestStandardCommand)

Показать (Show)

Доступен, начиная с версии 8.3.24.

Описание:

Доступно только при запросе "Поделиться" для одного файла поддерживаемого внутреннего формата (например, mxl).  
При выполнении команды файлы открывается системное окно просмотра файла.  
В контекстном меню команда будет иметь представление "Открыть".

Использование в версии:

Доступен, начиная с версии 8.3.24.
