ЭлементОбластиМакетаОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateAreaItem)

Уровень (Level)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Положительное число, обозначающее номер уровня иерархии. Если равно 0, то считается, что оформление применяется по умолчанию ко всем макетам области определенного типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
