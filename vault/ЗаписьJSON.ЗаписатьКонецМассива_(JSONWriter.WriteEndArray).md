ЗаписьJSON (JSONWriter)

ЗаписатьКонецМассива (WriteEndArray)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗаписатьКонецМассива()

Описание:

Записывает конец массива JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; // Запись документа JSON  ЗаписьJSON.ЗаписатьНачалоМассива();  ЗаписьJSON.ЗаписатьЗначение("Строковое значение");  ЗаписьJSON.ЗаписатьЗначение(123);  ЗаписьJSON.ЗаписатьНачалоОбъекта(); ЗаписьJSON.ЗаписатьИмяСвойства("Булево"); ЗаписьJSON.ЗаписатьЗначение(Истина); ЗаписьJSON.ЗаписатьКонецОбъекта();  ЗаписьJSON.ЗаписатьКонецМассива(); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
