Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка задач. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ЗадачаСписок.](ЗадачаСписок.Имя_задачи.md)<[Имя задачи](ЗадачаСписок.Имя_задачи.md)>.
