ЭлементыШкалыВремени (TimeScaleItems)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента.

Возвращаемое значение:

Тип: [ЭлементШкалыВремени](ЭлементШкалыВремени.md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
