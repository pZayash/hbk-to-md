Глобальный контекст (Global context)

ИмяКомпьютера (ComputerName)

Доступен, начиная с версии 8.0.

Синтаксис:

ИмяКомпьютера()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Сетевое имя компьютера.

Описание:

Получает сетевое имя компьютера.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СетевоеИмяКомпьютера = ИмяКомпьютера(); |

Использование в версии:

Доступен, начиная с версии 8.0.
