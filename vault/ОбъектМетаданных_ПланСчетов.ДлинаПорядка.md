ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ДлинаПорядка (OrderLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Максимальная длина поля [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок.md) плана счетов. Если значение этого свойства равно 0, то по умолчанию список счетов упорядочивается по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код.md). В противном случае - по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
