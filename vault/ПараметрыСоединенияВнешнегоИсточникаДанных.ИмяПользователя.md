ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

ИмяПользователя (UserName)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка, Неопределено.   
Содержит имя пользователя, которое нужно использовать при стандартной аутентификации.  
Если содержит Неопределено, то имя пользователя получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
