# Объект внешней компоненты.<ИмяМетода>Асинх

Объект внешней компоненты (Add-in object)

<ИмяМетода>Асинх (<MethodName>Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

<ИмяМетода>Асинх(<Параметры>)

Параметры:

<Параметры> (необязательный)

Тип: Произвольный.   
Список входных параметров внешней компоненты в том же порядке, в каком они указаны в компоненте.

Возвращаемое значение:

Тип: [РезультатАсинхВызоваВнешнейКомпоненты](РезультатАсинхВызоваВнешнейКомпоненты_(AddInAsyncCallResult).md).   

Описание:

Асинхронно вызывает метод объекта внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Объект внешней компоненты](Объект_внешней_компоненты_(Add-in_object).md), метод [НачатьВызов\<ИмяМетода\>](Объект_внешней_компоненты.НачатьВызовИмяМетода_(Add-in_object.BeginCallingMethodName).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
