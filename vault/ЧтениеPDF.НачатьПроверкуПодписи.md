ЧтениеPDF (PDFReader)

НачатьПроверкуПодписи (BeginVerifySignature)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПроверкуПодписи(<ОписаниеОповещения>, <Имя>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <Сертификат> - тип [СертификатКриптографии](СертификатКриптографии.md). В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Сертификат> - тип [СертификатКриптографии](СертификатКриптографии.md). В параметре возвращается сертификат, с помощью которого была произведена подпись (если сертификат включен в данные подписи).  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).





<Имя> (обязательный)

Тип: Строка.   
Имя подписи для проверки.

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии.md).   
Менеджер криптографии.

Описание:

Проверяет действительность присоединенной к PDF-документу подписи с указанным именем. В случае ошибки генерируется исключение и вызывается процедура обработки ошибок, указанная в <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [ПроверитьПодписи](ЧтениеPDF.ПроверитьПодписи.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [ПроверитьПодписиАсинх](ЧтениеPDF.ПроверитьПодписиАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
