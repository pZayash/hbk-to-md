Описание:

Добавляет узел в список дочерних. Если узел уже присутствует в дереве документа, то он сначала удаляется из прежнего места.  
Если узлом является [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md), то все содержимое фрагмента будет добавлено в список дочерних.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Исключение вызывается в случаях:

- если данный узел не допускает в качестве дочерних добавляемый узел или добавляемый узел является одним из предков данного узел добавляется сам в себя или данный узел - [Документ](ТипУзлаDOM.Документ.md) и второй узел [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) или [Элемент](ТипУзлаDOM.Элемент.md);
- если добавляемый узел был создан документом, отличным от создавшего данный узел;
- если данный узел в режиме Только для чтения или если предыдущий родитель добавляемого узла в режиме Только для чтения;
- если добавляемый узел является дочерним узлом узла [Документ](ТипУзлаDOM.Документ.md) и реализация не поддерживает удаления узлов [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) и [Элемент](ТипУзлаDOM.Элемент.md) у узла [Документ](ТипУзлаDOM.Документ.md).

Использование в версии:

Доступен, начиная с версии 8.2.
