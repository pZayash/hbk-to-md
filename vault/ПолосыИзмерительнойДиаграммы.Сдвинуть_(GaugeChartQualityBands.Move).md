ПолосыИзмерительнойДиаграммы (GaugeChartQualityBands)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Полоса>, <Смещение>)

Параметры:

<Полоса> (обязательный)

Тип: [ПолосаИзмерительнойДиаграммы](ПолосаИзмерительнойДиаграммы_(GaugeChartQualityBand).md).   
Сдвигаемая полоса.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает полосу на указанное число позиций.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
