ИсточникКомандПоляПланировщика (PlannerCommandSource)

ПустаяОбластьЭлементов (EmptyItemsArea)

Доступен, начиная с версии 8.3.23.

Описание:

Значение, которое устанавливается при нажатии на пустую область поля планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд.md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md) содержит Неопределено,
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md) содержит Неопределено,
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md) заполнено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата.md) заполнено,
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md) содержит Неопределено,
- [Команды](html.md) заполнено стандартными командами планировщика - [СоздатьЭлемент](СтандартнаяКомандаПланировщика.СоздатьЭлемент.md),
- [КомандаПоУмолчанию](html.md) заполнено командой [СоздатьЭлемент](СтандартнаяКомандаПланировщика.СоздатьЭлемент.md).

Использование в версии:

Доступен, начиная с версии 8.3.23.
