РегистрБухгалтерииКлючЗаписи.<Имя регистра бухгалтерии> (AccountingRegisterRecordKey.<Accounting register name>)

НомерСтроки (LineNumber)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрБухгалтерииКлючЗаписи.Имя_регистра_бухгалтерии.Регистратор_(AccountingRegisterRecordKey.Accounting_register_name.Recorder).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
