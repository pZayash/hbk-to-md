Описание:

Содержит варианты режимов ограничения количества серий в диаграмме. Используется для определения свойства [МаксимумСерий](Диаграмма.МаксимумСерий.md) диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: MaxSeries.
