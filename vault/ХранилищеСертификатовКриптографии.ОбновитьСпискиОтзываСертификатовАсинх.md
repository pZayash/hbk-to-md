ХранилищеСертификатовКриптографии (CryptoCertificateStore)

ОбновитьСпискиОтзываСертификатовАсинх (UpdateCertificateRevocationListAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ОбновитьСпискиОтзываСертификатовАсинх(<Сертификаты>, <ОбновлятьПриНаличииРасширенияСАдресомOCSP>)

Параметры:

<Сертификаты> (необязательный)

Тип: [Массив](Массив.md).   
Массив объектов типа [СертификатКриптографии](СертификатКриптографии.md) – список сертификатов, по которым необходимо обновить списки отзыва. По умолчанию выполняется обновление всех сертификатов хранилища.

<ОбновлятьПриНаличииРасширенияСАдресомOCSP> (необязательный)

Тип: Булево.   
Определяет, требуется ли обновлять списки отзыва по сертификату, если в нем есть адрес OCSP-сервера.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
[Обещание](Обещание.md) с результатом выполнения типа Булево – удалось ли обновить хотя бы один список отзыва или не требуется обновлять списки отзыва.

Описание:

Асинхронная функция. Обновляет списки отзыва по списку сертификатов и их цепочек. Рекомендуется добавить в регламентные задания при необходимости использования улучшенной подписи. Позволяет поддерживать списки отзыва в актуальном состоянии в асинхронном режиме (не задерживая функции создания подписи).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Хранилище = МенеджерКриптографии.ПолучитьХранилищеСертификатов(Тип); Сертификаты = Хранилище.ПолучитьВсе(); Результат = Ждать Хранилище.ОбновитьСпискиОтзываСертификатовАсинх(Сертификаты, Ложь); |
