НастройкаПорядка (OrderSetting)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента управления порядком.

Возвращаемое значение:

Тип: [ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md).   

Описание:

Получает элемент управления порядком по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
