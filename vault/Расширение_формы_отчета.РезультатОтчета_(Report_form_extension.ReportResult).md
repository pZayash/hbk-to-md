Расширение формы отчета (Report form extension)

РезультатОтчета (ReportResult)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Содержит элемент управления, в который будет осуществляться вывод результата.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.1.
