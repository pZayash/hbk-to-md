ЗаписьДанных (DataWriter)

ЗаписатьЦелое16 (WriteInt16)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьЦелое16(<Число>, <ПорядокБайтов>)

Параметры:

<Число> (обязательный)

Тип: Число.   
Число, которое будет записано в целевой поток.  
Значение числа должно находиться в диапазоне от 0 до 65535.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов.md).   
Порядок байтов, который будет использован для кодировки числа при записи. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных.md).  
Значение по умолчанию: Неопределено.

Описание:

Записывает 16-разрядное число в целевой поток.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [НачатьЗаписьЦелого16](ЗаписьДанных.НачатьЗаписьЦелого16.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьЦелое16Асинх](ЗаписьДанных.ЗаписатьЦелое16Асинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
