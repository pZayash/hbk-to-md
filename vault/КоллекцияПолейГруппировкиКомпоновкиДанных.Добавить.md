КоллекцияПолейГруппировкиКомпоновкиДанных (DataCompositionGroupFieldCollection)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [ПолеГруппировкиКомпоновкиДанных](ПолеГруппировкиКомпоновкиДанных.md), [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [ПолеГруппировкиКомпоновкиДанных](ПолеГруппировкиКомпоновкиДанных.md), [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
