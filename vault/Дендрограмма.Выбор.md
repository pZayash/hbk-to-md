Дендрограмма (Dendrogram)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<ЗначениеДиаграммы>, <СтандартнаяОбработка>)

Параметры:

<ЗначениеДиаграммы>

Тип: [ЭлементДендрограммы](ЭлементДендрограммы.md), [СвязьДендрограммы](СвязьДендрограммы.md).   
Значение дендрограммы.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Возникает при двойном щелчке мыши на дендрограмме.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если дендрограмма расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.

Использование в версии:

Доступен, начиная с версии 8.0.
