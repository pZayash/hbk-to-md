СтраницаПанели (PanelPage)

КартинкаЗаголовка (TitlePicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Содержит картинку, отображаемую на закладке страницы. Картинка отображается слева от текста при горизонтальной ориентации закладок и сверху от текста при вертикальной ориентации. Картинка автоматически уменьшается (если её размеры больше) до размера 96 пикселя по ширине и 64 по высоте с сохранением пропорций оригинального изображения.

Доступность:

Толстый клиент.

Примечание:

Рекомендуемый размер картинки 16х16 пикселей.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.0.
