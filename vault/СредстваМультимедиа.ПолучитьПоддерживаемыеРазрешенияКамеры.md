СредстваМультимедиа (MultimediaTools)

ПолучитьПоддерживаемыеРазрешенияКамеры (GetSupportedCameraResolutions)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьПоддерживаемыеРазрешенияКамеры(<ТипКамеры>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства.md).   
Показывает, для какой камеры будут получены разрешения.  
Значение по умолчанию: [Авто](ТипКамерыУстройства.Авто.md).

Возвращаемое значение:

Тип: [Массив](Массив.md), Неопределено.   
Массив [РазрешениеКамерыУстройства](РазрешениеКамерыУстройства.md). Для iOS будет возвращен тип Неопределено.

Описание:

Получает массив доступных разрешений для выбранной камеры.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа.md), метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
