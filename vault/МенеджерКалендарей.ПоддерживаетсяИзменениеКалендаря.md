МенеджерКалендарей (CalendarsManager)

ПоддерживаетсяИзменениеКалендаря (CalendarEditingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяИзменениеКалендаря(<Календарь>)

Параметры:

<Календарь> (обязательный)

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря.md).   
Локальный ключ календаря.

Возвращаемое значение:

Тип: Булево.   
На устройствах под управлением Windows Phone всегда возвращает Ложь.  
Для устройств под управлением Android (начиная с версии 4.1) и iOS всегда возвращает Истина.

Описание:

Определяет возможность редактирования или удаления календаря.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МенеджерКалендарей](МенеджерКалендарей.md), метод [УдалитьКалендарь](МенеджерКалендарей.УдалитьКалендарь.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
