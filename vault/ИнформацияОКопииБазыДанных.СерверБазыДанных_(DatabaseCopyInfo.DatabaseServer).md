ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

СерверБазыДанных (DatabaseServer)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя сервера соответствующее [СерверБазыДанных](МенеджерКопииБазыДанных.СерверБазыДанных_(DatabaseCopyManager.DatabaseServer).md) или имя компьютера (host), на котором запущен Дата акселератор. Для разных экземпляров копии может содержать разное значение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
