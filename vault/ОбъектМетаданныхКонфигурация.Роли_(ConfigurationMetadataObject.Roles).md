ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Роли (Roles)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все роли, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Роль](ОбъектМетаданных_Роль_(MetadataObject_Role).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
