ПользовательИнформационнойБазы (InfoBaseUser)

ОбработкаНастроекВторогоФактораАутентификации (SecondAuthenticationFactorSettingsProcessing)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [ТипОбработкиНастроекВторогоФактораАутентификации](ТипОбработкиНастроекВторогоФактораАутентификации_(SecondAuthenticationFactorSettingsProcessingType).md).   
Указывает, будет ли выполнен переход к следующему провайдеру при возврате ошибки от текущего.  
Значение по умолчанию - [ИспользоватьСледующуюПриОшибке](ТипОбработкиНастроекВторогоФактораАутентификации.ИспользоватьСледующуюПриОшибке_(SecondAuthenticationFactorSettingsProcessingType.UseNextIfFailed).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.15.
