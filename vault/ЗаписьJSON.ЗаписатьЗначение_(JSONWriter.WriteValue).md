# ЗаписьJSON.ЗаписатьЗначение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [JSON](JSON-2.md) › [ЗаписьJSON](ЗаписьJSON_(JSONWriter).md) › [Методы](ЗаписьJSON__Методы.md)

ЗаписьJSON (JSONWriter)

ЗаписатьЗначение (WriteValue)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ЗаписатьЗначение(<Значение>, <ИспользоватьФорматСЭкспонентой>)

Параметры:

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md), [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Записываемое значение.

<ИспользоватьФорматСЭкспонентой> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Использование экспоненциальной формы записи для числовых значений. Параметр имеет смысл только если записывается значение числового типа.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Записывает значение свойства JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьJSON = Новый ЗаписьJSON; // Запись документа JSON  // Запись строковое значение  ЗаписьJSON.ЗаписатьИмяСвойства("Строка"); ЗаписьJSON.ЗаписатьЗначение("Строковое значение"); // Запись числовое значение  ЗаписьJSON.ЗаписатьИмяСвойства("Число"); ЗаписьJSON.ЗаписатьЗначение(123); // Запись значение с плавающей точкой  // в экспоненциальном формате  ЗаписьJSON.ЗаписатьИмяСвойства("ЧислоВЭкпоненциальномФормате"); ЗаписьJSON.ЗаписатьЗначение(123.45); // Запись булево значение  ЗаписьJSON.ЗаписатьИмяСвойства("Булево"); ЗаписьJSON.ЗаписатьЗначение(Истина); // Запись значение Неопределено  ЗаписьJSON.ЗаписатьИмяСвойства("Неопределено"); ЗаписьJSON.ЗаписатьЗначение(Неопределено); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
