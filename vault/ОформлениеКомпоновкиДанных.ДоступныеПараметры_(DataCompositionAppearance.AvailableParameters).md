ОформлениеКомпоновкиДанных (DataCompositionAppearance)

ДоступныеПараметры (AvailableParameters)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДоступныеПараметрыКомпоновкиДанных](ДоступныеПараметрыКомпоновкиДанных_(DataCompositionAvailableParameters).md).   
Содержит набор доступных параметров. Определяет состав оформления компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
