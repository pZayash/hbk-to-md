СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

СоздатьЭлемент (CreateItem)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьЭлемент()

Возвращаемое значение:

Тип: [СправочникОбъект.\<Имя справочника\>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md).   

Описание:

Создает новый элемент справочника.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Пример:

|  |
| --- |
| НовыйОбъект = Справочники.Валюты.СоздатьЭлемент(); НовыйОбъект.Код = "840"; НовыйОбъект.Наименование = "USD"; НовыйОбъект.ПолноеНаименование = "Доллары США"; НовыйОбъект.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
