БазовыеВидыРасчета.<Имя плана видов расчета> (BaseCalculationTypes.<Chart of calculation types name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [БазовыеВидыРасчетаСтрока.\<Имя плана видов расчета\>](БазовыеВидыРасчетаСтрока.Имя_плана_видов_расчета_(BaseCalculationTypesRow.Chart_of_calculation_types_name).md).   
Строка табличной части, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной строки в табличной части. Если не найдено, то возвращается -1.

Описание:

Получает индекс строки в табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
