COM-класс (ISecurityProfileCOMClass)

ComputerName (ComputerName)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя компьютера, на котором может быть создан COM-объект. Используется при выполнении Новый COMОбъект. Специальные значения:

- localhost – текущий компьютер;
- пустая строка – произвольный компьютер.

localhost – текущий компьютер;  
пустая строка – произвольный компьютер.





Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
