Описание:

С помощью данного объекта осуществляется управление конструктором форматной строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.   
Сериализуется.

Пример:

|  |
| --- |
| Конструктор = Новый КонструкторФорматнойСтроки; // форматная строка только для типов Число и Дата  Конструктор.ДоступныеТипы = Новый ОписаниеТипов("Число, Дата"); Если Конструктор.ОткрытьМодально() Тогда     // обработка форматной строки      ФорматнаяСтрока = Конструктор.Текст;     // ...  КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
