СтандартнаяКомандаПланировщика (PlannerStandardCommand)

ПерейтиПоНавигационнойСсылке (GotoURL)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ПерейтиПоНавигационнойСсылке](СтандартнаяКомандаПоляПланировщика.ПерейтиПоНавигационнойСсылке_(PlannerStandardCommand.GotoURL).md)

Доступен, начиная с версии 8.3.18.

Описание:

Переход к событию [НажатиеНаНавигационнойСсылке](Расширение_поля_формы_для_поля_планировщика.НажатиеНаНавигационнойСсылке_(Form_field_extension_for_a_planner.URLClick).md).  
Используется только для источника вида [НавигационнаяСсылка](ИсточникКомандПланировщика.НавигационнаяСсылка_(PlannerCommandSource.URL).md).  
В контекстном меню команда будет иметь представление "Перейти".

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
