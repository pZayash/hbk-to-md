ФабрикаXDTO (XDTOFactory)

ПрочитатьJSON (ReadJSON)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПрочитатьJSON(<ЧтениеJSON>, <Тип>, <ИмяФункцииВосстановления>, <МодульФункцииВосстановления>, <ДополнительныеПараметрыФункцииВосстановления>, <ТипыДляОбработкиВосстановления>, <ИменаСвойствДляВосстановления>, <НазначениеТипа>)

Параметры:

<ЧтениеJSON> (обязательный)

Тип: [ЧтениеJSON](ЧтениеJSON_(JSONReader).md).   
Объект чтения JSON.

<Тип> (необязательный)

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO_(XDTOValueType).md), [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md).   
Тип элемента данных XDTO. Если тип не указан, то будет сделана попытка определить тип данных по текущему состоянию объекта [ЧтениеJSON](ЧтениеJSON_(JSONReader).md).

<ИмяФункцииВосстановления> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя функции восстановления значения.  
Функция должна иметь следующие параметры:

- <Свойство> - значение типа [Строка](lang__def_String.md),
- <Тип> - значение типа [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md),
- <Значение> - значение допустимого для сериализации типа,
- <ДополнительныеПараметры>.

Возвращаемое значение - допустимого для XDTO сериализации типа.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<МодульФункцииВосстановления> (необязательный)

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md), [ОбщийМодуль](ОбщийМодуль_(CommonModule).md).   
Указывает модуль, процедура которого будет использована для восстановления значения. В зависимости от типа параметра будет вызван соответствующий метод:

- [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md) - будет вызван метод модуля указанной формы клиентского приложения.
- [КомандаКомандногоИнтерфейса](КомандаКомандногоИнтерфейса_(CommandInterfaceCommand).md) - будет вызван метод модуля команды командного интерфейса.
- [ОбщийМодуль](ОбщийМодуль_(CommonModule).md) - будет вызван метод неглобального общего модуля.

  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ДополнительныеПараметрыФункцииВосстановления> (необязательный)

Тип: Произвольный.   
Дополнительные параметры для функции восстановления.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ТипыДляОбработкиВосстановления> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов [ТипОбъектаXDTO](ТипОбъектаXDTO_(XDTOObjectType).md), соответствующих типам XDTO, для которых требуется вызвать функцию восстановления.  
Если функция восстановления не задана, то параметры <ТипыДляОбработкиВосстановления> и <ИменаСвойствДляВосстановления> игнорируются.

<ИменаСвойствДляВосстановления> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Имена свойств, для которых нужно вызвать функцию восстановления.  
Если функция восстановления не задана, то параметры <ТипыДляОбработкиВосстановления> и <ИменаСвойствДляВосстановления> игнорируются.

<НазначениеТипа> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Назначение типа данных произвольных свойства при чтении.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Возвращаемое значение:

Тип: [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   

Описание:

Читает элемент данных XDTO указанного типа из объекта чтения JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.7.

Описание изменено в версии 8.3.25.
