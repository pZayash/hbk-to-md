ИдентификаторТекущегоПользователя (CurrentUserID)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ИдентификаторТекущегоПользователя()

Возвращаемое значение:

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   

Описание:

Возвращает идентификатор текущего пользователя системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
