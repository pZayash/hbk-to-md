Описание:

Представляет собой отдельную колонку таблицы значений. Доступ к объекту осуществляется через свойства коллекции колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
