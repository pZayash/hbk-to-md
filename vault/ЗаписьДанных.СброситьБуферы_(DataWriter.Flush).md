ЗаписьДанных (DataWriter)

СброситьБуферы (Flush)

Доступен, начиная с версии 8.3.9.

Синтаксис:

СброситьБуферы()

Описание:

Сбрасывает все внутренние буферы в целевой поток, после чего вызывает метод [СброситьБуферы](Поток.СброситьБуферы_(Stream.Flush).md) целевого потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [НачатьСбросБуферов](ЗаписьДанных.НачатьСбросБуферов_(DataWriter.BeginFlush).md)  
[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [СброситьБуферыАсинх](ЗаписьДанных.СброситьБуферыАсинх_(DataWriter.FlushAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
