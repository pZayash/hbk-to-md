Расширение формы документа (Document form extension)

ПараметрОбъектКопирования (CopyingObjectParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Содержит ссылку на копируемый объект.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
