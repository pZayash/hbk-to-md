ВариантОтображенияСообщенияОбОшибке (ErrorMessageDisplayVariant)

СообщениеОбОшибкеДляПользователя (ErrorMessageForUser)

Доступен, начиная с версии 8.3.17.

Описание:

Отображается сообщение об ошибке для пользователя.  
Дополнительно может выводиться текст, помогающий пользователю исправить ошибку

Использование в версии:

Доступен, начиная с версии 8.3.17.
