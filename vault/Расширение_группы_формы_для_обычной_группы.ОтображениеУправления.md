Расширение группы формы для обычной группы (Form group extension for a usual group)

ОтображениеУправления (ControlRepresentation)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеУправленияОбычнойГруппы](ОтображениеУправленияОбычнойГруппы.md).   
Содержит режим отображения управления группы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
