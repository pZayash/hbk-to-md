Описание:

Клиентский прокси для вызова веб-сервиса.  
Для защиты соединения может быть использован протокол TLS 1.2.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
