РасписаниеРегламентногоЗадания (JobSchedule)

ДатаНачала (BeginDate)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата начала расписания. Регламентные задания могут быть запущены только в том случае, если текущая дата больше и равна дате начала. Если дата начала не задана, начало запуска неограничено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
