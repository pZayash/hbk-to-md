ОписанияХарактеристик (CharacteristicsDescriptions)

Индекс (IndexOf)

Доступен, начиная с версии 8.2.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ОписаниеХарактеристик](ОписаниеХарактеристик_(CharacteristicsDescription).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс объекта в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
