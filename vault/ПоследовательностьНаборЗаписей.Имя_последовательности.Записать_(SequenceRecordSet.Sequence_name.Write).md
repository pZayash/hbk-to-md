# ПоследовательностьНаборЗаписей.<Имя последовательности>.Записать

ПоследовательностьНаборЗаписей.<Имя последовательности> (SequenceRecordSet.<Sequence name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим замещения существующих записей в соответствии с текущими установками отбора. [Истина](lang__def_BooleanTrue.md) - перед записью существующие записи будут удалены. [Ложь](lang__def_BooleanFalse.md) - записи будут дописаны к уже существующим в информационной базе записям.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает набор записей регистрации документа в базу данных. В зависимости от переданного параметра, может быть выполнено добавление записей или их замещение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу набор записей очищается (удаляются записи из набора).

Использование в версии:

Доступен, начиная с версии 8.0.
