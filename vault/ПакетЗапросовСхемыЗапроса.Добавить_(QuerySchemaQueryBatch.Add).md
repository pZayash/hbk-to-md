ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).  
Значение по умолчанию: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md).

Возвращаемое значение:

Тип: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод возвращает [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
