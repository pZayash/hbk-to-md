ВертикальноеПоложениеЭлемента (ItemVerticalAlign)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

Вертикальное положение определяется автоматически.  
Для однострочной декорации будет использовано значение [Центр](ВертикальноеПоложениеЭлемента.Центр.md).  
Для многострочной - [Верх](ВертикальноеПоложениеЭлемента.Верх.md).

Использование в версии:

Доступен, начиная с версии 8.2.
