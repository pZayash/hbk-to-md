ДанныеФормыСтруктура (FormDataStructure)

ИсходныйКлючЗаписи (SourceRecordKey)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РегистрСведенийКлючЗаписи.\<Имя регистра сведений\>](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md).   
Содержит ключ записи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Существует у объектов, которые были получены из записей регистров сведений.

Использование в версии:

Доступен, начиная с версии 8.2.
