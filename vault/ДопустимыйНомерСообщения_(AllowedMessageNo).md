Описание:

Содержит варианты указаний, какой номер сообщения обмена данными следует считать допустимым при приеме сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[ЧтениеСообщенияОбмена](ЧтениеСообщенияОбмена_(ExchangeMessageReader).md), метод [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
