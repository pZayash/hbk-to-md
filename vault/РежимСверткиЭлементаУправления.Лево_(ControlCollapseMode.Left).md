РежимСверткиЭлементаУправления (ControlCollapseMode)

Лево (Left)

Доступен, начиная с версии 8.0.

Описание:

Элемент управления свертывается к своей левой стороне (правая сторона прижимается к левой).

Использование в версии:

Доступен, начиная с версии 8.0.
