Расширение табличного поля структуры настроек компоновки данных (Settings structure table box extension)

ПодробнаяНастройка (DetailedSetting)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Режим подробной настройки влияет на состав полей текущих данных табличного поля и на состав свойств объекта [ТекущиеДанныеСтруктурыНастроекКомпоновкиДанных](ТекущиеДанныеСтруктурыНастроекКомпоновкиДанных.md).  
Полями текущих данных могут быть: [ПараметрыСхемыКомпоновкиДанных](ПараметрыСхемыКомпоновкиДанных.md), [ПоляГруппировкиКомпоновкиДанных](ПоляГруппировкиКомпоновкиДанных.md), [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных.md), [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных.md), [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных.md), [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных.md), [ЗначенияПараметровВыводаТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаТаблицыКомпоновкиДанных.md), [ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаДиаграммыКомпоновкиДанных.md), [ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиТаблицыКомпоновкиДанных.md), [ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиКомпоновкиДанных.md), [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных.md), [ЗначенияПараметровВыводаКомпоновкиДанных](ЗначенияПараметровВыводаКомпоновкиДанных.md), [ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных](ЗначенияПараметровВыводаГруппировкиДиаграммыКомпоновкиДанных.md), [ПользовательскиеПоляКомпоновкиДанных](ПользовательскиеПоляКомпоновкиДанных.md).  
В случае подробных настроек текущие данные всегда относятся к текущей строке структуры настроек, а их состав определяется наличием у объекта соответствующих свойств.  
В случае кратких настроек напрямую от текущей строки зависят только поля группировки. Остальные поля текущих данных постоянно позволяют редактировать основные настройки, т.е. соответствующие свойства объекта "Настройки компоновки данных", которому принадлежит текущая строка структуры настроек.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
