РасписаниеРегламентногоЗадания (JobSchedule)

ДеньНеделиВМесяце (WeekDayInMonth)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Номер дня недели в месяце, когда задание может быть запущено (например, второе воскресенье месяца). Положительные значения считают относительно начала месяца, отрицательные - относительно конца (1 - первый день недели в месяце, -1 - последний день недели в месяце).   
Например: ДеньНеделиВМесяце = 2, ДниНедели={3,5} - выполнять каждую вторую среду и пятницу месяца.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
