ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет режимом выбора значения.   
[Истина](lang__def_BooleanTrue.md) - выбор будет выполняться из выпадающего списка.  
[Ложь](lang__def_BooleanFalse.md) - из окна выбора вида расчета.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
