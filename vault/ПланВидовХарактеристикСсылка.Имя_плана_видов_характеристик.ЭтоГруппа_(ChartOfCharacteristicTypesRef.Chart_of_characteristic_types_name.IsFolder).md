ПланВидовХарактеристикСсылка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesRef.<Chart of characteristic types name>)

ЭтоГруппа (IsFolder)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Позволяет определить, является ли элемент вида характеристики группой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - элемент является группой;  
[Ложь](lang__def_BooleanFalse.md) - элемент не является группой.

Использование в версии:

Доступен, начиная с версии 8.0.
