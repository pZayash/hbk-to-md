ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

ПерейтиКПредыдущемуОкну (GotoPreviousWindow)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПерейтиКПредыдущемуОкну()

Описание:

Для объекта [ТестируемоеОкноКлиентскогоПриложения](ТестируемоеОкноКлиентскогоПриложения_(TestedClientApplicationWindow).md), у которого свойство [Основное](ТестируемоеОкноКлиентскогоПриложения.Основное_(TestedClientApplicationWindow.IsMain).md) установлено в значение [Истина](lang__def_BooleanTrue.md), выполняет переход на к предыдущему открытому окну.  
Для остальных окон генерирует исключение.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
