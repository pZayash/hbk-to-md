CurrentServiceName (CurrentServiceName)

Доступен, начиная с версии 8.3.12.

Только чтение.

Описание:

Тип: Строка.   
Идентификатор сервиса кластера, который вызывается в данный момент, или пустая строка, если вызов сервиса кластера не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.12.
