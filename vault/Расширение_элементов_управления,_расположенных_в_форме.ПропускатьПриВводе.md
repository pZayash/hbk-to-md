Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак, при установке которого, элемент управления будет пропускаться при обходе элементов управления по клавише Enter. При обычном обходе (по клавише Tab) элемент управления пропущен не будет.  
Истина - пропускается при вводе.

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
