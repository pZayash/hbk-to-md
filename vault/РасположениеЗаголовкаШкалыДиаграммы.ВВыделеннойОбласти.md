РасположениеЗаголовкаШкалыДиаграммы (ChartScaleTitlePlacement)

ВВыделеннойОбласти (SpecialArea)

Доступен, начиная с версии 8.3.10.

Описание:

Заголовок располагается в выделенной области: для горизонтальной оси - снизу, для вертикальной оси - слева.

Использование в версии:

Доступен, начиная с версии 8.3.10.
