НастройкаНастройкиОформления (AppearanceSettingSettings)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента управления настройкой.

<Доступность> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойНастройкиОформления](ЭлементУправленияНастройкойНастройкиОформления_(AppearanceSettingsSettingControl).md).   

Описание:

Добавляет элемент управления настройкой оформления в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
