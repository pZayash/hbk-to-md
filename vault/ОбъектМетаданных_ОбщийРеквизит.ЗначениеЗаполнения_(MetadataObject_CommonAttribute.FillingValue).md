# ОбъектМетаданных: ОбщийРеквизит.ЗначениеЗаполнения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: ОбщийРеквизит](ОбъектМетаданных_ОбщийРеквизит_(MetadataObject_CommonAttribute).md) › [Свойства](ОбъектМетаданных_ОбщийРеквизит__Свойства.md)

ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

ЗначениеЗаполнения (FillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Значение типа, совпадающего с типом реквизита. Данное значение будет присваиваться реквизиту процедурой стандартной обработки заполнения, если реквизит не заполнен из данных заполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
