МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьТекущийАдресОбновления (GetCurrentUpdateAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТекущийАдресОбновления()

Возвращаемое значение:

Тип: Строка.   
Если адрес был установлен с помощью метода [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления_(ApplicationUsageStatisticsManager.SetUpdateAddress).md), со значением не равным [Неопределено](lang__def_Undefined.md), то возвращается именно этот адрес. Иначе возвращается адрес, установленный в Сборщике мобильного приложения.

Описание:

Возвращает действующий адрес автоматического обновления настроек регистрации событий/экранов статистики использования приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
