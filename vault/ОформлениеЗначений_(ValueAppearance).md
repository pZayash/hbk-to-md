Описание:

Содержит оформление для значений диаграммы.  
Для ключа одного типа может содержаться не более 100 записей. Количество записей с разными типами значений не ограничено.  
При записи в системное хранилище, если количество элементов какого-либо типа превышает 100, то записи, которые дольше всех не использовались (для них не был вызван метод [Получить](ОформлениеЗначений.Получить_(ValueAppearance.Get).md)), будут удалены.

Доступность:

Сервер, толстый клиент.   
Сериализуется.

Пример:

|  |
| --- |
| // заполнение цветов для диаграмм на форме   НаСервере Процедура ЗаполнитьЦветаДиаграмм()             // использовать ОформлениеЗначений         // из хранилища системных настроек        ОформлениеЗначений = ХранилищеСистемныхНастроек.                                         Загрузить("Общее/ЦветаДиаграммы");             Диаграмма1.ЗаполнитьЦветаСерий(ОформлениеЗначений);       Диаграмма2.ЗаполнитьЦветаСерий(ОформлениеЗначений);       Диаграмма3.ЗаполнитьЦветаСерий(ОформлениеЗначений);             // записать ОформлениеЗначений        ХранилищеСистемныхНастроек.Сохранить("Общее/ЦветаДиаграммы",                                                  , ОформлениеЗначений);       КонецПроцедуры |

Использование в версии:
