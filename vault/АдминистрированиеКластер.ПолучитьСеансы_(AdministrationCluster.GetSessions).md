АдминистрированиеКластер (AdministrationCluster)

ПолучитьСеансы (GetSessions)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСеансы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеСеанс](АдминистрированиеСеанс_(AdministrationSession).md).

Получает список сеансов, работающих с данным кластером.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
