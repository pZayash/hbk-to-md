ВнешниеОтчетыМенеджер (ExternalReportsManager)

Создать (Create)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени файла

Синтаксис:

Создать(<ПолноеИмяФайла>, <БезопасныйРежим>, <ЗащитаОтОпасныхДействий>)

Параметры:

<ПолноеИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла внешнего отчета.

<БезопасныйРежим> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Строка](lang__def_String.md).   
Признак создания внешнего отчета в безопасном режиме.   
Если указана строка, то внешняя обработка будет исполняться в безопасном режиме под управлением профиля безопасности с этим именем. Профиль безопасности должен иметь значение [SafeModeProfile](Профиль_безопасности.SafeModeProfile_(ISecurityProfile.SafeModeProfile).md)=[Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ЗащитаОтОпасныхДействий> (необязательный)

Тип: [ОписаниеЗащитыОтОпасныхДействий](ОписаниеЗащитыОтОпасныхДействий_(UnsafeOperationProtectionDescription).md).   
Структура, описывающая параметры защиты от опасных действий для подключаемого модуля.

Вариант синтаксиса: По имени отчета

Синтаксис:

Создать(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя внешнего отчета, полученное после подключения.

Описание варианта метода:

Предварительно обработка должна быть подключена методом [Подключить](ВнешниеОтчетыМенеджер.Подключить_(ExternalReportsManager.Connect).md).

Возвращаемое значение:

Тип: [ВнешнийОтчет](ВнешнийОтчет_(ExternalReport).md).   

Описание:

Создает объект внешнего отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
