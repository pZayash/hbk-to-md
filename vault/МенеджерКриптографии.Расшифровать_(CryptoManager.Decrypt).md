МенеджерКриптографии (CryptoManager)

Расшифровать (Decrypt)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: Данные в возвращаемом значении

Синтаксис:

Расшифровать(<ИсходныеДанные>, <Сертификат>, <ПарольДоступаКЗакрытомуКлючу>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Зашифрованные данные.  
Данные могут размещаться в файле (в этом случае указывается имя файла), а также представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md).

<Сертификат> (необязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Сертификат получателя зашифрованных данных, для которого выполняется расшифровка данных.

<ПарольДоступаКЗакрытомуКлючу> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к ключевому контейнеру получателя зашифрованных данных.

Описание варианта метода:

Возвращает расшифрованные данные в виде объекта [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Сохранение данных в файл

Синтаксис:

Расшифровать(<ИсходныеДанные>, <ВыходныеДанные>, <Сертификат>, <ПарольДоступаКЗакрытомуКлючу>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Зашифрованные данные.  
Данные могут размещаться в файле (в этом случае указывается имя файла), а также представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md).

<ВыходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Содержит имя выходного файла или поток, в который будут сохранены расшифрованные данные.

<Сертификат> (необязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Сертификат получателя зашифрованных данных, для которого выполняется расшифровка данных.

<ПарольДоступаКЗакрытомуКлючу> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к ключевому контейнеру получателя зашифрованных данных.

Описание варианта метода:

Помещает расшифрованные данные в файл. Возвращает [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Неопределено](lang__def_Undefined.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Расшифровывает зашифрованные данные.  
Чтобы метод успешно расшифровал данные, необходимо чтобы:

- В хранилище персональных сертификатов был установлен сертификат, связанный с контейнером открытого ключа;
- Сертификат был указан одним из получателей зашифрованных данных (перечислен в параметре <СписокПолучателей> метода [Зашифровать](МенеджерКриптографии.Зашифровать_(CryptoManager.Encrypt).md)).

Размер исходных зашифрованных данных ограничен 2 Гб (2147483647 байт).  
  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В случае ошибки генерируется исключительная ситуация.
