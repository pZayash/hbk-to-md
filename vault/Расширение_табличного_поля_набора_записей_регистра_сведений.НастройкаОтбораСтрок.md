Расширение табличного поля набора записей регистра сведений (Information register recordset table box extension)

НастройкаОтбораСтрок (RowFilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора.md).   
Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md). Наименования свойств совпадают с наименованиями полей, которые могут быть использованы для отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
