ДвоичныеДанные (BinaryData)

На основании файла

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ДвоичныеДанные(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из которого будет загружено значение в виде двоичных данных.

Описание:

Создает объект и помещает в него двоичные данные, взятые из файла.  
В режиме веб-клиента конструктор не доступен, для создания двоичных данных из файла нужно использовать метод глобального контекста [НачатьСозданиеДвоичныхДанныхИзФайла](Глобальный_контекст.НачатьСозданиеДвоичныхДанныхИзФайла_(Global_context.BeginCreateBinaryDataFromFile).md).

Пример:

|  |
| --- |
| ФайлКартинки = "C:\Мои документы\Картинка1.bmp"; ДвоичныеДанные = Новый ДвоичныеДанные(ФайлКартинки); |

Использование в версии:

Доступен, начиная с версии 8.0.
