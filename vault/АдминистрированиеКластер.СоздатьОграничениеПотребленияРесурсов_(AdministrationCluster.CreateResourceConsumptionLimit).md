АдминистрированиеКластер (AdministrationCluster)

СоздатьОграничениеПотребленияРесурсов (CreateResourceConsumptionLimit)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьОграничениеПотребленияРесурсов()

Возвращаемое значение:

Тип: [АдминистрированиеОграничениеПотребленияРесурсов](АдминистрированиеОграничениеПотребленияРесурсов_(AdministrationResourceConsumptionLimit).md).   

Описание:

Создает ограничение потребления ресурсов с пустыми значениями свойств. Этот объект может быть использован для регистрации нового ограничения потребления ресурсов после установки правильных значений его свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
