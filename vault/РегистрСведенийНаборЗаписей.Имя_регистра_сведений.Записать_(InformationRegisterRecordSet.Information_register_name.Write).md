# РегистрСведенийНаборЗаписей.<Имя регистра сведений>.Записать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md) › [РегистрСведенийНаборЗаписей.<Имя регистра сведений>](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md) › [Методы](РегистрСведенийНаборЗаписей.Имя_регистра_сведений__Методы.md)

РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [РежимЗамещения](РежимЗамещения_(ReplacementMode).md).   
Определяет режим замещения существующих записей в соответствии с текущими установками отбора:

- [Истина](lang__def_BooleanTrue.md) или [Замещение](РежимЗамещения.Замещение_(ReplacementMode.Replace).md) - перед записью существующие записи будут удалены.
- [Ложь](lang__def_BooleanFalse.md) или [Добавление](РежимЗамещения.Добавление_(ReplacementMode.Append).md) - записи будут дописаны к уже существующим в информационной базе записям.
- [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) - существующие записи регистра, совпадающие по значениям ключевых полей (Период, измерения, разделители - для независимого регистра, Регистратор, НомерСтроки, разделители - для подчиненного) с записями набора записей будут перезаписаны. Если для какой-либо записи набора записей нет соответствующей записи регистра, будет сгенерировано исключение.
- [Слияние](РежимЗамещения.Слияние_(ReplacementMode.Merge).md) - будет произведено слияние записей из набора записей с существующими записями регистра. Существующие записи регистра, совпадающие с записями набора записей по значениям ключевых полей (Период, измерения, значения разделителей), будут перезаписаны, а записи набора, для которых нет соответствия среди существующих записей регистра, будут добавлены в регистр. Данное значение параметра допустимо только для независимых регистров сведений.
- [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) - записи регистра, совпадающие с записями набора записей по значениям ключевых полей, будут удалены из регистра.

Значения параметров [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) или [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) для набора записей подчиненного регистра сведений могут быть использованы только в том случае, если свойство [ДлинаНомераСтроки](ОбъектМетаданных_ТабличнаяЧасть.ДлинаНомераСтроки_(MetadataObject_TabularSection.LineNumberLength).md) набора записей имеет значение [Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает набор записей в базу данных. В зависимости от переданного параметра, может быть выполнено добавление записей или их замещение.   
Если для регистра сведений подчиненного регистратору производится запись в режиме добавления, то после выполнения записи набор очищается.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для регистров сведений, подчиненных регистратору, при вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу набор записей очищается (удаляются записи из набора).

Пример:

|  |
| --- |
| НаборКурсов.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
