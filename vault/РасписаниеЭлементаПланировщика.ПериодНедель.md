РасписаниеЭлементаПланировщика (PlannerItemSchedule)

ПериодНедель (WeeksPeriod)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Период времени в неделях, через которые повторяется интервал:

- 1 - задание может быть запущено каждую неделю;
- 2 - через неделю, и т.д.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
