Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), связанного с отбором. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [ЭлементОтбора](ЭлементОтбора_(FilterItem).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
