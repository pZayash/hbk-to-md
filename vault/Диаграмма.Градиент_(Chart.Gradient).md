Диаграмма (Chart)

Градиент (Gradient)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак градиентной закраски плоских диаграмм и пузырьковой диаграммы.  
[Истина](lang__def_BooleanTrue.md) - градиентная закраска.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется только для вертикальных и горизонтальных плоских гистограмм, и пузырьковой диаграммы.

Использование в версии:

Доступен, начиная с версии 8.0.
