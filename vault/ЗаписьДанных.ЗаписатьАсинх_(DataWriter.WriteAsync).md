# ЗаписьДанных.ЗаписатьАсинх

ЗаписьДанных (DataWriter)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Запись двоичных данных

Синтаксис:

ЗаписатьАсинх(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Записать экземпляр объекта [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) в поток.

Описание варианта метода:

Записывает в поток данные из объекта типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Запись результата чтения данных

Синтаксис:

ЗаписатьАсинх(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).   
Записать экземпляр объекта [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md) в поток.

Описание варианта метода:

Записывает в целевой поток данные из объекта типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   

Описание:

Асинхронная функция. Записывает данные в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.
