Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементНастройкиОформления](ЭлементНастройкиОформления.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента коллекции (нумерация с 0).  

Свойства:

[\<Имя настройки\> (\<Setting name \>)](НастройкаОформления.Имя_настройки.md)  

Конструкторы:

[По умолчанию](НастройкаОформления.По_умолчанию.md)  

Описание:

Коллекция элементов типа [ЭлементНастройкиОформления](ЭлементНастройкиОформления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

[ЭлементУсловногоОформления](ЭлементУсловногоОформления.md), свойство [Оформление](ЭлементУсловногоОформления.Оформление.md)
