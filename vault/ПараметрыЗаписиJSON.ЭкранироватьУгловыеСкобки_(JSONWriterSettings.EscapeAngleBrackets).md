# ПараметрыЗаписиJSON.ЭкранироватьУгловыеСкобки

ПараметрыЗаписиJSON (JSONWriterSettings)

ЭкранироватьУгловыеСкобки (EscapeAngleBrackets)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли экранированы знаки угловых скобок при записи документа JSON.  
После создания объекта данное свойство имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
