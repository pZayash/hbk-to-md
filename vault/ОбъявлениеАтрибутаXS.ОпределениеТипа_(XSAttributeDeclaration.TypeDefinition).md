ОбъявлениеАтрибутаXS (XSAttributeDeclaration)

ОпределениеТипа (TypeDefinition)

Доступен, начиная с версии 8.1.

Синтаксис:

ОпределениеТипа()

Возвращаемое значение:

Тип: [ОпределениеПростогоТипаXS](ОпределениеПростогоТипаXS_(XSSimpleTypeDefinition).md), [ОпределениеСоставногоТипаXS](ОпределениеСоставногоТипаXS_(XSComplexTypeDefinition).md).   

Описание:

Получает определение типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет значение свойства type definition информационного пространства атрибута (<http://www.w3.org/TR/xmlschema-1/#a-simple_type_definition>) или элемента (<http://www.w3.org/TR/xmlschema-1/#type_definition>).  
Значением будет либо анонимное определение типа (дочернее), если имеется, либо разыменованное значение типа по имени типа, либо определение типа элемента, указанного в свойстве substitutionGroup (группа подстановки) - только для элемента, либо значение базового типа (Any Type | Any Simple Type).

Использование в версии:

Доступен, начиная с версии 8.1.
