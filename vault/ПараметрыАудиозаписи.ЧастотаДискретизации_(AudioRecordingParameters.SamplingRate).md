ПараметрыАудиозаписи (AudioRecordingParameters)

ЧастотаДискретизации (SamplingRate)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Устанавливает частоту дискретизации звука для записи.  
По умолчанию: 44100.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
