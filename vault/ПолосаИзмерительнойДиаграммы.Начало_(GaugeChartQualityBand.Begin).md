ПолосаИзмерительнойДиаграммы (GaugeChartQualityBand)

Начало (Begin)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит начало полосы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Измеряется в абсолютной величине. Значение не должно превышать значение свойства [Конец](ПолосаИзмерительнойДиаграммы.Конец_(GaugeChartQualityBand.End).md).

Использование в версии:

Доступен, начиная с версии 8.0.
