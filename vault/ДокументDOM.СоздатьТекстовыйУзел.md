ДокументDOM (DOMDocument)

СоздатьТекстовыйУзел (CreateTextNode)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьТекстовыйУзел(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.   
Строка, представляющая собой значение текстового узла.

Возвращаемое значение:

Тип: [ТекстDOM](ТекстDOM.md).   

Описание:

Создает узел DOM - текст.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
