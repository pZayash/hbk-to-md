АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Резервный (Enable)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, является ли рабочий процесс резервным:

- [Истина](lang__def_BooleanTrue.md) - процесс является резервным.
- [Ложь](lang__def_BooleanFalse.md) - процесс не является резервным.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
