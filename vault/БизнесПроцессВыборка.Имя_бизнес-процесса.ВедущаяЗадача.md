БизнесПроцессВыборка.<Имя бизнес-процесса> (BusinessProcessSelection.<Business process name>)

ВедущаяЗадача (HeadTask)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   
Содержит ссылку на ведущую задачу, которую бизнес-процесс будет автоматически помечать как выполненную при своем завершении. Ведущая задача может быть использована во вложенном бизнес-процессе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
