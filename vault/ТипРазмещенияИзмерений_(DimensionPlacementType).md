Доступен, начиная с версии 8.0.

Значения

[Вместе (Together)](ТипРазмещенияИзмерений.Вместе_(DimensionPlacementType.Together).md)  
[Отдельно (Separately)](ТипРазмещенияИзмерений.Отдельно_(DimensionPlacementType.Separately).md)  
[ОтдельноИТолькоВИтогах (SeparatelyAndInTotalsOnly)](ТипРазмещенияИзмерений.ОтдельноИТолькоВИтогах_(DimensionPlacementType.SeparatelyAndInTotalsOnly).md)  

Описание:

Содержит варианты расположения измерений в сводной таблице и построителе отчета.  
Для объекта [СводнаяТаблица](СводнаяТаблица_(PivotTable).md) значения Отдельно и ОтдельноИТолькоВИтогах интерпретируются как одно и то же значение - ОтдельноИТолькоВИтогах.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.
