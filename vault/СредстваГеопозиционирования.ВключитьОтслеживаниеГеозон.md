СредстваГеопозиционирования (LocationTools)

ВключитьОтслеживаниеГеозон (EnableGeofencesMonitoring)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ВключитьОтслеживаниеГеозон(<Геозона>)

Параметры:

<Геозона> (обязательный)

Тип: [Геозона](Геозона.md), [Массив](Массив.md).   
Описание отслеживаемых геозон.  
Идентификатор геозоны не должно быть пустым.  
Радиус геозоны должен быть больше нуля.

Описание:

Включает отслеживание одной или нескольких геозон для текущей информационной базы. Если для геозоны заданы некорректные параметры (пустой идентификатор, нулевой или отрицательный радиус), будет вызвано исключение.   
Отслеживаемые геозоны закрепляют за приложением, но разделяются по информационным базам.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

Особенности поведения в различных операционных системах:

- Windows Phone и Android - Если приложение не активно (не работает или свернуто), отображается локальное уведомление. У этого уведомления в качестве заголовка используется имя приложения, а в качестве текста - информация о входе/выходе из геозоны. Нажатие на уведомление запускает (восстанавливает) приложение и передает ему данные о геозоне;
- Android - количество геозон в приложении не должно превышать 100. Попытка отслеживать большее количество геозон приведет к вызову исключения. Если приложение не активно (свернуто или закрыто), при пересечении границы геозоны отображается локальное уведомление, содержащее имя приложения. Нажатие на уведомление запускает приложение и передает ему данные о геозоне;
- iOS - количество геозон в приложении не должно превышать 20. Попытка отслеживать большее количество геозон, приведет к вызову исключения.

Использование в версии:

Доступен, начиная с версии 8.3.10.
