ЭлементОтбора (FilterItem)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение, по которому будет осуществляться фильтрация.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если [ВидСравнения](ЭлементОтбора.ВидСравнения_(FilterItem.ComparisonType).md) принимает значение [ВСписке](ВидСравнения.ВСписке_(ComparisonType.InList).md), [ВСпискеПоИерархии](ВидСравнения.ВСпискеПоИерархии_(ComparisonType.InListByHierarchy).md), [НеВСписке](ВидСравнения.НеВСписке_(ComparisonType.NotInList).md) или [НеВСпискеПоИерархии](ВидСравнения.НеВСпискеПоИерархии_(ComparisonType.NotInListByHierarchy).md), то тип значения [СписокЗначений](СписокЗначений_(ValueList).md).

Использование в версии:

Доступен, начиная с версии 8.0.
