ОболочкаHTMLДокумента (HTMLDocumentShell)

Получить (Get)

Не рекомендуется использовать, начиная с версии 8.2.

Рекомендуется использовать:

- [ПолучитьCOMОбъект](ОболочкаHTMLДокумента.ПолучитьCOMОбъект.md)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить()

Возвращаемое значение:

Тип: [COMОбъект](COMОбъект.md).   

Описание:

Предоставляет доступ к HTML-документу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.2.

Описание изменено в версии 8.3.15.
