МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьПредставленияШаблонаСообщенияАсинх (SetMessageTemplatePresentationsAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПредставленияШаблонаСообщенияАсинх(<Идентификатор>, <ПредставлениеШаблона>, <ПредставленияПодстановок>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplateID).md).   
Идентификатор шаблона сообщения, для которого устанавливается описание.

<ПредставлениеШаблона> (обязательный)

Тип: [Строка](lang__def_String.md).   
Представление шаблона.

<ПредставленияПодстановок> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив строк - список представлений подстановок.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Для интеграции типа WhatsApp устанавливает представление шаблона и представления его подстановок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [УстановитьПредставленияШаблонаСообщения](МенеджерСистемыВзаимодействия.УстановитьПредставленияШаблонаСообщения_(CollaborationSystemManager.SetMessageTemplatePresentations).md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
