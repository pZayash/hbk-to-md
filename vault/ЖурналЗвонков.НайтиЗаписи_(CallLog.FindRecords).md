# ЖурналЗвонков.НайтиЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Телефония](Телефония-2.md) › [ЖурналЗвонков](ЖурналЗвонков_(CallLog).md) › [Методы](ЖурналЗвонков__Методы.md)

ЖурналЗвонков (CallLog)

НайтиЗаписи (FindRecords)

Доступен, начиная с версии 8.3.5.

Синтаксис:

НайтиЗаписи(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md).   
Задает условия поиска.  
В качестве условий отбора могут быть указаны следующие поля:

- НомерТелефона - тип [Строка](lang__def_String.md). Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно_(DataCompositionComparisonType.Equal).md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно_(DataCompositionComparisonType.NotEqual).md), [Содержит](ВидСравненияКомпоновкиДанных.Содержит_(DataCompositionComparisonType.Contains).md), [НеСодержит](ВидСравненияКомпоновкиДанных.НеСодержит_(DataCompositionComparisonType.NotContains).md).
- Дата - тип [Дата](lang__def_Date.md). Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше_(DataCompositionComparisonType.Greater).md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше_(DataCompositionComparisonType.Less).md), [Равно](ВидСравненияКомпоновкиДанных.Равно_(DataCompositionComparisonType.Equal).md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно_(DataCompositionComparisonType.GreaterOrEqual).md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно_(DataCompositionComparisonType.LessOrEqual).md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно_(DataCompositionComparisonType.NotEqual).md).
- ТипЗвонка - тип [ТипЗвонкаЖурналаЗвонков](ТипЗвонкаЖурналаЗвонков_(CallLogCallType).md). Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно_(DataCompositionComparisonType.Equal).md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно_(DataCompositionComparisonType.NotEqual).md).
- Продолжительность - тип [Число](lang__def_Number.md). Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше_(DataCompositionComparisonType.Greater).md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше_(DataCompositionComparisonType.Less).md), [Равно](ВидСравненияКомпоновкиДанных.Равно_(DataCompositionComparisonType.Equal).md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно_(DataCompositionComparisonType.GreaterOrEqual).md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно_(DataCompositionComparisonType.LessOrEqual).md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно_(DataCompositionComparisonType.NotEqual).md).

Элементы отбора могут быть сгруппированы по "И" или по "ИЛИ". Группировка по "НЕ" в отборе не поддерживается.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Элементом возвращаемого массива является [ЗаписьЖурналаЗвонков](ЗаписьЖурналаЗвонков_(CallLogRecord).md).

Описание:

Осуществляет поиск записей в журнале звонков.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
