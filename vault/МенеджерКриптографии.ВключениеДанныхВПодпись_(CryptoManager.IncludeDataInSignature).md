МенеджерКриптографии (CryptoManager)

ВключениеДанныхВПодпись (IncludeDataInSignature)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак включения данных в подпись при ее формировании.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
