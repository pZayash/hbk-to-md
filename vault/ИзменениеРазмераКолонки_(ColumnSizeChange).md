# ИзменениеРазмераКолонки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.0.

Значения

[Изменять (Change)](ИзменениеРазмераКолонки.Изменять_(ColumnSizeChange.Change).md)  
[НеИзменять (DontChange)](ИзменениеРазмераКолонки.НеИзменять_(ColumnSizeChange.DontChange).md)  

Описание:

Определяет набор режимов изменения размера колонки. Используется для определения свойства [ИзменениеРазмера](КолонкаТабличногоПоля.ИзменениеРазмера_(TableBoxColumn.SizeChange).md) колонки табличного поля.

Доступность:

Толстый клиент.

См. также:

[КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md), свойство [ИзменениеРазмера](КолонкаТабличногоПоля.ИзменениеРазмера_(TableBoxColumn.SizeChange).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
