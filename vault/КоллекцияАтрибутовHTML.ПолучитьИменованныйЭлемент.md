КоллекцияАтрибутовHTML (HTMLAttributeMap)

ПолучитьИменованныйЭлемент (GetNamedItem)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя узла-атрибута HTML.

Возвращаемое значение:

Тип: [АтрибутHTML](АтрибутHTML.md).   

Описание:

Возвращает атрибут HTML по указанному имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
