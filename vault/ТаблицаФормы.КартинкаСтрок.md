ТаблицаФормы (FormTable)

КартинкаСтрок (RowsPicture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Для получения картинки строк в таблице может быть указана колонка c типом Число или Булево. Если указана колонка с типом Число, то значение в колонке представляет собой индекс в наборе картинок, задаваемых данным свойством. Если указана колонка с типом Булево, значение Истина означает индекс 0, а значение Ложь означает индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.2.
