ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

ПринадлежностьПоследовательностям (BelongingToSequences)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ФиксированнаяКоллекция](ФиксированнаяКоллекция.md).   
Предоставляет доступ к наборам записей последовательностей, в которых регистрируется документ. Представляет собой коллекцию значений типа [ПоследовательностьНаборЗаписей.](ПоследовательностьНаборЗаписей.Имя_последовательности.md)<[Имя последовательности](ПоследовательностьНаборЗаписей.Имя_последовательности.md)>. Свойства коллекции содержат наборы записей последовательностей. Имена свойств совпадают с именами последовательностей как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
