РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRow)

Вставить (Insert)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Вставить(<Индекс>, <Действие>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить кнопку панели кнопок сообщения системы взаимодействия.

<Действие> (необязательный)

Тип: [ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.md).   
Действие кнопки панели кнопок сообщения системы взаимодействия.  
Значение по умолчанию: [ОтправитьСообщение](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщение.md).

<Текст> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Надпись на кнопке панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.md).   

Описание:

Вставляет кнопку панели кнопок сообщения системы взаимодействия в заданную позицию и возвращает ее.   
Если метод вызван не в модуле бота, будет сгенерировано исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
