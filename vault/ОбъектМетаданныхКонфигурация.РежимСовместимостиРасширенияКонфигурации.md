Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [РежимСовместимости](РежимСовместимости.md).   
Показывает, установлен ли режим совместимости расширения с платформой определенной версии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для основной конфигурации - Неопределено.

Использование в версии:

Доступен, начиная с версии 8.3.8.
