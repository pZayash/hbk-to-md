МенеджерСредствУстройства (DeviceToolsManager)

ВключитьФонарик (TurnFlashlightOn)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ВключитьФонарик(<Включить>)

Параметры:

<Включить> (необязательный)

Тип: Булево.   
Определяет, будет ли включен фонарик:

- Истина - включить фонарик;
- Ложь - выключить фонарик.

Описание:

Управляет состоянием фонарика.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом метода следует использовать метод [ПоддерживаетсяФонарик](МенеджерСредствУстройства.ПоддерживаетсяФонарик.md).

Доступность:

Толстый клиент.

См. также:

[МенеджерСредствУстройства](МенеджерСредствУстройства.md), метод [ПоддерживаетсяФонарик](МенеджерСредствУстройства.ПоддерживаетсяФонарик.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
