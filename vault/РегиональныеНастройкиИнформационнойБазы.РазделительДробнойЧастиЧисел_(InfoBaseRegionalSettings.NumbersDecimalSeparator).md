РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

РазделительДробнойЧастиЧисел (NumbersDecimalSeparator)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Строка из одного символа. Символ, который должен использоваться для разделения целой и дробной части при получении представления данных типа [Число](lang__def_Number.md).   
Если пустая строка, то используется соответствующая региональная настройка текущего сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
