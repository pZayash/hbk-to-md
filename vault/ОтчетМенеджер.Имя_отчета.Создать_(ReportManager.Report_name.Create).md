ОтчетМенеджер.<Имя отчета> (ReportManager.<Report name>)

Создать (Create)

Доступен, начиная с версии 8.0.

Синтаксис:

Создать()

Возвращаемое значение:

Тип: [ОтчетОбъект.\<Имя отчета\>](ОтчетОбъект.Имя_отчета_(ReportObject.Report_name).md).   

Описание:

Создает новый экземпляр отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ОтчетОстаткиНоменклатуры = Отчеты.ОстаткиНоменклатуры.Создать(); ОтчетОстаткиНоменклатуры.ПолучитьФорму().Открыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
