ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередВыполнением (BeforeExecute)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md), в которой производится проверка выполнения.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   
Выполняемая задача.

<Отказ>

Тип: Булево.   
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то выполнение задачи произведено не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при вызове метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу.md) у задачи и после обработки задачей события [ПередВыполнением](ЗадачаОбъект.Имя_задачи.ПередВыполнением.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md).  
В этом обработчике не допускаются интерактивные вызовы (открытие форм, предупреждений).

Использование в версии:

Доступен, начиная с версии 8.0.
