ПроверкаКачестваСканированияДокументов (DocumentScanningCheckingQuality)

ТребоватьСреднееПредупреждатьНижеВысокого (RequireMediumWarnBelowHigh)

Доступен, начиная с версии 8.3.23.

Описание:

Пользователь не может принять документ, чье обобщенное качество ниже Среднего (меньше 4).   
Пользователю показываются предупреждения о качестве документа, чье обобщенное качество ниже Высокого (меньше 7).

Использование в версии:

Доступен, начиная с версии 8.3.23.
