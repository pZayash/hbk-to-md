РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных (BinaryDataStorageDataCopiesPlacementMode)

РазмещениеДанныхВХранилищеИВБазеДанныхПриНедоступностиХранилища (DataPlacementToStorageAndToDatabaseOnInaccessibleStorage)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных_(BinaryDataStorageDataCopiesPlacementMode).md).   
Двоичные данные размещаются в хранилище двоичных данных.   
Если запись в хранилище двоичных данных невозможна, то двоичные данные будут помещены в базу данных.  
Если хранилище двоичных данных недоступно, то прочитать двоичные данные можно, только если они размещены в базе данных.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
