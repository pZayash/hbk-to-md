РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

Итог (Total)

Доступен, начиная с версии 8.0.

Синтаксис:

Итог(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля записей набора, по которому необходимо подсчитать итог.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Числовое значения результата вычисления.

Описание:

Суммирует значения всех строк в указанном поле набора.  
Если для поля набора записей установлен тип и он единственный, то при суммировании будет предприниматься попытка преобразования значения к типу [Число](lang__def_Number.md).  
Если полю не присвоены типы, то в процессе суммирования будут принимать участие только значения, имеющие тип [Число](lang__def_Number.md), значения других типов будут игнорироваться.  
Если для поля несколько типов и среди них есть тип [Число](lang__def_Number.md), то в процессе суммирования будут принимать участие только значения, имеющие тип [Число](lang__def_Number.md), значения других типов будут игнорироваться.  
Если для поля несколько типов и среди них нет типа [Число](lang__def_Number.md), то результатом будет значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
