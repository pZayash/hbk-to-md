# РежимПолнотекстовогоПоиска

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Разные](Разные.md)

Доступен, начиная с версии 8.2.

Значения

[Запретить (Disable)](РежимПолнотекстовогоПоиска.Запретить_(FullTextSearchMode.Disable).md)  
[Разрешить (Enable)](РежимПолнотекстовогоПоиска.Разрешить_(FullTextSearchMode.Enable).md)  

Описание:

Разрешает (запрещает) все операции полнотекстового поиска: обновление индексов, очистка индексов, поиск.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
