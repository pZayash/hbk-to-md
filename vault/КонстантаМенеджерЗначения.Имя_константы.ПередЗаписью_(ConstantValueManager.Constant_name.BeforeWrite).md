# КонстантаМенеджерЗначения.<Имя константы>.ПередЗаписью

КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи константы. Процедура-обработчик вызывается после начала транзакции записи, но до начала записи константы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
