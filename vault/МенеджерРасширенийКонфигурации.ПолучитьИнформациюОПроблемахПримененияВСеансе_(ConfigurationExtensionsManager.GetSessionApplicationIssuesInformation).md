МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

ПолучитьИнформациюОПроблемахПримененияВСеансе (GetSessionApplicationIssuesInformation)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьИнформациюОПроблемахПримененияВСеансе()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Возвращает массив объектов типа [ИнформацияОПроблемеПримененияРасширенияКонфигурации](ИнформацияОПроблемеПримененияРасширенияКонфигурации_(ConfigurationExtensionApplicationIssueInformation).md).

Описание:

Возвращает список проблем применения расширения конфигурации в текущем сеансе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
