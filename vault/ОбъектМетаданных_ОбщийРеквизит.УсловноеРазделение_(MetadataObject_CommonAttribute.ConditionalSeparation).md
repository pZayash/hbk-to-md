# ОбъектМетаданных: ОбщийРеквизит.УсловноеРазделение

ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

УсловноеРазделение (ConditionalSeparation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Константа](ОбъектМетаданных_Константа_(MetadataObject_Constant).md).   
Указывает константу конфигурации булевого типа, не входящую в состав общего реквизита. Значение константы задает использование разделителя в данном сеансе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
