Глобальный контекст (Global context)

ХранилищеСистемныхНастроек (SystemSettingsStorage)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md).   
Содержит объект менеджера стандартного хранилища настроек, предназначенный для доступа к системным настройкам.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
