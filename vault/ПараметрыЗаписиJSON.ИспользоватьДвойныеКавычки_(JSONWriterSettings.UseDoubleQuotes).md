# ПараметрыЗаписиJSON.ИспользоватьДвойныеКавычки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [JSON](JSON-2.md) › [ПараметрыЗаписиJSON](ПараметрыЗаписиJSON_(JSONWriterSettings).md) › [Свойства](ПараметрыЗаписиJSON__Свойства.md)

ПараметрыЗаписиJSON (JSONWriterSettings)

ИспользоватьДвойныеКавычки (UseDoubleQuotes)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет использование двойных кавычек при записи свойств и значений JSON.  
После создания объекта данное свойство имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
