Расширение формы клиентского приложения для объекта таблицы внешнего источника данных

ЗначениеКопирования (CopyingValue)

Доступен, начиная с версии 8.3.5.

Описание:

Значение для копирования.

Примечание:

Тип параметра соответствует типу объекта, копия которого будет создана.

Использование в версии:

Доступен, начиная с версии 8.3.5.
