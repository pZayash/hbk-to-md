Запрос (Query)

На основании текста запроса

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Запрос(<ТекстЗапроса>)

Параметры:

<ТекстЗапроса> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст запроса.  
Значение по умолчанию: Пустая строка.

Описание:

Создает запрос с указанным текстом.

Пример:

|  |
| --- |
| // Создание запроса с текстом.  Запрос = Новый Запрос(ТекстЗапроса); // Создание запроса без текста.  Запрос = Новый Запрос; |

Использование в версии:

Доступен, начиная с версии 8.0.
