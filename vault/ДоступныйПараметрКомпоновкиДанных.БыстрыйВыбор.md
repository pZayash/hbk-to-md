ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Управляет режимом выбора значения.   
Истина - выбор будет выполняться из выпадающего списка.  
Ложь - из окна выбора вида расчета.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
