ОписаниеМоделиРаспознаванияРечи (SpeechToTextModelDescription)

ПредставлениеАкустики (AcousticsPresentation)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Строка на языке текущего пользователя или указанном языке из набора строк на разных языках конфигурации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [НСтр](Глобальный_контекст.НСтр.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
