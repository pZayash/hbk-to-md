Описание:

Содержит варианты типов соединения значений по сериям в гистограммах с накоплением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartValuesBySeriesConnectionType.

Использование в версии:

Доступен, начиная с версии 8.3.1.
