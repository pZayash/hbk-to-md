Рабочий процесс (IWorkingProcessInfo)

MainPort (MainPort)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер основного IP-порта рабочего процесса. Этот порт выделяется динамически при старте рабочего процесса из диапазонов портов, определенных для соответствующего рабочего сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
