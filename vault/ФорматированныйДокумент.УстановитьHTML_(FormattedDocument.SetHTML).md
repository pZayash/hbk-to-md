ФорматированныйДокумент (FormattedDocument)

УстановитьHTML (SetHTML)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьHTML(<ТекстHTML>, <Вложения>)

Параметры:

<ТекстHTML> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текстовая строка, содержащая HTML.

<Вложения> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая вложения, которые будут установлены в форматированный документ.  
Ключ структуры - название или имя файла картинки. Значение - [Картинка](Картинка_(Picture).md).

Описание:

Формирует форматированный документ из переданных HTML-составляющих.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
