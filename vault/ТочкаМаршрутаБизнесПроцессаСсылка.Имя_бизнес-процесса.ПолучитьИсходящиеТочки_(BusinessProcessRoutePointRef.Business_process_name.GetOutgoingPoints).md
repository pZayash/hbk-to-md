ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПолучитьИсходящиеТочки (GetOutgoingPoints)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьИсходящиеТочки()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает массив точек маршрута бизнес-процесса, исходящих из указанной точки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение_(BusinessProcessRoutePointType.Split).md).

Использование в версии:

Доступен, начиная с версии 8.0.
