ТипДиаграммы (ChartType)

ГистограммаСНакоплением (StackedColumn)

Доступен, начиная с версии 8.0.

Описание:

Вертикальная гистограмма с накоплением: значения в каждой серии суммируются со значениями в предыдущих. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График.md).

Использование в версии:

Доступен, начиная с версии 8.0.
