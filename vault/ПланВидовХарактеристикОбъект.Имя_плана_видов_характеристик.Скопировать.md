Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md).   

Описание:

Создает новый элемент плана видов характеристик копированием существующего.

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.
