МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблоныСообщенияАсинх (GetMessageTemplatesAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблоныСообщенияАсинх(<Интеграция>)

Параметры:

<Интеграция> (обязательный)

Тип: [ИдентификаторИнтеграцииСистемыВзаимодействия](ИдентификаторИнтеграцииСистемыВзаимодействия_(CollaborationSystemIntegrationID).md).   
Идентификатор интеграции, для которой возвращается список шаблонов сообщений.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив объектов типа [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplate).md).

Описание:

Асинхронная функция. Получает массив, содержащий шаблоны сообщения системы взаимодействия.  
Для интеграции типа WhatsApp возвращает список шаблонов сообщений.  
Для остальных интеграций возвращает пустой массив.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [ПолучитьШаблоныСообщения](МенеджерСистемыВзаимодействия.ПолучитьШаблоныСообщения_(CollaborationSystemManager.GetMessageTemplates).md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
