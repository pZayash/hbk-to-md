Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ОбработкаПолученияСпискаНавигационныхСсылок (URLListGetProcessing)

Доступен, начиная с версии 8.3.21.

Синтаксис:

ОбработкаПолученияСпискаНавигационныхСсылок(<Элемент>, <СписокНавигационыхСсылок>, <КлючПоУмолчанию>)

Параметры:

<Элемент>

Тип: [ТаблицаФормы](ТаблицаФормы.md).   
Таблица, для которой выполняется получение списка навигационных ссылок.

<СписокНавигационыхСсылок>

Тип: [СписокВыбораНавигационнойСсылки](СписокВыбораНавигационнойСсылки.md).   
Если у таблицы свойство [РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки](Расширение_таблицы_формы_для_динамического_списка.РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки.md) имеет значение Истина, и объект/запись в текущей строке поддерживает навигационную ссылку, то в данный параметр добавляется элемент – вариант ссылки на объект/запись в текущей строке динамического списка. В обработчике можно добавить новые варианты, переставить или удалить стандартные варианты, указать ключ любого из элементов списка, в т.ч. стандартного, в качестве ключа по умолчанию.  
Если у таблицы установлено свойство [РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки](Расширение_таблицы_формы_для_динамического_списка.РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки.md) имеет значение Ложь, то обработчику передается пустой <СписокНавигационыхСсылок>.

<КлючПоУмолчанию>

Тип: Произвольный.   
В данный параметр следует поместить значение ключа из списка ссылок, соответствующего ссылке, которую требуется использовать по умолчанию.  
Значение по умолчанию: Неопределено.

Описание:

Вызывается перед открытием диалога получения навигационной ссылки. Если у таблицы свойство [РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки](Расширение_таблицы_формы_для_динамического_списка.РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки.md) имеет значение Истина, и объект/запись в текущей строке поддерживает навигационную ссылку,  
в параметр <СписокНавигационыхСсылок> добавляется элемент – вариант ссылки на объект/запись в текущей строке динамического списка. В обработчике можно добавить новые варианты, переставить или удалить стандартные варианты, указать ключ любого из элементов списка, в т.ч. стандартного, в качестве ключа по умолчанию.  
Если свойство [РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки](Расширение_таблицы_формы_для_динамического_списка.РазрешитьПолучатьНавигационнуюСсылкуТекущейСтроки.md) таблицы установлено в Ложь, то обработчику передается пустой <СписокНавигационыхСсылок>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.21.
