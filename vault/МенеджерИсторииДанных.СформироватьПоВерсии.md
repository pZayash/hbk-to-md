МенеджерИсторииДанных (DataHistoryManager)

СформироватьПоВерсии (GenerateFromVersion)

Доступен, начиная с версии 8.3.11.

Синтаксис:

СформироватьПоВерсии(<Данные>, <НомерВерсии>)

Параметры:

<Данные> (обязательный)

Тип: [СправочникСсылка.](СправочникСсылка.Имя_справочника.md)<[Имя справочника](СправочникСсылка.Имя_справочника.md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса.md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи.md)<[Имя задачи](ЗадачаСсылка.Имя_задачи.md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов.md)>, [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>, [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.md)>, [КонстантаКлючЗначения.](КонстантаКлючЗначения.Имя_константы.md)<[Имя константы](КонстантаКлючЗначения.Имя_константы.md)>.   
Ссылка на объект или ключ записи регистра сведений, по которым создана версия.

<НомерВерсии> (обязательный)

Тип: Число.   
Номер версии.

Возвращаемое значение:

Тип: [СправочникОбъект.](СправочникОбъект.Имя_справочника.md)<[Имя справочника](СправочникОбъект.Имя_справочника.md)>, [ДокументОбъект.](ДокументОбъект.Имя_документа.md)<[Имя документа](ДокументОбъект.Имя_документа.md)>, [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>, [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>, [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>, [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>, [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>, [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.md)>, [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы.md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы.md)>.   

Описание:

Получить объект для доступа к данным, сконструированный по данным версии. Если указанная версия объекта - удаление данных, будет восстановлено состояние данных перед удалением.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Версия  = ИсторияДанных.СформироватьПоВерсии( Справочники.Валюты.НайтиПоНаименованию("Рубль"), 2); Версия.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.3.11.
