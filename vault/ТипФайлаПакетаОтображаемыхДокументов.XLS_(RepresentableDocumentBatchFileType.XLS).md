ТипФайлаПакетаОтображаемыхДокументов (RepresentableDocumentBatchFileType)

XLS (XLS)

Доступен, начиная с версии 8.3.12.

Описание:

Файл в формате MS Excel. Сохранение в данном формате выполняет сохранение в формате MS Excel, используемом по умолчанию (эквивалентно [XLS97](ТипФайлаТабличногоДокумента.XLS97_(SpreadsheetDocumentFileType.XLS97).md)).

Использование в версии:

Доступен, начиная с версии 8.3.12.
