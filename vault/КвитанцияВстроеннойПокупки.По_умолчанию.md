КвитанцияВстроеннойПокупки (InAppPurchaseReceipt)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый КвитанцияВстроеннойПокупки(<Квитанция>, <Сервис>)

Параметры:

<Квитанция> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Строка](lang__def_String.md).   
Квитанция в виде двоичных данных или имя файла квитанции.

<Сервис> (обязательный)

Тип: [СервисВстроенныхПокупок](СервисВстроенныхПокупок_(InAppPurchaseService).md).   
Сервис встроенных покупок.

Описание:

Создает квитанцию встроенной покупки на основании тела квитанции и сервиса встроенных покупок.

Использование в версии:

Доступен, начиная с версии 8.3.14.
