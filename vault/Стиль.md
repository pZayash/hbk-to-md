Описание:

Представляет собой стиль, определенный в конфигурации. Может быть ссылкой на конкретный стиль конфигурации или значением, созданным конструктором стиля без параметров (стиль по умолчанию).  
Редактор стиля позволяет модифицировать для обычного приложения все предопределенные элементы стиля, а для приложения только следующие предопределенные цвета стиля:

- [ЦветАктивности](ЦветаСтиля.ЦветАктивности.md),
- [ЦветАкцента](ЦветаСтиля.ЦветАкцента.md),
- [ЦветНавигации](ЦветаСтиля.ЦветНавигации.md),
- [ЦветДополнительнойНавигации](ЦветаСтиля.ЦветДополнительнойНавигации.md),
- [ЦветВажного](ЦветаСтиля.ЦветВажного.md).

Также можно редактировать значения для собственных элементов стиля вида [Цвет](Цвет.md). Вне зависимости от возможности редактирования, существующие предопределенные элементы стиля разыменовываются в реальные цвета, используемые платформой. Для оформления элементов формы рекомендуется использовать именно цвета стиля, а не конкретные значения цвета. Другими словами, если для фона какого-либо элемента требуется использовать тот же цвет, что и цвет формы, то свойству, отвечающему за цвет фона элемента, следует установить значение стиля [ЦветФонаФормы](ЦветаСтиля.ЦветФонаФормы.md) (значение из системного набора значений [ЦветаСтиля](ЦветаСтиля.md)), а не конкретное значение цвета. В этом случае, при изменении цвета фона формы, цвет фона выбранного элемента также изменится.  
Если необходимо отключить используемый стиль и установить стиль оформления платформы по умолчанию, следует использовать конструкцию вида:   

|  |
| --- |
| ГлавныйСтиль = Новый Стиль(); |

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.
