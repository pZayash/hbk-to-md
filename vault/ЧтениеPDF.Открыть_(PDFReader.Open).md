ЧтениеPDF (PDFReader)

Открыть (Open)

Доступен, начиная с версии 8.3.16.

Вариант синтаксиса: Открыть файл

Синтаксис:

Открыть(<ИмяФайла>, <ОписаниеОповещения>)

Параметры:

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для чтения PDF-документа.

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения PDF-документа.

Описание:

Открывает PDF-документ для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [НачатьОткрытие](ЧтениеPDF.НачатьОткрытие_(PDFReader.BeginOpen).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
