Диаграмма (Chart)

ПропускатьБазовоеЗначение (HideBaseValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает на необходимость пропускать показ значений, равных базовому.   
Если Истина, то во всех типах диаграмм кроме:

- График с областями;
- График с областями и накоплением;
- Круговая;
- Круговая объёмная;
- Биржевая;
- Биржевая свеча;
- Изометрическая непрерывная;
- Изометрическая лента.

значения, равные базовому значению, не отображаются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
