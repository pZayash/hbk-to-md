Расширение поля формы для поля ввода (Form field extension for a text box)

ОбновлениеТекстаРедактирования (EditTextUpdate)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОбновлениеТекстаРедактирования](ОбновлениеТекстаРедактирования_(EditTextUpdate).md).   
Управляет режимом обновления текста.  
Значение по умолчанию [Авто](ОбновлениеТекстаРедактирования.Авто_(EditTextUpdate.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме совместимости конфигурации [Версия8\_2\_13](РежимСовместимости.Версия8_2_13_(CompatibilityMode.Version8_2_13).md) не влияет на поведение поля ввода управляемой формы.  
В остальных случаях влияет на то, в каких ситуациях будет обновляться текст в поле ввода при установке значения.

Использование в версии:

Доступен, начиная с версии 8.2.
