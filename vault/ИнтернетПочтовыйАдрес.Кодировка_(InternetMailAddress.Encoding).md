ИнтернетПочтовыйАдрес (InternetMailAddress)

Кодировка (Encoding)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит кодировку для отображаемых имен. Если кодировка не указана, будет использоваться значение кодировки из свойства [Кодировка](ИнтернетПочтовоеСообщение.Кодировка_(InternetMailMessage.Encoding).md), объекта [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
