ОписаниеЭлементаСпискаВыбораНавигационнойСсылки (URLChoiceListItemDescription)

Представление (Presentation)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Представление ссылки, которое в форме получения ссылки будет использовано в списке выбора ссылки.   
Если не задано, в качестве представления используется представление значения свойства [Ключ](ОписаниеЭлементаСпискаВыбораНавигационнойСсылки.Ключ.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
