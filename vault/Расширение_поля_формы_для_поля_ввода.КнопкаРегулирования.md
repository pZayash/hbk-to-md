Расширение поля формы для поля ввода (Form field extension for a text box)

КнопкаРегулирования (SpinButton)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Неопределено, Булево.   
Содержит признак наличия/отсутствия кнопки регулирования (стрелки вверх и вниз).  
Неопределено - автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
