Расширение поля ввода для описания типа (Text box for type description extension)

ДоступныеТипы (AvailableTypes)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов.md).   
Позволяет задать набор типов, доступных для выбора. Так же задается ограничение для квалификаторов примитивных типов Число, Строка, Дата.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
