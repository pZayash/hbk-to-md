СредстваКриптографии (CryptoTools)

НачатьПолучениеИнформацииМодуляКриптографии (BeginGettingCryptoModuleInformation)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеИнформацииМодуляКриптографии(<ОписаниеОповещения>, <ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после после получения информации о модуле криптографии со следующими параметрами:

- <ИнформацияМодуля> - значение объекта [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии_(CryptoModuleInformation).md) или [Неопределено](lang__def_Undefined.md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяМодуляКриптографии> (обязательный)

Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу-библиотеке, реализующей требуемый набор функций.  
Параметр используется, если менеджер создается для операционной системы Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: [Число](lang__def_Number.md).   
Задает тип модуля криптографии.

Возвращаемое значение:

Если модуль криптографии не поддерживается (не найден по имени для MS Windows, либо не экспортируются требуемые функции на Linux), то при вызове метода из объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md) первым параметром возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Начинает получение информации о модуле криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СредстваКриптографии](СредстваКриптографии_(CryptoTools).md), метод [ПолучитьИнформациюМодуляКриптографии](СредстваКриптографии.ПолучитьИнформациюМодуляКриптографии_(CryptoTools.GetCryptoModuleInformation).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
