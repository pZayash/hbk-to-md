# МенеджерКриптографии.НайтиКлючевойКонтейнерАсинх

МенеджерКриптографии (CryptoManager)

НайтиКлючевойКонтейнерАсинх (FindKeyContainerAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

НайтиКлючевойКонтейнерАсинх(<Сертификат>, <ИспользоватьКлюч>)

Параметры:

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Сертификат ключа проверки электронной подписи, для которого выполняется поиск контейнера ключа электронной подписи (закрытого ключа).

<ИспользоватьКлюч> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения поиска контейнера по значению ключа проверки электронной подписи (открытого ключа).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результатом выполнения обещания является объект типа [Строка](lang__def_String.md), содержащий наименование (идентификатор) ключевого контейнера.

Описание:

Асинхронная функция. Выполняет поиск контейнера ключа (закрытого ключа) электронной подписи. Поиск выполняется по наличию параметров контейнера и провайдера в расширенных свойствах сертификата, находящегося в системном хранилище.   
Если сертификата нет в хранилище, или у него отсутствует привязка к контейнеру, то если параметр <ИспользоватьКлюч> имеет значение [Истина](lang__def_BooleanTrue.md), поиск контейнера будет выполнен по значению ключа проверки электронной подписи (открытого ключа) сертификата, указанного в параметре <Сертификат>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
