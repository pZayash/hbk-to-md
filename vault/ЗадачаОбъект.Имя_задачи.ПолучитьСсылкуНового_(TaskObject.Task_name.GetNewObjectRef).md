# ЗадачаОбъект.<Имя задачи>.ПолучитьСсылкуНового

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Методы](ЗадачаОбъект.Имя_задачи__Методы.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](ЗадачаОбъект.Имя_задачи.УстановитьСсылкуНового_(TaskObject.Task_name.SetNewObjectRef).md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи_(TaskManager.Task_name).md), метод [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку_(TaskManager.Task_name.GetRef).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
