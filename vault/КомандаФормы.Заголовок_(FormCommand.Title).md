Заголовок (Title)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит отображаемый текст команды.

Доступность:

Мобильный клиент, сервер, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Изменение доступно только на сервере.

Использование в версии:

Доступен, начиная с версии 8.2.
