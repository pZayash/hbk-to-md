Прикладной объект, преобразованный из данных формы в значение, для которого необходимо получить соответствие с именем реквизита формы.

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает имя атрибута формы, связанного с объектом, передаваемым в качестве аргумента функции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Во внешнем соединении метод игнорируется.
