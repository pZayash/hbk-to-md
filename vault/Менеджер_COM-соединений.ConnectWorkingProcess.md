Менеджер COM-соединений (COM connector)

ConnectWorkingProcess (ConnectWorkingProcess)

Доступен, начиная с версии 8.1.

Синтаксис:

ConnectWorkingProcess(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя или IP-адрес рабочего процесса, с которым должно быть установлено соединение, в формате: <протокол>://<адрес\_или\_имя>:<порт>  
Например:  
testcomp  
tcp://testcomp  
tcp://testcomp:2354  
tcp://43.73.34.11:5342.

Возвращаемое значение:

Тип: [Соединение с рабочим процессом](Соединение_с_рабочим_процессом.md).   

Описание:

Создает административное соединение с рабочим процессом кластера серверов 1С:Предприятия 8.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
