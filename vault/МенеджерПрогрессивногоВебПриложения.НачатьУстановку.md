МенеджерПрогрессивногоВебПриложения (ProgressiveWebApplicationManager)

НачатьУстановку (BeginInstallation)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьУстановку()

Описание:

Инициирует установку прогрессивного веб-приложения, показывая соответствующий диалог.  
Если прогрессивное веб-приложение не доступно к установке (свойство [Доступно](МенеджерОкнаВнешнегоСайта.Доступно.md) в значении Ложь), тогда при вызове данного метода будет сгенерировано исключение "Прогрессивное веб-приложение не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
