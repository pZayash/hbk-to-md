ПараметрыКачестваСканированияДокументов (DocumentScanningQualityParameters)

КачествоБелого (WhitePaperQuality)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Указывает, на сколько бумага документа белая и однотонная.  
Значение данного свойства учитывается при расчете значения [ОбобщенноеКачество](ПараметрыКачестваСканированияДокументов.ОбобщенноеКачество_(DocumentScanningQualityParameters.GeneralizedQuality).md), если в параметре <ПараметрыСканирования> для методов [ПоказатьСканированиеДокументов](СредстваМультимедиа.ПоказатьСканированиеДокументов_(MultimediaTools.ShowDocumentScanning).md) и [СканироватьДокументыАсинх](СредстваМультимедиа.СканироватьДокументыАсинх_(MultimediaTools.ScanDocumentsAsync).md) используется свойство [БелыйФонДокумента](ПараметрыСканированияДокументов.БелыйФонДокумента_(DocumentScanningParameters.WhiteBackgroundDocument).md).  
Значение – число, нормированное от 1 до 10:

- [1.0, 4.0) – низкое качество, документ совсем не похож на документ, напечатанный на белой однотонной бумаге.
- [4.0, 7.0) – среднее качество, документ похож на напечатанный на белой однотонной бумаге, но освещен не совсем белым цветом.
- [7.0, 10.0] – высокое качество, бумага документа белая, однотонная.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
