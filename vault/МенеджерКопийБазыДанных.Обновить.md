МенеджерКопийБазыДанных (DatabaseCopiesManager)

Обновить (Update)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Обновить(<Копия>, <Порционное>, <КоличествоЗаданийОбновления>)

Параметры:

<Копия> (обязательный)

Тип: [МенеджерКопииБазыДанных](МенеджерКопииБазыДанных.md), Строка.   
Объект менеджера копии или имя копии.

<Порционное> (необязательный)

Тип: Булево.   
Обновление выполняется в течение максимум 20 минут, после чего прерывается.  
Значение по умолчанию: Истина.

<КоличествоЗаданийОбновления> (необязательный)

Тип: Число.   
Число заданий, используемых в процессе первоначального копирования таблицы в копию.   
По умолчанию - 0, при этом количество заданий - 4.  
Количество заданий всегда = 1, вне зависимости от значения параметра, в случае, если копия базы данных расположена на сервере Oracle.  
Дополнительно, для копии на масштабируемом Дата акселераторе определяет количество параллельных заданий обновления экземпляров копии. Значение по умолчанию = 0, при этом количество параллельных заданий - 4.  
Значение по умолчанию: 0.

Описание:

Обновляет таблицы в копии. Для новых таблиц выполняется первоначальное копирование данных из основной базы данных в базу копии.  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если указанная <Копия> базы данных уже используется другой копией или информационной базой, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.26.
