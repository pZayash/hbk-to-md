# ФормаКлиентскогоПриложения.НачатьРедактированиеЭлемента

ФормаКлиентскогоПриложения (ClientApplicationForm)

НачатьРедактированиеЭлемента (BeginEditingItem)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьРедактированиеЭлемента()

Описание:

Если [ТекущийЭлемент](ФормаКлиентскогоПриложения.ТекущийЭлемент_(ClientApplicationForm.CurrentItem).md) является полем ввода, выполняет переход в режим редактирования и одно из следующих действий:

- открытие панели ввода,
- отображение экранной клавиатуры,
- отображение календаря,
- отображение списка выбора,
- автоподбор значения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
