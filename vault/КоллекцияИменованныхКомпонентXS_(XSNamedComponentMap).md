# КоллекцияИменованныхКомпонентXS

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Схема XML](Схема_XML.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ОпределениеПростогоТипаXS](ОпределениеПростогоТипаXS_(XSSimpleTypeDefinition).md), [ОпределениеСоставногоТипаXS](ОпределениеСоставногоТипаXS_(XSComplexTypeDefinition).md), [ОпределениеГруппыАтрибутовXS](ОпределениеГруппыАтрибутовXS_(XSAttributeGroupDefinition).md), [ОпределениеГруппыМоделиXS](ОпределениеГруппыМоделиXS_(XSModelGroupDefinition).md), [ОбъявлениеАтрибутаXS](ОбъявлениеАтрибутаXS_(XSAttributeDeclaration).md), [ОбъявлениеЭлементаXS](ОбъявлениеЭлементаXS_(XSElementDeclaration).md), [ОпределениеОграниченияИдентичностиXS](ОпределениеОграниченияИдентичностиXS_(XSIdentityConstraintDefinition).md), [ОпределенияXPathXS](ОпределенияXPathXS_(XSXPathDefinition).md), [ОбъявлениеНотацииXS](ОбъявлениеНотацииXS_(XSNotationDeclaration).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются компоненты схемы XML.  
Возможно обращение к компоненте схемы XML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияИменованныхКомпонентXS.Количество_(XSNamedComponentMap.Count).md)  
[Получить (Get)](КоллекцияИменованныхКомпонентXS.Получить_(XSNamedComponentMap.Get).md)  

Описание:

Коллекция компонентов схемы XML, имеющих имя.

Доступность:

Сервер, толстый клиент, внешнее соединение.
