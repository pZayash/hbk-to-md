РазрешенныйВнешнийМодуль (AllowedExternalModule)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового внешнего модуля или обновление существующего. Внешний модуль идентифицируется своей контрольной суммой.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
