МенеджерКриптографии (CryptoManager)

ДобавитьАрхивнуюМеткуВремениАсинх (AddArchiveTimestampAsync)

Доступен, начиная с версии 8.3.20.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ДобавитьАрхивнуюМеткуВремениАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md) или [ФайловыйПоток](ФайловыйПоток_(FileStream).md).

Описание варианта метода:

Данные подписи возвращаются методом как [Обещание](Обещание_(Promise).md) с результатом [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Данные подписи сохраняются в файл

Синтаксис:

ДобавитьАрхивнуюМеткуВремениАсинх(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Подпись может размещаться в файле (в этом случае указывается имя файла) или представлена как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md) или [ФайловыйПоток](ФайловыйПоток_(FileStream).md).

<ВыходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Имя файла или поток, в который будут выгружены данные подписи.  
Если [Неопределено](lang__def_Undefined.md), то результат работы метода вернется в возвращаемом значении типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание варианта метода:

Данные подписи записываются в файл. Возвращает [Обещание](Обещание_(Promise).md) с результатом [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
[Обещание](Обещание_(Promise).md). Результат выполнения обещания – двоичные данные подписи с добавленной архивной меткой времени (если ВыходныеДанные = [Неопределено](lang__def_Undefined.md)) иначе [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Добавляет в подпись архивную метку времени версии 3.   
При добавлении метки выполняется обращение к серверам меток времени, указанным в поле [АдресаСерверовМетокВремени](МенеджерКриптографии.АдресаСерверовМетокВремени_(CryptoManager.TimestampServersAddresses).md).   
  
Для добавления архивной метки времени подпись должна иметь тип CAdES-BES или выше.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.
