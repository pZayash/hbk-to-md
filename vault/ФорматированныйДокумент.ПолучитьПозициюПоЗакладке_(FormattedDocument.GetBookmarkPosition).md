ФорматированныйДокумент (FormattedDocument)

ПолучитьПозициюПоЗакладке (GetBookmarkPosition)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПозициюПоЗакладке(<Закладка>)

Параметры:

<Закладка> (обязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Закладка в форматированном документе, для которой надо получить позицию.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает позицию символа по переданной закладке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
