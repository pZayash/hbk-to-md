МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьЛицензиюПоТелефону (AcquireLicenseByPhone)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьЛицензиюПоТелефону(<ЗапросНаПолучениеЛицензии>, <ОтветЦентраЛицензирования>)

Параметры:

<ЗапросНаПолучениеЛицензии> (обязательный)

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии_(LicenseAcquisitionRequest).md).   
Данные для запроса на получение лицензии.   
Свойства [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md), [Порт](ЗапросНаПолучениеЛицензии.Порт_(LicenseAcquisitionRequest.Port).md), [ДляВсехПользователей](ЗапросНаПолучениеЛицензии.ДляВсехПользователей_(LicenseAcquisitionRequest.ForAllUsers).md), [ИмяФайлаЛицензии](ЗапросНаПолучениеЛицензии.ИмяФайлаЛицензии_(LicenseAcquisitionRequest.LicenseFileName).md) не используются.

<ОтветЦентраЛицензирования> (обязательный)

Тип: [Строка](lang__def_String.md).   
Последовательность цифр, которую сообщил по телефону центр лицензирования в ответ на запрос по телефону. Может содержать только десятичные цифры от 0 по 9.

Описание:

Создает файл лицензии на основании параметров текущего компьютера, данных о лицензии и ответа центра лицензирования, переданного по телефону.  
Файл лицензии помещается на текущем компьютере в каталог поиска программной лицензиии для всех пользователей или для текущего пользователя.   
При любой ошибке генерируется исключение и файл лицензии не создается.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
