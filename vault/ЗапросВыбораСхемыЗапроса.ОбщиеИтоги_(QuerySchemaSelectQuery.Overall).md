ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

ОбщиеИтоги (Overall)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость вывода в результат общих итогов.  
Соответствует ключевому слову ОБЩИЕ в выражении ИТОГИ ПО языка запросов.  
Доступно только, если у схемы запроса выключен режим компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
