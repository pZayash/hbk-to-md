Глобальный контекст (Global context)

ТекущийРежимЗапуска (CurrentRunMode)

Доступен, начиная с версии 8.2.

Синтаксис:

ТекущийРежимЗапуска()

Возвращаемое значение:

Тип: [РежимЗапускаКлиентскогоПриложения](РежимЗапускаКлиентскогоПриложения.md), Неопределено.   
Неопределено, если режим запуска не является ни обычным, ни управляемым - например, внешнее соединение.

Описание:

Получает режим запуска клиентского приложения текущего сеанса работы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
