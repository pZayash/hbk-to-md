Расширение тестируемого дополнения элемента формы для отображения состояния просмотра (Tested form item addition extension for a view status representation)

ПолучитьТекстыЭлементовСостоянияПросмотра (GetViewStatusItemTexts)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьТекстыЭлементовСостоянияПросмотра()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Если в момент вызова метода нет ни одного элемента состояния просмотра (поиск в таблице не выполняется), возвращается пустой массив.

Описание:

Получает фиксированный массив с текстами элементов состояния просмотра.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
