АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПрофильБезопасногоРежима (SafeModeProfile)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, может ли профиль использоваться в качестве профиля безопасного режима: его имя может быть указано в конфигурациях при использовании внешних отчетов, обработок и расширений конфигурации:

- [Истина](lang__def_BooleanTrue.md) – профиль можно использовать.
- [Ложь](lang__def_BooleanFalse.md) – профиль нельзя использовать.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
