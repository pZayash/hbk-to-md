РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ГраницаСлева (LeftBorder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак наличия левой границы у рисунка. Если значение свойства Истина, рисунок будет иметь левую границу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо для рисунков любого типа, кроме [Прямая](ТипРисункаТабличногоДокумента.Прямая.md), [Эллипс](ТипРисункаТабличногоДокумента.Эллипс.md).

Использование в версии:

Доступен, начиная с версии 8.0.
