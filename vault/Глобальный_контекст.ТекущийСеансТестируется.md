Глобальный контекст (Global context)

ТекущийСеансТестируется (CurrentSessionIsTested)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ТекущийСеансТестируется()

Возвращаемое значение:

Тип: Булево.   
Истина – клиентское приложение запущено с параметром /TestClient, механизм тестирования корректно работает и готов к выполнению команд от менеджера тестирования.  
Ложь – клиент тестирования не смог стартовать корректно или клиентское приложение запущено без параметра /TestClient.

Описание:

Возвращает признак работы клиента в режиме клиента тестирования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
