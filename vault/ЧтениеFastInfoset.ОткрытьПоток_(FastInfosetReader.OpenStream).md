ЧтениеFastInfoset (FastInfosetReader)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОткрытьПоток(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток, содержащий данные в формате Fast Infoset.

Описание:

Устанавливает Fast Infoset-поток для чтения данным объектом. Если перед вызовом данного метода уже выполнялось чтение данных из другого файла или потока, то чтение прекращается и объект инициализируется для чтения из указанного потока.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
