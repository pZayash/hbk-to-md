Расширение поля формы для поля календаря (Form field extension for a calendar field)

Ширина (Width)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит ширину календаря. Задается в символах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется, если значение свойства [ШиринаВМесяцах](Расширение_поля_формы_для_поля_календаря.ШиринаВМесяцах.md)=0 или для режимов совместимости до [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md).

Использование в версии:

Доступен, начиная с версии 8.2.
