Расширение тестируемого поля формы для форматированного документа (Tested form field extension for a formatted document)

ЗаписатьСодержимоеВФайл (WriteContentToFile)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ЗаписатьСодержимоеВФайл(<СохранитьКак>)

Параметры:

<СохранитьКак> (необязательный)

Тип: Булево.   
Истина - отображает диалоговое окно "Сохранить как" для выбора имени файла.  
Ложь - сохраняет файл под текущим именем.  
Значение по умолчанию: Ложь.

Описание:

Вызывает сохранение форматированного документа в файл.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

Из-за особенностей реализации веб-клиента метод применим только при тестировании тонкого клиента.

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.25.
