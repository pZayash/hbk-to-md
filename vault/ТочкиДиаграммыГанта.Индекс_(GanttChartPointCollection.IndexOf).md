ТочкиДиаграммыГанта (GanttChartPointCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Точка>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной серии в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс точки в коллекции точек диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
