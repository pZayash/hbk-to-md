ОбъектМетаданных: Таблица (MetadataObject: Table)

РасширенноеПредставлениеОбъекта (ExtendedObjectPresentation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку, которая является названием элемента справочника и используется при формировании заголовка формы элемента таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для объектных таблиц.

Использование в версии:

Доступен, начиная с версии 8.2.
