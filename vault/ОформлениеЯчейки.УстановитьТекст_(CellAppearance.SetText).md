ОформлениеЯчейки (CellAppearance)

УстановитьТекст (SetText)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст ячейки.

Описание:

Устанавливает текст ячейки. Устанавливает свойство [ОтображатьТекст](ОформлениеЯчейки.ОтображатьТекст_(CellAppearance.ShowText).md) в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
