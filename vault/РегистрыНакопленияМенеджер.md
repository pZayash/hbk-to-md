# РегистрыНакопленияМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Регистры накопления](Регистры_накопления.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[РегистрНакопленияМенеджер.\<Имя регистра накопления\>](РегистрНакопленияМенеджер.Имя_регистра_накопления.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры регистров накопления.  
Возможно обращение к менеджеру регистра накопления посредством оператора [...]. В качестве аргумента передается имя регистра накопления, как оно задано в конфигурации.  

Свойства:

[\<Имя регистра накопления\> (\<Accumulation register name\>)](РегистрыНакопленияМенеджер.Имя_регистра_накопления.md)  

Описание:

Предназначен для управления регистрами накопления и предоставляет доступ к значениям типа [РегистрНакопленияМенеджер.](РегистрНакопленияМенеджер.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияМенеджер.Имя_регистра_накопления.md)>. Доступ к объекту осуществляется через свойство глобального контекста [РегистрыНакопления](Глобальный_контекст.РегистрыНакопления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [РегистрыНакопления](Глобальный_контекст.РегистрыНакопления.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
