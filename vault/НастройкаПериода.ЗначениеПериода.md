НастройкаПериода (PeriodSettings)

ЗначениеПериода (PeriodValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Дата периода настраиваемого интервала. От этой даты будет отсчитываться период, если задано значение свойства [ВариантПериода](НастройкаПериода.ВариантПериода.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
