КонтейнерКлючейКриптографии (CryptoKeysContainer)

Из двоичных данных

Доступен, начиная с версии 8.3.10.

Синтаксис:

Новый КонтейнерКлючейКриптографии(<Данные>, <Пароль>)

Параметры:

<Данные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные, на основе которых будет построен контейнер ключей криптографии.

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль контейнера ключей криптографии.

Использование в версии:

Доступен, начиная с версии 8.3.10.
