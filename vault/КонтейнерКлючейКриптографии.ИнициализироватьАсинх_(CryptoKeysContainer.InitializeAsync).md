КонтейнерКлючейКриптографии (CryptoKeysContainer)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из данных которого будет инициализирован контейнер ключей криптографии.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль контейнера ключей криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - созданный [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md).

Описание:

Асинхронная функция. Инициализирует контейнер.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md), метод [НачатьИнициализацию](КонтейнерКлючейКриптографии.НачатьИнициализацию_(CryptoKeysContainer.BeginInitialization).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
