Описание:

Диалог установки параметров отбора версий истории данных, вызывается из формы истории изменений.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Пример:

|  |
| --- |
| Окно = Новый ДиалогОтбораВерсийИсторииДанных(Параметры.Данные); Окно.Отбор = Отбор; Окно.Показать(Новый ОписаниеОповещения("ВыполнитьПослеЗакрытияОтбора",                                         ЭтотОбъект)); |
