АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

РазрешитьВыполнениеВнешнегоКодаВНебезопасномРежиме (AllowExternalCodeExecutionInUnsafeMode)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет разрешение запуска внешнего кода (внешние отчеты, внешние обработки, оператор Выполнить, Вычислить, расширения конфигурации) без безопасного режима, если это указано в конфигурации:

- [Истина](lang__def_BooleanTrue.md) – для любого кода.
- [Ложь](lang__def_BooleanFalse.md) – разрешен запуск без включения безопасного режима только внешних модулей, перечисленных в списке разрешенных внешних модулей.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
