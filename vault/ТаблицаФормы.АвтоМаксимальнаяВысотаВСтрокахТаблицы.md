ТаблицаФормы (FormTable)

АвтоМаксимальнаяВысотаВСтрокахТаблицы (AutoMaxHeightInTableRows)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
На максимальную высоту таблицы накладывается автоматическое ограничение.  
Свойство используется, в одном из случаев:

- если свойство [ВариантУправленияВысотой](ТаблицаФормы.ВариантУправленияВысотой.md) имеет значение [ВСтрокахТаблицы](ВариантУправленияВысотойТаблицы.ВСтрокахТаблицы.md) или [ПоСодержимому](ВариантУправленияВысотойТаблицы.ПоСодержимому.md)
- если значение [Авто](ВариантУправленияВысотойТаблицы.Авто.md) было разыменовано в эти значения.

В противном случае, значение свойства игнорируется.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.10.
