РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии> (AccountingRegisterManager.<Accounting register name>)

ВыбратьПоРегистратору (SelectByRecorder)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьПоРегистратору(<Регистратор>)

Параметры:

<Регистратор> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Значение регистратора.

Возвращаемое значение:

Тип: [РегистрБухгалтерииВыборка.\<Имя регистра бухгалтерии\>](РегистрБухгалтерииВыборка.Имя_регистра_бухгалтерии_(AccountingRegisterSelection.Accounting_register_name).md).   

Описание:

Формирует выборку записей регистра бухгалтерии по регистратору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
