ОписаниеСистемыЛинейныхУравнений (SystemOfLinearEquationsDescription)

КолонкаКоэффициентовВУзлах (FactorsInNodesColumn)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число, Строка.   
Индекс или имя колонки в [ИсточникДанныхУзлов](РасчетСистемЛинейныхУравнений.ИсточникДанныхУзлов.md), которая представляет колонку свободных коэффициентов для этой системы.   
Индексация начинается с 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
