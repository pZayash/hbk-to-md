ДанныеФормыСтруктураСКоллекцией (FormDataStructureAndCollection)

НайтиПоИдентификатору (FindByID)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Число](lang__def_Number.md).   
Идентификатор элемента коллекции.

Возвращаемое значение:

Тип: [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - элемент не найден.

Описание:

Возвращает элемент коллекции по идентификатору.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
