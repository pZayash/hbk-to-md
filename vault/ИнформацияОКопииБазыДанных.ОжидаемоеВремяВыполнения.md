ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ОжидаемоеВремяВыполнения (EstimatedUpdateTime)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Число, Неопределено.   
Прогнозируемое время (в секундах), через которое будет завершено обновление копии базы данных. Рассчитывается на основании статистики предыдущих обновлений. При отсутствии статистики содержит значение Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
