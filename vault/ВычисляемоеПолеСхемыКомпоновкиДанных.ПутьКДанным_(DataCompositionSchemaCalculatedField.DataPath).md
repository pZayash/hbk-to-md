ВычисляемоеПолеСхемыКомпоновкиДанных (DataCompositionSchemaCalculatedField)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя, под которым данное поле будет фигурировать в настройках.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В системе компоновки данных не допускается использовать в качестве пути к данным имена, совпадающие с ключевыми словами:  
ВЫБОР (CASE)  
ЕСТЬ (IS)  
ИНАЧЕ (ELSE)  
КОГДА (WHEN)  
КОНЕЦ (END)  
НЕ (NOT)  
ПОДОБНО (LIKE)  
РАЗЛИЧНЫЕ (DISTINCT)  
СПЕЦСИМВОЛ (ESCAPE)  
ТОГДА (THEN)

Использование в версии:

Доступен, начиная с версии 8.1.
