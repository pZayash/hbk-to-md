Не рекомендуется использовать, начиная с версии 8.3.19.

Рекомендуется использовать:

- [Поля](ДинамическийСписок.Поля_(DynamicList.Fields).md)
- [ОграничениеИспользования](ПолеНабораДанныхСхемыКомпоновкиДанных.ОграничениеИспользования_(DataCompositionSchemaDataSetField.UseRestriction).md)

Синтаксис:

ПолучитьОграниченияИспользованияВОтборе()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Возвращает массив строк-путей компоновки данных, на которые ранее было установлено ограничение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Не рекомендуется использовать, начиная с версии 8.3.19.

Описание изменено в версии 8.3.19.
