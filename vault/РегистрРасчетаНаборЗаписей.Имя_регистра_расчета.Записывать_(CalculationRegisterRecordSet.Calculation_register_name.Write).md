РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Записывать (Write)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Ложь](lang__def_BooleanFalse.md) - не происходит записи набора в информационную базу при вызове [Записать](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.Записать_(CalculationRegisterRecordSet.Calculation_register_name.Write).md) коллекции движений документа, которой принадлежит набор, а также при стандартной обработке проведения документа, если значение свойства метаданного документа "Запись движений при проведении" в Конфигураторе выставлено в "Записывать выбранные".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
