ВременныеТаблицыЗапроса (QueryTempTables)

Найти (Find)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Найти(<НаименованиеТаблицы>)

Параметры:

<НаименованиеТаблицы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы для поиска.

Возвращаемое значение:

Тип: [ВременнаяТаблицаЗапроса](ВременнаяТаблицаЗапроса_(QueryTempTable).md), [Неопределено](lang__def_Undefined.md).   
Если временная таблица с таким именем отсутствует, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает временную таблицу по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если РезультатЗапроса.Колонки.Найти("Цена") <> Неопределено Тогда     ЦеныВЗапросе = Истина; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.3.8.
