КоллекцияНотацийDOM (DOMNotationMap)

УдалитьИменованныйЭлемент (RemoveNamedItem)

Доступен, начиная с версии 8.1.

Синтаксис:

УдалитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя удаляемого узла DOM.

Возвращаемое значение:

Тип: [НотацияDOM](НотацияDOM_(DOMNotation).md).   

Описание:

Удаляет узел из коллекции по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
