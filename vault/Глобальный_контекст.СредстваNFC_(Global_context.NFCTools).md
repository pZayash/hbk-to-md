Глобальный контекст (Global context)

СредстваNFC (NFCTools)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [СредстваNFC](СредстваNFC_(NFCTools).md).   
Свойство предоставляет доступ к средствам NFC мобильной платформы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.23.
