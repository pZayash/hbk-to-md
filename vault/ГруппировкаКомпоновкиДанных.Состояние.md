ГруппировкаКомпоновкиДанных (DataCompositionGroup)

Состояние (State)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [СостояниеЭлементаНастройкиКомпоновкиДанных](СостояниеЭлементаНастройкиКомпоновкиДанных.md).   
Определяет состояние группировки.  
Поведение отключенных и удаленных группировок аналогично неиспользуемым группировкам (Использование = Ложь) с тем отличием, что состояние группировки не влияет на вложенные в нее группировки.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
