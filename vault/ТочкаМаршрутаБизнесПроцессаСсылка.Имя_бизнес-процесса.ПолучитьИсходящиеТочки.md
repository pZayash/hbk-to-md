ПолучитьИсходящиеТочки (GetOutgoingPoints)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьИсходящиеТочки()

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Получает массив точек маршрута бизнес-процесса, исходящих из указанной точки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение.md).

Использование в версии:

Доступен, начиная с версии 8.0.
