КнопкиКоманднойПанели (CommandBarButtons)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Имя>, <Тип>, <Текст>, <Действие>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция в коллекции кнопок, в которую необходимо вставить кнопку.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя кнопки - должно соответствовать правилам имен. Уникально в пределах командной панели.

<Тип> (необязательный)

Тип: [ТипКнопкиКоманднойПанели](ТипКнопкиКоманднойПанели_(CommandBarButtonType).md).   
Тип кнопки командной панели (разделитель, действие, подменю).

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст, который будет выводиться на кнопке.

<Действие> (необязательный)

Тип: [Действие](Действие_(Action).md).   
Действие, которое необходимо выполнить при нажатии на данную кнопку. Для типов кнопок [Разделитель](ТипКнопкиКоманднойПанели.Разделитель_(CommandBarButtonType.Separator).md) и [Подменю](ТипКнопкиКоманднойПанели.Подменю_(CommandBarButtonType.Popup).md), действие указывать не нужно.

Возвращаемое значение:

Тип: [КнопкаКоманднойПанели](КнопкаКоманднойПанели_(CommandBarButton).md).   

Описание:

Вставляет кнопку в указанную позицию коллекции кнопок командной панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| КП = ЭлементыФормы.КоманднаяПанель; НовоеДействие = Новый Действие("МояПроцедураВМодулеФормы"); ТипКнопки = ТипКнопкиКоманднойПанели.Действие; НоваяКнопка = КП.Кнопки.Вставить(0, "Действие1", ТипКнопки,, НовоеДействие); |

Использование в версии:

Доступен, начиная с версии 8.0.
