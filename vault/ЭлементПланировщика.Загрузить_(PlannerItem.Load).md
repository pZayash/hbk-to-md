# ЭлементПланировщика.Загрузить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md) › [ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md) › [Методы](ЭлементПланировщика__Методы.md)

ЭлементПланировщика (PlannerItem)

Загрузить (Load)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Загрузить(<Источник>)

Параметры:

<Источник> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, из которой необходимо заполнить свойства элемента планировщика.

Описание:

Заполняет соответствующие свойства элемента планировщика из значений переданной структуры.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md), метод [Выгрузить](ЭлементПланировщика.Выгрузить_(PlannerItem.Unload).md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
