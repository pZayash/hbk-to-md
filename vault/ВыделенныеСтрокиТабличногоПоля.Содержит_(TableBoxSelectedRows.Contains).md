ВыделенныеСтрокиТабличногоПоля (TableBoxSelectedRows)

Содержит (Contains)

Доступен, начиная с версии 8.0.

Синтаксис:

Содержит(<Строка>)

Параметры:

<Строка> (обязательный)

Строка табличного поля, наличие которой определяется в коллекции выделенных строк.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, имеется ли в данной коллекции выделенных строк указанная строка табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
