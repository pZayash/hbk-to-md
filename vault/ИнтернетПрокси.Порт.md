ИнтернетПрокси (InternetProxy)

Порт (Port)

Доступен, начиная с версии 8.0.

Синтаксис:

Порт(<Протокол>)

Параметры:

<Протокол> (необязательный)

Тип: Строка.   
Имя протокола, для которого запрашивается порт прокси-сервера.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: Число.   
При указании пустой строки возвращается порт прокси-сервера, установленного по умолчанию.

Описание:

Получает порт прокси-сервера по имени протокола.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
