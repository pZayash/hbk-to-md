Элементы коллекции:

Произвольный  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения типа [КлючИЗначение](КлючИЗначение_(KeyAndValue).md), где Ключ - путь ключевого поля, Значение - значение этого ключевого поля.  
Возможно обращение к значению ключевого поля посредством оператора [...]. В качестве аргумента передается строка (путь компоновки данных соответствующего ключевого поля).

Описание:

Набор значений, однозначно идентифицирующих запись динамического списка.   
Используется для динамических списков с типом ключа Составной. Объекты данного типа используются в таблицах формы, связанных с такими списками, в качестве значения свойства [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен . Имя типа XDTO:
