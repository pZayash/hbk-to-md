Расширение поля формы для поля ввода (Form field extension for a text box)

ПолучитьВыделенныеМножественныеЗначения (GetSelectedMultipleValues)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьВыделенныеМножественныеЗначения()

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Возвращает массив идентификаторов объектов [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md) или [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md) для выделенных множественных значений. Этот идентификатор не привязан к позиции элемента в списке или таблице значений.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
