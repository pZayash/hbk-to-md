СерияДиаграммы (ChartSeries)

Раздвигать (Separate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак выдвижения сектора круговой диаграммы.  
Истина - сектор выдвигается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для круговых диаграмм в случае, если не установлено автоматическое раздвижение, т.е. свойство диаграммы [АвтоРаздвижениеСерий](Диаграмма.АвтоРаздвижениеСерий.md) равно Ложь.

См. также:

[Диаграмма](Диаграмма.md), свойство [АвтоРаздвижениеСерий](Диаграмма.АвтоРаздвижениеСерий.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
