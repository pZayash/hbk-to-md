# СредстваТелефонии.ПоддерживаетсяНаборНомера

СредстваТелефонии (TelephonyTools)

ПоддерживаетсяНаборНомера (NumberDialingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяНаборНомера(<ВыбратьСразу>)

Параметры:

<ВыбратьСразу> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет режим проверки поддержки набора номера:

- [Истина](lang__def_BooleanTrue.md) - проверка выполняется непосредственно.
- [Ложь](lang__def_BooleanFalse.md) - при проверке требуется подтверждение пользователя.
- [Неопределено](lang__def_Undefined.md) - проверяется доступность набора номера любым способом.

  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
С подтверждением пользователя возвращает [Истина](lang__def_BooleanTrue.md):

- на Android - при наличии телефонного модуля,
- на iOS - при наличии системного приложения Телефон,
- на Windows Phone - никогда.

Без подтверждения пользователя возвращает [Истина](lang__def_BooleanTrue.md):

- на Android - при наличии телефонного модуля,
- на iOS - для iOS младше версии 10.3 при наличии системного приложения Телефон,
- на Windows Phone - всегда на телефоне, никогда на планшете.

Описание:

Определяет, поддерживается ли набор номера на данном устройстве.  
В зависимости от переданного параметра, проверка выполняется непосредственно или с подтверждением пользователем. Если значение переданного параметра - [Неопределено](lang__def_Undefined.md), проверяется доступность набора номера любым способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
