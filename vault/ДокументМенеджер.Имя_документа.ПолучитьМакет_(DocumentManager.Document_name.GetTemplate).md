# ДокументМенеджер.<Имя документа>.ПолучитьМакет

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументМенеджер.<Имя документа>](ДокументМенеджер.Имя_документа_(DocumentManager.Document_name).md) › [Методы](ДокументМенеджер.Имя_документа__Методы.md)

ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет_(MetadataObject_Template).md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md); другой объект, который может быть макетом.   

Описание:

Получает один из макетов, определенных для документа в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
