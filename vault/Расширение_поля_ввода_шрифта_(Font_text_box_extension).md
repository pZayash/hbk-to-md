# Расширение поля ввода шрифта

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Общие интерфейсные объекты](Общие_интерфейсные_объекты.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность редактирования и хранения в элементе управления [ПолеВвода](ПолеВвода_(TextBox).md) шрифта.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
