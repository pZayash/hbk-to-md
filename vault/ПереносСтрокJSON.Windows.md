ПереносСтрокJSON (JSONLineBreak)

Windows (Windows)

Доступен, начиная с версии 8.3.6.

Описание:

Строки переносятся. Для переноса используются символы переноса строк Microsoft Windows - \r\n.

Использование в версии:

Доступен, начиная с версии 8.3.6.
