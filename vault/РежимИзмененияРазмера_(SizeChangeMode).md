# РежимИзмененияРазмера

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[БыстроеИзменение (QuickChange)](РежимИзмененияРазмера.БыстроеИзменение_(SizeChangeMode.QuickChange).md)  
[Обычный (Normal)](РежимИзмененияРазмера.Обычный_(SizeChangeMode.Normal).md)  

Описание:

Содержит варианты изменения размера.   
Используется, например, для указания режима изменения размера колонок табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: SizeChangeMode.

См. также:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md), свойство [РежимИзмененияРазмераКолонки](ОбластьЯчеекТабличногоДокумента.РежимИзмененияРазмераКолонки_(SpreadsheetDocumentRange.ColumnSizeChangeMode).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
