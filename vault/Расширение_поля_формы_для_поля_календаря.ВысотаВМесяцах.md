Расширение поля формы для поля календаря (Form field extension for a calendar field)

ВысотаВМесяцах (HeightInMonths)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Высота календаря в месяцах. Если равна 0, то используется значение свойства [Высота](ОписаниеОтображаемогоОбъектаPDF.Высота.md). Значение по умолчанию 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Имеет смысл только для режимов совместимости выше [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md). В противном случае свойство доступно, но его изменение не производится.

Использование в версии:

Доступен, начиная с версии 8.3.3.
