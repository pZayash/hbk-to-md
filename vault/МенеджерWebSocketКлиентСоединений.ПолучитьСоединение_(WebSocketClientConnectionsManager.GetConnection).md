МенеджерWebSocketКлиентСоединений (WebSocketClientConnectionsManager)

ПолучитьСоединение (GetConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьСоединение(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор соединения.

Возвращаемое значение:

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md), [Неопределено](lang__def_Undefined.md).   
Возвращает объект [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md) или  
[Неопределено](lang__def_Undefined.md) - если соединение отсутствует.

Описание:

Получает объект соединения с WebSocket-клиентом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
