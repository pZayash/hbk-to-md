# Расширения конфигурации

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › [Общие объекты](Общие_объекты.md)

Механизм расширений конфигурации позволяет создавать специальные конфигурации-расширения, которые подключаются к расширяемой (основной) конфигурации и содержат расширяющие и собственные объекты, добавляемые к расширяемой (основной) конфигурации, которые добавляют или изменяют поведение объектов из неё.
