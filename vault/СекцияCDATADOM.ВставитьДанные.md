СекцияCDATADOM (DOMCDATASection)

ВставитьДанные (InsertData)

Доступен, начиная с версии 8.1.

Синтаксис:

ВставитьДанные(<Смещение>, <Строка>)

Параметры:

<Смещение> (обязательный)

Тип: Число.   
Смещение, по которому определяется позиция вставки символьных данных.

<Строка> (обязательный)

Тип: Строка.   
Строка, вставляемая в символьные данные.

Описание:

Вставляет строку в символьные данные.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных;
- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
