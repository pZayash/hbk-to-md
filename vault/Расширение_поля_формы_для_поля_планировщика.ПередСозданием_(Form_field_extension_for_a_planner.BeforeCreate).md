Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередСозданием (BeforeCreate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередСозданием(<Начало>, <Конец>, <ЗначенияИзмерений>, <Текст>, <СтандартнаяОбработка>)

Параметры:

<Начало>

Тип: [Дата](lang__def_Date.md).   
Дата, на которой начинается выделенная область. Эта дата будет отображаться в окне быстрого создания элемента планировщика и применена к создаваемому элементу планировщика.  
Возвращаемый параметр.

<Конец>

Тип: [Дата](lang__def_Date.md).   
Дата, на которой заканчивается выделенная область. Эта дата будет отображаться в окне быстрого создания элемента планировщика и применена к создаваемому элементу планировщика.  
Возвращаемый параметр.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Коллекция значений измерений. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения, в качестве значения выступает [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения.   
Будет применено к создаваемому элементу планировщика.  
Возвращаемый параметр.

<Текст>

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, который будет содержаться в окне быстрого создания элемента планировщика. Значение параметра будет отображаться в окне быстрого редактирования и применено к создаваемому элементу планировщика.  
Возвращаемый параметр.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметру установлено значение [Ложь](lang__def_BooleanFalse.md), то окно быстрого создания элемента планировщика не будет открыто.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает после выделения мышью некоторой области и после успешного перетаскивания элемента. При стандартной обработке показывается окно быстрого создания элемента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
