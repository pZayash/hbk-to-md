Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Расписание](ДиалогРасписанияЭлементаПланировщика.Расписание_(PlannerItemScheduleDialog.Schedule).md) диалог закрыли нажатием кнопки "ОК", [Неопределено](lang__def_Undefined.md) - в противном случае.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
