Лицензия (ILicenseInfo)

RMngrAddress (RMngrAddress)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит адрес сервера, на котором запущен процесс, получивший лицензию.

Доступность:

Интеграция.

Примечание:

Если лицензию получило клиентское приложение, то содержит пустую строку.

Использование в версии:

Доступен, начиная с версии 8.2.
