ТочкиДиаграммыГанта (GanttChartPointCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс точки.

Возвращаемое значение:

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   

Описание:

Вставляет точку с указанным индексом.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
