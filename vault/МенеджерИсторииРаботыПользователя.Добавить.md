МенеджерИсторииРаботыПользователя (UserWorkHistoryManager)

Добавить (Add)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: По навигационной ссылке

Синтаксис:

Добавить(<НавигационнаяСсылка>)

Параметры:

<НавигационнаяСсылка> (обязательный)

Тип: Строка.   
Ссылка на объект информационной базы, точку навигации или другой объект.

Вариант синтаксиса: По ссылке на объект

Синтаксис:

Добавить(<Ссылка>)

Параметры:

<Ссылка> (обязательный)

Ссылка на объект информационной базы - элемент справочника, документ, бизнес-процесс, задачу, ключ записи регистра сведений.

Описание:

Добавляет элемент в историю работы пользователя.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
