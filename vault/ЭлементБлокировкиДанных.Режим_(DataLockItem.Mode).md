ЭлементБлокировкиДанных (DataLockItem)

Режим (Mode)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [РежимБлокировкиДанных](РежимБлокировкиДанных_(DataLockMode).md).   
Определяет режим управляемой блокировки данных.  
Значение по умолчанию: [Исключительный](РежимБлокировкиДанных.Исключительный_(DataLockMode.Exclusive).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
