ДанныеФормыЭлементДерева (FormDataTreeItem)

<Имя свойства> (<Property name>)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Набор свойств элемента дерева зависит от того, на основе какого объекта построено данное дерево. Может включать в себя свойства, значения которых будут являться другие универсальные структуры, универсальные коллекции и [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
