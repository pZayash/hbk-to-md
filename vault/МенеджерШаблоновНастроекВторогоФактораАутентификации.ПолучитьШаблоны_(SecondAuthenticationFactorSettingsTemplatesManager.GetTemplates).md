МенеджерШаблоновНастроекВторогоФактораАутентификации (SecondAuthenticationFactorSettingsTemplatesManager)

ПолучитьШаблоны (GetTemplates)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьШаблоны()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [ШаблонНастройкиВторогоФактораАутентификации](ШаблонНастройкиВторогоФактораАутентификации_(SecondAuthenticationFactorSettingTemplate).md).

Описание:

Возвращает все шаблоны настройки второго фактора аутентификации, существующие в информационной базе. Выполнение метода требует наличия права "Администратор".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
