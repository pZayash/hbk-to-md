НаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSets)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [НаборДанныхЗапросМакетаКомпоновкиДанных](НаборДанныхЗапросМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetQuery).md), [НаборДанныхОбъектМакетаКомпоновкиДанных](НаборДанныхОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetObject).md), [НаборДанныхОбъединениеМакетаКомпоновкиДанных](НаборДанныхОбъединениеМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetUnion).md).

Возвращаемое значение:

Тип: [НаборДанныхЗапросМакетаКомпоновкиДанных](НаборДанныхЗапросМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetQuery).md), [НаборДанныхОбъектМакетаКомпоновкиДанных](НаборДанныхОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetObject).md), [НаборДанныхОбъединениеМакетаКомпоновкиДанных](НаборДанныхОбъединениеМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetUnion).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод вернет [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
