ИнтернетПочтовоеСообщение (InternetMailMessage)

ДатаОтправления (PostingDate)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата отправления письма. Значение представлено датой, локальной для почтового клиента отправителя.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
