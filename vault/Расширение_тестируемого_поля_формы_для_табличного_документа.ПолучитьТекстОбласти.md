Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ПолучитьТекстОбласти (GetAreaText)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьТекстОбласти(<Область>)

Параметры:

<Область> (необязательный)

Тип: Строка.   
Область табличного документа.

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает текст области.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
