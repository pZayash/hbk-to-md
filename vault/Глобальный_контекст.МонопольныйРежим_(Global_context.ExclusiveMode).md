Глобальный контекст (Global context)

МонопольныйРежим (ExclusiveMode)

Доступен, начиная с версии 8.0.

Синтаксис:

МонопольныйРежим()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - монопольная блокировка была установлена на текущую область данных сеанса или текущая область данных является подобластью монопольно заблокированной этим же сеансом области; [Ложь](lang__def_BooleanFalse.md) - в противном случае.  
Для режима запуска "Мобильное приложение (сервер)" результат всегда [Истина](lang__def_BooleanTrue.md).

Описание:

Определяет, используется ли в данный момент монопольный режим работы с информационной базой.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если МонопольныйРежим() Тогда     // Выполнить обработку  Иначе     Предупреждение(НСтр("ru = 'Для выполнения обработки необходим монопольный режим!';          | en = 'Exclusive mode is needed in order to execute the processing!'"), 10); КонецЕсли; |
