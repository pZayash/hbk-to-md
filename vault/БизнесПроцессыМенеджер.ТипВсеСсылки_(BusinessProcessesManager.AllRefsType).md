БизнесПроцессыМенеджер (BusinessProcessesManager)

ТипВсеСсылки (AllRefsType)

Доступен, начиная с версии 8.0.

Синтаксис:

ТипВсеСсылки()

Возвращаемое значение:

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   

Описание:

Получает описание типов, содержащее типы ссылок бизнес-процессов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| МассивТипов = ЭлементыФормы.ПолеВводаВариант.ТипЗначения.Типы(); Для каждого ЭлементМассива Из МассивТипов Цикл     Если БизнесПроцессы.ТипВсеСсылки().СодержитТип(ЭлементМассива) Тогда         МассивНужныхТипов.Добавить(ЭлементМассива);     КонецЕсли;  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
