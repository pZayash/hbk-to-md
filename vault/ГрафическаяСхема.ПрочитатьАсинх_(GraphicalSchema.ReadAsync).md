ГрафическаяСхема (GraphicalSchema)

ПрочитатьАсинх (ReadAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Из файла

Синтаксис:

ПрочитатьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла графической схемы.

Описание варианта метода:

Чтение графической схемы из файла.

Вариант синтаксиса: Из потока

Синтаксис:

ПрочитатьАсинх(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения графической схемы.

Описание варианта метода:

Чтение графической схемы из потока.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Считывает графическую схему из файла.

Доступность:

Тонкий клиент, толстый клиент.
