Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), предназначенные для интерактивной работы с записями регистра расчета. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [РегистрРасчетаКлючЗаписи.](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета_(CalculationRegisterRecordKey.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета_(CalculationRegisterRecordKey.Calculation_register_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
