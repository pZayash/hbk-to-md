ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

ТипДиаграммы (ChartType)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [ТипДиаграммы](ТипДиаграммы_(ChartType).md).   
Содержит тип диаграммы, устанавливаемый при открытии диалога.  
По умолчанию имеет значение [ГистограммаОбъемная](ТипДиаграммы.ГистограммаОбъемная_(ChartType.Column3D).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
