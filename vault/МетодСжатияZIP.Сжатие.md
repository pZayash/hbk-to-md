МетодСжатияZIP (ZIPCompressionMethod)

Сжатие (Deflate)

Не рекомендуется использовать, начиная с версии 8.3.26.

Доступен, начиная с версии 8.0.

Описание:

Осуществляется сжатие 32-битным алгоритмом Deflate.   
Требуется больше времени работы, чем при копировании.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
