ДокументDOM (DOMDocument)

СоздатьОбходДерева (CreateTreeWalker)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьОбходДерева(<Узел>, <Отбор>, <ОбходитьСсылкиНаСущности>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [ДокументDOM](ДокументDOM_(DOMDocument).md), [ЭлементDOM](ЭлементDOM_(DOMElement).md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [НотацияDOM](НотацияDOM_(DOMNotation).md), [СущностьDOM](СущностьDOM_(DOMEntity).md), [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md), [ТекстDOM](ТекстDOM_(DOMText).md), [КомментарийDOM](КомментарийDOM_(DOMComment).md), [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM_(DOMEntityReference).md), [ПространствоИменXPath](ПространствоИменXPath_(XPathNamespace).md).   
Узел DOM - стартовый узел итератора.

<Отбор> (необязательный)

Тип: [ФильтрУзловDOM](ФильтрУзловDOM_(DOMNodeFilter).md).   
Отбор узлов DOM.  
Если не указан, то будут проитерированы все узлы.

<ОбходитьСсылкиНаСущности> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак обхода ссылок на сущности DOM.  
[Истина](lang__def_BooleanTrue.md) - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ОбходДереваDOM](ОбходДереваDOM_(DOMTreeWalker).md).   

Описание:

Создает новый иерархический объект обхода дерева документа DOM, начиная с указанного корневого узла.  
Метод соответствует интерфейсу Treewalker стандарта W3C (Подробнее см. <http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113/traversal.html#Traversal-TreeWalker>).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
