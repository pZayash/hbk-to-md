КатегорияОшибки (ErrorCategory)

НеподдерживаемыйФормат (UnsupportedFormat)

Доступен, начиная с версии 8.3.26.

Описание:

Формат данных не поддерживается.   
Например, ошибка этой категории отображается при попытке открыть архив неподдерживаемого формата.

Использование в версии:

Доступен, начиная с версии 8.3.26.
