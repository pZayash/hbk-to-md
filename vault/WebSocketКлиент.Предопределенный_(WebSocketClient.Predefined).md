WebSocketКлиент (WebSocketClient)

Предопределенный (Predefined)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, что данный объект создан на основании предопределенного объекта конфигурации (имеет значение [Истина](lang__def_BooleanTrue.md)).   
Иначе имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
