НастройкиСериализацииJSON (JSONSerializerSettings)

СериализовыватьМассивыКакОбъекты (SerializeArraysAsObjects)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство имеет значение Истина, то массив будет сериализован как объект JSON следующего вида:  
{  
 "1":"Значение1",  
 "2":"Значение2",  
 "3":"Значение3"  
}  
Если свойство имеет значение Ложь, то массив будет записан как массив JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
