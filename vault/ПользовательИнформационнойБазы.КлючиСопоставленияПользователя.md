ПользовательИнформационнойБазы (InfoBaseUser)

КлючиСопоставленияПользователя (UserMatchingKeys)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие.md).   
Содержит пары "ключ-значение". Ключом является наименование провайдера OpenID Connect-аутентификации или эмитент токена доступа, значение - ключ сопоставления пользователя.  
  
Ключи сопоставления могут использоваться при OpenID Connect аутентификации и аутентификации токенами доступа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
