НастройкиВосстановленияПароля (PasswordRecoverySettings)

СрокДействияКодаПодтверждения (VerificationCodeEffectivePeriod)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет срок действия кода подтверждения восстановления пароля (в секундах ).  
Значение по умолчанию: 600 (10 минут).  
Диапазон значений: от 1 до 4294967295.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
