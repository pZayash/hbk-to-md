РасписаниеЭлементаПланировщика (PlannerItemSchedule)

ДеньВМесяце (DayInMonth)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Номер дня в месяце. Положительные значения отсчитываются относительно начала месяца, отрицательные - относительно конца (1 - первый день месяца, -1 - последний день месяца).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
