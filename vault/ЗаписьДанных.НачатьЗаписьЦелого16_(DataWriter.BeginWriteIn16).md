ЗаписьДанных (DataWriter)

НачатьЗаписьЦелого16 (BeginWriteIn16)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗаписьЦелого16(<ОписаниеОповещения>, <Число>, <ПорядокБайтов>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число, которое будет записано в поток.  
Значение числа должно находиться в диапазоне от 0 до 65535.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа перед записью в поток. Если не установлен, то будет использован порядок байтов, заданный для текущего экземпляра объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Значение по умолчанию: Неопределено.

Описание:

Вызывает начало записи 16-разрядного целого числа в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [ЗаписатьЦелое16Асинх](ЗаписьДанных.ЗаписатьЦелое16Асинх_(DataWriter.WriteInt16Async).md)

Доступен, начиная с версии 8.3.8.
