ФиксированнаяСтруктура (FixedStructure)

Из структуры

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ФиксированнаяСтруктура(<Структура>)

Параметры:

<Структура> (обязательный)

Тип: [Структура](Структура.md).   
Исходная структура. Все ее свойства и значения будут скопированы.

Описание:

Создает неизменяемую копию объекта типа [Структура](Структура.md).

Использование в версии:

Доступен, начиная с версии 8.2.
