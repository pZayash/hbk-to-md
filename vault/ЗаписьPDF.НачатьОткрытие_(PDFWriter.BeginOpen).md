ЗаписьPDF (PDFWriter)

НачатьОткрытие (BeginOpen)

Доступен, начиная с версии 8.3.16.

Вариант синтаксиса: Открыть файл

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для чтения PDF-документа.

Вариант синтаксиса: Открыть поток

Синтаксис:

НачатьОткрытие(<ОписаниеОповещения>, <Поток>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи PDF-документа.

Описание:

Открывает файл PDF-документа для записи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [Открыть](ЗаписьPDF.Открыть_(PDFWriter.Open).md)  
[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [ОткрытьАсинх](ЗаписьPDF.ОткрытьАсинх_(PDFWriter.OpenAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
