ВидПодписейКДиаграмме (ChartLabelType)

СерияТочкаРазмер (SeriesPointSize)

Доступен, начиная с версии 8.3.10.

Описание:

В качестве подписи выводится представление серии плюс представление точки, дополнительно в пузырьковой диаграмме выводится значение размера пузырька.

Использование в версии:

Доступен, начиная с версии 8.3.10.
