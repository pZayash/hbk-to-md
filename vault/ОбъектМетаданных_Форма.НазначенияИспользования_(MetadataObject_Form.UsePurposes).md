ОбъектМетаданных: Форма (MetadataObject: Form)

НазначенияИспользования (UsePurposes)

Доступен, начиная с версии 8.3.2.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Содержит назначение использования формы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.

Использование в версии:

Доступен, начиная с версии 8.3.2.
