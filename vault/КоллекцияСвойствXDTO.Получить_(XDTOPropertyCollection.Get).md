КоллекцияСвойствXDTO (XDTOPropertyCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

Получить(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя свойства.

Описание варианта метода:

Если свойства с указанным именем в коллекции не существует, будет возвращено [Неопределено](lang__def_Undefined.md).

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс свойства в коллекции.

Возвращаемое значение:

Тип: [СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Получает свойство.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
