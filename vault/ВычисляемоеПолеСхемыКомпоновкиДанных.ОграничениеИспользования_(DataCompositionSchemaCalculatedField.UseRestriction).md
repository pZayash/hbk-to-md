ВычисляемоеПолеСхемыКомпоновкиДанных (DataCompositionSchemaCalculatedField)

ОграничениеИспользования (UseRestriction)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОграничениеИспользованияПоляСхемыКомпоновкиДанных](ОграничениеИспользованияПоляСхемыКомпоновкиДанных_(DataCompositionSchemaFieldUseRestriction).md).   
Содержит вид ограничения использования и указывает, каким образом нельзя использовать данное поле. Ограничение использования описывается так же, как и ограничение использования поля набора данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
