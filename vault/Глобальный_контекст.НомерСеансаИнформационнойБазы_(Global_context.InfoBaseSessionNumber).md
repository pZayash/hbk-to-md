Глобальный контекст (Global context)

НомерСеансаИнформационнойБазы (InfoBaseSessionNumber)

Доступен, начиная с версии 8.2.

Синтаксис:

НомерСеансаИнформационнойБазы()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает номер текущего сеанса информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[СеансИнформационнойБазы](СеансИнформационнойБазы_(InfoBaseSession).md), свойство [НомерСеанса](СеансИнформационнойБазы.НомерСеанса_(InfoBaseSession.SessionNumber).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
