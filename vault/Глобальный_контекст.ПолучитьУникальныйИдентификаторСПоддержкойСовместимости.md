Глобальный контекст (Global context)

ПолучитьУникальныйИдентификаторСПоддержкойСовместимости (GetUUIDWithCompatibilitySupport)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПолучитьУникальныйИдентификаторСПоддержкойСовместимости(<Совместимость>, <Версия>, <ПространствоИмен>, <Имя>)

Параметры:

<Совместимость> (обязательный)

Тип: Булево.   
Определяет поведение функции при генерации уникальных идентификаторов версий 3 и 5:

- Истина – идентификатор генерируется по такому же алгоритму, как в версиях до 8.3.24 включительно. Такие идентификаторы могут содержать ошибку (не полностью удовлетворять стандарту RFC4122),
- Ложь – идентификатор генерируется без ошибки.

<Версия> (обязательный)

Тип: [ВерсияУникальногоИдентификатора](ВерсияУникальногоИдентификатора.md).   
Задает версию уникального идентификатора, которую необходимо сгенерировать.

<ПространствоИмен> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Задает пространство имен, по которому генерируется уникальный идентификатор. Используется только для уникальных идентификаторов версий 3 и 5.

<Имя> (необязательный)

Тип: Строка.   
Задает имя, по которому генерируется уникальный идентификатор. Используется только для уникальных идентификаторов версий 3 и 5.

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   

Описание:

Функция генерирует уникальные идентификаторы для версий 3 и 5 в зависимости от значения параметра <Совместимость>.  
Для версий 1 и 4 уникальные идентификаторы генерируются по тому же алгоритму, что и при использовании конструктора объекта [УникальныйИдентификатор](УникальныйИдентификатор.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.

Описание изменено в версии 8.3.27.
