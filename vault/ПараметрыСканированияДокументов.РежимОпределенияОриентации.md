ПараметрыСканированияДокументов (DocumentScanningParameters)

РежимОпределенияОриентации (OrientationDetectionMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОпределенияОриентацииСканированияДокументов](РежимОпределенияОриентацииСканированияДокументов.md).   
Устанавливает режим распознавания естественного расположения документа для чтения пользователем.   
При серийной съемке позволяет применить одинаковую ориентацию ко всем страницам документа.   
Значение по умолчанию: [ПоГоризонтальнымСтрокамТекста](РежимОпределенияОриентацииСканированияДокументов.ПоГоризонтальнымСтрокамТекста.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
