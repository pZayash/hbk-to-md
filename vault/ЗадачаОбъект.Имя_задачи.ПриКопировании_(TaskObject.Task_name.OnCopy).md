# ЗадачаОбъект.<Имя задачи>.ПриКопировании

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [События](ЗадачаОбъект.Имя_задачи__События.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ПриКопировании (OnCopy)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриКопировании(<ОбъектКопирования>)

Параметры:

<ОбъектКопирования>

Тип: [ЗадачаОбъект.\<Имя задачи\>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md).   
Исходный элемент, который является источником копирования.

Описание:

Возникает при создании элемента задачи копированием.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
