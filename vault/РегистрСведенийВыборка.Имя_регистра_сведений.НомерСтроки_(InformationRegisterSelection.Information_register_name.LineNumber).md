# РегистрСведенийВыборка.<Имя регистра сведений>.НомерСтроки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md) › [РегистрСведенийВыборка.<Имя регистра сведений>](РегистрСведенийВыборка.Имя_регистра_сведений_(InformationRegisterSelection.Information_register_name).md) › [Свойства](РегистрСведенийВыборка.Имя_регистра_сведений__Свойства.md)

РегистрСведенийВыборка.<Имя регистра сведений> (InformationRegisterSelection.<Information register name>)

НомерСтроки (LineNumber)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрСведенийВыборка.Имя_регистра_сведений.Регистратор_(InformationRegisterSelection.Information_register_name.Recorder).md). Используется для регистров, управляемых регистратором.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
