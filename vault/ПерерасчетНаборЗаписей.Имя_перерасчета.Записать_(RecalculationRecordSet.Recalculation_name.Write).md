# ПерерасчетНаборЗаписей.<Имя перерасчета>.Записать

ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим замещения существующих записей в соответствии с текущими установками отбора (т.е. по объекту перерасчета). [Истина](lang__def_BooleanTrue.md) - перед записью существующие записи будут удалены. [Ложь](lang__def_BooleanFalse.md) - записи будут дописаны к уже существующим в информационной базе записям.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает набор записей в базу данных. В зависимости от переданного параметра может быть выполнено добавление записей или замещение.   
Если указано замещение, то перед выполнением записи будет выполнено удаление всех существующих записей в соответствии с текущими установками отбора (т.е. по объекту перерасчета).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу набор записей очищается (удаляются записи из набора).

Пример:

|  |
| --- |
| Регистр = РегистрыРасчета.ОсновныеНачисленияСотрудников; Перерасчет = Регистр.Перерасчеты.Перерасчет;  // очищаем набор записей  НаборЗаписей = Перерасчет.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.ОбъектПерерасчета.Значение = СтрокаДок.Документ; НаборЗаписей.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
