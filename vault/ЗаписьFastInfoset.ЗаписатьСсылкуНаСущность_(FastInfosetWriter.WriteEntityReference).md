ЗаписьFastInfoset (FastInfosetWriter)

ЗаписатьСсылкуНаСущность (WriteEntityReference)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьСсылкуНаСущность(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя ссылки на сущность.

Описание:

Осуществляет запись ссылки на сущность.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
