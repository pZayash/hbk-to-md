Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfileAddIn (UnregSecurityProfileAddIn)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfileAddIn(<Кластер>, <ИмяПрофиля>, <AddInName>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должна быть отменена регистрация разрешенного внешнего компонента.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<AddInName> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя разрешенного внешнего компонента, регистрация которого отменяется. Если разрешенного внешнего компонента с этим именем нет, то метод ничего не делает.

Описание:

Отменяет регистрацию разрешенного внешнего компонента.

Доступность:

Интеграция.

Примечание:

Если профиля с именем, указанным в параметре <ИмяПрофиля>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
