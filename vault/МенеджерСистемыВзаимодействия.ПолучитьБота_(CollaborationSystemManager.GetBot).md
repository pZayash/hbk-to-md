МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьБота (GetBot)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьБота(<Пользователь>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Идентификатор пользователя системы взаимодействия.

Возвращаемое значение:

Тип: [БотСистемыВзаимодействия](БотСистемыВзаимодействия_(CollaborationSystemBot).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - если переданный пользователь не является ботом.

Описание:

Возвращает описание бота, назначенного переданному пользователю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
