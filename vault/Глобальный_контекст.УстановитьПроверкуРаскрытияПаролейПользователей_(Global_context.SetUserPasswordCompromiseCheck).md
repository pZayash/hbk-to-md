Глобальный контекст (Global context)

УстановитьПроверкуРаскрытияПаролейПользователей (SetUserPasswordCompromiseCheck)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьПроверкуРаскрытияПаролейПользователей(<ПроверкаРаскрытияПаролейПользователей>)

Параметры:

<ПроверкаРаскрытияПаролейПользователей> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Проверка раскрытия паролей пользователей.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Устанавливает проверку раскрытия паролей пользователей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
