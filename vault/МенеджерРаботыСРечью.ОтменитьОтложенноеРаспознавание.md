МенеджерРаботыСРечью (SpeechProcessingManager)

ОтменитьОтложенноеРаспознавание (CancelDelayedSpeechToText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОтменитьОтложенноеРаспознавание(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [ИдентификаторОтложенногоРаспознаванияРечи](ИдентификаторОтложенногоРаспознаванияРечи.md).   
Ключ задания отложенного распознавания.

Описание:

Отменяет отложенное распознавание речи по ключу.  
При отмене для объекта [РезультатОтложенногоРаспознаванияРечи](РезультатОтложенногоРаспознаванияРечи.md) устанавливаются следующие значения свойств:

- [Завершено](РезультатОтложенногоРаспознаванияРечи.Завершено.md) - Истина,
- [Успешно](РезультатОтложенногоРаспознаванияРечи.Успешно.md) - Ложь,
- [Ошибка](РезультатОтложенногоРаспознаванияРечи.Ошибка.md) - "Распознавание отменено".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
