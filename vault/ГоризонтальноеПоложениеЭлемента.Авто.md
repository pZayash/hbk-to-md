ГоризонтальноеПоложениеЭлемента (ItemHorizontalLocation)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

Горизонтальное выравнивание текста элемента управления будет определяться автоматически, исходя из типа отображаемого значения. Числовые значения будут прижиматься к правой границе, остальные - к левой.

Использование в версии:

Доступен, начиная с версии 8.2.
