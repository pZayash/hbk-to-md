Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка документов. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ДокументСсылка.](ДокументСсылка.Имя_документа.md)<[Имя документа](ДокументСсылка.Имя_документа.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ДокументСписок.](ДокументСписок.Имя_документа.md)<[Имя документа](ДокументСписок.Имя_документа.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
