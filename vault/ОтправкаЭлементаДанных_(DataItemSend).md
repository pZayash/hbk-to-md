Значения

[Авто (Auto)](ОтправкаЭлементаДанных.Авто_(DataItemSend.Auto).md)

Описание:

Содержит варианты значений выходных параметров обработчиков событий [ПриОтправкеДанныхГлавному](ПланОбменаОбъект.Имя_плана_обмена.ПриОтправкеДанныхГлавному_(ExchangePlanObject.Exchange_plan_name.OnSendDataToMaster).md) и [ПриОтправкеДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПриОтправкеДанныхПодчиненному_(ExchangePlanObject.Exchange_plan_name.OnSendDataToSlave).md) объектов [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>. Данное значение определяет действия системы относительно элемента данных при формировании сообщения, содержащего изменения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.
