Описание:

Дополнительные свойства объекта [Форма](Форма_(Form).md), в том случае, если основной реквизит формы имеет тип [РегистрНакопленияСписок.](РегистрНакопленияСписок.Имя_регистра_накопления_(AccumulationRegisterList.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияСписок.Имя_регистра_накопления_(AccumulationRegisterList.Accumulation_register_name).md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Приходные = Документы.ПриходнаяНакладная; Док = Приходные.НайтиПоНомеру("00003",ТекущаяДата()); Остатки = РегистрыНакопления.УчетНоменклатуры; Форма = Остатки.ПолучитьФормуСписка(); Форма.ПараметрОтборПоРегистратору = Док; Форма.Открыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
