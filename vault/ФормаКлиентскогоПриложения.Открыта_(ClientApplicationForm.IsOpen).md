ФормаКлиентскогоПриложения (ClientApplicationForm)

Открыта (IsOpen)

Доступен, начиная с версии 8.2.

Синтаксис:

Открыта()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - форма открыта; [Ложь](lang__def_BooleanFalse.md) - не открыта.

Описание:

Определяет, открыта ли данная форма.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
