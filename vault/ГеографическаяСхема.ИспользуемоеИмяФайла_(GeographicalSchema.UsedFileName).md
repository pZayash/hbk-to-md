ГеографическаяСхема (GeographicalSchema)

ИспользуемоеИмяФайла (UsedFileName)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, которое будет использоваться при сохранении географической схемы пользователем.   
Данное свойство будет заполнятся:  
1. при программной установке значения;  
2. при интерактивном открытии документа из файла;  
3. при вызове метода [Показать](ГеографическаяСхема.Показать_(GeographicalSchema.Show).md), если в нем указано имя файла;  
4. при вызове пользователем команды "Сохранить как…".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
