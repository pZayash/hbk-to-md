# ПланВидовРасчетаОбъект.<Имя плана видов расчета>.ПроверитьЗаполнение

ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ОбработкаПроверкиЗаполнения_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.FillCheckProcessing).md).
