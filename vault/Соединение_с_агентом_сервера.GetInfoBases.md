Соединение с агентом сервера (IServerAgentConnection)

GetInfoBases (GetInfoBases)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBases(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, для которого необходимо получить список информационных баз. Может быть элементом полученного методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md) массива кластеров, зарегистрированных в текущем центральном сервере. Еще кластер может быть создан методом [CreateClusterInfo](Соединение_с_агентом_сервера.CreateClusterInfo.md) с последующим заполнением его свойств.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив кратких описаний информационных баз. Каждое краткое описание информационной базы является объектом с интерфейсом [Описание информационной базы](Описание_информационной_базы.md).

Описание:

Получает массив информационных баз, зарегистрированных в кластере. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера. Про каждую информационную базу выдается только такая информация, для получения которой не требуется аутентификация пользователя информационной базы. Получение всех параметров зарегистрированных информационных баз, а так же создание и удаление информационных баз может быть выполнено только посредством интерфейса [Соединение с рабочим процессом](Соединение_с_рабочим_процессом.md) после аутентификации пользователей.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
