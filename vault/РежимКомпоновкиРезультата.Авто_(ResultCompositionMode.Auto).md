РежимКомпоновкиРезультата (ResultCompositionMode)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

Режим формирования определяется автоматически.  
Для файлового варианта - [Непосредственно](РежимКомпоновкиРезультата.Непосредственно_(ResultCompositionMode.Directly).md).  
Для клиент-серверного - [Фоновый](РежимКомпоновкиРезультата.Фоновый_(ResultCompositionMode.Background).md).

Использование в версии:

Доступен, начиная с версии 8.2.
