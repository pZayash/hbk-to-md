# ОтборОбсужденийСистемыВзаимодействия.ДатаНачала

ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

ДатаНачала (StartDate)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md), [Неопределено](lang__def_Undefined.md).   
Если свойство установлено, возвращаются обсуждения, в которых последнее сообщение имеет дату большую (или меньшую - в зависимости от направления сортировки), чем заданная. Устанавливается в формате UTC.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
