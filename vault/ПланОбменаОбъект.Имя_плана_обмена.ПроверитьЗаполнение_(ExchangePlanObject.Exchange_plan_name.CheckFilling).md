# ПланОбменаОбъект.<Имя плана обмена>.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [Методы](ПланОбменаОбъект.Имя_плана_обмена__Методы.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](ПланОбменаОбъект.Имя_плана_обмена.ОбработкаПроверкиЗаполнения_(ExchangePlanObject.Exchange_plan_name.FillCheckProcessing).md).

Использование в версии:

Доступен, начиная с версии 8.2.
