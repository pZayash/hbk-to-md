РазмерКартинки (PictureSize)

Черепица (Tile)

Доступен, начиная с версии 8.0.

Описание:

Картинка, сохраняя свои оригинальные размеры, будет периодически повторяться, заполняя всю отведенную для нее область.

Использование в версии:

Доступен, начиная с версии 8.0.
