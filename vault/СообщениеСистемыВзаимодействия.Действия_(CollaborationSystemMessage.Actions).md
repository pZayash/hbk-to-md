СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

Действия (Actions)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [ПанельКнопок](СообщениеСистемыВзаимодействия.ПанельКнопок_(CollaborationSystemMessage.ButtonPanel).md)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияДействийСообщенияСистемыВзаимодействия](КоллекцияДействийСообщенияСистемыВзаимодействия_(CollaborationSystemMessageActionCollection).md).   
Содержит список действий, которые будут выведены в виде гиперссылок при отображении сообщения:

- Текст ссылки – значение свойства [Текст](ДействиеСообщенияСистемыВзаимодействия.Текст_(CollaborationSystemMessageAction.Text).md) объекта типа [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md). Если [Текст](ДействиеСообщенияСистемыВзаимодействия.Текст_(CollaborationSystemMessageAction.Text).md) содержит пустую строку, то гиперссылка для этого элемента не отображается.
- В обработчик, подключенный при помощи вызова [ПодключитьОбработчикДействияСообщения](МенеджерСистемыВзаимодействия.ПодключитьОбработчикДействияСообщения_(CollaborationSystemManager.AttachMessageActionHandler).md) передается значение, заданное в свойстве [Значение](ДействиеСообщенияСистемыВзаимодействия.Значение_(CollaborationSystemMessageAction.Value).md) объекта типа [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
