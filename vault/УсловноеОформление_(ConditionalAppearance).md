# УсловноеОформление

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Оформление](Оформление-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы условного оформления.  
Возможно обращение к элементу оформления посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Доступность:
