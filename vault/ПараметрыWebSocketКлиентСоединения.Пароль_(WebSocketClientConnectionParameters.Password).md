ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

Пароль (Password)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Пароль пользователя, который используется для стандартной аутентификации или аутентификации ОС на WebSocket-сервере.  
Значение по умолчанию: Пустая строка.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
