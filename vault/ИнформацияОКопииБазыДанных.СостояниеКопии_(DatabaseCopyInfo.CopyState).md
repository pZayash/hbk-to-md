ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

СостояниеКопии (CopyState)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [СостояниеКопииБазыДанных](СостояниеКопииБазыДанных_(DatabaseCopyState).md).   
Описывает текущее состояние копии базы данных для использования в отборе копий при выполнении запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
