ИнтернетПочтовыеВложения (InternetMailAttachments)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<ИндексЭлементаКоллекции>)

Параметры:

<ИндексЭлементаКоллекции> (обязательный)

Тип: Число; элемент коллекции.   
Индекс элемента коллекции или элемент коллекции.

Описание:

Удаляет почтовое вложение из коллекции.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
