ЗаписьДанных (DataWriter)

ЗаписатьСимволы (WriteChars)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьСимволы(<Строка>, <КодировкаТекста>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, символы которой будут записаны в поток.

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Определяет кодировку текста для записи строки. Если не установлена, используется кодировка, заданная для данного объекта [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.
