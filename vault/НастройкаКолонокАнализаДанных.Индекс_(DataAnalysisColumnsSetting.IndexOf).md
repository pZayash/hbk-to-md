# НастройкаКолонокАнализаДанных.Индекс

НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияКолонкамиАнализаДанных](ЭлементУправленияКолонкамиАнализаДанных_(DataAnalysisColumnsControl).md).   
Искомый элемент управления колонками анализа данных.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
