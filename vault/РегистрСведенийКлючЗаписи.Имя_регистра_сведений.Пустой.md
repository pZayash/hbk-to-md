РегистрСведенийКлючЗаписи.<Имя регистра сведений> (InformationRegisterRecordKey.<Information register name>)

Доступен, начиная с версии 8.2.

Синтаксис:

Пустой()

Возвращаемое значение:

Тип: Булево.   

Описание:

Возвращает Истина, если все поля ключа имеют значения по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
