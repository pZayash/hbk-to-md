ЭлементыФормы (FormItems)

<Имя элемента управления> (<Control name>)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Набор свойств содержит элементы формы клиентского приложения, подчиненные данному. Доступ к элементу формы осуществляется по имени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
