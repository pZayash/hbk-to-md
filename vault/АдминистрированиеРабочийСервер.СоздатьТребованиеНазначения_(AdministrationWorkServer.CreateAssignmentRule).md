АдминистрированиеРабочийСервер (AdministrationWorkServer)

СоздатьТребованиеНазначения (CreateAssignmentRule)

Доступен, начиная с версии 8.3.14.

Синтаксис:

СоздатьТребованиеНазначения()

Возвращаемое значение:

Тип: [АдминистрированиеТребованиеНазначения](АдминистрированиеТребованиеНазначения_(AdministrationAssignmentRule).md).   

Описание:

Создает новое требование назначения для его последующего добавления в список требований. После создания необходимо установить значения его свойств.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
