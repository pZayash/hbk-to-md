ПланВидовРасчетаСсылка.<Имя плана видов расчета> (ChartOfCalculationTypesRef.<Chart of calculation types name>)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ПланВидовРасчетаСсылка.<Имя плана видов расчета>(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор ссылки.   
Если идентификатор не указан, создается пустая ссылка.

Описание:

Создает новый объект на основе переданных параметров. Такая форма записи будет вызывать ошибку при проверке модулей в конфигураторе, так как при проверке модулей в конфигураторе не определены типы прикладных объектов.  
Также может быть создан с использованием функциональной формы оператора Новый.   
Например, ВидРасчета = Новый(Тип("ПланВидовРасчетаСсылка.ОсновныеНачисленияОрг"));

Использование в версии:

Доступен, начиная с версии 8.2.
