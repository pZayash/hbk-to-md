ПорядокСерийВЛегендеДиаграммы (ChartSeriesOrderInLegend)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Порядок серий в легенде рассчитывается в зависимости от типа диаграммы:

- обратный порядок используется для типов:
  - [ГистограммаГоризонтальная](ТипДиаграммы.ГистограммаГоризонтальная.md),
  - [ГистограммаГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаГоризонтальнаяОбъемная.md),
  - [ГрафикСНакоплением](ТипДиаграммы.ГрафикСНакоплением.md),
  - [ГрафикСОбластямиИНакоплением](ТипДиаграммы.ГрафикСОбластямиИНакоплением.md),
  - [ГрафикСОбластямиНормированный](ТипДиаграммы.ГрафикСОбластямиНормированный.md),
  - [ГистограммаСНакоплением](ТипДиаграммы.ГистограммаСНакоплением.md),
  - [ГистограммаСНакоплениемОбъемная](ТипДиаграммы.ГистограммаСНакоплениемОбъемная.md),
  - [ГистограммаНормированная](ТипДиаграммы.ГистограммаНормированная.md),
  - [ГистограммаНормированнаяОбъемная](ТипДиаграммы.ГистограммаНормированнаяОбъемная.md).
- для остальных типов используется прямой порядок.

Использование в версии:

Доступен, начиная с версии 8.3.10.
