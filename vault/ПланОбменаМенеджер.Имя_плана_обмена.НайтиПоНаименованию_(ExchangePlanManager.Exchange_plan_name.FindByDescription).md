# ПланОбменаМенеджер.<Имя плана обмена>.НайтиПоНаименованию

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаМенеджер.<Имя плана обмена>](ПланОбменаМенеджер.Имя_плана_обмена_(ExchangePlanManager.Exchange_plan_name).md) › [Методы](ПланОбменаМенеджер.Имя_плана_обмена__Методы.md)

ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

НайтиПоНаименованию (FindByDescription)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоНаименованию(<Наименование>, <ТочноеСоответствие>)

Параметры:

<Наименование> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, содержащая искомое наименование.

<ТочноеСоответствие> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим поиска по полному соответствию.  
Поиск будет успешным, если строка поиска: в случае значения параметра [Ложь](lang__def_BooleanFalse.md) - будет соответствовать левой части наименования; в случае значения параметра [Истина](lang__def_BooleanTrue.md) - будет полностью совпадать с наименованием (за исключением "хвостовых" пробелов в наименовании).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   
Ссылка на найденный узел. Если не существует ни одного узла с требуемым наименованием, то будет возвращена пустая ссылка.

Описание:

Осуществляет поиск узла плана обмена по его наименованию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если существует несколько узлов с указанным наименованием, то будет найден только один из них.

Использование в версии:

Доступен, начиная с версии 8.0.
