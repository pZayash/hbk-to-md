РежимОткрытияФайла (FileOpenMode)

Обрезать (Truncate)

Доступен, начиная с версии 8.3.9.

Описание:

Существующий файл будет перезаписан. При открытии длина файла будет равна 0.  
Если файл не существует, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
