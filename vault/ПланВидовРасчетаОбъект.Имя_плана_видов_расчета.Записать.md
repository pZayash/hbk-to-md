ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает объект вид расчета. При записи выполняется проверка правильности таблицы вытесняющих видов расчета и ведущих видов расчета. При заполнении предопределенных табличных частей [ВедущиеВидыРасчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета.ВедущиеВидыРасчета.md) и [ВытесняющиеВидыРасчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ВытесняющиеВидыРасчета.md) проверяется отсутствие циклических ссылок.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
