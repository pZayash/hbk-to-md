ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаНавигационнойСсылки (URLProcessing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОбработкаНавигационнойСсылки(<НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: [Строка](lang__def_String.md).   
Значение гиперссылки форматированной строки.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при нажатии на гиперссылку форматированной строки, расположенной в элементе формы типа [Расширение поля формы для поля надписи](Расширение_поля_формы_для_поля_надписи_(Form_field_extension_for_a_label_field).md) или в элементе формы [Расширение декорации формы для надписи](Расширение_декорации_формы_для_надписи_(Form_decoration_extension_for_a_label).md), у которого свойство Гиперссылка имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.1.
