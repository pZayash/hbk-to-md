Расширение формы клиентского приложения для объекта таблицы внешнего источника данных

Ключ (Key)

Доступен, начиная с версии 8.3.5.

Использование:

Ключевой.

Описание:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   
Ключ объекта.   
В случае отсутствия или неверного значения создается новый объект с использованием остальных параметров.

Использование в версии:

Доступен, начиная с версии 8.3.5.
