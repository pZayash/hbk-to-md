ПостроительОтчета (ReportBuilder)

ПолучитьЗапрос (GetQuery)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗапрос()

Возвращаемое значение:

Тип: [Запрос](Запрос_(Query).md).   

Описание:

Получить запрос для выполнения отчета. Данный метод недоступен в случае, если установлено свойство [ИсточникДанных](ПостроительОтчета.ИсточникДанных_(ReportBuilder.DataSource).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| РезультатЗапроса = ПостроительОтчета.ПолучитьЗапрос().Выполнить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
