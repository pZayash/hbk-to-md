Глобальный контекст (Global context)

РазблокироватьДанныеДляРедактирования (UnlockDataForEdit)

Доступен, начиная с версии 8.2.

Синтаксис:

РазблокироватьДанныеДляРедактирования(<Ключ>, <ИдентификаторФормы>)

Параметры:

<Ключ> (необязательный)

Тип: Любая ссылка на объект информационной базы, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений.md)>, [ВнешнийИсточникДанныхТаблицаСсылка.](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md)<[Имя внешнего источника](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md)>, [ВнешнийИсточникДанныхТаблицаКлючЗаписи.](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md)<[Имя внешнего источника](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md)>.   
Ссылка на объект, который должен быть разблокирован, или ключ записи независимого регистра сведений, которая должна быть разблокирована.   
Если параметр не указан, то разблокируются все объекты, заблокированные с данным идентификатором формы или без указания идентификатора формы.

<ИдентификаторФормы> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор формы.   
Если параметр не указан, то разблокируются только те объекты, которые были заблокированы без указания идентификатора формы.

Описание:

Снимает блокировку объекта.   
Если блокировка не была установлена от имени этой формы или без указания формы, никаких действий не выполняется.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ЗаблокироватьДанныеДляРедактирования](Глобальный_контекст.ЗаблокироватьДанныеДляРедактирования.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
