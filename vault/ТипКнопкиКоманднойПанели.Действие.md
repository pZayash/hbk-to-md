ТипКнопкиКоманднойПанели (CommandBarButtonType)

Действие (Action)

Доступен, начиная с версии 8.0.

Описание:

Кнопка данного типа представляет собой обычную кнопку панели. Кнопке может быть установлено действие, которое будет выполнено при нажатии на кнопку.

Использование в версии:

Доступен, начиная с версии 8.0.
