Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

ОбработкаАктивации (ActivationProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаАктивации()

Описание:

Возникает на клиенте перед выполнением команды "Активировать" управляемой формы Задачи.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
