Глобальный контекст (Global context)

XMLЗначение (XMLValue)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLЗначение(<Тип>, <СтрокаXML>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип, значение которого надо получить при преобразовании из строкового представления XML.

<СтрокаXML> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, содержащая строковое представление значения, полученное из XML.

Возвращаемое значение:

Тип: [Null](lang__def_Null.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md); значения перечислений, значения системных перечислений ([ВидСчета](ВидСчета_(AccountType).md), [ВидДвиженияБухгалтерии](ВидДвиженияБухгалтерии_(AccountingRecordType).md), [ВидДвиженияНакопления](ВидДвиженияНакопления_(AccumulationRecordType).md), [ДопустимыйЗнак](ДопустимыйЗнак_(AllowedSign).md), [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md), [ЧастиДаты](ЧастиДаты_(DateFractions).md), [Неопределено](lang__def_Undefined.md) или все ссылки на объекты базы данных.   

Описание:

Выполняет преобразование из строки, полученной из текста элемента или значения атрибута XML, в значение в соответствии с указанным типом. Выполняет действие обратное действию метода [XMLСтрока](Глобальный_контекст.XMLСтрока_(Global_context.XMLString).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЧитаемДату = Ложь; Пока ЧтениеXML.Прочитать() Цикл     Если ЧтениеXML.ТипУзла = ТипУзлаXML.НачалоЭлемента Тогда         Если ЧтениеXML.ЛокальноеИмя = "Дата" Тогда             ЧитаемДату = Истина;         КонецЕсли;     ИначеЕсли ЧтениеXML.ТипУзла = ТипУзлаXML.Текст Тогда         Если ЧитаемДату Тогда             Дата = XMLЗначение(Тип("Дата"), ЧтениеXML.Значение);         КонецЕсли     КонецЕсли КонецЕсли |
