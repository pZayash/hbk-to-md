СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

ВыбратьНастройкиПоУмолчанию (SelectDefaultSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьНастройкиПоУмолчанию(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Задает поле и значение отбора открываемой выборки. Ключ структуры описывает имя поля, а значение структуры – значение отбора по этому полю. В качестве поля для отбора может задаваться поле "КлючОбъекта" ("ObjectKey") – ключ объекта настройки. Если параметр не указан, то отбор не используется.

Возвращаемое значение:

Тип: [СтандартноеХранилищеНастроекВыборкаНастроекПоУмолчанию](СтандартноеХранилищеНастроекВыборкаНастроекПоУмолчанию.md).   

Описание:

Формирует выборку настроек по умолчанию из стандартного хранилища настроек по заданным условиям.  
Вызов метода допускается только для хранилища системных настроек (ХранилищеСистемныхНастроек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
