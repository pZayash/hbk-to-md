ОбъектМетаданных: Документ (MetadataObject: Document)

Имя (Name)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имя должно состоять из одного слова, начинаться с буквы и не содержать специальных символов, кроме «\_».

Использование в версии:

Доступен, начиная с версии 8.1.
