МенеджерСистемыАналитики (AnalyticsSystemManager)

Открыть (Open)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Открыть(<НавигационнаяСсылкаСистемыАналитики>)

Параметры:

<НавигационнаяСсылкаСистемыАналитики> (необязательный)

Тип: Строка.   
Внутренняя ссылка на объект системы аналитики. Если не задана, то открывается рабочий стол системы аналитики.

Описание:

Открывает систему аналитики в новом окне веб-браузера.   
В тонком клиенте в режиме прямого соединения не работает.

Доступность:

Тонкий клиент, веб-клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
