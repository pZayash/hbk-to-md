СредстваБуфераОбмена (ClipboardTools)

ПоддерживаетсяФорматДанных (DataFormatSupported)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПоддерживаетсяФорматДанных(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена_(ClipboardDataStandardFormat).md), [Строка](lang__def_String.md).   
Значение параметра типа [Строка](lang__def_String.md) должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В тонком и толстом клиентах всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Проверяет, поддерживается ли переданный формат данных буфером обмена.  
Данный метод рекомендуется использовать перед вызовом метода [ПоместитьДанныеАсинх](СредстваБуфераОбмена.ПоместитьДанныеАсинх_(ClipboardTools.PutDataAsync).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте имеются следующие особенности:

- в браузере Chrome (а также всех браузерах на базе движка Chromium) поддерживаются все форматы данных буфера обмена, начиная с версии 104 Chromium. До версии 104 Chromium не поддерживаются форматы данных, заданные строкой,
- в браузерах Firefox, Safari, Internet Explorer поддерживается только формат данных буфера обмена [Текст](СтандартныйФорматДанныхБуфераОбмена.Текст_(ClipboardDataStandardFormat.Text).md).

Использование в версии:

Доступен, начиная с версии 8.3.24.
