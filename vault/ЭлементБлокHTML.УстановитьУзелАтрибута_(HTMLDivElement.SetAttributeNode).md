ЭлементБлокHTML (HTMLDivElement)

УстановитьУзелАтрибута (SetAttributeNode)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьУзелАтрибута(<НовыйАтрибут>)

Параметры:

<НовыйАтрибут> (обязательный)

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md).   
Устанавливаемый атрибут HTMLDOM.

Возвращаемое значение:

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md), [Неопределено](lang__def_Undefined.md).   
Возвращает атрибут, который был замещен новым атрибутом или [Неопределено](lang__def_Undefined.md), если такого нет.

Описание:

Устанавливает атрибут для элемента.  
Если значение локального имени совпадает с одним из существующих атрибутов, то данный атрибут заменяется на новый.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- узел-атрибут был создан другим документом;
- данный узел в режиме Только для чтения;
- устанавливаемый атрибут является уже узлом-атрибутом другого узла [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), необходимо явное клонирование атрибута.

Использование в версии:

Доступен, начиная с версии 8.2.
