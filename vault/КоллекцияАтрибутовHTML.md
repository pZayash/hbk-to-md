Доступен, начиная с версии 8.2.

Элементы коллекции:

[АтрибутHTML](АтрибутHTML.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются атрибуты HTML.  
Возможно обращение к атрибуту HTML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция узлов атрибутов узла HTML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
