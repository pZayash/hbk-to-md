# Расширение формы элемента вида характеристик.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [Расширение формы элемента вида характеристик](Расширение_формы_элемента_вида_характеристик_(Characteristic_type_item_form_extension).md) › [События](Расширение_формы_элемента_вида_характеристик__События.md)

Расширение формы элемента вида характеристик (Characteristic type item form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.ПриЗаписи_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.OnWrite).md) объекта [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
