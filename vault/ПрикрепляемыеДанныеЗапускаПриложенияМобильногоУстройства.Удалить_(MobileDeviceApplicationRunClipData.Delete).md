ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipData)

Удалить (Delete)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента прикрепляемых данных в контейнере. Значение должно быть целым числом в диапазоне от 0 до [Количество](ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства.Количество_(MobileDeviceApplicationRunClipData.Count).md) – 1.

Описание:

Удаляет элемент прикрепляемых данных из контейнера. Все элементы с номером больше переданного сдвигаются.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
