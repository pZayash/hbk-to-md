ДействиеПриНедоступностиНастроекКомпоновкиДанных (OnUnavailabilityDataCompositionSettingsAction)

ИзменятьДоступностьПоля (DisableControl)

Доступен, начиная с версии 8.1.

Описание:

Редакторы, которые не используются при редактировании текущих данных, становятся недоступными.

Использование в версии:

Доступен, начиная с версии 8.1.
