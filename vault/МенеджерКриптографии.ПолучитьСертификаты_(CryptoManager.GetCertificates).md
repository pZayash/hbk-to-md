МенеджерКриптографии (CryptoManager)

ПолучитьСертификаты (GetCertificates)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьСертификаты(<ИсходныеДанные>, <Хранилище>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Исходные данные электронной подписи или зашифрованных данных в формате PKCS#7, из которых будут извлечены сертификаты.  
Исходные данные могут размещаться в файле (в этом случае указываются именем исходного файла) или представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

<Хранилище> (необязательный)

Тип: [ХранилищеСертификатовКриптографии](ХранилищеСертификатовКриптографии_(CryptoCertificateStore).md).   
Хранилище сертификатов. Используется для поиска сертификатов в случае, если какой-либо сертификат автора подписи или получателя зашифрованных данных отсутствует.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Описание:

Извлекает массив сертификатов из указанных данных электронной подписи или зашифрованных данных в формате PKCS#7.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
