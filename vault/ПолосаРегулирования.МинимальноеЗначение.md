ПолосаРегулирования (TrackBar)

МинимальноеЗначение (MinValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит минимальное значение диапазона изменения. Это значение устанавливается при перемещении "движка" в крайнее левое (верхнее) положение.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
