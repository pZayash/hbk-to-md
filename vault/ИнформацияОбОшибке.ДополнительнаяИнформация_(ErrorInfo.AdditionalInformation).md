ИнформацияОбОшибке (ErrorInfo)

ДополнительнаяИнформация (AdditionalInformation)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Предназначено для хранения дополнительной информации, передаваемой в исключение для последующего анализа, например, для расследования ошибок.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
