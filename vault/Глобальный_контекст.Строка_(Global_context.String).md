Глобальный контекст (Global context)

Строка (String)

Доступен, начиная с версии 8.0.

Синтаксис:

Строка(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Исходное значение.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Полученное значение.

Описание:

Преобразует полученный параметр в значение типа [Строка](lang__def_String.md).  
При преобразовании к строке числа формируется его полное представление в стандартном виде, соответствующем национальным установкам.  
Преобразование значений типа [Булево](lang__Булево_(Boolean).md) зависит от национальных установок и содержит строковое представление этих значений.  
При преобразовании к строке даты формируется ее полное представление в стандартном удобочитаемом виде, соответствующем национальным установкам.  
Преобразование к строке значений остальных типов производится по мере возможности и обычно содержит удобочитаемое представление значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Сообщить(Строка(Количество) + " единиц товара на  складе не обнаружено"); |

Использование в версии:

Доступен, начиная с версии 8.0.
