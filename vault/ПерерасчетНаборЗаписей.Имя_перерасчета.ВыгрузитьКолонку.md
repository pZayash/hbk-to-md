ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: Строка.   
Имя выгружаемой колонки.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Создает массив и копирует в него значения заданного поля записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения записей выгружаются в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.0.
