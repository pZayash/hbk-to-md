# ОбъектМетаданных: Ресурс.ПризнакУчета

ОбъектМетаданных: Ресурс (MetadataObject: Resource)

ПризнакУчета (AccountingFlag)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: ПризнакУчетаПланаСчетов](ОбъектМетаданных_ПризнакУчетаПланаСчетов_(MetadataObject_AccountingFlag).md).   
Объект метаданных, описывающий признак учета соответствующего плана счетов. Заполнение данного ресурса возможно только в том случае, если в проводке участвует счет, использующий данный признак учета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
