ГлубинаЦвета (ColorDepth)

БитНаПиксел4 (BitPerPixel4)

Доступен, начиная с версии 8.3.14.

Описание:

Для хранения одного пикселя используется 4 бита (картинка использует 16 цветов).

Использование в версии:

Доступен, начиная с версии 8.3.14.
