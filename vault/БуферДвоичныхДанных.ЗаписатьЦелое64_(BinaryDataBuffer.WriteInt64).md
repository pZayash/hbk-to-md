БуферДвоичныхДанных (BinaryDataBuffer)

ЗаписатьЦелое64 (WriteInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗаписатьЦелое64(<Позиция>, <Значение>, <ПорядокБайтов>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, на которой требуется записать число.

<Значение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число, которое требуется записать.  
Если значение не помещается в 64 бита, будет вызвано исключение.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, который будет использован для кодировки числа при записи в буфер. Если не установлен, то используется порядок байтов, заданный для текущего экземпляра [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).  
Значение по умолчанию: Неопределено.

Описание:

Записать целое 64-битное положительное число в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
