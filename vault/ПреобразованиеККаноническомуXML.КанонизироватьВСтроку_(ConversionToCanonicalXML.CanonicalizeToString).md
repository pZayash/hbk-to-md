ПреобразованиеККаноническомуXML (ConversionToCanonicalXML)

КанонизироватьВСтроку (CanonicalizeToString)

Доступен, начиная с версии 8.3.9.

Синтаксис:

КанонизироватьВСтроку(<ДокументDOM>, <ТипКаноническогоXML>, <ФильтрXPath>, <ПрефиксыДляВключения>)

Параметры:

<ДокументDOM> (обязательный)

Тип: [ДокументDOM](ДокументDOM_(DOMDocument).md).   
Документ DOM, который нужно записать в каноническом виде.

<ТипКаноническогоXML> (необязательный)

Тип: [ТипКаноническогоXML](ТипКаноническогоXML_(XMLCanonicalizationType).md).   
Тип алгоритма приведения к каноническому виду.  
Значение по умолчанию: [КаноническийXML1\_1](ТипКаноническогоXML.КаноническийXML1_1_(XMLCanonicalizationType.XMLCanonicalization1_1).md).

<ФильтрXPath> (необязательный)

Тип: [РезультатXPath](РезультатXPath_(XPathResult).md), [Строка](lang__def_String.md).   
Если передана [Строка](lang__def_String.md), то перед преобразованием к каноническому виду будет выполнен запрос XPath из этой строки, и каноническая форма XML-документа будет содержать только результирующее подмножество документа.  
Если передан объект типа [РезультатXPath](РезультатXPath_(XPathResult).md), то к каноническому виду будет преобразовано подмножество узлов из переданного параметра.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ПрефиксыДляВключения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит список префиксов пространств имен для включения в формате: "Префикс1 Префикс2", и так далее.  
Префиксы для включения используются в алгоритме исключающего канонического XML (см. стандарт W3C Exclusive XML Canonicalization - https://www.w3.org/TR/xml-exc-c14n/).   
В остальных случаях параметр игнорируется.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Результат возвращается в виде строки.

Описание:

Преобразует документ DOM к каноническому виду.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
