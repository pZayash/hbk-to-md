Доступен, начиная с версии 8.0.

Свойства:

[ПараметрВыборГруппИЭлементов (ChoiceFoldersAndItemsParameter)](Расширение_формы_списка_видов_характеристик.ПараметрВыборГруппИЭлементов_(Characteristic_type_list_form_extension.ChoiceFoldersAndItemsParameter).md)

Описание:

Дополнительные свойства объекта [Форма](Форма_(Form).md) списка видов характеристик, в том случае, если основной реквизит формы имеет тип [ПланВидовХарактеристикСписок.](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesList.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesList.Chart_of_characteristic_types_name).md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |

Использование в версии:

Доступен, начиная с версии 8.0.
