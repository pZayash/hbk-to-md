Расширение поля формы для поля картинки (Form field extension for a picture field)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что поле картинки является гиперссылкой. Если значение свойства Истина, то при щелчке мышью в поле картинки возникает событие [Нажатие](ПолеКартинки.Нажатие.md).  
В колонке таблицы значение данного свойства игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
