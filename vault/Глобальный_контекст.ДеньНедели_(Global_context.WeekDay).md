# Глобальный контекст.ДеньНедели

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Функции работы со значениями типа Дата](Функции_работы_со_значениями_типа_Дата.md)

Глобальный контекст (Global context)

ДеньНедели (WeekDay)

Доступен, начиная с версии 8.0.

Синтаксис:

ДеньНедели(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Исходная дата.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Номер дня недели.

Описание:

Определяет номер дня недели для указанной даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Сообщить(ДеньНедели(ТекущаяДата())); |

Использование в версии:

Доступен, начиная с версии 8.0.
