Панель (Panel)

РаспределятьПоСтраницам (SeparateByPage)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет, изменяется ли состав отображаемых на панели элементов управления при переключении между страницами. Если Ложь, то страница будет только одна, а закладок много.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
