# ЗадачаОбъект.<Имя задачи>.Разблокировать

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Разблокировать (Unlock)

Доступен, начиная с версии 8.0.

Синтаксис:

Разблокировать()

Описание:

Выполняет разблокировку задачи, если она заблокирована данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Объект.Разблокировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
