РегистрБухгалтерииВыборка.<Имя регистра бухгалтерии> (AccountingRegisterSelection.<Accounting register name>)

Активность (Active)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, попадает ли в учет данная запись.  
Используется различными методами (являются некоторыми функциями от регистра: вычисление остатка, оборотов и т.д.), которые не должны учитывать неактивные записи. С другой стороны, методы перебора записей должны учитывать неактивные записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
