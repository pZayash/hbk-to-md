Соединение с рабочим процессом (IWorkingProcessConnection)

UpdateInfoBase (UpdateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateInfoBase(<ИнформационнаяБаза>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база.md).   
Информационная база с новыми значениями параметров. Новые значения могут иметь только следующие свойства объекта:

- [Descr](Информационная_база.Descr.md),
- [SessionsDenied](Информационная_база.SessionsDenied.md),
- [DBMS](Информационная_база.DBMS.md),
- [dbServerName](Информационная_база.dbServerName.md),
- [dbName](Информационная_база.dbName.md),
- [dbUser](Информационная_база.dbUser.md),
- [dbPassword](Информационная_база.dbPassword.md),
- [DeniedFrom](Информационная_база.DeniedFrom.md),
- [DeniedTo](Информационная_база.DeniedTo.md),
- [DeniedMessage](Информационная_база.DeniedMessage.md),
- [DeniedParameter](Информационная_база.DeniedParameter.md),
- [ExternalSessionManagerConnectionString](Информационная_база.ExternalSessionManagerConnectionString.md),
- [ExternalSessionManagerRequired](Информационная_база.ExternalSessionManagerRequired.md),
- [LicenseDistributionAllowed](Информационная_база.LicenseDistributionAllowed.md),
- [PermissionCode](Информационная_база.PermissionCode.md),
- [SafeModeSecurityProfileName](Информационная_база.SafeModeSecurityProfileName.md),
- [ScheduledJobsDenied](Информационная_база.ScheduledJobsDenied.md),
- [SecurityProfileName](Информационная_база.SecurityProfileName.md)

.

Описание:

Устанавливает новые параметры существующей информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
