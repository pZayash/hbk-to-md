Глобальный контекст (Global context)

ПослеОбменаДаннымиСОсновнымСервером (AfterExchangeDataWithMainServer)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПослеОбменаДаннымиСОсновнымСервером(<ВключеноОповещениеОСинхронизации>, <СтандартнаяОбработка>, <ДатаОбмена>)

Параметры:

<ВключеноОповещениеОСинхронизации>

Тип: Булево.   
Пользователь включил показ оповещений о синхронизации данных.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

<ДатаОбмена>

Тип: Дата.   
Дата последнего обмена с основным сервером.

Описание:

Вызывается после успешного (не пустого) обмена с основным сервером.

Доступность:

Мобильный клиент.

Использование в версии:

Доступен, начиная с версии 8.3.22.

Описание изменено в версии 8.3.23.
