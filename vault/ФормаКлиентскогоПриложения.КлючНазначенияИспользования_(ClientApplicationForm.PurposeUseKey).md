ФормаКлиентскогоПриложения (ClientApplicationForm)

КлючНазначенияИспользования (PurposeUseKey)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет назначение использования формы.  
См. также [КлючНазначенияИспользования](ФормаКлиентскогоПриложения.КлючНазначенияИспользования_(ClientApplicationForm.PurposeUseKey).md).  
Если режим совместимости [Версия8\_3\_4](РежимСовместимости.Версия8_3_4_(CompatibilityMode.Version8_3_4).md) или ниже не установлен, то при попытке открытия формы клиентского приложения с параметром [КлючНазначенияИспользования](ФормаКлиентскогоПриложения.КлючНазначенияИспользования_(ClientApplicationForm.PurposeUseKey).md), длина значения которого превышает 128 символов, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.2.
