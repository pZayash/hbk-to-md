# ПланОбменаОбъект.<Имя плана обмена>.ПриОтправкеДанныхУзлаПодчиненному

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПриОтправкеДанныхУзлаПодчиненному (OnSendNodeDataToSlave)

Доступен, начиная с версии 8.1.

Синтаксис:

ПриОтправкеДанныхУзлаПодчиненному(<ЭлементДанных>, <Игнорировать>)

Параметры:

<ЭлементДанных>

Тип: [ПланОбменаОбъект.\<Имя плана обмена\>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md).   
Узел плана обмена, записываемый в сообщение обмена.

<Игнорировать>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак, управляющий записью узла плана обмена в сообщение обмена.  
Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), узел плана обмена в сообщение обмена записан не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выгрузкой узлов плана обмена (источника и приемника) в рамках распределенной информационной базой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.1.
