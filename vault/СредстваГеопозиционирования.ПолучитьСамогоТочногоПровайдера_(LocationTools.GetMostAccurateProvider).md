СредстваГеопозиционирования (LocationTools)

ПолучитьСамогоТочногоПровайдера (GetMostAccurateProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьСамогоТочногоПровайдера(<ТолькоДоступного>)

Параметры:

<ТолькоДоступного> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр содержит [Истина](lang__def_BooleanTrue.md), то информация о самом точном провайдере будет возращена только если провайдер является доступным. Если самый точный провайдер недоступен, будет возвращено [Неопределено](lang__def_Undefined.md).  
Если параметр содержит [Ложь](lang__def_BooleanFalse.md), то будет возвращена информация о самом точном провайдере на устройстве.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования_(LocationProviderInformation).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - ни один провайдер не найден, или если параметр <ТолькоДоступно> содержал значение [Истина](lang__def_BooleanTrue.md), а самый точный провайдер не доступен.

Описание:

Получает информацию о самом точном провайдере геопозиционирования.   
Если параметр <ТолькоДоступно> содержит значение [Истина](lang__def_BooleanTrue.md), информация о самом точном провайдере будет возвращена только в случае, если провайдер доступен.  
Если параметр <ТолькоДоступно> содержит значение [Ложь](lang__def_BooleanFalse.md), информация о самом точном провайдере будет возвращена независимо от доступности провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
