# СоставКопииБазыДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Копии базы данных](Копии_базы_данных-2.md)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Элементы коллекции:

[ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных_(DatabaseCopyContentItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются [ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных_(DatabaseCopyContentItem).md).  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента.

Описание:

Представляет собой коллекцию объектов копии базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md), свойство [Состав](МенеджерКопииБазыДанных.Состав_(DatabaseCopyManager.Content).md)  

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.27.
