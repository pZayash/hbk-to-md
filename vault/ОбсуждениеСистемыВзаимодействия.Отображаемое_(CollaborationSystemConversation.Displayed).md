ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

Отображаемое (Displayed)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения обсуждения в пользовательском интерфейсе:  
[Истина](lang__def_BooleanTrue.md) - обсуждение отображается.  
Метод [ПолучитьОповещенияТекущегоПользователя](МенеджерСистемыВзаимодействия.ПолучитьОповещенияТекущегоПользователя_(CollaborationSystemManager.GetCurrentUserNotifications).md) возвращает оповещения для всех обсуждений, независимо от значения данного свойства.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).  
Свойство доступно для записи только на сервере.

Использование в версии:

Доступен, начиная с версии 8.3.11.
