СерииДиаграммы (ChartSeriesCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Текст>)

Параметры:

<Текст> (необязательный)

Тип: Строка.   
Название серии.

Возвращаемое значение:

Тип: [СерияДиаграммы](СерияДиаграммы.md).   

Описание:

Добавляет новую серию в конец коллекции серий диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Диаграмма.Очистить(); Для Серия = 1 По 4 Цикл     НоваяСерия = Диаграмма.Серии.Добавить("Серия №" + Серия); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
