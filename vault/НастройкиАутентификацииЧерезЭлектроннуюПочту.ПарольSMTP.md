НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

ПарольSMTP (SMTPPassword)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит пароль пользователя для аутентификации на SMTP-сервере.   
Поддерживаются только символы, входящие в кодовую таблицу US-ASCII.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
