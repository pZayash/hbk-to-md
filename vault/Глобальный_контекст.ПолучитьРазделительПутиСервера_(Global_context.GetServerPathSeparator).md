Глобальный контекст (Global context)

ПолучитьРазделительПутиСервера (GetServerPathSeparator)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьРазделительПутиСервера()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Разделитель пути возвращается в виде строки. Например: "\".

Описание:

Получает значение разделителя пути, используемого в операционной системе, в которой работает рабочий процесс сервера.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вызывается исключение, если код выполняется на клиенте в клиент-серверном варианте.

Использование в версии:

Доступен, начиная с версии 8.3.3.
