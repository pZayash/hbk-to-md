Описание:

Содержит данные для уточнения фильтров, которые будут использованы для поиска приложения мобильного устройства при запуске с помощью методов [ЗапуститьБезОжидания](ЗапускПриложенияМобильногоУстройства.ЗапуститьБезОжидания_(MobileDeviceApplicationRun.RunWithoutWaiting).md) и [ЗапуститьАсинх](ЗапускПриложенияМобильногоУстройства.ЗапуститьАсинх_(MobileDeviceApplicationRun.RunAsync).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
