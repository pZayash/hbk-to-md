Свойства:

[НастройкаНастройкиОформления (AppearanceSettingSettings)](Расширение_табличного_поля_настроек_оформления.НастройкаНастройкиОформления.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для поля настроек оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ЭлементНастройкиОформления](ЭлементНастройкиОформления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
