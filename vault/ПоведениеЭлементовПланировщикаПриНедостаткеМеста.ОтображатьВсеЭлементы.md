ПоведениеЭлементовПланировщикаПриНедостаткеМеста (PlannerItemsBehaviorOnLackOfSpace)

ОтображатьВсеЭлементы (ShowAllItems)

Доступен, начиная с версии 8.3.15.

Описание:

При нехватке места в поле планировщика отображаются все элементы. Высота каждой строки планировщика вычисляется таким образом, чтобы вместить все элементы и, по возможности, избежать появления полос прокрутки.

Использование в версии:

Доступен, начиная с версии 8.3.15.
