АдминистрированиеОграничениеПотребленияРесурсов (AdministrationResourceConsumptionLimit)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового или обновление существующего ограничения потребления ресурсов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
