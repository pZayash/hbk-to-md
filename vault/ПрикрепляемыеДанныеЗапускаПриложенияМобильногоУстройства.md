Элементы коллекции:

Строка, [ЗапускПриложенияМобильногоУстройства](ЗапускПриложенияМобильногоУстройства.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы прикрепляемых данных в контейнере.  
Возможно обращение к элементу прикрепляемых данных посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

Описание:

Представляет собой прикрепляемые данные в буфере обмена между приложениями.   
Это сложный тип, содержащий один или несколько элементов, каждый из которых может содержать одно или несколько представлений данных элемента. Для отображения пользователю объекту можно установить надпись. Каждый элемент может быть одного из следующих типов: простой текст, один объект [ЗапускПриложенияМобильногоУстройства](ЗапускПриложенияМобильногоУстройства.md) или строковая ссылка на ресурс.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
