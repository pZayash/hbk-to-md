ОбъявлениеЭлементаXS (XSElementDeclaration)

Ограничение (Constraint)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОграничениеЗначенияXS](ОграничениеЗначенияXS_(XSConstraint).md).   
Содержит вид ограничения значения свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет атрибуты ограничения значения объявления атрибута (<http://www.w3.org/TR/xmlschema-1/#a-value_constraint>) или объявления элемента (<http://www.w3.org/TR/xmlschema-1/#e-value_constraint>).

Использование в версии:

Доступен, начиная с версии 8.1.
