Расширение таблицы формы для значений параметров компоновки данных (Form table extension for data composition parameter values)

ОграниченияИспользования (UseRestrictions)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОграниченияИспользованияДоступныхПараметровКомпоновкиДанных](ОграниченияИспользованияДоступныхПараметровКомпоновкиДанных.md).   
Содержит коллекцию ограничений для использования элементов в данной таблице.  
С помощью ограничений разработчик может исключить часть элементов этой коллекции, показ которых для пользователя нецелесообразен.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
