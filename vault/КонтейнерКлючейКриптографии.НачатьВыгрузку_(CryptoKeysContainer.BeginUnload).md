# КонтейнерКлючейКриптографии.НачатьВыгрузку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Криптография](Криптография-2.md) › [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md) › [Методы](КонтейнерКлючейКриптографии__Методы.md)

КонтейнерКлючейКриптографии (CryptoKeysContainer)

НачатьВыгрузку (BeginUnload)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: Данные в двоичном виде

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- ВыгруженныеДанные - данные типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md);
- ДополнительныеПараметры - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание варианта метода:

Передает выгруженные данные в параметре типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Вариант синтаксиса: Данные в файл

Синтаксис:

НачатьВыгрузку(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения выгрузки сертификата со следующими параметрами:

- ИмяФайла - имя файла, в который выгружен сертификат;
- ДополнительныеПараметры - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выгружен контейнер ключей криптографии.

Описание варианта метода:

Выгружает контейнер ключей криптографии в файл.

Описание:

Возвращает данные, использованные при создании или инициализации объекта.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
