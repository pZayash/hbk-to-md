ВидИерархии (HierarchyType)

ИерархияЭлементов (HierarchyOfItems)

Доступен, начиная с версии 8.0.

Описание:

Все элементы равноправны. Любой элемент может являться родителем.

Использование в версии:

Доступен, начиная с версии 8.0.
