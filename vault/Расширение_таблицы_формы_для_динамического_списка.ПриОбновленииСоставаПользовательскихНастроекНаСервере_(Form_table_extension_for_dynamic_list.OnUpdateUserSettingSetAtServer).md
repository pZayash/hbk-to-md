Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ПриОбновленииСоставаПользовательскихНастроекНаСервере (OnUpdateUserSettingSetAtServer)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПриОбновленииСоставаПользовательскихНастроекНаСервере(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в обработчике установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), то после обработчика не будет выполняться стандартная обработка данного события.  
В стандартной обработке события расширение формы динамического списка при помощи метода [СоздатьЭлементыФормыПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_компоновщика_настроек.СоздатьЭлементыФормыПользовательскихНастроек_(Client_application_form_extension_for_settings_composer.CreateUserSettingsFormItems).md) создает в указанной группе элементы формы для быстрого редактирования пользовательских настроек.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при обновлении состава пользовательских настроек:

- При получении формы;
- После загрузки настроек;
- После изменения настроек, если изменился состав настроек.

.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
