Описание:

Содержит варианты положений картинки кнопки относительно текста. Используется при определении свойства [ПоложениеКартинки](Кнопка.ПоложениеКартинки_(Button.PictureLocation).md) для элемента управления [Кнопка](Кнопка_(Button).md).

Доступность:

Толстый клиент.

См. также:

[Кнопка](Кнопка_(Button).md), свойство [ПоложениеКартинки](Кнопка.ПоложениеКартинки_(Button.PictureLocation).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
