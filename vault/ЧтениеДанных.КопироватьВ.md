ЧтениеДанных (DataReader)

КопироватьВ (CopyTo)

Доступен, начиная с версии 8.3.10.

Синтаксис:

КопироватьВ(<Поток>, <Количество>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md): [ЗаписьДанных](ЗаписьДанных.md).   
Поток или объект [ЗаписьДанных](ЗаписьДанных.md), в который требуется скопировать данные.

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые необходимо скопировать. Если не задано, будут скопированы все данные до конца потока.

Возвращаемое значение:

Тип: Число.   
Фактически записанное количество байтов.

Описание:

Копирует заданное количество байтов из объекта [ЧтениеДанных](ЧтениеДанных.md) в указанный поток или объект [ЗаписьДанных](ЗаписьДанных.md).  
Если количество не указано, копируются все данные из объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЧтениеДанных](ЧтениеДанных.md), метод [НачатьКопированиеВ](ЧтениеДанных.НачатьКопированиеВ.md)  
[ЧтениеДанных](ЧтениеДанных.md), метод [КопироватьВАсинх](ЧтениеДанных.КопироватьВАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
