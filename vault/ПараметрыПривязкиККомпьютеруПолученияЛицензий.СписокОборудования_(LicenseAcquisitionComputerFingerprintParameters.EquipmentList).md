ПараметрыПривязкиККомпьютеруПолученияЛицензий (LicenseAcquisitionComputerFingerprintParameters)

СписокОборудования (EquipmentList)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Фиксированный массив элементов типа [Строка](lang__def_String.md). Все параметры устройств компьютера, участвующие в привязке лицензии, в формате:  
"имя\_устройства: параметры\_устройства".

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
