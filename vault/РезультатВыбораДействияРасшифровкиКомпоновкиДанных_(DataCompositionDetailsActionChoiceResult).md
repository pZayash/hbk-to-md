Описание:

Содержит результат выбора действия расшифровки компоновки данных. Результат включает две составляющие, находящиеся в свойствах объекта: <ВыбранноеДействие> и <ПараметрВыбранногоДействия>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ОбработкаРасшифровкиКомпоновкиДанных](ОбработкаРасшифровкиКомпоновкиДанных_(DataCompositionDetailsProcess).md), метод [ВыбратьДействие](ОбработкаРасшифровкиКомпоновкиДанных.ВыбратьДействие_(DataCompositionDetailsProcess.ChooseAction).md)

Доступен, начиная с версии 8.3.18.
