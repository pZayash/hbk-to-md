ПользовательИнформационнойБазы (InfoBaseUser)

АутентификацияOpenID (OpenIDAuthentication)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, разрешена ли пользователю аутентификация по протоколу OpenID.   
[Истина](lang__def_BooleanTrue.md) - аутентификация разрешена.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.2.
