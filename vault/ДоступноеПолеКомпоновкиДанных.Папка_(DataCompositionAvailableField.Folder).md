ДоступноеПолеКомпоновкиДанных (DataCompositionAvailableField)

Папка (Folder)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, является ли поле папкой.  
[Истина](lang__def_BooleanTrue.md) - является папкой.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
