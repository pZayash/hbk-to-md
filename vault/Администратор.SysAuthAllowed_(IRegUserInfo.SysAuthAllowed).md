# Администратор.SysAuthAllowed

Администратор (IRegUserInfo)

SysAuthAllowed (SysAuthAllowed)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак разрешения администратору аутентификации пользователем операционной системы.  
[Истина](lang__def_BooleanTrue.md) - аутентификация разрешена. Данный администратор будет считаться аутентифицированным, если клиентское приложение работает от имени пользователя операционной системы, имя которого указано в качестве значения свойства [SysUserName](Администратор.SysUserName_(IRegUserInfo.SysUserName).md). При этом необходимо, чтобы компьютеры, на которых работают агент сервера и клиентское приложение, использовали один и тот же сервер аутентификации (например, контроллер домена Windows).   
[Ложь](lang__def_BooleanFalse.md) - запрещена.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
