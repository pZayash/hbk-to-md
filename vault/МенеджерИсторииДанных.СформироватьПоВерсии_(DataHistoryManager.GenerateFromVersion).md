Описание:

Получить объект для доступа к данным, сконструированный по данным версии. Если указанная версия объекта - удаление данных, будет восстановлено состояние данных перед удалением.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Версия  = ИсторияДанных.СформироватьПоВерсии( Справочники.Валюты.НайтиПоНаименованию("Рубль"), 2); Версия.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.3.11.
