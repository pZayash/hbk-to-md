ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ОсновнойМакетОформленияОтчета (DefaultReportAppearanceTemplate)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Позволяет указать макет, который будет использован в отчетах в качестве макета по умолчанию.  
Можно выбрать макет оформления компоновки данных или макет из ветки "Общие макеты" конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
