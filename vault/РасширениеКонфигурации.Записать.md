РасширениеКонфигурации (ConfigurationExtension)

Записать (Write)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Записать(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Если объект [РасширениеКонфигурации](РасширениеКонфигурации.md) представляет собой только что созданное расширение, при создании которого не были переданы данные, параметр должен быть установлен. Если он не установлен, будет сгенерирована ошибка "При создании расширения необходимы данные".  
При записи контролируется уникальность имени расширения. Если имя не уникально, генерируется ошибка "Имя расширения ... не уникально".  
Запись расширений в безопасном режиме запрещена.  
Значение по умолчанию: Неопределено.

Описание:

Сохраняет изменения в информационной базе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерРасширенийКонфигурации](МенеджерРасширенийКонфигурации.md), метод [Создать](МенеджерРасширенийКонфигурации.Создать.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
