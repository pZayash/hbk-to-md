ПостроительОтчета (ReportBuilder)

ПолучитьЗапрос (GetQuery)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗапрос()

Возвращаемое значение:

Тип: [Запрос](Запрос.md).   

Описание:

Получить запрос для выполнения отчета. Данный метод недоступен в случае, если установлено свойство [ИсточникДанных](ПостроительОтчета.ИсточникДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| РезультатЗапроса = ПостроительОтчета.ПолучитьЗапрос().Выполнить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
