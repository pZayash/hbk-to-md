# ДокументОбъект.<Имя документа>.ПриУстановкеНовогоНомера

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

ПриУстановкеНовогоНомера (OnSetNewNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриУстановкеНовогоНомера(<СтандартнаяОбработка>, <Префикс>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная генерация нового номера производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Префикс>

Тип: [Строка](lang__def_String.md).   
Префикс, который будет использоваться для генерации номера. Может быть изменен в процедуре-обработчике события.

Описание:

Возникает в момент, когда выполняется установка нового номера документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Параметр <Префикс> может быть указан, если [Номер](ДокументСсылка.Имя_документа.Номер_(DocumentRef.Document_name.Number).md) имеет тип [Строка](lang__def_String.md).

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

См. также:

[ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md), метод [УстановитьНовыйНомер](ДокументОбъект.Имя_документа.УстановитьНовыйНомер_(DocumentObject.Document_name.SetNewNumber).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
