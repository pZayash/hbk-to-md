СписокВыбораНавигационнойСсылки (URLChoiseList)

Удалить (Delete)

Доступен, начиная с версии 8.3.19.

Синтаксис:

Удалить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ удаляемого элемента.

Описание:

Удаляет элемент из коллекции.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.19.
