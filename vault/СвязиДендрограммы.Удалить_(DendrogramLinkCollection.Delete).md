СвязиДендрограммы (DendrogramLinkCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Связь>)

Параметры:

<Связь> (обязательный)

Тип: [СвязьДендрограммы](СвязьДендрограммы_(DendrogramLink).md).   
Удаляемая связь.

Описание:

Удаляет связь дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
