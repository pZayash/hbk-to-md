Планировщик (Planner)

ФиксироватьЗаголовокШкалыВремени (FixTimeScaleHeader)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет, будет ли зафиксирован заголовок шкалы времени планировщика при прокрутке.   
[Неопределено](lang__def_Undefined.md) - необходимость фиксации заголовка определяется автоматически.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы.

Примечание:

Значение [Неопределено](lang__def_Undefined.md) заполняется по умолчанию, в том числе и для уже добавленных на форму реквизитов типа [Планировщик](Планировщик_(Planner).md).  
В веб-клиенте под управлением Internet Explorer свойство всегда имеет значение [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
