Доступен, начиная с версии 8.0.

Только чтение.

Описание:

Тип: [Индексирование](Индексирование.md).   
Содержит варианты индексирования по реквизиту справочника, документа (а также по реквизиту табличной части), измерению регистра.

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
