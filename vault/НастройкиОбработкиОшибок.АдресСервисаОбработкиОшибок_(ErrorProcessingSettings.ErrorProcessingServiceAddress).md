НастройкиОбработкиОшибок (ErrorProcessingSettings)

АдресСервисаОбработкиОшибок (ErrorProcessingServiceAddress)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Строка, содержащая адрес сервиса регистрации ошибок.  
Если строка пустая в настройках пользователя, то адрес получается из общих настроек.  
Если пустая строка получена из общих настроек, то отчет не отправляется.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - Пустая строка.

Использование в версии:

Доступен, начиная с версии 8.3.21.
