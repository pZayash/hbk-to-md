ЭлементСтрокаТаблицыHTML (HTMLTableRowElement)

ВставитьЯчейку (InsertCell)

Доступен, начиная с версии 8.2.

Синтаксис:

ВставитьЯчейку(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: Число.   
Индекс ячейки, после которой будет добавлена новая ячейка.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML.md).   

Описание:

Предназначен для добавления ячейки в строку таблицы HTML-документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
