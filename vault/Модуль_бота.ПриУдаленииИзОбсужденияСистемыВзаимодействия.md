Модуль бота (Bot module)

ПриУдаленииИзОбсужденияСистемыВзаимодействия (OnDeleteFromCollaborationSystemConversation)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПриУдаленииИзОбсужденияСистемыВзаимодействия(<ИдентификаторОбсуждения>, <ДополнительныеПараметры>)

Параметры:

<ИдентификаторОбсуждения>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения, из которого удаляется бот.

<ДополнительныеПараметры>

Тип: Произвольный.   
Параметры, заданные при создании бота.

Описание:

Обработчик, вызываемый при удалении бота из обсуждения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
