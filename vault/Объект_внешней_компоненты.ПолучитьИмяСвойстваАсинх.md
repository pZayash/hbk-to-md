Объект внешней компоненты (Add-in object)

Получить<ИмяСвойства>Асинх (Get<PropertyName>Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Получить<ИмяСвойства>Асинх()

Возвращаемое значение:

Результат выполнения обещания - значение произвольного типа.

Описание:

Асинхронно получает значение свойства внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
