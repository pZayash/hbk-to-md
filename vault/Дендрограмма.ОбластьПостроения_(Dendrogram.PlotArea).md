ОбластьПостроения (PlotArea)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ОбластьПостроенияДендрограммы](ОбластьПостроенияДендрограммы_(DendrogramPlotArea).md).   
Содержит область построения дендрограммы для установки параметров оформления.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

[ОбластьПостроенияДиаграммы](ОбластьПостроенияДиаграммы_(ChartPlotArea).md)
