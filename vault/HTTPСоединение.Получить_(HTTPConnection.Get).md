HTTPСоединение (HTTPConnection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<HTTPЗапрос>, <ИмяВыходногоФайла>)

Параметры:

<HTTPЗапрос> (обязательный)

Тип: [HTTPЗапрос](HTTPЗапрос_(HTTPRequest).md).   
HTTP-запрос.

<ИмяВыходногоФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который помещаются данные полученного ресурса. Если не указан или содержит пустую строку, то тело ответа может быть получено из объекта [HTTPОтвет](HTTPОтвет_(HTTPResponse).md).

Возвращаемое значение:

Тип: [HTTPОтвет](HTTPОтвет_(HTTPResponse).md).   

Описание:

Получает данные с HTTP-сервера, используя GET-запрос.  
При вызове на клиенте данный метод, начиная с [Версия8\_3\_21](РежимСовместимости.Версия8_3_21_(CompatibilityMode.Version8_3_21).md), признан устаревшим и вместо него требуется использовать [ПолучитьАсинх](HTTPСоединение.ПолучитьАсинх_(HTTPConnection.GetAsync).md).

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
