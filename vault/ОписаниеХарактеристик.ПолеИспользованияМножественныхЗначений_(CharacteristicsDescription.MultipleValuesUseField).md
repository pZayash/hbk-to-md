ОписаниеХарактеристик (CharacteristicsDescription)

ПолеИспользованияМножественныхЗначений (MultipleValuesUseField)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [Поле](Поле_(Field).md).   
Содержит имя поля таблицы списка характеристик, в котором содержится признак того, что свойство содержит несколько значений.   
Характеристика будет считаться множественной, если в этом поле будет находиться значение [Истина](lang__def_BooleanTrue.md) и не множественной в противном случае.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
