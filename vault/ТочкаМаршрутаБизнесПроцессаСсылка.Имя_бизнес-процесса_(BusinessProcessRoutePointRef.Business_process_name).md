Описание:

Точка маршрута бизнес-процесса. Точки маршрута отличаются видом - [Старт](ВидТочкиМаршрутаБизнесПроцесса.Старт_(BusinessProcessRoutePointType.Start).md), [Завершение](ВидТочкиМаршрутаБизнесПроцесса.Завершение_(BusinessProcessRoutePointType.End).md), [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md) и т.д. У точек некоторых видов есть специфические свойства и события. Например, свойство [Групповая](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.Групповая_(BusinessProcessRoutePointRef.Business_process_name.Group).md) есть только у точки вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md), а событие [ПроверкаУсловия](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПроверкаУсловия_(BusinessProcessRoutePointRef.Business_process_name.ConditionCheck).md) - только у точки условного перехода. При попытке обращения к свойству, которое отсутствует у данной точке будет выдано сообщение об ошибке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise/current-config}. Имя типа XDTO: BusinessProcessRoutePointRef.
