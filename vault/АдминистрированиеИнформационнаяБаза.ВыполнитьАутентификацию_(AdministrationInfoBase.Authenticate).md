АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ВыполнитьАутентификацию (Authenticate)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ВыполнитьАутентификацию(<ИмяПользователя>, <Пароль>)

Параметры:

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя информационной базы.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя для аутентификации пользователя паролем. Пользователь будет аутентифицирован, если совпадает его имя и пароль.

Описание:

Выполняет аутентификацию пользователя в информационной базе.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
