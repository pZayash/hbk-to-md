ПодписьКриптографии (CryptoSignature)

Данные (Data)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md), Неопределено.   
Содержит подписанные данные, хранящиеся в самой подписи. Если подпись отсоединенная и не хранит данных, то содержит Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.24.
