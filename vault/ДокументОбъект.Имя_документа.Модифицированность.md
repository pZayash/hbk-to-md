ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: Булево.   
Истина - объект изменен; Ложь - в противном случае.

Описание:

Определяет, был ли изменен объект после считывания из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не позволяет определить, был ли изменен объект другими пользователями.

Пример:

|  |
| --- |
| Если РасходнаяНакладная.Модифицированность() Тогда     Предупреждение(НСтр("ru = 'Документ был изменен.';"      + " en = 'Document was changed.'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
