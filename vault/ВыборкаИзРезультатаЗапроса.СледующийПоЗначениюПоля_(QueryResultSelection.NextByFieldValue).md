ВыборкаИзРезультатаЗапроса (QueryResultSelection)

СледующийПоЗначениюПоля (NextByFieldValue)

Доступен, начиная с версии 8.0.

Синтаксис:

СледующийПоЗначениюПоля(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля результата запроса, в котором будет осуществляться поиск следующего значения.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующая запись выбрана; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующую запись по значению указанного поля. Выборка позиционируется на следующую запись со значением, отличающимся от текущего значения, по указанному полю. При первом вызове - остается на текущей записи.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Пока Выборка.СледующийПоЗначениюПоля("Товар") Цикл     ... КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
