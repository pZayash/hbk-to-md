ПараметрыМакетаТабличногоДокумента (SpreadsheetDocumentTemplateParameters)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс параметра.

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение.

Описание:

Устанавливает значение по индексу. Работает аналогично оператору [].

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
