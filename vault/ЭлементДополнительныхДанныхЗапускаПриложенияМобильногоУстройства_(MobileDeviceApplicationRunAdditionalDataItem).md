Описание:

Описание дополнительного данного, передаваемого параметром в запускаемое приложение на мобильном устройстве.  
Используется для преобразования значений типов "1С:Предприятия" в значения Java-типов.  
Допустимые имена типов: byte, char, double, float, int, long, short, boolean, String, Uri, Bitmap, Account, Address, Location, Bundle.  
Указание имени Java-типа необязательно, поскольку может быть определено по типу значения. По умолчанию выполняются следующие преобразования типов:

- Строка –> String,
- Массив строк –> String[],
- Булево –> boolean,
- Массив значений типа Булево –> boolean[],
- Число –> double,
- Массив чисел –> double[],
- НавигационнаяСсылка –> String,
- ДанныеМестоположения –> Location,
- ДанныеАдреса –> Address,
- Картинка –> Bitmap,
- УчетнаяЗаписьКонтакта –> Account,
- ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства –> Bundle (иерархическая структура дополнительных данных),
- Массив строк при указании в имени типа ArrayList -> ArrayList<String>,
- Массив значений типов ДанныеМестоположения, ДанныеАдреса, Картинка или УчетнаяЗаписьКонтакта -> Parcelable[],
- Массив значений типов ДанныеМестоположения, ДанныеАдреса, Картинка или УчетнаяЗаписьКонтакта при указании в имени типа ArrayList -> ArrayList<? extends Parcelable>,
- File -> Uri c "content"-ссылкой на файл, для доступа снаружи приложения,
- Массив значений типа File -> Uri[] c "content"-ссылками на файлы, для доступа снаружи приложения,
- Массив значений типа File при указании в имени типа ArrayList -> ArrayList<Uri> c "content" ссылками на файлы, для доступа снаружи приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.23.
