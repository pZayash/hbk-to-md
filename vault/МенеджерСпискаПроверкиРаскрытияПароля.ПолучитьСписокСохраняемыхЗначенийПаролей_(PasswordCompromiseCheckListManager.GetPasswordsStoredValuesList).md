МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

ПолучитьСписокСохраняемыхЗначенийПаролей (GetPasswordsStoredValuesList)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСписокСохраняемыхЗначенийПаролей(<ВыходныеДанные>)

Параметры:

<ВыходныеДанные> (необязательный)

Тип: [Строка](lang__def_String.md), [Поток](Поток_(Stream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [Неопределено](lang__def_Undefined.md).   
Если передан параметр типа [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md) – список хэш-значений записывается в соответствующий поток.   
Если передана строка с путем к файлу - списо сохраняемых значений будет записан в этот файл.  
Если передано [Неопределено](lang__def_Undefined.md), метод вернет [Массив](Массив_(Array).md) сохраняемых значений паролей.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Неопределено](lang__def_Undefined.md), [Массив](Массив_(Array).md).   
Элементы массива имеют тип [Строка](lang__def_String.md) и содержат хеш-значения паролей, вычисленные алгоритмом SHA1 и закодированные в base64, разделенные переносом строки.

Описание:

Возвращает список сохраняемых значений раскрытых паролей.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| СписокПроверкиРаскрытияПароляМассив =      ДополнительныеНастройкиАутентификации.СписокПроверкиРаскрытияПароля.     ПолучитьСписокСохраняемыхЗначенийПаролей(); Для Каждого СохрЗнач Из СписокПроверкиРаскрытияПароляМассив Цикл     Сообщить(СохрЗнач); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.3.26.
