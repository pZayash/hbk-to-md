Планировщик (Planner)

ОтступСНачалаПереносаШкалыВремени (TimeScaleWrapBeginIndent)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Устанавливает отступ с начала отображения после переноса шкалы времени в единицах, указанных в свойстве [ЕдиницаПериодическогоВарианта](Планировщик.ЕдиницаПериодическогоВарианта.md).   
Пример: рабочий день начинается в 9:00 и заканчивается в 17:00, но на шкале времени отображается полный диапазон времени – с 00:00 до 24:00. Данное свойство и свойство [ОтступСКонцаПереносаШкалыВремени](Планировщик.ОтступСКонцаПереносаШкалыВремени.md) предназначены для настройки диапазона отображаемого времени – так, чтобы на календаре отображался только рабочий день в указанном выше диапазоне. Для этого следует установить данному свойству значение "9", а свойству [ОтступСКонцаПереносаШкалыВремени](Планировщик.ОтступСКонцаПереносаШкалыВремени.md) – "7".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
