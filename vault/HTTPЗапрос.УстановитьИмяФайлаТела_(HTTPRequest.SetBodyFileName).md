# HTTPЗапрос.УстановитьИмяФайлаТела

HTTPЗапрос (HTTPRequest)

УстановитьИмяФайлаТела (SetBodyFileName)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьИмяФайлаТела(<ПутьКФайлу>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: [Строка](lang__def_String.md).   
Указывает путь к файлу.

Описание:

Устанавливает имя файла, из которого будет прочитано тело запроса.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Во всех клиентах, кроме веб-клиента, в момент вызова метода файл открывается на чтение, но само чтение происходит только в момент отправки запроса. В веб-клиенте метод поддерживается только при подключенном расширении работы с файлами, но проверка на подключенность, открытие файла на чтение производятся в момент отправки запроса.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
