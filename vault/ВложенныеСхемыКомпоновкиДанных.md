Доступен, начиная с версии 8.1.

Элементы коллекции:

[ВложеннаяСхемаКомпоновкиДанных](ВложеннаяСхемаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются вложенные схемы.  
Возможно обращение к вложенной схеме посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Описание:

Содержит вложенные схемы компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
