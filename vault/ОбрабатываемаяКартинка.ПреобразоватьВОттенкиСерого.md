ОбрабатываемаяКартинка (ProcessingPicture)

ПреобразоватьВОттенкиСерого (ConvertToGrayScale)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПреобразоватьВОттенкиСерого()

Описание:

Преобразует все цвета картинки в градации серого цвета и записывает значение для соответствующего свойства картинки.  
Следует учитывать, что такое свойство используется для форматов PNG и TIFF.  
  
На мобильной платформе соответствующее свойство в картинке не записывается, даже если оно поддерживается форматом PNG.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
