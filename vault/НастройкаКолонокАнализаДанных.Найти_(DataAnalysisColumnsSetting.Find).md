# НастройкаКолонокАнализаДанных.Найти

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md) › [НастройкаКолонокАнализаДанных](НастройкаКолонокАнализаДанных_(DataAnalysisColumnsSetting).md) › [Методы](НастройкаКолонокАнализаДанных__Методы.md)

НастройкаКолонокАнализаДанных (DataAnalysisColumnsSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента настройки колонок.

Возвращаемое значение:

Тип: [ЭлементУправленияКолонкамиАнализаДанных](ЭлементУправленияКолонкамиАнализаДанных_(DataAnalysisColumnsControl).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента настройки колонок по имени элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
