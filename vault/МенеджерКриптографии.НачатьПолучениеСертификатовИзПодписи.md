МенеджерКриптографии (CryptoManager)

НачатьПолучениеСертификатовИзПодписи (BeginGettingCertificatesFromSignature)

Не рекомендуется использовать, начиная с версии 8.3.24.

Рекомендуется использовать:

- [ПолучитьСертификаты](МенеджерКриптографии.ПолучитьСертификаты.md)
- [ПолучитьСертификатыАсинх](МенеджерКриптографии.ПолучитьСертификатыАсинх.md)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеСертификатовИзПодписи(<ОписаниеОповещения>, <ИсходныеДанные>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения получения сертификатов из подписи со следующими параметрами:

- <Сертификаты> - массив объектов [СертификатКриптографии](СертификатКриптографии.md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИсходныеДанные> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md).   
Исходные данные, из которых будет извлечен сертификат.  
Данные могут размещаться в файле (в этом случае указывается имя файла) или представлены как [ДвоичныеДанные](ДвоичныеДанные.md).

Описание:

Начинает извлечение массива сертификатов из данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПолучитьСертификатыИзПодписи](МенеджерКриптографии.ПолучитьСертификатыИзПодписи.md)  
[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПолучитьСертификатыИзПодписиАсинх](МенеджерКриптографии.ПолучитьСертификатыИзПодписиАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.24.
