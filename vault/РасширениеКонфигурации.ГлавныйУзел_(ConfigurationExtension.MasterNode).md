РасширениеКонфигурации (ConfigurationExtension)

ГлавныйУзел (MasterNode)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md), [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>.   
Содержит узел распределенной информационной базы, в котором создано данное расширение конфигурации. Если текущая информационная база не является узлом распределенной информационной базы или расширение создано локально, то содержит [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.3.12.
