УстановитьНаименованиеПриложения (SetApplicationDescription)

Доступен, начиная с версии 8.3.13.

Синтаксис:

УстановитьНаименованиеПриложения(<НаименованиеПриложения>)

Параметры:

<НаименованиеПриложения> (обязательный)

Устанавливает текущее наименование приложения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
