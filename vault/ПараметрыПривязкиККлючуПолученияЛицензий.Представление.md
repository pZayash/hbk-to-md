ПараметрыПривязкиККлючуПолученияЛицензий (LicenseAcquisitionKeyFingerprintParameters)

Представление (Presentation)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Локализованное словесное описание ключа, как его показывает встроенный диалог получения лицензии. Например: "На 100 пользователей 28374623".

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
