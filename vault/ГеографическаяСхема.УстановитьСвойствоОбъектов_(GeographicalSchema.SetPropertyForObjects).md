ГеографическаяСхема (GeographicalSchema)

УстановитьСвойствоОбъектов (SetPropertyForObjects)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьСвойствоОбъектов(<СписокОбъектов>, <ИмяСвойства>, <Значение>)

Параметры:

<СписокОбъектов> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Список объектов, для которых будет устанавливаться указанное свойство.

<ИмяСвойства> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя свойства, которое будет устанавливаться у переданных объектов.

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение.

Описание:

Устанавливает свойство объектов географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
