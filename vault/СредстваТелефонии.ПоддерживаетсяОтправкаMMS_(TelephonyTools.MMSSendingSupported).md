СредстваТелефонии (TelephonyTools)

ПоддерживаетсяОтправкаMMS (MMSSendingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяОтправкаMMS(<Интерактивно>)

Параметры:

<Интерактивно> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, используется ли интерактивный режим для отправки MMS-сообщений.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
При интерактивной отправке возвращает:

- [Истина](lang__def_BooleanTrue.md) - на устройствах под управлением Android,
- [Ложь](lang__def_BooleanFalse.md) - для остальных операционных систем.

При неинтерактивной отправке всегда возвращает [Ложь](lang__def_BooleanFalse.md).

Описание:

Определяет доступность отправки сообщений MMS с данного устройства.  
Если не поддерживается, то при вызове метода [ПослатьSMS](СредстваТелефонии.ПослатьSMS_(TelephonyTools.SendSMS).md) вложения сообщения будут игнорироваться.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПослатьSMS](СредстваТелефонии.ПослатьSMS_(TelephonyTools.SendSMS).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
