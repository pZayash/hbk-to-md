# КоллекцияСтрокДереваЗначений.Удалить

КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По объекту

Синтаксис:

Удалить(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md).   
Строка, которую необходимо удалить из данной коллекции строк [ДеревоЗначений](ДеревоЗначений_(ValueTree).md).

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемой строки.

Описание:

Удаляет строку из коллекции строк данного уровня дерева значений. Если у строки имеются подчиненные строки, они также будут удалены.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СтрокиСостава.Удалить(6); |

Использование в версии:

Доступен, начиная с версии 8.0.
