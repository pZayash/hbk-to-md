УсловноеОформление (ConditionalAppearance)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Имя>, <Представление>, <Позиция>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя добавляемого оформления области.

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md).   
Представление добавляемого оформления.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо добавить оформление.

Возвращаемое значение:

Тип: [ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md).   

Описание:

Вставить область в указанную позицию коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
