Глобальный контекст (Global context)

ОбработкаФормированияКомандВходящегоЗапросаПоделиться (IncomingShareRequestCommandGenerateProcessing)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ОбработкаФормированияКомандВходящегоЗапросаПоделиться(<ДанныеЗапросаПоделиться>, <Команды>)

Параметры:

<ДанныеЗапросаПоделиться>

Тип: [ДанныеЗапросаПоделиться](ДанныеЗапросаПоделиться.md).   
Описывает параметры вызова обработчика.

<Команды>

Тип: [ОписаниеКомандыВходящегоЗапросаПоделиться](ОписаниеКомандыВходящегоЗапросаПоделиться.md).   
Значения элементов массива имеют тип [ОписаниеКомандыВходящегоЗапросаПоделиться](ОписаниеКомандыВходящегоЗапросаПоделиться.md).  
Значения другого типа игнорируются. Задает список команд для меню. При вызове список заполнен командами меню по умолчанию.  
Если в списке после вызова обработчика останется 1 команда, она сразу будет вызвана без показа меню.

Описание:

Возникает при формировании команд входящего запроса "Поделиться".

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
