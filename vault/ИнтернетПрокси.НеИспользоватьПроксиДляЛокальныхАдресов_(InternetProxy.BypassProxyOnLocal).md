# ИнтернетПрокси.НеИспользоватьПроксиДляЛокальныхАдресов

ИнтернетПрокси (InternetProxy)

НеИспользоватьПроксиДляЛокальныхАдресов (BypassProxyOnLocal)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено в [Истина](lang__def_BooleanTrue.md), то при запросе ресурсов с локальных адресов прокси не используется.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
