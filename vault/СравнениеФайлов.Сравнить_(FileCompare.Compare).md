СравнениеФайлов (FileCompare)

Сравнить (Compare)

Доступен, начиная с версии 8.0.

Синтаксис:

Сравнить()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - файлы равны, [Ложь](lang__def_BooleanFalse.md) – в противном случае.

Описание:

Сравнивает файлы. Позволяет узнать, есть ли в них различия.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
