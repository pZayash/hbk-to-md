ЭлементКолонкаТаблицыHTML (HTMLTableColElement)

Нормализовать (Normalize)

Доступен, начиная с версии 8.2.

Синтаксис:

Нормализовать()

Описание:

Приводит все узлы [Текст](ТипУзлаDOM.Текст.md) на полной глубине поддерева данного узла, включая узлы [Атрибут](ТипУзлаDOM.Атрибут.md), в "нормальную" форму, где только структуры (например, элементы, комментарии) разделяются текстовыми узлами, т.е. смежные текстовые узлы отсутствуют.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
