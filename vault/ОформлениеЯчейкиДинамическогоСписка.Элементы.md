ОформлениеЯчейкиДинамическогоСписка (DynamicListCellAppearance)

Элементы (Items)

Доступен, начиная с версии 8.3.10.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийПараметровКомпоновкиДанных](КоллекцияЗначенийПараметровКомпоновкиДанных.md).   
Коллекция значений оформления компоновки данных.  
Для оформления ячейки динамического списка используются следующие элементы:

- ЦветФона (BackColor) - тип значения [Цвет](Цвет.md);
- ЦветТекста (TextColor) - тип значения [Цвет](Цвет.md);
- Шрифт (Font) - тип значения [Шрифт](Шрифт.md);
- ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md);
- Видимость (Visible) – тип значения Булево;
- Доступность (Enabled) – тип значения Булево;
- Отображать (Show) – тип значения Булево;
- Текст (Text) - тип значения Строка;
- Формат (Format) - тип значения Строка;
- Картинка (Picture) - тип значения [Картинка](Картинка.md).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
