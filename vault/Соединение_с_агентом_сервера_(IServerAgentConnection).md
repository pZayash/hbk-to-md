Описание:

Соединение с агентом сервера. С его помощью можно управлять кластерами.  
Представляет собой объект с интерфейсом IV8AgentConnection.

Доступность:

Интеграция.

[Менеджер COM-соединений](Менеджер_COM-соединений_(COM_connector).md), метод [ConnectAgent](Менеджер_COM-соединений.ConnectAgent_(COM_connector.ConnectAgent).md)

Доступен, начиная с версии 8.1.
