ФорматированныйДокумент (FormattedDocument)

НайтиТекст (FindText)

Доступен, начиная с версии 8.3.1.

Синтаксис:

НайтиТекст(<Строка>, <Начало>, <Конец>, <ИскатьВперед>, <ИгнорироватьРегистр>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомый текст.

<Начало> (необязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Закладка, указывающая на начало диапазона поиска. Если не задан, поиск будет выполнен с начала форматиорванного документа.

<Конец> (необязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Закладка, указывающая на конец диапазона поиска. Если не задан, поиск будет выполнен до конца форматиорванного документа.

<ИскатьВперед> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет направление поиска: [Истина](lang__def_BooleanTrue.md) - искать вперед; [Ложь](lang__def_BooleanFalse.md) - искать назад.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ИгнорироватьРегистр> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Учет регистра.  
[Истина](lang__def_BooleanTrue.md) - без учета регистра, [Ложь](lang__def_BooleanFalse.md) - поиск с учетом регистра.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ОбластьФорматированногоДокумента](ОбластьФорматированногоДокумента_(FormattedDocumentRange).md), [Неопределено](lang__def_Undefined.md).   
[ОбластьФорматированногоДокумента](ОбластьФорматированногоДокумента_(FormattedDocumentRange).md) - содержит закладки начала и конца ближайшего найденного текста.  
Если текст не найден, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск текста в форматированном документе.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
