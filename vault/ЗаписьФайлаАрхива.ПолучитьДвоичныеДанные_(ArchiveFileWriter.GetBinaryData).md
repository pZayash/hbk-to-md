# ЗаписьФайлаАрхива.ПолучитьДвоичныеДанные

ЗаписьФайлаАрхива (ArchiveFileWriter)

ПолучитьДвоичныеДанные (GetBinaryData)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьДвоичныеДанные(<Пароль>, <ТипФайлаАрхива>, <Комментарий>, <МетодСжатия>, <УровеньСжатия>, <МетодШифрования>)

Параметры:

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль, который будет назначен архиву.   
Если пароль не назначен или является пустой строкой, то шифрование не происходит.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива_(ArchiveFileType).md).   
Тип файла архива, который требуется открыть для чтения, определяется по расширению имени файла.   
Допускается только значение ZIP.  
Значение по умолчанию: [ZIP](ТипФайлаАрхива.ZIP_(ArchiveFileType.ZIP).md).  
Значение по умолчанию: [ZIP](ТипФайлаАрхива.ZIP_(ArchiveFileType.ZIP).md).

<Комментарий> (необязательный)

Тип: [Строка](lang__def_String.md).   
Комментарий, описывающий файл архива.

<МетодСжатия> (необязательный)

Тип: [МетодСжатияФайлаАрхива](МетодСжатияФайлаАрхива_(ArchiveFileCompressionMethod).md).   
Метод сжатия, которым будет сжиматься файл архива.  
Значение по умолчанию: Сжатие.

<УровеньСжатия> (необязательный)

Тип: [УровеньСжатияФайлаАрхива](УровеньСжатияФайлаАрхива_(ArchiveFileCompressionLevel).md).   
Уровень сжатия данных.  
Значение по умолчанию: [Оптимальный](УровеньСжатияФайлаАрхива.Оптимальный_(ArchiveFileCompressionLevel.Optimal).md).

<МетодШифрования> (необязательный)

Тип: [МетодШифрованияФайлаАрхива](МетодШифрованияФайлаАрхива_(ArchiveFileEncryptionMethod).md).   
Метод шифрования, которым будет зашифрован архив.  
Значение по умолчанию: [Zip20](МетодШифрованияФайлаАрхива.Zip20_(ArchiveFileEncryptionMethod.Zip20).md).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Получает файл архива в виде двоичных данных и закрывает файл.  
При использовании данного метода объект [ЗаписьФайлаАрхива](ЗаписьФайлаАрхива_(ArchiveFileWriter).md) должен быть создан с помощью конструктора без параметров.   
При этом открывать архив явно с помощью метода [Открыть](ЗаписьФайлаАрхива.Открыть_(ArchiveFileWriter.Open).md) не требуется.  
При завершении работы данный метод закрывает архив, аналогично методу [Записать](ЗаписьФайлаАрхива.Записать_(ArchiveFileWriter.Write).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
