Расширение формы документа (Document form extension)

ИспользоватьРежимПроведения (UsePostingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеРежимаПроведения](ИспользованиеРежимаПроведения.md).   
Содержит режим проведения документа. Значение данного свойства будет использоваться для определения режима проведения, если оно будет вызываться средствами формы (методом [ЗаписатьВФорме](Расширение_формы_документа.ЗаписатьВФорме.md), стандартным действием команды).

Доступность:

Толстый клиент.

См. также:

[Расширение формы документа](Расширение_формы_документа.md), метод [ЗаписатьВФорме](Расширение_формы_документа.ЗаписатьВФорме.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
