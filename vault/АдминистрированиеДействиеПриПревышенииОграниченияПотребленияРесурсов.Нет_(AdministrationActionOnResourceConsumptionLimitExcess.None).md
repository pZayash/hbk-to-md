АдминистрированиеДействиеПриПревышенииОграниченияПотребленияРесурсов (AdministrationActionOnResourceConsumptionLimitExcess)

Нет (None)

Доступен, начиная с версии 8.3.14.

Описание:

При превышении предельных значений, указанных в ограничении потребления ресурсов, никакие действия не выполняются.

Использование в версии:

Доступен, начиная с версии 8.3.14.
