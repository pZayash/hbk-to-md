# Кластер серверов.PingTimeout

Кластер серверов (IClusterInfo)

PingTimeout (PingTimeout)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет таймаут получения ответов на проверочные запросы системы отслеживания соединений, в миллисекундах. Допустимые значения: от 0 до 2147483648.   
Если данное свойство и свойство [Каталог](objects__catalog63__catalog2843__BinaryDataExternalStorageManager__properties__Directory13855.md) имеют значение 0, то используются параметры системы отслеживания соединений, установленные с помощью командной строки.  
Значение свойства должно быть установлено вместе со значением свойства [PingPeriod](Кластер_серверов.PingPeriod_(IClusterInfo.PingPeriod).md).  
Значения по умолчанию:

- PingPeriod - 1000 мс,
- PingTimeout - 5000 мс.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
