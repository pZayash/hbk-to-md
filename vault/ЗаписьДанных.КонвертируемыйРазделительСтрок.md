ЗаписьДанных (DataWriter)

КонвертируемыйРазделительСтрок (ConvertibleSplitterOfLines)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Конвертируемый разделитель строк. Этот параметр влияет на поведение метода [ЗаписатьСимволы](ЗаписьДанных.ЗаписатьСимволы.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьСимволы](ЗаписьДанных.ЗаписатьСимволы.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
