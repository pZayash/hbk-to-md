ИсточникКомандПланировщика (PlannerCommandSource)

Элементы (Items)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [Элементы](ИсточникКомандПоляПланировщика.Элементы_(PlannerCommandSource.Items).md)

Доступен, начиная с версии 8.3.16.

Описание:

Тип: [ИсточникКомандПланировщика](ИсточникКомандПоляПланировщика_(PlannerCommandSource).md).   
Значение, которое устанавливается при нажатии на элементы поля планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд_(Form_field_extension_for_a_planner.CommandGenerateProcessing).md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы_(PlannerCommandGenerateParameters.Items).md) заполнено,
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения_(PlannerCommandGenerateParameters.DimensionItem).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений_(PlannerCommandGenerateParameters.DimensionValues).md) содержит [Неопределено](lang__def_Undefined.md),
- [Дата](ПараметрыФормированияКомандПланировщика.Дата_(PlannerCommandGenerateParameters.Date).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandGenerateParameters.TimeScaleItem).md) содержит [Неопределено](lang__def_Undefined.md),
- Команды заполнены следующими стандартными командами планировщика: [УдалитьЭлементы](СтандартнаяКомандаПланировщика.УдалитьЭлементы_(PlannerStandardCommand.DeleteItems).md), [РедактироватьЭлемент](СтандартнаяКомандаПланировщика.РедактироватьЭлемент_(PlannerStandardCommand.EditItem).md) (недоступно, если элементов несколько),
- [КомандаПоУмолчанию](html.md) заполнено командой [БыстроРедактироватьЭлемент](СтандартнаяКомандаПланировщика.БыстроРедактироватьЭлемент_(PlannerStandardCommand.QuickEditItem).md) (игнорируется, если элементов несколько).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
