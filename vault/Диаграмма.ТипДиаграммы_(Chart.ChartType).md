Диаграмма (Chart)

ТипДиаграммы (ChartType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипДиаграммы](ТипДиаграммы_(ChartType).md).   
Определяет тип диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Следует учитывать, что для некоторых видов диаграмм отдельные свойства диаграммы могут отсутствовать или быть недоступны.

См. также:

[ТипДиаграммы](ТипДиаграммы_(ChartType).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
