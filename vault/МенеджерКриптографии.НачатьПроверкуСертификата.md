МенеджерКриптографии (CryptoManager)

НачатьПроверкуСертификата (BeginCheckingCertificate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПроверкуСертификата(<ОписаниеОповещения>, <Сертификат>, <РежимПроверки>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения проверки сертификата со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

В случае ошибки генерируется исключительная ситуация и вызывается процедура обработки ошибок из [ОписаниеОповещения](ОписаниеОповещения.md).

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии.md).   
Проверяемый сертификат.

<РежимПроверки> (необязательный)

Тип: [РежимПроверкиСертификатаКриптографии](РежимПроверкиСертификатаКриптографии.md), [Массив](Массив.md).   
Указывает режим проверки.   
Содержит один объект или массив объектов режимов проверки.

Описание:

Начинает проверку сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПроверитьСертификат](МенеджерКриптографии.ПроверитьСертификат.md)  
[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПроверитьСертификатАсинх](МенеджерКриптографии.ПроверитьСертификатАсинх.md)  

Использование в версии:
