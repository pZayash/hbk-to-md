РасширениеКонфигурации (ConfigurationExtension)

ПроверитьВозможностьПрименения (CheckCanApply)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПроверитьВозможностьПрименения(<Данные>, <ПроверятьСледующие>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Проверка применимости будет учитывать, что текущее расширение конфигурации обновлено предоставленными двоичными данными.  
Значение по умолчанию: Неопределено.

<ПроверятьСледующие> (необязательный)

Тип: Булево.   
Если содержит значение Истина, будут проверены все расширения, последующие за данным в порядке загрузки расширения конфигурации.  
Значение по умолчанию: Ложь.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Возвращает массив объектов типа [ИнформацияОПроблемеПримененияРасширенияКонфигурации](ИнформацияОПроблемеПримененияРасширенияКонфигурации.md)

Описание:

Проверяет возможность применения данного расширения конфигурации. При проверке учитываются расширения, загруженные ранее.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
