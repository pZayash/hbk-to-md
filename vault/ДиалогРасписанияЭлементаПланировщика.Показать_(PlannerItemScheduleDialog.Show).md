ДиалогРасписанияЭлементаПланировщика (PlannerItemScheduleDialog)

Показать (Show)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога со следующими параметрами:

- <Расписание> - [Расписание](ДиалогРасписанияЭлементаПланировщика.Расписание_(PlannerItemScheduleDialog.Schedule).md) диалог закрыли, нажав кнопку"OK"; [Неопределено](lang__def_Undefined.md) - в противном случае;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершению никакая процедура вызвана не будет.

Описание:

Открывает диалог расписания не модально.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
