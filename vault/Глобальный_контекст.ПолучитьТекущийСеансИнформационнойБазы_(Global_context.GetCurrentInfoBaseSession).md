Глобальный контекст (Global context)

ПолучитьТекущийСеансИнформационнойБазы (GetCurrentInfoBaseSession)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьТекущийСеансИнформационнойБазы()

Возвращаемое значение:

Тип: [СеансИнформационнойБазы](СеансИнформационнойБазы_(InfoBaseSession).md).   

Описание:

Возвращает текущий сеанс информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
