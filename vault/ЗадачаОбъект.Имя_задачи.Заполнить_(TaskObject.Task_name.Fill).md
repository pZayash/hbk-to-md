# ЗадачаОбъект.<Имя задачи>.Заполнить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [Методы](ЗадачаОбъект.Имя_задачи__Методы.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение задачи.

Описание:

Заполняет данные задачи данными из другого объекта. Инициирует событие [ОбработкаЗаполнения](ЗадачаОбъект.Имя_задачи.ОбработкаЗаполнения_(TaskObject.Task_name.Filling).md) и вызов его процедуры-обработчика в модуле задачи. В ней может размещаться алгоритм, заполняющий данные задачи из переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для заполнения может использоваться значение любого типа.

Пример:

|  |
| --- |
| Объект.Заполнить(ОбъектОснование); |

Использование в версии:

Доступен, начиная с версии 8.0.
