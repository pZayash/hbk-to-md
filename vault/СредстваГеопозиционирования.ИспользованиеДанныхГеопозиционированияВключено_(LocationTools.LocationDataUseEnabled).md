СредстваГеопозиционирования (LocationTools)

ИспользованиеДанныхГеопозиционированияВключено (LocationDataUseEnabled)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ИспользованиеДанныхГеопозиционированияВключено()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если обнаружение местоположения включено в настройках устройства.  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Возвращает признак использования обнаружения месположения в настройках конкретного устройства.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
