УровеньИзоляцииТранзакций (TransactionsIsolationLevel)

ЧтениеНезафиксированных (ReadUncommitted)

Доступен, начиная с версии 8.3.5.

Описание:

Уровень изоляции транзакций внешнего источника данных, при котором операция чтения возвращает данные, не зафиксированные в других транзакциях на момент начала транзакции, операция записи на измененных в других транзакциях данных перезаписывает данные (может быть обеспечено поведение уровня [ЧтениеЗафиксированных](УровеньИзоляцииТранзакций.ЧтениеЗафиксированных.md) или выше, зависит от внешнего источника данных).

Использование в версии:

Доступен, начиная с версии 8.3.5.
