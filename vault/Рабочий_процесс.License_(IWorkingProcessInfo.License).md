Рабочий процесс (IWorkingProcessInfo)

License (License)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Лицензия](Лицензия_(ILicenseInfo).md), [Неопределено](lang__def_Undefined.md).   
Содержит информацию о серверной лицензии, используемой рабочим процессом.   
[Неопределено](lang__def_Undefined.md) - рабочий процесс не использует серверную лицензию.

Доступность:

Интеграция.

Примечание:

Рабочий процесс использует серверную лицензию только, если имеется хотя бы одно соединение клиентского приложения с информационной базой через этот рабочий процесс.

Использование в версии:

Доступен, начиная с версии 8.2.
