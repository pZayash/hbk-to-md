Расширение табличного поля списка задач (Task list table box extension)

ИзменятьСпособРедактирования (ChangeEditingMode)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает доступность интерактивного способа редактирования объекта (в строке или в отдельной форме).

Доступность:

Толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - интерактивная установка способа редактирования объекта доступна пользователю; [Ложь](lang__def_BooleanFalse.md) - не доступна.

Использование в версии:

Доступен, начиная с версии 8.0.
