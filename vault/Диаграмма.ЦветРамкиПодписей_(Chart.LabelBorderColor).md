Диаграмма (Chart)

ЦветРамкиПодписей (LabelBorderColor)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет рамки подписей к значениям. Используется, если [ПрозрачныйФонПодписей](Диаграмма.ПрозрачныйФонПодписей_(Chart.TransparentLabel).md) = [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
