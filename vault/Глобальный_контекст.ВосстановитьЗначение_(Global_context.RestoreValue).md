# Глобальный контекст.ВосстановитьЗначение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Процедуры и функции сохранения значений](Процедуры_и_функции_сохранения_значений.md)

Глобальный контекст (Global context)

ВосстановитьЗначение (RestoreValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ВосстановитьЗначение(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя сохраненного значения.

Возвращаемое значение:

Тип: Произвольный.   
Восстановленное значение. Если восстановить не удалось, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Получает значение, сохраненное ранее с помощью метода [СохранитьЗначение](Глобальный_контекст.СохранитьЗначение_(Global_context.SaveValue).md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| СрокИсполнения = ВосстановитьЗначение("СрокИсполненияЗаказа")); ДатаИсполнения = ДатаДокумента + Макс(СрокИсполнения, 5); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [СохранитьЗначение](Глобальный_контекст.СохранитьЗначение_(Global_context.SaveValue).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
