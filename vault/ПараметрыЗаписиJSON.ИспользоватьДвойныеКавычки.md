ПараметрыЗаписиJSON (JSONWriterSettings)

ИспользоватьДвойныеКавычки (UseDoubleQuotes)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Определяет использование двойных кавычек при записи свойств и значений JSON.  
После создания объекта данное свойство имеет значение Истина.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
