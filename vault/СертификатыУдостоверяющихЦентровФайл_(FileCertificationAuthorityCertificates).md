# СертификатыУдостоверяющихЦентровФайл

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Интернет](Интернет-2.md) › [Сертификаты защищенных соединений](Сертификаты_защищенных_соединений.md)

Доступен, начиная с версии 8.3.1.

Свойства:

[ИмяФайла (FileName)](СертификатыУдостоверяющихЦентровФайл.ИмяФайла_(FileCertificationAuthorityCertificates.FileName).md)  

Конструкторы:

[На основании файла](СертификатыУдостоверяющихЦентровФайл.По_умолчанию.md)  

Описание:

Объект сертификатов удостоверяющих центров OpenSSL, загружаемых из файла.  
Формат файла определяется по его расширению:  
\*.PFX; \*.P12 - файл сертификатов формата P12;  
\*.PEM - файл сертификатов формата PEM.  
по умолчанию - файл сертификатов формата PEM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.
