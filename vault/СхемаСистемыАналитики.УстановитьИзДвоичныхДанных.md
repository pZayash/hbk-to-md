СхемаСистемыАналитики (AnalyticsSystemSchema)

УстановитьИзДвоичныхДанных (SetFromBinaryData)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьИзДвоичныхДанных(<СхемаКакДвоичныеДанные>)

Параметры:

<СхемаКакДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Содержит схему системы аналитики в виде двоичных данных.

Описание:

Устанавливает схему системы аналитики по переданным двоичным данным.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
