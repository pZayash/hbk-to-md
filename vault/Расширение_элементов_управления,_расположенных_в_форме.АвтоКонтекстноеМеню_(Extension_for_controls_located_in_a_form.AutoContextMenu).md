# Расширение элементов управления, расположенных в форме.АвтоКонтекстноеМеню

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Форма](Форма.md) › [Расширение элементов управления, расположенных в форме](Расширение_элементов_управления,_расположенных_в_форме_(Extension_for_controls_located_in_a_form).md) › [Свойства](Расширение_элементов_управления,_расположенных_в_форме__Свойства.md)

Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

АвтоКонтекстноеМеню (AutoContextMenu)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного свойства разрешает использование для элемента управления автоматического контекстного меню, иначе будет использовано контекстное меню, выбранное в свойстве [КонтекстноеМеню](Расширение_элементов_управления,_расположенных_в_форме.КонтекстноеМеню_(Extension_for_controls_located_in_a_form.ContextMenu).md).

Доступность:

Толстый клиент.

Примечание:

Свойство используется только для следующих типов элементов управления: [ПолеВвода](ПолеВвода_(TextBox).md), [ПолеВыбора](ПолеВыбора_(ComboBox).md), [ПолеСписка](ПолеСписка_(ListBox).md), [ТабличноеПоле](ТабличноеПоле_(TableBox).md), [ПолеТекстовогоДокумента](ПолеТекстовогоДокумента_(TextDocumentField).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).  
  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
