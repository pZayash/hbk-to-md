ДокументDOM (DOMDocument)

СоздатьСсылкуНаСущность (CreateEntityReference)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьСсылкуНаСущность(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя сущности DOM.

Возвращаемое значение:

Тип: [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md).   

Описание:

Создает узел DOM - ссылка на сущность.  
Если сущность уже определена в документе, то в создаваемый узел будут автоматически добавлены узлы в соответствии с определением сущности.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- цель инструкции обработки содержит символы, недопустимые стандартом XML.

Использование в версии:

Доступен, начиная с версии 8.1.
