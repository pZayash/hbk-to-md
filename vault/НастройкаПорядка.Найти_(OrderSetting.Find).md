НастройкаПорядка (OrderSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого элемента.

Возвращаемое значение:

Тип: [ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента управления порядком по имени элемента порядка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
