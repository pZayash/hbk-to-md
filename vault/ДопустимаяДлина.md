# ДопустимаяДлина

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Системные_перечисления.md">Системные перечисления</a> › [Разные](Разные.md)

Доступен, начиная с версии 8.0.

Значения

[Переменная (Variable)](ДопустимаяДлина.Переменная.md)  
[Фиксированная (Fixed)](ДопустимаяДлина.Фиксированная.md)  

Описание:

Определяет варианты ограничений длины строки. Используется для определения свойства [ДопустимаяДлина](КвалификаторыСтроки.ДопустимаяДлина.md) и в конструкторе объекта [КвалификаторыСтроки](КвалификаторыСтроки.md), если задается длина строки отличная от нуля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/core}. Имя типа XDTO: AllowedLength.
