Диаграмма (Chart)

Окантовка (Outline)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак окантовки черным цветом областей на диаграмме. Применяется в диаграммах, имеющих закрашиваемые области. Вне зависимости от цвета областей (серий) окантовка наносится тонкой черной линией.  
[Истина](lang__def_BooleanTrue.md) - окантовка используется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для диаграмм с закрашиваемыми областями.

Использование в версии:

Доступен, начиная с версии 8.0.
