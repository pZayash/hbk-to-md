ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьОбновлениеСписковОтзываСертификатов (BeginUpdatingCertificateRevocationList)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НачатьОбновлениеСписковОтзываСертификатов(<ОписаниеОповещения>, <Сертификаты>, <ОбновлятьПриНаличииРасширенияСАдресомOCSP>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после обновления списков отзыва сертификатов со следующими параметрами:

- <СпискиОтзываСертификатовОбновлены> - [Булево](lang__Булево_(Boolean).md) c результатом.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Сертификаты> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md) – список сертификатов, по которым необходимо обновить списки отзыва. По умолчанию выполняется обновление всех сертификатов хранилища.

<ОбновлятьПриНаличииРасширенияСАдресомOCSP> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, требуется ли обновлять списки отзыва по сертификату, если в нем есть адрес OCSP-сервера.  
Значение по умолчанию: Ложь.

Описание:

Обновляет списки отзыва по списку сертификатов и их цепочек. Рекомендуется добавить в регламентные задания при необходимости использования улучшенной подписи. Позволяет поддерживать списки отзыва в актуальном состоянии в асинхронном режиме (не задерживая функции создания подписи).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
