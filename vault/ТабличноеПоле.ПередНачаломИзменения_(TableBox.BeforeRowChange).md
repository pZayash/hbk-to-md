ТабличноеПоле (TableBox)

ПередНачаломИзменения (BeforeRowChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломИзменения(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), интерактивное изменение строки не начнется.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед началом интерактивного изменения существующей строки табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
