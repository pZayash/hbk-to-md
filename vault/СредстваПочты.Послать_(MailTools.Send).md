СредстваПочты (MailTools)

Послать (Send)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Послать(<Сообщение>, <ОбрабатыватьТексты>)

Параметры:

<Сообщение> (обязательный)

Тип: [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md).   
Почтовое сообщение.  
Внимание! Тип текстов почтовых сообщений [РазмеченныйТекст](ТипТекстаПочтовогоСообщения.РазмеченныйТекст_(InternetMailTextType.RichText).md) не поддерживается.

<ОбрабатыватьТексты> (необязательный)

Тип: [ОбработкаТекстаИнтернетПочтовогоСообщения](ОбработкаТекстаИнтернетПочтовогоСообщения_(InternetMailTextProcessing).md).   
Управляет обработкой текстов перед отправкой:

- [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать_(InternetMailTextProcessing.Process).md) - Локальные картинки из текста, помещаются в сообщение в виде вложений. Содержимое текста письма изменяется таким образом, чтобы теги <img> указывали на картинки из вложений.
- [НеОбрабатывать](ДопустимыеДействияПеретаскивания.НеОбрабатывать_(DragAllowedActions.DontProcess).md) - Текст посылается без внутренней обработки.
- Загрузка картинок при отправке HTML-документов отключена.

  
Значение по умолчанию: [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать_(InternetMailTextProcessing.Process).md).

Описание:

Отправляет почтовое сообщение.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяОтправка](СредстваПочты.ПоддерживаетсяОтправка_(MailTools.SendingSupported).md).  
Внимание! Тип текстов почтовых сообщений [РазмеченныйТекст](ТипТекстаПочтовогоСообщения.РазмеченныйТекст_(InternetMailTextType.RichText).md) не поддерживается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваПочты](СредстваПочты_(MailTools).md), метод [ПоддерживаетсяОтправка](СредстваПочты.ПоддерживаетсяОтправка_(MailTools.SendingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
