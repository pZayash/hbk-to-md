СообщениеВнешнегоСайта (MessageFromExternalSite)

Источник (Origin)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит строку, соответствующую основному адресу внешнего сайта (в формате "<протокол>://<домен>:<порт>"), от которого было получено сообщение.  
Например: https://example.org:8080/example.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
