# ЧтениеXML.ПерейтиКСодержимому

ЧтениеXML (XMLReader)

ПерейтиКСодержимому (MoveToContent)

Доступен, начиная с версии 8.2.

Синтаксис:

ПерейтиКСодержимому(<ПробельныеСимволы>)

Параметры:

<ПробельныеСимволы> (необязательный)

Тип: [ПробельныеСимволыXML](ПробельныеСимволыXML_(XMLSpace).md).   
Указывает способ проверки пробельных символов.  
Значение по умолчанию: [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию_(XMLSpace.Default).md).

Возвращаемое значение:

Тип: [ТипУзлаXML](ТипУзлаXML_(XMLNodeType).md).   
Значение узла, в который был перемещен текущий узел; [Ничего](ТипУзлаXML.Ничего_(XMLNodeType.None).md) - в случае достижения конца источника данных XML.

Описание:

Проверяет является ли текущий узел "содержимым" ([Текст](ТипУзлаXML.Текст_(XMLNodeType.Text).md), [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента_(XMLNodeType.StartElement).md), [КонецЭлемента](ТипУзлаXML.КонецЭлемента_(XMLNodeType.EndElement).md), [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность_(XMLNodeType.EntityReference).md), [КонецСущности](ТипУзлаXML.КонецСущности_(XMLNodeType.EndEntity).md) или [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы_(XMLNodeType.Whitespace).md)) в случае значения параметра равному [Сохранять](ПробельныеСимволыXML.Сохранять_(XMLSpace.Preserve).md) или если текущее значения свойства [ПробельныеСимволы](ЧтениеXML.ПробельныеСимволы_(XMLReader.Space).md) равно [Сохранять](ПробельныеСимволыXML.Сохранять_(XMLSpace.Preserve).md)).  
Если текущий узел не является "содержимым", то выполняется пропуск узлов до "содержимого" или же до конца источника данных XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Будут пропущены следующие типы узлов: [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML_(XMLNodeType.XMLDeclaration).md), [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки_(XMLNodeType.ProcessingInstruction).md), [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md), [Комментарий](ТипУзлаXML.Комментарий_(XMLNodeType.Comment).md), а также [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы_(XMLNodeType.Whitespace).md), в случае значения параметра равному [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию_(XMLSpace.Default).md) и если текущее значения свойства [ПробельныеСимволы](ЧтениеXML.ПробельныеСимволы_(XMLReader.Space).md) равно [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию_(XMLSpace.Default).md).

Использование в версии:

Доступен, начиная с версии 8.2.
