Очистить (Clear)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Очистить()

Описание:

Удаляет все элементы из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
