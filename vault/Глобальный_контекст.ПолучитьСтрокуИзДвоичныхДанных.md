Глобальный контекст (Global context)

ПолучитьСтрокуИзДвоичныхДанных (GetStringFromBinaryData)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьСтрокуИзДвоичныхДанных(<ДвоичныеДанные>, <КодировкаТекста>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные, которые требуется преобразовать в строку.

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

.  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8.md).

Возвращаемое значение:

Тип: Строка.   

Описание:

Преобразует двоичные данные в строку с заданной кодировкой текста.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
