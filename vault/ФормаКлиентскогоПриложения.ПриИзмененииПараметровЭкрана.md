ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриИзмененииПараметровЭкрана (OnChangeDisplaySettings)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПриИзмененииПараметровЭкрана()

Описание:

Возникает при изменении информации, собранной в объекте [ИнформацияЭкранаКлиента](ИнформацияЭкранаКлиента.md): разрешение, ориентация и т.п.  
На мобильной платформе событие возникает только в активной форме. Если форма не активна, то обработчик события сработает после активизации формы, если изменения еще актуальны (например, устройство можно повернуть обратно, и тогда изменение ориентации станет неактуальным).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
