Соединение с агентом сервера (IServerAgentConnection)

GetConnectionLocks (GetConnectionLocks)

Доступен, начиная с версии 8.1.

Синтаксис:

GetConnectionLocks(<Кластер>, <Соединение>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, из которого должны быть получены блокировки. Массив кластеров может быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md).

<Соединение> (обязательный)

Тип: [Описание соединения](Описание_соединения.md).   
Описание соединения, которое установило получаемые блокировки. Массив описаний соединений может быть получен методом [GetConnections](Соединение_с_агентом_сервера.GetConnections.md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив блокировок объектов, установленных клиентами кластера. Каждая блокировка представлена объектом с интерфейсом [Блокировка](Блокировка.md).

Описание:

Получает массив блокировок объектов соединения.  
Блокировки могут быть трех видов:

- блокировки информационной базы устанавливаются при запуске клиентских приложений, внешних соединений или других клиентов кластера;
- блокировки объектов устанавливаются при начале интерактивного редактирования объектов 1С:Предприятия или при вызове методов Заблокировать.
- блокировки номеров устанавливаются 1С:Предприятием при автоматическом получении нового номера (кода).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
