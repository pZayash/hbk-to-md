Глобальный контекст (Global context)

ПоказатьИнформациюОбОшибке (ShowErrorInfo)

Не рекомендуется использовать, начиная с версии 8.3.17.

Рекомендуется использовать:

- [ПоказатьИнформациюОбОшибке](МенеджерОбработкиОшибок.ПоказатьИнформациюОбОшибке.md)

Доступен, начиная с версии 8.1.

Синтаксис:

ПоказатьИнформациюОбОшибке(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Информация об ошибке, которую надо показать пользователю.

Описание:

Предназначен для показа информации об ошибке пользователю в стандартном диалоге платформы "1С:Предприятие".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Попытка     ... Исключение     Инфо = ИнформацияОбОшибке();     ПоказатьИнформациюОбОшибке(Инфо); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.17.

Описание изменено в версии 8.3.17.
