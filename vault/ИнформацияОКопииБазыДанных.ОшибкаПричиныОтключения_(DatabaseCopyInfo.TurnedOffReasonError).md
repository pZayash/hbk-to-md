ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ОшибкаПричиныОтключения (TurnedOffReasonError)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md), [Неопределено](lang__def_Undefined.md).   
При отсутствии ошибок свойство содержит значение [Неопределено](lang__def_Undefined.md). При возникновении ошибки объект, описывающий ошибку, которая привела к недоступности копии для выполнения запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
