# ПланВидовРасчетаОбъект.<Имя плана видов расчета>.Заполнить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [ПланВидовРасчетаОбъект.<Имя плана видов расчета>](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md) › [Методы](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета__Методы.md)

ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Объект, на основании которого производится заполнение.

Описание:

Заполняет данные объекта вид расчета данными из другого объекта (вида расчета или иного объекта базы данных). Метод инициирует событие [ОбработкаЗаполнения](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ОбработкаЗаполнения_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Filling).md) и вызов соответствующей процедуры-обработчика в модуле объекта. В процедуре может размещаться алгоритм, заполняющий данные объекта на основании переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
