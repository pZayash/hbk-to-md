Расширение формы клиентского приложения для документа (Client application form extension for documents)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура.md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Состав параметров произвольный. Содержит предопределенные параметры записи РежимЗаписи и РежимПроведения. Предопределенные параметры передаются в метод объекта [Записать](ДокументОбъект.Имя_документа.Записать.md).  
Параметры записи можно обработать в событиях формы [ПередЗаписью](Расширение_формы_клиентского_приложения_для_документа.ПередЗаписью.md), [ПередЗаписьюНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПередЗаписьюНаСервере.md), [ПриЗаписиНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПриЗаписиНаСервере.md), [ПослеЗаписиНаСервере](Расширение_формы_клиентского_приложения_для_документа.ПослеЗаписиНаСервере.md), [ПослеЗаписи](Расширение_формы_клиентского_приложения_для_документа.ПослеЗаписи.md).

Возвращаемое значение:

Тип: Булево.   
Истина - успешно записан; Ложь - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
