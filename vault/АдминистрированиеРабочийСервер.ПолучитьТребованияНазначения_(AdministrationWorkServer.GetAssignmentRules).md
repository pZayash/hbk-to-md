АдминистрированиеРабочийСервер (AdministrationWorkServer)

ПолучитьТребованияНазначения (GetAssignmentRules)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьТребованияНазначения()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеТребованиеНазначения](АдминистрированиеТребованиеНазначения_(AdministrationAssignmentRule).md).

Описание:

Получает массив требований назначения сервера. Требования в массиве располагаются в том же порядке, в котором они будут подбираться при назначении новых сервисов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
