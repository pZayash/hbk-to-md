УровеньИспользованияЗащищенногоСоединенияFTP (FTPSecureConnectionUsageLevel)

Требовать (Require)

Доступен, начиная с версии 8.3.9.

Описание:

Для передачи управляющих команд и данных требуется использование защищенного SSL-соединения.

Использование в версии:

Доступен, начиная с версии 8.3.9.
