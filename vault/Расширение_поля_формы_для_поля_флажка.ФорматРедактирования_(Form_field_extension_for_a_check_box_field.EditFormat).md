Расширение поля формы для поля флажка (Form field extension for a check box field)

ФорматРедактирования (EditFormat)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Используется для формирования текстового представления значений типа [Булево](lang__Булево_(Boolean).md) при виде флажка [Тумблер](ВидФлажка.Тумблер_(CheckBoxType.Tumbler).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
