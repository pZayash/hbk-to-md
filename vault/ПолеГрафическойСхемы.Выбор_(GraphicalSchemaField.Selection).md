Доступен, начиная с версии 8.0.

Синтаксис:

Выбор()

Возникает при двойном щелчке мыши (нажатии клавиши Enter) на элементе графической схемы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
