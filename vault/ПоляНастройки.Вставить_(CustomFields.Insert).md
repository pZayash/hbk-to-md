ПоляНастройки (CustomFields)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Имя>, <Представление>, <Тип>, <СписокДопустимыхЗначений>, <Позиция>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

<Представление> (обязательный)

Тип: [Строка](lang__def_String.md).   
Представление поля.  
Если не задано, то выбирается из параметра <Имя>.

<Тип> (необязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Тип поля.  
Значение по умолчанию: Произвольный.

<СписокДопустимыхЗначений> (необязательный)

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Список допустимых значений для поля.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить поле.

Возвращаемое значение:

Тип: [ПолеНастройки](ПолеНастройки_(CustomField).md).   

Описание:

Вставляет поле в указанную позицию коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
