ЭкранированиеСимволовJSON (JSONCharactersEscapeMode)

СимволыВнеBMP (SymbolsNotInBMP)

Доступен, начиная с версии 8.3.6.

Описание:

Экранировать символы, которые в UTF-16 будут представлены "суррогатной парой" (двумя символами UTF-16).

Использование в версии:

Доступен, начиная с версии 8.3.6.
