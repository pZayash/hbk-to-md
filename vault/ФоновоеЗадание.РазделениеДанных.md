ФоновоеЗадание (BackgroundJob)

РазделениеДанных (DataSeparation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Структура](Структура.md).   
Содержит значения разделителей. Имя элемента структуры содержит имя общего реквизита; значение – значение общего реквизита.  
Элементами структуры являются значения всех установленных на момент запуска фонового задания разделителей с типом [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
