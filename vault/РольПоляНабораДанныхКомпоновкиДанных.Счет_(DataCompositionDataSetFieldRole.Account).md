РольПоляНабораДанныхКомпоновкиДанных (DataCompositionDataSetFieldRole)

Счет (Account)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что поле является полем счета. Данная информация используется при расчете по полям бухгалтерского остатка.  
[Истина](lang__def_BooleanTrue.md) - поле является полем счета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
