БлокировкаСеансов (SessionsLock)

Начало (Begin)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату и время начала актуальности блокировки. Блокировка начинает действовать, если текущее время превышает значение данного свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
