КомпоновщикНастроекКомпоновкиДанных (DataCompositionSettingsComposer)

ПолучитьИсточникДоступныхНастроек (GetAvailableSettingsSource)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьИсточникДоступныхНастроек()

Возвращаемое значение:

Тип: [ИсточникДоступныхНастроекКомпоновкиДанных](ИсточникДоступныхНастроекКомпоновкиДанных.md).   
Неопределено, если источник доступных настроек не установлен.

Описание:

Возвращает источник доступных настроек компоновщика настроек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
