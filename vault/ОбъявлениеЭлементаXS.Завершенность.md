ОбъявлениеЭлементаXS (XSElementDeclaration)

Завершенность (Final)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ОбъединениеИсключенийГруппПодстановкиXS](ОбъединениеИсключенийГруппПодстановкиXS.md).   
Определяет значение исключения вхождения в группу подстановки.   
Если значение свойства не задано, то элемент может быть рассмотрен в качестве кандидатуры на включение в группу подстановки другого объявления элемента, имеющего одинаковое определение типа или типов, производных от данного. В противном случае значение исключения указывает на тип наследования, по которому вхождение в группу подстановки запрещается. Если выбраны все виды наследования, то элемент не может входить в группу подстановки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
