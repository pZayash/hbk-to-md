ПолитикаПаролейПользователей (UserPasswordPolicy)

МинимальнаяДлинаПаролей (PasswordMinLength)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Численное значение минимальной длины паролей пользователей.  
  
Значение минимальной длины должно быть в диапазоне 0…99. Если включена проверка сложности паролей, то значение должно быть в интервале 7…99.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.22.
