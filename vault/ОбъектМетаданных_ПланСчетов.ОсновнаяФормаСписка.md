ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

ОсновнаяФормаСписка (DefaultListForm)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Форма](ОбъектМетаданных_Форма.md) , Неопределено.   
Описывает форму, которая будет вызвана системой для выполнения стандартных действий со списком плана счетов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и система генерирует форму по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.1.
