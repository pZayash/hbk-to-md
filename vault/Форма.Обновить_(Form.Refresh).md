Форма (Form)

Обновить (Refresh)

Доступен, начиная с версии 8.0.

Синтаксис:

Обновить()

Описание:

Обновляет данные, представленные в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Обновить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
