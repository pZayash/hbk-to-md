ДиаграммаГанта (GanttChart)

НайтиИнтервалПоИдентификатору (FindIntervalByID)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиИнтервалПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторИнтервалаДиаграммыГанта](ИдентификаторИнтервалаДиаграммыГанта_(GanttChartIntervalID).md).   
Идентификатор искомого интервала.

Возвращаемое значение:

Тип: [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md).   
[Неопределено](lang__def_Undefined.md) - если интервал таким идентификатором не найден.

Описание:

Осуществляет поиск интервала в диаграмме по идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
