СтандартнаяКомандаПланировщика (PlannerStandardCommand)

ВыполнитьДействие (ExecuteAction)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ВыполнитьДействие](СтандартнаяКомандаПоляПланировщика.ВыполнитьДействие_(PlannerStandardCommand.ExecuteAction).md)

Доступен, начиная с версии 8.3.18.

Описание:

Переход к событию [НажатиеНаДействиеПланировщика](Расширение_поля_формы_для_поля_планировщика.НажатиеНаДействиеПланировщика_(Form_field_extension_for_a_planner.PlannerActionClick).md).  
Используется только для источника вида [Действие](ИсточникКомандПланировщика.Действие_(PlannerCommandSource.Action).md).  
В контекстном меню команда будет иметь представление "Выполнить действие".

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
