ЭлементыФормы (Controls)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции.

Возвращаемое значение:

Элемент формы.

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Толстый клиент.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
