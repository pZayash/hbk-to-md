ИнтернетПочтовыйПрофиль (InternetMailProfile)

АутентификацияПоТокену (TokenAuthentication)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеАутентификацииИнтернетПочтыПоТокену](ИспользованиеАутентификацииИнтернетПочтыПоТокену_(UseInternetMailTokenAuthentication).md).   
Содержит признак выполнения аутентификации по протоколу OAuth2 при подключении по протоколу IMAP.  
Значение по умолчанию - [Авто](ИспользованиеАутентификацииИнтернетПочтыПоТокену.Авто_(UseInternetMailTokenAuthentication.Auto).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
