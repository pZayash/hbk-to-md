[ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.  
Возможно обращение к элементам посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).
