СредстваГеопозиционирования (LocationTools)

ОтключитьОбработчикПересеченияГраницОтслеживаемыхГеозон (DetachMonitoredGeofencesBordersCrossingDetectionHandler)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОтключитьОбработчикПересеченияГраницОтслеживаемыхГеозон(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md), Строка.   
Описание процедуры, которая будет вызвана после завершения операции.  
Процедура должна содержать следующие параметры:

- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Отключает обработчик пересечения границ геозон.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.22.
