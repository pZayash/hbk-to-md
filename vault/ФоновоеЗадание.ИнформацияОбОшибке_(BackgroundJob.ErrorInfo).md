ФоновоеЗадание (BackgroundJob)

ИнформацияОбОшибке (ErrorInfo)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md), [Неопределено](lang__def_Undefined.md).   
Информация об ошибке, если задание было завершено аварийно.  
[Неопределено](lang__def_Undefined.md), если задание было завершено успешно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
