Получить(<Ключ>)

<Ключ> (обязательный)

Тип: Тип ключа строки (ссылка для ссылочных объектов, ключ записи для регистров, число при отсутствии ключа выборки).   
Ключ, для которого требуется получить строку.

Получает строку по ключу. Работает аналогично оператору [].

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
