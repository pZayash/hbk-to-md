ФиксированныйМассив (FixedArray)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Искомое значение.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Если элемент найден, возвращается его порядковый номер. Если элемент не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Выполняет поиск элемента в массиве.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод эффективно использовать для поиска уникальных значений.

Использование в версии:

Доступен, начиная с версии 8.1.
