ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ИзмерениеРегистра (RegisterDimension)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: РегистрРасчета](ОбъектМетаданных_РегистрРасчета.md).   
Объект метаданных, описывающий измерение регистра расчета. На это измерение будет установлен отбор при анализе существования зависимых записей, которые необходимо перерасчитать.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется для перерасчета.

Использование в версии:

Доступен, начиная с версии 8.1.
