МенеджерОтображенияРекламы (AdRepresentationManager)

НачатьЗагрузкуВидеообъявленияСВознаграждением (BeginRewardedVideoDownloading)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьЗагрузкуВидеообъявленияСВознаграждением(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Указывает обработчик, вызываемый при завершении загрузки регламы из сети Интернет.  
Процедура должна иметь следующие параметры:

- Статус (Status) – тип [СтатусРекламы](СтатусРекламы_(AdStatus).md). Показывает статус рекламного блока по результатам загрузки.
- Идентификатор (ID) – тип [Строка](lang__def_String.md). Рекламный идентификатор для которого закончилась загрузка.
- <ДополнительныеПараметры> – Значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Загружает из сети видеообъявление. По завершению загрузки вызывается обработчик.   
После загрузки, при вызове метода [ПоказатьВидеообъявлениеСВознаграждением](МенеджерОтображенияРекламы.ПоказатьВидеообъявлениеСВознаграждением_(AdRepresentationManager.ShowRewardedVideo).md) видеообъявление будет показано без задержки на загрузку.   
Вызов метода не обязателен для показа рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МенеджерОтображенияРекламы](МенеджерОтображенияРекламы_(AdRepresentationManager).md), метод [ЗагрузитьВидеообъявлениеСВознаграждениемАсинх](МенеджерОтображенияРекламы.ЗагрузитьВидеообъявлениеСВознаграждениемАсинх_(AdRepresentationManager.DownloadRewardedVideoAsync).md)
