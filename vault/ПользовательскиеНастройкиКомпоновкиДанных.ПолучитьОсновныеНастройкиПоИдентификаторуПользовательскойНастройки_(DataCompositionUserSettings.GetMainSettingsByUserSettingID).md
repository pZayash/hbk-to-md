# ПользовательскиеНастройкиКомпоновкиДанных.ПолучитьОсновныеНастройкиПоИдентификаторуПользовательскойНастройки

ПользовательскиеНастройкиКомпоновкиДанных (DataCompositionUserSettings)

ПолучитьОсновныеНастройкиПоИдентификаторуПользовательскойНастройки (GetMainSettingsByUserSettingID)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьОсновныеНастройкиПоИдентификаторуПользовательскойНастройки(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор пользовательской настройки.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Возвращаемый массив может содержать элементы следующих типов:

- [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных_(DataCompositionFilter).md),
- [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных_(DataCompositionFilterItem).md),
- [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных_(DataCompositionFilterItemGroup).md),
- [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных_(DataCompositionOrder).md),
- [ВыбранныеПоляКомпоновкиДанных](ВыбранныеПоляКомпоновкиДанных_(DataCompositionSelectedFields).md),
- [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных_(DataCompositionConditionalAppearance).md),
- [ЭлементУсловногоОформленияКомпоновкиДанных](ЭлементУсловногоОформленияКомпоновкиДанных_(DataCompositionConditionalAppearanceItem).md),
- [ЗначениеПараметраНастроекКомпоновкиДанных](ЗначениеПараметраНастроекКомпоновкиДанных_(DataCompositionSettingsParameterValue).md),
- [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных_(DataCompositionGroup).md),
- [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных_(DataCompositionTableGroup).md),
- [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных_(DataCompositionChartGroup).md),
- [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных_(DataCompositionTable).md),
- [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных_(DataCompositionChart).md),
- [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных_(DataCompositionNestedObjectSettings).md),
- [КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных](КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных_(DataCompositionSettingStructureItemCollection).md),

Если объекты не найдены, метод возвращает пустой массив.

Описание:

Получает массив объектов основных настроек по идентификатору пользовательской настройки. Эти объекты можно использовать для доступа к свойствам, значения которых не установлены у объектов пользовательских настроек, поскольку они недоступны для редактирования пользователю. Например, свойство [ЛевоеЗначение](ЭлементОтбораКомпоновкиДанных.ЛевоеЗначение_(DataCompositionFilterItem.LeftValue).md) объекта [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных_(DataCompositionFilterItem).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если идентификатор пользовательской настройки объекта имеет тип [СтруктураНастроекКомпоновкиДанных](СтруктураНастроекКомпоновкиДанных_(DataCompositionSettingStructure).md) из состава пользовательских настроек, то возвращаемый массив будет содержать элемент полной коллекции настроек типа [КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных](КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных_(DataCompositionSettingStructureItemCollection).md).

Использование в версии:

Доступен, начиная с версии 8.3.13.
