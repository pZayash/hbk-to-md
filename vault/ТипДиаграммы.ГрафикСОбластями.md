ТипДиаграммы (ChartType)

ГрафикСОбластями (Area)

Доступен, начиная с версии 8.0.

Описание:

Линейный график с закрашиванием области между линией графика и линией базового значения. Разные серии перекрывают друг друга. Строится на основании данных, аналогичных данным диаграммы вида [График](ТипДиаграммы.График.md).

Использование в версии:

Доступен, начиная с версии 8.0.
