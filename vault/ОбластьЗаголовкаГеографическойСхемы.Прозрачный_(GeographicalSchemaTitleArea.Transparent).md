ОбластьЗаголовкаГеографическойСхемы (GeographicalSchemaTitleArea)

Прозрачный (Transparent)

Не рекомендуется использовать, начиная с версии 8.2.

Рекомендуется использовать:

- [ПрозрачныйФон](ОбластьЗаголовкаГеографическойСхемы.ПрозрачныйФон_(GeographicalSchemaTitleArea.Transparent).md)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет прозрачность фона области заголовка географической схемы.  
[Истина](lang__def_BooleanTrue.md) - фон прозрачный.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.2.

Описание изменено в версии 8.2.
