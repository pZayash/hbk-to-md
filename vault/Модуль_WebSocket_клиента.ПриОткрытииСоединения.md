Модуль WebSocket клиента (WebSocket client module)

ПриОткрытииСоединения (OnOpenConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПриОткрытииСоединения(<Соединение>)

Параметры:

<Соединение>

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md).   
Ссылка на объект соединения.

Описание:

Обработчик открытия соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
