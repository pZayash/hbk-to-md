ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

ЗапрещатьНезаполненныеЗначения (DenyIncompleteValues)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак необходимости проверки заполнения значения параметра при выполнении отчета.   
Данное свойство заполняется автоматически на основании свойства [ЗапрещатьНезаполненныеЗначения](ПараметрСхемыКомпоновкиДанных.ЗапрещатьНезаполненныеЗначения.md) параметра схемы компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
