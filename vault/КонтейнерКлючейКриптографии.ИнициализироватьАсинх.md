КонтейнерКлючейКриптографии (CryptoKeysContainer)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (необязательный)

Тип: Строка.   
Имя файла, из данных которого будет инициализирован контейнер ключей криптографии.

<Пароль> (необязательный)

Тип: Строка.   
Пароль контейнера ключей криптографии.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - созданный [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии.md).

Описание:

Асинхронная функция. Инициализирует контейнер.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии.md), метод [НачатьИнициализацию](КонтейнерКлючейКриптографии.НачатьИнициализацию.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
