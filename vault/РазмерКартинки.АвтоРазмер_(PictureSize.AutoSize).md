РазмерКартинки (PictureSize)

АвтоРазмер (AutoSize)

Доступен, начиная с версии 8.0.

Описание:

Если область, в которой отображается картинка, больше чем размеры картинки, используется режим [РеальныйРазмер](РазмерКартинки.РеальныйРазмер_(PictureSize.RealSize).md), в противном случае используется режим [Пропорционально](РазмерКартинки.Пропорционально_(PictureSize.Proportionally).md).

Использование в версии:

Доступен, начиная с версии 8.0.
