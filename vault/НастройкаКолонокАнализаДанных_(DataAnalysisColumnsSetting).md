# НастройкаКолонокАнализаДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияКолонкамиАнализаДанных](ЭлементУправленияКолонкамиАнализаДанных_(DataAnalysisColumnsControl).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки колонок анализа данных.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Доступность:
