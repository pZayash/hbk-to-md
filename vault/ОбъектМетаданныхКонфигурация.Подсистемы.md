ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Подсистемы (Subsystems)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Коллекция объектов метаданных, которые описывают все подсистемы, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Подсистема](ОбъектМетаданных_Подсистема.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При конвертации из версии 8.1 свойство заполняется на основе информации о принадлежности объектов метаданных подсистемам.

Использование в версии:

Доступен, начиная с версии 8.1.
