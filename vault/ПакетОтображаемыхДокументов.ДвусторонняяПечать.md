ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ДвусторонняяПечать (DuplexPrinting)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ТипДвустороннейПечати](ТипДвустороннейПечати.md), Неопределено.   
Указывает необходимость использования двусторонней печати. Неопределено - значение определяется родительским пакетом. Если родительского пакета нет, то считается значением [Нет](ТипДвустороннейПечати.Нет.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
