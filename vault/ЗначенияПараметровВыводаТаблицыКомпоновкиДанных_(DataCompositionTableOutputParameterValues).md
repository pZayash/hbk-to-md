Описание:

Значения параметров вывода для таблиц.  
Содержит значения следующих параметров:  
1. РасположениеИтогов (TotalsPlacement) - расположение итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных_(DataCompositionTotalPlacement).md));  
2. РасположениеПолейГруппировки (GroupFieldsPlacement) - расположение полей группировки (тип [РасположениеПолейГруппировкиКомпоновкиДанных](РасположениеПолейГруппировкиКомпоновкиДанных_(DataCompositionGroupFieldsPlacement).md));  
3. РасположениеГруппировки (GroupPlacement) - расположение группировки (тип [РасположениеГруппировкиКомпоновкиДанных](РасположениеГруппировкиКомпоновкиДанных_(DataCompositionGroupPlacement).md));  
4. РасположениеРеквизитов (AttributePlacement) - расположение реквизитов (тип [РасположениеРеквизитовКомпоновкиДанных](РасположениеРеквизитовКомпоновкиДанных_(DataCompositionAttributesPlacement).md));  
5. РасположениеРесурсов (ResourcePlacement) - расположение ресурсов (тип [РасположениеРесурсовКомпоновкиДанных](РасположениеРесурсовКомпоновкиДанных_(DataCompositionResourcesPlacement).md));  
6. ГоризонтальноеРасположениеОбщихИтогов (HorizontalOverallPlacement) - горизонтальное расположение общих итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных_(DataCompositionTotalPlacement).md));  
7. ВертикальноеРасположениеОбщихИтогов (VerticalOverallPlacement) - вертикальное расположение общих итогов (тип [РасположениеИтоговКомпоновкиДанных](РасположениеИтоговКомпоновкиДанных_(DataCompositionTotalPlacement).md));  
8. ТипЗаголовкаПолей (FieldsTitleType) - тип заголовка полей ([ТипЗаголовкаПолейКомпоновкиДанных](ТипЗаголовкаПолейКомпоновкиДанных_(DataCompositionFieldsTitleType).md));  
9. ВыводЗаголовка (TitleOutput) - вывод заголовка (тип [ТипВыводаТекстаКомпоновкиДанных](ТипВыводаТекстаКомпоновкиДанных_(DataCompositionTextOutputType).md));  
10. Заголовок (Title) - заголовок (тип [Строка](lang__def_String.md));  
11. МакетОформления (AppearanceTemplate) - имя макета оформления (тип [Строка](lang__def_String.md));  
12. АвтоПозицияРесурсов (ResourcesAutoPosition) - авто позиция ресурсов (тип [АвтоПозицияРесурсовКомпоновкиДанных](АвтоПозицияРесурсовКомпоновкиДанных_(DataCompositionResourcesAutoPosition).md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/settings}. Имя типа XDTO: TableOutputParameterValues.
