# МенеджерМетокNDEF.ВключитьАктивноеСканирование

МенеджерМетокNDEF (TagManagerNDEF)

ВключитьАктивноеСканирование (StartActiveScanning)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВключитьАктивноеСканирование(<Заголовок>, <ОбработчикСканирования>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка, который будет показан в окне сканирования меток.

<ОбработчикСканирования> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание асинхронной функции, которая будет вызвана при перехвате метки или завершении сессии сканирования со следующими параметрами:

- <Метка> - [МеткаNDEF](МеткаNDEF_(TagNDEF).md) или [Неопределено](lang__def_Undefined.md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Возвращаемое значение: Нет.  
Если была отсканирована метка, то она передается в обработчик.  
После выполнения обработчика сессия сканирования будет завершена и интерфейс сканирования будет скрыт. Поскольку в обработчике потребуется обращение к другим асинхронным функциям, сам обработчик должен быть объявлен как асинхронная функция.  
Если сессия завершена до обнаружения метки, то в обработчик будет передано значение [Неопределено](lang__def_Undefined.md).

Описание:

При вызове этого метода начинается сессия активного сканирования и пользователю показывается интерфейс сканирования. В каждый момент времени может существовать только одна сессия сканирования. Если метод вызван до завершения другой сессии сканирования NDEF-меток, то старая сессия завершается и начинается новая. Пользователь может завершить сессию, нажав кнопку "Отмена" в интерфейсе сканирования.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
