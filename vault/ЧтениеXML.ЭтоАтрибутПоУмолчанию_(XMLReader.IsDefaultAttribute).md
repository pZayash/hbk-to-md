ЧтениеXML (XMLReader)

ЭтоАтрибутПоУмолчанию (IsDefaultAttribute)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак включения значения текущего узла как значения по умолчанию, полученного из DTD или XMLSchema.  
Имеет смысл только для узла [Атрибут](ТипУзлаXML.Атрибут_(XMLNodeType.Attribute).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - значение атрибута было получено подстановкой значения по умолчанию из DTD или XMLSchema, [Ложь](lang__def_BooleanFalse.md) - значение было указано явно.

Использование в версии:

Доступен, начиная с версии 8.1.
