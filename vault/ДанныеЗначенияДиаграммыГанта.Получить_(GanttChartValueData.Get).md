Получить (Get)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
