ПолеВвода (TextBox)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, нужно ли формировать автоматически состояние свойства [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного_(TextBox.MarkIncomplete).md) в зависимости от устанавливаемого или формируемого в поле ввода значения. Если [Истина](lang__def_BooleanTrue.md), тогда при установке и формировании значения в поле ввода будет меняться состояние свойства [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного_(TextBox.MarkIncomplete).md): если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного_(TextBox.MarkIncomplete).md) будет выставлено в [Истина](lang__def_BooleanTrue.md), иначе - в [Ложь](lang__def_BooleanFalse.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
