ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтключитьОтметкуНезаполненного (ClearMarkIncomplete)

Доступен, начиная с версии 8.2.

Синтаксис:

ОтключитьОтметкуНезаполненного()

Описание:

Снимает свойство ОтметкаНезаполненного у всех полей, у которых свойство АвтоОтметкаНезаполненного = [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
