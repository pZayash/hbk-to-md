Расширение поля формы для поля календаря (Form field extension for a calendar field)

ОтображатьТекущуюДату (ShowCurrentDate)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак отображения строки с текущей датой.   
Истина - текущая дата отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
