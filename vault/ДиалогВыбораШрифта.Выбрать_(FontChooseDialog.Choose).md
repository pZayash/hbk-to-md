# ДиалогВыбораШрифта.Выбрать

ДиалогВыбораШрифта (FontChooseDialog)

Выбрать (Choose)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбрать()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - шрифт выбран; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает диалог выбора шрифта.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [Показать](ДиалогВыбораШрифта.Показать_(FontChooseDialog.Show).md).

См. также:

[ДиалогВыбораШрифта](ДиалогВыбораШрифта_(FontChooseDialog).md), метод [ВыбратьАсинх](ДиалогВыбораШрифта.ВыбратьАсинх_(FontChooseDialog.ChooseAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
