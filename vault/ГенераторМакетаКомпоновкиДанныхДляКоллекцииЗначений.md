Доступен, начиная с версии 8.3.5.

Описание:

Используется для генерации макета для вывода в коллекцию значений. Данный объект не может быть создан. Он может быть только передан в качестве параметра метода [Выполнить](КомпоновщикМакетаКомпоновкиДанных.Выполнить.md).

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
