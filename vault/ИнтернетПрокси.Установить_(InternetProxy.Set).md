ИнтернетПрокси (InternetProxy)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Протокол>, <Сервер>, <Порт>, <Пользователь>, <Пароль>, <ИспользоватьАутентификациюОС>)

Параметры:

<Протокол> (обязательный)

Тип: [Строка](lang__def_String.md).   
Протокол, для которого устанавливаются параметры прокси-сервера.

<Сервер> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя сервера.

<Порт> (необязательный)

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Порт прокси-сервера. Если в качестве протокола указано [Неопределено](lang__def_Undefined.md), то все обращения к прокси-серверу для каждого протокола будут идти через порт по умолчанию для этого протокола.  
Значение по умолчанию: Порт по умолчанию для указанного протокола.

<Пользователь> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя для аутентификации на прокси-сервере.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для аутентификации на прокси-сервере.

<ИспользоватьАутентификациюОС> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Включает использование NTLM-аутентификации на прокси.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Устанавливает параметры прокси-серверов для различных протоколов.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
