Описание:

Объект этого типа возвращается методами [Выбрать](РегистрСведенийМенеджер.Имя_регистра_сведений.Выбрать.md) и [ВыбратьПоРегистратору](РегистрСведенийМенеджер.Имя_регистра_сведений.ВыбратьПоРегистратору.md) у объекта типа [РегистрСведенийМенеджер.](РегистрСведенийМенеджер.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийМенеджер.Имя_регистра_сведений.md)> и представляет собой специализированный способ перебора записей регистра сведений.  
Обход записей выполняется системой динамически. Это означает, что использование выборки не считывает все записи сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большое количество записей и не загружать в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
