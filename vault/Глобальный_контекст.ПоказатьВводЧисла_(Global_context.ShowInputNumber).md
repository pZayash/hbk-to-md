Глобальный контекст (Global context)

ПоказатьВводЧисла (ShowInputNumber)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьВводЧисла(<ОписаниеОповещенияОЗавершении>, <Число>, <Подсказка>, <Длина>, <Точность>)

Параметры:

<ОписаниеОповещенияОЗавершении> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия окна ввода числа со следующими параметрами:

- <Число> - введенное значение числа или [Неопределено](lang__def_Undefined.md), если пользователь отказался от ввода,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Значение числа, которое будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода числа. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<Длина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Длина вводимого числа включая дробную часть (символы разделителей не учитываются).  
Значение по умолчанию: 0.

<Точность> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество знаков в дробной части вводимого числа.  
Значение по умолчанию: 0.

Описание:

Вызывает диалог для ввода строки, при этом не ожидает завершения ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Оповещение = Новый ОписаниеОповещения("ПослеВводаКоличества", ЭтотОбъект); ПоказатьВводЧисла(Оповещение, 1, "Введите количество", 10, 2); //...   &НаКлиенте Процедура ПослеВводаКоличества(Число, Параметры) Экспорт     Если НЕ Число = Неопределено Тогда         // обработка введенного количества      КонецЕсли; КонецПроцедуры; |

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.22.
