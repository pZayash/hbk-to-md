# ДокументОбъект.<Имя документа>.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](ДокументОбъект.Имя_документа.ОбработкаПроверкиЗаполнения_(DocumentObject.Document_name.FillCheckProcessing).md).
