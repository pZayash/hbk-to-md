ЗаписьXML (XMLWriter)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: По потоку и типу кодировки

Синтаксис:

ОткрытьПоток(<Поток>, <ТипКодировки>, <ДобавлятьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, в который будет записан текст XML.

<ТипКодировки> (необязательный)

Тип: Строка.   
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае поток будет записан в соответствующей кодировке. При этом в объявлении XML-документа (если таковое будет записано) будет присутствовать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то для записи потока будет использована кодировка [UTF8](КодировкаТекста.UTF8.md), а атрибут 'encoding' не будет присутствовать в объявлении XML-документа.

<ДобавлятьBOM> (необязательный)

Тип: Булево.   
Определяет, будет ли добавлена метка порядка байт (BOM) к результирующему потоку XML.  
Значение по умолчанию: Истина.

Описание варианта метода:

Позволяет указать тип кодировки, который будет использован для записи потока XML.

Вариант синтаксиса: С параметрами записи

Синтаксис:

ОткрытьПоток(<Поток>, <ПараметрыЗаписи>, <ДобавлятьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, в который будет записан текст XML.

<ПараметрыЗаписи> (необязательный)

Тип: [ПараметрыЗаписиXML](ПараметрыЗаписиXML.md).   
Параметры записи XML.

<ДобавлятьBOM> (необязательный)

Тип: Булево.   
Определяет, будет ли добавлена метка порядка байт (BOM) к результирующему потоку XML.  
Значение по умолчанию: Истина.

Описание варианта метода:

Позволяет указать параметры записи XML.

Описание:

Открывает поток для записи документов и фрагментов XML. Позволяет указать тип кодировки, который будет использован для записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
