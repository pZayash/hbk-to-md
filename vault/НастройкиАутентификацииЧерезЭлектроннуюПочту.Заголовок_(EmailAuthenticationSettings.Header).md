НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

Заголовок (Header)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит заголовок сообщения в том виде, в котором письмо хранится на сервере.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
