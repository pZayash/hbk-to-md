ОбъектМетаданных: Подсистема (MetadataObject: Subsystem)

Состав (Content)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийСвойстваОбъектаМетаданных](КоллекцияЗначенийСвойстваОбъектаМетаданных_(MetadataObjectPropertyValueCollection).md).   
Содержит список идентификаторов объектов метаданных, принадлежащих подсистеме.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При конвертации из версии 8.1 свойство заполняется на основе информации о принадлежности объектов метаданных подсистемам.  
  
Сохранение порядка элементов в коллекции между сеансами не гарантируется.

Использование в версии:

Доступен, начиная с версии 8.2.
