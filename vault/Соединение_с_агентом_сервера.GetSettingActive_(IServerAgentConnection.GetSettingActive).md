# Соединение с агентом сервера.GetSettingActive

Соединение с агентом сервера (IServerAgentConnection)

GetSettingActive (GetSettingActive)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetSettingActive(<Кластер>, <Сервер>, <Настройка>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого требуется проверить активность настройки.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   
Сервер, для которого требуется проверить активность настройки.

<Настройка> (обязательный)

Тип: [Настройка сервиса](Настройка_сервиса_(IServiceSetting).md).   
Настройка сервиса кластера.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если настройка активна,  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Получает значение активности настройки.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
