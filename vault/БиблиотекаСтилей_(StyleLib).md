# БиблиотекаСтилей

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Общие интерфейсные объекты](Общие_интерфейсные_объекты.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[Стиль](Стиль_(Style).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Свойства:

[\<Имя стиля\> (\<Style name\>)](БиблиотекаСтилей.Имя_стиля_(StyleLib.Style_name).md)  

Методы:

[Индекс (IndexOf)](БиблиотекаСтилей.Индекс_(StyleLib.IndexOf).md)  
[Количество (Count)](БиблиотекаСтилей.Количество_(StyleLib.Count).md)  
[Получить (Get)](БиблиотекаСтилей.Получить_(StyleLib.Get).md)  

Описание:

Представляет собой библиотеку стилей конфигурации.

Доступность:

Сервер, толстый клиент.

[Глобальный контекст](Глобальный_контекст.md), свойство [БиблиотекаСтилей](Глобальный_контекст.БиблиотекаСтилей_(Global_context.StyleLib).md)

Доступен, начиная с версии 8.0.
