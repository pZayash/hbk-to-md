ПараметрТаблицыСхемыЗапроса (QuerySchemaTableParameter)

Выражение (Expression)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md).   
Выражение. Для установки свойства используется [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md). Оно должно быть создано при помощи конструктора и присвоено свойству.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
