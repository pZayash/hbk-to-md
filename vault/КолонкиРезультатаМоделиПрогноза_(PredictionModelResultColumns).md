**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаРезультатаМоделиПрогноза](КолонкаРезультатаМоделиПрогноза_(PredictionModelResultColumn).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонкам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит колонки, выводимые в результат модели прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.
