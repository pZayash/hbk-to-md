ФрагментДокументаDOM (DOMDocumentFragment)

ДочерниеУзлы (ChildNodes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [СписокУзловDOM](СписокУзловDOM_(DOMNodeList).md).   
Список дочерних узлов DOM. В случае отсутствия дочерних узлов содержит пустой список.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
