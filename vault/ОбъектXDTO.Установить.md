ОбъектXDTO (XDTODataObject)

Установить (Set)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По выражению XPath

Синтаксис:

Установить(<Выражение>, <Значение>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.   
Выражение XPath, определяющее свойство для установки значения.  
Если по указанному выражению значение определить не удалось или найдено несколько результатов, будет сгенерировано исключение.  
Корнем выполнения выражения XPath является сам объект.

<Значение> (необязательный)

Тип: [ОбъектXDTO](ОбъектXDTO.md), [ЗначениеXDTO](ЗначениеXDTO.md), Неопределено.   
Устанавливаемый элемент данных XDTO.

Вариант синтаксиса: По свойству

Синтаксис:

Установить(<Свойство>, <Значение>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO.md).   
Свойство, в которое необходимо установить значение.

<Значение> (обязательный)

Тип: [ОбъектXDTO](ОбъектXDTO.md), [ЗначениеXDTO](ЗначениеXDTO.md), Неопределено.   
Элемент данных XDTO, устанавливаемый в качестве значения данного свойства.

Описание:

Устанавливает значение свойства объекта.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.26.
