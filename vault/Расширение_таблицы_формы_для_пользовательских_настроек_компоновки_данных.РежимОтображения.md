Расширение таблицы формы для пользовательских настроек компоновки данных (Form table extension for data composition user settings)

РежимОтображения (ViewMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтображенияНастроекКомпоновкиДанных](РежимОтображенияНастроекКомпоновкиДанных.md).   
Свойство появляется у таблицы, если она связана с данными типа [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных.md).  
Свойство определяет режим отображения таблицы. Показываются все элементы отбора или только те, которые имеют режим отображения [БыстрыйДоступ](РежимОтображенияЭлементаНастройкиКомпоновкиДанных.БыстрыйДоступ.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
