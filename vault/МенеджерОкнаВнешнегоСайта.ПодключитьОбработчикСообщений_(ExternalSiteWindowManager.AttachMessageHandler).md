МенеджерОкнаВнешнегоСайта (ExternalSiteWindowManager)

ПодключитьОбработчикСообщений (AttachMessageHandler)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПодключитьОбработчикСообщений(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана при получении сообщения из внешнего сайта со следующими параметрами:

- <Сообщение> - тип [СообщениеВнешнегоСайта](СообщениеВнешнегоСайта_(MessageFromExternalSite).md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Подключает процедуру-обработчик, вызываемую при получении сообщения из внешнего сайта.  
Допускается наличие нескольких обработчиков. В этом случае при получении сообщения из внешнего сайта будут вызваны все обработчики. Порядок вызова обработчиков не определен.  
Вызов метода с тем же обработчиком замещает обработчик.  
Сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Если окно внешнего сайта не доступно (свойство [Доступно](МенеджерОкнаВнешнегоСайта.Доступно_(ExternalSiteWindowManager.Enabled).md) имеет значение [Ложь](lang__def_BooleanFalse.md)), тогда при вызове данного метода будет сгенерировано исключение "Окно внешнего сайта не доступно".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[СообщениеВнешнегоСайта](СообщениеВнешнегоСайта_(MessageFromExternalSite).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.
