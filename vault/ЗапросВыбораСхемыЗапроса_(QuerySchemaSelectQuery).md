Описание:

Объект предназначен для создания и изменения запросов при помощи объектов встроенного языка.  
Соответствует запросу, начинающемуся с ключевого слова ВЫБРАТЬ.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.25.
