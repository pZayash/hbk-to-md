НастройкаПорядка (OrderSetting)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента порядка.

<Доступность> (обязательный)

Тип: Булево.   
Определяет доступность элемента для редактирования.

Возвращаемое значение:

Тип: [ЭлементУправленияПорядком](ЭлементУправленияПорядком.md).   

Описание:

Добавляет элемент.  
Внимание! Метод не доступен для настройки порядка динамических списков (свойство НастройкаПорядка объектов [СправочникСписок.](СправочникСписок.Имя_справочника.md)<[Имя справочника](СправочникСписок.Имя_справочника.md)>, [ДокументСписок.](ДокументСписок.Имя_документа.md)<[Имя документа](ДокументСписок.Имя_документа.md)> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
