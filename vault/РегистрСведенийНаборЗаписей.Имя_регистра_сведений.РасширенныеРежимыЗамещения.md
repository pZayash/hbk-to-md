РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

РасширенныеРежимыЗамещения (ExtendedReplaceMode)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет возможность использования расширенных режимов замещения [Удаление](РежимЗамещения.Удаление.md) или [Обновление](РежимЗамещения.Обновление.md) в качестве значения параметра метода [Записать](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.Записать.md) для наборов записей подчиненных регистров.   
Также определяет доступность для записи свойства [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки.md) у записей, принадлежащих набору записей.  
На поведение наборов записей независимых регистров сведений значение данного свойства не влияет.  
Если свойство имеет значение Истина, то режимы замещения [Удаление](РежимЗамещения.Удаление.md) или [Обновление](РежимЗамещения.Обновление.md) могут быть использованы в качестве значения параметра метода [Записать](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.Записать.md), а свойство [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки.md) у записей, принадлежащих набору, доступно для записи.   
Иначе режимы [Удаление](РежимЗамещения.Удаление.md) или [Обновление](РежимЗамещения.Обновление.md) не могут быть использованы, а свойство [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки.md) у записей, принадлежащих набору, недоступно для записи.  
При создании набора записей свойству присваивается значение Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
