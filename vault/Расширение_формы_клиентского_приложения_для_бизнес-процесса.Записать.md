Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура.md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Содержит предопределенный параметр Старт. Если параметр Старт=Истина, то бизнес-процесс будет стартован.  
Состав других параметров произвольный.

Возвращаемое значение:

Тип: Булево.   
Истина - успешно записан; Ложь - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
