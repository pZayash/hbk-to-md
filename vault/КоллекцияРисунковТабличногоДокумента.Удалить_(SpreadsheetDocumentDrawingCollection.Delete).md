КоллекцияРисунковТабличногоДокумента (SpreadsheetDocumentDrawingCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Рисунок>)

Параметры:

<Рисунок> (обязательный)

Тип: [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md).   
Удаляемый рисунок.

Описание:

Удаляет рисунок из коллекции рисунков табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
