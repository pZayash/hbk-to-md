# ФабрикаXDTO.ЗаписатьXML

ФабрикаXDTO (XDTOFactory)

ЗаписатьXML (WriteXML)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьXML(<ЗаписьXML>, <Элемент>, <ЛокальноеИмя>, <URIПространстваИмен>, <Форма>, <УказаниеТипа>)

Параметры:

<ЗаписьXML> (обязательный)

Тип: [ЗаписьУзловDOM](ЗаписьУзловDOM_(DOMNodeWriter).md), [ЗаписьFastInfoset](ЗаписьFastInfoset_(FastInfosetWriter).md), [ЗаписьXML](ЗаписьXML_(XMLWriter).md).   
Объект записи XML.

<Элемент> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [Неопределено](lang__def_Undefined.md).   
Записываемое значение.

<ЛокальноеИмя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя записываемого элемента данных.

<URIПространстваИмен> (необязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен записываемого элемента данных.

<Форма> (необязательный)

Тип: [ФормаXML](ФормаXML_(XMLForm).md).   
Форма представления элемента данных в XDTO.  
Значение по умолчанию: [Элемент](ФормаXML.Элемент_(XMLForm.Element).md).

<УказаниеТипа> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Вариант назначения типа элемента данных XDTO.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Описание:

Записывает указанный элемент данных XDTO в объект записи XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
