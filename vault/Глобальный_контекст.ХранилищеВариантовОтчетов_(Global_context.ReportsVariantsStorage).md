# Глобальный контекст.ХранилищеВариантовОтчетов

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Свойства](Свойства.md)

Глобальный контекст (Global context)

ХранилищеВариантовОтчетов (ReportsVariantsStorage)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md), [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)>.   
Содержит объект, с помощью которого осуществляется чтение, изменение, добавление и удаление настроек вариантов отчета.   
Если в свойстве конфигурации "ХранилищеВариантовОтчетов" не указано хранилище, то содержит объект [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md), предназначенный для чтения, изменения, добавления и удаления настроек вариантов отчетов.  
Если в свойстве конфигурации "ХранилищеВариантовОтчетов" указано хранилище, созданное в конфигурации, то содержит объект [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)>.  
  
Для чтения, изменения, добавления и удаления настроек следует в параметрах методов объектов [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md) и [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)> использовать следующие значения:

- <Ключ объекта> – указывается полное имя отчета. Например: "Отчет.Продажи".
- <Ключ настройки> – указывается текстовый идентификатор варианта. Например: "ПродажиПоРегионам".

Тип сохраняемого значения – [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
