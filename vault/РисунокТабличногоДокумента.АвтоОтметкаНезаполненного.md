РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет, необходимо ли выделять рисунок, в котором не заполнено значение, либо оно выставлено в значение по умолчанию.   
Если нет заполненного текста, будет установлено [ОтметкаНезаполненного](РисунокТабличногоДокумента.ОтметкаНезаполненного.md) в Истина, иначе Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо только для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), [Примечание](ТипРисункаТабличногоДокумента.Примечание.md).

Использование в версии:

Доступен, начиная с версии 8.0.
