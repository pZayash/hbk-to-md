РазрешенныйВнешнийМодуль (AllowedExternalModule)

ХешСумма (HashSum)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит контрольную сумму внешнего модуля, вычисленную алгоритмом SHA-1, преобразованную к строке base64. Для построения контрольной суммы можно использовать одно из следующих:

- объект [ХешированиеДанных](ХешированиеДанных.md) с указанием хеш-функции SHA-1;
- метод глобального контекста [Base64Строка](Глобальный_контекст.Base64Строка.md).

Для расширений конфигурации хеш-сумма может быть получена с помощью свойства [ХешСумма](РасширениеКонфигурации.ХешСумма.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
