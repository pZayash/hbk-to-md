ИнтернетПочтовоеСообщение (InternetMailMessage)

УстановитьИсходныеДанные (SetSourceData)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьИсходныеДанные(<ИсходныеДанные>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Исходные данные почтового сообщения.

Описание:

Устанавливает исходные данные сообщения и инициализирует на основе них объект типа [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md). В случае ошибки разбора указанных данных генерируетcя исключение.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
