СредстваБуфераОбмена (ClipboardTools)

ПолучитьДанныеАсинх (GetDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПолучитьДанныеАсинх(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена_(ClipboardDataStandardFormat).md), [Строка](lang__def_String.md).   
Формат получаемых данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – данные типа [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Картинка](Картинка_(Picture).md) или [Неопределено](lang__def_Undefined.md).  
Если формат не известен, то возвращается [Неопределено](lang__def_Undefined.md).  
Если значения, соответствующего формату, не найдено, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Получает из буфера обмена данные указанного формата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте:

- может появиться запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается [Обещание](Обещание_(Promise).md), результат которого – [Неопределено](lang__def_Undefined.md).
- значение параметра <Формат> типа [Строка](lang__def_String.md) должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Использование в версии:

Доступен, начиная с версии 8.3.24.
