Расширение поля формы для поля ввода (Form field extension for a text box)

ОткрытиеМножественногоЗначения (MultipleValueOpening)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОткрытиеМножественногоЗначения(<Идентификатор>, <СтандартнаяОбработка>)

Параметры:

<Идентификатор>

Тип: Число.   
Идентификатор для объекта [ЭлементСпискаЗначений](ЭлементСпискаЗначений.md) или [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции.md).   
Эти идентификаторы не привязаны к позиции элемента в списке или таблице значений.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается при нажатии на гиперссылку множественного значения или выборе пункта контекстного меню "Открыть".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
