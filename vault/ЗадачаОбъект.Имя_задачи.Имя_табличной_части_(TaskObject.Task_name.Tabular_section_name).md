# ЗадачаОбъект.<Имя задачи>.<Имя табличной части>

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ЗадачаТабличнаяЧасть.<Имя задачи>.<Имя табличной части>.   
Набор свойств содержит табличные части задачи. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ЗадачаТабличнаяЧасть.<Имя задачи>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
