КолонкиТабличногоПоля (TableBoxColumns)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <ТекстШапки>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс вставляемой колонки. Если индекс равен количеству колонок, то происходит добавление колонки в конец коллекции.

<ТекстШапки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст, который будет выводиться в заголовке колонки табличного поля.

Возвращаемое значение:

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md).   
Вставленная колонка табличного поля.

Описание:

Вставляет колонку в заданную позицию коллекции колонок табличного поля.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ЭлементыФормы.ТабличноеПоле.Колонки.Вставить(0, "Наименование"); |

Использование в версии:

Доступен, начиная с версии 8.0.
