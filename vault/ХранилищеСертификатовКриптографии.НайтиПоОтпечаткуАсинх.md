ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НайтиПоОтпечаткуАсинх (FindByThumbprintAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НайтиПоОтпечаткуАсинх(<Отпечаток>)

Параметры:

<Отпечаток> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Отпечаток, по которому производится поиск.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - массив объектов типа [СертификатКриптографии](СертификатКриптографии.md).

Описание:

Асинхронная функция. Осуществляет поиск сертификатов в хранилище по заданному отпечатку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
