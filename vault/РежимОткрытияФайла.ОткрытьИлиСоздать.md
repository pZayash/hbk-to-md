РежимОткрытияФайла (FileOpenMode)

ОткрытьИлиСоздать (OpenOrCreate)

Доступен, начиная с версии 8.3.9.

Описание:

Открытие существующего файла. Если файл не существует, будет создан новый файл. Если файл существует, то при записи данные в нем будут перезаписаны.

Использование в версии:

Доступен, начиная с версии 8.3.9.
