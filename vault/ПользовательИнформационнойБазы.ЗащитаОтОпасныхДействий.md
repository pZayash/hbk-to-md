ПользовательИнформационнойБазы (InfoBaseUser)

ЗащитаОтОпасныхДействий (UnsafeOperationProtection)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеЗащитыОтОпасныхДействий](ОписаниеЗащитыОтОпасныхДействий.md).   
Структура для настройки предупреждений пользователю о загрузке внешних отчетов, внешних обработок и расширений конфигурации, на которые пользователь ранее не давал согласия и выполнении потенциально опасных функций языка в коде загруженных модулей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
