ДанныеМестоположения (LocationData)

Направление (Heading)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Содержит направление движения. Задается в градусах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Неопределено](lang__def_Undefined.md), если не поддерживается провайдером геопозиционирования.

Использование в версии:

Доступен, начиная с версии 8.3.3.
