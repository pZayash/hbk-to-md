# Глобальный контекст.ОсновнойСерверДоступен

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Мобильный клиент с возможностью автономной работы](Мобильный_клиент_с_возможностью_автономной_работы-3.md)

Глобальный контекст (Global context)

ОсновнойСерверДоступен (MainServerAvailable)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОсновнойСерверДоступен()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Показывает доступность основного сервера с клиента или автономного сервера.   
[Неопределено](lang__def_Undefined.md) - в случае, если отсутствует возможность автономной работы (используется мобильное приложение или мобильный клиент без возможности автономной работы).

Доступность:

Мобильный клиент, мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
