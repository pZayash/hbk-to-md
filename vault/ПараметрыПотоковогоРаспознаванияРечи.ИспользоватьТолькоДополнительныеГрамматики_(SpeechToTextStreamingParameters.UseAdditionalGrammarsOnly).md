ПараметрыПотоковогоРаспознаванияРечи (SpeechToTextStreamingParameters)

ИспользоватьТолькоДополнительныеГрамматики (UseAdditionalGrammarsOnly)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак использования только дополнительных грамматик для распознавания.  
Если установлено [Истина](lang__def_BooleanTrue.md), используются только дополнительние грамматики.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
