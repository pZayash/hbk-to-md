ПолеКалендаря (CalendarBox)

РазрешитьНачалоПеретаскивания (EnableStartDrag)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешает начало операции перетаскивания в поле календаря - источнике данных.

Доступность:

Толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - перетаскивание разрешено; [Ложь](lang__def_BooleanFalse.md) - запрещено.

Использование в версии:

Доступен, начиная с версии 8.0.
