ГеографическаяСхема (GeographicalSchema)

Записать (Write)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет записан текущий документ географической схемы.

Описание варианта метода:

Запись географической схемы в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи географической схемы.

Описание варианта метода:

Запись географической схемы в поток.

Описание:

Записывает объект географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
