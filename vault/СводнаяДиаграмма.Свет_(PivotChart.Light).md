СводнаяДиаграмма (PivotChart)

Свет (Light)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак моделирования эффекта освещенности объемных сводных диаграмм.  
[Истина](lang__def_BooleanTrue.md) - установлен эффект освещенности.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл для объемных сводных диаграмм.

Использование в версии:

Доступен, начиная с версии 8.0.
