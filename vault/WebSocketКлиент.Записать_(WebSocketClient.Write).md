WebSocketКлиент (WebSocketClient)

Записать (Write)

Доступен, начиная с версии 8.3.27.

Синтаксис:

Записать()

Описание:

Записывает объект, если в него были внесены изменения. Позволяет переопределить некоторые данные, описывающие WebSocket-клиента относительно тех, которые заданы в метаданных.  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если ключ не уникальный, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
