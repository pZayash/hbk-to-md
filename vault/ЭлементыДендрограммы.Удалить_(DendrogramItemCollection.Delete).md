ЭлементыДендрограммы (DendrogramItemCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементДендрограммы](ЭлементДендрограммы_(DendrogramItem).md).   
Удаляемый элемент.

Описание:

Удаляет элемент дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
