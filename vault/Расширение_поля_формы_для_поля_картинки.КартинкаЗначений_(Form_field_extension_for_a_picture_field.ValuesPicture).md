Расширение поля формы для поля картинки (Form field extension for a picture field)

КартинкаЗначений (ValuesPicture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Поле вида [ПолеКартинки](ВидПоляФормы.ПолеКартинки_(FormFieldType.PictureField).md) может отображать данные типов [Число](lang__def_Number.md) и [Булево](lang__Булево_(Boolean).md). В этом случае значение представляет собой индекс в наборе картинок, задаваемых этим свойством. Для типа [Булево](lang__Булево_(Boolean).md) значение [Истина](lang__def_BooleanTrue.md) означает индекс 0, а значение [Ложь](lang__def_BooleanFalse.md) означает индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.2.
