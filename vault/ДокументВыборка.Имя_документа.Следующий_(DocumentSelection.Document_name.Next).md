# ДокументВыборка.<Имя документа>.Следующий

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументВыборка.<Имя документа>](ДокументВыборка.Имя_документа_(DocumentSelection.Document_name).md) › [Методы](ДокументВыборка.Имя_документа__Методы.md)

ДокументВыборка.<Имя документа> (DocumentSelection.<Document name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующий документ выбран. [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующий документ из выборки. Для обхода списка документов нужно после получения выборки использовать данный метод для позиционирования на первый и последующие документы выборки до тех пор, пока он не вернет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Выборка = Документы.ПоступлениеТоваров.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка записи выборки  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
