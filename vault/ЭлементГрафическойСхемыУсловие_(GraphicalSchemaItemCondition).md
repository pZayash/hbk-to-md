Описание:

Элемент графической схемы "Условие". Отображает точку маршрута бизнес-процесса, из которой есть два выхода - в зависимости от логического условия.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
