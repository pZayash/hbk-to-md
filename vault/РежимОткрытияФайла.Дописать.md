РежимОткрытияФайла (FileOpenMode)

Дописать (Append)

Доступен, начиная с версии 8.3.9.

Описание:

Открывает существующий файл и перемещает позицию потока в конец файла. Если файл не существует, будет создан новый файл.

Использование в версии:

Доступен, начиная с версии 8.3.9.
