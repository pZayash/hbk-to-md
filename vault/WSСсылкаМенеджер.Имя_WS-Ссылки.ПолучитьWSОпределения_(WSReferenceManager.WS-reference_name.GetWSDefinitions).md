WSСсылкаМенеджер.<Имя WS-Ссылки> (WSReferenceManager.<WS-reference name>)

ПолучитьWSОпределения (GetWSDefinitions)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьWSОпределения()

Возвращаемое значение:

Тип: [WSОпределения](WSОпределения_(WSDefinitions).md).   

Описание:

Получает определения для данного Web-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Определения = WSСсылки.Заказы.ПолучитьWSОпределения(); |

Использование в версии:

Доступен, начиная с версии 8.1.
