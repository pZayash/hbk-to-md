Глобальный контекст (Global context)

УстановитьСоответствиеОбъектаИФормы (SetObjectAndFormConformity)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьСоответствиеОбъектаИФормы(<Объект>, <ИдентификаторФормыИИмяРеквизита>)

Параметры:

<Объект> (обязательный)

Тип: [СправочникОбъект.](СправочникОбъект.Имя_справочника.md)<[Имя справочника](СправочникОбъект.Имя_справочника.md)>, [ДокументОбъект.](ДокументОбъект.Имя_документа.md)<[Имя документа](ДокументОбъект.Имя_документа.md)>, [ПоследовательностьНаборЗаписей.](ПоследовательностьНаборЗаписей.Имя_последовательности.md)<[Имя последовательности](ПоследовательностьНаборЗаписей.Имя_последовательности.md)>, [ОтчетОбъект.](ОтчетОбъект.Имя_отчета.md)<[Имя отчета](ОтчетОбъект.Имя_отчета.md)>, [ОбработкаОбъект.](ОбработкаОбъект.Имя_обработки.md)<[Имя обработки](ОбработкаОбъект.Имя_обработки.md)>, [ВнешняяОбработка](ВнешняяОбработка.md), [ВнешнийОтчет](ВнешнийОтчет.md), [ПланВидовХарактеристикОбъект.](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md)>, [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>, [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>, [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>, [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.md)>, [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)>, [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии.md)>, [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>.   
Прикладной объект, преобразованный из универсального объекта в значение, для которого необходимо установить соответствие с именем реквизита формы.

<ИдентификаторФормыИИмяРеквизита> (обязательный)

Тип: [Структура](Структура.md).   
Содержит идентификатор формы и имя реквизита элемента управления, для которых выполняется установка соответствия.   
Структура, передаваемая в параметр, может содержать два элемента с именами ключей:  
1. "ИдентификаторФормы" ("FormID") – уникальный идентификатор формы (тип [УникальныйИдентификатор](УникальныйИдентификатор.md));  
2. "ИмяРеквизита" ("AttributeName") – имя реквизита в форме (тип Строка). Этот элемент может не указываться.

Описание:

Устанавливает соответствие формы клиентского приложения и объектов вида [СообщениеПользователю](СообщениеПользователю.md) для последующей правильной привязки сообщений к элементам управления формы. Для того, чтобы сообщение использовало данные соответствия объекта и формы, у него необходимо заполнить свойство [ИдентификаторНазначения](СообщениеПользователю.ИдентификаторНазначения.md). Соответствие может быть получено методом [ПолучитьСоответствиеОбъектаИФормы](Глобальный_контекст.ПолучитьСоответствиеОбъектаИФормы.md). Получать соответствие можно до тех пор, пока существует объект, для которого оно зарегистрированно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
