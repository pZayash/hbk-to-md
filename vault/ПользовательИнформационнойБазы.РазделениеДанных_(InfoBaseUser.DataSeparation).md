# ПользовательИнформационнойБазы.РазделениеДанных

ПользовательИнформационнойБазы (InfoBaseUser)

РазделениеДанных (DataSeparation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Содержит установленные пользователю значения разделителей.  
Ключ - имя общего реквизита, являющегося разделителем.  
Значение - [Строка](lang__def_String.md), строковое представление значения разделителя в зависимости от типа общего реквизита:

- [Булево](lang__Булево_(Boolean).md) - 0 или 1;
- [Число](lang__def_Number.md) - десятичное представление числа;
- [Дата](lang__def_Date.md) - yyyymmddhhmmss, где yyyy - номер года, mm - номер месяца (01 - 12), dd - день месяца (01 - 31), hh - номер часа (00 - 23), mm - номер минуты (00 - 59), ss - номер секунды (00 - 59);
- [Строка](lang__def_String.md) - содержимое строки без модификации;
- Ссылка на объект информационной базы - номер или код объекта. В зависимости от типа номера или кода:

- [Число](lang__def_Number.md) - десятичное представление номера;
- [Строка](lang__def_String.md) - содержимое строки без модификации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Проверка правильности структуры выполняется при присвоении значения свойству [РазделениеДанных](ПользовательИнформационнойБазы.РазделениеДанных_(InfoBaseUser.DataSeparation).md) и при выполнении метода [Записать](ПользовательИнформационнойБазы.Записать_(InfoBaseUser.Write).md) объекта [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).
