Описание:

Содержит параметры диалога помещения файлов.  
Используется в качестве параметра <ПараметрыДиалога> методов [НачатьПомещениеФайлаНаСервер](Глобальный_контекст.НачатьПомещениеФайлаНаСервер_(Global_context.BeginPutFileToServer).md) и [НачатьПомещениеФайловНаСервер](Глобальный_контекст.НачатьПомещениеФайловНаСервер_(Global_context.BeginPutFilesToServer).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
