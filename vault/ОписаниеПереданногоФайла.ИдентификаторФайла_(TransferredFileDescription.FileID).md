ОписаниеПереданногоФайла (TransferredFileDescription)

ИдентификаторФайла (FileID)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Содержит идентификатор объекта [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md), соответствующий помещенному файлу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
