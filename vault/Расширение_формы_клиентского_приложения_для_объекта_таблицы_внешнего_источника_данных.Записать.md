Расширение формы клиентского приложения для объекта таблицы внешнего источника данных

Записать (Write)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Записать()

Возвращаемое значение:

Тип: Булево.   
Истина - успешно записан; Ложь - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
