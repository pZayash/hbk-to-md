Расширение поля формы для поля географической схемы (Form field extension for a geographical schema field)

РастягиватьПоВертикали (VerticalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает возможность растягивания географической схемы по вертикали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
