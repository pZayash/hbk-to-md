ТаблицыДляИзмененияСхемыЗапроса (QuerySchemaTablesForUpdate)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: По источнику

Синтаксис:

Добавить(<Источник>)

Параметры:

<Источник> (обязательный)

Тип: [ИсточникСхемыЗапроса](ИсточникСхемыЗапроса_(QuerySchemaSource).md).   
Источник.

Описание варианта метода:

Добавляет таблицу для изменения по источнику.

Вариант синтаксиса: По имени таблицы

Синтаксис:

Добавить(<ИмяТаблицы>)

Параметры:

<ИмяТаблицы> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя таблицы. Если таблица с заданным именем не найдена, будет сгенерировано исключение.

Описание варианта метода:

Добавляет таблицу для изменения по имени источника.

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
