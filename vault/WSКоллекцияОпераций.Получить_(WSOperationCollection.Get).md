WSКоллекцияОпераций (WSOperationCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Индекс в коллекции или имя операции.

Возвращаемое значение:

Тип: [WSОперация](WSОперация_(WSOperation).md).   

Описание:

Получает элемент коллекции по индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
