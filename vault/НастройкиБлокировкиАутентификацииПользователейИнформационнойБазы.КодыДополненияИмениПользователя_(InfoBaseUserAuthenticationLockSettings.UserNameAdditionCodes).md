НастройкиБлокировкиАутентификацииПользователейИнформационнойБазы (InfoBaseUserAuthenticationLockSettings)

КодыДополненияИмениПользователя (UserNameAdditionCodes)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит строку в формате:  
<КодДополнения>;...;<КодДополнения>,  
где <КодДополнения> - строка, которую требуется добавить к имени заблокированного пользователя, чтобы выполнить аутентификацию.  
Например, если <КодДополнения> - "123", то "Пользователь123" является алиасосм для пользователя с именем "Пользователь".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если свойство не задано, то содержит пустую строку.

Использование в версии:

Доступен, начиная с версии 8.3.16.
