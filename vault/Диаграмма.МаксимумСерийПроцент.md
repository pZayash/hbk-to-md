Диаграмма (Chart)

МаксимумСерийПроцент (MaxSeriesPercent)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит процент, ограничивающий вывод отдельных серий в диаграмму. Используется, когда свойство [МаксимумСерий](Диаграмма.МаксимумСерий.md) имеет значение [Процент](МаксимумСерий.Процент.md). Серии для показа отбираются по следующему алгоритму: выбирается абсолютный максимум по всем точкам всех серий, вычисляется пороговое значение отбора умножением выбранного максимума на МаксимумСерий.Процент/100. Если в серии есть значения, превышающие этот порог, эта серия будет показана. Значения непоказанных серий суммируются и выводятся в серии "Сводная".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
