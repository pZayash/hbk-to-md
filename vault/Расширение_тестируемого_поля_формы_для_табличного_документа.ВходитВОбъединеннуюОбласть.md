Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ВходитВОбъединеннуюОбласть (IncludedInMergedArea)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ВходитВОбъединеннуюОбласть(<Область>)

Параметры:

<Область> (необязательный)

Тип: Строка.   
Содержит имя области в формате RNCN[:RMCM].  
Если не указан, проверяется текущая ячейка.

Возвращаемое значение:

Тип: Строка, Неопределено.   
Строка - Имя объединенной области, если указанный диапазон (область) ячеек входит в объединенную область.  
Неопределено - указанный диапазон (область) ячеек не входит в объединенную область.

Описание:

Проверяет включение указанного диапазона (области) ячеек в объединенную область ячеек табличного документа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
