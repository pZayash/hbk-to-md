Глобальный контекст (Global context)

НачатьУстановкуРасширенияРаботыСКриптографией (BeginInstallCryptoExtension)

Доступен, начиная с версии 8.3.3.

Синтаксис:

НачатьУстановкуРасширенияРаботыСКриптографией(<ОписаниеОповещенияОЗавершении>)

Параметры:

<ОписаниеОповещенияОЗавершении> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения установки расширения работы с криптографией со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Описание:

Устанавливает расширение для работы с криптографией в браузер.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска толстого и тонкого клиентов, а также на клиенте мобильного приложения информирует о том, что расширение установлено.

Пример:

| --- |
| НаКлиенте Процедура Команда(Команда)     Подключить(Истина); КонецПроцедуры  НаКлиенте  Процедура Подключить(УстановитьЕслиНеПодключено) Экспорт     НачатьПодключениеРасширенияРаботыСКриптографией(         Новый ОписаниеОповещения(             "ПослеПодключения",             ЭтотОбъект,             УстановитьЕслиНеПодключено)); КонецПроцедуры  НаКлиенте Процедура ПослеПодключения(Подключено, УстановитьЕслиНеПодключено) Экспорт     Если Подключено Тогда         // Расширение работы с криптографией подключено      ИначеЕсли УстановитьЕслиНеПодключено Тогда         НачатьУстановкуРасширенияРаботыСКриптографией(             Новый ОписаниеОповещения(                 "Подключить",                 ЭтотОбъект,                 Ложь));     Иначе         // Не удалось установить или подключить          // расширение работы с криптографией      КонецЕсли;     КонецПроцедуры |
