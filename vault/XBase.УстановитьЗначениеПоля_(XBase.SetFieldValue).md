XBase (XBase)

УстановитьЗначениеПоля (SetFieldValue)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьЗначениеПоля(<ИмяПоля>, <Значение>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Имя или индекс поля. Индекс полей начинается с 0.

<Значение> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [Булево](lang__Булево_(Boolean).md).   
Значение поля.

Описание:

Устанавливает значение поля по его индексу или имени.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
