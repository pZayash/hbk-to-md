ФильтрУзловDOM (DOMNodeFilter)

На основании параметров отбора

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ФильтрУзловDOM(<ОтображатьВсе>, <ОтображатьЭлемент>, <ОтображатьАтрибут>, <ОтображатьТекст>, <ОтображатьСекцииCDATA>, <ОтображатьСсылкуНаСущность>, <ОтображатьСущность>, <ОтображатьИнструкциюОбработки>, <ОтображатьКомментарий>, <ОтображатьДокумент>, <ОтображатьОпределениеТипаДокумента>, <ОтображатьФрагментДокумента>, <ОтображатьНотацию>)

Параметры:

<ОтображатьВсе> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать все узлы.

<ОтображатьЭлемент> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы элементов.

<ОтображатьАтрибут> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы атрибутов.

<ОтображатьТекст> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы текста.

<ОтображатьСекцииCDATA> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы секций CDATA.

<ОтображатьСсылкуНаСущность> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы ссылок на сущности DOM.

<ОтображатьСущность> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы сущностей.

<ОтображатьИнструкциюОбработки> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы инструкций обработки.

<ОтображатьКомментарий> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы комментариев.

<ОтображатьДокумент> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узел документа DOM.

<ОтображатьОпределениеТипаДокумента> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узел определения типа документа.

<ОтображатьФрагментДокумента> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы фрагмента DOM.

<ОтображатьНотацию> (необязательный)

Тип: [ПараметрыОтбораУзловDOM](ПараметрыОтбораУзловDOM_(DOMNodeFilterParameters).md).   
Отображать узлы нотаций DOM.

Описание:

Создает объект отбора узлов DOM по переданным параметрам отбора.  
Если какой-либо параметр не указан, то в объект отбора не включается соответствующий вид отбора.

Использование в версии:

Доступен, начиная с версии 8.1.
