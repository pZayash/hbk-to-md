ИсточникиСхемыЗапроса (QuerySchemaSources)

ПолучитьКорневыеИсточники (GetRootSources)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьКорневыеИсточники()

Возвращаемое значение:

Тип: [ИсточникиСхемыЗапроса](ИсточникиСхемыЗапроса.md).   

Описание:

Получает коллекцию источников, которые не соединяются с другими источниками. Полученная коллекция не будет доступна для редактирования.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
