# КомандыФормы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Форма клиентского приложения](Форма_клиентского_приложения.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[КомандаФормы](КомандаФормы_(FormCommand).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются команды.  
Возможно обращение к команде посредством оператора [...]. В качестве аргумента передается имя команды.  

Методы:

[Добавить (Add)](КомандыФормы.Добавить_(FormCommands.Add).md)  
[Количество (Count)](КомандыФормы.Количество_(FormCommands.Count).md)  
[Найти (Find)](КомандыФормы.Найти_(FormCommands.Find).md)  
[Удалить (Delete)](КомандыФормы.Удалить_(FormCommands.Delete).md)  

Описание:

Содержит коллекцию команд формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), свойство [Команды](ФормаКлиентскогоПриложения.Команды_(ClientApplicationForm.Commands).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
