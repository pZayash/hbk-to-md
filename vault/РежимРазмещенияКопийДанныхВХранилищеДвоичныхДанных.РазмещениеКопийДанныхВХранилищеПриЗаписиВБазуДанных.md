РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных (BinaryDataStorageDataCopiesPlacementMode)

РазмещениеКопийДанныхВХранилищеПриЗаписиВБазуДанных (DataCopiesPlacementToStorageOnWritingToDatabase)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных.md).   
Данные записываются в хранилище и служебную таблицу. При чтении сначала выполняется попытка чтения данных из хранилища, а в случае ошибки чтения - из служебной таблицы.  
Данный режим предназначен для обеспечения отказоустойчивого хранения данных в базе данных с использованием хранилища в качестве кэша для быстрого доступа.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
