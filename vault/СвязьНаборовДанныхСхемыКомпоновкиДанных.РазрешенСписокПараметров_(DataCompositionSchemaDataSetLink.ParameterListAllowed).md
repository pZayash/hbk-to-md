СвязьНаборовДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetLink)

РазрешенСписокПараметров (ParameterListAllowed)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возможность использования в качестве значения параметра список значений. Список значений допустим в случае, если параметр используется в операции В.  
[Истина](lang__def_BooleanTrue.md) - список значений допускается к использованию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
