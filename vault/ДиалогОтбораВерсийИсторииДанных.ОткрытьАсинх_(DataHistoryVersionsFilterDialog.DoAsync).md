ДиалогОтбораВерсийИсторииДанных (DataHistoryVersionsFilterDialog)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - если диалог был закрыт нажатием кнопки "OK", содержит структуру, идентичную структуре в параметре <Отбор> метода [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии_(DataHistoryManager.SelectVersions).md), иначе [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Показывает диалога отбора версий.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
