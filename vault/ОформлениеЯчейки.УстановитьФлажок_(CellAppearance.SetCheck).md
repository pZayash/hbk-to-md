ОформлениеЯчейки (CellAppearance)

УстановитьФлажок (SetCheck)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьФлажок(<Флажок>)

Параметры:

<Флажок> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md).   
Значение флажка.

Описание:

Устанавливает свойство [Флажок](ОформлениеЯчейки.Флажок_(CellAppearance.Check).md) объекта. Устанавливает свойство [ОтображатьФлажок](ОформлениеЯчейки.ОтображатьФлажок_(CellAppearance.ShowCheckBox).md) в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
