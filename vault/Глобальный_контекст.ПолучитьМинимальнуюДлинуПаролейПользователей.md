Глобальный контекст (Global context)

ПолучитьМинимальнуюДлинуПаролейПользователей (GetUserPasswordMinLength)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьМинимальнуюДлинуПаролейПользователей()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает ограничение на минимальную длину паролей, которое установлено для пользователей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
