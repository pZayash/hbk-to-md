# ТестируемаяТаблицаФормы.ПолучитьВыделенныеСтроки

ТестируемаяТаблицаФормы (TestedFormTable)

ПолучитьВыделенныеСтроки (GetSelectedRows)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьВыделенныеСтроки()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   

Описание:

Возвращает массив соответствий с описаниями выделенных строк. Ключ соответствия содержит заголовок колонки, значение - текст ячейки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
