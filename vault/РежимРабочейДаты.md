Значения

[ИспользоватьТекущуюДату (UseCurrentDate)](РежимРабочейДаты.ИспользоватьТекущуюДату.md)  
[Назначать (Assign)](РежимРабочейДаты.Назначать.md)  

Описание:

Определяет набор вариантов использования рабочей даты в глобальном контексте. Используется для определения свойства [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты.md) глобального контекста.

Доступность:

Толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
