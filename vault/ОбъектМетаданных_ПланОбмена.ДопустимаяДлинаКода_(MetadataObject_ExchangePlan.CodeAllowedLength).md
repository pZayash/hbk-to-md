ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

ДопустимаяДлинаКода (CodeAllowedLength)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md).   
Содержит тип допустимой длины кода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство имеет смысл, если тип кода - [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
