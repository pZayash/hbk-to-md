СущностьDOM (DOMEntity)

СравнитьПозициюВДокументе (CompareDocumentPosition)

Доступен, начиная с версии 8.1.

Синтаксис:

СравнитьПозициюВДокументе(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Узел DOM, с которым происходит сравнение.

Возвращаемое значение:

Тип: [ПозицияВДокументеDOM](ПозицияВДокументеDOM.md).   

Описание:

Сравнивает данный узел с узлом, переданным в качестве параметра, для определения позиции относительно документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
