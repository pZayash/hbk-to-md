ГрафическаяСхема (GraphicalSchema)

ИспользуемоеИмяФайла (UsedFileName)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла, которое будет использоваться при сохранении графической схемы пользователем.   
Данное свойство будет заполнятся:  
1. при программной установке значения;  
2. при интерактивном открытии документа из файла;  
3. при вызове метода [Показать](ГрафическаяСхема.Показать_(GraphicalSchema.Show).md), если в нем указано имя файла;  
4. при вызове пользователем команды "Сохранить как…".

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.1.
