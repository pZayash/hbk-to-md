МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтключитьОбработчикНовыхСообщенийАсинх (DetachNewMessagesHandlerAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По идентификатору

Синтаксис:

ОтключитьОбработчикНовыхСообщенийАсинх(<Обсуждение>, <Обработчик>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения, для которого отключается обработчик.

<Обработчик> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, заданной при вызове метода [НачатьПодключениеОбработчикаНовыхСообщений](МенеджерСистемыВзаимодействия.НачатьПодключениеОбработчикаНовыхСообщений_(CollaborationSystemManager.BeginAttachNewMessagesHandler).md).

Описание варианта метода:

Выполняет отключение обработчика по идентификатору обсуждения.

Вариант синтаксиса: По ключу

Синтаксис:

ОтключитьОбработчикНовыхСообщенийАсинх(<Ключ>, <Обработчик>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ обсуждения, для которого отключается обработчик.

<Обработчик> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, заданной при вызове метода [НачатьПодключениеОбработчикаНовыхСообщений](МенеджерСистемыВзаимодействия.НачатьПодключениеОбработчикаНовыхСообщений_(CollaborationSystemManager.BeginAttachNewMessagesHandler).md).

Описание варианта метода:

Выполняет отключение обработчика по ключу обсуждения.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Отключает ранее подключенный обработчик.  
Сравнение обработчиков происходит по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Не рекомендуется выполнять отключение обработчика в коде самого обработчика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [НачатьОтключениеОбработчикаНовыхСообщений](МенеджерСистемыВзаимодействия.НачатьОтключениеОбработчикаНовыхСообщений_(CollaborationSystemManager.BeginDetachNewMessagesHandler).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
