# СредстваМультимедиа.СканироватьДокументыАсинх

СредстваМультимедиа (MultimediaTools)

СканироватьДокументыАсинх (ScanDocumentsAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

СканироватьДокументыАсинх(<Заголовок>, <ПараметрыСканирования>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка, который будет показан в окне сканирования документов.

<ПараметрыСканирования> (необязательный)

Тип: [ПараметрыСканированияДокументов](ПараметрыСканированияДокументов_(DocumentScanningParameters).md).   
Набор входных параметров для сканирования документов.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – массив значений типа [СтраницаСканированияДокументов](СтраницаСканированияДокументов_(DocumentScanningPage).md) или [Неопределено](lang__def_Undefined.md).  
Возвращает данные сканирования документов, в порядке заданным пользователем, или [Неопределено](lang__def_Undefined.md), если пользователь закрыл интерфейс сканирования документов.

Описание:

Асинхронная функция. Выводит на экран интерфейс сканирования документов. Пользователь фотографирует страницы документа с помощью камеры, встроенной в мобильное устройство. После получения фотоснимка производится автоматическое сканирование на наличие страницы документа. Найденная страница документа вырезается в отдельную картинку, которая обрабатывается выбранным пользователем фильтром из списка, указанного в параметрах сканирования. Метод позволяет производить сканирование как отдельных документов, так и серии страниц одного документа.   
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяСканированиеДокументов](СредстваМультимедиа.ПоддерживаетсяСканированиеДокументов_(MultimediaTools.DocumentScanningSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
