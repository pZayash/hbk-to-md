ОбъектМетаданных: WebСервис (MetadataObject: WebService)

ПакетыXDTO (XDTOPackages)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийСвойстваОбъектаМетаданных](КоллекцияЗначенийСвойстваОбъектаМетаданных_(MetadataObjectPropertyValueCollection).md).   
Коллекция объектов метаданных, описывающих пакеты XDTO, типы которых могут использоваться в качестве типов возвращаемого значения операций и типов параметров операций данного Web-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
