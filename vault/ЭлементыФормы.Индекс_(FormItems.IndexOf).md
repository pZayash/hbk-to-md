# ЭлементыФормы.Индекс

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Форма клиентского приложения](Форма_клиентского_приложения.md) › [ЭлементыФормы](ЭлементыФормы_(FormItems).md) › [Методы](ЭлементыФормы__Методы-2.md)

ЭлементыФормы (FormItems)

Индекс (IndexOf)

Доступен, начиная с версии 8.2.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md), [КнопкаФормы](КнопкаФормы_(FormButton).md).   
Элемент в коллекции.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
