Глобальный контекст (Global context)

ПродолжитьВызов (ProceedWithCall)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПродолжитьВызов(<<Значение1>,...,<ЗначениеN>>)

Параметры:

<<Значение1>,...,<ЗначениеN>> (необязательный)

Тип: Произвольный.   
Набор параметров, который передается в переопределенный метод из метода модуля заимствованного объекта. Количество и типы параметров определяются методом из модуля расширяемого объекта.  
Максимальное возможное количество параметров - 50.

Возвращаемое значение:

Тип: Произвольный.   
Возвращаемое значение переопределенной функции.

Описание:

Вызывает переопределенный метод или обработчик события из расширяемой конфигурации.  
Функция предназначена только для однократного вызова. Использовать функцию можно только в расширениях и только с аннотацией &Вместо.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Пример:

|  |
| --- |
| // пример с переопределенной процедурой  &Вместо("Процедура1") Процедура Расш1\_Процедура1(Парам1)     ...     ПродолжитьВызов(Парам1);     ... КонецПроцедуры  // пример с переопределенной функцией  &Вместо("Функция1") Функция Расш1\_Функция1(Парам1)     ...     Перем1 = ПродолжитьВызов(Парам1);     Возврат Перем1; КонецПроцедуры  // пример с переопределенным обработчиком  &Вместо("ПриЗаписи") Процедура Расш1\_ПриЗаписи(Отказ)     ...     ПродолжитьВызов(Отказ); КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.21.
