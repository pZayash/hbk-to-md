МенеджерКриптографии (CryptoManager)

ПолучитьХранилищеСертификатов (GetCertificateStore)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьХранилищеСертификатов(<ТипХранилища>, <РасположениеХранилища>)

Параметры:

<ТипХранилища> (необязательный)

Тип: [ТипХранилищаСертификатовКриптографии](ТипХранилищаСертификатовКриптографии.md).   
Определяет тип хранилища сертификатов, который должен вернуть метод.   
Если параметр не задан, возвращает хранилище, содержащее сертификаты всех доступных типов.

<РасположениеХранилища> (необязательный)

Тип: [РасположениеХранилищаСертификатовКриптографии](РасположениеХранилищаСертификатовКриптографии.md).   
Определяет, к какому хранилищу предоставить доступ: к хранилищу сертификатов пользователя операционной системы или к хранилищу сертификатов локального компьютера.  
На мобильной платформе по умолчанию используется значение [ДанныеПриложения](РасположениеХранилищаСертификатовКриптографии.ДанныеПриложения.md).  
Значение по умолчанию: [ДанныеПользователяОС](РасположениеХранилищаСертификатовКриптографии.ДанныеПользователяОС.md).

Возвращаемое значение:

Тип: [ХранилищеСертификатовКриптографии](ХранилищеСертификатовКриптографии.md).   
На мобильной платфоме возвращает пустое хранилище, если устройство не поддерживает указанное расположение хранилища сертификатов криптографии. Поддерживается только расположение [ДанныеПриложения](РасположениеХранилищаСертификатовКриптографии.ДанныеПриложения.md). В остальных платформах поддерживаются расположения [ДанныеКомпьютера](РасположениеХранилищаСертификатовКриптографии.ДанныеКомпьютера.md) и [ДанныеПользователяОС](РасположениеХранилищаСертификатовКриптографии.ДанныеПользователяОС.md).

Описание:

Получает хранилище сертификатов, содержащее сертификаты заданного типа.   
  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для типов сертификатов [СертификатыУдостоверяющихЦентров](ТипХранилищаСертификатовКриптографии.СертификатыУдостоверяющихЦентров.md) и [КорневыеСертификаты](ТипХранилищаСертификатовКриптографии.КорневыеСертификаты.md) возвращаемое хранилище доступно только для чтения.
