Описание:

Содержит варианты контроля использования методов, которые открывают модальные окна, т.к. реализация веб-клиента для некоторых платформ не поддерживает создания модальных окон.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
