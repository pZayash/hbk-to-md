РазрешенныйCOMКласс (AllowedCOMClass)

ИмяФайла (FileName)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя файла, по которому создается объект методом [ПолучитьCOMОбъект](Глобальный_контекст.ПолучитьCOMОбъект_(Global_context.GetCOMObject).md) с пустым значением второго параметра. В общем случае строка используется как текстовое представление объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
