Порядок (Order)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс удаляемого элемента.

Описание:

Удаляет элемент порядка по индексу.  
Внимание! Метод не доступен для упорядочивания динамических списков (свойство Порядок объектов [СправочникСписок.](СправочникСписок.Имя_справочника.md)<[Имя справочника](СправочникСписок.Имя_справочника.md)>, [ДокументСписок.](ДокументСписок.Имя_документа.md)<[Имя документа](ДокументСписок.Имя_документа.md)> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Очистим порядок  Пока ПостроительОтчета.Порядок.Количество()>0 Цикл     ПостроительОтчета.Порядок.Удалить(0); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
