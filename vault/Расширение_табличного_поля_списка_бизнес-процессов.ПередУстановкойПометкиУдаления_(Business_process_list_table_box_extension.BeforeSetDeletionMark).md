Расширение табличного поля списка бизнес-процессов (Business process list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак изменения пометки удаления бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке бизнес-процессов.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](БизнесПроцессОбъект.Имя_бизнес-процесса.ПередЗаписью_(BusinessProcessObject.Business_process_name.BeforeWrite).md) и [ПриЗаписи](БизнесПроцессОбъект.Имя_бизнес-процесса.ПриЗаписи_(BusinessProcessObject.Business_process_name.OnWrite).md) объекта [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
