РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: РегистрБухгалтерии](ОбъектМетаданных_РегистрБухгалтерии_(MetadataObject_AccountingRegister).md).   

Описание:

Предоставляет доступ к объекту описания метаданных регистра бухгалтерского учета. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.РегистрыБухгалтерии.Хозрасчетный.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
