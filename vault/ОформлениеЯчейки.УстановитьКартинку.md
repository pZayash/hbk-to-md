ОформлениеЯчейки (CellAppearance)

УстановитьКартинку (SetPicture)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьКартинку(<Картинка>)

Параметры:

<Картинка> (обязательный)

Тип: [Картинка](Картинка.md).   
Картинка.

Описание:

Устанавливает свойство [Картинка](ОформлениеЯчейки.Картинка.md) объекта [ОформлениеЯчейки](ОформлениеЯчейки.md) в значение, переданное в параметре <Картинка>, и устанавливает свойство [ОтображатьКартинку](ОформлениеЯчейки.ОтображатьКартинку.md) в значение Истина.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
