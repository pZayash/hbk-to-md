СоединениеССерверомСистемыАналитики (AnalyticsSystemServerConnection)

УдалитьСхему (DeleteSchema)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УдалитьСхему(<Путь>, <Идентификатор>)

Параметры:

<Путь> (обязательный)

Тип: Строка.   
Путь к удаляемой схеме на сервере системы аналитики.

<Идентификатор> (необязательный)

Тип: Строка.   
Идентификатор объекта системы аналитики, который будет удален.

Описание:

Удаляет схему по указанному пути и идентификатору из системы аналитики.  
Содержание параметров определяется сервером системы аналитики и описано в его документации.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
