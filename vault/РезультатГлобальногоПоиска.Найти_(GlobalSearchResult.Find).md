РезультатГлобальногоПоиска (GlobalSearchResult)

Найти (Find)

Доступен, начиная с версии 8.3.15.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение результата поиска.

Возвращаемое значение:

Тип: [ЭлементРезультатаГлобальногоПоиска](ЭлементРезультатаГлобальногоПоиска_(GlobalSearchResultItem).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента коллекции результата глобального поиска по его значению.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
