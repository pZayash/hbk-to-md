ФормаКлиентскогоПриложения (ClientApplicationForm)

НачатьПолучениеОбсуждения (BeginGetConversation)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьПолучениеОбсуждения(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <Обсуждение> - тип [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md) или Неопределено,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если у формы нет навигационной ссылки (например, форма нового незаписанного объекта), то в параметре <ИдентификаторОбсуждения> передается Неопределено.

Описание:

Начинает получение контекстного обсуждения формы.  
Если обсуждение с указанным идентификатором не найдено, оно будет создано.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md), метод [ПолучитьОбсуждениеАсинх](ФормаКлиентскогоПриложения.ПолучитьОбсуждениеАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.16.
