СтрокаГруппировкиДинамическогоСписка (DynamicListGroupRow)

РодительскаяГруппировка (ParentGrouping)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтрокаГруппировкиДинамическогоСписка](СтрокаГруппировкиДинамическогоСписка.md), Неопределено.   
Содержит значение группировки верхнего уровня.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Неопределено - если группировка осуществлялась по полю, в котором есть значения типа Null.

Использование в версии:

Доступен, начиная с версии 8.2.
