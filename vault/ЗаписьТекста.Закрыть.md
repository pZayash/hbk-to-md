ЗаписьТекста (TextWriter)

Закрыть (Close)

Доступен, начиная с версии 8.0.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый текстовый файл. Закрытие файла выполняется автоматически при удалении объекта [ЗаписьТекста](ЗаписьТекста.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
