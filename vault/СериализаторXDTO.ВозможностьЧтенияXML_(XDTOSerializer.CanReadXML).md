# СериализаторXDTO.ВозможностьЧтенияXML

СериализаторXDTO (XDTOSerializer)

ВозможностьЧтенияXML (CanReadXML)

Доступен, начиная с версии 8.1.

Синтаксис:

ВозможностьЧтенияXML(<ЧтениеXML>)

Параметры:

<ЧтениеXML> (обязательный)

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM_(DOMNodeReader).md), [ЧтениеFastInfoset](ЧтениеFastInfoset_(FastInfosetReader).md), [ЧтениеXML](ЧтениеXML_(XMLReader).md).   
Объект, через который производится чтение XML.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - тип 1С:Предприятия существует; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Производит проверку возможности чтения из объекта [ЧтениеXML](ЧтениеXML_(XMLReader).md) значения.   
Данный метод получает тип данных XML из объекта [ЧтениеXML](ЧтениеXML_(XMLReader).md) аналогично методу [ПолучитьXMLТип](СериализаторXDTO.ПолучитьXMLТип_(XDTOSerializer.GetXMLType).md), а затем пытается определить, имеется ли соответствующий тип 1С:Предприятия аналогично тому, как это делает метод [ИзXMLТипа](СериализаторXDTO.ИзXMLТипа_(XDTOSerializer.FromXMLType).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не производит чтения. Он определяет возможность чтения указанного типа в текущем положении объекта чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
