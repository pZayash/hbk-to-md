ГлубинаЦвета (ColorDepth)

БитНаПиксел8 (BitPerPixel8)

Доступен, начиная с версии 8.3.14.

Описание:

Для хранения одного пикселя используется 8 бит (картинка использует 256 цветов).

Использование в версии:

Доступен, начиная с версии 8.3.14.
