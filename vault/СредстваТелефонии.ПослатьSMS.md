СредстваТелефонии (TelephonyTools)

ПослатьSMS (SendSMS)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПослатьSMS(<SMSСообщение>, <Интерактивно>)

Параметры:

<SMSСообщение> (обязательный)

Тип: [SMSСообщение](SMSСообщение.md).   
Содержит передаваемое SMS-сообщение.

<Интерактивно> (обязательный)

Тип: Булево.   
Указывает, используется ли интерактивный режим для отправки SMS-сообщений.

Описание:

Если параметр <Интерактивно> установлен в значение Истина, то будет запущено системное приложение для отправки SMS-сообщений.   
В противном случае, SMS-сообщение будет отправлено средствами мобильной платформы.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать один из методов:

- [ПоддерживаетсяОтправкаSMS](СредстваТелефонии.ПоддерживаетсяОтправкаSMS.md) - если вложения не используются,
- [ПоддерживаетсяОтправкаMMS](СредстваТелефонии.ПоддерживаетсяОтправкаMMS.md) - если вложения используются.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии.md), метод [ПоддерживаетсяОтправкаSMS](СредстваТелефонии.ПоддерживаетсяОтправкаSMS.md)  
[СредстваТелефонии](СредстваТелефонии.md), метод [ПоддерживаетсяОтправкаMMS](СредстваТелефонии.ПоддерживаетсяОтправкаMMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
