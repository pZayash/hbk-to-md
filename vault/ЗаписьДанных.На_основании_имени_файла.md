ЗаписьДанных (DataWriter)

На основании имени файла

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый ЗаписьДанных(<ИмяФайла>, <КодировкаТекста>, <ПорядокБайтов>, <РазделительСтрок>, <Дописать>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выполнена запись.

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста для создаваемого экземпляра [ЗаписьДанных](ЗаписьДанных_(DataWriter).md). Если не задана, то используется UTF-8.  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый по умолчанию для кодирования целых чисел при записи в поток.  
Значение по умолчанию: [LittleEndian](ПорядокБайтов.LittleEndian_(ByteOrder.LittleEndian).md).

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Разделитель по умолчанию для строк, записываемых в поток. Если разделитель строк не задан, то используется строка [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<Дописать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли данные записаны в начало или в конец файла:

- Если [Истина](lang__def_BooleanTrue.md), то при открытии существующего файла запись будет выполнена в конец файла.
- Иначе данные будут записываться с начала файла, перезаписывая существующие данные.

Если заданный файл не существует, будет создан новый файл с указанным именем и значение параметра не повлияет на поведение конструктора.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если в начало файла или потока требуется записать метку порядка байтов (BOM) для используемой кодировки текста, то данный параметр должен иметь значение [Истина](lang__def_BooleanTrue.md).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Создает объект [ЗаписьДанных](ЗаписьДанных_(DataWriter).md) для записи в указанный файл. Если файл с таким именем не существует, он будет создан. Параметр <Дописать> определяет, будут ли данные записаны в начало или в конец файла.  
После завершения работы с объектом, до закрытия потока, переданного в конструктор, объект необходимо закрыть с помощью метода [Закрыть](ЗаписьДанных.Закрыть_(DataWriter.Close).md) или [НачатьЗакрытие](ЗаписьДанных.НачатьЗакрытие_(DataWriter.BeginClose).md). При этом используемый файл будет закрыт автоматически.  
В режиме запуска "Веб-клиент" конструктор не используется.

Использование в версии:

Доступен, начиная с версии 8.3.9.
