ЗаписьДанных (DataWriter)

На основании имени файла

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый ЗаписьДанных(<ИмяФайла>, <КодировкаТекста>, <ПорядокБайтов>, <РазделительСтрок>, <Дописать>, <КонвертируемыйРазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который будет выполнена запись.

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста для создаваемого экземпляра [ЗаписьДанных](ЗаписьДанных.md). Если не задана, то используется UTF-8.  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8.md).

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов.md).   
Порядок байтов, используемый по умолчанию для кодирования целых чисел при записи в поток.  
Значение по умолчанию: [LittleEndian](ПорядокБайтов.LittleEndian.md).

<РазделительСтрок> (необязательный)

Тип: Строка.   
Разделитель по умолчанию для строк, записываемых в поток. Если разделитель строк не задан, то используется строка [ПС](Символы.ПС.md).  
Значение по умолчанию: [ПС](Символы.ПС.md).

<Дописать> (необязательный)

Тип: Булево.   
Определяет, будут ли данные записаны в начало или в конец файла:

- Если Истина, то при открытии существующего файла запись будет выполнена в конец файла.
- Иначе данные будут записываться с начала файла, перезаписывая существующие данные.

Если заданный файл не существует, будет создан новый файл с указанным именем и значение параметра не повлияет на поведение конструктора.  
Значение по умолчанию: Ложь.

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<ЗаписатьBOM> (необязательный)

Тип: Булево.   
Если в начало файла или потока требуется записать метку порядка байтов (BOM) для используемой кодировки текста, то данный параметр должен иметь значение Истина.  
Значение по умолчанию: Ложь.

Описание:

Создает объект [ЗаписьДанных](ЗаписьДанных.md) для записи в указанный файл. Если файл с таким именем не существует, он будет создан. Параметр <Дописать> определяет, будут ли данные записаны в начало или в конец файла.  
После завершения работы с объектом, до закрытия потока, переданного в конструктор, объект необходимо закрыть с помощью метода [Закрыть](ЗаписьДанных.Закрыть.md) или [НачатьЗакрытие](ЗаписьДанных.НачатьЗакрытие.md). При этом используемый файл будет закрыт автоматически.  
В режиме запуска "Веб-клиент" конструктор не используется.

Использование в версии:

Доступен, начиная с версии 8.3.9.
