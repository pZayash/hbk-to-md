ПолеГеографическойСхемы (GeographicalSchemaField)

ОбработкаРасшифровки (DetailProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаРасшифровки(<Расшифровка>, <СтандартнаяОбработка>, <ОбъектСхемы>, <Серия>)

Параметры:

<Расшифровка>

Тип: Произвольный.   
Значение расшифровки.  
Если щелчок произведен в области объекта схемы, то содержит значение расшифровки объекта.  
Если щелчок произведен по какому-либо значению в одной из диаграмм объекта, то содержит расшифровку значения серии. Если она отсутствует, расшифровку объекта, которому принадлежит значение.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ОбъектСхемы>

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   
Объект или серия объекта, для которого выполняется обработка расшифровки.

<Серия>

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы_(GeographicalSchemaLayerDataSeries).md).   
Серия, требующая расшифровки (щелчок выполнен в области серии).

Описание:

Возникает при щелчке пользователя по объекту географической схемы или значению одной из диаграмм объекта.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
