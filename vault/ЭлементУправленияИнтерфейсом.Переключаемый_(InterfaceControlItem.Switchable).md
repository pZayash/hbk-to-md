ЭлементУправленияИнтерфейсом (InterfaceControlItem)

Переключаемый (Switchable)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак, определяющий видимость интерфейса. Первоначальное значение свойства для интерфейса задается в Конфигураторе. Интерфейсы, у которых данное свойство установлено в [Ложь](lang__def_BooleanFalse.md), останутся видимы после переключения интерфейсов.

Доступность:

Толстый клиент.

См. также:

[КоллекцияЭлементовУправленияИнтерфейсами](КоллекцияЭлементовУправленияИнтерфейсами_(InterfaceControlItemCollection).md), метод [ПереключитьИнтерфейс](КоллекцияЭлементовУправленияИнтерфейсами.ПереключитьИнтерфейс_(InterfaceControlItemCollection.SwitchInterface).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
