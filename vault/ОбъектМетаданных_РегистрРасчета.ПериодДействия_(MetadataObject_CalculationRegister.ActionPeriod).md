ОбъектМетаданных: РегистрРасчета (MetadataObject: CalculationRegister)

ПериодДействия (ActionPeriod)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то в данном регистре можно учитывать записи, протяженные во времени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
