Форма (Form)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования формы в режиме просмотра. Если свойство установлено в [Истина](lang__def_BooleanTrue.md), то становятся не доступны для изменения все дочерние элементы вне зависимости от значения их свойств ТолькоПросмотр.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
