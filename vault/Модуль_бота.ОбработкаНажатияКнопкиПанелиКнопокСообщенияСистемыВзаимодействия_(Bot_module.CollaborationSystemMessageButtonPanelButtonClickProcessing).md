Модуль бота (Bot module)

ОбработкаНажатияКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonClickProcessing)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ОбработкаНажатияКнопкиПанелиКнопокСообщенияСистемыВзаимодействия(<Сообщение>, <Кнопка>, <Пользователь>, <ДополнительныеПараметры>)

Параметры:

<Сообщение>

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).   
Сообщение, в котором была нажата кнопка.

<Кнопка>

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   
Кнопка, которая была нажата.

<Пользователь>

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Идентификатор пользователя, который нажал на кнопку.

<ДополнительныеПараметры>

Тип: Произвольный.   
Параметры, заданные при создании бота.

Описание:

Обработчик, вызываемый при нажатии кнопки панели кнопок сообщения системы взаимодействия, если действие кнопки [ОбработатьБотом](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОбработатьБотом_(CollaborationSystemMessageButtonPanelButtonAction.ProcessByBot).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Бот обрабатывает только нажатия на кнопки в сообщениях, созданных этим же ботом.

Использование в версии:

Доступен, начиная с версии 8.3.25.
