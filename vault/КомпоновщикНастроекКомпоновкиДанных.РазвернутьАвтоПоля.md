КомпоновщикНастроекКомпоновкиДанных (DataCompositionSettingsComposer)

РазвернутьАвтоПоля (ExpandAutoFields)

Доступен, начиная с версии 8.1.

Синтаксис:

РазвернутьАвтоПоля()

Описание:

Преобразует все автополя ([АвтоВыбранноеПолеКомпоновкиДанных](АвтоВыбранноеПолеКомпоновкиДанных.md), [АвтоПолеГруппировкиКомпоновкиДанных](АвтоПолеГруппировкиКомпоновкиДанных.md), [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных.md)), содержащиеся в настройках, в наборы соответствующих элементов.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
