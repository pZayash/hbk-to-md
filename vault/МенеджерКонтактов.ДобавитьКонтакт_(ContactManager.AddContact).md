МенеджерКонтактов (ContactManager)

ДобавитьКонтакт (AddContact)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ДобавитьКонтакт(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ДанныеКонтактаУчетнойЗаписи](ДанныеКонтактаУчетнойЗаписи_(AccountContactData).md).   
Данные контакта учетной записи.

Возвращаемое значение:

Тип: [ЛокальныйКлючКонтакта](ЛокальныйКлючКонтакта_(LocalContactKey).md).   

Описание:

Добавляет новый контакт и возвращает его локальный ключ.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяДобавлениеКонтактов](МенеджерКонтактов.ПоддерживаетсяДобавлениеКонтактов_(ContactManager.ContactsAddingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКонтактов](МенеджерКонтактов_(ContactManager).md), метод [ПоддерживаетсяДобавлениеКонтактов](МенеджерКонтактов.ПоддерживаетсяДобавлениеКонтактов_(ContactManager.ContactsAddingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
