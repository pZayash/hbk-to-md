РасширениеКонфигурации (ConfigurationExtension)

ИспользуетсяВРаспределеннойИнформационнойБазе (UsedInDistributedInfoBase)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то данное расширение конфигурации будет передаваться в распределенных информационных базах, организованных планами обмена, у которых свойство [РаспределеннаяИнформационнаяБаза](ОбъектМетаданных_ПланОбмена.РаспределеннаяИнформационнаяБаза_(MetadataObject_ExchangePlan.DistributedInfoBase).md) установлено в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Изменение свойства на клиенте требует обращения к серверу.

Доступен, начиная с версии 8.3.12.
