Глобальный контекст (Global context)

ПолучитьИспользованиеЖурналаРегистрации (GetEventLogUsing)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьИспользованиеЖурналаРегистрации()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив состоит из значений типа [УровеньЖурналаРегистрации](УровеньЖурналаРегистрации_(EventLogLevel).md). для которых включена регистрация событий в журнале.  
Если регистрация отключена, возвращается пустой массив.

Описание:

Проверяет, осуществляется ли в журнале регистрация события заданного уровня важности.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
