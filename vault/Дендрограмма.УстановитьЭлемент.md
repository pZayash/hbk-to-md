Дендрограмма (Dendrogram)

УстановитьЭлемент (SetItem)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьЭлемент(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Устанавливаемое значение элемента.

Возвращаемое значение:

Тип: [ЭлементДендрограммы](ЭлементДендрограммы.md).   
Если элемента с указанным значением нет, то создается новый элемент и он возвращается методом, иначе возвращается существующий.

Описание:

Устанавливает элемента дендрограммы по значению.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
