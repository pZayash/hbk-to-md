Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияНастройкойОбластиОформления](ЭлементУправленияНастройкойОбластиОформления_(AppearanceAreasSettingControl).md).

Удаляет элемент управления областью оформления.

Использование в версии:

Доступен, начиная с версии 8.0.
