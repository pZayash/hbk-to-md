ВытесняющиеВидыРасчета.<Имя плана видов расчета> (DisplacingCalculationTypes.<Chart of calculation types name>)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ВытесняющиеВидыРасчетаСтрока.\<Имя плана видов расчета\>](ВытесняющиеВидыРасчетаСтрока.Имя_плана_видов_расчета_(DisplacingCalculationTypesRow.Chart_of_calculation_types_name).md).   

Описание:

Добавляет строку в конец табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода допустимо только в том случае, если табличная часть получена из свойства объекта. Если табличная часть получена из свойства ссылки (или выборки), то использование этого метода будет вызывать ошибку выполнения.

Пример:

| --- |
| Процедура УстановитьВытесняющиеРасчеты(ВидРасчета,          ВытесняющиеВидыРасчета)     ВидРасчета = ВидРасчета.ПолучитьОбъект();     ВидРасчета.ВытесняющиеВидыРасчета.Очистить();     Если ВытесняющиеВидыРасчета <> Неопределено Тогда         Для каждого Элемент Из ВытесняющиеВидыРасчета Цикл             НоваяСтрока = ВидРасчета.ВытесняющиеВидыРасчета.Добавить();             НоваяСтрока.ВидРасчета = Элемент.Значение;         КонецЦикла;     КонецЕсли;      ВидРасчета.Записать(); КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
