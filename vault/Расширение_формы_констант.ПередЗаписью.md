Расширение формы констант (Constant form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением записи объекта из формы. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](КонстантаМенеджерЗначения.Имя_константы.ПередЗаписью.md)объекта [КонстантаМенеджерЗначения.](КонстантаМенеджерЗначения.Имя_константы.md)<[Имя константы](КонстантаМенеджерЗначения.Имя_константы.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
