ФабрикаXDTO (XDTOFactory)

ЭкспортСхемыXML (ExportXMLSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

ЭкспортСхемыXML(<URIПространствИмен>)

Параметры:

<URIПространствИмен> (обязательный)

Тип: [Строка](lang__def_String.md), [Массив](Массив_(Array).md).   
Массив строк или строка, соответствующая URI пространства имен экспортируемых пакетов.

Возвращаемое значение:

Тип: [НаборСхемXML](НаборСхемXML_(XMLSchemaSet).md).   

Описание:

Выполняет экспорт пакетов XDTO, имеющих указанные URI пространства имен в набор схем XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
