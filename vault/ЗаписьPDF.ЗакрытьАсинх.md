ЗаписьPDF (PDFWriter)

ЗакрытьАсинх (CloseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗакрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Закрывает открытый файл. Закрытие файла выполняется автоматически при удалении объекта [ЗаписьPDF](ЗаписьPDF.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[ЗаписьPDF](ЗаписьPDF.md), метод [Закрыть](ЗаписьPDF.Закрыть.md)  
[ЗаписьPDF](ЗаписьPDF.md), метод [НачатьЗакрытие](ЗаписьPDF.НачатьЗакрытие.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
