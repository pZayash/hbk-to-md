ФормаКлиентскогоПриложения (ClientApplicationForm)

ПриЗакрытии (OnClose)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЗакрытии(<ЗавершениеРаботы>)

Параметры:

<ЗавершениеРаботы>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак того, что форма закрывается в процессе завершения работы приложения:

- [Истина](lang__def_BooleanTrue.md) - если в процессе завершения приложения; [Ложь](lang__def_BooleanFalse.md) - если закрывается только форма.

Описание:

Возникает на клиенте при закрытии формы.  
В процессе завершения работы приложения (параметр [ЗавершениеРаботы](html.md) имеет значение [Истина](lang__def_BooleanTrue.md)) запрещены серверные вызовы и открытие окон.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В обработчике события отказаться от закрытия формы нельзя.

Использование в версии:

Доступен, начиная с версии 8.2.
