СистемнаяИнформация (SystemInfo)

ВариантПриложения (AppVariant)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит признак специального варианта поставки платформы "1С:Предприятие". Обычная поставка в данном свойстве содержит пустую строку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
