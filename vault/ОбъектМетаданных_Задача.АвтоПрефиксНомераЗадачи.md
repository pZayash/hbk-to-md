ОбъектМетаданных: Задача (MetadataObject: Task)

АвтоПрефиксНомераЗадачи (TaskNumberAutoPrefix)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [АвтоПрефиксНомераЗадачи](АвтоПрефиксНомераЗадачи.md).   
Если это свойство установлено в значение [НомерБизнесПроцесса](АвтоПрефиксНомераЗадачи.НомерБизнесПроцесса.md), то при создании новой задачи ее номер автоматически дополняется номером соответствующего ей бизнес-процесса.  
Если это свойство установлено в значение [НеИспользовать](АвтоПрефиксНомераЗадачи.НеИспользовать.md), то номер новой задачи не изменяется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
