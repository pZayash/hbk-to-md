ДиалогВыбораФайла (FileDialog)

ТипыСодержимого (MIMETypes)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Массив строк с допустимыми типами содержимого для выбора.  
Фильтрация по типам содержимого всегда объединяется с активным фильтром по И.  
Если [Разделы](ДиалогВыбораФайла.Разделы_(FileDialog.Sections).md) имеет значение [Неопределено](lang__def_Undefined.md), то при указании определенного типа содержимого разделы автоматически настраиваются.

- image, video и любой их подтип – Галерея, Документы, Файлы,
- audio и любой подтип – Аудио, Документы, Файлы.

Используется только в мобильной платформе для iOS и Android.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.20.
