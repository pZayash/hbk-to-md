# МенеджерКриптографии.ПроверитьСертификатАсинх

МенеджерКриптографии (CryptoManager)

ПроверитьСертификатАсинх (CheckCertificateAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьСертификатАсинх(<Сертификат>, <РежимПроверки>, <УниверсальнаяДатаПроверки>)

Параметры:

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).   
Проверяемый сертификат.

<РежимПроверки> (необязательный)

Тип: [РежимПроверкиСертификатаКриптографии](РежимПроверкиСертификатаКриптографии_(CryptoCertificateCheckMode).md), [Массив](Массив_(Array).md).   
Указывает режим проверки.   
Содержит один объект или массив объектов режимов проверки.

<УниверсальнаяДатаПроверки> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Время проверки и построения цепочки сертификатов, указывается в универсальном времени (UTC).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
В случае успешного результата проверки результатом выполнения обещания является [Неопределено](lang__def_Undefined.md). В случае ошибки объект [Обещание](Обещание_(Promise).md) будет содержать исключение, которое будет выброшено из оператора Ждать, аргументом которого будет данное [Обещание](Обещание_(Promise).md).

Описание:

Асинхронная функция. Проверяет сертификат. В случае ошибки генерируется исключительная ситуация.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
