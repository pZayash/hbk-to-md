ЗаписьДвиженийПриПроведении (RegisterRecordsWritingOnPost)

ЗаписыватьМодифицированные (WriteModified)

Доступен, начиная с версии 8.2.

Описание:

Все наборы записей, участвующие в регистрации движений документа, имеют значение свойства Записывать установленным системой по умолчанию в [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.2.
