МенеджерWebSocketКлиентСоединений (WebSocketClientConnectionsManager)

ОткрытьСоединение (OpenConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ОткрытьСоединение(<Ключ>, <URLСервера>, <Обработчики>, <Параметры>, <ИмяПользователяИнформационнойБазы>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор соединения.

<URLСервера> (обязательный)

Тип: [Строка](lang__def_String.md).   
URL WebSocket-сервера, с которым создано соединение.

<Обработчики> (необязательный)

Тип: [ОбработчикиWebSocketКлиентСоединения](ОбработчикиWebSocketКлиентСоединения_(WebSocketClientConnectionHandlers).md).   
Имена процедур-обработчиков, модуля, пользователя информационной базы.

<Параметры> (необязательный)

Тип: [ПараметрыWebSocketКлиентСоединения](ПараметрыWebSocketКлиентСоединения_(WebSocketClientConnectionParameters).md).   
Сетевые настройки.

<ИмяПользователяИнформационнойБазы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, от имени которого которым будет запущен сеанс, в котором будут вызваны обработчики событий. Если имя пользователя не задано, сеанс будет запущен от имени пользователя по умолчанию.

Возвращаемое значение:

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md).   
Если соединение с таким ключом уже существует, то возвращает ссылку на него.

Описание:

Асинхронный. Создает и открывает соединение WebSocket между клиентом и сервером.  
Полученная ссылку на соединение:

- можно использовать для отправки сообщений немедленно - они будут сохранены и отправлены после успешной установки соединения,
- можно проигнорировать - обнуление счетчика ссылок не приводит к закрытию соединения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
