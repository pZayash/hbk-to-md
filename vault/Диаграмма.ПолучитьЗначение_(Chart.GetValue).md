Диаграмма (Chart)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначение(<Точка>, <Серия>)

Параметры:

<Точка> (обязательный)

Тип: [Число](lang__def_Number.md), [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md).   
Индекс точки либо значение сама точка диаграммы.

<Серия> (обязательный)

Тип: [Число](lang__def_Number.md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Индекс серии либо сама серия диаграммы.

Возвращаемое значение:

Тип: [ЗначениеДиаграммы](ЗначениеДиаграммы_(ChartValue).md).   
Значение диаграммы в заданной точке и серии.

Описание:

Получает значение из заданной точки и серии диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ОбъемПродаж = Диаграмма.ПолучитьЗначение(Точка, Серия).Значение; |

См. также:

[Диаграмма](Диаграмма_(Chart).md), метод [УстановитьЗначение](Диаграмма.УстановитьЗначение_(Chart.SetValue).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
