# МенеджерВстроенныхПокупок.ПоддерживаютсяПокупки

МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПоддерживаютсяПокупки (PurchasesSupported)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоддерживаютсяПокупки()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, доступен ли в данное время механизм встроенных покупок.  
Возвращает [Истина](lang__def_BooleanTrue.md) в следующих случаях:   
{bullet }для Android - если установлена служба Google Play Services,  
для iOS - в проекте установлено разрешение на покупки,  
для ОС Windows Phone - всегда.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
