КоллекцияИндексовXBase (XBaseIndexesCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество индексов таблицы базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Количество индексов отлично от нуля, если открыт или создан индексный файл.

Пример:

|  |
| --- |
| xB = Новый xBase; xB.ОткрытьФайл("c:\test.dbf", "c:\test.cdx"); // Получим количество индексов  КоличествоИндексов = xB.Индексы.Количество(); |

Использование в версии:

Доступен, начиная с версии 8.0.
