СредстваПочты (MailTools)

ПоддерживаетсяОтправка (SendingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяОтправка()

Возвращаемое значение:

Тип: Булево.   
Возвращает Истина:

- на Android - всегда,
- на iOS - если на устройстве настроено приложение для отправки почты,
- на Windows Phone - на телефонах всегда. Никогда на планшетах.

Описание:

Определяет доступность отправки почты на данном устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваПочты](СредстваПочты.md), метод [Послать](СредстваПочты.Послать.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
