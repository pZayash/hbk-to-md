Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ТекстЗаполнения (FillingText)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: [Строка](lang__def_String.md).   
Текст, введенный в поле ввода, при открытии формы в режиме выбора.  
Если основное представление [ВВидеНаименования](ОсновноеПредставлениеВидаХарактеристики.ВВидеНаименования_(CharacteristicTypeMainPresentation.AsDescription).md) и реквизит [Наименование](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.Наименование_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.Description).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_ПланВидовХарактеристик.ВводПоСтроке_(MetadataObject_ChartOfCharacteristicTypes.InputByString).md), то значение присваивается реквизиту [Наименование](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.Наименование_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.Description).md).  
Если основное представление [ВВидеКода](ОсновноеПредставлениеВидаХарактеристики.ВВидеКода_(CharacteristicTypeMainPresentation.AsCode).md) и реквизит [Код](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.Код_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.Code).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_ПланВидовХарактеристик.ВводПоСтроке_(MetadataObject_ChartOfCharacteristicTypes.InputByString).md) и не установлен признак [Автонумерация](ОбъектМетаданных_ПланВидовХарактеристик.Автонумерация_(MetadataObject_ChartOfCharacteristicTypes.Autonumbering).md), то значение присваивается реквизиту [Код](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.Код_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name.Code).md).  
В противном случае присваивание не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.3.
