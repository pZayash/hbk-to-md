СправочникСсылка.<Имя справочника> (CatalogRef.<Catalog name>)

ИмяПредопределенныхДанных (PredefinedDataName)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя предопределенного элемента, как оно задано в конфигурации.  
Если указано пустое имя, элемент не будет предопределенным.  
Если указано недопустимое имя, будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Поле обладает особенностями сортировки (сортировка выполняется не по алфавиту).

Использование в версии:

Доступен, начиная с версии 8.3.3.
