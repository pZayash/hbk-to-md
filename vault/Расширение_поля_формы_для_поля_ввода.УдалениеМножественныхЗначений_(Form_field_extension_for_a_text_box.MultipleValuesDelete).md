Расширение поля формы для поля ввода (Form field extension for a text box)

УдалениеМножественныхЗначений (MultipleValuesDelete)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УдалениеМножественныхЗначений(<Идентификаторы>, <СтандартнаяОбработка>)

Параметры:

<Идентификаторы>

Тип: [Массив](Массив_(Array).md).   
Массив идентификаторов для объекта [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md) или [ДанныеФормыЭлементКоллекции](ДанныеФормыЭлементКоллекции_(FormDataCollectionItem).md). Эти идентификаторы не привязаны к позиции элемента в списке или таблице значений.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается перед вызовом обработчика [ПриИзменении](ПолеВвода.ПриИзменении_(TextBox.OnChange).md) при интерактивном удалении множественных значений:

- нажатием на кнопку удаления множественного значения;
- нажатием средней кнопкой мыши на множественном значении;
- нажатием клавиши Delete или Backspace в области ввода текста;
- при выборе пункта "Удалить" контекстного меню множественного значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
