# СредстваБуфераОбмена.ПоместитьДанныеАсинх

СредстваБуфераОбмена (ClipboardTools)

ПоместитьДанныеАсинх (PutDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПоместитьДанныеАсинх(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ЭлементБуфераОбмена](ЭлементБуфераОбмена_(ClipboardItem).md), [Массив](Массив_(Array).md).   
Элемент буфера обмена или массив элементов типа [ЭлементБуфераОбмена](ЭлементБуфераОбмена_(ClipboardItem).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Булево](lang__Булево_(Boolean).md).

Описание:

Асинхронная функция. Помещает переданные данные в буфер обмена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

При помещении в буфер обмена данных формата [HTML](СтандартныйФорматДанныхБуфераОбмена.HTML_(ClipboardDataStandardFormat.HTML).md) часть оформления HTML-текста может быть утеряна.  
  
В веб-клиенте:

- может появиться запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается [Обещание](Обещание_(Promise).md), результат которого – [Булево](lang__Булево_(Boolean).md).
- значение параметра <Формат> типа [Строка](lang__def_String.md) должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Использование в версии:

Доступен, начиная с версии 8.3.24.
