МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

Создать (Create)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Создать(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные файла формата \*.cfe.

Возвращаемое значение:

Тип: [РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md).   

Описание:

Создает новый объект типа [РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md).  
Для объектов, созданных с указанием параметра Данные:

- свойства [Версия](РасширениеКонфигурации.Версия_(ConfigurationExtension.Version).md), [Имя](РасширениеКонфигурации.Имя_(ConfigurationExtension.Name).md), [Назначение](РасширениеКонфигурации.Назначение_(ConfigurationExtension.Purpose).md), [ОсновныеРоли](РасширениеКонфигурации.ОсновныеРоли_(ConfigurationExtension.DefaultRoles).md), [Синоним](РасширениеКонфигурации.Синоним_(ConfigurationExtension.Synonym).md) и [ХешСумма](РасширениеКонфигурации.ХешСумма_(ConfigurationExtension.HashSum).md) имеют такое же значение, какое будет у расширения конфигурации, установленного из файла с идентичными двоичными данными,
- метод [ИзменяетСтруктуруДанных](РасширениеКонфигурации.ИзменяетСтруктуруДанных_(ConfigurationExtension.ModifiesDataStructure).md) возвращает [Истина](lang__def_BooleanTrue.md), если после установки расширение будет влиять на структуру данных конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
