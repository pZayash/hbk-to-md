ОбъектМетаданных: СервисИнтеграции (MetadataObject: IntegrationService)

Каналы (Channels)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Каналы сервиса интеграции, в которые отправляются и из которых получаются сообщения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Коллекция содержит объекты типа [ОбъектМетаданных: КаналСервисаИнтеграции](ОбъектМетаданных_КаналСервисаИнтеграции_(MetadataObject_IntegrationServiceChannel).md).

Использование в версии:

Доступен, начиная с версии 8.3.17.
