# КвалификаторыДаты

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Описание типов значений](Описание_типов_значений.md)

Доступен, начиная с версии 8.0.

Свойства:

[ЧастиДаты (DateFractions)](КвалификаторыДаты.ЧастиДаты_(DateQualifiers.DateFractions).md)  

Конструкторы:

[На основании описания даты](КвалификаторыДаты.По_умолчанию.md)  

Описание:

Используется для описания допустимых значений типа [Дата](lang__def_Date.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/core}. Имя типа XDTO: DateQualifiers.

Пример:

|  |
| --- |
| ПараметрыДаты = Новый КвалификаторыДаты(ЧастиДаты.Дата); |
