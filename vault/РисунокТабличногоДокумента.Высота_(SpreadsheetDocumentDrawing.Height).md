РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Высота (Height)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту рисунка (в миллиметрах).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
