# ПоляЭлементаБлокировкиДанных

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеЭлементаБлокировкиДанных](ПолеЭлементаБлокировкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элемент коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](ПоляЭлементаБлокировкиДанных.Количество.md)  
[Получить (Get)](ПоляЭлементаБлокировкиДанных.Получить.md)  

Описание:

Коллекция полей блокировки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементБлокировкиДанных](ЭлементБлокировкиДанных.md), свойство [Поля](ЭлементБлокировкиДанных.Поля.md)  

Использование в версии:

Доступен, начиная с версии 8.1.
