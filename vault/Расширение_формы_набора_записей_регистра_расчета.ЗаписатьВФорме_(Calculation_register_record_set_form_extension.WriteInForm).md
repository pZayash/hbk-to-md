# Расширение формы набора записей регистра расчета.ЗаписатьВФорме

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры расчета](Регистры_расчета.md) › [Расширение формы набора записей регистра расчета](Расширение_формы_набора_записей_регистра_расчета_(Calculation_register_record_set_form_extension).md) › [Методы](Расширение_формы_набора_записей_регистра_расчета__Методы.md)

Расширение формы набора записей регистра расчета (Calculation register record set form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает набор, открытый в форме.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
