ПроверкаЗаполнения (FillChecking)

ВыдаватьОшибку (ShowError)

Доступен, начиная с версии 8.2.

Описание:

Если для проверяемого значения метод [ЗначениеЗаполнено](Глобальный_контекст.ЗначениеЗаполнено_(Global_context.ValueIsFilled).md) вернет [Ложь](lang__def_BooleanFalse.md), значение будет считаться незаполненным и пользователю будет выдано соответствующее сообщение с указанием места исправления.

Использование в версии:

Доступен, начиная с версии 8.2.
