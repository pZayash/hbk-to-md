НастройкаПериода (PeriodSettings)

Редактировать (Edit)

Доступен, начиная с версии 8.0.

Синтаксис:

Редактировать()

Возвращаемое значение:

Тип: Булево.   
Истина - нажата кнопка "ОК" (выбор сделан); Ложь - в противном случае.

Описание:

Открывает диалог ввода периода в модальном режиме. В зависимости от значения свойств [РедактироватьКакИнтервал](НастройкаПериода.РедактироватьКакИнтервал.md) и [РедактироватьКакПериод](НастройкаПериода.РедактироватьКакПериод.md) диалог визуальной настройки имеет обе закладки или одну из них. Если использовано обе закладки активность той или иной из них задается свойством [ВариантНастройки](НастройкаПериода.ВариантНастройки.md).

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод; НастройкаПериода.Редактировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
