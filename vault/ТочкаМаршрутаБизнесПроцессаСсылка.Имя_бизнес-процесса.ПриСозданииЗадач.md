ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриСозданииЗадач (OnCreateTask)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриСозданииЗадач(<ТочкаМаршрута>, <ФормируемыеЗадачи>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса, на которой происходит создание задач.

<ФормируемыеЗадачи>

Тип: [Массив](Массив.md).   
Массив формируемых задач (объектов типа [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>).

<Отказ>

Тип: Булево.   
Признак отказа от добавления задачи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись задач произведена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает после выполнения стандартной или переопределенной в обработчике события [ПередСозданиемЗадач](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПередСозданиемЗадач.md) механики генерации списка задач. Обработчик этого события может "донастроить" автоматически сформированные задачи или удалить некоторые из них, а также добавить к ним новые задачи.   
Запись списка задач будет осуществлена после завершения обработчика.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md) и [ВложенныйБизнесПроцесс](ВидТочкиМаршрутаБизнесПроцесса.ВложенныйБизнесПроцесс.md).  
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода [Сообщить](Глобальный_контекст.Сообщить.md).

Использование в версии:

Доступен, начиная с версии 8.0.
