ДокументHTML (HTMLDocument)

Тело (Body)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ЭлементТелоHTML](ЭлементТелоHTML_(HTMLBodyElement).md).   
Содержит ссылку на тело HTML документа. Если документ содержит несколько тел, то указывает на первое.  
При указании в свойстве нового элемента, старый замещается.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
