Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередРазворачиваниемЭлементаИзмерения (BeforeExpandDimensionItem)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПередРазворачиваниемЭлементаИзмерения(<ЭлементИзмерения>, <ЗначенияИзмерений>, <Отказ>)

Параметры:

<ЭлементИзмерения>

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика.md).   
Элемент измерения, на кнопку разворачивания которого было произведено нажатие.

<ЗначенияИзмерений>

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие.md).   
Фиксированное соответствие, однозначно определяющее элемент измерения, на кнопку разворачивания которого было произведено нажатие. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение.md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md) элемента измерения.

<Отказ>

Тип: Булево.   
Если значение Истина, то разворачивание элемента не выполняется.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед разворачиванием элемента измерения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
