# ЭлементОтбораКомпоновкиДанных.ПравоеЗначение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Настройки компоновки данных](Настройки_компоновки_данных.md) › [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных_(DataCompositionFilterItem).md) › [Свойства](ЭлементОтбораКомпоновкиДанных__Свойства.md)

ЭлементОтбораКомпоновкиДанных (DataCompositionFilterItem)

ПравоеЗначение (RightValue)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Содержит правое значение элемента отбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Тип правого значения может быть полем, остальные варианты зависят от левого значения. Это может быть его тип левого значения, либо, если слева выбрано поле, может использоваться тип поля.

Использование в версии:

Доступен, начиная с версии 8.1.
