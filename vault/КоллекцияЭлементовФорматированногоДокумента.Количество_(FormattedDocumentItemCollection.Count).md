КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Количество (Count)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество элементов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
