ЗаписьТекста (TextWriter)

По имени файла

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЗаписьТекста(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <Дописывать>, <КонвертируемыйРазделительСтрок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя текстового файла для записи.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка записываемого текста (согласована с текстовым документом).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые кодировки:

- UTF-8
- UTF-16
- UTF-16BE
- UTF-16LE
- ISO-8859-1
- US-ASCII
- gb18030
- iso-8859-2
- iso-8859-3
- iso-8859-4
- iso-8859-5
- iso-8859-6
- iso-8859-7
- iso-8859-8
- iso-8859-9
- iso-8859-13
- iso-8859-15
- Shift\_JIS
- EUC-JP
- Big5
- Big5-HKSCS
- GBK
- GB2312
- GB\_2312-80
- EUC-KR
- KSC\_5601
- windows-949
- windows-874
- cp866
- KOI8-R
- KOI8-U
- windows-1250
- windows-1251
- windows-1252
- windows-1253
- windows-1254
- windows-1255
- windows-1256
- windows-1257
- windows-1258
- macintosh
- x-mac-cyrillic
- ISO-2022-JP
- ISO-2022-KR
- ISO-2022-CN
- ISO-2022-CN-EXT
- HZ-GB-2312

  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8.md).

<РазделительСтрок> (необязательный)

Тип: Строка.   
Символ разделителя строк в записываемом файле.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<Дописывать> (необязательный)

Тип: Булево.   
Если параметр принимает значение Истина, то содержимое файла сохраняется, если файл уже существует. Если файла не существует, создается новый с именем, указанным в первом параметре конструктора.  
Значение по умолчанию: Ложь.

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

Описание:

Открывает текстовый файл для записи. Аналогичен методу [Открыть](ЗаписьТекста.Открыть.md).

Пример:

|  |
| --- |
| ДлинныйТекст = Новый ЗаписьТекста("c:\temp\file.txt", КодировкаТекста.UTF16); |

См. также:

[ЗаписьТекста](ЗаписьТекста.md), метод [Открыть](ЗаписьТекста.Открыть.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
