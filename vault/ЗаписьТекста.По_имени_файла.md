ЗаписьТекста (TextWriter)

По имени файла

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЗаписьТекста(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <Дописывать>, <КонвертируемыйРазделительСтрок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя текстового файла для записи.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка записываемого текста (согласована с текстовым документом).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые кодировки:

- UTF-8
- UTF-16
- UTF-16BE
- UTF-16LE
- ISO-8859-1
- US-ASCII
- gb18030
- iso-8859-2
- iso-8859-3
- iso-8859-4
- iso-8859-5
- iso-8859-6
- iso-8859-7
- iso-8859-8
- iso-8859-9
- iso-8859-13
- iso-8859-15
- Shift\_JIS
- EUC-JP
- Big5
- Big5-HKSCS
- GBK
- GB2312
- GB\_2312-80
- EUC-KR
- KSC\_5601
- windows-949
- windows-874
- cp866
- KOI8-R
- KOI8-U
- windows-1250
- windows-1251
- windows-1252
- windows-1253
- windows-1254

- x-mac-cyrillic
- ISO-2022-JP
- ISO-2022-KR
- ISO-2022-CN
- ISO-2022-CN-EXT
- HZ-GB-2312

  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Символ разделителя строк в записываемом файле.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<Дописывать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр принимает значение [Истина](lang__def_BooleanTrue.md), то содержимое файла сохраняется, если файл уже существует. Если файла не существует, создается новый с именем, указанным в первом параметре конструктора.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание:

Открывает текстовый файл для записи. Аналогичен методу [Открыть](ЗаписьТекста.Открыть_(TextWriter.Open).md).

Пример:

|  |
| --- |
| ДлинныйТекст = Новый ЗаписьТекста("c:\temp\file.txt", КодировкаТекста.UTF16); |

См. также:

[ЗаписьТекста](ЗаписьТекста_(TextWriter).md), метод [Открыть](ЗаписьТекста.Открыть_(TextWriter.Open).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
