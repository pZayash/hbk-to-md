Описание:

Действие, возможное при обработке расшифровки. Используется для определения доступных действий и для информирования о том, какое действие выполнено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ОбработкаРасшифровкиКомпоновкиДанных](ОбработкаРасшифровкиКомпоновкиДанных_(DataCompositionDetailsProcess).md), метод [Выполнить](ОбработкаРасшифровкиКомпоновкиДанных.Выполнить_(DataCompositionDetailsProcess.Execute).md)

Доступен, начиная с версии 8.1.
