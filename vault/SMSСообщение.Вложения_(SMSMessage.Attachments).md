SMSСообщение (SMSMessage)

Вложения (Attachments)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит вложения в SMS-сообщения.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Элементом массива является объект типа [MMSВложение](MMSВложение_(MMSAttachment).md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
