Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Структура, содержащая параметры записи. Для формы бизнес-процесса существует один предопределенный параметр Старт. При отработке команд "Старт", "Стартовать и закрыть" он устанавливается в [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
