Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfileCOMClass (CreateSecurityProfileCOMClass)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfileCOMClass()

Возвращаемое значение:

Тип: [COM-класс](COM-класс_(ISecurityProfileCOMClass).md).   

Описание:

Создает новый разрешенный [COM-класс](COM-класс_(ISecurityProfileCOMClass).md) для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
