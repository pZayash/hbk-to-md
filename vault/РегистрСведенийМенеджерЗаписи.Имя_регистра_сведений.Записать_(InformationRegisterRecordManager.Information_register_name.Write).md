# РегистрСведенийМенеджерЗаписи.<Имя регистра сведений>.Записать

РегистрСведенийМенеджерЗаписи.<Имя регистра сведений> (InformationRegisterRecordManager.<Information register name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим замещения существующей записи с тем же набором значений измерений регистра и с тем же периодом (для периодических регистров). [Истина](lang__def_BooleanTrue.md) - если запись регистра сведений с таким набором существует, то выполняется замещение; [Ложь](lang__def_BooleanFalse.md) - запись будет дописана к уже существующим в информационной базе записям.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает в базу данных запись регистра сведений с текущими значениями свойств.   
  
С помощью параметра <Замещать> регулируется, будет ли метод замещать запись, если в момент выполнения метода существует другая (исключая считанную) запись с такими же значениями измерений регистра и с таким же периодом для периодических регистров.  
  
Если запись регистра сведений была предварительно считана из базы данных, то возможны два варианта поведения:

- В режиме совместимости конфигурации [Версия8\_3\_26](РежимСовместимости.Версия8_3_26_(CompatibilityMode.Version8_3_26).md) и ниже считанная запись удаляется, а затем производится запись новой записи.
- Если режим совместимости не используется или выше [Версия8\_3\_26](РежимСовместимости.Версия8_3_26_(CompatibilityMode.Version8_3_26).md), выполняется сравнение значений ключевых полей (период, измерения). При совпадении значений ключевых полей считанная запись замещается на новую. Если значения хотя бы одного поля отличаются, то считанная запись удаляется, а затем выполняется запись новой записи.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу запись очищается.
