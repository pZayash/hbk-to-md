ФайловыйПоток (FileStream)

НачатьПолучениеРазмера (BeginGetSize)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьПолучениеРазмера(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - тип Число. Размер данных в байтах.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Возвращает размер данных в байтах.  
Если свойство [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции.md) установлено в значение Ложь, то при вызове данного метода будет сгенерировано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

См. также:

[Поток](Поток.md), метод [РазмерАсинх](Поток.РазмерАсинх.md)  
[Поток](Поток.md), метод [Размер](Поток.Размер.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
