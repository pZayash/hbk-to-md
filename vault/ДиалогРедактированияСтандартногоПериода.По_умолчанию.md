ДиалогРедактированияСтандартногоПериода (StandardPeriodEditDialog)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ДиалогРедактированияСтандартногоПериода()

Описание:

Создает неинициализированный объект диалога.  
Перед открытием диалога, если необходимо, можно установить требуемый стандартный период в свойстве [Период](ДиалогРедактированияСтандартногоПериода.Период.md).

Использование в версии:

Доступен, начиная с версии 8.2.
