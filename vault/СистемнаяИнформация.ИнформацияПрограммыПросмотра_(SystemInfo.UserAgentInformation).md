СистемнаяИнформация (SystemInfo)

ИнформацияПрограммыПросмотра (UserAgentInformation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Информация о веб-браузере (User-Agent).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Во всех режимах, кроме веб-клиента, содержит пустую строку.

Использование в версии:

Доступен, начиная с версии 8.2.
