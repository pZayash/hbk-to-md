ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceContentSettingsItem)

Имя (Name)

Доступен, начиная с версии 8.3.4.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя панели.  
Имена стандартных панелей:

- ПанельРазделов (SectionsPanel),
- ПанельФункцийТекущегоРаздела (CurrentSectionFunctionsPanel),
- ПанельИзбранного (FavoritesPanel),
- ПанельИстории (HistoryPanel),
- ПанельОткрытых (OpenItemsPanel).

Доступность:

Сервер, толстый клиент.

Примечание:

В режиме совместимости [Версия8\_3\_14](РежимСовместимости.Версия8_3_14.md) и ниже добавление элемента "ПанельИнструментов"(ToolsPanel) в группу панелей игнорируется.  
В режиме совместимости выше [Версия8\_3\_14](РежимСовместимости.Версия8_3_14.md) создание элемента "ПанельИнструментов"(ToolsPanel) не допускается.

Использование в версии:

Доступен, начиная с версии 8.3.4.
