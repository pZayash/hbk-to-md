МаксимумСерий (MaxSeries)

Процент (Percent)

Доступен, начиная с версии 8.0.

Описание:

Отображаются только те серии, чей максимум больше или равен заданному проценту от абсолютного. Для этого для каждой серии находится максимальное абсолютное значение и абсолютный максимум по всем сериям. Процент определяется значением свойства [МаксимумСерийПроцент](Диаграмма.МаксимумСерийПроцент_(Chart.MaxSeriesPercent).md) объекта [Диаграмма](Диаграмма_(Chart).md).

Использование в версии:

Доступен, начиная с версии 8.0.
