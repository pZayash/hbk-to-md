ТелоГруппировкиТаблицыМакетаКомпоновкиДанных (DataCompositionTemplateTableGroupBody)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип добавляемого элемента. Элементами коллекции могут быть [МакетГруппировкиТаблицыМакетаКомпоновкиДанных](МакетГруппировкиТаблицыМакетаКомпоновкиДанных.md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных.md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных.md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных.md).

Возвращаемое значение:

Тип: [ТелоГруппировкиТаблицыМакетаКомпоновкиДанных](ТелоГруппировкиТаблицыМакетаКомпоновкиДанных.md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных.md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных.md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод возвращает Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
