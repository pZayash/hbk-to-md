# Расширение таблицы формы для таблицы значений

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Таблица формы](Таблица_формы.md)

Доступен, начиная с версии 8.2.

Свойства:

[ОтборСтрок (RowFilter)](Расширение_таблицы_формы_для_таблицы_значений.ОтборСтрок_(Form_table_extension_for_value_table.RowFilter).md)  

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы_(FormTable).md), применимые для формы таблицы значений.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель_(FormTable.CurrentParent).md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки_(FormTable.SelectedRows).md), используется идентификатор строки.  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные_(FormTable.CurrentData).md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки_(FormTable.RowData).md) возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Таблица1 = ЭлементыФормы.ТаблицаОстатков; Отбор = Новый Структура; Отбор.Вставить("НашДолг", 10); Таблица1.ОтборСтрок = Новый ФиксированнаяСтруктура(Отбор); |

Использование в версии:

Доступен, начиная с версии 8.2.
