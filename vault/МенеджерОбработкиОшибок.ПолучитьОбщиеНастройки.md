МенеджерОбработкиОшибок (ErrorProcessingManager)

ПолучитьОбщиеНастройки (GetCommonSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьОбщиеНастройки()

Возвращаемое значение:

Тип: [НастройкиОбработкиОшибок](НастройкиОбработкиОшибок.md).   

Описание:

Возвращает общие настройки обработки ошибки. Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.17.
