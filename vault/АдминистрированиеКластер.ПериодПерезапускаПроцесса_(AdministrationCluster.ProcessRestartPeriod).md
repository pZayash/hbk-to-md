АдминистрированиеКластер (AdministrationCluster)

ПериодПерезапускаПроцесса (ProcessRestartPeriod)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Интервал времени (в секундах) после запуска процессов, по истечении которого будет выполнен перезапуск рабочих процессов кластера.   
Если рабочий процесс активен в течение этого времени, то будет выполнен запуск нового процесса, а старый процесс будет выключен. Выключенный процесс будет обслуживать соединения, установленные ранее, не устанавливая при том новых соединений.   
Если значение равно 0, то перезапуск рабочих процессов не производится.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
