ОбластьЗаголовкаДендрограммы (DendrogramTitleArea)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если свойство [ПрозрачныйФон](ОбластьЗаголовкаДендрограммы.ПрозрачныйФон_(DendrogramTitleArea.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
