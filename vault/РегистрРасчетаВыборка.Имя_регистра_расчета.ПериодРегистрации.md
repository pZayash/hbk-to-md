РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

ПериодРегистрации (RegistrationPeriod)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Дата.   
Период регистрации - дата, которая принимает дискретные значения в зависимости от периодичности регистра расчета.  
Например, если регистр помесячный, то дата всегда начало какого-нибудь месяца.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
