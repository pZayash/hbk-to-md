ОписанияСистемЛинейныхУравнений (SystemDescriptions)

Добавить (Add)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md).   
У нового созданного объекта [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md) свойства [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях_(SystemOfLinearEquationsDescription.FactorsInRelationsColumn).md) и [КолонкаКоэффициентовВУзлах](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВУзлах_(SystemOfLinearEquationsDescription.FactorsInNodesColumn).md) имеют значение [Неопределено](lang__def_Undefined.md).

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
