Описание:

Коллекция элементов оформления компоновки данных.   
Для оформления используются следующие элементы оформления:

- АвтоОтступ (AutoIndent) - тип значения Число;
- ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение.md);
- ВертикальноеПоложениеКартинки (PictureVerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение.md). Размещение картинки по вертикали;
- ВесовойКоэффициентШирины (WidthWeightFactor) - тип значения Число. Весовой коэффициент для автоматического расчета ширины колонки;
- Видимость (Visible) - Булево. Используется только для поля, расположенного в таблице (колонки);
- ВыводитьКартинку (PictureOutput) - тип значения [ТипВыводаКартинкиКомпоновкиДанных](ТипВыводаКартинкиКомпоновкиДанных.md). Способ вывода картинки;
- ВыделятьОтрицательные (MarkNegatives) - тип значения Булево;
- ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md);
- ГоризонтальноеПоложениеКартинки (PictureHorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md). Размещение картинки по горизонтали;
- Доступность (Enabled) - Булево. Используется только для поля, расположенного в таблице (колонки);
- ИндикаторВДиаграмме (IndicatorInChart) - тип значения Булево. Указывает, что серия, полученная из данной группировки является индикатором. Используется только для серий диаграммы;
- ИнформационнаяЛинияВДиаграмме (ReferenceLineInChart) - тип значения [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы.md). Информационная линия в диаграмме;
- ИнформационныйИнтервалВДиаграмме (ReferenceBandInChart) - тип значения [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы.md). Информационный интервал в диаграмме;
- ИспользуемаяОсьЗначенийВДиаграмме (UsedValueAxisInChart) - тип значения [ИспользуемаяОсьЗначенийДиаграммы](ИспользуемаяОсьЗначенийДиаграммы.md) Настраивает использование дополнительной оси значений в диаграмме;
- Картинка (Picture) - тип значения [Картинка](Картинка.md);
- ЛинииТрендаВДиаграмме (ChartTrendlines) - тип значения [ЛинииТрендаДиаграммы](ЛинииТрендаДиаграммы.md). Используется только для серий диаграммы;
- ЛинияВДиаграмме (LineInChart) - тип значения [Линия](Линия.md). Тип линии серии, полученной из данной группировки. Используется только для серий диаграммы;
- МаксимальнаяВысота (MaxHeight) - тип значения Число;
- МаксимальнаяШирина (MaxWidth) - тип значения Число;
- МаркерВДиаграмме (MarkerInChart) - тип значения [ТипМаркераДиаграммы](ТипМаркераДиаграммы.md). Тип маркера серии, полученной из данной группировки. Используется только для серий диаграммы;
- МинимальнаяВысота (MinHeight) - тип значения Число;
- МинимальнаяШирина (MinWidth) - тип значения Число;
- ОриентацияТекста (TextOrientation) - тип значения Число;
- ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную - тип значения Булево;
- Отображать (Show) - Булево. Используется только для поля, расположенного в таблице (колонки);
- ОтображатьГрафическоеПредставлениеДанныхВДиаграмме (ShowGraphicalRepresentationOfDataOnChart) тип значения [ОтображениеВДиаграмме](ОтображениеВДиаграмме.md). Используется только для серий диаграммы;
- ОтображатьГрафическоеПредставлениеДанныхВЛегендеДиаграммы (ShowGraphicalRepresentationOfDataInChartLegend) - тип значения [ОтображениеВЛегендеДиаграммы](ОтображениеВЛегендеДиаграммы.md). Используется только для серий диаграммы;
- Отступ (Indent) - тип значения Число;
- ОтступСверху (TopMargin) - тип значения Число;
- ОтступСлева (LeftMargin) - тип значения Число;
- ОтступСнизу (BottomMargin) - тип значения Число.
- ОтступСправа (RightMargin) - тип значения Число;
- ПоложениеТекстаОтносительноКартинки (TextPositionRelativeToPicture) - тип значения [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки.md). Положение текста относительно картинки;
- РазделительМножественныхЗначений (MultipleValuesSeparator) - тип значения Строка. Содержит разделитель, который вставляется между значениями множественных характеристик. Если параметр не задан, то используется запятая с пробелом;
- РазмерКартинки (PictureSize) - тип значения [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки.md). Положение текста относительно картинки;
- Размещение (Placement) - размещение текста - тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных.md);
- РастягиватьПоГоризонтали (HorizontalStretch) - тип значения Булево. Указывает, что ширина текущей колонки рассчитывается автоматически;
- СтильГраницы (BorderStyle) - тип значения [Линия](Линия.md);
- СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия.md);
- СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия.md);
- СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия.md);
- СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия.md);
- Текст (Text) - тип значения Строка или [ФорматированнаяСтрока](ФорматированнаяСтрока.md);
- ТипГрафическогоПредставленияВДиаграмме (VisualTypeInChart) - тип значения [ТипГрафическогоПредставленияСерииДиаграммы](ТипГрафическогоПредставленияСерииДиаграммы.md).Тип графического представления серии в диаграмме;
- ТипНакопленияВДиаграмме (StackTypeInChart) - тип значения [ТипНакопленияСерииДиаграммы](ТипНакопленияСерииДиаграммы.md). Тип накопления серии, отличный от типа диаграммы. Группа накопления серии создаётся автоматически, на основе элемента условного оформления. Разные элементы условного оформления будут имеют разные группы накопления;
- ТипОтметки (MarkType) - тип значения [ТипОтметкиЯчейкиТаблицы](objects__catalog2__catalog277__TableCellMarkType.md);
- ТолькоПросмотр (ReadOnly) - Булево. Используется только для поля, расположенного в таблице (колонки);
- Формат (Format) - тип значения Строка (форматная строка);
- ЦветВДиаграмме (ColorInChart) - тип значения [Цвет](Цвет.md). Используется только для серии диаграммы;
- ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет.md);
- ЦветОтметки (MarkColor) - тип значения [Цвет](Цвет.md);
- ЦветТекста (TextColor) - тип значения [Цвет](Цвет.md);
- ЦветФона (BackColor) - тип значения [Цвет](Цвет.md);
- Шрифт (Font) - тип значения [Шрифт](Шрифт.md);

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: Appearance.
