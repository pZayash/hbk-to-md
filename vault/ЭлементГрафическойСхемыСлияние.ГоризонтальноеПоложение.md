ЭлементГрафическойСхемыСлияние (GraphicalSchemaItemJoin)

ГоризонтальноеПоложение (HorizontalAlign)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md).   
Определяет способ горизонтального расположения текста элемента графической схемы.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Допустимыми являются только значения [Лево](ГоризонтальноеПоложение.Лево.md), [Право](ГоризонтальноеПоложение.Право.md) или [Центр](ГоризонтальноеПоложение.Центр.md).

Использование в версии:

Доступен, начиная с версии 8.0.
