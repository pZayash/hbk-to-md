ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

Ключ (Key)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Ключ интеграции системы взаимодействия.  
Если не пустая строка, то должен быть уникальным.  
Уникальность поддерживается в пределах информационной базы в разрезе всех независимых разделителей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
