# ЭлементСоставаКопииБазыДанных.Отбор

ЭлементСоставаКопииБазыДанных (DatabaseCopyContentItem)

Отбор (Filter)

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Поддерживается только для элементов состава копии, у которых свойство [Метаданные](ЭлементСоставаКопииБазыДанных.Метаданные_(DatabaseCopyContentItem.Metadata).md) содержит периодический регистр свдеений или регистр накопления.  
Отбор содержит один элемент "Период". Для элемента отбора доступны следующие виды сравнения:

- [Больше](ВидСравнения.Больше_(ComparisonType.Greater).md),
- [БольшеИлиРавно](ВидСравнения.БольшеИлиРавно_(ComparisonType.GreaterOrEqual).md),
- [Интервал](ВидСравнения.Интервал_(ComparisonType.Interval).md),
- [ИнтервалВключаяГраницы](ВидСравнения.ИнтервалВключаяГраницы_(ComparisonType.IntervalIncludingBounds).md),
- [ИнтервалВключаяНачало](ВидСравнения.ИнтервалВключаяНачало_(ComparisonType.IntervalIncludingLowerBound).md),
- [ИнтервалВключаяОкончание](ВидСравнения.ИнтервалВключаяОкончание_(ComparisonType.IntervalIncludingUpperBound).md),
- [Меньше](ВидСравнения.Меньше_(ComparisonType.Less).md),
- [МеньшеИлиРавно](ВидСравнения.МеньшеИлиРавно_(ComparisonType.LessOrEqual).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15 (в режиме совместимости с версией 8.3.14 и последующими).
