ДиаграммаГанта (GanttChart)

ОтображатьПустыеЗначения (ShowEmptyValues)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения пустых (не содержащих ни одного интервала) значений.  
[Истина](lang__def_BooleanTrue.md) - отображаются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
