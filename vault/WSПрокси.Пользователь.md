WSПрокси (WSProxy)

Пользователь (User)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Пользователь, вызывающий веб-сервис. Используется для аутентификации пользователя при вызове веб-сервиса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
