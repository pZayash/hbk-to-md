ЗаписьPDF (PDFWriter)

ЗаписатьПодписьАсинх (WriteSignatureAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьПодписьАсинх(<МенеджерКриптографии>, <ПредставлениеПодписи>, <ОписаниеПодписи>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

<ПредставлениеПодписи> (необязательный)

Тип: [Картинка](Картинка_(Picture).md), [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Объект, представляющий собой оформление (штамп) электронной подписи PDF-документа. Если параметр не указан, оформление не создается (подпись не отображается).

<ОписаниеПодписи> (обязательный)

Тип: [ОписаниеПодписиPDF](ОписаниеПодписиPDF_(PDFSignatureDescription).md).   
Объект, содержащий описание электронной подписи PDF-документа.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Создает присоединенную к PDF-документу подпись. Располагает оформление подписи (штамп) на указанной странице по указанным координатам.

Доступность:

Тонкий клиент, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [ЗаписатьПодпись](ЗаписьPDF.ЗаписатьПодпись_(PDFWriter.WriteSignature).md)  
[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [НачатьЗаписьПодписи](ЗаписьPDF.НачатьЗаписьПодписи_(PDFWriter.BeginWriteSignature).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
