ТипФайлаТабличногоДокумента (SpreadsheetDocumentFileType)

XLS (XLS)

Доступен, начиная с версии 8.0.

Описание:

Файл в формате MS Excel. Сохранение в данном формате выполняет сохранение в формате MS Excel, используемом по умолчанию (эквивалентно [XLS97](ТипФайлаТабличногоДокумента.XLS97.md)).

Использование в версии:

Доступен, начиная с версии 8.0.
