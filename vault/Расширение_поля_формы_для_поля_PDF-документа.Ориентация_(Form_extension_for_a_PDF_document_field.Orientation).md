# Расширение поля формы для поля PDF-документа.Ориентация

Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

Ориентация (Orientation)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет угол поворота (в градусах) PDF-документа. Значение угла поворота должно быть кратно 90 градусам. При установке значения, не кратного 90, генерируется исключение. Положительное значение указывает поворот против часовой стрелки, отрицательное – по часовой стрелке.  
Значение по умолчанию: 0.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
