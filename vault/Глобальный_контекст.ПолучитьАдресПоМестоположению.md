Глобальный контекст (Global context)

ПолучитьАдресПоМестоположению (GetAddressByLocation)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьАдресПоМестоположению(<Местоположение>)

Параметры:

<Местоположение> (обязательный)

Тип: [ГеографическиеКоординаты](ГеографическиеКоординаты.md).   
Местоположение, адрес которого нужно получить.

Возвращаемое значение:

Тип: [ДанныеАдреса](ДанныеАдреса.md), Неопределено.   
Адрес местоположения. Его состав может различаться в разных государствах.  
Для России определены поля:

- Страна (Country);
- Регион (State);
- Город (City);
- Улица (Street);
- Дом (House);
- Индекс (ZipCode).

Неопределено - адрес не получен.

Описание:

Получает адрес по географическим координатам.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
