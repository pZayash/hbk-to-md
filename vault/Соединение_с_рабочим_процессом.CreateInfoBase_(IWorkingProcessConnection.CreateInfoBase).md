# Соединение с рабочим процессом.CreateInfoBase

Соединение с рабочим процессом (IWorkingProcessConnection)

CreateInfoBase (CreateInfoBase)

Доступен, начиная с версии 8.1.

Синтаксис:

CreateInfoBase(<ИнформационнаяБаза>, <Режим>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база_(IInfoBaseInfo).md).   
Информационная база. Все свойства, необходимые для создания информационной базы ([Name](Информационная_база.Name_(IInfoBaseInfo.Name).md), [dbServerName](Информационная_база.dbServerName_(IInfoBaseInfo.dbServerName).md), [dbName](Информационная_база.dbName_(IInfoBaseInfo.dbName).md), [dbUser](Информационная_база.dbUser_(IInfoBaseInfo.dbUser).md), [dbPassword](Информационная_база.dbPassword_(IInfoBaseInfo.dbPassword).md), [Locale](Информационная_база.Locale_(IInfoBaseInfo.Locale).md), [[DateOffset](Информационная_база.DateOffset_(IInfoBaseInfo.DateOffset).md)]) должны быть заполнены.

<Режим> (обязательный)

Тип: [Число](lang__def_Number.md).   
Режим создания информационной базы:  
0 - при создании информационной базы базу данных не создавать;  
1 - при создании информационной базы создавать базу данных.

Возвращаемое значение:

Тип: [Информационная база](Информационная_база_(IInfoBaseInfo).md).   

Описание:

Создает информационную базу с заданными параметрами. Требуется аутентификация администратора кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
