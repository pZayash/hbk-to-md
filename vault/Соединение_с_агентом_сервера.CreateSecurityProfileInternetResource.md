Соединение с агентом сервера (IServerAgentConnection)

CreateSecurityProfileInternetResource (CreateSecurityProfileInternetResource)

Доступен, начиная с версии 8.3.3.

Синтаксис:

CreateSecurityProfileInternetResource()

Возвращаемое значение:

Тип: [Ресурс интернет](Ресурс_интернет.md).   

Описание:

Создает новый разрешенный ресурс Интернета для его последующей настройки и регистрации. Все свойства имеют значения по умолчанию для соответствующих типов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
