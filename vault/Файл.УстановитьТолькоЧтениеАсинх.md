Файл (File)

УстановитьТолькоЧтениеАсинх (SetReadOnlyAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьТолькоЧтениеАсинх(<ТолькоЧтение>)

Параметры:

<ТолькоЧтение> (обязательный)

Тип: Булево.   
Истина - установить атрибут "Только чтение", Ложь - снять атрибут "Только чтение".

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Устанавливает значение атрибута файла "ReadOnly". Применима только к файлам или каталогам, которые реально существуют.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.
