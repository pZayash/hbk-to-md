# РегистрСведенийНаборЗаписей.<Имя регистра сведений>.Сдвинуть

РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Запись>, <Смещение>)

Параметры:

<Запись> (обязательный)

Тип: [Число](lang__def_Number.md), [РегистрСведенийЗапись.](РегистрСведенийЗапись.Имя_регистра_сведений_(InformationRegisterRecord.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийЗапись.Имя_регистра_сведений_(InformationRegisterRecord.Information_register_name).md)>.   
Индекс сдвигаемой записи либо сама запись.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца списка записей набора (увеличение индекса), отрицательное - к началу.

Описание:

Сдвигает запись в наборе на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Доступен только для набора записей регистра сведений, в котором записи подчинены регистратору.

Использование в версии:

Доступен, начиная с версии 8.0.
