Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

ПериодАвтоОбновления (AutoRefreshPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Задает период (в секундах), с которым будет производиться автоматическое обновление списка. Не может быть меньше 5 секунд. Если свойство [АвтоОбновление](Расширение_табличного_поля_списка_записей_регистра_сведений.АвтоОбновление.md) установлено как Ложь, то обновление не производится.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
