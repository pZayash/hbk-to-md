# ФасетПеречисленияXS.ЭлементDOM

ФасетПеречисленияXS (XSEnumerationFacet)

ЭлементDOM (DOMElement)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ЭлементDOM](ЭлементDOM_(DOMElement).md).   
Элемент DOM, который соответствует данному компоненту схемы XML.  
Синхронизация значений свойств компонента схемы XML и структуры элемента DOM обеспечивается вызовом метода [ФасетПеречисленияXS.ОбновитьЭлементDOM](ФасетПеречисленияXS.ОбновитьЭлементDOM_(XSEnumerationFacet.UpdateDOMElement).md).  
Перед первым вызовом данного метода значение данного свойства может быть [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
