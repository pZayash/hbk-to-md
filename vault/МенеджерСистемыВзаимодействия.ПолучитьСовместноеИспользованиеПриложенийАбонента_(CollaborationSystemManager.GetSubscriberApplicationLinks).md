МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьСовместноеИспользованиеПриложенийАбонента (GetSubscriberApplicationLinks)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьСовместноеИспользованиеПриложенийАбонента()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [СовместноеИспользованиеПриложенийСистемыВзаимодействия](СовместноеИспользованиеПриложенийСистемыВзаимодействия_(CollaborationSystemApplicationLinks).md).  
В свойстве [Приложения](СовместноеИспользованиеПриложенийСистемыВзаимодействия.Приложения_(CollaborationSystemApplicationLinks.Applications).md) объекта [СовместноеИспользованиеПриложенийСистемыВзаимодействия](СовместноеИспользованиеПриложенийСистемыВзаимодействия_(CollaborationSystemApplicationLinks).md) всегда возвращается массив из двух элементов.

Описание:

Возвращает описание совместного использования приложений системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
