ВидКлючаДинамическогоСписка (DynamicListKeyType)

КлючСтроки (RowKey)

Доступен, начиная с версии 8.3.14.

Описание:

Используется, если у динамического списка несколько ключевых полей.  
В качестве значений свойства ТекущаяСтрока используются значения типа [КлючСтрокиДинамическогоСписка](КлючСтрокиДинамическогоСписка.md).

Использование в версии:

Доступен, начиная с версии 8.3.14.
