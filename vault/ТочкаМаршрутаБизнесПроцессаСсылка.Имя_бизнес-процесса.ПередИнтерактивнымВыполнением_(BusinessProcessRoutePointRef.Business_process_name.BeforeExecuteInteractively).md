# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередИнтерактивнымВыполнением

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередИнтерактивнымВыполнением (BeforeExecuteInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИнтерактивнымВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md), в которой выполняется обработка метода [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно_(TaskObject.Task_name.ExecuteTaskInteractively).md) у задачи.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Выполняемая задача.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то выполнение задачи произведено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при вызове метода [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно_(TaskObject.Task_name.ExecuteTaskInteractively).md) у задачи. Этот обработчик является единственным местом в котором можно разместить интерактивное взаимодействие с пользователем перед переходом выполнения бизнес-процесса дальше по маршруту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md).  
Допускаются интерактивные вызовы (открытие форм, предупреждений).  
В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) не вызывается.

Использование в версии:

Доступен, начиная с версии 8.0.
