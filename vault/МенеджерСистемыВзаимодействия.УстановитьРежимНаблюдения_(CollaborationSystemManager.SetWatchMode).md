# МенеджерСистемыВзаимодействия.УстановитьРежимНаблюдения

МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьРежимНаблюдения (SetWatchMode)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьРежимНаблюдения(<Обсуждение>, <РежимНаблюдения>, <Пользователь>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения.

<РежимНаблюдения> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - при появлении новых сообщений в обсуждения генерируется оповещение.  
[Ложь](lang__def_BooleanFalse.md) - оповещение не генерируется.

<Пользователь> (необязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Если параметр задан, то устанавливается режим наблюдения для соответствующего пользователя.  
Если передано значение, отличное от идентификатора текущего пользователя, и не установлен привилегированный режим, то генерируется исключение.

Описание:

Устанавливает режим получения оповещений по контекстному обсуждению.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
