# Глобальный контекст.СоздатьДвоичныеДанныеИзФайлаАсинх

Глобальный контекст (Global context)

СоздатьДвоичныеДанныеИзФайлаАсинх (CreateBinaryDataFromFileAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СоздатьДвоичныеДанныеИзФайлаАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, из которого надо создать [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), содержащий данные из файла.

Описание:

Асинхронная функция. Создает двоичные данные из файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [НачатьСозданиеДвоичныхДанныхИзФайла](Глобальный_контекст.НачатьСозданиеДвоичныхДанныхИзФайла_(Global_context.BeginCreateBinaryDataFromFile).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
