РежимУправленияБлокировкойДанных (DataLockControlMode)

Автоматический (Automatic)

Доступен, начиная с версии 8.1.

Описание:

При работе транзакции в этом режиме система автоматически обеспечивает целостность данных, то есть устанавливает все необходимые блокировки.

Использование в версии:

Доступен, начиная с версии 8.1.
