Глобальный контекст (Global context)

ПриНачалеРаботыСистемы (OnStart)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриНачалеРаботыСистемы()

Описание:

Возникает при запуске в режиме 1С:Предприятие в режиме внешнего соединения. В процедуре-обработчике могут быть выполнены различные действия, необходимые при запуске программы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), событие [ПередНачаломРаботыСистемы](Глобальный_контекст.Глобальный_контекст.ПередНачаломРаботыСистемы.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
