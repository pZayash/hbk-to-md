ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Доступно для записи только у нового обсуждения и требует наличия права "Администрирование данных" (или привилегированного режима).  
Если Истина, то в обсуждении интерактивно нельзя:

- создавать сообщения,
- изменять сообщения,
- удалять сообщения,
- изменять тему обсуждения,
- покидать обсуждение,
- изменять состав участников обсуждения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
