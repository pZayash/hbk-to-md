КнопкаФормы (FormButton)

ПоложениеВКоманднойПанели (LocationInCommandBar)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеКнопкиВКоманднойПанели](ПоложениеКнопкиВКоманднойПанели_(ButtonLocationInCommandBar).md).   
Определяет расположение кнопки в командной панели и/или в подменю "Еще".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.15.
