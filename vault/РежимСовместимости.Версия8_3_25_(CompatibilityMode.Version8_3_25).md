РежимСовместимости (CompatibilityMode)

Версия8\_3\_25 (Version8\_3\_25)

Доступен, начиная с версии 8.3.26.

Описание:

Усовершенствования версии 8.3.26 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.25.

Использование в версии:

Доступен, начиная с версии 8.3.26.
