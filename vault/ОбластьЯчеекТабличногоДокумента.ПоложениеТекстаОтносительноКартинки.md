ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ПоложениеТекстаОтносительноКартинки (TextPositionRelativeToPicture)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки.md).   
Определяет расположение текста в ячейке относительно картинки. Если свойство [РазмерКартинки](ОбластьЯчеекТабличногоДокумента.РазмерКартинки.md) содержит значение [Черепица](РазмерКартинки.Черепица.md) или [Растянуть](РазмерКартинки.Растянуть.md), то игнорируется и считается равным [Поверх](ПоложениеТекстаОтносительноКартинки.Поверх.md).   
Значение по умолчанию – [Авто](ПоложениеТекстаОтносительноКартинки.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
