СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника.md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](СправочникОбъект.Имя_справочника.УстановитьСсылкуНового.md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СправочникМенеджер.\<Имя справочника\>](СправочникМенеджер.Имя_справочника.md), метод [ПолучитьСсылку](СправочникМенеджер.Имя_справочника.ПолучитьСсылку.md)

Доступен, начиная с версии 8.0.
