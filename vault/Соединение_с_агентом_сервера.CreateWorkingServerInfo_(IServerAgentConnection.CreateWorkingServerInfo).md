Соединение с агентом сервера (IServerAgentConnection)

CreateWorkingServerInfo (CreateWorkingServerInfo)

Доступен, начиная с версии 8.1.

Синтаксис:

CreateWorkingServerInfo()

Возвращаемое значение:

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   

Описание:

Создает новый рабочий сервер. Все его свойства содержат пустые строки, кроме свойства [MainPort](Рабочий_сервер.MainPort_(IWorkingServerInfo.MainPort).md), которое устанавливается в значение по умолчанию (1540).   
Новый сервер может быть использован, например, для регистрации сервера. Для этого ему должны быть явно заданы необходимые значения свойств.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
