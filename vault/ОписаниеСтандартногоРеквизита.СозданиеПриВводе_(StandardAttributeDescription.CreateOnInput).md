ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

СозданиеПриВводе (CreateOnInput)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [СозданиеПриВводе](СозданиеПриВводе_(CreateOnInput).md).   
Определяет режим создания при вводе. Если значение равно [Авто](СозданиеПриВводе.Авто_(CreateOnInput.Auto).md), то участие в механизме создания при вводе определяется значением свойства СозданиеПриВводе объекта метаданных этого типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
