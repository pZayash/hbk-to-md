ПараметрыЗапроса (QueryOptions)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Содержит параметры запроса (в строке URL-адреса параметры следуют после знака вопроса).

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
