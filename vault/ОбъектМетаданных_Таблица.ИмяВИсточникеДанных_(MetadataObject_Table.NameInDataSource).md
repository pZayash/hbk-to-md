ОбъектМетаданных: Таблица (MetadataObject: Table)

ИмяВИсточникеДанных (NameInDataSource)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя таблицы во внешнем источнике данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для таблиц со значением [Таблица](ВидТаблицыВнешнегоИсточникаДанных.Таблица_(ExternalDataSourceTableType.Table).md) свойства [ВидТаблицы](ОбъектМетаданных_Таблица.ВидТаблицы_(MetadataObject_Table.TableType).md).

Использование в версии:

Доступен, начиная с версии 8.2.
