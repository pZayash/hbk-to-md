Расширение группы формы для командной панели (Form group extension for a command bar)

ВажностьПриОтображении (DisplayImportance)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ВажностьПриОтображении](ВажностьПриОтображении.md).   
Указывает важность элемента при отображении (используется для адаптации формы к ширине экрана мобильного устройства).

Доступность:

Мобильный клиент, сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
