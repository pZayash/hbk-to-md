# Расширение формы обработки.ПередСохранениемЗначений

Расширение формы обработки (Data processor form extension)

ПередСохранениемЗначений (BeforeSaveValues)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСохранениемЗначений(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак сохранения значений формы. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), сохранение не будет выполнено.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением сохранения значений формы из списка сохраняемых значений.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
