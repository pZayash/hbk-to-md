ИспользованиеПараметраКомпоновкиДанных (DataCompositionParameterUse)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

Значение параметра получается из настроек компоновки данных только тогда, когда у параметра в настройках включено использование.

Использование в версии:

Доступен, начиная с версии 8.2.
