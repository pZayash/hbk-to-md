Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

ПолучитьГраницыВыделения (GetTextSelectionBounds)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьГраницыВыделения(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Возвращается позиция начала выделения.

<Конец> (обязательный)

Тип: [ЗакладкаФорматированногоДокумента](ЗакладкаФорматированногоДокумента_(FormattedDocumentBookmark).md).   
Возвращается позиция конца выделения.

Возвращаемое значение:

Возвращает позицию курсора.

Описание:

Получает границы выделенной области текста.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Метод можно использовать при открытом окне, в том числе в серверных вызовах.

Использование в версии:

Доступен, начиная с версии 8.2.
