ИспользованиеПодчинения (SubordinationUse)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеПодчинения](ИспользованиеПодчинения.md).   
Определяет, какие элементы могут являться владельцами данного справочника (например, элементы, группы или группы и элементы).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
