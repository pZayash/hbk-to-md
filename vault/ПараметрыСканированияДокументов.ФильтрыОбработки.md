ПараметрыСканированияДокументов (DocumentScanningParameters)

ФильтрыОбработки (ProcessingFilters)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Массив значений типа [ФильтрОбработкиСканированияДокументов](ФильтрОбработкиСканированияДокументов.md).  
Набор фильтров для обработки найденного документа, из которого пользователь выберет наилучший, чтобы его в дальнейшем напечатать или показать на форме.   
По умолчанию используется массив со значениями: [Текст](ФильтрОбработкиСканированияДокументов.Текст.md), [ТекстСКартинками](ФильтрОбработкиСканированияДокументов.ТекстСКартинками.md) и [Нет](ФильтрОбработкиСканированияДокументов.Нет.md).  
Передача пустого массива не допускается и приведет к вызову исключения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
