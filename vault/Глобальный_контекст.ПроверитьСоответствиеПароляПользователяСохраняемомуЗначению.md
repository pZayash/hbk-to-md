Глобальный контекст (Global context)

ПроверитьСоответствиеПароляПользователяСохраняемомуЗначению (CheckUserPasswordComplianceWithStoredValue)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПроверитьСоответствиеПароляПользователяСохраняемомуЗначению(<Пароль>, <ЗначениеДляПроверки>)

Параметры:

<Пароль> (обязательный)

Тип: Строка.   
Пароль пользователя информационной базы для проверки.

<ЗначениеДляПроверки> (обязательный)

Тип: Строка, [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы.md).   
Хранимое значение пароля для проверки соответствия.

- Если Строка, то она может быть получена из значения свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля.md) или с помощью метода [ВычислитьСохраняемоеЗначениеПароляПользователя](Глобальный_контекст.ВычислитьСохраняемоеЗначениеПароляПользователя.md).
- Если указан [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы.md), то используется значение его свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля.md).

Возвращаемое значение:

Тип: Булево.   
Истина, если пароль соответствует сохраняемому значению пароля, переданному в параметре <ЗначениеДляПроверки>, или содержащемуся в объекте [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы.md), переданному в параметре <ЗначениеДляПроверки>, с учетом установленного в информационной базе режима проверки сложности паролей пользователей, получаемого с помощью метода [ПолучитьПроверкуСложностиПаролейПользователей](Глобальный_контекст.ПолучитьПроверкуСложностиПаролейПользователей.md).

Описание:

Проверяет, соответствует ли заданный пароль сохраняемому значению пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
