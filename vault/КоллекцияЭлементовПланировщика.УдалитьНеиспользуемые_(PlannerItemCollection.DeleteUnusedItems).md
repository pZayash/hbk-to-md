# КоллекцияЭлементовПланировщика.УдалитьНеиспользуемые

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md) › [КоллекцияЭлементовПланировщика](КоллекцияЭлементовПланировщика_(PlannerItemCollection).md) › [Методы](КоллекцияЭлементовПланировщика__Методы.md)

КоллекцияЭлементовПланировщика (PlannerItemCollection)

УдалитьНеиспользуемые (DeleteUnusedItems)

Доступен, начиная с версии 8.3.11.

Синтаксис:

УдалитьНеиспользуемые()

Описание:

Удаляет из коллекции элементов события, которые не перетаскиваются пользователем в момент вызова метода и не попадают в текущие периоды отображения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
