Сервис кластера (IClusterServiceInfo)

ClusterManagers (ClusterManagers)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [COMSafeArray](COMSafeArray.md).   
Содержит массив объектов [Менеджер кластера](Менеджер_кластера.md), на которые назначен данный сервис. Каждый из сервисов может быть назначен только на один менеджер кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
