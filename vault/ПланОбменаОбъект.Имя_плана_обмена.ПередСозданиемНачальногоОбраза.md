ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПередСозданиемНачальногоОбраза (BeforeCreateInitialImage)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПередСозданиемНачальногоОбраза(<ТолькоЗарегистрированные>, <ПотокДанныхОбмена>)

Параметры:

<ТолькоЗарегистрированные>

Тип: Булево.   
Содержит значение Истина, если выполняется обход не всех данных исходной информационной базы.

<ПотокДанныхОбмена>

Тип: [ПотокОбменаДанными](ПотокОбменаДанными.md).   
Поток данных обмена между мобильным клиентом с автономным режимом и основным сервером.

Описание:

Вызывается перед началом создания образа информационной базы.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
