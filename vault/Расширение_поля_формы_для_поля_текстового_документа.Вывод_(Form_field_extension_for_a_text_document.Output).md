# Расширение поля формы для поля текстового документа.Вывод

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля текстового документа](Расширение_поля_формы_для_поля_текстового_документа_(Form_field_extension_for_a_text_document).md) › [Свойства](Расширение_поля_формы_для_поля_текстового_документа__Свойства.md)

Расширение поля формы для поля текстового документа (Form field extension for a text document)

Вывод (Output)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеВывода](ИспользованиеВывода_(UseOutput).md).   
Ограничение вывода на печать, сохранения, работы с буфером обмена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Может иметь значение, отличное от значения свойства [Вывод](ТекстовыйДокумент.Вывод_(TextDocument.Output).md) документа, с которым это поле связано.

Использование в версии:

Доступен, начиная с версии 8.2.
