СтандартнаяКомандаВходящегоЗапросаПоделиться (IncomingShareRequestStandardCommand)

Сохранить (Save)

Доступен, начиная с версии 8.3.24.

Описание:

Доступно только при запросе "Поделиться" для файлов.  
При выполнении команды файлы будут доступны для выбора в диалоге выбора файлов в разделе "Документы - Входящие".  
Пользователю отображается сообщение "Файлы сохранены для выбора в "Документы - Входящие"".  
В сохраненных входящих файлах хранятся последние десять входящих групп файлов.  
В контекстном меню команда будет иметь представление "Сохранить во "Входящие"".

Использование в версии:

Доступен, начиная с версии 8.3.24.
