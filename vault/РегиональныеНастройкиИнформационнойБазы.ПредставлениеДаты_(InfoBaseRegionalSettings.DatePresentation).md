# РегиональныеНастройкиИнформационнойБазы.ПредставлениеДаты

РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

ПредставлениеДаты (DatePresentation)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПредставлениеДаты(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение даты, представление которой требуется получить.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Представление значения, аналогичное тому, которое строится методом глобального контекста [Строка](Глобальный_контекст.Строка_(Global_context.String).md). В качестве региональных установок используются данные этого объекта [РегиональныеНастройкиИнформационнойБазы](РегиональныеНастройкиИнформационнойБазы_(InfoBaseRegionalSettings).md).  
Если в качестве параметра указана [Дата](lang__def_Date.md), то строится представление только даты без времени.

Описание:

Получает представление значения даты.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
