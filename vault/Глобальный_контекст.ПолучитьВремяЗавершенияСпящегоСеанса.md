Глобальный контекст (Global context)

ПолучитьВремяЗавершенияСпящегоСеанса (GetHibernateSessionTerminateTime)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьВремяЗавершенияСпящегоСеанса()

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает интервал времени в секундах, по истечении которого спящий сеанс завершается.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
