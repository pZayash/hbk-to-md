МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

ОтключитьОбработчикИзмененияСостояния (DetachStateChangeHandler)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОтключитьОбработчикИзмененияСостояния(<Обработчик>)

Параметры:

<Обработчик> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, заданной при вызове метода [ПодключитьОбработчикИзмененияСостояния](МенеджерАгентаКлиентскогоПриложения.ПодключитьОбработчикИзмененияСостояния_(ClientApplicationAgentManager.AttachStateChangeHandler).md).

Описание:

Отключает ранее подключенную процедуру-обработчик.   
Сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
