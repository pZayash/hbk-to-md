# КоллекцияЭлементовПланировщика.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md) › [КоллекцияЭлементовПланировщика](КоллекцияЭлементовПланировщика_(PlannerItemCollection).md) › [Методы](КоллекцияЭлементовПланировщика__Методы.md)

КоллекцияЭлементовПланировщика (PlannerItemCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [ЭлементПланировщика](ЭлементПланировщика_(PlannerItem).md);.   
Индекс удаляемого элемента в коллекции или сам элемент.

Описание:

Удаляет элемент планировщика в заданной позиции коллекции элементов планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
