ВыражениеОтбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionFilterExpression)

ИспользоватьРеквизиты (UseAttributes)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования подчиненных элементов для поля, указанного в выражении.  
Значение [Истина](lang__def_BooleanTrue.md) может быть установлено только в том случае, если выражение содержит имя доступного поля схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
