ДокументСписок.<Имя документа> (DocumentList.<Document name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Содержит объект [Отбор](Отбор_(Filter).md), по которому осуществляется текущая фильтрация документов в списке. Имена элементов отбора совпадают с именами реквизитов документа и критериев отбора, как они заданы в Конфигураторе. Элементы отбора представляют собой колонки списка документов и критерии отбора, относящиеся к документу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
