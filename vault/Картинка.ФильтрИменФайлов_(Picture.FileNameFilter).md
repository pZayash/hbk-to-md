Картинка (Picture)

ФильтрИменФайлов (FileNameFilter)

Доступен, начиная с версии 8.2.

Синтаксис:

ФильтрИменФайлов(<ВсеВарианты>, <ПодобратьВариант>)

Параметры:

<ВсеВарианты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки. Если значение [Ложь](lang__def_BooleanFalse.md) - фильтр будет подготовлен для картинки варианта. Если значение [Истина](lang__def_BooleanTrue.md) - для архива.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ПодобратьВариант> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md), [Неопределено](lang__def_Undefined.md).   
Данный параметр используется, если картинка представлена архивом, содержащим варианты этой картинки, и <ВсеВарианты> установлен в значение [Ложь](lang__def_BooleanFalse.md).  
Пары "Ключ-Значение" соответствуют xml-тегам в манифесте вариантов картинки. Если параметры выбора варианта не заданы, будет выбран вариант по умолчанию. Если параметры указывают на несуществующий в архиве вариант, то будет возвращено [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Фильтр для диалога выбора файлов.

Описание:

Получает фильтры для диалога выбора файлов, соответствующие формату картинки.

Доступность:

Тонкий клиент, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ФильтрДиалога = Картинка.ФильтрИменФайлов(); |

Использование в версии:

Доступен, начиная с версии 8.2.
