ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

ДобавитьКопию (AddCopy)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ДобавитьКопию(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента, который нужно скопировать.

Возвращаемое значение:

Тип: [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).   
[ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md) или [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md), в зависимости от того, что находилось по переданному индексу.

Описание:

Добавляет копию запроса. В параметре передается индекс запроса, который требуется скопировать.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
