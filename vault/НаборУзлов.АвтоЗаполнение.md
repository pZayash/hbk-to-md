НаборУзлов (NodeSet)

АвтоЗаполнение (AutoFill)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если значение свойства Истина, то метод [Заполнить](НаборУзлов.Заполнить.md) будет вызван автоматически перед обращением к обработчику ПередЗаписью() объекта данных, которому принадлежит объект [ПараметрыОбменаДанными](ПараметрыОбменаДанными.md), которому, в свою очередь, принадлежит данный объект. Если значение свойства [АвтоЗаполнение](НаборУзлов.АвтоЗаполнение.md) Ложь, то никаких действий предприниматься не будет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
