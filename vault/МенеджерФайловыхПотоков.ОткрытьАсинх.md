МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьАсинх (OpenAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: С указанием режима открытия

Синтаксис:

ОткрытьАсинх(<ИмяФайла>, <РежимОткрытия>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла.md).   
Режим открытия файла.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер буфера для операций с файлом.

Описание варианта метода:

Открывает файл в заданном режиме с возможностью чтения и записи. Файл открывается в режиме разделяемого чтения. Результат Обещания - объект типа [ФайловыйПоток](ФайловыйПоток.md) для файла с указанным именем.

Вариант синтаксиса: С указанием режимов открытия и доступа

Синтаксис:

ОткрытьАсинх(<ИмяФайла>, <РежимОткрытия>, <РежимДоступа>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла.md).   
Режим открытия файла.

<РежимДоступа> (обязательный)

Тип: [ДоступКФайлу](ДоступКФайлу.md).   
Режим доступа к файлу.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер буфера для операций с файлом.

Описание варианта метода:

Открывает файл в выбранном режиме, с заданным уровнем доступа, с общим доступом на чтение. Результат Обещания - объект типа [ФайловыйПоток](ФайловыйПоток.md) для файла с указанным именем.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [ФайловыйПоток](ФайловыйПоток.md) для файла с заданным именем.

Описание:

Асинхронная функция. Открывает файл в заданном режиме с возможностью чтения и записи.   
Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, толстый клиент.
