Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

ВыполнитьВыборИзСпискаВыбора (ExecuteChoiceFromChoiceList)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВыполнитьВыборИзСпискаВыбора(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Строка, Число.   
Представление выбираемого значения или номер значения (индекс строки, нумерация с 0).

Описание:

Выбирает значение из открытого списка выбора по его представлению в списке выбора.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
