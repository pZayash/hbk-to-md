УсловноеОформление (ConditionalAppearance)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Оформление>, <Смещение>)

Параметры:

<Оформление> (обязательный)

Тип: [ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md).   
Перемещаемое оформление.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает оформление на заданное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
