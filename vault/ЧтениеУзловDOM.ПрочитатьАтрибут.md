ЧтениеУзловDOM (DOMNodeReader)

ПрочитатьАтрибут (ReadAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

ПрочитатьАтрибут()

Возвращаемое значение:

Тип: Булево.   
Истина - очередной атрибут прочитан; Ложь - атрибутов больше нет.

Описание:

Считывает очередной атрибут элемента XML. После прочтения с помощью метода [ЧтениеУзловDOM.Прочитать](ЧтениеУзловDOM.Прочитать.md) очередного узла, имеющего тип [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md), можно получить доступ к атрибутам данного элемента. Каждый вызов данного метода позволяет прочитать очередной атрибут. После прочтения очередного атрибута свойствам [ЧтениеУзловDOM.ТипУзла](ЧтениеУзловDOM.ТипУзла.md), [ЧтениеУзловDOM.Имя](ЧтениеУзловDOM.Имя.md), [ЧтениеУзловDOM.ЛокальноеИмя](ЧтениеУзловDOM.ЛокальноеИмя.md), [ЧтениеУзловDOM.Префикс](ЧтениеУзловDOM.Префикс.md), [ЧтениеУзловDOM.URIПространстваИмен](ЧтениеУзловDOM.URIПространстваИмен.md) и [ЧтениеУзловDOM.Значение](ЧтениеУзловDOM.Значение.md) присваиваются значения в соответствии с прочитанным атрибутом.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
