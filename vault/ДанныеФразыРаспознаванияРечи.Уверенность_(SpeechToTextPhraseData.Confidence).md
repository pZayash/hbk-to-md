ДанныеФразыРаспознаванияРечи (SpeechToTextPhraseData)

Уверенность (Confidence)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Влияет на положение фразы в коллекции [ДанныеФраз](РезультатРаспознаванияРечи.ДанныеФраз_(SpeechToTextResult.PhrasesData).md), которая сортируется в порядке уменьшения значения данного свойства.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
