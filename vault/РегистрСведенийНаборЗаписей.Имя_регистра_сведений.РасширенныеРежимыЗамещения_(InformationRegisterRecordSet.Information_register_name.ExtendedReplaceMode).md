РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

РасширенныеРежимыЗамещения (ExtendedReplaceMode)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность использования расширенных режимов замещения [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) или [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) в качестве значения параметра метода [Записать](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.Записать_(AccumulationRegisterRecordSet.Accumulation_register_name.Write).md) для наборов записей подчиненных регистров.   
Также определяет доступность для записи свойства [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки_(AccumulationRegisterRecord.Accumulation_register_name.LineNumber).md) у записей, принадлежащих набору записей.  
На поведение наборов записей независимых регистров сведений значение данного свойства не влияет.  
Если свойство имеет значение [Истина](lang__def_BooleanTrue.md), то режимы замещения [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) или [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) могут быть использованы в качестве значения параметра метода [Записать](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.Записать_(AccumulationRegisterRecordSet.Accumulation_register_name.Write).md), а свойство [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки_(AccumulationRegisterRecord.Accumulation_register_name.LineNumber).md) у записей, принадлежащих набору, доступно для записи.   
Иначе режимы [Удаление](РежимЗамещения.Удаление_(ReplacementMode.Delete).md) или [Обновление](РежимЗамещения.Обновление_(ReplacementMode.Update).md) не могут быть использованы, а свойство [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки_(AccumulationRegisterRecord.Accumulation_register_name.LineNumber).md) у записей, принадлежащих набору, недоступно для записи.  
При создании набора записей свойству присваивается значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
