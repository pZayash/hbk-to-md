Глобальный контекст (Global context)

БлокировкаАутентификации (AuthenticationLock)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [МенеджерБлокировкиАутентификацииПользователейИнформационнойБазы](МенеджерБлокировкиАутентификацииПользователейИнформационнойБазы_(InfoBaseUserAuthenticationLockManager).md).   
Предоставляет доступ к управлению блокировками аутентификации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
