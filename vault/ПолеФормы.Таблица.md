ПолеФормы (FormField)

Таблица (Table)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [ТаблицаФормы](ТаблицаФормы.md).   
Для объекта с значением свойства [Вид](ПолеФормы.Вид.md), равным [ПолеДиаграммыГанта](ВидПоляФормы.ПолеДиаграммыГанта.md), возвращается объект типа [ТаблицаФормы](ТаблицаФормы.md), содержащий предопределенную таблицу поля, для остальных - выбрасывается исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
