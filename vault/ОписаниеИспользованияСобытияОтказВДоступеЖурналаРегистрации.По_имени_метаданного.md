ОписаниеИспользованияСобытияОтказВДоступеЖурналаРегистрации (EventLogAccessDeniedEventUseDescription)

По имени метаданного

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ОписаниеИспользованияСобытияОтказВДоступеЖурналаРегистрации(<Объект>)

Параметры:

<Объект> (необязательный)

Имя объекта метаданных.

Описание:

Создает объект с указанным полным именем метаданного в качестве объекта и пустыми массивами в свойствах [Объект](ОписаниеИспользованияСобытияОтказВДоступеЖурналаРегистрации.Объект_(EventLogAccessDeniedEventUseDescription.Object).md) и [ПоляРегистрации](ОписаниеИспользованияСобытияОтказВДоступеЖурналаРегистрации.ПоляРегистрации_(EventLogAccessDeniedEventUseDescription.RegistrationFields).md).

Доступен, начиная с версии 8.2.
