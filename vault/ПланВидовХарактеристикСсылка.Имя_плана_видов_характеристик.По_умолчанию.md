ПланВидовХарактеристикСсылка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesRef.<Chart of characteristic types name>)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ПланВидовХарактеристикСсылка.<Имя плана видов характеристик>(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор ссылки.   
Если идентификатор не указан, создается пустая ссылка.

Описание:

Создает новый объект на основе переданных параметров.   
Такая форма записи будет вызывать ошибку при проверке модулей в конфигураторе, так как при проверке модулей в конфигураторе не определены типы прикладных объектов.  
Также может быть создан с использованием функциональной формы оператора Новый.   
Например, ХарактеристикаНоменклатуры = Новый(Тип("ПланВидовХарактеристикСсылка.ВидыХарактеристикТоваров"));

Пример:

|  |
| --- |
| ПараметрыОбъекта = НовыйМассив; ПараметрыОбъекта.Добавить (Новый УникальныйИдентификатор("05dbe824-a4c6-11dd-bf56-00145e3710ab"));    СозданиеОбъекта = Новый(Тип("ПланВидовХарактеристикСсылка.ВидыХарактеристик"), ПараметрыОбъекта); |

Использование в версии:

Доступен, начиная с версии 8.2.
