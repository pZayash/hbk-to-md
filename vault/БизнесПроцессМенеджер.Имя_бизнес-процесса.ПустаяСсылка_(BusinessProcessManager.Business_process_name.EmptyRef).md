ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [БизнесПроцессСсылка.\<Имя бизнес-процесса\>](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md).   

Описание:

Получает пустое значение ссылки на бизнес-процесс данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
