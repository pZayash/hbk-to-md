ОтчетОбОшибке (ErrorReport)

Записать (Write)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Записать(<Файл>, <Интерактивно>)

Параметры:

<Файл> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла на диске, в который будет записан отчет.

<Интерактивно> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр имеет значение [Истина](lang__def_BooleanTrue.md), то перед сохранением откроется форма создания отчета (где пользователь сможет дополнить отчет).   
Если параметр имеет значение [Ложь](lang__def_BooleanFalse.md), то отчет сохранится без свойств, которые не разрешены для автоматического сохранения.   
Если параметр не установлен, то форма откроется, если есть свойства, для записи которых требуется разрешение пользователя.

Описание:

Записывает отчет в файл. Файл имеет формат архива. Архив может содержать файл отчета, приложенные пользователм файлы и дамп, сгенерированный при возникновении ошибки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[НастройкиОбработкиОшибок](НастройкиОбработкиОшибок_(ErrorProcessingSettings).md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.
