ОбработкиМенеджер (DataProcessorsManager)

<Имя обработки> (<Data processor name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ОбработкаМенеджер.\<Имя обработки\>](ОбработкаМенеджер.Имя_обработки.md).   
Набор свойств содержит менеджеры отдельных обработок. Доступ к менеджеру осуществляется по имени. Имена свойств совпадают с именами обработок, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
