ЧтениеДанных (DataReader)

КодировкаТекста (TextEncoding)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста, используемая по умолчанию для данного экземпляра [ЧтениеДанных](ЧтениеДанных.md).  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

- UTF-16BE

- HZ-GB-2312

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.9.
