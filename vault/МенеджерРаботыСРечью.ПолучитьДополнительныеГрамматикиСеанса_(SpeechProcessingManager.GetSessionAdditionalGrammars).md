МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьДополнительныеГрамматикиСеанса (GetSessionAdditionalGrammars)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДополнительныеГрамматикиСеанса()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив ключей грамматик (типа [Строка](lang__def_String.md)), сохраненных для сеанса.

Описание:

Возвращает описания всех доступных дополнительных грамматик сеанса.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.25.
