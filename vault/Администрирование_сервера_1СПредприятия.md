# Администрирование сервера "1С:Предприятия"

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md)

Объекты, предназначенные для доступа к функциям администрирования серверов "1С:Предприятия" из встроенного языка.  
Управление осуществляется через административный интерфейс ras.   
Для работы данных объектов требуется наличие запущенного сервера администрирования (ras).
