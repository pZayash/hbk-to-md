# МенеджерРегламентныхЗаданий.ПолучитьРегламентныеЗадания

МенеджерРегламентныхЗаданий (ScheduledJobsManager)

ПолучитьРегламентныеЗадания (GetScheduledJobs)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьРегламентныеЗадания(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, задающая отбор. Значениями структуры могут быть: [УникальныйИдентификатор](РегламентноеЗадание.УникальныйИдентификатор_(ScheduledJob.UUID).md), [Ключ](РегламентноеЗадание.Ключ_(ScheduledJob.Key).md), [Метаданные](РегламентноеЗадание.Метаданные_(ScheduledJob.Metadata).md), [Предопределенное](РегламентноеЗадание.Предопределенное_(ScheduledJob.Predefined).md), [Использование](РегламентноеЗадание.Использование_(ScheduledJob.Use).md), [Наименование](РегламентноеЗадание.Наименование_(ScheduledJob.Description).md). Если отбор не задан, получаются все регламентные задания.  
Если выполняется отбор по метаданным, то в качестве значения [Метаданные](РегламентноеЗадание.Метаданные_(ScheduledJob.Metadata).md) можно указать либо объект метаданных регламентного задания, либо его имя.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Получает массив регламентных заданий по заданному отбору. Получение регламентных заданий возможно только для администратора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
