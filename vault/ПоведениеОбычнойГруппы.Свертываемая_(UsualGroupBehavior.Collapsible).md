ПоведениеОбычнойГруппы (UsualGroupBehavior)

Свертываемая (Collapsible)

Доступен, начиная с версии 8.3.3.

Описание:

Группа обладает возможностью свертываться и развертываться.

Использование в версии:

Доступен, начиная с версии 8.3.3.
