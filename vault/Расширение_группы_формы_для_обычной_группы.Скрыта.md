Расширение группы формы для обычной группы (Form group extension for a usual group)

Скрыта (Hidden)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Скрыта()

Возвращаемое значение:

Тип: Булево.   
В зависимости от значения свойства [Поведение](Расширение_группы_формы_для_обычной_группы.Поведение.md) группы, метод возвращает:

- [Обычное](ПоведениеОбычнойГруппы.Обычное.md) - Ложь,
- [Свертываемая](ПоведениеОбычнойГруппы.Свертываемая.md) - Истина, если группа свернута, и Ложь в обратном случае,
- [Всплывающая](ПоведениеОбычнойГруппы.Всплывающая.md) - Истина, если группа в данный момент отображается, и Ложь в обратном случае.

Описание:

Проверяет состояние группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
