ОписаниеКонфигурации (ConfigurationDescription)

Имя (Name)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Краткое наименование конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имя должно состоять из одного слова, начинаться с буквы и не содержать специальных символов, кроме «\_».

Использование в версии:

Доступен, начиная с версии 8.3.13.
