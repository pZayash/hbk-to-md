СертификатКриптографии (CryptoCertificate)

Для инициализации

Доступен, начиная с версии 8.3.6.

Синтаксис:

Новый СертификатКриптографии()

Описание:

Создает объект, который должен быть инициализирован при помощи метода [НачатьИнициализацию](СертификатКриптографии.НачатьИнициализацию_(CryptoCertificate.BeginInitialization).md).  
В веб-клиенте, если не установлено расширение работы с криптографией или если недоступно использование синхронных вызовов, при вызове конструктора генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
