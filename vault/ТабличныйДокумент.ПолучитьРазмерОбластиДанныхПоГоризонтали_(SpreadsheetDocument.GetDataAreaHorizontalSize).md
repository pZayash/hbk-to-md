ТабличныйДокумент (SpreadsheetDocument)

ПолучитьРазмерОбластиДанныхПоГоризонтали (GetDataAreaHorizontalSize)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьРазмерОбластиДанныхПоГоризонтали()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает максимальный по всем типам колонок табличного документа номер колонки, содержащей данные.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.10.
