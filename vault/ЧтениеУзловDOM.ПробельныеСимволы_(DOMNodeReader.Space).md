ЧтениеУзловDOM (DOMNodeReader)

ПробельныеСимволы (Space)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ПробельныеСимволыXML](ПробельныеСимволыXML_(XMLSpace).md).   
Содержит значение атрибута xml:space для данного контекста чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
