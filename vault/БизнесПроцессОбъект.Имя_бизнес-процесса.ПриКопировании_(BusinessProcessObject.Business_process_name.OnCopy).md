БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ПриКопировании (OnCopy)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриКопировании(<ОбъектКопирования>)

Параметры:

<ОбъектКопирования>

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md).   
Исходный бизнес-процесс, который является источником копирования.

Описание:

Возникает при создании бизнес-процесса копированием.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

См. также:

[БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md), метод [Скопировать](БизнесПроцессОбъект.Имя_бизнес-процесса.Скопировать_(BusinessProcessObject.Business_process_name.Copy).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
