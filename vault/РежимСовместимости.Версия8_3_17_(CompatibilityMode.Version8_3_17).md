РежимСовместимости (CompatibilityMode)

Версия8\_3\_17 (Version8\_3\_17)

Доступен, начиная с версии 8.3.18.

Описание:

Усовершенствования версии 8.3.18 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.17.

Использование в версии:

Доступен, начиная с версии 8.3.18.
