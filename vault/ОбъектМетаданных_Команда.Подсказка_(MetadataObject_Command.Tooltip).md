ОбъектМетаданных: Команда (MetadataObject: Command)

Подсказка (Tooltip)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст всплывающей подсказки, которая появляется при наведении курсора на команду.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
