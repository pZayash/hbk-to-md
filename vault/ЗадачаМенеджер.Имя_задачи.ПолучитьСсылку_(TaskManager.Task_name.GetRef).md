# ЗадачаМенеджер.<Имя задачи>.ПолучитьСсылку

ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ЗадачаСсылка.Имя_задачи.УникальныйИдентификатор_(TaskRef.Task_name.UUID).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

См. также:

[ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md), метод [УникальныйИдентификатор](ЗадачаСсылка.Имя_задачи.УникальныйИдентификатор_(TaskRef.Task_name.UUID).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
