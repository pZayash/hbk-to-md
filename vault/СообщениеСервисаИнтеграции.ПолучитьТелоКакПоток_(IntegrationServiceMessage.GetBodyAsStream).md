# СообщениеСервисаИнтеграции.ПолучитьТелоКакПоток

СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ПолучитьТелоКакПоток (GetBodyAsStream)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТелоКакПоток()

Возвращаемое значение:

Тип: [Поток](Поток_(Stream).md).   
Возвращает поток со следующими свойствами в зависимости от состояния сообщения:

- При создании сообщения - [ДоступнаЗапись](Поток.ДоступнаЗапись_(Stream.CanWrite).md) имеет значение [Истина](lang__def_BooleanTrue.md), [ДоступноЧтение](Поток.ДоступноЧтение_(Stream.CanRead).md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) имеют значение [Ложь](lang__def_BooleanFalse.md).
- При отправке и для отправленного сообщения - [ДоступноЧтение](Поток.ДоступноЧтение_(Stream.CanRead).md) имеет значение [Истина](lang__def_BooleanTrue.md), [ДоступнаЗапись](Поток.ДоступнаЗапись_(Stream.CanWrite).md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) имеют значение [Ложь](lang__def_BooleanFalse.md).
- Для полученного сообщения - [ДоступноЧтение](Поток.ДоступноЧтение_(Stream.CanRead).md) имеет значение [Истина](lang__def_BooleanTrue.md), [ДоступнаЗапись](Поток.ДоступнаЗапись_(Stream.CanWrite).md) и [ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) имеют значение [Ложь](lang__def_BooleanFalse.md).

Описание:

Получить тело сообщения как поток, доступный для чтения и записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.21.
