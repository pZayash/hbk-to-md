Расширение тестируемого поля формы для поля переключателя (Tested form field extension for a radio button field)

ВыбратьВариант (SelectOption)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВыбратьВариант(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Строка](lang__def_String.md), [Число](lang__def_Number.md).   
Представление выбираемого значения или номер значения (индекс значения, нумерация с 0).

Описание:

Устанавливает новое значение переключателя.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
