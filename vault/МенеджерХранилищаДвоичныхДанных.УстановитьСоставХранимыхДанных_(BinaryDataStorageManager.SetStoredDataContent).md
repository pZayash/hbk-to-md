МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьСоставХранимыхДанных (SetStoredDataContent)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьСоставХранимыхДанных(<Состав>)

Параметры:

<Состав> (обязательный)

Тип: [СоставХранимыхДанныхХранилищаДвоичныхДанных](СоставХранимыхДанныхХранилищаДвоичныхДанных_(BinaryDataStorageDataContent).md).   
Состав хранимых данных.

Описание:

Устанавливает состав хранимых данных для настройки записи конкретных объектов метаданных.  
Один и тот же объект метаданных не может быть настроен для использования с разными хранилищами двоичных данных в [СоставХранимыхДанныхХранилищаДвоичныхДанных](СоставХранимыхДанныхХранилищаДвоичныхДанных_(BinaryDataStorageDataContent).md). Если состав хранимых данных содержит объект метаданных, который уже был настроен для использования в другом внешнем или встроенном хранилище двоичных данных, будет сгененировано исключение.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
