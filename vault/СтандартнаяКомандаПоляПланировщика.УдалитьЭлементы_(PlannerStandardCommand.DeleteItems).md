СтандартнаяКомандаПоляПланировщика (PlannerStandardCommand)

УдалитьЭлементы (DeleteItems)

Доступен, начиная с версии 8.3.23.

Описание:

Удалить выбранные элементы. Используется только для источника типа [Элементы](ИсточникКомандПланировщика.Элементы_(PlannerCommandSource.Items).md).  
В контекстном меню команда будет иметь представление "Удалить".

Использование в версии:

Доступен, начиная с версии 8.3.23.
