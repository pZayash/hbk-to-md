СредстваМультимедиа (MultimediaTools)

ОтключитьАудиозапись (StopAudioRecording)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ОтключитьАудиозапись()

Описание:

Останавливает запись аудио в фоновом для пользователя режиме.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
