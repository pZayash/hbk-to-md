Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - если диалог был закрыт нажатием кнопки "OK", содержит структуру, идентичную структуре в параметре <Отбор> метода [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии.md), иначе Неопределено.

Описание:

Асинхронная функция. Показывает диалога отбора версий.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
