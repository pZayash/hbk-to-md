ЧтениеPDF (PDFReader)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЧтениеPDF()

Описание:

Формирует неинициализированный объект [ЧтениеPDF](ЧтениеPDF_(PDFReader).md). Для инициализации объекта используется метод [Открыть](ЧтениеPDF.Открыть_(PDFReader.Open).md) или [НачатьОткрытие](ЧтениеPDF.НачатьОткрытие_(PDFReader.BeginOpen).md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
