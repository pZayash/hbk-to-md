Формирование неинициализированного объекта

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЧтениеPDF()

Описание:

Формирует неинициализированный объект [ЧтениеPDF](ЧтениеPDF.md). Для инициализации объекта используется метод [Открыть](ЧтениеPDF.Открыть.md) или [НачатьОткрытие](ЧтениеPDF.НачатьОткрытие.md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
