ОписаниеКомандыВходящегоЗапросаПоделиться (IncomingShareRequestCommandDescription)

Команда (Command)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: [СтандартнаяКомандаВходящегоЗапросаПоделиться](СтандартнаяКомандаВходящегоЗапросаПоделиться.md), [ОписаниеОповещения](ОписаниеОповещения.md), Неопределено.   
Содержит описание команды:

- [СтандартнаяКомандаВходящегоЗапросаПоделиться](СтандартнаяКомандаВходящегоЗапросаПоделиться.md) - стандартная команда;
- [ОписаниеОповещения](ОписаниеОповещения.md) - процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
  - <Данные> - тип [ДанныеЗапросаПоделиться](ДанныеЗапросаПоделиться.md),
  - <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md);;
- Неопределено - разделитель;
- Любой другой тип - пункт меню не отображается.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
