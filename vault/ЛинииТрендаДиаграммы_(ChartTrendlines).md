Доступен, начиная с версии 8.3.10.

Элементы коллекции:

[ЛинияТрендаДиаграммы](ЛинияТрендаДиаграммы_(ChartTrendline).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются линии тренда.  
Возможно обращение к линии тренда посредством оператора [...]. В качестве аргумента передается индекс линии тренда (нумерация с 0).

Описание:

Представляет собой коллекцию линий тренда.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartTrendlines.

Использование в версии:

Доступен, начиная с версии 8.3.10.
