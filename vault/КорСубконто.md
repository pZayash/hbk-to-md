КорСубконто (необязательный)

Тип параметра: [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>, [Массив](Массив.md), Неопределено.   
Задает набор и порядок кор. субконто, которыми можно оперировать в запросе. Накладывает фильтр на записи по видам кор. субконто (если массив то по И).  
Параметр не существует, если при конфигурировании для плана счетов не указаны субконто.  
Если задан массив, то тип значения элемента массива [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик.md)>.  
Значение по умолчанию: Неопределено.
