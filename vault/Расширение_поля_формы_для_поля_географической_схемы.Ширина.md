Расширение поля формы для поля географической схемы (Form field extension for a geographical schema field)

Ширина (Width)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит ширину географической схемы. Задается в символах.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
