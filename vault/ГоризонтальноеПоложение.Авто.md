ГоризонтальноеПоложение (HorizontalAlign)

Авто (Auto)

Доступен, начиная с версии 8.0.

Описание:

Горизонтальное выравнивание текста элемента управления или области табличного документа будет определяться автоматически, исходя из типа отображаемого значения. Числовые значения будут прижиматься к правой границе, остальные - к левой.

Использование в версии:

Доступен, начиная с версии 8.0.
