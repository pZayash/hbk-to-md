Соединение с агентом сервера (IServerAgentConnection)

RegAgentAdmin (RegAgentAdmin)

Доступен, начиная с версии 8.1.

Синтаксис:

RegAgentAdmin(<Администратор>)

Параметры:

<Администратор> (обязательный)

Тип: [Администратор](Администратор.md).   
Описание администратора центрального сервера, информацию о котором нужно добавить или обновить.

Описание:

Добавить (или обновить информацию) администратора центрального сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
