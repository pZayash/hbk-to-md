Расширение табличного поля структуры настроек компоновки данных (Settings structure table box extension)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
