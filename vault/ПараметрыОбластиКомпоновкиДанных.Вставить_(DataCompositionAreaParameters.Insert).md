ПараметрыОбластиКомпоновкиДанных (DataCompositionAreaParameters)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных_(DataCompositionExpressionAreaParameter).md) и [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных_(DataCompositionDetailsAreaParameter).md).

Возвращаемое значение:

Тип: [ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных_(DataCompositionExpressionAreaParameter).md), [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных_(DataCompositionDetailsAreaParameter).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
