ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

ОбработкаПослеЗаписиВерсийИсторииДанных (AfterWriteDataHistoryVersionsProcessing)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ОбработкаПослеЗаписиВерсийИсторииДанных(<ИнформацияОЗаписиВерсий>)

Параметры:

<ИнформацияОЗаписиВерсий>

Тип: [КоллекцияИнформацииОЗаписиВерсииИсторииДанных](КоллекцияИнформацииОЗаписиВерсииИсторииДанных_(WriteDataHistoryVersionInformationCollection).md).   
Содержит информацию о записи версий истории данных.

Описание:

Обработчик вызывается после создания версий, отдельным вызовом метода [ВыполнитьОбработкуПослеЗаписиВерсий](МенеджерИсторииДанных.ВыполнитьОбработкуПослеЗаписиВерсий_(DataHistoryManager.ExecuteAfterWriteVersionsProcessing).md) либо в рамках вызова [ОбновитьИсторию](МенеджерИсторииДанных.ОбновитьИсторию_(DataHistoryManager.UpdateHistory).md) при указании необходимости выполнения обработки.  
Обработчик выполняется с установленным безопасным режимом разделения для всех независимых разделителей конфигурации - переключение в другую область данных будет запрещено.  
Исключение в обработчике приводит к прерыванию выполнения обработки остальных версий в очереди, поэтому рекомендуется не допускать возникновения исключений.   
Элементы коллекции информации о записи версий отсортированы по возрастанию данных, по которым создаются версии, и номеров версий.   
В обработчик передается не более 1000 элементов.   
Рекомендуется обрабатывать исключения, возникающие в вызываемых механизмах, с целью не прерывать обработку после записи.  
Рекомендуется создать регламентное задание, удаляющее из обработки версии, созданные ранее определенной даты. Для этого в задании следует вызвать метод [УдалитьИзОбработкиПослеЗаписиВерсий](МенеджерИсторииДанных.УдалитьИзОбработкиПослеЗаписиВерсий_(DataHistoryManager.DeleteFromAfterWriteVersionsProcessing).md) с указанием даты.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
