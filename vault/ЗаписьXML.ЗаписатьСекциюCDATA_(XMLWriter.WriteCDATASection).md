# ЗаписьXML.ЗаписатьСекциюCDATA

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md) › [ЗаписьXML](ЗаписьXML_(XMLWriter).md) › [Методы](ЗаписьXML__Методы.md)

ЗаписьXML (XMLWriter)

ЗаписатьСекциюCDATA (WriteCDATASection)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьСекциюCDATA(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [Строка](lang__def_String.md).   
Символьные данные - содержимое секции CDATA.

Описание:

Осуществляет запись секции CDATA.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
