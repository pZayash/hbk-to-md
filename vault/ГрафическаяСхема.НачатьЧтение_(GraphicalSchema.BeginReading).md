ГрафическаяСхема (GraphicalSchema)

НачатьЧтение (BeginReading)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЧтение(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения чтения файла со следующими параметрами:

- <Результат> - результат операции, в настоящее время не используется, всегда имеет значение [Истина](lang__def_BooleanTrue.md),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла графической схемы.

Описание:

Вызывает начало чтения графической схемы из файла.

Доступность:

Тонкий клиент, толстый клиент.

[ГрафическаяСхема](ГрафическаяСхема_(GraphicalSchema).md), метод [ПрочитатьАсинх](ГрафическаяСхема.ПрочитатьАсинх_(GraphicalSchema.ReadAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.7.
