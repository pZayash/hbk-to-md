Соединение с агентом сервера (IServerAgentConnection)

SetClusterRecyclingKillProblemProcesses (SetClusterRecyclingKillProblemProcesses)

Доступен, начиная с версии 8.3.6.

Синтаксис:

SetClusterRecyclingKillProblemProcesses(<Кластер>, <ПринудительноЗавершатьПроблемныеПроцессы>)

Параметры:

<Кластер> (необязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер серверов, для которого устанавливается принудительное завершение проблемных процессов.  
Значение по умолчанию: Число.

<ПринудительноЗавершатьПроблемныеПроцессы> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), то процессы, признанные проблемными по результатам мониторинга, будут принудительно завершены.  
Возможные причины признания процесса проблемным:

- превышение значения <ДопустимоеОтклонениеКоличестваОшибокСервера>,
- превышение количества памяти, занимаемой процессом,
- зависание процесса (перестает отвечать на запросы),
- процесс не был завершен через некоторое время после исключения из кластера.

Описание:

Устанавливает принудительное завершение для проблемных процессов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.6.
