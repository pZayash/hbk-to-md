Описание:

Определяет набор картинок, используемых в конфигурации. Значения этого набора имеют тип [Картинка](Картинка_(Picture).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // Для обычных форм  ЭлементыФормы.Надпись.Картинка = БиблиотекаКартинок.ИнтервалДат; // Для управляемых форм  Элементы.Надпись.КартинкаКнопкиВыбора = БиблиотекаКартинок.ИнтервалДат; |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [БиблиотекаКартинок](Глобальный_контекст.БиблиотекаКартинок_(Global_context.PictureLib).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.25.
