КолонкиСписка (ListColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: Строка.   
Имя колонки.

Возвращаемое значение:

Тип: [КолонкаСписка](КолонкаСписка.md), Неопределено.   
Если колонка не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск указанной колонки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
