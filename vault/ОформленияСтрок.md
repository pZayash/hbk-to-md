Доступен, начиная с версии 8.0.

Элементы коллекции:

[ОформлениеСтроки](ОформлениеСтроки.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются оформления строк.  
Возможно обращение к оформлению строки посредством оператора [...]. В качестве аргумента передается индекс строки (нумерация с 0).  

Методы:

[Количество (Count)](ОформленияСтрок.Количество.md)  
[Получить (Get)](ОформленияСтрок.Получить.md)  

Описание:

Представляет собой коллекцию значений типа [ОформлениеСтроки](ОформлениеСтроки.md).

Доступность:

Толстый клиент.
