ЗначенияПараметровДанныхКомпоновкиДанных (DataCompositionDataParameterValues)

НайтиЗначениеПараметра (FindParameterValue)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиЗначениеПараметра(<Параметр>)

Параметры:

<Параметр> (необязательный)

Тип: [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных.md).   
Параметр, значение которого нужно найти.

Возвращаемое значение:

Тип: [ЗначениеПараметраНастроекКомпоновкиДанных](ЗначениеПараметраНастроекКомпоновкиДанных.md), Неопределено.   
Если значение не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск значения параметра.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
