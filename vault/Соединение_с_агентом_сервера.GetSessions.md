Соединение с агентом сервера (IServerAgentConnection)

GetSessions (GetSessions)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessions(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого необходимо получить список сеансов. Должен быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив, который содержит список объектов [Сеанс](Сеанс.md).

Описание:

Получает список сеансов, работающих с данным кластером.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
