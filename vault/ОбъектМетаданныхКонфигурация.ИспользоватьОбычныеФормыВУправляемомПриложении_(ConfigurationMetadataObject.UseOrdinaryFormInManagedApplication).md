ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ИспользоватьОбычныеФормыВУправляемомПриложении (UseOrdinaryFormInManagedApplication)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то при проверке модулей для режима работы "Толстый клиент (приложение)" и "Толстый клиент (приложение) вариант клиент-сервер" модуль обычной формы не проверяется. При этом при получении формы по умолчанию в толстом клиенте в режиме приложения, если в качестве формы по умолчанию используется обычная форма, будет сгенерирована форма клиентского приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
