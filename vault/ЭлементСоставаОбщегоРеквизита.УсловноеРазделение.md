ЭлементСоставаОбщегоРеквизита (CommonAttributeContentItem)

УсловноеРазделение (ConditionalSeparation)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: Константа](ОбъектМетаданных_Константа.md), Булево.   
Содержит условия использования общего реквизита как разделителя данного объекта. Может быть константой логического типа, не входящей в состав общего реквизита, или реквизитом логического типа объекта метаданных, не входящего в состав общего реквизита, причем тип ссылки, образуемый этим объектом метаданных, является типом общего реквизита.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
