ПоддержкаМасштаба (ScaleKeeping)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ПоддержкаМасштабаДиаграммыГанта](ПоддержкаМасштабаДиаграммыГанта_(GanttChartScaleKeeping).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
