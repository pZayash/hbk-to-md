ПараметрВыбора (ChoiceParameter)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ПараметрВыбора(<Имя>, <Значение>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Значение присваивается свойству Имя данного объекта.

<Значение> (обязательный)

Может быть любого типа. Значение присваивается свойству [Значение](ПараметрВыбора.Значение.md).

Описание:

Создает объект и заполняет его свойства значениями из параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
