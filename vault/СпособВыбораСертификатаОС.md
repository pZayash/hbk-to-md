# СпособВыбораСертификатаОС

Доступен, начиная с версии 8.3.21.

Значения

[Авто (Auto)](СпособВыбораСертификатаОС.Авто.md)  
[Выбирать (Choose)](СпособВыбораСертификатаОС.Выбирать.md)  

Описание:

Содержит варианты способов выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы.  
При работе в ОС Linux значение игнорируется.  
В ОС Windows и macOS действует аналогично объектам [СпособВыбораСертификатаWindows](СпособВыбораСертификатаWindows.md) и [СпособВыбораСертификатаMacOS](СпособВыбораСертификатаMacOS.md).

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

См. также:

[СертификатКлиентаОС](СертификатКлиентаОС.md)  

Использование в версии:

Доступен, начиная с версии 8.3.21.

Описание изменено в версии 8.3.24.
