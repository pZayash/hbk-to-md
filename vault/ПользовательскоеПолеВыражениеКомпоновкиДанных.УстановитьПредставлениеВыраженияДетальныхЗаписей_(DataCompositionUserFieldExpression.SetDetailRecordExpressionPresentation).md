ПользовательскоеПолеВыражениеКомпоновкиДанных (DataCompositionUserFieldExpression)

УстановитьПредставлениеВыраженияДетальныхЗаписей (SetDetailRecordExpressionPresentation)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьПредставлениеВыраженияДетальныхЗаписей(<ПользовательскоеПредставлениеВыражения>)

Параметры:

<ПользовательскоеПредставлениеВыражения> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пользовательское представление выражения, используемое при выводе поля в детальных записях.

Описание:

Устанавливает пользовательское представление выражения, используемое при выводе поля в детальных записях.  
В случае, если настройки связаны с источником доступных настроек, выражение тоже будет обновлено.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
