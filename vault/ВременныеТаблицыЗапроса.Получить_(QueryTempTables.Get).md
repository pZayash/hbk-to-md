ВременныеТаблицыЗапроса (QueryTempTables)

Получить (Get)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ВременнаяТаблицаЗапроса](ВременнаяТаблицаЗапроса_(QueryTempTable).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.3.8.
