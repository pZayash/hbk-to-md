ПараметрВыбора (ChoiceParameter)

Значение (Value)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Содержит конкретное значение, которое будет поставлено в отбор при генерации формы выбора. Может принимать любые значения, кроме [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
