МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ВызватьИсходноеСобытие (CallSourceEvent)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ВызватьИсходноеСобытие(<Событие>, <Параметры>)

Параметры:

<Событие> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя исходного нестандартного события из настроек регистрации событий/экранов статистики использования приложения. Если переданное событие не найдено, вызов игнорируется.

<Параметры> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, выступающая в качестве объекта, порождающего исходное событие.

Описание:

Вызывает исходное нестандартное событие.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
