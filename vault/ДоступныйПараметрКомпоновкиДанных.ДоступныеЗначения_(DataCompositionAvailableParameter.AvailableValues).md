ДоступныйПараметрКомпоновкиДанных (DataCompositionAvailableParameter)

ДоступныеЗначения (AvailableValues)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [СписокЗначений](СписокЗначений_(ValueList).md), [Неопределено](lang__def_Undefined.md).   
Содержит список значений, которые может принимать значение параметра.  
[Неопределено](lang__def_Undefined.md) - значения поля неограничены.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
