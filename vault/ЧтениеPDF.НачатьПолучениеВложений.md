ЧтениеPDF (PDFReader)

НачатьПолучениеВложений (BeginGetAttachments)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПолучениеВложений(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <Вложения> - объект типа [КоллекцияВложенийPDF](КоллекцияВложенийPDF.md), содержащий вложения PDF-документов.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Инициирует получение массива, содержащего описания присоединенных к PDF-документу вложений.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [ПолучитьВложения](ЧтениеPDF.ПолучитьВложения.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [ПолучитьВложенияАсинх](ЧтениеPDF.ПолучитьВложенияАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
