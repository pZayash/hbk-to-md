СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СочетаниеКлавиш](СочетаниеКлавиш_(Shortcut).md).   
Содержит сочетание клавиш, при нажатии на которые произойдет выполнение команды.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
