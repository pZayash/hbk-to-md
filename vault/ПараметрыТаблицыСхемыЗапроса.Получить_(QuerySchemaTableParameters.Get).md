ПараметрыТаблицыСхемыЗапроса (QuerySchemaTableParameters)

Получить (Get)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ПараметрТаблицыСхемыЗапроса](ПараметрТаблицыСхемыЗапроса_(QuerySchemaTableParameter).md).   

Описание:

Получает элемент коллекции по индексу. Если передаваемый индекс больше количества элементов в коллекции, то добавляется новый элемент по переданному индексу. При использовании оператора [], если индекс выходит за границы коллекции, происходит ошибка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
