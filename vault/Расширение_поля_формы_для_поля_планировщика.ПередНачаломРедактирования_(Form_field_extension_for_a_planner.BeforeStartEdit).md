Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередНачаломРедактирования (BeforeStartEdit)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередНачаломРедактирования(<НовыйЭлемент>, <СтандартнаяОбработка>)

Параметры:

<НовыйЭлемент>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак указывает, что создается новый элемент, а не редактируется существующий.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события - отображение большой формы полного редактирования элемента - производиться не будет.   
Элементы, над которыми выполнялось действие, можно получить с помощью свойства [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы_(Form_field_extension_for_a_planner.SelectedItems).md) поля.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при нажатии кнопки "Изменить" в окне быстрого редактирования, при выборе команды "Изменить" в контексном меню, после события [Выбор](Расширение_поля_формы_для_поля_планировщика.Выбор_(Form_field_extension_for_a_planner.Selection).md), если разрешена стандартная обработка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
