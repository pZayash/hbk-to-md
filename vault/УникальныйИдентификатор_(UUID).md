# УникальныйИдентификатор

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.0.

Конструкторы:

[Из строки](УникальныйИдентификатор.Из_строки.md)  
[По умолчанию](УникальныйИдентификатор.По_умолчанию.md)  
[С указанием версии](УникальныйИдентификатор.С_указанием_версии.md)  

Описание:

Предназначен для создания и хранения глобального уникального идентификатора GUID.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/core}. Имя типа XDTO: UUID.

Пример:

|  |
| --- |
| КлючУникальности = Новый УникальныйИдентификатор; |
