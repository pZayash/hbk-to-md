ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Родитель (Parent)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   
Содержит ссылку на родителя элемента плана счетов. Для элементов верхнего уровня содержит пустую ссылку.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл только для многоуровневых планов счетов.

Использование в версии:

Доступен, начиная с версии 8.0.
