РасширениеКонфигурации (ConfigurationExtension)

ЗащитаОтОпасныхДействий (UnsafeActionProtection)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [ОписаниеЗащитыОтОпасныхДействий](ОписаниеЗащитыОтОпасныхДействий.md).   
Устанавливает использование защиты от опасных действий для расширения. Свойство также влияет на выполнение метода расширения [Записать](РасширениеКонфигурации.Записать.md).  
Если защита отключена, также отключается отображение предупреждений во время записи расширений.  
Изменение свойства возможно только при наличии у текущего пользователя права "Администрирование".  
Чтобы сохранить значение свойства, следует использовать метод [Записать](РасширениеКонфигурации.Записать.md) объекта [РасширениеКонфигурации](РасширениеКонфигурации.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.12.
