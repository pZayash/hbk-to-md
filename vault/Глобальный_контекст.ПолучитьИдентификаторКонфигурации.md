Глобальный контекст (Global context)

ПолучитьИдентификаторКонфигурации (GetConfigurationID)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьИдентификаторКонфигурации()

Возвращаемое значение:

Тип: Строка.   
Строка, содержащая зашифрованный и закодированный в Base64 идентификатор конфигурации.

Описание:

Возвращает идентификатор конфигурации, с которой работает информационная база. Идентификатор составляется путем анализа и учета метаданных конфигурации.

Доступность:

Сервер.

Примечание:

Доступен только пользователям с административными правами.

Пример:

|  |
| --- |
| ИдентификаторКонфигурации = ПолучитьИдентификаторКонфигурации(); |

Использование в версии:

Доступен, начиная с версии 8.3.7.
