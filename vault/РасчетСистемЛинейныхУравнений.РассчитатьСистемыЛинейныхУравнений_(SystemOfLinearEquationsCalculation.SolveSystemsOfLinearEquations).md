РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

РассчитатьСистемыЛинейныхУравнений (SolveSystemsOfLinearEquations)

Доступен, начиная с версии 8.3.14.

Синтаксис:

РассчитатьСистемыЛинейныхУравнений(<РазрешитьРедуцированиеГрафа>)

Параметры:

<РазрешитьРедуцированиеГрафа> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - механизм решения использует все возможные оптимизации решения.   
[Ложь](lang__def_BooleanFalse.md) - механизм решения разделяет граф на компоненты и начинает решать их, пропуская этап редуцирования графа.   
Данный параметр не влияет на процесс выбора алгоритма решения, определяемого свойством [ГраницаИзмененияАлгоритмаРешения](РасчетСистемЛинейныхУравнений.ГраницаИзмененияАлгоритмаРешения_(SystemOfLinearEquationsCalculation.AlgorithmSelectionBorder).md), а только отключает дополнительные оптимизации.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   
Таблица значений содержит следующие столбцы:

- столбец с номерами переменных (НомерУзла),
- К столбцов с решениями (Решение1(Solution1),..., РешениеК(SolutionK))

где К - количество систем описанных в [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений_(SystemDescriptions).md).

Описание:

Решает системы уравнений и возвращает объект [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
