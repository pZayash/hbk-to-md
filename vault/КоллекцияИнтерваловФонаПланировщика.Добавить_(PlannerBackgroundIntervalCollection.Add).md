КоллекцияИнтерваловФонаПланировщика (PlannerBackgroundIntervalCollection)

Добавить (Add)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Добавить(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата начала интервала фона.

<Конец> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата конца интервала фона.

Возвращаемое значение:

Тип: [ИнтервалФонаПланировщика](ИнтервалФонаПланировщика_(PlannerBackgroundInterval).md).   

Описание:

Добавляет интервал фона в конец коллекции интервалов фона.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
