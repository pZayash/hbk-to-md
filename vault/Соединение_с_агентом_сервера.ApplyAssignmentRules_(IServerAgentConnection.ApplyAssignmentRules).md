# Соединение с агентом сервера.ApplyAssignmentRules

Соединение с агентом сервера (IServerAgentConnection)

ApplyAssignmentRules (ApplyAssignmentRules)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ApplyAssignmentRules(<Кластер>, <Режим>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должно быть применено изменение настроек.

<Режим> (обязательный)

Тип: [Число](lang__def_Number.md).   
Способ применения изменений конфигурации кластера.

- 0 - частичное. В изменении назначения не могут принимать участия сервисы, использующие файловые хранилища данных:

- сервис журналов регистрации;
- сервис полнотекстового поиска;
- сервис пользовательских настроек.

- 1 - полное. Выполняется переназначение всех сервисов.

Описание:

Применяет требования назначения.  
После выполнения метода внесенные изменения в конфигурацию кластера начинают оказывать влияние на управление работой кластера.  
Если изменения настроек кластера противоречиво, то применение изменений не выполняется и порождается исключение.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
