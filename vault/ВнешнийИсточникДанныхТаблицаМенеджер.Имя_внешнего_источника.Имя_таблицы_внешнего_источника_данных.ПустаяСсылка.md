ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.2.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   

Описание:

Получает пустое значение ссылки на таблицу внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.  
Только для таблиц с типом данных [ОбъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.ОбъектныеДанные.md).

Использование в версии:

Доступен, начиная с версии 8.2.
