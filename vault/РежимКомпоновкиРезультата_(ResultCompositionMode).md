**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Система компоновки данных](Система_компоновки_данных.md)

Доступен, начиная с версии 8.2.

Значения

[Авто (Auto)](РежимКомпоновкиРезультата.Авто_(ResultCompositionMode.Auto).md)  
[Непосредственно (Directly)](РежимКомпоновкиРезультата.Непосредственно_(ResultCompositionMode.Directly).md)  
[Фоновый (Background)](РежимКомпоновкиРезультата.Фоновый_(ResultCompositionMode.Background).md)  

Описание:

Содержит возможные варианты компоновки результата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise}. Имя типа XDTO: ResultCompositionMode.

Использование в версии:

Доступен, начиная с версии 8.2.
