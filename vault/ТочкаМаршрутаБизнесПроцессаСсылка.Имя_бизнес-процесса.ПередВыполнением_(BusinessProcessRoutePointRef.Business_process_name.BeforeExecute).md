# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередВыполнением

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередВыполнением (BeforeExecute)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md), в которой производится проверка выполнения.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md).   
Выполняемая задача.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то выполнение задачи произведено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при вызове метода [ВыполнитьЗадачу](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачу_(TaskObject.Task_name.ExecuteTask).md) у задачи и после обработки задачей события [ПередВыполнением](ЗадачаОбъект.Имя_задачи.ПередВыполнением_(TaskObject.Task_name.BeforeExecute).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md).  
В этом обработчике не допускаются интерактивные вызовы (открытие форм, предупреждений).

Использование в версии:

Доступен, начиная с версии 8.0.
