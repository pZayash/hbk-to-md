КоллекцияДействийЭлементаПланировщика (PlannerItemActionCollection)

Найти (Find)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению действия элемента планировщика.

Возвращаемое значение:

Тип: [ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика.md), Неопределено.   

Описание:

Осуществляет поиск действия элемента планировщика по значению свойства [Значение](ДействиеЭлементаПланировщика.Значение.md). Возвращает первое действие с совпадающим значением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
