ОбъектМетаданных: Таблица (MetadataObject: Table)

ВыражениеВИсточникеДанных (ExpressionInDataSource)

Доступен, начиная с версии 8.3.4.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение языка запросов источника с параметрами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для таблиц со значением [Выражение](ВидТаблицыВнешнегоИсточникаДанных.Выражение_(ExternalDataSourceTableType.Expression).md) свойства [ВидТаблицы](ОбъектМетаданных_Таблица.ВидТаблицы_(MetadataObject_Table.TableType).md).

Использование в версии:

Доступен, начиная с версии 8.3.4.
