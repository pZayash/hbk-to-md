# ПланСчетовОбъект.<Имя плана счетов>.ОбменДанными

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Свойства](ПланСчетовОбъект.Имя_плана_счетов__Свойства.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md).   
Используется для управления обменом данных.  
С помощью данного свойства настраивается состав узлов-получателей, для которых будут регистрироваться изменения данных, узел-отправитель, из которого получена записываемая информация, а также устанавливается режим [Загрузка](ПараметрыОбменаДанными.Загрузка_(DataExchangeParameters.Load).md), указывающий, что выполняется перенос информации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
