Диаграмма (Chart)

ПорядокОтображенияТочекГоризонтальнойГистограммы (BarChartPointsOrder)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [ПорядокОтображенияТочекГоризонтальнойГистограммы](ПорядокОтображенияТочекГоризонтальнойГистограммы_(BarChartPointsOrder).md).   
Определяет порядок вывода точек в горизонтальных диаграммах.  
Используется только для диаграмм следующих видов:

- [ГистограммаГоризонтальная](ТипДиаграммы.ГистограммаГоризонтальная_(ChartType.Bar).md),
- [ГистограммаГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаГоризонтальнаяОбъемная_(ChartType.Bar3D).md),
- [ГистограммаНормированнаяГоризонтальная](ТипДиаграммы.ГистограммаНормированнаяГоризонтальная_(ChartType.NormalizedBar).md),
- [ГистограммаНормированнаяГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаНормированнаяГоризонтальнаяОбъемная_(ChartType.NormalizedBar3D).md),
- [ГистограммаСНакоплениемГоризонтальная](ТипДиаграммы.ГистограммаСНакоплениемГоризонтальная_(ChartType.StackedBar).md),
- [ГистограммаСНакоплениемГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаСНакоплениемГоризонтальнаяОбъемная_(ChartType.StackedBar3D).md)

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
