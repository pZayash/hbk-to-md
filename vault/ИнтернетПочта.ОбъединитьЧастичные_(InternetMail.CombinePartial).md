ИнтернетПочта (InternetMail)

ОбъединитьЧастичные (CombinePartial)

Доступен, начиная с версии 8.1.

Синтаксис:

ОбъединитьЧастичные(<МассивСообщений>)

Параметры:

<МассивСообщений> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий частичные сообщения для объединения.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md).

Описание:

Объединяет частичные сообщения. Если в переданном массиве находятся не все части, то сообщение не объединяется. В передаваемом массиве могут находится части нескольких сообщений. После объединения они возвратятся в массиве. У каждого объединенного сообщения свойство [Идентификатор](ИнтернетПочтовоеСообщение.Идентификатор_(InternetMailMessage.UID).md) будет содержать несколько значений, соответствующих взятым из частичных сообщений при объединении.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
