ДанныеМультимедиа (MultimediaData)

По имени файла

Доступен, начиная с версии 8.3.25.

Синтаксис:

Новый ДанныеМультимедиа(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, из которого будет загружено значение в виде двоичных данных.

Описание:

Создает объект и помещает в него двоичные данные, взятые из файла. Определяет тип данных в стандарте MIME по расширению файла.

Использование в версии:

Доступен, начиная с версии 8.3.25.
