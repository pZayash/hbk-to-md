# ТаблицаЗначений.ВыгрузитьКолонку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Таблица значений](Таблица_значений.md) › [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md) › [Методы](ТаблицаЗначений__Методы.md)

ТаблицаЗначений (ValueTable)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [КолонкаТаблицыЗначений](КолонкаТаблицыЗначений_(ValueTableColumn).md).   
Колонка, значения которой необходимо выгрузить. В качестве значения параметра может выступать индекс колонки, имя колонки, сама колонка.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Создает массив и копирует в него значения, содержащиеся в колонке таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТаблицаЗначений.Свернуть("Номенклатура"); Запрос.УстановитьПараметр("МассивНоменклатуры", ТаблицаЗначений.ВыгрузитьКолонку("Номенклатура")); |

Использование в версии:

Доступен, начиная с версии 8.0.
