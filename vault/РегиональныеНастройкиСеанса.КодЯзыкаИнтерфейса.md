КодЯзыкаИнтерфейса (InterfaceLanguageCode)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Код языка интерфейса платформы. Поддерживаются следующие коды:

- az - Азербайджанский,
- en - Английский,
- ar - Арабский,
- hy - Армянский,
- bg - Болгарский,
- hu - Венгерский,
- vi - Вьетнамский,
- el - Греческий,
- ka - Грузинский,
- el - Испанский,
- it - Итальянский,
- kk - Казахский,
- zh - Китайский,
- lv - Латышский,
- lt - Литовский,
- de - Немецкий,
- pl - Польский,
- ro - Румынский,
- ru - Русский,
- tr - Турецкий,
- uk - Украинский,
- fr - Французский.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
