ДиаграммаГанта (GanttChart)

ПриАктивизацииИнтервала (OnActivateInterval)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПриАктивизацииИнтервала()

Описание:

Вызывается при активизации интервала диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
