# Глобальный контекст.ПодключитьРасширениеРаботыСФайламиАсинх

Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСФайламиАсинх (AttachFileSystemExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПодключитьРасширениеРаботыСФайламиАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Истина](lang__def_BooleanTrue.md), расширение успешно подключено, в противном случае - [Ложь](lang__def_BooleanFalse.md).  
  
Для тонкого клиента и клиента мобильного приложения результат всегда [Истина](lang__def_BooleanTrue.md).

Описание:

Асинхронная функция. При первом обращении выполняет подключение расширения для работы с файлами.  
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
