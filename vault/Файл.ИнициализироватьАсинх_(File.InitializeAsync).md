Файл (File)

ИнициализироватьАсинх (InitializeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ИнициализироватьАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла или каталога, с которым будет связан объект.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - инициализированный объект типа [Файл](Файл_(File).md).

Описание:

Асинхронная функция. Начинает инициализацию объекта [Файл](Файл_(File).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Файл](Файл_(File).md), метод [НачатьИнициализацию](Файл.НачатьИнициализацию_(File.BeginInitialization).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
