НастройкаОбластиОформления (AppearanceAreaSetting)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента управления областью оформления.

<Доступность> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойОбластиОформления](ЭлементУправленияНастройкойОбластиОформления_(AppearanceAreasSettingControl).md).   

Описание:

Добавляет элемент управления областью оформления в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
