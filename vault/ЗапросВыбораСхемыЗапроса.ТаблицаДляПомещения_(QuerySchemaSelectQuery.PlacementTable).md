ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

ТаблицаДляПомещения (PlacementTable)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя временной таблицы, которая будет создана при выполнении запроса и в которую будет помещен результат запроса.  
Соответствует ключевому слову ПОМЕСТИТЬ языка запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
