ЗаписьFastInfoset (FastInfosetWriter)

ЗаписатьНачалоАтрибута (WriteStartAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Запись с полным именем атрибута

Синтаксис:

ЗаписатьНачалоАтрибута(<ПолноеИмя>)

Параметры:

<ПолноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя записываемого атрибута. Может включать префикс. Например: "nsp:weight".

Вариант синтаксиса: Запись с локальным именем и URI пространства имен

Синтаксис:

ЗаписатьНачалоАтрибута(<ЛокальноеИмя>, <URIПространстваИмен>)

Параметры:

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута (без префикса).

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен для имени атрибута.

Описание варианта метода:

Соответствие пространства имен должно быть записано при записи вышестоящего атрибута или при записи данного атрибута.

Описание:

Записывает начало атрибута. Может вызываться после вызова метода [ЗаписьFastInfoset.ЗаписатьНачалоЭлемента](ЗаписьFastInfoset.ЗаписатьНачалоЭлемента_(FastInfosetWriter.WriteStartElement).md), но до вызова методов [ЗаписьFastInfoset.ЗаписатьКонецЭлемента](ЗаписьFastInfoset.ЗаписатьКонецЭлемента_(FastInfosetWriter.WriteEndElement).md), [ЗаписьFastInfoset.ЗаписатьТекст](ЗаписьFastInfoset.ЗаписатьТекст_(FastInfosetWriter.WriteText).md), [ЗаписьFastInfoset.ЗаписатьКомментарий](ЗаписьFastInfoset.ЗаписатьКомментарий_(FastInfosetWriter.WriteComment).md), [ЗаписьFastInfoset.ЗаписатьСекциюCDATA](ЗаписьFastInfoset.ЗаписатьСекциюCDATA_(FastInfosetWriter.WriteCDATASection).md) или [ЗаписьFastInfoset.ЗаписатьНачалоАтрибута](ЗаписьFastInfoset.ЗаписатьНачалоАтрибута_(FastInfosetWriter.WriteStartAttribute).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
