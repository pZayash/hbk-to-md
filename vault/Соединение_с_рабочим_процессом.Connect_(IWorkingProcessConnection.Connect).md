Соединение с рабочим процессом (IWorkingProcessConnection)

Connect (Connect)

Доступен, начиная с версии 8.1.

Синтаксис:

Connect(<ИнформационнаяБаза>, <ИмяПользователя>, <Пароль>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база_(IInfoBaseInfo).md).   
Информационная база, для которой должно быть установлено значение свойства [Name](Информационная_база.Name_(IInfoBaseInfo.Name).md).

<ИмяПользователя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя информационной базы 1С:Предприятия, который устанавливает соединение.

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя информационной базы. Если пароля нет, то нужно указать пустую строку.

Возвращаемое значение:

Тип: [Внешнее соединение](Внешнее_соединение_(External_connection).md).   

Описание:

Устанавливает соединение с информационной базой, зарегистрированной в кластере серверов 1С:Предприятия, аналогичное результату метода [Connect](Менеджер_COM-соединений.Connect_(COM_connector.Connect).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
