ЭлементЯчейкаТаблицыHTML (HTMLTableCellElement)

ПолучитьЭлементыПоИмени (GetElementByTagName)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЭлементыПоИмени(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: Строка.   
Имя требующегося элемента (коллекции элементов).

Возвращаемое значение:

Тип: [СписокЭлементовDOM](СписокЭлементовDOM.md).   
Список узлов DOM (элементов), являющихся дочерними для данного узла, в порядке их следования в документе.

Описание:

Выбирает все элементы с указанным именем.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может использоваться предопределенный шаблон '\*', означающий необходимость выбора всех узлов [Элемент](ТипУзлаDOM.Элемент.md) дерева.

Использование в версии:

Доступен, начиная с версии 8.2.
