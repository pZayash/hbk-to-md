ЭлементФормаHTML (HTMLFormElement)

ПоддерживаемыеНаборыЗнаков (AcceptCharset)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Список кодировок символов, используемых при вводе данных, которые должны быть приняты обрабатывающим эту форму сервером.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
