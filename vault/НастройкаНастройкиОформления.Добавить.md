НастройкаНастройкиОформления (AppearanceSettingSettings)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента управления настройкой.

<Доступность> (обязательный)

Тип: Булево.   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойНастройкиОформления](ЭлементУправленияНастройкойНастройкиОформления.md).   

Описание:

Добавляет элемент управления настройкой оформления в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
