ОбъектМетаданных: WebСервис (MetadataObject: WebService)

ВремяЖизниСеанса (SessionMaxAge)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Время бездействия сеанса (в секундах), после которого он завершается. Действует в случае, если свойство &РежимПовторногоИспользованияСеансов установлено в [Использовать](РежимПовторногоИспользованияСеансов.Использовать_(SessionReuseMode.Use).md) или [ИспользоватьАвтоматически](РежимПовторногоИспользованияСеансов.ИспользоватьАвтоматически_(SessionReuseMode.AutoUse).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
