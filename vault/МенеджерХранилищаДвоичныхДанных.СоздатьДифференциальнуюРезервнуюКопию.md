МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

СоздатьДифференциальнуюРезервнуюКопию (CreateDifferentialBackup)

Доступен, начиная с версии 8.3.23.

Синтаксис:

СоздатьДифференциальнуюРезервнуюКопию(<Путь>, <БазовыйПуть>, <УниверсальнаяДатаДоКоторойВыполнятьОчистку>)

Параметры:

<Путь> (обязательный)

Тип: Строка.   
Путь к файлу с дифференциальной резервной копией.

<БазовыйПуть> (обязательный)

Тип: Строка.   
Путь к файлу с полной резервной копией, от которой будет создаваться дифференциальная резервная копия.

<УниверсальнаяДатаДоКоторойВыполнятьОчистку> (необязательный)

Тип: Дата.   
Указывает дату и время в формате UTC, до которого будет выполнена однократная очистка неиспользуемого места для данных после успешного окончания резервного копирования.  
  
Если Неопределено – очистка не будет выполнена.  
Значение по умолчанию: Неопределено.

Описание:

Создает дифференциальную резервную копию хранилища двоичных данных  
Дифференциальная резервная копия - это резервная копия, которая хранит только изменения относительно момента начала создания полной резервной копии.   
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены.  
Формат создаваемого файла: \*.sbd (storage differential backup).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.26.
