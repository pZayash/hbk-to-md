НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.25.

Только чтение.

Содержит навигационную ссылку формы. Получаемая ссылка идентична ссылке, которую можно получить с помощью свойства [НавигационнаяСсылка](ТестируемаяКнопкаКомандногоИнтерфейса.НавигационнаяСсылка_(TestedCommandInterfaceButton.URL).md) в панели открытых.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.25.
