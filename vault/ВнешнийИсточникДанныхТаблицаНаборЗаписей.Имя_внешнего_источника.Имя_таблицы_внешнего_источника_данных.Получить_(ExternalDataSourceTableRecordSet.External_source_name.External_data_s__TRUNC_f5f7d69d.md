Получить (Get)

Доступен, начиная с версии 8.2.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс записи.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаЗапись.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаЗапись.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecord.External_source_name.Table_of_external_data_source_name).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.2.
