РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.1.

Синтаксис:

Выгрузить(<Строки>, <Колонки>)

Параметры:

<Строки> (необязательный)

Тип: [Массив](Массив.md).   
Массив строк для выгрузки. Если не указан, выгружаются все строки набора записей.

<Колонки> (необязательный)

Тип: Строка.   
Список колонок для выгрузки в формате: "Колонка1, Колонка2...". Если список не задан, то будут выгружены все колонки.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   

Описание:

Создает таблицу значений и копирует в нее записи набора. Структура полученной таблицы совпадает со структурой набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Рег = РегистрыСведений.КурсыВалют; НаборЗаписей = Рег.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.Валюта.Значение = Ссылка; ТаблицаКурсов = НаборЗаписей.Выгрузить(); |

Использование в версии:

Доступен, начиная с версии 8.1.
