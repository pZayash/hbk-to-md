ОформлениеЯчейки (CellAppearance)

ЗначениеКартинки (PictureValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит значение картинки, отображаемой в колонке. Если колонка, отображающая данное значение связана с данными, то данное свойство доступно только для чтения.

Доступность:

Толстый клиент.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
