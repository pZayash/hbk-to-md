Соединение с агентом сервера (IServerAgentConnection)

GetClusterManagers (GetClusterManagers)

Доступен, начиная с версии 8.2.

Синтаксис:

GetClusterManagers(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер серверов. Он может быть элементом массива кластеров, зарегистрированных в текущем центральном сервере, полученного методом [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md). Также кластер может быть создан методом [CreateClusterInfo](Соединение_с_агентом_сервера.CreateClusterInfo_(IServerAgentConnection.CreateClusterInfo).md) с последующим заполнением его свойств.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [Менеджер кластера](Менеджер_кластера_(IClusterManagerInfo).md). Один из них представляет главный менеджер кластера, а остальные – дополнительные менеджеры, зарегистрированные в кластере.

Описание:

Получает список менеджеров, зарегистрированных в кластере.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
