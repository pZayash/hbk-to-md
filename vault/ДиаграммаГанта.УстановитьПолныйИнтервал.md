<!-- signature:end -->

ДиаграммаГанта (GanttChart)

УстановитьПолныйИнтервал (SetWholeInterval)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПолныйИнтервал(<ДатаНачала>, <ДатаОкончания>)

Параметры:

<ДатаНачала> (обязательный)

Тип: Дата.   
Дата начала интервала. Не может быть меньше даты "01.01.1000".

<ДатаОкончания> (обязательный)

Тип: Дата.   
Дата окончания интервала. Не может быть больше даты "01.01.3000".

Описание:

Устанавливает полный интервал.  
Следует использовать при [АвтоОпределениеПолногоИнтервала](ДиаграммаГанта.АвтоОпределениеПолногоИнтервала.md) = Ложь.   
ВНИМАНИЕ! Установку полного интервала можно производить только с помощью этого метода. Свойства [НачалоПолногоИнтервала](ДиаграммаГанта.НачалоПолногоИнтервала.md) и [КонецПолногоИнтервала](ДиаграммаГанта.КонецПолногоИнтервала.md) только для чтения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ДиаграммаГанта](ДиаграммаГанта.md), свойство [НачалоПолногоИнтервала](ДиаграммаГанта.НачалоПолногоИнтервала.md)  
[ДиаграммаГанта](ДиаграммаГанта.md), свойство [КонецПолногоИнтервала](ДиаграммаГанта.КонецПолногоИнтервала.md)  
[ДиаграммаГанта](ДиаграммаГанта.md), свойство [АвтоОпределениеПолногоИнтервала](ДиаграммаГанта.АвтоОпределениеПолногоИнтервала.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
