МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ЗагрузитьЗапросНаПолучениеЛицензии (LoadLicenseAcquisitionRequest)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ЗагрузитьЗапросНаПолучениеЛицензии()

Возвращаемое значение:

Тип: [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md), Неопределено.   
Сохраненный [ЗапросНаПолучениеЛицензии](ЗапросНаПолучениеЛицензии.md) или Неопределено, если нет сохраненных данных.

Описание:

Загружает сохраненные данные для получения лицензии.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
