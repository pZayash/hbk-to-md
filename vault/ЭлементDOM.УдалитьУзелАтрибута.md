ЭлементDOM (DOMElement)

УдалитьУзелАтрибута (RemoveAttributeNode)

Доступен, начиная с версии 8.1.

Синтаксис:

УдалитьУзелАтрибута(<Атрибут>)

Параметры:

<Атрибут> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md).   
Удаляемый атрибут элемента.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md).   
Удаленный атрибут.

Описание:

Удаляет атрибут из элемента.  
В случае удаления атрибута, не явно указанного в элементе, а взятого из схемы документа, будет создан новый узел [Атрибут](ТипУзлаDOM.Атрибут.md), и в него будет установлено значение по умолчанию.  
Причины вызова исключений:

- указанный атрибут не является атрибутом данного элемента;
- данный узел в режиме Только для чтения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
