ПоддержкаМасштабаСводнойДиаграммы (PivotChartScaleKeeping)

МинимальнаяШирина (MinimumWidth)

Доступен, начиная с версии 8.0.

Описание:

Минимальная ширина элемента задается свойством [МинимальнаяШиринаЗначенияГоризонтальнойШкалы](СводнаяДиаграмма.МинимальнаяШиринаЗначенияГоризонтальнойШкалы_(PivotChart.HorizontalScaleValueMinWidth).md). Если размер области построения не позволяет отобразить все элементы с шириной каждого, не меньше заданной, то используется прокрутка.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.21.
