# МенеджерПроверкиВстроенныхПокупок.ПроверитьКвитанциюВстроеннойПокупкиНаМобильномУстройстве

МенеджерПроверкиВстроенныхПокупок (InAppPurchasesValidationManager)

ПроверитьКвитанциюВстроеннойПокупкиНаМобильномУстройстве (ValidateInAppPurchaseReceiptAtMobileDevice)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПроверитьКвитанциюВстроеннойПокупкиНаМобильномУстройстве(<Квитанция>)

Параметры:

<Квитанция> (обязательный)

Тип: [КвитанцияВстроеннойПокупки](КвитанцияВстроеннойПокупки_(InAppPurchaseReceipt).md).   
Квитанция, которую необходимо проверить.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Выполняет локальную проверку квитанции на устройстве:

- Если квитанция подлинная - [Истина](lang__def_BooleanTrue.md),
- Иначе - [Ложь](lang__def_BooleanFalse.md).

Для наиболее полной проверки подлинности совершенной покупки рекомендуется также проверять валидность покупки на сервере с помощью объекта платформы для ПК.  
  
В режиме совместимости с 8.3.18 и последующими, проверка квитанции на устройствах под управлением ОС Android требует наличия ключа проверки встроенных покупок в ресурсах приложения, который указывается в сборщике мобильных приложений в настройках приложения под Android. Если ключ не найден, то генерируется исключение "Ключ проверки квитанции не найден в ресурсах приложения".

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
