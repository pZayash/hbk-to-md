РамкаГруппы (GroupBox)

ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона рамки группы.

Доступность:

Толстый клиент.

Примечание:

Не используется, если свойство [ПрозрачныйФон](РамкаГруппы.ПрозрачныйФон_(GroupBox.Transparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
