ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

РаспределеннаяИнформационнаяБаза (DistributedInfoBase)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то данный план обмена может быть задействован для организации распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
