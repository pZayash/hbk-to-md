ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать()

Описание:

Считывает данные задачи из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Позволяет прочесть данные заново. Недопустим для нового объекта.

Пример:

|  |
| --- |
| Задача.Прочитать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
