Глобальный контекст (Global context)

ПриПробужденииКлиентскогоПриложения (OnClientApplicationResume)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПриПробужденииКлиентскогоПриложения()

Описание:

Вызывается при активизации приложения из фонового режима.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.20.
