ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ВысотаСтроки (RowHeight)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит высоту строк в области (в пунктах). Если свойство [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки.md) установлено Истина, - содержит максимальную высоту строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При присвоении свойству значения 0, значение свойства [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки.md) выставляется в Истина. При присвоении других значений свойство [АвтоВысотаСтроки](ОбластьЯчеекТабличногоДокумента.АвтоВысотаСтроки.md) выставляется в Ложь.

Использование в версии:

Доступен, начиная с версии 8.2.
