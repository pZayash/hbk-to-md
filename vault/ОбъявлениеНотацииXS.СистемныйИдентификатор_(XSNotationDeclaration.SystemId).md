ОбъявлениеНотацииXS (XSNotationDeclaration)

СистемныйИдентификатор (SystemId)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит системный идентификатор нотации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Представляет свойство system identifier (<http://www.w3.org/TR/xmlschema-1/#system_identifier>) информационного пространства объявления.

Использование в версии:

Доступен, начиная с версии 8.1.
