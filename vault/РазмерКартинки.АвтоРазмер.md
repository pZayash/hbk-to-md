РазмерКартинки (PictureSize)

АвтоРазмер (AutoSize)

Доступен, начиная с версии 8.0.

Описание:

Если область, в которой отображается картинка, больше чем размеры картинки, используется режим [РеальныйРазмер](РазмерКартинки.РеальныйРазмер.md), в противном случае используется режим [Пропорционально](РазмерКартинки.Пропорционально.md).

Использование в версии:

Доступен, начиная с версии 8.0.
