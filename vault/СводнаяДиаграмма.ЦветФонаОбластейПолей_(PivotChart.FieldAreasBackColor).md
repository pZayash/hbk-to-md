СводнаяДиаграмма (PivotChart)

ЦветФонаОбластейПолей (FieldAreasBackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона областей полей сводной диаграммы.  
Области полей - это области, на которых расположены выбранные на соответствующей оси измерения (и, возможно, данные, если одновременно используются несколько ресурсов).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Не используется, если свойство [ПрозрачныйФонОбластейПолей](СводнаяДиаграмма.ПрозрачныйФонОбластейПолей_(PivotChart.FieldAreasTransparent).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Использование в версии:

Доступен, начиная с версии 8.0.
