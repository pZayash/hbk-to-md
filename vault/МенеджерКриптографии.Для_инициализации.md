МенеджерКриптографии (CryptoManager)

Для инициализации

Доступен, начиная с версии 8.3.6.

Синтаксис:

Новый МенеджерКриптографии(<ИспользованиеИнтерактивногоРежима>)

Параметры:

<ИспользованиеИнтерактивногоРежима> (необязательный)

Определяет режим работы модуля криптографии.

Описание:

Создает объект, который впоследствии должен быть инициализирован с помощью метода [НачатьИнициализацию](МенеджерКриптографии.НачатьИнициализацию.md). При попытке обращения к неициализированному объекту генерируется исключение.  
В веб-клиенте, если не установлено расширение работы с криптографией или если недоступно использование синхронных вызовов, при вызове конструктора генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.24.
