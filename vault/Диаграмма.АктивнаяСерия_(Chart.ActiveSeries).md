Диаграмма (Chart)

АктивнаяСерия (ActiveSeries)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Содержит серию, для которой производится построение биржевой диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для биржевых диаграмм.

См. также:

[ТипДиаграммы](ТипДиаграммы_(ChartType).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
