ТочкаДиаграммыГанта (GanttChartPoint)

ПриоритетЦвета (ColorPriority)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Вместе со свойством [ПриоритетЦвета](СерияДиаграммыГанта.ПриоритетЦвета_(GanttChartSeries.ColorPriority).md) соответствующей серии определяет цвет значения. При равных значениях свойства используется цвет серии.  
[Истина](lang__def_BooleanTrue.md) - будет использоваться значение [Цвет](ТочкаДиаграммыГанта.Цвет_(GanttChartPoint.Color).md) точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
