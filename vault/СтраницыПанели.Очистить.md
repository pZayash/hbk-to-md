СтраницыПанели (PanelPages)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все страницы из коллекции страниц панели. После выполнения этого метода панель имеет одну страницу и одну закладку.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
