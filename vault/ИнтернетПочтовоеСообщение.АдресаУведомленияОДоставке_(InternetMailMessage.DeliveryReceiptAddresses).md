ИнтернетПочтовоеСообщение (InternetMailMessage)

АдресаУведомленияОДоставке (DeliveryReceiptAddresses)

Не рекомендуется использовать, начиная с версии 8.3.13.

Рекомендуется использовать:

- [УведомитьОДоставке](ИнтернетПочтовоеСообщение.УведомитьОДоставке_(InternetMailMessage.RequestDeliveryReceipt).md)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИнтернетПочтовыеАдреса](ИнтернетПочтовыеАдреса_(InternetMailAddresses).md).   
Коллекция адресов, на которые будет высылаться уведомление о доставке.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При отправке указанные адреса будут добавлены в заголовок "Return-Receipt-To" отправляемого письма. Разные почтовые клиенты по-разному обрабатывают данный заголовок: некоторые клиенты автоматически отсылают уведомление о доставке, другие позволяют пользователю проигнорировать это действие.

См. также:

[ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md), свойство [УведомитьОДоставке](ИнтернетПочтовоеСообщение.УведомитьОДоставке_(InternetMailMessage.RequestDeliveryReceipt).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.13.

Описание изменено в версии 8.3.20.
