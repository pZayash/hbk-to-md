ПользовательСистемыВзаимодействия (CollaborationSystemUser)

АдресЭлектроннойПочтыДляАутентификации (AuthenticationEmail)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Адрес электронной почты, используемый для аутентификации внешнего пользователя.  
Если уже есть внешний пользователь с таким же значением АдресЭлектроннойПочтыДляАутентификации, то генерируется исключение.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
