ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Имя>, <Тип>, <Область>, <ОбластьИерархия>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным для измерения.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя измерения.

<Тип> (необязательный)

Тип: [ТипИзмеренияПостроителяОтчета](ТипИзмеренияПостроителяОтчета_(ReportBuilderDimensionType).md).   
Тип измерения.

<Область> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Область для вывода.

<ОбластьИерархия> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Область для иерархического вывода.

Возвращаемое значение:

Тип: [ИзмерениеПостроителяОтчета](ИзмерениеПостроителяОтчета_(ReportBuilderDimension).md).   

Описание:

Добавляет измерение в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ПостроительОтчетов.ИзмеренияСтроки.Добавить(Имя, Имя); |

Использование в версии:

Доступен, начиная с версии 8.0.
