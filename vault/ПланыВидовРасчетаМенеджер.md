# ПланыВидовРасчетаМенеджер

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПланВидовРасчетаМенеджер.\<Имя плана видов расчета\>](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры планов видов расчета.  
Возможно обращение к менеджеру плана видов расчета посредством оператора [...]. В качестве аргумента передается имя плана видов расчета, как оно задано в конфигурации.  

Свойства:

[\<Имя плана видов расчета\> (\<Chart of calculation types name\>)](ПланыВидовРасчетаМенеджер.Имя_плана_видов_расчета.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПланыВидовРасчетаМенеджер.ТипВсеСсылки.md)  

Описание:

Предназначен для управления планами видов расчета и предоставляет доступ к значениям типа [ПланВидовРасчетаМенеджер.](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.md)<[Имя плана видов расчета](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.md)>. Доступ к объекту осуществляется через свойство глобального контекста [ПланыВидовРасчета](Глобальный_контекст.ПланыВидовРасчета.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.

[Глобальный контекст](Глобальный_контекст.md), свойство [ПланыВидовРасчета](Глобальный_контекст.ПланыВидовРасчета.md)
