ФормаКлиентскогоПриложения (ClientApplicationForm)

ПолучитьРеквизиты (GetAttributes)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьРеквизиты(<Путь>)

Параметры:

<Путь> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает путь к родительскому реквизиту. Если параметр опущен или указана пустая строка, возвращаются реквизиты верхнего уровня.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов [РеквизитФормы](РеквизитФормы_(FormAttribute).md).

Описание:

Получает описание реквизитов формы.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
