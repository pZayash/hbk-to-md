НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента управления параметрами.

Возвращаемое значение:

Тип: [ЭлементУправленияПараметрамиАнализаДанных](ЭлементУправленияПараметрамиАнализаДанных_(DataAnalysisParametersControl).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
