ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Сдвинуть(<Интервал>, <Смещение>)

Параметры:

<Интервал> (обязательный)

Тип: [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы.md).   
Интервал, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: Число.   
Количество позиций, на которое необходимо сдвинуть информационный интервал.

Описание:

Сдвигает информационный интервал на указанное количество позиций.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
