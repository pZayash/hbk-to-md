Глобальный контекст (Global context)

ПриНачалеРаботыСистемы (OnStart)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриНачалеРаботыСистемы()

Описание:

Возникает при запуске в режиме 1С:Предприятие после открытия главного окна. В процедуре-обработчике могут быть выполнены различные действия, необходимые при запуске программы, например, открыты некоторые формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), событие [ПередНачаломРаботыСистемы](Глобальный_контекст.ПередНачаломРаботыСистемы_(Global_context.BeforeStart).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
