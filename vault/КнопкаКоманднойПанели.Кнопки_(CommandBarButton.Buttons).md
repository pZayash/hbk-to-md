КнопкаКоманднойПанели (CommandBarButton)

Кнопки (Buttons)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КнопкиКоманднойПанели](КнопкиКоманднойПанели_(CommandBarButtons).md).   
Содержит коллекцию кнопок, связанную с данной. Имеет смысл только для кнопок типа [Подменю](ТипКнопкиКоманднойПанели.Подменю_(CommandBarButtonType.Popup).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
