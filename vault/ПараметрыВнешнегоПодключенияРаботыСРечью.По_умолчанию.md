ПараметрыВнешнегоПодключенияРаботыСРечью (SpeechProcessingExternalConnectionParameters)

По токену и адресу

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ПараметрыВнешнегоПодключенияРаботыСРечью(<Токен>, <АдресСервера>)

Параметры:

<Токен> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заполняется, если требуется настройка внешнего распознавания для информационной базы.

<АдресСервера> (необязательный)

Тип: [Строка](lang__def_String.md).   
Адрес внешней модели распознавания.

Использование в версии:

Доступен, начиная с версии 8.3.23.
