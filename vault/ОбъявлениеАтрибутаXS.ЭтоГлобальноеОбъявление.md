ЭтоГлобальноеОбъявление (IsGlobal)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает, имеет ли объявление глобальную область видимости.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - объявление имеет глобальную область видимости; Ложь - объявление содержится внутри определений составного типа, группы атрибутов, группы модели.

Использование в версии:

Доступен, начиная с версии 8.1.
