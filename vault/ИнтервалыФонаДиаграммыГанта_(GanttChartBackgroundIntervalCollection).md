Элементы коллекции:

[ИнтервалФонаДиаграммыГанта](ИнтервалФонаДиаграммыГанта_(GanttChartBackgroundInterval).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются интервалы.  
Возможно обращение к интервалу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция интервалов фона диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
