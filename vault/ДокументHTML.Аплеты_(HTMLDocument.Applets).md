ДокументHTML (HTMLDocument)

Аплеты (Applets)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML_(HTMLCollection).md).   
Содержит коллекцию элементов всех аплетов HTML документа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Коллекция содержит элементы типа [ЭлементАплетHTML](ЭлементАплетHTML_(HTMLAppletElement).md).

Использование в версии:

Доступен, начиная с версии 8.2.
