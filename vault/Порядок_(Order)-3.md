# Порядок

Порядок (Order)

Тип: [Строка](lang__def_String.md).   
Хранит строку с порядком, по которому можно выполнить упорядочивание выборки.  
Свойство предназначено для сортировки счетов в случаях, когда порядок следования по полю [Код](Код_(Code).md) или [Наименование](Наименование_(Description).md) не приемлем. Если длина поля [Порядок](Порядок_(Order).md) > 0, то при формировании выборок будет автоматически устанавливаться сортировка по полю [Порядок](Порядок_(Order).md). При этом основное представление по полю [Код](Код_(Code).md) или [Наименование](Наименование_(Description).md) влиять на упорядочивание счетов не будет.  
Если при конфигурировании у плана счетов установлено свойство "Автопорядок по коду", то при формировании результатов запроса будет отмечено, что сортировка выполняется по полю [Код](Код_(Code).md), однако на самом деле будет использоваться сортировка по полю [Порядок](Порядок_(Order).md).
