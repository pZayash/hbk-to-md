ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [ПерерасчетЗапись.\<Имя перерасчета\>](ПерерасчетЗапись.Имя_перерасчета_(RecalculationRecord.Recalculation_name).md).   
Запись, индекс которой требуется определить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной записи в наборе. Если не найдено, то возвращается -1.

Описание:

Получает индекс записи в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
