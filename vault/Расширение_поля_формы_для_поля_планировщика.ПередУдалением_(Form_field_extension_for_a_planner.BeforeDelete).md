# Расширение поля формы для поля планировщика.ПередУдалением

Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередУдалением (BeforeDelete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередУдалением(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) приведет к отмене удаления элементов.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при интерактивном удалении элемента. Элементы, над которыми было выполнено действие, можно получить с помощью свойства поля - [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы_(Form_field_extension_for_a_planner.SelectedItems).md) .

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
