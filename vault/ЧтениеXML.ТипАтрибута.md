ЧтениеXML (XMLReader)

ТипАтрибута (AttributeType)

Доступен, начиная с версии 8.1.

Синтаксис:

ТипАтрибута(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс атрибута.

Возвращаемое значение:

Тип: [ТипАтрибутаXML](ТипАтрибутаXML.md).   

Описание:

Получает тип атрибута с указанным индексом.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
