Описание:

Содержит объекты, описывающие оформление ячейки таблицы.  
Для оформления ячейки таблицы используются следующие элементы оформления:  
1. ЦветФона (BackColor) - тип значения [Цвет](Цвет_(Color).md);  
2. ЦветТекста (TextColor) - тип значения [Цвет](Цвет_(Color).md);  
3. ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет_(Color).md);  
4. СтильГраницы (BorderStyle) - тип значения [Линия](Линия_(Line).md);  
5. СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия_(Line).md);  
6. СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия_(Line).md);  
7. СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия_(Line).md);  
8. СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия_(Line).md);  
9. Шрифт (Font) - тип значения [Шрифт](Шрифт_(Font).md);  
10. Отступ (Indent) - тип значения [Число](lang__def_Number.md);  
11. ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md);  
12. ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md);  
13. Размещение (Placement) - размещение текста - тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных_(DataCompositionTextPlacementType).md);  
14. ОриентацияТекста (TextOrientation) - тип значения [Число](lang__def_Number.md);  
15. ВыделятьОтрицательные (MarkNegatives) - тип значения [Булево](lang__Булево_(Boolean).md);  
16. МинимальнаяШирина (MinWidth) - тип значения [Число](lang__def_Number.md);  
17. МаксимальнаяШирина (MaxWidth) - тип значения [Число](lang__def_Number.md);  
18. МинимальнаяВысота (MinHeight) - тип значения [Число](lang__def_Number.md);  
19. МаксимальнаяВысота (MaxHeight) - тип значения [Число](lang__def_Number.md);  
20. ОбъединятьПоГоризонтали (HorizontalMerge) - тип значения [Булево](lang__Булево_(Boolean).md);  
21. ОбъединятьПоВертикали (VerticalMerge) - тип значения [Булево](lang__Булево_(Boolean).md);  
22. ГоризонтальныйУровень (HorizontalLevel) - тип значения [Число](lang__def_Number.md);  
23. ВертикальныйУровень (VerticalLevel) - тип значения [Число](lang__def_Number.md);  
24. Расшифровка (Details) - тип [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных_(DataCompositionDetailsID).md);  
25. РежимИзмененияРазмераКолонки (ColumnSizeChangeMode) - возможность изменять размер колонки ячейки [РежимИзмененияРазмера](РежимИзмененияРазмера_(SizeChangeMode).md);  
26. ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную - тип значения [Булево](lang__Булево_(Boolean).md);  
27. ФиксироватьСлева (FixLeft) - признак того, что ячейку нужно фиксировать слева [Булево](lang__Булево_(Boolean).md);  
28. ФиксироватьСверху (FixTop) - признак того, что ячейку нужно фиксировать сверху [Булево](lang__Булево_(Boolean).md);  
29. РастягиватьПоГоризонтали (HorizontalStretch) - тип значения [Булево](lang__Булево_(Boolean).md). Указывает, что ширина текущей колонки рассчитывается автоматически;  
30. ВесовойКоэффициентШирины (WidthWeightFactor) - тип значения [Число](lang__def_Number.md). Весовой коэффициент для автоматического расчета ширины колонки;  
31. Картинка (Picture) - тип [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных_(DataCompositionParameter).md) или [Картинка](Картинка_(Picture).md). Если содержит параметр, то картинка получается из выражения параметра.  
32. РазмерКартинки (PictureSize) - тип [РазмерКартинки](РазмерКартинки_(PictureSize).md). Способ определения размера картинки;  
33. ПоложениеТекстаОтносительноКартинки (PictureSizeTextPositionRelativeToPicture) - тип [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки_(TextPositionRelativeToPicture).md). Положение текста относительно картинки;  
34. ГоризонтальноеПоложениеКартинки (PictureHorizontalAlign)- тип [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md). Размещение картинки по горизонтали;  
35. ВертикальноеПоложениеКартинки (PictureVerticalAlign) - тип [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md). Размещение картинки по вертикали;  
36. ОтступСлева (LeftMargin) - тип значения [Число](lang__def_Number.md);  
37. ОтступСправа (RightMargin) - тип значения [Число](lang__def_Number.md);  
38. ОтступСверху (TopMargin) - тип значения [Число](lang__def_Number.md);  
39. ОтступСнизу (BottomMargin) - тип значения [Число](lang__def_Number.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/area-template}. Имя типа XDTO: TableCellAppearance.
