Расширение поля формы для поля форматированного документа (Form field extension for a formatted document)

Вывод (Output)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеВывода](ИспользованиеВывода.md).   
Ограничение вывода на печать, сохранения, работы с буфером обмена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Может иметь значение, отличное от значения свойства [Вывод](Расширение_поля_формы_для_поля_форматированного_документа.Вывод.md) документа, с которым это поле связано.

Использование в версии:

Доступен, начиная с версии 8.2.
