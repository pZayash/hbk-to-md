Расширение формы документа (Document form extension)

ПриЗаписиПерепроводить (RepostOnWrite)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак режима записи проведенного документа. Если установлено значение [Истина](lang__def_BooleanTrue.md), проведение будет автоматически выполняться при записи проведенного документа.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
