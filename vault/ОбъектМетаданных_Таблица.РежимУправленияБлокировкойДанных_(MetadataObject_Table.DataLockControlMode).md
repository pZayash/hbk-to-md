РежимУправленияБлокировкойДанных (DataLockControlMode)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [РежимУправленияБлокировкойДанныхПоУмолчанию](РежимУправленияБлокировкойДанныхПоУмолчанию_(DefaultDataLockControlMode).md).   
Определяет режим управления блокировкой данных объекта конфигурации (например,[Автоматический](РежимУправленияБлокировкойДанныхПоУмолчанию.Автоматический_(DefaultDataLockControlMode.Automatic).md), [Управляемый](РежимУправленияБлокировкойДанныхПоУмолчанию.Управляемый_(DefaultDataLockControlMode.Managed).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
