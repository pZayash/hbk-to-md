`ПолучитьЗначениеПоля`(`<ИмяПоля>`: `Строка`) → `Число, Строка, Дата, Булево`
<!-- signature:end -->

XBase (XBase)

ПолучитьЗначениеПоля (GetFieldValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначениеПоля(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.   
Имя или индекс поля. Индекс полей начинается с 0.

Возвращаемое значение:

Тип: Число, Строка, Дата, Булево.   
Тип возвращаемого значения определяется типом поля.

Описание:

Получает значение поля по его индексу или имени поля.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
