ВложениеСистемыВзаимодействия (CollaborationSystemAttachment)

ОткрытьПотокДляЧтенияАсинх (OpenStreamForReadAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьПотокДляЧтенияАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [Поток](Поток_(Stream).md) для содержимого вложения.

Описание:

Асинхронная функция. Открывает поток для чтения содержимого вложения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
