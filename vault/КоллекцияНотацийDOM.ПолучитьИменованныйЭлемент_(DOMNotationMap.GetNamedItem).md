КоллекцияНотацийDOM (DOMNotationMap)

ПолучитьИменованныйЭлемент (GetNamedItem)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя узла DOM.

Возвращаемое значение:

Тип: [НотацияDOM](НотацияDOM_(DOMNotation).md).   

Описание:

Получить узел DOM по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
