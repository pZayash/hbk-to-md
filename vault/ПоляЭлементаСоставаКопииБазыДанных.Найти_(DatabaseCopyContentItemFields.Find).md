ПоляЭлементаСоставаКопииБазыДанных (DatabaseCopyContentItemFields)

Найти (Find)

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

Возвращаемое значение:

Тип: [ПолеЭлементаСоставаКопииБазыДанных](ПолеЭлементаСоставаКопииБазыДанных_(DatabaseCopyContentItemField).md).   

Описание:

Осуществляет поиск в элементе состава копии базы данных поля с указанным именем.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16 (в режиме совместимости с версией 8.3.14 и последующими).
