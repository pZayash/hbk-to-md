РежимСовместимости (CompatibilityMode)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.2.

Описание:

Возможности версии 8.3 используются полностью, работа конфигураций, созданных на платформе версии 8.2, не гарантируется.

Использование в версии:

Доступен, начиная с версии 8.2.
