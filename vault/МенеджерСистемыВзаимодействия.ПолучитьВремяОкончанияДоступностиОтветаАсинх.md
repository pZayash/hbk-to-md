МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьВремяОкончанияДоступностиОтветаАсинх (GetAnswerAvailabilityEndTimeAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьВремяОкончанияДоступностиОтветаАсинх(<Обсуждение>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания:

- Для интеграции типа WhatsApp - Дата в часовом поясе компьютера,
- Для других интеграций - Неопределено.

Описание:

Асинхронная функция. Для интеграции типа WhatsApp получает время окончания 24-х часового окна для ответа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
