СлойГеографическойСхемы (GeographicalSchemaLayer)

УстановитьЗначение (SetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьЗначение(<Объект>, <Серия>, <Значение>, <Подсказка>, <Расшифровка>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md).   
Объект, для которого будет устанавливаться значение серии.

<Серия> (обязательный)

Тип: [СерияДанныхСлояГеографическойСхемы](СерияДанныхСлояГеографическойСхемы.md).   
Серия, для которой устанавливается значение.

<Значение> (необязательный)

Тип: Произвольный.   
Устанавливаемое значение. Если значение не задано, текущее значение будет удалено.

<Подсказка> (необязательный)

Тип: Строка.   
Содержит текст всплывающей подсказки.

<Расшифровка> (необязательный)

Тип: Произвольный.   
Значение, используемое при расшифровке.

Описание:

Устанавливает значение для указанной серии. Дополнительно может быть установлена расшифровка значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
