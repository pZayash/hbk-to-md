Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ТипАдресаМгновенныхСообщенийДанныхКонтакта](ТипАдресаМгновенныхСообщенийДанныхКонтакта.md).

Например, тип данных [Рабочий](ТипАдресаМгновенныхСообщенийДанныхКонтакта.Рабочий.md) показывает, что в значении хранится рабочий адрес.

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
