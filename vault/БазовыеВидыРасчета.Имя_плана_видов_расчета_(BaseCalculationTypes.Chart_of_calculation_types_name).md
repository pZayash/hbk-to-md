Описание:

Предопределенная табличная часть вида расчета - список базовых видов расчета. Такая табличная часть (свойство [БазовыеВидыРасчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.БазовыеВидыРасчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.BaseCalculationTypes).md)) определена только для планов видов расчета со свойством "Зависимость от базы" не равным значению "Не зависит".  
Имеет колонки - ВидРасчета(CalculationKind) типа [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)> и Предопределенный(Predefined) типа [Булево](lang__Булево_(Boolean).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
