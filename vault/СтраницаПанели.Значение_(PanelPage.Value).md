СтраницаПанели (PanelPage)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение, которое может быть назначено странице и затем получено. Может использоваться для связи страницы с данными.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
