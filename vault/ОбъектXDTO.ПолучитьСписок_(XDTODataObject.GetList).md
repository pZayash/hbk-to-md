# ОбъектXDTO.ПолучитьСписок

ОбъектXDTO (XDTODataObject)

ПолучитьСписок (GetList)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По свойству

Синтаксис:

ПолучитьСписок(<Свойство>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md).   
Свойство, для которого необходимо получить список.

Описание варианта метода:

Если свойство не является списковым, вызывается исключение.

Вариант синтаксиса: По выражению XPath

Синтаксис:

ПолучитьСписок(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение XPath, соответствующее свойству, для которого необходимо получить список значений.

Возвращаемое значение:

Тип: [СписокXDTO](СписокXDTO_(XDTOList).md), [Неопределено](lang__def_Undefined.md).   
Если для свойства список значения не определен, будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Получает список значений для указанного свойства.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
