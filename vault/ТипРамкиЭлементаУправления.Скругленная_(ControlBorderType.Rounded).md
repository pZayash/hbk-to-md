ТипРамкиЭлементаУправления (ControlBorderType)

Скругленная (Rounded)

Доступен, начиная с версии 8.0.

Описание:

Скругленная рамка используется только для рамки группы. Может иметь толщину 1.

Использование в версии:

Доступен, начиная с версии 8.0.
