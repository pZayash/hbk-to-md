Описание:

Предназначен для запуска внешних приложений.  
В отличие от метода [ЗапуститьПриложение](Глобальный_контекст.ЗапуститьПриложение.md) позволяет передать в приложение параметры.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.

Описание изменено в версии 8.3.23.
