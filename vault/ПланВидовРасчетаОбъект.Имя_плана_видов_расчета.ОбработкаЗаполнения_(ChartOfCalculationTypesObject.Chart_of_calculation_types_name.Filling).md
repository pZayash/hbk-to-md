ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

ОбработкаЗаполнения (Filling)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаЗаполнения(<ДанныеЗаполнения>, <ТекстЗаполнения>, <СтандартнаяОбработка>)

Параметры:

<ДанныеЗаполнения>

Тип: Произвольный.   
Значение, которое используется как основание для заполнения.  
Если обработчик вызван при выполнении метода Заполнить, параметр равен параметру <ДанныеЗаполнения> метода Заполнить.  
При обработке заполнения во время ввода на основании параметр имеет значение основания.  
При интерактивном вводе нового из формы списка параметр является структурой, элементы которой соответствуют тем параметрам отбора формы списка, у которых способ сравнения [Равно](ВидСравнения.Равно_(ComparisonType.Equal).md) или [ВСписке](ВидСравнения.ВСписке_(ComparisonType.InList).md) с единственным элементом списка. Если таких элементов отбор списка не содержит, данный параметр равен [Неопределено](lang__def_Undefined.md). Если параметр <СтандартнаяОбработка> по окончании работы обработчика равен [Истина](lang__def_BooleanTrue.md) (установлен по умолчанию), система автоматически произведет заполнение одноименными элементами структуры.

<ТекстЗаполнения>

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если у объекта основное представление [ВВидеНаименования](ОсновноеПредставлениеВидаРасчета.ВВидеНаименования_(CalculationTypeMainPresentation.AsDescription).md) и реквизит [Наименование](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.Наименование_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Description).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_ПланВидовРасчета.ВводПоСтроке_(MetadataObject_ChartOfCalculationTypes.InputByString).md), то значение присваивается реквизиту [Наименование](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.Наименование_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Description).md).  
Если основное представление [ВВидеКода](ОсновноеПредставлениеВидаРасчета.ВВидеКода_(CalculationTypeMainPresentation.AsCode).md) и реквизит [Код](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.Код_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Code).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_ПланВидовРасчета.ВводПоСтроке_(MetadataObject_ChartOfCalculationTypes.InputByString).md), то значение присваивается реквизиту [Код](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.Код_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Code).md).  
В противном случае присваивание не выполняется.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при выполнении заполнения объекта из какого-либо значения. Вызывается при выполнении метода [Заполнить](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.Заполнить_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.Fill).md), при вводе на основании, а также при интерактивном вводе нового.  
В этой процедуре должно быть выполнено заполнение реквизитов объекта на основании переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При копировании (как интерактивном, так и средствами встроенного языка), данный обработчик не вызывается.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
