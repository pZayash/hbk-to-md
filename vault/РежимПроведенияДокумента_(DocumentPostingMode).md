# РежимПроведенияДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Свойства прикладных объектов](Свойства_прикладных_объектов.md)

Доступен, начиная с версии 8.0.

Значения

[Неоперативный (Regular)](РежимПроведенияДокумента.Неоперативный_(DocumentPostingMode.Regular).md)  
[Оперативный (RealTime)](РежимПроведенияДокумента.Оперативный_(DocumentPostingMode.RealTime).md)  

Описание:

Определяет набор режимов проведения документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.
