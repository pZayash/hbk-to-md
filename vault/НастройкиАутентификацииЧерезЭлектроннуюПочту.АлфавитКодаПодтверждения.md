НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

АлфавитКодаПодтверждения (VerificationCodeAlphabet)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Определяет допустимые символы для кода аутентификации через электронную почту.  
Строка длиной до 256 символов.  
Строка должна содержать минимум10 различных символов.  
Если задана пустая строка, то в алфавит входят цифры от 0 до 9.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
