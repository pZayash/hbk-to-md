Удалить (Delete)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [КолонкаДанныхДиаграммыГанта](КолонкаДанныхДиаграммыГанта_(GanttChartDataColumn).md).

Удаляет колонку из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
