ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаПерехода (NavigationProcessing)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОбработкаПерехода(<ОбъектПерехода>, <СтандартнаяОбработка>)

Параметры:

<ОбъектПерехода>

Тип: Ссылка на объект информационной базы.   
Объект, к которому требуется перейти.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки перехода на объект. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка производиться не будет.   
Стандартная обработка заключается в позиционировании на переданный объект основной таблицы формы списка (если форма является формой списка типа записываемого объекта).  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает на клиенте при вызове команды перехода к объекту или при вызове метода формы [ВыполнитьПереход](ФормаКлиентскогоПриложения.ВыполнитьПереход_(ClientApplicationForm.ExecuteNavigation).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
