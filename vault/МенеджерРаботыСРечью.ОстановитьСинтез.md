МенеджерРаботыСРечью (SpeechProcessingManager)

ОстановитьСинтез (StopTextToSpeech)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ОстановитьСинтез(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: Строка.   
Идентификатор потокового синтеза.  
Если не указан, останавливаются все сессии потокового синтеза.

Описание:

Останавливает сессию потокового синтеза речи по ключу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.
