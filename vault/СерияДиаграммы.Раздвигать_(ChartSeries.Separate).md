СерияДиаграммы (ChartSeries)

Раздвигать (Separate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак выдвижения сектора круговой диаграммы.  
[Истина](lang__def_BooleanTrue.md) - сектор выдвигается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется для круговых диаграмм в случае, если не установлено автоматическое раздвижение, т.е. свойство диаграммы [АвтоРаздвижениеСерий](Диаграмма.АвтоРаздвижениеСерий_(Chart.AutoSeriesSeparation).md) равно [Ложь](lang__def_BooleanFalse.md).

См. также:

[Диаграмма](Диаграмма_(Chart).md), свойство [АвтоРаздвижениеСерий](Диаграмма.АвтоРаздвижениеСерий_(Chart.AutoSeriesSeparation).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
