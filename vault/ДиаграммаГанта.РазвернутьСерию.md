ДиаграммаГанта (GanttChart)

РазвернутьСерию (ExpandSeries)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутьСерию(<Серия>, <СПодчиненными>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта.md).   
Раскрываемая серия.

<СПодчиненными> (обязательный)

Тип: Булево.   
Определяет необходимость раскрытия серии вместе с подчиненными сериями. Истина - раскрывать вместе с подчиненными.

Описание:

Раскрывает серию, возможно вместе с подчинёнными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
