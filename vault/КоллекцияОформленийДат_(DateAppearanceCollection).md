# КоллекцияОформленийДат

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Типы параметров событий](Типы_параметров_событий.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ОформлениеДаты](ОформлениеДаты_(DateAppearance).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается дата.  

Методы:

[Количество (Count)](КоллекцияОформленийДат.Количество_(DateAppearanceCollection.Count).md)  
[Получить (Get)](КоллекцияОформленийДат.Получить_(DateAppearanceCollection.Get).md)  

Описание:

Коллекция, содержащая оформление дат календаря. Коллекция является перечислимой. Доступ к элементам осуществляется по значению даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[ОформлениеПериода](ОформлениеПериода_(PeriodAppearance).md), свойство [Даты](ОформлениеПериода.Даты_(PeriodAppearance.Dates).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
