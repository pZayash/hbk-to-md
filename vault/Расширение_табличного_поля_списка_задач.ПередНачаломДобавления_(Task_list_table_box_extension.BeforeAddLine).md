# Расширение табличного поля списка задач.ПередНачаломДобавления

Расширение табличного поля списка задач (Task list table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то строка списка задач добавлена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

Описание:

Возникает перед добавлением строки списка задач. Данное событие для табличного поля списка задач возникает "вместо" одноименного события объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

Доступность:

Толстый клиент.

Примечание:

Расширяется набор параметров по сравнению с одноименным событием объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

См. также:

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [ПередНачаломДобавления](ТабличноеПоле.ПередНачаломДобавления_(TableBox.BeforeAddRow).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
