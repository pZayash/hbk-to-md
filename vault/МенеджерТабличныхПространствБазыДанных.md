Элементы коллекции:

[МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных.md).  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя пространства.

Описание:

Предназначен для работы с табличными пространствами.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ИмяПространства = "Пространство с данными"; ПутьКПространству = "C:\Data\";  Таблица = МенеджерТабличныхПространствБазыДанных.                         Найти(ИмяПространства); Если Таблица = Неопределено Тогда     Таблица = МенеджерТабличныхПространствБазыДанных.     Добавить(ИмяПространства, ПутьКПространству); КонецЕсли;  Элемент = Новый ЭлементСоставаТабличногоПространства; // указываем выбранный заранее объект, например, справочник Товары  Элемент.Метаданные = Метаданные.Справочники.Товары;  // добавим данные в пространство  СодержимоеДанных = Таблица.ИзмененныйСоставДанных; ИзмененныйЭлемент = СодержимоеДанных.Найти(Элемент); Если ИзмененныйЭлемент = Неопределено Тогда     СодержимоеДанных.Добавить(Элемент); КонецЕсли;  // удалим индекс из пространства  СоставИндексов = Таблица. МенеджерТабличногоПространстваБазыДанных; ЭлементИндекса = СоставИндексов.Найти(Элемент); Если ЭлементИндекса <> Неопределено Тогда     ЭлементИндекса.Удалить(Элемент); КонецЕсли;  Таблица.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.3.23.
