АдминистрированиеСервис (AdministrationService)

ПоддерживаетсяТолькоГлавнымМенеджером (SupportedOnlyByMainManager)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак поддержки главным и дополнительным менеджером кластера:

- [Истина](lang__def_BooleanTrue.md) - если сервис поддерживается только главным менеджером кластера и не может быть перенесен в процесс дополнительного менеджера кластера.
- [Ложь](lang__def_BooleanFalse.md) - если сервис может поддерживаться как главным, так и дополнительным менеджером кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
