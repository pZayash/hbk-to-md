ОписаниеПодписиPDF (PDFSignatureDescription)

Ширина (Width)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит ширину подписи (в миллиметрах) на странице PDF-документа.   
Если при подписании не указана, то ширина будет определена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
