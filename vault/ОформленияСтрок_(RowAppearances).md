Доступен, начиная с версии 8.0.

Элементы коллекции:

[ОформлениеСтроки](ОформлениеСтроки_(RowAppearance).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются оформления строк.  
Возможно обращение к оформлению строки посредством оператора [...]. В качестве аргумента передается индекс строки (нумерация с 0).  

Методы:

[Количество (Count)](ОформленияСтрок.Количество_(RowAppearances.Count).md)  
[Получить (Get)](ОформленияСтрок.Получить_(RowAppearances.Get).md)  

Описание:

Представляет собой коллекцию значений типа [ОформлениеСтроки](ОформлениеСтроки_(RowAppearance).md).

Доступность:

Толстый клиент.
