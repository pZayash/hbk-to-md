АдминистрированиеХранилищеДвоичныхДанных (AdministrationBinaryDataStorage)

СоздатьДифференциальнуюРезервнуюКопию (CreateDifferentialBackup)

Доступен, начиная с версии 8.3.27.

Синтаксис:

СоздатьДифференциальнуюРезервнуюКопию(<Путь>, <ПутьКПолнойРезервнойКопии>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу резервной копии на сервере.

<ПутьКПолнойРезервнойКопии> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу полной резервной копии на сервере.

Описание:

Создает дифференциальную резервную копию хранилища двоичных данных. Только для встроенного хранилища.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
