БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: Булево.   
Истина - объект изменен; Ложь - в противном случае.

Описание:

Определяет, был ли изменен объект после считывания из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, был ли изменен объект другими пользователями.

Пример:

|  |
| --- |
| Если БизнесПроцесс.Модифицированность() Тогда     Предупреждение("Бизнес-процесс был изменен."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
