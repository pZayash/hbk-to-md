Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

АвтоКонтекстноеМеню (AutoContextMenu)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Установка данного свойства разрешает использование для элемента управления автоматического контекстного меню, иначе будет использовано контекстное меню, выбранное в свойстве [КонтекстноеМеню](Расширение_элементов_управления,_расположенных_в_форме.КонтекстноеМеню.md).

Доступность:

Толстый клиент.

Примечание:

Свойство используется только для следующих типов элементов управления: [ПолеВвода](ПолеВвода.md), [ПолеВыбора](ПолеВыбора.md), [ПолеСписка](ПолеСписка.md), [ТабличноеПоле](ТабличноеПоле.md), [ПолеТекстовогоДокумента](ПолеТекстовогоДокумента.md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента.md).  
  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

См. также:

[Расширение элементов управления, расположенных в форме](Интерфейс.Расширение_элементов_управления,_расположенных_в_форме.md), свойство [КонтекстноеМеню](Расширение_элементов_управления,_расположенных_в_форме.КонтекстноеМеню.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
