# Расширение поля формы для поля календаря.Обновить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля календаря](Расширение_поля_формы_для_поля_календаря_(Form_field_extension_for_a_calendar_field).md) › [Методы](Расширение_поля_формы_для_поля_календаря__Методы.md)

Расширение поля формы для поля календаря (Form field extension for a calendar field)

Обновить (Refresh)

Доступен, начиная с версии 8.2.

Синтаксис:

Обновить()

Описание:

Инициирует событие [ПриВыводеПериода](Расширение_поля_формы_для_поля_календаря.ПриВыводеПериода_(Form_field_extension_for_a_calendar_field.OnPeriodOutput).md) для всех видимых в календаре месяцев.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
