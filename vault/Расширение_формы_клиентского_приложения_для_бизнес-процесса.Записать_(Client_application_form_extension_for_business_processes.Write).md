Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Содержит предопределенный параметр Старт. Если параметр Старт=[Истина](lang__def_BooleanTrue.md), то бизнес-процесс будет стартован.  
Состав других параметров произвольный.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - успешно записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
