СвойствоXDTO (XDTOProperty)

ВерхняяГраница (UpperBound)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Верхняя граница появления значений данного свойства у объекта.  
Свойства с верхней границей большей 1 представляются у объекта списками элементов XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
