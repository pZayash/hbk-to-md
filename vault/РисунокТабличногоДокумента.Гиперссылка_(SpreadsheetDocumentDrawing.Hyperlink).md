РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - рисунок является гиперссылкой. Все команды обработки для рисунка, являющегося гиперссылкой, вызываются по одинарному щелчку мыши на рисунке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не применимо для рисунков типа [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
