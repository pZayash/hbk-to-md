КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md).   
Индекс сдвигаемого элемента в коллекции или сам элемент.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое необходимо сдвинуть элемент. Положительное значение обозначает смещение к концу, отрицательное - к началу коллекции.

Описание:

Сдвигает элемент с заданным индексом на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
