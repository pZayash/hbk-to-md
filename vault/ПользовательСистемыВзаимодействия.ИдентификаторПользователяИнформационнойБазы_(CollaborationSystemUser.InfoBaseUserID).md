ПользовательСистемыВзаимодействия (CollaborationSystemUser)

ИдентификаторПользователяИнформационнойБазы (InfoBaseUserID)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [Неопределено](lang__def_Undefined.md).   
Идентификатор пользователя информационной базы, соответствующий пользователю системы взаимодействия.  
Изменение свойства возможно только в привилегированном режиме или при наличии у текущего пользователя права "Администрирование данных".

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.23.
