Доступен, начиная с версии 8.1.

Элементы коллекции:

[ОписаниеМакетаОбластиМакетаКомпоновкиДанных](ОписаниеМакетаОбластиМакетаКомпоновкиДанных_(DataCompositionTemplateAreaTemplateDefinition).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются описания макетов.  
Возможно обращение к описанию макета посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.

Описание:

Коллекция описаний макетов областей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
