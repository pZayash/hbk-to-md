СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

ФорматДанныхСтроки (StringDataFormat)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если свойство [Данные](СообщениеСистемыВзаимодействия.Данные_(CollaborationSystemMessage.Data).md) содержит данные типа [Строка](lang__def_String.md), то содержит MIME-тип.данных. Иначе – [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
