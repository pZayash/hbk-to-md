Соединение с агентом сервера (IServerAgentConnection)

RegSecurityProfileUnSafeExternalModule (RegSecurityProfileUnSafeExternalModule)

Доступен, начиная с версии 8.3.3.

Синтаксис:

RegSecurityProfileUnSafeExternalModule(<Кластер>, <ИмяПрофиля>, <ВнешнийМодуль>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должен быть зарегистрирован внешний модуль.

<ИмяПрофиля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности.

<ВнешнийМодуль> (обязательный)

Тип: [Внешний модуль](Внешний_модуль_(ISecurityProfileExternalModule).md).   
Регистрируемый внешний модуль.

Описание:

Выполняет регистрацию нового внешнего модуля или обновление существующего. Внешний модуль идентифицируется своей контрольной суммой.

Доступность:

Интеграция.

Примечание:

Если внешний модуль существует, то выполняется обновление, в противном случае - добавление.  
Если профиль не найден, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
