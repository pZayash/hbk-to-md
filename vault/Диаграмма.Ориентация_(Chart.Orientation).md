Диаграмма (Chart)

Ориентация (Orientation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОриентацияДиаграммы](ОриентацияДиаграммы_(ChartOrientation).md).   
Определяет ориентацию осей координатной системы для изометрических диаграмм.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для изометрических диаграмм.

Использование в версии:

Доступен, начиная с версии 8.0.
