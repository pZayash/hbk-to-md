ПоляВыбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionSelectionFields)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [ПолеВыбораКомпоновкиДанныхСхемыЗапроса](ПолеВыбораКомпоновкиДанныхСхемыЗапроса_(QuerySchemaDataCompositionSelectionField).md), [Число](lang__def_Number.md).   
Индекс элемента или сам элемент, который требуется переместить.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое нужно сдвинуть указанный элемент. Положительное смещение обозначает сдвиг в конец, отрицательное - в начало коллекции.

Описание:

Сдвигает элемент в коллекции на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
