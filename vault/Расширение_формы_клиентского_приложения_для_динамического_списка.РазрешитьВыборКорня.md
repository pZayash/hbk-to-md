Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

РазрешитьВыборКорня (AllowRootChoice)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.   
Определяет возможность выбора корня в форме с динамическим списком, отображаемом в виде дерева.  
При открытии формы списка со значением параметра Истина и параметром [РежимВыбора](Расширение_формы_клиентского_приложения_для_динамического_списка.РежимВыбора.md)=Истина для всех таблиц, связанных с основным реквизитом типа [ДинамическийСписок](ДинамическийСписок.md), автоматически будут переопределены значения некоторых свойств:

- свойство [ОтображатьКорень](Расширение_таблицы_формы_для_динамического_списка.ОтображатьКорень.md) будет установлено в Истина;
- свойство [РазрешитьВыборКорня](Расширение_таблицы_формы_для_динамического_списка.РазрешитьВыборКорня.md) будет установлено в Истина;
- свойство [Отображение](ТаблицаФормы.Отображение.md) будет установлено в [Дерево](ОтображениеТаблицы.Дерево.md), если оно было не [Дерево](ОтображениеТаблицы.Дерево.md).

Команда "Переместить в группу" при открытии формы передает параметр со значением Истина.  
В конструкторах форм списков для всех списков, связанных с основным реквизитом, свойству [НачальноеОтображениеДерева](ТаблицаФормы.НачальноеОтображениеДерева.md) устанавливается значение [РаскрыватьВерхнийУровень](НачальноеОтображениеДерева.РаскрыватьВерхнийУровень.md).

Использование в версии:

Доступен, начиная с версии 8.2.
