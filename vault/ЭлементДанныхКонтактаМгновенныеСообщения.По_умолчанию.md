ЭлементДанныхКонтактаМгновенныеСообщения (ContactDataItemInstantMessaging)

По умолчанию

Доступен, начиная с версии 8.3.4.

Синтаксис:

Новый ЭлементДанныхКонтактаМгновенныеСообщения(<ТипДанных>, <ТипМгновенныхСообщений>, <Идентификатор>)

Параметры:

<ТипДанных> (обязательный)

Тип: [Строка](lang__def_String.md), [ТипАдресаМгновенныхСообщенийДанныхКонтакта](ТипАдресаМгновенныхСообщенийДанныхКонтакта_(ContactDataInstantMessagingAddressType).md).   
Указывает, как интерпретировать значение.  
Например, тип данных [Рабочий](ТипАдресаМгновенныхСообщенийДанныхКонтакта.Рабочий_(ContactDataInstantMessagingAddressType.Work).md) показывает, что в значении хранится рабочий адрес.

<ТипМгновенныхСообщений> (обязательный)

Тип: [Строка](lang__def_String.md).   
Тип мгновенных сообщений (например: "Skype").

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор учетной записи мгновенных сообщений.

Описание:

Создает элемент данных контакта мгновенных сообщений.

Использование в версии:

Доступен, начиная с версии 8.3.4.
