Глобальный контекст (Global context)

СтрСравнить (StrCompare)

Доступен, начиная с версии 8.3.5.

Синтаксис:

СтрСравнить(<Строка1>, <Строка2>)

Параметры:

<Строка1> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка для сравнения

<Строка2> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка для сравнения.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Возвращает:

- -1 - первая строка меньше второй.
- 1 - первая строка больше второй.
- 0 - первая строка равна второй.

Описание:

Выполняет сравнение строк без учета регистра, аналогичное сравнению строк с помощью объекта [СравнениеЗначений](СравнениеЗначений_(CompareValues).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СравнениеЗначений](СравнениеЗначений_(CompareValues).md), метод [Сравнить](СравнениеЗначений.Сравнить_(CompareValues.Compare).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
