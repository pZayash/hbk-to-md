КонфигурацияПостроителяDOM (DOMBuilderConfiguration)

ПолучитьПараметр (GetParameter)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьПараметр(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя параметра.

Возвращаемое значение:

Тип: Произвольный.   

Описание:

Получает значение параметра.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- указанный параметр не определен в данной конфигурации.

Использование в версии:

Доступен, начиная с версии 8.1.
