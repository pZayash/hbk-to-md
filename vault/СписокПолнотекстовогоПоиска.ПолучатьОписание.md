СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолучатьОписание (GetDescription)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Признак получения описания для результатов поиска. Если установлено Истина, то для каждого из результатов поиска будет заполнено значение [Описание](ЭлементСпискаПолнотекстовогоПоиска.Описание.md), помогающее понять контекст найденных слов. Установка [ПолучатьОписание](СписокПолнотекстовогоПоиска.ПолучатьОписание.md) в Ложь ускоряет поиск.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
