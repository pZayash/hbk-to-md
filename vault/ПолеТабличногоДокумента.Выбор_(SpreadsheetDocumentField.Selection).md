ПолеТабличногоДокумента (SpreadsheetDocumentField)

Выбор (Selection)

Доступен, начиная с версии 8.0.

Синтаксис:

Выбор(<Область>, <СтандартнаяОбработка>)

Параметры:

<Область>

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md).   
Область, с которой связано событие.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке мыши (нажатии клавиши Enter) на ячейке или рисунке таблицы.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
