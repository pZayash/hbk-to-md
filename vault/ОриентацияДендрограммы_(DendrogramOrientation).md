Доступен, начиная с версии 8.0.

Значения

[Верх (Top)](ОриентацияДендрограммы.Верх_(DendrogramOrientation.Top).md)  
[Лево (Left)](ОриентацияДендрограммы.Лево_(DendrogramOrientation.Left).md)  
[Низ (Bottom)](ОриентацияДендрограммы.Низ_(DendrogramOrientation.Bottom).md)  
[Право (Right)](ОриентацияДендрограммы.Право_(DendrogramOrientation.Right).md)  

Описание:

Содержит варианты ориентации связей дендрограммы. Метки будут располагаться противоположно ориентации связей. Например: если для связей задана ориентация [Верх](ОриентацияДендрограммы.Верх_(DendrogramOrientation.Top).md), метки будут расположены снизу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: DendrogramOrientation.

См. также:

[Дендрограмма](Дендрограмма_(Dendrogram).md), свойство [Ориентация](Дендрограмма.Ориентация_(Dendrogram.Orientation).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
