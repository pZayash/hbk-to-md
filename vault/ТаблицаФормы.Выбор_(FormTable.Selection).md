ТаблицаФормы (FormTable)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<ВыбраннаяСтрока>, <Поле>, <СтандартнаяОбработка>)

Параметры:

<ВыбраннаяСтрока>

Значение выбранной строки. Тип значения зависит от типа данных, отображаемых в таблице. Конкретные типы значения выбранной строки указаны в описаниях расширений таблицы.  
Массив значений выделенных строк, если [МножественныйВыбор](ТаблицаФормы.МножественныйВыбор_(FormTable.MultipleChoice).md) = [Истина](lang__def_BooleanTrue.md).

<Поле>

Тип: [ПолеФормы](ПолеФормы_(FormField).md).   
Активное поле (колонка).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке мыши (нажатии клавиши Enter) в ячейке таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
