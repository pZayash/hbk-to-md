ДиаграммаГанта (GanttChart)

КратностьПериодическогоВарианта (PeriodicVariantRepetition)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Величина просматриваемого интервала в единицах, указанных в свойстве [ЕдиницаПериодическогоВарианта](ДиаграммаГанта.ЕдиницаПериодическогоВарианта.md). При изменении размеров формы значение интервала не меняется.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Используется, если свойство ПоддержкаМасштаба имеет значение Период.

Использование в версии:

Доступен, начиная с версии 8.0.
