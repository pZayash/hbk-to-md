ТабличныйДокумент (SpreadsheetDocument)

НайтиЗначение (FindValue)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиЗначение(<Значение>, <Начало>, <Где>, <ИскатьПоСтрокам>, <ИскатьВперед>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Искомое значение.

<Начало> (необязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Область, после которой начинать поиск. Если параметр не задан, то поиск с начала табличного документа.

<Где> (необязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Область, в которой осуществляется поиск. Если параметр не указан, поиск будет выполнен по всем ячейкам табличного документа.

<ИскатьПоСтрокам> (необязательный)

Тип: Булево.   
Определяет последовательность обхода ячеек при поиске.  
Истина - искать по текущей строке, затем переходить на следующую и т.д. в общем направлении поиска, Ложь - искать по текущей колонке, затем переходить на следующую и т.д. в общем направлении поиска.  
Значение по умолчанию: Истина.

<ИскатьВперед> (необязательный)

Тип: Булево.   
Определяет направление поиска: Истина - искать вперед; Ложь - искать назад.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md), [РисунокТабличногоДокумента](РисунокТабличногоДокумента.md), Неопределено.   
Если значение не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск значения в ячейках табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Область = ТабДок.НайтиЗначение(ВыбЗначение); Если ТипЗнч(Область)=Тип("Неопределено") Тогда     Сообщить("Значение не найдено!"); Иначе     Сообщить("Значение найдено."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.1.
