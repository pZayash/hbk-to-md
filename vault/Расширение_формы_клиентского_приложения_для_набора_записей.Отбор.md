Расширение формы клиентского приложения для набора записей (Managed form extension for record set)

Отбор (Filter)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Структура](Структура.md).   
Используется для установки отбора в наборе записей. Ключ структуры - имя поля набора.

Использование в версии:

Доступен, начиная с версии 8.2.
