СредстваГеопозиционирования (LocationTools)

ПолучитьСамогоЭнергоЭкономичногоПровайдера (GetMostPowerSavingProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьСамогоЭнергоЭкономичногоПровайдера(<ТолькоДоступного>)

Параметры:

<ТолькоДоступного> (необязательный)

Тип: Булево.   
Если передано значение Истина, информация о самом экономичном провайдере будет возвращена только если провайдер доступен. Если самый энергоэкономичный провайдер недоступен, будет возвращено Неопределено.  
Если передано значение Ложь, будет возвращена информация о самом экономичном провайдере на устройстве.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования.md), Неопределено.   
Неопределено - ни один провайдер не найден, или если параметр <ТолькоДоступного> содержал значение Истина, а самый экономичный провайдер не доступен.

Описание:

Получает информацию о самом энергоэкономичном провайдере геопозиционирования.  
Если параметр <ТолькоДоступного> содержит значение Истина, информация о самом экономичном провайдере будет возвращена только в случае, если провайдер доступен.  
Если параметр <ТолькоДоступного> содержит значение Ложь, информация о самом экономичном провайдере будет возвращена независимо от доступности провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
