МенеджерОбработкиОшибок (ErrorProcessingManager)

КатегорияОшибкиДляПользователя (ErrorCategoryForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

КатегорияОшибкиДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md).   
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: [КатегорияОшибки](КатегорияОшибки_(ErrorCategory).md).   
Возвращает одно из следующих значений:

- [ИсключениеВызванноеИзВстроенногоЯзыка](КатегорияОшибки.ИсключениеВызванноеИзВстроенногоЯзыка_(ErrorCategory.ExceptionRaisedFromScript).md),
- [НарушениеПравДоступа](КатегорияОшибки.НарушениеПравДоступа_(ErrorCategory.AccessViolation).md),
- [НеправильныйПароль](КатегорияОшибки.НеправильныйПароль_(ErrorCategory.InvalidPassword).md),
- [ОтсутствиеРазрешенияДляИспользованияФункциональности](КатегорияОшибки.ОтсутствиеРазрешенияДляИспользованияФункциональности_(ErrorCategory.NoPermissionToUseFunctionality).md),
- [ОшибкаВнешнегоИсточникаДанных](КатегорияОшибки.ОшибкаВнешнегоИсточникаДанных_(ErrorCategory.ExternalDataSourceError).md),
- [ОшибкаДоступаКЛокальномуФайлу](КатегорияОшибки.ОшибкаДоступаКЛокальномуФайлу_(ErrorCategory.LocalFileAccessError).md),
- [ОшибкаКонфигурации](КатегорияОшибки.ОшибкаКонфигурации_(ErrorCategory.ConfigurationError).md),
- [ОшибкаКопииБазыДанных](КатегорияОшибки.ОшибкаКопииБазыДанных_(ErrorCategory.DatabaseCopyError).md),
- [ОшибкаНастроекКомпоновкиДанных](КатегорияОшибки.ОшибкаНастроекКомпоновкиДанных_(ErrorCategory.DataCompositionSettingsError).md),
- [ОшибкаПереходаПоНавигационнойСсылке](КатегорияОшибки.ОшибкаПереходаПоНавигационнойСсылке_(ErrorCategory.GotoURLError).md),
- [ОшибкаПолнотекстовогоПоиска](КатегорияОшибки.ОшибкаПолнотекстовогоПоиска_(ErrorCategory.FullTextSearchError).md),
- [ОшибкаПреобразованияДокумента](КатегорияОшибки.ОшибкаПреобразованияДокумента_(ErrorCategory.DocumentConversionError).md),
- [ОшибкаПроверкиПодписи](КатегорияОшибки.ОшибкаПроверкиПодписи_(ErrorCategory.SignatureVerificationError).md),
- [ОшибкаРаботыСПринтером](КатегорияОшибки.ОшибкаРаботыСПринтером_(ErrorCategory.PrinterError).md),
- [ОшибкаРаботыСРечью](КатегорияОшибки.ОшибкаРаботыСРечью_(ErrorCategory.SpeechProcessingError).md),
- [ОшибкаСеанса](КатегорияОшибки.ОшибкаСеанса_(ErrorCategory.SessionError).md),
- [ОшибкаСети](КатегорияОшибки.ОшибкаСети_(ErrorCategory.NetworkError).md),
- [ОшибкаСистемыВзаимодействия](КатегорияОшибки.ОшибкаСистемыВзаимодействия_(ErrorCategory.CollaborationSystemError).md),
- [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа_(ErrorCategory.MultimediaToolsError).md),
- [ОшибкаТабличногоПространстваБазыДанных](КатегорияОшибки.ОшибкаТабличногоПространстваБазыДанных_(ErrorCategory.DatabaseTablespaceError).md),
- [ОшибкаХранимыхДанных](КатегорияОшибки.ОшибкаХранимыхДанных_(ErrorCategory.StoredDataError).md),
- [ПринудительноеЗавершениеРаботы](КатегорияОшибки.ПринудительноеЗавершениеРаботы_(ErrorCategory.ForcedShutdown).md),
- [ПрочаяОшибка](КатегорияОшибки.ПрочаяОшибка_(ErrorCategory.OtherError).md).

Не возвращает:

- [ВсеОшибки](КатегорияОшибки.ВсеОшибки_(ErrorCategory.AllErrors).md),
- [ОшибкаВоВремяВыполненияВстроенногоЯзыка](КатегорияОшибки.ОшибкаВоВремяВыполненияВстроенногоЯзыка_(ErrorCategory.ScriptRuntimeError).md),
- [ОшибкаИспользованияВстроенногоЯзыка](КатегорияОшибки.ОшибкаИспользованияВстроенногоЯзыка_(ErrorCategory.ScriptUseError).md),
- [ОшибкаКомпиляцииВстроенногоЯзыка](КатегорияОшибки.ОшибкаКомпиляцииВстроенногоЯзыка_(ErrorCategory.ScriptCompileError).md).

Описание:

Возвращает категорию для пользователя. По этой категории будет формироваться сообщение в методе [СообщениеОбОшибкеДляПользователя](МенеджерОбработкиОшибок.СообщениеОбОшибкеДляПользователя_(ErrorProcessingManager.ErrorMessageForUser).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.26.
