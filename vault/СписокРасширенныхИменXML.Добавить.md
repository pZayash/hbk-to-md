Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [РасширенноеИмяXML](РасширенноеИмяXML.md).

Добавляет расширенное имя XML в конец коллекции.

Доступен, начиная с версии 8.1.
