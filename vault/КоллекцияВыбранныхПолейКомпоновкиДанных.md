# КоллекцияВыбранныхПолейКомпоновкиДанных

Элементы коллекции:

[ВыбранноеПолеКомпоновкиДанных](ВыбранноеПолеКомпоновкиДанных.md), [ГруппаВыбранныхПолейКомпоновкиДанных](ГруппаВыбранныхПолейКомпоновкиДанных.md), [АвтоВыбранноеПолеКомпоновкиДанных](АвтоВыбранноеПолеКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выбранные поля.  
Возможно обращение к выбранным полям посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция выбранных полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
