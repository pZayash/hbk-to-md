Расширение группы формы для группы кнопок (Form group extension for a button group)

Отображение (Representation)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеГруппыКнопок](ОтображениеГруппыКнопок.md).   
Задает отображение группы кнопок командной панели.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
