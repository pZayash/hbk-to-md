Сеанс (ISessionInfo)

durationLast5MinService (durationLast5MinService)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит время, которое затрачено сеансом на вызовы сервисов кластера за последние 5 минут, в миллисекундах.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.12.
