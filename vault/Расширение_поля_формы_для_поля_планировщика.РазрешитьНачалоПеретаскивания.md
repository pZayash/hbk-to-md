Расширение поля формы для поля планировщика (Form field extension for a planner)

РазрешитьНачалоПеретаскивания (EnableStartDrag)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешает начало перетаскивания из поля планировщика - источника данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Истина - перетаскивание разрешено; Ложь - запрещено.

Использование в версии:

Доступен, начиная с версии 8.3.6.
