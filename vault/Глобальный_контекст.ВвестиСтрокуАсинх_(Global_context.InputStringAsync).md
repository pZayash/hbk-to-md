# Глобальный контекст.ВвестиСтрокуАсинх

Глобальный контекст (Global context)

ВвестиСтрокуАсинх (InputStringAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВвестиСтрокуАсинх(<Строка>, <Подсказка>, <Длина>, <Многострочность>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Доступная в модуле переменная. Начальное значение переменной будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода строки. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<Длина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Длина вводимой строки. Если параметр не указан, то строка неограниченной длины.  
Значение по умолчанию: 0.

<Многострочность> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим ввода многострочного текста: [Истина](lang__def_BooleanTrue.md) - ввод многострочного текста с разделителями строк; [Ложь](lang__def_BooleanFalse.md) - ввод простой строки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - введенное значение строки или [Неопределено](lang__def_Undefined.md), если пользователь отказался от ввода,

Описание:

Асинхронная функция. Вызывает диалог для ввода строки, при этом не ожидает завершения ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ВвестиСтроку](Глобальный_контекст.ВвестиСтроку_(Global_context.InputString).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
