ТекстовыйДокумент (TextDocument)

УстановитьТипФайла (SetFileType)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТипФайла(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Устанавливаемый тип файла.  
  
Тип файла может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые названия кодировок:

- UTF-8
- UTF-16
- UTF-16BE
- UTF-16LE
- ISO-8859-1
- US-ASCII
- gb18030
- iso-8859-2
- iso-8859-3
- iso-8859-4
- iso-8859-5
- iso-8859-6
- iso-8859-7
- iso-8859-8
- iso-8859-9
- iso-8859-13
- iso-8859-15
- Shift\_JIS
- EUC-JP
- Big5
- Big5-HKSCS
- GBK
- GB2312
- GB\_2312-80
- EUC-KR
- KSC\_5601
- windows-949
- windows-874
- cp866
- KOI8-R
- KOI8-U
- windows-1250
- windows-1251
- windows-1252
- windows-1253
- windows-1254
- windows-1255
- windows-1256
- windows-1257
- windows-1258
- macintosh
- x-mac-cyrillic
- ISO-2022-JP
- ISO-2022-KR
- ISO-2022-CN
- ISO-2022-CN-EXT
- HZ-GB-2312

  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8.md).

Описание:

Устанавливает кодировку файла, в которой будет сохранен текстовый документ при интерактивной записи (например, при выборе пункта "Файл - Печать").

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
