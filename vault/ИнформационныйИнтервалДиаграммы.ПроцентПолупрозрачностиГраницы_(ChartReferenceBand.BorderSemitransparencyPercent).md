ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

ПроцентПолупрозрачностиГраницы (BorderSemitransparencyPercent)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит степень прозрачности линии границы, указанную в процентах. Целочисленное значение от 0 до 100. 100% - полностью прозрачный.  
Если указано [Неопределено](lang__def_Undefined.md), прозрачность расчитывается автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
