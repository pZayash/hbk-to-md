ХешированиеДанных (DataHashing)

ХешСумма (HashSum)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: Число, [ДвоичныеДанные](ДвоичныеДанные.md).   
Содержит текущее значение хеш-суммы. Для функций [MD5](ХешФункция.MD5.md), [SHA1](ХешФункция.SHA1.md), [SHA256](ХешФункция.SHA256.md), [SHA512](ХешФункция.SHA512.md), и [Неактивно](СостояниеОбновленияКопииБазыДанных.Неактивно.md) – значение типа [ДвоичныеДанные](ДвоичныеДанные.md), для функции [CRC32](ХешФункция.CRC32.md) – Число.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Описание изменено в версии 8.3.26.
