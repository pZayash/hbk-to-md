Глобальный контекст (Global context)

ПрочитатьДатуJSON (ReadJSONDate)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПрочитатьДатуJSON(<Строка>, <Формат>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, которую требуется преобразовать в дату.

<Формат> (обязательный)

Тип: [ФорматДатыJSON](ФорматДатыJSON_(JSONDateFormat).md).   
Формат, в котором представлена дата в строке, подлежащей преобразованию.

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   

Описание:

Выполняет преобразование строки, прочитанной в JSON-формате, в значение типа [Дата](lang__def_Date.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если входящая дата записана в форме без указания часового пояса (например, 2009-02-15T10:00:00), она будет интерпретирована как дата в поясе локального компьютера.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.23.
