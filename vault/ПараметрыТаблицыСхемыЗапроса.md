Элементы коллекции:

[ПараметрТаблицыСхемыЗапроса](ПараметрТаблицыСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры таблицы.  
Возможно обращение к параметру таблицы посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПараметрыТаблицыСхемыЗапроса.Индекс.md)  
[Количество (Count)](ПараметрыТаблицыСхемыЗапроса.Количество.md)  
[Найти (Find)](ПараметрыТаблицыСхемыЗапроса.Найти.md)  
[Получить (Get)](ПараметрыТаблицыСхемыЗапроса.Получить.md)  

Описание:

Содержит коллекцию параметров таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
