Глобальный контекст (Global context)

ИсторияРаботыПользователя (UserWorkHistory)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [МенеджерИсторииРаботыПользователя](МенеджерИсторииРаботыПользователя_(UserWorkHistoryManager).md).   
Используется для управления историей работы пользователя.  
Для использования свойства требуется, чтобы для пользователя было доступно право "СохранениеДанныхПользователя".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
