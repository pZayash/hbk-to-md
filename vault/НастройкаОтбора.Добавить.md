НастройкаОтбора (FilterSettings)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента отбора.

<Доступность> (обязательный)

Тип: Булево.   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ЭлементУправленияОтбором](ЭлементУправленияОтбором.md).   

Описание:

Добавляет элемент управления отбором в коллекцию.  
Внимание! Метод не доступен для настройки отбора динамических списков (свойство НастройкаОтбора объектов [СправочникСписок.](СправочникСписок.Имя_справочника.md)<[Имя справочника](СправочникСписок.Имя_справочника.md)>, [ДокументСписок.](ДокументСписок.Имя_документа.md)<[Имя документа](ДокументСписок.Имя_документа.md)> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
