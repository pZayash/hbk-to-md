Глобальный контекст (Global context)

ПолучитьОтключениеБезопасногоРежима (GetSafeModeDisabled)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПолучитьОтключениеБезопасногоРежима()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - выключение безопасного режима для текущего метода; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет выключение безопасного режима для текущего метода командой на время выполнения обработчиков событий объектов метаданных или вызовом метода [УстановитьОтключениеБезопасногоРежима](Глобальный_контекст.УстановитьОтключениеБезопасногоРежима_(Global_context.SetSafeModeDisabled).md) с параметром [Истина](lang__def_BooleanTrue.md) непосредственно в коде метода.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
