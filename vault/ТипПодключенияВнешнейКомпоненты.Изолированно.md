ТипПодключенияВнешнейКомпоненты (AddInConnectionType)

Изолированно (Isolated)

Доступен, начиная с версии 8.3.21.

Описание:

Подключение выполняется к хост-процессу информационной базы. Если этот режим не поддерживается компонентой, то генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
