# НастройкаПараметровАнализаДанных.Найти

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md) › [НастройкаПараметровАнализаДанных](НастройкаПараметровАнализаДанных_(DataAnalysisParametersSetting).md) › [Методы](НастройкаПараметровАнализаДанных__Методы.md)

НастройкаПараметровАнализаДанных (DataAnalysisParametersSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента параметров.

Возвращаемое значение:

Тип: [ЭлементУправленияПараметрамиАнализаДанных](ЭлементУправленияПараметрамиАнализаДанных_(DataAnalysisParametersControl).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента управления параметров по имени элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
