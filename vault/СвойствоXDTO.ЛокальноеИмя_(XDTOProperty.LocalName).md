СвойствоXDTO (XDTOProperty)

ЛокальноеИмя (LocalName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Локальное имя представления данного свойства в XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для свойства с формой представления "Текст" локальное имя неопределено.

Использование в версии:

Доступен, начиная с версии 8.1.
