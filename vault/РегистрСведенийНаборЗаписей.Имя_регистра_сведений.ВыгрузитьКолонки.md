РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

ВыгрузитьКолонки (UnloadColumns)

Доступен, начиная с версии 8.1.

Синтаксис:

ВыгрузитьКолонки(<Колонки>)

Параметры:

<Колонки> (необязательный)

Тип: Строка.   
Список колонок для выгрузки в формате: "Колонка1, Колонка2...". Если список не задан, то будут выгружены все колонки.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   

Описание:

Создает таблицу значений с заданным списком колонок набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выгружается только структура набора записей, данные не выгружаются.

Использование в версии:

Доступен, начиная с версии 8.1.
