ФорматированныйДокумент (FormattedDocument)

Добавить (Add)

Доступен, начиная с версии 8.2.

Синтаксис:

Добавить(<Значение>, <ТипЭлемента>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Добавляемое в форматированный документ значение.  
В зависимости от значения параметра <ТипЭлемента> данного метода значение интерпретируется по-разному.  
Если значение параметра [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), то в указанную позицию документа будет добавлено строковое представление переданного значения.  
Если значение [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md), то тип значения должен быть [Картинка](Картинка.md), [ДвоичныеДанные](ДвоичныеДанные.md) или Строка, содержащая URL картинки. В противном случае вызывается исключение.   
Если значение параметра [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), то значение параметра игнорируется и добавляется перевод строки.  
Значение по умолчанию: Неопределено.

<ТипЭлемента> (необязательный)

Тип: Тип.   
Указывает тип элемента, который нужно добавить. В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Элементами коллекции могут быть:  
1) [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md)  
2) [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md)  
3) [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md)  
В зависимости от значения этого параметра интерпретируется значение параметра <Значение> данного метода.  
Значение по умолчанию: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md).

Возвращаемое значение:

Тип: [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md).   
Возвращается элемент, добавленный в форматированный документ.

Описание:

Добавляет строку, картинку или перевод строки в конец форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Добавление картинки доступно только на сервере.  
Картинка должна быть абсолютной.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.21.
