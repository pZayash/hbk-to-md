Расширение формы клиентского приложения для отчета (Client application form extension for reports)

КлючВарианта (VariantKey)

Доступен, начиная с версии 8.2.

Описание:

Тип: Произвольный.   
Содержит ключ варианта, который будет использоваться в отчете.   
Если не установлен параметр формы [Вариант](Расширение_формы_клиентского_приложения_для_отчета.Вариант_(Client_application_form_extension_for_reports.Variant).md), то будут загружены настройки с указанным ключом.

Использование в версии:

Доступен, начиная с версии 8.2.
