Расширение тестируемого поля формы для поля ввода (Tested form field extension for an input field)

УменьшитьЗначение (DecreaseValue)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УменьшитьЗначение()

Описание:

Эмулирует нажатие кнопки Уменьшить значение поля ввода

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
