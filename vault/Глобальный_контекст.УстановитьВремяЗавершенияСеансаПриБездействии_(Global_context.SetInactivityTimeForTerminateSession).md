Глобальный контекст (Global context)

УстановитьВремяЗавершенияСеансаПриБездействии (SetInactivityTimeForTerminateSession)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьВремяЗавершенияСеансаПриБездействии(<Таймаут>)

Параметры:

<Таймаут> (обязательный)

Тип: [Число](lang__def_Number.md).   
Интервал бездействия (в секундах), по истечении которого сеанс завершается.

Описание:

Устанавливает интервал времени (в секундах), по истечении которого сеанс завершается по причине бездействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
