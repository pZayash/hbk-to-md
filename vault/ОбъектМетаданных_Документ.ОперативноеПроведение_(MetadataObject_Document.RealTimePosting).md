ОбъектМетаданных: Документ (MetadataObject: Document)

ОперативноеПроведение (RealTimePosting)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОперативноеПроведение](ОперативноеПроведение_(RealTimePosting).md).   
Если это свойство установлено в значение [Разрешить](ОперативноеПроведение.Разрешить_(RealTimePosting.Allow).md), то оперативное проведение документа разрешено. В противном случае оперативное проведение документа запрещено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
