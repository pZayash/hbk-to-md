ВнешнийИсточникДанныхТаблицаМенеджер.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableManager.<External source name>.<External data source table name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.2.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаСсылка.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRef.External_source_name.External_data_source_table_name).md).   

Описание:

Получает пустое значение ссылки на таблицу внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.  
Только для таблиц с типом данных [ОбъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.ОбъектныеДанные_(ExternalDataSourceTableDataType.ObjectData).md).

Использование в версии:

Доступен, начиная с версии 8.2.
