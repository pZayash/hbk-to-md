КоллекцияРисунковТабличногоДокумента (SpreadsheetDocumentDrawingCollection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс рисунка.

Возвращаемое значение:

Тип: [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.  
Нельзя получить рисунок типа [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.0.
