# ДоступныеПоляКомпоновкиДанных.НайтиПоле

ДоступныеПоляКомпоновкиДанных (DataCompositionAvailableFields)

НайтиПоле (FindField)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПоле(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md).   
Поле системы компоновки данных.

Возвращаемое значение:

Тип: [ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных_(DataCompositionAvailableField).md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных_(DataCompositionFilterAvailableField).md), [Неопределено](lang__def_Undefined.md).   
Если поле не найдено, возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск доступного поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
