Доступен, начиная с версии 8.0.

Только чтение.

Описание:

Тип: Число.   
Содержит уникальный номер строки данной записи в списке записей по регистратору, указанному в значении свойства [Регистратор](РегистрНакопленияЗапись.Имя_регистра_накопления.Регистратор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
