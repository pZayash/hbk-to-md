Расширение формы клиентского приложения для справочника (Client application form extension for catalogs)

ВыборДоступен (ChoiceAvailable)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что значения реквизитов объекта соответствуют параметрам выбора и это значение можно выбрать.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
