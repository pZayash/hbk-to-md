Глобальный контекст (Global context)

РабочийКаталогДанныхПользователяАсинх (UserDataWorkDirAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РабочийКаталогДанныхПользователяАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - тип Строка. Имя каталога.  
В случае, если создание каталога не возможно, будет вызвано исключение.

Описание:

Асинхронная функция. Предоставляет доступ к каталогу данного пользователя данной информационной базы внутри стандартного каталога данных приложения.   
В случае, если этот каталог отсутствует, он будет создан при обращении к методу.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Для работы функции в веб-клиенте необходимо предварительно подключить расширение работы с файлами.
