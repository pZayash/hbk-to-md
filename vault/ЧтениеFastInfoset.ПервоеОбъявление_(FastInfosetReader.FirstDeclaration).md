ЧтениеFastInfoset (FastInfosetReader)

ПервоеОбъявление (FirstDeclaration)

Доступен, начиная с версии 8.1.

Синтаксис:

ПервоеОбъявление()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - позиционирование на объявление прошло удачно, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Осуществляет переход к первому определению DTD.  
В случае, если текущий узел является [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md) (DTD), текущий узел позиционируется на первое объявление DTD.  
Вызов метода [ЧтениеFastInfoset.Прочитать](ЧтениеFastInfoset.Прочитать_(FastInfosetReader.Read).md) прерывает процесс чтения DTD и позиционирует текущий узел на следующий после DTD.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
