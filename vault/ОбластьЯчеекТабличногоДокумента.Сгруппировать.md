ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

Сгруппировать (Group)

Доступен, начиная с версии 8.2.

Синтаксис:

Сгруппировать(<ИмяГруппы>, <РасположениеЗаголовка>)

Параметры:

<ИмяГруппы> (необязательный)

Тип: Строка.   
Название группы.

<РасположениеЗаголовка> (необязательный)

Тип: [РасположениеЗаголовкаГруппировкиТабличногоДокумента](РасположениеЗаголовкаГруппировкиТабличногоДокумента.md).   
Расположение заголовка группировки.  
Значение по умолчанию: [Авто](РасположениеЗаголовкаГруппировкиТабличногоДокумента.Авто.md).

Описание:

Группирует область, состоящую из строк или колонок.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Область.Сгруппировать("Товар"); |

См. также:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md), метод [Разгруппировать](ОбластьЯчеекТабличногоДокумента.Разгруппировать.md)  

Использование в версии:

Доступен, начиная с версии 8.2.
