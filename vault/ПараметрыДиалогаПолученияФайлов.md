Описание:

Содержит настройки диалога получения файлов.  
Используется в качестве параметра <ПараметрыДиалога> для методов [НачатьПолучениеФайлаССервера](Глобальный_контекст.НачатьПолучениеФайлаССервера.md) и [НачатьПолучениеФайловССервера](Глобальный_контекст.НачатьПолучениеФайловССервера.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
