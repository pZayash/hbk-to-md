ФакторЛинииТрендаДиаграммы (ChartTrendlineFactor)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Фактор устанавливается автоматически в зависимости от значения всех точек диаграммы.  
Если значения всех точек являются датами или числами, то используется [ЗначениеТочки](ФакторЛинииТрендаДиаграммы.ЗначениеТочки_(ChartTrendlineFactor.PointValue).md).  
В противном случае используется [НомерТочки](ФакторЛинииТрендаДиаграммы.НомерТочки_(ChartTrendlineFactor.PointNumber).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
