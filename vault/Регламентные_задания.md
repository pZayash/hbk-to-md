# Регламентные задания

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › [Общие объекты](Общие_объекты.md)

Регламентные задания предназначены для выполнения фоновых задач по расписанию.   
Механизм регламентных заданий решает следующие задачи:

- Регистрацию регламентных заданий в информационной базе;
- Назначение расписания регламентному заданию;
- Определение условно-постоянных регламентных заданий с использованием метаданных;
- Запуск заданий в соответствии с расписанием регламентного задания;
- Управление регламентными заданиями (включение/выключение и др.).
