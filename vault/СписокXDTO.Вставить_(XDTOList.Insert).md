СписокXDTO (XDTOList)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция в списке, в которую необходимо вставить элемент данных.

<Элемент> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Вставляемый элемент данных.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Вставленный элемент данных.

Описание:

Вставляет элемент данных в указанную позицию списка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
