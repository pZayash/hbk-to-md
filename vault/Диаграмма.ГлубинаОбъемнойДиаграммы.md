Диаграмма (Chart)

ГлубинаОбъемнойДиаграммы (Chart3DDepth)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: Число, Неопределено.   
Глубина объемной диаграммы.  
Используется в круговой, кольцевой и объемных гистограммах.  
Если Неопределено - используемое значение зависит от типа диаграммы:

- Для круговой и кольцевой диаграммы - 20, в процентах от радиуса.
- Для объемных гистограмм – 50. Для объемных гистограмм глубина задает значение в процентах от ширины столбца.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.24.
