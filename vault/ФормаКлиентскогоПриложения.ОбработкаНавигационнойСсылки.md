ФормаКлиентскогоПриложения (ClientApplicationForm)

ОбработкаНавигационнойСсылки (URLProcessing)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ОбработкаНавигационнойСсылки(<НавигационнаяСсылкаФорматированнойСтроки>, <СтандартнаяОбработка>)

Параметры:

<НавигационнаяСсылкаФорматированнойСтроки>

Тип: Строка.   
Значение гиперссылки форматированной строки.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается при нажатии на гиперссылку форматированной строки, расположенной в элементе формы типа [Расширение поля формы для поля надписи](Расширение_поля_формы_для_поля_надписи.md) или в элементе формы [Расширение декорации формы для надписи](Расширение_декорации_формы_для_надписи.md), у которого свойство Гиперссылка имеет значение Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.1.
