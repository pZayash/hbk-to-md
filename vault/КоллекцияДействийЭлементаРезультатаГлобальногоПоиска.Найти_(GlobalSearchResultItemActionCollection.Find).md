КоллекцияДействийЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemActionCollection)

Найти (Find)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению действия элемента результата глобального поиска.

Возвращаемое значение:

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Осуществляет поиск действия элемента результата глобального поиска по значению свойства [Значение](ЭлементРезультатаГлобальногоПоиска.Значение_(GlobalSearchResultItem.Value).md). Возвращает первое действие с совпадающим значением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
