ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

КоличествоЭкземпляров (Copies)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Число, Неопределено.   
Количество экземпляров каждого документа.   
Неопределено - количество экземпляров определяется родительским пакетом или считается 1, если родительского пакета нет.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
