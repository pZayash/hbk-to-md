ЧтениеFastInfoset (FastInfosetReader)

ЭтоСимвольныеДанные (IsCharacters)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, имеет ли текущий узел текстовое значение. Текстовое значение имеется у узлов: [Текст](ТипУзлаXML.Текст_(XMLNodeType.Text).md), [СекцияCDATA](ТипУзлаXML.СекцияCDATA_(XMLNodeType.CDATASection).md), [ПробельныеСимволы](ЧтениеFastInfoset.ПробельныеСимволы_(FastInfosetReader.Space).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
