ИнтернетПрокси (InternetProxy)

Сервер (Server)

Доступен, начиная с версии 8.0.

Синтаксис:

Сервер(<Протокол>)

Параметры:

<Протокол> (необязательный)

Тип: Строка.   
Имя протокола, для которого запрашивается установленный прокси-сервер.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: Строка.   
При указании пустой строки возвращается прокси-сервер, установленный по умолчанию.

Описание:

Получает прокси-сервер по имени протокола.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
