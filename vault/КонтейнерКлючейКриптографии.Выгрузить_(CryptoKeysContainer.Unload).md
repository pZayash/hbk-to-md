# КонтейнерКлючейКриптографии.Выгрузить

КонтейнерКлючейКриптографии (CryptoKeysContainer)

Выгрузить (Unload)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: В двоичные данные

Синтаксис:

Выгрузить()

Описание варианта метода:

Метод возвращает двоичные данные выгруженного закрытого ключа.

Вариант синтаксиса: В поток

Синтаксис:

Выгрузить(<Поток>)

Параметры:

<Поток> (необязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток, в который будет записан закрытый ключ.

Описание варианта метода:

Метод возвращает данные контейнера ключей криптографии, использованные при создании или инициализации объекта, в поток.

Вариант синтаксиса: В файл

Синтаксис:

Выгрузить(<ИмяФайла>)

Параметры:

<ИмяФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выгружен сертификат.

Описание варианта метода:

Сохраняет данные контейнера ключей криптографии в файл.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает данные, использованные при создании или инициализации объекта.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии_(CryptoKeysContainer).md), метод [ВыгрузитьАсинх](КонтейнерКлючейКриптографии.ВыгрузитьАсинх_(CryptoKeysContainer.UnloadAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.18.
