ОформлениеЯчейкиДинамическогоСписка (DynamicListCellAppearance)

НайтиЗначениеПараметра (FindParameterValue)

Доступен, начиная с версии 8.3.10.

Синтаксис:

НайтиЗначениеПараметра(<Параметр>)

Параметры:

<Параметр> (обязательный)

Тип: [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных.md).   
Параметр, значение которого нужно найти.

Возвращаемое значение:

Тип: [ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных.md), Неопределено.   
Если значение не найдено, то возвращается значение Неопределено.

Описание:

Осуществляет поиск значения параметра.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
