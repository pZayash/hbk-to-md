Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) строки.

Описание:

Предопределенная табличная часть вида расчета - список вытесняющих видов расчета. Такая табличная часть определена только для планов видов расчета с признаком "Использует период действия".  
Имеет колонки - ВидРасчета(CalculationKind) типа [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)> и Предопределенный(Predefined) типа [Булево](lang__Булево_(Boolean).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
