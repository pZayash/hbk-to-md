ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

ПоддерживаетсяДействиеКнопкиПанелиКнопокСообщения (MessageButtonPanelButtonActionSupported)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПоддерживаетсяДействиеКнопкиПанелиКнопокСообщения(<ДействиеКнопки>)

Параметры:

<ДействиеКнопки> (обязательный)

Тип: [ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonAction).md).   
Действие кнопки.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Определяет, поддерживает ли данная интеграция кнопки, выполняющие данное действие.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
