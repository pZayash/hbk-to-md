ЧтениеJSON (JSONReader)

ТипТекущегоЗначения (CurrentValueType)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ТипЗначенияJSON](ТипЗначенияJSON_(JSONValueType).md).   
Тип текущего значения в документе JSON.  
[Неопределено](lang__def_Undefined.md) - если чтение еще не началось или достигнут конец файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
