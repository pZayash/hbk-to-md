МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

ТекущийСпособБиометрическойПроверки (CurrentBiometricVerificationMethod)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ТекущийСпособБиометрическойПроверки()

Возвращаемое значение:

Тип: [СпособБиометрическойПроверки](СпособБиометрическойПроверки_(BiometricVerificationMethod).md), [Неопределено](lang__def_Undefined.md).   
Если получить информацию о наличии биометрической проверки невозможно (в Windows Runtime), возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает способ биометрической проверки, который поддерживает устройство.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
