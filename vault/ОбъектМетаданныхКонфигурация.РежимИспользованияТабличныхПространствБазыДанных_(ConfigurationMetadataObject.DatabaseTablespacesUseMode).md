ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

РежимИспользованияТабличныхПространствБазыДанных (DatabaseTablespacesUseMode)

Доступен, начиная с версии 8.3.23 (в режиме совместимости с версией 8.3.1 и последующими).

Использование:

Только чтение.

Описание:

Тип: [РежимИспользованияТабличныхПространствБазыДанных](РежимИспользованияТабличныхПространствБазыДанных_(DatabaseTablespacesUseMode).md).   
Задает режим использования табличных пространств базы данных.  
В режиме совместимости конфигурации [Версия8\_3\_22](РежимСовместимости.Версия8_3_22_(CompatibilityMode.Version8_3_22).md) и ниже устанавливается в значение [НеИспользовать](РежимИспользованияТабличныхПространствБазыДанных.НеИспользовать_(DatabaseTablespacesUseMode.DontUse).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23 (в режиме совместимости с версией 8.3.1 и последующими).
