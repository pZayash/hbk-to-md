ПотокВПамяти (MemoryStream)

НачатьУстановкуРазмера (BeginSetSize)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьУстановкуРазмера(<ОписаниеОповещения>, <Размер>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Размер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Устанавливаемый размер потока.

Описание:

Начинает установку размера потока.  
Если текущий размер превышает заданный, поток будет сокращен до заданного размера, а информация, превышающая заданный размер, будет потеряна.  
Если текущий размер потока меньше заданного, то содержимое потока между старым и новым размером не определено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент, интеграция.

Примечание:

Установить длину потока можно только в том случае, если в потоке разрешены запись и изменение позиции. В противном случае будет сгенерировано исключение.

См. также:

[Поток](Поток_(Stream).md), метод [УстановитьРазмер](Поток.УстановитьРазмер_(Stream.SetSize).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
