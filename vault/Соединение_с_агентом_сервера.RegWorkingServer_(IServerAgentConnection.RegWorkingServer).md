Соединение с агентом сервера (IServerAgentConnection)

RegWorkingServer (RegWorkingServer)

Доступен, начиная с версии 8.1.

Синтаксис:

RegWorkingServer(<Кластер>, <Сервер>, <УстановитьЗапрещающееТребованиеНазначенияФункциональности>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должен быть зарегистрирован новый рабочий сервер.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md).   
Рабочий сервер, который должен быть зарегистрирован в кластере. Перед регистрацией он должен быть создан методом [CreateWorkingServerInfo](Соединение_с_агентом_сервера.CreateWorkingServerInfo_(IServerAgentConnection.CreateWorkingServerInfo).md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера_(IServerAgentConnection).md), после чего ему должны быть установлены значения свойств.

<УстановитьЗапрещающееТребованиеНазначенияФункциональности> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли при добавлении рабочего сервера в кластер автоматически создаваться требование назначение функциональности на этом сервере "Любой объект требования - Не назначать - пусто - пусто - 0". Если требование создается (значение параметра - [Истина](lang__def_BooleanTrue.md)), оно считается примененным. После добавления рабочего сервера на него не назначается никакая функциональность.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Выполняет регистрацию рабочего сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.25.
