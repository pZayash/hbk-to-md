ОбъектМетаданных: Поле (MetadataObject: Field)

ЗначениеЗаполнения (FillingValue)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Значение типа, совпадающего с типом поля. Данное значение будет присваиваться полю процедурой стандартной обработки заполнения, если поле не заполнено из данных заполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
