Доступен, начиная с версии 8.3.5.

<Индекс> (обязательный)

Индекс позиции, на которую надо вставить новую запись.

Тип: [ВнешнийИсточникДанныхТаблицаЗапись.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаЗапись.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecord.External_source_name.Table_of_external_data_source_name).md).   

Описание:

Вставляет новую запись в набор на указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
