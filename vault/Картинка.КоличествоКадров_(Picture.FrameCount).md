Картинка (Picture)

КоличествоКадров (FrameCount)

Доступен, начиная с версии 8.3.15.

Синтаксис:

КоличествоКадров()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает количество кадров картинки (применимо для многостраничных файлов TIFF).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.16.
