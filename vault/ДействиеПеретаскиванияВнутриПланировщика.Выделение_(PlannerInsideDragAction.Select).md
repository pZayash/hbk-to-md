ДействиеПеретаскиванияВнутриПланировщика (PlannerInsideDragAction)

Выделение (Select)

Доступен, начиная с версии 8.3.18.

Описание:

Указывает на то, что происходит выделение существующих элементов с помощью с зажатой клавиши Shift.

Использование в версии:

Доступен, начиная с версии 8.3.18.
