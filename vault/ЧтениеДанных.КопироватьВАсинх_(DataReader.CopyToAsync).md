# ЧтениеДанных.КопироватьВАсинх

ЧтениеДанных (DataReader)

КопироватьВАсинх (CopyToAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КопироватьВАсинх(<Поток>, <Количество>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).   
Поток или объект [ЗаписьДанных](ЗаписьДанных_(DataWriter).md), в который требуется скопировать данные.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые необходимо скопировать. Если не задано, будут скопированы все данные до конца потока.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - фактическое количество скопированных байтов типа [Число](lang__def_Number.md).

Описание:

Асинхронная функция. Копирует заданное количество байтов из объекта [ЧтениеДанных](ЧтениеДанных_(DataReader).md) в указанный поток или объект [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Если количество не указано, копируются все данные из объекта.

Доступность:

Тонкий клиент, толстый клиент.
