МенеджерОбработкиОшибок (ErrorProcessingManager)

КраткоеПредставлениеОшибки (BriefErrorDescription)

Доступен, начиная с версии 8.3.17.

Синтаксис:

КраткоеПредставлениеОшибки(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md).   
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает строку с кратким представлением ошибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
