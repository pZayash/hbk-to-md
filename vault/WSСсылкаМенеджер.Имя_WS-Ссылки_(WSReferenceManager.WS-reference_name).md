# WSСсылкаМенеджер.<Имя WS-Ссылки>

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Web-сервисы](Web-сервисы-2.md)

Доступен, начиная с версии 8.1.

Методы:

[ПолучитьWSОпределения (GetWSDefinitions)](WSСсылкаМенеджер.Имя_WS-Ссылки.ПолучитьWSОпределения_(WSReferenceManager.WS-reference_name.GetWSDefinitions).md)  
[СоздатьWSПрокси (CreateWSProxy)](WSСсылкаМенеджер.Имя_WS-Ссылки.СоздатьWSПрокси_(WSReferenceManager.WS-reference_name.CreateWSProxy).md)  

Описание:

Предназначен для управления менеджером ссылки на внешний Web-сервис. Через данный менеджер можно создавать объекты-прокси для обращения к операциям, предоставляемым внешним Web-сервисом, а также получать WSОпределения, содержащие набор определений, описывающих Web-сервис.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[WSСсылкиМенеджер](WSСсылкиМенеджер_(WSReferencesManager).md), свойство [\<Имя WS-ссылки\>](WSСсылкиМенеджер.Имя_WS-ссылки_(WSReferencesManager.WS_Reference_Name).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
