Форма (Form)

МодальныйРежим (ModalMode)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак монопольного открытия формы.  
[Истина](lang__def_BooleanTrue.md) - форма открыта в модальном режиме.

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), метод [ОткрытьМодально](Форма.ОткрытьМодально_(Form.DoModal).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
