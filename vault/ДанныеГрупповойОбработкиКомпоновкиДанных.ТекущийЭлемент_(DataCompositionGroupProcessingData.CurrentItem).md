ДанныеГрупповойОбработкиКомпоновкиДанных (DataCompositionGroupProcessingData)

ТекущийЭлемент (CurrentItem)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений_(ValueTableRow).md), [Неопределено](lang__def_Undefined.md).   
Содержит текущую строку таблицы значений, находящейся в свойстве [Данные](ДанныеГрупповойОбработкиКомпоновкиДанных.Данные_(DataCompositionGroupProcessingData.Data).md).  
Для итоговых строк будет содержать значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
