ДиаграммаГанта (GanttChart)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначение(<Точка>, <Серия>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта.md).   
Точка диаграммы Ганта.

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта.md).   
Серия диаграммы Ганта.

Возвращаемое значение:

Тип: [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта.md).   
Значение диаграммы Ганта, соответствующее заданной точке и серии.

Описание:

Получает значение, соответствующее заданной точке и серии диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Значение = ДиаграммаГанта.ПолучитьЗначение(Точка, Серия); |

Использование в версии:

Доступен, начиная с версии 8.0.
