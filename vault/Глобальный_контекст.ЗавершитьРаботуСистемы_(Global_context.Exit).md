Глобальный контекст (Global context)

ЗавершитьРаботуСистемы (Exit)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗавершитьРаботуСистемы(<ЗапрашиватьВозможность>, <ПерезапуститьПриЗавершении>, <ДополнительныеПараметрыКоманднойСтроки>)

Параметры:

<ЗапрашиватьВозможность> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Ложь](lang__def_BooleanFalse.md), то завершение работы программы будет выполнено безусловно (независимо от наличия редактируемых данных).  
Если значение параметра [Истина](lang__def_BooleanTrue.md), то будут вызываться обработчики закрытия для каждой открытой формы и для всего приложения. Для редакторов текстов, табличных документов и HTML-документов будет произведен анализ модифицированности. Если нет факторов, препятствующих закрытию (например, данные модифицированы и не сохранены), то выполняется завершение работы. В противном случае (такие данные есть) будет выдан вопрос о необходимости сохранения документов.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ПерезапуститьПриЗавершении> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость выполнения запуска нового сеанса с текущими установками после завершения работы.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ДополнительныеПараметрыКоманднойСтроки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит дополнительные параметры, которые будут переданы новому сеансу. Если такие параметры уже есть среди текущих установок, переданные заменят текущие.  
Параметр указывается, если значение параметра <ПерезапуститьПриЗавершении>=[Истина](lang__def_BooleanTrue.md).  
Если режим приложения ([УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md)/ [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md)) не задан явно дополнительным параметром командной строки, при запуске текущего приложения учитывается использование параметров принудительного включения режима управляемого или обычного приложения /RunModeManagedApplication и /RunModeOrdinaryApplication.

Описание:

Позволяет принудительно вызвать завершение работы 1С:Предприятия. В зависимости от переданного параметра будут вызываться обработчики закрытия и завершения работы системы или завершает работу безусловно.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если вызов осуществляется из модуля формы, открытой в модальном режиме, завершение работы системы не выполняется.  
Если параметр <ПерезапуститьПриЗавершении>=[Истина](lang__def_BooleanTrue.md), то после завершения работы системы будет открыт новый сеанс 1С:Предприятия.   
Если в процессе завершения системы пользователь отменит завершение работы, запуск нового сеанса выполнен не будет.

Пример:

|  |
| --- |
| // безусловное завершение работы программы  ЗавершитьРаботуСистемы(Ложь); |

Использование в версии:

Доступен, начиная с версии 8.1.
