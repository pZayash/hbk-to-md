Глобальный контекст (Global context)

СократитьЖурналРегистрации (TruncateEventLog)

Доступен, начиная с версии 8.3.20.

Синтаксис:

СократитьЖурналРегистрации(<КонецПериода>, <ИмяФайла>)

Параметры:

<КонецПериода> (обязательный)

Тип: Дата.   
Конец периода сокращения журнала. События старше этой даты будут удалены из журнала регистрации.

<ИмяФайла> (необязательный)

Тип: Строка.   
Имя файла, в который будут сохранены удаленные события.   
Если журнал регистрации информационной базы имеет последовательный формат, то имя будет использовано в качестве имени файла-легенды. Имена файлов с данными будут использовать это имя без расширения в качестве префикса, за которым будут следовать исходные имена с расширениями файлов данных журнала регистрации.   
Если журнал регистрации имеет формат базы данных SQLite, то имя используется в качестве имени файла базы данных SQLite, в которую будут сохранены удаляемые события.   
В случае ошибки генерируется исключение, и события журнала регистрации не удаляются. Если файл с указанным именем уже есть, то будет выполнена попытка добавления к нему удаляемых событий.   
Если пустая строка, то удаляемые события нигде не сохраняются.  
Значение по умолчанию: Пустая строка.

Описание:

Удаляет события журнала регистрации старше заданной даты.  
Требуется наличие у текущего пользователя права "Администрирование".

Доступность:

Сервер, толстый клиент, внешнее соединение.
