КонтрольныеТочкиИтоговСхемыЗапроса (QuerySchemaTotalCalculationFields)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: По выражению

Синтаксис:

Добавить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md).   
Содержит выражение.

Описание варианта метода:

Добавляет новый элемент по выражению. Если выражение содержит ошибку, будет выдано исключение, включающее текст описания ошибки.

Вариант синтаксиса: По колонке

Синтаксис:

Добавить(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса.md).   
Колонка схемы запроса.

Описание варианта метода:

Добавляет новый элемент по колонке.

Вариант синтаксиса: По строке

Синтаксис:

Добавить(<КолонкаИлиВыражение>)

Параметры:

<КолонкаИлиВыражение> (обязательный)

Тип: Строка.   
По переданной строке сначала будет выполнен поиск колонки. Если не удачно, будет создано выражение, которое добавляется в коллекцию. Если полученное выражение не корректно, будет вызвано исключение.

Описание варианта метода:

По переданной строке выполняется поиск колонки. Если ничего не найдено, создается выражение и добавляется в коллекцию.  
Если полученное выражение не корректно, выводится исключение.

Возвращаемое значение:

Тип: [КонтрольнаяТочкаИтоговСхемыЗапроса](КонтрольнаяТочкаИтоговСхемыЗапроса.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
