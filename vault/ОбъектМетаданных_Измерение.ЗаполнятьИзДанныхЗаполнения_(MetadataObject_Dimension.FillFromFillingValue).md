ОбъектМетаданных: Измерение (MetadataObject: Dimension)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство определяет возможность заполнения измерения из данных заполнения. Используется для заполнения измерений из значений текущего отбора списков. Если установлено [Истина](lang__def_BooleanTrue.md), то процедура стандартной обработки заполнения будет заполнять измерение из данных заполнения. Если в данных заполнения нет данных для заполнения этого измерения или свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то измерение будет заполнено из свойства [ЗначениеЗаполнения](ОбъектМетаданных_Реквизит.ЗначениеЗаполнения_(MetadataObject_Attribute.FillingValue).md), если оно установлено. Данное свойство устанавливается системой в значение [Истина](lang__def_BooleanTrue.md) для некоторых стандартных измерений (Родитель, Владелец, ведущие измерения регистров сведений).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
