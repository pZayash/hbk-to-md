ЗадачаМенеджер.<Имя задачи> (TaskManager.<Task name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор.md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ЗадачаСсылка.Имя_задачи.УникальныйИдентификатор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

См. также:

[ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md), метод [УникальныйИдентификатор](ЗадачаСсылка.Имя_задачи.УникальныйИдентификатор.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
