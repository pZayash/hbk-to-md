МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПоддерживаетсяВнешнийДоступ (ExternalAccessSupported)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПоддерживаетсяВнешнийДоступ()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если внешний доступ поддерживается (на сервере системы взаимодействия опубликован сайт для внешних пользователей).

Описание:

Определяет, подерживается ли внешний доступ на сервере системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
