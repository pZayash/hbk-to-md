Глобальный контекст (Global context)

СтрПолучитьСтроку (StrGetLine)

Доступен, начиная с версии 8.0.

Синтаксис:

СтрПолучитьСтроку(<Строка>, <НомерСтроки>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Исходная строка.

<НомерСтроки> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер получаемой строки. Нумерация строк в многострочной строке начинается с 1.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Полученная строка.

Описание:

Получает строку многострочной строки по номеру.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Для Счетчик = 1 По СтрЧислоСтрок(ИсходныйТекст) Цикл     ТекСтрока = СтрПолучитьСтроку(ИсходныйТекст, Счетчик);     ... КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
