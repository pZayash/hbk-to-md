ДвоичныеДанные (BinaryData)

Записать (Write)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Запись в файл

Синтаксис:

Записать()

Описание варианта метода:

Записывает двоичные данные в файл с указанным именем.

Вариант синтаксиса: Запись в файл

Синтаксис:

Записать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который требуется записать двоичные данные. Если файл не существует, он будет создан. В противном случае, его содержимое будет перезаписано.

Описание варианта метода:

Записать двоичные данные в файл с заданным именем.

Вариант синтаксиса: Запись в поток

Синтаксис:

Записать(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md).   
Поток, в который необходимо записать двоичные данные.

Описание варианта метода:

Записывает двоичные данные в заданный поток.

Описание:

Записывает двоичные данные в файл или поток.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
