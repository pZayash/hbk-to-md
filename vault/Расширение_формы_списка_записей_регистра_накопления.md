# Расширение формы списка записей регистра накопления

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Регистры накопления](Регистры_накопления.md)

Доступен, начиная с версии 8.0.

Свойства:

[ПараметрОтборПоРегистратору (FilterByRecorderParameter)](Расширение_формы_списка_записей_регистра_накопления.ПараметрОтборПоРегистратору.md)  
[ПараметрТекущаяСтрока (CurrentLineParameter)](Расширение_формы_списка_записей_регистра_накопления.ПараметрТекущаяСтрока.md)  

Описание:

Дополнительные свойства объекта [Форма](Форма.md), в том случае, если основной реквизит формы имеет тип [РегистрНакопленияСписок.](РегистрНакопленияСписок.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияСписок.Имя_регистра_накопления.md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Приходные = Документы.ПриходнаяНакладная; Док = Приходные.НайтиПоНомеру("00003",ТекущаяДата()); Остатки = РегистрыНакопления.УчетНоменклатуры; Форма = Остатки.ПолучитьФормуСписка(); Форма.ПараметрОтборПоРегистратору = Док; Форма.Открыть(); |

Использование в версии:

Доступен, начиная с версии 8.0.
