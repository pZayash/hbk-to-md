# МенеджерХранилищаДвоичныхДанных.СоздатьПолнуюРезервнуюКопию

МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

СоздатьПолнуюРезервнуюКопию (CreateFullBackup)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СоздатьПолнуюРезервнуюКопию(<Путь>, <УниверсальнаяДатаДоКоторойВыполнятьОчистку>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу с резервной копией.

<УниверсальнаяДатаДоКоторойВыполнятьОчистку> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Указывает дату и время в формате UTC, до которого будет выполнена однократная очистка неиспользуемого места для данных после успешного окончания резервного копирования.  
  
Если [Неопределено](lang__def_Undefined.md) – очистка не будет выполнена.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Создает полную резервную копию хранилища двоичных данных.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены.  
Формат создаваемого файла: \*.sbf (storage full backup).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.

Описание изменено в версии 8.3.26.
