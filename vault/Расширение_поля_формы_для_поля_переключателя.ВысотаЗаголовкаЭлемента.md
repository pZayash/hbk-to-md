Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ВысотаЗаголовкаЭлемента (ItemTitleHeight)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Высота заголовка элемента. 0 - определять автоматически.  
Используется при виде переключателя [Тумблер](ВидПереключателя.Тумблер.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
