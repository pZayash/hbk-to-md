# COMОбъект

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.0.

Конструкторы:

[По имени приложения](COMОбъект.По_умолчанию.md)  

Описание:

Создает COM-объект (например, Word, Excel и т.д.). Методы и свойства СОМ-объектов в дальнейшем становятся доступными через данный объект.  
  
В веб-клиенте данный объект поддерживается только в браузере Internet Explorer.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
