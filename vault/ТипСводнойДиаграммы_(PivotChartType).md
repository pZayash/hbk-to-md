Значения

Описание:

Содержит варианты типов сводных диаграмм. Используется для определения свойства [ТипСводнойДиаграммы](СводнаяДиаграмма.ТипСводнойДиаграммы_(PivotChart.PivotChartType).md) объекта [СводнаяДиаграмма](СводнаяДиаграмма_(PivotChart).md).

Доступность:

Тонкий клиент, сервер, толстый клиент.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/ui}. Имя типа XDTO: PivotChartType.

См. также:

[СводнаяДиаграмма](СводнаяДиаграмма_(PivotChart).md), свойство [ТипСводнойДиаграммы](СводнаяДиаграмма.ТипСводнойДиаграммы_(PivotChart.PivotChartType).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
