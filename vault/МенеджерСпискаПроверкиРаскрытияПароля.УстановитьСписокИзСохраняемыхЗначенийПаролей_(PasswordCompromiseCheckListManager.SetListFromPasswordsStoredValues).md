МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

УстановитьСписокИзСохраняемыхЗначенийПаролей (SetListFromPasswordsStoredValues)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьСписокИзСохраняемыхЗначенийПаролей(<СписокПроверкиРаскрытияПароля>)

Параметры:

<СписокПроверкиРаскрытияПароля> (обязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md).   
Список проверки раскрытия пароля.  
Список может быть размещен:

- в массиве строк,
- в файле, содержащем хэш-значения паролей, вычисленные алгоритмом SHA1 и закодированные в base64, разделенные переносом строки (в этом случае указывается имя файла),
- передан в виде потока или представлен как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Описание:

Формирует список проверки раскрытия пароля из сохраняемых значений пароля, удаляя дубликаты. Количество значений не может превышать 100000.

Доступность:

Сервер, толстый клиент.

Доступен, начиная с версии 8.3.26.
