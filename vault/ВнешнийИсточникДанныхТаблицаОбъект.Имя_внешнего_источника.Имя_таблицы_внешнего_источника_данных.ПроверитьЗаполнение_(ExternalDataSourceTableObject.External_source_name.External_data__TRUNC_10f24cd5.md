ВнешнийИсточникДанныхТаблицаОбъект.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableObject.<External source name>.<External data source table name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - в процессе проверки ошибок не обнаружено,  
[Ложь](lang__def_BooleanFalse.md) - в процессе проверки были ошибки.

Описание:

Вызывает проверку заполнения полей. Для незаполненных полей, у которых свойство ПроверкаЗаполнения установлено в значение "Показывать ошибку", будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
