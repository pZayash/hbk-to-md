# ИнтернетПочта.УдалитьПочтовыйЯщик

ИнтернетПочта (InternetMail)

УдалитьПочтовыйЯщик (DeleteMailbox)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьПочтовыйЯщик(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя почтового ящика. Может содержать в себе иерархию. В качестве разделителя используется символ, который доступен через свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель_(InternetMail.DelimeterChar).md). В большинстве случаев это символ '/'.

Описание:

Удаляет заданный почтовый ящик IMAP со всем его содержимым.  
Символ-разделитель иерархии почтовых ящиков, используемый текущим почтовым сервером, доступен с помощью свойства [СимволРазделитель](ИнтернетПочта.СимволРазделитель_(InternetMail.DelimeterChar).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Работает только при подключении, установленном по протоколу IMAP.

Пример:

|  |
| --- |
| Почта.УдалитьПочтовыйЯщик("inbox/maibox1/maibox2"); |

См. также:

[ИнтернетПочта](ИнтернетПочта_(InternetMail).md), свойство [СимволРазделитель](ИнтернетПочта.СимволРазделитель_(InternetMail.DelimeterChar).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
