ОграниченияИспользованияДоступныхПолейКомпоновкиДанных (DataCompositionAvailableFieldUseRestrictions)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор поля.

Возвращаемое значение:

Тип: [ОграничениеИспользованияДоступногоПоляКомпоновкиДанных](ОграничениеИспользованияДоступногоПоляКомпоновкиДанных_(DataCompositionAvailableFieldUseRestriction).md), [Неопределено](lang__def_Undefined.md).   
Если элемент не найден, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
