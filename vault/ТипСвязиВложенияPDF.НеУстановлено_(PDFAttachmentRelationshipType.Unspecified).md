ТипСвязиВложенияPDF (PDFAttachmentRelationshipType)

НеУстановлено (Unspecified)

Доступен, начиная с версии 8.3.16.

Описание:

Взаимоотношение между вложенным файлом и содержимым документа PDF неизвестно или не может быть описано одним значением.

Использование в версии:

Доступен, начиная с версии 8.3.16.
