МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВидеоконференцииДоступны (VideoconferenceAllowed)

Не рекомендуется использовать, начиная с версии 8.3.20.

Рекомендуется использовать:

- [ПоддерживаютсяВидеоконференции](МенеджерСистемыВзаимодействия.ПоддерживаютсяВидеоконференции_(CollaborationSystemManager.VideoconferenceSupported).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВидеоконференцииДоступны()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если есть возможность совершать видеозвонки.

Описание:

Определяет возможность совершать видеозвонки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.20.

Описание изменено в версии 8.3.20.
