Расширение табличного поля условного оформления компоновки данных (Conditional appearance table box extension)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в [Ложь](lang__def_BooleanFalse.md), именованные части условного оформления (свойство [Представление](ЭлементУсловногоОформленияКомпоновкиДанных.Представление_(DataCompositionConditionalAppearanceItem.Presentation).md) не пустое) отображаются в табличном поле одной строкой с одной ячейкой, в которой выводится представление.  
В противном случае представление редактируется в отдельной колонке, отображение именованных и неименованных элементов условного оформления не отличается.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
