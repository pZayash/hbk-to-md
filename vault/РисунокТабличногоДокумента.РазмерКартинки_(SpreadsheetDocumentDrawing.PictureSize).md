РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

РазмерКартинки (PictureSize)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РазмерКартинки](РазмерКартинки_(PictureSize).md).   
Содержит способ приспособления оригинальных размеров картинки к размеру рисунка.  
Значение [АвтоРазмерБезУчетаМасштаба](РазмерКартинки.АвтоРазмерБезУчетаМасштаба_(PictureSize.AutoSizeIgnoreScale).md) интерпретируется как [АвтоРазмер](РазмерКартинки.АвтоРазмер_(PictureSize.AutoSize).md).  
Значение [РеальныйРазмерБезУчетаМасштаба](РазмерКартинки.РеальныйРазмерБезУчетаМасштаба_(PictureSize.RealSizeIgnoreScale).md) интерпретируется как [РеальныйРазмер](РазмерКартинки.РеальныйРазмер_(PictureSize.RealSize).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунков типа [Картинка](ТипРисункаТабличногоДокумента.Картинка_(SpreadsheetDocumentDrawingType.Picture).md).

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
