# РазыменовательПространствИменDOM.НайтиURIПространстваИмен

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [РазыменовательПространствИменDOM](РазыменовательПространствИменDOM_(DOMNamespaceResolver).md) › [Методы](РазыменовательПространствИменDOM__Методы.md)

РазыменовательПространствИменDOM (DOMNamespaceResolver)

НайтиURIПространстваИмен (LookupNamespaceURI)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиURIПространстваИмен(<Префикс>)

Параметры:

<Префикс> (обязательный)

Тип: [Строка](lang__def_String.md).   
Префикс пространства имен.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
URI пространства имен, соответствующее указанному префиксу, или [Неопределено](lang__def_Undefined.md).

Описание:

Производит поиск URI пространства имен для указанного префикса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
