СводнаяДиаграмма (PivotChart)

Градиент (Gradient)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак градиентной закраски плоских сводных диаграмм.  
[Истина](lang__def_BooleanTrue.md) - градиентная закраска.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется только для плоских сводных гистограмм.

Использование в версии:

Доступен, начиная с версии 8.0.
