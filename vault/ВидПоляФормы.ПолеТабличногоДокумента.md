ВидПоляФормы (FormFieldType)

ПолеТабличногоДокумента (SpreadsheetDocumentField)

Доступен, начиная с версии 8.2.

Описание:

Поле представляется в виде табличного документа. Данный вид недоступен для полей, расположенных в таблице.

Использование в версии:

Доступен, начиная с версии 8.2.
