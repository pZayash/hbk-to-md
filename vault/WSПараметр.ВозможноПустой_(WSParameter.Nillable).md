WSПараметр (WSParameter)

ВозможноПустой (Nillable)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что параметр может содержать значения типа [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - данное свойство может содержать значения [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.1.
