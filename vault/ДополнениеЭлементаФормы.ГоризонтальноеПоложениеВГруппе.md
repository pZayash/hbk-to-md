ДополнениеЭлементаФормы (FormItemAddition)

ГоризонтальноеПоложениеВГруппе (HorizontalAlignInGroup)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента.md).   
Устанавливает горизонтальное выравнивание элемента внутри группы. Значение [Авто](ГоризонтальноеПоложениеЭлемента.Авто.md) означает, что будет использоваться значение свойства ГоризонтальноеПоложениеВГруппе родительской группы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
