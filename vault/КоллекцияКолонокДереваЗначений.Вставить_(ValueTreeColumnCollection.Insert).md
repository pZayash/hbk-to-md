# КоллекцияКолонокДереваЗначений.Вставить

КоллекцияКолонокДереваЗначений (ValueTreeColumnCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Имя>, <Тип>, <Заголовок>, <Ширина>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс в коллекции колонок, с которым необходимо вставить колонку.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки.  
Значение по умолчанию: Пустая строка.

<Тип> (необязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Объект, описывающий допустимые типы значений для колонки. Если параметр не указан, в колонке можно будет хранить значение любого типа.

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок колонки. Используется при визуальном отображении дерева значений.  
Значение по умолчанию: Пустая строка.

<Ширина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Ширина колонки в символах. Используется при визуальном отображении дерева значений.  
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: [КолонкаДереваЗначений](КолонкаДереваЗначений_(ValueTreeColumn).md).   
Вставленная колонка.

Описание:

Вставляет колонку в заданную позицию коллекции колонок дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| МассивТипов = Новый Массив; МассивТипов.Добавить(Тип("Число")); КЧ = Новый КвалификаторыЧисла(10,3); ОписаниеЧисло = Новый ОписаниеТипов(МассивТипов,КЧ); КолонкиСостава = ДеревоСостава.Колонки; КолонкиСостава.Вставить(2, "Количество", ОписаниеЧисло, "Кол-во", 14); |

Использование в версии:

Доступен, начиная с версии 8.0.
