Глобальный контекст (Global context)

ПолучитьИспользуемыйСервер (GetUsedServer)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьИспользуемыйСервер()

Возвращаемое значение:

Тип: [ИспользуемыйСервер](ИспользуемыйСервер_(UsedServer).md).   

Описание:

Возвращает текущее значение, установленное методом [УстановитьИспользуемыйСервер](Глобальный_контекст.УстановитьИспользуемыйСервер_(Global_context.SetUsedServer).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьИспользуемыйСервер](Глобальный_контекст.УстановитьИспользуемыйСервер_(Global_context.SetUsedServer).md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
