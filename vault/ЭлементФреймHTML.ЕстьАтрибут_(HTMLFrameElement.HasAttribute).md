# ЭлементФреймHTML.ЕстьАтрибут

ЭлементФреймHTML (HTMLFrameElement)

ЕстьАтрибут (HasAttribute)

Доступен, начиная с версии 8.2.

Синтаксис:

ЕстьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если атрибут с указанным именем явно присутствует в элементе, или имеет значение по умолчанию; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет наличие атрибута с указанным именем в данном элементе.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
