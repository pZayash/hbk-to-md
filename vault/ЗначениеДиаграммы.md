Описание:

Используется для доступа к свойствам значения диаграммы. Содержит собственно значение диаграммы в определенной точке и серии, а также значение расшифровки в точке и серии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
