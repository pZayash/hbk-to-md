Расширение тестируемого поля формы для поля календаря (Tested form field extension for a calendar field)

ПерейтиНаМесяцВперед (GotoNextMonth)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПерейтиНаМесяцВперед()

Описание:

Передвигает дату на месяц вперёд.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
