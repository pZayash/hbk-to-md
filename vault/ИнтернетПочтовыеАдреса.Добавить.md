ИнтернетПочтовыеАдреса (InternetMailAddresses)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Добавить адрес строкой

Синтаксис:

Добавить(<ПочтовыйАдрес>)

Параметры:

<ПочтовыйАдрес> (необязательный)

Тип: Строка.   
Почтовый адрес.

Описание варианта метода:

Добавляет полный почтовый адрес в коллекцию почтовых адресов.

Вариант синтаксиса: Добавить адрес парой (пользователь, сервер)

Синтаксис:

Добавить(<Пользователь>, <Сервер>)

Параметры:

<Пользователь> (необязательный)

Тип: Строка.   
Имя пользователя.

<Сервер> (необязательный)

Тип: Строка.   
Имя сервера.

Описание варианта метода:

Добавляет почтовый адрес в коллекцию почтовых адресов как пару (пользователь, сервер).

Возвращаемое значение:

Тип: [ИнтернетПочтовыйАдрес](ИнтернетПочтовыйАдрес.md).   

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
