Глобальный контекст (Global context)

УведомленияКлиента (ClientNotifications)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [МенеджерУведомленийКлиента](МенеджерУведомленийКлиента_(ClientNotificationManager).md).   
Предоставляет доступ к функциям уведомления клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
