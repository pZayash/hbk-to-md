ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ТипЗначения (ValueType)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов.md).   
Содержит объект, описывающий допустимые типы значений для ячеек области.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство доступно, если ячейка содержит значение (свойство СодержитЗначение равно Истина).  
Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.2.
