МеткаNDEF (TagNDEF)

МаксимальныйРазмер (MaxSize)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Число, Неопределено.   
Максимальный размер сообщения, которое может быть записано на метку (в байтах), или Неопределено, если размер неизвестен.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
