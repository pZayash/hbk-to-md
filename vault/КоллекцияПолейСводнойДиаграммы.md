# КоллекцияПолейСводнойДиаграммы

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Сводная диаграмма](objects__catalog63__catalog810.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы.md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной диаграммы или коллекция полей сводной диаграммы.  
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс (нумерация с 0) поля.
