# КоллекцияКолонокТаблицыЗначений.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Таблица значений](Таблица_значений.md) › [КоллекцияКолонокТаблицыЗначений](КоллекцияКолонокТаблицыЗначений_(ValueTableColumnCollection).md) › [Методы](КоллекцияКолонокТаблицыЗначений__Методы.md)

КоллекцияКолонокТаблицыЗначений (ValueTableColumnCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [КолонкаТаблицыЗначений](КолонкаТаблицыЗначений_(ValueTableColumn).md), [Строка](lang__def_String.md).   
Удаляемая колонка (индекс, колонка или имя колонки).

Описание:

Удаляет заданную колонку из коллекции колонок таблицы значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
