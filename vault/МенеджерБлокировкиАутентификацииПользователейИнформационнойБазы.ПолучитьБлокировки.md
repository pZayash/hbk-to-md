МенеджерБлокировкиАутентификацииПользователейИнформационнойБазы (InfoBaseUserAuthenticationLockManager)

ПолучитьБлокировки (GetLocks)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПолучитьБлокировки(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Структура может содержать поля "ИмяПользователя" и "РазделениеДанных".  
Поле "РазделениеДанных" представляет собой структуру, в которой ключом является наименование разделителя, а значением - значение разделителя.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив элементов типа [БлокировкаАутентификацииПользователяИнформационнойБазы](БлокировкаАутентификацииПользователяИнформационнойБазы.md).

Описание:

Получает блокировки аутентификации пользователей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.16.
