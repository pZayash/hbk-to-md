ФормаКлиентскогоПриложения (ClientApplicationForm)

Окно (Window)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md).   
Содержит окно клиентского приложения, в котором расположена форма.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме совместимости [Версия8\_3\_19](РежимСовместимости.Версия8_3_19_(CompatibilityMode.Version8_3_19).md) и ниже для формы на начальной странице в тонком и веб-клиенте содержит [Неопределено](lang__def_Undefined.md).  
В режиме совместимости выше [Версия8\_3\_19](РежимСовместимости.Версия8_3_19_(CompatibilityMode.Version8_3_19).md) для формы на начальной странице в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md) содержит объект типа {ссылка:Объекты; 1753; ОкноКлиентскогоПриложения}, у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница_(ClientApplicationWindow.HomePage).md) имеет значение [Истина](lang__def_BooleanTrue.md). И свойство [Содержимое](ОкноКлиентскогоПриложения.Содержимое_(ClientApplicationWindow.Content).md) содержит формы начальной страницы.  
Если активна начальная страница, то и данное свойство, и метод [АктивноеОкно](Глобальный_контекст.АктивноеОкно_(Global_context.ActiveWindow).md) для формы на начальной странице возвращают один и тот же объект.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
