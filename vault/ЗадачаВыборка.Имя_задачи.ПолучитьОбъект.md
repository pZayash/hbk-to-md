ЗадачаВыборка.<Имя задачи> (TaskSelection.<Task name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи.md)<[Имя задачи](ЗадачаОбъект.Имя_задачи.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает объект для чтения, изменения, добавления и удаления элемента, на котором в данный момент спозиционирована выборка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
