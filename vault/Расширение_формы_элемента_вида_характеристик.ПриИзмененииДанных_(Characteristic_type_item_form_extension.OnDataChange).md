# Расширение формы элемента вида характеристик.ПриИзмененииДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [Расширение формы элемента вида характеристик](Расширение_формы_элемента_вида_характеристик_(Characteristic_type_item_form_extension).md) › [События](Расширение_формы_элемента_вида_характеристик__События.md)

Расширение формы элемента вида характеристик (Characteristic type item form extension)

ПриИзмененииДанных (OnDataChange)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриИзмененииДанных()

Описание:

Возникает при инициализации формы, перечитывании данных, смене объекта основного реквизита.  
Предназначено для отображения данных объекта в элементах управления, которые не связаны непосредственно с данными, например, отображение картинки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
