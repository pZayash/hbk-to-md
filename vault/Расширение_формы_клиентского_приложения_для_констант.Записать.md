Расширение формы клиентского приложения для констант (Client application form extension for constants)

Записать (Write)

Доступен, начиная с версии 8.2.

Синтаксис:

Записать(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [Структура](Структура.md).   
Структура параметров записи, которые могут быть обработаны в обработчиках событий записи. Состав параметров произвольный.

Возвращаемое значение:

Тип: Булево.   
Истина - успешно записаны; Ложь - в противном случае.

Описание:

Записывает константы в базу данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
