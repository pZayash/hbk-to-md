АдминистрированиеЛицензия (AdministrationLicense)

СетевойКлюч (NetworkKey)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает статус использования аппаратного ключа защиты:

- Истина, если для аппаратной лицензии ключ защиты программы является сетевым, лицензия получена через менеджер лицензий Aladdin License Manager;
- Ложь в противном случае.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
