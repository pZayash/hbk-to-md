# ПостроительОтчета.Вывести

ПостроительОтчета (ReportBuilder)

Вывести (Put)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Вывод в табличный документ

Синтаксис:

Вывести(<ТабличныйДокумент>)

Параметры:

<ТабличныйДокумент> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Имя элемента управления "Табличный документ", в который будет осуществляться вывод. В случае, если параметр не указан, будет создан новый табличный документ.

Вариант синтаксиса: Вывод в диаграмму

Синтаксис:

Вывести(<Диаграмма>, <Ресурс>)

Параметры:

<Диаграмма> (обязательный)

Тип: [Диаграмма](Диаграмма_(Chart).md).   
Имя элемента управления "Диаграмма", в который будет осуществляться вывод.

<Ресурс> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя ресурса, который будет выдаваться в диаграмму.

Описание:

Выполняет и выводит отчет в табличный документ или диаграмму.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Вывод отчета происходит только в случае, если отчет еще не выводился или если были изменены настройки (см. [НастройкиПостроителяОтчета](НастройкиПостроителяОтчета_(ReportBuilderSettings).md)) или вызван метод [Выполнить](ПостроительОтчета.Выполнить_(ReportBuilder.Execute).md).

Пример:

|  |
| --- |
| // Вывод в табличный документ  ПостроительОтчета.Вывести(ДокументРезультат); // Вывод в диаграмму  Диаграмма.Очистить(); ПостроительОтчета.Вывести(Диаграмма, ИмяВыводимогоПоказателя); |

Использование в версии:

Доступен, начиная с версии 8.0.
