МенеджерОтображенияРекламы (AdRepresentationManager)

УстановитьИндентификаторВидеообъявленияСВознаграждением (SetRewardedVideoID)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьИндентификаторВидеообъявленияСВознаграждением(<ИдентификаторРекламногоБлока>)

Параметры:

<ИдентификаторРекламногоБлока> (обязательный)

Тип: [Строка](lang__def_String.md).   
Индентификатор видеообъявления.

Описание:

Устанавливает идентификатор видеообъявления, который будет использован при загрузке рекламы из сети Интернет.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.14.
