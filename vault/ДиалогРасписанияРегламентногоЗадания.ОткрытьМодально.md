ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

ОткрытьМодально (DoModal)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьМодально()

Возвращаемое значение:

Тип: Булево.   

Описание:

Открывает модально диалог расписания.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент.

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [Показать](ДиалогРасписанияРегламентногоЗадания.Показать.md).

См. также:

[ДиалогРасписанияРегламентногоЗадания](ДиалогРасписанияРегламентногоЗадания.md), метод [ОткрытьАсинх](ДиалогРасписанияРегламентногоЗадания.ОткрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.18.
