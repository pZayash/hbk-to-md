ТаблицаФормы (FormTable)

ОбработкаЗаписиНового (NewWriteProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаЗаписиНового(<НовыйОбъект>, <Источник>, <СтандартнаяОбработка>)

Параметры:

<НовыйОбъект>

Тип: Ссылка на объект информационной базы.   
Объект, который был добавлен.  
Если событие вызывается из формы выбора, то через этот параметр можно вернуть новый объект. В режиме выбора, если форма нового объекта закрыта, будет предложено немедленно выбрать переопределенный объект. Если отказаться от выбора, то на этот объект при возможности будет осуществленно позиционирование.  
Тип возвращаемого значения параметра должен соответствовать типу ключей строк таблицы.

<Источник>

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md).   
Форма - источник события.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.   
Стандартная обработка заключается в позиционировании на записанный элемент основной таблицы формы списка (если форма является формой списка типа записываемого объекта).  
Значение по умолчанию: Истина.

Описание:

Вызывается при добавлении объекта в одной из подчиненных форм или при вызове в одной из подчиненных форм метода формы [ОповеститьОЗаписиНового](ФормаКлиентскогоПриложения.ОповеститьОЗаписиНового.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.2.
