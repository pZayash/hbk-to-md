ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

Пароль (Password)

Доступен, начиная с версии 8.2.

Использование:

Только запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Содержит пароль, используемый при стандартной аутентификации.  
Если содержит [Неопределено](lang__def_Undefined.md), то получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
