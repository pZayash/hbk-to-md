КоллекцияСтраницPDF (PDFPagesCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [СтраницаPDF](СтраницаPDF_(PDFPage).md).   
Элемент, индекс которого требуется получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции. Если такого элемента нет, возвращается -1.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
