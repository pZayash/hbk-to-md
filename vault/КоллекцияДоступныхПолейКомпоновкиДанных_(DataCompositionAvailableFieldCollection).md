# КоллекцияДоступныхПолейКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Настройки компоновки данных](Настройки_компоновки_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных_(DataCompositionAvailableField).md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных_(DataCompositionFilterAvailableField).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля.  
Возможно обращение к доступному полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияДоступныхПолейКомпоновкиДанных.Индекс_(DataCompositionAvailableFieldCollection.IndexOf).md)  
[Количество (Count)](КоллекцияДоступныхПолейКомпоновкиДанных.Количество_(DataCompositionAvailableFieldCollection.Count).md)  
[Найти (Find)](КоллекцияДоступныхПолейКомпоновкиДанных.Найти_(DataCompositionAvailableFieldCollection.Find).md)  
[Получить (Get)](КоллекцияДоступныхПолейКомпоновкиДанных.Получить_(DataCompositionAvailableFieldCollection.Get).md)  

Описание:

Коллекция доступных полей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
