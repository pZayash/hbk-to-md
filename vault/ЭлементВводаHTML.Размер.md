ЭлементВводаHTML (HTMLInputElement)

Размер (Size)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Свойство устанавливает ширину поля ввода.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Внимание! Свойство не устанавливает количество символов, которое может быть введено в элементе ввода.

Использование в версии:

Доступен, начиная с версии 8.2.
