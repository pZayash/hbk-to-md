Расширение поля формы для поля флажка (Form field extension for a check box field)

ШиринаЭлемента (ItemWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет ширину элемента тумблера. 0 - определять автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Используется при виде флажка [Тумблер](ВидФлажка.Тумблер.md).  
Не используется, если итоговое значение свойства [ОдинаковаяШиринаЭлементов](Расширение_поля_формы_для_поля_флажка.ОдинаковаяШиринаЭлементов.md) Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.2.
