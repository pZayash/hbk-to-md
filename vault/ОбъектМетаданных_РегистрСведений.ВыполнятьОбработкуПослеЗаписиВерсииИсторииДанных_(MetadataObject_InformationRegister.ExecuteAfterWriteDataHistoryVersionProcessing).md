ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если имеет значение [Истина](lang__def_BooleanTrue.md), то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](РегистрСведенийМенеджер.Имя_регистра_сведений.ОбработкаПослеЗаписиВерсийИсторииДанных_(InformationRegisterManager.Information_register_name.AfterWriteDataHistoryVersionsProcessing).md) можно обработать запись версии после обновления истории.  
При записи данных значение, указанное в конфигураторе, устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_РегистрСведений.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных_(MetadataObject_InformationRegister.ExecuteAfterWriteDataHistoryVersionProcessing).md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
