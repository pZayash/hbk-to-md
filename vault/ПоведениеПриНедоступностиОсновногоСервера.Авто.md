ПоведениеПриНедоступностиОсновногоСервера (OnMainServerUnavalableBehavior)

Авто (Auto)

Доступен, начиная с версии 8.3.16.

Описание:

При недоступности основного сервера, если форма была открыта с автономного сервера, элементы формы, связанные с внешними ссылками, становятся недоступными.  
Остальные элементы формы остаются доступными.

Использование в версии:

Доступен, начиная с версии 8.3.16.
