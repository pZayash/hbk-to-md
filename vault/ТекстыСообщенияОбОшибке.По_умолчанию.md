ТекстыСообщенияОбОшибке (ErrorMessageTexts)

По умолчанию

Доступен, начиная с версии 8.3.17.

Синтаксис:

Новый ТекстыСообщенияОбОшибке(<ТекстДляОшибкиНеТребующейПерезапуск>, <ТекстДляОшибкиТребующейПерезапуск>)

Параметры:

<ТекстДляОшибкиНеТребующейПерезапуск> (обязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Шаблон сообщения об ошибке, которое отобразится в форме информации об ошибке, если ошибка не требует завершения работы.

<ТекстДляОшибкиТребующейПерезапуск> (обязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Шаблон сообщения об ошибке, которое отобразится в форме информации об ошибке, если ошибка требует завершения работы.

Описание:

Создает новый объект типа [ТекстыСообщенияОбОшибке](ТекстыСообщенияОбОшибке_(ErrorMessageTexts).md), инициализируя одноименные свойства объекта переданными параметрами.

Использование в версии:

Доступен, начиная с версии 8.3.17.
