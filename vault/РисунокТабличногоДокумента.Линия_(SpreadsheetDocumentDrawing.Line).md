РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Линия (Line)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Линия](Линия_(Line).md).   
Содержит тип линии рисунка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не применимо для рисунков типа [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
