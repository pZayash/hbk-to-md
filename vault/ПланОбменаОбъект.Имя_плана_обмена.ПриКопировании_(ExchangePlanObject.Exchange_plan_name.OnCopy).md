ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПриКопировании (OnCopy)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриКопировании(<ОбъектКопирования>)

Параметры:

<ОбъектКопирования>

Тип: [ПланОбменаОбъект.\<Имя плана обмена\>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md).   
Исходный элемент, который является источником копирования.

Описание:

Возникает при создании узла копированием.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
