РежимСовместимости (CompatibilityMode)

Версия8\_3\_20 (Version8\_3\_20)

Доступен, начиная с версии 8.3.21.

Описание:

Усовершенствования версии 8.3.21 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.20.

Использование в версии:

Доступен, начиная с версии 8.3.21.
