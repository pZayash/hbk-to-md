ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

СодержитЗначение (ContainsValue)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что ячейки области содержат значения.  
[Истина](lang__def_BooleanTrue.md) - содержат значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для прямоугольных областей.

Использование в версии:

Доступен, начиная с версии 8.2.
