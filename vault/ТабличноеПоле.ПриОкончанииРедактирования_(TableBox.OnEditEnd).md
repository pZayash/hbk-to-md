ТабличноеПоле (TableBox)

ПриОкончанииРедактирования (OnEditEnd)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриОкончанииРедактирования(<НоваяСтрока>, <ОтменаРедактирования>)

Параметры:

<НоваяСтрока>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак редактирования новой строки. Имеет значение [Истина](lang__def_BooleanTrue.md), если строка была добавлена или скопирована.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если произошла отмена редактирования.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает после окончания редактирования строки табличного поля.

Доступность:

Толстый клиент.

Примечание:

Возникает, если редактирование производится непосредственно в табличном поле.

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [ПриНачалеРедактирования](ТабличноеПоле.ПриНачалеРедактирования_(TableBox.OnStartEdit).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
