АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

РолиОграничивающиеРасширениеПравДоступа (AccessRightsExtensionLimitingRoles)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Роли, ограничивающие расширение прав доступа. Перечень ролей, разделенных символом ";".  
При изменении списка ролей изменения в составе ролей учитываются только для новых сеансов и после перезапуска текущих сеансов.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
