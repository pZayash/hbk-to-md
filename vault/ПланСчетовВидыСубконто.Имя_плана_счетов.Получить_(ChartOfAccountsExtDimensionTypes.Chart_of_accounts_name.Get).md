Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс строки.

Возвращаемое значение:

Тип: [ПланСчетовВидыСубконтоСтрока.\<Имя плана счетов\>](ПланСчетовВидыСубконтоСтрока.Имя_плана_счетов_(ChartOfAccountsExtDimensionTypesRow.Chart_of_accounts_name).md).   

Описание:

Получает значение по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
