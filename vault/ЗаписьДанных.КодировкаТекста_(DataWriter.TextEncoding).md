ЗаписьДанных (DataWriter)

КодировкаТекста (TextEncoding)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста по умолчанию для данного экземпляра [ЗаписьДанных](ЗаписьДанных_(DataWriter).md).  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

- UTF-16BE

- HZ-GB-2312

.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
