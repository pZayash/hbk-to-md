# СредстваМультимедиа.СделатьВидеозапись

СредстваМультимедиа (MultimediaTools)

СделатьВидеозапись (MakeVideoRecording)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СделатьВидеозапись(<ТипКамеры>, <Качество>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства_(DeviceCameraType).md).   
Показывает, какой камерой будет сделана видеозапись.

<Качество> (необязательный)

Тип: [КачествоВидеозаписи](КачествоВидеозаписи_(VideoQuality).md).   
Определяет качество видеозаписи.  
В устройствах под управлением iOS параметр игнорируется.

Возвращаемое значение:

Тип: [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - запись не выполнена.

Описание:

Выполняет видеозапись встроенной фото-видео камерой.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяВидеозапись](СредстваМультимедиа.ПоддерживаетсяВидеозапись_(MultimediaTools.VideoRecordingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ПоддерживаетсяВидеозапись](СредстваМультимедиа.ПоддерживаетсяВидеозапись_(MultimediaTools.VideoRecordingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.
