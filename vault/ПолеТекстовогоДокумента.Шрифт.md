ПолеТекстовогоДокумента (TextDocumentField)

Шрифт (Font)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт.md).   
Содержит шрифт, используемый в поле текстового документа.  
Если шрифт не указан явно (значение [АвтоШрифт](ВидШрифта.АвтоШрифт.md)), он выбирается из расширения поля текстового документа.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
