ОформлениеЯчейки (CellAppearance)

ЗначениеФлажка (CheckValue)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число, Булево.   
Содержит значение флажка, отображаемого в колонке. Если колонка, отображающая данное значение связана с данными, то данное свойство доступно только для чтения.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
