# КонстантыМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КонстантаМенеджер.\<Имя константы\>](КонстантаМенеджер.Имя_константы_(ConstantManager.Constant_Name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры констант.  
Возможно обращение к менеджеру константы посредством оператора [...]. В качестве аргумента передается имя константы, как оно задано в конфигурации.  

Свойства:

[\<Имя константы\> (\<Constant name\>)](КонстантыМенеджер.Имя_константы_(ConstantsManager.Constant_name).md)  

Методы:

[ПолучитьФорму (GetForm)](КонстантыМенеджер.ПолучитьФорму_(ConstantsManager.GetForm).md)  
[СоздатьНабор (CreateSet)](КонстантыМенеджер.СоздатьНабор_(ConstantsManager.CreateSet).md)  

Описание:

Применяется в виде свойства [Константы](Глобальный_контекст.Константы_(Global_context.Constants).md) глобального контекста для предоставления доступа к константам.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| Константы.ПолучитьФорму().ОткрытьМодально(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Константы](Глобальный_контекст.Константы_(Global_context.Constants).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
