РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать()

Описание:

Считывает записи из базы данных по установленному отбору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Регистр = РегистрыРасчета.ОсновныеНачисленияСотрудниковОрг; НаборЗаписей = Регистр.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.Регистратор.Значение = Документ.Регистратор; НаборЗаписей.Прочитать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
