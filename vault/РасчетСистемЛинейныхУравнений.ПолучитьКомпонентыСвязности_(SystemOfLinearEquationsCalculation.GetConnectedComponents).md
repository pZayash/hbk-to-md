# РасчетСистемЛинейныхУравнений.ПолучитьКомпонентыСвязности

РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ПолучитьКомпонентыСвязности (GetConnectedComponents)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьКомпонентыСвязности(<СпособПолученияКомпонент>)

Параметры:

<СпособПолученияКомпонент> (необязательный)

Тип: [СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений_(ConnectedComponentsOfSystemOfLinearEquationsCalculationGettingMethod).md).   
Определяет способ получения компонент связности.  
В режиме совместимости [Версия8\_3\_14](РежимСовместимости.Версия8_3_14_(CompatibilityMode.Version8_3_14).md) значение по умолчанию – [КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент_(ConnectedComponentsOfSystemOfLinearEquationsCalculationGettingMethod.StronglyConnectedComponentsWithNoInnerConnectionRequired).md).  
Значение по умолчанию: [КомпонентыСлабойСвязности](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.КомпонентыСлабойСвязности_(ConnectedComponentsOfSystemOfLinearEquationsCalculationGettingMethod.WeaklyConnectedComponents).md).

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   
Таблица значений с двумя колонками: в первой колонке содержится номер узла, а во второй - номер компоненты, которой принадлежит узел.

Описание:

Возвращает компоненты связности.   
Для корректной работы метода требуется:

- установить значения свойств [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей_(SystemOfLinearEquationsCalculation.RelationDataSource).md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях_(SystemOfLinearEquationsCalculation.EquationsInRelationsColumn).md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях_(SystemOfLinearEquationsCalculation.VariablesInRelationsColumn).md),
- в [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений_(SystemDescriptions).md) добавлен объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md), в котором указана только [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях_(SystemOfLinearEquationsDescription.FactorsInRelationsColumn).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
