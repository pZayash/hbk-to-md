ОформлениеЯчейки (CellAppearance)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Истина - отображает значение ячейки табличного поля как гиперссылку. При выборе ячейки с установленным свойством будет порождаться событие [Выбор](ТабличноеПоле.Выбор.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
