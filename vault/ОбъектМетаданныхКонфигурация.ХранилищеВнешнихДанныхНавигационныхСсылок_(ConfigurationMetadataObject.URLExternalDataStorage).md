ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ХранилищеВнешнихДанныхНавигационныхСсылок (URLExternalDataStorage)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md).   
Содержит хранилище настроек, в котором будут сохраняться внешние данные ссылок.  
Если хранилище не установлено:

- Если режим совместимости не используется, используется стандартное хранилище.
- В режиме совместимости [Версия8\_3\_18](РежимСовместимости.Версия8_3_18_(CompatibilityMode.Version8_3_18).md):
  - при обращении из встроенного языка выдается ошибка,
  - формы отчета и динамического списка не предоставляют возможность получать ссылки с полными настройками.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.

Описание изменено в версии 8.3.26.
