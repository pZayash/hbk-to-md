НастройкаПериода (PeriodSettings)

ДатаОкончания (EndDate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату окончания периода.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если свойство [ВариантОкончания](НастройкаПериода.ВариантОкончания_(PeriodSettings.EndingVariant).md) имеет значение [КонкретнаяДата](ВариантГраницыИнтервала.КонкретнаяДата_(IntervalBoundVariant.SpecificDate).md).

Использование в версии:

Доступен, начиная с версии 8.0.
