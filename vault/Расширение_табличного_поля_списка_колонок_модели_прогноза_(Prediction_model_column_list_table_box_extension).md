Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для колонок модели прогноза. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [КолонкаМоделиПрогноза](КолонкаМоделиПрогноза_(PredictionModelColumn).md).
