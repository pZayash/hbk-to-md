# ЭлементыZipФайла

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [ZIP](ZIP-2.md)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [ЭлементыФайлаАрхива](ЭлементыФайлаАрхива_(ArchiveFileEntries).md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементZipФайла](ЭлементZipФайла_(ZipFileEntry).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элемент коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Методы:

[Количество (Count)](ЭлементыZipФайла.Количество_(ZipFileEntries.Count).md)  
[Найти (Find)](ЭлементыZipФайла.Найти_(ZipFileEntries.Find).md)  
[Получить (Get)](ЭлементыZipФайла.Получить_(ZipFileEntries.Get).md)  

Описание:

Содержит коллекцию элементов ZIP файла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЧтениеZipФайла](ЧтениеZipФайла_(ZipFileReader).md), свойство [Элементы](ЧтениеZipФайла.Элементы_(ZipFileReader.Items).md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.

Описание изменено в версии 8.3.26.
