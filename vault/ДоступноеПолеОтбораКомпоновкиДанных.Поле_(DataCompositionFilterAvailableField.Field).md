ДоступноеПолеОтбораКомпоновкиДанных (DataCompositionFilterAvailableField)

Поле (Field)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md).   
Поле, доступное для использования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для поиска нужного поля в доступных полях (перебором) данное свойство использовать нельзя, т.к. доступное поле может быть многоязыковым, но свойство всегда возвращают один и тот же синоним. Для поиска нужно использовать метод [НайтиПоле](ДоступныеПоляКомпоновкиДанных.НайтиПоле_(DataCompositionAvailableFields.FindField).md).

Использование в версии:

Доступен, начиная с версии 8.1.
