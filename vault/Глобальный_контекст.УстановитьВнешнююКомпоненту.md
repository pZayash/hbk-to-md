Глобальный контекст (Global context)

УстановитьВнешнююКомпоненту (InstallAddIn)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [НачатьУстановкуВнешнейКомпоненты](Глобальный_контекст.НачатьУстановкуВнешнейКомпоненты.md)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьВнешнююКомпоненту(<Местоположение>)

Параметры:

<Местоположение> (обязательный)

Тип: Строка.   
Строка, определяющая местоположение внешнего компонента.  
В качестве строки может использоваться:

- полное имя макета, в котором хранится внешний компонент в архиве;
- URL к внешнему компоненту, упакованному в ZIP-архив, в формате, аналогичном [ПолучитьНавигационнуюСсылку](Глобальный_контекст.ПолучитьНавигационнуюСсылку.md)().

Описание:

Доставляет объект внешнего компонента с сервера на клиент, после чего он становится доступен для метода [ПодключитьВнешнююКомпоненту](Глобальный_контекст.ПодключитьВнешнююКомпоненту.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Метод работает только с компонентами, хранящимися в архиве.  
Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [НачатьУстановкуВнешнейКомпоненты](Глобальный_контекст.НачатьУстановкуВнешнейКомпоненты.md).
