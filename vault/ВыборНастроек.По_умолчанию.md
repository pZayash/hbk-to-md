ВыборНастроек (SettingsChoice)

На основании ключа

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ВыборНастроек(<КлючНастройки>)

Параметры:

<КлючНастройки> (необязательный)

Тип: Произвольный.   
Ключ настройки, устанавливаемый объекту.

Описание:

Создает объект с ключем настройки.  
Также может быть создан с использованием функциональной формы оператора Новый.

Использование в версии:

Доступен, начиная с версии 8.2.
