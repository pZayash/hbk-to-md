Глобальный контекст (Global context)

ПолучитьПараметрыМонопольногоРежима (GetExclusiveModeParameters)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПолучитьПараметрыМонопольногоРежима()

Возвращаемое значение:

Тип: [ПараметрыМонопольногоРежима](ПараметрыМонопольногоРежима.md), Неопределено.   
Неопределено - если для текущей области данных не установлен монопольный режим.  
[ПараметрыМонопольногоРежима](ПараметрыМонопольногоРежима.md) - если для текущей области данных установлен монопольный режим.

Описание:

Если монопольный режим установлен, то получает параметры монопольного режима.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
