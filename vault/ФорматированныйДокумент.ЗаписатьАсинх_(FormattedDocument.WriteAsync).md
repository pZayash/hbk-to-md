ФорматированныйДокумент (FormattedDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет записан форматированный документ.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента_(FormattedDocumentFileType).md).   
Указывает тип файла.

Описание варианта метода:

Запись форматированного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи документа.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента_(FormattedDocumentFileType).md).   
Указывает формат записи документа.

Описание варианта метода:

Запись форматированного документа в поток.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания:

- [Истина](lang__def_BooleanTrue.md) - если запись была инициирована с помощью расширения браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
- [Ложь](lang__def_BooleanFalse.md) - если запись была инициирована с помощью расширения браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
- [Неопределено](lang__def_Undefined.md) - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

.

Описание:

Асинхронная функция. Сохраняет форматированный документ в файл.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
