Расширение формы клиентского приложения для отчета (Client application form extension for reports)

КлючПользовательскихНастроек (UserSettingsKey)

Доступен, начиная с версии 8.2.

Описание:

Тип: Произвольный.   
Содержит ключ пользовательских настроек, которые нужно использовать для отчета.   
Если не установлен параметр формы [ПользовательскиеНастройки](Расширение_формы_клиентского_приложения_для_отчета.ПользовательскиеНастройки.md), то будут загружены пользовательские настройки по указанному ключу.  
Если настройки, заданные ключом, не найдены, используются настройки варианта по умолчанию.

Использование в версии:

Доступен, начиная с версии 8.2.
