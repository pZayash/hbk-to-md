КоллекцияФасетовXDTO (XDTOFacetCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По виду фасета

Синтаксис:

Получить(<ВидФасета>)

Параметры:

<ВидФасета> (обязательный)

Тип: [ВидФасетаXDTO](ВидФасетаXDTO_(XDTOFacetType).md).   
Вид получаемого фасета XDTO.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс фасета в коллекции.

Возвращаемое значение:

Тип: [ФасетXDTO](ФасетXDTO_(XDTOFacet).md).   

Описание:

Получает фасет коллекции.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
