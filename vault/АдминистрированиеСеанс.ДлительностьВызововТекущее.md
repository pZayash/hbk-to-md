АдминистрированиеСеанс (AdministrationSession)

ДлительностьВызововТекущее (CallDurationCurrent)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит интервал времени, прошедший с момента начала обращения, в случае, если сеанс выполняет обращение к серверу "1С:Предприятия", в миллисекундах. Иначе – 0.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
