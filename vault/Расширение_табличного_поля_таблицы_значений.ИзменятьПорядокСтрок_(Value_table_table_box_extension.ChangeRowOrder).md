Расширение табличного поля таблицы значений (Value table table box extension)

ИзменятьПорядокСтрок (ChangeRowOrder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность перемещения строк и изменение упорядочивания таблицы значений.  
[Истина](lang__def_BooleanTrue.md) - менять разрешено.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
