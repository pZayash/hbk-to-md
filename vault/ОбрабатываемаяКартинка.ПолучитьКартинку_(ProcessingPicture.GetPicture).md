ОбрабатываемаяКартинка (ProcessingPicture)

ПолучитьКартинку (GetPicture)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьКартинку()

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md).   
Если в результате получается картинка векторного формата (WMF, EMF, SVG), будет выдано сообщение об ошибке.  
На мобильной платформе поддерживаются только форматы JPEG и PNG. Если в результате получается картинка, не поддерживаемого формата, будет выдано сообщение об ошибке.

Описание:

Создает новую картинку, на основе исходной картинки и выполненных для нее преобразований.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.19.
