Глобальный контекст (Global context)

НачатьПолучениеИнформацииОСетевыхАдаптерах (BeginGettingNetworkAdaptersInformation)

Доступен, начиная с версии 8.3.18.

Синтаксис:

НачатьПолучениеИнформацииОСетевыхАдаптерах(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения получения информации со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив объектов типа [ИнформацияОСетевомАдаптере](ИнформацияОСетевомАдаптере.md).

Описание:

Процедура заполняет массив объектов типа [ИнформацияОСетевомАдаптере](ИнформацияОСетевомАдаптере.md) и передает в функцию, указанную в обработчике <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьИнформациюОСетевыхАдаптерахАсинх](Глобальный_контекст.ПолучитьИнформациюОСетевыхАдаптерахАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
