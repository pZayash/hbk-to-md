Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиПоИспользуемомуСоставуДанных(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [ЭлементСоставаТабличногоПространстваБазыДанных](ЭлементСоставаТабличногоПространстваБазыДанных.md), ОбъектМетаданных.   
Объект метаданных, для которого выполняется поиск табличного пространства.

Возвращаемое значение:

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных.md).   

Описание:

Возвращает табличное пространство, в котором физически хранятся данные таблиц объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
