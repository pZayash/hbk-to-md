ДоступныеПоляКомпоновкиДанных (DataCompositionAvailableFields)

ПолучитьИдентификаторПоОбъекту (GetIDByObject)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИдентификаторПоОбъекту(<Объект>)

Параметры:

<Объект> (необязательный)

Тип: [ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных_(DataCompositionAvailableField).md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных_(DataCompositionFilterAvailableField).md).   
Объект, для которого требуется получить идентификатор.

Возвращаемое значение:

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - переданный объект не является частью данного объекта.

Описание:

Получает идентификатор для оформления компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
