Панель (Panel)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, отображаемую на панели в качестве фона. Многостраничная панель использует одну картинку для всех своих страниц. Картинка не отображается на закладках - переключателях страниц.

Доступность:

Толстый клиент.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
