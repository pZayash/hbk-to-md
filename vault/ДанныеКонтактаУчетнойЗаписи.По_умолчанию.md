ДанныеКонтактаУчетнойЗаписи (AccountContactData)

По умолчанию

Доступен, начиная с версии 8.3.4.

Синтаксис:

Новый ДанныеКонтактаУчетнойЗаписи(<ДанныеКонтакта>, <УчетнаяЗапись>, <ГлобальныйКлючКонтакта>)

Параметры:

<ДанныеКонтакта> (обязательный)

Тип: [ДанныеКонтакта](ДанныеКонтакта.md).   
Данные контакта.

<УчетнаяЗапись> (обязательный)

Тип: [УчетнаяЗаписьКонтактов](УчетнаяЗаписьКонтактов.md).   
Учетная запись, к которой привязан контакт.

<ГлобальныйКлючКонтакта> (необязательный)

Тип: Строка.   
Глобальный ключ контакта, идентифицирующий его во внешнем сервисе хранения контактов.

Описание:

Создает данные контакта учетной записи.

Использование в версии:

Доступен, начиная с версии 8.3.4.
