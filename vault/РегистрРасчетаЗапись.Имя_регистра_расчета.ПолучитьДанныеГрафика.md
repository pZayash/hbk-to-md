РегистрРасчетаЗапись.<Имя регистра расчета> (CalculationRegisterRecord.<Calculation register name>)

ПолучитьДанныеГрафика (GetScheduleData)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДанныеГрафика(<ВидПериода>)

Параметры:

<ВидПериода> (обязательный)

Тип: [ВидПериодаРегистраРасчета](ВидПериодаРегистраРасчета.md).   
Вид периода регистра расчета.  
Если [ПериодДействия](ВидПериодаРегистраРасчета.ПериодДействия.md), возвращаются данные календаря, собранные по фактическому периоду действия.  
Если [БазовыйПериод](ВидПериодаРегистраРасчета.БазовыйПериод.md) - данные по базовому периоду.  
Если [ПериодРегистрации](ВидПериодаРегистраРасчета.ПериодРегистрации.md) - по данные календаря по периоду регистрации. Если, например, регистр расчета имеет месячную периодичность, то суммируются данные календаря за весь месяц, в котором зарегистрирована запись регистра расчета.  
Если выбран [ФактическийПериодДействия](ВидПериодаРегистраРасчета.ФактическийПериодДействия.md), то приводит к вычислению данных графика по фактическому периоду действия записей (записи) регистра расчета.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   

Описание:

Получает данные календаря для движений по заданному регистратору в виде таблицы значений. Таблица содержит одну колонку c полученными данными календаря.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
