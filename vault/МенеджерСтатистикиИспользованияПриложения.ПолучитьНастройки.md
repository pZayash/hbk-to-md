МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьНастройки (GetSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьНастройки()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает текст настроек событий/экранов статистики использования приложения, установленный с помощью метода [УстановитьНастройки](МенеджерСтатистикиИспользованияПриложения.УстановитьНастройки.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МенеджерСтатистикиИспользованияПриложения](МенеджерСтатистикиИспользованияПриложения.md), метод [ПолучитьНастройки](МенеджерСтатистикиИспользованияПриложения.ПолучитьНастройки.md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.
