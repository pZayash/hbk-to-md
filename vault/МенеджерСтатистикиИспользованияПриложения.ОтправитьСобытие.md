МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ОтправитьСобытие (SendEvent)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ОтправитьСобытие(<Событие>, <Параметр1>, <Параметр2>)

Параметры:

<Событие> (обязательный)

Тип: Строка.   
Имя отправляемого события.

<Параметр1> (необязательный)

Тип: Строка.   
Дополнительный параметр вызова.

<Параметр2> (необязательный)

Тип: Строка.   
Дополнительный параметр вызова.

Описание:

Передает отправляемое нестандартное событие в текущий сервис статистики без порождения исходного в обход настроек регистрации событий/экранов.  
Внимание! Не рекомендуется использовать данный метод в тиражных конфигурациях, так как он не управляется механизмом настройки регистрации статистики использования приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
