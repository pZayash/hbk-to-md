Информационная база (IInfoBaseInfo)

dbPassword (dbPassword)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Пароль пользователя сервера баз данных.

Доступность:

Интеграция.

Примечание:

Если объект получен методом [GetInfoBases](Соединение_с_рабочим_процессом.GetInfoBases.md), то результат чтения отличается от настоящего пароля пользователя. Пароль пользователя базы данных для зарегистрированной информационной базы прочитать нельзя.

Использование в версии:

Доступен, начиная с версии 8.1.
