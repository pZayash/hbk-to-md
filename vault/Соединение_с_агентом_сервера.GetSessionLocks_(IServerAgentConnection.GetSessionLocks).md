Соединение с агентом сервера (IServerAgentConnection)

GetSessionLocks (GetSessionLocks)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessionLocks(<Кластер>, <Сеанс>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер серверов, из которого должны быть получены блокировки. Должен быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера_(IServerAgentConnection).md).

<Сеанс> (обязательный)

Тип: [Сеанс](Сеанс_(ISessionInfo).md).   
Сеанс, от имени которого установлены блокировки объектов. Должен быть получен методом [GetSessions](Соединение_с_агентом_сервера.GetSessions_(IServerAgentConnection.GetSessions).md) или [GetInfoBaseSessions](Соединение_с_агентом_сервера.GetInfoBaseSessions_(IServerAgentConnection.GetInfoBaseSessions).md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера_(IServerAgentConnection).md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив элементов [Блокировка](Блокировка_(IObjectLock).md).

Описание:

Получает объектные блокировки кластера, установленные от имени заданного сеанса.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
