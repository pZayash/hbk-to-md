Расширение поля ввода перечисления (Enum text box extension)

БыстрыйВыбор (QuickChoice)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Предоставляет доступ к управлению режимом выбора значения. Если свойство установлено в Истина, то выбор будет выполняться из выпадающего списка.  
Данное свойство имеет более низкий приоритет, чем свойство "Способ выбора" объекта метаданных Перечисления и оказывает влияние на способ выбора только в том случае, если режим выбора перечисления равен "Обоими способами".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
