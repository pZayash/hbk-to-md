Глобальный контекст (Global context)

ПолучитьВремяЗавершенияСеансаПриБездействии (GetInactivityTimeForTerminateSession)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьВремяЗавершенияСеансаПриБездействии()

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает интервал времени (в секундах), по истечении которого будет завершен сеанс, в котором не наблюдалась активность пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
