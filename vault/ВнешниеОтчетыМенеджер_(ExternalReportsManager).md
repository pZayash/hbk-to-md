Описание:

Предназначен для управления внешними отчетами и позволяет получать значения типа [ВнешнийОтчет](ВнешнийОтчет_(ExternalReport).md) при помощи метода [Создать](ВнешниеОтчетыМенеджер.Создать_(ExternalReportsManager.Create).md). Доступ к объекту осуществляется через свойство глобального контекста [ВнешниеОтчеты](Глобальный_контекст.ВнешниеОтчеты_(Global_context.ExternalReports).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ВнешниеОтчеты](Глобальный_контекст.ВнешниеОтчеты_(Global_context.ExternalReports).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.23.
