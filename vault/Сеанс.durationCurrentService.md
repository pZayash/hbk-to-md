Сеанс (ISessionInfo)

durationCurrentService (durationCurrentService)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит время, в течение которого сеанс выполняет текущий вызов сервиса кластера.   
0, если в данный момент вызов сервиса кластера не выполняется, в миллисекундах.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.12.
