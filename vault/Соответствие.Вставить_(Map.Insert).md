Соответствие (Map)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Ключ>, <Значение>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ устанавливаемого элемента.

<Значение> (необязательный)

Тип: Произвольный.   
Значение устанавливаемого элемента. Если не указан, будет установлено [Неопределено](lang__def_Undefined.md).

Описание:

Устанавливает значение элемента соответствия по ключу. Если элемент с переданным значением ключа существует, то его значение заменяется, в противном случае добавляется новый элемент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ПараметрыОбработки = Новый Соответствие; ПараметрыОбработки.Вставить("Дата начала", ДатаНач); ПараметрыОбработки.Вставить("Дата конца", ДатаКон); |

Использование в версии:

Доступен, начиная с версии 8.0.
