ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

Начало (Begin)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Начало интервала.  
Допускаются следующие типы значений:

- для информационного интервала в значениях серий диаграммы (свойство диаграммы [ИнформационныеИнтервалыЗначений](Диаграмма.ИнформационныеИнтервалыЗначений_(Chart.ValuesReferenceBands).md)) - [Число](lang__def_Number.md);
- для информационного интервала в значениях точек диаграммы (свойство диаграммы [ИнформационныеИнтервалыТочек](Диаграмма.ИнформационныеИнтервалыТочек_(Chart.PointsReferenceBands).md)):
  - если на оси точек откладываются точки - [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md);
  - если на оси точек откладываются числа (например, в точечной и пузырьковой диаграммах) - [Число](lang__def_Number.md);
  - если на оси точек откладываются даты (например, в точечной и пузырьковой диаграмме) - [Дата](lang__def_Date.md);
- для информационного интервала, расположенного в параметрах вывода компоновки данных:
  - [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md);
  - для информационного интервала значений - [Число](lang__def_Number.md);
  - для информационного интервала точек - тип, установленный в свойстве [Точки](ДиаграммаКомпоновкиДанных.Точки_(DataCompositionChart.Points).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
