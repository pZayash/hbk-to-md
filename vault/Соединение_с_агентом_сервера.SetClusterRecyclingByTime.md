Соединение с агентом сервера (IServerAgentConnection)

SetClusterRecyclingByTime (SetClusterRecyclingByTime)

Доступен, начиная с версии 8.2.

Синтаксис:

SetClusterRecyclingByTime(<Кластер>, <ПериодПерезапуска>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого требуется изменить настройку автоматического перезапуска рабочих процессов. Должен быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md) объекта [Соединение с агентом сервера](Соединение_с_агентом_сервера.md).

<ПериодПерезапуска> (обязательный)

Тип: Число.   
Период перезапуска рабочих процессов в секундах.   
0 - периодический перезапуск не выполняется.

Описание:

Устанавливает периодический перезапуск рабочих процессов. Если после запуска рабочего процесса прошло время, превышающее значение, указанное в параметре <ПериодПерезапуска>, то рабочий процесс будет автоматически перезапущен.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
