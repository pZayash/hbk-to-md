# ПеречисленияМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Перечисления](Перечисления.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПеречислениеМенеджер.\<Имя перечисления\>](ПеречислениеМенеджер.Имя_перечисления.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры перечислений.  
Возможно обращение к менеджеру перечисления посредством оператора [...]. В качестве аргумента передается имя перечисления, как оно задано в конфигурации.  

Свойства:

[\<Имя перечисления\> (\<Enumeration name\>)](ПеречисленияМенеджер.Имя_перечисления.md)  

Методы:

[ТипВсеСсылки (AllRefsType)](ПеречисленияМенеджер.ТипВсеСсылки.md)  

Описание:

Предназначен для управления перечислениями и предоставляет доступ к значениям типа [ПеречислениеМенеджер.](ПеречислениеМенеджер.Имя_перечисления.md)<[Имя перечисления](ПеречислениеМенеджер.Имя_перечисления.md)>. Доступ к объекту осуществляется через свойство глобального контекста [Перечисления](Глобальный_контекст.Перечисления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| ВидКонтрагента = Перечисления.ВидыКонтрагентов.Организация; |

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Перечисления](Глобальный_контекст.Перечисления.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
