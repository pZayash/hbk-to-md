Диаграмма (Chart)

ПорядокОтображенияТочекГоризонтальнойГистограммы (BarChartPointsOrder)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [ПорядокОтображенияТочекГоризонтальнойГистограммы](ПорядокОтображенияТочекГоризонтальнойГистограммы.md).   
Определяет порядок вывода точек в горизонтальных диаграммах.  
Используется только для диаграмм следующих видов:

- [ГистограммаГоризонтальная](ТипДиаграммы.ГистограммаГоризонтальная.md),
- [ГистограммаГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаГоризонтальнаяОбъемная.md),
- [ГистограммаНормированнаяГоризонтальная](ТипДиаграммы.ГистограммаНормированнаяГоризонтальная.md),
- [ГистограммаНормированнаяГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаНормированнаяГоризонтальнаяОбъемная.md),
- [ГистограммаСНакоплениемГоризонтальная](ТипДиаграммы.ГистограммаСНакоплениемГоризонтальная.md),
- [ГистограммаСНакоплениемГоризонтальнаяОбъемная](ТипДиаграммы.ГистограммаСНакоплениемГоризонтальнаяОбъемная.md)

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
