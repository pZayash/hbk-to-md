ТаблицаФормы (FormTable)

Заголовок (Title)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Заголовок группы формы. Пустая строка означает автоматическое формирование. Для того чтобы не располагать в группе заголовок, следует установить свойство [ОтображатьЗаголовок](Расширение_группы_формы_для_обычной_группы.ОтображатьЗаголовок.md) в Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
