ЧтениеJSON (JSONReader)

УстановитьСтроку (SetString)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьСтроку(<СтрокаJSON>)

Параметры:

<СтрокаJSON> (обязательный)

Тип: Строка.   
Строка, содержащая текст в формате JSON.

Описание:

Устанавливает строку, содержащую текст JSON для чтения данным объектом. Если перед вызовом данного метода уже производилось чтение JSON из другого файла или строки, то чтение прекращается и объект инициализируется для чтения из указанной строки.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЧтениеJSON = Новый ЧтениеJSON; ЧтениеJSON.УстановитьСтроку("{}"); |

Использование в версии:

Доступен, начиная с версии 8.3.6.
