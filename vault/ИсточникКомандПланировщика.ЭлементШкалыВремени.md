ИсточникКомандПланировщика (PlannerCommandSource)

ЭлементШкалыВремени (TimeScaleItem)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ЭлементШкалыВремени](ИсточникКомандПоляПланировщика.ЭлементШкалыВремени.md)

Доступен, начиная с версии 8.3.16.

Описание:

Тип: [ИсточникКомандПланировщика](ИсточникКомандПоляПланировщика.md).   
Значение, которое устанавливается при нажатии на шкалу времени планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд.md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md) содержит Неопределено,
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md) содержит Неопределено,
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md) содержит Неопределено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата.md) заполнено,
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md) заполнено,
- Команды не содержат стандартных команд,
- [КомандаПоУмолчанию](html.md) заполнено командой [ВыбратьЭлементШкалыВремени](СтандартнаяКомандаПланировщика.ВыбратьЭлементШкалыВремени.md).

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
