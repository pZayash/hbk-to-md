# Расширение поля формы для поля планировщика.МаксимальнаяВысота

Расширение поля формы для поля планировщика (Form field extension for a planner)

МаксимальнаяВысота (MaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную высоту элемента.  
Используется, если свойство [АвтоМаксимальнаяВысота](Расширение_поля_формы_для_поля_планировщика.АвтоМаксимальнаяВысота_(Form_field_extension_for_a_planner.AutoMaxHeight).md) установлено в значение [Ложь](lang__def_BooleanFalse.md).  
Задается в символах. Значение 0 означает, что максимальная высота не ограничена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
