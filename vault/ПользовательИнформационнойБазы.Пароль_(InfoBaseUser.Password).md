ПользовательИнформационнойБазы (InfoBaseUser)

Пароль (Password)

Доступен, начиная с версии 8.0.

Использование:

Только запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Пароль, используемый при стандартной аутентификации.  
Установка свойства определяет пароль нового пользователя (по умолчанию - пустая строка) или меняет пароль существующего пользователя.  
Изменения начнут действовать только после записи пользователя.  
Не следует одному пользователю устанавливать значения этого свойства совместно со значением свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md). Иначе пароль записанному пользователю будет установлен в соответствии с установленным значением свойства [Пароль](ПользовательИнформационнойБазы.Пароль_(InfoBaseUser.Password).md), а установленное значение свойства [СохраняемоеЗначениеПароля](ПользовательИнформационнойБазы.СохраняемоеЗначениеПароля_(InfoBaseUser.StoredPasswordValue).md) будет проигнорировано.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
