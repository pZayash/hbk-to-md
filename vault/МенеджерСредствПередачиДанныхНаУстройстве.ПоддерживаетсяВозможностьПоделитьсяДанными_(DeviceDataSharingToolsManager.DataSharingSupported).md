МенеджерСредствПередачиДанныхНаУстройстве (DeviceDataSharingToolsManager)

ПоддерживаетсяВозможностьПоделитьсяДанными (DataSharingSupported)

Доступен, начиная с версии 8.3.25.

Синтаксис:

ПоддерживаетсяВозможностьПоделитьсяДанными()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md) на устройствах под управлением Android и iOS.  
Возвращает [Ложь](lang__def_BooleanFalse.md) на устройствах под управлением Windows.

Описание:

Определяет, поддерживается ли возможность делиться данными с другими приложениями в данной операционной системе.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.
