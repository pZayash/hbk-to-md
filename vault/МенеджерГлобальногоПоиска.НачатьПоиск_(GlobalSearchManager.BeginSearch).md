МенеджерГлобальногоПоиска (GlobalSearchManager)

НачатьПоиск (BeginSearch)

Доступен, начиная с версии 8.3.15.

Синтаксис:

НачатьПоиск(<СтрокаПоиска>)

Параметры:

<СтрокаПоиска> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка поиска.

Описание:

Начинает поиск по переданной строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

При работе в веб-клиенте, если поле ввода глобального поиска недоступно для ввода (по любой причине), то вызов в этот момент метода НачатьПоиск() не приведет к началу поиска.

Использование в версии:

Доступен, начиная с версии 8.3.15.
