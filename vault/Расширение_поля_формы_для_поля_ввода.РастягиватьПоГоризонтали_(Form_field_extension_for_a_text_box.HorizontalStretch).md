# Расширение поля формы для поля ввода.РастягиватьПоГоризонтали

Расширение поля формы для поля ввода (Form field extension for a text box)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Устанавливает возможность растягивания поля по горизонтали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.  
Значение [Неопределено](lang__def_Undefined.md) означает автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
