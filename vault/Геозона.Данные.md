Геозона (Geofence)

Данные (Data)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Ключ для поиска объекта конфигурации, ассоциированного с данной геозоной. Параметр должен поддерживать сериализацию.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
