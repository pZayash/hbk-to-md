ТекстовыйДокумент (TextDocument)

ПолучитьОбласть (GetArea)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбласть(<ИмяОбласти>)

Параметры:

<ИмяОбласти> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя области текстового документа.

Возвращаемое значение:

Тип: [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md).   

Описание:

Получает область текстового документа в качестве нового текстового документа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
