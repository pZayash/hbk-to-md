# БизнесПроцессМенеджер.<Имя бизнес-процесса>.НайтиПоНомеру

БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

НайтиПоНомеру (FindByNumber)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Для бизнес-процессов с непериодической нумерацией

Синтаксис:

НайтиПоНомеру(<НомерБизнесПроцесса>)

Параметры:

<НомерБизнесПроцесса> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Номер искомого документа. Тип передаваемого значения зависит от установки в конфигураторе представления номера бизнес-процесса.

Описание варианта метода:

Для бизнес-процессов со сквозной нумерацией.

Вариант синтаксиса: Для бизнес-процессов с нумерацией в пределах периода

Синтаксис:

НайтиПоНомеру(<НомерБизнесПроцесса>, <ДатаИнтервала>)

Параметры:

<НомерБизнесПроцесса> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Номер искомого бизнес-процесса. Тип передаваемого значения зависит от установки в конфигураторе представления номера бизнес-процесса.

<ДатаИнтервала> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата из интервала, в котором проводится поиск по номеру. Сам интервал определяется как период уникальности номеров бизнес-процесса, в который входит указанная дата. Например, если номера бизнес-процессов уникальны в пределах месяца и задана дата 10 декабря 2001 года, то поиск будет проводиться в интервале с 01 по 31 декабря 2001 года.

Описание варианта метода:

Для бизнес-процессов с периодической нумерацией.

Возвращаемое значение:

Тип: [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)>, [Неопределено](lang__def_Undefined.md).   
Ссылка на найденный бизнес-процесс. Если бизнес-процесс не найден, возвращается пустая ссылка.  
Если номер не задан (дина = 0), то будет возвращено [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск бизнес-процесса по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
