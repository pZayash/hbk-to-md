БуферДвоичныхДанных (BinaryDataBuffer)

Записать (Write)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Записать(<Позиция>, <Байты>, <Количество>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, начиная с которой требуется записать содержимое буфера.

<Байты> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Байты, которыми нужно заполнить часть буфера.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байт, которые требуется заменить.

Описание:

Заменить значения, начиная с заданной позиции, значениями из заданного буфера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
