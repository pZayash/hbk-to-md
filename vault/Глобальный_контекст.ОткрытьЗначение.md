Глобальный контекст (Global context)

ОткрытьЗначение (OpenValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьЗначение(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение любого типа, который может быть показан системой в диалоге.

Описание:

Показывает в диалоге переданное значение. Вид диалога зависит от типа переданного значения.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [ПоказатьЗначение](Глобальный_контекст.ПоказатьЗначение.md).

Пример:

|  |
| --- |
| Товар = Справочники.Номенклатура.НайтиПоКоду(КодТовара); ОткрытьЗначение(Товар); |
