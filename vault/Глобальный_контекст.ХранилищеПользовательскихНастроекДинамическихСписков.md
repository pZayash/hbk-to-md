Глобальный контекст (Global context)

ХранилищеПользовательскихНастроекДинамическихСписков (DynamicListsUserSettingsStorage)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md), [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища.md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища.md)>.   
Содержит объект, с помощью которого осуществляется чтение, изменение, добавление и удаление настроек динамических списков.   
Если в свойстве конфигурации "ХранилищеПользовательскихНастроекДинамическихСписков" не указано хранилище, то содержит объект [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md), предназначенный для чтения, изменения, добавления и удаления настроек динамических списков.  
Если в свойстве конфигурации "ХранилищеПользовательскихНастроекДинамическихСписков" указано хранилище, созданное в конфигурации, то содержит объект [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища.md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища.md)>.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
