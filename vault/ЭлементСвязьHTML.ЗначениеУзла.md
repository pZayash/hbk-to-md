ЭлементСвязьHTML (HTMLLinkElement)

ЗначениеУзла (NodeValue)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит значение узла HTML. Значение узла присутствует не для каждого типа узла HTML.  
Узел Значение узла

- [Атрибут](ТипУзлаDOM.Атрибут.md) - [Значение](АтрибутDOM.Значение.md);
- [Комментарий](ТипУзлаDOM.Комментарий.md) - [ТекстHTML.Данные](ТекстHTML.Данные.md), [КомментарийHTML.Данные](КомментарийHTML.Данные.md), содержит комментарий;
- [Документ](ТипУзлаDOM.Документ.md) - Неопределено;
- [Элемент](ТипУзлаDOM.Элемент.md) - Неопределено;
- [Текст](ТипУзлаDOM.Текст.md) - [ТекстHTML.Данные](ТекстHTML.Данные.md), [КомментарийHTML.Данные](КомментарийHTML.Данные.md), содержит текст узла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В качестве значения узла HTML используется значение узла DOM.

Использование в версии:

Доступен, начиная с версии 8.2.
