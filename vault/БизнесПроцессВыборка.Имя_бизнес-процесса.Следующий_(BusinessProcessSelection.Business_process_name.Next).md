# БизнесПроцессВыборка.<Имя бизнес-процесса>.Следующий

БизнесПроцессВыборка.<Имя бизнес-процесса> (BusinessProcessSelection.<Business process name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующий бизнес-процесс выбрана; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает очередной бизнес-процесс из выборки. Для обхода списка бизнес-процессов нужно после получения выборки использовать данный метод для позиционирования на первый и последующие бизнес-процессы выборки до тех пор, пока он не вернет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Выборка = БизнесПроцессы.ОбработкаПоступлений.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка выборки  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
