МакетОформления (AppearanceTemplate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [СтандартноеОформление](СтандартноеОформление_(StandardAppearance).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
