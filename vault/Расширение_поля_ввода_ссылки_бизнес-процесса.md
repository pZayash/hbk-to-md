# Расширение поля ввода ссылки бизнес-процесса

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Бизнес-процессы](Бизнес-процессы.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора бизнес-процесса в элементе управления [ПолеВвода](ПолеВвода.md) по введенному тексту. По умолчанию поиск осуществляется по полю "Номер".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
