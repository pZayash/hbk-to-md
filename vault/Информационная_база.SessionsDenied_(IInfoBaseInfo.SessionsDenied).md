Информационная база (IInfoBaseInfo)

SessionsDenied (SessionsDenied)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет режимом блокировки сеансов.   
[Истина](lang__def_BooleanTrue.md) и текущее время находится в диапазоне, определяемом свойствами [DeniedFrom](Информационная_база.DeniedFrom_(IInfoBaseInfo.DeniedFrom).md) и [DeniedTo](Информационная_база.DeniedTo_(IInfoBaseInfo.DeniedTo).md), то начало новых сеансов с информационной базой запрещено. Если при этом попытаться установить новый сеанс , то будет выдано сообщение об ошибке, содержащее текст из значения свойства [DeniedMessage](Информационная_база.DeniedMessage_(IInfoBaseInfo.DeniedMessage).md).   
Для начала сеанса вопреки блокировке сеансов необходимо указать код разрешения, определяемый свойством [PermissionCode](Информационная_база.PermissionCode_(IInfoBaseInfo.PermissionCode).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
