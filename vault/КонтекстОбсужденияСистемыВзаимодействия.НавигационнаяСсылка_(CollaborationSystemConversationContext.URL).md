КонтекстОбсужденияСистемыВзаимодействия (CollaborationSystemConversationContext)

НавигационнаяСсылка (URL)

Доступен, начиная с версии 8.3.10.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит навигационную ссылку на объект базы данных или пустую строку, если объект базы данных не имеет представления в виде ссылки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
