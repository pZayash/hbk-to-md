Свойства:

[НастройкаОбластиОформления (AppearanceAreaSetting)](Расширение_табличного_поля_области_оформления.НастройкаОбластиОформления.md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для области оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ЭлементОбластиОформления](ЭлементОбластиОформления.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
