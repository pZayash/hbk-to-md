Глобальный контекст (Global context)

КаталогВременныхФайловАсинх (TempFilesDirAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

КаталогВременныхФайловАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Строка](lang__def_String.md), имя каталога.

Описание:

Асинхронная функция. Функция получает имя каталога, который используется программой для размещения временных файлов

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции в веб-клиенте необходимо предварительно подключить расширение работы с файлами.
