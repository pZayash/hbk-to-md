СерииДиаграммы (ChartSeriesCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Серия>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Серия диаграммы.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс указанной серии диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ИндексСерии = Диаграмма.Серии.Индекс(ВыбраннаяСерия); |

Использование в версии:

Доступен, начиная с версии 8.0.
