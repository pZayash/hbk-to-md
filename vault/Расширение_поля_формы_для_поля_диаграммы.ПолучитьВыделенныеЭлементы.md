Расширение поля формы для поля диаграммы (Form field extension for a chart field)

ПолучитьВыделенныеЭлементы (GetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьВыделенныеЭлементы()

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Элементами массива являются значения следующих типов: [ТочкаДиаграммы](ТочкаДиаграммы.md), [СерияДиаграммы](СерияДиаграммы.md), [ЗначениеДиаграммы](ЗначениеДиаграммы.md).

Описание:

Возвращает массив выделенных элементов диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
