Файл (File)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [Файл](Файл_(File).md).   
Содержит файл, с которым связан объект, если файл доступен, и [Неопределено](lang__def_Undefined.md) в противном случае.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
