ПолеЭлементаБлокировкиДанных (DataLockItemField)

Поле (Field)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md), [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)>.   
Содержит имя поля пространства имен блокировок или ссылка на вид характеристики (может применяться для блокировки по субконто у регистра бухгалтерии).  
Возможны следующие имена: Регистратор, Период, <Измерение>.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
