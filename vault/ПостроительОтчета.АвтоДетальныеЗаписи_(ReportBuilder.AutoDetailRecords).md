ПостроительОтчета (ReportBuilder)

АвтоДетальныеЗаписи (AutoDetailRecords)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство [Истина](lang__def_BooleanTrue.md), то необходимость вывода детальных записей будет определяться автоматически.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
