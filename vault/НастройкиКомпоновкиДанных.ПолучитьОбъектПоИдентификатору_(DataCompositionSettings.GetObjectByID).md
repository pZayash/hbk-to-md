НастройкиКомпоновкиДанных (DataCompositionSettings)

ПолучитьОбъектПоИдентификатору (GetObjectByID)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбъектПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md).   
Идентификатор объекта.

Возвращаемое значение:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md), [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных_(DataCompositionGroup).md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных_(DataCompositionTable).md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных_(DataCompositionChart).md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных_(DataCompositionNestedObjectSettings).md), [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных_(DataCompositionChartGroup).md), [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных_(DataCompositionTableGroup).md), [КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных](КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных_(DataCompositionChartStructureItemCollection).md), [КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных](КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных_(DataCompositionTableStructureItemCollection).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объект, для которого был передан идентификатор, не найден или если он не является частью данного объекта.

Описание:

Получает объект структуры настроек компоновки данных, соответствующие переданному идентификатору.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
