# ПоследовательностьМенеджер.<Имя последовательности>.Проверить

ПоследовательностьМенеджер.<Имя последовательности> (SequenceManager.<Sequence name>)

Проверить (Validate)

Доступен, начиная с версии 8.0.

Синтаксис:

Проверить(<МоментВремени>, <Отбор>)

Параметры:

<МоментВремени> (обязательный)

Тип: [Дата](lang__def_Date.md), [МоментВремени](МоментВремени_(PointInTime).md).   
Момент времени, по который проверяется последовательность.

<Отбор> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, задающая отбор по измерениям последовательности. Ключ структуры определяет измерение последовательности, по которому будет осуществляться отбор, а значение структуры - значение измерения.  
Если измерение отсутствует в структуре, то по нему отбор осуществляться не будет.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - последовательность не нарушена; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, что по указанный момент времени последовательность по заданным измерениям не нарушена.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Момент = Новый МоментВремени(Док.Дата, Док.Ссылка); Отбор = Новый Структура; Отбор.Вставить("Товар", ТекТовар); Непрерывна = Последовательности.УчетТоваров.Проверить(Момент, Отбор); |

Использование в версии:

Доступен, начиная с версии 8.0.
