ИнформацияОбИнтернетСоединении (InternetConnectionInformation)

ПолучитьИспользованиеРоуминга (GetRoamingUsage)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьИспользованиеРоуминга()

Возвращаемое значение:

Тип: [ИспользованиеРоуминга](ИспользованиеРоуминга.md).   
[Используется](ИспользованиеРоуминга.Используется.md) - соединение установлено в роуминговой сети, использование интернета может привести к дополнительным расходам.  
[НеИспользуется](ИспользованиеРоуминга.НеИспользуется.md) - трафик учитывается согласно тарифу оператора.  
[Неизвестно](ИспользованиеРоуминга.Неизвестно.md) - используется Wi-fi, локальная сеть или соединение отсутствует

Описание:

Возвращает признак использования роуиинга в мобильной сети.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

На устройствах с ОС iOS существуют следующие особенности: метод использует сравнение данных из SIM-карты устройства (MCC) и данные геолокации (SOcountryCode), поэтому точный результат не гарантирован. При использовании роуминга внутри одной страны, метод вернет значение [Неизвестно](ИспользованиеРоуминга.Неизвестно.md).  
Метод работает только если приложение получило разрешение на использование службы геолокации, в противном случае будет возвращать значение [Неизвестно](ИспользованиеРоуминга.Неизвестно.md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
