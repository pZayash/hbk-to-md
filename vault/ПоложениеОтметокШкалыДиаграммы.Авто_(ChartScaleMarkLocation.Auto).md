ПоложениеОтметокШкалыДиаграммы (ChartScaleMarkLocation)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Положение отметок определяется автоматически:

- если подписи диаграммы отображаются снаружи -- [Снаружи](ПоложениеОтметокШкалыДиаграммы.Снаружи_(ChartScaleMarkLocation.Outside).md),
- если подписи диаграммы отображаются внутри -- [Внутри](ПоложениеОтметокШкалыДиаграммы.Внутри_(ChartScaleMarkLocation.Inside).md).

Использование в версии:

Доступен, начиная с версии 8.3.11.
