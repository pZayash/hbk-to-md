Элементы коллекции:

[ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса.md), [ДоступнаяВложеннаяТаблицаСхемыЗапроса](ДоступнаяВложеннаяТаблицаСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные поля и доступные вложенные таблицы.  
Возможно обращение к доступным полям и доступным вложенным таблицам посредством оператора [...]. В качестве аргумента передается индекс.
