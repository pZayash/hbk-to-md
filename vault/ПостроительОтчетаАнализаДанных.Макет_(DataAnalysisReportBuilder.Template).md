ПостроительОтчетаАнализаДанных (DataAnalysisReportBuilder)

Макет (Template)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   
Содержит макет для вывода результата. Макет генерируется автоматически. Для переопределения макета необходимо записать желаемый макет в данное свойство. При необходимости возобновить автоматическую генерацию макета необходимо записать в это свойство значение [Неопределено](lang__def_Undefined.md).  
Названия областей и параметров макета для каждого вида анализа можно получить из автоматически сгенерированного макета.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
