ЧтениеТекста (TextReader)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЧтениеТекста()

Описание:

Формирует неинициализированный объект [ЧтениеТекста](ЧтениеТекста.md). Для инициализации объекта используется метод [Открыть](ЧтениеТекста.Открыть.md).

Пример:

|  |
| --- |
| Текст = Новый ЧтениеТекста; |

Использование в версии:

Доступен, начиная с версии 8.0.
