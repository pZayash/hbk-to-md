ДинамическийСписок (DynamicList)

ПараметрыДанных (DataParameters)

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).

Использование:

Только чтение.

Описание:

Тип: [ПараметрыСхемыКомпоновкиДанных](ПараметрыСхемыКомпоновкиДанных_(DataCompositionSchemaParameters).md).   
Содержит описания параметров.  
  
Свойство [ПараметрФункциональныхОпций](ПараметрСхемыКомпоновкиДанных.ПараметрФункциональныхОпций_(DataCompositionSchemaParameter.FunctionalOptionsParameter).md) не поддерживается в динамическом списке.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).
