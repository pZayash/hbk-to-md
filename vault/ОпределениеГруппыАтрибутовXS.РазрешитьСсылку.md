ОпределениеГруппыАтрибутовXS (XSAttributeGroupDefinition)

РазрешитьСсылку (ResolveReference)

Доступен, начиная с версии 8.1.

Синтаксис:

РазрешитьСсылку()

Возвращаемое значение:

Тип: [ОпределениеГруппыАтрибутовXS](ОпределениеГруппыАтрибутовXS.md), Неопределено.   
Если данная группа атрибутов является ссылкой на глобальную группу атрибутов, то возвращаемым значением будет глобальная группа атрибутов (определенная в схеме-владельце данного компонента или в одной из схем, указанных в директивах импорта или включения) или Неопределено, если для имени, указанного в свойстве [Ссылка](ОпределениеГруппыАтрибутовXS.Ссылка.md) группы атрибутов не обнаружено.

Описание:

Производит разыменование ссылки на группу атрибутов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
