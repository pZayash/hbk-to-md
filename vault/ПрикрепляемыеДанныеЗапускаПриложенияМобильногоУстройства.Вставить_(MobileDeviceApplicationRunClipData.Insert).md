ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipData)

Вставить (Insert)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Вставить(<Индекс>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента в контейнере. Значение должно быть в диапазоне от 0 до [Количество](ПрикрепляемыеДанныеЗапускаПриложенияМобильногоУстройства.Количество_(MobileDeviceApplicationRunClipData.Count).md) – 1.

<Элемент> (обязательный)

Тип: [ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства](ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства_(MobileDeviceApplicationRunClipDataItem).md).   
Вставляемый элемент прикрепляемых данных.

Описание:

Вставляет элемент прикрепляемых данных в контейнер. Все элементы с таким номером и больше переданного сдвигаются.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
