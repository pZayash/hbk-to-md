НастройкаОбластиОформления (AppearanceAreaSetting)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Доступность>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя элемента управления областью оформления.

<Доступность> (обязательный)

Тип: Булево.   
Определяет доступность элемента для редактирования.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойОбластиОформления](ЭлементУправленияНастройкойОбластиОформления.md).   

Описание:

Добавляет элемент управления областью оформления в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
