ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

Показать (Show)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога со следующими параметрами:

- <ТипДиаграммы> - [ТипДиаграммы](ТипДиаграммы.md) – тип диаграммы выбран. Неопределено - в противном случае;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).
- Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Описание:

Открывает немодальный диалог выбора типа диаграммы.  
По умолчанию будет установлен тип диаграммы, установленный в свойстве [ТипДиаграммы](ДиалогВыбораТипаДиаграммы.ТипДиаграммы.md).  
Для выбора будут доступны типы диаграммы, установленные в свойстве [ДоступныеТипыДиаграммы](ДиалогВыбораТипаДиаграммы.ДоступныеТипыДиаграммы.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
