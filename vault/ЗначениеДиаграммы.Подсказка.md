ЗначениеДиаграммы (ChartValue)

Подсказка (ToolTip)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Содержит текст всплывающей подсказки, которая появляется при наведении курсора к значению диаграммы. Если не задана, будет сформирована по общим правилам.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
