# Форма.ПроверитьЗаполнение

Форма (Form)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Толстый клиент.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](Форма.ОбработкаПроверкиЗаполнения_(Form.FillCheckProcessing).md).

[Форма](Форма_(Form).md), событие [ОбработкаПроверкиЗаполнения](Форма.ОбработкаПроверкиЗаполнения_(Form.FillCheckProcessing).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
