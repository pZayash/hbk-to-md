# ТабличноеПоле.ПередОкончаниемРедактирования

ТабличноеПоле (TableBox)

ПередОкончаниемРедактирования (BeforeEditEnd)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередОкончаниемРедактирования(<НоваяСтрока>, <ОтменаРедактирования>, <Отказ>)

Параметры:

<НоваяСтрока>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установлен в [Истина](lang__def_BooleanTrue.md), если строка была добавлена или скопирована.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если произошла отмена редактирования.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от выполнения действия.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается перед окончанием редактирования строки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
