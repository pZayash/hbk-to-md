ПолеФормы (FormField)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Указывает необходимость пропуска поля при вводе данных.  
Если [Неопределено](lang__def_Undefined.md), то определяется значением свойства [ОтображениеПредупрежденияПриРедактировании](ПолеФормы.ОтображениеПредупрежденияПриРедактировании_(FormField.WarningOnEditRepresentation).md). Если указан режим [Отображать](ОтображениеПредупрежденияПриРедактировании.Отображать_(WarningOnEditRepresentation.Show).md) или [Авто](ОтображениеПредупрежденияПриРедактировании.Авто_(WarningOnEditRepresentation.Auto).md) и в конфигурации указан режим отображения, то это эквивалентно значению [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
