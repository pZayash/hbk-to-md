ОбъектМетаданных: WebSocketКлиент (MetadataObject: WebSocketClient)

Предопределенный (Predefined)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает, является ли WebSocket-клиент предопределенным.   
Предопределенные клиенты определяются в метаданных. Предопределенные клиенты можно изменять, но нельзя удалять.  
Предопределенные клиенты создаются автоматически.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
