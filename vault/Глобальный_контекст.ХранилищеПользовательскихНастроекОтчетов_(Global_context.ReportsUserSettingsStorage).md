Глобальный контекст (Global context)

ХранилищеПользовательскихНастроекОтчетов (ReportsUserSettingsStorage)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md), [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)>.   
Содержит объект, с помощью которого осуществляется чтение, изменение, добавление и удаление настроек отчетов.   
Если в свойстве конфигурации "ХранилищеПользовательскихНастроекОтчетов" не указано хранилище, то содержит объект [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md), предназначенный для чтения, изменения, добавления и удаления настроек вариантов отчетов.  
Если в свойстве конфигурации "ХранилищеПользовательскихНастроекОтчетов" указано хранилище, созданное в конфигурации, то содержит объект [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)>.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
