ПоляНастройки (CustomFields)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Поле>, <Смещение>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеНастройки](ПолеНастройки_(CustomField).md).   
Перемещаемое поле.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает заданный элемент коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
