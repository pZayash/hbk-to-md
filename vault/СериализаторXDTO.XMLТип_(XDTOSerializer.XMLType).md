# СериализаторXDTO.XMLТип

СериализаторXDTO (XDTOSerializer)

XMLТип (XMLType)

Доступен, начиная с версии 8.1.

Синтаксис:

XMLТип(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип значения, для которого необходимо определить соответствующий XML тип.

Возвращаемое значение:

Тип: [ТипДанныхXML](ТипДанныхXML_(XMLDataType).md).   
[Неопределено](lang__def_Undefined.md), если никакой тип данных XML не соответствует указанному типу.

Описание:

Позволяет получить тип данных XML, соответствующий переданному в качестве параметра типу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
