ЗаписьДанных (DataWriter)

ЗакрытьАсинх (CloseAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗакрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Инициирует вызов метода [СброситьБуферы](ЗаписьДанных.СброситьБуферы.md). Если целевой поток был создан при создании объекта [ЗаписьДанных](ЗаписьДанных.md), также вызывает начало закрытие целевого потока.

Доступность:

Тонкий клиент, толстый клиент.

[ЗаписьДанных](ЗаписьДанных.md), метод [НачатьЗакрытие](ЗаписьДанных.НачатьЗакрытие.md)

Использование в версии:

Доступен, начиная с версии 8.3.18.
