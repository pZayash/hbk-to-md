Расширение формы набора записей регистра сведений (Information register record set form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: Булево.   
Истина - объект записан; Ложь - в противном случае.

Описание:

Записывает набор, открытый в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Объект записан."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
