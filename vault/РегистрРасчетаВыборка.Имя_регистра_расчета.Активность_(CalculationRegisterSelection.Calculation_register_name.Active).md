РегистрРасчетаВыборка.<Имя регистра расчета> (CalculationRegisterSelection.<Calculation register name>)

Активность (Active)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак активности записи. Влияние записи на фактический период действия, перерасчеты, работу методов [ПолучитьБазу](РегистрРасчетаМенеджер.Имя_регистра_расчета.ПолучитьБазу_(CalculationRegisterManager.Calculation_register_name.GetBase).md) и [ПолучитьДанныеГрафика](РегистрРасчетаМенеджер.Имя_регистра_расчета.ПолучитьДанныеГрафика_(CalculationRegisterManager.Calculation_register_name.GetScheduleData).md).  
[Истина](lang__def_BooleanTrue.md) - запись активна.

Доступность:

Сервер, толстый клиент, внешнее соединение.
