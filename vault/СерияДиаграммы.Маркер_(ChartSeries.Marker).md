СерияДиаграммы (ChartSeries)

Маркер (Marker)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипМаркераДиаграммы](ТипМаркераДиаграммы_(ChartMarkerType).md).   
Содержит тип маркера, которым отмечаются значения в точках на графике.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для диаграмм вида [График](ТипДиаграммы.График_(ChartType.Line).md).

Использование в версии:

Доступен, начиная с версии 8.0.
