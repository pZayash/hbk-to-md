ТабличноеПоле (TableBox)

ТекущаяКолонка (CurrentColumn)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [КолонкаТабличногоПоля](КолонкаТабличногоПоля.md).   
Содержит текущую колонку табличного поля. Если данному свойству присвоить [КолонкаТабличногоПоля](КолонкаТабличногоПоля.md), то произойдет активизация указанной колонки, если устанавливается значение Неопределено, то активная колонка сбрасывается.

Доступность:

Толстый клиент.

См. также:

[КолонкаТабличногоПоля](КолонкаТабличногоПоля.md)  
[ТабличноеПоле](ТабличноеПоле.md), свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
