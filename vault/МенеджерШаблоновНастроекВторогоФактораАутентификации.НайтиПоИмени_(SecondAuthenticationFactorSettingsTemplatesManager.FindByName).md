МенеджерШаблоновНастроекВторогоФактораАутентификации (SecondAuthenticationFactorSettingsTemplatesManager)

НайтиПоИмени (FindByName)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НайтиПоИмени(<ИмяШаблона>)

Параметры:

<ИмяШаблона> (обязательный)

Тип: [Строка](lang__def_String.md).   
Уникальное имя шаблона настроек второго фактора аутентификации, хранящегося в базе.

Возвращаемое значение:

Тип: [ШаблонНастройкиВторогоФактораАутентификации](ШаблонНастройкиВторогоФактораАутентификации_(SecondAuthenticationFactorSettingTemplate).md).   
[Неопределено](lang__def_Undefined.md) - если шаблон с указанным именем не найден.

Описание:

Выполняет поиск шаблона по полному имени.  
Для выполнения требуется наличие права "Администратор".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
