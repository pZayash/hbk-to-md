# Планировщик.ВыравниватьГраницыЭлементовПоШкалеВремени

Планировщик (Planner)

ВыравниватьГраницыЭлементовПоШкалеВремени (AlignItemBoundariesByTimeScale)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство определяет, выравнивать ли границы элементов по делениям шкалы времени или нет. Если установлено в [Истина](lang__def_BooleanTrue.md), то границы элементов будут автоматически раздвигаться в большую сторону до ближайшего деления шкалы времени.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
