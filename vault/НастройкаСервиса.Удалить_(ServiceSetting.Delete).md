НастройкаСервиса (ServiceSetting)

Удалить (Delete)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Удалить()

Описание:

Удаляет настройку сервиса кластера на рабочем сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
