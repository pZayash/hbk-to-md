КлиентскоеПриложение (ClientApplication)

ПолучитьЗаголовок (GetCaption)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьЗаголовок()

Возвращаемое значение:

Тип: Строка.   

Описание:

Получает заголовок приложения, который используется при формировании заголовков окон.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение.md) получает заголовок главного окна.

[КлиентскоеПриложение](КлиентскоеПриложение.md), метод [УстановитьЗаголовок](КлиентскоеПриложение.УстановитьЗаголовок.md)  

Использование в версии:

Доступен, начиная с версии 8.3.10.
