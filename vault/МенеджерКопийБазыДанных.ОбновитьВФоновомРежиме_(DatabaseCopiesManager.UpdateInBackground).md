МенеджерКопийБазыДанных (DatabaseCopiesManager)

ОбновитьВФоновомРежиме (UpdateInBackground)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ОбновитьВФоновомРежиме(<Копия>, <Порционное>, <КоличествоЗаданийОбновления>)

Параметры:

<Копия> (обязательный)

Тип: [МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md), [Строка](lang__def_String.md).   
Объект менеджера копии или имя копии.

<Порционное> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Обновление выполняется в течение максимум 20 минут, после чего прерывается.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<КоличествоЗаданийОбновления> (необязательный)

Тип: [Число](lang__def_Number.md).   
Число заданий, используемых в процессе первоначального копирования таблицы в копию.   
По умолчанию - 0, при этом количество заданий - 4.  
Количество заданий всегда = 1, вне зависимости от значения параметра, в случае, если копия базы данных расположена на сервере Oracle.  
Дополнительно, для копии на масштабируемом Дата акселераторе определяет количество параллельных заданий обновления экземпляров копии. Значение по умолчанию = 0, при этом количество параллельных заданий - 4.  
Значение по умолчанию: 0.

Описание:

Выполняет обновление копии в системном фоновом задании.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
