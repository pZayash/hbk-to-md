ДокументыМенеджер (DocumentsManager)

ТипВсеСсылки (AllRefsType)

Доступен, начиная с версии 8.0.

Синтаксис:

ТипВсеСсылки()

Возвращаемое значение:

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   

Описание:

Получает описание типов, содержащее типы ссылок документов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| МассивТипов = ЭлементыФормы.ПолеВводаВариант.ТипЗначения.Типы(); Для каждого ЭлементМассива Из МассивТипов Цикл     Если Документы.ТипВсеСсылки().СодержитТип(ЭлементМассива) Тогда         МассивНужныхТипов.Добавить(ЭлементМассива);     КонецЕсли;  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
