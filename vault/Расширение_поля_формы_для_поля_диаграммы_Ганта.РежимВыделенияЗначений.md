Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

РежимВыделенияЗначений (ValuesSelectionMode)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [РежимВыделенияЗначенийДиаграммыГанта](РежимВыделенияЗначенийДиаграммыГанта.md).   
Определяет режим выделения значений в диаграмме. Значение по умолчанию - [Авто](РежимВыделенияЗначенийДиаграммыГанта.Авто.md).  
Если установлено значение [Нет](РежимВыделенияЗначенийДиаграммыГанта.Нет.md), то свойства [РежимВыделения](ТаблицаФормы.РежимВыделения.md) таблицы принимает значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный.md).  
Если установлено значение [Авто](РежимВыделенияЗначенийДиаграммыГанта.Авто.md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения.md) в режиме совместимости с [Версия8\_3\_22](РежимСовместимости.Версия8_3_22.md) и ниже устанавливается значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный.md). В остальных случаях устанавливается значение {ссылка:Атрибуты; 13383; Множественный}.  
Если данное свойство принимает значение [Множественный](РежимВыделенияЗначенийДиаграммыГанта.Множественный.md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения.md) таблицы также принимает значение [Множественный](РежимВыделенияТаблицы.Множественный.md), и наоборот.  
Если данное свойство принимает значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный.md), то и свойство [РежимВыделения](ТаблицаФормы.РежимВыделения.md) таблицы также принимает значение [Одиночный](РежимВыделенияТаблицы.Одиночный.md), и наоборот.   
Если установлено значение [Нет](РежимВыделенияЗначенийДиаграммыГанта.Нет.md), то свойство [РежимВыделения](ТаблицаФормы.РежимВыделения.md) таблицы принимает значение [Одиночный](РежимВыделенияТаблицы.Одиночный.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
