ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ВместеСоСледующим (StayWithNext)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Объединяет строки или колонки области таким образом, чтобы при разделении на страницы они всегда оказывались на одной странице.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - строки или колонки области объединяются.  
Применимо только к областям, состоящим из строк или колонок.

Использование в версии:

Доступен, начиная с версии 8.2.
