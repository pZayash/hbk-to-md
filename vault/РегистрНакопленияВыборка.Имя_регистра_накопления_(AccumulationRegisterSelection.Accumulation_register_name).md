Описание:

Объект этого типа возвращается методами [Выбрать](РегистрНакопленияМенеджер.Имя_регистра_накопления.Выбрать_(AccumulationRegisterManager.Accumulation_register_name.Select).md) и [ВыбратьПоРегистратору](РегистрНакопленияМенеджер.Имя_регистра_накопления.ВыбратьПоРегистратору_(AccumulationRegisterManager.Accumulation_register_name.SelectByRecorder).md) у объекта типа [РегистрНакопленияМенеджер.](РегистрНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegisterManager.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияМенеджер.Имя_регистра_накопления_(AccumulationRegisterManager.Accumulation_register_name).md)> и представляет собой специализированный способ перебора записей регистра накопления.  
Обход записей выполняется системой динамически. Это означает, что использование выборки не считывает все записи сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большое количество записей и не загружать в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
