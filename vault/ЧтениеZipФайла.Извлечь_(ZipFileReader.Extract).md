ЧтениеZipФайла (ZipFileReader)

Извлечь (Extract)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Извлечь](ЧтениеФайлаАрхива.Извлечь_(ArchiveFileReader.Extract).md)

Доступен, начиная с версии 8.0.

Синтаксис:

Извлечь(<Элемент>, <Путь>, <ВосстанавливатьКаталоги>, <Пароль>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементZipФайла](ЭлементZipФайла_(ZipFileEntry).md).   
Элемент, который требуется распаковать.

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь, по которому требуется распаковать элемент.

<ВосстанавливатьКаталоги> (необязательный)

Тип: [РежимВосстановленияПутейФайловZIP](РежимВосстановленияПутейФайловZIP_(ZIPRestoreFilePathsMode).md).   
Указывает, требуется ли восстанавливать структуру папок.  
Значение по умолчанию: [Восстанавливать](РежимВосстановленияПутейФайловZIP.Восстанавливать_(ZIPRestoreFilePathsMode.Restore).md).

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывается пароль для расшифровки элемента архива, в случае, если он зашифрован с паролем, отличным от пароля общего для файла.  
Если пароль не указан, используется пароль, заданный в конструкторе или при вызове метода [Открыть](ЧтениеZipФайла.Открыть_(ZipFileReader.Open).md), если он был там указан.

Описание:

Распаковывает переданный элемент в указанную папку.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
