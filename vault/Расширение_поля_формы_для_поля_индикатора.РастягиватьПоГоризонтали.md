Расширение поля формы для поля индикатора (Form field extension for a progress bar field)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает возможность растягивания поля индикатора по горизонтали.   
Истина - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
