Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПрименениеРежимаОтображенияПриУстановкеРезультата (ViewModeApplicationOnSetResult)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [ПрименениеРежимаОтображенияПриУстановкеРезультатаОтчета](ПрименениеРежимаОтображенияПриУстановкеРезультатаОтчета_(ViewModeApplicationOnSetReportResult).md).   
Применение режима отображения при установке результата.   
Значение по умолчанию - [Авто](ПрименениеРежимаОтображенияПриУстановкеРезультатаОтчета.Авто_(ViewModeApplicationOnSetReportResult.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
