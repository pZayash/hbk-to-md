ВытесняющиеВидыРасчета (DisplacingCalculationTypes)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ВытесняющиеВидыРасчета.\<Имя плана видов расчета\>](ВытесняющиеВидыРасчета.Имя_плана_видов_расчета_(DisplacingCalculationTypes.Chart_of_calculation_types_name).md).   
Предопределенная табличная часть вида расчета, которая содержит список вытесняющих видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство определено только для объектов тех планов видов расчета, которые имеют признак "Использует период действия".

Использование в версии:

Доступен, начиная с версии 8.0.
