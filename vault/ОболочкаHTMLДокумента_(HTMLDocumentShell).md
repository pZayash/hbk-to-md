Используется для доступа к HTML-документу.  
Может быть получен методами ПолучитьМакет() и [ПолучитьОбщийМакет](Глобальный_контекст.ПолучитьОбщийМакет_(Global_context.GetCommonTemplate).md), если тип макета [HTMLДокумент](ТипМакета.HTMLДокумент_(TemplateType.HTMLDocument).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
