КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Итог (Total)

Доступен, начиная с версии 8.0.

Синтаксис:

Итог(<Колонка>, <ВключатьПодчиненные>)

Параметры:

<Колонка> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [КолонкаДереваЗначений](КолонкаДереваЗначений_(ValueTreeColumn).md).   
Колонка, по которой подсчитывается итог.  
В качестве значения параметра может быть передан индекс колонки, имя колонки, либо колонка дерева значений.

<ВключатьПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли включены в сумму итоги по подчиненным строкам (если таковые имеются). [Истина](lang__def_BooleanTrue.md) - итоги будут включены.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   

Описание:

Суммирует значения всех строк в указанной колонке.  
Если в колонке установлен тип и он единственный, то при суммировании будет предприниматься попытка преобразования значения к типу [Число](lang__def_Number.md).  
Если колонке не присвоены типы, то в процессе суммирования будут принимать участие только значения, имеющие тип [Число](lang__def_Number.md), значения других типов будут игнорироваться.  
Если в колонке несколько типов и среди них есть тип [Число](lang__def_Number.md), то в процессе суммирования будут принимать участие только значения, имеющие тип [Число](lang__def_Number.md), значения других типов будут игнорироваться.  
Если в колонке несколько типов и среди них нет типа [Число](lang__def_Number.md), то результатом будет значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СтоимостьПродукции = СтрокиСостава.Итог("Сумма"); |

Использование в версии:

Доступен, начиная с версии 8.0.
