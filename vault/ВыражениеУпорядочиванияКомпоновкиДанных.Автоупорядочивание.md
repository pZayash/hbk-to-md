ВыражениеУпорядочиванияКомпоновкиДанных (DataCompositionOrderExpression)

Автоупорядочивание (AutoOrder)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Автоматическое определение полей, по которым необходимо упорядочивать результат упорядочивания. Например, для ссылочных полей информационной базы 1С:Предприятия будет осуществляться упорядочивание по тем полям, по которым следует упорядочивать таблицу, на которую осуществляется ссылка.  
Истина - выполнять автоматическое упорядочивание.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
