МенеджерОформленияОтчетов (ReportsAppearanceManager)

УстановитьМакетОформленияПользователя (SetUserAppearanceTemplate)

Доступен, начиная с версии 8.3.22.

Синтаксис:

УстановитьМакетОформленияПользователя(<ИмяМакета>, <ИмяПользователя>)

Параметры:

<ИмяМакета> (обязательный)

Тип: Строка.   
Имя макета оформления компоновки данных. Чтобы использовать системный макет оформления, необходимо передать пустую строку.

<ИмяПользователя> (необязательный)

Тип: Строка.   
Имя пользователя, для которого требуется установить макет оформления. При отсутствии передаваемого значения параметра макет оформления устанавливается для текущего пользователя.

Описание:

Устанавливает для пользователя макет оформления компоновки данных, используемый по умолчанию. Для доступа к параметрам другого пользователя требуется наличие права "Администрирование данных".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.22.
