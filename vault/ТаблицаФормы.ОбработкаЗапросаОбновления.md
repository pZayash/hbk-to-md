ТаблицаФормы (FormTable)

ОбработкаЗапросаОбновления (RefreshRequestProcessing)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ОбработкаЗапросаОбновления()

Описание:

Возникает на клиенте мобильного приложения, когда пользователь отправляет запрос на обновление таблицы.  
Порядок выполнения запроса устанавливается в свойстве [ЗапросОбновления](ТаблицаФормы.ЗапросОбновления.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.6.
