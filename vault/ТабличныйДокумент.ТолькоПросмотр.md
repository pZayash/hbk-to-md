ТабличныйДокумент (SpreadsheetDocument)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что табличный документ открыт в режиме просмотра. В режиме просмотра в таблице не доступны для редактирования ячейки и рисунки, у которых установлено свойство [Защита](ТабличныйДокумент.Защита.md).  
Истина - режим просмотра.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если табличный документ расположен на форме, то у него могут возникать события [Выбор](ПолеТабличногоДокумента.Выбор.md) и [ОбработкаРасшифровки](ПолеТабличногоДокумента.ОбработкаРасшифровки.md) только в режиме просмотра.

Использование в версии:

Доступен, начиная с версии 8.0.
