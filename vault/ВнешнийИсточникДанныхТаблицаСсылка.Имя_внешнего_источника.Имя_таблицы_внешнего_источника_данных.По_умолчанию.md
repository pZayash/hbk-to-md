ВнешнийИсточникДанныхТаблицаСсылка.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRef.<External source name>.<External data source table name>)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ВнешнийИсточникДанныхТаблицаСсылка.<Имя внешнего источника>.<Имя таблицы внешнего источника данных>(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение ссылки.  
Если параметр не указан, то создается пустая ссылка.

Описание:

Создает ссылку на основании значения ссылки.

Использование в версии:

Доступен, начиная с версии 8.2.
