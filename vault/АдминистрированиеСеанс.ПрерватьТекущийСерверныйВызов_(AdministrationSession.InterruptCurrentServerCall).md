АдминистрированиеСеанс (AdministrationSession)

ПрерватьТекущийСерверныйВызов (InterruptCurrentServerCall)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПрерватьТекущийСерверныйВызов(<СообщениеОбОшибке>)

Параметры:

<СообщениеОбОшибке> (необязательный)

Тип: [Строка](lang__def_String.md).   
Сообщение для пользователя, которое содержит причину прерывания выполнения текущей операции.

Описание:

Прерывает выполнение текущего серверного вызова.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.20.
