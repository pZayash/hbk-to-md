НаборыДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSets)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [НаборДанныхЗапросСхемыКомпоновкиДанных](НаборДанныхЗапросСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetQuery).md), [НаборДанныхОбъектСхемыКомпоновкиДанных](НаборДанныхОбъектСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetObject).md), [НаборДанныхОбъединениеСхемыКомпоновкиДанных](НаборДанныхОбъединениеСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetUnion).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
