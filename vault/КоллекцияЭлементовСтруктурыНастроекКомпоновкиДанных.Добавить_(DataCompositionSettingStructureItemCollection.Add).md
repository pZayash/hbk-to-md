КоллекцияЭлементовСтруктурыНастроекКомпоновкиДанных (DataCompositionSettingStructureItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных_(DataCompositionGroup).md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных_(DataCompositionTable).md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных_(DataCompositionChart).md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных_(DataCompositionNestedObjectSettings).md).

Возвращаемое значение:

Тип: [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных_(DataCompositionGroup).md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных_(DataCompositionTable).md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных_(DataCompositionChart).md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных_(DataCompositionNestedObjectSettings).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
