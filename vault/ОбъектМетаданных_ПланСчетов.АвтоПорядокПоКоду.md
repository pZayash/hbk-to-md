ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

АвтоПорядокПоКоду (AutoOrderByCode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если свойство установлено в значение Истина, то упорядочивание по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код.md) будет фактически подменяться упорядочиванием по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок.md). Другими словами, упорядочивание по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код.md) и по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок.md) будут давать одинаковый результат. При этом по умолчанию список счетов будет упорядочиваться по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код.md).  
Если свойство установлено в значение Ложь, то упорядочивание по полю Код будет фактическим, а по умолчанию список счетов будет упорядочиваться (по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код.md) или по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок.md)) в зависимости от значения свойства [ДлинаПорядка](ОбъектМетаданных_ПланСчетов.ДлинаПорядка.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
