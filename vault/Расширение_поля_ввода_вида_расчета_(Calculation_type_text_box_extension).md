# Расширение поля ввода вида расчета

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md)

Доступен, начиная с версии 8.0.

Свойства:

[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_вида_расчета.БыстрыйВыбор_(Calculation_type_text_box_extension.QuickChoice).md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на вид расчета.  
Расширение предоставляет возможность автоматического поиска и подбора элемента вида расчета по введенному тексту. По умолчанию поиск осуществляется по полям код и наименование.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
