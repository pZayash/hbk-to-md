НастройкиВложенногоОбъектаКомпоновкиДанных (DataCompositionNestedObjectSettings)

УстановитьИдентификатор (SetIdentifier)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьИдентификатор(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Строка](lang__def_String.md).   
Устанавливаемый идентификатор объекта настройки.

Описание:

Устанавливает идентификатор объекта настройки.  
При первом изменении идентификатора, когда предыдущий идентификатор был пустым, происходит загрузка настроек "по умолчанию", подготовленных для этого объекта в источнике доступных настроек (см. [Инициализировать](КомпоновщикНастроекКомпоновкиДанных.Инициализировать_(DataCompositionSettingsComposer.Initialize).md)).  
Настроек "по умолчанию" две (они складываются). Одна подготовлена для данного объекта, а другая для добавления настроек такого объекта в конкретные родительские настройки (глобальный фильтр и параметры данных).  
Изменение настроек происходит только в том случае, если есть связь с источником доступных настроек.  
После инициализации компоновщика настроек (КомпоновщикНастроекСистемыКомпоновкиДанных) методом [Инициализировать](КомпоновщикНастроекКомпоновкиДанных.Инициализировать_(DataCompositionSettingsComposer.Initialize).md), его [Настройки](КомпоновщикНастроекКомпоновкиДанных.Настройки_(DataCompositionSettingsComposer.Settings).md) имеют связь с источником доступных настроек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
