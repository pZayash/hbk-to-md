ДиаграммаГанта (GanttChart)

РазвернутаТочка (PointExpanded)

Доступен, начиная с версии 8.0.

Синтаксис:

РазвернутаТочка(<Точка>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - точка развернута; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, развернута ли точка диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
