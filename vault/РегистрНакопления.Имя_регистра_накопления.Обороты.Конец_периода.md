Конец периода (необязательный)

Тип параметра: Дата, [МоментВремени](Прикладные_объекты.МоментВремени.md), [Граница](Граница.md).   
Указывается конец периода расчета оборотов. Значение этого параметра по умолчанию включается в период расчета оборотов. Для исключения значения из периода его следует задавать объектом [Граница](Граница.md) с исключением граничного значения.  
Если параметр не задан, обороты рассчитываются по самую последнюю запись.
