ИнтернетСоединение (InternetConnection)

Установить (Connect)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить()

Описание:

Устанавливает соединение при подключении по коммутируемым линиям.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Интернет = Новый ИнтернетСоединение; Интернет.Установить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
