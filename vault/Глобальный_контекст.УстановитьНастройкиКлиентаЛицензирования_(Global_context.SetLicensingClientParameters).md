Глобальный контекст (Global context)

УстановитьНастройкиКлиентаЛицензирования (SetLicensingClientParameters)

Доступен, начиная с версии 8.3.7.

Синтаксис:

УстановитьНастройкиКлиентаЛицензирования(<Имя>, <Пароль>, <ДополнительныйПараметр>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя клиента лицензирования.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль клиента лицензирования.

<ДополнительныйПараметр> (необязательный)

Тип: [Строка](lang__def_String.md).   
Дополнительный параметр клиента лицензирования, используемый для технической поддержки конфигурации.

Описание:

Предназначен для сохранения в информационной базе имени пользователя и пароля клиента лицензирования. Эти данные используются при проверке лицензионности конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только пользователю с административными правами.

Использование в версии:

Доступен, начиная с версии 8.3.7.
