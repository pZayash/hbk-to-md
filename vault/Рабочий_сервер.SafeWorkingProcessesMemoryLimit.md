Рабочий сервер (IWorkingServerInfo)

SafeWorkingProcessesMemoryLimit (SafeWorkingProcessesMemoryLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет размер максимального объема памяти рабочих процессов.  
Суммарный объем виртуального адресного пространства, занимаемого всеми рабочими процессами этого кластера на этом рабочем сервере, превышение которого может привести к снижению производительности рабочего сервера.  
Допустимые значения от -1 до 9223372036854775807.  
Если значение 0, то максимальный объем памяти рабочих процессов определяется автоматически как 80% объема оперативной памяти рабочего сервера.  
Если значение -1, то максимальный объем памяти рабочих процессов не ограничен.

Доступность:

Интеграция.

[Рабочий сервер](Рабочий_сервер.md), свойство [SafeCallMemoryLimit](Рабочий_сервер.SafeCallMemoryLimit.md)

Доступен, начиная с версии 8.3.1.
