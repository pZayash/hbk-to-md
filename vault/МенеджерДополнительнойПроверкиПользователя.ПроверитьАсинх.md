МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

ПроверитьАсинх (VerifyAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьАсинх(<Способ>, <СообщениеПользователю>)

Параметры:

<Способ> (обязательный)

Тип: [СпособДополнительнойПроверкиПользователя](СпособДополнительнойПроверкиПользователя.md).   
Способ проверки пользователя.

<СообщениеПользователю> (обязательный)

Тип: Строка.   
Текст сообщения, отображаемый пользователю при проверке доступа.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - тип Булево. Признак того, что проверка пользователя была отменена пользователем.  
При возникновении ошибки вызывается обработчик ошибки.

Описание:

Асинхронная функция. Начинает проверку дополнительную проверку пользователя указанным способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерДополнительнойПроверкиПользователя](МенеджерДополнительнойПроверкиПользователя.md), метод [НачатьПроверку](МенеджерДополнительнойПроверкиПользователя.НачатьПроверку.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
