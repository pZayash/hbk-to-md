ПараметрыПолученияАрхиваФайлов (GetFilesArchiveParameters)

По имени архива и режиму

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ПараметрыПолученияАрхиваФайлов(<ИмяАрхива>, <Режим>)

Параметры:

<ИмяАрхива> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя архива.

<Режим> (необязательный)

Тип: [РежимПолученияАрхиваФайлов](РежимПолученияАрхиваФайлов_(GetFilesArchiveMode).md).   
Режим получения архива файлов.

Использование в версии:

Доступен, начиная с версии 8.3.15.
