# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередСозданиемВложенныхБизнесПроцессов

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередСозданиемВложенныхБизнесПроцессов (BeforeCreateSubBusinessProcesses)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСозданиемВложенныхБизнесПроцессов(<ТочкаМаршрута>, <ФормируемыеБизнесПроцессы>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, на которой происходит создание вложенных бизнес-процессов.

<ФормируемыеБизнесПроцессы>

Тип: [Массив](Массив_(Array).md).   
Массив формируемых бизнес-процессов.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления вложенных бизнес-процессов. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись и старт вложенных бизнес-процессов произведен не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Служит для создания собственного набора бизнес-процессов, которые будут созданы и запущены как вложенные. Для этого их нужно добавить в массив <ФормируемыеБизнесПроцессы>. Если же после выхода из обработчика массив будет пустым, то платформа самостоятельно сформирует набор вложенных бизнес-процессов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [ВложенныйБизнесПроцесс](ВидТочкиМаршрутаБизнесПроцесса.ВложенныйБизнесПроцесс_(BusinessProcessRoutePointType.SubBusinessProcess).md).  
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода [Сообщить](Глобальный_контекст.Сообщить_(Global_context.Message).md).

Использование в версии:

Доступен, начиная с версии 8.0.
