ЧтениеУзловDOM (DOMNodeReader)

ПервыйАтрибут (FirstAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

ПервыйАтрибут()

Возвращаемое значение:

Тип: Булево.   
Истина - атрибут прочитан, Ложь - в противном случае.

Описание:

Позиционирует текущее состояние объекта на первый атрибут текущего элемента.   
При этом свойствам [ЧтениеУзловDOM.ТипУзла](ЧтениеУзловDOM.ТипУзла.md), [ЧтениеУзловDOM.Имя](ЧтениеУзловDOM.Имя.md), [ЧтениеУзловDOM.ЛокальноеИмя](ЧтениеУзловDOM.ЛокальноеИмя.md), [ЧтениеУзловDOM.Префикс](ЧтениеУзловDOM.Префикс.md), [ЧтениеУзловDOM.URIПространстваИмен](ЧтениеУзловDOM.URIПространстваИмен.md), [ЧтениеУзловDOM.Значение](ЧтениеУзловDOM.Значение.md) и т.д. присваиваются значения, соответствующие прочитанным данным атрибута.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл, если текущим узлом является [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md) или [Атрибут](ТипУзлаXML.Атрибут.md).

Использование в версии:

Доступен, начиная с версии 8.1.
