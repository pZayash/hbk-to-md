МенеджерПолученияЛицензий (LicenseAcquisitionManager)

ПолучитьЛицензионноеСоглашениеРазработчика (GetDeveloperLicenseAgreement)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьЛицензионноеСоглашениеРазработчика(<ПорталРазработчиков>)

Параметры:

<ПорталРазработчиков> (необязательный)

Тип: Строка.   
Может быть указано одно из значений:

- Пустая строка или не указан - язык текста соглашения определяется в соответствии с текущим языком системы того сеанса, в котором вызван метод,
- "developer.1c.ru" - возвращается текст соглашения на русском языке,
- "1c-dn.com" - возвращается текст соглашения на английском языке.

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает текст лицензионного соглашения лицензии для разработчиков.  
Текст требуется отобразить пользователю перед активацией лицензии для разработчиков.

Доступность:

Тонкий клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.27.
