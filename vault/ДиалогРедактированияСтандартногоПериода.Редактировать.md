ДиалогРедактированияСтандартногоПериода (StandardPeriodEditDialog)

Редактировать (Edit)

Доступен, начиная с версии 8.2.

Синтаксис:

Редактировать()

Возвращаемое значение:

Тип: Булево.   
Истина - была нажата кнопка "ОК", Ложь - в противном случае.

Описание:

Открывает диалог редактирования.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [Показать](ДиалогРедактированияСтандартногоПериода.Показать.md).
