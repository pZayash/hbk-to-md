ТипДвустороннейПечати (DuplexPrintingType)

ПереворотВлево (FlipPagesLeft)

Доступен, начиная с версии 8.3.1.

Описание:

Использовать двустороннюю печать таким образом, чтобы страницу нужно было переворачивать влево (переворот выполняется относительно вертикальной оси листа).

Использование в версии:

Доступен, начиная с версии 8.3.1.
