ИнтернетПочтовоеСообщение (InternetMailMessage)

ОбратныйАдрес (ReplyTo)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ИнтернетПочтовыеАдреса](ИнтернетПочтовыеАдреса_(InternetMailAddresses).md).   
Содержит коллекцию объектов [ИнтернетПочтовыйАдрес](ИнтернетПочтовыйАдрес_(InternetMailAddress).md). Используется в случае, когда адресант хочет получить ответ от адресата на другой адрес, отличный от того, с которого отправляется письмо.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
