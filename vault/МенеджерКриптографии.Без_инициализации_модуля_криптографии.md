МенеджерКриптографии (CryptoManager)

Без инициализации модуля криптографии

Доступен, начиная с версии 8.3.24.

Синтаксис:

Новый МенеджерКриптографии(<ИспользованиеИнтерактивногоРежима>)

Параметры:

<ИспользованиеИнтерактивногоРежима> (обязательный)

Определяет режим работы модуля криптографии.

Описание:

Создает объект без инициализации модуля криптографии.

Использование в версии:

Доступен, начиная с версии 8.3.24.
