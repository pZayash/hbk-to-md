# ЗащищенноеСоединениеNSS

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Интернет](Интернет-2.md) › [Сертификаты защищенных соединений](Сертификаты_защищенных_соединений.md)

Не рекомендуется использовать, начиная с версии 8.3.8.

Рекомендуется использовать:

- [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md)

Доступен, начиная с версии 8.3.1.

Описание:

Объект защищенного соединения NSS.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| nss1 = Новый ЗащищенноеСоединениеNSS(                 "c:/nss/nssdb","password", Истина, "user1");  nss2 = Новый ЗащищенноеСоединениеNSS(     "C:/Users/usrnm/AppData/Roaming/Mozilla/Firefox/Profiles/usrnm"); |

Использование в версии:

Доступен, начиная с версии 8.3.1.

Не рекомендуется использовать, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
