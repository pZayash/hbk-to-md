Расширение поля ввода для описания типа (Text box for type description extension)

ТипыВВидеДерева (TypesAsTree)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Устанавливает способ отображения типов - в виде дерева или в виде списка.  
Если Истина, то при редактировании типа доступные типы будут представлены в виде дерева, иначе - в виде списка.  
По умолчанию указывается способ отображения в виде списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
