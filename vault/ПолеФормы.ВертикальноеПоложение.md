ПолеФормы (FormField)

ВертикальноеПоложение (VerticalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложениеЭлемента](ВертикальноеПоложениеЭлемента.md).   
Управляет вертикальным положением поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Используется для полей надписи, флажка и переключателя.

Использование в версии:

Доступен, начиная с версии 8.2.
