Описание:

Предназначен для получения и изменения настроек проверки раскрытия пароля:

- Если проверка выполняется при аутентификации:
  - Пользователь выполняет вход в базу,
  - Выполняется запись в журнал регистрации;
- Если проверка выполняется при помощи метода [ПроверитьСоответствиеПароляПолитике](МенеджерПолитикПаролейПользователей.ПроверитьСоответствиеПароляПолитике_(UserPasswordPolicyManager.CheckPasswordComplianceWithPolicy).md), исключение не генерируется;
- Если проверка выполняется при помощи метода глобального контекста [ПроверитьРаскрытиеПароля](Глобальный_контекст.ПроверитьРаскрытиеПароля_(Global_context.CheckPasswordCompromise).md), генерируется исключение соединения с сервисом проверки раскрытия пароля;
- Если проверка выполняется при установке нового пароля пользователю:
  - Не влияет на установку пароля,
  - Выполняется запись в журнал регистрации;
- Если свойству [ТаймаутЗапросаСервисаПроверкиРаскрытияПароля](НастройкиПроверкиРаскрытияПароля.ТаймаутЗапросаСервисаПроверкиРаскрытияПароля_(PasswordCompromiseCheckSettings.PasswordCompromiseCheckServiceRequestTimeout).md) установлено значение 0:
  - Ожидается ответ от сервера,
  - При ошибке соединения в любом сценарии проверки генерируется исключение.

Доступность:

Сервер, толстый клиент.

Пример:
