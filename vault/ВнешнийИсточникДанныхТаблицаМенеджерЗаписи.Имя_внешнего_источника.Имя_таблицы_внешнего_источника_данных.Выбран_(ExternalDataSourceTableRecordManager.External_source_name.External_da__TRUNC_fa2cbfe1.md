ВнешнийИсточникДанныхТаблицаМенеджерЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordManager.<External source name>.<External data source table name>)

Выбран (Selected)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Выбран()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - запись выбрана, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, считана запись или нет.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если Не Запись.Выбран() Тогда     Предупреждение("Запись не считана");     Предупреждение(НСтр("ru = 'Запись не считана';"      + " en = 'Record is not read.'")); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.3.3.
