<!-- signature:end -->

Расширение табличного поля журнала документов (Document journal table box extension)

ПередОтменойПроведения (BeforeUndoPosting)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередОтменойПроведения(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отмены проведения документа. Если в теле процедуры-обработчика установить данному параметру значение Истина, то отмена проведения документа выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед интерактивной отменой проведения документа в списке.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
