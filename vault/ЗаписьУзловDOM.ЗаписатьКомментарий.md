ЗаписьУзловDOM (DOMNodeWriter)

ЗаписатьКомментарий (WriteComment)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьКомментарий(<Комментарий>)

Параметры:

<Комментарий> (обязательный)

Тип: Строка.   
Текст комментария, помещаемый в XML-документ.

Описание:

Записывает XML-комментарий.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запись.ЗаписатьКомментарий("Это же комментарий!!!"); |

Использование в версии:

Доступен, начиная с версии 8.1.
