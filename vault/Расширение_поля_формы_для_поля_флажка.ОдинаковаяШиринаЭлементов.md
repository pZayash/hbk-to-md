Расширение поля формы для поля флажка (Form field extension for a check box field)

ОдинаковаяШиринаЭлементов (EqualItemsWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Определяет одинаковую ширину элементов.   
Неопределено трактуется как Истина.   
Если итоговое значение отлично от Истина, свойство [ШиринаЭлемента](Расширение_поля_формы_для_поля_флажка.ШиринаЭлемента.md) не используется.   
Данное свойство используется только если [ВидФлажка](ВидФлажка.md) = [Тумблер](ВидФлажка.Тумблер.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
