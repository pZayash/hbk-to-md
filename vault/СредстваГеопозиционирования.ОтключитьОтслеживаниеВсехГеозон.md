СредстваГеопозиционирования (LocationTools)

ОтключитьОтслеживаниеВсехГеозон (DisableAllGeofencesMonitoring)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОтключитьОтслеживаниеВсехГеозон()

Описание:

Отключает отслеживание всех геозон для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
