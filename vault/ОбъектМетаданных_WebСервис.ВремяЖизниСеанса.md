ОбъектМетаданных: WebСервис (MetadataObject: WebService)

ВремяЖизниСеанса (SessionMaxAge)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: Число.   
Время бездействия сеанса (в секундах), после которого он завершается. Действует в случае, если свойство &РежимПовторногоИспользованияСеансов установлено в [Использовать](РежимПовторногоИспользованияСеансов.Использовать.md) или [ИспользоватьАвтоматически](РежимПовторногоИспользованияСеансов.ИспользоватьАвтоматически.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
