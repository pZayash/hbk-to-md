# ПараметрыСеанса

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Параметры сеанса](Параметры_сеанса.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

Произвольный  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры сеанса.  

Свойства:

[\<Имя параметра сеанса\> (\<Session parameter name\>)](ПараметрыСеанса.Имя_параметра_сеанса_(SessionParameters.Session_parameter_name).md)  

Методы:

[Очистить (Clear)](ПараметрыСеанса.Очистить_(SessionParameters.Clear).md)  

Описание:

Является значением свойства глобального контекста [ПараметрыСеанса](Глобальный_контекст.ПараметрыСеанса_(Global_context.SessionParameters).md) и служит для доступа к значениям параметров сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| // Установка значения параметра сеанса.  ПараметрыСеанса.ТекущийОтдел = "Производство"; // Получение значения параметра сеанса.  МойОтдел = ПараметрыСеанса.ТекущийОтдел; // Обращение к параметру сеанса из ограничения доступа к данным.  ГДЕ Отдел = &ТекущийОтдел |

Доступен, начиная с версии 8.0.
