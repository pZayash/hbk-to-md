Соединение с агентом сервера (IServerAgentConnection)

UnregSecurityProfile (UnregSecurityProfile)

Доступен, начиная с версии 8.3.3.

Синтаксис:

UnregSecurityProfile(<Кластер>, <ИмяПрофиля>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, в котором должна быть отменена регистрация профиля безопасности.

<ИмяПрофиля> (обязательный)

Тип: Строка.   
Имя профиля безопасности, регистрация которого должна быть отменена.

Описание:

Отменяет регистрацию профиля безопасности кластера. Все объекты (виртуальные каталоги, [COM-класс](COM-класс.md), внешние компоненты, приложения, адреса Интернета, внешние модули), связанные с этим профилем, удаляются.

Доступность:

Интеграция.

Примечание:

Если профиля, указанного в параметре <ИмяПрофиля>, нет, то метод ничего не делает.

Использование в версии:

Доступен, начиная с версии 8.3.3.
