ДиалогВыбораФайла (FileDialog)

Разделы (Sections)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md), [Неопределено](lang__def_Undefined.md).   
Массив элементов перечисления типа [РазделДиалогаВыбораФайла](РазделДиалогаВыбораФайла_(FileDialogSection).md). Значение по умолчанию [Неопределено](lang__def_Undefined.md). По умолчанию доступны все разделы в порядке сортировки (Недавние, Галерея, Аудио, Документы,Файлы).  
  
Используется только в мобильной платформе для iOS и Android.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.21.
