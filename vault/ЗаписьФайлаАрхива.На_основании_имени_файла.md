ЗаписьФайлаАрхива (ArchiveFileWriter)

На основании имени файла

Доступен, начиная с версии 8.3.26.

Синтаксис:

Новый ЗаписьФайлаАрхива(<ИмяФайла>, <Пароль>, <ТипФайлаАрхива>, <Комментарий>, <МетодСжатия>, <УровеньСжатия>, <МетодШифрования>, <Кодировка>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, куда будет записан архив.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль, который будет назначен архиву.   
Если пароль не назначен или является пустой строкой, то шифрование не происходит.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива_(ArchiveFileType).md).   
Тип файла архива, который требуется записать. По умолчанию имеет значение [ZIP](ТипФайлаАрхива.ZIP_(ArchiveFileType.ZIP).md).   
Примечание. Допускается только использование значения [ZIP](ТипФайлаАрхива.ZIP_(ArchiveFileType.ZIP).md).  
Значение по умолчанию: [ZIP](ТипФайлаАрхива.ZIP_(ArchiveFileType.ZIP).md).

<Комментарий> (необязательный)

Тип: [Строка](lang__def_String.md).   
Комментарий, описывающий файл архива.

<МетодСжатия> (необязательный)

Тип: [МетодСжатияФайлаАрхива](МетодСжатияФайлаАрхива_(ArchiveFileCompressionMethod).md).   
Метод сжатия, которым будет сжиматься архив.  
Значение по умолчанию: [Сжатие](МетодСжатияФайлаАрхива.Сжатие_(ArchiveFileCompressionMethod.Deflate).md).

<УровеньСжатия> (необязательный)

Тип: [УровеньСжатияФайлаАрхива](УровеньСжатияФайлаАрхива_(ArchiveFileCompressionLevel).md).   
Уровень сжатия данных.  
Значение по умолчанию: [Оптимальный](УровеньСжатияФайлаАрхива.Оптимальный_(ArchiveFileCompressionLevel.Optimal).md).

<МетодШифрования> (необязательный)

Тип: [МетодШифрованияФайлаАрхива](МетодШифрованияФайлаАрхива_(ArchiveFileEncryptionMethod).md).   
Метод шифрования, которым будет зашифрован архив.  
Значение по умолчанию: [Zip20](МетодШифрованияФайлаАрхива.Zip20_(ArchiveFileEncryptionMethod.Zip20).md).

<Кодировка> (необязательный)

Тип: [КодировкаИменФайловВФайлеАрхива](КодировкаИменФайловВФайлеАрхива_(FileNamesEncodingInArchiveFile).md).   
Определяет способ кодировки имен файлов внутри файла архива.  
Значение по умолчанию: [Авто](КодировкаИменФайловВФайлеАрхива.Авто_(FileNamesEncodingInArchiveFile.Auto).md), [КодировкаИменФайловВФайлеАрхива](КодировкаИменФайловВФайлеАрхива_(FileNamesEncodingInArchiveFile).md).

Описание:

Открывает файл архива для записи. Аналогичен методу [Открыть](ЗаписьФайлаАрхива.Открыть_(ArchiveFileWriter.Open).md).

Использование в версии:

Доступен, начиная с версии 8.3.26.
