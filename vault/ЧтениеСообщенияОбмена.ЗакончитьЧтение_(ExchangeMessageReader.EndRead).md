ЧтениеСообщенияОбмена (ExchangeMessageReader)

ЗакончитьЧтение (EndRead)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗакончитьЧтение()

Описание:

Вызывает попытку нормального завершения чтения сообщения. При этом проверяется нормальное завершение сообщения. Если обнаружен конец элемента XML, соответствующего телу сообщения, то в случае превышения номером сообщения максимального номера принятого сообщения, он помещается в реквизит [НомерПринятого](ПланОбменаОбъект.Имя_плана_обмена.НомерПринятого_(ExchangePlanObject.Exchange_plan_name.ReceivedNo).md) узла плана обмена. Блокировка записи узла плана обмена снимается и сообщение считается принятым.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для нормального завершения чтения требуется явное обращение к методу [ЗакончитьЧтение](ЧтениеСообщенияОбмена.ЗакончитьЧтение_(ExchangeMessageReader.EndRead).md). Если при чтении сообщения объект [ЧтениеСообщенияОбмена](ЧтениеСообщенияОбмена_(ExchangeMessageReader).md) был освобожден или произошло повторное обращение к методу [НачатьЧтение](ЧтениеСообщенияОбмена.НачатьЧтение_(ExchangeMessageReader.BeginRead).md), то предыдущее чтение будет прервано через неявное обращение к методу [ПрерватьЧтение](ЧтениеСообщенияОбмена.ПрерватьЧтение_(ExchangeMessageReader.CancelRead).md).

Пример:

|  |
| --- |
| ЧтениеСообщения = ПланыОбмена.СоздатьЧтениеСообщения(); ЧтениеXML = Новый ЧтениеXML; ЧтениеXML.ОткрытьФайл(ИмяФайла); ЧтениеСообщения.НачатьЧтение(ЧтениеXML); // Действия по чтению тела сообщения  ЧтениеСообщения.ЗакончитьЧтение(); |

Использование в версии:

Доступен, начиная с версии 8.0.
