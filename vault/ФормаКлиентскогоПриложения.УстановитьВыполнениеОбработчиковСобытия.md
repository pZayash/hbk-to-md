ФормаКлиентскогоПриложения (ClientApplicationForm)

УстановитьВыполнениеОбработчиковСобытия (SetEventHandlersExecution)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьВыполнениеОбработчиковСобытия(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: Булево.   
Если установлено Ложь - после завершения текущего обработчика выполнение цепочки обработчиков будет прервано.  
Если установлено Истина - выполнение цепочки обработчиков будет продолжено.

Описание:

Управляет выполнением цепочки обработчиков события.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Метод может быть вызван только в формах конфигураций расширений.

Использование в версии:

Доступен, начиная с версии 8.3.6.
