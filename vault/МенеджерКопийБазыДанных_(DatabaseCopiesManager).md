# МенеджерКопийБазыДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Копии базы данных](Копии_базы_данных-2.md)

Доступен, начиная с версии 8.3.14.

Элементы коллекции:

[МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются объекты типа [МенеджерКопииБазыДанных](МенеджерКопииБазыДанных_(DatabaseCopyManager).md).  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается имя копии.

Описание:

Предназначен для взаимодействия с копиями базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Копия1 = КопииБазыДанных.Добавить("Копия1", ТипРепликацииКопииБазыДанных.Стандартная, ТипСУБДКопииБазыДанных.PostgreSQL, "localhost:5432", "DbCopy1", "postgres", "postgres");  Копия1.Состав.Добавить(Метаданные.Справочники.Товары); Копия1.Состав.Добавить(Метаданные.Документы.РасходТовара); Копия1.Состав.Добавить(Метаданные.РегистрыНакопления.ТоварныеЗапасы); Копия1.Записать();  КопииБазыДанных.Обновить(Копия1); |

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.27.
