ИнтернетПочтовоеСообщение (InternetMailMessage)

Заголовок (Header)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит заголовок сообщения в том виде, в котором письмо хранится на сервере.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
