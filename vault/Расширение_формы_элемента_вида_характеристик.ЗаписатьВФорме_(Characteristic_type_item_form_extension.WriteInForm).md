# Расширение формы элемента вида характеристик.ЗаписатьВФорме

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [Расширение формы элемента вида характеристик](Расширение_формы_элемента_вида_характеристик_(Characteristic_type_item_form_extension).md) › [Методы](Расширение_формы_элемента_вида_характеристик__Методы.md)

Расширение формы элемента вида характеристик (Characteristic type item form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Объект записан."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
