ТестируемаяТаблицаФормы (TestedFormTable)

ЗакончитьРедактированиеСтроки (EndEditRow)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ЗакончитьРедактированиеСтроки(<ОтменаРедактирования>)

Параметры:

<ОтменаРедактирования> (необязательный)

Тип: Булево.   
Признак отмены редактирования.  
Значение по умолчанию: Ложь.

Описание:

Завершает режим редактирования в текущей колонке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
