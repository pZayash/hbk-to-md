Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСКриптографиейАсинх (AttachCryptoExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПодключитьРасширениеРаботыСКриптографиейАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
В случае успешного подключения результат выполнения обещания - Истина, иначе - Ложь.  
Для тонкого и толстого клиентов, а также для клиента мобильного приложения результат всегда Истина.

Описание:

Асинхронная функция. При первом обращении выполняет подключение расширения для работы с криптографией.  
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
