ТипДиаграммы (ChartType)

Точечная (Scatter)

Доступен, начиная с версии 8.3.10.

Описание:

Точечная диаграмма. Строится на основании данных, представленных одной или несколькими сериями значений для нескольких точек.

Использование в версии:

Доступен, начиная с версии 8.3.10.
