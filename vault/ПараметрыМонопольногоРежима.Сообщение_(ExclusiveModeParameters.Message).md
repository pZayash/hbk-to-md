ПараметрыМонопольногоРежима (ExclusiveModeParameters)

Сообщение (Message)

Доступен, начиная с версии 8.3.19.

Использование:

Чтение и запись.

Описание:

Тип: [Неопределено](lang__def_Undefined.md), [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст сообщения, которое будет отображено пользователю при попытке входа в информационную базу или область данных, на которую установлен режим монопольного доступа.   
По умолчанию - [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
