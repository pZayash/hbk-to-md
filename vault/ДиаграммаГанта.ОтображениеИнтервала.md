ДиаграммаГанта (GanttChart)

ОтображениеИнтервала (IntervalRepresentation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеИнтервалаДиаграммыГанта](ОтображениеИнтервалаДиаграммыГанта.md).   
Указывает вариант отображения интервала диаграммы Ганта.  
В режиме совместимости [Версия8\_3\_22](РежимСовместимости.Версия8_3_22.md) и ниже используется значение [Градиент](ОтображениеИнтервалаДиаграммыГанта.Градиент.md).  
В режиме совместимости выше [Версия8\_3\_22](РежимСовместимости.Версия8_3_22.md) используется значение [Плоский](ОтображениеИнтервалаДиаграммыГанта.Плоский.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.23.
