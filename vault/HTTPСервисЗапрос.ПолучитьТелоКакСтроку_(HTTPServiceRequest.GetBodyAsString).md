HTTPСервисЗапрос (HTTPServiceRequest)

ПолучитьТелоКакСтроку (GetBodyAsString)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьТелоКакСтроку(<Кодировка>)

Параметры:

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Указывает кодировку, в которой должно интерпретироваться тело полученного запроса.   
Если не задан, то кодировка определяется из HTTP-заголовка "Content-Type".  
Если кодировка не указана в заголовке, то используется [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые названия кодировок:

- UTF-16
- UTF-16BE
- UTF-16LE
- ISO-8859-1
- US-ASCII
- gb18030
- iso-8859-2
- iso-8859-3

- EUC-JP

- ISO-2022-JP
- ISO-2022-KR
- ISO-2022-CN
- ISO-2022-CN-EXT
- HZ-GB-2312

.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает тело запроса в виде строки.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
