# ПланСчетовОбъект.<Имя плана счетов>.Ссылка

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Ссылка (Ref)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   
Содержит ссылку на счет. Это значение может быть записано в базу данных для полей соответствующего типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Например, данное свойство можно использовать, если ссылку на созданный и записанный счет нужно присвоить реквизиту другого счета.

Использование в версии:

Доступен, начиная с версии 8.0.
