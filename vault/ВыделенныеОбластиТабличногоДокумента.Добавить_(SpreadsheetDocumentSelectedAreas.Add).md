ВыделенныеОбластиТабличногоДокумента (SpreadsheetDocumentSelectedAreas)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Область>)

Параметры:

<Область> (обязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md), [РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md).   
Область, добавляемая в коллекцию выделенных областей.

Описание:

Добавляет область ячеек или рисунок в коллекцию выделенных областей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
