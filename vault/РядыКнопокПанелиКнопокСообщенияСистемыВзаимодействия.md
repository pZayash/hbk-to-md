Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются ряд кнопок панели кнопок сообщения системы взаимодействия.  
Возможно обращение к элементам коллекции посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Предоставляет доступ к методам коллекции рядов кнопок панели кнопок сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
