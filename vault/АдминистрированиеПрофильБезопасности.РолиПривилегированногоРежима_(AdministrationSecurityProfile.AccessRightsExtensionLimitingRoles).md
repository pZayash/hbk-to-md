АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

РолиПривилегированногоРежима (AccessRightsExtensionLimitingRoles)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Список ролей через ";", использующихся во время активности привилегированного режима вместо ролей пользователя, если включен безопасный режим и для активного профиля безопасности не установлен флаг [ПолныйПривилегированныйРежим](АдминистрированиеПрофильБезопасности.ПолныйПривилегированныйРежим_(AdministrationSecurityProfile.FullPrivilegedMode).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.21.
