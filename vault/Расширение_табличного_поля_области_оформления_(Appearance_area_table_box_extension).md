Свойства:

[НастройкаОбластиОформления (AppearanceAreaSetting)](Расширение_табличного_поля_области_оформления.НастройкаОбластиОформления_(Appearance_area_table_box_extension.AppearanceAreaSetting).md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md) для области оформления. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) возвращает значение типа [ЭлементОбластиОформления](ЭлементОбластиОформления_(AppearanceAreaItem).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
