Расширение поля формы для поля ввода (Form field extension for a text box)

КнопкаВыпадающегоСписка (DropListButton)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Определяет, будет ли показываться кнопка открытия выпадающего списка. Истина - кнопка показывается.  
Если режим совместимости не используется или выше [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md) и значение равно Неопределено, то кнопка отображается у полей ввода, редактирующих значения следующих типов:

- Ссылочный тип, в том числе перечисление;
- Системное перечисление;
- Булево;
- [СтандартнаяДатаНачала](СтандартнаяДатаНачала.md);
- [СтандартныйПериод](СтандартныйПериод.md);
- Любого типа, если свойство [РежимВыбораИзСписка](Расширение_поля_формы_для_поля_ввода.РежимВыбораИзСписка.md) имеет значение Истина.

При установленном режиме совместимости [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md) и ниже значение Неопределено игнорируется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
