МенеджерКриптографии (CryptoManager)

ПолучитьИнформациюМодуляКриптографии (GetCryptoModuleInformation)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИнформациюМодуляКриптографии()

Возвращаемое значение:

Тип: [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии_(CryptoModuleInformation).md).   

Описание:

Получает информацию о модуле криптографии, если была выполнена инициализация объекта.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Доступен, начиная с версии 8.2.
