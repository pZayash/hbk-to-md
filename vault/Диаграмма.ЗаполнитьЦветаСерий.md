Диаграмма (Chart)

ЗаполнитьЦветаСерий (FillSeriesColors)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ЗаполнитьЦветаСерий(<Оформление>)

Параметры:

<Оформление> (необязательный)

Тип: [ОформлениеЗначений](ОформлениеЗначений.md).   
Содержит информацию о цветах серий.

Описание:

Для всех цветов серий заменяет автоцвета серий на абсолютный цвет.  
Если параметр <Оформление> не передан, то цвета будут получены из палитры диаграммы. Если же параметр <Оформление> передан, то меняет цвета в следующем порядке:

- Для всех серий получается ключ серии: свойство серии [Значение](СерияДиаграммы.Значение.md), если оно установлено, либо свойство [Текст](СерияДиаграммы.Текст.md). У переданного параметра вызывается метод [Установить](ОформлениеЗначений.Установить.md) для каждого ключа. Таким образом добавляются все ключи серий.
- Для каждой серии получается ключ. Для ключа у переданного параметра вызывается метод [ПолучитьВсе](ОформлениеЗначений.ПолучитьВсе.md). В полученном соответствии ищутся и сохраняются все цвета.
- Из текущей палитры на время работы этого метода удаляются абсолютные цвета, полученные на предыдущем шаге.
- Для каждой серии получается ключ. У переданного параметра вызывается метод [Получить](ОформлениеЗначений.Получить.md) для этого ключа:
  - Если метод возвращает [Цвет](Цвет.md), то этот цвет устанавливается для серии.
  - Если метод возвращает Число, то по полученному индексу получается цвет из палитры и устанавливается для серии.

Доступность:

Сервер, толстый клиент.

Примечание:

Если в одной диаграмме для серий используется один цвет (количество ключей больше количества цветов в палитре), не выполняет никаких действий.

Использование в версии:

Доступен, начиная с версии 8.3.16.
