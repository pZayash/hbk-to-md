`ЕстьАтрибут`(`<Имя>`: `Строка`)

`ЕстьАтрибут`(`<URIПространстваИмен>`: `Строка`, `<ЛокальноеИмя>`: `Строка`) → `Булево`
<!-- signature:end -->

ЭлементDOM (DOMElement)

ЕстьАтрибут (HasAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ЕстьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя атрибута.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ЕстьАтрибут(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен.

<ЛокальноеИмя> (обязательный)

Тип: Строка.   
Локальное имя атрибута.

Возвращаемое значение:

Тип: Булево.   
Истина, если атрибут с указанным именем явно присутствует в элементе, или имеет значение по умолчанию; Ложь - в противном случае.

Описание:

Определяет наличие атрибута с указанным именем в данном элементе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
