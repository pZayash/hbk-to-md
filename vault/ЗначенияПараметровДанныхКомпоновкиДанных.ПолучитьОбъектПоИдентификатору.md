ЗначенияПараметровДанныхКомпоновкиДанных (DataCompositionDataParameterValues)

ПолучитьОбъектПоИдентификатору (GetObjectByID)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОбъектПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных.md).   
Идентификатор объекта.

Возвращаемое значение:

Тип: [ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных.md), Неопределено.   
Неопределено - объект, для которого был передан идентификатор, не найден или если он не является частью данного объекта.

Описание:

Получает значения параметров данных компоновки данных, соответствующие переданному идентификатору.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
