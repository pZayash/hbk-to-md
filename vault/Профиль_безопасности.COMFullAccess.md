Профиль безопасности (ISecurityProfile)

COMFullAccess (COMFullAccess)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешено использование произвольных COM-объектов.  
Истина – разрешено использование любых COM-объектов. Ложь – допустимо использование только COM-объектов, перечисленных в списке разрешенных COM-объектов профиля безопасности.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
