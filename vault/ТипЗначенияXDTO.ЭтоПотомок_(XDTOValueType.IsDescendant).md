ТипЗначенияXDTO (XDTOValueType)

ЭтоПотомок (IsDescendant)

Доступен, начиная с версии 8.1.

Синтаксис:

ЭтоПотомок(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO_(XDTOValueType).md).   
Анализируемый тип.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - переданный тип является потомком данного, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, является ли переданный тип значения потомком данного типа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
