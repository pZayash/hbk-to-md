Описание:

На основе данного объекта метаданных может быть сгенерирован новый тип данных. Сгенерированный тип данных может быть использован в качестве типа других данных (реквизиты, измерения, и т.д.).  
**Внимание!** Сгенерированный тип нельзя использовать в качестве типа данных для плана видов характеристик.

Доступность:

Сервер, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
