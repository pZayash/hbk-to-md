ЧтениеJSON (JSONReader)

ТекущееЗначение (CurrentValue)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: Число, Строка, Булево, Неопределено.   
Содержит текущее значение:

- Число - если [ТипТекущегоЗначения](ЧтениеJSON.ТипТекущегоЗначения.md) имеет значение [Число](ТипЗначенияJSON.Число.md);
- Строка - если [ТипТекущегоЗначения](ЧтениеJSON.ТипТекущегоЗначения.md) имеет значение:
  - [Комментарий](ТипЗначенияJSON.Комментарий.md),
  - [ИмяСвойства](ТипЗначенияJSON.ИмяСвойства.md),
  - [Строка](ТипЗначенияJSON.Строка.md);
- Булево - если [ТипТекущегоЗначения](ЧтениеJSON.ТипТекущегоЗначения.md) имеет значение [Булево](ТипЗначенияJSON.Булево.md),
- Неопределено - если [ТипТекущегоЗначения](ЧтениеJSON.ТипТекущегоЗначения.md) имеет значение [Null](ТипЗначенияJSON.Null.md).

Исключение генерируется в случае, если [ТипТекущегоЗначения](ЧтениеJSON.ТипТекущегоЗначения.md) имеет одно из следующих значений:

- [НачалоМассива](ТипЗначенияJSON.НачалоМассива.md),
- [КонецМассива](ТипЗначенияJSON.КонецМассива.md),
- [НачалоОбъекта](ТипЗначенияJSON.НачалоОбъекта.md),
- [КонецОбъекта](ТипЗначенияJSON.КонецОбъекта.md),
- [Ничего](ТипЗначенияJSON.Ничего.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
