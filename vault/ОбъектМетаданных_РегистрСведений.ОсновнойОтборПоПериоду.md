ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ОсновнойОтборПоПериоду (MainFilterOnPeriod)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то поле [Период](РегистрСведенийЗапись.Имя_регистра_сведений.Период.md)входит в основной отбор. В противном случае поле Период не включается в основной отбор.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
