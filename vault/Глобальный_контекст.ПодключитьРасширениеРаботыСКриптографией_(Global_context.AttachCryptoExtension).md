# Глобальный контекст.ПодключитьРасширениеРаботыСКриптографией

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Процедуры и функции работы с криптографией](Процедуры_и_функции_работы_с_криптографией.md)

Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСКриптографией (AttachCryptoExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [НачатьПодключениеРасширенияРаботыСКриптографией](Глобальный_контекст.НачатьПодключениеРасширенияРаботыСКриптографией_(Global_context.BeginAttachingCryptoExtension).md)

Доступен, начиная с версии 8.2.

Синтаксис:

ПодключитьРасширениеРаботыСКриптографией()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
В случае успешного подключения возвращает [Истина](lang__def_BooleanTrue.md), иначе - [Ложь](lang__def_BooleanFalse.md).  
Для тонкого и толстого клиентов, а также для клиента мобильного приложения результат всегда [Истина](lang__def_BooleanTrue.md).

Описание:

При первом обращении выполняет подключение расширения для работы с криптографией.  
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
