Сеанс (ISessionInfo)

dbProcInfo (dbProcInfo)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит номер соединения с СУБД в терминах СУБД в том случае, если в момент получения списка выполняется запрос к СУБД, открыта транзакция или определены временные таблицы (это означает, что захвачено соединение с СУБД).   
Пустая строка - соединение с СУБД не захвачено.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
