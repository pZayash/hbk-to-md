Индекс (IndexOf)

Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ПараметрСхемыКомпоновкиДанных](ПараметрСхемыКомпоновкиДанных_(DataCompositionSchemaParameter).md).

Тип: [Число](lang__def_Number.md).

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
