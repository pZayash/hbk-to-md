ФорматированныйДокумент (FormattedDocument)

ПолучитьHTML (GetHTML)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьHTML(<ТекстHTML>, <Вложения>)

Параметры:

<ТекстHTML> (обязательный)

Тип: [Строка](lang__def_String.md).   
В параметр метод помещает строку, содержащую текст HTML.

<Вложения> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
В параметр метод помещает структуру вложенных в форматированный документ рисунков.   
Ключ элемента структуры - название или имя файла рисунка, значение - [Картинка](Картинка_(Picture).md).  
Если рисунок задан через URL, то он не добавляется в структуру.

Описание:

Получает HTML-составляющие форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
