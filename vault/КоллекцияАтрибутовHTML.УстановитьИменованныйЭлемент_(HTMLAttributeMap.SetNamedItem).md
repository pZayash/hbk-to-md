КоллекцияАтрибутовHTML (HTMLAttributeMap)

УстановитьИменованныйЭлемент (SetNamedItem)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьИменованныйЭлемент(<Атрибут>)

Параметры:

<Атрибут> (обязательный)

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md).   
Атрибут HTML.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   

Описание:

Устанавливает в коллекцию атрибут HTML, используя в качестве ключа локальное имя .  
Если в коллекции присутствует атрибут с таким же локальным именем, то он будет замещен и возвращен в качестве возвращаемого значения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
