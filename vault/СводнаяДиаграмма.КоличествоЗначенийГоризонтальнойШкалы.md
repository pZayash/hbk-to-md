СводнаяДиаграмма (PivotChart)

КоличествоЗначенийГоризонтальнойШкалы (HorizontalScaleValueCount)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет количество значений, одновременно отображаемых на горизонтальной оси (оси точек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется при [ГоризонтальнаяПоддержкаМасштаба](СводнаяДиаграмма.ГоризонтальнаяПоддержкаМасштаба.md) = [КоличествоЗначений](ПоддержкаМасштабаСводнойДиаграммы.КоличествоЗначений.md).
