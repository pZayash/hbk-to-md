Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

ПолучитьНавигационнуюСсылкуСписка (GetListURL)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПолучитьНавигационнуюСсылкуСписка(<ПредставлениеНавигационнойСсылки>, <ВнешниеДанныеНавигационнойСсылки>, <ВосстанавливатьТекущуюСтроку>)

Параметры:

<ПредставлениеНавигационнойСсылки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит представление, которое будет помещено в хранилище внешних данных навигационных ссылок. Если этот параметр указан, а <ВнешниеДанныеНавигационнойСсылки> – нет, будет вызвано исключение.

<ВнешниеДанныеНавигационнойСсылки> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит значение, которое будет помещено в хранилище внешних данных навигационных ссылок. Если не указан, то внешний параметр в навигационной ссылке создаваться не будет.   
Если данный параметр указан, а <ПредставлениеНавигационнойСсылки> – нет, будет вызвано исключение.

<ВосстанавливатьТекущуюСтроку> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость сохранять настройках текущую строку. Если задан, то формируется ссылка с параметром stngs, а в хранилище помещается объект типа [НастройкиТаблицыДинамическогоСписка](НастройкиТаблицыДинамическогоСписка_(DynamicListTableSettings).md), у которого заполнено только свойство [ТекущаяСтрока](НастройкиТаблицыДинамическогоСписка.ТекущаяСтрока_(DynamicListTableSettings.CurrentRow).md).  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает ссылку для динамического списка, являющегося основным реквизитом формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
