# ОтчетОбъект.<Имя отчета>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Отчеты](Отчеты.md) › [ОтчетОбъект.<Имя отчета>](ОтчетОбъект.Имя_отчета_(ReportObject.Report_name).md) › [Методы](ОтчетОбъект.Имя_отчета__Методы.md)

ОтчетОбъект.<Имя отчета> (ReportObject.<Report name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Отчет](ОбъектМетаданных_Отчет_(MetadataObject_Report).md).   

Описание:

Предоставляет доступ к объекту описания метаданных отчета. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.Отчеты.Продажи.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = Справочники.Конкуренты.НайтиПоКоду(1); МетаданныеСправочника = Объект.Метаданные(); Сообщить("Представление справочника - " +     МетаданныеСправочника.Представление()); |

Использование в версии:

Доступен, начиная с версии 8.0.
