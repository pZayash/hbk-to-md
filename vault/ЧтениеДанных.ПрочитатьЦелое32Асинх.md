ЧтениеДанных (DataReader)

ПрочитатьЦелое32Асинх (ReadInt32Async)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьЦелое32Асинх(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов.md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - результат чтения типа Число.  
Если не удалось прочитать число из-за того, что был достигнут конец потока, результатом обещания будет значение Неопределено.

Описание:

Асинхронная функция. Считывает 32-битное целое число из потока. Результат выполнения обещания - результат чтения типа Число.

Доступность:

Тонкий клиент, толстый клиент.
