ФормаКлиентскогоПриложения (ClientApplicationForm)

АвтоПодборПользователейСистемыВзаимодействия (CollaborationSystemUsersAutoComplete)

Доступен, начиная с версии 8.3.15.

Синтаксис:

АвтоПодборПользователейСистемыВзаимодействия(<Текст>, <ДанныеВыбора>, <НазначениеВыбора>, <СтандартнаяОбработка>)

Параметры:

<Текст>

Тип: Строка.   
Строка текста, введенная в поле ввода выбора пользователя.

<ДанныеВыбора>

Тип: [СписокЗначений](СписокЗначений.md).   
В качестве значения элемента списка могут использоваться значения следующих типов:

- [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md),
- [УникальныйИдентификатор](УникальныйИдентификатор.md) - идентификатор пользователя информационной базы,
- Массив - должен содержать значения только следующих типов:
  - [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md),
  - [УникальныйИдентификатор](УникальныйИдентификатор.md) - идентификатор пользователя информационной базы,
  - Неопределено - допустимо только, если [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия.md) имеет значение [ПолучательСообщения](НазначениеВыбораПользователейСистемыВзаимодействия.ПолучательСообщения.md).

<НазначениеВыбора>

Тип: [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия.md).   
Назначение выбора пользователя системы взаимодействия.

<СтандартнаяОбработка>

Тип: Булево.   
Признак выполнения стандартной обработки.  
Если после вызова обработчика данный параметр содержит значение Ложь, то обработчик события клиентского управляемого приложения [АвтоПодборПользователейСистемыВзаимодействия](Глобальный_контекст.АвтоПодборПользователейСистемыВзаимодействия.md)не вызывается.

Описание:

Возникает на клиенте при автоподборе пользователя системы взаимодействия (получателя сообщения, участника видеозвонка).  
Данный обработчик не вызывается при добавлении участников в существующую видеоконференцию.  
При вызове обработчика в <ДанныеВыбора> передается сформированный системой список выбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
