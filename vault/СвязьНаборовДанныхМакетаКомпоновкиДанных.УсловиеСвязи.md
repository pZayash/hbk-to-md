СвязьНаборовДанныхМакетаКомпоновкиДанных (DataCompositionTemplateDataSetLink)

УсловиеСвязи (LinkConditionExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Выражение, результат которого проверяется перед осуществлением связи. Связь будет осуществляться только в случае, если результатом исполнения выражение является значение Истина. Условие связи может быть пустым. В таком случае связь будет осуществляться всегда.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
