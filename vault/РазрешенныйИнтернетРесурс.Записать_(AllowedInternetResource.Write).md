РазрешенныйИнтернетРесурс (AllowedInternetResource)

Записать (Write)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Записать()

Описание:

Выполняет регистрацию нового разрешенного ресурса сети Интернет или обновление существующего.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
