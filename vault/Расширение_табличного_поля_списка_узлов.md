Описание:

Дополнительные свойства и события элемента управления [ТабличноеПоле](ТабличноеПоле.md) для списка узлов. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) содержит значение типа [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена.md)>. Свойство [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные.md) содержит коллекцию значений данных строки, набор значений которой определяется колонками объекта [ПланОбменаСписок.](ПланОбменаСписок.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаСписок.Имя_плана_обмена.md)>.
