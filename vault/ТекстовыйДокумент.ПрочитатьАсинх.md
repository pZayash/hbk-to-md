ТекстовыйДокумент (TextDocument)

ПрочитатьАсинх (ReadAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Из файла

Синтаксис:

ПрочитатьАсинх(<ИмяФайла>, <ТипФайла>, <РазделительСтрок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя текстового файла.

<ТипФайла> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Указывается кодировка текста в открываемом файле.   
В случае, если параметр не задан, формат текста будет определен автоматически (если он хранится в файле с сигнатурой BOM в начале). Если автоматически определить формат файла не удается, файл будет открыт в кодировке [UTF8](КодировкаТекста.UTF8.md).  
  
Тип файла может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые названия кодировок:

<РазделительСтрок> (необязательный)

Тип: Строка.   
Задается строка, являющаяся разделителем строк, из которой при чтении файла будет сформирован символ [ПС](Символы.ПС.md).   
Если строка пустая, то разделители строк при чтении не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК.md);
- [ПС](Символы.ПС.md);
- [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

Описание варианта метода:

Считывает текстовый документ из файла.

ПрочитатьАсинх(<Поток>, <ТипФайла>, <РазделительСтрок>)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для чтения.
