HTTPСоединение (HTTPConnection)

Защищенное (IsSecure)

Не рекомендуется использовать, начиная с версии 8.3.1.

- [ЗащищенноеСоединение](HTTPСоединение.ЗащищенноеСоединение.md)

Использование:

Только чтение.

Описание:

Тип: Булево.   
Определяет использование протокола HTTP или HTTPS.  
Истина - HTTPS.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
