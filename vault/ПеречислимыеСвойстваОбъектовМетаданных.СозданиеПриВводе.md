ПеречислимыеСвойстваОбъектовМетаданных (MetadataObjectEnumeratedProperties)

СозданиеПриВводе (CreateOnInput)

Доступен, начиная с версии 8.3.7.

Использование:

Только чтение.

Описание:

Тип: [СозданиеПриВводе](СозданиеПриВводе.md).   
Режим создания при вводе. Если значение = [Авто](СозданиеПриВводе.Авто.md), то участие в механизме создания при вводе определяется значением свойства СозданиеПриВводе объекта метаданных этого типа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.7.
