# ПланВидовРасчетаМенеджер.<Имя плана видов расчета>.НайтиПоРеквизиту

ПланВидовРасчетаМенеджер.<Имя плана видов расчета> (ChartOfCalculationTypesManager.<Chart of calculation types name>)

НайтиПоРеквизиту (FindByAttribute)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоРеквизиту(<ИмяРеквизита>, <Значение>)

Параметры:

<ИмяРеквизита> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя реквизита, как он задан в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и строк произвольной длины.

<Значение> (обязательный)

Тип: Произвольный.   
Значение реквизита искомого вида расчета, по значению которого осуществляется поиск.

Возвращаемое значение:

Тип: [ПланВидовРасчетаСсылка.\<Имя плана видов расчета\>](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md).   
Ссылка на найденный вид расчета. Если не существует ни одного элемента с требуемым значением реквизита, то будет возвращена пустая ссылка.

Описание:

Осуществляет поиск вида расчета по значению реквизита.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если существует несколько видов расчета с указанным значением реквизита, то будет найдет первый найденный.  
Для реквизитов типа [Строка](lang__def_String.md) поиск осуществляется по точному соответствию.

Использование в версии:

Доступен, начиная с версии 8.0.
