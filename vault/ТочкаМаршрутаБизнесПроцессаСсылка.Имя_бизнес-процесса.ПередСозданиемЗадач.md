ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередСозданиемЗадач (BeforeCreateTasks)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСозданиемЗадач(<ТочкаМаршрута>, <ФормируемыеЗадачи>, <СтандартнаяОбработка>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса, на которой происходит создание задач.

<ФормируемыеЗадачи>

Тип: [Массив](Массив.md).   
Массив формируемых задач (объектов типа [ЗадачаОбъект](БиблиотекаКартинок.ЗадачаОбъект.md)).

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартного формирования списка задач производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Возникает перед выполнением стандартной механики генерации задач. Если обработчик сам формирует список задач, то должен установить <СтандартнаяОбработка> в Ложь. В противном случае созданные им в этом обработчике задачи удаляются и заново формируются стандартной процедурой создания задач. Вне зависимости от <СтандартнаяОбработка> после этого обработчика обязательно будет вызван обработчик [ПриСозданииЗадач](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПриСозданииЗадач.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md) и [ВложенныйБизнесПроцесс](ВидТочкиМаршрутаБизнесПроцесса.ВложенныйБизнесПроцесс.md).  
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода [Сообщить](Глобальный_контекст.Сообщить.md).

Использование в версии:

Доступен, начиная с версии 8.0.
