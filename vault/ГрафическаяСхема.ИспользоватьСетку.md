ГрафическаяСхема (GraphicalSchema)

ИспользоватьСетку (GridEnabled)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак использования сетки при отрисовке графической схемы.   
Истина - используется.  
Дополнительным свойством регулируется режим отрисовки сетки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
