ЗаписьZipФайла (ZipFileWriter)

Открыть (Open)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Открыть](ЗаписьФайлаАрхива.Открыть_(ArchiveFileWriter.Open).md)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть для записи в файл

Синтаксис:

Открыть(<ИмяФайла>, <Пароль>, <Комментарий>, <МетодСжатия>, <УровеньСжатия>, <МетодШифрования>, <КодировкаИменФайлов>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет записан архив.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль, который будет назначен архиву.   
Если пароль не назначен или является пустой строкой, то шифрование не происходит.

<Комментарий> (необязательный)

Комментарий, описывающий ZIP-файл.

<МетодСжатия> (необязательный)

Тип: [МетодСжатияZIP](МетодСжатияZIP_(ZIPCompressionMethod).md).   
Метод сжатия, который будет использован для сжатия архива.  
Значение по умолчанию: [Сжатие](МетодСжатияZIP.Сжатие_(ZIPCompressionMethod.Deflate).md).

<УровеньСжатия> (необязательный)

Тип: [УровеньСжатияZIP](УровеньСжатияZIP_(ZIPCompressionLevel).md).   
Уровень сжатия данных.  
Значение по умолчанию: [Оптимальный](УровеньСжатияZIP.Оптимальный_(ZIPCompressionLevel.Optimal).md).

<МетодШифрования> (необязательный)

Тип: [МетодШифрованияZIP](МетодШифрованияZIP_(ZIPEncryptionMethod).md).   
Метод шифрования, которым будет зашифрован архив.  
Значение по умолчанию: [Zip20](МетодШифрованияZIP.Zip20_(ZIPEncryptionMethod.Zip20).md).

<КодировкаИменФайлов> (необязательный)

Тип: [КодировкаИменФайловВZipФайле](КодировкаИменФайловВZipФайле_(FileNamesEncodingInZipFile).md).   
Определяет способ кодировки имен файлов внутри ZIP-файла.  
Значение по умолчанию: [Авто](КодировкаИменФайловВZipФайле.Авто_(FileNamesEncodingInZipFile.Auto).md).

Описание варианта метода:

Устанавливает файл для записи архива.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Пароль>, <Комментарий>, <МетодСжатия>, <УровеньСжатия>, <МетодШифрования>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток, в который будет записан архив.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль, который будет назначен архиву.   
Если пароль не назначен или является пустой строкой, то шифрование не происходит.

<Комментарий> (необязательный)

Тип: [Строка](lang__def_String.md).   
Комментарий, описывающий ZIP-файл.

<МетодСжатия> (необязательный)

Тип: [МетодСжатияZIP](МетодСжатияZIP_(ZIPCompressionMethod).md).   
Метод сжатия, который будет использован для сжатия архива.  
Значение по умолчанию: [Сжатие](МетодСжатияZIP.Сжатие_(ZIPCompressionMethod.Deflate).md).

<УровеньСжатия> (необязательный)

Тип: [УровеньСжатияZIP](УровеньСжатияZIP_(ZIPCompressionLevel).md).   
Уровень сжатия данных.  
Значение по умолчанию: [Оптимальный](УровеньСжатияZIP.Оптимальный_(ZIPCompressionLevel.Optimal).md).

<МетодШифрования> (необязательный)

Тип: [МетодШифрованияZIP](МетодШифрованияZIP_(ZIPEncryptionMethod).md).   
Метод шифрования, которым будет зашифрован архив.  
Значение по умолчанию: [Zip20](МетодШифрованияZIP.Zip20_(ZIPEncryptionMethod.Zip20).md).

Описание варианта метода:

Устанавливает поток для записи архива.

Описание:

Открывает ZIP-архив для записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
