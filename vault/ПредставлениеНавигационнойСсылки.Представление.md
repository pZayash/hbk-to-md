ПредставлениеНавигационнойСсылки (URLPresentation)

Представление (Presentation)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Представление()

Возвращаемое значение:

Тип: Строка.   
Для корректных ссылок - значение свойства [Текст](ПредставлениеНавигационнойСсылки.Текст.md).  
Для некорректных ссылок (ссылка неправильно сформирована, объект отсутствует в базе, объект метаданных отсутствует, или у пользователя нет права на объект) - значение свойства [НавигационнаяСсылка](ПредставлениеНавигационнойСсылки.НавигационнаяСсылка.md).

Описание:

Возвращает представление ссылки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
