РежимСовместимости (CompatibilityMode)

Версия8\_3\_14 (Version8\_3\_14)

Доступен, начиная с версии 8.3.15.

Описание:

Усовершенствования версии 8.3.15 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.14.

Использование в версии:

Доступен, начиная с версии 8.3.15.
