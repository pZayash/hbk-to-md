ОписаниеИзмененийКонфигурацииВСообщенииОбмена (ConfigurationChangesDescriptionInExchangeMessage)

КонфигурацияИзменена (ConfigurationChanged)

Доступен, начиная с версии 8.3.22.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что чтение сообщения приводит к изменению конфигурации информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
