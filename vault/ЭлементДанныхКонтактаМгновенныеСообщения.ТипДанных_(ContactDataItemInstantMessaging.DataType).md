ЭлементДанныхКонтактаМгновенныеСообщения (ContactDataItemInstantMessaging)

ТипДанных (DataType)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ТипАдресаМгновенныхСообщенийДанныхКонтакта](ТипАдресаМгновенныхСообщенийДанныхКонтакта_(ContactDataInstantMessagingAddressType).md).   
Показывает, как интерпретировать значение.  
Например, тип данных [Рабочий](ТипАдресаМгновенныхСообщенийДанныхКонтакта.Рабочий_(ContactDataInstantMessagingAddressType.Work).md) показывает, что в значении хранится рабочий адрес.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
