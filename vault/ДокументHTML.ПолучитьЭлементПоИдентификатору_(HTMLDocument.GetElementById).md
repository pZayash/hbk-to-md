# ДокументHTML.ПолучитьЭлементПоИдентификатору

ДокументHTML (HTMLDocument)

ПолучитьЭлементПоИдентификатору (GetElementById)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьЭлементПоИдентификатору(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор искомого элемента.

Возвращаемое значение:

Тип: [ЭлементHTML](ЭлементHTML_(HTMLElement).md), [Неопределено](lang__def_Undefined.md).   
Узел, у которого имеется атрибут со свойством [ЯвляетсяИдентификатором](АтрибутHTML.ЯвляетсяИдентификатором_(HTMLAttribute.IsId).md), установленным в [Истина](lang__def_BooleanTrue.md).

Описание:

Возвращает элемент по идентификатору.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
