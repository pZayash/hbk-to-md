# Соединение с агентом сервера.GetLocks

Соединение с агентом сервера (IServerAgentConnection)

GetLocks (GetLocks)

Доступен, начиная с версии 8.1.

Синтаксис:

GetLocks(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер серверов, из которого должен быть получен массив зарегистрированных информационных баз. Может быть получено при помощи метода [GetServerWorkingProcesses](Соединение_с_агентом_сервера.GetServerWorkingProcesses_(IServerAgentConnection.GetServerWorkingProcesses).md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив блокировок объектов, установленных клиентами кластера. Каждая блокировка представлена объектом с интерфейсом [Блокировка](Блокировка_(IObjectLock).md).

Описание:

Получает массив блокировок, установленных клиентами кластера, и хранящихся в кластере. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера. Предоставляемая информация о блокировках не требует аутентификации пользователей информационных баз.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
