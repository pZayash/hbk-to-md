ОбъектМетаданных: Константа (MetadataObject: Constant)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](КонстантаМенеджер.Имя_константы.ОбработкаПослеЗаписиВерсийИсторииДанных_(ConstantManager.Constant_Name.AfterWriteDataHistoryVersionsProcessing).md) предоставляется возможность обработать запись версии после обновления истории.  
При записи данных указанное в конфигураторе значение устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_Константа.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных_(MetadataObject_Constant.ExecuteAfterWriteDataHistoryVersionProcessing).md). Значение свойства можно переопределить.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
