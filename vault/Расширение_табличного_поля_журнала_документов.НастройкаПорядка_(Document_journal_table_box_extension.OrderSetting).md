Расширение табличного поля журнала документов (Document journal table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаПорядка](НастройкаПорядка_(OrderSetting).md).   
Предоставляет доступ к настройке порядка списка документов, регистрируемых в журнале. Представляет собой коллекцию значений типа [ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md). Для журнала документов эта коллекция состоит из одного элемента, соответствующего упорядочиванию списка документов по дате. Свойство коллекции содержит объект для управления элементом порядка и имеет имя "Дата".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
