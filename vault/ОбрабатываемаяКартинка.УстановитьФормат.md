ОбрабатываемаяКартинка (ProcessingPicture)

УстановитьФормат (SetFormat)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьФормат(<Формат>)

Параметры:

<Формат> (обязательный)

Тип: [ФорматКартинки](ФорматКартинки.md).   
Устанавливаемый формат картинки.

Описание:

Устанавливает новый формат картинки.   
Не поддерживаются векторные форматы.  
  
На мобильной платформе поддерживаются только форматы JPEG и PNG.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
