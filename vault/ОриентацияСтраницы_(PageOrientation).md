# ОриентацияСтраницы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Табличный документ](Табличный_документ.md)

Доступен, начиная с версии 8.0.

Значения

[Ландшафт (Landscape)](ОриентацияСтраницы.Ландшафт_(PageOrientation.Landscape).md)  
[Портрет (Portrait)](ОриентацияСтраницы.Портрет_(PageOrientation.Portrait).md)  

Описание:

Определяет варианты ориентации страницы при выводе табличного документа на печать. Используется для определения свойства ОриентацияСтраницы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ГрафическаяСхема](ГрафическаяСхема_(GraphicalSchema).md), свойство [ОриентацияСтраницы](ГрафическаяСхема.ОриентацияСтраницы_(GraphicalSchema.PageOrientation).md)  
[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), свойство [ОриентацияСтраницы](ТабличныйДокумент.ОриентацияСтраницы_(SpreadsheetDocument.PageOrientation).md)  
[ГеографическаяСхема](ГеографическаяСхема_(GeographicalSchema).md), свойство [ОриентацияСтраницы](ГеографическаяСхема.ОриентацияСтраницы_(GeographicalSchema.PageOrientation).md)
