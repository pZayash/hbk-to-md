Расширение формы клиентского приложения для объектов (Client application form extension for objects)

ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)>, [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)>, [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>.   
Записанный объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи.

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
