# Расширение табличного поля списка видов характеристик.АвтоОбновление

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [Расширение табличного поля списка видов характеристик](Расширение_табличного_поля_списка_видов_характеристик_(Characteristic_types_list_table_box_extension).md) › [Свойства](Расширение_табличного_поля_списка_видов_характеристик__Свойства.md)

Расширение табличного поля списка видов характеристик (Characteristic types list table box extension)

АвтоОбновление (AutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического обновления табличного поля. Если признак установлен как [Истина](lang__def_BooleanTrue.md), то автоматически будет вызываться автоматическое обновление списка. Время между обновлениями списка задается свойством [ПериодАвтоОбновления](Расширение_табличного_поля_списка_видов_характеристик.ПериодАвтоОбновления_(Characteristic_types_list_table_box_extension.AutoRefreshPeriod).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
