ПоляПостроителяОтчета (ReportBuilderFields)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ПутьКДанным>, <Имя>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным поля.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

Возвращаемое значение:

Тип: [ПолеПостроителяОтчета](ПолеПостроителяОтчета_(ReportBuilderField).md).   

Описание:

Добавляет поле в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
