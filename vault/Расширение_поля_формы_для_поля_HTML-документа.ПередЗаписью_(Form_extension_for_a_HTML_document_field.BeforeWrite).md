Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если выполняется команда "Сохранить как".

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) позволяет отказаться от записи.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед интерактивной записью HTML-документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
