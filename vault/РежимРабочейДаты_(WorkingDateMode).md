# РежимРабочейДаты

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Разные](Разные.md)

Доступен, начиная с версии 8.0.

Значения

[ИспользоватьТекущуюДату (UseCurrentDate)](РежимРабочейДаты.ИспользоватьТекущуюДату_(WorkingDateMode.UseCurrentDate).md)  
[Назначать (Assign)](РежимРабочейДаты.Назначать_(WorkingDateMode.Assign).md)  

Описание:

Определяет набор вариантов использования рабочей даты в глобальном контексте. Используется для определения свойства [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты_(Global_context.WorkingDateUse).md) глобального контекста.

Доступность:

Толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты_(Global_context.WorkingDateUse).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
