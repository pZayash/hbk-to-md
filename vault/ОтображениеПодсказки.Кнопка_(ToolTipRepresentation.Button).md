ОтображениеПодсказки (ToolTipRepresentation)

Кнопка (Button)

Доступен, начиная с версии 8.3.1.

Описание:

Рядом с элементом выводится кнопка.  
Подсказка отображается при нажатии на эту кнопку.

Использование в версии:

Доступен, начиная с версии 8.3.1.
