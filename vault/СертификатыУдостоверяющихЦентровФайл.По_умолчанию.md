СертификатыУдостоверяющихЦентровФайл (FileCertificationAuthorityCertificates)

На основании файла

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый СертификатыУдостоверяющихЦентровФайл(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Полное имя файла сертификатов удостоверяющих центров.

<Пароль> (необязательный)

Тип: Строка.   
Пароль файла сертификатов удостоверяющих центров.

Описание:

Создает объект сертификатов удостоверяющих центров OpenSSL, загружаемых из указанного файла.

Использование в версии:

Доступен, начиная с версии 8.3.1.
