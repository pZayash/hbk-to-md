Описание:

Дополнительные свойства объекта [Форма](Форма.md) списка видов характеристик, в том случае, если основной реквизит формы имеет тип [ПланВидовХарактеристикСписок.](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик.md)<[Имя плана видов характеристик](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик.md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |

Использование в версии:

Доступен, начиная с версии 8.0.
