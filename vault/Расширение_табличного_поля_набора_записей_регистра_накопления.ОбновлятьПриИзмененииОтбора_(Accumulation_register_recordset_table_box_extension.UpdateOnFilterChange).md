Расширение табличного поля набора записей регистра накопления (Accumulation register recordset table box extension)

ОбновлятьПриИзмененииОтбора (UpdateOnFilterChange)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического обновления набора записей при изменении параметров отбора. Если значение данного свойства установлено в [Ложь](lang__def_BooleanFalse.md), то при изменении параметров отбора обновления не произойдет.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
