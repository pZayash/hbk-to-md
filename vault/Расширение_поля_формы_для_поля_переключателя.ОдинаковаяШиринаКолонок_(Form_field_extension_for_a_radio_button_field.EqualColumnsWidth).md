Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ОдинаковаяШиринаКолонок (EqualColumnsWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет одинаковую ширину колонок элементов.   
[Неопределено](lang__def_Undefined.md) трактуется как [Ложь](lang__def_BooleanFalse.md).   
Если значение свойства отлично от [Истина](lang__def_BooleanTrue.md), свойство "[ШиринаЭлемента](Расширение_поля_формы_для_поля_переключателя.ШиринаЭлемента_(Form_field_extension_for_a_radio_button_field.ItemWidth).md)" не используется.  
Данное свойство используется только, если [ВидПереключателя](Расширение_поля_формы_для_поля_переключателя.ВидПереключателя_(Form_field_extension_for_a_radio_button_field.RadioButtonType).md) = [Тумблер](ВидПереключателя.Тумблер_(RadioButtonType.Tumbler).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
