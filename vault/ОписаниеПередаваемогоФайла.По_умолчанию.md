ОписаниеПередаваемогоФайла (TransferableFileDescription)

По хранению и имени

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ОписаниеПередаваемогоФайла(<Имя>, <Хранение>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Положение файла на клиенте.

<Хранение> (необязательный)

Тип: [Строка](lang__def_String.md).   
Положение файла на сервере.

Описание:

Задает расположение передаваемого файла на сервере и на клиенте.

Использование в версии:

Доступен, начиная с версии 8.2.
