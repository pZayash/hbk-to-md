ЖурналЗвонков (CallLog)

НайтиЗаписи (FindRecords)

Доступен, начиная с версии 8.3.5.

Синтаксис:

НайтиЗаписи(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных.md).   
Задает условия поиска.  
В качестве условий отбора могут быть указаны следующие поля:

- НомерТелефона - тип Строка. Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md), [Содержит](ВидСравненияКомпоновкиДанных.Содержит.md), [НеСодержит](ВидСравненияКомпоновкиДанных.НеСодержит.md).
- Дата - тип Дата. Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше.md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше.md), [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно.md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).
- ТипЗвонка - тип [ТипЗвонкаЖурналаЗвонков](ТипЗвонкаЖурналаЗвонков.md). Допускаются виды сравнения: [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).
- Продолжительность - тип Число. Допускаются виды сравнения: [Больше](ВидСравненияКомпоновкиДанных.Больше.md), [Меньше](ВидСравненияКомпоновкиДанных.Меньше.md), [Равно](ВидСравненияКомпоновкиДанных.Равно.md), [БольшеИлиРавно](ВидСравненияКомпоновкиДанных.БольшеИлиРавно.md), [МеньшеИлиРавно](ВидСравненияКомпоновкиДанных.МеньшеИлиРавно.md), [НеРавно](ВидСравненияКомпоновкиДанных.НеРавно.md).

Элементы отбора могут быть сгруппированы по "И" или по "ИЛИ". Группировка по "НЕ" в отборе не поддерживается.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Элементом возвращаемого массива является [ЗаписьЖурналаЗвонков](ЗаписьЖурналаЗвонков.md).

Описание:

Осуществляет поиск записей в журнале звонков.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
