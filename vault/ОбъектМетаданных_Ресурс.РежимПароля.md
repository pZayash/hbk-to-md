ОбъектМетаданных: Ресурс (MetadataObject: Resource)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Устанавливает отображение данных ресурса в режиме пароля (символы заменяются звездочками при отображении).  
Истина - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для ресурсов типа Строка.

Использование в версии:

Доступен, начиная с версии 8.2.
