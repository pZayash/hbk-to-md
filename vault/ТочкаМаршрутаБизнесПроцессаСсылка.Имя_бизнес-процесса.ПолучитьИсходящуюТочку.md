ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПолучитьИсходящуюТочку (GetOutgoingPoint)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Для точки вида "ВыборВарианта"

Синтаксис:

ПолучитьИсходящуюТочку(<Вариант>)

Параметры:

<Вариант> (обязательный)

Тип: [ВариантТочкиМаршрутаБизнесПроцесса](ВариантТочкиМаршрутаБизнесПроцесса.md).   
Вариант.

Описание варианта метода:

Для точки вида [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта.md) возможны несколько точек перехода. Число точек определяется числом вариантов.

Вариант синтаксиса: Для точки вида "Условие"

Синтаксис:

ПолучитьИсходящуюТочку(<Условие>)

Параметры:

<Условие> (обязательный)

Тип: Булево.   
Указывает точку маршрута бизнес-процесса. Если значение параметра Истина, то требуется получить исходящую точку выхода по "Да", Ложь - по "Нет".

Описание варианта метода:

Для точки вида [Условие](ВидТочкиМаршрутаБизнесПроцесса.Условие.md) возможны две исходящие точки: при значении условия "Да" и "Нет". Параметр уточняет, какую точку требуется получить.

Вариант синтаксиса: Для всех точек, кроме точек вида "Разделение", "Условие", "ВыборВарианта".

Синтаксис:

ПолучитьИсходящуюТочку()

Описание варианта метода:

Для всех точек, кроме точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение.md), [Условие](ВидТочкиМаршрутаБизнесПроцесса.Условие.md) и [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта.md) существует только одна следующая точка.

Возвращаемое значение:

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   

Описание:

Получает точку маршрута бизнес-процесса, следующую за указанной точкой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для всех точек маршрута бизнес-процесса, кроме точек вида [Разделение](ВидТочкиМаршрутаБизнесПроцесса.Разделение.md).

Использование в версии:

Доступен, начиная с версии 8.0.
