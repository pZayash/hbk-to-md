Соединение с агентом сервера (IServerAgentConnection)

GetInfoBaseConnections (GetInfoBaseConnections)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBaseConnections(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, для которого должен быть получен массив описаний соединений.

<ИнформационнаяБаза> (обязательный)

Тип: [Описание информационной базы](Описание_информационной_базы.md).   
Информационная база, для которой должен быть получен массив описаний соединений.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив описаний соединений кластера. Каждое описание соединения представлено объектом с интерфейсом [Описание соединения](Описание_соединения.md).

Описание:

Получает массив описаний соединений информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
