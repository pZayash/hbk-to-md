РежимПроведенияДокумента (DocumentPostingMode)

Неоперативный (Regular)

Доступен, начиная с версии 8.1.

Описание:

Неоперативное проведение, т.е. проведение выполняется не в реальном времени. Например, используется при проведении задним числом.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.23.
