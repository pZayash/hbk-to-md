РасширениеКонфигурации (ConfigurationExtension)

ПолучитьДанные (GetData)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Для нового расширения возвращает Неопределено либо [ДвоичныеДанные](ДвоичныеДанные.md), если они были переданы при создании объекта.

Описание:

Возвращает текущие используемые данные расширения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерРасширенийКонфигурации](МенеджерРасширенийКонфигурации.md), метод [Создать](МенеджерРасширенийКонфигурации.Создать.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
