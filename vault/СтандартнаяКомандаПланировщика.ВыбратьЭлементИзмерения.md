ВыбратьЭлементИзмерения (SelectDimensionItem)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ВыбратьЭлементИзмерения](СтандартнаяКомандаПоляПланировщика.ВыбратьЭлементИзмерения.md)

Описание:

Переход к событию [НажатиеНаЭлементеИзмерения](Расширение_поля_формы_для_поля_планировщика.НажатиеНаЭлементеИзмерения.md).  
Используется только для источника вида [ЭлементИзмерения](ИсточникКомандПланировщика.ЭлементИзмерения.md).  
В контекстном меню команда будет иметь представление "Открыть".

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
