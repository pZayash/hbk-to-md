ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

Порт (Port)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
IP-порт агента сервера 1С:Предприятия. От 0 по 65535.   
Значение 0 предполагает использование порта агента по умолчанию - 1540.   
Не используется, если [IPАдресАгента](ЗапросНаПолучениеЛицензии.IPАдресАгента_(LicenseAcquisitionRequest.AgentIPAddress).md) - пустая строка.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
