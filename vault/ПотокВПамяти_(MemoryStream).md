Описание:

Специализированный вариант объекта [Поток](Поток_(Stream).md) для работы с данными, расположенными в оперативной памяти.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЧтениеДанных](ЧтениеДанных_(DataReader).md)  
[ЗаписьДанных](ЗаписьДанных_(DataWriter).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
