# ПланОбменаОбъект.<Имя плана обмена>.УстановитьПометкуУдаления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [Методы](ПланОбменаОбъект.Имя_плана_обмена__Методы.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак установки / снятие пометки на удаление.  
[Истина](lang__def_BooleanTrue.md) - пометка будет установлена; [Ложь](lang__def_BooleanFalse.md) - пометка будет снята.

Описание:

Устанавливает/снимет пометку на удаление. Объект при этом не удаляется из базы данных. Окончательное удаление помеченных узлов происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](ПланОбменаОбъект.Имя_плана_обмена.ПередЗаписью_(ExchangePlanObject.Exchange_plan_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](ПланОбменаОбъект.Имя_плана_обмена.ПометкаУдаления_(ExchangePlanObject.Exchange_plan_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Использование в версии:

Доступен, начиная с версии 8.0.
