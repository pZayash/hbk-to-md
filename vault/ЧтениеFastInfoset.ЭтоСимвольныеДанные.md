ЧтениеFastInfoset (FastInfosetReader)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.   
Показывает, имеет ли текущий узел текстовое значение. Текстовое значение имеется у узлов: [Текст](ТипУзлаXML.Текст.md), [СекцияCDATA](ТипУзлаXML.СекцияCDATA.md), [ПробельныеСимволы](ЧтениеFastInfoset.ПробельныеСимволы.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
