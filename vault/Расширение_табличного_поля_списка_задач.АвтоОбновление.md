Расширение табличного поля списка задач (Task list table box extension)

АвтоОбновление (AutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак автоматического обновления табличного поля. Если признак установлен как Истина, то автоматически будет вызываться обновление списка. Время между обновлениями списка задается свойством [ПериодАвтоОбновления](Расширение_табличного_поля_списка_задач.ПериодАвтоОбновления.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
