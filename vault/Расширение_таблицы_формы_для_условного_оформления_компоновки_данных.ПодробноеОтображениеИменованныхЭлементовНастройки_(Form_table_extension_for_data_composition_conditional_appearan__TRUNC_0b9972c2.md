Расширение таблицы формы для условного оформления компоновки данных (Form table extension for data composition conditional appearance)

ПодробноеОтображениеИменованныхЭлементовНастройки (SettingsNamedItemDetailedRepresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство появляется у таблицы, если она связана с данными типа [УсловноеОформлениеКомпоновкиДанных](УсловноеОформлениеКомпоновкиДанных_(DataCompositionConditionalAppearance).md).  
[Ложь](lang__def_BooleanFalse.md) - именованные элементы (свойство [Отображение](ТаблицаФормы.Отображение_(FormTable.Representation).md) не пустое) отображаются в таблице одной строкой с одной ячейкой, в которой выводится представление.  
В противном случае, представление редактируется в отдельной колонке.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
