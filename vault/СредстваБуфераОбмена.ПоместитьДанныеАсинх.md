СредстваБуфераОбмена (ClipboardTools)

ПоместитьДанныеАсинх (PutDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ПоместитьДанныеАсинх(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: [ЭлементБуфераОбмена](ЭлементБуфераОбмена.md), [Массив](Массив.md).   
Элемент буфера обмена или массив элементов типа [ЭлементБуфераОбмена](ЭлементБуфераОбмена.md).

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Булево.

Описание:

Асинхронная функция. Помещает переданные данные в буфер обмена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

При помещении в буфер обмена данных формата [HTML](СтандартныйФорматДанныхБуфераОбмена.HTML.md) часть оформления HTML-текста может быть утеряна.  
  
В веб-клиенте:

- может появиться запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается [Обещание](Обещание.md), результат которого – Булево.
- значение параметра <Формат> типа Строка должно соответствовать формату MIME-типов. Например, "video/mp4" или "image/svg+xml".

Использование в версии:

Доступен, начиная с версии 8.3.24.
