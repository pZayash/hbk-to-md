Описание:

Определяет набор шрифтов, используемых в стиле. Значения этого набора имеют тип [Шрифт](Шрифт.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
