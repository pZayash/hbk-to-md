ПолеФормы (FormField)

РежимРедактирования (EditMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимРедактированияКолонки](РежимРедактированияКолонки_(ColumnEditMode).md).   
Определяет режим редактирования поля в таблице. Данное свойство имеет смысл только для полей, расположенных в таблице (представляющих собой колонки).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
