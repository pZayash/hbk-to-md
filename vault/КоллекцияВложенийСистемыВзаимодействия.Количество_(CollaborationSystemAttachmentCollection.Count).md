КоллекцияВложенийСистемыВзаимодействия (CollaborationSystemAttachmentCollection)

Количество (Count)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает количество элементов в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
