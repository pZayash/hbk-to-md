ГруппировкаТаблицыКомпоновкиДанных (DataCompositionTableGroup)

Порядок (Order)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПорядокКомпоновкиДанных](ПорядокКомпоновкиДанных_(DataCompositionOrder).md).   
Содержит порядок для упорядочивания групповых записей.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
