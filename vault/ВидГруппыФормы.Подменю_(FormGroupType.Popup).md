ВидГруппыФормы (FormGroupType)

Подменю (Popup)

Доступен, начиная с версии 8.2.

Описание:

Подменю командной панели. Группа данного вида может содержать кнопки и группы видов [Подменю](ВидГруппыФормы.Подменю_(FormGroupType.Popup).md) и [ГруппаКнопок](ВидГруппыФормы.ГруппаКнопок_(FormGroupType.ButtonGroup).md).

Использование в версии:

Доступен, начиная с версии 8.2.
