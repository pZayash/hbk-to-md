ПолеКалендаря (CalendarBox)

ПриАктивизацииДаты (OnActivateDate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриАктивизацииДаты()

Описание:

Возникает при активизации даты в календаре (указание даты мышью или перемещение на дату с помощью клавиатуры).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
