Файл (File)

ПолучитьУниверсальноеВремяИзменения (GetModificationUniversalTime)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьУниверсальноеВремяИзменения()

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   
Универсальное время последнего изменения файла.

Описание:

Получает универсальное время последнего изменения файла.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применим только к файлам и каталогам, которые реально существуют.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
