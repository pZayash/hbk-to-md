Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПриСохраненииВариантаНаСервере (OnSaveVariantAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСохраненииВариантаНаСервере(<Настройки>)

Параметры:

<Настройки>

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md).   
Сохраняемые настройки.

Описание:

Вызывается при сохранении варианта отчета.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
