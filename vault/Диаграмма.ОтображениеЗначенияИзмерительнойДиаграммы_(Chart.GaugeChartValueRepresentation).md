ОтображениеЗначенияИзмерительнойДиаграммы (GaugeChartValueRepresentation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеЗначенияИзмерительнойДиаграммы](ОтображениеЗначенияИзмерительнойДиаграммы_(GaugeChartValueRepresentation).md).   
Определяет способ графического представления значений серий измерительной диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
