НастройкиВосстановленияПароля (PasswordRecoverySettings)

ДлинаКодаПодтверждения (VerificationCodeLength)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет длину кода подтверждения для восстановления пароля - целое число в диапазоне от 8 до 50.   
Значение по умолчанию - 8.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
