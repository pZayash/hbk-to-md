АдминистрированиеКластер (AdministrationCluster)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить()

Описание:

Выполняет отмену регистрации кластера. Для успешного выполнения метода необходима аутентификация одного из пользователей кластера. Отмена регистрации возможна только для пустого кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
