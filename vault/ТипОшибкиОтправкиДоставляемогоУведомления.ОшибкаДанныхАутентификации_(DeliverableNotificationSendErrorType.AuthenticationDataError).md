ТипОшибкиОтправкиДоставляемогоУведомления (DeliverableNotificationSendErrorType)

ОшибкаДанныхАутентификации (AuthenticationDataError)

Доступен, начиная с версии 8.3.11.

Описание:

Ошибка данных аутентификации (в файле сертификата, необходимом для подключения к службе доставки "Apple Push Notification Service", либо строке с ключом авторизации для подключения к службе доставки "Google Cloud Messaging").

Использование в версии:

Доступен, начиная с версии 8.3.11.
