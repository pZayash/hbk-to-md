КолонкиАнализаДанных (DataAnalysisColumns)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомой колонки.

Возвращаемое значение:

Тип: [КолонкаАнализаДанных](КолонкаАнализаДанных.md), Неопределено.   
Если колонка не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск колонки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
