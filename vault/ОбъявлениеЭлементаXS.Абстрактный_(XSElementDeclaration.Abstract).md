ОбъявлениеЭлементаXS (XSElementDeclaration)

Абстрактный (Abstract)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак абстрактного объявления элемента.  
Абстрактное объявление элемента может появляться в модели содержания только в случае, когда разрешена подстановка. Абстрактные объявления элементов сами в процессе проверки содержимого элемента не участвуют.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - тип абстрактный.

Использование в версии:

Доступен, начиная с версии 8.1.
