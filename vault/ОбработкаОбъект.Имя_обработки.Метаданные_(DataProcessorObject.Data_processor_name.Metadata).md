# ОбработкаОбъект.<Имя обработки>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Обработки](Обработки.md) › [ОбработкаОбъект.<Имя обработки>](ОбработкаОбъект.Имя_обработки_(DataProcessorObject.Data_processor_name).md) › [Методы](ОбработкаОбъект.Имя_обработки__Методы.md)

ОбработкаОбъект.<Имя обработки> (DataProcessorObject.<Data processor name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Обработка](ОбъектМетаданных_Обработка_(MetadataObject_DataProcessor).md).   

Описание:

Предоставляет доступ к объекту описания метаданных обработки. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.Обработки.ВыгрузкаДанных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Объект = Справочники.Конкуренты.НайтиПоКоду(1); МетаданныеСправочника = Объект.Метаданные(); Сообщить("Представление справочника - " +     МетаданныеСправочника.Представление()); |

Использование в версии:

Доступен, начиная с версии 8.0.
