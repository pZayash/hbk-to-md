# Расширение формы узла.ПриЗаписи

Расширение формы узла (Node form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланОбменаОбъект.Имя_плана_обмена.ПриЗаписи_(ExchangePlanObject.Exchange_plan_name.OnWrite).md) объекта [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
