ОбъектМетаданных: Документ (MetadataObject: Document)

ПривилегированныйРежимПриОтменеПроведения (PrivilegedUnpostingMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает возможность использования привилегированного режима при проведении документа. Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то проведение документа будет всегда выполняться в привилегированном режиме (на сервере и в файловом варианте).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если проведение документа выполняется на клиенте в клиент-серверном варианте, то привилегированный режим не устанавливается.

Использование в версии:

Доступен, начиная с версии 8.2.
