БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на бизнес-процессы.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [БизнесПроцессВыборка.\<Имя бизнес-процесса\>](БизнесПроцессВыборка.Имя_бизнес-процесса.md).   

Описание:

Формирует выборку бизнес-процессов по заданному источнику ссылок на бизнес-процессы соответствующего типа. Если источник содержит значения, отличные от ссылок на бизнес-процессы соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    СогласованиеПродажи.Ссылка КАК Ссылка     |ИЗ     |    БизнесПроцесс.СогласованиеПродажи КАК СогласованиеПродажи     |"; Результат = Запрос.Выполнить(); Выборка = БизнесПроцессы.СогласованиеПродажи     .ВыбратьПоСсылкам(Результат); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
