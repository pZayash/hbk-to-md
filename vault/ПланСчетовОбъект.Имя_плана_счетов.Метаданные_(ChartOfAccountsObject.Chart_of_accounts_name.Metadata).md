# ПланСчетовОбъект.<Имя плана счетов>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы счетов](Планы_счетов.md) › [ПланСчетовОбъект.<Имя плана счетов>](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md) › [Методы](ПланСчетовОбъект.Имя_плана_счетов__Методы.md)

ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов_(MetadataObject_ChartOfAccounts).md).   

Описание:

Предоставляет доступ к объекту описания метаданных плана счетов. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.ПланыСчетов.Основной.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ВыборкаПоСчету = ПланыСчетов[Счет.Метаданные().Имя].Выбрать(Счет); |

Использование в версии:

Доступен, начиная с версии 8.0.
