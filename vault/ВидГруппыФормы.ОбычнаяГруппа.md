ВидГруппыФормы (FormGroupType)

ОбычнаяГруппа (UsualGroup)

Доступен, начиная с версии 8.2.

Описание:

Обычная группа. Группа данного вида может содержать поля, таблицы, кнопки и группы видов [ОбычнаяГруппа](ВидГруппыФормы.ОбычнаяГруппа.md), [КоманднаяПанель](ВидГруппыФормы.КоманднаяПанель.md), [Страницы](ВидГруппыФормы.Страницы.md).

Использование в версии:

Доступен, начиная с версии 8.2.
