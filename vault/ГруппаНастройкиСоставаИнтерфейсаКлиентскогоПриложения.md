Элементы коллекции:

[ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения.md), [ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
