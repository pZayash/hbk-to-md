СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

ШаблонСообщения (MessageTemplate)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplateID).md), [Неопределено](lang__def_Undefined.md).   
Если свойство установлено, то при записи создается шаблонное сообщение.  
При записи шаблонного сообщения игнорируются свойства [Текст](СообщениеСистемыВзаимодействия.Текст_(CollaborationSystemMessage.Text).md), [Вложения](СообщениеСистемыВзаимодействия.Вложения_(CollaborationSystemMessage.Attachments).md), [Данные](СообщениеСистемыВзаимодействия.Данные_(CollaborationSystemMessage.Data).md) и [Действия](СообщениеСистемыВзаимодействия.Действия_(CollaborationSystemMessage.Actions).md).  
Установка свойства возможна только для нового незаписанного сообщения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
