Доступен, начиная с версии 8.3.13.

Элементы коллекции:

[ГруппаФормы](ГруппаФормы.md), [КнопкаФормы](КнопкаФормы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.  

Методы:

[Вставить (Insert)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Вставить.md)

[Индекс (IndexOf)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Индекс.md)  
[Количество (Count)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Количество.md)  
[Найти (Find)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Найти.md)  
[Сдвинуть (Move)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Сдвинуть.md)  
[Удалить (Delete)](СоставКоманднойПанелиФормыНаМобильномУстройстве.Удалить.md)  

Описание:

Содержит коллекцию элементов формы, входящих в командную панель мобильного клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.13.
