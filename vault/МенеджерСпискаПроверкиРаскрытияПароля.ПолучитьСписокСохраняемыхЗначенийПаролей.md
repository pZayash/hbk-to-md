МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

ПолучитьСписокСохраняемыхЗначенийПаролей (GetPasswordsStoredValuesList)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСписокСохраняемыхЗначенийПаролей(<ВыходныеДанные>)

Параметры:

<ВыходныеДанные> (необязательный)

Тип: Строка, [Поток](Поток.md), [ФайловыйПоток](ФайловыйПоток.md), [ПотокВПамяти](ПотокВПамяти.md), Неопределено.   
Если передан параметр типа [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md) – список хэш-значений записывается в соответствующий поток.   
Если передана строка с путем к файлу - списо сохраняемых значений будет записан в этот файл.  
Если передано Неопределено, метод вернет [Массив](Массив.md) сохраняемых значений паролей.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Неопределено, [Массив](Массив.md).   
Элементы массива имеют тип Строка и содержат хеш-значения паролей, вычисленные алгоритмом SHA1 и закодированные в base64, разделенные переносом строки.

Описание:

Возвращает список сохраняемых значений раскрытых паролей.

Доступность:

Сервер, толстый клиент.

Пример:

Использование в версии:

Доступен, начиная с версии 8.3.26.
