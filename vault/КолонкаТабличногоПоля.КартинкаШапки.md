КолонкаТабличногоПоля (TableBoxColumn)

КартинкаШапки (HeaderPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку, расположенную в шапке колонки.

Доступность:

Толстый клиент.

Примечание:

Рекомендуемый размер картинки 14х14 пикселей.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
