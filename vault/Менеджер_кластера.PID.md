Менеджер кластера (IClusterManagerInfo)

PID (PID)

Доступен, начиная с версии 8.2.

Только чтение.

Описание:

Тип: Строка.   
Идентификатор процесса операционной системы дополнительного менеджера кластера.

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
