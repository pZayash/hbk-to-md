Доступен, начиная с версии 8.3.8.

Элементы коллекции:

[ВременнаяТаблицаЗапроса](ВременнаяТаблицаЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются временные таблицы.  
Возможно обращение к временной таблице посредством оператора [...]. В качестве аргумента передается индекс, полное имя.  

Методы:

[Индекс (IndexOf)](ВременныеТаблицыЗапроса.Индекс.md)  
[Количество (Count)](ВременныеТаблицыЗапроса.Количество.md)  
[Найти (Find)](ВременныеТаблицыЗапроса.Найти.md)  
[Получить (Get)](ВременныеТаблицыЗапроса.Получить.md)  

Описание:

Содержит коллекцию временных таблиц запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
