СхемаXML (XMLSchema)

ДокументDOM (DOMDocument)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ДокументDOM](ДокументDOM.md).   
Документ DOM, соответствующий схеме XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Наличие документа DOM не является обязательным. Для его создания можно воспользоваться методом [СхемаXML.ОбновитьЭлементDOM](СхемаXML.ОбновитьЭлементDOM.md).

Использование в версии:

Доступен, начиная с версии 8.1.
