ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗакрыватьПриВыборе (CloseOnChoice)

Доступен, начиная с версии 8.2.

Описание:

Тип: Булево.   
Указывает необходимость закрытия формы при выборе значения.  
При создании формы значение данного параметра устанавливается в свойство [ЗакрыватьПриВыборе](ФормаКлиентскогоПриложения.ФормаКлиентскогоПриложения.ЗакрыватьПриВыборе.md).  
Истина - форма будет закрыта при выборе.

Использование в версии:

Доступен, начиная с версии 8.2.
