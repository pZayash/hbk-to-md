# МенеджерWebSocketКлиентов.СоздатьКлиента

МенеджерWebSocketКлиентов (WebSocketClientsManager)

СоздатьКлиента (CreateClient)

Доступен, начиная с версии 8.3.27.

Синтаксис:

СоздатьКлиента(<Метаданные>, <Ключ>)

Параметры:

<Метаданные> (обязательный)

Тип: [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент_(MetadataObject_WebSocketClient).md).   
Объект метаданных, который описывает создаваемого клиента.

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор соединения.

Возвращаемое значение:

Тип: [WebSocketКлиент](WebSocketКлиент_(WebSocketClient).md).   

Описание:

Создает нового непредопределенного клиента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
