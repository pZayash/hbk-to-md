КоллекцияЗначенийПараметровКомпоновкиДанных (DataCompositionParameterValueCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных_(DataCompositionParameterValue).md) или [ЗначениеПараметраНастроекКомпоновкиДанных](ЗначениеПараметраНастроекКомпоновкиДанных_(DataCompositionSettingsParameterValue).md).   
Элемент, индекс которого определяется.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс элемента в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
