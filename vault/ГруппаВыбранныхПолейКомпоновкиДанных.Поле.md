Описание:

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных.md).   
Содержит путь к данным доступного поля, связанного с папкой. Используется для получения заголовка папки по умолчанию. Очищается при интерактивном изменении заголовка папки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
