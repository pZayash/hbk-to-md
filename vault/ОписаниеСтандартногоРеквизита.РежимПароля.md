ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Устанавливает отображение данных реквизита в режиме пароля (символы заменяются звездочками при отображении).   
Истина - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для реквизитов типа Строка.

Использование в версии:

Доступен, начиная с версии 8.2.
