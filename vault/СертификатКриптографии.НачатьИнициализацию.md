СертификатКриптографии (CryptoCertificate)

НачатьИнициализацию (BeginInitialization)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьИнициализацию(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения загрузки сертификата из файла со следующими параметрами:

- <Сертификат> – созданный [СертификатКриптографии](СертификатКриптографии.md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИмяФайла> (обязательный)

Тип: Строка.   
Полный путь к файлу сертификата.

Описание:

Начинает инициализацию объекта [СертификатКриптографии](СертификатКриптографии.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии.md), метод [ИнициализироватьАсинх](СертификатКриптографии.ИнициализироватьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
