ОбъектМетаданных: Роль (MetadataObject: Role)

Права (Rights)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
Для получения прав можно использовать методы глобального контекста [ПравоДоступа](Глобальный_контекст.ПравоДоступа_(Global_context.AccessRight).md) и [ПараметрыДоступа](Глобальный_контекст.ПараметрыДоступа_(Global_context.AccessParameters).md)().

Использование в версии:

Доступен, начиная с версии 8.1.
