# СравнениеФайлов.УчитыватьРазделителиСтрок

СравнениеФайлов (FileCompare)

УчитыватьРазделителиСтрок (EOLSensitive)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость сравнения разделителей строк при сравнении текстов. [Истина](lang__def_BooleanTrue.md) - разделители строк учитываются. Значение по умолчания - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Толстый клиент.

Примечание:

При сравнении табличных документов и двоичном сравнении свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.1.
