# ОформлениеКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ЗначениеПараметраКомпоновкиДанных](ЗначениеПараметраКомпоновкиДанных_(DataCompositionParameterValue).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения параметров.  
Возможно обращение к значению параметра посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция элементов оформления компоновки данных.   
Для оформления используются следующие элементы оформления:

- АвтоОтступ (AutoIndent) - тип значения [Число](lang__def_Number.md);
- ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md);
- ВертикальноеПоложениеКартинки (PictureVerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md). Размещение картинки по вертикали;
- ВесовойКоэффициентШирины (WidthWeightFactor) - тип значения [Число](lang__def_Number.md). Весовой коэффициент для автоматического расчета ширины колонки;
- Видимость (Visible) - [Булево](lang__Булево_(Boolean).md). Используется только для поля, расположенного в таблице (колонки);
- ВыводитьКартинку (PictureOutput) - тип значения [ТипВыводаКартинкиКомпоновкиДанных](ТипВыводаКартинкиКомпоновкиДанных_(DataCompositionPictureOutputType).md). Способ вывода картинки;
- ВыделятьОтрицательные (MarkNegatives) - тип значения [Булево](lang__Булево_(Boolean).md);
- ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md);
- ГоризонтальноеПоложениеКартинки (PictureHorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md). Размещение картинки по горизонтали;
- Доступность (Enabled) - [Булево](lang__Булево_(Boolean).md). Используется только для поля, расположенного в таблице (колонки);
- ИндикаторВДиаграмме (IndicatorInChart) - тип значения [Булево](lang__Булево_(Boolean).md). Указывает, что серия, полученная из данной группировки является индикатором. Используется только для серий диаграммы;
- ИнформационнаяЛинияВДиаграмме (ReferenceLineInChart) - тип значения [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы_(ChartReferenceLine).md). Информационная линия в диаграмме;
- ИнформационныйИнтервалВДиаграмме (ReferenceBandInChart) - тип значения [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы_(ChartReferenceBand).md). Информационный интервал в диаграмме;
- ИспользуемаяОсьЗначенийВДиаграмме (UsedValueAxisInChart) - тип значения [ИспользуемаяОсьЗначенийДиаграммы](ИспользуемаяОсьЗначенийДиаграммы_(UsedChartValuesAxis).md) Настраивает использование дополнительной оси значений в диаграмме;
- Картинка (Picture) - тип значения [Картинка](Картинка_(Picture).md);
- ЛинииТрендаВДиаграмме (ChartTrendlines) - тип значения [ЛинииТрендаДиаграммы](ЛинииТрендаДиаграммы_(ChartTrendlines).md). Используется только для серий диаграммы;
- ЛинияВДиаграмме (LineInChart) - тип значения [Линия](Линия_(Line).md). Тип линии серии, полученной из данной группировки. Используется только для серий диаграммы;
- МаксимальнаяВысота (MaxHeight) - тип значения [Число](lang__def_Number.md);
- МаксимальнаяШирина (MaxWidth) - тип значения [Число](lang__def_Number.md);
- МаркерВДиаграмме (MarkerInChart) - тип значения [ТипМаркераДиаграммы](ТипМаркераДиаграммы_(ChartMarkerType).md). Тип маркера серии, полученной из данной группировки. Используется только для серий диаграммы;
- МинимальнаяВысота (MinHeight) - тип значения [Число](lang__def_Number.md);
- МинимальнаяШирина (MinWidth) - тип значения [Число](lang__def_Number.md);
- ОриентацияТекста (TextOrientation) - тип значения [Число](lang__def_Number.md);
- ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную - тип значения [Булево](lang__Булево_(Boolean).md);
- Отображать (Show) - [Булево](lang__Булево_(Boolean).md). Используется только для поля, расположенного в таблице (колонки);
- ОтображатьГрафическоеПредставлениеДанныхВДиаграмме (ShowGraphicalRepresentationOfDataOnChart) тип значения [ОтображениеВДиаграмме](ОтображениеВДиаграмме_(ShowInChart).md). Используется только для серий диаграммы;
- ОтображатьГрафическоеПредставлениеДанныхВЛегендеДиаграммы (ShowGraphicalRepresentationOfDataInChartLegend) - тип значения [ОтображениеВЛегендеДиаграммы](ОтображениеВЛегендеДиаграммы_(ShowInChartLegend).md). Используется только для серий диаграммы;
- Отступ (Indent) - тип значения [Число](lang__def_Number.md);
- ОтступСверху (TopMargin) - тип значения [Число](lang__def_Number.md);
- ОтступСлева (LeftMargin) - тип значения [Число](lang__def_Number.md);
- ОтступСнизу (BottomMargin) - тип значения [Число](lang__def_Number.md).
- ОтступСправа (RightMargin) - тип значения [Число](lang__def_Number.md);
- ПоложениеТекстаОтносительноКартинки (TextPositionRelativeToPicture) - тип значения [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки_(TextPositionRelativeToPicture).md). Положение текста относительно картинки;
- РазделительМножественныхЗначений (MultipleValuesSeparator) - тип значения [Строка](lang__def_String.md). Содержит разделитель, который вставляется между значениями множественных характеристик. Если параметр не задан, то используется запятая с пробелом;
- РазмерКартинки (PictureSize) - тип значения [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки_(TextPositionRelativeToPicture).md). Положение текста относительно картинки;
- Размещение (Placement) - размещение текста - тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных_(DataCompositionTextPlacementType).md);
- РастягиватьПоГоризонтали (HorizontalStretch) - тип значения [Булево](lang__Булево_(Boolean).md). Указывает, что ширина текущей колонки рассчитывается автоматически;
- СтильГраницы (BorderStyle) - тип значения [Линия](Линия_(Line).md);
- СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия_(Line).md);
- СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия_(Line).md);
- СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия_(Line).md);
- СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия_(Line).md);
- Текст (Text) - тип значения [Строка](lang__def_String.md) или [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md);
- ТипГрафическогоПредставленияВДиаграмме (VisualTypeInChart) - тип значения [ТипГрафическогоПредставленияСерииДиаграммы](ТипГрафическогоПредставленияСерииДиаграммы_(ChartSeriesGraphicalRepresentationType).md).Тип графического представления серии в диаграмме;
- ТипНакопленияВДиаграмме (StackTypeInChart) - тип значения [ТипНакопленияСерииДиаграммы](ТипНакопленияСерииДиаграммы_(ChartSeriesStackType).md). Тип накопления серии, отличный от типа диаграммы. Группа накопления серии создаётся автоматически, на основе элемента условного оформления. Разные элементы условного оформления будут имеют разные группы накопления;
- ТипОтметки (MarkType) - тип значения [ТипОтметкиЯчейкиТаблицы](objects__catalog2__catalog277__TableCellMarkType.md);
- ТолькоПросмотр (ReadOnly) - [Булево](lang__Булево_(Boolean).md). Используется только для поля, расположенного в таблице (колонки);
- Формат (Format) - тип значения [Строка](lang__def_String.md) (форматная строка);
- ЦветВДиаграмме (ColorInChart) - тип значения [Цвет](Цвет_(Color).md). Используется только для серии диаграммы;
- ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет_(Color).md);
- ЦветОтметки (MarkColor) - тип значения [Цвет](Цвет_(Color).md);
- ЦветТекста (TextColor) - тип значения [Цвет](Цвет_(Color).md);
- ЦветФона (BackColor) - тип значения [Цвет](Цвет_(Color).md);
- Шрифт (Font) - тип значения [Шрифт](Шрифт_(Font).md);

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: Appearance.
