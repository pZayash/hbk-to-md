СерияДиаграммы (ChartSeries)

ФорматПроцентовВПодписях (LabelPercentFormat)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Задает формат процентов в подписях этой серии. Если формат пуст, то используется формат общий для диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
