# Расширение поля формы для поля картинки.ПроверкаПеретаскивания

Расширение поля формы для поля картинки (Form field extension for a picture field)

ПроверкаПеретаскивания (DragCheck)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверкаПеретаскивания(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания_(DragParameters).md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при движении курсора в поле приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В веб-клиенте при перетаскивании файла (файлов) как [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md) обработчик данного события у элемента управления вызывается один раз в момент отпускания кнопки мыши перед обработчиком события [Перетаскивание](Расширение_поля_формы_для_поля_картинки.Перетаскивание_(Form_field_extension_for_a_picture_field.Drag).md) в отличие от тонкого клиента, в котором обработчик данного события вызывается в процессе перетаскивания файла (файлов).  
В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл_(FileDragMode.AsFile).md), то перетаскивание файлов игнорируется, и обработчик данного события не вызывается.

Использование в версии:

Доступен, начиная с версии 8.2.
