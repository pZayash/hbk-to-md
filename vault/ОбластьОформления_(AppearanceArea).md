Элементы коллекции:

[ЭлементОбластиОформления](ЭлементОбластиОформления_(AppearanceAreaItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы области оформления.

[По умолчанию](ОбластьОформления.По_умолчанию.md)  

Описание:

Содержит описание области, к которой применяется оформление.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.
