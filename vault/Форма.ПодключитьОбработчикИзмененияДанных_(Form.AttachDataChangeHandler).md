Форма (Form)

ПодключитьОбработчикИзмененияДанных (AttachDataChangeHandler)

Доступен, начиная с версии 8.0.

Синтаксис:

ПодключитьОбработчикИзмененияДанных(<ИмяДанных>, <ИмяПроцедуры>, <ОтслеживатьПодчиненныеДанные>)

Параметры:

<ИмяДанных> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя данных (путь к данным), изменения которых необходимо отслеживать. Могут быть указаны данные, связанные с реквизитами формы, табличными частями, а также подчиненные данные - реквизиты этих объектов. Например, "Реквизит1.Наименование".  
Параметр может содержать только один путь к данным, т.е. метод применяется для одного реквизита.

<ИмяПроцедуры> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры, подключаемой в качестве обработчика изменения данных. Процедура должна иметь один параметр, в который при вызове процедуры будет передана строка - путь к измененным данным, по структуре аналогичная параметру <Имя данных>.

<ОтслеживатьПодчиненныеДанные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость вызова процедуры-обработчика при изменении данных, подчиненных заданным в параметре <ИмяДанных>, Например, при изменении их реквизитов.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Подключает процедуру-обработчик, вызываемую при изменении указанных данных в форме или подчиненных формах.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ПодключитьОбработчикИзмененияДанных("ПолеВвода1", "ПриИзменении"); |
