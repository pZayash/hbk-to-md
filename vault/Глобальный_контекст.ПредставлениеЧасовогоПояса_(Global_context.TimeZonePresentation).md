# Глобальный контекст.ПредставлениеЧасовогоПояса

Глобальный контекст (Global context)

ПредставлениеЧасовогоПояса (TimeZonePresentation)

Доступен, начиная с версии 8.2.

Синтаксис:

ПредставлениеЧасовогоПояса(<ЧасовойПояс>)

Параметры:

<ЧасовойПояс> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор часового пояса.  
Список допустимых часовых поясов может быть получен методом [ПолучитьДопустимыеЧасовыеПояса](Глобальный_контекст.ПолучитьДопустимыеЧасовыеПояса_(Global_context.GetAvailableTimeZones).md).   
Кроме того, возможно задание часовых поясов смещением от универсального времени (часовые пояса GMT) в формате:  
GMT{+/-}h[h][:mm]  
где hh=0:23, mm=0:59.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает локализованное наименование часового пояса с заданным идентификатором.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьДопустимыеЧасовыеПояса](Глобальный_контекст.ПолучитьДопустимыеЧасовыеПояса_(Global_context.GetAvailableTimeZones).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
