СертификатКлиентаMacOS (MacOSClientCertificate)

По способу выбора сертификата

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый СертификатКлиентаMacOS(<СпособВыбораСертификатаMacOS>)

Параметры:

<СпособВыбораСертификатаMacOS> (обязательный)

Тип: [СпособВыбораСертификатаMacOS](СпособВыбораСертификатаMacOS.md).   
Указывает способ выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы macOS.

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы macOS.  
Доступность: Толстый клиент, тонкий клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
