Соединение с агентом сервера (IServerAgentConnection)

RegCluster (RegCluster)

Доступен, начиная с версии 8.1.

Синтаксис:

RegCluster(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Описание кластера. Этот объект должен быть предварительно создан методом [CreateClusterInfo](Соединение_с_агентом_сервера.CreateClusterInfo.md) и должны быть установлены правильные значения его свойств. Объект, являющийся элементом массива зарегистрированных кластеров, также может быть использован для создания нового кластера после изменения значений соответствующих свойств.

Описание:

Позволяет зарегистрировать существующий кластер или создать новый кластер и зарегистрировать его.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
