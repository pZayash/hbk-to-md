ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

ВключатьРасширенияКонфигурации (IncludeConfigurationExtensions)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то в распределенной информационной базе, организованной данным планом обмена, будут передаваться расширения конфигурации, у которых свойство [ИспользуетсяВРаспределеннойИнформационнойБазе](РасширениеКонфигурации.ИспользуетсяВРаспределеннойИнформационнойБазе_(ConfigurationExtension.UsedInDistributedInfoBase).md) установлено в значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md), свойство [ИспользуетсяВРаспределеннойИнформационнойБазе](РасширениеКонфигурации.ИспользуетсяВРаспределеннойИнформационнойБазе_(ConfigurationExtension.UsedInDistributedInfoBase).md)  
[РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md), свойство [ГлавныйУзел](РасширениеКонфигурации.ГлавныйУзел_(ConfigurationExtension.MasterNode).md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
