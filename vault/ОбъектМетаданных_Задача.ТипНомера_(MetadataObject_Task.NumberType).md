ОбъектМетаданных: Задача (MetadataObject: Task)

ТипНомера (NumberType)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ТипНомераЗадачи](ТипНомераЗадачи_(TaskNumberType).md).   
Определяет тип номера задачи (например, [Число](ТипНомераБизнесПроцесса.Число_(BusinessProcessNumberType.Number).md), [Строка](ТипНомераБизнесПроцесса.Строка_(BusinessProcessNumberType.String).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
