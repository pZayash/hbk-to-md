ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

Отчеты (Reports)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все отчеты, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: Отчет](ОбъектМетаданных_Отчет_(MetadataObject_Report).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
