РегистрРасчетаМенеджер.<Имя регистра расчета> (CalculationRegisterManager.<Calculation register name>)

СоздатьКлючЗаписи (CreateRecordKey)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьКлючЗаписи(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (обязательный)

Тип: [Структура](Структура_(Structure).md), [РегистрРасчетаЗапись.](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаЗапись.Имя_регистра_расчета_(CalculationRegisterRecord.Calculation_register_name).md)>.   
Если используется структура, то состоит из двух элементов - регистратор и номер строки.

Возвращаемое значение:

Тип: [РегистрРасчетаКлючЗаписи.\<Имя регистра расчета\>](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета_(CalculationRegisterRecordKey.Calculation_register_name).md).   

Описание:

Создает ключ записи регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Отбор = Новый Структура; Отбор.Вставить("ФизЛицо", ВыбФизЛицо); Отбор.Вставить("Приказ", ВыбПриказ); Отбор.Вставить("Организация", ВыбОрганизация); Рег = РегистрыРасчета.ОсновныеНачисленияСотрудниковОрганизации; КлючЗаписи = Рег.СоздатьКлючЗаписи(Отбор); |

Использование в версии:

Доступен, начиная с версии 8.0.
