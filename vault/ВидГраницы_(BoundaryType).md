Значения

[Включая (Including)](ВидГраницы.Включая_(BoundaryType.Including).md)  
[Исключая (Excluding)](ВидГраницы.Исключая_(BoundaryType.Excluding).md)  

Описание:

Определяет набор видов границ по отношению к граничному значению. Используется для определения значения свойства [ВидГраницы](Граница.ВидГраницы_(Boundary.BoundaryType).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Граница](Граница_(Boundary).md), конструктор [На основании значения и вида](Граница.По_умолчанию.md)

Доступен, начиная с версии 8.0.
