СвойствоXDTO (XDTOProperty)

URIПространстваИмен (NamespaceURI)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
URI пространства имен представления свойства в XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для свойства с формой представления "Текст" URI пространства имен неопределено.

Использование в версии:

Доступен, начиная с версии 8.1.
