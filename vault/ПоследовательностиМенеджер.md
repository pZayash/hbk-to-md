Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПоследовательностьМенеджер.\<Имя последовательности\>](ПоследовательностьМенеджер.Имя_последовательности.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры последовательностей.  
Возможно обращение к менеджеру последовательности посредством оператора [...]. В качестве аргумента передается имя последовательности, как оно задано в конфигурации.  

Свойства:

[\<Имя последовательности\> (\<Sequence name\>)](ПоследовательностиМенеджер.Имя_последовательности.md)  

Методы:

[Восстановить (Restore)](ПоследовательностиМенеджер.Восстановить.md)  

Описание:

Предназначен для управления последовательностями и предоставляет доступ к значениям типа [ПоследовательностьМенеджер.](ПоследовательностьМенеджер.Имя_последовательности.md)<[Имя последовательности](ПоследовательностьМенеджер.Имя_последовательности.md)>. Объект также позволяет восстанавливать сразу несколько последовательностей. Доступ к объекту осуществляется через свойство глобального контекста [Последовательности](Глобальный_контекст.Последовательности.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Последовательности](Глобальный_контекст.Последовательности.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
