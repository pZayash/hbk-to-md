АдминистрированиеСервера (ServerAdministration)

ПолучитьАдминистраторов (GetAdministrators)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьАдминистраторов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеАдминистратор](АдминистрированиеАдминистратор_(AdministrationAdministrator).md).

Описание:

Получает массив администраторов центрального сервера. Для успешного выполнения метода необходима аутентификация одного из администраторов центрального сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
