# СтандартноеХранилищеНастроекМенеджер.Удалить

СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

Удалить (Delete)

Доступен, начиная с версии 8.2.

Синтаксис:

Удалить(<КлючОбъекта>, <КлючНастроек>, <ИмяПользователя>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Ключ объекта настройки.  
Если [Неопределено](lang__def_Undefined.md), то удаляются настройки для всех объектов.  
Подробнее см. в [Настройки, автоматически сохраняемые в системном хранилище](lang__SavingSettings.md).

<КлючНастроек> (обязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Ключ удаляемых настроек.  
Если [Неопределено](lang__def_Undefined.md), то удаляются настройки со всеми ключами.

<ИмяПользователя> (обязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Имя пользователя, настройки которого удаляются.  
Если [Неопределено](lang__def_Undefined.md), то удаляются настройки всех пользователей.  
Если указан пользователь, не являющийся текущим пользователем, т.е. делается попытка обратиться к настройкам, не принадлежащим текущему пользователю, то требуется наличие у текущего пользователя права АдминистрированиеДанных.

Описание:

Удаляет настройку из хранилища.  
Вызов метода требует наличия права "СохранениеДанныхПользователя".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
