ВариантСостоянияОкна (WindowStateVariant)

Обычное (Normal)

Доступен, начиная с версии 8.0.

Описание:

Обычное: окно располагается только внутри рабочей области программы.

Использование в версии:

Доступен, начиная с версии 8.0.
