Соединение с агентом сервера (IServerAgentConnection)

GetServiceSettings (GetServiceSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetServiceSettings(<Кластер>, <Сервер>)

Параметры:

<Кластер> (необязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого требуется получить список настроек сервисов.

<Сервер> (необязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Сервер, для которого требуется получить список настроек сервисов.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив содержит элементы типа [Настройка сервиса](Настройка_сервиса.md).

Описание:

Получается список настроек сервисов кластера на сервере.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
