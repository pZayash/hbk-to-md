ФормаКлиентскогоПриложения (ClientApplicationForm)

КлючНазначенияИспользования (PurposeUseKey)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Ключ назначения используется при сохранении/загрузке текущих настроек.  
Использование зависит от расширения формы.  
Во всех формах ключ назначения используется при сохранении и загрузке автоматически сохраняемых значений реквизитов формы.  
Формы, открытые с различными ключами назначения использования будут иметь собственные наборы значений автоматически сохраняемых реквизитов.  
Если режим совместимости [Версия8\_3\_4](РежимСовместимости.Версия8_3_4_(CompatibilityMode.Version8_3_4).md) или с более ранней не установлен:

- Во всех формах ключ назначения также используется при сохранении и загрузке пользовательских настроек формы. Таким образом, формы, открытые с различными ключами назначения использования, будут иметь собственные настройки состава, взаимного расположения и свойств элементов.
- При попытке присвоить свойству значение длинее 128 символов будет выдано исключение.

При создании формы заполняется на основе параметра формы [КлючНазначенияИспользования](ФормаКлиентскогоПриложения.КлючНазначенияИспользования_(ClientApplicationForm.PurposeUseKey).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
