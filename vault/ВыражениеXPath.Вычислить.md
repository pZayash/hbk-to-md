ВыражениеXPath (XPathExpression)

Вычислить (Evaluate)

Доступен, начиная с версии 8.1.

Синтаксис:

Вычислить(<УзелКонтекста>, <ТипРезультата>)

Параметры:

<УзелКонтекста> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Узел DOM - контекст для вычисления выражения XPath.   
Узел должен принадлежать тому же документу DOM, которым было создано выражение XPath (в противном случае будет вызвано исключение) и может быть следующих видов: [Документ](ТипУзлаDOM.Документ.md), [Элемент](ТипУзлаDOM.Элемент.md), [Атрибут](ТипУзлаDOM.Атрибут.md), [Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md), [Комментарий](ТипУзлаDOM.Комментарий.md), [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки.md), [ПространствоИменXPath](ТипУзлаDOM.ПространствоИменXPath.md).

<ТипРезультата> (необязательный)

Тип: [ТипРезультатаDOMXPath](ТипРезультатаDOMXPath.md).   
Тип результата вычисления выражения XPath.  
Значение по умолчанию: [Любой](ТипРезультатаDOMXPath.Любой.md).

Возвращаемое значение:

Тип: [РезультатXPath](РезультатXPath.md).   

Описание:

Производит вычисление выражения XPath для указанного узла контекста.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
