ЧтениеFastInfoset (FastInfosetReader)

ПервыйАтрибут (FirstAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

ПервыйАтрибут()

Возвращаемое значение:

Тип: Булево.   
Истина - атрибут прочитан, Ложь - в противном случае.

Описание:

Позиционирует текущее состояние объекта на первый атрибут текущего элемента.   
При этом свойствам [ЧтениеFastInfoset.ТипУзла](ЧтениеFastInfoset.ТипУзла.md), [ЧтениеFastInfoset.Имя](ЧтениеFastInfoset.Имя.md), [ЧтениеFastInfoset.ЛокальноеИмя](ЧтениеFastInfoset.ЛокальноеИмя.md), [ЧтениеFastInfoset.Префикс](ЧтениеFastInfoset.Префикс.md), [ЧтениеFastInfoset.URIПространстваИмен](ЧтениеFastInfoset.URIПространстваИмен.md), [ЧтениеFastInfoset.Значение](ЧтениеFastInfoset.Значение.md) и т.д. присваиваются значения, соответствующие прочитанным данным атрибута.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл, если текущим узлом является [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md) или [Атрибут](ТипУзлаXML.Атрибут.md).

Использование в версии:

Доступен, начиная с версии 8.1.
