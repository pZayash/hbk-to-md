ФорматированнаяСтрока (FormattedString)

На основании описания форматирования

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый ФорматированнаяСтрока(<Содержимое>, <Шрифт>, <ЦветТекста>, <ЦветФона>, <Ссылка>)

Параметры:

<Содержимое> (обязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md), [Картинка](Картинка_(Picture).md), [Массив](Массив_(Array).md).   
Содержимое строки.   
Если тип параметра [Массив](Массив_(Array).md), то элементами массива должны быть значения типа [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md) или [Картинка](Картинка_(Picture).md).   
Результирующая строка является конкатенацией элементов массива.  
Картинка должна содержаться в объекте [БиблиотекаКартинок](БиблиотекаКартинок_(PictureLib).md).

<Шрифт> (необязательный)

Тип: [Шрифт](Шрифт_(Font).md).   
Шрифт строки.

<ЦветТекста> (необязательный)

Тип: [Цвет](Цвет_(Color).md).   
Цвет текста строки.

<ЦветФона> (необязательный)

Тип: [Цвет](Цвет_(Color).md).   
Цвет фона строки.

<Ссылка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Если параметр задан и его значение не пустая строка, то конструируемая строка представляет собой гиперссылку, по нажатию на которой происходит переход по ссылке, заданной в этом параметре.

Описание:

Создает значение на основании описания форматирования.

Пример:

|  |
| --- |
| Текст = Новый ФорматированнаяСтрока("Предупреждение!", ,          WebЦвета.Красный); |

Использование в версии:

Доступен, начиная с версии 8.3.1.
