Запрос (Query)

ТребуемаяАктуальностьДанных (RequiredDataRelevance)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Чтение и запись.

Описание:

Тип: [ТребуемаяАктуальностьДанных](ТребуемаяАктуальностьДанных_(RequiredDataRelevance).md).   
Актуальность данных, которые необходимо получить. Значение по умолчанию - [Авто](ТребуемаяАктуальностьДанных.Авто_(RequiredDataRelevance.Auto).md) .   
Для запроса [Авто](ТребуемаяАктуальностьДанных.Авто_(RequiredDataRelevance.Auto).md) равно значению [Актуальные](ТребуемаяАктуальностьДанных.Актуальные_(RequiredDataRelevance.Relevant).md).  
Если у запроса свойство [ИспользованиеКопийБазыДанных](Запрос.ИспользованиеКопийБазыДанных_(Query.DatabaseCopiesUse).md) имеет значение [ИспользоватьТолькоКопии](ИспользованиеКопийБазыДанных.ИспользоватьТолькоКопии_(DatabaseCopiesUse.UseCopiesOnly).md), то свойство может влиять только на то, какая именно копия будет выбрана.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.19.
