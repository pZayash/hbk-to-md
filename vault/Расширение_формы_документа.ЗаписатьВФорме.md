Расширение формы документа (Document form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме(<РежимЗаписи>)

Параметры:

<РежимЗаписи> (необязательный)

Тип: [РежимЗаписиДокумента](РежимЗаписиДокумента.md).   
Позволяет указать режим записи документа.  
Значение по умолчанию: [Запись](РежимЗаписиДокумента.Запись.md).

Возвращаемое значение:

Тип: Булево.   
Истина - документ успешно записан; Ложь - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

См. также:

[Расширение формы документа](Расширение_формы_документа.md), свойство [ИспользоватьРежимПроведения](Расширение_формы_документа.ИспользоватьРежимПроведения.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
