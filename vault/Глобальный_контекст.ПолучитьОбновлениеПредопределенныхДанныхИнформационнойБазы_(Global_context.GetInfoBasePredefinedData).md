Глобальный контекст (Global context)

ПолучитьОбновлениеПредопределенныхДанныхИнформационнойБазы (GetInfoBasePredefinedData)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ПолучитьОбновлениеПредопределенныхДанныхИнформационнойБазы()

Возвращаемое значение:

Тип: [ОбновлениеПредопределенныхДанных](ОбновлениеПредопределенныхДанных_(PredefinedDataUpdate).md).   

Описание:

Возвращает текущее установленное в информационной базе значение режима обновления предопределенных данных.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [УстановитьОбновлениеПредопределенныхДанныхИнформационнойБазы](Глобальный_контекст.УстановитьОбновлениеПредопределенныхДанныхИнформационнойБазы_(Global_context.SetInfoBasePredefinedDataUpdate).md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
