ИзбранноеРаботыПользователя (UserWorkFavorites)

Удалить (Delete)

Доступен, начиная с версии 8.2.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Описание:

Удаляет элемент избранного из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
