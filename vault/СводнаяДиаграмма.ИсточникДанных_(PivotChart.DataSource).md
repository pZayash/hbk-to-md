СводнаяДиаграмма (PivotChart)

ИсточникДанных (DataSource)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [ПостроительОтчета](ПостроительОтчета_(ReportBuilder).md).   
Содержит источник данных сводной диаграммы. При помощи этого свойства можно наполнять сводную диаграмму данными, содержащимися в результате запроса или в построителе отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
