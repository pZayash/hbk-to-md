МенеджерКриптографии (CryptoManager)

ПолучитьКонтейнерПодписейКриптографииАсинх (GetCryptoSignaturesContainerAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьКонтейнерПодписейКриптографииАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Подпись для разбора.  
Если параметр типа [Строка](lang__def_String.md), то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) или [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md), то он задает данные подписи.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [КонтейнерПодписейКриптографии](КонтейнерПодписейКриптографии_(CryptoSignaturesContainer).md).

Описание:

Выполняет разбор двоичных данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
