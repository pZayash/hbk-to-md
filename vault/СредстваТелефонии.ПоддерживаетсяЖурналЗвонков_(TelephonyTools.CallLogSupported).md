СредстваТелефонии (TelephonyTools)

ПоддерживаетсяЖурналЗвонков (CallLogSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяЖурналЗвонков()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

- [Истина](lang__def_BooleanTrue.md) - для Android при наличии телефонного модуля,
- [Ложь](lang__def_BooleanFalse.md) - для остальных операционных систем.

Описание:

Определяет, доступен ли журнал звонков на данном устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПолучитьЖурналЗвонков](СредстваТелефонии.ПолучитьЖурналЗвонков_(TelephonyTools.GetCallLog).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
