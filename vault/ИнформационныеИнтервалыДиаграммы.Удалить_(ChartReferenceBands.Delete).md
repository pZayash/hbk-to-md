ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Удалить (Delete)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Удалить(<Интервал>)

Параметры:

<Интервал> (обязательный)

Тип: [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы_(ChartReferenceBand).md).   
Интервал, который требуется удалить.

Описание:

Удаляет указанный информационный интервал из коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
