Элементы коллекции:

[ВыражениеУпорядочиванияКомпоновкиДанных](ВыражениеУпорядочиванияКомпоновкиДанных_(DataCompositionOrderExpression).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выражения упорядочивания.  
Возможно обращение к выражению упорядочивания посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
