WebSocketКлиент (WebSocketClient)

ИмяПользователяИнформационнойБазы (InfoBaseUserName)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя пользователя, от имени которого будет запущен сеанс, в котором будет вызван обработчик события. Если имя пользователя не задано, сеанс будет запущен от имени пользователя по умолчанию.   
Изначально получается из метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
