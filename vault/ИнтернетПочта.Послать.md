ИнтернетПочта (InternetMail)

Послать (Send)

Доступен, начиная с версии 8.0.

Синтаксис:

Послать(<Сообщение>, <ОбрабатыватьТексты>, <ПротоколОтправкиПочты>)

Параметры:

<Сообщение> (обязательный)

Тип: [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение.md).   
Почтовое сообщение.

<ОбрабатыватьТексты> (необязательный)

Тип: [ОбработкаТекстаИнтернетПочтовогоСообщения](ОбработкаТекстаИнтернетПочтовогоСообщения.md).   
Управляет обработкой текстов перед отправкой:

- [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать.md) - Локальные картинки из текста, помещаются в сообщение в виде вложений. Содержимое текста письма изменяется таким образом, чтобы теги <img> указывали на картинки из вложений.
- [НеОбрабатывать](ДопустимыеДействияПеретаскивания.НеОбрабатывать.md) - Текст посылается без внутренней обработки.
- Загрузка картинок при отправке HTML-документов отключена.

  
Значение по умолчанию: [Обрабатывать](ОбработкаТекстаИнтернетПочтовогоСообщения.Обрабатывать.md).

<ПротоколОтправкиПочты> (необязательный)

Тип: [ПротоколИнтернетПочты](ПротоколИнтернетПочты.md).   
Определяет тип протокола отправки. Допустимые варианты [IMAP](ПротоколИнтернетПочты.IMAP.md) или [SMTP](ПротоколИнтернетПочты.SMTP.md).   
Указание [POP3](ПротоколИнтернетПочты.POP3.md) приведет к возникновению исключения.  
Значение по умолчанию: [SMTP](ПротоколИнтернетПочты.SMTP.md).

Возвращаемое значение:

Тип: [Соответствие](Соответствие.md).   
Если почтовый сервер не принял какой-нибудь из адресов получателей, то возвращаемое значение содержит соответствие ошибочный получатель - сообщение об ошибке.  
При возникновении других ошибочных ситуаций, генерируется исключение.

Описание:

Посылает сообщение типа [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение.md).  
При отправке сообщения, которое содержит html-текст (свойство [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста.md) установлено в значение [HTML](ТипТекстаПочтовогоСообщения.HTML.md)) также формируется простой текст. Этот текст извлекается из html-текста c некоторыми особенностями. Например, командой перевода строки считается тег <DIV>. Тег <BR> как перевод строки не трактуется и игнорируется.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если используется [IMAP](ПротоколИнтернетПочты.IMAP.md) протокол, сообщение будет послано в почтовый ящик IMAP сессии, который задан свойством [ТекущийПочтовыйЯщик](ИнтернетПочта.ТекущийПочтовыйЯщик.md).

Пример:

|  |
| --- |
| Почта = Новый ИнтернетПочта; Сообщение = Новый ИнтернетПочтовоеСообщение; Почта.Подключиться(Профиль); ОшибочныеПолучатели = Почта.Послать(Сообщение); Почта.Отключиться(); |

Использование в версии:

Доступен, начиная с версии 8.0.
