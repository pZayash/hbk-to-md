ИнтернетПочтовоеСообщение (InternetMailMessage)

ПолучитьИсходныйТекст (GetSourceText)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИсходныйТекст(<Кодировка>)

Параметры:

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка, которая будет использована для перекодирования исходных данных в текст. Если параметр не указан, то используется кодировка UTF-8.  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые названия кодировок:

- UTF-16
- UTF-16BE
- UTF-16LE
- ISO-8859-1
- US-ASCII
- gb18030
- iso-8859-2

- iso-8859-7

- iso-8859-15
- Shift\_JIS

.  
Значение по умолчанию: "UTF-8".

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает исходный текст почтового сообщения.  
Если сообщение принято с сервера (с помощью метода [Выбрать](ИнтернетПочта.Выбрать.md) объекта [ИнтернетПочта](ИнтернетПочта.md)) и не было изменено, то возвращаемый методом текст соответствуют тексту, полученному с сервера  
Если объект [ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение.md) создан программно или получен с сервера и был изменен, то текст генерируется на основе содержимого этого объекта.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
