МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьОтображениеОповещенийОбсуждения (SetConversationNotificationRepresentation)

Доступен, начиная с версии 8.3.15.

Синтаксис:

УстановитьОтображениеОповещенийОбсуждения(<Обсуждение>, <ОтображениеОповещений>, <Пользователь>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения системы взаимодействия, для которого устанавливается режим отображения оповещений.

<ОтображениеОповещений> (обязательный)

Тип: [ОтображениеОповещенийСистемыВзаимодействия](ОтображениеОповещенийСистемыВзаимодействия.md).   
Устанавливаемый режим отображения оповещений.

<Пользователь> (необязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Если задано, то устанавливает режим оповещений для соответствующего пользователя.  
Если передано значение отличное от идентификатора текущего пользователя и не установлен привилегированный режим, то генерируется исключение.

Описание:

Устанавливает режим отображения оповещений системы взаимодействия.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
