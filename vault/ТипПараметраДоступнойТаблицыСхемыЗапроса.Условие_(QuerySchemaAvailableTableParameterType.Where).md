ТипПараметраДоступнойТаблицыСхемыЗапроса (QuerySchemaAvailableTableParameterType)

Условие (Where)

Доступен, начиная с версии 8.3.5.

Описание:

В параметре указывается выражение условия, которое будет проверяться при выполнении запроса.  
Поля, которые можно использовать в условии, содержатся в свойстве [ДоступныеПоля](ПараметрДоступнойТаблицыСхемыЗапроса.ДоступныеПоля_(QuerySchemaAvailableTableParameter.AvailableFields).md) объекта типа [ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса_(QuerySchemaAvailableTableParameter).md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
