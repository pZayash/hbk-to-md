# ОбработкаРасшифровкиКомпоновкиДанных.Расшифровать

ОбработкаРасшифровкиКомпоновкиДанных (DataCompositionDetailsProcess)

Расшифровать (DrillDown)

Доступен, начиная с версии 8.1.

Синтаксис:

Расшифровать(<ИдентификаторРасшифровки>, <Поле>)

Параметры:

<ИдентификаторРасшифровки> (обязательный)

Тип: [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных_(DataCompositionDetailsID).md).   
Идентификатор обрабатываемой расшифровки.

<Поле> (обязательный)

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md).   
Поле, по которому должна быть произведена расшифровка.

Возвращаемое значение:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Если в метод передаются значения из различных группировок, то генерируется исключение категории [ОшибкаНастроекКомпоновкиДанных](КатегорияОшибки.ОшибкаНастроекКомпоновкиДанных_(ErrorCategory.DataCompositionSettingsError).md).

Описание:

Позволяет получить более детальную информацию, развернув ее по заданному полю.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
