Соединение с рабочим процессом (IWorkingProcessConnection)

CreateInfoBaseInfo (CreateInfoBaseInfo)

Доступен, начиная с версии 8.1.

Синтаксис:

CreateInfoBaseInfo()

Возвращаемое значение:

Тип: [Информационная база](Информационная_база.md).   
Значения всех строковых свойств объекта равны пустым строкам, а числовых – нулям. Свойствам созданного объекта могут быть явно установлены значения. После этого объект можно использовать в качестве значений входных параметров соответствующих методов.

Описание:

Создает объект описания одной информационной базы. Аутентификации не требуется.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
