АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолучитьРазрешенныеВнешниеМодули (GetAllowedExternalModules)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьРазрешенныеВнешниеМодули()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [РазрешенныйВнешнийМодуль](РазрешенныйВнешнийМодуль_(AllowedExternalModule).md).

Описание:

Получает список внешних модулей, исполнение которых возможно без включения безопасного режима.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
