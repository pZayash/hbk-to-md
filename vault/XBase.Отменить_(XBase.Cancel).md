# XBase.Отменить

XBase (XBase)

Отменить (Cancel)

Доступен, начиная с версии 8.0.

Синтаксис:

Отменить()

Описание:

Отменяет запись изменений в таблицу базы при смене текущей записи, если установлен признак [АвтоСохранение](XBase.АвтоСохранение_(XBase.AutoSave).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Текст = "Сохранить изменения?"; Режим = РежимДиалогаВопрос.ДаНет; Если Вопрос(Текст, Режим) = КодВозвратаДиалога.Нет Тогда     xB.Отменить(); КонецЕсли; |

См. также:

[XBase](XBase_(XBase).md), свойство [АвтоСохранение](XBase.АвтоСохранение_(XBase.AutoSave).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
