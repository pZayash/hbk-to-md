# Расширение группы формы для обычной группы.ИспользуемаяТаблица

Расширение группы формы для обычной группы (Form group extension for a usual group)

ИспользуемаяТаблица (AssociatedTable)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [ТаблицаФормы](ТаблицаФормы_(FormTable).md).   
Используется вместе со свойством [ИспользованиеТекущейСтроки](Расширение_группы_формы_для_обычной_группы.ИспользованиеТекущейСтроки_(Form_group_extension_for_a_usual_group.CurrentRowUse).md) для указания таблицы, текущая строка которой используется.  
Таблица формы, которая устанавливается значением данного свойства, должна относиться к той же форме, что и владелец свойства. При выборе таблицы из другой формы, будет сгенерировано исключение.  
Данную особенность следует учитывать при использовании метода [ЗаполнитьЗначенияСвойств](Глобальный_контекст.ЗаполнитьЗначенияСвойств_(Global_context.FillPropertyValues).md) для объектов разных форм.

Доступность:

Мобильный клиент, сервер.

Использование в версии:

Доступен, начиная с версии 8.3.12.
