Расширение поля формы для поля ввода (Form field extension for a text box)

АвтоОтметкаНезаполненного (AutoMarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Указывает, нужно ли формировать автоматически состояние свойства [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) в зависимости от устанавливаемого или формируемого в поле ввода значения.   
Если Истина, тогда при установке и формировании значения в поле ввода будет меняться состояние свойства [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного.md): если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного.md) будет выставлено в Истина, иначе - в Ложь.  
В случае, если для поля в свойствах метаданных "Автоотметка незаполненного" установлено "Авто":

- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Выдавать ошибку", то свойство принимает значение Истина, и соответственно, [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) должно зависеть от того, является ли значение реквизита значением по умолчанию;
- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Не проверять", то свойство [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) должно принимать значение Ложь, и соответственно, [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) не зависит от того, является ли значение реквизита значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
