РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По записи

Синтаксис:

Сдвинуть(<Запись>, <Смещение>)

Параметры:

<Запись> (обязательный)

Тип: [РегистрБухгалтерииЗапись.\<Имя регистра бухгалтерии\>](РегистрБухгалтерииЗапись.Имя_регистра_бухгалтерии_(AccountingRegisterRecord.Accounting_register_name).md).   
Перемещаемая запись набора.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца списка записей набора (увеличение индекса), отрицательное - к началу.

Вариант синтаксиса: По индексу

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция сдвигаемой записи в наборе.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое следует сдвинуть запись. Положительное значение соответствует смещению записи в сторону конца списка записей набора (увеличение индекса), отрицательное - к началу.

Описание:

Сдвигает запись в наборе на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
