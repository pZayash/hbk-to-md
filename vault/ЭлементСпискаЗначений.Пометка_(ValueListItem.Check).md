ЭлементСпискаЗначений (ValueListItem)

Пометка (Check)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит связанное с элементом списка значение пометки.  
[Ложь](lang__def_BooleanFalse.md) - пометка не установлена, [Истина](lang__def_BooleanTrue.md) - пометка установлена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
