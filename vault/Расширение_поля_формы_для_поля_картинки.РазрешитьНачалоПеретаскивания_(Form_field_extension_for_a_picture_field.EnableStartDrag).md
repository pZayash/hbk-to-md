Расширение поля формы для поля картинки (Form field extension for a picture field)

РазрешитьНачалоПеретаскивания (EnableStartDrag)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Включает режим перетаскивания у источника.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - перетаскивание разрешено; [Ложь](lang__def_BooleanFalse.md) - запрещено.

Использование в версии:

Доступен, начиная с версии 8.2.
