Описание:

Содержит список агрегатов регистра накопления.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise}. Имя типа XDTO: AggregatesInformation.
