ЗначениеПараметраНастроекКомпоновкиДанных (DataCompositionSettingsParameterValue)

Значение (Value)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Определяет значение параметра. Его тип определяется соответствующим доступным параметром.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
