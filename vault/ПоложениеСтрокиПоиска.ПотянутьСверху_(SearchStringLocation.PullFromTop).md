ПоложениеСтрокиПоиска (SearchStringLocation)

ПотянутьСверху (PullFromTop)

Доступен, начиная с версии 8.3.9.

Описание:

Строка поиска располагается над первой строкой таблицы, и первоначально скрыта. Чтобы отобразить строку поиска, необходимо перетянуть содержимое таблицы сверху вниз.

Использование в версии:

Доступен, начиная с версии 8.3.9.
