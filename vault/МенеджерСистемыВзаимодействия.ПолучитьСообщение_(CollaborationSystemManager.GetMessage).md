МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьСообщение (GetMessage)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьСообщение(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [ИдентификаторСообщенияСистемыВзаимодействия](ИдентификаторСообщенияСистемыВзаимодействия_(CollaborationSystemMessageID).md).   
Идентификатор сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).   

Описание:

Возвращает сообщение системы взаимодействия по его идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
