ТочкиМаршрута (BusinessProcessRoutePoints)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ТочкаМаршрутаБизнесПроцесса>)

Параметры:

<ТочкаМаршрутаБизнесПроцесса> (необязательный)

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс переданной точки маршрута бизнес-процесса. Если точка не принадлежит коллекции точек маршрута бизнес-процесса, возвращает -1.

Описание:

Получает индекс точки в коллекции точек маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
