Доступен, начиная с версии 8.1.

Элементы коллекции:

[СвязьДендрограммы](СвязьДендрограммы_(DendrogramLink).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются связи коллекции.  
Возможно обращение к связи коллекции посредством оператора [...]. В качестве аргумента передается индекс связи (нумерация с 0).
