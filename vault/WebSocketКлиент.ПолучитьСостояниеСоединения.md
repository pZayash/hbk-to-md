WebSocketКлиент (WebSocketClient)

ПолучитьСостояниеСоединения (GetConnectionState)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьСостояниеСоединения()

Возвращаемое значение:

Тип: [СостояниеWebSocketСоединения](СостояниеWebSocketСоединения.md).   

Описание:

Возвращает состояние WebSocket-соединения.  
Если у клиента нет объекта соединения, возвращает [Закрыто](СостояниеWebSocketСоединения.Закрыто.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
