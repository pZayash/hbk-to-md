СерииДиаграммы (ChartSeriesCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Вставить(<Индекс>, <Текст>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Если индекс больше текущего количества серий, то новая серия вставляется в конец коллекции. Если индекс меньше нуля, то новая серия вставляется в начало коллекции.

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Название серии.

Возвращаемое значение:

Тип: [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Возвращает вставленную серию.

Описание:

Вставляет новую серию по указанному индексу. Все серии с индексом больше, либо равным переданному, сдвигаются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
