ТекстовыйДокумент (TextDocument)

НачатьЗапись (BeginWriting)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ИмяФайла>, <ТипФайла>, <РазделительСтрок>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения записи файла со следующими параметрами:

- <Результат> - результат операции.
  - [Истина](lang__def_BooleanTrue.md), если запись была инициирована с помощью расширений для браузера и пользователь согласился на его получение, то есть запись файла была завершена.
  - [Ложь](lang__def_BooleanFalse.md), если запись была инициирована с помощью расширений для браузера, но пользователь не согласился на его получение, то есть запись файла не была завершена.
  - [Неопределено](lang__def_Undefined.md) если запись была инициирована скачиванием файла средствами браузера и неизвестно - был ли файл записан или нет.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершении никакая процедура вызвана не будет.

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла для сохранения текста.

<ТипФайла> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Тип кодировки файла. Тип файла может быть задан как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Задается строка, являющаяся разделителем строк, к которой при записи файла будет преобразован символ [ПС](Символы.ПС_(Chars.LF).md). Если строка пустая, то разделители строк при записи не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК_(Chars.CR).md);
- [ПС](Символы.ПС_(Chars.LF).md);
- [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

Описание:

Вызывает начало записи текстового документа в файл с указанным именем.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

При работе на веб-клиенте вызов метода выполняет обращение к серверу.
