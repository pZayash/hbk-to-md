GCM (GCM)

Не рекомендуется использовать, начиная с версии 8.3.13.

Рекомендуется использовать:

- [FCM](ТипПодписчикаДоставляемыхУведомлений.FCM.md)

Доступен, начиная с версии 8.3.6.

Описание:

Google Cloud Messaging. Сервис, используемый для доставки push-уведомлений на устройства под управлением ОС Android.  
Данный сервис в апреле 2018 был деактивирован и удален из API Google.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Не рекомендуется использовать, начиная с версии 8.3.13.

Описание изменено в версии 8.3.13.
