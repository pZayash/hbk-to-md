МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьПредставленияШаблонаСообщения (SetMessageTemplatePresentations)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьПредставленияШаблонаСообщения(<Идентификатор>, <ПредставлениеШаблона>, <ПредставленияПодстановок>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия.md).   
Идентификатор шаблона сообщения, для которого устанавливается представление.

<ПредставлениеШаблона> (обязательный)

Тип: Строка.   
Представление шаблона.

<ПредставленияПодстановок> (необязательный)

Тип: [Массив](Массив.md).   
Массив строк - список представлений подстановок.

Описание:

Для интеграции типа WhatsApp устанавливает представление шаблона и представления его подстановок.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [УстановитьПредставленияШаблонаСообщенияАсинх](МенеджерСистемыВзаимодействия.УстановитьПредставленияШаблонаСообщенияАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
