Соединение с агентом сервера (IServerAgentConnection)

GetSessions (GetSessions)

Доступен, начиная с версии 8.2.

Синтаксис:

GetSessions(<Кластер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого необходимо получить список сеансов. Должен быть получен методом [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив, который содержит список объектов [Сеанс](Сеанс_(ISessionInfo).md).

Описание:

Получает список сеансов, работающих с данным кластером.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
