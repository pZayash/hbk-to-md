# ИтераторУзловDOM.Отсоединить

ИтераторУзловDOM (DOMNodeIterator)

Отсоединить (Detach)

Доступен, начиная с версии 8.1.

Синтаксис:

Отсоединить()

Описание:

Отсоединяет итератор от итерируемого документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Обращение к следующему или предыдущему узлу после отсоединения итератора приведет к вызову исключения.

Использование в версии:

Доступен, начиная с версии 8.1.
