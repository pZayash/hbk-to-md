ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

IPАдресАгента (AgentIPAddress)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Адрес агента сервера 1С:Предприятия, на который будет установлена лицензия. Параметры этого компьютера будут использованы в запросе к центру лицензирования.   
Если пустая строка, то используются параметры текущего компьютера.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
