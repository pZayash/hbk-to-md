# АтрибутDOM.ТекстовоеСодержимое

АтрибутDOM (DOMAttribute)

ТекстовоеСодержимое (TextContent)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Текстовое содержимое узла DOM.  
Содержит текстовое значение узла и всех его потомков. Значение не содержит разметки.  
Никакой нормализации пробельных символов не выполняется, и возвращаемая строка не содержит пробельных символов в содержимом элемента (см. свойство [ЭтоПробельныеСимволыЭлемента](ТекстDOM.ЭтоПробельныеСимволыЭлемента_(DOMText.IsElementContentWhitespace).md)).  
Принцип формирования текстового значения узла:  
[Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md), [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), [Сущность](ТипУзлаDOM.Сущность_(DOMNodeType.Entity).md), [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность_(DOMNodeType.EntityReference).md), [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md) - объединение текстового содержимого каждого из дочерних узлов, исключая [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md); [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки_(DOMNodeType.ProcessingInstruction).md). Пустая строка, если дочерних узлов нет;  
[Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA_(DOMNodeType.CDATASection).md), [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md); [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки_(DOMNodeType.ProcessingInstruction).md) - значение свойства [АтрибутDOM.ЗначениеУзла](АтрибутDOM.ЗначениеУзла_(DOMAttribute.NodeValue).md);  
[Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md), [Нотация](ТипУзлаDOM.Нотация_(DOMNodeType.Notation).md) - [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
