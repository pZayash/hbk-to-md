Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Признак автоматической генерации текста запроса по указанной главной таблице.  
Ложь - выполняется автоматическая генерация текста запроса.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Вне сервера "1С:Предприятие" не используется.

Использование в версии:

Доступен, начиная с версии 8.2.
