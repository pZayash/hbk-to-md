# Глобальный контекст.СтрНайтиВсеПоРегулярномуВыражению

Глобальный контекст (Global context)

СтрНайтиВсеПоРегулярномуВыражению (StrFindAllByRegularExpression)

Доступен, начиная с версии 8.3.23.

Синтаксис:

СтрНайтиВсеПоРегулярномуВыражению(<Строка>, <РегулярноеВыражение>, <ИгнорироватьРегистр>, <МногострочныйПоиск>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Исходная строка.

<РегулярноеВыражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Регулярное выражение.

<ИгнорироватьРегистр> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - без учета регистра, [Ложь](lang__def_BooleanFalse.md) - с учетом регистра.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<МногострочныйПоиск> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - выполняется многострочный поиск: символу "^" в шаблоне регулярного выражения соответствует начало каждой строки значения параметра <Строка>, символу "$" - конец каждой строки.  
  
[Ложь](lang__def_BooleanFalse.md) - символу "^" в шаблоне регулярного выражения соответствует начало значения в параметре <Строка>, символу "$" - конец текста.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [РезультатПоискаПоРегулярномуВыражению](РезультатПоискаПоРегулярномуВыражению_(ResultOfSearchByRegularExpression).md).

Описание:

Находит все вхождения регулярного выражения в исходной строке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.26.
