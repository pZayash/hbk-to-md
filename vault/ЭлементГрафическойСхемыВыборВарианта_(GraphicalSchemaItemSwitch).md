Описание:

Отображает точку маршрута бизнес-процесса [ВыборВарианта](ВидТочкиМаршрутаБизнесПроцесса.ВыборВарианта_(BusinessProcessRoutePointType.Switch).md), имеющую несколько выходов (вариантов), из которых в зависимости от значения возвращаемого параметра <Вариант> обработчика события [ОбработкаВыбораВарианта](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ОбработкаВыбораВарианта_(BusinessProcessRoutePointRef.Business_process_name.SwitchProcessing).md) выбирается только один.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
