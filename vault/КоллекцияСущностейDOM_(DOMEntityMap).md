# КоллекцияСущностейDOM

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СущностьDOM](СущностьDOM_(DOMEntity).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются узле сущностей DOM.  
Возможно обращение к узлу сущности DOM посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияСущностейDOM.Количество_(DOMEntityMap.Count).md)  
[ПолучитьИменованныйЭлемент (GetNamedItem)](КоллекцияСущностейDOM.ПолучитьИменованныйЭлемент_(DOMEntityMap.GetNamedItem).md)  
[УдалитьИменованныйЭлемент (RemoveNamedItem)](КоллекцияСущностейDOM.УдалитьИменованныйЭлемент_(DOMEntityMap.RemoveNamedItem).md)  
[УстановитьИменованныйЭлемент (SetNamedItem)](КоллекцияСущностейDOM.УстановитьИменованныйЭлемент_(DOMEntityMap.SetNamedItem).md)  
[Элемент (Item)](КоллекцияСущностейDOM.Элемент_(DOMEntityMap.Item).md)  

Описание:

Коллекция узлов сущностей DOM.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), свойство [Сущности](ОпределениеТипаДокументаDOM.Сущности_(DOMDocumentType.Entities).md)
