СредстваПочты (MailTools)

ПоддерживаетсяОтправка (SendingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяОтправка()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md):

- на Android - всегда,
- на iOS - если на устройстве настроено приложение для отправки почты,
- на Windows Phone - на телефонах всегда. Никогда на планшетах.

Описание:

Определяет доступность отправки почты на данном устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваПочты](СредстваПочты_(MailTools).md), метод [Послать](СредстваПочты.Послать_(MailTools.Send).md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
