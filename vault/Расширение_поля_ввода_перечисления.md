Описание:

Расширение предоставляет возможность автоматического поиска и подбора значения перечисления в элементе управления [ПолеВвода](ПолеВвода.md) по введенному тексту. При этом поиск осуществляется по его текстовому представлению.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
