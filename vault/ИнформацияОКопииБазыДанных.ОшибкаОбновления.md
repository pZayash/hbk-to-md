ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ОшибкаОбновления (UpdateError)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md), Неопределено.   
При отсутствии ошибок предыдущего процесса обновления свойство содержит значение Неопределено. При возникновении ошибки - объект, описывающий ошибку, которая послужила причиной прекращения предыдущего процесса обновления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
