Элементы коллекции:

[ЗначениеXDTO](ЗначениеXDTO.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются значения XDTO.  
Возможно обращение к значению XDTO посредством оператора [...]. В качестве аргумента передается индексу.
