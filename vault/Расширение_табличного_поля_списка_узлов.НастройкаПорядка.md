Расширение табличного поля списка узлов (Node list table box extension)

НастройкаПорядка (OrderSetting)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаПорядка](НастройкаПорядка.md).   
Предоставляет доступ к настройкам порядка списка узлов. Представляет собой коллекцию значений типа [ЭлементУправленияПорядком](ЭлементУправленияПорядком.md). Свойства коллекции содержат объекты для управления элементом порядка. Имена свойств совпадают с именами доступных для списка узлов элементов упорядочивания.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
