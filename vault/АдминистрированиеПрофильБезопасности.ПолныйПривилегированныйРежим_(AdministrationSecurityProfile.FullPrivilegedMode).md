АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйПривилегированныйРежим (FullPrivilegedMode)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, разрешена ли установка привилегированного режима в безопасном режиме исполнения кода:

- [Истина](lang__def_BooleanTrue.md) – разрешена.
- [Ложь](lang__def_BooleanFalse.md) – запрещена.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.16.
