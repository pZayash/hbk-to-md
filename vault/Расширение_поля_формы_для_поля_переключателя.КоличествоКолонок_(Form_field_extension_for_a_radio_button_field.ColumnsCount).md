# Расширение поля формы для поля переключателя.КоличествоКолонок

Расширение поля формы для поля переключателя (Form field extension for a radio button field)

КоличествоКолонок (ColumnsCount)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Количество колонок, в которых будут располагаться переключатели.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
