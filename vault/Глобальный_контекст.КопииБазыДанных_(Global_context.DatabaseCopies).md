Глобальный контекст (Global context)

КопииБазыДанных (DatabaseCopies)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: [МенеджерКопийБазыДанных](МенеджерКопийБазыДанных_(DatabaseCopiesManager).md).   
Используется для доступа к копиям базы данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [КопииБазыДанных](Глобальный_контекст.КопииБазыДанных_(Global_context.DatabaseCopies).md)  

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
