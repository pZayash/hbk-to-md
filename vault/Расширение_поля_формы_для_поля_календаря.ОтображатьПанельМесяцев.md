Расширение поля формы для поля календаря (Form field extension for a calendar field)

ОтображатьПанельМесяцев (ShowMonthsPanel)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, требуется ли отображать панель месяцев. Истина - панель отображается.  
Значение по умолчанию Ложь.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
