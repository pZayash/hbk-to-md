ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

Автонумерация (Autonumbering)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если свойство установлено в значение Истина, то новому объекту базы данных система будет автоматически присваивать очередной порядковый номер/код.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
