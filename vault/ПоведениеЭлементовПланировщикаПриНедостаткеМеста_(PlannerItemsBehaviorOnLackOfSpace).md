Описание:

Содержит варианты поведения элементов планировщика при недостатке места в поле планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. PlannerItemsBehaviorOnCompression.

Использование в версии:

Доступен, начиная с версии 8.3.15.
