ТабличныйДокумент (SpreadsheetDocument)

УдалитьОбласть (DeleteArea)

Доступен, начиная с версии 8.0.

Синтаксис:

УдалитьОбласть(<ИсходнаяОбласть>, <Сдвиг>)

Параметры:

<ИсходнаяОбласть> (обязательный)

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md).   
Удаляемая область.

<Сдвиг> (необязательный)

Тип: [ТипСмещенияТабличногоДокумента](ТипСмещенияТабличногоДокумента.md).   
Определяет смещение ячеек таблицы при удалении.  
Значение по умолчанию: [БезСмещения](ТипСмещенияТабличногоДокумента.БезСмещения.md).

Описание:

Удаляет область из табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Смещать = ТипСмещенияТабличногоДокумента.ПоВертикали; Область = ТабДок.Область("R2C2:R2C2"); ТабДок.УдалитьОбласть(Область, Смещать); |

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), метод [ВставитьОбласть](ТабличныйДокумент.ВставитьОбласть.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
