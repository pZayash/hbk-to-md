# ОбъектМетаданных: БизнесПроцесс.КартаМаршрута

ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

КартаМаршрута (Flowchart)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   
Содержит карту маршрута бизнес-процесса, которая описывает логику бизнес-процесса от точки старта до точки завершения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
Для получения карты маршрута можно использовать метод [ПолучитьКартуМаршрута](БизнесПроцессМенеджер.Имя_бизнес-процесса.ПолучитьКартуМаршрута_(BusinessProcessManager.Business_process_name.GetFlowchart).md)() менеджера соответствующего бизнес-процесса.

Использование в версии:

Доступен, начиная с версии 8.1.
