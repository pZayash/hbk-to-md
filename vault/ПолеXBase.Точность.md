ПолеXBase (XBaseField)

Точность (Precision)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит количество разрядов дробной части.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Используется для числовых полей.

Использование в версии:

Доступен, начиная с версии 8.0.
