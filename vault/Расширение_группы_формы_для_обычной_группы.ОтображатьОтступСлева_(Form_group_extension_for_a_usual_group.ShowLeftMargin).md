# Расширение группы формы для обычной группы.ОтображатьОтступСлева

Расширение группы формы для обычной группы (Form group extension for a usual group)

ОтображатьОтступСлева (ShowLeftMargin)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержимое свертываемой группы отображается в развернутом виде с отступом слева.  
Данное свойство не используется, если [Отображение](Расширение_группы_формы_для_обычной_группы.Отображение_(Form_group_extension_for_a_usual_group.Representation).md) = [СильноеВыделение](ОтображениеОбычнойГруппы.СильноеВыделение_(UsualGroupRepresentation.StrongSeparation).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.3.
