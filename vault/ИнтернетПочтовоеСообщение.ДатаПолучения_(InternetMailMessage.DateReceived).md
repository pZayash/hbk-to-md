ИнтернетПочтовоеСообщение (InternetMailMessage)

ДатаПолучения (DateReceived)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Дата получения письма. Имеет смысл только для полученных писем.  
Значение представлено локальной датой клиента или сервера "1С:Предприятие", в зависимости от того, где был вызван метод [Выбрать](ИнтернетПочта.Выбрать_(InternetMail.Get).md) или метод [ПолучитьЗаголовки](ИнтернетПочта.ПолучитьЗаголовки_(InternetMail.GetHeaders).md) объекта [ИнтернетПочта](ИнтернетПочта_(InternetMail).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
