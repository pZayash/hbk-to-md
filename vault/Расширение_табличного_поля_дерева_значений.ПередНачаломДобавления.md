Расширение табличного поля дерева значений (Value tree table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>, <Родитель>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение Истина, то строка списка узлов добавлена не будет.  
Значение по умолчанию: Ложь.

<Копирование>

Тип: Булево.   
Определяет режим копирования. Если установлено Истина, то происходит копирование строки.

<Родитель>

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений.md).   
Строка-родитель, в которую будет добавляться новая строка. Если добавляется в корень дерева, то значение Неопределено

Описание:

Вызывается перед началом добавления строки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
