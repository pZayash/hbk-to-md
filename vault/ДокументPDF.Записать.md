ДокументPDF (PDFDocument)

Записать (Write)

Доступен, начиная с версии 8.3.21.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>, <Пароль>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла для записи PDF-документа.

<Пароль> (необязательный)

Тип: Строка.   
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: [ТипФайлаДокументаPDF](ТипФайлаДокументаPDF.md), Неопределено.   
Тип файла документа PDF. Если содержит Неопределено, то в качестве значения параметра используется свойство [ТипФайла](ДокументPDF.ТипФайла.md) объекта [ДокументPDF](ДокументPDF.md).

Описание варианта метода:

Записывает документ PDF.  
Если документ имеет электронную цифровую подпись и выполняется запись с паролем, то генерируется исключение.  
  
Доступность:  
Тонкий клиент, сервер, толстый клиент.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>, <Пароль>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи PDF-документа.

<Пароль> (необязательный)

Тип: Строка.   
Пароль для доступа к PDF-документу.

<ТипФайла> (необязательный)

Тип: [ТипФайлаДокументаPDF](ТипФайлаДокументаPDF.md), Неопределено.   
Тип файла документа PDF. Если содержит Неопределено, то в качестве значения параметра используется свойство [ТипФайла](ДокументPDF.ТипФайла.md) объекта [ДокументPDF](ДокументPDF.md).

Описание варианта метода:

Записывает документ в поток PDF-документа.  
Если документ имеет электронную цифровую подпись и производится запись с паролем, то генерируется исключение.

Описание:

Записывает изменения в PDF-документ.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.
