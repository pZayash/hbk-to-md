Описание:

Тип: Булево.   
Содержит признак генерации текста для новых точек по схеме "ТочкаN". Если Ложь, то у новой точки будет пустой текст.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
