Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступныйОбъектНастройкиКомпоновкиДанных](ДоступныйОбъектНастройкиКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные объекты настройки.  
Возможно обращение к доступному объекту настройки посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Индекс.md)  
[Количество (Count)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Количество.md)  
[Найти (Find)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Найти.md)  
[Получить (Get)](КоллекцияДоступныхОбъектовНастройкиКомпоновкиДанных.Получить.md)  

Описание:

Коллекция доступных объектов настройки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
