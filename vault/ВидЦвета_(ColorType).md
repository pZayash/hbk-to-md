Описание:

Определяет набор значений видов цветов. Используется для определения вида цвета объекта типа [Цвет](Цвет_(Color).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Цвет](Цвет_(Color).md), свойство [Вид](Цвет.Вид_(Color.Type).md)  

Использование в версии:
