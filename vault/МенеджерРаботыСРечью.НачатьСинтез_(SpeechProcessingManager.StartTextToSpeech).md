МенеджерРаботыСРечью (SpeechProcessingManager)

НачатьСинтез (StartTextToSpeech)

Доступен, начиная с версии 8.3.25.

Синтаксис:

НачатьСинтез(<Текст>, <Ключ>, <Голос>, <ПараметрыСинтеза>, <ОбработчикОкончанияСинтеза>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст, по которому генерируется речь.

<Ключ> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор генерации речи.  
Если уже существует активное задание с указанным идентификатором, генерируется исключение.

<Голос> (необязательный)

Тип: [Строка](lang__def_String.md).   
Голос синтеза речи.  
Если не указан – голос определяется сервисом.

<ПараметрыСинтеза> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Дополнительные параметры синтеза. Возможный список параметров и их значений зависит от выбранного голоса; будет получен с помощью метода [ПолучитьДоступныеГолосаСинтеза](МенеджерРаботыСРечью.ПолучитьДоступныеГолосаСинтеза_(SpeechProcessingManager.GetTextToSpeechAvailableVoices).md).  
  
Заполняется на основании [ОписаниеПараметраГолосаСинтезаРечи](ОписаниеПараметраГолосаСинтезаРечи_(TextToSpeechVoiceParameterDescription).md).

<ОбработчикОкончанияСинтеза> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после окончания генерации речи со следующими параметрами:

- <РезультатГенерации> – тип [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md). Файл WAV с результатом.
- <ДополнительныеПараметры> – значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Запускает синтез речи. Полученные данные в потоковом режиме отправляются на аудиоустройство по умолчанию. При параллельном начале синтеза речи, речь синтезируется по очереди в порядке начала синтеза, т.е. одно воспроизведение синтезированной речи не накладывается на другое воспроизведение синтезированной речи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.
