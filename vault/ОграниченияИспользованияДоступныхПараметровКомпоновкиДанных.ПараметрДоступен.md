ОграниченияИспользованияДоступныхПараметровКомпоновкиДанных (DataCompositionAvailableParameterUseRestrictions)

ПараметрДоступен (ParameterEnabled)

Доступен, начиная с версии 8.2.

Синтаксис:

ПараметрДоступен(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ОграничениеИспользованияДоступногоПоляКомпоновкиДанных](ОграничениеИспользованияДоступногоПоляКомпоновкиДанных.md).   
Доступное поле компоновки данных, доступность которого (отсутствие ограничений) проверяется.

Возвращаемое значение:

Тип: Булево.   

Описание:

Позволяет определить, наложено ли ограничение использования на данный параметр.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
