# РегистрСведенийМенеджерЗаписи.<Имя регистра сведений>.Прочитать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md) › [РегистрСведенийМенеджерЗаписи.<Имя регистра сведений>](РегистрСведенийМенеджерЗаписи.Имя_регистра_сведений_(InformationRegisterRecordManager.Information_register_name).md) › [Методы](РегистрСведенийМенеджерЗаписи.Имя_регистра_сведений__Методы.md)

РегистрСведенийМенеджерЗаписи.<Имя регистра сведений> (InformationRegisterRecordManager.<Information register name>)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать()

Описание:

Считывает запись регистра сведений по установленным значениям ключевых полей, включая разделители с уровнем разделения [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно_(CommonAttributeSeparatedDataUse.IndependentlyAndSimultaneously).md), в состав которых входит регистр. При этом, если указать значение разделителя, отличное от используемого в сеансе или не указать не используемое в сеансе, [Выбран](РегистрСведенийМенеджерЗаписи.Имя_регистра_сведений.Выбран_(InformationRegisterRecordManager.Information_register_name.Selected).md) вернет [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Курс = РегистрыСведений.КурсыВалют.СоздатьМенеджерЗаписи(); Курс.Валюта = ВыбВалюта; Курс.Период = ТекущаяДата(); Курс.Прочитать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
