МаскаXS (XSWildcard)

Допускает (Allow)

Доступен, начиная с версии 8.1.

Синтаксис:

Допускает(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен, которое проверяется на соответствие маске.

Возвращаемое значение:

Тип: Булево.   

Описание:

Определяет допустимость URI пространства имен данной маской.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
