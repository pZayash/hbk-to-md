ПараметрыПотоковогоРаспознаванияРечи (SpeechToTextStreamingParameters)

ПолучатьПромежуточныеРезультаты (GetIntermediateResults)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак получения промежуточных гипотез для фраз.  
Если установлено [Истина](lang__def_BooleanTrue.md), то для каждого из результатов распознавания речи будут получены промежуточные гипотезы.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
