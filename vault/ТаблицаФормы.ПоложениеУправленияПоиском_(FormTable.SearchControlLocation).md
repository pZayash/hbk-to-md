ПоложениеУправленияПоиском (SearchControlLocation)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеУправленияПоиском](ПоложениеУправленияПоиском_(SearchControlLocation).md).   
Задает расположение автоматически сформированного элемента управления поиском в таблице.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
