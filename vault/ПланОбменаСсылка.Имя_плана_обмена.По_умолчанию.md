ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый ПланОбменаСсылка.<Имя плана обмена>(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор ссылки.   
Если идентификатор не указан, создается пустая ссылка.

Описание:

Создает новый объект на основе переданных параметров.  
Такая форма записи будет вызывать ошибку при проверке модулей в конфигураторе, так как при проверке модулей в конфигураторе не определены типы прикладных объектов.  
Также может быть создан с использованием функциональной формы оператора Новый.   
Например, ПланОбмена = Новый(Тип("ПланОбмена.Основной"));

Использование в версии:

Доступен, начиная с версии 8.2.
