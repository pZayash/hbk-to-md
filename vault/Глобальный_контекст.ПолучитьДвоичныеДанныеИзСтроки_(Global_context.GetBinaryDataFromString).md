Глобальный контекст (Global context)

ПолучитьДвоичныеДанныеИзСтроки (GetBinaryDataFromString)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьДвоичныеДанныеИзСтроки(<Строка>, <КодировкаТекста>, <ЗаписатьBOM>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, которую требуется преобразовать в [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Кодировка текста может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.  
Поддерживаемые названия кодировок:

.  
Значение по умолчанию: [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли добавлена метка порядка байт (BOM) кодировки текста в начало данных.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   

Описание:

Преобразует строку в значение типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md) с учетом кодировки текста.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
