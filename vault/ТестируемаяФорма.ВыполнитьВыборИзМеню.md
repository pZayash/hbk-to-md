ТестируемаяФорма (TestedForm)

ВыполнитьВыборИзМеню (ExecuteChoiceFromMenu)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ВыполнитьВыборИзМеню(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Строка, Число.   
Представление выбираемого значения или номер значения (индекс строки, нумерация с 0).

Описание:

Выбирает значение из открытого меню по его представлению в меню.  
Важно! Выбор из элементов вложенного контекстного меню не поддерживается.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.8.
