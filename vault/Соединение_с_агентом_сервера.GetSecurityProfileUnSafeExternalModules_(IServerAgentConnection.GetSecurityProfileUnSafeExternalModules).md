Соединение с агентом сервера (IServerAgentConnection)

GetSecurityProfileUnSafeExternalModules (GetSecurityProfileUnSafeExternalModules)

Доступен, начиная с версии 8.3.3.

Синтаксис:

GetSecurityProfileUnSafeExternalModules(<Кластер>, <Профиль>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, для которого должен быть получен список разрешенных внешних модулей.

<Профиль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя профиля безопасности, которому принадлежит список.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [Внешний модуль](Внешний_модуль_(ISecurityProfileExternalModule).md).

Описание:

Получает список внешних модулей ([ВнешнийОтчет](ВнешнийОтчет_(ExternalReport).md), [ВнешняяОбработка](ВнешняяОбработка_(ExternalDataProcessor).md)), исполнение которых возможно без включения безопасного режима.

Доступность:

Интеграция.

Примечание:

Если профиль с этим именем не найден, то исключение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
