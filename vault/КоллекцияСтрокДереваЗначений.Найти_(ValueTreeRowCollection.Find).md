КоллекцияСтрокДереваЗначений (ValueTreeRowCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Значение>, <Колонки>, <ВключатьПодчиненные>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Искомое значение.

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Список имен колонок, в которых будет осуществляться поиск, разделенных запятыми.  
Если параметр не указан, поиск осуществляется по всем колонкам дерева.  
Значение по умолчанию: Пустая строка.

<ВключатьПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли участвовать в поиске строки подчиненных коллекций (если таковые имеются). [Истина](lang__def_BooleanTrue.md) - строки подчиненных коллекций участвуют в поиске.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md), [Неопределено](lang__def_Undefined.md).   
Строка, которая содержит искомое значение. Если значение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск значения в дереве в указанных колонках коллекции строк дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод эффективно использовать для поиска уникальных значений.

Пример:

|  |
| --- |
| НайденнаяСтрока = СтрокиСостава.Найти(ТекЗатрата, "Затрата"); Если НайденнаяСтрока = Неопределено Тогда     Предупреждение(НСтр("ru = 'Затрата не найдена!';"      + " en = 'Expense not found!'")); Иначе     Количество = НайденнаяСтрока.Норма; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
