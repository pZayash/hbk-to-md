# ДокументОбъект.<Имя документа>.Записать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md) › [ДокументОбъект.<Имя документа>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md) › [Методы](ДокументОбъект.Имя_документа__Методы.md)

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<РежимЗаписи>, <РежимПроведения>)

Параметры:

<РежимЗаписи> (необязательный)

Тип: [РежимЗаписиДокумента](РежимЗаписиДокумента_(DocumentWriteMode).md).   
Позволяет выбрать один из возможных режимов записи.  
Значение по умолчанию: [Запись](РежимЗаписиДокумента.Запись_(DocumentWriteMode.Write).md).

<РежимПроведения> (необязательный)

Тип: [РежимПроведенияДокумента](РежимПроведенияДокумента_(DocumentPostingMode).md).   
Позволяет выбрать один из возможных режимов проведения.  
Значение по умолчанию: [Неоперативный](РежимПроведенияДокумента.Неоперативный_(DocumentPostingMode.Regular).md).

Описание:

Записывает документ в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для документов с поддержкой уникальности номеров в процессе записи, если нужно, проверяется уникальность номера.

Пример:

|  |
| --- |
| Попытка     НовДокумент.Записать(РежимЗаписиДокумента.Проведение); Исключение     Предупреждение(НСтр("ru = 'Невозможно провести документ.';"      + " en = 'Document needs reposting.'")); КонецПопытки; |
