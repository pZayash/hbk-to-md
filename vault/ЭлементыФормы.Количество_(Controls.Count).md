ЭлементыФормы (Controls)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество размещенных на форме элементов управления.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| КоличествоЭлементов = ЭлементыФормы.Количество(); |

Использование в версии:

Доступен, начиная с версии 8.0.
