ВидСравненияКомпоновкиДанных (DataCompositionComparisonType)

Заполнено (Filled)

Доступен, начиная с версии 8.1.

Описание:

Осуществляет проверку поля на то, что его значение отличается от значения данного поля по умолчанию, отличное от значения [Null](lang__def_Null.md), отличное от пустой ссылки, отличное от значения [Неопределено](lang__def_Undefined.md). Для логических значений осуществляется проверка на значение [Null](lang__def_Null.md). Для строк осуществляется проверка на отсутствие непробельных символов.

Использование в версии:

Доступен, начиная с версии 8.1.
