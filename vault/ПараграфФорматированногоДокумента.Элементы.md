ПараграфФорматированногоДокумента (FormattedDocumentParagraph)

Элементы (Items)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовФорматированногоДокумента](КоллекцияЭлементовФорматированногоДокумента.md).   
Содержит коллекцию элементов параграфа форматированного документа.  
В коллекцию помещаются элементы только типов [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента.md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента.md) и [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
