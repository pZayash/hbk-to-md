СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на элементы справочника.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [СправочникВыборка.\<Имя справочника\>](СправочникВыборка.Имя_справочника.md).   

Описание:

Формирует выборку элементов справочника по заданному источнику ссылок на элементы справочника соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы справочника соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    Товары.Ссылка КАК Ссылка     |ИЗ     |    Справочник.Товары КАК Товары     |"; РезультатЗапроса = Запрос.Выполнить(); Выборка = Справочники.Товары.ВыбратьПоСсылкам(РезультатЗапроса); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
