# ТестируемаяТаблицаФормы.ПерейтиКСтроке

ТестируемаяТаблицаФормы (TestedFormTable)

ПерейтиКСтроке (GotoRow)

Доступен, начиная с версии 8.3.2.

Синтаксис:

ПерейтиКСтроке(<ОписаниеСтроки>, <Направление>, <ПереключитьВыделение>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.  
Текст ячейки может содержать знаки подстановки "\*" и "?".  
Если не указан, изменяется выделение текущей строки.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Направление> (необязательный)

Тип: [НаправлениеПереходаКСтроке](НаправлениеПереходаКСтроке_(RowGotoDirection).md).   
Указывает направление перехода от текущей строки.  
Значение по умолчанию: [Вниз](НаправлениеПереходаКСтроке.Вниз_(RowGotoDirection.Down).md).

<ПереключитьВыделение> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Ложь](lang__def_BooleanFalse.md), поведение метода эмулирует щелчок мышью в строке.  
Если установлено [Истина](lang__def_BooleanTrue.md), поведение метода эмулирует щелчок мышью в строке с нажатой клавишей Ctrl.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - переход на указанную строку выполнен, [Ложь](lang__def_BooleanFalse.md) - в противном случае .

Описание:

Выполняет переход к указанной строке.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.2.

Описание изменено в версии 8.3.25.
