Расширение декорации формы для картинки (Form decoration extension for a picture)

Картинка (Picture)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Отображаемая картинка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.2.
