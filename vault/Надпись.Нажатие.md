Надпись (Label)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на надпись, если у надписи свойство [Гиперссылка](Надпись.Гиперссылка.md) имеет значение Истина.

Доступность:

Толстый клиент.

См. также:

[Надпись](Надпись.md), свойство [Гиперссылка](Надпись.Гиперссылка.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
