НастройкиОбработкиОшибокПриЗапуске (ErrorOnStartupProcessingSettings)

ДополнительнаяИнформацияЗапроса (AdditionalRequestInformation)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Дополнительная информация, которая будет помещена в заголовке запроса по адресу сервиса обработки ошибок при запуске.  
Значение по умолчанию: Пустая строка.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
