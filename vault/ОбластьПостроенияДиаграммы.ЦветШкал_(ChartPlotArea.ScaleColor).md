Доступен, начиная с версии 8.0.

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет шкалы.

Имеет смысл для всех диаграмм, кроме круговых.

Использование в версии:

Доступен, начиная с версии 8.0.
