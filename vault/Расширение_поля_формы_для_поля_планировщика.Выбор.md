Расширение поля формы для поля планировщика (Form field extension for a planner)

Выбор (Selection)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Выбор(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка – вызов события ПриНачалеРедактирования – производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Возникает при двойном щелчке левой кнопки мыши на элементе. Выбранный элемент можно получить с помощью свойства поля [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
