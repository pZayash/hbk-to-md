Расширение элементов управления, расположенных на панели (Panel controls extension)

АктивизироватьПоУмолчанию (DefaultControl)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - элемент управления будет активизирован при открытии формы.

Доступность:

Толстый клиент.

Примечание:

На одной странице панели только один элемент управления может иметь значение данного свойства [Истина](lang__def_BooleanTrue.md). Установка свойства для другого элемента приводит к снятию свойства у элемента, для которого оно было установлено ранее.

Использование в версии:

Доступен, начиная с версии 8.0.
