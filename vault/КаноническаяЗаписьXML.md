Описание:

Предназначен для организации последовательной записи документов и фрагментов XML в каноническом виде (см. w3c стандарты - Canonical XML, Exclusive XML Canonicalization).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ЗаписатьXML](Глобальный_контекст.ЗаписатьXML.md)

[КаноническийDOM](КаноническийDOM.md), метод [Канонизировать](КаноническийDOM.Канонизировать.md)
