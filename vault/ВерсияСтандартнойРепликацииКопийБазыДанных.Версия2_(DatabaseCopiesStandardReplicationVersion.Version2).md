ВерсияСтандартнойРепликацииКопийБазыДанных (DatabaseCopiesStandardReplicationVersion)

Версия2 (Version2)

Доступен, начиная с версии 8.3.21.

Описание:

Версия 2 стандартной репликации копий базы данных.  
Использует возможности СУБД основной базы данных для записи изменений. Доступна для SQL Server (2008+).

Использование в версии:

Доступен, начиная с версии 8.3.21.
