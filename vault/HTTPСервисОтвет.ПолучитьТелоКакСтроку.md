HTTPСервисОтвет (HTTPServiceResponse)

ПолучитьТелоКакСтроку (GetBodyAsString)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьТелоКакСтроку()

Возвращаемое значение:

Описание:

Сервер, толстый клиент.
