ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ИмяПринтера (PrinterName)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Имя принтера, на котором нужно напечатать.   
[Неопределено](lang__def_Undefined.md) - значение определяется родительским пакетом. Если родительского пакета нет, то печать выполняется на принтер по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
