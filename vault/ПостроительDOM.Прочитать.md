ПостроительDOM (DOMBuilder)

Прочитать (Read)

Доступен, начиная с версии 8.1.

Синтаксис:

Прочитать(<ИсточникДанныхXML>)

Параметры:

<ИсточникДанныхXML> (обязательный)

Тип: [ЧтениеУзловDOM](ЧтениеУзловDOM.md), [ЧтениеFastInfoset](ЧтениеFastInfoset.md), [ЧтениеXML](ЧтениеXML.md), [ЧтениеHTML](ЧтениеHTML.md).   
Объект чтения данных XML или чтения данных HTML.

Возвращаемое значение:

Тип: [ДокументDOM](ДокументDOM.md), [ДокументHTML](ДокументHTML.md).   
Если выполняется чтение HTML с помощью объекта [ЧтениеHTML](ЧтениеHTML.md), данный метод вернет объект типа [ДокументHTML](ДокументHTML.md).

Описание:

Создает документ DOM. Производит чтение документа XML из источника данных XML.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Параметры DOM в созданный документ не переносятся.  
При чтении документа XML обработка схемы Document Type Definition (DTD) из читаемого документа не выполняется.

Использование в версии:

Доступен, начиная с версии 8.1.
