Series (Series)

Доступен, начиная с версии 8.2.

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит серию ключа защиты программы для аппаратной лицензии или регистрационный номер комплекта для программной лицензии платформы.

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
