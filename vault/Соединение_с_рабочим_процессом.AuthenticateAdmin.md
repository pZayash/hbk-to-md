Соединение с рабочим процессом (IWorkingProcessConnection)

AuthenticateAdmin (AuthenticateAdmin)

Доступен, начиная с версии 8.1.

Синтаксис:

AuthenticateAdmin(<Имя>, <Пароль>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя администратора кластера.

<Пароль> (обязательный)

Тип: Строка.   
Пароль администратора кластера.

Описание:

Выполняет аутентификацию администратора кластера. Администратор кластера должен быть аутентифицирован для создания в этом кластере новой информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
