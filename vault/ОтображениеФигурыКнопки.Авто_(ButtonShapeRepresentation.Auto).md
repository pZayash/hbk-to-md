ОтображениеФигурыКнопки (ButtonShapeRepresentation)

Авто (Auto)

Доступен, начиная с версии 8.3.8.

Описание:

Способ отображения фигуры кнопки определяется автоматически:

- Для стандартных команд используется вариант отображения, установленный для команды.
- В остальных случаях используется значение [Всегда](ОтображениеФигурыКнопки.Всегда_(ButtonShapeRepresentation.Always).md).

Использование в версии:

Доступен, начиная с версии 8.3.8.
