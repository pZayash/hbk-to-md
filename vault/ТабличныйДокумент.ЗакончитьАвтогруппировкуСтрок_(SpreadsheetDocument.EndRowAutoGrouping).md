ТабличныйДокумент (SpreadsheetDocument)

ЗакончитьАвтогруппировкуСтрок (EndRowAutoGrouping)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗакончитьАвтогруппировкуСтрок()

Описание:

Заканчивает автоматическую группировку строк, ранее начатую при помощи метода [НачатьАвтогруппировкуСтрок](ТабличныйДокумент.НачатьАвтогруппировкуСтрок_(SpreadsheetDocument.StartRowAutoGrouping).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
