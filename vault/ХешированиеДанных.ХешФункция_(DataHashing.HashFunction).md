ХешированиеДанных (DataHashing)

ХешФункция (HashFunction)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [ХешФункция](ХешФункция_(HashFunction).md).   
Содержит тип хеш-функции. Определяет способ вычисления хеш-суммы и тип вычисляемого значения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
