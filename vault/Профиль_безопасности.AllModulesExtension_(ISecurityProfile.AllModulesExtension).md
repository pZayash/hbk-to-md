# Профиль безопасности.AllModulesExtension

Профиль безопасности (ISecurityProfile)

AllModulesExtension (AllModulesExtension)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, разрешено ли изменение всех модулей в расширении конфигурации:  
[Истина](lang__def_BooleanTrue.md) - разрешено расширять любой модуль,  
[Ложь](lang__def_BooleanFalse.md) - разрешено расширять модули из списка разрешенных (указаны в свойстве [ModulesAvailableForExtension](Профиль_безопасности.ModulesAvailableForExtension_(ISecurityProfile.ModulesAvailableForExtension).md)).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
