ТаблицаФормы (FormTable)

ПоложениеСтрокиПоиска (SearchStringLocation)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеСтрокиПоиска](ПоложениеСтрокиПоиска_(SearchStringLocation).md).   
Задает расположение автоматически формируемой строки поиска для таблицы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
