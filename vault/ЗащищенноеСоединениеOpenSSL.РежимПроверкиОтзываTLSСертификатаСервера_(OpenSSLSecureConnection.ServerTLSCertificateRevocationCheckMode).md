РежимПроверкиОтзываTLSСертификатаСервера (ServerTLSCertificateRevocationCheckMode)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [РежимПроверкиОтзываTLSСертификатаСервера](РежимПроверкиОтзываTLSСертификатаСервера_(ServerTLSCertificateRevocationCheckMode).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.
