Расширение поля формы для поля картинки (Form field extension for a picture field)

СпособПеретаскиванияФайлов (FileDragMode)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [СпособПеретаскиванияФайлов](СпособПеретаскиванияФайлов_(FileDragMode).md).   
Указывает способ перетаскивания файлов. Если установлен способ [КакФайл](СпособПеретаскиванияФайлов.КакФайл_(FileDragMode.AsFile).md), то при перетаскивании файла (файлов) на элемент управления в качестве значения параметра перетаскивания будет передан объект типа [Файл](Файл_(File).md) (массив объектов типа [Файл](Файл_(File).md)), а если установлен способ [КакСсылкаНаФайл](СпособПеретаскиванияФайлов.КакСсылкаНаФайл_(FileDragMode.AsFileRef).md) – объект типа [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md) (массив объектов типа [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md)).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В веб-клиенте при перетаскивании файла (файлов) как [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md) обработчик события [ПроверкаПеретаскивания](Расширение_поля_формы_для_поля_картинки.ПроверкаПеретаскивания_(Form_field_extension_for_a_picture_field.DragCheck).md) у элементов управления вызывается один раз в момент отпускания кнопки мыши перед обработчиком события [Перетаскивание](Расширение_поля_формы_для_поля_картинки.Перетаскивание_(Form_field_extension_for_a_picture_field.Drag).md) в отличие от тонкого клиента, в котором обработчик события [ПроверкаПеретаскивания](Расширение_поля_формы_для_поля_картинки.ПроверкаПеретаскивания_(Form_field_extension_for_a_picture_field.DragCheck).md) вызывается в процессе перетаскивания файла (файлов).  
В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл_(FileDragMode.AsFile).md), то перетаскивание файлов игнорируется, и обработчики событий [ПроверкаПеретаскивания](Расширение_поля_формы_для_поля_картинки.ПроверкаПеретаскивания_(Form_field_extension_for_a_picture_field.DragCheck).md) и [Перетаскивание](Расширение_поля_формы_для_поля_картинки.Перетаскивание_(Form_field_extension_for_a_picture_field.Drag).md) не вызываются.

Использование в версии:

Доступен, начиная с версии 8.3.12.
