ПотокВПамяти (MemoryStream)

ЗакрытьИПолучитьДвоичныеДанные (CloseAndGetBinaryData)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ЗакрытьИПолучитьДвоичныеДанные()

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   

Описание:

Возвращает экземпляр объекта [ДвоичныеДанные](ДвоичныеДанные.md), содержащего данные, записанные в поток.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если поток создан на основе объекта[БуферДвоичныхДанных](БуферДвоичныхДанных.md) и буфер позволяет изменения ([ТолькоЧтение](БуферДвоичныхДанных.ТолькоЧтение.md)установлено в Ложь), то выполняется копирование данных.  
В любом другом случае копирование данных не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.9.
