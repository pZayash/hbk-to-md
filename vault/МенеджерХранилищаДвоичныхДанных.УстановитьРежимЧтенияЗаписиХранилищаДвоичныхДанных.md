МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьРежимЧтенияЗаписиХранилищаДвоичныхДанных (SetBinaryDataStorageReadWriteMode)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьРежимЧтенияЗаписиХранилищаДвоичныхДанных(<РежимЧтенияЗаписи>)

Параметры:

<РежимЧтенияЗаписи> (обязательный)

Тип: [РежимЧтенияЗаписиХранилищаДвоичныхДанных](РежимЧтенияЗаписиХранилищаДвоичныхДанных.md).   
Режим работы хранилища двоичных данных, который требуется установить.

Описание:

Устанавливает режим хранилища двоичных данных (только для чтения, чтение и запись).  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
