КритерийОтбораСписок.<Имя критерия> (FilterCriterionList.<Criterion name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Содержит объект [Отбор](Отбор_(Filter).md), по которому осуществляется текущая фильтрация списка критерия отбора.  
Данный отбор состоит из одного элемента отбора ЗначениеОтбора (SelectValue), свойство которого [Значение](ЭлементОтбора.Значение_(FilterItem.Value).md) имеет тип, совпадающий с типом объекта метаданных критерий отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
