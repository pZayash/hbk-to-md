# РезультатXPath.НеверноеСостояниеИтератора

РезультатXPath (XPathResult)

НеверноеСостояниеИтератора (InvalidIteratorState)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак недействительного состояния итератора.  
[Истина](lang__def_BooleanTrue.md), если итератор становится недействительным (тип результата является [НеупорядоченныйИтераторУзлов](ТипРезультатаDOMXPath.НеупорядоченныйИтераторУзлов_(DOMXPathResultType.UnorderedNodeIterator).md) или [УпорядоченныйИтераторУзлов](ТипРезультатаDOMXPath.УпорядоченныйИтераторУзлов_(DOMXPathResultType.OrderedNodeIterator).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Устанавливается в [Истина](lang__def_BooleanTrue.md), если документ был изменен после получения данного результата.

Использование в версии:

Доступен, начиная с версии 8.1.
