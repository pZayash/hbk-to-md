ПараметрыДиалогаПолученияФайлов (GetFilesDialogParameters)

ВыборКаталога (ChooseDirectory)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли открыт диалог для выбора каталога. Если значение [Ложь](lang__def_BooleanFalse.md), то диалог будет открыт для получения каждого файла.  
В веб-клиенте значение [Истина](lang__def_BooleanTrue.md) игнорируется.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
