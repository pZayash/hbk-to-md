ДанныеАдреса (AddressData)

ИсходныеДанные (RawData)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Описывает адрес в том виде, в котором он возвращен операционной системой. В соответствии содержатся поля адреса, но их состав зависит от конкретной платформы, на которой адрес получен.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
