Глобальный контекст (Global context)

ПолучитьСпособПроверкиПодписиМобильногоКлиента (GetMobileClientSignatureVerificationMethod)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьСпособПроверкиПодписиМобильногоКлиента()

Описание:

Возвращает текущий способ проверки подписи мобильного клиента.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
