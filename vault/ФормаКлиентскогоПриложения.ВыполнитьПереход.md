ФормаКлиентскогоПриложения (ClientApplicationForm)

ВыполнитьПереход (ExecuteNavigation)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ВыполнитьПереход(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: Ссылка на объект информационной базы.   
Объект, на который необходимо перейти.

Описание:

Если основным реквизитом формы является динамический список, и тип значения параметра <Объект> является ссылкой на объект, который содержится в главной таблице динамического списка, то у таблицы формы, отображающей этот динамический список (первой, если таких таблиц несколько) текущая строка устанавливается на переданный объект.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
