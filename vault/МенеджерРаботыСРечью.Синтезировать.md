МенеджерРаботыСРечью (SpeechProcessingManager)

Синтезировать (Synthesize)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Синтезировать(<Текст>, <Голос>, <ПараметрыСинтеза>)

Параметры:

<Текст> (обязательный)

Тип: Строка.   
Текст, по которому синтезируется речь.

<Голос> (необязательный)

Тип: Строка.   
Голос синтеза речи.  
Если не указан – голос определяется сервисом.

<ПараметрыСинтеза> (необязательный)

Тип: [Соответствие](Соответствие.md).   
Дополнительные параметры синтеза. Возможный список параметров и их значений зависит от выбранного голоса; будет получен с помощью метода [ПолучитьДоступныеГолосаСинтеза](МенеджерРаботыСРечью.ПолучитьДоступныеГолосаСинтеза.md).  
  
Заполняется на основании [ОписаниеПараметраГолосаСинтезаРечи](ОписаниеПараметраГолосаСинтезаРечи.md).

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   

Описание:

Синтезирует речь и возвращает результат в виде аудиофайла (OPUS в контейнере OGG).

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.25.
