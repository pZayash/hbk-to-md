ДокументDOM (DOMDocument)

КодировкаXML (XMLEncoding)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Кодировка, указанная в декларации XML документа. Значение псевдоатрибута "encoding" в объявлении XML документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Пустая строка, если кодировка неизвестна (например, документ формируется в памяти).
