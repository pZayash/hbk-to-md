Модифицированность (Modified)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Модифицированность()

Возвращаемое значение:

Тип: Булево.   
Истина - записи изменены; Ложь - в противном случае.

Описание:

Определяет, были ли изменены записи в наборе после последнего считывания или записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не позволяет определить, были ли изменены соответствующие записи другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.3.5.
