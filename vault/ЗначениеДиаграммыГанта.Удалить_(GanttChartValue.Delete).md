ЗначениеДиаграммыГанта (GanttChartValue)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Интервал>)

Параметры:

<Интервал> (обязательный)

Тип: [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md).   
Удаляемый интервал.

Описание:

Удаляет интервал.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
