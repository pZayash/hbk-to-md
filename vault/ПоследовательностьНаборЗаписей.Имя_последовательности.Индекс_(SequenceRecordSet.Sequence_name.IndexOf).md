ПоследовательностьНаборЗаписей.<Имя последовательности> (SequenceRecordSet.<Sequence name>)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: [ПоследовательностьЗапись.\<Имя последовательности\>](ПоследовательностьЗапись.Имя_последовательности_(SequenceRecord.Sequence_name).md).   
Запись, индекс которой необходимо получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной записи в наборе. Если не найдено, то возвращается -1.

Описание:

Получает индекс указанной записи в наборе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
