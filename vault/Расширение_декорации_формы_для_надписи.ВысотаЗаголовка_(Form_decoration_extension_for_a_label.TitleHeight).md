# Расширение декорации формы для надписи.ВысотаЗаголовка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Декорация формы](Декорация_формы.md) › [Расширение декорации формы для надписи](Расширение_декорации_формы_для_надписи_(Form_decoration_extension_for_a_label).md) › [Свойства](Расширение_декорации_формы_для_надписи__Свойства.md)

Расширение декорации формы для надписи (Form decoration extension for a label)

ВысотаЗаголовка (TitleHeight)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит высоту заголовка. Указывается в строках. 0 - высота определяется автоматически.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
