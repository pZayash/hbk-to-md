МенеджерКриптографии (CryptoManager)

ПолучитьКонтейнерПодписейКриптографииАсинх (GetCryptoSignaturesContainerAsync)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьКонтейнерПодписейКриптографииАсинх(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Подпись для разбора.  
Если параметр типа Строка, то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные.md) или [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md), то он задает данные подписи.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [КонтейнерПодписейКриптографии](КонтейнерПодписейКриптографии.md).

Описание:

Выполняет разбор двоичных данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.

Описание изменено в версии 8.3.22.
