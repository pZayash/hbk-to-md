Расширение формы обработки (Data processor form extension)

ПослеВосстановленияЗначений (AfterRestoreValues)

Доступен, начиная с версии 8.0.

Синтаксис:

ПослеВосстановленияЗначений()

Описание:

Возникает после считывания сохраненных ранее значений.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
