МенеджерКалендарей (CalendarsManager)

УдалитьКалендарь (DeleteCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

УдалитьКалендарь(<Календарь>)

Параметры:

<Календарь> (обязательный)

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря.md).   
Локальный ключ календаря.

Описание:

Удаляет календарь по локальному ключу.  
Данный метод поддерживается не для любого календаря. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИзменениеКалендаря](МенеджерКалендарей.ПоддерживаетсяИзменениеКалендаря.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей.md), метод [ПоддерживаетсяИзменениеКалендаря](МенеджерКалендарей.ПоддерживаетсяИзменениеКалендаря.md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
