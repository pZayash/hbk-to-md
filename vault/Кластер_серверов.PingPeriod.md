Кластер серверов (IClusterInfo)

PingPeriod (PingPeriod)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Определяет период отправки проверочных запросов системы отслеживания соединений, в миллисекундах. Допустимые значения от 0 до 65535.  
Если данное свойство и свойство [PingTimeout](Кластер_серверов.PingTimeout.md) имеют значение 0, будут использованы параметры системы отслеживания соединений, установленные с помощью командной строки.  
Значение свойства должно быть установлено вместе со значением свойства [PingTimeout](Кластер_серверов.PingTimeout.md).  
Значения по умолчанию:

- PingPeriod - 1000 мс,
- PingTimeout - 5000 мс.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
