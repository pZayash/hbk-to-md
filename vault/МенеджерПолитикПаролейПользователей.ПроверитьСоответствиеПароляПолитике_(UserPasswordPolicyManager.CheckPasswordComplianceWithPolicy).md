МенеджерПолитикПаролейПользователей (UserPasswordPolicyManager)

ПроверитьСоответствиеПароляПолитике (CheckPasswordComplianceWithPolicy)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПроверитьСоответствиеПароляПолитике(<Пароль>, <Политика>, <ПользовательИнформационнойБазы>)

Параметры:

<Пароль> (обязательный)

Тип: [Строка](lang__def_String.md).   
Пароль пользователя.

<Политика> (необязательный)

Тип: [ПолитикаПаролейПользователей](ПолитикаПаролейПользователей_(UserPasswordPolicy).md).   
Политика паролей пользователей.  
По умолчанию: [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ПользовательИнформационнойБазы> (необязательный)

Тип: [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md).   
Пользователь информационной базы.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив значений типа [РезультатПроверкиСоответствияПароляПолитике](РезультатПроверкиСоответствияПароляПолитике_(PasswordPolicyComplianceCheckResult).md).

Описание:

Проверяет, соответствует ли пароль политике паролей пользователей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
