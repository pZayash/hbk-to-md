РежимВосстановленияПутейФайловZIP (ZIPRestoreFilePathsMode)

НеВосстанавливать (DontRestore)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [НеВосстанавливать](РежимВосстановленияПутейФайлаАрхива.НеВосстанавливать_(ArchiveFileRestoreFilePathsMode.DontRestore).md)

Доступен, начиная с версии 8.0.

Описание:

При извлечении из архива пути файлов игнорируются.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
