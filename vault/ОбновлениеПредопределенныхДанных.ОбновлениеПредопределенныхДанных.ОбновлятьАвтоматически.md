ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

ОбновлятьАвтоматически (AutoUpdate)

Доступен, начиная с версии 8.3.3.

Описание:

При реструктуризации информационной базы будет выполняться автоматическое создание предопределенных элементов и обновление существующих значений.

Использование в версии:

Доступен, начиная с версии 8.3.3.
