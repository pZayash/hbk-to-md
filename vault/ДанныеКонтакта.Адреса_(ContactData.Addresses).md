ДанныеКонтакта (ContactData)

Адреса (Addresses)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив адресов в виде объектов типа [ЭлементДанныхКонтакта](ЭлементДанныхКонтакта_(ContactDataItem).md). Свойство [ТипДанных](ЭлементДанныхКонтакта.ТипДанных_(ContactDataItem.DataType).md) содержит значение перечисления [ТипАдресаДанныхКонтакта](ТипАдресаДанныхКонтакта_(ContactDataAddressType).md) или строку. Свойство [Значение](ЭлементДанныхКонтакта.Значение_(ContactDataItem.Value).md) содержит объект [ДанныеАдреса](ДанныеАдреса_(AddressData).md).

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
