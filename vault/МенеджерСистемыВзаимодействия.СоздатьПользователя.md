МенеджерСистемыВзаимодействия (CollaborationSystemManager)

СоздатьПользователя (CreateUser)

Доступен, начиная с версии 8.3.11.

Синтаксис:

СоздатьПользователя(<ПользовательИнформационнойБазы>)

Параметры:

<ПользовательИнформационнойБазы> (необязательный)

Тип: [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы.md).   
Пользователь информационной базы, для которого создается пользователь системы взаимодействия.  
Если параметр не задан, то создается пользователь системы взаимодействия, для которого нет соответствующего пользователя информационной базы.

Возвращаемое значение:

Тип: [ПользовательСистемыВзаимодействия](ПользовательСистемыВзаимодействия.md).   

Описание:

Создает объект типа [ПользовательСистемыВзаимодействия](ПользовательСистемыВзаимодействия.md) и заполняет его свойства [Имя](ПользовательСистемыВзаимодействия.Имя.md) и [ПолноеИмя](ПользовательСистемыВзаимодействия.ПолноеИмя.md) по данным пользователя информационной базы.  
Объект будет создан после вызова метода [Записать](ПользовательСистемыВзаимодействия.Записать.md).  
Вызов метода возможен только в привилегированном режиме.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если такой пользователь уже существует, генерируется исключение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
