ПользовательскоеПолеВыражениеКомпоновкиДанных (DataCompositionUserFieldExpression)

УстановитьВыражениеИтоговыхЗаписей (SetTotalRecordExpression)

Не рекомендуется использовать, начиная с версии 8.3.10.

Рекомендуется использовать:

- [УстановитьВыражения](ПользовательскоеПолеВыражениеКомпоновкиДанных.УстановитьВыражения.md)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьВыражениеИтоговыхЗаписей(<Выражение>)

Параметры:

<Выражение> (необязательный)

Тип: Строка.   
Устанавливаемое выражение.

Описание:

Устанавливает выражение, используемое при выводе поля в итоговых записях.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.10.

Описание изменено в версии 8.3.15.
