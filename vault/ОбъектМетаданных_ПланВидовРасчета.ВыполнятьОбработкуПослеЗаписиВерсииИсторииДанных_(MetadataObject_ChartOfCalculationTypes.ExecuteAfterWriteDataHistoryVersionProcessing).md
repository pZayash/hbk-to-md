ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если имеет значение [Истина](lang__def_BooleanTrue.md), то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.ОбработкаПослеЗаписиВерсийИсторииДанных_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name.AfterWriteDataHistoryVersionsProcessing).md) можно обработать запись версии после обновления истории.  
При записи данных значение, указанное в конфигураторе, устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_ПланВидовРасчета.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных_(MetadataObject_ChartOfCalculationTypes.ExecuteAfterWriteDataHistoryVersionProcessing).md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
