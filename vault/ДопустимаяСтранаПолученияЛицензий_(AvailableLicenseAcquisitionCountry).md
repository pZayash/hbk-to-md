# ДопустимаяСтранаПолученияЛицензий

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Получение лицензий](Получение_лицензий.md)

Доступен, начиная с версии 8.3.20.

Свойства:

[Код (Code)](ДопустимаяСтранаПолученияЛицензий.Код_(AvailableLicenseAcquisitionCountry.Code).md)  
[Представление (Presentation)](ДопустимаяСтранаПолученияЛицензий.Представление_(AvailableLicenseAcquisitionCountry.Presentation).md)  

Описание:

Содержит код и локализованное наименование страны.   
Рекомендуется использовать двухбуквенный код страны.   
При повторном получении лицензии, полученной платформой версии 8.3.10 и ниже (в 2017 году и раньше), требуется указание наименования страны, как оно было указано при первоначальном получении лицензии.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолучитьДопустимыеСтраны](objects__Global_context__methods__catalogmethods__GetAvailableCountries6192.md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
