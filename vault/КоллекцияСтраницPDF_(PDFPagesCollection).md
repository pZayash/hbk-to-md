# КоллекцияСтраницPDF

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с PDF](Работа_с_PDF-2.md)

Доступен, начиная с версии 8.3.26.

Элементы коллекции:

[СтраницаPDF](СтраницаPDF_(PDFPage).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются страницы документа PDF.  
Возможно обращение к странице документа PDF посредством оператора [...]. В качестве аргумента передается .  

Методы:

[Индекс (IndexOf)](КоллекцияСтраницPDF.Индекс_(PDFPagesCollection.IndexOf).md)  
[Количество (Count)](КоллекцияСтраницPDF.Количество_(PDFPagesCollection.Count).md)  
[Получить (Get)](КоллекцияСтраницPDF.Получить_(PDFPagesCollection.Get).md)  

Описание:

Предоставляет методы для работы со страницами PDF-документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
