ИнтервалДиаграммыГанта (GanttChartInterval)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Конец>)

Параметры:

<Конец> (обязательный)

Тип: [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md).   
Конец создаваемой связи. Недопустимо создавать связь между одним и тем же интервалом.

Возвращаемое значение:

Тип: [СвязьДиаграммыГанта](СвязьДиаграммыГанта_(GanttChartLink).md).   

Описание:

Добавляет связь.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
