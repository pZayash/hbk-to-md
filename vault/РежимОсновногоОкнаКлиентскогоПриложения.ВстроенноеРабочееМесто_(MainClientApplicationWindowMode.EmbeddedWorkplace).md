РежимОсновногоОкнаКлиентскогоПриложения (MainClientApplicationWindowMode)

ВстроенноеРабочееМесто (EmbeddedWorkplace)

Доступен, начиная с версии 8.3.16.

Описание:

Режим встроенного рабочего места основного окна клиентского приложения.  
Применяется только в веб-клиенте, используется для встраивания веб-клиента во внешний сайт.  
В тонком, толстом, мобильном клиентах данное значение игнорируется и интерпретируется как [Обычный](РежимОсновногоОкнаКлиентскогоПриложения.Обычный_(MainClientApplicationWindowMode.Normal).md).  
Имеет следующие особенности:

- все особенности режима [РабочееМесто](РежимОсновногоОкнаКлиентскогоПриложения.РабочееМесто_(MainClientApplicationWindowMode.Workplace).md),
- системная командная панель скрывается,
- в правом нижнем углу отображается надпись "РАБОТАЕТ НА 1С".

Использование в версии:

Доступен, начиная с версии 8.3.16.
