# Расширение декорации формы для надписи.ВертикальноеПоложение

Расширение декорации формы для надписи (Form decoration extension for a label)

ВертикальноеПоложение (VerticalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложениеЭлемента](ВертикальноеПоложениеЭлемента_(ItemVerticalAlign).md).   
Определяет способ вертикального расположения текста надписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
