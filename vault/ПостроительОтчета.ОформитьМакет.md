ПостроительОтчета (ReportBuilder)

ОформитьМакет (ApplyAppearanceTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ОформитьМакет()

Описание:

Оформить макет с применением макета оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // Инициализация оформления  Оформление = ПолучитьМакетОформления(СтандартноеОформление.Весна); ПостроительОтчета.МакетОформления = Оформление; // ...  ПостроительОтчета.Выполнить(); ПостроительОтчета.ОформитьМакет(); ПостроительОтчета.Вывести(ДокументРезультат); |

Использование в версии:

Доступен, начиная с версии 8.0.
