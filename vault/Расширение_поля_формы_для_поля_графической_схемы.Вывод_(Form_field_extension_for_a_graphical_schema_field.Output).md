# Расширение поля формы для поля графической схемы.Вывод

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля графической схемы](Расширение_поля_формы_для_поля_графической_схемы_(Form_field_extension_for_a_graphical_schema_field).md) › [Свойства](Расширение_поля_формы_для_поля_графической_схемы__Свойства.md)

Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

Вывод (Output)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеВывода](ИспользованиеВывода_(UseOutput).md).   
Управляет ограничением вывода на печать, сохранения, работы с буфером обмена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Может иметь значение, отличное от значения свойства [Вывод](ГрафическаяСхема.Вывод_(GraphicalSchema.Output).md) документа, с которым это поле связано.

Использование в версии:

Доступен, начиная с версии 8.2.
