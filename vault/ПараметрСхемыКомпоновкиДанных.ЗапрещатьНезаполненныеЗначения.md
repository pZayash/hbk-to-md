ПараметрСхемыКомпоновкиДанных (DataCompositionSchemaParameter)

ЗапрещатьНезаполненныеЗначения (DenyIncompleteValues)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство установлено в Истина, то при выполнении компоновки данных, если включено использование данного параметра и значение параметра содержит значение по умолчанию для типа значения параметра, система выдаст ошибку, в которой будет указано, что нужно ввести значение данного параметра.   
Если свойство установлено в Ложь, то отчет будет исполняться и с пустым значением параметра.  
Значение по умолчанию Ложь.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
