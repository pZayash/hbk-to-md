СообщениеСистемыВзаимодействия (CollaborationSystemMessage)

ШаблонСообщения (MessageTemplate)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия.md), Неопределено.   
Если свойство установлено, то при записи создается шаблонное сообщение.  
При записи шаблонного сообщения игнорируются свойства [Текст](СообщениеСистемыВзаимодействия.Текст.md), [Вложения](СообщениеСистемыВзаимодействия.Вложения.md), [Данные](СообщениеСистемыВзаимодействия.Данные.md) и [Действия](СообщениеСистемыВзаимодействия.Действия.md).  
Установка свойства возможна только для нового незаписанного сообщения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
