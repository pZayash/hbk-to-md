ЗаписьДанных (DataWriter)

ЗаписатьБайтАсинх (WriteByteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьБайтАсинх(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Целое число, которое будет записано в целевой поток. Значение числа должно находиться в диапазоне от 0 до 255.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Записывает один байт в целевой поток.

Доступность:

Тонкий клиент, толстый клиент.
