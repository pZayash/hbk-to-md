Глобальный контекст (Global context)

ПолучитьОкна (GetWindows)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьОкна()

Возвращаемое значение:

Тип: [ОкнаКлиентскогоПриложения](ОкнаКлиентскогоПриложения_(ClientApplicationWindows).md).   

Описание:

Получает окна приложения. Порядок элементов в коллекции не определен.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) возвращает [Неопределено](lang__def_Undefined.md).

См. также:

[ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
