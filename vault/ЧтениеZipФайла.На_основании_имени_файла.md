ЧтениеZipФайла (ZipFileReader)

На основании имени файла

Не рекомендуется использовать, начиная с версии 8.3.26.

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЧтениеZipФайла(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя ZIP-файла, который требуется открыть для чтения.

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пароль для доступа к ZIP файлу, если файл зашифрован.

Описание:

Открывает для чтения указанный ZIP файл. Аналогичен методу [Открыть](ЧтениеZipФайла.Открыть_(ZipFileReader.Open).md).

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
