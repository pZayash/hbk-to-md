Элементы коллекции:

[МакетГруппировкиДиаграммыМакетаКомпоновкиДанных](МакетГруппировкиДиаграммыМакетаКомпоновкиДанных.md), [ГруппировкаДиаграммыМакетаКомпоновкиДанных](ГруппировкаДиаграммыМакетаКомпоновкиДанных.md), [ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных](ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы тела группировки диаграммы.  
Возможно обращение к элементу тела группировки диаграммы посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Тело группировки диаграммы. Данный объект является коллекцией объектов следующих типов:   
[МакетГруппировкиДиаграммыМакетаКомпоновкиДанных](МакетГруппировкиДиаграммыМакетаКомпоновкиДанных.md);  
[ГруппировкаДиаграммыМакетаКомпоновкиДанных](ГруппировкаДиаграммыМакетаКомпоновкиДанных.md);  
[ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных](ИерархическаяГруппировкаДиаграммыМакетаКомпоновкиДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
