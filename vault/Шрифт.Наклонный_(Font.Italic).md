Шрифт (Font)

Наклонный (Italic)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак наклонного начертания шрифта.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если шрифт задан не абсолютным, то содержит [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.0.
