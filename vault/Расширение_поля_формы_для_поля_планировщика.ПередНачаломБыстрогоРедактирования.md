Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередНачаломБыстрогоРедактирования (BeforeStartQuickEdit)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередНачаломБыстрогоРедактирования(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: Булево.   
Установка данного параметра в Ложь приведет к отмене появления окна быстрого редактирования.  
Значение по умолчанию: Истина.

Описание:

Возникает перед отображением окна быстрого редактирования. Элементы, над которыми выполнялось действие, можно получить с помощью свойства [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы.md) поля.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
