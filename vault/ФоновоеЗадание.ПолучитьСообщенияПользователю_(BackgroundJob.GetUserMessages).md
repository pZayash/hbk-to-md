ФоновоеЗадание (BackgroundJob)

ПолучитьСообщенияПользователю (GetUserMessages)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьСообщенияПользователю(<УдалятьПолученные>)

Параметры:

<УдалятьПолученные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак необходимости удаления полученных сообщений.  
[Истина](lang__def_BooleanTrue.md) - удалять.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   

Описание:

Получает массив объектов [СообщениеПользователю](СообщениеПользователю_(UserMessage).md), которые были выведены в процессе работы фонового задания. Получение сообщений может выполняться как в процессе работы задания, так и по его завершении.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
