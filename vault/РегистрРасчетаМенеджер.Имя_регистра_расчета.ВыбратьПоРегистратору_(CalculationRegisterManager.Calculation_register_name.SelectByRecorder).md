РегистрРасчетаМенеджер.<Имя регистра расчета> (CalculationRegisterManager.<Calculation register name>)

ВыбратьПоРегистратору (SelectByRecorder)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьПоРегистратору(<Регистратор>)

Параметры:

<Регистратор> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Документ, по которому будут отобраны движения.

Возвращаемое значение:

Тип: [РегистрРасчетаВыборка.\<Имя регистра расчета\>](РегистрРасчетаВыборка.Имя_регистра_расчета_(CalculationRegisterSelection.Calculation_register_name).md).   

Описание:

Формирует выборку записей регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Движения = РегистрыРасчета.РасчетСреднегоЗаработка.ВыбратьПоРегистратору(Ссылка); |

Использование в версии:

Доступен, начиная с версии 8.0.
