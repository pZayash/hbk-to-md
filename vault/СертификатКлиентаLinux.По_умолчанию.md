СертификатКлиентаLinux (LinuxClientCertificate)

По умолчанию

Доступен, начиная с версии 8.3.21.

Синтаксис:

Новый СертификатКлиентаLinux()

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из хранилища сертификатов ОС Linux. Сертификат подбирается автоматически.

Использование в версии:

Доступен, начиная с версии 8.3.21.
