МенеджерСпискаПроверкиРаскрытияПароля (PasswordCompromiseCheckListManager)

УстановитьСписокИзПаролей (SetListFromPasswords)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьСписокИзПаролей(<СписокПроверкиРаскрытияПароля>)

Параметры:

<СписокПроверкиРаскрытияПароля> (обязательный)

Тип: [Массив](Массив.md), Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ФайловыйПоток](ФайловыйПоток.md), [ПотокВПамяти](ПотокВПамяти.md).   
Список проверки раскрытия пароля.  
Список может быть размещен в массиве строк, в файле, содержащем пароли, разделенные переносом строки (в этом случае указывается имя файла), передан в виде потока или представлен как [ДвоичныеДанные](ДвоичныеДанные.md).

Описание:

Формирует список проверки раскрытия пароля, удаляя дубликаты. Количество значений не может превышать 100000.

Доступность:

Сервер, толстый клиент.

Пример:

|  |
| --- |
| СписокПроверкиРаскрытияПароля = Новый Массив(); СписокПроверкиРаскрытияПароля.Добавить("Qwerty"); СписокПроверкиРаскрытияПароля.Добавить("12345"); СписокПроверкиРаскрытияПароля.Добавить("Password"); ДополнительныеНастройкиАутентификации. СписокПроверкиРаскрытияПароля.УстановитьСписокИзПаролей(СписокПроверкиРаскрытияПароля); |
