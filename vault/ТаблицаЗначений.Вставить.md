ТаблицаЗначений (ValueTable)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс вставляемой строки.

Возвращаемое значение:

Тип: [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений.md).   
Вставленная строка.

Описание:

Вставляет строку на позицию в таблице значений, соответствующую указанному индексу.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| НоваяСтрока = ТаблицаЦен.Вставить(2); НоваяСтрока.Товар = ТекТовар; НоваяСтрока.Цена = ПолученнаяЦена; |

Использование в версии:

Доступен, начиная с версии 8.0.
