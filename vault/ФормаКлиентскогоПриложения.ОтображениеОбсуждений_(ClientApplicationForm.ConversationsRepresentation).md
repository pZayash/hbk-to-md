ФормаКлиентскогоПриложения (ClientApplicationForm)

ОтображениеОбсуждений (ConversationsRepresentation)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеОбсужденийФормы](ОтображениеОбсужденийФормы_(FormConversationsRepresentation).md).   
Устанавливает режим отображения обсуждений формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.10.
