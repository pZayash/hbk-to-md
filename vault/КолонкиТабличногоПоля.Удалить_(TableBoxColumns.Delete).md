КолонкиТабличногоПоля (TableBoxColumns)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [КолонкаТабличногоПоля](КолонкаТабличногоПоля_(TableBoxColumn).md).   
Индекс удаляемой колонки либо сама колонка.

Описание:

Удаляет заданную колонку из табличного поля.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| // Удаление первой колонки:  ЭлементыФормы.ТабличноеПоле.Колонки.Удалить(0); |

Использование в версии:

Доступен, начиная с версии 8.0.
