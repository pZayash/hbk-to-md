ДиаграммаГанта (GanttChart)

АвтоУстановкаТекстаТочек (AutoPointText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак генерации текста для новых точек по схеме "ТочкаN". Если [Ложь](lang__def_BooleanFalse.md), то у новой точки будет пустой текст.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
