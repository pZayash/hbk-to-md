Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ОдинаковаяШиринаКолонок (EqualColumnsWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Определяет одинаковую ширину колонок элементов.   
Неопределено трактуется как Ложь.   
Если значение свойства отлично от Истина, свойство "[ШиринаЭлемента](Расширение_поля_формы_для_поля_переключателя.ШиринаЭлемента.md)" не используется.  
Данное свойство используется только, если [ВидПереключателя](Расширение_поля_формы_для_поля_переключателя.ВидПереключателя.md) = [Тумблер](ВидПереключателя.Тумблер.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.2.
