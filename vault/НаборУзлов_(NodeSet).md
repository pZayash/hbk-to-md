Описание:

Представляет собой значение реквизита [НаборУзлов](НаборУзлов_(NodeSet).md) объекта типа [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md), который, в свою очередь является значением свойства ОбменДанными, объектов, представляющих данные, по которым ведется учет изменений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md), свойство [Получатели](ПараметрыОбменаДанными.Получатели_(DataExchangeParameters.Recipients).md)
