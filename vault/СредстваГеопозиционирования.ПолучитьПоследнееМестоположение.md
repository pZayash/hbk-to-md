СредстваГеопозиционирования (LocationTools)

ПолучитьПоследнееМестоположение (GetLastLocation)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьПоследнееМестоположение(<ИмяПровайдера>)

Параметры:

<ИмяПровайдера> (обязательный)

Тип: Строка.   
Содержит имя провайдера.

Возвращаемое значение:

Тип: [ДанныеМестоположения](ДанныеМестоположения.md), Неопределено.   
Неопределено - отсутствует информация о последнем местоположении.

Описание:

Получает последние данные о местоположении, полученные от этого провайдера.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
