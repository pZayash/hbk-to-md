Значения

[Авто (Auto)](ОтправкаЭлементаДанных.Авто.md)  
[Игнорировать (Ignore)](ОтправкаЭлементаДанных.Игнорировать.md)  
[Удалить (Delete)](ОтправкаЭлементаДанных.Удалить.md)  

Описание:

Содержит варианты значений выходных параметров обработчиков событий [ПриОтправкеДанныхГлавному](ПланОбменаОбъект.Имя_плана_обмена.ПриОтправкеДанныхГлавному.md) и [ПриОтправкеДанныхПодчиненному](ПланОбменаОбъект.Имя_плана_обмена.ПриОтправкеДанныхПодчиненному.md) объектов [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>. Данное значение определяет действия системы относительно элемента данных при формировании сообщения, содержащего изменения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.
