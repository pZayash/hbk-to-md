ТипАппроксимацииЛинииТрендаДиаграммы (ChartTrendlineApproximationType)

Экспоненциальный (Exponential)

Доступен, начиная с версии 8.3.10.

Описание:

Экспоненциальная аппроксимация.  
Все значения данных по оси Y должны быть строго больше нуля, в противном случае линия тренда не будет построена.

Использование в версии:

Доступен, начиная с версии 8.3.10.
