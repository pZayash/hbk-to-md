# Расширение поля ввода документа

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Документы](Документы.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора документа в элементе управления [ПолеВвода](ПолеВвода.md) по введенному тексту. При этом поиск осуществляется по полю "Номер".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
