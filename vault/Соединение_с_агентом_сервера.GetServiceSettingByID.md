Соединение с агентом сервера (IServerAgentConnection)

GetServiceSettingByID (GetServiceSettingByID)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetServiceSettingByID(<Кластер>, <Сервер>, <Идентификатор>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого требуется получить настройку.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Сервер, для которого требуется получить настройку.

<Идентификатор> (обязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор настройки.

Возвращаемое значение:

Тип: [Настройка сервиса](Настройка_сервиса.md).   

Описание:

Получает настройку сервиса кластера по идентификатору.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
