Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьПользователей()

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Получает массив, элементами которого являются объекты [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы.md).

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данный метод доступен пользователям, имеющим административные права.

Использование в версии:

Доступен, начиная с версии 8.0.
