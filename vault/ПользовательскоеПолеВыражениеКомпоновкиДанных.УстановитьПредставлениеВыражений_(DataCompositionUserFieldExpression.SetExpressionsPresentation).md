ПользовательскоеПолеВыражениеКомпоновкиДанных (DataCompositionUserFieldExpression)

УстановитьПредставлениеВыражений (SetExpressionsPresentation)

Доступен, начиная с версии 8.3.10.

Синтаксис:

УстановитьПредставлениеВыражений(<ПредставлениеВыраженияДетальныхЗаписей>, <ПредставлениеВыраженияИтоговыхЗаписей>)

Параметры:

<ПредставлениеВыраженияДетальныхЗаписей> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пользовательское представление выражения, используемого при выводе поля в детальных записях.

<ПредставлениеВыраженияИтоговыхЗаписей> (необязательный)

Тип: [Строка](lang__def_String.md).   
Пользовательское представление выражения, используемого при выводе поля в итоговых записях.

Описание:

Устанавливает пользовательские представления выражений, используемых при выводе поля в детальных и итоговых записях.  
Если настройки связаны с источником доступных настроек, выражения также будут обновлены.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.10.
