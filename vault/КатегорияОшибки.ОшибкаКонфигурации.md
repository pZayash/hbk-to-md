КатегорияОшибки (ErrorCategory)

ОшибкаКонфигурации (ConfigurationError)

Доступен, начиная с версии 8.3.21.

Описание:

Ошибка возникла вследствие исключения, вызванного прикладным разработчиком во время нештатной ситуации при вызове метода или процедуры (из оператора ВызватьИсключение).  
Например: неправильное количество параметров, неправильное значение параметра, некорректное внутреннее состояние.  
Рекомендуется устанавливать данную категорию при генерации исключений, которые сигнализируют об ошибке в конфигурации.

Использование в версии:

Доступен, начиная с версии 8.3.21.
