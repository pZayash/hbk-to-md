Модуль WebSocket клиента (WebSocket client module)

ПриОшибке (OnError)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПриОшибке(<Соединение>, <КодОшибки>, <Описание>)

Параметры:

<Соединение>

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md).   
Ссылка на объект соединения.

<КодОшибки>

Тип: Строка.   
Код ошибки.

<Описание>

Тип: Строка.   
Описание ошибки.

Описание:

Обработчик ошибки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
