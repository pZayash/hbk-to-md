ТаблицаФормы (FormTable)

ПроверкаПеретаскивания (DragCheck)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверкаПеретаскивания(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Строка>, <Поле>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания_(DragParameters).md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Строка>

Тип: Произвольный, [Неопределено](lang__def_Undefined.md).   
Строка таблицы, над которой находится курсор.  
Тип значения строки таблицы приведен в описаниях расширений табличного поля у соответствующих объектов.   
Если строка не определена (тип [Неопределено](lang__def_Undefined.md)), то это означает, что курсор не находится над ячейкой. Такой случай может быть обработан, например, как нестандартное добавление новой строки.

<Поле>

Тип: [ПолеФормы](ПолеФормы_(FormField).md).   
Элемент формы клиентского приложения, с которым связана данная колонка таблицы.

Описание:

Возникает при движении курсора в поле приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В веб-клиенте при перетаскивании файла (файлов) как [СсылкаНаФайл](СсылкаНаФайл_(FileRef).md) обработчик данного события элемента управления вызывается один раз в момент отпускания кнопки мыши перед обработчиком события [Перетаскивание](ТаблицаФормы.Перетаскивание_(FormTable.Drag).md) в отличие от тонкого клиента, в котором обработчик данного события вызывается в процессе перетаскивания файла (файлов).  
В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл_(FileDragMode.AsFile).md), то перетаскивание файлов игнорируется, и обработчик данного не вызывается.

Использование в версии:

Доступен, начиная с версии 8.2.
