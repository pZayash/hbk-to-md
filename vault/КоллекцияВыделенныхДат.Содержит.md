КоллекцияВыделенныхДат (SelectedDateCollection)

Содержит (Contains)

Доступен, начиная с версии 8.0.

Синтаксис:

Содержит(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: Дата.   
Дата, наличие которой в коллекции необходимо проверить.

Возвращаемое значение:

Тип: Булево.   
Истина - дата содержится в коллекции; Ложь - в противном случае.

Описание:

Определяет, содержится ли указанная дата в коллекции.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
