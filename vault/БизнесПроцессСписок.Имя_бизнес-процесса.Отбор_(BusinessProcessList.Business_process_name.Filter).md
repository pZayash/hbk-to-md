БизнесПроцессСписок.<Имя бизнес-процесса> (BusinessProcessList.<Business process name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Содержит объект [Отбор](Отбор_(Filter).md), по которому осуществляется текущая фильтрация бизнес-процессов в списке. Имена элементов отбора совпадают с именами реквизитов бизнес-процесса и критериев отбора, как они заданы в Конфигураторе. Элементы отбора представляют собой колонки списка бизнес-процессов и критерии отбора, относящиеся к бизнес-процессу.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
