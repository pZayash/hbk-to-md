КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

<Имя общего реквизита> (<Common attribute name>)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор таких свойств определяется набором общих реквизитов, в которых участвует данная константа и для которых [РазделениеДанных](ОбъектМетаданных_ОбщийРеквизит.РазделениеДанных.md) = [Разделять](РазделениеДанныхОбщегоРеквизита.Разделять.md) и [ИспользованиеРазделяемыхДанных](ОбъектМетаданных_ОбщийРеквизит.ИспользованиеРазделяемыхДанных.md) = [НезависимоИСовместно](ИспользованиеРазделяемыхДанныхОбщегоРеквизита.НезависимоИСовместно.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
