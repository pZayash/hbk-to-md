Глобальный контекст (Global context)

ПолучитьСтандартноеХранилищеПользовательскихНастроекДинамическихСписков (GetStandardDynamicListsUserSettingsStorage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСтандартноеХранилищеПользовательскихНастроекДинамическихСписков()

Возвращаемое значение:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md).   

Описание:

Получает объект, с помощью которого осуществляется доступ к содержимому стандартного хранилища общих настроек, независимо от того, заполнено ли свойство конфигурации [ХранилищеПользовательскихНастроекДинамическихСписков](ОбъектМетаданныхКонфигурация.ХранилищеПользовательскихНастроекДинамическихСписков.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
