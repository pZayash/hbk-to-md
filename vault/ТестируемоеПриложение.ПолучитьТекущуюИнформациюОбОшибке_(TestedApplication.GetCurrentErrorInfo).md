ТестируемоеПриложение (TestedApplication)

ПолучитьТекущуюИнформациюОбОшибке (GetCurrentErrorInfo)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьТекущуюИнформациюОбОшибке()

Возвращаемое значение:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md), [Неопределено](lang__def_Undefined.md).   
[ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md), если в тестируемом клиенте отображается окно с ошибкой. [Неопределено](lang__def_Undefined.md) в противном случае.

Описание:

Получает информацию об ошибке встроенного языка, сведения о которой в данный момент отображаются в тестируемом клиенте.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
