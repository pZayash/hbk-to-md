Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Элементы коллекции:

[ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются [ЭлементСоставаКопииБазыДанных](ЭлементСоставаКопииБазыДанных.md).  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента.

Описание:

Представляет собой коллекцию объектов копии базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
