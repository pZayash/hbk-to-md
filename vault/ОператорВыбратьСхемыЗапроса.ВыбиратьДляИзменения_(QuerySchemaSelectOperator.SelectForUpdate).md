ОператорВыбратьСхемыЗапроса (QuerySchemaSelectOperator)

ВыбиратьДляИзменения (SelectForUpdate)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость выбора данных для изменения.  
Соответствует выражению ДЛЯ ИЗМЕНЕНИЯ языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
