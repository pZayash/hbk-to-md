НастройкаВходныхКолонокМоделиПрогноза (PredictionModelInputColumnsSetting)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомой настройки.

Возвращаемое значение:

Тип: [НастройкаВходнойКолонкиМоделиПрогноза](НастройкаВходнойКолонкиМоделиПрогноза.md), Неопределено.   
Если настройка не найдена, то возвращается значение Неопределено.

Описание:

Осуществляет поиск настройки по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
