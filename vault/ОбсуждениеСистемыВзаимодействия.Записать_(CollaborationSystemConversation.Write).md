ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

Записать (Write)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Записать()

Описание:

Записывает изменения обсуждения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если задано свойство [КонтекстОбсуждения](ОбсуждениеСистемыВзаимодействия.КонтекстОбсуждения_(CollaborationSystemConversation.ConversationContext).md) и свойство [Отображаемое](ОбсуждениеСистемыВзаимодействия.Отображаемое_(CollaborationSystemConversation.Displayed).md) установлено в [Ложь](lang__def_BooleanFalse.md), и свойство [Участники](ОбсуждениеСистемыВзаимодействия.Участники_(CollaborationSystemConversation.Members).md) содержит не пустую коллекцию, то созданное обсуждение будет доступно только программно и только тем пользователям, которые указаны в свойстве [Участники](ОбсуждениеСистемыВзаимодействия.Участники_(CollaborationSystemConversation.Members).md).

Использование в версии:

Доступен, начиная с версии 8.3.10.
