КоллекцияИнтерваловФонаПланировщика (PlannerBackgroundIntervalCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: [Цвет](Цвет_(Color).md).   
Цвет, который должен быть установлен у искомого элемента.

Возвращаемое значение:

Тип: [ИнтервалФонаПланировщика](ИнтервалФонаПланировщика_(PlannerBackgroundInterval).md), [Неопределено](lang__def_Undefined.md).   
Если указанный элемент отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента измерения планировщика по указанному значению свойства [Цвет](ИнтервалФонаПланировщика.Цвет_(PlannerBackgroundInterval.Color).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
