ЧтениеДанных (DataReader)

НачатьЧтениеБайта (BeginReadByte)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьЧтениеБайта(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Результат> - тип Число. Прочитанный байт.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает чтение одного байта из потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Доступен, начиная с версии 8.3.9.
