XBase (XBase)

Первая (First)

Доступен, начиная с версии 8.0.

Синтаксис:

Первая()

Возвращаемое значение:

Тип: Булево.   
Истина - первая запись получена; Ложь - в противном случае.

Описание:

Позиционирует объект на первой записи. Если не установлен текущий индекс, объект позиционируется на первую запись в таблице базы данных, если установлен - на запись, имеющую самое младшее значение выражения текущего индекса (если индекс создавался с установленным флагом "Убывание", то наоборот).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB.Первая(); |

Использование в версии:

Доступен, начиная с версии 8.0.
