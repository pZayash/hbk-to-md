МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьТекущийАдресОбновления (GetCurrentUpdateAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТекущийАдресОбновления()

Возвращаемое значение:

Тип: Строка.   
Если адрес был установлен с помощью метода [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления.md), со значением не равным Неопределено, то возвращается именно этот адрес. Иначе возвращается адрес, установленный в Сборщике мобильного приложения.

Описание:

Возвращает действующий адрес автоматического обновления настроек регистрации событий/экранов статистики использования приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерСтатистикиИспользованияПриложения](МенеджерСтатистикиИспользованияПриложения.md), метод [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления.md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.
