# СерияДиаграммы.Значение

СерияДиаграммы (ChartSeries)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение, идентифицирующее серию в методе диаграммы [УстановитьСерию](Диаграмма.УстановитьСерию_(Chart.SetSeries).md). Строковое представление значения используется для представления серии, если свойство [Текст](СерияДиаграммы.Текст_(ChartSeries.Text).md) не установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
