ВыборкаДанных (DataSelection)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить()

Возвращаемое значение:

Тип: Данные, [УдалениеОбъекта](УдалениеОбъекта_(ObjectDeletion).md), [Неопределено](lang__def_Undefined.md).   
Если текущей позиции выборки соответствует удаленный объект, то возвращается значение типа [УдалениеОбъекта](УдалениеОбъекта_(ObjectDeletion).md). Если выборка находится в позиции "перед первым элементом" или "после последнего", то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Производит считывание из базы данных элемента данных, соответствующего текущей позиции выборки и возвращает соответствующий объект.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Выб = ПланыОбмена.ВыбратьИзменения(Получатель, НомерСообщения); Пока Выб.Следующий() Цикл     Объект = Выб.Получить();     ЗаписатьXML(Запись, Объект); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
