ЧтениеДанных (DataReader)

РазделитьАсинх (SplitAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: Разделение по двоичному разделителю

Синтаксис:

РазделитьАсинх()

Описание варианта метода:

Разделить данные по двоичному разделителю.  
Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Вариант синтаксиса: По двоичному разделителю

Синтаксис:

РазделитьАсинх(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Двоичный разделитель данных.

Описание варианта метода:

Разделяет остаток данных по двоичному разделителю. Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Вариант синтаксиса: Разделение по строковому разделителю

Синтаксис:

РазделитьАсинх(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста. Если не задана, то используется кодировка, заданная для текущего экземпляра объекта [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

Описание варианта метода:

Разделить данные по строковому разделителю.  
Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Вариант синтаксиса: По строковому разделителю

Синтаксис:

РазделитьАсинх(<Разделитель>)

Параметры:

<Разделитель> (обязательный)

Тип: Строка.   
Разделитель данных в виде строки.

Описание варианта метода:

Разделяет остаток данных по строковому разделителю. Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Вариант синтаксиса: Разделение по набору смешанных разделетелей

Синтаксис:

РазделитьАсинх(<КодировкаТекста>)

Параметры:

<КодировкаТекста> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Кодировка текста. Если не установлена, используется кодировка, заданная для текущего экземпляра [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

Описание варианта метода:

Разделить данные по набору смешанных разделителей.  
Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Вариант синтаксиса: По набору смешанных разделителей

Синтаксис:

РазделитьАсинх(<Разделители>)

Параметры:

<Разделители> (обязательный)

Тип: [Массив](Массив.md).   
Массив разделителей может содержать элементы типа [БуферДвоичныхДанных](БуферДвоичныхДанных.md) или Строка.

Описание варианта метода:

Разделяет остаток данных по набору смешанных разделителей. Результат Обещания - массив, содержащий значения типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - массив значений типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Описание:

Асинхронная функция. Разделяет остаток данных по заданным разделителям.

Доступность:

Тонкий клиент, толстый клиент.
