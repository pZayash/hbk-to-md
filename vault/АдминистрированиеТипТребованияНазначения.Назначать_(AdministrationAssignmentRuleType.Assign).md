АдминистрированиеТипТребованияНазначения (AdministrationAssignmentRuleType)

Назначать (Assign)

Доступен, начиная с версии 8.3.14.

Описание:

Рабочий сервер, для которого создано данное требование, будет являться одним из кандидатов на обслуживание данного объекта требования (если рабочих серверов будет несколько).

Использование в версии:

Доступен, начиная с версии 8.3.14.
