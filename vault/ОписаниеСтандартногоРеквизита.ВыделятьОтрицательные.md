ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

ВыделятьОтрицательные (MarkNegatives)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Истина - при выводе реквизита с отрицательным значением значение будет отображено особым образом.  
Имеет смысл только для реквизитов типа Число.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
