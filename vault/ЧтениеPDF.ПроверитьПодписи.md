ЧтениеPDF (PDFReader)

ПроверитьПодписи (VerifySignatures)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПроверитьПодписи(<МенеджерКриптографии>, <Сертификаты>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии.md).   
Менеджер криптографии.

<Сертификаты> (необязательный)

Тип: [Массив](Массив.md).   
В параметре возвращаются сертификаты, с помощью которых были произведены подписи (если сертификаты включены в данные подписей).

Описание:

Проверяет действительность присоединенных к PDF-документу подписей.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Примечание:

Если проверка подписи завершилась неудачно, генерируется исключение.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [ПроверитьПодписиАсинх](ЧтениеPDF.ПроверитьПодписиАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
