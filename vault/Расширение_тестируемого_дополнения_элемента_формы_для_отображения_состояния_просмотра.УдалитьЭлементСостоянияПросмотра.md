Расширение тестируемого дополнения элемента формы для отображения состояния просмотра (Tested form item addition extension for a view status representation)

УдалитьЭлементСостоянияПросмотра (DeleteViewStatusItem)

Доступен, начиная с версии 8.3.16.

Синтаксис:

УдалитьЭлементСостоянияПросмотра(<ЭлементСостоянияПросмотра>)

Параметры:

<ЭлементСостоянияПросмотра> (обязательный)

Тип: Строка, Число.   
Представление выбираемого значения или номер значения (индекс значения, нумерация с 0).

Описание:

Имитирует удаление элемента состояния просмотра (нажатие на кнопку Х).  
Если элемент с указанным заголовком или индексом не найден, генерируется исключение с текстом: "В элементе управления отсутствует указанное значение".

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
