ЗаписьЖурналаЗвонков (CallLogRecord)

НомерТелефона (PhoneNumber)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит номер телефона, с которого или по которому был осуществлен вызов.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.5.
