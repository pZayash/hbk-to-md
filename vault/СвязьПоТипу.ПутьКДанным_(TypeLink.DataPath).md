# СвязьПоТипу.ПутьКДанным

СвязьПоТипу (TypeLink)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Путь к полю формы клиентского приложения, или объекту метаданных, или свойству объекта метаданных, откуда будут браться данные для отбора.  
Пример:  
"Элементы.ТабличнаяЧасть1.ТекущиеДанные.НомерСтроки" - указывает на поле НомерСтроки табличной части ТабличнаяЧасть1 в форме клиентского приложения.  
"Наименование" - указывает на свойство "Наименование" - свойство объекта метаданных, например какого-нибудь Справочника.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
