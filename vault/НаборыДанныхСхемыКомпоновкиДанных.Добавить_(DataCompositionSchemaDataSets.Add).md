# НаборыДанныхСхемыКомпоновкиДанных.Добавить

НаборыДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSets)

Добавить (Add)

Доступен, начиная с версии 8.1.

Синтаксис:

Добавить(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [НаборДанныхЗапросСхемыКомпоновкиДанных](НаборДанныхЗапросСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetQuery).md), [НаборДанныхОбъектСхемыКомпоновкиДанных](НаборДанныхОбъектСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetObject).md), [НаборДанныхОбъединениеСхемыКомпоновкиДанных](НаборДанныхОбъединениеСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetUnion).md).

Возвращаемое значение:

Тип: [НаборДанныхЗапросСхемыКомпоновкиДанных](НаборДанныхЗапросСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetQuery).md), [НаборДанныхОбъектСхемыКомпоновкиДанных](НаборДанныхОбъектСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetObject).md), [НаборДанныхОбъединениеСхемыКомпоновкиДанных](НаборДанныхОбъединениеСхемыКомпоновкиДанных_(DataCompositionSchemaDataSetUnion).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. Если тип нового элемента не подходит для данной коллекции, добавления не происходит. В этом случае метод возвращает [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
