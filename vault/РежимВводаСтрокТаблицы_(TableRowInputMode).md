Описание:

Содержит варианты режимов ввода строк в таблице формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы_(FormTable).md), свойство [РежимВводаСтрок](ТаблицаФормы.РежимВводаСтрок_(FormTable.RowInputMode).md)
