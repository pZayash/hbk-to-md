# ЭлементСвязьHTML.ЕстьДочерниеУзлы

ЭлементСвязьHTML (HTMLLinkElement)

ЕстьДочерниеУзлы (HasChildNodes)

Доступен, начиная с версии 8.2.

Синтаксис:

ЕстьДочерниеУзлы()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - узел имеет дочерние узлы, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, имеются ли у узла дочерние узлы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
