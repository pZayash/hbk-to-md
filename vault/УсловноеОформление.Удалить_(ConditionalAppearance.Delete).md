УсловноеОформление (ConditionalAppearance)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<ОформлениеОбласти>)

Параметры:

<ОформлениеОбласти> (обязательный)

Тип: [ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md).   
Удаляемое оформление области.

Описание:

Удалить область из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
