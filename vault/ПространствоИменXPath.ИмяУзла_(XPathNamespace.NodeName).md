ПространствоИменXPath (XPathNamespace)

ИмяУзла (NodeName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя узла DOM. Имя узла присутствует не у каждого типа узла DOM.  
Узел Имя узла

- [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - Attribute.name;
- [СекцияCDATA](ТипУзлаDOM.СекцияCDATA_(DOMNodeType.CDATASection).md) - "#cdata-section";
- [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - "#comment";
- [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - "#document";
- [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md) - "#document-fragment";
- [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md) - DocumentType.name;
- [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) - Element.tagName;
- [Сущность](ТипУзлаDOM.Сущность_(DOMNodeType.Entity).md) - имя сущности;
- [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность_(DOMNodeType.EntityReference).md) - имя ссылки на сущность;
- [Нотация](ТипУзлаDOM.Нотация_(DOMNodeType.Notation).md) - имя нотации;
- [ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки_(DOMNodeType.ProcessingInstruction).md) - ProcessingInstruction.target;
- [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) - "#text".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
