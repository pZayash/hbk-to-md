ТипРазмещенияТекстаТабличногоДокумента (SpreadsheetDocumentTextPlacementType)

Забивать (Block)

Доступен, начиная с версии 8.0.

Описание:

Видимая часть текста будет ограничена левой и правой границами ячейки, если при этом текст не умещается полностью, то все символы видимой части будут отображаться как "#". (Применяется для предотвращения неправильного восприятия обрезанного текста).

Использование в версии:

Доступен, начиная с версии 8.0.
