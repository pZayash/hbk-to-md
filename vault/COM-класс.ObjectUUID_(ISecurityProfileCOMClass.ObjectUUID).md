COM-класс (ISecurityProfileCOMClass)

ObjectUUID (ObjectUUID)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит представление идентификатора COM-класса в формате системного реестра MS Windows без фигурных скобок, по которому он может быть создан операционной системой. Например, 7F58F27D-5AD8-43a1-AA1E-C982F41BED5C. Регистр букв не различается. Используется при выполнении оператора Новый COMОбъект и методом [ПолучитьCOMОбъект](Глобальный_контекст.ПолучитьCOMОбъект_(Global_context.GetCOMObject).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
