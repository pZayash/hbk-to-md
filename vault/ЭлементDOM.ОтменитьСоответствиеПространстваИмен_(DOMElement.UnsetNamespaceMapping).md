# ЭлементDOM.ОтменитьСоответствиеПространстваИмен

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ЭлементDOM](ЭлементDOM_(DOMElement).md) › [Методы](ЭлементDOM__Методы.md)

ЭлементDOM (DOMElement)

ОтменитьСоответствиеПространстваИмен (UnsetNamespaceMapping)

Доступен, начиная с версии 8.1.

Синтаксис:

ОтменитьСоответствиеПространстваИмен(<URIПространстваИмен>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

Описание:

Удаляет из элемента атрибут соответствия пространства имен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
