ВыраженияИндексаСхемыЗапроса (QuerySchemaIndexExpressions)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: Добавить выражение

Синтаксис:

Добавить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса.md), Строка.   
Выражение схемы запроса или строковое выражение.

Описание варианта метода:

Добавляет новое выражение итогов по выражению схемы запроса. Если выражение содержит ошибку, будет выдано исключение, включающее текст описания ошибки.

Вариант синтаксиса: Добавить колонку

Синтаксис:

Добавить(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса.md).   
Колонка схемы запроса.

Описание варианта метода:

Добавляет новое выражение итогов по колонке схемы запроса.

Вариант синтаксиса: Добавить по строке

Синтаксис:

Добавить(<КолонкаИлиВыражение>)

Параметры:

<КолонкаИлиВыражение> (обязательный)

Тип: Строка.   
Имя колонки или выражение.

Описание варианта метода:

По переданной строке сначала выполняется поиск колонки. Если колонка не найдена, выполняется поиск выражения из выбираемых полей. Если найдена колонка или выражение, то оно добавляется в коллекцию, иначе генерируется исключение, содержащее описание ошибки.

Возвращаемое значение:

Тип: [ВыражениеИндексаСхемыЗапроса](ВыражениеИндексаСхемыЗапроса.md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
