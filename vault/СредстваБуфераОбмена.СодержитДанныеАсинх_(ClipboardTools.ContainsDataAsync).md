СредстваБуфераОбмена (ClipboardTools)

СодержитДанныеАсинх (ContainsDataAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СодержитДанныеАсинх(<ФорматДанных>)

Параметры:

<ФорматДанных> (обязательный)

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена_(ClipboardDataStandardFormat).md), [Строка](lang__def_String.md).   
Проверяемый формат данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Булево](ТипЗначенияJSON.Булево_(JSONValueType.Boolean).md).

Описание:

Асинхронная функция. Возвращает признак того, что буфер обмена содержит данные указанного формата. Данный метод рекомендуется использовать перед вызовом метода [ПолучитьДанныеАсинх](СредстваБуфераОбмена.ПолучитьДанныеАсинх_(ClipboardTools.GetDataAsync).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Примечание:

В веб-клиенте может быть отображен запрос к пользователю. Если пользователь откажется или на сайте запрещен доступ к буферу обмена, то возвращается [Обещание](Обещание_(Promise).md), результат которого – [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.24.
