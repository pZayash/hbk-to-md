КолонкиДанныхДиаграммыГанта (GanttChartDataColumns)

Найти (Find)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Найти(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: Строка.   
Имя колонки.

Возвращаемое значение:

Тип: [КолонкаДанныхДиаграммыГанта](КолонкаДанныхДиаграммыГанта.md), Неопределено.   
Если указанная колонка отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск колонки данных диаграммы Ганта по имени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
