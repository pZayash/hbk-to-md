ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

ЗапуститьАсинх (RunAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗапуститьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания – Значение типа [РезультатЗапускаПриложенияМобильногоУстройства](РезультатЗапускаПриложенияМобильногоУстройства.md) или Неопределено, если мобильное приложение не найдено на устройстве. Возвращает результат работы запущенного мобильного приложения.

Описание:

Асинхронная функция. Запускает приложение и возвращает результат работы этого приложения. Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЗапуск](ЗапускПриложенияМобильногоУстройства.ПоддерживаетсяЗапуск.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
