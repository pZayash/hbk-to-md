# ЗаписьУзловDOM.ЗаписатьКонецЭлемента

ЗаписьУзловDOM (DOMNodeWriter)

ЗаписатьКонецЭлемента (WriteEndElement)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаписатьКонецЭлемента()

Описание:

Завершает запись элемента XML. Начало элемента записывается при вызове метода [ЗаписьУзловDOM.ЗаписатьНачалоЭлемента](ЗаписьУзловDOM.ЗаписатьНачалоЭлемента_(DOMNodeWriter.WriteStartElement).md). После записи начала элемента все, что записывается, относится к данному элементу: вложенные элементы, атрибуты, текст и т. п. Для завершения элемента следует вызвать данный метод. Каждому вызову метода [ЗаписьУзловDOM.ЗаписатьНачалоЭлемента](ЗаписьУзловDOM.ЗаписатьНачалоЭлемента_(DOMNodeWriter.WriteStartElement).md) должен соответствовать вызов данного метода.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запись.ЗаписатьНачалоЭлемента("fruits"); Запись.ЗаписатьНачалоЭлемента("apple"); Запись.ЗаписатьКонецЭлемента(); Запись.ЗаписатьНачалоЭлемента("orange"); Запись.ЗаписатьКонецЭлемента(); Запись.ЗаписатьКонецЭлемента(); |

Использование в версии:

Доступен, начиная с версии 8.1.
