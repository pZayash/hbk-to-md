ТабличноеПоле (TableBox)

ИзменятьНастройкуКолонок (ChangeSettingOfColumns)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность настройки всех колонок табличного поля.

Доступность:

Толстый клиент.

Примечание:

Если свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то при открытии формы у табличного поля будет восстановлена стандартная настройка колонок.

Использование в версии:

Доступен, начиная с версии 8.0.
