ПараметрыДоступа (AccessParameters)

Доступность (Accessibility)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит информацию о наличии права текущего пользователя на заданный объект метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - права есть, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Доступен, начиная с версии 8.0.
