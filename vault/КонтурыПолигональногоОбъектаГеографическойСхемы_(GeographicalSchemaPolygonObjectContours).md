# КонтурыПолигональногоОбъектаГеографическойСхемы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[КонтурПолигональногоОбъектаГеографическойСхемы](КонтурПолигональногоОбъектаГеографическойСхемы_(GeographicalSchemaPolygonObjectContour).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Количество (Count)](КонтурыПолигональногоОбъектаГеографическойСхемы.Количество_(GeographicalSchemaPolygonObjectContours.Count).md)  
[Очистить (Clear)](КонтурыПолигональногоОбъектаГеографическойСхемы.Очистить_(GeographicalSchemaPolygonObjectContours.Clear).md)  
[Получить (Get)](КонтурыПолигональногоОбъектаГеографическойСхемы.Получить_(GeographicalSchemaPolygonObjectContours.Get).md)  
[Удалить (Delete)](КонтурыПолигональногоОбъектаГеографическойСхемы.Удалить_(GeographicalSchemaPolygonObjectContours.Delete).md)  

Описание:

Предоставляет доступ к коллекции контуров полигонального объекта (замкнутых линий), из которых состоит полигон.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md), свойство [Контуры](ПолигональныйОбъектГеографическойСхемы.Контуры_(GeographicalSchemaObjectPolygon.Contours).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
