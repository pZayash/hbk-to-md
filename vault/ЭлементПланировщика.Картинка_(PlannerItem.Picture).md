ЭлементПланировщика (PlannerItem)

Картинка (Picture)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Картинка элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Требуемый размер картинки 16х16 пикселей.  
Картинка не должна быть [Абсолютная](ВидКартинки.Абсолютная_(PictureType.Absolute).md).

Использование в версии:

Доступен, начиная с версии 8.3.6.
