Поле (Field)

Имя (Name)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя поля ввода по строке.  
Для реквизитов объектов имена совпадает с именем соответствующего реквизита, как оно задано в Конфигураторе. Кроме того, могут быть предопределенные реквизиты "Код", "Наименование", "Дата", "Номер".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
