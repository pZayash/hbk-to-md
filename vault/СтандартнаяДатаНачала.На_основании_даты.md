СтандартнаяДатаНачала (StandardBeginningDate)

На основании даты

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СтандартнаяДатаНачала(<Дата>)

Параметры:

<Дата> (необязательный)

Тип: [Дата](lang__def_Date.md).   
Дата, которая будет использоваться как значение стандартной даты начала.

Описание:

Создает стандартную дату начала на указанную дату.

Использование в версии:

Доступен, начиная с версии 8.2.
