ПолеФормы (FormField)

ПропускатьПриВводе (SkipOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Указывает необходимость пропуска поля при вводе данных.  
Если Неопределено, то определяется значением свойства [ОтображениеПредупрежденияПриРедактировании](ПолеФормы.ОтображениеПредупрежденияПриРедактировании.md). Если указан режим [Отображать](ОтображениеПредупрежденияПриРедактировании.Отображать.md) или [Авто](ОтображениеПредупрежденияПриРедактировании.Авто.md) и в конфигурации указан режим отображения, то это эквивалентно значению Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
