ЧтениеДанных (DataReader)

КонвертируемыйРазделительСтрок (ConvertedLineDelimiter)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Конвертируемый разделитель строк. Этот параметр влияет на поведение метода [ПрочитатьСимволы](ЧтениеДанных.ПрочитатьСимволы.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЧтениеДанных](ЧтениеДанных.md), метод [ПрочитатьСимволы](ЧтениеДанных.ПрочитатьСимволы.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
