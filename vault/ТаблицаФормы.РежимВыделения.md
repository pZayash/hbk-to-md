ТаблицаФормы (FormTable)

РежимВыделения (SelectionMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимВыделенияТаблицы](РежимВыделенияТаблицы.md).   
Определяет режим выделения строк в таблице.  
Для таблицы, связанной с диаграммой Ганта:

- если установлено значение [Множественный](РежимВыделенияТаблицы.Множественный.md), то [РежимВыделенияЗначений](Расширение_поля_формы_для_поля_диаграммы_Ганта.РежимВыделенияЗначений.md) диаграммы Ганта принимает значение [Множественный](РежимВыделенияЗначенийДиаграммыГанта.Множественный.md), и наоборот;
- если свойству установлено значение [Одиночный](РежимВыделенияТаблицы.Одиночный.md), то свойство [РежимВыделенияЗначений](Расширение_поля_формы_для_поля_диаграммы_Ганта.РежимВыделенияЗначений.md) диаграммы Ганта принимает значение [Одиночный](РежимВыделенияЗначенийДиаграммыГанта.Одиночный.md), и наоборот.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
