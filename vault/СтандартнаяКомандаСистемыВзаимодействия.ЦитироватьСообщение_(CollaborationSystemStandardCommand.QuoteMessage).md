СтандартнаяКомандаСистемыВзаимодействия (CollaborationSystemStandardCommand)

ЦитироватьСообщение (QuoteMessage)

Доступен, начиная с версии 8.3.18.

Описание:

Команда цитирования сообщения.  
Добавляется в список команд по умолчанию, передаваемых в обработчик формирования команд, если источником команд является [Сообщение](ИсточникКомандСистемыВзаимодействия.Сообщение_(CollaborationSystemCommandSource.Message).md).

Использование в версии:

Доступен, начиная с версии 8.3.18.
