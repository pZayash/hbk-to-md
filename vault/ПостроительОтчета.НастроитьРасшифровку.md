ПостроительОтчета (ReportBuilder)

НастроитьРасшифровку (InitDetails)

Доступен, начиная с версии 8.0.

Синтаксис:

НастроитьРасшифровку(<ПостроительОтчета>, <Расшифровка>)

Параметры:

<ПостроительОтчета> (обязательный)

Тип: [ПостроительОтчета](ПостроительОтчета.md).   
Настраиваемый построитель отчета. Возможна передача самого построителя.

<Расшифровка> (необязательный)

Тип: [Структура](Структура.md).   
Структура, содержащая расшифровываемые имена измерений, указанные ключами структуры, и их значения, указанные соответствующими ключам значениями структуры.

Описание:

Настраивает отчет для получения расшифровки. Устанавливает у построителя отчета отбор по указанным в параметре <Расшифровка> измерениям, замещая имеющиеся по ним отборы. Добавляет в настраиваемый построитель отбор, который установлен у исходного построителя на момент выполнения метода.  
Отбор при этом ограничивается только теми измерениями, которые указаны в секции "ГДЕ" свойства [Текст](ПостроительОтчета.Текст.md). Наличие в отборе исходного построителя измерений, не присутствующих в секции "ГДЕ" настраиваемого построителя не приводит к установке отбора построителя отчета и не вызывает ошибки времени выполнения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Процедура ОтработатьРасшифровку(Расшифровка, СтандартнаяОбработка)     // ...      ПостроительОтчетов.НастроитьРасшифровку(ПостроительОтчетов, Расшифровка);     // ...  КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.0.
