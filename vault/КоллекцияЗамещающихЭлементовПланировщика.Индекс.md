КоллекцияЗамещающихЭлементовПланировщика (PlannerReplacementItemCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементПланировщика](ЭлементПланировщика.md).   
Элемент, для которого необходимо получить индекс.

Возвращаемое значение:

Тип: Число.   

Описание:

Получает индекс указанного замещающего элемента в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
