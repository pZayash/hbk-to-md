МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

Получить (Get)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Получить(<Отбор>, <ИсточникРасширенийКонфигурации>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Если параметр не задан, возвращаются все возможные расширения.  
Структура для указания отбора может содержать следующие поля:

- <УникальныйИдентификатор> (UUID) - идентификатор расширения.
- <Имя> (Name) - имя расширения.

  
Значение по умолчанию: Неопределено.

<ИсточникРасширенийКонфигурации> (необязательный)

Тип: [ИсточникРасширенийКонфигурации](ИсточникРасширенийКонфигурации.md).   
Устанавливает источник получаемого расширения.  
Значение по умолчанию: БазаДанных.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив из объектов вида [РасширениеКонфигурации](РасширениеКонфигурации.md), удовлетворяющих заданному критерию отбора.

Описание:

Получает массив расширений конфигурации

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
