НастройкиОбработкиОшибок (ErrorProcessingSettings)

ВключатьСведенияОбИнформационнойБазеВОтчет (IncludeInfobaseInformationInReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ВключатьСведенияОбИнформационнойБазеВОтчетНаКлиенте](НастройкиОбработкиОшибок.ВключатьСведенияОбИнформационнойБазеВОтчетНаКлиенте_(ErrorProcessingSettings.IncludeInfobaseInformationInReportOnClient).md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость добавлять в отчет об ошибке сведения об информационной базе.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
