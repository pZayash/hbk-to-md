Диапазон (Range)

Начало (Begin)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md), [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Нижняя граница диапазона. Значение нижней границы диапазона должно быть не больше значения верхней границы. [Неопределено](lang__def_Undefined.md) - диапазон не ограничен снизу.   
Тип значения должен совпадать с типом значения верхней границы.  
Если свойство [Конец](Диапазон.Конец_(Range.End).md) имеет тип значения, отличный от [Неопределено](lang__def_Undefined.md), то данное свойство должно иметь такой же тип значения или [Неопределено](lang__def_Undefined.md).  
Если свойство [Конец](Диапазон.Конец_(Range.End).md) имеет тип значения [Неопределено](lang__def_Undefined.md), то данное свойство не может иметь такой же тип значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
