ОбъектМетаданных: РеквизитАдресации (MetadataObject: AddressingAttribute)

ВыборГруппИЭлементов (ChoiceFoldersAndItems)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеГруппИЭлементов](ИспользованиеГруппИЭлементов_(FoldersAndItemsUse).md).   
Устанавливает вид данных, которые могут быть использованы в качестве значения реквизита: Элементы, Группы, Группы и элементы.   
Имеет смысл для ссылочных типов данных, описывающих ссылки на справочники или планы видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
