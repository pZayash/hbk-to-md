# НастройкаПериода.ПолучитьДатуОкончания

НастройкаПериода (PeriodSettings)

ПолучитьДатуОкончания (GetDateTo)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДатуОкончания()

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   

Описание:

Получает дату конца периода.

Доступность:

Толстый клиент.

Примечание:

Если граница интервала задана как [БезОграничения](ВариантГраницыИнтервала.БезОграничения_(IntervalBoundVariant.WithoutRestriction).md), то будет возвращена пустая дата.

Пример:

|  |
| --- |
| НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод; ДатаКон = НастройкаПериода.ПолучитьДатуОкончания(); |

Использование в версии:

Доступен, начиная с версии 8.0.
