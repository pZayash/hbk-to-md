# РегистрСведенийНаборЗаписей.<Имя регистра сведений>.ПроверитьЗаполнение

РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов записей. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](РегистрСведенийНаборЗаписей.Имя_регистра_сведений.ОбработкаПроверкиЗаполнения_(InformationRegisterRecordSet.Information_register_name.FillCheckProcessing).md).
