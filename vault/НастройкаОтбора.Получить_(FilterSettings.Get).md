НастройкаОтбора (FilterSettings)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента управления отбором.

Возвращаемое значение:

Тип: [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md).   

Описание:

Получает элемент управления отбором по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
