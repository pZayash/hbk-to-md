ЧтениеPDF (PDFReader)

ПроверитьПодписиАсинх (VerifySignaturesAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьПодписиАсинх(<МенеджерКриптографии>, <Сертификаты>)

Параметры:

<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

<Сертификаты> (необязательный)

Тип: [Массив](Массив_(Array).md).   
В параметре возвращаются сертификаты, с помощью которых были произведены подписи (если сертификаты включены в данные подписей).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив сертификатов, с помощью которых была произведена подпись (если сертификат включен в данные подписи).  
  
Если проверка подписи завершилась неудачно, генерируется исключение.

Описание:

Асинхронная функция. Проверяет действительность присоединенных к PDF-документу подписей.

Доступность:

Тонкий клиент, толстый клиент.
