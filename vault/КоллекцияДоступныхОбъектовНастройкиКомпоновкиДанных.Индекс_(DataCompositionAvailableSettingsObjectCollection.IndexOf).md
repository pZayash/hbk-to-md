Индекс (IndexOf)

Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ДоступныйОбъектНастройкиКомпоновкиДанных](ДоступныйОбъектНастройкиКомпоновкиДанных_(DataCompositionAvailableSettingsObject).md).

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
