Доступен, начиная с версии 8.0.

Описание:

Тип анализа данных - кластеризация.  
Параметры анализа:

- КоличествоКластеров (ClusterCount) - количество искомых кластеров (Число);
- ТипЗаполненияТаблицы (TableFillType) - какие поля выводить в таблицу. (ТипЗаполненияТаблицыРезультатаАнализаДанных);
- Стандартизация (Standardization) - необходимость стандартизации данных (ТипСтандартизацииДанных);
- МераРасстояния (DistanceMetric) - каким образом вычислять расстояние между объектами (ТипМерыРасстояния);
- МетодКластеризации (ClusterizationMethod) - каким методом выполнять кластеризацию (МетодКластеризации).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
