# СписокПолнотекстовогоПоиска.ПолучатьПредставление

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Полнотекстовый поиск](Полнотекстовый_поиск.md) › [СписокПолнотекстовогоПоиска](СписокПолнотекстовогоПоиска_(FullTextSearchList).md) › [Свойства](СписокПолнотекстовогоПоиска__Свойства.md)

СписокПолнотекстовогоПоиска (FullTextSearchList)

ПолучатьПредставление (GetPresentation)

Доступен, начиная с версии 8.3.11.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак получения представления для результатов поиска. Если установлено [Истина](lang__def_BooleanTrue.md), то для каждого из результатов поиска будет заполнено значение [Представление](ЭлементСпискаПолнотекстовогоПоиска.Представление_(FullTextSearchListItem.Presentation).md).  
Если свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), поиск выполняется быстрее.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
