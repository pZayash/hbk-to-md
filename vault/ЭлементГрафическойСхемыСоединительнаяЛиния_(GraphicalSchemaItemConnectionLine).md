Описание:

Элемент графической схемы "Соединительная линия". Предназначен для соединения элементов графической схемы, отображающих точки маршрута бизнес-процесса. Автоматически перестраивается при изменении положения элементов схемы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
