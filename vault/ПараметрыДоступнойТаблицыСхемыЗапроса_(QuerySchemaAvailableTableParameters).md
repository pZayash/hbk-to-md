**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Работа с запросами](Работа_с_запросами.md) › [Схема запроса](Схема_запроса-2.md)

Доступен, начиная с версии 8.3.5.

Элементы коллекции:

[ПараметрДоступнойТаблицыСхемыЗапроса](ПараметрДоступнойТаблицыСхемыЗапроса_(QuerySchemaAvailableTableParameter).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры доступной таблицы.  
Возможно обращение к параметру доступной таблицы посредством оператора [...]. В качестве аргумента передается индекс.  

Методы:

[Индекс (IndexOf)](ПараметрыДоступнойТаблицыСхемыЗапроса.Индекс_(QuerySchemaAvailableTableParameters.IndexOf).md)  
[Количество (Count)](ПараметрыДоступнойТаблицыСхемыЗапроса.Количество_(QuerySchemaAvailableTableParameters.Count).md)  
[Найти (Find)](ПараметрыДоступнойТаблицыСхемыЗапроса.Найти_(QuerySchemaAvailableTableParameters.Find).md)  
[Получить (Get)](ПараметрыДоступнойТаблицыСхемыЗапроса.Получить_(QuerySchemaAvailableTableParameters.Get).md)  

Описание:

Коллекция параметров доступной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
