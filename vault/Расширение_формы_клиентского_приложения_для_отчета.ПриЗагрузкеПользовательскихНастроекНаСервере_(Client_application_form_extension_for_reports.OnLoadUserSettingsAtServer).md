Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПриЗагрузкеПользовательскихНастроекНаСервере (OnLoadUserSettingsAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЗагрузкеПользовательскихНастроекНаСервере(<Настройки>, <ИспользуютсяСтандартныеНастройки>)

Параметры:

<Настройки>

Тип: [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных_(DataCompositionUserSettings).md).   
Загружаемые настройки.

<ИспользуютсяСтандартныеНастройки>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если имеет значение [Истина](lang__def_BooleanTrue.md), настройки были сформированы на базе основных настроек отчета. Если [Ложь](lang__def_BooleanFalse.md) - настройки загружены из хранилища.

Описание:

Вызывается при восстановлении пользовательских настроек, после помещения настроек в компоновщик настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
