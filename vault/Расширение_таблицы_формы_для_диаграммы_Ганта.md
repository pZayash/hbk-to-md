# Расширение таблицы формы для диаграммы Ганта

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Интерфейс.md">Интерфейс</a> › [Таблица формы](Таблица_формы.md)

Доступен, начиная с версии 8.3.21.

Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы.md), применимые для таблицы значений диаграммы Ганта.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока.md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель.md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки.md), используется идентификатор типа [ИдентификаторЗначенияДиаграммыГанта](ИдентификаторЗначенияДиаграммыГанта.md).  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные.md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки.md) возвращают объект типа [ДанныеФормыСтруктура](ДанныеФормыСтруктура.md). Именами свойств структуры будут являться имена колонок диаграммы Ганта. Изменение этих данных не изменит диаграмму Ганта.

- [ВариантУправленияВысотой](ТаблицаФормы.ВариантУправленияВысотой.md),
- [ВертикальнаяПолосаПрокрутки](ТаблицаФормы.ВертикальнаяПолосаПрокрутки.md),
