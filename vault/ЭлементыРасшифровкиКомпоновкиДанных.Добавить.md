ЭлементыРасшифровкиКомпоновкиДанных (DataCompositionDetailsItems)

Добавить (Add)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: Группировка

Синтаксис:

Добавить(<Тип>, <Родители>, <ИмяГруппировки>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип элемента расшифровки.

<Родители> (необязательный)

Тип: [Массив](Массив.md), Неопределено.   
Массив родительских элементов расшифровки.  
Если значение параметра равно Неопределено, то считается, что родителей у элемента нет.

<ИмяГруппировки> (обязательный)

Тип: Строка.   
Имя группировки, для которой нужно добавлять элемент.

Вариант синтаксиса: Значения полей

Синтаксис:

Добавить(<Тип>, <Родители>, <ЗначенияПолей>)

Параметры:

<Тип> (обязательный)

Тип: Тип.   
Тип элемента расшифровки.

<Родители> (необязательный)

Тип: [Массив](Массив.md), Неопределено.   
Массив родительских элементов расшифровки.  
Если значение параметра равно Неопределено, то считается, что родителей у элемента нет.

<ЗначенияПолей> (обязательный)

Тип: [ЗначенияПолейРасшифровкиКомпоновкиДанных](ЗначенияПолейРасшифровкиКомпоновкиДанных.md).   
Значения полей для расшифровки.

Возвращаемое значение:

Тип: [ЭлементРасшифровкиКомпоновкиДанныхПоля](ЭлементРасшифровкиКомпоновкиДанныхПоля.md), [ЭлементРасшифровкиКомпоновкиДанныхГруппировка](ЭлементРасшифровкиКомпоновкиДанныхГруппировка.md).   

Описание:

Добавляет элемент расшифровки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
