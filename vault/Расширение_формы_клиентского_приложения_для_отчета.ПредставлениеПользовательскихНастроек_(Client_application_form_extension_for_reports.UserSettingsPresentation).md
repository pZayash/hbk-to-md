Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПредставлениеПользовательскихНастроек (UserSettingsPresentation)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Строка](lang__def_String.md).   
Представление пользовательских настроек.   
Если не указан и при этом указан параметр формы [КлючПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.КлючПользовательскихНастроек_(Client_application_form_extension_for_reports.UserSettingsKey).md), то будет получено представление пользовательских настроек для переданного ключа пользовательских настроек.

Использование в версии:

Доступен, начиная с версии 8.2.
