Глобальный контекст (Global context)

ПолучитьОграничениеПовторенияПаролейПользователейСредиПоследних (GetUserPasswordReuseLimit)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьОграничениеПовторенияПаролейПользователейСредиПоследних()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает численное значение ограничения повторения паролей пользователей среди последних.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
