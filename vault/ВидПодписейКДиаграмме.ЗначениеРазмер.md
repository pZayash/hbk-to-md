ВидПодписейКДиаграмме (ChartLabelType)

ЗначениеРазмер (ValueSize)

Доступен, начиная с версии 8.3.10.

Описание:

В качестве подписи выводится значение серии в точке, дополнительно в пузырьковой диаграмме выводится значение размера.

Использование в версии:

Доступен, начиная с версии 8.3.10.
