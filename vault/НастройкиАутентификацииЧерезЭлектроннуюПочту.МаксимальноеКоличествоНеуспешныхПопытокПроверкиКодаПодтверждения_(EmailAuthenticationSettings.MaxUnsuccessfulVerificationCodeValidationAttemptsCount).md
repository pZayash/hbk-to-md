НастройкиАутентификацииЧерезЭлектроннуюПочту (EmailAuthenticationSettings)

МаксимальноеКоличествоНеуспешныхПопытокПроверкиКодаПодтверждения (MaxUnsuccessfulVerificationCodeValidationAttemptsCount)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Максимальное количество неуспешных попыток проверки кода подтверждения при аутентификации через электронную почту.  
Значение по умолчанию: 3.  
Диапазон значений: от 1 до 999999999.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.27.
