ДействиеПеретаскиванияВнутриПланировщика (PlannerInsideDragAction)

Копирование (Copy)

Доступен, начиная с версии 8.3.18.

Описание:

Указывает на то, что выполняется копирование существующих элементов планировщика путем перетаскивания с зажатой клавишей Ctrl и не зажатой клавишей Shift.

Использование в версии:

Доступен, начиная с версии 8.3.18.
