Глобальный контекст (Global context)

ПриЗасыпанииКлиентскогоПриложения (OnClientApplicationSuspend)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПриЗасыпанииКлиентскогоПриложения()

Описание:

Вызывается при уходе приложения в фоновый режим, спящий режим (на смартфонах спящий режим возникает при засыпании устройства или при переключении на другое приложение).  
Не рекомендуется выполнять длительные операции. Ограничение по времени зависит от требований используемой операционной системы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.20.
