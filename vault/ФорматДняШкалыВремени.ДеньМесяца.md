ФорматДняШкалыВремени (TimeScaleDayFormat)

ДеньМесяца (MonthDay)

Доступен, начиная с версии 8.0.

Описание:

Предпочтительный формат дня для элемента шкалы времени может быть описан как "ДЛФ=ДД". Например, дата 20.11.2004 будет представлена как "20 ноября 2004".

Использование в версии:

Доступен, начиная с версии 8.0.
