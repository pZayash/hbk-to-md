РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных (BinaryDataStorageDataCopiesPlacementMode)

РазмещениеКопийДанныхВХранилищеПриЧтенииИзБазыДанных (DataCopiesPlacementToStorageOnReadingFromDatabase)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных_(BinaryDataStorageDataCopiesPlacementMode).md).   
Данные записываются в служебную таблицу. При чтении данных сначала выполняется попытка чтения из хранилища, а в случае ошибки чтения - из служебной таблицы. Если данные отсутствуют в хранилище, то они синхронно копируются туда из служебной таблицы.   
В данном режиме хранилище используется в качестве кэша для быстрого доступа к данным, поэтому помещение данных в хранилище происходит после первого чтения.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
