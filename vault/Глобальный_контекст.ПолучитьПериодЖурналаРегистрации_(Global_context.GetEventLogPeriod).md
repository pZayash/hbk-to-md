Глобальный контекст (Global context)

ПолучитьПериодЖурналаРегистрации (GetEventLogPeriod)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьПериодЖурналаРегистрации()

Возвращаемое значение:

Тип: [Диапазон](Диапазон_(Range).md).   
[Начало](Диапазон.Начало_(Range.Begin).md) - дата самого старого события журнала регистрации,   
[Конец](Диапазон.Конец_(Range.End).md) - дата самого нового события журнала регистрации.   
Если и Начало, и Конец имеют значение [Неопределено](lang__def_Undefined.md), то журнал не содержит ни одного события.

Описание:

Возвращает начало и конец периода, за который журнал регистрации хранит события.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[Глобальный контекст](Глобальный_контекст.md), метод [СократитьЖурналРегистрации](Глобальный_контекст.СократитьЖурналРегистрации_(Global_context.TruncateEventLog).md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
