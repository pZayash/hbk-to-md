КатегорияОшибки (ErrorCategory)

ОшибкаПреобразованияДокумента (DocumentConversionError)

Доступен, начиная с версии 8.3.20.

Описание:

Ошибка является ошибкой конвертации документа. Например например, при сохранении табличного документа в формате XLS.

Использование в версии:

Доступен, начиная с версии 8.3.20.
