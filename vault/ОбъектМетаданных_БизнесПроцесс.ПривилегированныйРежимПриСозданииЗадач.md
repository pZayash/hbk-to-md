ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

ПривилегированныйРежимПриСозданииЗадач (TaskCreatingPrivilegedMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если установлено, то система все действия по формированию задач выполняет в привилегированном режиме (на сервере и в файловом варианте, однако привилегированный режим не будет установлен, если задачи бизнес-процесса формируются в клиент-серверном варианте на клиенте).  
Истина - установлен привилегированный режим.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
