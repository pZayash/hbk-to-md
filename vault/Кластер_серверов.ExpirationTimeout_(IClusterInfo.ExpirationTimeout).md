# Кластер серверов.ExpirationTimeout

Кластер серверов (IClusterInfo)

ExpirationTimeout (ExpirationTimeout)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Время принудительного завершения.  
Интервал времени (в секундах), по истечении которого выключенный процесс будет завершен принудительно даже, если еще имеются установленные с ним соединения. Если процесс выключен автоматически по достижении времени перезапуска, то после принудительного завершения процесс будет удален из кластера.   
Если значение равно 0, то принудительное завершение не производится.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
