ЧтениеFastInfoset (FastInfosetReader)

ПубличныйИдентификатор (PublicId)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит публичный идентификатор PiblicId для типов узлов, у которых идентификатор имеет смысл: [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md), [Сущность](ТипУзлаXML.Сущность_(XMLNodeType.Entity).md), [Нотация](ТипУзлаXML.Нотация_(XMLNodeType.Notation).md).  
Для типов узлов, для которых идентификатор не определен, содержит пустую строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
