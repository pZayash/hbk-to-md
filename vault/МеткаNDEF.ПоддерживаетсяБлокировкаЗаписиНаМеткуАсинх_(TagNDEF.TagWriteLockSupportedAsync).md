МеткаNDEF (TagNDEF)

ПоддерживаетсяБлокировкаЗаписиНаМеткуАсинх (TagWriteLockSupportedAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяБлокировкаЗаписиНаМеткуАсинх()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если физическая метка перезаписываемая и может быть заблокирована на запись,  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, что физическая метка NFC поддерживает блокировку на запись.  
После блокировки запись на метку будет недоступна.  
Если произошла ошибка, будет сгенерировано исключение категории [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа_(ErrorCategory.MultimediaToolsError).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МеткаNDEF](МеткаNDEF_(TagNDEF).md), метод [ЗаблокироватьЗаписьНаМеткуАсинх](МеткаNDEF.ЗаблокироватьЗаписьНаМеткуАсинх_(TagNDEF.MakeTagReadOnlyAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
