МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

ПодключитьОбработчикИзмененияСостояния (AttachStateChangeHandler)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПодключитьОбработчикИзмененияСостояния(<Обработчик>)

Параметры:

<Обработчик> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после смены состояния приложения со следующими параметрами:

- <Состояние> - Тип [СостояниеАгентаКлиентскогоПриложения](СостояниеАгентаКлиентскогоПриложения.md)
- <Дополнительно> - Значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md)

Описание:

Подключает процедуру-обработчик, вызываемую после смены состояния агента клиентского приложения.  
Допускается наличие нескольких обработчиков. В этом случае после подключения будут вызваны все обработчики. Порядок вызова не определен.  
Вызов метода с тем же обработчиком замещает обработчик.  
Сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
