МенеджерКопииБазыДанных (DatabaseCopyManager)

ПолучитьИнформациюОКопии (GetCopyInfo)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьИнформациюОКопии()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [ИнформацияОКопииБазыДанных](ИнформацияОКопииБазыДанных_(DatabaseCopyInfo).md).

Описание:

Получает информацию о текущем состоянии экземпляров копии базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
