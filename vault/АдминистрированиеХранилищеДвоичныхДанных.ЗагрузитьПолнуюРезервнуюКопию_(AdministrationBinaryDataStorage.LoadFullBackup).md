АдминистрированиеХранилищеДвоичныхДанных (AdministrationBinaryDataStorage)

ЗагрузитьПолнуюРезервнуюКопию (LoadFullBackup)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ЗагрузитьПолнуюРезервнуюКопию(<Путь>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу резервной копии на сервере.

Описание:

Восстанавливает хранилище двоичных данных из полной резервной копии. Только для встроенного хранилища.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
