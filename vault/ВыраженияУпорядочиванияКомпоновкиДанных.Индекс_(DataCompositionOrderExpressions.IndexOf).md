Доступен, начиная с версии 8.1.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ВыражениеУпорядочиванияКомпоновкиДанных](ВыражениеУпорядочиванияКомпоновкиДанных_(DataCompositionOrderExpression).md).

Использование в версии:

Доступен, начиная с версии 8.1.
