Профиль безопасности (ISecurityProfile)

ExternalAppFullAccess (ExternalAppFullAccess)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет разрешение запуска приложений.  
Истина – разрешен запуск любых приложений. Ложь– разрешен запуск только приложений, перечисленных в списке разрешенных приложений профиля.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
