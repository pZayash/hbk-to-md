# ДокументВыборка.<Имя документа>.<Имя табличной части>

ДокументВыборка.<Имя документа> (DocumentSelection.<Document name>)

<Имя табличной части> (<Tabular section name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: ДокументТабличнаяЧасть.<Имя документа>.<Имя табличной части>.   
Набор свойств содержит табличные части документа. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (ДокументТабличнаяЧасть.<Имя документа>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
