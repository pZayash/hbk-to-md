ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

Отказ (Cancel)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено значение [Истина](lang__def_BooleanTrue.md), то запись данных не отражается в истории данных.  
Если данные не используются в истории, то данное свойство игнорируется.  
Не влияет на работу [ЗаписатьВерсию](МенеджерИсторииДанных.ЗаписатьВерсию_(DataHistoryManager.WriteVersion).md), поскольку при вызове метода запись данных не выполняется.  
Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
