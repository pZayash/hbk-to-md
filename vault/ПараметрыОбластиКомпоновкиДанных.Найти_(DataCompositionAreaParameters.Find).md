ПараметрыОбластиКомпоновкиДанных (DataCompositionAreaParameters)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого элемента.

Возвращаемое значение:

Тип: [ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных_(DataCompositionExpressionAreaParameter).md), [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных_(DataCompositionDetailsAreaParameter).md), [Неопределено](lang__def_Undefined.md).   
Если элемент не найден, будет возвращено значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента коллекции по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
