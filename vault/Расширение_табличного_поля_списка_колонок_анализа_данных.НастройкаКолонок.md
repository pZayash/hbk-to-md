Расширение табличного поля списка колонок анализа данных (Data analysis column list table box extension)

НастройкаКолонок (ColumnsSetting)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [НастройкаКолонокАнализаДанных](НастройкаКолонокАнализаДанных.md).   
Коллекция значений настроек колонок анализа данных. Наименования свойств совпадают с наименованиями полей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
