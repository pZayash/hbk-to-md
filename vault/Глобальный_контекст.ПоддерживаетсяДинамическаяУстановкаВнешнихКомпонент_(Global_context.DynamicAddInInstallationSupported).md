Глобальный контекст (Global context)

ПоддерживаетсяДинамическаяУстановкаВнешнихКомпонент (DynamicAddInInstallationSupported)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПоддерживаетсяДинамическаяУстановкаВнешнихКомпонент()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - если в собранном приложении мобильного клиента есть возможность использовать динамические установленные внешние компоненты с сервера.  
Для тонкого клиента, веб-клиента и толстого клиента всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Определяет, поддерживается ли динамическая установка внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.12.

Описание изменено в версии 8.3.16.
