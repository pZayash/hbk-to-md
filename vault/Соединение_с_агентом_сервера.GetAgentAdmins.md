Соединение с агентом сервера (IServerAgentConnection)

GetAgentAdmins (GetAgentAdmins)

Доступен, начиная с версии 8.1.

Синтаксис:

GetAgentAdmins()

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив администраторов центрального сервера. Каждый администратор центрального сервера является объектом с интерфейсом [Администратор](Администратор.md).

Описание:

Получает массив администраторов центрального сервера. Для успешного выполнения метода необходима аутентификация одного из администраторов центрального сервера. Если не зарегистрирован ни один администратор центрального сервера, то перед выполнением этого метода следует выполнить метод [AuthenticateAgent](Соединение_с_агентом_сервера.AuthenticateAgent.md)("", "").

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
