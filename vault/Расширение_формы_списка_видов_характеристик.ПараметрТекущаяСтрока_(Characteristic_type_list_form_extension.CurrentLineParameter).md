# Расширение формы списка видов характеристик.ПараметрТекущаяСтрока

Расширение формы списка видов характеристик (Characteristic type list form extension)

ПараметрТекущаяСтрока (CurrentLineParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ПланВидовХарактеристикСсылка.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md).   
Осуществляет позиционирование на строку содержащегося в форме списка, отображающую устанавливаемое значение. Позиционирование происходит при открытии или повторном открытии формы. Если форма используется для выбора или подбора, данная установка имеет более высокий приоритет, чем свойство формы [НачальноеЗначениеВыбора](Форма.НачальноеЗначениеВыбора_(Form.ChoiceInitialValue).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
