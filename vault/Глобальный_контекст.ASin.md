Глобальный контекст (Global context)

ASin (ASin)

Доступен, начиная с версии 8.0.

Синтаксис:

ASin(<Число>)

Параметры:

<Число> (обязательный)

Тип: Число.   
Аргумент функции. Определен в диапазоне -1 ... 1.

Возвращаемое значение:

Тип: Число.   
Результат вычисления арксинуса в радианах.

Описание:

Вычисляет арксинус от аргумента <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

Если аргумент выходит за пределы диапазона, то происходит ошибка "Неправильное значение аргумента встроенной функции (ASin)".
