ЧтениеУзловDOM (DOMNodeReader)

СледующийАтрибут (NextAttribute)

Доступен, начиная с версии 8.1.

Синтаксис:

СледующийАтрибут()

Возвращаемое значение:

Тип: Булево.   
Истина - атрибут прочитан, Ложь - атрибутов больше нет.

Описание:

Позиционирует текущее состояние объекта на следующий атрибут текущего элемента. Если вызов осуществляется при текущем узле [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md), то вызов аналогичен методу [ЧтениеУзловDOM.ПервыйАтрибут](ЧтениеУзловDOM.ПервыйАтрибут.md).  
При этом свойствам [ЧтениеУзловDOM.ТипУзла](ЧтениеУзловDOM.ТипУзла.md), [ЧтениеУзловDOM.Имя](ЧтениеУзловDOM.Имя.md), [ЧтениеУзловDOM.ЛокальноеИмя](ЧтениеУзловDOM.ЛокальноеИмя.md), [ЧтениеУзловDOM.Префикс](ЧтениеУзловDOM.Префикс.md), [ЧтениеУзловDOM.URIПространстваИмен](ЧтениеУзловDOM.URIПространстваИмен.md), [ЧтениеУзловDOM.Значение](ЧтениеУзловDOM.Значение.md) и т.д. присваиваются значения, соответствующие прочитанным данным атрибута.  
Вызов метода [ЧтениеУзловDOM.Прочитать](ЧтениеУзловDOM.Прочитать.md) прерывает процесс чтения атрибутов элемента и позиционирует текущий узел на следующий атрибут после [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
