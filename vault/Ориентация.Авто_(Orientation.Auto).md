Ориентация (Orientation)

Авто (Auto)

Доступен, начиная с версии 8.0.

Описание:

Ориентация элемента управления определяется автоматически.  
Ориентация выбирается по соотношению сторон. Если высота больше ширины, то элемент ориентирован вертикально, если высота меньше ширины, то элемент ориентирован горизонтально.  
Для элементов управления [Индикатор](Индикатор_(ProgressBar).md), [КоманднаяПанель](КоманднаяПанель_(CommandBar).md), [ПолосаРегулирования](ПолосаРегулирования_(TrackBar).md), [Разделитель](Разделитель_(Splitter).md) и [РамкаГруппы](РамкаГруппы_(GroupBox).md) привязка к границам формы осуществляется в зависимости от ориентации элемента управления.

Использование в версии:

Доступен, начиная с версии 8.0.
