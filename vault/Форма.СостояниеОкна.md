Форма (Form)

СостояниеОкна (WindowState)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВариантСостоянияОкна](ВариантСостоянияОкна.md).   
Определяет состояние окна формы. Данное состояние можно установить окну только в том случае, если состояние окна не было ранее сохранено в профиле.

Доступность:

Толстый клиент.

Примечание:

Управление свойством должно производиться до открытия формы.

Использование в версии:

Доступен, начиная с версии 8.0.
