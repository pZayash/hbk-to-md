WSВозвращаемоеЗначение (WSReturnValue)

ВозможноПустое (Nillable)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак того, что возвращаемое значение может принимать значения типа Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - данное свойство может содержать значения Неопределено.

Использование в версии:

Доступен, начиная с версии 8.1.
