АтрибутDOM (DOMAttribute)

ЯвляетсяИдентификатором (IsId)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Определяет, является ли данный атрибут идентификатором владеющего им элемента.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - данный атрибут является идентификатором владеющего им элемента, Ложь - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.1.
