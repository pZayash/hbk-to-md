# ОформлениеМакетаОформленияКомпоновкиДанных

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › <a href="obsidian://open?file=objects__catalog63__catalog1064.md">Система компоновки данных</a> › [Макет оформления](Макет_оформления.md)

Доступен, начиная с версии 8.1.

Описание:

Коллекция элементов оформления макета оформления компоновки данных. Для оформления используются следующие элементы оформления:  
1. ЦветФона (BackColor) - тип значения [Цвет](Цвет.md);  
2. ЦветТекста (TextColor) - тип значения [Цвет](Цвет.md);  
3. ЦветГраницы (BorderColor) - тип значения [Цвет](Цвет.md);  
4. СтильГраницы (BorderStyle) - тип значения [Линия](Линия.md);  
5. СтильГраницы.Слева (BorderStyle.Left) - тип значения [Линия](Линия.md);  
6. СтильГраницы.Сверху (BorderStyle.Top) - тип значения [Линия](Линия.md);  
7. СтильГраницы.Справа (BorderStyle.Right) - тип значения [Линия](Линия.md);  
8. СтильГраницы.Снизу (BorderStyle.Bottom) - тип значения [Линия](Линия.md);  
9. Шрифт (Font) - тип значения [Шрифт](Шрифт.md);  
10. Отступ (Indent) - тип значения Число;  
11. АвтоОтступ (AutoIndent) - тип значения Число;  
12. ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение.md);  
13. ВертикальноеПоложение (VerticalAlign) - тип значения [ВертикальноеПоложение](ВертикальноеПоложение.md);  
14. Размещение (Placement) - размещение текста; тип значения [ТипРазмещенияТекстаКомпоновкиДанных](ТипРазмещенияТекстаКомпоновкиДанных.md);  
15. ОриентацияТекста (TextOrientation) - тип значения Число;  
16. Формат (Format) - тип значения Строка (форматная строка);  
17. ВыделятьОтрицательные (MarkNegatives) - тип значения Булево;  
18. МинимальнаяШирина (MinWidth) - тип значения Число;  
19. МаксимальнаяШирина (MaxWidth) - тип значения Число;  
20. МинимальнаяВысота (MinHeight) - тип значения Число;  
21. МаксимальнаяВысота (MaxHeight) - тип значения Число;  
22. РежимИзмененияРазмераКолонки (ColumnSizeChangeMode) - возможность изменять размер колонки ячейки [РежимИзмененияРазмера](РежимИзмененияРазмера.md).  
23. ОтметкаНезаполненного (MarkIncomplete) - отмечать ячейку как незаполненную; тип значения Булево.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data-composition-system/core}. Имя типа XDTO: Appearance.
