ДокументHTML (HTMLDocument)

ИзвлечьТолькоТекст (ExtractPlainText)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ИзвлечьТолькоТекст()

Возвращаемое значение:

Тип: Строка.   
Строка, содержащая весь отображаемый текст из HTML-документа, разбираемого анализатором.

Описание:

Извлекает текст из HTML-документа, записывает его в строку и возвращает эту строку.

Доступность:

Тонкий клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
