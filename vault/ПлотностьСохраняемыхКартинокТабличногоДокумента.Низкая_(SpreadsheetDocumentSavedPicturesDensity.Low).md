ПлотностьСохраняемыхКартинокТабличногоДокумента (SpreadsheetDocumentSavedPicturesDensity)

Низкая (Low)

Доступен, начиная с версии 8.3.22.

Описание:

Соответствует плотности 96 пикселей на дюйм.  
Рекомендуется использовать, если предполагается отправлять документ по электронной почте.

Использование в версии:

Доступен, начиная с версии 8.3.22.
