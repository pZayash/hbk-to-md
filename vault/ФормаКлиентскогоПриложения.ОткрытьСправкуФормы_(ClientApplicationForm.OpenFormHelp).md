ФормаКлиентскогоПриложения (ClientApplicationForm)

ОткрытьСправкуФормы (OpenFormHelp)

Доступен, начиная с версии 8.2.

Синтаксис:

ОткрытьСправкуФормы()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - окно со справочной информацией по форме открыто; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Открывает окно справочной информации по форме.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.  
Вызов метода выполняет обращение к серверу.

Примечание:

Результат выполнения кэшируется. Кэширование зависит от версии платформы и версии конфигурации для режимов запуска "Тонкий клиент" и "Веб-клиент".

Использование в версии:

Доступен, начиная с версии 8.2.
