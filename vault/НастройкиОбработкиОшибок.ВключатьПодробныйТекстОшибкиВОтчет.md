ВключатьПодробныйТекстОшибкиВОтчет (IncludeDetailErrorDescriptionInReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ВключатьПодробныйТекстОшибкиВОтчетНаКлиенте](НастройкиОбработкиОшибок.ВключатьПодробныйТекстОшибкиВОтчетНаКлиенте.md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке.md).   
Указывает необходимость добавлять отчет об ошибке подробный текст ошибки.

Доступность:

Сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
