ПланВидовРасчетаВыборка.<Имя плана видов расчета> (ChartOfCalculationTypesSelection.<Chart of calculation types name>)

ИмяПредопределенныхДанных (PredefinedDataName)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Имя предопределенного элемента, как оно задано в конфигурации.  
Если указано пустое имя пустое, элемент будет не предопределенным.  
Если указано недопустимое имя, будет вызвано исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Поле обладает особенностями сортировки (сортировка выполняется не по алфавиту).

Использование в версии:

Доступен, начиная с версии 8.3.3.
