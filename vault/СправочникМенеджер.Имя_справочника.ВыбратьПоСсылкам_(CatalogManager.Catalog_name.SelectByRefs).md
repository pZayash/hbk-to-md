# СправочникМенеджер.<Имя справочника>.ВыбратьПоСсылкам

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникМенеджер.<Имя справочника>](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md) › [Методы](СправочникМенеджер.Имя_справочника__Методы.md)

СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [РезультатЗапроса](РезультатЗапроса_(QueryResult).md), [Массив](Массив_(Array).md), [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Источник ссылок на элементы справочника.

<Колонка> (необязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Колонка источника ссылок. [Число](lang__def_Number.md) представляет индекс колонки источника. [Строка](lang__def_String.md) представляет имя колонки. Если значением параметра является [Неопределено](lang__def_Undefined.md), то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив_(Array).md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [СправочникВыборка.\<Имя справочника\>](СправочникВыборка.Имя_справочника_(CatalogSelection.Catalog_name).md).   

Описание:

Формирует выборку элементов справочника по заданному источнику ссылок на элементы справочника соответствующего типа. Если источник содержит значения, отличные от ссылок на элементы справочника соответствующего типа, или ссылки на несуществующие элементы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    Товары.Ссылка КАК Ссылка     |ИЗ     |    Справочник.Товары КАК Товары     |"; РезультатЗапроса = Запрос.Выполнить(); Выборка = Справочники.Товары.ВыбратьПоСсылкам(РезультатЗапроса); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
