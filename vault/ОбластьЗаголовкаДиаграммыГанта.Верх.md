ОбластьЗаголовкаДиаграммыГанта (GanttChartTitleArea)

Верх (Top)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Относительное положение верхней границы.  
Может содержать значения в интервале от 0 до 1.  
0 - верхний край, 0,5 - центр, 1 - нижний край.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
