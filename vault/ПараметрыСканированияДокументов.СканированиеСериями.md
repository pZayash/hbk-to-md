ПараметрыСканированияДокументов (DocumentScanningParameters)

СканированиеСериями (ScanSeries)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Запустить сканирование серии документов:

- Истина – сканирование выполняется, пока пользователь не закроет интерфейс сканирования.
- Ложь - интерфейс закрывается после предварительного просмотра полученного документа.

Значение по умолчанию: Истина.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
