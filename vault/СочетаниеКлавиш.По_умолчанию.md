СочетаниеКлавиш (Shortcut)

На основании сочетания клавиш

Доступен, начиная с версии 8.0.

Синтаксис:

Новый СочетаниеКлавиш(<Клавиша>, <Alt>, <Control>, <Shift>)

Параметры:

<Клавиша> (обязательный)

Тип: [Клавиша](Клавиша_(Key).md).   
Основная клавиша сочетания.

<Alt> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - комбинация клавиш включает Alt, [Ложь](lang__def_BooleanFalse.md) - иначе.

<Control> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - комбинация клавиш включает Control, [Ложь](lang__def_BooleanFalse.md) - иначе.

<Shift> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - комбинация клавиш включает Shift, [Ложь](lang__def_BooleanFalse.md) - иначе.

Описание:

Формирует сочетание из заданной клавиши и клавиш Alt, Control и Shift.

Использование в версии:

Доступен, начиная с версии 8.0.
