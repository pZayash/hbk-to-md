ОбъектXDTO (XDTODataObject)

Сбросить (Unset)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По выражению XPath

Синтаксис:

Сбросить(<Выражение>)

Параметры:

<Выражение> (обязательный)

Тип: Строка.   
Выражение XPath, соответствующее свойству, у которого необходимо сбросить значение.

Вариант синтаксиса: По свойству

Синтаксис:

Сбросить(<Свойство>)

Параметры:

<Свойство> (обязательный)

Тип: [СвойствоXDTO](СвойствоXDTO.md).   
Свойство, значение которого необходимо сбросить.

Описание:

Сбрасывает значения указанного свойства. Метод [Установлено](ОбъектXDTO.Установлено.md) будет возвращать значение Ложь.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для списковых свойств выполнение данного метода приведет к очистке списка.

Использование в версии:

Доступен, начиная с версии 8.1.
