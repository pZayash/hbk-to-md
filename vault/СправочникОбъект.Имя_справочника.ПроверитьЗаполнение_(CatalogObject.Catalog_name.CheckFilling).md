# СправочникОбъект.<Имя справочника>.ПроверитьЗаполнение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Методы](СправочникОбъект.Имя_справочника__Методы.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ошибок не обнаружено, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых при конфигурировании свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполнения](СправочникОбъект.Имя_справочника.ОбработкаПроверкиЗаполнения_(CatalogObject.Catalog_name.FillCheckProcessing).md).
