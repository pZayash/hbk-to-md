СредстваГеопозиционирования (LocationTools)

ОтключитьОтслеживаниеГеозон (DisableGeofencesMonitoring)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОтключитьОтслеживаниеГеозон(<Геозона>)

Параметры:

<Геозона> (обязательный)

Тип: [Геозона](Геозона_(Geofence).md), [Строка](lang__def_String.md), [Массив](Массив_(Array).md).   
Параметр может содержать идентификатор геозоны, объект или массив объектов типа [Геозона](Геозона_(Geofence).md).

Описание:

Отключает отслеживание одной или нескольких геозон для текущей информационной базы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
