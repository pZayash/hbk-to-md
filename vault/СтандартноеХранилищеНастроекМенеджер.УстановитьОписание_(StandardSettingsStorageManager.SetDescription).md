# СтандартноеХранилищеНастроекМенеджер.УстановитьОписание

СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

УстановитьОписание (SetDescription)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьОписание(<КлючОбъекта>, <КлючНастроек>, <ОписаниеНастроек>, <ИмяПользователя>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ объекта настройки.  
Подробнее см. в [Настройки, автоматически сохраняемые в системном хранилище](lang__SavingSettings.md).

<КлючНастроек> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ сохраняемых настроек.  
Если содержит пустую строку, то будет сгенерирован новый ключ сохраняемых настроек.  
Значение по умолчанию: Пустая строка.

<ОписаниеНастроек> (обязательный)

Тип: [ОписаниеНастроек](ОписаниеНастроек_(SettingsDescription).md).   
Описание настроек.   
Для стандартного хранилища настроек из данного параметра используется только свойство [Представление](ОписаниеНастроек.Представление_(SettingsDescription.Presentation).md).

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, настройки которого сохраняются.  
Если не указан, то сохраняются настройки текущего пользователя.  
Если указан пользователь, не являющийся текущим пользователем, т.е. делается попытка обратиться к настройкам, не принадлежащим текущему пользователю, то требуется наличие у текущего пользователя права АдминистрированиеДанных.

Описание:

Изменяет описание настройки или добавляет новую настройку, если такой настройки не было.  
Вызов метода требует наличия права "СохранениеДанныхПользователя".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
