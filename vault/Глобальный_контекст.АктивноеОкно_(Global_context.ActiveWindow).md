Глобальный контекст (Global context)

АктивноеОкно (ActiveWindow)

Доступен, начиная с версии 8.2.

Синтаксис:

АктивноеОкно()

Возвращаемое значение:

Тип: [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md), [Неопределено](lang__def_Undefined.md).   
В режиме совместимости [Версия8\_3\_19](РежимСовместимости.Версия8_3_19_(CompatibilityMode.Version8_3_19).md) и ниже, если активна начальная страница, метод возвращает:

- в тонком клиенте - [Неопределено](lang__def_Undefined.md),
- в веб-клиенте - объект типа [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md), у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница_(ClientApplicationWindow.HomePage).md) имеет значение [Ложь](lang__def_BooleanFalse.md).

В режиме совместимости выше [Версия8\_3\_19](РежимСовместимости.Версия8_3_19_(CompatibilityMode.Version8_3_19).md) в тонком и веб-клиенте, в интерфейсе [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md) возвращает объект типа [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md), у которого свойство [НачальнаяСтраница](ОкноКлиентскогоПриложения.НачальнаяСтраница_(ClientApplicationWindow.HomePage).md) имеет значение [Истина](lang__def_BooleanTrue.md). И свойство [Содержимое](ОкноКлиентскогоПриложения.Содержимое_(ClientApplicationWindow.Content).md) содержит формы начальной страницы.  
Если активна начальная страница то и данный метод, и свойство [Окно](ФормаКлиентскогоПриложения.Окно_(ClientApplicationForm.Window).md) формы начальной страницы возвращают один и тот же объект.

Описание:

Получает текущее активное окно.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В режиме запуска клиентского приложения "Обычное приложение" возвращает [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
