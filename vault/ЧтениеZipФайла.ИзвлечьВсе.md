ЧтениеZipФайла (ZipFileReader)

ИзвлечьВсе (ExtractAll)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [ИзвлечьВсе](ЧтениеФайлаАрхива.ИзвлечьВсе.md)

Доступен, начиная с версии 8.0.

Синтаксис:

ИзвлечьВсе(<Путь>, <ВосстанавливатьКаталоги>)

Параметры:

<Путь> (обязательный)

Тип: Строка.   
Путь, по которому необходимо распаковать элемент ZIP файла.

<ВосстанавливатьКаталоги> (необязательный)

Тип: [РежимВосстановленияПутейФайловZIP](РежимВосстановленияПутейФайловZIP.md).   
Указывает, требуется ли восстанавливать структуру каталогов.  
Значение по умолчанию: [Восстанавливать](РежимВосстановленияПутейФайловZIP.Восстанавливать.md).

Описание:

Распаковывает все элементы в ZIP-файле по указанному пути.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
