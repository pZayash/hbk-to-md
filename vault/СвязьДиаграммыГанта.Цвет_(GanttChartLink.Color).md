СвязьДиаграммыГанта (GanttChartLink)

Цвет (Color)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет связи. Допустимо значение [АвтоЦвет](ВидЦвета.АвтоЦвет_(ColorType.AutoColor).md). В этом случае используется соответствующее свойство области построения.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
