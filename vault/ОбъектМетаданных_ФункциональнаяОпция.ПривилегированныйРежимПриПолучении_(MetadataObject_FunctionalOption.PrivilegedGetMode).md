ОбъектМетаданных: ФункциональнаяОпция (MetadataObject: FunctionalOption)

ПривилегированныйРежимПриПолучении (PrivilegedGetMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то получение значения опции осуществляется в привилегированном режиме. В противном случае, значение получается в обычном режиме.  
Если [Истина](lang__def_BooleanTrue.md), то безопасный режим игнорируется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
