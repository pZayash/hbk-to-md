ЧтениеXML (XMLReader)

ПерейтиКСодержимому (MoveToContent)

Доступен, начиная с версии 8.2.

Синтаксис:

ПерейтиКСодержимому(<ПробельныеСимволы>)

Параметры:

<ПробельныеСимволы> (необязательный)

Тип: [ПробельныеСимволыXML](ПробельныеСимволыXML.md).   
Указывает способ проверки пробельных символов.  
Значение по умолчанию: [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию.md).

Возвращаемое значение:

Тип: [ТипУзлаXML](ТипУзлаXML.md).   
Значение узла, в который был перемещен текущий узел; [Ничего](ТипУзлаXML.Ничего.md) - в случае достижения конца источника данных XML.

Описание:

Проверяет является ли текущий узел "содержимым" ([Текст](ТипУзлаXML.Текст.md), [НачалоЭлемента](ТипУзлаXML.НачалоЭлемента.md), [КонецЭлемента](ТипУзлаXML.КонецЭлемента.md), [СсылкаНаСущность](ТипУзлаXML.СсылкаНаСущность.md), [КонецСущности](ТипУзлаXML.КонецСущности.md) или [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы.md)) в случае значения параметра равному [Сохранять](ПробельныеСимволыXML.Сохранять.md) или если текущее значения свойства [ПробельныеСимволы](ЧтениеXML.ПробельныеСимволы.md) равно [Сохранять](ПробельныеСимволыXML.Сохранять.md)).  
Если текущий узел не является "содержимым", то выполняется пропуск узлов до "содержимого" или же до конца источника данных XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Будут пропущены следующие типы узлов: [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML.md), [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки.md), [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента.md), [Комментарий](ТипУзлаXML.Комментарий.md), а также [ПробельныеСимволы](ТипУзлаXML.ПробельныеСимволы.md), в случае значения параметра равному [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию.md) и если текущее значения свойства [ПробельныеСимволы](ЧтениеXML.ПробельныеСимволы.md) равно [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию.md).

Использование в версии:

Доступен, начиная с версии 8.2.
