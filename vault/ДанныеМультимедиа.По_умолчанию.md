ДанныеМультимедиа (MultimediaData)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ДанныеМультимедиа(<Данные>, <ТипСодержимого>, <РасширениеФайла>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные.

<ТипСодержимого> (необязательный)

Тип: [Строка](lang__def_String.md).   
Определяет тип данных мультимедиа в стандарте MIME.

<РасширениеФайла> (необязательный)

Тип: [Строка](lang__def_String.md).   
Рекомендуемое расширение файла. Используется при сохранении данных в файл.

Использование в версии:

Доступен, начиная с версии 8.3.23.
