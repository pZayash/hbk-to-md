ТипЗаполненияОбластиТабличногоДокумента (SpreadsheetDocumentAreaFillType)

Шаблон (Template)

Доступен, начиная с версии 8.0.

Описание:

При выводе области в результирующий документ выведется текст по шаблону.

Использование в версии:

Доступен, начиная с версии 8.0.
