ЧтениеФайлаАрхива (ArchiveFileReader)

Извлечь (Extract)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Извлечь(<Элемент>, <Путь>, <ВосстанавливатьКаталоги>, <Пароль>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементФайлаАрхива](ЭлементФайлаАрхива_(ArchiveFileEntry).md).   
Элемент, который требуется распаковать.

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь, по которому требуется распаковать элемент.

<ВосстанавливатьКаталоги> (необязательный)

Тип: [РежимВосстановленияПутейФайлаАрхива](РежимВосстановленияПутейФайлаАрхива_(ArchiveFileRestoreFilePathsMode).md).   
Указывает, требуется ли восстанавливать структуру папок.  
Значение по умолчанию: [Восстанавливать](РежимВосстановленияПутейФайлаАрхива.Восстанавливать_(ArchiveFileRestoreFilePathsMode.Restore).md).

<Пароль> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывается пароль для расшифровки элемента архива, в случае, если он зашифрован с паролем, отличным от пароля общего для файла.  
Если пароль не указан, используется пароль, заданный в конструкторе или при вызове метода [Открыть](ЧтениеФайлаАрхива.Открыть_(ArchiveFileReader.Open).md), если он был там указан.

Описание:

Распаковывает переданный элемент в указанную папку.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
