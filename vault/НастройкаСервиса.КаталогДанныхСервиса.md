НастройкаСервиса (ServiceSetting)

КаталогДанныхСервиса (ServiceDataDir)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Указывает текущий каталог расположения данных сервиса на рабочем сервере. Если пустая строка, то используется каталог кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
