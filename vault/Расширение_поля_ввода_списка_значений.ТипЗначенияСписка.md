Расширение поля ввода списка значений (Value list text box extension)

ТипЗначенияСписка (ListValueType)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов.md).   
Свойство показывает, значения каких типов могут быть в списке значений у поля ввода.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
