ЧтениеДанных (DataReader)

РазделитьНаЧастиПо (SplitInPartsOf)

Доступен, начиная с версии 8.3.9.

Синтаксис:

РазделитьНаЧастиПо(<РазмерЧасти>)

Параметры:

<РазмерЧасти> (обязательный)

Тип: Число.   
Размер части данных в байтах.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив значений типа [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Описание:

Разделяет остаток данных на части заданного размера. Размер части указывается в байтах.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЧтениеДанных](ЧтениеДанных.md), метод [РазделитьНаЧастиПоАсинх](ЧтениеДанных.РазделитьНаЧастиПоАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
