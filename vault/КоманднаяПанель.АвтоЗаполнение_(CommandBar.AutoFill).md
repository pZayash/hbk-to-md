КоманднаяПанель (CommandBar)

АвтоЗаполнение (AutoFill)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак автоматического заполнения командной панели.  
Если установлено значение - [Истина](lang__def_BooleanTrue.md), командная панель будет заполнена автоматически действиями, предоставленными источником действий (см. свойство [ИсточникДействий](Расширение_элементов_управления,_расположенных_в_форме.ИсточникДействий_(Extension_for_controls_located_in_a_form.ActionSource).md) объекта [Расширение элементов управления, расположенных в форме](Расширение_элементов_управления,_расположенных_в_форме_(Extension_for_controls_located_in_a_form).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
