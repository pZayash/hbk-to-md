МакетОбластиКомпоновкиДанных (DataCompositionAreaTemplate)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableRow).md).

Возвращаемое значение:

Тип: [СтрокаТаблицыОбластиКомпоновкиДанных](СтрокаТаблицыОбластиКомпоновкиДанных_(DataCompositionAreaTemplateTableRow).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
