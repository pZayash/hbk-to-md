КоллекцияВложенийСистемыВзаимодействия (CollaborationSystemAttachmentCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс объекта или сам объект, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Размер смещения. Положительные значения производят смещение вперед, отрицательные - назад.

Описание:

Сдвигает указанный элемент на заданное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.

Описание изменено в версии 8.3.26.
