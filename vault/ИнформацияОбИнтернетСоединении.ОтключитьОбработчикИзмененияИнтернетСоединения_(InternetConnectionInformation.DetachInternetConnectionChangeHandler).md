ИнформацияОбИнтернетСоединении (InternetConnectionInformation)

ОтключитьОбработчикИзмененияИнтернетСоединения (DetachInternetConnectionChangeHandler)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ОтключитьОбработчикИзмененияИнтернетСоединения(<Процедура>)

Параметры:

<Процедура> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Имя процедуры, которая была подключена в качестве обработчика изменения интернет-соединения.

Описание:

Отключает обработчик, вызываемый при смене текущего типа или скорости соединения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
