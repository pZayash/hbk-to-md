ФормаКлиентскогоПриложения (ClientApplicationForm)

УстановитьВыполнениеПослеОбработчиковСобытия (SetExecutionAfterEventHandlers)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьВыполнениеПослеОбработчиковСобытия(<Имя>, <ДополнительныйПараметр>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя обработчика, который будет вызван следующим в цепочке.

<ДополнительныйПараметр> (необязательный)

Тип: Произвольный.   
Дополнительный параметр произвольного типа, который будет передан при вызове указанного обработчика.

Описание:

Позволяет добавить вызов обработчика после вызова базового (в обработчиках как на клиенте, так и на сервере).  
Состав параметров передаваемого метода должен быть таким же, как у обработчика, из которого был вызван метод [УстановитьВыполнениеПослеОбработчиковСобытия](ФормаКлиентскогоПриложения.УстановитьВыполнениеПослеОбработчиковСобытия_(ClientApplicationForm.SetExecutionAfterEventHandlers).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
