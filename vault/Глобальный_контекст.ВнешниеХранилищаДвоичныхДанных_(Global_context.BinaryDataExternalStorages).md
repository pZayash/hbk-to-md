Глобальный контекст (Global context)

ВнешниеХранилищаДвоичныхДанных (BinaryDataExternalStorages)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [МенеджерВнешнихХранилищДвоичныхДанных](МенеджерВнешнихХранилищДвоичныхДанных_(BinaryDataExternalStoragesManager).md).   
Используется для доступа к внешним хранилищам двоичных данных.   
Получение свойства возможно только при наличии у текущего пользователя права "Администрирование".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

См. также:

[МенеджерВнешнихХранилищДвоичныхДанных](МенеджерВнешнихХранилищДвоичныхДанных_(BinaryDataExternalStoragesManager).md)  

Использование в версии:

Доступен, начиная с версии 8.3.24.
