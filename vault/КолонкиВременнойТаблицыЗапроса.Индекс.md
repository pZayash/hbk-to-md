КолонкиВременнойТаблицыЗапроса (QueryTempTableColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаВременнойТаблицыЗапроса](КолонкаВременнойТаблицыЗапроса.md).   
Колонка результата запроса, для которой нужно определить индекс.

Возвращаемое значение:

Тип: Число.   
Индекс указанной в коллекции колонки. Если не найдена, возвращает -1.

Описание:

Возвращает индекс колонки временной таблицы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| РезультатЗапроса.Колонки.Индекс(Колонка); |

Использование в версии:

Доступен, начиная с версии 8.0.
