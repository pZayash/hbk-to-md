СлоиГеографическойСхемы (GeographicalSchemaLayers)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md).   
Объект, который необходимо удалить из коллекции слоев.

Описание:

Удаляет указанный объект из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
