HTTPЗапрос (HTTPRequest)

УстановитьТелоИзСтроки (SetBodyFromString)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьТелоИзСтроки(<ТелоКакСтрока>, <Кодировка>, <ИспользоватьBOM>)

Параметры:

<ТелоКакСтрока> (обязательный)

Тип: [Строка](lang__def_String.md).   
Содержит тело запроса в виде строки.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Определяет кодировку, в которой будет отправлен HTTP-запрос. Если параметр не задан, то тело отправляется в кодировке [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md).  
Установка этого параметра не вызывает автоматической установки соответствующего HTTP-заголовка.  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.  
  
В веб-клиенте параметр игнорируется. Строка интерпретируется в кодировке UTF-8.

<ИспользоватьBOM> (необязательный)

Тип: [ИспользованиеByteOrderMark](ИспользованиеByteOrderMark_(ByteOrderMarkUse).md).   
Использовать метку порядка байтов (BOM) для кодировок, которые ее поддерживают: UTF-8, UTF-16 и др.  
В режиме совместимости конфигурации значение по умолчанию - [Использовать](ИспользованиеByteOrderMark.Использовать_(ByteOrderMarkUse.Use).md).  
  
В веб-клиенте параметр игнорируется.  
Значение по умолчанию: [Авто](ИспользованиеByteOrderMark.Авто_(ByteOrderMarkUse.Auto).md).

Описание:

Устанавливает тело запроса из строки.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

 В веб-клиенте параметры <Кодировка>, <ИспользоватьBOM> игнорируются. Строка интерпретируется в кодировке utf-8.

Использование в версии:

Доступен, начиная с версии 8.2.
