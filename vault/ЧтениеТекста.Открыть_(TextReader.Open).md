ЧтениеТекста (TextReader)

Открыть (Open)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Открыть файл

Синтаксис:

Открыть(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <МонопольныйРежим>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя читаемого файла.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста_(TextEncoding).md), [Строка](lang__def_String.md).   
Определяет кодировку текста читаемого текстового файла (согласован с текстовым документом). Если параметр не задан, формат текста будет определен автоматически по сигнатуре BOM в начале файла; если сигнатура BOM в файле отсутствует, файл будет открыт в кодировке [ANSI](КодировкаТекста.ANSI_(TextEncoding.ANSI).md).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста_(TextEncoding).md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает строку, являющуюся разделителем строк в читаемом файле.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Указывает разделитель строк в файле при конвертации в стандартный перевод строк [ПС](Символы.ПС_(Chars.LF).md).  
Значение по умолчанию: [ВК](Символы.ВК_(Chars.CR).md) + [ПС](Символы.ПС_(Chars.LF).md).

<МонопольныйРежим> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим открытия файла. [Истина](lang__def_BooleanTrue.md) - монопольный, [Ложь](lang__def_BooleanFalse.md) - разделяемый.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание варианта метода:

Открывает текстовый файл для чтения в монопольном режиме.

Вариант синтаксиса: Установить поток

Синтаксис:

Открыть(<Поток>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения текста.

Определяет кодировку текста, читаемого из потока (согласован с текстовым документом). Описание варианта метода:

Установить поток для чтения.

Описание:

Открывает текстовый файл или поток для чтения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Текст.Открыть("d:\dos.txt", КодировкаТекста.OEM); |

Использование в версии:

Доступен, начиная с версии 8.0.
