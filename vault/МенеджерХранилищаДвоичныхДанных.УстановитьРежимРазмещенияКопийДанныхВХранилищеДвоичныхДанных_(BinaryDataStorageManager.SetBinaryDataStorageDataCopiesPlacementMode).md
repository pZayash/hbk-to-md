МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

УстановитьРежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных (SetBinaryDataStorageDataCopiesPlacementMode)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьРежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных(<РежимРазмещенияДанных>)

Параметры:

<РежимРазмещенияДанных> (необязательный)

Тип: [РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных_(BinaryDataStorageDataCopiesPlacementMode).md).   
Режим размещения копий данных.

Описание:

Устанавливает режим использования данных для встроенного хранилища двоичных данных. Требуется включенный режим оптимизированного хранения двоичных данных в базе данных ([ОшибкаСоединенияСБазойДанных](ПричинаОтключенияКопииБазыДанных.ОшибкаСоединенияСБазойДанных_(DatabaseCopyTurnedOffReason.DatabaseConnectionError).md)).  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
