ПользовательИнформационнойБазы (InfoBaseUser)

ПарольНеСоответствуетТребованиям (PasswordDoesNotSatisfyRequirements)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Показывает, соответствует ли пароль пользователя требованиям:

- Если выявлен несоответствующий пароль при аутентификации, устанавливается Истина;
- Если несоответствие не выявлено, устанавливается Ложь;
- Если меняется пароль пользователя, свойство сбрасывается (устанавливается Ложь);
- Значение по умолчанию - Ложь.

Свойство недоступно для записи, доступно для чтения только пользователю с административными правами.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
