ОформлениеЯчейки (CellAppearance)

Флажок (Check)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Булево](lang__Булево_(Boolean).md).   
Содержит значение флажка.

Доступность:

Толстый клиент.

Примечание:

Флажок со значением типа [Булево](lang__Булево_(Boolean).md) может иметь только два состояния ([Истина](lang__def_BooleanTrue.md) - установлен, [Ложь](lang__def_BooleanFalse.md) - снят), со значением типа Число может иметь три состояния (1 - установлен, 0 - снят, 2 - не определено).

Использование в версии:

Доступен, начиная с версии 8.0.
