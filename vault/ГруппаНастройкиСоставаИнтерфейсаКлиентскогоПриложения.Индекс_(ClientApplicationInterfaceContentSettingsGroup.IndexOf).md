ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceContentSettingsGroup)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.4.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения_(ClientApplicationInterfaceContentSettingsItem).md), [ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения_(ClientApplicationInterfaceContentSettingsGroup).md).   
Элемент коллекции, индекс которого требуется определить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента в коллекции.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.4.
