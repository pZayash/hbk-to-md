# КонтурПолигональногоОбъектаГеографическойСхемы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Географическая схема](Географическая_схема.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ГеографическиеКоординаты](ГеографическиеКоординаты_(GeographicCoordinates).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Описание:

Предоставляет доступ к коллекции точек, из которых состоит контур полигонального объекта.

Доступность:

Сервер, толстый клиент, внешнее соединение.
