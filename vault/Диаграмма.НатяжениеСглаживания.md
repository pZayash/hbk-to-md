Диаграмма (Chart)

НатяжениеСглаживания (SplineStrain)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит степень "натяжения" гладкой кривой. Целое значение от 0 до 100 (100 - максимальное "натяжение").  
Значение 100 означает, что кривая, проходящая между двумя точками, полностью вписана в прямоугольник, диагональю которого является отрезок, образованный этими точками.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
