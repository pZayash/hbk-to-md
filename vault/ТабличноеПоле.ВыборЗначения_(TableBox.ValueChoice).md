ТабличноеПоле (TableBox)

ВыборЗначения (ValueChoice)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыборЗначения(<СтандартнаяОбработка>, <Значение>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Значение>

Выбранное значение или массив выбранных значений в случае, если свойство [МножественныйВыбор](Форма.МножественныйВыбор_(Form.MultipleChoice).md) = [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке мыши (нажатии клавиши Enter) на ячейке табличного поля, а также при выборе строки в табличном поле посредством команды "Выбрать значение" по кнопке или кнопке командного интерфейса.

Доступность:

Толстый клиент.

Примечание:

При двойном щелчке мыши (нажатии клавиши Enter) на ячейке табличного документа данное событие следует после события [Выбор](ТабличноеПоле.Выбор_(TableBox.Selection).md). При этом если в обработчике события [Выбор](ТабличноеПоле.Выбор_(TableBox.Selection).md) установить параметру <СтандартнаяОбработка> значение [Ложь](lang__def_BooleanFalse.md), то событие ВыборЗначения не возникнет.

См. также:

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [Выбор](ТабличноеПоле.Выбор_(TableBox.Selection).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
