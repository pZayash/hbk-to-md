Категории (Categories)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Массив строк.  
Категории предоставляют дополнительные сведения о Действии (Action). При поиске приложения будут использованы только действия, которые предоставляют все запрошенные категории.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
