КоллекцияДействийСообщенияСистемыВзаимодействия (CollaborationSystemMessageActionCollection)

Добавить (Add)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Добавить](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Добавить_(CollaborationSystemMessageButtonPanelButtonRow.Add).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Добавить(<Значение>, <Текст>, <Картинка>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение действия сообщения системы взаимодействия.  
Значение по умолчанию: Неопределено.

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Текст, отображаемый в действии сообщения системы взаимодействия.  
Значение по умолчанию: Пустая строка.

<Картинка> (необязательный)

Тип: [Картинка](Картинка_(Picture).md).   
Картинка, отображаемая в действии сообщения системы взаимодействия  
Значение по умолчанию: Пустая картинка.

Возвращаемое значение:

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md).   

Описание:

Добавляет новый элемент в конец коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
