Глобальный контекст (Global context)

РабочаяДата (WorkingDate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Дата.   
Содержит рабочую дату, используемую в текущем сеансе работы с конфигурацией. Доступно для записи в случае, если свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты.md) имеет значение [Назначать](РежимРабочейДаты.Назначать.md).

Доступность:

Толстый клиент.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ИспользованиеРабочейДаты](Глобальный_контекст.ИспользованиеРабочейДаты.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
