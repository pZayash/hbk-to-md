Диаграмма (Chart)

СоединениеЗначенийПоСериям (ValuesBySeriesConnection)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [ТипСоединенияЗначенийПоСериямДиаграммы](ТипСоединенияЗначенийПоСериямДиаграммы_(ChartValuesBySeriesConnectionType).md).   
Определяет тип соединения значений по сериям в гистограммах с накоплением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
