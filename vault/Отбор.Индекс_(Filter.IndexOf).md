Отбор (Filter)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<ЭлементОтбора>)

Параметры:

<ЭлементОтбора> (обязательный)

Тип: [ЭлементОтбора](ЭлементОтбора_(FilterItem).md).   
Элемент отбора, индекс которого требуется определить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс элемента отбора в коллекции отборов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
