РасписаниеРегламентногоЗадания (JobSchedule)

ИнтервалЗавершения (CompletionInterval)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Интервал времени в секундах от начала запуска регламентного задания, после которого задание будет принудительно завершено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
