Блокировка (IObjectLock)

Connection (Connection)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Описание соединения](Описание_соединения_(IConnectionShort).md), [Неопределено](lang__def_Undefined.md).   
Содержит объект с информацией о соединении с информационной базой, установившем данную блокировку, или [Неопределено](lang__def_Undefined.md), если источником блокировки не является соединение с информационной базой.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
