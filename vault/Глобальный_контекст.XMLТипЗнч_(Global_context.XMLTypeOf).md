Глобальный контекст (Global context)

XMLТипЗнч (XMLTypeOf)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLТипЗнч(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, для типа которого нужно получить соответствующий тип данных XML.

Возвращаемое значение:

Тип: [ТипДанныхXML](ТипДанныхXML_(XMLDataType).md), [Неопределено](lang__def_Undefined.md).   
Если тип данных XML, соответствующий типу переданного в качестве параметра значения определен, то метод возвращает тип данных XML. Если не определен, то метод возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Получает тип данных XML, соответствующий типу переданного в качестве параметра значения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // Получение типа данных XML, соответствующего типу Число  ТипXML = XMLТипЗнч(10); |

Использование в версии:

Доступен, начиная с версии 8.0.
