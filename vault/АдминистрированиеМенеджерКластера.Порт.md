АдминистрированиеМенеджерКластера (AdministrationClusterManager)

Порт (Port)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Число.   
Основной порт менеджера кластера.   
От 0 до 65535. Выбирается кластером серверов автоматически из диапазонов портов для динамического занятия, определенных для рабочего сервера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
