ДанныеФормыЭлементДерева (FormDataTreeItem)

Свойство (Property)

Доступен, начиная с версии 8.2.

Синтаксис:

Свойство(<Ключ>, <НайденноеЗначение>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ для поиска свойства.

<НайденноеЗначение> (необязательный)

Тип: Произвольный.   
Содержит найденное значение. Если ключа нет в структуре, то возвращается [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ключ найден; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Получает значение элемента по указанному имени, а также проверяет, имеется ли указанное свойство.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
