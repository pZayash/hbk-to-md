# МенеджерКриптографии.НачатьПроверкуМеткиВремени

МенеджерКриптографии (CryptoManager)

НачатьПроверкуМеткиВремени (BeginVerifyingTimestamp)

Доступен, начиная с версии 8.3.20.

Синтаксис:

НачатьПроверкуМеткиВремени(<ОписаниеОповещения>, <Подпись>, <МеткаВремени>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения проверки метки времени со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Подпись> (необязательный)

Тип: [ПодписьКриптографии](ПодписьКриптографии_(CryptoSignature).md).   
Подпись из контейнера подписей.

<МеткаВремени> (необязательный)

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии_(CryptoTimestamp).md).   
Проверяемая метка времени.

Описание:

Метод начинает проверку метки времени для переданной подписи криптографии:

- Соответствие хеша, указанного в метке времени, хешу полей подписи.
- Корректность сертификата метки.
- Корректность подписи на метке времени.

При возникновении ошибки проверки, будет сгенерировано исключение с описанием ошибки.  
  
По завершении будет вызвана процедура, указанная в <ОписаниеОповещения>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.20.
