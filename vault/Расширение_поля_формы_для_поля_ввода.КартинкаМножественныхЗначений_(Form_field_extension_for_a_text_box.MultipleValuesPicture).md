Расширение поля формы для поля ввода (Form field extension for a text box)

КартинкаМножественныхЗначений (MultipleValuesPicture)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Используется, если для получения картинки множественных значений в свойстве поля ввода [ПутьКДаннымКартинкиМножественногоЗначения](Расширение_поля_формы_для_поля_ввода.ПутьКДаннымКартинкиМножественногоЗначения_(Form_field_extension_for_a_text_box.MultipleValuePictureDataPath).md) может быть указана колонка c типом [Число](lang__def_Number.md) или [Булево](lang__Булево_(Boolean).md):

- Для колонки с типом [Число](lang__def_Number.md), значение в колонке является индексом элемента в наборе картинок, задаваемых данным свойством.
- Для колонки с типом [Булево](lang__Булево_(Boolean).md), если [Истина](lang__def_BooleanTrue.md) используется индекс 0, а если [Ложь](lang__def_BooleanFalse.md) - используется индекс 1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:
