СвязьПоТипуКомпоновкиДанных (DataCompositionTypeLink)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СвязьПоТипуКомпоновкиДанных(<Поле>, <ЭлементСвязи>)

Параметры:

<Поле> (необязательный)

Тип: [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md).   
Поле, с которым осуществляется связь.  
Значение по умолчанию: Поле без имени.

<ЭлементСвязи> (необязательный)

Тип: [Число](lang__def_Number.md).   
Элемент связи по типу.  
Значение по умолчанию: 0.

Описание:

Создает объект.

Использование в версии:

Доступен, начиная с версии 8.2.
