# ВнешнийОтчет.ПолучитьМакет

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Внешние обработки и отчеты](Внешние_обработки_и_отчеты.md) › [ВнешнийОтчет](ВнешнийОтчет_(ExternalReport).md) › [Методы](ВнешнийОтчет__Методы.md)

ВнешнийОтчет (ExternalReport)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет_(MetadataObject_Template).md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md); другой объект, который может быть макетом.   

Описание:

Получает макет внешнего отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
