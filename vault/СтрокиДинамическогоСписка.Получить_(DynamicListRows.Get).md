СтрокиДинамическогоСписка (DynamicListRows)

Получить (Get)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Получить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Тип ключа строки (ссылка для ссылочных объектов, ключ записи для регистров, число при отсутствии ключа выборки).   
Ключ, для которого требуется получить строку.

Возвращаемое значение:

Тип: [СтрокаДинамическогоСписка](СтрокаДинамическогоСписка_(DynamicListRow).md).   

Описание:

Получает строку по ключу. Работает аналогично оператору [].

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
