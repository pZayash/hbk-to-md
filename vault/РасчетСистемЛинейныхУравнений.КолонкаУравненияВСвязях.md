Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Число, Строка.   
Индекс или имя колонки в [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей.md) с номерами уравнений.

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
