# Расширение поля формы для поля периода.Выбор

Расширение поля формы для поля периода (Form field extension for a period field)

Выбор (Selection)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Выбор(<ВыбраннаяДата>)

Параметры:

<ВыбраннаяДата>

Тип: [Дата](lang__def_Date.md).   
Выбранная дата.

Описание:

Возникает при выборе месяца в элементе управления, отображающем период.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
