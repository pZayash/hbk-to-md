# ДополнительныйРежимОтображения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[Неактуальность (Irrelevance)](ДополнительныйРежимОтображения.Неактуальность_(AdditionalShowMode.Irrelevance).md)  
[НеИспользовать (DontUse)](ДополнительныйРежимОтображения.НеИспользовать_(AdditionalShowMode.DontUse).md)  

Описание:

Виды дополнительного отображения состояния.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/mngsrv/ws}. Имя типа XDTO: AdditionalShowMode.

Использование в версии:

Доступен, начиная с версии 8.2.
