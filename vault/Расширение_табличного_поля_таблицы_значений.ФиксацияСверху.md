Расширение табличного поля таблицы значений (Value table table box extension)

ФиксацияСверху (FixedTop)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит количество строк таблицы значений, фиксированных сверху, то есть не прокручиваемых при прокрутке табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
