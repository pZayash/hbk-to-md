Дата (Date)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату сообщения.  
Значение свойства может быть установлено только в привилегированном сеансе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
