Доступен, начиная с версии 8.3.8.

Значения

[Авто (Auto)](ОбновлениеПредопределенныхДанных.Авто_(PredefinedDataUpdate.Auto).md)  
[НеОбновлятьАвтоматически (DontAutoUpdate)](ОбновлениеПредопределенныхДанных.НеОбновлятьАвтоматически_(PredefinedDataUpdate.DontAutoUpdate).md)

Описание:

Определяет режим обновления предопределенных данных.  
Используется в методах &ПолучитьОбновлениеПредопределенныхДанных, &УстановитьОбновлениеПредопределенныхДанных объектов [СправочникМенеджер.](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md)<[Имя справочника](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md)>, [ПланВидовРасчетаМенеджер.](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)>, [ПланВидовХарактеристикМенеджер.](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md)>, [ПланСчетовМенеджер.](ПланСчетовМенеджер.Имя_плана_счетов_(ChartOfAccountsManager.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовМенеджер.Имя_плана_счетов_(ChartOfAccountsManager.Chart_of_accounts_name).md)>, а также в методах [ПолучитьОбновлениеПредопределенныхДанныхИнформационнойБазы](Глобальный_контекст.ПолучитьОбновлениеПредопределенныхДанныхИнформационнойБазы_(Global_context.GetInfoBasePredefinedData).md) и [УстановитьОбновлениеПредопределенныхДанныхИнформационнойБазы](Глобальный_контекст.УстановитьОбновлениеПредопределенныхДанныхИнформационнойБазы_(Global_context.SetInfoBasePredefinedDataUpdate).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.1/data/enterprise. Имя типа XDTO: PredefinedDataUpdate.

Использование в версии:

Доступен, начиная с версии 8.3.8.
