ЭлементЛегендыГеографическойСхемы (GeographicalSchemaLegendItem)

ОтображатьШкалу (ShowScale)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТипОтображенияШкалыЭлементаЛегендыГеографическойСхемы](ТипОтображенияШкалыЭлементаЛегендыГеографическойСхемы_(GeographicalSchemaLegendItemShowScaleType).md).   
Определяет способ отображения шкалы значений для элемента легенды географической схемы.  
Имеет смысл только в том случае, если с элементом связана серия слоя географической схемы и тип её отрисовки установлен как один из перечисленных - [Гистограмма](ТипОтображенияСерииСлояГеографическойСхемы.Гистограмма_(GeographicalSchemaLayerSeriesShowMode.Column).md), [РазмерФигуры](ТипОтображенияСерииСлояГеографическойСхемы.РазмерФигуры_(GeographicalSchemaLayerSeriesShowMode.ShapeSize).md), [ОттенокЦветаФигуры](ТипОтображенияСерииСлояГеографическойСхемы.ОттенокЦветаФигуры_(GeographicalSchemaLayerSeriesShowMode.ShapeColorHue).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
