ГеографическаяСхема (GeographicalSchema)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все объекты из географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
