СредстваТелефонии (TelephonyTools)

ПоддерживаетсяОтправкаMMS (MMSSendingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяОтправкаMMS(<Интерактивно>)

Параметры:

<Интерактивно> (обязательный)

Тип: Булево.   
Указывает, используется ли интерактивный режим для отправки MMS-сообщений.

Возвращаемое значение:

Тип: Булево.   
При интерактивной отправке возвращает:

- Истина - на устройствах под управлением Android,
- Ложь - для остальных операционных систем.

При неинтерактивной отправке всегда возвращает Ложь.

Описание:

Определяет доступность отправки сообщений MMS с данного устройства.  
Если не поддерживается, то при вызове метода [ПослатьSMS](СредстваТелефонии.ПослатьSMS.md) вложения сообщения будут игнорироваться.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии.md), метод [ПослатьSMS](СредстваТелефонии.ПослатьSMS.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
