ЗаписьPDF (PDFWriter)

На основании имени файла

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЗаписьPDF(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет выполнена запись.

Описание:

Создает PDF-документ для записи.

Использование в версии:

Доступен, начиная с версии 8.3.16.
