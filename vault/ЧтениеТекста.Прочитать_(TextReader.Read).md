ЧтениеТекста (TextReader)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать(<РазмерСтроки>)

Параметры:

<РазмерСтроки> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер возвращаемой строки. Если параметр не указан, текстовый файл читается до конца.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
В случае, если читаемый файл завершен, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Считывает строку произвольной длины.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Строка нулевой длины не является признаком конца файла.

Пример:

|  |
| --- |
| Текст = Новый ЧтениеТекста("d:\win.txt", КодировкаТекста.ANSI); Стр = Текст.Прочитать(10); Пока Стр <> Неопределено Цикл // чтение файла по 10 символов      Сообщить(Стр);     Стр = Текст.Прочитать(10); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
