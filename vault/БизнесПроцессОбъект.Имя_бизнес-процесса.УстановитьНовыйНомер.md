БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

УстановитьНовыйНомер (SetNewNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьНовыйНомер(<ПрефиксНомера>)

Параметры:

<ПрефиксНомера> (необязательный)

Тип: Строка.   
Префикс устанавливаемого номера.  
Значение по умолчанию: Пустая строка.

Описание:

Устанавливает новый номер бизнес-процесса для заданного префикса номера.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для бизнес-процессов с нумерацией в пределах периода, номер устанавливается в соответствии с установленной датой бизнес-процесса.

Пример:

|  |
| --- |
| ВыбБизнесПроцесс.УстановитьНовыйНомер(ПрефиксИБ); |
