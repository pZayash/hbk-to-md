МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПодтвердитьПокупку (AcknowledgePurchase)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПодтвердитьПокупку(<Покупка>)

Параметры:

<Покупка> (обязательный)

Тип: [ВстроеннаяПокупка](ВстроеннаяПокупка_(InAppPurchase).md), [Строка](lang__def_String.md).   
Покупка, которую необходимо подтвердить. Параметр может быть, как строковым идентификатором покупки, так и объектом, у которого будет запрошен идентификатор.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Ложь](lang__def_BooleanFalse.md) - если покупка не была приобретена, или уже подтверждена.

Описание:

Подтверждает встроенные покупки на устройствах под управлением ОС Android.  
При работе в других ОС генерируется исключение.  
Разработчик приложения должен подтвердить, что пользователь получил доступ ко всем купленным встроенным покупкам в приложении.  
  
Внимание! Все покупки требуют подтверждения, в том числе подписки. В противном случае, покупка будет аннулирована и деньги возвращены. Для расходуемых покупок убедитесь, что вы используете метод [ИзрасходоватьПокупку](МенеджерВстроенныхПокупок.ИзрасходоватьПокупку_(InAppPurchasesManager.ConsumePurchase).md), который действует как неявное подтверждение, или можно явно подтвердить покупку с помощью данного метода.  
  
Метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяПодтверждениеПокупок](МенеджерВстроенныхПокупок.ПоддерживаетсяПодтверждениеПокупок_(InAppPurchasesManager.PurchaseAcknowledgementSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
