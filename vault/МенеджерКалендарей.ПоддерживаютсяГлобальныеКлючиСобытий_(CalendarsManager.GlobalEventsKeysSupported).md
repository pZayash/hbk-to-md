МенеджерКалендарей (CalendarsManager)

ПоддерживаютсяГлобальныеКлючиСобытий (GlobalEventsKeysSupported)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПоддерживаютсяГлобальныеКлючиСобытий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Определяет, поддерживаются ли глобальные ключи событий на устройстве.  
Предназначен для реализации возможности передачи событий между клиентом и сервером.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
