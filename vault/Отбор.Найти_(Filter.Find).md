Отбор (Filter)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяЭлементаОтбора>)

Параметры:

<ИмяЭлементаОтбора> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента отбора.

Возвращаемое значение:

Тип: [ЭлементОтбора](ЭлементОтбора_(FilterItem).md), [Неопределено](lang__def_Undefined.md).   
Если элемента с таким именем нет, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента отбора, входящего в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
