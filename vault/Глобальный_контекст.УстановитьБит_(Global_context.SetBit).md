Глобальный контекст (Global context)

УстановитьБит (SetBit)

Доступен, начиная с версии 8.3.11.

Синтаксис:

УстановитьБит(<Число>, <НомерБита>, <Значение>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Число, у которого требуется изменить значение бита.  
Значение должно быть целым числом в диапазоне от 0 до 2^32-1. Если число не целое или не укладывается в данный диапазон - генерируется исключение.

<НомерБита> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер бита, значение которого необходимо изменить.  
Значение должно быть целым числом в дипазоне от 0 до 31. Если число не является целым или не укладывается в данный диапазон, генерируется исключение.

<Значение> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Новое значение бита:

- [Истина](lang__def_BooleanTrue.md) - 1.
- [Ложь](lang__def_BooleanFalse.md) - 0.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Устанавливает заданный бит заданного числа в указанное значение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.11.
