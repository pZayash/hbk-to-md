РазрешениеКамерыУстройства (DeviceCameraResolution)

По умолчанию

Доступен, начиная с версии 8.3.6.

Синтаксис:

Новый РазрешениеКамерыУстройства(<Ширина>, <Высота>)

Параметры:

<Ширина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Значение присваивается свойству Ширина данного объекта.

<Высота> (необязательный)

Тип: [Число](lang__def_Number.md).   
Значение присваивается свойству Высота данного объекта.

Описание:

Создает описание разрешения камеры устройства.

Использование в версии:

Доступен, начиная с версии 8.3.6.
