СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает элемент справочника в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В процессе записи проверяется уникальность кода, если данные режим установлен в конфигурации.

Пример:

|  |
| --- |
| Попытка     Объект.Записать(); Исключение     Предупреждение("Не удалось записать объект """ + Объект + """!                    |" + ОписаниеОшибки(), 60); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
