СоставТабличногоПространстваБазыДанных (DatabaseTablespaceContent)

Получить (Get)

Доступен, начиная с версии 8.3.23.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента состава.

Возвращаемое значение:

Тип: [ЭлементСоставаТабличногоПространстваБазыДанных](ЭлементСоставаТабличногоПространстваБазыДанных_(DatabaseTablespaceContentItem).md).   

Описание:

Получает элемент состава табличного пространства по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
