РежимСовместимости (CompatibilityMode)

Версия8\_3\_26 (Version8\_3\_26)

Доступен, начиная с версии 8.3.27.

Описание:

Усовершенствования версии 8.3.27 частично не используются, гарантируется работа конфигураций, сделанных на платформе версии 8.3.26.

Доступен, начиная с версии 8.3.27.
