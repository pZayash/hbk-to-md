ФорматированныйДокумент (FormattedDocument)

Записать (Write)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который будет записан форматированный документ.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента.md).   
Указывает тип файла.

Описание варианта метода:

Запись форматированного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи документа.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента.md).   
Указывает формат записи документа.

Описание варианта метода:

Запись форматированного документа в поток.

Описание:

Записывает форматированный документ в файл.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ФорматированныйДокумент](ФорматированныйДокумент.md), метод [ЗаписатьАсинх](ФорматированныйДокумент.ЗаписатьАсинх.md)  
[ФорматированныйДокумент](ФорматированныйДокумент.md), метод [НачатьЗапись](ФорматированныйДокумент.НачатьЗапись.md)  

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.18.
