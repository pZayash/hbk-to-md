ПоляПостроителяЗапроса (QueryBuilderFields)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Поле>, <Смещение>)

Параметры:

<Поле> (обязательный)

Тип: [ПолеПостроителяЗапроса](ПолеПостроителяЗапроса_(QueryBuilderField).md).   
Перемещаемое поле.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Указывается, на сколько позиций осуществить сдвиг.

Описание:

Сдвигает поле на указанное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
