ЗаписиТаблицыМакетаКомпоновкиДанных (DataCompositionTemplateTableRecords)

ОбязательныеНаборыДанных (RequiredDataSets)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив содержит строки - имена наборов данных, записи которых должны обязательно присутствовать для выводимых в результат записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
