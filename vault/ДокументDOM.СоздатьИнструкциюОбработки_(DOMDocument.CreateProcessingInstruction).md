ДокументDOM (DOMDocument)

СоздатьИнструкциюОбработки (CreateProcessingInstruction)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьИнструкциюОбработки(<Цель>, <Данные>)

Параметры:

<Цель> (обязательный)

Тип: [Строка](lang__def_String.md).   
Цель инструкции обработки.

<Данные> (обязательный)

Тип: [Строка](lang__def_String.md).   
Данные инструкции обработки.

Возвращаемое значение:

Тип: [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md).   

Описание:

Создает узел инструкции обработки DOM.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
