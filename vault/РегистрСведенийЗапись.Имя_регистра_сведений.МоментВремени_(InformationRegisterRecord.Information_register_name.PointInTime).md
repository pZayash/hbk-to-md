# РегистрСведенийЗапись.<Имя регистра сведений>.МоментВремени

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры сведений](Регистры_сведений.md) › [РегистрСведенийЗапись.<Имя регистра сведений>](РегистрСведенийЗапись.Имя_регистра_сведений_(InformationRegisterRecord.Information_register_name).md) › [Методы](РегистрСведенийЗапись.Имя_регистра_сведений__Методы.md)

РегистрСведенийЗапись.<Имя регистра сведений> (InformationRegisterRecord.<Information register name>)

МоментВремени (PointInTime)

Доступен, начиная с версии 8.1.

Синтаксис:

МоментВремени()

Возвращаемое значение:

Тип: [МоментВремени](МоментВремени_(PointInTime).md).   

Описание:

Получает момент времени, соответствующий записи регистра. Применяется для регистров сведений, для которых в Конфигураторе установлен режим записи "Подчинение регистратору".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
