# КонстантаМенеджер.<Имя константы>.Установить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md) › [КонстантаМенеджер.<Имя константы>](КонстантаМенеджер.Имя_константы_(ConstantManager.Constant_Name).md) › [Методы](КонстантаМенеджер.Имя_константы__Методы.md)

КонстантаМенеджер.<Имя константы> (ConstantManager.<Constant Name>)

Установить (Set)

Доступен, начиная с версии 8.0.

Синтаксис:

Установить(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Новое значение константы.

Описание:

Устанавливает значение константы в базе данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Константы.Бухгалтер.Установить("Сидоров Петр Иванович"); |

Использование в версии:

Доступен, начиная с версии 8.0.
