ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ВертикальноеПоложениеКартинки (PictureVerticalAlign)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ВертикальноеПоложение](ВертикальноеПоложение_(VerticalAlign).md).   
Определяет способ вертикального выравнивания картинки внутри ячейки табличного документа. По умолчанию – [Низ](ВертикальноеПоложение.Низ_(VerticalAlign.Bottom).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
