# Null

**↑** [Главная](_index.md) › [Встроенный язык 1С](_index__Встроенный_язык_1С.md)

**Описание:**  
Значения данного типа используются исключительно для определения отсутствующего значения при работе с базой данных, например, при соединении таблиц.

**Доступность:** Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер). Возможен обмен с сервером. Сериализуется.    
Данный объект может быть сериализован в/из XDTO. Значение NULL представляется в XDTO в виде пустой строки.
