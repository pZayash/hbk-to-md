# ПланОбменаОбъект.<Имя плана обмена>.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [Методы](ПланОбменаОбъект.Имя_плана_обмена__Методы.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить()

Описание:

Удаляет узел из базы данных.  
Важно! В отличие от метода [УстановитьПометкуУдаления](ПланОбменаОбъект.Имя_плана_обмена.УстановитьПометкуУдаления_(ExchangePlanObject.Exchange_plan_name.SetDeletionMark).md), данный метод производит непосредственное удаление узла без возможности восстановления и без проверки ссылочной целостности. Использовать данный метод нужно крайне осмотрительно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
