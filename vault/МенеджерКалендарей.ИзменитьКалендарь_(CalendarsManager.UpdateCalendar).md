МенеджерКалендарей (CalendarsManager)

ИзменитьКалендарь (UpdateCalendar)

Доступен, начиная с версии 8.3.4.

Синтаксис:

ИзменитьКалендарь(<Календарь>, <Данные>)

Параметры:

<Календарь> (обязательный)

Тип: [ДанныеКалендаряУчетнойЗаписи](ДанныеКалендаряУчетнойЗаписи_(AccountCalendarData).md).   
Данные календаря учетной записи.

<Данные> (обязательный)

Тип: [ДанныеКалендаряУчетнойЗаписи](ДанныеКалендаряУчетнойЗаписи_(AccountCalendarData).md).   
Данные календаря учетной записи.

Описание:

Изменяет календарь в соответствии с переданными данными.  
Данный метод поддерживается не для любого календаря. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИзменениеКалендаря](МенеджерКалендарей.ПоддерживаетсяИзменениеКалендаря_(CalendarsManager.CalendarEditingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей_(CalendarsManager).md), метод [ПоддерживаетсяИзменениеКалендаря](МенеджерКалендарей.ПоддерживаетсяИзменениеКалендаря_(CalendarsManager.CalendarEditingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
