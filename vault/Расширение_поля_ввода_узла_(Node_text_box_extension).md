# Расширение поля ввода узла

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md)

Доступен, начиная с версии 8.0.

Свойства:

[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_узла.БыстрыйВыбор_(Node_text_box_extension.QuickChoice).md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на узел.  
Расширение предоставляет возможность автоматического поиска и подбора узла по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
