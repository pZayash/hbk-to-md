Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: [БизнесПроцессОбъект.\<Имя бизнес-процесса\>](БизнесПроцессОбъект.Имя_бизнес-процесса.md).   
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
