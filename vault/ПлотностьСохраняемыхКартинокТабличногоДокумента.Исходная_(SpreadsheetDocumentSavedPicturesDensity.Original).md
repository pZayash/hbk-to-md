ПлотностьСохраняемыхКартинокТабличногоДокумента (SpreadsheetDocumentSavedPicturesDensity)

Исходная (Original)

Доступен, начиная с версии 8.3.22.

Описание:

Соответствует исходной плотности картинки без изменения.  
Рекомендуется использовать, если требуется сохранить оригинальное качество картинок. Например, если после сохранения документа предполагается дальнейшая работа с картинками из этого документа.

Использование в версии:

Доступен, начиная с версии 8.3.22.
