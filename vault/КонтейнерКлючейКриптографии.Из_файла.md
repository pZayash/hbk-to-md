КонтейнерКлючейКриптографии (CryptoKeysContainer)

Из файла

Доступен, начиная с версии 8.3.11.

Синтаксис:

Новый КонтейнерКлючейКриптографии(<ИмяФайла>, <Пароль>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, из данных которого будет создан контейнер ключей криптографии. Стандартные расширения файлов для контейнера ключей криптографии – ".pfx" или ".p12".

<Пароль> (обязательный)

Тип: Строка.   
Пароль контейнера ключей криптографии.

Описание:

Создает контейнер ключей криптографии из данных файла.

Использование в версии:

Доступен, начиная с версии 8.3.11.
