Расширение поля формы для поля планировщика (Form field extension for a planner)

ПередУдалением (BeforeDelete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПередУдалением(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Установка данного параметра в Истина приведет к отмене удаления элементов.  
Значение по умолчанию: Ложь.

Описание:

Возникает при интерактивном удалении элемента. Элементы, над которыми было выполнено действие, можно получить с помощью свойства поля - [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы.md) .

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
