РасширениеКонфигурации (ConfigurationExtension)

Активно (Active)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если данное свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то при следующем запуске расширение не будет подключено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.12.
