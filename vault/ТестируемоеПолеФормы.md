Описание:

Представляет абстракцию элемента управления поле ввода. Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством [Вид](ТестируемоеПолеФормы.Вид.md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
