МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПолучитьИсториюПриобретений (GetPurchaseHistory)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьИсториюПриобретений(<ИдентификаторПокупки>)

Параметры:

<ИдентификаторПокупки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор покупки, для которого необходимо получить историю.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Важно! Если передан идентификатор покупки, которой нет в данном приложении или покупка ещё не приобреталась, будет возвращен пустой массив.

Описание:

Возвращает массив объектов типа [Дата](lang__def_Date.md), содержащий даты всех приобретений или продлений покупки.   
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИсторияПриобретений](МенеджерВстроенныхПокупок.ПоддерживаетсяИсторияПриобретений_(InAppPurchasesManager.PurchaseHistorySupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.13.
