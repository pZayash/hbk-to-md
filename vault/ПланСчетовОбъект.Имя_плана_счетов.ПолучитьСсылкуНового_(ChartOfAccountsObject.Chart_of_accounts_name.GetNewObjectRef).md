ПланСчетовОбъект.<Имя плана счетов> (ChartOfAccountsObject.<Chart of accounts name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [ПланСчетовСсылка.\<Имя плана счетов\>](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](ПланСчетовОбъект.Имя_плана_счетов.УстановитьСсылкуНового_(ChartOfAccountsObject.Chart_of_accounts_name.SetNewObjectRef).md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[ПланСчетовМенеджер.\<Имя плана счетов\>](ПланСчетовМенеджер.Имя_плана_счетов_(ChartOfAccountsManager.Chart_of_accounts_name).md), метод [ПолучитьСсылку](ПланСчетовМенеджер.Имя_плана_счетов.ПолучитьСсылку_(ChartOfAccountsManager.Chart_of_accounts_name.GetRef).md)

Доступен, начиная с версии 8.0.
