СредстваТелефонии (TelephonyTools)

ПолучитьЖурналЗвонков (GetCallLog)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьЖурналЗвонков()

Возвращаемое значение:

Тип: [ЖурналЗвонков](ЖурналЗвонков.md), Неопределено.   
Если журнал звонков недоступен, а также в мобильном приложении для Windows, вернет Неопределено.

Описание:

Получает журнал звонков.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяЖурналЗвонков](СредстваТелефонии.ПоддерживаетсяЖурналЗвонков.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии.md), метод [ПоддерживаетсяЖурналЗвонков](СредстваТелефонии.ПоддерживаетсяЖурналЗвонков.md)  

Использование в версии:

Доступен, начиная с версии 8.3.5.
