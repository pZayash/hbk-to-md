ТипДиаграммы (ChartType)

Круговая (Pie)

Доступен, начиная с версии 8.0.

Описание:

Круговая диаграмма. Строится на основании данных, представленных набором серий значений для одной точки.

Использование в версии:

Доступен, начиная с версии 8.0.
