МенеджерУведомленийКлиента (ClientNotificationManager)

ПодключитьОбработчик (AttachHandler)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПодключитьОбработчик(<Ключ>, <Обработчик>)

Параметры:

<Ключ> (обязательный)

Тип: Неопределено, Строка.   
Ключ уведомлений, для которых будет вызван обработчик.  
Если Неопределено, обработчик будет вызван для всех полученных уведомлений.

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана при получении новых уведомлений со следующими параметрами:

- <Данные> - тип Произвольный. Поддерживаются все типы, которые могут быть переданы на клиент в клиент-серверном вызове;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Подключает обработчик уведомлений клиента, который будет вызван при получении указанных уведомлений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.

Описание изменено в версии 8.3.27.
