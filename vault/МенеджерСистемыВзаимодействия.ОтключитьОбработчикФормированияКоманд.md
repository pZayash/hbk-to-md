МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтключитьОбработчикФормированияКоманд (DetachGenerateCommandsHandler)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ОтключитьОбработчикФормированияКоманд(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Описание оповещения, заданное при вызове метода [ПодключитьОбработчикФормированияКоманд](МенеджерСистемыВзаимодействия.ПодключитьОбработчикФормированияКоманд.md).

Описание:

Отключает обработчик, установленный вызовом метода [ПодключитьОбработчикФормированияКоманд](МенеджерСистемыВзаимодействия.ПодключитьОбработчикФормированияКоманд.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
