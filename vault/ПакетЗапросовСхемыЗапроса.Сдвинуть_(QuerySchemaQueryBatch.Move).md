ПакетЗапросовСхемыЗапроса (QuerySchemaQueryBatch)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [Число](lang__def_Number.md), [ЗапросВыбораСхемыЗапроса](ЗапросВыбораСхемыЗапроса_(QuerySchemaSelectQuery).md), [ЗапросУничтоженияТаблицыСхемыЗапроса](ЗапросУничтоженияТаблицыСхемыЗапроса_(QuerySchemaTableDropQuery).md).   
Индекс элемента или сам элемент, который требуется сдвинуть.  
В режиме совместимости с [Версия8\_3\_10](РежимСовместимости.Версия8_3_10_(CompatibilityMode.Version8_3_10).md) и ниже тип элемента может быть только [Число](lang__def_Number.md).

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое нужно сдвинуть указанный элемент. Положительное смещение обозначает сдвиг в конец, отрицательное - в начало коллекции.

Описание:

Сдвигает элемент в коллекции на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме совместимости с [Версия8\_3\_10](РежимСовместимости.Версия8_3_10_(CompatibilityMode.Version8_3_10).md) выбирается максимальный индекс из переданных в параметрах, и элемент по этому индексу перемещается в позицию по второму индексу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
