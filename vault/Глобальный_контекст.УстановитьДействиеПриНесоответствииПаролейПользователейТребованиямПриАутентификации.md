Глобальный контекст (Global context)

УстановитьДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации (SetActionOnUserPasswordRequirementsViolationOnAuthentication)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации(<ДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации>)

Параметры:

<ДействиеПриНесоответствииПаролейПользователейТребованиямПриАутентификации> (обязательный)

Тип: [ДействиеПриНесоответствииПароляТребованиямПриАутентификации](ДействиеПриНесоответствииПароляТребованиямПриАутентификации.md).   
Действие при несоответствии паролей требованиям при аутентификации.  
Значение по умолчанию: [Нет](ДействиеПриНесоответствииПароляТребованиямПриАутентификации.Нет.md).

Описание:

Устанавливает действие при несоответствии паролей пользователей требованиям при аутентификации – при аутентификации будет проверяться соответствие пароля требованиям:

- Минимальная длина паролей пользователей,
- Проверка сложности паролей пользователей,
- Проверка раскрытия паролей пользователей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
