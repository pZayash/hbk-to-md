РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ГраницаСнизу (BottomBorder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак наличия нижней границы у рисунка. Если значение свойства [Истина](lang__def_BooleanTrue.md), рисунок будет иметь нижнюю границу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо для рисунков любого типа, кроме [Прямая](ТипРисункаТабличногоДокумента.Прямая_(SpreadsheetDocumentDrawingType.Line).md), [Эллипс](ТипРисункаТабличногоДокумента.Эллипс_(SpreadsheetDocumentDrawingType.Ellipse).md).

Использование в версии:

Доступен, начиная с версии 8.0.
