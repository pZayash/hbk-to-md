ОбъектМетаданных: Задача (MetadataObject: Task)

ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных (ExecuteAfterWriteDataHistoryVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если имеет значение [Истина](lang__def_BooleanTrue.md), то в обработчике события [ОбработкаПослеЗаписиВерсийИсторииДанных](ЗадачаМенеджер.Имя_задачи.ОбработкаПослеЗаписиВерсийИсторииДанных_(TaskManager.Task_name.AfterWriteDataHistoryVersionsProcessing).md) можно обработать запись версии после обновления истории.  
При записи данных значение, указанное в конфигураторе, устанавливается в качестве значения свойства [ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных](ОбъектМетаданных_Задача.ВыполнятьОбработкуПослеЗаписиВерсииИсторииДанных_(MetadataObject_Task.ExecuteAfterWriteDataHistoryVersionProcessing).md). Значение свойства может быть переопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.15.
