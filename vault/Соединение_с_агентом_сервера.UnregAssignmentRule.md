Соединение с агентом сервера (IServerAgentConnection)

UnregAssignmentRule (UnregAssignmentRule)

Доступен, начиная с версии 8.3.1.

Синтаксис:

UnregAssignmentRule(<Кластер>, <Сервер>, <Требование>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, из которого нужно удалить требование.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Рабочий сервер, из которого нужно удалить требование.

<Требование> (обязательный)

Тип: [Требование назначения сервисов](Требование_назначения_сервисов.md).   
Удаляемое требование назначения сервисов.

Описание:

Удаляет требование <Требование> из списка требований сервера <Сервер> кластера <Кластер>.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
