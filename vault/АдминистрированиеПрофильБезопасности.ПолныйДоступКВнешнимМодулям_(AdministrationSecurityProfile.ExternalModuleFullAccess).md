АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКВнешнимМодулям (ExternalModuleFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет разрешение на доступ к внешним модулям:

- [Истина](lang__def_BooleanTrue.md) - разрешен доступ к любым внешним модулям,
- [Ложь](lang__def_BooleanFalse.md) - разрешен доступ только к модулям, перечисленным в списке разрешенных модулей (список разрешенных модулей можно получить с помощью метода [ПолучитьРазрешенныеВнешниеМодули](АдминистрированиеПрофильБезопасности.ПолучитьРазрешенныеВнешниеМодули_(AdministrationSecurityProfile.GetAllowedExternalModules).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
