ЭлементСпискаЗначений (ValueListItem)

Представление (Presentation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Содержит внешнее представление элемента списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[ФорматированнаяСтрока](ФорматированнаяСтрока.md) отображается с учетом формата только в выпадающем списке поля ввода управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.0.
