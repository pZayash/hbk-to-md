Описание варианта метода:

Получить актуальную версию метаданных, расширенную описаниями полей и табличных частей дополнительных данных, добавленных хотя бы в одну из версий данных, созданных по указанным данным.

Возвращаемое значение:

Тип: [Структура](Структура_(Structure).md).   
Структура вида:

- Представление (Presentation) - тип [Строка](lang__def_String.md). Обязательный. Представление объекта метаданных.
- Поля (Fields) - тип [Структура](Структура_(Structure).md). Содержит все поля версии метаданных. Ключом структуры выступает имя поля, значением - представление поля. Если метаданные версии не содержат реквизиты, элемент отсутствует.
- ТабличныеЧасти (TabularSections) - тип [Структура](Структура_(Structure).md). Содержит табличные части версии метаданных. Ключ структуры - имя табличной части. Значение - структура табличной части. Если метаданные версии не содержат табличных частей, элемент отсутствует.

  
Структура табличной части содержит поля:

- Представление (Presentation) - тип [Строка](lang__def_String.md). Представление табличной части.
- Поля (Fields) - тип [Структура](Структура_(Structure).md). Содержит все поля табличной части версии метаданных. Ключ - имя поля, значение - представление поля. Если метаданные версии не содержат полей табличной части, элемент отсутствует.

Описание:

Получает метаданные на момент создания версии данных или актуальные метаданные.  
В версии хранятся синонимы данных на всех языках конфигурации. При получении метаданных возвращается представление на языке, установленном для пользователя.  
Имя стандартного реквизита возвращается в варианте встроенного языка конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Пример:

|  |
| --- |
| // Выбор метаданных версии   МетаданныеВерсии = ИсторияДанных.ПолучитьМетаданные(     Справочники.Валюты.НайтиПоНаименованию("Рубль"), 1);  // Выбор актуальных метаданных   АктуальныеМетаданные = ИсторияДанных.ПолучитьМетаданные(     Метаданные.Справочники.Валюты); |

Использование в версии:

Доступен, начиная с версии 8.3.11.
