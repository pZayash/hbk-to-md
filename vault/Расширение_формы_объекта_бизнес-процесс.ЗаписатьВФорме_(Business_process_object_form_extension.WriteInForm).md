# Расширение формы объекта бизнес-процесс.ЗаписатьВФорме

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [Расширение формы объекта бизнес-процесс](Расширение_формы_объекта_бизнес-процесс_(Business_process_object_form_extension).md) › [Методы](Расширение_формы_объекта_бизнес-процесс__Методы.md)

Расширение формы объекта бизнес-процесс (Business process object form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - бизнес-процесс успешно записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
