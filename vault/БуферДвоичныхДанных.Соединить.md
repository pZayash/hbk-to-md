БуферДвоичныхДанных (BinaryDataBuffer)

Соединить (Concat)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Соединить(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер, который будет соединен с исходным.

Возвращаемое значение:

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   

Описание:

Создает новый буфер, содержащий элементы текущего буфера и, за ними, элементы заданного буфера.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
