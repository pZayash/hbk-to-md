# ДинамическийСписок.ПолучениеПредставленийДляНевидимыхПолей

ДинамическийСписок (DynamicList)

ПолучениеПредставленийДляНевидимыхПолей (GetInvisibleFieldPresentations)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет способ получения представлений для полей списка, не связанных с элементами формы или связанных с фактически невидимыми элементами:

- [Истина](lang__def_BooleanTrue.md) - Для тонкого и веб-клиентов динамический список получает представления для всех ссылок, которые передаются на клиента. В том числе, для полей не связанных с элементами формы или связанных с фактически невидимыми элементами.
- [Ложь](lang__def_BooleanFalse.md) - Для тонкого и веб клиентов динамический список получает представления только для ссылок расположенных в полях, связанных с фактически видимыми элементами. Если таблица, с которой связан динамический список, находится в режиме выбора, для ключевых полей представления получаются.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
