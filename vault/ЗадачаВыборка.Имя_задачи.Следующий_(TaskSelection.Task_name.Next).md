# ЗадачаВыборка.<Имя задачи>.Следующий

ЗадачаВыборка.<Имя задачи> (TaskSelection.<Task name>)

Следующий (Next)

Доступен, начиная с версии 8.0.

Синтаксис:

Следующий()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - следующая задача выбрана; [Ложь](lang__def_BooleanFalse.md) - достигнут конец выборки.

Описание:

Получает следующую задачу из выборки. Для обхода задач нужно после получения выборки вызвать данный метод для позиционирования на первую задачу и далее вызывать до тех пор, пока не будет возвращено значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Выборка = Задачи.Основные.Выбрать(); Пока Выборка.Следующий() Цикл     // обработка выборки  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
