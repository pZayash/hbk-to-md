ИнтернетПочта (InternetMail)

ПолучитьФлагиСообщений (GetMessagesFlags)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьФлагиСообщений(<МассивИдентификаторовСообщений>)

Параметры:

<МассивИдентификаторовСообщений> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив серверных идентификаторов сообщений, для которых нужно получить флаги.

Возвращаемое значение:

Тип: [Соответствие](Соответствие_(Map).md).   
Соответствие, в котором ключом является идентификатор почтового сообщения, а значением - объект типа [ФлагиИнтернетПочтовогоСообщения](ФлагиИнтернетПочтовогоСообщения_(InternetMailMessageFlags).md) .

Описание:

Получает с почтового сервера флаги, установленные для сообщений.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.12.
