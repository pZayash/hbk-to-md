КоллекцияРисунковТабличногоДокумента (SpreadsheetDocumentDrawingCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ТипРисунка>)

Параметры:

<ТипРисунка> (обязательный)

Тип: [ТипРисункаТабличногоДокумента](ТипРисункаТабличногоДокумента.md).   
Тип добавляемого рисунка.

Возвращаемое значение:

Тип: [РисунокТабличногоДокумента](РисунокТабличногоДокумента.md).   

Описание:

Добавляет рисунок в конец коллекции рисунков табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Нельзя добавлять в коллекцию рисунок типа [Примечание](ТипРисункаТабличногоДокумента.Примечание.md).

Использование в версии:

Доступен, начиная с версии 8.0.
