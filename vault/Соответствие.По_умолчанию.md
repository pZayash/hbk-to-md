Соответствие (Map)

По умолчанию

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Соответствие()

Описание:

Создает пустое соответствие (без элементов).

Пример:

|  |
| --- |
| ПараметрыОбработки = Новый Соответствие; |

Использование в версии:

Доступен, начиная с версии 8.0.
