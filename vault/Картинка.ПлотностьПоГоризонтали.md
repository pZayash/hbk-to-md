Картинка (Picture)

ПлотностьПоГоризонтали (HorizontalDensity)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПлотностьПоГоризонтали()

Возвращаемое значение:

Тип: Число, Неопределено.   
Если картинка создана с вариантами, то возвращает информацию по текущему варианту.  
Если картинка является частью коллекции картинок, возвращает Неопределено.  
На мобильной платформе и мобильных клиентах всегда возвращает Неопределено.

Описание:

Возвращает плотность изображения по горизонтали - число точек на дюйм (DPI).  
Важно! Если формат картинки не предоставляет эту информацию, то будет возвращена стандартная плотность, равная 72 точкам на дюйм (разрешение экрана).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
