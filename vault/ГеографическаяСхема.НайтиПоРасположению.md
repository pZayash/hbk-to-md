ГеографическаяСхема (GeographicalSchema)

НайтиПоРасположению (FindByLocation)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоРасположению(<Объект>, <ГдеИскать>, <ТипПоиска>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md).   
Объект поиска.

<ГдеИскать> (обязательный)

Тип: [Массив](Массив.md).   
Массив может содержать объекты следующих типов:  
[ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы.md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы.md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы.md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы.md), [ГеографическиеКоординаты](ГеографическиеКоординаты.md),   
[ПрямоугольникГеографическойСхемы](ПрямоугольникГеографическойСхемы.md).

<ТипПоиска> (обязательный)

Тип: [ТипПоискаОбъектовГеографическойСхемы](ТипПоискаОбъектовГеографическойСхемы.md).   
Определяет способ поиска объекта географической схемы.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Проверяет объекты по взаимному расположению.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
