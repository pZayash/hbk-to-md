КоллекцияЭлементовУсловногоОформленияКомпоновкиДанных (DataCompositionConditionalAppearanceItemCollection)

Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [ЭлементУсловногоОформленияКомпоновкиДанных](ЭлементУсловногоОформленияКомпоновкиДанных_(DataCompositionConditionalAppearanceItem).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
