РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ОриентацияТекста (TextOrientation)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит угол наклона текста по отношению к горизонтальному положению в градусах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md), [Примечание](ТипРисункаТабличногоДокумента.Примечание_(SpreadsheetDocumentDrawingType.Comment).md).

Использование в версии:

Доступен, начиная с версии 8.0.
