ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Запись>)

Параметры:

<Запись> (обязательный)

Тип: Число, [ПерерасчетЗапись.](ПерерасчетЗапись.Имя_перерасчета.md)<[Имя перерасчета](ПерерасчетЗапись.Имя_перерасчета.md)>.   
Индекс записи или сама запись, которую требуется удалить.

Описание:

Удаляет запись из набора по указанному индексу записи или объекту записи перерасчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.
