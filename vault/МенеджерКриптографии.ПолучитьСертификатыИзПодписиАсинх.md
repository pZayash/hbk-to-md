МенеджерКриптографии (CryptoManager)

ПолучитьСертификатыИзПодписиАсинх (GetCertificatesFromSignatureAsync)

Не рекомендуется использовать, начиная с версии 8.3.24.

Рекомендуется использовать:

- [ПолучитьСертификаты](МенеджерКриптографии.ПолучитьСертификаты.md)
- [ПолучитьСертификатыАсинх](МенеджерКриптографии.ПолучитьСертификатыАсинх.md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьСертификатыИзПодписиАсинх(<ИсходныеДанные>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md).   
Исходные данные, из которых будет извлечен сертификат.  
Данные могут размещаться в файле (в этом случае указывается имя файла) или представлены как [ДвоичныеДанные](ДвоичныеДанные.md).

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - массив объектов [СертификатКриптографии](СертификатКриптографии.md).

Описание:

Асинхронная функция. Извлекает массив сертификатов из данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
