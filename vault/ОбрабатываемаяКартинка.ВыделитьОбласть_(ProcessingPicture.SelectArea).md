# ОбрабатываемаяКартинка.ВыделитьОбласть

ОбрабатываемаяКартинка (ProcessingPicture)

ВыделитьОбласть (SelectArea)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ВыделитьОбласть(<Лево>, <Верх>, <Ширина>, <Высота>)

Параметры:

<Лево> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество пикселей слева относительно левой верхней точки исходной картинки.

<Верх> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество пикселей сверху относительно левой верхней точки исходной картинки.

<Ширина> (обязательный)

Тип: [Число](lang__def_Number.md).   
Ширина области.

<Высота> (обязательный)

Тип: [Число](lang__def_Number.md).   
Высота области в пикселях.

Описание:

Выделяет из картинки прямоугольную область, ограниченную переданными параметрами.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
