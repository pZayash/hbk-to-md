ТипSMSСредствТелефонии (TelephonyToolsSMSType)

Исходящее (Outgoing)

Доступен, начиная с версии 8.3.11.

Описание:

Исходящее сообщение. Данный статус является временным. При отправке статус сообщения изменится на один из статусов: [ОшибкаОтправки](ТипSMSСредствТелефонии.ОшибкаОтправки_(TelephonyToolsSMSType.Failed).md), [ВОчереди](ТипSMSСредствТелефонии.ВОчереди_(TelephonyToolsSMSType.Queued).md), [Отправленное](ТипSMSСредствТелефонии.Отправленное_(TelephonyToolsSMSType.Sent).md).

Использование в версии:

Доступен, начиная с версии 8.3.11.
