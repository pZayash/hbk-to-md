Глобальный контекст (Global context)

ПолучитьBase64БуферДвоичныхДанныхИзБуфераДвоичныхДанных (GetBase64BinaryDataBufferFromBinaryDataBuffer)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьBase64БуферДвоичныхДанныхИзБуфераДвоичныхДанных(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер двоичных данных.

Возвращаемое значение:

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   

Описание:

Преобразует буфер двоичных данных в формат Base64.  
Полученный текст разбивается на строки длиной 64 символа. В качестве разделителя строк используется сочетание символов CR+LF.   
Если целевой декодер Base64 не поддерживает переводы строк, требуется дополнительно обработать строки, чтобы удалить из нее разделители.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
