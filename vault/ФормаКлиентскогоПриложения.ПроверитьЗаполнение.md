ФормаКлиентскогоПриложения (ClientApplicationForm)

ПроверитьЗаполнение (CheckFilling)

Доступен, начиная с версии 8.2.

Синтаксис:

ПроверитьЗаполнение()

Возвращаемое значение:

Тип: Булево.   
Истина - ошибок не обнаружено, Ложь - в противном случае.

Описание:

Проверяет заполнение реквизитов. Для реквизитов, у которых свойство "Проверка заполнения" установлено в значение "Показывать ошибку" и реквизит не заполнен, будет сформировано сообщение об ошибке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

Примечание:

Выполнение метода приводит к возникновению события [ОбработкаПроверкиЗаполненияНаСервере](ФормаКлиентскогоПриложения.ОбработкаПроверкиЗаполненияНаСервере.md).

Использование в версии:

Доступен, начиная с версии 8.2.
