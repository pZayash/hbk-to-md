МенеджерМетокNDEF (TagManagerNDEF)

ПоддерживаетсяФоновоеСканирование (BackgroundScanningSupported)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяФоновоеСканирование()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - устройство поддерживает фоновое сканирование и NFC включено,  
[Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет, что устройство поддерживает сканирование меток в фоне.  
Для устройств на Android дополнительно проверяется, что пользователь включил NFC.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
