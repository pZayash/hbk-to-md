Расширение декорации формы для надписи (Form decoration extension for a label)

Нажатие (Click)

Доступен, начиная с версии 8.2.

Синтаксис:

Нажатие()

Описание:

Вызывается при нажатии мышью или при помощи клавиатуры при установленном свойстве [Гиперссылка](Расширение_декорации_формы_для_надписи.Гиперссылка_(Form_decoration_extension_for_a_label.Hyperlink).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
