Расширение поля формы для поля ввода (Form field extension for a text box)

РастягиватьПоГоризонтали (HorizontalStretch)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Устанавливает возможность растягивания поля по горизонтали.   
Истина - растягивание разрешено.  
Значение Неопределено означает автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
