# ПланОбменаСсылка.<Имя плана обмена>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаСсылка.<Имя плана обмена>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md) › [Методы](ПланОбменаСсылка.Имя_плана_обмена__Методы.md)

ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланОбмена](ОбъектМетаданных_ПланОбмена_(MetadataObject_ExchangePlan).md).   

Описание:

Предоставляет доступ к объекту описания метаданных плана обмена. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.ПланыОбменов.Офисы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
