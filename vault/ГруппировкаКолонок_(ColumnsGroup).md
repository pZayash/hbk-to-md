Описание:

Содержит варианты группировки колонок в группе вида [ГруппаКолонок](ВидГруппыФормы.ГруппаКолонок_(FormGroupType.ColumnGroup).md) формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Расширение группы формы для группы колонок](Расширение_группы_формы_для_группы_колонок_(Form_group_extension_for_a_group_of_columns).md), свойство [Группировка](Расширение_группы_формы_для_группы_колонок.Группировка_(Form_group_extension_for_a_group_of_columns.Group).md)

Доступен, начиная с версии 8.2.
