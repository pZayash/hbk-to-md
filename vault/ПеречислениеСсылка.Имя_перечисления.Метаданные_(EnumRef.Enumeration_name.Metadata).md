# ПеречислениеСсылка.<Имя перечисления>.Метаданные

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Перечисления](Перечисления.md) › [ПеречислениеСсылка.<Имя перечисления>](ПеречислениеСсылка.Имя_перечисления_(EnumRef.Enumeration_name).md) › [Методы](ПеречислениеСсылка.Имя_перечисления__Методы.md)

ПеречислениеСсылка.<Имя перечисления> (EnumRef.<Enumeration name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Перечисление](ОбъектМетаданных_Перечисление_(MetadataObject_Enum).md).   

Описание:

Предоставляет доступ к объекту описания метаданных перечисления. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.Перечисления.ВидыКонтрагентов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
