РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

ИспользоватьНастройкиТекущегоСеанса (UseCurrentSessionSettings)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак использования региональных настроек текущего сеанса при получении представления данных типа Число, Дата, Булево:

- Истина - используются региональные настройки сеанса, выполняющего получение представления,
- Ложь - используются значения других свойств этого объекта, полученного методом [ПолучитьРегиональныеНастройкиИнформационнойБазы](Глобальный_контекст.ПолучитьРегиональныеНастройкиИнформационнойБазы.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
