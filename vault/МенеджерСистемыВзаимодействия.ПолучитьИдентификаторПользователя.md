МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьИдентификаторПользователя (GetUserID)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьИдентификаторПользователя(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Идентификатор пользователя информационной базы.

Возвращаемое значение:

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Идентификатор пользователя системы взаимодействия.

Описание:

Возвращает идентификатор того пользователя системы взаимодействия, который соответствует пользователю информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
