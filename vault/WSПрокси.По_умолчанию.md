WSПрокси (WSProxy)

На основании определения веб-сервиса

Доступен, начиная с версии 8.1.

Синтаксис:

Новый WSПрокси(<Определение>, <URIПространстваИменСервиса>, <ИмяСервиса>, <ИмяТочкиПодключения>, <ИнтернетПрокси>, <Таймаут>, <ЗащищенноеСоединение>, <Местоположение>, <ИспользоватьАутентификациюОС>)

Параметры:

<Определение> (обязательный)

Тип: [WSОпределения](WSОпределения_(WSDefinitions).md).   
Определение веб-сервисов, на основе которого создается данный прокси.

<URIПространстваИменСервиса> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен веб-сервиса, с которым связывается данный прокси.

<ИмяСервиса> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя веб-сервиса, с которым связывается данный прокси.

<ИмяТочкиПодключения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя точки подключения веб-сервиса, с которой связывается данный прокси.

<ИнтернетПрокси> (необязательный)

Тип: [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md).   
Содержит объект [ИнтернетПрокси](ИнтернетПрокси_(InternetProxy).md) для осуществления HTTP-соединения с веб-сервисом объектом [WSПрокси](WSПрокси_(WSProxy).md),  
Если не указан, то используются настройки прокси по умолчанию.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Таймаут> (необязательный)

Тип: [Число](lang__def_Number.md).   
Таймаут соединения и операций, осуществляемых объектом WSПрокси, в секундах. 0 - не устанавливать таймаут.  
Значение по умолчанию: 0.

<ЗащищенноеСоединение> (необязательный)

Тип: [ЗащищенноеСоединениеOpenSSL](ЗащищенноеСоединениеOpenSSL_(OpenSSLSecureConnection).md), [Неопределено](lang__def_Undefined.md).   
Объект защищенного соединения для осуществления HTTPS соединения с веб-сервисом.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Местоположение> (необязательный)

Тип: [Строка](lang__def_String.md).   
Позволяет указать фактический адрес сервиса.  
Если параметр не задан или задана пустая строка, то адрес сервиса выбирается из [Местоположение](WSТочкаПодключения.Местоположение_(WSEndpoint.Location).md).  
Адрес указывается в виде строки, которая автоматически кодируется под требования стандарта RFC 3986, с учетом особенностей:

- не кодируются разделители сегментов пути,
- если адрес ресурса содержит символ "%", то считается, что после него указан код закодированного символа и повторно он не кодируется.

Недопустимые символы, не являющиеся специальными символами URI, будут автоматически кодированы.  
Примеры:  
Обращение к ресурсу по адресу "/example%segment":  

|  |
| --- |
| /example%25segment |

  
Обращение к ресурсу "/example#segment", содержащему символ "#", в качестве сегмента, а не разделителя сегмента пути:  

|  |
| --- |
| /example%23segment |

  
Обращение к ресурсу "/пример"("/%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80"):  

|  |
| --- |
| "/пример" |

  
или  

|  |
| --- |
| "/%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80" |

  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<ИспользоватьАутентификациюОС> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Включает использование аутентификации NTLM или Negotiate на сервере.  
Значение по умолчанию: Ложь.

Описание:

Создает прокси на основе определения веб-сервиса и связывает его с точкой подключения веб-сервиса.

Использование в версии:

Доступен, начиная с версии 8.1.
