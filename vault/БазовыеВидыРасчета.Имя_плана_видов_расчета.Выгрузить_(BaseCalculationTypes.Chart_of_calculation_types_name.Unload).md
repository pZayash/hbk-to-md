БазовыеВидыРасчета.<Имя плана видов расчета> (BaseCalculationTypes.<Chart of calculation types name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.0.

Синтаксис:

Выгрузить()

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   

Описание:

Создает таблицу значений и копирует в нее табличную часть. Имена колонок таблицы значений совпадают с именами колонок табличной части. Колонка, содержащая номер строки, выгружается в колонку с именем "НомерСтроки".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
