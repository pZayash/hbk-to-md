СерияДиаграммыГанта (GanttChartSeries)

ШтриховкаПерекрывающихсяИнтервалов (OverlappedIntervalsHatch)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Истина](lang__def_BooleanTrue.md), то перекрывающиеся интервалы выводятся штриховкой, а не сплошным цветом. Цвет выбирается из свойства [ДополнительныйЦвет](СерияДиаграммыГанта.ДополнительныйЦвет_(GanttChartSeries.SecondColor).md) с учётом свойства [ПриоритетЦвета](СерияДиаграммыГанта.ПриоритетЦвета_(GanttChartSeries.ColorPriority).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.0.
