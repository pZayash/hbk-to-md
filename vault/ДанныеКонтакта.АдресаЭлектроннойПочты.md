ДанныеКонтакта (ContactData)

АдресаЭлектроннойПочты (Emails)

Доступен, начиная с версии 8.3.4.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Содержит массив адресов электронной почты в виде объектов типа [ЭлементДанныхКонтакта](ЭлементДанныхКонтакта.md). Свойство [ТипДанных](ЭлементДанныхКонтакта.ТипДанных.md) содержит значение перечисления [ТипАдресаЭлектроннойПочтыДанныхКонтакта](ТипАдресаЭлектроннойПочтыДанныхКонтакта.md) или строку. Свойство [Значение](ЭлементДанныхКонтакта.Значение.md) содержит строку – адрес электронной почты.

Доступность:

Мобильный клиент, сервер, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.4.
