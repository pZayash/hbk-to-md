СтатусРазбораИнтернетПочтовогоСообщения (InternetMailMessageParseStatus)

ОбнаруженыОшибки (ErrorsDetected)

Доступен, начиная с версии 8.3.6.

Описание:

При разборе почтового сообщения обнаружены ошибки. Некоторые поля сообщения могут быть не заполнены.

Использование в версии:

Доступен, начиная с версии 8.3.6.
