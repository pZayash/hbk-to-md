ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Вставить (Insert)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс добавляемого информационного интервала.

Возвращаемое значение:

Тип: [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы_(ChartReferenceBand).md).   
Созданный информационный интервал.

Описание:

Добавляет новый элемент коллекции по указаному индексу.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
