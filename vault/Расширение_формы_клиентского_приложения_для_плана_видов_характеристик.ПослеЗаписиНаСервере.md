Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [ПланВидовХарактеристикОбъект.\<Имя плана видов характеристик\>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик.md).   
Записанный объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура.md).   
Структура, содержащая параметры записи.

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
