КонструкторФорматнойСтроки (FormatStringWizard)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания: [Текст](КонструкторФорматнойСтроки.Текст.md) - если конструктор был закрыт нажатием кнопки "OK", Неопределено - в противном случае.

Описание:

Асинхронная функция. Открывает конструктор форматной строки немодально.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[КонструкторФорматнойСтроки](КонструкторФорматнойСтроки.md), метод [ОткрытьМодально](КонструкторФорматнойСтроки.ОткрытьМодально.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
