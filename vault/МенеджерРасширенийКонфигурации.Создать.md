МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

Создать (Create)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Создать(<Данные>)

Параметры:

<Данные> (необязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Данные файла формата \*.cfe.

Возвращаемое значение:

Тип: [РасширениеКонфигурации](РасширениеКонфигурации.md).   

Описание:

Создает новый объект типа [РасширениеКонфигурации](РасширениеКонфигурации.md).  
Для объектов, созданных с указанием параметра Данные:

- свойства [Версия](РасширениеКонфигурации.Версия.md), [Имя](РасширениеКонфигурации.Имя.md), [Назначение](РасширениеКонфигурации.Назначение.md), [ОсновныеРоли](РасширениеКонфигурации.ОсновныеРоли.md), [Синоним](РасширениеКонфигурации.Синоним.md) и [ХешСумма](РасширениеКонфигурации.ХешСумма.md) имеют такое же значение, какое будет у расширения конфигурации, установленного из файла с идентичными двоичными данными,
- метод [ИзменяетСтруктуруДанных](РасширениеКонфигурации.ИзменяетСтруктуруДанных.md) возвращает Истина, если после установки расширение будет влиять на структуру данных конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
