# НастройкаОтбора

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Отбор](Отбор.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы настройки отбора.  
Возможно обращение к элементу посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Добавить (Add)](НастройкаОтбора.Добавить_(FilterSettings.Add).md)  
[Индекс (IndexOf)](НастройкаОтбора.Индекс_(FilterSettings.IndexOf).md)  
[Количество (Count)](НастройкаОтбора.Количество_(FilterSettings.Count).md)  
[Найти (Find)](НастройкаОтбора.Найти_(FilterSettings.Find).md)  
[Очистить (Clear)](НастройкаОтбора.Очистить_(FilterSettings.Clear).md)  
[Получить (Get)](НастройкаОтбора.Получить_(FilterSettings.Get).md)  
[Удалить (Delete)](НастройкаОтбора.Удалить_(FilterSettings.Delete).md)  

Описание:

Набор элементов управления отбором.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[Расширение табличного поля отбора](Расширение_табличного_поля_отбора_(Filter_table_box_extension).md), свойство [НастройкаОтбора](Расширение_табличного_поля_отбора.НастройкаОтбора_(Filter_table_box_extension.FilterSettings).md)
