СтандартноеХранилищеНастроекМенеджер (StandardSettingsStorageManager)

ЗагрузитьНастройкуПоУмолчанию (LoadDefaultSetting)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ЗагрузитьНастройкуПоУмолчанию(<КлючОбъекта>)

Параметры:

<КлючОбъекта> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ объекта настройки.

Возвращаемое значение:

Тип: Произвольный, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md), если настройка по умолчанию не найдена.

Описание:

Загружает настройку по умолчанию.  
Вызов метода допускается только для хранилища системных настроек (ХранилищеСистемныхНастроек).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
