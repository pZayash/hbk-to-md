РезультатВыбораДействияРасшифровкиКомпоновкиДанных (DataCompositionDetailsActionChoiceResult)

ПараметрВыбранногоДействия (ChosenActionParameter)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Параметр выбранного действия.  
Если действие "Открыть", то свойство содержит значение, которое нужно открыть.  
Если действие "Расшифровать", "Открыть", "Отфильтровать", "Упорядочить", "Сгруппировать", "Оформить" то свойство содержит настройки, пригодные для использования в методе [ПрименитьНастройки](ОбработкаРасшифровкиКомпоновкиДанных.ПрименитьНастройки_(DataCompositionDetailsProcess.ApplySettings).md).  
Если выбрано дополнительное действие , то свойство не заполняется.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ОбработкаРасшифровкиКомпоновкиДанных](ОбработкаРасшифровкиКомпоновкиДанных_(DataCompositionDetailsProcess).md), метод [ПоказатьВыборДействия](ОбработкаРасшифровкиКомпоновкиДанных.ПоказатьВыборДействия_(DataCompositionDetailsProcess.ShowActionChoice).md)

Доступен, начиная с версии 8.3.18.
