ЭлементБуфераОбмена (ClipboardItem)

ФорматДанных (DataFormat)

Доступен, начиная с версии 8.3.24.

Использование:

Только чтение.

Описание:

Тип: [СтандартныйФорматДанныхБуфераОбмена](СтандартныйФорматДанныхБуфераОбмена_(ClipboardDataStandardFormat).md), [Строка](lang__def_String.md).   
Формат данных элемента буфера обмена.  
Значение параметра типа [Строка](lang__def_String.md) используется для задания произвольного формата данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.24.
