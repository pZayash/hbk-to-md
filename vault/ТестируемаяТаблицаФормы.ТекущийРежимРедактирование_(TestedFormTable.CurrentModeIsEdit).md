ТестируемаяТаблицаФормы (TestedFormTable)

ТекущийРежимРедактирование (CurrentModeIsEdit)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущийРежимРедактирование()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - таблица редактируется; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Определяет, редактируется ли таблица.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
