ЗаписьТекста (TextWriter)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЗаписьТекста()

Описание:

Формирует неинициализированный объект [ЗаписьТекста](ЗаписьТекста.md). Для инициализации объекта используется метод [Открыть](ЗаписьТекста.Открыть.md).

Пример:

|  |
| --- |
| Текст = Новый ЗаписьТекста; |

Использование в версии:

Доступен, начиная с версии 8.0.
