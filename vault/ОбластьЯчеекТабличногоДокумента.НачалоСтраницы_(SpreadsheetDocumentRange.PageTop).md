ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

НачалоСтраницы (PageTop)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак вывода области со следующей страницы.  
[Истина](lang__def_BooleanTrue.md) - выводить со следующей страницы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Применимо только для областей из строк или колонок.

Использование в версии:

Доступен, начиная с версии 8.2.
