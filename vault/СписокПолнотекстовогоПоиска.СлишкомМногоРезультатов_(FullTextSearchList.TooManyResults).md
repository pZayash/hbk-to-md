СписокПолнотекстовогоПоиска (FullTextSearchList)

СлишкомМногоРезультатов (TooManyResults)

Доступен, начиная с версии 8.1.

Синтаксис:

СлишкомМногоРезультатов()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - во время поиска по соображениям производительности было произведено усечение количества результатов, что может сказаться на точности поиска (не все объекты будут найдены).

Описание:

Осуществляет проверку числа результатов поиска.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
