АдминистрированиеКластер (AdministrationCluster)

ПолучитьРабочиеСерверы (GetWorkServers)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьРабочиеСерверы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеРабочийСервер](АдминистрированиеРабочийСервер_(AdministrationWorkServer).md).

Описание:

Получает массив рабочих серверов, зарегистрированных в кластере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
