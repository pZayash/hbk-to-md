АдминистрированиеСервера (ServerAdministration)

ПолучитьКластеры (GetClusters)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьКластеры()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеКластер](АдминистрированиеКластер_(AdministrationCluster).md).

Описание:

Получает список кластеров, зарегистрированных в центральном сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
