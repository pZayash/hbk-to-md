ДокументDOM (DOMDocument)

Полный

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ДокументDOM(<URIПространстваИмен>, <Имя>, <ОпределениеТипаДокумента>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен корневого элемента документа.

<Имя> (обязательный)

Тип: Строка.   
Квалифицированное имя корневого элемента документа DOM.

<ОпределениеТипаДокумента> (обязательный)

Тип: [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md).   
Определение типа создаваемого документа.

Описание:

Создает документ DOM с корневым элементом и описанием типа документа.

Использование в версии:

Доступен, начиная с версии 8.1.
