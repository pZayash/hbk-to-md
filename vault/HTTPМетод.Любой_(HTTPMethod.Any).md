Любой (Any)

Доступен, начиная с версии 8.3.5.

Указывает, что может быть использован любой HTTP-метод из перечисления [HTTPМетод](HTTPМетод_(HTTPMethod).md).

Использование в версии:

Доступен, начиная с версии 8.3.5.
