# МенеджерОбработкиОшибок.ПредставлениеОшибкиДляПользователя

МенеджерОбработкиОшибок (ErrorProcessingManager)

ПредставлениеОшибкиДляПользователя (ErrorDescriptionForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПредставлениеОшибкиДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md).   
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   

Описание:

Возвращает текст ошибки, понятный пользователю.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
