ИдентификаторШаблонаСообщенияСистемыВзаимодействия (CollaborationSystemMessageTemplateID)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ИдентификаторШаблонаСообщенияСистемыВзаимодействия(<Строка>)

Параметры:

<Строка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строковое представление идентификатора.

Описание:

Создает идентификатор шаблона сообщения из указанной строки.  
Корректность идентификатора не проверяется.

Использование в версии:

Доступен, начиная с версии 8.3.23.
