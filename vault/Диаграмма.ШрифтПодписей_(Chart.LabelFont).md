Диаграмма (Chart)

ШрифтПодписей (LabelFont)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт_(Font).md).   
Содержит шрифт подписей. Если указан шрифт, вид которого [АвтоШрифт](ВидШрифта.АвтоШрифт_(FontType.AutoFont).md), то используется шрифт области построения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
