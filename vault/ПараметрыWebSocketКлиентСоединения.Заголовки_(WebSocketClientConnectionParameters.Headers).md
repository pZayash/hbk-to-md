ПараметрыWebSocketКлиентСоединения (WebSocketClientConnectionParameters)

Заголовки (Headers)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Заголовки, которые платформа добавит к заголовкам HTTP-запроса GET (с заголовком Upgrade: websocket). Этим запросом начинается попытка установки соединения с WebSocket-сервером.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
