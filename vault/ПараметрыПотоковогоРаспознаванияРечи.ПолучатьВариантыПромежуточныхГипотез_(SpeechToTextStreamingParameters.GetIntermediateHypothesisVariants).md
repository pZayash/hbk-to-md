ПараметрыПотоковогоРаспознаванияРечи (SpeechToTextStreamingParameters)

ПолучатьВариантыПромежуточныхГипотез (GetIntermediateHypothesisVariants)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак получения вариантов промежуточных гипотез для фраз.  
Если установлено [Истина](lang__def_BooleanTrue.md), то в коллекции будет гораздо больше вариантов результата потокового распознания.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
