ПараметрыЗаписиJSON (JSONWriterSettings)

ПереносСтрок (NewLines)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ПереносСтрокJSON](ПереносСтрокJSON.md).   
Управляет размещением начала и конца объектов и массивов, ключей и значений на новой строке.  
Также установка данного свойства в значение, отличное от [Нет](ПереносСтрокJSON.Нет.md), добавляет пробел между именем свойства, двоеточием и значением.  
После создания объекта данное свойство имеет значение [Авто](ПереносСтрокJSON.Авто.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
