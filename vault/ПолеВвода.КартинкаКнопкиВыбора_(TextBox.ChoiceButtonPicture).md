ПолеВвода (TextBox)

КартинкаКнопкиВыбора (ChoiceButtonPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка_(Picture).md).   
Содержит картинку, отображаемую в кнопке выбора.

Доступность:

Толстый клиент.

Примечание:

Оптимальный размер картинки определяется размером элемента управления и размером его шрифта.  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая_(PictureType.Empty).md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки_(PictureType.FromLib).md).

Использование в версии:

Доступен, начиная с версии 8.0.
