ДанныеФормыКоллекция (FormDataCollection)

Выгрузить (Unload)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: Выгрузить колонки

Синтаксис:

Выгрузить(<Строки>, <Колонки>)

Параметры:

<Строки> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив строк для выгрузки. Если не указан, выгружаются все строки коллекции.

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Строка, задающая список колонок для копирования в формате: "Колонка1, Колонка2...". Если список не задан, то будут скопированы все колонки.

Вариант синтаксиса: Выгрузить по отбору

Синтаксис:

Выгрузить(<ПараметрыОтбора>, <Колонки>)

Параметры:

<ПараметрыОтбора> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Ключ структуры - идентификатор колонки, а значение структуры - значение отбора.

<Колонки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Список колонок для копирования в формате: "Колонка1, Колонка2,...,КолонкаN".

Описание варианта метода:

Если указан отбор, то только строки из отбора будут выгружены.   
Если отбор не указан, то будут выгружены все строки коллекции.   
Если указаны колонки, то только эти колонки будут выгружены.   
Иначе, будут выгружены все колонки коллекции.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   

Описание:

Создает таблицу значений и копирует в нее данные коллекции. Имена колонок таблицы значений совпадают с именами колонок коллекции. Колонка, содержащая номер строки, выгружается в колонку с именем [НомерСтроки](Строка_табличной_части.НомерСтроки_(Line_of_a_tabular_section.LineNumber).md).

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
