ЗаписьHTML (HTMLWriter)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОткрытьПоток(<Поток>, <ТипКодировки>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи.

<ТипКодировки> (необязательный)

Тип: Строка.   
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае поток будет записан в соответствующей кодировке. При этом в объявлении HTML-документа (если таковое будет записано) будет присутствовать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то для записи файла будет использована кодировка [UTF8](КодировкаТекста.UTF8.md), а атрибут 'encoding' не будет присутствовать в объявлении HTML-документа.

<ЗаписатьBOM> (необязательный)

Тип: Булево.   
Определяет, будет ли добавлена метка порядка байт (BOM) в начало потока.  
Значение по умолчанию: Ложь.

Описание:

Устанавливает поток для записи HTML-документа. Позволяет указать тип кодировки, который будет использован для записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
