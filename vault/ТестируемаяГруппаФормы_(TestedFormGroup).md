Описание:

Представляет абстракцию элемента управления [ГруппаФормы](ГруппаФормы_(FormGroup).md). Доступная функциональность зависит от вида тестируемого элемента управления, который определяется свойством [Вид](ТестируемаяГруппаФормы.Вид_(TestedFormGroup.Type).md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
