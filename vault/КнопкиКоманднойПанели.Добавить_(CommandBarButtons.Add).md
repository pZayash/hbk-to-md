КнопкиКоманднойПанели (CommandBarButtons)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <ТипКнопки>, <Текст>, <Действие>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя кнопки - должно соответствовать правилам имен. Уникально в пределах командной панели.

<ТипКнопки> (необязательный)

Тип: [ТипКнопкиКоманднойПанели](ТипКнопкиКоманднойПанели_(CommandBarButtonType).md).   
Тип кнопки интерфейса (разделитель, подменю, действие).  
Значение по умолчанию: [Разделитель](Разделитель_(Splitter).md).

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст кнопки.

<Действие> (необязательный)

Тип: [Действие](Действие_(Action).md).   
Действие, которое необходимо выполнить при нажатии на данную кнопку. Для типов кнопок [Разделитель](ТипКнопкиКоманднойПанели.Разделитель_(CommandBarButtonType.Separator).md) и [Подменю](ТипКнопкиКоманднойПанели.Подменю_(CommandBarButtonType.Popup).md), действие указывать не нужно.

Возвращаемое значение:

Тип: [КнопкаКоманднойПанели](КнопкаКоманднойПанели_(CommandBarButton).md).   

Описание:

Добавляет кнопку в конец коллекции кнопок командной панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| КП = ЭлементыФормы.КоманднаяПанель; НовоеДействие = Новый Действие("МояПроцедураВМодулеФормы"); ТипКнопки = ТипКнопкиКоманднойПанели.Действие; НоваяКнопка = КП.Кнопки.Добавить("Действие1", ТипКнопки, , НовоеДействие); |

Использование в версии:

Доступен, начиная с версии 8.0.
