ЗначенияПараметровДанныхКомпоновкиДанных (DataCompositionDataParameterValues)

УстановитьЗначениеПараметра (SetParameterValue)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьЗначениеПараметра(<Параметр>, <Значение>)

Параметры:

<Параметр> (обязательный)

Тип: [Строка](lang__def_String.md), [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных_(DataCompositionParameter).md).   
Имя параметра или параметр компоновки данных, значение которого нужно установить.

<Значение> (обязательный)

Значение, которое нужно установить.

Описание:

Устанавливает значение параметра и включает свойство [Использование](ЗначениеПараметраКомпоновкиДанных.Использование_(DataCompositionParameterValue.Use).md). Если параметр с указанным именем не найден, будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
