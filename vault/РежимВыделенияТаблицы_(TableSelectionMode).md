# РежимВыделенияТаблицы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[Множественный (MultiRow)](РежимВыделенияТаблицы.Множественный_(TableSelectionMode.MultiRow).md)  
[Одиночный (SingleRow)](РежимВыделенияТаблицы.Одиночный_(TableSelectionMode.SingleRow).md)  

Описание:

Содержит варианты выделения строк таблицы формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТаблицаФормы](ТаблицаФормы_(FormTable).md), свойство [РежимВыделения](ТаблицаФормы.РежимВыделения_(FormTable.SelectionMode).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
