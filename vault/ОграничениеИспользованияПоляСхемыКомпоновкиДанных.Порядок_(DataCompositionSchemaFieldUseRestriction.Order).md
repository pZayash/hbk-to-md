Порядок (Order)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли доступно поле для упорядочивания.  
[Истина](lang__def_BooleanTrue.md) - упорядочивание недоступно.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
