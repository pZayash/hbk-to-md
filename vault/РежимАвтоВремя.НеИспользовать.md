РежимАвтоВремя (AutoTimeMode)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.0.

Описание:

Не использовать установку времени объекта (документа, бизнес-процесса, задачи). Это означает, что если в дате объекта время не нулевое, оно будет сброшено (время - 0:00:00).

Использование в версии:

Доступен, начиная с версии 8.0.
