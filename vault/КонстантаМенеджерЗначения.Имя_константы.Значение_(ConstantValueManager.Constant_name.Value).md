# КонстантаМенеджерЗначения.<Имя константы>.Значение

КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

Значение (Value)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Содержит значение константы. Чтение и запись производится методами [Прочитать](КонстантаМенеджерЗначения.Имя_константы.Прочитать_(ConstantValueManager.Constant_name.Read).md) и [Записать](КонстантаМенеджерЗначения.Имя_константы.Записать_(ConstantValueManager.Constant_name.Write).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
