РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Картинка (Picture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Картинка](Картинка.md).   
Содержит картинку рисунка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунков типа [Картинка](ТипРисункаТабличногоДокумента.Картинка.md).  
Если картинка задается программно, может быть установлена картина вида [Пустая](ВидКартинки.Пустая.md) или [ИзБиблиотеки](ВидКартинки.ИзБиблиотеки.md).

Использование в версии:

Доступен, начиная с версии 8.0.
