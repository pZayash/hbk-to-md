Соединение с рабочим процессом (IWorkingProcessConnection)

GetInfoBaseConnections (GetInfoBaseConnections)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBaseConnections(<ИнформационнаяБаза>)

Параметры:

<ИнформационнаяБаза> (обязательный)

Тип: [Информационная база](Информационная_база.md).   
Информационная база, для которой должен быть получен список соединений.

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Одномерный массив, содержащий объекты [Соединение](Соединение.md).

Описание:

Получает массив соединений информационной базы. Для выполнения требуются административные права на информационную базу.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
