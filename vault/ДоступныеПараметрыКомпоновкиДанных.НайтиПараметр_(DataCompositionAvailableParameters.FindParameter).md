ДоступныеПараметрыКомпоновкиДанных (DataCompositionAvailableParameters)

НайтиПараметр (FindParameter)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПараметр(<Параметр>)

Параметры:

<Параметр> (обязательный)

Тип: [ПараметрКомпоновкиДанных](ПараметрКомпоновкиДанных_(DataCompositionParameter).md).   
Параметр.

Возвращаемое значение:

Тип: [ДоступныйПараметрКомпоновкиДанных](ДоступныйПараметрКомпоновкиДанных_(DataCompositionAvailableParameter).md), [Неопределено](lang__def_Undefined.md).   
Если параметр не найден, возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск доступного параметра.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
