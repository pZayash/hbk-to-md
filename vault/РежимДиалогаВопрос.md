Описание:

Определяет набор вариантов состава кнопок окна диалога вопроса. Используется в качестве параметра функции [Вопрос](Глобальный_контекст.Вопрос.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Вопрос](Глобальный_контекст.Вопрос.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
