Рабочий процесс (IWorkingProcessInfo)

AvgCallTime (AvgCallTime)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Число.   
Показывает среднее время обслуживания рабочим процессом одного клиентского обращения. Оно складывается из: значений свойств [AvgServerCallTime](Рабочий_процесс.AvgServerCallTime.md), [AvgDBCallTime](Рабочий_процесс.AvgDBCallTime.md), [AvgLockCallTime](Рабочий_процесс.AvgLockCallTime.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
