ТестируемаяФорма (TestedForm)

ТекущаяМодифицированность (CurrentModified)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ТекущаяМодифицированность()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект модифицирован; [Ложь](lang__def_BooleanFalse.md) в противном случае.

Описание:

Возвращает текущее значение признака модифицированности для тестируемой формы

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
