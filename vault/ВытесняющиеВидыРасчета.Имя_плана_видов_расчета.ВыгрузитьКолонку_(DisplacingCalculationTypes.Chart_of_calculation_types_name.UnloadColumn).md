ВытесняющиеВидыРасчета.<Имя плана видов расчета> (DisplacingCalculationTypes.<Chart of calculation types name>)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Колонка, из которой нужно выгрузить значения. В качестве значения параметра может быть передан индекс колонки или имя колонки.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Выгружает значения, содержащиеся в колонке табличной части, в массив.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения из колонки выгружаются в порядке индексов строк в табличной части.

Использование в версии:

Доступен, начиная с версии 8.0.
