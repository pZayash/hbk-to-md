# ПользовательИнформационнойБазы.ПарольНеСоответствуетТребованиям

ПользовательИнформационнойБазы (InfoBaseUser)

ПарольНеСоответствуетТребованиям (PasswordDoesNotSatisfyRequirements)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, соответствует ли пароль пользователя требованиям:

- Если выявлен несоответствующий пароль при аутентификации, устанавливается [Истина](lang__def_BooleanTrue.md);
- Если несоответствие не выявлено, устанавливается [Ложь](lang__def_BooleanFalse.md);
- Если меняется пароль пользователя, свойство сбрасывается (устанавливается [Ложь](lang__def_BooleanFalse.md));
- Значение по умолчанию - [Ложь](lang__def_BooleanFalse.md).

Свойство недоступно для записи, доступно для чтения только пользователю с административными правами.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
