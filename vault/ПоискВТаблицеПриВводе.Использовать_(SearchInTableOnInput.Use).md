ПоискВТаблицеПриВводе (SearchInTableOnInput)

Использовать (Use)

Доступен, начиная с версии 8.2.

Описание:

При вводе символов в таблицу, связанную с данными типа [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md) выполняется поиск в таблице.

Использование в версии:

Доступен, начиная с версии 8.2.
