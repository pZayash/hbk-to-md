Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md) для полей настройки.  
Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [ПолеНастройки](ПолеНастройки.md).
