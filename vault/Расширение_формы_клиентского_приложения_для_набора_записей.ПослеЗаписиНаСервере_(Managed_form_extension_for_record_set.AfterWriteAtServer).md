ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [РегистрСведенийНаборЗаписей.](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийНаборЗаписей.Имя_регистра_сведений_(InformationRegisterRecordSet.Information_register_name).md)>, [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления_(AccumulationRegisterRecordSet.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления_(AccumulationRegisterRecordSet.Accumulation_register_name).md)>, [РегистрБухгалтерииНаборЗаписей.](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)<[Имя регистра бухгалтерии](РегистрБухгалтерииНаборЗаписей.Имя_регистра_бухгалтерии_(AccountingRegisterRecordSet.Accounting_register_name).md)>, [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета_(CalculationRegisterRecordSet.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета_(CalculationRegisterRecordSet.Calculation_register_name).md)>.   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи.

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
