Форма (Form)

ЗакрыватьПриЗакрытииВладельца (CloseOnOwnerClose)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что форма должна закрываться при закрытии формы-владельца.  
[Ложь](lang__def_BooleanFalse.md) - остается открытой.

Доступность:

Толстый клиент.

См. также:

[Форма](Форма_(Form).md), свойство [ВладелецФормы](Форма.ВладелецФормы_(Form.FormOwner).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
