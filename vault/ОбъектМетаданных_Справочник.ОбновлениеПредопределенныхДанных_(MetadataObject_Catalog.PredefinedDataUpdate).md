ОбъектМетаданных: Справочник (MetadataObject: Catalog)

ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ОбновлениеПредопределенныхДанных](ОбновлениеПредопределенныхДанных_(PredefinedDataUpdate).md).   
Содержит режим обновления предопределенных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для режимов [Авто](ОбновлениеПредопределенныхДанных.Авто_(PredefinedDataUpdate.Auto).md) и [ОбновлятьАвтоматически](ОбновлениеПредопределенныхДанных.ОбновлятьАвтоматически_(PredefinedDataUpdate.AutoUpdate).md) дополнительно учитывается флаг, установленный в базе данных с помощью метода [УстановитьОбновлениеПредопределенныхДанных](СправочникМенеджер.Имя_справочника.УстановитьОбновлениеПредопределенныхДанных_(CatalogManager.Catalog_name.SetPredefinedDataUpdate).md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
