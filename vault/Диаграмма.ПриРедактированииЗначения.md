Диаграмма (Chart)

ПриРедактированииЗначения (OnValueEdit)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПриРедактированииЗначения(<ЗначениеДиаграммы>, <СостояниеРедактирования>)

Параметры:

<ЗначениеДиаграммы>

Тип: [ЗначениеДиаграммы](ЗначениеДиаграммы.md).   
Содержит измененное значение (с новым числовым значением).

<СостояниеРедактирования>

Тип: [СостояниеРедактированияЗначенияДиаграммы](СостояниеРедактированияЗначенияДиаграммы.md).   
Состояние редактирования значения на момент вызова обработчика.

Описание:

Вызывается при интерактивном изменении значений диаграммы.   
В параметр <ЗначениеДиаграммы> устанавливается измененное значение (с новым числовым значением). Параметр <СостояниеРедактирования> показывает текущее состояние редактирования на момент вызова обработчика.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
