ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество измерений в коллекции измерений построителя отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| // ПостроительОтчетов - реквизит отчета типа "ПостроительОтчетов"  ДоступныеИзмерения = Новый Соответствие; Если Не ИспользоватьМакет Тогда     Для Индекс=0 По ПостроительОтчетов.ДоступныеПоля.Количество()-1 Цикл         Поле = ПостроительОтчетов.ДоступныеПоля[Индекс];         Если Поле.Измерение Тогда             ДоступныеИзмерения.Вставить(Поле.Имя, Поле.Представление);         КонецЕсли;     КонецЦикла;      Для Индекс=0 По ПостроительОтчетов.ИзмеренияСтроки.Количество()-1 Цикл         Измерение = ПостроительОтчетов.ИзмеренияСтроки[Индекс];         ДоступныеИзмерения.Удалить(Измерение.Имя);     КонецЦикла;      Для Индекс=0 По ПостроительОтчетов.ИзмеренияКолонки.Количество()-1 Цикл         Измерение = ПостроительОтчетов.ИзмеренияКолонки[Индекс];         ДоступныеИзмерения.Удалить(Измерение.Имя);     КонецЦикла; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
