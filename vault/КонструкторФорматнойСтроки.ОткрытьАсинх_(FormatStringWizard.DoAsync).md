# КонструкторФорматнойСтроки.ОткрытьАсинх

КонструкторФорматнойСтроки (FormatStringWizard)

ОткрытьАсинх (DoAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания: [Текст](КонструкторФорматнойСтроки.Текст_(FormatStringWizard.Text).md) - если конструктор был закрыт нажатием кнопки "OK", [Неопределено](lang__def_Undefined.md) - в противном случае.

Описание:

Асинхронная функция. Открывает конструктор форматной строки немодально.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[КонструкторФорматнойСтроки](КонструкторФорматнойСтроки_(FormatStringWizard).md), метод [ОткрытьМодально](КонструкторФорматнойСтроки.ОткрытьМодально_(FormatStringWizard.DoModal).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
