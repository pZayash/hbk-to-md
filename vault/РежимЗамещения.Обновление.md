РежимЗамещения (ReplacementMode)

Обновление (Update)

Доступен, начиная с версии 8.3.26.

Описание:

Данное значение означает, что должно быть произведено обновление существующих записей регистра, совпадающих по значениям ключевых полей с записями, содержащимися в наборе записей. Добавление новых записей не выполняется. При обновлении выполняется перезапись существующих записей регистра с использованием значений полей, содержащихся в записях набора записей.

Использование в версии:

Доступен, начиная с версии 8.3.26.
