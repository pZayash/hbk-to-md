БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ОбработкаИнтерактивнойАктивации (InteractiveActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаИнтерактивнойАктивации(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при выполнении метода [АктивироватьИнтерактивно](БизнесПроцессОбъект.Имя_бизнес-процесса.АктивироватьИнтерактивно_(BusinessProcessObject.Business_process_name.ActivateInteractively).md). Если стандартная обработка не изменена, то после этого события будет вызван обработчик события [ОбработкаИнтерактивнойАктивации](ЗадачаОбъект.Имя_задачи.ОбработкаИнтерактивнойАктивации_(TaskObject.Task_name.InteractiveActivationProcessing).md) для всех невыполненных задач по данному бизнес-процессу для текущего исполнителя. Текущий пользователь определяется параметром сеанса, который установлен в соответствующем свойстве задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) не вызывается.
