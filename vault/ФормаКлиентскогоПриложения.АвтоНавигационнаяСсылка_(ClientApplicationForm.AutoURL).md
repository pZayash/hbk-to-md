ФормаКлиентскогоПриложения (ClientApplicationForm)

АвтоНавигационнаяСсылка (AutoURL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет автоматическим формированием навигационной ссылки, возвращаемой формой.  
Если [Ложь](lang__def_BooleanFalse.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку_(ClientApplicationWindow.GetURL).md) объекта [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md) возвращает значение свойства НавигационнаяСсылка основной формы.  
Если [Истина](lang__def_BooleanTrue.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку_(ClientApplicationWindow.GetURL).md) возвращает навигационную ссылку основной формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку_(ClientApplicationWindow.GetURL).md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.
