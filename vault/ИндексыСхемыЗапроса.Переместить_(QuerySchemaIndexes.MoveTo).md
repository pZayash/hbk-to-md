ИндексыСхемыЗапроса (QuerySchemaIndexes)

Переместить (MoveTo)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Переместить(<Элемент>, <НовыйИндекс>)

Параметры:

<Элемент> (необязательный)

Тип: [Число](lang__def_Number.md), [ИндексСхемыЗапроса](ИндексСхемыЗапроса_(QuerySchemaIndex).md).   
Индекс элемента или сам элемент, который требуется переместить.

<НовыйИндекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс, в который нужно переместить указанный элемент.

Описание:

Перемещает элемент в коллекции в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
