# РежимИспользованияПараметраКоманды

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Интерфейсные](Интерфейсные.md)

Доступен, начиная с версии 8.2.

Значения

[Множественный (Multiple)](РежимИспользованияПараметраКоманды.Множественный_(CommandParameterUseMode.Multiple).md)  
[Одиночный (Single)](РежимИспользованияПараметраКоманды.Одиночный_(CommandParameterUseMode.Single).md)  

Описание:

Содержит варианты режимов использования параметра команды.  
Если свойствах команды [РежимИспользованияПараметра](ОбъектМетаданных_Команда.РежимИспользованияПараметра_(MetadataObject_Command.ParameterUsageMode).md) и [РежимИспользованияПараметра](ОбъектМетаданных_ОбщаяКоманда.РежимИспользованияПараметра_(MetadataObject_CommonCommand.ParameterUsageMode).md) имеет значение [Множественный](РежимИспользованияПараметраКоманды.Множественный_(CommandParameterUseMode.Multiple).md), то в обработчик команды передается массив значений, а не одно значение даже, если в конкретном случае значение одно.  
Массив значений может быть передан в том случае, если источником для параметра команды является динамический список в режиме множественного выделения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
