Расширение декорации формы для картинки (Form decoration extension for a picture)

РазрешитьПеретаскивание (EnableDrag)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Включает режим перетаскивания у приёмника.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - перетаскивание разрешено; [Ложь](lang__def_BooleanFalse.md) - запрещено.

Использование в версии:

Доступен, начиная с версии 8.2.
