Расширение группы формы для страниц (Form group extension for pages)

ПриСменеСтраницы (OnCurrentPageChange)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриСменеСтраницы(<ТекущаяСтраница>)

Параметры:

<ТекущаяСтраница>

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md).   
Новая активная страница.

Описание:

Возникает при переключении с одной страницы на другую.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
