АдминистрированиеЛицензия (AdministrationLicense)

МаксимальноеКоличествоПользователейПрограммнойЛицензии (MaxSoftwareLicenseUsers)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальное количество пользователей, допустимое для данного комплекта, если используется программная лицензия платформы, иначе совпадает со значением свойства [МаксимальноеКоличествоПользователей](АдминистрированиеЛицензия.МаксимальноеКоличествоПользователей_(AdministrationLicense.MaxUsers).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
