Глобальный контекст (Global context)

НайтиОкноПоНавигационнойСсылке (FindWindowByURL)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиОкноПоНавигационнойСсылке(<НавигационнаяСcылка>)

Параметры:

<НавигационнаяСcылка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Навигационная ссылка.

Возвращаемое значение:

Тип: [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения_(ClientApplicationWindow).md), [Неопределено](lang__def_Undefined.md).   
Если окно не открыто, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Получает окно клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В коллекцию попадают только главные и вспомогательные окна приложения, в том числе и окна документов.  
В коллекцию окон не попадают служебные окна, модальные окна сообщений, подсказки и пр.   
В режиме запуска [ОбычноеПриложение](РежимЗапускаКлиентскогоПриложения.ОбычноеПриложение_(ClientRunMode.OrdinaryApplication).md) возвращает [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.2.
