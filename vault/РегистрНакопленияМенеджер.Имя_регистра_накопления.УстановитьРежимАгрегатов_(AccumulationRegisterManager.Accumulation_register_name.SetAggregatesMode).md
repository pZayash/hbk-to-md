РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

УстановитьРежимАгрегатов (SetAggregatesMode)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьРежимАгрегатов(<РежимАгрегатов>)

Параметры:

<РежимАгрегатов> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - вместо итогов используются агрегаты.   
[Ложь](lang__def_BooleanFalse.md) - используются итоги (режим соответствует поведению регистра накопления версии 1С:Предприятие 8.1).

Описание:

Устанавливает режим использования агрегатов.  
Если в составе конфигурации для регистра нет агрегатов, метод вызывает исключение.  
При включении режима агрегатов итоги очищаются.  
При выключении режима агрегатов очищаются агрегаты, а итоги пересчитываются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
