ЧтениеДанных (DataReader)

ПрочитатьЦелое32 (ReadInt32)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьЦелое32(<ПорядокБайтов>)

Параметры:

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
[Неопределено](lang__def_Undefined.md) - если не удалось прочитать число, поскольку был достигнут конец потока.

Описание:

Прочитать 32-битное целое число из потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
