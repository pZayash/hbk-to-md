# ОбъектМетаданных: Справочник.КоличествоУровней

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: Справочник](ОбъектМетаданных_Справочник_(MetadataObject_Catalog).md) › [Свойства](ОбъектМетаданных_Справочник__Свойства.md)

ОбъектМетаданных: Справочник (MetadataObject: Catalog)

КоличествоУровней (LevelCount)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Максимальное количество уровней иерархии справочника при условии, что свойство [ОграничиватьКоличествоУровней](ОбъектМетаданных_Справочник.ОграничиватьКоличествоУровней_(MetadataObject_Catalog.LimitLevelCount).md) имеет значение [Истина](lang__def_BooleanTrue.md). Если [ОграничиватьКоличествоУровней](ОбъектМетаданных_Справочник.ОграничиватьКоличествоУровней_(MetadataObject_Catalog.LimitLevelCount).md) имеет значение [Ложь](lang__def_BooleanFalse.md), то максимальное количество уровней иерархии неограниченно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
