ГеографическаяСхема (GeographicalSchema)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Из файла

Синтаксис:

Прочитать(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла географической схемы, который будет прочитан в текущий документ.

Описание варианта метода:

Чтение географической схемы из файла.

Вариант синтаксиса: Из потока

Синтаксис:

Прочитать(<Поток>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для чтения географической схемы.

Описание варианта метода:

Чтение географической схемы из потока.

Описание:

Считывает документ географической схемы из файла.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
