КоллекцияАтрибутовDOM (DOMAttributeMap)

ПолучитьИменованныйЭлемент (GetNamedItem)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя узла-атрибута DOM.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ПолучитьИменованныйЭлемент(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен атрибута DOM.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя узла атрибута DOM.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md).   

Описание:

Возвращает атрибут DOM по указанному имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
