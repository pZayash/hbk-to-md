ЧтениеУзловDOM (DOMNodeReader)

Значение (Value)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Текстовое значение текущего узла XML.  
Если для узла текстовое значение неопределено, то содержит пустую строку.  
Узлы, для которых определено текстовое значение:

- [Атрибут](ТипУзлаXML.Атрибут_(XMLNodeType.Attribute).md) - значение атрибута;
- [СекцияCDATA](ТипУзлаXML.СекцияCDATA_(XMLNodeType.CDATASection).md) - содержимое секции CDATA;
- [Комментарий](ТипУзлаXML.Комментарий_(XMLNodeType.Comment).md) - содержимое комментария;
- [ОпределениеТипаДокумента](ТипУзлаXML.ОпределениеТипаДокумента_(XMLNodeType.DocumentTypeDefinition).md) - внутреннее подмножество (internal subset);
- [ИнструкцияОбработки](ТипУзлаXML.ИнструкцияОбработки_(XMLNodeType.ProcessingInstruction).md) - содержимое инструкции обработки, исключая адрес (target);
- [Текст](ТипУзлаXML.Текст_(XMLNodeType.Text).md) - содержимое текстового узла;
- [ОбъявлениеXML](ТипУзлаXML.ОбъявлениеXML_(XMLNodeType.XMLDeclaration).md) - содержимое объявления.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
