АссоциированнаяГруппа (AssociationGroup)

Элементы (Items)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Фиксированный массив элементов, входящих в группу. Каждый элемент - объект анализа данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
