МенеджерТабличныхПространствБазыДанных (DatabaseTablespacesManager)

НайтиПоИзмененномуСоставуИндексов (FindByChangedIndexesContent)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НайтиПоИзмененномуСоставуИндексов(<Объект>)

Параметры:

<Объект> (необязательный)

Тип: [ЭлементСоставаТабличногоПространстваБазыДанных](ЭлементСоставаТабличногоПространстваБазыДанных.md), ОбъектМетаданных.   
Объект метаданных, для которого выполняется поиск табличного пространства.

Возвращаемое значение:

Тип: [МенеджерТабличногоПространстваБазыДанных](МенеджерТабличногоПространстваБазыДанных.md).   

Описание:

Возвращает табличное пространство, которое будет хранить индексы таблиц объекта метаданных после обновления размещения таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
