Описание:

Содержит типы маркера, которым отмечаются значения в точках для диаграммы [График](ТипДиаграммы.График_(ChartType.Line).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartMarkerType.

См. также:

[СерияДиаграммы](СерияДиаграммы_(ChartSeries).md), свойство [Маркер](СерияДиаграммы.Маркер_(ChartSeries.Marker).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
