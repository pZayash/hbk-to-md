Описание:

Объект является контейнером данных в формате PKCS#12 для хранения и/или транспортировки закрытого ключа, цепочки доверия от сертификата пользователя до корневого сертификата удостоверяющего центра и списка отзыва сертификатов (CRL). В упрощенном варианте содержит только сертификат и закрытый ключ, являющийся парой для открытого ключа из сертификата.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
