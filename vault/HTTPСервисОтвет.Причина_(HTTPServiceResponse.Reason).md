HTTPСервисОтвет (HTTPServiceResponse)

Причина (Reason)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текстовую расшифровку кода HTTP-статуса.  
Для стандартных статусов в ответе формируется автоматически.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
