ИсточникКомандПоляПланировщика (PlannerCommandSource)

Элементы (Items)

Доступен, начиная с версии 8.3.23.

Описание:

Тип: [ИсточникКомандПланировщика](ИсточникКомандПоляПланировщика.md).   
Значение, которое устанавливается при нажатии на элементы поля планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд.md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md) заполнено,
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md) содержит Неопределено,
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md) содержит Неопределено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата.md) содержит Неопределено,
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md) содержит Неопределено,
- Команды заполнены следующими стандартными командами планировщика: [УдалитьЭлементы](СтандартнаяКомандаПланировщика.УдалитьЭлементы.md), [РедактироватьЭлемент](СтандартнаяКомандаПланировщика.РедактироватьЭлемент.md) (недоступно, если элементов несколько),
- КомандаПоУмолчанию заполнено командой [БыстроРедактироватьЭлемент](СтандартнаяКомандаПланировщика.БыстроРедактироватьЭлемент.md) (игнорируется, если элементов несколько).

Использование в версии:

Доступен, начиная с версии 8.3.23.
