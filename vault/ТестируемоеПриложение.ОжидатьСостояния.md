ТестируемоеПриложение (TestedApplication)

ОжидатьСостояния (WaitForCondition)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОжидатьСостояния(<ОписаниеОповещения>, <Таймаут>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание функции, которая будет возвращать:

- Истина - если состояние достигнуто,
- Ложь - в противном случае.

<Таймаут> (необязательный)

Тип: Число.   
Время в секундах, в течение которого выполняется ожидание состояния.  
Значение по умолчанию: 60.

Возвращаемое значение:

Тип: Булево.   
Истина - если функция, проверяющая достижение состояния, вернула Истина.  
Ложь - закончился тайм-аут или функция проверки вернула Ложь.

Описание:

Ожидает наступления некоторого состояния за счет вызова функции, проверяющей условия его достижения с периодичностью 1 раз в секунду (без цикла ожидания по времени).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
