ВнешнийИсточникДанныхКубКлючЗаписи.<Имя внешнего источника>.<Имя куба> (ExternalDataSourceCubeRecordKey.<External source name>.<Cube name>)

По умолчанию

Доступен, начиная с версии 8.3.3.

Синтаксис:

Новый ВнешнийИсточникДанныхКубКлючЗаписи.<Имя внешнего источника>.<Имя куба>(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (необязательный)

Тип: [Структура](Структура_(Structure).md), [Неопределено](lang__def_Undefined.md).   
Содержит пары Ключ/Значение. Имя ключа - имя поля ключа, в качестве значения поля – необходимое значение этого поля. Значения должны быть перечислены в порядке полей ключа, как они определены в конфигураторе.  
Если параметр не задан или равен [Неопределено](lang__def_Undefined.md), или отсутствует, создается пустой ключ таблицы внешнего источника данных.

Описание:

Создает новый объект на основе переданных параметров.

Использование в версии:

Доступен, начиная с версии 8.3.3.
