Расширение элементов управления, расположенных в поле табличного документа (Extension for controls located in a spreadsheet document field)

ВыводитьНаПечать (Print)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак вывода на печать элемента управления.

Доступность:

Толстый клиент.

Примечание:

Истина - выводить рисунок на печать; Ложь - не выводить.  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
