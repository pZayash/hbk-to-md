**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов характеристик](Планы_видов_характеристик.md) › [ПланВидовХарактеристикОбъект.<Имя плана видов характеристик>](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesObject.Chart_of_characteristic_types_name).md) › [Методы](ПланВидовХарактеристикОбъект.Имя_плана_видов_характеристик__Методы.md)

ПланВидовХарактеристикОбъект.<Имя плана видов характеристик> (ChartOfCharacteristicTypesObject.<Chart of characteristic types name>)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.0.

Синтаксис:

Заблокирован()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект заблокирован; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, заблокирован ли объект.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Следует учитывать, что этот метод используется для проверки блокировки объекта базы данных конкретным объектом встроенного языка. Он не может быть использован для проверки, заблокирован ли вообще объект базы данных, например, другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.0.
