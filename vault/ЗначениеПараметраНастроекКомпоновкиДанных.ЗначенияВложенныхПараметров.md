ЗначениеПараметраНастроекКомпоновкиДанных (DataCompositionSettingsParameterValue)

ЗначенияВложенныхПараметров (NestedParameterValues)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийПараметровКомпоновкиДанных](КоллекцияЗначенийПараметровКомпоновкиДанных.md).   
Коллекция значений вложенных параметров.  
Данное свойство недоступно для чтения и записи у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
