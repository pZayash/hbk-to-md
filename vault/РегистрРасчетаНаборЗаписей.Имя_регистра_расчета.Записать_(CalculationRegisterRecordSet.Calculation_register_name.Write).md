# РегистрРасчетаНаборЗаписей.<Имя регистра расчета>.Записать

РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать(<Замещать>, <ТолькоЗапись>, <ЗаписьФактическогоПериодаДействия>, <ЗаписьПерерасчетов>)

Параметры:

<Замещать> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим замещения существующей записи в соответствии с текущими установками отбора (т.е. по регистратору набора записей). [Истина](lang__def_BooleanTrue.md) - перед записью существующие записи будут удалены. [Ложь](lang__def_BooleanFalse.md) - записи будут дописаны к уже существующим в информационной базе записям.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ТолькоЗапись> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим записи набора. [Истина](lang__def_BooleanTrue.md) - производится только запись набора; [Ложь](lang__def_BooleanFalse.md) - производится пересчет фактического периода действия и ввод записей перерасчетов.  
По умолчанию при записи набора производится не только собственно запись, но и пересчет периода действия (для регистров расчета с поддержкой периода действия) и ввод записей перерасчета (если необходимо).  
Внимание! Значение [Истина](lang__def_BooleanTrue.md) для параметра можно задавать только при записи уже записанного набора записей и только в том случае, если в наборе записей изменялись только ресурсы и реквизиты. Если в наборе записей изменялись любые другие поля записей набора, то запись набора с параметром, равным [Истина](lang__def_BooleanTrue.md) не выполняется и выдается сообщение об ошибке.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<ЗаписьФактическогоПериодаДействия> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если запись набора записей выполняется со значением параметра <ТолькоЗапись> равным [Ложь](lang__def_BooleanFalse.md) и при этом данный параметр равен [Ложь](lang__def_BooleanFalse.md), то при записи набора записей фактический период действия не рассчитывается.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ЗаписьПерерасчетов> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если запись набора записей выполняется со значением параметра <ТолькоЗапись> равным [Ложь](lang__def_BooleanFalse.md) и при этом данный параметр равен [Ложь](lang__def_BooleanFalse.md), то при записи набора записей перерасчеты не регистрируются.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Записывает набор записей в базу данных. В зависимости от переданного параметра может быть выполнено добавление записей или замещение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При вызове с параметром <Замещать> равным [Ложь](lang__def_BooleanFalse.md) после записи в информационную базу набор записей очищается (удаляются записи из набора).

Использование в версии:

Доступен, начиная с версии 8.0.
