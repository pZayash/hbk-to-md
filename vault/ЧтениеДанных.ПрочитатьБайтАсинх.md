ЧтениеДанных (DataReader)

ПрочитатьБайтАсинх (ReadByteAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПрочитатьБайтАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - прочитанный байт типа Число.  
Если не удалось прочитать байт из-за того, что был достигнут конец потока, результатом выполнения обещания будет значение Неопределено.

Описание:

Асинхронная функция. Читает один байт из потока.

Доступность:

Тонкий клиент, толстый клиент.
