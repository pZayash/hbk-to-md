Доступен, начиная с версии 8.3.15.

Элементы коллекции:

[ИнформацияОЗаписиВерсииИсторииДанных](ИнформацияОЗаписиВерсииИсторииДанных_(WriteDataHistoryVersionInformation).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.  
Возможно обращение к элементам посредством оператора [...]. В качестве аргумента передается индекс элемента коллекции (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияИнформацииОЗаписиВерсииИсторииДанных.Индекс_(WriteDataHistoryVersionInformationCollection.IndexOf).md)  
[Количество (Count)](КоллекцияИнформацииОЗаписиВерсииИсторииДанных.Количество_(WriteDataHistoryVersionInformationCollection.Count).md)  
[Получить (Get)](КоллекцияИнформацииОЗаписиВерсииИсторииДанных.Получить_(WriteDataHistoryVersionInformationCollection.Get).md)  

Описание:

Представляет коллекцию версий истории данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
