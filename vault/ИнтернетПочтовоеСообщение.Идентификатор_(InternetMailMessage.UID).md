ИнтернетПочтовоеСообщение (InternetMailMessage)

Идентификатор (UID)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит строку, идентифицирующую сообщение. Данный идентификатор сообщения уникален в пределах почтового ящика и остается неизменным на протяжении всего времени существования этого сообщения в почтовом ящике на сервере. После объединения частичных сообщений, результирующее сообщение может содержать несколько идентификаторов, поэтому свойство имеет тип [Массив](Массив_(Array).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИнтернетПочта](ИнтернетПочта_(InternetMail).md), метод [ПолучитьИдентификаторы](ИнтернетПочта.ПолучитьИдентификаторы_(InternetMail.GetUIDL).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
