ОписаниеКомандыПоляВвода (InputFieldCommandDescription)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый ОписаниеКомандыПоляВвода(<Команда>, <Представление>)

Параметры:

<Команда> (необязательный)

Тип: [СтандартнаяКомандаПоляВвода](СтандартнаяКомандаПоляВвода_(InputFieldStandardCommand).md), [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md), [Массив](Массив_(Array).md), [Неопределено](lang__def_Undefined.md).   
Содержит описание команды:

- [СтандартнаяКомандаПоляВвода](СтандартнаяКомандаПоляВвода_(InputFieldStandardCommand).md) – стандартная команда;
- [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md) – процедура, которая будет вызвана при выборе пункта меню. Процедура должна содержать следующие параметры:
  - <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md);
- [Массив](Массив_(Array).md) – задает подменю. Массив может содержать элементы обозначенных выше типов;
- [Неопределено](lang__def_Undefined.md) – разделитель;
- Любой другой тип - пункт меню не отображается.

  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Представление> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст команды.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.24.
