ОписаниеПараметровЗапроса (QueryParametersDescription)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя искомого параметра.

Возвращаемое значение:

Тип: [ОписаниеПараметраЗапроса](ОписаниеПараметраЗапроса.md), Неопределено.   
Если искомое значение не найдено, то возвращается Неопределено.

Описание:

Осуществляет поиск параметра запроса по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
