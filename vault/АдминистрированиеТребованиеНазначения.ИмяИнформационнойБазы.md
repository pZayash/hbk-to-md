Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя информационной базы, для которой действует требование.   
Пустая строка, если требование должно действовать для всех информационных баз.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
