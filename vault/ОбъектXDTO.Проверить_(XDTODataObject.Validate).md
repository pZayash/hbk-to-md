**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md) › [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md) › [Методы](ОбъектXDTO__Методы.md)

ОбъектXDTO (XDTODataObject)

Проверить (Validate)

Доступен, начиная с версии 8.1.

Синтаксис:

Проверить()

Описание:

Проверяет данный объект на соответствие модели XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
