# ТекущиеДанныеСписка

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md)

Доступен, начиная с версии 8.0.

Свойства:

[\<Имя колонки списка\> (\<List column name\>)](ТекущиеДанныеСписка.Имя_колонки_списка_(CurrentDataOfList.List_column_name).md)  

Описание:

Элемент выборки динамического списка. Содержит значения колонок одного элемента динамического списка. Доступ к значениям осуществляется через свойства. Имя свойства совпадает с именем колонки списка. Возвращается в качестве значения свойства [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) расширения табличного поля для динамических списков, если табличное поле не находится в режиме редактирования записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
