РежимОткрытияФайла (FileOpenMode)

СоздатьНовый (CreateNew)

Доступен, начиная с версии 8.3.9.

Описание:

Создание нового файла. Если такой файл существует, будет сгенерировано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
