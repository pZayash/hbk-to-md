КонстантаМенеджерЗначения.<Имя константы> (ConstantValueManager.<Constant name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает значение константы из свойства [Значение](КонстантаМенеджерЗначения.Имя_константы.Значение_(ConstantValueManager.Constant_name.Value).md) в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
