АдминистрированиеРабочийСервер (AdministrationWorkServer)

ПолучитьНастройкуСервиса (GetServiceSetting)

Доступен, начиная с версии 8.3.26.

Вариант синтаксиса: По имени сервиса и информационной базы

Синтаксис:

ПолучитьНастройкуСервиса(<ИмяСервиса>, <ИмяИнформационнойБазы>)

Параметры:

<ИмяСервиса> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя сервиса кластера как в [АдминистрированиеСервис](АдминистрированиеСервис_(AdministrationService).md).

<ИмяИнформационнойБазы> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя информационной базы, для которой определена настройка сервиса кластера на рабочем сервере. Пустая строка, если данные сервиса не распределены по информационным базам.  
Значение по умолчанию: Пустая строка.

Вариант синтаксиса: По идентификатору настройки

Синтаксис:

ПолучитьНастройкуСервиса(<ИдентификаторНастройки>)

Параметры:

<ИдентификаторНастройки> (обязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор настройки сервиса кластера.

Возвращаемое значение:

Тип: [НастройкаСервиса](НастройкаСервиса_(ServiceSetting).md).   

Описание:

Возвращает настройку сервиса кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.

Описание изменено в версии 8.3.27.
