Форма (Form)

ВнешнееСобытие (ExternalEvent)

Доступен, начиная с версии 8.0.

Синтаксис:

ВнешнееСобытие(<Источник>, <Событие>, <Данные>)

Параметры:

<Источник>

Тип: [Строка](lang__def_String.md).   
Источник внешнего события.

<Событие>

Тип: [Строка](lang__def_String.md).   
Наименование события.

<Данные>

Тип: [Строка](lang__def_String.md).   
Данные для события.

Описание:

Возникает при посылке внешним приложением сообщения, сформированного в специальном формате. Внешнее событие сначала обрабатывается всеми открытыми формами, имеющими обработчик этого события, а затем может быть обработано в процедуре модуля обычного приложения с именем [ОбработкаВнешнегоСобытия](Глобальный_контекст.ОбработкаВнешнегоСобытия_(Global_context.ExternEventProcessing).md).

Доступность:

Толстый клиент.
