ВидСравнения (ComparisonType)

ИнтервалВключаяОкончание (IntervalIncludingUpperBound)

Доступен, начиная с версии 8.0.

Описание:

Сравниваемое значение лежит в заданном интервале, не включая начальную и включая конечную границу интервала.

Использование в версии:

Доступен, начиная с версии 8.0.
