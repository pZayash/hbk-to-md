ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение задачи.

Описание:

Заполняет данные задачи данными из другого объекта. Инициирует событие [ОбработкаЗаполнения](ЗадачаОбъект.Имя_задачи.ОбработкаЗаполнения.md) и вызов его процедуры-обработчика в модуле задачи. В ней может размещаться алгоритм, заполняющий данные задачи из переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для заполнения может использоваться значение любого типа.

Пример:

|  |
| --- |
| Объект.Заполнить(ОбъектОснование); |

Использование в версии:

Доступен, начиная с версии 8.0.
