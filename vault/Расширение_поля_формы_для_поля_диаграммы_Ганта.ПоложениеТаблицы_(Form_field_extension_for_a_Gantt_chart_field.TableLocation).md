Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ПоложениеТаблицы (TableLocation)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеТаблицыДиаграммыГанта](ПоложениеТаблицыДиаграммыГанта_(GanttChartTableLocation).md).   
Настройка положения предопределенной таблицы относительно поля диаграммы Ганта.   
Значение [Авто](ПоложениеТаблицыДиаграммыГанта.Авто_(GanttChartTableLocation.Auto).md) интерпретируется как [Авто](ПоложениеТаблицыДиаграммыГанта.Авто_(GanttChartTableLocation.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
