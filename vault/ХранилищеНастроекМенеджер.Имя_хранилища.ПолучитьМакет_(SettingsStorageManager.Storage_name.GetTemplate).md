# ХранилищеНастроекМенеджер.<Имя хранилища>.ПолучитьМакет

ХранилищеНастроекМенеджер.<Имя хранилища> (SettingsStorageManager.<Storage name>)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьМакет()

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md); другой объект, который может быть макетом.   

Описание:

Получает макет хранилища настроек по идентификатору макета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
