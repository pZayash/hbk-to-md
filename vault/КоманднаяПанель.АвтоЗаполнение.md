КоманднаяПанель (CommandBar)

АвтоЗаполнение (AutoFill)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Признак автоматического заполнения командной панели.  
Если установлено значение - Истина, командная панель будет заполнена автоматически действиями, предоставленными источником действий (см. свойство [ИсточникДействий](Расширение_элементов_управления,_расположенных_в_форме.ИсточникДействий.md) объекта [Расширение элементов управления, расположенных в форме](Интерфейс.Расширение_элементов_управления,_расположенных_в_форме.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
