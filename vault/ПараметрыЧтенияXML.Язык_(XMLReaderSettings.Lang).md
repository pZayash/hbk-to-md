ПараметрыЧтенияXML (XMLReaderSettings)

Язык (Lang)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Язык документа XML - начальное значение атрибута xml:lang для документа XML.  
Значение данного свойства является начальным значением свойства Язык объекта [ЧтениеXML](ЧтениеXML_(XMLReader).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
