БуферДвоичныхДанных (BinaryDataBuffer)

Получить (Get)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Получить(<Позиция>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция элемента в буфере. Нумерация начинается с 0.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает значение элемента на указанной позиции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
