ФайловыйПоток (FileStream)

НачатьКопированиеВ (BeginCopyTo)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьКопированиеВ(<ОписаниеОповещения>, <ЦелевойПоток>, <Количество>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Количество> - тип [Число](lang__def_Number.md). Количество фактически скопированных байтов.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ЦелевойПоток> (обязательный)

Тип: [Поток](Поток_(Stream).md).   
Поток, в который будет производиться копирование.

<Количество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, которые требуется скопировать в целевой поток. Если не задано, то будут скопированы все данные до конца текущего потока.

Описание:

Начинает копирование данных из текущего потока в другой поток. Дополнительно можно установить количество байтов, которые будут скопированы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[Поток](Поток_(Stream).md), метод [КопироватьВ](Поток.КопироватьВ_(Stream.CopyTo).md)  
[Поток](Поток_(Stream).md), метод [КопироватьВАсинх](Поток.КопироватьВАсинх_(Stream.CopyToAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
