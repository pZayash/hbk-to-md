# ВыделенныеОбластиТабличногоДокумента

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Табличный документ](objects__catalog63__catalog64.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента.md), [РисунокТабличногоДокумента](РисунокТабличногоДокумента.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются выделенные области.  
Возможно обращение к выделенной области посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит коллекцию выделенных областей табличного документа.  
Может содержать либо рисунки табличного документа, либо области ячеек.  
Первый элемент коллекции содержит текущую область табличного документа.  
При изменении выделенной области табличного документа осуществляется автоматическое прокручивание документа таким образом, чтобы выделенная область находилась в видимой области.  
На мобильной платформе может содержать только одну ячейку.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
