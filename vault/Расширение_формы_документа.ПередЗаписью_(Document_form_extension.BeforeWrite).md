# Расширение формы документа.ПередЗаписью

Расширение формы документа (Document form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>, <РежимЗаписи>, <РежимПроведения>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи документа. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись документа выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<РежимЗаписи>

Тип: [РежимЗаписиДокумента](РежимЗаписиДокумента_(DocumentWriteMode).md).   
Режим записи документа. Позволяет определить выполняется запись, проведение или отмена проведения.  
Изменение значения параметра позволяет изменить режим записи.

<РежимПроведения>

Тип: [РежимПроведенияДокумента](РежимПроведенияДокумента_(DocumentPostingMode).md).   
Позволяет определить, выполняется оперативное проведение или нет.  
Изменение значения параметра позволяет изменить режим проведения.

Описание:

Возникает перед выполнением записи объекта из формы. Процедура-обработчик данного события вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](ДокументОбъект.Имя_документа.ПередЗаписью_(DocumentObject.Document_name.BeforeWrite).md) объекта [ДокументОбъект.](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)<[Имя документа](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
