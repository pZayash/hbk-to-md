# МенеджерПользователейИнформационнойБазы.НайтиПоИмени

МенеджерПользователейИнформационнойБазы (InfoBaseUsersManager)

НайтиПоИмени (FindByName)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПоИмени(<ИмяПользователя>)

Параметры:

<ИмяПользователя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя.

Возвращаемое значение:

Тип: [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md), [Неопределено](lang__def_Undefined.md).   
Если значение не найдено, то для пользователя с административными правами возвращается значение [Неопределено](lang__def_Undefined.md), для других пользователей вызывается исключение.

Описание:

Осуществляет поиск информации о пользователе с указанным именем.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В случае, если пользователь имеет административные права, то допускается поиск любого пользователя. Если пользователь не имеет административных прав, то допускается поиск только того пользователя, под которым данный пользователь аутентифицировался.   
Если указано пустое имя пользователя, то возвращается объект, который можно использовать в тех случаях, когда пользователь не задан (например, в фильтре журнала регистрации).  
Если в конфигурации имеются разделители, разделяющие аутентификацию, и хотя бы один из них не используется сеансом или не разделяет пользователей, то при выполнении метода генерируется исключение в том случае, если доступен хотя бы один пользователь с таким именем в списке пользователей информационной базы.

Использование в версии:

Доступен, начиная с версии 8.1.
