ТипРазмещенияИзмерений (DimensionPlacementType)

Отдельно (Separately)

Доступен, начиная с версии 8.0.

Описание:

Каждое измерение располагается в отдельной строке/колонке.

Использование в версии:

Доступен, начиная с версии 8.0.
