МенеджерКриптографии (CryptoManager)

ПолучитьДанныеИзПодписи (GetDataFromSignature)

Доступен, начиная с версии 8.3.24.

Вариант синтаксиса: Данные подписи в возвращаемом значении

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.  
Если параметр типа Строка, то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные.md) или [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md), то он задает данные подписи.

Описание варианта метода:

Данные возвращаются методом как [ДвоичныеДанные](ДвоичныеДанные.md).

Вариант синтаксиса: Сохранение данных в файл или запись их в поток

Синтаксис:

ПолучитьДанныеИзПодписи(<Подпись>, <ВыходныеДанные>)

Параметры:

<Подпись> (обязательный)

Тип: Строка, [ДвоичныеДанные](ДвоичныеДанные.md), [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Присоединенная электронная подпись, из которой будут извлечены подписанные данные.  
Если параметр типа Строка, то обозначает имя файла с подписью.  
Если параметр типа [ДвоичныеДанные](ДвоичныеДанные.md) или [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md), то он задает данные подписи.

<ВыходныеДанные> (обязательный)

Тип: Строка, [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Имя файла или поток, в который будут выгружены данные, содержащиеся в подписи.

Возвращаемое значение:

Тип: [ДвоичныеДанные](ДвоичныеДанные.md), Неопределено.   
В зависимости от варианта метода, либо [ДвоичныеДанные](ДвоичныеДанные.md), либо Неопределено.

Описание:

Извлекает подписанные данные из электронной подписи.   
Если данные отсутствуют в подписи, т.е. она является отсоединенной, то будет вызвано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[МенеджерКриптографии](МенеджерКриптографии.md), метод [ПолучитьДанныеИзПодписиАсинх](МенеджерКриптографии.ПолучитьДанныеИзПодписиАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.24.
