# КоллекцияПолейСводнойТаблицы

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › <a href="obsidian://open?file=objects__catalog63__catalog64.md">Табличный документ</a> › [Сводная таблица](Общие_объекты.Сводная_таблица.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеСводнойТаблицы](ПолеСводнойТаблицы.md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной таблицы или коллекция полей сводной таблицы.  
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс элемента (нумерация с 0).
