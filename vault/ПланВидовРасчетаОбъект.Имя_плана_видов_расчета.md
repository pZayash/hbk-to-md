Описание:

Предназначен для чтения, изменения, добавления и удаления вида расчета, как объекта базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/enterprise/current-config}. Имя типа XDTO: ChartOfCalculationTypesObject.<Имя плана видов расчета>.
