ОпределениеТипаДокументаDOM (DOMDocumentType)

На основании имени документа XML

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ОпределениеТипаДокументаDOM(<Имя>, <ПубличныйИдентификатор>, <СистемныйИдентификатор>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя документа XML.

<ПубличныйИдентификатор> (необязательный)

Тип: Строка.   
Публичный идентификатор определения типа документа.

<СистемныйИдентификатор> (необязательный)

Тип: Строка.   
Системный идентификатор определения типа документа.

Описание:

Создает экземпляр объекта.

Использование в версии:

Доступен, начиная с версии 8.1.
