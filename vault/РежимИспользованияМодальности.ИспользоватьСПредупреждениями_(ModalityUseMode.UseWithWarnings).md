РежимИспользованияМодальности (ModalityUseMode)

ИспользоватьСПредупреждениями (UseWithWarnings)

Доступен, начиная с версии 8.3.3.

Описание:

Разрешает использование в конфигурации методов, открывающих модальные окна. Однако при вызове такого метода в окно сообщений будет выведено предупреждение. Если такие методы используются, то конфигурация не будет работать в полном объеме через веб-клиент на тех платформах, где поддержка модальных окон отсутствует.

Использование в версии:

Доступен, начиная с версии 8.3.3.
