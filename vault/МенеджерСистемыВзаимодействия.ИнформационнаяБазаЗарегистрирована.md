МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ИнформационнаяБазаЗарегистрирована (InfoBaseRegistered)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ИнформационнаяБазаЗарегистрирована()

Возвращаемое значение:

Тип: Булево.   
Истина - информационная база зарегистрирована в системе взаимодействия и регистрация не отключена.

Описание:

Определяет, зарегистрирована ли информационная база в системе взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
