ТаблицаФормы (FormTable)

ВертикальныеЛинии (VerticalLines)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Управляет отображением вертикальных линий в таблице:

- Истина - линии видны.
- Ложь - линии не видны.
- Неопределено - в таблице со списком в виде дерева линии не видны, в других таблицах линии видны.

Значение по умолчанию: Неопределено.  
Для таблиц, сохраненных в старых версиях платформы, при открытии форм, отображающих таблицы, значение Истина интерпретируется как Неопределено(Авто).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
