КнопкиКоманднойПанели (CommandBarButtons)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Удаляет все кнопки из командной панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| КП = ЭлементыФормы.КоманднаяПанель; КП.Кнопки.Очистить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
