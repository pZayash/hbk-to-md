# КоллекцияЭлементовУправленияИнтерфейсами

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Работа с пользовательским интерфейсом](Работа_с_пользовательским_интерфейсом.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементУправленияИнтерфейсом](ЭлементУправленияИнтерфейсом_(InterfaceControlItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).  

Свойства:

[\<Имя интерфейса\> (\<Interface name\>)](КоллекцияЭлементовУправленияИнтерфейсами.Имя_интерфейса_(InterfaceControlItemCollection.Interface_name).md)  

Методы:

[Индекс (IndexOf)](КоллекцияЭлементовУправленияИнтерфейсами.Индекс_(InterfaceControlItemCollection.IndexOf).md)  
[Количество (Count)](КоллекцияЭлементовУправленияИнтерфейсами.Количество_(InterfaceControlItemCollection.Count).md)  
[Найти (Find)](КоллекцияЭлементовУправленияИнтерфейсами.Найти_(InterfaceControlItemCollection.Find).md)  
[ПереключитьИнтерфейс (SwitchInterface)](КоллекцияЭлементовУправленияИнтерфейсами.ПереключитьИнтерфейс_(InterfaceControlItemCollection.SwitchInterface).md)  
[Получить (Get)](КоллекцияЭлементовУправленияИнтерфейсами.Получить_(InterfaceControlItemCollection.Get).md)  

Описание:

Предназначен для управления видимостью командных интерфейсов.

Доступность:

Толстый клиент.
