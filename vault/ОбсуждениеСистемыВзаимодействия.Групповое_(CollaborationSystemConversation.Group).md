ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

Групповое (Group)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак, что обсуждение является групповым.  
Установка свойства возможна только для нового незаписанного обсуждения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
