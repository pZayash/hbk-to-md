# ВнешнийОбъект

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › [Общие объекты](Общие_объекты.md)

Доступен, начиная с версии 8.2.

Описание:

Свойства, методы и события объектов данного типа являются динамическими, т.е. зависят от самого объекта.  
Для значений типа [ВнешнийОбъект](ВнешнийОбъект.md), свойства, являющиеся динамическими объектами, имеют тип [ВнешнийОбъект](ВнешнийОбъект.md).

Доступность:

Веб-клиент.

Доступен, начиная с версии 8.2.
