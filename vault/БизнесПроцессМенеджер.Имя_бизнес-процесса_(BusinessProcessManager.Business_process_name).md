Описание:

Предназначен для управления бизнес-процессом, как объектом конфигурации. С помощью этого объекта осуществляется поиск бизнес-процессов, получение выборки бизнес-процессов, создание нового бизнес-процесса, работа с формами и макетами бизнес-процесса. Доступ к объекту осуществляется через свойства объекта [БизнесПроцессыМенеджер](БизнесПроцессыМенеджер_(BusinessProcessesManager).md). Полное имя типа объекта определяется с учетом имени бизнес-процесса конфигурации. Например, для бизнес-процесса "Выписка счета" имя типа будет выглядеть БизнесПроцессМенеджер.ВыпискаСчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.
