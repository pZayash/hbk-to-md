ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ВключатьСправкуВСодержание (IncludeHelpInContents)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если свойство установлено в значение Истина, то cправочная информация, относящаяся к объекту метаданных конфигурация (свойство [Справка](ОбъектМетаданныхКонфигурация.Справка.md)), включается в общее описание конфигурации.  
В противном случае справочная информация не включается в общее описание конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
