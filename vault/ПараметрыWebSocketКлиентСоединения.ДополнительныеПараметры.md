Доступен, начиная с версии 8.3.27.

Чтение и запись.

Описание:

Тип: Произвольный.   
Свойство предназначено для передачи в прикладной код, использующий объект [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md) данных, которые требуются для корректной обработки событий.

Использование в версии:

Доступен, начиная с версии 8.3.27.
