Расширение формы клиентского приложения для задачи (Managed form extension for tasks)

ПриЧтенииНаСервере (OnReadAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЧтенииНаСервере(<ТекущийОбъект>)

Параметры:

<ТекущийОбъект>

Тип: [ЗадачаОбъект.\<Имя задачи\>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md).   
Объект, который будет прочитан.

Описание:

Вызывается при чтении объекта на сервере.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
