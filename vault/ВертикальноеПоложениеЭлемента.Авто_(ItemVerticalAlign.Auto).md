ВертикальноеПоложениеЭлемента (ItemVerticalAlign)

Авто (Auto)

Доступен, начиная с версии 8.2.

Описание:

Вертикальное положение определяется автоматически.  
Для однострочной декорации будет использовано значение [Центр](ВертикальноеПоложениеЭлемента.Центр_(ItemVerticalAlign.Center).md).  
Для многострочной - [Верх](ВертикальноеПоложениеЭлемента.Верх_(ItemVerticalAlign.Top).md).

Использование в версии:

Доступен, начиная с версии 8.2.
