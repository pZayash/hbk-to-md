ЗаписьFastInfoset (FastInfosetWriter)

ОткрытьФайл (OpenFile)

Доступен, начиная с версии 8.1.

Синтаксис:

ОткрытьФайл(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла для записи.

Описание:

Открывает файл для записи данных в формате Fast Infoset.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
