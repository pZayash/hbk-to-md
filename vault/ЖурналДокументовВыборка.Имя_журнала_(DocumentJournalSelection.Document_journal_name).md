Описание:

Объект этого типа возвращается методом [Выбрать](ЖурналДокументовМенеджер.Имя_журнала_документов.Выбрать_(DocumentJournalManager.Document_journal_name.Select).md) у объекта [ЖурналДокументовМенеджер.](ЖурналДокументовМенеджер.Имя_журнала_документов_(DocumentJournalManager.Document_journal_name).md)<[Имя журнала документов](ЖурналДокументовМенеджер.Имя_журнала_документов_(DocumentJournalManager.Document_journal_name).md)> и представляет собой специализированный способ перебора документов, регистрируемых в журнале.  
Обход документов выполняется системой динамически. Это означает, что использование выборки не считывает все документы сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большие списки документов и не загружает в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
