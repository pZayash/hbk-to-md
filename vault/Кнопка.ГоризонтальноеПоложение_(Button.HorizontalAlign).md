Кнопка (Button)

ГоризонтальноеПоложение (HorizontalAlign)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md).   
Определяет способ горизонтального расположения текста на кнопке.

Доступность:

Толстый клиент.

Примечание:

Допустимыми являются только значения [Лево](ГоризонтальноеПоложение.Лево_(HorizontalAlign.Left).md), [Право](ГоризонтальноеПоложение.Право_(HorizontalAlign.Right).md) или [Центр](ГоризонтальноеПоложение.Центр_(HorizontalAlign.Center).md).

Использование в версии:

Доступен, начиная с версии 8.0.
