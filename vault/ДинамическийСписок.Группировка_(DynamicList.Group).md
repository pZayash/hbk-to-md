ДинамическийСписок (DynamicList)

Группировка (Group)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ПоляГруппировкиКомпоновкиДанных](ПоляГруппировкиКомпоновкиДанных_(DataCompositionGroupFields).md).   
Описывает поля, по которым в динамическом списке будет осуществляться группировка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
