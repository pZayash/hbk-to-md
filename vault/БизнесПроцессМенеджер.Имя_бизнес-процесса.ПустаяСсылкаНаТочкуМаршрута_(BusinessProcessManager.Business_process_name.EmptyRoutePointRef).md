ПустаяСсылкаНаТочкуМаршрута (EmptyRoutePointRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылкаНаТочкуМаршрута()

Возвращаемое значение:

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   

Описание:

Получает пустое значение ссылки на точку маршрута бизнес-процесса данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Может использоваться, когда нужно передать пустую ссылку в параметр метода.

Использование в версии:

Доступен, начиная с версии 8.0.
