ЭлементОтбораКомпоновкиДанных (DataCompositionFilterItem)

Применение (Application)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [ТипПримененияОтбораКомпоновкиДанных](ТипПримененияОтбораКомпоновкиДанных.md).   
Определяет тип применения отбора к записям результата компоновки.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
