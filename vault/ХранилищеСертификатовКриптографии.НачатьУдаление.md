ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НачатьУдаление (BeginDeleting)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьУдаление(<ОписаниеОповещения>, <Сертификат>)

Параметры:

<ОписаниеОповещения> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения добавления со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Сертификат> (обязательный)

Тип: [СертификатКриптографии](СертификатКриптографии.md), [КонтейнерКлючейКриптографии](КонтейнерКлючейКриптографии.md).   
Сертификат, который нужно удалить из хранилища.   
На мобильной платформе вместо сертификата может быть использован контейнер ключей криптографии.

Описание:

Начинает удаление сертификата из хранилища.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В мобильной платформе под управлением Windows в хранилище с типом [ПерсональныеСертификаты](ТипХранилищаСертификатовКриптографии.ПерсональныеСертификаты.md) метод удаляет только сертификаты и контейнеры ключей криптографии с истекшим сроком действия.
