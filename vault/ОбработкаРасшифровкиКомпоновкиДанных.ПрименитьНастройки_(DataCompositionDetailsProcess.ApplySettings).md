ОбработкаРасшифровкиКомпоновкиДанных (DataCompositionDetailsProcess)

ПрименитьНастройки (ApplySettings)

Доступен, начиная с версии 8.1.

Синтаксис:

ПрименитьНастройки(<ИдентификаторРасшифровки>, <Настройки>)

Параметры:

<ИдентификаторРасшифровки> (обязательный)

Тип: [ИдентификаторРасшифровкиКомпоновкиДанных](ИдентификаторРасшифровкиКомпоновкиДанных_(DataCompositionDetailsID).md).   
Идентификатор обрабатываемой расшифровки.

<Настройки> (обязательный)

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Применяемые настройки.

Возвращаемое значение:

Тип: [Настройки компоновки данных](Настройки_компоновки_данных.md), [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных_(DataCompositionUserSettings).md).   

Описание:

Позволяет модифицировать настройки, описав изменения в виде другого объекта настроек.  
Состав применяемых настроек имеет ограничения:

- применяемые настройки не должны иметь пользовательских полей, либо состав пользовательских полей обоих настроек не должен различаться;
- структура применяемых настроек не должна содержать больше двух группировок, причем одна из них может иметь только фиксированное имя "ТекущаяГруппировка", она используется для модификации настроек текущей расшифровываемой группировки. Обе эти группировки не должны содержать вложенной структуры.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
