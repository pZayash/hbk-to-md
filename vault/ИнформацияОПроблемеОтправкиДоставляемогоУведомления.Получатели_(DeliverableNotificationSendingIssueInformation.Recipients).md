ИнформацияОПроблемеОтправкиДоставляемогоУведомления (DeliverableNotificationSendingIssueInformation)

Получатели (Recipients)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив получателей, запрос на отправку уведомления для которых завершился неудачно с одним и тем же типом ошибки.  
Массив содержит элементы типа [ИдентификаторПодписчикаДоставляемыхУведомлений](ИдентификаторПодписчикаДоставляемыхУведомлений_(DeliverableNotificationSubscriberID).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.11.
