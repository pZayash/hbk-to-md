ТабличныйДокумент (SpreadsheetDocument)

ТекущаяОбласть (CurrentArea)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ОбластьЯчеекТабличногоДокумента](ОбластьЯчеекТабличногоДокумента_(SpreadsheetDocumentRange).md).   
Содержит текущую активную область или рисунок табличного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В случае отсутствия визуального представления табличного документа или если в данный момент нет выделенной области, имеет значение [Неопределено](lang__def_Undefined.md).
