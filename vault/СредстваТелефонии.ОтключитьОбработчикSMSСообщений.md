СредстваТелефонии (TelephonyTools)

ОтключитьОбработчикSMSСообщений (DetachSMSMessageHandler)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ОтключитьОбработчикSMSСообщений(<ИмяПроцедуры>)

Параметры:

<ИмяПроцедуры> (необязательный)

Тип: Строка, [ОписаниеОповещения](ОписаниеОповещения.md).   
Имя отключаемой процедуры.

Описание:

Отключает ранее подключенный обработчик SMS-сообщений.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

В мобильном приложении для Windows будет выполнен возврат из метода без вызова обработчика.

Использование в версии:

Доступен, начиная с версии 8.3.5.
