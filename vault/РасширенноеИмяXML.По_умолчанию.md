РасширенноеИмяXML (XMLExpandedName)

По локальному имени и URI пространства имен

Доступен, начиная с версии 8.1.

Синтаксис:

Новый РасширенноеИмяXML(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя.

Описание:

Создает экземпляр расширенного имени XML.

Использование в версии:

Доступен, начиная с версии 8.1.
