Свойства:

[ПараметрТекущаяСтрока (CurrentLineParameter)](Расширение_формы_списка_плана_счетов.ПараметрТекущаяСтрока_(Chart_of_accounts_list_form_extension.CurrentLineParameter).md)  

Описание:

Дополнительные свойства объекта [Форма](Форма_(Form).md) в том случае, если основной реквизит формы имеет тип [ПланСчетовСписок.](ПланСчетовСписок.Имя_плана_счетов_(ChartOfAccountsList.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовСписок.Имя_плана_счетов_(ChartOfAccountsList.Chart_of_accounts_name).md)>.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Форма.ПараметрВыборГруппИЭлементов = ИспользованиеГруппИЭлементов.Элементы; |

Использование в версии:

Доступен, начиная с версии 8.0.
