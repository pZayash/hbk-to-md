Расширение поля формы для поля периода (Form field extension for a period field)

Высота (Height)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Устанавливает высоту элемента управления в строках.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
