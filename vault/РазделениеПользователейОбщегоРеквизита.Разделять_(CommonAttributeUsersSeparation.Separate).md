РазделениеПользователейОбщегоРеквизита (CommonAttributeUsersSeparation)

Разделять (Separate)

Доступен, начиная с версии 8.2.

Описание:

Общий реквизит разделяет список пользователей информационной базы. Если сеанс использует разделитель, то доступны только пользователи информационной базы, у которых установлено используемое в сеансе значение разделителя.

Использование в версии:

Доступен, начиная с версии 8.2.
