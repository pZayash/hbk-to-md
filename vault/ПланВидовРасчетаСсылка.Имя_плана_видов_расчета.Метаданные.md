Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Тип: [ОбъектМетаданных: ПланВидовРасчета](ОбъектМетаданных_ПланВидовРасчета.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
