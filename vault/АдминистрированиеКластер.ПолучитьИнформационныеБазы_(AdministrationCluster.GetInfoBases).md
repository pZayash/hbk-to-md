АдминистрированиеКластер (AdministrationCluster)

ПолучитьИнформационныеБазы (GetInfoBases)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьИнформационныеБазы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеИнформационнаяБаза](АдминистрированиеИнформационнаяБаза_(AdministrationInfoBase).md).

Описание:

Получает массив информационных баз, зарегистрированных в кластере. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
