ОформлениеЯчейки (CellAppearance)

УстановитьТекст (SetText)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьТекст(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: Строка.   
Текст ячейки.

Описание:

Устанавливает текст ячейки. Устанавливает свойство [ОтображатьТекст](ОформлениеЯчейки.ОтображатьТекст.md) в значение Истина.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
