# Расширение поля ввода перечисления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Перечисления](Перечисления.md)

Доступен, начиная с версии 8.0.

Свойства:

[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_перечисления.БыстрыйВыбор_(Enum_text_box_extension.QuickChoice).md)  
[ДоступныеЗначения (AvailableValues)](Расширение_поля_ввода_перечисления.ДоступныеЗначения_(Enum_text_box_extension.AvailableValues).md)  

Описание:

Расширение предоставляет возможность автоматического поиска и подбора значения перечисления в элементе управления [ПолеВвода](ПолеВвода_(TextBox).md) по введенному тексту. При этом поиск осуществляется по его текстовому представлению.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
