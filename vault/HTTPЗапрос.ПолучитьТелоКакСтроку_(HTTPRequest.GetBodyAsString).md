# HTTPЗапрос.ПолучитьТелоКакСтроку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Интернет](Интернет-2.md) › [Работа с HTTP](Работа_с_HTTP.md) › [HTTPЗапрос](HTTPЗапрос_(HTTPRequest).md) › [Методы](HTTPЗапрос__Методы.md)

HTTPЗапрос (HTTPRequest)

ПолучитьТелоКакСтроку (GetBodyAsString)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьТелоКакСтроку()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - тело не было установлено или был установлен файл тела.

Описание:

Получает тело запроса как строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
