**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Функции работы со значениями типа Число](Функции_работы_со_значениями_типа_Число.md)

Глобальный контекст (Global context)

Цел (Int)

Доступен, начиная с версии 8.0.

Синтаксис:

Цел(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Исходное число.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат выделения целой части.

Описание:

Вычисляет целую часть переданного числа, полностью отсекая дробную часть.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| МожноКупить = Цел(Наличность/Цена); |

Использование в версии:

Доступен, начиная с версии 8.0.
