ВнешнийОтчет (ExternalReport)

ИспользуемоеИмяФайла (UsedFileName)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит полное имя файла внешнего отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если внешний отчет открыт в веб-клиенте, то содержит имя файла (без пути).

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.18.
