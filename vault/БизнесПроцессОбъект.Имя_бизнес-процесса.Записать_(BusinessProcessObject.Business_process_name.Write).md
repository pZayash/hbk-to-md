# БизнесПроцессОбъект.<Имя бизнес-процесса>.Записать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [БизнесПроцессОбъект.<Имя бизнес-процесса>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md) › [Методы](БизнесПроцессОбъект.Имя_бизнес-процесса__Методы.md)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает бизнес-процесс в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для бизнес-процессов с поддержкой уникальности номеров в процессе записи, если нужно, проверяется уникальность номера.

Пример:

|  |
| --- |
| Попытка     БизнесПроцесс.Записать(); Исключение     Предупреждение(НСтр("ru = 'Невозможно записать бизнес-процесс.';"      + " en = 'Can't write the business process.'")); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
