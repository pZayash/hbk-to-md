РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.   
Имя поля записей набора, значения из которого будут выгружены в массив.

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Создает массив и копирует в него значения заданного поля записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Значения записей выгружаются в порядке индексов.

Пример:

|  |
| --- |
| МассивКурсов = НаборКурсов.ВыгрузитьКолонку("Курс"); |

Использование в версии:

Доступен, начиная с версии 8.0.
