ВыраженияПорядкаСхемыЗапроса (QuerySchemaOrderExpressions)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: По полю

Синтаксис:

Добавить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ДоступноеПолеСхемыЗапроса](ДоступноеПолеСхемыЗапроса_(QuerySchemaAvailableField).md), [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса_(QuerySchemaColumn).md), [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md).   
Добавляемое доступное поле, полученное из доступных полей у [Источник](ИсточникСхемыЗапроса.Источник_(QuerySchemaSource.Source).md), колонка, или выражение.

Описание варианта метода:

Добавить выражение порядка по полю.

Вариант синтаксиса: По строке

Синтаксис:

Добавить(<КолонкаИлиВыражение>)

Параметры:

<КолонкаИлиВыражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
По переданной строке сначала выполняется поиск колонки. Если колонка не найдена, создается выражение по указанной строки. Если колонка найдена или выражение получилось корректным, то оно добавляется в коллекцию. В противном случае вызывается исключение, содержащее описание ошибки.

Описание варианта метода:

Добавить выражение порядка по строке.

Возвращаемое значение:

Тип: [ВыражениеПорядкаСхемыЗапроса](ВыражениеПорядкаСхемыЗапроса_(QuerySchemaOrderExpression).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
