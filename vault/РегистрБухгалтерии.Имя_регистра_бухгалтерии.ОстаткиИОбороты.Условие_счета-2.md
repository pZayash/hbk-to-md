Условие счета (необязательный)

Конструкция языка запросов. Условие, по каким счетам получать данные. В условие можно использовать только поле [Счет](РегистрБухгалтерии.Имя_регистра_бухгалтерии.ОстаткиИОбороты.Счет.md).  
Параметр "Условие по счету" не существует, если при конфигурировании для плана счетов не указаны субконто.  
Значение по умолчанию: Неопределено.
