Модуль бота (Bot module)

ПриСозданииОбсужденияСистемыВзаимодействия (OnCreateCollaborationSystemConversation)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПриСозданииОбсужденияСистемыВзаимодействия(<ИдентификаторОбсуждения>, <ДополнительныеПараметры>)

Параметры:

<ИдентификаторОбсуждения>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения, в которое добавлен бот.

<ДополнительныеПараметры>

Тип: Произвольный.   
Параметры, заданные при создании бота.

Описание:

Обработчик, вызываемый при создании нового обсуждения с ботом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
