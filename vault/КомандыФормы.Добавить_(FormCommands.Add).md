КомандыФормы (FormCommands)

Добавить (Add)

Доступен, начиная с версии 8.2.

Синтаксис:

Добавить(<ИмяКоманды>)

Параметры:

<ИмяКоманды> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя добавляемой команды.

Возвращаемое значение:

Тип: [КомандаФормы](КомандаФормы_(FormCommand).md).   

Описание:

Добавляет команду в коллекцию.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
