Расширение формы документа (Document form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме(<РежимЗаписи>)

Параметры:

<РежимЗаписи> (необязательный)

Тип: [РежимЗаписиДокумента](РежимЗаписиДокумента_(DocumentWriteMode).md).   
Позволяет указать режим записи документа.  
Значение по умолчанию: [Запись](РежимЗаписиДокумента.Запись_(DocumentWriteMode.Write).md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - документ успешно записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает объект, редактируемый в данный момент в форме.

Доступность:

Толстый клиент.

См. также:

[Расширение формы документа](Расширение_формы_документа_(Document_form_extension).md), свойство [ИспользоватьРежимПроведения](Расширение_формы_документа.ИспользоватьРежимПроведения_(Document_form_extension.UsePostingMode).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
