ОбъектМетаданных: ОбщаяКоманда (MetadataObject: CommonCommand)

ТипПараметраКоманды (CommandParameterType)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов.md).   
Содержит тип значения, которое будет использовано в качестве параметра команды.  
Имеет смысл только для команд, размещаемых в формах.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
