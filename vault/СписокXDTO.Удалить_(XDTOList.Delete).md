# СписокXDTO.Удалить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md) › [СписокXDTO](СписокXDTO_(XDTOList).md) › [Методы](СписокXDTO__Методы.md)

СписокXDTO (XDTOList)

Удалить (Delete)

Доступен, начиная с версии 8.1.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Позиция удаляемого элемента данных.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Элемент данных, удаленный из списка.

Описание:

Удаляет элемент данных из указанной позиции.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
