ПараметрыСканированияДокументов (DocumentScanningParameters)

СканированиеСериями (ScanSeries)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Запустить сканирование серии документов:

- [Истина](lang__def_BooleanTrue.md) – сканирование выполняется, пока пользователь не закроет интерфейс сканирования.
- [Ложь](lang__def_BooleanFalse.md) - интерфейс закрывается после предварительного просмотра полученного документа.

Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
