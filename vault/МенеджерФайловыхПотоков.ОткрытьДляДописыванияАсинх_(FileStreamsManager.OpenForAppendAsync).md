МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьДляДописыванияАсинх (OpenForAppendAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьДляДописыванияАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ФайловыйПоток](ФайловыйПоток_(FileStream).md) для файла с указанным именем.

Описание:

Асинхронная функция. Открывает существующий файл для записи в конец. Если файл не существует, то создается новый файл. Запись в существующий файл производится с конца файла. Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, толстый клиент.
