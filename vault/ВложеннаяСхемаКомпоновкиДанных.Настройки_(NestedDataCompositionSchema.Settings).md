ВложеннаяСхемаКомпоновкиДанных (NestedDataCompositionSchema)

Настройки (Settings)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md).   
Содержит настройки, применяемые к настройкам вложенного объекта при добавлении этого объекта в настройки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
