СтраницыПанели (PanelPages)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Заголовок>, <Значение>, <КартинкаЗаголовка>)

Параметры:

<Имя> (необязательный)

Тип: Строка.   
Имя добавляемой страницы.

<Заголовок> (необязательный)

Тип: Строка.   
Текст заголовка добавляемой страницы. Если страницы панели показываются в виде закладок, то определяет текст на закладке.

<Значение> (необязательный)

Тип: Произвольный.   
Значение, связанное со страницей.

<КартинкаЗаголовка> (необязательный)

Тип: [Картинка](Картинка.md).   
Картинка, отображаемая на заголовке закладки.

Возвращаемое значение:

Тип: [СтраницаПанели](СтраницаПанели.md).   

Описание:

Добавляет страницу в конец коллекции страниц панели.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Страницы = ЭлементыФормы.ОсновнаяПанель.Страницы; СтраницаПанели = Страницы.Добавить(Имя, Представление); ЭлементыФормы.ОсновнаяПанель.ТекущаяСтраница = СтраницаПанели; |

Использование в версии:

Доступен, начиная с версии 8.0.
