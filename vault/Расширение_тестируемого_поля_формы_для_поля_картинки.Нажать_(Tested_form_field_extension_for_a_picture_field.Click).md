Расширение тестируемого поля формы для поля картинки (Tested form field extension for a picture field)

Нажать (Click)

Доступен, начиная с версии 8.3.2.

Синтаксис:

Нажать()

Описание:

Эмулирует нажатие для поля ввода в режиме отображения картинки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.2.
