ОформлениеЯчейки (CellAppearance)

ОтображатьФлажок (ShowCheckBox)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения флажка в ячейке.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл для колонок типа [Число](lang__def_Number.md) и [Булево](lang__Булево_(Boolean).md). Флажок со значением типа [Булево](lang__Булево_(Boolean).md) может иметь только два состояния ([Истина](lang__def_BooleanTrue.md) - установлен, [Ложь](lang__def_BooleanFalse.md) - снят), со значением типа [Число](lang__def_Number.md) может иметь три состояния (1 - установлен, 0 - снят, 2 - не определено).

Использование в версии:

Доступен, начиная с версии 8.0.
