ЭлементШкалыВремени (TimeScaleItem)

Формат (Format)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Пользовательский формат подписей. Если задан, то при выборе определенного масштаба подписи могут не поместиться целиком.  
Элемент с установленным форматом не может быть автоматически выключен.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
