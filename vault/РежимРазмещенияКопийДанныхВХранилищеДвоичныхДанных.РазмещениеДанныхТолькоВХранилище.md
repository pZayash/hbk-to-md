РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных (BinaryDataStorageDataCopiesPlacementMode)

РазмещениеДанныхТолькоВХранилище (DataPlacementToStorageOnly)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Тип: [РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных.md).   
Данные записываются в хранилище. При чтении данных сначала выполняется попытка чтения из хранилища, а в случае ошибки чтения - из служебной таблицы.  
Данный режим используется по умолчанию.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
