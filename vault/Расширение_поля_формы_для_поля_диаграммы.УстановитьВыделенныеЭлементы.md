Расширение поля формы для поля диаграммы (Form field extension for a chart field)

УстановитьВыделенныеЭлементы (SetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

УстановитьВыделенныеЭлементы(<МассивЭлементов>)

Параметры:

<МассивЭлементов> (необязательный)

Тип: [Массив](Массив.md).   
Массив выделяемых элементов.

Описание:

Устанавливает новое выделение. Элементы, тип которых отличается от [ТочкаДиаграммы](ТочкаДиаграммы.md), [СерияДиаграммы](СерияДиаграммы.md) , [ЗначениеДиаграммы](ЗначениеДиаграммы.md) игнорируются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
