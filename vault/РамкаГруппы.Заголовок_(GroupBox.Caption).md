# РамкаГруппы.Заголовок

РамкаГруппы (GroupBox)

Заголовок (Caption)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст подписи рамки группы.

Доступность:

Толстый клиент.

Примечание:

Для рамок типа [Одинарная](ТипРамкиЭлементаУправления.Одинарная_(ControlBorderType.Single).md) и Двойная заголовок отображается на самой рамке. Для рамок типа [Выпуклая](ТипРамкиЭлементаУправления.Выпуклая_(ControlBorderType.Embossed).md) и [Вдавленная](ТипРамкиЭлементаУправления.Вдавленная_(ControlBorderType.Indented).md) заголовок отображается внутри рамки.

Использование в версии:

Доступен, начиная с версии 8.0.
