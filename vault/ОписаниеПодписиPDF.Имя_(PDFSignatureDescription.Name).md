ОписаниеПодписиPDF (PDFSignatureDescription)

Имя (Name)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя электронной подписи PDF-документа.  
Если при подписании не указано, то генерируется автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
