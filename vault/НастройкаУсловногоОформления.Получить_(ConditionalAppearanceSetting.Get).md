НастройкаУсловногоОформления (ConditionalAppearanceSetting)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Номер>)

Параметры:

<Номер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Номер элемента настройки условного оформления.

Возвращаемое значение:

Тип: [ЭлементУправленияНастройкойУсловногоОформления](ЭлементУправленияНастройкойУсловногоОформления_(ConditionalAppearanceSettingControl).md).   

Описание:

Получает элемент управления настройкой условного оформления по номеру.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
