ПоследовательностьЗапись.<Имя последовательности> (SequenceRecord.<Sequence name>)

<Имя измерения> (<Dimension name>)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Набор свойств содержит значения измерений последовательности, с которыми документ зарегистрирован в последовательности. Доступ к значению осуществляется по имени, как оно задано в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В качестве типа допускается использовать любые типы, кроме [УникальныйИдентификатор](УникальныйИдентификатор.md), [ДвоичныеДанные](ДвоичныеДанные.md) и строки неограниченной длины.

Использование в версии:

Доступен, начиная с версии 8.0.
