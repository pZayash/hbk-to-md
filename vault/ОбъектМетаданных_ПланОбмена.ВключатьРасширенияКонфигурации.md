`ВключатьРасширенияКонфигурации`: `Булево`
<!-- signature:end -->

ОбъектМетаданных: ПланОбмена (MetadataObject: ExchangePlan)

ВключатьРасширенияКонфигурации (IncludeConfigurationExtensions)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то в распределенной информационной базе, организованной данным планом обмена, будут передаваться расширения конфигурации, у которых свойство [ИспользуетсяВРаспределеннойИнформационнойБазе](РасширениеКонфигурации.ИспользуетсяВРаспределеннойИнформационнойБазе.md) установлено в значение Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации.md), свойство [ИспользуетсяВРаспределеннойИнформационнойБазе](РасширениеКонфигурации.ИспользуетсяВРаспределеннойИнформационнойБазе.md)  
[РасширениеКонфигурации](РасширениеКонфигурации.md), свойство [ГлавныйУзел](РасширениеКонфигурации.ГлавныйУзел.md)  

Использование в версии:

Доступен, начиная с версии 8.3.12.
