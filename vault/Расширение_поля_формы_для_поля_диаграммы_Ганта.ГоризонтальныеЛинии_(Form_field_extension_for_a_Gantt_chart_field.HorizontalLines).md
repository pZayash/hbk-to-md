Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ГоризонтальныеЛинии (HorizontalLines)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Управляет отображением вертикальных линий в области построения диаграммы.   
Если содержит [Ложь](lang__def_BooleanFalse.md), то все вертикальные линии в области построения диаграммы не отображаются.   
Если содержит [Истина](lang__def_BooleanTrue.md), то все вертикальные линии в области построения диаграммы отображаются.   
Если содержит [Неопределено](lang__def_Undefined.md), то все вертикальные линии в области текстов точек и области таблицы данных не отображаются, а в области значений и интервалов – отображаются.   
Значение по умолчанию - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
