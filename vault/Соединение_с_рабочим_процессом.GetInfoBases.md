Соединение с рабочим процессом (IWorkingProcessConnection)

GetInfoBases (GetInfoBases)

Доступен, начиная с версии 8.1.

Синтаксис:

GetInfoBases()

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Одномерный массив [COMSafeArray](COMSafeArray.md), содержащий объекты [Информационная база](Информационная_база.md).

Описание:

Получает массив информационных баз, зарегистрированных в кластере серверов. Для выполнения не требуется аутентификации. При этом у информационной базы можно получить только значения свойств [Name](Информационная_база.Name.md) и [Descr](Информационная_база.Descr.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
