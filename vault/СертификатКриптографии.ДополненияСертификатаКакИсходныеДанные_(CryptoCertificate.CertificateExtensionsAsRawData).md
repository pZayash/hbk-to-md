СертификатКриптографии (CryptoCertificate)

ДополненияСертификатаКакИсходныеДанные (CertificateExtensionsAsRawData)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит список дополнений сертификата, каждый элемент которого представляет собой тип [КлючИЗначение](КлючИЗначение_(KeyAndValue).md), ключом которого является строка, содержащая идентификатор объекта (OID) дополнения сертификата, а значение - [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md). Значения расширений представлены в том виде, как они находятся в самом сертификате, без расшифровки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md), свойство [ДополненияСертификата](СертификатКриптографии.ДополненияСертификата_(CryptoCertificate.CertificateExtensions).md)  

Использование в версии:

Доступен, начиная с версии 8.3.27.
