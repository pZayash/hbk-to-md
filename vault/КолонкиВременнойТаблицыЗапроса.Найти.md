КолонкиВременнойТаблицыЗапроса (QueryTempTableColumns)

Найти (Find)

Доступен, начиная с версии 8.3.8.

Синтаксис:

Найти(<НаименованиеКолонки>)

Параметры:

<НаименованиеКолонки> (обязательный)

Тип: Строка.   
Имя колонки результата запроса.

Возвращаемое значение:

Тип: [КолонкаВременнойТаблицыЗапроса](КолонкаВременнойТаблицыЗапроса.md)Неопределено.   
Если колонка с таким именем отсутствует, возвращает Неопределено.

Описание:

Возвращает колонку временной таблицы по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если РезультатЗапроса.Колонки.Найти("Цена") <> Неопределено Тогда     ЦеныВЗапросе = Истина; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.3.8.
