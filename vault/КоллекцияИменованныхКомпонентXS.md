Элементы коллекции:

[ОпределениеПростогоТипаXS](ОпределениеПростогоТипаXS.md), [ОпределениеСоставногоТипаXS](ОпределениеСоставногоТипаXS.md), [ОпределениеГруппыАтрибутовXS](ОпределениеГруппыАтрибутовXS.md), [ОпределениеГруппыМоделиXS](ОпределениеГруппыМоделиXS.md), [ОбъявлениеАтрибутаXS](ОбъявлениеАтрибутаXS.md), [ОбъявлениеЭлементаXS](ОбъявлениеЭлементаXS.md), [ОпределениеОграниченияИдентичностиXS](ОпределениеОграниченияИдентичностиXS.md), [ОпределенияXPathXS](ОпределенияXPathXS.md), [ОбъявлениеНотацииXS](ОбъявлениеНотацииXS.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются компоненты схемы XML.  
Возможно обращение к компоненте схемы XML посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияИменованныхКомпонентXS.Количество.md)  
[Получить (Get)](КоллекцияИменованныхКомпонентXS.Получить.md)  

Описание:

Коллекция компонентов схемы XML, имеющих имя.

Доступность:

Сервер, толстый клиент, внешнее соединение.
