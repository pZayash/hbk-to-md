РежимОтображенияВыделения (SelectionShowMode)

ПриВыделенииНесколькихЯчеекПриАктивности (WhenMultipleCellsSelectedWhenActive)

Доступен, начиная с версии 8.3.10.

Описание:

Выделение отображается только, если элемент активен и если выбрано больше одной ячейки.

Использование в версии:

Доступен, начиная с версии 8.3.10.
