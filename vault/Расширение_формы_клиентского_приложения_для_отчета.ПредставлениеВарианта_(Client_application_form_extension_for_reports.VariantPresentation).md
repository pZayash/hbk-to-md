Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПредставлениеВарианта (VariantPresentation)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит представление варианта.   
Если не указан и при этом указан параметр формы [КлючВарианта](Расширение_формы_клиентского_приложения_для_отчета.КлючВарианта_(Client_application_form_extension_for_reports.VariantKey).md), то будет получено представление варианта для переданного ключа варианта.

Использование в версии:

Доступен, начиная с версии 8.2.
