# ЭлементDOM.ПолучитьАтрибут

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ЭлементDOM](ЭлементDOM_(DOMElement).md) › [Методы](ЭлементDOM__Методы.md)

ЭлементDOM (DOMElement)

ПолучитьАтрибут (GetAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Описание варианта метода:

Если значение отсутствует или атрибут не обнаружен, что будет возвращено значение [Неопределено](lang__def_Undefined.md).

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ПолучитьАтрибут(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен атрибута.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает значение атрибута по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если значение отсутствует или атрибут не обнаружен, что будет возвращено значение [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.1.
