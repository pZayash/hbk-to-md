# ИнтернетТекстыПочтовогоСообщения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Почта](Почта-2.md) › [ИнтернетПочта](ИнтернетПочта.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента коллекции.  

Методы:

[Добавить (Add)](ИнтернетТекстыПочтовогоСообщения.Добавить_(InternetMailTexts.Add).md)  
[Количество (Count)](ИнтернетТекстыПочтовогоСообщения.Количество_(InternetMailTexts.Count).md)  
[Очистить (Clear)](ИнтернетТекстыПочтовогоСообщения.Очистить_(InternetMailTexts.Clear).md)  
[Получить (Get)](ИнтернетТекстыПочтовогоСообщения.Получить_(InternetMailTexts.Get).md)  
[Удалить (Delete)](ИнтернетТекстыПочтовогоСообщения.Удалить_(InternetMailTexts.Delete).md)  

Описание:

Представляет собой коллекцию объектов типа [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ИнтернетПочтовоеСообщение](ИнтернетПочтовоеСообщение_(InternetMailMessage).md), свойство [Тексты](ИнтернетПочтовоеСообщение.Тексты_(InternetMailMessage.Texts).md)
