СоединениеИнформационнойБазы (InfoBaseConnection)

НомерСеанса (SessionNumber)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит номер сеанса, назначенного этому соединению.   
0, если соединению не назначен сеанс.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
