ВертикальнаяПрокруткаФормы (VerticalFormScroll)

ИспользоватьПриНеобходимости (UseIfNecessary)

Доступен, начиная с версии 8.3.3.

Описание:

Включает вертикальную прокрутку.   
Перед включением элементы формы максимально сжимаются по вертикали.

Использование в версии:

Доступен, начиная с версии 8.3.3.
