Надпись (Label)

Гиперссылка (Hyperlink)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак того, что надпись является гиперссылкой. Если значение свойства Истина, то текст надписи выводится с подчеркиванием, и при нажатии на надпись возникает событие [Нажатие](Надпись.Нажатие.md).

Доступность:

Толстый клиент.

См. также:

[Надпись](Надпись.md), событие [Нажатие](Надпись.Нажатие.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
