ЧтениеXML (XMLReader)

ЭтоСимвольныеДанные (IsCharacters)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Показывает, имеет ли текущий узел текстовое значение. Текстовое значение имеется у узлов: [Текст](ТипУзлаXML.Текст_(XMLNodeType.Text).md), [СекцияCDATA](ТипУзлаXML.СекцияCDATA_(XMLNodeType.CDATASection).md), [ПробельныеСимволы](ЧтениеXML.ПробельныеСимволы_(XMLReader.Space).md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
