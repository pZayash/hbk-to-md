Расширение поля формы для поля ввода (Form field extension for a text box)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает, нужно ли сейчас в поле ввода отображать особую отметку незаполненного или нет. Если свойство имеет значение Истина, то независимо от хранимых в поле ввода данных будет отображаться отметка, если Ложь, то независимо от хранимых в поле ввода данных отметка отображаться не будет.  
При начале редактирования свойство сбрасывается, даже если [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного.md) равно Ложь.  
  
Если свойство [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного.md) равно Истина, то при установке и формировании значения в поле ввода будет меняться состояние свойства [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного.md): если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного.md) будет установлено Истина, иначе - Ложь.  
Если для поля в свойствах метаданных "Автоотметка незаполненного" установлено "Авто":

- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Выдавать ошибку", то свойство принимает значение Истина, и соответственно, свойство [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) зависит от того, является ли значение реквизита значением по умолчанию;
- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Не проверять", то свойство [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) принимает значение Ложь, и соответственно, [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного.md) не зависит от того, является ли значение реквизита значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
