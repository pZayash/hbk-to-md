ОформлениеЯчейкиДинамическогоСписка (DynamicListCellAppearance)

Элементы (Items)

Доступен, начиная с версии 8.3.10.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗначенийПараметровКомпоновкиДанных](КоллекцияЗначенийПараметровКомпоновкиДанных_(DataCompositionParameterValueCollection).md).   
Коллекция значений оформления компоновки данных.  
Для оформления ячейки динамического списка используются следующие элементы:

- ЦветФона (BackColor) - тип значения [Цвет](Цвет_(Color).md);
- ЦветТекста (TextColor) - тип значения [Цвет](Цвет_(Color).md);
- Шрифт (Font) - тип значения [Шрифт](Шрифт_(Font).md);
- ГоризонтальноеПоложение (HorizontalAlign) - тип значения [ГоризонтальноеПоложение](ГоризонтальноеПоложение_(HorizontalAlign).md);
- Видимость (Visible) – тип значения [Булево](lang__Булево_(Boolean).md);
- Доступность (Enabled) – тип значения [Булево](lang__Булево_(Boolean).md);
- Отображать (Show) – тип значения [Булево](lang__Булево_(Boolean).md);
- Текст (Text) - тип значения [Строка](lang__def_String.md);
- Формат (Format) - тип значения [Строка](lang__def_String.md);
- Картинка (Picture) - тип значения [Картинка](Картинка_(Picture).md).

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
