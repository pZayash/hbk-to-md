ПолеТабличногоДокумента (SpreadsheetDocumentField)

ПриАктивизацииОбласти (OnActivateArea)

Не рекомендуется использовать, начиная с версии 8.3.14.

Рекомендуется использовать:

- [ПриАктивизации](ПолеТабличногоДокумента.ПриАктивизации_(SpreadsheetDocumentField.OnActivate).md)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриАктивизацииОбласти()

Описание:

Событие, возникающее при смене текущей области поля табличного документа.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.14.
