ЭлементПланировщика (PlannerItem)

Загрузить (Load)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Загрузить(<Источник>)

Параметры:

<Источник> (обязательный)

Тип: [Структура](Структура.md).   
Структура, из которой необходимо заполнить свойства элемента планировщика.

Описание:

Заполняет соответствующие свойства элемента планировщика из значений переданной структуры.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементПланировщика](ЭлементПланировщика.md), метод [Выгрузить](ЭлементПланировщика.Выгрузить.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.
