Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

Выбор (Selection)

Доступен, начиная с версии 8.2.

Синтаксис:

Выбор(<Значения>, <СтандартнаяОбработка>, <Дата>)

Параметры:

<Значения>

Тип: [Массив](Массив.md).   
Массив, содержащий выбранные объекты.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

<Дата>

Тип: Дата.   
Дата, соответствующая точке выбора. Применима для значения и интервала диаграммы Ганта. При выборе других объектов принимает значение Неопределено.

Описание:

Событие, возникающее при выборе диаграммы Ганта (например, при двойном щелчке мыши).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
