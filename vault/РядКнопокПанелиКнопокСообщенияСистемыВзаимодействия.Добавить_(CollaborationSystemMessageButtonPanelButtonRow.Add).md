РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRow)

Добавить (Add)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Добавить(<Действие>, <Текст>)

Параметры:

<Действие> (необязательный)

Тип: [ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonAction).md).   
Действие кнопки панели кнопок сообщения системы взаимодействия.  
Значение по умолчанию: [ОтправитьСообщение](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщение_(CollaborationSystemMessageButtonPanelButtonAction.SendMessage).md).

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Надпись на кнопке панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   

Описание:

Добавляет кнопку панели кнопок сообщения системы взаимодействия в конец коллекции и возвращает ее.   
Если метод вызван не в модуле бота, будет сгенерировано исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
