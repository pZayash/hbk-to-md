ПараметрыЗаписиИсторииДанных (WriteDataHistoryParameters)

ВыполнятьОбработкуПослеЗаписиВерсии (ExecuteAfterWriteVersionProcessing)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, добавляется ли информация о записи версии в очередь обработки после записи. Очередь обрабатывается при вызове метода [ВыполнитьОбработкуПослеЗаписиВерсий](МенеджерИсторииДанных.ВыполнитьОбработкуПослеЗаписиВерсий_(DataHistoryManager.ExecuteAfterWriteVersionsProcessing).md):

- [Истина](lang__def_BooleanTrue.md) – информация добавляется.
- [Ложь](lang__def_BooleanFalse.md) – информация не добавляется.

Значение данного свойства игнорируется в следующих случаях:

- Если свойству [Отказ](ПараметрыЗаписиИсторииДанных.Отказ_(WriteDataHistoryParameters.Cancel).md) установлено значение [Истина](lang__def_BooleanTrue.md).
- Если данные не используются в истории.

  
По умолчанию свойство имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
