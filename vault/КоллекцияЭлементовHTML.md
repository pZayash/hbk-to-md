Доступен, начиная с версии 8.2.

Элементы коллекции:

[ЭлементHTML](ЭлементHTML.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элемент коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

[Количество (Count)](КоллекцияЭлементовHTML.Количество.md)  
[Элемент (Item)](КоллекцияЭлементовHTML.Элемент.md)  

Описание:

Коллекция элементов HTML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
