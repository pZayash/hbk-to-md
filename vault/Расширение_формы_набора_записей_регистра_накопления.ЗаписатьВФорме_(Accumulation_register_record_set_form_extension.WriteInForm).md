# Расширение формы набора записей регистра накопления.ЗаписатьВФорме

Расширение формы набора записей регистра накопления (Accumulation register record set form extension)

ЗаписатьВФорме (WriteInForm)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписатьВФорме()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект записан; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Записывает набор, открытый в форме.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Если ЗаписатьВФорме() Тогда     Сообщить("Объект записан."); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
