ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПриАвтоматическомСозданииНовогоУзла (OnAutoCreateNewNode)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПриАвтоматическомСозданииНовогоУзла(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от создания нового узла задачи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то новый узел записан не будет.

Описание:

Вызывается при создании нового узла плана обмена для синхронизации данных между мобильным клиентом с автономным режимом и основным сервером.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
