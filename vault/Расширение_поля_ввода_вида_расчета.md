[БыстрыйВыбор (QuickChoice)](Расширение_поля_ввода_вида_расчета.БыстрыйВыбор.md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода.md), предназначенного для ввода ссылки на вид расчета.  
Расширение предоставляет возможность автоматического поиска и подбора элемента вида расчета по введенному тексту. По умолчанию поиск осуществляется по полям код и наименование.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
