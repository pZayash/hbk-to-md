ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

УстановитьВремя (SetTime)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьВремя(<РежимОпределенияВремени>)

Параметры:

<РежимОпределенияВремени> (необязательный)

Тип: [РежимАвтоВремя](РежимАвтоВремя.md).   
Устанавливает режим автоматического определения времени при записи новой задачи.  
Значение по умолчанию: [ТекущееИлиПоследним](РежимАвтоВремя.ТекущееИлиПоследним.md).

Описание:

Устанавливает режим определения времени при записи задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
