ПерерасчетНаборЗаписей.<Имя перерасчета> (RecalculationRecordSet.<Recalculation name>)

ЗагрузитьКолонку (LoadColumn)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗагрузитьКолонку(<ИмяПоля>, <Массив>)

Параметры:

<ИмяПоля> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор поля записи, в которую будут загружены значения из массива.

<Массив> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив, содержащий значения для заполнения колонки набора.

Описание:

Загружает значения из массива в заданное поле набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения из массива загружаются в поле в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.1.
