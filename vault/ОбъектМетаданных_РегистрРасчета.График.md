ОбъектМетаданных: РегистрРасчета (MetadataObject: CalculationRegister)

График (Schedule)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ОбъектМетаданных: РегистрСведений](ОбъектМетаданных_РегистрСведений.md).   
Описывает непериодический регистр сведений, в структуре которого есть хотя бы одно измерение типа Дата и хотя бы один ресурс типа Число.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если данное свойство не было задано на этапе конфигурирования, то оно имеет значение Неопределено, и регистр расчета не учитывает записи, протяженные во времени.

Использование в версии:

Доступен, начиная с версии 8.1.
