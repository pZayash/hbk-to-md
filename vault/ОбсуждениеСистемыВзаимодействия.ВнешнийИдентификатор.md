ОбсуждениеСистемыВзаимодействия (CollaborationSystemConversation)

ВнешнийИдентификатор (ExternalID)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит идентификатор обсуждения во внешней системе.  
Установка свойства возможна только для нового незаписанного обсуждения.  
  
Задание свойства имеет смысл для интеграций типа WhatsApp и WebHook.  
Для интеграций типа WhatsApp содержит номер телефона внешнего участника.  
Для интеграций типа Webhook - заданный при создании внешний идентификатор.  
Для интеграций типа Telegram и ВКонтакте - идентификатор чата в этих системах.  
  
Запись обсуждения с установленным свойством ВнешнийИдентификатор требует привилегированный режим.  
При получении первого сообщения в это обсуждение в список участников будет добавлен пользователь интеграции. Если список участников был пуст, то также будут добавлены "пользователи интеграции по умолчанию".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
