# ЭлементОбъектHTML.ЗначениеУзла

ЭлементОбъектHTML (HTMLObjectElement)

ЗначениеУзла (NodeValue)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит значение узла HTML. Значение узла присутствует не для каждого типа узла HTML.  
Узел Значение узла

- [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - [Значение](АтрибутDOM.Значение_(DOMAttribute.Value).md);
- [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - [ТекстHTML.Данные](ТекстHTML.Данные_(HTMLText.Data).md), [КомментарийHTML.Данные](КомментарийHTML.Данные_(HTMLComment.Data).md), содержит комментарий;
- [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - [Неопределено](lang__def_Undefined.md);
- [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) - [Неопределено](lang__def_Undefined.md);
- [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) - [ТекстHTML.Данные](ТекстHTML.Данные_(HTMLText.Data).md), [КомментарийHTML.Данные](КомментарийHTML.Данные_(HTMLComment.Data).md), содержит текст узла.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В качестве значения узла HTML используется значение узла DOM.

Использование в версии:

Доступен, начиная с версии 8.2.
