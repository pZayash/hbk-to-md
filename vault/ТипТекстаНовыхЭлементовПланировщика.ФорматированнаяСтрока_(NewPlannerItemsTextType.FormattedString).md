ТипТекстаНовыхЭлементовПланировщика (NewPlannerItemsTextType)

ФорматированнаяСтрока (FormattedString)

Доступен, начиная с версии 8.3.18.

Описание:

При создании элемента планировщика свойство [Текст](ЭлементПланировщика.Текст_(PlannerItem.Text).md) будет иметь тип [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).

Использование в версии:

Доступен, начиная с версии 8.3.18.
