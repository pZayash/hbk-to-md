# КоллекцияИндексовXBase.Найти

КоллекцияИндексовXBase (XBaseIndexesCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяИндекса>)

Параметры:

<ИмяИндекса> (обязательный)

Тип: [Строка](lang__def_String.md).   
Наименование индекса XBase, который требуется найти.

Возвращаемое значение:

Тип: [ИндексXBase](ИндексXBase_(XBaseIndex).md), [Неопределено](lang__def_Undefined.md).   
Если указанного индекса нет, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск индекса XBase по его имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
