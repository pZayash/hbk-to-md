ТочкаДиаграммыГанта (GanttChartPoint)

Уровень (Level)

Доступен, начиная с версии 8.0.

Синтаксис:

Уровень()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает уровень точки в иерархии.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Для точки, не имеющей родителя, уровень будет равен 0.

Использование в версии:

Доступен, начиная с версии 8.0.
