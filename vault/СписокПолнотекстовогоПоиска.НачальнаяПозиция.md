СписокПолнотекстовогоПоиска (FullTextSearchList)

НачальнаяПозиция (StartPosition)

Доступен, начиная с версии 8.1.

Синтаксис:

НачальнаяПозиция()

Возвращаемое значение:

Тип: Число.   

Описание:

Начальная позиция поиска. После вызова метода [ПерваяЧасть](СписокПолнотекстовогоПоиска.ПерваяЧасть.md) равна 0. При вызове методов [СледующаяЧасть](СписокПолнотекстовогоПоиска.СледующаяЧасть.md), [ПредыдущаяЧасть](СписокПолнотекстовогоПоиска.ПредыдущаяЧасть.md), начальная позиция будет меняться.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
