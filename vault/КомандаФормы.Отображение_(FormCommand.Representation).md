КомандаФормы (FormCommand)

Отображение (Representation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеКнопки](ОтображениеКнопки_(ButtonRepresentation).md).   
Указывает вид отображения связанной кнопки по умолчанию.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Изменение доступно только на сервере.

Использование в версии:

Доступен, начиная с версии 8.2.
