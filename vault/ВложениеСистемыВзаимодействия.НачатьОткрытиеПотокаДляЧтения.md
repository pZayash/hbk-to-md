ВложениеСистемыВзаимодействия (CollaborationSystemAttachment)

НачатьОткрытиеПотокаДляЧтения (BeginOpenStreamForRead)

Доступен, начиная с версии 8.3.14.

Синтаксис:

НачатьОткрытиеПотокаДляЧтения(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Поток> - Объект типа [Поток](Поток.md) для содержимого вложения.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает открытие потока для чтения содержимого вложения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.
