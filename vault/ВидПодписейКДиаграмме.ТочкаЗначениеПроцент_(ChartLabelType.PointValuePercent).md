ВидПодписейКДиаграмме (ChartLabelType)

ТочкаЗначениеПроцент (PointValuePercent)

Доступен, начиная с версии 8.0.

Описание:

В качестве подписи выводится представление точки плюс значение серии в точке плюс процент от общей суммы.

Использование в версии:

Доступен, начиная с версии 8.0.
