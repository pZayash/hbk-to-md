ИтераторУзловDOM (DOMNodeIterator)

Фильтр (Filter)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ФильтрУзловDOM](ФильтрУзловDOM_(DOMNodeFilter).md).   
Содержит отбор узлов DOM, участвующих в обходе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
