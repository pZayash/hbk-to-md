Расширение поля формы для поля ввода (Form field extension for a text box)

ОбновлениеТекстаРедактирования (EditTextUpdate)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОбновлениеТекстаРедактирования](ОбновлениеТекстаРедактирования.md).   
Управляет режимом обновления текста.  
Значение по умолчанию [Авто](ОбновлениеТекстаРедактирования.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В режиме совместимости конфигурации [Версия8\_2\_13](РежимСовместимости.Версия8_2_13.md) не влияет на поведение поля ввода управляемой формы.  
В остальных случаях влияет на то, в каких ситуациях будет обновляться текст в поле ввода при установке значения.

Использование в версии:

Доступен, начиная с версии 8.2.
