КнопкаКоманднойПанели (CommandBarButton)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СочетаниеКлавиш](СочетаниеКлавиш.md).   
Содержит сочетание клавиш, при нажатии на которые произойдет выполнение действия, связанного с кнопкой.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
