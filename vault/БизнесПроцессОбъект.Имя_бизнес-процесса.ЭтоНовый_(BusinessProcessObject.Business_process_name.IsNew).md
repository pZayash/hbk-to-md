БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - изменяется еще ни разу не записанный бизнес-процесс; [Ложь](lang__def_BooleanFalse.md) - бизнес-процесс уже записан.

Описание:

Определяет, записан ли бизнес-процесс в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Если ВыбБизнесПроцесс.ЭтоНовый() Тогда     ВыбБизнесПроцесс.Записать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
