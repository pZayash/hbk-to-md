Описание:

Содержит варианты элементов сообщения, для которых вызывается обработчик, установленный при вызове метода [ПодключитьОбработчикФормированияКоманд](МенеджерСистемыВзаимодействия.ПодключитьОбработчикФормированияКоманд_(CollaborationSystemManager.AttachGenerateCommandsHandler).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
