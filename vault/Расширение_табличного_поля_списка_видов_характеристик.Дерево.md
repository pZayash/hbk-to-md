Расширение табличного поля списка видов характеристик (Characteristic types list table box extension)

Дерево (Tree)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит способ отображения поля списка.  
Если свойство установить в Истина, то поле будет отображаться в виде дерева.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
