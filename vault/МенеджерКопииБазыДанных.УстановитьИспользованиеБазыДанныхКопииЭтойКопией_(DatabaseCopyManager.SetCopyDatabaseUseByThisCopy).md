МенеджерКопииБазыДанных (DatabaseCopyManager)

УстановитьИспользованиеБазыДанныхКопииЭтойКопией (SetCopyDatabaseUseByThisCopy)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьИспользованиеБазыДанныхКопииЭтойКопией()

Описание:

Устанавливает использование базы данных текущей копией информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.26.
