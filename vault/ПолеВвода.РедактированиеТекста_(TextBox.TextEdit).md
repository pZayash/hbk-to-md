ПолеВвода (TextBox)

РедактированиеТекста (TextEdit)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак возможности редактирования содержимого поля как обычного текста. Если свойство имеет значение [Ложь](lang__def_BooleanFalse.md) изменить содержимое можно лишь с помощью встроенных в элемент управления кнопок.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только для значений, которые могут редактироваться непосредственно в поле ввода.

Использование в версии:

Доступен, начиная с версии 8.0.
