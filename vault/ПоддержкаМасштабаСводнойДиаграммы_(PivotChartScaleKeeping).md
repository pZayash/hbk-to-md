Описание:

Содержит варианты способов поддержки масштаба (количества одновременно отображаемых значений) при изменении размеров области построения.  
Вариант [ВсеЗначения](ПоддержкаМасштабаСводнойДиаграммы.ВсеЗначения_(PivotChartScaleKeeping.AllValues).md) предусматривает представление в области построения всех значений. Прокрутка не применяется.  
Вариант [КоличествоЗначений](ПоддержкаМасштабаСводнойДиаграммы.КоличествоЗначений_(PivotChartScaleKeeping.ValuesCount).md) предусматривает представление области построения определенного количества значений, заданное соответствующим свойством. Если реальных значений больше, применяется прокрутка.  
Вариант [МинимальнаяШирина](ПоддержкаМасштабаДендрограммы.МинимальнаяШирина_(DendrogramScaleKeeping.MinimumWidth).md) предусматривает задание минимальной ширины значения (в символах). Если размер области построения не позволяет отобразить все значения с шириной каждого, не меньше заданной, применяется прокрутка.

Доступность:

Тонкий клиент, сервер, толстый клиент.

См. также:

[СводнаяДиаграмма](СводнаяДиаграмма_(PivotChart).md), свойство [ГоризонтальнаяПоддержкаМасштаба](СводнаяДиаграмма.ГоризонтальнаяПоддержкаМасштаба_(PivotChart.HorizontalScaleKeeping).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
