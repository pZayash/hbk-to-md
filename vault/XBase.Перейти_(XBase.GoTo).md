# XBase.Перейти

XBase (XBase)

Перейти (GoTo)

Доступен, начиная с версии 8.0.

Синтаксис:

Перейти(<НомерЗаписи>)

Параметры:

<НомерЗаписи> (обязательный)

Тип: [Число](lang__def_Number.md).   
Физический порядковый номер записи в базе данных.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - получена искомая запись; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Переходит на запись по ее физическому порядковому номеру в таблице базы данных.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Позволяет перейти на запись, помеченную удаленной, даже если не установлен режим просмотра удаленных записей.

Пример:

|  |
| --- |
| xB.Перейти(НомерЗаписи); |

Использование в версии:

Доступен, начиная с версии 8.0.
