ЗаписьДанных (DataWriter)

РазделительСтрок (LineSplitter)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Разделитель строк по-умолчанию. Это свойство влияет на поведение метода [ЗаписатьСтроку](ЗаписьДанных.ЗаписатьСтроку_(DataWriter.WriteLine).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗаписьДанных](ЗаписьДанных_(DataWriter).md), метод [ЗаписатьСтроку](ЗаписьДанных.ЗаписатьСтроку_(DataWriter.WriteLine).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
