# ПланОбменаМенеджер.<Имя плана обмена>.ПолучитьСсылку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаМенеджер.<Имя плана обмена>](ПланОбменаМенеджер.Имя_плана_обмена_(ExchangePlanManager.Exchange_plan_name).md) › [Методы](ПланОбменаМенеджер.Имя_плана_обмена__Методы.md)

ПланОбменаМенеджер.<Имя плана обмена> (ExchangePlanManager.<Exchange plan name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ПланОбменаСсылка.Имя_плана_обмена.УникальныйИдентификатор_(ExchangePlanRef.Exchange_plan_name.UUID).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

Доступен, начиная с версии 8.0.
