КолонкаТабличногоПоля (TableBoxColumn)

ТриСостоянияФлажка (CheckBoxThreeState)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что флажок в колонке может иметь три состояния: "Включен", "Выключен", "Не определено".  
[Истина](lang__def_BooleanTrue.md) - состояние разрешено.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если для колонки задано значение флажка.

Использование в версии:

Доступен, начиная с версии 8.0.
