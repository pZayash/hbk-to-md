ПотокВПамяти (MemoryStream)

НачатьПереход (BeginSeek)

Доступен, начиная с версии 8.3.9.

Синтаксис:

НачатьПереход(<ОписаниеОповещения>, <Смещение>, <НачальнаяПозиция>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <Позиция> - тип type [Число](lang__def_Number.md). Результат выполнения метода.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество байтов, на которое нужно передвинуть позицию в потоке.

<НачальнаяПозиция> (необязательный)

Тип: [ПозицияВПотоке](ПозицияВПотоке_(PositionInStream).md).   
Начальная позиция, от которой отсчитывается смещение.

Описание:

Начинает сдвиг текущей позиции потока на заданное количество байтов относительно начальной позиции. Если указано отрицательное смещение, позиция сдвигается в направлении к началу потока.  
Если изменение позиции недоступно ([ДоступноИзменениеПозиции](Поток.ДоступноИзменениеПозиции_(Stream.CanSeek).md) установлено в [Ложь](lang__def_BooleanFalse.md)), будет сгенерировано исключение.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Если результирующая позиция меньше 0, будет сгенерировано исключение.  
После записи в поток за пределами текущего размера, размер потока соответствующим образом увеличивается, а все байты после старого размера потока и до новой позиции остаются в неопределенном состоянии.
