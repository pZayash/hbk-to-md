ЗаписьДанных (DataWriter)

Доступен, начиная с версии 8.3.9.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Разделитель строк по-умолчанию. Это свойство влияет на поведение метода [ЗаписатьСтроку](ЗаписьДанных.ЗаписатьСтроку.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьСтроку](ЗаписьДанных.ЗаписатьСтроку.md)
