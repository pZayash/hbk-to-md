# РегистрСведенийМенеджер.<Имя регистра сведений>.ВыбратьПоРегистратору

РегистрСведенийМенеджер.<Имя регистра сведений> (InformationRegisterManager.<Information register name>)

ВыбратьПоРегистратору (SelectByRecorder)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьПоРегистратору(<Регистратор>)

Параметры:

<Регистратор> (обязательный)

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   
Ссылка на документ, являющийся регистратором для записей регистра.

Возвращаемое значение:

Тип: [РегистрСведенийВыборка.\<Имя регистра сведений\>](РегистрСведенийВыборка.Имя_регистра_сведений_(InformationRegisterSelection.Information_register_name).md).   

Описание:

Формирует выборку по регистратору записей регистра сведений. Применяется для регистров сведений, для которых в Конфигураторе установлен режим записи "Подчинение регистратору".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ВыборкаЦен =  РегистрыСведений.МаркетингЦен.ВыбратьПоРегистратору(ВыбРегистратор); Пока ВыборкаЦен.Следующий() Цикл     Сообщить("Дата " + ВыборкаЦен.Регистратор.ДатаДокумента +               " Номенклатура "+СокрЛП(ВыборкаЦен.Номенклатура) +              ", Конкурент "+  ВыборкаЦен.Конкурент ); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
