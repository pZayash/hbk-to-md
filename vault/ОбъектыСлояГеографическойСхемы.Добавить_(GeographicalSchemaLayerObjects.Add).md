ОбъектыСлояГеографическойСхемы (GeographicalSchemaLayerObjects)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   

Описание:

Добавляет новый объект в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
