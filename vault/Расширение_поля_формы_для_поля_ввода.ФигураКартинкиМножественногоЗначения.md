Расширение поля формы для поля ввода (Form field extension for a text box)

ФигураКартинкиМножественногоЗначения (MultipleValuePictureShape)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ФигураКартинкиМножественногоЗначенияПоляВвода](ФигураКартинкиМножественногоЗначенияПоляВвода.md).   
Устанавливает фигуру картинки множественного значения. Это свойство также влияет на внешний вид картинки в выпадающем списке поля ввода.  
Значение по умолчанию: [Авто](ФигураКартинкиМножественногоЗначенияПоляВвода.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
