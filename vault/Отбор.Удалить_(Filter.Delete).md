Отбор (Filter)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого элемента.

Описание:

Удаляет элемент отбора по индексу.  
Внимание! Метод не доступен для отбора динамических списков (свойство Отбор объектов [СправочникСписок.](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)<[Имя справочника](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)>, [ДокументСписок.](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)<[Имя документа](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // Очистим отбор  ОтборКоличество = ПостроительОтчета.Отбор.Количество(); Для Н = 1 По ОтборКоличество Цикл     ПостроительОтчета.Отбор.Удалить(ОтборКоличество - Н); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
