# Расширение табличного поля списка видов характеристик.ПередНачаломДобавления

Расширение табличного поля списка видов характеристик (Characteristic types list table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>, <Родитель>, <ЭтоГруппа>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то строка списка видов характеристик добавлена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

<Родитель>

Тип: [ПланВидовХарактеристикСписок.\<Имя плана видов характеристик\>](ПланВидовХарактеристикСписок.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesList.Chart_of_characteristic_types_name).md).   
Ссылка на элемент, который будет использован при добавлении в качестве родителя.

<ЭтоГруппа>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, что добавляется - группа или вид характеристики. [Истина](lang__def_BooleanTrue.md) - добавляется группа.

Описание:

Возникает перед добавлением строки списка видов характеристик. Данное событие для табличного поля вида характеристик возникает "вместо" одноименного события объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

Доступность:

Толстый клиент.

Примечание:

Расширяется набор параметров по сравнению с одноименным событием объекта [ТабличноеПоле](ТабличноеПоле_(TableBox).md).

См. также:

[ТабличноеПоле](ТабличноеПоле_(TableBox).md), событие [ПередНачаломДобавления](ТабличноеПоле.ПередНачаломДобавления_(TableBox.BeforeAddRow).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
