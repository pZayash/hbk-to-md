ПолеНабораДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetField)

НаборДанныхПроверкиИерархии (HierarchyCheckDataSet)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
В случае, если для поля условие [ВИерархии](ВидСравненияКомпоновкиДанных.ВИерархии_(DataCompositionComparisonType.InHierarchy).md) необходимо отрабатывать нестандартным образом, в этом свойстве указывается имя набора данных, в котором получаются данные для проверки принадлежности ссылки иерархии некоторого значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
