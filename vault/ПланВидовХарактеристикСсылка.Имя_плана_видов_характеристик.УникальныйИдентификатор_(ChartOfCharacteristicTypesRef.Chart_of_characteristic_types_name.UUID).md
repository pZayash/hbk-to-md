ПланВидовХарактеристикСсылка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesRef.<Chart of characteristic types name>)

УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик.ПолучитьСсылку_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name.GetRef).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Доступен, начиная с версии 8.0.
