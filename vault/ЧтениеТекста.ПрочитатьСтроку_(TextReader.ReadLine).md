ЧтениеТекста (TextReader)

ПрочитатьСтроку (ReadLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПрочитатьСтроку(<Разделитель>)

Параметры:

<Разделитель> (необязательный)

Тип: [Строка](lang__def_String.md).   
Разделитель строк. Если не указан, то совпадает с указанным при открытии файла.  
Значение по умолчанию: [ПС](Символы.ПС_(Chars.LF).md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
В случае, если читаемый файл завершен, то возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Считывает строку текста, ограниченную разделителем.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Строка нулевой длины не является признаком конца файла.

Пример:

|  |
| --- |
| Текст = Новый ЧтениеТекста("d:\win.txt", КодировкаТекста.ANSI); Стр = Текст.ПрочитатьСтроку(); Пока Стр <> Неопределено Цикл // строки читаются до символа перевода строки      Сообщить(Стр);     Стр = Текст.ПрочитатьСтроку(); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
