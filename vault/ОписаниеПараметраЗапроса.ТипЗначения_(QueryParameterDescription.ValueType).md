ОписаниеПараметраЗапроса (QueryParameterDescription)

ТипЗначения (ValueType)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Содержит рекомендуемый тип параметра, определяемый на основе анализа операции, производимой с параметром.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
