НаборУзлов (NodeSet)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Узел>)

Параметры:

<Узел> (обязательный)

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена.md).   
Ссылка на узел плана обмена, который надо удалить из набора узлов.

Описание:

Удаляет узел из набора узлов.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Узел = ПланыОбмена.ПО1.НайтиПоКоду("Маг11"); Об.ОбменДанными.Получатели.Удалить(Узел); |

Использование в версии:

Доступен, начиная с версии 8.0.
