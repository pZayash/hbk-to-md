Соединение с агентом сервера (IServerAgentConnection)

TurnOffWorkingProcess (TurnOffWorkingProcess)

Доступен, начиная с версии 8.3.27.

Синтаксис:

TurnOffWorkingProcess(<Кластер>, <Процесс>)

Параметры:

<Кластер> (необязательный)

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   
Кластер, в котором должен быть остановлен рабочий процесс. Список кластеров может быть получен при помощи метода [GetClusters](Соединение_с_агентом_сервера.GetClusters_(IServerAgentConnection.GetClusters).md).

<Процесс> (необязательный)

Тип: [Рабочий процесс](Рабочий_процесс_(IWorkingProcessInfo).md).   
Рабочий процесс, который требуется остановить.

Описание:

Выключает указанный рабочий процесс.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
