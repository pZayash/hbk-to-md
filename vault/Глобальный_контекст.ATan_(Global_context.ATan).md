Глобальный контекст (Global context)

ATan (ATan)

Доступен, начиная с версии 8.0.

Синтаксис:

ATan(<Число>)

Параметры:

<Число> (обязательный)

Тип: [Число](lang__def_Number.md).   
Аргумент функции.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат вычисления арктангенса в радианах.

Описание:

Вычисляет арктангенс от аргумента <Число>.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Pi = 3.1415926535897932; Аргумент = 1; Попытка     УголВГрадусах = 360\*ATan(Аргумент)/Pi/2;     Сообщить("Значение угла: " + Окр(УголВГрадусах,2)); Исключение     Сообщить("При значении аргумента Tan()" + Аргумент         + " выдается сообщение об ошибке:"         + Символы.ПС + """" + ОписаниеОшибки() + """"); КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.0.
