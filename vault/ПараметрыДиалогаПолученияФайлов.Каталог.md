ПараметрыДиалогаПолученияФайлов (GetFilesDialogParameters)

Каталог (Directory)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит значение текущего каталога при открытии диалога.  
В веб-клиенте параметр игнорируется.   
Значение по умолчанию: пустая строка.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
