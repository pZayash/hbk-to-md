# ДокументМенеджер.<Имя документа>.ПолучитьСсылку

ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](ДокументСсылка.Имя_документа.УникальныйИдентификатор_(DocumentRef.Document_name.UUID).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан или передано значение [Неопределено](lang__def_Undefined.md), то будет сформирована новая уникальная ссылка.

См. также:

[ДокументСсылка.\<Имя документа\>](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md), метод [УникальныйИдентификатор](ДокументСсылка.Имя_документа.УникальныйИдентификатор_(DocumentRef.Document_name.UUID).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
