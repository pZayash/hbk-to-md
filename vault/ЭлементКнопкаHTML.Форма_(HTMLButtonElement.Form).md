ЭлементКнопкаHTML (HTMLButtonElement)

Форма (Form)

Доступен, начиная с версии 8.2.

Синтаксис:

Форма()

Возвращаемое значение:

Тип: [ЭлементФормаHTML](ЭлементФормаHTML_(HTMLFormElement).md).   

Описание:

Возвращает объект - форму, к которой относится данная кнопка.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
