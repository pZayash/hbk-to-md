ОбъектМетаданных: Задача (MetadataObject: Task)

СозданиеПриВводе (CreateOnInput)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Режим создания при вводе. Если значение = [Авто](СозданиеПриВводе.Авто.md), то участие в механизме создания при вводе определяется значением свойства СозданиеПриВводе объекта метаданных этого типа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
