ХранилищеСертификатовКриптографии (CryptoCertificateStore)

НайтиПоСубъекту (FindBySubject)

Доступен, начиная с версии 8.2.

Синтаксис:

НайтиПоСубъекту(<СтрокаПоиска>)

Параметры:

<СтрокаПоиска> (обязательный)

Тип: Строка.   
Строка поиска субъекта.  
Данные субъекта должны содержать переданную в этом параметре строку.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Содержит объекты [СертификатКриптографии](СертификатКриптографии.md).

Описание:

Осуществляет поиск сертификатов криптографии по переданному имени субъекта, которому выдан сертификат.  
  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с криптографией. В веб-клиенте метод поддерживается только в браузере Internet Explorer, если в конфигурации используется режим синхронных вызовов расширений платформы и внешних компонент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
