ГенераторСлучайныхПаролей (RandomPasswordGenerator)

СлучайныйПароль (RandomPassword)

Доступен, начиная с версии 8.3.22.

Синтаксис:

СлучайныйПароль(<ДлинаПароля>, <ЗапоминающийсяПароль>)

Параметры:

<ДлинаПароля> (необязательный)

Тип: Число.   
Длина генерируемого пароля.  
  
По умолчанию используется минимальная длина паролей, установленная в информационной базе, но не менее 7.

<ЗапоминающийсяПароль> (необязательный)

Тип: Булево.   
Значение данного параметра определяет, будут ли сгенерированы простые для запоминания пароли или нет.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: Строка.   

Описание:

Генерирует случайный пароль с указанными параметрами.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.22.
