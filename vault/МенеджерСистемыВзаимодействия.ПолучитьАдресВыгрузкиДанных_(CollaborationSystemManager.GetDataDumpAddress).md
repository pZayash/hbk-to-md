МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьАдресВыгрузкиДанных (GetDataDumpAddress)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьАдресВыгрузкиДанных(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Ссылка (адрес) на скачивание выгрузки данных, если она расположена в хранилище, доступ к которому предоставлен по протоколу Amazon S3 (https://aws.amazon.com/ru/s3/). (Полученная ссылка имеет ограниченное время жизни, которое определяется настройками хранилища S3).  
Или путь к файлу в файловой системе сервера системы взаимодействия.

Описание:

Получает адрес выгрузки данных системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
