УдалениеОбъекта (ObjectDeletion)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md).   
Содержит параметры обмена данными, которые будут использованы при удалении объекта, на который показывает ссылка.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
