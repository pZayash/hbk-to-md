Описание:

Описывает действие элемента планировщика.  
Примечание. Действия в элементе отображаются только при наличии свободного пространства.   
В противном случае действия будут скрыты и будут отображаться только в окнах редактирования.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен . Имя типа XDTO:
