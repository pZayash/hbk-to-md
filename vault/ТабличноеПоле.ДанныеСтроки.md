ТабличноеПоле (TableBox)

ДанныеСтроки (RowData)

Доступен, начиная с версии 8.0.

Синтаксис:

ДанныеСтроки(<Строка>)

Параметры:

<Строка> (обязательный)

Строка табличного поля, данные которой необходимо получить. Тип параметра зависит от типа редактируемого в табличном поле значения.

Возвращаемое значение:

Тип возвращаемого значения зависит от типа редактируемого в табличном поле значения. Для динамических списков данных, выбираемых из базы данных, возвращает объект, в свойствах которого содержатся значения ячеек указанной строки.

Описание:

Получает данные указанной строки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
