Расширение формы клиентского приложения для бизнес-процесса (Client application form extension for business processes)

Ключ (Key)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Тип: [БизнесПроцессСсылка.\<Имя бизнес-процесса\>](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md).   
Ключ объекта.   
В случае отсутствия или неверного значения создается новый объект с использованием остальных параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
