Соединение с агентом сервера (IServerAgentConnection)

UpdateWorkingServer (UpdateWorkingServer)

Доступен, начиная с версии 8.1.

Синтаксис:

UpdateWorkingServer(<Кластер>, <Сервер>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Определяет кластер, в котором должно быть модифицировано описание рабочего сервера. Список кластеров может быть получен при помощи метода [GetClusters](Соединение_с_агентом_сервера.GetClusters.md).

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Содержит модифицированное описание сервера. Его значение должно быть получено методом [GetWorkingServers](Соединение_с_агентом_сервера.GetWorkingServers.md), после чего в нем может быть изменен список диапазонов IP-портов для динамического распределения при помощи методов [InsertPortRange](Рабочий_сервер.InsertPortRange.md) и [ErasePortRange](Рабочий_сервер.ErasePortRange.md).

Описание:

Модифицирует описание зарегистрированного в кластере рабочего сервера. Можно модифицировать только список диапазонов портов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
