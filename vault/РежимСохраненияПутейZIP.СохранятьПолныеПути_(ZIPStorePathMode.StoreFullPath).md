РежимСохраненияПутейZIP (ZIPStorePathMode)

СохранятьПолныеПути (StoreFullPath)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [СохранятьПолныеПути](РежимСохраненияПутейФайлаАрхива.СохранятьПолныеПути_(ArchiveFileStorePathMode.StoreFullPath).md)

Описание:

При помещении в архив сохраняются полные пути файлов.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
