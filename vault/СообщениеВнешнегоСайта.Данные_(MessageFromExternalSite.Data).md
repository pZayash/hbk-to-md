СообщениеВнешнегоСайта (MessageFromExternalSite)

Данные (Data)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит данные сообщения от внешнего сайта. Если от внешнего сайта были отправлены данные не строкового типа, то они будут преобразованы к типу [Строка](lang__def_String.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
