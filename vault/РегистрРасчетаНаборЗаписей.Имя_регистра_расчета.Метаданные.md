Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: РегистрРасчета](ОбъектМетаданных_РегистрРасчета.md).   

Описание:

Предоставляет доступ к объекту описания метаданных регистра расчета. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.РегистрыРасчета.РасчетСреднегоЗаработка.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
