ВидПодписейКДиаграмме (ChartLabelType)

СерияЗначениеПроцент (SeriesValuePercent)

Доступен, начиная с версии 8.0.

Описание:

В качестве подписи выводится строка, состоящая из имени серии, значения в серии и процента от общей суммы.

Использование в версии:

Доступен, начиная с версии 8.0.
