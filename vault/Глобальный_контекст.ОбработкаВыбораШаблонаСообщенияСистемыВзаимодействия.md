Глобальный контекст (Global context)

ОбработкаВыбораШаблонаСообщенияСистемыВзаимодействия (CollaborationSystemMessageTemplateChoiceProcessing)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ОбработкаВыбораШаблонаСообщенияСистемыВзаимодействия(<Обсуждение>, <СтандартнаяОбработка>)

Параметры:

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения.

<СтандартнаяОбработка>

Тип: Булево.   
Если после вызова обработчика параметр <СтандартнаяОбработка> содержит значение Истина, отображается системный диалог отправки шаблонного сообщения.  
Значение по умолчанию: Истина.

Описание:

Вызывается при выборе команды "Отправить шаблонное сообщение".  
Если после завершения работы процедуры обработки события <СтандартнаяОбработка> установлено в Истина, отображается системный диалог отправки шаблонного сообщения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.23.
