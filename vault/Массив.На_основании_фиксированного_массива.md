Массив (Array)

На основании фиксированного массива

Доступен, начиная с версии 8.3.2.

Синтаксис:

Новый Массив(<Массив>)

Параметры:

<Массив> (необязательный)

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Фиксированный массив.

Описание:

Создаваемый массив заполняется значениями из исходного фиксированного массива. К элементам исходного массива применяется рекурсивное преобразование, т.е. элементы фиксированного массива, в свою очередь являющиеся фиксированными массивами, будут помещены в массив с преобразованием к типу [Массив](Массив_(Array).md).

Использование в версии:

Доступен, начиная с версии 8.3.2.
