# НастройкаОформления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md) › [Оформление](Оформление-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ЭлементНастройкиОформления](ЭлементНастройкиОформления_(AppearanceSettingItem).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс элемента коллекции (нумерация с 0).  

Свойства:

[\<Имя настройки\> (\<Setting name \>)](НастройкаОформления.Имя_настройки_(AppearanceSetting.Setting_name_).md)  

Конструкторы:

[По умолчанию](НастройкаОформления.По_умолчанию.md)  

Описание:

Коллекция элементов типа [ЭлементНастройкиОформления](ЭлементНастройкиОформления_(AppearanceSettingItem).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[ЭлементУсловногоОформления](ЭлементУсловногоОформления_(ConditionalAppearanceItem).md), свойство [Оформление](ЭлементУсловногоОформления.Оформление_(ConditionalAppearanceItem.Appearance).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
