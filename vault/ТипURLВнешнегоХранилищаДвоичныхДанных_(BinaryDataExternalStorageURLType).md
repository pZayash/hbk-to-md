Описание:

Тип URL-адреса внешнего хранилища двоичных данных. Для хранилищ S3 поддерживаются схемы Path-style и Virtual-hosted-style.

Доступность:

Сервер.

Пример:

|  |
| --- |
| ТипURLS3 = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3; ТипСБакетом = ТипURLВнешнегоХранилищаДвоичныхДанных                     .ХостS3СБакетомВСоставеПути; |

Использование в версии:

Доступен, начиная с версии 8.3.24.
