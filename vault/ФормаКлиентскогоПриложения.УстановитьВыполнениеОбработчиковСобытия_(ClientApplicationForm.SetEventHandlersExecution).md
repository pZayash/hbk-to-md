ФормаКлиентскогоПриложения (ClientApplicationForm)

УстановитьВыполнениеОбработчиковСобытия (SetEventHandlersExecution)

Доступен, начиная с версии 8.3.6.

Синтаксис:

УстановитьВыполнениеОбработчиковСобытия(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Ложь](lang__def_BooleanFalse.md) - после завершения текущего обработчика выполнение цепочки обработчиков будет прервано.  
Если установлено [Истина](lang__def_BooleanTrue.md) - выполнение цепочки обработчиков будет продолжено.

Описание:

Управляет выполнением цепочки обработчиков события.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

Метод может быть вызван только в формах конфигураций расширений.

Использование в версии:

Доступен, начиная с версии 8.3.6.
