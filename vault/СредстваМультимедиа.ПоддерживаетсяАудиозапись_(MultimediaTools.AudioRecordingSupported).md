СредстваМультимедиа (MultimediaTools)

ПоддерживаетсяАудиозапись (AudioRecordingSupported)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоддерживаетсяАудиозапись()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
Возвращает [Истина](lang__def_BooleanTrue.md) в следующих случаях:

- на Android - всегда,
- на iOS - всегда,
- на Windows Phone - при наличии микрофона.

Описание:

Определяет, поддерживается ли аудиозапись системой.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [СделатьАудиозапись](СредстваМультимедиа.СделатьАудиозапись_(MultimediaTools.MakeAudioRecording).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.
