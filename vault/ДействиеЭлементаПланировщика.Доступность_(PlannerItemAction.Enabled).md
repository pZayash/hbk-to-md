# ДействиеЭлементаПланировщика.Доступность

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md) › [ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика_(PlannerItemAction).md) › [Свойства](ДействиеЭлементаПланировщика__Свойства.md)

ДействиеЭлементаПланировщика (PlannerItemAction)

Доступность (Enabled)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Истина](lang__def_BooleanTrue.md), то при нажатии на элемент будет вызвано событие [НажатиеНаДействиеПланировщика](Расширение_поля_формы_для_поля_планировщика.НажатиеНаДействиеПланировщика_(Form_field_extension_for_a_planner.PlannerActionClick).md).   
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.18.
