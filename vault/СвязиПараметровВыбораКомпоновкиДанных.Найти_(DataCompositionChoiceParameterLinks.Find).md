СвязиПараметровВыбораКомпоновкиДанных (DataCompositionChoiceParameterLinks)

Найти (Find)

Доступен, начиная с версии 8.2.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор параметра.

Возвращаемое значение:

Тип: [ПараметрВыбораКомпоновкиДанных](ПараметрВыбораКомпоновкиДанных_(DataCompositionChoiceParameter).md), [Неопределено](lang__def_Undefined.md).   
Если значение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
