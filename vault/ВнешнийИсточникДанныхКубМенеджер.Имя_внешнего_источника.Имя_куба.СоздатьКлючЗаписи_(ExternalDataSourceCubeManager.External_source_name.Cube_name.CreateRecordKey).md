ВнешнийИсточникДанныхКубМенеджер.<Имя внешнего источника>.<Имя куба> (ExternalDataSourceCubeManager.<External source name>.<Cube name>)

СоздатьКлючЗаписи (CreateRecordKey)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СоздатьКлючЗаписи(<ЗначенияКлюча>)

Параметры:

<ЗначенияКлюча> (обязательный)

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая значения для заполнения свойств ключа записи.  
Идентификаторы элементов структуры должны соответствовать идентификаторам ключевых полей.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхКубКлючЗаписи.\<Имя внешнего источника\>.\<Имя куба\>](ВнешнийИсточникДанныхКубКлючЗаписи.Имя_внешнего_источника.Имя_куба_(ExternalDataSourceCubeRecordKey.External_source_name.Cube_name).md).   

Описание:

Создает ключ записи по переданной структуре.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
