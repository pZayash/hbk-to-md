КонтрольныеТочкиИтоговСхемыЗапроса (QuerySchemaTotalCalculationFields)

НайтиПоИмениКолонки (FindByColumnName)

Доступен, начиная с версии 8.3.7.

Синтаксис:

НайтиПоИмениКолонки(<ИмяКолонки>)

Параметры:

<ИмяКолонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки для поиска.

Возвращаемое значение:

Тип: [КонтрольнаяТочкаИтоговСхемыЗапроса](КонтрольнаяТочкаИтоговСхемыЗапроса_(QuerySchemaTotalCalculationField).md).   

Описание:

Выполняет поиск по свойству [ИмяКолонки](КонтрольнаяТочкаИтоговСхемыЗапроса.ИмяКолонки_(QuerySchemaTotalCalculationField.ColumnName).md) контрольных точек итогов схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
