ОбъектМетаданных: КаналСервисаИнтеграции (MetadataObject: IntegrationServiceChannel)

НаправлениеСообщения (MessageDirection)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [НаправлениеСообщенияКаналаСервисаИнтеграции](НаправлениеСообщенияКаналаСервисаИнтеграции_(IntegrationServiceChannelMessageDirection).md).   
Если [Отправка](НаправлениеСообщенияКаналаСервисаИнтеграции.Отправка_(IntegrationServiceChannelMessageDirection.Send).md) - канал сервиса интеграции доступен для отправки сообщений и не доступен для получения сообщений. Если для канала установлен обработчик полученных сообщений, при использовании конфигурации выводится сообщение об ошибке.  
Если [Получение](НаправлениеСообщенияКаналаСервисаИнтеграции.Получение_(IntegrationServiceChannelMessageDirection.Receive).md) - канал сервиса интеграции доступен для получения сообщения и не доступен для отправки сообщения. Если для канала не установлен обработчик полученных сообщений, при использовании конфигурации выводится сообщение об ошибке, а при вызове метода [ОтправитьСообщение](КаналСервисаИнтеграцииМенеджер.Имя_канала_сервиса_интеграции.ОтправитьСообщение_(IntegrationServiceChannelManager.Integration_service_channel_name.SendMessage).md) генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
