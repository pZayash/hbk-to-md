ОбъектМетаданных: WSСсылка (MetadataObject: WSReference)

WSОпределение (WSDefinition)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
Для получения WsОпределения можно использовать метод [ПолучитьWSОпределения](WSСсылкаМенеджер.Имя_WS-Ссылки.ПолучитьWSОпределения_(WSReferenceManager.WS-reference_name.GetWSDefinitions).md)() менеджера соответствующей WS-Ссылки.

Использование в версии:

Доступен, начиная с версии 8.1.
