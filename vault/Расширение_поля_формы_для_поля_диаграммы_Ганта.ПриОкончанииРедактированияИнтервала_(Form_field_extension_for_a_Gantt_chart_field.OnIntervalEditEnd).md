Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

ПриОкончанииРедактированияИнтервала (OnIntervalEditEnd)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриОкончанииРедактированияИнтервала(<Интервал>, <ОтменаРедактирования>)

Параметры:

<Интервал>

Тип: [ИнтервалДиаграммыГанта](ИнтервалДиаграммыГанта_(GanttChartInterval).md).   
Интервал, который был изменен. Передаётся временное значение. Если в обработчике параметр <ОтменаРедактирования> не будет выставлен в [Истина](lang__def_BooleanTrue.md), то после выполнения обработчика данные из временного объекта будут перенесены в реквизит.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) приведет к отмене изменения интервала.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при окончании интерактивного редактирования интервала.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
