ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

РежимАвтонумерацииОбъектов (ObjectAutonumerationMode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [РежимАвтонумерацииОбъектов](РежимАвтонумерацииОбъектов_(ObjectAutonumerationMode).md).   
Определяет, переиспользовать или нет автоматически полученные номера объектов, если они не записаны в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
