МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьОбсуждения (GetConversations)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьОбсуждения(<Отбор>)

Параметры:

<Отбор> (обязательный)

Тип: [ОтборОбсужденийСистемыВзаимодействия](ОтборОбсужденийСистемыВзаимодействия.md).   
Содержит настройки отбора и сортировки.

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив объектов типа [ОбсуждениеСистемыВзаимодействия](ОбсуждениеСистемыВзаимодействия.md).

Описание:

Возвращает список обсуждений, удовлетворяющих заданным критериям отбора.

- Если [КонтекстноеОбсуждение](ОтборОбсужденийСистемыВзаимодействия.КонтекстноеОбсуждение.md) установлено в значение Ложь и <ТекущийПользовательЯвляетсяУчастником> установлен в Истина, то возвращаются неконтекстные обсуждения, удовлетворяющие отбору, в которых текущий пользователь входит в список участников.
- Если [КонтекстноеОбсуждение](ОтборОбсужденийСистемыВзаимодействия.КонтекстноеОбсуждение.md) установлено в значение Ложь и <ТекущийПользовательЯвляетсяУчастником> установлен в Ложь, то возвращаются все неконтекстные обсуждения, удовлетворяющие отбору. Требуется использование привилегированного режима или наличие права "Администрирование данных" у текущего пользователя информационной базы. В противном случае, будет сгенерировано исключение.
- Если [КонтекстноеОбсуждение](ОтборОбсужденийСистемыВзаимодействия.КонтекстноеОбсуждение.md) установлено в значение Истина, то значение свойства <ТекущийПользовательЯвляетсяУчастником> игнорируется.
- Если [КонтекстноеОбсуждение](ОтборОбсужденийСистемыВзаимодействия.КонтекстноеОбсуждение.md) установлено в значение Истина, и [КонтекстОбсуждения](ОтборОбсужденийСистемыВзаимодействия.КонтекстОбсуждения.md) задан, то возвращается обсуждение (одно) переданного объекта контекста, если оно удовлетворяет заданному отбору и у текущего пользователя есть права (с учетом ограничения прав доступа на уровне записей (RLS)) на переданный объект контекста. Если установлен привилегированный режим, то проверка прав и RLS не производится.
- Если [КонтекстноеОбсуждение](ОтборОбсужденийСистемыВзаимодействия.КонтекстноеОбсуждение.md) установлено в значение Истина, и [КонтекстОбсуждения](ОтборОбсужденийСистемыВзаимодействия.КонтекстОбсуждения.md) не задан, то возвращаются все контекстные обсуждения, удовлетворяющие отбору. Если не установлен привилегированный режим, то будет сгенерировано исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.

Описание изменено в версии 8.3.21.
