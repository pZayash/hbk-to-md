АдминистрированиеРабочийСервер (AdministrationWorkServer)

ПрименитьНастройкиСервисов (ApplyServiceSettings)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПрименитьНастройкиСервисов()

Описание:

Применяет настройки сервисов кластера.   
Для активации настроек требуется перезагрузка кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
