ПоляКолонкиСхемыЗапроса (QuerySchemaColumnFields)

Получить (Get)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md), [ВложеннаяТаблицаСхемыЗапроса](ВложеннаяТаблицаСхемыЗапроса_(QuerySchemaNestedTable).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
