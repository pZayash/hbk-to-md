ОбъектМетаданных: ОбщийМодуль (MetadataObject: CommonModule)

КлиентОбычноеПриложение (ClientOrdinaryApplication)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Если это свойство установлено в значение Истина, то процедуры и функции общего модуля могут быть использованы в толстом клиенте в режиме обычного приложения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
