НастройкиОбработкиОшибок (ErrorProcessingSettings)

ДополнительнаяИнформацияОтчета (AdditionalReportInformation)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Произвольная дополнительная информация, которая будет добавлена в отчет.  
Если в настройках для пользователя пустая строка, то значение параметра будет получено из общих настроек.  
Данная строка может содержать имя/код базы и/или области, либо любую другую указанную разработчиком информацию.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
