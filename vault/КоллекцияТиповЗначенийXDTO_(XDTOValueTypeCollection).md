# КоллекцияТиповЗначенийXDTO

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ТипЗначенияXDTO](ТипЗначенияXDTO_(XDTOValueType).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются типы значений XDTO.  
Возможно обращение к типу значения XDTO посредством оператора [...]. В качестве аргумента передается индексу (нумерация с 0).  

Методы:

[Количество (Count)](КоллекцияТиповЗначенийXDTO.Количество_(XDTOValueTypeCollection.Count).md)  
[Получить (Get)](КоллекцияТиповЗначенийXDTO.Получить_(XDTOValueTypeCollection.Get).md)  

Описание:

Коллекция типов значений модели XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
