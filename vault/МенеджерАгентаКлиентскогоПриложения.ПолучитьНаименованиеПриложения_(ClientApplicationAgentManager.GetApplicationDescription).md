МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

ПолучитьНаименованиеПриложения (GetApplicationDescription)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьНаименованиеПриложения()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   

Описание:

Возвращает текущее наименование приложения в агенте клиентского приложения. Если агент не запущен, то возвращает [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
