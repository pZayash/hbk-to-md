КоллекцияДействийЭлементаПланировщика (PlannerItemActionCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Текст>, <Картинка>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить действие элемента планировщика.

<Значение> (необязательный)

Тип: Произвольный.   
Значение действия элемента планировщика.

<Текст> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Текст, отображаемый в действии элемента планировщика.

<Картинка> (необязательный)

Тип: [Картинка](Картинка.md).   
Картинка, отображаемая в действии элемента планировщика.

Возвращаемое значение:

Тип: [ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика.md).   

Описание:

Вставляет действие в коллекцию действий элемента планировщика по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
