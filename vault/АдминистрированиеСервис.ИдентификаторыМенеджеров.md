АдминистрированиеСервис (AdministrationService)

ИдентификаторыМенеджеров (ManagerIDs)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив.md).   
Содержит массив объектов [УникальныйИдентификатор](УникальныйИдентификатор.md), на которые назначен данный сервис. Каждый из сервисов может быть назначен только на один менеджер кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
