ТокенДоступа (AccessToken)

ВремяСоздания (CreationTime)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Числовое значение времени создания токена доступа в формате UnixTime (количество секунд, прошедших с полуночи 01.01.1970). Соответствует ключам 'iat' и 'nbf'.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
