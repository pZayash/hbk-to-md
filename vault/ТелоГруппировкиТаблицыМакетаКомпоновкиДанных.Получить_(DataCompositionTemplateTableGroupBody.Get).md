Получить (Get)

Доступен, начиная с версии 8.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

Возвращаемое значение:

Тип: [ТелоГруппировкиТаблицыМакетаКомпоновкиДанных](ТелоГруппировкиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroupBody).md), [ГруппировкаТаблицыМакетаКомпоновкиДанных](ГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableGroup).md), [ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных](ИерархическаяГруппировкаТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableHierarchicalGroup).md), [ЗаписиТаблицыМакетаКомпоновкиДанных](ЗаписиТаблицыМакетаКомпоновкиДанных_(DataCompositionTemplateTableRecords).md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
