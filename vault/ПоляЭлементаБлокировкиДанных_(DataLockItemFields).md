# ПоляЭлементаБлокировкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Управление блокировкой данных](Управление_блокировкой_данных.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеЭлементаБлокировкиДанных](ПолеЭлементаБлокировкиДанных_(DataLockItemField).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элемент коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Количество (Count)](ПоляЭлементаБлокировкиДанных.Количество_(DataLockItemFields.Count).md)  
[Получить (Get)](ПоляЭлементаБлокировкиДанных.Получить_(DataLockItemFields.Get).md)  

Описание:

Коллекция полей блокировки данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ЭлементБлокировкиДанных](ЭлементБлокировкиДанных_(DataLockItem).md), свойство [Поля](ЭлементБлокировкиДанных.Поля_(DataLockItem.Fields).md)  

Использование в версии:

Доступен, начиная с версии 8.1.
