Значения

[Авто (Auto)](ПолучениеЭлементаДанных.Авто.md)

[Принять (Accept)](ПолучениеЭлементаДанных.Принять.md)  

Описание:

Содержит варианты значений выходных параметров обработчиков событий [ПриПолученииДанныхОтГлавного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтГлавного.md) и [ПриПолученииДанныхОтПодчиненного](ПланОбменаОбъект.Имя_плана_обмена.ПриПолученииДанныхОтПодчиненного.md) объектов [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>. Данное значение определяет поведение системы при получении элементов данных из сообщения распределенной информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером.
