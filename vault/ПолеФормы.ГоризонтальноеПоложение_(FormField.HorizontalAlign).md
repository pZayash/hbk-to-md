ПолеФормы (FormField)

ГоризонтальноеПоложение (HorizontalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента_(ItemHorizontalLocation).md).   
Определяет способ горизонтального расположения текста в колонке. Данное свойство имеет смысл только для полей, расположенных в таблице (представляющих собой колонки).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
