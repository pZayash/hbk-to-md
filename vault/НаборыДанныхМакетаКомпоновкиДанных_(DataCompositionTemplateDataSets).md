Элементы коллекции:

[НаборДанныхЗапросМакетаКомпоновкиДанных](НаборДанныхЗапросМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetQuery).md), [НаборДанныхОбъектМакетаКомпоновкиДанных](НаборДанныхОбъектМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetObject).md), [НаборДанныхОбъединениеМакетаКомпоновкиДанных](НаборДанныхОбъединениеМакетаКомпоновкиДанных_(DataCompositionTemplateDataSetUnion).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются наборы.  
Возможно обращение к набору посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
