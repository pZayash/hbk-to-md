КолонкаСписка (ListColumn)

АвтоУдаление (AutoDelete)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, необходимо ли удалять колонку при удалении ее из табличного поля.  
[Истина](lang__def_BooleanTrue.md) - необходимо удалять.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
