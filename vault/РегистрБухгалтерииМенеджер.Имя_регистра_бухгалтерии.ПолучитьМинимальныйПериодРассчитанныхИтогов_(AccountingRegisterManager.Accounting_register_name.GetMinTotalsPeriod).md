РегистрБухгалтерииМенеджер.<Имя регистра бухгалтерии> (AccountingRegisterManager.<Accounting register name>)

ПолучитьМинимальныйПериодРассчитанныхИтогов (GetMinTotalsPeriod)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМинимальныйПериодРассчитанныхИтогов()

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   
Начало периода.

Описание:

Получает минимальный период рассчитанных итогов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме совместимости конфигурации [Версия8\_3\_2](РежимСовместимости.Версия8_3_2_(CompatibilityMode.Version8_3_2).md) и ниже возвращает 01.01.0001.

Пример:

|  |
| --- |
| ВыбДата = Регистры.Остатки.ПолучитьМинимальныйПериодРассчитанныхИтогов(); |
