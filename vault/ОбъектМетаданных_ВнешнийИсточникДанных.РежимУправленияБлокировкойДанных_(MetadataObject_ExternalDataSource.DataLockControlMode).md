ОбъектМетаданных: ВнешнийИсточникДанных (MetadataObject: ExternalDataSource)

РежимУправленияБлокировкойДанных (DataLockControlMode)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [РежимУправленияБлокировкойДанныхПоУмолчанию](РежимУправленияБлокировкойДанныхПоУмолчанию_(DefaultDataLockControlMode).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
