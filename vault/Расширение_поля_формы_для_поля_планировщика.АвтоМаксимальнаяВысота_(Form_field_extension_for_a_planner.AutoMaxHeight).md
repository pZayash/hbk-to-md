Расширение поля формы для поля планировщика (Form field extension for a planner)

АвтоМаксимальнаяВысота (AutoMaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет использование автоматического ограничения максимальной высоты элемента.  
[Истина](lang__def_BooleanTrue.md) - автоматическое ограничение используется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
