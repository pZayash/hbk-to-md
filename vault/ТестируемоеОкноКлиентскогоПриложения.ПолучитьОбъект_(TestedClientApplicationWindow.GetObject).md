ТестируемоеОкноКлиентскогоПриложения (TestedClientApplicationWindow)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ТестируемоеОкноКлиентскогоПриложения](ТестируемоеОкноКлиентскогоПриложения_(TestedClientApplicationWindow).md), [ТестируемаяФорма](ТестируемаяФорма_(TestedForm).md), [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы_(TestedFormGroup).md), [ТестируемаяКнопкаФормы](ТестируемаяКнопкаФормы_(TestedFormButton).md), [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы_(TestedFormTable).md), [ТестируемоеПолеФормы](ТестируемоеПолеФормы_(TestedFormField).md), [ТестируемаяДекорацияФормы](ТестируемаяДекорацияФормы_(TestedFormDecoration).md), [Неопределено](lang__def_Undefined.md).   
Первый найденный объект. Если объект не найден, генерируется исключительная ситуация.

Описание:

Выполняет поиск объекта среди иерархии подчиненных объектов по указанным параметрам.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
