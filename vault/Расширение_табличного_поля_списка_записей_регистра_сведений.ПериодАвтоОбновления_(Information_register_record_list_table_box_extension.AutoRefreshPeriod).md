# Расширение табличного поля списка записей регистра сведений.ПериодАвтоОбновления

Расширение табличного поля списка записей регистра сведений (Information register record list table box extension)

ПериодАвтоОбновления (AutoRefreshPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Задает период (в секундах), с которым будет производиться автоматическое обновление списка. Не может быть меньше 5 секунд. Если свойство [АвтоОбновление](Расширение_табличного_поля_списка_записей_регистра_сведений.АвтоОбновление_(Information_register_record_list_table_box_extension.AutoRefresh).md) установлено как [Ложь](lang__def_BooleanFalse.md), то обновление не производится.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
