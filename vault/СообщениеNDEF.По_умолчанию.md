СообщениеNDEF (MessageNDEF)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый СообщениеNDEF(<Записи>)

Параметры:

<Записи> (необязательный)

Тип: [ФиксированныйМассив](ФиксированныйМассив.md), Неопределено.   
Фиксированный массив записей. Элементами массива являются объекты, представляющие NDEF-запись, например, [ТекстоваяЗаписьNDEF](ТекстоваяЗаписьNDEF.md), [URIЗаписьNDEF](URIЗаписьNDEF.md), и др.  
Если передан пустой массив или Неопределено, будет создано пустое сообщение - сообщение с единственной записью [ПустаяЗаписьNDEF](ПустаяЗаписьNDEF.md). Запись на метку пустого сообщения означает очистку данных на метке.

Использование в версии:

Доступен, начиная с версии 8.3.23.
