СообщениеNDEF (MessageNDEF)

По умолчанию

Доступен, начиная с версии 8.3.23.

Синтаксис:

Новый СообщениеNDEF(<Записи>)

Параметры:

<Записи> (необязательный)

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md), [Неопределено](lang__def_Undefined.md).   
Фиксированный массив записей. Элементами массива являются объекты, представляющие NDEF-запись, например, [ТекстоваяЗаписьNDEF](ТекстоваяЗаписьNDEF_(TextRecordNDEF).md), [URIЗаписьNDEF](URIЗаписьNDEF_(URIRecordNDEF).md), и др.  
Если передан пустой массив или Неопределено, будет создано пустое сообщение - сообщение с единственной записью [ПустаяЗаписьNDEF](ПустаяЗаписьNDEF_(EmptyRecordNDEF).md). Запись на метку пустого сообщения означает очистку данных на метке.

Использование в версии:

Доступен, начиная с версии 8.3.23.
