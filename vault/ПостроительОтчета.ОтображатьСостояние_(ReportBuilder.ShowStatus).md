ПостроительОтчета (ReportBuilder)

ОтображатьСостояние (ShowStatus)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости вывода в панель состояния информации о выводе отчета.   
[Истина](lang__def_BooleanTrue.md) - отображается.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значение по умолчанию [Истина](lang__def_BooleanTrue.md) (выводить информацию).

Использование в версии:

Доступен, начиная с версии 8.0.
