МеткаNDEF (TagNDEF)

ПоддерживаетсяБлокировкаЗаписиНаМеткуАсинх (TagWriteLockSupportedAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПоддерживаетсяБлокировкаЗаписиНаМеткуАсинх()

Возвращаемое значение:

Тип: Булево.   
Истина - если физическая метка перезаписываемая и может быть заблокирована на запись,  
Ложь - в противном случае.

Описание:

Проверяет, что физическая метка NFC поддерживает блокировку на запись.  
После блокировки запись на метку будет недоступна.  
Если произошла ошибка, будет сгенерировано исключение категории [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МеткаNDEF](МеткаNDEF.md), метод [ЗаблокироватьЗаписьНаМеткуАсинх](МеткаNDEF.ЗаблокироватьЗаписьНаМеткуАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.23.
