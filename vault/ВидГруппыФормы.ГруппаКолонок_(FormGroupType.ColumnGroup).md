ВидГруппыФормы (FormGroupType)

ГруппаКолонок (ColumnGroup)

Доступен, начиная с версии 8.2.

Описание:

Группа колонок. Группа данного вида может содержать поля и группы вида [ГруппаКнопок](ВидГруппыФормы.ГруппаКнопок_(FormGroupType.ButtonGroup).md).

Использование в версии:

Доступен, начиная с версии 8.2.
