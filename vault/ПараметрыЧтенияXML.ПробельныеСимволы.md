ПараметрыЧтенияXML (XMLReaderSettings)

ПробельныеСимволы (Space)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПробельныеСимволыXML](ПробельныеСимволыXML.md).   
Содержит значение по умолчанию атрибута xml:space, используемое при чтении XML. При создании объекта содержит значение [ПоУмолчанию](ПробельныеСимволыXML.ПоУмолчанию.md).  
Влияет на свойство ПробельныеСимволы объекта [ЧтениеXML](ЧтениеXML.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
