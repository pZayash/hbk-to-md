ВариантУправленияВысотойТаблицы (TableHeightControlVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Вариант управления высотой таблицы определяется автоматически:

- Для динамического списка - [ВСтрокахФормы](ВариантУправленияВысотойТаблицы.ВСтрокахФормы_(TableHeightControlVariant.UseHeightInFormRows).md),
- в остальных случаях - [ПоСодержимому](ВариантУправленияВысотойТаблицы.ПоСодержимому_(TableHeightControlVariant.UseContentHeight).md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
