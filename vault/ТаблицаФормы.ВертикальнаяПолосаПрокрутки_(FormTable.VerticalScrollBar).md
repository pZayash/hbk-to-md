ТаблицаФормы (FormTable)

ВертикальнаяПолосаПрокрутки (VerticalScrollBar)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеПолосыПрокрутки](ИспользованиеПолосыПрокрутки_(ScrollBarUse).md).   
Режим использования вертикальной полосы прокрутки.  
Для таблицы формы, отображающей динамический список, управляет отображением кнопок прокрутки списка.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
