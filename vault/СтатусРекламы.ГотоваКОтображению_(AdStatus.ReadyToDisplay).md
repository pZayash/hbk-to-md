СтатусРекламы (AdStatus)

ГотоваКОтображению (ReadyToDisplay)

Доступен, начиная с версии 8.3.14.

Описание:

Реклама загружена. При вызове метода [ПоказатьВидеообъявлениеСВознаграждением](МенеджерОтображенияРекламы.ПоказатьВидеообъявлениеСВознаграждением_(AdRepresentationManager.ShowRewardedVideo).md)или [ПоказатьПолноэкраннуюРекламу](МенеджерОтображенияРекламы.ПоказатьПолноэкраннуюРекламу_(AdRepresentationManager.ShowFullScreenAd).md) реклама будет показана без задержки.

Использование в версии:

Доступен, начиная с версии 8.3.14.
