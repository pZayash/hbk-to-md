ПеречислениеМенеджер.<Имя перечисления> (EnumManager.<Enumeration name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [ПеречислениеСсылка.\<Имя перечисления\>](ПеречислениеСсылка.Имя_перечисления.md).   

Описание:

Получает пустое значение ссылки на перечисление данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Пример:

|  |
| --- |
| ВидКонтрагента = Перечисления.ВидыКонтрагентов.ПустаяСсылка(); |

Использование в версии:

Доступен, начиная с версии 8.0.
