ЗаписьPDF (PDFWriter)

НачатьЗаписьВложений (BeginWriteAttachments)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьЗаписьВложений(<ОписаниеОповещения>, <Вложения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Вложения> (обязательный)

Тип: [КоллекцияВложенийPDF](КоллекцияВложенийPDF.md).   
Объект, содержащий вложения PDF-документов.

Описание:

Дописывает вложения в существующий PDF-документ. Если в документе уже существуют вложения с именами, указанными в коллекции вложений, то вложения в документе заменяются на вложения из коллекции.  
Если документ записывается в формате PDF или PDF/A-3 и указан параметр <Вложения>, то вложения добавляются в документ в формате PDF и PDF/A-3.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF.md), метод [ЗаписатьВложения](ЗаписьPDF.ЗаписатьВложения.md)  
[ЗаписьPDF](ЗаписьPDF.md), метод [ЗаписатьВложенияАсинх](ЗаписьPDF.ЗаписатьВложенияАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.21.
