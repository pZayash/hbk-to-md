НастройкиВосстановленияПароля (PasswordRecoverySettings)

ДлительностьБлокировкиЗапросаОбновленияКодаПодтверждения (VerificationCodeRefreshRequestLockDuration)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет длительность блокировки запроса отправки нового кода для подтверждения пароля (в секундах) -  
целое число в диапазоне от 1 до 999999999. Значение по умолчанию: 30.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
