Глобальный контекст (Global context)

УстановитьИспользованиеДополнительныхИндексов (SetAdditionalIndexesUsage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьИспользованиеДополнительныхИндексов(<ИспользованиеДополнительныхИндексов>)

Параметры:

<ИспользованиеДополнительныхИндексов> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - устанавливает использование дополнительных индексов,  
[Ложь](lang__def_BooleanFalse.md) - отключает использование дополнительных индексов.

Описание:

Устанавливает признак использования дополнительных индексов.  
При вызове метода дополнительные индексы не создаются/не удаляются.  
Создание/удаление индексов выполняется при следующих действиях:

- тестирование и исправление информационной базы в режиме "Проверка и включение функциональности уровня КОРП",
- реструктуризация объектов метаданных,
- тестирование и исправление в режиме "Реструктуризация таблиц информационной базы".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
