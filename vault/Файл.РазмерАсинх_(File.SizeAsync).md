Файл (File)

РазмерАсинх (SizeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазмерАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Число](lang__def_Number.md), размер файла в байтах.

Описание:

Асинхронная функция. Получает размер файла в байтах.  
Применима только к файлам, которые реально существуют.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами
