ОбъектМетаданных: ОбщийРеквизит (MetadataObject: CommonAttribute)

Индексирование (Indexing)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Индексирование](Индексирование_(Indexing).md).   
Вид дополнительных индексов, создаваемых для работы с данными объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для всех реквизитов, кроме реквизитов, описывающих отчеты, обработки и табличные части отчетов и обработок.

Использование в версии:

Доступен, начиная с версии 8.2.
