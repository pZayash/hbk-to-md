МаксимумСерий (MaxSeries)

Ограничено (Limited)

Доступен, начиная с версии 8.0.

Описание:

Отображается не более определенного количества серий. Количество серий определяется свойством [МаксимумСерийКоличество](Диаграмма.МаксимумСерийКоличество.md) объекта [Диаграмма](Диаграмма.md).

Использование в версии:

Доступен, начиная с версии 8.0.
