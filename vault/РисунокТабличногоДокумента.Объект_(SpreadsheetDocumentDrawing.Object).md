РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

Объект (Object)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Произвольный.   
Содержит объект, содержащийся в рисунке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо только для рисунков типа [Объект](ОписаниеОтображаемогоОбъектаPDF.Объект_(PDFRepresentationObjectDescription.Object).md), [Диаграмма](ТипРисункаТабличногоДокумента.Диаграмма_(SpreadsheetDocumentDrawingType.Chart).md), [ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md), [СводнаяДиаграмма](СводнаяДиаграмма_(PivotChart).md), [Дендрограмма](Дендрограмма_(Dendrogram).md) и [ГеографическаяСхема](ГеографическаяСхема_(GeographicalSchema).md).

Использование в версии:

Доступен, начиная с версии 8.0.
