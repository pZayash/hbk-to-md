АдминистрированиеМенеджерКластера (AdministrationClusterManager)

ГлавныйМенеджер (MainManager)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Указывает, является ли менеджер главным или дополнительным менеджером кластера:

- Истина - главный менеджер;
- Ложь - дополнительный менеджер (не главный менеджер).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.

Описание изменено в версии 8.3.23.
