Глобальный контекст (Global context)

ХранилищеВнешнихДанныхНавигационныхСсылок (URLExternalDataStorage)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер.md), [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища.md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища.md)>.   
Предоставляет доступ к хранилищу внешних данных навигационных ссылок.  
При использовании стандартного хранилища, все пользователи имеют доступ ко всем настройкам на чтение.   
При использовании стандартного хранилища для каждой настройки, сохраняемой в хранилище, записывается дата сохранения настройки. Ее можно получить через свойство [ДатаИзменения](СтандартноеХранилищеНастроекВыборка.ДатаИзменения.md). В режиме совместимости [Версия8\_3\_18](РежимСовместимости.Версия8_3_18.md) и ниже это свойство всегда содержит значение Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
