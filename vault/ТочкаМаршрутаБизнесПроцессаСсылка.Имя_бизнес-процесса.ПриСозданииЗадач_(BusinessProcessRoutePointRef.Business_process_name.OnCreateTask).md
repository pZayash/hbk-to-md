# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПриСозданииЗадач

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПриСозданииЗадач (OnCreateTask)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриСозданииЗадач(<ТочкаМаршрута>, <ФормируемыеЗадачи>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, на которой происходит создание задач.

<ФормируемыеЗадачи>

Тип: [Массив](Массив_(Array).md).   
Массив формируемых задач (объектов типа [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)<[Имя задачи](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)>).

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления задачи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись задач произведена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает после выполнения стандартной или переопределенной в обработчике события [ПередСозданиемЗадач](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПередСозданиемЗадач_(BusinessProcessRoutePointRef.Business_process_name.BeforeCreateTasks).md) механики генерации списка задач. Обработчик этого события может "донастроить" автоматически сформированные задачи или удалить некоторые из них, а также добавить к ним новые задачи.   
Запись списка задач будет осуществлена после завершения обработчика.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md) и [ВложенныйБизнесПроцесс](ВидТочкиМаршрутаБизнесПроцесса.ВложенныйБизнесПроцесс_(BusinessProcessRoutePointType.SubBusinessProcess).md).  
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода [Сообщить](Глобальный_контекст.Сообщить_(Global_context.Message).md).

Использование в версии:

Доступен, начиная с версии 8.0.
