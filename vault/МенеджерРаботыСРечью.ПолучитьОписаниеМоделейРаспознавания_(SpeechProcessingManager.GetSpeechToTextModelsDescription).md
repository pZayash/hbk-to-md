МенеджерРаботыСРечью (SpeechProcessingManager)

ПолучитьОписаниеМоделейРаспознавания (GetSpeechToTextModelsDescription)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьОписаниеМоделейРаспознавания(<ПараметрыМодели>)

Параметры:

<ПараметрыМодели> (необязательный)

Тип: [ПараметрыМоделиРаспознаванияРечи](ПараметрыМоделиРаспознаванияРечи_(SpeechToTextModelParameters).md), [Неопределено](lang__def_Undefined.md).   
Если [Неопределено](lang__def_Undefined.md) - будет получено описание всех доступных моделей.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив объектов типа [ОписаниеМоделиРаспознаванияРечи](ОписаниеМоделиРаспознаванияРечи_(SpeechToTextModelDescription).md).

Описание:

Возвращает описания всех доступных моделей распознавания. Если задан отбор в виде параметров модели, то возвращает описания всех доступных моделей с учетом отбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
