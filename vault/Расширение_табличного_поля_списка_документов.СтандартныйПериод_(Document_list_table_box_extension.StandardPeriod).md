Расширение табличного поля списка документов (Document list table box extension)

СтандартныйПериод (StandardPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [НастройкаПериода](НастройкаПериода_(PeriodSettings).md).   
Содержит объект [НастройкаПериода](НастройкаПериода_(PeriodSettings).md), в соответствии с которым осуществляется настройка периода списка документов при открытии формы, содержащей данное табличное поле.

Доступность:

Толстый клиент.

См. также:

[НастройкаПериода](НастройкаПериода_(PeriodSettings).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
