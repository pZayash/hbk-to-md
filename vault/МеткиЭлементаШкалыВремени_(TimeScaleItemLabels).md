**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Диаграмма Ганта](Диаграмма_Ганта.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени_(TimeScaleItemLabel).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются метки.  
Возможно обращение к меткам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Добавить (Add)](МеткиЭлементаШкалыВремени.Добавить_(TimeScaleItemLabels.Add).md)  
[Количество (Count)](МеткиЭлементаШкалыВремени.Количество_(TimeScaleItemLabels.Count).md)  
[Очистить (Clear)](МеткиЭлементаШкалыВремени.Очистить_(TimeScaleItemLabels.Clear).md)  
[Удалить (Delete)](МеткиЭлементаШкалыВремени.Удалить_(TimeScaleItemLabels.Delete).md)  

Описание:

Коллекция непериодических меток элемента шкалы времени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

[ЭлементШкалыВремени](ЭлементШкалыВремени_(TimeScaleItem).md), свойство [Метки](ЭлементШкалыВремени.Метки_(TimeScaleItem.Labels).md)
