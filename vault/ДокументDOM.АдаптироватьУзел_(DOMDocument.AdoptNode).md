# ДокументDOM.АдаптироватьУзел

ДокументDOM (DOMDocument)

АдаптироватьУзел (AdoptNode)

Доступен, начиная с версии 8.1.

Синтаксис:

АдаптироватьУзел()

Возвращаемое значение:

Тип: [ДокументDOM](ДокументDOM_(DOMDocument).md), [Неопределено](lang__def_Undefined.md).   
Адаптированный узел или [Неопределено](lang__def_Undefined.md), если адаптация не удалась.

Описание:

Выполняет адаптацию (заимствование) узла другого документа. Производит попытку заимствования узла из другого документа в данный.  
Если адаптация поддерживается для узла, то производится замена документа-владельца у данного узла, всех его дочерних узлов и атрибутов рекурсивно.  
Если узел имеет родительский узел, то вначале он удаляется из родительского узла.  
Правила заимствования для типов узлов:  
[Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - документ-владелец устанавливается в [Неопределено](lang__def_Undefined.md), свойство [Указан](АтрибутDOM.Указан_(DOMAttribute.Specified).md) устанавливается в [Истина](lang__def_BooleanTrue.md), заимствуются все потомки узла;  
[ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md) - все потомки фрагмента рекурсивно заимствуются;  
[Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - не может быть адаптирован;  
[ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md) - не может быть адаптирован;  
[Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) - атрибуты с установленным свойством [Указан](АтрибутDOM.Указан_(DOMAttribute.Specified).md) заимствуются, для нового узла-элемента создаются атрибуты, имеющие значения по умолчанию. Все потомки рекурсивно заимствуются;  
[Сущность](ТипУзлаDOM.Сущность_(DOMNodeType.Entity).md) - не может быть адаптирован;  
[СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность_(DOMNodeType.EntityReference).md) - заимствуется только данный узел. Если в данном документе определена сущность с таким же именем, то формируется значение сущности для заимствованной ссылки;  
[Нотация](ТипУзлаDOM.Нотация_(DOMNodeType.Notation).md) - не может быть адаптирован;  
[ИнструкцияОбработки](ТипУзлаDOM.ИнструкцияОбработки_(DOMNodeType.ProcessingInstruction).md), [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA_(DOMNodeType.CDATASection).md), [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - заимствуются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключений:

- попытка адаптации узлов [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md), [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md);
- текущий документ адаптируемого узла в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
