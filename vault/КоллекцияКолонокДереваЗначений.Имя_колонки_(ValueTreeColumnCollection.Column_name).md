КоллекцияКолонокДереваЗначений (ValueTreeColumnCollection)

<Имя колонки> (<Column name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КолонкаДереваЗначений](КолонкаДереваЗначений_(ValueTreeColumn).md).   
Набор свойств содержит значения колонок дерева значений. Доступ к значению осуществляется по имени колонки. Имена свойств совпадают с именами колонок дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
