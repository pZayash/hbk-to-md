ОбъектМетаданных: КаналСервисаИнтеграции (MetadataObject: IntegrationServiceChannel)

ОбработчикПолученияСообщения (ReceivingMessageHandler)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя обработчика в модуле СервисИнтеграцииМенеджер.<Имя сервиса интеграции>, вызываемого при получении сообщения.  
Обработчик выполняется в транзакции и имеет следующие параметры:

- Сообщение (Message) - тип [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md),
- Отказ (Cancel) - тип [Булево](lang__Булево_(Boolean).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
