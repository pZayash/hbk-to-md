ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения (ClientApplicationInterfaceContentSettingsGroup)

Получить (Get)

Доступен, начиная с версии 8.3.4.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента.

Возвращаемое значение:

Тип: [ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ЭлементНастройкиСоставаИнтерфейсаКлиентскогоПриложения.md), [ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения](ГруппаНастройкиСоставаИнтерфейсаКлиентскогоПриложения.md).   

Описание:

Получает элемент коллекции по индексу. Работает аналогично оператору [].

Доступность:

Сервер, толстый клиент.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.3.4.
