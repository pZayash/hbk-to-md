Глобальный контекст (Global context)

УдалитьНедопустимыеСимволыXML (DeleteDisallowedXMLCharacters)

Доступен, начиная с версии 8.3.20.

Синтаксис:

УдалитьНедопустимыеСимволыXML(<СтрокаСимволов>, <Версия>)

Параметры:

<СтрокаСимволов> (обязательный)

Тип: Строка.   
Строка, в которой необходимо проверить символы.

<Версия> (необязательный)

Тип: Строка.   
Версия спецификации XML, используемая при проверке допустимости.  
Возможные значения параметра: "1.0" или "1.1".  
Значение по умолчанию: 1.0.

Возвращаемое значение:

Тип: Строка.   
Строка, из которой удалены все символы, не соответствующие стандарту XML 1.0 или 1.1.

Описание:

Удаляет символы в указанной строке, не удовлетворяющие рекомендации XML 1.0 или 1.1.  
Данная функция позволяет решить проблему, например, когда в отображаемых в HTML-документе данных (которые нужно отображать на форме) присутствуют недопустимые символы.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
