ГеографическаяСхема (GeographicalSchema)

ПолучитьКартинку (GetPicture)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьКартинку(<Ширина>, <Высота>, <ФорматКартинки>, <ЧерноБелая>)

Параметры:

<Ширина> (обязательный)

Тип: [Число](lang__def_Number.md).   
Ширина картинки.

<Высота> (обязательный)

Тип: [Число](lang__def_Number.md).   
Высота картинки.

<ФорматКартинки> (необязательный)

Тип: [ФорматКартинки](ФорматКартинки_(PictureFormat).md).   
Требуемый формат картинки.  
Допустимые значения для растровых картинок: [BMP](ФорматКартинки.BMP_(PictureFormat.BMP).md), [PNG](ФорматКартинки.PNG_(PictureFormat.PNG).md) и исходный тип. Для векторных картинок допускается указывать только исходный тип.  
Значение по умолчанию: [BMP](ФорматКартинки.BMP_(PictureFormat.BMP).md).

<ЧерноБелая> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак получения черно-белой картинки.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Картинка](Картинка_(Picture).md).   

Описание:

Получает картинку заданных размеров в указанном формате.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
