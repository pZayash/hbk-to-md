УникальныйИдентификатор (UUID)

Доступен, начиная с версии 8.0.

Синтаксис:

УникальныйИдентификатор()

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   

Описание:

Получает уникальный идентификатор ссылки.  
Ссылка может быть получена из уникального идентификатора с помощью метода менеджера [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЗадачаМенеджер.\<Имя задачи\>](ЗадачаМенеджер.Имя_задачи.md), метод [ПолучитьСсылку](ЗадачаМенеджер.Имя_задачи.ПолучитьСсылку.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
