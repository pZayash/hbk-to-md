# ТестируемаяТаблицаФормы.Развернут

ТестируемаяТаблицаФормы (TestedFormTable)

Развернут (Expanded)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Развернут(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки перехода.  
Если не указан, проверка выполняется для текущей строки.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - узел для текущей строки развернут.

Описание:

Проверяет развернутость узла для текущей строки.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
