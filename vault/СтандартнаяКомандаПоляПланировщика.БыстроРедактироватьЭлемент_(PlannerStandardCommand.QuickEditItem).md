СтандартнаяКомандаПоляПланировщика (PlannerStandardCommand)

БыстроРедактироватьЭлемент (QuickEditItem)

Доступен, начиная с версии 8.3.23.

Описание:

Переход в режим быстрого редактирования выбранного элемента планировщика.  
Используется только для источника типа [Элементы](ИсточникКомандПланировщика.Элементы_(PlannerCommandSource.Items).md), когда свойство [Элементы](ИзмерениеПланировщика.Элементы_(PlannerDimension.Items).md) содержит массив из одного элемента планировщика.  
В контекстном меню команда будет иметь представление "Быстро редактировать".

Использование в версии:

Доступен, начиная с версии 8.3.23.
