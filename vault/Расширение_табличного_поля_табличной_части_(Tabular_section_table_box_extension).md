Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), предназначеные для интерактивной работы с табличной частью. Для данного расширения свойства [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) и [ТекущиеДанные](ТабличноеПоле.ТекущиеДанные_(TableBox.CurrentData).md) имеют значения типа [Строка табличной части](Строка_табличной_части_(Line_of_a_tabular_section).md)

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
