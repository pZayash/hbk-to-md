РежимОтправкиИнформацииОбОшибке (ErrorReportingMode)

Авто (Auto)

Доступен, начиная с версии 8.3.17.

Описание:

В настройках пользователя указывает, что будет использовано значение из общих настроек.  
Если из общих параметров получено значение &Авто, то пользователю будет задан вопрос.

Использование в версии:

Доступен, начиная с версии 8.3.17.
