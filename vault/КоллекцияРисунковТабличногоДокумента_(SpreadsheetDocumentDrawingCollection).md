# КоллекцияРисунковТабличногоДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличный документ](Табличный_документ-2.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[РисунокТабличногоДокумента](РисунокТабличногоДокумента_(SpreadsheetDocumentDrawing).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс рисунка (нумерация с 0).  

Свойства:

[\<Имя рисунка\> (\<Drawing name\>)](КоллекцияРисунковТабличногоДокумента.Имя_рисунка_(SpreadsheetDocumentDrawingCollection.Drawing_name).md)  

Методы:

[Добавить (Add)](КоллекцияРисунковТабличногоДокумента.Добавить_(SpreadsheetDocumentDrawingCollection.Add).md)  
[Индекс (IndexOf)](КоллекцияРисунковТабличногоДокумента.Индекс_(SpreadsheetDocumentDrawingCollection.IndexOf).md)  
[Количество (Count)](КоллекцияРисунковТабличногоДокумента.Количество_(SpreadsheetDocumentDrawingCollection.Count).md)  
[Очистить (Clear)](КоллекцияРисунковТабличногоДокумента.Очистить_(SpreadsheetDocumentDrawingCollection.Clear).md)  
[Получить (Get)](КоллекцияРисунковТабличногоДокумента.Получить_(SpreadsheetDocumentDrawingCollection.Get).md)  
[Удалить (Delete)](КоллекцияРисунковТабличногоДокумента.Удалить_(SpreadsheetDocumentDrawingCollection.Delete).md)  

Описание:

Представляет собой коллекцию рисунков, вставленных в табличный документ.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), свойство [Рисунки](ТабличныйДокумент.Рисунки_(SpreadsheetDocument.Drawings).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
