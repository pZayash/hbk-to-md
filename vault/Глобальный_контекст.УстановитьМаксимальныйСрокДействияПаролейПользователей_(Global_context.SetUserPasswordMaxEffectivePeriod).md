Глобальный контекст (Global context)

УстановитьМаксимальныйСрокДействияПаролейПользователей (SetUserPasswordMaxEffectivePeriod)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьМаксимальныйСрокДействияПаролейПользователей(<МаксимальныйСрокДействияПаролей>)

Параметры:

<МаксимальныйСрокДействияПаролей> (необязательный)

Тип: [Число](lang__def_Number.md).   
Максимальный срок действия паролей.  
Значение по умолчанию: 0.

Описание:

Устанавливает максимальный срок действия паролей пользователей.   
  
Максимальный срок действия паролей не может быть меньше минимального срока действия паролей.  
  
Значение по умолчанию: 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.24.
