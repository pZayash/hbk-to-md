Глобальный контекст (Global context)

ПриИзмененииДоступностиОсновногоСервера (OnMainServerAvailabilityChange)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ПриИзмененииДоступностиОсновногоСервера(<НачалоСеансаОсновногоСервера>)

Параметры:

<НачалоСеансаОсновногоСервера>

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак создания нового сеанаса на основном сервере:

- [Истина](lang__def_BooleanTrue.md) - на основном сервере был инициализирован новый сеанс,
- [Ложь](lang__def_BooleanFalse.md) - произошла кратковременная потеря связи, без потери сеанса на основном сервере.

Описание:

Возникает при недоступности основного сервера.

Доступность:

Мобильный клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
