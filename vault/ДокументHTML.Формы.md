ДокументHTML (HTMLDocument)

Формы (Forms)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовHTML](КоллекцияЭлементовHTML.md).   
Содержит коллекцию элементов всех форм в HTML документе.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Коллекция содержит элементы типа [ЭлементФормаHTML](ЭлементФормаHTML.md).

Использование в версии:

Доступен, начиная с версии 8.2.
