НастройкиПроверкиРаскрытияПароля (PasswordCompromiseCheckSettings)

ИгнорироватьОшибкиСервисаПроверкиРаскрытияПароля (IgnorePasswordCompromiseCheckServiceErrors)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет поведением системы при ошибке соединения с сервисом проверки раскрытия пароля. Ошибки могут возникать не только по истечении Таймаута, но и, например, при неправильно указанном адресе сервиса проверки раскрытия пароля:

- [Истина](lang__def_BooleanTrue.md) (по умолчанию) - при ошибке выполняется только запись в журнал регистрации. Не влияет на аутентификацию, на запись пользователей, на вызовы методов проверки раскрытия пароля и соответствия пароля требованиям.
- [Ложь](lang__def_BooleanFalse.md) - при ошибке генерируется исключение.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
