ТаблицаФормы (FormTable)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [СочетаниеКлавиш](СочетаниеКлавиш.md).   
Содержит сочетание клавиш таблицы. Действие сочетания клавиш приводит к установке фокуса таблице. Выбирается колонка текущая или колонка по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
