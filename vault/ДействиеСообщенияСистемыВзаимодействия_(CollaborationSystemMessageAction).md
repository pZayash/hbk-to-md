Описание:

Представляет собой действие сообщения системы взаимодействия. Отображаются в виде гиперссылок при отображении сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md), свойство [Действия](СообщениеСистемыВзаимодействия.Действия_(CollaborationSystemMessage.Actions).md)
