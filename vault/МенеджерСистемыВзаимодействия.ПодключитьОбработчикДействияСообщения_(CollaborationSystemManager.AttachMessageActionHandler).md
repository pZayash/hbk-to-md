МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПодключитьОбработчикДействияСообщения (AttachMessageActionHandler)

Не рекомендуется использовать, начиная с версии 8.3.18.

Рекомендуется использовать:

- [ПриВыбореДействияСообщенияСистемыВзаимодействия](Глобальный_контекст.ПриВыбореДействияСообщенияСистемыВзаимодействия_(Global_context.OnCollaborationSystemMessageActionChoice).md)

Доступен, начиная с версии 8.3.11.

Синтаксис:

ПодключитьОбработчикДействияСообщения(<Обработчик>)

Параметры:

<Обработчик> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после интерактивной отправки сообщения в обсуждение со следующими параметрами:

- <Сообщение> - тип [СообщениеСистемыВзаимодействия](СообщениеСистемыВзаимодействия_(CollaborationSystemMessage).md).Сообщение, в котором нажата гиперссылка действия. В обработчик передается копия объекта;
- <Действие> - тип Произвольный. Значение, заданное для нажатой гиперссылки.
- <Дополнительные параметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

Описание:

Подключает процедуру-обработчик, вызываемую при нажатии на гиперссылку в сообщении, заданную в свойстве [Действия](СообщениеСистемыВзаимодействия.Действия_(CollaborationSystemMessage.Actions).md).  
Допускается наличие нескольких обработчиков. В этом случае при нажатии на гиперссылку вызываются все обработчики. Порядок вызова обработчиков не определен.  
Повторный вызов метода для подключенного ранее обработчика замещает обработчик.  
При повторном вызове метода сравнение обработчиков выполняется по равенству значений свойств:

- ИмяПроцедуры,
- ИмяПроцедурыОбработкиОшибки,
- Модуль,
- МодульОбработкиОшибки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.
