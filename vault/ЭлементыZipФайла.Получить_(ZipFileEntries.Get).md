ЭлементыZipФайла (ZipFileEntries)

Получить (Get)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Получить](ЭлементыФайлаАрхива.Получить_(ArchiveFileEntries.Get).md)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента в коллекции.

Возвращаемое значение:

Тип: [ЭлементZipФайла](ЭлементZipФайла_(ZipFileEntry).md).   

Описание:

Получает элемент по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
