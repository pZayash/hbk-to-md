ОбъектМетаданных: ПризнакУчетаПланаСчетов (MetadataObject: AccountingFlag)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство определяет возможность заполнения признака учета плана счетов из данных заполнения. Используется для заполнения признака учета плана счетов из значений текущего отбора списков. Если установлено [Истина](lang__def_BooleanTrue.md), то процедура стандартной обработки заполнения будет заполнять признак учета плана счетов из данных заполнения. Если в данных заполнения нет данных для заполнения этого признака учета плана счетов или свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то признак учета плана счетов будет заполнен из свойства [ЗначениеЗаполнения](ОбъектМетаданных_Реквизит.ЗначениеЗаполнения_(MetadataObject_Attribute.FillingValue).md), если оно установлено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
