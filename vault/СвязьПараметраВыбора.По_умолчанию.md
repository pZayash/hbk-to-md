СвязьПараметраВыбора (ChoiceParameterLink)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СвязьПараметраВыбора(<Имя>, <ПутьКДанным>, <ИзменениеЗначения>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Значение, указанное в параметре, присваивается свойству [Имя](СвязьПараметраВыбора.Имя_(ChoiceParameterLink.Name).md).

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Значение, указанное в параметре, присваивается свойству [ПутьКДанным](СвязьПараметраВыбора.ПутьКДанным_(ChoiceParameterLink.DataPath).md).

<ИзменениеЗначения> (необязательный)

Тип: [РежимИзмененияСвязанногоЗначения](РежимИзмененияСвязанногоЗначения_(LinkedValueChangeMode).md).   
Указывает, каким образом изменять связаное значение.  
Значение по умолчанию: [Очищать](РежимИзмененияСвязанногоЗначения.Очищать_(LinkedValueChangeMode.Clear).md).

Описание:

Создает объект и заполняет его свойства значениями из параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
