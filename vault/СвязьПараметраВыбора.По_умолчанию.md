СвязьПараметраВыбора (ChoiceParameterLink)

По умолчанию

Доступен, начиная с версии 8.2.

Синтаксис:

Новый СвязьПараметраВыбора(<Имя>, <ПутьКДанным>, <ИзменениеЗначения>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Значение, указанное в параметре, присваивается свойству [Имя](СвязьПараметраВыбора.Имя.md).

<ПутьКДанным> (обязательный)

Тип: Строка.   
Значение, указанное в параметре, присваивается свойству [ПутьКДанным](СвязьПараметраВыбора.ПутьКДанным.md).

<ИзменениеЗначения> (необязательный)

Тип: [РежимИзмененияСвязанногоЗначения](РежимИзмененияСвязанногоЗначения.md).   
Указывает, каким образом изменять связаное значение.  
Значение по умолчанию: [Очищать](РежимИзмененияСвязанногоЗначения.Очищать.md).

Описание:

Создает объект и заполняет его свойства значениями из параметров.

Использование в версии:

Доступен, начиная с версии 8.2.
