КонстантаКлючЗначения.<Имя константы> (ConstantValueKey.<Constant name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Константа](ОбъектМетаданных_Константа.md).   

Описание:

Предоставляет доступ к объекту описания метаданных константы. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные.md). Например: Метаданные.Константы.Валюта.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
