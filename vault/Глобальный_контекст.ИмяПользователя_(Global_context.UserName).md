Глобальный контекст (Global context)

ИмяПользователя (UserName)

Доступен, начиная с версии 8.0.

Синтаксис:

ИмяПользователя()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Имя пользователя.

Описание:

Получает имя пользователя, указанного при запуске программы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТекущийПользователь = ИмяПользователя(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ПолноеИмяПользователя](Глобальный_контекст.ПолноеИмяПользователя_(Global_context.UserFullName).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
