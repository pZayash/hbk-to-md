Существующий дочерний узел.

Возвращаемое значение:

Значение прежнего узла.

Описание:

Заменяет дочерний узел, указанный параметром <СтарыйУзел>, на узел, указанный параметром <НовыйУзел>.  
Если новый узел - это [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента.md), то заменяемый узел будет заменен всеми дочерними узлами фрагмента документа.   
Если новый узел уже содержится в дереве документа, то он сначала удаляется из дочерних узлов старого родителя и вставляется в список дочерних узлов нового родителя.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Исключение вызывается в случаях:

- если данный узел не допускает в качестве дочерних добавляемый узел или добавляемый узел является одним из предков данного;
- если узел добавляется сам в себя;
- если данный узел - [Документ](ТипУзлаDOM.Документ.md) и добавляется второй узел [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) или [Элемент](ТипУзлаDOM.Элемент.md);
- если добавляемый узел был создан документом отличным от создавшего данный узел;
- если данный узел в режиме только для чтения или если предыдущий родитель добавляемого узла в режиме Только для чтения;
- если заменяемый узел не найден;
- если добавляемый узел является дочерним узлом узла [Документ](ТипУзлаDOM.Документ.md) и реализация не поддерживает удаления узлов [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента.md) и [Элемент](ТипУзлаDOM.Элемент.md) у узла [Документ](ТипУзлаDOM.Документ.md).

Использование в версии:

Доступен, начиная с версии 8.2.
