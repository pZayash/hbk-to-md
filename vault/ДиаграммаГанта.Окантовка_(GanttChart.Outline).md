ДиаграммаГанта (GanttChart)

Окантовка (Outline)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак окантовки черным цветом интервалов на диаграмме Ганта. Вне зависимости от цвета интервалов окантовка наносится тонкой черной линией.  
[Истина](lang__def_BooleanTrue.md) - окантовка используется.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
