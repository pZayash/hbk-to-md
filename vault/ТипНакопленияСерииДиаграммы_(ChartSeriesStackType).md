Описание:

Задает тип накопления серии, отличный от типа накопления диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: ChartSeriesAddType.

Использование в версии:

Доступен, начиная с версии 8.3.16.
