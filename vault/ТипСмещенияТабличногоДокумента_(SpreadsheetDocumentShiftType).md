Доступен, начиная с версии 8.0.

Значения

[БезСмещения (WithoutShift)](ТипСмещенияТабличногоДокумента.БезСмещения_(SpreadsheetDocumentShiftType.WithoutShift).md)  
[ПоВертикали (Vertical)](ТипСмещенияТабличногоДокумента.ПоВертикали_(SpreadsheetDocumentShiftType.Vertical).md)  
[ПоГоризонтали (Horizontal)](ТипСмещенияТабличногоДокумента.ПоГоризонтали_(SpreadsheetDocumentShiftType.Horizontal).md)  

Описание:

Содержит варианты смещения ячеек при вставке или удалении областей табличного документа.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется.

См. также:

[ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), метод [ВставитьОбласть](ТабличныйДокумент.ВставитьОбласть_(SpreadsheetDocument.InsertArea).md)

Использование в версии:

Доступен, начиная с версии 8.0.
