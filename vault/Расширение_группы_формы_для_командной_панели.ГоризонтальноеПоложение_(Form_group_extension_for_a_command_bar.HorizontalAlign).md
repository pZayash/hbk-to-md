# Расширение группы формы для командной панели.ГоризонтальноеПоложение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Группа формы](Группа_формы.md) › [Расширение группы формы для командной панели](Расширение_группы_формы_для_командной_панели_(Form_group_extension_for_a_command_bar).md) › [Свойства](Расширение_группы_формы_для_командной_панели__Свойства.md)

Расширение группы формы для командной панели (Form group extension for a command bar)

ГоризонтальноеПоложение (HorizontalAlign)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ГоризонтальноеПоложениеЭлемента](ГоризонтальноеПоложениеЭлемента_(ItemHorizontalLocation).md).   
Содержит вид выравнивания элементов в командной панели.  
Запрещено использовать значение [Авто](ГоризонтальноеПоложениеЭлемента.Авто_(ItemHorizontalLocation.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
