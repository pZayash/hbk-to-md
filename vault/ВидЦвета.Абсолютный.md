ВидЦвета (ColorType)

Абсолютный (Absolute)

Доступен, начиная с версии 8.0.

Описание:

Цвет задается абсолютным значением, то есть сочетанием красной, зеленой и синей составляющих.

Использование в версии:

Доступен, начиная с версии 8.0.
