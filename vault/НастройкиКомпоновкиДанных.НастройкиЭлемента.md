НастройкиЭлемента (ItemSettings)

Доступен, начиная с версии 8.2.

Синтаксис:

НастройкиЭлемента(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md), [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных.md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных.md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных.md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных.md), [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных.md), [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных.md) , [КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных](КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных.md), [КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных](КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных.md).   
Элемент, настройки которого нужно получить.

Возвращаемое значение:

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных.md).   

Описание:

Получает настройки компоновки данных, к которым относится переданный элемент.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
