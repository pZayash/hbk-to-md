Действие (Action)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Действие](Действие_(Action).md).   
Содержит действие, ассоциированное с кнопкой командного интерфейса. Имеет смысл только для кнопок типа [Действие](ТипКнопкиКоманднойПанели.Действие_(CommandBarButtonType.Action).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
