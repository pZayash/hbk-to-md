Диаграмма (Chart)

ИнформационныеЛинииТочек (PointsReferenceLines)

Доступен, начиная с версии 8.3.12.

Использование:

Только чтение.

Описание:

Тип: [ИнформационныеЛинииДиаграммы](ИнформационныеЛинииДиаграммы_(ChartReferenceLines).md).   
Коллекция информационных линий, которые соответствуют точкам.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Может использоваться в реквизитах управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.3.12.
