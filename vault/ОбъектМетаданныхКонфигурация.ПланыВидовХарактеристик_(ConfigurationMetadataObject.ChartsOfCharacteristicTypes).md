ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПланыВидовХарактеристик (ChartsOfCharacteristicTypes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все планы видов характеристик, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ПланВидовХарактеристик](ОбъектМетаданных_ПланВидовХарактеристик_(MetadataObject_ChartOfCharacteristicTypes).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.20.
