ОтправлятьОтчетОбАварийномЗавершенииВСервисРегистрацииОшибокПлатформы (SendCrashReportToPlatformErrorRegistrationService)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ОтправлятьОтчетОбАварийномЗавершенииНаКлиентеВСервисРегистрацииОшибокПлатформы](НастройкиОбработкиОшибок.ОтправлятьОтчетОбАварийномЗавершенииНаКлиентеВСервисРегистрацииОшибокПлатформы.md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке.md).   
Указывает необходимость отправки информации об аварийном завершении в техподдержку фирмы "1С".

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
