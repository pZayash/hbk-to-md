# ПланОбменаОбъект.<Имя плана обмена>.ПолучитьСсылкуНового

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

ПолучитьСсылкуНового (GetNewObjectRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылкуНового()

Возвращаемое значение:

Тип: [ПланОбменаСсылка.\<Имя плана обмена\>](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md).   

Описание:

Позволяет для нового (созданного и еще не записанного) объекта получить ранее установленное методом [УстановитьСсылкуНового](ПланОбменаОбъект.Имя_плана_обмена.УстановитьСсылкуНового_(ExchangePlanObject.Exchange_plan_name.SetNewObjectRef).md) значение ссылки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[ПланОбменаМенеджер.\<Имя плана обмена\>](ПланОбменаМенеджер.Имя_плана_обмена_(ExchangePlanManager.Exchange_plan_name).md), метод [ПолучитьСсылку](ПланОбменаМенеджер.Имя_плана_обмена.ПолучитьСсылку_(ExchangePlanManager.Exchange_plan_name.GetRef).md)

Доступен, начиная с версии 8.0.
