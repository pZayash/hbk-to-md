МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ПроверитьИндекс (CheckIndex)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПроверитьИндекс()

Возвращаемое значение:

Тип: Булево.   
Ложь - если файлах индекса содержатся некорректные данные. В этом случае рекомендуется использовать методы [ОчиститьИндекс](МенеджерПолнотекстовогоПоиска.ОчиститьИндекс.md) и [ОбновитьИндекс](МенеджерПолнотекстовогоПоиска.ОбновитьИндекс.md).

Описание:

Проверяет целостность файлов полнотекстового поиска.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если в момент запуска проверки выполняется обновление, очистка или проверка данных полнотекстового поиска, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
