# Расширение табличного поля дерева значений.ПередНачаломДобавления

Расширение табличного поля дерева значений (Value tree table box extension)

ПередНачаломДобавления (BeforeAddLine)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>, <Родитель>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления строки. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то строка списка узлов добавлена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

<Родитель>

Тип: [СтрокаДереваЗначений](СтрокаДереваЗначений_(ValueTreeRow).md).   
Строка-родитель, в которую будет добавляться новая строка. Если добавляется в корень дерева, то значение [Неопределено](lang__def_Undefined.md)

Описание:

Вызывается перед началом добавления строки.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
