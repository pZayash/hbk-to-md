ВыраженияПорядкаСхемыЗапроса (QuerySchemaOrderExpressions)

Найти (Find)

Доступен, начиная с версии 8.3.7.

Синтаксис:

Найти(<КолонкаИлиВыражение>)

Параметры:

<КолонкаИлиВыражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки или выражение для поиска.

Возвращаемое значение:

Тип: [ВыражениеПорядкаСхемыЗапроса](ВыражениеПорядкаСхемыЗапроса_(QuerySchemaOrderExpression).md).   

Описание:

Выполняет поиск по свойству [Элемент](ВыражениеПорядкаСхемыЗапроса.Элемент_(QuerySchemaOrderExpression.Item).md).   
В зависимости от элемента проверяется выражение или имя колонки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
