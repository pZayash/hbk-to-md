НастройкиВторогоФактораАутентификации (SecondAuthenticationFactorSettings)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Массив содержит настройки взаимодействия с провайдерами, которые доступны пользователю.  
Элементы массива представляют собой настройки провайдеров второго фактора аутентификации и имеют тип [НастройкаВторогоФактораАутентификации](НастройкаВторогоФактораАутентификации.md).

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
