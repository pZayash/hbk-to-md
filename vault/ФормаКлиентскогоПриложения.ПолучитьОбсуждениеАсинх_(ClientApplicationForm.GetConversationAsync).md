ФормаКлиентскогоПриложения (ClientApplicationForm)

ПолучитьОбсуждениеАсинх (GetConversationAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По идентификатору

Синтаксис:

ПолучитьОбсуждениеАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения.

Описание варианта метода:

Возвращает обсуждение по идентификатору.

Вариант синтаксиса: По ключу

Синтаксис:

ПолучитьОбсуждениеАсинх(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ обсуждения.  
Значение по умолчанию: "".

Описание варианта метода:

Возвращает неконтекстное обсуждение по заданному ключу. Если обсуждение не найдено, возвращает [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md) или [Неопределено](lang__def_Undefined.md), если у формы нет навигационной ссылки (например, форма нового незаписанного объекта).

Описание:

Асинхронная функция. Выполняет получение контекстного обсуждения формы.  
Если обсуждение с указанным идентификатором не найдено, оно будет создано.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md), метод [НачатьПолучениеОбсуждения](ФормаКлиентскогоПриложения.НачатьПолучениеОбсуждения_(ClientApplicationForm.BeginGetConversation).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
