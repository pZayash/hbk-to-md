ТипКурсоровТабличногоДокумента (SpreadsheetDocumentPointerType)

Специальные (Special)

Доступен, начиная с версии 8.3.10.

Описание:

В табличном документе нужно использовать специальные указатели мыши (перекрестие с увеличительным стеклом).

Использование в версии:

Доступен, начиная с версии 8.3.10.
