ПараметрыСканированияДокументов (DocumentScanningParameters)

ПрямоугольныйДокумент (RectangularDocument)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, что форма документа будет выпрямлена до прямоугольника.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
