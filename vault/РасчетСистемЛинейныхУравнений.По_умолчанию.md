РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый РасчетСистемЛинейныхУравнений()

Описание:

Создает объект и устанавливает следующие значения для свойств:

- [ОписанияСистем](РасчетСистемЛинейныхУравнений.ОписанияСистем.md) – новый объект [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений.md) (пустая коллекция);
- [КоличествоИтераций](РасчетСистемЛинейныхУравнений.КоличествоИтераций.md) = 200;
- [ТребуемаяТочность](РасчетСистемЛинейныхУравнений.ТребуемаяТочность.md) = 0.000001;
- [ИспользованиеВычислительныхРесурсов](РасчетСистемЛинейныхУравнений.ИспользованиеВычислительныхРесурсов.md) = 0;
- [ГраницаИзмененияАлгоритмаРешения](РасчетСистемЛинейныхУравнений.ГраницаИзмененияАлгоритмаРешения.md) = 10000.

Значения остальных свойств устанавливаются равными Неопределено.  
Для корректной работы механизма после вызова конструктора необходимо установить значения для свойств: [ИсточникДанныхУзлов](РасчетСистемЛинейныхУравнений.ИсточникДанныхУзлов.md), [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей.md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях.md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях.md), [КолонкаУравненияВУзлах](РасчетСистемЛинейныхУравнений.КолонкаУравненияВУзлах.md), и добавить в коллекцию [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений.md) хотя бы один объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md).

Использование в версии:

Доступен, начиная с версии 8.3.14.
