РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

По умолчанию

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый РасчетСистемЛинейныхУравнений()

Описание:

Создает объект и устанавливает следующие значения для свойств:

- [ОписанияСистем](РасчетСистемЛинейныхУравнений.ОписанияСистем_(SystemOfLinearEquationsCalculation.SystemsDescription).md) – новый объект [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений_(SystemDescriptions).md) (пустая коллекция);
- [КоличествоИтераций](РасчетСистемЛинейныхУравнений.КоличествоИтераций_(SystemOfLinearEquationsCalculation.IterationCount).md) = 200;
- [ТребуемаяТочность](РасчетСистемЛинейныхУравнений.ТребуемаяТочность_(SystemOfLinearEquationsCalculation.AdequateAccuracy).md) = 0.000001;
- [ИспользованиеВычислительныхРесурсов](РасчетСистемЛинейныхУравнений.ИспользованиеВычислительныхРесурсов_(SystemOfLinearEquationsCalculation.ComputationalResourcesUsage).md) = 0;
- [ГраницаИзмененияАлгоритмаРешения](РасчетСистемЛинейныхУравнений.ГраницаИзмененияАлгоритмаРешения_(SystemOfLinearEquationsCalculation.AlgorithmSelectionBorder).md) = 10000.

Значения остальных свойств устанавливаются равными [Неопределено](lang__def_Undefined.md).  
Для корректной работы механизма после вызова конструктора необходимо установить значения для свойств: [ИсточникДанныхУзлов](РасчетСистемЛинейныхУравнений.ИсточникДанныхУзлов_(SystemOfLinearEquationsCalculation.NodeDataSource).md), [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей_(SystemOfLinearEquationsCalculation.RelationDataSource).md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях_(SystemOfLinearEquationsCalculation.EquationsInRelationsColumn).md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях_(SystemOfLinearEquationsCalculation.VariablesInRelationsColumn).md), [КолонкаУравненияВУзлах](РасчетСистемЛинейныхУравнений.КолонкаУравненияВУзлах_(SystemOfLinearEquationsCalculation.EquationsInNodesColumn).md), и добавить в коллекцию [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений_(SystemDescriptions).md) хотя бы один объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений_(SystemOfLinearEquationsDescription).md).

Использование в версии:

Доступен, начиная с версии 8.3.14.
