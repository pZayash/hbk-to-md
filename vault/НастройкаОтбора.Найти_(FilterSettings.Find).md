НастройкаОтбора (FilterSettings)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя элемента отбора.

Возвращаемое значение:

Тип: [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента по имени элемента отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
