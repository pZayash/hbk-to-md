Глобальный контекст (Global context)

ПолучитьВремяОжиданияБлокировкиДанных (GetLockWaitTime)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьВремяОжиданияБлокировкиДанных()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает заданное в информационной базе время ожидания блокировки данных (в секундах).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
