ОбъявлениеЭлементаXS (XSElementDeclaration)

ПрисоединениеКГруппеПодстановки (SubstitutionGroupAffiliation)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [РасширенноеИмяXML](РасширенноеИмяXML_(XMLExpandedName).md).   
Содержит имя группы подстановки, членом которой может потенциально являться данный элемент.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Представляет свойство substitution group affiliation (<http://www.w3.org/TR/xmlschema-1/#class_exemplar>) информационного пространства объявления элемента.

Использование в версии:

Доступен, начиная с версии 8.1.
