ПолеГрафическойСхемы (GraphicalSchemaField)

Напечатать (Print)

Доступен, начиная с версии 8.0.

Синтаксис:

Напечатать(<ДиалогПечати>)

Параметры:

<ДиалогПечати> (необязательный)

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати_(PrintDialogUseMode).md).   
Определяет необходимость показа диалога печати перед печатью.  
Значение по умолчанию: [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать_(PrintDialogUseMode.DontUse).md).

Описание:

Выводит графическую схему на печать.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ЭлементыФормы.ПолеГрафическойСхемы1.Напечатать(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
