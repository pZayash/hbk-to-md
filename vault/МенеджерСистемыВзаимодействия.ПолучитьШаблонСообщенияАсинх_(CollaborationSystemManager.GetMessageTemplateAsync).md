МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьШаблонСообщенияАсинх (GetMessageTemplateAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьШаблонСообщенияАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторШаблонаСообщенияСистемыВзаимодействия](ИдентификаторШаблонаСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplateID).md).   
Идентификатор шаблона сообщения.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - значение типа [ШаблонСообщенияСистемыВзаимодействия](ШаблонСообщенияСистемыВзаимодействия_(CollaborationSystemMessageTemplate).md) или [Неопределено](lang__def_Undefined.md):

- Для интеграции типа WhatsApp возвращает шаблон сообщения по его идентификатору.
- Для остальных интеграций или если шаблон не найден возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Получает шаблон сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
