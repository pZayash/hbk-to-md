Глобальный контекст (Global context)

НомерСоединенияИнформационнойБазы (InfoBaseConnectionNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

НомерСоединенияИнформационнойБазы()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает номер текущего соединения с информационной базой.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[СоединениеИнформационнойБазы](СоединениеИнформационнойБазы.md), свойство [НомерСоединения](СоединениеИнформационнойБазы.НомерСоединения.md)
