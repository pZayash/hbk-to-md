ПостроительОтчета (ReportBuilder)

ВыводитьПодвалТаблицы (PutTableFooter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости вывода подвала таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

[Истина](lang__def_BooleanTrue.md) - выводить подвал таблицы;   
[Ложь](lang__def_BooleanFalse.md) - не выводить.

Использование в версии:

Доступен, начиная с версии 8.0.
