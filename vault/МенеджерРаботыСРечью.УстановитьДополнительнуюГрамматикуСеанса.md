МенеджерРаботыСРечью (SpeechProcessingManager)

УстановитьДополнительнуюГрамматикуСеанса (SetSessionAdditionalGrammar)

Доступен, начиная с версии 8.3.23.

Синтаксис:

УстановитьДополнительнуюГрамматикуСеанса(<Ключ>, <ДополнительнаяГрамматика>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Идентификатор грамматики. В случае если грамматика с таким ключом уже есть, она будет заменена.

<ДополнительнаяГрамматика> (обязательный)

Тип: [Массив](Массив.md).   
Массив строк, дополняющих или заменяющих грамматику модели.

Описание:

Устанавливает дополнительную грамматику распознавания речи для сеанса в сервис распознавания речи.   
Дополнительная грамматика становится доступной для использования.  
Дополнительная грамматика хранится в сеансовых данных.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.
