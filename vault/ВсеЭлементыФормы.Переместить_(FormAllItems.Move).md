# ВсеЭлементыФормы.Переместить

ВсеЭлементыФормы (FormAllItems)

Переместить (Move)

Доступен, начиная с версии 8.2.

Синтаксис:

Переместить(<Элемент>, <Родитель>, <МестоРасположения>)

Параметры:

<Элемент> (обязательный)

Тип: [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md), [ГруппаФормы](ГруппаФормы_(FormGroup).md), [КнопкаФормы](КнопкаФормы_(FormButton).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md).   
Перемещаемый элемент.

<Родитель> (обязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения_(ClientApplicationForm).md).   
Новый родитель элемента. Может совпадать со старым.

<МестоРасположения> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md), [ДекорацияФормы](ДекорацияФормы_(FormDecoration).md), [КнопкаФормы](КнопкаФормы_(FormButton).md), [ТаблицаФормы](ТаблицаФормы_(FormTable).md), [ПолеФормы](ПолеФормы_(FormField).md).   
Элемент, перед которым нужно разместить перемещаемый элемент. Если не задан, то перемещается в конец коллекции.

Описание:

Перемещает элемент коллекции.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
