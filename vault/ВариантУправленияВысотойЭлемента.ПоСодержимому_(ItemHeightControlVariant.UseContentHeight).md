ВариантУправленияВысотойЭлемента (ItemHeightControlVariant)

ПоСодержимому (UseContentHeight)

Доступен, начиная с версии 8.3.9.

Описание:

Высота элемента задается так, чтобы высота рабочей области элемента совпадала с высотой содержимого.

Использование в версии:

Доступен, начиная с версии 8.3.9.
