# ТестируемаяТаблицаФормы.ВозможноРазвернуть

ТестируемаяТаблицаФормы (TestedFormTable)

ВозможноРазвернуть (CanBeExpanded)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ВозможноРазвернуть(<ОписаниеСтроки>)

Параметры:

<ОписаниеСтроки> (необязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Содержит набор пар "Заголовок колонки" - "Текст ячейки" для поиска строки.  
Если не указан, выполняется проверка возможности развернуть текущую строку.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - разворачивание узла возможно; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Проверяет возможность разворачивания узла.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
