Линия (Line)

Для линии табличного документа

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Линия(<ТипЛинии>, <Толщина>, <Отступ>)

Параметры:

<ТипЛинии> (обязательный)

Тип: [ТипЛинииЯчейкиТабличногоДокумента](ТипЛинииЯчейкиТабличногоДокумента_(SpreadsheetDocumentCellLineType).md).   
Тип линии. Тип параметра зависит от того, в каком объекте будет использоваться линия.  
Если выбран тип [ТипЛинииЯчейкиТабличногоДокумента](ТипЛинииЯчейкиТабличногоДокумента_(SpreadsheetDocumentCellLineType).md), то толщина линии может быть только 1.

<Толщина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Толщина линии (в логических единицах). Если в параметре <ТипЛинии> указано значение НетЛинии, то максимальная толщина линии - 1.

<Отступ> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Отступ линии.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Создает линию заданного типа и толщины, а также устанавливает отступ линии.

Пример:

|  |
| --- |
| Линия = Новый Линия(ТипЛинииЯчейкиТабличногоДокумента.Сплошная,2); |

Использование в версии:

Доступен, начиная с версии 8.0.
