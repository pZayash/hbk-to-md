БуферДвоичныхДанных (BinaryDataBuffer)

ПрочитатьЦелое32 (ReadInt32)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьЦелое32(<Позиция>, <ПорядокБайтов>)

Параметры:

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, на которой требуется прочитать число.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных_(DataReader).md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Прочитать целое 32-битное положительное число на заданной позиции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
