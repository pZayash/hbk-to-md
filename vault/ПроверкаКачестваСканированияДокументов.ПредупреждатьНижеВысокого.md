ПроверкаКачестваСканированияДокументов (DocumentScanningCheckingQuality)

ПредупреждатьНижеВысокого (WarnBelowHigh)

Доступен, начиная с версии 8.3.23.

Описание:

Пользователь может принять документ любого качестве.  
Пользователю показываются предупреждения о качестве документа, чье обобщенное качество ниже Высокого (меньше 7).

Использование в версии:

Доступен, начиная с версии 8.3.23.
