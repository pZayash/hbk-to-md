ОбъектМетаданных: ОперацияWebСервиса (MetadataObject: WebServiceOperation)

ВТранзакции (Transactioned)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то код процедуры, описывающей операцию Web-сервиса, будет выполняться в транзакции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
