# ДокументDOM.СоздатьТекстовыйУзел

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ДокументDOM](ДокументDOM_(DOMDocument).md) › [Методы](ДокументDOM__Методы.md)

ДокументDOM (DOMDocument)

СоздатьТекстовыйУзел (CreateTextNode)

Доступен, начиная с версии 8.1.

Синтаксис:

СоздатьТекстовыйУзел(<Текст>)

Параметры:

<Текст> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, представляющая собой значение текстового узла.

Возвращаемое значение:

Тип: [ТекстDOM](ТекстDOM_(DOMText).md).   

Описание:

Создает узел DOM - текст.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
