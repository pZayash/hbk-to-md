# МенеджерХранилищаДвоичныхДанных.ЗагрузитьДифференциальнуюРезервнуюКопию

МенеджерХранилищаДвоичныхДанных (BinaryDataStorageManager)

ЗагрузитьДифференциальнуюРезервнуюКопию (LoadDifferentialBackup)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ЗагрузитьДифференциальнуюРезервнуюКопию(<Путь>, <БазовыйПуть>)

Параметры:

<Путь> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу с дифференциальной резервной копией.

<БазовыйПуть> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу с полной резервной копией хранилища двоичных данных, от которой была сделана дифференциальная резервная копия, которую требуется восстановить.

Возвращаемое значение:

Нет

Описание:

Загружает дифференциальную резервную копию хранилища двоичных данных.  
Дифференциальная резервная копия - это резервная копия, которая хранит только изменения относительно момента начала создания полной резервной копии.   
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены. Если до вызова метода режим монопольного доступа к базе данных не был установлен, то он устанавливается на время вызова. Если установить монопольный доступ невозможно, будет сгенерировано исключение.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.26.
