ДиаграммаГанта (GanttChart)

ВертикальнаяПрокрутка (VerticalScroll)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, разрешена ли вертикальная прокрутка области построения.  
[Истина](lang__def_BooleanTrue.md) - прокрутка разрешена.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
