# ХранилищаНастроекМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Хранилища настроек](Хранилища_настроек.md)

Доступен, начиная с версии 8.2.

Элементы коллекции:

[ХранилищеНастроекМенеджер.\<Имя хранилища\>](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются хранилища настроек.  
Возможно обращение к хранилищу настроек посредством оператора [...]. В качестве аргумента передается имя хранилища, как оно задано в конфигурации.  

Свойства:

[\<Имя хранилища\> (\<Имя хранилища\>)](ХранилищаНастроекМенеджер.Имя_хранилища_(SettingsStoragesManager.Имя_хранилища).md)  

Описание:

Коллекция хранилищ настроек, определенных в конфигурации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [ХранилищаНастроек](Глобальный_контекст.ХранилищаНастроек_(Global_context.SettingsStorages).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
