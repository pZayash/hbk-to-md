Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПриОбновленииСоставаПользовательскихНастроекНаСервере (OnUpdateUserSettingSetAtServer)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПриОбновленииСоставаПользовательскихНастроекНаСервере(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
При стандартной обработке события система генерирует элементы формы для редактирования пользовательских настроек аналогично методу [СоздатьЭлементыФормыПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.СоздатьЭлементыФормыПользовательскихНастроек_(Client_application_form_extension_for_reports.CreateUserSettingsFormItems).md).   
При этом, если тип формы [Основная](ТипФормыОтчета.Основная_(ReportFormType.Main).md), то генерируются элементы формы для быстрых пользовательских настроек. Если тип формы [Настройка](ТипФормыОтчета.Настройка_(ReportFormType.Settings).md) или [Вариант](ТипФормыОтчета.Вариант_(ReportFormType.Variant).md), то генерируются элементы формы для всех пользовательских настроек.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Вызывается при обновлении состава пользовательских настроек:

- При получении формы отчета;
- После загрузки варианта отчета;
- После изменения варианта отчета.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
