Глобальный контекст (Global context)

ЗакрытьСправку (CloseHelp)

Доступен, начиная с версии 8.2.

Синтаксис:

ЗакрытьСправку()

Описание:

Закрывает окно со справочной информацией. Когда выбран режим показа справки в нескольких окнах, метод закроет все окна справки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ЗакрытьСправку(); |

Использование в версии:

Доступен, начиная с версии 8.2.
