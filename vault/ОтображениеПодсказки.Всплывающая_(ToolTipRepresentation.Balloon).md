ОтображениеПодсказки (ToolTipRepresentation)

Всплывающая (Balloon)

Доступен, начиная с версии 8.3.1.

Описание:

Подсказка отображается в виде всплывающего текста.  
На мобильной платформе вариант эквивалентен варианту [Нет](ОтображениеПодсказки.Нет_(ToolTipRepresentation.None).md).

Использование в версии:

Доступен, начиная с версии 8.3.1.
