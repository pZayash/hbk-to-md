ИзмеренияПостроителяЗапроса (QueryBuilderDimensions)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Искомое измерение.

Возвращаемое значение:

Тип: [ИзмерениеПостроителяЗапроса](ИзмерениеПостроителяЗапроса_(QueryBuilderDimension).md), [Неопределено](lang__def_Undefined.md).   
Если измерение не найдено, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск измерения по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
