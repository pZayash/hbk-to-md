ОбъектМетаданных: ОбщаяКоманда (MetadataObject: CommonCommand)

ИзменяетДанные (ModifiesData)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если выполняется команда, у которой свойство установлено в [Истина](lang__def_BooleanTrue.md), то после ее выполнения форма перечитывает данные.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
