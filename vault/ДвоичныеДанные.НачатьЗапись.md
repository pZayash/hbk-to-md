ДвоичныеДанные (BinaryData)

НачатьЗапись (BeginWrite)

Доступен, начиная с версии 8.3.9.

Вариант синтаксиса: Запись в файл

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <ИмяФайла>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения работы метода со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в который требуется записать двоичные данные.

Описание варианта метода:

Записать двоичные данные в файл с заданным именем.

Вариант синтаксиса: Запись в файл

Синтаксис:

НачатьЗапись()

Описание варианта метода:

Записывает двоичные данные в файл с указанным именем.

Вариант синтаксиса: Запись в поток

Синтаксис:

НачатьЗапись(<ОписаниеОповещения>, <Поток>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Поток> (обязательный)

Тип: [Поток](Поток.md).   
Поток, в который требуется записать двоичные данные.

Описание варианта метода:

Записывает двоичные данные в заданный поток.

Описание:

Начинает запись двоичных данных в файл или поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

При вызове этого метода в веб-клиенте запрашивается разрешение пользователя на выполнение операции.
