МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ОтменитьСовместноеИспользованиеПриложенийАбонента (CancelSubscriberApplicationLinks)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ОтменитьСовместноеИспользованиеПриложенийАбонента(<Приложения>)

Параметры:

<Приложения> (необязательный)

Тип: [КоллекцияИдентификаторовПриложенийСистемыВзаимодействия](КоллекцияИдентификаторовПриложенийСистемыВзаимодействия_(CollaborationSystemApplicationIDCollection).md).   
Задает приложения, для которых отменяется совместное использование.

Описание:

Отменяет совместное использование приложений системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
