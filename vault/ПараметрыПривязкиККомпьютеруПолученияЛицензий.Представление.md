ПараметрыПривязкиККомпьютеруПолученияЛицензий (LicenseAcquisitionComputerFingerprintParameters)

Представление (Presentation)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Локализованное словесное описание компьютера, как его показывает встроенный диалог получения лицензии. Например: "MyComputerName, Intel64 Family 6 Model 158 Stepping 9, 32646Mb RAM, Майкрософт Windows 10 Корпоративная".

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
