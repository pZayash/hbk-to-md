ШаблонНастройкиВторогоФактораАутентификации (SecondAuthenticationFactorSettingTemplate)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить()

Описание:

Удаляет шаблон из информационной базы. Требует наличия у текущего пользователя права "Администрирование".

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
