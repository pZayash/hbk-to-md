ПолеВвода (TextBox)

СписокВыбора (ChoiceList)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Содержит список значений для осуществления выбора по нажатию кнопки списка выбора.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только при наличии кнопки выбора из списка.

Использование в версии:

Доступен, начиная с версии 8.0.
