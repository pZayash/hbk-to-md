Расширение формы клиентского приложения для документа (Client application form extension for documents)

ПослеЗаписиНаСервере (AfterWriteAtServer)

Доступен, начиная с версии 8.2.

Синтаксис:

ПослеЗаписиНаСервере(<ТекущийОбъект>, <ПараметрыЗаписи>)

Параметры:

<ТекущийОбъект>

Тип: [ДокументОбъект.\<Имя документа\>](ДокументОбъект.Имя_документа_(DocumentObject.Document_name).md).   
Записываемый объект.

<ПараметрыЗаписи>

Тип: [Структура](Структура_(Structure).md).   
Структура, содержащая параметры записи. Для формы документа существует два предопределенных параметра РежимЗаписи и РежимПроведения. Они добавляются автоматически при вызове команд "Провести", "Провести и закрыть", "Отмена проведения".

Описание:

Вызывается после записи объекта на сервере и после завершения транзакции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
