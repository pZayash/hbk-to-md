РезультатЗапроса (QueryResult)

Выгрузить (Unload)

Доступен, начиная с версии 8.0.

Синтаксис:

Выгрузить(<ТипОбхода>)

Параметры:

<ТипОбхода> (необязательный)

Тип: [ОбходРезультатаЗапроса](ОбходРезультатаЗапроса_(QueryResultIteration).md).   
Задает тип обхода записей в получаемой выборке.  
Значение по умолчанию: [Прямой](ОбходРезультатаЗапроса.Прямой_(QueryResultIteration.Linear).md).

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md), [ДеревоЗначений](ДеревоЗначений_(ValueTree).md).   
Если тип обхода задан [Прямой](ОбходРезультатаЗапроса.Прямой_(QueryResultIteration.Linear).md), результат выгружается в таблицу значений, в противном случае в дерево значений.

Описание:

Создает таблицу значений (или дерево значений) и копирует в нее все записи набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ТаблицаРезультатов = РезультатЗапроса.Выгрузить(); |

Использование в версии:

Доступен, начиная с версии 8.0.
