# ОбъектМетаданных: ПланСчетов.АвтоПорядокПоКоду

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: ПланСчетов](ОбъектМетаданных_ПланСчетов_(MetadataObject_ChartOfAccounts).md) › [Свойства](ОбъектМетаданных_ПланСчетов__Свойства.md)

ОбъектМетаданных: ПланСчетов (MetadataObject: ChartOfAccounts)

АвтоПорядокПоКоду (AutoOrderByCode)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то упорядочивание по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код_(ChartOfAccountsObject.Chart_of_accounts_name.Code).md) будет фактически подменяться упорядочиванием по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок_(ChartOfAccountsObject.Chart_of_accounts_name.Order).md). Другими словами, упорядочивание по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код_(ChartOfAccountsObject.Chart_of_accounts_name.Code).md) и по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок_(ChartOfAccountsObject.Chart_of_accounts_name.Order).md) будут давать одинаковый результат. При этом по умолчанию список счетов будет упорядочиваться по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код_(ChartOfAccountsObject.Chart_of_accounts_name.Code).md).  
Если свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то упорядочивание по полю Код будет фактическим, а по умолчанию список счетов будет упорядочиваться (по полю [Код](ПланСчетовОбъект.Имя_плана_счетов.Код_(ChartOfAccountsObject.Chart_of_accounts_name.Code).md) или по полю [Порядок](ПланСчетовОбъект.Имя_плана_счетов.Порядок_(ChartOfAccountsObject.Chart_of_accounts_name.Order).md)) в зависимости от значения свойства [ДлинаПорядка](ОбъектМетаданных_ПланСчетов.ДлинаПорядка_(MetadataObject_ChartOfAccounts.OrderLength).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
