ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ПоложениеТекстаОтносительноКартинки (TextPositionRelativeToPicture)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [ПоложениеТекстаОтносительноКартинки](ПоложениеТекстаОтносительноКартинки_(TextPositionRelativeToPicture).md).   
Определяет расположение текста в ячейке относительно картинки. Если свойство [РазмерКартинки](ОбластьЯчеекТабличногоДокумента.РазмерКартинки_(SpreadsheetDocumentRange.PictureSize).md) содержит значение [Черепица](РазмерКартинки.Черепица_(PictureSize.Tile).md) или [Растянуть](РазмерКартинки.Растянуть_(PictureSize.Stretch).md), то игнорируется и считается равным [Поверх](ПоложениеТекстаОтносительноКартинки.Поверх_(TextPositionRelativeToPicture.OnTop).md).   
Значение по умолчанию – [Авто](ПоложениеТекстаОтносительноКартинки.Авто_(TextPositionRelativeToPicture.Auto).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
