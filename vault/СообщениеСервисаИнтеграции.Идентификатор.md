СообщениеСервисаИнтеграции (IntegrationServiceMessage)

Идентификатор (ID)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Идентификатор сообщения формируется при отправке сообщения.  
Может использоваться для связывания запроса и ответа: при формировании ответа в [ИдентификаторСообщенияЗапроса](СообщениеСервисаИнтеграции.ИдентификаторСообщенияЗапроса.md) ответного сообщения устанавливается значение параметра <Идентификатор> сообщения запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
