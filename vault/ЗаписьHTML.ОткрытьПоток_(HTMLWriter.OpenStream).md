ЗаписьHTML (HTMLWriter)

ОткрытьПоток (OpenStream)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ОткрытьПоток(<Поток>, <ТипКодировки>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи.

<ТипКодировки> (необязательный)

Тип: [Строка](lang__def_String.md).   
В качестве типа кодировки может быть указана одна из возможных кодировок текста. В этом случае поток будет записан в соответствующей кодировке. При этом в объявлении HTML-документа (если таковое будет записано) будет присутствовать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то для записи файла будет использована кодировка [UTF8](КодировкаТекста.UTF8_(TextEncoding.UTF8).md), а атрибут 'encoding' не будет присутствовать в объявлении HTML-документа.

<ЗаписатьBOM> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли добавлена метка порядка байт (BOM) в начало потока.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Устанавливает поток для записи HTML-документа. Позволяет указать тип кодировки, который будет использован для записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
