# ЭлементDOM.ПолучитьУзелАтрибута

ЭлементDOM (DOMElement)

ПолучитьУзелАтрибута (GetAttributeNode)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ПолучитьУзелАтрибута(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ПолучитьУзелАтрибута(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен атрибута.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [Неопределено](lang__def_Undefined.md).   
Если у элемента нет атрибута с указанными параметрами, то будет возвращено значение [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает узел атрибута по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
