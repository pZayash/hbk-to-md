МеткаВремениПодписи (SignatureTimestamp)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [МеткаВремениКриптографии](МеткаВремениКриптографии_(CryptoTimestamp).md), [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
