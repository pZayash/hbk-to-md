Соединение с агентом сервера (IServerAgentConnection)

RegClusterAdmin (RegClusterAdmin)

Доступен, начиная с версии 8.1.

Синтаксис:

RegClusterAdmin(<Кластер>, <Администратор>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Описание кластера с интерфейсом [Администратор](Администратор.md), в котором необходимо зарегистрировать администратора.

<Администратор> (обязательный)

Тип: [Администратор](Администратор.md).   
Описание регистрируемого администратора. Если в списке администраторов кластера уже присутствует администратор с таким же именем, то описание администратора обновляется, в противном случае добавляется описание нового администратора.

Описание:

Выполняет регистрацию нового администратора кластера. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
