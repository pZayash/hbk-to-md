Описание:

Содержит варианты положения шкалы времени. Шкала времени применяется в диаграмме Ганта и планировщике.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/chart}. Имя типа XDTO: TimeScalePosition.

См. также:

[ШкалаВремени](ШкалаВремени_(TimeScale).md), свойство [Положение](ШкалаВремени.Положение_(TimeScale.Location).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
