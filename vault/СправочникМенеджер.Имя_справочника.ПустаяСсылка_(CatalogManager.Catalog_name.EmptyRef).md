# СправочникМенеджер.<Имя справочника>.ПустаяСсылка

СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

ПустаяСсылка (EmptyRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПустаяСсылка()

Возвращаемое значение:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   

Описание:

Получает пустое значение ссылки на справочник данного вида.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Может использоваться, например, когда нужно передать пустую ссылку в параметр метода.

Пример:

|  |
| --- |
| Если НайденнаяСсылка = Справочники.Валюты.ПустаяСсылка() Тогда     Сообщить("Такой валюты еще нет"); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
