РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

РасчетСистемЛинейныхУравнений

Доступен, начиная с версии 8.3.14.

Синтаксис:

Новый РасчетСистемЛинейныхУравнений()

Описание:

Создает объект и устанавливает следующие значения свойств:

- [ИсточникДанныхУзлов](РасчетСистемЛинейныхУравнений.ИсточникДанныхУзлов.md) - Неопределено,
- [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей.md) – Неопределено,
- [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях.md) – Неопределено,
- [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях.md) – Неопределено,
- [КолонкаУравненияВУзлах](РасчетСистемЛинейныхУравнений.КолонкаУравненияВУзлах.md) – Неопределено,
- [ОписанияСистем](РасчетСистемЛинейныхУравнений.ОписанияСистем.md) – новый объект типа [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений.md) (пустая коллекция),
- [КоличествоИтераций](РасчетСистемЛинейныхУравнений.КоличествоИтераций.md) = 200,
- [ТребуемаяТочность](РасчетСистемЛинейныхУравнений.ТребуемаяТочность.md) = 0.000001,
- [ИспользованиеВычислительныхРесурсов](РасчетСистемЛинейныхУравнений.ИспользованиеВычислительныхРесурсов.md) = 0,
- [ГраницаИзмененияАлгоритмаРешения](РасчетСистемЛинейныхУравнений.ГраницаИзмененияАлгоритмаРешения.md) = 10000.

Использование в версии:

Доступен, начиная с версии 8.3.14.
