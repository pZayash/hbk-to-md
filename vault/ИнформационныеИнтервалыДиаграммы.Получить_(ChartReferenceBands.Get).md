ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Получить (Get)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс информационного интервала, который требуется получить.

Возвращаемое значение:

Тип: [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы_(ChartReferenceBand).md).   

Описание:

Возвращает элемент с указанным индексом.  
Если индекс находится за пределами коллекции, генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
