МакетДокументаОбластиКомпоновкиДанных (DataCompositionAreaDocumentTemplate)

ПлотностьСохраняемыхКартинок (SavedPicturesDensity)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: [ПлотностьСохраняемыхКартинокТабличногоДокумента](ПлотностьСохраняемыхКартинокТабличногоДокумента_(SpreadsheetDocumentSavedPicturesDensity).md).   
Содержит значение плотности картинок, используемое при сохранении макета документа области компоновки данных.  
Значение по умолчанию: [Исходная](ПлотностьСохраняемыхКартинокТабличногоДокумента.Исходная_(SpreadsheetDocumentSavedPicturesDensity.Original).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.22.
