РежимМасштабированияПросмотра (ViewScalingMode)

Авто (Auto)

Доступен, начиная с версии 8.3.4.

Описание:

Режим масштабирования просмотра определяется автоматически:

- для интерфейса [Такси](ВариантИнтерфейсаКлиентскогоПриложения.Такси_(ClientApplicationInterfaceVariant.Taxi).md) - [Крупный](РежимМасштабированияПросмотра.Крупный_(ViewScalingMode.Large).md),
- для интерфейса [Версия8\_2](ВариантИнтерфейсаКлиентскогоПриложения.Версия8_2_(ClientApplicationInterfaceVariant.Version8_2).md) - [Обычный](РежимМасштабированияПросмотра.Обычный_(ViewScalingMode.Normal).md).

Использование в версии:

Доступен, начиная с версии 8.3.4.
