Глобальный контекст (Global context)

КонфигурацияИзменена (ConfigurationChanged)

Доступен, начиная с версии 8.0.

Синтаксис:

КонфигурацияИзменена()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - основная конфигурация отличается от конфигурации базы данных, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет факт отличия основной конфигурации от конфигурации базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
