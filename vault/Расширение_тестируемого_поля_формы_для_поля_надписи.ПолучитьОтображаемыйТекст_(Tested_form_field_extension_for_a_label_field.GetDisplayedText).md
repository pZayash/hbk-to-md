Расширение тестируемого поля формы для поля надписи (Tested form field extension for a label field)

ПолучитьОтображаемыйТекст (GetDisplayedText)

Доступен, начиная с версии 8.3.12.

Синтаксис:

ПолучитьОтображаемыйТекст()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает отображаемое представление значения в поле ввода.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
