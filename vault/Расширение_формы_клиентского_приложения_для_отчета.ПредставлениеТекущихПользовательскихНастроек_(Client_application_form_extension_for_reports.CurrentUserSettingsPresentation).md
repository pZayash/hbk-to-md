Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПредставлениеТекущихПользовательскихНастроек (CurrentUserSettingsPresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит представление текущих пользовательских настроек.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
