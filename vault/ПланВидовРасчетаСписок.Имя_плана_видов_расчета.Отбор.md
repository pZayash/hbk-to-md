Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор.md).   
Содержит объект [Отбор](Отбор.md), по которому осуществляется фильтрация (отбор) видов расчета в списке. Имена элементов отбора совпадают с именами реквизитов вида расчета и именами критериев отбора, затронувших план видов расчета, как они заданы в Конфигураторе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
