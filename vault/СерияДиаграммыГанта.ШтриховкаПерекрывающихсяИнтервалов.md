СерияДиаграммыГанта (GanttChartSeries)

ШтриховкаПерекрывающихсяИнтервалов (OverlappedIntervalsHatch)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если установлено Истина, то перекрывающиеся интервалы выводятся штриховкой, а не сплошным цветом. Цвет выбирается из свойства [ДополнительныйЦвет](СерияДиаграммыГанта.ДополнительныйЦвет.md) с учётом свойства [ПриоритетЦвета](СерияДиаграммыГанта.ПриоритетЦвета.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[СерияДиаграммыГанта](СерияДиаграммыГанта.md), свойство [ПриоритетЦвета](СерияДиаграммыГанта.ПриоритетЦвета.md)  
[СерияДиаграммыГанта](СерияДиаграммыГанта.md), свойство [ДополнительныйЦвет](СерияДиаграммыГанта.ДополнительныйЦвет.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
