ПоГоризонтальнымСтрокамТекста (ByHorizontalTextLines)

Доступен, начиная с версии 8.3.23.

Поворачивать страницы исходя из того, что строки должны располагаться горизонтально.

Доступен, начиная с версии 8.3.23.
