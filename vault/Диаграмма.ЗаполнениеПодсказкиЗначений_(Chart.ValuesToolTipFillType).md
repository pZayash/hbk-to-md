Диаграмма (Chart)

ЗаполнениеПодсказкиЗначений (ValuesToolTipFillType)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [ЗаполнениеПодсказкиЗначенийДиаграммы](ЗаполнениеПодсказкиЗначенийДиаграммы_(ChartValuesToolTipFillType).md).   
Определяет режим заполнения всплывающей подсказки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
