# АтрибутHTML.ЯвляетсяИдентификатором

АтрибутHTML (HTMLAttribute)

ЯвляетсяИдентификатором (IsId)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, является ли данный атрибут идентификатором владеющего им элемента.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Истина](lang__def_BooleanTrue.md) - данный атрибут является идентификатором владеющего им элемента, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Использование в версии:

Доступен, начиная с версии 8.2.
