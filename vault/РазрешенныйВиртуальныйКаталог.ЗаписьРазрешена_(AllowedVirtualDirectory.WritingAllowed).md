РазрешенныйВиртуальныйКаталог (AllowedVirtualDirectory)

ЗаписьРазрешена (WritingAllowed)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, разрешено ли создание файлов, создание каталогов и запись в файлы.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
