ПолеXBase (XBaseField)

Тип (Type)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит тип поля: "S"- строка, "N" - число, "D" - дата, "L" - булево, "F" - число, "M" - мемо поле (не поддерживается).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
