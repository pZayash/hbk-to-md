СообщениеСервисаИнтеграции (IntegrationServiceMessage)

ИдентификаторСообщенияЗапроса (CorrelationId)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Идентификатор сообщения, на которое отправляемое сообщение является ответом.  
Можно устанавливать только при отправке, при попытке изменить в полученном сообщении генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
