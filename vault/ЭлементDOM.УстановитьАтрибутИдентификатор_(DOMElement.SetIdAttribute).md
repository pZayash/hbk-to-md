# ЭлементDOM.УстановитьАтрибутИдентификатор

ЭлементDOM (DOMElement)

УстановитьАтрибутИдентификатор (SetIdAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

УстановитьАтрибутИдентификатор(<Имя>, <ЭтоИД>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

<ЭтоИД> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак идентификатора.

Описание варианта метода:

Если параметр <ЭтоИД> = [Истина](lang__def_BooleanTrue.md), то указанный атрибут становится "идентификатором, определенным пользователем".

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

УстановитьАтрибутИдентификатор(<URIПространстваИмен>, <ЛокальноеИмя>, <ЭтоИД>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута.

<ЭтоИД> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак идентификатора.

Описание варианта метода:

Если значение параметра <ЭтоИД> = [Истина](lang__def_BooleanTrue.md), указанный атрибут становится "идентификатором, определенным пользователем".

Описание:

Устанавливает атрибуту признак идентификатора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Метод не меняет установленной в документе схемы.  
Причины вызова исключений:

- данный узел в режиме Только для чтения;
- указанный атрибут не является атрибутом данного элемента.

Использование в версии:

Доступен, начиная с версии 8.1.
