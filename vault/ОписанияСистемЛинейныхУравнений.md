Элементы коллекции:

[ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы.  
Возможно обращение к к элементу посредством оператора [...]. В качестве аргумента передается индекс элемента (нумерация с 0).

[Удалить (Delete)](ОписанияСистемЛинейныхУравнений.Удалить.md)  

Описание:

Представляет собой коллекцию объектов типа [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.
