КоллекцияМетокИнтервалаФонаПланировщика (PlannerBackgroundIntervalLabelCollection)

Найти (Find)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Найти(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению метки интервала фона планировщика.

Возвращаемое значение:

Тип: [МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика_(PlannerBackgroundIntervalLabel).md), [Неопределено](lang__def_Undefined.md).   
Если указанный элемент отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск метки интервала фона планировщика по значению свойства [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md). Возвращает первую метку с совпадающим значением.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
