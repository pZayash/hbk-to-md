ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

Конец (End)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Произвольный.   
Конец интервала.  
Допускаются следующие типы значений:

- для информационного интервала в значениях серий диаграммы (свойство диаграммы [ИнформационныеИнтервалыЗначений](Диаграмма.ИнформационныеИнтервалыЗначений.md)) - Число;
- для информационного интервала в значениях точек диаграммы (свойство диаграммы [ИнформационныеИнтервалыТочек](Диаграмма.ИнформационныеИнтервалыТочек.md)):
  - если на оси точек откладываются точки - [ТочкаДиаграммы](ТочкаДиаграммы.md);
  - если на оси точек откладываются числа (например, в точечной и пузырьковой диаграммах) - Число;
  - если на оси точек откладываются даты (например, в точечной и пузырьковой диаграмме) - Дата;
- для информационного интервала, расположенного в параметрах вывода компоновки данных:
  - [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных.md);
  - для информационного интервала значений - Число;
  - для информационного интервала точек - тип, используемый в свойстве [Точки](ДиаграммаКомпоновкиДанных.Точки.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
