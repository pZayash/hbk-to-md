Глобальный контекст (Global context)

ВвестиЧисло (InputNumber)

Доступен, начиная с версии 8.0.

Синтаксис:

ВвестиЧисло(<Число>, <Подсказка>, <Длина>, <Точность>)

Параметры:

<Число> (обязательный)

Тип: Число.   
Имя доступной в модуле переменной. В эту переменную будет помещено введенное число. Начальное значение переменной будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: Строка.   
Текст заголовка окна диалога ввода числа. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<Длина> (необязательный)

Тип: Число.   
Длина вводимого числа включая дробную часть (символы разделителей не учитываются).  
Значение по умолчанию: 0.

<Точность> (необязательный)

Тип: Число.   
Количество знаков в дробной части вводимого числа.  
Значение по умолчанию: 0.

Возвращаемое значение:

Тип: Булево.   
Истина - число введено; Ложь - пользователь отказался от ввода.

Описание:

Вызывает диалог для ввода числа.  
При вызове метода в веб-клиенте генерируется исключение.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности.md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать.md), следует использовать метод [ПоказатьВводЧисла](Глобальный_контекст.ПоказатьВводЧисла.md).

Пример:

| --- |
| Количество = 1; Если ВвестиЧисло(Количество, "Введите количество", 10, 2) Тогда     // обработка введенного количества  КонецЕсли; |
