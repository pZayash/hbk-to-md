ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

ДоступныеТипыДиаграммы (AvailableChartTypes)

Доступен, начиная с версии 8.3.13.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Содержит массив типов диаграммы, доступных для выбора в диалоге. Значения, не являющиеся значениями перечисления [ТипДиаграммы](ТипДиаграммы_(ChartType).md), игнорируются. По умолчанию содержит все типы диаграммы.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
