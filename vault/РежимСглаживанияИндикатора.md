Описание:

Содержит варианты стилей заполнения индикатора. Используется для определения свойства [СтильОтображения](Индикатор.СтильОтображения.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент.
