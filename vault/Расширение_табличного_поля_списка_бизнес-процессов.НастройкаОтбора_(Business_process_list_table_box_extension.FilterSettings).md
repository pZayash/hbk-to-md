Расширение табличного поля списка бизнес-процессов (Business process list table box extension)

НастройкаОтбора (FilterSettings)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаОтбора](НастройкаОтбора_(FilterSettings).md).   
Предоставляет доступ к настройкам отбора списка бизнес-процессов. Представляет собой коллекцию значений типа [ЭлементУправленияОтбором](ЭлементУправленияОтбором_(FilterItemControl).md). Свойства коллекции содержат объекты для управления элементом отбора. Имена свойств совпадают с именами доступных для бизнес-процесса элементов отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
