ТипНакопленияСерииДиаграммы (ChartSeriesStackType)

СНакоплениемНормированная (StackedNormalized)

Доступен, начиная с версии 8.3.16.

Описание:

Серия накапливается в процентном соотношении с предыдущей серией с такой же группой накопления.

Доступен, начиная с версии 8.3.16.
