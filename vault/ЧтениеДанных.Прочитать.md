ЧтениеДанных (DataReader)

Прочитать (Read)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Прочитать(<Количество>)

Параметры:

<Количество> (необязательный)

Тип: Число.   
Количество байтов, которые требуется прочитать. Если не задано, то выполняется чтение всех данных до конца потока.  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [РезультатЧтенияДанных](РезультатЧтенияДанных.md).   
Экземпляр объекта [РезультатЧтенияДанных](РезультатЧтенияДанных.md).

Описание:

Выполняет чтение двоичных данных заданного размера из потока. Если размер не указан, будут прочитаны все данные до конца потока.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
