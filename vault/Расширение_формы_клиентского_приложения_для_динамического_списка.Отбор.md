Расширение формы клиентского приложения для динамического списка (Client application form extension for dynamic lists)

Отбор (Filter)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Структура](Структура.md).   
Используется для установки отбора в динамическом списке. Ключ структуры - имя поля основной таблицы.  
Значение элемента структуры содержит значение отбора.  
Если значение является массивом, фиксированным массивом или списком значений с не одним элементом, то отбор будет применяться с видом сравнения [ВСписке](ВидСравнения.ВСписке.md).

Использование в версии:

Доступен, начиная с версии 8.2.
