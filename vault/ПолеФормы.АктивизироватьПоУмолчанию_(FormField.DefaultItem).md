ПолеФормы (FormField)

АктивизироватьПоУмолчанию (DefaultItem)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости активизирования по умолчанию поля в форме, странице или таблице.  
[Истина](lang__def_BooleanTrue.md) - активизация необходима.  
В мобильной платформе установка данного свойства у поля ввода не переводит поле в режим интерактивного редактирования, а только устанавливает значение свойства [ТекущийЭлемент](ФормаКлиентскогоПриложения.ТекущийЭлемент_(ClientApplicationForm.CurrentItem).md) формы в соответствующее значение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
