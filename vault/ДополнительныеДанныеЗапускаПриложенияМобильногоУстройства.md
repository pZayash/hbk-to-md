Описание:

Содержит список дополнительных данных в виде пар: "Ключ – Значение", где:

- Ключ – тип Строка, уникальный идентификатор данных,
- Значение – объект типа [ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства](ЭлементДополнительныхДанныхЗапускаПриложенияМобильногоУстройства.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.5.
