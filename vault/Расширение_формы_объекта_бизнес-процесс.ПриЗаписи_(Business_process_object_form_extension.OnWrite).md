# Расширение формы объекта бизнес-процесс.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [Расширение формы объекта бизнес-процесс](Расширение_формы_объекта_бизнес-процесс_(Business_process_object_form_extension).md) › [События](Расширение_формы_объекта_бизнес-процесс__События.md)

Расширение формы объекта бизнес-процесс (Business process object form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись бизнес-процесса выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](БизнесПроцессОбъект.Имя_бизнес-процесса.ПриЗаписи_(BusinessProcessObject.Business_process_name.OnWrite).md) объекта [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Доступен, начиная с версии 8.0.
