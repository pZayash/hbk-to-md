Расширение табличного поля списка записей регистра накопления (Accumulation register record list table box extension)

ИзменятьАвтоОбновление (ChangeAutoRefresh)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если свойство установлено как Ложь, то пользователь не имеет доступа к интерактивному управлению параметрами автообновления, устанавливаемыми на закладке "Прочее" диалога настройки просмотра списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
