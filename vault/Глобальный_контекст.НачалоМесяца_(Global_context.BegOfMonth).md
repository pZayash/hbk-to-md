# Глобальный контекст.НачалоМесяца

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Функции работы со значениями типа Дата](Функции_работы_со_значениями_типа_Дата.md)

Глобальный контекст (Global context)

НачалоМесяца (BegOfMonth)

Доступен, начиная с версии 8.0.

Синтаксис:

НачалоМесяца(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Исходная дата.

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md).   
Дата и время начала месяца.

Описание:

Определяет дату и время начала месяца для указанной даты.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Сообщить(НачалоМесяца(ТекущаяДата())); |

Использование в версии:

Доступен, начиная с версии 8.0.
