Расширение табличного поля списка параметров анализа данных (Data analysis parameter list table box extension)

НастройкаПараметров (ParametersSetting)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [НастройкаПараметровАнализаДанных](НастройкаПараметровАнализаДанных.md).   
Предоставляет доступ к настройкам параметров анализатора данных. Представляет собой коллекцию значений типа [ЭлементУправленияПараметрамиАнализаДанных](ЭлементУправленияПараметрамиАнализаДанных.md). Свойства коллекции содержат объекты для управления элементом настройки параметров. Наименования свойств совпадают с наименованиями полей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
