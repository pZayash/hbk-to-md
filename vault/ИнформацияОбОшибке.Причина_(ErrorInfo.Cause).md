ИнформацияОбОшибке (ErrorInfo)

Причина (Cause)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке_(ErrorInfo).md), [Неопределено](lang__def_Undefined.md).   
Содержит информацию о причине для данной ошибки. Если таковой не было, то [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
