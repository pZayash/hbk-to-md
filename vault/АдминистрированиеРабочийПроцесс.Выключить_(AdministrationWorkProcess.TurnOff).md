# АдминистрированиеРабочийПроцесс.Выключить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Администрирование сервера "1С:Предприятия"](Администрирование_сервера_1СПредприятия.md) › [АдминистрированиеРабочийПроцесс](АдминистрированиеРабочийПроцесс_(AdministrationWorkProcess).md) › [Методы](АдминистрированиеРабочийПроцесс__Методы.md)

АдминистрированиеРабочийПроцесс (AdministrationWorkProcess)

Выключить (TurnOff)

Доступен, начиная с версии 8.3.27.

Синтаксис:

Выключить()

Описание:

Останавливает рабочий процесс. Процесс будет остановлен после отключения всех пользователей или после истечения времени, заданного свойством [ВремяПринудительногоЗавершения](АдминистрированиеКластер.ВремяПринудительногоЗавершения_(AdministrationCluster.ForceTerminationTime).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.27.
