Элементы коллекции:

[ИсточникСхемыЗапроса](ИсточникСхемыЗапроса.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются источники.  
Возможно обращение к источнику посредством оператора [...]. В качестве аргумента передается индекс.
