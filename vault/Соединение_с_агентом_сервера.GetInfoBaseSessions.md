Соединение с агентом сервера (IServerAgentConnection)

GetInfoBaseSessions (GetInfoBaseSessions)

Доступен, начиная с версии 8.2.

Синтаксис:

GetInfoBaseSessions(<Кластер>, <ИнформационнаяБаза>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, для которого необходимо получить список сеансов. Должен содержать объект [Кластер серверов](Кластер_серверов.md), полученный методом [GetClusters](Соединение_с_агентом_сервера.GetClusters.md).

<ИнформационнаяБаза> (обязательный)

Тип: [Описание информационной базы](Описание_информационной_базы.md).   
Информационная база, для которой необходимо получить список сеансов. Должен содержать объект, полученный методом [GetInfoBases](Соединение_с_агентом_сервера.GetInfoBases.md).

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray.md).   
Массив, содержащий объекты [Сеанс](Сеанс.md).

Описание:

Получить список сеансов информационной базы.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
