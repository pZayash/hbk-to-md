UnregAgentAdmin (UnregAgentAdmin)

Доступен, начиная с версии 8.1.

Синтаксис:

UnregAgentAdmin(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя администратора центрального сервера.

Описание:

Удаляет администратора центрального сервера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
