МенеджерКалендарей (CalendarsManager)

ПоддерживаетсяИзменениеКалендаря (CalendarEditingSupported)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПоддерживаетсяИзменениеКалендаря(<Календарь>)

Параметры:

<Календарь> (обязательный)

Тип: [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря_(LocalCalendarKey).md).   
Локальный ключ календаря.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
На устройствах под управлением Windows Phone всегда возвращает [Ложь](lang__def_BooleanFalse.md).  
Для устройств под управлением Android (начиная с версии 4.1) и iOS всегда возвращает [Истина](lang__def_BooleanTrue.md).

Описание:

Определяет возможность редактирования или удаления календаря.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.10.
