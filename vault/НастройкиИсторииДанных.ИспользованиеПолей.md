НастройкиИсторииДанных (DataHistorySettings)

ИспользованиеПолей (FieldsUse)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие.md).   
Содержит настройки использования для полей объекта метаданных.  
В качестве ключа соответствия используются Строка, имя поля (Реквизит, Измерение, Ресурс, СтандартныйРеквизит, Реквизит ТабличнойЧасти). Для реквизита табличной части указывается имя табличной части.  
Значение имеет тип Булево:

- Истина - поле используется в истории данных,
- Ложь - поле не используется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
