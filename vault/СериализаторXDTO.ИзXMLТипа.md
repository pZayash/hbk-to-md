СериализаторXDTO (XDTOSerializer)

ИзXMLТипа (FromXMLType)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По типу XML

Синтаксис:

ИзXMLТипа(<ТипXML>)

Параметры:

<ТипXML> (обязательный)

Тип: [ТипДанныхXML](ТипДанныхXML.md).   
Тип XML, для которого нужно получить соответствующий тип.

Описание варианта метода:

Возвращает тип, соответствующий типу данных XML.

Вариант синтаксиса: По имени типа и URI пространства имен

Синтаксис:

ИзXMLТипа(<ИмяТипа>, <URIПространстваИмен>)

Параметры:

<ИмяТипа> (обязательный)

Тип: Строка.   
Имя типа XML.

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен типа XML.

Описание варианта метода:

Возвращает тип, соответствующий типу данных XML и URI пространства имен.

Возвращаемое значение:

Тип: Тип, Неопределено.   
Неопределено, если соответствующий тип данных не обнаружен.

Описание:

Получает тип, соответствующий типу данных XML.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
