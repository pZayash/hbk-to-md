# Расширение формы набора записей регистра накопления.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры накопления](Регистры_накопления.md) › [Расширение формы набора записей регистра накопления](Расширение_формы_набора_записей_регистра_накопления_(Accumulation_register_record_set_form_extension).md) › [События](Расширение_формы_набора_записей_регистра_накопления__События.md)

Расширение формы набора записей регистра накопления (Accumulation register record set form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.ПриЗаписи_(AccumulationRegisterRecordSet.Accumulation_register_name.OnWrite).md) объекта [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления_(AccumulationRegisterRecordSet.Accumulation_register_name).md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления_(AccumulationRegisterRecordSet.Accumulation_register_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
