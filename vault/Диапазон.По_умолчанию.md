Диапазон (Range)

По описанию диапазона

Доступен, начиная с версии 8.1.

Синтаксис:

Новый Диапазон(<Начало>, <Конец>)

Параметры:

<Начало> (необязательный)

Тип: [Дата](lang__def_Date.md), [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Нижняя граница диапазона. Если не задана, считается, что диапазон снизу не ограничен.

<Конец> (необязательный)

Тип: [Дата](lang__def_Date.md), [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Верхняя граница диапазона. Если не задана, считается, что диапазон не ограничен сверху.

Описание:

Создает объект.

Использование в версии:

Доступен, начиная с версии 8.1.
