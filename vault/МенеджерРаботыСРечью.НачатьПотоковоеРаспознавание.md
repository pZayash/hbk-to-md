МенеджерРаботыСРечью (SpeechProcessingManager)

НачатьПотоковоеРаспознавание (StartStreamingSpeechToText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

НачатьПотоковоеРаспознавание(<Ключ>, <ОбработчикРаспознавания>, <ПараметрыМодели>, <ВариантИспользованияРасположенияРаботыСРечью>, <ПараметрыПотоковогоРаспознавания>)

Параметры:

<Ключ> (обязательный)

Тип: Строка.   
Идентификатор потокового распознавания.  
Если уже есть активное задание с указанным идентификатором, генерируется исключение.  
Внимание! Ключ должен быть уникальным для области данных, в которой выполняется распознавание.

<ОбработчикРаспознавания> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Процедура, которая будет вызвана по мере появления новых гипотез распознавания со следующими параметрами:

- <РезультатРаспознавания> - тип [РезультатРаспознаванияРечи](РезультатРаспознаванияРечи.md),
- <ПродолжитьРаспознавание> - тип Булево, признак необходимости продолжения работы (по умолчанию, Истина),
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ПараметрыМодели> (необязательный)

Тип: [ПараметрыМоделиРаспознаванияРечи](ПараметрыМоделиРаспознаванияРечи.md).   
Параметры модели распознавания речи с указанным кодом языка.  
По умолчанию используется язык текущего сеанса, остальные параметры не определены.

<ВариантИспользованияРасположенияРаботыСРечью> (необязательный)

Тип: [ВариантИспользованияРасположенияРаботыСРечью](ВариантИспользованияРасположенияРаботыСРечью.md).   
Вариант использования расположения модели работы с речью.  
Значение по умолчанию: [Авто](ВариантИспользованияРасположенияРаботыСРечью.Авто.md).

<ПараметрыПотоковогоРаспознавания> (необязательный)

Тип: [ПараметрыПотоковогоРаспознаванияРечи](ПараметрыПотоковогоРаспознаванияРечи.md).   
Дополнительные параметры потокового распознавания.

Описание:

Запускает захват звука с микрофона и отправку потока для распознавания.  
  
Запуск потокового распознавания речи c с помощью данного метода рекомендуется выполнять только интерактивно, чтобы пользователь явно знал, что действие приводит к захвату звука с целью распознавания речи.  
  
Поиск модели распознавания речи выполняется в следующем порядке:

- Проверяется наличие подключенной подходящей модели для локального распознавания - если обнаружена, используется локальная.
- Если локальная модель не обнаружена, выполняется обращение в облачном сервисе.
- Если в облачном сервисе модель не обнаружена, генерируется исключение.

При попытке запустить более одного потока распознавания по одному и тому же ключу, генерируется исключение.  
  
При использовании внешнего распознавания речи, дополнительные грамматики, установленные с помощью метода [УстановитьДополнительнуюГрамматикуИнформационнойБазы](МенеджерРаботыСРечью.УстановитьДополнительнуюГрамматикуИнформационнойБазы.md), не доступны.  
  
В операционных системах MS Windows Server 2003/Windows XP при вызове данного метода генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
