# ОбъектМетаданных: Ресурс.РежимПароля

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: Ресурс](ОбъектМетаданных_Ресурс_(MetadataObject_Resource).md) › [Свойства](ОбъектМетаданных_Ресурс__Свойства.md)

ОбъектМетаданных: Ресурс (MetadataObject: Resource)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает отображение данных ресурса в режиме пароля (символы заменяются звездочками при отображении).  
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Имеет смысл для ресурсов типа [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
