АдминистрированиеПрофильБезопасности (AdministrationSecurityProfile)

ПолныйДоступКВнешнимКомпонентам (AddInFullAccess)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешено использование произвольных внешних компонент.

- [Истина](lang__def_BooleanTrue.md) – разрешено использование произвольных внешних компонент.
- [Ложь](lang__def_BooleanFalse.md) – допустимо использование только внешних компонентов, перечисленных в списке разрешенных внешних компонент профиля.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
