# Расширение формы вида расчета.ПриЗаписи

Расширение формы вида расчета (Calculation type form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета.ПриЗаписи_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name.OnWrite).md) объекта [ПланВидовРасчетаОбъект.](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
