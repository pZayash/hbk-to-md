Соединение с рабочим процессом (IWorkingProcessConnection)

Disconnect (Disconnect)

Доступен, начиная с версии 8.1.

Синтаксис:

Disconnect(<Соединение>)

Параметры:

<Соединение> (обязательный)

Тип: [Соединение](Соединение.md).   
Описание соединения с информационной базой. Объект может быть получен ка элемент массива, возвращаемого методом [GetInfoBaseConnections](Соединение_с_рабочим_процессом.GetInfoBaseConnections.md).

Описание:

Разрывает соединение клиента с рабочим процессом.

Доступность:

Интеграция.

Примечание:

После разрыва соединения соответствующее клиентское приложение завершится аварийно.

Использование в версии:

Доступен, начиная с версии 8.1.
