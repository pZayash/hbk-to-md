СредстваТелефонии (TelephonyTools)

ПослатьSMS (SendSMS)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПослатьSMS(<SMSСообщение>, <Интерактивно>)

Параметры:

<SMSСообщение> (обязательный)

Тип: [SMSСообщение](SMSСообщение_(SMSMessage).md).   
Содержит передаваемое SMS-сообщение.

<Интерактивно> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, используется ли интерактивный режим для отправки SMS-сообщений.

Описание:

Если параметр <Интерактивно> установлен в значение [Истина](lang__def_BooleanTrue.md), то будет запущено системное приложение для отправки SMS-сообщений.   
В противном случае, SMS-сообщение будет отправлено средствами мобильной платформы.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать один из методов:

- [ПоддерживаетсяОтправкаSMS](СредстваТелефонии.ПоддерживаетсяОтправкаSMS_(TelephonyTools.SMSSendingSupported).md) - если вложения не используются,
- [ПоддерживаетсяОтправкаMMS](СредстваТелефонии.ПоддерживаетсяОтправкаMMS_(TelephonyTools.MMSSendingSupported).md) - если вложения используются.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
