Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаЭлементеШкалыВремени (TimeScaleItemClick)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НажатиеНаЭлементеШкалыВремени(<ЭлементШкалыВремени>, <Дата>)

Параметры:

<ЭлементШкалыВремени>

Тип: [ЭлементШкалыВремени](ЭлементШкалыВремени.md).   
Элемент шкалы времени, на котором было произведено нажатие.

<Дата>

Тип: Дата.   
Дата, на которую было произведено нажатие.

Описание:

Возникает при нажатии левой кнопкой мыши на элементе шкалы времени в случае, если свойство поля планировщика [ГиперссылкаЭлементаШкалыВремени](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаЭлементаШкалыВремени.md) имеет значение Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
