КнопкаКоманднойПанели (CommandBarButton)

ПорядокКнопок (ButtonOrder)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ПорядокКнопокКоманднойПанели](ПорядокКнопокКоманднойПанели_(CommandBarButtonOrder).md).   
Признак, определяющий сортировку кнопок по тексту подменю в пределах разделителей.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только для кнопок, у которых установлено значение свойства [ТипКнопки](КнопкаКоманднойПанели.ТипКнопки_(CommandBarButton.ButtonType).md) равное [Подменю](ТипКнопкиКоманднойПанели.Подменю_(CommandBarButtonType.Popup).md).

Использование в версии:

Доступен, начиная с версии 8.0.
