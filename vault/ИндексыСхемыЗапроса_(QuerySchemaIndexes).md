Доступен, начиная с версии 8.3.25.

Элементы коллекции:

[ИндексСхемыЗапроса](ИндексСхемыЗапроса_(QuerySchemaIndex).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются таблицы для изменения.  
Возможно обращение к таблице для изменения посредством оператора [...]. В качестве аргумента передается индекс.

Описание:

Представляет собой коллекцию описаний индексов схемы запроса (объекты типа [ИндексСхемыЗапроса](ИндексСхемыЗапроса_(QuerySchemaIndex).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
