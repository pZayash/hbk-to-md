ПараметрыАудиозаписи (AudioRecordingParameters)

По умолчанию

Доступен, начиная с версии 8.3.17.

Синтаксис:

Новый ПараметрыАудиозаписи(<ФорматАудиозаписи>, <ИспользованиеКаналовАудиозаписи>, <ЧастотаДискретизации>, <СкоростьКодирования>)

Параметры:

<ФорматАудиозаписи> (необязательный)

Устаналивает используемый формат аудиозаписи.  
Значение по умолчанию: Mpeg4AAC.

<ИспользованиеКаналовАудиозаписи> (необязательный)

Тип: [ИспользованиеКаналовАудиозаписи](ИспользованиеКаналовАудиозаписи_(AudioRecordingChannelUse).md).   
Определяет способ использования каналов аудиозаписи.  
Значение по умолчанию: [Моно](ИспользованиеКаналовАудиозаписи.Моно_(AudioRecordingChannelUse.Mono).md).

<ЧастотаДискретизации> (необязательный)

Тип: [Число](lang__def_Number.md).   
Устанавливает частоту дискретизации звука для записи.  
Для кодировки AАC рекомендуемые частоты: 8000, 11025, 22050, 44100 и 96000.  
Значение по умолчанию: 44100.

<СкоростьКодирования> (необязательный)

Тип: [Число](lang__def_Number.md).   
Устанавливает скорость кодирования звука в битах в секунду для записи.  
Значение по умолчанию: 128000.

Описание:

Создает объект, содержащий параметры аудиозаписи.

Использование в версии:

Доступен, начиная с версии 8.3.17.
