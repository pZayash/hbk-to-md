ЗаписьДанных (DataWriter)

НачатьЗакрытие (BeginClose)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗакрытие(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Инициирует вызов метода [СброситьБуферы](ЗаписьДанных.СброситьБуферы.md). Если целевой поток был создан при создании объекта [ЗаписьДанных](ЗаписьДанных.md), также вызывает начало закрытие целевого потока.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

См. также:

[ЗаписьДанных](ЗаписьДанных.md), метод [Закрыть](ЗаписьДанных.Закрыть.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗакрытьАсинх](ЗаписьДанных.ЗакрытьАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.18.
