ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

УстановитьПараметрыСоединенияСеанса (SetSessionConnectionParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьПараметрыСоединенияСеанса(<ПараметрыСоединения>)

Параметры:

<ПараметрыСоединения> (обязательный)

Тип: [ПараметрыСоединенияВнешнегоИсточникаДанных](ПараметрыСоединенияВнешнегоИсточникаДанных_(ExternalDataSourceConnectionParameters).md).   
Параметры соединения, которые нужно установить.

Описание:

Устанавливает параметры соединения с внешним источником данных для текущего сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если у пользователя нет права на изменение свойства и значение свойства изменено, то выдается исключение. Например, у пользователя нет права ИзменениеСтандартнойАутентификацииСеанса и в параметре было установлено свойство [ИмяПользователя](ПараметрыСоединенияВнешнегоИсточникаДанных.ИмяПользователя_(ExternalDataSourceConnectionParameters.UserName).md), то будет выдано исключение.  
Для свойств [ИмяПользователя](ПараметрыСоединенияВнешнегоИсточникаДанных.ИмяПользователя_(ExternalDataSourceConnectionParameters.UserName).md), [Пароль](ПараметрыСоединенияВнешнегоИсточникаДанных.Пароль_(ExternalDataSourceConnectionParameters.Password).md), [ПарольУстановлен](ПараметрыСоединенияВнешнегоИсточникаДанных.ПарольУстановлен_(ExternalDataSourceConnectionParameters.PasswordIsSet).md) проверяется права ИзменениеСтандартнойАутентификацииСеанса.  
Для остальных свойств проверяются административные права.

Использование в версии:

Доступен, начиная с версии 8.2.
