# ОбработкаМенеджер.<Имя обработки>.ПолучитьМакет

ОбработкаМенеджер.<Имя обработки> (DataProcessorManager.<Data processor name>)

ПолучитьМакет (GetTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьМакет(<Макет>)

Параметры:

<Макет> (обязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: Макет](ОбъектМетаданных_Макет_(MetadataObject_Template).md).   
Имя макета, как оно задано в конфигураторе, или объект описания метаданного требуемого макета.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ТекстовыйДокумент](ТекстовыйДокумент_(TextDocument).md); другой объект, который может быть макетом.   

Описание:

Получает макет обработки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ИмяМакета = "ОбработкаДокументов"; Макет = Обработки.ОбработкаДокументов.ПолучитьМакет(ИмяМакета); |

Использование в версии:

Доступен, начиная с версии 8.0.
