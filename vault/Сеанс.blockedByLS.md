Сеанс (ISessionInfo)

blockedByLS (blockedByLS)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Число.   
Содержит номер сеанса, который является причиной ожидания управляемой транзакционной блокировки, в случае, если сеанс выполняет установку управляемых транзакционных блокировок и ожидает блокировки, установленные другим сеансом. Иначе - 0.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
