ИспользованиеКопийБазыДанных (DatabaseCopiesUse)

НеИспользоватьКопии (DontUseCopies)

Доступен, начиная с версии 8.3.19.

Описание:

Не использовать копии базы данных, использовать только основную базу данных.

Использование в версии:

Доступен, начиная с версии 8.3.19.
