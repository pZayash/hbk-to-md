<Имя ресурса>Расход (<Имя ресурса>Expense)

Тип: Число.   
Набор полей содержит суммы всех движений [Расход](ВидДвиженияНакопления.Расход.md) по ресурсам регистра. Имена полей соответствуют именам ресурсов, как они заданы в конфигураторе, с добавлением слова [Расход](ВидДвиженияНакопления.Расход.md).  

Примечание:

Существует только для регистров остатков.
