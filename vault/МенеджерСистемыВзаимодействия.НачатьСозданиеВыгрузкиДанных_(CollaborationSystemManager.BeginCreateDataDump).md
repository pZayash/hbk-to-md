МенеджерСистемыВзаимодействия (CollaborationSystemManager)

НачатьСозданиеВыгрузкиДанных (BeginCreateDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

НачатьСозданиеВыгрузкиДанных(<Описание>)

Параметры:

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   

Описание:

Начинает создание выгрузки данных системы взаимодействия.  
Вызов метода возвращает управление при начале процесса создания выгрузки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [НачатьСозданиеВыгрузкиДанныхАсинх](МенеджерСистемыВзаимодействия.НачатьСозданиеВыгрузкиДанныхАсинх_(CollaborationSystemManager.BeginCreateDataDumpAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
