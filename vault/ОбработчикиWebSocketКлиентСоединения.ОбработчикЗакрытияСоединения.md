ОбработчикиWebSocketКлиентСоединения (WebSocketClientConnectionHandlers)

ОбработчикЗакрытияСоединения (OnCloseHandler)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя экспортируемой процедуры-обработчика события закрытия соединения со следующими параметрами:

- <Соединение> - объект типа [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md).
- <КодЗакрытия > - объект типа Число. Содержит статус-код закрытия соединения:
  - 1005 - если не получен.

Пустая строка - если это соединение [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
