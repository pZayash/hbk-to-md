СоединениеССерверомСистемыАналитики (AnalyticsSystemServerConnection)

ЗаписатьСхему (WriteSchema)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ЗаписатьСхему(<Путь>, <Схема>)

Параметры:

<Путь> (обязательный)

Тип: Строка.   
Путь для записи схемы.

<Схема> (обязательный)

Тип: [СхемаСистемыАналитики](СхемаСистемыАналитики.md).   
Схема для записи.

Описание:

Записывает схему по переданному пути в системе аналитики.  
Содержание параметров определяется сервером системы аналитики и описано в его документации.

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
