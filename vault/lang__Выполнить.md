# Выполнить

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › [Встроенный язык 1С](_index__Встроенный_язык_1С.md)

**Синтаксис:**Выполнить(<Строка>)

**Параметры:**  
<Строка>  
Строка, содержащая текст исполняемого кода.

**Описание:**Позволяет выполнить фрагмент кода, который передается ему в качестве строкового значения.

**Примечание:**  
В режиме запуска веб-клиент оператор не поддерживается, при его вызове будет сгенерировано исключение.

Данный оператор не поддерживается при работе через COM-соединение.

В режимах запуска мобильного приложения оператор не поддерживается в операционной системе iOS.

**Использование в версии:**

Доступен, начиная с версии 8.0.
