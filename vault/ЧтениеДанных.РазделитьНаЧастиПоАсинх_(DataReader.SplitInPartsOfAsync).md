# ЧтениеДанных.РазделитьНаЧастиПоАсинх

ЧтениеДанных (DataReader)

РазделитьНаЧастиПоАсинх (SplitInPartsOfAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

РазделитьНаЧастиПоАсинх(<РазмерЧасти>)

Параметры:

<РазмерЧасти> (обязательный)

Тип: [Число](lang__def_Number.md).   
Размер части данных в байтах.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив значений типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Описание:

Асинхронная функция. Делит остаток данных на части заданного размера.  
Размер части задается в байтах.

Доступность:

Тонкий клиент, толстый клиент.
