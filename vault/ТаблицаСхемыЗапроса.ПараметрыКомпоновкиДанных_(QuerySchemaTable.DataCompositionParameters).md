ТаблицаСхемыЗапроса (QuerySchemaTable)

ПараметрыКомпоновкиДанных (DataCompositionParameters)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыКомпоновкиДанныхТаблицыСхемыЗапроса](ПараметрыКомпоновкиДанныхТаблицыСхемыЗапроса_(QuerySchemaTableDataCompositionParameters).md).   
Содержит коллекцию параметры компоновки данных таблицы схемы запроса.  
Соответствует параметрам в фигурных скобках в тексте запроса.  
Например:  

|  |
| --- |
| ТоварныеЗапасыОстатки( {Парам}){exampl} |

ТоварныеЗапасыОстатки( {Парам})

|  |
| --- |
|  |

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
