ОбъектМетаданных: ПланВидовРасчета (MetadataObject: ChartOfCalculationTypes)

ОбновлениеПредопределенныхДанных (PredefinedDataUpdate)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [ОбновлениеПредопределенныхДанных](Системные_перечисления.ОбновлениеПредопределенныхДанных.md).   
Содержит режим обновления предопределенных данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для режимов [Авто](ОбновлениеПредопределенныхДанных.ОбновлениеПредопределенныхДанных.Авто.md) и [ОбновлятьАвтоматически](ОбновлениеПредопределенныхДанных.ОбновлениеПредопределенныхДанных.ОбновлятьАвтоматически.md) дополнительно учитывается флаг, установленный в базе данных с помощью метода [УстановитьОбновлениеПредопределенныхДанных](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.УстановитьОбновлениеПредопределенныхДанных.md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
