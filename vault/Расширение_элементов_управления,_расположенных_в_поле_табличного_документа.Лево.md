Расширение элементов управления, расположенных в поле табличного документа (Extension for controls located in a spreadsheet document field)

Лево (Left)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит положение левой границы элемента управления, отсчитываемое от левой границы табличного документа (в миллиметрах).

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
