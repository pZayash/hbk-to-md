Форма (Form)

Модифицированность (Modified)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Признак изменения (модифицированности) данных в форме. Модифицированность отображается символом (\*) в заголовке окна формы.  
Истина - изменена.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
