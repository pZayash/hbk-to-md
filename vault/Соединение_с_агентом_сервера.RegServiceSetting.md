Соединение с агентом сервера (IServerAgentConnection)

RegServiceSetting (RegServiceSetting)

Доступен, начиная с версии 8.3.26.

Синтаксис:

RegServiceSetting(<Кластер>, <Сервер>, <Настройка>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер серверов, для которого требуется создать настройку.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Сервер, для которого требуется нужно создать настройку.

<Настройка> (обязательный)

Тип: [Настройка сервиса](Настройка_сервиса.md).   
Настройка сервиса кластера.

Описание:

Обновляет или добавляет новую настройку сервиса кластера.

Доступность:

Интеграция.

Примечание:

При обновлении настройки не допускается изменение имен сервиса и информационной базы.

Использование в версии:

Доступен, начиная с версии 8.3.26.
