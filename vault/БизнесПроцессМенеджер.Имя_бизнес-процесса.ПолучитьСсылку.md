БизнесПроцессМенеджер.<Имя бизнес-процесса> (BusinessProcessManager.<Business process name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [БизнесПроцессСсылка.\<Имя бизнес-процесса\>](БизнесПроцессСсылка.Имя_бизнес-процесса.md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор.md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](БизнесПроцессСсылка.Имя_бизнес-процесса.УникальныйИдентификатор.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

Доступен, начиная с версии 8.0.
