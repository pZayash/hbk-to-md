# МенеджерРаботыСРечью.ВыполнитьОтложенноеРаспознавание

МенеджерРаботыСРечью (SpeechProcessingManager)

ВыполнитьОтложенноеРаспознавание (ExecuteDelayedSpeechToText)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ВыполнитьОтложенноеРаспознавание(<Данные>, <ПараметрыМодели>, <ВариантИспользованияРасположенияРаботыСРечью>)

Параметры:

<Данные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Данные для распознавания речи.

<ПараметрыМодели> (необязательный)

Тип: [ПараметрыМоделиРаспознаванияРечи](ПараметрыМоделиРаспознаванияРечи_(SpeechToTextModelParameters).md).   
Параметры модели распознавания речи.

<ВариантИспользованияРасположенияРаботыСРечью> (необязательный)

Тип: [ВариантИспользованияРасположенияРаботыСРечью](ВариантИспользованияРасположенияРаботыСРечью_(SpeechProcessingLocationUseVariant).md).   
Вариант использования расположения работы с речью.  
В мобильной платформе значение по умолчанию - [ТолькоВнешнее](ВариантИспользованияРасположенияРаботыСРечью.ТолькоВнешнее_(SpeechProcessingLocationUseVariant.ExternalOnly).md).  
Значение по умолчанию: [ТолькоЛокальное](ВариантИспользованияРасположенияРаботыСРечью.ТолькоЛокальное_(SpeechProcessingLocationUseVariant.LocalOnly).md).

Возвращаемое значение:

Тип: [ИдентификаторОтложенногоРаспознаванияРечи](ИдентификаторОтложенногоРаспознаванияРечи_(SpeechToTextDelayedID).md).   

Описание:

Отправляет сервису распознавания задание на распознавание файла. Отправка файла в сервис распознавания речи является синхронной.   
Ожидание результата является асинхронным: по окончании вызова метода файл будет передан в сервис распознавания речи, но не будет распознан.  
Получение результатов заданий выполняется в рамках вызова метода [ПолучитьРезультатыОтложенныхРаспознаваний](МенеджерРаботыСРечью.ПолучитьРезультатыОтложенныхРаспознаваний_(SpeechProcessingManager.GetDelayedSpeechToTextResults).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.25.
