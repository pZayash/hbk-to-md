ЭлементDOM (DOMElement)

Префикс (Prefix)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Префикс узла DOM.   
Свойство имеет значение Неопределено для узлов, отличных [Атрибут](ТипУзлаDOM.Атрибут.md) и [Элемент](ТипУзлаDOM.Элемент.md), или для узлов, созданных при помощи метода DOM Level 1 (например, [СоздатьЭлемент](ДокументDOM.СоздатьЭлемент.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
