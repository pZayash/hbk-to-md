# БизнесПроцессОбъект.<Имя бизнес-процесса>.УстановитьПометкуУдаления

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

УстановитьПометкуУдаления (SetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьПометкуУдаления(<ПометкаУдаления>)

Параметры:

<ПометкаУдаления> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак установки / снятие пометки на удаление.  
[Истина](lang__def_BooleanTrue.md) - пометка будет установлена; [Ложь](lang__def_BooleanFalse.md) - пометка будет снята.

Описание:

Устанавливает/снимает пометку на удаление бизнес-процесса. Бизнес-процесс при этом не удаляется из базы данных. Окончательное удаление помеченных бизнес-процессов происходит при удалении помеченных объектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Применение метода влечет инициирование события [ПередЗаписью](БизнесПроцессОбъект.Имя_бизнес-процесса.ПередЗаписью_(BusinessProcessObject.Business_process_name.BeforeWrite).md), т.к. происходит сохранение измененного свойства [ПометкаУдаления](БизнесПроцессОбъект.Имя_бизнес-процесса.ПометкаУдаления_(BusinessProcessObject.Business_process_name.DeletionMark).md).  
Если объект модифицирован, то выполнение метода вызывает исключение.

Пример:

|  |
| --- |
| ВыбБизнесПроцесс.УстановитьПометкуУдаления(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
