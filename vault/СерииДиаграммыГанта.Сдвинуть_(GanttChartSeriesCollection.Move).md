СерииДиаграммыГанта (GanttChartSeriesCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Серия>, <Смещение>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md).   
Серия диаграммы Ганта.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество серий, на которое необходимо переместить указанную. Положительное значение означает, что серия будет передвинута ближе к концу диаграммы (вниз), отрицательное - ближе к началу (вверх).

Описание:

Сдвигает указанную серию на указанное смещение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
