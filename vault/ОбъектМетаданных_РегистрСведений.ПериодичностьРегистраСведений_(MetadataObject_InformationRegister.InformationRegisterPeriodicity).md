ОбъектМетаданных: РегистрСведений (MetadataObject: InformationRegister)

ПериодичностьРегистраСведений (InformationRegisterPeriodicity)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ПериодичностьРегистраСведений](ПериодичностьРегистраСведений_(InformationRegisterPeriodicity).md).   
Указывает, с какой периодичностью регистр сохраняет значения ресурсов (например, [Месяц](ПериодичностьРегистраСведений.Месяц_(InformationRegisterPeriodicity.Month).md), [День](ПериодичностьРегистраСведений.День_(InformationRegisterPeriodicity.Day).md), [Секунда](ПериодичностьРегистраСведений.Секунда_(InformationRegisterPeriodicity.Second).md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
