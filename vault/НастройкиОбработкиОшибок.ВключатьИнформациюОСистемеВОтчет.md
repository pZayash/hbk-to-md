ВключатьИнформациюОСистемеВОтчет (IncludeSystemInfoInReport)

Не рекомендуется использовать, начиная с версии 8.3.21.

Рекомендуется использовать:

- [ВключатьИнформациюОСистемеВОтчетНаКлиенте](НастройкиОбработкиОшибок.ВключатьИнформациюОСистемеВОтчетНаКлиенте.md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке.md).   
Указывает необходимость добавлять в отчет об ошибке информацию о компьютере пользователя.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто.md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.21.

Описание изменено в версии 8.3.21.
