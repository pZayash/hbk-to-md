Описание:

Тип: Строка.   
Имя сервера соответствующее [СерверБазыДанных](МенеджерКопииБазыДанных.СерверБазыДанных.md) или имя компьютера (host), на котором запущен Дата акселератор. Для разных экземпляров копии может содержать разное значение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
