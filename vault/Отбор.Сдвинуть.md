Отбор (Filter)

Сдвинуть (Move)

Доступен, начиная с версии 8.0.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс сдвигаемого элемента отбора.

<Смещение> (обязательный)

Тип: Число.   
Смещение сдвига. Если положительное, то смещение производится вперед, если отрицательное - назад.

Описание:

Сдвигает элемент отбора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
