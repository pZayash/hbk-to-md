Расширение колонки табличного поля списка записей регистра сведений (Information register record list table box column extension)

ОтображатьСтандартнуюКартинку (DisplayDefaultPicture)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак отображения в колонке стандартной картинки, обозначающей запись регистра сведений и ее состояние.

Доступность:

Толстый клиент.

Примечание:

Если [Истина](lang__def_BooleanTrue.md), то при выводе свойство [КартинкиСтрок](КолонкаТабличногоПоля.КартинкиСтрок_(TableBoxColumn.RowsPictures).md) колонки табличного поля не используется.

Использование в версии:

Доступен, начиная с версии 8.0.
