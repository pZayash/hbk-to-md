# РежимПолученияАрхиваФайлов.ПолучатьАрхивПриНеобходимости

РежимПолученияАрхиваФайлов (GetFilesArchiveMode)

ПолучатьАрхивПриНеобходимости (GetArchiveWhenRequired)

Доступен, начиная с версии 8.3.15.

Описание:

В веб-клиенте, если не установлено расширение работы с файлами, файлы упаковываются в архив, если они получены с помощью метода [НачатьПолучениеФайловССервера](Глобальный_контекст.НачатьПолучениеФайловССервера.md). Во всех остальных случаях не упаковывать файлы в архив.

Использование в версии:

Доступен, начиная с версии 8.3.15.
