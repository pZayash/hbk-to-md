МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьВремяОкончанияДоступностиОтвета (GetAnswerAvailabilityEndTime)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьВремяОкончанияДоступностиОтвета(<Обсуждение>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Идентификатор обсуждения.

Возвращаемое значение:

Тип: [Дата](lang__def_Date.md), [Неопределено](lang__def_Undefined.md).   
Дата возвращается в часовом поясе текущего сеанса.  
Для интеграции других типов возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Для интеграции типа WhatsApp получает время окончания 24-х часового окна для ответа.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.23.
