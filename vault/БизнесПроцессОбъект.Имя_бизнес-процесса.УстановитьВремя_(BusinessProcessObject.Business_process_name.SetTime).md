# БизнесПроцессОбъект.<Имя бизнес-процесса>.УстановитьВремя

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Бизнес-процессы](Бизнес-процессы.md) › [БизнесПроцессОбъект.<Имя бизнес-процесса>](БизнесПроцессОбъект.Имя_бизнес-процесса_(BusinessProcessObject.Business_process_name).md) › [Методы](БизнесПроцессОбъект.Имя_бизнес-процесса__Методы.md)

БизнесПроцессОбъект.<Имя бизнес-процесса> (BusinessProcessObject.<Business process name>)

УстановитьВремя (SetTime)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьВремя(<РежимОпределенияВремени>)

Параметры:

<РежимОпределенияВремени> (необязательный)

Тип: [РежимАвтоВремя](РежимАвтоВремя_(AutoTimeMode).md).   
Устанавливает режим автоматического определения времени при записи нового бизнес-процесса.  
Значение по умолчанию: [ТекущееИлиПоследним](РежимАвтоВремя.ТекущееИлиПоследним_(AutoTimeMode.CurrentOrLast).md).

Описание:

Устанавливает режим определения времени при записи бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
