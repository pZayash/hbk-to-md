# Расширение элементов управления, расположенных в форме.КонтекстноеМеню

Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

КонтекстноеМеню (ContextMenu)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Неопределено](lang__def_Undefined.md), [КоманднаяПанель](КоманднаяПанель_(CommandBar).md), [КнопкаКоманднойПанели](КнопкаКоманднойПанели_(CommandBarButton).md).   
Свойство определяет, какое контекстное меню будет показано у элемента управления. Значением свойства может являться элемент управления типа [КоманднаяПанель](КоманднаяПанель_(CommandBar).md) или одна из его кнопок-подменю. Если значением является [Неопределено](lang__def_Undefined.md), то контекстное меню открыто не будет.

Доступность:

Толстый клиент.

Примечание:

При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

См. также:

[Расширение элементов управления, расположенных в форме](Расширение_элементов_управления,_расположенных_в_форме_(Extension_for_controls_located_in_a_form).md), свойство [АвтоКонтекстноеМеню](Расширение_элементов_управления,_расположенных_в_форме.АвтоКонтекстноеМеню_(Extension_for_controls_located_in_a_form.AutoContextMenu).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
