ДанныеПереходаПоНавигационнойСсылке (URLNavigationData)

ДополнительныеДанныеПереходаВМобильноеПриложение (MobileApplicationNavigationAdditionalData)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства](ДополнительныеДанныеЗапускаПриложенияМобильногоУстройства.md).   
Дополнительные параметры, которые будут заполнены в случае, если переход в приложение выпол выполнен с помощью объекта Intent в ОС Android.   
В остальных операционных системах всегда Неопределено.

Доступность:

Тонкий клиент, веб-клиент, сервер, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
