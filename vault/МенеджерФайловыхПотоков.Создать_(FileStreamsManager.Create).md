МенеджерФайловыхПотоков (FileStreamsManager)

Создать (Create)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Создать(<ИмяФайла>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя создаваемого файла.

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера.

Возвращаемое значение:

Тип: [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   

Описание:

Создает или перезаписывает файл и открывает поток с возможностью чтения и записи в файл. Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
