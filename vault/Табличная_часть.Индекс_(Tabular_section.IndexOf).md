Табличная часть (Tabular section)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка табличной части](Строка_табличной_части_(Line_of_a_tabular_section).md).   
Строка табличной части, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанной строки табличной части. Если не найдено, то возвращается -1.

Описание:

Получает индекс строки в табличной части.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ИндексТекСтроки = Состав.Индекс(ЭлементыФормы.Состав.ТекущаяСтрока); |

Использование в версии:

Доступен, начиная с версии 8.0.
