ГруппаФормы (FormGroup)

ПолучитьДействие (GetAction)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьДействие(<ИмяСобытия>)

Параметры:

<ИмяСобытия> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя события, для которого устанавливается новое действие.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает имя установленного обработчика события.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
