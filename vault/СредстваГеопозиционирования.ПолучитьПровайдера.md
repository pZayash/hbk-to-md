СредстваГеопозиционирования (LocationTools)

ПолучитьПровайдера (GetProvider)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьПровайдера(<Имя>, <ТолькоДоступного>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя провайдера геопозиционирования.

<ТолькоДоступного> (необязательный)

Тип: Булево.   
Истина - информация о требуемом провайдере будет возвращена только если провайдер присутствует в системе и является доступным.  
Ложь - информация о требуемом провайдере будет возвращена только если провайдер присутствует в системе.  
Значение по умолчанию: Истина.

Возвращаемое значение:

Тип: [ИнформацияПровайдераГеопозиционирования](ИнформацияПровайдераГеопозиционирования.md), Неопределено.   
Неопределено - провайдер не найден.

Описание:

Получает информацию о провайдере геопозиционирования с заданным именем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
