# Расширение поля формы для поля диаграммы.ПолучитьВыделенныеЭлементы

Расширение поля формы для поля диаграммы (Form field extension for a chart field)

ПолучитьВыделенныеЭлементы (GetSelectedItems)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьВыделенныеЭлементы()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Элементами массива являются значения следующих типов: [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md), [ЗначениеДиаграммы](ЗначениеДиаграммы_(ChartValue).md).

Описание:

Возвращает массив выделенных элементов диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
