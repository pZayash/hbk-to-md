КнопкаПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButton)

ТекстСообщения (MessageText)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит текст сообщения, которое будет отправлено при нажатии на кнопку, если действие кнопки [ОтправитьСообщение](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщение_(CollaborationSystemMessageButtonPanelButtonAction.SendMessage).md) или [ОтправитьСообщениеСДанными](ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия.ОтправитьСообщениеСДанными_(CollaborationSystemMessageButtonPanelButtonAction.SendMessageWithData).md).  
Если свойство не заполнено, то текст сообщения будет содержать значение из свойства [Текст](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Текст_(CollaborationSystemMessageButtonPanelButton.Text).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
