Расширение поля формы для поля планировщика (Form field extension for a planner)

НачалоПеретаскивания (DragStart)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачалоПеретаскивания(<ПараметрыПеретаскивания>, <Выполнение>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания_(DragParameters).md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<Выполнение>

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет выполнением перетаскивания. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), перетаскивание не начнется.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает в тот момент, когда во время перетаскивания курсор находится вне поля планировщика.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
