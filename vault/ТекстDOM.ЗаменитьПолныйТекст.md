ТекстDOM (DOMText)

ЗаменитьПолныйТекст (ReplaceWholeText)

Доступен, начиная с версии 8.1.

Синтаксис:

ЗаменитьПолныйТекст(<Данные>)

Параметры:

<Данные> (обязательный)

Тип: Строка.   
Новый текст.

Возвращаемое значение:

Тип: [ТекстDOM](ТекстDOM.md), Неопределено.   
Неопределено, если замещающий текст пустой.  
Созданный узел:

- данный узел, исключая случай, когда данный узел в режиме Только для чтения;
- новый текстовый узел того же типа, что и данный.

Описание:

Производит замену текста у данного узла и логически смежных текстовых узлов. При этом удаляются все логически смежные текстовые узлы, включая данный. Создается новый текстовый узел с типом, соответствующим данному, ему устанавливается новый текст, и вставляется в дерево документа на место данного.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- при замене могут удаляться узлы [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md), находящиеся в режиме только для чтения, но только в том случае, если они не содержат узлы отличные от [СсылкаНаСущность](ТипУзлаDOM.СсылкаНаСущность.md), [Текст](ТипУзлаDOM.Текст.md), [СекцияCDATA](ТипУзлаDOM.СекцияCDATA.md).

Использование в версии:

Доступен, начиная с версии 8.1.
