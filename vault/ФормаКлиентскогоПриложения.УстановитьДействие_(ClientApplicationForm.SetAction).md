ФормаКлиентскогоПриложения (ClientApplicationForm)

УстановитьДействие (SetAction)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьДействие(<ИмяСобытия>, <Действие>)

Параметры:

<ИмяСобытия> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя события.

<Действие> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя процедуры обработчика.

Описание:

Устанавливает обработчик события.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
