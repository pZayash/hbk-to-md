Соединение с агентом сервера (IServerAgentConnection)

GetSettingActive (GetSettingActive)

Доступен, начиная с версии 8.3.26.

Синтаксис:

GetSettingActive(<Кластер>, <Сервер>, <Настройка>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, для которого требуется проверить активность настройки.

<Сервер> (обязательный)

Тип: [Рабочий сервер](Рабочий_сервер.md).   
Сервер, для которого требуется проверить активность настройки.

<Настройка> (обязательный)

Тип: [Настройка сервиса](Настройка_сервиса.md).   
Настройка сервиса кластера.

Возвращаемое значение:

Тип: Булево.   
Истина - если настройка активна,  
Ложь - в противном случае.

Описание:

Получает значение активности настройки.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
