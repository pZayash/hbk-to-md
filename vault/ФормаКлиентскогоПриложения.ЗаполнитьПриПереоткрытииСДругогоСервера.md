ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗаполнитьПриПереоткрытииСДругогоСервера (FillOnReopenFromOtherServer)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ЗаполнитьПриПереоткрытииСДругогоСервера(<Источник>, <ПараметрыЗаполнения>)

Параметры:

<Источник> (обязательный)

Тип: [ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md).   
Исходная форма.

<ПараметрыЗаполнения> (необязательный)

Тип: [ПараметрыЗаполненияПриПереоткрытииФормы](ПараметрыЗаполненияПриПереоткрытииФормы.md).   
Параметры заполнения при переоткрытии формы.

Описание:

Позволяет заполнить параметры, используемые при переоткрытии формы.  
Имеет смысл только при работе мобильного клиента с возможностью автономной работы. При вызове в другом режиме работы будет сгененировано исключение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
