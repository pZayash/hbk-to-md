# СправочникМенеджер.<Имя справочника>.ПолучитьСсылку

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникМенеджер.<Имя справочника>](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md) › [Методы](СправочникМенеджер.Имя_справочника__Методы.md)

СправочникМенеджер.<Имя справочника> (CatalogManager.<Catalog name>)

ПолучитьСсылку (GetRef)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьСсылку(<УникальныйИдентификатор>)

Параметры:

<УникальныйИдентификатор> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор, из которого будет формироваться ссылка.

Возвращаемое значение:

Тип: [СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md).   

Описание:

Формирует ссылку из значения типа [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).  
Данный уникальный идентификатор может быть в дальнейшем получен из ссылки методом [УникальныйИдентификатор](СправочникСсылка.Имя_справочника.УникальныйИдентификатор_(CatalogRef.Catalog_name.UUID).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если параметр не указан, то будет сформирована новая уникальная ссылка.

См. также:

[СправочникСсылка.\<Имя справочника\>](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md), метод [УникальныйИдентификатор](СправочникСсылка.Имя_справочника.УникальныйИдентификатор_(CatalogRef.Catalog_name.UUID).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
