РасписаниеРегламентногоЗадания (JobSchedule)

ПаузаПовтора (RepeatPause)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Минимальный интервал времени (в секундах) между повторными запусками задания. Интервал считается от времени завершения предыдущего запуска до времени начала последующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
