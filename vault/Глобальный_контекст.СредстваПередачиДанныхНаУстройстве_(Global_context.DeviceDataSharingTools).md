Глобальный контекст (Global context)

СредстваПередачиДанныхНаУстройстве (DeviceDataSharingTools)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Свойство предоставляет доступ к средствам передачи данных в другие приложения из мобильного приложения платформы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.25.
