ОбъявлениеЭлементаXS (XSElementDeclaration)

ИмяТипа (TypeName)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [РасширенноеИмяXML](РасширенноеИмяXML_(XMLExpandedName).md).   
Содержит имя типа данного свойства.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Представляет значение атрибута type объявления атрибута (<http://www.w3.org/TR/xmlschema-1/#element-attribute>) или элемента (<http://www.w3.org/TR/xmlschema-1/#element-element>).

Использование в версии:

Доступен, начиная с версии 8.1.
