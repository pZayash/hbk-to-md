ВычисляемоеПолеСхемыКомпоновкиДанных (DataCompositionSchemaCalculatedField)

ВыражениеПредставления (PresentationExpression)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит выражение, результат которого будет выводиться в результат компоновки для данного поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если свойство не заполнено, будет выдаваться значение свойства [Выражение](ВычисляемоеПолеСхемыКомпоновкиДанных.Выражение_(DataCompositionSchemaCalculatedField.Expression).md).

Использование в версии:

Доступен, начиная с версии 8.1.
