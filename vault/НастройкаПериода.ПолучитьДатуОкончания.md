НастройкаПериода (PeriodSettings)

ПолучитьДатуОкончания (GetDateTo)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьДатуОкончания()

Возвращаемое значение:

Тип: Дата.   

Описание:

Получает дату конца периода.

Доступность:

Толстый клиент.

Примечание:

Если граница интервала задана как [БезОграничения](ВариантГраницыИнтервала.БезОграничения.md), то будет возвращена пустая дата.

Пример:

|  |
| --- |
| НастройкаПериода = ЭлементыФормы.Список.СтандартныйПериод; ДатаКон = НастройкаПериода.ПолучитьДатуОкончания(); |

Использование в версии:

Доступен, начиная с версии 8.0.
