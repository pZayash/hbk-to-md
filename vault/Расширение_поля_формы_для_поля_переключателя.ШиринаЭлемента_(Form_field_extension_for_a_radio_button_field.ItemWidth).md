# Расширение поля формы для поля переключателя.ШиринаЭлемента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля переключателя](Расширение_поля_формы_для_поля_переключателя_(Form_field_extension_for_a_radio_button_field).md) › [Свойства](Расширение_поля_формы_для_поля_переключателя__Свойства.md)

Расширение поля формы для поля переключателя (Form field extension for a radio button field)

ШиринаЭлемента (ItemWidth)

Доступен, начиная с версии 8.3.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет ширину элемента. 0 - определять автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Примечание:

Используется при виде переключателя [Тумблер](ВидПереключателя.Тумблер_(RadioButtonType.Tumbler).md).  
Не используется, если итоговое значение свойства [ОдинаковаяШиринаКолонок](Расширение_поля_формы_для_поля_переключателя.ОдинаковаяШиринаКолонок_(Form_field_extension_for_a_radio_button_field.EqualColumnsWidth).md) = [Ложь](lang__def_BooleanFalse.md).

Использование в версии:

Доступен, начиная с версии 8.3.2.
