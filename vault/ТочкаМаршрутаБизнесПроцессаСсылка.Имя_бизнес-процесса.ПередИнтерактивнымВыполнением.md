ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередИнтерактивнымВыполнением (BeforeExecuteInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередИнтерактивнымВыполнением(<ТочкаМаршрута>, <Задача>, <Отказ>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md), в которой выполняется обработка метода [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно.md) у задачи.

<Задача>

Тип: [ЗадачаСсылка.\<Имя задачи\>](ЗадачаСсылка.Имя_задачи.md).   
Выполняемая задача.

<Отказ>

Тип: Булево.   
Признак отказа от выполнения задачи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то выполнение задачи произведено не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при вызове метода [ВыполнитьЗадачуИнтерактивно](ЗадачаОбъект.Имя_задачи.ВыполнитьЗадачуИнтерактивно.md) у задачи. Этот обработчик является единственным местом в котором можно разместить интерактивное взаимодействие с пользователем перед переходом выполнения бизнес-процесса дальше по маршруту.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие.md).  
Допускаются интерактивные вызовы (открытие форм, предупреждений).  
В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение.md) не вызывается.

Использование в версии:

Доступен, начиная с версии 8.0.
