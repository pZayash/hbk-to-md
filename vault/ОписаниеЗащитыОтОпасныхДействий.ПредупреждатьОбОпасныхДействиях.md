ОписаниеЗащитыОтОпасныхДействий (UnsafeOperationProtectionDescription)

ПредупреждатьОбОпасныхДействиях (UnsafeOperationWarnings)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Если установлено в значение Ложь, будет отключена защита от опасных действий.  
При включенном механизме защиты "1С:Предприятие" выдает пользователю предупреждение о загрузке внешнего исполняемого кода, ранее еще не загружавшегося пользователем. После загрузки внешнего модуля внутри его кода разрешается исполнение всех потенциально опасных команд.   
Рекомендуется устанавливать значение Ложь, если только разработчик конфигурации полностью уверен в источнике происхождения загружаемого кода. При исполнении из кода внешней обработки, отчета или расширения, данный параметр игнорируется.  
Значение по умолчанию - Истина.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.8.
