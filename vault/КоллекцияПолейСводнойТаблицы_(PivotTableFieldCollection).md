# КоллекцияПолейСводнойТаблицы

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Табличный документ](Табличный_документ-2.md) › [Сводная таблица](Сводная_таблица.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля сводной таблицы или коллекция полей сводной таблицы.  
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается имя или индекс элемента (нумерация с 0).
