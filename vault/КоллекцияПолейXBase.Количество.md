КоллекцияПолейXBase (XBaseFieldsCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество полей в таблице базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Сообщить("Всего полей в таблице: " + xB.Поля.Количество()); |

Использование в версии:

Доступен, начиная с версии 8.0.
