Расширение табличного поля списка справочника (Catalog list table box extension)

ПроверкаОтображенияНовойСтроки (NewRowShowCheck)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ВариантПроверкиОтображенияНовойСтроки](ВариантПроверкиОтображенияНовойСтроки.md).   
Если данное свойство имеет значение [СообщатьОНесоответствииОтбору](ВариантПроверкиОтображенияНовойСтроки.СообщатьОНесоответствииОтбору.md), то после окончания ввода новой строки системой будет осуществлена проверка строки и, если строка будет не соответствовать установленному отбору, будет выдано сообщение.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
