**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Глобальный контекст](Глобальный_контекст.md) › [Методы](Методы.md) › [Процедуры и функции работы с XML](Процедуры_и_функции_работы_с_XML.md)

Глобальный контекст (Global context)

XMLТип (XMLType)

Доступен, начиная с версии 8.0.

Синтаксис:

XMLТип(<Тип>)

Параметры:

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип значения, для которого необходимо определить соответствующий XML тип.

Возвращаемое значение:

Тип: [ТипДанныхXML](ТипДанныхXML_(XMLDataType).md).   
[Неопределено](lang__def_Undefined.md), если никакой тип данных XML не соответствует указанному типу.

Описание:

Получает тип данных XML, соответствующий переданному в качестве параметра типу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
