# Глобальный контекст.УстановитьВнешнююКомпонентуАсинх

Глобальный контекст (Global context)

УстановитьВнешнююКомпонентуАсинх (InstallAddInAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьВнешнююКомпонентуАсинх(<Местоположение>)

Параметры:

<Местоположение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, определяющая местоположение внешней компоненты.  
В качестве строки может использоваться:

- полное имя макета, в котором хранится внешний компонент в архиве;
- URL к внешнему компоненту, упакованному в ZIP-архив, в формате, аналогичном [ПолучитьНавигационнуюСсылку](Глобальный_контекст.ПолучитьНавигационнуюСсылку_(Global_context.GetURL).md)().

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Доставляет объект внешнего компонента с сервера на клиент, после чего он становится доступен для метода [ПодключитьВнешнююКомпонентуАсинх](Глобальный_контекст.ПодключитьВнешнююКомпонентуАсинх_(Global_context.AttachAddInAsync).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Метод работает только с компонентами, хранящимися в ZIP-архиве.

Пример:

| --- |
| НаКлиенте Процедура Команда(Команда)     Подключить(); КонецПроцедуры  НаКлиенте  Асинх Процедура Подключить()     Подключено = Ждать ПодключитьВнешнююКомпонентуАсинх(         "ОбщийМакет.ВнешняяКомпонента", "ВнешняяКомпонента");     Если Подключено Тогда         // Внешняя компонента подключена      Иначе         Ждать УстановитьВнешнююКомпонентуАсинх(             "ОбщийМакет.ВнешняяКомпонента");         Подключено = Ждать ПодключитьВнешнююКомпонентуАсинх(             "ОбщийМакет.ВнешняяКомпонента", "ВнешняяКомпонента");         Если Подключено Тогда             // Внешняя компонента установлена и подключена          Иначе             // Не удалось установить или подключить              // внешнюю компоненту          КонецЕсли;     КонецЕсли; КонецПроцедуры |
