ЭлементыДендрограммы (DendrogramItemCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ЭлементДендрограммы](ЭлементДендрограммы_(DendrogramItem).md).   

Описание:

Добавляет элемент в коллекцию элементов дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
