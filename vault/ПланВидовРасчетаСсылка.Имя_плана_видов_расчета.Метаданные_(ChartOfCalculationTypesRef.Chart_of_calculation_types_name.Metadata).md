ПланВидовРасчетаСсылка.<Имя плана видов расчета> (ChartOfCalculationTypesRef.<Chart of calculation types name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланВидовРасчета](ОбъектМетаданных_ПланВидовРасчета_(MetadataObject_ChartOfCalculationTypes).md).   

Описание:

Предоставляет доступ к объекту метаданных плана видов расчета. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.ПланыВидовРасчета.ОсновныеНачисления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
