СлоиГеографическойСхемы (GeographicalSchemaLayers)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md).   
Объект, для которого надо получить индекс.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает индекс объекта в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
