Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле.md), предназначенные для интерактивной работы с записями регистра расчета. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока.md) возвращает значение типа [РегистрРасчетаКлючЗаписи.](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета.md)<[Имя регистра расчета](РегистрРасчетаКлючЗаписи.Имя_регистра_расчета.md)>.
