СписокСтрокDOM (DOMStringList)

Содержит (Contains)

Доступен, начиная с версии 8.1.

Синтаксис:

Содержит(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Проверяемая строка.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - строка входит в коллекцию; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет наличие строки в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
