ИнформацияОбОшибке (ErrorInfo)

Код (Code)

Доступен, начиная с версии 8.3.21.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Предназначено для идентификации исключения и последующего анализа.  
Например, для маркировки ошибок подсистемы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
