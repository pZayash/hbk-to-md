Глобальный контекст (Global context)

СоздатьКаталогАсинх (CreateDirectoryAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

СоздатьКаталогАсинх(<ИмяКаталога>)

Параметры:

<ИмяКаталога> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя создаваемого каталога.  
Допускается использование схемы ftp для адресации файлов. При использовании этой схемы в адресах необходимо указывать прямые слеши '/', а не обратные '\'.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - строка, содержащая путь к созданному каталогу.

Описание:

Асинхронная функция. Создает новый каталог файлов.  
Допускается использование схемы FTP для адресации файлов. При использовании этой схемы в адресах необходимо указывать прямые косые черты '/', а не обратные '\'.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.  
Существование каталога <ИмяКаталога> не приводит к ошибке.  
Для работы метода на веб-клиенте необходимо предварительно подключить расширение работы с файлами.
