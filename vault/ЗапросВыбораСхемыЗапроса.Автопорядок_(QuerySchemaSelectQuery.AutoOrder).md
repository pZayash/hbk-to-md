ЗапросВыбораСхемыЗапроса (QuerySchemaSelectQuery)

Автопорядок (AutoOrder)

Доступен, начиная с версии 8.3.5.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, используется ли автоматическое упорядочивание результата запроса. Соответствует ключевому слову АВТОУПОРЯДОЧИВАНИЕ языка запросов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
