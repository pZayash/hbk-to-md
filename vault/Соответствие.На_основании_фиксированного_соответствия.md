Соответствие (Map)

На основании фиксированного соответствия

Доступен, начиная с версии 8.3.2.

Синтаксис:

Новый Соответствие(<ФиксированноеСоответствие>)

Параметры:

<ФиксированноеСоответствие> (необязательный)

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие.md).   
Фиксированное соответствие.

Описание:

Создаваемое соответствие заполняется элементами из исходного фиксированного соответствия.

Использование в версии:

Доступен, начиная с версии 8.3.2.
