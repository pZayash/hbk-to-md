Расширение поля формы для поля PDF-документа (Form extension for a PDF document field)

НажатиеНаНавигационнойСсылке (URLClick)

Доступен, начиная с версии 8.3.21.

Синтаксис:

НажатиеНаНавигационнойСсылке(<НавигационнаяСсылка>, <СтандартнаяОбработка>)

Параметры:

<НавигационнаяСсылка>

Тип: Строка.   
Текст гиперссылки.

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: Истина.

Описание:

Возникает при нажатии левой кнопкой мыши на навигационной ссылке в поле PDF-документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.21.
