МенеджерКриптографии (CryptoManager)

ОпределениеАлгоритмаХеширования (HashAlgorithmDefinition)

Доступен, начиная с версии 8.3.24.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие.md).   
Содержит список связанных элементов: OID-значение открытого ключа и OID-значение алгоритма функции хеширования.   
В качестве значения элемента связанного списка используется массив OID-значений алгоритма функции хеширования в порядке приоритета их использования и с учетом поддержки соответствующим криптопровайдером.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Свойство используется, если [АлгоритмХеширования](МенеджерКриптографии.АлгоритмХеширования.md) не установлен.

Использование в версии:

Доступен, начиная с версии 8.3.24.
