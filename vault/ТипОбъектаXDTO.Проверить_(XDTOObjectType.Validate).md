ТипОбъектаXDTO (XDTOObjectType)

Проверить (Validate)

Доступен, начиная с версии 8.1.

Синтаксис:

Проверить(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Проверяемый объект.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Проверяет соответствие объекта модели XDTO.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
