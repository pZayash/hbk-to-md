ВнешнийИсточникДанныхТаблицаМенеджерЗаписи.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordManager.<External source name>.<External data source table name>)

Заполнить (Fill)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (необязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение.

Описание:

Вызывает заполнение элемента данными из какого-либо другого объекта. При этом в модуле объекта вызывается процедура ОбработкаЗаполнения. В ней может размещаться алгоритм, заполняющий данные элемента из переданного значения.  
Для заполнения может использоваться значение любого типа. Для этого типа в процедуре ОбработкаЗаполнения должен быть описан алгоритм заполнения.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
