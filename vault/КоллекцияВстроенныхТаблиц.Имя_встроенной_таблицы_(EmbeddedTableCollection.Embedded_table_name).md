КоллекцияВстроенныхТаблиц (EmbeddedTableCollection)

<Имя встроенной таблицы> (<Embedded table name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [СводнаяТаблица](СводнаяТаблица_(PivotTable).md).   
Набор свойств содержит встроенные таблицы. Доступ к встроенной таблице осуществляется по имени таблицы. Имена свойств совпадают с именами встроенных таблиц, как они заданы в табличном документе.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
