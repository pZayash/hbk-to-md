Глобальный контекст (Global context)

ПерейтиПоНавигационнойСсылке (GotoURL)

Доступен, начиная с версии 8.2.

Синтаксис:

ПерейтиПоНавигационнойСсылке(<НавигационнаяСсылка>)

Параметры:

<НавигационнаяСсылка> (обязательный)

Тип: [Строка](lang__def_String.md).   
Навигационная ссылка.

Описание:

Выполняет переход по ссылке так, как если бы пользователь выполнил интерактивную вспомогательную команду "Перейти по ссылке ...".  
Если выполняется переход по ссылке на объект информационной базы, отчет или обработку, то происходит открытие нового вспомогательного окна (если окно с этим объектом еще не открыто).  
Если выполняется переход по ссылке на точку навигации, то переход будет выполнен в основном окне приложения, и оно станет активным.  
Если выполняется переход по ссылке на обсуждение системы взаимодействия:

- для неконтекстных обсуждений активизируется основная форма системы взаимодействия и соответствующее обсуждение становится текущим,
- для контекстных обсуждений открывается форма объекта контекста.

Возможен переход по внешней ссылке. Для внешней ссылки, которая содержит сигнатуру "e1cib" выполняются следующие действия:

- Если начало ссылки совпадает с навигационной ссылкой информационной базы текущего сеанса, то выполняется переход по локальной ссылке;
- Если начало ссылки не совпадает с навигационной ссылкой информационной базы, то осуществляется поиск запущенного клиента, подключенного к базе, адрес которой соответствует ссылке. Если такой клиент найден, окно его приложения активизируется и в нем выполняется переход по ссылке.
- Если соответствующего запущенного экземпляра тонкого клиента нет, то выполняется попытка запуска нового экземляра тонкого клиента с соответствующей строкой соединения и в нем выполняется переход.

Если выполняется переход по ссылке вида "http(s)" на объект в другой информационной базе, то:

- Выполняется попытка найти запущенный экземпляр тонкого клиента, работающего с соответствующей информационной базой.
- Если запущенного экземпляра тонкого клиента нет, или в нем открыт модальный диалог, выполняется попытка запуска нового экземпляра тонкого клиента с соответствующей строкой соединения.
- Если попытка запуска не успешна (например, тонкий клиент не установлен), переданная ссылка обрабатывается операционной системой.

Если внешняя ссылка не является навигационной ссылкой информационной базы (например, ссылки на сайты или файлы на диске), то в тонком клиенте данная ссылка выполняется как команда операционной системы. Ссылка открывается ассоциированной с данным типом ссылки программой. Если ссылка указывает на текстовый документ, табличный документ и т.п., в тонком клиенте документ будет открыт в клиентском приложении.В веб-клиенте отрабатывают только http-ссылки, которые открываются в том же браузере в отдельном окне.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).  
Вызов метода выполняет обращение к серверу.

Примечание:

Может быть вызван только в управляемом режиме.  
В веб-клиенте внешние ссылки всегда открываются в новом окне.  
Вызов метода для ссылок типа Данные (конкретный элемент справочника или документ) всегда приводит к вызову сервера.  
Вызов метода для ссылок типа НавигационнаяСсылка (например, открыть раздел) кэшируется. Зависит от версии платформы, версии конфигурации, функциональных опций.  
Вызов метода для перехода по ссылке других типов (например, справка, файловая система) не приводит к вызову сервера.  
При переходе по навигационной ссылке не учитываются пробельные символы в начале и в конце строки:

- В тонком и толстом клиентах - символы, относящиеся к пробельным с точки зрения ICU, и символ параграфа, который вставляет поле ввода вместо символов возврат каретки/перевод строки.
- В веб-клиенте - символы, относящиеся к пробельным с точки зрения реализации регулярных выражений в браузере.

Ссылка, переданная в данный метод, передается на выполнение операционной системе в исходном виде.  
В режиме совместимости конфигурации [Версия8\_3\_6](РежимСовместимости.Версия8_3_6_(CompatibilityMode.Version8_3_6).md) или [Версия8\_3\_5](РежимСовместимости.Версия8_3_5_(CompatibilityMode.Version8_3_5).md), ссылка, переданная в метод, перед передачей на выполнение операционной системе, декодируется.  
В режиме совместимости конфигурации [Версия8\_3\_4](РежимСовместимости.Версия8_3_4_(CompatibilityMode.Version8_3_4).md) и ниже ссылка, переданная в метод, передается на выполнение операционной системе в исходном виде.

Использование в версии:

Доступен, начиная с версии 8.2.
