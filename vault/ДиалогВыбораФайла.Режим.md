ДиалогВыбораФайла (FileDialog)

Режим (Mode)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [РежимДиалогаВыбораФайла](РежимДиалогаВыбораФайла.md).   
Содержит режим диалога для выбора файла.  
Для записи следует использовать метод [ПолучитьФайл](Глобальный_контекст.ПолучитьФайл.md) с установленным параметром <Интерактивно>.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

В веб-клиенте, если расширение работы с файлами не установлено, то доступен только режим [Открытие](РежимДиалогаВыбораФайла.Открытие.md).

Использование в версии:

Доступен, начиная с версии 8.0.
