Информационная база (IInfoBaseInfo)

ConfigurationUnloadDelayByWorkingProcessWithoutActiveUsers (ConfigurationUnloadDelayByWorkingProcessWithoutActiveUsers)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет время в секундах после разрыва последнего соединения с информационной базой в рабочем процессе, через которое рабочий процесс освобождает данные информационной базы. Пока данные не освобождены, установка новых соединений, и запуск фоновых заданий выполняются существенно быстрее. Данные могут быть освобождены раньше, если завершается рабочий процесс или запрошен монопольный доступ к информационной базе или базе данных.  
Значение по умолчанию: 0.  
Значение должно находиться в диапазоне от 0 до 4294967.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
