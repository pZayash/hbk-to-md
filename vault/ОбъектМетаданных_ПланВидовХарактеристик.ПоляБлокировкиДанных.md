ОбъектМетаданных: ПланВидовХарактеристик (MetadataObject: ChartOfCharacteristicTypes)

ПоляБлокировкиДанных (DataLockFields)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [СписокПолей](СписокПолей.md).   
Список полей, по которым возможна блокировка плана видов характеристик.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не рекомендуется использовать поля следующих видов:

- Строка неограниченной длины,
- Поле ТипЗначения плана видов характеристик,
- [ХранилищеЗначения](ХранилищеЗначения.md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
