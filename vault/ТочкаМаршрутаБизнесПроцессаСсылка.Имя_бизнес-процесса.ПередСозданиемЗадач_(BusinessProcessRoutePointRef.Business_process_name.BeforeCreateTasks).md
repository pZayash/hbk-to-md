# ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса>.ПередСозданиемЗадач

ТочкаМаршрутаБизнесПроцессаСсылка.<Имя бизнес-процесса> (BusinessProcessRoutePointRef.<Business process name>)

ПередСозданиемЗадач (BeforeCreateTasks)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередСозданиемЗадач(<ТочкаМаршрута>, <ФормируемыеЗадачи>, <СтандартнаяОбработка>)

Параметры:

<ТочкаМаршрута>

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса_(BusinessProcessRoutePointRef.Business_process_name).md).   
Точка маршрута бизнес-процесса, на которой происходит создание задач.

<ФормируемыеЗадачи>

Тип: [Массив](Массив_(Array).md).   
Массив формируемых задач (объектов типа [ЗадачаОбъект](БиблиотекаКартинок.ЗадачаОбъект_(PictureLib.TaskObject).md)).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартного формирования списка задач производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает перед выполнением стандартной механики генерации задач. Если обработчик сам формирует список задач, то должен установить <СтандартнаяОбработка> в [Ложь](lang__def_BooleanFalse.md). В противном случае созданные им в этом обработчике задачи удаляются и заново формируются стандартной процедурой создания задач. Вне зависимости от <СтандартнаяОбработка> после этого обработчика обязательно будет вызван обработчик [ПриСозданииЗадач](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.ПриСозданииЗадач_(BusinessProcessRoutePointRef.Business_process_name.OnCreateTask).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Только для точек вида [Действие](ВидТочкиМаршрутаБизнесПроцесса.Действие_(BusinessProcessRoutePointType.Activity).md) и [ВложенныйБизнесПроцесс](ВидТочкиМаршрутаБизнесПроцесса.ВложенныйБизнесПроцесс_(BusinessProcessRoutePointType.SubBusinessProcess).md).  
Не допускаются интерактивные вызовы (открытие форм, предупреждений). Допускается использование метода [Сообщить](Глобальный_контекст.Сообщить_(Global_context.Message).md).

Использование в версии:

Доступен, начиная с версии 8.0.
