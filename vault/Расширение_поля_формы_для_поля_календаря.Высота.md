Расширение поля формы для поля календаря (Form field extension for a calendar field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит высоту поля календаря. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется, если значение свойства [ВысотаВМесяцах](Расширение_поля_формы_для_поля_календаря.ВысотаВМесяцах.md)=0 или для режимов совместимости до [Версия8\_3\_2](РежимСовместимости.Версия8_3_2.md).

Использование в версии:

Доступен, начиная с версии 8.2.
