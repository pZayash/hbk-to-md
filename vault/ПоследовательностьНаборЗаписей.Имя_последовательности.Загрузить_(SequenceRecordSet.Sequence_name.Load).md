# ПоследовательностьНаборЗаписей.<Имя последовательности>.Загрузить

ПоследовательностьНаборЗаписей.<Имя последовательности> (SequenceRecordSet.<Sequence name>)

Загрузить (Load)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Загрузить(<ТаблицаЗначений>)

Параметры:

<ТаблицаЗначений> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений_(ValueTable).md).   
Таблица значений, содержащая данные для заполнения набора записей.

Описание:

Загружает набор записей значениями из переданной таблицы значений. При этом все прежние записи набора удаляются.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
