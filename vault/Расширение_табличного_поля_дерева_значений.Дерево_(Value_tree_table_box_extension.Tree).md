# Расширение табличного поля дерева значений.Дерево

Расширение табличного поля дерева значений (Value tree table box extension)

Дерево (Tree)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет тип отображения дерева значений. Если указано [Истина](lang__def_BooleanTrue.md) - в виде дерева, если [Ложь](lang__def_BooleanFalse.md) - в виде иерархического списка.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
