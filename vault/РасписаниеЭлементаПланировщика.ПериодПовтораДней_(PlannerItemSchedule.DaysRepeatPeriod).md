# РасписаниеЭлементаПланировщика.ПериодПовтораДней

РасписаниеЭлементаПланировщика (PlannerItemSchedule)

ПериодПовтораДней (DaysRepeatPeriod)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Период времени (в днях), с которым повторяется интервал. Если &ПериодПовтораДней=1, интервал будет повторен каждый день, &ПериодПовтораДней=2, интервал будет повторен через день, и т. д.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
