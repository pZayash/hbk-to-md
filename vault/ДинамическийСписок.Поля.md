ДинамическийСписок (DynamicList)

Поля (Fields)

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).

Использование:

Только чтение.

Описание:

Тип: [ПоляНабораДанныхСхемыКомпоновкиДанных](ПоляНабораДанныхСхемыКомпоновкиДанных.md).   
Содержит описания полей (набора данных) динамического списка.  
  
Свойства [Роль](ПолеНабораДанныхСхемыКомпоновкиДанных.Роль.md), [ВыражениеПредставления](ПолеНабораДанныхСхемыКомпоновкиДанных.ВыражениеПредставления.md), [НаборДанныхПроверкиИерархии](ПолеНабораДанныхСхемыКомпоновкиДанных.НаборДанныхПроверкиИерархии.md) и [ПараметрНабораДанныхПроверкиИерархии](ПолеНабораДанныхСхемыКомпоновкиДанных.ПараметрНабораДанныхПроверкиИерархии.md) не поддерживаются в динамическом списке.  
  
Кроме того, при установленной основной таблице:

- для многих стандартных полей (например, Ссылка, Наименование, ПометкаУдаления) не поддерживается установка значение свойства [Поле](ОграничениеИспользованияПоляСхемыКомпоновкиДанных.Поле.md) в Истина,
- для ключевых полей, стандартных полей сортировки и поля Родитель дополнительно не поддерживается установка значений свойств [Условие](ОграничениеИспользованияПоляСхемыКомпоновкиДанных.Условие.md) и [Порядок](ОграничениеИспользованияПоляСхемыКомпоновкиДанных.Порядок.md) в Истина.

Доступность:

Сервер, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19 (в режиме совместимости с версией 8.3.19 и последующими).
