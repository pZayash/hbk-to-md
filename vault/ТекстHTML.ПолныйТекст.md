ТекстHTML (HTMLText)

ПолныйТекст (WholeText)

Не рекомендуется использовать, начиная с версии 8.2.

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит текст логически смежных текстовых узлов.  
Текст формируется из текстовых узлов, смежных с данным, в порядке следования в документе.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.2.
