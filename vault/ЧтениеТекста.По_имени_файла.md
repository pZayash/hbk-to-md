ЧтениеТекста (TextReader)

По имени файла

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ЧтениеТекста(<ИмяФайла>, <Кодировка>, <РазделительСтрок>, <КонвертируемыйРазделительСтрок>, <МонопольныйРежим>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя текстового файла для чтения.

<Кодировка> (необязательный)

Тип: [КодировкаТекста](КодировкаТекста.md), Строка.   
Определяет кодировку текста читаемого текстового файла (согласован с текстовым документом). Если параметр не задан, формат текста будет определен автоматически по сигнатуре BOM в начале файла; если сигнатура BOM в файле отсутствует, файл будет открыт в кодировке [ANSI](КодировкаТекста.ANSI.md).  
  
Кодировка может быть задана как в виде значения перечисления [КодировкаТекста](КодировкаТекста.md), так и в виде строки с указанием названия кодировки.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Определяет строку, разделяющую строки текстового файла.  
Значение по умолчанию: [ПС](Символы.ПС.md).

<КонвертируемыйРазделительСтрок> (необязательный)

Тип: Строка.   
Определяет разделение строк в файле для конвертации в стандартный перевод строк [ПС](Символы.ПС.md).  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<МонопольныйРежим> (необязательный)

Тип: Булево.   
Определяет режим открытия файла. Истина - монопольный, Ложь - разделяемый.  
Значение по умолчанию: Истина.

Описание:

Открывает текстовый файл для чтения. Аналогичен методу [Открыть](ЧтениеТекста.Открыть.md).

Пример:

|  |
| --- |
| ДлинныйТекст = Новый ЧтениеТекста("c:\temp\file.txt", КодировкаТекста.UTF16); |

См. также:

[ЧтениеТекста](ЧтениеТекста.md), метод [Открыть](ЧтениеТекста.Открыть.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
