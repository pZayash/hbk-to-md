# Расширение поля формы для поля календаря.ПеремещениеПоКалендарю

Расширение поля формы для поля календаря (Form field extension for a calendar field)

ПеремещениеПоКалендарю (CalendarNavigation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак разрешения перемещения по датам календаря вне видимых в текущий момент дат.   
[Истина](lang__def_BooleanTrue.md) - перемещение разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
