ФормаКлиентскогоПриложения (ClientApplicationForm)

ПолучитьОбсуждениеАсинх (GetConversationAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По идентификатору

Синтаксис:

ПолучитьОбсуждениеАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения.

Описание варианта метода:

Возвращает обсуждение по идентификатору.

Вариант синтаксиса: По ключу

Синтаксис:

ПолучитьОбсуждениеАсинх(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: Строка.   
Ключ обсуждения.  
Значение по умолчанию: "".

Описание варианта метода:

Возвращает неконтекстное обсуждение по заданному ключу. Если обсуждение не найдено, возвращает Неопределено.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - объект типа [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md) или Неопределено, если у формы нет навигационной ссылки (например, форма нового незаписанного объекта).

Описание:

Асинхронная функция. Выполняет получение контекстного обсуждения формы.  
Если обсуждение с указанным идентификатором не найдено, оно будет создано.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[ФормаКлиентскогоПриложения](ФормаКлиентскогоПриложения.md), метод [НачатьПолучениеОбсуждения](ФормаКлиентскогоПриложения.НачатьПолучениеОбсуждения.md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
