ИнтернетПрокси (InternetProxy)

ИспользоватьАутентификациюОС (UseOSAuthentication)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ИспользоватьАутентификациюОС(<Протокол>)

Параметры:

<Протокол> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя протокола, для которого запрашивается признак использования аутентификации средствами операционной системы прокси-сервера. Если в параметре указана пустая строка, возвращается порт сервера, установленного по умолчанию.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

Описание:

Возвращает признак использования аутентификации средствами операционной системы прокси для указанного протокола.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.7.
