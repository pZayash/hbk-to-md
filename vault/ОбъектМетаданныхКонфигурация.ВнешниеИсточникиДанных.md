ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ВнешниеИсточникиДанных (ExternalDataSources)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных.md).   
Содержит объекты метаданных для внешних источников данных.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ВнешнийИсточникДанных](ОбъектМетаданных_ВнешнийИсточникДанных.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
