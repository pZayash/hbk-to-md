ЧтениеУзловDOM (DOMNodeReader)

ЭтоСимвольныеДанные (IsCharacters)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Показывает, имеет ли текущий узел текстовое значение. Текстовое значение имеется у узлов: [Текст](ТипУзлаXML.Текст.md), [СекцияCDATA](ТипУзлаXML.СекцияCDATA.md), [ПробельныеСимволы](ЧтениеУзловDOM.ПробельныеСимволы.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
