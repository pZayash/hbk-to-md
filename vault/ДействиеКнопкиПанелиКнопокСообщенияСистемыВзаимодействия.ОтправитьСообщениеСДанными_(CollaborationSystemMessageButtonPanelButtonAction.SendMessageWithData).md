ДействиеКнопкиПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonAction)

ОтправитьСообщениеСДанными (SendMessageWithData)

Доступен, начиная с версии 8.3.25.

Описание:

Действие кнопки создает сообщение с текстом кнопки.  
В свойство [Данные](СообщениеСистемыВзаимодействия.Данные_(CollaborationSystemMessage.Data).md) создаваемого сообщения копируется значение свойства [Данные](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Данные_(CollaborationSystemMessageButtonPanelButton.Data).md) нажатой кнопки.   
Действие не доступно в интеграциях с мессенджерами.

Использование в версии:

Доступен, начиная с версии 8.3.25.
