Доступен, начиная с версии 8.0.

Элементы коллекции:

[Поле](Поле_(Field).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.  

Методы:

[Индекс (IndexOf)](СписокПолей.Индекс_(FieldList.IndexOf).md)  
[Количество (Count)](СписокПолей.Количество_(FieldList.Count).md)  
[Найти (Find)](СписокПолей.Найти_(FieldList.Find).md)  
[Получить (Get)](СписокПолей.Получить_(FieldList.Get).md)  

Описание:

Представляет собой коллекцию полей ввода по строке.  
Используется для свойств ВводПоСтроке объектов метаданных и свойства ПоляКлюча объектов метаданных - таблиц внешних источников данных.  
 Свойство ВводПоСтроке такого типа есть у всех объектов описания метаданного, которые определяют наличие в системе типа ...Ссылка, таких как [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [СправочникСсылка.](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)<[Имя справочника](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)>, [ДокументСсылка.](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)<[Имя документа](ДокументСсылка.Имя_документа_(DocumentRef.Document_name).md)>, [ПланВидовХарактеристикСсылка.](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)<[Имя плана видов характеристик](ПланВидовХарактеристикСсылка.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesRef.Chart_of_characteristic_types_name).md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)>, [ПланВидовРасчетаСсылка.](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаСсылка.Имя_плана_видов_расчета_(ChartOfCalculationTypesRef.Chart_of_calculation_types_name).md)>, [БизнесПроцессСсылка.](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)<[Имя бизнес-процесса](БизнесПроцессСсылка.Имя_бизнес-процесса_(BusinessProcessRef.Business_process_name).md)>, [ЗадачаСсылка.](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)<[Имя задачи](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md)>.  
В качестве полей ввода по строке могут выступать такие предопределенные реквизиты, как "Код", "Наименование", "Дата", "Номер" и реквизиты объектов, имеющие примитивный тип данных, для которых свойство объекта метаданных "Индексировать" принимает значение "Индексирование" или "Индексировать с доп. упорядочиванием".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
