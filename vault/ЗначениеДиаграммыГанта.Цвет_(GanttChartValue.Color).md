ЗначениеДиаграммыГанта (GanttChartValue)

Цвет (Color)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет интервалов. Допустимо значение [АвтоЦвет](ВидЦвета.АвтоЦвет_(ColorType.AutoColor).md). В этом случае цвет определяется в зависимости от значения свойства ПриоритетЦвета серии или точки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
