ОбъектМетаданных: РегламентноеЗадание (MetadataObject: ScheduledJob)

Расписание (Schedule)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Неопределено](lang__def_Undefined.md).   

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Данное свойство всегда имеет значение [Неопределено](lang__def_Undefined.md), т.к. во встроенном языке не предусмотрена работа со значением данного свойства.  
Для получения расписания можно использовать свойство [Расписание](РегламентноеЗадание.Расписание_(ScheduledJob.Schedule).md):  

|  |
| --- |
| Задание = РегламентныеЗадания.НайтиПредопределенное(Объект\_Метаданных); РасписаниеЗадания = Задание.Расписание; |

Использование в версии:

Доступен, начиная с версии 8.1.
