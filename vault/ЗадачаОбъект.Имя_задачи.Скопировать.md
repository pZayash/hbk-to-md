ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ЗадачаОбъект.\<Имя задачи\>](ЗадачаОбъект.Имя_задачи.md).   

Описание:

Создает новую задачу копированием существующей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.0.
