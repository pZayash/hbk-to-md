ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

ПараметрыФункциональныхОпций (FunctionalOptionsParameters)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все параметры функциональных опций, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: ПараметрФункциональныхОпций](ОбъектМетаданных_ПараметрФункциональныхОпций_(MetadataObject_FunctionalOptionsParameter).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
