КоллекцияВыделенныхДат (SelectedDateCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Дата>)

Параметры:

<Дата> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Дата, добавляемая к коллекции выделенных дат.

Описание:

Добавляет дату к коллекции выделенных дат календаря.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
