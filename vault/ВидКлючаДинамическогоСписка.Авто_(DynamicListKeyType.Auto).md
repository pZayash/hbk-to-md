ВидКлючаДинамическогоСписка (DynamicListKeyType)

Авто (Auto)

Доступен, начиная с версии 8.3.14.

Описание:

Вид ключа определяется по следующему правилу:

- Если у динамического списка указана основная таблица - фактически используемый вид ключа определяется основной таблицей.
- Если основная таблица не указана - вид ключа определяется в зависимости от количества указанных ключевых полей:
  - ни одного поля - <НомерСтроки>,
  - одно поле - <ЗначениеПоля>,
  - более одного поля - <КлючЗаписи>.

Использование в версии:

Доступен, начиная с версии 8.3.14.
