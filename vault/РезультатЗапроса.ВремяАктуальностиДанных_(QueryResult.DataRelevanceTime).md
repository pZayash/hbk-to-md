РезультатЗапроса (QueryResult)

ВремяАктуальностиДанных (DataRelevanceTime)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Фактическое время актуальности данных. Для актуальных данных содержит время начала выполнения запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).
