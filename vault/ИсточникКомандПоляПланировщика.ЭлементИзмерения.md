ИсточникКомандПоляПланировщика (PlannerCommandSource)

ЭлементИзмерения (DimensionItem)

Доступен, начиная с версии 8.3.23.

Описание:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика.md), Неопределено.   
Значение, которое устанавливается при нажатии на измерение поля планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд.md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы.md) содержит Неопределено,
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения.md) заполнено,
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений.md) заполнено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата.md) содержит Неопределено,
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени.md) содержит Неопределено,
- [Команды](html.md) не содержит команд,
- [КомандаПоУмолчанию](html.md) заполнено командой [ВыбратьЭлементИзмерения](СтандартнаяКомандаПланировщика.ВыбратьЭлементИзмерения.md).

Использование в версии:

Доступен, начиная с версии 8.3.23.
