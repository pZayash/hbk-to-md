СвязиДендрограммы (DendrogramLinkCollection)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс точки.

Возвращаемое значение:

Тип: [СвязьДендрограммы](СвязьДендрограммы_(DendrogramLink).md).   

Описание:

Вставляет связь в заданную позицию коллекции связей дендрограммы.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
