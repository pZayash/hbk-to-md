ФормаКлиентскогоПриложения (ClientApplicationForm)

ОповеститьОВыборе (NotifyChoice)

Доступен, начиная с версии 8.2.

Синтаксис:

ОповеститьОВыборе(<ЗначениеВыбора>, <ВыборДобавлением>)

Параметры:

<ЗначениеВыбора> (обязательный)

Тип: Произвольный.   
Результат выбора.

<ВыборДобавлением> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Определяет, как поле ввода обработает выбранные значения при вводе множественных значений:

- [Ложь](lang__def_BooleanFalse.md) - выбранное значение заменяет текущее значение реквизита.
- [Истина](lang__def_BooleanTrue.md) - выбранное значение добавляется как новый элемент списка в значение реквизита.
- [Неопределено](lang__def_Undefined.md) - Автоматическое определение. Используется значение одноименного параметра формы <ВыборДобавлением>.

  
Если поле ввода находится в режиме ввода единственного значения, то параметр игнорируется.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Посылает оповещение владельцу формы о выполнении выбора или подбора, передает ему выбранное значение. Закрывает форму в соответствии со значением свойства [ЗакрыватьПриВыборе](ФормаКлиентскогоПриложения.ЗакрыватьПриВыборе_(ClientApplicationForm.CloseOnChoice).md). Данное действие аналогично выполнению выбора в форме, открытой для выбора в поле ввода или ячейке табличного поля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Может применяться в случае, когда открытие формы выбора или подбора выполнено средствами языка.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.23.
