ЭлементПланировщика (PlannerItem)

ЗамещающиеЭлементы (ReplacementItems)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЗамещающихЭлементовПланировщика](КоллекцияЗамещающихЭлементовПланировщика_(PlannerReplacementItemCollection).md), [Неопределено](lang__def_Undefined.md).   
Коллекция элементов, которые используются для того, чтобы заменить повторенные элементы для случая, когда задано свойство [Расписание](ЭлементПланировщика.Расписание_(PlannerItem.Schedule).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
