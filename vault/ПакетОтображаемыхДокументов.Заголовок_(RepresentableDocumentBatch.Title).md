ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

Заголовок (Title)

Доступен, начиная с версии 8.3.8.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Заголовок пакета отображается в окне очереди печати принтера и в диалоге, отображаемом при печати пакета. Если заголовок не установлен, то отображается надпись "Пакет документов".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.8.
