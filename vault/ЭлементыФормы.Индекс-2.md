ЭлементыФормы (FormItems)

Индекс (IndexOf)

Доступен, начиная с версии 8.2.

Синтаксис:

Индекс(<Элемент>)

Параметры:

<Элемент> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы.md), [ТаблицаФормы](ТаблицаФормы.md), [ПолеФормы](ПолеФормы.md), [КнопкаФормы](КнопкаФормы.md).   
Элемент в коллекции.

Возвращаемое значение:

Тип: Число.   

Описание:

Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
