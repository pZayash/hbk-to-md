ПолеКалендаря (CalendarBox)

КонецПериодаОтображения (EndOfDisplayPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md), [Неопределено](lang__def_Undefined.md).   
Верхняя граница периода видимых в календаре дат. За указанную границу перемещение в календаре невозможно.  
Для снятия ограничения следует указать значение [Неопределено](lang__def_Undefined.md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
