# Рабочий сервер.ConnectionsPerWorkingProcessLimit

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Средства интеграции и администрирования](Средства_интеграции_и_администрирования.md) › [Менеджер COM-соединений](Менеджер_COM-соединений.md) › [Администрирование кластера серверов](Администрирование_кластера_серверов.md) › [Рабочий сервер](Рабочий_сервер_(IWorkingServerInfo).md) › [Свойства](Рабочий_сервер__Свойства.md)

Рабочий сервер (IWorkingServerInfo)

ConnectionsPerWorkingProcessLimit (ConnectionsPerWorkingProcessLimit)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит ограничение количества соединений с информационными базами через один рабочий процесс. При достижении этого количества на рабочий процесс не могут назначаться новые соединения с информационными базами.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
