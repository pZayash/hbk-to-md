Расширение поля формы для поля надписи (Form field extension for a label field)

Ширина (Width)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Ширина представления данных в форме. Задается в символах. Используется при расчете макета формы.   
0 - автоматическое определение.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
