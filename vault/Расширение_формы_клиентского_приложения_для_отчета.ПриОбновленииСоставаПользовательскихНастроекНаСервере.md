Расширение формы клиентского приложения для отчета (Client application form extension for reports)

ПриОбновленииСоставаПользовательскихНастроекНаСервере (OnUpdateUserSettingSetAtServer)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПриОбновленииСоставаПользовательскихНастроекНаСервере(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: Булево.   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение Ложь, стандартная обработка события производиться не будет.  
При стандартной обработке события система генерирует элементы формы для редактирования пользовательских настроек аналогично методу [СоздатьЭлементыФормыПользовательскихНастроек](Расширение_формы_клиентского_приложения_для_отчета.СоздатьЭлементыФормыПользовательскихНастроек.md).   
При этом, если тип формы [Основная](ТипФормыОтчета.Основная.md), то генерируются элементы формы для быстрых пользовательских настроек. Если тип формы [Настройка](ТипФормыОтчета.Настройка.md) или [Вариант](ТипФормыОтчета.Вариант.md), то генерируются элементы формы для всех пользовательских настроек.  
Значение по умолчанию: Истина.

Описание:

Вызывается при обновлении состава пользовательских настроек:

- При получении формы отчета;
- После загрузки варианта отчета;
- После изменения варианта отчета.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
