# РольПоляНабораДанныхКомпоновкиДанных.ИгнорироватьЗначенияNULL

РольПоляНабораДанныхКомпоновкиДанных (DataCompositionDataSetFieldRole)

ИгнорироватьЗначенияNULL (IgnoreNULLValues)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что в результат не нужно включать групповые записи по данному полю в случае, если поле содержит значение NULL.  
[Истина](lang__def_BooleanTrue.md) - не нужно включать.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
