СправочникСсылка.<Имя справочника> (CatalogRef.<Catalog name>)

Пустая (IsEmpty)

Доступен, начиная с версии 8.0.

Синтаксис:

Пустая()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - ссылка не указывает ни на какой объект (пустая ссылка); [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, является ли ссылка пустой или нет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Если ТоварСсылка.Пустая() Тогда     Предупреждение(НСтр("ru = 'Не выбран товар!';"      + " en = 'Article of trade is not selected!'"),60);     Возврат; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
