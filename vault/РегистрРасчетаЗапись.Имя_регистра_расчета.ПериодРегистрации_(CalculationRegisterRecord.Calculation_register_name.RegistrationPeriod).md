РегистрРасчетаЗапись.<Имя регистра расчета> (CalculationRegisterRecord.<Calculation register name>)

ПериодРегистрации (RegistrationPeriod)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Период регистрации - дата, которая принимает дискретные значения в зависимости от периодичности регистра расчета. Например, если регистр помесячный, то дата всегда начало какого-нибудь месяца.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
