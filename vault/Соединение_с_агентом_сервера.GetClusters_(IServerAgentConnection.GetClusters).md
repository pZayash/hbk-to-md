Соединение с агентом сервера (IServerAgentConnection)

GetClusters (GetClusters)

Доступен, начиная с версии 8.1.

Синтаксис:

GetClusters()

Возвращаемое значение:

Тип: [COMSafeArray](COMSafeArray_(COMSafeArray).md).   
Массив объектов [Кластер серверов](Кластер_серверов_(IClusterInfo).md).

Описание:

Позволяет получить список кластеров, зарегистрированных на центральном сервере.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
