БуферДвоичныхДанных (BinaryDataBuffer)

ПрочитатьЦелое64 (ReadInt64)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПрочитатьЦелое64(<Позиция>, <ПорядокБайтов>)

Параметры:

<Позиция> (обязательный)

Тип: Число.   
Позиция, на которой требуется прочитать число.

<ПорядокБайтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов.md).   
Порядок байтов, используемый при чтении числа.  
Если не задан, используется порядок, определенный для текущего экземпляра [ЧтениеДанных](ЧтениеДанных.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Число.   

Описание:

Выполняет чтение целого 64-битного положительного числа на заданной позиции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.9.
