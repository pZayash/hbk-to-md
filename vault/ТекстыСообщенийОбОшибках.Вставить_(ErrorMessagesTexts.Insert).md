ТекстыСообщенийОбОшибках (ErrorMessagesTexts)

Вставить (Insert)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Вставить(<Категория>, <ТекстыСообщенияОбОшибке>)

Параметры:

<Категория> (обязательный)

Тип: [КатегорияОшибки](КатегорияОшибки_(ErrorCategory).md).   
Категория ошибки.

<ТекстыСообщенияОбОшибке> (обязательный)

Тип: [ТекстыСообщенияОбОшибке](ТекстыСообщенияОбОшибке_(ErrorMessageTexts).md).   
Тексты сообщения об ошибке.

Описание:

Вставляет новое значение по ключу, заменяет уже существующее значение.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.17.
