# Расширение формы набора записей регистра расчета.ПриЗаписи

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры расчета](Регистры_расчета.md) › [Расширение формы набора записей регистра расчета](Расширение_формы_набора_записей_регистра_расчета_(Calculation_register_record_set_form_extension).md) › [События](Расширение_формы_набора_записей_регистра_расчета__События.md)

Расширение формы набора записей регистра расчета (Calculation register record set form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись объекта выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета.ПриЗаписи_(CalculationRegisterRecordSet.Calculation_register_name.OnWrite).md) объекта [РегистрРасчетаНаборЗаписей.](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета_(CalculationRegisterRecordSet.Calculation_register_name).md)<[Имя регистра расчета](РегистрРасчетаНаборЗаписей.Имя_регистра_расчета_(CalculationRegisterRecordSet.Calculation_register_name).md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
