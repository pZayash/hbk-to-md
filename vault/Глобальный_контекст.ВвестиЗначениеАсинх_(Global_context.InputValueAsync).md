# Глобальный контекст.ВвестиЗначениеАсинх

Глобальный контекст (Global context)

ВвестиЗначениеАсинх (InputValueAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ВвестиЗначениеАсинх(<Значение>, <Подсказка>, <Тип>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Имя доступной в модуле переменной. Если параметр <Тип> не задан или имеет значение [Неопределено](lang__def_Undefined.md), тип данной переменной используется в качестве типа вводимого в диалоге данного. Начальное значение переменной будет использовано в качестве начального значения в диалоге.

<Подсказка> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст заголовка окна диалога ввода значения. Может использоваться в качестве подсказки пользователю.  
Значение по умолчанию: Пустая строка.

<Тип> (необязательный)

Тип: [Тип](lang__def_Type.md), [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Тип вводимого значения. Если параметр не указан, используется тип параметра <Значение>.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - выбранное значение или [Неопределено](lang__def_Undefined.md), если пользователь отказался от ввода.

Описание:

Асинхронная функция. Вызывает диалог для ввода значения заданного типа. Если тип начального значения не определен и <Тип> представляет собой составной тип данных, то в поле ввода появляется кнопка выбора типа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ВвестиЗначение](Глобальный_контекст.ВвестиЗначение_(Global_context.InputValue).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
