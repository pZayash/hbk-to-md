Расширение табличного поля списка справочника (Catalog list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то пометка удаления изменена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед установкой пометки удаления в списке справочника.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](СправочникОбъект.Имя_справочника.ПередЗаписью.md) и [ПриЗаписи](СправочникОбъект.Имя_справочника.ПриЗаписи.md) объекта [СправочникОбъект.](СправочникОбъект.Имя_справочника.md)<[Имя справочника](СправочникОбъект.Имя_справочника.md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
