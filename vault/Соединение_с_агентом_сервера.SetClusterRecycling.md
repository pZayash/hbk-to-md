Соединение с агентом сервера (IServerAgentConnection)

SetClusterRecycling (SetClusterRecycling)

Доступен, начиная с версии 8.2.

Синтаксис:

SetClusterRecycling(<Кластер>, <ПериодПерезапуска>, <ТаймаутЗавершения>)

Параметры:

<Кластер> (обязательный)

Тип: [Кластер серверов](Кластер_серверов.md).   
Кластер, который должен быть модифицирован.

<ПериодПерезапуска> (обязательный)

Тип: Число.   
Период перезапуска рабочих процессов кластера.  
Интервал времени ( в секундах), по истечении которого будет выполнен перезапуск рабочих процессов кластера. Если рабочий процесс активен в течение этого времени, то будет выполнен запуск нового процесса, а старый процесс будет выключен. Выключенный процесс будет обслуживать соединения, установленные ранее, не устанавливая при том новых соединений.   
Если параметр не указан, то перезапуск рабочих процессов не производится.

<ТаймаутЗавершения> (обязательный)

Тип: Число.   
Время принудительного завершения.  
Интервал времени (в секундах), по истечении которого выключенный процесс будет завершен принудительно даже, если еще имеются установленные с ним соединения. Если процесс выключен автоматически по достижении времени перезапуска, то после принудительного завершения процесс будет удален из кластера.   
Если параметр не указан, принудительное завершение не производится.

Описание:

Устанавливает параметры автоматического перезапуска рабочих процессов кластера.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
