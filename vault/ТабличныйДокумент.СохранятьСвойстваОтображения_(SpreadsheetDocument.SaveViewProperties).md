ТабличныйДокумент (SpreadsheetDocument)

СохранятьСвойстваОтображения (SaveViewProperties)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак сохранения свойств отображения табличного документа: отображения заголовков, сетки, групп, имен, позиции фиксации.  
[Истина](lang__def_BooleanTrue.md) - свойства сохраняются.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
