# ДиалогВыбораТипаДиаграммы.Показать

ДиалогВыбораТипаДиаграммы (ChartTypeChooseDialog)

Показать (Show)

Доступен, начиная с версии 8.3.13.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога со следующими параметрами:

- <ТипДиаграммы> - [ТипДиаграммы](ТипДиаграммы_(ChartType).md) – тип диаграммы выбран. [Неопределено](lang__def_Undefined.md) - в противном случае;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).
- Если параметр не указан или имеет тип [Неопределено](lang__def_Undefined.md), то по завершении никакая процедура вызвана не будет.

Описание:

Открывает немодальный диалог выбора типа диаграммы.  
По умолчанию будет установлен тип диаграммы, установленный в свойстве [ТипДиаграммы](ДиалогВыбораТипаДиаграммы.ТипДиаграммы_(ChartTypeChooseDialog.ChartType).md).  
Для выбора будут доступны типы диаграммы, установленные в свойстве [ДоступныеТипыДиаграммы](ДиалогВыбораТипаДиаграммы.ДоступныеТипыДиаграммы_(ChartTypeChooseDialog.AvailableChartTypes).md).

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
