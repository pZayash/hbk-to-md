ОбластьПостроенияДиаграммыГанта (GanttChartPlotArea)

ОтображатьТекстТочек (ShowPointsText)

Доступен, начиная с версии 8.3.21.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеВДиаграммеГанта](ОтображениеВДиаграммеГанта.md).   
Настройка отображения текста точек в области построения.   
Значение [Авто](ОтображениеВДиаграммеГанта.Авто.md) интерпретируется следующим образом: если у диаграммы как элемента формы включено отображение таблицы в качестве предопределенного элемента (значение свойства [ПоложениеТаблицы](Расширение_поля_формы_для_поля_диаграммы_Ганта.ПоложениеТаблицы.md) поля диаграммы не равно [Нет](ПоложениеТаблицыДиаграммыГанта.Нет.md)), то [НеОтображать](ОтображениеВДиаграммеГанта.НеОтображать.md), иначе - [Отображать](ОтображениеВДиаграммеГанта.Отображать.md).  
Значение по умолчанию - [Авто](ОтображениеВДиаграммеГанта.Авто.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.21.
