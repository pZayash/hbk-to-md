ПараметрПеретаскиванияВнутриПланировщика (PlannerInsideDragParameter)

Конец (End)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: Дата.   

- В случае создания элемента планировщика путем выделения пустой области является датой конца области выделения.
- В случае выделения элементов планировщика является датой конца области выделения.
- В случае редактирования элемента планировщика путем перетаскивания является датой конца редактируемого элемента.
- В случае копирования существующего элемента планировщика является датой конца создаваемого элемента.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
