Расширение поля формы для поля ввода (Form field extension for a text box)

ГиперссылкаМножественныхЗначений (MultipleValuesHyperlink)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак того, что множественное значение отображается в виде гиперссылки:

- [Истина](lang__def_BooleanTrue.md) – текст множественных значений в виде гиперссылки.

- [Ложь](lang__def_BooleanFalse.md) — гиперссылка отсутствует.

- [Неопределено](lang__def_Undefined.md) – гиперссылка множественных значений отображается только для ссылочных типов данных.

Значение по умолчанию: [Неопределено](lang__def_Undefined.md).  
Используется, если в поле ввода редактируются данные типа [СписокЗначений](СписокЗначений_(ValueList).md) или [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
