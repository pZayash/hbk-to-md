ТабличноеПоле (TableBox)

НачальноеОтображениеДерева (InitialTreeView)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [НачальноеОтображениеДерева](НачальноеОтображениеДерева_(InitialTreeView).md).   
Определяет вариант начального отображения дерева связанного с табличным полем списка при открытии.

Доступность:

Толстый клиент.

Примечание:

Используется, если связанный с табличным полем список отображается в виде дерева.

Использование в версии:

Доступен, начиная с версии 8.0.
