Описание:

Представляет абстракцию формы (управляемой и обычной). Позволяет вызвать закрытие окна и получить список подчиненных элементов управления. Может содержать объекты типа [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы.md), [ТестируемаяКнопкаФормы](ТестируемаяКнопкаФормы.md), [ТестируемаяТаблицаФормы](ТестируемаяТаблицаФормы.md), [ТестируемоеПолеФормы](ТестируемоеПолеФормы.md), [ТестируемаяДекорацияФормы](ТестируемаяДекорацияФормы.md).

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.1.
