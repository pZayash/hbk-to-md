# КоллекцияИнтерваловФонаПланировщика

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Планировщик](Планировщик-2.md)

Доступен, начиная с версии 8.3.6.

Элементы коллекции:

[ИнтервалФонаПланировщика](ИнтервалФонаПланировщика_(PlannerBackgroundInterval).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются интервалы фона.  
Возможно обращение к интервалу фона посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию интервалов фона планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
