# НаправлениеСортировки

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Системные_перечисления.md">Системные перечисления</a> › [Разные](Разные.md)

Доступен, начиная с версии 8.0.

Значения

[Возр (Asc)](НаправлениеСортировки.Возр.md)  
[Убыв (Desc)](НаправлениеСортировки.Убыв.md)  

Описание:

Определяет возможные способы сортировки.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

Пример:

|  |
| --- |
| СписокТиповЦен.СортироватьПоЗначению(НаправлениеСортировки.Возр); |
