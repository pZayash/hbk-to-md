Описание:

Содержит варианты отображения группы формы клиентского приложения вида [ОбычнаяГруппа](ВидГруппыФормы.ОбычнаяГруппа.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Расширение группы формы для обычной группы](Расширение_группы_формы_для_обычной_группы.md), свойство [Отображение](Расширение_группы_формы_для_обычной_группы.Отображение.md)

Доступен, начиная с версии 8.2.
