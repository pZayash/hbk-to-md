ЗапускПриложенияМобильногоУстройства (MobileDeviceApplicationRun)

ПоддерживаетсяЗапуск (RunningSupported)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПоддерживаетсяЗапуск()

Возвращаемое значение:

Тип: Булево.   

- Истина - для Android,
- Ложь - для остальных операционных систем.

Описание:

Определяет, поддерживается ли запуск приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[ЗапускПриложенияМобильногоУстройства](ЗапускПриложенияМобильногоУстройства.md), метод [Запустить](ЗапускПриложенияМобильногоУстройства.Запустить.md)
