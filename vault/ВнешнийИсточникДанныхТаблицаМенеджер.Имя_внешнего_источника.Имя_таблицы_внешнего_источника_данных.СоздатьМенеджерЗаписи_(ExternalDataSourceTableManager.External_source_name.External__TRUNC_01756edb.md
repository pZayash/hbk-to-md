СоздатьМенеджерЗаписи (CreateRecordManager)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СоздатьМенеджерЗаписи()

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаМенеджерЗаписи.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаМенеджерЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecordManager.External_source_name.External_data_source_table_name).md).   

Описание:

Создает объект для управления записью таблицы внешнего источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных [НеобъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.НеобъектныеДанные_(ExternalDataSourceTableDataType.NonobjectData).md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
