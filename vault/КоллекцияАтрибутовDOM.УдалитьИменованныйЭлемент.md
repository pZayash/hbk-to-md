КоллекцияАтрибутовDOM (DOMAttributeMap)

УдалитьИменованныйЭлемент (RemoveNamedItem)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

УдалитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя удаляемого атрибута DOM.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

УдалитьИменованныйЭлемент(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен удаляемого атрибута.

<ЛокальноеИмя> (обязательный)

Тип: Строка.   
Локальное имя атрибута.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md).   

Описание:

Удаляет узел атрибут из коллекции по указанному имени.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
