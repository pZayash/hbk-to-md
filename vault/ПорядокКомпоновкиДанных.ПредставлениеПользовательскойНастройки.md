Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Используется для отображения объекта в пользовательских настройках.  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
