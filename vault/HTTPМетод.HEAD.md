HTTPМетод (HTTPMethod)

HEAD (HEAD)

Доступен, начиная с версии 8.3.5.

Описание:

Соответствует HTTP-методу HEAD.  
Предназначен для получения информации данных (списка заголовков без получения тела).  
Описан в стандарте RFC 2616 (http://tools.ietf.org/html/rfc2616).

Использование в версии:

Доступен, начиная с версии 8.3.5.
