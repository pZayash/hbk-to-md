РасписаниеРегламентногоЗадания (JobSchedule)

ДниНедели (WeekDays)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив номеров дней недели, по которым задание может быть запущено (понедельник - 1, вторник - 2 и т.д.).   
Например: ДеньНеделиВМесяце = 2, ДниНедели={3,5} - выполнять каждую вторую среду и пятницу месяца.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Нецелесообразно вызывать модифицирующие методы для значений данного свойства (поскольку методы используют копию значения, а реальное значение при этом не редактируется).   
При этом значение свойства может быть полностью заменено.

Использование в версии:

Доступен, начиная с версии 8.1.
