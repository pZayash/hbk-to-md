ПостроительОтчета (ReportBuilder)

МакетДетальныхЗаписей (DetailRecordsTemplate)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [Строка](lang__def_String.md).   
Содержит макет (табличный документ) или имя области в макете, используемые для вывода детальных записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
