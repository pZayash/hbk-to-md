ЭлементФреймHTML (HTMLFrameElement)

ПолучитьУзелАтрибута (GetAttributeNode)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьУзелАтрибута(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Возвращаемое значение:

Тип: [АтрибутHTML](АтрибутHTML_(HTMLAttribute).md), [Неопределено](lang__def_Undefined.md).   
Если у элемента нет атрибута с указанными параметрами, то будет возвращено значение [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает узел атрибута по имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
