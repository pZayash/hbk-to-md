ЭлементПакетаОтображаемыхДокументов (RepresentableDocumentBatchItem)

Данные (Data)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка, [ПакетОтображаемыхДокументов](ПакетОтображаемыхДокументов.md).   
Содержит данные пакета отображаемых документов.  
Строка - адрес временного хранилища, в котором расположен документ. [ПакетОтображаемыхДокументов](ПакетОтображаемыхДокументов.md) - вложенный пакет документов. В качестве документов может выступать табличный документ.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
