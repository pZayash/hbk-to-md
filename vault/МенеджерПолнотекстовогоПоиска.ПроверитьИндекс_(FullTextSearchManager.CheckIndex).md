# МенеджерПолнотекстовогоПоиска.ПроверитьИндекс

МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

ПроверитьИндекс (CheckIndex)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПроверитьИндекс()

Возвращаемое значение:

Тип: Булево.   
[Ложь](lang__def_BooleanFalse.md) - если файлах индекса содержатся некорректные данные. В этом случае рекомендуется использовать методы [ОчиститьИндекс](МенеджерПолнотекстовогоПоиска.ОчиститьИндекс_(FullTextSearchManager.ClearIndex).md) и [ОбновитьИндекс](МенеджерПолнотекстовогоПоиска.ОбновитьИндекс_(FullTextSearchManager.UpdateIndex).md).

Описание:

Проверяет целостность файлов полнотекстового поиска.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если в момент запуска проверки выполняется обновление, очистка или проверка данных полнотекстового поиска, будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.6.
