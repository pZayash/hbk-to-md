Расширение табличного поля списка плана счетов (Chart of accounts list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от изменения пометки удаления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то пометка удаления изменена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед установкой пометки удаления в списке счетов.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](ПланСчетовОбъект.Имя_плана_счетов.ПередЗаписью_(ChartOfAccountsObject.Chart_of_accounts_name.BeforeWrite).md) и [ПриЗаписи](ПланСчетовОбъект.Имя_плана_счетов.ПриЗаписи_(ChartOfAccountsObject.Chart_of_accounts_name.OnWrite).md) объекта [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов_(ChartOfAccountsObject.Chart_of_accounts_name).md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
