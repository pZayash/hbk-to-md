Диаграмма (Chart)

ЛинииСоединенияЗначенийПоСериям (ValuesBySeriesConnectionLines)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Линия](Линия.md).   
Определяет стиль и толщину линий соединений значений по сериям. При выборе значения [НетЛинии](ТипЛинииДиаграммы.НетЛинии.md), соединения выполняются для каждой серии собственными линиями, соответствующими свойству [Линия](СерияДиаграммы.Линия.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
