Описание:

Данный объект представляет собой коллекцию объектов, описывающих оформление диаграммы.  
Для оформления ячейки таблицы используются следующие элементы оформления:

- ТипДиаграммы (ChartType) - тип диаграммы (тип [ТипДиаграммы](ТипДиаграммы_(ChartType).md));
- ТипДиаграммы.АвтоРаздвижениеСерий (ChartType.AutoSeriesSeparation) - режим раздвижения (тип [АвтоРаздвижениеСерий](АвтоРаздвижениеСерий_(AutoSeriesSeparation).md));
- ТипДиаграммы.БазовоеЗначение (ChartType.BaseValue) – базовое значение (тип [Число](lang__def_Number.md));
- ТипДиаграммы.ВидПодписей (ChartType.LabelType) - состав подписей (тип [ВидПодписейКДиаграмме](ВидПодписейКДиаграмме_(ChartLabelType).md));
- ТипДиаграммы.ВлияниеРазмераНаПузырек (ChartType.ChartBubbleSizing) - определяет зависимость размера пузырька от значения (тип [ВлияниеРазмераНаПузырекДиаграммы](ВлияниеРазмераНаПузырекДиаграммы_(ChartBubbleSizing).md));
- ТипДиаграммы.ВнутреннийРадиусКольцевойДиаграммы (ChartType.DonutChartInnerRadius) - настройка размера внутреннего радиуса кольцевой диаграммы (тип [Число](lang__def_Number.md)).
- ТипДиаграммы.ГлубинаОбъемнойДиаграммы (ChartType.Chart3DDepth) - глубина объемной диаграммы. Используется в круговой, кольцевой и объемных гистограммах (тип [Число](lang__def_Number.md));
- ТипДиаграммы.Градиент (ChartType.Gradient) – градиент (тип [Булево](lang__Булево_(Boolean).md));
- ТипДиаграммы.ДополнительнаяОсьЗначений (ChartType.AdditionalValuesAxis) - настройка дополнительной оси значений (тип [ОсьДиаграммы](ОсьДиаграммы_(ChartAxis).md));
- ТипДиаграммы.ДополнительнаяШкалаЗначений (ChartType.AdditionalValuesScale) - настройка дополнительной шкалы значений (тип [ШкалаДиаграммы](ШкалаДиаграммы_(ChartScale).md));
- ТипДиаграммы.ЗаполнениеПодсказкиЗначений (ChartType.ValuesToolTipFillType) - настройка режима заполнения всплывающей подсказки диаграммы (тип [ЗаполнениеПодсказкиЗначенийДиаграммы](ЗаполнениеПодсказкиЗначенийДиаграммы_(ChartValuesToolTipFillType).md));
- ТипДиаграммы.ИнформационныеИнтервалыЗначений (ChartType.ValuesReferenceBands) - информационные интервалы значений в диаграмме (тип [ИнформационныеИнтервалыДиаграммы](ИнформационныеИнтервалыДиаграммы_(ChartReferenceBands).md));
- ТипДиаграммы.ИнформационныеИнтервалыТочек (ChartType.PointsReferenceBands) - информационные интервалы точек в диаграмме (тип [ИнформационныеИнтервалыДиаграммы](ИнформационныеИнтервалыДиаграммы_(ChartReferenceBands).md));
- ТипДиаграммы.ИнформационныеЛинииЗначений (ChartType.ValuesReferenceLines) - информационные линии значений в диаграмме (тип [ИнформационныеЛинииДиаграммы](ИнформационныеЛинииДиаграммы_(ChartReferenceLines).md));
- ТипДиаграммы.ИнформационныеЛинииТочек (ChartType.PointsReferenceLines) - информационные линии точек в диаграмме (тип [ИнформационныеЛинииДиаграммы](ИнформационныеЛинииДиаграммы_(ChartReferenceLines).md));
- ТипДиаграммы.ИспользованиеНечисловыхЗначений (ChartType.NonnumericValuesUse) - настройка использования значений, которые не приводятся к числу (тип [ИспользованиеНечисловыхЗначенийДиаграммы](ИспользованиеНечисловыхЗначенийДиаграммы_(NonnumericChartValueUse).md));
- ТипДиаграммы.ИсточникЗначенийОсиТочек (ChartType.ChartPointsAxisValuesSource) - определяет вид значений на оси точек (тип [ИсточникЗначенийОсиТочекДиаграммы](ИсточникЗначенийОсиТочекДиаграммы_(ChartPointsAxisValuesSource).md));
- ТипДиаграммы.ИсточникЗначенияРазмераПузырька (ChartType.BubbleSizeValueSource) - источник значения размера пузырьков (тип [ИсточникЗначенияРазмераПузырькаДиаграммы](ИсточникЗначенияРазмераПузырькаДиаграммы_(ChartBubbleSizeValueSource).md));
- ТипДиаграммы.КлючСохраненияЦветов (ChartType.SaveColorsKey) - ключ, по которому в системном хранилище настроек лежит [СохранениеОформленияКомпоновкиДанных](СохранениеОформленияКомпоновкиДанных_(SaveDataCompositionAppearance).md). Если по ключу ничего нет, то создает новую запись. Если по ключу лежит объект другого типа, то параметр ТипДиаграммы.КлючСохраненияЦветов игнорируется (тип [Строка](lang__def_String.md)).
- ТипДиаграммы.КонечныйУгол (ChartType.EndAngle) - угол конца отображения секторов. Используется в круговой и кольцевой диагарммах (тип [Число](lang__def_Number.md));
- ТипДиаграммы.ЛинииСоединенияЗначенийПоСериям (ChartType.ValuesBySeriesConnectionLines) - стиль и толщина линий соединений (тип [Линия](Линия_(Line).md));
- ТипДиаграммы.ЛинииШкал (ChartType.ScaleLines) - стиль линий шкалы (тип [Линия](Линия_(Line).md));
- ТипДиаграммы.МаксимальноеЗначение (ChartType.MaxValue) - максимальное значение (тип [Число](lang__def_Number.md));
- ТипДиаграммы.МаксимальныйРазмерПузырька (ChartType.MaxBubbleSize) - максимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения) если для размера пузырьков указана серия значений (тип [Число](lang__def_Number.md));
- ТипДиаграммы.МинимальноеЗначение (ChartType.MinValue) - минимальное значение (тип [Число](lang__def_Number.md));
- ТипДиаграммы.МинимальныйРазмерПузырька (ChartType.MinBubbleSize) - минимальный диаметр пузырька (в процентах от минимума из ширины и высоты области построения), если для размера пузырьков указана серия значений (тип [Число](lang__def_Number.md));
- ТипДиаграммы.НатяжениеСглаживания (ChartType.SplineStrain) - целочисленное значение определяет "натяжение" кривой для режима сглаживания [ГладкаяКривая](РежимСглаживанияДиаграммы.ГладкаяКривая_(ChartSplineMode.SmoothCurve).md);
- ТипДиаграммы.НачальныйУгол (ChartType.BeginAngle) - угол начала отображения секторов по часовой стрелке от вертикальной оси. Используется в круговой и кольцевой диагарммах (тип [Число](lang__def_Number.md));
- ТипДиаграммы.ОбщаяСерияРазмераПузырька (ChartType.BubbleSizeCommonSeries) - серия, значения которой будут использоваться для размеров пузырьков для всех серий по оси ординат (тип Произвольн);
- ТипДиаграммы.Окантовка (ChartType.Outline) – окантовка (тип [Булево](lang__Булево_(Boolean).md));
- ТипДиаграммы.ОписаниеПалитрыЦветов (ChartType.ColorPaletteDescription) - настройка палитры цветов серий в диаграмме (тип [ОписаниеПалитрыЦветовДиаграммы](ОписаниеПалитрыЦветовДиаграммы_(ChartColorPaletteDescription).md));
- ТипДиаграммы.ОписаниеПалитрыЦветовИнформационныхИнтервалов (ChartType.ReferenceBandsColorPaletteDescription) - настройка палитры цветов для информационных интервалов (тип ОписаниеПалитрыЦветовДиаграммы)
- ТипДиаграммы.ОсьЗначений (ChartType.ValuesAxis) - настройки оси значений в диаграмме (тип [ОсьДиаграммы](ОсьДиаграммы_(ChartAxis).md));
- ТипДиаграммы.ОсьТочек (ChartType.PointsAxis) - настройки оси точек в диаграмме (тип [ОсьДиаграммы](ОсьДиаграммы_(ChartAxis).md));
- ТипДиаграммы.ОтображатьТаблицуДанных (ChartType.ShowDataTable) - отображать таблицу данных (тип [Булево](lang__Булево_(Boolean).md));
- ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииЗначений (ChartType.ShowValuesPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале значений при показе подсказки (тип [ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы](ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы_(ShowChartPopupReferenceLine).md));
- ТипДиаграммы.ОтображениеВсплывающейИнформационнойЛинииТочек (ChartType.ShowPointsPopupReferenceLine) - настройка режима отображения всплывающей информационной линии к шкале точек при показе подсказки диаграммы (тип [ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы](ОтображатьВсплывающуюИнформационнуюЛиниюДиаграммы_(ShowChartPopupReferenceLine).md));
- ТипДиаграммы.ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы (ChartType.BubbleChartNegativeValuesShowMode) - указывает вариант работы с отрицательными значениями размера (тип [ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы](ОтображениеОтрицательныхЗначенийПузырьковойДиаграммы_(BubbleChartNegativeValuesShowMode).md));
- ТипДиаграммы.ПорядокОтображенияТочекГоризонтальнойГистограммы (ChartType.BarChartPointsOrder) - порядок вывода точек в горизонтальных диаграммах (тип [ПорядокОтображенияТочекГоризонтальнойГистограммы](ПорядокОтображенияТочекГоризонтальнойГистограммы_(BarChartPointsOrder).md));
- ТипДиаграммы.ПорядокСерийВЛегенде (ChartType.SeriesOrderInLegend) - определяет порядок отображения серий в легенде диаграммы (тип [ПорядокСерийВЛегендеДиаграммы](ПорядокСерийВЛегендеДиаграммы_(ChartSeriesOrderInLegend).md));
- ТипДиаграммы.ПропускатьБазовоеЗначение (ChartType.HideBaseValue) - пропускать точки с базовым значением (тип [Булево](lang__Булево_(Boolean).md));
- ТипДиаграммы.РазмерПузырькаПоУмолчанию (ChartType.DefaultBubbleSize) - указывает базовый диаметр пузырька в процентах от минимума из ширины и высоты области построения (тип [Число](lang__def_Number.md));
- ТипДиаграммы.РасположениеЛегенды (ChartType.LegendPlacement) - расположение легенды в диаграммы (тип [РасположениеЛегендыДиаграммыКомпоновкиДанных](РасположениеЛегендыДиаграммыКомпоновкиДанных_(DataCompositionChartLegendPlacement).md));
- ТипДиаграммы.РасположениеРесурсов (ChartType.ResourcesPlacement) - расположение ресурсов в диаграмме (тип [РасположениеРесурсовВДиаграммеКомпоновкиДанных](РасположениеРесурсовВДиаграммеКомпоновкиДанных_(DataCompositionResourcesPlacementInChart).md));
- ТипДиаграммы.РежимВыделения (ChartType.SelectionMode) - настройка режима выделения диаграммы (тип [РежимВыделенияДиаграммы](РежимВыделенияДиаграммы_(ChartSelectionMode).md));
- ТипДиаграммы.РежимОтображенияПодсказкиЗначений (ChartType.ValuesToolTipShowMode) - настройка режима заполнения всплывающей подсказки диаграммы (тип [ЗаполнениеПодсказкиЗначенийДиаграммы](ЗаполнениеПодсказкиЗначенийДиаграммы_(ChartValuesToolTipFillType).md));
- ТипДиаграммы.РежимПолупрозрачности (ChartType.SemitransparencyMode) - определяет режим отображения для перекрывающихся элементов диаграмм (тип [РежимПолупрозрачностиДиаграммы](РежимПолупрозрачностиДиаграммы_(ChartSemitransparencyMode).md));
- ТипДиаграммы.РежимСглаживания (ChartType.SplineMode) - определяет режим сглаживания линий в диаграммах "непрерывного" вида (тип [РежимСглаживанияДиаграммы](РежимСглаживанияДиаграммы_(ChartSplineMode).md));
- ТипДиаграммы.СерияНаОсиТочек (ChartType.PointsAxisSeries) - серия, значения которой откладываются на оси точек (тип Произвольный);
