ОтображениеВДиаграмме (ShowInChart)

Авто (Auto)

Доступен, начиная с версии 8.3.10.

Описание:

Отображение серии зависит от свойства диаграммы [МаксимумСерий](Диаграмма.МаксимумСерий_(Chart.MaxSeries).md): отображается, если режим ограничения по сериям отключен.

Использование в версии:

Доступен, начиная с версии 8.3.10.
