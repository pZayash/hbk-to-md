ВложенныеНаборыДанныхМакетаКомпоновкиДанных (DataCompositionTemplateNestedDataSets)

Добавить (Add)

Доступен, начиная с версии 8.2.

Синтаксис:

Добавить()

Возвращаемое значение:

Тип: [ВложенныйНаборДанныхМакетаКомпоновкиДанных](ВложенныйНаборДанныхМакетаКомпоновкиДанных_(DataCompositionTemplateNestedDataSet).md).   

Описание:

Добавляет новый элемент коллекции и возвращает его. В случае, если тип нового элемента не подходит для данной коллекции, добавления не происходит, и метод возвращает [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
