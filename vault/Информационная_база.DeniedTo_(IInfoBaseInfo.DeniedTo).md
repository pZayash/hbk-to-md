Информационная база (IInfoBaseInfo)

DeniedTo (DeniedTo)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Конец интервала времени, в течение которого действует режим блокировки сеансов.   
Если указана пустая дата ('00010101'), то без ограничения.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
