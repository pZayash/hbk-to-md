НастройкаВходныхКолонокМоделиПрогноза (PredictionModelInputColumnsSetting)

<Имя колонки прогноза> (<Forecast column name>)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [НастройкаВходнойКолонкиМоделиПрогноза](НастройкаВходнойКолонкиМоделиПрогноза_(PredictionModelInputColumnSetting).md).   
Набор свойств содержит значения колонок. Доступ к колонке модели прогноза осуществляется по имени. Имена свойств совпадают с именами колонок прогноза.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
