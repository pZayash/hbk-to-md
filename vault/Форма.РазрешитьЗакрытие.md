Форма (Form)

РазрешитьЗакрытие (AllowClose)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет интерактивной возможностью закрытия формы.

Доступность:

Толстый клиент.

Примечание:

Ложь - форму нельзя закрыть интерактивно. В этом случае форму можно закрыть только методом [Закрыть](Форма.Закрыть.md).

Использование в версии:

Доступен, начиная с версии 8.1.
