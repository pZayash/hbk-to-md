НастройкаПорядка (OrderSetting)

Очистить (Clear)

Доступен, начиная с версии 8.0.

Синтаксис:

Очистить()

Описание:

Очищает коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Метод не доступен для настройки отбора динамических списков (свойство НастройкаПорядка объектов [СправочникСписок.](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)<[Имя справочника](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)>, [ДокументСписок.](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)<[Имя документа](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)> и т.д.).

Использование в версии:

Доступен, начиная с версии 8.0.
