ОписаниеКонфигурации (ConfigurationDescription)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ОписаниеКонфигурации(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Двоичные данные файла, из которого будет сформирован объект [ОписаниеКонфигурации](ОписаниеКонфигурации_(ConfigurationDescription).md).

Описание:

Создает объект на основе двоичных данных файла формата \*.cf или \*.cfe.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации_(ConfigurationExtension).md), метод [ПолучитьДанные](РасширениеКонфигурации.ПолучитьДанные_(ConfigurationExtension.GetData).md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
