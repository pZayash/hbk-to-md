ОписаниеКонфигурации (ConfigurationDescription)

По умолчанию

Доступен, начиная с версии 8.3.13.

Синтаксис:

Новый ОписаниеКонфигурации(<ДвоичныеДанные>)

Параметры:

<ДвоичныеДанные> (обязательный)

Тип: [ДвоичныеДанные](ДвоичныеДанные.md).   
Двоичные данные файла, из которого будет сформирован объект [ОписаниеКонфигурации](ОписаниеКонфигурации.md).

Описание:

Создает объект на основе двоичных данных файла формата \*.cf или \*.cfe.

См. также:

[РасширениеКонфигурации](РасширениеКонфигурации.md), метод [ПолучитьДанные](РасширениеКонфигурации.ПолучитьДанные.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
