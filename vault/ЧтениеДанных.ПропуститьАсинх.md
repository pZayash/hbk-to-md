ЧтениеДанных (DataReader)

ПропуститьАсинх (SkipAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПропуститьАсинх(<Количество>)

Параметры:

<Количество> (обязательный)

Тип: Число.   
Количество байтов, которые требуется пропустить.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - количество фактически пропущенных данных.

Описание:

Асинхронная функция. Пропускает заданное количество байтов в потоке.

Доступность:

Тонкий клиент, толстый клиент.
