ДействиеПриНесоответствииПароляТребованиямПриАутентификации (ActionOnThePasswordRequirementsViolationOnAuthentication)

ПотребоватьСменуПароля (RequirePasswordChange)

Доступен, начиная с версии 8.3.26.

Использование:

Только чтение.

Описание:

Отображается диалог смены пароля без возможности пропустить действие, и выполняется запись события в журнал регистрации.  
Если пользователю запрещено изменять пароль, будет сгенерировано исключение.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
