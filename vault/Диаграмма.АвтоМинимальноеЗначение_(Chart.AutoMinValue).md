Диаграмма (Chart)

АвтоМинимальноеЗначение (AutoMinValue)

Не рекомендуется использовать, начиная с версии 8.3.10.

Рекомендуется использовать:

- [ОсьЗначений](Диаграмма.ОсьЗначений_(Chart.ValuesAxis).md)
- [СпособОпределенияМинимальногоЗначения](ОсьДиаграммы.СпособОпределенияМинимальногоЗначения_(ChartAxis.MinValueDetectionMethod).md)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость автоматического определения минимального значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.10.

Описание изменено в версии 8.3.15.
