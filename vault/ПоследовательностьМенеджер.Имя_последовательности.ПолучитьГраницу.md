ПоследовательностьМенеджер.<Имя последовательности> (SequenceManager.<Sequence name>)

ПолучитьГраницу (GetBound)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьГраницу(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [Структура](Структура.md).   
Структура, задающая отбор по измерениям последовательности. Ключ структуры определяет измерение последовательности, по которому будет осуществляться отбор, а значение структуры - значение измерения.  
Если измерение отсутствует в структуре, то по нему отбор осуществляться не будет.  
Если параметр не указан, то получение границы последовательности происходит по всем значениям измерений.

Возвращаемое значение:

Тип: [МоментВремени](Прикладные_объекты.МоментВремени.md).   
Момент времени, соответствующий минимальной границе последовательности.

Описание:

Получает минимальную границу последовательности по заданным значениям измерений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Анализируются все границы, без учета наличия или отсутствия зарегистрированных документов по данным границам.

Пример:

|  |
| --- |
| Отбор = Новый Структура ("Товар"); Отбор.Вставить("Товар", ТекТовар); Граница = Последовательности.УчетТоваров.ПолучитьГраницу(Отбор); |

Использование в версии:

Доступен, начиная с версии 8.0.
