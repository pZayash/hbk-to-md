ОписаниеОтображаемогоОбъектаPDF (PDFRepresentationObjectDescription)

НомерСтраницы (PageNumber)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер страницы PDF-документа, на которой будет расположен объект. Начинается с 1.  
Если не указано, то объект будет выведен на последней странице. Если указан номер несуществующей страницы, то генерируется исключение.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.
