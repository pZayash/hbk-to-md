# ОбъектМетаданных: ОбщаяКартинка.ДоступностьДляОформления

ОбъектМетаданных: ОбщаяКартинка (MetadataObject: CommonPicture)

ДоступностьДляОформления (AvailabilityForAppearance)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает доступность картинки для выбора в форме выбора картинки, открываемой из формы редактирования элемента условного оформления в режиме 1С:Предприятие. Доступно для редактирования только тогда, когда свойство [ДоступностьДляВыбора](ОбъектМетаданных_ОбщаяКартинка.ДоступностьДляВыбора_(MetadataObject_CommonPicture.AvailabilityForChoice).md) содержит [Истина](lang__def_BooleanTrue.md).   
По умолчанию содержит [Ложь](lang__def_BooleanFalse.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.
