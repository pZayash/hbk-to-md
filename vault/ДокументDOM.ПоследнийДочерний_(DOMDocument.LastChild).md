ДокументDOM (DOMDocument)

ПоследнийДочерний (LastChild)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ДокументDOM](ДокументDOM_(DOMDocument).md), [Неопределено](lang__def_Undefined.md).   
Последний дочерний узел DOM или [Неопределено](lang__def_Undefined.md), если дочерних узлов нет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
