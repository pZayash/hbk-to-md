Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

РежимОтображенияВыделения (SelectionShowMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтображенияВыделения](РежимОтображенияВыделения_(SelectionShowMode).md).   
Управляет режимом отображения выделения в табличном документе.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
