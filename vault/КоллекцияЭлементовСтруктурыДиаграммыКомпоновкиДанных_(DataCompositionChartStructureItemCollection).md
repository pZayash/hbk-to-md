Элементы коллекции:

[ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных_(DataCompositionChartGroup).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются группировки.  
Возможно обращение к группировке посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Содержит элементы структуры настроек, вложенных в диаграмму или в группировку диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
