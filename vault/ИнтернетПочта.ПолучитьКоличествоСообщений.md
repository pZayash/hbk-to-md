ИнтернетПочта (InternetMail)

ПолучитьКоличествоСообщений (GetMessageCount)

Доступен, начиная с версии 8.3.1.

Синтаксис:

ПолучитьКоличествоСообщений()

Возвращаемое значение:

Тип: Число.   

Описание:

Получает количество сообщений [IMAP](ПротоколИнтернетПочты.IMAP.md) в текущем почтовом ящике ([ТекущийПочтовыйЯщик](ИнтернетПочта.ТекущийПочтовыйЯщик.md)) или количество сообщений в ящике POP3.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
