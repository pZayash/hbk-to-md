КоллекцияИндексовXBase (XBaseIndexesCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Выражение>, <Уникальность>, <Убывание>, <Фильтр>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя создаваемого индекса. Поддерживаются только латинские имена индексов длиной до 10 символов.

<Выражение> (обязательный)

Тип: [Строка](lang__def_String.md).   
Выражение индекса.

<Уникальность> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак уникальности создаваемого индекса. [Истина](lang__def_BooleanTrue.md) - уникальный; [Ложь](lang__def_BooleanFalse.md) - неуникальный.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Убывание> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будет ли создаваемый индекс убывающим. [Истина](lang__def_BooleanTrue.md) - убывает; [Ложь](lang__def_BooleanFalse.md) - возрастает.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Фильтр> (необязательный)

Тип: [Строка](lang__def_String.md).   
Выражение фильтра индекса.  
Значение по умолчанию: ".T.".

Описание:

Добавляет новый индекс в коллекцию индексов таблицы базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB = Новый xBase; xB.Поля.Добавить("CODE", "S", 5); xB.Поля.Добавить("NAME", "S", 40); // Добавим индекс по полю CODE:  xB.Индексы.Добавить("IDXCODE","CODE", Истина); xB.СоздатьФайл("c:\test.dbf", "c:\test.cdx"); |

Использование в версии:

Доступен, начиная с версии 8.0.
