# МенеджерРегламентныхЗаданий.НайтиПредопределенное

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Регламентные задания](Регламентные_задания.md) › [МенеджерРегламентныхЗаданий](МенеджерРегламентныхЗаданий_(ScheduledJobsManager).md) › [Методы](МенеджерРегламентныхЗаданий__Методы.md)

МенеджерРегламентныхЗаданий (ScheduledJobsManager)

НайтиПредопределенное (FindPredefined)

Доступен, начиная с версии 8.1.

Синтаксис:

НайтиПредопределенное(<Метаданные>)

Параметры:

<Метаданные> (обязательный)

Тип: [Строка](lang__def_String.md), [ОбъектМетаданных: РегламентноеЗадание](ОбъектМетаданных_РегламентноеЗадание_(MetadataObject_ScheduledJob).md).   
Имя или объект метаданных.

Возвращаемое значение:

Тип: [РегламентноеЗадание](РегламентноеЗадание_(ScheduledJob).md).   

Описание:

Осуществляет поиск предопределенного регламентного задания по его метаданным. Получение возможно только для администратора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
