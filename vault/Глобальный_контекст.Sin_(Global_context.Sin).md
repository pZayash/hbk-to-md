Глобальный контекст (Global context)

Sin (Sin)

Доступен, начиная с версии 8.0.

Синтаксис:

Sin(<Угол>)

Параметры:

<Угол> (обязательный)

Тип: [Число](lang__def_Number.md).   
Аргумент функции.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Результат вычисления синуса.

Описание:

Вычисляет синус от аргумента <Угол>, заданного в радианах.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

При вычислениях полезно привести значение числа Пи, точности которого достаточно для вычисления тригонометрических функций:  
Pi=3.1415926535897932  
Указание более точного значения числа Пи не оказывает влияния на результат вычислений, т.к. выходит за рамки ограничений представления числа.

Пример:

|  |
| --- |
| Pi = 3.1415926535897932; УголРадианы = 0; УголГрадусы = 0; Для Н=0 По 6 Цикл     Сообщить("Sin("+УголГрадусы+")="+Формат(Sin(УголРадианы),"ЧДЦ=6"));     УголРадианы = УголРадианы + Pi/6;     УголГрадусы = УголГрадусы + 30; КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
