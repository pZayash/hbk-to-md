ИнформационнаяЛинияДиаграммы (ChartReferenceLine)

Значение (Value)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md), [Дата](lang__def_Date.md), [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md), Произвольный.   
Значение, по которому выводится линия.  
Допускаются следующие типы значений:

- для информационной линии, расположенной в значениях серий диаграммы (свойство диаграммы [ИнформационныеЛинииЗначений](Диаграмма.ИнформационныеЛинииЗначений_(Chart.ValuesReferenceLines).md)) - [Число](lang__def_Number.md);
- для информационной линии, расположенной в значениях точек диаграммы (свойство диаграммы [ИнформационныеЛинииТочек](Диаграмма.ИнформационныеЛинииТочек_(Chart.PointsReferenceLines).md):
  - если на оси точек откладываются точки - [ТочкаДиаграммы](ТочкаДиаграммы_(ChartPoint).md);
  - если на оси точек откладываются числа (например, в точечной и пузырьковой диаграммах) - [Число](lang__def_Number.md);
  - если на оси точек откладываются даты (например, в точечной и пузырьковой диаграмме) - [Дата](lang__def_Date.md);
- для информационной линии, расположенной в параметрах вывода компоновки данных:
  - [ПолеКомпоновкиДанных](ПолеКомпоновкиДанных_(DataCompositionField).md);
  - для информационной линии значений - [Число](lang__def_Number.md);
  - для информационной линии точек - тип, установленный в свойстве [Точки](ДиаграммаКомпоновкиДанных.Точки_(DataCompositionChart.Points).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
