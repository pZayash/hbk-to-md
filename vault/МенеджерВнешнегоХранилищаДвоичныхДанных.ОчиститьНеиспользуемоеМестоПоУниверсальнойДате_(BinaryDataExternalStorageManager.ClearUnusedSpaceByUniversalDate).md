# МенеджерВнешнегоХранилищаДвоичныхДанных.ОчиститьНеиспользуемоеМестоПоУниверсальнойДате

МенеджерВнешнегоХранилищаДвоичныхДанных (BinaryDataExternalStorageManager)

ОчиститьНеиспользуемоеМестоПоУниверсальнойДате (ClearUnusedSpaceByUniversalDate)

Доступен, начиная с версии 8.3.24.

Синтаксис:

ОчиститьНеиспользуемоеМестоПоУниверсальнойДате(<ДатаОчистки>)

Параметры:

<ДатаОчистки> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Верхняя граница очистки хранилища от удаленных записей.

Описание:

Очищает хранилище от записей, удаленных раньше указанной даты.  
Вызов метода требует административных прав в сеансе, в котором все разделители не установлены.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.24.

Описание изменено в версии 8.3.26.
