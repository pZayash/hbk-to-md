Файл (File)

ПолучитьУниверсальноеВремяИзмененияАсинх (GetModificationUniversalTimeAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьУниверсальноеВремяИзмененияАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [Дата](lang__def_Date.md), универсальное время последнего изменения файла.

Описание:

Асинхронная функция. Получает универсальное время последнего изменения файла.  
Применима только к файлам и каталогам, которые реально существуют.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Для работы функции на веб-клиенте, необходимо предварительно подключить расширение работы с файлами.
