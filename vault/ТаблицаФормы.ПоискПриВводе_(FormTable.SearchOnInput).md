ТаблицаФормы (FormTable)

ПоискПриВводе (SearchOnInput)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ПоискВТаблицеПриВводе](ПоискВТаблицеПриВводе_(SearchInTableOnInput).md).   
Задает режим выполнения поиска при вводе символов в таблице, связанной с данными типа [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md) или [ДанныеФормыСтруктураСКоллекцией](ДанныеФормыСтруктураСКоллекцией_(FormDataStructureAndCollection).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
