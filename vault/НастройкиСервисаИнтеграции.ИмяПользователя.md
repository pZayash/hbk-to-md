НастройкиСервисаИнтеграции (IntegrationServiceSettings)

ИмяПользователя (UserName)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Если пользователь указан,то обработка полученных сообщений выполняется от имени указанного пользователя.  
Пользователь должен иметь право "Использование" для канала сервиса интеграции.  
Если пользователь не указан, то обработка полученных сообщений происходит с правами, которые определяются набором ролей, указанных в свойстве конфигурации [ОсновныеРоли](ОбъектМетаданныхКонфигурация.ОсновныеРоли.md). В том случае, если в этом свойстве не указано ни одной роли, выполнение происходит без ограничения прав доступа.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.18.
