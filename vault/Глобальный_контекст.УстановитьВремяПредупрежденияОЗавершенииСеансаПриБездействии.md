Глобальный контекст (Global context)

УстановитьВремяПредупрежденияОЗавершенииСеансаПриБездействии (SetInactivityTimeForTerminateSessionNotification)

Доступен, начиная с версии 8.3.26.

Синтаксис:

УстановитьВремяПредупрежденияОЗавершенииСеансаПриБездействии(<Таймаут>)

Параметры:

<Таймаут> (обязательный)

Тип: Число.   
Интервал бездействия в секундах, по истечении которого пользователю отображается предупреждение о завершении сеанса.

Описание:

Устанавливает интервал времени (в секундах), по истечении которого отобразится предупреждение о завершении сеанса по причине бездействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
