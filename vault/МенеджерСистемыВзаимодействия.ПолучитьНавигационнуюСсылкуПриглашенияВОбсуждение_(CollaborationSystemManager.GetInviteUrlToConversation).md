МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьНавигационнуюСсылкуПриглашенияВОбсуждение (GetInviteUrlToConversation)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьНавигационнуюСсылкуПриглашенияВОбсуждение(<Обсуждение>, <ТолькоВидеоконференция>)

Параметры:

<Обсуждение> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Обсуждение.

<ТолькоВидеоконференция> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если [Ложь](lang__def_BooleanFalse.md), то в обсуждении есть возможность читать и писать сообщения.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Если подключена работа с внешними пользователями, то возвращает внешнюю ссылку, иначе – внутреннюю.  
  
Если не подключена работа с видеозвонками и <ТолькоВидеоконференция> имеет значени [Истина](lang__def_BooleanTrue.md), то возвращает пустую ссылку.

Описание:

Возвращает навигационную ссылку-приглашение, по которой можно получить доступ к видеозвонку и обсуждению, если <ТолькоВидеоконференция> имеет значение [Ложь](lang__def_BooleanFalse.md).

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.26.
