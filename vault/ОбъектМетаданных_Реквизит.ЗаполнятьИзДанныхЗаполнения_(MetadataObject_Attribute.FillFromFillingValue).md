# ОбъектМетаданных: Реквизит.ЗаполнятьИзДанныхЗаполнения

ОбъектМетаданных: Реквизит (MetadataObject: Attribute)

ЗаполнятьИзДанныхЗаполнения (FillFromFillingValue)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Свойство определяет возможность заполнения реквизита из данных заполнения. Используется для заполнения реквизитов из значений текущего отбора списков. Если установлено [Истина](lang__def_BooleanTrue.md), то процедура стандартной обработки заполнения будет заполнять реквизит из данных заполнения. Если в данных заполнения нет данных для заполнения этого реквизита или свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то реквизит будет заполнен из свойства [ЗначениеЗаполнения](ОбъектМетаданных_Реквизит.ЗначениеЗаполнения_(MetadataObject_Attribute.FillingValue).md), если оно установлено. Данное свойство устанавливается системой в значение [Истина](lang__def_BooleanTrue.md) для некоторых стандартных реквизитов (Родитель, Владелец, ведущие измерения регистров сведений).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
