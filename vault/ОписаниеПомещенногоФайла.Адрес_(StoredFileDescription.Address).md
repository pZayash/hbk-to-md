ОписаниеПомещенногоФайла (StoredFileDescription)

Адрес (Address)

Доступен, начиная с версии 8.3.15.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Адрес файла во временном хранилище.   
Если файл не был помещен (параметр <Отказ> имеет значение [Истина](lang__def_BooleanTrue.md)), то данное свойство имеет значение [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.15.
