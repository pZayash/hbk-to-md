МенеджерОбработкиОшибок (ErrorProcessingManager)

УстановитьНастройкиПользователя (SetUserSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

УстановитьНастройкиПользователя(<Настройки>, <ИмяПользователя>)

Параметры:

<Настройки> (обязательный)

Тип: [НастройкиОбработкиОшибок](НастройкиОбработкиОшибок_(ErrorProcessingSettings).md).   
Устанавливаемые настройки.

<ИмяПользователя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя пользователя, для которого устанавливаются настройки.  
Если параметр не передан, то используется текущий пользователь.

Описание:

Устанавливает настройки обработки ошибки для переданного пользователя. Требуется наличие у текущего пользователя права "Администрирование данных".

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

См. также:

[МенеджерОбработкиОшибок](МенеджерОбработкиОшибок_(ErrorProcessingManager).md), метод [ПолучитьНастройкиПользователя](МенеджерОбработкиОшибок.ПолучитьНастройкиПользователя_(ErrorProcessingManager.GetUserSettings).md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.
