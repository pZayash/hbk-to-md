Расширение поля формы для поля ввода (Form field extension for a text box)

ПутьКДаннымКартинкиМножественногоЗначения (MultipleValuePictureDataPath)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит путь к реквизиту, содержащему картинку множественного значения.   
Для получения картинки множественных значений в свойстве поля ввода "ПутьКДаннымКартинкиМножественногоЗначения" может быть указана колонка c типом [Картинка](Картинка.md), Строка, Число или Булево:

- Строка - значение в колонке представляет собой навигационную ссылку с адресом картинки.
- Число - значение в колонке представляет собой индекс в наборе картинок, задаваемых свойством [КартинкаМножественныхЗначений](Расширение_поля_формы_для_поля_ввода.КартинкаМножественныхЗначений.md).
- Булево - если Истина используется индекс 0, если Ложь используется индекс 1.

Доступность:

Сервер.
