Глобальный контекст (Global context)

ЗаблокироватьДанныеДляРедактирования (LockDataForEdit)

Доступен, начиная с версии 8.2.

Синтаксис:

ЗаблокироватьДанныеДляРедактирования(<Ключ>, <ВерсияДанных>, <ИдентификаторФормы>)

Параметры:

<Ключ> (обязательный)

Тип: Любая ссылка на объект информационной базы, [РегистрСведенийКлючЗаписи.](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)<[Имя регистра сведений](РегистрСведенийКлючЗаписи.Имя_регистра_сведений_(InformationRegisterRecordKey.Information_register_name).md)>, [ВнешнийИсточникДанныхТаблицаСсылка.](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRef.External_source_name.External_data_source_table_name).md)<[Имя внешнего источника](ВнешнийИсточникДанныхТаблицаСсылка.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRef.External_source_name.External_data_source_table_name).md)>, [ВнешнийИсточникДанныхТаблицаКлючЗаписи.](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecordKey.External_source_name.External_data_source_table_name).md)<[Имя внешнего источника](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных_(ExternalDataSourceTableRecordKey.External_source_name.External_data_source_table_name).md)>.   
Ссылка на объект, который должен быть заблокирован, или ключ записи независимого регистра сведений, которая должна быть заблокирована.

<ВерсияДанных> (необязательный)

Тип: [Строка](lang__def_String.md).   
Версия блокируемого объекта в формате Base64. Используется, если параметр <Ключ> имеет тип Любая ссылка на объект информационной базы.   
Если указан, то при блокировке объекта проверяется версия объекта базы данных. При несовпадении версии или отсутствии объекта в базе данных генерируется исключение.

<ИдентификаторФормы> (необязательный)

Тип: [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md).   
Уникальный идентификатор формы. Если параметр указан, то блокировка устанавливается на все время жизни формы, в которой установлена блокировка. Важно, что если за время отображения формы не были установлены другие блокировки от имени этой формы (при интерактивном редактировании или методом [ЗаблокироватьДанныеФормыДляРедактирования](ФормаКлиентскогоПриложения.ЗаблокироватьДанныеФормыДляРедактирования_(ClientApplicationForm.LockFormDataForEdit).md)), то при закрытии формы блокировка может быть отменена не сразу, а через некоторое время. Блокировка может быть снята:

- автоматически при закрытии формы или завершении сеанса;
- с помощью метода глобального контекста [РазблокироватьДанныеДляРедактирования](Глобальный_контекст.РазблокироватьДанныеДляРедактирования_(Global_context.UnlockDataForEdit).md) с указанием того же идентификатора формы, который указывался для установки блокировки. В этом случае блокировка снимается сразу.

Если параметр не указан, то блокировка устанавливается на ограниченное время и может быть снята:

- с помощью метода [РазблокироватьДанныеДляРедактирования](Глобальный_контекст.РазблокироватьДанныеДляРедактирования_(Global_context.UnlockDataForEdit).md) без указания идентификатора формы;
- при окончании транзакции, если блокировка установлена в транзакции;
- при окончании работы встроенного языка (для режимов запуска "Тонкий клиент" и "Веб-клиент"; при возврате управления с сервера);
- при завершении сеанса.

Описание:

Заблокировать данные для редактирования в форме клиентского приложения.  
Вызывает исключение, если объект уже заблокирован, в том числе и методом Заблокировать.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [РазблокироватьДанныеДляРедактирования](Глобальный_контекст.РазблокироватьДанныеДляРедактирования_(Global_context.UnlockDataForEdit).md)  

Использование в версии:

Доступен, начиная с версии 8.2.
