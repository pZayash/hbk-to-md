ОбъектМетаданных: WebСервис (MetadataObject: WebService)

ПространствоИмен (Namespace)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Для Web-сервиса содержит унифицированный глобальный идентификатор (URI) пространства имен веб-сервиса. Каждый Web-сервис может быть однозначно идентифицирован по своему имени и URI пространству имен, которому он принадлежит.  
Для пакета XDTO содержит унифицированный глобальный идентификатор (URI) пространства имен, которое определяет данный пакет.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
