ИнтернетПочтовыйПрофиль (InternetMailProfile)

ТолькоЗащищеннаяАутентификацияSMTP (SMTPSecureAuthenticationOnly)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает на использование для протокола SMTP только защищенных способов аутентификации из поддерживаемых сервером (на данный момент CRAM-MD5).  
[Ложь](lang__def_BooleanFalse.md) - разрешить использовать незащищенные способы аутентификации, если сервер поддерживает только их.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - соответствует устаревшим значениям свойств АутентификацияSMTP Plain, Login, ПоУмолчанию.  
[Истина](lang__def_BooleanTrue.md) - соответствует устаревшему значению свойства АутентификацияSMTP.CramMD5.

Использование в версии:

Доступен, начиная с версии 8.3.1.
