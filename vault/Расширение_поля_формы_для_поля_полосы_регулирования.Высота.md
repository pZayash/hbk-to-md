Расширение поля формы для поля полосы регулирования (Form field extension for a track bar field)

Высота (Height)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Содержит высоту поля полосы регулирования. Задается в строках.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
