ДокументDOM (DOMDocument)

СоздатьРазыменовательПИ (CreateNSResolver)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: На основе документа DOM

Синтаксис:

СоздатьРазыменовательПИ()

Описание варианта метода:

Разрешение пространств имен выполняется по всем пространствам имен, определенным в документе. В случае совпадения префиксов пространств имен используется то пространство имен, которое определено раньше.

Вариант синтаксиса: На основании узла DOM

Синтаксис:

СоздатьРазыменовательПИ(<УзелКонтекста>)

Параметры:

<УзелКонтекста> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Узел документа DOM, который будет использоваться в качестве контекста, для разрешения пространств имен.

Описание варианта метода:

Разрешение пространств имен выполняется в контексте данного узла, т.е. в разрешении используются пространства имен данного узла и его родительских узлов.

Вариант синтаксиса: На основании Соответствия

Синтаксис:

СоздатьРазыменовательПИ(<Соответствие>)

Параметры:

<Соответствие> (обязательный)

Тип: [Соответствие](Соответствие.md).   
Ключом соответствия выступает префикс пространства имен, а значением - URI пространства имен.

Описание варианта метода:

Создает адаптер для разрешения пространств имен на основании [Соответствие](Соответствие.md).

Вариант синтаксиса: На основании конкретного префикса и URI пространства имен

Синтаксис:

СоздатьРазыменовательПИ(<Префикс>, <URIПространстваИмен>)

Параметры:

<Префикс> (обязательный)

Тип: Строка.   
Префикс.

<URIПространстваИмен> (обязательный)

Тип: Строка.   
URI пространства имен.

Возвращаемое значение:

Тип: [РазыменовательПространствИменDOM](РазыменовательПространствИменDOM.md).   

Описание:

Создает адаптер для разрешения пространств имен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
