ИнтервалФонаПланировщика (PlannerBackgroundInterval)

ЗначенияИзмерений (DimensionValues)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие.md).   
Содержит коллекцию значений измерений. В качестве ключа выступает уникальное [Значение](ИзмерениеПланировщика.Значение.md) измерения (ИзмерениеПланировщика.Значение), в качестве значения выступает уникальное [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md) элемента измерения (ИзмерениеПланировщика.<Элемент>.Значение) или [ФиксированныйМассив](ФиксированныйМассив.md) объектов вида [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение.md) элементов измерения.  
Если для какого-либо измерения нет значения в данном соответствии, фон отображается на всех элементах измерения.  
Если в качестве значения в соответствии находится [ФиксированныйМассив](ФиксированныйМассив.md), интервал фона будет отображён на всех элементах измерения, значения которых входят в массив.  
Если в одной области оказывается несколько интервалов фона, правила их перекрытия определяются их положением в коллекции интервалов фона планировщика (первый элемент коллекции в самом низу, последний – в самом верху).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
