Описание:

Объект этого типа возвращается методами [Выбрать](СправочникМенеджер.Имя_справочника.Выбрать_(CatalogManager.Catalog_name.Select).md) и [ВыбратьИерархически](СправочникМенеджер.Имя_справочника.ВыбратьИерархически_(CatalogManager.Catalog_name.SelectHierarchically).md) у объекта типа [СправочникМенеджер.](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md)<[Имя справочника](СправочникМенеджер.Имя_справочника_(CatalogManager.Catalog_name).md)> и представляет собой специализированный способ перебора элементов справочника. Обход элементов выполняется системой динамически. Это означает, что использование выборки не считывает все элементы сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большие списки справочников и не загружает в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
