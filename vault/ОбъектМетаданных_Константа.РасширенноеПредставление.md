ОбъектМетаданных: Константа (MetadataObject: Constant)

Только чтение.

Описание:

Тип: Строка.   
Содержит строку, которая является названием константы и используется при формировании команды открытия формы константы в командном интерфейсе.

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
