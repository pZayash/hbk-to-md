ФормаКлиентскогоПриложения (ClientApplicationForm)

АвтоНавигационнаяСсылка (AutoURL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет автоматическим формированием навигационной ссылки, возвращаемой формой.  
Если Ложь, метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md) объекта [ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md) возвращает значение свойства НавигационнаяСсылка основной формы.  
Если Истина, метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md) возвращает навигационную ссылку основной формы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ОкноКлиентскогоПриложения](ОкноКлиентскогоПриложения.md), метод [ПолучитьНавигационнуюСсылку](ОкноКлиентскогоПриложения.ПолучитьНавигационнуюСсылку.md)
