Шрифт (Font)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Шрифт](Шрифт_(Font).md).   
Содержит шрифт, которым отображаются надписи в области легенды (подписи к элементам легенды и масштабной линейке).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
