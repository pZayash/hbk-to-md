ФормаКлиентскогоПриложения (ClientApplicationForm)

АвтоПодборПользователейСистемыВзаимодействия (CollaborationSystemUsersAutoComplete)

Доступен, начиная с версии 8.3.15.

Синтаксис:

АвтоПодборПользователейСистемыВзаимодействия(<Текст>, <ДанныеВыбора>, <НазначениеВыбора>, <СтандартнаяОбработка>)

Параметры:

<Текст>

Тип: [Строка](lang__def_String.md).   
Строка текста, введенная в поле ввода выбора пользователя.

<ДанныеВыбора>

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
В качестве значения элемента списка могут использоваться значения следующих типов:

- [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md),
- [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md) - идентификатор пользователя информационной базы,
- Массив - должен содержать значения только следующих типов:
  - [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md),
  - [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md) - идентификатор пользователя информационной базы,
  - [Неопределено](lang__def_Undefined.md) - допустимо только, если [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md) имеет значение [ПолучательСообщения](НазначениеВыбораПользователейСистемыВзаимодействия.ПолучательСообщения_(CollaborationSystemUsersChoicePurpose.MessageRecipient).md).

<НазначениеВыбора>

Тип: [НазначениеВыбораПользователейСистемыВзаимодействия](НазначениеВыбораПользователейСистемыВзаимодействия_(CollaborationSystemUsersChoicePurpose).md).   
Назначение выбора пользователя системы взаимодействия.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак выполнения стандартной обработки.  
Если после вызова обработчика данный параметр содержит значение [Ложь](lang__def_BooleanFalse.md), то обработчик события клиентского управляемого приложения [АвтоПодборПользователейСистемыВзаимодействия](Глобальный_контекст.АвтоПодборПользователейСистемыВзаимодействия_(Global_context.CollaborationSystemUsersAutoComplete).md)не вызывается.

Описание:

Возникает на клиенте при автоподборе пользователя системы взаимодействия (получателя сообщения, участника видеозвонка).  
Данный обработчик не вызывается при добавлении участников в существующую видеоконференцию.  
При вызове обработчика в <ДанныеВыбора> передается сформированный системой список выбора.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.15.
