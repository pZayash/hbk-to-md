ЭлементАплетHTML (HTMLAppletElement)

Атрибуты (Attributes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияАтрибутовHTML](КоллекцияАтрибутовHTML_(HTMLAttributeMap).md).   
Содержит коллекцию атрибутов узла. Коллекция атрибутов доступна только для узла Element.  
Узел Атрибуты

- [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md) - [Неопределено](lang__def_Undefined.md);
- [Комментарий](ТипУзлаDOM.Комментарий_(DOMNodeType.Comment).md) - [Неопределено](lang__def_Undefined.md);
- [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md) - [Неопределено](lang__def_Undefined.md);
- [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md) - [Неопределено](lang__def_Undefined.md);
- [ОпределениеТипаДокумента](ТипУзлаDOM.ОпределениеТипаДокумента_(DOMNodeType.DocumentType).md) - [Неопределено](lang__def_Undefined.md);
- [Элемент](ТипУзлаDOM.Элемент_(DOMNodeType.Element).md) - [КоллекцияАтрибутовHTML](КоллекцияАтрибутовHTML_(HTMLAttributeMap).md);
- [Текст](ТипУзлаDOM.Текст_(DOMNodeType.Text).md) - [Неопределено](lang__def_Undefined.md).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
