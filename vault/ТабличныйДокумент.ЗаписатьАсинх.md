ТабличныйДокумент (SpreadsheetDocument)

ЗаписатьАсинх (WriteAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: В файл

Синтаксис:

ЗаписатьАсинх(<ИмяФайла>, <ТипФайлаТаблицы>, <Вложения>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, в котором сохраняется табличный документ.

<ТипФайлаТаблицы> (необязательный)

Тип: [ТипФайлаТабличногоДокумента](ТипФайлаТабличногоДокумента.md).   
Формат, в котором будет сохранен табличный документ.  
Поддерживаются форматы: ANSITXT, DOCX, MXL, MXL7, ODS, PDF, TXT.  
Значение по умолчанию: [MXL](ТипФайлаТабличногоДокумента.MXL.md).

<Вложения> (необязательный)

Тип: [КоллекцияВложенийPDF](КоллекцияВложенийPDF.md).   
Объект, содержащий вложения PDF-документов.  
Используется только при записи в форматы PDF и PDF/A-3.  
Если табличный документ записывается в формате PDF или PDF/A-3 и указан параметр <Вложения>, то вложения добавляются в документ в формате PDF и PDF/A-3.

Описание варианта метода:

Записывает табличного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

ЗаписатьАсинх(<Поток>, <ТипФайлаТаблицы>, <Вложения>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи табличного документа.

<ТипФайлаТаблицы> (необязательный)

Тип: [ТипФайлаТабличногоДокумента](ТипФайлаТабличногоДокумента.md).   
Формат, в котором будет сохранен табличный документ.  
Поддерживаются форматы: ANSITXT, DOCX, MXL, MXL7, ODS, PDF, TXT.  
Значение по умолчанию: [MXL](ТипФайлаТабличногоДокумента.MXL.md).

<Вложения> (необязательный)

Тип: [КоллекцияВложенийPDF](КоллекцияВложенийPDF.md).   
Объект, содержащий вложения PDF-документов.  
Если табличный документ записывается в формате PDF или PDF/A-3 и указан параметр <Вложения>, то вложения добавляются в документ в формате PDF и PDF/A-3.

Описание варианта метода:

Записывает табличного документа в поток.  
При записи в поток допускается использование только форматов: ANSITXT, DOCX, XLSX, MXL, MXL7, ODS, PDF, TXT.  
Внимание! Данный вариант метода не поддерживается в веб-клиенте.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания:

- Истина - если запись была инициирована с помощью расширения браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
- Ложь - если запись была инициирована с помощью расширения браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
- Неопределено - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

Описание:

Асинхронная функция. Записывает таблицу в файл.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

При работе на сервере, веб-клиенте или через внешнее соединение:

- нет возможности сохранять в формате XLS95;
- нет возможности сохранять в формате MXL7 табличные документы, содержащие картинки. При попытке сохранить такой документ в формате MXL7 будет сгенерировано исключение.

Не поддерживается сохранение в [PDF](ТипФайлаТабличногоДокумента.PDF.md) картинок типа Метафайл.  
При работе на веб-клиенте вызов метода выполняет обращение к серверу.  
При работе на мобильной платформе возможно сохранение только в форматах MXL и PDF.  
При выводе в формат PDF шрифты, которые нельзя внедрять, а также шрифты, не являющиеся TrueType или OpenType, будут заменены на шрифт Arial. Если данный шрифт не обнаружен, то будет использован шрифт Helvetica.
