РежимСовместимостиИнтерфейса (InterfaceCompatibilityMode)

Версия8\_2РазрешитьТакси (Version8\_2EnableTaxi)

Доступен, начиная с версии 8.3.3.

Описание:

Клиентское приложение по умолчанию использует интерфейс [Версия8\_2](ВариантИнтерфейсаКлиентскогоПриложения.Версия8_2_(ClientApplicationInterfaceVariant.Version8_2).md).  
Возможно изменение режима через [НастройкиКлиентскогоПриложения](НастройкиКлиентскогоПриложения_(ClientSettings).md) или параметр строки запуска.

Использование в версии:

Доступен, начиная с версии 8.3.3.
