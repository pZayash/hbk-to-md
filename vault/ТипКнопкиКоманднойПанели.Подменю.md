ТипКнопкиКоманднойПанели (CommandBarButtonType)

Подменю (Popup)

Доступен, начиная с версии 8.0.

Описание:

Кнопка данного типа аналогична кнопке типа [Действие](ТипКнопкиКоманднойПанели.Действие.md), но при нажатии на нее будет открыто связанное с данной кнопкой подменю, а не выполнено действие.

Использование в версии:

Доступен, начиная с версии 8.0.
