ПолитикаПаролейПользователей (UserPasswordPolicy)

МинимальныйСрокДействияПаролей (PasswordMinEffectivePeriod)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Численное значение минимального срока действия паролей в секундах. Минимальный срок действия паролей не может превышать максимальный.  
  
Значение должно быть в интервале от 0 до 4294967295.  
  
Значение по умолчанию: 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
