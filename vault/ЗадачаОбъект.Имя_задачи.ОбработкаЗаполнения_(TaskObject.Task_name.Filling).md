# ЗадачаОбъект.<Имя задачи>.ОбработкаЗаполнения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаОбъект.<Имя задачи>](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md) › [События](ЗадачаОбъект.Имя_задачи__События.md)

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ОбработкаЗаполнения (Filling)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаЗаполнения(<ДанныеЗаполнения>, <ТекстЗаполнения>, <СтандартнаяОбработка>)

Параметры:

<ДанныеЗаполнения>

Тип: Произвольный.   
Значение, которое используется как основание для заполнения.  
Если обработчик вызван при выполнении метода Заполнить, параметр равен параметру <ДанныеЗаполнения> метода Заполнить.  
При обработке заполнения во время ввода на основании параметр имеет значение основания.  
При интерактивном вводе нового из формы списка параметр является структурой, элементы которой соответствуют тем параметрам отбора формы списка, у которых способ сравнения [Равно](ВидСравнения.Равно_(ComparisonType.Equal).md) или [ВСписке](ВидСравнения.ВСписке_(ComparisonType.InList).md) с единственным элементом списка. Если таких элементов отбор списка не содержит, данный параметр равен [Неопределено](lang__def_Undefined.md). Если параметр <СтандартнаяОбработка> по окончании работы обработчика равен [Истина](lang__def_BooleanTrue.md) (установлен по умолчанию), система автоматически произведет заполнение одноименными элементами структуры.

<ТекстЗаполнения>

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если у объекта основное представление - [ВВидеНаименования](ОсновноеПредставлениеЗадачи.ВВидеНаименования_(TaskMainPresentation.AsDescription).md) и реквизит [Наименование](ЗадачаОбъект.Имя_задачи.Наименование_(TaskObject.Task_name.Description).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_Задача.ВводПоСтроке_(MetadataObject_Task.InputByString).md), то значение присваивается реквизиту [Наименование](ЗадачаОбъект.Имя_задачи.Наименование_(TaskObject.Task_name.Description).md).  
Если основное представление [ВВидеНомера](ОсновноеПредставлениеЗадачи.ВВидеНомера_(TaskMainPresentation.AsNumber).md) и реквизит [Номер](ЗадачаОбъект.Имя_задачи.Номер_(TaskObject.Task_name.Number).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_Задача.ВводПоСтроке_(MetadataObject_Task.InputByString).md) и не установлен признак [Автонумерация](ОбъектМетаданных_Задача.Автонумерация_(MetadataObject_Task.Autonumbering).md), то значение присваивается реквизиту [Номер](ЗадачаОбъект.Имя_задачи.Номер_(TaskObject.Task_name.Number).md).  
В противном случае присваивание не выполняется.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при вводе задачи на основании, а также при выполнении метода [Заполнить](ЗадачаОбъект.Имя_задачи.Заполнить_(TaskObject.Task_name.Fill).md), при вводе на основании, а также при интерактивном вводе нового. В процедуре-обработчике этого события должен быть описан сам алгоритм заполнения реквизитов задачи на основании переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При копировании (как интерактивном, так и средствами встроенного языка), данный обработчик не вызывается.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
