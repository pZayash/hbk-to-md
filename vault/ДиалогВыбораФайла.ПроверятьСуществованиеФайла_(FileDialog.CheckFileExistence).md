ДиалогВыбораФайла (FileDialog)

ПроверятьСуществованиеФайла (CheckFileExistence)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак проверки существования выбираемого файла.  
[Истина](lang__def_BooleanTrue.md) - существование проверяется.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Примечание:

Не используется в режиме выбора каталога.  
Не поддерживается в мобильном приложении для Windows.   
В веб-клиенте данное свойство игнорируется.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.18.
