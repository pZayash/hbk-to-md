КоллекцияПолейСводнойДиаграммы (PivotChartFieldCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: <Имя поля>, [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы_(PivotChartFieldCollection).md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы_(PivotChartField).md).   
Добавляемое поле.

Возвращаемое значение:

Тип: [КоллекцияПолейСводнойДиаграммы](КоллекцияПолейСводнойДиаграммы_(PivotChartFieldCollection).md), [ПолеСводнойДиаграммы](ПолеСводнойДиаграммы_(PivotChartField).md).   

Описание:

Добавляет поле или другую коллекцию в конец коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
