ПланОбменаСписок.<Имя плана обмена> (ExchangePlanList.<Exchange plan name>)

Отбор (Filter)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Отбор](Отбор_(Filter).md).   
Позволяет установить отбор по реквизитам узла. Имена элементов отбора совпадают с именами колонок списка плана обмена, а также дополняются критериями отбора, затрагивающими план обмена.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
