# ОтчетыМенеджер

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Отчеты](Отчеты.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ОтчетМенеджер.\<Имя отчета\>](ОтчетМенеджер.Имя_отчета.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры отчетов.  
Возможно обращение к менеджеру отчета посредством оператора [...]. В качестве аргумента передается имя отчета, как оно задано в конфигурации.  

Свойства:

[\<Имя отчета\> (\<Report name\>)](ОтчетыМенеджер.Имя_отчета.md)  

Описание:

Предназначен для управления отчетами и предоставляет доступ к значениям типа [ОтчетМенеджер.](ОтчетМенеджер.Имя_отчета.md)<[Имя отчета](ОтчетМенеджер.Имя_отчета.md)>. Доступ к объекту осуществляется через свойство глобального контекста [Отчеты](Глобальный_контекст.Отчеты.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Отчеты](Глобальный_контекст.Отчеты.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
