РегистрНакопленияЗапись.<Имя регистра накопления> (AccumulationRegisterRecord.<Accumulation register name>)

Период (Period)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату и время записи регистра накопления. Совместно с [Регистратор](РегистрНакопленияЗапись.Имя_регистра_накопления.Регистратор_(AccumulationRegisterRecord.Accumulation_register_name.Recorder).md) и [НомерСтроки](РегистрНакопленияЗапись.Имя_регистра_накопления.НомерСтроки_(AccumulationRegisterRecord.Accumulation_register_name.LineNumber).md) определяет последовательность движений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
