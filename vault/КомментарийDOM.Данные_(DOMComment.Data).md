КомментарийDOM (DOMComment)

Данные (Data)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит данные узла.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

При установке значения свойства может возникнуть исключение в случае, если данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.1.
