УровеньИзоляцииТранзакций (TransactionsIsolationLevel)

ЧтениеЗафиксированных (ReadCommitted)

Доступен, начиная с версии 8.3.5.

Описание:

Уровень изоляции транзакций внешнего источника данных, при котором операция чтения возвращает данные, зафиксированные в других транзакциях на момент чтения данных, операция записи на измененных в других транзакциях данных перезаписывает данные (может быть обеспечено поведение уровня [ПовторяемоеЧтение](УровеньИзоляцииТранзакций.ПовторяемоеЧтение_(TransactionsIsolationLevel.RepeatableRead).md) или выше, зависит от внешнего источника данных).

Использование в версии:

Доступен, начиная с версии 8.3.5.
