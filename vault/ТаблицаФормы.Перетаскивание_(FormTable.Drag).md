ТаблицаФормы (FormTable)

Перетаскивание (Drag)

Доступен, начиная с версии 8.2.

Синтаксис:

Перетаскивание(<ПараметрыПеретаскивания>, <СтандартнаяОбработка>, <Строка>, <Поле>)

Параметры:

<ПараметрыПеретаскивания>

Тип: [ПараметрыПеретаскивания](ПараметрыПеретаскивания_(DragParameters).md).   
Содержит перетаскиваемое значение, тип действия и возможные действия при перетаскивании.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<Строка>

Содержит порядковый номер строки или ссылку на текущий объект.  
Если таблица связана с [ДанныеФормыКоллекция](ДанныеФормыКоллекция_(FormDataCollection).md) или [ДанныеФормыДерево](ДанныеФормыДерево_(FormDataTree).md), то параметр содержит идентификатор строки над которой находится курсор.   
Для таблиц, связанных с [ДинамическийСписок](ДинамическийСписок_(DynamicList).md), содержит ссылку на текущий объект строки таблицы.

<Поле>

Поле, с которым связана данная колонка таблицы, над которой находится объект.

Описание:

Возникает при окончании перетаскивания в поле - приемнике данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

В веб-клиенте, если установлен способ перетаскивания файлов [КакФайл](СпособПеретаскиванияФайлов.КакФайл_(FileDragMode.AsFile).md), то перетаскивание файлов игнорируется, и обработчик данного события не возникает.

Использование в версии:

Доступен, начиная с версии 8.2.
