ПользовательСистемыВзаимодействия (CollaborationSystemUser)

ТипВнешнейСистемы (ExternalSystemType)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Если не пустая строка, то содержит тип внешней системы, которой принадлежит данный пользователь.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
