ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ПричинаОтключения (TurnedOffReason)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [ПричинаОтключенияКопииБазыДанных](ПричинаОтключенияКопииБазыДанных.md), Неопределено.   
Если копия базы данных может выполнять запросы, то данное свойство имеет значение Неопределено. В противном случае - причина отключения копии.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
