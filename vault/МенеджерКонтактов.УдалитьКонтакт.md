МенеджерКонтактов (ContactManager)

УдалитьКонтакт (DeleteContact)

Доступен, начиная с версии 8.3.4.

Синтаксис:

УдалитьКонтакт(<Контакт>)

Параметры:

<Контакт> (обязательный)

Тип: [ЛокальныйКлючКонтакта](ЛокальныйКлючКонтакта.md).   
Локальный ключ контакта.

Описание:

Удаляет контакт по локальному ключу.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИзменениеКонтакта](МенеджерКонтактов.ПоддерживаетсяИзменениеКонтакта.md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКонтактов](МенеджерКонтактов.md), метод [ПоддерживаетсяИзменениеКонтакта](МенеджерКонтактов.ПоддерживаетсяИзменениеКонтакта.md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
