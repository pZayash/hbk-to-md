# ПолеФормы.ОтображатьВПодвале

ПолеФормы (FormField)

ОтображатьВПодвале (ShowInFooter)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак представления данного поля в подвале таблицы. Данное свойство имеет смысл только для полей, расположенных в таблице (представляющих собой колонки).  
[Истина](lang__def_BooleanTrue.md) - отображается.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
