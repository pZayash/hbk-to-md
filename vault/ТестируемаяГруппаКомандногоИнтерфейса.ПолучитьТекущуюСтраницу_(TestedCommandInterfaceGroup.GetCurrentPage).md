ТестируемаяГруппаКомандногоИнтерфейса (TestedCommandInterfaceGroup)

ПолучитьТекущуюСтраницу (GetCurrentPage)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПолучитьТекущуюСтраницу()

Возвращаемое значение:

Тип: [ТестируемаяГруппаФормы](ТестируемаяГруппаФормы_(TestedFormGroup).md).   

Описание:

Для группы вида [Страницы](ВидГруппыФормы.Страницы_(FormGroupType.Pages).md) возвращает объект, описывающий текущую активную страницу.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
