БиблиотекаМакетовОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateLib)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя макета оформления.

Возвращаемое значение:

Тип: [ЭлементБиблиотекиМакетовОформленияКомпоновкиДанных](ЭлементБиблиотекиМакетовОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplateLibItem).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента макетов оформления компоновки данных по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
