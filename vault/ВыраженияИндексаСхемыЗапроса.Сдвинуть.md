ВыраженияИндексаСхемыЗапроса (QuerySchemaIndexExpressions)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.5.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: Число, [ВыражениеИндексаСхемыЗапроса](ВыражениеИндексаСхемыЗапроса.md).   
Индекс элемента или сам элемент, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: Число.   
Количество элементов, на которое нужно сдвинуть указанный элемент. Положительное смещение обозначает сдвиг в конец, отрицательное - в начало коллекции.

Описание:

Сдвигает элемент в коллекции на указанное число позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме совместимости [Версия8\_3\_10](РежимСовместимости.Версия8_3_10.md) и ниже перемещает элемент коллекции в указанную позицию.

Использование в версии:

Доступен, начиная с версии 8.3.5.
