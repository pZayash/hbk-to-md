Глобальный контекст (Global context)

ОткрытьЗначениеАсинх (OpenValueAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ОткрытьЗначениеАсинх(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение любого типа, который может быть показан системой в диалоге.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Неопределено.

Описание:

Асинхронная функция. Показывает в диалоге содержимое указанного значения. Вид диалога зависит от типа переданного значения.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
