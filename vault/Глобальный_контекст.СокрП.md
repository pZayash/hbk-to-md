Глобальный контекст (Global context)

СокрП (TrimR)

Доступен, начиная с версии 8.0.

Синтаксис:

СокрП(<Строка>)

Параметры:

<Строка> (обязательный)

Тип: Строка.   
Исходная строка.

Возвращаемое значение:

Тип: Строка.   
Строка, полученная в результате отсечения пробелов .

Описание:

Отсекает незначащие символы, стоящие справа от последнего значащего символа в строке.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Примечание:

К незначащим символам относятся символы "пробел", "неразрывный пробел" ([НПП](Символы.НПП.md)), "табуляция" (горизонтальная [Таб](Символы.Таб.md) и вертикальная [ВТаб](Символы.ВТаб.md)), "возврат каретки" ([ВК](Символы.ВК.md)), "перевод строки" ([ПС](Символы.ПС.md)), "перевод формы (страницы)" ([ПФ](Символы.ПФ.md)).

Пример:

|  |
| --- |
| НаименованиеДляПечати = СокрП(Товар.Наименование); |

Использование в версии:

Доступен, начиная с версии 8.0.
