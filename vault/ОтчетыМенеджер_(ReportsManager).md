# ОтчетыМенеджер

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Отчеты](Отчеты.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ОтчетМенеджер.\<Имя отчета\>](ОтчетМенеджер.Имя_отчета_(ReportManager.Report_name).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры отчетов.  
Возможно обращение к менеджеру отчета посредством оператора [...]. В качестве аргумента передается имя отчета, как оно задано в конфигурации.  

Свойства:

[\<Имя отчета\> (\<Report name\>)](ОтчетыМенеджер.Имя_отчета_(ReportsManager.Report_name).md)  

Описание:

Предназначен для управления отчетами и предоставляет доступ к значениям типа [ОтчетМенеджер.](ОтчетМенеджер.Имя_отчета_(ReportManager.Report_name).md)<[Имя отчета](ОтчетМенеджер.Имя_отчета_(ReportManager.Report_name).md)>. Доступ к объекту осуществляется через свойство глобального контекста [Отчеты](Глобальный_контекст.Отчеты_(Global_context.Reports).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Отчеты](Глобальный_контекст.Отчеты_(Global_context.Reports).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
