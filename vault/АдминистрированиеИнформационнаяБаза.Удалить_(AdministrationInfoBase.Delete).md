# АдминистрированиеИнформационнаяБаза.Удалить

АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

Удалить (Delete)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Удалить(<РежимУдаленияИнформационнойБазы>)

Параметры:

<РежимУдаленияИнформационнойБазы> (обязательный)

Тип: [АдминистрированиеРежимУдаленияИнформационнойБазы](АдминистрированиеРежимУдаленияИнформационнойБазы_(AdministrationInfoBaseDeletionMode).md).   
Режим удаления информационной базы.  
Значение по умолчанию: [НеВыполнятьДействийСБазойДанных](АдминистрированиеРежимУдаленияИнформационнойБазы.НеВыполнятьДействийСБазойДанных_(AdministrationInfoBaseDeletionMode.DontPerformActionsWithDatabase).md).

Возвращаемое значение:

Тип: [АдминистрированиеРежимУдаленияИнформационнойБазы](АдминистрированиеРежимУдаленияИнформационнойБазы_(AdministrationInfoBaseDeletionMode).md).   

Описание:

Удаляет информационную базу. Для выполнения требуются административные права в удаляемой информационной базе на сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
