МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УстановитьАдресЭлектроннойПочтыАбонента (SetSubscriberEmail)

Доступен, начиная с версии 8.3.25.

Синтаксис:

УстановитьАдресЭлектроннойПочтыАбонента(<АдресЭлектроннойПочты>)

Параметры:

<АдресЭлектроннойПочты> (обязательный)

Тип: [Строка](lang__def_String.md).   
Устанавливаемый адрес электронной почты абонента.

Описание:

Изменяет адрес электронной почты абонента.  
Если текущий пользователь не является администратором абонента, то генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
