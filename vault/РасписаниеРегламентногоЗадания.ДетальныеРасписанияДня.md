РасписаниеРегламентногоЗадания (JobSchedule)

ДетальныеРасписанияДня (DetailedDailySchedules)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Массив](Массив.md).   
Массив детальных расписаний на день. Расписание может содержать только значения следующих свойств: [ВремяНачала](РасписаниеРегламентногоЗадания.ВремяНачала.md), [ВремяКонца](РасписаниеРегламентногоЗадания.ВремяКонца.md), [ВремяЗавершения](РасписаниеРегламентногоЗадания.ВремяЗавершения.md), [ИнтервалЗавершения](РасписаниеРегламентногоЗадания.ИнтервалЗавершения.md), [ПериодПовтораВТечениеДня](РасписаниеРегламентногоЗадания.ПериодПовтораВТечениеДня.md), [ПаузаПовтора](РасписаниеРегламентногоЗадания.ПаузаПовтора.md). Если задано свойство [ДетальныеРасписанияДня](РасписаниеРегламентногоЗадания.ДетальныеРасписанияДня.md), то основное расписание не должно содержать вышеперечисленные свойства.  
Свойство возвращает копию массива, поэтому просто его модификация не имеет смысла. Измененный массив должен быть обратно присвоен свойству для применения:  

|  |
| --- |
| ДетальноеРасписание = Расписание.ДетальныеРасписанияДня; ДетальноеРасписание.Добавить(Новый РасписаниеРегламентногоЗадания()); Расписание.ДетальныеРасписанияДня = ДетальноеРасписание; |

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.

Описание изменено в версии 8.3.17.
