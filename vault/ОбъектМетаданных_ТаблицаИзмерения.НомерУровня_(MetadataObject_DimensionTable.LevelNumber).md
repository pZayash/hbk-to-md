ОбъектМетаданных: ТаблицаИзмерения (MetadataObject: DimensionTable)

НомерУровня (LevelNumber)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер уровня в иерархии таблицы измерения куба во внешнем источнике данных.  
Для иерархических таблиц измерений и таблиц измерений, представляющих измерения во внешнем источнике данных, номер уровня - 0.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
