ВыраженияИтоговСхемыЗапроса (QuerySchemaTotalExpressions)

Добавить (Add)

Доступен, начиная с версии 8.3.5.

Вариант синтаксиса: По колонке или выражению

Синтаксис:

Добавить(<КолонкаИлиВыражение>)

Параметры:

<КолонкаИлиВыражение> (обязательный)

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса_(QuerySchemaColumn).md), [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md), [Строка](lang__def_String.md).   
Колонка или выражение схемы запроса. Если передана строка, то по переданной строке сначала выполняется поиск колонки. Если не найдена, по указанной строке будет создано выражение.

Описание варианта метода:

Добавляется выражение итогов по колонке.

Вариант синтаксиса: По выражению

Синтаксис:

Добавить(<Выражение>, <Колонка>)

Параметры:

<Выражение> (обязательный)

Тип: [ВыражениеСхемыЗапроса](ВыражениеСхемыЗапроса_(QuerySchemaExpression).md), [Строка](lang__def_String.md).   
Выражение, при помощи которого нужно рассчитывать итог.   
В случае ошибки в выражении будет сгенерировано исключение.

<Колонка> (обязательный)

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса_(QuerySchemaColumn).md), [Строка](lang__def_String.md).   
Колонка для выражения итога. Если указана строка, то выполняется поиск колонки с соответствующим псевдонимом.

Описание варианта метода:

Добавляется выражение итогов по выражению.

Описание:

Добавляет новый элемент коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
