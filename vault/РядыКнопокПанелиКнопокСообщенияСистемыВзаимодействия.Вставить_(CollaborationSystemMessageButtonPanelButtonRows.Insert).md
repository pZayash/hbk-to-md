РядыКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRows)

Вставить (Insert)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить ряд кнопок панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonRow).md).   

Описание:

Вставляет ряд кнопок панели кнопок сообщения системы взаимодействия в заданную позицию и возвращает его.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
