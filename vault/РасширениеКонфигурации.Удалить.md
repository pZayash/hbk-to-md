РасширениеКонфигурации (ConfigurationExtension)

Удалить (Delete)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Удалить()

Описание:

Удаляет расширение из информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

При попытке удалить незаписанное расширение будет получена ошибка "Нельзя удалить незаписанное расширение".

Использование в версии:

Доступен, начиная с версии 8.3.6.
