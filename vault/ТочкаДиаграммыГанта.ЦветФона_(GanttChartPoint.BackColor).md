ЦветФона (BackColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет фона области вывода точки. Допустимо значение [АвтоЦвет](ВидЦвета.АвтоЦвет_(ColorType.AutoColor).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
