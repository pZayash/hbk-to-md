ЭлементыГрафическойСхемы (GraphicalSchemaItems)

Найти (Find)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: По точке бизнес-процесса

Синтаксис:

Найти(<ТочкаМаршрутаБизнесПроцесса>)

Параметры:

<ТочкаМаршрутаБизнесПроцесса> (необязательный)

Тип: [ТочкаМаршрутаБизнесПроцессаСсылка.\<Имя бизнес-процесса\>](ТочкаМаршрутаБизнесПроцессаСсылка.Имя_бизнес-процесса.md).   
Точка маршрута бизнес-процесса, для которой ищется элемент графической схемы.

Вариант синтаксиса: По имени

Синтаксис:

Найти(<ИмяЭлементаГрафическойСхемы>)

Параметры:

<ИмяЭлементаГрафическойСхемы> (необязательный)

Тип: Строка.   
Имя элемента графической схемы, которую требуется найти.

Возвращаемое значение:

Тип: [ЭлементГрафическойСхемыСоединительнаяЛиния](ЭлементГрафическойСхемыСоединительнаяЛиния.md), [ЭлементГрафическойСхемыДекоративнаяЛиния](ЭлементГрафическойСхемыДекоративнаяЛиния.md), [ЭлементГрафическойСхемыДекорация](ЭлементГрафическойСхемыДекорация.md), [ЭлементГрафическойСхемыДействие](ЭлементГрафическойСхемыДействие.md), [ЭлементГрафическойСхемыРазделение](ЭлементГрафическойСхемыРазделение.md), [ЭлементГрафическойСхемыУсловие](ЭлементГрафическойСхемыУсловие.md), [ЭлементГрафическойСхемыЗавершение](ЭлементГрафическойСхемыЗавершение.md), [ЭлементГрафическойСхемыСтарт](ЭлементГрафическойСхемыСтарт.md), [ЭлементГрафическойСхемыСлияние](ЭлементГрафическойСхемыСлияние.md), [ЭлементГрафическойСхемыВложенныйБизнесПроцесс](ЭлементГрафическойСхемыВложенныйБизнесПроцесс.md), [ЭлементГрафическойСхемыОбработка](ЭлементГрафическойСхемыОбработка.md), [ЭлементГрафическойСхемыВыборВарианта](ЭлементГрафическойСхемыВыборВарианта.md), Неопределено.   
Элемент графической схемы. Если элемент не найден, то возвращается значение Неопределено.

Описание:

Осуществляет поиск элемента графической схемы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
