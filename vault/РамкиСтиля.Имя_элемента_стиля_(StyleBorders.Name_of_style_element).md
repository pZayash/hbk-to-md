РамкиСтиля (StyleBorders)

<Имя элемента стиля> (<Name of style element>)

Доступен, начиная с версии 8.0.

Описание:

Тип: [Рамка](Рамка_(Border).md).   
Набор значений содержит рамки элементов стиля, созданные при конфигурировании прикладной задачи. Доступ к рамке элемента стиля осуществляется по имени. Имена значений совпадают с именами элементов стиля типа [Рамка](ВидЭлементаСтиля.Рамка_(StyleElementType.Border).md), как они заданы в Конфигураторе.

Использование в версии:

Доступен, начиная с версии 8.0.
