ВариантГраницыИнтервала (IntervalBoundVariant)

Год (Year)

Доступен, начиная с версии 8.0.

Описание:

Начало интервала задается датой начала года рабочей даты, конец интервала - датой конца года рабочей даты.

Использование в версии:

Доступен, начиная с версии 8.0.
