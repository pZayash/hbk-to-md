Описание:

Определяет набор статусов сообщений, обозначаемых пиктограммами при выводе сообщений в окно сообщений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Сообщить](Глобальный_контекст.Сообщить_(Global_context.Message).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
