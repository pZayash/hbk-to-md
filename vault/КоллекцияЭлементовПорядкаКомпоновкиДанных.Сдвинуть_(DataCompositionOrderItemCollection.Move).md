КоллекцияЭлементовПорядкаКомпоновкиДанных (DataCompositionOrderItemCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.1.

Синтаксис:

Сдвинуть(<Элемент>, <Смещение>)

Параметры:

<Элемент> (обязательный)

Тип: [АвтоЭлементПорядкаКомпоновкиДанных](АвтоЭлементПорядкаКомпоновкиДанных_(DataCompositionAutoOrderItem).md), [ЭлементПорядкаКомпоновкиДанных](ЭлементПорядкаКомпоновкиДанных_(DataCompositionOrderItem).md).   
Перемещаемый элемент.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Смещение элемента в коллекции.

Описание:

Перемещает элемент в коллекции.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
