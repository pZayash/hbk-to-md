ФиксированноеСоответствие (FixedMap)

Получить (Get)

Доступен, начиная с версии 8.2.

Синтаксис:

Получить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ значения.

Возвращаемое значение:

Тип: Произвольный.   
[Неопределено](lang__def_Undefined.md) - ключ не найден.

Описание:

Получает значение по ключу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
