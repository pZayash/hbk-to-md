СвязиДендрограммы (DendrogramLinkCollection)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Связь>)

Параметры:

<Связь> (обязательный)

Тип: [СвязьДендрограммы](СвязьДендрограммы_(DendrogramLink).md).   
Искомая связь.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента в коллекции. Если не найдено, то возвращается -1.

Описание:

Получает индекс указанного элемента в коллекции элементов.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
