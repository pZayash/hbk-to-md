ИзбранноеРаботыПользователя (UserWorkFavorites)

Вставить (Insert)

Доступен, начиная с версии 8.2.

Синтаксис:

Вставить(<Индекс>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Элемент> (необязательный)

Тип: [ЭлементИзбранногоРаботыПользователя](ЭлементИзбранногоРаботыПользователя_(UserWorkFavoritesItem).md).   
Добавляемый элемент избранного.

Описание:

Добавляет элемент избранного в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
