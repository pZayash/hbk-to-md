ПараметрыПривязки (FingerprintParameters)

Доступен, начиная с версии 8.3.20.

Использование:

Чтение и запись.

Описание:

Тип: [ПараметрыПривязкиККлючуПолученияЛицензий](ПараметрыПривязкиККлючуПолученияЛицензий_(LicenseAcquisitionKeyFingerprintParameters).md), [ПараметрыПривязкиККомпьютеруПолученияЛицензий](ПараметрыПривязкиККомпьютеруПолученияЛицензий_(LicenseAcquisitionComputerFingerprintParameters).md).

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
