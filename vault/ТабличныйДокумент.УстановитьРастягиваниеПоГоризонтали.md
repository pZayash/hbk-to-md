ТабличныйДокумент (SpreadsheetDocument)

УстановитьРастягиваниеПоГоризонтали (SetHorizontalStretch)

Доступен, начиная с версии 8.3.13.

Синтаксис:

УстановитьРастягиваниеПоГоризонтали(<Растягивать>)

Параметры:

<Растягивать> (обязательный)

Тип: Булево.   
Значение, которое будет использовано в свойстве [РастягиватьПоГоризонтали](ОбластьЯчеекТабличногоДокумента.РастягиватьПоГоризонтали.md) для всех колонок.   
Если передано значение Истина, то дополнительно для всех колонок в свойство [ВесовойКоэффициентШирины](ОбластьЯчеекТабличногоДокумента.ВесовойКоэффициентШирины.md) устанавливается текущая ширина ячейки.

Описание:

Для всех колонок табличного документа изменяет значение свойства [РастягиватьПоГоризонтали](ОбластьЯчеекТабличногоДокумента.РастягиватьПоГоризонтали.md).

Доступность:

Сервер, толстый клиент.

См. также:

[ТабличныйДокумент](ТабличныйДокумент.md), метод [ПолучитьКомпактныйДокумент](ТабличныйДокумент.ПолучитьКомпактныйДокумент.md)  

Использование в версии:

Доступен, начиная с версии 8.3.13.
