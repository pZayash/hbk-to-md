ДиалогРасписанияРегламентногоЗадания (ScheduledJobDialog)

Показать (Show)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога со следующими параметрами:

- <Расписание> - [Расписание](ДиалогРасписанияРегламентногоЗадания.Расписание.md) - диалог закрыли по кнопке "OK"; Неопределено - в противном случае;
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершению никакая процедура вызвана не будет.

Описание:

Открывает немодальный диалог расписания.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
