# ДиаграммаГанта.ПолучитьЗначение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Диаграмма Ганта](Диаграмма_Ганта.md) › [ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md) › [Методы](ДиаграммаГанта__Методы.md)

ДиаграммаГанта (GanttChart)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьЗначение(<Точка>, <Серия>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md).   
Серия диаграммы Ганта.

Возвращаемое значение:

Тип: [ЗначениеДиаграммыГанта](ЗначениеДиаграммыГанта_(GanttChartValue).md).   
Значение диаграммы Ганта, соответствующее заданной точке и серии.

Описание:

Получает значение, соответствующее заданной точке и серии диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Значение = ДиаграммаГанта.ПолучитьЗначение(Точка, Серия); |

Использование в версии:

Доступен, начиная с версии 8.0.
