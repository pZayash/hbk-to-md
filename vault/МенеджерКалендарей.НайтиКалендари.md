МенеджерКалендарей (CalendarsManager)

НайтиКалендари (FindCalendars)

Доступен, начиная с версии 8.3.4.

Синтаксис:

НайтиКалендари(<Отбор>)

Параметры:

<Отбор> (необязательный)

Тип: [ОтборКомпоновкиДанных](ОтборКомпоновкиДанных.md), Строка.   
Задает условия отбора календарей.  
Отбор может быть установлен по следующим полям:

- Имя (Name),
- Цвет (Color),
- УчетнаяЗапись (Account) – для отбора по объекту типа [УчетнаяЗаписьКалендарей](УчетнаяЗаписьКалендарей.md),
- ИдентификаторУчетнойЗаписи (AccountID),
- ИсточникУчетнойЗаписи (AccountSource).

Возвращаемое значение:

Тип: [Массив](Массив.md).   
Массив [ЛокальныйКлючКалендаря](ЛокальныйКлючКалендаря.md) с заданным именем.   
Если имя не задано, возвращает ключи всех календарей.

Описание:

Осуществляет поиск календарей.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Примечание:

При использовании строки в качестве параметра, в строке следует указать имя календаря.

Пример:

|  |

| Фильтр = новый ОтборКомпоновкиДанных(); элементОтбора = Фильтр.Элементы                     .Добавить(Тип("ЭлементОтбораКомпоновкиДанных")); элементОтбора.ЛевоеЗначение =      Новый ПолеКомпоновкиДанных("ИдентификаторУчетнойЗаписи"); элементОтбора.ВидСравнения = ВидСравненияКомпоновкиДанных.Равно; элементОтбора.ПравоеЗначение = ИдентификаторУчетнойЗаписи;   Календари = Менеджер.НайтиКалендари(Фильтр);   Для Каждого КлючКалендаря Из Календари Цикл     Календарь = Менеджер.ПолучитьКалендарь(КлючКалендаря); КонецЦикла; |
