Описание:

Представляет собой индекс таблицы базы данных. Используется для получения свойств индекса. Функции, применяемые в выражении и фильтре индекса:

- CTOD(<Строка>) - Дата. Осуществляет преобразование значение исходной строки <Строка> в формате ММ/ДД/ГГ в дату;
- STOD(<Строка>) - Дата. Осуществляет преобразование значение исходной строки <Строка> в формате ГГГГММДД в дату;
- DTOC(<Дата>) - Строка. Осуществляет преобразование даты <Дата> в строку в формате ММ/ДД/ГГ;
- DTOS(<Дата>) - Строка. Осуществляет преобразование даты <Дата> в строку в формате ГГГГММДД;
- VAL(<Строка>) - Число. Возвращает результат преобразования строки <Строка> в число;
- STR(<Число>, <Длина>, <Точность>) - Строка. Осуществляет преобразование числа <Число> в строковое представление. <Длина> - количество символов в возвращаемой строке, включая десятичную точку, <Точность> - желаемое количество знаков после десятичной точки;
- DATE() - Дата. Возвращает системную дату;
- TIME() - Строка. Возвращает системное время в формате ЧЧ:ММ:СС;
- DEL() - Строка. Возвращает "\*" (звездочка), если запись помечена на удаление, "" - иначе;
- DELETED() - Булево. Возвращает Истина, если запись помечена на удаление, Ложь - иначе;
- IIF(<Выражение>, <Значение1>, <Значение2>) - возвращает <Значение1>, если значение вычисленного выражения <Выражение> равно Истина, и <Значение2> в противном случае;
- LTRIM(<Строка>) - Строка. Возвращает исходную строку <Строка> без ведущих пробелов;
- TRIM(<Строка>) - Строка. Возвращает исходную строку <Строка> без заключительных пробелов;
- UPPER(<Строка>) - Строка. Возвращает строку <Строка>, символы которой преобразованы к верхнему регистру;
- SUBSTR(<Строка>, <НачПоз>, <Количество>) - Строка. Возвращает подстроку, состоящую из <Количество> символов от начальной позиции исходной строки <Строка>. <НачПоз> - начальная позиция, <Количество> - количество символов;
- MONTH(<Дата>) - Число. Вычисляет номер месяца в году по исходной дате <Дата>;
- YEAR(<Дата>) - Число. Вычисляет номер года по исходной дате <Дата>;
- RECCOUNT() - Число. Возвращает общее количество записей в таблице базы данных;
- RECNO() - Число. Возвращает номер текущей записи.

Доступность:

Сервер, толстый клиент, внешнее соединение.
