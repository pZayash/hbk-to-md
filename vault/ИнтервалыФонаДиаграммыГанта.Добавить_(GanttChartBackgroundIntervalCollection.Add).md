ИнтервалыФонаДиаграммыГанта (GanttChartBackgroundIntervalCollection)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Начало>, <Конец>)

Параметры:

<Начало> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Начало интервала.

<Конец> (обязательный)

Тип: [Дата](lang__def_Date.md).   
Конец интервала.

Возвращаемое значение:

Тип: [ИнтервалФонаДиаграммыГанта](ИнтервалФонаДиаграммыГанта_(GanttChartBackgroundInterval).md).   

Описание:

Добавляет интервал фона.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
