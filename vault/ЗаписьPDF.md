Описание:

Предназначен для организации записи документов PDF.  
В веб-клиенте требуется установка расширения для работы с криптографией.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
