ОтборОбсужденийСистемыВзаимодействия (CollaborationSystemConversationsFilter)

КонтекстноеОбсуждение (ContextConversation)

Доступен, начиная с версии 8.3.10.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает вид запрашиваемых обсуждений.  
По умолчанию - [Ложь](lang__def_BooleanFalse.md) (обсуждение не привязано к объекту информационной базы).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
