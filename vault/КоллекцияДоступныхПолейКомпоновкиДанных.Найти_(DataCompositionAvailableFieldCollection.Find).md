КоллекцияДоступныхПолейКомпоновкиДанных (DataCompositionAvailableFieldCollection)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор поля.

Возвращаемое значение:

Тип: [ДоступноеПолеКомпоновкиДанных](ДоступноеПолеКомпоновкиДанных_(DataCompositionAvailableField).md), [ДоступноеПолеОтбораКомпоновкиДанных](ДоступноеПолеОтбораКомпоновкиДанных_(DataCompositionFilterAvailableField).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
