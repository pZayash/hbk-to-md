Информационная база (IInfoBaseInfo)

MaxScheduledJobsStartShiftWithoutActiveUsers (MaxScheduledJobsStartShiftWithoutActiveUsers)

Доступен, начиная с версии 8.3.25.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет максимальное опоздание момента запуска каждого регламентного задания информационной базы в секундах относительно положенного по расписанию с учетом значения предыдущего параметра, если в информационной базе нет ни одного неспящего сеанса пользователей. Используется только если "Минимальный период запуска…" не равен 0.  
Значение по умолчанию: 0.  
Значение должно находиться в диапазоне от 0 до 4294967.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
