ЭлементПланировщика (PlannerItem)

ЗначенияИзмерений (DimensionValues)

Доступен, начиная с версии 8.3.6.

Использование:

Только чтение.

Описание:

Тип: [ФиксированноеСоответствие](ФиксированноеСоответствие_(FixedMap).md).   
Коллекция значений измерений. В качестве ключа выступает [Значение](ИзмерениеПланировщика.Значение_(PlannerDimension.Value).md) измерения. В качестве значения - [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md) элемента измерения (ИзмерениеПланировщика.<Элемент>.Значение).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
