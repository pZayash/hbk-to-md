**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md) › [Макеты областей](Макеты_областей.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ПараметрОбластиВыражениеКомпоновкиДанных](ПараметрОбластиВыражениеКомпоновкиДанных_(DataCompositionExpressionAreaParameter).md), [ПараметрОбластиРасшифровкаКомпоновкиДанных](ПараметрОбластиРасшифровкаКомпоновкиДанных_(DataCompositionDetailsAreaParameter).md), [МакетДокументаОбластиКомпоновкиДанных](МакетДокументаОбластиКомпоновкиДанных_(DataCompositionAreaDocumentTemplate).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются параметры.  
Возможно обращение к параметру посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя.
