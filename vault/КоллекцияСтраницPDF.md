Доступен, начиная с версии 8.3.26.

Элементы коллекции:

[СтраницаPDF](СтраницаPDF.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются страницы документа PDF.  
Возможно обращение к странице документа PDF посредством оператора [...]. В качестве аргумента передается .  

Методы:

[Индекс (IndexOf)](КоллекцияСтраницPDF.Индекс.md)  
[Количество (Count)](КоллекцияСтраницPDF.Количество.md)  
[Получить (Get)](КоллекцияСтраницPDF.Получить.md)  

Описание:

Предоставляет методы для работы со страницами PDF-документа.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.
