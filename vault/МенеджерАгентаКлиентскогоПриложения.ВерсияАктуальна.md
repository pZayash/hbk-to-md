МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

ВерсияАктуальна (IsVersionActual)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ВерсияАктуальна()

Возвращаемое значение:

Тип: Булево, Неопределено.   

Описание:

Проверяет, актуальная ли версия агента установлена и запущена (возвращает Истина или Ложь). Если агент не запущен, то возвращает Неопределено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
