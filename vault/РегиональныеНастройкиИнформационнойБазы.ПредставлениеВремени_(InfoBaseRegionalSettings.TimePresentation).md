РегиональныеНастройкиИнформационнойБазы (InfoBaseRegionalSettings)

ПредставлениеВремени (TimePresentation)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПредставлениеВремени(<Значение>)

Параметры:

<Значение> (необязательный)

Тип: Произвольный.   
Значение времени, представление которого требуется получить.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Представление значения, аналогичное тому, которое строится методом глобального контекста [Строка](Глобальный_контекст.Строка_(Global_context.String).md). В качестве региональных установок используются данные этого объекта [РегиональныеНастройкиИнформационнойБазы](РегиональныеНастройкиИнформационнойБазы_(InfoBaseRegionalSettings).md).  
Если в качестве параметра указана [Дата](lang__def_Date.md), то строится представление только времени без даты.

Описание:

Получает представление времени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
