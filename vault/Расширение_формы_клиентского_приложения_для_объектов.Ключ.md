Расширение формы клиентского приложения для объектов (Client application form extension for objects)

Ключ (Key)

Доступен, начиная с версии 8.2.

Использование:

Ключевой.

Описание:

Ключ объекта.   
В случае отсутствия или неверного значения создается новый объект с использованием остальных параметров.

Примечание:

Тип параметра соответствует типу объекта, редактируемого в форме.

Использование в версии:

Доступен, начиная с версии 8.2.
