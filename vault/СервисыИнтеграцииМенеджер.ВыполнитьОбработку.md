СервисыИнтеграцииМенеджер (IntegrationServicesManager)

ВыполнитьОбработку (ExecuteProcessing)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ВыполнитьОбработку()

Описание:

Выполняет запуск фоновых заданий обработки сервисов интеграции. Требует наличия права "Администрирование".  
Не допускается запуск, если в сеансе установлены разделители.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
