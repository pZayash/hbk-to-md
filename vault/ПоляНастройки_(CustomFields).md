# ПоляНастройки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Универсальные объекты](Универсальные_объекты.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[ПолеНастройки](ПолеНастройки_(CustomField).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются поля.  
Возможно обращение к полю посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0), имя поля.

Описание:

Коллекция допустимых полей для порядка, отбора, коллекций полей построителя отчета.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
