КонструкторФорматнойСтроки (FormatStringWizard)

Показать (Show)

Доступен, начиная с версии 8.3.3.

Синтаксис:

Показать(<ОписаниеОповещенияОЗакрытии>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (необязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога конструктора со следующими параметрами:

- <Текст> - [Текст](КонструкторФорматнойСтроки.Текст.md) - конструктор закрыли по кнопке "OK"; Неопределено - в противном случае,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Если параметр не указан или имеет тип Неопределено, то по завершении никакая процедура вызвана не будет.

Описание:

Открывает немодальное окно конструктора форматной строки.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.3.
