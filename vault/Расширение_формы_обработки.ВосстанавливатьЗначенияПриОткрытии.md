Расширение формы обработки (Data processor form extension)

ВосстанавливатьЗначенияПриОткрытии (RestoreValuesOnOpen)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Указывает необходимость восстановления настроек.   
Если свойство имеет значение Истина и одна из сохраненных настроек имеет установленный признак "Использовать при открытии", во время открытия формы сохраненные в данной настройке значения будут восстановлены. В противном случае восстановления значений при открытии формы происходить не будет.  
Сразу после создания формы данное свойство имеет значение, совпадающее со значением флажка "Восстанавливать настройки автоматически", установленном для данной формы в Конфигураторе.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
