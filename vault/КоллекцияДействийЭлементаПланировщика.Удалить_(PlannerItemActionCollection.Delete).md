КоллекцияДействийЭлементаПланировщика (PlannerItemActionCollection)

Удалить (Delete)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого действия в коллекции.

Описание варианта метода:

Удаляет действие по заданному индексу из коллекции действий элемента планировщика.

Вариант синтаксиса: По действию

Синтаксис:

Удалить(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеЭлементаПланировщика](ДействиеЭлементаПланировщика_(PlannerItemAction).md).   
Удаляемое действие в коллекции.

Описание варианта метода:

Удаляет заданное действие из коллекции действий элемента планировщика.

Описание:

Удаляет действие из коллекции действий элемента планировщика.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
