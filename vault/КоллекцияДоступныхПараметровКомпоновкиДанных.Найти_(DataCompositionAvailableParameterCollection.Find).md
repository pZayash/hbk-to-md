КоллекцияДоступныхПараметровКомпоновкиДанных (DataCompositionAvailableParameterCollection)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Идентификатор>)

Параметры:

<Идентификатор> (необязательный)

Тип: [Строка](lang__def_String.md).   
Идентификатор параметра.

Возвращаемое значение:

Тип: [ДоступныйПараметрКомпоновкиДанных](ДоступныйПараметрКомпоновкиДанных_(DataCompositionAvailableParameter).md), [Неопределено](lang__def_Undefined.md).   

Описание:

Осуществляет поиск элемента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
