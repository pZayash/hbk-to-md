Расширение формы клиентского приложения для отчета (Client application form extension for reports)

РежимОтображенияРезультата (ResultViewMode)

Доступен, начиная с версии 8.3.14.

Описание:

Тип: [РежимОтображенияРезультатаОтчета](РежимОтображенияРезультатаОтчета_(ReportResultViewMode).md).   
Содержит режим отображения результата, который будет использован в отчете.  
Если параметр задан, результат отчета будет сформирован в соответствии с переданным значением.  
Если параметр не задан, режим отображения результата будет получен из системного хранилища настроек.

Использование в версии:

Доступен, начиная с версии 8.3.14.
