Формат (Format)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Определяет формат отображения значения. Формат задается текстовой строкой.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
