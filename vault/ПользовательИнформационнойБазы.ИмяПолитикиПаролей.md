ПользовательИнформационнойБазы (InfoBaseUser)

ИмяПолитикиПаролей (PasswordPolicyName)

Доступен, начиная с версии 8.3.22.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя политики паролей пользователей для пользователя информационной базы.  
Если для пользователя установлено пустое значение свойства не существует политики с именем, соответствующим значению свойства, используются параметры информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер).

Использование в версии:

Доступен, начиная с версии 8.3.22.
