НастройкиОбработкиОшибок (ErrorProcessingSettings)

ОтправлятьОчет (SendReport)

Не рекомендуется использовать, начиная с версии 8.3.20.

Рекомендуется использовать:

- [ОтправлятьОтчет](НастройкиОбработкиОшибок.ОтправлятьОтчет_(ErrorProcessingSettings.SendReport).md)

Доступен, начиная с версии 8.3.17.

Использование:

Чтение и запись.

Описание:

Тип: [РежимОтправкиИнформацииОбОшибке](РежимОтправкиИнформацииОбОшибке_(ErrorReportingMode).md).   
Указывает необходимость отправки отчета.

Доступность:

Сервер, толстый клиент.

Примечание:

Значение по умолчанию - [Авто](РежимОтправкиИнформацииОбОшибке.Авто_(ErrorReportingMode.Auto).md).

Использование в версии:

Доступен, начиная с версии 8.3.17.

Не рекомендуется использовать, начиная с версии 8.3.20.

Описание изменено в версии 8.3.20.
