ConnID (ConnID)

Доступен, начиная с версии 8.1.

Описание:

Тип: Число.   
Содержит номер соединения. Имеет смысл, если с данного компьютера установлено несколько соединений.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
