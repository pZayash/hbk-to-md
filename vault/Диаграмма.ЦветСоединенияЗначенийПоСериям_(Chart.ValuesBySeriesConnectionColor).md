Диаграмма (Chart)

ЦветСоединенияЗначенийПоСериям (ValuesBySeriesConnectionColor)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет линий соединений. При выборе значения цвета "Авто", соединения выполняются для каждой серии собственным цветом, соответствующим свойству [Цвет](СерияДиаграммы.Цвет_(ChartSeries.Color).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.1.
