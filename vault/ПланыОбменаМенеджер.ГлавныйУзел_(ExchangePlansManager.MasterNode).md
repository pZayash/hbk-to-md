# ПланыОбменаМенеджер.ГлавныйУзел

ПланыОбменаМенеджер (ExchangePlansManager)

ГлавныйУзел (MasterNode)

Доступен, начиная с версии 8.0.

Синтаксис:

ГлавныйУзел()

Возвращаемое значение:

Тип: [ПланОбменаСсылка.](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)<[Имя плана обмена](ПланОбменаСсылка.Имя_плана_обмена_(ExchangePlanRef.Exchange_plan_name).md)>, [Неопределено](lang__def_Undefined.md).   
Если текущая информационная база не является узлом распределенной информационной базы или главный узел для нее не определен (она сама является корневым узлом) - метод возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Получает узел распределенной информационной базы, являющийся главным для текущей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ГлУзел = ПланыОбмена.ГлавныйУзел(); |

Использование в версии:

Доступен, начиная с версии 8.0.
