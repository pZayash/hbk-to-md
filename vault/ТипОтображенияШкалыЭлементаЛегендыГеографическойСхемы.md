Описание:

Определяет тип отображения шкалы элемент легенды географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Данный объект может быть сериализован в/из XML. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/geo}. Имя типа XDTO: GeographicalSchemaLegendItemShowScaleType.
