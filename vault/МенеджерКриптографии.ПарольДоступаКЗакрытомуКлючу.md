МенеджерКриптографии (CryptoManager)

ПарольДоступаКЗакрытомуКлючу (PrivateKeyAccessPassword)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Устанавливает пароль доступа к закрытому ключу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

На мобильной платформе игнорируется.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.20.
