Расширение поля формы для поля планировщика (Form field extension for a planner)

НажатиеНаПеренесенномЗаголовкеШкалыВремени (WrappedTimeScaleHeaderClick)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НажатиеНаПеренесенномЗаголовкеШкалыВремени(<Дата>)

Параметры:

<Дата>

Тип: [Дата](lang__def_Date.md).   
Дата, определяющая перенесенный заголовок шкалы времени, на котором было произведено нажатие.

Описание:

Возникает при нажатии левой кнопки мыши на перенесенном заголовке шкалы времени в случае, если значение свойства поля планировщика [ГиперссылкаПеренесенногоЗаголовкаШкалыВремени](Расширение_поля_формы_для_поля_планировщика.ГиперссылкаПеренесенногоЗаголовкаШкалыВремени_(Form_field_extension_for_a_planner.WrappedTimeScaleHeaderHyperlink).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
