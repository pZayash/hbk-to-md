ОбъектМетаданных: КритерийОтбора (MetadataObject: FilterCriterion)

ИспользоватьСтандартныеКоманды (UseStandardCommands)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено в значение [Ложь](lang__def_BooleanFalse.md), принадлежащие данному объекту метаданных команды не будут включаться в соответствующие фрагменты командного интерфейса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
