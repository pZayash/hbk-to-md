МенеджерБезопасногоХранилища (SecureStorageManager)

ПолучитьДанныеАсинх (GetDataAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьДанныеАсинх(<Ключ>, <СообщениеПользователю>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ, под которым были сохранены данные.

<СообщениеПользователю> (необязательный)

Тип: [Строка](lang__def_String.md).   
Текст сообщения, отображаемый пользователю при проверке доступа.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md), [Неопределено](lang__def_Undefined.md).   
Если переданный ключ не существует, возвращается ошибка.  
Результат выполнения обещания - тип Произвольный. Данные, хранимые под переданным ключом.  
Если операция проверки пользователя отменена пользователем, метод возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Получает данные из защищенного хранилища при выполнении условний доступа, переданных при сохранении данных.  
Вызов метода не блокирует исполнение кода языка, все результаты возвращаются в переданный обработчик оповещения.   
Если при помещении данных <СпособЗащитыДоступа> имеет значение [ТребуетсяДополнительнаяПроверкаПользователя](СпособЗащитыДоступаБезопасногоХранилища.ТребуетсяДополнительнаяПроверкаПользователя_(SecureStorageAccessProtectionMethod.AdditionalUserVerificationRequired).md), то при получении выполняется проверка пользователя в соответствии с заданным способом проверки.  
Выполняемая операция может быть отменена пользователем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
