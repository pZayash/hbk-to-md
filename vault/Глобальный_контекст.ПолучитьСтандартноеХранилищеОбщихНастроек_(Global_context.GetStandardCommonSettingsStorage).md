Глобальный контекст (Global context)

ПолучитьСтандартноеХранилищеОбщихНастроек (GetStandardCommonSettingsStorage)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ПолучитьСтандартноеХранилищеОбщихНастроек()

Возвращаемое значение:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md).   

Описание:

Получает объект, с помощью которого осуществляется доступ к содержимому стандартного хранилища общих настроек, независимо от того, заполнено ли свойство конфигурации [ХранилищеОбщихНастроек](ОбъектМетаданныхКонфигурация.ХранилищеОбщихНастроек_(ConfigurationMetadataObject.CommonSettingsStorage).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.26.
