Описание:

Используется для настройки структуры (состава группировок) отчета, группировки, группировок строк/колонок таблиц, группировок серий/точек диаграмм в составе пользовательских настроек.  
Состав группировок определяется коллекцией "Структура" при применении пользовательской настройки, коллекция преобразуется в последовательность вложенных друг в друга группировок.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
