Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСФайламиАсинх (AttachFileSystemExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПодключитьРасширениеРаботыСФайламиАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания - Истина, расширение успешно подключено, в противном случае - Ложь.  
  
Для тонкого клиента и клиента мобильного приложения результат всегда Истина.

Описание:

Асинхронная функция. При первом обращении выполняет подключение расширения для работы с файлами.  
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
