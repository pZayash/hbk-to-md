Форма (Form)

ЗакрыватьПриВыборе (CloseOnChoice)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что форму необходимо закрывать после осуществления выбора. Если свойство имеет значение [Ложь](lang__def_BooleanFalse.md), можно использовать форму для выбора нескольких позиций или элементов. По умолчанию свойство имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.
