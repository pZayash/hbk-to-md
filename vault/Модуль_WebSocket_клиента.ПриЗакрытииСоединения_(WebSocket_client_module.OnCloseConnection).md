Модуль WebSocket клиента (WebSocket client module)

ПриЗакрытииСоединения (OnCloseConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПриЗакрытииСоединения(<Соединение>, <КодЗакрытия>)

Параметры:

<Соединение>

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение_(WebSocketClientConnection).md).   
Ссылка на объект соединения.

<КодЗакрытия>

Тип: [Число](lang__def_Number.md).   
Статус-код закрытия соединения.  
1005 - если не получен.

Описание:

Обработчик закрытия соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
