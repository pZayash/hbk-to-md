Расширение элементов управления, расположенных в форме (Extension for controls located in a form)

КнопкаПоУмолчанию (DefaultButton)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак, является ли кнопка "кнопкой по умолчанию".   
Истина - является.  
Нажатие такой кнопки отрабатывает при нажатии сочетания клавиш Ctrl + Enter.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл только для элементов управления типа [Кнопка](Кнопка.md).  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы, свойство становится недоступно.

Использование в версии:

Доступен, начиная с версии 8.0.
