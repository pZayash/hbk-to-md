Глобальный контекст (Global context)

ХранилищеВнешнихДанныхНавигационныхСсылок (URLExternalDataStorage)

Доступен, начиная с версии 8.3.19.

Использование:

Только чтение.

Описание:

Тип: [СтандартноеХранилищеНастроекМенеджер](СтандартноеХранилищеНастроекМенеджер_(StandardSettingsStorageManager).md), [ХранилищеНастроекМенеджер.](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)<[Имя хранилища](ХранилищеНастроекМенеджер.Имя_хранилища_(SettingsStorageManager.Storage_name).md)>.   
Предоставляет доступ к хранилищу внешних данных навигационных ссылок.  
При использовании стандартного хранилища, все пользователи имеют доступ ко всем настройкам на чтение.   
При использовании стандартного хранилища для каждой настройки, сохраняемой в хранилище, записывается дата сохранения настройки. Ее можно получить через свойство [ДатаИзменения](СтандартноеХранилищеНастроекВыборка.ДатаИзменения_(StandardSettingsStorageSelection.ChangeDate).md). В режиме совместимости [Версия8\_3\_18](РежимСовместимости.Версия8_3_18_(CompatibilityMode.Version8_3_18).md) и ниже это свойство всегда содержит значение [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.19.
