ТаблицаФормы (FormTable)

ПередНачаломДобавления (BeforeAddRow)

Доступен, начиная с версии 8.2.

Синтаксис:

ПередНачаломДобавления(<Отказ>, <Копирование>, <Родитель>, <ЭтоГруппа>, <Параметр>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от добавления объекта. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то добавление объекта выполнено не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Копирование>

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет режим копирования. Если установлено [Истина](lang__def_BooleanTrue.md), то происходит копирование строки.

<Родитель>

Тип: [Неопределено](lang__def_Undefined.md), [СправочникСсылка.](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)<[Имя справочника](СправочникСсылка.Имя_справочника_(CatalogRef.Catalog_name).md)>, [ПланСчетовСсылка.](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)<[Имя плана счетов](ПланСчетовСсылка.Имя_плана_счетов_(ChartOfAccountsRef.Chart_of_accounts_name).md)>.   
Ссылка на элемент, который будет использован при добавлении в качестве родителя.

<ЭтоГруппа>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак добавления группы.  
[Истина](lang__def_BooleanTrue.md) - будет добавлена группа.

<Параметр>

Тип: Произвольный.   
Параметр команды, выполняемой при добавлении строки в таблицу.  
Для стандартной команды "Создать по параметру" содержит тип создаваемого документа.  
В остальных случаях значение параметра равно [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Вызывается перед началом интерактивного добавления строки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Примечание:

В обработчике данного события нельзя использовать серверные методы формы с директивой компиляции &НаСервере, а также изменять свойства, которые могут привести к серверному вызову.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.15.
