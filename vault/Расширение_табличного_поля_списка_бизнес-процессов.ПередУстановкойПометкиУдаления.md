Расширение табличного поля списка бизнес-процессов (Business process list table box extension)

ПередУстановкойПометкиУдаления (BeforeSetDeletionMark)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУстановкойПометкиУдаления(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак изменения пометки удаления бизнес-процесса. Если в теле процедуры-обработчика установить данному параметру значение Истина, то пометка удаления изменена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед установкой пометки удаления в списке бизнес-процессов.

Доступность:

Толстый клиент.

Примечание:

Установка пометки удаления приводит к записи объекта и, соответственно, вызывает события [ПередЗаписью](БизнесПроцессОбъект.Имя_бизнес-процесса.ПередЗаписью.md) и [ПриЗаписи](БизнесПроцессОбъект.Имя_бизнес-процесса.ПриЗаписи.md) объекта [БизнесПроцессОбъект.](БизнесПроцессОбъект.Имя_бизнес-процесса.md)<[Имя бизнес-процесса](БизнесПроцессОбъект.Имя_бизнес-процесса.md)>.

Использование в версии:

Доступен, начиная с версии 8.0.
