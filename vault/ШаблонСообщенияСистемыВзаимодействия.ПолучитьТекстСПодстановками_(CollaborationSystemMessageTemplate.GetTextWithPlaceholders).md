ШаблонСообщенияСистемыВзаимодействия (CollaborationSystemMessageTemplate)

ПолучитьТекстСПодстановками (GetTextWithPlaceholders)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьТекстСПодстановками(<ЗначенияПодстановок>)

Параметры:

<ЗначенияПодстановок> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив значений подстановок.

Возвращаемое значение:

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   

Описание:

Возвращает текст шаблонного сообщения, заполненный значениями подстановок.  
Значения подстановок в форматированной строке выделены стандартным оформлением.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
