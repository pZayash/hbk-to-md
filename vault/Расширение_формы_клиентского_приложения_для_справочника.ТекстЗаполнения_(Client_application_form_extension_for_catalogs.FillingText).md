Расширение формы клиентского приложения для справочника (Client application form extension for catalogs)

ТекстЗаполнения (FillingText)

Доступен, начиная с версии 8.3.3.

Описание:

Тип: [Строка](lang__def_String.md).   
Текст, введенный в поле ввода, при открытии формы в режиме выбора.  
Если основное представление [ВВидеНаименования](ОсновноеПредставлениеСправочника.ВВидеНаименования_(CatalogMainPresentation.AsDescription).md) и реквизит [Наименование](СправочникОбъект.Имя_справочника.Наименование_(CatalogObject.Catalog_name.Description).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_Справочник.ВводПоСтроке_(MetadataObject_Catalog.InputByString).md), то значение присваивается реквизиту [Наименование](СправочникОбъект.Имя_справочника.Наименование_(CatalogObject.Catalog_name.Description).md).  
Если основное представление [ВВидеКода](ОсновноеПредставлениеСправочника.ВВидеКода_(CatalogMainPresentation.AsCode).md) и реквизит [Код](СправочникОбъект.Имя_справочника.Код_(CatalogObject.Catalog_name.Code).md) находится в списке [ВводПоСтроке](ОбъектМетаданных_Справочник.ВводПоСтроке_(MetadataObject_Catalog.InputByString).md) и не установлен признак [Автонумерация](ОбъектМетаданных_Справочник.Автонумерация_(MetadataObject_Catalog.Autonumbering).md), то значение присваивается реквизиту [Код](СправочникОбъект.Имя_справочника.Код_(CatalogObject.Catalog_name.Code).md).  
В противном случае присваивание не выполняется.

Использование в версии:

Доступен, начиная с версии 8.3.3.
