Расширение группы формы для обычной группы (Form group extension for a usual group)

Объединенная (United)

Доступен, начиная с версии 8.3.7.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если данное свойство имеет значение [Ложь](lang__def_BooleanFalse.md), то элементы группы располагаются в родительской группе.  
Для такой группы игнорируются все свойства, кроме свойств [ЦветФона](Расширение_группы_формы_для_обычной_группы.ЦветФона_(Form_group_extension_for_a_usual_group.BackColor).md) и [Группировка](Расширение_группы_формы_для_обычной_группы.Группировка_(Form_group_extension_for_a_usual_group.Group).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.7.
