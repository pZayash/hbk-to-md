Точная (Accurate)

Доступен, начиная с версии 8.2.

Описание:

Печать без изменения полей и масштаба.  
При печати из веб-браузера Google Chrome способом PDF печатаемый документ возвращается в виде файла.

Использование в версии:

Доступен, начиная с версии 8.2.
