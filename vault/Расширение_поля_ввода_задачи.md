# Расширение поля ввода задачи

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Прикладные_объекты.md">Прикладные объекты</a> › [Задачи](Задачи.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора задачи в элементе управления [ПолеВвода](ПолеВвода.md) по введенному тексту. По умолчанию поиск осуществляется по полям "Номер" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
