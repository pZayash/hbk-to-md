HTTPМетод (HTTPMethod)

OPTIONS (OPTIONS)

Доступен, начиная с версии 8.3.5.

Описание:

Соответствует HTTP-методу OPTIONS.  
Предназначен для получения информации о способах работы с данным URI (доступные HTTP-методы, форматы представления и т.п.).  
Описан в стандарте RFC 2616 (http://tools.ietf.org/html/rfc2616).

Использование в версии:

Доступен, начиная с версии 8.3.5.
