Расширение элементов управления, расположенных на панели (Panel controls extension)

Видимость (Visible)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет признак видимости элемента управления.

Доступность:

Толстый клиент.

Примечание:

[Ложь](lang__def_BooleanFalse.md) - элемент не отображается.

Использование в версии:

Доступен, начиная с версии 8.0.
