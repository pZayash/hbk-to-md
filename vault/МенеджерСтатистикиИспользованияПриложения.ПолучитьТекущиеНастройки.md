МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьТекущиеНастройки (GetCurrentSettings)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьТекущиеНастройки()

Возвращаемое значение:

Тип: Строка.   
Если результат равен пустой строке – настройки регистрации в статистике отключены.

Описание:

Возвращает текст с текущими настройками регистрации событий/экранов статистики использования приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
