ИнформационныйИнтервалДиаграммы (ChartReferenceBand)

НачалоОбластиОтображения (DisplayAreaBegin)

Доступен, начиная с версии 8.3.12.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Положение начала области отображения информационного интервала относительно диаграммы.  
Может содержать значения от 0 до 1.  
Значение по умолчанию - 0.  
Для интервалов значений свойство отсчитывается вдоль оси точек,  
а для интервалов точек - вдоль оси значений.  
Например, для интервала значений диаграммы, размещенной горизонтально:

- 0 - начало области интервала располагается у левой границы области диаграммы;
- 0,5 - начало области интервала располагается в середине области диаграммы;
- 1 - начало области интервала располагается у правой границы области диаграммы.

Если значение больше или равно свойству [КонецОбластиОтображения](ИнформационныйИнтервалДиаграммы.КонецОбластиОтображения.md), интервал отображаться не будет.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
