МенеджерСистемыВзаимодействия (CollaborationSystemManager)

УдалитьСообщение (DeleteMessage)

Доступен, начиная с версии 8.3.11.

Синтаксис:

УдалитьСообщение(<ИдентификаторСообщения>)

Параметры:

<ИдентификаторСообщения> (обязательный)

Тип: [ИдентификаторСообщенияСистемыВзаимодействия](ИдентификаторСообщенияСистемыВзаимодействия_(CollaborationSystemMessageID).md).   
Идентификатор удаляемого сообщения.

Описание:

Удаляет указанное сообщение из обсуждения.  
Вызов метода возможен только в привилегированном режиме.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
