Расширение поля формы для поля планировщика (Form field extension for a planner)

РазрешитьНачалоПеретаскивания (EnableStartDrag)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Разрешает начало перетаскивания из поля планировщика - источника данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Примечание:

[Истина](lang__def_BooleanTrue.md) - перетаскивание разрешено; [Ложь](lang__def_BooleanFalse.md) - запрещено.

Использование в версии:

Доступен, начиная с версии 8.3.6.
