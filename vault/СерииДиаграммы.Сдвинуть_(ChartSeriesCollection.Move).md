# СерииДиаграммы.Сдвинуть

СерииДиаграммы (ChartSeriesCollection)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.14.

Синтаксис:

Сдвинуть(<Серия>, <Смещение>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммы](СерияДиаграммы_(ChartSeries).md).   
Сдвигаемая серия.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество позиций, на которое необходимо сдвинуть серию.

Описание:

Сдвигает серию на указанное количество позиций.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
