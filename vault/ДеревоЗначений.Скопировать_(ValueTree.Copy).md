ДеревоЗначений (ValueTree)

Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ДеревоЗначений](ДеревоЗначений_(ValueTree).md).   

Описание:

Создает полную копию исходного дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // ДеревоСостава имеет тип ДеревоЗначений  НовоеДеревоСостава = ДеревоСостава.Скопировать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
