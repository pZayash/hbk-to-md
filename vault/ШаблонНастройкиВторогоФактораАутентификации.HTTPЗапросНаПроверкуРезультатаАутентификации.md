HTTPЗапросНаПроверкуРезультатаАутентификации (AuthenticationResultVerificationHTTPRequest)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [HTTPЗапрос](HTTPЗапрос.md), Неопределено.   
Запрос к провайдеру, проверяющий результат аутентификации в случае, если провайдер выполняет аутентификацию самостоятельно. Неопределено, если опрос результата не требуется.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
