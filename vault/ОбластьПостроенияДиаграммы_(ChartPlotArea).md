Описание:

Используется для доступа к области построения диаграммы для настройки параметров оформления. В этой области выводятся координатное пространство и фигура диаграммы.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
