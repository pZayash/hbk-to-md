ДоставляемоеУведомление (DeliverableNotification)

Наклейка (Badge)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Число.   
Числовое значение, которое будет отображаться рядом со значком приложения на домашнем экране в iOS и в уведомлении Android:

- Для платформы iOS: Значение задает наклейку (badge), которая будет установлена у иконки мобильного приложения на домашнем экране. Значение наклейки можно установить с помощью метода [УстановитьНаклейку](МенеджерДоставляемыхУведомлений.УстановитьНаклейку.md) и получить с помощью метода [ПолучитьНаклейку](МенеджерДоставляемыхУведомлений.ПолучитьНаклейку.md).
- Для платформы Android: Значение задает число, которое будет отображаться в сообщении на панели уведомлений Android. Значение будет видно пользователю только когда мобильное приложение неактивно (свернуто). Обработчику доставляемых уведомлений в конфигурации значение не передается.

Для платформы Windows Phone уведомление не меняет значение наклейки на иконке приложения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
