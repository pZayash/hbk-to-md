ПользовательскоеПолеВыборКомпоновкиДанных (DataCompositionUserFieldCase)

ПутьКДанным (DataPath)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит путь к данным поля. Является его идентификатором, создаваемым автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

В системе компоновки данных не допускается использовать в качестве пути к данным имена, совпадающие с ключевыми словами:  
ВЫБОР (CASE)  
ЕСТЬ (IS)  
ИНАЧЕ (ELSE)  
КОГДА (WHEN)  
КОНЕЦ (END)  
НЕ (NOT)  
ПОДОБНО (LIKE)  
РАЗЛИЧНЫЕ (DISTINCT)  
СПЕЦСИМВОЛ (ESCAPE)  
ТОГДА (THEN)

Использование в версии:

Доступен, начиная с версии 8.1.
