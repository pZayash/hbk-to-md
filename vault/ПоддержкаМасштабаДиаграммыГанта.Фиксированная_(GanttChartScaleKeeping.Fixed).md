ПоддержкаМасштабаДиаграммыГанта (GanttChartScaleKeeping)

Фиксированная (Fixed)

Доступен, начиная с версии 8.0.

Описание:

Задается определенный масштаб (в символах на единицу времени), который не меняется при изменении размеров окна.

Использование в версии:

Доступен, начиная с версии 8.0.
