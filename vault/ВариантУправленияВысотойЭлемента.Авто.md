ВариантУправленияВысотойЭлемента (ItemHeightControlVariant)

Авто (Auto)

Доступен, начиная с версии 8.3.9.

Описание:

Вариант управления высотой определяется автоматически:

- Для многострочного поля ввода - [ПоСодержимому](ВариантУправленияВысотойЭлемента.ПоСодержимому.md),
- Для однострочного поля ввода - [ВСтрокахФормы](ВариантУправленияВысотойЭлемента.ВСтрокахФормы.md).

Использование в версии:
