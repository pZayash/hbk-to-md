ПолучитьВыгрузкуДанныхАсинх (GetDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ПолучитьВыгрузкуДанныхАсинх(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   
Идентификатор выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – [ВыгрузкаДанныхСистемыВзаимодействия](ВыгрузкаДанныхСистемыВзаимодействия_(CollaborationSystemDataDump).md).

Описание:

Асинхронная функция. Получает выгрузку данных системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.
