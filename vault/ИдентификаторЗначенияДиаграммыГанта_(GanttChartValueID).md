# ИдентификаторЗначенияДиаграммыГанта

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Диаграмма Ганта](Диаграмма_Ганта.md)

Доступен, начиная с версии 8.3.21.

Описание:

Используется для поиска значения в диаграмме Ганта. Значения этого типа имеют строковое представление (от числового идентификатора) и поддерживают сериализацию в XDTO.  
Данный тип не содержит свойств и методов.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.   
Возможен обмен с сервером. Сериализуется. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен http://v8.1c.ru/8.2/data/chart. Имя типа XDTO: GanttChartValueID.

Использование в версии:

Доступен, начиная с версии 8.3.21.
