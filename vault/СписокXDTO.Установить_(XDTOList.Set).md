СписокXDTO (XDTOList)

Установить (Set)

Доступен, начиная с версии 8.1.

Синтаксис:

Установить(<Индекс>, <Элемент>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо установить элемент данных.

<Элемент> (обязательный)

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Элемент данных XDTO, устанавливаемый в указанную позицию списка.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md).   
Установленный элемент данных.

Описание:

Устанавливает элемент данных XDTO в список по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
