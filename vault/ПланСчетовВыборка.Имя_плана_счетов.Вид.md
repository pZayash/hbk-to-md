ПланСчетовВыборка.<Имя плана счетов> (ChartOfAccountsSelection.<Chart of accounts name>)

Вид (Type)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ВидСчета](ВидСчета.md).   
Определяет вид счета.   
На активном счете будет всегда дебетовый остаток, на пассивном кредитовый, на активно пассивном, если остаток больше нуля, то дебетовый, если меньше, то кредитовый.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
