ГлубинаЦвета (ColorDepth)

БитНаПиксел32 (BitPerPixel32)

Доступен, начиная с версии 8.3.14.

Описание:

Для хранения одного пикселя используется 32 бита (картинка полноцветная с прозрачностью).

Использование в версии:

Доступен, начиная с версии 8.3.14.
