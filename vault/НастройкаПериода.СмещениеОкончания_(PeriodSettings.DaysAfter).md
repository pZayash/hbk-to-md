НастройкаПериода (PeriodSettings)

СмещениеОкончания (DaysAfter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Количество дней после рабочей даты для конца интервала.

Доступность:

Толстый клиент.

Примечание:

Имеет смысл, если свойство [ВариантОкончания](НастройкаПериода.ВариантОкончания_(PeriodSettings.EndingVariant).md) имеет значение [Смещение](ВариантГраницыИнтервала.Смещение_(IntervalBoundVariant.BeforeAfter).md).

Использование в версии:

Доступен, начиная с версии 8.0.
