# Расширение формы элемента справочника.ПередЗаписью

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [Расширение формы элемента справочника](Расширение_формы_элемента_справочника_(Catalog_item_form_extension).md) › [События](Расширение_формы_элемента_справочника__События.md)

Расширение формы элемента справочника (Catalog item form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи элемента. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то запись элемента выполнена не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Возникает перед выполнением записи объекта из формы. Процедура-обработчик данного события вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](СправочникОбъект.Имя_справочника.ПередЗаписью_(CatalogObject.Catalog_name.BeforeWrite).md) объекта [СправочникОбъект.](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)<[Имя справочника](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
