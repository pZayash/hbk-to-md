Диаграмма (Chart)

УстановитьПалитру (SetPalette)

Не рекомендуется использовать, начиная с версии 8.3.12.

Рекомендуется использовать:

- [УстановитьПалитру](ОписаниеПалитрыЦветовДиаграммы.УстановитьПалитру.md)

Доступен, начиная с версии 8.3.10 (в режиме совместимости с версией 8.3.12 и последующими).

Синтаксис:

УстановитьПалитру(<Палитра>)

Параметры:

<Палитра> (необязательный)

Тип: [Массив](Массив.md).   
Массив цветов палитры. Используются только элементы типа [Цвет](Цвет.md). Элементы другого типа игнорируются.

Описание:

Устанавливает свойство диаграммы [ПалитраЦветов](Диаграмма.ПалитраЦветов.md) в значение [Произвольная](ПалитраЦветовДиаграммы.Произвольная.md). В качестве палитры используются цвета из переданного массива. Цвета записываются в палитру в том же порядке, как они расположены в массиве.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10 (в режиме совместимости с версией 8.3.12 и последующими).

Не рекомендуется использовать, начиная с версии 8.3.12.

Описание изменено в версии 8.3.15.
