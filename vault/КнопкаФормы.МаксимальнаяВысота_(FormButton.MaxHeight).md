# КнопкаФормы.МаксимальнаяВысота

КнопкаФормы (FormButton)

МаксимальнаяВысота (MaxHeight)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальную высоту кнопки. Используется, если [АвтоМаксимальнаяВысота](КнопкаФормы.АвтоМаксимальнаяВысота_(FormButton.AutoMaxHeight).md) установлена в значение [Ложь](lang__def_BooleanFalse.md).  
Задается в символах.  
0 - максимальная высота не ограничена.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.6.
