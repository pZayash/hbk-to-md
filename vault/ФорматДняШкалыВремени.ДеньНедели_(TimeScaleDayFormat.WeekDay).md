ФорматДняШкалыВремени (TimeScaleDayFormat)

ДеньНедели (WeekDay)

Доступен, начиная с версии 8.0.

Описание:

Предпочтительный формат дня для элемента шкалы времени может быть описан как "ДФ=дддд". Например, дата 20.11.2004 будет представлена как "четверг".

Использование в версии:

Доступен, начиная с версии 8.0.
