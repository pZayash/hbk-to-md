Описание:

Содержит данные о возможности записи файла программной лицензии в каталог лицензий.  
Объект может быть получен только с помощью метода [ПолучитьДоступностьПолученияЛицензии](МенеджерПолученияЛицензий.ПолучитьДоступностьПолученияЛицензии.md).

Доступность:

Сериализуется.
