СредстваКриптографии (CryptoTools)

НачатьПолучениеИнформацииМодуляКриптографии (BeginGettingCryptoModuleInformation)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПолучениеИнформацииМодуляКриптографии(<ОписаниеОповещения>, <ИмяМодуляКриптографии>, <ПутьМодуляКриптографии>, <ТипМодуляКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после после получения информации о модуле криптографии со следующими параметрами:

- <ИнформацияМодуля> - значение объекта [ИнформацияМодуляКриптографии](ИнформацияМодуляКриптографии.md) или Неопределено,
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ИмяМодуляКриптографии> (обязательный)

Имя модуля криптографии.  
Параметр используется, если менеджер создается для операционной системы Windows.

<ПутьМодуляКриптографии> (обязательный)

Тип: Строка.   
Путь к файлу-библиотеке, реализующей требуемый набор функций.  
Параметр используется, если менеджер создается для операционной системы Linux и macOS.

<ТипМодуляКриптографии> (обязательный)

Тип: Число.   
Задает тип модуля криптографии.

Возвращаемое значение:

Если модуль криптографии не поддерживается (не найден по имени для MS Windows, либо не экспортируются требуемые функции на Linux), то при вызове метода из объекта [ОписаниеОповещения](ОписаниеОповещения.md) первым параметром возвращает Неопределено.

Описание:

Начинает получение информации о модуле криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[СредстваКриптографии](СредстваКриптографии.md), метод [ПолучитьИнформациюМодуляКриптографии](СредстваКриптографии.ПолучитьИнформациюМодуляКриптографии.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.22.
