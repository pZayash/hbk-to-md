# АдминистрированиеКластер.ТаймаутПроверкиСоединения

АдминистрированиеКластер (AdministrationCluster)

ТаймаутПроверкиСоединения (PingTimeout)

Доступен, начиная с версии 8.3.26.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Определяет таймаут получения ответов на проверочные запросы системы отслеживания соединений, в миллисекундах. Допустимые значения: от 0 до 2147483648.   
Если данное свойство и свойство [ПериодПроверкиСоединения](АдминистрированиеКластер.ПериодПроверкиСоединения_(AdministrationCluster.PingPeriod).md) имеют значение 0, то используются параметры системы отслеживания соединений, установленные с помощью командной строки.  
Значение свойства должно быть установлено вместе со значением свойства [ПериодПроверкиСоединения](АдминистрированиеКластер.ПериодПроверкиСоединения_(AdministrationCluster.PingPeriod).md).  
Значения по умолчанию:

- ПериодПроверкиСоединения - 1000 мс,
- ТаймаутПроверкиСоединения - 5000 мс.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
