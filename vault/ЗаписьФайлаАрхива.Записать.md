ЗаписьФайлаАрхива (ArchiveFileWriter)

Записать (Write)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Записать()

Описание:

Записывает и закрывает файл архива.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если файл с таким именем уже существует, он перезаписывается.  
При записи файлов-архивов средствами платформы имена файлов в архиве записываются в кодировке Unicode. Встроенный архиватор ОС Windows 2008 и младше, некорректно отображают такие символы. Для ОС Windows 7/2008 имеется возможность установить исправление https://support.microsoft.com/ru-ru/kb/2704299, которое решает данную проблему. Если установить это исправление нет возможности, то рекомендуется использовать или символы латинского алфавита для имен файлов или использовать внешние (по отношению к ОС) архиваторы, которые не имеют проблем с отображением имен файлов, заданных в Unicode. Также рекомендуется использовать символы латинского алфавита в том случае, если предполагается использование получившегося архива конечным пользователем и нет возможности гарантировать нужный архиватор у этого пользователя.

Использование в версии:

Доступен, начиная с версии 8.3.26.
