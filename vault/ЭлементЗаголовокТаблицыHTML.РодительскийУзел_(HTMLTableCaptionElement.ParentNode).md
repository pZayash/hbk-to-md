ЭлементЗаголовокТаблицыHTML (HTMLTableCaptionElement)

РодительскийУзел (ParentNode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ЭлементЗаголовокТаблицыHTML](ЭлементЗаголовокТаблицыHTML_(HTMLTableCaptionElement).md), [Неопределено](lang__def_Undefined.md).   
Родительский узел узла HTML или [Неопределено](lang__def_Undefined.md), в случае отсутствия такового.  
Все узлы, за исключением [Атрибут](ТипУзлаDOM.Атрибут_(DOMNodeType.Attribute).md), [Документ](ТипУзлаDOM.Документ_(DOMNodeType.Document).md), [ФрагментДокумента](ТипУзлаDOM.ФрагментДокумента_(DOMNodeType.DocumentFragment).md), могут иметь родительский узел.  
Если узел только что был создан и еще не добавлен в дерево или был уже удален из него, родительский узел отсутствует.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
