СериализаторXDTO (XDTOSerializer)

ЗаписатьJSON (WriteJSON)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ЗаписатьJSON(<ЗаписьJSON>, <Значение>, <НазначениеТипаXML>)

Параметры:

<ЗаписьJSON> (обязательный)

Тип: [ЗаписьJSON](ЗаписьJSON_(JSONWriter).md).   
Объект, через который осуществляется запись JSON.

<Значение> (обязательный)

Тип: Произвольный.   
Записываемое в поток JSON значение.

<НазначениеТипаXML> (необязательный)

Тип: [НазначениеТипаXML](НазначениеТипаXML_(XMLTypeAssignment).md).   
Вариант назначения типа элемента данных XDTO.  
Значение по умолчанию: [Неявное](НазначениеТипаXML.Неявное_(XMLTypeAssignment.Implicit).md).

Описание:

Записывает значения в формате JSON.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В JSON могут быть записаны значения следующих типов: [Неопределено](lang__def_Undefined.md), [Null](lang__def_Null.md), [Булево](lang__Булево_(Boolean).md), [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md), [Тип](lang__def_Type.md), [УникальныйИдентификатор](УникальныйИдентификатор_(UUID).md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md), [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md), а также ссылки объектов данных и сами объекты данных, наборы записей регистров, менеджер значения константы. В дальнейшем список может пополняться.  
При попытке записи значения неподходящего типа будет вызвано исключение.

Использование в версии:

Доступен, начиная с версии 8.3.7.
