МенеджерПолнотекстовогоПоиска (FullTextSearchManager)

УстановитьРежимПолнотекстовогоПоиска (SetFullTextSearchMode)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьРежимПолнотекстовогоПоиска(<РежимПолнотекстовогоПоиска>)

Параметры:

<РежимПолнотекстовогоПоиска> (обязательный)

Тип: [РежимПолнотекстовогоПоиска](РежимПолнотекстовогоПоиска_(FullTextSearchMode).md).   
Устанавливаемый режим полнотекстового поиска.

Описание:

Устанавливает режим полнотекстового поиска.  
Требуется режим монопольного доступа к информационной базе. Если до вызова метода монопольный доступ не установлен, то генерируется исключение.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.25.
