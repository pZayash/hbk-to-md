# РежимЗаписиДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Свойства прикладных объектов](Свойства_прикладных_объектов.md)

Доступен, начиная с версии 8.0.

Значения

[Запись (Write)](РежимЗаписиДокумента.Запись_(DocumentWriteMode.Write).md)  
[ОтменаПроведения (UndoPosting)](РежимЗаписиДокумента.ОтменаПроведения_(DocumentWriteMode.UndoPosting).md)  
[Проведение (Posting)](РежимЗаписиДокумента.Проведение_(DocumentWriteMode.Posting).md)  

Описание:

Определяет набор режимов записи документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.
