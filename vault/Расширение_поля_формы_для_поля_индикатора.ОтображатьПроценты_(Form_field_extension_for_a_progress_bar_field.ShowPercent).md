Расширение поля формы для поля индикатора (Form field extension for a progress bar field)

ОтображатьПроценты (ShowPercent)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит способ отображения процентов в индикаторе.   
[Истина](lang__def_BooleanTrue.md) - отображаются.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
