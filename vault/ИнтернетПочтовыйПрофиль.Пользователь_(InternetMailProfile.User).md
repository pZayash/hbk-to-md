ИнтернетПочтовыйПрофиль (InternetMailProfile)

Пользователь (User)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Логин пользователя – часть «пользователь» почтового адреса. Поддерживаются только символы, входящие в кодовую таблицу US-ASCII.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[ИнтернетПочтовыйАдрес](ИнтернетПочтовыйАдрес_(InternetMailAddress).md), свойство [Пользователь](ИнтернетПочтовыйАдрес.Пользователь_(InternetMailAddress.User).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
