Глобальный контекст (Global context)

ПолучитьМакетОформления (GetAppearanceTemplate)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьМакетОформления(<Оформление>)

Параметры:

<Оформление> (обязательный)

Тип: [СтандартноеОформление](СтандартноеОформление_(StandardAppearance).md).   
Содержит стандартный вариант оформления.

Возвращаемое значение:

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md).   

Описание:

Получает макет оформления.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| ПостроительОтчета.МакетОформления = ПолучитьМакетОформления(СтандартноеОформление.Интерфейс); |

Использование в версии:

Доступен, начиная с версии 8.0.
