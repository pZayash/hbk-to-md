МенеджерАгентаКлиентскогоПриложения (ClientApplicationAgentManager)

ПолучитьНаименованиеПриложения (GetApplicationDescription)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьНаименованиеПриложения()

Возвращаемое значение:

Тип: Строка, Неопределено.   

Описание:

Возвращает текущее наименование приложения в агенте клиентского приложения. Если агент не запущен, то возвращает Неопределено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
