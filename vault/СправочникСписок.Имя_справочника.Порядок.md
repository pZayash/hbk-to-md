Порядок (Order)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Порядок](Универсальные_объекты.Порядок.md).   
Содержит объект [Порядок](Универсальные_объекты.Порядок.md), по которому осуществляется текущее упорядочивание элементов справочника в списке. В качестве имен упорядочивания объекта могут выступать имена предопределенных реквизитов Код и Наименование и имена реквизитов справочника.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
