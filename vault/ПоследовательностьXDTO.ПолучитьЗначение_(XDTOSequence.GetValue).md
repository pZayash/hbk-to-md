ПоследовательностьXDTO (XDTOSequence)

ПолучитьЗначение (GetValue)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьЗначение(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента последовательности.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO_(XDTODataValue).md), [ОбъектXDTO](ОбъектXDTO_(XDTODataObject).md), [Неопределено](lang__def_Undefined.md).   
Если не найдено, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Получает значение (элемент данных) по указанному индексу.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Если по указанному индексу записан текст, будет инициировано исключение.

Использование в версии:

Доступен, начиная с версии 8.1.
