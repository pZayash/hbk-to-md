КоллекцияАтрибутовHTML (HTMLAttributeMap)

УдалитьИменованныйЭлемент (RemoveNamedItem)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьИменованныйЭлемент(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя удаляемого атрибута HTML.

Возвращаемое значение:

Тип: [АтрибутDOM](АтрибутDOM.md).   

Описание:

Удаляет узел атрибут из коллекции по указанному имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
