# ДиаграммаГанта.СвернутьТочку

ДиаграммаГанта (GanttChart)

СвернутьТочку (CollapsePoint)

Доступен, начиная с версии 8.0.

Синтаксис:

СвернутьТочку(<Точка>, <СПодчиненными>)

Параметры:

<Точка> (обязательный)

Тип: [ТочкаДиаграммыГанта](ТочкаДиаграммыГанта_(GanttChartPoint).md).   
Точка диаграммы Ганта.

<СПодчиненными> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость сворачивания точки вместе с подчиненными точками. [Истина](lang__def_BooleanTrue.md) - свернуть вместе с подчиненными.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Сворачивает точки, возможно вместе с подчиненными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
