КолонкиДанныхДиаграммыГанта (GanttChartDataColumns)

Вставить (Insert)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Вставить(<Индекс>, <Имя>, <Тип>, <Заголовок>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс вставляемой колонки.

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки.

<Тип> (обязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Объект, описывающий допустимые типы значений для колонки.

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок колонки.  
Значение по умолчанию: Пустая строка.

Возвращаемое значение:

Тип: КолонкаДанныхДиаграммыГанта.   

Описание:

Вставляет колонку в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
