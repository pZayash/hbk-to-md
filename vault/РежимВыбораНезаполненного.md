Значения

[ПриАктивизации (OnActivate)](РежимВыбораНезаполненного.ПриАктивизации.md)  
[ПриНажатииEnter (OnEnterPressed)](РежимВыбораНезаполненного.ПриНажатииEnter.md)  

Описание:

Перечисление описывает варианты выбора незаполненных значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Сериализуется.
