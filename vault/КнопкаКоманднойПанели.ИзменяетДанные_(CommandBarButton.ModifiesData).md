КнопкаКоманднойПанели (CommandBarButton)

ИзменяетДанные (ModifiesData)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что действие, выполняемое кнопкой, приводит к изменению данных. Например, заполнение или изменение объекта.  
[Истина](lang__def_BooleanTrue.md) - приводит к изменению.

Доступность:

Толстый клиент.

Примечание:

Если изменение данных объекта запрещено, то действие по кнопке выполнено не будет.

Использование в версии:

Доступен, начиная с версии 8.0.
