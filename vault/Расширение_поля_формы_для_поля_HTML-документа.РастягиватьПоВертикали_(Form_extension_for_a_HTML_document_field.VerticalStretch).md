Расширение поля формы для поля HTML-документа (Form extension for a HTML document field)

РастягиватьПоВертикали (VerticalStretch)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает возможность растягивания поля по вертикали.   
[Истина](lang__def_BooleanTrue.md) - растягивание разрешено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
