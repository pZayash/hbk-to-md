Описание:

Тип: СправочникТабличнаяЧасть.<Имя справочника>.<Имя табличной части>.   
Набор свойств содержит табличные части справочника. Доступ к табличной части осуществляется по имени, как оно задано в Конфигураторе (СправочникТабличнаяЧасть.<Имя справочника>.<Имя табличной части>).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
