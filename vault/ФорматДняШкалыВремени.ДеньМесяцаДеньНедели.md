ФорматДняШкалыВремени (TimeScaleDayFormat)

ДеньМесяцаДеньНедели (MonthDayWeekDay)

Доступен, начиная с версии 8.0.

Описание:

Предпочтительный формат дня для элемента шкалы времени может быть представлен как ДеньМесяца + ДеньНедели.

Использование в версии:

Доступен, начиная с версии 8.0.
