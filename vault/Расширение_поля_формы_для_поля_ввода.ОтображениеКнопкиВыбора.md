Расширение поля формы для поля ввода (Form field extension for a text box)

ОтображениеКнопкиВыбора (ChoiceButtonRepresentation)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеКнопкиВыбора](ОтображениеКнопкиВыбора.md).   
Если указано значение [Авто](ОтображениеКнопкиВыбора.Авто.md), то значение интерпретируется в зависимости от типа, редактируемого в поле ввода значения:

- Ссылочный тип (в том числе пользовательское перечисление) - [ОтображатьВВыпадающемСписке](ОтображениеКнопкиВыбора.ОтображатьВВыпадающемСписке.md);
- Для остальных типов - [ОтображатьВПолеВвода](ОтображениеКнопкиВыбора.ОтображатьВПолеВвода.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
