СвязьНаборовДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSetLink)

Обязательная (Required)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак необходимости создания связи.  
[Истина](lang__def_BooleanTrue.md) - связь будет обязательно добавлена в макет компоновки, если задействованы поля набора данных - приемника.  
[Ложь](lang__def_BooleanFalse.md) - связь будет добавляться только в случае, если в исходном наборе данных задействованы поля, используемые в выражении связи источника.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
