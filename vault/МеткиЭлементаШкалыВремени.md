Элементы коллекции:

[МеткаЭлементаШкалыВремени](МеткаЭлементаШкалыВремени.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются метки.  
Возможно обращение к меткам посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция непериодических меток элемента шкалы времени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.
