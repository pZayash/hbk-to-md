Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются части компоновки данных.  
Возможно обращение к части компоновки данных посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).
