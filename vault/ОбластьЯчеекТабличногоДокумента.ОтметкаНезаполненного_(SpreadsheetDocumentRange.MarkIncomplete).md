ОбластьЯчеекТабличногоДокумента (SpreadsheetDocumentRange)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Позволяет определить, выведена ли в области отметка незаполненного значения. [Истина](lang__def_BooleanTrue.md) - отметка выведена, [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
