ИспользованиеКопийБазыДанных (DatabaseCopiesUse)

ИспользоватьПреимущественноКопии (PreferUseCopies)

Доступен, начиная с версии 8.3.19.

Описание:

Использовать копии базы данных, если есть подходящие копии.   
В противном случае использовать основную базу данных.  
Выбор базы зависит от требуемой актуальности данных.

Использование в версии:

Доступен, начиная с версии 8.3.19.
