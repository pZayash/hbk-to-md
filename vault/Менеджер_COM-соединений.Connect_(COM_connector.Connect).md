Менеджер COM-соединений (COM connector)

Connect (Connect)

Доступен, начиная с версии 8.1.

Синтаксис:

Connect(<СтрокаСоединения>)

Параметры:

<СтрокаСоединения> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка параметров ([Строка соединения](lang__Строка_соединения.md)), используемая 1С:Предприятием для соединения с информационной базой.

Возвращаемое значение:

Тип: [Внешнее соединение](Внешнее_соединение_(External_connection).md).   

Описание:

Создает соединение с информационной базой 1С:Предприятия 8.  
Устанавливает соединение с информационной базой 1С:Предприятия 8 и возвращает ссылку на объект [Внешнее соединение](Внешнее_соединение_(External_connection).md).   
Установка соединения выполняется по следующему алгоритму:  
1. [Внешнее соединение](Внешнее_соединение_(External_connection).md) с требуемыми параметрами ищется в пуле соединений. Если найдено, то используется оно.  
2. Если [Внешнее соединение](Внешнее_соединение_(External_connection).md) с требуемыми параметрами не найдено, то анализируется не исчерпан ли лимит по числу одновременно существующих соединений. Если не исчерпан, то создается новое [Внешнее соединение](Внешнее_соединение_(External_connection).md).  
3. Если лимит исчерпан, но при этом в пуле имеются неиспользуемые [Внешнее соединение](Внешнее_соединение_(External_connection).md), то соединение, дольше всех находящееся в пуле, удаляется и создается новое [Внешнее соединение](Внешнее_соединение_(External_connection).md).  
4. Если никаким способом найти подходящее или создать новое [Внешнее соединение](Внешнее_соединение_(External_connection).md) не удалось, то происходит ожидание освобождения [Внешнее соединение](Внешнее_соединение_(External_connection).md) другим потоком, после чего весь процесс повторяется, начиная с пункта 1.

Доступность:

Интеграция.

Пример:

|  |
| --- |
| // Создается объект-коннектор  cntr = Новый COMObject("V83.COMConnector"); // создается объект COM-соединение  connection = cntr.Connect("File=""c:\InfoBases\Trade"";         Usr=""Director"";") |

Использование в версии:

Доступен, начиная с версии 8.1.
