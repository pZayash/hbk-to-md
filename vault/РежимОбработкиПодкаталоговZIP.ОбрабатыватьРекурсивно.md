Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [ОбрабатыватьРекурсивно](РежимОбработкиПодкаталоговФайлаАрхива.ОбрабатыватьРекурсивно.md)

Описание:

При задании маски в качестве пути для архивации подкаталоги также будут рекурсивно обработаны.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
