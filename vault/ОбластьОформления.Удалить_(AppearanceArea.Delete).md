ОбластьОформления (AppearanceArea)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [ЭлементОбластиОформления](ЭлементОбластиОформления_(AppearanceAreaItem).md).   
Удаляемое поле.

Описание:

Удаляет поле из коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
