Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаОписанияИсточникаДанных](КолонкаОписанияИсточникаДанных_(DataSourceDescriptionColumn).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки.  
Возможно обращение к колонке описания посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Свойства:

[\<Имя колонки\> (\<Column name\>)](КолонкиОписанияИсточникаДанных.Имя_колонки_(DataSourceDescriptionColumns.Column_name).md)

[Получить (Get)](КолонкиОписанияИсточникаДанных.Получить_(DataSourceDescriptionColumns.Get).md)  

Описание:

Содержит колонки описания источника данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
