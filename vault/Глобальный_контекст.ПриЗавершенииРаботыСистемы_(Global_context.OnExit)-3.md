Глобальный контекст (Global context)

ПриЗавершенииРаботыСистемы (OnExit)

Доступен, начиная с версии 8.2.

Синтаксис:

ПриЗавершенииРаботыСистемы()

Описание:

Возникает перед завершением работы в режиме приложения после закрытия главного окна. В данной процедуре могут быть выполнены действия, необходимые при выходе из программы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

В данной процедуре не допускаются:

- открытие форм и других окон,
- использование серверных вызовов,
- использование асинхронных вызовов.

Не поддерживаются:

- выдача сообщений,
- установка текста в панели состояния,
- другие действия, требующие наличия главного окна.

Доступен, начиная с версии 8.2.
