# ТабличноеПоле.ТекущиеДанные

ТабличноеПоле (TableBox)

ТекущиеДанные (CurrentData)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Содержит данные текущей строки табличного поля.

Доступность:

Толстый клиент.

Примечание:

Тип значения данных строки табличного поля приведен в описаниях расширений табличного поля у соответствующих объектов.  
Для табличных полей, отображающих иерархический список в виде дерева, свойство может принимать значение [Неопределено](lang__def_Undefined.md).

Использование в версии:

Доступен, начиная с версии 8.0.
