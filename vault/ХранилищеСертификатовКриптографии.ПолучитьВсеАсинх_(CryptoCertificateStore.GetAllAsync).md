ПолучитьВсеАсинх (GetAllAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьВсеАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив объектов типа [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Описание:

Асинхронная функция. Получает массив всех сертификатов криптографии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
