Расширение формы клиентского приложения для обработки (Client application form extension for processings)

ПолучитьНавигационнуюСсылкуОбработки (GetDataProcessorURL)

Доступен, начиная с версии 8.3.19.

Синтаксис:

ПолучитьНавигационнуюСсылкуОбработки(<ПредставлениеНавигационнойСсылки>, <ВнешниеДанныеНавигационнойСсылки>)

Параметры:

<ПредставлениеНавигационнойСсылки> (необязательный)

Тип: [Строка](lang__def_String.md).   
Содержит представление, которое будет помещено в хранилище внешних данных навигационных ссылок.   
Если этот параметр указан, а <ВнешниеДанныеНавигационнойСсылки> – нет, будет вызвано исключение.

<ВнешниеДанныеНавигационнойСсылки> (необязательный)

Тип: [Структура](Структура_(Structure).md).   
Содержит значение, которое будет помещено в хранилище внешних данных навигационных ссылок. Если не указан, то внешний параметр в навигационной ссылке создаваться не будет. Если данный параметр указан, а <ПредставлениеНавигационнойСсылки> – нет, будет вызвано исключение.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Получает навигационную ссылку текущей обработки.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.19.
