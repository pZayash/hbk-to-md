ДокументМенеджер.<Имя документа> (DocumentManager.<Document name>)

ВыбратьПоСсылкам (SelectByRefs)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ВыбратьПоСсылкам(<Источник>, <Колонка>)

Параметры:

<Источник> (обязательный)

Тип: [ТаблицаЗначений](ТаблицаЗначений.md), [РезультатЗапроса](РезультатЗапроса.md), [Массив](Массив.md), [ФиксированныйМассив](ФиксированныйМассив.md).   
Источник ссылок на документы.

<Колонка> (необязательный)

Тип: Число, Строка, Неопределено.   
Колонка источника ссылок. Число представляет индекс колонки источника. Строка представляет имя колонки. Если значением параметра является Неопределено, то используется первая колонка источника. Значение данного параметра игнорируется, если понятие "колонка" не имеет смысла для источника. Например, если в качестве источника используется [Массив](Массив.md).  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: [ДокументВыборка.\<Имя документа\>](ДокументВыборка.Имя_документа.md).   

Описание:

Формирует выборку документов по заданному источнику ссылок на документы соответствующего типа. Если источник содержит значения, отличнные от ссылок на документы соответствующего типа, или ссылки на несуществующие документы, то такие значения игнорируются.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Запрос = Новый Запрос(); Запрос.Текст = "     |ВЫБРАТЬ     |    РасходТовара.Ссылка КАК Ссылка     |ИЗ     |    Документ.РасходТовара КАК РасходТовара     |"; Таблица = Запрос.Выполнить()     .Выгрузить(ОбходРезультатаЗапроса.Прямой); Выборка = Документы.РасходТовара.ВыбратьПоСсылкам(Таблица, 0); |

Использование в версии:

Доступен, начиная с версии 8.3.26.
