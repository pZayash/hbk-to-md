# СвязьПараметраВыбора.ИзменениеЗначения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Типы свойств](Типы_свойств.md) › [СвязьПараметраВыбора](СвязьПараметраВыбора_(ChoiceParameterLink).md) › [Свойства](СвязьПараметраВыбора__Свойства.md)

СвязьПараметраВыбора (ChoiceParameterLink)

ИзменениеЗначения (ValueChange)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [РежимИзмененияСвязанногоЗначения](РежимИзмененияСвязанногоЗначения_(LinkedValueChangeMode).md).   
Содержит способ изменения связанного значения при изменении ведущего значения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
