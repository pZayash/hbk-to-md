# Расширение элементов управления, расположенных в форме

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Форма клиентского приложения](Форма_клиентского_приложения.md)

Доступен, начиная с версии 8.2.

События:

[\<Имя события\> (\<Event name\>)](Расширение_элементов_управления,_расположенных_в_форме.Имя_события_(Extension_for_controls_located_in_a_form.Event_name).md)  

Описание:

Дополнительные свойства и методы элемента управления в форме. Эти свойства и методы появляются у любого элемента управления, расположенного на форме.  
  
ЗАМЕЧАНИЕ. При установке поля ввода или табличного поля в качестве владельца формы все свойства расширения становятся недоступны.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
