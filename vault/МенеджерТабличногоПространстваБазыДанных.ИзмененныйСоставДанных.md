Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [СоставТабличногоПространстваБазыДанных](СоставТабличногоПространстваБазыДанных.md).   
Состав объектов, данные таблиц которых будут физически располагаться в этом табличном пространстве после обновления размещения таблиц.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
