МенеджерWebSocketКлиентов (WebSocketClientsManager)

ПолучитьКлиентов (GetClients)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьКлиентов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
