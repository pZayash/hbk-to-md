XBase (XBase)

Ключ (Key)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КлючXBase](КлючXBase.md).   
Предоставляет доступ к объекту [КлючXBase](КлючXBase.md). Состав свойств объекта повторяет набор свойств полей объекта [XBase](XBase.md). Значения свойств используются для вычисления выражения индекса при использовании метода [НайтиПоКлючу](XBase.НайтиПоКлючу.md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[XBase](XBase.md), метод [НайтиПоКлючу](XBase.НайтиПоКлючу.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
