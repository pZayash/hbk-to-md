ПолеКалендаря (CalendarBox)

ЦветТекстаКнопки (ButtonTextColor)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Цвет](Цвет_(Color).md).   
Содержит цвет текста выбранной даты, заголовка календаря, кнопок перехода на предыдущий и последующий месяц и год, выпадающего списка для выбора стандартных дат.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
