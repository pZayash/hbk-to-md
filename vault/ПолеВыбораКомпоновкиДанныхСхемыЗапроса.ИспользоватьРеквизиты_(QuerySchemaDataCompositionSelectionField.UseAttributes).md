ПолеВыбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionSelectionField)

ИспользоватьРеквизиты (UseAttributes)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак использования подчиненных элементов для поля, указанного в выражении.  
Возможно установить значение [Истина](lang__def_BooleanTrue.md) только в том случае, если поле содержит колонку схемы запроса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
