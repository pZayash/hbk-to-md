ПараметрыЗаполненияПриПереоткрытииФормы (OnReopenFormFillParameters)

ИсключитьРеквизиты (ExcludedAttributes)

Доступен, начиная с версии 8.3.16.

Использование:

Только чтение.

Описание:

Тип: [Массив](Массив_(Array).md).   
Массив путей к реквизитам, для которых не будет выполнено копирование значений в новую форму.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.16.
