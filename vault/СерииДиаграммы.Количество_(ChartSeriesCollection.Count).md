СерииДиаграммы (ChartSeriesCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество серий диаграммы.

Доступность:

Веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| СерииДиаграммы = Диаграмма.Серии; КоличествоСерий = СерииДиаграммы.Количество(); Сообщить("Диаграмма содержит " + КоличествоСерий + " серий"); |

Использование в версии:

Доступен, начиная с версии 8.0.
