Глобальный контекст (Global context)

ПолучитьМаскуВсеФайлы (GetAllFilesMask)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПолучитьМаскуВсеФайлы()

Возвращаемое значение:

Тип: Строка.   
Маска возвращается в виде строки. Например: "\*.\*".

Описание:

Получает файловую маску, принятую для выбора всех файлов в операционной системе, в контексте которой выполняется вызов.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
