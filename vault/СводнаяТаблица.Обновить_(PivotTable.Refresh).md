СводнаяТаблица (PivotTable)

Обновить (Refresh)

Доступен, начиная с версии 8.0.

Синтаксис:

Обновить()

Описание:

Обновляет содержимое сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
