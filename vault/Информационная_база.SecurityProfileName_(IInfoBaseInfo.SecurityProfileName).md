Информационная база (IInfoBaseInfo)

SecurityProfileName (SecurityProfileName)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Содержит имя профиля безопасности информационной базы. Если ссылка не указана, то информационная база имеет полные права на внешнюю активность. Если указано имя профиля безопасности, то поведение информационной базы регламентируется этим профилем.

Доступность:

Интеграция.

Примечание:

Если профиль с заданным именем не найден, то внешняя активность запрещена.

Использование в версии:

Доступен, начиная с версии 8.3.3.
