# ИзмеренияПостроителяОтчета.Вставить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Построитель отчета](Построитель_отчета-2.md) › [ИзмеренияПостроителяОтчета](ИзмеренияПостроителяОтчета_(ReportBuilderDimensions).md) › [Методы](ИзмеренияПостроителяОтчета__Методы.md)

ИзмеренияПостроителяОтчета (ReportBuilderDimensions)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<ПутьКДанным>, <Имя>, <Тип>, <Область>, <ОбластьИерархия>, <Позиция>)

Параметры:

<ПутьКДанным> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к данным для измерения.

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя измерения.

<Тип> (необязательный)

Тип: [ТипИзмеренияПостроителяОтчета](ТипИзмеренияПостроителяОтчета_(ReportBuilderDimensionType).md).   
Тип измерения.

<Область> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Область для вывода.

<ОбластьИерархия> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент_(SpreadsheetDocument).md), [ПолеТабличногоДокумента](ПолеТабличногоДокумента_(SpreadsheetDocumentField).md).   
Область для иерархического вывода.

<Позиция> (обязательный)

Тип: [Число](lang__def_Number.md).   
Позиция, в которую необходимо вставить измерение.

Возвращаемое значение:

Тип: [ИзмерениеПостроителяОтчета](ИзмерениеПостроителяОтчета_(ReportBuilderDimension).md).   

Описание:

Вставляет измерение в указанную позицию.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| СтрПутьКДанным = ПостроительОтчета.ИзмеренияКолонки[Индекс].ПутьКДанным; Измерение = ПостроительОтчета.ИзмеренияКолонки[Индекс]; ПостроительОтчета.ИзмеренияКолонки.Удалить(Измерение); ПостроительОтчета.ИзмеренияКолонки.Вставить(СтрПутьКДанным,,,,,Индекс); |

Использование в версии:

Доступен, начиная с версии 8.0.
