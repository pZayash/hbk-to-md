Глобальный контекст (Global context)

ПолучитьПроверкуСложностиПаролейПользователей (GetUserPasswordStrengthCheck)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьПроверкуСложностиПаролейПользователей()

Возвращаемое значение:

Тип: Булево.   
Истина - ограничения установлены; Ложь - в противном случае.

Описание:

Проверяет, установлены ли ограничения на сложность паролей пользователей информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
