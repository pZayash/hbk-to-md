РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

АвтоРазмер (AutoSize)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Содержит признак автоматического расчета размера для рисунка.  
Истина - выполняется автоматический расчет размера, при котором весь текст рисунка вмещался.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применимо для рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст.md), [Примечание](ТипРисункаТабличногоДокумента.Примечание.md).

Использование в версии:

Доступен, начиная с версии 8.0.
