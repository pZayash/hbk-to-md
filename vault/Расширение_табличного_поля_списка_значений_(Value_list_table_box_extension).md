Свойства:

[ИзменятьПорядокСтрок (ChangeRowOrder)](Расширение_табличного_поля_списка_значений.ИзменятьПорядокСтрок_(Value_list_table_box_extension.ChangeRowOrder).md)  

Описание:

Дополнительные свойства элемента управления [ТабличноеПоле](ТабличноеПоле_(TableBox).md), предназначенные для интерактивной работы со списком значений. Для данного расширения свойство [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md) имеет значение типа [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
