ГеографическаяСхема (GeographicalSchema)

НайтиПоРасположению (FindByLocation)

Доступен, начиная с версии 8.0.

Синтаксис:

НайтиПоРасположению(<Объект>, <ГдеИскать>, <ТипПоиска>)

Параметры:

<Объект> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   
Объект поиска.

<ГдеИскать> (обязательный)

Тип: [Массив](Массив_(Array).md).   
Массив может содержать объекты следующих типов:  
[ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md), [ГеографическиеКоординаты](ГеографическиеКоординаты_(GeographicCoordinates).md),   
[ПрямоугольникГеографическойСхемы](ПрямоугольникГеографическойСхемы_(GeographicalSchemaRectangle).md).

<ТипПоиска> (обязательный)

Тип: [ТипПоискаОбъектовГеографическойСхемы](ТипПоискаОбъектовГеографическойСхемы_(GeographicalSchemaObjectFindType).md).   
Определяет способ поиска объекта географической схемы.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Проверяет объекты по взаимному расположению.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
