# ЧтениеДанных.РазделитьНаЧастиПо

ЧтениеДанных (DataReader)

РазделитьНаЧастиПо (SplitInPartsOf)

Доступен, начиная с версии 8.3.9.

Синтаксис:

РазделитьНаЧастиПо(<РазмерЧасти>)

Параметры:

<РазмерЧасти> (обязательный)

Тип: [Число](lang__def_Number.md).   
Размер части данных в байтах.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив значений типа [РезультатЧтенияДанных](РезультатЧтенияДанных_(ReadDataResult).md).

Описание:

Разделяет остаток данных на части заданного размера. Размер части указывается в байтах.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ЧтениеДанных](ЧтениеДанных_(DataReader).md), метод [РазделитьНаЧастиПоАсинх](ЧтениеДанных.РазделитьНаЧастиПоАсинх_(DataReader.SplitInPartsOfAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
