# ЧтениеXML.ПолучитьАтрибут

ЧтениеXML (XMLReader)

ПолучитьАтрибут (GetAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По номеру

Синтаксис:

ПолучитьАтрибут(<НомерАтрибута>)

Параметры:

<НомерАтрибута> (обязательный)

Тип: [Число](lang__def_Number.md).   
Порядковый номер атрибута, начиная с 0.

Вариант синтаксиса: По полному имени

Синтаксис:

ПолучитьАтрибут(<ПолноеИмяАтрибута>)

Параметры:

<ПолноеИмяАтрибута> (обязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя атрибута, включающее префикс.

Вариант синтаксиса: По локальному имени и пространству имен

Синтаксис:

ПолучитьАтрибут(<ЛокальноеИмяАтрибута>, <URIПространстваИмен>)

Параметры:

<ЛокальноеИмяАтрибута> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута, не включая префикс.

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен для локального имени атрибута.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Если атрибут найден, то метод возвращает строку, представляющую значение атрибута. Если атрибут в соответствии с заданными параметрами не найден, то метод возвращает значение [Неопределено](lang__def_Undefined.md).

Описание:

Получает значение атрибута.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Чтение.ОткрытьФайл(ИмяФайла); Попытка     ИдГруппы = Чтение.ПолучитьАтрибут("ИдГруппы"); Исключение КонецПопытки; |

Использование в версии:

Доступен, начиная с версии 8.1.
