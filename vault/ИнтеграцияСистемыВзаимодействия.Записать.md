ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

Записать (Write)

Доступен, начиная с версии 8.3.17.

Синтаксис:

Записать()

Описание:

Создает новую интеграцию или обновляет существующую.  
Требуется наличие права "Администрирование данных" или привилегированный режим.  
Если интеграция получена без права "Администрирование данных" (или не в привилегированном режиме), то запись невозможна.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.17.
