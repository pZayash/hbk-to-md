ИнтернетТекстПочтовогоСообщения (InternetMailText)

ПроизвольныйТипТекста (CustomTextType)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Текстовые данные сообщения.  
Если при отправке тип текста у [ИнтернетТекстПочтовогоСообщения](ИнтернетТекстПочтовогоСообщения_(InternetMailText).md) установлен в [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст_(InternetMailTextType.CustomText).md), то содержимое поля Content-Type для отправляемого письма будет получено из данного свойства.  
Если в получаемом сообщении не удалось определить тип текста, то свойство [ТипТекста](ИнтернетТекстПочтовогоСообщения.ТипТекста_(InternetMailText.TextType).md) принимает значение [ПроизвольныйТекст](ТипТекстаПочтовогоСообщения.ПроизвольныйТекст_(InternetMailTextType.CustomText).md). Но в любом случае, данное свойство при чтении, будет содержать актуальный Content-Type полученного сообщения.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.
