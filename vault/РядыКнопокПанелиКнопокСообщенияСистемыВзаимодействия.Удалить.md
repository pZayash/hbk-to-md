РядыКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRows)

Удалить (Delete)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс удаляемого ряда кнопок.

Описание:

Удаляет элемент из коллекции по индексу.   
Индексация начинается с 0.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
