Элементы коллекции:

[ЭлементЛегендыГеографическойСхемы](ЭлементЛегендыГеографическойСхемы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0) элемента.

Описание:

Содержит коллекцию элементов легенды географической схемы. Элементы отрисовываются в том порядке, в котором они расположены в коллекции.

Доступность:

Сервер, толстый клиент, внешнее соединение.
