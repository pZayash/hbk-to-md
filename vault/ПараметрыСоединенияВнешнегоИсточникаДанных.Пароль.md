ПараметрыСоединенияВнешнегоИсточникаДанных (ExternalDataSourceConnectionParameters)

Пароль (Password)

Доступен, начиная с версии 8.2.

Использование:

Только запись.

Описание:

Тип: Строка, Неопределено.   
Содержит пароль, используемый при стандартной аутентификации.  
Если содержит Неопределено, то получается из родительских параметров соединения.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
