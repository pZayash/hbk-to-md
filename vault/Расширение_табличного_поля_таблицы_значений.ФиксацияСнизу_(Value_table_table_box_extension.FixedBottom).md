Расширение табличного поля таблицы значений (Value table table box extension)

ФиксацияСнизу (FixedBottom)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит количество строк таблицы значений, фиксированных снизу, то есть не прокручиваемых при прокрутке табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
