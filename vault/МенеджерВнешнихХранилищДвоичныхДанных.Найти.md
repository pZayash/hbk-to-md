МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Найти (Find)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя внешнего хранилища двоичных данных.

Возвращаемое значение:

Тип: Неопределено, [МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.md).   
Если менеджер существует и не удален, вернет [МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных.md), иначе - Неопределено.

Описание:

Возвращает внешнее хранилище по имени.

Доступность:

Сервер.
