МенеджерКриптографии (CryptoManager)

ПолучитьСертификатыИзПодписиАсинх (GetCertificatesFromSignatureAsync)

Не рекомендуется использовать, начиная с версии 8.3.24.

Рекомендуется использовать:

- [ПолучитьСертификаты](МенеджерКриптографии.ПолучитьСертификаты_(CryptoManager.GetCertificates).md)
- [ПолучитьСертификатыАсинх](МенеджерКриптографии.ПолучитьСертификатыАсинх_(CryptoManager.GetCertificatesAsync).md)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьСертификатыИзПодписиАсинх(<ИсходныеДанные>)

Параметры:

<ИсходныеДанные> (обязательный)

Тип: [Строка](lang__def_String.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Исходные данные, из которых будет извлечен сертификат.  
Данные могут размещаться в файле (в этом случае указывается имя файла) или представлены как [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - массив объектов [СертификатКриптографии](СертификатКриптографии_(CryptoCertificate).md).

Описание:

Асинхронная функция. Извлекает массив сертификатов из данных подписи.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

[МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md), метод [ПолучитьСертификатыИзПодписи](МенеджерКриптографии.ПолучитьСертификатыИзПодписи_(CryptoManager.GetCertificatesFromSignature).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.24.

Описание изменено в версии 8.3.24.
