# ДопустимаяДлина

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Разные](Разные.md)

Доступен, начиная с версии 8.0.

Значения

[Переменная (Variable)](ДопустимаяДлина.Переменная_(AllowedLength.Variable).md)  
[Фиксированная (Fixed)](ДопустимаяДлина.Фиксированная_(AllowedLength.Fixed).md)  

Описание:

Определяет варианты ограничений длины строки. Используется для определения свойства [ДопустимаяДлина](КвалификаторыСтроки.ДопустимаяДлина_(StringQualifiers.AllowedLength).md) и в конструкторе объекта [КвалификаторыСтроки](КвалификаторыСтроки_(StringQualifiers).md), если задается длина строки отличная от нуля.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером. Данный объект может быть сериализован в/из XML. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.1/data/core}. Имя типа XDTO: AllowedLength.
