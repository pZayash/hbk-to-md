БотСистемыВзаимодействия (CollaborationSystemBot)

Удалить (Delete)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Удалить()

Описание:

Удаляет бота из информационной базы.  
Если бот не существует (создан методом [СоздатьБота](МенеджерСистемыВзаимодействия.СоздатьБота_(CollaborationSystemManager.CreateBot).md) и не записан), то вызов данного метода игнорируется.  
Требует наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
