Описание:

Объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы:

- В ОС Windows работает полностью аналогично объекту [СертификатКлиентаWindows](СертификатКлиентаWindows_(WindowsClientCertificate).md).
- В ОС Linux работает полностью аналогично объекту [СертификатКлиентаLinux](СертификатКлиентаLinux_(LinuxClientCertificate).md).
- В ОС macOS работает полностью аналогично объекту [СертификатКлиентаMacOS](СертификатКлиентаMacOS_(MacOSClientCertificate).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
