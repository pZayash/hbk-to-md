МоментВремениСУточнениемПериода (PointInTimeWithPeriodAdjustment)

Сравнить (Compare)

Доступен, начиная с версии 8.3.9.

Синтаксис:

Сравнить(<МоментВремениСУточнениемПериода>)

Параметры:

<МоментВремениСУточнениемПериода> (обязательный)

Момент времени с уточнением периода, с которым выполняется сравнение.

Возвращаемое значение:

Тип: Число.   
Возвращает Число:

- -1 - если первый момент времени с уточнением периода меньше второго момента времени с уточнением периода,
- 0 - если первый момент времени с уточнением периода равен второму моменту времени с уточнением периода,
- 1 - если первый момент времени с уточнением периода больше второго момента времени с уточнением периода.

Описание:

Выполняет сравнение одного момента времени с уточнением периода со другим моментом времени с уточнением периода.  
Сначала сравнение выполняется по дате. Если они равны, сравниваются уточнения периодов. Если они также равны – выполняется сравнение по ссылке.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Момент1 = Новый МоментВремениСУточнениемПериода(Док1.Дата, 0, Док1); Момент2 = Новый МоментВремениСУточнениемПериода(Док2.Дата, 0, Док2); Если Момент1.Сравнить(Момент2) = -1 Тогда     Момент1 = Момент2; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.3.9.
