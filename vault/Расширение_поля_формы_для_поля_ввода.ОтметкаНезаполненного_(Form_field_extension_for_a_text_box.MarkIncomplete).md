# Расширение поля формы для поля ввода.ОтметкаНезаполненного

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Поле формы](Поле_формы.md) › [Расширение поля формы для поля ввода](Расширение_поля_формы_для_поля_ввода_(Form_field_extension_for_a_text_box).md) › [Свойства](Расширение_поля_формы_для_поля_ввода__Свойства.md)

Расширение поля формы для поля ввода (Form field extension for a text box)

ОтметкаНезаполненного (MarkIncomplete)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает, нужно ли сейчас в поле ввода отображать особую отметку незаполненного или нет. Если свойство имеет значение [Истина](lang__def_BooleanTrue.md), то независимо от хранимых в поле ввода данных будет отображаться отметка, если [Ложь](lang__def_BooleanFalse.md), то независимо от хранимых в поле ввода данных отметка отображаться не будет.  
При начале редактирования свойство сбрасывается, даже если [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного_(Form_field_extension_for_a_text_box.AutoMarkIncomplete).md) равно [Ложь](lang__def_BooleanFalse.md).  
  
Если свойство [АвтоОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.АвтоОтметкаНезаполненного_(Form_field_extension_for_a_text_box.AutoMarkIncomplete).md) равно [Истина](lang__def_BooleanTrue.md), то при установке и формировании значения в поле ввода будет меняться состояние свойства [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного_(TextBox.MarkIncomplete).md): если значение в поле ввода эквивалентно значению по умолчанию текущего типа, то [ОтметкаНезаполненного](ПолеВвода.ОтметкаНезаполненного_(TextBox.MarkIncomplete).md) будет установлено [Истина](lang__def_BooleanTrue.md), иначе - [Ложь](lang__def_BooleanFalse.md).  
Если для поля в свойствах метаданных "Автоотметка незаполненного" установлено "Авто":

- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Выдавать ошибку", то свойство принимает значение [Истина](lang__def_BooleanTrue.md), и соответственно, свойство [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного_(Form_field_extension_for_a_text_box.MarkIncomplete).md) зависит от того, является ли значение реквизита значением по умолчанию;
- если поле ввода связано с реквизитом, у которого в свойстве метаданных "Проверка заполнения" установлено "Не проверять", то свойство [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного_(Form_field_extension_for_a_text_box.MarkIncomplete).md) принимает значение [Ложь](lang__def_BooleanFalse.md), и соответственно, [ОтметкаНезаполненного](Расширение_поля_формы_для_поля_ввода.ОтметкаНезаполненного_(Form_field_extension_for_a_text_box.MarkIncomplete).md) не зависит от того, является ли значение реквизита значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
