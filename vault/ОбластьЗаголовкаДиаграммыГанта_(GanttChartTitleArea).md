Описание:

Используется для доступа к области заголовка диаграммы Ганта для настройки параметров оформления. В области заголовка может содержаться название диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

См. также:

[ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md), свойство [ОбластьЗаголовка](ДиаграммаГанта.ОбластьЗаголовка_(GanttChart.TitleArea).md)
