ОбъектМетаданных: ТабличнаяЧасть (MetadataObject: TabularSection)

Использование (Use)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [ИспользованиеРеквизита](ИспользованиеРеквизита.md), Булево.   
Для табличных частей справочников и планов видов характеристик это свойство указывает способ использования данных подчиненного объекта конфигурации (например, [ДляЭлемента](ИспользованиеРеквизита.ДляЭлемента.md), [ДляГруппы](ИспользованиеРеквизита.ДляГруппы.md), [ДляГруппыИЭлемента](ИспользованиеРеквизита.ДляГруппыИЭлемента.md)).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для табличных частей справочников и планов видов характеристик.

Использование в версии:

Доступен, начиная с версии 8.1.
