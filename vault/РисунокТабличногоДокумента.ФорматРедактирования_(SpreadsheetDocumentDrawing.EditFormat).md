РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ФорматРедактирования (EditFormat)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Формат, используемый для редактирования значения.  
Применимо только рисунков типа [Текст](ТипРисункаТабличногоДокумента.Текст_(SpreadsheetDocumentDrawingType.Text).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
