ЭлементПланаГлобальногоПоиска (GlobalSearchPlanItem)

По стандартному виду поиска

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ЭлементПланаГлобальногоПоиска(<ВидПоиска>, <Порядок>)

Параметры:

<ВидПоиска> (обязательный)

Тип: [СтандартныйВидГлобальногоПоиска](СтандартныйВидГлобальногоПоиска.md).   
Вид стандартного поиска.

<Порядок> (необязательный)

Тип: Число.   
Порядок выдачи результатов поиска.  
Значение по умолчанию: 0.

Описание:

Создает элемент плана глобального поиска по стандартному виду поиска.

Использование в версии:

Доступен, начиная с версии 8.3.15.
