ПолеФормы (FormField)

ОтображениеПредупрежденияПриРедактировании (WarningOnEditRepresentation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеПредупрежденияПриРедактировании](ОтображениеПредупрежденияПриРедактировании.md).   
Содержит режим отображения предупреждения перед началом редактирования.  
Если для поля выдается предупреждение при редактировании, то поле отображается с фоном, аналогичным фону поля с режимом [ТолькоПросмотр](ПолеФормы.ТолькоПросмотр.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
