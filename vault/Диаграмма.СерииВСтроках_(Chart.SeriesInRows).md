Диаграмма (Chart)

СерииВСтроках (SeriesInRows)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак размещения серий диаграммы в строках.  
[Истина](lang__def_BooleanTrue.md) - размещаются в строках.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Используется только при работе диаграммы с источником данных.

Использование в версии:

Доступен, начиная с версии 8.0.
