ГруппаФормы (FormGroup)

РазрешитьИзменениеСостава (EnableContentChange)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Управляет возможностью настройки состава группы. Ложь - запрещает пользователю менять состав группы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
