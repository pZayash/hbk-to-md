СерииДиаграммыГанта (GanttChartSeriesCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество серий диаграммы Ганта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для Н = 0 По Диаграмма.Серии.Количество() - 1 Цикл     // обработка серии  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
