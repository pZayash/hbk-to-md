ДействиеСообщенияСистемыВзаимодействия (CollaborationSystemMessageAction)

Текст (Text)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Текст](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия.Текст_(CollaborationSystemMessageButtonPanelButton.Text).md)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md), [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Содержит текст, отображаемый в действии сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если не задан ни <Текст>, ни <Картинка>, то в качестве текста гиперссылки выводится представление <Значения>.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
