Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Разрешено использование произвольных внешних компонент.

- Истина – разрешено использование произвольных внешних компонент.
- Ложь – допустимо использование только внешних компонентов, перечисленных в списке разрешенных внешних компонент профиля.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
