СредстваМультимедиа (MultimediaTools)

СделатьФотоснимок (MakePhoto)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СделатьФотоснимок(<ТипКамеры>, <Разрешение>, <Качество>, <ЧерноБелый>, <ТипПодсветки>, <Отметка>, <АвтоПоворот>)

Параметры:

<ТипКамеры> (необязательный)

Тип: [ТипКамерыУстройства](ТипКамерыУстройства_(DeviceCameraType).md).   
Показывает, какой камерой будет сделан снимок.  
Значение по умолчанию: [Авто](ТипКамерыУстройства.Авто_(DeviceCameraType.Auto).md).

<Разрешение> (необязательный)

Тип: [РазрешениеКамерыУстройства](РазрешениеКамерыУстройства_(DeviceCameraResolution).md).   
Определяет разрешение, с которым будет сделан снимок.  
Значение по умолчанию: 0;0 (параметр не будет использован при открытии камеры):

- если задано разрешение, не поддерживаемое устройством, будет использован ближайшее подходящее разрешение.
- если задано отрицательное разрешение, то будет инициировано исключение.

В устройствах под управлением iOS параметр игнорируется.

<Качество> (необязательный)

Тип: [Число](lang__def_Number.md).   
Определяет качество снимка в процентах.  
Допустимый интервал значений: 1 - 100.  
По умолчанию - 100 % (съемка в максимальном качестве).  
Если значение не входит в допустимый интервал, будет вызвано исключение.

<ЧерноБелый> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр установлен в значение "Да" ([Истина](lang__def_BooleanTrue.md)), фотоснимок будет сделан монохромным. В противном случае, монохромность не используется.  
По умолчанию - "Нет".  
В устройствах под управлением iOS при создании фотоснимка эффект монохромности на камеру не накладывается, но фотография будет получена в монохромном формате.

<ТипПодсветки> (необязательный)

Тип: [ТипПодсветкиКамеры](ТипПодсветкиКамеры_(CameraLightingType).md).   
Определяет начальное использование подсветки камеры во время создания снимка.  
Значение по умолчанию: [Авто](ТипПодсветкиКамеры.Авто_(CameraLightingType.Auto).md).

<Отметка> (необязательный)

Тип: [ОтметкаНаФотоснимке](ОтметкаНаФотоснимке_(PhotoStamp).md).   
Если передано значение [Неопределено](lang__def_Undefined.md), то на фото не будет отображена подпись. Если передан объект типа [ОтметкаНаФотоснимке](ОтметкаНаФотоснимке_(PhotoStamp).md), то в нижнем правом углу фотографии (согласно ее ориентации) будет размещена подпись.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<АвтоПоворот> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Если параметр установлен в значение [Ложь](lang__def_BooleanFalse.md), фотоснимок останется в том виде, в каком его предоставила камера мобильного устройства. Если параметр установлен в значение [Истина](lang__def_BooleanTrue.md), полученный фотоснимок будет автоматически поворачиваться под текущее положение экрана устройства.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Возвращаемое значение:

Тип: [ДанныеМультимедиа](ДанныеМультимедиа_(MultimediaData).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - запись не выполнена.

Описание:

Выполняет фотоснимок встроенной фото-видео камерой.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок_(MultimediaTools.PhotoSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваМультимедиа](СредстваМультимедиа_(MultimediaTools).md), метод [ПоддерживаетсяФотоснимок](СредстваМультимедиа.ПоддерживаетсяФотоснимок_(MultimediaTools.PhotoSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.3.
