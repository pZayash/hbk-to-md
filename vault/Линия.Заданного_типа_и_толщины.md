Линия (Line)

Заданного типа и толщины

Доступен, начиная с версии 8.0.

Синтаксис:

Новый Линия(<ТипЛинии>, <Толщина>)

Параметры:

<ТипЛинии> (обязательный)

Тип: [ТипЛинииРисункаТабличногоДокумента](ТипЛинииРисункаТабличногоДокумента_(SpreadsheetDocumentDrawingLineType).md), [ТипЛинииДиаграммы](ТипЛинииДиаграммы_(ChartLineType).md), [ТипЛинииГеографическойСхемы](ТипЛинииГеографическойСхемы_(GeographicalSchemaLineType).md).   
Тип линии. Тип параметра зависит от того, в каком объекте будет использоваться линия.  
Если выбран тип [ТипЛинииЯчейкиТабличногоДокумента](ТипЛинииЯчейкиТабличногоДокумента_(SpreadsheetDocumentCellLineType).md), то толщина линии может быть только 1.

<Толщина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Толщина линии (в логических единицах). Если в параметре <ТипЛинии> указано значение НетЛинии, то максимальная толщина линии - 1.

Описание:

Создает линию заданного типа и толщины.

Пример:

|  |
| --- |
| Линия = Новый Линия(ТипЛинииЯчейкиТабличногоДокумента.Сплошная,2); |

Использование в версии:

Доступен, начиная с версии 8.0.
