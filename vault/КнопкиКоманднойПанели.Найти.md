КнопкиКоманднойПанели (CommandBarButtons)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяКнопки>)

Параметры:

<ИмяКнопки> (обязательный)

Тип: Строка.   
Наименование кнопки.

Возвращаемое значение:

Тип: [КнопкаКоманднойПанели](КнопкаКоманднойПанели.md), Неопределено.   
Если кнопка с указанным именем отсутствует, то возвращается значение Неопределено.

Описание:

Осуществляет поиск кнопки командной панели по ее имени.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
