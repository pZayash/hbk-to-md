МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьДляЗаписи (OpenForWrite)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ОткрытьДляЗаписи(<ИмяФайла>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя открываемого файла.

<РазмерБуфера> (необязательный)

Тип: Число.   
Размер буфера для операций с файлом (в байтах).  
При записи в поток данные сначала пишутся в буфер, а при заполнении буфера - данные перемещаются на диск.  
Целое число больше 0, максимальный размер 2^32-1.

Возвращаемое значение:

Тип: [ФайловыйПоток](ФайловыйПоток.md).   

Описание:

Открывает существующий файл для записи. Файл открывается в режиме разделяемого чтения. Если файл не найден, будет создан новый файл. Запись в существующий файл производится с начала файла, замещая ранее сохраненные данные.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.
