# ТипФайлаФорматированногоДокумента

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Системные перечисления](Системные_перечисления.md) › [Форматированный документ](Форматированный_документ.md)

Доступен, начиная с версии 8.2.

Значения

[ANSITXT (ANSITXT)](ТипФайлаФорматированногоДокумента.ANSITXT_(FormattedDocumentFileType.ANSITXT).md)  
[HTML (HTML)](ТипФайлаФорматированногоДокумента.HTML_(FormattedDocumentFileType.HTML).md)  
[PDF (PDF)](ТипФайлаФорматированногоДокумента.PDF_(FormattedDocumentFileType.PDF).md)  
[TXT (TXT)](ТипФайлаФорматированногоДокумента.TXT_(FormattedDocumentFileType.TXT).md)  

Описание:

Содержит варианты типов файлов форматированного документа.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/formatted-document}. Имя типа XDTO: FormattedDocumentFileType.

Использование в версии:

Доступен, начиная с версии 8.2.
