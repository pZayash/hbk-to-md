ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ВыполнитьЗадачуИнтерактивно (ExecuteTaskInteractively)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыполнитьЗадачуИнтерактивно()

Описание:

Интерактивно выполняет задачу с вызовом события [ПередИнтерактивнымВыполнением](ЗадачаОбъект.Имя_задачи.ПередИнтерактивнымВыполнением.md) у задачи и соответствующей ей точке маршрута. Для задачи свойство Выполнена устанавливается равным Истина и она записывается. При выполнении этого метода бизнес-процесс, породивший задачу, получает соответствующую нотификацию и выполняет переход к следующей точке маршрута бизнес-процесса.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
