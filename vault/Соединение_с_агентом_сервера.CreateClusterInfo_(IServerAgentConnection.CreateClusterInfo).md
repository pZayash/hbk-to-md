Соединение с агентом сервера (IServerAgentConnection)

CreateClusterInfo (CreateClusterInfo)

Доступен, начиная с версии 8.1.

Синтаксис:

CreateClusterInfo()

Возвращаемое значение:

Тип: [Кластер серверов](Кластер_серверов_(IClusterInfo).md).   

Описание:

Создает новый кластер. Все его свойства содержат пустые строки, кроме свойства [MainPort](Кластер_серверов.MainPort_(IClusterInfo.MainPort).md), которое устанавливается в значение по умолчанию (1541).   
Новый кластер может быть использован, например, для регистрации кластера. Для этого ему должны быть явно заданы необходимые значения свойств.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.1.
