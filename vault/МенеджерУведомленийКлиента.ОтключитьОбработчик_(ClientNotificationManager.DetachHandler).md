МенеджерУведомленийКлиента (ClientNotificationManager)

ОтключитьОбработчик (DetachHandler)

Доступен, начиная с версии 8.3.26.

Синтаксис:

ОтключитьОбработчик(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: [Неопределено](lang__def_Undefined.md), [Строка](lang__def_String.md).   
Ключ обработчика.  
Если [Неопределено](lang__def_Undefined.md), будут отключены все обработчики.

Описание:

Отключает обработчик уведомлений клиента.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.26.

Описание изменено в версии 8.3.27.
