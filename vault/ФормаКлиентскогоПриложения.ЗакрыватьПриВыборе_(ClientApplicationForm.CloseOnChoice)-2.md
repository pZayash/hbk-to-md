ФормаКлиентскогоПриложения (ClientApplicationForm)

ЗакрыватьПриВыборе (CloseOnChoice)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак того, что форму необходимо закрывать после осуществления выбора. Если свойство имеет значение [Ложь](lang__def_BooleanFalse.md), можно использовать форму для выбора нескольких позиций или элементов.   
По умолчанию свойство имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
