РегиональныеНастройкиСеанса (SessionRegionalSettings)

КодЛокализации (LocaleCode)

Доступен, начиная с версии 8.3.20.

Использование:

Только чтение.

Описание:

Тип: [Строка](lang__def_String.md).   
Один из возможных кодов локализации. Используется при получении представлений значений типа [Число](lang__def_Number.md), [Дата](lang__def_Date.md), [Булево](lang__Булево_(Boolean).md), при выполнении методов [ЧислоПрописью](Глобальный_контекст.ЧислоПрописью_(Global_context.NumberInWords).md) и [ПредставлениеПериода](Глобальный_контекст.ПредставлениеПериода_(Global_context.PeriodPresentation).md). Допустимые коды локализации могут быть получены методом глобального контекста [ПолучитьДопустимыеКодыЛокализации](Глобальный_контекст.ПолучитьДопустимыеКодыЛокализации_(Global_context.GetAvailableLocaleCodes).md). Представления кодов локализации можно получить методом глобального контекста [ПредставлениеКодаЛокализации](Глобальный_контекст.ПредставлениеКодаЛокализации_(Global_context.LocaleCodePresentation).md).

Доступность:

Сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.20.
