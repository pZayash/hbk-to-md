Надпись (Label)

Нажатие (Click)

Доступен, начиная с версии 8.0.

Синтаксис:

Нажатие()

Описание:

Возникает при нажатии на надпись, если у надписи свойство [Гиперссылка](Надпись.Гиперссылка_(Label.Hyperlink).md) имеет значение [Истина](lang__def_BooleanTrue.md).

Доступность:

Толстый клиент.

См. также:

[Надпись](Надпись_(Label).md), свойство [Гиперссылка](Надпись.Гиперссылка_(Label.Hyperlink).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
