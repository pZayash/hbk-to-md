МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьНавигационнуюСсылкуВнешнегоДоступа (GetExternalAccessURL)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ПолучитьНавигационнуюСсылкуВнешнегоДоступа(<Пользователь>, <Обсуждение>)

Параметры:

<Пользователь> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия_(CollaborationSystemUserID).md).   
Внешний пользователь, для которого формируется ссылка внешнего доступа.

<Обсуждение> (необязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Если параметр задан, то возвращается ссылка, при переходе по которой, будет открыто указанное обсуждение.

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   

Описание:

Возвращает навигационную ссылку-приглашение для внешнего доступа.  
Если переданный пользователь не является внешним (свойство [Внешний](ПользовательСистемыВзаимодействия.Внешний_(CollaborationSystemUser.External).md) имеет значение [Ложь](lang__def_BooleanFalse.md)), то возвращается пустая строка.  
Требуется наличие права "Администрирование данных" у текущего пользователя.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение.

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [ОтправитьНавигационнуюСсылкуВнешнегоДоступа](МенеджерСистемыВзаимодействия.ОтправитьНавигационнуюСсылкуВнешнегоДоступа_(CollaborationSystemManager.SendExternalAccessURL).md)  

Использование в версии:

Доступен, начиная с версии 8.3.20.
