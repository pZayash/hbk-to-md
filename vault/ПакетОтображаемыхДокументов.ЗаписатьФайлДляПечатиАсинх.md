ПакетОтображаемыхДокументов (RepresentableDocumentBatch)

ЗаписатьФайлДляПечатиАсинх (WriteFileForPrintingAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ЗаписатьФайлДляПечатиАсинх(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Содержит полное имя файла, в который нужно записать документы.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания:

- Истина - если запись была инициирована с помощью расширений для браузера и пользователь подтвердил его получение, то есть запись файла была завершена.
- Ложь - если запись была инициирована с помощью расширений для браузера, но пользователь не подтвердил его получение, то есть запись файла не была завершена.
- Неопределено - если запись была инициирована скачиванием файла средствами браузера и неизвестно, был ли файл записан или нет.

.

Описание:

Асинхронная функция. Записывает файл, который в дальнейшем можно использовать для печати пакета.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
