Расширение поля формы для поля текстового документа (Form field extension for a text document)

ПередПечатью (BeforePrint)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПередПечатью(<ДиалогПечати>, <Отказ>)

Параметры:

<ДиалогПечати>

Тип: [РежимИспользованияДиалогаПечати](РежимИспользованияДиалогаПечати.md).   
Способ вызова печати:

- [НеИспользовать](РежимИспользованияДиалогаПечати.НеИспользовать.md) - если выбрана команда "Печать с текущими настройками",
- [Использовать](РежимИспользованияДиалогаПечати.Использовать.md) - если выбрана команда "Печать".

<Отказ>

Тип: Булево.   
Признак отказа от печати. Если в теле процедуры-обработчика установить данному параметру значение Истина, то печать выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед интерактивной печатью объекта (при выборе пользователем команды "Печать" или "Печать с текущими настройками").

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.
