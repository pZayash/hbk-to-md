# Расширение поля ввода документа

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Документы](Документы.md)

Доступен, начиная с версии 8.0.

Описание:

Расширение предоставляет возможность автоматического поиска и подбора документа в элементе управления [ПолеВвода](ПолеВвода_(TextBox).md) по введенному тексту. При этом поиск осуществляется по полю "Номер".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
