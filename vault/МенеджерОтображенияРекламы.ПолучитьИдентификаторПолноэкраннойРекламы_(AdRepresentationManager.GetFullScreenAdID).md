ПолучитьИдентификаторПолноэкраннойРекламы (GetFullScreenAdID)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ПолучитьИдентификаторПолноэкраннойРекламы()

Возвращаемое значение:

Получает идентификатор рекламного блока, который используется для полноэкранной рекламы.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.8.
