СущностьDOM (DOMEntity)

ПервыйДочерний (FirstChild)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [СущностьDOM](СущностьDOM_(DOMEntity).md), [Неопределено](lang__def_Undefined.md).   
Первый дочерний узел DOM или [Неопределено](lang__def_Undefined.md), если дочерних узлов нет.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
