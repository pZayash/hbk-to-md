ПланОбменаСсылка.<Имя плана обмена> (ExchangePlanRef.<Exchange plan name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ПланОбменаОбъект.](ПланОбменаОбъект.Имя_плана_обмена.md)<[Имя плана обмена](ПланОбменаОбъект.Имя_плана_обмена.md)>, Неопределено.   
Неопределено - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект, предназначенный для чтения, изменения, добавления и удаления.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
