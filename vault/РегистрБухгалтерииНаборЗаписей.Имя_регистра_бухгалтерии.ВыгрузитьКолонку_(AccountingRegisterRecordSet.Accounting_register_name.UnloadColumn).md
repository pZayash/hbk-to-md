РегистрБухгалтерииНаборЗаписей.<Имя регистра бухгалтерии> (AccountingRegisterRecordSet.<Accounting register name>)

ВыгрузитьКолонку (UnloadColumn)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыгрузитьКолонку(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля записей набора, значения из которого будут выгружены в массив.

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   

Описание:

Создает массив и копирует в него значения заданного поля записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Значения записей выгружаются в порядке индексов.

Использование в версии:

Доступен, начиная с версии 8.0.
