Описание:

Используется для доступа к свойствам области легенды географической схемы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Схема.ОбластьЛегенды.Показывать = Истина; Схема.ОбластьЛегенды.Прозрачность = Истина; Схема.ОбластьЛегенды.МасштабнаяЛинейка = Ложь; |
