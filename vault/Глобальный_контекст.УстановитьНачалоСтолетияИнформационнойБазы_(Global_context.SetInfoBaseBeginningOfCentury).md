Глобальный контекст (Global context)

УстановитьНачалоСтолетияИнформационнойБазы (SetInfoBaseBeginningOfCentury)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

УстановитьНачалоСтолетияИнформационнойБазы(<НачалоСтолетия>)

Параметры:

<НачалоСтолетия> (необязательный)

Тип: [Число](lang__def_Number.md), [Неопределено](lang__def_Undefined.md).   
Устанавливаемое значение года начала столетия: число в диапазоне [1..9900] или [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Устанавливает новое значение начала столетия для информационной базы и сохраняет его в базе данных. Если параметр имеет значение [Неопределено](lang__def_Undefined.md), то установка начала столетия информационной базы отменяется. В этом случае в качестве начала столетия используется значение по умолчанию, равное 1950.   
Требует монопольного режима доступа к базе данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.21.
