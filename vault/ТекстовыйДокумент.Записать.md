ТекстовыйДокумент (TextDocument)

Записать (Write)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>, <ТипФайла>, <РазделительСтрок>)

<РазделительСтрок> (необязательный)

Тип: Строка.   
Задается строка, являющаяся разделителем строк, к которой при записи файла будет преобразован символ [ПС](Символы.ПС.md). Если строка пустая, то разделители строк при записи не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК.md);
- [ПС](Символы.ПС.md);
- [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

Описание варианта метода:

Запись текстового документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>, <ТипФайла>, <РазделительСтрок>, <ЗаписатьBOM>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток для записи документа.

<РазделительСтрок> (необязательный)

Тип: Строка.   
Задается строка, являющаяся разделителем строк, из которой при чтении файла будет сформирован символ [ПС](Символы.ПС.md).   
Если строка пустая, то разделители строк при чтении не конвертируются.  
Допустимые варианты разделителей строк:

- [ВК](Символы.ВК.md);
- [ПС](Символы.ПС.md);
- [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md) (значение по умолчанию - может не указываться).

При использовании других строк в качестве разделителей будет вызвано исключение.  
Значение по умолчанию: [ВК](Символы.ВК.md) + [ПС](Символы.ПС.md).

<ЗаписатьBOM> (необязательный)

Тип: Булево.   
Определяет, будет ли добавлена метка порядка байт (BOM) при сохранении документа.  
Значение по умолчанию: Ложь.

Описание варианта метода:

Запись текстового документа в поток.

Описание:

Записывает текстовый документ в файл с указанным именем.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

При работе на веб-клиенте вызов метода выполняет обращение к серверу.

| ТекстДок.Записать("price.txt"); |
