# ПерерасчетМенеджер.<Имя перерасчета>.СоздатьНаборЗаписей

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Регистры расчета](Регистры_расчета.md) › [Перерасчеты](Перерасчеты.md) › [ПерерасчетМенеджер.<Имя перерасчета>](ПерерасчетМенеджер.Имя_перерасчета_(RecalculationManager.Recalculation_name).md) › [Методы](ПерерасчетМенеджер.Имя_перерасчета__Методы.md)

ПерерасчетМенеджер.<Имя перерасчета> (RecalculationManager.<Recalculation name>)

СоздатьНаборЗаписей (CreateRecordSet)

Доступен, начиная с версии 8.0.

Синтаксис:

СоздатьНаборЗаписей()

Возвращаемое значение:

Тип: [ПерерасчетНаборЗаписей.\<Имя перерасчета\>](ПерерасчетНаборЗаписей.Имя_перерасчета_(RecalculationRecordSet.Recalculation_name).md).   

Описание:

Создает набор записей перерасчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Набор записей создается пустым.

Пример:

|  |
| --- |
| Регистр = РегистрыРасчета.ОсновныеНачисления.Перерасчеты; НаборЗаписей = Регистр.Перерасчет.СоздатьНаборЗаписей(); НаборЗаписей.Отбор.ОбъектПерерасчета.Значение = Документ; НаборЗаписей.Отбор.ФизЛицо.ВидСравнения = ВидСравнения.ВСписке; НаборЗаписей.Отбор.ФизЛицо.Значение = ФизЛицаСписок; НаборЗаписей.Отбор.ФизЛицо.Использование = Истина; НаборЗаписей.Записать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
