ЗаписьXML (XMLWriter)

УстановитьСтроку (SetString)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По типу кодировки

Синтаксис:

УстановитьСтроку(<ТипКодировки>)

Параметры:

<ТипКодировки> (необязательный)

Тип: [Строка](lang__def_String.md).   
В качестве типа кодировки может быть указано имя одной из возможных кодировок. Это никак не влияет на формируемую строку. Но если будет записано объявление XML-документа, то объявление будет содержать атрибут 'encoding', имеющий соответствующее значение. Если же в качестве параметра указана пустая строка или ничего не указано, то атрибут 'encoding' не будет присутствовать в объявлении XML-документа.

Вариант синтаксиса: С параметрами записи

Синтаксис:

УстановитьСтроку(<ПараметрыЗаписи>)

Параметры:

<ПараметрыЗаписи> (необязательный)

Тип: [ПараметрыЗаписиXML](ПараметрыЗаписиXML_(XMLWriterSettings).md).   
Параметры записи XML.

Описание варианта метода:

Позволяет указать параметры записи XML.

Описание:

Инициализирует объект для вывода результирующего XML в строку.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ЗаписьXML = Новый ЗаписьXML; ЗаписьXML.УстановитьСтроку(); // Запись документа XML  ... СтрXML = ЗаписьXML.Закрыть(); |

Использование в версии:

Доступен, начиная с версии 8.1.
