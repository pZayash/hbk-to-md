Описание:

Используется для доступа к Active document. Объект может быть записан в файл.  
Может быть получен методами ПолучитьМакет() и [ПолучитьОбщийМакет](Глобальный_контекст.ПолучитьОбщийМакет.md), если тип макета [ActiveDocument](ТипМакета.ActiveDocument.md).

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
