ВнешнийИсточникДанныхМенеджер.<Имя внешнего источника> (ExternalDataSourceManager.<External source name>)

УстановитьОбщиеПараметрыСоединения (SetCommonConnectionParameters)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьОбщиеПараметрыСоединения(<ПараметрыСоединения>)

Параметры:

<ПараметрыСоединения> (обязательный)

Тип: [ПараметрыСоединенияВнешнегоИсточникаДанных](ПараметрыСоединенияВнешнегоИсточникаДанных.md).   
Устанавливаемые параметры.

Описание:

Устанавливает параметры соединения общие для всех пользователей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Для изменения параметров нужны права на администрирование внешнего источника данных.

Использование в версии:

Доступен, начиная с версии 8.2.
