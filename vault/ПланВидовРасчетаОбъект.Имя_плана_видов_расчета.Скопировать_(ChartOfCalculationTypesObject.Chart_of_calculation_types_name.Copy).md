Скопировать (Copy)

Доступен, начиная с версии 8.0.

Синтаксис:

Скопировать()

Возвращаемое значение:

Тип: [ПланВидовРасчетаОбъект.\<Имя плана видов расчета\>](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md).   

Описание:

Создает новый вид расчета копированием существующего.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Использование метода не приводит к записи созданного объекта в базу данных.

Использование в версии:

Доступен, начиная с версии 8.0.
