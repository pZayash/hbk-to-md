СводнаяДиаграмма (PivotChart)

ОбработкаРасшифровки (DetailProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаРасшифровки(<Расшифровка>, <СтандартнаяОбработка>)

Параметры:

<Расшифровка>

Тип: Произвольный.   
Значение расшифровки точки, серии или значения сводной диаграммы.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при двойном щелчке мыши на значении, точке, серии сводной диаграммы, для которых установлено значение расшифровки.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Если сводная диаграмма расположена в форме (или табличном документе, расположенном в форме), то используется процедура-обработчик формы.
