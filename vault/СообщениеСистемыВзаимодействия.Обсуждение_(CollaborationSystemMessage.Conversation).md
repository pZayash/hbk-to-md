Обсуждение (Conversation)

Доступен, начиная с версии 8.3.11.

Использование:

Только чтение.

Описание:

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.11.
