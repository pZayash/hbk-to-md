Расширение таблицы формы для динамического списка (Form table extension for dynamic list)

ПриСохраненииПользовательскихНастроекНаСервере (OnSaveUserSettingsAtServer)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПриСохраненииПользовательскихНастроекНаСервере(<Элемент>, <Настройки>)

Параметры:

<Элемент>

Тип: [ТаблицаФормы](ТаблицаФормы.md).   
Таблица, для которой выполняется сохранение настроек.

<Настройки>

Тип: [ПользовательскиеНастройкиКомпоновкиДанных](ПользовательскиеНастройкиКомпоновкиДанных.md).   
Сохраняемые настройки.

Описание:

Вызывается при сохранении пользовательских настроек. В обработчике можно добавить в объект настроек нужные значения. Обработчик вызывается при закрытии формы для сохранения настроек, а также при интерактивном вызове сохранения настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
