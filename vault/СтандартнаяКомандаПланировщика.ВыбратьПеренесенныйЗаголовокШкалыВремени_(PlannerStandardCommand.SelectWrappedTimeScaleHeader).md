ВыбратьПеренесенныйЗаголовокШкалыВремени (SelectWrappedTimeScaleHeader)

Не рекомендуется использовать, начиная с версии 8.3.23.

Рекомендуется использовать:

- [ВыбратьПеренесенныйЗаголовокШкалыВремени](СтандартнаяКомандаПоляПланировщика.ВыбратьПеренесенныйЗаголовокШкалыВремени_(PlannerStandardCommand.SelectWrappedTimeScaleHeader).md)

Описание:

Переход к событию [НажатиеНаПеренесенномЗаголовкеШкалыВремени](Расширение_поля_формы_для_поля_планировщика.НажатиеНаПеренесенномЗаголовкеШкалыВремени_(Form_field_extension_for_a_planner.WrappedTimeScaleHeaderClick).md).  
Используется только для источника вида [ОбластьПеренесенногоЗаголовкаШкалыВремени](ИсточникКомандПланировщика.ОбластьПеренесенногоЗаголовкаШкалыВремени_(PlannerCommandSource.WrappedTimeScaleHeaderArea).md).  
В контекстном меню команда будет иметь представление "Выбрать".

Использование в версии:

Доступен, начиная с версии 8.3.16.

Не рекомендуется использовать, начиная с версии 8.3.23.

Описание изменено в версии 8.3.23.
