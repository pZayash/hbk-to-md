АтрибутDOM (DOMAttribute)

ПолучитьПользовательскиеДанные (GetUserData)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьПользовательскиеДанные(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ пользовательских данных.

Возвращаемое значение:

Тип: Произвольный.   
[Неопределено](lang__def_Undefined.md), если указанному ключу не соответствовали никакие пользовательские данные.

Описание:

Получает пользовательские данные.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
