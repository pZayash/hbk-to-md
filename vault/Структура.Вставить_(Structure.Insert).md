Структура (Structure)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Ключ>, <Значение>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ устанавливаемого элемента. Ключ должен соответствовать правилам, установленным для идентификаторов:

- Первым символом ключа должна быть буква или символ подчеркивания (\_).
- Каждый из последующих символов может быть буквой, цифрой или символом подчеркивания (\_).

<Значение> (необязательный)

Тип: Произвольный.   
Значение устанавливаемого элемента.

Описание:

Устанавливает значение элемента структуры по ключу. Если элемент с переданным значением ключа существует, то его значение заменяется, в противном случае добавляется новый элемент.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| Отбор = Новый Структура("Дата, Клиент"); Отбор.Вставить("Клиент", ТекКлиент); Отбор.Вставить("Дата", РабочаяДата); |

Использование в версии:

Доступен, начиная с версии 8.0.
