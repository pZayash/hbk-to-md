Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ПолучитьРазмерОбластиДанныхДокументаПоВертикали (GetDocumentDataAreaVerticalSize)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ПолучитьРазмерОбластиДанныхДокументаПоВертикали()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Возвращает максимальный номер строки, содержащей данные любого типа.

Доступность:

Тонкий клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
