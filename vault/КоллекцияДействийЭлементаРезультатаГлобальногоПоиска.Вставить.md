КоллекцияДействийЭлементаРезультатаГлобальногоПоиска (GlobalSearchResultItemActionCollection)

Вставить (Insert)

Доступен, начиная с версии 8.3.18.

Синтаксис:

Вставить(<Индекс>, <Значение>, <Текст>, <Картинка>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Позиция, в которую необходимо вставить действие элемента результата глобального поиска.

<Значение> (необязательный)

Тип: Произвольный.   
Значение действия элемента результата глобального поиска.

<Текст> (необязательный)

Тип: Строка, [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   
Текст, отображаемый в действии элемента результата глобального поиска.

<Картинка> (необязательный)

Тип: [Картинка](Картинка.md).   
Картинка, отображаемая в действии элемента результата глобального поиска.

Возвращаемое значение:

Тип: [ДействиеЭлементаРезультатаГлобальногоПоиска](ДействиеЭлементаРезультатаГлобальногоПоиска.md).   

Описание:

Вставляет действие в коллекцию действий элемента результата глобального поиска по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
