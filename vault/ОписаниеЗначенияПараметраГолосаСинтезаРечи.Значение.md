ОписаниеЗначенияПараметраГолосаСинтезаРечи (TextToSpeechVoiceParameterValueDescription)

Значение (Value)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Ключ значения для использования в методах генерации речи [НачатьСинтез](МенеджерРаботыСРечью.НачатьСинтез.md) и [Синтезировать](МенеджерРаботыСРечью.Синтезировать.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
