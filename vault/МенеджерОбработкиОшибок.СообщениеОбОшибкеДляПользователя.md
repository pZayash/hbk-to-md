МенеджерОбработкиОшибок (ErrorProcessingManager)

СообщениеОбОшибкеДляПользователя (ErrorMessageForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

СообщениеОбОшибкеДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока.md).   

Описание:

Возвращает текст сообщения, которое рекомендуется показать пользователю.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
