ЗаписьZipФайла (ZipFileWriter)

Добавить (Add)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Добавить](ЗаписьФайлаАрхива.Добавить.md)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<ИмяФайла>, <МетодСохраненияПутей>, <РекурсивнаяОбработкаПодпапок>)

Параметры:

<ИмяФайла> (обязательный)

Тип: Строка.   
Имя файла, помещаемого в архив, или маска.  
Недопустимо использование в именах файлов и папок букв национальных алфавитов, которые при преобразовании из символов UNICODE в узкие символы могут быть преобразованы с потерей информации. Рекомендуется использовать в именах файлов и папок символы латинского алфавита.  
Запрещены символы: > < | ? \* / \ : ".

<МетодСохраненияПутей> (необязательный)

Тип: [РежимСохраненияПутейZIP](РежимСохраненияПутейZIP.md).   
Метод сохранения путей к файлу в архиве.  
Значение по умолчанию: [НеСохранятьПути](РежимСохраненияПутейZIP.НеСохранятьПути.md).

<РекурсивнаяОбработкаПодпапок> (необязательный)

Тип: [РежимОбработкиПодкаталоговZIP](РежимОбработкиПодкаталоговZIP.md).   
В случае добавления файлов по маске обрабатывать дополнительно подкаталоги.  
Значение по умолчанию: [НеОбрабатывать](РежимОбработкиПодкаталоговZIP.НеОбрабатывать.md).

Описание:

Добавляет новый файл в архив.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если при открытии в параметре <ИмяФайла> метода [Открыть](ЗаписьZipФайла.Открыть.md) был указан существующий файл ZIP архива, то выполнение данного метода не приводит к добавлению указанного файла к существующему архиву. Прежний архив при выполнении метода [Записать](ЗаписьZipФайла.Записать.md) будет полностью заменен на новый, состав которого формируется данным методом.  
Добавление в архив файла текущего архива вызывает исключительную ситуацию при выполнении метода [Записать](ЗаписьZipФайла.Записать.md).  
Параметр <МетодСохраненияПутей> имеет смысл указывать, если в архив добавляется один файл или группа файлов по указанной в параметре <ИмяФайла> маске.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
