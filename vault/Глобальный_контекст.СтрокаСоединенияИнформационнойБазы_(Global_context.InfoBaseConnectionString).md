Глобальный контекст (Global context)

СтрокаСоединенияИнформационнойБазы (InfoBaseConnectionString)

Доступен, начиная с версии 8.0.

Синтаксис:

СтрокаСоединенияИнформационнойБазы()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
В клиентском приложении и на сервере от имени этого клиентского приложения метод возвращает строку соединения в том виде, как она указана при регистрации информационной базы в клиентском приложении.   
В регламентных и фоновых заданиях строка соединения возвращается в том виде, как она указана при регистрации информационной базы на сервере.

Описание:

Возвращает [Строка соединения](lang__Строка_соединения.md) информационной базы, которую, например, можно использовать для запуска 1С:Предприятия средствами встроенного языка в Automation, [Менеджер COM-соединений](Менеджер_COM-соединений_(COM_connector).md). Пример результата метода: File="C:\1cv8\Base".

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
