КвалификаторыДаты (DateQualifiers)

На основании описания даты

Доступен, начиная с версии 8.0.

Синтаксис:

Новый КвалификаторыДаты(<ЧастиДаты>)

Параметры:

<ЧастиДаты> (необязательный)

Тип: [ЧастиДаты](ЧастиДаты.md).   
Допустимые части даты.  
Значение по умолчанию: [ДатаВремя](ЧастиДаты.ДатаВремя.md).

Описание:

Создает квалификатор даты заданием допустимых частей даты.

Пример:

|  |
| --- |
| ПараметрыДаты = Новый КвалификаторыДаты(ЧастиДаты.Дата); |

Использование в версии:

Доступен, начиная с версии 8.0.
