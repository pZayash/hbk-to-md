СредстваТелефонии (TelephonyTools)

ПоддерживаетсяОбработкаЗвонков (CallsHandlingSupported)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПоддерживаетсяОбработкаЗвонков()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   

- [Истина](lang__def_BooleanTrue.md) - для Android при наличии телефонного модуля,
- [Ложь](lang__def_BooleanFalse.md) - для остальных операционных систем.

Описание:

Определяет доступность получения событий о звонках на устройстве.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[СредстваТелефонии](СредстваТелефонии_(TelephonyTools).md), метод [ПодключитьОбработчикЗвонков](СредстваТелефонии.ПодключитьОбработчикЗвонков_(TelephonyTools.AttachCallsHandler).md)  

Использование в версии:

Доступен, начиная с версии 8.3.9.
