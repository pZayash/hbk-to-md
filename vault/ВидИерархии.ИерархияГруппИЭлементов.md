ВидИерархии (HierarchyType)

ИерархияГруппИЭлементов (HierarchyFoldersAndItems)

Доступен, начиная с версии 8.0.

Описание:

Все элементы делятся на группы и обычные элементы. В качестве родителей могут выступать только группы.

Использование в версии:

Доступен, начиная с версии 8.0.
