# ИнтеграцияСистемыВзаимодействия.ДоступныВидеоконференции

ИнтеграцияСистемыВзаимодействия (CollaborationSystemIntegration)

ДоступныВидеоконференции (VideoconferencesAvailable)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность совершать видеозвонки пользователям, принадлежащим данной интеграции системы взаимодействия.  
[Истина](lang__def_BooleanTrue.md) возвращается только для интеграций с типом внешней системы "WebChat", если для них параметр allowVideoconferences установлен в значение "true".

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
