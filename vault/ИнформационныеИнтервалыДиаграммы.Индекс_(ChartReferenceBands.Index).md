ИнформационныеИнтервалыДиаграммы (ChartReferenceBands)

Индекс (Index)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Индекс(<Интервал>)

Параметры:

<Интервал> (обязательный)

Тип: [ИнформационныйИнтервалДиаграммы](ИнформационныйИнтервалДиаграммы_(ChartReferenceBand).md).   
Интервал, индекс которого в коллекции требуется получить.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс указанного элемента коллекции.  
Если элемент не принадлежит этой коллекции, генерируется исключение.

Описание:

Получает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
