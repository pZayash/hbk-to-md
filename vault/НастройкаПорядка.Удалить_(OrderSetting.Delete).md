НастройкаПорядка (OrderSetting)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияПорядком](ЭлементУправленияПорядком_(OrderingItemControl).md).   
Удаляемый элемент управления порядком.

Описание:

Удаляет элемент из коллекции.  
Внимание! Метод не доступен для настройки порядка динамических списков (свойство НастройкаПорядка объектов [СправочникСписок.](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)<[Имя справочника](СправочникСписок.Имя_справочника_(CatalogList.Catalog_name).md)>, [ДокументСписок.](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)<[Имя документа](ДокументСписок.Имя_документа_(DocumentList.Document_name).md)> и т.д.).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
