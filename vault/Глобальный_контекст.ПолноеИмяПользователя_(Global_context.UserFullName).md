Глобальный контекст (Global context)

ПолноеИмяПользователя (UserFullName)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолноеИмяПользователя()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Полное имя пользователя.

Описание:

Получает полное имя пользователя, указанного при запуске программы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Пользователь = ПолноеИмяПользователя(); |

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ИмяПользователя](Глобальный_контекст.ИмяПользователя_(Global_context.UserName).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
