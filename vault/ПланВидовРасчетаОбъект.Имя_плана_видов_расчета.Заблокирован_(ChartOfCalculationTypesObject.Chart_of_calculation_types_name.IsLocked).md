# ПланВидовРасчетаОбъект.<Имя плана видов расчета>.Заблокирован

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы видов расчета](Планы_видов_расчета.md) › [ПланВидовРасчетаОбъект.<Имя плана видов расчета>](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета_(ChartOfCalculationTypesObject.Chart_of_calculation_types_name).md) › [Методы](ПланВидовРасчетаОбъект.Имя_плана_видов_расчета__Методы.md)

ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

Заблокирован (IsLocked)

Доступен, начиная с версии 8.0.

Синтаксис:

Заблокирован()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - объект заблокирован; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет, заблокирован ли вид расчета данным объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Следует учитывать, что этот метод используется для проверки блокировки объекта базы данных конкретным объектом встроенного языка. Он не может быть использован для проверки, заблокирован ли вообще объект базы данных, например, другими пользователями.

Использование в версии:

Доступен, начиная с версии 8.0.
