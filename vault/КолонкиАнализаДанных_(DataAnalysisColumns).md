**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Анализ данных](Анализ_данных-2.md)

Доступен, начиная с версии 8.0.

Элементы коллекции:

[КолонкаАнализаДанных](КолонкаАнализаДанных_(DataAnalysisColumn).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются колонки анализа данных.  
Возможно обращение к колонке посредством оператора [...]. В качестве аргумента передается индекс колонки (нумерация с 0).

Описание:

Содержит набор колонок анализа данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.   
Сериализуется.
