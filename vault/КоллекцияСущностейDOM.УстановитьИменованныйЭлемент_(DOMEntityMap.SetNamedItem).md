КоллекцияСущностейDOM (DOMEntityMap)

УстановитьИменованныйЭлемент (SetNamedItem)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьИменованныйЭлемент(<Сущность>)

Параметры:

<Сущность> (обязательный)

Тип: [СущностьDOM](СущностьDOM_(DOMEntity).md).   
Узел сущности DOM.

Возвращаемое значение:

Тип: [СущностьDOM](СущностьDOM_(DOMEntity).md).   

Описание:

Устанавливает сущность DOM.  
В качестве ключа в коллекции будет использовано имя узла.  
Если в коллекции присутствует узел нотации с таким же именем, то он будет замещен и возвращен в качестве возвращаемого значения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
