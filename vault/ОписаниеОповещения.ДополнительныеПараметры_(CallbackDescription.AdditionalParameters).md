ОписаниеОповещения (CallbackDescription)

ДополнительныеПараметры (AdditionalParameters)

Доступен, начиная с версии 8.3.3.

Использование:

Только чтение.

Описание:

Тип: Произвольный.   
Содержит дополнительные параметры. При вызове передается в указанную процедуру последним параметром.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.3.
