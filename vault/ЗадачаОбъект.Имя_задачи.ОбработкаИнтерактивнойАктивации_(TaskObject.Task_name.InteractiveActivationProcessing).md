ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ОбработкаИнтерактивнойАктивации (InteractiveActivationProcessing)

Доступен, начиная с версии 8.0.

Синтаксис:

ОбработкаИнтерактивнойАктивации(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Возникает при выполнении метода [АктивироватьИнтерактивно](ЗадачаОбъект.Имя_задачи.АктивироватьИнтерактивно_(TaskObject.Task_name.ActivateInteractively).md) перед вызовом аналогичного обработчика у соответствующей точки маршрута бизнес-процесса. Если не изменялась стандартная обработка, то после этого события будет вызван обработчик интерактивной активации у точки маршрута бизнес-процесса.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

В режиме запуска [УправляемоеПриложение](РежимЗапускаКлиентскогоПриложения.УправляемоеПриложение_(ClientRunMode.ManagedApplication).md) не вызывается.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
