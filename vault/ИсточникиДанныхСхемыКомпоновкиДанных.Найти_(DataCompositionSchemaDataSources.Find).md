ИсточникиДанныхСхемыКомпоновкиДанных (DataCompositionSchemaDataSources)

Найти (Find)

Доступен, начиная с версии 8.1.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (необязательный)

Тип: [Строка](lang__def_String.md).   
Имя искомого элемента.

Возвращаемое значение:

Тип: [ИсточникДанныхСхемыКомпоновкиДанных](ИсточникДанныхСхемыКомпоновкиДанных_(DataCompositionSchemaDataSource).md), [Неопределено](lang__def_Undefined.md).   
Если не найден, возвращается [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента коллекции по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
