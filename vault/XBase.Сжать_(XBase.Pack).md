XBase (XBase)

Сжать (Pack)

Доступен, начиная с версии 8.0.

Синтаксис:

Сжать()

Описание:

Уменьшает размер файла таблицы базы данных, убирая из него записи, помеченные как удаленные.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Текст = "Удаленные записи будут уничтожены. Продолжить?"; Режим = РежимДиалогаВопрос.ДаНет; Если Вопрос(Текст, Режим) = КодВозвратаДиалога.Да Тогда     xB.Сжать(); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
