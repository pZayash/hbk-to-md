РегистрРасчетаЗапись.<Имя регистра расчета> (CalculationRegisterRecord.<Calculation register name>)

ФактическийПериодДействия (ActualActionPeriod)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Фактический период действия. Отличается от периода действия, задаваемого датами начала и окончания периода действия. Период действия - это свойство учетной записи, которое отражает тот факт, что данная учетная запись пытается получить такой период действия. Фактический период действия - это тот период, который сложился за счет правил взаимной конкуренции учетных записей за период действия, определяемых списками вытесняющих видов расчета плана расчетов данного регистра расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определен для регистров расчета с признаком "Период действия".   
В качестве элементов массива - [ЭлементФактическогоПериодаДействия](ЭлементФактическогоПериодаДействия_(ActualActionPeriodItem).md).

См. также:

[ЭлементФактическогоПериодаДействия](ЭлементФактическогоПериодаДействия_(ActualActionPeriodItem).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
