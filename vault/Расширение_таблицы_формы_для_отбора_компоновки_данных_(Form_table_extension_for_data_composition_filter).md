Описание:

Дополнительные свойства элемента управления [ТаблицаФормы](ТаблицаФормы_(FormTable).md), применимые для отбора компоновки данных.  
В качестве значений для свойств [ТекущаяСтрока](ТаблицаФормы.ТекущаяСтрока_(FormTable.CurrentRow).md), [ТекущийРодитель](ТаблицаФормы.ТекущийРодитель_(FormTable.CurrentParent).md) и элемента коллекции, содержащейся в свойстве [ВыделенныеСтроки](ТаблицаФормы.ВыделенныеСтроки_(FormTable.SelectedRows).md), используется идентификатор типа [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md).  
Свойство [ТекущиеДанные](ТаблицаФормы.ТекущиеДанные_(FormTable.CurrentData).md) и метод [ДанныеСтроки](ТаблицаФормы.ДанныеСтроки_(FormTable.RowData).md) возвращают структуру, заполненную копией данных. Их изменение не отразится на данных формы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
