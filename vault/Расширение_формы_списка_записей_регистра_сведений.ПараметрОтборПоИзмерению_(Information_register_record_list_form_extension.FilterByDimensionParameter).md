# Расширение формы списка записей регистра сведений.ПараметрОтборПоИзмерению

Расширение формы списка записей регистра сведений (Information register record list form extension)

ПараметрОтборПоИзмерению (FilterByDimensionParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Предоставляет возможность установить начальное значение отбора по значениям измерений при открытии или повторном открытии формы списка регистра сведений. Содержит объект [Структура](Структура_(Structure).md), имена свойств которого должны соответствовать именам измерений регистра, значения свойств - устанавливаемым значениям отбора.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
