ЗадачаВыборка.<Имя задачи> (TaskSelection.<Task name>)

Номер (Number)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md), [Строка](lang__def_String.md).   
Содержит номер задачи. Тип значения зависит от установки в Конфигураторе представления номера задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
