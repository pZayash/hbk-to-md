РасширениеКонфигурации (ConfigurationExtension)

ИзменяетСтруктуруДанных (ModifiesDataStructure)

Доступен, начиная с версии 8.3.13.

Синтаксис:

ИзменяетСтруктуруДанных()

Возвращаемое значение:

Тип: Булево.   

Описание:

Определяет, содержит ли данное расширение объекты метаданных, влияющие на структуры данных, и, соответственно, изменяло ли оно структуры для хранения данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.13.
