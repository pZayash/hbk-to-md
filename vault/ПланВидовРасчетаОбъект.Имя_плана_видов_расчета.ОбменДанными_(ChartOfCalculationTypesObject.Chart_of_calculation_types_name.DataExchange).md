ПланВидовРасчетаОбъект.<Имя плана видов расчета> (ChartOfCalculationTypesObject.<Chart of calculation types name>)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md).   
Используется для управления обменом данных.  
С помощью данного свойства настраивается состав узлов-получателей, для которых будут регистрироваться изменения данных, узел-отправитель, из которого получена записываемая информация, а также устанавливается режим [Загрузка](ПараметрыОбменаДанными.Загрузка_(DataExchangeParameters.Load).md), указывающий, что выполняется перенос информации.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
