Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на план видов характеристик.  
Расширение предоставляет возможность автоматического поиска и подбора элемента вида характеристики по введенному тексту. По умолчанию поиск осуществляется по полям "Код" и "Наименование".

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
