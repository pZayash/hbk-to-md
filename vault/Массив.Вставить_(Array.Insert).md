Массив (Array)

Вставить (Insert)

Доступен, начиная с версии 8.0.

Синтаксис:

Вставить(<Индекс>, <Значение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс вставляемого значения.

<Значение> (необязательный)

Тип: Произвольный.   
Вставляемое значение. Если не указан, то будет добавлено значение типа [Неопределено](lang__def_Undefined.md).

Описание:

Вставляет значение в массив по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Допускается указание индекса большего, чем [ВГраница](Массив.ВГраница_(Array.UBound).md). В этом случае массив дополняется элементами с индексами от ВГраница()+1 до указанного в параметре индекса.

Пример:

|  |
| --- |
| Массив.Вставить(0, "Первый"); |

Использование в версии:

Доступен, начиная с версии 8.0.
