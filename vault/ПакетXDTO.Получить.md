ПакетXDTO (XDTOPackage)

Получить (Get)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По индексу

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: Число.   
Индекс элемента коллекции.

Вариант синтаксиса: По имени

Синтаксис:

Получить(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: Строка.   
Имя получаемого типа XDTO.

Возвращаемое значение:

Тип: [ТипЗначенияXDTO](ТипЗначенияXDTO.md), [ТипОбъектаXDTO](ТипОбъектаXDTO.md).   

Описание:

Получает тип XDTO по индексу или имени.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
