Доступен, начиная с версии 8.0.

Чтение и запись.

Описание:

Тип: [ТипКолонкиАнализаДанныхКластеризация](ТипКолонкиАнализаДанныхКластеризация_(DataAnalysisColumnTypeClusterization).md), [ТипКолонкиАнализаДанныхПоискАссоциаций](ТипКолонкиАнализаДанныхПоискАссоциаций_(DataAnalysisColumnTypeAssociationRules).md), [ТипКолонкиАнализаДанныхПоискПоследовательностей](ТипКолонкиАнализаДанныхПоискПоследовательностей_(DataAnalysisColumnTypeSequentialPatterns).md), [ТипКолонкиАнализаДанныхДеревоРешений](ТипКолонкиАнализаДанныхДеревоРешений_(DataAnalysisColumnTypeDecisionTree).md), [ТипКолонкиАнализаДанныхОбщаяСтатистика](ТипКолонкиАнализаДанныхОбщаяСтатистика_(DataAnalysisColumnTypeSummaryStatistics).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
