ОбъектМетаданных: Нумератор (MetadataObject: DocumentNumerator)

ДлинаНомера (NumberLength)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Максимальная длина номера объекта базы данных (например, документа, бизнес-процесса).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
