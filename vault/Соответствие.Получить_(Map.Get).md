Соответствие (Map)

Получить (Get)

Доступен, начиная с версии 8.0.

Синтаксис:

Получить(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: Произвольный.   
Ключ значения.

Возвращаемое значение:

Значение элемента соответствия.  
[Неопределено](lang__def_Undefined.md) - если указанный ключ отсутствует.

Описание:

Получает значение по ключу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Применяется в случаях, когда использование оператора [] невозможно.

Использование в версии:

Доступен, начиная с версии 8.0.
