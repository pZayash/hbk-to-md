Расширение тестируемого поля формы для табличного документа (Tested form field extension for a spreadsheet document)

ЗаписатьСодержимоеВФайл (WriteContentToFile)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ЗаписатьСодержимоеВФайл(<СохранитьКак>)

Параметры:

<СохранитьКак> (необязательный)

Тип: Булево.   
Истина - отображает диалоговое окно "Сохранить как" для выбора имени файла.  
Ложь - сохраняет файл под текущим именем.  
  
В зависимости от значения свойства [Редактирование](Расширение_поля_формы_для_поля_табличного_документа.Редактирование.md) расширения поля формы для поля табличного документа, диалог сохранения в файл может быть отображен независимо от значения данного свойства.  
Значение по умолчанию: Ложь.

Описание:

Вызывает сохранение табличного документа в файл.

Доступность:

Тонкий клиент, толстый клиент.

Примечание:

Из-за особенностей реализации веб-клиента метод применим только при тестировании тонкого клиента.

Использование в версии:

Доступен, начиная с версии 8.3.8.

Описание изменено в версии 8.3.25.
