НастройкаУсловногоОформления (ConditionalAppearanceSetting)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Элемент>)

Параметры:

<Элемент> (обязательный)

Тип: [ЭлементУправленияНастройкойУсловногоОформления](ЭлементУправленияНастройкойУсловногоОформления_(ConditionalAppearanceSettingControl).md).

Описание:

Удаляет элемент управления условного оформления.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
