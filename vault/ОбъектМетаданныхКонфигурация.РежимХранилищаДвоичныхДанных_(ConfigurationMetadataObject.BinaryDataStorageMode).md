# ОбъектМетаданныхКонфигурация.РежимХранилищаДвоичныхДанных

ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

РежимХранилищаДвоичныхДанных (BinaryDataStorageMode)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [РежимХранилищаДвоичныхДанных](РежимХранилищаДвоичныхДанных_(BinaryDataStorageMode).md).   
Управляет режимом совместимости с хранилищем двоичных данных.  
При режиме совместимости с версией 8.3.22 и выше хранилище двоичных данных всегда разрешено.  
При режиме совместимости меньшем, чем [Версия8\_3\_15](РежимСовместимости.Версия8_3_15_(CompatibilityMode.Version8_3_15).md), управление режимом совместимости с хранилищем двоичных данных недоступно.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
