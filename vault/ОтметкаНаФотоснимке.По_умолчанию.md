ОтметкаНаФотоснимке (PhotoStamp)

По умолчанию

Доступен, начиная с версии 8.3.15.

Синтаксис:

Новый ОтметкаНаФотоснимке(<ОтметкаДаты>, <ФорматДаты>, <Текст>)

Параметры:

<ОтметкаДаты> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Выводить отметку текущей даты.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ФорматДаты> (необязательный)

Тип: [Строка](lang__def_String.md).   
Если параметр установлен, он будет использован при выводе фотографии.  
Если параметр не установлен или его не удалось распознать, то будет использован формат, определенный для установленного на устройстве региона.  
Значение по умолчанию: Пустая строка.

<Текст> (необязательный)

Тип: [Строка](lang__def_String.md).   
Произвольная строка с дополнительным текстом, который будет размещен на фотографии.

Использование в версии:

Доступен, начиная с версии 8.3.15.
