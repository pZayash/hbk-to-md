ДиалогОтбораВерсийИсторииДанных (DataHistoryVersionsFilterDialog)

Отбор (Filter)

Доступен, начиная с версии 8.3.13.

Использование:

Чтение и запись.

Описание:

Тип: [Структура](Структура_(Structure).md).   
Используется для установки параметров отбора при открытии диалога.  
Структура идентична структуре, содержащейся в параметре <Отбор> метода [ВыбратьВерсии](МенеджерИсторииДанных.ВыбратьВерсии_(DataHistoryManager.SelectVersions).md).  
Если диалог был закрыт нажатием кнопки "OK", содержит параметры отбора, установленные пользователем.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.13.
