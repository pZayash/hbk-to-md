КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Удалить (Delete)

Доступен, начиная с версии 8.0.

Синтаксис:

Удалить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: <Индекс поля>, <Имя поля>, [ПолеСводнойТаблицы](ПолеСводнойТаблицы_(PivotTableField).md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы_(PivotTableFieldCollection).md).   
Удаляемое поле.

Описание:

Удаляет поле из коллекции полей сводной таблицы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
