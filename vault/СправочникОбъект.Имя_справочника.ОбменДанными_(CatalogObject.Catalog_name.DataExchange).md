# СправочникОбъект.<Имя справочника>.ОбменДанными

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Справочники](Справочники.md) › [СправочникОбъект.<Имя справочника>](СправочникОбъект.Имя_справочника_(CatalogObject.Catalog_name).md) › [Свойства](СправочникОбъект.Имя_справочника__Свойства.md)

СправочникОбъект.<Имя справочника> (CatalogObject.<Catalog name>)

ОбменДанными (DataExchange)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПараметрыОбменаДанными](ПараметрыОбменаДанными_(DataExchangeParameters).md).   
Используется для управления обменом данных.  
С помощью данного свойства настраивается состав узлов-получателей, для которых будут регистрироваться изменения данных, узел-отправитель, из которого получена записываемая информация, а также устанавливается режим [Загрузка](ПараметрыОбменаДанными.Загрузка_(DataExchangeParameters.Load).md), указывающий, что выполняется перенос информации.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
