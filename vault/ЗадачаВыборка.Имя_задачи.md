Описание:

Объект этого типа возвращается методом [Выбрать](ЗадачаМенеджер.Имя_задачи.Выбрать.md) объекта типа [ЗадачаМенеджер.](ЗадачаМенеджер.Имя_задачи.md)<[Имя задачи](ЗадачаМенеджер.Имя_задачи.md)> и представляет собой специализированный способ перебора задач. Обход задач выполняется системой динамически. Это означает, что использование выборки не считывает все задачи сразу, а выбирает их порциями из базы данных. Такой подход позволяет достаточно быстро обходить с помощью выборки большие списки задач и не загружает в память всех элементов выборки.

Доступность:

Сервер, толстый клиент, внешнее соединение.
