РегистрСведенийКлючЗаписи.<Имя регистра сведений> (InformationRegisterRecordKey.<Information register name>)

Период (Period)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [Дата](lang__def_Date.md).   
Содержит дату и время записи периодического регистра сведений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для непериодических регистров сведений смысла не имеет.

Использование в версии:

Доступен, начиная с версии 8.0.
