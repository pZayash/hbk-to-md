ОформленияЯчеекДинамическогоСписка (DynamicListCellAppearances)

Получить (Get)

Доступен, начиная с версии 8.3.10.

Синтаксис:

Получить(<Поле>)

Параметры:

<Поле> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь компоновки данных колонки ячейки, для которой требуется получить оформление.

Возвращаемое значение:

Тип: [ОформлениеЯчейкиДинамическогоСписка](ОформлениеЯчейкиДинамическогоСписка_(DynamicListCellAppearance).md).   

Описание:

Получает оформление ячейки по пути. Работает аналогично оператору [].

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
