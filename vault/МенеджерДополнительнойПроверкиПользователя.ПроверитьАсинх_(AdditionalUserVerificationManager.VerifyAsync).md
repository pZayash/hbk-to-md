# МенеджерДополнительнойПроверкиПользователя.ПроверитьАсинх

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Биометрия](Биометрия-2.md) › [МенеджерДополнительнойПроверкиПользователя](МенеджерДополнительнойПроверкиПользователя_(AdditionalUserVerificationManager).md) › [Методы](МенеджерДополнительнойПроверкиПользователя__Методы.md)

МенеджерДополнительнойПроверкиПользователя (AdditionalUserVerificationManager)

ПроверитьАсинх (VerifyAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПроверитьАсинх(<Способ>, <СообщениеПользователю>)

Параметры:

<Способ> (обязательный)

Тип: [СпособДополнительнойПроверкиПользователя](СпособДополнительнойПроверкиПользователя_(AdditionalUserVerificationMethod).md).   
Способ проверки пользователя.

<СообщениеПользователю> (обязательный)

Тип: [Строка](lang__def_String.md).   
Текст сообщения, отображаемый пользователю при проверке доступа.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Булево](lang__Булево_(Boolean).md). Признак того, что проверка пользователя была отменена пользователем.  
При возникновении ошибки вызывается обработчик ошибки.

Описание:

Асинхронная функция. Начинает проверку дополнительную проверку пользователя указанным способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

[МенеджерДополнительнойПроверкиПользователя](МенеджерДополнительнойПроверкиПользователя_(AdditionalUserVerificationManager).md), метод [НачатьПроверку](МенеджерДополнительнойПроверкиПользователя.НачатьПроверку_(AdditionalUserVerificationManager.BeginVerification).md)  

Использование в версии:

Доступен, начиная с версии 8.3.18.
