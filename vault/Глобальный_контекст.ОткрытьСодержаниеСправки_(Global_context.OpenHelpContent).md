Глобальный контекст (Global context)

ОткрытьСодержаниеСправки (OpenHelpContent)

Доступен, начиная с версии 8.0.

Синтаксис:

ОткрытьСодержаниеСправки()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Ложь](lang__def_BooleanFalse.md) - показ справки невозможен. Например, используется Internet Explorer версии ранее 5,5.

Описание:

Открывает окно с содержанием справки по приложению.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент.  
Вызов метода выполняет обращение к серверу.

Примечание:

Результат выполнения кэшируется. Кэширование зависит от версии платформы и версии конфигурации для режимов запуска "Тонкий клиент" и "Веб-клиент".

Пример:

|  |
| --- |
| ОткрытьСодержаниеСправки(); |

Использование в версии:

Доступен, начиная с версии 8.0.
