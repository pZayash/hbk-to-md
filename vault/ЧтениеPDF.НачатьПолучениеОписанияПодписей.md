ЧтениеPDF (PDFReader)

НачатьПолучениеОписанияПодписей (BeginGetSignatureDescriptions)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПолучениеОписанияПодписей(<ОписаниеОповещения>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ОписанияПодписей> - тип [Массив](Массив.md). В параметре возвращается массив, содержащий объекты типа [ОписаниеПодписиPDF](ОписаниеПодписиPDF.md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<ОписанияПодписей> - тип [Массив](Массив.md). В параметре возвращается массив, содержащий объекты типа [ОписаниеПодписиPDF](ОписаниеПодписиPDF.md).  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).





<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии.md).   
Менеджер криптографии.

Описание:

Инициирует получение массива, содержащего описания присоединенных к PDF-документу электронных подписей. В веб-клиенте требуется установка расширения для работы с криптографией.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЧтениеPDF](ЧтениеPDF.md), метод [ПолучитьОписанияПодписей](ЧтениеPDF.ПолучитьОписанияПодписей.md)  
[ЧтениеPDF](ЧтениеPDF.md), метод [ПолучитьОписанияПодписейАсинх](ЧтениеPDF.ПолучитьОписанияПодписейАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
