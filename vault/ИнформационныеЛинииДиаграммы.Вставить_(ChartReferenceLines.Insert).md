ИнформационныеЛинииДиаграммы (ChartReferenceLines)

Вставить (Insert)

Доступен, начиная с версии 8.3.12.

Синтаксис:

Вставить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс добавляемой информационной линии.

Возвращаемое значение:

Тип: [ИнформационнаяЛинияДиаграммы](ИнформационнаяЛинияДиаграммы_(ChartReferenceLine).md).   
Вставленный элемент.

Описание:

Добавляет новый элемент коллекции по указанному индексу.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.12.
