Глобальный контекст (Global context)

ПодключитьРасширениеРаботыСКриптографией (AttachCryptoExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [НачатьПодключениеРасширенияРаботыСКриптографией](Глобальный_контекст.НачатьПодключениеРасширенияРаботыСКриптографией.md)

Доступен, начиная с версии 8.2.

Синтаксис:

ПодключитьРасширениеРаботыСКриптографией()

Возвращаемое значение:

Тип: Булево.   
В случае успешного подключения возвращает Истина, иначе - Ложь.  
Для тонкого и толстого клиентов, а также для клиента мобильного приложения результат всегда Истина.

Описание:

При первом обращении выполняет подключение расширения для работы с криптографией.  
При каждом обращении проверяет готовность расширения к работе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
