Расширение поля формы для поля графической схемы (Form field extension for a graphical schema field)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.3.15.

Синтаксис:

ПередЗаписью(<Копирование>, <Отказ>)

Параметры:

<Копирование>

Тип: Булево.   
Истина, если выполняется команда "Сохранить как".

<Отказ>

Тип: Булево.   
Установка данного параметра в Истина позволяет отказаться от записи.  
Значение по умолчанию: Ложь.

Описание:

Событие, возникающее перед интерактивной записью.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.15.

Описание изменено в версии 8.3.18.
