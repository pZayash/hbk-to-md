АдминистрированиеКластер (AdministrationCluster)

ПолучитьАдминистраторовКластера (GetClusterAdministrators)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьАдминистраторовКластера()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [АдминистрированиеАдминистратор](АдминистрированиеАдминистратор_(AdministrationAdministrator).md).

Описание:

Получает массив администраторов кластера. Про каждого администратора доступна вся информация, кроме его пароля. Для успешного выполнения метода необходима аутентификация одного из администраторов кластера.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
