СертификатКлиентаWindows (WindowsClientCertificate)

По способу выбора сертификата

Доступен, начиная с версии 8.3.1.

Синтаксис:

Новый СертификатКлиентаWindows(<СпособВыбораСертификатаWindows>)

Параметры:

<СпособВыбораСертификатаWindows> (обязательный)

Тип: [СпособВыбораСертификатаWindows](СпособВыбораСертификатаWindows.md).   
Указывает способ выбора клиентского сертификата из подходящих сертификатов, установленных в системном хранилище сертификатов операционной системы MS Windows.

Описание:

Создает объект клиентского сертификата OpenSSL, загружаемого из системного хранилища сертификатов операционной системы MS Windows.  
Доступность: Толстый клиент, тонкий клиент.

См. также:

[СпособВыбораСертификатаWindows](СпособВыбораСертификатаWindows.md)  

Использование в версии:

Доступен, начиная с версии 8.3.1.
