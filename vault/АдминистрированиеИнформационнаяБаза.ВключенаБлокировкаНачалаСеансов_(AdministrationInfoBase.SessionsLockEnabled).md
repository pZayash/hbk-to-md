АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ВключенаБлокировкаНачалаСеансов (SessionsLockEnabled)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Управляет режимом блокировки сеансов.   
Если установлено значение [Истина](lang__def_BooleanTrue.md) и текущее время находится в диапазоне, определяемом свойствами [ВремяНачалаБлокировки](АдминистрированиеИнформационнаяБаза.ВремяНачалаБлокировки_(AdministrationInfoBase.LockBeginTime).md) и [ВремяОкончанияБлокировки](АдминистрированиеИнформационнаяБаза.ВремяОкончанияБлокировки_(AdministrationInfoBase.LockEndTime).md), то начало новых сеансов с информационной базой запрещено. Если при этом попытаться установить новый сеанс, то будет сгенерировано сообщение об ошибке, содержащее текст из значения свойства [СообщениеБлокировки](АдминистрированиеИнформационнаяБаза.СообщениеБлокировки_(AdministrationInfoBase.LockMessage).md).   
Для начала сеанса вопреки блокировке сеансов необходимо указать код разрешения, определяемый свойством [КодРазрешенияНачалаСеансов](АдминистрированиеИнформационнаяБаза.КодРазрешенияНачалаСеансов_(AdministrationInfoBase.SessionStartPermissionCode).md).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
