КоллекцияЭлементовОтбораКомпоновкиДанных (DataCompositionFilterItemCollection)

Вставить (Insert)

Доступен, начиная с версии 8.1.

Синтаксис:

Вставить(<Индекс>, <Тип>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Расположение элемента в коллекции.

<Тип> (обязательный)

Тип: [Тип](lang__def_Type.md).   
Тип добавляемого элемента. Элементами коллекции могут быть [ЭлементУсловногоОформленияКомпоновкиДанных](ЭлементУсловногоОформленияКомпоновкиДанных_(DataCompositionConditionalAppearanceItem).md), [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных_(DataCompositionFilterItemGroup).md).

Возвращаемое значение:

Тип: [ЭлементОтбораКомпоновкиДанных](ЭлементОтбораКомпоновкиДанных_(DataCompositionFilterItem).md), [ГруппаЭлементовОтбораКомпоновкиДанных](ГруппаЭлементовОтбораКомпоновкиДанных_(DataCompositionFilterItemGroup).md).   

Описание:

Добавляет элемент в заданную позицию.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
