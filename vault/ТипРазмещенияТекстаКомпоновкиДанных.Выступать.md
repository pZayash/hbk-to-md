ТипРазмещенияТекстаКомпоновкиДанных (DataCompositionTextPlacementType)

Выступать (Overflow)

Доступен, начиная с версии 8.1.

Описание:

Текст, не вмещающийся в ячейки, будет выступать на следущие ячейки в случае, если они пустые.

Использование в версии:

Доступен, начиная с версии 8.1.
