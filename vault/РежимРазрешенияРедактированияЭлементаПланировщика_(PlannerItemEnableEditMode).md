Описание:

Определяет варианты режима редактирования элемента планировщика. Значение свойства [РежимРазрешенияРедактирования](ЭлементПланировщика.РежимРазрешенияРедактирования_(PlannerItem.EnableEditMode).md) не влияет на копирование элемента планировщика и на перетаскивание элемента планировщика вовне и извне поля планировщика. При копировании элемента планировщика данное свойство заполняется значением по умолчанию.

Доступность:

Тонкий клиент, веб-клиент, сервер, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
