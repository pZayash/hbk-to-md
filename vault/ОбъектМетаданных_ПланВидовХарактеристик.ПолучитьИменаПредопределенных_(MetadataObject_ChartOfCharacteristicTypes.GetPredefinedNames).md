ОбъектМетаданных: ПланВидовХарактеристик (MetadataObject: ChartOfCharacteristicTypes)

ПолучитьИменаПредопределенных (GetPredefinedNames)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПолучитьИменаПредопределенных()

Возвращаемое значение:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   

Описание:

Возвращает имена предопределенных, заданные в конфигурации в виде фиксированного массива из имен предопределенных типа [Строка](lang__def_String.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.  
Вызов метода выполняет обращение к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.5.
