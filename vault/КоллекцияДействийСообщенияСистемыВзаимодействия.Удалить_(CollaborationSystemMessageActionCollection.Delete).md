КоллекцияДействийСообщенияСистемыВзаимодействия (CollaborationSystemMessageActionCollection)

Удалить (Delete)

Не рекомендуется использовать, начиная с версии 8.3.25.

Рекомендуется использовать:

- [Удалить](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия.Удалить_(CollaborationSystemMessageButtonPanelButtonRow.Delete).md)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: По индексу

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс удаляемого действия в коллекции.

Описание варианта метода:

Удаляет действие по заданному индексу из коллекции действий сообщения системы взаимодействия.

Вариант синтаксиса: По действию

Синтаксис:

Удалить(<Действие>)

Параметры:

<Действие> (обязательный)

Тип: [ДействиеСообщенияСистемыВзаимодействия](ДействиеСообщенияСистемыВзаимодействия_(CollaborationSystemMessageAction).md).   
Удаляемое действие сообщения системы взаимодействия.

Описание варианта метода:

Удаляет заданное действие из коллекции действий сообщения системы взаимодействия.

Описание:

Удаляет действие из коллекции действий сообщения системы взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Не рекомендуется использовать, начиная с версии 8.3.25.

Описание изменено в версии 8.3.25.
