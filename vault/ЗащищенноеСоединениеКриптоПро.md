Описание:

Объект защищенного соединения КриптоПро.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| ssl1 = Новый ЗащищенноеСоединениеКритоПро(             Новый СертификатКлиентаОС,             Новый СертификатыУдостоверяющихЦентровОС,             РежимПроверкиОтзываСертификатовУдостоверяющихЦентров.Строгий);  ssl2 = Новый ЗащищенноеСоединениеКритоПро(             Новый СертификатКлиентаОС(СпособВыбораСертификатаОС.Авто),             Новый СертификатыУдостоверяющихЦентровФайл("C:/certs/myca.pem"));  ssl3 = Новый ЗащищенноеСоединениеКритоПро(             Новый СертификатКлиентаФайл("C:/certs/user.p12", "password" ),             Новый СертификатыУдостоверяющихЦентровФайл("C:/certs/myca.pem"));  ssl4 = Новый ЗащищенноеСоединениеКритоПро(             неопределено,             неопределено); |
