СписокXDTO (XDTOList)

Удалить (Delete)

Доступен, начиная с версии 8.1.

Синтаксис:

Удалить(<Индекс>)

Параметры:

<Индекс> (необязательный)

Тип: Число.   
Позиция удаляемого элемента данных.

Возвращаемое значение:

Тип: [ЗначениеXDTO](ЗначениеXDTO.md), [ОбъектXDTO](ОбъектXDTO.md).   
Элемент данных, удаленный из списка.

Описание:

Удаляет элемент данных из указанной позиции.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
