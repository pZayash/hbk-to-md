Расширение формы клиентского приложения для плана видов характеристик (Client application form extension for chart of characteristics types)

ВыборЗначения (ValueChoice)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ВыборЗначения(<СтандартнаяОбработка>)

Параметры:

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Вызывается на клиенте перед закрытием формы, находящейся в режиме выбора.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
