Расширение формы клиентского приложения для компоновщика настроек (Client application form extension for settings composer)

СоздатьЭлементыФормыПользовательскихНастроек (CreateUserSettingsFormItems)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СоздатьЭлементыФормыПользовательскихНастроек(<ГруппаПользовательскихНастроек>, <Режим>, <КоличествоКолонок>)

Параметры:

<ГруппаПользовательскихНастроек> (необязательный)

Тип: [ГруппаФормы](ГруппаФормы_(FormGroup).md).   
Группа, внутри которой требуется создать элементы для редактирования пользовательских настроек.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<Режим> (необязательный)

Тип: [РежимОтображенияНастроекКомпоновкиДанных](РежимОтображенияНастроекКомпоновкиДанных_(DataCompositionSettingsViewMode).md).   
Указывает, нужно ли в группе создавать элементы для всех пользовательских настроек или только для быстрых.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<КоличествоКолонок> (необязательный)

Тип: [Число](lang__def_Number.md).   
Задает количество колонок, в которых размещаются элементы формы в группе редактирования пользовательских настроек.  
Если параметр не задан, то элементы группы размещаются в двух колонках.

Описание:

Создает элементы формы для редактирования пользовательских настроек.

Доступность:

Сервер.

Использование в версии:

Доступен, начиная с версии 8.3.3.
