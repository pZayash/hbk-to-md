СообщениеПользователю (UserMessage)

УстановитьДанные (SetData)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьДанные(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: Произвольный.   
Объект, с которым должно быть связано сообщение.

Описание:

Устанавливает на основе переданного объекта свойства [ПутьКДанным](СообщениеПользователю.ПутьКДанным.md) и [КлючДанных](СообщениеПользователю.КлючДанных.md).  
Для установки свойства [ПутьКДанным](СообщениеПользователю.ПутьКДанным.md) используется установленное автоматически или методом [УстановитьСоответствиеОбъектаИФормы](Глобальный_контекст.УстановитьСоответствиеОбъектаИФормы.md) соответствие объекта и реквизита формы.  
Для установки свойства [КлючДанных](СообщениеПользователю.КлючДанных.md) используется ссылка объекта или значение регистратора для наборов записей.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
