КвалификаторыДвоичныхДанных (BinaryDataQualifiers)

На основании описания двоичных данных

Доступен, начиная с версии 8.2.

Синтаксис:

Новый КвалификаторыДвоичныхДанных(<Длина>, <ДопустимаяДлина>)

Параметры:

<Длина> (необязательный)

Тип: [Число](lang__def_Number.md).   
Длина двоичных данных.  
Значение по умолчанию: 0.

<ДопустимаяДлина> (необязательный)

Тип: [ДопустимаяДлина](ДопустимаяДлина_(AllowedLength).md).   
Допустимая длина.  
Значение по умолчанию: [Переменная](ДопустимаяДлина.Переменная_(AllowedLength.Variable).md).

Использование в версии:

Доступен, начиная с версии 8.2.
