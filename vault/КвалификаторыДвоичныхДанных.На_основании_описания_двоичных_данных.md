КвалификаторыДвоичныхДанных (BinaryDataQualifiers)

На основании описания двоичных данных

Доступен, начиная с версии 8.2.

Синтаксис:

Новый КвалификаторыДвоичныхДанных(<Длина>, <ДопустимаяДлина>)

Параметры:

<Длина> (необязательный)

Тип: Число.   
Длина двоичных данных.  
Значение по умолчанию: 0.

<ДопустимаяДлина> (необязательный)

Тип: [ДопустимаяДлина](ДопустимаяДлина.md).   
Допустимая длина.  
Значение по умолчанию: [Переменная](ДопустимаяДлина.Переменная.md).

Использование в версии:

Доступен, начиная с версии 8.2.
