# КаноническаяЗаписьXML

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XML](XML-2.md)

Не рекомендуется использовать, начиная с версии 8.3.9.

Рекомендуется использовать:

- [ПреобразованиеККаноническомуXML](ПреобразованиеККаноническомуXML_(ConversionToCanonicalXML).md)

Доступен, начиная с версии 8.1.

Конструкторы:

[По умолчанию](КаноническаяЗаписьXML.По_умолчанию.md)  

Описание:

Предназначен для организации последовательной записи документов и фрагментов XML в каноническом виде (см. w3c стандарты - Canonical XML, Exclusive XML Canonicalization).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [ЗаписатьXML](Глобальный_контекст.ЗаписатьXML_(Global_context.WriteXML).md)  
[ЗаписьСообщенияОбмена](ЗаписьСообщенияОбмена_(ExchangeMessageWriter).md), метод [НачатьЗапись](ЗаписьСообщенияОбмена.НачатьЗапись_(ExchangeMessageWriter.BeginWrite).md)  
[КаноническийDOM](КаноническийDOM_(DOMCanonicalization).md), метод [Канонизировать](КаноническийDOM.Канонизировать_(DOMCanonicalization.Canonicalize).md)  

Использование в версии:

Доступен, начиная с версии 8.1.

Не рекомендуется использовать, начиная с версии 8.3.9.

Описание изменено в версии 8.3.18.
