МенеджерВстроенныхПокупок (InAppPurchasesManager)

ПоддерживаетсяПодтверждениеПокупок (PurchaseAcknowledgementSupported)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПоддерживаетсяПодтверждениеПокупок()

Возвращаемое значение:

Тип: Булево.   

Описание:

Определяет доступность операции "Подтверждение" для встроенных покупок на текущей ОС:

- Истина - в ОС Android при наличии установленных служб Google Play Services телефонного модуля,
- Ложь – для iOS и Windows Phone.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
