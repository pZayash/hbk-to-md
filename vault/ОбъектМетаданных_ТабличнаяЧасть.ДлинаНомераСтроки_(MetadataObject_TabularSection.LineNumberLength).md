# ОбъектМетаданных: ТабличнаяЧасть.ДлинаНомераСтроки

ОбъектМетаданных: ТабличнаяЧасть (MetadataObject: TabularSection)

ДлинаНомераСтроки (LineNumberLength)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Для табличных частей справочников, документов, планов видов характеристик, планов счетов, планов видов расчета, бизнес-процессов, задач и планов обмена это свойство указывает число десятичных разрядов, используемое для хранения значения номера строки табличной части в базе данных.  
Может принимать значения в диапазоне от 5 до 9.  
При создании новой табличной части в режиме совместимости выше [Версия8\_3\_26](РежимСовместимости.Версия8_3_26_(CompatibilityMode.Version8_3_26).md) свойству присваивается значение 9.  
В режиме совместимости [Версия8\_3\_26](РежимСовместимости.Версия8_3_26_(CompatibilityMode.Version8_3_26).md) и ниже значением свойства может быть только 5.  
  
В расширениях заимствованная табличная часть заимствуется вместе с длиной номера строки, заданной в основной конфигурации. Для заимствованных табличных частей ДлинаНомераСтроки не переопределяется. Для добавляемых в расширении табличных частей ДлинаНомераСтроки определяется в расширении.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.27.
