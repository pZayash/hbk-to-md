Форма (Form)

ВладелецФормы (FormOwner)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Форма](Форма_(Form).md), Элемент управления.   
Содержит форму или элемент управления, который является владельцем данной формы. При помощи этого свойства устанавливается связь между формой и владельцем формы. Например, форма элемента подчиненного справочника может иметь в качестве владельца форму элемента справочника владельца, из которой она была открыта. При открытии по текущей строке табличного поля в качестве владельца устанавливается табличное поле.

Доступность:

Толстый клиент.

Примечание:

Если у формы есть владелец, то в свойстве [ЗакрыватьПриЗакрытииВладельца](Форма.ЗакрыватьПриЗакрытииВладельца_(Form.CloseOnOwnerClose).md) устанавливается необходимость закрытия данной формы при закрытии формы-владельца.

См. также:

[Форма](Форма_(Form).md), свойство [ЗакрыватьПриЗакрытииВладельца](Форма.ЗакрыватьПриЗакрытииВладельца_(Form.CloseOnOwnerClose).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
