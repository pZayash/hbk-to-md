ОбластьПостроенияДендрограммы (DendrogramPlotArea)

Низ (Bottom)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Относительное положение нижней границы.   
Может содержать значения от 0 до 1.   
0 - верхний край, 0,5 - центр, 1 - нижний край.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
