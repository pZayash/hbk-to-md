ИнтернетПочтовыйПрофиль (InternetMailProfile)

ИспользоватьSSLSMTP (SMTPUseSSL)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Использовать SSL соединение для протокола SMTP.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Если предполагается использование защищенного STARTTLS-соединения, то должно быть установлено в значение Ложь.

Использование в версии:

Доступен, начиная с версии 8.3.1.
