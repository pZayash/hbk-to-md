МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ОтправитьЭкран (SendScreen)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ОтправитьЭкран(<Экран>, <Параметр1>, <Параметр2>)

Параметры:

<Экран> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя отправляемого экрана.

<Параметр1> (необязательный)

Тип: [Строка](lang__def_String.md).   
Дополнительный параметр вызова.

<Параметр2> (необязательный)

Тип: [Строка](lang__def_String.md).   
Дополнительный параметр вызова.

Описание:

Передает нестандартный экран в текущий сервис статистики без вызова исходного события, игнорируя настройки регистрации событий/экранов.  
Внимание! Не рекомендуется использовать данный метод в тиражных конфигурациях, так как он не управляется механизмом настройки регистрации статистики использования приложения.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.17.
