ТаблицаФормы (FormTable)

ГоризонтальныеЛинии (HorizontalLines)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md), [Неопределено](lang__def_Undefined.md).   
Содержит признак видимости горизонтальных линий в таблице:

- [Истина](lang__def_BooleanTrue.md) - линии видны.
- [Ложь](lang__def_BooleanFalse.md) - линии не видны.
- [Неопределено](lang__def_Undefined.md) - в таблице со списком в виде дерева линии не видны, в других таблицах линии видны.

Значение по умолчанию: [Неопределено](lang__def_Undefined.md)

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.27.
