# СписокЗначений.ЗаполнитьПометки

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Универсальные коллекции значений](Универсальные_коллекции_значений.md) › [Список значений](Список_значений.md) › [СписокЗначений](СписокЗначений_(ValueList).md) › [Методы](СписокЗначений__Методы.md)

СписокЗначений (ValueList)

ЗаполнитьПометки (FillChecks)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаполнитьПометки(<ЗначениеПометки>)

Параметры:

<ЗначениеПометки> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Значение пометки, устанавливаемой для всех элементов списка значений.

Описание:

Устанавливает пометку у всех элементов списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| // Установить пометку у всех строк.      Список.ЗаполнитьПометки(Истина); // Снять пометку у всех строк.      Список.ЗаполнитьПометки(Ложь); |

Использование в версии:

Доступен, начиная с версии 8.0.
