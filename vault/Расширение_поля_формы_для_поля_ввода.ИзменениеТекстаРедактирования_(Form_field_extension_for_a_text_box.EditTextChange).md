# Расширение поля формы для поля ввода.ИзменениеТекстаРедактирования

Расширение поля формы для поля ввода (Form field extension for a text box)

ИзменениеТекстаРедактирования (EditTextChange)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ИзменениеТекстаРедактирования(<Текст>, <СтандартнаяОбработка>)

Параметры:

<Текст>

Тип: [Строка](lang__def_String.md).   
Содержит редактируемый текст.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если после вызова обработчика данный параметр содержит значение [Ложь](lang__def_BooleanFalse.md), то не будут выполняться действия по автоподбору (в том числе, не будет вызван обработчик события [АвтоПодбор](Расширение_поля_формы_для_поля_ввода.АвтоПодбор_(Form_field_extension_for_a_text_box.AutoComplete).md)).  
Значение по умолчанию: Истина.

Описание:

Вызывается при изменении текста в поле ввода – при наборе текста, при работе с буфером обмена, при вызове команд "Отменить"/"Вернуть", а также при отмене ввода.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

В обработчике данного события нельзя использовать серверные методы формы с директивой компиляции &НаСервере, а также изменять свойства, которые могут привести к серверному вызову.

Использование в версии:

Доступен, начиная с версии 8.3.5.
