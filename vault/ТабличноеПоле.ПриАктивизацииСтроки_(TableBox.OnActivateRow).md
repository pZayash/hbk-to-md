ТабличноеПоле (TableBox)

ПриАктивизацииСтроки (OnActivateRow)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриАктивизацииСтроки()

Описание:

Возникает при активизации строки табличного поля.  
Событие также возникает при изменении значения [ТекущаяСтрока](ТабличноеПоле.ТекущаяСтрока_(TableBox.CurrentRow).md).

Доступность:

Толстый клиент.
