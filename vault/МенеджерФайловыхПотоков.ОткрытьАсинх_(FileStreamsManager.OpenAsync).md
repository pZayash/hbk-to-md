# МенеджерФайловыхПотоков.ОткрытьАсинх

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с двоичными данными](Работа_с_двоичными_данными-2.md) › [МенеджерФайловыхПотоков](МенеджерФайловыхПотоков_(FileStreamsManager).md) › [Методы](МенеджерФайловыхПотоков__Методы.md)

МенеджерФайловыхПотоков (FileStreamsManager)

ОткрытьАсинх (OpenAsync)

Доступен, начиная с версии 8.3.18.

Вариант синтаксиса: С указанием режима открытия

Синтаксис:

ОткрытьАсинх(<ИмяФайла>, <РежимОткрытия>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла_(FileOpenMode).md).   
Режим открытия файла.

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера для операций с файлом.

Описание варианта метода:

Открывает файл в заданном режиме с возможностью чтения и записи. Файл открывается в режиме разделяемого чтения. Результат Обещания - объект типа [ФайловыйПоток](ФайловыйПоток_(FileStream).md) для файла с указанным именем.

Вариант синтаксиса: С указанием режимов открытия и доступа

Синтаксис:

ОткрытьАсинх(<ИмяФайла>, <РежимОткрытия>, <РежимДоступа>, <РазмерБуфера>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя открываемого файла.

<РежимОткрытия> (обязательный)

Тип: [РежимОткрытияФайла](РежимОткрытияФайла_(FileOpenMode).md).   
Режим открытия файла.

<РежимДоступа> (обязательный)

Тип: [ДоступКФайлу](ДоступКФайлу_(FileAccess).md).   
Режим доступа к файлу.

<РазмерБуфера> (необязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера для операций с файлом.

Описание варианта метода:

Открывает файл в выбранном режиме, с заданным уровнем доступа, с общим доступом на чтение. Результат Обещания - объект типа [ФайловыйПоток](ФайловыйПоток_(FileStream).md) для файла с указанным именем.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - объект типа [ФайловыйПоток](ФайловыйПоток_(FileStream).md) для файла с заданным именем.

Описание:

Асинхронная функция. Открывает файл в заданном режиме с возможностью чтения и записи.   
Файл открывается в режиме разделяемого чтения.

Доступность:

Тонкий клиент, толстый клиент.
