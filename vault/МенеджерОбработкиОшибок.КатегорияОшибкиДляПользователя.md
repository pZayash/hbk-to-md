МенеджерОбработкиОшибок (ErrorProcessingManager)

КатегорияОшибкиДляПользователя (ErrorCategoryForUser)

Доступен, начиная с версии 8.3.17.

Синтаксис:

КатегорияОшибкиДляПользователя(<ИнформацияОбОшибке>)

Параметры:

<ИнформацияОбОшибке> (обязательный)

Тип: [ИнформацияОбОшибке](ИнформацияОбОшибке.md).   
Структурированная информация об ошибке.

Возвращаемое значение:

Тип: [КатегорияОшибки](КатегорияОшибки.md).   
Возвращает одно из следующих значений:

- [ИсключениеВызванноеИзВстроенногоЯзыка](КатегорияОшибки.ИсключениеВызванноеИзВстроенногоЯзыка.md),

- [НеправильныйПароль](КатегорияОшибки.НеправильныйПароль.md),
- [ОтсутствиеРазрешенияДляИспользованияФункциональности](КатегорияОшибки.ОтсутствиеРазрешенияДляИспользованияФункциональности.md),
- [ОшибкаВнешнегоИсточникаДанных](КатегорияОшибки.ОшибкаВнешнегоИсточникаДанных.md),

- [ОшибкаКопииБазыДанных](КатегорияОшибки.ОшибкаКопииБазыДанных.md),
- [ОшибкаНастроекКомпоновкиДанных](КатегорияОшибки.ОшибкаНастроекКомпоновкиДанных.md),
- [ОшибкаПереходаПоНавигационнойСсылке](КатегорияОшибки.ОшибкаПереходаПоНавигационнойСсылке.md),
- [ОшибкаПолнотекстовогоПоиска](КатегорияОшибки.ОшибкаПолнотекстовогоПоиска.md),
- [ОшибкаПреобразованияДокумента](КатегорияОшибки.ОшибкаПреобразованияДокумента.md),
- [ОшибкаПроверкиПодписи](КатегорияОшибки.ОшибкаПроверкиПодписи.md),
- [ОшибкаРаботыСПринтером](КатегорияОшибки.ОшибкаРаботыСПринтером.md),
- [ОшибкаРаботыСРечью](КатегорияОшибки.ОшибкаРаботыСРечью.md),

Не возвращает:

- [ВсеОшибки](КатегорияОшибки.ВсеОшибки.md),
- [ОшибкаВоВремяВыполненияВстроенногоЯзыка](КатегорияОшибки.ОшибкаВоВремяВыполненияВстроенногоЯзыка.md),

Описание:

Возвращает категорию для пользователя. По этой категории будет формироваться сообщение в методе [СообщениеОбОшибкеДляПользователя](МенеджерОбработкиОшибок.СообщениеОбОшибкеДляПользователя.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.

Описание изменено в версии 8.3.26.
