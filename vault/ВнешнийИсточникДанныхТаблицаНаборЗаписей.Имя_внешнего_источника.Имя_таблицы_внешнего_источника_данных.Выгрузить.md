ВнешнийИсточникДанныхТаблицаНаборЗаписей.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableRecordSet.<External source name>.<External data source table name>)

Выгрузить (Unload)

Доступен, начиная с версии 8.2.

Синтаксис:

Выгрузить(<Строки>, <Колонки>)

Параметры:

<Строки> (необязательный)

Тип: [Массив](Массив.md).   
Массив строк для выгрузки. Если не указан, выгружаются все строки набора записей.

<Колонки> (необязательный)

Тип: Строка.   
Список колонок для выгрузки в формате: "Колонка1, Колонка2...". Если список не задан, то будут выгружены все колонки.

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   

Описание:

Создает таблицу значений и копирует в нее записи набора. Структура полученной таблицы совпадает со структурой набора записей.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
