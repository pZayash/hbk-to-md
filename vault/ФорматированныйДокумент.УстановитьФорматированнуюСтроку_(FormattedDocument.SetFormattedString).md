ФорматированныйДокумент (FormattedDocument)

УстановитьФорматированнуюСтроку (SetFormattedString)

Доступен, начиная с версии 8.3.1.

Синтаксис:

УстановитьФорматированнуюСтроку(<ФорматированнаяСтрока>)

Параметры:

<ФорматированнаяСтрока> (обязательный)

Тип: [ФорматированнаяСтрока](ФорматированнаяСтрока_(FormattedString).md).   
Форматированная строка, из которой будет сформирован форматированный документ.

Описание:

Формирует форматированный документ из переданной форматированной строки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
