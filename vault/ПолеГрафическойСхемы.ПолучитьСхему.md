ПолеГрафическойСхемы (GraphicalSchemaField)

ПолучитьСхему (GetSchema)

Доступен, начиная с версии 8.1.

Синтаксис:

ПолучитьСхему()

Возвращаемое значение:

Тип: [ГрафическаяСхема](ГрафическаяСхема.md).   

Описание:

Получает у элемента управления "Поле графической схемы" значение "Графическая схема" для отображения.

Доступность:

Толстый клиент.

Пример:

|  |
| --- |
| Процедура ПриОткрытии()     Схема = ЭлементыФормы.ПолеСхемы1.ПолучитьСхему();     Схема.ЦветФона = WebЦвета.Аквамарин;     ЭлементыФормы.ПолеСхемы1.УстановитьСхему(Схема); КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.1.
