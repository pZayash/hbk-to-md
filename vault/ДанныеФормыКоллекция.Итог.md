ДанныеФормыКоллекция (FormDataCollection)

Итог (Total)

Доступен, начиная с версии 8.2.

Синтаксис:

Итог(<Колонка>)

Параметры:

<Колонка> (необязательный)

Тип: Строка.   
Имя колонки, по которой возвращается итог.

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает сумму значений всех строк в указанной колонке.   
Итог подсчитывается только для колонок, у которых указан тип Число.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| ОбщийИтог = Товары.Итог("Сумма"); |

Использование в версии:

Доступен, начиная с версии 8.2.
