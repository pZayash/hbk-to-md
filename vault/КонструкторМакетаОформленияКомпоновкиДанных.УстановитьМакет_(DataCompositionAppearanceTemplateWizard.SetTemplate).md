КонструкторМакетаОформленияКомпоновкиДанных (DataCompositionAppearanceTemplateWizard)

УстановитьМакет (SetTemplate)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьМакет(<МакетОформления>)

Параметры:

<МакетОформления> (необязательный)

Тип: [МакетОформленияКомпоновкиДанных](МакетОформленияКомпоновкиДанных_(DataCompositionAppearanceTemplate).md).   
Макет оформления компоновки данных, предназначенный для редактирования.

Описание:

Устанавливает макет оформления компоновки данных для редактирования.

Доступность:

Толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
