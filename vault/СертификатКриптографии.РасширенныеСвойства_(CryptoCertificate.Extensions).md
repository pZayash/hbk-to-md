СертификатКриптографии (CryptoCertificate)

РасширенныеСвойства (Extensions)

Не рекомендуется использовать, начиная с версии 8.3.27.

Рекомендуется использовать:

- [ДополненияСертификата](СертификатКриптографии.ДополненияСертификата_(CryptoCertificate.CertificateExtensions).md)
- [ДополненияСертификатаКакИсходныеДанные](СертификатКриптографии.ДополненияСертификатаКакИсходныеДанные_(CryptoCertificate.CertificateExtensionsAsRawData).md)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ФиксированнаяСтруктура](ФиксированнаяСтруктура_(FixedStructure).md).   
Содержит расширенные свойства сертификата.   
Структура может содержать один элемент со строковым ключом EKU и значением - [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md) строк.  
Каждый элемент массива - это вариант использования сертификата.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.

Не рекомендуется использовать, начиная с версии 8.3.27.

Описание изменено в версии 8.3.27.
