РисунокТабличногоДокумента (SpreadsheetDocumentDrawing)

ПараметрРасшифровки (DetailsParameter)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Строка](lang__def_String.md).   
Имя параметра, значение которого будет использоваться для заполнения расшифровки при выводе.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Не применимо для рисунков типа [Группа](ТипРисункаТабличногоДокумента.Группа_(SpreadsheetDocumentDrawingType.Group).md).

Использование в версии:

Доступен, начиная с версии 8.0.
