**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [XDTO](XDTO-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[СвойствоXDTO](СвойствоXDTO_(XDTOProperty).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются свойства объекта XDTO.  
Возможно обращение к свойству объекта XDTO посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Коллекция свойств XDTO.

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
