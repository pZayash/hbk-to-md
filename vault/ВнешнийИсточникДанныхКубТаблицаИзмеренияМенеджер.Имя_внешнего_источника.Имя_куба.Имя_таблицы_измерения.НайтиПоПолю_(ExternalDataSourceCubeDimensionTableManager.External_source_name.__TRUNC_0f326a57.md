ВнешнийИсточникДанныхКубТаблицаИзмеренияМенеджер.<Имя внешнего источника>.<Имя куба>.<Имя таблицы измерения> (ExternalDataSourceCubeDimensionTableManager.<External source name>.<Cube name>.<Dimension table name>)

НайтиПоПолю (FindByField)

Доступен, начиная с версии 8.3.3.

Синтаксис:

НайтиПоПолю(<ИмяПоля>, <ЗначениеПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля, как оно задано в конфигураторе, по значению которого осуществляется поиск. Тип значения произвольный, кроме [ХранилищеЗначения](ХранилищеЗначения_(ValueStorage).md) и строк произвольной длины.

<ЗначениеПоля> (обязательный)

Тип: Произвольный.   
Значение поля, по которому должен выполняться поиск.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхКубТаблицаИзмеренияСсылка.\<Имя внешнего источника\>.\<Имя куба\>.\<Имя таблицы измерения\>](ВнешнийИсточникДанныхКубТаблицаИзмеренияСсылка.Имя_внешнего_источника.Имя_куба.Имя_таблицы_измерения_(ExternalDataSourceCubeDimensionTableRef.External_source_name.Cube_name.Dimension_table_name).md).   

Описание:

Осуществляет поиск элемента по значению поля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
