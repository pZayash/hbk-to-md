Лицензия (ILicenseInfo)

MaxUsersAll (MaxUsersAll)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит максимальное количество пользователей, допустимое для данного комплекта, если используется программная лицензия платформы. Иначе совпадает с значением свойства [MaxUsersCur](Лицензия.MaxUsersCur_(ILicenseInfo.MaxUsersCur).md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.2.
