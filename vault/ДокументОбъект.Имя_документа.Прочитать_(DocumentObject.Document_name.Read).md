# ДокументОбъект.<Имя документа>.Прочитать

ДокументОбъект.<Имя документа> (DocumentObject.<Document name>)

Прочитать (Read)

Доступен, начиная с версии 8.0.

Синтаксис:

Прочитать()

Описание:

Считывает данные документа из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Позволяет прочесть данные заново. Недопустим для нового объекта.

Пример:

|  |
| --- |
| РасходнаяНакладная.Прочитать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
