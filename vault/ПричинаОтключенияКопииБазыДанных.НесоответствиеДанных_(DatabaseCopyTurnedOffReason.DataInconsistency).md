ПричинаОтключенияКопииБазыДанных (DatabaseCopyTurnedOffReason)

НесоответствиеДанных (DataInconsistency)

Доступен, начиная с версии 8.3.27.

Описание:

Копия не может быть использована для выполнения запроса из-за того, что данные в копии не актуальны (не синхронизированы с основной базой).

Использование в версии:

Доступен, начиная с версии 8.3.27.
