ЗначениеПараметраКомпоновкиДанных (DataCompositionParameterValue)

Использование (Use)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет использование параметра при компоновке данных.  
[Истина](lang__def_BooleanTrue.md) - параметр используется.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
