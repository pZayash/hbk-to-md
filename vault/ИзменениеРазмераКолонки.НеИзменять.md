ИзменениеРазмераКолонки (ColumnSizeChange)

НеИзменять (DontChange)

Доступен, начиная с версии 8.0.

Описание:

Размер колонки не может изменяться при масштабировании или пользователем.

Использование в версии:

Доступен, начиная с версии 8.0.
