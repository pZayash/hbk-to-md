ПолеФормы (FormField)

СочетаниеКлавиш (Shortcut)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [СочетаниеКлавиш](СочетаниеКлавиш.md).   
Содержит сочетание клавиш поля. Для полей, которые принадлежат группе или форме, действие сочетания клавиш приводит к установке фокуса полю. Для полей, которые принадлежат таблице (являются колонкой таблицы), действие сочетания клавиш приводит к установке фокуса непосредственно таблице и выбирается колонка, которая связана с этим полем.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
