ПараметрыВыбора (ChoiceParameters)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Определяет параметры выбора. Фиксированный массив из элементов типа [ПараметрВыбора](ПараметрВыбора_(ChoiceParameter).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
