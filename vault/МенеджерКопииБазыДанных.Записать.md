МенеджерКопииБазыДанных (DatabaseCopyManager)

Записать (Write)

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Синтаксис:

Записать()

Описание:

Записывает измененный объект копии базы данных.  
Требуется наличие права "Администрирование данных" у текущего пользователя информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14 (в режиме совместимости с версией 8.3.14 и последующими).

Описание изменено в версии 8.3.19.
