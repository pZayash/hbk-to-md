Рабочий сервер (IWorkingServerInfo)

ClusterMainPort (ClusterMainPort)

Доступен, начиная с версии 8.3.1.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит номер порта главного менеджера кластера, запущенного на данном рабочем сервере. Используется, если [MainPort](Рабочий_сервер.MainPort_(IWorkingServerInfo.MainPort).md) = [Истина](lang__def_BooleanTrue.md).

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.1.
