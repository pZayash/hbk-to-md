ЗаписьДанных (DataWriter)

НачатьЗаписьБайта (BeginWriteByte)

Доступен, начиная с версии 8.3.8.

Синтаксис:

НачатьЗаписьБайта(<ОписаниеОповещения>, <Число>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

<Число> (обязательный)

Тип: Число.   
Целое число, определяющее байт, который будет записан в поток. Значение числа должно находиться в диапазоне от 0 до 255.

Описание:

Вызывает начало записи одного байта в целевой поток.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьБайт](ЗаписьДанных.ЗаписатьБайт.md)  
[ЗаписьДанных](ЗаписьДанных.md), метод [ЗаписатьБайтАсинх](ЗаписьДанных.ЗаписатьБайтАсинх.md)
