Приложение (ISecurityProfileApplication)

CommandMask (CommandMask)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит шаблон строки запуска приложения. Состоит из последовательности шаблонных слов, разделенных пробелами.   
Шаблонное слово - произвольная последовательность символов. Может содержать символы шаблонов. Если содержит пробелы, то должно заключаться в кавычки. В этом случае кавычки в тексте удваиваются. Кавычки в начале и в конце шаблонного слова тоже удваиваются.   
Шаблонные символы:

- % - произвольная последовательность символов;
- \_ - один произвольный символ;
- \* - имя файла. Если шаблонное слово начинается с символа "\*", то параметр является маршрутом от виртуального каталога. После проверки соответствия параметра шаблонному слову перед выполнением виртуальный каталог будет заменен на физический каталог;
- \ - префикс символа. Его запись перед шаблонным символом делает его нешаблонным.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.3.
