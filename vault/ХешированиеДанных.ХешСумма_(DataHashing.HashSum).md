ХешированиеДанных (DataHashing)

ХешСумма (HashSum)

Доступен, начиная с версии 8.3.18.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md), [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md).   
Содержит текущее значение хеш-суммы. Для функций [MD5](ХешФункция.MD5_(HashFunction.MD5).md), [SHA1](ХешФункция.SHA1_(HashFunction.SHA1).md), [SHA256](ХешФункция.SHA256_(HashFunction.SHA256).md), [SHA512](ХешФункция.SHA512_(HashFunction.SHA512).md), и [Неактивно](СостояниеОбновленияКопииБазыДанных.Неактивно_(DatabaseCopyUpdateState.Inactive).md) – значение типа [ДвоичныеДанные](ДвоичныеДанные_(BinaryData).md), для функции [CRC32](ХешФункция.CRC32_(HashFunction.CRC32).md) – [Число](lang__def_Number.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.18.

Описание изменено в версии 8.3.26.
