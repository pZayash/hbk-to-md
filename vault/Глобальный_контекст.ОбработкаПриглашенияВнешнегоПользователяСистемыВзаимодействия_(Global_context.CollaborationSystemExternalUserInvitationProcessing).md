Глобальный контекст (Global context)

ОбработкаПриглашенияВнешнегоПользователяСистемыВзаимодействия (CollaborationSystemExternalUserInvitationProcessing)

Доступен, начиная с версии 8.3.20.

Синтаксис:

ОбработкаПриглашенияВнешнегоПользователяСистемыВзаимодействия(<Обсуждение>, <СтандартнаяОбработка>)

Параметры:

<Обсуждение>

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия_(CollaborationSystemConversationID).md).   
Если команда приглашения вызвана из списка обсуждения, то в параметре передается [Неопределено](lang__def_Undefined.md).

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
Если после вызова обработчика параметр <СтандартнаяОбработка> содержит значение [Истина](lang__def_BooleanTrue.md), отображается системный диалог приглашения внешнего участника.  
Значение по умолчанию: Истина.

Описание:

Обработчик вызывается при выборе команды "Пригласить внешнего участника".  
Если при выходе из обработчика <СтандартнаяОбработка> имеет значение [Истина](lang__def_BooleanTrue.md), то отображается системный диалог приглашения внешнего участника.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.20.
