ЭлементИзмеренияПланировщика (PlannerDimensionItem)

Родитель (Parent)

Доступен, начиная с версии 8.3.17.

Использование:

Только чтение.

Описание:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md), {ссылка:Объекты; 281; Неопределено}.   
Содержит ссылку на элемент измерения, которому принадлежит данный элемент измерения, или [Неопределено](lang__def_Undefined.md), если элемент измерения находится на верхнем уровне иерархии.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.17.
