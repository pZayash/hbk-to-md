Глобальный контекст (Global context)

СмещениеЛетнегоВремени (DaylightTimeOffset)

Доступен, начиная с версии 8.2.

Синтаксис:

СмещениеЛетнегоВремени(<ЧасовойПояс>, <УниверсальноеВремя>)

Параметры:

<ЧасовойПояс> (необязательный)

Тип: [Строка](lang__def_String.md), [Неопределено](lang__def_Undefined.md).   
Идентификатор часового пояса. Если [Неопределено](lang__def_Undefined.md), то часовой пояс компьютера.  
Список допустимых часовых поясов может быть получен методом [ПолучитьДопустимыеЧасовыеПояса](Глобальный_контекст.ПолучитьДопустимыеЧасовыеПояса_(Global_context.GetAvailableTimeZones).md).   
Кроме того, возможно задание часовых поясов смещением от универсального времени (часовые пояса GMT) в формате:  
GMT{+/-}h[h][:mm]  
где hh=0:23, mm=0:59.  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

<УниверсальноеВремя> (необязательный)

Тип: [Дата](lang__def_Date.md), [Неопределено](lang__def_Undefined.md).   
Универсальное время (UTC), для которого нужно определить смещение или [Неопределено](lang__def_Undefined.md).  
Значение по умолчанию: [Неопределено](lang__def_Undefined.md).

Описание:

Получает смещение в секундах летнего времени заданного часового пояса относительно стандартного времени этого часового пояса для заданного универсального времени. Если <УниверсальноеВремя> = [Неопределено](lang__def_Undefined.md), то возвращает смещение летнего времени, действующее в заданном часовом поясе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
