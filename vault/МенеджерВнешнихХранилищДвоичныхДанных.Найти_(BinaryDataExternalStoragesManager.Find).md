МенеджерВнешнихХранилищДвоичныхДанных (BinaryDataExternalStoragesManager)

Найти (Find)

Доступен, начиная с версии 8.3.24.

Синтаксис:

Найти(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя внешнего хранилища двоичных данных.

Возвращаемое значение:

Тип: [Неопределено](lang__def_Undefined.md), [МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных_(BinaryDataExternalStorageManager).md).   
Если менеджер существует и не удален, вернет [МенеджерВнешнегоХранилищаДвоичныхДанных](МенеджерВнешнегоХранилищаДвоичныхДанных_(BinaryDataExternalStorageManager).md), иначе - [Неопределено](lang__def_Undefined.md).

Описание:

Возвращает внешнее хранилище по имени.

Доступность:

Сервер.

Пример:

|  |
| --- |
| cp = Новый ПараметрыПодключенияВнешнегоХранилищаДвоичныхДанных(); cp.URL = "https://bucket.example.com"; cp.ТипURL = ТипURLВнешнегоХранилищаДвоичныхДанных.ВиртуальныйХостS3;  ap = Новый ПараметрыДоступаВнешнегоХранилищаДвоичныхДанных(); ap.ИдентификаторДоступа = "EXAMPLE"; ap.СекретныйКлюч = "SecretKey"; ap.Регион = "US";  ВнешниеХранилищаДвоичныхДанных.Добавить("manager", cp, ap);  Мнджр = ВнешниеХранилищаДвоичныхДанных.Найти("manager"); |

Использование в версии:

Доступен, начиная с версии 8.3.24.
