# ЭлементDOM.ЕстьАтрибут

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [DOM](DOM-2.md) › [ЭлементDOM](ЭлементDOM_(DOMElement).md) › [Методы](ЭлементDOM__Методы.md)

ЭлементDOM (DOMElement)

ЕстьАтрибут (HasAttribute)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: По имени

Синтаксис:

ЕстьАтрибут(<Имя>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя атрибута.

Вариант синтаксиса: По имени и URI пространства имен

Синтаксис:

ЕстьАтрибут(<URIПространстваИмен>, <ЛокальноеИмя>)

Параметры:

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

<ЛокальноеИмя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Локальное имя атрибута.

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md), если атрибут с указанным именем явно присутствует в элементе, или имеет значение по умолчанию; [Ложь](lang__def_BooleanFalse.md) - в противном случае.

Описание:

Определяет наличие атрибута с указанным именем в данном элементе.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
