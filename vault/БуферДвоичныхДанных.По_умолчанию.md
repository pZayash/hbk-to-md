БуферДвоичныхДанных (BinaryDataBuffer)

По умолчанию

Доступен, начиная с версии 8.3.9.

Синтаксис:

Новый БуферДвоичныхДанных(<Размер>, <Порядок байтов>)

Параметры:

<Размер> (обязательный)

Тип: [Число](lang__def_Number.md).   
Размер буфера в байтах.

<Порядок байтов> (необязательный)

Тип: [ПорядокБайтов](ПорядокБайтов_(ByteOrder).md).   
Порядок байтов.  
Значение по умолчанию: [LittleEndian](ПорядокБайтов.LittleEndian_(ByteOrder.LittleEndian).md).

Использование в версии:

Доступен, начиная с версии 8.3.9.
