Значения

[Окр15как10 (Round15as10)](РежимОкругления.Окр15как10_(RoundMode.Round15as10).md)

Описание:

Определяет режим округления.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), метод [Окр](Глобальный_контекст.Окр_(Global_context.Round).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
