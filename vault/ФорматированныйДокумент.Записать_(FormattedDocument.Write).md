ФорматированныйДокумент (FormattedDocument)

Записать (Write)

Доступен, начиная с версии 8.2.

Вариант синтаксиса: В файл

Синтаксис:

Записать(<ИмяФайла>, <ТипФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла, в который будет записан форматированный документ.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента_(FormattedDocumentFileType).md).   
Указывает тип файла.

Описание варианта метода:

Запись форматированного документа в файл.

Вариант синтаксиса: В поток

Синтаксис:

Записать(<Поток>, <ТипФайла>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток_(Stream).md), [ПотокВПамяти](ПотокВПамяти_(MemoryStream).md), [ФайловыйПоток](ФайловыйПоток_(FileStream).md).   
Поток для записи документа.

<ТипФайла> (необязательный)

Тип: [ТипФайлаФорматированногоДокумента](ТипФайлаФорматированногоДокумента_(FormattedDocumentFileType).md).   
Указывает формат записи документа.

Описание варианта метода:

Запись форматированного документа в поток.

Описание:

Записывает форматированный документ в файл.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

[ФорматированныйДокумент](ФорматированныйДокумент_(FormattedDocument).md), метод [ЗаписатьАсинх](ФорматированныйДокумент.ЗаписатьАсинх_(FormattedDocument.WriteAsync).md)  
[ФорматированныйДокумент](ФорматированныйДокумент_(FormattedDocument).md), метод [НачатьЗапись](ФорматированныйДокумент.НачатьЗапись_(FormattedDocument.BeginWriting).md)  

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.18.
