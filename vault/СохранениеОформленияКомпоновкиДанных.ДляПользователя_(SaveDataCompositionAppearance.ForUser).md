СохранениеОформленияКомпоновкиДанных (SaveDataCompositionAppearance)

ДляПользователя (ForUser)

Доступен, начиная с версии 8.3.16.

Описание:

Используются значения, содержащиеся в объекте [ОформлениеЗначений](ОформлениеЗначений_(ValueAppearance).md) для всей конфигурации.  
При этом значение параметра вывода ТипДиаграммы.КлючСохраненияЦветов игнорируется.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.20.
