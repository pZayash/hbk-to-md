ДокументDOM (DOMDocument)

СоздатьРазыменовательПИ (CreateNSResolver)

Доступен, начиная с версии 8.1.

Вариант синтаксиса: На основе документа DOM

Синтаксис:

СоздатьРазыменовательПИ()

Описание варианта метода:

Разрешение пространств имен выполняется по всем пространствам имен, определенным в документе. В случае совпадения префиксов пространств имен используется то пространство имен, которое определено раньше.

Вариант синтаксиса: На основании узла DOM

Синтаксис:

СоздатьРазыменовательПИ(<УзелКонтекста>)

Параметры:

<УзелКонтекста> (обязательный)

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [ДокументDOM](ДокументDOM_(DOMDocument).md), [ЭлементDOM](ЭлементDOM_(DOMElement).md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [НотацияDOM](НотацияDOM_(DOMNotation).md), [СущностьDOM](СущностьDOM_(DOMEntity).md), [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md), [ТекстDOM](ТекстDOM_(DOMText).md), [КомментарийDOM](КомментарийDOM_(DOMComment).md), [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM_(DOMEntityReference).md), [ПространствоИменXPath](ПространствоИменXPath_(XPathNamespace).md).   
Узел документа DOM, который будет использоваться в качестве контекста, для разрешения пространств имен.

Описание варианта метода:

Разрешение пространств имен выполняется в контексте данного узла, т.е. в разрешении используются пространства имен данного узла и его родительских узлов.

Вариант синтаксиса: На основании Соответствия

Синтаксис:

СоздатьРазыменовательПИ(<Соответствие>)

Параметры:

<Соответствие> (обязательный)

Тип: [Соответствие](Соответствие_(Map).md).   
Ключом соответствия выступает префикс пространства имен, а значением - URI пространства имен.

Описание варианта метода:

Создает адаптер для разрешения пространств имен на основании [Соответствие](Соответствие_(Map).md).

Вариант синтаксиса: На основании конкретного префикса и URI пространства имен

Синтаксис:

СоздатьРазыменовательПИ(<Префикс>, <URIПространстваИмен>)

Параметры:

<Префикс> (обязательный)

Тип: [Строка](lang__def_String.md).   
Префикс.

<URIПространстваИмен> (обязательный)

Тип: [Строка](lang__def_String.md).   
URI пространства имен.

Возвращаемое значение:

Тип: [РазыменовательПространствИменDOM](РазыменовательПространствИменDOM_(DOMNamespaceResolver).md).   

Описание:

Создает адаптер для разрешения пространств имен.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
