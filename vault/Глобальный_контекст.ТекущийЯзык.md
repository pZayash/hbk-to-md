Глобальный контекст (Global context)

ТекущийЯзык (CurrentLanguage)

Доступен, начиная с версии 8.0.

Синтаксис:

ТекущийЯзык()

Возвращаемое значение:

Тип: [ОбъектМетаданных: Язык](ОбъектМетаданных_Язык.md), Строка.   
В режиме выполнения "Сервер", "Толстый клиент" и "Внешнее соединение" возвращается [ОбъектМетаданных: Язык](ОбъектМетаданных_Язык.md).   
В режиме выполнения "Тонкий клиент, "Веб-клиент", "Мобильный клиент", "Мобильное приложение (клиент)", и "Мобильное приложение (сервер)" возвращается значение свойства [КодЯзыка](ОбъектМетаданных_Язык.КодЯзыка.md).

Описание:

Получает язык интерфейса, выбранный для текущего пользователя или основной язык, если нет текущего пользователя или для него не указан язык.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[МенеджерПользователейИнформационнойБазы](МенеджерПользователейИнформационнойБазы.md), метод [ТекущийПользователь](МенеджерПользователейИнформационнойБазы.ТекущийПользователь.md)  

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.16.
