АдминистрированиеСеанс (AdministrationSession)

ДлительностьТекущегоВызоваСУБД (CurrentDBMSCallDuration)

Доступен, начиная с версии 8.3.25.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит длительность текущего вызова СУБД в милисекундах. Если вызов не выполняется - 0.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.25.
