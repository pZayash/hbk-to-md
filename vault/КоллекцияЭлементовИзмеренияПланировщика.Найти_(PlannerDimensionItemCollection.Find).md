КоллекцияЭлементовИзмеренияПланировщика (PlannerDimensionItemCollection)

Найти (Find)

Доступен, начиная с версии 8.3.6.

Синтаксис:

Найти(<Значение>, <ВключатьПодчиненные>)

Параметры:

<Значение> (обязательный)

Тип: Произвольный.   
Значение, которое соответствует искомому значению элемента измерения.

<ВключатьПодчиненные> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет, будут ли участвовать в поиске элемента измерения подчиненные коллекции (если таковые имеются).   
[Истина](lang__def_BooleanTrue.md) – элементы измерения подчиненных коллекций участвуют в поиске.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Возвращаемое значение:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md), [Неопределено](lang__def_Undefined.md).   
Если указанный элемент отсутствует, то возвращается значение [Неопределено](lang__def_Undefined.md).

Описание:

Осуществляет поиск элемента измерения планировщика по указанному значению свойства [Значение](ГруппаРезультатаПоискаПоРегулярномуВыражению.Значение_(ResultOfSearchByRegularExpressionGroup.Value).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
