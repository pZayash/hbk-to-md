НастройкаСервиса (ServiceSetting)

Записать (Write)

Доступен, начиная с версии 8.3.26.

Синтаксис:

Записать()

Описание:

Добавляет или обновляет настройку сервиса кластера на рабочем сервере.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.26.
