Рекомендуется использовать:

- [Путь](ЭлементФайлаАрхива.Путь.md)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит путь файла.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Без режима совместимости содержит части пути в значении свойства разделены в зависимости от используемой операционной системы:

- В Linux элементы пути разделяются символом "/".
- В Windows элементы пути разделяются символом"\".
- В режиме совместимости конфигурации [Версия8\_3\_8](РежимСовместимости.Версия8_3_8.md) и ниже части пути разделены символом "/".
