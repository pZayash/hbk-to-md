ПланВидовХарактеристикСсылка.<Имя плана видов характеристик> (ChartOfCharacteristicTypesRef.<Chart of characteristic types name>)

Метаданные (Metadata)

Доступен, начиная с версии 8.0.

Синтаксис:

Метаданные()

Возвращаемое значение:

Тип: [ОбъектМетаданных: ПланВидовХарактеристик](ОбъектМетаданных_ПланВидовХарактеристик_(MetadataObject_ChartOfCharacteristicTypes).md).   

Описание:

Предоставляет доступ к объекту описания метаданных плана видов характеристик. Другой путь получения того же значения - через свойство глобального контекста [Метаданные](Глобальный_контекст.Метаданные_(Global_context.Metadata).md). Например: Метаданные.ПланыВидовХарактеристик.СвойстваОбъектов.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Для каждого Реквизит Из ПланВидовХарактеристикСсылка.Метаданные().Реквизиты Цикл     Сообщить(Реквизит.Представление()); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
