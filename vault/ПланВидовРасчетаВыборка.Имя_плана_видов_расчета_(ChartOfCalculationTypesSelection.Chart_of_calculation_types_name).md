Описание:

Объект этого типа возвращается методом [Выбрать](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета.Выбрать_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name.Select).md) объекта типа [ПланВидовРасчетаМенеджер.](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)<[Имя плана видов расчета](ПланВидовРасчетаМенеджер.Имя_плана_видов_расчета_(ChartOfCalculationTypesManager.Chart_of_calculation_types_name).md)> и представляет собой специализированный способ перебора объектов вид расчета. Обход элементов выполняется системой динамически. Это означает, что при использовании выборки не считываются все объекты сразу, а происходит считывание "порциями" из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.
