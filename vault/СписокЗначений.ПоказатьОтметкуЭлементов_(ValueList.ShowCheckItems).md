СписокЗначений (ValueList)

ПоказатьОтметкуЭлементов (ShowCheckItems)

Доступен, начиная с версии 8.3.3.

Синтаксис:

ПоказатьОтметкуЭлементов(<ОписаниеОповещенияОЗакрытии>, <Заголовок>)

Параметры:

<ОписаниеОповещенияОЗакрытии> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после закрытия диалога выбора элементов со следующими параметрами:

- <Список> - если нажата кнопка "OK", то [СписокЗначений](СписокЗначений_(ValueList).md), отметка элементов которого была завершена; [Неопределено](lang__def_Undefined.md) - если была нажата кнопка "Отмена";
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок окна диалога. Если параметр не указан, то используется стандартный заголовок.  
Значение по умолчанию: Пустая строка.

Описание:

Предназначен для интерактивной установки состояний пометок у элементов списка значений.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Пример:

|  |
| --- |
| Оповещение = Новый ОписаниеОповещения("ПослеОтметкиЭлементов", ЭтотОбъект); СписокТиповЦен.ПоказатьОтметкуЭлементов(Оповещение, "Удаляемые типы цен."); //...   НаКлиенте Процедура ПослеОтметкиЭлементов(Список, Параметры) Экспорт     Если Список = Неопределено Тогда         Сообщить("Отказ от пометки на удаление типов цен.");     Иначе         Для каждого Элемент из Список Цикл             Если Элемент.Пометка Тогда                 Элемент.Значение.ПолучитьОбъект().                 УстановитьПометкуУдаления(Истина);             КонецЕсли;         КонецЦикла;     КонецЕсли; КонецПроцедуры |

Использование в версии:

Доступен, начиная с версии 8.3.3.

Описание изменено в версии 8.3.22.
