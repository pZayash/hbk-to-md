РежимРазделенияСоставныхСлов (CompositeWordsSeparationMode)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.3.15.

Описание:

В индекс полнотекстового поиска значащие части составных слов не включаются. Подсветка в результатах поиска не меняет поведение.

Использование в версии:

Доступен, начиная с версии 8.3.15.
