ПолучитьБотов (GetBots)

Доступен, начиная с версии 8.3.18.

Синтаксис:

ПолучитьБотов()

Возвращаемое значение:

Тип: [Массив](Массив_(Array).md).   
Массив элементов типа [БотСистемыВзаимодействия](БотСистемыВзаимодействия_(CollaborationSystemBot).md).

Описание:

Возвращает массив ботов системы взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.18.
