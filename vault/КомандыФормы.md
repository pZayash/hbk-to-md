Доступен, начиная с версии 8.2.

Элементы коллекции:

[КомандаФормы](КомандаФормы.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются команды.  
Возможно обращение к команде посредством оператора [...]. В качестве аргумента передается имя команды.  

Методы:

[Добавить (Add)](КомандыФормы.Добавить.md)  
[Количество (Count)](КомандыФормы.Количество.md)  
[Найти (Find)](КомандыФормы.Найти.md)  
[Удалить (Delete)](КомандыФормы.Удалить.md)  

Описание:

Содержит коллекцию команд формы клиентского приложения.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
