XBase (XBase)

ЗаписьУдалена (RecDeleted)

Доступен, начиная с версии 8.0.

Синтаксис:

ЗаписьУдалена()

Возвращаемое значение:

Тип: Булево.   
Истина - запись удалена; Ложь - в противном случае.

Описание:

Получает пометки удаления текущей записи.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| xB.ОтображатьУдаленные = Истина; xB.Первая(); Пока Не xB.ВКонце() Цикл     Если xB.ЗаписьУдалена() Тогда         Сообщить("Удаленная запись: " + xB.НомерЗаписи());     КонецЕсли;     xB.Следующая(); КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
