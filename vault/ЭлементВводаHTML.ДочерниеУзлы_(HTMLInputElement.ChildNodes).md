ЭлементВводаHTML (HTMLInputElement)

ДочерниеУзлы (ChildNodes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [СписокУзловHTML](СписокУзловHTML_(HTMLNodeList).md).   
Список дочерних узлов HTML. В случае отсутствия дочерних узлов содержит пустой список.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
