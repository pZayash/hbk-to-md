РегистрРасчетаНаборЗаписей.<Имя регистра расчета> (CalculationRegisterRecordSet.<Calculation register name>)

УстановитьАктивность (SetActive)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьАктивность(<Активность>)

Параметры:

<Активность> (обязательный)

Тип: Булево.   
Признак активности записей. Истина - записи учитываются в итогах регистра. Ложь - записи не учитываются в итогах регистра.

Описание:

Устанавливает значение свойства [Активность](РегистрРасчетаЗапись.Имя_регистра_расчета.Активность.md) у всех записей, входящих в набор.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| НаборЗаписей.УстановитьАктивность(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
