WebSocketКлиент (WebSocketClient)

ПолучитьТекущееСоединение (GetCurrentConnection)

Доступен, начиная с версии 8.3.27.

Синтаксис:

ПолучитьТекущееСоединение()

Возвращаемое значение:

Тип: [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md), Неопределено.   
Возвращает объект [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md) или  
Неопределено - если соединение отсутствует.

Описание:

Получает объект текущего соединения с WebSocket-клиентом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
