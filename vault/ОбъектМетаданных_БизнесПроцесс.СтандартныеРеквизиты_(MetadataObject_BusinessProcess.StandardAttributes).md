ОбъектМетаданных: БизнесПроцесс (MetadataObject: BusinessProcess)

СтандартныеРеквизиты (StandardAttributes)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ОписанияСтандартныхРеквизитов](ОписанияСтандартныхРеквизитов_(StandardAttributeDescriptions).md).   
Свойство содержит коллекцию настроек стандартных реквизитов объекта метаданных. Состав стандартных реквизитов зависит от вида объекта метаданных.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
