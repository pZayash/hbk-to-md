ВводПоСтроке (InputByString)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [СписокПолей](СписокПолей.md).   
Список полей объекта конфигурации, для которых будет поддерживаться возможность ввода ссылки на объект базы данных путем непосредственного ввода значения данного поля.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
