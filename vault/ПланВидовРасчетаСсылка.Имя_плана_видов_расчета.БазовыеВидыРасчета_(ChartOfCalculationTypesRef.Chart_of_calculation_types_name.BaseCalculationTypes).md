БазовыеВидыРасчета (BaseCalculationTypes)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [БазовыеВидыРасчета.\<Имя плана видов расчета\>](БазовыеВидыРасчета.Имя_плана_видов_расчета_(BaseCalculationTypes.Chart_of_calculation_types_name).md).   
Предопределенная табличная часть вида расчета, которая содержит список базовых видов расчета.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Определено только для объектов тех планов видов расчета, у которых выставлено свойство "Зависимость от базы".

Использование в версии:

Доступен, начиная с версии 8.0.
