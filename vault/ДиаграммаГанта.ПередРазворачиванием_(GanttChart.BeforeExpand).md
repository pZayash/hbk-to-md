ДиаграммаГанта (GanttChart)

ПередРазворачиванием (BeforeExpand)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПередРазворачиванием(<ИдентификаторЗначения>, <Отказ>)

Параметры:

<ИдентификаторЗначения>

Тип: [ИдентификаторЗначенияДиаграммыГанта](ИдентификаторЗначенияДиаграммыГанта_(GanttChartValueID).md).   
Содержит идентификатор значения, точка/серия которого будет развернута.

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от стандартной обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), то стандартной обработки события не будет.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается перед разворачиванием точки/серии диаграммы.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
