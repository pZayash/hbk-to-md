Расширение поля формы для поля диаграммы Ганта (Form field extension for a Gantt chart field)

АвтоМаксимальнаяШирина (AutoMaxWidth)

Доступен, начиная с версии 8.3.6.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет использование автоматического ограничения максимальной ширины элемента.  
Истина - автоматическое ограничение используется

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.6.
