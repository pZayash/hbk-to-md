ДиаграммаГанта (GanttChart)

СвернутьСерию (CollapseSeries)

Доступен, начиная с версии 8.0.

Синтаксис:

СвернутьСерию(<Серия>, <СПодчиненными>)

Параметры:

<Серия> (обязательный)

Тип: [СерияДиаграммыГанта](СерияДиаграммыГанта_(GanttChartSeries).md).   
Сворачиваемая серия.

<СПодчиненными> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет необходимость сворачивания серии вместе с подчиненными сериями. [Истина](lang__def_BooleanTrue.md) - свернуть вместе с подчиненными.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Сворачивает серии, возможно вместе с подчиненными.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
