Элементы коллекции:

[МеткаИнтервалаФонаПланировщика](МеткаИнтервалаФонаПланировщика.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются метки интервала фона планировщика.  
Возможно обращение к метке интервала фона планировщика посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).

Описание:

Представляет собой коллекцию меток интервала фона планировщика.  
Индекс в коллекции определяет порядок отображения меток интервала фона планировщика: чем больше индекс, тем ниже расположена метка относительно другой метки с совпадающим горизонтальным и вертикальным выравниванием.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
