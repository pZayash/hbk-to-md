# ПолеФормы.ФиксацияВТаблице

ПолеФормы (FormField)

ФиксацияВТаблице (FixingInTable)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ФиксацияВТаблице](ФиксацияВТаблице_(FixingInTable).md).   
Определяет, будет ли фиксироваться колонка таблицы.   
Если значение свойства установлено в [Лево](ФиксацияВТаблице.Лево_(FixingInTable.Left).md), то колонка прижимается к левому краю таблицы и фиксируется.   
Если значение установлено в [Право](ФиксацияВТаблице.Право_(FixingInTable.Right).md), то колонка прижимается к правому краю и фиксируется.   
Если значение установлено в значение [Нет](ФиксацияВТаблице.Нет_(FixingInTable.None).md), то колонка выводится обычным образом.   
Если свойство установлено в [Лево](ФиксацияВТаблице.Лево_(FixingInTable.Left).md) или [Право](ФиксацияВТаблице.Право_(FixingInTable.Right).md) для колонки в группе, то колонка переносится на верхний уровень таблицы и фиксируется так, как описано выше.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Примечание:

Свойство имеет смысл, если поле является колонкой таблицы управляемой формы.

Использование в версии:

Доступен, начиная с версии 8.2.
