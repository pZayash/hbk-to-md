# ЗадачаОбъект.<Имя задачи>.ЭтоНовый

ЗадачаОбъект.<Имя задачи> (TaskObject.<Task name>)

ЭтоНовый (IsNew)

Доступен, начиная с версии 8.0.

Синтаксис:

ЭтоНовый()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - изменяется еще ни разу не записанная задача; [Ложь](lang__def_BooleanFalse.md) - задача уже записана.

Описание:

Определяет, записана ли задача в базу данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Если Объект.ЭтоНовый() Тогда     Объект.Наименование = "Новая задача"; КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
