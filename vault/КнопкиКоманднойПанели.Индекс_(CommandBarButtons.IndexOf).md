КнопкиКоманднойПанели (CommandBarButtons)

Индекс (IndexOf)

Доступен, начиная с версии 8.0.

Синтаксис:

Индекс(<Кнопка>)

Параметры:

<Кнопка> (обязательный)

Тип: [КнопкаКоманднойПанели](КнопкаКоманднойПанели_(CommandBarButton).md).   
Кнопка командной панели.

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   
Индекс кнопки командной панели. Если такой кнопки не найдено, возвращает -1.

Описание:

Получает индекс кнопки в коллекции кнопок командной панели.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
