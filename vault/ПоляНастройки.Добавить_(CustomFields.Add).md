ПоляНастройки (CustomFields)

Добавить (Add)

Доступен, начиная с версии 8.0.

Синтаксис:

Добавить(<Имя>, <Представление>, <Тип>, <СписокДопустимыхЗначений>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя поля.

<Представление> (обязательный)

Тип: [Строка](lang__def_String.md).   
Представление поля.   
Если не задано, то выбирается из параметра <Имя>.

<Тип> (необязательный)

Тип: [ОписаниеТипов](ОписаниеТипов_(TypeDescription).md).   
Тип поля.  
Значение по умолчанию: Произвольный.

<СписокДопустимыхЗначений> (необязательный)

Тип: [СписокЗначений](СписокЗначений_(ValueList).md).   
Список допустимых значений для поля.

Возвращаемое значение:

Тип: [ПолеНастройки](ПолеНастройки_(CustomField).md).   

Описание:

Добавляет поле в коллекцию.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.
