XBase (XBase)

На основании пути к базе

Доступен, начиная с версии 8.0.

Синтаксис:

Новый XBase(<ПутьКБазе>, <ПутьКИндексу>, <ТолькоЧтение>)

Параметры:

<ПутьКБазе> (необязательный)

Тип: [Строка](lang__def_String.md).   
Полное имя файла таблицы базы данных в формате DBF.

<ПутьКИндексу> (необязательный)

Тип: [Строка](lang__def_String.md).   
Путь к индексному файлу таблицы.

<ТолькоЧтение> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
[Истина](lang__def_BooleanTrue.md) - открывать файл только для чтения, [Ложь](lang__def_BooleanFalse.md) - открывать файл для чтения и записи.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Создает объект XBase. При создании могут быть сразу открыты файлы таблицы и индекса.

Пример:

|  |
| --- |
| xB = Новый xBase("C:\temp.dbf"); |

См. также:

[XBase](XBase_(XBase).md), метод [ОткрытьФайл](XBase.ОткрытьФайл_(XBase.OpenFile).md)  

Использование в версии:

Доступен, начиная с версии 8.0.
