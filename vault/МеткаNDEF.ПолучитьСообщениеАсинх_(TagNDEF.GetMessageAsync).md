МеткаNDEF (TagNDEF)

ПолучитьСообщениеАсинх (GetMessageAsync)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьСообщениеАсинх()

Возвращаемое значение:

Тип: [СообщениеNDEF](СообщениеNDEF_(MessageNDEF).md).   

Описание:

Получает сообщение NDEF с физической метки.  
В случае ошибки генерируется исключение категории [ОшибкаСредствМультимедиа](КатегорияОшибки.ОшибкаСредствМультимедиа_(ErrorCategory.MultimediaToolsError).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
