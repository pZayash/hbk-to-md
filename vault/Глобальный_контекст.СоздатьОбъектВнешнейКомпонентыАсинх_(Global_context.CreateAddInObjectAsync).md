Глобальный контекст (Global context)

СоздатьОбъектВнешнейКомпонентыАсинх (CreateAddInObjectAsync)

Доступен, начиная с версии 8.3.24.

Синтаксис:

СоздатьОбъектВнешнейКомпонентыАсинх(<Тип>)

Параметры:

<Тип> (необязательный)

Тип: [Строка](lang__def_String.md), [Тип](lang__def_Type.md).   
Имя типа или значение типа [Тип](lang__def_Type.md).

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания – объект внешней компоненты.

Описание:

Асинхронная функция. Создает объект внешней компоненты.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.
