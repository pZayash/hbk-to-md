# Расширение группы формы для подменю.Отображение

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(управляемый).md) › [Группа формы](Группа_формы.md) › [Расширение группы формы для подменю](Расширение_группы_формы_для_подменю_(Form_group_extension_for_a_popup).md) › [Свойства](Расширение_группы_формы_для_подменю__Свойства.md)

Расширение группы формы для подменю (Form group extension for a popup)

Отображение (Representation)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [ОтображениеКнопки](ОтображениеКнопки_(ButtonRepresentation).md).   
Содержит способ отображения подменю.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте требует обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
