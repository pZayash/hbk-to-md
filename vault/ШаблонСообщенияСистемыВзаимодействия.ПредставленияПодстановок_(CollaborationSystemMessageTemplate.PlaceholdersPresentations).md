ШаблонСообщенияСистемыВзаимодействия (CollaborationSystemMessageTemplate)

ПредставленияПодстановок (PlaceholdersPresentations)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Содержит список представлений подстановок в тексте шаблона.  
Фиксированный массив из элементов типа [Строка](lang__def_String.md).  
Количество элементов массива соответствует количеству подстановок в тексте шаблона.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.
