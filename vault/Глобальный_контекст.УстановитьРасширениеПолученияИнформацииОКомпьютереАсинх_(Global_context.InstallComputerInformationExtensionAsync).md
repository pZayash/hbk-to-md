Глобальный контекст (Global context)

УстановитьРасширениеПолученияИнформацииОКомпьютереАсинх (InstallComputerInformationExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьРасширениеПолученияИнформацииОКомпьютереАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Устанавливает расширение для работы с информацией о компьютере в браузер.  
В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
