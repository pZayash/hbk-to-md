СредстваТелефонии (TelephonyTools)

ПоддерживаетсяНаборНомера (NumberDialingSupported)

Доступен, начиная с версии 8.3.5.

Синтаксис:

ПоддерживаетсяНаборНомера(<ВыбратьСразу>)

Параметры:

<ВыбратьСразу> (необязательный)

Тип: Булево, Неопределено.   
Определяет режим проверки поддержки набора номера:

- Истина - проверка выполняется непосредственно.
- Ложь - при проверке требуется подтверждение пользователя.
- Неопределено - проверяется доступность набора номера любым способом.

  
Значение по умолчанию: Неопределено.

Возвращаемое значение:

Тип: Булево.   
С подтверждением пользователя возвращает Истина:

- на Android - при наличии телефонного модуля,
- на iOS - при наличии системного приложения Телефон,
- на Windows Phone - никогда.

Без подтверждения пользователя возвращает Истина:

- на Android - при наличии телефонного модуля,
- на iOS - для iOS младше версии 10.3 при наличии системного приложения Телефон,
- на Windows Phone - всегда на телефоне, никогда на планшете.

Описание:

Определяет, поддерживается ли набор номера на данном устройстве.  
В зависимости от переданного параметра, проверка выполняется непосредственно или с подтверждением пользователем. Если значение переданного параметра - Неопределено, проверяется доступность набора номера любым способом.

Доступность:

Мобильный клиент, мобильное приложение (клиент).
