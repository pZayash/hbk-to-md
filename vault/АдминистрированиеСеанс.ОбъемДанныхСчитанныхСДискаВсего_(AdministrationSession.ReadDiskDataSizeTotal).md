АдминистрированиеСеанс (AdministrationSession)

ОбъемДанныхСчитанныхСДискаВсего (ReadDiskDataSizeTotal)

Доступен, начиная с версии 8.3.14.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит количество данных, прочитанных с диска сеансом с момента начала сеанса (в байтах).

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
