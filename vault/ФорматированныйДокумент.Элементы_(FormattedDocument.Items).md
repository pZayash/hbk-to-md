ФорматированныйДокумент (FormattedDocument)

Элементы (Items)

Доступен, начиная с версии 8.3.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияЭлементовФорматированногоДокумента](КоллекцияЭлементовФорматированногоДокумента_(FormattedDocumentItemCollection).md).   
Содержит коллекцию элементов форматированного документа.  
В коллекцию помещаются элементы только типа [ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента_(FormattedDocumentParagraph).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
