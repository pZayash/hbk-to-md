`ПриЗаписи`(`<Отказ>`)
<!-- signature:end -->

Расширение формы элемента плана счетов (Chart of accounts item form extension)

ПриЗаписи (OnWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПриЗаписи(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает при записи объекта из формы. Процедура-обработчик вызывается после записи объекта (после события [ПриЗаписи](ПланСчетовОбъект.Имя_плана_счетов.ПриЗаписи.md) объекта [ПланСчетовОбъект.](ПланСчетовОбъект.Имя_плана_счетов.md)<[Имя плана счетов](ПланСчетовОбъект.Имя_плана_счетов.md)>) в базу данных, но до окончания транзакции записи.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
