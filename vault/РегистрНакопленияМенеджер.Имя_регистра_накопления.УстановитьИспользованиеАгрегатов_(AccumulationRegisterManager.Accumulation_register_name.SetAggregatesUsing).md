РегистрНакопленияМенеджер.<Имя регистра накопления> (AccumulationRegisterManager.<Accumulation register name>)

УстановитьИспользованиеАгрегатов (SetAggregatesUsing)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьИспользованиеАгрегатов(<Использование>)

Параметры:

<Использование> (обязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Режим использования агрегатов.  
[Истина](lang__def_BooleanTrue.md) – использование агрегатов разрешается.   
[Ложь](lang__def_BooleanFalse.md) – все агрегаты перестают использоваться и очищаются.

Описание:

Устанавливает использование агрегатов.  
Заполнение агрегатов можно выполнить при помощи метода [ОбновитьАгрегаты](РегистрНакопленияМенеджер.Имя_регистра_накопления.ОбновитьАгрегаты_(AccumulationRegisterManager.Accumulation_register_name.UpdateAggregates).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
