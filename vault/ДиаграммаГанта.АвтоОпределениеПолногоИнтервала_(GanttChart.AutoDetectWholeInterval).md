ДиаграммаГанта (GanttChart)

АвтоОпределениеПолногоИнтервала (AutoDetectWholeInterval)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если установлено [Истина](lang__def_BooleanTrue.md), то будет вычислен полный интервал ([НачалоПолногоИнтервала](ДиаграммаГанта.НачалоПолногоИнтервала_(GanttChart.BeginOfWholeInterval).md) и [КонецПолногоИнтервала](ДиаграммаГанта.КонецПолногоИнтервала_(GanttChart.EndOfWholeInterval).md)) как максимальное и минимальное значения времени с учетом максимальной единицы шкалы времени.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
