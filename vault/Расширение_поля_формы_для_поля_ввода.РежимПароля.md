Расширение поля формы для поля ввода (Form field extension for a text box)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Содержит признак специального режима для ввода пароля, в котором все вводимые символы отображаются символом "\*" (звездочка).  
Если тип значения Неопределено, то поведение осуществляется в соответствии с настройками конфигурации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.2.
