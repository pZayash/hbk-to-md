РасчетСистемЛинейныхУравнений (SystemOfLinearEquationsCalculation)

ПолучитьКомпонентыСвязности (GetConnectedComponents)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ПолучитьКомпонентыСвязности(<СпособПолученияКомпонент>)

Параметры:

<СпособПолученияКомпонент> (необязательный)

Тип: [СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.md).   
Определяет способ получения компонент связности.  
В режиме совместимости [Версия8\_3\_14](РежимСовместимости.Версия8_3_14.md) значение по умолчанию – [КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.КомпонентыСильнойСвязностиБезТребованияСвязиВнутриКомпонент.md).  
Значение по умолчанию: [КомпонентыСлабойСвязности](СпособПолученияКомпонентСвязностиРасчетаСистемЛинейныхУравнений.КомпонентыСлабойСвязности.md).

Возвращаемое значение:

Тип: [ТаблицаЗначений](ТаблицаЗначений.md).   
Таблица значений с двумя колонками: в первой колонке содержится номер узла, а во второй - номер компоненты, которой принадлежит узел.

Описание:

Возвращает компоненты связности.   
Для корректной работы метода требуется:

- установить значения свойств [ИсточникДанныхСвязей](РасчетСистемЛинейныхУравнений.ИсточникДанныхСвязей.md), [КолонкаУравненияВСвязях](РасчетСистемЛинейныхУравнений.КолонкаУравненияВСвязях.md), [КолонкаПеременныеВСвязях](РасчетСистемЛинейныхУравнений.КолонкаПеременныеВСвязях.md),
- в [ОписанияСистемЛинейныхУравнений](ОписанияСистемЛинейныхУравнений.md) добавлен объект [ОписаниеСистемыЛинейныхУравнений](ОписаниеСистемыЛинейныхУравнений.md), в котором указана только [КолонкаКоэффициентовВСвязях](ОписаниеСистемыЛинейныхУравнений.КолонкаКоэффициентовВСвязях.md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
