ФрагментДокументаDOM (DOMDocumentFragment)

Атрибуты (Attributes)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияАтрибутовDOM](КоллекцияАтрибутовDOM_(DOMAttributeMap).md).   
Содержит коллекцию атрибутов узла. Коллекция атрибутов доступна только для узла Element.  
Узел Атрибуты

- Attribute - [Неопределено](lang__def_Undefined.md);
- CDATASection - [Неопределено](lang__def_Undefined.md);
- Comment - [Неопределено](lang__def_Undefined.md);
- Document - [Неопределено](lang__def_Undefined.md);
- DocumentFragment - [Неопределено](lang__def_Undefined.md);
- DocumentType - [Неопределено](lang__def_Undefined.md);
- Element - [КоллекцияАтрибутовDOM](КоллекцияАтрибутовDOM_(DOMAttributeMap).md);
- Entity - [Неопределено](lang__def_Undefined.md);
- EntityReference - [Неопределено](lang__def_Undefined.md);
- Notation - [Неопределено](lang__def_Undefined.md);
- ProcessingInstruction - [Неопределено](lang__def_Undefined.md);
- Text - [Неопределено](lang__def_Undefined.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.1.
