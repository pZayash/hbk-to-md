Глобальный контекст (Global context)

ПолучитьБуферДвоичныхДанныхИзBase64БуфераДвоичныхДанных (GetBinaryDataBufferFromBase64BinaryDataBuffer)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьБуферДвоичныхДанныхИзBase64БуфераДвоичныхДанных(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   
Буфер двоичных данных.

Возвращаемое значение:

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных.md).   

Описание:

Преобразует буфер двоичных данных из формата Base64 в [БуферДвоичныхДанных](БуферДвоичныхДанных.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
