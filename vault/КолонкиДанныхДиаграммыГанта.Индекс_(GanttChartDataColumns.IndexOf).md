КолонкиДанныхДиаграммыГанта (GanttChartDataColumns)

Индекс (IndexOf)

Доступен, начиная с версии 8.3.21.

Синтаксис:

Индекс(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаДанныхДиаграммыГанта](КолонкаДанныхДиаграммыГанта_(GanttChartDataColumn).md).   
Колонка, для которой необходимо получить индекс.

Возвращаемое значение:

Тип: Число.   

Описание:

Возвращает индекс элемента коллекции.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.21.
