Расширение формы клиентского приложения для отчета (Client application form extension for reports)

СформироватьПриОткрытии (GenerateOnOpen)

Доступен, начиная с версии 8.2.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает необходимость формирования отчета при открытии.  
[Истина](lang__def_BooleanTrue.md) - отчет будет выполнен сразу на сервере, после обработчика события [ПриСозданииНаСервере](ФормаКлиентскогоПриложения.ПриСозданииНаСервере_(ClientApplicationForm.OnCreateAtServer).md).  
Если в обработчике события установить СтандартнаяОбработка = [Ложь](lang__def_BooleanFalse.md), то параметр не отрабатывается, т.е. отчет не формируется.

Использование в версии:

Доступен, начиная с версии 8.2.
