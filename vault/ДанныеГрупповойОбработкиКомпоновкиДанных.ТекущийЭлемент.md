Только чтение.

Описание:

Тип: [СтрокаТаблицыЗначений](СтрокаТаблицыЗначений.md), Неопределено.   
Содержит текущую строку таблицы значений, находящейся в свойстве [Данные](ДанныеГрупповойОбработкиКомпоновкиДанных.Данные.md).  
Для итоговых строк будет содержать значение Неопределено.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
