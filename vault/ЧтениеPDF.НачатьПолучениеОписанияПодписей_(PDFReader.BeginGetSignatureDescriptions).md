ЧтениеPDF (PDFReader)

НачатьПолучениеОписанияПодписей (BeginGetSignatureDescriptions)

Доступен, начиная с версии 8.3.16.

Синтаксис:

НачатьПолучениеОписанияПодписей(<ОписаниеОповещения>, <МенеджерКриптографии>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).   
Содержит описание процедуры, которая будет вызвана после завершения операции со следующими параметрами:

- <ОписанияПодписей> - тип [Массив](Массив_(Array).md). В параметре возвращается массив, содержащий объекты типа [ОписаниеПодписиPDF](ОписаниеПодписиPDF_(PDFSignatureDescription).md).
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).

<ОписанияПодписей> - тип [Массив](Массив_(Array).md). В параметре возвращается массив, содержащий объекты типа [ОписаниеПодписиPDF](ОписаниеПодписиPDF_(PDFSignatureDescription).md).  
<ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения_(CallbackDescription).md).





<МенеджерКриптографии> (обязательный)

Тип: [МенеджерКриптографии](МенеджерКриптографии_(CryptoManager).md).   
Менеджер криптографии.

Описание:

Инициирует получение массива, содержащего описания присоединенных к PDF-документу электронных подписей. В веб-клиенте требуется установка расширения для работы с криптографией.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

[ЧтениеPDF](ЧтениеPDF_(PDFReader).md), метод [ПолучитьОписанияПодписейАсинх](ЧтениеPDF.ПолучитьОписанияПодписейАсинх_(PDFReader.GetSignatureDescriptionsAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
