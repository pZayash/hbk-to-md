HTTPСоединение (HTTPConnection)

ИспользоватьАутентификациюОС (UseOSAuthentication)

Доступен, начиная с версии 8.3.7.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит текущее значение использования аутентификации NTLM или Negotiate на сервере.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

В веб-клиенте всегда возвращает значение Истина.

Использование в версии:

Доступен, начиная с версии 8.3.7.

Описание изменено в версии 8.3.21.
