Глобальный контекст (Global context)

ПолучитьИмяКлиентаЛицензирования (GetLicensingClientName)

Доступен, начиная с версии 8.3.7.

Синтаксис:

ПолучитьИмяКлиентаЛицензирования()

Возвращаемое значение:

Тип: [Строка](lang__def_String.md).   
Имя клиента лицензирования.

Описание:

Возвращает имя клиента лицензирования из информационной базы.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только пользователю с административными правами.

Пример:

|  |
| --- |
| ИмяКлиентаЛицензирования = ПолучитьИмяКлиентаЛицензирования(); |

Использование в версии:

Доступен, начиная с версии 8.3.7.
