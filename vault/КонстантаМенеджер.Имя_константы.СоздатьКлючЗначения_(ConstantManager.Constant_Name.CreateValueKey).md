# КонстантаМенеджер.<Имя константы>.СоздатьКлючЗначения

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Константы](Константы.md) › [КонстантаМенеджер.<Имя константы>](КонстантаМенеджер.Имя_константы_(ConstantManager.Constant_Name).md) › [Методы](КонстантаМенеджер.Имя_константы__Методы.md)

КонстантаМенеджер.<Имя константы> (ConstantManager.<Constant Name>)

СоздатьКлючЗначения (CreateValueKey)

Доступен, начиная с версии 8.3.13.

Синтаксис:

СоздатьКлючЗначения(<ЗначениеКлюча>)

Параметры:

<ЗначениеКлюча> (необязательный)

Тип: [Структура](Структура_(Structure).md), [Неопределено](lang__def_Undefined.md).   
В качестве ключа структуры используется имя общего реквизита, в качестве значения – необходимое значение этого поля.   
Если параметр не задан или равен [Неопределено](lang__def_Undefined.md), будет создан пустой ключ константы.

Возвращаемое значение:

Тип: [КонстантаКлючЗначения.\<Имя константы\>](КонстантаКлючЗначения.Имя_константы_(ConstantValueKey.Constant_name).md).   

Описание:

Создает ключ значения константы на основе переданных параметров.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.
