ПараметрПеретаскиванияВнутриПланировщика (PlannerInsideDragParameter)

ЗначенияИзмерений (DimensionValues)

Доступен, начиная с версии 8.3.18.

Использование:

Чтение и запись.

Описание:

Тип: [Соответствие](Соответствие_(Map).md).   
Коллекция значений измерений. В качестве ключа соответствия выступает Значение измерения, в качестве значения выступает Значение элемента измерения.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.18.
