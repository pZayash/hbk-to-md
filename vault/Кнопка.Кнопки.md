Кнопка (Button)

Кнопки (Buttons)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [КнопкиКоманднойПанели](КнопкиКоманднойПанели.md).   
Содержит коллекцию кнопок, образующих командную панель.

Доступность:

Толстый клиент.

Примечание:

При ее нажатии кнопки выдается меню или меню будет выдаваться только при нажатии в треугольник в правой части кнопки.

Использование в версии:

Доступен, начиная с версии 8.0.
