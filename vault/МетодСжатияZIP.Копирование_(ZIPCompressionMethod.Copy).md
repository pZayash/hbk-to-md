МетодСжатияZIP (ZIPCompressionMethod)

Копирование (Copy)

Не рекомендуется использовать, начиная с версии 8.3.26.

Рекомендуется использовать:

- [Копирование](МетодСжатияФайлаАрхива.Копирование_(ArchiveFileCompressionMethod.Copy).md)

Доступен, начиная с версии 8.0.

Описание:

Файлы при помещении в архив не сжимаются, а копируются.  
Требуется минимальное время работы.

Использование в версии:

Доступен, начиная с версии 8.0.

Не рекомендуется использовать, начиная с версии 8.3.26.
