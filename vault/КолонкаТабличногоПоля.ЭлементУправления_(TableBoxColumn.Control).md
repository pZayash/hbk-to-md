ЭлементУправления (Control)

Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПолеВвода](ПолеВвода_(TextBox).md), [ПолеВыбора](ПолеВыбора_(ComboBox).md), [Флажок](Флажок_(CheckBox).md).

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
