# Расширение поля ввода системного перечисления

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Интерфейс](Интерфейс_(обычный).md) › [Элементы управления](Элементы_управления.md) › [Поле ввода](Поле_ввода.md)

Доступен, начиная с версии 8.0.

Свойства:

[ДоступныеЗначения (AvailableValues)](Расширение_поля_ввода_системного_перечисления.ДоступныеЗначения_(System_enumeration_text_box_extension.AvailableValues).md)  

Описание:

Дополнительные свойства элемента управления [ПолеВвода](ПолеВвода_(TextBox).md), предназначенного для ввода ссылки на системное перечисление. Предназначен для задания набора значений перечисления, которые попадают в список выбора. Порядок следования значений в списке выбора в этом случае совпадает с порядком в списке доступных значений. В случае, если список пустой, доступными являются все значения.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
