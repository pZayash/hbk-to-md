МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьОбсуждение (GetConversation)

Доступен, начиная с версии 8.3.10.

Вариант синтаксиса: По идентификатору

Синтаксис:

ПолучитьОбсуждение(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторОбсужденияСистемыВзаимодействия](ИдентификаторОбсужденияСистемыВзаимодействия.md).   
Идентификатор обсуждения.

Описание варианта метода:

Возвращает обсуждение по идентификатору.

Вариант синтаксиса: По ключу

Синтаксис:

ПолучитьОбсуждение(<Ключ>)

Параметры:

<Ключ> (необязательный)

Тип: Строка.   
Ключ обсуждения.  
Значение по умолчанию: "".

Описание варианта метода:

Возвращает неконтекстное обсуждение по заданному ключу. Если обсуждение не найдено, возвращает Неопределено.

Возвращаемое значение:

Тип: [ОбсуждениеСистемыВзаимодействия](ОбсуждениеСистемыВзаимодействия.md).   

Описание:

Получает обсуждение по его идентификатору.

Доступность:

Сервер, толстый клиент, внешнее соединение.
