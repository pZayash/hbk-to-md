РежимИспользованияПараметраКоманды (CommandParameterUseMode)

Множественный (Multiple)

Доступен, начиная с версии 8.2.

Описание:

В обработчик команды передается массив значений, а не одно значение даже, если в конкретном случае значение одно.

Использование в версии:

Доступен, начиная с версии 8.2.
