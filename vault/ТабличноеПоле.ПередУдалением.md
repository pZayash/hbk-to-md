ТабличноеПоле (TableBox)

ПередУдалением (BeforeDeleteRow)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередУдалением(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от удаления строки. Если в теле процедуры-обработчика установить данному параметру значение Истина, удаление строки табличного поля не будет выполнено.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед удалением строки или группы строк табличного поля.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
