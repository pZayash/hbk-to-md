ПолеСписка (ListBox)

ТолькоПросмотр (ReadOnly)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Определяет возможность редактирования поля списка. Если значение свойства Истина, то возможен только просмотр поля списка.

Доступность:

Толстый клиент.

Примечание:

В отличие от свойства [Доступность](ПолеСписка.Доступность.md), если свойство ТолькоПросмотр равно Истина, становится невозможным редактирование поля списка, однако на остальную работу с полем это не влияет. Например, можно выделить строковое представление содержимого поля списка и скопировать в буфер обмена.

Использование в версии:

Доступен, начиная с версии 8.0.
