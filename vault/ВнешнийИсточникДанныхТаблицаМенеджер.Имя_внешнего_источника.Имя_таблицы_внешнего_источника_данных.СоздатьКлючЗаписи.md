ВнешнийИсточникДанныхТаблицаМенеджер.<Имя внешнего источника>.<Имя таблицы внешнего источника данных> (ExternalDataSourceTableManager.<External source name>.<External data source table name>)

СоздатьКлючЗаписи (CreateRecordKey)

Доступен, начиная с версии 8.3.3.

Синтаксис:

СоздатьКлючЗаписи(<ЗначенияКлюча>)

Параметры:

<ЗначенияКлюча> (обязательный)

Тип: [Структура](Структура.md).   
Структура, содержащая значения для заполнения свойств ключа записи.  
Имена элементов структуры должны соответствовать именам ключевых полей.

Возвращаемое значение:

Тип: [ВнешнийИсточникДанныхТаблицаКлючЗаписи.\<Имя внешнего источника\>.\<Имя таблицы внешнего источника данных\>](ВнешнийИсточникДанныхТаблицаКлючЗаписи.Имя_внешнего_источника.Имя_таблицы_внешнего_источника_данных.md).   

Описание:

Создает ключ записи по переданной структуре.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Доступен только для таблиц с типом данных [НеобъектныеДанные](ТипДанныхТаблицыВнешнегоИсточникаДанных.НеобъектныеДанные.md).

Использование в версии:

Доступен, начиная с версии 8.3.3.
