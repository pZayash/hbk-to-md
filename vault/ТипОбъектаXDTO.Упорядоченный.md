ТипОбъектаXDTO (XDTOObjectType)

Упорядоченный (Ordered)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: Булево.   
Содержит признак упорядоченности свойств объекта данного типа.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Истина - порядок следования свойств в объекте должен соответствовать порядку следования свойств в типе объекта. В противном случае порядок следования свойств может быть произвольным.

Использование в версии:

Доступен, начиная с версии 8.1.
