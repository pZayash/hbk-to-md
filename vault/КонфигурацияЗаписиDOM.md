Описание:

Конфигурация объекта записи DOM.  
"discard-default-content" - признак удаления неявных атрибутов (не присутствующих в исходном документе, но добавленных по схеме) (Булево);  
"xml-declaration" - признак необходимости записывать объявление XML (Булево).

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
