ГруппировкаКолонок (ColumnsGroup)

ВЯчейке (InCell)

Доступен, начиная с версии 8.2.

Описание:

Группировка колонок в одной ячейке. При этом варианте текстовые значения отображаются через запятую. Редактирование возможно только для первого значения. Помимо текстовых возможно отображение одного значения в виде флажка и одного в виде картинки.

Использование в версии:

Доступен, начиная с версии 8.2.
