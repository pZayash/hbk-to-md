Глобальный контекст (Global context)

УстановитьРасширениеРаботыСФайлами (InstallFileSystemExtension)

Не рекомендуется использовать, начиная с версии 8.3.16.

Рекомендуется использовать:

- [НачатьУстановкуРасширенияРаботыСФайлами](Глобальный_контекст.НачатьУстановкуРасширенияРаботыСФайлами_(Global_context.BeginInstallFileSystemExtension).md)

Доступен, начиная с версии 8.2.

Синтаксис:

УстановитьРасширениеРаботыСФайлами()

Описание:

Устанавливает расширение работы с файлами.  
В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.  
Для клиента мобильного приложения, информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [НачатьУстановкуРасширенияРаботыСФайлами](Глобальный_контекст.НачатьУстановкуРасширенияРаботыСФайлами_(Global_context.BeginInstallFileSystemExtension).md).
