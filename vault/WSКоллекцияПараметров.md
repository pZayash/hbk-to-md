# WSКоллекцияПараметров

**↑** <a href="obsidian://open?file=_index.md">Главная</a> › <a href="obsidian://open?file=_index__Объекты.md">Объекты</a> › <a href="obsidian://open?file=Общие_объекты.md">Общие объекты</a> › [Web-сервисы](objects__catalog63__catalog1031.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[WSПараметр](WSПараметр.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются элементы коллекции.  
Возможно обращение к элементу коллекции посредством оператора [...]. В качестве аргумента передается индекс коллекции (нумерация с 0).  

Методы:

[Количество (Count)](WSКоллекцияПараметров.Количество.md)  
[Получить (Get)](WSКоллекцияПараметров.Получить.md)  

Описание:

Коллекция параметров.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

См. также:

[WSОперация](WSОперация.md), свойство [Параметры](WSОперация.Параметры.md)  

Использование в версии:
