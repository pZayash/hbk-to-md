МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВыполнитьРегистрациюВыгрузкиДанныхАсинх (RegisterDataDumpAsync)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ВыполнитьРегистрациюВыгрузкиДанныхАсинх(<ПутьКФайлу>, <Описание>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: Строка.   
Путь к файлу выгрузки данных на сервере системы взаимодействия.

<Описание> (обязательный)

Тип: Строка.   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [Обещание](Обещание.md).   
Результат выполнения обещания – [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия.md).

Описание:

Асинхронная функция. Регистрирует выгрузку данных, если администратор самостоятельно записал ее на сервер взаимодействия.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия.md), метод [ВыполнитьРегистрациюВыгрузкиДанных](МенеджерСистемыВзаимодействия.ВыполнитьРегистрациюВыгрузкиДанных.md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
