# HTTPСервисОтвет.УстановитьИмяФайлаТела

HTTPСервисОтвет (HTTPServiceResponse)

УстановитьИмяФайлаТела (SetBodyFileName)

Доступен, начиная с версии 8.3.5.

Синтаксис:

УстановитьИмяФайлаТела(<ИмяФайла>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Строка, содержащая имя файла. Чтение файла будет происходить в момент формирования HTTP-ответа.

Описание:

Устанавливает имя файла, из которого будет прочитано тело HTTP-ответа.

Доступность:

Сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.5.
