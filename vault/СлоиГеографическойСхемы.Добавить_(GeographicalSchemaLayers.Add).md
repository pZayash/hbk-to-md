СлоиГеографическойСхемы (GeographicalSchemaLayers)

Добавить (Add)

Доступен, начиная с версии 8.0.

Вариант синтаксиса: Добавить пустой слой объектов

Синтаксис:

Добавить(<Имя>, <ТипОбъектов>)

Параметры:

<Имя> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя, под которым слой будет добавлен в коллекцию.

<ТипОбъектов> (обязательный)

Тип: [ТочечныйОбъектГеографическойСхемы](ТочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPoint).md), [МноготочечныйОбъектГеографическойСхемы](МноготочечныйОбъектГеографическойСхемы_(GeographicalSchemaObjectMultiPoint).md), [ПолилинейныйОбъектГеографическойСхемы](ПолилинейныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolyline).md), [ПолигональныйОбъектГеографическойСхемы](ПолигональныйОбъектГеографическойСхемы_(GeographicalSchemaObjectPolygon).md).   
Тип топологических объектов географической схемы, которые будет хранить слой.

Описание варианта метода:

Добавляет слой, хранящий указанный тип объектов, в коллекцию.

Вариант синтаксиса: Добавить слой объектов из файла

Синтаксис:

Добавить(<ИмяФайла>, <ТипИмпорта>)

Параметры:

<ИмяФайла> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя файла топологической схемы формата ESRI Shapefile (расширение "shp"), из которого будет осуществляться импорт.

<ТипИмпорта> (необязательный)

Тип: [ТипИмпортаСерийСлояГеографическойСхемы](ТипИмпортаСерийСлояГеографическойСхемы_(GeographicalSchemaLayerSeriesImportModeType).md).   
Указывает вид действия с присоединёнными к файлу топологической схемы данными.  
Значение по умолчанию: [ИмпортироватьВсе](ТипИмпортаСерийСлояГеографическойСхемы.ИмпортироватьВсе_(GeographicalSchemaLayerSeriesImportModeType.ImportAll).md).

Описание варианта метода:

Импортирует в коллекцию новый слой из SHP-файла ArcGIS.

Возвращаемое значение:

Тип: [СлойГеографическойСхемы](СлойГеографическойСхемы_(GeographicalSchemaLayer).md).   

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Схема = Новый ГеографическаяСхема(); Слой = Схема.Слои.Добавить("Слой",              Тип("ПолигональныйОбъектГеографическойСхемы")); НовыйПолигональныйОбъект = Слой.Объекты.Добавить(); Схема.Показать(); |

Использование в версии:

Доступен, начиная с версии 8.0.
