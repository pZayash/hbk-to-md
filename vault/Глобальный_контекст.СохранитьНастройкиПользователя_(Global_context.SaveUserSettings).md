Глобальный контекст (Global context)

СохранитьНастройкиПользователя (SaveUserSettings)

Доступен, начиная с версии 8.1.

Синтаксис:

СохранитьНастройкиПользователя()

Описание:

Выполняет сохранение пользовательских настроек в процессе работы, не дожидаясь завершения приложения.

Доступность:

Толстый клиент.

Примечание:

Важно учитывать, что настройки форм формируются при закрытии формы. Поэтому для незакрытых окон метод не будет сохранять новые варианты настроек.

Использование в версии:

Доступен, начиная с версии 8.1.
