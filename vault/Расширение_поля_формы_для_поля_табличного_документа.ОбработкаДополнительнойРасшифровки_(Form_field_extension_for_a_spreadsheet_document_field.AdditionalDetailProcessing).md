Расширение поля формы для поля табличного документа (Form field extension for a spreadsheet document field)

ОбработкаДополнительнойРасшифровки (AdditionalDetailProcessing)

Доступен, начиная с версии 8.2.

Синтаксис:

ОбработкаДополнительнойРасшифровки(<Расшифровка>, <СтандартнаяОбработка>, <ДополнительныеПараметры>)

Параметры:

<Расшифровка>

Тип: Произвольный.   
Значение расшифровки.

<СтандартнаяОбработка>

Тип: [Булево](lang__Булево_(Boolean).md).   
В данный параметр передается признак выполнения стандартной (системной) обработки события. Если в теле процедуры-обработчика установить данному параметру значение [Ложь](lang__def_BooleanFalse.md), стандартная обработка события производиться не будет. Отказ от стандартной обработки не отменяет действие.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

<ДополнительныеПараметры>

Тип: [Структура](Структура_(Structure).md).   
Если событие сформировано элементом формы [ДиаграммаГанта](ДиаграммаГанта_(GanttChart).md) и выполняется обработка расшифровки для значения или интервала диаграммы Ганта, то в этот параметр помещается структура со свойством Дата(Date) типа [Дата](lang__def_Date.md), значение которого соответствует точке выбора.  
В других случаях этот параметр будет содержать пустую структуру.  
Параметр недоступен в режиме совместимости конфигурации [Версия8\_3\_9](РежимСовместимости.Версия8_3_9_(CompatibilityMode.Version8_3_9).md) и ниже.

Описание:

Вызывается при нажатии на правую кнопку мыши или клавишу меню на клавиатуре, на ячейке или рисунке, содержащей значение расшифровки.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.

Описание изменено в версии 8.3.27.
