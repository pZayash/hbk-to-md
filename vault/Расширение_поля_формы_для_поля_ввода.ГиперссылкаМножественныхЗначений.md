Расширение поля формы для поля ввода (Form field extension for a text box)

ГиперссылкаМножественныхЗначений (MultipleValuesHyperlink)

Доступен, начиная с версии 8.3.23.

Использование:

Чтение и запись.

Описание:

Тип: Булево, Неопределено.   
Содержит признак того, что множественное значение отображается в виде гиперссылки:

- Истина – текст множественных значений в виде гиперссылки.

- Ложь — гиперссылка отсутствует.

- Неопределено – гиперссылка множественных значений отображается только для ссылочных типов данных.

Значение по умолчанию: Неопределено.  
Используется, если в поле ввода редактируются данные типа [СписокЗначений](СписокЗначений.md) или [ДанныеФормыКоллекция](ДанныеФормыКоллекция.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.23.
