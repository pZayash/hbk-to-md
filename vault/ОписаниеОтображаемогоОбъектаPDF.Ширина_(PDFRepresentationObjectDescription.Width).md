ОписаниеОтображаемогоОбъектаPDF (PDFRepresentationObjectDescription)

Ширина (Width)

Доступен, начиная с версии 8.3.16.

Использование:

Чтение и запись.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит ширину объекта (в миллиметрах) на странице PDF-документа.   
Если не указано, то ширина будет определена автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
