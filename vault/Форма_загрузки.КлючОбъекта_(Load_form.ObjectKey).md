Форма загрузки (Load form)

КлючОбъекта (ObjectKey)

Доступен, начиная с версии 8.2.

Описание:

Идентификатор объекта, для которого будет загружаться настройка.

Использование в версии:

Доступен, начиная с версии 8.2.
