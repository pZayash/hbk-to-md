Расширение формы набора записей регистра накопления (Accumulation register record set form extension)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>)

Параметры:

<Отказ>

Тип: Булево.   
Признак отказа от записи объекта. Если в теле процедуры-обработчика установить данному параметру значение Истина, то запись объекта выполнена не будет.  
Значение по умолчанию: Ложь.

Описание:

Возникает перед выполнением записи набора из формы. Процедура-обработчик вызывается перед вызовом процедуры-обработчика события [ПередЗаписью](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.ПередЗаписью.md) объекта [РегистрНакопленияНаборЗаписей.](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)<[Имя регистра накопления](РегистрНакопленияНаборЗаписей.Имя_регистра_накопления.md)>.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
