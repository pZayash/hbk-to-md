Глобальный контекст (Global context)

ОсновнойСерверДоступен (MainServerAvailable)

Доступен, начиная с версии 8.3.16.

Синтаксис:

ОсновнойСерверДоступен()

Возвращаемое значение:

Тип: Булево, Неопределено.   

Описание:

Показывает доступность основного сервера с клиента или автономного сервера.   
Неопределено - в случае, если отсутствует возможность автономной работы (используется мобильное приложение или мобильный клиент без возможности автономной работы).

Доступность:

Мобильный клиент, мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
