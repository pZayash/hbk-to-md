ЭлементШкалыВремени (TimeScaleItem)

Видимость (Visible)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: Булево.   
Позволяет без добавления или удаления элементов управлять составом шкалы времени.  
Элемент шкалы может стать невидимым и автоматически, в зависимости от масштаба. На значение данного свойства автоматическое выключение влияния не оказывает.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Примечание:

Ложь - элемент шкалы не отображается.

См. также:

[ЭлементШкалыВремени](ЭлементШкалыВремени.md), свойство [Формат](ЭлементШкалыВремени.Формат.md)  

Использование в версии:

Доступен, начиная с версии 8.0.
