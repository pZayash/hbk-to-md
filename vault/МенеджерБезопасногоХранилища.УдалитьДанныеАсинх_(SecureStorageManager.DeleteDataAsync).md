МенеджерБезопасногоХранилища (SecureStorageManager)

УдалитьДанныеАсинх (DeleteDataAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УдалитьДанныеАсинх(<Ключ>)

Параметры:

<Ключ> (обязательный)

Тип: [Строка](lang__def_String.md).   
Ключ, под которым были сохранены данные.

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - тип [Булево](lang__Булево_(Boolean).md). Признак того, что проверка пользователя была отменена пользователем.  
Если переданный ключ не существует, будет сгенерировано сообщение об ошибке.

Описание:

Асинхронная функция. Удаляет данные из защищенного хранилища.  
Вызов метода не блокирует исполнение кода языка, все результаты возвращаются в переданный обработчик оповещения.   
Выполняемая операция не является интерактивной и не может быть отменена пользователем.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.18.
