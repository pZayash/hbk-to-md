Глобальный контекст (Global context)

ПолучитьБуферДвоичныхДанныхИзHexБуфераДвоичныхДанных (GetBinaryDataBufferFromHexBinaryDataBuffer)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьБуферДвоичныхДанныхИзHexБуфераДвоичныхДанных(<Буфер>)

Параметры:

<Буфер> (обязательный)

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   
Буфер двоичных данных в формате Base 16 (Hex).

Возвращаемое значение:

Тип: [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).   

Описание:

Преобразует буфер двоичных данных из формата Base 16 (Hex) в [БуферДвоичныхДанных](БуферДвоичныхДанных_(BinaryDataBuffer).md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.10.
