# ПоляВыбораКомпоновкиДанныхСхемыЗапроса.Добавить

ПоляВыбораКомпоновкиДанныхСхемыЗапроса (QuerySchemaDataCompositionSelectionFields)

Добавить (Add)

Доступен, начиная с версии 8.3.14.

Вариант синтаксиса: Из строки

Синтаксис:

Добавить(<Имя колонки>)

Параметры:

<Имя колонки> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя колонки схемы запроса, добавляемой в поля выбора компоновки данных.

Описание варианта метода:

Добавляет поле выбора компоновки данных из строки.

Вариант синтаксиса: Из колонки

Синтаксис:

Добавить(<Колонка>)

Параметры:

<Колонка> (обязательный)

Тип: [КолонкаСхемыЗапроса](КолонкаСхемыЗапроса_(QuerySchemaColumn).md).   
Колонка схемы запроса, добавляемая в поля выбора компоновки данных.

Описание варианта метода:

Добавляет поле выбора компоновки данных из колонки схемы запроса.

Возвращаемое значение:

Тип: [ПолеВыбораКомпоновкиДанныхСхемыЗапроса](ПолеВыбораКомпоновкиДанныхСхемыЗапроса_(QuerySchemaDataCompositionSelectionField).md).   

Описание:

Добавляет новый элемент коллекции. Если тип нового элемента не подходит для данной коллекции, добавление не выполняется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.14.
