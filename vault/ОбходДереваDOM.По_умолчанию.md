ОбходДереваDOM (DOMTreeWalker)

На основании документа DOM

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ОбходДереваDOM(<Документ>, <Узел>, <Отбор>, <ОбходитьСсылкиНаСущности>)

Параметры:

<Документ> (обязательный)

Тип: [ДокументDOM](ДокументDOM.md).   
Документ DOM, которому принадлежат узлы.

<Узел> (необязательный)

Тип: [АтрибутDOM](АтрибутDOM.md), [ДокументDOM](ДокументDOM.md), [ЭлементDOM](ЭлементDOM.md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM.md), [НотацияDOM](НотацияDOM.md), [СущностьDOM](СущностьDOM.md), [ФрагментДокументаDOM](ФрагментДокументаDOM.md), [ТекстDOM](ТекстDOM.md), [КомментарийDOM](КомментарийDOM.md), [СекцияCDATADOM](СекцияCDATADOM.md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM.md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM.md), [ПространствоИменXPath](ПространствоИменXPath.md).   
Если узел не указан, то в качестве корневого узла будет использован документ DOM.

<Отбор> (необязательный)

Тип: [ФильтрУзловDOM](ФильтрУзловDOM.md).   
Отбор узлов DOM. Если отбор не указан, то будут проитерированы все узлы.

<ОбходитьСсылкиНаСущности> (необязательный)

Тип: Булево.   
Признак обхода ссылок на сущности DOM. Истина - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.  
Значение по умолчанию: Истина.

Описание:

Создает экземпляр объекта.

Использование в версии:

Доступен, начиная с версии 8.1.
