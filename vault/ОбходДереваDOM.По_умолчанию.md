ОбходДереваDOM (DOMTreeWalker)

На основании документа DOM

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ОбходДереваDOM(<Документ>, <Узел>, <Отбор>, <ОбходитьСсылкиНаСущности>)

Параметры:

<Документ> (обязательный)

Тип: [ДокументDOM](ДокументDOM_(DOMDocument).md).   
Документ DOM, которому принадлежат узлы.

<Узел> (необязательный)

Тип: [АтрибутDOM](АтрибутDOM_(DOMAttribute).md), [ДокументDOM](ДокументDOM_(DOMDocument).md), [ЭлементDOM](ЭлементDOM_(DOMElement).md), [ОпределениеТипаДокументаDOM](ОпределениеТипаДокументаDOM_(DOMDocumentType).md), [НотацияDOM](НотацияDOM_(DOMNotation).md), [СущностьDOM](СущностьDOM_(DOMEntity).md), [ФрагментДокументаDOM](ФрагментДокументаDOM_(DOMDocumentFragment).md), [ТекстDOM](ТекстDOM_(DOMText).md), [КомментарийDOM](КомментарийDOM_(DOMComment).md), [СекцияCDATADOM](СекцияCDATADOM_(DOMCDATASection).md), [ИнструкцияОбработкиDOM](ИнструкцияОбработкиDOM_(DOMProcessingInstruction).md), [СсылкаНаСущностьDOM](СсылкаНаСущностьDOM_(DOMEntityReference).md), [ПространствоИменXPath](ПространствоИменXPath_(XPathNamespace).md).   
Если узел не указан, то в качестве корневого узла будет использован документ DOM.

<Отбор> (необязательный)

Тип: [ФильтрУзловDOM](ФильтрУзловDOM_(DOMNodeFilter).md).   
Отбор узлов DOM. Если отбор не указан, то будут проитерированы все узлы.

<ОбходитьСсылкиНаСущности> (необязательный)

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак обхода ссылок на сущности DOM. [Истина](lang__def_BooleanTrue.md) - при обходе узла ссылки на сущность будет производиться обход узлов содержимого сущности, в противном случае обход производиться не будет.  
Значение по умолчанию: [Истина](lang__def_BooleanTrue.md).

Описание:

Создает экземпляр объекта.

Использование в версии:

Доступен, начиная с версии 8.1.
