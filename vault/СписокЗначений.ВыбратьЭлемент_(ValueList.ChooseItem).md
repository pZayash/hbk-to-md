СписокЗначений (ValueList)

ВыбратьЭлемент (ChooseItem)

Доступен, начиная с версии 8.0.

Синтаксис:

ВыбратьЭлемент(<Заголовок>, <Элемент>)

Параметры:

<Заголовок> (необязательный)

Тип: [Строка](lang__def_String.md).   
Заголовок окна выбора. Если параметр не указан, то используется стандартный заголовок.  
Значение по умолчанию: Пустая строка.

<Элемент> (необязательный)

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md).   
Элемент, на который следует изначально спозиционировать список значений при интерактивном выборе. Если значением параметра не является элемент списка значений, входящий в данный список, позиционирования не произойдет.

Возвращаемое значение:

Тип: [ЭлементСпискаЗначений](ЭлементСпискаЗначений_(ValueListItem).md), [Неопределено](lang__def_Undefined.md).   
Если пользователь отказался от выбора, возвращает [Неопределено](lang__def_Undefined.md).

Описание:

Вызывает окно для интерактивного выбора одного из элементов, входящих в список значений.

Доступность:

Тонкий клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

Примечание:

Если для конфигурации свойство [РежимИспользованияМодальности](ОбъектМетаданныхКонфигурация.РежимИспользованияМодальности_(ConfigurationMetadataObject.ModalityUseMode).md) установлено в [НеИспользовать](РежимИспользованияМодальности.НеИспользовать_(ModalityUseMode.DontUse).md), следует использовать метод [ПоказатьВыборЭлемента](СписокЗначений.ПоказатьВыборЭлемента_(ValueList.ShowChooseItem).md).

Пример:

|  |
| --- |
| ВыбЭлемент = СписокТиповЦен.НайтиПоЗначению(ВыбЗначение); ВыбЭлемент = СписокТиповЦен.ВыбратьЭлемент("Выберите тип цен.", ВыбЭлемент); Если ВыбЭлемент = Неопределено Тогда      Сообщить("Не выбран тип цен."); Иначе     ВыбЗначение = ВыбЭлемент.Значение; КонецЕсли; |
