ЭлементУсловногоОформленияКомпоновкиДанных (DataCompositionConditionalAppearanceItem)

ИспользоватьВОтборе (UseInFilter)

Доступен, начиная с версии 8.3.3.

Использование:

Чтение и запись.

Описание:

Тип: [ИспользованиеУсловногоОформленияКомпоновкиДанных](ИспользованиеУсловногоОформленияКомпоновкиДанных.md).   
Указывает необходимость применения условного оформления области, в которой выводятся отборы. Значение по умолчанию [Использовать](ИспользованиеУсловногоОформленияКомпоновкиДанных.Использовать.md).  
Данное свойство недоступно у объектов, содержащихся в коллекции пользовательских настроек компоновки данных.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.3.
