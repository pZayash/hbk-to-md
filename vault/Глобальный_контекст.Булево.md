`Булево`(`<Значение>`: `Число, Булево, Строка`) → `Булево`
<!-- signature:end -->

Глобальный контекст (Global context)

Булево (Boolean)

Доступен, начиная с версии 8.0.

Синтаксис:

Булево(<Значение>)

Параметры:

<Значение> (обязательный)

Тип: Число, Булево, Строка.   
Исходное значение.

Возвращаемое значение:

Тип: Булево.   
Полученное значение.

Описание:

Преобразует полученный параметр в значение типа Булево.  
Преобразование числа к типу Булево производится по следующим правилам:

- 0 преобразуется в Ложь;
- остальные значения преобразуются в Истина.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

  
**Внимание:** Данный метод не поддерживается при работе через COM-соединение

Пример:

|  |
| --- |
| Если ТипЗнч(Реквизит) = Тип("Булево")  Тогда     Значение = Булево(Значение); КонецЕсли; |

Использование в версии:

Доступен, начиная с версии 8.0.
