МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ВыполнитьРегистрациюВыгрузкиДанных (RegisterDataDump)

Доступен, начиная с версии 8.3.22.

Синтаксис:

ВыполнитьРегистрациюВыгрузкиДанных(<ПутьКФайлу>, <Описание>)

Параметры:

<ПутьКФайлу> (обязательный)

Тип: [Строка](lang__def_String.md).   
Путь к файлу с выгрузкой данных на сервере системы взаимодействия.

<Описание> (обязательный)

Тип: [Строка](lang__def_String.md).   
Описание выгрузки данных.

Возвращаемое значение:

Тип: [ИдентификаторВыгрузкиДанныхСистемыВзаимодействия](ИдентификаторВыгрузкиДанныхСистемыВзаимодействия_(CollaborationSystemDataDumpID).md).   

Описание:

Регистрирует выгрузку данных, если администратор самостоятельно записал ее на сервер взаимодействия.

Доступность:

Сервер, толстый клиент, внешнее соединение.

См. также:

[МенеджерСистемыВзаимодействия](МенеджерСистемыВзаимодействия_(CollaborationSystemManager).md), метод [ВыполнитьРегистрациюВыгрузкиДанныхАсинх](МенеджерСистемыВзаимодействия.ВыполнитьРегистрациюВыгрузкиДанныхАсинх_(CollaborationSystemManager.RegisterDataDumpAsync).md)  

Использование в версии:

Доступен, начиная с версии 8.3.22.
