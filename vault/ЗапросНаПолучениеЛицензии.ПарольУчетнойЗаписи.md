ЗапросНаПолучениеЛицензии (LicenseAcquisitionRequest)

ПарольУчетнойЗаписи (AccountPassword)

Доступен, начиная с версии 8.3.23.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Пароль учетной записи пользователя лицензии. Не должен быть пустой строкой при получении лицензии на основании данных учетной записи.   
При получении лицензии для разработчиков должно быть указано имя учетной записи разработчика (на портале developer.1c.ru или 1c-dn.com), а остальные свойства объекта должны иметь значения по умолчанию.

Доступность:

Тонкий клиент, сервер, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.23.

Описание изменено в версии 8.3.26.
