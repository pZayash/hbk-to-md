# РегистрСведенийНаборЗаписей.<Имя регистра сведений>.ПередЗаписью

РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

ПередЗаписью (BeforeWrite)

Доступен, начиная с версии 8.0.

Синтаксис:

ПередЗаписью(<Отказ>, <Замещение>)

Параметры:

<Отказ>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак отказа от записи набора регистра сведений. Если в теле процедуры-обработчика установить данному параметру значение [Истина](lang__def_BooleanTrue.md), запись выполнена не будет и будет вызвано исключение.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

<Замещение>

Тип: [Булево](lang__Булево_(Boolean).md).   
Режим записи набора.  
[Истина](lang__def_BooleanTrue.md) - запись осуществляется с заменой существующих в базе данных записей набора.  
[Ложь](lang__def_BooleanFalse.md) - запись осуществляется с "дописыванием" текущего набора записей.

Описание:

Возникает перед выполнением записи набора регистра сведений. Процедура-обработчик вызывается после начала транзакции записи, но до начала записи набора регистра сведений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Замечание:

Безопасный режим на время исполнения данного обработчика будет отключен. Данное замечание не относится к обработчикам событий, расположенным в расширении конфигурации.

Использование в версии:

Доступен, начиная с версии 8.0.
