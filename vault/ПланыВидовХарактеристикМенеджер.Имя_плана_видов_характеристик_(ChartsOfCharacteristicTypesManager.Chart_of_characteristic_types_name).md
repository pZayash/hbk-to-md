Доступен, начиная с версии 8.0.

Использование:

Только чтение.

Описание:

Тип: [ПланВидовХарактеристикМенеджер.\<Имя плана видов характеристик\>](ПланВидовХарактеристикМенеджер.Имя_плана_видов_характеристик_(ChartOfCharacteristicTypesManager.Chart_of_characteristic_types_name).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
