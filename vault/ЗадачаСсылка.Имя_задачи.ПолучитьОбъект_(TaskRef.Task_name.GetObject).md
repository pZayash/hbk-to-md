# ЗадачаСсылка.<Имя задачи>.ПолучитьОбъект

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Задачи](Задачи.md) › [ЗадачаСсылка.<Имя задачи>](ЗадачаСсылка.Имя_задачи_(TaskRef.Task_name).md) › [Методы](ЗадачаСсылка.Имя_задачи__Методы.md)

ЗадачаСсылка.<Имя задачи> (TaskRef.<Task name>)

ПолучитьОбъект (GetObject)

Доступен, начиная с версии 8.0.

Синтаксис:

ПолучитьОбъект()

Возвращаемое значение:

Тип: [ЗадачаОбъект.](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)<[Имя задачи](ЗадачаОбъект.Имя_задачи_(TaskObject.Task_name).md)>, [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - объекта в базе нет (например, при обмене данными или после непосредственного удаления объекта, на который есть ссылки).

Описание:

Получает по ссылке объект для модификации и записи задачи.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Пример:

|  |
| --- |
| Запрос = Новый Запрос; Запрос.Текст =  "ВЫБРАТЬ |    Ссылка  |ИЗ |    Задача.Задача | |ГДЕ |    Выполнена <> Истина"; Выборка = Запрос.Выполнить().Выбрать(); Пока Выборка.Следующий() Цикл     ТекущаяЗадача = Выборка.Ссылка.ПолучитьОбъект();     // Выполнение требуемых действий с задачей  КонецЦикла; |

Использование в версии:

Доступен, начиная с версии 8.0.
