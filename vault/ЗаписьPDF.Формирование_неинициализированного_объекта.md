ЗаписьPDF (PDFWriter)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЗаписьPDF()

Описание:

Формирует неинициализированный объект [ЗаписьPDF](ЗаписьPDF_(PDFWriter).md). Для инициализации объекта используется метод [Открыть](ЗаписьPDF.Открыть_(PDFWriter.Open).md) или [НачатьОткрытие](ЗаписьPDF.НачатьОткрытие_(PDFWriter.BeginOpen).md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
