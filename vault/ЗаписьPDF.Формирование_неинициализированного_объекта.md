ЗаписьPDF (PDFWriter)

Формирование неинициализированного объекта

Доступен, начиная с версии 8.3.16.

Синтаксис:

Новый ЗаписьPDF()

Описание:

Формирует неинициализированный объект [ЗаписьPDF](ЗаписьPDF.md). Для инициализации объекта используется метод [Открыть](ЗаписьPDF.Открыть.md) или [НачатьОткрытие](ЗаписьPDF.НачатьОткрытие.md).

Использование в версии:

Доступен, начиная с версии 8.3.16.
