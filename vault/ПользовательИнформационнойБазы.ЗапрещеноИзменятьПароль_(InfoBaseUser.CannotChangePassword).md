ПользовательИнформационнойБазы (InfoBaseUser)

ЗапрещеноИзменятьПароль (CannotChangePassword)

Доступен, начиная с версии 8.1.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Определяет возможность пользователя изменять свой пароль.  
[Истина](lang__def_BooleanTrue.md) - замена запрещена.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл, только если установлено свойство [АутентификацияСтандартная](ПользовательИнформационнойБазы.АутентификацияСтандартная_(InfoBaseUser.StandardAuthentication).md).  
Доступно только пользователю с административными правами.

Использование в версии:

Доступен, начиная с версии 8.1.
