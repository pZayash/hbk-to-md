Профиль безопасности (ISecurityProfile)

RightExtensionDefinitionRoles (RightExtensionDefinitionRoles)

Доступен, начиная с версии 8.3.9.

Использование:

Только чтение.

Описание:

Тип: Строка.   
Содержит список имен ролей, влияющих на изменение прав доступа из расширения.  
При изменении списка ролей изменения в составе ролей учитываются только после перезапуска текущих сеансов и для новых сеансов.

Доступность:

Интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.9.
