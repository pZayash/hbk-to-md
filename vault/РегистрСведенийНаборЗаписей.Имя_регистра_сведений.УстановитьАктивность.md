РегистрСведенийНаборЗаписей.<Имя регистра сведений> (InformationRegisterRecordSet.<Information register name>)

УстановитьАктивность (SetActive)

Доступен, начиная с версии 8.0.

Синтаксис:

УстановитьАктивность(<Активность>)

Параметры:

<Активность> (обязательный)

Тип: Булево.   
Признак активности записей. Истина - записи активны. Ложь - записи не активны.

Описание:

Устанавливает значение свойства [Активность](РегистрСведенийЗапись.Имя_регистра_сведений.Активность.md) для всех записей набора.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| НаборКурсов.УстановитьАктивность(Истина); |

Использование в версии:

Доступен, начиная с версии 8.0.
