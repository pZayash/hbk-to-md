МенеджерКалендарей (CalendarsManager)

ПоддерживаютсяГлобальныеКлючиКалендарей (GlobalCalendarsKeysSupported)

Доступен, начиная с версии 8.3.9.

Синтаксис:

ПоддерживаютсяГлобальныеКлючиКалендарей()

Возвращаемое значение:

Тип: Булево.   
Всегда возвращает Ложь.

Описание:

Определяет, поддерживаются ли системой глобальные ключи календарей.  
Предназначен для реализации возможности передачи календаря между клиентом и сервером.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.9.
