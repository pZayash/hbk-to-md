ИнтернетПочта (InternetMail)

ПолучитьПочтовыеЯщики (GetMailboxes)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьПочтовыеЯщики()

Возвращаемое значение:

Тип: [Массив](Массив.md).   

Описание:

Получает массив имен всех почтовых ящиков на сервере для данной учётной записи.

Доступность:

Тонкий клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Работает только при подключении, установленном по протоколу IMAP.

Использование в версии:

Доступен, начиная с версии 8.2.
