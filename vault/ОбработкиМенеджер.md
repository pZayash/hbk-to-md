Доступен, начиная с версии 8.1.

Элементы коллекции:

[ОбработкаМенеджер.\<Имя обработки\>](ОбработкаМенеджер.Имя_обработки.md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются менеджеры обработок.  
Возможно обращение к менеджеру обработки посредством оператора [...]. В качестве аргумента передается имя обработки, как оно задано в конфигурации.  

Свойства:

[\<Имя обработки\> (\<Data processor name\>)](ОбработкиМенеджер.Имя_обработки.md)  

Описание:

Используется для управления обработками и предоставляет доступ к значениям типа [ОбработкаМенеджер.](ОбработкаМенеджер.Имя_обработки.md)<[Имя обработки](ОбработкаМенеджер.Имя_обработки.md)>. Доступ к объекту осуществляется через свойство глобального контекста [Обработки](Глобальный_контекст.Обработки.md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.   
Возможен обмен с сервером.

См. также:

[Глобальный контекст](Глобальный_контекст.md), свойство [Обработки](Глобальный_контекст.Обработки.md)  

Использование в версии:

Доступен, начиная с версии 8.1.
