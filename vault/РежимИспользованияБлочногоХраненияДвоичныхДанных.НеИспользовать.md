# РежимИспользованияБлочногоХраненияДвоичныхДанных.НеИспользовать

РежимИспользованияБлочногоХраненияДвоичныхДанных (BinaryDataBlockStorageUseMode)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.3.26.

Описание:

Не использовать блочное хранение двоичных данных. Режимы размещения копий двоичных данных будут недоступны. При выключении режима выполняется реструктуризация данных: данные из таблицы BinaryData перемещаются в базу данных и хранилища двоичных данных в соответствии с правилами размещении. После перемещения данных все хранилища работают в режиме [РазмещениеДанныхТолькоВХранилище](РежимРазмещенияКопийДанныхВХранилищеДвоичныхДанных.РазмещениеДанныхТолькоВХранилище.md).

Использование в версии:

Доступен, начиная с версии 8.3.26.
