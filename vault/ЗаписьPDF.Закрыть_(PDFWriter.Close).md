ЗаписьPDF (PDFWriter)

Закрыть (Close)

Доступен, начиная с версии 8.3.16.

Синтаксис:

Закрыть()

Описание:

Закрывает открытый файл.   
Закрытие файла выполняется автоматически при удалении объекта [ЗаписьPDF](ЗаписьPDF_(PDFWriter).md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

См. также:

[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [ЗакрытьАсинх](ЗаписьPDF.ЗакрытьАсинх_(PDFWriter.CloseAsync).md)  
[ЗаписьPDF](ЗаписьPDF_(PDFWriter).md), метод [НачатьЗакрытие](ЗаписьPDF.НачатьЗакрытие_(PDFWriter.BeginClose).md)  

Использование в версии:

Доступен, начиная с версии 8.3.16.

Описание изменено в версии 8.3.18.
