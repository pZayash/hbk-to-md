ОтображениеЛинийСеткиДиаграммы (ChartGridLinesShowMode)

Авто (Auto)

Доступен, начиная с версии 8.3.11.

Описание:

Способ отображения линий определяется автоматически:

- Для шкалы значений - [Отображать](ОтображениеЛинийСеткиДиаграммы.Отображать_(ChartGridLinesShowMode.Show).md).
- Для шкалы точек в точечной и пузырьковой диаграммах - [Отображать](ОтображениеЛинийСеткиДиаграммы.Отображать_(ChartGridLinesShowMode.Show).md).
- Во всех остальных случаях - [НеОтображать](ОтображениеЛинийСеткиДиаграммы.НеОтображать_(ChartGridLinesShowMode.DontShow).md).

Использование в версии:

Доступен, начиная с версии 8.3.11.
