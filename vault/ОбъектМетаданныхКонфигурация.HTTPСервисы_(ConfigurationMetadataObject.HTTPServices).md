ОбъектМетаданныхКонфигурация (ConfigurationMetadataObject)

HTTPСервисы (HTTPServices)

Доступен, начиная с версии 8.3.5.

Использование:

Только чтение.

Описание:

Тип: [КоллекцияОбъектовМетаданных](КоллекцияОбъектовМетаданных_(MetadataObjectCollection).md).   
Коллекция объектов метаданных, которые описывают все HTTP-Сервисы, определенные в конфигурации.  
Элементами коллекции являются объекты типа [ОбъектМетаданных: HTTPСервис](ОбъектМетаданных_HTTPСервис_(MetadataObject_HTTPService).md).

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.5.
