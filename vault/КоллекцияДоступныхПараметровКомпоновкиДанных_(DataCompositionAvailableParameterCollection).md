# КоллекцияДоступныхПараметровКомпоновкиДанных

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Система компоновки данных](Система_компоновки_данных-2.md)

Доступен, начиная с версии 8.1.

Элементы коллекции:

[ДоступныйПараметрКомпоновкиДанных](ДоступныйПараметрКомпоновкиДанных_(DataCompositionAvailableParameter).md)  
Для объекта доступен обход коллекции посредством оператора Для каждого … Из … Цикл. При обходе выбираются доступные параметры.  
Возможно обращение к доступному параметру посредством оператора [...]. В качестве аргумента передается индекс (нумерация с 0).  

Методы:

[Индекс (IndexOf)](КоллекцияДоступныхПараметровКомпоновкиДанных.Индекс_(DataCompositionAvailableParameterCollection.IndexOf).md)  
[Количество (Count)](КоллекцияДоступныхПараметровКомпоновкиДанных.Количество_(DataCompositionAvailableParameterCollection.Count).md)  
[Найти (Find)](КоллекцияДоступныхПараметровКомпоновкиДанных.Найти_(DataCompositionAvailableParameterCollection.Find).md)  
[Получить (Get)](КоллекцияДоступныхПараметровКомпоновкиДанных.Получить_(DataCompositionAvailableParameterCollection.Get).md)  

Описание:

Коллекция доступных параметров.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.
