ИсточникДоступныхНастроекКомпоновкиДанных (DataCompositionAvailableSettingsSource)

На основании схемы

Доступен, начиная с версии 8.1.

Синтаксис:

Новый ИсточникДоступныхНастроекКомпоновкиДанных(<СхемаКомпоновкиДанных>, <ПараметрыФункциональныхОпций>)

Параметры:

<СхемаКомпоновкиДанных> (обязательный)

Тип: [СхемаКомпоновкиДанных](СхемаКомпоновкиДанных.md).   
Из схемы источник извлекает необходимую компоновщику настроек информацию.

<ПараметрыФункциональныхОпций> (необязательный)

Тип: [Структура](Структура.md).   
Содержит параметры функциональных опций которые будут использоваться при построении доступных полей.  
Значение по умолчанию: Неопределено.

Описание:

Инициализирует источник доступных настроек на основе схемы компоновки данных.

Использование в версии:

Доступен, начиная с версии 8.1.
