# ОбъектМетаданных: РеквизитАдресации.РежимПароля

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа с объектами метаданных](Работа_с_объектами_метаданных.md) › [Объекты метаданных](Объекты_метаданных.md) › [ОбъектМетаданных: РеквизитАдресации](ОбъектМетаданных_РеквизитАдресации_(MetadataObject_AddressingAttribute).md) › [Свойства](ОбъектМетаданных_РеквизитАдресации__Свойства.md)

ОбъектМетаданных: РеквизитАдресации (MetadataObject: AddressingAttribute)

РежимПароля (PasswordMode)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Устанавливает отображение данных реквизита адресации в режиме пароля (символы заменяются звездочками при отображении).   
[Истина](lang__def_BooleanTrue.md) - режим ввода пароля.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Примечание:

Имеет смысл для реквизитов типа [Строка](lang__def_String.md).

Использование в версии:

Доступен, начиная с версии 8.2.
