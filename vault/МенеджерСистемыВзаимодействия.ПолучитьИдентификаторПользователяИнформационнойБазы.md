МенеджерСистемыВзаимодействия (CollaborationSystemManager)

ПолучитьИдентификаторПользователяИнформационнойБазы (GetInfoBaseUserID)

Доступен, начиная с версии 8.3.10.

Синтаксис:

ПолучитьИдентификаторПользователяИнформационнойБазы(<Идентификатор>)

Параметры:

<Идентификатор> (обязательный)

Тип: [ИдентификаторПользователяСистемыВзаимодействия](ИдентификаторПользователяСистемыВзаимодействия.md).   
Идентификатор пользователя системы взаимодействия.

Возвращаемое значение:

Тип: [УникальныйИдентификатор](УникальныйИдентификатор.md), Неопределено.   

Описание:

Возвращает идентификатор того пользователя информационной базы, который соответствует пользователю системы взаимодействия. Если соответствующего пользователя в информационной базе нет, возвращается Неопределено.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.10.
