МенеджерРасширенийКонфигурации (ConfigurationExtensionsManager)

ВыполнитьФоновоеЗаданиеСРасширениямиБазыДанных (ExecuteBackgroundJobWithDatabaseExtensions)

Доступен, начиная с версии 8.3.8.

Синтаксис:

ВыполнитьФоновоеЗаданиеСРасширениямиБазыДанных(<ИмяМетода>, <Параметры>, <Ключ>, <Наименование>)

Параметры:

<ИмяМетода> (обязательный)

Тип: [Строка](lang__def_String.md).   
Имя экспортируемой процедуры или функции неглобального общего модуля, который может быть выполнен на сервере. Указывается в формате :"ИмяМодуля.ИмяМетода".

<Параметры> (необязательный)

Тип: [Массив](Массив_(Array).md).   
Массив параметров, передаваемых в метод. Количество и типы параметров должны соответствовать параметрам метода.  
Все передаваемые параметры должны поддерживать сериализацию. В противном случае будет выдано исключение и фоновое задание не будет запущено.  
Если последние параметры метода имеют значения по-умолчанию, то их можно не задавать в массиве.  
Если все параметры метода имеют значение по-умолчанию, то массив можно не передавать вообще.

<Ключ> (необязательный)

Тип: [Строка](lang__def_String.md).   
Ключ задания. Если ключ задан, то он должен быть уникальным среди ключей активных фоновых заданий, имеющих такое же имя метода, что и у данного фонового задания.

<Наименование> (необязательный)

Тип: [Строка](lang__def_String.md).   
Описание задания.

Возвращаемое значение:

Тип: [ФоновоеЗадание](ФоновоеЗадание_(BackgroundJob).md).   

Описание:

Инициирует выполнение фонового задания, набор расширений для которого будет получен из базы данных.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.8.
