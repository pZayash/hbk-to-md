Расширение поля формы для поля планировщика (Form field extension for a planner)

ПриОкончанииРедактирования (OnEditEnd)

Доступен, начиная с версии 8.3.6.

Синтаксис:

ПриОкончанииРедактирования(<НовыйЭлемент>, <ОтменаРедактирования>)

Параметры:

<НовыйЭлемент>

Тип: [Булево](lang__Булево_(Boolean).md).   
Признак указывает, что был создан новый элемент, а не редактировался существующий.

<ОтменаРедактирования>

Тип: [Булево](lang__Булево_(Boolean).md).   
Установка данного параметра в [Истина](lang__def_BooleanTrue.md) приведет к отмене изменения элемента планировщика.  
Значение по умолчанию: [Ложь](lang__def_BooleanFalse.md).

Описание:

Вызывается при окончании интерактивного редактировании элемента планировщика. Если был создан новый элемент, [НовыйЭлемент](html.md) будет равен [Истина](lang__def_BooleanTrue.md). Элементы, над которыми выполнялось действие, можно получить с помощью свойства [ВыделенныеЭлементы](Расширение_поля_формы_для_поля_планировщика.ВыделенныеЭлементы_(Form_field_extension_for_a_planner.SelectedItems).md) поля.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.6.
