# ПользовательИнформационнойБазы.Записать

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Общие объекты](Общие_объекты.md) › [Работа со списком пользователей информационной базы](Работа_со_списком_пользователей_информационной_базы.md) › [ПользовательИнформационнойБазы](ПользовательИнформационнойБазы_(InfoBaseUser).md) › [Методы](ПользовательИнформационнойБазы__Методы.md)

ПользовательИнформационнойБазы (InfoBaseUser)

Записать (Write)

Доступен, начиная с версии 8.0.

Синтаксис:

Записать()

Описание:

Записывает сведения о пользователе.  
После выполнения метода Записать и фиксации транзакции, если Записать выполнен при активной транзакции, получение пользователя методами [НайтиПоИмени](МенеджерПользователейИнформационнойБазы.НайтиПоИмени_(InfoBaseUsersManager.FindByName).md), [НайтиПоУникальномуИдентификатору](МенеджерПользователейИнформационнойБазы.НайтиПоУникальномуИдентификатору_(InfoBaseUsersManager.FindByUUID).md), и [ПолучитьПользователей](МенеджерПользователейИнформационнойБазы.ПолучитьПользователей_(InfoBaseUsersManager.GetUsers).md) объекта [МенеджерПользователейИнформационнойБазы](МенеджерПользователейИнформационнойБазы_(InfoBaseUsersManager).md) во всех сеансах этой информационной базы будет возвращать записанное состояние этого пользователя. Но метод [ТекущийПользователь](МенеджерПользователейИнформационнойБазы.ТекущийПользователь_(InfoBaseUsersManager.CurrentUser).md) во всех сеансах, включая текущий, будет возвращать состояние пользователя на момент старта соответствующего сеанса.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.0.

Описание изменено в версии 8.3.26.
