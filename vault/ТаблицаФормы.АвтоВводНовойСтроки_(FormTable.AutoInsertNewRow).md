АвтоВводНовойСтроки (AutoInsertNewRow)

Доступен, начиная с версии 8.2.

Использование:

Чтение и запись.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Содержит признак автоматического ввода новой строки после заполнения текущей.  
[Истина](lang__def_BooleanTrue.md) - новая строка вводится автоматически.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.2.
