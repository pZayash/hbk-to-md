ЭлементПрикрепляемыхДанныхЗапускаПриложенияМобильногоУстройства (MobileDeviceApplicationRunClipDataItem)

ПолучитьДанныеЗапуска (GetRunData)

Доступен, начиная с версии 8.3.23.

Синтаксис:

ПолучитьДанныеЗапуска()

Возвращаемое значение:

Тип: [ЗапускПриложенияМобильногоУстройства](ЗапускПриложенияМобильногоУстройства.md), Неопределено.   
Данные запуска, если они есть в прикрепляемых данных, иначе Неопределено.

Описание:

Возвращает данные запуска из элемента прикрепляемых данных.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

Использование в версии:

Доступен, начиная с версии 8.3.23.
