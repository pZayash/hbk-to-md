Прикладной объект, преобразованный из универсального объекта в значение, для которого необходимо получить соответствие с именем реквизита формы.

Возвращаемое значение:

Тип: [Структура](Структура_(Structure).md).   
Содержит элементы с именами ключей "ИдентификаторФормы" ("FormID"), "ИмяРеквизита" ("AttributeName").

Описание:

Получает информацию об идентификаторе формы и имени атрибута формы, связанного с передаваемым в качестве аргумента функции объектом.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
