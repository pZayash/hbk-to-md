Глобальный контекст (Global context)

ЗапрещеноОткрытиеФорм (CannotOpenForm)

Доступен, начиная с версии 8.3.14.

Синтаксис:

ЗапрещеноОткрытиеФорм()

Возвращаемое значение:

Тип: [Булево](lang__Булево_(Boolean).md).   
На толстом и тонком клиентах возвращает [Истина](lang__def_BooleanTrue.md) при вызове в обработчике [ПередНачаломРаботыСистемы](Глобальный_контекст.ПередНачаломРаботыСистемы_(Global_context.BeforeStart).md), если установлен ключ DisableStartupDialogs.   
После окончания работы обработчика - [Ложь](lang__def_BooleanFalse.md).  
На веб-клиенте всегда возвращает [Ложь](lang__def_BooleanFalse.md).

Описание:

Возвращает текущий режим открытия форм.

Доступность:

Тонкий клиент, веб-клиент, толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.3.14.
