КоллекцияКолонокДереваЗначений (ValueTreeColumnCollection)

Количество (Count)

Доступен, начиная с версии 8.0.

Синтаксис:

Количество()

Возвращаемое значение:

Тип: [Число](lang__def_Number.md).   

Описание:

Получает количество колонок дерева значений.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Пример:

|  |
| --- |
| КолонкиСостава = ДеревоСостава.Колонки; КоличествоКолонок = КолонкиСостава.Количество(); |

Использование в версии:

Доступен, начиная с версии 8.0.
