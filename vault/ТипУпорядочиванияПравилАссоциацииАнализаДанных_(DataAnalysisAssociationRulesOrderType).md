Описание:

Содержит варианты упорядочивания найденных правил ассоциации.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.   
Возможен обмен с сервером. Сериализуется. Может использоваться в реквизитах управляемой формы. Данный объект может быть сериализован в/из XDTO. Тип XDTO, соответствующий данному объекту, определяется в пространстве имен {http://v8.1c.ru/8.2/data/data-analysis}. Имя типа XDTO: DataAnalysisAssociationRulesOrderType.

Использование в версии:

Доступен, начиная с версии 8.0.
