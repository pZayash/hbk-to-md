Глобальный контекст (Global context)

УстановитьРасширениеРаботыСФайламиАсинх (InstallFileSystemExtensionAsync)

Доступен, начиная с версии 8.3.18.

Синтаксис:

УстановитьРасширениеРаботыСФайламиАсинх()

Возвращаемое значение:

Тип: [Обещание](Обещание_(Promise).md).   
Результат выполнения обещания - [Неопределено](lang__def_Undefined.md).

Описание:

Асинхронная функция. Устанавливает расширение работы с файлами.  
В режиме запуска толстого и тонкого клиентов информирует о том, что расширение установлено.  
Для клиента мобильного приложения, информирует о том, что расширение установлено.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).
