КоллекцияПолейСводнойТаблицы (PivotTableFieldCollection)

Найти (Find)

Доступен, начиная с версии 8.0.

Синтаксис:

Найти(<ИмяПоля>)

Параметры:

<ИмяПоля> (обязательный)

Тип: Строка.   
Имя искомого поля.

Возвращаемое значение:

Тип: [ПолеСводнойТаблицы](ПолеСводнойТаблицы.md), [КоллекцияПолейСводнойТаблицы](КоллекцияПолейСводнойТаблицы.md), Неопределено.   
Если поле отсутствует, возвращает значение Неопределено.

Описание:

Осуществляет поиск поля сводной таблицы по имени.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.0.
