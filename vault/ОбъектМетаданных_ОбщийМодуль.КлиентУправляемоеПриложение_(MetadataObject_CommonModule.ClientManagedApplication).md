ОбъектМетаданных: ОбщийМодуль (MetadataObject: CommonModule)

КлиентУправляемоеПриложение (ClientManagedApplication)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то процедуры и функции общего модуля могут быть использованы в толстом клиенте в режиме приложения, в тонком клиенте и в веб-клиенте.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
