РежимСглаживанияДиаграммы (ChartSplineMode)

Нет (None)

Доступен, начиная с версии 8.3.1.

Описание:

Режим сплайнов - "сглаживания" отключен. Соединения между точками визуализируются на основе прямых линий.

Использование в версии:

Доступен, начиная с версии 8.3.1.
