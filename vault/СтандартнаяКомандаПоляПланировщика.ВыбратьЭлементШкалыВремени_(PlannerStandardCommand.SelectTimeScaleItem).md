СтандартнаяКомандаПоляПланировщика (PlannerStandardCommand)

ВыбратьЭлементШкалыВремени (SelectTimeScaleItem)

Доступен, начиная с версии 8.3.23.

Описание:

Переход к событию [НажатиеНаЭлементеШкалыВремени](Расширение_поля_формы_для_поля_планировщика.НажатиеНаЭлементеШкалыВремени_(Form_field_extension_for_a_planner.TimeScaleItemClick).md).  
Используется только для источника типа [ЭлементШкалыВремени](ИсточникКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandSource.TimeScaleItem).md).  
В контекстном меню команда будет иметь представление "Выбрать".

Использование в версии:

Доступен, начиная с версии 8.3.23.
