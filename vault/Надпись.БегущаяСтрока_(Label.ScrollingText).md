Надпись (Label)

БегущаяСтрока (ScrollingText)

Доступен, начиная с версии 8.0.

Использование:

Чтение и запись.

Описание:

Тип: [РежимБегущейСтроки](РежимБегущейСтроки_(ScrollingTextMode).md).   
Задает режим бегущей строки. В режиме бегущей строки содержимое надписи будет двигаться справа налево.

Доступность:

Толстый клиент.

Использование в версии:

Доступен, начиная с версии 8.0.
