Описание:

Описывает поддерживаемые глубины цвета картинки.  
При работе в режиме совместимости [Версия8\_3\_13](РежимСовместимости.Версия8_3_13_(CompatibilityMode.Version8_3_13).md) поддерживается только английский вариант терма.

Доступность:

Тонкий клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.3.14.
