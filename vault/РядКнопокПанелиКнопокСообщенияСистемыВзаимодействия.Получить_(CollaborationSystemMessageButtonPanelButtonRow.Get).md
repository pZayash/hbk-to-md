Получить (Get)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс кнопки панели кнопок сообщения системы взаимодействия.

Возвращаемое значение:

Тип: [КнопкаПанелиКнопокСообщенияСистемыВзаимодействия](КнопкаПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButton).md).   

Описание:

Получает кнопку панели кнопок сообщения по индексу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
