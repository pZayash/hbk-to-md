МенеджерСтатистикиИспользованияПриложения (ApplicationUsageStatisticsManager)

ПолучитьАдресОбновления (GetUpdateAddress)

Доступен, начиная с версии 8.3.17.

Синтаксис:

ПолучитьАдресОбновления()

Возвращаемое значение:

Тип: Строка.   

Описание:

Возвращает адрес автоматического обновления настроек регистрации событий/экранов статистики использования приложения, установленный методом [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления.md).   
Если в последний раз метод [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления.md) был вызван со параметром Неопределено – адрес не установлен.

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерСтатистикиИспользованияПриложения](МенеджерСтатистикиИспользованияПриложения.md), метод [УстановитьАдресОбновления](МенеджерСтатистикиИспользованияПриложения.УстановитьАдресОбновления.md)  

Использование в версии:

Доступен, начиная с версии 8.3.17.
