РежимСверткиЭлементаУправления (ControlCollapseMode)

Верх (Top)

Доступен, начиная с версии 8.0.

Описание:

Элемент управления свертывается к своей верхней стороне (нижняя сторона прижимается к верхней).

Использование в версии:

Доступен, начиная с версии 8.0.
