ОписаниеТипов (TypeDescription)

На основании типов и квалификаторов

Доступен, начиная с версии 8.0.

Синтаксис:

Новый ОписаниеТипов(<Типы>, <КвалификаторыЧисла>, <КвалификаторыСтроки>, <КвалификаторыДаты>, <КвалификаторыДвоичныхДанных>)

Параметры:

<Типы> (обязательный)

Тип: [Массив](Массив_(Array).md), [Строка](lang__def_String.md).   
Массив значений типа [Тип](lang__def_Type.md), состоящий из типов, которые будут использованы в объекте, или строка, содержащая имена типов, разделенных запятыми.

<КвалификаторыЧисла> (необязательный)

Тип: [КвалификаторыЧисла](КвалификаторыЧисла_(NumberQualifiers).md).   
Квалификаторы числа, описывающие допустимые значения числового типа. Если параметр не указан, то числа хранить нельзя.

<КвалификаторыСтроки> (необязательный)

Тип: [КвалификаторыСтроки](КвалификаторыСтроки_(StringQualifiers).md).   
Квалификаторы строки, описывающие допустимые значения строкового типа. Если параметр не указан, то используется неограниченная длина строки.

<КвалификаторыДаты> (необязательный)

Тип: [КвалификаторыДаты](КвалификаторыДаты_(DateQualifiers).md).   
Квалификаторы даты, описывающие допустимые значения типа [Дата](lang__def_Date.md). Если параметр не указан, то используются обе части даты (дата + время).

<КвалификаторыДвоичныхДанных> (необязательный)

Тип: [КвалификаторыДвоичныхДанных](КвалификаторыДвоичныхДанных_(BinaryDataQualifiers).md).   
Квалификаторы двоичных данных.

Описание:

Создает описание типов на основании заданных типов и квалификаторов для типов [Число](lang__def_Number.md), [Строка](lang__def_String.md), [Дата](lang__def_Date.md).

Пример:

|  |
| --- |
| // составной тип из строки и ссылки на справочник Номенклатуры.  МассивТипов = Новый Массив; МассивТипов.Добавить(Тип("СправочникСсылка.Номенклатура")); МассивТипов.Добавить(Тип("Строка")); ПараметрыСтроки = Новый КвалификаторыСтроки(20);  ДопустимыеТипы = Новый ОписаниеТипов(МассивТипов, , ПараметрыСтроки); |

Использование в версии:

Доступен, начиная с версии 8.0.
