ОбработчикиWebSocketКлиентСоединения (WebSocketClientConnectionHandlers)

ОбработчикПолученияСообщения (OnMessageHandler)

Доступен, начиная с версии 8.3.27.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Имя экспортируемой процедуры- обработчика события получения сообщения со следующими параметрами:

- <Соединение> - объект типа [WebSocketКлиентСоединение](WebSocketКлиентСоединение.md).
- <Сообщение> - объект типа строка или [ДвоичныеДанные](ДвоичныеДанные.md). Содержит данные сообщения.

  
Пустая строка, если это соединение [ОбъектМетаданных: WebSocketКлиент](ОбъектМетаданных_WebSocketКлиент.md).

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
