ИсточникКомандПоляПланировщика (PlannerCommandSource)

ЭлементИзмерения (DimensionItem)

Доступен, начиная с версии 8.3.23.

Описание:

Тип: [ЭлементИзмеренияПланировщика](ЭлементИзмеренияПланировщика_(PlannerDimensionItem).md), [Неопределено](lang__def_Undefined.md).   
Значение, которое устанавливается при нажатии на измерение поля планировщика.  
Во время возникновения события [ОбработкаФормированияКоманд](Расширение_поля_формы_для_поля_планировщика.ОбработкаФормированияКоманд_(Form_field_extension_for_a_planner.CommandGenerateProcessing).md):

- [Элементы](ПараметрыФормированияКомандПланировщика.Элементы_(PlannerCommandGenerateParameters.Items).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЭлементИзмерения](ПараметрыФормированияКомандПланировщика.ЭлементИзмерения_(PlannerCommandGenerateParameters.DimensionItem).md) заполнено,
- [ЗначенияИзмерений](ПараметрыФормированияКомандПланировщика.ЗначенияИзмерений_(PlannerCommandGenerateParameters.DimensionValues).md) заполнено,
- [Дата](ПараметрыФормированияКомандПланировщика.Дата_(PlannerCommandGenerateParameters.Date).md) содержит [Неопределено](lang__def_Undefined.md),
- [ЭлементШкалыВремени](ПараметрыФормированияКомандПланировщика.ЭлементШкалыВремени_(PlannerCommandGenerateParameters.TimeScaleItem).md) содержит [Неопределено](lang__def_Undefined.md),
- [Команды](html.md) не содержит команд,
- [КомандаПоУмолчанию](html.md) заполнено командой [ВыбратьЭлементИзмерения](СтандартнаяКомандаПланировщика.ВыбратьЭлементИзмерения_(PlannerStandardCommand.SelectDimensionItem).md).

Использование в версии:

Доступен, начиная с версии 8.3.23.
