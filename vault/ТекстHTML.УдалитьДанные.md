ТекстHTML (HTMLText)

УдалитьДанные (DeleteData)

Доступен, начиная с версии 8.2.

Синтаксис:

УдалитьДанные(<Смещение>, <Длина>)

Параметры:

<Смещение> (обязательный)

Тип: Число.   
Смещение от начала символьных данных.

<Длина> (обязательный)

Тип: Число.   
Длина удаляемых символьных данных.

Описание:

Удаляет часть символьных данных.

Доступность:

Мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент), мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Причины вызова исключения:

- смещение превышает размер символьных данных;
- данный узел в режиме Только для чтения.

Использование в версии:

Доступен, начиная с версии 8.2.
