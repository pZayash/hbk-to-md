ВстроеннаяПокупка (InAppPurchase)

Цена (Price)

Доступен, начиная с версии 8.3.8.

Использование:

Только чтение.

Описание:

Тип: [Число](lang__def_Number.md).   
Содержит численное значение стоимости покупки. При использовании необходимо учитывать значение свойства [Валюта](ВстроеннаяПокупка.Валюта_(InAppPurchase.Currency).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).  
Изменение свойства на клиенте может потребовать обращения к серверу.

Использование в версии:

Доступен, начиная с версии 8.3.8.
