КоллекцияЭлементовФорматированногоДокумента (FormattedDocumentItemCollection)

Получить (Get)

Доступен, начиная с версии 8.3.1.

Синтаксис:

Получить(<Индекс>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md).   
Индекс элемента.

Возвращаемое значение:

Тип: [ПараграфФорматированногоДокумента](ПараграфФорматированногоДокумента_(FormattedDocumentParagraph).md), [ТекстФорматированногоДокумента](ТекстФорматированногоДокумента_(FormattedDocumentText).md), [ПереводСтрокиФорматированногоДокумента](ПереводСтрокиФорматированногоДокумента_(FormattedDocumentLinefeed).md), [КартинкаФорматированногоДокумента](КартинкаФорматированногоДокумента_(FormattedDocumentPicture).md).   

Описание:

Получает элемент форматированного документа по индексу. Работает аналогично оператору [].

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.1.
