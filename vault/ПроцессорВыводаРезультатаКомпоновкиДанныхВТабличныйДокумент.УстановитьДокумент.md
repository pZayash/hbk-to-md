ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент (DataCompositionResultSpreadsheetDocumentOutputProcessor)

УстановитьДокумент (SetDocument)

Доступен, начиная с версии 8.1.

Синтаксис:

УстановитьДокумент(<Документ>)

Параметры:

<Документ> (необязательный)

Тип: [ТабличныйДокумент](ТабличныйДокумент.md).   
Документ, в который будет осуществляться вывод.

Описание:

Устанавливает табличный документ, в который нужно выводить результат.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
