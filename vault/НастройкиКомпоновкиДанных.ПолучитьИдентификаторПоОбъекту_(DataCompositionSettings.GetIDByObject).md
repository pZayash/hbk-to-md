НастройкиКомпоновкиДанных (DataCompositionSettings)

ПолучитьИдентификаторПоОбъекту (GetIDByObject)

Доступен, начиная с версии 8.2.

Синтаксис:

ПолучитьИдентификаторПоОбъекту(<Объект>)

Параметры:

<Объект> (обязательный)

Тип: [НастройкиКомпоновкиДанных](НастройкиКомпоновкиДанных_(DataCompositionSettings).md), [ГруппировкаКомпоновкиДанных](ГруппировкаКомпоновкиДанных_(DataCompositionGroup).md), [ТаблицаКомпоновкиДанных](ТаблицаКомпоновкиДанных_(DataCompositionTable).md), [ДиаграммаКомпоновкиДанных](ДиаграммаКомпоновкиДанных_(DataCompositionChart).md), [НастройкиВложенногоОбъектаКомпоновкиДанных](НастройкиВложенногоОбъектаКомпоновкиДанных_(DataCompositionNestedObjectSettings).md), [ГруппировкаДиаграммыКомпоновкиДанных](ГруппировкаДиаграммыКомпоновкиДанных_(DataCompositionChartGroup).md), [ГруппировкаТаблицыКомпоновкиДанных](ГруппировкаТаблицыКомпоновкиДанных_(DataCompositionTableGroup).md) , [КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных](КоллекцияЭлементовСтруктурыДиаграммыКомпоновкиДанных_(DataCompositionChartStructureItemCollection).md), [КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных](КоллекцияЭлементовСтруктурыТаблицыКомпоновкиДанных_(DataCompositionTableStructureItemCollection).md).   
Объект, для которого требуется получить идентификатор.

Возвращаемое значение:

Тип: [ИдентификаторКомпоновкиДанных](ИдентификаторКомпоновкиДанных_(DataCompositionID).md), [Неопределено](lang__def_Undefined.md).   
[Неопределено](lang__def_Undefined.md) - переданный объект не является частью данного объекта.

Описание:

Получает идентификатор для переданного объекта.

Доступность:

Тонкий клиент, веб-клиент, сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.2.
