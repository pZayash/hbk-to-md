ИнформацияОКопииБазыДанных (DatabaseCopyInfo)

ВыполняетсяОбработкаДанныхДатаАкселератором (DataAcceleratorProcessingData)

Доступен, начиная с версии 8.3.27.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Указывает на то, что Дата акселератор занят процессами обслуживания своей базы данных (значение [Истина](lang__def_BooleanTrue.md)) - очисткой, построением индексов и статистики.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.27.
