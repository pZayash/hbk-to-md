ЧтениеФайлаАрхива (ArchiveFileReader)

На основании потока

Доступен, начиная с версии 8.3.26.

Синтаксис:

Новый ЧтениеФайлаАрхива(<Поток>, <Пароль>, <ТипФайлаАрхива>)

Параметры:

<Поток> (обязательный)

Тип: [Поток](Поток.md), [ПотокВПамяти](ПотокВПамяти.md), [ФайловыйПоток](ФайловыйПоток.md).   
Поток, из которого будут прочитаны данные архива.

<Пароль> (необязательный)

Тип: Строка.   
Пароль для доступа к потоку архива, если поток зашифрован.

<ТипФайлаАрхива> (необязательный)

Тип: [ТипФайлаАрхива](ТипФайлаАрхива.md).   
Тип файла архива, который требуется прочитать.  
Значение по умолчанию: {ссылка:Атрибуты; 13749; ZIP}.

Описание:

Создает объект чтения потока архива и устанавливает поток для его чтения.

Использование в версии:

Доступен, начиная с версии 8.3.26.
