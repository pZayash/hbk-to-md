Описание:

Определяет набор вариантов раскрытия дерева в табличном поле. Используется для определения свойства [НачальноеОтображениеДерева](ТабличноеПоле.НачальноеОтображениеДерева.md).

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, сервер, толстый клиент, внешнее соединение, мобильное приложение (клиент).   
Возможен обмен с сервером. Сериализуется.
