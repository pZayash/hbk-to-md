МенеджерКалендарей (CalendarsManager)

УдалитьСобытие (DeleteEvent)

Доступен, начиная с версии 8.3.4.

Синтаксис:

УдалитьСобытие(<Событие>)

Параметры:

<Событие> (обязательный)

Тип: [ЛокальныйКлючСобытияКалендаря](ЛокальныйКлючСобытияКалендаря_(LocalCalendarEventKey).md).   
Локальный ключ события календаря.

Описание:

Удаляет событие по локальному ключу.  
Данный метод поддерживается не на всех устройствах. Для проверки совместимости устройства перед вызовом данного метода следует использовать метод [ПоддерживаетсяИзменениеСобытий](МенеджерКалендарей.ПоддерживаетсяИзменениеСобытий_(CalendarsManager.EventEditingSupported).md).

Доступность:

Мобильный клиент, мобильное приложение (клиент).

См. также:

[МенеджерКалендарей](МенеджерКалендарей_(CalendarsManager).md), метод [ПоддерживаетсяИзменениеСобытий](МенеджерКалендарей.ПоддерживаетсяИзменениеСобытий_(CalendarsManager.EventEditingSupported).md)  

Использование в версии:

Доступен, начиная с версии 8.3.4.
