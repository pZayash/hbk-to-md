ОписаниеСтандартногоРеквизита (StandardAttributeDescription)

СвязиПараметровВыбора (ChoiceParameterLinks)

Доступен, начиная с версии 8.2.

Использование:

Только чтение.

Описание:

Тип: [ФиксированныйМассив](ФиксированныйМассив_(FixedArray).md).   
Предоставляет программный доступ к связям параметров выбора.  
Фиксированный массив из элементов типа [СвязьПараметраВыбора](СвязьПараметраВыбора_(ChoiceParameterLink).md).

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Использование в версии:

Доступен, начиная с версии 8.2.
