# ПланОбменаОбъект.<Имя плана обмена>.Заполнить

**↑** [Главная](_index.md) › [Объекты](_index__Объекты.md) › [Прикладные объекты](Прикладные_объекты.md) › [Планы обмена](Планы_обмена.md) › [ПланОбменаОбъект.<Имя плана обмена>](ПланОбменаОбъект.Имя_плана_обмена_(ExchangePlanObject.Exchange_plan_name).md) › [Методы](ПланОбменаОбъект.Имя_плана_обмена__Методы.md)

ПланОбменаОбъект.<Имя плана обмена> (ExchangePlanObject.<Exchange plan name>)

Заполнить (Fill)

Доступен, начиная с версии 8.0.

Синтаксис:

Заполнить(<ДанныеЗаполнения>)

Параметры:

<ДанныеЗаполнения> (обязательный)

Тип: Произвольный.   
Значение, на основании которого выполняется заполнение узла.

Описание:

Заполняет данные узла данными из другого объекта. Инициирует событие [ОбработкаЗаполнения](ПланОбменаОбъект.Имя_плана_обмена.ОбработкаЗаполнения_(ExchangePlanObject.Exchange_plan_name.Filling).md) и вызов его процедуры-обработчика в модуле плана обмена. В ней может размещаться алгоритм, заполняющий данные узла из переданного значения.

Доступность:

Сервер, толстый клиент, внешнее соединение, мобильное приложение (сервер), мобильный автономный сервер.

Примечание:

Для заполнения может использоваться значение любого типа.

Использование в версии:

Доступен, начиная с версии 8.0.
