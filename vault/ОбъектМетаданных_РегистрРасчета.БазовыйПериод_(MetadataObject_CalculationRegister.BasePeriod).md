# ОбъектМетаданных: РегистрРасчета.БазовыйПериод

ОбъектМетаданных: РегистрРасчета (MetadataObject: CalculationRegister)

БазовыйПериод (BasePeriod)

Доступен, начиная с версии 8.1.

Использование:

Только чтение.

Описание:

Тип: [Булево](lang__Булево_(Boolean).md).   
Если это свойство установлено в значение [Истина](lang__def_BooleanTrue.md), то для записей регистра расчета возможно использование механизма зависимости по базовому периоду и получение значения базы.   
Если это свойство установлено в значение [Ложь](lang__def_BooleanFalse.md), то механизм зависимости по базовому периоду не используется.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.1.
