Файл (File)

НачатьПроверкуЭтоКаталог (BeginCheckingIsDirectory)

Доступен, начиная с версии 8.3.6.

Синтаксис:

НачатьПроверкуЭтоКаталог(<ОписаниеОповещения>)

Параметры:

<ОписаниеОповещения> (обязательный)

Тип: [ОписаниеОповещения](ОписаниеОповещения.md).   
Содержит описание процедуры, которая будет вызвана после завершения со следующими параметрами:

- <ЭтоКаталог> - результат проверки.Тип: Булево. Истина - объект соответствует каталогу; Ложь - объект соответствует файлу.
- <ДополнительныеПараметры> - значение, которое было указано при создании объекта [ОписаниеОповещения](ОписаниеОповещения.md).

Описание:

Начинает определение, соответствует ли файловый объект каталогу или файлу.

Доступность:

Тонкий клиент, веб-клиент, мобильный клиент, толстый клиент, мобильное приложение (клиент).

См. также:

[Файл](Файл.md), метод [ЭтоКаталог](Файл.ЭтоКаталог.md)  
[Файл](Файл.md), метод [ЭтоКаталогАсинх](Файл.ЭтоКаталогАсинх.md)  

Использование в версии:

Доступен, начиная с версии 8.3.6.

Описание изменено в версии 8.3.18.
