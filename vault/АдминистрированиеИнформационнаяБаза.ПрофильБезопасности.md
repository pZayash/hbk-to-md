АдминистрированиеИнформационнаяБаза (AdministrationInfoBase)

ПрофильБезопасности (SecurityProfile)

Доступен, начиная с версии 8.3.14.

Использование:

Чтение и запись.

Описание:

Тип: Строка.   
Содержит имя профиля безопасности информационной базы.   
Если не указано, то информационная база имеет полные права на внешнюю активность.   
Если имя профиля безопасности указано, то поведение информационной базы регламентируется этим профилем.

Доступность:

Тонкий клиент, сервер, толстый клиент, интеграция.

Использование в версии:

Доступен, начиная с версии 8.3.14.
