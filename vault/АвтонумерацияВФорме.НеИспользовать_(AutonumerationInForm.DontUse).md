АвтонумерацияВФорме (AutonumerationInForm)

НеИспользовать (DontUse)

Доступен, начиная с версии 8.1.

Описание:

Автонумерация в форме не используется. Номер формируется при записи объекта.

Использование в версии:

Доступен, начиная с версии 8.1.
