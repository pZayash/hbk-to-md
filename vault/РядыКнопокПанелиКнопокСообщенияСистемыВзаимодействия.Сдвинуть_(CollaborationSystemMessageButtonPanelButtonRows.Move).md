РядыКнопокПанелиКнопокСообщенияСистемыВзаимодействия (CollaborationSystemMessageButtonPanelButtonRows)

Сдвинуть (Move)

Доступен, начиная с версии 8.3.25.

Синтаксис:

Сдвинуть(<Индекс>, <Смещение>)

Параметры:

<Индекс> (обязательный)

Тип: [Число](lang__def_Number.md), [РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия](РядКнопокПанелиКнопокСообщенияСистемыВзаимодействия_(CollaborationSystemMessageButtonPanelButtonRow).md).   
Индекс объекта или сам объект, который требуется сдвинуть.

<Смещение> (обязательный)

Тип: [Число](lang__def_Number.md).   
Количество элементов, на которое необходимо переместить элемент.

Описание:

Сдвигает указанный ряд кнопок панели кнопок сообщения системы взаимодействия на заданное количество позиций.

Доступность:

Сервер, толстый клиент, внешнее соединение.

Использование в версии:

Доступен, начиная с версии 8.3.25.
